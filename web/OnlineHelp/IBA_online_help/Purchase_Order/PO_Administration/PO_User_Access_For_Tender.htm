PO User Access For Tender

From this function you can Provide rights for accessing the tenders and Query for the existing user access rights units.

This function allows providing access rights for creating, finalizing/authorizing, cancelling and deleting the tenders.

Providing PO User Access For Tender

From the PO Administration, select the PO User Access For Tender function.

The PO User Access For Tender screen gets displayed.

  1. Click [Create] to provide user access rights for tender.

  2. Click Master Type [?] to select the user name. The logged in user name is displayed, by default. This is a mandatory field.

  3. Click [Search] to display transaction details.

  4. Enter the details in the mandatory (*) fields. <Related Field Description>

  5. Click [Apply] to save the PO User Access rights for tender or [Reset] to clear the entered data.

  6. Click [Query] to invoke the Query screen.
