Purchase Order

From this function you can Create purchase orders. You can Query for the existing purchase orders.

This function allows the Purchase Units to create purchase orders to the Suppliers in two ways:

  • Direct Order – Allows you to raise an order directly by adding items and defining the quantity of the required items in the order without a purchase request.

  • Order Based on Request – Allows you to raise a purchase order from a single or multiple purchase requests where all the items in the purchase request can be taken up for the purchase order.

This function supports generation of Blanket Purchase orders. Blanket Purchase Orders help in avoiding preparation of purchase orders for ongoing purchases of goods like consumable supplies that are frequently required, with a commitment from the vendor over a specified period of time at the agreed prices.

Creating Purchase Orders

From the PO Transactions, select the PO Purchase Order function.

The PO Purchase Order screen gets displayed.

  1. Click [Create] to invoke the Create screen. The Purchase Orders can be created in two ways:

Direct Purchase Order

Order Based on Request

  1. Click [Query] to query for the existing purchase orders.
