Enter the following details:

  1. PO Mode – Select ‘Order Based on Request’ from the PO Mode list to create purchase orders using the purchase requests.

  2. Purchase Unit – Select the Purchase Unit from which the purchase order is raised. This is a mandatory field.

  3. Supplier – Select the Supplier to which the purchase order is placed. This is a mandatory field.

  4. Purchase Type – Select the Purchase Type from the list. This is a mandatory field.

  5. PO Date – Select the Date by clicking the calendar function. By default, the current date is displayed.

  6. PO No – Enter the purchase order number. If the purchase number generation rule is set as ‘Manual entry’, you can enter Number up to 20 characters maximum length.  If the purchase number generation rule is set as ‘Autogenerate’, then this field is disabled.

  7. PO Ref – Enter the PO Ref if any.

  8. Order Remarks – Click ‘Order Remarks’ hyperlink to enter remarks for the purchase order.

  9. Additional Details – Click ‘Additional Details’  hyperlink to view or modify the details of the supplier.

    1. Quotation No/Tender Ref – Enter the quotation number or the tender reference number in alphanumeric format of 10 characters maximum length.

    2. Supplier Account Number – The Supplier Account is displayed from the Supplier master. It can be modified if required.

    3. Supplier Address Line 1, Line 2 and Line 3 – The Supplier Address is displayed from the Supplier master.  The supplier address can be modified if required.

    4. Payment Terms - Select the Payment Terms from the list.

    5. Terms and Conditions – Select the Terms and Conditions from the list.

    6. Delivery Mode – Select the Delivery Mode from the list

    7. Deliver To Address – The Deliver To address for the purchase order is displayed from the last entered deliver address for the supplier from the purchase unit. It can be modified if required.

    8. Bill To Address – The Bill To address for the purchase order is displayed from the last entered bill address for the supplier from the purchase unit. It can be modified if required.

    9. Multi-Facility Deliver To & Bill To - Click Multi-Facility Deliver To and Bill To hyperlink to enter the Deliver To and Bill To address individually for each facility.

    10. OK – Click [OK] to close the Additional Details dialog.

  10. Req Doc No – Select the Requested Purchase Order number using the [?] function. The Purchase Request Details dialog is displayed.

    1. Item – Enter the Item or select using [?] function.

    2. Facility – Select the Facility from the list.

    3. Store – Select the Store from the list.

    4. From Date and To Date – Select the Date by clicking the calendar function.

    5. Req Doc No – Enter the Requested Document Number.

    6. Search – Click [Search] to view the purchase requests with the corresponding items.  If you click [Search] without entering any search criteria, the list of all finalized purchase requests and partially ordered purchase requests are displayed.

    7. Select the required Purchase Request check box and select the check box against the corresponding items for the purchase order.

    8. Modify the Order Quantity if required and click [Add]. The items are listed under the Purchase Order Details.

  11. Expected Delivery Date – Select the Expected Delivery Date by clicking the calendar function. This is a mandatory field.

  12. Transaction Currency - Select the Transaction Currency from the list. The Transaction Currency value is listed based on the Supplier selected. This is a mandatory field.

  13. Exchange Rate – Enter the Exchange Rate for the transaction currency. This is a mandatory field.

  14. Local Currency – The Local Currency for the facility is displayed by default.

  15. Old Purchase Order No – Enter the Old Purchase Order Number if any, in alphanumeric format of 20 characters maximum length.
