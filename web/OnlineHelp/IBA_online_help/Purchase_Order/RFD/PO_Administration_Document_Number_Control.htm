Enter the following details:

  1. Item Level Order Approval Required - Select the Item Level Order Approval Required check box to enter the approval reference number for each item during the PO transaction.

  2. Allow Override of Order Delivery Date – Select the value from the drop-down list. The listed values are ‘Warning’, ‘Error’ and ‘Not Applicable’.

  3. Warning – When the value ‘Warning’ is selected, the system allows receiving of items within a Goods Received Note (GRN) if the GRN date is not equal to the PO Delivery Due date but with a prompt message.

  4. Error – When the value ‘Error’ is selected, the system does not allow to proceed with the transaction.

  5. Not Applicable – When the value ‘Not Applicable’ is selected, there is no warning or restriction.

  1. Allow Override of Order Quantity During Receiving – Select the value from the drop-down list. The listed values are ‘Warning’, ‘Error’ and ‘Not Applicable’.

  2. Warning – When the value ‘Warning’ is selected, the system allows receiving of items within a GRN if the total GRN quantity is more or less than total PO quantity but with a prompt message.

  3. Error – When the value ‘Error’ is selected, the system does not allow to proceed with the transaction.

  4. Not Applicable – When the value ‘Not Applicable’ is selected, there is no warning or restriction.

  1. Allow Override of Order Price During Receiving – Select the value from the drop-down list. The listed values are ‘Warning’, ‘Error’ and ‘Not Applicable’.

  2. Warning – When the value ‘Warning’ is selected, the system allows receiving of items within a GRN if the GRN price is not equal to PO price but with a prompt message.

  3. Error – When the value ‘Error’ is selected, the system does not allow to proceed with the transaction.

  4. Not Applicable – When the value ‘Not Applicable’ is selected, there is no warning or restriction.

  1. Purchase Order Mandatory for Delivery Order – If Purchase Order Mandatory for Delivery Order check box is selected, purchase order details are mandatory for a delivery order. If it is unchecked purchase order details are not mandatory.

  2. Allow Change to Purchase Price During Delivery Order – If Allow Change to Purchase Price During Delivery Order check box is selected, you can modify price details while creating the delivery orders using Delivery Order screen.

  3. Header Remarks 1 – Enter the remarks to be printed in the report.

  4. Header Remarks 2 – Enter the remarks to be printed in the report.
