Enter the following details:

  1. Delivery Order Type – Select the Delivery Order Type from the list. This is a mandatory field.

  2. DO No – Enter the delivery order number up to 20 characters maximum length.

  3. DO Date – Select the Date by clicking the calendar function. By default, the current date is displayed.

  4. DO Ref – Enter the DO Ref if any.

  5. Supplier – Select the Supplier to which the purchase order is placed. This is a mandatory field.

  6. PO Search – Click [PO Search] Purchase Order Details dialog gets displayed. Enter the details. This field is mandatory if Purchase Order Mandatory for Delivery Order is selected in the PO Parameter screen.

  7. Additional Details – Click ‘Additional Details ’  hyperlink to view or modify the details of the supplier.

    1. Invoice No – Enter the invoice number in alphanumeric format of 10 characters maximum length.

    2. Payment Terms - Select the Payment Terms from the list.

    3. Invoice Date – Select the date using the calendar function.

    4. Terms and Conditions – Select the Terms and Conditions from the list.

    5. Invoice Received By – Enter the name of the person who has received the invoice items.

    6. Delivery Mode – Select the Delivery Mode from the list.

    7. Delivery Remarks – Enter the Delivery Remarks in the text box provided.

    8. OK – Click [OK] to close the Additional Details dialog

  8. Transaction Currency - Select the Transaction Currency from the list. The Transaction Currency value is listed based on the Supplier selected. This is a mandatory field.

  9. Exchange Rate – Enter the Exchange Rate for the transaction currency. This is a mandatory field.

  10. Local Currency – The Local Currency for the facility is displayed by default.
