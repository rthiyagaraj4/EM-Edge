Purchase Request Status

From this function you can Query about the current status of a purchase request.

This function allows querying about the current status of a purchase request against a store or a group of stores. It helps to list the purchase requests as per status. The purchase request statuses are Unfinalized Requests, Cancelled Request, Ordered Requests and Fully Approved Requests.

Querying Purchase Request Status

From the PO Queries, select the Purchase Request Status function.

The Purchase Request Status screen gets displayed.

  1. Enter the details in the respective fields. <Related Field Description>

  2. Click [Search].  The search result is displayed based on the entered search criteria.

  3. Click ‘Document Number’ hyperlink to view the details of the items in the purchase request in a ‘Item Details’ dialog. The item details include Item name, Description, UOM, Requested Quantity, Pending Order Quantity and Remarks.

  4. Click ‘Request Status’ hyperlink to view the request status details. The details include name of the user who entered the order, order entered date, last accessed user name and date.
