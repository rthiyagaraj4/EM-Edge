Nutrients

From this function you can Define a Nutrient and associate it to a Unit of measurement to define equivalent unit of measurements. You can Query for the existing Nutrient records.

The Nutrients function helps in defining different nutrients such as Vitamin A, B, C and Proteins against a valid Unit of Measurement along with the energy conversion details. These details are used for RDA (Recommended Daily Allowance Chart) comparison and checks.

Creating a Nutrient

From the Setup, select the Nutrients function.

The Nutrients screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a created Nutrient or [Reset] to clear the entered details.

  4. Click [Query] to search for the existing records.
