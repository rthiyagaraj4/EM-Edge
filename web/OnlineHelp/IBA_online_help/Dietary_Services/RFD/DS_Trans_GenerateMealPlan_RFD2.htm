Print Meal Plan

You can print the generated meal plan details using this tab. A meal list number is generated internally on generating the meal plan. You can select to print the meal diet list for the specific list or all lists generated using ‘Generate Meal Plan’ tab. To print the meal plan for entire combination, select  only the serving date. To print the meal plan for only the selected list, specify the combination selected under ‘Generate Meal Plan’ tab.

 

Enter the following details to print a meal plan:

 

  1. Diet Category - Select the Diet Category from the list for which the meal plan is generated. The system-defined values are Enternal, Normal and Therapeutic Diet. The Meal Plan is printed for the selected Diet Category.

  2. Meal Type - Select the Meal Type from the list. The values defined using the ‘Meal Type’ function get listed.

  3. Meal Class - Select the Meal Class from the list. The values defined using the ‘Meal Class’ function get listed.

  4. Diet Type - Select the Diet Type from the list. The values defined using the ‘Diet Type’ function get listed.  You can select more than one diet type.

  5. Feed Type – Select the required Feed Type from the drop-down list. The list displays the values defined in the Feed Type master.

  6. Nursing Unit - Select the Nursing Unit from the list. Nursing Units are defined in Inpatient and Day Care Management module.

  7. List Type - Select the required List Type from the list. The system-defined values are ‘Entire List’ and ‘Specific List’. Select ‘Entire List’ to print all the Diet Orders of the generated Meal Plan for the specified Serving Date. On selecting Entire List, the report is printed irrespective of the List Number. Select ‘Specific List’ to print the specified Diet Order of the generated Meal Plan for the selected Serving Date.

  8. List No - Select the List Number from the list. You can select a list number only if the List Type is selected as ‘Specific List’. List Numbers are generated for regenerated meal plans.  

Meal plans are printed for the given criteria, if the values are selected. Meal plan is generated for all Diet Orders, if the Serving Date is selected.
