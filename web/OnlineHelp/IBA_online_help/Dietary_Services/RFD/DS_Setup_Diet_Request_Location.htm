Enter the following details

 

  1. Location Code – Enter the location Code in alphanumeric format of four characters maximum length. This code is used for identifying location. This code is unique and cannot be changed after creation.

  2. Long Description – Enter the long description in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description - Enter the short description in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Default Diet Request location – Select the default diet request location. Based on this setup three diet types 'Vegetarian', 'Full Diet' and 'Western' are displayed in Meal Class drop down list in the 'Staff Order List' screen. You can select this check box for only one diet location code. If you try to select checkbox for some other diet location, an alert message 'Default diet location is already available' gets displayed

  5. Enabled - Select this checkbox to enable the record and clear the checkbox to disable the record.
