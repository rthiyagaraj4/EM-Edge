Nutrient Supplement

Select  to expand the Nutrient Supplement option and click to add or edit the food items for the Nutrient Supplement.

 

  1. Date [From-To] - Click the Calendar function to set the date and time range during which the Nutrient Supplement is to be served along with the diet order placed.

  2. Item Type – Select the Item Type from the list. The values defined using the ‘Item Type’ function get listed. This is a mandatory field.

  3. Item – Select the Item using [?] function. The Food Items associated to the selected Item Type get listed. This is a mandatory field.

  4. Meal Type – Select the Meal Type from the list. The values defined using the ‘Meal Type’ function get listed. ‘*All’ is displayed, by default.

  5. Qty – Enter the quantity to be served in numeric format.

  6. [Select] - Click [Select] to display the selected food item in the table. The item name, serving date range and the food item quantity gets displayed along with the preparation UOM selected for the respective Food item and the Energy UOM selected in ‘DS Parameter’ function. Select the Item Type displayed as a hyperlink. The details get displayed in the respective fields. Click [Cancel] to remove the selected food item from the list. Click [Close] to exit from the Nutrient Supplement window. The total calorie content of Nutrient Supplement is calculated and displayed.

If the patient dislikes a food item, it is displayed in red color. Click to collapse the Nutrient Supplement option.
