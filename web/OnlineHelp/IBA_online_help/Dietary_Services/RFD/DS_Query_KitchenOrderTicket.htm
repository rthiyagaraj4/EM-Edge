Enter the following details:

 

  1. Serving Date - Click the Calendar function to select the Serving Date. The system date is displayed, by default. You cannot modify the meal plan for the previous dates. This is a mandatory field.

  2. Kitchen - Select the Kitchen from the list. The values defined using the ‘Kitchen/Kitchen Area’ function get listed..

  3. Diet Type - Select the Diet Type from the list. The values defined using the ‘Diet Type’ function get listed. This is a mandatory field.

  4. Meal Type - Select the Meal Type from the list. The values defined using the ‘Meal Type’ function get listed.

  5. Meal Class - Select the Meal Class from the list. The values defined using the ‘Meal Class’ function get listed.

  6. Diet Category - Select the Diet Category from the list. The system-defined values are Enteral, Normal and Therapeutic Diet. The Meal Plan is cancelled for the selected Diet Category for all discharged inpatients.

  7. Order Type- Select the required order type from the list. The values are system defined.

 

  • Inpatients – Select Inpatients, if the Billing Class is applicable for Inpatients admitted in the Nursing Unit of a Facility.

  • Outpatients - Select Outpatients, if the Billing Class is applicable for Outpatients visiting Clinic in a Facility.

  • Daycare – Select Daycare, if the Billing Class is applicable for Daycare patients admitted in the Clinic of a Facility.

  • Emergency – Select Emergency, if the Billing Class is applicable for Emergency patients.

  • External – Select External, if the Billing Class is applicable for Referral or External patients.
