Enter the following details to create a Meal Type:

 

  1. Meal Type – Enter the Meal Type in alphanumeric format of four characters maximum length. The Meal Type has to be unique and cannot be changed after creation. This is a mandatory field.

  2. Long Description - Enter the long description of the Meal Type in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description of the Meal Type in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Order - Enter the Order in numeric format of two digits maximum length. The order number of the Meal Types has to be unique and cannot be changed after creation.

  5. Enabled – Select the check box to enable the Meal Type in the transaction functions. The check box is selected, by default.

  6. Place Inpatient Meal Order – Select the checkbox to map the meal type to inpatient. This checkbox is selected by default.

  7. Place Outpatient Meal Order – Select this check box to place the meal order for outpatients in the Place Meal Order for Outpatients function.

  8. Place Emergency Patient Meal Order – Select this check box to place the meal order for Emergency Patients in the Place Meal Order for Outpatients function.

  9. Place Supplementary Diet Order – Select this checkbox to allow configuring a meal type for Supplementary Diet Order.

  10. Place Staff Meal Order – Select the checkbox in the section. The Meal categories ‘Doctor On Call’, ‘OT/Paramedic Staff’, ‘On Call Staff’, ‘Staff’ and ‘Students’ are displayed with checkbox. The selected Meal Category is mapped with Meal Type. By default, the check boxes are not selected.

  11. Doctor On Call - Select the checkbox to allow configuring a meal type for Doctor On call. By default, the checkbox is cleared.

  12. OT/Paramedic Staff - Select the checkbox to allow configuring a meal type for OT/Paramedic Staff. By default, the checkbox is cleared.

  13. On Call Staff - Select the checkbox to allow configuring a meal type for On Call Staff. By default, the checkbox is cleared.

  14. Staff - Select the checkbox to allow configuring a meal type for Staff. By default, the checkbox is cleared.

  15. Students- Select the checkbox to allow configuring a meal type for Students. By default, the checkbox is cleared.

  16. Serving Start Time – Enter the time in hh:min format. The serving time is stared based on this time. By default this field is blank.

  17. End Time - Enter the time in hh:min format. The serving time is ended based on this time. By default this field is blank.

  18. Late/Irregular Diet Order Start Time - Enter the time in hh:min format. The irregular serving time is started based on this time. By default this field is blank.

  19. End Time - Enter the time in hh:min format. The serving time is ended based on this time. By default this field is blank.
