Meal Exclusions

 

Click to expand the Meal Exclusions option. Dietitians can view the meal exclusions and the food details of a patient by selecting the respective hyperlink. All excluded meals are displayed in ascending order of the Diet Order Date for orders

 

• Future active orders

• Past and Current orders active for current date – By default will be scrolled to the current date meal excluded.

• Past/Current/Future inactive orders.

 

  1. Click [Edit] button to display the ‘Meal Exclusions’ dialog with the followings fields.

  2. Date [From-To] – Select the date range using calendar function. These are mandatory fields.

  3. Meal Type – Select the required meal type check box based on which the Diet order date details will be displayed. The meal types are Breakfast, Dinner and Lunch

  4. Click [Select] button to view the diet order details. The details are displayed with following symbols.

 - Meals, which are excluded for past date, cannot be revoked.

 - Meals, which are excluded for current and future date, can be revoked.

A tool tip ‘Click to revoke’ is displayed when the mouse is hovered over this icon. You can revoke a meal by clicking on this icon. Once clicked, this icon gets disappeared against the respective date and meal rowl. On clicking [OK] button, the respective meal (and date) is removed from ‘Meal Exclusions’ menu list view in Place Diet Order screen. A meal is permanently revoked only when the Place Diet Order screen is saved.
