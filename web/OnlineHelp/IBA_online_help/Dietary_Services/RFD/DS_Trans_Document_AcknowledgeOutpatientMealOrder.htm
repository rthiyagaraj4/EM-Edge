Enter the following details:

 

Meal Order For

 

  1. Outpatient – Select the radio button to place meal order for outpatients.

  2. Emergency Patient - Select the radio button to place meal order for Emergency Patients.

  3. Date – Select the date using the calendar function. This is a mandatory field.

  4. Meal Type - Select the Meal Type from the list. The values defined using the ‘Meal Type’ function get listed. This is a mandatory field. The values are sorted based on the order of the meal type defined in the Meal type screen.

  5. Specialty – Select the Specialty from the drop-down list. This is a mandatory field.

  6. Location Type - Select the location type from the drop-down list. This field is mandatory. The Location Type values listed are ‘*All’, ‘Clinic’, ‘Day Care’ and ‘Procedure Unit’. On selecting the required Location Type, the Locations listed are based on the following conditions:

  7. All the Clinics/Daycare/Procedure units defined under the selected location type that allow appointments (based on the ‘Allow Appointment’ value set in ‘Clinic’, Daycare Unit’ and ‘Procedure Unit’ setup function of EM Outpatient Management module).

  8. All the defined Clinics/Daycare/Procedure units must be associated to the operator station assigned to the logged in user.

  1. Location – Select the Location using [?] function.

  2. Resource Type - Select the Resource Type from the list. These values are populated from the Application Masters module based on the Resource Class identified. This field is optional and is used to identify if any specific Resource Type is required to cater to the selected Visit Type

  3. Resource - Select the Resource Class from the list. These values are system defined and the list of values provided are ‘Practitioner’, ‘Room’, ‘Equipment’, and ‘Others’.

  4. Set Status – The values – Meal Prepared, Meal Served, Meal Received and Meal Reject are system defined. Select a Status from the list. You can acknowledge or document about the meal order against the Status selected.

  5. Treatment Area – Select the treatment area from the drop-down list. The treatment area associated with location will be listed out.

  6. Date Time for all Patients - Select the Date and Time using the calendar function.

If you select the date/time different from the event date in the ‘Date/Time’ field in list view, an error message ‘Acknowledgement date should be equal to event date’ is displayed.

If you select the future date as status date, the error message ‘Status Date Cannot be Future Date; is displayed on clicking the Apply button.

If you select the Status Date different from Visit date or blank, the error message ‘Status Date Should be Same as Visit Date’ is displayed when Apply button is clicked.
