Enter the following to modify a Meal Plan:

 

  1. Serving Date - Click the Calendar function to select the Serving Date. The system date is displayed, by default. You cannot modify the meal plan for the previous dates. This is a mandatory field.

  2. Kitchen - Select the Kitchen from the list. The values defined using the ‘Kitchen/Kitchen Area’ function get listed. This is a mandatory field.

  3. Encounter ID – Enter the Encounter ID of an inpatient to update the meal plan for the respective encounter. This is a mandatory field.

  4. Frequency? – Select the Check box if the Meal Plan generated for a patient is ‘Frequency Based’. Clear the check box, if the meal plan is non-frequency based. 

  5. Diet Type - Select the Diet Type from the list. The values defined using the ‘Diet Type’ function get listed. This is a mandatory field.

  6. Meal Type - Select the Meal Type from the list. The values defined using the ‘Meal Type’ function get listed. This is a mandatory field. The values are sorted based on the order of the meal type defined in the Meal type screen.

 

Item Details

  1. Srl No. – Enter the Serial Number in numeric format of two digits maximum length.

  2. Item Type – Select the Item Type from the list. The values defined using the ‘Item Type’ function get listed. This is a mandatory field.

  3. Item - Select the Item using [?] function. The Food Items associated to the selected Item Type get listed. This is a mandatory field.

  4. Serving Qty – Enter the Serving Quantity in numeric format of six digits maximum length.

  5. Schedule Time – The time schedule recorded for serving the Diet Type based on the Meal Type is displayed and disabled. You cannot modify this field.

  6. Special Food – The check box is selected and disabled if the Meal Plan is generated for Special Food items. You cannot modify this field.

  7. Frequency – The check box is selected and disabled if the Meal Plan is generated for Frequency based Special Food Items. You cannot modify this field.
