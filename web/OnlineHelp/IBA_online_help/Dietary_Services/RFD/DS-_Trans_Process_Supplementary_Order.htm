Enter the following details to process the supplementary order:

 

  1. Date – Click Calendar function to select a date to acknowledge the meal prepared and served or record complaints on meal prepared, meal received, meal served or meal rejected. The current date is displayed, by default.

  2. Patient ID – Select the Patient ID using [?].

  3. Kitchen - Select a Kitchen from the list. The values defined using the ‘Kitchen/Kitchen Area’ function get listed. The Meal Plan is generated for the selected Kitchen.

  4. Nursing Unit - Select the Nursing Unit from the list. Nursing Units are defined in Inpatient and Day Care Management module.

  5. Supplementary Order For – Select ‘Both’ or ‘Patient’ or ‘Attendant’ from drop down list. By default, the value ‘Both’ is selected.

  6. Set Status for all Patients – Select a value from the Status drop down. Based on the selected value, the diet order is displayed. The listed values are ‘All’, ‘Processed by Kitchen’, ‘Rejected by Kitchen’, ‘Accepted by Patient/Attendant’ and ‘Rejected by Patient/Attendant’. By default the value ‘All’ is displayed.
