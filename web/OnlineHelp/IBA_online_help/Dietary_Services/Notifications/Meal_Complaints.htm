Meal Complaints

From this function you can View notifications about the complaints registered in ’Document/Acknowledge Meal’ function for all the meal status except Reject.

The Meal Complaints function is used to view notifications every time a complaint is registered on the 'Document / Acknowledge meal' screen for all the meal status except Reject. You can acknowledge the status of the meal complaint.  

Viewing Meal Complaints

From the Notifications, select the Meal Complaints function.

The Meal Complaints screen gets displayed.

  1. Enter the details in the in the respective fields. <Related Field Description>

  2. Click [Search] to view notifications about the complaints registered in  ’Document/Acknowledge meal’ screen for all the meal status except Reject.

  3. The search details include Serving Date, Patient ID, Patient Name, Patient Class, Location, Meal Type, Meal Status, Complaints, Complaint Date & Time, Acknowledged By, Acknowledged Date and Time.

  4. To acknowledge a meal complaint, select  ‘Yes’ value  from Acknowledge from drop-down list. By default, the value ‘No’ is selected.

  5. Click [Apply] to save the Acknowledgement status of Meal complaint or [Reset] to clear the entered data.

  6. On applying, the Acknowledge column with Yes value gets disabled. The Acknowledged By and Acknowledged Date and time gets displayed in the respective columns.
