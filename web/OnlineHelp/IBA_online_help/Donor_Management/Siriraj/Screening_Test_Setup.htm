Screening Test Setup

From this function you can Create tests for screening the blood units and blood bag and Query for the existing records.

The Screening Test Setup function is used to create tests for screening the blood units and blood bag. The screening tests created are applicable for Apheresis, Blood Bag and Deferred screening if the check box is selected in the Applicable for section. You can select the tests for screening, define type of answers, validations for tests with numeric results (if any) and list of values and actions for the respective values. Screen based tests are normal tests, whereas Rules based tests are requested while entering the questionnaire in the Pre Donation function. A suspended test cannot be used for blood screening unless you reinstate it. The suspended test codes are displayed in grey color.

Creating Screening Tests

From the Administration, double-click the Screening Test Setup function.

The Screening Test Setup screen gets displayed.

  1. Select the value from the list to specify whether the test setup made is valid for ‘Screening’ or ‘Rule Based’ tests.  

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Commit Form] or press [F10] to save or click [Clear Record] to clear the entered details.

  4. Click [Execute Query] or press [F8] to search for the existing records within the facility.
