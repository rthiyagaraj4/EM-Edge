Donor Audit Trail

From this function you can Query to view the changes that are made to the donor details.

The Donor Audit Trail function is used to view the changes that are made to a selected donor. You can view the name of the user who has made the changes, the old and new values. Search option is available to filter the results based on user or period.

Donor Audit Trail

From the Transactions > Blood Donor, double-click Donor Audit Trail function.

The Donor Audit Trail screen gets displayed.

  1. Select the date range from the Calendar function to search for audit trail within a specified date range.

  2. Click [List of Values] to select the name of the user who has made changes to the donor details.

  3. Click [List of Values] to select the Donor ID and click [Search] to display the audit trail details.

  4. The Donor ID, Name, Gender and Age of the Donor are displayed.

  5. You can also view the modifications done to the Field Names, namely, Old Field and New Field.
