Screening Test Setup

From this function you can Define the validation for screening tests to be accepted or rejected . You can Query to view the existing records.

The Screening Test Setup function enables you to define the validation for screening tests to be accepted or rejected.  You can define any screening test for the blood units. The tests are defined through ‘Test Codes’ function in Laboratory module. You can also suspend a particular test for a temporary period.

Defining Screening Test Setup

From the Administration, double-click Screening Test Setup function.

The Screening Test Setup screen gets displayed.

  1. Click [List of Values] to select a Screen Test Code. The screen test codes are created in the ‘Test Codes’ function in Laboratory module.

  2. Select the check box ‘Suspend Screen Test?’ to suspend the selected test for a temporary period. If you clear the check box, the test is mandatory.  When you clear the check box an error is displayed ‘This code cannot be reinstated’, if the selected code’s status in 'Referred in Tests' function in Clinical Laboratory module is ‘Suspended’.

  3. Enter test details and associate valid values. <Related Field Description>

  4. Click [Commit] or press [F10] to save or click [Clear Record] to clear the entered details.

  5. Click [Execute Query] or press [F8] to search for the existing records.
