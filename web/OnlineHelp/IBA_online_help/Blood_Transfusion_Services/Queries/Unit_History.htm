Unit History

From this function you can Query the Unit History of the blood unit.

The Unit History function enables you to view the details of the stages a unit has undergone before acquiring the current status. The various stages can be allocated, reserved, expired, issued, returned, available, transfused, etc.

Querying Unit History

From the Queries, double-click Unit History function.

The Unit History screen gets displayed.

  1. Click the Calendar function to select the date range.  

  2. Click [Execute Query] or press [F7] to view all the records matching the search criteria.

  3. To search for specific records enter the Unit Number/Specimen Number or Patient ID.

  4. Enter the Patient ID or right-click on ‘Right Click for User Option’ to select it. You can search a patient record either using Patient Look or External Patients option.  

  5. Enter the Unit Number, Storage Location, Specimen Number and click [Execute Query] or press [F8] to view the unit history details. The unit history details are displayed.

  6. Double-click the selected unit to view complete details.
