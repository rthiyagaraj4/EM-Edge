Transfusion History

From this function you can Query the Transfusion History of a selected patient and Print the Transfusion History.

The Transfusion History query function enables you to view the transfusion details of a selected patient. You can print the Transfusion History of a patient. The Transfusion details include Unit Number, Product code, Blood Group, Rhesus code, Specimen Number and Date, Reserved by, Practitioner, Start and End time, Volume Transfused and Reactions, if any. The Transfusion Remarks and Partial Transfusion Reason and Description that are recorded during 'Blood Unit Consumption' are displayed. The Current Unit Numbers are highlighted in red.

Querying Transfusion History

From the Queries, double-click Transfusion History function.

The Transfusion History screen gets displayed.

  1. Enter the Patient ID or right-click on ‘Right Click for User Option’ link to select it. You can select a patient ID using Patient Lookup or External Patients option. Select any option to query and select a patient ID.

  2. Click [Execute Query] or press [F8] to view the transfusion details. You can also press [Tab] to view the transfusion details.

  3. Click [Print] to print the transfusion details. You can also view the file in a PDF format and save the file in HTML, Delimited, RTF and PDF formats.

  4. Click [Details] to view the patient details.

  5. Click [Antibodies] to view the recorded Antibodies for the selected patient.

  6. Click [Requirements] to view  the recorded Transfusion Requirement details of the selected patient.
