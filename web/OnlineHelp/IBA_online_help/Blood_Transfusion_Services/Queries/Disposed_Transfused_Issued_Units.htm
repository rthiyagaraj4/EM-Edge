Disposed Transfused Issued Units

From this function you can Query and Print the details of blood units that are transfused, issued or disposed from a Storage Location.

The Disposed/Transfused/Issued Units function enables you to view the details of the blood units that have been issued, transfused or disposed during the selected date range. You can print the displayed details.

Querying Issued, Transfused and Disposed Blood Units

From the Queries, double-click Disposed/Transfused/Issued Units function.

The Disposed/Transfused/Issued Units screen gets displayed.

  1. Click the Calendar function to select date range and view Disposed/Transfused/Issued Unit details. This is a mandatory field.

  2. Select the Indicator Disposed/Transfused/Issued from the list to view the Disposed/Transfused/Issued blood units.

  3. Click [List of Values] to select the reaction range. The records matching the query criteria are displayed.

  4. Press [Tab] to view the queried details. The queried details are displayed based on the indicator selected.

  5. Click [Print]  to print the Disposed/Transfused/Issued Units details of the blood unit. You can also view the file in a PDF format and save the file in HTML, Delimited, RTF and PDF formats.
