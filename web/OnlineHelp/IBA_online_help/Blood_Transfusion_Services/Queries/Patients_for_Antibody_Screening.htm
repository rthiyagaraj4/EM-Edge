Patients for Antibody Screening

From this function you can Query about the details of patients available for Antibody Screening and Print the request letter for Antibody Screening.

The Patients for Antibody Screening query function enables you to search for patients available for antibody screening and print the request letter for Antibody Screening. You can print the letter whether the request letter for screening is already printed or not. You can filter the display of records by selecting the date range. If the letter is printed already, the specimen is highlighted with the color indicator.

Querying the Patients for Antibody Screening

From the Queries, double-click Patients for Antibody Screening function.

The Patients for Antibody Screening screen gets displayed.

  1. Select the value from the ‘Letter Printed’ field. The listed values are Printed, Not Printed and Both. The value Not Printed is displayed, by default. The records are listed based on the value selected.

  2. Click the Calendar function to select the date range and click [Search] to view the details. The details are displayed based on the query criteria.

  3. Select the ‘Select All’ check box to print letter for all patients or select the ‘Print’ check box against each Specimen to print the letter for the selected patient.

  4. Click [Print] to print the request letter. Select View, Print or File to view the report, print the report or save the report.
