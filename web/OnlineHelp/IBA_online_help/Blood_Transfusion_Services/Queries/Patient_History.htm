Patient History

From this function you can Query the transfusion history of a patient and Print the details.

The Patient History function enables you to view the transfusion history of the patient or the details of Reserved blood units for the patient. You can print the transfusion history of a patient using this function.  

Querying Patient History

From the Queries, double-click Patient History function.

The Patient History screen gets displayed.

  1. Enter the Patient ID or right-click on ‘Right Click for User Option’ to select it. You can select a patient ID using Patient Lookup or External Patients option.

  2. Select the Status ‘Reserved’ to view the history of the list of Reserved blood units for the patient. Select the Status as ‘Transfused’ to view details on the blood units transfused to the patient. Select 'Reserved' to view the details on blood units reserved for the patient. You can select 'Both' to view the details matching both the values.

  3. Click [Execute Query] or press [F8] to view the details matching the query criteria. You can also press [Tab] to view the details. The Transfusion Details and Donor Details are displayed.

  4. Click [Print] to print the Patient History. You can also view the file in a PDF format and save the file in HTML, Delimited, RTF and PDF formats.
