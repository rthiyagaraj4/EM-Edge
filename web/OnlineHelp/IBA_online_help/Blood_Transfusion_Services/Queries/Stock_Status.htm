Stock Status

From this function you can Query the Stock Status of a blood product and Print the details of the specimens.

The Stock Status function enables you to view the current stock details of a selected blood product. The number of units that has been used in any of the transactions is displayed under the heading In Process. The number of aliquoted units that has been used in any of the transactions is displayed in the Aliquot B1 field. The number of units available in the stock for use is displayed in Available for Use heading. The number of aliquoted units available in the stock for use is displayed in the Aliquot C1 field. The calculated Total Stock and the calculated aliquoted is displayed under Total Stock (B+B1+C+C1) heading. You can also print the details of the selected specimens or all the specimens.

Several color indicators are used to indicate the status:

- In Process

- Aliquot Units

- Available for Use

- Expired not Disposed

Querying Stock Status

From the Queries, double-click the Stock Status function.

The Stock Status screen gets displayed.

  1. Click [List of Values] to select the Product code .

  2. Click [Tab] to view the details of the selected blood product.

  3. Click [Details] to view the stock details of a selected Product. The color code on the screen indicates the status of the blood units such as In Process, Aliquot Units, Available for Use and Expired not Disposed.

  4. Click [Print] to print the details of a selected specimen or all the blood products. You can view, print or save the report in a file.
