Enter the following details:

 

  1. Centre Code – The Centre Code defined in the BD Parameters function of the Blood Donor module is displayed.

  2. Display – Select the criteria for displaying the details from Display list. The valid values are, ‘All’ and ‘Pending for Acknowledged.

  3. Date [From–To] - Select the Date range to perform the query to acknowledge the receipt of blood units from blood donor stock to the blood transfusion stock.

  4. Requisition Number – The Requisition Number for which Blood Unit issued is displayed. The requisition number is generated in the Request Blood Unit function.

  5. Product Code – The Product Code for which the requisition is made is displayed. The list displays the products defined in the Product Codes function.

  6. Blood Group – The Blood Group of the requested Unit for which acknowledgement is to be done is displayed. The list displays the values defined in the ‘Blood Groups’ function of the Medical Records module.

  7. Rhesus Code - The Requested Rhesus Code for which Blood Unit acknowledgement is to be done is displayed. The list displays the values defined in the Rhesus factor function of the Medical Records module.

  8. Barcode Unit - Product Code – Scan the blood unit bag that is going to be issued. When the blood bag is scanned, the system displays all the details of the blood bag. You can also enter the barcode in the text box.
