Click [View Multi-Allocation] to view the units allocated for multiple specimens. The following details are displayed.

 

  1. Specimen No – The Specimen Number for which the selected unit has been allocated is displayed. The current specimen gets highlighted with a color code to indicate the status of the specimen. The color codes are defined in the screen.

  2. Patient ID – The Patient ID associated with the specimen is displayed.

  3. Unit Number – The Unit Number associated to the specimen is displayed.

  4. Product – The Product code of the selected unit number is displayed.

  5. Unit Status – The status of the selected blood unit is displayed.

  6. Allocated/Reserved/Issued Date – The Date when the unit is Allocated, Reserved or Issued is displayed.

  7. [Unit Antibodies] – Click [Unit Antibodies] to view the Antibody details of the unit.

  8. [Close] – Click [Close] to close the screen.

The current specimen is highlighted in yellow color.
