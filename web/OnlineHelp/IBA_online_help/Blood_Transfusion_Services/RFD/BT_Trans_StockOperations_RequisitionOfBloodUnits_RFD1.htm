Enter the following details:

 

  1. Product Code – Click [List of Values] to select the Product Code to request for the blood unit of the selected product. The list displays the values defined in ‘Product Codes’ function. On selecting the product code, the description of the product code is displayed.

  2. Blood Group – Click [List of Values] to select the Blood Group to request for the blood unit of the selected Blood Group. The list displays the values defined in the ‘Blood Groups’ function of the Medical Records module.

  3. Rhesus – Click [List of Values] to select the Rhesus factor of the blood group to request for the blood unit of the selected Rhesus factor. The list displays the values defined in the Rhesus factor function of the Medical Records module.
