The following details are displayed:

 

  1. Unit Number – The Unit Number satisfying the specified query criteria is displayed.

  2. Product Code – The Product Code of the blood unit is displayed. The list displays the values defined in Product Codes function.

  3. Description – On selecting the product code, the description of the product code is displayed.

  4. Blood Group – The Blood Group of the blood unit is displayed. The list displays the values defined in the Blood Groups function of the Terminology and Coding Management module.

  5. Rhesus – The Rhesus factor of the blood unit is displayed. The list displays the values defined in the Rhesus factor function of the Terminology and Coding Management module.

  6. Donation Date – The Donation Date of the unit recorded in Record Donation function Blood Donor module is displayed.

  7. Expiry Date – The Expiry Date of the blood unit is displayed. The expiry date is calculated based on the donation date and the expiry period defined for the products in the Products Codes function.

  8. Added To Stock Date – The Date on which the unit added to the stock is displayed. The blood unit is added to the stock using the Blood Units Entry function.
