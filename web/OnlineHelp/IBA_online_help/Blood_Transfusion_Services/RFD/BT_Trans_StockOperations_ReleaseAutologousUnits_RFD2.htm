The following details are displayed:

 

Autologous Unit Details

  1. Patient ID – The Patient ID of the Patient satisfying the search criteria is displayed. If the patient ID is autologous, color is displayed near the patient ID. If the patient ID is directed,   color is displayed near the patient ID.

  2. Patient Name – The Name of the Patient recorded during patient registration in Register Patient function in Blood Donor is displayed.

  3. Unit Number – The Unit Number of the blood unit that is to be released is displayed.

  4. Product Code – The Product Code of the unit to be released is displayed. The product codes are defined in Product Codes function.

  5. Description – The Description of the product code defined in Product Codes function is displayed.

  6. Blood Group - The Blood Group of the unit is displayed. The list displays the values defined in the Blood Groups function of the Terminology and Coding Management module.

  7. Rhesus Code – The Rhesus Code of the unit is displayed. The list displays the values defined in the Rhesus factor function of the Terminology and Coding Management module.

  8. Added Date – The Added Date of the Blood Unit is displayed. The added date is recorded in the Pre Donation function.

  9. Storage Location – The current Storage Location of the blood unit is displayed. The values are defined in Storage Locations function.
