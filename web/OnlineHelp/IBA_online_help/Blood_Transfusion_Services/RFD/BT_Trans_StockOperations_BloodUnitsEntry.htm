Enter the following details:

 

  1. Blood Group – Click [List of Values] to select the Blood Group. The list displays the values defined in the ‘Blood Groups’ function of Medical Records module.

  2. Rhesus – Click [List of Values] to select the Rhesus factor of the Blood Group. The list displays the values defined in the ‘Rhesus Factors’ function in Medical Records module.

  3. Volume – The Volume of the product specified in the 'Product Code' function is displayed. Enter a volume in numeric format of five digits maximum length to modify the existing volume. The unit of measurement is displayed adjacent to the Volume.

  4. Collection/Supplier – Click [List of Values] to select the Collection/Supplier. The list displays the values defined in the ‘Collection/Supplier’ function.

 

Unit No

  1. Unit Number – Enter a unique Unit Number in alphanumeric format of 20 characters maximum length. The actual length of the Unit number is defined in the ‘BD Parameter’ function. The Unit Number is generated automatically, if the 'Generate Unit Number' check box is selected in 'Product Codes' and 'Collection Centre/Supplier' functions. First, system checks for 'Generate Unit Number' check box in 'Collection Centre/Supplier' function and then in the 'Product Codes' ' function. If the check box is cleared in both the functions, the unit number is not generated automatically.

  2. Original Unit Number – Enter the Original Unit Number of the product. The Original Unit number is the number of the blood unit brought from the external facility.

  3. Donation Date – The system displays the current date, by default. Click the Calendar function to enter the donation date.

  4. Date Of Expiry – The expiry date of the blood unit is displayed. You can modify expiry date without exceeding the Expiry date specified for the selected product in the Product Codes function.

  5. Storage Location – Click [List of Values] to select the Storage Location. The list displays the Storage Locations that is set as the default storage location in BD Parameter function.

  6. Donation Type – Select the Donation Type from the list. The system-defined values are ‘Autologous’, ‘Allogenic’ and ‘Allogenic-Direct’. The Donation types are selected for the specimen in the Pre donation function. It is used while transfusing the blood units.

  7. Patient ID – If the Donation Type is Autologous or Allogenic-Direct, it is mandatory to enter the patient ID of the specific patient.

  8. Volume Milli Liter(s) - By default, the volume entered in the ‘Volume’ field gets displayed. You can view and edit the value if required.

  9. Fate – The Fate code is displayed during query if a unit has been disposed of during the any transactions performed earlier.  

  10. Patient ID - The Patient ID for which the selected blood unit used during the any transactions performed earlier is displayed on query mode.

  11. Specimen No - The Specimen Number used during the any transactions performed earlier is on query mode.

  12. Unit Status - The unit status of the product if it has been used in any transactions earlier is displayed on query mode.

  13. Last Reserved Date - The last reserved date of the blood unit is displayed on query mode.
