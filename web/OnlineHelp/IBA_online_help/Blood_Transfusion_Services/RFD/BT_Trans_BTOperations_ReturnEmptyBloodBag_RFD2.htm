Enter the following details:

 

  1. Transfusion Tag Return Indicator – Select a value from the list. The system-defined values are ‘Both’, ‘Blood Bag’ and ‘Tag’. The current system date and time is displayed in the respective fields based on the selected Transfusion Tag Return Indicator value. When the value is selected as ‘Both’, date and time is displayed in ‘Tag Returned Date’ and ‘Bag Returned Date’ fields. Click the Calendar function to select any other date and time. The selected date and time has to be lesser than the current system date and time.

  2. Tag Returned Date – The current system date and time is displayed, if the ‘Transfusion Tag Return Indicator’ is selected as either ‘Both’ or ‘Tag’. Click the Calendar function to select any other date and time. The selected date and time has to be lesser than the current system date and time.

  3. Bag Retuned Date – The current system date and time is displayed, if the ‘Transfusion Tag Return Indicator’ is selected as either ‘Both’ or ‘Blood Bag’. Click the Calendar function to select any other date and time. The selected date and time has to be lesser than the current system date and time.
