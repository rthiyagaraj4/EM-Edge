The following details are displayed:

 

  1. Blood Typing patient – The Blood Group and Rhesus factor of the patient to whom the blood is issued is displayed.  

  2. Test – The Test code associated in the ‘Test and Allowed Values for Cross Match’ function in Laboratory module is displayed .

  3. Result Value – Click [List of Values] to enter the result value. The listed values are defined in the ‘Test and Allowed Values for Cross Match’ function in Laboratory module.

  4. Result Indicator – On selecting the Result value, the Result Indicator defined in the ‘Test and Allowed Values for Cross Match’ function in Clinical Laboratory module is displayed.

  5. Validation Status – The validation status of the blood unit is displayed. The displayed statuses are Compatible, Incompatible and Least Compatible. If the blood unit is not validated ‘Not Validated’ is displayed.  The validation status is displayed, on clicking [Validate].

  6. X Match Result – The result of the crossmatch test after validating the result value is displayed. The listed values are Compatible, Incompatible and Least Compatible.  Only if Compatible is displayed, you can issue the blood unit to the patient. If the results are Incompatible and Least Compatible, you have to reject the blood units.

  7. [Validate] – Click [Validate] to validate the test results.

  8. [OK] – Click [OK] to accept the results and return to the Result Emergency Blood Units screen or [Cancel] to quit the cross match process.

Mandatory tests are highlighted using color.
