Enter the following selection criteria:

 

  1. Unit Number – Enter the Unit Number of the aliquot units to search for a specific record and press [F7] or click [Execute Query].

  2. Product – Enter the Product code of the aliquot units to search for a specific record and press  [F7] or click [Execute Query].  

On entering query criteria, press [Tab] to view details satisfying selected query criteria.

 

Unit Aliquoted

  1. Unit Number – The aliquoted Unit Number of the specified query criteria is displayed.

  2. Product Code – The Product Code of the aliquoted unit is displayed.

  3. Description – The Description of the product defined in the ‘Product Codes’ function is displayed.

 

Aliquoted Unit Details

  1. Product – The Product code of the aliquoted unit of the selected Unit is displayed.

  2. Description – The Description of the product code is displayed.

  3. Unit Number – The Unit Number of the aliquoted unit is displayed.

  4. Date – The processed Date of the unit is displayed.

  5. Volume – The Volume of the unit is displayed.

  6. Units – The Unit of the volume defined in the Product Codes function is displayed.

  7. Donation Date – The Donation Date of the blood unit is displayed.

  8. Expiry Date – The calculated Expiry Date of the unit is displayed based on the expiry period specified for the product in Product Codes function.

  9. Remarks – The remarks recorded in Aliquoted Units transaction function is displayed (if any).
