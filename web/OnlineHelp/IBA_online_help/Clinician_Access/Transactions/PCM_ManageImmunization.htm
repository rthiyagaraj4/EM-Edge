Manage Immunization

From this function you can Manage Immunization for patients.

You can administer the scheduled vaccine, view the administered vaccine details, administer Adhoc vaccines using this function. You can view the Vaccine Schedules based on the Status. The statuses are displayed with a Unique color code. The administered vaccine details cannot be modified. You can record the allergy details on administering the Vaccine schedule.

Managing Immunization

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Charting, select Manage Immunization function.

The Manage Immunization screen gets displayed.

  1. Select the required Schedule from the Create Schedule list. The Categories defined in the Vaccine Category function with the Adhoc check box cleared are displayed in the list.

  2. Click [Go] to display the Confirm Schedule screen.

    The Confirm Schedule screen displays the list of Vaccine details that are scheduled in the Vaccine for Category function.

    The following details are displayed:

    1. Schedule Start Date – The system date and time is displayed as the Schedule Start Date. You can enter the Schedule Start Date in the format of dd/mm/yyyy hh:mm or select the required date using the Calendar function. The entered date and time should be greater than the selected patients date of birth. This is a mandatory field.

    2. Vaccine Name – The scheduled Vaccine Name is displayed. The Vaccines associated to the selected Vaccine Category in the Vaccine for Category function are displayed.

    3. Frequency – The recorded Frequency is displayed. The Frequency for the Vaccine is displayed based on the number of doses entered in the Vaccine for Category function. If the Frequency is not recorded, the value ‘0’ is displayed.

    4. Select – Select the check box to confirm the scheduled vaccine for administration. If the mandatory check box is selected in the ‘Vaccine for Category’ function, the Select check box is selected by default and disabled.

    Click [OK] to confirm the Schedule or [Cancel] to exit from the Confirm Schedule screen.

  3. The confirmed schedules are displayed in the Existing Schedules frame.

    1. The Existing Schedules frame displays Categories (with # symbol against the Category) defined in the Vaccine Category function with the Adhoc check box selected.

    2. Click the required Schedule hyperlink from the Existing Schedules frame to view the confirmed schedules. Click ‘All’ to view details of all the Schedules.

    3. Undo  icon  appears next to the created schedule. Click the icon to revert the vaccine added into the existing schedule while creating schedule or delete the entire vaccine category records based on below criteria.

    • Schedule within 48 hrs - login user who has added the vaccine schedules within 48 hrs can add or remove it.

    • If the Vaccine category schedules were not completed.

    • The statuses 'Completed, 'In Error' and 'Not Administered' will not be considered while adding or removing.

    1. The Undo icon will disappear from Existing schedule - against applicable vaccine category after 48 hrs.

    1. The Undo icon is not applicable for Age specific and Adhoc vaccine category.

  4. Click the required Schedule hyperlink from the Existing Schedules frame. The scheduled vaccine details are displayed in the right hand side of the screen. If Adhoc# Schedule is selected, the vaccine details are not displayed.

  5. Click ‘Vaccine Name’ hyperlink to administer the scheduled Vaccine and the Administer Details Tab is displayed.

  6. Click [Record] to record the administration details or [Clear] to reset the entered details.

  7. The recorded Administer details are displayed on the top half of the screen.

  8. Click Vaccine Name hyperlink to view the administration details. You can only view the Administered details.

  9. On administering the Immunization, an option to change the status of the administered vaccine to 'In Error' is introduced in case of incorrect entry while administering the Immunization. The status is changed from 'Completed' to 'In Error'. Select the administered vaccine and click [Mark Error] to record the error remarks. On marking the administered vaccine as erroraneous entry, the administered vaccine details are displayed with a unique color code.

  10. Click Allergy Tab if Allergy is recorded for the patient after recording details in the Administer Details Tab.

  11. Click All from the Existing Schedules frame to enable the Group By Schedule check box. The check box is selected, by default. The Vaccine Administer details are grouped based on the Vaccine Category and Schedule Dates.

  12. Select the 'Include In Error Records' to include the vaccination administered details which are marked as error.

  13. Select the check box against the required immunization record and click [Print]. The immunization detail gets printed. You can select only one immunization record for printing.

  14. You can view the Administered Schedule based on the status.  

    Select the Status from the ‘Show Status’ list on the top right corner of the screen. The listed values are Completed, Due, Elapsed, In Error and Not Administered. The Status is displayed based on the Schedule Date.

    On selecting the value ‘Not Administered’ from the Show Status list, the details of Vaccines not administered are displayed. The ‘Not Administered’ Vaccine details are displayed with a unique color code .

    On selecting the 'In Error' status, Show Status list displays the vaccination details which are marked as error. The 'In Error' vaccination details are displayed with a unique color code .

    The elapsed Vaccine details are displayed on selecting the value ‘Elapsed’ from the Show Status list. The elapsed administered Vaccine details are displayed with a unique color code .

    The Vaccine details that are due for administration are displayed on selecting the value ‘Due’ from the Show Status list. The Vaccine details which are due for administration are displayed with a unique color code .

    On selecting ‘Completed’ from the Show Status list, the completed administered Vaccine details are displayed. The Completed administered Vaccine details are displayed with a unique color code .
