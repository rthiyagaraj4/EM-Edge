View Tanner Staging History

From this function you can View the History of Tanner Staging details for selected patient for any encounter.

This function allows to view and print the history of tanner staging details for selected patient for any encounter.  You can search tanner staging history details based on date range and encounter ID.

Click  icon in the patient chart to invoke the View Tanner Staging History screen.

Viewing Tanner Staging History

From the CA Transactions > Clinician Access > Desktop Menu > View Chart Summary > View History > View Tanner Staging History.

The View Tanner Staging History screen gets displayed.

  1. By default, the Tanner Staging History details based on Encounter ID and Period gets displayed.

    • Period From - By default, the date populates for the one month. 'To Date' is current date and 'From Date' is previous month date same as current date.

    • Encounter ID - By default, the current encounter ID of the selected patient gets displayed.

  1. If required, you can change the 'Period From' and 'Encounter ID' fields and click [Search] to view data accordingly.

  2. The View Tanner Staging History details include Practitioner Name, Recorded Date/Time. Click ‘Recorded Date/Time’ hyperlink to view the already recorded tanner staging history details.

  3. You can only view the recorded Tanner Staging History detail and cannot edit it.

  4. Click [Print] button to print the tanner staging detail.
