Pathologist Work List

From this function you can Assign specimen number to pathologist across facilities for result reporting in multi-facility environment.

This function allows to select Specimen Numbers and assign it to Pathologist across Facility for Result-Report in the Multi-Facility environment.  

Also, allows users to view all the assigned Specimen Numbers for Result Reporting in Multi-Facility environment.

Assigning Pathologist across facilities

From the CA Desktop Menu > Order Management > select the Pathologist Work List function.  

The Pathologist Work List screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to search for the required specimen numbers

  3. Select the check box against the specimen record. You can select more than one specimen if required.

  4. Select the required Reporting Facility from the lookup. The list displays the active Facilities in the multifacility. This field is mandatory.

  5. Select the required Pathologist (Lab User) in that Reporting facility and one who has access to Laboratory Dept from the lookup. This field is mandatory.

  6. Remarks - Enter the remarks if required.

  7. Click [Assign] to assign the selected specimen numbers to the pathologist. Once the specimen number is assigned, only that assigned Pathologist can progress that specimen number to 'Resulted - Complete'.
