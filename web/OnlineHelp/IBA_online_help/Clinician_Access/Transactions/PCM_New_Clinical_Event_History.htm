New Clinical Event History

From this function you can View Clinical Event History details for laboratory, radiology, clinical note and medication type.  

This function allows you to view the laboratory, radiology, clinical note and medication history type based on Encounter ID, Date Range and Facility.

This screen displays search results in two frames, Left and Right. The Left Frame displays date time within the selected date range in descending order with vertical scroll bar. The Right Frame displays search results only for one history type.

Viewing New Clinical Event History

From the CA Transactions > Clinician Access > Desktop Menu > View Chart Summary > View History > New Clinical Event History.

The New Clinical Event History screen gets displayed.

  1. By default, displays the lab results for the current encounter ID for one week duration.

  2. Period [From - To] - By default, the system displays current week date in the From and To fields. You can change the data range if required. Current Date is displayed in the To field.

  3. Encounter ID - By default, the current encounter ID gets displayed. You can change the encounter ID if required.

  4. Facility Name - By default, the login facility gets displayed. If required, you can select desired facility of that organization from the drop-down list.

 

Laboratory tab

  1. By default, displays the lab results for the current encounter ID for one week duration.

  2. Select the ‘View By’ option as Individual or Panel. By default, the Panel option is selected.

  3. Click [Search] to view the lab results matching the search criteria.

  4. The Search results are displayed in two frames:

  5. Left Frame - Displays date time within the selected date range in descending order with vertical scroll bar.

  6. Right Frame - Displays search results only for one history type i.e., Laboratory.

  1. Three Sub tabs are displayed:

  2. Self - By default, this tab is selected. The Login Practitioner who has ordered the order catalogs for the selected patient, those results only shall be displayed in self-sub tab.

  3. My Specialty - Displays the results related to the Login practitioner specialty. The specialties mapped in practitioner setup are considered.

  4. All – All lab results are displayed without considering either Self and My specialty.

  1. Select the Quick Link from the filter drop-down list to view records based on quick link setup done in masters.

  2. To print the lab results, click the Print icon. You can print both the panel and individual tests.

  3. To generate graph, select the check box against the events and click [View Graph] button to generate the graph.

Clinical Note

  1. By default, displays the signed clinical note results for the current encounter ID for one week duration.

  2. Click [Search] to view the notes in collapsed mode by default in the right frame with latest one note in expanded mode.

  3. The Search results are displayed in two frames:

  4. Left Frame - Displays date time within the selected date range in descending order with vertical scroll bar. Click on the Desired date to view the signed notes in the right frame.

  5. Right Frame - Displays all notes in collapsed mode by default in the right frame with latest one note in expanded mode.

  1. Three Sub tabs are displayed:

  2. Self - By default, this tab is selected. The Login Practitioner who has signed the notes for the selected patient, those notes only shall be displayed in self-sub tab. Signed status to be considered in displaying notes in this tab.

  3. My Specialty - Login practitioner having specialties mapped in AM practitioner setup (Primary and Secondary) is considered while displaying notes in My specialty tab with applicability to Signed status.

  4. All – Displays all the signed notes without considering either Self and My Specialty but applicability to display signed notes only.

  1. Expanded and Collapse buttons allow to view the data in either in expanded and collapsible mode as required. For notes that are modified, a pencil icon appears next to the date hyperlink.

Radiology/Imaging

  1. By default, displays the radiology results for the current encounter ID for one week duration.

  2. Click [Search] to view the radiology results matching the search criteria.

  3.  The Search results are displayed in two frames:

  4. Left Frame - Displays date time within the selected date range in descending order with vertical scroll bar.

  5. Right Frame - Displays search results only for the history type i.e., radiology in collapsed mode by default in the right frame with latest one note in expanded mode.

  1. Three Sub tabs are displayed:

  2. Self - By default, this tab is selected. The Login Practitioner who has placed the radiology orders for the selected patient, those results only shall be displayed in self-sub tab.

  3. My Specialty – The Login practitioner who has placed the radiology orders for the selected patient, having specialties mapped in AM practitioner setup is considered while displaying results in My Specialty tab.

  1. Expanded and Collapsed buttons allows to view the data in either in expanded and collapsible mode as required.

Medication

  1. By default, the system displays the medication results for the current encounter ID for one week duration.

  2. Enter the required search criteria and click [Search] to view the medication results matching the search criteria.

  3. The Search results are displayed in two frames:

  4. Left Frame - Displays date time within the selected date range in descending order with vertical scroll bar.

  5. Right Frame – Displays three tabs namely Self, My Specialty and All.  

  1. Three Sub tabs are displayed:

  2. Self - By default, this tab is selected. The Login practitioner is the ordering practitioner who ordered items are displayed.

  3. My Specialty – The login practitioner primary and secondary specialty defined in AM practitioner setup is considered to filter the medication details and displayed in the My Specialty tab.

  4. All – There is no restriction for this tab.

The additional search fields:

  1. Generic Name – Select the required generic name from the lookup to search the desired generic name.  

  2. Drug Name- Based on the generic name selection, drug names is sorted in the lookup against the drug name field. In case, if drug name is selected, generic name to be default selected in Generic Name field.

  3. Status – Select the required status from the drop-down list. By default, the value ‘Active’ is displayed.

  4. Based on the value Drug Name selected in the Generic Name, Drug Name and Status field, the search results get displayed in three tabs:

  5. Drug Profile - By default, the system opens with Drug profile tab to display the Drug profile content as per input criteria.

  6. External Drug Profile – Click the tab to view if any external drug profile is available.

  7. Medical Item – Click the tab to view if any medical items is available.

Surgery Procedure

  1. By default, displays the surgery/procedure results for the current encounter ID for one week duration

  2. Click [Search] to view the surgery/procedure results matching the search criteria.  

  3. The Search results are displayed in two frames:

  4. Left Frame - Displays date time within the selected date range in descending order with vertical scroll bar.

  5. Right Frame - Displays search results only for the history type i.e., surgery/procedure in collapsed mode by default in the right frame with latest one note in expanded mode.

  1. Three Sub tabs are displayed:

  2. Self - By default, this tab is selected. The Performed by Practitioner who has performed Surgery/Procedures orders for the selected patient, those results get displayed in the Self tab.

  3. My Specialty –The Performed by Practitioner who has performed Surgery/Procedures orders for the selected patient, having specialties mapped in AM practitioner setup is considered while displaying results in My Specialty tab.

  4. All – All surgery/procedure results are displayed without considering either Self and My specialty tab.

  1. Expanded and Collapsed buttons allows to view the data in either in expanded and collapsible mode as required.

Medical Report

  1. By default, displays the medical report for the current encounter ID for one week duration.

  2. Click [Search] to view the medical report results matching the search criteria.

  3. The Search results are displayed in two frames:

  4. Left Frame - Displays date time within the selected date range in descending order with vertical scroll bar.

  5. Right Frame - Displays search results only for the history type i.e., medical report in collapsed mode by default in the right frame with latest one note in expanded mode.

  1. Three Sub tabs are displayed:

  2. Self - By default, this tab is selected. The Login practitioner who has prepared the medical report shall be displayed in this tab.

  3. My Specialty - The login practitioner mapped primary and secondary specialty in AM practitioner setup matching same specialties for other practitioners who prepare medical report also gets displayed in My Specialty tab.
