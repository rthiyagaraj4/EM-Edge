Assign Coder for Coding function

From this function Supervisors can Assign patient records  to Coders to perform coding function.

This function allows the supervisors to assign patient records to Coders to perform coding function. The hyperlink is visible only if there are records.

 

All the insurance patient records who are discharged/checked out are available for all Supervisors.

Assigning Coder for Coding function

From the CA Transactions > Clinician Access > Summary Menu > Coders Review > select the Assign Coder for Coding function.

The Assign Coder for Coding Function screen gets displayed.

  1. By default, the patient details for the last 7 days get displayed. The sorting is done based on Discharge Date/Time (Descending Order).

  2. Enter the search criteria if required.

    1. Patient ID – Enter the required patient ID or select it from the lookup.

    2. Period – By default displays the date of last 7 days. You can change the date range if required

    3. Click [Search], the patient records matching the criteria appear.

  3. Click the Patient ID hyperlink to invoke the Record Clinical Notes window. (To open the Record Clinical Notes function by default, you need to define it as the first function in ‘CA Menu” assigned to the coder).

  4. To assign the patient record to a coder, select Coder name from the lookup and the Select check box against the record gets selected automatically.

  5. To assign multiple patients to coder, enter the From and To Serial No of the patient records. Select the Coder from the lookup for those patients and click [Assign]. The Selected Coder name gets displayed in the Coder lookup in the patient record.

• You can enter only numeric values in the S.No From and To.

• When To value is greater than From value and on clicking [Assign], then the message ‘To S.no should be Greater than or equal to from S.no’ gets displayed.

  1. To complete the record, select the check box and click [Record Completion] button. The record gets removed from list in summary.

  2. Click [Export To Excel]to download the records in excel format.
