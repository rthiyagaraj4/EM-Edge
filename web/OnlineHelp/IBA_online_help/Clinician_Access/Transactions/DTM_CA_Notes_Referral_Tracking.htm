Notes Referral Tracking

From this function you can Track, Review and Print the Notes Referral details for patients.

This function is used to track down the clinical note referral details and print the clinical note referral details if required. It allows the practitioner to review the pending clinical note if he searches with his own logged in User ID.

Tracking CA Notes Referral

From the CA Transactions > Clinician Access > Desktop Menu > Schedules, select Notes Referral Tracking  

The Notes Referral Tracking screen gets displayed.

  1. Enter the details in mandatory (*) fields. <Related Field Description>

  2. Click [Search] to view clinical note referral details or [Reset] to clear the entered search criteria.

  3. In the search results, two columns ‘Referred From’ and ‘Referred To’ appear if both the Referred From and Referred To check boxes are selected in search criteria. If only ‘Referred From’ check box is selected, only ‘Referred From’ column will appear and vice versa.

  4. Click  icon to view expand or collapse the search results.

  5. Click Date/Time hyperlink and the note referral details such as Patient Details, Note Type, Referred Location Details and Status appear.

  6. Click Note Type hyperlink of the patient to view the Note Content in a separate dialog.

  7. Click Status hyperlink to view the audit trails details of the status in a separate dialog.

  8. Click [Hide Header] to view the clinical note search results alone or click [Show Header] to view the search results along with the search criteria.

  9. If you have logged in as a practitioner and if there any referral requests waiting your review, then select the check box against the required clinical note referral request and click [Review] to confirm the clinical note referral request / authorization request for selected cases. After signing the respective Clinical Note referral requests, the system changes the status as confirmed. You can confirm multiple clinical note requests at a time.

  10. Click [Print] if you want to print the Clinical notes referral details.
