Record Birth

Select Record Birth List from the Record Birth task to record birth details of the event Delivery and the Delivery Stage 2 and 3 if the Delivery type is recorded as 'Normal' delivery.

 

Record Birth Details

Delivery Stage 2

Enter the following details:

  1. 2nd Stage Date and Time – Enter the 2nd Stage Date and Time using Calendar function. This is a mandatory field.

  2. Duration – Enter the 2nd Stage Duration of labour in minutes.

  3. Delivery Presentation – Select the Delivery Presentation form the list. The listed values are OA, OP, Breech, Hand, Face, Foot, Shoulder and SA.

  4. Membrane Rupture Date and Time – Enter the Membrane Rupture Date and Time or select using Calendar function. Select the Membrane rupture from the listed values ARM and SRM.

  5. Delivery Position – Select the position of delivery from the list. The listed values are OL, OR, OP, SL and SR.

  6. Foetus Heart Rate – Select the Foetus Heart Rate from the list. The listed values are Late Deceleration, Normal, Prolonged bradycardia, Variable deceleration and Tachycardia.

  7. Fetal Complications – Select the Fetal Complications from the list. The listed values are Birth asphyxia, Nerve injury, Fracture Clavicle, Fracture Femur, Fetal death and Cephalhematoma.

  8. No of round of cord around neck - Enter the number of rounds the cord is around the neck of the fetus.

  9. 2nd Stage Maternal Complications (s) – Select 2nd Stage Maternal Complication from the listed values. You can select three values for 2nd stage maternal complication. The listed values are Abnormal presentation, Abruptio placenta, Cord prolapse/presentation, Vasa previa, Cephalopelvic disproportion, Eclampsia, Maternal death, Maternal exhaustion, Multiple pregnancy, Pelvic injury, Prolonged labour, Retained second twin, Shoulder dystocia, Uterine rupture, Amniotic fluid embolism, Pulmonary Edema and Congestive Heart Failure.

Personal Details

  1. Attending Doctor – Enter the name of the Attending Doctor or select using [?] function. The practitioners applicable to physicians and surgeons are displayed. If attending doctor is recorded in ‘Delivery Record’ it is displayed, by default.  

  2. Nurse – Enter the name of the attending Nurse or select using [?] function. If attending Nurse is recorded in ‘Delivery Record’ it is displayed, by default.

Click [Save] to save the record and the record can be modified. Click [Finalize] to save the record permanently and the record cannot be modified. On finalizing the record the date and time of recording will be displayed.

Only the Finalized records are posted to Clinical Event History.

Delivery Stage 3

Enter the following details:

  1. 3rd Stage Duration - Enter the 3rd Stage Duration in minutes format.

  2. Method of Delivery – Select the Method of Delivery from the list. The listed values are Modified Crede Manuever, Controlled cord traction and Manual removal.

  3. Completeness of Placental Delivery – Select the Check box if placental delivery is complete.

  4. Placenta weight – Enter the placenta weight in grams.  

  5. 3rd stage Maternal Complication(s) – Select the 3rd Stage Maternal complication from the list. The listed values are Placenta accrete, Placenta increta, Placenta previa, Placenta percreta, Post partum Haemorrhage, Retained Placenta, Uterine inversion, Uterine atony, Vaginal hematoma and Vulva hematoma. You can select three values for 3rd Stage maternal complication.

  6. Umbilical Cord – Select the umbilical cord from the list. The listed values are Normal, True knot and Hematoma.

  7. Cord Blood Collection - Select the check box indicate blood sample of cord is collected.

Obstetric Trauma and Perineal Tear

Perineal Tear

Select from the following options to indicate Perineal Tear. The options are 1st Degree, 2nd Degree, 3rd Degree, 4th degree and Cervical Tear. Enter any other Perineal Tear in the Specify text area, if ‘Others’ check box is selected.

Perineorhaphy

  1. No - Select the required option button to indicate if Perineorrhapy is performed or clear the check box to if not performed.

  2. Continuous Subcuticular – Select the check box if Continuous Subcuticular is performed or clear the check box if not performed.

  3. Interrupted Suture - Select the check box if Continuous Subcuticular is performed or clear the check box if not performed.

  4. Inside – Select the ‘Inside’ check box if Interrupted Suture was performed inside and mention the number of stitches in the numeric format of two digits in the text area provided.

  5. Outside – Select the ‘Outside check box if Interrupted Suture was performed outside and mention the number of stitches in the numeric format of two digits in the text area provided.

  6. Stitches with – Select the check box if Interrupted Suture stitches were performed With Silk or Select the check box if Interrupted Suture stitches were performed With Catugut.

  7. Continuous interrupted - Select the check box if Continuous Subcuticular is performed ‘Outside’ and mention the number of stitches in the numeric format of two digits in the text area provided.

  8. Interrupted Subcuticular - Select the check box if Continuous Subcuticular is performed ‘Inside’ and mention the number of stitches in the numeric format of two digits in the text area provided.

  9. By – Enter the name of the practitioner who has performed the stitches or select using [?] function.

Click [Save] to save the record and the record can be modified.

Click [Finalize] to save the record permanently and the record cannot be modified. On finalizing the record the date and time of recording will be displayed.

Only the Finalized records are posted to repository. If Stage 2 is recorded and finalized, Stage 1 cannot be modified even if not recorded.
