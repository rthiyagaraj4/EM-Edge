Enter the following details:

 

  1. Code - Enter the code in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Short Description - Enter the Short Description of the Intervention in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  3. Long Description - Enter the Long Description of the Intervention in alphanumeric format of 400 characters maximum length. This is a mandatory field.

  4. Enabled - Select the check box to enable the Intervention record or clear the check box to disable the record.

 

Attributes

  1. Intervention Type - Select the Intervention Type from the list. This is a mandatory field.

  2. Dependency - Select the Dependency level for the selected intervention type.

  3. Duration to perform Intervention - Enter the Duration to perform the Intervention in numeric format of two digits maximum length. Select the duration as day(s), hour(s), minute(s), month(s) or week(s).  This is a mandatory field.

  4. Duration before start time when task becomes due - Enter the Duration before start time when task becomes due in numeric format of two digits maximum length. Select the duration as day(s), hour(s); minute(s), month(s) or week(s). This is a mandatory field.

  5. Duration after start time when task becomes overdue - Enter the Duration after start time when task becomes overdue in numeric format of two digits maximum length. Select the duration as day(s), hour(s); minute(s), month(s) or week(s). This is a mandatory field.

  6. Duration after end time for marking tasks as expired - Enter the Duration after end time for marking tasks as expired in numeric format of two digits maximum length. Select the duration mentioned is for day(s), hour(s); minute(s), month(s) or week(s). This is a mandatory field.

  7. Backdate limit for resulting of tasks - Enter the Backdate limit for resulting of tasks in numeric format of two digits maximum length. Select the duration as day(s), hour(s); minute(s), month(s) or week(s). This is a mandatory field.

  8. Use as Ad Hoc Intervention - Select the check box for the intervention to be available on clicking [Adhoc] on the task list for a patient in the transactions.

  9. Result Type - Select the Result Type from the list. The listed values are Chart, Discrete Measure Atomic, Discrete Measure Panel and Orderable.

  10. Result Measure - Select the Result Measure based on the 'Result Type' selected. This field becomes mandatory if 'Result Type' is selected.

  11. Guidelines to perform Intervention - Enter the Guidelines to perform Interventions in alphanumeric format of 4000 characters maximum length.

  12. Patient Instructions – Select the patient instruction using the function [?] to associate with the Intervention.
