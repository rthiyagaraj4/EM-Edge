Enter the following details:

 

  1. Order Srl No – The serial number of selecting the Vaccine is displayed and hyperlinked.

  2. Vaccine Description – The description of the selected Vaccine is displayed.

  3. Number of Doses - Enter the Number of Doses of the Vaccine which has to be administered. If the Number of Doses specified is more than one, you can schedule the doses using ‘Schedule’ hyperlink. Click ‘Schedule’ hyperlink to invoke the Schedule for Vaccination screen.

    1. The Vaccine Category details are displayed in the Schedule for Vaccination frame.

    2. The number of doses is displayed.

    3. Enter the Duration in numeric format of 2 digits maximum length. This is a mandatory field.

    4. Select the Duration Type from the list. The listed values are Day(s), Hour(s), Month(s), Minute(s), Week(s), and Year(s). This is a mandatory field.

    5. Select the ‘Mandatory’ check box if administration of the defined Vaccine schedule is mandatory.

    6. Select the Default check box to make the defined schedule as default.

    7. Click [OK] record the Vaccination schedule or [Cancel] to exit from the Schedule for Vaccination screen.

  4. Default Site - Select the Default Site from the list. The Anatomical Sites associated to the selected Vaccine in Site for Vaccine function are displayed.

  5. Mandatory - Select the check box if administration of the Vaccine schedule defined is mandatory.

  6. Check Previous Administered - Select the check box to check if the scheduled vaccine is previously administered.

  7. Check Previous Days - Enter the number of days before which the particular Vaccine was administered.

  8. Check Previous Action – Select the value from the list. The listed values are ‘Alert’ and ‘Stop’. This indicates if the previous administration was alerted or stopped.

  9. Frequency – Enter the Duration to schedule the vaccine and select the Duration Type. The Duration Type values are populated from ‘Duration Type’ function of AM Frequency Setup.
