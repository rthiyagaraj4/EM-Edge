Enter the following details:

 

  1. No Known Allergy - Select the check box to indicate that the patient has no known allergies. This check box is selected, by default. Clear the check box to enter details in the following fields.

  2. Hypersensitivity – Select any one of the radio buttons from the options: Allergy, Type II, Type III and Type IV.

  3. Event Type – Select the Event Type from the list. The values in the list are populated from the Adverse Events master function in Terminology & Coding Management module. The Event Types are displayed based on the Hypersensitivity selected. This field is mandatory. On selecting the Event Type the screen as shown is displayed.

  4. Allergen – Select the Allergen that the patient is allergic from [?] function. The allergens for selection are displayed from the ‘Causative Agent for Adverse Event’ master function. This field is mandatory.

For example, while placing the order for Paracap 500mg Tablet for the selected patient, if the value ‘Drug’ is selected in Allergy Alert by and Allergen as Generic Name ‘Paracetamol’ and the Causative Substance is set as ‘Paracap 500mg Tablet’, the system will display the Allergy alert.  

  1. Causative Substance – Enter the details of the substance causing the allergy. This field is mandatory if ‘Drug’ is selected in the Allergy Alert By drop-down list.

  2. Onset Date - This field denotes the date from which the Patient is having the problem. It is defaulted to the current system date, which can be changed to an earlier date. To change the date, choose the date from the Calendar function. This date cannot be greater than the current system date/time.

  3. Estimated Duration - Select the check box to indicate that the onset date recorded for the allergy is an estimated duration. An icon is displayed against the Onset Date field on selecting the check box.

  4. From Last - This field provides an alternate method of entering the onset date. The three fields represent the days, months and years respectively. Enter the value in numeric format of two digits maximum length in all the fields or any one of them as the case may be to indicate that the patient has been experiencing the problem from the last number of days, months or years.

  5. Allergy Alert By – Based on the value selected, The available values are ‘None’, ‘ATC’, ‘Generic’ and ‘Drug’.

    • If you select ‘None’, the current functionality will continue.

    • If you select ‘ATC’, the allergy alert will be based on the setup done at the ATC Classification level.

    • If you select ‘Generic’, the allergy alert will be based on the selected Generic Name.

    • If you select ‘Drug’, the allergy alert will be based on the selected drug and the existing Causative Substance field will become mandatory.

  6. Onset – By default this field is disabled and shows ‘Not Known’ option when the ‘No Known Allergy’ check box is selected. It gets enabled and you can select either Immediate or Delayed option for the allergy when the ‘No Known Allergy’ check box is cleared.

  7. Comments - By default, the text box is disabled when the ‘No Known Allergy’ check box is selected.  It gets enabled when the ‘No Known Allergy’ check box is cleared. Enter your comments in alphanumeric format of 300 characters maximum length.

  8. Add Reaction – This field is enabled on selecting the Allergen. Select the Adverse Reaction from [?] function. The values displayed for selection are associated to the Reaction in the Adverse Reaction master function of Medical Records module. This is a mandatory field. The screen displays the following on selecting [?] function.

  9. Reaction Code – The code of the reaction is displayed.

  10. Reaction Description – The description of the reaction is displayed.

  11. Select - Select the required Reaction by selecting the corresponding check box.

  12. Other Reactions – Enter any other reactions in the text area provided.

Click [Close] and the screen as shown is displayed.

The selected reaction and the Other Reaction as entered are displayed on the main screen.

  1. Reaction Description – The reaction field name selected from the Reaction Search screen is displayed. If you enter a value in the ‘Other Reaction’ screen from the Reaction Search screen, the reaction description entered is displayed along with the legend ‘Other Reaction’.

  2. Reaction Date – The current system date and time is displayed. Enter a date in dd/mm/yyyy format or select the date from Calendar function. If a ‘Reaction’ is recorded for a patient against an allergen for a particular date and time, the same reaction cannot be recorded again for the same patient and for the same date and time. Reaction date entered must be lesser than current date but cannot be greater than the current date. This field is mandatory.

  3. Severity – Select the ‘Severity’ from the list. The value selected in this field must be defaulted in the Severity Field while recording reaction. The values listed are Mild, Moderate, Severe and Unknown.

  4. Site of Reaction – Enter the anatomical site where the reaction has occurred in the text area provided.

 

Click [Event Type Details] to record additional details.
