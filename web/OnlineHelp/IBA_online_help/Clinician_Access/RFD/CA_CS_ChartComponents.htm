Enter the following details to create Chart Components:

 

  1. Chart – Enter the required Chart in alphanumeric format or select the Chart using [?] function.

Click [Show Detail]. The Sections associated to the Chart are displayed.

  1. Order Sequence No – The sequence number in which the discrete measures are associated to the section are displayed.

  2. Discrete Measure Code - The Code of the discrete measure is displayed.

  3. Discrete Measure - The description of the Discrete Measure is displayed.

  4. Formula – For Discrete Measures, which belong to the type ‘Formula’, the system displays a hyperlink ‘Def’. On clicking the hyperlink, the formula can be defined.

    The Formula Definition Chart Components screen displays the following.

    1. Panel ID – The system displays the numerical components associated to the section with a hyperlink. On clicking the hyperlink of any component, which has to be included in the formula, the component is shown to the user in the calculator frame.

    2. Panel Description – The system displays the Description of the components as defined in the Application Masters – Discrete Measure Setup function.

    Calculator Frame

    1. Constants – The system allows you to insert the constants for the calculation. The numbers are given as buttons. On clicking a Number, the system inserts the number into the number display field. Choose a maximum of 40 characters in this field that can be used for calculation. The number entered in this field is inserted into the formula based on the need.

    2. Insert – Click [Insert] to insert the number entered in the number field into the formula frame.

    3. Operators – The system displays the basic operators such as +, -, *, /, (, and ) as buttons. Choose the operator as needed in framing the formula. On clicking the button, the system inserts the operator into the formula frame.

    4. Placing Option – The system displays the basic placing options like the ‘Replace’, ‘Insert Left’ and ‘Insert Right’. Use this in placing the constants or operators in the required position of a formula.

    5. Replace – Select the Constant/Operator to be replaced, click the Replace radio button and select a new operator/constant. The selected constant/operator gets replaced.

    6. Insert Left – Select the Constant/Operator from the formula bar to the left of which a Constant/Operator needs to be added. Click [Insert Left] and select the Constant/Operator needed to be inserted, which is added to the left of the selected Constant/Operator.

    7. Insert Right – Select the Constant/Operator from the formula bar to the right of which a Constant/Operator needs to be added. Click [Insert Right] and select the Constant/Operator needed to be inserted, which is added to the right of the selected Constant/Operator.

    8. Remove – Select the Constant/Operator from the formula bar and click [Remove] to remove the Constant/Operator.

    9. Reset – Click [Reset] to remove the existing formula and reframe it. A message gets displayed on clicking the button, if the entire formula to be deleted. Click [OK] and the existing formula is removed and the Placing option is disabled until an Operator/Constant is selected.

    10. Validate – Once the formula framed on the formula bar is complete, click [Validate]. If the expression is invalid, on clicking [Validate], a message gets displayed indicating the Expression is invalid. A dialog box is displayed if the Expression is valid. Click [OK] to save the formula defined and get back to the chart component screen.

  5. Summary Reqd - The check box is enabled only if the Summary Required check box in the Chart Setup for the selected chart component is selected. The period of validity for the component depends on the summary period created in the setup.

  6. Summary Column - On enabling the Summary Required for the selected Chart Component, the Summary Column is enabled. Click the Summary Column [?] function to search the discrete measures for the selected summary component. Click Discrete Measure Description hyperlink to display it in the Summary Column field.

  7. Summary Type – Select the Summary Type for the selected summary component. The values are ‘Sum’, ‘Avg’, ‘Max’ and ‘Min’.

  8. Select – Clear the selected component if not required.

  9. Mandatory – Select the check box if entering data in the field for the selected component is mandatory.

  10. Element Type – Select the Element Type from the list. This field is enabled for Special Charts and not for Generic Charts. If chart type is "Growth Chart" this field gets enabled with system defined values. The list displays Length, Weight, Head Circumference and Chest Circumference. It is used to create graph for pediatrics department using Special Graph Element master in CA code setup.

  11. Default – If for the selected chart component the Configuration in the chart setup is selected as Fixed, the check box is enabled, or it is disabled. This field is also selected and disabled if Mandatory check box is selected.
