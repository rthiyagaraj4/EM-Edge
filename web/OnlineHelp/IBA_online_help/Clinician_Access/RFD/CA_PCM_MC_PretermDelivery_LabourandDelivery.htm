Labour and Delivery

  1. Select Delivery Record from the Labour and Delivery task to record labour and delivery details of the event Preterm Delivery. Click [Record] to save the Delivery Record. The Delivery Type field is disabled on selecting the Delivery Record hyperlink from the left side of the screen if it is not yet finalized and other details can be modified. If the record is Finalised, you cannot modify any of the recorded details.

    Delivery Record

    1. Delivery Type - Select the Delivery Type from the list.  The Delivery Types are defined in the Master Patient Index module with Delivery indicators - Normal, C Section and Others.

    2. LMP - Select the LMP using the calendar function. This is validated based gestation period recorded on the MP Parameter setup. This is a mandatory field.

    3. Attending Practitioner - Enter name of the Attending Practitioner or select using [?] function.

    4. Attending Nurse - Enter name of the Attending Nurse or select using [?] function.

    5. Procedure Classification - Enter the Procedure Classification or select using [?] function. The Procedure Classifications are defined in the Patient Identity Management module.

    6. Delivery Surgical Classification - Enter the Delivery Surgical Classification or select using [?] function. The Delivery Surgical Classifications are defined in the Master Patient Index module.

    7. Vaginal Delivery Type - Enter the Vaginal Delivery Type or select using [?] function. This field is disabled if 'Delivery Type' is selected as a record with Delivery Indicator - 'Normal'.

    8. No. of Birth - Enter the number of births.

    9. Complication in Stage 1, 2 and 3 of Delivery - Select the Complication of Stage 1, 2 and 3 of Delivery from the list. The Complications of Delivery are created in the Complications of Delivery function with 'Type of Indicator' field selected as Mother for each of the complication stages in the Patient Identity Management module. If 'Delivery Type' is selected as a record with Delivery Indicator - 'C Section', these fields are disabled.  

    10. Complication of Delivery - Enter the Complication of Delivery or select using [?] function. The Complications of Delivery are created in the Complications of Delivery function in the Master Patient Index module. This field is disabled if Delivery Type is selected as a record with Delivery Indicator - 'Normal'.

    11. Foetus Complication of Delivery - Enter the Foetus Complication of Delivery or select using [?] function. The Foetus Complications of Delivery records are created in the Complications of Delivery function with 'Type of Indicator' field selected as Fetus in the Master Patient Index module.

    12. Finalize - Select the check box to Finalize the delivery record. Selecting the check box makes the No. of Birth field mandatory.

  2. If the Delivery Type is selected as a record with Delivery Indicator as 'C Section', you get the option to Record Procedure.

  3. If the Delivery Type is selected as a record with Delivery Indicator as 'Normal', you get the following options to record the delivery stages.

Delivery Stage 1

Delivery Stage 1 Summary

Delivery

  1. Delivery - Select delivery as Full Term, Pre Term or Others. Specify details if Others is selected.

  2. Delivery Suite Admission Time - Select the date and time of admission into the delivery suite using the calendar function.

  3. Date and Time of Labour Pain - Select the date and time of labour pain using the calendar function.

  4. Labour Established Time - Select the date and time of when labour was established.

  5. Enema Given - Enter the date and time of when enema was given to the patient. An enema is the procedure of introducing liquids into the rectum and colon via the anus.

  6. Date and Time of Bloody Flux - Select the date and time of bloody flux using the calendar function. Dysentery (formerly known as flux or the bloody flux) is frequent, small-volume, severe diarrhea that shows blood in the feces along with intestinal cramping and tenesmus (painful straining to pass stool).

  7. Character of Bloody Flux - Enter the character of bloody flux. Example: Mucous, Bloody.

  8. Membrane Rupture Date and Time - Select the date and time of membrane rupture using the calendar function.

  9. Type of Membrane Rupture - Select the type of membrane rupture from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup. Example ARM, SRM.

  10. Fluid Quantity - Select the Fluid quantity from the list box. Example: Normal, Reduced or Excessive. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

Liquor

  1. Select the radio button against the required options to indicate the type of Liquor. The options are Clear, MSL Grade 1, MSL Grade 2, MSL Grade 3, Blood Stained, Offensive and Others. Specify details if Others is selected.

  2. Stage 1 Duration - Enter the first stage duration of labour in minutes.

  3. FHR at Admission - Enter the Fetal Heart Rate per minute at admission.

  4. Labour Analgesia - Select Labour Analgesia (pain relief given during labour) from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  5. Labour Opiate Doses - Select Labour Opiate Doses from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  6. Epidural Doses - Select Epidural Doses from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup. Epidural steroid injections or doses are given into spine for immediate pain relief.

  7. Labour Anaesthesia - Select the Labour Anaesthsia from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  8. Labour Type - Select the Type of Labour from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup. Example: Induced, Spontaneous.

  9. Induction Started - Select the date and time of induction of labour from the calendar function.

  10. Induction Indicators - Select Induction Indicators from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup. Example: Prolonged 1st stage labour, Premature rupture of membrane.

  11. Induction Details - Enter the details of induction. This is a mandatory field.

  12. 1st Stage Mother Complication - Select the 1st stage mother complication from the list box. The values are defined in the Complication of Delivery function in MPI module.

  13. 1st Stage Fetus Complication - Select the 1st stage fetus complication from the list box. The values are defined in the Complication of Delivery function in MPI module.

  14. Procedure Done - Select the check box if procedure is done.

  15. Procedure Details - Enter the Procedure details. This is a mandatory field.

Personnel Details

  1. Attending Doctor - Select an Attending Doctor using the [?] function. All the practitioners defined in the Application Masters module are displayed for selection. This field is displayed if recorded while entering the Delivery Record.

  2. Assistant Doctor - Select an Assistant Doctor using the [?] function. All the practitioners defined in the Application Masters module are displayed for selection.

  3. Nurse - Select a Nurse using the [?] function. All nurses are defined in the Practitioner function of Application Masters module are displayed for selection. This field is displayed if recorded while entering the Delivery Record.

Delivery Stage 2

Click Delivery Stage 2 tab to record 2nd stage delivery details for the event Preterm Delivery.

Delivery Stage 2 Summary

  1. Duration of Second Stage - Enter the second stage duration of labour in minutes. This is a mandatory field.

  2. Full Dilation of Cervix – Enter the date and time of full dilation of cervix. This is a mandatory field.

  3. Bearing Down Effort Date/Time – Enter the date and time of bearing down effort during labour.

  4. Membrane Rupture Date/Time – Enter the date and time of membrane rupture.

  5. Type of Membrane Rupture – Select the type of membrane rupture from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  6. Liquor Quantity – Select the Liquor Quantity from the list box. The listed values are Normal, Increased and Decreased.

Liquor

  1. Select the radio button against the required options to indicate the type of Liquor. The options are Clear, MSL Grade 1, MSL Grade 2, MSL Grade 3, Blood Stained, Offensive and Others. Specify details if Others is selected.

  2. Time of Delivery – Enter the date and time of delivery using the calendar function. This is a mandatory field.

Delivery Presentation

  1. Select the check boxes against the required delivery presentation options. Enter any other delivery presentation in the Specify text area if Others check box is selected.

  2. Delivery Position - Select the position of delivery from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  3. Delivery Method - Select the method of delivery from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  4. Labour Analgesia - Select Labour Analgesia (pain relief given during labour) from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  5. Abnormal Delivery Reason - Select the reason for abnormal delivery from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  6. Mother Complication – Select Complication of mother from the list. The values are defined in the Complication of Delivery function in MPI module.

  7. Fetal HR/Min - Enter the fetal heart rate per minute. This is a mandatory field.

  8. Fetus Complication - Select the complication of fetus from the list box. The values are defined in the Complication of Delivery function in MPI module.

  9. Cord Problems - Select the check box if cord problem exists.

  10. No of round of cord around neck - Enter the number of rounds the cord is around the neck of the fetus.

  11. Treatment/Medication – Enter the treatment or medication details. This is a mandatory field.

  12. Remarks – Enter the remarks. This is a mandatory field.

Personnel Details

  1. Attending Doctor - Select an Attending Doctor using the [?] function. All the practitioners defined in the Application Masters module are displayed for selection. This field is displayed if recorded.

  2. Attending Pediatrician - Select an Attending Pediatrician from the [?] function. All the practitioners defined in the Application Masters module is displayed for selection.

  3. Assisting Doctor - Select an Assisting Doctor using the [?] function. All practitioners defined in the Practitioner function of Application Masters module are displayed for selection.

  4. Nurse - Select a Nurse from the [?] function. All nurses are defined in the Practitioner function of Application Masters module. This field is displayed if recorded.

Delivery Stage 3

  1. 3rd Stage Duration - Enter the third stage duration of labour in minutes.

  2. Method of Delivery of Placenta - Select the method of delivery of placenta from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  3. Delivery of Placenta and Membrane Date and Time - Select the date and time of delivery of placenta and membrane from the calendar function.

Placenta in toto

  1. Select the required radio button – Yes, No or Others. Specify details if Others is selected.

  2. Placenta – Select the characteristics of placenta from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  3. Membranes - Select the Membrane from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  4. Placenta's Weight - Enter the weight of the placenta in grams.

  5. Umbalical Cord - Select the type of Umbalical Cord from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  6. Retroplacental Clot - Select the Retroplacental Clot details from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  7. Complication of Mother in 3rd Stage of Delivery - Select the mother complication in the 3rd stage of delivery from the list box. The values are defined in the Complication of Delivery function in MPI module.

  8. Cord Sample Collection - Select the check box to indicate blood sample of cord is collected.

  9. PPH Prophylaxis – Enter PPH Prophylaxis details. This is a mandatory field.

  10. Estimated Blood Loss - Enter the estimated blood loss in ml.

  11. Blood Transfused - Select the check box if blood has been transfused.

  12. No. of Units – Enter the number of units of blood transfused. This field is enabled if Blood Transfused check box is selected.

  13. Blood Group – Select the Blood Group from the list box. The listed values are defined in Medical Records function

  14. Transfusion Details - Enter the transfusion details.

  15. Medication Details – Enter medication details of the patient. This is a mandatory field.

  1. Click [Record] to save details of labour and delivery recorded for the event Preterm Delivery. Click the required Delivery Stage hyperlink and the details recorded for Labour and Delivery for the event Preterm Delivery gets displayed on the right hand side of the screen. Click Delivery Stage 1 hyperlink to view first stage delivery details recorded for the patient. Click Delivery Stage 2 hyperlink to view second stage delivery details recorded for the patient and the details are displayed. Click Delivery Stage 3 hyperlink to view third stage delivery details recorded for the patient and the details are displayed.

  2. Once the record is finalized, you can record the birth details. Record Birth option is displayed based on the No. of Births recorded. If two births are recorded, you can record the birth details for both the babies.

  1. Click Register Baby to register the baby. Enter details is mandatory field and click [Register] to register the baby. The record can be finalized by selecting the Newborn ID hyperlink. Enter mandatory details and click [Finalize] to finalize the record. The newborn ID is displayed on the left side of the screen. If the record is not finalized, text is displayed as (Not Finalized). Select the newborn ID to view details.

 

Assessment

  1. You can record the newborn document details only if the newborn is registered. Click Newborn Document from the menu to record clinical notes that reflect the clinical condition of the newborn for the event Preterm Delivery. Note Types created with Note Group of Applicable Task as 'Mothercare Consultation' is used for recording.

  2. You can record the newborn chart details only if the newborn is registered. Click Newborn Chart from the menu to record clinical charts for the newborn for the event Preterm Delivery.
