Enter the following details to create a Sections for Chart record:

 

  1. Chart – Enter the required Chart in alphanumeric format or select the Chart using [?] function. All charts defined in the Chart Setup function for which components are not applied in the function ‘Chart Components’ are displayed. This field is mandatory. On selecting the required Chart, click [Show Detail].

  2. Section Starts With – Enter the required section description in alphanumeric format of 20 characters and click [Search] to search for the required section or click [Search] without entering the Search Criteria to list all the Sections.

  3. Section Description - All the Sections created in the Application Masters module with the type as ‘Chart Section’ in the Discrete Measure Panel function is displayed.

  4. Select - Select the check box to select the Section to be associated to the selected Chart. The selected sections are displayed under the ‘Selected Sections’ on the top part of the screen. You can exclude the added Sections by selecting the ‘Select’ check box provided under selected Section header.
