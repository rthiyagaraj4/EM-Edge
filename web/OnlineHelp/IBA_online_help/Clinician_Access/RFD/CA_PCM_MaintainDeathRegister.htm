The following details are displayed:

 

  1. Patient Details - The displayed patient details are the Patient Name, Age, Gender and Patient ID, Practitioner name, Nursing Unit Name, Room Number, Bed Number and the Encounter ID.

 

Death Registration Details

  1. Admission Date/Time – The admission date and time of the patient is displayed.

  2. Admission Type – The Admission Type selected during admission is displayed.

  3. Death Date/Time – The date and time of death is displayed.

  4. Authorizing Practitioner – The practitioner who declared the patient to be dead is displayed. The name of the practitioner can be changed or selected using [?] function.

  5. Remarks – Enter the remarks in alphanumeric format of 200 characters maximum length.

 

Diagnosis

This feature enables recording a diagnosis which led to the death of the patient. A normal diagnosis or a sensitive diagnosis can be recorded for the patient.

  1. Diagnosis Type – Select the Diagnosis Type from the list. The listed values are Provisional, Differential and Final.

  2. Code – Enter the ICD code in alphanumeric format or select the code using [?] function. This field is mandatory.

  3. Description – Once the code is selected, the description is automatically displayed.

  4. Remarks – Enter the remarks in the text area provided in alphanumeric format of 200 characters maximum length.
