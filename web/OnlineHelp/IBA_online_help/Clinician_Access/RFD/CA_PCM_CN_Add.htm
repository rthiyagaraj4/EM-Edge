By default this tab is displayed on selecting the Clinical Notes function.

  1. Enter details in mandatory fields. <Related Field Description>

  2. On selecting the Note Type, Sections and Subsections associated to the Note is displayed on the left side of the screen under Notes View. Each of the Section or the Sub Section appears as a hyperlink to invoke the screen for entering the relevant notes.

  3. An icon (c) is displayed if Chief Complaint defined in CA Codes Setup is associated to any section of the selected Note Type in the function ‘Section for Note Type’.

    If a Chief Complaint defined in CA Codes Setup is associated to any Section of the selected Note Type through ‘Section for Note Type’ function of CA Codes Setup, an icon (c) is displayed.

    1. Click [Chief Complaints] to display the Chief Complaints screen.

    2. Scope - Select Scope as Current Encounter, Across Encounter or Open List.

    3. Search Text - On selecting Scope as Open List, entering the text for searching the chief complaint becomes mandatory.

    4. Search Criteria – Select the Search Criteria as Starts With, Ends With or Contains from the list box.

    5. Click [Search] to display Chief Complaints. Select the check box against the required Chief Complaint and click [Select] to copy the Chief Complaint to the editor screen. Click [Cancel] to exit from the Chief Complaints screen.

  4. An icon (i) is displayed for an Image Linked Section and this setup is done in Section function of CA Code Setup by selecting the Image Linked checkbox.

    An Image linked section is used to represent the details that need to be recorded with the help of an Image. The section of the note type is marked as Image linked by selecting the check box ‘Image linked ‘in CA Code Set up using the ‘Section’ function. The Image is linked to the section using ‘Image for Section’ in CA code setup. If a section of the note type is Image linked, an icon (i) is displayed.

    1. Click the Section or Subsection with (i) icon from the ‘Notes View’ part of the ‘Record Clinical Notes’ screen.

    2. The editor is displayed to enter details of the Note.

    3. Click [Show Image] to select an image. If an Image is set default to a particular Section in ‘Image for Section’ function, the Image is displayed on selecting that Section and clicking [Show Image]. The tool palate in the image editor is used for marking or recording parts or details of the Image. Click [Show Editor] to switch to the editor screen.

    4. Select an Image from the Available Images list.

    5. Click [Show Both] to display the Text Editor and the Image Editor screen and record details.

  5. The [Upload] button is available at the bottom of the screen if the Note Type has the check box ‘Document Linked Note’ selected in the Note Type function.

    1. Click [Upload] to upload files/documents for the selected Note Type.

    2. File Reference – Browse to the path where the files are stored for uploading.

    3. Remarks – Enter remarks if any.

    4. Click [View] to view the file to be uploaded. Click [Record] to upload the file for the Note Type.

  6. The [Req Transcription] is displayed for Note Types associated to Note Groups for which 'Transcription Applicable' check box is selected. Click [Req Transcription] to request for transcription.   

    The Transcribe Notes screen is displayed.  

    1. Transcriber - By default, the default Transcriber set in the 'Default Transcriber Setup' function for the Practitioner is displayed. Select the Transcriber using [?] function.

    2. Remarks - Enter the remarks.

    Click [Apply] to request transcription for the Notes. Click [Cancel] to exit from the Transcriber Notes screen without saving the request for transcription.

The Sections displayed can be of Types:

Free Format

Result Linked

Main Section

Template Based

  1. Click [Copy from Previous Note] if you want to copy the pre-recorded clinical note to the currently selected clinical note. This button is visible only if you select a clinical note in the Record Clinical Notes screen. You cannot copy past records to the same note with New Format

    The Recording  Previous Notes Search dialog box is displayed.  

    1. Period - By default, the clinical note records based on the defaulted period appear in the Recording Clinical Notes Search dialog. The default Period is 30 days for Outpatients. The default Period is 10 days for inpatients.

    2. Facility Name - By default, the logged in facility gets displayed in the Facility Name field. User can search the required facility from the drop-down list.  Note: ‘All Facilities’ value is not available in the list.

    3. Ref No - This field will be blank by default. You can search the clinical note using the reference number for the selected time period.

    4. Click [Search] to filter clinical notes based on the entered search criteria.

    5. From the clinical note search results, select the required clinical note detail to be copied to the current clinical note.

    For day care and in patients, the system displays the clinical note across encounters based on the ‘No of Encounters to Copy Previous Notes’ in Note Parameter screen. For example, if the ‘No of Encounters to Copy Previous Notes’ value is 1, the system displays the notes created against the current Encounter.  If the ‘No of Encounters to Copy Previous Notes’ value is 2, the system displays the last created Encounter + current Encounter Notes.

    1. Click [OK], the system copies all the details from the clinical note in the respective fields with an Alert message "Copying from selected note will clear all filled details in current note. Do you want to continue?"

    2. Click [OK], the system overrides any previous details captured against the current clinical note by user after copying  the previous note. You can change the values of the configured note components if required.

    An alert message appears if you try to copy past records to the same note with New Format.

  2. Click [Canned Text] a Canned Text dialog box opens for you to select the list of text blocks available.

    The Canned Text dialog box is displayed.  

    1. Select – Select the check box against the appropriate text block that needs to be inserted into the editor.

    2. Description - The content of the Text Block is displayed. This field is hyperlinked if the Text Block contains more contents.

    3. Click [Select] to copy the text to the editor screen. Click [Cancel] to exit from the screen.

  3. Click [Prev Notes] to preview the notes before recording or signing. Close the Preview Notes screen on previewing the contents. You can also view the images, if they are linked to the note.

  4. Click [Notes View] to record or sign the clinical notes.

    • Click [Record] to record the notes. You can record notes multiple times.

    • Click [Sign] to sign the notes. You can sign the notes only once.

  5. Click [Print Preview] to preview and print the Clinical Note.

Spell Check function has been introduced. This is enabled for the logged in facility according to the setup done in the Note Parameter screen.
