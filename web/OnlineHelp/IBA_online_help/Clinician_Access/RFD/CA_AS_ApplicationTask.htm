Enter the following:

 

  1. To search for a specific Task, enter the name of the task in the text box, select ‘All’, ‘Checked’ or ‘Unchecked’ from Applicable list and click [Search] to list all the Tasks matching the specified criteria.

  2. All Tasks are displayed, by default. Select the required Tasks by selecting the ‘Applicable’ Check box to make the Task applicable to the facility.

  3. Select a value from the list under header ‘Access Restriction’ corresponding to Amend Order, Cancel Order, Discontinue Order and Hold Order. The listed values are All, Self Orders and Self Specialty Orders. Select Self Orders to allow you to Amend, Cancel, Discontinue and Hold the order placed. Select Self Specialty Orders to allow users of the specialty to Amend, Cancel, Discontinue and Hold the order placed. All is displayed, by default.
