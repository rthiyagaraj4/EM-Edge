Enter the following details:

 

  1. Chart Component – The components are displayed based on the setup in the masters.

  2. Format – Select the value from the list. The listed values are ‘Select’, ‘Graph’ and ‘Table’.

  3. Display Order – Enter the order in which the components have to be displayed. The values can be interchanged. This is a mandatory field.

  4. X – Axis Interval – Enter the interval for X – Axis values. This is a mandatory field.

  5. Y – Axis Interval – Enter the interval for Y – Axis values. This is a mandatory field.

  6. Min Y – Axis value – Enter the minimum value on the Y – Axis. This value should be greater than zero and less than maximum value. This is a mandatory field.

  7. Max Y – Axis value – Enter the maximum value the Y – Axis. This value should be greater than Min Y value.  This is a mandatory field.
