Existing List

Existing List Tab enables you to search for the Consultation List based on the Status. Consultation List that is pending for charges and consultation charges completed can be viewed. This tab also allows cancelling charges for completed consultation charges.

 

  1. Consultation List - Select the Consultation List ID from the list. The list item lists based on the parameter 'Charge Consultation for Past Days' in the billing module.

  2. Status - Select the Status from the list. The List displays All, Completed and Pending. If All is selected, records which are to be charged and those for which the consultation charge is already charged are listed. Pending lists patients for whom the consultation has to be charged and Completed lists patients for whom the consultation is already charged. If the consultation is pending for a particular patient then a colour indicator is displayed against that patient's record indicating that the consultation is to be charged. Click on [Charge Consultation] to charge those patients records. [Cancel Charge] is displayed if the status selected is completed. Select the records and click on cancel charge button to cancel the charges which are posted.

  3. Click [Search] to display the generated consultation list of patients. Patient details, Admission date and Charge Applicable for Consultation are displayed. The total Consultation charges, the charges paid by the patient and whether approval is required for posting the charges are displayed.

  4. View/Edit - Click the hyperlink to view/edit the Charges.

  5. Select - The check box is selected, by default. Records that are charged are indicated with a colour indicator and the check box is disabled. Check box is enabled only for records that are pending for consultation charge.

  6. Click [Charge Consultation] to post the consultation charges.
