Post Natal Check List

Select Post Natal Check List from the Post Natal Check List task to record post natal details of the event Preterm Delivery.

 

Post Natal Check List

  1. Mother and Baby Received By - Select the person, who received the mother and baby from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  2. Type of Delivery - Select the type of delivery from the list box.

  3. Place Taken - Select the place where the delivery took place from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  4. Baby Count – Enter the number of babies. This is a mandatory field.

  5. Enter the Baby Tag Details, Sex of the baby and Outcome.

Mothers Post Natal Examination

  1. Vaginal Examination - Select the check box if vaginal examination has been performed for the mother.

  2. Perineal Inspection - Select the check box if perineal inspection has been performed for the mother.

  3. Lochia - Select the lochia from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  4. Uterus Fundal Height - Enter the fundal height of uterus in weeks.

  5. Breast Engorged - Select the check box if breast is engorged. Breast engorgement occurs in the mammary glands when too much breast milk is contained within them. It is caused by insufficient breastfeeding and/or blocked milk ducts. When engorged the breasts may swell, throb, and cause mild to extreme pain.

  6. Uterus Contractility - Select the status of uterus contractility from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

Advice to Mother On

  1. Select the check box against Breast Care, Breast Feeding, Baby Immunization, Family Planning and Post Natal Exercise if advice has been given to the mother.

  2. Click [Record] to save details of Post Natal Check List recorded for the event Preterm Delivery. The date and time of recording the Post Natal details is displayed on the left side menu of the screen under Post Natal Check List task.

  3. Click the Date and Time hyperlink and the details recorded for Post Natal Check List for the event Delivery gets displayed on the right hand side of the screen.
