Template Based

A Template Based section is used to represent the details that need to be recorded in a pre-formatted structure. The template is built by arranging components in a formatted way so that it forms a meaningful and easy to read format to record clinical data for a patient.

 

  1. Click the Section or Subsection defined with type ‘Template Based’ from the ‘Notes View’ part of the ‘Record Primary Assessment’ screen.

  2. The contents of the section ‘Template Based’ document are displayed as constructed using the function ‘Section Template’ under the Code Setup functions.

  3. Enter the required details in the fields provided.

 

Additional buttons available on the screen are

Upload

  1. Click [Upload] to upload files/documents for the selected Note Type.

  2. File Reference – Browse to the path where the files are stored for uploading.

  3. Remarks – Enter remarks if any.

  4. Click [View] to view the file to be uploaded. Click [Record] to upload the file for the Note Type.

Record

The Note created is partially saved by using [Record] after which the Note is still available for editing or deletion.

On entering the required details for the Note Type, click [Record] to save the clinical notes recorded. The recorded Primary Assessment Notes are displayed. Select the Section hyperlink under Notes View on the left hand side of the screen to edit the contents recorded for the Note Type.

Sign

Once a Note is finalized, it is confirmed by using [Sign] after which the Note is available only for modification, forwarding or marking error and cannot be deleted.

  1. Click on [Sign] to sign a Note.

  2. A message is displayed indicating confirmation of signing the Note.

  3. Click [OK] to confirm signing or click [Cancel] to cancel signing.
