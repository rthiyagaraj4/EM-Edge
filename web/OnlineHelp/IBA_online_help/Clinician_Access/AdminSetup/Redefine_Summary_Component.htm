Redefine Summary Component

From this function you can Redefine the system defined summary description to user specific.  

You can modify the system defined summary components to user defined summary components. These components redefined are applicable in the Transaction functions. From this function, you can bold, highlight and also set background colour for the summary components.

Redefining Summary Components

From the CA Admin Setup, select Redefine Summary Component function.

The Redefine Summary Component screen gets displayed.

  1. The System Defined Summary Description gets displayed by default.

  2. Enter the User Defined Summary Description in the text box.

  3. Select the required User Defined Summary Description and click [B] to bold the description.

  4. Click on the colour picker under FC column to set the font colour for the description.

  5. Click on the colour picker under BC column to set the background colour for the description.

  6. Click [D] to reset the default font colour as black and background colour as white.

  7. Click [Apply] to save the changes made or [Reset] to clear the entered data.
