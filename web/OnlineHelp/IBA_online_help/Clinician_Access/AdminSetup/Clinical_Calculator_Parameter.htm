Clinical Calculator Parameter

From this function you can Assign a particular formulae as default for the Clinical Calculations.

The Formulae is used while accessing the Clinical Calculator.

Defining Clinical Calculator Parameter

From the CA Admin Setup, select Clinical Calculator Parameter function.

The Clinical Calculator Parameter screen gets displayed.

  1. Select the required Clinical Calculation as Body Surface Area or Ideal Body Weight. This is a mandatory field.

  2. Select the required Formulae for the Clinical Calculation selected. The field displays formulas based on the selected Clinical Calculation. On selecting the Formulae, the calculation of the formulae is displayed at the bottom of the screen.

  3. Click [Apply] to default the formulae for the Clinical Calculation or [Reset] to clear the entered data.
