Note Template Query List

From this function you can Create Query list for template based note types.

Creating Note Template Query List

From the CA Code Setup >  CA Clinical Note Setup, select Note Template Query List.

The Note Template Query List screen gets displayed.

  1. Click [Create] to create to invoke the Create screen.

  2. Enter the Query ID in the alphanumeric format of 10 characters maximum length. This is a mandatory field.

  3. Enter the Query Description in the alphanumeric format 60 characters maximum length. This is a mandatory field.

  4. Enter the Note Type or select the Note Type using [?] function. The enabled Note Types defined in the Note Type function are displayed. This is a mandatory field.

  5. Select the check box to enable the record or clear the check box to disable the record.

  6. The Component ID, Component name and Include (All) is displayed, by default. The components with ‘Include for Analysis’ selected in Section Template function and  component types belonging to Integer Numeric, Decimal Numeric, List Item, Check box and Formula are displayed.

  7. Select 'Include' check box against the required component or select Include (All) to select all the components by default. Only 10 components can be selected to create a query list.

  8. Click [Apply] to record Note Template Query List.

  9. Click [Reset] to clear the entered data.
