Specialty Outcome

From this function you can Identify all the outcomes pertaining to the Mothercare specialty module.

Every specialty event should have an outcome recorded which triggers another event in the transactions. The set of Specialty Outcomes in this function are system defined and new outcomes cannot be created. The Specialty Outcomes of this module are Close, Ectopic Pregnancy, Delivery, Still Birth, Abortion, Preterm Delivery, Delivered, Pregnant, Fertility Issues and Birth Control. The description of any Specialty Outcome can be modified. The user is also allowed to associate a Specialty Outcome to a particular age group.

Identifying Specialty Outcomes

From the CA Code Setup, CA Specialty Code Setup, select Specialty Outcome function.

The Specialty Outcome screen gets displayed.

  1. Click [Query] to query for system defined Specialty Outcomes.

  2. Enter the query criteria and click [Execute].

  3. The Specialty Outcomes are displayed. The Module, Code, Description and Age Group are displayed.

  4. Click the Code Hyperlink to associate an Age Group to the Specialty Outcome.

  5. Click [Apply] to save the association of Age Group to the Specialty Outcome.
