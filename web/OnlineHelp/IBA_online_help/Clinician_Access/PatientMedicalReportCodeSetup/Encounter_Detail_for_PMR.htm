Encounter Detail for PMR

From this function, you can Define the encounter details for deriving the Patient Medical Report.

Defining Encounter Detail for PMR

From CA Code Setup > Patient Medical Report Code Setup, select Encounter Detail for PMR function.

The Encounter Detail for PMR screen gets displayed.

  1. Select Patient Medical Report.

    Select the Patient Medical Report using [?] function. The [?] function displays Note Type created in the Note Type function of CA Code Setup with Note Groups applicable to Patient Medical Report.

  2. Click [Search] to display additional fields.

  3. Enter the Encounter Details.

    1. Past Days – Enter the number of days, months or years from when the Patient Medical Report has to be derived.

    2. Current Encounter – Select the check box to derive the Patient Medical Report for the current encounter. This check box is selected, by default.

    3. Previous Encounters – Enter the number of previous encounters to derive the Patient Medical Report.

    4. Facility Scope – Select the Facility Scope from the list. The listed values are All Facilities and Logged in Facility to derive the Patient Medical Report from All Facilities or the Logged in Facility. If the Current Encounter check box is selected, this field is displayed as Logged in Facility, by default.

  4. Click [Apply] to save the record or [Reset] to clear the entered details.
