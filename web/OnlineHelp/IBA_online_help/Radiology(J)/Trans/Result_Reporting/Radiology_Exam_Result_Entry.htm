Radiology Exam Result Entry

From this function you can Review or Finalize the pending examination reports.

This function enables the radiologist to review/authorize examination reports that are marked pending for authorization. The reviewing radiologist can make necessary corrections in the entered Report Text, Diagnosis/Analysis Codes entered and finally authorize the Result Report.

Radiology Exam Result Entry

From the Transactions > Result Reporting > Radiologist Exam Result Entry function.

The Radiologist Exam Result Entry screen gets displayed.

  1. The pending examination results, which are authorized by a Reporter in the Performing Facility and assigned to the Radiologist, are listed. Enter the search criteria and click [Search] to search for a specific patient record.

    Enter the search criteria:

    1. Ordering Facility - By default, the logged in facility appears in the Ordering Facility field. User can change the facility name if required.

    2. Assigned Radiologist - The logged in user name appears in the Assigned Radiologist field. User cannot change this value. Hence, logged in user cannot view other radiologist Exams as well as unassigned.

    3. Registration Date – By default, the current week date range is displayed. This field is mandatory. Maximum date range is one month. User can change this date range if required. Date cannot be future date.

    4. Registration No – Enter the registration number of the examination.

    5. Priority – By default, the value ‘All’ is displayed. The drop-down list values are All, Stat, Urgent, and Routine.

    6. Order Type – Select the required order type from the lookup.

    7. Exam – Click [List of Values] to select the examination code. The examination reports are listed based on the Exam selected from the list.

    8. Exam Status – By default, the status ‘Pending’ is displayed. Select the status of the examination. The list values are – All, Exam Completed, Pending, On hold, Prelim, Finalized, and Modified.

    9. Performing Facility – Select the facility where the exam is being performed from the list of values. The name of the performing facility where the exam was last conducted is displayed by default.

    10. Performed By – Select the Performed By name from the lookup.

    11. Reporting Facility – Select the Reporting Facility from the lookup.

    12. Assigned Radiologist – The name of the Practitioner associated to the logged in User gets listed. Select Radiologist from the list to view the examination reports pending for authorization.

    13. Patient ID - Enter the patient ID or select it from the lookup.

  2. Based on the search criteria, list of examinations get displayed. Details like Patient ID, Patient Name, Exam, Status, Result Indicator, Reg No, Regn Date/Time, Completion Date/Time, Awaiting Time (in min), Laterality, Contrast Type, Source, Prelim Radiologist, Radiologist, Section, Wing, Ordering Facility Name, Episode Type, Doctor and Study Location are displayed.

  3. Select the ‘Select’ check box against the patient record in the search result to view the patient details or to result report.

  4. Click the patient ID and sub menu with following options appear:

• View Image

Click [View Image] to open the PACS url to view the image if PACS is with interfaced with HIS system.

• Request Clinical Comments

Click [Request Clinical Comments] to view the clinical comments entered by the radiologist while performing the examination.

• Technologist Comments

Click [Technologist Comments] to view the technologist comments recorded while preparing the initial report and the views for the examination.

• Clinical Event History

Click [Clinical Event History] to view the clinical event history details of the patient. On clicking the button, the patient chart summary gets displayed. Click the CEH icon in the tool bar to open the clinical event history screen.

• Patient History

Click [Patient History] to view the patient chart summary.

  1. To result the report, select the 'Select check box against the patient record and click the hyperlink under 'Result Indicator' column. The Result Indicator column displays the value as R – Result Entry, P – Prelim, M – Modify, H - On Hold based on the Exam Status of the patient record.

• If the Exam Status is completed, 'R' hyperlink appears under Result Indicator column.  

• If the Exam Status is Reported-Prelim, 'P' hyperlink appears under Result Indicator column.  

• If the Exam Status is put on Hold, 'H' hyperlink appears under Result Indicator column.

• If the Exam Status is Report-Final or Report-Modified, 'M' hyperlink appears under Result Indicator column.  

  1. On clicking the ‘Result Indicator’ hyperlink, the system invokes Clinical Note or Report Text window as per Master set-up for that Exam.

  2. Enter the text required in the Report Text dialog and click [Save]. If Clinical Note window opens then record the clinical note details.

  3. Select the ‘Type’ from the list. The system-defined values are ‘Hold’, ‘Prelim’ and ‘Finalize’.

  4. Select ‘On Hold’ if the examination report is not confirmed. Select ‘Prelim’ if the authorization report is in the preliminary stages.

  5. Select ‘Finalize’, to authorize and finalize the examination reports. You cannot change the 'Prelim' Status to ‘On Hold’, but you can finalize a report for which the status is either ‘On Hold’ or ‘Prelim’.

  6. Click [Submit] to submit the selected patient record. Once submitted, the listed patient records get refreshed.
