Interim Charge Report

From this function you can Generate a report to view the bills generated for the services utilized by the Inpatients in the facility.  

Interim Charge Report function is used to generate an interim report for the services utilized by the inpatients in the facility. You can view the billing details of the current inpatients in the facility. The financial details, admission number, date and time, bed and ward number, last bed charges and the cash amount collected and adjusted towards the services gets displayed.

Generating Interim Charge Reports

From the Reports, select Interim Charge Report Function.

The Interim Charge Report screen gets displayed.

  1. Click [?] to select the range of Nursing Units. The interim charges for the current inpatients in the facility get listed.

  2. Click [?] to select the range of Admission Numbers.  A report is generated for the admission numbers of the inpatients in the selected Nursing Unit.

  3. Select a Payable Type from the list. The values are system defined – Any, Payer and Patient. Select Payer to generate a report where the Payer has settled the amount. Select Patient to generate a report where the Patient has settled the amount. Select Any to generate a report where the settlement has been made by either the Payer or the Patient.  

  4. Click [?] to select a Payer. The values defined using the ‘Customer’ function in the Accounts Receivable or the Patient Billing Manager module gets listed. A Payer can be selected when the ‘Payable Type’ is selected as ‘Payer’.

  5. Enter the Payable Amount Exceeding in numeric format. The report is generated for the records where the payable amount for the services exceeds the specified amount.

  6. Click [As on dd/mm/yyyy hh:mm:ss] to generate the report for the selected date.

  7. Click [Submit Bed Charge Process] to compute the bed charge. The job is queued on clicking and a confirmation ‘Job Submitted For Bed Charge Process’ is displayed.
