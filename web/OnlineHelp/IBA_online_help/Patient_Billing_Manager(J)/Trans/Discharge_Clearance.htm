Discharge Clearance

From this function you can Generate discharge clearance slip for an inpatient and emergency patient. Also, you can Reprint or Cancel generated discharge clearance slips.

Generate Discharge Clearance function is used to generate discharge clearance slip. The finance team generates a discharge clearance slip for an inpatient after the discharge advice is generated by the Nurse. Discharge clearance slip is generated irrespective of bill settlement against the discharge bill raised. Users are restricted from placing new orders for the patient after the discharge clearance slip is raised. You can generate a discharge clearance slip only if the discharge advice is raised. You can access this function only if the check box 'Discharge Clearance in Use' is selected in 'BL Parameters' function under the 'Inpatient/Daycare' tab. Authorized users with valid access rights are permitted to generate discharge clearance slip. You can generate discharge clearance slips for patients' with or without discharge bill.

 

Reprint/Cancel Discharge Clearance function is used either to reprint or cancel the discharge clearances. The discharge clearances generated for inpatients using the function Record Discharge Clearance are listed. You can access this function only if the check box 'Discharge Clearance in Use' is selected in 'BL Parameters' function under the 'Inpatient/Daycare' tab. Authorized users with valid access rights are permitted to cancel discharge clearance slips. To reprint the slips, you have to define print routing. The Discharge Document is available for discharged patient's also, so that the document can be reprinted after a patient is discharged from the ward.

IP Discharge Clearance - Generate Discharge Clearance tab

By default, the 'IP Discharge Clearance' option is selected.

 

From the Transactions > select IP Discharge Clearance

The Generate Discharge Clearance screen gets displayed.

  1. By default, the patient records for the logged in Facility ID gets displayed. You can change the facility ID by clicking [?]. This is a mandatory field.

  2. Enter the search criteria <Related Field Description>. The system gets refreshed when the search criteria is entered and displays record matching based on selected criteria.

  3. Click [Search] to search for patient records matching the search criteria. On clicking [Search], the legend changes to [As On: Current Date Time].

  4. Patient records matching the search criteria gets listed. Inpatient records get listed with the Patient ID, Patient Name, Episode ID, Episode Type, Admission Date, Discharge Advice Date, Discharge Bill Document Type and Number and the Patient Outstanding Amount.  

  5. Select the check box against the patient record. An authorization screen pops-up requesting the login details. Enter the details and click [Accept] to generate discharge clearance slip or [Exit] to cancel.

    1. User ID - Enter the User ID in alphanumeric format. Authorized users are only permitted to login and generate discharge clearance slip. This is a mandatory field.

    2. If the user is not authorised, an error message 'The User is not authorised to generate discharge clearance without bill' is displayed. User is permitted to authorize discharge clearance with discharge bill and without discharge bill for inpatients' by selecting the check boxes 'Discharge Clearance With Discharge Bill' and 'Disch Clearance W/O Disch Bill/Outst Bill' respectively through the function 'User Privileges and Cash Counter Access'.

    3. Pin Number - Enter the Pin Number of the respective User. This is a mandatory field.

    4. Reason - Click [List of Values] to select a reason. The action reasons defined through the 'Reasons for Billing Actions' function with the check box 'Discharge Clearance W/O Bill' as selected are listed. This is a mandatory field.

    5. Remarks - Enter the Remarks in alphanumeric format. This is a mandatory field.

  6. After accepting, click [Generate Discharge Clearance] button at the bottom. The Discharge Clearance Slip number gets generated successfully.

 

IP Discharge Clearance - Reprint/Cancel Discharge Clearance tab

To reprint or cancel the generated Discharge Clearance, click Reprint/Cancel Discharge Clearance tab.

Users can access this function only if the check box 'Discharge Clearance in Use' is selected in 'BL Parameters' function under the 'Inpatient/Daycare' tab. Authorized users with valid access rights are permitted to cancel discharge clearance slips. To reprint the slips, you have to define print routing. The Discharge Document is available for discharged patient's also, so that the document can be reprinted after a patient is discharged from the ward.

 

  1. By default, the patient records for the logged in Facility ID gets displayed. You can change the facility ID by clicking [?]. This is a mandatory field.

  2. Enter the search criteria <Related Field Description>. The system gets refreshed when the search criteria is entered and displays record matching based on selected criteria.

  3. Click [Search] to search for patient records matching the search criteria. On clicking [Search], the legend changes to [As On: Current Date Time].

  4. Patient records matching the search criteria gets listed. Inpatient records get listed with the Patient ID, Episode ID, Episode Type, Admission Date, Discharge Clearance Date, Discharge Clearance Document Type and Number, Outstanding Amount and the name of the authorized user.  

  5. Select the check box against patient record. An authorization screen pops-up requesting the login details. Enter the details and click [Accept] to cancel /reprint discharge clearance slip or [Exit] to cancel.

    1. User ID - Enter the User ID in alphanumeric format. Authorized users are only permitted to login and cancel discharge clearance slip. This is a mandatory field.

    2. If the user is not authorised, an error message 'The User is not authorised to Cancel discharge clearance' is displayed. User is authorized to cancel discharge clearance for inpatients' by selecting the check box 'Discharge Clearance'' through the function 'User Privileges and Cash Counter Access'.

    3. Pin Number - Enter the Pin Number of the respective User. This is a mandatory field.

    4. Reason - Click [List of Values] to select a reason. The action reasons defined through the 'Reasons for Billing Actions' function with the check box 'Cancel Discharge Clearance' as selected are listed. This is a mandatory field.

    5. Remarks - Enter the Remarks in alphanumeric format. This is a mandatory field.

  6. After accepting, click [Cancel Discharge Clearance] button at bottom to cancel generated discharge clearance slip.  The discharge clearance gets cancelled successfully.

  7. After accepting, click [Reprint Discharge Clearance] button at bottom to reprint the generated discharge clearance.
