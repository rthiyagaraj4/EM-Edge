Package Refund

From this function you can Refund the amount for the services in a package, if the services are not utilized.

The Package Refund function is used to issue a refund to the patient when the services associated to a package are not utilized. Refund can be issued only if the services are ordered. You cannot issue a refund if the order is registered. The refund amount is calculated based on the refund amount or percentage recorded during Package Definition.

Issuing Refund for a Package

From the Transactions > Package, select Package Refund Function.

The Package Refund screen gets displayed.

  1. Click [?] function to select the patient ID. The name of the patient, gender, age and ID id is displayed. This is a mandatory field.

  2. Click [?] function to select a package and issue refund. The packages associated to the patient are listed. This is a mandatory field.

  3. Click Package Details hyperlink to view the details.

    The package code, description and sequence number, effective date during which the package is valid, Billing Class code and description, Payer Group and Payer details are listed. The package amount, total amount utilized by the patient from the package for the services that are utilized is displayed. Deposit amount, if any, that is accepted from the patient is also displayed.

    Click [Close] to exit from the screen.

  4. Click [Search] to view the individual services that are included in the package. <Related Field Description>

  5. Click [Apply] to save the changes or [Reset] to clear the details.

  6. A bill number is generated on recording the refund details.

  7. Bill Settlement Details screen is displayed with the services that are selected. The refund amount that is recorded for service is listed under the 'Charge Amount' column with a '-ve' sign to indicate the refund process. Enter the details in the Bill Settlement Details screen.

  8. A receipt number is generated on recording the details.
