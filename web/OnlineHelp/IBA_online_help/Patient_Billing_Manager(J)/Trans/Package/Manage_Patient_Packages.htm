Manage Patient Packages

From this function you can Modify the package definition that is associated to a patient. You can Replace an existing service with the replaceable service.

The Manage Patient Packages function is used to associate additional services to the package. The additional service associated to the package is charged based on the standard price that is defined for the respective Billing Service during definition. Authorized users' are only permitted to modify the treatment package. You can update the services associated to the package before the bill generation. Any package variance is validated when the details are modified. You can modify the details only if there is a positive package variance. You cannot modify the details when the package variance is negative. You can modify only the packages that are open. Modification is finalized for positive variance.

 

Package can be associated/de-associated, cancelled based on the user privileges defined in the 'Function Privilege Setup for Policy' screen. Multiple packages can be cancelled during a single transaction.

Managing Patient Packages

From the Transactions > Packages, Manage Patient Packages Function.

The Manage  Patient Packages screen gets displayed.

  1. Enter the Patient ID in alphanumeric format of 10 characters maximum length. The patient details get displayed on entering the Patient ID. This is a mandatory field.

  2. The Package details and the Services associated to the respective package are displayed. The package details are populated based on the package subscription for the respective patient encounter. <Related Field Description>

  3. Click [Replace Service] to replace the selected Billing Service. You can replace only rate based services. You cannot replace a charge based service, service group or service classification. A confirmation message 'Do you want to Replace the Service?' is displayed. Click [Yes] to replace or [No] to cancel. The selected service is replaced with the value selected for 'Replaceable Service'.

  4. Click [Apply] to save the records or click [Reset] to clear the data.

  5. A confirmation message 'Recalculation will be done. Confirm?' is displayed. Click [OK] to confirm. The package details are recalculated.
