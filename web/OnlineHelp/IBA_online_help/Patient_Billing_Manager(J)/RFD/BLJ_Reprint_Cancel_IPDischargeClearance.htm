Enter the following details:

 

  1. Facility ID - Select a Facility ID from the list by clicking the [?]. The logged in Facility is displayed, by default. The Facilities associated to the logged in User are listed. This is a mandatory field.

  2. Nursing Unit - Click [?] to select the Nursing Unit. A list of patients' admitted to the selected Nursing Unit and matching the query criteria are listed.

  3. Patient ID - Click [?] to select the patient ID.

  4. Episode ID - Click [?] to select the Episode ID. A list of inpatients' with the discharge advice are listed.

  5. Discharge Clearance Date [From-To] – Click the calendar icon to select date range on which the discharge clearance was prepared. The inpatients’ with the discharge clearance generated during the selected period are listed.  

  6. Options – Select an option from the list. By default, the ‘Both’ option is selected.

• Select ‘With Discharge Bill’ to view discharge clearance generated for inpatients’ with discharge bill.

• Select ‘Without Discharge Bill’ to view discharge clearance generated for inpatients’ without discharge bill.

• Select ‘Both’ to generate discharge clearance for inpatients’ with and without discharge bill.
