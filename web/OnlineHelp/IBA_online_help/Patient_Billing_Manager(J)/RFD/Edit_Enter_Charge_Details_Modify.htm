Enter the following details to update charge details:

 

  1. Service - The service associated to the patient is displayed, by default. You cannot change the service. This is a mandatory field.

  2. Included/Excluded - The value is either displayed as Include or Exclude based on the policy coverage definition. If the value is set as 'Included',  the default value listed for 'Included/Excluded Action' is set as 'Exclude'. If the value is set as 'Excluded',  the default value listed for 'Included/Excluded Action' is set as 'Include'.

You can override the value for 'Included/Excluded Action' only if 'Override' is selected as 'Yes' for the respective Billing Service.

  1. Included/Excluded Action - The value is either displayed as Include or Exclude based on the policy coverage definition and the value set in 'Included/Excluded'. You can override the setup through this field only if 'Override' is selected as 'Yes' for the respective Billing Service.

  2. Reason - Click [?] function to select a reason. The reasons that are defined through the 'Reasons for Billing Actions' function with the 'Service Override Incl/Excl' check box as selected are listed.

  3. Rate/Charge - The value is displayed as 'Rate' or 'Charge' based on the value selected during Billing Service definition.

  4. Original Quantity per Panel - The original quantity of services associated to the Panel are displayed.

  5. UOM - The unit of measure of the service is displayed.

  6. Revised Quantity per Panel - The updated quantity of service in the associated panel is displayed. This is a mandatory field.

  7. Total Quantity - Total quantity of the service is displayed.

  8. Rate - Rate is displayed for the quantity of services associated. This is a mandatory field.

  9. Practitioner/Staff Type - Select a value from the list to associate either a 'Practitioner' or a 'Staff Type' to the encounter.

  10. Practitioner/Staff - Select a value from the list to associate either a 'Practitioner' or a 'Staff Type' to the encounter. The values in the list vary based on the 'Practitioner/Staff Type' selected.

  11. Total Payable - The total payable amount is displayed.

  12. Patient Payable - Total amount to be paid by the patient for the service is displayed.

  13. Patient Paid - Total amount paid by the patient for the service is displayed.

  14. Preapproved Reqd - The check box is selected, if pre-approval is required for insurance coverage. The check box is cleared, if pre-approval is not required for insurance coverage.

  15. [Accept] - Click [Accept] to record the changes or [Cancel] to abort the transaction.
