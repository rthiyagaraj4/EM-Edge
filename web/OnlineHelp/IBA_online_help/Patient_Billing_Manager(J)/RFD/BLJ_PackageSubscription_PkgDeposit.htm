Package Deposit

Enter the following details to record package deposit details:

 

  1. Receipt Nature - The receipt nature is displayed as 'Patient Deposit', by default. You cannot edit the value.

  2. Receipt/Refund - The value is displayed as 'Receipt', by default. You cannot edit the value.

  3. Receipt Type - Click [?] function to select the receipt type. The document type selected for 'Patient Deposit for Package' in the 'Package Parameters' function gets displayed, by default. This is a mandatory field.

  4. Deposit Type - The value is displayed as 'Package Type', by default. You cannot edit the value. This is a mandatory field.

  5. Remarks - Enter the remarks in alphanumeric format.

  6. Amount - The amount to be settled as deposit for the respective package is displayed, by default. You can update the entry to specify the actual deposit amount collected from the patient or payer.

  7. Package Seq No - The Package Sequence Number is displayed, by default. You cannot edit the value.

  8. Through mail - Select the check box, if the settlement amount is received through mail.

 

Payment Details

  1. Select - Select the check box to modify or dissociate the selected settlement record.

  2. Amount Collected - The amount collected through a specific Settlement Type is displayed.

  3. Settlement Type - The Settlement Type through which the amount is settled is displayed.

  4. Cash Given - The cash amount given by the patient during bill settlement is displayed.

  5. Cash Returned - Cash amount returned to the patient after deducting the bill amount is displayed.

  6. Cheque/Card/Bank A/c No. - The cheque number, credit card number or bank account number are displayed based on the settlement type selected and details recorded during bill settlement.

  7. Date - This indicates the transaction date.

  8. Credit Card Co. - The name of the credit card company recorded during bill settlement is displayed.

  9. Bank Name - The name of the bank recorded during bill settlement is displayed for cheque settlement types.

  10. Branch Name - The name of the branch associated to the selected bank is displayed.

  11. Batch No - The batch number recorded during bill settlement for credit card settlement type is displayed.

  12. Approval Code - The approval code recorded during bill settlement for credit card settlement type is displayed.

  13. Sale Draft No - The sale draft number recorded during bill settlement is displayed.

  14. Payer's Name - The Payer's Name recorded during bill settlement is displayed.

  15. Payer Ref Id/Deposit Ref No. - The Payer's Reference ID and Deposit Reference number is displayed.

  16. Online Approval - This check box is selected for Credit Card transactions. You cannot select or clear the check box.

  17. [Add] - Click [Add] to record bill settlement details. You can record settlement details using multiple payment modes by clicking [Add].

  18. [Modify] - Click [Modify] to update bill settlement details.

  19. [Remove] - Click [Remove] to dissociate respective settlement record. A confirmation message 'Do you want to Delete the Record?' is displayed. Click [OK] to delete the record or [Cancel] to abort the transaction.  

  20. Total - The total deposit amount that is collected is displayed. You cannot edit the details.

  21. Cash Counter/Shift - The Cash Counter code and the shift to which the user has logged in is displayed. You cannot edit the details.

  22. Date - System date and time is displayed. You cannot edit the details.

  23. Click [Record] to record the settlement details or [Cancel] to cancel the process. A Receipt Number is generated on clicking [Record].
