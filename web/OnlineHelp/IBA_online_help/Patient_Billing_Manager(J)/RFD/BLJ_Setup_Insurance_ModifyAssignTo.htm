Enter the following details:

 

  1. Facility – By default, the logged in facility name gets displayed. The Facilities defined in the System Manager module and associated to the logged in User get listed. The Policy Types defined are valid within the selected Facility. This field is mandatory.

  1. Policy Type Code – Enter a valid Policy Type code in alphanumeric format of 20 characters maximum length. The Policy Type code has to be unique and cannot be changed after creation. This is a mandatory field.

  2. Long Description - Enter the long description in alphanumeric format of 80 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Enabled – Select the check box to enable the policy type. If the check box is not selected, the policy type will not be available in transactions.
