Enter the following details:

 

  1. User Type – Select the User Type from the drop-down list. The user Types defined in the system manager are listed. This is a mandatory field.

  2. User Code – Click [?] to select the User Code. The User Codes defined in the System Management function are listed. This is a mandatory field.

  3. Function Id - Select the Function Id from the drop-down list. The Function Ids defined in the System management are listed. You can provide user access rights for policy creation, policy update and policy association in the selected function id. This is a mandatory field.

  4. Privilege – Select this check box to define privilege for policies.

  5. Policy Create – Select this check box to provide user access rights for policy creation in the selected function Id.

  6. Policy Update - Select this check box to provide user access rights for policy update in the selected function Id.

  7. Policy Associate - Select this check box to provide user access rights for policy association in the selected function Id.

  8. Effective Date [From-To] - Select the date range using calendar function. The user access rights for policy definition, policy update and policy association will be effective within the date range.

  9. Click [Add] to select the Payer Group, Payer and Policy from the dialog. The Payer Group, Payers and Policies defined in the Setup Payer Billing Group function are listed.

  10. Select the check box against the record.

  11. Click [Delete] to delete the record.
