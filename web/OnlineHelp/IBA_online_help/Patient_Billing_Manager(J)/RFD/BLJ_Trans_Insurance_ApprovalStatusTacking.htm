Enter the following details:

  1. Patient ID – Select the patient ID from the lookup.

  2. Patient Name – On selecting the patient ID, the patient name gets displayed.

  3. Encounter ID – Select the required encounter ID of the patient.

  4. Episode Type – Select the episode type from the drop-down list. The list values include Inpatient, Day Care, Emergency, Outpatient, All. By default, the value ‘All’ is displayed.

  5. Payer Group – Select the payer group from the lookup.

  6. Payer – Select the required payer from the lookup.

  7. Policy – Select the required policy from the lookup.

  8. Encounter Date [From – To] – By default, the current week range is displayed. This field is mandatory. User can change the encounter date range by clicking the calendar icon. Only one week range can be selected.

  9. From Date should not be greater than To Date and Current Date

  10. To Date should not be greater than the Current system Date

  11. User Type – The User Type drop-down list values include Insurance Team, Front Office Team. The user type value gets displayed based on the user privileges defined in the Function Privileges setup for Policy.
