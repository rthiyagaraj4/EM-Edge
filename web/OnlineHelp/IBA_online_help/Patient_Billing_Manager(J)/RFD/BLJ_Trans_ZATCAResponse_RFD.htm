Enter the following details:

  1. Patient ID – Select the required patient ID from the lookup.

  2. Patient Name – Once the patient ID is selected, the patient name gets defaulted.

  3. Status – Select the required status from the drop-down list. The listed values include Valid Document, Accepted Document with warnings, Invalid Document.

  4. Episode Type – By default, the value ‘Outpatient’ is displayed. Select the required episode type from the drop-down list. The list values include Outpatient, Referral and Emergency.

  5. Encounter ID – Enter or select the encounter ID from the lookup.

  6. Payer Group – Select the required payer group from the lookup.

  7. Payer – Select the required payer from the lookup.

  8. Encounter Date [From-To] – By default current date is displayed. This field is mandatory. Change the required date range by clicking the calendar icon.

    • To date cannot be greater than system date.
