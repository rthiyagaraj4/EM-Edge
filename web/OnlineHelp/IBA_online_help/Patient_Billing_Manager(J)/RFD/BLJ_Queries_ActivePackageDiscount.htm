Enter the following details to query active package discount:

 

  1. Customer Group –  Select the required customer group from the lookup.

  2. Customer – Select the required customer from the lookup. This field is mandatory.

  3. Policy Type – Select the required policy type from the lookup. This field is mandatory.

  4. Billing Group – Select the billing group from the lookup. This field is mandatory.

  5. Package – Select the required package from the drop-down list. This field is mandatory

  6. Effective From and To – Enter the effective from and to date by clicking the calendar icon. The Effective From date is mandatory.
