Multi Package Discount

From this function you can Define discount for multiple package.

The Multi package Discount function is used to define discount percentage for multiple packages.

Issuing Refund for a Package

From the Transactions > Package, select Multi Package Discount Function.

The Multi Package Discount screen gets displayed.

  1. Click [Create] to display create screen.

  2. Enter the search criteria <Related Field Description>

  3. Click [Apply] to define multi package discount or [Reset] to clear the entered data.

  4. Click [Query] to search for existing records.
