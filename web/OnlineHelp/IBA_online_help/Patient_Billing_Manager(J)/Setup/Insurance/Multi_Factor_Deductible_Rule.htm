Multi Factor Deductible Rule

From this function you can Define the multi factor deductible rule.                    

This function is accessible only when the Multi Factor Deductible Rule check box is selected in BL Parameters (J). Else, an error message ‘This menu cannot be accessed’ will be displayed.

This co-payment rule is applicable only for pharmacy drugs and not for services, orders, and Material items.  

 

Deductible will be calculated based on

• Agreement Items

• Generic Items

• Brand Items (With alternate)

• Brand Items (Without alternate)

Defining Multi Factor Deductible Rule

From the Setup > Insurance, select Multi Factor Deductible Rule function.

The Multi Factor Deductible Rule screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. After entering the required details, click [Apply] to save the changes done or [Reset] to clear the entered details.

  4. Click [Query] to search for existing records.
