BL Parameters

From this function you can Define the time limit and colour indicator for the orders when the order status exceeds the pre-defined time for change.

This function allows to define the time limit and colour indicator for the orders for each request status such as Pending, Sent to Insurance, Hold, Approved, Rejected and Clarification. You can change the setting at any time, it is not a one-time setup.

Based on setup done, when the order status exceeds the pre-defined time for change, the colour indicator against each status gets reflected in Record Approval screen and Service Approval Status screen.  

Defining Billing Parameters

From the Setup > select the BL Parameters function.

The BL Parameters screen gets displayed.

  1. By default, the Request Status such as Pending, Sent to Insurance, Hold, Approved, Rejected and Clarification gets displayed.

  2. Enter the time limit manually for each status in HH:MM format. By default, it is mentioned as 00:00. Maximum time limit will be 99:59 (HH:MM)

  3. After entering the time limit, click the Color icon and choose any colour of your own choice and configure the colour indicator for each status.

Insurance Tab

  1. Click [Apply] to save the settings done.
