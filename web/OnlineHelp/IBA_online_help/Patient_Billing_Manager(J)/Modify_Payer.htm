Modify Payer

You can update the Payer Group, Payer and Policy Type combination associated to a patient. The adjustment rule indicator, amount indicator, adjustment value and interface code are displayed based on the rule defined in Insurance Management module for respective Payer Group, Payer and Policy Type combination.

 

Enter the following details to update Payer details:

 

  1. Encounter Date/Time -  Current date and time gets displayed as the Encounter date and time.  

  2. Search By - Select a value from the list to query Payer details based on the 'Country' or 'Region'. The values in the associated field gets updated based on the value selected for 'Search By'.

  3. Country/Region - Click [?] function to select a Country or Region. The legend changes based on the value selected for 'Search By'.

  4. Payer Group - Click [?] function to select a Payer Group. Customers defined in the Patient Billing Manager module gets listed. This is a mandatory field.

  5. Payer - The legend is displayed as a hyperlink. Click the hyperlink to view Customer details. Click [?] function to select a Customer. This is a mandatory field. The values defined through the 'Customer' function in Patient Billing Manager is displayed.

  6. Priority – The Priority recorded for the Payer is displayed. Enter the Priority in numeric format with a maximum of two digits. The Priority indicates the precedence of the Payer in the list. This is a mandatory field.

  7. Initial Credit Authorization - The Initial credit Authorization recorded is displayed. Enter the details in numeric format to update the credit authorization. This is a mandatory field.

  8. Initial Credit Auth Date - The Initial credit Authorization date recorded is displayed. Click the calendar function to update the authorization date. This is a mandatory field.

  9. Approved Amount - The approved amount that is recorded is displayed. Enter the coverage amount in numeric format to update the value. This is a mandatory field.

  10. Approved Days - The number of days selected is displayed. Enter the number of days during which the approved amount can be availed by the patient. This is a mandatory field.

  11. Policy - Click [?] function to select a Policy. Policy Types defined in the Insurance Management module gets listed. This is a mandatory field.

  12. Policy No – The Policy Number recorded is displayed. Enter the Policy Number in numeric format with a maximum of 30 digits. This is a mandatory field.

  13. Policy Start Date – Click the Calendar function to select a date. The selected Policy Type is considered valid from the date selected. This is a mandatory field.

  14. Policy Expiry Date – Click the Calendar function to select a date. The selected Policy Type is considered valid until the date selected. This is a mandatory field.

  15. Policy Effective From – Click the Calendar function to select the date beginning when the Policy is effective. The Policy Start Date is set, by default.

  16. Policy Effective To – Click the Calendar function to select the date until when the Policy is effective. The Policy Expiry Date is set, by default.

  17. Adjustment Rule Indicator - The adjustment rule indicator selected for the selected Payer Group, Payer and Policy combination in Insurance Management module is displayed.

  18. %Amount Indicator - The percentage amount indicator selected for the selected Payer Group, Payer and Policy combination in Insurance Management module is displayed.

  19. Adjustment Value - The adjustment value selected for the selected Payer Group, Payer and Policy combination in Insurance Management module is displayed.  

  20. Adjustment Interface Code - The adjustment interface code selected for the selected Payer Group, Payer and Policy combination in Insurance Management module is displayed.

  21. Additional Approvals - Click 'Dtls' displayed as a hyperlink to associate and approve a package for the encounter. The details are displayed in tabular format.

  22. Credit Authorization - Enter the credit authorization value in numeric format.

  23. Credit Auth Date - Click the Calendar function to record credit authorization date.

  24. Approved Amount - Enter the approved amount in numeric format.

  25. Approved Days - Enter the approved number of days in numeric format.

  26. Package Approvals - Select the check box to approve the package. Authorized users are only permitted to approve the package. A legend Dtls is displayed on selecting the check box.

    Enter the following details:

    Package Code - Desc - Seq No - Select a value from the list. The packages that are associated to the patient and are open are listed.

    Package Code - The code of the selected package is displayed.

     

  27. Add - Click '+' displayed as a hyperlink to associate additional records.

  28. Click [Accept] to record the details or [Cancel] to exit from the screen.

  29. Modify Policy Details – Click this hyperlink to invoke Authorization dialog. This dialog is invoked only if you do not have modification rights. Enter User ID, PIN and Remarks for authorization and proceed with policy details modification.
