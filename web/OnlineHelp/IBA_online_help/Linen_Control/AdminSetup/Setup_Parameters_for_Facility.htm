Setup Parameters for Facility

From this function you can Define rules that influence functioning of the Linen Control department within the facility.

The Setup Parameters for Facility function helps to define facility specific rules that influence the functioning of the ‘Linen Control’ module. You can define the cut-off time to the laundry to accept or deliver linen items. You can choose to display an alert or stop the user from adding linen items more than the standard stock items defined for the specific source.

Defining Setup Parameters for Facility

From the Setup > Admin Setup, select the Setup Parameters for Facility function.

The Setup Parameters for Facility screen gets displayed.

  1. Select the Cut-Off Timings from the list to accept the linen items from the facility and deliver it to the facility. The system-defined values are Not Applicable, 2 and 3. Based on the value selected, the fields are displayed under ‘Cut-Off Time for accepting linen by laundry’ and ‘Cut-Off Time for linen delivery by laundry’.

    Cut-Off Time for linen delivery by laundry:

    • If ‘No. of Cut-Off Time’ is selected as ‘Not Applicable’, the Cut-Off Timings for delivering the linen by laundry cannot be defined. A message ‘Delivery Date to be entered during Request for Cleaning’ is displayed. If ‘Not Applicable’ is selected, the field ‘Schedule Delivery Date’ in ‘Request for Cleaning’ function is enabled and you can select a date using the Calendar function. If ‘2’ or ‘3’ is selected the field ‘Schedule Delivery Date’ in ‘Request for Cleaning’ function is displayed and defaulted as ‘Null’. You cannot modify this field.

    • If ‘No. of Cut-Off Time’ is selected as ‘2’ or ‘3’, select the value from the list. The system-defined values are ‘Same Day’, ‘Next Day’ and ‘After’. Enter the cut-off timings in hh:mm format.

    • If ‘Same Day’ is selected, the cut-off time defined is applicable for the same day when the linen was given to the laundry for washing.

    • If ‘Next Day’ is selected, the cut-off time defined is applicable for the ‘Next Day’ on when the linen was delivered to the laundry for washing.

    • If ‘After’ is selected, a text box gets displayed to enter the number of days in numeric format of two digits maximum length after which the linen has to delivered by the laundry.

    Cut-Off Time for accepting linen by laundry:

    • If ‘No. of Cut-Off Time’ is selected as ‘Not Applicable’, the Cut-Off Timings for accepting linen by laundry cannot be defined. A message ‘Delivery Date to be entered during Request for Cleaning’ is displayed. If ‘Not Applicable’ is selected, the field ‘Schedule Delivery Date’ in ‘Request for Cleaning’ function is enabled and you can select a date using the Calendar function. If ‘2’ or ‘3’ is selected, the field ‘Schedule Delivery Date’ in ‘Request for Cleaning’ function is displayed and defaulted as ‘Null’. You cannot modify this field.

    • If ‘No. of Cut-Off Time’ is selected as ‘2’, the Starting time of the first slot is displayed as 00:00. Enter the end time of the first cut-off time slot and the start time of the second cut-off time slot. The end time of the second cut-off time slot is displayed as 23:59. The values 00:00 and 23:59 are displayed and disabled.

    • If ‘No. of Cut-Off Time’ is selected as ‘3’, the Starting time of the first slot is displayed as 00:00. Enter the end time of the first and second cut-off time slot. Enter the start time of the second and third cut-off time slot. The end time of the third cut-off time slot is displayed as 23:59. The values 00:00 and 23:59 are displayed and disabled.

  2. Select the check box ‘Dirty Linen to be entered by Laundry’ to authorize laundry to collect and enter the dirty linen items. If the check box is cleared, the laundry personnel are not allowed to collect the dirty linen items from wards.

  3. Select the ‘Ward should confirm the Dirty Linen counted by Laundry’ check box to authorize the ward to confirm the dirty linen given by Laundry. Clear the check box, if the Ward is not allowed to confirm the count of dirty Linen given by the laundry.

  4. Select a value from the list to allow or alert the user when a request for linen item exceeds the standard stock mentioned for the source in a facility. The system-defined values are ‘Allow’, ‘Stop’ and ‘Alert’.  

    • If ‘Allow’ is selected, the system allows you to add more linen than the standard stock quantity.

    • If ‘Stop’ is selected, the system does not allow you to add linen to the standard stock quantity.

    • If ‘Alert’ is selected, the system alerts you whether there is any addition or deletion in the standard stock quantity of the linen item by the laundry.

  5. Click [Apply] to save the created record or [Reset] to clear the entered details.

The Setup Parameters for Facility screen gets displayed for the logged in Facility without any values, on invoking the function from the menu for the first time. If the parameter record exists for the logged in Facility, existing parameter details are displayed in the updated mode on invoking the function.
