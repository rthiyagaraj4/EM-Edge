Laundry Users

From this function you can Authorize a User to perform the transactions in the linen control department. You can Query for an existing Laundry User.

The Laundry Users function is used to authorize users in the Linen Control department to issue and enter the cleaned linen items. The users defined in the System Manager module and associated to the logged in facility get listed. Only the authorized users can perform transactions using the Linen Control module.

Authorizing a User

From the Setup > Code Setup, select the Laundry Users function.

The Laundry Users screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the User using [?] function. The values defined using the 'User' function in System Manager module get listed. This is a mandatory field.

  3. Select the check box ‘Override Ward Confirmation’ to authorize the selected user to override the details. Clear the check box, if the user is not authorized to override the details.

  4. Click [Apply] to save a created Laundry User or [Reset] to clear the entered details.

  5. Click [Query] to search for the existing records.

  6. Click [Delete] to delete an existing record.
