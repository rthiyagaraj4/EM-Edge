Request for Linen

From this function you can Place a request for a new Linen Item to a laundry department within the Facility. You can Cancel or View the Request List and View the Stock Status and Work Status of the Linen Items.

The Request for Linen function is used to place an order for new linen items. This helps to maintain a standard stock level in the respective Nursing Units/Clinic/Department. When new linen items are purchased, details of the Linen and respective quantity are recorded. The order placed can be viewed, modified or cancelled according to the requirement. A request number is generated by the system for each request placed. The Request Number generated is used to track the request for a new linen item from the Nursing Units/Clinics/Department to avoid shortage of linen in emergency. You have to select a Source Type and Source for which the linen request has to be placed. The values selected are defaulted and displayed in the transaction functions.

Placing a request for new linen items

From the Linen > Linen Transaction, select the Request for Linen function.

The Request for Linen screen gets displayed with two tabs. Place Linen Request tab is enabled.

  1. Enter the details in the mandatory (*) field in the tabs:

Place Linen Request

Requests List

  1. You can place a new request, view or cancel the requests placed.

  2. The Stock Status and Work Status of the linen items in the laundry are displayed. You can choose to view the status of the linen items in laundry by selecting a source Type and a Source. The current system is displayed. Click [Go] to search for the details in the respective Source and Source Type.
