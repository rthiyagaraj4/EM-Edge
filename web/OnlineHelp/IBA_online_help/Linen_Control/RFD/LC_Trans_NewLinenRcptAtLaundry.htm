Enter the following details:

 

  1. Receipt Num – The Receipt Number is system generated. The Receipt Number is incremented by one to record the new linen received. You cannot edit this field. This is a mandatory field.

  2. Received Date – The system date and time is displayed, by default. This indicates the date and time when the linen item is received by the laundry. You cannot change the data. This is a mandatory field.

  3. Linen Item – Select the Linen Item using [?] function. Linen items for which standard stock is defined in ‘Standard Stock for Sources’ function gets listed. This is a mandatory field.

  4. Received Qty – Enter the quantity received in numeric format.

  5. Received By – Enter the name of the person who is responsible for collecting the new linen at the laundry. The user name is displayed, by default. This is a mandatory field.

  6. Remarks – Enter the remarks in alphanumeric format of 100 characters maximum length.
