Place Linen Request

You can place a order for new linen item using this tab.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Select] to display a Linen Request placed or [Cancel] to clear the entered or displayed details by clicking the Linen Item name, which is displayed as a hyperlink.

• If ‘Allow linen more than std. stock qty.’ value is selected as ‘Allow’, the system allows you to place request for linen items more than the standard stock defined in ‘Standard Stock for Sources’ function in Admin Setup for the selected location.

• If ‘Allow linen more than std. stock qty.’ value is selected as ‘Stop’, the system do not allow you to place request for linen items more than the standard stock defined in ‘Standard Stock for Sources’ function in Admin Setup for the selected location. It displays an error message ‘Requested Quantity should be less than or equal to Std. Stock’.

• If ‘Allow linen more than std. stock qty.’ value is selected as ‘Alert’, an alert message ‘The requested quantity exceeds the standard stock for the selected location. Do you want to continue?’ is displayed. Place request for linen items more than the standard stock defined in ‘Standard Stock for Sources’ function in Admin Setup for the selected location. It displays an error message ‘Requested Quantity should be less than or equal to Std. Stock’. Click [OK] to proceed with placing the request for linen items or click [Cancel] to exit without placing an order.

You cannot place request for same linen item while placing an order under a Request Number.

  1. The selected Linen Item is displayed along with the requested quantity and remarks. The Linen Item and Remark fields are hyperlinked. On clicking the Linen Item hyperlink, the selected Linen Item and its requested quantity is displayed in the respective fields for editing. Click Remark hyperlink to view the remarks recorded. To edit the remark, select the Linen Item hyperlink and enter the details by clicking ‘Remark’ hyperlink. Click [OK] to save the details.

  2. Click [Submit] to save the new linen request placed for a Source Type. A confirmation message ‘Please note your Request Number’ is displayed. Click [OK] and the control shifts back to ‘Place Linen Request’ tab to place the next request.
