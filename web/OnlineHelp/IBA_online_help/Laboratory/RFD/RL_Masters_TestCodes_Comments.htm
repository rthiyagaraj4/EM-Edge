Comments

Comments tab allows you to define Result Comments, Test Comments and Request Comments for the test.

 

Result Comments

This tab enables you to associate ‘Result Comments’ to the ‘Test Codes’.  

  1. Code - Enter the Result Comment Code or click [List of Values] function to view a list of result comments and select the required comment code. The listed values are created in Result Comment master.

  2. Description - The corresponding description for the selected Result Comments Code is displayed, by default.

  3. Reflex Text - Click [List of Values] to select the required code. List of values displays data entered through the 'Result' function.

  4. Prompted - Select the check box to display the result comment in result entry, by default for the selected test.

  5. Review - Select the check box to send the result for 2nd level review after releasing the result.

  6. Notify? - Select the check box to send a sms alert/email notification to the concerned practitioners, facility and department while releasing the test results. Clear the check box, if notification is not required.

 

Test Comments

Test comment is used to enter the standard comments for the test. The standard comments are printed in the result report against the test code, by default.

  1. Test Comments - Enter the Test Comments in alphanumeric format of 40 characters maximum length. These test comments will be displayed in the Laboratory reports.

 

Request Comments

This tab enables you to define result comments for the test. This tab gets enabled on selecting ‘Comments’ check box under ‘Validations’ tab.

  1. Request - Click [List of Values] function to view a list of request codes and respective description entered through the 'Request' function.

  2. Description - The Description of the selected Request code is displayed.

 

Result Text

  1. Code – Select the code using lookup function. The codes defined in the masters are displayed.

  2. Description –The description for selected code is displayed.

  3. Seq. No - Select the Seq.No using lookup function. The Sequence Numbers defined in the masters are displayed. This is a mandatory field.
