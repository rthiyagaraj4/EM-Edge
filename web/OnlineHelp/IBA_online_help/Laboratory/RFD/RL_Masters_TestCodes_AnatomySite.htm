Anatomy Site

The ‘Anatomy Site’ tab enables you to define multiple anatomy site details that are required as a part of the investigation. This tab is  enabled on selecting ‘Anatomy Site’ check box under ‘General’ sub tab of ‘Attribute’ tab.

 

Enter the following details:

  1. Anatomy Site - Click [List of Values] to select the Anatomy Site. The listed values are created in 'Anatomy Site' masters.

  2. Description - The corresponding description for the selected Anatomy Site is displayed, by default.

  3. Fixative - Click [List of Values] to select a Fixative. The listed values are created in 'Fixative' function.

  4. Description - The corresponding description for the selected Fixative is displayed, by default.
