Enter the following details:

  1. Search With - Select the criteria for search from the list. The listed values are 'All' and 'Patient ID'. The value ‘All’ is selected, by default. This is a mandatory field.

  2. Patient ID - Enter the Patient ID in the length as specified in the MP Parameter. This is a mandatory field, if the value ‘Patient ID’ is selected in the ‘Search With’ field.

 

Appointment Details

  1. Ref No - Enter the Appointment Reference Number in alphanumeric format of 30 characters maximum length.

  2. Date [From – To] - Enter the 'From Date' and ‘To Date’ or select the date range using the Calendar function. The ‘From Date’ entered should not be less than the current system date. This is a mandatory field.

  3. Specialty - Enter the Specialty code in alphanumeric format of four characters maximum length or click [List of Values] to select the Specialty. The listed Specialties are defined in the Application Masters module.

  4. Location - Select the Location Type from the list. The value listed is ‘Procedure Unit’. On selecting the Location Type, select the Location. Enter the Location or click [List of Values] to select the Location.
