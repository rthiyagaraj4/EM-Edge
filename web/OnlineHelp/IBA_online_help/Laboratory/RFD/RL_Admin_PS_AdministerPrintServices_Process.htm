The Process tab includes two tabs: Administer Tab and Active Tab

 

Administer Tab

Enter the details in the Administer Tab:

  1. Process – The Process displays a list of active print services.

  2. Facility ID - The processes which are defined as multiple instances, for those processes facility can be entered by selecting individual facility from list or '*ALL' for all facilities. This field is disabled, if the selected process is of single instance. If single facility is selected, server process processes the request for that facility only. If '*ALL' is selected, the server process processes the request for all facilities. You cannot restart the process, if it has been started for a single facility. Similarly, if process has started for all facilities, it cannot be restarted for a single facility.

  3. Priority - The process which are defined as multiple instances, for those processes priority can be entered by selecting individual priority from list box or '*ALL' for all priorities. This field is disabled if the selected process is of single instance. If single priority is selected, server process will process the request for the respective priority. If '*ALL' is selected, the server process processes the request for all priorities. Once the process has started for a single priority, it cannot be started for all the priorities. Similarly, if process has started for all the priorities, it cannot be started for single priority.

  4. Serviced Requests - The service requests can be entered for the server process 'Print Services' only. Currently two services named 'Lab Result Reporting' and 'Patient Bill Printing' are available for selection. Select '*ALL' to process print requests generated from both processes. This field is disabled for other processes. If Lab Result Reporting' service is selected, the 'Print Services' process prints the requests created by 'Lab Result Reporting' based on the priorities defined. If 'Patient Bill Printing' service is selected, the 'Print Services' process prints the requests created by 'Patient Bill Printing' process.

  5. Click [Start] to start the process. Once the process has started for all facilities (by choosing ‘*ALL’ for facility), it cannot be restarted for an individual facility.

 

Active Tab

  1. Click [Stop] to stop the server process. The users who have started the process only can stop the process.

  2. Process – Select the name of the active Process from the list of valid values.

  3. Date/Time - Process start date and time is displayed.

  4. Machine Name – Name of the computer for which the process has started is displayed.

  5. Started By – User ID who has started the process is displayed.

  6. Facility – Name of the facility for which the process has started is displayed.

  7. Priority - Priority for which the process has started is displayed.

  8. Serviced requests – Service requests for which the process ‘Print services’ has started is displayed.
