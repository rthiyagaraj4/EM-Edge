Enter the following details:

  1. Group Test Code - Enter a valid Group Test Code in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Department - The logged in Department Code is displayed, by default. Select the required department from the list if you have access to more than one department.

  3. Long Description - Enter the Long Description of the Group Test in alphanumeric format of 40 characters maximum length. You can search a test based on long description. This is a mandatory field.

  4. Sub Department – Click [List of Values] to select the sub department. The sub departments defined in the Department masters are listed.

  5. Sequence No – Enter a valid Sequence Number of the test code in numeric format of five digits maximum length. This number is used to sequence the test codes in the result reports.

  6. Short Description – Enter the Short Description of the Group Test in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  7. Avg. TAT – Enter the average turn around time required for the Group Test in numeric format of five digits maximum length.

  8. Avg. TAT Unit – Select the required unit for average turn around time from the Avg. TAT Unit list. The list displays ‘Day’, ‘Hours’ and ‘Minutes’.

  9. Specimen Type – Select the Specimen Type check box to associate a Specimen Type to the Group Test. Clear the check box, if specimen is not required for the Group Test. Click [List of Values] to associate a specimen. The values defined through the ‘Specimen Type’ function get listed. The respective description of the selected specimen type is displayed.

  10. Volume – Enter the required Volume of the specimen required to perform the group test in numeric format of four digits maximum length. Click [List of Values] to select the required unit of volume.

  11. Notes for Test – Double-click the text box to invoke an editor to enter required notes for the group test. Click [OK] to save the notes or [Cancel] to discard. The required data can be searched by clicking [Search].

  12. Name on Report - Select the Name on Report check box to print the name of the Group Test in the test result report.

  13. Reflex Test - Select the Reflex Test check box to indicate that the group test requires you to perform other specified tests. On selecting check box, the system enables Reflex Test tab where you can specify a list of other tests that has to be performed along with this test.

  14. Anatomy Site – Select the Anatomy Site check box, if the group test requires you to enter the anatomy site details as a part of the investigation. The Anatomy Site tab is enabled to specify multiple anatomy site details on selecting the check box.

  15. Profile – Select the check box to indicate that the group test is a Profile test.

  16. Process Date – Select the Process Date check box, if the group test requires the process date as a part of registration. You can also use the process date while creating batch process.

  17. Display Result – The ‘Display Result’ check box gets enabled on selecting the Process Date check box. Select the Display Result check box if the result is to be displayed as a part of result entry by batch.

  18. Aliquot Label - Select Aliquot Label check box if the test requires aliquot label to be printed while registration.

  19. No. of Aliquot Labels – Enter the required number of Aliquot Labels to be printed in numeric format of three digits maximum length. You can specify the number of aliquot labels only on selecting the ‘Aliquot Label’ check box.

  20. Slide Label – Select the Slide Label check box if the test requires slide labels to be printed in Performing Task Function.

  21. No. of Slide Labels – Enter the Number of the Slide Labels to be printed in numeric format of three digits maximum length. You can specify the number of slide labels only when the check box ‘Slide Label’ is selected.
