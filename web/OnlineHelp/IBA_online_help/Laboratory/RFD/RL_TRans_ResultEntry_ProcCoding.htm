Enter the following details:

  1. Released Date [From-To] - Enter the Released Date and Time range or select the released date and time using Calendar function.

  2. Specimen No [From-To] - Enter the Specimen Number range in numeric format of 12 digits maximum length.

  3. Patient ID - Enter the Patient ID or right-click on the ‘Right Click for User Options’ function to search for a patient record using the ‘Standard Patient/Outpatient Episode/Visit’ screen. Double-click the Patient ID in the displayed list to select the patient record.

  4. Search Criteria - Select the search criteria value from the list. The listed values are Procedure Coded, Procedure Not Coded and All.
