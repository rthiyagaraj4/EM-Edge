Enter the following details:

  1. Patient ID – Enter a valid Patient ID or right click on ‘Click for User Option’ hyperlink to select from the user options.

  2. Specimen No [From-To] - Enter the range for Specimen Number in alphanumeric format of 40 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes the lowest value and if ‘To’ value is not entered, the system assumes the highest value.

  3. Regd Date [From-To] - Enter the range for Registered date or select it using calendar function. Both the values are optional both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value. If the value is not entered, the system assumes the lowest value for the ‘From’ and if ‘To’ value is not entered, the system assumes the highest value for the ‘To’ value. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  4. Specimen Status – Select the Specimen Status from the list. The valid values are ‘All’ and ‘Outstanding’. Specimen details get populated based the selected criteria. You can select the particular specimen, whose label needs to be reprinted.
