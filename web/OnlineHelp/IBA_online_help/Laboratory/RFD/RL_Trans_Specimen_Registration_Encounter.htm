Select Encounter

Enter the following details:

  1. Ordering Facility – Select the Ordering Facility name from the list. The logged in facility name is displayed, by default.

  2. Set Default Facility – Click [Set Default Facility] to set the default facility name.

 

Enterprise Patients

  1. Patient ID – Enter the Patient identification number in the length as specified in the MP Parameter or right click ‘Right Click for User Option’ link to select the Patient ID using ‘Standard Patient Lookup’ or ‘Standard Patient/IP Episode/ADT Transaction’ function.

  2. Episode Type – Select the Episode Type from the list. The listed values are ‘All’, Inpatient’ and ‘Outpatient’.

  3. Perform Site – Select the desired Perform Site to process the specimen from the list. The listed values are defined in the Perform Site masters function.

  4. Search – Click [Search] to search the enterprise patients’ details.

  5. Select as Referral – Click [Select as Referral] to select the selected patient as referral.

 

Encounter List

On searching, the encounter details are displayed in the ‘Encounter’ tab. Select the ‘Select’ check box to select the ‘Test’ in the ‘Test’ field under Investigations section for registration. If the check box is cleared, you will not be able to select the ‘Test’ from the Investigations section to register the specimen. Select the ‘Select Order’ check box to select the order for registration which is placed in the Order Entry and Result Reporting module.

 

Orders

The order details are displayed in the Orders tab. Select the ‘Select Order’ check box to select the order to register.

 

External Patients

  1. Patient ID – Enter the patient identification number to search for the external patient records.

  2. First Name – Enter the first name of the patient in alphanumeric format of 30 characters maximum length.

  3. Last Name – Enter the last name of the patient in alphanumeric format of 30 characters maximum length.

  4. Ref Source – Enter the referral source of the patient in alphanumeric format of 30 characters maximum length.

  5. Referral Patient ID – Enter the referral patient identification number in alphanumeric format of 30 characters maximum length.

 

Click [Search] to search for the patients based on the entered criteria or [New Patient] to register the external patient.

On searching the external patient details are displayed. Select the ‘Select’ check box to select the external patient record for registration.
