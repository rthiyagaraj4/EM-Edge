Enter the following details:

  1. Department - The logged in Department Code is displayed, by default. Select desired department from the list if you have access to more than one department. This is a mandatory field.

  2. Quality Code – Enter the Quality code or select the desired quality code using [List of Values] functions. The list displays the values defined in the Quality Codes masters function. Quality control material code is a unique code to identify the quality control material. This code is used in the worklist format setup to define a Cup Type. This is a mandatory field.

  3. Manufacturer ID - The corresponding Manufacturer ID and description associated to the 'Quality Codes' in the 'Quality Code' function is displayed, by default.

  4. Batch/Lot No. - Enter the vendor supplied Batch Number for the quality control material in numeric format of 20 characters of maximum length. This is a mandatory field.

  5. Expiry Date - The current system date is displayed, by default. You can select the desired date using the Calendar function.

  6. Analyzer Code – Click [List of Values] to select a Analyzer Code for which the Quality Code material is applicable.  The list displays the values defined in ‘Analyzers’ function. This is a mandatory field.

  7. QC Specimen No - The Batch number is displayed as Quality Control Specimen Number. Enter the specimen number in alphanumeric format of 20 characters to record a different specimen number.

 

Quality Tests

  1. Test Code - Click [List of Values] to view and select the required test code. The list displays the values defined in the ‘Test Codes’ function. You can associate the same test code to different quality control materials, but duplication is not allowed in the same quality control material. Numeric result has to be applicable for the selected test code. This is a mandatory field.

  2. Mean - Enter the Quality Control materials Mean Value for the test. This is a mandatory field.

  3. Std Deviation - Enter the Quality Control materials Standard Deviation for the test. This is a mandatory field.

  4. Coefficient of Variation - Enter the Quality Control materials Coefficient of Variation for the test.

  5. Low - Enter the low value of the normal range of numeric result. This is a mandatory field.

  6. High - Enter the high value of the normal range of numeric result. This is a mandatory field.

  7. Last - The result for the Quality Codes and test combination updated manually or from work list.

  8. Last Result Date/Time - Enter the Date and Time of resulting.

  9. Result Status - The status gets updated based on the validation for the Quality Code result.
