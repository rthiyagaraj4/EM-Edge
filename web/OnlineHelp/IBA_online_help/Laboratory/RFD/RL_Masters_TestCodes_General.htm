General

The Attribute tab and General sub tab is displayed, by default.

  1. Specimen Required - Select the check box, if the test requires a specimen. The Specimen Type tab is enabled on selecting check box to specify required specimen or specimens’ details.

  2. Spec Type – Click [List of Values] to select a Specimen Type. The values created through the function ‘Specimen Type’ function get listed.

  3. No. of Specimens – Enter the Number of Specimens required for processing the test in numeric format of three digits maximum length.

  4. Volume – Click [List of Values] to select the required specimen Volume.  The list displays the values defined in Units of Measurement function of the ‘Application Masters’ module.

  5. Aliquot Label - Select Aliquot Label check box, if the test requires aliquot label to be printed while registration.

  6. No. of Aliquot Labels – Enter the required Number of Aliquot Labels to be printed in registration function.

  7. Slide Label – Select the Slide Label check box if the test requires slide label to be printed in Performing Task function.

  8. No of Slide Labels – Enter the Number of Slide Labels to be printed Performing Task function.

  9. Confidential Test – Select the check box to specify the test as a confidential test. The result of these tests ordered is confidential. System provides a separate print option to print confidential test reports.

  10. Inhibit Report - Select the Inhibit Report check box to indicate not to print or display the test report.

  11. Name on Report – Select the Name on Report check box to print the name of the test in the test result report.

  12. Anatomy Site – Select the Anatomy Site check box if the test requires the anatomy site details as a part of the investigation. On selecting the check box, the ‘Anatomy Site’ tab is enabled to specify multiple anatomy site details.

  13. Essential – Select the Essential check box to indicate the test as an essential test. System allows you to group individual test to create a panel. This is used to decide the status of a group test based on the status of all its individual tests. The panel test is complete if all the essential individual tests associated to it are completed.

  14. Patient Marker – Select the Patient Marker check box if the test result is to be available as a marker.

  15. Interval Test - Select the Interval Test check box if the test needs to be conducted in specified intervals. The ‘Interval Tests’ tab is enabled on selecting the code to record details of the interval test. An Interval test requires multiple specimens and you can generate a single report for all the specimens.

  16. Reflex Test – Select the Reflex Test check box to indicate that the test requires additional specified tests. On selecting the code, the ‘Reflex Test’ tab is enabled to specify a list of other tests that has to be performed along with the Reflex test.

During specimen registration if the user selects a reflex test, additional tests defined in Reflex Tab of the selected text has to be performed.

  1. Text Block – Select the Text Block check box to enter the result of the test in text format.

  2. Nomenclature – Select the Nomenclature check box to enter Nomenclature for the required test.

  3. Culture Test – Select the Culture Test check box if the report for the test is based on culture result.

  4. Validity – This field is enabled, if Culture Test check box is selected. Select the required value from the Validity list to validate the organism for susceptibility. The listed values are Manual, MIC (Minimal Inhibitory Concentration), KB (Kirby Bauer Test) and KB/MIC.

  5. Result Indicator - This field gets enabled if Culture Test check box is not selected. Select the required value from the Result Indicator list. The valid values are ‘Preliminary’ and ‘Addendum’.

  6. Preliminary Release – Select the check box to indicate the test requires Preliminary Release.

  7. Exclude 1st Level Review – Select the check box to exclude the test for first level review.

At the time of specimen registration if you select a reflex test, you have to book other tests defined in Reflex Tab of the selected text.

  1. Process Date – Select the Process Date check box to specify the process date for the test. You can create a batch of tests based on the test process date.

  2. Display Result – This field is enabled, if ‘Process Date’ check box is selected. Select the Display Result check box if the result is to be displayed as a part of result entry by batch.

  3. Avg. TAT – Enter the Average Turn Around Time for the test.

  4. Avg. TAT Unit – Select the required value from the Average Turn Around Time Unit list. The list displays ‘Day’, by default. The valid values are ‘Day’, ‘Hours’ and ‘Minutes’.

  5. Include Status – Select the Include Status check box to include the test in the periodic workload statistics report.

  6. Work Units – This field is enabled on selecting the Include Status check box. Enter the work units for the test. This is used in deriving the workload statistics for the test.

  7. Work Units Addl. Factor – Enter the additional work units for the test. This is used in deriving the workload statistics for the test.

  8. Prompt Result – Select the Prompt Result check box if you want to enter the numeric result values for the test at the time of specimen registration. If the test is a part of a panel test, the first individual test with prompt result is used at the specimen registration stage.

  9. Legend – Enter the Legend in alphanumeric format of 10 characters maximum length to be displayed before the result field while resulting.

  10. Flag – Select the required value from the list. The selected flag value sets the condition for reporting if the result is not entered. The valid values are ‘Error’ and ‘Warning’.

  11. NABL Accredited hyperlink – Click the hyperlink to invoke the NABL Accredited dialog where user can select the NABL Accredited check box for All facilities independently. Select the 'NABL Accredited' check box to print NABL logo in the Specimen Result report. For any specimen number having at least one test for which ‘NABL Accredited?’ check box is selected, the logo gets printed at the header of the specimen result report printed from both 2T and 3T. This is applicable for users in Artemis.
