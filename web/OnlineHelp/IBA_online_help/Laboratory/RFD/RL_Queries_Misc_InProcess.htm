Enter the following details:

  1. Department - The department name associated with the logged in user is displayed. If the user has access to multiple departments, select the desired department name from Department list.

  2. Perform Site - Enter the Perform Site code or select the Perform Site using [List of Values] function. The list displays the values defined in Perform Site masters function.

  3. Include? - Select the values from the list. The listed values are 'Outstanding Test' and 'All Tests'. The value ‘All Tests’ is displayed, by default.

  4. Specimen No [From-To] – Enter the specimen number in numeric format of 15 characters maximum length or select the Specimen Number using [List of Values] function.

  5. Priority - The Priority of the order placed is displayed. The listed values are ‘All’, 'Routine', 'Stat' and 'Urgent'. The value ‘All’ is displayed, by default.

  6. Priority Specimen – Select the check box to consider the specimen as high priority.

  7. Category No [From – To] – Enter the category code range or select the Category Number using [List of Values] function. The listed values are created in Category master.

  8. Assign To - The name of the person to whom the selected specimen processing is assigned, is displayed.

  9. Workgroup Code - Select workgroup name from the [List of Values] function. The list displays the values defined in Work Group masters function.

  10. Test Code - Enter the Individual Tests code or click [List of Values] function to view a list of tests created while setting up in 'Tests Codes' masters function and select the desired Test Code.

  11. Reference Laboratory - Enter the Reference Laboratory code to which the test is contracted or select the reference laboratory name using [List of Values] function. The list displays the values defined in the Referral Lab masters function. When a valid code is entered, the short name of the code is displayed for user verification.

  12. Source - Select the source from the list. The listed values are 'Ward', 'Clinic' and 'External'. Based on the selected Source, you can enter or select the values using [List of Values] function.

  13. Start and End Date - Enter the start and end date of the specimen result entry using Calendar function.

  14. Type – Select the Type from the drop down list. The listed values are ‘Specimen No’, ‘Registration Date’, ‘Source Code’, ‘Test code’ and ‘Sorting list’.

  15. Order By - Select the Order By from the drop down list. The listed values are ‘AESC’ and ‘DESC’.
