Cup Details tab

Cup Details tab displays the details of the cups specified. You can specify a maximum of 250 cups. The information for individual specimen cups comprise of patient samples and quality control samples. The cups meant for patient samples are entered as 'P' and the quality codes entered as 'Q', where required. The pattern for the cups entered here must be equal to the number of cups.

 

Quality Code tab

Quality Code tab enables you to associate respective quality codes to the cups. This tab is enabled, only if a Quality Sample is entered in the ‘Cup Details’.

  1. Cup Position - The ‘Number of Cups’ entered is displayed, by default.

  2. Quality Code – Click [List of Functions] to associate a Quality code. The values defined using the ‘Quality Codes’ function gets listed to be associated to the corresponding quality cup.

  3. Description - Description of the quality code is displayed.

  4. Click [Close] to exit from the screen.
