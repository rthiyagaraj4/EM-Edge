Organism Tab

Enter the details in the Organism Tab:

  1. Code - Enter a unique organism code in alphanumeric format of four digits maximum length. This is a mandatory field.

  2. Long Description - Enter the long description for the Organism code in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description for the organism Code in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Antibiotic Group Code – Enter the antibiotic code or click [List of Values] to select the antibiotic group code used to treat the isolated organism. These codes are created in Antibiotics tab. This is a mandatory field.

  5. Description - The corresponding Description of the selected antibiotic group code is displayed, by default.

  6. Classify Organism - Select a value from the list to classify the organism with the infected type of organism. The system-defined values are ‘Bacteria’, ‘Myco Bacteria’, ‘Yeast’, ‘Virus’, ‘Parasite’ and ‘Fungus’.

  7. Infectious? - Select the check box to indicate that the organism is infectious. Clear the check box to indicate that the organism is not infectious.

  8. Notify? - Select the check box to notify  the concerned personnel through sms alert. . Clear the check box, if notification is not mandatory for the respective organism.

 

KB Tab

Click KB (Kirby Bauer test) tab to invoke the tab page and define microbiological susceptibility for the defined antibiotic as Zone Break Point.

  1. Organism - The Organism code selected in the Organism Tab frame is displayed, by default.

  2. Description - The Description of the corresponding organism is displayed, by default.

  3. Antibiotic - The Antibiotic codes associated to the Antibiotic Group gets displayed, by default.

  4. Description - The Description of the corresponding Antibiotic Code is displayed, by default.

 

Zone Break Points

You can define the zone break points for each antibiotic.

  1. Susceptibility - Select the required value from the list. The system-defined values are 'Resistant', 'Intermediate' and 'Sensitive'.

  2. Min Value - Enter the Minimum Value for susceptibility in numeric format of six digits maximum length.

  3. Max Value - Enter the Maximum Value for susceptibility in numeric format of six digits maximum length.

The zone range values should not overlap. You can delete the zone break points associated to the Organism by clicking [Delete Record]. A confirmation message ‘Do you want to delete this record?’ is displayed. Click [Yes] to delete or [No] to cancel.

 

MIC Tab

Click MIC (Minimal Inhibitory Concentration) tab to define the lowest concentration of antibiotic sufficient to inhibit the bacterial growth.

  1. Organism - The Organism Code and the description of the corresponding Organism is displayed, by default.

  2. Antibiotic - The Antibiotic Code and the description of the Antibiotic Code is displayed, by default.

  3. Susceptibility - Select the required Susceptibility from the list.

  4. Mic Value - Enter the MIC Value for the susceptibility.

 

Bio Chemical Tests Tab

Click Bio Chemical Tests tab to define Bio Chemical Tests for an Organism.

  1. Organism Code - The Organism Code and the corresponding description of the Organism is displayed, by default.

  2. Description – The description for the selected Organism Code is displayed, by default.

  3. Bio Chem Test - Click [List of Values] to select a Bio Chemical Test code. The Bio Chemical Test validates the type of infection. The created organism values are displayed in this list.

  4. Test Description – The description for the selected Bio Chemical Test is displayed, by default.
