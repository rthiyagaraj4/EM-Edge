Enter the following details:

  1. Code – Enter a unique SNOMED code in alphanumerical format of 18 characters maximum length. This is a mandatory field.

  2. Description – Enter the Description for the SNOMED Code in alphanumeric format of 300 characters maximum length. This is a mandatory field.

  3. SNOMED Marker – Click [List of Values] to select a SNOMED code. The listed values are defined in Nomenclature (SNOMED) Marker function.

  4. Print Code – Select the check box to print the code while generating lab reports or clear the check box not to print the code.

  5. Print Description – Select the check box to print the description along with the code while generating lab reports or clear the check box not to print the description along with the code.

  6.  Terminology Set ID/Description – Select the terminology set ID/description from the drop-down list. The drop-down list value is populated from the Terminology Set master setup in Medical Records module. If this field is blank, then it is considered  SNOMED by default.
