Enter the following details:

  1. Department - The logged in Department Code is displayed by default. Select desired department from the list if you have access to more than one department.

  2. Worklist Name - Click [List of Values] to view and select a valid worklist name. The list displays the values defined in ‘Worklist Format’ function. This is a mandatory field.

  3. Worklist Date – System date is displayed, by default. Click the Calendar function to select any other date.

  4. Worklist No - The number last used as recorded in the ‘Worklist Format’ is displayed. You can edit this worklist number. The ‘Last Number Used’ gets updated in ‘Worklist Format’ function.

  5. Analyzer Code - Click [List of Values] to select an Analyzer, if the worklist has to be processed by analyzers.

  6. Perform Site – Perform site associated to the analyzer is displayed, by default.

  7. Assign To – Enter the name of assign to in alphanumeric format of 30 characters maximum length or select it using [List of Values] function. The is a mandatory field.

  8. Category No [From - To] – Enter  the category number range 'From' and 'To' that needs to be included in the Worklist.

  9. Specimen No [From-To] – Click [List of Values] to select the specimen numbers. The specimen number generated during specimen registration is listed. This is a mandatory field.

  10. Default – Select ‘Default’ option from the list box to start the specimen range from the Default number that was processed in the previous worklist.

  11. Last Specimen Number – Select the ‘Last Specimen No’ option from the list to start the specimen range from the last specimen number that was processed in the previous worklist.

  12. Print Worklist? –Select the check box to print the worklist.
