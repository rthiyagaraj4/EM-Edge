Enter the following details:
 

  1. Patient Type – Select ‘Enterprise Patient’ or ‘External Patient’ from the drop down list. Based on the selected value the high risk condition details of the patient gets displayed.

  2. Patient ID – Enter the Patient ID or select the Patient ID from the Patient lookup function.

If you select ‘Enterprise Patient’ as Patient Type, you have to select patient ID for the enterprise patient, else error message ‘Patient ID is invalid’ is displayed. If you select ‘External Patient’ as Patient Type, you have to select patient ID for the external patient else error message ‘Patient ID is invalid’ is displayed.

  1. Patient Name – The description of the Patient ID defined in the masters is displayed.

  2. High Risk Condition – Click [List of Values] to select High risk condition values which are already defined in the CA masters. This is mandatory field.

  3. Description – The description of the selected High Risk Condition value is displayed. You cannot edit the data.

  4. Remarks – Enter remarks of the high risk condition in alphanumeric character format.

  5. Onset Date - Select the Date using the calendar function. By default the current date is displayed. The Onset date is not future date. If you select future date, the error message is displayed. This is mandatory field.

  6. Day – If patient is unaware of the Onset date of high risk condition, you can enter the date. If the checkbox next to the Onset Date is selected, this field gets enabled. The value should be greater than 30. After the value entered, the Onset date will be adjusted accordingly.

  7. Month – If patient is unaware of the Onset date of high risk condition, you can enter the month. If the checkbox next to the Onset Date is selected, this field gets enabled. The value should be greater than 12. After the value entered, the Onset date will be adjusted accordingly.

  8. Year – If patient is unaware of the Onset date of high risk condition, you can enter the year. If the checkbox next to the Onset Date is selected, this field gets enabled. After the value entered, the Onset date will be adjusted accordingly.

  9. Close Date – The closing date of the high risk condition is selected using the calendar function. This date should not be less then the Onset date.

  10. Day – If patient is unaware of the Close date of high risk condition, you can enter the date. If the checkbox next to the Onset Date is selected, this field gets enabled. The value should be greater than 30. After the value entered, the Onset date will be adjusted accordingly.

  11. Month – If patient is unaware of the close date of high risk condition, you can enter the month. If the checkbox next to the Onset Date is selected, this field gets enabled. The value should be greater than 12. After the value entered, the Onset date will be adjusted accordingly.

  12. Year – If patient is unaware of the close date of high risk condition, you can enter the year. If the checkbox next to the Onset Date is selected, this field gets enabled. After the value entered, the Onset date will be adjusted accordingly.
