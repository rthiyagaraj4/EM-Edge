Enter the following details:

  1. Date [From – To] – Enter the WHONET Data Extraction [From-To] Date range release date in dd/mm/yyyy format or select the date range using the calendar function. The WHONET Data Extraction details are displayed for the selected date range. Current date range is displayed, by default.  

  2. File Name – Enter the File Name in alphanumeric format of 500 characters maximum length. The path at which the WHONET Data Extraction is saved is displayed, by default.

  3. [Process WHONET Data] – Click [Process WHONET Data] to display the WHONET Data Extraction details.

  4. [Error Log Tab] - Click [ERROR LOG] to display the errors occurred while WHONET Data Extraction.

  5. [WHONET DATA] - Click [WHONET] to display the data extracted while WHONET Data Extraction.
