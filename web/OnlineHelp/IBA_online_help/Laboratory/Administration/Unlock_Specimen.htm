Unlock Specimen

From this function you can Unlock the result entry for a specimen.

Unlock Specimen function is used to unlock the specimen when two users are accessing the same specimen in Result Entry By Specimen and Review functions.

Unlocking Specimen

From Administration, double-Unlock Specimen function.

The Unlock Specimen screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Execute Query] or press [F8] to display the specimen current user details with the date /time.

  3. Select the User to be unlocked.

  4. Click [Unlock] to unlock the user.

When you query the specimen number which is not accessed, in the unlock specimen function, the system displays the data with the user name, department details. You can unlock the specimen.
