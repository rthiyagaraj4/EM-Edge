Infection Alert

From this function you can Define an alert against the organisms causing infection in a ward or clinic specific to a facility. You can Query for the existing records.

Infection Alert function is used to identify the organisms that cause infection in a facility. You can also define the number of days during which the infection has prevailed. The number of isolates from the specific department during the infected period can also be recorded using this function. A report is generated and sent to the person in charge, on identifying infection as per the setup made in this function.

Defining Infection Alert

From Administration, double-click the Infection Alert function.

The Infection Alert screen gets displayed.

  1. The name of the logged in facility is displayed, by default. This is a mandatory field.

  2. Select the 'Source Type' from list. The listed values are 'Ward', 'Clinic' and 'External'. Click [List of Values] to select a Source Type. The values differ based on the Source Type selected. The listed values are created in 'Outpatient Management’ and ‘Inpatient and Daycare Management’ modules. This is a mandatory field.

  3. Enter the details in the mandatory (*) fields.<Related Field Description>

  4. Click [Commit] or press [F10] to save the created records or click [Clear Record] to clear the entered data.

  5. Click [Execute Query] or press [F8] to search for the existing records.
