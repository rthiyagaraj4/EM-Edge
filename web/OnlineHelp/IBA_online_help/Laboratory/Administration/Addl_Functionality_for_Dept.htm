Additional Functionality for Department

From this function you can Associate sample anatomy and section indicator to a Department. You can Query for the existing records.

Additional Functionality for Department function is used to associate additional indicators to a department. The values selected in the ‘Use Sample Anatomy Indicator’ list against the department reflects in specimen registration screen. The registration function allows you to add anatomy sample details.

Associating Indicators to a Department

From Administration, select the Additional Functionality for Department function.

The Additional Functionality for Department screen gets displayed.

  1. The departments in the respective facility get listed. The description of the department gets displayed in the adjacent field.

  2. Select the required value from the list. The list displays ‘Sample – Anatomy’ and ‘Anatomy’.

  3. Select the desired Section Indicator from the list. The system-defined values are ‘Blood Transfusion’, ‘Clinical Chemistry’, ‘Cytology’, ‘Hematology’, ‘Histopathology’, ‘Microbiology’, ‘Serology’ and ‘Virology’.

  4. Click [Commit] or press [F10] to save the records or click [Clear Record] to clear the entered data.

  5. Click [Execute Query] or press [F8] to search for the existing records.

Sample Anatomy is used only for Histopathology (Surgical) departments where we enter the sample details of the Anatomy of Biopsy. Eg: Appendix.

Anatomy is used for Cytology department where the specimen of the body is collected. Eg: FNAC.
