Patient Markers

From this function you can Query the patient markers of the patient.

The patient marker is used to query the tests which are marked and to view the details of the patient and the results of the marker tests. When you enter the patient number, the function lists out all the marker tests and their results, if available. In the Test Codes function, if ‘Patient Marker’ check box is selected; the test will be marked for specific tests.  For example, HIV Tests.

Querying Patient Marker

From Query > Miscellaneous, double-click the Patient Marker function.

The Patient Marker screen gets displayed.

  1. Enter the Patient ID or right-click on the ‘Right Click for User Options’ function to search for a patient record using the ‘Hospital Patient Lookup’ or the ‘External Patient Lookup’ screen. Double-click the Patient ID in the displayed list to select the patient record.

  2. On selecting the Patient ID, the following details are displayed.

  3. The patient identification number is displayed.

  4. Name of the patient corresponding to the Patient ID selected, is displayed .

  5. The Gender of Patient corresponding to the Patient ID selected, is displayed.

  6. The name of the Test Code is displayed.

  7. The description of the test Code is displayed.

  8. The numeric result of the test Is displayed.

  9. The result comment for result entered while resulting is displayed.
