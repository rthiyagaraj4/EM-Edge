Confidential Results

From this function you can Query the status of all the registered requests and their results (including Confidential Results) for a given patient number.

The function is used to query the status of all registered requests and their results (including Confidential Results) for a given patient number. Authorized personnel can use the query as per the setup in Result and Request module. Enter a valid patent number for which the requests and their test results are to be displayed. It is also possible to enter an order number and query the results.

Querying Confidential Results details

From Query > Patient Results, double-click the Confidential Results function.

The Confidential Results screen gets displayed.

  1. If the Order Number is available, enter the Order Number, for which the status has to be queried.

  2. Enter the Patient ID or right-click on the ‘Right Click for User Options’ function to search for a patient record using the ‘Hospital Patient Lookup’ or the ‘External Patient Lookup’ screen. Double-click the Patient ID in the displayed list to select the patient record.

  3. Click ‘Tab’ to view the patient details.

    On selecting the Patient ID, the following details are displayed.

    1. Patient Name – The patient name corresponding to the selected Patient ID, is displayed.

    2. Gender – The patient gender corresponding to the selected Patient ID, is displayed.

    3. Referral No – The Referral Number if any, corresponding to the selected Patient ID, is displayed.

    4. Specimen No – Specimen number is displayed. You can double click the Specimen number to view the Patient Result.

    5. Regd Date and Time – The Specimen Registration date and time is displayed.

    6. Test codes (1-5) – Test codes associated to the specimen are displayed.

    7. Ordered Facility ID – The name of the Ordering Facility is displayed.

    8. Perform Facility ID – The name of the Performing Facility is displayed.

    9. Source - Name of the source is displayed.

  4. Click [Patient Details] to view the patient details of the patient.

  5. Click [Print Report] to print the report. On clicking [Print Report] an authorization screen gets displayed. Enter the authorize password to print the report.

  6. Click [Result] to view the results for the tests performed. If the results have been printed, the status is displayed as 'P'. If the result has only been processed and not printed, the status is displayed as 'R'.

  7. Click [View Confidential Tests] to view the confidential tests of the patient.

  8. Click [Specimen] to view specimen registration details.

  9. Click [More Tests] to view all the tests requested for the specimen.

    All the requests booked for the patient are displayed in reverse chronological order, that is, the latest request is displayed at the top and the first request at the bottom. The number of requests booked may be more and can be viewed by using the vertical scroll bar. For each request the specimen number, date/time of request registration, the tests requested and the source from where the request originated is displayed. Completed and outstanding requests are displayed in unique colour indicators. Click [More Tests] to view all the tests requested for the specimen, if the request has more tests to be displayed.

  10. Click [Transfusion History] to view the transfusion history of the patient.

    The following details are displayed:

    1. Unit Number/Practitioner – The unit number of the practitioner is displayed.

    2. Product Code/Start Time – The product code or start time of the transfusion is displayed.

    3. Description/End Time – The description of the product code and the end time of transfusion is displayed.

    4. Blood Grp – The blood group of the patient is displayed.

    5. Rhesus – The rhesus name of the patient is displayed.

    6. Specimen No/Volume –  The specimen number and the volume of the unit is displayed.

    7. Units – The name of the unit is displayed.

    8. Specimen Date/Reaction – The specimen and reaction date of the transfusion is displayed.

    9. Reserved By – The name of the reserved person is displayed.

    10. Print – Click [Print] to print the transfusion details of the patient.

  11. Click [View Clinical History] to view the clinical history of the patient.

  12. Click [Clinical Comments] to view the clinical comments of the patient.

  13. Click [Clear] to clear the entered details.
