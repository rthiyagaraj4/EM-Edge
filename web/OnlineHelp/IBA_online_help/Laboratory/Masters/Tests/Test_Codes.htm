Test Codes

From this function you can Define various individual tests or investigations specific to a department. You can Query for the existing records.

The tests defined are performed in the laboratory by an authorized user of a specific department by associating attributes, validations, order catalogs, comments, age/sex ranges, and perform sites. The system displays the department to which the logged in user has access rights.

Defining Test Codes
