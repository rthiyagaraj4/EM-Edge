Group Tests

From this function you can Define various Group Tests specific to a department. You can Query for the existing records.

A panel test or a Group Test can have two or more tests defined in ‘Test Codes’ function. You can define and modify the details associated to the Group Test code. By associating Individual Tests, Specimen Type, Order Setup, Reflex Test, Perform Sites, Anatomy Site and Volume.

Defining Group Tests

From Masters > Tests, double-click the Group Tests function.

The Group Tests screen gets displayed.

  1. Enter the data in the mandatory fields. <Related Field Description>

  2. The screen consists of the following tabs:

Individual Tests

Specimen Type  

Order Setup  

Reflex Test

Perform Sites  

Anatomy Site  

Volume  

  1. Click [Commit] or press [F10] to save the created records or click [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for existing records.
