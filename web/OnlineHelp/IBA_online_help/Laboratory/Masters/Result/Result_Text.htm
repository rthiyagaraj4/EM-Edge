Result Text

From this function you can Define the Result Text specific to a department to assist you while processing specimen results. You can Query for the existing records.

Result Test function allows you to work with the data of a specific laboratory department to which you are assigned. This department is displayed and can be modified if the user belongs to more than one department.

Defining Result Text

From the Masters > Result, double-click Result Text function.

The Result Text screen gets displayed.

  1. Enter the data in the mandatory (*) fields. <Related Field Description>

  2. The Result Text tab is selected by default. Enter the required result text.

  3. Click SNOMED Pair tab to record the details.

    Enter the following details:

    1. SNOMED Code1 – Click [List of Values] to select a SNOMED code. The listed values are created in Nomenclature (SNOMED) masters.

    2. Description – The corresponding description for the selected SNOMED code is displayed, by default.

    3. SNOMED Code 2 – Click [List of Values] to select a SNOMED code. The listed values are created in Nomenclature (SNOMED) masters.

    4. Description - The description corresponding to the selected SNOMED code is displayed, by default.

  4. Click [Commit] or press [F10] to save the records or click [Clear Record] to clear the entered data.

  5. Click [Execute Query] or press [F8] to search for existing records.
