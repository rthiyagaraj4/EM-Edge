Result Notify/BB Comments

From this function, you can Create set of notifications/blood bank comment codes for the specimen. The created notification codes can be used in Result Entry by Specimen and Reject Specimen/Tests functions.

This function allows you to create notification codes and its description for department wise, which will be used in the Result Entry by Specimen and Reject Specimen/Tests functions. These codes can also be Suspended or later activated in the Suspend/Reinstate master. The created blood bank comment codes can be used in BT module.

Creating Result Notify/BB Comments

From the Masters > Result, double-click Result Notify/BB Comments function.

The Result Notify/BB Comments screen gets displayed.

  1. Enter the data in the mandatory (*) fields. <Related Field Description>

  2. Click [Save] or press [F10] to save the records or click [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for the existing records.

If you try to delete the created notification code, an error message ‘Deletes are not allowed in this screen’ appear.
