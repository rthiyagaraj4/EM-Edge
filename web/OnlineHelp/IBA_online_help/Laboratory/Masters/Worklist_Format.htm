Worklist Format

From this function you can Define a work list by associating one or more tests specific to a department. You can Query for the existing records.

You can enter the tests that need to be included in the worklist format. The criteria for including the specimen on the worklist are based on whether a test is to be performed on a particular specimen. If a test is defined as mandatory and the specimen does not have the corresponding test, though it may have the other tests of the worklist, the specimen will not be included in the worklist. If tests have been defined as optional and the specimen has any one or more of the tests, then the specimen is included in the worklist. The maximum number of investigations or tests that can be defined in a single worklist format is 25. It is mandatory to specify the number of cups required for a worklist.

Defining Worklist Format

From the Masters, double-click the Worklist Format function.

The Worklist Format screen gets displayed.

  1. Enter the data in the mandatory (*) fields. <Related Field Description>

  2. Click [Arrange Cups] to define the cup details. Click [Clear] to clear the cups specified or [Close] to close the cup details screen. <Related Field Description>

  3. Click [Setup Print Format] to customize the print layout of the worklist or [Close] to close the screen. You can view the worklist format by clicking [View Format]. A pop-up appears on clicking [View Format] indicating that the process is successful. Click [OK] to view the format or [Cancel] to cancel the process.

  4. Click [Commit] or press [F10] to save the created records or click [Clear Record] to clear the entered data.

  5. Click [Execute Query] or press [F8] to search for the existing records.
