Source Name on Specimen Label

From this function you can Create a Source Name to be printed on the specimen label specific to a source type. You can Query for the existing records.

In this function, you can select a desired source type to define specimen labels. These source names on specimen labels are used to identify the source from which specimens are received.

Creating Source Name to be printed on Specimen Label

From the Masters, select the Source Name on Specimen Label function.

The Source Name on Specimen Label screen gets displayed.

  1. Enter the data in the mandatory (*) fields. <Related Field Description>

  2. Click [Save] or press [F10] to save the records or click [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for existing records.
