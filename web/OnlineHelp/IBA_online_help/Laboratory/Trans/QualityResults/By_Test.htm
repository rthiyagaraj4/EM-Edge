By Test

From this function you can Record the details for entering the result for quality control materials.

Quality Test Results function is used to record the results for quality control material. The quality results are used for the quality validations (quality sample test) on an analyzer periodically applicable for the selected test. The quality results are validated based on the inbuilt in Enterprise Management.

Recording Quality Test Results

From Laboratory Transactions > Quality Results menu, double-click the By Test function.

The Quality Test Results screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Rules Chart] to view the Rule Details defined for validating the quality of the Specimen as per the standards set in ‘Quality Codes’ function in the masters.

    You can also add the Rule Details to the defined Rules Chart.

    1. No – Enter the Serial Number in alphanumeric format of four characters maximum length.

    2. Rule – Enter the Rule in alphanumeric format of four characters maximum length.

    3. Rule Description – Enter the Rule Description in alphanumeric format of 500 characters maximum length.

    Click [OK] to exit from the ‘Rules Chart’ screen.

  3. Click [Validate] to view the results of the validation done for the quality tests of the Specimen between the low and high value and the result value.

    • If ‘N’ is displayed, the quality test result is normal.

    • If ‘Y’ is displayed, the quality test result is failed.

    Click [Close] to exit from the ‘Validate’ screen.

     

  4. On entering the details, click [Save] to save the record or [Clear Record] to clear the entered details.

 

The failed tests are highlighted in red colour.
