Grossing

From this function you can Record the Grossing information during the dissection of the Histology and Cytology samples.

The samples received in Histopathology or Cytology department need to undergo grossing before the consultants study them and report the findings. The samples can be dissected into different blocks to enable the Technician to do the investigations. You can do multiple regrossing if required.

Recording Grossing

From Laboratory Transactions > Anatomic Pathology menu, double-click the Grossing function. The Grossing screen gets displayed.

  1. Select the Department from the list. The logged in Department Code is displayed, by default. Select required department from the list if you have access to more than one department.

  2. Enter the Specimen Number or click [List of Values] to select the Specimen Number to record the grossing details. The system displays a list of specimens for which the sample details are recorded. Press [Tab] to view the specimen details. <Related Field Description>

    The following details are displayed:

    1. Priority – The priority of the Specimen is displayed. The valid values are ‘All’, ‘Routine’, ‘Urgent’ and ‘Stat’.

    2. Assign To – Enter the ‘Assign To’ for which the sample consumption details are to be processed. Click [List of Values] to view and select the appropriate user name.

    3. Patient ID – The unique number assigned to the patient is displayed. The name of the patient is displayed in the adjacent filed.

    4. Sex – The patient’s gender is displayed as recoded during registration.

    5. Age – The patient’s age is displayed as recorded during registration.

  3. Enter the details in the following tab pages.

Text Tab

The Text tab frame is displayed, by default. The current system date and time is displayed as grossing date and time. Click [List of Values] to select the required text code to record the textual information as a part of grossing. The list displays the values defined in Result Text masters function.

 

You can do multiple regrossing. The system shows multiple Regrossing Date time field in the detail information section. You can enter multiple re-grossing time and re-grossing text for a single sample.

 

Enter the following details:

  1. Grossing Date Time - Select the grossing date and time using Calendar function.  

  2. Grossing Done By – Displays the logged in user name who has done grossing. You cannot edit the data.

  3. Re-grossing Date Time - Select the re-grossing date and time using Calendar function. You can select the re-grossing date and time, if the grossing is already done for the selected specimen.

  4. ReGrossing Done By – Displays the logged in user name who has done regrossing. You cannot edit the data.

  5. Code - Enter the code or select it using the [List of Values] function. The codes created in 'Result Text' Master are displayed.

  6. Performed By – Select the user name who has performed Grossing or Re-grossing using lookup function.

Blocks Tab

Click ‘Blocks’ tab to invoke the tab frame.

Enter the following details:

  1. Block ID - Serial number for the Block ID is displayed on clicking [Create Block].

  2. Number of Tissues - Enter the Number of Tissue for the block depending on the specimen grossing.

  3. Description - Enter tissue description in the free text.

  4. [Create Block] - Click [Create Block] to create different blocks to enable the Technician to do the investigations. Enter the number of tissue and description of the tissue.

  5. [Delete Block] - Click [Delete Block] to delete the block in case it was wrongly created.  On clicking Delete Block button, a confirmation message appears. Click [Confirm] to delete the block.

  6. The deleted blocks are highlighted in grey background. You cannot select or perform any action on those blocks.

You cannot delete the block when the sample is consumed fully or when task assigned to the block is started or completed.

  1. Click [Requesting Task] tab to invoke Requesting Task screen to request tasks that should be performed before the consultants study them and report the findings.

  2. Click [Patient Details] to view the details of the Patient.

  3. Click [Add Clinical History] to add the details of the clinical history of the patients. These clinical history codes are defined in the Clinical History master function. Click [Close] to return to Sample Details screen.

The Order Comments button is enabled if recorded order comments exist for the selected specimen.

  1. Click [Clinical Comments] to view the recorded clinical history and clinical comments details of the patient. Click [Close] to return to the Sample Details screen.

  2. Click [Patient Details] to view the complete information of the patient demographic.

  3. Click [Order Comments] to view the Order details. This button will get enabled, if the order is placed from the Clinician Access else it will be disabled.

  4. The Sample ID is displayed. The Sample ID is generated in Sample Details function.

  5. Select the ‘Sample Consumed?’ check box to indicate that the selected sample is consumed.

  6. Select the ‘Regrossing?’ check box to indicate re-grossing is required for the selected specimen.

You can perform Re-grossing if at least one task is completed for the selected specimen.

  1. On entering the details, click [Save] to record the grossing details or [Clear Record] to clear the entered details.
