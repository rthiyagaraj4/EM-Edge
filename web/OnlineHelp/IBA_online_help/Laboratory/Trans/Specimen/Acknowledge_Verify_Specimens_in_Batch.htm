Acknowledge/Verify Specimens in Batch

From this function you can Verify and Acknowledge the registered specimens in Batch.

This function allows you to acknowledge/verify the specimen by batch. This function is applicable for Laboratory, BD and BT modules. Based on the setup in the Authorized Users, the User will be allowed to verify the requests for a Department.

Acknowledge/Verify Specimens in Batch

From Laboratory > Transactions > Specimen menu, select the Acknowledge/Verify Specimens in Batch function.

The Acknowledge/Verify Specimens in Batch screen gets displayed.

  1. By default, the logged in user department and the list of specimens to be verified for that department gets displayed.  If a department is suspended, it will not get listed in the list.

  2. Scan the barcode of the specimen number and the specimen number gets displayed in the Specimen Number field. If you enter the specimen number manually, on tabbing out the system prompts an error message 'Enter Category code first' in case if the department has the 'Usage of Category No' check box selected in Departmental Parameter for Facility screen.

  3. Once the specimen barcode is scanned, the system selects the 'Verify Y/N' checkbox by default. For a specific specimen number, if the Category code and number is not present, then the system clears the 'Verify Y/N checkbox'.

  4. The Priority check box is displayed next to the 'Verify Y/N column'. When you select the Priority check box, the system considers it as a priority specimen.

  5. Select the Assign To name from the LOV. Only the user who are Pathologists or Lab technicians and are linked to the particular department are allowed for selection. This field is optional.

  6. The Category Number gets generated automatically by the system when ‘Generate Category No’ check box is selected in Departmental Parameter for Facility screen. Otherwise, you need to enter the Category number manually. First, you need to enter the Category Code, Category Year and Category number.

 

• If the category number is entered manually, the system checks for duplicates for a given combination of Category code, category year and category number.

• For automatic generation of category number, a sequence mentioned in the Category code master setup is followed.

• The Category field will not be displayed for departments where the ‘Usage of Category No’ check box is not selected in the Departmental Parameter for Facility screen.

  1. On clicking any record, the tests placed for the specimen gets displayed at the bottom of the screen.

  2. Click [Specimen] button to invoke the specimen Registration screen where you can add new tests if required and verify the specimen. If additional tests are added in the Registration screen, it gets displayed in the Test field under Investigations legend.

  3. Select the Verify Y/N check box against the record or select the 'Verify All' check box if you want to verify all the records at a time. Click [Verify] button to verify the specimens. The system prints the barcode after verifying.

To enable the Category Code in Acknowledge/Verify function, create a Category Codes in Masters and in Administration setup under Departmental Parameter for Facility select ‘Usage of Category No’ check box.
