Edit Work List

From this function you can Modify the worklists created using the Create Worklist function.

Edit work list includes additional specimens to the worklist, which were not available at the time of worklist creation. The specimens added to the worklist are evaluated against the Worklist Format defined. A Specimen or a test within a specimen, which is cancelled after creating of the worklist, are displayed in the worklist in a different color code. On creating the worklist, the specimen added to the worklist is deleted from the individual specimen list and is displayed in the worklist with a different color code.

Editing a Work List

From Laboratory Transactions > Worklist menu, double-click the Edit Worklist function.

The Edit Worklist screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [Create Empty Worklist] to invoke the Create Empty Worklist screen to add more worklist.

    Enter the following details:

    1. Worklist Name - Enter a valid worklist name, which has already been created or click [List of Values] function to view and select a valid worklist name.

    2. Worklist Date - Enter the work list date or click [List of Values] function to select a valid date.

    3. Worklist No - Enter a worklist number or click [List of Values] function to view and select a valid worklist number already created.

    4. Analyzer Code - Enter the Analyzer Code or select it using [List of Values] function to select from the list. The list displays the values defined in Analyzers masters function of this module.

    5. Perform Site - Enter the Perform Site code or select it using [List of Values] function. Listed values are created in 'Perform Site' function in Masters.

    6. [Create] - Click [Create] to create the worklist

    7. [Close] – Click [Close] to close the 'Create Empty Worklist' screen.
