Medication Administration Rights

From this function you can Define rights to administer the Medication Administration function.

This function is used to define rights to a user to administer the Medication Administration function. It provides right to a user to cancel the administered drug in Medication Administration function.

Defining Medication Administration Rights

From the PH Code Setup > PH Other Admin Setup, select the Medication Administration Rights function.

The Medication Administration Rights screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the User name from lookup function. This field is mandatory field.

  3. Once the user is selected, the Medication Administration Rights Type grid appear.

  4. For all users the ‘Cancel Admin’ check box is not selected, and other Medication Administration Rights types check boxes will be selected and in disable mode. You can select/clear the Cancel Admin check box as required.

  5. Click [Apply] to save the Medication Administration Rights defined or [Reset] to clear the entered data.

  6. Click [Query] to search for an existing Medication Administration Rights.
