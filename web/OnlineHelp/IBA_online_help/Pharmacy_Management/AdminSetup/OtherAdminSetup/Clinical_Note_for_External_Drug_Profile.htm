Clinical Note for External Drug Profile

From this function you can Associate the note type to be linked with the clinical note in external drug profile for each drug.

This function allows to link the clinical note in external drug profile for each drug.

Linking Clinical Note for external drug profile for each drug

From the PH Admin Setup > PH Other Admin Setup > select the Clinical Note for External Drug Profile function.

The Clinical Note for External Drug Profile screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select Note Group using lookup function. The Note Groups defined in the masters are listed.

  3. Click [Search] to display Note Type Code, Note Type Description, Class, Enabled, Confidential details based on selected Note Group. In Enabled column,is displayed against enabled Note Type and is displayed against disabled Note Type. Select the checkbox against Note Type to be displayed in Record Clinical Notes in External Drug Profile screen.

  4. Click [Apply] to save the details or [Reset] to clear the entered data.
