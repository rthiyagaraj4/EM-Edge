Order Routing for Billing Group

From this function you can route the order based on the billing group and also define the Alternate Dispensing location.

Routing an Order for Billing Group

From the PH Admin Setup > PH Other Admin Setup, select the Order Routing for Billing Group function.

The Order Routing for Billing Group screen is displayed.

  1. Click [Create] to invoke the Create screen; the Ordering Facility is displayed, by default, with the logged in facility.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the defined Order Routing for billing group or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing Order Routing for billing group defined.
