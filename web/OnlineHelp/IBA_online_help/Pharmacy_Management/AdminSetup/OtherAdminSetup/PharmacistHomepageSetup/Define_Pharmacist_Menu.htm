Define Pharmacist Menu

From this function you can Define desktop menu for the logged in pharmacist. You can Query to view the Defined Pharmacist menu.

The Define Pharmacist Menu function allows to link desktop IDs created in ‘Create Desktop User’ screen with user IDs.

Defining Pharmacist Menu

From the PH Admin Setup > PH Other Admin Setup > Pharmacist Homepage Setup, select the Define Pharmacist Menu function.

The Define Pharmacist Menu screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the define pharmacist menu or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing pharmacist menu record.
