Formulary Billing Group

From this function you can Create formulary (drug and medical item) and tag it to a billing group.

This function is used to create the formulary (drug and medical item) and tag it to a billing group. The system allows to add one formulary with multiple billing. But one billing group can be mapped to only one formulary.

While placing Pharmacy Orders based on the patient billing group the system will show the applicable formulary drugs to prescribe.

Creating Formulary Billing Group

From the PH Code Setup > PH Other Admin Setup, select the Formulary Billing Group function.

The Formulary Billing Group screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the search criteria in the respective fields.

    Enter the search criteria:

    1. Formulary Code –  Enter the formulary code in alphanumeric format of 20 characters maximum length. This field is mandatory. On tab out, the system checks whether the formulary code is duplicate or not.

    2. Formulary Type – Select the Formulary Type value from the drop-down list. The listed values are Cash and Credit. By default, the value ‘Cash’ is displayed.

    3. Formulary Description – Enter the formulary description in alphanumeric format of 60 characters maximum length. This field is mandatory.

    4. Item Type -  Select the item type from the drop-down list. The listed values are Drug and Medical Item. By default, the value ‘Drug’ gets displayed.

    5. When the ‘Item type’ value is selected as ‘Drug’, the fields Generic name, Form and Strength are displayed.  

    6. When the Item Type is selected as ‘Medical Item’, the field ‘Item Technical Name’ gets displayed.

    1. Generic Name – Select the generic name from the lookup.

    2. Form - Select the Form of the drug from the drop-down list. This field is mandatory.

    3. Strength – Select the drug strength from the drop-down list. This field is mandatory.

    4. Billing Group – All the Billing Groups are displayed in the list box. Select the required billing group. For new formulary, the system disables the used billing group check box.

    5. By default, all billing groups are displayed in Alphabetical Order. If you type ‘A’ then system displays all the billing group which contains ‘A’. You can check the ‘A’ related billing group and can select any other also.

    6. For new formulary, system will disable the used billing group check box. If a formulary is disabled, then system automatically releases the associated billing group and it can be used for another formulary.

    1. Item Technical Name – Select the item technical name from the drop-down list.

    2. Enabled – Select the check box to enable the created formulary code. By default, the check box is selected.

  3. Click [Search]. The Drug Search webpage dialog gets displayed.

  4. Select the Preference Item from the drop-down list. The drop-down list includes Preference 1, Preference 2 and Preference 3.

  • On selection of preference, the system will check whether the same preference is selected or not. If the same preference is used, an alert will be displayed. You cannot add duplicate preference item in the list.

  • For Cash, at least 3 preferences to be selected and for credit at least 1 preference to be selected.

  1. After selection, click [Add]. The drug details such as Drug Name, Form, Strength, Admin Route, Unit Price, Preferred Item are displayed.

  2. Click [Apply] button, added drug/medical item gets saved for the formulary.

  3. Once any item is added for the formulary, the Formulary Type field gets disabled so that you cannot change the formulary type. You can modify by clicking Generic Name/Item Technical Name hyperlink.

  4. Click [Print] to invoke the Formulary Details Report dialog for printing.

  5. Click [Query] to search for an existing formulary Billing Group.
