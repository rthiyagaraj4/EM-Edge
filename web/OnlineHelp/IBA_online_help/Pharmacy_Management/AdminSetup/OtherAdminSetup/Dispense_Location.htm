Dispense Location

From this function you can Create Dispense Locations of various types and Query for the desired Dispense Location.

Dispensing Locations are of the type ‘Pharmacy’, ‘Clinic’ and ‘Nursing Unit’. Option to link Dispense Locations with existing Stores is provided. A ‘Pharmacy’ type Dispensing Location can cater to Outpatients/Inpatients, Outpatients only or Inpatients only. Drug Classes dispensed by all these locations are specified. The Dispensing Locations defined in the function are specific to the logged in facility. Rules specified for Pharmacy Type locations catering to outpatients are: applicability of various dispensing stages, generation of outputs, drug return rules, rules to handle undelivered drugs and printer locations for orders placed. Inpatient rules defined are applicability of various dispensing stages and printer locations. The dispensing rules defined in the function ‘PH Parameter for Facility’ is inherited by location. These default rules are modified to suit the location requirements. But, the default values cannot be exceeded.

Creating a Dispense Location

From the PH Admin Setup > PH Other Admin Setup, select the Dispense Location function.

The Dispense Location screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Dispense Location created or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing Dispense Location.
