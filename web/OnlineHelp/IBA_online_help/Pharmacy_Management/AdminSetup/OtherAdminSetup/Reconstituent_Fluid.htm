Reconstituent Fluid

From this function you can Define various Re-constituent Fluids, associate Inventory Items and Query for desired Re-constituent Fluid.

Re-constituent Fluids are used with drugs for preparing Admixtures.

Defining Reconstituent Fluids, Associating Inventory Items

From the PH Code Setup > PH Other Admin Setup, select the Reconstituent Fluid function.

The Re-constituent Fluid screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Reconstituent Fluid and Add/Modify Inventory Items details in the mandatory (*) fields.

    1. Item Code - Enter Item Code in alphanumeric format of 20 characters maximum length or select the required Item Code using [?] function. This is a mandatory field.

    2. Content in Base Unit – If the selected Item has equivalent Unit of Measurement defined in terms of Re-constituent Fluid; Base Unit and Conversion Factor defined for the equivalent is displayed and disabled. The conversion factor is defined in ‘Equivalent Unit of Measurement’ function in Application Masters module.

    1. Fluid ID – Enter the ID in alphanumeric format of 15 characters maximum length. This is a mandatory field.

    2. Fluid Name – Enter the Fluid Name in alphanumeric format of 60 characters maximum length. This is a mandatory field.

    3. Base Unit – Select the Base Unit of Measurement from the list. The listed values are defined in ‘Application Masters’ UOM function. This is a mandatory field.

  3. Click [Add] to add the Inventory Items or [Clear] to clear the entered data.

  4. Click Item hyperlink to modify the Inventory Items details.

  5. Click [Modify] to modify the changes or [Delete] to delete the record.

  6. Click [Apply] to save the record or [Reset] to clear the entered data.

  7. Click [Query] to search for an existing Reconstituent Fluid.
