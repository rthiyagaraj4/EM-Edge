Quota Limit Query

From this function you can Query and View all ordered drugs for which quota limit is defined.

This function is used to query and view all ordered drugs for which quota limit is defined with the current order status. On querying, the list of patient and order details for which quota limit approval has been accepted will be displayed.

Querying Drug Quota Limit

From the PH Queries > select the Quota Limit Query function.

The Drug Limit Query screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to view records matching the search criteria.

  3. The query result details include Patient ID, Patient Details, Drug Name, Order Status, Prescription No, Order Date/time, and Specialty.

  4. Click [Reset] to clear the specified criteria.
