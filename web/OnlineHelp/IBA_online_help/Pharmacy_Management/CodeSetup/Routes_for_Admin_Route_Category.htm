Routes for Admin Route Category

From this function you can Associate Admin Routes to a Admin Route Category.

The Administration Routes defined in the Admin Route function can be associated to the defined Admin Route Category.

Association of Admin Routes to Category

From the PH Code Setup > PH Code Setup, select the Routes for Admin Route Category function.

The Routes for Admin Route Category screen gets displayed.

  1. Enter the Admin Route Category or select the Admin Route Category using [?] function. The Admin Route Categories defined in the Admin Route Category function are displayed. This is a mandatory field.

  2. Enter the search test in the Admin Route Description field and select the 'Search Criteria' from the list. The listed values are 'Starts With', 'Contains' and 'Ends With'.

  3. Click 'Search' to display the Admin Route values.

  4. English alphabets from 'A to Z' are displayed along with 'Others' and hyperlinked.  Click the required alphabet hyperlink to associate Administration Routes of that particular alphabet to the Admin Route Category. Click ‘Others’ hyperlink to select and associate Administration Routes beginning with numeric or special characters.

  5. Select the check box against the Administration Routes to associate it to the Admin Route Category.

  6. Click Associated hyperlink to view the Administration Routes associated to the Admin Route Category. Clear the check box to dissociate relevant Administration Routes.

  7. Click [Apply] to associate or dissociate the Administration Routes to the Admin Route Category or [Reset] to clear the entered data.
