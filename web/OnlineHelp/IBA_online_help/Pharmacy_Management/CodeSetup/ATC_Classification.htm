ATC Classification

From this function you can Define Anatomical, Therapeutic, Chemical classification and their sub class classification of a drug and Modify or Delete the desired classes.

The Anatomical Therapeutic Chemical Classification System is used for the classification of drugs. In the ATC classification system, drugs are divided into different groups according to the organ or system on which they act and their chemical and therapeutic properties. Drugs are classified into groups at five different levels.

An Anatomical Class consists of a Therapeutic Main Class; the Therapeutic Main Class has the Therapeutic Sub Class; Therapeutic Sub Class has the Chemical Main Class and the Chemical Main Class has the Chemical Sub Class grouped under it.

Defining an ATC Classification

From the PH Code Setup > PH Code Setup, select the ATC Classification function.

The ATC Classification screen gets displayed.

  1. The ATC Classification screen displays the Anatomical Class details, by default.
