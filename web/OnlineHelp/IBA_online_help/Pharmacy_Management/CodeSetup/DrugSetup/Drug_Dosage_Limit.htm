Drug Dosage Limit

From this function you can Create and maintain Dosage Limits for a Drug and Query for the desired Dosage Limit record.

The Drug Dosage Limit is to alert a Practitioner against prescribing an over dose. Setting up of dosage limits can help in preventing Drug Over Dose or Medication error.

The Drug Dosage Limit specified influence the prescription function. The usage of Drug Dosage Limit details are recorded optionally. An appropriate Age Group must be created using the function ‘Age Group’ in Application Masters module. It is essential to have the drug details created using the function ‘Drug’ to which the dosage limit control can be established.

Creating a Drug Dosage Limit

From the PH Code Setup > PH Drug Setup, select the Drug Dosage Limit function.

The Drug Dosage Limit screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Drug Dosage Limit created.

  4. Click [Query] to search for an existing Drug Dosage Limit.

  5. Click [Delete] to delete the queried records.
