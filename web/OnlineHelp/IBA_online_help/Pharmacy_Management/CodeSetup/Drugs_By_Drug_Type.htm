Drugs by Drug Type

From this function you can Create an association of Drugs to Drug Type and Query for the desired association of Drug to Drug Types.

Drugs are associated with the Drug Types. For example, Amoxicillin is a Drug, which is an Antibiotic. This drug is grouped with the Antibiotic group using this function.

Creating Drugs by Drug Type

From the PH Code Setup > PH Code Setup, select the Drugs by Drug Type function.

The Drugs by Drug Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Drug Name in alphanumeric format of 60 characters maximum length or select the desired Drug Name using [?] function.

  3. On selecting the Drug Name, the Drug Types are displayed.

  4. Select the check box against the desired Drug Type to associate the Drug to Drug Type. The check boxes are cleared, by default. If the Drug Type is already associated to the selected Drug, the check box is selected, by default.

  5. Click [Apply] to save the association of Drug to Drug Types or [Reset] to clear the entered data.

  6. Click [Query] to search for an existing association of Drugs to Drug Type.
