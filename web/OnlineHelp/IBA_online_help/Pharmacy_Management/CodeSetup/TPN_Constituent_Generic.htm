TPN Constituent Generic

From this function you can Query for the TPN Constituent Generic and Modify the queried record.

TPN Constituent Generic provides details on generics that form TPN Constituent groups. The displayed details are Generic Names, Display Order and UOM Flag. These Generic Names are maintained internally and used during Medication Ordering. You can modify or change the Generic Names. In the Medication Order function, these constituents are included as part of the Non-Standard Regimen.

Querying the TPN Constituent Generic

From the PH Code Setup > PH TPN Setup, select the TPN Constituent Generic function.

The TPN Constituent Generic screen gets displayed.

  1. Click [Query] to invoke the Query screen.

  2. Click Constituent Generic Code hyperlink to modify the queried records.

  3. You can modify the Constituent Generic Name. Enter the Constituent Generic Name in alphanumeric format of 30 characters maximum length.

  4. Select the ‘Selection of Preferred Item Fluid Applicable’ check box to indicate whether the selection of preferred Item Fluid is applicable.

  5. Click [Apply] to save the changes on TPN Constituent Generic or [Reset] to clear the entered data.
