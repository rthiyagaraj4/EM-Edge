Label Text

From this function you can Define Label texts and Query for desired Label Text.

These Label Texts are attached to a Drug in the ‘Drug’ function. Label texts specific to cautions and special instructions are defined.

Defining the Label Text

From the PH Code Setup > PH Code Setup, select the Label Text function.

The Label Text screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the defined Label Text or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing Label Text.

Label Text can be specified in local language provided ‘Patient Instruction in Local Language’ is selected in PH Parameter for Facility.
