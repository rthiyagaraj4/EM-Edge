EDL Classification

From this function you can Create Essential Drug Groups and Non-Essential Drug Groups Query for the desired association of Drug to Drug Types.

The Essential List is grouped to National Essential Drug List and Non National Essential Drug List. The National Essential Drug List consists Group A, Group B, Group C, Group D and Group F.  The Group F consists two sub groups Group F1 and Group F2. The Non Essential List is grouped to National Non Essential Drug List and Non National Non Essential Drug List.

The EDL Classification is linked to a Drug while creating the Drug. While placing Pharmacy Orders an icon is displayed adjacent to the Drug Code indicating the Group to which the Drug belongs.

Creating EDL Classification

From the PH Code Setup > PH Code Setup, select the EDL Classification function.

The EDL Classification screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [Record] to record the details.

  3. Click Tree Structure hyperlink to modify the recorded details.

  4. Click [Modify] to modify the changes.

  5. Click [Reset] to reset the entered details.
