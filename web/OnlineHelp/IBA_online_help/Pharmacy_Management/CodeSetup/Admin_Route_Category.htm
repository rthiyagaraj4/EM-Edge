Admin Route Category

From this function you can Define Admin Route Category and Query for the existing records.

The Administration Routes defined in the Admin Route function can be associated to the defined Admin Route Category. You can search for a record in the Medication Administration details for a particular Admin Route Category.

Define Admin Route Category

From the PH Code Setup > PH Code Setup, select the Admin Route Category function.

The Admin Route Category screen gets displayed.

  1. Enter the Admin Route Category code in alphanumeric format of ten characters maximum length. This is a mandatory field.

  2. Enter the Admin Route Category description in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Select the ‘Enabled’ check box to enable the defined record.

  4. Click [Apply] to save the defined Admin Route Category or [Reset] to clear the entered data.

  5. Click [Query] to search for an existing Admin Route Category.
