Release Batches

From this function you can Release the allocated batches.

The Dispensing Location allocates drug stock from various batches before delivering them to patients. For various reasons, the reserved stock is released by the Pharmacy. On releasing the allocated batches, stock quantity is updated.

Releasing the Allocated Batches

From the PH Transactions, select the Release Batches function.

The Release Batches screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to display the allocated batch details.

  3. Click ‘Dispense No’ hyperlink to view the dispense details.

  4. Select the Reason from the list to release the allocated batch. The listed values are defined in ‘Reason Codes’ function in PH Code Setup.

  5. Select the check box to release the drugs/allocated batch of the drug. If the Payment Status is ‘Paid’, the Select check box is disabled.

  6. Click [Apply] to release the allocated batch; the Release Batches screen is closed and the Dispense Medication screen is refreshed.

On refreshing of the Dispense Medication screen, the following happen:

  • If any of the released drug(s) is part of the Order IDs, that drug(s) will not appear.

  • If all the drugs that are part of the Order IDs are released for the patient, the patients will be listed.

  • If there is no other patient or orders available for the stage, the following message is displayed

 “Query caused no records to be displayed” message.

  1. Click [Reset] to clear the entered data.
