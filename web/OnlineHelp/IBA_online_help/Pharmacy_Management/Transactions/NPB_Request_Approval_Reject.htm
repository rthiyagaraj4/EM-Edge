NPB Request Approval/Reject

From this function you can Approve or Reject the NPB drug requests. You can Query for the existing NPB requests.

This function allows authorized pharmacists to ‘Acknowledge’, ‘Approve’ or ‘Reject’ the request placed for a Named Patient Basis (NPB) drug. A NPB drug request can be approved only if there is a drug code available for the requested drug. If you try to approve a request without the drug code, a validation message is displayed.

Approving/Rejecting NPB Requests

From the PH Transactions, select the NPB Request Approval/Reject function.

The NPB Request Approval/Reject screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Refresh]. The NPB requests are displayed based on the search criteria.

  3. Click on ‘NPB Approve Form’ hyperlink to view the request form.

  4. Select the Valid Upto date by clicking the calendar function. This is a mandatory field.  

  5. Select the required value from the Approve/Reject drop-down list. The listed values are Approve, Acknowledge and Reject. Acknowledging the request is optional.

  6. Click on ‘Remarks’ hyperlink to enter the remarks for approving/rejecting/acknowledging the request in the ‘Pharmacist Remarks’ dialog.

  7. Select the ‘Select’ check box against the NPB request.

  8. Click [Apply] to approve/acknowledge/reject the request placed by the practitioner.

  9. Click [Query] to search for the existing requests.
