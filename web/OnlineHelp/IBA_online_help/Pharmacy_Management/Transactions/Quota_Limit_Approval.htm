Quota Limit Approval

From this function you can Query and View the orders pertaining to quota limit approval.

This function is used to query and view all orders pertaining to quota limit approval. The quota limit is considered based on the primary specialty of the ordering practitioner. New patient is admitted and a Doctor whose primary speciality is cardiology has ordered for Quota item and patient is referred to another doctor whose primary speciality is immunology. In this case, quota will get deducted from cardiology speciality.

 

Quota Limit approval Access is given to the practitioner/responsibility in the “Users for Order Review” setup in OR module.

 

Note: When the ‘Active Prescription Applicable for Quota Limit’ check box gets selected in PH Parameter for Facility, for an active prescription if the same drug is prescribed again, then the quota count will not be considered for the patient.

Approving Quota Limit for drugs

From the PH Transactions > select the Quota Limit Approval function.

The Quota Limit Approval screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search], the records matching the search criteria appear.

  3. In the search results, click the Order Catalog to View the order details in a popup window.

  4. Click the ‘Patient History’ hyper link to invoke the Clinical Event History dialog.  

  5. To approve an order, select the check box against the record and click [Approve] button. Once Quota Limit is ‘Approved’, the quota limit will be deducted and status of the order gets updated as ‘Ordered’.

  6. The status change is recorded in audit trail.
