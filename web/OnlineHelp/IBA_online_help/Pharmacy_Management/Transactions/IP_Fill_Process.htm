IP Fill Process

From this function you can Generate IP fill list for a Dispense Location.  

The fill list contains orders placed by the nursing units on the Dispense Location. The following characteristics of Dispense Location influence generation of fill list.

• Filling process: Filling period specified for the Dispense Location.

• Exclude new order placed within [] schedule start.

• Continuous/Intermittent Order Schedule start time.

During fill list generation, batches are auto allocated, if stocks are available and these details are defaulted during IP Filling Stage. Certain conditions are evaluated before generation of fill list. The following conditions are evaluated before generation of fill list (Routine fill list). For Incremental fill list generation, the conditions from one to eight holds good. But, orders with status as Outstanding or Verified are only considered.

• Patient must not be deceased

• Order Category must belong to Pharmacy

• Only Inpatients are considered

• Medication start date has to be in between schedule start date/time and schedule end date/time

Or

• Medication End Date has to be in between schedule start date/time and schedule end date/time

Or

• Schedule start date/time has to be in between medication start and end date

Or

• Schedule end date/time has to be in between medication start and end date.

• Admitted and transferred inpatients are considered.

• Only Routine Order priority is considered.

• Narcotics drugs are not considered for fill list generation.

• Order Date has to be less than or equal to current system date and time.

• Order Date/Time has to be less than (schedule start – exclude new orders)

• Orders with status as Outstanding, or Verified or partially dispensed are only considered.

• Dispense Location must be same.

Generating IP Fill List

From the PH Transactions, select the IP Fill Process function.

The IP Fill Process screen gets displayed.

  1. Enter the details in mandatory fields. <Related Field Description>

  2. Click [Generate Fill List] to generate Fill List and unique Fill ID.

IP Fill Process is applicable only for Inpatients.
