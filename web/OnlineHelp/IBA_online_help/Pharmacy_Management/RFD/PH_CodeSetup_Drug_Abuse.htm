Enter the following details:

  1. Patient ID – Select the patient ID from the lookup. This field is mandatory.

  2. Patient Name – Once the patient ID is selected, the patient name gets displayed.

  3. Restriction Type – Select the required restriction type from the drop-down list. There are two types of Restriction: Drug Class and Drug Name. By default, the ‘Drug Class’ is selected as Restriction Type.

  4. Drug Class – The ‘Drug Class’ field will be displayed If ‘Restriction Type’ is selected as Drug Class. Select the required drug class option from the drop-down list. The list values are General, Narcotics and Controlled.

  5. Drug Name – The Drug Name field is displayed if the ‘Restriction Type’ is selected as ‘Drug Name’. Select the drug to be restricted from being ordered for the patient from the lookup.

  6. Restrict Transaction – Select the check box to restrict the defined class or drug while ordering. By default, the check box will not be selected.

• If the Restrict Transaction check box is selected, then the defined class or drug cannot be ordered for a patient.

• If the Restrict Transaction check box is not selected, the system will allow to order the defined class or drug for a patient after the user has entered the override reason. The Override reason will be similar to existing override reasons.

  1. Action – Select the action type from the drop-down list. The listed values are Block, Unblock. User can select block  or unblock a particular drug/class. By default the value ‘Block’ will be displayed.

  • When user selects the action as ‘Unblock’, then the ‘Restrict Transaction’ check box will get disabled.

  • When a drug class or drug is blocked, then based on the 'Restrict Transaction' the ordering of drug will behave.

  • When a drug class or drug is unblocked, then the drug can be ordered without any restrictions.

  1. Remarks – Enter the remarks in the text box in alphanumeric format up to 100 characters. The entered remarks will be displayed in 'Drug Abuse Dtls' while ordering a blocked drug for the patient.
