Enter the following details to get the administration details:

  1. Patient Class - Select the Patient Class Option. The options are 'Inpatient', 'Daycare', 'Emergency', 'Outpatient'. The option 'Inpatient' is selected, by default.

  2. Location – Enter the Location based on the Patient Class selected or select the Location using [?] function. The Locations listed are based on Operator Station defined for the logged in user. The Operator Station is defined in ‘Application Masters’ module. This is a mandatory field.

  3. Administration Window – The range gets displayed as read-only text. Based on the value specified for ‘Record Administration Within [] hrs’ in PH Parameter for Facility these values are calculated. The system considers current system date/time and displays it in the second value of ‘Administration Window’. The first value is calculated by subtracting the ‘Record Administration Within [] hrs’ from the current system date/time.

  4. Administration Status – Select the status from the list. The listed values are ‘Due for Administration’ and ‘All’.

 

Due for Administration

The details displayed correspond to medications that are due for administration.

All

Details displayed correspond to medication administration based on the ‘From’ and ‘To’ values. The ‘Record Administration within [] hrs’ is not considered while displaying the result set. All the details like Orders Held, Discontinued, Administered, Not Administered and Verbal Orders are displayed.

  1. Display Order By - Select the value from the list. The available values are Drug Name, Dosage Form and Scheduled Date/Time. The value Scheduled Date/Time is displayed, by default.

  2. Bed no. [From-To] – Select the bed number from the list. Specify the range. The listed beds correspond to the selected Nursing Unit. Based on the location selected, the Bed From and To fields are populated.

  3. Admin Route Category - Select the value from the list. The listed values are defined in the 'Admin Route Category' function in PH Code Setup. On selecting the Admin Route Category, the administration routes associated to the selected category in 'Routes for Admin Route Category' are listed for selection.

  4. Route of Administration - Select the Route of Administration value from the list to filter the Medication Order by Route of Administration. The available values are As directed and Dilute before use.

  5. Include -  Select the Value from the list. The listed values are 'All', 'Rx', 'Compounding Rx', 'IV Rx', 'PRN', 'Stat', 'TPN' and 'Oncology'. The value 'All' is displayed by default.

  6. Display Held/Discontinued Orders - Select the check box to display orders that are held/discontinued. These orders are indicated using a unique color-code.

The balance administration dose in MAR is displayed on user editing the PRN order. An alert message with prescribed details and administered dosage is displayed on selecting the ‘Select’ check box.
