Deliver

The dispensing stage is selected as Delivery. The dispensing medication screen gets displayed depending on the Category selected. This section provides information on delivery to:

  1. Outpatient

  2. Inpatient

• With Fill List

• Without Fill List.

Dispensing the Order

  1. Click [Refresh] to display the Orders for delivery based on the Category selected and the entered criteria.

  2. Click [Deliver] to invoke the Dispense Details screen.

    The receiver and dispensing Pharmacist details get recorded before completing delivery process. Enter the following details:

    Receiver Details

    1. Received By – Enter name of the person who receive the order. This field is mandatory. The patient name gets displayed, by default.

    2. Gender - Select the Gender from the list. This field is mandatory. The listed values are Unknown, Male and Female. The patient’s gender gets displayed, by default.

    3. Nationality - Select the Nationality from the list. This field is mandatory. The patient’s Nationality gets displayed, by default.

     

    Dispensing Pharmacist Details

    1. Dispensed By – The dispensing Pharmacist name gets displayed.

    2. Date/Time - The system Date and Time gets displayed. It indicates delivery Date and Time.

    3. Received At - Select the value from the list. The listed values are ‘Collected at Pharmacy’ and ‘Delivered at the Bed side’.

    4. Counter Check by - If ‘Collect at Pharmacy’ is selected as ‘Received At’, enter the counter checking nursing name, or use the lookup function to select the nurse. If ‘Delivered at the bed side’ is selected, enter either Pharmacist’s or nurse’s name. This field is mandatory for ‘Delivered at the bed side’.

     

    Click [OK] to confirm entered values or click [Cancel] not to confirm the action. The report screen gets displayed with two tabs, namely, ‘Print Reports’ and ‘Balance Medication Detail’. The ‘Balance Medication Detail’ tab gets displayed only when drugs are partially dispensed to the selected patient. The ‘Balance Medication Detail’ tab is selected, by default. Pharmacist directs the patient to the appropriate Pharmacy to collect the balance drugs.

    Enter the following details:

    1. Balance Medication Sheet – Select the check box to print balance medication sheet.

    2. Next Collection Detail – Select either ‘Current Facility’ or ‘External Facility’.

    Current Facility Details

    If the current facility is selected as ‘Next Collection Detail’, the following details are recorded. The Pharmacist directs the patient to collect the balance medication from the specified Pharmacy on the specified date.

    1. Next Collection Date – Enter the date in date format of dd/mm/yyyy or use the calendar function to select the required date. This field is mandatory.

    2. Next Collection From – Select the Dispense Location to collect the balance drugs from the list. All the Dispense Locations of the logged in facility are listed. This field is mandatory.

    External Facility Detail

    If current facility is selected as ‘External Facility’, the following details are recorded. Pharmacist directs the patient to collect the balance medication from the external facility on the specified date.

    1. Facility – Select the Facility name from the list. All the facilities of the enterprise are listed. This field is mandatory.

    2. Other Facility - To direct the patient to an external facility outside the enterprise, select the external facility from the list.

    3. Next Collection Date – Enter the date in date format of dd/mm/yyyy or use the calendar function to select the required date. This field is mandatory.

    4. Remarks – Enter the remarks in alphanumeric format. This field is mandatory.

    5. Referred By – Enter the name of the person, who refer the patient to external facility, or use the lookup function to select the required person.

    6. Print External Prescription - Select the check box to Print the External Prescription.

    On entering the required details, click [Record] to save the details.

  3. The Prescribed quantity, Allocated quantity, Fill quantity, Balance quantity, and the payment status are displayed in the Prescription Details frame.

  4. Select Complete Order Reason from the list.

  5. Click [Complete Deliver] on considering the Allocated/ Fill drug for dispensing; displays the Print/Edit Label screen.

 

Supporting Order registration before Patient Arrival

The orders can be registered before the patient arrives to the pharmacy to support processing the prescription before patient arrival. Once the patient arrives after order registration, the patient arrival is acknowledged. The patient ‘Arrival Time’ and ‘Token Number’ are displayed in green colour in the Patient ID column.

 

In the Delivery stage, a ‘Keep in Delivery Bin’ check box or ‘Patient Called’ check box is displayed in the ‘Patient Name’ column. At a time, only one of these check boxes is displayed.

• ‘Keep in Delivery Bin’ check box gets displayed if both the ‘Delivery Bin Applicable’ and ‘Register before Patient Arrival’ check boxes are selected in the Dispense Location screen. This allows you  to allocate drugs to a delivery bin.

• ‘Patient Called’ check box gets displayed only if the ‘Register before Patient Arrival’ check box is selected and ‘Delivery Bin Applicable’ check box is cleared in the Dispense Location screen. This allows you to record the patient called time.

 

Keep in Delivery Bin check box

If the patient has not arrived to the pharmacy to collect the drugs, the pharmacist allocates the drugs in the bin specified in the allocation stage or allocate to a different bin as follows:

  1. Select the ‘Delivery Bin Applicable’ check box in the Patient ID column.  A ‘Delivery Bin Location’ dialog is displayed.

  2. Select the required Bin from the ‘Delivery Bin Location’ list and click [OK]. The drugs are allocated to the newly selected bin location. The Delivery Bin is displayed in the Patient ID column.

If the patient has arrived to the pharmacy to collect drugs, the pharmacist allocates the drug by selecting the ‘Keep in Delivery’ check box. The selected delivery bin is displayed along with the 'Patient Called Time' in the Patient ID column.

 

Patient Called check box

The pharmacist dispenses the medication to the patient on arrival to the pharmacy. When the ‘Patient Called’ check box is selected, the delivery bin selected in the allocation stage is prompted. If required, the delivery bin can be changed. The ‘Patient Called Time’ is displayed in the ‘Patient ID’ column.

The Patient Called Time is significant in calculating the patient waiting time in the pharmacy.

 

If another user is accessing the same order, an icon is displayed in the 'User Locked' column. On clicking the Patient ID, a alert message 'This Patient Record is opened by another user. Access Denied' is displayed. Click icon to unlock and proceed.
