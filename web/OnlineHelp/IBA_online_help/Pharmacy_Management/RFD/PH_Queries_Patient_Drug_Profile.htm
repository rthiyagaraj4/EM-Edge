Enter the following search criteria:

 

  1. Patient ID – Enter the Patient ID in alphanumeric format of length as specified in MP Parameter or select the required Patient ID using [?] function. This is a mandatory field.

  2. Date [From-To] – Enter the date range in dd/mm/yyyy format or select the required date range using the Calendar function. The result set will display the Drug Profile for the specified date range.

  3. TPN – Select the check box to include TPN Orders in the Drug Profile.

  4. Generic Name - Enter the Generic Name in alphanumeric format of 40 characters maximum length or select the Generic Name using [?] function.

  5. Item Name - Enter the Drug Name in alphanumeric format of 60 characters maximum length or select the required Drug Name using [?] function.

  6. Patient Class – Select the required Patient Class from the list. The valid values are ‘Daycare’, ‘Emergency’, ‘External Patient’, ‘Inpatient’ and ‘Outpatient’.

  7. Status – Select the required Status from the list. The listed values are All, Completed, Active, Future Rx, Discontinued, On Hold, Returned and Cancelled.

  8. Drug - When you select this radio-button and click Search, the Drug Profile tab is selected. By default this radio button is  selected.

  9. Medical Item – If you select this radio-button and click Search, the Medical Item tab is selected.
