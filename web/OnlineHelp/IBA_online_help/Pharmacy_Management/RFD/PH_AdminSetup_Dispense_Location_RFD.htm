Enter the following details to create a dispensing location of required type:

 

Location Details

  1. Link to an Existing Store - Select the check box to link existing store to the Dispense Location or clear the check box, if Dispense Location is not linked to an existing Store. This field is enabled if ‘Inventory Management’ module is interfaced with Pharmacy Management module.

    When ‘Link to Existing Store’ check box is cleared, the ‘Store Code’ is disabled. You have to enter the following details to define the Dispense Location.

    1. Dispense Location Code - Enter the code in alphanumeric format of six characters maximum length. This is a mandatory field.

    2. Long Description – Enter the long description in alphanumeric format of 30 characters maximum length. This is a mandatory field.

    3. Short Description – Enter the short description in alphanumeric format of 15 characters maximum length. This is a mandatory field.

    1. Store Code - Enter the Store Code in alphanumeric format of 15 characters maximum length or select the code using the [?] function. The displayed Stores are of the Store Type ‘Pharmacy’ defined in the ‘Store’ function of ‘Inventory Management’ module. This is a mandatory field.

    A store can be linked to one Dispense Location only.

    On selecting the Store, the following details are displayed:

    1. Dispense Location Code – The Store Code is displayed as the Dispense Location Code as the Dispense Location is linked to the Store and the displayed value is disabled.

    2. Long Description - The long description of the selected Store is displayed and disabled.

    3. Short Description - The short description of the selected Store is displayed and disabled.

  2. Admixture Preparation Allowed – Select the check box, if Admixture Preparation is allowed. The Impact of this check is in Order Routing function to handle Special Orders.

  3. Type – Select the ‘Type’ from the list. The type ‘Pharmacy’, ‘Clinic’ and ‘Nursing Unit’ are listed. This is a mandatory field.

    • On selecting type as ‘Nursing Unit’, you can specify only the dispense details as ‘General’, ‘Narcotics’ and ‘Controlled’.

    • On selecting type as ‘Clinic’, you can specify only the dispense details as ‘General’, ‘Narcotics’ and ‘Controlled’.

  4. Direct Dispensing – Select the check box, if Direct Dispensing is applicable for the Dispense Location.

  5. External Dispensing – Select the check box if External Dispensing is applicable for the Dispense Location.

  6. External Prescription – Select the check box if External Prescription is applicable for the Dispense Location.

  7. Bar Code Scanning – Select the check box to capture the batch ID while scanning the bar code in Dispense Medication – All Stages.

  8. Enabled – Select the check box to enable the Dispense Location. Only the enabled records are available in required function.

 

Dispense Details

Dispensing Drug Classes

  • General – Select the check box to allow dispensing of drugs belonging to general class from the location.

  • Narcotics - Select the check box to allow dispensing of drugs belonging to narcotics class from the location.

  • Controlled - Select the check box to allow dispensing of drugs belonging to controlled class from the location.

 

There are three tabs available Outpatient Prescription Tab, Inpatient Prescription Tab and Other Details

Other Details Tab

  1. Online Print Setup - Click 'Online Print Setup' hyperlink to invoke Online Print Setup screen. All the Online Reports applicable for the enterprise are listed.

    All the Online Reports applicable for the enterprise are listed.

    • Select the 'Select All' value form the list. The listed values are 'No Print', 'Print', and 'Print Default'. The list options are available only when more than one report is printing at that stage. On selecting the 'Select All' value, the selected list value is displayed, by default for all reports. You can change the list values individually for the reports.

    • If ‘No Print’ is selected, the print screen will not appear.

     

    Click [OK] to record the selected values or [Cancel] to cancel the details.

  2. Print Dispensing Sheet in Direct Dispensing - Select the check box is selected, if ‘Print Dispensing Sheet’ is applicable in Direct Dispensing. If the check box is selected, it indicates dispensing sheet is printed after dispensing for the Dispense Location.

  3. Print Dispense Label in Direct Dispensing -  Select the check box is selected, if ‘Print Dispensing Label’ is applicable in Direct Dispensing. If the check box is selected, it indicates dispensing label is printed after dispensing for the Dispense Location.

 

Spill Quantity Details

  1. Usage of Spill Quantity during Admixture - Select the check box, if the usage of the Spill Quantity is permitted during admixture. If the check box is cleared, the usage of the spill quantity is not permitted.

  2. Charge Patient for Spill Quantity – Select the check box, if the usage of the Spill Quantity is chargeable.

 

Billing Interface

  1. Bill Stage – The value specified in ‘PH Parameter for Facility’ is defaulted. If Patient Billing Manager module is interfaced, the Bill Stage is applicable. Select the value from the list. The values listed are ‘Not Applicable’, ‘Upon Allocation’ and ‘Upon Delivery’.  

  2. Receipt Stage – The value specified in ‘PH Parameter for Facility’ is made default. If Patient Billing Manager module is interfaced, Bill/Receipt is applicable. The ‘Receipt Stage’ field is enabled if ‘Upon Allocation’ and ‘Upon Delivery’ is selected in Bill Stage field. Select the value from the list. The values listed are ‘Not Applicable’, ‘With Delivery’ and ‘Separate’.

 

Inpatient Prescription Tab

The dispensing rules for Inpatients are specified in this tab. The dispensing rules defined for Inpatient in ‘PH Parameter for Facility’ are made default. You can change these values to suit working of the Dispense Location.

 

Enter the following dispensing rules for Inpatients.

Stage

  1. Verification – The check box is selected, if ‘Verification’ is applicable for Inpatient in ‘PH Parameter for Facility’. It indicates verification of drugs is required during dispensing for inpatients. Clear the check box to indicate ‘Verification’ is not required.

  2. New Prescriptions Only – Select the check box if the Dispensing Location is applicable for New Prescriptions only.

  3. Fill – Select the check box if ‘Fill’ is applicable for the Dispensing Location.

  4. Fill Process - Fill Period – This field is used to compute the period for generating the fill sheet. The value specified in ‘PH Parameter for Facility’ is made default and can be modified but not exceeding the value defaulted. Specify the fill period in numeric format of two digits maximum length. The period can be defined in terms of ‘Hours’, ‘Days’ or can be based on the ‘Prescribe Period’. A fill sheet is generated for the specified period. If the fill period is based on hours or days, the period computed for fill sheet generation shall include the time specified in ‘Continuous/Intermittent Orders Schedule Start’.

  5. Deliver - The check box is selected, if ‘Delivery’ is applicable for Inpatient in ‘PH Parameter for Facility’. It indicates delivery stage is applicable for the Dispense Location. If delivery stage is not applicable to the Dispense Location.

  6. Exclude Routine Orders Placed within [] Hours of Schedule Start – The value specified in ‘PH Parameter for Facility’ is made default. Specify the period in numeric format of two digits maximum length. Routine orders placed within the specified duration are not listed in the generated fill sheet. The parameter value cannot be exceeded.

  7. Allocate – If allocation stage is applicable to the Dispense Location, select the check box. Clear the check box, if allocation is not required.

  8. Auto Dispense At Administration – Select the check box to indicate automatic dispensing of the drug during medication administration. Stock of the drug is updated on administration to the patient.

  9. IP fill process Schedule Start time – The value specified in ‘PH Parameter for Facility’ is made default. Enter the start time in the time format of ‘hh:mm’. Fill sheet is generated from the specified start time.

  10. Allow Medication Return - Select the check box to allow returning of Inpatient Medication. If the check box is selected, it indicates dispensed medication can be returned back to the Dispense Location.

  11. Daily Sequence Number Range - Enter the ‘Display Sequence Number Range’ in numeric format of five digits maximum length. Click on the ‘View Ranges’ hyperlink to display ‘Defined Sequence Ranges’ screen.  The Daily sequence number is generated at the Dispense Location level, which is mapped to different labels/reports printed in different printer in a Dispense Location for a patient.    

    • The screen displays the sequence number range defined for other Dispense Locations within the facility. This helps in avoiding duplication of sequence number.

 

Printer Location

  1. For Stat Order - Enter the Printer Location for stat orders in alphanumeric format of six characters maximum length.

  2. For Routine Order - Enter the Printer Location for routine orders in alphanumeric format of six characters maximum length.

  3. Fill Period: Fill Without Fill List – Specify the fill period in numeric format of two digits maximum length. The period can be defined in terms of ‘Hours’, ‘Days’ or can be based on the ‘Prescribe Period’. On loading, the ‘Fill Period: Fill Without Fill List’ value is same as the ‘Fill Process: Fill Period’ value but the user can modify as per the requirement, to display in 'Fill Without Fill List' screen only.

  4. Priority: Fill Without Fill List – Select the priority from the list. The listed values are ‘All’, ‘Urgent’ and ‘Routine’.

Outpatient Prescription Tab

The dispensing rules defined for Outpatient in ‘PH Parameter for Facility’ are made default. You can change these values to suit the Outpatient Pharmacy activities. Specify the following dispensing rules for Outpatients.

 

Stages

  1. Registration - The check box is selected, if ‘Registration’ is applicable for outpatient in ‘PH Parameter for Facility’. Clear the check box to indicate ‘Registration’ of prescription is not required before dispensing for the Dispense Location.

  2. Print Prescription – To print prescription at the Dispensing Location, select the check box. This field is enabled, if ‘Registration’ is applicable.

  3. Issue Token – To issue tokens for dispensing from the location after registration, select the check box. This field is enabled, if ‘Registration’ is applicable.

  4. Fill - The check box is selected, if ‘Fill’ is applicable for outpatient in ‘PH Parameter for Facility’. Clear the check box to indicate ‘Fill’ stage is not applicable to the Dispense Location.

  5. Fill Stage – The value specified in ‘PH Parameter for Facility’ is made default. Select the required value from the list. The values listed are After Allocate and Before Allocate.

  6. Before Allocation – The drugs are filled and then batches allocated.

  7. After Allocation – Batches are allocated and then drugs filled.

  1. Pharmacist Verification – The value specified in ‘PH Parameter for Facility’ is made default. Select the value from the list. The values listed are ‘Separate before Fill’ and ‘Combined with Allocation’.

  2. Separate Before Fill – The drugs are verified before batch allocation.

  3. Combined with Allocation – Drug verification and batch allocation are combined. This means, stock verification and batch allocation are performed together.

  1. Deliver - The check box is selected, if ‘Delivery’ is applicable for outpatient in ‘PH Parameter for Facility’. If ‘Delivery’ stage is applicable to the Dispense Location, select the check box.

 

Drug Return

  1. Allow Medication Return – The check box is selected, if ‘Allow Medication Return’ is applicable for Outpatient in ‘PH Parameter for Facility’.

  2. Within [] days – The value specified in ‘PH Parameter for Facility’ is made default. Enter the period in numeric format of two digits within which the medications are to be returned from the date of dispensing. This field is enabled only if ‘Allow Medication Return’ is applicable. The value specified in the parameter setting cannot be violated. This is a mandatory field.

 

Undelivered Drugs

  1. Max Drugs Retention Period Before Return to [] days - The value specified in ‘PH Parameter for Facility’ is made default. Specify the number of days in numeric format of two digits maximum length to return undelivered drugs held by the Dispensing Location back to the Stores. The specified period must not exceed the parameter setting.

 

Reissue Medication

  1. Generate Token upon Reissue – The check box is selected, if Token has to be generated on re-issuing medication.

 

Return Medication

  1. Generate Token upon Return - The check box is selected, if Token has to be generated on returning the medication.

 

Direct Dispensing

  1. Generate Token upon Direct Dispensing - The check box is selected, if Token has to be generated while Direct Dispensing the medication.

 

Printer Location

  1. For Stat Order - Enter the Printer Location for stat orders in alphanumeric format of six characters maximum length.

  2. For Routine Order - Enter the Printer Location for routine orders in alphanumeric format of six characters maximum length.

 

Other Dispensing Rules

  1. Register before Patient Arrival – Select this check box to register the order before the patient arrives to the pharmacy counter. By default, this check box is cleared.

  2. Generate Token Number before Patient Arrival – Select this check box to generate token number before the patient arrival is acknowledged. By default, this check box is cleared.

  3. Delivery Bin Applicable – Select this check box to allocate the delivery bin location before the patient arrives to the pharmacy counter.  By default, this check box is cleared.

  4. Default Next BMS Location – Select the Defaulr Next BMS Location from the drop-down list. Based on the selected value in the drop down list, you can define the default Value of 'Next Collection Location' for BMS. It will allow Pharmacist to partially dispense the discharge prescription without changing the Next Dispense Location.

  5. Same Location – Select this checkbox to default the same BMS location as dispensing Location. Once you select this checkbox the Default Next BMS Location drop-down is disabled.

The ‘Register before Patient Arrival’, ‘Generate Token Number before Patient Arrival’ and ‘Delivery Bin Applicable’ check boxes are displayed only if the ‘Registration’ and ‘Issue Token’ check boxes are selected.
