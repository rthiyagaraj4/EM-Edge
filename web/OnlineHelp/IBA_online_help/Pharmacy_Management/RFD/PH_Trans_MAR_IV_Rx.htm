IV Rx

Click ‘Scheduled Time’ hyperlink to view Multi Flow Rate details.

Multi Flow Rate details

  1. Flow Rate – The number of multi flow rate entered while placing IV medication is displayed.

  2. Admin Dose – The admin dose is displayed and disabled.

  3. Sch.Time – The scheduled date and time for the multi flow rate is displayed.

  4. Status – The ‘Status’ of the multi flow rate is displayed and hyperlinked.

    Click ‘Status’ hyperlink and select the ‘Start’ option to record the multiple flow start details in the Start MFR screen.

    • Start

      Enter the following details in the Start MFR screen:

      1. Drug Name - The name of the prescribed additive is displayed.

      2. Starts Remarks – Enter the start remarks in the free text format to start the multi flow rate.

      3. Start By – The name of the logged in user is displayed, by default. This is a mandatory field.

      4. Start Date/Time – The start date and time of the multi flow rate is displayed, by default. Select the start date and time using the calendar function.

      Click [OK] to save the entered record and the status is changed from ‘Not Started’ to ‘Started’ or click [Cancel] to cancel the start request.

      The Infusion date can be back time only before the multi flow rate is started.

      The change in the flow rate must be calculated and the admin details must be recorded mandatorily.

      The system prompts if you have entered the admin details during the complete stage of the transaction. If the infusion rate is changed, the multi flow rate end time for the particular flow rate and consecutive flow rate's start time and end times will be changed accordingly.

    • Hold

      Click ‘Hold’ option from the list to hold the multi flow rate administration.

      Enter the following details in the Hold MFR screen:

      1. Drug Name - The name of the prescribed drug is displayed.

      2. Hold Remarks – Enter the hold remarks in the free text to hold the multi flow rate administration.

      3. Hold By – The name of the logged in user is displayed, by default. This is a mandatory field.

      4. Hold Date/Time – The hold date and time of the multi flow rate is displayed, by default. Select the hold date and time using the calendar function.

      Click [OK] to hold the entered record and the status is changed from ‘Started’ to ‘Hold’ or click [Cancel] to cancel the hold request.

    • Resume

      Click ‘Resume’ option from the list to resume the IV Rx multi flow rate. The Resume option is applicable only for the prescriptions which are placed on Hold.

      Enter the following details Resume MFR screen:

      1. Drug Name - The name of the prescribed drug is displayed.

      2. Resume Remarks – Enter the resume remarks in the free text to resume the multi flow rate administration.

      3. Resume By – The name of the logged in user is displayed, by default. This is a mandatory field.

      4. Resume Date/Time – The resume date and time of the multi flow rate is displayed, by default. Select the resume date and time using the calendar function.

      Click [OK] to resume the entered record and the status is changed from ‘Hold’ to ‘Resumed’ or click [Cancel] to cancel the resume request.

      If you have resumed the flow after one hour the scheduled time for the consecutive flow rates should be one hour ahead accordingly on resuming. The infusion end time, schedule time of the consecutive flow rate, end times of the consecutive flow rates should be ahead one hour.

    • Complete

      Click ‘Complete’ option from the list to complete the multi flow rate.

      Enter the following details in the Complete MFR screen:

      1. Drug Name - The name of the prescribed drug is displayed.

      2. Complete Remarks – Enter the complete remarks in the free text to complete resume the multi flow rate administration.

      3. Complete By – The name of the logged in user is displayed, by default. This is a mandatory field.

      4. Complete Date/Time – The complete date and time of the multi flow rate is displayed, by default. Select the complete date and time using the calendar function.

      Click [OK] to complete the entered record or [Cancel] to cancel the complete request.

      The IV Rx is completed if the current system date and time is equal to or more than the end time of that flow rate. If you have made any changes, the scheduled time and end time should be one hour ahead or adjusted properly as per the recalculated time. After completion of all the flow rates, select the ‘Select’ check box at the header level and click ‘Apply’ to complete the administering of all the flow rates. You can enter the post administration details.

    Click ‘Started’ hyperlink to ‘Hold’, ‘Complete’ and ‘Resume’ the multi flow rate.
