Characteristics Tab

Enter the following characteristics of the defined Drug:

 

  1. Type – Select the type from the list. The values are ‘Infection Control’,’ Vaccine’ and ‘Poisonous/Scheduled’. If the defined Drug controls infection, select ‘Infection Control’. If the defined Drug is a Vaccine, select ‘Vaccine’. If the defined Drug is Poisonous or Scheduled, select ‘Poisonous/Scheduled’.

  2. Schedule – This field is enabled when ‘Poisonous/Scheduled’ check box is selected, that is, if the defined Drug is a Scheduled Drug, the field is enabled. Select the schedule from the list. The schedules displayed in the list are defined in the function ‘Drug Schedule’. The rules defined for the selected schedule are associated to the Drug and validated during Medication Ordering. This is a mandatory field.

  3. Non-Essential Restricted Drug – Select the check box, if the defined Drug is Non-essential Restricted Drug. The check box is cleared, by default.

  4. Trial Drug - Select the check box, if the defined Drug is a Trail Drug. The check box is cleared, by default.

  5. Affects Pregnancy - Select the check box, if the defined drug affects Pregnancy. The check box is cleared, by default.

  6. Pregnancy Category For Drug - enter the Pregnancy Category defined for the Drug or select the Pregnancy category using the [?] function. The displayed values are defined in the Pregnancy Category For Drug functions. This field is enabled, if 'Affects Pregnancy' check box is selected.

  7. First Trimester - Select the alphabet code from the drop-down list for the drug effect entered in Drug effect to pregnancy field.

  8. Warning Reqd. – Select this check box, if you want an alert message to be displayed in the Pharmacy Orders screen, to indicate the Practitioner, that the patient is pregnant. This field is displayed only if Affects Pregnancy check box is selected.

  9. Drug effect  to pregnancy – Enter the adverse effects of the drug, if prescribed to the patient during the First Trimester. This field is displayed only if Affects Pregnancy check box is selected.

  10. Second Trimester - Select the second level of risk for the drug from the list. The displayed values are the categories based on the Pregnancy Category for Drug selected.

  11. Late/Early Administration – Select this checkbox to display late/early administration option in MAR screen. By default this checkbox is not selected.

  12. Drug effect to pregnancy – Enter the adverse effects of the drug, if prescribed to the patient during the Second Trimester. This field is displayed only if Affects Pregnancy check box is selected.

  13. Third Trimester - Select the third level of risk for the drug from the list. The displayed values are the categories based on the Pregnancy Category for Drug selected.

  14. Warning Reqd. – Select this check box, if you want an alert message to be displayed in the Pharmacy Orders screen, to indicate the Practitioner, that the patient is pregnant. This field is displayed only if Affects Pregnancy check box is selected.

  15. Drug effect to pregnancy – Enter the adverse effects of the drug, if prescribed to the patient during the Third Trimester. This field is displayed only if Affects Pregnancy check box is selected.

  16. Med Admin Detail Required – Select the check box, if Medication Administration details are required. The check box is cleared, by default.

  17. Weight Recording - Select the Age Group from the list. If an Age Group is associated, it is mandatory to record patient’s height/weight while prescribing the Drug.

  18. Verbal Order Applicable during MAR – During Medication Administration for patients, select the check box, if the selected Drug can be placed as verbal order.

  19. Affects Breast Feeding - Select the check box, if the defined Drug affects breast-feeding. The check box is cleared, by default.

  20. Affects Lab Results - Select the check box, if the defined Drug affects lab-results. The check box is cleared, by default.

  21. Restricted Antimicrobials Justification Form Required – Select the check box if restricted antimicrobials justification form is required while ordering a drug. By default, the check box is cleared. This is applicable only for KDAH users.

  22. Restricted Antimicrobials Justification Form Duration – Only when the ‘Restricted Antimicrobials Justification Form Required’ check box is selected, the text box becomes enabled. Enter the duration to be considered while re-ordering the drug again in numeric digits up to two characters. This field is mandatory. When user tries to select the drug again in the time defined in the ‘Restricted Antimicrobials Justification form duration’ text box, then there will not be any restriction and user can proceed to order without filling any justification form.

  23. Stability Applicable – If stability factor is applicable, select the check box.

  24. Stability (Hrs) – If stability is applicable, enter the stability of the Drug in numeric format of four digits maximum length. This is a mandatory field.

  25. Patient Counseling Required - Select the check box to indicate patient required to be counseled on usage of Drug on dispensing.

  26. NPB Drug – Select the NPB Drug check box to mark the drug as Named-Patient-Basis drug. On selecting the NPB Drug check box, the ‘Dispense By Alternate Form’ and ‘Dispense By Multi-Strength’ check boxes are disabled.

  27. TDM Applicable - If Therapeutic Drug Monitoring is applicable, select the check box.

  28. Drug effect to pregnancy – Enter the comments about the drug effect to pregnant patient in the character format of 200 character maximum length.

  29. Used for Extemporaneous preparation - Select the check box to make the Drug available for compounding while prescribing ‘Compounding Rx’.

  30. Image Filename – Enter the path of the Image file related to the defined Drug.

  31. EDL Classification - Click [?] function to associate the EDL Classification to the Drug. The groups defined in the EDL Classification function is displayed. Select the check box against the group to associate the Drug to the required Essential Drug group.

  32. EDL Classification Remarks - Enter the EDL Classification remarks in alphanumeric format of 500 characters maximum length.  

  33. Drug Information – Enter the information related to the Drug in the text box.

  34. Patient Education – Enter the information related to the Drug to educate the patient regarding the Drug.

  35. Alert Drug Expiry Before [ ] Days - Enter the number of days to be considered to alert the user about drug expiry in Dispense Medication function. By default, the field is blank. User can enter maximum three digits in numeric format. (i.e., till 999 days). Based on the number of days defined, the system will display information about drug expiry in the Dispense Medication function.
