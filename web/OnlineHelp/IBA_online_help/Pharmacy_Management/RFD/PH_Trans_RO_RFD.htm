If ‘Issue Token’ is applicable for the selected Dispense Location:

 

Patient collecting Medication

If the patient is collecting the token, enter the following details:

  1. Patient Collecting Medication – The check box is selected by default, indicating that the patient is collecting the token.

 

Medication Collector Details

The details displayed are:

  • Name of the patient

  • Gender of the patient

  • Nationality

  1. Patient Arrived – Select this check box to indicate that the patient has arrived to the pharmacy counter. By default, this check box is selected.  The ‘Patient Arrived’ check box is displayed only if the ‘Register before Patient Arrival’ check box is selected in the ‘Dispense Location’ screen.

  2. Token Series – The Token Series applicable to the selected Dispense Location are displayed. Select the required Token Series from the list. The Token Number is generated based on the selected series. Based on the Gender and Nationality, the Token Series gets displayed, by default.

  3. Print Token - Select the check box to print the token.

 

Patient not collecting Medication

If patient has not collected the token:

Enter the following details:

  1. Patient Collecting Medication – Clear the check box, if the patient is not collecting the medication.

 

Medication Collection Details

Enter the following details:

  1. Name – Enter the name of the person collecting the medication. This is a mandatory field.

  2. Gender - Select the Gender of the person collecting the medication from the list. The listed values are Male, Female and Both. This is a mandatory field.

  3. Nationality - Select the Nationality of the person collecting the medication from the list. This is a mandatory field.

  4. Token Series – The Token Series applicable to the selected Dispense Location are displayed. Select the required Token Series from the list. The Token Number is generated based on the selected series.

  5. Print Token - Select the check box to print the token.
