Enter the following details to record the Incident Reports:

  1. Drug Name – Select the Drug Name for which the details have to be updated using [?] function.

  2. Location Type – Select Location Type from the list. The listed values are ‘Clinic’ and ‘Nursing Unit’. This is to indicate the source of the event. Select the location from the adjacent list. The Locations, which belong to the Location Type selected, are listed.

  3. Date of Incident – Enter the date and time in the format of dd/mm/yyyy hh:mm or select the required date using the Calendar function. The current system date and time is displayed, by default.

  4. Incident Remarks – Enter the remarks of the incident in the alphanumeric format. This is to provide information about the incident.

  5. Cause of Spill – Select the ‘Cause of Spill’ value from the list. The values are Container Leakage, Container Pierced, Container Dropped, Container Defective and Others.

  6. Witnessed by – Select the User Name using [?] function. On selecting the required User Name, the ID of the User gets displayed in the first field and the full name of the User is made default according to the setup in the next field.

  7. Dosage form of drug – Select Dosage form of Drug from the list. The values given are Plastic Bag, Syringe, Dry Powder and Liquid in Vial/Amp. On selecting the Drug, the default form of drug as mentioned in the Drug Master Setup is made default.

  8. Strength Supplied – Enter the volume of the drug supplied in the numeric format of 10 digits maximum length. The last two digits are taken as decimals automatically.

  9. Weight – Enter the Weight of the Drug in the numeric format of 10 digits maximum length. The last two digits are taken as decimals automatically. Select the Unit of Measure from the list. The values displayed are Gram, Mg, Mcg and Ncg.

  10. Spill Kit returned date – Enter the date and time in the format of dd/mm/yyyy hh:mm or select the required date using Calendar function.

  11. Spill Kit replaced date - Enter the date and time in the format of dd/mm/yyyy hh:mm or select the required date using the Calendar function.

  12. Reported By - Select the User Name using [?] function. On selecting the User Name, the ID of the User gets displayed in the first field and the full name of the User is made default according to the setup in the next field.

  13. Comment on Incident – Enter the incident comments in the alphanumeric format of 255 characters maximum length.

  14. Checked By – Select the User Name using [?] function. On selecting the User name, the ID of the User gets displayed in the first field and the full name of the User is made default according to the setup in the next field.

  15. Checked Date – Enter the Checked Date and time in the format of dd/mm/yyyy hh:mm or select the required date using the Calendar function.
