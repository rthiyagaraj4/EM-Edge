Refill Applicability

  1. Allow Refill – To allow prescription refill, select the check box.

  2. Max No of Refill allowed – If ‘Allow Refill’ check box is selected, this field is enabled. Enter the maximum number of refill allowed in numeric format of two digits maximum length. The specified value cannot be exceeded while ordering. This is a mandatory field.

  3. Max Days for Refill Prescription - If ‘Allow Refill’ check box is selected, this field gets enabled. Enter the maximum days for refill prescription in numeric format of three digits maximum length. This is a mandatory field.

 

For Example:

Outpatients

  • Max prescription duration for outpatient/take home medication = 30 days.

  • Maximum number of refill allowed = 5 times.

  • Maximum days for refill prescription = 100 days.

  • A practitioner prescribes Drug ‘A’ - 2 times a day for 30 days and refill 5 times.

In this scenario, medication order fails even though the prescription duration and number of refills have not exceeded the set parameter. The system multiplies the prescription duration (that is 30 days in the example) with the number of refills (five times). The product of which is 150 days, that is greater than the set maximum days for refill prescription at enterprise level (that is 100 days).

 

  1. Print Refill Prescriptions along with Main – Select the check box to Print Refill Prescriptions along with the Main Prescription. If the check box is cleared, only the main prescription is printed. This field gets enabled only when ‘Allow Refill’ is selected.
