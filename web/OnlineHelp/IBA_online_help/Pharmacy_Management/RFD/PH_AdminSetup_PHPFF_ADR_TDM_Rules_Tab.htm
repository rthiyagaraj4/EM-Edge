ADR/TDM Rules Tab

Rules that influence recording of Adverse Drug Reaction and Therapeutic Drug Monitoring are specified in the tab.

 

Enter the following details:

  1. ADR Start No. – Enter the number to be generated on recording Adverse Drug Reaction. Enter the value in numeric format.

  2. Verify TDM Order - If TDM orders require authorization on ordering, select the check box.

  3. TDM Reporting Note - Select the Reporting Note from the list. The selected note is used while the TDM order resulting in ‘Order Entry and Result Reporting’ module.

  4. Patient Counseling Note - Select the ‘Note from the list. The selected ‘Note’ is used for resulting counseling in ‘Order Entry and Result Reporting’ module.

  5. TPN Assessment Applicable – Select this check box to display the TPN Regimen tab in Pharmacy Order screen. By default, this check box is selected and enabled.

  6. Dispense  By – Select ‘Multi Strength’ or  ‘Strength’ from the drop-down list. This field is enabled only if the  ‘Dispense by Alternate Form’ check box is selected. If the ‘Dispense by Alternate Form’ is not selected, then select the check box to allow dispensing of multi strength and strength of the defined Drug.

  7. Default Generic Value for Therapeutic Drug Monitoring Statistics – Select the Default Generic Value for Therapeutic Drug Monitoring Statistics using lookup function. By default, the value is null.

  8. Office Hour for TDM From and To Time – Enter office hours in HH:MM format for releasing TDM orders from Pharmacy Module. Based on this configuration, you can determine the number of orders placed after office hours in respective  report. This field is mandatory.
