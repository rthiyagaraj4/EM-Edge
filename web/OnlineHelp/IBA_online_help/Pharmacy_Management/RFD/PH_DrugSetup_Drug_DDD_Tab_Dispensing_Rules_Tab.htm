Dispensing Rules Tab

The Dispensing Rules of the Drug are defined in this tab. Cautions, Special Instructions and direction to patients are recorded. These labels are generated on dispensing.

 

Dispensing Rules

  1. Fractional Dose Applicable – Select the check box to split the dose while prescribing.

  2. Fractional Dose Round Up – Select the check box to indicate rounding off the fractional dose prescribed. If the ordered dosage is greater than or lesser than the dispensing quantity, the pharmacist dispenses the actual dispensing quantity only.

  3. Default Dosage By For Normal RX - Select the default Dosage By value from drop-down list. The listed values are Quantity and Strength. The ‘Default Dosage By For Normal Rx’ is enabled only when the following setup is done.

  4. Strength value is entered in Prescribing Details tab.

  5. ‘Calculate by Frequency’ and ‘Fractional Dose Applicable’ checkboxes are selected in Prescribing Details tab and Dispensing Rules tab respectively.

  1. Dispense By Alternate Form – Select the check box to allow dispensing of alternate form of the Drug. This field is enabled, if ‘Dispense By: Alternate Form’ is applicable in ‘PH Parameter’. If dispensing by alternate form is applicable, all forms of the prescribed drug with same administration route can be dispensed. This field is not applicable for Compounding Drugs.

  2. Dispense  By – Select ‘Multi Strength’ or ‘Strength’ from the drop-down list. This field is enabled only if the ‘Dispense by Alternate Form’ check box is selected. If the ‘Dispense by Alternate Form’ is not selected, then select the check box to allow dispensing of multi strength and strength of the defined Drug.

  3. PRN Doses: For Prescribed Period Fill – Specify the value in numeric format of three digits maximum length. The specified value is made default for the ‘Ordered Quantity’ while ordering when PRN Frequency is selected.

  4. For Continuous Fill [] per day – Specify the value in numeric format of three digits maximum length. The specified value is the fill quantity per day for inpatients when IP Fill Process is executed.

 

Dispensing Controls

  1. Authorize on Dispensing – Select the check box to indicate that the authorization is required for dispensing the drug.

  2. Dispense via Prescription only - Select the check box to indicate that the dispensing is performed with prescription only.

  3. Reissue Applicable - Select the check box to indicate dispense against damage of a Drug. Drug marked ‘Re-issue Applicable’ can only be re-issued in the transaction function ‘Reissue Medication’.

  4. Reissue Type - Controls the reissuing of medication to a certain number of quantities and number of times that is defined. The available values are ‘No Restriction’, ‘Fixed’ and ‘Variable’. This drop-down is enabled only if the Reissue Applicable check box is checked. The default value will be ‘No Restriction’.

• If you select the default value ‘No Restriction’, the existing functionality will continue.

• If you select the value ‘Fixed’, two more additional fields, a drop-down and a text box will appear. The drop down will have the values ‘No of Quantity’ and ‘No of Times’. The default value will be ‘No of Quantity’. The label ‘Qty’ or ‘Time(s)’would appear based on the value selected in the drop-down.

o If you select ‘No of Quantity’, you must enter the number of quantity in the text field. The number of quantity mentioned in this text field will determine the maximum number of quantity the user can reissue of the medication to be reissued.

o Similarly, if you select ‘No of Times’, you must enter the number of times in the text field to determine the maximum number of times you can reissue the of medication reissue. In this case, the number of quantity to be reissued will be based on the already dispensed quantity.

• If you select the value ‘Variable’, you can reissue the medication based on the already dispensed quantity. The reissue quantity cannot exceed the already dispensed quantity for each reissue. However, you can reissue the medication any number of times.

  1. Verification Required - Select the check box to indicate that the Verification is required for dispensing the drug.

  2. Allow More Than Prescribed Quantity - Select this check box to allow dispensing of the Drugs more than the prescribed quantity.

  3. Print Number of Labels Based on Filled/Delivered Quantity  – Select the checkbox to enter the ‘No of Copies’ in Edit Dispense Label screen. If the checkbox is cleared then ‘No of Copies’ is default to 1.

 

Labels

These instructions are printed as labels on dispensing the defined drug.

  1. Caution – Select the Caution Label from the list. The Label Texts defined for ‘Cautions’ in ‘Label Text’ function are listed. You can select two Caution Labels.

  2. Special Instruction - Select the Special Instruction from the list. The Label Texts defined for ‘Special Instruction’ in ‘Label Text’ function are listed.

  3. Direction To Patient – Enter the Direction to Patient in alphanumeric format of 40 characters maximum length.

  4. Drug Indications – Enter the drug indications to be printed on the Medication Plan Sheet.
