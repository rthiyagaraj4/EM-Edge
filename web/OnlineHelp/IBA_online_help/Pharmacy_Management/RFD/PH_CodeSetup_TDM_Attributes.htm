Enter the following details to set the TDM Attributes:

  1. Drug Name –Enter the Drug Name (for which TDM is applicable) in alphanumeric format of 60 characters maximum length or select the required Drug using [?] function. The search result display drugs for which ‘TDM Applicable’ is selected in the ‘Drug’ function during its creation. This is a mandatory field.

  2. Age Group - Select the required Age Group from the list. The listed values are defined in ‘Age Group’ function of ‘Application Masters’ module. This field is mandatory.

  3. Min/Max Age – Based on the selected Age group, the minimum and maximum age values are displayed as read-only text.

  4. Gender - Gender specified for the selected Age Group is displayed as read-only text.

  5. Steady State - Specify the Steady State in terms of days, hours or minutes. Enter the value in numeric format of two digits maximum length. The period taken by the drug to reach the Steady State is specified. This is a mandatory field.

 

On entering the required details, the sampling details are displayed. The Sampling Types are ‘Peak’, ‘Trough’, ‘Extended Dose’ and ‘Random’.

Sampling Types – Random

 

  • Laboratory Orders - Select the check box to place Laboratory Orders. On selecting the check box, the ‘Associate’ text is hyperlinked. Click on the link to view the Laboratory Orders in the ‘Laboratory Search’ screen. The Order Catalogs belonging to the ‘Laboratory’ Order Category are displayed. Select the check box to select the Order Catalog against it for placing order.

Sampling Types – Extended Dose

 

  • Sampling Time - Specify the Sampling Time for trough type in numeric format of two digits maximum length. Specify the sampling time in terms of days, hours or minutes.

  • Laboratory Orders - Select the check box to place Laboratory Orders. On selecting the box, the ‘Associate’ text is hyperlinked. Click on the link to view the Laboratory Orders in the ‘Laboratory Search’ screen. The order catalogs belonging to the ‘Laboratory’ Order Category are displayed. Select the check box to select the Order Catalog against it for placing order.
