Enter the following details for specifying the Order Routing for Billing Group:

Ordering Characteristics:

  1. Ordering Facility – The user’s logged-in facility is displayed, by default.

  2. Location Type - Select Location Type; based on the value selected, the Location drop-down list is populated with the values accordingly. The values displayed here are defined in the function Care Location Type of Application Masters module. For example, Nursing Unit, Clinic

  3. Location - Select the Physical location of the Nursing Unit. For example, Burns unit, ICU and so on.

Routing Locations:

  1. Routing Level – The Routing Level for the order is displayed. For example, Source, Order Type, Drug Class, Drug, Special Orders, Medical Item and Home Leave.

  2. Routing – Based on the Routing level, the routing for the order  is displayed.

  3. Performing Facility - The logged in Facility is displayed.

  4. Routine - The default Dispensing Location to route Routine Orders of Inpatients is displayed. The locations are defined in ‘Dispense Location’ function. Dispensing locations of type ‘Pharmacy’ and catering to both Outpatient/Inpatients and exclusively to Inpatients are listed.

  5. Stat - The default dispensing location to route stat orders of Outpatients. The locations are defined in ‘Dispense Location’ function. The dispensing locations of type ‘Pharmacy’ and catering to both Outpatient/Inpatients and exclusively to Outpatients are listed.

  6. Discharge/Take Home Medication – Select the default dispense location for the Discharge/Take Home medication.

  7. Billing Group -  The Billing Group for the order is displayed. Billing Group denotes various types of payments the Facility accepts from the patients or the customers paying on behalf of the patients. A Billing Group is assigned to a patient during registration.

  8. Customer Group – The group to which the Customer belongs. Typical examples are Insurance Companies, Group Companies, Staff, etc.

  9. Customer – The Customer name is displayed. Customers can be patient related or non-patient related customers.

  10. Policy Type - The associated policy types for the customer are displayed. You can select a Facility to define the Policy Types pertinent to the respective Facility. The existing Policy Type records are displayed.

  11. Select – You can select the required routing details for the order.

Routing Locations:

  1. Routing Level - Select Routing Level. For example, Source, Order Type, Drug Class, Drug, Special Orders, Medical Item and Home Leave. The default Routing Level will be ‘Drug Class’. You can select any other value from the drop-down list based on the order.

  2. Performing Facility - Select the performing facility, if the order routing is performed by any other facility. By default, the logged-in facility is displayed.

  3. Billing Group - Select Billing Group as the order routing is done based on the billing group. Billing Group denotes various types of payments the Facility accepts from the patients or the customers paying on behalf of the patients. A Billing Group is assigned to a patient during registration.

  4. Customer Group – Select a customer group from the drop-down list. The values in the Customer Group drop-down list are displayed, based on the billing group selected. If the Billing Group is selected as ‘Cash’, the fields ‘Customer Group’, ‘Customer’ and ‘Policy Type’ will be blank.

  5. Customer – Select a customer from the drop-down list, which displays values based on the customer group selected. Customers can be patient related or non-patient related customers.

  6. Policy Type – Select a policy type from the drop-down list, which displays the associated policy types for the customer. You can select a Facility to define the Policy Types pertinent to the respective Facility. The existing Policy Type records are displayed.

Default Dispense Location:

  1. Routine Orders – Select the default dispense location for the Routine order for Inpatients. The locations are defined in ‘Dispense Location’ function. Dispensing locations of type ‘Pharmacy’ and catering to both Outpatient/Inpatients and exclusively to Inpatients are listed.

  2. Alternate – Click Alternate to set the alternate location for the dispense location. The Day Type drop-down list will be displayed.

  3. Day Type - The Alternate Dispense Location will be set for the days selected within this drop-down list. For example, Holidays, Sunday, Monday, Tuesday, Wednesday, Thursday, Friday, Saturday and All Days

  4. Search – Click Search to display the results based on the criteria specified.

  5. Search results grid – Displays details such as Dispense location, From Time, To Time and allows you to select the details using the Select check box. Adds the selected details under the Routing Locations section.

  6. Stat Order – Select the default dispense location for the stat order.

  7. Alternate – Click Alternate to set the alternate location for the dispense location.

  8. Discharge/Take Home Medication – Select the default dispense location for the Discharge/Take Home medication.

  9. Alternate - Click Alternate to set the alternate location for the dispense location.

  10. Add – Select the rows from the search results grid and click Add; adds the details under the Routing Locations section. Routing Level column will be displayed as a hyperlink.

You can modify the selected order routing details by clicking the Routing Level hyperlink.

  1. Clear - Click Clear to clear the data in the fields.

The dispense Location set in the Order Routing for Billing Group screen will be set as default in the Pharmacy Orders screen, as well, when placing the pharmacy orders for the patient. The order will then be routed to the specified Dispense Location.
