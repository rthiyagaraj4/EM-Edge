Medication Administration Tab

Rules required for Medication Administration are specified in this tab.

 

Enter the following details:

  1. Medication Administration Required - Select the check box to indicate that the Medication Administration is applicable to the logged in facility. If it is cleared, ‘Medication Administration’ transaction function is not applicable to the logged in facility.

  2. PIN Authentication for Drug Category Required -  Select the check box to indicate Authentication is required for a Drug Category.

  3. Verification Required before Drug Administration - Select the check box to indicate verification is required before drug administration. If it is cleared, the verification is not performed before Drug Administration for the logged in facility.

  4. Record Administration Within [] hrs – Enter the period in hours within which Drug Administration has to be recorded. Enter the period in numeric format of two digits maximum length. During MAR, the schedules are displayed based on the set period.

  5. Allow Medication Administration Without Order – Select the check box to allow medication administration without order. During Medication Administration verbal orders can be placed.

  6. Witness for Verbal Order - By default, this check box will be unchecked and disabled. It will be enabled  only if the ‘Allow Medication Administration without Order’ is checked. The system will prompt you to capture the witness details while releasing a verbal order on behalf of others. The order will be witnessed by another user.  For self orders, the system will not prompt you to capture the witness details even if this check box is checked.  

  7. Patient Stock Required – Select this checkbox to enable the Patient Stock functionality in Place Order and Medication Administration screens for the item. By default this checkbox is not selected

  8. Record Administration Before Schedule Time - Enter the period in minutes or hours to administer the medication before the scheduled time. Enter the period in numeric format of two digits maximum length.

  9. PIN Authentication in MAR – Select this check box to prompt a PIN Authentication dialog in Medication Administration screen. This checkbox is selected by default.

    1. Administrated By - The logged in user name is displayed by default. You can modify the user name using lookup function. This is a mandatory field.

    2. PIN Number – Enter the PIN Number in the alphanumeric format of 64 maximum character length.

     

  10. Allow Alternate Drug for Floor Stock Item – Select the check box to allow alternate drug for floor stock item. If you select the check box, Alternate Drug icon  gets displayed in the Medication Administration screen. By default, the checkbox is cleared.

  11. Print Label – Select this check box to print the MAR label.

  12. Display Held/Discontinued Orders – Includes held/discontinued orders also in the search results.

  13. Allow Shared Drug - If you select this check box , the shared drug check box within the Medication Administration screen will appear else it will not appear.

  14. Medication Administration Schedule Level - By default, this check box is unchecked. When you select the Medication Administration Schedule Level check box, all the below mentioned fields under Medication Administration Schedule Level are enabled and you can select/deselect, as required.

  15. Display Records From - The available options are Starting day of the week, Current Date and User Defined Option. The default value will be ‘Starting day of the week (Monday)’.

    • If you select the day as “Tuesday”, the system shows data from Tuesday to Monday.

    • If you select “Current Day” in ‘Display records from’, the system will display the records from current day to the following one week. For example, if you select the current date as 07/07/2013, then the system shows data from 07/07/2013 to 07/13/2013 (MM/DD/YYYY) days medication administration data.

    • If you select “User Defined Option” in ‘Display records from’ drop-down list, two new text fields “Past” and “Future” will be displayed. You must enter the value in the past and future days fields mandatorily. Both the new fields will accept numeric value and the system will display the Medication administration records.

  16. Display Order By - The available options are Drug Name, Dosage Form and Order Date/Time. The default value will be Order Date/Time.

  17. Allow Record Discontinued Dose - By default, this check box is unchecked. If you select this check box, the system allows you to administer the discontinued medications by displaying an alert.

  18. IV Completion Date Time - The  available options are Current Date/Time and Calculate by infuse over. The default value will be Calculate by infuse over.

    • If you select “Current Date/Time” or “Calculate by infuse over” option and complete the IV infusion, the system will display the completion time as “Current Date/Time” or Completion time based on the Infuse over time respectively. 

  19. Allow MAR for Dispensed Unbilled Medication – Select the check box to allow MAR only if the medication is billed. By default, the check box is cleared. This is applicable only for Normal Rx for OP Patient class. This is applicable only for NMC users.

Applicable Patient Classes

  1. Select the Patient Class to which Medication Administration is applicable. The Patient Classes are ‘Inpatient’, ‘Outpatient’, ‘Daycare’, ‘Emergency’ and ‘Discharge Medication’.

  2. On selecting the Patient Class, a Rule Based check box near the respective Patient Class gets active. When the Rule Based check box is selected, the patient class value gets displayed in the ‘Applicable Patient Class’ list in the Build MAR Rule screen.
