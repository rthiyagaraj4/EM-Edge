Enter the following details to create Token Series:

 

  1. Dispense Location - Select the Dispense Location from the list. The locations listed are of type ‘Pharmacy’ and Token Issued are applicable to a particular Dispense Location. This is a mandatory field.

  2. Code – Enter code in alphanumeric format of two characters maximum length. This is a mandatory field.

  3. Description – Enter the Token Series description in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Generate Token – Select the check box to generate the Token. The check box is selected, by default.

  5. Start Serial No. – Enter the start number for the series in numeric format of four digits maximum length. This is a mandatory field.

 

Applicability

  • Gender – Select the Gender from the list. The listed values are Both, Male and Female. The value ‘Both’ is displayed, by default.

  • Nationality - Select the Nationality from the list. The listed values are Both, Nationals and Expatriates. The value ‘Both’ is displayed, by default. The listed values are pre-defined.

 

  1. Generation Upon – Select the ‘Generation Upon’ value from the list to generate the token. The listed values are based on the Dispense Location setup. You can generate Token on ‘Register Order’, ‘Reissue Medication’, ‘Return Medication’, ‘Direct Dispensing’, and ‘Order Release’. The Release Order can be associated to the Token Series once only.

  2. Enabled – Select the check box to enable the Token Series.

 

QMS

  1. Required - Select the Required check box to interface Queue Management System (QMS) with Pharmacy Management module.

  2. Edit Token – This field gets enabled only if the ‘Required’ check box is selected. Select the Edit Token check box to edit the token number generated from QMS.

  3. Select - Select the check box to associate the Token Series to the selected Dispense Location.
