Enquiry Details Tab

The Enquiry details tab consists of two sections, namely, ‘Enquirer Details’ and ‘Enquirer Contact Details’.

 

Enquirer Details section

  1. Source of Request - Select the Source of Request from the list. The listed values are Within Facility, Other Enterprise Facility, External Facility and Public. On selecting, ‘Within Facility’, the Logged in Facility Name is displayed and disabled.

  2. Facility –

  3. The Logged in Facility Name is displayed on selecting the Source of Request ‘Within Facility’.

  4. If the ‘Source of Request’ value selected is ‘Other Enterprise Facility’, enter the Facility Name in alphanumeric format of 30 characters maximum length.

  5. If the ‘Source of Request’ value selected is ‘External Facility’, select the required external facility from the list. The listed values are defined in the Facility function of System Manager module of Enterprise Management.

  6. This field is not available, if the ‘Source of Request’ value selected is ‘Public’.

  1. Name of the Person - Select the name of the request source using [?] function. This field is enabled on selecting the ‘Source of Request’ value ‘Within Facility’ or ‘External Facility’. This field is disabled on selecting the ‘Source of Request’ value ‘Other Enterprise Facility’ or ‘Public’. The name is defaulted based on the logged in user details, if the value ‘Self’ is selected in the Mode of Request drop-down list.

  2. Others – You can enter persons name in case of ‘Other Enterprise Facility’ or ‘Public’.

  3. Designation – The designation of the enquirer is made default based on the logged in user details if the value ‘Self’ is selected in the Mode of Request drop-down list.  

  4. Priority of the Request – Select from the Priority of the Request value from the list or enter any other priority into the text box the possible values are Urgent, immediate (Within 1-2 hours), Within 24 Hours, in a week’s time. This is a mandatory field.

  5. Others – You can enter any other priority, if the value is not available in the List item Priority of the Request.

  6. Enquiry Date/Time – Enter the Enquiry date and time in the dd/mm/yyyy hh:mm format. The date has to be greater than the system date.

  7. Mode of Request – Select from the list of possible values the mode of request or enter the mode. The mode of request must be in the form of Telephone, In Person, Facsimile, Email or Referral. This is a mandatory field. If you select the value ‘Self’, then the Name of the Person and Designation fields are populated based on the logged in user details.

  8. Others – You can enter a value here, if the desired value is not listed in Mode of Request list item.

 

Enquirer Contact Details section

This section captures the contact details of the enquirer like the Contact Address, Telephone number, Facsimile and Email these details are copied to the reply details with the help [COPY] hyperlink. Click copy hyperlink to copy the Contact Address details to the Reply Address fields.
