To view specific orders, enter the following details:

 

  1. Order Date [From-To] – The current system date gets displayed, by default. Enter the order date in date format of dd/mm/yyyy. Specify the order date range. The orders placed within the specified range are listed.

  2. Scope - Select the prescription scope from the list. The listed values are ‘Dept. Order on Hold’ and ‘External Prescriptions’. The table below explains the scope.

 

Scope

Description

Dept. Order on Hold

Orders placed on Hold.

External Prescriptions

Prescriptions placed using the function ‘Transcribe External Prescription’.

 

  1. Group By - Based on the dispense level set for the selected Dispense Location, the value gets displayed as read-only text. The displayed values are Order wise or Patient wise. You can specify the ‘Dispense Level’ under ‘Show Criteria’ and group the result set accordingly.

  2. Dispense Location – The selected Dispense Location in the login window gets displayed as read-only text.

  3. Dispense Stage – The Dispense Stage ‘Verification’ gets displayed as read-only text.

  4. Priority – Select the Priority from the list. The listed values are All, Urgent and Routine. ‘All’ gets displayed, by default.

  5. Dispense Level - Select Dispense Level from the list. The listed values are Order Wise and Pat Wise. Dispense level is ‘Order wise’, if Type of Order is selected as ‘Compounding’, ‘IV with Additives (Admx)’, ‘IV intermittent Admx’, ‘IV without Additives’, ‘TPN Admixture’, ‘Oncology’.

 

Dispense Level

Description

Order Wise

Order ID(s) gets displayed followed by the prescribed drugs under each Order ID.

Patient Wise

All the Drugs prescribed for the patient irrespective of order ID are displayed in result section.

 

  1. Type of Order - Select the required order type from the list. The listed values are pre-defined. Refer the table below for more details.

 

Order Type

Description

All

Al types of orders are displayed.

Rx Order

Orders placed using ‘Rx’ function.

Compounding Order

Orders placed using ‘Compounding’ function.

IV-With Additives(Direct or Admx)

Orders placed using ‘IV’ function with type as ‘IV with Additives’.

IV-Without Additives

Orders placed using ‘IV’ function with type as ‘IV without Additives’.

IV-Intermittent(Direct or Admx)

Orders placed using ‘IV’ function with type as ‘IV Intermittent’.

TPN Direct

Orders placed using ‘TPN Regimen’ function.

Oncology Order

Orders placed using ‘Oncology’ function.

TPN Standard

Order Placed using ‘TPN Standard’ function.

TPN Admixture

Orders placed using ‘TPN’ function for Non-Standard regimen.

Oncology Admixture

Orders placed using ‘Oncology’ function for which admixture has to be prepared.

 

  1. Patient ID – Enter the Patient ID in alphanumeric format of length as specified in MP Parameter or click [?] function to search for the required patient.

  2. Ordering Location – Select the Ordering Location from the list. The listed values are ‘Clinic’ and ‘Nursing Unit‘. ‘Clinic’ gets displayed, by default. Select the Location associated to the selected location using [?] function.
