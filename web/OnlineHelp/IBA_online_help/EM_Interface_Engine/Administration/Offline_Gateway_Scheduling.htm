Offline Gateway Scheduling

From this function you can Schedule to Record the Offline Gateway Attributes.

Offline Gateway Scheduling

From EM Interface Engine > Administration > select Offline Gateway Scheduling function.

The Offline Gateway Scheduling screen gets displayed.

  1. Enter the required details.

    1. Protocol Link - Enter or Select the Protocol Link using the [?] function to create interface gateway attributes

    2. Type - Select the Type from the list. ‘Daily’ is displayed, by default. The other options available are ‘Monthly’, ‘Weekly’ and ‘Yearly’.  

    3. Month – This field is disabled by default. This field is enabled, only if the ‘Monthly’ option is selected in the ‘Type’ Field.

    4. Day – This field is disabled by default. This field is enabled, only if the ‘Weekly’ option is selected in the ‘Type’ field.

    5. Date Number – This field is disabled by default. This field is enabled, only if the ‘Monthly’ or ‘Yearly’ option is selected in the ‘Type’ field.

    6. From Sleep Time – Enter the time in valid format. For the specified period, the communication process will be inactive.

    7. To Sleep Time – Enter the time in valid format. For the specified period, the communication process will be inactive.

  2. Click [Record] to record the Offline Gateway Scheduling or click [Reset] to reset the entered data.
