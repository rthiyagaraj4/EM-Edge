Segments

Click the Segments Tab. Segment is a logical grouping of data fields and each Segment is identified by a unique Segment ID.

  1. Segment ID – The Segment ID of the Segment is displayed.

  2. Segment Name – The name of the segment is displayed.

  3. Standard Code –

  4. Optional – The check boxes are disabled. Segments may be mandatory or optional in the message. Enterprise Management displays the optional segments by placing a check mark in the corresponding check box. The mandatory segments will be by default included in the message. The optional segments that need to be included in the message are defined by the user.

  5. Select - Only optional segments can be disabled or enabled by selecting the check box. Select the corresponding check box to include the segment in the message.

  6. Repetition Applicable –

  7. Message Control –

  8. File Control Type – Select File Header or File Footer from the list.

  9. Click [Apply] to save the record.

This tab is a global level setting for the segments; the user has an option to not include the ‘selected’ segments in the message in the ‘Event wise Segments’ tab. The user is allowed only to enable or disable the optional segments. The segments without a check mark in the corresponding check boxes (under the ‘Optional’ column) indicate mandatory segments.
