Enter the following details to insert a new record:

 

  1. Age Group - Enter the Age Group in alphanumeric format of 15 characters maximum length, or search for the age group by clicking [?] function. On invoking [?] function, Age Group search screen gets displayed. This field is mandatory.

Enter the following search criteria to select the required value.

  1. Search By – Select either the Description or Code as the value.

  2. Search Criteria - Select the required search criteria from the list. The listed options are Starts With, Ends With and Contains.

  3. Search text - Enter the required search string.

On entering the required criteria, click [Search]. The result records are displayed based on the search criteria.

  1. Code – The Code for the Age Group defined is displayed and hyperlinked.

  2. Description – The descriptions for the Codes are displayed and hyperlinked.

Click on the Code or Description hyperlink to select the required value. The search screen gets closed and the selected value gets displayed in the corresponding field.

  • [Reset] – Click [Reset] to clear the displayed records and to enter the new search criteria.

  • [Close] – Click [Close] to exit the search window.

  1. Bed Class – The Bed Class defined is displayed.
