Enter the following details to create a Pseudo Bed:

 

  1. Nursing Unit – Enter the required Nursing Unit in alphanumeric format or select the Nursing Unit using [?] function. This field is mandatory.

  2. Room/Cubicle No – Select a Room/Cubicle Number from the list. The list of values corresponds to the rooms/cubicles set for the selected Nursing Unit. This field is mandatory.

  3. Bed No. – Enter the Bed Number in alphanumeric format of eight characters maximum length. The Bed Number has to be unique for a Nursing Unit and cannot be changed after creation. This field is mandatory.

  4. Enabled - Select the check box to enable the Pseudo Bed for Nursing Unit, or clear the check box to disable it.

 

Characteristics

  1. Bed Class – Select the required Bed Class from the list for the selected Nursing Unit.  This field is mandatory.

  2. Bed Type - Select the required Bed Type from the list for the selected Nursing Unit. The Bed Type values correspond to the bed class selected. This is a mandatory field.

  3. Gender Specific Ind – Select the option from the list. The values listed are Male, Female and Unknown. If no value is selected, the bed created cannot be gender specific. If the selected Nursing Unit is defined to be gender specific (if the defined Nursing Unit is attached to an age group that is gender specific), the default gender set up for the Nursing Unit is set in this field and the field gets disabled.

  4. Specialty – Select the option from the list. Depending on the Nursing Unit selected, the applicable values get populated. If no value is specified, the pseudo bed is open to all specialties.
