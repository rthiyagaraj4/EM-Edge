Enter the Mother Details:

 

Mother Details

  1. Mother’s Previous HN - The previous hospital number details specified for the mother patient during patient registration is displayed as read-only text.

  2. Multiple Birth – Select the check box if the selected mother patient has delivered more than one baby during current encounter.

  3. No. of Births – This field is enabled only if ‘Multiple Birth’ check box is selected. Enter the number of babies delivered by the mother patient in numeric format of one digit maximum length. If ‘Multiple Birth’ is not selected, the number of birth is  disabled and displayed as ‘1’.

  4. Gestation – Gestation refers to the duration between the conceiving date and date of delivery. Enter the gestation period in numeric format of three digits maximum length. The entered value must be within the Gestation range specified in MP Parameter or, an alert is displayed. The unit of measure specified in the MP Parameter is displayed as read-only value. This field is mandatory.

  5. Delivery Type – Select the Delivery Type from the list. If the Delivery Type selected is created with Indicator 'Normal', all the three Complication in stages of Labour fields in Mother Details Section of Register New Born function are displayed and Complications of Delivery field is disabled. If the Delivery Type selected is created with Indicator 'C Section', Complications of Delivery field is enabled and all the three Complication in stages of Labour fields are disabled. If the Delivery Type selected is created with Indicator 'Others', both 'Complications of Delivery' field and 'Complications in stages of labour' fields are enabled.

  6. Gravida – Enter a value in numeric format of two digits maximum length. This field is mandatory.

  7. Parity – Enter a value in numeric format of two digits maximum length. The value of parity cannot be greater than gravida. This field is mandatory.

  8. Attending Practitioner – Enter the name of Practitioner attending the patient in alphanumeric format of 30 characters maximum length or select the practitioner name using the [?] function. This field is mandatory.

  9. Attending Nurse - Select the name of the nurse attending the birth from the list. The Practitioners of type ‘Nurse’ and ‘Mid-Wife’ associated to the logged in facility are listed.

  10. Complications Related to Pregnancy - Select the Complication of Pregnancy from the list. The values created with ‘Related to Pregnancy’ check box selected in Complications of Pregnancy function are listed.

  11. Complications not Related to Pregnancy - Select the Complication of Pregnancy from the list. The listed values are created in Complications of Pregnancy function for which ‘Related to Pregnancy’ check box is not selected.

  12. Complications in First Stage of Labour - Select the Complications in First Stage of labour from the list. The listed values are defined in Complications of Delivery in Master Patient Index module.

  13. Complications in second Stage of Labour - Select the Complications of second Stage of labour from the list. The listed values are defined in Complications of Delivery function in Master Patient Index module.

  14. Complications in Third Stage of Labour - Select the Complications of Third Stage of labour from the list. The listed values are defined in Complications of Delivery function in Master Patient Index module.

  15. Complications of Delivery – Select the problem that occurred during delivery from the list. The listed values are defined in the ‘Complications of Delivery’ function of Master Patient Index module.

  16. Delivery Indication - Select the delivery indication from the list. The listed values are defined in the ‘Delivery Indication’ function of Master Patient Index module.

  17. Procedure Classification – Select the Procedure Classification using [?] function.

  18. Vaginal Delivery Type - Select the Vaginal Delivery Type using [?] function.

  19. Delivery Surgical Classification – Select the Delivery Surgical Classification using [?] function.

  20. Remarks – Enter remarks if any.

  21. Antenatal Care Given  - Select the ‘Antenatal Care Given’ check box to enter the Antenatal Care Details. Once the ‘Antenatal  Care Given’ check box is selected, the ‘Antenatal Care Given Location’ field becomes enabled.

  22. Antenatal Care Given Location - Enter the Location from which the care is availed. There are two options listed. They are In Hospital and External Hospital.
