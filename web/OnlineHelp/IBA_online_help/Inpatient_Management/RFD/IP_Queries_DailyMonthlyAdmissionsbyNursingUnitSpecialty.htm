Enter the following query criteria:

  1. Admissions - Select the admissions as daily or monthly from the list. By default, ‘Daily’ is displayed in the list.

  2. Nursing Unit - Select the nursing unit using [?] function.

  3. Specialty - Select the specialty using [?] function.

  4. Service – Select the required service using [?] function.

  5. Statistics By – Select the required value from the drop-down list value. The list includes Nursing Unit, Specialty, Service. This field is mandatory.

  6. Admission Date [From-To] – Enter the admission Date From in the date format of dd/mm/yyyy for daily basis (admissions) and in mm/yyyy format for monthly basis (admissions). The admission date from appears on selection of admissions as ‘Daily’. The admission month from appears on selection of admissions as ‘Monthly’.
