The following details are displayed:

 

  1. Admission Date/Time - The Admission Date and Time of the inpatient is displayed.

  2. Admission Type - The Admission Type of the inpatient is displayed.

  3. Emergency Transfer Date/Time – The current system date and time is displayed, by default. Enter the Emergency Transfer Date and Time in dd/mm/yyyy hh:mm format or select the date using the Calendar function. The entered date must not be greater than system date and time. This is a mandatory field.

  4. Reason for Transfer – Select the Reason for Transfer from the list. The values are created in the Transfer Type function of IP Code Setup. This is a mandatory field.

 

Transfer To

  1. Nursing Unit – Select the Nursing Unit to transfer the patient from [?] function. Transfer to the same nursing unit is not allowed. This field is mandatory.

  2. Service - Select a Service from list. This field is mandatory.

  3. Sub-Service – Select the required Sub-service from the list. The listed values correspond to the selected Service.

  4. Request Bed Class Change – Select the check box to request for bed class change. This field is enabled only if the ‘Request for Bed Class Change’ check box is selected in the IP Parameter. Once the check box is selected, the system allows the change of Bed Class and Bed Type.

  5. Bed Class - The Bed Class associated to the patient is displayed. The value can be changed if the ‘Request for Bed Class Change’ check box is selected. This is a mandatory field.

  6. Bed Type - The Bed Type associated to the patient is displayed. The value can be changed if the ‘Request for Bed Class Change’ check box is selected. The values are displayed based on the Bed Class selected. This field is mandatory.

  7. Bed No – The Bed to which the patient has to be transferred is entered in alphanumeric format of eight characters maximum length or select the bed from the ‘Bed availability Chart’ by clicking on [?] function. The beds associated to the Bed Type and Bed Class is displayed.

  8. Room/Cubicle No – The room/cubicle number within which the selected bed is located is displayed.

  9. Specialty – If the Nursing Unit caters to multiple Specialties, a different Specialty can be selected by entering the Specialty or using the [?] function. This is a mandatory field.

  10. Attending Practitioner – Enter the name of the Practitioner or select the Practitioner from [?] function for the selected Specialty. This is a mandatory field.

  11. Medical Team - Select the Medical Team from the list. The values populated correspond to the selected practitioner.
