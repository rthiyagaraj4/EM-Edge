Removal of

This section is used to record details of any minor interventions performed and any items being removed during discharge which may have been used for the treatment of the patient during the stay of the patient.

  1. Item – List of all the items that have been removed as part of the discharge process is displayed. A check box against each Item is provided to mark that the item has been removed.

  2. Removed/Done By – Enter the name of the Practitioner who performed the intervention (removal of item) in alphanumeric format of 30 characters maximum length. This field is enabled only if the check box against the Item is selected.

  3. Date Time – Enter the Date/Time when the intervention (removal of item) was done in the format of dd/mm/yyyy hh:mm or select the date using the calendar function. This field is enabled if the check box against the Item is selected.

 

Properties

This section is used to record the details of any belongings or valuables that have been returned.

  1. Belongings Returned – Select the check box to indicate that all belongings of the patient that was held at the time of admission are returned.

  2. Date/Time - Enter the Date/Time as to when the belongings were returned in date format of dd/mm/yyyy hh:mm or select the date using the calendar function. This field is enabled if the check box against the ‘Belongings Returned’ field is selected.

  3. Remarks - Enter the remarks, if any, in alphanumeric format of 100 characters maximum length. This field is enabled if the check box against the ‘Belongings Returned’ field is selected.

  4. Valuables Returned – Select the check box to indicate that all valuables of the patient that were registered with the facility at the time of admission are returned.

  5. View Details - This field is hyperlinked and this hyperlink is displayed if any valuables of the patient were collected and a record of the same is maintained. On clicking the hyperlink, the system displays the details regarding the valuables collected.

  6. Returned By – Enter the name of the person who returned the valuables to the patient or select the name using the [?] function. This field is enabled if the ‘Valuables Returned’ check box is selected.

  7. Witness - Enter the Name of the person who was a witness to the return of the valuables to the patient or select the name using the [?] function. This field is enabled if the ‘Valuables Returned’ field is selected.

 

Transport

This section is used to record the details of the transport arranged for the patient as part of the discharge process.

  1. Transport Arranged – Select the check box to indicate that transportation is arranged for the patient.

  2. Date Time - Enter the Date/time as to when the transport has been arranged in the format dd/mm/yyyy hh:mm or select the date using the calendar function. This field is enabled if the check box against the ‘Transport Arranged’ field is selected.

  3. Remarks - Enter the Remarks, if any, in alphanumeric format of 100 characters maximum length. This field is enabled only if the check box against the ‘Transport Arranged’ field is selected.

  4. Recorded By – The name of the logged in person is displayed by default as the person recording the discharge checklist. This field cannot be changed.

  5. On – The date and time as to when the recording of the discharge checklist commenced is displayed. This field cannot be changed.
