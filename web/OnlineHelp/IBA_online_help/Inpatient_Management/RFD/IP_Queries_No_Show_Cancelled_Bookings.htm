Enter the following search criteria:

  1. Nursing Unit - Enter the Nursing Unit or select the Nursing Unit using [?] function.

  2. Preferred Date [From-To] - Enter the preferred Admission Date range in date format of dd/mm/yyyy or select it using the Calendar function.

  3. Specialty - Enter the Specialty or select the Specialty using [?] function.

  4. Booking Status - Select the status of Booking from the list. The values displayed are 'Both', 'No show' and 'Cancel'.

  5. Practitioner - Enter the Practitioner name or select the Practitioner using [?] function.

  6. Booking Type – Select the Booking Type from the list. The listed values are defined in the Booking Type function.

  7. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or click [?] function to select the Patient ID.

  8. Gender - Select the Gender of the patient from the list. The listed values are 'Male', 'Female' and 'Unknown'.

  9. Search Name with Local Lang – Select the check box to search the Patient name with local language.

  10. Name - Specify the name of the patient in the alphanumeric format of 40 characters maximum length. The query components and the legends pertaining to the patient name depend upon the set up done in MP Parameter under the 'Legal Names' Tab. The available components for the patient name are Title, First Name, Second Name, Third Name, Family Name and Suffix. The components that are made applicable for the facility appears as query criteria here.
