The Discharge Details Tab is displayed, by default.

 

Enter the following details:

 

  1. Authorizing Practitioner – Enter the Authorizing Practitioner in the alphanumeric format as specified or select the practitioner using [?] function. The practitioners listed in [?] function are filtered based on the Specialty of the Nursing Unit. This field is mandatory. The assigned practitioner is displayed as the Authorizing Practitioner, by default.

  2. Expected Discharge Date/Time – Enter the date in the format dd/mm/yyyy hh:mm. The date and time entered should be greater than or equal to system date and time. This is a mandatory field. The expected discharge date and time is displayed, by default. This displayed value is specified in Admit Patient function or Record Expected Discharge functions.

  3. Discharge Type - Select a Discharge Type from the list. This is a mandatory field. The list displays values defined in the Discharge Type function. Discharge Types are set for a Discharge Type Indicator as Absconded, At Own Risk, Deceased, Discharged/Home, Transfer to Other Hospital. If Discharge Type is selected as Deceased, An alert message ‘This Patient is marked as Deceased. Do you want to continue?’ is displayed. Click ‘OK’ to proceed or ‘Cancel’ to select a new value of Discharge Type.  

  4. Discharge Type values created with Maternal Death check box selected in Discharge Type function is displayed only for female patients.

  5. If a ‘No Show’ Patient is selected, the Discharge Type field displays values created with Indicators – Deceased and Transfer to Other Hospital.

  6. If a ‘Abscond’ Patient is selected, the Discharge Type field displays values created with Indicators – Absconded, At Own Risk and Deceased.

  7. If a ‘Leave Expiry’ Patient is selected, the Discharge Type field displays values created with Indicators – Absconded, At Own Risk and Deceased.

  1. Discharge Status - Select the Discharge Status from the list. Based on the Discharge Type selected the Discharge Status values are populated in the list. If the Discharge Type selected is Death, then the Discharge Status list displays only those values defined for the Discharge Status indicator Deceased. This is a mandatory field.

  2. Death Date/Time – Enter or select the death date and time of the patient in format of dd/mm/yyyy hh:mm. This field is displayed if a value with Type Deceased is selected as the Discharge Type. This field is mandatory.

  3. Post-Mortem Requested - Select the check box if post-mortem is requested for the deceased patient. The field is enabled only if the selected Discharge Type has the Indicator ‘Deceased’ selected during its creation.

  4. Remarks – Enter the remarks if any in the alphanumeric format of 100 characters maximum length.

 

MLC Details

  1. Medico Legal Case - Select the check box if the encounter is a Medico Legal Case.

  2. Police Report No. – Enter the Police Report Number in alphanumeric format of 30 characters maximum length. This field is enabled only if the Medico Legal Case check box is selected.

  3. Police Station – Enter the Police Station where the case has to be registered in alphanumeric format of 30 characters maximum length. This field is enabled only if the Medico Legal Case check box is selected.

  4. Police ID – Enter the Police ID in alphanumeric format of 30 characters maximum length. This field is enabled only if the Medico Legal Case check box is selected.

 

Informed Details

  1. Informed To – Select the valid contact to inform about the patient’s discharge from the list. The listed values are Next to Kin, First to Notify and Others. On selection of either ‘Next of Kin’ or ‘First to Notify’, [Patient Details] is enabled, allowing you to view these details if they had been entered during the registration process for the selected inpatient. On selecting ‘Others’ from the list, the [Patient Details] is disabled.

  2. Name – Enter the Name of the person informed in alphanumeric format of 200 characters maximum length.

  3. Date Time – Enter the informed date and time in the format dd/mm/yyyy hh:mm or select the date and time by clicking the calendar function.

 

Absconding Details

  1. Absconding – Select the check box if the patient has absconded. This is recorded for patients absconded from the facility or those patients that have gone on leave and not yet returned. This check box is enabled on selecting Discharge Type with a value created with Discharge Type Indicator ‘Absconded’ or ‘At Own Risk’ in the Discharge Type function of IP Code Setup.

  2. Date/Time – This field is enabled on selecting the Absconding check box. Enter the date and time in date format of dd/mm/yyyy hh:mm from when the patient is absconding. The entered date and time has to be greater than current system date/time and not be less than admission date/time. This field is mandatory.

  3. Informed Others - This field is enabled on selecting the Absconding check box. Enter the name of the person to whom absconding of the inpatient is reported. This field is mandatory.
