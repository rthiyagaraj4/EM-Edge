Enter the following details:

 

  1. Patient ID - Enter the patient ID from the lookup.

  2. Patient Name - Enter the patient name in the text field

  3. Episode Type - Select the episode type from the drop-down list. The list includes IP, OP, DC and EM. This field is mandatory.

  4. Encounter ID - Select the encounter ID from the lookup.

  5. Specialty - Select the specialty from the lookup.

  6. Payer Group - Select the payer group from the lookup. This field is mandatory.

  7. Payer - Select the payer from the lookup.

  8. Policy - Select the policy from the lookup.
