Enter the following search criteria:

 

  1. Nursing Unit - Select the Nursing Unit assigned to the patient using [?] function.

  2. Discharge Type – Select the Discharge Type from the list.

  3. Specialty - Select the Specialty associated to the patient using [?] function.

  4. Practitioner - Select the treating Practitioner using [?] function.

  5. Encounter ID - Enter the Encounter ID of the patient in numeric format of 12 digits maximum length.

  6. Gender - Select the Gender from the list. The listed values are ‘All’, ‘Male’, ‘Female’ and ‘Unknown’. ‘All’ is displayed, by default.

  7. Cancelled By - Select the Practitioner who cancelled the Discharge using [?] function.

  8. Cancelled Date [From-To] - Enter the Cancelled Date range of Discharge in date format of dd/mm/yyyy or select the Date using the Calendar function.

  9. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as defined in MP Parameter of the Master Patient Index module or click [?] function to select the Patient ID.
