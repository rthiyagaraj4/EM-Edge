Enter the following details:

 

  1. Booking Type – Select the Booking Type from the list for which the serial number is to be generated. The values contained are Block Bed, Block Bed on Discharge and Patient Transfer. This field is mandatory.

 

Booking serial details

  1. Next Serial No – Enter the Next Serial Number in numeric format of six digits maximum length. The next serial number entered is the next number to be generated by the system for the booking type. The next number specified is the first booking number generated by the system. Subsequently, this is automatically incremented by one by the system for every booking. The value entered must be greater than zero. This field is mandatory.

  2. Max Serial No – Enter the Maximum number in numeric format of six digits maximum length. The maximum serial number entered is the maximum number to be generated by the system for the booking type. This number cannot be less than the Next number. When the maximum serial number is reached, no further numbers are generated. At this stage, the System Administrator has to reset appropriately the Max Serial Number. This field is mandatory.
