Booking

The Booking tab is displayed, by default.

  1. Enter the Date of Procedure/Surgery or select the Date by clicking the Calendar function. On entering the Procedure/Surgery Date, the Preferred Admission Date/Time displays the date entered for Procedure/Surgery.

  2. Enter the number of days for preparation before Operation. If the Procedure/Surgery Date is entered and the Pre-Op Preparations Days specified, the Preferred Admission Date/Time is displayed automatically. If Pre-Op Preparations Days is specified after entering the Preferred Admission Date/Time, the system automatically calculates and displays the Procedure/Surgery Date.

  3. Enter the Preferred Date and Time for Admission in the dd/mm/yyyy hh:mm format or use the Calendar function to select the required date and time. Advance booking can be performed within the period specified in IP Parameter ('Max allowable period for advance bed booking [] days') from the current system date and time. This field is mandatory.

  4. Select a Medical Service Group from the list. Only those Medical Service Groups that are attached to Nursing Unit Types, which are in turn, attached to single/multiple Nursing Units are displayed. The selected Medical Service Group indicates the group under which the patient is being booked. This field is mandatory. For the selected Medical Service Group, if the maximum daily bookings or the maximum monthly bookings are specified, the system checks for the rules set during every booking. Once the values set are exceeded, subsequent bookings are not allowed.

Generating Booking Reference Number Manually

The Booking Reference Number is auto-generated or manually entered  based on the selected Medical Service Group.

If ‘Generate Booking Number’ check box is not selected for the desired Medical Service Group in the Medical Service Group function of IP Code Setup, the Booking Reference Number needs to be entered manually.

 

Enter the Booking Reference Number in alphanumeric format of 12 characters maximum length.

The system prompts you to enter the booking reference number by making the field entry as mandatory.

Auto Generating Booking Reference Number

The Booking Reference Number is auto-generated or manually entered  based on the selected Medical Service Group.

If ‘Generate Booking Number’ check box is selected for the desired Medical Service Group in the Medical Service Group function of IP Code Setup, the Booking Reference Number is generated automatically.

If the system generates booking reference number for the selected Medical Service Group, you are not allowed to enter the number.

Creating Booking for Registered/Unregistered Patients with Referral ID

  1. Enter the Referral ID of the patient in alphanumeric format of 14 characters or select the Referral ID by clicking [?] function to get the ‘Referral Search’ screen.

  2. Enter the referral details and click [Search] to search for a particular referral patient or click [Search] without entering the referral details in the fields provided to display all referral patients registered in the facility. The Referral Details of the referral patients are displayed.

  3. Click the Referral ID hyperlink and the patient details are displayed on the Create Booking screen.

  4. Click [Clear] to clear the entered data or [Close] to exit from the screen.

  5. On selecting the required Referral ID, the ‘Referral Search’ screen closes and control shifts back to the ‘Create Booking’ screen. The system displays patient details available for the selected Referral ID in the booking screen along with the preferred date/time of admission as mentioned while registering referral. Enter the other details in the ‘Create Booking’ screen.

On checking out an outpatient, if a referral has been registered for an inpatient location with the ‘Referred for’ value selected as ‘Booking’ the same is displayed in the result screen of ‘Referral Search’.

Booking for Unregistered patients is allowed If ‘Allow Booking for Unregistered Patients’ field is selected in the IP Parameter function.

Creating Booking for Registered/Unregistered Patients without Referral ID

  1. Click the [?] function to get the ‘Referral Search’ screen.

  2. Click [Register Patient Referral] in the Referral Search screen to register the referral patient. On successfully registering the referral, the patient details are displayed on the Create Booking screen. The Referral ID generated is based on the value set in the Referral ID Numbering function of the Application Masters module.

  3. Click [Clear] to clear the entered data or [Close] to exit from the screen.

Booking for Unregistered patients is allowed if ‘Allow Booking for Unregistered Patients’ field is selected in the IP Parameter function.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

If ‘Allow Booking for Unregistered Patients’ field is selected in the IP Parameter function, booking for Unregistered patients is allowed.

If ‘Allowed Booking’ is set as ‘Referral’ in the ‘IP Parameter’ function, the ‘Create Booking’ screen allows booking only for referral patients.
