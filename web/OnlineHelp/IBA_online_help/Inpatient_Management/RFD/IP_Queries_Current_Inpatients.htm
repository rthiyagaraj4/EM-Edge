Enter the following search criteria:

 

  1. Nursing Unit - Select the Nursing Unit using [?] function.

  2. Admission Date [From-To] - Enter the Admission Date range in date format of dd/mm/yyyy or select the Date from the Calendar function.

  3. Specialty - Select the Specialty using [?] function.

  4. Practitioner - Select the Practitioner using [?] function.

  5. Service - Select the Service using [?] function. The services are defined in the Application Masters module.

  6. Sub-service - Select the appropriate Sub-service from the list. The listed Sub-services correspond to the selected Service.

  7. Encounter ID - Enter the Encounter ID in numeric format of 12 digits maximum length.

  8. Incl Disc Patients - Select the check box to include the discharged patients in the result list or clear the check box.

  9. No of Days - This field is enabled only when ‘Incl Disc Patients’ check box is selected. Enter the number of days in numeric format of one digit maximum length. The result set contains those patients discharged within the specified period.

  10. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select using [?] function.

  11. Patient Name - Enter the Patient Name in alphanumeric format of maximum 15 characters length. The system displays the patients list that contains the string entered in the field. If any string is entered in the filed to search a particular patient, the system displays all patient names that contain the entered string.
