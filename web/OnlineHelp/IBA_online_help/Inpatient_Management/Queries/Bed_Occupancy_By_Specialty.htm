Bed Occupancy By Specialty

From this function you can  Query about bed occupancy statistics for each specialty and nursing unit.

This query function provides bed occupancy statistics for each specialty and nursing unit. The query result provides details on the bed status for each specialty and nursing unit. The user can know the number of beds available for each specialty and nursing unit, sthe total beds occupied, number of beds vacant and beds blocked.

Querying for Bed Occupancy by Specialty

From IP Queries> Statistics, select the Bed Occupancy By Specialty function.

The Bed Occupancy By Specialty screen gets displayed.

  1. Enter the details in the fields to query the bed occupancy by specialty. <Related Field Description>

  2. Click [Search] to view the Bed Occupancy details by Specialty or [Reset] to clear the entered search criteria.

  3. The details are grouped under each specialty. The nursing units attached to each specialty are displayed with details of the Total beds, Occupied Beds, Blocked Bed, and Vacant Beds.

  4. Click [Clear] to clear the result screen.
