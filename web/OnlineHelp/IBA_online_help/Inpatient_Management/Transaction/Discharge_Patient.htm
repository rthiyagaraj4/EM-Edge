Discharge Patient

From this function you can Record the Discharge details and Discharge the Inpatient.

On discharging an Inpatient, the bed occupied by the Inpatient is released. The bed is blocked for a specified period based on the IP Parameter setup. To release blocked beds, you have to invoke the Release Bed function and release the bed explicitly. The Discharge Type for deceased patients is entered as ‘Deceased’. On discharging a Deceased Patient, the Patient Record is updated in the Master Patient Index module as deceased.

While discharging a patient, the system checks for medication to be provided to the patient (as defined while preparing the discharge advice). In case discharge medication is required, the system checks if the ‘Pharmacy Management’ module is installed and necessary transactions are processed. The Inpatients can be discharged if the discharge advice is recorded.

Discharging a Patient

From the IP Transactions > Discharge, select the Discharge Patient function.

The Discharge Patient screen gets displayed.

  1. Click [Search] to display the list of all the Inpatients with discharge advice prepared or enter the search criteria and click [Search] to select a specific Inpatient with discharge advice prepared. Click [Clear] to reset the records displayed in the result area.

    Enter the following query criteria to select the required patient record:

    1. Nursing Unit – Select the Nursing Unit using the [?] function. The Nursing Units filtered in the lookup function based on the Locations associated to the Operator Station of the logged in user.

    2. Admission Date From-To – Enter the Admission Date range in the format of dd/mm/yyyy.

    3. Specialty – Select the Specialty using the [?] function.

    4. Practitioner – Select the Practitioner using the [?] function.

    5. Encounter ID – Enter the Encounter ID in numeric format of 12 digits maximum length.

    6. Patients with Discharge Advice - Select the check box to filter and list inpatients with Discharge Advice. If the check box is cleared, the Discharge Patient function will filter and list inpatients, with and without Discharge Advice. If the value for ‘Display Discharge Patient List’ in the IP Parameter function is selected as ‘Only Patient with Discharge Advice’, the check box is selected, by default and will filter and list inpatients with Discharge Advice.

    7. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the Patient ID using the [?] function.

  2. On searching, Inpatient Details are displayed based on the entered criteria.

  3. Click ‘+’ hyperlink to view the Current Encounter Details.

  4. Click the Encounter ID hyperlink to display the Discharge Patient screen. The screen displays Encounter Details, Nursing Unit Details. You can enter the patient weight if required in the ‘Weight on Admission field’.  An alert message 'The Patient is Deceased' is displayed, if the selected patient for discharging is deceased. This message is displayed only for patients whose Discharge Type in 'Prepare Discharge Advice function is selected as 'Deceased'.

  5. Enter the details in the mandatory (*) fields of Discharge Details tab and Additional Discharge Details tab.

Discharge Details

Additional Discharge Details

  1. Click [Stay Details] to view the Stay Details.

  2. Click [Discharge Advice] to view the Discharge Advice details or [Close] to exit from the Discharge Advice screen.

  3. Click [Confirm Discharge] to save the record and discharge the patient or [Abort Discharge] to cancel the process.

  4. If a person is accompanying the patient, the Accompanying Person Details screen is displayed. The details displayed are Accompanying Person ID, Name of the accompanying person and Accompanying Period (from/to). Click [Checkout] to check-out the accompanying person or [Close] to exit from the screen.

  5. If internal reports are defined, the Internal Reports screen gets displayed.

  6. Select the report name, Required and number of Copies check boxes.

  7. Click [OK] to generate Internal Reports or [Cancel] to exit from the screen.

You can generate the following internal reports:

    • Appointment Slip

    • Patient ID Card

    • Medical Record Release Form

    • Medical Record Request Form

    • Operating Site Identification

    • Order Summary for Discharge
