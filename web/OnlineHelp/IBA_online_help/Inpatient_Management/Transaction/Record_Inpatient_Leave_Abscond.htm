Record Inpatient Leave/Abscond

From this function you can Record Leave or Abscond details of an Inpatient.

Record Inpatient Leave/Abscond function lists all the current Inpatients in Nursing Units, which belong to the Operator Station that the logged in user has been assigned to. The system validates if the logged in user has access to record leave or abscond for an Inpatient, which is also defined in the Operator Station.

If the Nursing Unit, from which the Inpatient is sent on leave, has the flag ‘Release Bed on Leave’ selected, ‘Release bed on Leave’ flag in this function is enabled. You can release the bed or retain the bed for the Inpatient. There is also a provision to assign a pseudo bed to the Inpatient in case; bed is released on granting leave. The duration of leave granted cannot be greater than the maximum duration set up for leave setup in the IP Parameter.

Recording Inpatient Leave/Abscond Details

From the IP Transactions > Leave Pass, select the Record Inpatient Leave/Abscond function.

The Record Inpatient Leave/Abscond screen gets displayed.

  1. Click [Search] to display the list of all Inpatients or enter the search criteria and click [Search] to select a specific Inpatient. Click [Clear] to reset the records displayed in the result area.

    Enter the following query criteria:

    1. Nursing Unit – Select the Nursing Unit using the [?] function.

    2. Admission Date From-To – Enter the Admission Date range in date format of dd/mm/yyyy.

    3. Specialty – Select the Specialty using the [?] function.

    4. Practitioner – Select the Practitioner using the [?] function.

    5. Encounter ID – Enter the Encounter ID in numeric format of 12 digits maximum length.

    6. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the Patient ID using the [?] function.

  2. On searching, Inpatient Details are displayed based on the entered criteria.

  3. Click ‘+’ hyperlink to view the Current Encounter Details.

  4. Click the ‘Encounter ID’ hyperlink to display the Record Inpatient Leave/Abscond screen.

  5. Enter the details in the mandatory (*) fields. <Related Field Description>

  6. Click [Record] to record the leave details, [Clear] to clear the entered data or [Close] to exit from the screen.
