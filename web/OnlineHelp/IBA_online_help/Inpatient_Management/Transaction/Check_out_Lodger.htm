Check-out Lodger

From this function you can Check-out lodgers who have checked-in and Enquire about the details of lodgers overstaying.

The lodging charges applicable are calculated. The bill is generated and printed while checking out if the Patient Billing Manager module is installed.

Checking out Lodger

From the IP Transactions > Lodger/Accompanying Person, select the Check out Lodger function.

The Check-out Lodger screen gets displayed.

  1. Click [Search] to display the list of all lodgers or enter the search criteria and click [Search] to select a specific lodger. Click [Clear] to reset the records displayed in the result area.

    Enter the following query criteria:

    1. Nursing Unit – Select the Nursing Unit using [?] function.

    2. Lodger Person ID – Enter the ID of the Lodger in alphanumeric format of length as specified in the MP Parameter or select using [?] function.

    3. Checked In Date (From-To) – Enter the check-in date range of the Lodger in date format of dd/mm/yyyy or select the date from the calendar function.

    4. Gender – Select the Gender of the Lodger from the list.

  2. On searching, Lodger Details are displayed based on the entered criteria.

  3. Click ‘Lodger Name’ hyperlink to display the Check-Out Lodger screen. The screen displays the Lodger Details and Stay Details recorded while checking in the lodger.

  4. Click [Contact Details] to view the contact details of the lodger.

  5. Enter or select the Check-out Date and Time of the lodger using the calendar function. The current system date and time is displayed, by default. The entered value must not be greater than current system date and time and lesser than the checked-in date and time.

  6. Click [Release Lodger] to check the lodger out or [Close] to exit from the screen.
