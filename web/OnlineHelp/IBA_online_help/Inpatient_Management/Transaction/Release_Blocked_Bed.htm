Release Blocked Bed

From this function you can Release a Blocked Bed manually before the specified blocking period.

Blocked Beds are either released automatically by the system or manually. The system releases the Blocked Beds after the specified blocking period has elapsed. All the beds blocked in ‘Block Bed’, ‘Transfer Patient’ and ‘Discharge Patient’ functions are listed.

Releasing a Blocked Bed

From the IP Transactions > Bed Maintenance, select the Release Blocked Bed function.

The Release Blocked Bed screen gets displayed.

  1. Click [Search] to display all the blocked bed details or enter the search criteria and click [Search] to select specific blocked beds. Click [Clear] to reset the records displayed in the result area.

    Enter the following query criteria:

    1. Nursing Unit – Select the Nursing Unit using [?] function to display the blocked beds within the Nursing Unit. All the Nursing Units associated to the operator station to which the logged in user has access are listed.

    2. Bed Class - Select the Bed Class using [?] function or enter the required bed class.

    3. Blocked Through - Select the Blocked through option from the list. The values available are Blocking Function, Transfer Function and Discharge Function.

    4. Blocked From Date - Enter the ‘From’ date for blocking the bed in date format of dd/mm/yyyy.

    5. Until - Enter the blocked period ‘Until’ in date format of dd/mm/yyyy. The block period ‘Until’ is displayed based on the value set in IP Parameter function.

    6. Block Type – Select the type of blocking from the list. This lists all the Blocking Types defined in the ‘Blocking Type’ Master function.

    7. Include Reserved Beds – Select the values from the list. The listed values are 'Blocked Beds',  'Reserved Beds' and 'All'. The value 'Blocked Beds' is displayed, by default.

  2. On searching, Blocked Bed details are displayed based on the entered criteria.

  3. Click Remarks hyperlink to view the Bed Blocking remarks.

  4. Select the check box against the desired blocked bed to release the blocked bed.

  5. Click [Apply] to release the blocked bed or [Reset] to clear the entered data.

 

You must have rights defined in the Operator Station attached to the User to release blocked bed.

The beds that are reserved for the patient are released while discharging the patient.
