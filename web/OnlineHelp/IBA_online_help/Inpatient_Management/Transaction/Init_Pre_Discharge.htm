Initiate Pre-Discharge

From this function you can Initiate the pre-discharge activity for an inpatient and notify the nurse in-charge that they can start the discharge activities for the patient.

The practitioner needs to indicate that a patient is ready to be discharged. Using the Initiate Pre-discharge function, the practitioner can indicate the patients who are ready to be discharged for the nurses’ knowledge. The nurses in-charge of the can then start the actual discharge activities such as preparing discharge advice, reports and such.

 

The in-patient details will be available both in the Initiate Pre-Discharge screen and Prepare Advice screen. However, if you have selected the Enable Initiate Pre-Discharge check box in the IP Parameter screen, unless the practitioner initiates the pre-discharge activity, the nurses cannot prepare the discharge advice; an error message “Pre-discharge not initiated; cannot proceed” will be displayed.

 

This function will be available only if you select the Enable Initiate Pre-Discharge check box in the IP Parameter screen. Else, an error message “This function is not applicable since Enable Pre-Discharge is unchecked in IP Parameter” will be displayed.

Initiating Pre-Discharge

From the IP Transactions > Discharge, select the Initiate Pre-Discharge function.

The Initiate Pre-Discharge screen gets displayed.

  1. Click [Search] to display the list of all the Inpatients or enter the search criteria and click [Search] to select a specific Inpatient. Click [Clear] to reset the records displayed in the result area. On searching, Inpatient Details are displayed based on the entered criteria.

    Enter the following query criteria:

    Nursing Unit – Enter the Nursing Unit in alphanumeric format of 12 characters maximum length or select the Nursing Unit using [?] function. The Nursing Units that are displayed in the lookup are filtered based on the locations associated with the Operator Station of the logged in user.

    Admission Date From-To – Enter the Admission Date range in date format of dd/mm/yyyy or select the date range from Calendar function.

    Specialty – Enter the Specialty in alphanumeric format of 15 characters maximum length or select the Specialty using [?] function.

    Practitioner – Enter the name of the Practitioner in alphanumeric format of 15 characters maximum length or select the practitioner using [?] function.

    Encounter ID – Enter the Encounter ID in numeric format of 12 digits maximum length.

    Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the Patient ID using [?] function.

  2. Select the check box Pre-Discharge Initiated for the required patient.

  3. Click [Apply] to save the changes.

 

1. Even when there is pending orders, the user will be able to” Initiate Pre-Discharge” and this will be applicable for Inpatients and Daycare Patients.

2. You can initiate pre-discharge without assigning bed. The validation to assign bed will be triggered in “Prepare Discharge Advice’”

3. In CA, the Initiate Pre-Discharge will be available in “Desktop” and it will not be available in “Patient Chart”.
