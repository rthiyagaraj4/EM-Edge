Discharge Status

From this function you can Create various Discharge Statuses for a Discharge Status Indicator and Query for a desired Discharge Status record.

Discharge Status is recorded for an inpatient while preparing the Discharge Advice. A Discharge Advice is used to notify the discharge desk that a patient is ready for discharge. Only those inpatients with prepared discharge advice can be discharged.

Creating Discharge Status

From IP Setup > IP Code Setup > Discharge Status, select the Discharge Status function.

The Discharge Status screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Discharge Status created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Discharge Status records.
