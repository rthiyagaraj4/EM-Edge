MP Audit Trial Parameter

From this function you can Define rules for tracking Demographics/Status changes.

Demographics and status of patients are vital information for the enterprise. Changes to these details are to be updated regularly and maintained. The system maintains log records on changes performed to the elements set in this function.

Setting up Audit Trial Parameter

From the MP Admin Setup > MP Admin Setup, select the MP Audit Trial Parameter function.

The MP Audit Trial Parameter screen gets displayed.

  1. Select the required Audit Trail Parameter settings. <Related Field Description>

  2. Click [Apply] to save the MP Audit Trial Parameter settings or [Reset] to clear the entered data.
