MP Access Rights

From this function you can Define access rights to view patient photo and patient financial details.

An enterprise can restrict viewing patients photograph and patients financial details to Enterprise Management users. This is achieved by assigning rights to view photograph and patients financial details for each Responsibility. Users assigned to these Responsibilities only can view the photograph and patient financial details.

Defining Access Rights

From the MP Admin Setup > MP Admin Setup, select the MP Access Rights function.

The MP Access Rights screen gets displayed.

  1. Enter the Responsibility Name to list the required Responsibility. This is a mandatory field.

  2. Select the value 'Starts with', 'Ends with', and 'Contains' from the Criteria field.

  3. Click [Search] to display the Responsibilities based on the entered criteria.

  4. Select the 'View Patient Photo' check box beside the required Responsibility to define access rights.  

  5. Select the 'View Patient Financial Details' check box beside the required Responsibility to define access rights.

  6. Click [Apply] to save the MP Access Rights or [Reset] to clear the entered data.
