Patient / Encounter Context Banner

From this function you can Customize Patient Banner and Encounter Context Banner.

You can configure components in the patient banner area and define the display order and formatting in the patient banner. This function provides the option to edit or change white spacing on banner and preview the changes before saving the patient banner. You can customize the display text and length of the legend text display and the tool tip. The customized Patient Banner values are displayed throughout Enterprise Management modules. The Patient Banner is displayed with unique color indicator for active patients and deceased patients.

Customizing Patient/Encounter Context Banner

From the MP Admin Setup > MP Admin Setup, select the Patient/Encounter Context Banner function.

The Patient/Encounter Context Banner screen gets displayed.

  1. Select the Banner Type from the list. The listed values are Patient Context Banner and Encounter Context Banner.

  2. On selecting the Banner Type the values are displayed. Specify the rules for Line 1 and Line 2 in the Patient Banner.

Values

Enter the following values:

  1. Select - Select the check box to display the item in the Patient Banner.

  2. Display Order – The default display order is displayed. Select the ‘Display Order’ hyperlink to move the elements to the respective order.

  3. Field Name – The Name of the field is displayed by default. Click on the Field Name hyperlink to move the item to Line 1 or Line 2.

  4. Actual Length - The actual maximum length of the field is displayed by default. This value is for reference only in this screen.

  5. Display Length - You can set the character length which needs to be displayed in the patient banner. Enter the Display Length in numeric format of three digits maximum length.

Icons are displayed for specific Field Names which will be displayed in the Enterprise Management modules if selected.

Legend

Enter the following legends:

  1. No. of Spaces Before Text – Enter the Number of Spaces between two field names in numeric format of two digits maximum length.  

  2. Display Text – Enter the legend value to display against the selected Field Name. You cannot specify the display text for the icons.

  3. Tool Tip - Enter the values to display as ToolTip text against the selected Field Name.

  1. Click [View Default Banner] to reset the entered values and retain the default banner values.

  2. Click [Preview Banner] to invoke the Preview Banner screen.

    On customizing the Patient Banner values, you can preview the customized Patient Banner. You can preview the actual banner as it will be displayed in the application before saving it to the system. Click [Close] to exit from the Preview Banner screen.

  3. Click [Apply] to save the record or [Reset] to clear the entered details.
