First to Notify

 

  1. Name - Enter the Name of the person in alphanumeric format of 30 characters maximum length.

  2. Relationship - Select the Relationship the patient shares with the person from the list. The values are listed from Relation ship function.

  3. Citizen ID – Enter the Citizen ID of the person in the required format as specified in MP Parameter. To copy Citizen ID, click on the hyperlink copy next of kin.

  4. Other Alt ID - Select the Other Alternate ID from the List. The listed values are ‘Previous HN’, ‘Govt No’, ‘Driving License No’ and ‘Expatriate No’.

  5. Occupation - Enter the Occupation of the person in alphanumeric format of 30 characters maximum length.

  6. Address Line (1 to 4) – Enter the Address details in alphanumeric format of 30 characters maximum length in each line. The legends are set in System Manager module.

  7. Area - Enter the name of the Area in alphanumeric format of 30 characters maximum length or select it using the function [?]. The field legend is set in System Manager module.

  8. Town - Enter the name of the Town in alphanumeric format of 30 characters maximum length. The entered name must be listed under the selected area. If the selected area has more towns attached to it, select the required town using the function [?]. The field legend is set in System Manager module.

  9. Region – Enter the Region in alphanumeric format of 30 characters maximum length or select it using the function [?]. The area must correspond to the entered region.  The field legend is set in System Manager module.

  10. Postal Code - Enter the Postal Code in alphanumeric format of eight characters maximum length. The entered value must correspond to the selected town. The user can select the required code using the function [?].The field legend is set in System Manager module.

  11. Country - The Country name gets automatically populated when the Region is selected.

  12. Home Phone No – Enter Residence Telephone Number of the person in alphanumeric format of 20 characters maximum length.

  13. Mobile No - Enter Mobile Number of the person in alphanumeric format of 20 characters maximum length.

  14. Office Phone No - Enter Office Telephone Number of the person in alphanumeric format of 20 characters maximum length.

  15. Email – Enter the Email address of the person in alphanumeric format of 50 characters maximum length.

  16. Copy Next of Kin – Click on ‘Copy Next of Kin’ hyperlink to copy address details of the patient’s Next of Kin.

  17. Copy Patient Address - Click on ‘Copy Patient Address’ hyperlink to copy address details of the Patient’s First to Notify.

The field legends for Address 1-4, Town, Area, State, and Postal Code are defined in Site function in System Manger module.
