Enter the following details to create Patient File:

 

  1. File Storage Locn - Select the file storage location from the list box. The permanent file storage locations of the logged-in facility are available in the list box. This field is mandatory. If only one File Storage (FS) Location exists for the logged in facility, then all the files created are assigned to this FS location. If more then one FS Location exists, then this field is enabled and the list box contains all these FS locations. Select the required location and assign the patient file. This field appears if Patient File Tracking  module is installed and File Storage Location is defined.

  2. File Type – Select the file type to be associated from the list. The listed types are defined in 'File Type' function. This function is mandatory. The file type defined should be associated to Storage Location using the function ‘File Type for File Storage Location’ in Patient File Tracking module.

  3. Old File No. – Enter the old file number, if exists for the patient. This field is displayed, if 'Prompt for Old MRN' is checked for the MR Section attached to the selected Patient Series.

 

The Internal Reports screen displays the following details:

 

The patient name in English, age of the patient, gender and the Patient ID

  1. Report Name – Displays the appropriate report name.

  2. Required – Check the box to select the report for printing.

  3. Copies – Displays ‘1’ as number of copies. Enter the number of copies required.
