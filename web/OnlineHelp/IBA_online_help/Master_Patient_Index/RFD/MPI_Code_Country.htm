Enter the following details to create a Country record:

 

  1. Code – Enter the Country code in alphanumeric format of four characters maximum length. The Country code should be unique and cannot be changed after creation. This field is mandatory.

  2. Long Name - Enter long name of the Country in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Name - Enter short name of the Country in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Citizenship - Enter description of the Citizenship in alphanumeric format of length 30 characters maximum length. This field is mandatory.

  5. Enabled - Select the check box to enable the record. Only enabled records are available in the transaction functions.

  6. Telephone Code – Enter the telephone code of the country in the text field. This field is mandatory. This is applicable for users in Ml-MMOH sites.
