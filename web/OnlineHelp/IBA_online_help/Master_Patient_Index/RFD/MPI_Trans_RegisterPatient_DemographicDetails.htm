Enter the following details:

 

  1. Patient Series - The default value appears based on the settings in MP Parameter. All the patient series defined in the function 'Patient Series' are listed. Select the required value from the list. Refer to the table below for default value of Patient Series. On selecting the Patient Series, you can capture the mandatory fields which is controlled internally.

 

Value for DEFAULT in MP Parameter (under Patient Numbering tab > Patient Series)

Default Patient series in Register Patient function

General

Patient Series specified for the field 'General' in MP Parameter is displayed as default. The list contains all the patient series defined in 'Patient Series' function. Select the required series.

Alternate ID 1

Patient Series specified for the field 'With Alternate Series' in MP Parameter is displayed as default. The list contains all the Patient Series defined in 'Patient Series' function. Select the required series.

National ID

Patient Series specified for the field 'With National ID' in MP Parameter is displayed as default. The list contains all the Patient Series defined in 'Patient Series' function. Select the required series.

 

Patient ID generation is based on the selected Patient Series. If the system automatically generates Patient ID for the selected series, you are not allowed to enter the Patient ID.

 

  1. Preferred Facility – Select the Facility Preferred by the patient from the list. The logged-in facility is the default value. All the facilities of the enterprise are listed.

  2. NRIC - Enter the National Registration Identity Card (NRIC) number of the patient. This field is available only when the following setting is done in the ‘MP Parameter'.   

    • Clear the 'Invoke Routine' check box.

    • Disable 'Check Digit Scheme' drop-down list.

    • Select the 'External Interface Required' check box.

On entering the NRIC number, the following fields: Government No, Expiry Date, Prefix, First Name, Gender, Date of Birth, Residency, Nationality, Race, Religion, Address1, Address 2, Address 3 and Address 4 are populated automatically.

 

Note: Users in UAE, click the button 'M' available next to the field Emirates ID. The screen gets populated with the patient’s information.

 

  1. National ID - Enter the National ID in the numeric format as specified in MP Parameter. The displayed field legend corresponds to the set 'Prompt' in MP Parameter.  

 

The National ID field is enabled by default irrespective of the user access rights in MR if the National ID has not been recorded earlier. There by, you can record the National ID.  

In case, if the National ID has already been recorded and if you try to update it, then the National ID field will be enabled only when 'Change National ID' check box is selected in User Access Rights screen.  Else, you cannot update the national ID.

 

  1. Form No 60 – Select the value from the drop-down list. The listed values are Yes and No.  By default, the value ‘No’ is selected.

    • If the Form No 60 drop-down list is selected as ‘No’, the PAN Card field gets enabled and it is not mandatory.

    • If the Form No 60 drop-down list value is selected as ‘Yes’, the PAN Card field is disabled.

  2. Other Alt No - Enter the Other Alternate No from the List.

  3. Previous HN – Enter the Previous Hospital Number in alphanumeric format of maximum length as specified in the MP Parameter and enter the expiry date of the Previous HN.

  4. Government No - Enter the Government Number in alphanumeric format of maximum length as specified in the MP Parameter and enter the expiry date of the Government number.

  5. Driving License No. - Enter the Driving License Number in alphanumeric format of maximum length as specified in the MP Parameter and enter the expiry date of the Driving License number.

  6. PAN Card – Enter the PAN Number of the patient in alphanumeric format. This field gets enabled only if the ‘Form No 60’ is selected as ‘No’ and it is not mandatory.  The ‘Alternate ID 3’ is configured as the PAN Card.

The ‘Form No 60’ and ‘PAN Card’ fields are applicable only for users in India. The fields ‘Form No 60’ or ‘PAN Card’ can be recorded  for all patients irrespective of whether 'Residency' is selected as 'Citizen' or ‘Non-Citizen’.  

  1. Expatriate No. - Enter the Expatriate Number in alphanumeric format of maximum length as specified in the MP Parameter and enter the expiry date of the Expatriate number.

  2. Membership - Select the Membership from the list. The listed values are Organization Member, Related Member - Family, Related Member - Non Family, Non-Organization Member. It is mandatory to select the Alternate ID as set in the field 'Alternate ID Type for Membership ID' in MP Parameter function. This field is displayed if 'Maintain Organization Membership' check box is selected in the MP Parameter function.

  3. Organization - Select the Organization using [?] function. The Organization Sub Unit is displayed on selecting the Organization. If more than one Sub Unit is associated to the Organization, select the Sub Unit from the list.

  4. Head of Family Patient ID - This field is displayed based on the ‘Maintain Family Link’ set in MP Parameter. If family link is to be maintained, enter the Head Patient ID in the required format as specified in the MP Parameter, or click [?] function to select the required Head Patient ID. The Head patient name is displayed next to the field.

  5. Relationship - The Relationship is displayed as set in the 'Relationship for Organization Member' field in the MP Parameter function on selecting the Membership as 'Organization Member'. Select the Relationship to the displayed Head of Family. This is a mandatory field.

  6. Relationship Level - This field is displayed on selecting the Relationship with a level defined in the Relationship function. Select the Relationship Level using [?] function. The Relationship Levels defined in the Relationship Level function is displayed. This is a mandatory field.

  7. Title - Select the Prefix from the list. The listed values are defined in the function 'Name Prefix'.

  8. First Name - Enter the First Name in alphanumeric format as specified in MP Parameter.

  9. Second Name - Enter the Second Name in alphanumeric format as specified in MP Parameter.

  10. Third Name - Enter the Third Name in alphanumeric format as specified in MP Parameter.

  11. Family Name - Enter the Family Name in alphanumeric format as specified in ‘Legal Names’ tab in MP Parameter. This is applicable irrespective of whether the 'Accept Patient Name in Multi parts' check box, Suffix or Prefix check box is selected or not. But if any other Name component check box is selected other than the ‘Family Name’ check box  in MP Parameter, the Family Name field size will be changed to 20 characters.  This is a mandatory field.

  12. Suffix - Select the Suffix from the list provided. The values are listed from Name suffix function.

  13. Gender - Select the Gender of the Patient from list. The listed values are Male, Female and Unknown. This is a mandatory field. Based on the routine defined for the National ID in MP Parameter, the gender is displayed automatically on entering the national ID.

  14. Y/M/D - Enter the Age of the patient in years, months and days in the respective fields. If National ID is entered, the age is calculated and displayed automatically based on the age validation logic defined in the routine associated with the National ID.

  15. Age & Birth Date & Time – Select the check box and enter the Birth date and time in yyyy/mm/hh format. If the age of patient is entered in years, months and days and not through the calendar function, the system identifies the age of the patient as estimated. If the age of the patient entered while registering is an estimated age, an estimated age indicator (Estd.) is displayed in Patient Context Banner and Patient Encounter Banner across all modules in the functions, where Patient Banner and Patient Context Banner are displayed. If you change the age of the patient using the Calendar function in the Change Patient Details function, the indicator is not displayed in Patient Context Banner and Patient Encounter Banner. On entering the birth date, the system calculates the Patient's age and displays it in the Age (Y/M/D) fields. If National ID is entered, the date of birth is displayed automatically based on the age validation logic defined in the routine associated with the National ID or select the date of birth using the 'Calendar' function. This is a mandatory field.

If the hours in 'Display New Born Age in Hrs When age is less than or equal to [] Hours' is recorded, the age for the new born is displayed in hours and the time is displayed in the 'T' text box.

  1. Blood Group - Select the Blood Group from the list. The listed values are defined in the Medical Records module.

  2. RH(D) Factor - Select the RH (D) Factor of the Patient from the list. The listed values are created in Rhesus Factors function in Terminology & Coding Management  module.

  3. Known Allergy - Select the Known Allergy of the Patient from the list. The listed values are Yes and No. If you select Yes from the list, it displays the Known Allergy recorded in the Patient Banner as Known Allergy. If you select No, it displays No Known Allergy. If Known Allergy is recorded as No in the Patient Registration Screen and in Allergy recording Screen, the icon is disabled. If Known Allergy is recorded as Yes in the Patient Registration screen and in Allergy Recording Screen, the icon is displayed as enabled.

  4. Birth Place - Enter the Birth Place of the Patient in alphanumeric format of 30 characters maximum length, or use the Calendar function to select the required birthplace. If National ID is entered, the birthplace is displayed based on the birthplace logic defined in the routine associated with the National ID. If the required birthplace is not available in the search list, enter the birthplace in alphanumeric format of 30 characters maximum length in the text box provided next to the [?] function.

  5. Marital Status - Select the Marital Status of the Patient from the list. The listed values are defined in the function 'Marital Status'.

  6. Residency – The residency of the patient is automatically validated by the system on selecting the Nationality of the patient (other than the default nationality) at registration. The system automatically validates the Nationality on selecting the Residency and the ‘Nationality’ of the patient is changed from default value to other value. Select either 'Citizen' or 'Non-citizen'.

  7. Status - Select either 'Legal' or 'Illegal' as citizen status. By default, 'Legal' is selected. If residency is set as 'Citizen', 'Legal' is the only option available. If the residency is set as 'Non-Citizen', the status can either be 'Legal' or 'Illegal'.  

  8. Patient Category - Select the Category of the Patient from the list. The listed values are defined in the function 'Patient Category' and the patient category must be associated to the Membership using ‘Pat Cat by Org. Membership’ master function.

  9. Nationality - The default Nationality set in MP Parameter is displayed. Select the required value using the [?] function. If National ID is entered, the default nationality set in MP Parameter is displayed and disabled. This is a mandatory field.

  10. Race – Select the Race from the list. The listed values are defined in 'Race' function. This is a mandatory field if 'Race Required During Registration' check box is selected in the Others Tab of MP Parameter function.    

  11. Ethnicity - Select the Ethnicity from the list. The listed values correspond to the selected race.

  12. Religion - Select the Religion of the Patient from the list. The listed values are defined in the function 'Religion'.

  13. Language - Enter the Language in alphanumeric format of 20 characters maximum length or select the Language using [?] function.

  14. Sub Department - Enter the sub department in alphanumeric format.

  15. Alias Name - Enter the Alias Name of the Patient in alphanumeric format of 30 characters maximum length.

  16. Father Name - Enter the Father Name of the Patient in alphanumeric format of 12 characters maximum length.

  17. Mother Name - Enter the Mother Name of the Patient in alphanumeric format of 12 characters maximum length.

  18. Spouse – Enter the Spouse Name of the patient in alphanumeric format of 20 characters maximum length.

  19. Remarks – Enter the remarks of the patient in alphanumeric format of 200 characters maximum length, if any.

  20. Registration Date and Time – The system Date and Time is displayed by default. You can select the required Registration date and time by clicking Date picker. The Registration Date and Time cannot be lesser than the Birth date and time. Similarly, the Birth date and time cannot be greater than the registration date and time.

  21. Consent for MyHIX – Select ‘Yes’ value from the drop-down list if the patient gives consent to share information to MyHIX System. Else, select the value ‘No’. The default value is ‘Yes’.  Once the patient is registered with “Consent for MyHIX” recorded as Yes, the 'Patient Registration message’ is sent from EM to MyHIX system and the MyHIX system sends response message with the MyHIX global ID.

The fields displayed in the screen may vary depending on the MP Parameter Setup.
