Employment Details

 

Employer Details

 

  1. Organization Name - Enter the Organization Name in alphanumeric format of 30 characters maximum length.

  2. Contact Person Name - Enter Contact Person Name in the organization. Specify the name in alphanumeric format of 30 characters maximum length.

  3. Address Line (1 to 4) – Enter the Address details in alphanumeric format of 30 characters maximum length in each line. The legends are set in System Manager module.

  4. Town - Enter the name of the town in alphanumeric format of 30 characters maximum length. The entered name must be listed under the selected area. If the selected area has more towns attached to it, select the required town using the function [?]. The field legend is set in System Manager Module.

  5. Area - Enter the name of the Area in alphanumeric format of 30 characters maximum length or select it using the function [?]. The field legend is set in System Manager module.

  6. Region – Enter the name of the State in alphanumeric format of 30 characters maximum length or select it using the function [?]. The area must correspond to the entered region. The field legend is set in System Manager module.

  7. Postal Code - Enter the Postal Code in alphanumeric format of eight characters maximum length. The entered value must correspond to the selected town. The user can select the required code using the function [?].The field legend is set in System Manager module.

  8. Country - The Country name gets automatically populated when the Region is selected.

  9. Home Phone No – Enter Residence Telephone Number of the contact person (in the organization) in alphanumeric format of 20 characters maximum length.

  10. Office Phone No - Enter Office Telephone Number in alphanumeric format of 20 characters maximum length.

 

Employee Details

 

  1. Employment ID - Enter the Employment ID of the Patient in alphanumeric format of 15 characters maximum length.

  2. Occupation Class – Select the Occupation class from the list. The listed values are defined in the function Occupation Class.

  3. Occupation of the Person - Select the Occupation from the list. The listed values correspond to the selected occupation class. If the required occupation is not available, enter the same in alphanumeric format of 15 characters maximum length in the text box provided.

  4. Status - Select the Employment Status of the Patient from the list. The listed values are pre-defined.

The field legends for Address 1-4, Town, Area, State, and Postal Code are defined in Site function in System Manger module.
