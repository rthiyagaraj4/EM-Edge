External Referral Source

From this function you can Record and maintain details of the External Referral Sources and Query for the desired External Referral Source record.

Patients are referred from/to various external sources for healthcare management by the facilities of the enterprise.

Creating an External Referral Source

From the MP Code Setup > MP Code Setup, select the External Referral Source function.

The External Referral Source screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the External Referral Source or [Reset] to clear the entered data.

  4. Click [Query] to search for existing External Referral Source records.
