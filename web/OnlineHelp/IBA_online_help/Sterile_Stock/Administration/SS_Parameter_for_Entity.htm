SS Parameter for Entity

From this function you can Define business logics or rules that influence the functioning of transactions in the Sterile Stock module at the Entity Level.

The SS Parameter for Entity function is used to maintain the various parameters of the Sterile Stock module at the entity level. The parameter setup controls certain aspects and functionality of the module functions at the entity level. Only one record can be created. It is mandatory to set up the parameter before any operations are performed as these record stores important details about the Sterile Stock module.

Defining Entity Parameter

From SS Administration, select the SS Parameter for Entity function.

The SS Parameter for Entity screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [Apply] to save the defined Entity Parameter rules or [Reset] to clear the entered data.

The ‘Current Processing Month/Year’ option displayed based on the installation. During installation of Enterprise Management application, the ‘Current Processing Month/Year’ is set at the Entity Level or the Facility Level.
