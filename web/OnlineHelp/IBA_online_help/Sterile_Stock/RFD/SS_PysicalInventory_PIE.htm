Select the following details to record the Physical Inventory details:

  1. Group – Select the group from the list. The groups defined for the selected owner and current store will be displayed in the list. This is a mandatory field.

  2. Tray No – Enter the tray number in numeric format of 12 digits maximum length. This is a mandatory field.

  3. Bin Location – Select the value from the list. The Bin Locations associated to the selected owner store will be displayed in the list.
