Enter the following details

  1. Doc Type – Select the request document type from the list. The values listed are defined in the ‘SS Document Type’ function and the default value displayed is specified in the ‘SS Parameter for Facility’ function for Request group. This is a mandatory field.

  2. Doc No. – The request document number generation is based on the selected document type. If the document number has to be manually generated for the selected document type, the user is informed. Enter the document number in numeric format of eight digits maximum length. If the system generates the number, you are not allowed to enter the document number.

  3. From Store – Select the store name from which the stocks are to be returned. The stores defined in the master screen are displayed. This is a mandatory field.

  4. To Vendor – Select the vendor to whom you return the stocks.  The vendors defined in the master screen are displayed. This is a mandatory field.

  5. Date [From-To] – Select the date using the calendar function. Based on the date range the stock details are displayed. Document Date should not be greater than System Date or Request To Date.

  6. Entry Completed – Select the ‘Entry Completed’ check box to indicate that entry is completed and request document is available for authorizing.  This check box is selected by default.

  7. Entered By – The logged in user name is displayed and disabled.

  8. Request Type - Select the Request Type from the list. The values displayed are Normal, Urgent and Elective. The value Normal is displayed, by default.
