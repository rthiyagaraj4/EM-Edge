Enter the following details to create SS Parameter for Facility record:

  1. Current Processing Month/Year –This period denotes the processing period of Sterile Stock module. The current processing month and year should always denote the current processing month and year and not the Calendar month and year. The current processing month and year are automatically updated to the next period during month end processing. Once the current processing month and year is defined and recorded, the processing month and year cannot be changed. This is a mandatory field.

Interface

  1. Billing – Select the check box to interface Patient Billing Manager module with Sterile Stock module. This check box is selected by default, if Patient Billing Manager module is interfaced with Sterile Stock module.

  2. Billing Service – Select the required Billing Service from the list. The listed values are defined in the ‘Billing Services’ function of the Revenue Management module. This field is enabled if ‘Billing check’ box is selected.

Acknowledgement For

  1. Issue - Select this check box to make the issue transactions applicable for acknowledgement. By default, this check box is cleared.

  2. Return - Select this check box to make the issue return transactions applicable for acknowledgement. By default, this check box is cleared.

  3. Issue against Loan – Select this check box for acknowledging Issue against Loan.

  4. Return of Loan - Select this check box for acknowledging Return of Loan.

Default Document Types

  1. Prepare Group – Select the default Document Type for Prepare Group Transaction from the list. All the Document Types applicable at the Facility Level and Entity Level and associated to the Prepare Group transaction in SS Document Type function are displayed in the list.

  2. Request Group – Select the default Document Type for Request Group Transaction from the list. All the Document Types applicable at the Facility Level and Entity Level and associated to the Request Group transaction in SS Document Type function are displayed in the list.

  3. Issue Group – Select the default Document Type for Issue Group Transaction from the list. All the Document Types applicable at the Facility Level and Entity Level and associated to the Issue Group transaction in SS Document Type function are displayed in the list.

  4. Return Group – Select the default Document Type for Return Group Transaction from the list. All the Document Types applicable at the Facility Level and Entity Level and associated to the Return Group transaction in SS Document Type function are displayed in the list.

  5. Request for Loan - Map the default Document type value ‘Request for Loan Group’ to this transaction type as mapped in System manager  module .

  6. Issue against Loan - Map the default Document type value ‘Issue against Loan Group’ to this transaction type as mapped in System manager module .

  7. Return of Loan - Map the default Document type value ‘Return of Loan Group’ to this transaction type as mapped in System manager module.

  8. Return Instrument – Select the default Document Type for Return Instrument Transaction from the list. All the Document Types applicable at the Facility Level and Entity Level and associated to the Return Instrument transaction in SS Document Type function are displayed in the list.

  9. Tray Discrepancy – Select the default Document Type for Tray Discrepancy Transaction from the list. All the Document Types applicable at the Facility Level and Entity Level and associated to the Tray Discrepancy transaction in SS Document Type function are displayed in the list.

  10. Write Off – Select the default Document Type for Write off Transaction from the list. All the Document Types applicable at the Facility Level and Entity Level and associated to the Write Off transaction in SS Document Type function are displayed in the list.

  11. Variance – Select the default Document Type for Variance Transaction from the list. All the Document Types applicable at the Facility Level and Entity Level and associated to the Variance transaction in SS Document Type function are displayed in the list.

  12. Process Sterilization Request – Select the default Document Type for Process Sterilization Request Transaction from the list. All the Document Types applicable at the Facility Level and Entity Level and associated to the Process ROF transaction in SS Document Type function are displayed in the list.

  13. Auto Request Group – Select the group that is to be set as default document type for the Auto Request Group.  

Other Defaults

  1. Store for Prepare Group – Select the default Store from the list. The selected store is made default in Prepare Group transaction. The list displays all the owner stores defined in the ‘CSSD Store’ function.

  2. Service Location - Select the default Service Location from the list. The selected service location is made default in the respective transactions. The service locations listed are defined in the ‘Service Location’ function.

  3. Write Off Reason - Select the default Write off Reason from the list. The Write Off Reasons defined in the ‘Write off Reason’ function are listed for selection.

  4. Auto Request on Store – Select the Auto Request Store only if it is a non-owner store.

  5. Enable Auto Request for Group based on Reorder Level of Non Owner Store - The auto request will be raised only if you select this check box. By default, this check box is unchecked

  6. Surgery Type linked to OT – Select this check box to setup new Surgery Types and link them to OT Surgery Names. By default, this check box is unchecked as the Surgery Name, Surgery Name  and Nature fields are not available in all transactions and reports. The application uses the existing surgery types defined within CSSD that are not linked to the OT Surgery Name.  

  7. Allow Multi Group Types in Sterilization Request - Select the check box to allow mutli-group types in sterilization request. When this check box is selected, the system disables label selection for all group types in Packing Label section. By default, the check box is selected. If you clear the check box, then system will label selection for group types in Packing label section.

  8. Group Type Mandatory in Sterilization Request – Select the check box to make group type mandatory in Generate sterilization request function and Sterilization Request function. By default, this check box is cleared.

Packaging Label

  1. Instruction on Packaging Label: Enter the instructions to be used in printing of printed in Pack Label 6X9 and Pack Label 3X5 in alphanumeric format up to 50 characters.

  1. Print Packed By in Label – Select the check box to print ‘Packed By legend only in Pack Label 6X9 and Pack Label 3X5. By default, the check box will be selected. If the check box is cleared, no value will be printed in pack label.

  2. Label Size for Linen Group Type – Select the label size for linen group type from the drop-down list. The list values include 'Pack Label 6X9' and 'Pack Label 3X5'.

  3. Label Size for Single Group Type - Select the label size for single group type from the drop-down list. The list values include 'Pack Label 6X9' and 'Pack Label 3X5'.

  4. Label Size for Tray Group Type - Select the label size for tray group type from the drop-down list. The list values include 'Pack Label 6X9' and 'Pack Label 3X5'.

  5. Label Size for Set Group Type - Select the label size for set group type from the drop-down list. The list values include 'Pack Label 6X9' and 'Pack Label 3X5'.

  6. Label Size for Soft Pack Group Type - Select the label size for soft pack group type from the drop-down list. The list values include 'Pack Label 6X9' and 'Pack Label 3X5'.

All the Label Size fields are enabled only if the if 'Allow Multi Group Types in Sterilization Request' checkbox is not selected.
