Loan Acknowledgement

From this function you can acknowledge the loan request.

You can acknowledge all Issues made against Loan Requests and Returns of Loaned Items, if they are configured in the Parameter for Facility screen.

Acknowledging Loan Request

From the SS Transactions > Loan > select the Loan Acknowledgement function.

The Loan Acknowledgement screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. If you select ‘Issue against Loan’ as TRN Type, all the document types associated with ‘Issue against Loan’ are displayed in the Document Type drop-down list. The default value is 'Select’. The available Action Types are ‘Acknowledge’ and ‘Confirm’ with the default value being ‘Acknowledge’.

  3. If you select ‘Return of Loan’ as TRN Type, the only available Action Type is ‘Acknowledge’, which is the default value.

  4. Select the Document Type as ‘Issue against Loan Request’ or ‘Loan Return’ and enter the other mandatory (*) search criteria fields. Refer <Related Field Description >  

    1. TRN Type – Select one of the two Transaction Types: Issue against Loan Request (Default) and Loan Return. This field is mandatory.

    2. Document Type–Displays the document types associated with ‘Request for Loan’ document types within a date range as defined through the ‘Document Type’ Setup in ‘System Manager’ module. If you select ‘Issue against Loan’ as TRN Type, all the document types associated with ‘Issue against Loan’ are displayed in the Document Type drop-down list. The default value is ‘Select’. This field is mandatory.

    3. Action Type - The available Action Types are: Acknowledge, Confirm, Reject. If you select ‘Issue against Loan’ as TRN Type, the available Action Types are ‘Acknowledge’ and ‘Confirm’ with the default value being ‘Acknowledge’. If you select ‘Return of Loan’ as TRN Type, the only available Action Type is ‘Acknowledge’ which is also the default value. You can Confirm Acknowledge only Issue against Loan Request. You cannot Confirm Acknowledge Return of Loan.

    4. From Store – Displays the name of the non-owner stores from which the Issue against Loan request document or Loan Return document was received.

    5. To Store- Displays the name of the non-owner stores from which the Issue against Loan request document or Loan Return document was issued. This field is mandatory.

    6. From Doc Date – This field is mandatory.

    7. To Doc Date – This field is mandatory.

    8. Doc No – Number of the Issue against Loan Request document or Loan Return document.

    9. Doc Ref – Reference number of the Issue against Loan Request document or Loan Return document.

    10. Group – Group to which the Issue against Loan Request document or Loan Return document belongs.

  5. Click [Search] to display all the Issue against Loan Request documents or Loan Return documents that match the search criteria.  Refer <Search results>

    Search results:

    1. Document type –  You can click on the Doc Type hyperlink to view details of all the Sterile Groups on each loan request document. You can configure the default value of the default document type for Loan Request in the Parameter for Facility screen.

    2. Doc Date – Date when the Issue against Loan Request document or Loan Return document was issued.

    3. From Store - Displays the name of the non-owner stores from which the Issue against Loan request document or Loan Return document was received.

    4. To Store – Displays the name of the non-owner stores from which the Issue against Loan request document or Loan Return document was issued. This field is mandatory.

    5. Group – Group to which the Issue against Loan Request document or Loan Return document belongs.

    6. Acknowledge – Select this check box to acknowledge the of the corresponding document.

  6. Select one or multiple documents to acknowledge at once by clicking on the Acknowledge checkbox of the corresponding document.

If you do not select the ‘Acknowledge’ checkbox for any record and click Apply, the following error message is displayed:

‘Please select at least one document to acknowledge’  

  1. To select the constituent Sterile Groups (i.e., Trays) of the selected Loan Issue Group or Loan Return document:

    1. Click on the Loan Request document or Loan Return Document Type hyperlink, the constituent trays are displayed in the bottom frame of the page.

    2. In the Acknowledge drop-down list, mark the trays as ‘Acknowledged’, ‘Rejected’ or ‘Pending’.

Except for the pending acknowledgement records, all other selected documents in the upper frame as well as the corresponding Sterile Groups and their trays are removed from the screen display

  1. If you select ‘Rejected’, click the Ack. Remarks hyperlink to enter the Acknowledgement Remarks for Sterile Groups whose acknowledgement status is ‘Reject’.

If you click Apply or click another Loan Issue/Loan Return document number without recording remarks for a  tray in ‘Reject’ status, the following error message is displayed:

‘Remarks field cannot be blank for Rejected Trays’

  1. Click the document and view the Sterile Groups that are in ‘Pending’ status in the bottom frame.

The rest of the Sterile Groups are already ‘Acknowledged’ or ‘Rejected’. The pending documents will be shown during search operation also. If you mark all the documents as Pending, the following error message is displayed:

‘At least one tray should be Acknowledged/Rejected’

  1. Click Discrepancy Remarks; displays ‘Tray Discrepancy Remarks’ and ‘Loan Return Discrepancy Remarks’. These two options are enabled or disabled based on availability of ‘Tray Discrepancy’ or ‘Loan Discrepancy’ details. Clicking these links displays the discrepancy details. If neither the ‘Tray Discrepancy Remarks’ nor the ‘Loan Return Discrepancy Remarks’ are available for the tray, the ‘Discrepancy Remarks’ link is in disabled state.

  2. Click Apply when all the constituent Sterile Groups (i.e., Trays) of a Loan Issue Group or Loan Return document are marked as ‘Acknowledged’ or ‘Rejected’
