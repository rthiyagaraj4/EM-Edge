Return of Loan

From this function you can Record the returning details of the trays loaned.

This function allows you to multi-select the trays to be returned.

Returning of Loan

From the SS Transactions > Loan > select the Return of Loan function.

The Return of Loan screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

    1. Doc Type – Select the document type as defined in the ‘Document Type’ Setup in ‘System Manager’ module. This field is mandatory.

    2. Doc No - Look up the number of the return of loan document. This field is mandatory.

    3. Doc Date - Select the date when the return of loan was placed. This field is mandatory.

    4. Return from - Select the store name from which the trays are returned. This field is mandatory.

    5. Return to - Select the store name to which the trays are returned. This field is mandatory.

    Only non-owner stores are available in the Return From and Return To dropdowns and both have ‘Select’ as the default value.  

    1. Doc Ref - Enter the document reference number, if any.

    2. Entry completed - You can open a record in Query mode and edit it, if the document is in ‘Entry Not Completed’ status. Records in ‘Entry Completed’ status are opened in read-only mode.

    3. Entered By – Displays the name of the person who enters the return of loan details. You cannot edit this value.

    4. Display Groups Linked to patient – Select the sterile groups linked to patient.

    5. Select Trays - Click Select Trays to select multiple trays from the Select Trays popup.

    6. Del - This checkbox is unchecked and enabled by default. You can click this check box to mark the document for deletion.

    7. Group Description – You can click this hyperlink to view the group item details in the Group Item Details popup.

    8. Tray No – Displays the number of the trays.

    9. Patient ID  - Displays the patient identification code.

    10. Patient Name - Displays the name of the patient.

    11. Bin Location - Displays the bin location of the sterile group. This field is mandatory.

    12. Expired – This check box is selected and disabled if the tray is in expired status.

    13. Used – This check box is selected and disabled if the tray is in used status.

    14. Sterilized – This check box is selected and disabled if the tray is sterilized.

    15. Wash – This check box is selected and disabled if the tray is washed.

    16. Pack – This check box is selected and disabled if the tray is packed.

    17. Contaminated – This check box is selected and disabled if the tray is contaminated.

    18. Charge Consum – This check box is selected and disabled if the tray is charged.

    19. Reason - Click this hyperlink to record the reason for returning the trays in the Loan Return Reason popup that is displayed.

    20. Discrepancy - Click this hyperlink to enter the Tray Discrepancy details while returning the trays that are in ‘Used’ status in the Discrepancy Details popup that is displayed.

      Enter the following details:

      1. Instruments – Select the instrument which has a discrepancy.

      2. Discrepancy Reason – Select the reason for the discrepancy.

      3. Remarks – Enter the remarks for the discrepancy.

      4. Discrepancy Quantity – Enter the quantity of the discrepancy.

      5. Patient ID – Enter the patient identification.

      6. Returnable Instrument – Select this check box if the instrument is returnable.

      7. Encounter ID – Select the encounter ID.

      8. Charge option – Select the payment option.

      9. Cost – Enter the cost of damage to the instrument.

      10. Receipt Reference – Enter the reference number of the receipt.

      11. Add – Adds the details to the adjacent table.

      12. Cancel – Cancels the discrepancy details entry.

      13. Close – Closes the Discrepancy screen without saving the details.

      14. Del – Marks for instrument details for deletion

      15. Instrument Description – Displays the description of the instrument.

      16. Discrepancy Reason – Displays the reason of discrepancy.

      17. Inst Quantity – Displays the instrument quantity.

      18. Disc Quantity - Displays the discrepancy quantity.

  3. Click Select Trays to multi-select the trays from the Select Trays popup that is displayed.

    Enter the following details:

    1. Loaned from Store – Displays the store name from which the trays were loaned.

    2. Bin Location– Displays the bin location of the trays.

    3. Current Store– Displays the current store name to which the trays are returned.

    4. Group – Displays the sterile group name.

    5. Tray No– Displays the number of the tray which is returned.

    6. Patient ID– Displays the patient identification.

    7. Issue Doc.ID– Displays the issue document identification.

    8. Used– Select the status of the tray i.e., Used or Unused.

    9. Select– Select the trays details which are being returned. The selected details are displayed in the Return for loan screen.

  4. Mark the trays as either ‘Used’ or ‘Unused’ while returning.

  5. Click Ok in the Select Trays popup, the tray details are displayed in the Return of Loan screen.

  6. Click the Group Description hyperlink to view the group item details in the Group Item Details popup.

  7. Mark the Tray details for deletion by selecting the Del check box. This Del checkbox is unchecked and enabled by default.

If all the Trays are marked as ‘Delete’ or when no trays are available and you click Apply, the following error is displayed: “At least tray must be available to perform Return of Loan.”

  1. Click the Reason hyperlink to record the reason for returning the trays in the Loan Return Reason popup that is displayed

    1. Reason – Enter the reason for returning the trays in free-text or look up from the pre-defined text. This is mandatory for all trays, if the tray is not associated to a patient.

You must enter or select the ‘Loan Return Reason’ for all trays, if the tray is not associated to a patient.. If the tray is marked for deletion, you need not enter the reason even if the Loan Return Reason field is mandatory.Click Discrepancy hyperlink to enter the tray discrepancy details while returning the trays in ‘Used’ status in the Discrepancy Details popup.

  1. Click Apply.

If you do not record the Loan Return Reason for a tray that is returned because it is not associated to a patient, the following message is displayed when you click Apply:

“It is mandatory to record the Loan Return Reason if the tray is not associated to a patient.”
