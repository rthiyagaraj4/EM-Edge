Cancel Loan Request

From this function you can cancel the loan request.

This function allows you to cancel either:

• individual Sterile Groups in the Loan Request that are not yet issued.

• entire loan request document, if none of the Sterile Groups in the document are issued but not acknowledged or just issued.

But you cannot cancel the loan Sterile Groups that are issued but pending acknowledgement.

Cancelling Loan Request

From the SS Transactions > Loan > select the Cancel Loan Request function.

The Cancel Loan Request screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the mandatory (*) search criteria fields. Refer <Related Field Description >  

    Enter the following details:

    1. Doc Type – Select the Document Type associated with ‘Request for Loan’ document types within a date range. Based on the setup in the Parameter for Facility screen, one of the values is displayed by default.

    2. Doc No - Enter the Request for Loan document number directly, if it is known.

    3. Doc Date - Select the date when the Request for Loan document was placed.

    4. Requested by Store - Select the non-owner store name from which the user has Cancel Loan Request privilege. This field defaults to ‘Select’.

    5. Requested On Store - Select the non-owner store name from which the user has Cancel Loan Request privilege. This field defaults to ‘Select’.

    6. Request Type - Select the type of request from the drop-down list.  The Request Type defaults to the value ‘Normal’.

  3. Click [Search] to display all the Loan Request Documents that match the search criteria or if there is at-least one Sterile Group available for cancellation in the particular Loan request. Refer Search results

    Search results:

    1. Doc type – Displays the Document Type as defined through the ‘Document Type’ Setup in ‘System Manager’ module. This field is mandatory. You can click on the Doc Type hyperlink to view details of all the Sterile Groups on each loan request document. You can configure the default value of the default document type for Loan Request in the Parameter for Facility screen.

    2. Doc No – Displays the number of the Request for Loan document.

    3. Doc Date – Displays the date when the Request for Loan was placed.

    4. Requested by Store – Displays the name of the non-owner stores from which the Loan request was received.

    5. Requested on Store – Displays the name of the non-owner stores from which the Loan request is issued.

  4. Click the Document Type hyperlink to display the Group Code, Description, Requested Quantity, Issued Quantity, Pending Ack./Conf. Quantity.

  5. Select the check box against the Group Code to be cancelled or select the check box next to cancel label to select all the records.

You cannot cancel Sterile Groups that satisfy the following conditions:

• Sterile Groups that are issued whether partially or fully.

• Sterile Groups that are issued whether partially or fully but not yet acknowledged or awaiting ‘Confirm Acknowledge’ i.e., pending acknowledgement.

• Sterile Groups that are issued and Rejected.  

  1. Enter the reason for cancelling the loan request in Cancellation Remarks field. This is mandatory field.

  2. Click [Apply] to confirm the cancellation of the loan requests or [Reset] to clear the entered data
