Issue against Loan

From this function you can Record the Issue details of Sterile Groups within a Loan Request document

This function eliminates the need to search and select the IssueGroup and Tray numbers as they are pre-loaded.

Issuing against Loan

From the SS Transactions > Loan > select the Issue against Loan function.

The Issue against Loan screen gets displayed.

 

  1. Click [Create]; the Pending Requests for Loan popup is displayed

    1. Doc Type - Select the document type.

    2. Doc No - Enter the number of the document.

    3. Doc Date - Enter the date of the issue document.

    4. Requested By - Enter the name of the requested by store.

    5. Requested On - Enter the name of the requested by store.

    6. Search - Performs the search based on the search criteria.

    Search results:

    1. Doc No - Click this hyperlink to select the required Loan Request document and all the Loan Request sterile groups that are part of the selected Loan Request document are displayed in multiple rows in the Issue against Loan screen.

    2. Doc Type - Displays the document type of the loan request document.

    3. Doc Date - Displays the document date of the loan request document.

    4. Requested by Store - Displays the Requested by store name.

    5. Requested on Store - Displays the Requested on store name.

  2. Enter the mandatory (*) search criteria. Refer <Related Field Description>.

    1. Req Doc Type – Select the document type as defined through the ‘Document Type’ Setup in ‘System Manager’ module. This field is mandatory.

    2. Req Doc No - Look up the number of the pending request for loan document. This field is mandatory.

    3. Issue Store - Select the store name from which the pending request for loan is being issued. This field is mandatory.

    4. Receiving Store - Select the store name from which the pending request for loan is being received. This field is mandatory.

    5. Authorized By - Enter the name of the user who authorises the document.

    6. Doc Type - Select the document type.

    7. Doc No - Enter the document number of the pending request for loan document.

    8. Doc Date - Select the date when the pending request for loan was placed. This field is mandatory.

    9. Doc Ref - Enter the document reference, if any.

    10. Del - Select this check box to mark the documents for deletion.

    11. Request Group - Displays the request sterile group.

    12. Authorized Group - Displays the authorized sterile group.

    13. IssueGroup - Select the required issue group; the Tray drop-down list is displayed with associated available tray numbers. For each requested sterile group, all the valid Issue Groups , namely, the requested groups and their alternate groups associated with both the Requesting and Issuing Stores are displayed in the column.

    14. Tray No. - The tray numbers associated with the issue group are displayed. The default value is ‘Select’.

    Only the tray numbers currently selected are displayed and not the ones selected earlier.

    1. Bin Location - Displays the bin location of the sterile group.

    2. Expiry Date - Displays the expiry date of the sterile group.

    3. Issue Cost - Displays the issue cost of the sterile group.

    4. Patient ID  - Displays the patient identification.

    5. Patient Name - Displays the name of the patient.

  3. Click Search to view the Pending Request for Loan documents.

  4. Select the required document by clicking the Doc No hyperlink and all the pending documents for issue are displayed in multiple rows in the Issue against Loan screen.

  5. Select the required Issue Group and the system loads the Tray drop-down list with associated available tray numbers.

For each requested sterile group, all the valid Issue Groups i.e., Requested Groups and its Alternate Groups associated with both the Requesting and Issuing Stores are pre-loaded in the IssueGroup column. The Tray numbers are pre-loaded in the Tray No. column thereby eliminating the need to search for trays. The Tray drop-down list excludes tray numbers that are already chosen for the same Issue Group earlier. The default value is ‘Select’.

  1. Select the Del check box to delete the documents.

  2. Click Apply.
