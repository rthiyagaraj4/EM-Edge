Change Bin Location

From this function you can Change the Bin Location of a Tray from Current Location to a New Location.

The movement of tray(s) from the current location to the new location is allowed within the same owner location. You can change the Bin Location for complete set of trays held in one Bin Location to the other or individually move each tray from one Bin Location to the other.

Changing Bin Location

From SS Transactions > Tray Maintenance, select the Change Bin Location function.

The Change Bin Location screen gets displayed.

  1. Click [Query] to display all the trays based on the default parameters set in the SS Parameter for Facility function or enter the search criteria and click [Search] to select a specific record.

    Enter the following search criteria to display the prepared documents:

    1. Owner - Select the owner from the list. The default value for owner is displayed based on the value set in ‘Store for Prepare Group’ in ‘SS Parameter for Facility’ function. The stores are displayed based on the restrictions set in ‘SS User Access for Store’ and applicability of the store to perform the specific transaction as set in ‘CSSD Store’. This is a mandatory field.

    2. Old Bin Location - Select the old Bin Location from the list. The list displays all the Bin Locations associated to the selected owner in the function ‘Bin Location’ in Materials Base module.

    3. New Bin Location - Select the new Bin Location from the list. The list displays all the Bin Locations associated to the selected owner in the function ‘Bin Location’ in Materials Base module.

    4. Group - Enter the group in alphanumeric format or click [?] function to search for the required group. All the groups defined in ‘CSSD Store’ function are displayed.

    5. Tray No – Enter the tray number to limit the search for a particular tray in numeric format of eight digits maximum length.

    6. Doc Type – Select the prepare document type from the list. The default ‘Document Type’ set in ‘SS Parameter for Facility’ is displayed, by default. This value can be modified.

    7. Doc No – Enter the document number in numeric format of eight digits maximum length.

  2. On searching, the tray details are displayed based on the entered criteria. The search result displays first 14 records. Click [Next] to view the next 14 tray details or [Previous] to view the previous tray details.

  3. Click ‘Owner Store’ hyperlink to invoke the ‘New Bin Location’ screen.

  4. On assigning the new Bin Location, the assigned new bin location is displayed in the New Bin Location column.

    • On selecting the Owner Store hyperlink, the Owner Store, Group, Tray Number, Old Bin Location details are displayed.

    • Select the required new Bin Location from the list. All the Bin Locations associated to the selected owner in the function ‘Bin Location’ in Materials Base module are displayed.

    • Click [OK] to assign the new Bin Location to the selected Tray or [Cancel] to cancel the process.

  5. Click [Apply] to record the New Bin Location or [Reset] to clear the entered data.
