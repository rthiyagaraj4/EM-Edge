Cancel Request

From this function you can Cancel the request raised by the requestor.

You can cancel the request by providing a cancellation reason. You can cancel the authorized request documents which are not issued. The following conditions are validated by the system.

  • The Logged in user should have rights to cancel the request from the requesting store

  • The requesting and owner stores must have access to the requesting group.

Requesting Trays from Owner Store

From SS Transactions > Issue, select the Cancel Request function.

The Cancel Request screen gets displayed.

  1. Click [Query] to display all the authorized requests which are not issued or enter the search criteria and click [Search] to select a specific record.

    To select a particular request document type, specify the following query criteria.

    1. Doc Type – Select the request document type from the list. The values listed are defined in the ‘SS Document Type’ function and the default value displayed is specified in the ‘SS Parameter for Facility’ function for Request group.

    2. Doc No. – Enter the request document number in numeric format of eight digits maximum length.

    3. Doc Date - Enter the request document generation date in the format of dd/mm/yyyy or select the date using the Calendar function. The entered date must not be less than the processing month/year specified in ‘SS Parameter for Facility’ function and not greater than current system date.

    4. Requested By Store – Select the requested by store from the list. The stores are listed based on the access rights of the user and applicable transaction.

    5. Requested On Store - Select the owner store from the list. The stores are listed based on the access rights of the user and applicable transaction.

    6. Request Type - Select the request type from the list. The listed values are Normal, Urgent and Elective.

  2. On searching, the authorized request documents are displayed based on the entered criteria.

  3. Click ‘Doc Type’ hyperlink to display the requested details. The details like Doc Type, Doc No, Doc Date, Requested By, Requested On, Doc Ref, Entered By and Request Type are displayed.

  4. The details of the Groups to which the request is associated are displayed. You can view details such as Group code, Description, Requested Quantity, Issued Quantity and Cancel check box.

  5. Select the Cancel check box against the group Code(s) and enter the Cancellation Remarks in alphanumeric format of 100 characters maximum length. This is a mandatory field.

  6. Click [Apply] to display the Confirm Cancel dialog or [Reset] to clear the entered data

  7. Click [Yes] to cancel the selected request or [No] to close the dialog.
