Variance Report

From this function you can Generate a Variance Report between the computer stock and physically verified stock on recording the Physical Inventory details.

You can generate a details or a summary report for a particular Inventory ID.

Generating Variance Report

From SS Physical Inventory, select the Variance Report function.

The Variance Report screen gets displayed.

  1. Select the Physical Inventory ID from the list. The listed Physical Inventory ID are the ones for which Physical Inventory Entry is recorded. This is a mandatory field.

  2. Select the Report Type from the list. The listed values are ‘Detail’ and ‘Summary’.

  3. Click [Run] to generate Variance Report or [Reset] to clear the entered data.

  4. Select ‘Report Option’ to generate the report.

  5. Click [OK] to generate Variance Report and spool it in the file on the print server or [Cancel] to cancel the action.
