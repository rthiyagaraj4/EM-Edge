Dispatch

From this function you can Record the dispatch details of a Tray.

All the trays that have undergone Washing/Packing/Autoclaving will be available for dispatch in this function. The user can enter the dispatch date for the selected tray(s) and complete the dispatch process.

Recording the Dispatch Details

From SS Transactions > Sterilization Process, select the Dispatch function.

The Dispatch screen gets displayed.

  1. Click [Query] to display all the trays which are available for dispatch or enter the search criteria and select a specific record.

    Enter the following search criteria:

    1. Ster Req Doc Type – Select the required sterilization request document type from the list. The document type as set for ‘Process Ster Req’ in SS Parameter for Facility will be displayed, by default.

    2. Ster Req Doc No – Enter the sterilization request document number in numeric format of eight digits maximum length.

    3. Doc Date – Enter the document date in dd/mm/yyyy format or select the date using the Calendar function.

    4. Service Location – Select the Service Location from the list. The Service Location as set in ‘SS Parameter for facility’ will be displayed, by default.

    5. Store – Select the Store from the list. The Store as set in ‘SS Parameter for Facility’ will be displayed, by default.

    6. Doc Ref – Enter the document reference number in alphanumeric format of 15 characters maximum length.

  2. Click Search; displays the Ster Doc Type, Ster Req Doc No, Doc Date, Service Location, Store and Dispatch check box.

  3. Click Ster Req Doc Type hyperlink. The application displays the group and tray details such as Ster Req Doc No, Group, Tray No, Dispatch Date and Dispatch check box relevant to the selected document number in a popup.

The Dispatch Date must be greater than or equal to the Autoclaving Date but lesser than equal to the current date i.e., it allows past dates if it is after the Autoclave Date but it cannot be a future date.

  1. The default dispatch date is current date and the dispatch date is mandatory for all trays that are marked for dispatch. All the other fields will only be display fields.

  2. Modify the Dispatch Date and select the 'Dispatch' check box to consider the tray for dispatch process. The check box is selected, by default. Click Ok;  saves the modified details and the popup is closed.

  3. Click [Apply] to confirm the dispatch or [Reset] to clear the entered data.
