Packing

From this function you can Record the packing details.

The Packing function is used to record packing details required for packing tray(s). The packing details entered in the ‘CSSD Group’ function are displayed, by default. Trays can be held back by specifying a relevant reason for holding the tray. Tray discrepancy (if any) can be recorded.

Recording the Packing Details

From SS Transactions > Sterilization Process, select the Packing function.

The Packing screen gets displayed.

  1. Click [Query] to display all the trays which are available for packing or enter the search criteria and select a specific record.

    Enter the following search criteria:

    1. Doc Type – Select the sterilization request document type from the list. The values listed are defined in the function ‘SS Document Type’ and the default value displayed is specified in the function ‘SS Parameter for Facility’.

    2. Doc No. – Enter the sterilization request document number in numeric format of eight digits maximum length.

    3. Doc Date - Enter the Sterilization Request generation date in date format of dd/mm/yyyy.

    4. Store – Select the Store from the list. The owner stores with rights to perform ‘Sterilization Request Processing’ and assigned to the logged in user are displayed.  

    5. Service Location - Select the Service Location from the list. The location specified in ‘SS Parameter for Facility’ is displayed, by default.

  2. Click ‘Doc Type’ hyperlink to record the packing details.

  3. The Document Type, Document Number, Document Date, Store, Service Location, Document Reference details are displayed. The Sterilize, Wash and Pack check boxes are selected or cleared based on the Sterilization Request for the selected document.

  4. The Tray details are displayed in the right hand side of the screen.

  5. Click ‘Group’ hyperlink to modify the Tray details. The Group name and the tray number is displayed and disabled.

    Enter the following details:

    1. Pack Item – The pack item defined for the group is displayed. You can select the pack item using [?] function. This is a mandatory field.

    2. Pack Item Qty - The packing quantity as defined for the group is displayed. You can enter the Pack item quantity.

    3. Pack Cost - The packing cost is displayed and disabled. The displayed value is calculated by the system using the formula [(Item*unit cost*item quantity) + Additional packing cost]. On changing the pack quantity, the pack cost is calculated.

    4. Hold – Select the check box to place the selected tray on hold.

    5. Hold Reason – Enter the hold reason in alphanumeric format. This field is enabled if the 'Hold' check box is selected.

    6. Packing Date – The system date is displayed. This date is not displayed if the tray is put on hold. Select the Packing Date using the Calendar function. This is a mandatory field.

  6. Click [Discrepancy] to invoke the Discrepancy Details screen. The discrepancy details recorded in the Instruments Usage/Discrepancy at Units function are displayed.

    1. Enter the details in the mandatory (*) fields. <Related Field Description>

    2. Click [Add] to add the discrepancy details or [Cancel] to clear the entered details.

    3. On adding the discrepancy details, the details are displayed in the right hand side of the discrepancy details screen. Click Instrument Description hyperlink to modify the discrepancy details and click [Modify] to modify the changes.

    4. Click [Close] to exit from the Discrepancy Details screen.

  7. Click [Modify] to modify the packing details or [Cancel] to clear the entered data.

  8. Select the ‘Update’ check box to consider the tray for packing process. The check box is selected, by default.

  9. Click [Apply] to record the packing details or [Reset] to clear the entered data.

  10. On clicking [Apply], a ‘Confirm Printing’ dialog is displayed.  Click [Yes] to print the packing label for the selected trays or click [No] to cancel the printing.

The trays are packed after the Sterilization request is generated. If washing is applicable, the trays are packed after the washing process is completed. The trays are available for packing if packing is applicable for the particular request.
