Tray Detail

From this function you can View the complete details of trays with their current status.

The Tray Details are grouped to the Group Type and Group of the Tray. The current status of the trays is displayed if the trays are at the service location, sterilized or unsterilized or destroyed.

Viewing the Tray Details

From SS Queries, select the Tray Detail function.

The Tray Detail screen gets displayed.

  1. The details of all the trays are displayed. Enter the search criteria and click [Search] to view the details of a specific tray.

    Enter the following criteria to search for the specific trays:

    1. Group Type – The value ‘All’ is displayed, by default. Select the Group Type from the list. The listed values are Set, Tray, Single and Soft Pack.

    2. Group - Enter the group in alphanumeric format of 15 characters maximum length or select the group using [?] function.

    3. Tray No - Enter the tray number in numeric format of 15 digits maximum length.

    4. Owner Store - Select the owner store from the list.

    5. Current Store – Select the current store location of the selected tray from the list. All the stores (owners and requestors) are listed.

    6. Status - Select the status from the list. The value ‘All’ is displayed, by default. The listed values are ‘Sterilized’, ‘Unsterilized’, ‘At Service Location’ and ‘Destroyed’.

    7. Expiry Date - Enter the expiry date in the format of dd/mm/yyyy or select the date using the Calendar function.

  2. On searching, the tray details are displayed based on the entered criteria. The displayed details are: Group Type, Group, Tray Number, Owner Store, Current Store, Bin Location, Expiry Date and Status.
