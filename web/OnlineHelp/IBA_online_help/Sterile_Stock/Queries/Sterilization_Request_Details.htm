Sterilization Request Details

From this function you can View the sterilization request details and the status of the sterilization request.

From this function you can view the details like the date of washing, packing and sterilization for the request raised. All the sterilization request details with the default settings are displayed, by default.

Viewing the sterilization request details

From SS Queries, select the Sterilization Request Details function.

The Sterilization Request Details screen gets displayed.

  1. The sterilization request details of all the trays are displayed, by default. Enter the search criteria and click [Search] to view the specific sterilization request details.

    Enter the following search criteria:

    1. From Date – Enter the ‘From’ date in the format of dd/mm/yyyy or select the date using the Calendar function.

    2. To Date - Enter the ‘To’ date in the format of dd/mm/yyyy or select the date using the Calendar function. The ‘To date’ must be greater than ‘From date’.

    3. Store - All the owner stores are listed. Select the store from the list.

    4. Doc Type – The sterilization request document types are listed. Select the document type from the list.

    5. Ster Req Status – The system-defined statuses are listed. Select the status from the list. The listed statuses are: Acknowledged, Autoclaved, Dispatched, Hold, In Autoclaving, Packed, ROF Ent Compl, ROF Entered, Wash Completed and Washing Started.

  2. On searching, the sterilization request details of the selected store are displayed based on the entered criteria.

  3. The displayed details are: Group Type, group, Group Type, Owner Store/Current Store, Group Name, Total Tray's), Sterilized, Unsterilized, In Service Location details.

  4. Click Document Type hyperlink to view the request details. On clicking Document Type hyperlink, the details displayed are: Tray number, Entered Date, Checked Date, Washed Date, Packed Date, Autoclaved Date, Dispatched Date and Acknowledged Date.

The Sterilization requests which are on hold are displayed with a unique colour code.
