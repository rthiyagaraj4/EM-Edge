Loan Request Status

From this function you can search the status of the Loan Request documents.

You can enter search criteria such as From/To Date, Entry Completion, Authorization, Issue, Cancellation, Req By/On Store and Request Type.

Search for Loan Request status

From the SS Queries, select the Loan Request Status function.

The Loan Request Status screen gets displayed.

  1. Enter the search criteria and click [Search] to view the required loan request details.

  2. Click Request for Loan hyperlink under Doc Type column to open the Loan Request Details popup which displays the Sterile Group-Level issue status of each group in the Loan Request. You can view details such as Request Group, Group Type, Request Quantity, Issued Quantity, Pending for Ackn./Conf. Quantity and Reject Quantity.

  3. Click [Reset] to clear the entered data.
