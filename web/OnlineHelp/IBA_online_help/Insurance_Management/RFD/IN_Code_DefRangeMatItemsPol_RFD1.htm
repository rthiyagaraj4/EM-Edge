Enter the following details to select the items for Policy Coverage:

 

Range of Items to be Copied/Deleted for Inclusion/Exclusion

  1. Material Item Exclude/Include Criteria – Select a value from the list. The values are system defined – ‘Sale Category’, ‘Material Item’ and ‘Enterprise Material Group’.

  2. Code [From-To] – Click [List of Values] to view and select a valid item Code. If the code is valid, the short description of the item code gets displayed. A range of material items and can be included or excluded from the Policy Types based on the value selected for Exclude/Include Criteria.. This is a mandatory field.

The legend changes as per the value selected in the Exclude/Include Criteria. The values in the list differ based on the ‘Material Item Exclude/Include Criteria’ selected.

  1. Excl/Incl Ind – Select a value from the list. The system defined values are ‘Include’ and ‘Exclude’. Select Include and the set of items are associated to the range of policies selected. Select Exclude and the set of items are excluded from the range of policies selected.

  2. Override Inclusion/Exclusion – Select the check box to override the Inclusion/Exclusion criteria. The override performs inverse action based on the value selected for ‘Excl/Incl Ind’.

  3. Patient Class – Click [List of Values] to view and to select a Patient Class. The system defined values are ‘All’, ‘Daycare’, ‘Emergency’, ‘Inpatient’, ‘Outpatient’ and ‘External’. Select All and the range of services copied/deleted for inclusion/exclusion is applicable for all the patient class in the facility.

  4. Co-pay Ind - On selecting the value as ‘Include’ in the ‘Excl/Incl Ind’ function, ‘Percentage’ is displayed. You can select a value from the list. The system defined values are ‘Percentage’, ‘Rate’ and ‘Fixed’. The Copay value can be set based on the Co-pay Indicator set. The amount to be paid by the co-payer towards the items in the range of policies is calculated based on the Co-Pay Indicator selected. You can select a value from the list, if the ‘Excl/Incl Ind’ is selected as ‘Include’.

  5. Include Policy Level Coverage – Select the check box to include policy level coverage details for the co-payer.

  6. Copay Value – Enter the Copay Value in numeric format. The amount to be settled by the Co-payer is calculated based on the Co-Pay Indicator selected and the value specified for Copay value. The entered value must be between 1 and 99.

  7. Pre Apprv – On selecting the value as ‘Include’ in the ‘Excl/Incl Ind’ function, ‘Required’ is displayed. Select a value from the list. The system defined values are ‘Required’ and ‘Not Required’. Select Required to indicate that an approval is required from the insurance company to include or exclude the items to the range of Policies selected. Select ‘Not Required’, if an approval is not mandatory from the insurance company to include or exclude the items from the range of policies selected.

  8. Pref Check – Select the check box to associate preferred items.

  9. Pref Basis – Select a value from the list. The system defined values are ‘Specific Drug’ and ‘Generic Drug’. The field is enabled only on selecting ‘Pref Check’.

  10. Pref Item – Click [List of Values] to select a preferred item. The items in the list differ based on the value selected for ‘Pref Basis’.

  11. Special Service – Select the check box, if special services can be associated to the range of policies selected. Clear the check box and you cannot associate special services to the range of policies.
