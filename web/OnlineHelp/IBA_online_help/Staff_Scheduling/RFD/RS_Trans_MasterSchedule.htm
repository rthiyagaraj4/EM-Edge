Enter the following details:

 

  1. Location type - Select the required location type from the list. The listed values are defined in 'Location Type' function of System module.

  2. Work cycle basis - Based on the value set for 'Work Cycle Basis' in the parameter setup, the value is displayed as read only text.

  3. Workplace - Enter the workplace in alphanumeric format of 10 characters maximum length or select the required workplace using the [?] function. If the Location Type is selected, workplaces listed in the [?] function corresponds to the selected location type or all the workplaces created in the "Workplace" function are displayed. This is a mandatory field.

  4. Requirement - Enter the requirement in alphanumeric format of 10 characters maximum length or select the requirement using [?] function. The [?] function lists all the requirements as set in the 'Workplace Requirement' function for the selected workplace. This is a mandatory field.

  5. On selecting the requirement code, the description of the requirement is displayed as read only text. If a date range is specified in the 'Workplace Requirement' function, the same is also displayed as read only text.

  6. Role Type - Select the role type as 'Practitioner' or 'Other Staff' from the list.

  7. Staff type - Select the staff type from the list. The staff type displayed is based on the selected role type. If Role Type is selected as Practitioner, the list displays the Practitioner Type as defined in the Application Masters module. If Role Type is selected as Other Staff, the list displays the Other Staff Type as defined in the Application Masters module.

  8. Position - Select the position from the list. The listed values are defined in the "Position" function of Application Masters module. If the 'Staff Type' is selected, the positions listed in the list corresponds to the selected Staff Type or the [?] function displays all the positions defined in Position function in System Configuration module. This is a mandatory field.

  9. Staff ID - Enter the staff ID in alphanumeric format of 10 characters maximum length or select the required staff ID using [?] function.

  10. View - Select an option as 'All Staff' or 'Allocated Staff' from the list. If All Staff is selected, all the staff of the facility is displayed irrespective of whether a schedule is allocated to them or not. If Allocated Staff is selected, the staff for which atleast one schedule is allocated is displayed.
