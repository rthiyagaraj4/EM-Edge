Enter the following details:

 

'By Position' Tab

  1. Role type - Select the Role Type as Practitioner or 'Other Staff' from the list. This is a mandatory field.

  2. Staff type - Select the Staff Type from the list. The Staff Types displayed are based on the selected Role Type. If Role Type is selected as Practitioner, the list displays the Practitioner Types as defined in the Application Masters module. If Role Type is selected as Other Staff, the list displays the Other Staff Types as defined in the Application Masters module.

  3. Position - Enter the Position in alphanumeric format of 20 characters maximum length or select using [?] function. If the 'Staff Type' is selected then the positions listed in the [?] function correspond to the selected Staff Type or the [?] function displays all the positions defined in Position function in System Configuration module. This is a mandatory field..

  4. Click [Show Skills]

  5. Skill - The skills defined in the 'Skill' function are displayed.

  6. Mandatory - Select the check box if the skill is mandatory for the selected position.

  7. Select - Select the check the box to indicate applicability of the skill for the selected position. To select all the displayed skills, select the check box provided in the column header.  

 

'By Skill' tab

  1. Skill - Select the Skill from the list. The listed values are defined in the 'Skill' function. This is a mandatory field.

  2. Role type - Select the Role Type as 'Practitioner' or 'Other Staff Type' from the list.

  3. Staff type - Select the Staff Type from the list. The Staff Types displayed are based on the selected Role Type. If Role Type is selected as Practitioner, the list displays the Practitioner Types as defined in the Application Masters module. If Role Type is selected as Other Staff Type, the list displays the Other Staff Types as defined in the Application Masters module.

  4. Click [Show Positions]

  5. Role Type - The selected Role Type is displayed. If the Role Type is not selected, both 'Practitioners' and 'Other Staff Type' are displayed.

  6. Staff type - The Staff Types of the selected Role Type is displayed. If the Role Type is not selected, both 'Practitioner Types' and 'Other Staff Types' are displayed as defined in the Application Masters module.

  7. Positions - The Positions of the selected Staff Type defined in the 'Position' function (Application Masters Module) is displayed. If the Staff Type is not selected, all the positions as defined in the 'Position' of Application Masters module are displayed.

  8. Mandatory - Select the check box if position is mandatory for the selected skill.

  9. Select - Select the check box to indicate applicability of the position for the selected skill. To select all the displayed positions, select the check box provided in the column header.
