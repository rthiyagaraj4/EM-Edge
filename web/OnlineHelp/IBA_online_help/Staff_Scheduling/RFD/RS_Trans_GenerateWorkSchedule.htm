Enter the following details:

 

  1. Location type - Select the required location type from the list. The listed values are defined in 'Location Type' function of Application Masters module.

  2. Workplace - Enter the workplace or select the required workplace using [?] function. If the Location Type is selected, workplaces listed in the [?] function corresponds to the selected location type or all the workplaces created in the 'Workplace' function are displayed. This is a mandatory field.

  3. Role Type - Select the role type as 'Practitioner' or 'Other Staff' from the list.

  4. Staff type - Select the staff type from the list. The staff type displayed is based on the selected role type. If Role Type is selected as Practitioner, the list displays the Practitioner Type as defined in the Application Masters module. If Role Type is selected as Other Staff, the list displays the Other Staff Type as defined in the Application Masters module.

  5. Position - Select the position using [?] function. If the 'Staff Type' is selected, the positions listed in the [?] function corresponds to the selected Staff Type or all the positions created in the "Position" function of Application Masters module are displayed.

  6. Staff - Enter the name of the staff or click the [?] function to search for the required staff.

  7. Generate Schedule From/To - Enter the date range till when the work schedule is to be generated in date (dd/mm/yyyy) format or select the date using the calendar function. From date corresponds to the start date when the generated work schedule comes into effect. The 'From' date should be greater than or equal to the system date. 'To' date corresponds to the date when the generated work schedule comes to an end. This is a mandatory field.
