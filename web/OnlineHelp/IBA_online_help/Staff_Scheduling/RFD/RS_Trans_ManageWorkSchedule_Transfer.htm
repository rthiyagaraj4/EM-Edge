Transfer

Select [Transfer] from the menu to transfer a staff from the current workplace of the staff to another workplace for the same shift and position.

 

The following details are displayed:

  1. Workplace - The current workplace is displayed.

  2. Shift - The shifts scheduled to the staff are displayed.

  3. Start Time - The start time and date of the shift is displayed.

  4. End Time - The end time and date of the shift is displayed.

 

Enter the following details:

  1. Schedule Date - The scheduled date is displayed.

  2. Transfer from - The current workplace from where the staff has to be transferred is displayed.

  3. Allocated Shifts - The shifts scheduled to the staff are displayed.

  4. Select - Select the check box to select the required allocated shifts. This is a mandatory field.

  5. Transfer to - Select the workplace to which the staff has to be transferred from the list box. This is a mandatory field. The staff will be transferred to the same shift that was selected by clicking the Select checkbox.

Transfer cannot be done for a shift in which the staff is currently working i.e. the start time of the shift getting transferred should be greater than the end time of the current shift if the transfer is on the same day.

  1. Reason - Select a valid reason from the list. The reason is defined in the "Reason" function for Transfer. This is a mandatory field.

  2. Remarks - Enter the remarks if required.

  3. Click [Apply] to save the record. The system prompts you before finalizing the action. Click [OK] to confirm the action.
