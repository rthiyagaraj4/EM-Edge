Skill for Position

From this function you can Create an association of one or more Skills to a particular Position or associate one or more Positions to a particular Skill. You can Query for the required Skill for Position records.

This function is used to limit relevant skills to certain positions. You can define skills mandatory for a particular position that helps in planning staff training and evaluation. The Positions are defined in the Application Masters module.

Creating an association of Skills for Positions

From the RS Code Setup, select Skill for Position function.

The Skill for Position screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select By Position tab to associate one or more Skills to a particular Position or select By Skill tab to associate one or more Positions to a particular Skill.

  3. Enter the details in the mandatory (*) fields. <Related Field Description>

  4. Click [Apply] to save the association of Skills to Position or [Reset] to clear the entered data.

  5. Click [Query] to search for associated Skill for Position records.
