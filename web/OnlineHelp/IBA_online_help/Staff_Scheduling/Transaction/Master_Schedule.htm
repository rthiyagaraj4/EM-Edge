Master Schedule

From this function, you can Create a Master Schedule for a workplace.

This function allows the users to define shift patterns for Workplace/Staff combinations. It is particularly helpful in facilities where staff is scheduled for a given period, in a specific shift pattern, well in advance. Master Schedule is created based on the Work Cycle Basis as defined in the Parameter setup. To further enable easy definition of the master schedules, shift patterns are also used. Using these master schedules, work schedules are generated for the staff for a specified period. There is also a provision to view the staff allocation against the actual requirement defined using the function "Workplace Requirement", enabling utilizing the staff appropriately.

Access to the workplaces for which you can create a master schedule is controlled through a flag 'Allocate', setup using the admin function 'Access Rights'. If you do not have access to any workplace, then you are alerted accordingly and the control exits from the function.

The Staff (Practitioner and/or Other Staff) pertaining to the logged in facility only must be considered for the Master Schedule.

Any update to Master Schedules play a role only during the next generation of schedules. If any work schedules are already created, no action is taken on such generated work schedule.

Creating Master Schedule

From RS Transactions, select Master Schedule function.

The Master Schedule screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Show Details] to view the workplace requirements.

  3. The schedule status for each day has a distinct color to indicate different statuses (Optimal, Below Optimal, Above Optimal, Unproductive and On Call).

  4. The screen displays the workplace details.

  5. Click [Apply] to save the master schedule.
