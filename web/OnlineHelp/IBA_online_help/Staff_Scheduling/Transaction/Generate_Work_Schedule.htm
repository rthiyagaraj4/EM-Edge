Generate Work Schedule

From this function, you can Generate a Work Schedule created for a workplace.

This function allows you to generate a Work Schedule based on master schedules. Work schedules cannot be generated without setting up master schedules. You can generate a schedule for the workplaces to which you have access.

Generating Work Schedule

From RS Transactions, select Generate Work Schedule function.

The Generate Work Schedule screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Generate Work Schedule].

  3. Select the check box to indicate the selection of the staff to generate the work schedule. To generate a work schedule for all the displayed staffs, select the check box provided in the column header

  4. Click [Apply] to generate the work schedule.
