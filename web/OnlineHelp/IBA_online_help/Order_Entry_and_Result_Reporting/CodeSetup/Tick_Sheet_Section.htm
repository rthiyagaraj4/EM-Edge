Tick Sheet Section

From this function you can Create sections for a Tick Sheet and specify the sequence in which the section has to display for the selected Tick Sheet. You can Query for a desired Tick Sheet Section.

The function Tick Sheet Section is used to define various sections of a Tick Sheet. Order Catalogs are grouped under sections of a ‘Tick Sheet’ in the function ‘Order Catalog by Tick Sheet Section’. It is mandatory to define a section for a Tick Sheet. An Order Catalog can be associated to a tick sheet only if the Tick Sheet Section is defined. This function helps to categorize the Tick Sheet into various sections. You can create 10 Tick Sheet Section records at a time and associate it to an Order Category and Tick Sheet.

Creating a Tick Sheet Section

From the OR Code Setup, select the Tick Sheet Section function.

The Tick Sheet Section screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a record or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Tick Sheet Section.

You can define 10 Tick Sheet Sections at a time and associate it to an Order Category and a Tick Sheet.
