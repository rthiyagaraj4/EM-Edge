Replaceable Orderable

From this function you can Associate replaceable Order Catalogs for an Order Catalog and Query for an existing orderable.

Replaceable Orderable function helps you to associate replaceable Order Catalogs for a selected Order Catalog. When a single replaceable Order Catalog is associated, the selected Order Catalog is replaced with the associated Order Catalog. If there are, multiple replaceable Order Catalogs associated to the Order Catalog, a list of the replaceable Order Catalogs associated to the Order Catalog gets displayed. The Practitioner can select from the list to replace the existing Order Catalog during Result Reporting.  

Associating a Replaceable Order Catalog to an Order Catalog

From the OR Code Setup menu, select the Replaceable Orderable function.

The Replaceable Orderable screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select an Order Catalog by clicking [?] function. This is a mandatory field.

  3. Select a Replaceable Order Catalog by clicking [?] function. Selecting a Replaceable Order Catalog is mandatory. You can associate up to 10 replaceable Order Catalogs to the main Order Catalog.  

  4. Click [Apply] to save the record or [Reset] to clear the entered data.

  5. Click [Delete] to delete a queried record.
