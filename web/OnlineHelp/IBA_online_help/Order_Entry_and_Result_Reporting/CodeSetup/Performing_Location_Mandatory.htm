Performing Location Mandatory

From this function you can Make Performing Locations as mandatory while placing orders by defining Order Category for required Patient Class.

This function allows you to select the order category for each patient class such as inpatient, outpatient, emergency, day care and external. While placing orders, the performing location is mandatory only for the patient classes selected for each order category.

 

For Radiology order category, all patient class types are selected by default in disable mode.

For Laboratory order category, the system does not consider BT orders so while placing BT orders the performing location is not mandatory.

Defining Order Category for Patient Class

From the OR Code Setup > OR Perform Locn Setup, select the Performing Location Mandatory function.

The Performing Location Mandatory screen gets displayed.

  1. By default, the Order Category appear column wise while the Patient Class (Day Care, OP, IP, Emergency, External) appear row wise.

  2. Select the check box against the Order Category for each patient class as required.

  3. Click [Apply] to save the record or [Reset] to clear the entered data.
