Order Type

From this function you can Create an Order Type and define the characteristics like auto registration, default timings and default order repeats for an Order Type and Query for a desired Order Type.

The Order Type function helps in defining various Order Types. An Order Type is attached to a system defined Order Category. The characteristics for an Order Type differs based on the Order Category selected. The Order Category attributes are system defined and the associated Order Type inherits the characteristics of the Order Category. An Order Catalog forms the basic unit, which is grouped under an Order Type.

Creating an Order Type

From the OR Code Setup, select the Order Type function.

The Order Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a created Order Type or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Order Type.
