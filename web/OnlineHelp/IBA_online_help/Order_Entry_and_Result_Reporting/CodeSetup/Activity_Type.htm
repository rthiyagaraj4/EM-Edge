Activity Type

From this function you can Create an Activity Type and Query for an existing Activity Type record.

Activity Type function helps to define different Activity Types. Activity type is the third level of classification apart from Order Category and Order Type for searching and analysis of Order Catalog. You can create multiple Activity Types under an Order Type. The Activity Type defined is used while creating an Order Catalog. You can create 10 Activity Type records at a time using this function.

Creating an Activity Type

From the OR Code Setup > OR Code Setup, select the Activity Type function.

The Activity Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save an Activity Type created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Activity Type record.
