Order Set

From this function you can Create an Order Set and Query for an existing Order Set.

The function Order Set is used to define Order Sets and their characteristics. An Order Set is a collection of Order Catalogs. Order Catalogs associated to a particular Order Category gets listed. You can select the Order Catalogs from the list and associate it to the Order Set. An Order Set is designed specifically for a Service, Practitioner, Location or Diagnosis. While placing an order, you can select the Order Set. An order is placed for the Order Catalogs associated to the Order Set.

Creating an Order Set

From the OR Code Setup, select the Order Set function.

The Order Set screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Enter the details in the following Tab pages.
