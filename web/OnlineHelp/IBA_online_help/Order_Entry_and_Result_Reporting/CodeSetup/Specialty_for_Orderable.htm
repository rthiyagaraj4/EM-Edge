Specialty for Orderable

From this function, you can Create an association of an Order Catalog to a Specialty.

Associating Specialty to Orderables

From the OR Code Setup > select the Specialty for Orderable function.

The Specialty for Orderable screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select Group By as ‘Order Catalog’ or ‘Specialty’.

  3. If Group By is selected as Order Catalog, select the Order Catalog using the [?] function. The Order Catalogs created in Order Catalog function are displayed. If Group By is selected as Specialty, select the Specialty using the [?] function. The Specialties created in the Specialty function of Application Masters module are displayed. This is a mandatory field.

  4. The Select Tab is displayed, by default. The screen displays alphabets from ‘A to Z’ and ‘Others’ which are hyperlinked. Each alphabet contains Specialties or Order Catalogs based on the selected 'Group By' field. The ‘Others’ hyperlink contains Specialties or Order Catalogs which begins with characters other than alphabets. Click on the required alphabet and the Specialties or Order Catalogs beginning with the selected alphabet are listed.

  5. Select the check box against the required Specialties or Order Catalogs for association.

  6. Click 'Associate' Tab to view the selected Specialties or Order Catalogs. Clear the check box to dissociate the association.

  7. Click [Apply] to save the association of Specialties to Order Catalogs or vice versa. Click [Reset] to clear the entered data.
