Index Order Set

From this function you can Create an association of indexed Order Sets to Practitioner, Source, Service or Diagnosis and Query for the existing records.

The function Index Order Set is used to associate indexed Order Sets to a practitioner, facility, service or diagnosis. Order Sets are indexed, if the 'Index' check box is selected while defining the Order Set.  If the Order Set is indexed, only the associated source, service and practitioner can access the Order Set selected while placing the order. The function contains two tabs, namely, Select and Associate. By default, the Select tab is selected. Using the Associate tab you can view the practitioner, location or service associated previously.

Creating an association of Indexed Order Set to Practitioner, Location, Service or Diagnosis

From the OR Code Setup, select the Index Order Set function.

The Index Order Set screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to associate an indexed Order Set to Practitioners, Locations, Services or Diagnosis Code. Click [Reset] to clear the entered data.

  4. Click [Query] to search for an existing record.
