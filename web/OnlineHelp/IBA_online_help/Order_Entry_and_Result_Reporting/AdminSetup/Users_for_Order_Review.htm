Users for Order Review

From this function you can Assign order review rights to Practitioners or Responsibilities.

Order review rights like Authorization, Co-sign and/or Special Approval are assigned to a Practitioner or Responsibility. The rights defined for a Practitioner overrides the rights defined for Responsibility. The rights to review orders of each category are defined in this function.

Assigning Order Review Rights

From the OR Admin Setup, select the Users for Order Review function.

The Users for Order Review screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Practitioner or Responsibility option.

  3. On selecting ‘Practitioner’ option, the Practitioner search function is displayed.

    1. Select the required Practitioner using the [?] function.

    2. The Order Categories are displayed with the Order Review rights check boxes against the Order Category.

    3. Select the 'Authorization' check box to assign rights to authorize orders of the order category. The selected practitioner is provided right to authorize orders of the order category. This check box gets enabled or disabled as per the setup made in ‘Control by Privilege group Setup’ under OR Parameter.

    4. Select the Cosign check box to assign 'Cosign' rights to the selected Practitioner to cosign the orders of the Order Category.

    5. Select the Spl Approval check box to provide rights for Special Approval of orders belonging to the Order Category.

    6. Select the Quota Limit Approval check box to provide rights to allow quota limit approval for pharmacy order Category.

  4. On selecting, ‘Responsibility’ option, the list option to select the Responsibility is displayed.

    1. Select the required Responsibility from the list. The Responsibilities defined in the System Manager module are displayed in the list. The rights are assigned for the selected Responsibility. This field is mandatory.

    2. The Order Categories are displayed with the Order Review rights check boxes against the Order Category.

    3. Select the 'Authorization' check box to assign rights to authorize orders of the order category. The selected practitioner is provided right to authorize orders of the order category. This check box gets enabled or disabled as per the setup made in ‘Control by Privilege group Setup’ under OR Parameter.

    4. Select the Cosign check box to assign 'Cosign' rights to the selected Practitioner to cosign the orders of the Order Category.

    5. Select the Spl Approval check box to provide rights for Special Approval of orders belonging to the Order Category.

    6. Select the Quota Limit Approval check box to provide rights to allow quota limit approval for pharmacy order Category.

  5. Click [Apply] to save the assigned Order Review Rights or [Reset] to clear the entered data.

  6. Click [Query] to search for existing Order Review Rights.

  7. Click [Delete] to delete the queried record.

 

If the Authorization, Cosign and Special Approval are not selected for the Order Categories, the Practitioner or Responsibility will not have the rights for Order Review for the respective category. The access for Order authorization, Special Approval and Cosign function are restricted, if the Practitioner or Responsibility does not have rights to review any Order Category.
