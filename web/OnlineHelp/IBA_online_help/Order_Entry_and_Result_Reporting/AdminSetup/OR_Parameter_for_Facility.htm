OR Parameter for Facility

From this function you can Define and maintain the OR Parameter Setup specific to a Facility.

The Grace Period for inclusion of expired orders and discontinued orders into the active order list are set using this function. Using this function you can set up print defaults for Order forms, consent forms and Result reports. The time for renewing and reactivating single continuing orders on hold are set. Only one record can be created for a facility. It is mandatory to set up the Parameter before any operations are performed as these record stores important details about Order Entry and Results Reporting module specific to a facility.

Creating Parameter for Facility

From the OR Admin Setup, select the OR Parameter for Facility function.

The OR Parameter for Facility screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [Apply] to define OR Parameter for Facility values or [Reset] to revert to the previously defined values.
