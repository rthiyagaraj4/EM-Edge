MDS for Diagnosis Mandatory

From this function, you can Define Minimum data set setup to be considered while placing order if the patient primary diagnosis is not recorded for selected patient encounter.

This function allows you to define Minimum data set setup to be considered while placing order if the patient primary diagnosis is not recorded for selected patient encounter.

There by, the system restricts the user to place order if the patient primary diagnosis is not recorded for selected patient encounter in Existing Orders – New Orders, Copy Previous Orders, Previous Orders, Place Order screens in CA and OR modules.

Defining MDS for Diagnosis Mandatory

From the OR Admin Setup, select the MDS for Diagnosis Mandatory function.

The MDS for Diagnosis Mandatory screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the required minimum data set from the ‘Minimum Data Set’ drop-down list. The listed values include Specialty, Order Category and Patient Class. By default, the value 'Specialty' gets displayed. This field is mandatory.  

  3. Select the Terminology Set from the drop-down list. The Terminology Set values get populated in the list based on the Minimum Data Set selected. By default, the value 'ICD10' gets displayed. This field is mandatory.

  4. When Minimum Data Set is selected as 'Specialty', all Specialties get displayed with a check box. Select the required Specialty or select the 'Select All' check box to select all the specialties. It is mandatory to select a specialty.

  5. When Minimum  Data Set is selected as 'Specialty', the Primary Diagnosis Mandatory check box gets displayed. Select the check box to restrict the user to place order if the patient primary diagnosis is not recorded for selected patient encounter in existing orders, New orders. By default, the check box is cleared.

  6. When Minimum Data Set is selected as 'Order Category', all Order Categories get displayed with a check box. Select the required Order Category or select the 'Select All' check box to select all the order categories. It is mandatory to select an order category.

  7. When Minimum Data Set is selected as 'Patient Class', all Patient Class get displayed with a check box. Select the required Patient Class or select the 'Select All' check box to select all the patient classes. It is mandatory to select a patient class.

  8. Click [Apply] to save the settings done or click [Reset] to clear the data.

  9. Based on the setup done, the system restricts the user to place order if the patient primary diagnosis is not recorded for selected patient encounter in Existing Orders – New Orders, Copy Previous Orders, Place Order screens in CA and OR modules.
