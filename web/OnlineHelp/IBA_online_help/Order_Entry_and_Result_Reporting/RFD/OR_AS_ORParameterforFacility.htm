Enter the following details to define OR Parameter for Facility record:

 

  1. Facility - The logged in facility is displayed and disabled.

 

Order Forms

  1. Order Forms - To print Order Forms at the Resulting Location, select the corresponding check box under ‘Print at Destination’ heading. To print Order Forms at the source (order placing department), select the corresponding check box under 'Print at Source' heading. The check box is cleared, by default.

 

Result Reports

  1. Print at Department - Select the check box to print reports at the Resulting Department or the report will not be printed at the Resulting Department.

  2. Print at Other Location - Select the required location from the list. During the initial setup 'Do not print' is displayed, by default. The listed values are 'Patient Loc', 'Specific Loc' and 'Do not print'. Select the required location from the list. The results are printed at the selected location. If ‘Specific Loc’ is selected, a Location Type and Location has to be selected.

• Patient Location - Select 'Patient Location' to print Result Reports at the respective 'Clinic' or 'Nursing Unit', where the patient availed consultation (Outpatient) or admitted (for Inpatients).

• Specific Location - Select 'Specific Location' to print 'Result Reports' at a specified location like 'Clinic' or 'Nursing Unit'.

If 'Clinic' is selected, all the available 'Clinics' in the logged in facility are listed in the corresponding list. The Result Report is printed at the selected 'Clinic'.

If 'Nursing Unit' is selected, all the available 'Nursing Units' in the logged in facility are listed in the corresponding list. The Result Report is printed at the selected 'Nursing Unit'.

• Do not Print - Select 'Do not Print' and 'Result Reports' are not printed in other locations. If 'Print at Dept' check box is selected and 'Do not Print' is selected from 'Print at Other Location' list, the Result Reports cannot be printed at other locations and can only be printed at the Ordering Location.

  1. Print Location for Discharged Patients - Select the Print Location for Discharged Patients from the list. The values listed are 'Patient Location' 'Specific Location' and 'Do not Print'.

• Patient Location - Select 'Patient Location' to print Result Reports at the respective 'Clinic' or 'Nursing Unit', where the patient has taken consultation (Outpatient) or was admitted (for Inpatients).

•  Specific Location - Select 'Specific Location' to print 'Result Reports' at a specified location like 'clinic' or 'Nursing Unit'.

If 'Clinic' is selected, all the available 'Clinics' are listed in the corresponding list. The Result Report is printed at the selected 'Clinic'.

If 'Nursing Unit' is selected, all the available 'Nursing Units' are listed in the corresponding list. The Result Report is printed at the selected 'Nursing Unit'.

•  Do not Print - Select 'Do not Print' and 'Result Reports' are not printed for discharged patients.

 

Parameters for Single Continuing Orders

  1. Renew: Period Behind [Hrs] - Enter the duration in numeric format of three digits maximum length. You can renew single continuing orders placed within the specified duration.

  2. Renew: Period Ahead [Hrs] - Enter the maximum hours in numeric format of three digits. You can renew single continuing orders placed (future orders) within the specified duration.

  3. Reactivate on Hold: Period Ahead [Hrs] - Enter the maximum hours in numeric format of three digits to reactivate orders, which are on hold.

 

Grace time for Inclusion in Active Orders

  1. Expired Orders [Hrs] - Enter the duration in numeric format of three digits. The Grace Period to include Expired Orders into the active orders list is specified. This is applicable only for single continuing orders.

  2. Discontinued Orders [Hrs] - Enter the duration in numeric format of three digits. The Grace Period to include discontinued orders into the active orders list is specified. This is applicable only for single continuing orders.

 

Specimen Collection

  1. Period Behind [] - The number of days specified in this field is applicable while listing 'Print Specimen Labels' in the Clinician Access homepage. It is the number of days based on which the old specimens are searched for in 'Print Specimen Labels' in Clinician Access homepage.

The value specified here is the number of days the system allows users to generate specimen labels.

  1. Period Ahead [] - The number of days specified in this field is applicable while listing 'Print Specimen Labels' in Clinician Access activities. It is the number of days ahead, based on which the future specimens are searched for in 'Print Specimen Labels' in clinician homepage.

The value specified here is the number of days ahead the system allows you to generate specimen labels.

  1. Select all option for Record Collection –  Select the check box if you want to enable the ‘Select’ check box in Record Collection screen so that you can select all the records of a particular patient and record the collection. By default, this check box is not selected.

 

Interface Particulars

  1. Billing - Select the check box to interface the 'Billing' module with 'Order Entry and Results Reporting' module. The check box gets displayed if Billing module is installed.

  2. Display Billing information – Select this check box if you want to view the billing information in the Pharmacy Orders screen. This check box is selected by default. If this check box is cleared, no billing information will be displayed to the user. Note: This check box will be enabled only if the check box Billing is selected.

  3. Default Refund Option For Discontinue Order  - Select the system-defined values from the list. The listed values are ‘Yes’ and ‘No’. The selected values will be defaulted in the discontinue reason screen.

  4. Invoke Settlement Screen For Patient Class - Select the check box to settle the bill or issue a discount while placing orders for the respective patient class. Select 'XT' to settle the bill or issue discount for external patients. Select 'OP' to settle the bill or issue discount for Outpatients. Select 'EM' to settle the bill or issue discount for Emergency patients.  

  5. Invoke Print Report For Billing - Select the system-defined values from the list. The listed values are ‘Yes’ and ‘No’. Select 'Yes' to invoke print report for billing.

  6. External Link – Select this check box to display the External Link  icon in the Results screen for radiology orders in CA module.  This icon is displayed in the Clinical Event History, View Chart Summary, Order Tracking – Results Order screens and 'Events Occurred In 24 Hours' widget in View Chart Summary screen.

  7. Split Unbilled orders and create New ID during Register Order –  Select the check box to split the unbilled orders and create new order ID during registration. This is applicable only for orders registered from Task List in CA module. By default, the checkbox is not selected.

 

External Patient Order

  1. Referral - Enter the Referral source in alphanumeric format of 30 characters maximum length or select the Referral Source using the [?] function. The Referral Source is defined in External/Referral Source function in Application Masters module. The value selected in this field is made default internally while placing an external order for a new born without an encounter.

 

Future Order Linking Applicability

  1. Posted to Folio but not Billed / Receipted - Select the check box to consider the service charges as future orders, which are posted to folio but not billed or receipt not generated. Clear the check box to consider the service charges as the current orders, which are posted to folio and billed or receipted.

  2. Billed but not Receipted - Select the check box to consider the service charges as future orders, which are billed but the receipt not generated. Clear the check box to consider the service charges as current orders, for the services that are billed and the receipt not generated.

  3. Max allowed Period from Current Day - Enter the number of days in numeric format of two digits maximum length, to indicate maximum number of days for future orders. The days are calculated based on the current date.

 

Others

  1. Specialty Filter in Copy Previous Order – Based on the value selected in the drop-down list, a ‘Specialty’ field will appear or not in the Copy Previous Order screen. The drop-down list values include Not Applicable, All, Primary Specialty and Encounter Specialty. The value ‘Not Applicable’ is displayed by default.

  2. Default Period in Copy Previous Orders – Enter the number of days to be defaulted as the search Period in the Copy Previous Order screen. By default, the value ‘Null’ is displayed. Based on the entered number of days, the ‘From’ Date appears in the Copy Previous Order screen while the current date is displayed as ‘To’ date.

  3. Hide Specimen Type In Results - Select the check box to hide the Specimen Type in the lab results displayed in Clinical Event History, Existing Orders, Results in Chart Summary Widget, Pending Results Review on Homepage, Order Tracking, and Events in 24 Hours. By default, the check box is not selected.

  4. Hide Reporting Practitioner In Results - Select the check box to hide the Reporting Practitioner name in the lab results displayed  in Existing Orders, Results Widget and Orders Widget in View Chart Summary, Order Tracking. By default, the check box is not selected.

  5. Apply Ordering Rules by Practitioner Type To Restrict Departmental Orders from Copy Previous order – Select the check box to consider the ‘Ordering Rule by Practitioner Type’ setup to restrict placing of departmental orders from Copy Previous Orders. By default, the check box is cleared. This is applicable for all Order Catalogues except for Pharmacy order category.

 

Lab Specimen Not Registered

  1. Time Frame to alert Urgent/Stat Lab Specimen X minutes - Enter the time frame to alert when the urgent/stat lab specimens are not registered. By default, the value ‘1’ min is displayed. You can enter value from 1 to 9999 Minutes.
