Enter the following details:

 

  1. Order Catalog - Select the Order Catalog by clicking [?] function. The Order Catalogs where the Procedure Link is selected as Manual get listed. This is a mandatory field.

  2. Term Set - Select Term Set from the list. The listed values appear from Terminology set function with Procedure specific check box selected for the term set in Medical Records module. This is a mandatory field.

  3. Term Code - Enter a Term Code or select by clicking [?] function. The Term Code is displayed from Terminology Code function in Medical Records module and the 'Type' for the terminology code has to be selected as 'Procedure'. This is a mandatory field.

  4. Default - Select the check box to make the term code as 'Default' to the term set. Only one record can be selected, by default.
