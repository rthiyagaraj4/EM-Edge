Enter the following details:

 

  1. Order Category – Select the Order Category from the drop-down list. The order categories that are defined in the OR Order Category function gets listed. Based on the Order category selected here, it will be applicable for all the order types as well as order Catalog defined under it. This field is mandatory.

  2. Order Type – Select the Order Type from the drop-down list. Based on the Order Type selected here, it will be applicable for the entire order Catalog defined under it. You can define different periodicity for different order type falling under same order Category. This field is mandatory. By default, the value ‘All’ is selected.

  3. Order Catalog - Select the Order Catalog from the lookup. You can define periodicity for each order Catalog separately. This field is mandatory. By default, the value ‘All’ is displayed.

  4. Duration Value – Enter the Duration value in three characters limit for which the overrule of authorization is applicable. For example, 365 days. This field is mandatory. And within this time period, system will not ask for authorization by senior doctor, if the order is previously authorized by the senior doctor for the same patient.

  5. Duration Type  – Select the type of duration from the drop-down list. The available duration types are Minutes / Hours / Days / Weeks / Months / Years. This field is mandatory.

  6. Select Row to Remove –  Select the check box against the particular record and click ‘-‘ to remove  the selected row. Click ‘+’ button to add a new row.
