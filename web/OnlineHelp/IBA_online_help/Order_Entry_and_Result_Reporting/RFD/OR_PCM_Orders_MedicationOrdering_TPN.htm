TPN Regimen

TPN is a complete form of nutrition containing nutrients, electrolytes, minerals, trace elements and vitamins. TPN Mixtures are administered to patients through Intravenous Infusion. These infusions can be administered either using either a central line or a peripheral line. TPN mixtures are available as either standard regimens or can be custom prepared in the pharmacy based on nutritional requirements of patients.

TPN orders can be placed only after capturing the height and weight values of the selected patient. On entering the height and weight values, the BSA and BMI are calculated. Enter the required header level details as explained under ‘Rx’ tab.

 

Placing TPN Orders

  1. Enter the details in the tab pages.

Assessment

The 'Assessment ' tab is displayed by default, on selecting 'TPN Regimen'. This tab provides details on daily energy requirements including standard requirement of protein, lipids and fluid based on the age group and computed body surface area of the patient. The energy requirement is computed considering the default stress factor value. Clinician can change the default values by entering appropriate values and the corresponding macro nutrient values are calculated.

  1. Enter the details in the Energy Expenditure frame.

    1. Stress Factor – The stress factor set as default in the master code is displayed on selecting the tab. Select the required Stress Factor from the list. All the values entered in ‘Stress Factor’ master of PH master are displayed.

    2. Recommended Energy – This value is made default based on selected Stress factor. The Recommended Energy is calculated based on the weight age specified for the selected stress factor. The displayed value cannot be modified

    3. Fluid – This value is made default from ‘TPN Constituent Limit’ function of PH master code. The standard value defined for fluids is displayed. Enter the required value in numeric format of five digits maximum length.

    4. Energy to be provided – The Recommended Energy is the energy to be provided. Specify the required energy to be provided in numeric format.

    5. Protein – The standard value specified for protein in the function ‘TPN Constituent Limit’ is displayed as read only text. Normal’ is displayed as the value for protein, by default. The standard value for normal is displayed next to the list. Select the required value from the list. The listed values are High and Low. Based on the selected value, corresponding value is displayed. The maximum value of protein for ‘High’ and minimum value for ‘Low’.

    6. Lipid – The standard value specified for lipid in the function ‘TPN Constituent Limit’ is displayed as read only text. ‘Normal’ is displayed as the value for lipid, by default. The standard value for normal is displayed next to the list. Select the required value from the list. The listed values are High and Low. Based on the selected value, corresponding value is displayed. The maximum value of protein for ‘High’ and minimum value for ‘Low’.

    [Calculate Nutrients] – On updating the system defaulted value in the above fields, click [Calculate Nutrients] to calculate the recommended macronutrients for the specified values.

    Recommended Macro Nutrients

    The energy values, order quantity and strength/weight for each macronutrient are defaulted by the system for the default stress factor value and standard values of Proteins and Lipids, by default. On modifying the default values and clicking [Calculate Nutrients], the energy, order quantity and strength/weight for the macronutrients are updated accordingly.

  2. Click [Record] to store the values to order ‘Standard TPN Regimen’.

Standard Regimen

This tab is used to prescribe TPN based on a standard regimen. This tab allows selection of any one of the predefined standard regimens, which has predetermined proportion of constituents. The energy values of constituents cannot be modified to suit the nutritional requirements of the patient since this is a standard regimen, which has predetermined values.

  1. Enter the Standard Regimen Details and Administration Details in the mandatory (*) fields.

    1. Infusion Route – The infusion route specified for the selected regimen in ‘TPN Regimen’ master is displayed. Select the infusion route from the list. The listed values are ‘Central’ and ‘Peripheral’.

    2. Volume – The volume set for the selected regimen in ‘TPN Regimen’ function is displayed, by default. Specify the required volume in numeric format of three digits maximum length. The field is mandatory.

    3. Infuse Over – The Infuse over specified for the selected regimen in ‘TPN Regimen’ function is displayed as default value. The clinician can change infuse over period. Enter the period in numeric format of two digits maximum length.

    4. Frequency – The default frequency specified for the TPN order in PH Parameter function is displayed as read only text.

    5. Duration – Enter the duration in numeric format of two digits maximum length. This field is mandatory. The duration specified while defining the Order catalog attributes of the selected standard regimen is made default, by default. The entered value must be within the range specified for duration. On violating the duration value, the system performs the necessary action as specified for the regimen.

    6. Start Date – The current system start date/time are displayed, by default. The practitioner can modify the medication start date, that is pre-date or post-date the medication starting date/time based on the setting in OR Parameter. This field is mandatory.

    7. End Date – The end date/time is calculated by the system by adding the specified duration with the start date/time. The end date and time cannot be modified.

    1. Standard Regimen – Enter the standard regimen value in alphanumeric format of 30 character maximum length or select the Standard Regimen using [?] function.

    On selecting the Standard Regimen, click 'Constituents' hyperlink to view the constituents of the selected regimen.

    1. Build MAR - Medication Administration is applicable for all patient classes depending on the Parameter settings. For Inpatients, Build MAR is default checked and enabled. For remaining patient classes based upon parameter settings it will be checked and enabled.

  2. Click [Release Rx] to release the prescription or [Cancel] to clear the entered details.

Non Standard Regimen

Non Standard Regimen comprises of TPN Mixtures, which are prepared in the Pharmacy based on specific nutritive requirements of the patient.

The following details are displayed:

  1. Regimen - Default non-standard regimen provided through scripts during installation is displayed.

  2. NCP: N Ratio – The NCP: N ratio derived is displayed. The displayed ratio is based on the pre-defined macronutrients.

  3. Total - the total energy required by the patient is displayed.

  4. Build MAR – The Build MAR is default checked and enabled for inpatients. For remaining patient classes based upon parameter settings it will be checked and enabled

  5. TPN Constituent Group Tabs - All the TPN constituent groups defined in ‘TPN Constituent Group’ function are displayed. The constituents are segregated under identified generic names. They are Fluid, Macro Nutrient, Electrolytes, Trace Elements, Vitamins and Other Components. Each group comprises of one or more constituents.

  6. Constituents - On clicking individual TPN Constituent Group tabs, the element under them are listed.

  7. Require Quantity - The standard value set in ‘TPN Constituents Limit’ for individual constituent is displayed. The displayed value can be changed.

  8. Strength/Kg - This value is auto calculated by the system by dividing the required quantity and weight of the selected patient.
