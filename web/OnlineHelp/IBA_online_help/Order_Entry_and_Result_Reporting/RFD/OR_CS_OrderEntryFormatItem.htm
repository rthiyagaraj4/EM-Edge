Enter the following details:

 

  1. Field Mnemonic - Enter the Field Mnemonic in alphanumeric format of 20 characters maximum length. Mnemonics are short codes or abbreviations used. The field mnemonic is used to associate the Format Item to an Order Entry Format. This field is mandatory.

  2. Field Mnemonic Desc - Enter the Field Mnemonic Description in alphanumeric format of 20 characters maximum length. This field is mandatory. The message 'More than three records will not be accepted for this Mnemonic' is displayed if more than three clinical comments for Lab Format Items are selected.

  3. Field Type - Select the Field Type from the list. The values are system defined. The table below provides the field types and values to be entered. This field is mandatory.

 

Type

 Values

Alphanumeric

Mnemonic field is created to accept data in alphanumeric format.

Clinical Event

Specify the Clinical Event to be associated.

Date

Mnemonic field is created to accept valid date format.

Date time

Mnemonic field is created to accept valid date/time format.

Decimal Numeric

Decimal Numbers are valid for the Mnemonic defined. You can specify Minimum and Maximum value in the fields Min and Max respectively.

Discrete Measure

Specify the Discrete Measure to be associated.

Integer Numeric

Integer Numbers are valid for the Mnemonic defined. You can specify Minimum and Maximum value in the fields Min and Max respectively.

List

Drop down list is displayed for the Mnemonic defined. An additional screen pops-up, where you can specify the sequence and the description for the items to appear in the list.

Radio Group

Select Radio Group and enter the sequence number and description for the field items.   

Time

Mnemonic field is created to accept the valid Time format.

Yes/No

Check box is provided for the defined mnemonic.

 

  1. Order Category - Select the Order Category from the list. The values are system defined. This field is mandatory.

  2. Definition Type - The value 'User Defined' appears by default and you cannot change it. This indicates that the created Mnemonic is User-defined.

  3. Presentation - Select the Presentation from the list. The values displayed are Single and Multi. The presentation is enabled, only if the Field Type is selected as ‘List’.

  4. Selection - Select the value from the list. The values displayed are Single and Multi. By default, 'Single' is displayed. If single is selected, the mnemonic is used only once in an order entry format or it can be used multiple times.

  5. Hint - Enter the Hint in alphanumeric format of 200 characters maximum length. This text recorded is displayed as a tool tip for the mnemonic when the order entry format to which the mnemonic is attached is invoked in the 'Place Order' function.

  6. Enabled - Select the check box to enable the record in the transaction functions. The check box is selected, by default.
