Enter the following details:

 

  1. Practitioner Type - Select the required Practitioner Type from the list. On selecting the Practitioner Type, the list of Order Categories are displayed with the Ordering Rules.

  2. Order Category – The Order Categories are listed. These categories are system defined.

  3. Order ‘Dept Only’ Catalogs – Select the check box corresponding to the required Order Category. Practitioner of the selected Practitioner Type can place ‘Department Only’ orders for the selected Order Categories only.

  4. Force Authorization – Select the values from the list. The listed values are ‘Authorize Force Authorization’ and ‘Force Authorization’. If ‘Authorize Force Authorization’ is selected the orders which requires force authorization is forwarded to the selected practitioner for authorization.

  5. Force Co–Sign – Select the check box if the selected Order Category requires co-signing.

  6. Time Based Rule – Select the required value from the drop-down list. The list values include ‘Day After discharge/OP or AE Checkout’, ‘Hour After Discharge/OP or AE Checkout’, ‘Day After Discharge Advice’, and ‘Hour After Discharge Advice’. By default, the value ‘Select’ is displayed.

  7. Expiry Period – Enter the Expiry Period for the selected rule in numeric format of two digits maximum length. This is a mandatory field. You can enter from 0 to 99 days.

  8. These two fields appear only if the Apply Time Based Rules applicable in Ordering Rule By Practitioner Type check box is selected in the OR Parameter screen.

  9. Based on the defined Time Based Rules and Expiry period, you can place orders after discharge or checkout. The system considers this rule as first priority and overrides with Ordering Rule by Patient class.
