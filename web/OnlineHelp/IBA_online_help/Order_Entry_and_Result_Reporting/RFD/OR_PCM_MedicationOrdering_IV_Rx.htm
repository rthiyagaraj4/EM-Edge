IV Rx

Practitioner use ‘IV Rx’ function to prescribe intravenous medications to the patients of the logged in facility. Practitioner can place ‘Continuous’ and ‘Intermittent’ orders under ‘IV Rx’. The Continuous Orders are orders without frequency and duration which cannot exceed more than 24 hours. The Intermittent Orders are orders with specified frequency and duration that can exceed a day (more than 24 hours). The function provides for three types of IV medication namely:

• IV with Additives – This option allows a practitioner to prescribe Intravenous fluid with additives. A practitioner is permitted to select only one IV fluid and required additive drugs. Only continuous orders can be placed.

• IV Without Additives – This option allows a practitioner to prescribe only the Intravenous fluid. A maximum of four Intravenous fluids can be prescribed. Duplication of IV fluid is allowed. This option allows placing of ‘Continuous’ orders only.

• IV Intermittent – This option allows a practitioner to prescribe one Intravenous fluid and an IV drug. Only Intermittent orders can be placed using this option.

 

Placing IV orders

 

  1. Click [IV Rx] to prescribe Intravenous Medication.

  2. Select the Order Type from the list. The listed Order Types are IV with Additives, IV without Additives and IV Intermittent. The value ‘IV with Additives’ is displayed, by default. The listed values for ‘IV with Additives’ and ‘IV Intermittent’ are recorded under ‘Process Name’ tab in ‘PH Parameter’ function.

    A Practitioner can place ‘Intermittent’ orders using IV Intermittent option. The system allows selecting an IV Drug and vehicle or fluid only.

    1. Enter the Drug Name in alphanumeric format of 60 characters maximum length or select the required Drug using the [?] function. The Drugs with IV characteristic as ‘IV Ingredient’ are available for selection. This is a mandatory field. On selecting the Drug, the Dosage details are displayed.

    Calculate By

    1. The Dosage option is selected, by default. Select the Dosage or Body wt/BSA option.

      Enter the following details:

      1. Body wt/BSA - Select the option to define the additive by Strength and by Quantity.

      2. Dose - Enter the Dose in numeric format of four digits maximum length and select the UOM from the list. The Strength UOM of the Drug is displayed, by default.

      3. /KG or M2 - Based on the Drug Master setup KG or M2 is displayed, by default. If Standard Unit Dose for the selected Item is defined in the Drug Dosage Limit function, where Calculate Dosage By field is selected as 'BSA'; the system automatically displays the value as BSA.

       

      The Total Dose per Administration is displayed. The system displays the dose based on the setup made in Drug Dosage Limit function. The Standard Dose is multiplied with BSA or the weight based on the Calculate Dosage By setup.

      Enter the following details:

      1. Dosage By - Based on the Drug Setup, if ‘Dosage By’ is ‘Strength’, the list contains Strength and Quantity. The practitioner can define dosage details either as Strength or Quantity. If ‘Dosage By’ is ‘Quantity’, the displayed value cannot be changed.

      2. Dosage By: Strength

      This value is displayed provided base strength is defined for the selected drug. The ‘Strength’ field displays the base strength value and unit of measurement. The practitioner can change only the strength value.

      • Dosage By: Quantity

      The base quantity and unit of measurement set for the selected drug are displayed for ‘Dose’ field. The practitioner can change both quantity and unit of measurement. The unit of measurement values is listed based on the following conditions:

      The unit of measurement defined with nature as ‘quantity’ and belonging to the list used for defining prescription base unit.

      The unit of measurement defined with nature as ‘Strength’ and having an equivalent value, which is a quantity.

      The unit of measurement defined as ‘Quantity’ and having an equivalent to a value that is of nature ‘Strength’ and belonging to the list used for defining prescription base unit.

    2. Select the Admixture check box to indicate the Pharmacist to prepare admixture for the order.

    3. Select the Build MAR check box if the order is applicable for Medication Administration. Based on the PH Parameter for Facility settings the check box is selected, by default. Medication Administration is applicable for all Patient Classes based on the Parameter settings. For Inpatients, Build MAR is selected by default.

    4. Enter the required Fluid Name in alphanumeric format of 40 characters maximum length or select the required Fluid Name using [?] function. The Drugs for which the IV Fluid check box is selected in the IV Characteristics tab of Drug function in the Pharmacy Management module are displayed for selection. This is a mandatory field.

    5. On selecting a Fluid, the Volume is displayed.

    6. The Administration details of the Intravenous Intermittent orders are displayed.

      Enter the following details:

      1. Infusion Rate – The infusion rate is calculated by dividing the Fluid volume with Infuse over value. The displayed value can be modified. For ‘Continuous’ orders, ‘Hours’ displays, by default that cannot be changed. But for ‘Intermittent’, a list displays for the practitioner to select. The listed values are ‘Hours’ and ‘Minutes’.

      2. Infuse Over – The ‘Infuse Over’ specified for the selected IV Fluid (in Drug Master function) is displayed by default. The practitioner can specify the required period in numeric format of two digits. Based on the modified Infuse over value, the Infusion rate is calculated. The unit of measurement displayed corresponds to the time component value selected for Infusion Rate.

      3. Frequency – The frequency specified for the selected drug in the drug master is displayed. Select the required value from the list. The listed values are Scheduled frequencies associated to the Order Category ‘Pharmacy’ and associated to Drug in ‘Drug master’ function.

    7. Click Admin Rate hyperlink to view the Intravenous Intermittent administration rate. The ‘Admin Rate’ hyperlink displays once the system calculates the ‘Infusion Rate’.

    8. Click [Release Rx] to release the prescribed prescription or [Reset] to clear the entered details.

    9. You are prompted with a message to print the Intravenous Intermittent details on releasing. Click [OK] to print the prescription or [Cancel] to cancel the printing.

    10. On releasing the prescription, a unique Order ID is generated. Order details can be viewed under ‘Existing Orders’ function.

    The Practitioner can place Intravenous Medication without Additives under IV Rx. The system allows selecting a maximum of six fluids at a time for prescribing. Each IV Fluid is considered as a single order. Duplication of a fluid is permitted.

    1. Select the Build MAR if Medication Administration is applicable for the selected patient. Based on the PH Parameter for Facility settings the check box is selected, by default. Medication Administration is applicable for all Patient Classes based on the Parameter settings. For Inpatients, Build MAR is selected by default.

    2. Enter the required Fluid Name in alphanumeric format of 40 characters maximum length or select the required Fluid Name using [?] function. The Drugs for which the IV Fluid check box is selected in the IV Characteristics tab of Drug function in the Pharmacy Management module are displayed for selection. This is a mandatory field.

    3. On selecting a Fluid, the Fluid details are displayed and the next frame to add the Fluid is enabled.

      1. Dispense Location – The Dispense Location of the fluid is displayed as set in the Routing Location field in the Order Routing screen.

      2. Volume - Measure of the fluid in terms of base unit is displayed and you can change the volume.

      3. Infusion Rate – The infusion rate is calculated by dividing the Fluid volume and Infuse over values.

      4. Infuse Over – The ‘Infuse Over’ specified for the selected IV Fluid (in Drug Master Function) is displayed by default. The Practitioner can specify the required period in numeric format of two digits. The Infuse Over value is displayed in hours and minutes format. For ‘Continuous’ orders, ‘Hours’ displays by default that cannot be changed. Infuse over cannot exceed more than 24 hours for ‘Continuous’ orders.  

      5. Duration for Repeat – Enter the duration in numeric format of two digits maximum length. The specified duration has to be greater than or equal to ‘Infuse Over’ value. Duration cannot exceed 24 hours for ‘Continuous’ orders.  

      6. Start Time - The system start date/time are displayed, by default. The practitioner can modify, that is pre-date or post-date the medication starting date/time based on the setting in Order Entry and Result Reporting Parameter. This field is mandatory.

      7. End Time – The end date/time is calculated by the system by adding the specified duration with the start time.

      Duplication of fluid is allowed for the Type ‘IV without Additives’. The system defaults start date/time of the duplicated drug with the end date/time of the original drug. This start date/time cannot be modified. The end date/time is calculated by adding duration to the start date/time.

    4. Click [Preview Rx] to view the Patient details and entered Fluid details.

    5. Click [Release Rx] to release the prescribed prescription or [Reset] to clear the entered details.

    6. You are prompted with a message to print the Intravenous Prescription details on releasing. Click [OK] to print the prescription or [Cancel] to cancel the printing.

    7. On releasing the prescription, a unique Order ID is generated. Order details can be viewed under ‘Existing Orders’ function.

    7. The Indicators are displayed with unique color code against to Additive Drug.

    8. Enter the Fluid details in the Fluids frame.

    9. The Administration details of the Intravenous medication are displayed.

    10. Click Admin Rate hyperlink to view the IV administration rate. The ‘Admin Rate’ hyperlink displays once the system calculates the ‘Infusion Rate’.

    11. Click [Preview Rx] to view the Intravenous medication details before releasing the order. The Patient details and the details recorded while placing Intravenous Medication are displayed as read-only text.

    12. Click [Release Rx] to release the prescribed prescription or [Reset] to clear the entered details.

  3. Click Multi Flow Rate hyperlink to allow the prescribing practitioner to record multi flow rate for a particular IV Medication with different infusion over time.

  4. Click 'MFR Remarks' hyperlink to record the Multi Flow Rate remarks. The MFR Hyperlink is displayed if the Multiple Flow Rates are recorded. Enter the remarks in alphanumeric format of 255 characters maximum length.

  5. Click [Prev Orders] to view the Previous Order details.

    Enter the following query criteria to get the previous orders:

    1. Patient Class – Select the required class from the list. The listed values are Both, Inpatient and Outpatient.

    2. Orders – Select the option as either Self Related or All Practitioners. If ‘Self Related’ is selected, then all previous orders placed by the prescribing practitioner are listed. If ‘All Practitioners’ is selected, then orders placed by all the practitioners are listed.

    3. Order status - Select the order status from the list. The listed values are All, Cancelled, Discontinued, and Completed.

    4. Group By – Select the option as either Medication Orders or Drug. If the selected option is ‘medication Orders’, then drugs are grouped under their respective order numbers. If the selected option is ‘Drug’, then prescribed drugs are grouped alphabetically.

    5. Patient ID - The selected patient ID is displayed as read-only text.

    6. Order Date [From-To] - The order period displayed is (system date – three months) for ‘From’ date and current system date as ‘To’ date, by default. Specify the required period in date format of dd/mm/yyyy. These fields are mandatory.

    On entering the required details click [Search] to invoke the search process. The following details are displayed.

    1. Select – Select the check box to select the drug to be copied for the current prescription. On checking the check box, the drug(s) under the order are selected. Order date/time, ordering practitioner’s ID and ordering location are displayed as header contents.

    2. Status Indicator – The following indicators are displayed based on the status.

     

    Click [Preview] to view the selected Order, [Copy] to copy the selected Order, [Close] to exit from the Previous Orders screen.

  6. Click [Current Rx] to view the medications prescribed and currently under usage.

  7. Click [Drug Profile] to view the Drug Profile of the selected patient.

    The ‘Patient Drug Profile’ screen gets displayed with the active medication orders. The practitioner can view complete information on Drugs prescribed, Prescribed quantity, Dispensing status, Dispensing quantity, Prescribing practitioner, Ordering location and others under drug profile. Option to view the profile based on patient class is provided. In addition to viewing the profile, option to cancel or discontinue drug based on status is provided.

    You can view the orders with the Status: All, Active, Completed, Future Rx, Discontinued, On Hold, Returned and Cancelled.

    Click Remarks hyperlink to view the Drug remarks. Click [Close] to exit from the remarks screen. Click Dispensed Locns hyperlink to view the Dispensed Location details.
