Enter the following details:

 

  1. Function - Select the function from the list. The values displayed are 'All', 'Orders', 'External Orders' and 'Medication Administration'.

  2. Patient Class - Select the Patient Class from the list. The values displayed are 'Inpatient', 'Outpatient', 'External', 'Day care' and 'Emergency'.

  3. Practitioner - Select the Practitioner for whom the 'User Defined Tab Name' has to appear in patient chart. Practitioner(s) or 'All' practitioners can be selected using [?] function. This is a mandatory field.

  4. Tab Name - The system defined tab names gets displayed.

  5. User Defined Tab Name - Enter the User defined tab name for a system defined tab name in the text box. The tab name recorded appears in the Inpatient chart.

  6. Sequence No - Enter the Sequence Number in numeric format not exceeding one digit. The tabs are displayed in the Inpatient chart based on the order specified.

  7. Applicable To - Select the check box to indicate that the tab is displayed in the patient Chart. The check box is selected, by default.
