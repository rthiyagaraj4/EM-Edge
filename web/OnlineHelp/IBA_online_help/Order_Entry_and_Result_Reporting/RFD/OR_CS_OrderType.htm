Enter the following details:

 

  1. Code - Enter the Code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Long Description - Enter the Long Description of the Order Type in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the Short Description of the Order Type in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Order Category - Select the Order Category from the list. The values are pre-defined. This field is mandatory. The system defined Order Categories are as follows:

  5. Consumables

  7. Dietary

  8. Laboratory

  9. Miscellaneous

  10. Monitoring

  11. OT Surgery

  12. Order Set

  13. Patient Care

  14. Oral Health

  15. Pharmacy

  16. Radiology

  17. Trmt/Procedures

  1. Enabled - Select the check box to enable the record in the transaction functions. The check box is selected, by default.

Auto Registration

  1. Required - Select the check box and the orders placed for the selected Order Category is registered automatically. The orders are registered automatically on placing the orders. Clear the check box and the orders placed have to be registered automatically using the function Register Order.

Default Timing

  1. Frequency Applicable - Select the check box to indicate that frequency is applicable for the defined Order Type.

  2. If Frequency is applicable, in the 'Order Catalog' function, under the Timing Tab, the 'Frequency Required' check box is selected, by default. If frequency is not applicable, the 'Frequency Required' check box is displayed as cleared and disabled.

  3. If Frequency is not applicable, the frequency selection is not mandatory for the Order Catalog associated to the selected Order Type in transaction function.

  1. Soft Stop for Inpatients – Select the check box, if the duration or End Date is not mandatory while placing an order. When the encounter is closed for the patient, the order is terminated, or you have to 'Stop' (Discontinue) the order explicitly. This rule is applicable for inpatient orders only.
