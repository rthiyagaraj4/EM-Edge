Enter the following details:

 

  1. By – Select a value from the list. The values are system defined – Consent Form and Practitioner.

  2. Consent Form - Enter the required Consent Form or select the Consent Form using [?] function. This is a mandatory field.

Two tabs 'Select' and 'Associate' are provided for selecting and associating the Practitioners to the Consent Forms and vice versa . The 'Select' tab is selected, by default.

 

Select Tab

  1. Alphabets/Others - The alphabets 'A to Z' and 'Others' are displayed and hyperlinked. Practitioners' names beginning with the selected alphabet are displayed. Click 'Others' hyperlink to select Practitioners, whose name begin with numeric or special characters.

  2. Practitioner - A list of the practitioners whose names begin with the selected alphabet gets listed. The practitioners are listed only if the value for ‘By’ field is selected as ‘Practitioner’. This is a mandatory field.

  3. Consent Form - The Consent Form is displayed automatically on selecting the value for ‘By’ as ‘Consent Form’. This is a mandatory field.

  4. Select - Select the check box to associate the practitioner to the consent form or vice versa.

 

Associate Tab

Click 'Associate' tab to view the associated records. You are able to dissociate the records by clearing the check boxes against the records, which are associated. Click [Apply] to save the changes.
