Record Refusal

From this function you can Record refusal details.

A registered patient at the logged in facility may refuse to undergo an advised procedure placed as an order or refuse service advised. This function is used to record refusal details.  

Recording Refusal

From the CA Transactions > Clinician Access > Desktop Menu > Order Management, select Record Refusal function.

The Record Refusal screen gets displayed.

  1. Select the Patient ID using [?] function. The Patient Name, Age and Sex is displayed.

The screen displays two tabs:

Existing Refusal

  1. The Existing Refusals for the patient are displayed for the default date range. Enter the query criteria and click [Search] to display specific existing refusals. Click [Clear] to clear the displayed data.

    1. Refusal Type – Select the Refusal Type from the list. The listed values are All, Orderable and Non-Orderable.

    2. Catalog – Select the Order Catalog using [?] function.

    3. Period From-To – Select the period range to search for refusals recorded.

    4. Order By – This field is displayed as Date/Time.

    1. Refusal Date/Time – The Date and Time of Refusal is displayed.

    2. Orderable/Non Orderable – The Orderable/Non Orderable are displayed.

    3. Reason – Click Reason hyperlink to view the reason for refusal recorded.

    4. Practitioner – The Practitioner recording Refusal is displayed.

    5. Refusal Forms – The Refusal Form is displayed on clicking View Details hyperlink. Click [Print] to print the refusal form or [Close] to exit from the Preview screen.

  2. Click Reason hyperlink to enter reason for refusal.

  3. Click View Details to view details of refusal form.

New Refusal

  1. Click New Refusal tab to record new refusal for the patient.

  2. Enter the details in the mandatory (*) fields. < Related Field Description >

  3. Click [Record] to record refusal form or [Clear] to clear the selected data.
