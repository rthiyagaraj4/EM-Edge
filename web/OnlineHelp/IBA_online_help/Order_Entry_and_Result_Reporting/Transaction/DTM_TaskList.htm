Task List

From this function you can Perform tasks that are pending for the Practitioner.

Pending Activities is a set of functions that allow practitioners to view tasks that are pending. Tasks such as reviews, verifications and tasks, reviewing results and signing notes are grouped under three sub menus, Verification, Reviews and Tasks.

Task List function enables you to view the tasks, which are pending to complete the care planned for the patient. This also updates the user with the status of the individual task performed or not performed.

The system defaults the nursing location in ‘Quick Link’ function based on the user login; and automatically display the Task List with all tasks in ‘Due’ and ‘Over Due’ status when Task list is defined in ‘Define Quick Link’ setup and the Task list is mapped for a particular responsibility in Quick Link for Responsibility.

Performing Tasks

  1. Select the Date for which the Task List is to be displayed. The system date is displayed, by default.

  2. Select the Location Type as Clinic or Nursing Unit. The Location Type is displayed as Nursing Unit, by default.

  3. Select the required Location. The Locations are listed based on the selected Location Type.

  4. Select the required radio button against All Patients or Search Patient. All Patients is selected by default. Select Search Patient to search for a required patient. Select the required patient using the [?] function.

  5. Select Order Category and Order Type. The Order Types are listed based on the selected Order Category. Both Order Category List and Order Type List display ‘All’, by default. Click [Go].

  6. Select ‘Group By’ from the list. The listed values are ‘Patient Name’ and ‘Time Interval’. This field allows you to select criteria based on which the task list is to be generated. ‘Patient Name’ is displayed, by default.

  7. Click 'Filter' hyperlink to displays the Filter criteria window. Select the Discharge Patient check box to include discharge patients in the Task List. You can select the Date range from the calendar function.

  8. Click 'Order Catalog' hyperlink to view different tasks such as Note Done, Register Order Status, Review Line Status, Register Order, Result Entry, Order Consumption, Line View in Existing Order and Manage Specimen. All the pharmacy orders are displayed with dosage and frequency details.
