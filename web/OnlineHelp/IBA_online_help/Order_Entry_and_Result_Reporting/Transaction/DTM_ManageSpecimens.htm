Manage Specimens

From this function you can Manage Specimen for laboratory investigations.

This function provides various options for managing specimen labels required to handle laboratory investigations. This function is used to generate new labels, print labels, reprint already printed labels, stamp date and time of collection, and update status of collected specimens as dispatched to the lab. The function also provides an option to print a collection list in an inpatient location for use by nurses or phlebotomists to perform specimen collection.

Managing Specimens

From the CA Transactions > Clinician Access > Desktop Menu > Pending Activities > Tasks, select the Manage Specimens function.

The Manage Specimens screen gets displayed.

  1. Click [Search] to display orders for the default query criteria or enter the query criteria and click [Search] to display specific orders for managing.

    1. Disease Code (HAAD) - The infectious disease color codes get displayed for the patient diagnosed with the particular infectious diseases in that encounter. When you mouse over the Disease Code colour, the name of the disease gets displayed as a tool tip.  This column appears only for users complying with HAAD regulations and color codes appear only if patient is diagnosed with infectious disease. The colour coding is based on the patient diagnosis and colour code configuration done in the MR Code Setup.

    2. Patient Order Detail – Patient order details are grouped under location type and location. The details displayed are Patient Name, Gender, Age, ID, Encounter ID, Location Type and Location Name.  

    3. Order ID – The Order ID generated on placing order is displayed. The Order ID provides information on the Order Category, Patient Class and system generated number.

    4. Order Catalog – The Order Catalog is displayed and hyperlinked. If specimen number is generated for the order it is displayed and in New List Option, specimen number is not displayed since it is not yet generated. Specimen number generation can either be automated upon confirming the order or can be generated manually after placing the order.

    1. Start Date and Time –The Start Date and Time entered while placing the order is displayed.

    2. Select – Select the check box, to select the order catalog. The purpose of selection varies based on the option chosen. The selection can be for generation of specimen label and printing if the option is either ‘New List’ or ‘Reprint’. This column also represents the status of the specimen collection, which could be either ‘Future’, ‘Due’, ‘Overdue’ or ‘Unbilled' The status is computed based on the time frame defined for specimen label generation. The following colors are used to represent the status.

    Future

    Due

    Overdue

    Unbilled

     

    When generating a new list, if you select the check box for a future order, the following message will be displayed:

    “This action will update the start date of the selected future order, to current date. Do you want to proceed?”

    Click OK if you wish to collect the specimens of future orders.

  2. Click Order Catalog hyperlink to view Order Details.

    1. The screen displays Order ID, Order Category, Order Type, Priority, and Patient details, Encounter ID, Order Date Time, Order Status, Ordering Location, Ordering Practitioner, Ordering Facility, Performing Facility, Order Format and Clinical Comments. The screen also displays the Order Catalog, Priority, Frequency, Start Date/Time, End Date/Time, Quantity, Status, Formats and Comments.

    2. Click Format hyperlink to view details in the Order Line Format screen. Click [Close] to close the Order Line Format screen.

    3. Click [Schd] hyperlink to view the Schedule Frequency of the order and [Close] to close the Schedule Frequency screen.

    4. Click Preps, Dept Instrn, Pat Instrn hyperlink to view details if recorded while creating the Order Catalog.

    5. Click [View Audit Trail] to display View Audit Trail screen and view audit trail details. Click the Remark (R) hyperlink to display remarks for the order. Click [Close] to close the remark screen. Click [x] to exit from the View Audit Trail screen.

    6. Click [Close] to close View Order screen.

  3. Select the check box against the required order details.

  4. Click [Redirect Order] to redirect specimen to another performing location.

    • The Change Performing Location dialog box is displayed.

    • Select the Performing Facility and Performing Location from the drop-down list. The Performing Facility is a mandatory field, if left blank an error message appears.

    • Click [OK].

  5. Click [Record] to save the task based on the Options selected or click [Clear] to clear the selected orders.

    New List

    If the Option is New List, select the Select check box against the required orders and click [Record].

    Record Collection

    If Option is Record Collection, select the Select check box against the required orders to display the Collection Details screen. You can select all the records of a particular patient by clicking ‘Select All’ check box which is enabled only if ‘Select all option for Record Collection’ check box is selected in OR Parameter for Facility screen. If there are records in Next page, you need to navigate to that page and select the records.

    1. Collection Date and Time – Enter the date and time of sample collection. System displays current system date and time and allows backdating, by default. This field is mandatory.

    2. Collected By – Enter the name of the person who has collected the specimen. By default, the logged in user name is displayed. This field is mandatory.

    3. Remarks – Enter any remarks on collection. This field is optional.

    4. Click [OK] to confirm collection details

    5. A 'Dtls' hyperlink gets displayed on recording collection details and is clicked to view details of collection recorded.

    6. Click [Record] to confirm collection of specimen.

    Record Dispatch

    Specimen dispatch to the laboratory can be automated or can be manually performed based on the collection stage defined against each laboratory order. If collection stage for an order is with order, then the system automatically updates dispatch status of the order upon placing the order. If collection stage is separate, it indicates that the specimen requires manual dispatch, in which case this function can be used to list all such specimens, which have been stamped as collected and record dispatch. If collection stage is not applicable, it indicates that the specimen dispatch is not required for the order and such specimens are not listed for dispatch.

    If the Option is Record Dispatch, select the Select check box against the required orders to display the Dispatch Details screen.

    1. Dispatch Date Time – Enter the date and time of dispatch. The system defaults current date and time, which can be backdated. This field is mandatory.

    2. Dispatch Mode – Select the dispatch mode from list box. The values are populated based on transport mode defined for specimens in AM Module. This field is mandatory.

    3. Dispatched By – Enter the name of the person performing the dispatch. The system defaults logged in user name. This field is mandatory.

    4. Remarks – Enter remarks on dispatch if any. This field is optional.

    5. Click [OK] to confirm dispatch details.

    6. A 'Dtls' hyperlink gets displayed on recording dispatch details and is clicked to view details of dispatch recorded.

    7. Click [Record] to confirm dispatch details.

    Reprint Specimen

    On Selecting Option ‘Reprint Specimen’, the orderable can be printed in the default location. The option allows reprinting specimens against orders for which specimen numbers are already generated and printed.   

    1. Select the Select check box against the required orders to Reprint Specimens.

    2. Click [Reprint] to reprint the Specimen.

    Reprint Dispatch Slip

    On selecting Option as ‘Reprint Dispatch Slip’, the orders having Dispatch Slip generated are displayed on searching. You can Reprint the Dispatch Slip using this function. The Print Location has to be specified.

    1. Select the Select check box against the required orders to Reprint Dispatch Slip.

    2. Click [Reprint] to reprint the Dispatch Slip.

The Performing Facility can be updated in the Change Performing Location dialog box only before generating a New List.
