Interface Vital Signs Data

This sub-menu is applicable only when the ‘Third party vitals data required to acknowledge’ check box is selected in CA Parameter. This sub-menu allows you to acknowledge/delete the vitals if the ‘Third party vitals data required to acknowledge’ checkbox is selected.

  1. Enter the Encounter ID, Nursing Unit, Location and Patient ID fields. They are mandatory.

  2. Click [Go]. The interfaced vital sign data appear based on input criteria.

  3. In the search results, the columns include Chart Component Description, Recorded Date/Time, Value and Select All check box.

  4. Select the check box against any individual vital parameter record or select all the vital parameter records by clicking 'Select All' check box at the header.

  5. Once selected, click [Acknowledge] to acknowledge the data or click [Delete] button to delete the records.

  6. If only few vital parameters are confirmed, that record still appears in the screen. Once all the vital parameters are confirmed, then the record gets removed from the list.
