Enter the following to create associated patients to a Customer:

 

  1. Customer – Click [List of Values] to select a valid Customer. This is a  mandatory field.

  2. Credit Doc Reference – Enter the Credit Document Reference in alphanumeric format. This is a  mandatory field.

  3. Credit Doc Ref Start Date – Click the Calendar function to set the date. This is a  mandatory field.

  4. Credit Doc Ref End Date – Click the Calendar function to set the date. This is a  mandatory field.

  5. [Edit List] - Click [Edit List] to edit the Customer Details.

  6. [Finalize] - Click [Finalize] to authorize the entries.

 

Patient Details

  1. Patient No – Click [List of Values] to select a valid Patient Number. This is a  mandatory field.

  2. Patient Name – The Patient Name is displayed automatically when the Patient Number is set.

  3. National ID – Click [List of Values] to set a National ID. This is a  mandatory field.

  4. Current Billing Group – The Current Billing Group is displayed.

  5. New Billing Group – Click [List of Values] to set a New Billing Group. This is a  mandatory field.

  6. Description – The Description for the selected Billing Group is displayed.

  7. [Fin Details] - Click [Fin Details] to view the Financial Details of the relevant patient.
