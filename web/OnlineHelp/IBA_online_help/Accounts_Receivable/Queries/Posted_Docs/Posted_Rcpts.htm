Posted Receipts

From this function you can Query the receipts which are posted.

Posted Receipts those pertaining to the period for which the month end process is not done only are displayed. When the month end process is done, Posted Receipts are deleted. In addition to the document details, related bills details, User accessed list, Text Document details are also displayed.

Querying Posted Receipts

From the Queries > Posted Documents, double-click Posted Receipts function.

The Posted Receipts screen gets displayed.

  1. Click [Execute Query] or press [F8] to view the receipts posted to the Customer from the Accounts Receivable, Patient Billing Manager and the Laboratory module.

OR

  1. Click Calendar function to select the date range. Click [Execute Query] or press [F8] to view the receipts posted to the Customer from the Accounts Receivable, Patient Billing Manager and the Laboratory module during the specified date range.

  2. Click [Refresh] to view all the Posted Receipts Details associated with the facility for the specified Date range. When the Date range is not specified, clicking on [Refresh] displays all the Posted Receipts details from the database.

  3. After entering date field values, click [Execute Query] or press [F8]  by placing the cursor on document type, and related records are populated for view. In case if all the records are to be displayed, just click [Execute Query] or press [F8]  by placing the cursor on document type, system displays all the Posted Receipts record.

  4. Click [Receipt Details] to view the information regarding the Posted Receipts, Patient Bills, User access and Remarks pertaining to a relevant Document Type.

  5. Click [Close] to exit the Receipts Details Screen and return to the Posted Receipts main screen.
