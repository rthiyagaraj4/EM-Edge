Enter Receipts Allocation

From this function you can Allocate receipts for customer transactions in the facility. You can Query to view the existing records.

This function is used to allocate Receipts for Customers. Allocations can be done to an already existing receipt document, which is not posted. New receipt documents can be entered only through "Enter Official Receipts" option and allocations can be done here only when the official receipt is posted. After posting, the receipt document can be recalled through this function to allocate the receipt amount against several Invoices for a relevant customer. The receipt document date must be greater than or equal to the allocated document date.

Enter Receipt Allocation

From the Transactions > Receipts, double-click Enter Receipts Allocation function.

The Enter Receipts Allocation screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Name/Addr]  to record the miscellaneous Customer details. Click [Close] to exit from the screen.

  3. Enter the details in the mandatory (*) fields under the Receipt Lines. <Related Field Description>

  4. Click [Commit Form] or press [F10] to save a generated receipts or [Clear Record] to clear the entered or queried data.

  5. Click [Execute Query] or press [F8] to search for an existing records.
