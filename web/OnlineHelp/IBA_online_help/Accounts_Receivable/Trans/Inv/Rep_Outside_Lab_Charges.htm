Report Outside Lab Charges

From this function you can Generate external lab service reports and consolidate Lab Invoices for the selected Customers.

If a patient visits a Lab outside the hospital based on the instruction given by the hospital, the hospital pays to the lab on behalf of the patient in the Patient Billing Manager module and the customer is Invoiced towards such payments paid on behalf of the patient in the Accounts Receivable module. Separate Invoices can be raised for each order in the lab or for each patient. System carries several validations for the  generation of this report.

• Lab order for which customer has to be charged

• Orders which are yet to be invoiced

• Orders that are not cancelled

Generating External Lab Service Reports

From the Transactions > Invoices > Lab Invoices, double-click Report Outside Lab Charges function.

The Report Outside Lab Charges screen gets displayed.

  1. Enter the required date in the format of mm/dd/yyyy or click the Calendar function to select the required date range. Both the values are mandatory. The ‘To’ value has to be greater than or equal to the ‘From’ value.

  2. Enter the Customer Code in alphanumeric format of eight characters maximum length or select the required Customer Code from [List of Values]. Both the values are optional. If ‘From’ value is not entered, the system assumes the lowest value for the ‘From’ and if ‘To’ value is not entered, the system assumes the highest value for the ‘To’ value. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  3. Select the ‘Report By’ option. The options are ‘Summary’ and ‘Detail’. The option ‘Summary’ is selected, by default. Select ‘Detail’ to view for the detailed report.  

  4. Click [Run Process] to view, print or save the adjustment details in a file in the local machine.
