Print Official Receipts – Edit List

From this function you can Edit Official Receipts before posting and printing.

This function is used to check for all the field values entered in Official Receipts and provide information on Errors, if any, encountered. The information is provided in the form of report with details.

Editing Official Receipts

From the Transactions > Cashier Menu, double-click Print Official Receipts – Edit List function.

The Print Official Receipts – Edit List screen gets displayed.

  1. Enter the range for Document Type or click [List of Values] to select the required Document Type. Both the values are optional. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  2. Enter the range for Document Number or click [List of Values] to select the required Document Number. Both the values are optional. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  3. Enter the range for Document Date or select the required date using the Calendar function. Both the values are optional. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  4. Click [Run Process] to view, print or save the report in a file in the local machine.

  5. A log file gets generated. Click [OK] to view the log file or [Cancel] to exit from the screen.
