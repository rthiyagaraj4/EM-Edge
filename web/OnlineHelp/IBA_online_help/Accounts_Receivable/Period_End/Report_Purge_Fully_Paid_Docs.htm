Report/Purge Fully Paid documents

From this function you can Delete the documents with the outstanding amounts fully settled.

This function is used to delete the fully paid documents for the selected set of customers based on the input parameters entered. The printouts are the only records in future for any reference of deleted documents. Hence, print the documents before deleting them. Documents with outstanding balance as zero are considered for deletion.

Generate a report or purge the fully paid documents

From the Period End, double-click Report/Purge Fully Paid Documents function.

The Report/Purge Fully Paid Documents screen gets displayed.

  1. Click [List of Values] to select a range of Customer Code. Both the values are optional. If ‘From’ value is not entered, the system assumes the lowest value for the ‘From’ and if ‘To’ value is not entered, the system assumes the highest value for the ‘To’ value. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  2. Click [List of Values] to select a Customer Group.  Both the values are optional. If ‘From’ value is not entered, the system assumes the lowest value for the ‘From’ and if ‘To’ value is not entered, the system assumes the highest value for the ‘To’ value. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  3. Select the check box to generate and view the report. If the check box is cleared, fully settled records are deleted from the database. You cannot view or access the deleted records for verification thereafter.

  4. Click [Run Process] to either view, print or save the output in a file in the local machine.

  5. A log file gets generated. Click [OK] to view the log file or [Cancel] to exit from the screen.
