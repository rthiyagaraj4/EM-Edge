Export Transactions to GL

From this function you can Export the transaction records from the Accounts Receivable module to the GL interface.

The Export Transactions to GL function is used to export all transaction records from the Accounts Receivable module to the General Ledger module. This function can be used only if the Accounts Receivable to General Ledger interface in AR parameters is set to "Batch". The General Ledger distribution has to be recorded prior to the export. A check is done to filter the records, which are already exported to General Ledger.

Exporting Transactions to GL

From the Period End, double-click the Export Transactions to GL function.

The Export Transactions to GL screen gets displayed.

  1. Enter the Month and Year, upto when the transactions are to be exported to General Ledger. Current processing month and year is defaulted.

  2. The options are “Detail” and “Summary”. Select an option to generate either a detailed or a summarized report. If the option is summary, a summary line is interfaced to GL. The details have to be looked into the GL distribution report only. Summarization is done separately for each Transaction Type. For example, all invoice transactions affecting a posting account within a month and year is summarized into one entry and exported. If the option is Detail, each individual transaction header and detail lines are interfaced. However, the detail interface is done for all the accounts where the bank reconciliation is required.

  3. The Document Types screen gets displayed on selecting the option. Click [List of Values] to select a Document Type for each Transaction Type. The details get displayed for the selected Document Type. Click [Clear Record] to exit from the screen without selecting the Document Types. Click [Close] to exit from the screen on selecting the Document Types.

  4. Click [Run Process] to either view, print or save the output in a file in the local machine.

  5. Click [Accept] to process or [Cancel] to re-enter the selection criteria for the report.
