Enter the following details:

 

  1. Surgeon - Enter the required surgeon's name or select from the list of values by clicking [?] function.

  2. Applicability - Select the applicability of the Surgeon Preference from the list. The listed values are All Operations, Specific Specialty, Specific Operation Category (within the Specialty), Specific Operation Sub Category (within the Specialty) and Specific Operation. This is a mandatory field.

  3. Specialty - Select the Specialty using the [?] function. The listed values are defined in the 'Specialties' function. This field is enabled if the Applicability is selected as 'Specific Specialty', 'Specific Operation Category' or 'Specific Operation Sub-Category'.

  4. Category - Select the Category using the [?] function. This field is enabled if the Applicability is selected as 'Specific Operation Category (within the specialty)' and 'Specific Operation Sub-Category (within the specialty)'. The listed values associated to the selected specialty in the Operation Category function.

  5. Sub Category - Select the Sub Category using the [?] function. This field is enabled if the Applicability is selected as 'Specific Operation Sub-Category (within the Category)'. The listed values associated to the selected specialty and operation category in the Operation Sub Category function.

  6. Operation - Select the operation using the [?] function. This field is enabled if the Applicability is selected as 'Specific Operation'. The listed values are defined in the Operations/Procedures function.

 

Consumables Tab

  1. Description - Select the required consumables using the [?] function. The listed values are defined in the 'Consumable Packs' function. This is a mandatory field.

  2. No. Of Units - Enter the number of units in alphanumerical format of three characters maximum length.

 

Equipments Tab

  1. Description - Select the required equipments using the [?] function. The listed values are defined in the 'Equipments' function. This is a mandatory field.

  2. No. Of Units - Enter the number of units in alphanumerical format of three characters maximum length.

 

Instruments/CSSD Packs Tab

  1. Description - Select the required instruments/cssd packs using the [?] function. The listed values are defined in the 'Instruments/CSSD Packs' function. This is a mandatory field.

  2. No. Of Units - Enter the number of units in alphanumerical format of three characters maximum length.

 

Prosthesis Tab

  1. Description - Select the required prosthesis using the [?] function. The listed values are defined in the 'Prosthesis/Implants' function. This is a mandatory field.

  2. No. Of Units - Enter the number of units in alphanumerical format of three characters maximum length.

 

Personnel Required Tab

  1. Description - Select the required personnel using the [?] function. The listed values are defined in the 'Roles' function in the Admin setup of this module. This is a mandatory field.

  2. No. Of Persons - Enter the number of persons required for the operation in alphanumerical format of three characters maximum length.

The Consumables, Equipments, Instruments/CSSD Packs, Prosthesis and Personnel assigned to one surgeon cannot be assigned to another surgeon.
