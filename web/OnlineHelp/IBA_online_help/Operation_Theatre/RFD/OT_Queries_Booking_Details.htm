Enter the following details:

  1. Booking Date [From-To] – Enter the Booking Date range in the format of dd/mm/yyyy or select the Booking Date range using Calendar function. This is a mandatory field.

  2. Theatre Type – Select the Theatre Type using [?] function.

  3. Theatre - Select a Theatre using [?] function. The theatres defined in the Operating Room function are displayed in the list. This is a mandatory field.

  4. Surgeon – Select the Surgeon using [?] function. The practitioners with practitioner type ‘Surgeon’, as defined in the Application Masters module are displayed in the list.

  5. Anesthetist - Select the Anesthetist using [?] function. The practitioners with practitioner type ‘Anesthetist’, as defined in the Application Masters module are displayed in the list.

  6. Specialty - Select the Specialty using [?] function. The specialties defined in the Operations master are displayed in the list.

  7. Operation - Select the Operation using [?] function. The Operations/Procedures defined in the Operations function are displayed in the list.

  8. Patient ID – Select the Patient identification number using [?] function.

  9. Patient Name – The Patient Name corresponding to the Patient ID gets populated in this field based on the selected Patient ID.

  10. Gender – Select the Gender from the list. The listed values are Male, Female and Unknown.

  11. Birth Date – Enter the Date of Birth of the patient in dd/mm/yyyy format.

  12. Stages – This text box gets enabled if the selected Status is ‘Booked Cases’. Select the Stages from the list of values.

  13. Order By - Select the ‘Order By’ value from the list. This is a mandatory field.

  14. Source Type – Select the Source Type from the list. The options available are ‘Nursing Unit’, Clinic’ and ‘Referral’. On selecting the Source Type, the Sources for the selected Source Type are available for selection in the adjacent field. The  icon appears beside the Source field if the long name for the External Referral Source is recorded, in ‘External/Referral Source’ function of Application Masters module. Place the mouse on the  icon to display the long name of the External or Referral Source recorded in ‘External/Referral Source’ function of Application Masters module.

  15. Pre-operative clinical comments – Enter the Pre-operative clinical comments as entered in Booking Verification function.

  16. Status - Select the Status from the list. The system defined values are: All Cases, Booked, Waitlist, Cancelled and Reschedule Cases. This is a mandatory field.
