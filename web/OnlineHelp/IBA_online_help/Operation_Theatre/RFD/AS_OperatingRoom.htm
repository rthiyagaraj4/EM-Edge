Enter the following details:

 

  1. Theatre Type – Select the Theatre Type using the [?] function. The listed values are predefined. This is a mandatory field.

  2. Operating Room – Enter a code for the Operating Room in alphanumeric format of six characters maximum length. This is a mandatory field.

  3. Long Description – Enter the Long Description for the Operation Room in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  4. Short Description – Enter the Short Description for the Operation Room in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  5. Theatre Sequence No. -  Enter the Theatre Sequence number. The 3 digit Sequence Number is used to denote the order in which the Operating Room is displayed on all OT Bookings and the related screens. This is a mandatory field.

  6. Restrict by Specialty Y/N – Select the check box to indicate that the Operations in the Theatre are restricted by Specialty. The check box is selected, by default. If this check box is selected, the operating room is available in the Booking function (code setup), where the operation rooms are blocked by the specialty.

  7. Over Booking Allowed Y/N – Select the check box to indicate that over booking is allowed for the Operating Theatre.

  8. Count – Enter the Count in numeric format to define the maximum number of over bookings that can be made in the OT room. Maximum value that can be entered is ‘999’. This field gets enabled only if the ‘Over Booking Allowed Y/N’ check box is selected.

  9. Service - Select the Service using the [?] function to specify the Service for the Operating Room. This is a mandatory field.

  10. Working Day Start Time - Enter the start time of working days in the Operating Room in the format hh:mm. This is a mandatory field.

  11. End Time - Enter the end time of working days in the Operating Room in hh:mm format . This is a mandatory field.

  12. User to override booking restriction – Click the ‘Select User’ hyperlink to invoke Authorize user dialog.

    1. Name - Select the Name using the [?] function.

    2. Select - Click [Select] to select the user as authorized user with booking rights. The user name gets displayed in the grid below.  Multiple users can be added as authorized users with booking rights.

    3. Save – Click [Save] to save the authorized users with booking rights.

  13. Holiday Start Time - Enter the start time of Holidays in the Operating Room in hh:mm format . This is a mandatory field.

  14. End Time - Enter the end time of Holidays in the Operating Room in hh:mm format . This is a mandatory field.

  15. Chargeable Room? - Select the check box if the usage of the Operating Room is chargeable. This check box is enabled if 'Bill Operating Room Usage?' check box is selected in 'Setup Parameter for Facility' function.

  16. Default Recovery Room - Enter the Default Recovery Room in alphanumeric format of  20 characters maximum length or select the Recovery Room using the [?] function. The listed values are defined in the 'Recovery Room' function.
