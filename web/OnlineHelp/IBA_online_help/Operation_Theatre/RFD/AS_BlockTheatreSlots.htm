Enter the following details:

 

  1. Theatre - Select the Theatre from the list. The listed values are defined in the 'Operating Room' function. This is a mandatory field.

  2. Reason - Enter the Reason for blocking the Theatre. This is a mandatory field.

  3. Day Specific - Select the check box to block for a specific day.

  4. Block Date - Enter the blocking period using the calendar function.  This is a mandatory field.

  5. Slot Time - Enter the Time Range to block the Theatre.  This is a mandatory field.

 

Weeks

If the blocking has to be done for all the weeks in the specified Date range, then, select the 'All Weeks' check box, or select the check box suitable to specific weeks.  The check boxes are enabled if the Day Specific check box is selected.

  1. Day - Select the required Day from the list.
