Enter the following details:

 

  1. Operating Room - Select the Operating Room from the list. The listed Operating Rooms are defined in the 'Operating Rooms' function. This is a mandatory field.

 

Diary Schedule for Surgeon

  1. Surgeon - Select the required surgeon using [?] function to define the schedule. This is a mandatory field.  

  2. Specialty - Enter the Specialty or select the specialty using the [?] function. The listed values are defined in the 'Specialties' function in Application Masters module. This is a mandatory field.

  3. Start Time - Enter the Start Time in hh:mm format for the surgeon. This is a mandatory field.

  4. End Time - Enter the End Time in hh:mm format for the surgeon. This is a mandatory field.

The Start and End time entered for the Operating Room should be between the time set for working days in the Operating Room function.

  1. Day - Select All Days or a specific day to schedule the surgeon.

  2. Schedule Type - Select the 'Schedule Type' from the list. The options available are, 'All Weeks' and 'Specific Weeks'. If 'Specific Weeks' is selected, you can select the weeks to block slots. If 'All Weeks' is selected, all the 'Weeks' check boxes are selected and disabled.

  3. Weeks - The check boxes are enabled if Schedule Type is selected as 'Specific Weeks'. Select the required check boxes to indicate that the slots are scheduled for the surgeon for the specific weeks selected.

  4. Total slots available – The total number of slots allotted for surgery is displayed.

  5. Booking allowed for X Slots – Enter the number of slots allowed for surgery without authorized person permission. The number of slots must be less than or equal to the Total slots available. The total number of slots allotted for surgery is displayed by default. You can modify the value.

If the ‘Booking allowed for ‘is greater than the ‘Total slots available’, an  alert message is prompted.
