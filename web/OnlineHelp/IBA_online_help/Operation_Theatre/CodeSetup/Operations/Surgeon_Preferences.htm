Surgeon Preferences

From this function you can Define Surgeon Preferences and Query for existing Surgeon Preference records.

The Surgeon Preferences function allows you to define the Consumables, Equipments, Instruments/CSSD Packs, Prosthesis and Personnel required for All Operations, Specific Specialty, Specific Operation Category, Specific Operation Sub Category and Specific Operation for the selected Surgeon. The surgeon preferences defined reflects in the OT Slate function (Verify checklist, Swab and Instrument count) and while generating the pick list for an operation.

Defining Surgeon Preferences

From OT Code Setup > Operations, select the Surgeon Preferences function.

The Surgeon Preferences screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the data in the mandatory (*) fields. <Related Field Description>

  3. Click [Select] to display the details in the right hand side of the screen. Select the Description hyperlink to edit the units or cancel the preferences.

  4. Click [Apply] to save a created Surgeon Preference or [Reset] to clear the entered data.

  5. Click [Query] to search for existing Surgeon Preference records.
