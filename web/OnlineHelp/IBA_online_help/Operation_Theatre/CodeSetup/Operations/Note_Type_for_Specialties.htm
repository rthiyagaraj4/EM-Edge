Note Type for Specialties

From this function you can Associate Note Types to a particular Specialty.

The 'Note Type for Specialties' function enables you to associate Note Types created in the Clinician Access module to a particular Specialty. The associated Note Types are used for OT Booking/Operations. Each Note Type is associated to a Location Type which caters to Holding Area Notes, Surgeon Notes or Recovery Notes.  You are prompted to enter Notes at various stages of an operation.

Associating Note Types to a Specialty

From OT Code Setup > Operations, select the Note Type for Specialties function.

The Note Type for Specialties screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the required Specialty from the list. The Specialties created in the Specialties function is displayed. This is a mandatory field.

  3. Click [Search] to display the Note Types or [Clear] to reset the selected data.

  4. Select the Location Type for the Note Type from the list. The listed values are Holding Area Notes, Surgeon Notes and Recovery Notes.

  5. Select the check box against the required Note Type to associate to the Specialty.

  6. Click [Apply] to save a Note Types associated to the Specialty or [Reset] to reset the screen.
