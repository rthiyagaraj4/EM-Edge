Drug for Notification

From this function, you can Define the drug required for notification and Query for the existing records.

Drug for Notification function enables you to select the drugs that have to be notified at Pre-Anesthesia Stage. The drugs in this function are displayed from the 'Drug Masters' setup in the Pharmacy Management Module.

Drug Notification of Pre-Anaesthesia

From OT Code Setup > Others, select the Drug for Notification function.

The Drug for Notification screen gets displayed.

  1. Click [Create] to invoke the create screen

  2. Select the Definition as either 'Drug' or 'Generic' from the list.

  3. Click [Search] to display the Drugs at the bottom of the screen.

  4. Select the check box under 'Notify at Pre-Anesthesia' against the required drug which needs to be notified at Pre-Anesthesia stage.

  5. Click [Apply] to Notify Pre-Anesthesia stage for the selected drug or [Reset] to clear the entered data.

  6. Click [Query] to search for existing records of the Drugs, which are to be notified at Pre-Anesthesia stage.
