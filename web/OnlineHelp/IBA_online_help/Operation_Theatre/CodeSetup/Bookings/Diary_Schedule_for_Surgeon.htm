Diary Schedule for Surgeon

From this function you can Set the schedule for a surgeon and Query for the existing records.

This function allows setting the schedule for a surgeon. The schedule is generated based on the Specialty, the days of the week and the start and end time of the required day. This schedule can be made common for all the weeks of a month or for a particular week. In the 'Booking' function, if you enter the criteria for booking as surgeon, the schedule defined is displayed.

Setting Diary Schedule for Surgeon

From OT Code Setup > Bookings, select the Diary Schedule for Surgeon function.

The Diary Schedule for Surgeon screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Select] to display the recorded details at the bottom of the screen.

  4. Click Surgeon hyperlink to edit the End Time, Schedule Type and Weeks. Click [Cancel] to clear the entered details.

  5. Click [Apply] to save the record or [Reset] to reset the entered details.

  6. Click [Query] to search for existing records.

 

This function is displayed if the 'Create Diary Schedule By' field is selected as Surgeon in the 'Setup Parameters for Facility' function.
