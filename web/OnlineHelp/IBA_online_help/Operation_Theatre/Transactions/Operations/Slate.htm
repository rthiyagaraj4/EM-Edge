Slate

  1. Select User Role and click [Search] to display scheduled cases for the default current date or enter the search criteria to search for a specific scheduled case.

    1. Theatre Period [From - To] - The date as specified in the Setup Parameter for Facility function is displayed, by default. Select the date using the Calendar function. This is a mandatory field.

    2. User Role - Select the User Role from the list. The roles defined in the OT Admin setup is populated in the list. This is a mandatory field. The default value is based on the role type assigned to the practitioner for the logged in user . If there is no role type assigned to the practitioner for the logged in user, then the value ‘Select’ is displayed in the drop-down list.

    3. Theatre - Select a Theatre from the list. The Theatres are displayed based on the selected Theatre Type. The theatres are defined in the Operating Room function.

    4. Surgeon - Select the Surgeon using [?] function. The logged in user name is displayed by default in the Surgeon field only if the logged in user is configured for the Role Type 'Operating Surgeon'

    5. Status - Select the required Status from the list.

    6. View By - Select the option for viewing from the list. The listed values are 'Scheduled Cases', 'Unscheduled cases' and 'All'. 'All' is displayed, by default.

    Click icon to view the additional criteria or to hide the additional criteria

    1. Elective/Emergency -  Select the values 'All', 'Elective' and 'Emergency' from the list. If 'Elective/Emergency' is selected as Emergency, OT Slate displays the emergency cases for the selected date range. If 'Elective/Emergency' is selected as Elective, OT Slate displays the elective cases for the selected date range. If 'Elective/Emergency' is selected as All, OT Slate displays all cases for the selected date range.

    2. Specialty - Select a Specialty from the list. The specialties defined in the Operations function is listed.

    3. Theatre Type - Select the Theatre Type from the list. 'All Theatres' is displayed, by default.

    4. Scan Patient - Select the required value from the list. The listed values are 'Check-In' and 'Check-In to OR'.

    5. Click [Clear] to reset all the values.

  2. The details of the Scheduled Cases - Date/Time Slot, Patient ID, Patient Name, Theatre, Procedure, ASA Score, Specialty, Surgeon Status and Location are displayed. An icon is displayed when the items in Pick list screen is not finalized or requested or partially finalized. This icon will appear only if Operation Theatre and Sterile Stock modules are interfaced.

  3. Place the cursor over icon to view the Order Result Status/Additional Information of Patient.

  4. Place  the cursor over icon to view the high risk cases. The reason for risk is also displayed.

  5. Click Patient Name hyperlink to view the menu to perform various activities. The Menu is displayed based on the Status, Role and User. The setups are done in OT Admin Setup in functions – Functions for Status, Functions for Role and Functions for User. The listed functions are given below.

  6. Select the required function and a screen to enter Login details are displayed if ‘Slate Login Required?’ check box is selected in the ‘Setup Parameters for Facility’ function. Enter the Login Username and Password. Click [OK] to login and [Cancel] to exit from the screen.
