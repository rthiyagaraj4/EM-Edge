Check Out from Ward  

From this function you can Check-Out a patient from the ward.   

Checking Out From Ward

From Operations, select the Check Out From Ward function.

The Check Out From Ward screen gets displayed.

  1. Select the Nursing Unit and click [Search] to display Patient records for the default Surgery Date or enter the search criteria to search for specific patient records.

    1. Surgery Date - Select a Surgery Date using the Calendar function. The current date is displayed, by default. This is a mandatory field.

    2. Nursing Unit - Select a Nursing Unit using [?] function. This is a mandatory field.

    3. Patient - Select the Patient ID using [?] function. The name of the patient gets displayed in the corresponding field.

    4. View By - Select a View option from the list. The listed values are 'Scheduled cases', 'Unscheduled Cases' and 'All'.

    5. Type - Select a value from the list. The listed values are 'Surgery Bookings' and 'Radiology Bookings'.

    6. Consider Patient Current Nursing Unit for search - Select the check box to search for the patient from the ward where the patient is currently admitted.

  2. The Patient Details displayed are Patient ID, Patient Name, Procedure, Surgeon, From-To time of booking, Status.

  3. Click the Patient ID hyperlink to perform the following:

View Booking Details

Verify Checklist

Check-Out Patient

  1. Enter the Check-out time.

  2. Click [Apply] to save the check out details recorded.

 

You can directly enter the check out time and select the ‘Check out Patient’ option without selecting the other two options.  

If the patient is not checked out from ward using this function, the ‘Checkout from Ward’ appears as an option in the OT Slate function.
