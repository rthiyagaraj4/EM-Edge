Widget Selection

From this function you can Select widgets for OT Rooms and staff assignment and View the selected widgets in the dashboard.

The Widget Selection function is used to select the maximum three number of OT rooms. You can preview and view the widgets in the dashboard.

Selecting Widgets

From the Bookings > OT External Display, select the Widget Selection function.

The Widget Selection screen gets displayed.

  1. Select the Staff Assignment check box to display the staff name, role and time in the dashboard.

  2. Select Room check box under the section the Room List to display room details in dashboard. You can select maximum three OT rooms.

  3. Select Auto refresh check box to enable the auto refresh time field. This check box is selected by default.

  4. You can enter the auto refresh time in this field. By default, the auto fresh time is 30.

  5. You can set auto fresh time in seconds and minutes by selecting ‘Seconds’ or ‘Minutes’ from the drop-down list next to the Auto refresh time field. The value ‘Seconds’ is selected by default.

  6. The selected rooms and staff assignment are displayed.

  7. Click [Preview] to preview the dashboard.

  8. Click [Apply] to apply the widget selection based on which the dashboard will be displayed.

  9. Click [Display Dashboard] to display the dashboard. This button is enabled only if the dashboard is closed.

  10. Click [Close Dashboard] to close the dashboard display. This button will be enabled only if the dashboard is open.

You can select maximum three OT rooms. You can set the auto refresh time only if the Auto refresh check box is selected. If you select more than 3 items and click on Apply button in Widget Selection screen, the system displays error message ‘Maximum 3 Items Can be Selected’.

If you enter the data other than 5 to 99 in ‘Auto refresh time’ field in Widget Selection screen, the system displays the error message ‘Auto Refresh Time Should be Minimum 5 and Maximum 99’.

If neither ‘Staff Assignment’ nor one ‘OT room name’ item is selected and you click on Apply button in Widget Selection screen, the system displays the error message ‘At least One Item Should be Selected’.
