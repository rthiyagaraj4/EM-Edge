Link Preoperative Orders

Link Pre-operative Orders function allows linking the orders raised for Order Categories - Laboratory, Monitoring, Patient Care and Radiology for the booked operation/procedure. You can view the Status and details of these orders from OT Slate.

  1. Click [Search] to display the orders or enter the query criteria to search for a specific order.

    1. Order Category – Select the required Order Category from the list. The listed values are All Categories, Laboratory, Monitoring, Patient Care and Radiology. This is a mandatory field.

    2. Order Type – Select the Order Type from the list. The Order Types listed are based on the selected Order Category.

    3. From/To Date – Select the date range to list the orders.

    4. Order ID – Enter the Order ID.

    5. Status – Select the required Status from the list. The listed values are All, Pending for Reviews, Outstanding, Active and Completed.

    6. Catalog – Select the Catalog using the [?] function.

  2. The details of the Order displayed are Order Category, Ordered Time, Order Type, Order Catalog, Ordered By, and Status.

  3. Select the check box against the required Orders to link the pre-operative orders.

  4. Click [Apply] save the link to pre-operative orders.
