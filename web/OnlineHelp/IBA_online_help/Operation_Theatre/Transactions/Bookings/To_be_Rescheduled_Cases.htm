To Be Rescheduled Cases  

From this function you can Search for operations that were marked for 'Reschedule' in the Booking Verification function and Reschedule or Cancel the bookings.

Reschedule is an option provided to reschedule an operation booking. The bookings marked for ‘Reschedule’ is rescheduled or cancelled by specifying a reason.

Rescheduling a Booking

From the Bookings, select the To Be Rescheduled Cases function.

The To Be Rescheduled Cases screen gets displayed.

  1. Click [Search] to display cases marked for rescheduling for the current date or enter the query criteria to search for specific case.

    1. Theatre Date - Select the date to display cases marked for reschedule. The current system date is displayed, by default. This is a mandatory field.

    2. Specialty - Select the Specialty using [?] function.

    3. Surgeon – Select the Surgeon using [?] function.

    4. Ordering Doctor - Enter the doctor who ordered for rescheduling.

  2. On searching, the cases marked for reschedule are displayed. Patient ID, Patient Name, Procedure, Rescheduled Reason and Surgeon are displayed.

  3. Select the Patient ID hyperlink to view and perform the following options:

Reschedule
