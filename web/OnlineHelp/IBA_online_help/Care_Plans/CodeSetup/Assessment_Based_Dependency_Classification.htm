Assessment Based Dependency Classification

From this function you can Associate  dependency levels for the numerical values allotted for the assessment criteria.

The system assigns a dependency level for the patient based on the assessment score arrived at, as a result of the patient assessment.

Associating a Dependency Level to an Assessment Note

From the CP Codes Setup > CP Code Setup, select the Assessment Based Dependency Classification function.

The Assessment Based Dependency Classification screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Assessment Note using the [?] function. This is a mandatory field. The Min Score and Max Score gets displayed for the selected Assessment Note.

  3. Click [Search]. Dependency, Level and Min Score gets displayed.

  4. Enter the Min Score in the numerical format. The dependency is assigned based on the minimum score once the assessment note template is recorded.

  5. Select the check box against the Dependency to be associated with the Assessment Note.

  6. Click [Apply] to save the changes or [Reset] to clear the entered data.

  7. Click [Query] to invoke the Query screen and to search for the existing records.
