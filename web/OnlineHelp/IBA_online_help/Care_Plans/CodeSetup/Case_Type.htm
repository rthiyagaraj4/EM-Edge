Case Type

From this function you can Define and maintain the Case Type applicable for the generation of a care plan. You can Query for an existing Case Type.

Case Types are used to group a set of symptoms or problems. This helps in faster planning while generating a care plan for a patient. Case Types can be classified into the Illness or Wellness based on the conditions, which are associated to it. Based on the nature selected as Illness or Wellness, the system differentiates the case type’s applicability.

Defining a Case Type

From the CP Codes Setup > CP Code Setup, select the Case Type function.

The Case Type screen gets displayed.

  1. Click [Create] to define a new Case Type.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Case Type or [Reset] to clear the entered data.

  4. Click [Query] to search the existing Case Type record within the facility.
