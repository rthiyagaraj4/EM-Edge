Terminology Code for Case Type

From this function you can Associate the problems or diagnosis to a Case Type.

The system suggests the care provider with the eligible Case Types during the generation of care plan when the terminology codes are recorded for the patient.

Creating an association of Terminology Code to Case Type

From the CP Codes Setup > CP Code Setup, select the Terminology Code for Case Type.

The Terminology Code for Case Type screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to search the existing terminology code details.

  3. Click ‘Associated’ hyperlink to view the already associated terminology code to the case type.

  4. Click on respective Alphabet hyperlinks to view the terminology code description starting with the corresponding alphabet.

  5. You can update the Terminology Code records displayed in the search results.

  6. Click [Apply] to save the association of Terminology Code to a Case Type or [Reset] to clear the entered data.
