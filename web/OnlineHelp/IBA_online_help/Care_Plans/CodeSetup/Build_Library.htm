Build Library

From this function you can Create the libraries.

This function is used to build libraries that can be used in generating a care plan for a patient, which is predefined. Libraries are built with one or more terminology codes, one or more goals associated with each terminology code and one or more group of interventions and outcomes associated to each goal.

The library serves as a template while generating a care plan for the patient.

Creating Build Library

From the CP Codes Setup > CP Code Setup, select the Build Library function.

The Build Library screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Library using [?] function. This is a mandatory field. The Library Description, Version No and Status get displayed.

  3. Click Associate Term Code hyperlink, the Terminology Code Search dialog gets displayed.

  4. Enter the respective fields in the Terminology Code Search dialog.

    Enter the following details in order to associate term codes to the Library:

    1. Terminology Set – Select the Terminology Set from the list of values. The values listed are the terminology sets defined in the ‘Terminology Set’ master in the Medical Records Module. This is a mandatory field. Only those terminology sets that are Diagnosis Specific and Procedure Specific are displayed.

    2. Type – Select the Type from the list.  The listed values depend on the terminology set selected. The values are ‘Procedure’ or ‘Diagnosis’.

    3. Classification Level - Select the Classification Level from the list of valid values. The listed values depend on the terminology set selected. The number of levels defined for the terminology set is displayed.

    4. Terminology Code – Enter the Terminology Code in alphanumeric format of 30 characters maximum length.

    5. Terminology Code Description - Enter the Terminology Code Description in alphanumeric format of 50 characters maximum length.

  5. Click [Search]. The Terminology Code, Description, Actual, Potential, Wellness and Sensitive details gets displayed. On clicking any of the alphabets the list of terminology codes belonging to the selected terminology set, selected classification level and which are starting with the alphabet clicked are listed for selection.

  6. Select the checkbox against the Terminology Code to be associated and click [Close].

  7. Click Goal hyperlink, the Goal dialog gets displayed.

    Enter the following details:

    1. Terminology Set - The Terminology Set from which the terminology code is selected is displayed.

    2. Terminology Code – The short description of the Terminology Code is displayed.

    3. Associated – This is a hyperlink. The system displays the goals associated to the terminology code, by default. Click the hyperlink to view the goals, which are associated already to the term code when required.

    4. Alphabets – The goals are listed in the alphabetical order. To view the goals in the alphabetical order, click on the respective alphabet, which is hyperlinked. For example to view the goals, which start with ‘T’, click on the hyperlink ‘T’.

    5. Others – This is a hyperlink. The goals that start with a number or symbol other than an alphabet are displayed on clicking the hyperlink.

    6. Goal Description – The description as entered for the goal in the Outcome master is displayed.

    7. Target Period – This consists of two fields. The first field is to enter the duration. The second field is to capture the duration units.

      1. Duration Units – Select from the list of values. The list values are Day(s), Hour(s), Minute(s), Month(s) and Week(s). By default, the value Day(s) is selected.

      2. Duration – Enter the duration in the numeric format of maximum five characters. The duration entered in this field along with the duration units is taken as the period to achieve the goal selected.

    1. Practitioner Type - Select the Practitioner Type from the list values. The list consists of all the practitioner types, which are system defined.

     

  8. Select the check box against the Goal Description to be associated with the Library. Once you associate the goal(s) by clicking the check boxes against them, the intervention that needs to be performed to achieve the goal has to be mapped.

  9. Click Interventions/Outcomes hyperlink, the Interventions/Outcomes dialog gets displayed.

    Enter the following details:

    1. Intervention Description – The description recorded for the intervention is displayed.

    2. Alt – Enter the number, which is considered as an Alt ID. Interventions with the same Alt ID is taken as Alternative Interventions in a library for a goal.

    3. Relative Start – Enter the period in numerical format and select the period units from the list value. The combination of the two is taken as the relative start period.

    4. Frequency – Select from the list values. The frequency values are defined in the AM Frequency master.

    5. Duration – Enter the duration by selecting it from the list, the duration multiplied with the frequency gives the number of tasks in the task list.

    6. Dependency – Select from the list of values the dependency levels. The values displayed are from one to nine.

    7. Stage – This is from the Stage master meant for mentioning the stage at which the intervention has to be performed.

    8. Select – Select the check box to select the intervention.

    When the intervention(s) are selected, the outcome for the intervention(s) is to be associated by selecting an outcome.

    1. Outcome Description – The Outcome Description recorded for the intervention is displayed.

    2. Nature – The drop-down list has system defined values. The listed values are ‘Actual’, ‘Potential’, ‘Both’ and ‘Wellness’. Select the nature based on the term code associated.

    3. Target Period – This consists of two fields. The first field is to enter the duration. The second field is to capture the duration units.

    4. Select – Select the check box to select the outcomes of the intervention.

    5. Create New Group – Click [Create New Group] to form a group of the interventions and the outcomes selected. The system displays the group number as a hyperlink.

    6. Close - Click [Close] to close the window.

  10. Close the Goal dialog.

  11. Click [Preview] to preview the selected Library. The Term set, Term Code, Goals, Intervention/Outcome groups associated to the library are displayed. If all these are not associated, it is highlighted in red. You need to associate all the above to view the preview.

  12. Click [Save] to save the changes made.

  13. Click [Reset] to clear the entered data.

  14. Click [Query] to invoke the Query screen and to search for the existing records.
