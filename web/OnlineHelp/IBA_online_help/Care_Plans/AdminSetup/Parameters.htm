Parameters

From this function you can Define rules that govern the functioning of the Care Plans module.

This function allows defining rules that influence the work flow of the Care Plans module. The rules defined govern the functioning of the module in general. It is mandatory to setup the parameter before any operations are performed as this record stores important detail about the Care Plans module.

Defining Parameters

From the CP Codes Setup > CP Admin Setup, select the Parameters function.

The Parameters screen gets displayed.

  1. Click [Create] to define the parameters.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the parameters defined or [Reset] to clear the entered data.
