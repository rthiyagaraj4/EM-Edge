Reprint Count Sheet

From this function you can Reprint Count Sheet for the Items posted for Physical Inventory Process.

Count sheet(s) are reprinted for those Physical Inventory ID's, which are not posted for Variance Reporting.

Reprinting Count Sheets

From the ST Physical Inventory, select the Reprint Count Sheet function.

The Reprint Count Sheet screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run] to Reprint count sheet for the items posted for Physical Inventory or [Reset] to clear the entered data.

  3. Select ‘Report Option’ to generate the report.

  4. Click [OK] to Reprint count sheet for the Items posted for Physical Inventory or [Cancel] to cancel the action.
