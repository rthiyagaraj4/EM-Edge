Physical Inventory Entry By File

From this function you can Upload file containing item details required for Physical Stock check function.

This function helps to upload a file containing list of items for doing the operations in physical inventory check function.

Uploading files for Physical Stock Check

From the ST Physical Inventory, select the Physical Inventory Entry By File function.

The Physical Inventory Entry By File screen gets displayed.

  1. Click [Query] to invoke the Query screen.

  2. Select the Physical Inventory ID from the list. The Physical Inventory ID generated in the ‘Phy Inventory - Specific Items’ function is listed in the list.

  3. On selecting the Physical Inventory ID, the Store value is displayed in the Store field.

  4. Click [Browse] to select the required CSV file and selected file path gets displayed in the ‘Upload File’ text box.

  5. If the file is successfully processed, the View button gets enabled.

  6. Click [View] button, all the data in csv file will be loaded into the columns.

  7. Select the ‘Entry Completed’ check box. The check box will be enabled for selection only if uploaded document does not have any remarks.

  8. Click [Apply] to record the Physical Inventory transaction or [Reset] to clear the entered data.

Only the browsed file (csv) data will be shown here, no external data can be added.
