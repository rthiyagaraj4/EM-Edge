Opening Stock

From this function you can Record the Opening Stock details of an Item into the system.

The Logged in User must have the rights to perform Opening Stock function in the associated Store. Opening stock function is performed only for Stock Items in the specified Store and this function is performed only once after the Item is created.

Entering the Opening Stock Details

From the ST Transactions > Receipt, select the Opening Stock function.

The Opening Stock screen gets displayed.

  1. Modify Document Details screen is displayed, by default or click ‘Document’ hyperlink to modify the document details.

  2. On entering the Document Details, click [OK] to record the Opening Stock details or [Reset] to clear the entered data.

  3. Enter the Item to record the Opening Stock details in alphanumeric format of 60 characters maximum length or select the Item using [?] function. This is a mandatory field.

  4. On selecting the Item, the Unit of Measure is displayed against the Item.

  5. Enter the Item details and Batch details in the mandatory (*) fields. <Related Field Description>

  6. Click [Add] to add the Item or [Cancel] to cancel the selected Item details.

  7. The selected Item details are displayed in the Item Details frame on the left hand side. Click ‘Item Details’ hyperlink to modify the Opening Stock details. Click [Modify] to modify the entered details or [Cancel] to cancel the modified changes.

  8. You can select the ‘Delete’ check box to delete the selected record.

  9. Click [Apply] to record the Item consumption details or [Reset] to clear the entered data.

  10. The ‘Confirm Printing’ dialog is displayed based on the attribute set for the Document Type. Click [Yes] to print the Opening Stock details or [No] to cancel the action.

If Trade Name is not associated to the selected Item, the system alerts with a message ‘Trade Name not associated for the Item’. Associate the Trade Name to the Item in Materials Base module to record Opening Stock for the Item.

Opening stock is created only for Items defined as Stock Item in the Store or the system alerts with a message ‘Not a stock Item’ on selection of a Non-Stock Item for Opening Stock.
