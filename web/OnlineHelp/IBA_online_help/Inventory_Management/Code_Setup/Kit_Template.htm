Kit Template

From this function you can Define Kit Template details for an Item and Query for the desired Kit Template records.

The Kit Template function helps to  define a Kit which include Medical Items (non drugs) or Drugs. The non-drugs are grouped in to various Sets/Kits. Kit Template is a template which contains various Items and the corresponding Quantity and UOM. The defined Kit can be used for surgical purpose. The usage of the Kit is recorded in Operation Theatre module.

Defining Kit Template

From the Code Setup, select the Kit Template function.

The Kit Template screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Add] to add the Items and the Items are displayed on the top half of the screen or [Cancel] to cancel the entered Item details.

  4. Click ‘Item Code’ hyperlink to modify the Item details.

  5. Click [Modify] to add the modified Item details and the Item details are displayed in the top half of the screen or [Cancel] to reset the changes.

  6. Click [Apply] to save the Kit Template details or [Reset] to clear the entered data.

  7. Click [Query] to search for existing Kit Template records.
