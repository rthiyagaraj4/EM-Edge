Batch Setup for Disposable Adjustment

From this function you can Define a Batch ID generation rule for disposable drugs received from the patient without Expiry Date, Trade Name and Batch ID.

Patients return the unused items to the pharmacy for which details like Expiry Date, Trade Name and Batch ID are not available. This function allows you to create a predefined batch generation rule for disposable drugs received from the patient.

Based on the batch setup done, a running sequence number will be generated for the batches while disposing the items in the Stock Adjustment for Disposable Items function.

Defining Batch Setup for Disposable Adjustment

From the Code Setup, select the Batch Setup for Disposable Adjustment function.

  1. The Batch Setup for Disposable Adjustment screen gets displayed.

  2. Click [Create] to invoke the Create screen.

  3. Enter the details in the respective fields. <Related Field Description>

  4. Click [Apply] to set the batch ID generation rule.  

  5. Click [Query] to query the setup related for the stores.
