Enter the following details to create Prepare Location records:

  1. Code – Enter the code in alphanumeric format of eight characters maximum length. The code defined indicates the Prepare Location. Once the code is defined and saved, it cannot be changed in query function. This is a mandatory field.

  2. Long Description - Enter the long description of the Prepare Location in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description of the Prepare Location in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Generate Batch – Select the check box to generate the Batch ID automatically for all the Items prepared in the defined Prepare Location or clear the check box to enter the Batch ID manually.

  5. Batch Series – Enter the Batch Series to prefix the Batch Series to the generated batch in alphanumeric format of two characters maximum length. This field is enabled on selecting the ‘Generate Batch’ check box.

  6. Batch ID – Enter the required batch for the Prepare Location in numeric format of eight digits maximum length. This field is enabled on selecting the ‘Generate Batch’ check box.

  7. Enabled – Select the check box to enable the defined Prepare Location. Clear the check box to disable the defined Prepare Location. The check box is selected, by default. Only enabled Prepare Locations are available for selection in the required functions.

You can define 10 Prepare Location codes at a time. If mandatory values are not entered, the system alerts you to enter them. The mandatory field is highlighted with a unique color code. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate codes with a unique color code.
