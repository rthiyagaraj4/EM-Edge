The following report criteria can be specified to generate the Stock Fast/Slow Movement Report:

  1. Report Option – Select the required report option from the list. The listed values are ‘All Items’, ‘By Quantity’, ‘By Value’ or ‘No Movement’. The value ‘All Items’ is displayed, by default.

  2. Store Code [From-To] - Enter the range of Store codes for which the report is to be generated in alphanumeric format of six characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Store code and required values. On selecting the Report Type as ‘Item Consolidated’, this field is disabled.

  3. Item Code [From-To] - Enter the range of Item codes for which the report is to be generated in alphanumeric format of 20 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item code and required values up.

  4. Item Class [From-To] - Enter the range of Item Class for which the report is to be generated in alphanumeric format of four characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Class code and select required values.

  5. Alpha code [From-To] - Enter the range of Alpha Codes for which the report is to be generated in alphanumeric format of 10 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Alpha Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Alpha Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  6. Range - Enter the range value for which the report is to be generated in numeric format of 10 digits maximum length. Both the values are optional.

  7. Period - Enter the period range for which the stock movement has to be generated in mm/yyyy format. ‘To’ value should be greater than or equal to the ‘From’ value.

  8. Report Order - Select the required report order from the list. The listed values are ‘Items Consolidated’ and ‘By Store’. The value ‘Items Consolidated’ is displayed, by default.
