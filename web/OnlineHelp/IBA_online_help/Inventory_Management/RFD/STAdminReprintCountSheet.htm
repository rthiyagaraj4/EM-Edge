Enter the following details to Reprint Count Sheets:

  1. Phy Inventory. ID – Select the required Physical Inventory ID from the list s for which the count sheet has to be reprinted. The Physical Inventory ID's generated and that are not posted for variance calculation are displayed. This is a mandatory field.

  2. Count Sheet Date – The count sheet generation date based on the selected Physical Inventory ID is displayed.

  3. Report Type - The Report Type is displayed based on the selected Physical Inventory ID.
