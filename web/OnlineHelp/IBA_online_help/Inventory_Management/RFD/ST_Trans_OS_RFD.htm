Enter the following details in the Item Entry Area:

  1. Quantity - Enter the Opening Stock Quantity of the Item in numeric format of six digits maximum length. This is a mandatory field.

  2. Value - Enter the Opening Stock Value of the selected Item in numeric format of six digits maximum length. This is a mandatory field.

  3. Cost – The Cost Value is computed by the system, based on the value entered for quantity and value.

  4. Total Quantity – The quantity recorded is displayed as the total quantity, by default.

  5. Remarks – Enter the Opening Stock transaction remarks for the Item in alphanumeric format of 200 characters maximum length.

 

Enter the following batch details:

 

  1. Expiry Date - Enter the Expiry Date for expiry related Items in dd/mm/yyyy format or select the required date using the Calendar function. For non-expiry related Items, the current date is displayed as the receipt date and this option is disabled. If the selected Item is expiry based, ‘Expiry Date’ is displayed as the heading or ‘Receipt Date’ is displayed for non-expiry based items. The current system date will be highlighted in the Calendar.

  2. Batch ID – Enter the Batch ID in numeric format as set in the ‘Batch ID’ label in Item function for expiry and non-expiry related Items as shown in the table.

  4. Sale Price – Enter the sale price.
