Enter the following details to create Repacking Group records:

  1. Code – Enter the code in alphanumeric format of eight characters maximum length. The code defined indicates the Repacking Group. Once the code is defined and saved, it cannot be changed in query function. This is a mandatory field.

  2. Long Description - Enter the long description of the Repacking Group in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description of the Repacking Group in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Generate Batch – Select the check box to generate Batch ID automatically for all the Repacked Groups or clear the check box to enter the Batch ID manually.

  5. Batch Series – Enter the Batch Series to prefix the Batch Series to the generated batch in alphanumeric format of two characters maximum length. This field is enabled on checking the ‘Generate Batch’ check box.

  6. Batch ID – Enter the required batch for the Repacked Group in numeric format of eight digits maximum length. This field is enabled on checking the ‘Generate Batch’ check box.

  7. Enabled – Select the check box to enable the defined Repacking Group. Clear the check box to disable the defined Repacking Group. The check box is selected, by default. Only enabled Repacking Group are available for selection in the required functions.

You can define 10  Repacking Group codes at a time. If mandatory values are not entered, the system alerts you to enter these values. The mandatory field is highlighted with a unique color code. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate codes with a unique color code.
