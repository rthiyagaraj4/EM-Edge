Enter the following details to create the Document Types:

  1. Document Type – Select the Document Type from the list. The Document Types defined for Inventory Management module in Financials Base module are displayed in the list. The Document Types defined at the Entity Level and Facility Level is displayed in the list. This is a mandatory field.

  2. Transaction Type – Select a Transaction Type value from the list. The Transaction Types of Inventory Management module are displayed in the list. This is a mandatory field.

  3. Applicable To Pharmacy – The check box is enabled for Pharmacy Management related transactions like ‘Sales’ (Dispensing) and ‘Sales Return’ (Medication Returns). On selecting the check box, the Document Type is applicable for Pharmacy Management related transactions.

  4. Print Alert Required – Select the check box, if a print alert has to be displayed on completing the transactions or clear the check box to print the documents by default without showing the print alert in the respective transactions.

  5. Print Alert Required – Select this check box to enable/disable online report printing and to reprint the online reports. This check box is not selected by default. It is visible only for Request and Return Transactions.
