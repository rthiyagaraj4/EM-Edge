The following report criteria can be specified to generate the workload statistics:

  1. User [From-To] – Enter the User Name against whom the workload statistic has to be generated in alphanumeric format of 60 characters maximum length or select the required User Name using the [?] function. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  2. Store Code [From-To] - Enter the range of Store codes for which the report is to be generated in alphanumeric format of six characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Store code and select required values.

  3. Doc Date [From-To] – Enter the date in format dd/mm/yyyy format or select the required date using the Calendar function. Entered ‘To’ value should be greater than or equal to the ‘From’ value.
