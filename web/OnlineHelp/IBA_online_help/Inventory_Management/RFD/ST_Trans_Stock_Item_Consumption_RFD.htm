Enter the following patient details in the Patient Details Entry Area:

 

  1. Type – Select the required type of consumption from the list. The list of valid values includes ‘Consumption’, ‘Inpatient’, ‘Outpatient’ and ‘Referral’. The value ‘Consumption’,’ is displayed, by default.

  2. Remarks – Enter the required remarks in alphanumeric format of 20 characters maximum length.

  3. Patient ID – If the consumption is recorded against a patient, enter the Patient ID generated during registration as specified in MP Parameter or select the Patient ID using [?] function. This is a mandatory field on selecting the value as ‘Inpatient’, ‘Outpatient’ or ‘Referral’.

  4. Patient Name – The patient name is displayed based on the selected Patient ID.

  5. Encounter – This field is enabled on selecting the value as ‘Inpatient’, ‘Outpatient’ or ‘Referral’. Select the required encounter number from the encounters listed. The active encounters are listed. If the selected Sale Type is ‘Inpatient’, the Inpatient encounters are displayed in the list, or if the selected Sale Type is ‘Outpatient’, the Outpatient encounters are displayed in the list and if the selected Sale Type is ‘Referrals’ this field is disabled. The Encounter ID has to be entered for Consumption Type of nature ‘Inpatient’ or ‘Outpatient’.
