Enter the following search criteria:

  1. Query Type – Select the query type from the list. The listed values are ‘Summary’ and ‘Detail’. The value ‘Summary’ is displayed, by default. Based on the Query Type value selected, the details of stock inflow and outflow of an Item are displayed.

  2. Result Criteria – Select the required value or result criteria from the list. The listed values are ‘Quantity’ and ‘Value’. The value ‘Quantity’ is displayed, by default. This is applicable only for the query type ‘Summary’.

  3. Item – Enter the required Item in alphanumeric format of 60 characters maximum length or select the Item using [?] function. This field is mandatory for Query Type ‘Detail’.

  4. Store Code – Enter the Store code for which the Item Movement details are to be searched or select the Store Code using the [?] function. This field is mandatory for Query Type ‘Detail’.

  5. Item Class – Enter the required Item Class in alphanumeric format of 15 characters maximum length or select the Item using [?] function.

  6. Transaction Type – This field is enabled only for Query Type ‘Detail’. You can select the required Transaction Type from the listed values.

  7. Stock Item – The value ‘Both’ is displayed by default and this field is disabled. This field is enabled for the Transaction Type ‘Issue/Issue Return/Urgent Issues’.

  8. Period [From-To] – Enter the period from which, details of stock movement has to be viewed. Enter the range of months in mm/yyyy format for which the report has to be generated. Both the values are optional. If ‘From’ value is not entered, the system assumes the lowest value for the ‘From’ and if ‘To’ value is not entered, the system assumes the highest value for the ‘To’ value. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  9. Stock Inflow/Outflow - Select the ‘Stock Inflow/Outflow’ value from the list. The listed values are ‘Inflow’, ‘Outflow’ and ‘Both’. The value ‘Both’ is displayed, by default. This field is displayed for the Query Type 'Details'.
