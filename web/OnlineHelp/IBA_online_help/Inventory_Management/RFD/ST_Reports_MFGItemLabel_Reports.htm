Select the required Report Type for which the report is to be generated:

  1. Report Type - Select the required report type for which the report is to be generated from the list. The listed values are Manufacturing Item Label and Repacking Label.

 

Enter the following report criteria:

  1. Doc Type – Select the required Doc type from the list. All the documents associated to the Manufacturing Receipts in ‘ST Facility Parameter’ of ST Admin Setup will be displayed in the list. This is a mandatory field.

  2. Store Code – Enter the Store Code in alphanumeric format of six characters maximum length or select the Store Code from the [?] function.

  3. Doc No. – Enter the document number in alphanumeric format of 20 characters maximum length.

  4. No of Copies – Enter the number of copies of the Item Label to be generated in numeric format of 10 digits maximum length. This is a mandatory field.

  5. Period Date [From-To] – Enter the period in date format of dd/mm/yyyy or select the date range using the Calendar function.

  6. Label Type – Select the required Label Type value from the list. The listed values are ‘With Formulary’ and ‘Without Formulary’. On selecting the value as ‘With Formulary’ the formulary details entered for manufacturing the Item will be printed on the label or the formulary details will not be printed on selecting the value ‘Without Formulary’. This field is not applicable for Report Type ‘Repacking Label’.
