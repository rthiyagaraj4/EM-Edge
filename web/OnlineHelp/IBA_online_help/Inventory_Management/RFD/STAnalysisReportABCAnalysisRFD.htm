The following report criteria can be specified to generate the Ageing Analysis Report:

  1. Item Code [From-To] - Enter the range of Item codes for which the report is to be generated in alphanumeric format of 20 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item code and select required values.

  2. Item Class [From-To] - Enter the range of Item Class for which the report is to be generated in alphanumeric format of four characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Class code and select required values.

  3. Alpha code [From-To] - Enter the range of Alpha Codes for which the report is to be generated in alphanumeric format of 10 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Alpha Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Alpha Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  4. Item Analysis1 – Select the required value for Item Analysis1 from the list.

  5. Item Analysis2 – Select the required value for Item Analysis2 from the list.

  6. Item Analysis3 – Select the required value for Item Analysis3 from the list.

  7. ABC Criteria – Select the required criteria for ABC analysis from the list. The listed values are ‘Unit Price’, ‘Value’, ‘Quantity’ and ‘ABC Code (Item Master)’.

  8. Report Type - Select the required Report Type from the list. The listed values are ‘All’, ‘Only A’, ‘Only B’ and ‘Only C’. The value ‘All’ is displayed, by default.

 

ABC Cutoff Range

  1. For A, B and C (Minimum) – Enter the minimum cutoff range to classify and group Items under A, B and C group.

  2. For A, B and C (Maximum) – Enter the minimum cutoff range to classify and group Items under A, B and C group.
