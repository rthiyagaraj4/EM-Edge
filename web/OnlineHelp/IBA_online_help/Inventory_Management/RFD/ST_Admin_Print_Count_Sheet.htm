Enter the following criteria to print the count sheet:

  1. Physical Inv Process Mode – Select the required value for the mode from the list. The list values are ‘Annual’ and 'Perpetual’. The value ‘Annual’ is displayed, by default. All the fields except Store Code is disabled, if ‘Physical Inv Process Mode’ is selected as ‘Annual’.

  2. Store Code [From-To] - Enter the Store Codes range for which the count sheets are to be generated in alphanumeric format of six characters maximum length and select the Store Codes using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store Code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  3. Item Code [From-To] - This field is enabled when the Physical Inv Process mode is selected as ‘Perpetual’ or on selection of ‘Annual’ this field is disabled. Enter the Item Codes range for which the count sheet is to be generated in alphanumeric format of 20 characters maximum length or select the Item Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected item code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ item code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  4. Item Class [From-To] - This field is enabled when the Physical Inv Process Mode is selected as ‘Perpetual’ or on selection of ‘Annual’ this field is disabled. Enter the Item Class range for which the count sheet is to be generated in alphanumeric format of four characters maximum length or select the Item Class using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  5. Alpha code [From-To] - This field is enabled when the Physical Inv Process Mode is selected as ‘Perpetual’ or on selection of ‘Annual’ this field is disabled. Enter the alpha codes range for which the count sheet is to be generated in alphanumeric format of 10 characters maximum length or select the Alpha Code using the [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Alpha Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Alpha Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  6. Unit Cost [From-To] - This field is enabled only when the Physical Inv Process Mode is selected as ‘Perpetual’ or on selection of ‘Annual’, this field is disabled. Enter the Unit Cost range for which the count sheet is to be generated in numeric format of 15 digits maximum length. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Unit Cost and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Unit Cost. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  7. Stock Item Value [From-To] - This field is enabled only when the Physical Inv Process Mode is selected as ‘Perpetual’ or on selection of ‘Annual’ this field is disabled. Enter the Stock Item Value range for which the count sheet is to be generated in numeric format of 15 digits maximum length. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Stock Item Value and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Total Item Value. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  8. Physical Inventory Type – This field is enabled only when the Physical Inv Process Mode is selected as ‘Perpetual’ or on selection of ‘Annual’ this field is disabled. Enter the Physical Inventory Type in alphanumeric format and the Physical Inventory is considered as perpetual count. When the Physical Inventory Type is left blank, the Items are considered for generating the count sheet, irrespective of whether the Item is already counted.

  9. Physical Count for Batch Qty - Select the value from the list. The listed values are 'All', 'Zero Quantity' and 'Quantity > 0'. The value 'All' is displayed, by default. This field is enabled, if 'Phy Inv Process Mode' is selected as 'Perpetual'.

  10. Physical Count – This field is enabled only when the Physical Inv Process Mode is selected as ‘Perpetual’ or on selection of ‘Annual’ this field is disabled. Select the required Physical Count Value from the list. The listed values are 1,2,3,4,6,12 and blank. On leaving the Physical Count field blank, the Items are considered for count sheet generation, irrespective of whether the Item is already counted. On selecting a Physical Count Value, the system considers those Items whose physical count per year as set in ‘Item’ function is same as the value selected here and based on the physical counts already done for the current financial year.

  11. Expiry Items – This field is enabled only when the Physical Inv Process Mode is selected as ‘Perpetual’ or on selection of ‘Annual’ this field is disabled. Select the required expiry value from the list. The list of valid values is ‘All Items’, ‘Expiry Items’ and ‘Non-Expiry Items’. The value ‘All items’ both expiry and non-expiry will be displayed in the item summary, by default.

  12. Report Order – This field is enabled only when the Physical Inv Process Mode is selected as ‘Perpetual’ or on selection of ‘Annual’ this field is disabled. Select the required Report Order from the list. The listed values are ‘By Item’ and ‘By Bin Location’. The value ‘By Item’ is displayed, by default.

 

Analysis

  1. Item Analysis Code 1 – Select the required value for Analysis Code 1 from the list.

  2. Item Analysis Code 2 – Select the required value for Analysis Code 2 from the list.

  3. Item Analysis Code 3 – Select the required value for Analysis Code 3 from the list.

  4. ABC Analysis – Select the required value for ABC analysis from the list. The lists of valid values are ‘A’, ‘B’ and ‘C’.

  5. Report Option – This field is enabled only when the Physical Inv Process Mode is selected as Perpetual. Select the Report Option from the list. The listed values are ‘Fast Moving’ and ‘Slow Moving’. By default, ‘Select’ is displayed.

  6. Consumption Period [From and To] – This field is enabled only when the Physical Inv Process Mode is selected as Perpetual. Enter the consumption period in MM/YYYY format. By default, current month is displayed. This is a mandatory field.

  7. Top Items By – This field is enabled only when the Physical Inv Process Mode is selected as Perpetual. Enter the number of items in the ‘Top’ text box and select ‘Quantity’ or ‘Value’ from the list.

The  items are classified as Slow or Fast Moving based on their consumption. If any two items have the same fast and slow movement, then the items are printed in alphabetic order of item description in the Count Sheet.
