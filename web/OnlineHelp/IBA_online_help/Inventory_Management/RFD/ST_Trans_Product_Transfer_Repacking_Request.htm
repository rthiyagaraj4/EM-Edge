Enter the following details to raise a Product Transfer/Repacking Request:

 

  1. Repack Store – Select the required Store raising the Transfer/Repack request from the list. The Stores to which the logged in user has access are displayed. This is a mandatory field.

  2. Required Item – Enter the required Repack Item in alphanumeric format of 60 characters maximum length or select the required Item using [?] function. On selecting the Item the corresponding UOM is displayed. This is a mandatory field.

  3. Item to be used – Enter the Item to be used for repacking in alphanumeric format of 60 characters maximum length or select the required Item using [?] function. You cannot select the same Item as the ‘Required Item’ or an alert is displayed as shown below. The repack group for both the ‘Required Item’ and ‘Item to be used’ has to be the same as set for the Item in the Item function or an error message is displayed. This is a mandatory field.

  4. Required Quantity – Enter the required quantity to be used for repacking in numeric format of 12 digits maximum length. This is a mandatory field.

  5. Repack Size – Enter the Repack Size in numeric format of 12 digits maximum length. The value one is displayed, by default. This is a mandatory field.

  6. Quantity – The quantity based on the value entered for ‘Required Quantity’ and ‘Repack factor’ is displayed. You can modify the displayed quantity by entering the required quantity in numeric format of 12 digits maximum length. This is a mandatory field.

 

Remarks

  1. Req By – The current logged in user name is displayed, by default. You cannot modify the Requested By name.

  2. Remarks - Enter the remarks in alphanumeric format of 400 characters maximum length.
