Enter the following details:

  1. Req UOM – Select the Request UOM Quantity from the list. The listed UOMs are defined in the Unit of Measure function in Application Masters module.

  2. Req UOM Qty – Enter the Request UOM Quantity in numeric format of seven digits maximum length. This is a mandatory field.

  3. Req Qty – On entering the Request UOM Quantity, the Request Quantity is displayed same as the Request UOM Quantity and the value is disabled. This is a mandatory field.
