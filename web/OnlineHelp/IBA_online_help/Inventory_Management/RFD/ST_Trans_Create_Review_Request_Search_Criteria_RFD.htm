Displays the following search criteria:

Search criteria:

  1. Item Code – Select this Radio button if you want to search by Item code. You have to enter the item code in the Search string.  

  2. Machine Code – Select this Radio button if you want to search by machine code. You have to enter the machine code in the Search string.

  3. Trade ID – Select this Radio button if you want to search by trade ID. You have to enter the trade id in the Search string.

  4. Alpha Code – Select this Radio button if you want to search by alpha code. You have to enter the alpha code in the Search string.

  5. Item Name – Select this Radio button if you want to search by Item name. You have to enter the item name in the Search string.

  6. Part Number – Select this Radio button if you want to search by part number. You have to enter the part number in the Search string.

  7. Trade Name – Select this Radio button if you want to search by trade name. You have to enter the trade name in the Search string.

  8. Consider item in TargetStore – Select this check box if you want to consider the item in Target store during search.

  9. Kit Template – Select this check box if you want to search in the Kit template. If you select this check box, all the fields are disabled.

  10. Store – Enter the store code in alphanumeric format or click [?] function to search and select the code of the store from the Store dialog. The enabled ‘Stores’ defined in Materials Base module are displayed.

  11. Search String – Enter the required details based on the radio button selection i.e., if you select the Item code radio button, you have to enter the item code here. This is a mandatory field.

  12. Patient Category – Select the patient category from the drop-down list. The available options are Medical, Non-medical and All. The value ‘All’ is selected by default.

  13. Sole Source - Select the sole source from the drop-down list. The available options are Sole Source, Non Sole Source and All. The value ‘All’ is selected by default.

  14. Pharmacy Stock Item - Select the pharmacy stock item from the drop-down list. The available options are Pharmacy Stock Items, Non PS items. The value ‘All’ is selected by default.

  15. ABC Analysis - Select the ABC analysis from the drop-down list. The available options are Select, A, B, and C. The value ‘Select’ is selected by default.

  16. Manufacturer – Select the manufacturer from the drop-down list. The Manufactureres defined in the Master Setup are listed. The value ‘Select’ is displayed by default.

  17. Consignment - Select the ABC analysis from the drop-down list. The available options are all, Consignment and Non Consignment. The value ‘All’ is selected by default.

  18. Search Criteria - Select the Search Criteria from the drop-down list. The available options are Starts With, Contains With and Ends With. The value ‘Start With’ is selected by default.

  19. Manufacturing - Select the value from the drop-down list. The listed values are All, Manufacturing and Non Manufacturing. The value ‘All’ is displayed by default.

  20. Expiry - Select value from the drop-down list. The listed values are All, Expiry and Non Expiry. The value ‘All’ is displayed by default.

  21. Item Class - Select the required Item class from the list.All the Item Classes with the enabled box selected in the ‘Item Class’ function is displayed in the list.

  22. Item Analysis 1 - Select the Item analysis from the drop-down list. All the records from Item Analysis master is displayed in the list.

  23. Item Analysis 2 - Select the Item analysis from the drop-down list. All the records from Item Analysis master is displayed in the list.

  24. Item Analysis 3 - Select the Item analysis 3 from the drop-down list. All the records from Item Analysis master is displayed in the list.

  25. Click [Previous] to return back to the previous screen.

  26. Click [Next] to redirect to the next screen.

  27. Click [Search] to displayed the item details based on the search criteria entered.

 

Search results:

  1. Item Code – Displays the item code that matches the search string.

  2. Item – Displays the item that matches with the item code. You can click the Information icon  near the item name to view either 'Item Remarks' or 'Item Specification' or both for the selected item in a popup or as a tool tip. The Information icon will be available only if either 'Remarks' or 'Specification' or both exists for that item.

  3. Expiry – Displays the Tick mark to indicate it is an expiry item or X mark to indicate it is a non-expiry item.

  4. Stock UOM – Displays the stock unit of measurement of the item.

  5. Item Class – Displays the class of the item.

  6. Alpha Code –Displays the alpha code of the item.

  7. Qty.On Hand – Displays the quantity of items on hand in the store so that you can view and select the items  in stock. This column is visible only when the Store value is selected in the search criteria.

  8. Available Qty –  Displays the available quantity of items in the store. This column is visible  only when the Store value is selected in the search criteria. The Sale/Issue will not proceed if adequate Available Qty is not available in the chosen store.

The system  performs  inadequate stock validation  if you select an item with inadequate stock in spite of the ‘Quantity on Hand’ and ‘Available Quantity’ displayed against each item.
