Enter the following details:

  1. Schedule Code - Enter the Schedule Code in alphanumeric format of 10 characters maximum length. Enter the description of the schedule code in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  2. Document Type - Select the required Document Type from the list. The Document Type as set for 'Request Issue' in ‘ST Parameter for Facility’ is displayed, by default. This is a mandatory field.

  3. Req By Store - Select the Requested by Store from the list. Only those Stores are displayed for which you have access and ‘Request Issue’ transaction function applicable. This is a mandatory field.

  4. Req On Store - Select the Requested on Store from the list. Only those Stores are displayed for which you have access and ‘Request Issue’ transaction function applicable. This is a mandatory field.

  5. Frequency - Enter the Frequency in the numeric format. The Frequency should be greater than one. This is a mandatory field.

  6. Starting From - Enter the Starting From date in the format of dd/mm/yyyy or select the schedule start date using the Calendar function. The entered date should not be less than current system date. This is a mandatory field.

  7. Item Class – Select the required Item Class from the list. The Item Class associated to the selected Requested By store and Requested On Store is displayed in the list.

  8. Consignment – Select the required value for consignment from the list. The listed values are ‘All’, ‘Consignment’ and ‘Non-Consignment’. The value ‘All’ is displayed, by default.

  9. Item Analysis 1 – Select the required value for Item Analysis 1 from the list.

  10. Item Analysis 2 - Select the required value for Item Analysis 2 from the list.

  11. Item Analysis 3 - Select the required value for Item Analysis 3 from the list.

  12. Auto Authorised? - Select the check box to automatically authorize the auto generated requests.

  13. Enabled - Select the check box to enable the defined record. Clear the check box to disable the defined record. The check box is selected, by default.
