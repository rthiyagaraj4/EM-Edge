Enter the following criteria to generate reports on Batch Complaints:

  1. Item Code [From-To] - Enter the Item codes range for which the report is to be generated in alphanumeric format of 20 characters maximum length or select the Item Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  2. Batch ID - Enter the Batch IDs for which the report is to be generated in alphanumeric format of 10 characters maximum length. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Batch ID and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Batch ID. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  3. Expiry Date [From-To] - Enter the required expiry date range for which the complaints recorded has to be viewed in dd/mm/yyyy format or select the required date using Calendar function.
