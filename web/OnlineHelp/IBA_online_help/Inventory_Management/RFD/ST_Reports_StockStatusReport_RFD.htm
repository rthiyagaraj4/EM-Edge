The following report criteria can be specified to generate the Stock Status Report:

  1. Report Type – Select the required Report Type from the list. The listed values are ‘Items Consolidated’, ‘By Item’, ‘By Store Batch’, ‘By Bin Store Location’, ‘By Store’ and ‘By Item Batch’. The value ‘Item Consolidated’ is displayed, by default.

  2. Store Code [From-To] - Enter the range of Store Codes for which the report is to be generated in alphanumeric format of six characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Store Code and select required values. This field is disabled on selecting the Report Type as ‘Item Consolidated’. If the ‘Restrict Viewing Stock Status’ privileges are set to ‘User access to stores’, only the Stores to which logged in user has access are displayed in Store Code [From - To] fields.

  3. Item Code [From-To] - Enter the range of Item Codes for which the report is to be generated in alphanumeric format of 20 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Code and select required values.

  4. Item Class [From-To] - Enter the range of Item Class for which the report is to be generated in alphanumeric format of four characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Class code and select required values.

  5. Alpha code [From-To] - Enter the range of alpha codes for which the report is to be generated in alphanumeric format of 10 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected alpha code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ alpha code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  6. Unit Cost [From-To] - Enter the range of unit costs of item(s) for which the report is to be generated in numeric format of 10 digits maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected unit cost and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ unit cost. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  7. Stock Value [From-To] - Enter the range of stock value of item(s) for which the report is to be generated in numeric format of 10 digits maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected stock value and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ stock value. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  8. Bin Location Range [From-To] - Enter the range of Bin Locations for which the report is to be generated in numeric format of 10 digits maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected bin location range and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ bin location range. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. This field gets enabled on selecting the Report Type as ‘By Store Batch’, ‘By Bin-Store Location’ and ‘By Item Batch’.

  9. Item Selection – This field is enabled on selecting the Report Type as ‘Item Consolidated’. Select the required item level from the list. The listed values are ‘All Items’, ‘Zero Stock Items’, ‘Item Less than Reorder Level’, ‘Item Less than Minimum Level’, ‘Item with Stock’ and ‘Item Greater than Maximum Level’. The value ‘All Items’ is displayed, by default.

  10. Stock Position Of – Select the required stock position of the item(s) from the list. The listed values are ‘All Stores’, ‘Reorderable Stores Stock’ and ‘Non-Reorderable Stores Stock’. The value ‘All Stores’ is displayed, by default.

  11. Expiry Selection – Select the required nature of expiry from the list. The listed values are ‘All’, ‘Expiry Items’ and ‘Non Expiry Items’. The value ‘All Items’ is displayed, by default.

  12. Batch Selection – Select the required value for batches from the list. The listed values are ‘All Batches’, ‘Suspended Batches’ and ‘Active Batches’. The value ‘All Batches’ is displayed, by default.

  13. Item Analysis1 – Select the required value for Item Analysis1 from the list.

  14. Item Analysis2 – Select the required value for Item Analysis2 from the list.

  15. Item Analysis3 – Select the required value for Item Analysis3 from the list.

  16. Consignment Items – Select the value from the list. The listed values are ‘All’, ‘Consignment’ and ‘Non Consignment’. The value ‘All’ is displayed, by default.

  17. Top (Item(s)) – Enter the required value to view the top () items with the at most stock in numeric format of four digits maximum length. This field is applicable for report type ‘Item Consolidated’.

  18. Report Order – Select the required report order from the list. The listed values are ‘Item Code’ and ‘Short Description’. The value ‘All Items’ is displayed, by default.

  19. Nature – Select the required value for nature from the list. The listed values are ‘Both’, ‘Enabled’ and ‘Disabled’.

  20. Sort Order By - Select the Sort Order By from the drop-down list based on which the results have to be sorted. The listed values are ‘Item Code’, ‘Short Description’, ‘Item Alpha Code’, ‘Generic Name’, and ‘Dosage Form’.
