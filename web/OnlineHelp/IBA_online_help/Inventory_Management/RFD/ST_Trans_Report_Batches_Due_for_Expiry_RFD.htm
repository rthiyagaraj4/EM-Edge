Enter the following report criteria to generate the required report:

 

  1. Report Type – Select the required Report Type from the list. The listed values are ‘By Item’ and ‘By Store’. The value ‘By Item’, by default.

  2. Cut Off Date [From-To] – Enter the cut off date range for which the batches due for expiry are to be searched in dd/mm/yyyy format or select the required date using the Calendar function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ date and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ date. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  3. Store Code [From-To] – Enter the Store codes range for which the batches due for expiry are to be searched or select the Store Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  4. Item Code [From-To] - Enter the Item Codes range for which the batches due for expiry are to be searched in alphanumeric format or select the Item Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  5. Item Class [From-To] - Enter the Item Class Range for which the batches due for expiry are to be searched in alphanumeric format or select the Item Class using the [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  6. Alpha Code [From-To] - Enter the Alpha Codes range for which the batches due for expiry are to be searched in alphanumeric format of 10 characters maximum length or select the Alpha Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Alpha Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Alpha code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  7. Batch ID – Enter the Batch ID for the condemned batch in alphanumeric format of 10 characters maximum length.

  8. Report Order – Select the required value for Report Order from the list. The listed values are ‘By Item Code’ and ‘By Short Description’. The value ‘By Item Code’ is displayed, by default.
