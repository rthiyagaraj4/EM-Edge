Enter the following details to create Assembly Details:

 

  1. Item – Enter the required Item in alphanumeric format of 60 characters maximum length or select the Item using [?] function. The Items with ‘Manufacturing Item’ check box selected are displayed in the list. This is a mandatory field.

  2. Stock UOM - On selecting or entering the required Item its stock Unit of Measure is displayed against the Item.

  3. Prep. Type – Select the required preparation type from the list of valid values. The list of valid values is  Extemporaneous.

  4. Extemporaneous – Ad Hoc, Standard Formulary – Extemp,

  5. Standard Formulary – Gale Internal, and

  6. Standard Formulary - Gale. External. By default, the value ‘Standard Formulary – Gale. Internal’ is displayed.  

  1. Preparation Mode - Select the required Preparation Mode from the list of valid values. The list of valid values is ‘Sterile’, ‘Non-Sterile’ and ‘Others’. The value ‘Sterile’ is displayed, by default.

  2. Qty – Enter the required quantity that needs to be prepared in numeric format of eight digits maximum length. This is a mandatory field.

  3. Label Type – Select applicable Label Type from list of valid values. The list of valid values is ‘With Formulary’ and ‘Without Formulary’.

  4. Prepare Instructions – Click ‘Prepare Instruction’ hyperlink to enter Manufacturing Instructions. On entering the Manufacturing Instructions, click [Close] to exit from the screen.

  5. Process Loss% - Enter the process loss that might occur in the manufacturing process in numeric format of eight digits maximum length. This is a mandatory field.

 

On entering the required Manufacturing Item details, associate the Ingredients which are used in manufacturing the main Item by entering the following details.

 

  1. Ingredients - Enter the required Ingredients description in alphanumeric format of 60 characters maximum length or select the Ingredients using [?] function. This is a mandatory field.

  2. Stock UOM - On selecting or entering the required Item its stock Unit of Measure is displayed against the Ingredient.

  3. Mfg Instructions – Select the Manufacturing Instructions from the list to associate with the Ingredients.

  4. Quantity – Enter the quantity of the Ingredients required in numeric format of eight digits maximum length. This is a mandatory field.

  5. MFG UOM – Select the Manufacturing UOM from the list. This field is mandatory. This is a mandatory field.

  6. Stock Conv Factor - On selection of the MFG UOM the corresponding Stock Conv. Factor is displayed against the MFG UOM.

  7. Sequence No – Enter the Sequence Number in alphanumeric format of two characters maximum length. The Sequence Number cannot be duplicated. This is a mandatory field.

  8. Component – Select the check box, if the selected Item is a component. The check box is selected, by default.

  9. Mandatory - Select the check box, if the selected Item is mandatory for the selected Manufacturing Item.

  10. Print On Label – Select the check box, to print the Item Name.
