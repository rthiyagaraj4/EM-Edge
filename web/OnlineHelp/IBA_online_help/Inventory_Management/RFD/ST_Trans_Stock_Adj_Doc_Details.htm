Enter the following details to modify the required document details:

 

  1. Doc Type – Select the required Document Type from the list. The documents associated to the transaction ‘Stock Adjustment’ are displayed in the list. The Document Type, as set in ‘ST Facility Parameter’ is displayed, by default. This field is mandatory.

  2. Doc No. - The Document Number generation depends on the selected Document Type. You are informed if the Document Number has to be manually generated for the selected Document Type. Enter the document number in numeric format of eight digits maximum length. If the system generates the number, then user is not allowed to enter the field value.  This field is mandatory.

  3. Doc Date - The system date is displayed, by default. You can change the document date by clicking on the calendar function or entering the date in date format of dd/mm/yyyy. The entered date must not be less than the Processing month and Year set in ‘ST Facility Parameter’ and greater than current system date. This field is mandatory.

  4. Doc Ref – Enter the Document Reference in alphanumeric format of 15 characters maximum length.

  5. Store – Select the Store from the list. Only those stores are displayed for which you have access and ‘Stock Adjustment’ transaction function applicable. This field is mandatory.

  6. Item Class – Select the required item class from the listed values.
