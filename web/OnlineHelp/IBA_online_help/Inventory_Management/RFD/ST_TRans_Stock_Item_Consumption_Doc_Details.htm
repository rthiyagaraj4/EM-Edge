Enter the following details to modify the required document details:

 

  1. Doc Type – Select the required Document Type from the list. The documents associated to the transaction ‘Stock Item Consumption’ are displayed in the list. The Document Type, as set in ‘ST Facility Parameter’ is displayed, by default. This is a mandatory field.

  2. Doc No. - The Document Number generation depends on the selected Document Type. If the Document Number has to be generated manually for the selected Document Type, you are informed. Enter the Document Number in numeric format of eight digits maximum length. If the system generates the number, you are not allowed to enter the field value. This is a mandatory field.

  3. Doc Date - The system date is displayed, by default. You can change the document date by entering the date in date format of dd/mm/yyyy or select the required date using the Calendar function. This is a mandatory field.

  4. Doc Ref – Enter the Document Reference in alphanumeric format of 15 characters maximum length.

  5. Store – Select the Store from the list. The Stores for which you have access and ‘Sales’ transaction function applicable is displayed. This is a mandatory field.

  6. Item Class – Select the required Item Class from the list.
