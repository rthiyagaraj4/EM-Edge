The following report criteria can be specified to generate the Transactions Audit Trail report:

  1. Store Code [From-To] - Enter the range of Store Codes for which the report is to be generated in alphanumeric format of six characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Store Code and select required values.

  2. Item Code [From-To] - Enter the range of Item Codes for which the report is to be generated in alphanumeric format of 20 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the item code and select required values.

  3. Item Class [From-To] - Enter the range of Item Class for which the report is to be generated in alphanumeric format of four characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Class code and select required values.

  4. Alpha code [From-To] - Enter the range of Alpha Codes for which the report is to be generated in alphanumeric format of 10 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Alpha Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Alpha Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  5. Date [From-To] – Enter the date in format dd/mm/yyyy format. Entered ‘To’ value should be greater than or equal to the ‘From’ value.

  6. Document Ref [From-To] – Enter the document reference number in alphanumeric format of 10 characters maximum length.

  7. Transaction Type – Select the required Transaction Type from the list. On selection of a Transaction Type the Audit Trail report will be generated for the selected transaction.

  8. Report Order - Select the Report Order as ‘By Item’ or ‘By Document Date’ to produce the report in the required sorting order. The value ‘By Item’ is displayed, by default.

  9. Report Type - Select the required report type to generate the report from the list. The listed values are ‘Both’, ‘Consumption’ and ‘Non-Consumption’. The value ‘Both’ is displayed, by default.
