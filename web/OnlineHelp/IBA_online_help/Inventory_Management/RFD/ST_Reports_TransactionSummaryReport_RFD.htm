Select the required code for which the report is to be generated as follows:

  1. Transaction - Select the required transaction code for which a report is to be generated from list of ST Transaction codes. The following screen gets displayed on selection of the code.

 

Enter the following report criteria to generate a report for the selected code:

  1. Store Code [From-To] - Enter the range of Store codes for which the report is to be generated in alphanumeric format of six characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Store code and select required values.

  2. Item Code [From-To] - Enter the range of Item codes for which the report is to be generated in alphanumeric format of 20 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected item code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item code and select required values.

  3. Alpha code [From-To] - Enter the range of Alpha codes for which the report is to be generated in alphanumeric format of 10 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Alpha code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Alpha code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  4. Item Class [From-To] - Enter the range of Item Class for which the report is to be generated in alphanumeric format of four characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Class code and select required values.

  5. Doc Type Code [From-To] - Enter the range of Document Type codes for which the report is to be generated in alphanumeric format of six characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected doc type code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ doc type code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the doc type code and select required values.

  6. Doc No - Enter the range of Document number for which the report is to be generated in alphanumeric format of 10 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Document Type code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Document Type code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  7. Doc Date [From-To] - Enter the range of dates for which the report is to be generated in alphanumeric format of 20 characters maximum length or click the Calendar function to select the required date. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected date and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ date. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  8. Item Analysis1 – Select the required value for Item Analysis1 from the list.

  9. Item Analysis2 – Select the required value for Item Analysis2 from the list.

  10. Item Analysis3 – Select the required value for Item Analysis3 from the list.
