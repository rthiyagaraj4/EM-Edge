Enter the following details to Notify, Suspend or Reinstate Batch:

  1. Facility - Select ‘All’ if you want to select all the facilities. Else, select the specific facility. By default, the logged in facility is displayed.

  2. Store Code – Select a specific store associated with chosen facility.  The store name is  displayed above the search result column header.

  3. All Stores – Select this check box is you want to select all the stores in the facility. When you select the ‘All Stores’ check box, the Store search is automatically disabled. When ‘All Stores’ checkbox is cleared, the Store search is mandatory.

  4. Item – Enter the required item name in alphanumeric format of 60 characters maximum length or select the item using [?] function. This field is mandatory. On selecting the required Item, the Unit of Measure and Item expiry based details are displayed.

  5. Expiry – Displays a Tick mark if the item is an expired item.

  6. Manufacturer – Select the manufacturer name from the list. The manufacturers associated to the selected item are displayed.

  7. Batch ID – Enter the Batch ID in numeric format of 10 digits maximum length.

  8. Select Batch For – Select the action from the drop-down list to display the batch details based on the selected action. The actions are ‘Notify’, ‘Suspend’ and ‘Reinstate’. The value ‘Notify’ is displayed, by default.

    • When you select ‘Notify’, the Include Notified Batches check box is selected.

    • When you select ‘Suspend’, the Only Notified Batches check box is cleared.

    • When you select ‘Re-instate’, the Include Notified Batches check box is cleared.

  9. Only Notified Batches – Select the check box to include only notified batches in the batch result area or clear the check box to display all the batches in the batch result area.
