The following report criteria can be specified to generate Purchase Advice Report:

  1. Store Code [From – To] – Enter the range of Store Code for which the report is to be generated in alphanumeric format of six characters maximum length.  Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Store Code and select required values.

  2. Item Code [From-To] - Enter the range of Item Codes for which the report is to be generated in alphanumeric format of 20 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected item code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Code and select required values.

  3. Item Class [From-To] - Enter the range of item class for which the report is to be generated in alphanumeric format of four characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Class code and select required values.

  4. Alpha code [From-To] - Enter the range of alpha codes for which the report is to be generated in alphanumeric format of 10 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected alpha code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ alpha code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  5. Special Item – Select the required value for special item to generate a report having only Special Item or Non-special Item. On selecting ‘Both’ the report would display both Special and Non-special Items.

  6. Item Analysis1 – Select the required value for Item Analysis1 from the list.

  7. Item Analysis2 – Select the required value for Item Analysis2 from the list.

  8. Item Analysis3 – Select the required value for Item Analysis3 from the list.

  9. Show Batch Info – Select the check box to include batch information of the selected item(s) in the report. The check box is cleared, by default.

  10. Item Selection – This field is enabled on selecting the Report Type as ‘Item Consolidated’. Select the required item level from the list. The listed values are ‘All Items’, ‘Item below ROL’, ‘Item below Minimum Level’ and ‘Zero Stock’. The value ‘All Items’ is displayed, by default.

  11. Stock Position Of – Select the required value for stock position from the list. The listed values are ‘All Stores’, ‘Reorderable Stores Stock’ and ‘Non Reorderable Stores Stock’. The value ‘All Stores’ are displayed, by default.
