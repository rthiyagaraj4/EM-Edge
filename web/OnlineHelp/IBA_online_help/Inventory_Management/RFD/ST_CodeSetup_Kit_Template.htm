Enter the following details to create Kit Template:

  1. Kit Template Code – Enter the Kit Template Code in alphanumeric format of 20 characters maximum length.

  2. Kit Template Description – Enter the Kit Template Description in alphanumeric format of 40 characters maximum length.

  3. Enabled – Select the Enabled check box to enable the record. The check box is selected, by default.

 

Enter details to Add/Modify Item:

 

  1. Item - Enter the required Item in alphanumeric format of 60 characters maximum length or select the Item using [?] function. This is a mandatory field.

  2. Stock UOM - On selecting or entering the required Item, its stock Unit of Measure is displayed against the selected Item.

  3. Quantity - Enter the quantity of the Items in numeric format of six digits maximum length. This field is mandatory.

  4. UOM – Select the UOM from the list. The UOMs defined in Application Masters module are listed.

  5. Sequence No - Enter the Sequence Number in numeric format of four digits maximum length. The Sequence Number cannot be duplicated. This is a mandatory field.

  6. Enabled – Select the Enabled check box to enable the record. The check box is selected by default.

  7. Usage Type – Select the Usage Type from the list. The listed values are ‘Required’, ‘Optional Required’, ‘Optional Pre-Selected’ and ‘Optional De-Selected’. This is a mandatory field.  By default, the value 'Optional Required' is displayed.

Based on the selected value, the Select check box is selected/cleared in ‘Create Request’ and the Delete check box is selected/cleared in the transactions. See the table below.

 

Usage Type drop-down list value

Select check box in ‘Create Request’

Delete check box in transactions

Required

Select check box is disabled and selected by default.

Delete check box is disabled and cleared by default.

Optional Required

Select check box is enabled and selected by default.

Delete check box is enabled and cleared by default.

Optional Pre-Selected

Select check box is enabled and cleared by default.

Delete check box is enabled and selected by default.

Optional De-Selected

Select check box is disabled and cleared by default.

Delete check box is disabled and selected by default.
