Enter the search criteria:

 

  1. Doc Type – Select the Document Type from the list. The Document Types applicable for the Purchase Order are displayed in the list.

  2. Doc No – Enter the Document Number in numeric format of eight digits maximum length.

  3. Req By Store – Select the Request By Store from the list. The Stores, which have access to Request for Purchase Order, are displayed in the list.

  4. Request On Department – Select the Request On Department from the list. The Stores, which have access to Request for Purchase Order, are displayed in the list.

  5. Item Class – Select the Item Class from the list.

  6. Doc Date [From–To] – Enter the ‘From’ and ‘To’ Document Date in the format of dd/mm/yyyy or select it using the Calendar function.

  7. Request Status – Select the Request Status from the list. The value ‘Pending For Approval’ is displayed, by default.
