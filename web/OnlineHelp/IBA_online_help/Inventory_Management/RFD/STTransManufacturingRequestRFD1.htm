Enter the following details to search for raised manufacturing request:

  1. Req By Store – Select the Requested by Store from the list. The Stores for which you have access and ‘Manufacturing’ transaction function applicable are displayed.

  2. Prepare Location - Select the required Location from the list. The Locations created using the function ‘Prepare Location’ and status as enabled are displayed.

  3. Preparation Type – Select the required preparation type from the list of valid values. The list of valid values is All,  Extemporaneous, Extemporaneous – Ad Hoc, Standard Formulary – Extemp, Standard Formulary – Gale Internal, and Standard Formulary - Gale. External. By   default, the value ‘All’ is displayed.

  4. Preparation Mode – Select the required Preparation Mode from the list. The list of valid values includes ‘All’, ‘Sterile’, ‘Non-sterile’ and ‘Others’. The value ‘All’ is displayed, by default.
