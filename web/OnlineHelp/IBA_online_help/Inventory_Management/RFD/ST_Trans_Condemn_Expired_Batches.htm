Enter the following details search for the required expired batch:

  1. Condemn Type – Select the Condemn type. The available values are Expiry and Non-Expiry. You can condemn multiple batches of either expiry items or non-expiry items at once. The default value is ‘Expiry Items’.

  2. Cut Off Date [From-To] – Based on the value selected in Condemn Type drop-down list, the system changes the label of the Cut Off Date field. When you select ‘Expiry Items’ in Condemn Type, the Cut-off Date for Expiry field is displayed and you can search batches with the expiry dates within a specified date range. If you select Non-expiry Items in Condemn Type, the ‘Cut-off Date for Receipt’field is displayed and the system searches for condemned items based on GRN Date range instead of Expiry date range.  Enter the date range for which the expired batches are to be searched in dd/mm/yyyy format or select the date using the Calendar function.

  3. Store Code [From-To] – Enter the Store codes range for which the expired batches are to be searched in alphanumeric format or select the Store using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  4. Item Code [From-To] - Enter the Item Codes range for which the expired batches are to be searched in alphanumeric format or select the Item Code using [?] function. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  5. Batch ID – Enter the Batch ID for the condemned batch in alphanumeric format of 10 characters maximum length.
