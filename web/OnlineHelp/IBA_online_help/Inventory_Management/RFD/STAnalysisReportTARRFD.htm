The following report criteria can be specified to generate the Request Statistics Report:

 

  1. Item Class [From-To] - Enter the range of Item Class for which the report is to be generated in alphanumeric format of four characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Class code and select required values.

  2. Req On Store Code [From-To] - Enter the range of Store codes for which the report is to be generated in alphanumeric format of Six characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected store code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ store code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the store code and select required values. On selecting the Report Type as ‘Item Consolidated’ this field is disabled.

  3. Request Date [From-To] – Enter the request date in format dd/mm/yyyy format. Entered ‘To’ value should be greater than or equal to the ‘From’ value.

  4. Request By Store Code – Select the required request by store from the list.
