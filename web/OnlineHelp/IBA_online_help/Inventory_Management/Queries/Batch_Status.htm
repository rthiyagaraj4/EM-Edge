Batch Status

From this function you can Query the Batch details of Items in a Store.

Batch Status is maintained when the Opening Stock for an item is generated. You can specify range of items and stores for which the details are to be viewed. All the details of an Item like Expiry/Receipt date, Batch ID, Trade Name, Manufacturer, Bin Location Code, Quantity on Hand and Available Quantity can be viewed.

Querying the Batch Details

From the ST Queries, select the Batch Status function.

The Batch Status screen gets displayed.

  1. Click [Search] to display the list of all the batches or enter the search criteria and click [Search] to select a specific Batch.

    Enter the following search criteria:

     

    1. Item [From-To] - Enter the range of Item for which the batch details are to be searched in alphanumeric format or select the required Item using [?] function. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected item code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

    2. Store Code [From-To] – Enter the range of Store Codes for which the batches details are to be searched or select the Store Code using the [?] function. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

    3. Manufacturer – Select the required Manufacturer from the list.

    4. Batch Expires Next (Days) – Enter the days in numeric format of three digits maximum length to include the details of batches which are due for expiry in the search result.

    5. Trade Name – Select the required Trade Name to be associated from the list.

    6. Batch ID – Enter the required Batch ID if batch status for a specific batch has to be viewed in alphanumeric format.

    7. Include Expired Batches – Select the check box to include expired batches in the search result.

    8. Include Zero Stock Batches – Select the check box to include batches having zero stock in the search result.

    9. Include Suspended Batches – Select the check box to include suspended batches in the search result.

  2. On searching, the Batch Details of Item are displayed based on the entered criteria.
