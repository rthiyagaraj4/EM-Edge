Sales/Return History

From this function you can Query the Sales and Sales Return Transaction History performed with respect to the Items sold out to a patient.  

This function allows to view both Sales and Sales Return Details. The Sales Types are ‘Inpatients’, ‘Outpatients’ ‘Referrals’ and ‘Discharge', 'Direct Sales', 'Emergency', 'Day Care' and 'All'.

Viewing the Sales/Sales Return Transaction History

From the ST Queries, select the Sales/Return History function.

The Sales/Return History screen gets displayed.

  1. Click [Search] to view all the Sales Transaction/Return Transaction History or enter the search criteria and click [Search] to display the required Sales/Return Transaction History.

    Enter the following search criteria:

    1. Sales, Sales Return and Both - Select the required Sales, Sales Return or Both ption to generate the query. By default, the value Sales is selected.

    2. Doc Type - Select the Doc Type from the drop-down list.

    • If Sales option is selected, the system lists Sales value in the Doc Type drop-down list.

    • If Sales Return option is selected, the system lists Sales Return value in the Doc Type drop-down list.

    • If Both option is selected, the system lists both the Sales and Sales Return value in the Doc Type drop-down list.

    1. Document Number - Enter the Document Number in numeric format. The details of the Sales History or Sales Return details associated to the respective Document Number are displayed.

    2. Sales Type - Select the required value for Sale type from the list of valid values. The lists of valid values are 'Inpatients', 'Outpatients', 'Discharges', 'Referrals', 'Direct Sales', 'Emergency', 'Day Care'  and ' 'All'. By default, the value 'All' is displayed.

    3. Store Code - Enter the store code required to be viewed in alphanumeric format of 8 characters maximum length. Click [?] function to query the store code and select required values.

    4. Item Code - Enter the required item in alphanumeric format of 21 characters maximum length or select the item using [?] function.

    5. Date [From-To] - Enter the 'From' and 'To' date for which the query to be done.

    6. From Date - Enter the 'From' and 'To' date for which the query to be done

    • The difference between From and To dates cannot be greater than 365 Days.

    • From Date cannot be greater than To date

    • By default, the current date is displayed as To date.

    1. Patient ID - Enter the Patient ID or search using the search patient lookup.

    2. Patient Name - The patient name for the Patient ID is displayed.

    3. Nationality- The Nationality for the selected Patient ID is displayed.

    4.  Encounter ID - Enter the required Encounter ID.

  2. On searching, the Sales Transaction, Return Transaction History or both is displayed based on the entered criteria.
