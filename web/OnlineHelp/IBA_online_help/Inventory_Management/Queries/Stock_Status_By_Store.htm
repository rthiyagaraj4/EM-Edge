Stock Status By Store

From this function you can Generate stock status of drugs maintained in a Store and provides information on ATC classification, QoH, Unit cost and Stock level.

This query can be further tailored to list only drugs with zero stock or items below reorder level. The status information displayed is as on date.

Generating stock status of drugs maintained in a Store

From the ST Queries, select the Stock Status by Store function.

The Stock Status by Store screen gets displayed.

  1. Enter the search criteria and click [Show Items] to view the required Item details.

    Enter the following search criteria:

    1. From Store Code – From Store Code for which the Items are to be searched in alphanumeric format of 15 characters maximum length or select the required Store using [?] function. If the  'Restrict Viewing Stock Status' privileges are set to 'User access to stores', only the Stores to which logged in user has access are displayed in Store Code [From - To] fields. This is a mandatory field.

    2. To Store Code – To Store Code for which the Items are to be searched in alphanumeric format of 15 characters maximum length or select the required Store using [?] function. If the  'Restrict Viewing Stock Status' privileges are set to 'User access to stores', only the Stores to which logged in user has access are displayed in Store Code [From - To] fields. This is a mandatory field.

    3. Stock Level – Select the required stock level from the list. The listed values are ‘All’, ‘Stock On Hand’, ‘Zero Stock’ and ‘Below reorder Level’. The value ‘All’ is displayed, by default.

    4. Item – Enter the required Item in alphanumeric format of 60 characters maximum length or select the Item using [?] function.

    5. Stock Item – Select the required value for stock item from the list. The listed values are ‘Both’, ‘Yes’ and ‘No’. The value ‘Both’ is displayed, by default.

    6. Item Class [From–To] – Enter the required Item Class range in alphanumeric format of 15 characters maximum length or select the Item Class using [?] function.

    7. Item Analysis Code 1-3 [From–To] - Select the required value for ‘Item Analysis 1’, ‘Item Analysis 2’ and ‘Item Analysis 3’ from the list. The Item Analysis codes are defined in the Item Analysis function.

  2. Click ‘Item Code’ hyperlink to invoke Batch Search screen and view the batch details of the item in a store.

  3. Click [Reset] to clear the entered data.
