Enter the following:

 

  1. User – Enter the User in alphanumeric format of 30 characters maximum length or select the User using the [?] function.

  2. Employee No - Enter the Employee Number in alphanumeric format of six characters maximum length.

  3. Enabled - Select the check box to enable the record or clear to disable the record. The check box is selected, by default.

 

Rights for DR Processes

  1. Search - The right to search for patient records is defined. Select one of the values as ‘None’, ‘Logged in Facility’ or ‘All Facilities’ from the list displayed. The value ‘None’ is displayed, by default.

  2. Identify Main/Duplicates - The right to identify Main/Duplicate patients is defined. Select one of the values as ‘None’, ‘Logged in Facility’ or ‘All Facilities’ from the list displayed. The value ‘None’ is displayed, by default.

  3. Merge - The right to merge patient records is defined. Select one of the values as ‘None’, ‘Logged in Facility’ or ‘All Facilities’ from the list displayed. The value ‘None’ is displayed by default.

  4. Administer Search Process - The right to administer search process is defined. Select one of the values as ‘None’, ‘Logged in Facility’ or ‘All Facilities’ from the list displayed. The value ‘None’ is displayed, by default.
