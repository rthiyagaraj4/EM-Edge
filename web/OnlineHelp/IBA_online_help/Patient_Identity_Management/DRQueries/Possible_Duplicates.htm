Possible Duplicates

From this function you can Query and view the possible Duplicate Patient records identified by the 'Search Duplicates' function.

The Possible Duplicates query function is used to select the required Patient ID and view the Duplicate details. You also have an option to view the search criteria specified for the search process.

Querying for the Possible Duplicates

From DR Queries, select the Possible Duplicates function.

The Possible Duplicates screen gets displayed.

  1. The Possible Duplicate screen gets displayed with the Search Duplicate Details.

  2. Click the ‘Sug. Main Patient ID’ hyperlink. The details of the Main and Duplicate patients of the selected Duplicate Group are displayed in the Group Details.

You can use the Criteria to search for patients under the duplicate groups as an alternate method of searching. If the number of duplicate groups is more, you can narrow down the patient search by specifying the Patient ID as search criteria.

  1. Click '+' hyperlink to view the patient details. The Patient details are displayed under six tabs namely, Demographic Details, Contact Addresses, Related Contacts, Documents & Others, Who Done It and Financial Details.

  2. Enter the valid Patient ID in alphanumeric format of length as specified in Master Patient Index module or click [?] to select the required Patient ID.

  3. On selecting or entering a valid Patient ID, click [Search]. For the selected patient, if the duplicate patient records exist in the search result, the duplicate records details are displayed, or the system displays an appropriate error message. Click [Reset] to clear the data entered.

  4. Click [Search Criteria] to view the search criteria specified for the search process.

  5. Click [Close] to close the Search Criteria window and control returns to the Possible Duplicates screen.

  6. Click [Duplicates Details] to view the additional details of the main and possible duplicate records.

  7. A comparison table is displayed with the patients' details. Click [Close] to close the 'Possible Duplicates Details' screen and the control returns to the main screen.
