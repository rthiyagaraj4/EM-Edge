Enter Duplicate Patients

From this function you can manually select the patients and Identify their record as Main or Duplicate and Query for the duplicate records manually to Delete or Modify the record.

You cannot select the patients fetched by the duplicate search process in this function. If the selected patient record is available in the search result, the system displays an error message. On identifying these records as Main/Duplicate, the identified records are available for merge process.

The access to this function is determined by the User Access Rights set for the logged-in user. The user is denied access to the function if rights are not defined for the logged in user or the right for the DR process ‘Identify Main/Duplicates’ is set as ‘None’. If the privilege level is set as Logged-in facility, the User can perform the DR Process ‘Enter Duplicate patients’ only for the Logged-in facility and if the privilege level is set as All Facilities, the User is allowed to access the function and can perform this process for all facilities.

Entering Duplicate Patients

From the DR Processes, select the Enter Duplicate Patients function.

The Enter Duplicate Patients screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Add] to create and add a record to the Group Details frame or [Cancel] to cancel the entered data. The details of the added records are displayed in the Group Details frame.

  4. Click ‘+’ hyperlink to view the patient details.

  5. Click ‘Patient ID’ hyperlink to modify the entered details. Click [Modify] to modify the changes or [Cancel] to rest the changes.

  6. Click [Apply] to save the created record(s) or [Reset] to clear the entered data. Enter minimum two records to save the transactions and one Main and one Duplicate patient should be identified. If the Patient IDs are already marked as duplicates, the message ‘Patient IDs are already present as duplicates’ is displayed.

  7. Click [Query] to invoke the Query screen.

  8. Enter the Patient ID or select the desired Patient ID using the [?] function.

  9. Click [Search] to view the main patient details in the left hand side frame.

  10. Click Patient ID hyperlink to view the duplicate details in the Group Details frame. Click ‘+’ hyperlink to view the patient details, Patient ID hyperlink to modify the details. Click [Modify] to save the changes or [Cancel] to reset the changes.

  11. Select ‘Delete’ check box against the desired patient to delete the record. Click [Delete] to delete the selected records or [Reset] to clear the entered data.
