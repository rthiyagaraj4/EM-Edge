Default Search Criteria

From this function you can Define the default search criteria for Patient Name, Demographic Data and Scope of Search to search for duplicate records.

The search criteria defined are used for searching duplicate records in the Master Patient Index module. You have to specify at least one criterion for the Patient Name. The scope of the search is either logged-in facility or all facilities.

Defining Default Search Criteria

From the DR Processes, select the Default Search Criteria function.

The Default Search Criteria screen gets displayed.

  1. Select the desired search criteria from the Patient Name, Demographics Data and Scope of Search frames. <Related Field Description>

  2. Click [Apply] to save the defined Default Search Criteria or [Reset] to clear the selected data.
