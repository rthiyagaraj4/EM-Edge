User Access for Materials Group

From this function you can Define Material Group access to the Users to process the Transactions.

The Material Groups defined in the Material Group function are displayed. These Material Groups are applicable only if access is provided for the User.

Defining Material Group Access to Users

From the Materials Setup, select the User Access for Material Group function.

The User Access for Material Group screen gets displayed.

  1. Enter the User ID in alphanumeric format of 60 characters maximum length or select the User ID using [?] function. This is a mandatory field.

  2. Click [Search] to display the enabled Material Groups defined in Material Group function.

  3. Select the check box against the desired Material Group to provide Material Group access for the selected User.

  4. Click [Apply] to save the defined Material Group Access or [Reset] to clear the entered data.
