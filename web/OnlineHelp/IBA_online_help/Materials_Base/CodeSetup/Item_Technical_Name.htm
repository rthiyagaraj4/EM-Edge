Item Technical Name

From this function you can Define names of Inventory and Non-inventory Items used across different sections of the Hospital/Enterprise for various processes, namely Purchasing and Pharmacy Stock.

The Inventory Items are Stock Items for which the stock positions are maintained and managed in Inventory Management module. The Non-inventory Items are direct Items, which is procured on request and given to the end users directly. The Item attributes pertaining to Inventory Management will be defined as a part of Inventory Management module. The attributes of the Items like Item Description, Item Class and Unit of Measure, if the Items are Inventory Management Items and/or CSSD Components, etc. are defined.

Defining Item Technical Name

From the Materials Setup, select the Item Technical Name function.

The Item Technical Name screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the defined Item Technical Name or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Item Technical Names.
