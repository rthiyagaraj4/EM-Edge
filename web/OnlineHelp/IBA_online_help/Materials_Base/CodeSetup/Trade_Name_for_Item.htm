Trade Name for Item

From this function you can Associate Trade Names to one or more Items and Query for the desired association of Trade Name for Item records.

You can associate various Trade Names to an Item. If the Item is a Sole-source Item, you cannot associate the Trade Name for this Item. The Trade Names are defined in the Application Masters module of Enterprise Management.

Associating Trade Names to Item

From the Materials Setup, select the Trade Name for Item function.

The Trade Name for Item screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the desired Item name or select the Item using [?] function. All the Item values created, which are not the Sole source type are displayed. This is a mandatory field.

  3. On selecting the Item, the associated Trade Name details get displayed.

  4. Enter the desired Trade Name to associate to the Item or select the Trade Name using [?] function. This is a mandatory field.

  5. Click [Add] to add the selected Trade Name or [Cancel] to cancel the selection.

  6. On adding, select the check box against the Trade Name.

  7. Click [Apply] to save the Trade Name to Item association or [Reset] to clear the entered data.

  8. Click [Query] to search for existing association of Trade Name to Item records.
