Enter the following details to generate the Master Codes reports:

 

  1. Code - Select the required master code for which a code list is to be generated from list of MM Master Codes and a report criteria screen is displayed.

 

Enter the following report criteria to generate a report for the selected code:

 

  1. Item Class [From-To] - Enter the range of Item Class codes for which the report is to be generated in alphanumeric format of four characters maximum length. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the From selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the To Item Class code. When both values are entered, ‘To’ value has to be greater than or equal to the From value. Click [?] function to query the Item Class code and select the required values.

  2. Nature - Select the report nature as, ‘Enabled’, ‘Disabled’ or ‘Both’. The Item records with the matching status are included in the report. The value ‘Both’ is displayed, by default.

  3. Order By - Select the Sort Order as ‘Item Class Code’ or ‘Item Class Description’ to generate the report in the required sorting order. The value ‘Item Class Code’ is displayed, by default.
