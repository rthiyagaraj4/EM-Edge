Enter the following details to create MM Parameter record:

 

  1. Audit Trail - Click [Audit Trail] to view the Audit Trail details. The Audit Trail screen is displayed. The modified details are displayed. The displayed details are: Modified By, Modified Date/Time, Facility Name, Allow Bonus Entry, Allow Sample Entry.

  2. Default Batch ID if Batch ID is Not Applicable – Enter the default Batch ID for Items in alphanumeric format of nine characters maximum length, if the Batch ID is not applicable. This is a mandatory field.

  3. Default Trade ID if Trade ID is Not applicable – Enter the default Trade ID for Items for which Trade ID is not applicable in alphanumeric format of 20 characters maximum length. This is a mandatory field.

  4. Allow Bonus Entry – Select the check box to allow Bonus Entry while recording Good Received Note.

If Allow Bonus Entry is selected in MM Parameter, you can enter Bonus Entry while recording the details of received Items in Goods Received Note function.

  1. Allow Sample Entry – Select the check box to allow Sample Entry while recording Good Received Note.

If Allow Sample Entry is selected in MM Parameter, you can enter Sample Entry while recording the details of received items in Goods Received Note function.

  1. Decentralized Inventory – This option is disabled as Enterprise Management is using centralized Inventory.

 

Period End Scope

  1. PO Number Mandatory in GRN – Select this checkbox to make the PO Number recording as mandatory in Goods Received Note screen. This check box is enabled only if ‘Interface to External PO’ checkbox is not selected in MM Entity Parameter.

  2. Purchase Control – This field is enabled if Purchase Control module of Enterprise Management is installed or 'Not applicable' option is made default and disabled. If Purchase Control module is installed, you can define the period end scope for Purchase control at the accounting entity level or facility level by selecting appropriate values from the list.

  3. Sterile Stock – This field is enabled if Sterile Stock module of Enterprise Management is installed or 'Not applicable' option is made default and disabled. If Sterile Stock module is installed, you can define the period end scope for Sterile Stock at the Inventory Management entity level or facility level by selecting appropriate values from the list.

  4. Inventory Management - This field is enabled if Inventory Management module of Enterprise Management is installed or 'Not applicable' option is made default and disabled. If Inventory Management module is installed, you can define the period end scope for Inventory Management at the entity level or facility level by selecting appropriate values from the list.
