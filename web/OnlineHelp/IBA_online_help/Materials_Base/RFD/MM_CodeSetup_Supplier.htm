Enter the following details to create Supplier record:

 

  1. Supplier Code - Enter the code of the supplier in alphanumeric format of eight characters maximum length. Once the code is defined and saved, it cannot be changed in query function.  This is a mandatory field.

  2. Alpha Code – Enter the Alpha code for the supplier in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  3. Long Description – Enter the Long Description of the Supplier in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  4. Short Description – Enter the Short Description of the Supplier in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  5. Supplier Address Line1. Address Line 2 and Address Line 3 – Enter the Supplier Address in alphanumeric format of 30 characters maximum length in each line field.

  6. Zip Code – Enter the Zip Code in alphanumeric format of 30 characters maximum length.

  7. Telephone No – Enter the Telephone Number in alphanumeric format of 30 characters maximum length.

  8. Fax No – Enter the Fax Number in alphanumeric format of 30 characters maximum length.

  9. E-Mail ID - Enter the E-mail ID in alphanumeric format of 30 characters maximum length. The system will validate the e-mail address. If an invalid email address is entered, an alert is displayed to enter the valid e-mail address.

  10. Bank No – Enter the Bank No in alphanumeric format of three characters maximum length.

  11. Account No – Enter the supplier Account Number in alphanumeric format of 16 characters maximum length.

  12. Bank Account Name – Enter the Bank Account Name in alphanumeric format of 20 characters maximum length.

  13. Branch No – Enter the Branch Number in alphanumeric format of three characters maximum length.

  14. Currency – Select the Currency from the drop-down list. The foreign currency value is defined in Financial Setup and the local currency value is defined in Accounting Parameters in the System Manager module.

  15. Contact Person For Purchase – Enter the Contact Person Name in alphanumeric format of 40 characters maximum length.

  16. Title of the Contact Person – Enter the Title of the Contact Person in alphanumeric format of 40 characters maximum length.

  17. Enabled - Select the Enabled check box, to enable the defined Supplier record.  This check box is selected by default. Clear the check box to disable the defined supplier record.
