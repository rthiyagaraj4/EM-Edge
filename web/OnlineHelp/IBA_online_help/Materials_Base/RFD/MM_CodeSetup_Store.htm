Enter the following details to create Store records:

 

  1. Store Code – Enter the code of the Store in alphanumeric format of six characters maximum length. The code defined indicates the Store. Once the code is defined and saved, it cannot be changed in query function. This is a mandatory field.

  2. Long Description - Enter the long description of the Store in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description of the Store in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. External – Select the check box, if the defined Store is an external store.

  5. Customer – This field appears only if the External check box is selected. Select the required customer from the drop-down list. This field is mandatory.

  6. Department – Select the required department from the list to associate to the Store. All the enabled Departments as set in ‘Department in Facility’ of the Application Masters module are displayed in the list. This is a mandatory field.

  7. Store Group – Select the store group from the drop-down list.

  8. Remarks –  Enter the remarks if any. You can enter maximum of 300 characters.

  9. Drug License No – Enter the drug license no in alphanumeric format of maximum 30 characters.

  10. Enabled – Select the check box to enable the defined Store. Clear the check box to disable the defined Store. The check box is selected, by default. The enabled Stores are available for selection in the required functions.

 

Store Incharge Details

  1. Incharge Name – Enter the name of the person in-charge of the Store in alphanumeric format of 30 characters maximum length.

  2. Incharge Title – Enter the Title/Designation of the person in-charge of the Store in alphanumeric format of 30 characters maximum length.

  3. Location – Enter the Location details of the in-charge in alphanumeric format of 30 characters maximum length.

  4. Contact Reference – Enter the contact reference of the in-charge in alphanumeric format of 40 characters maximum length.
