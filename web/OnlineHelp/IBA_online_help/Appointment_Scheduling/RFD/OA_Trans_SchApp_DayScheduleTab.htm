Day Schedule Tab

The Day Schedule Tab is enabled only from the Schedule Status Tab. This tab displays the schedules for a day based on the selected Location/Resource/Day combination. You can view either one, two or maximum of three combinations based on the selection.

  1. Select a required Schedule Date greater than the current date and click [Go] to view the Day Schedule or book appointments for that date.

    1. Schedule Date - The Appointment Schedule for the next or previous days can be viewed by entering the required date in dd/mm/yyyy format in the Schedule Date field or else click calendar function to select the required date and subsequently clicking on the [Go].

    2. Location – This field displays the selected Location Type and Location.

    3. Resource – This field displays the Resource Class and Resource.

     

    Maximum

    1. Maximum Slots – This field displays the Maximum number of Patients or slots allowed for booking.

    2. Maximum First Visit – This field displays the Maximum number of First Visits allowed.

    3. Maximum Other Visit – This field displays the Maximum number of Other Visits allowed

    4. Maximum Overbooking – This field displays the Maximum number of over bookings allowed.

    5. Maximum Global – This field displays the maximum number of global bookings allowed for the Location/Practitioner.

    6. Maximum Forced – This field displays the maximum number of forced bookings allowed for the Location/Practitioner.

     

    Booked

    1. Booked Patients – This field displays the total number of appointments already booked.

    2. Booked First Visit – This field displays the total number of appointments booked against First Visits.

    3. Booked Other Visit – This field displays the total number of appointments booked against Other Visits.

    4. Booked Overbooking – This field displays the total number of appointments booked against Over Booking.

    5. Booked Global – This field displays the number of patients booked as global.

    6. Booked Forced – This field displays the number of forced appointments booked for the selected Location/Resource.

     

    Count

    1. Normal Reschedule Count – The Count increases by one if one appointment is transferred using the Reschedule/Cancel Appointment Function.

    2. Bulk Reschedule Count - The count increases by one, if appointments are transferred from one practitioner to another Practitioner through Bulk Transfer function.

     

    Appointment Details

    1. From Time – This field displays the commencement time of an appointment based on the appointment type set up in timetable for scheduling.

    2. To Time – This field displays the end time of an appointment based on the appointment type set up in timetable.

    3. IP/DC – This field displays an icon if the patient is an inpatient.

    4. Patient Details – This field displays the details of the patient like Patient ID (Incase of registered patient), Name of the Patient, Gender and Age. This field is blank if no appointments exist.

    Note: An icon is displayed in this column in both single and multiple views, if the patient is of ‘VIP’ category.

    1. Visit Type – This field displays the applicable visit type recorded while giving the appointment. The Visit Type is highlighted based on the colour indicator setup done in the Outpatient Management module.

    2. Visit Indicator – This field displays the visit indicator recorded while giving the appointment.  The Visit Indicator setup is done in the Outpatient Management module.  

    3. Reason For Contact – This field displays the reason for the patient seeking an appointment.

    4. Contact Details – This field displays the contact number of the patient captured while booking an appointment.

    5. Secondary Resources – This field displays the Secondary Resource selected for an appointment with a hyperlink or this column is blank.

  2. Click the required ‘From Time’ hyperlink to book an appointment for a patient for the selected time slot. The time slot selected cannot be less than the system time. Book appointment screen gets displayed to Book an Appointment for the selected Schedule Date.

  3. Patient details are displayed against the selected time slot/slab in the Day Schedule Tab. Click the From Time hyperlink to book an appointment for the patient. Once the appointment is booked, you can click Select Order hyperlink to place an order from the order catalog.

  4. If the timetable generated for the schedule permits Forced Booking, select the required time slot/slab, which is already booked to perform Forced Booking.

    The system allows forced in bookings against a particular Location/Resource, thus accommodating special cases within a defined appointment schedule for a day. Forced Appointments for a particular Location/Resource can be performed only if the defined time table is of type ‘Slot’ or ‘Slab’ and value for ‘No of Forced in Booking’ is set while defining the timetable. Forced Bookings cannot be performed for a future date or the system prompts you with a message – Forced Booking possible for future date. You can perform ‘Forced Appointment’ by clicking on any of the already booked time.

  5. If the timetable generated for the schedule permits Over Booking, select the Overbooking hyperlink (which gets displayed when all the slots/slabs are booked) to perform Over Booking. If a location of an Operator Station is defined for Global booking in 'Location for Operator Station' in Application Masters module, users associated to that Operator Station are not allowed to do Overbooking.

    The system allows overbooking against a particular Location/Resource, thus allowing certain amount of flexibility in accommodating additional flow of patients within a defined appointment schedule for a day. Overbooking for a particular Location/Resource is performed only if value for ‘Maximum Over Bookings’ is set while defining the timetable.

  6. Click Details hyperlink under ‘Selected Resource’ to view the Secondary Resource associated to the patient. Click [Close] to exit from the screen.

    The system allows forced in bookings against a particular Location/Resource, thus accommodating special cases within a defined appointment schedule for a day. Forced Appointments for a particular Location/Resource can be performed only if the defined time table is of type ‘Slot’ or ‘Slab’ and value for ‘No of Forced in Booking’ is set while defining the timetable. Forced Bookings cannot be performed for a future date or the system prompts you with a message – Forced Booking possible for future date. You can perform ‘Forced Appointment’ by clicking on any of the already booked time.

    The Secondary Resources page displays the following details:

    Details

    The header displays the Patient details like Name, Gender, Age, Patient ID, Location/Resource, Appointment Ref No, and Appointment Date.

    1. Resource Class – The resource class of the secondary resource is displayed.

    2. Resource Type – The resource type of the secondary resource is displayed.

    3. Resource Name – The resource name of the secondary resource is displayed.

    4. From Time – The time from when the secondary resource is required for the appointment is displayed.

    5. To Time – The time until which the secondary resource is required for the appointment is displayed.

  7. Click ‘From Time’ hyperlink of an already booked appointment from the day schedule screen to view the appointment details of an already booked appointment.

  8. Appointment records with appointment time greater than the current system date or time permits Reschedule Appointments and Cancel Appointments but if appointment time is less than the current system time, only the details of the appointment are viewed. Click [View Audit Trail] to view the appointment and modified details.  Click [Patient Instructions] to view the patient and procedure instructions associated to the patient while scheduling an appointment. Click [Close] to exit from the screen.

 

Booking Group Appointments

Group appointments can be created only for slabs. When you click a slab, the following message is displayed:

“Do you want to book group appointment?” On clicking OK, the Appointment Group web page dialog gets displayed:

1. Appt Date/Time – The selected appointment date and time are displayed.

2. Location – The location is displayed automatically.

3. Service – Select the required service from the drop-down list. The services associated with the location are available for selection.

4. Visit Type – Select a visit type from the drop-down list. The visit types associated with the clinic will be available for selection. This is a mandatory field.

5. Resource – Displays the name of the practitioner whose schedule slab you have selected.

6. Priority – Select the appointment priority from the drop-down list. You can select “Urgent”, “Semi-urgent”, or “Normal”.

7. Group Name – Click [?] and select an appointment group from the Appointment Group webpage dialog. This is a mandatory field.

8. Reason for Contact - Click [?] and select a reason from the Reason webpage dialog. This is a mandatory field.

9. Receiving Date / Time – Select the date on which you received the request for appointment from the patient, using the calendar function.

10. Mode – Select the means through which you received the request for appointment from the drop-down list. The available options are: “Email”, “Telephone”, “Fax”, “In Person”, “Web”, and “Other”.

11. Remarks – Enter any remarks about the appointment.
