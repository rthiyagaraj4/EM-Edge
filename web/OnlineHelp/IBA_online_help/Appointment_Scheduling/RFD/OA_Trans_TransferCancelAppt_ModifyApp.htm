Modify Appointment

Appointments are modified from this function on patient’s request or for any other reasons.

The Book Appointment screen gets displayed. You can Modify the appointment details of the  patient and click [Confirm Booking] to save the changes.

 

The Referral ID field is enabled if Patient ID exist. If appointment has to be modified for the referred patients, Referral ID must be entered as generated during referral registration or click [?] function to perform Referral Patient Search to locate the referral patient details. The appointment is modified before registering a visit or prior to lapse of appointment time. A new referral can be registered and associated to the appointment. If Patient ID exists for an appointment, referrals related to the Patient ID are listed in the Referral list during Referral Patient search. You can associate the existing Referral to the appointment.
