Secondary Resources Tab

This tab is used to associate secondary resources against the booking. This tab is enabled based on the following conditions:

• The Resource has to belong to a Resource Type identified against the Visit Type indicator in OP module. This association is created using the function ‘Resource Type for Visit Type’ in Outpatient Management module. (Resource Type identified as primary resource in Location Setup master of Outpatient Management module is not displayed).

• The Resource has to be associated to the selected location using the function ‘Resource for Location’ in Outpatient Management module.

• Timetable and Schedule has to be generated for the Location/Resource/Day combination.

Secondary Bookings against the required resource is made by selecting the check box corresponding to the required time slot/slab provided.

Based on the ‘Resource Type for Visit Type’ set up (with ‘Required’ check box selected) in Outpatient Management module, Secondary Resources for a Visit Type has to be compulsorily identified or the system prompts you with a message on clicking [Confirm Booking]. Mandatory resources are marked with a * mark against it. You can select the required time slot/slab by selecting the check box and proceed with the booking.

 

If you select a secondary resource of type Practitioner, it will be automatically displayed in the Reassign Practitioner screen.
