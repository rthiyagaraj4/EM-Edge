Enter the following details to create a Appointment Group:

 

Note: All fields in this screen are mandatory except Enabled.

 

  1. Group ID – Enter a unique identifier for the appointment group that you want to create. You can enter a max of 5 alpha numeric characters. No Special characters will be allowed except * and _.

  2. Group Name – Enter the name of the appointment group. You can enter a max of 30 alphabetic characters.

  3. Location Type – Select the location type from the drop-down list. The available options are “Clinic”, ”Daycare Unit”, “Procedure Unit”.

  4. Location – Click [?] and select a location by searching from the Location webpage dialog.

  5. Min Patient – Enter the minimum number of patients to be included in the appointment group.

  6. Max Patient – Enter the minimum number of patients to be included in the appointment group.

  7. Enabled – Select this check box if you want the appointment group to be available for associating patients.
