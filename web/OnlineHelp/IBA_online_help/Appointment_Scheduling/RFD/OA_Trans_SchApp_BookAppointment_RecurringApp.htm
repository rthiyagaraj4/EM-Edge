Recurring Appointments

[Recurring Days] gets displayed only for bookings created against a Series Type of visit and is enabled only for registered patients having a Patient ID and for whom ‘First Visit’ has already been registered against the location or the system prompts a pop-up message – Patient has not visited the clinic, select the First Visit Type only.

Booking Recurring Appointments:

  1. Click [Recurring Days] to display recurring days screen with location details.

    1. Selected Location – The location type/location for which recurring appointment has to be made is displayed.

    2. Selected Resource - The resource class/resource for which recurring appointment has to be made is displayed.

    3. Visit Type – The visit type for which the recurring appointment has to be made is displayed.

    4. Selected Time Duration – The time duration for which recurring appointment has to be made is displayed.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Display Date] to view all the dates for which the schedule exist with details of recurring days.

    1. Date – This field displays the date for which schedule exists with a hyperlink.

    2. Day of Week – This field displays the day of week for which the schedule exists.

    3. Select – Select the check box to create a recurring appointment for the selected date or clear the check box.

    4. Time Duration – This field displays the Time Duration against which the recurring appointment has to be made.

  4. Select the check box to create a recurring appointment for the selected date and click [Accept Dates] to book an appointment against the dates specified.
