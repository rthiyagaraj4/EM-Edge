Block Appointment Slots/Slabs

From this function you can Block one or more appointment slots or slabs booked for a Location and Resource for a particular day by specifying a reason and Query for a desired blocked appointment slot or slabs. You can Delete a blocked appointment slot or slab record.

The ‘Block Appointments’ function is used to block appointments for a whole day. Blocking a slot or slab prevents booking an appointment against a patient. This function allows blocking of one or more slots or slabs only if the timetable set is based on the slots or slabs. Block Appointment Slots/Slabs function is used if a Location/Resource is not available for any reason for a specified period within a day. Blocking a slot or slab has to be justified by a reason and can be deleted if required. This function has a feature to block or cancel the appointment slots/slabs between a given date and time range. For example, if the appointment is only for half an hour but the clinic has scheduled slots for one hour, the system blocks the appointment slot for one hour instead of half an hour (actual appointment time). If the appointment slot to be blocked is for multiple date range the system does not consider the type of visit whether it is first visit, follow-up visit or some other visit.

Blocking Appointment Slots/Slabs

From the OA Schedules, select the Block Appointment Slot/Slabs function.

Block Appointment Slot/Slabs screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Select] to display the day schedule status for the selected date. Select the start time to block appointment slots by clicking on the ‘From Time’ hyperlink. All the From Time records are disabled. Select the end time by clicking on the ‘To Time’ hyperlink. The selected block time is displayed in the Block Time field.

    1. From Time – This field displays the commencement time of an appointment based on the appointment type set up in timetable for scheduling.

    2. To Time – This field displays the end time of an appointment based on the appointment type setup in timetable.

    3. Patient Details – This field displays the details of the patient like Patient ID (Incase of registered patient), name of the patient, gender and age. This field is blank if no appointments exist.

    4. Age – This field displays the Age of the patient.

    5. Gender – This field displays Gender of the patient.

    6. Visit Type – This field displays the applicable Visit Type recorded while giving the appointment.

    7. Reason For Contact – This field displays the reason for the patient seeking an appointment.

    8. Contact – This field displays the Contact details of the patient captured while booking an appointment.

  4. Click [Reset Block Time] to clear the block time selected if required.

  5. Click [Apply] to Block Appointment Slots.

  6. Click [Query] to search for blocked appointment slots.

  7. Click [Delete] to delete a queried record.

 

Day Schedule Status is displayed only if the schedule is generated for the specified day.

It is not possible to block time slots for the time range where appointment is booked.
