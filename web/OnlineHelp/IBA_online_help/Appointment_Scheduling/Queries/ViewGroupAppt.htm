Group Appointment Details

From this function you can Query all the details of group appointments booked within a facility.

The function ‘Group Appointment Details’ is used to query the details of group appointments within a facility. Group appointments come in handy for activities such as counseling, rehabilitation, medical check-ups, ante natal and post natal education. This will enable you to keep track of patient attending Group Counseling, ensure that they are scheduled as a Group and not missed out.

Querying to view the Wait List Audit Details

From the OA Queries, select the Group Appointment Details function.

The Group Appointment Details screen gets displayed.

  1. Select the location type from the drop-down list.

  2. Click [?] and select a location from the Location webpage dialog.

  3. Click [?] and select an appointment group from the appointment group webpage dialog.

  4. Select the appointment from and to dates using the Calendar function.

  5. Click [Search] to display the results in a grid. The grid displays the following details:

- Date

- Resource

- From time

- To time

- Appointment Group

- Print All

- Overlap Details

- No. of patients with appointments

If the number of patients in the group appointment are less than the minimum limit, due to cancellation etc, this cell will be displayed in pink color.

 

In the Group Appointment Details results grid, click the Appointment Group name hyperlink to view the group appointment details in a pop-up. The following details are displayed:

• Group name

• Clinic

• Appt Date

• Grid of appointments with the columns: Appt No, Patient ID, Patient Name, Appointment Slip

In the results grid, click the Appointment Slip to print the appointment slips for the selected group appointment.
