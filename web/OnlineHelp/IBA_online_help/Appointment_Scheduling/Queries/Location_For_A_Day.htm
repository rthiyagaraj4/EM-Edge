Locations for a Day

From this function you can Query all the locations that are operational on a specified day of the week.

Locations are either operational across all days of the week or on certain days of the week. This function generates a list of all locations that are operational on a specified day of the week. This function is used at enquiry desks, where enquiries on location availability is handled. This is also used by healthcare managers on a daily basis for allocation of resources to various locations depending on the day of the week.

Querying Locations for a Day

From the OA Queries, select the Locations for a Day function.

The Locations for a Day screen gets displayed.

  1. Select a Location Type from the list. Click [?] function to search for the corresponding Location. This field is mandatory.

  2. Select the required Resource Type from the list and the Resource text box is enabled. You can enter the required resource name in alphanumeric format or click [?] function to search for the required resource.

  3. Select the Day of the week from the valid list of days displayed.

  4. Click [Search] to select a specific record or [Clear] to clear the records displayed in the result area.

  5. Click [Reset] to clear the entered data.
