Bulk Cancel Appointments

From this function you can Cancel appointments in bulk for the selected criteria.

Bulk Cancel function allows you to cancel appointments in bulk by specifying a reason for cancellation.

 

Process Flow Diagram

Cancelling Appointments

From the OA Transactions, select the Bulk Cancel Appointments.

Bulk Cancel Appointments screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Select] to display details of transferred appointments to be cancelled or [Clear] to clear the displayed result.

    1. Appt No – This field displays the Appointment Number of the patient.

    2. Appt Date – This field displays the Appointment date of an appointment based on the Appointment Type set up in timetable.

    3. Secondary Resources – This field displays the secondary resources selected for an Appointment with a hyperlink or this column is blank.

    4. Patient ID – This field displays the Patient ID of the patient.

    5. Patient Details – This field displays the details of the patient like Patient ID (In case of a registered patient), Name of the Patient, Gender and Age. This field is blank if no appointments exist.

    6. From Time - This field displays the start time of an Appointment based on the appointment type set up in timetable for scheduling.

    7. To Time - This field displays the end time of an Appointment based on the Appointment Type set up in timetable.

    8. Duration – This field displays the time allotted to the patient for consultation.

    9. Visit Type – This field displays the visit type selected for the Appointment. The Visit Type is highlighted based on the colour indicator setup done in the Outpatient Management module.

    10. Location Type – This field displays the Location Type selected for the Appointment.

    11. Location – This field displays the Location selected for the Appointment.

    12. Resource Type – This field displays the Resource Type selected for the Appointment.

    13. Resource Name – This field displays the resource name selected for the Appointment.

    14. Referral ID – This field displays the Referral ID if the patient is a referral patient.

    15. Order ID – This field displays the order ID of the patient.

  3. Click ‘Details’ hyperlink to view the Secondary Resources.

  4. Select the check box to cancel appointments and [Apply] to cancel the selected appointment.

 

Appointments booked from Referral ID or Order ID cannot be cancelled. Such appointment check boxes are disabled by default.
