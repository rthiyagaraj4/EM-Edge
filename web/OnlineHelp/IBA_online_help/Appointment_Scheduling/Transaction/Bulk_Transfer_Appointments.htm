Bulk Reschedule Appointments

From this function you can Reschedule Appointments including Force Bookings, Over Bookings and Global Bookings in bulk across specialties.

Bulk Reschedule Appointments function is used for rescheduling appointments, over bookings and global bookings in bulk across the Specialties. For schedules based on ‘Slot’ and ‘Slab’ timetable types, bulk reschedule of appointments is possible, but transfer is not possible for ‘Free Format’ timetable type. Appointments booked with secondary resources are also transferred using Bulk Reschedule Appointments function.

 

Process Flow Diagram

Rescheduling Appointments

From the OA Transactions, select the Bulk Reschedule Appointments.

Bulk Reschedule Appointments screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to display the Day Schedule status for both ‘From’ and ‘To’ locations. Click [Clear] to clear the displayed result.

    1. From Time – This field displays the Start Time of an appointment based on the appointment type set up in timetable for scheduling.

    2. To Time – This field displays the End Time of an appointment based on the appointment type set up in timetable.

    3. Patient Name – This field displays the details of the patient like Patient ID (In case of registered patient), Name of the Patient, Gender and Age. This field is blank if no appointments exist.

    4. Visit Type – This field displays the applicable Visit Type recorded while giving the appointment. The Visit Type is highlighted based on the colour indicator setup done in the Outpatient Management module.

    5. Reason For Contact – This field displays the reason for the patient seeking an appointment.

    6. Secondary Resources – This field displays the Secondary Resources selected for an appointment with a hyperlink.

  3. Click ‘Details’ hyperlink to view the secondary resources.

  4. Click to move all appointments from Location/Resource to the specified Location/Resource.

  5. Select the check box to transfer the appointments. If secondary resources are associated, a Secondary Resources screen gets displayed to select the required time slot/slab to which the secondary resources are to be booked. Select the check box against the required time slot/slab. Click [OK] to accept the change and the control reverts back to the ‘Bulk Transfer Appointments’ screen or [Cancel] to exit from the screen.

  6. Click [Confirm Revision] to transfer the appointments to the new date/resource.
