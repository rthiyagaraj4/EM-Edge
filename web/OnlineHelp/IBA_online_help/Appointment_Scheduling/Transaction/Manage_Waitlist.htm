Manage Wait List

From this function you can Cancel and Modify patients details in Wait list and Create a Booking for a patient assigned to a wait list.

Manage Wait list function is used to manage patients assigned to a waitlist. This function allows creating a booking for a patient in Waitlist. If booking is not created, the waitlist details of a patient is modified to a different date or cancelled if not applicable.

 

Process Flow Diagram

Managing Wait List

From the OA Transactions, select the Manage Wait list.

Manage Wait List screen gets displayed.

  1. Select Specialty to manage patients in Wait list for that particular specialty or enter the query criteria to search for a particular patient in wait list. <Related Field Description>

  2. Click [Search] to manage patients in waiting list. The results are displayed in sequence based on the ‘Priority’ of the ‘Wait List’ number. [Collapse] gets displayed in place of [Search]. Click [Collapse] to view all the query fields. Click [Clear] to clear the displayed data.

    1. Wait List No – This field displays the appointment Wait List Number allocated by the system. This field is made hyperlink.

    2. Patient ID – This field displays the Patient ID of the patient with Appointment.

    3. Patient Name – This field displays the Patient Name.

    4. Preferred Date – This field displays the preferred appointment date of the patient.

    5. Reference No - The Reference Number of the patient captured while creating the waitlist is displayed. The Reference Number helps in identifying the patients referred to the hospital for appointment from the list of patients.

    6. Specialty – This field displays the Specialty against which the appointment waitlist was generated.

    7. Category – This field displays the Category against which the appointment waitlist was generated.

    8. Location – This field displays the Location against which the appointment waitlist was generated.

    9. Resource – This field displays the Resource against which the appointment waitlist was generated.

    10. Previous Encounter – This field is made hyperlink. Click the Previous Encounter hyperlink to display the Previous Encounters screen. Click the Encounter ID hyperlink to view the encounter details. Click [Close] to close the Previous Encounters screen.

    11. Reason – The reason as recorded while creating waitlist for the patient is displayed.

  3. Click Wait list No hyperlink to Modify Wait list, Cancel Wait list or Create Booking. The screen displays three options.

Cancel Wait List

Modify Wait List

Create Booking
