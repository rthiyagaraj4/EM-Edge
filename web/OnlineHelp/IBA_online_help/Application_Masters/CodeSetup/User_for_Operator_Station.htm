User for Operator Station

From this function you can Associate authorized users to the Operator Stations of a Facility for a specific period and Query for a required record.

User for Operator Station function is used to associate authorized users to a Operator Station in the Facility. The user can manage the Operator Station for a specified period.

Associating a User to an Operator Station

From the Operator Station Setup, select the User for Operator Station function.

The User for Operator Station screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a User for Operator Station record created or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing User for Operator Station record.

  5. Click [Delete] to delete a queried record.
