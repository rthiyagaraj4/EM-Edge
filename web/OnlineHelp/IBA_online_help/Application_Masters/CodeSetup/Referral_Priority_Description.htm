Referral Priority Description

From this function you can Define the required referral priority description.

This function allows to define the description for the Referral Priority Code. The created priority description gets displayed in the Priority drop-down list values in the Referral transactions.

Defining Referral Priority Description

From the AM Other Setup, select Referral Priority Description function.

The Referral Priority Description screen gets displayed.

  1. The pre-defined Referral Priority Code and Referral Priority Description are displayed.

  2. Against each Referral Priority code, the default referral priority description is displayed.

  1. User can change only the Referral Priority description if required by removing the default value against the code in the text box. Enter the new description in alpha-numeric format. Special characters are not allowed. This field is mandatory.

  1. Once the description is modified, click [Apply] to save the changes done.
