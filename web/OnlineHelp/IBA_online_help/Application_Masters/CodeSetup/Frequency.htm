Frequency

From this function you can Create a new Frequency and Query for a required Frequency record.

Frequency is an option to describe the rate of occurrence of the order catalog for specific periods. Frequencies are broadly understood in two scenarios. One with a duration defined at the time of ordering and the other without any duration being defined at the time of placing orders.

Creating a Frequency

From Frequency Setup, select the Frequency function.

The Frequency screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Descriptions>

  3. Select the ‘Nature’ as Interval.

    1. Every – Enter the interval value in numeric format of six digits maximum length. The entered value has to be in Minutes(s) or Hour(s). The user can select the required value from the pre-defined list. This field is mandatory.

    2. Scheduled – The check box is disabled on selecting the frequency nature as ‘Interval’.

    3. Start Time Assignment –  Current Time is displayed as the start time and made default on selecting the frequency nature as Interval.

    4. Round To – The Round To field is disabled on selecting the frequency nature as ‘Interval’.

    5. Enabled - Select the check box to enable the Frequency record. The check box is selected, by default. Clear the check box to disable the Frequency record.

  4. Select the ‘Nature’ as PRN/One-Time/Continuous.

    1. Scheduled – The check box is disabled on selecting the Frequency Nature as ‘PRN/One-time/Continuous’.

    2. Start Time Assignment – Current Time is displayed as the start time and made default on selecting the frequency nature as ‘PRN/One-time/Continuous’.

    3. Round To – The Round To field is disabled on selecting the frequency nature as ‘PRN/One-time/Continuous’.

    4. Enabled - Select the check box to enable the Frequency record. The check box is selected, by default. Clear the check box to disable the Frequency record.

  5. Click [Apply] to save a Frequency created or [Reset] to clear the entered data.

  6. Click [Query] to search for an existing Frequency record.
