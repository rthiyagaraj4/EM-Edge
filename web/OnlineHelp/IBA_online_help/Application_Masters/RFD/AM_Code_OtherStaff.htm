Enter the following details to create an Other Staff record:

 

  1. Other Staff Type - The Other Staff Type indicates the functional nature of the staff. Select the Type of Other Staff from a list of valid Other Staff Types under which the Other Staff is to be created. This field displays the Other Staff Type created using the ‘Other Staff Type’ function with the ‘enabled’ check box selected. This field is mandatory.

  2. ID - Enter the Other Staff ID in alphanumeric format of 15 characters maximum length. An Other Staff ID is used to identify the Other Staff. The Other staff ID has to be unique and cannot be changed after creation. This field is mandatory.

  3. Full Name - Enter the full name of the Other Staff in alphanumeric format of 30 characters maximum length. This field is mandatory.

  4. Short Name - Enter the short name of the Other Staff in alphanumeric format of 15 characters maximum length. This field is mandatory.

  5. Long Name – Enter the long name of the other staff in this field. You can enter upto 300 characters. This field is optional.

  6. Enabled - the check box to enable the Other Staff record. The check box is selected, by default. Clear the check box to disable the Other Staff record.

The Other Staff screen comprises of two tab buttons, namely, Identity and Contact No/Addr. The ‘Identity’ maintains the other Staff’s personal details like Gender, Age, Employment Status, etc. The ‘Contact No/Addr’ maintains the other staff’s contact addresses details like Residential Address, Mailing Address, etc.

 

Identity Tab

  1. Gender - Select the required Gender of the Other Staff from the list. The pre-defined values are ‘Male’, ‘Female’ and ‘Unknown’. This field is mandatory.

  2. Age - Enter the Age of the Other Staff in years/months/days in the respective text box ‘Y’, ‘M’ and ‘D’. If date of birth is not known, on entering the age, system calculates the date of birth and displays it in the ‘Date of Birth’ field.

  3. Birth Date - If the age is not entered, enter the Date of Birth of the Other Staff in this field, or select the date using the Calendar function. On entering the birth date, system calculates the staff’s age and correspondingly displays the age in the Age field. This field is mandatory.

  4. Birth Place - Enter the Place of Birth of the Other Staff in alphanumeric format of 30 characters maximum length.

  5. Marital Status - Select the Marital Status of the Other Staff from the list. This field displays the values defined in ‘Marital Status’ function in Master Patient Index module.

  6. Alias Name - Enter the Alias Name (a name that has been assumed temporarily) of the other staff in alphanumeric format of 30 characters maximum length.

  7. Citizenship - Select the Citizenship of the Other Staff from the list. This field displays the values defined in ‘Citizenship’ function in Master Patient Index module.

  8. Other Alt No – Select the required other alternate no value from the drop-down list. The drop-down list value is based on the ‘Alternate ID Type’ defined in the MP Code Setup which is in enabled mode. By default, the value ‘Select’ is displayed. The text box next to the ‘Other Alt No’ field is enabled only when the ‘Other Alt No’ field is enabled. You can enter the legend value up to maximum of 20 characters and all special characters are allowed. The captured legend with value is displayed wherever the ‘Other Staff Details’ dialog, and the ‘Practitioner Details’ dialog are invoked in the EM application. When no value is captured, only the legend name ‘Other Alternate No’ is displayed without value.

  9. Position - Select the Position of the Other Staff from a list of valid Positions. This field displays the Positions created using the ‘Position’ function in Practitioner Setup for the ‘Role type’ selected as ‘Other Staff’ with the ‘Enabled’ check box selected. The values also depend on ‘Other Staff Type’ selected.

  10. Degree - Enter the Educational Qualification of the Other Staff in alphanumeric format of 15 characters maximum length.

  11. Empl Status - Select the required Employee Status from the list. The pre-defined values are Associate, Full Time, Part Time and Visitor. This field is mandatory.

  12. Rep Facility - Select the required Reporting Facility of the Other Staff from the list. The list box displays facility records as created in  the ‘Facility’ function of the System Manager module.

  13. All Facilities - Select the check box to indicate that the other staff works for all the facilities of the Enterprise. If this is selected, system identifies that the other staff is associated to all the Facilities or applicable facilities of the enterprise are to be associated with the Other Staff using the function ‘Other Staff for Facility’.

 

Contact No./Addr Tab

  1. Res Tel No - Enter the residential telephone number of the Other Staff in alphanumeric format of 18 characters maximum length.

  2. Office Phone No - Enter the official telephone number of the Other Staff in alphanumeric format of 18 characters maximum length.

  3. Extension - Enter the extension of official telephone number of the Other Staff in alphanumeric format of six characters maximum length.

  4. Fax No - Enter the fax number of the Other Staff in alphanumeric format of 18 characters maximum length.

  5. Mobile No - Enter the mobile number (if any) of the Other Staff in alphanumeric format of 18 characters maximum length.

  6. Pager No - Enter the pager number (if any) of the Other Staff in alphanumeric format of 18 characters maximum length.

  7. Email ID - Enter the E-mail ID of the Other Staff in alphanumeric format of 50 characters maximum length.

  8. Pref Mode - Select the preferred mode of contact of the Other Staff from the list. The pre-defined values are E-Mail, Fax, Mobile, Office, Pager and Residence.

  9. Address (Residence/Office/Mail) - Enter the Address in alphanumeric format of 30 characters maximum length in each of the four lines provided.

  10. In the Residence address fields, the user can enter address, where the other staff is presently residing.

  11. In the Mailing address fields, enter the address, where the correspondences to the other staff are to be sent.

  12. In the Office address fields, the user can enter the other staff’s office address.

  1. Postal Code (Residence/Office/Mail) - Enter a valid Postal Code for Residence, Office and Mailing Address respectively in alphanumeric format of eight characters maximum length. The Postal Codes gets listed on clicking [?] function, if it is set up in the ‘Postal Code’ function in the Master Patient Index module.

  2. Country (Residence/Office/Mail) - Select the relevant Country from the list. This field displays the values defined in ‘Country’ function in the Master Patient Index module.
