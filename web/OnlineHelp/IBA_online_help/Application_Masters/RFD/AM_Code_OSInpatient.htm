Inpatient

The tabs displayed are Booking, Admission, Transfer, Discharge and Others.

  1. Booking Tab is displayed, by default. Select the appropriate check boxes against the required functions to perform those functions of the operator station of type ‘Inpatient’.

    1. Create Booking - Select the check box to enable the Operator Station to Create Inpatient Booking. Clear the check box to restrict the Operator Station from Creating Inpatient Booking. The system clears check box, by default. On selecting the check box, ‘Confirm on Create Booking’ check box is disabled and vice versa. This check box is enabled for selection if the 'Confirm on Create Booking' check box is cleared.

    2. Revise Booking - Select the check box to enable the Operator Station to Revise Inpatient Booking. Clear the check box to restrict the Operator Station from revising inpatient booking.

    3. Cancel Booking - Select the check box to enable the Operator Station to Cancel Inpatient Booking. Clear the check box to restrict the Operator Station from canceling inpatient booking.

    4. Confirm Booking – Select the check box to enable the Operator Station to Confirm Booking for Admission. Clear the check box to restrict the Operator Station from confirming booking for admission.

    5. Confirm on Create booking – The check box is enabled only when ‘Create Booking’ is cleared. Select the check box to enable the Operator Station to confirm Inpatient Admission on booking. Clear the check box to restrict the Operator Station from confirming inpatient admission on booking. This check box is enabled for selection if the 'Create Booking' check box is cleared.

  2. Click 'Admission' Tab and select the appropriate check boxes against the required functions to perform those functions of the operator station of type ‘Inpatient’.

    1. Quick Admission – Select this checkbox if you want to the Quick Admission field to be available in the Register Referral page in OP Checkout, AE Checkout Close Consultation, and Register Outgoing Referral functions.

    Note: This field will be enabled only if the Admit Patient check box is selected.

    1. Admit Patient - Select the check box to enable the Operator Station to perform Inpatient Admission. Clear the check box to restrict the Operator Station from performing inpatient admission.

    2. Admit Patient with Bed – The check box is enabled only if ‘Admit Patient’ is selected. Select the check box to enable the Operator Station to allocate bed on admission of inpatients. Clear the check box to restrict the Operator Station from allocating bed on admission of inpatients. This check box is enabled if the 'Admit Patient' check box is selected.

    3. Register Patient From Admit Patient – Select the check box to enable the Operator Station to perform Patient Registration while admitting the patient to the Nursing Unit. Clear the check box to restrict the Operator Station from performing patient registration while admitting the patient to the nursing unit.

    4. Assign Bed - Select the check box to enable the 'Assign Bed' function in Inpatient and Day Care Management module for the Operator Station. Clear the check box to restrict the Operator Station from accessing Assign Bed function.

    5. Allow Changing Nursing Unit in Assign Bed – The check box is enabled only when ‘Assign Bed’ is selected. Select the check box to allow the Operator Station to change the nursing unit while assigning bed on inpatient admission. Clear the check box to restrict the Operator Station from changing the Nursing Unit while assigning bed on inpatient admission.

    6. Assign Treating Practitioner - Select the check box to enable the Operator Station to assign Treating Practitioner to inpatient. Clear the check box to restrict the Operator Station from assigning treating practitioner to inpatient.

    7. Change Admission Details - Select the check box to enable the Operator Station to change admission details of an Inpatient. Clear the check box to restrict the Operator Station from changing the admission details of an Inpatient.

    8. Cancel Admission - Select the check box to enable the Operator Station to cancel inpatient admission. Clear the check box to restrict the Operator Station from canceling inpatient admission.

    9. Backdated Admission - Select the check box to allow the user to backdate the admission. Clear the check box to restrict the user from backdating the admission.

    10. Link Baby to Mother – Select the check box to enable the Operator Station to access the 'Link Baby to Mother' function in Inpatient and Day Care Management module. Clear the check box to restrict the Operator Station from accessing the 'Link Baby to Mother' function in Inpatient and Day Care Management module.

    11. Register Newborn for IP – Select the check box to enable the Operator Station to access 'Register Newborn for Inpatient' function. Clear the check box to restrict the Operator Station from accessing 'Register Newborn for Inpatient' function.

  3. Click 'Transfer' Tab and select the appropriate check boxes against the required functions to perform those functions of the operator station of type ‘Inpatient’.

    1. Transfer Patient - Select the check box to enable the Operator Station to perform Inpatient Transfer within the Nursing Unit. Clear the check box to restrict the Operator Station from performing Inpatient Transfer within the Nursing Unit.

    2. Request for Transfer - Select the check box to enable the Operator Station to access request for transfer function. Clear the check box to restrict the Operator Station from accessing inpatient Request for Transfer function.

    3. Cancel Request – Select the check box to enable the Operator Station to cancel the request raised for inpatient transfer. Clear the check box to restrict the Operator Station from canceling the request raised for inpatient transfer.

    4. Confirm Request - Select the check box to enable the Operator Station to confirm the request raised for inpatient transfer. Clear the check box to restrict the Operator Station from confirming the request raised for inpatient transfer.

    5. Transfer Out – Select the check box to enable the Operator Station to Transfer Out the inpatient. Clear the check box to restrict the Operator Station from transferring out the inpatient.

    6. Cancel Transfer Out - Select the check box to enable the Operator Station to cancel transferring the inpatients out. Clear the check box to restrict the Operator Station from canceling the transfer of inpatients out.

    7. Transfer In – Select the check box to enable the Operator Station to Transfer In the transferred out patients. Clear the check box to restrict the Operator Station from transferring in the transferred out patients.

    8. Swap Bed – Select the check box to enable the Operator Station to Swap Beds of inpatients within a Nursing Unit. Clear the check box to restrict the Operator Station from swapping beds of inpatients within a Nursing Unit.

    9. Emergency Transfer - Select the check box to enable the Operator Station for Emergency Transfer. Clear the check box to restrict the Operator Station for Emergency transfer.

    10. Practitioner Transfer – Select the check box to enable the Operator Station to Transfer Practitioner. Clear the check box to restrict the Operator Station from transferring a Practitioner.

    11. Cancel Reserve Beds - Select the check box to enable the Operator Station to cancel reserved beds. Clear the check box to restrict the Operator Station from cancelling beds that are reserved.

  4. Click 'Discharge' Tab and select the appropriate check boxes against the required functions to perform those functions of the operator station of type ‘Inpatient’.

    1. Discharge Advice – Select the check box to enable the Operator Station to Prepare Discharge Advice. Clear the check box to restrict the Operator Station from preparing Discharge Advice.

    2. Discharge Patient - Select the check box to enable the Operator Station to Discharge an Inpatient. Clear the check box to restrict the Operator Station from discharging the Inpatient.

    3. Bed Class Change - Select the check box to enable the Operator Station to allow Bed Class Change. Clear the check box to restrict the Operator Station from allowing Bed Class Change.

    4. Discharge Checklist – Select the check box to enable the Operator Station to access Discharge Checklist function. Clear the check box to restrict the Operator Station from accessing Discharge Checklist function.

    5. Cancel Discharge - Select the check box to enable the Operator Station to cancel the Inpatient discharge. Clear the check box to restrict the Operator Station from cancelling the discharge of an  Inpatient.

  5. Click 'Others' Tab and select the appropriate check boxes against the required functions to perform those functions of the operator station of type ‘Inpatient’.

    1. Block Bed – Select the check box to enable the Operator Station to Block Bed for inpatients. Clear the check box to restrict the Operator Station from blocking beds.

    2. Release Bed – Select the check box to enable the Operator Station to Release Bed for inpatients. Clear the check box to restrict the Operator Station from releasing beds for inpatients.

    3. Record/Amend Leave – Select the check box to enable the Operator Station to Record and Amend inpatient leave. Clear the check box to restrict the Operator Station from recording or amending inpatient leave.

    4. Check In/Check Out Lodger - Select the check box to enable the Operator Station to perform Check In and Check Out Lodger. Clear the check box to restrict the Operator Station from performing Check In and Check Out Lodger.

    5. Register Inpatient Referral - Select the check box to enable registration of Inpatient Referral. Clear the check box to restrict the registration of Inpatient Referral.

    6. Confirm Bedside Referral – Select the check box to enable confirmation of Bedside Referral. Clear the check box to restrict confirmation of Bedside Referral.

    7. Diagnosis / Procedure Recoding - Select the check box to enable Diagnosis Procedure Recoding. Clear the check box to restrict Diagnosis Procedure Recoding.

    8. Maintain Death Register – Select the check box to enable the records in 'Maintain Death Register' function. If the check box is cleared and the user is linked to the Operator station, a message ‘Operator Station does not have access to this Function’ is displayed.
