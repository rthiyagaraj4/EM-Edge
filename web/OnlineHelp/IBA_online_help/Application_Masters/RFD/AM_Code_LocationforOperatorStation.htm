Enter the following details to associate a Location to an Operator Station:

 

  1. Facility - Select a Facility from the list. The facility to which the user is associated through the ‘Facilities for User’ function in Enterprise Configuration module are listed. The logged in facility is displayed, by default. This is a mandatory field.

  2. Operator Station - Select an Operator Station to which a Location has to be associated. The values are listed from ‘Operator Station’ function. This is a mandatory field.

  3. Location Type - Select a Location Type from the list. The locations are listed based on the indicators set for a location in the ‘Care Location Type’ function. If the Operator Station of Type ‘Inpatient’ is selected, care locations of indicator ‘Nursing Unit’ are displayed. If the Operator Station of Type ‘Outpatient’ is selected, care locations of indicator ‘Clinic’, ‘Day Care Unit’ and ‘Procedure Unit’ are displayed. If the Operator Station of Type ‘Inpatient/Outpatient’ is selected, care locations of indicator ‘Nursing Unit’, ‘Clinic’, ‘Day Care Unit’ and ‘Procedure Unit’ are displayed. This is a mandatory field.

  4. Specialty - Select a Specialty from the list. The specialties created through ‘Specialty’ function are listed.

  5. Search By – Select the 'Search By' field as Location Code or Location Description and enter the required text in the text box provided to search for Locations. The records matching the search criteria are listed.
