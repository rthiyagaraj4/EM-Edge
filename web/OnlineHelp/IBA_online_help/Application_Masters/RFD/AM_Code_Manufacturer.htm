Enter the following details to create a Manufacturer record:

  1. ID - Enter the Manufacturer ID in alphanumeric format of 20 characters maximum length. A Manufacturer ID is used to identify the Manufacturer. The Manufacturer ID has to be unique and cannot be changed after creation. This field is mandatory.

  2. Long Name – Enter the long name of the Manufacturer in alphanumeric format of 40 characters maximum length. This field is mandatory.

  3. Short Name – Enter the short name of the Manufacturer in alphanumeric format of 20 characters maximum length. This field is mandatory.

  4. Enabled – Select the check box to enable the Manufacturer record. The check box is selected, by default. Clear the check box to disable the Manufacturer record.

 

Address Details

  1. Address Line (1-4) – Enter the address of the manufacturer in alphanumeric format of 30 characters maximum length in each of the four text boxes provided.

  2. Postal Code – Enter a valid Postal Code in alphanumeric format of eight characters maximum length. The Postal Codes gets listed on clicking [?] function, if it is set up in the ‘Postal Code’ function in the Master Patient Index module.

  3. Country – Select the relevant Country from the list of values. This field displays the values defined in ‘Country’ function in the Master Patient Index module.

 

Contact Details

  1. Tel No – Enter the Telephone Number of the Manufacturer in alphanumeric format of 18 characters maximum length.

  2. Fax No – Enter the Fax Number of the Manufacturer in alphanumeric format of 18 characters maximum length.

  3. Email – Enter the Email ID of the Manufacturer in alphanumeric format of 50 characters maximum length.

  4. Web-site – Enter the Website of the Manufacturer in alphanumeric format of 50 characters maximum length.
