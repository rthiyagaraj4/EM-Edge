Enter the following details to create a Frequency record:

 

  1. Code - Enter the Frequency Code in alphanumeric format of four characters maximum length. A Frequency Code is used to identify the Frequency. The Frequency code has to be unique and cannot be changed after creation. This field is mandatory.

  2. Description - Enter the description of the Frequency in alphanumeric format of 20 characters maximum length. This field is mandatory.

  3. Print Description 1 - Enter the Print Description 1 in alphanumeric format of 40 characters maximum length. This description is printed when the prescription is printed.

  4. Print Description 2 - Enter the Print Description 2 in alphanumeric format of 40 characters maximum length. This description is printed when the prescription is printed.

  5. Nature – Select the required Nature from the list of values. The pre-defined values are ‘Interval’, ‘Frequency’, 'PRN’, ‘One time’ and ‘Continuous’., Frequency is displayed by default. This field is mandatory.

  6. Repeat(s) s Per – Enter the Repeat(s) per value in numeric format of six digits maximum length. Select the required duration type from the list of values. The pre-defined values are ‘Minute’, ‘Hours’, ‘Days’, ‘Weeks’, ‘Months’ or ‘Years’. The entered repeat is of the selected duration type. This field is mandatory.

  7. Every – Enter the repeats interval value in numeric format of six digits maximum length. Whatever Repeat(s) per value type is selected, the same is displayed and disabled in the second text box for repeats interval type. This field is mandatory.

  8. Scheduled - If the check box is selected, the frequency gets displayed for scheduling in Frequency Schedule function. You are able to enter the exact time when the frequency is scheduled. If the check box is cleared, the frequency does not get displayed in the Frequency Schedule function for further scheduling. The frequency schedule is used by the user while placing orders.

  9. Start Time Assignment – Select the required Start Time of the frequency from the list of values. The pre-defined values are ‘Next’, ‘Previous’ and ‘Closest’. The start time of the frequency is attached to the rounding rule.

  10. Round To - Select the required Round To value from the list of values. The pre-defined values are ‘Minute’, ‘5th Minute’, ‘10th Minute’, ‘Quarter Hour’ and ‘Hour’.

  11. Enabled - Select the check box to enable the Frequency record. The check box is selected, by default. Clear the check box to disable the Frequency record.
