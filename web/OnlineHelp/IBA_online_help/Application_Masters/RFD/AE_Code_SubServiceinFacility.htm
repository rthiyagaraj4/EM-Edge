Enter the following details to associate a Sub-Service to a Facility:

 

  1. Facility - Select a Facility from the list of values to associate a Sub-service. The facilities associated to the logged in user and selected responsibility gets displayed in the list. The system displays the logged in facility, by default. This is a mandatory field.

  2. Service - Select a Service from the list of values. Associate the Sub-service to a facility from the list of Sub-services associated to the selected Service. This field displays the services associated to the facility and department with the status as enabled through ‘Service in Facility’ function. This is a mandatory field.

  3. Sub-Service - Select a Sub-service from the list to associate it to the Facility. This field displays the Sub-services associated to the selected Service. This is a mandatory field.

  4. Account Department - Select an Accounting Department from the list. This field displays the Accounting Departments associated to the selected Service for the given Facility of the enterprise. This is a mandatory field.

  5. Section - Select a Section of the Accounting Department from the list. This field displays the Sections associated to an Accounting Department. This is a mandatory field.

  6. Enabled - Select the check box to enable the record in the transaction functions. Clear the check box to disable the record. By default, the check box is selected.
