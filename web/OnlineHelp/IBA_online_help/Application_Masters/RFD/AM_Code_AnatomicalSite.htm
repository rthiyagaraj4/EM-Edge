Enter the following details to create an Anatomical Site record:

 

  1. Code – Enter the Code in alphanumeric format of four characters maximum length. An Anatomical Site Code is used to identify an Anatomical Site. The Anatomical Site code has to be unique and cannot be changed after creation. This field is mandatory.

  2. Long Description - Enter the long description of the Anatomical Site in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description - Enter the short description of the Anatomical Site in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Applicable to ISS - Select the check box, to indicate that the anatomical site is considered for ISS. According to International Trauma Organization, various Anatomical Sites of the patient are assessed to prioritize a patient. Examples of Anatomical Sites can be neck, knee, arm, etc. This function is to avoid duplication of Anatomical Site records applicable for ISS.

  5. Oral Cavity - Select the check box to list the created Anatomical Site as an Oral Cavity in the Record Diagnosis function in Clinician Care Management and Medical Records modules.

  6. Enabled - Select the check box to enable the Anatomical Site record. The check box is selected, by default. Clear the check box to disable the Anatomical Site record.
