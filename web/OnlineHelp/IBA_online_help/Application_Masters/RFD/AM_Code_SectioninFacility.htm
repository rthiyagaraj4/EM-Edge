Enter the following details to create a Section in Facility record:

 

  1. Facility - Select a Facility from the list. This field displays the facilities created through the Facility function of System Manager module. The system displays the logged in facility, by default. This is a mandatory field.

  2. Account Department - Select the Accounting Department from the valid list of values. The Departments associated to the selected Facility through ‘Facility for Acct Dept’ function are displayed. This is a mandatory field.

  3. Section - Select the required Section to be associated to the Facility from the list. The Sections associated to the selected Accounting Department through ‘Section’ function are listed. This is a mandatory field.

  4. Enabled - Select the check box to enable the record in the transaction functions. Clear the check box to disable the record. The check box is selected, by default.
