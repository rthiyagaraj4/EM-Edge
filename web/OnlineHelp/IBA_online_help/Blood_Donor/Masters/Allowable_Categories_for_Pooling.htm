Allowable Categories for Pooling

From this function you can Associate the allowable blood categories to a product category. You can Query to view the existing records.

The Allowable Categories for Pooling function enables you to define the blood categories that are valid for a Product Category. The values defined are used in the Pool Units function.

Creating Allowable Categories for Pooling

From the Masters, double-click the Allowable Categories for Pooling function.

The Allowable Categories for Pooling screen gets displayed.

  1. Enter a valid Category code or click [List of Values] to select it. The values defined using the Product Categories function get listed. If the code is valid, the description of the code is displayed for user verification.

  2. Click [List of Values] to select a valid Category code. The description is displayed on selection of the code. The values defined using the Product Categories function get listed.

  3. Click [Save] or press [F10] to save the record or click [Clear Record] clear the entered data.

  4. Click [Execute Query] or press [F8] to search for the existing records.
