Suspend/Reinstate Masters

From this function you can Suspend or Reinstate the records defined for the Master functions in the Blood Donor module.

The Suspend/Reinstate Masters function is used to suspend or reinstate a valid records defined for a function in the Master menu.  

Suspending/Reinstating the Masters

From Administration, double-click the Suspend/Reinstate Masters function.

The Suspend/Reinstate Masters screen gets displayed.

  1. Select the master function.

  2. Click [List of Values] to select the Code. The list displays the names of the master functions in the Blood Donor module.

  3. Select the action from the list. The valid values are ‘Suspend’ and ‘Reinstate’. Select ‘Suspend’ from the list to defer the code.

  4. Click [Save] or press [F10] to save or click [Clear Record] to clear the entered data.

  5.  An alert message gets displayed. Click [Yes] to suspend or [No] to cancel the suspension process.

  6. Select ‘Reinstate’ from the list to enable the suspended code. An alert message gets displayed. Click [Yes] to enable or [No] to cancel the suspension process.

 

The field legends change based on the function selected in the Master menu.
