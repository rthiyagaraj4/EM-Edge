Setup Fax Service

From this function you can Create the Fax services for the users.

The Setup Fax Service function is used to set up fax services for the users. The routing mode can be by Email or by Printer. The Email user option can be either common or individual. The Email profile created is the default one and will be displayed and disabled.

Setting up Fax Service

From Administration > Manage Services > Setup, double-click the Setup Fax Service function.

The Setup Fax Service screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>  

  2. Click [Save] or press [F10] to save or click [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for the existing records within the facility.
