User Security

From this function you can Assign access rights to a user to change the deferred date of a donor.

The User Security function enables you to give access right to a user to change the deferred date of a donor. It is a one-time setup done by the system administrator and you cannot update the details.

User Security

From the Administration, double-click the User Security function.

The User Security screen gets displayed.

  1. Click [List of Values] to select the User. This is a mandatory field.

  2. Click [List of Values] to select the Access Control details. The Access Control description is displayed.

  3. Select the check box to give access rights to the selected user. This is a one-time setup and you cannot update the records.

    Select the check box to authorize a user to perform the following actions:

    1. ALDF - Allowed deferred donor autologous donation - Select the check box to authorize user to allow deferred donor for autologous donation.

    2. MBDF - Allow mobile deferred donor to donate - Select the check box to authorize the user to permit mobile deferred donors to donate blood.  

    3. TMDF - Allow temporary deferred donor to donate - Select the check box to authorize the user to permit temporarily deferred donors to donate blood.

    4. VWRS - View Screened Results - Select the check box to authorize the user to view screened results.

  4. Click [Save] or press [F10] to save or click [Clear Record] to clear the entered data.
