Recall Donor for Donation

From this function you can Query to view the list of donors’ who are recalled for blood donation and View the specific donor history. You can Print the recall list and the recall letter.

The Recall Donor for Donation function allows you to query the details of donors who are recalled for blood donation. You can print the Recall letter of the donor or print the recall list.

Querying the Recalled Donors’ Details

From Query, double-click the Recall Donor for Donation function.

The Recall Donor for Donation screen gets displayed.

  1. Enter the Blood Group or click [List of Values] to select it. The values defined using the Blood Groups function get listed.

  2. Enter the Rhesus factor or click [List of Values] to select it. The values defined using the Rhesus function get listed.

  3. Enter the Required Date or click the Calendar function to recall the donor.

  4. Click [Execute Query] or press [F8] to search for the donor records within the facility.

  5. The Donor ID, Donor Name, Contact Number and the Last Donation Date get displayed.

  6. Select the ‘Select?’ check box to view the details or to print the recall list of a specific donor.

  7. Select the ‘Select All’ check box to view the donor history, print the recall list or recall letter of the listed donors.

  8. Click [Donor History] to view the details. The donor history with the unit details screen gets populated with the donor details.

  9. Click [Print Recall Letter] to print the recall letter for the donor.

  10. Click [Print Recall List] to print the recall list of the donor.
