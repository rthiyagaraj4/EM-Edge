Donor Query

From this function you can Query to view the donation details of a donor. You can View the donor history and print the donor details.

The Donor Query function enables you to search for a donor based on blood group, rhesus, risk factor, antibody or antigen details. The Blood Group and the Rhesus factor are mandatory view the donation details. The donation details for the selected Blood Group and the Rhesus Factor get listed.

Querying the Donor Details

From Query, double-click the Donor Query function.

The Donor Query screen gets displayed.

  1. Enter the Blood Group or click [List of Values] to select the blood group. The list displays the values defined using the Blood Group function.

  2. Enter the Rhesus Factor or click [List of Values] to select the Rhesus. The list displays the values defined using the Rhesus function. <Related Field Description>

  3. Click [Execute Query] or press [F8] to query for the donor details.

    The following details are displayed:

    1. Donor ID – The Donor ID satisfying the entered query criteria is displayed.

    2. Donor Name – The Name of the Donor recorded during registration in Donor Registration function is displayed.

    3. Contact Number – The Contact Number of the donor recorded in Donor Registration function is displayed.

    4. Last Donation Date – The Last Donation Date of the donor is displayed in descending order.

  4. Select the check box against the specific donor record to view donor history or print details.

  5. Select the check box ‘Select All’ the check box to select all the records.

  6. Click [Donor History] to history of the donor. The Donor History with Unit Details screen gets invoked with the donor details.

  7. Click [Print] to print the donor details.
