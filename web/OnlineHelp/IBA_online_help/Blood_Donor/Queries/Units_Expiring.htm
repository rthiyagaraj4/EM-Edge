Units Expiring

From this function you can Query the Status of the Units approaching the expiry date. You can View the donation details or Print the unit details.

The Units Expiring query function allows you to query the unit details, which are approaching the expiry date. You can also print the number of units expiring within the selected date range.

Querying the Expiring Units

From Query, double-click the Units Expiring function.

The Units Expiring screen gets displayed.

  1. Click the Calendar function to select the Expiry Date range for the products. Click [List of Values] to select the Product.

  2. Click [Execute Query] or press [F8] to search for the units expiring within the date range specified. <Related Field Description>

  3. Click [Donation Details] to view the donation details .

  4. Click [Print] to print the details of the units approaching the expiry date.
