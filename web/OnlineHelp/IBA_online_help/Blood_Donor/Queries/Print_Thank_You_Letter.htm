Print Thank You Letter

From this function you can Query the details of the donors who have donated blood and Print thank you letters for the donors.

The Print Thank You Letter function is used to print thank you letter to the donor for donating the blood. You can view the Donor History and print the donor details using this function.

Printing Thank You Letter

From Query, double-click the Print Thank You Letter function.

The Print Thank You Letter screen gets displayed.

  1. Click the Calendar function to select the date range. The list of donors who have donated blood within the date range selected gets displayed. This is a mandatory field.

  2. Enter a valid Batch Number to view donation details. The donor records matching the specific Batch Number get displayed.

  3. Click [Execute Query] or press [F8] to search for the donation details. The Donor ID, Donor Name, Donation Date, Donation Type Process, Product Code and Unit Number are displayed.

  4. Select the ‘Select All’ check box to either view donor history, print details or print thank you letter for the listed donors. Select the ‘Select’ check box to either view donor history, print details or print thank you letter for the selected donors.

  5. Click [Donor History] to view the history details of the donor. The donor history with unit details screen is invoked with the respective donor details.

  6. Click [Print] to print the donor details.

  7. Click [Print Thank You Letter] to print the thank you letter for the donors.
