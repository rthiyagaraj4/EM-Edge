Donor History With Unit Details

From this function you can Query to view the donor’s previous encounter details and Print the donor details.

The Donor History with Unit Details function enables you to query the donor’s previous encounter details in the facility. The units which are cancelled and for which the components are prepared, added to stock or not added to stock are displayed using   and colors respectively. Blood Groups and Rhesus Factors with initial Blood Typing are indicated using color indicator.

Querying the Donor History with Unit Details

From the Query, double-click the Donor History with Unit Details function.

The Donor History with Unit Details screen gets displayed.

  1. Enter the Donor ID or click [Donor Lookup] to select a donor. The name of the donor, age, blood group, rhesus and the gender gets displayed on selecting Donor ID.

  2. Click [Execute Query] or press [F8] to search for the existing donor records.

  3. Click [D] to view the Donation Date, Donation Type, Unit Number, Product Code and Description, if any previous encounter exists for the donor.

  4. Click [Antibodies] to view Antibodies associated to the unit number. This button gets enabled only if Antibody details are recorded for the donor.

  5. Click [Antigens] to view Antigens associated to the unit number. This button gets enabled only if Antigen details are recorded for the donor.

  6. Click [Donor Notes] to view Donor Notes associated to the unit number. This button gets enabled only if Donor Notes are recorded for the donor.

  7. Click [Print Donor History] to print the donor history.

  8. Click [Donor Details] to view the donor’s demographic details recorded through the ‘Register Donor’ function. On clicking [Donor Details], the Donor Details screen is displayed. The Donor Details recorded during donor registration is displayed.

  9. Click [Unit Details] to view the unit details such as blood unit screening details, component details along with questionnaire details and physical checkup details. The Unit Details screen is invoked.

The [Unit Details] is enabled, only if the donor has donated blood earlier.
