Enter the following details to search and select a set of units:

 

  1. Unit Source – Select the blood unit source from the list. The value ‘All’ is displayed, by default. The listed values are ‘All’, ‘In-house’ and ‘Mobile’.

  2. Stage – Select the screening level from the list. The valid values are ‘Level 1’ and ‘Level 2’. You can send the units for ‘Level 2’ screening only if the result of the ‘Level 1’ screening is not satisfactory.

  3. Result Status – Select the Result Status as ‘Resulted’ or ‘Awaiting’ to view a list of units based on the selected status within the specified batch number or donation date range. The Units are resulted during the screening process in the lab.

  4. Batch Number – Enter the Batch Number to search for the screening results based on the batch created.  

  5. Donation [From-To] – Enter the donation date range or click the Calendar function to select the donation date. The blood donation details recorded during the selected date range gets displayed.

  6. Barcode Unit Number – Scan the Product Bar Code to display the details of the selected product.
