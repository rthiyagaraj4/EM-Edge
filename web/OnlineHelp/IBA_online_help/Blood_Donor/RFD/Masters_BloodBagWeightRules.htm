Enter the following details:

 

  1. Donated Product Code/Description – Enter a valid donated product code or click [List of Values] to select the required donated product code. The Products defined with the value set as ‘Yes’ for ‘Donated Product?’ are listed in this function. The description is displayed based on the product code selected. This is a mandatory field.

  2. Blood Bag Code/Description – Enter a valid blood bag code or click [List of Values] to select the blood bag. The values defined using the Blood Bag function get listed. The description is displayed based on the Blood Bag code selected. This is a mandatory field.

  3. Wt From/Unit – Enter the minimum weight in numerical format of four digits maximum length. The unit of weight gets displayed in the ‘Unit’ field automatically, on entering the value. This is a mandatory field.

  4. Wt To/Unit - Enter the maximum weight in numerical format of four digits maximum length. The unit of weight gets displayed in the ‘Unit’ field automatically on entering the value. This is a mandatory field.
