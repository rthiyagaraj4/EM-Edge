Enter the following details to record the details after performing physical checkup:

 

  1. Weight – Enter the Weight of the donor in numeric format.

  2. Temperature – Enter the body Temperature of the donor.

  3. Systolic Pressure - Enter the Systolic Pressure of the donor.

  4. Diastolic Pressure – Enter the Diastolic Pressure of the donor.

  5. Pulse – Enter the Pulse rate of the donor.

  6. Hemoglobin – Enter the Hemoglobin level of the donor.

After you enter the Weight, press Tab or press [Enter] and the cursor moves to the Temperature field. On entering the value in Temperature field, the cursor moves to Systolic Pressure, Diastolic Pressure, Pulse and Hemoglobin fields respectively, by default.

  1. Units – The respective units defined in the Physical Questionnaire setup of Administration function is displayed.

  2. Performed By – The logged in user name is displayed, by default.

  3. Location – Enter a valid Location code or click [List of Values] to select the location code. The values defined using the ‘Room’ function  in the Application Masters module get listed.

  4. Description – The Description of the location code selected is displayed. The values defined in the 'Room' function through the Application Masters module get displayed.

  5. Performed Date – Click the Calendar function to record the date when the physical checkup is performed.
