Enter the following details to search for a specific donor record:

 

  1. Patient ID – Enter a valid Patient ID to view the demographic details of the patient, or right-click and use the Patients Lookup option to search for a patient. You can register a patient with past encounter as a donor.

  2. Series/Donor ID – The Series ID associated to the selected Patient ID is displayed. Series ID value is defined in the Patient Series function of the Master Patient Index module.

  3. NRIC – Enter the National Registration Identity Card (NRIC) number of the patient. This field is mandatory if the ‘Accept’ check box under National ID Checks label is selected in MP Parameter, and ‘National ID Mandatory?’ check box is selected in Blood Donor Parameter.

  4. Alt Type/No – Select an Alt Type/Number from the list for the donor. Alt Type list displays the values defined in the Alternate ID Type function in Master Patient Index module.

  5. Name – The Name of the patient recorded during registration is displayed on selecting the Patient ID or Alt Type/Number.

 

The field names in the screen depend on the Alternate ID description given in the ‘MP Parameter’ Alternate ID tab. Demographic tab is used to record donor’s demographic details. The system displays the demographic details, by default, if you select a patient, who has had any previous encounter with the facility. You can right-click and click the ‘Patients Lookup’ option to select a patient ID.
