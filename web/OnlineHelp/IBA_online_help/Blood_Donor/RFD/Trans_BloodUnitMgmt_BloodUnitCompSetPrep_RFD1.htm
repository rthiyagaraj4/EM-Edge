Enter the following details to query the details:

 

  1. Unit Source – Select the Unit Source from the list. The list displays ‘All’, ‘Mobile’ and ‘In-house’.

  2. Components – Select the component value from the list to search for the blood units. The listed values are All, Prepared and Not Prepared. The value All is selected, by default.

  3. Batch Number – Enter the Batch Number to search for the blood units based on the batches. The batch number created in the Create Batches function gets displayed.  

  4. Donation Date [From–To] – Click the Calendar function to select the donation date range to search for the units based on the Date range.

  5. Barcode Unit Number – Enter the Bar Code Unit Number in alphanumeric format of 30 characters maximum length or scan the blood unit to get the bar code automatically.
