Enter the following details:

 

  1. Donated Product – Enter a valid donated product code or click [List of Values] to select the donated product code. The values defined using the Products function get listed. The description of the selected product is displayed on entering the product code.

  2. Blood Bag Code – Enter a valid blood bag code or click [List of Values] to select the valid blood bags code. The values defined using the Blood Bag function get listed. The description of the selected blood bag code is displayed on entering the blood bag code.

  3. Weight Range [From-To] – Based on the selected donated product and the blood bag code, this field displays the data specified in ‘Wt From/To’ fields of the ‘Blood Bag Weight Rules’ function. The weight range defined for the selected Product and the Blood Bag gets displayed. The weight range is defined using the function ‘Blood Bag Weight Rules’.

  4. Number of Blood Bags – The number of blood bags gets displayed for the selected Product. The blood bag numbers are set in the function ‘Blood Bag Weight Rules’.

  5. Blood Bag Rule Seq Number – The Blood Bag Rule Sequence Number is displayed.

  6. Default? – Select the check box to make the selected blood bag as default blood bag for the selected product. Only one blood bag can be selected as the default blood bag. Clear the check box, if any other blood bag is selected as default blood bag or if you do not want to select the selected blood bag as the default blood bag.

  7. Product – Enter a valid product code or click [List of Values] to select the product. The values defined using the Products function get listed.

  8. Description – The description for the selected product gets displayed.

  9. Expiry Period – The expiry period of the selected product gets displayed.

  10. Units – The unit of the expiry period of the selected product gets displayed.

  11. Max Time Period between Donation - Preparation – Enter the Maximum Period between Donation and Preparation in numeric format of three digits maximum length.

  12. Units – Enter the unit for the respective maximum period between donation preparation periods. The listed values are system defined.
