Dispose Cancelled Units

From this function you can Dispose the cancelled blood units and Query for the cancelled blood units.  

The Dispose Cancelled Units function enables you to dispose the blood units which are cancelled using the 'Unit Cancellation’ function. You can query to view the cancelled blood units. The disposed blood units are not available for further use.

Disposing Cancelled Units

From the Transactions > Blood Unit Management, double-click Dispose Cancelled Units function.

The Dispose Cancelled Units screen gets displayed.

  1. Enter the query criteria to search for the cancelled units. <Related Field Description>

  2. Press [Tab] to view the cancelled units’ details.

    The following details are displayed based on the query criteria specified:

    1. Unit Number – The Unit Number of the cancelled units get displayed based on the query criteria selected.

    2. Product Code – The Product code associated to the unit number is displayed.

    3. Description – The corresponding Description of the fate code is displayed.

    4. Date Collected – The date when the unit is collected is displayed.

    5. Cancel Date – The date when the unit is cancelled gets displayed.

    6. Cancel Code – The Cancel Code associated while cancelling the unit is displayed.

    7. Description – The Description of the Cancel Code is displayed.

  3. Select the ‘Select All’ check box to select the cancelled units for disposal. Select the check box against an individual record to dispose the cancelled unit.

  4. Click [Print] to print a list of cancelled blood units.

  5. Select the unit number of cancelled units and click [Dispose Units] to dispose the cancelled units.

  6. Enter the details in the Dispose Units screen.  

    Enter the following details to dispose the cancelled units:

    1. Dispose Code – Enter the dispose code or click [List of Values] to select it. The values defined using the Fates function get listed.

    2. Disposed By – Enter the user name or click [List of Values] to select the user.

    3. Disposed On – Enter the Disposal Date or click the Calendar function to select the disposal date.

  7. Click [Save] to dispose the cancelled unit. A confirmation message ‘Selected Units Disposed’ gets displayed. Click [Cancel] to cancel the transaction.
