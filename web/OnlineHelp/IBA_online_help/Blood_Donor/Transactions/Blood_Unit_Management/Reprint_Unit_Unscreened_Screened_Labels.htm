Reprint Unit/Unscreened/Screened Labels

From this function you can Reprint the unit labels for screened and unscreened blood units.

The Reprint Unit/Unscreened/Screened Labels function enables you to reprint the unit labels for screened and unscreened blood units. You can query the details based on the Unit Number or Donation Date range.

Reprinting Unit/Unscreened/Screened Labels

From the Transactions > Blood Unit Management, double-click the Reprint Unit/Unscreened/Screened Labels function.

The Reprint Unit/Unscreened/Screened Labels screen gets displayed.

  1. Enter the details to search for the unit details.  

    Enter the following details to search for the unit details:

    1. Unit Number [From-To] – Enter a range of Unit Numbers to display the units for which the labels have to be printed.

    2. Donation Date [From-To] – Enter Donation Date or click the Calendar function to select the date range for which the labels have to be printed.

  2. Click [Show Details] to view a list of units within the specified query criteria. <Related Field Description>

  3. Select the ‘Select All’ check box to select all the records. Select the check box against each record to reprint Unit/Unscreened/Screened Labels for the selected records.

  4. Enter number of labels to be printed and click [Print] to reprint Unit/Unscreened/Screened Labels.
