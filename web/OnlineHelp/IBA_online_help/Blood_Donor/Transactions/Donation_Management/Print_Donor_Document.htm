Print Donor Documents

From this function you can Print the Questionnaire, Donor Card, Consent Form and Donor History. You can View Antibodies, Antigens, Donor Notes and Unit Details of the donor.

The Print Donor Documents function enables you to print Questionnaire, Donor Card, Consent Form and Donor History. You can view Antibodies, Antigens, Donor Notes and Unit Details recorded for each donor.

 

The units cancelled are highlighted with  color and the units for which components are to be prepared, or added to stock or not added to stock are highlighted with  color.

Printing Donor Documents

From Transactions > Donation Management, double-click the Print Donor Documents function.

The Print Donor Documents screen gets displayed.

  1. Click [Donor Lookup] to select the donor. Press [F8] in the or click [Execute Query] or [Search] to view the donor details.  

  2. On selecting the Donor ID, click [Execute Query] or press [F8] to view the donor details.

    The following details are displayed:

    1. Age – The Age of the donor recorded using the ‘Donor Registration’ function gets displayed.

    2. Sex – The Gender of the donor recorded using the ‘Donor Registration’ function is displayed.

    3. Blood Group & Rhesus – If the donor has donated blood earlier, the details of blood group and Rhesus factor of the donor recorded using the ‘Pre Donation’ function are displayed.

    4. Last Donation Date – If the donor has donated blood earlier, the date and time of the donation is displayed.

    Donations

    You can view the donation details, if the donor has donated blood earlier. The units cancelled are highlighted using color and the units for which components are to be prepared, or added to stock or not added to stock are highlighted using color.

    1. Donation Date – The Date of Donation recorded using the ‘Record Donation function is displayed.

    2. Donation Type-Process - The list displays the type of donation or process recorded using the ‘Record Bleeding ‘function. The valid values are ‘Autologous’, ‘Allogenic’ or ‘Blood Letting’.

    3. Unit Number – The Unit Number of the blood recorded using the ‘Unit Screening’ function is displayed.

    4. Product – The Product Code of the blood unit is displayed.

    5. Description – The Description of the product is displayed.

  3. Click [Print Questionnaire] to print the Questionnaire. The questionnaires recorded using Physical/Questionnaire setup function in the Administration section gets printed.

  4. Click [Print Donor Card] to print the Donor ID card.

  5. Click [Print Consent Form] to print the consent form of the selected donor.

  6. Click [Print Consent Donor Age Exception] to print the Consent form for the donors under the Age Exception Category set in the BD Parameter function. This button is displayed only if the donor’s age falls in the Age Exception Category.

  7. Click [Unit Details] to view blood unit details. The [Unit Details] is enabled, only if the donor has donated blood earlier.

  8. Click [Antibodies] to view the details. This button is enabled, if antibody details are recorded for the donor during the screening.

  9. Click [Antigens] to view the details. This button is enabled, if antigen details are recorded for the donor in the Record Donor/Unit Antigens function.

  10. Click [Donor Notes] to view the details. This button is enabled, if donor notes details are recorded for the donor. Donor Notes are recorded in the ‘Record Donor/Unit Antigens’ function.

  11. Click [Print Donor History] to print the selected donor’s blood donation history.
