Record Donor Awards

From this function you can Record an award for a donor. You can View and Print the award details of a donor.

The Record Donor Awards function enables you to record awards for the donors. An award is a Certificate of Appreciation issued to the donor at the end of blood donation. You can also print award details, view awards and view the donor history.

Recording Donor Awards

From Transactions > Blood Donor, double-click the Record Donor Awards function.

The Record Donor Awards screen gets displayed.

  1. Click the Calendar function to select the donation date. The current system date is displayed, by default.

  2. Enter the number of Donations made to search for the donor records. You can search for all the donations by leaving this field value as blank.

  3. Click [Display] to view the donor records satisfying the specified query criteria.

    1. Donor ID – On selecting the donation date range, the Donor ID's within the selected search criteria are displayed.

    2. Name – The corresponding donors’ names get displayed.

    3. Number of Donations – The Number of Donations made by the donors is displayed.

    4. Awards – The number of Awards recorded for the donors is displayed.

    5. Description – The Descriptions of awards recorded for the donor get displayed.

  4. Select the check box ‘Select All’ to select all the donors and record awards. Select the check box for a specific donor to record the award for the individual donor.

  5. Click [Print] to print the donor list.

  6. Click [View Award] to view the existing award details. This button is enabled only if the donor has received more than one award.  

    The following details are displayed:

    1. Award Code – The Award Code entered for the selected donor is displayed. The values defined in Awards function get listed.

    2. Description - The Award Code description entered for the selected donor is displayed.

    3. Award Date – The date when the Award is given to the donor is displayed.

    4. Remarks – The Remarks are displayed, if any.

    5. [Close] – Click [Close] to close the window.

  7. Click [Record Award] to record an award for a donor. <Related Field Description>

  8. Click [Donor History] to view donor history with the unit details.

  9. Click [Save] to save the record or [Cancel] to cancel the transaction.
