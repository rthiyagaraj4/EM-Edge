Registration for Screen Units

From this function, you can Register the blood samples sent for screening from the Blood Donor and Transfusion Management modules.

The Registration for Screening Units function is used to directly register the blood units sent to the laboratory for screening purpose from the Blood Donor or Blood Transfusion Services modules. The upper part of the screen displays the units pending for registration. On selecting a unit, the lower part of the screen displays the investigations ordered for the sample. You can register more than one blood unit at a time.

Different color indicators are used to differentiate between the blood units that are sent from Blood Donor and Blood Transfusion Services modules:

- BD units will be displayed in pick colour.

- BT units will be displayed in green colour.

Registration for Screen Units

From Transactions > Blood Screening menu, double-click the Registration for Screen Units function.

The Registration for Screen Units screen gets displayed.

  1. The default Department name is displayed, by default. Select a Department from the list to register the units from any other department.

  2. The Registration for Screen Units screen displays the units that are pending for registration. <Related Field Description>

  3. Select the ‘Register?’ check box against the unit number details.

  4. Click [Register Units] to register the selected registered units

  5. Select ‘Register All’ check box to select all the register units.

  6. Click [Register Units] to register the selected unit number.

  7. A confirmation message is displayed on successfully registering selected blood unit.
