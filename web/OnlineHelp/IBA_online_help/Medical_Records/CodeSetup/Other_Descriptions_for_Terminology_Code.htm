Other Descriptions for Terminology Code 

From this function, you can Define other terminology description for primary terminology code against ICD10.

This function allows you to define ‘other terminology description’ for a primary terminology code against ICD10. You can map maximum ten ‘Other Descriptions’ with one Primary Terminology Code.

Creating Other Descriptions for Terminology Code

From MR Code Setup > MR Terminology Setup, select the Other Descriptions for Terminology Code function.

The Other Descriptions for Terminology Code screen gets displayed.

  1. By default, the value 'ICD10' gets displayed in the Terminology Set drop-down list.

  2. Select the Primary Diagnosis Code from the drop-down list. This field is mandatory.

  3. Click [Search]. The Other Terminology Descriptions column appear with ten text fields to enter description.

  4. By default, all the 10 Descriptions field shall be blank for the all the Primary Terminology Code. Enter the description in the text box in alpha numeric format with 100 characters maximum. Special characters are allowed.

  5. Enter in each and individual text box field. You can map 10 ‘Other Descriptions’ with one Primary Terminology Code.

  6. After entering the description, click [Apply]. The entered descriptions get mapped with one Primary Terminology Code.
