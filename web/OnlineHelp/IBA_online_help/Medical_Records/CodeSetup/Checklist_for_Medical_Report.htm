Checklist for Medical Report

From this function you can Create the Checklist for Medical Reports and Query for the existing checklist records.

This function allows you to create checklists for medical reports to be used in Medical Reports function. You can query about the created checklist records.

Creating Checklist for Medical Report

From MR Code Setup, select the Checklist for Medical Report function.

The Checklist for Medical Report screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the checklists created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing checklists for medical report.
