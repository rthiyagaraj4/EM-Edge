Medical Events

From this function you can Record Medical Events for the patient.

Recording Medical Events

From the CA Transactions > Clinician Access > Desktop Menu > select required patient, Patient Chart > Other Clinical Event Recording, select the Medical Events function.

The Medical Events screen gets displayed.

  1. Any Medical Events already recorded for the selected patient are displayed in the top part of the screen.

    1. Event Type - The recorded Event Type for the patient is displayed. These values are defined in the function 'Medical Event Type'.

    2. Description - The description of the Event Type is displayed.

    3. Start Date - The Start Date of the recorded medical event is displayed.

    4. End Date - The End Date of the recorded medical event is displayed.

    5. Confirmed - Confirmation of the recorded medical event is displayed.

  2. Enter the details in the mandatory (*) fields to record a new Medical Event for the patient. <Related Field Description>

  3. Click [Record] to record the Medical Event or [Clear] to clear the entered data. Click [Cancel] to exit from the screen.

  4. On successfully recording the Medical Event, select the Event Type hyperlink to modify details.. The medical event details are displayed in the lower part of the screen. You can record only the End Date and Event Details of the Medical Event if confirmed.
