Patient for Inactive

From this function you can Mark Patient ID as Inactive.

This function allows to mark the Patient ID as Inactive. The system considers only patient records who does not have case of MLC, OSCC, Post Mortem, Blood Donor and Blood Transfusion, no visits in the defined parameter duration for specialty.

 

If you search the patient ID marked as inactive in the transaction, the system will not list the inactive patient. But the system allows you to search for Inactive patient ID’s in the below functions:

  • Register Visit

  • Admit Patient

  • Register Attendance

  • Schedule Appointment

  • Admission Booking

  • Change patient Details

  • View Patient Details

 

The following criteria is considered before marking the Patient ID's as Inactive.

  • If encounter is open, the system will not allow you to mark EMR status as Inactive.

  • The date Patient is checked out/discharged, will be considered to mark EMR status as Inactive.

  • The encounter Discharge /Checked out date will be considered to mark EMR status as Inactive.

  • The Past and Future Appointment/Referral/ Booking will not be checked while listing these patients in the transaction screen and marking as Inactive.

Inactivating Patient IDs

From the MR Transactions, select the Patient for Inactive function.

The Inactive Patient Details screen gets displayed.

  1. Enter the Patient ID or select it using the lookup.

  2. Select the From Date by clicking the calendar icon. This field is mandatory.

  3. Click [Search]. The patient records matching the search criteria appear.

  4. Select the check box against the patient record to be made inactive and click [Apply]. The patient ID status gets marked as inactive.
