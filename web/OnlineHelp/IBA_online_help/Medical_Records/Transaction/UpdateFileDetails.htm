Update File Details

Using this function you can Record or Update the old file number of patients.

This function consists of new fields like Patient ID, File number (System generated File number) and Old File Number to search the patient file and update the Old File Number. You can either use one of the fields to search the records.

 

The “Old File Number” of a patient will be displayed as a hyperlink in the Old File No column. If there is no old file number available for the patient, a hyperlink “Add” will be displayed, clicking on which you can add an old file number.

Updating File Details

From the MR Transactions, select the Update File Details function.

The Update File Details Screen gets displayed.

  1. Click Patient ID [?] and select a patient ID from the look up, or enter the current or old file number of the patient in the respective fields.

  2. Click [Search]. The patient details get displayed.

If a file number is yet to be updated for the patient, the hyperlink “Add” is displayed under the column Old File No. Else, the previously updated number is displayed as hyperlink.

 

  1. Click the “Old file No” hyperlink. The Old File No webpage dialog is displayed.

  2. Enter the new value for the Old File No in the webpage dialog and click [Save]. The old file number is updated for the selected patient.
