Active Problems List

From this function you can Query all the active diagnoses recorded for the patient across encounters in the facility.

The active diagnoses are problems that patient is suffering from currently and not yet cured or marked as ‘Closed’. The details populated in this function are the onset date of the problem (since when the patient started experiencing the problem), the actual diagnosis (problem) itself, the ICD coding for the recorded active diagnosis, the coding scheme that is used by the facility to record diagnosis and an identifier to show if the diagnosis was the primary reason for the encounter.

Querying for Active Problems List

From the MR Queries, select the Active Problems List function.

The Active Problems List screen gets displayed.

  1. Enter the search criteria to search for a specific record.

  2. Click [Search] to select a specific record or [Clear] to clear the records displayed in the result area.

  3. On selecting the required patient record, the diagnosis for the selected patient gets displayed.

  4. Click '+' hyperlink to view the Encounter details of the selected patient.
