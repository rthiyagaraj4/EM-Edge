Diagnosis Category

From this function you can Define the diagnosis categories.

This function is used to create diagnosis categories. The diagnosis categories created and enabled in this function will appear in the Diagnosis Category field in Record Diagnosis function. This is applicable only for users in Taiba.

Defining Diagnosis Category

From the Code Setup > MR Terminology > Setup, select the Diagnosis Category function.                     

The Diagnosis Category screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. After entering the required details, click [Apply] to save the changes done or [Reset] to clear the entered details.

  3. Click [Query] to search for existing records.
