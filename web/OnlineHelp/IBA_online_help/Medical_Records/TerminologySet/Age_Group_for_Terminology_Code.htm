Age Group for Terminology Code

From this function you can Create an association of Terminology Codes to an Age Group and Query for a desired Age Group for Terminology Code record.

The Age Group for Terminology Code function is used to limit relevant Terminology Codes to certain Age Groups. You can associate Terminology Codes applicable to a particular Age Group.

Creating an association of Terminology Codes to an Age Group

From MR Code Setup > MR Terminology Setup, select the Age Group for Terminology Code function.

The Age Group for Terminology Code screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to display Terminology Codes associated to the selected Terminology Set. More option to search for Terminology Codes are provided in alphabetical order. On clicking any of the alphabet hyperlinks, the related Terminology Codes are displayed. Click 'Others' to display Terminology Codes beginning with numbers or other characters other than alphabets.

  3. Select the check box against the required Terminology Codes to associate them to the Age Group. Some check boxes appear selected and disabled as the Terminology Codes are already associated to the Age Group.

  4. Click [Apply] to save the created Age Group for Terminology Code record or [Reset] to clear the entered data.

  5. Click [Query] to search for existing Age Group for Terminology Code records.
