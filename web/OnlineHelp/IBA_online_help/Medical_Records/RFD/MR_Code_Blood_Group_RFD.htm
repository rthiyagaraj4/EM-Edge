Enter the following details to create a Blood Group record:

 

  1. Code - Enter Blood Group Code in alphanumeric format of four characters maximum length. A Blood Group Code is used to identify the Blood Group. The code has to be unique and cannot be changed after creation. This field is mandatory.

  2. Long Description - Enter the long description of the Blood Group in alphanumeric format of 40 characters maximum length. This field is mandatory.

  3. Short Description - Enter the short description of the Blood Group in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Indeterminate - Select the check box if pooling of blood units with different blood groups is allowed or clear the checkbox if blood group is not allowed. You can select only one blood group with intermediate as ‘Yes’.

  5. Enabled - Select the check box to enable the record or clear the check box to disable the record.
