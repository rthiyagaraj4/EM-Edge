The following details are displayed:

 

  1. Registration No - The Request ID of the Medical Report is displayed as the Registration Number.

  2. Registration Date/Time - The date and time of placing the request for Medical Report is displayed.

  3. Click ‘View Encounter Details’ hyperlink to view encounter details.

 

Applicant Information

The Applicant Information as recorded in the 'Medical Report Request' screen is displayed and disabled. The fields cannot be edited.

  1. Click [Date Details] to view the Request Date, Application Completion Date, Forwarded Date, Prepare Date and Received Date. Click [Close] to exit from the Date Details screen.

  2. Click [Charges Details] to view the charge details of the Medical Report as recorded. The charge details are displayed as read-only text. Click [Close] to exit from the Charge Details screen.

  3. Click [Applicant Address] to view the address details of the applicant.  Click [Close] to exit from the Contact Details screen.

  4. Click  [Check List] to view the check lists in read-only mode.  

 

Enter the following details:

  1. Report Type - The type of report as selected in the 'Forward Medical Report Request' screen is displayed and disabled.

  2. Department/Specialty - The Department or Specialty as selected in the 'Forward Medical Report Request' screen is displayed and disabled.

  3. Remarks - The Remarks are displayed as recorded which can be updated.

  4. Collect Date - The Collect date as recorded in the 'Status of Medical Report Request' screen is displayed and disabled.

  5. Prepare Date - The date and time of when the report was prepared in the Clinician Access module is displayed as read-only text.

  6. Received Date - The date and time when the medical report is received is displayed as read-only text.

  7. Delivered Date - Enter the date or select the date of when the received Medical Report is delivered to the requestor using the calendar function. This field is mandatory.

  8. Deliver/Dispatch mode - Select the dispatch mode of the Medical Report from the list of valid values. The values in the list of values are 'Personal' and 'Others'.

  9. Date posted - Enter the date of when the report is posted in dd/mm/yyyy hh:mm format or select the date using the calendar function. If 'Personal' is selected as the 'Deliver/Dispatch Mode' this field is disabled.

  10. Forwarded  Practitioner – The name of the practitioner to whom the request is forwarded as selected in the ‘Forward Medical Report Request’ screen is displayed and disabled.

  11. Prepared by Practitioner -  The name of the practitioner who prepared the medical report is displayed.

  12. Requested Practitioner –  The name of the practitioner who requested for the medical report of the patient is displayed as read only text.

  13. Forwarded HOD/Co-ordinator - The name of the HOD/Coordinator to whom the request is forwarded is displayed.

 

[Delivery/Dispatch Add] is enabled only if the 'Deliver/Dispatch Mode' field is selected. On clicking this button a contact details screen gets displayed. The contact details of the patient as recorded while registering the patient is displayed. The displayed fields can be updated.

  1. Section/File No - Enter the section number in numeric format of 20 digits maximum length. If 'Personal' is selected as the 'Deliver/Dispatch Mode', this field is disabled.

  2. Report Taken By - If the 'Deliver/Dispatch Mode' is selected as 'Others', this field is disabled. If the 'Deliver/Dispatch Mode' is selected as 'Personal', the name of the applicant is displayed as recorded which can be updated.

  3. Citizen ID - Enter the Citizen ID of the Patient. This field is disabled on selecting 'Others' in the field 'Deliver/Dispatch Mode'.

  4. Other Alt Type - Select the Other Alt Type of the Patient from the list of valid values. This field is disabled on selecting 'Others' in the field 'Deliver/Dispatch Mode'.

  5. Other Alt ID - Enter the Other Alt ID of the Patient. This field is disabled on selecting 'Others' in the field 'Deliver/Dispatch Mode'.

Click [View Medical Report] to view the Medical Report prepared.

Click [Previous Delivered Medical Report] in the Medical Report Request screen to view the details of Medical Reports already delivered for the patient.
