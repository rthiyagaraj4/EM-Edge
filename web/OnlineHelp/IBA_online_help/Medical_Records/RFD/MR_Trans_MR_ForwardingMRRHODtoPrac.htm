Forwarding Medical Report Request from HOD to Practitioner

From the MR Transactions > Medical Reports, select the Status tab.

The Status tab of the Medical Report screen gets displayed.

  1. Select the Medical Report Status as ‘Forwarded to HOD’ and click [Search] to display all Medical Report Requests or enter the search criteria and click [Search] to select the required completed Medical Report Request.

  2. Click the Request ID hyperlink to invoke ‘Forward to Practitioner’ screen and forward the Medical Report Request to practitioner.

  3. The Registration No, Registration Date/Time, Applicant Information as recorded in the 'Medical Report Request' screen is displayed and disabled. The fields cannot be edited.

  4. You need to enter the Status value i.e, accept or reject the request. Based on this selection, the remaining fields are enabled or disabled.

 

Accept Status

When you select the Status value as ‘Accept’, the following fields are enabled while remaining fields are disabled.

  1. Remarks – Enter remarks if any.

  2. Forwarded Practitioner – Enter the name of the Practitioner to whom the request is forwarded or select the Practitioner from [?] function.

  3. Acknowledged by HOD/Coordinator – Enter the name of the HOD/Co-ordinator who has to acknowledge the request or select using [?] function. This is a mandatory field.

  4. Inform to Practitioner – Enter the name of the practitioner to be informed or select using [?] function. This is a mandatory.

  5. Accept Date - Displays the current system date/time.

  6. Forward Practitioner – Enter the Forward Practitioner or select using [?]. If this field is recorded, on completion of transaction the Medical Request Status changes to ‘Forwarded to Practitioner’.

In case, the Forwarded Practitioner field is not recorded, on completion of transaction Medical Request Status changes to ‘Accepted Request’ when the transaction is complete. You had to reopen the Request in ‘Accepted Request’ status. The Request opens in ‘Forward to Practitioner’ screen where you can record the ‘Forwarded Practitioner’ and complete the transaction.

• When the Status is selected as ‘Accept’ and Forwarded Practitioner is not recorded, then the Medical Request Status is ‘Accepted Request’.

• When the Status is ‘Accept’ and Forwarded Practitioner is recorded, then the Medical Request Status is ‘Forwarded to Practitioner’.

Reject Status

When you select the Status value as ‘Reject’, only ‘Remarks’ is enabled and the remaining fields display the already recorded information if available in disabled state.

  1. Remarks – Enter the reason for rejecting the request.

  2. The 'Accept Date' label changes as 'Reject Date' label and displays the current system date and time.

On completion of transaction, the Medical Request status changes as ‘Returned Request’.

 

  1. Click [Record] to save forwarding the Medical Report Request to a Practitioner or [Close] to exit the ‘Forward to Practitioner’ screen.
