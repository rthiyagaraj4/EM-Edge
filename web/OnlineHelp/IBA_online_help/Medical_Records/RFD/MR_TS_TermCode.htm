Enter the following details:

 

  1. Terminology Set – Select the Terminology Set from the list of valid values. The values defined in the ‘Terminology Set’ master are displayed. This field is mandatory. On selecting the Terminology Set, the codes defined for Classification Level 1 are displayed. Enter the details under header Level 1 at the bottom of the screen to add more Codes to Level 1.

  2. Search – Specify the Search text in text box and select the required value from the list. The values listed are Starts With, Contains and Ends With. Click [Search] to retrieve Terminology Codes of the selected Terminology Set based on the entered criteria.

  3. On selecting the Terminology Set, the number of classification levels of the selected Terminology Set is displayed next to the ‘Search’ field as read-only text as defined in the Terminology Set master.

  4. On selecting the Terminology Set, the first level of the Terminology Codes is displayed, by default. More option to search for Terminology Sets are provided in alphabetical order. On clicking any of the alphabet hyperlinks, the related Terminology Codes are displayed and hyperlinked.

  5. On selecting the Terminology Set, the number of Classification Levels with their respective headings and the code length for the classification level is displayed and hyperlinked.

  6. On selecting a particular Terminology code, it appears in the space provided under the header ‘Selected Classification levels’. Enter the details at the bottom of the screen to add Codes to Level 2 of the Terminology Set. The Codes recorded are displayed on the top part of the screen under the respective Level.

  1. Code – Enter a Code for the Terminology in alphanumeric characters of length as specified in the Terminology Set for that particular level. This field is mandatory.

  2. Long Description – Click the ‘Long Description’ hyperlink and enter the description for the Terminology Code. Click [Close] after entering the description. This field is mandatory.

  3. Short Description – Enter the short description in alphanumeric format of 100 characters maximum length. This field is mandatory.

  4. Type – Select the Type as ‘Procedure’ or ‘Diagnosis’ for the selected Terminology Code.

  5. Selectable – Select the check box if the Terminology Code has to be hyperlinked and selected in the Clinician Access module.

  6. Enabled – Select the check box to enable the Terminology Code.

  7. Sensitive - Select the check box to mark the Terminology Code as Sensitive. This field is enabled if applicable for the Terminology Set.

  8. High Risk Condition – Select the High Risk Condition to associate it with the Diagnosis Code. This field is enabled only if the Diagnosis Code is selected for configuration and Sensitive check box is selected.

  9. Notifiable - Select the check box to mark the Terminology Code as Notifiable. This field is enabled if applicable for the Terminology Set.

  10. Code Indicator - Select the Code Indicator from the list. The listed values are Dagger, External Cause, Asterisk and Consequence. The Code Indicator is applicable only for the ICD Terminology Set.

  11. Dagger Independent - This check box is displayed on selecting the Code Indicator value 'Asterisk'. Select the check box to identify the Terminology Code as Dagger Independent. If the check box is selected, you can record the code and cannot mark the diagnosis as principle diagnosis.
