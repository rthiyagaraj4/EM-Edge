Enter the following details:

  1. Code - Enter the Code in numeric format of two digits maximum length. The code has to be unique and cannot be changed after creation. This is a mandatory field.

  2. Long Description - Enter the Long Description of the Diagnosis Classification in 30 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the Short Description of the Diagnosis Classification in 15 characters maximum length. This is a mandatory field.

  4. Classification Type - Select the Type from the list. The list item contains External Cause, Complication, Principle Diagnosis and Co Morbidity.  

  5. Enabled - Select the check box to enable the Diagnosis Classification or clear the check box to disable it.

  6. Term Set - Select the check box against the required Term Set to associate it to the Diagnosis Classification.

  7. Default – Select the check box against the required Term Set to default the prinicipal diagnosis to the term set.  This check box is visible only if the ‘Classification Type’ is selected as ‘Principal Diagnosis’

You can map many ‘Principal Diagnosis’ to a ‘Term Set’ but only one principal diagnosis can be defaulted to a term set. If you try to default another principal diagnosis to same term set, an error message is displayed on selecting the Default check box.
