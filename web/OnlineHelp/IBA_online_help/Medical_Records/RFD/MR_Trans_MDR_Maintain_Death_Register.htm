Maintain Death Register

Header Section

  1. Attending Practitioner – The Attending Practitioner name captured during patient admission is displayed. You can change the name of the Attending Practitioner. If the Attending Practitioner is changed during the course of the treatment, the latest Attending Practitioner is displayed in this function. This is a mandatory field.

  2. Certified Practitioner - The Certified Practitioner who has certified the patient as ‘Deceased’ is displayed. You can change the name of the Certified Practitioner. The Name of the latest certified practitioner is printed on the Death Certificate. This is a mandatory field.

  3. Designation – The designation of the Certified Practitioner is displayed.

  4. Citizen ID Number – The Citizen ID Number of the Certified Practitioner is displayed and disabled.

  5. Registration No – The Registration number of the Certified Practitioner is displayed and disabled.

  6. Brought in Dead – The check box is selected and disabled, if the patient is brought in dead, or the check box is cleared and disabled. The brought in dead details are recorded for a patient in Register Attendance function in Accident and Emergency module.

  7. Length of Stay – The patient’s duration of stay at the hospital is displayed and disabled. The system calculates the patient’s duration of stay from the encounter registration time and the deceased date & time irrespective of the patient class. If the 'Brought in Dead' check box is selected, the values are displayed as zero.

  8. Deceased date and time – The deceased date and time recorded in the Prepare Discharge Advice function in Inpatient and Day Care Management module is displayed.

 

Diagnosis

Cause of Death

The cause of death details are displayed. The number of cause of deaths records are determined by the value entered in the ‘Max Primary Diagnosis to indicate cause of Death’ field in MR Parameter function. The diagnosis is considered as a cause of death, if the value in the ‘Move To’ field is selected as cause of death. Any newly added diagnosis in this function is also considered as the cause of death. It is possible to add new diagnosis from this function based on the number of cause of death defined in the MR Parameter function. It is necessary to identify at least one diagnosis as cause of dearth or the message ‘Recording one cause of Death is mandatory’ is displayed.

  1. Select - Select the check box against the required diagnosis categorize the Diagnosis.

  2. Code – The code of the diagnosis is displayed.

  3. Description - The description of the diagnosis is displayed.

  4. Onset Date – The Onset date of the diagnosis is displayed, if it is recorded in the Record Diagnosis function. You can update the date entered.

  5. Duration of Illness - The Duration of Illness is automatically displayed based on the onset date.

  6. Order – The order of the diagnosis is sequenced in the ascending order, by default. Select the order of cause of death from the Order field. Code 1 is considered, as the Primary case of death. You can change the order and click [Sequence] to rearrange the diagnosis in the sequential order.

Other Disease/Complication which caused Death

The cause of Other Disease/Complication which caused Death is displayed. The number of Other Disease/Complication which caused Death is determined by the value entered in the 'Max other disease or complication to indicate cause of Death' field in MR Parameter function.

  1. Select - Select the check box to select the diagnosis.

  2. Code – The code of the diagnosis for Other Disease/Complication which caused Death is displayed.

  3. Description - The description of the diagnosis for Other Disease/Complication which caused Death is displayed.

General Diagnosis List

The General Diagnosis List is displayed. All the diagnoses are displayed under the General Diagnosis List when the Maintain Death Register function is accessed for the first time.  The diagnoses you create in the prepare discharge advice function is categorized under this list.

  1. Select - Select the check box to select the diagnosis.

  2. Code – The code of the diagnosis for General Diagnosis List is displayed.

  3. Description - The description of the diagnosis for General Diagnosis List is displayed.

  1. Move to – Select the 'Select' check box against the diagnosis and select the value from the list. The listed values are: Cause of Death, Other Disease/Complication which caused Death and General Diagnosis List.

  2. Click [Add new Diagnosis] to add a new diagnosis to the patient. On clicking [Add new Diagnosis], 'Record Diagnosis' screen is displayed. The diagnosis you create are considered as the cause of death. If the number of cause of deaths defined in the MR Parameter is equal to the number of cause of deaths marked in the function, an alert message ‘You cannot add new diagnosis as per MR parameter limit’ is displayed.

  3. Click [Sequence] to change the sequence of the order of the cause of death. You can change the order and change the sequence before finalizing the death register.

Remarks

  1. Remarks – Enter the remarks in alphanumeric format of 200 characters maximum length. The diagnosis details entered by the officer is entered in this field.

Female Patient Status

This section is displayed only for the female patients. If the pregnancy details are captured for the patient in the Clinical Care Management module and Inpatient and Day Care Management module, the details are displayed.

  1. Not Pregnant – Select the option, if the patient is not pregnant. On selecting this option, the field 'Delivered less than 6 weeks' is displayed.

    • Delivered less than 6 weeks – Select the check box if the deceased patient has recently delivered (that is in the postnatal period).   

  2. Pregnant - Select the option, if the patient is pregnant. On selecting this option, the fields Gestational Age and Delivered less than 6 weeks are displayed.

    • Gestational Age (Weeks) – Enter the Gestational Age, the gestation age field has the units as Weeks or Days depending on the values set in MP Parameter function in Master Patient Index module.

    • Delivered less than 6 weeks – Select the check box if the deceased patient has recently delivered (that is in the postnatal period).   

     

  3. Details Unknown - Select the option if the details of pregnancy are unknown.

MLC Details

  1. MLC – Select the check box, if the death is related to Medico Legal Case. The check box is selected if the MLC check box is selected in Register Attendance function in Accident and Emergency module or in Prepare Discharge Advice function in Inpatient and Day Care Management module.

  2. Police Report No – Enter the Police Report Number in alphanumeric format of 20 characters maximum length. The Police Report Number is displayed if it is recorded in Register Attendance function in Accident and Emergency module or in Prepare Discharge Advice function in Inpatient and Day Care Management module.

  3. Station – Enter the Police Station details in alphanumeric format of 20 characters maximum length. The Police Station details are displayed if they are recorded in Register Attendance function in Accident and Emergency module or in Prepare Discharge Advice function in Inpatient and Day Care Management module.

  4. ID - Enter the Police Station ID in alphanumeric format of 20 characters maximum length. The Police Station ID is displayed if it is recorded in Register Attendance function in Accident and Emergency module or in Prepare Discharge Advice function in Inpatient and Day Care Management module.

  5. Click [Audit Trail] to view the modification details of the diagnosis of the deceased patient. This button is not displayed, when you invoke the Maintain Death Register function from Prepare Death Register function in Inpatient and Day Care Management module.

  6. Click [Save] to save the first stage of Maintain Death Register, if the ‘Single Stage Death Registration Applicable’ check box is not selected in MR Parameter function. This  button is not displayed if the ‘Single Stage Death Registration Applicable’ check box is selected.

  7. Click [Finalize] to save the final stage of Maintain Death Register, if the ‘Single Stage Death Registration Applicable’ check box is not selected in MR Parameter function. If the record is finalized, you cannot modify the details.

  8. Click [Print] to print the Death Certificate of the deceased patient. You can print the Death certificate using the print option only after the cause of Death is entered in the function. The death certificate is printed after saving and finalizing the records.

  9. Click [Clear] to clear the details entered. On clicking [Clear], an alert message ‘Do you want to clear the details?’ is displayed. Click [OK] to close the screen or [Cancel] to abort the process.

  10. Click [Close] to close the screen without saving any of the details entered.

The Additional Fields are displayed, if the Additional Fields are defined for the Header Section, Remarks and MLC Details section in the MR Parameter function.
