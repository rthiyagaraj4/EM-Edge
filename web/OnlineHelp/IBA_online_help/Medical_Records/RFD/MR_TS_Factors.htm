Enter the following details:

 

  1. Code – Enter the code in alphanumeric format of 10 characters maximum length. This field is mandatory.

  2. Long Description – Enter the long description in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description – Enter the short description in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Type – Select the Type from the list of valid values. The values are predefined. The values listed are Defining Characteristics, Related Factors, Related and Risk Factors and Risk Factor. This field is mandatory.

  5. If Defining Characteristics is selected, the factor defined is considered as a defining factor for the diagnosis.

  6. If a Related Factor is selected, the factor being defined is considered as a related factor for the diagnosis, as it contributes to the occurrence of a problem.

  7. If Related and Risk Factors is selected, factor defined is considered as related and risk factor for the diagnosis, as it can contribute to the occurrence of the diagnosis and increase the probability of developing the problem.

  8. If Risk Factor is selected, the factor defined is considered as a risk factor for the diagnosis, as it increases the probability of developing the problem.

  1. Enabled - Select the check box to enable the factor.
