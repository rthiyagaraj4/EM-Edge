Enter the following details to view the death register:

  1. Citizen ID - Enter the Citizen ID in alphanumeric format of maximum length as specified in the MP Parameter. The field title(s) is displayed based on the Alternate ID type(s) set in MP Parameter. Based on the MP Parameter Setup, the system displays the Alternate ID Type description(s). The Alternate ID gets displayed automatically on entering Patient ID of the registered patient. For a previously unregistered patient, you have to enter the Alternate ID.

  2. Other Alt Type - Select the Other Alt Type from the list of values. The listed values are Birth Certificate, Passport, and Mother's PMI, PMI, etc.

  3. Other Alt ID - Enter the Other Alternate ID in alphanumeric format of maximum length as specified in the MP Parameter.

Enter the values in the fields 'Previous HN', 'Government No', 'Driving Licence Number' and 'Expatriate No' as set in the MP Parameter under 'Alternate ID Checks' in the Master Patient Index module. The number of fields and their order displayed in the screen may vary depending on the MP Parameter Setup.

  1. Patient ID - Enter the Patient ID in the alphanumeric format of 10 characters maximum length, select the Patient ID by clicking [?] function.

  2. Encounter Period - Enter the Encounter date range of the patient in date format of dd/mm/yyyy or select the date using the Calendar function.

  3. Encounter Type - Select the Encounter Type of the encounter from the list. The listed values are Daycare, Emergency, Inpatient, Outpatient and External.

  4. Deceased Period - Enter the Deceased date range of the patient in date format of dd/mm/yyyy or select the date using the Calendar function.

  5. Encounter ID - Enter the Encounter ID in numeric format of 12 digits maximum length.

  6. Order By - Select either 'Encounter Date' or 'Encounter Type' to display the search in the selected order.

The recorded death register details are displayed with the color code and the finalized death register details are displayed in the white color.
