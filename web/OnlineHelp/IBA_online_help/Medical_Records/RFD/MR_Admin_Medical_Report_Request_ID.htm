Enter the following details:

 

  1. Year – Enter the required year in the text field. This field is mandatory. You can enter only four digits.

  2. Start Serial No – Enter the beginning number based on which sequence number will be generated concatenation with year field. Only numeric values are allowed. You can enter up to 8 digits. This field is mandatory.

• Start Serial No should be greater than zero

  1. Max Serial No – Enter the maximum range the sequence can reach. Only numeric values are allowed. You can enter up to 8 digits. This field is mandatory.

• ‘Max Serial No’ should be greater than ‘Start Serial No’.

  1. Next Serial No – Enter the next serial no value in the text field. You can enter up to 8 numeric digits.
