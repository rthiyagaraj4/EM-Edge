Enter the following details:

 

  1. Event Type - Select the Event Type from the list of valid values. These values are defined in the ‘Medical Event Type’ function. This field is mandatory.

  2. Since – Enter the value in numeric format of two digits maximum length. On entering the value, select the period in Days, Months or Years in the list provided next to the entered value. This field specifies the number of Days, Months or Years prior to the current date, the recorded event had commenced for the selected patient.

  3. Start Date – On entering the since value and selecting the period in Days, Months or Years, the relevant start date is displayed. You can modify this date. The Start date can be entered directly in the format dd/mm/yyyy without entering the ‘Since’ value, or select the date using the Calendar function. This field is mandatory.

  4. Till – Enter the value in numeric format of two digits maximum length. On entering the value, select the period in Days, Months or Years in the list of values provided next to the entered value. This field specifies the number of Days, Months or Years from the start date, the recorded event was completed for the selected patient.

  5. End Date – On entering the till value the End Date is displayed. You can modify this date. The End Date can be entered directly in the format dd/mm/yyyy without entering the ‘Till’ value, or select the date using the Calendar function.

  6. Event Details – Enter the Event Details in alphanumeric format of 2000 characters maximum length. This field is mandatory.

  7. Confirmed – Select the check box to confirm the entry. The check box is cleared, by default.
