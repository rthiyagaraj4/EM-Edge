Enter the following fields:

1. Code - Enter Notifiable Setup Code in alphanumeric format of four characters maximum length. A Notifiable Setup Code is used to identify the notifiable medical condition. The code has to be unique and cannot be changed after creation. This field is mandatory.

2. Long Description - Enter the long description of the Notifiable medical condition in alphanumeric format of 40 characters maximum length. This field is mandatory.

3. Short Description - Enter the short description of the Blood Group in alphanumeric format of 15 characters maximum length. This field is mandatory.

4. Default Note Type – Select the default note type that you want to be associated for the notifiable medical condition setup from the drop-down list. All note types of the Note group ID *NOTIFORM will be fetched from Clinician access and displayed in the list.

5. Email Applicable – Select this check box, if you want email to be triggered when the corresponding type of note is generated.

6. Enabled - Select the check box to enable the record or clear the check box to disable the record.
