Receive File at MRD

From this function you can Receive the files issued to the other locations by the permanent File Storage Locations.

The permanent File Storage Locations can receive the files with the status ‘Returned’.  The receiving locations return file(s) back to the permanent File Storage Locations using the function Return to MRD.

The system validates access rights of the logged in user to receive files at MRD. The authorized user can invoke this function.

Receiving the File at MRD

From the FM Transactions, select the Receive File at MRD function.

The Receive File at MRD function screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Add] to view existing files of the patient to receive the file at MRD, [Show All Files] to view all the files from the selected FS Location to receive the file at MRD or [Clear] to clear the entered data.

  3. Click ‘+’ hyperlink to View Patient Details.

  4. Click ‘View Remarks’ hyperlink to display and view the Issue Remarks. Click [OK] to exit from the ‘Issue Remarks’ screen.

  5. Click ‘Return Remarks’ hyperlink to display and enter the Return Remarks. Click [OK] to exit from the ‘Return Remarks’ screen.

  6. Select the ‘Receive’ check box to receive the file at MRD. A unique color code is assigned that indicates the file status. File with the status as Returned or Outside is received. The color codes adopted to indicate file status are displayed at the bottom of the screen. The check box is selected by default for the status Returned and Outside.

    Files issued to locations outside the facility is received.

    Files issued within the logged in facility, with status as Returned is received.

  7. On selecting, click [Apply] to receive the desired file(s) at MRD or [Reset] to clear the entered data.

When the files are scanned manually or through a bar code reader, the message ‘Deficiency exists for the file. Do you want to proceed?’ is displayed, if the file is an Incomplete file. Click [OK] to list the files in the result page.
