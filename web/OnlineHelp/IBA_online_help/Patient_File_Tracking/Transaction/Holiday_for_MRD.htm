Holiday for MRD  

From this function you can Create and Query the Holiday for Medical Record Department (MRD).

This function allows you to create Holiday for Medical Record Department (MRD). You can query to view the created Holidays for MRD as well as to edit and delete the holidays created for MRD.

The file Return Date and Time calculation has to be recalculated if there is a holiday for a MRD location.

Creating Holiday for MRD

From the FM Transactions, select Holiday for MRD function.  

The Holiday for MRD screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Select]. The MRD Location for each date starting from Start to End Date gets displayed.

  3. Enter the reason for holiday for declaring holiday for that MRD location in the ‘Reason for Holiday’ column against the respective date. It is mandatory to enter the holiday reason.

  4. Select the check box against the Holiday set record.

  5. Click [Apply] to save the changes done or [Reset] to clear the entered data.

  6. Click [Query] to view as well as to edit the Holidays already created for the MRD locations.

  7. To delete any holiday record, select the check box against the record and click [Delete].
