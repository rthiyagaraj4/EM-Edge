Auto Track Out to OPD

From this function you can Issue the confirmed files from permanent FS Location to the respective location for which the patients have appointments for the specified date.

The files confirmed using the function 'Confirm OP Pulling List' are tracked to the respective locations using this function. On tracking out the files from the permanent locations, the receiving locations have to acknowledge the receipt of the files. File status are updated based on the auto-receipt Parameter Setup in the FM Parameter. If auto-receipt is selected, the file status is ‘O’ - Outside or ‘T’ In Transit.

Auto Tracking out to OPD

From the FM Transactions> Track Out to OPD, select the Auto Track Out to OPD function.

The Auto Track Out to OPD function screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Track Out] to issue the patient files to the respective locations or [Reset] to clear the enter data.

If the 'Auto Track out Upon Confirm Pulling List' check box is selected in FM Parameter Setup, on clicking the 'Auto Track Out To OPD' function displays a pop-up message ‘This function is not applicable as Auto Track out is happening upon Confirm Pulling List’.

If the 'Auto Track out Upon Confirm Pulling List' check box is cleared in FM Parameter Setup, on selecting Auto Track Out To OPD function, the Auto Track Out to OPD function screen is displayed.

On tracking the files, you can view the file status by invoking the query function ‘Files Current FS Location’.
