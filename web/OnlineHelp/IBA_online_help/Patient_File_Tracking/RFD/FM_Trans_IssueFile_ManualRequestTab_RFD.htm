Enter the following details to issue files against manual request:

 

  1. Facility – The Logged in Facility is displayed, by default. The list is populated with the names of all the facilities of the enterprise. You can select the required facility. This field is mandatory.

  2. Accountable Person - Select the person name using the [?] function. The list of user names displayed depends on the selected facility.  On selecting the name, the system evaluates the access rights of the requestor. If access rights for manual request are not defined for the selected person, an alert pop-up message gets displayed. If the user has necessary rights defined, enter the other details. On selecting the Accountable Person, the Designation and Primary Specialty details are displayed. This field is mandatory.

  3. Third Party Name - Enter the Third party details in the text area provided.

  4. Request Mode - Select the request mode from the list. The listed values are defined in Request Mode function.

  5. Reason for Borrowing - The narrations created in the ‘Narration’ function are listed. Select the appropriate reason for borrowing.

  6. Remarks – This field appears only when ‘Others’ value is selected in the ‘Reason for Borrowing’ field. Enter the remarks in alphanumeric format up to 100 characters except these four " '~` special characters. But you can enter these four special characters in the text box by copy and paste function. This is a mandatory field.

  7. FS Location - Select an appropriate file storage location from the list. File Storage Locations are populated depending upon the Accountable Person selected. Only those locations associated to the selected name are populated.  The selected location is the requesting location. This field is mandatory.

  8. Requesting - The field label is displayed based on the selected FS Location's Identity. For example, if the selected FS Location is a Clinic, the Clinic label is displayed. Enter the required location requesting for the file or select it using [?] function. The search screen displays locations linked to the selected FS Location. This field is mandatory.
