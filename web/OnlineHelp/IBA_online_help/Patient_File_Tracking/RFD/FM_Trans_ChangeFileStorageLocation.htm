Enter the following details to change File Storage Location:

 

  1. Terminal Digit - Enter the Terminal Digit in alphanumeric format of 10 characters maximum length.

  2. Permanent FS Location - Select the Permanent File Storage Location from the list. The listed locations are populated based on the user access rights. This field is mandatory.

  3. Patient ID - Enter the Patient ID in alphanumeric format of length as specified in MP Parameter or select the Patient ID using [?] function. This field is mandatory.

  4. File Type - On selecting the Patient ID, the File Type(s) associated to the patient is displayed. If more than one File Type is associated, select the required File Type from the list. On selecting the Patient ID, the default File Type associated in the File Type For FS Location is displayed. This field is mandatory.

  5. Volume No - The active volume number is displayed. Select the required Volume Number from the list. This field is mandatory.

  6. File Creation Date [From – To] - Enter the File Creation Date range in the format of dd/mm/yyyy format.

 

Change

  1. Permanent FS Location - Select ‘Permanent FS Location’ option to change the Permanent FS Locations of the file(s).

  2. Current FS Location - Select ‘Current FS Location’ option to change the Current FS Locations of the file(s).
