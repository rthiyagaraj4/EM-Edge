Enter the following details to generate the main and supplementary lists:

 

  1. Pulling List Type - Select the Pulling List Type from the list. The values listed are Main and Supplementary. The value ‘Main’ is displayed, by default. The Main list can be generated only once.

  2. Appt Date - The Appointment Date is displayed based on the FM Parameter Setup. This field is mandatory. In FM Parameter function, the value specified for ‘No of Days up to which to check for appointments in File Inquiry [ ]’ is considered for generation of pulling list. The current system Date and the parameter value are added to arrive at the appointment Date value in this function. The list is generated for the calculated Date.

  3. Gender - Select the Gender from the list. The values contained in the list are Male, Female, Unknown and All. The value 'All' is displayed, by default.

  4. Location - Select the Location from the list. The listed values are Clinic, Daycare Unit and Procedure Unit.

  5. Location Code - Enter the From and To Location Code in alphanumeric format of four characters maximum length or use [?] function to query the Location Codes. Specify the code range to limit the generation of pulling list. This field is optional. The Location Code must correspond to the selected location.

  6. Resource Class - Select the Resource Class from the list. The listed values are Practitioner, Equipment, Room and Other.

  7. Resource Type - Select the Resource Type from the list. The listed values correspond to the selected Resource Class. Specify the Resource Type range to restrict the generation of list.

  8. Resource ID - Enter the Resource ID in alphanumeric format of 15 characters maximum length or use [?] function to query the Resource ID. Specify the Resource ID range. The Resource ID must correspond to the selected Resource Type.

  9. Print Pulling Label - Select the Print Pulling Label check box to print the generated label. On generating the label, the label is routed to the specified Printer for printing.

  10. Print Identification Label - Select the Print Identification Label check box to print the Identification Label.

  11. Print Tel. No in Pulling List - Select the Print Tel No in Pulling List check box to include Telephone Number of the patients in the Pulling List. This field is enabled only when Print Pulling List is selected.

  12. Print Bar Code - Select the Print Bar code check box to print Bar Code. This field is enabled only when Print Pulling List is selected.

  13. Order By - This field is disabled if Print Pulling List check box is cleared. If Print Pulling List is applicable, select the list values. The values displayed are Terminal Digits and Appt. time (Appointment Time). The generated list is order based on the value specified. On selecting the required order by value, you can specify Ascending or Descending value from the list.

  14. Print Patient File Request Slip – Select the ‘Print Patient File Request Slip’ check box to print the Patient file request slip for Appointment. The Patient file request slip is printed on confirming the appointment.
