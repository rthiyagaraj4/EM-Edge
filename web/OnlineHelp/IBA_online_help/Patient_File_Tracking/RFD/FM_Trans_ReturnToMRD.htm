Enter the following details to Return File to MRD:

 

Return From

  1. FS Location Identity - Select the File Storage Location Identity from the list. The listed values are Clinic, Nursing unit, Practitioner, Department, Day Care Unit and Procedure Unit.

  2. FS Location – Select the FS Location from the list. The FS Locations are listed based on the FS Location Identity selected. This field is mandatory.

  3. Returning - Select the Returning place to return the Medical Record. The Returning legend corresponds to the FS Location Identity.

 

Criteria

  1. Patient ID - Enter the Patient ID in alphanumeric format of length as specified in MP Parameter or use [?] function to select the required patient. This field is mandatory.

  2. File Type - On selecting the Patient ID, the default File Type associated in the File Type For FS Location is displayed. The active Volume Number is displayed, default. This field is mandatory.  

  3. Volume No - The active Volume Number is displayed, default. Select the Volume Number from the list.
