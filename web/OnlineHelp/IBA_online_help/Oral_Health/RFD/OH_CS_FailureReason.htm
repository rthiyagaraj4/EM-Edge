Enter the following details to record a failure reason:

  1. Reason Code – Enter the Reason Code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Reason Description – Enter the description for reason in alphanumeric format of 200 characters maximum length. This is a mandatory field.

  3. Enabled – Select the check box to enable the record in the transaction functions. The check box is selected, by default.
