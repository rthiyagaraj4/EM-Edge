Enter the following details to define Tasks:

 

  1. Task Code – Enter the Task Code in alphanumeric format of 20 characters maximum length. A Task Code has to be unique and it cannot be changed once defined. This is a mandatory field.

  2. Task Description – Enter the Task Description in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  3. Link Catalog – Select the check box to associate a Task Code to an Order Catalog. The task codes associated to the selected Order Catalog get listed while placing an order in the Order Entry and Result Reporting module.

  4. Catalog Cross Reference – Click [?] function to select an Order Catalog. The Order Catalogs defined in the Order Entry and Result Reporting module and are associated to the Oral Health module get listed. The Catalog Cross Reference field is enabled only when the ‘Link Catalog’ check box is selected. Clear the check box and you cannot associate a Task Code to the Order Catalog. This is a mandatory field.

  5. Performing Location Applicable - Select the check box, if external performing location is valid for the selected task. On selecting the check box 'Note Type' is enabled for selection. Clear the check box, if external performing location is not valid for the selected task. If the check box is selected, options are enabled to record send out and acknowledged details of the external location where the order is sent and acknowledged. Option is also available to record the results received during registration.

  6. Note Type - The field is enabled for selection if the 'Performing Location Applicable' check box is selected. The Note Types that are defined as applicable for Oral Health in Clinician Access module are listed.

  7. Enabled – Select the check box to enable the record in the transaction functions. The check box is selected, by default.
