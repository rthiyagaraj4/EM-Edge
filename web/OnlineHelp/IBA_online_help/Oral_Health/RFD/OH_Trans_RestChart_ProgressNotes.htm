Enter the following details to record progression:

 

  1. Progression - Select a value from the list. The system-defined values are '1', 'Show Active', 'Show Closed', 'All' and 'Add New'. Click 'Add New' to record new progression details. Click 'Show Active' to view the active progression notes. Click 'Show Closed' to view the progression notes that are closed. Click 'All' to view all the progression notes recorded for the encounter.

  2. Specialty - Select a Specialty from the list. The Specialties associated to the facility are listed and the specialty of the current patient encounter is displayed, by default. If the progression is not applicable to any specialty,  selection of specialty is not mandatory while recording progression.

  3. Progress Note - Select a Progress Note from the facility.

  4. [Go] - On clicking [Go], the note is selected under Progression 1 and the note group, note type are grouped based on the date.

  5. Note Group - Click the Note Group hyperlink to view all the encounters recorded for the note group.

  6. Add New - Click the Add New to record a new note group.  

  7. Progression - Click the Progression hyperlink to view notes recorded across the encounter for a progression.

Add

  1. Filter By - Select the Note Type from the list. The value is set to 'Note Type' by default/

  2. Note Type - Select a Note Type from the list. This is a mandatory field.

  3. Note Group - Select a Note Group from the list. This is a mandatory field.

  4. Med/Anc Service - Click [?] function to select the service. This is a mandatory field.

  5. Date/Time - Click Calendar function to select the date and time. This is a mandatory field.

  6. Performed By - The name of the practitioner associated to the encounter is displayed. This is a mandatory field.

  7. Note Title - Enter the note title in alphanumeric format.

  8. Click [Hide Header] to hide the details.

  9. Click [Show Editor] to view the editor and record notes.

  10. Click [Show Both] to view editor and upload images.

  11. Available Images - Select an image from the list. The selected image is associated to the note.

  12. Click [Prev Notes] to view preview notes.

  13. Click [Print Preview] to view the details before printing.

  14. Click [Record] to record the details. The details are saved on clicking [Record].

  15. Click [Sign] to sign the notes. The notes that are signed is displayed in blue.

 

Search

Click Search tab to view the progress notes associated to the specific patient encounter.

The latest Progression Note is displayed, by default.

  • The notes can be viewed and accessed only by the members who have the Specialty same as that of the Progression. If the user does not belong to the specialty, an error message 'The access is limited to the practitioner of the progression specialty' is displayed.

  • A Note is posted to repository when it is signed.

  • You can record the outcome of a progression note and mark it as closed.

  • You can record additional details into a note that is closed.
