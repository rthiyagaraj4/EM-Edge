Enter the following details:

 

  1. Component – Select a Component from the list. The components, which are associated to the selected Chart ID during Chart Definition, get listed.

  2. Tooth Applicable – Select the tooth from the list. You can associate the component to the entire tooth or to selected tooth. The component has to be recorded for the associated tooth while selecting the Chart in Periodontal Charting.

  3. Required? – Select the check box, if the component has to be recorded for the selected tooth when the respective Chart is selected during transactions in Periodontal Charting. The check box is selected, by default.

  4. [Record] – Click [Record] to save the records. On selecting the values and clicking [Record], the recorded components and the tooth are displayed in the screen. On clicking [Record] a message gets displayed in the message bar of the screen indicating that the record has been saved successfully.
