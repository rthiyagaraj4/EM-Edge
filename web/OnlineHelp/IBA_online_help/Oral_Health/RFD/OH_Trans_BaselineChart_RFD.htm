Enter the following details to create a Baseline Chart:

 

  1. Numbering System – The Numbering System selected while defining the parameters in 'Setup Parameters For Facility' function in the Admin Setup gets displayed. You can select a different numbering system, if needed. On selecting a different numbering system and clicking [Refresh], a message ‘Do you want to keep this Numbering System as your default Numbering System?’ gets displayed. Click [OK] to continue or [Cancel] to continue using the existing numbering system. This is a mandatory field.

  2. Chart – Select a Chart from the list. The Charts defined using the function Chart Definition in the OH Masters get listed. The components associated to the selected Chart during Chart definition are applicable for the selected Baseline Chart. This is a mandatory field.

  3. Arch – Select an Arch from the list. You can record the status for the Upper and the Lower Arch. This is a mandatory field.

  4. Status – Select a Status from the list. The system defined values are Active, Completed, Closed and Marked Error. A chart has to go through all the statuses, if it has to be closed.

    The charts, which are completed or closed, cannot be marked as error. You can mark only an active chart as an error. Select the Status as ‘Mark Error’. A message gets displayed ‘Do you want to mark the chart as error?’ Click [OK] to mark the chart as an error or [Cancel] to maintain the status as active. A screen gets displayed on clicking [OK].

    The charts, which are completed only, can be closed. The charts, which are closed, appear in the Periodontal Chart Summary in Restorative Charting. Select the Status as ‘Closed’. A message gets displayed ‘Do you want to close the chart?’ Click [OK] to mark the chart as Closed or [Cancel] to maintain the status of the chart. A closed Chart gets displayed under the Baseline Charts (Closed) menu.

    You can mark the status of an active Chart as Complete. Select the Status as ‘Completed’ from the list. A message ‘Do you want to mark the Chart as completed?’ gets displayed. Click [OK] to mark the chart as complete or [Cancel] to maintain the status of the chart. A message ‘Chart has completed’ gets displayed on selecting [OK]. Click [OK] to save the chart as complete.

    You can record the details only in an active chart. The recorded details get displayed under the respective tooth number. Only active charts can be marked as an error or completed.

  5. [Refresh] – Click [Refresh] to refresh the screen. A new screen gets displayed based on the parameters selected.

  6. [Reset] – Click [Reset] to clear the data entered.

  7. Key To Charting - The components associated to the selected Chart are listed along with the minimum, maximum and the Red if>= values. The Red if>= value is marked in red and displayed within brackets.

  8. Tooth No – Select a Tooth Number from the list to record the components. The tooth range associated to the selected Chart using the function ‘Tooth Applicability for Chart’ in the OH Master function only are listed. This is a mandatory field.

  9. Components – Enter the value in numeric format to indicate the severity. You can enter the numeric value between the minimum and the maximum value recorded for the component. If the value is greater than or equal to the red if value, a message gets displayed indicating the range of values valid for the selected Component.

  10. [Record] – Click [Record] to record the tooth and the component details.

  11. [Reset] – Click [Reset] to reset the values entered.
