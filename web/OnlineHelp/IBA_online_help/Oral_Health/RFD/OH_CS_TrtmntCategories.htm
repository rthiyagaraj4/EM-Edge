Enter the following details to create a Treatment Category:

 

  1. Treatment/Procedure Category - Enter the Treatment/Procedure Category in alphanumeric format of 20 characters maximum length.  The Treatment/Procedure Category has to be unique and cannot be changed once defined. This is a mandatory field.

  2. Description – Enter the Description for the Treatment/Procedure Category in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Specialty Module ID – Select the Specialty module from the list. The Treatment/Procedure Category is valid for the selected Specialty module. The system-defined values is ‘Oral Health’  . This is a mandatory field.

  4. Treatment/Procedure Category Type - Select a value from the list to associate a Treatment/Procedure Category Type to the Treatment/Procedure Category. The system defined values are ‘Bridge’, ‘Crown’, ‘Denture’, ‘Extraction’, ‘Impaction’, ‘Impact’, ‘Maxilla Facial Prosthesis’, ‘Oral Cavity’, ‘Orthodontic Appliance’, ‘Others’, ‘Advanced Periodontal Treatment’, ‘Root Treatment’ and ‘Tooth Treatment’.

  5. Removable – Select a check box to record the treatment category as removable. The check box is enabled for selection, if the ‘Treatment/Procedure Category Type’ is selected as ‘Denture’. Clear the check box, if the treatment category is fixed. The check box is cleared, by default. If you select the check box, the corresponding treatment category can be recorded multiple times until it is marked as failure or the reason for failure is captured. You cannot record the same treatment category, if the treatment is fixed. But, you can record the same fixed treatment category, if the fixed treatment category is marked as failure. You can record denture on a tooth, tooth range, arch or quadrant.

  6. Enabled – Select the check box to enable the record in the transaction functions. The check box is selected, by default.
