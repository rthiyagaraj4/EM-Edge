Enter the following details to record a treatment against a quadrant, tooth, tooth range or arch:

 

  1. Root – Place the cursor over the appropriate root of the tooth. Click left mouse button. All possible treatments that can be recorded on the root level are displayed as a list. Practitioner has to select the appropriate Specialty Treatment from the list. The possible treatments on root level are displayed based on the master setup data. Hence, you need to set up appropriate treatments while setting up the master data through OH Code setup function.

  2. Surface - Place the cursor over the appropriate surface of the tooth. Click left mouse button. All possible treatments that can be recorded on the surface level are displayed as a list. The Practitioner needs to select the appropriate Specialty Treatment from the list. The possible treatments on surface level are displayed based on the master setup data. Hence, you have to set up appropriate treatment categories while setting up the master data through OH Code setup function.

  3. Entire Tooth – Select a tooth to record the treatments and click the hyperlink of the appropriate tooth. All possible conditions those can be performed on the tooth level (based on the master setup) are displayed as a list. The Practitioner needs to select the appropriate treatment from the list. The possible treatments on tooth level are displayed based on the master setup data. Hence, you have to set up appropriate Specialty Treatments while setting up the master data through OH Code setup function.

Click History area (row below the tooth number in the dental chart) to view the treatment history. A screen is displayed listing the Treatments that are recorded, Treatments done and the External treatments that are recorded. You can view the notes, image, treatment category and name, remarks recorded, tooth number, date and the name of the user who has recorded the treatment. The treatment name is displayed as a hyperlink. Click the hyperlink to view the tasks that are linked to the treatment.

=> The task name, status, remarks, date and the name of the user who has performed the task is displayed.

=> An error message ‘Treatments are not recorded for tooth number’ is displayed, if you have not recorded any treatment for the tooth.
