Enter the following details to record a condition on a root from the Dental Condition tab:

 

Dental Condition

  1. Clinical Group – Select a Clinical Group from the list. The Clinical Group defined in the Code setup is listed. It identifies the Clinical Group, where the patient is treated.

  2. Condition Type – If the Condition Type is selected through the dental chart for root, surface or entire tooth, the system displays the selected Condition Type. If the Condition Type is not set through the dental chart, you can select a Condition Type from the list. The Condition Types are system defined. The system defined values are - 'Attrition, Abrasion and Erosion', 'Caries', 'Congenitally Missing Tooth', 'Deciduous Tooth', Erupted Tooth', 'Exfoliated Tooth', 'Impaction', 'Mac Occlusion', 'Mal Position', 'Maxilla Non Tooth Related', 'Mobility', 'Partial Erupted Tooth', 'Peri apical Involvement', 'Pulp Involvement', 'Retained Deciduous Tooth', 'Root Involvement', 'Supernumerary Tooth', 'Traumatic Tooth' and 'Unerupted Tooth'.  This is a mandatory field.

When you select the Condition Type as Supernumerary, the supernumerary tooth is indicated as s1 and s2. You can record a maximum of two supernumerary tooth for a Base tooth. The tooth number is displayed as an hyperlink and an image is displayed to indicate that the tooth is a supernumerary tooth. You can record conditions on the Base or the Supernumerary tooth. The Base tooth number is highlighted using a color indicator. Click on the image to view the conditions and external treatments recorded for the tooth.

  1. Site – Based on the Condition Type selected, the Site details are displayed. The values are system defined and differ based on the Condition Type. This is a mandatory field. The Site values can be:

  2. Arch – Select Arch and the Condition Type is applicable for the Arch.

  3. Quadrant – Select Quadrant and the Condition Type is applicable for the Quadrant.

  4. Tooth – Select Tooth and the Condition Type is applicable for a tooth.

  5. Tooth Range – Select Tooth Range and the Condition Type is applicable for a tooth range. You can select the tooth numbers for which the Condition Type is applicable.

When the condition type is selected as 'Maxilla Non Tooth Related', site value changes to ‘Maxilla Facial Skeleton’ and ‘Mouth’. Select ‘Maxilla Facial Skeleton’ to record the condition on maxilla facial skeleton. On selecting ‘Maxilla Facial Skeleton’ as the ‘Site’ value, the values for ‘Area’ are displayed as ‘Intra Oral’ and ‘Extra Oral’.  Select Mouth to record condition on the mouth. On selecting ‘Mouth’ as the ‘Site’ value, you can select any specific ‘Area’, but the ‘Dental Condition’ is mandatory. The conditions recorded on Mouth or Maxilla Facial Skeleton is visible on clicking ‘Maxilla Facial Details’ under ‘Other Links’.

  1. Area – The Area differs based on the Site selected. This is a mandatory field. The values are:

  2. If the Site is selected as ‘Arch’, the Area is set to ‘Arch’. You can select either the Upper Arch or the Lower Arch from the list. The values are system defined.

  3. If the Site is selected as ‘Quadrant’, the Area is set to ‘Quadrant’. You can either select the Upper Right, Upper Left, Lower Right or the Lower Left quadrants from the list. The values are system defined.

  4. If the Site is selected as ‘Tooth’, the Area is set to ‘Tooth No’. You can select a tooth from the list. The selected Condition Type is applicable for the tooth selected. The values are system defined.

  5. If the Site is selected as ‘Tooth Range’, the Area is set to ‘Tooth Nos’. You can select a range of tooth from the list using [Ctrl] key. The selected Condition Type is applicable for the selected tooth range. A minimum of two teeth should be selected for the tooth range. The values are system defined.

  1. Dental Condition – Select a Dental Condition from the list. The values are populated in the list based on the Specialty Conditions defined in the Code setup. The Dental Condition identifies the severity condition of a tooth. This is a mandatory field.

  2. Surface – Select the Surface from the list on which the Dental Condition and the Condition Type is applicable. The values are system defined. This is a mandatory field.

  3. Cusp Tip – Select the values from the list. The values are system defined and are applicable only on a tooth. When an Arch, Quadrant and Tooth Range are selected, the values for Surface and Cusp Tip are set, by default.

  4. Click [Record] to save the record or [Reset] to clear the data and to record a new Dental Condition.

 

The values for Area and Site differ based on the Condition Type selected.
