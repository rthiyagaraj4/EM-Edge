Tasks

From this function you can Create Tasks and Associate them to an Order Catalog. You can Query for a selected Task.

The Tasks function is used to define multiple activities for a given treatment. A task is also called as an order. The tasks defined in the Oral Health module are linked to the Order Entry and Result Reporting module. When a treatment is planned, each task is performed individually, updated in the dental chart and marked as performed. A treatment is complete only when the task is completely performed and marked.

 

A Task is associated to an Order Catalog defined in the Order Entry and Result Reporting module. A task is listed for selecting a treatment, if it is associated to the treatment using the function ‘Treatments Tasks Link’.

Creating Tasks

From the Code Setup > Conditions/Treatments menu, select the Tasks function.

The Tasks screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Tasks created or [Reset] to clear the entered data.

  4. Click [Query] to search for the existing Tasks.
