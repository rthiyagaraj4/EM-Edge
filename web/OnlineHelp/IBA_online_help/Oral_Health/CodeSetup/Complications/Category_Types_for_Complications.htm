Category Types for Complications

From this function you can Associate a Complication to pre-defined Treatment or Procedure Category Types and Query for the selected records.

Treatment within a facility may cater to one or more Complications. This function is used to assign one or more Treatment/Procedure Category Types to a Complication. The compositions, which are part of the Composition Class, are in turn associated to the Treatment/Procedure Category Types. The Composition Class is applicable for the selected Treatment/Procedure Category Types. The Complication Code is associated to the Treatment/Procedure Category Type. Therefore, it is easier to identify the treatment type to be given to the patient depending on the severity level.

Associating Treatment/Procedure Category Type to Complication

From the Code Setup > Complications menu, select the Category Types for Complications function.

The Category Types for Complications screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Click [?] function to select a Complication from the list. The Complications defined gets listed. This is a mandatory field.

  3. The pre-defined Treatment/Procedure Category Types are listed. Select the check box to associate selected Treatment/Procedure Category Types to a Complication. The system-defined values are ‘Advanced Periodontal Treatment’, ‘Bridge’, ‘Crown’, ‘Denture’, ‘Extraction’, ‘Impaction’, ‘Implant’, ‘Maxilla Facial Prosthesis’, ‘Oral Cavity’, ‘Orthodontic Appliance’, ‘Others’, ‘Root Treatment’ and ‘Tooth Treatment’. This defines the Complication for the selected Treatment/Procedure Category Type.

  4. Click [Apply] to save the record created or [Reset] to clear the entered data.

  5. Click [Query] to search for the existing records.
