GCS Score

From this function you can Define GCS scores for the system defined Responses to grade the consciousness level of a patient when admitted to an Emergency Location.

The Glasgow Comma Scale is used to establish the degree of consciousness in a patient. A GCS Score helps in taking decisions in patient treatment. It is based on the response to the stimuli. Ratings range from 3-15 with 15 being the highest. A patient is assessed against the criteria of the scale and the resulting points given by the Glasgow Coma Score (GCS). GCS score function allows associating a score against each response, where the minimum score is one and the maximum score is four in case of Eye response. The maximum score for Verbal response is five and six for Motor Response. This is based on the Standard Glasgow Comma Scale Scoring System.

The maximum score is 15/15, which is considered excellent. The minimum score is 3/15.

Assigning GCS Score against a Response

From the AE Code Setup > Code Setup, select the GCS Score function.

The GCS Score screen gets displayed.

  1. Select the appropriate score from the list for a particular GCS Response.

  2. Click [Apply] to save the GCS Score or [Reset] to clear the selected data.

 

The GCS for Response values are provided through scripts to restrict you to 15 observations only. As per the Standard Glasgow Comma Scale, the maximum score value for recording responses is 15 and the minimum is 3.
