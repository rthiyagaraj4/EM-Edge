Select the following:

 

  1. Practitioner Type - The system defined Practitioner Types are displayed.

  2. Secondary Triage - Select the check box against each Practitioner Type to define the rights for accessing secondary triage function in queue management.

  3. Consultation - Select the check box against each Practitioner Type to define the rights for accessing Consultation function in queue management. The check box is enabled only if the relevant Secondary Triage check box is selected.

  4. Register New Born for AE – Select the check box against the required Practitioner type to give access rights for Register Newborn for AE link in Current Emergency Patients screen. By default, the check box is not selected. This column appears only when ‘Invoke Register Newborn for AE’ check box is selected in AE Parameter.

  5. For Admission - Select the check box against each Practitioner Type to provide access rights for accessing ‘For Admission’ sub menu in Current Emergency Patients function. By default, the check box is cleared.

  6. Cancel for Admission - Select the check box against each Practitioner Type to provide access rights for accessing ‘Cancel For Admission’ sub menu in Current Emergency Patients function. By default, the check box is cleared.

  7. Modify Referral - Select the check box against each Practitioner Type to provide access rights for accessing ‘Modify Referral’ sub menu in Current Emergency Patients function. By default, the check box is cleared.
