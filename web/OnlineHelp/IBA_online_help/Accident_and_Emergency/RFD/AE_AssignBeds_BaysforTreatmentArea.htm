Enter the following details to associate Bed/Bay to a Treatment Area:

 

  1. Location – Select the Location from the list to associate Bed/Bay to a Treatment Area. The list of locations displayed are limited to the locations of the logged in Facility. The Locations are defined in the ‘Clinic’ function of Outpatient Management module. The Clinics with Level of Care set to ‘Emergency’ are listed for selection. This is a mandatory field.

  2. Treatment Area – Select the Treatment Area from the list to which the Bed/Bays must be associated. The Treatment Areas defined for the selected Location in ‘Treatment Area for Location’ function are displayed. This field is mandatory.

  3. Bed/Bays – Select the value for Bed/Bays from the list. The listed values are ‘All’, ‘Normal’ and ‘Pseudo’. The value ‘All’ is displayed, by default.
