Release Bed/Bay

From this function you can Release a Bed/Bay reserved for a particular period in an Emergency Location.

Release Bed/Bay function is used to release a normal bed that is reserved, since reserved beds cannot be used for patient occupancy. Release is performed before the blocking period is due or the reserved bed gets released automatically.

 

Process Flow Diagram

Releasing Bed or Bay

From the AE Transactions, select the Release Bed/Bay Function.

Release Bed/Bay screen gets displayed.

  1. Click [Query] to invoke the Query screen.

  2. Enter details in the required fields. <Related Field Description>

  3. Click [Search] to display reserved Bed or Bay details or click [Clear] to clear the search results.

    The following details are displayed based on the search criteria:

    1. Location - The Location where the Bed/Bay is reserved is displayed.

    2. Treatment Area - The Treatment Area for Location in which the Bed/Bay is reserved is displayed.

    3. Bed/Bay No – The reserved Bed/Bay Number is displayed.

    4. Bed/Bay Type – The Bed/Bay Type of the reserved Bed/Bay is displayed.

    5. Reserve Until – The date until when the Bed/Bay is reserved is displayed.

    6. Release – Select the check box to release a reserved Bed/Bay. The check box is selected, by default.

  4. Select the ‘Release’ check box to release the Bed or Bay. The check box is selected by default. If the check box is cleared, the corresponding Bed or Bay remains reserved.

  5. Click [Apply] to release the reserved Bed/Bay or [Reset] to clear the entered data.
