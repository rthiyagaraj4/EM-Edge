Quality Screening Test Setup

From this function you can Associate quality screening tests to a blood product or reagent and Query for the existing records.

The Quality Screening Test Setup function is used to associate the quality screening tests to a product or a reagent. The Products and Reagents are tested for quality based on the tests created in this function. You can associate more than one test to a product or a reagent. You can also query for the existing records.

Associating Quality Screening Tests to Products/Reagents

From the Administration, double-click Quality Screening Test Setup function.

The Quality Screening Test Setup screen gets displayed.

  1. Select the Product’ or ‘Reagent’ option from the list to associate tests to the product or reagent.

  2. Click [List of Values] to select the required Product Code. This field gets displayed, if ‘Product’ is selected from the Option list. The list displays the values defined in the Products function.

  3. Click [List of Values] to select the required reagent Code. This field gets displayed, if ‘Reagent’ is selected from the Option list. The list displays the values defined in the Reagents function.

  4. Click [Display] on selecting the required product/reagent code to view existing records. You can view the test codes and their descriptions already associated to a product/reagent. You can modify the existing details. The suspended products/reagents are highlighted in gray color.

  5. Click [List of Values] to select the required test Code. The description of the test is displayed on selecting the test. The tests are created in the Test Codes function in Laboratory module.

  6. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.
