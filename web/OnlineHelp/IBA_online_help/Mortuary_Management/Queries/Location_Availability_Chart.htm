Location Availability Chart

From this function you can Query the availability of locations in the Mortuary.

This function is used to query the availability of locations in the mortuary. The details such as the location number, the number of occupied locations and the number of vacant locations are displayed. The deceased/body part patient ID occupying the location is displayed by placing the cursor over the occupied location number. Color representation is provided to identify vacant, occupied and blocked location.

Querying for Location Availability Chart

From MO Queries, select Location Availability Chart function.

The Location Availability Chart screen gets displayed.

  1. Select the Area from the list. The listed values are defined in the 'Area' function.

  2. Select the Location Status from the list. The values are All, Occupied, Vacant and Blocked. If all the occupied locations are to be queried, select 'Occupied'. If vacant locations have to be queried, select 'Vacant'. If blocked locations have to be queried, select 'Blocked'. If 'All' is selected, all the locations irrespective of whether the location is occupied, vacant or blocked are displayed. By default 'All' is displayed.

  3. Click [Search] to view the Location Availability Chart

  4. Click [Clear] to clear the result screen.
