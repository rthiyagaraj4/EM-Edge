Assign Area

This sub function is used to assign areas to the deceased/body part in the mortuary for the first time. If an area is not assigned to a deceased patient/body part in the 'Deceased Registration' and 'Body Part Registration' functions, then area is assigned. A location is associated to the deceased/body part in the assigned area using this sub function.

Assign Area option is applicable to all deceased bodies and body parts that are not assigned to any specific area while registering in the mortuary.

 

  1. Area - Select the required area from the list to assign the deceased patient to that area. The values listed are defined in the function "Area". The areas listed depend on the postmortem status of the deceased patient and the value recorded for the 'Postmortem Required' field in the function "Deceased Registration". This is a mandatory field.

Postmortem Required

Applicable Values

Selected

Values listed with area type set as

1.Postmortem Area

2.Storage Area

3.Other Area

'Release Area' is not listed unless the postmortem status is recorded as ‘Completed’.

Cleared

Values listed with area type set as

1.Release Area

2.Storage Area

3.Other Area

'Postmortem Area’ is not listed.

  1. Location - Enter the location in alphanumeric format of 8 characters maximum length or select the location using [?] function. This field is enabled only on selecting the area.

  2. Date/Time - Enter the date/time of assigning the deceased patient to the area in dd/mm/yyyy hh:mi format or select the date and time using the calendar function. This is a mandatory field.

  3. Click [OK] to assign the deceased patient to an area for the first time. Click [Cancel] to close the screen and the details entered are not recorded. The Internal Reports screen is displayed to generate the report or print a label and the control shifts back to the previous screen.
