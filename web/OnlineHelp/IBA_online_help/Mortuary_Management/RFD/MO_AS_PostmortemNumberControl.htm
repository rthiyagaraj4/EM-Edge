Enter the following details:

 

  1. Starting Autopsy Number - Enter the mortuary Autopsy Start Number in numeric format of 10 digits maximum length. This is a mandatory field.

  2. Next Autopsy Number - Enter the mortuary Next Autopsy Number in numeric format of 10 digits maximum length. This is a mandatory field.

  3. End Autopsy Number - Enter the mortuary Ending Autopsy Number in numeric format of 10 digits maximum length. This is a mandatory field.

  4. Autopsy Number Associate Details Required – Select the check box if the autopsy number generated for the mortuary should have the facility ID/year as prefix/suffix. If selected, the Facility ID and Year field are enabled for Prefix and Suffix.

 

 

Prefix

Suffix

Facility ID

If this check box is selected, the facility id is added as prefix to the autopsy number. The suffix check box is disabled.

If this check box is selected, the facility id is added as suffix to the autopsy number. The prefix check box is disabled.

Year

If this check box is selected, the year is added as prefix to the autopsy number. The suffix check box is disabled.

If this check box is selected, the year is added as suffix to the autopsy number. The prefix check box is disabled.

 

  1. Autopsy Number Format – The Autopsy Number Format is generated based on the ‘Autopsy Number Associate Details Required’ and the setup like the Patient Series used for the logged in facility and the length of the Patient ID as set in ‘Master Patient Index’ module.
