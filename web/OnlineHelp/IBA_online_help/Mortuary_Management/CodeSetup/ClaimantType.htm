Claimant Type

From this function you can Create various Claimant Types and Query for existing Claimant Type records.

When a deceased body is released from mortuary, it is necessary to identify the claimant (to whom the body is handed over) of the body. This function maintains details of various claimant types that include organizations registered with the facility for the purpose of handing over unknown or unclaimed bodies.

The created Claimant Type values appear in the Claimant Type field in the following functions:

• Claimant

• Body Part Registration

• Deceased Registration

• Change Deceased Details

• Release Deceased/Body Part

• Revert Released Deceased/Body Part

• Registration Details

Creating a Claimant Type

From the MO Code Setup, select Claimant Type function.

Claimant Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Claimant Type or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Claimant Type records.
