Create Monthly Posting Accounts

From this function you can Create an account balance records for all the posting accounts created. You can create an account balance record for a specific year.

Create Monthly Posting Accounts is an option provided to define a combination of the posting accounts. The accounts defined are used to record the posting account balances. A posting account created is relevant for the selected year. The following validations are carried out:

• If the number of accounting periods defined in installation parameters is 12, the function creates 13 monthly balance records (one additional for opening balance for the year).

• If this step is not executed, the transactions are not posted.

• Along with the detail accounts, the balances records for the summary accounts defined are also created.

Defining Monthly Posting Accounts

From the Setup > GL Setup, double-click on Create Monthly Posting Accounts function.

The Create Monthly Posting Accounts screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run Process] to process the log file or [Clear Record] to clear the entered data.

 

• This function has to be done only after executing "Create Posting Account Combinations".

• The Posting Accounts Combination has to be executed in order to create monthly posting accounts.

• A balance record is created for the detailed account as well as for the summary account.

• If the accounting period specified is 12 in the parameters, an additional balance record is created using this function.
