Write-off Installment Approval/Rejn

From this function you can either Approve or Cancel a Write-Off request placed.

Write-Off Installment Approval/Rejn function is used either to approve or reject a Write-off Request placed. Authorized users only can login and approve or cancel the request. The check boxes ‘Write-off Approval’ and ‘Approved Write-off’ has to be selected under ‘Privileges’ and ‘Cancellation’ in the function ‘Users for Authorization/Refund/Cash Counters’. Authorized User is able to approve or reject the write-off request only if the specified check boxes are selected. Patients need not pay the installment amount against the records, which are written-off. The outstanding documents against which the write-off request is placed using the function ‘Generate Write-off Installment Requests’ gets displayed. The document reference number generated is displayed along with the date and serial number.

Write-off Installment Approval or Rejection

From the Deferred Cash Patients > Agreement for Installment Payment, double-click on Write-Off Installment Approval/Rejn function.

The Write-Off Installment Approval/Rejn screen gets displayed.

  1. Select an Outstanding Amount Range. The Outstanding amount range defined using the Reminder Letter Setup function gets listed. The documents with the selected outstanding amount details get listed. Select All to view and to generate the reminder letters either for the selected documents or for all the documents. This is a mandatory field.

  2. Click Calendar function to select the date range. The outstanding documents for which the write-off request were placed using the function ‘Generate Installment Write-off Request’ get displayed. The outstanding document details with the write-off request for the selected date range get listed.

  3. Click [Authorize] to login and either to cancel, reject or to approve the write-off request. A screen gets displayed requesting the details to login and to approve or reject the write-off request. Click [OK] to login and to authorize the request or click [Cancel] to exit from the screen.

    Enter the following details:

    1. User ID – Enter the User ID in alphanumeric format. Users, who are authorized to approve the write-off process are only permitted to login and reject or approve the request. This is a mandatory field.

    2. PIN No – Enter the PIN number. The PIN Number recorded for the specific user during definition in the System Manager module has to be entered. This is a mandatory field.

    3. Approval Reason – Click [List of Values] to select an Approval Reason. The reasons defined using the function ‘Reasons for Approval’ with the ‘Write-off Approval’ selected get listed. This is a mandatory field.

    4. Rejection Reason – Click [List of Values] to select a reason for Rejection. The reasons defined using the function ‘Reasons for Approval’ with the ‘Write-off Approval’ selected get listed. This is a mandatory field.

  4. The document reference number generated for the outstanding documents using the function ‘Generate Write-off Installment Requests’ get listed. The details are displayed under the following tabs:

Record Approval/Rejection

Cancel Approved Write-Off

  1. Select the records and click [Commit Form] or [Process] to approve, reject or to cancel the write-off request. A confirmation screen pops-up. Click [Yes] to commit the records or [No] to cancel the transaction.

  2. Click [Clear Record] to clear the entered data.
