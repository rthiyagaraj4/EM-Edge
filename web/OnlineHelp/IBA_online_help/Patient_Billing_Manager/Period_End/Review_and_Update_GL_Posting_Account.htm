Review and Update GL Posting Account

From this function you can View and Update the Posting Account details. The details posted to the General Ledger account while posting the billing transactions are displayed. You can Query to view the details.   

Review and Update GL Posting Account function is used to update the details posted to the General Ledger account. The billing details are posted using the function ‘Post Billing Transactions’. You are able to correct the posting accounts before GL distributions are exported to the General Ledger. The incorrect posting accounts, debit/credit transaction details transferred to the General Ledger are reduced, if the corrections are done prior to the posting. The rectification journal vouchers are also minimized. The Posting Account associated to the Transaction Reference Document can only be changed.  

Review and Update GL Posting Account

From the Period End, double-click Review and Update GL Posting Account function.

The Review and Update GL Posting Account screen gets displayed.

  1. Click either [Execute Query] or press [F8] to query and update the details. A message appears, if the patient is deceased or an inpatient. <Related Field Description>

  2. Click [Commit Form] or press [F10] to save the details.

 

• You can update only the Posting Account details.

• You have to update the posting account details before exporting the GL transactions.
