Audit Report For Change Of Bill Nature

From this function you can Generate a report to get an overview of the bill details using the function Update Bill Details.

Audit Report for Change of Bill Nature function is used to generate a report to view the bill details, which are updated. A report is generated listing all the bill details, updated using the function ‘Update Bill Details’.

Audit Report for Change of Bill Nature

From the Period End, double click Audit Report for Change of Bill Nature function.

The Audit Report for Change of Bill Nature screen gets displayed.

  1. Click Calendar function to select a date range.

  2. Click [List of Values] to select a range of Document Type/Number.

  3. Click Calendar function to select a document date range. The bills generated during the selected date range get listed.

  4. Click [List of Values] to select a range of Patient ID. The report is generated for the selected range of Patient ID.

  5. Click [Run Process] to generate an audit report. You can either view, print or save the output in a file.
