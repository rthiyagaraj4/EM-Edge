Multiple Claims Process Setup

From this function you can Define customer group codes to be used for multiple claim processing.

This function allows you to define customer group codes to be used for multiple claim processing. It is a one-time setup.

 

This function is applicable only if the ‘Multi Claims Processing’ check box is selected in BL Parameters screen.  If the check box is cleared, the message ‘Function not applicable for the facility’ gets displayed.

Defining Multiple Claims Process Setup

From the Period End, select the Multiple Claims Process Setup function.

The Multiple Claims Process Setup screen gets displayed.

  1. Select the customer group code from the Customer Grp Code Lov. This field is mandatory.

  2. On tab out, the description of the customer group gets displayed based on AR setup.

  3. Click [Save], the selected customer group code and description gets saved.
