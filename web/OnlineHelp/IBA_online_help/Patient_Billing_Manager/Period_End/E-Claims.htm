E-Claims

From this function you can View and Generate the E-claims report in excel format.

E Claims function is used to view and generate the e-claims report in excel format. The same excel sheet is sent for claim process.

Generating e-clam report

From the Period End > double click E-Claims Function.

The E-Claims screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Generate Excel] to export the data and to generate a report in excel format.

  3. The report gets generated based on the given search criteria. If the given search criteria does not match with any of the patients record, then blank excel report will be generated.
