Finalize Payer Acceptance

From this function you can Finalize the payments against an invoice dispatched to the Payer.

Initially an invoice is generated for the outstanding bills associated to a Customer. The invoices are later dispatched using the dispatch documents created. A payer can either accept the invoice or post the amount to the facility to settle the outstanding bills or reject the invoice. You can record the payer acceptance using this function and record the amount received against an invoice dispatched. You can cancel the records using this function before finalizing the payer acceptance. You can finalize the payer acceptance manually only if the check box ‘Finalize Payer Acceptance’ check box is cleared in the ‘Invoice Process Parameters’ function. If the check box is selected, the payer acceptance is finalized automatically. The invoice documents, which are verified or cancelled, are displayed using  and colors respectively.

Finalizing the Payer Acceptance

From the Invoice Management > Transaction, double-click on Finalize Payer Acceptance function.

The Finalize Payer Acceptance screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Commit Form] to save the records or [Clear Record] to delete the data entered.

  3. Click [Execute Query] to query for the existing records.
