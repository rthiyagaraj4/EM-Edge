Finalize Dispatch Document

From this function you can Finalize the dispatch documents used to post the invoice to the Customers in the facility.

The Finalize Dispatch Document function enables you to finalize the dispatch document used to post the invoice details to the Customers. The dispatch documents are created using the function ‘Generate Dispatch Document’. You have to associate the invoices to the dispatch document. The document is further finalized and posted to the Customer indicating them to settle the outstanding bills. The invoice documents, which are verified or cancelled, are displayed using  and   colors respectively.

Finalizing the Dispatch Document

From the Invoice Management > Transaction, double-click on Finalize Dispatch Document function.

The Finalize Dispatch Document  screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Commit Form] to save the records or [Clear Record] to delete the data entered.

  3. Click [Execute Query] to query for the existing records.
