Cancel Invoice/Debit Note/Credit Note

From this function you can Cancel the Invoice, Debit Note or a Credit Note generated for the encounters in the facility. You can View the Service Details and the additional invoice details captured for the patient.

The Cancel Invoice/Note/Credit Note function enables you to cancel the invoice, debit note or a credit note generated in the facility for various patient transactions. You have to either specify the Payer ID, Patient ID or the Encounter ID is mandatory to search for the details in the facility. The search results are filtered based on the query details entered. You can query the details for a specific facility. You also have an option to search for the details based on the Episode ID or a specific Episode ID of the patient. The encounter details are displayed. You can view the invoice details, debit note or the credit note details and the bill details for a specific patient encounter using the Invoice/Debit Note/Credit Note Details and the Bill Details tab. You can also view the additional invoice details captured for the patient and the service details associated to a specific bill. The invoice cannot be cancelled, if the respective record is already selected for payment or if the debit note or the credit note is generated for the specific document. You can cancel the original or the re-generated invoice/debit/credit note through this function.

Cancelling the invoice, debit or the credit note

From the Invoice Management > Transactions, double-click on Cancel Invoice/Note/Credit Note function.

The Cancel Invoice/Note/Credit Note screen gets displayed.

  1. Select the Facility from the list. The facilities associated to the logged in user gets displayed. <Related Field Description>

  2. Click [Refresh] to view the results based on the query criteria specified.

  3. The encounter details are generated for the invoices. Click [V] to view the additional details captured for the patient using the Capture Invoice Additional Info function.

  4. The invoice/debit note/credit note details tab displays the transaction details. Click [D] to view the service details. The services for which the bill is generated are listed under the 'Service Details' tab. The Special Items are highlighted using the color. The items and services associated to the patient are displayed.

  5. Right-Click on the specific record and select 'Cancel the Invoice' option under the 'Invoice/Debit Note/Credit Note' tab to cancel either the invoice or the debit note or the credit note. An authorization screen pops-up. Users with privilege rights to exempt the bill are authorized to login and cancel the invoice, debit note or the credit note.

    Enter the following details to login and cancel the record:

    1. User ID - Enter the User ID in alphanumerice format. The Users with the check box 'Bill Exemption' as selected in the 'User Privileges and Cash Counter Access' function are authorized to cancel the details. This is a mandatory field.

    2. PIN No - Enter the PIN Number for the respective User. This is a mandatory field.

    3. Reason - Click [List of Values] to select the Reason. The Reasons defined using the 'Reasons for Billing Actions' function get listed.

    Click [OK] to login and cancel or [Cancel] to exit from the authorization screen.

  6. The payment details for the specific bill is displayed under 'Payment Details'. You can view the settled amount, receipt number and the date when the receipt is generated for the respective invoice, debit note or the credit note.

  7. The 'Bill Details' tab displays the bill generated for the respective invoice, debit note or the credit note. The bill number, outstanding amount, invoice amount, discount details and the invoice number, date and the amount gets displayed.
