Setup Patient Billing Remarks

From this function, you can Define remarks for a specific Customer or a Billing Group. You can also set up remarks for specific functions in the Patient Billing Manager module. You can Query to view the Remarks for the Customer or the Billing Groups.

Setup Patient Billing Remarks function is used to record the remarks to be printed on a bill or a receipt for a specific Billing Group or a Customer in the Facility. You can also record the remarks for the selected functions.

Defining Patient Billing Remarks

From the Masters > Interfaced Modules Financial Dtls, double-click on Setup Patient Billing Remarks function.

The Setup Patient Billing Remarks screen gets displayed.

  1. Select a Facility from the list. The facilities associated to the logged in user gets displayed.

  2. Enter the details in the mandatory (*) fields found in the following tab pages.

Customer Specific Remarks

Customer Specific Remarks tab enables you to record the remarks for a Customer associated to a facility. The remarks recorded are pertinent to the Customer in the facility for a specific period.

Enter the following to define remarks for a selected Customer:

  1. Customer – Click [List of Values] to select the Customer. The values defined using the Customer function get listed. The description for the selected customer appears in the adjacent text box. This is a mandatory field.

Click [Execute Query] or press [F8] to record Customer Specific Remarks.

Customer Specific Billing Remarks

  1. Customer Billing Remarks - Enter the remarks in alphanumeric format with a maximum of 200 characters.

  2. Eff. From Date – Click Calendar function to select a date. The Customer Specific Remarks are valid for the selected Customer from the date selected.

  3. Eff. To Date - Click Calendar function to select a date. The Customer Specific Remarks are valid for the selected Customer until the date selected. The date selected has to be greater than the effective from date.

Billing Group Specific Remarks

Billing Group Specific Remarks tab enables you to record the remarks for a Billing Group associated to a facility. The remarks recorded are pertinent to the Billing Group in the facility for a specific period.

Enter the following to define Remarks for a selected Billing Group:

  1. Billing Group – Click [List of Values] to select a Billing Group. The description for the selected Billing Group appears in the adjacent text box. This is a mandatory field.

Billing Group Specific Remarks

  1. Billing Group Remarks - Enter the remarks for the selected billing group in alphanumeric format with a maximum of 200 characters.

  2. Eff. From Date - Click Calendar function to select a date. The Billing Group Specific Remarks are valid for the selected Billing Group from the date selected.

  3. Eff. To Date - Click Calendar function to select a date. The Billing Group Specific Remarks are valid for the selected Billing Group until the date selected. The date selected has to be greater than the effective from date.

Remarks Applicability Rule

Remarks Applicability Rule tab enables you to record the remarks for a function. You have an option to choose between the various remarks.

Enter the following to define Remarks for selected functions:

  1. Function Name – Click [List of Values] to select a Function Name. The Remarks Applicability Rule is applicable only for the functions, which are system defined. This is a mandatory field.

  2. Encounter Remarks – Select the check box to record the episode level remarks for the selected function.

  3. Patient Financial Remarks - Select the check box to record patient financial remarks for the selected function.

  4. General Remarks – Select the check box, if general remarks are applicable for the selected function.

  5. Customer Remarks – Select the check box, if the customer specific remarks are applicable for the selected function.

  6. Billing Group Remarks - Select the check box, if the billing group specific remarks are applicable for the selected function.

  1. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records within the facility.
