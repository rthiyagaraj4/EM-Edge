Document Types by Payers

From this function you can Define the document types for the Customer Group and the Customer in the facility. You can Query to view the existing records.

The Document Types by Payers function enables you to define the document types for the transactions in the facility. You can define the Customer Group, Customer and the Counter to which the default document types are valid. You can also define the default document types Customer and the Customer Group association linked to the special cash counter. The document types for the Customer and the patient bill has to be separate. Based on the Document Types selected for the Customer and the patient, separate bills are generated. The bills for the transactions at the special clinics have to be billed in the special Cash Counters set up.

Associating Document Types to Payers

From the Masters > Facility Specific Setup, double-click on Document Types by Payers function.

The Document Types by Payers screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Commit Form] to save the records or [Clear Record] to delete the data entered.

  3. Click [Execute Query] to query for the existing records.
