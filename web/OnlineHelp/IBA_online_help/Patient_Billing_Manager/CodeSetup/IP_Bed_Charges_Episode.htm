IP Bed Charges – Episode

From this function you can Define the bed charges for inpatient transactions in the facility and Query to view the existing records in the Facility.

The IP Bed Charges – Episode function is used to define Billing class and Billing service for the Bed type specific to a Patient and Episode.

IP Bed Charges – Episode

From the Masters > Interfaced Modules Financial Dtls, double-click IP Bed Charges – Episode function.

The IP Bed Charges – Episode screen gets displayed.

  1. Enter the details in the mandatory (*) fields found in the following pages.

By Bed Charges – Episode

Enter the following to calculate bed charges based on bed type.

  1. Patient No - Select the Patient from the list of values or enter directly.

  2. Episode Id – List of Episode Ids for the selected patient gets displayed.

  3. Adt SL No – Displays the Bed Transfers made against the Patient for the Episode selected. On pressing Tab,  the  Bed Type From, Bed Type To and Billing Class are defaulted.

  4. Bed Type From – Click [List of Values] to select a Bed Type. The values defined using the function Bed Type in the Inpatient and Day Care Management module gets listed.

  5. Description - The description for the selected Bed Type gets displayed.

  6. Bed Type To – Click [List of Values] to select a Bed Type. The values defined using the function Bed Type in the Inpatient and Day Care Management module gets listed. This is a mandatory field.

  7. Billing Class - Click [List of Values] to select a Billing Class. The values defined using the function Billing Class get listed. This is a mandatory field.

  8. Description - The description for the selected Billing Class gets displayed.

  9. Billing Service - Click [List of Values] to select a Billing Service. The values defined using the Billing Service function get listed. The Bed Type and the selected Billing Class is charged based on the prices defined for the selected Billing Service. You can add the billing service against the Bed Type Transfer. This is a mandatory field.

  10. Description - The description for the selected Billing Service gets displayed.

  11. Apply for Previous Charges If this is Higher Bed class – When selected, if it is a higher Bed class then the charges are applied for previous charges entered in this screen.

  1. Click [Recalculate] to recalculate the patient charges if the patient is moved from one bed class to another bed class.

  2. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for the existing records within the facility.
