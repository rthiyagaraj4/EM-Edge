Reasons for Billing Actions      

From this function you can Define the reasons for the billing transactions in the Facility. You can also set an Action Type for the Reason Codes. You can Query to view the Reasons defined.   

Reasons for Billing Actions function is used to define the reasons for the exemptions, cancellations or the bill discounts in the facility. An Action Type has to be associated to the Reason Code. The Reason Code is valid for the selected Action Type. An action reason is associated to the action types to identify the circumstances where the action reason has to be used.

Defining Reasons for Billing Actions

From the Masters > select Interfaced Modules Financial Dtls, double-click Reasons for Billing Actions function.

The Reasons for Billing Actions screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for the existing records within the facility.
