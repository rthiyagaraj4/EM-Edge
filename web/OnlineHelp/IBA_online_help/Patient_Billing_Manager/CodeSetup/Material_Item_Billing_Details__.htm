Material Item Billing Details  

From this function you can Define a charge towards a material item and Query to view the Material Item Billing Details defined.

Material Item Billing Details function is used to charge the pharmacy items or the medical supply items defined in the Material Management module. An item code is associated to a Sale Category. The price is set up based on the sale name or the trade name. The prices defined using this function is considered as the base price for the material item, if the sale category is selected as batch. Items consumed from any store associated to the facility are charged based on the parameters defined for the sale category using the function Material Item Charging Rule Setup.  

Defining Material Item Billing Details

From the Masters > Interfaced Modules Financial Dtls, double-click Material Item Billing Details function.

The Material Item Billing Details screen gets displayed.

  1. Click [List of Values] to select an Item Code. The items defined in the Inventory Management module get listed. The description for the selected Item Code gets listed.

  2. The Item Type gets displayed based on the Item Type selected while defining the Item in the Inventory Management module.

  3. Enter the details in the mandatory (*) fields found in the following tab pages.

Sale Details by Item

Sale Details by Trade Name

  1. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records within the facility.
