SS View Payment Reference Details

From this function you can Query to view invoices, bill details and payment details of a Social Security.

The SS View Payment Reference Details function is used to view the payment details of Social Security. The amount settled against the invoices dispatched for the outstanding bills get listed. You have to define a Printer for the selected Document Type to print the payment reference details. You can view the transactions against which the invoice is raised and the individual bill details. You also have an option to view the services utilized for which a bill was raised. You can specify only specific details in the data entry area to view the payment details of the Customer.

View SS Payment Reference Details

From the SS Invoice Management > Queries, double-click on SS View Payment Reference Details function.

The SS View Payment Reference Details screen gets displayed.

  1. Click [List of Values] to select the Payment Reference Document. This is a mandatory field.

  2. Click [Execute Query], [View Details] or press [F8] to view the settlement details and receipts or invoices generated for the selected patient transaction.

  3. On selecting the Payment Reference Document, date and allocated amount, balance amount, receipt amount and receipt option is displayed. The check box 'Finalized' is selected, if the reference documents are finalized.
