Update Internet Send Date for the Invoices

From this function you can Update date for the invoices to be dispatched. You can also Create, Cancel or View the printed invoices related to HIV.

The Update Internet Send Date for the Invoices function enables you to update the date for invoices dispatched through e-mail.

Updating Internet Send Date for the Invoices

From the SS Invoice Management > Transaction, double-click on Update Internet Send Date for the Invoices function.

The Update Internet Send Date for the Invoices screen gets displayed.

  1. Click the Calendar function to select the Internet Date. This is a mandatory field.

  2. The invoice details are displayed under the following tabs:

Create

Enter the following details:

  1. Visit Date - Click the Calendar function to select the visit date to update invoice details for the respective visit registration date.

  2. [Refresh] - Click [Refresh] query the records.

Patient ID, ARV Code for the respective patient, Invoice Number, date and amount are displayed. HIV Service Types and the net amount for each Service Type are listed when you select the check box.

  1. Select All - Select the check box to record internet date for the respective patient.Click [Commit Form] to save the records.

Cancel

You can cancel the records using this tab. The records saved through the 'Create' tab are listed. Patient ID, ARV Code for the respective patient, Invoice Number, date and amount are displayed. HIV Service Types and the net amount for each Service Type are listed when you select the check box.

  1. Select All - Select the check box to cancel the records. Click [Commit Form] to save the records.

Printed

The patient records that are printed on the selected internet date are displayed. Patient ID, ARV Code for the respective patient, Invoice Number, date and amount are displayed along with the associated HIV Service Types and net amount for each Service Type.
