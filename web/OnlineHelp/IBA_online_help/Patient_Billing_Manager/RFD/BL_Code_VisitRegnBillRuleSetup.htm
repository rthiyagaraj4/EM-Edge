Enter the following details to define a format for the visit registration bills:

 

  1. Facility - Click [List of Values] to select a Facility. The bill format is valid for the Clinics in the selected facility.

 

Visit Generation Rule by Clinics

  1. Billing Group - Click [List of Values] to select a billing group. The values defined using the Billing Group function get listed. This is a mandatory field.

  2. Description - The description for the selected billing group gets displayed.

  3. Clinic – Click [List of Values] to select a Clinic. The values defined using the Clinic function in the Outpatient Management module gets listed.

  4. Description - The description for the selected Clinic gets displayed.

  5. Generate Bill at Regn – Select the check box to generate the bill during the visit registration.

  6. Settle Bill Amount - Select the check box to generate the bill when the amount towards the service is settled.

  7. Print Option - Click [List of Values] to select a Print Option. The values are system defined and differ when the check box Settle Bill Amount is selected or cleared.

  8. Do not Print – Select Do not Print, if a hard copy of the bill is not required.

  9. Print Visit Bill-Cum-Receipt Automatically - Select Print Visit Bill-Cum-Receipt Automatically to print a receipt and bill automatically when the amount is settled during visit registration.

  10. Print Visit Receipt Automatically - Select Print Visit Receipt Automatically to print a receipt automatically when the amount is settled during visit registration.

  1. Generate Zero Visit Bill - Select the check box to generate a bill with zero amount during visit registration in the selected Clinic.

  2. Print Zero Visit Bill – Select the check box to get a printed copy of the bill with zero amount generated during visit registration in the selected Clinic.

  3. A/E Checkout Receipt Print Option - Select the check box to print receipts during accident and emergency patient checkouts. Clear the check box, if receipts are not to be printed.

  4. Print Zero Visit Bill for A/E Checkout – Select the check box to get a printed copy of the bill with zero amount during patient checkout from the accident and emergency clinics. Clear the check box, if receipts for zero visit charges are not necessary.
