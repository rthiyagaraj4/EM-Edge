Enter the following details to search patient records:

 

  1. Cust Group – Click [List of Values] to select a Customer Group. The values defined through the ‘Customer Group’ function are listed. The description of the selected Customer Group is displayed. This field is mandatory when you select the Transaction Type as Invoice.

  2. Customer - Click [List of Values] to select a Customer Group. The values defined through the ‘Customer’ function and associated to the ‘Customer Group’ that is selected are listed. The description of the selected Customer is displayed. This field is mandatory when you select the Transaction Type as Invoice.

  3. Policy Type – Click [List of Values] to select a Policy Type. The values defined through the ‘Policy Types’ function in the Insurance Management module is listed. The description of the selected Policy Type is displayed. This field is mandatory when you select the Transaction Type as Invoice.

  4. Patient ID - Click [List of Values] to select a Patient ID. The list displays the patients associated to the selected Customer Group, Customer and Policy Type combination. The name of the patient, age, gender and encounter ID is displayed on selecting the patient ID. This field is mandatory when you select the Transaction Type as Invoice.

  5. Encounter ID - Click [List of Values] to select an Encounter ID. The list displays the encounters of the selected patient. This field is mandatory when you select the Transaction Type as Invoice.

  6. Invoice – Click [List of Values] to select an Invoice document type and number. This field is mandatory if the ‘Transaction Type’ is selected as Debit Note or Credit Note. The patient details are displayed on selecting the invoice number.

  7. All – Select the check box to query patient records associated to any patient class. The check boxes Outpatient, Emergency, Inpatient, Daycare and External are selected automatically when you select ‘All’. Clear the check box to query patient records associated to a selected patient class.

  8. Visit/Adm Date [From – To] – Click the Calendar function to select the visit or admission date range. The patient records with the visit or admission date matching the search criteria are displayed.
