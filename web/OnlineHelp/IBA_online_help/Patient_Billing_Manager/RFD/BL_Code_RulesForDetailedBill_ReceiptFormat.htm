Receipt Format

The Receipt Format tab enables you to define format for receipts and refund acknowledgments. The format selected in the 'BL Parameters' function for 'Print Receipt Using' gets displayed. You can define various receipt formats for different Settlement Types like Cash, Cheque, Credit Card and other payment modes. Option is also available to load the default format. Various refund formats are available for different refund modes. System displays Hospital Details, Patient Details, Receipt/Refund Details and Text Formatting list values in the text field in each tab. Right-click to select a value from the list. Various formats with several values are displayed on selecting a value from the list. Double-click on a legend value to select it.

The format lists styles to format the legends in header, footer and body section of a receipt format.

The format lists receipt/refund related transaction details. A Reference ID is generated for receipts, if 'Print Receipt Using' value is selected as 'Format2' in 'BL Parameters' function. The Cash Counter, Document Number, Document Type Code, Episode Type, Settlement Mode, Specialty and Year gets displayed for the Receipt/Refund Reference. The mentioned receipt reference ID components are valid for Cash, Cheque, Credit Card and other mode of payments and applicable for Header, Footer and Body section.

If Bill and Receipt is printed as a single document, Bill Reference ID is printed instead of Receipt Reference ID. If there is a Single Reference ID for multiple entries of receipts, the reference ID is printed as asterisks to indicate multiple entries.

The format lists patient demographic details.

The format lists complete information of the hospital and demographic details.

 

Enter the following details to define a Receipt Format:

Format 2- Receipt

Cash

You can define the receipt format for Cash settlements using this tab.

  1. Load the default Setup - Select the check box to load the pre-defined template. A confirmation message 'Existing Setup Will be Replaced, Continue?' is displayed on selecting the check box. Click [Yes] to load the default template or [No] to cancel the process.

Header

  1. [Line 1- Line 6] - Enter the details in the fields with a maximum of 100 characters. The data recorded gets displayed in the header part of the receipts generated. You can also record the data using the pre-defined formats that are available. Right-click on each field to select a value from the pre-defined format. The options available are Hospital Details, Patient Details, Receipt/Refund details and Text Formatting. Select a value and the pre-defined values recorded for the selected value are listed. Double-click on the value to select it. The selected value is displayed in the respective field.
