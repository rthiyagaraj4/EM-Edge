Users for User Group

Enter the following details to associate a User to a User Group:

 

  1.  User Group – Click [List of Values] to select a User Group. The User Groups created under the tab ‘User Group’ get listed. The User Group gets listed only if the User Groups are created initially for the selected Service. This is a mandatory field.

 

Users for User Group

  1. User ID – Click [List of Values] to select a User ID. The Users defined in the System Manager module and associated to the logged in facility get listed. The relevant User under the User Group is able to access the selected functions. This is a mandatory field.

  2. User Name – The name of the selected User gets displayed.

  3. Enter Service Request – Select Enter Service Request to permit the User to associate services to the patient using the Inpatient and Outpatient transaction functions – ‘Enter Requests – OP’ and ‘Enter Requests – IP’.

  4. Confirm Service Request – Select Confirm Service Request to permit the User to confirm the service requests. If the check box ‘Autoconfirm’ is cleared for the service, the User has to manually confirm the services using the functions ‘Confirm Outpatient Service Requests’ and ‘Confirm Inpatient Service Request’ in the Inpatient and Outpatient transaction functions.

  5. Cancel Confirmed Service – Select Cancel Confirmed Service to permit the User to cancel the services, which are confirmed. User can cancel the services in the Inpatient and Outpatient transaction functions ‘Cancel OP Direct Billing Items’, ‘Cancel OP Indirect Services’, ‘Cancel IP Direct Billing Items’ and ‘Cancel IP Indirect Services’.

  6. Cancel Self Entered Services Only – Select Cancel Self Entered Services Only to permit the User to cancel the services. The services entered by the user can only be cancelled. The services entered and confirmed by any other Users cannot be cancelled.

  7. Cancel Indirect Service – Select Cancel Indirect Service to permit the User to cancel the services placed from external modules. User can use the functions ‘Cancel OP Indirect Services’ and ‘Cancel IP Indirect Services’ to cancel the services.

  8. Adjust Confirmed Service – Select Adjust Confirmed Service to permit the User to amend the services, which are already confirmed. The User can use the functions ‘Adjust OP Direct Billing Items’ and ‘Adjust Direct Billing Items’ to amend the service details for an Outpatient and an Inpatient respectively.

  9. Adjust Self Entered Services Only – Select Adjust Self Entered Services Only to permit the User to update the Self Entered Services Only. The User can use the functions ‘Adjust OP Direct Billing Items’ and ‘Adjust Direct Billing Items’ to update the service details, when the services are self entered.

  10. Enter Negative Request – Select the check box to authorize user to issue request with negative quantity for rate based or charge based services in the transaction functions. Clear the check box, if the user is not permitted to enter negative quantity request for rate based or charge based services.

  11. Cancel Invoice – Select Cancel Invoice, if the User in the User Group has to be given access to cancel the invoices generated for a patient or a customer. Clear the check box and the User in the User Group cannot cancel the invoices generated for a patient or a customer in the facility.

  12. Cancel Invoice Payment – Select Cancel Invoice Payment, if the User in the User Group has to be given access to cancel the payment against the invoices generated for a patient or a customer in the facility. Clear the check box and the User in the User Group cannot cancel the payment against the invoices generated for a patient or a customer in the facility.

  13. Reclassification Allowed - Select the check box to authorize the User to reclassify, finalize and renew the reclassification details of a patient. Clear the check box, if the User is not authorized to reclassify, finalize and renew the reclassification details of a patient.

  14. Un-Finalize Invoice Payment - Select the check box, if the user is permitted to un-finalize invoice payments in 'Record/Process Invoice Payments' function in 'Invoice Management' menu. Clear the check box, if the user is not permitted to un-finalize the invoice payments.

  15. Undo Pre-Invoice - Select the check box, if the user is permitted to cancel the pre-invoice that is generated through 'Generate Pre-Invoice' function in 'Invoice Management' menu. Clear the check box, if the user is not permitted to cancel pre-invoice.

  16. Restrict Cancelling Unpaid Services – Select this check to restrict cancel of direct/indirect service through following screens if the original bill of the service to be cancelled has outstanding cash balance

 

  • Cancel OP Direct Billing Items

  • Cancel OP Indirect Billing Items

  • Cancel IP Direct Billing Items

  • Cancel IP Indirect Services

  • Adjust Op Direct Billing Items

  • Adjust Direct Billing Items
