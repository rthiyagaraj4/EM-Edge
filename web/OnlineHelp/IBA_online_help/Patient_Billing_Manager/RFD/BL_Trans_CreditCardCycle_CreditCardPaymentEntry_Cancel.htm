You can query to cancel the recorded Payment details. Press [F8] or click [Execute Query] to view the payment batch processed records. If a particular payment record has to be cancelled and you have the batch details, the system displays the associated details on pressing [F8] or clicking [Execute Query].

 

  1. Instrument No./Date – The Instrument Number along with the Date gets displayed. You cannot update the instrument number or date.

  2. Bank – The system defaults the Bank name to the  one associated with the Instrument Number. You cannot update the value in this field.

  3. Date – The system defaults the Date field to the system date.

  4. Receipt Type – The system displays the Receipt Type automatically . You cannot update the value in this field.

  5. Payment Ref. Batch ID – The Batch ID generated while setting up the payment entry gets displayed, by default. You cannot update the Payment Reference Batch ID.

  6. Cancel Reason Code – Click [List of Values] to select and to associate a Reason for cancellation. The reasons defined using the function 'Reasons for Billing Actions' under the menu ‘Other Masters’ get listed. This is a mandatory field.

  7. Cancellation Reason – Enter the reason for cancellation of the payment entry in alphanumeric format of 50 characters maximum length. This is a mandatory field.

  8. Cancel – Select the check box to cancel the transaction. If the check box is selected, an authorization screen pops-up.  Enter the User ID, Pin Number to login and to cancel the payment batch entry.

 

Credit Card Batches for which Payment Received

  1. Approval No – The Approval number entered during receipt generation for Credit Card settlement gets displayed. You cannot alter the approval number.  

  2. Amount – The amount collected against each batch of Credit Card is displayed.

  3. Batch Type/No – The system defaults the batch  type or the batch number that is generated. You cannot update the Batch Type or the Batch Number.

  4. Date – The system date is displayed, by default. You cannot update the date.

  5. Credit Card Receipt No – The Credit Card Receipt Number associated to the batch gets displayed.

  6. Rcpt.Doc.Ref – The Credit Card Receipt Document Reference details associated to the Credit Card Receipt Number gets displayed.

  7. Patient Name – The Patient for whom the receipt is generated gets displayed.
