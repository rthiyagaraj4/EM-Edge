View Visit Bill Generation Rule by Clinics

The View Visit Bill Generation Rule by Clinics tab enables you to view the bill generation rule defined for a Clinic using the function Outpatient Visit Bill Rule Setup.

 

  1. Facility – The Visit Bill generation rule defined for the relevant Clinic in the Facility gets displayed. The Visit Bill Generation Rule defined using the function ‘Outpatient Visit Bill Rule Setup’ under ‘Interfaced Modules Financial Dtls’ function in the Masters gets displayed.

  2. Clinic – The name of the Clinic associated to the Facility for which the rule is defined gets displayed.

  3. Description – The Description for the selected Clinic is displayed.

  4. Generate Visit Bill – Generate Visit Bill is either selected or cleared based on the parameter set up using the function ‘Outpatient Visit Bill Rule Setup’. If the check box is selected, the bill is generated during visit registration to the respective Clinic. If the check box is cleared, you cannot generate a Visit Bill during Visit Registration.

  5. Settle Bill Amount – Settle Bill Amount is either selected or cleared based on the parameter set up using the function ‘Outpatient Visit Bill Rule Setup’. If the check box is selected, the bill amount for the visit registration has to be settled during registration in the Outpatient Clinic. If the check box is cleared, it is not mandatory to settle the bill amount during Visit Registration.

  6. Print Option - Print Option gets displayed based on the values selected in the ‘Outpatient Visit Bill Rule Setup function. Print Option is enabled only when ‘Settle Bill Amount’ is selected. The values are system defined – ‘Print Visit Bill-Cum-Receipt – Automatically’, ‘Print Visit Receipt – Automatically’ and ‘Do not print’.

  7. Print Visit Bill-Cum-Receipt – Automatically – Select Print Visit Bill-Cum-Receipt – Automatically to generate  the Bill Cum Receipt automatically for the respective Visit.

  8. Print Visit Receipt – Automatically – Select Print Visit Receipt – Automatically to generate a receipt automatically for the visit.

  9. Do not print – Select Do Not Print and the bill or receipt is not generated for a visit registration.

  1. Generate Zero Visit Bill – Generate Zero Visit Bill is either selected or cleared based on the parameter set up using the function ‘Outpatient Visit Bill Rule Setup’. If the check box is selected, a bill with zero amount is generated for the Billing Group. If the check box is cleared, no bill is generated.

  2. Print Zero Visit Bill – Print Zero Visit Bill is selected or cleared based on the parameter set up using the function ‘Outpatient Visit Bill Rule Setup’. Print Zero Visit Bill option is enabled only if the ‘Generate Zero Visit Bill’ generation is selected. If the check box is selected, the bills with zero amount are printed during visit registration. If the check box is cleared, the zero visit bills are not printed for the Billing Group.
