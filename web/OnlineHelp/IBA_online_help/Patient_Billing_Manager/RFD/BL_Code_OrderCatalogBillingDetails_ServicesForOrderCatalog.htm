Services for Order Catalog

The Services for Order Catalog tab enables you to associate a billing service or a billing panel to an Order Category. The Order Catalogs are charged based on the prices defined for a billing service or an associated billing panel.

 

Enter the following details to associate Billing Service/Panel to the Order Catalogs:

Order Catalogs

 

  1. Order Category – Select an Order Category from the list. The values are system defined - Dietary, Laboratory, Miscellaneous, Monitoring, Oral Health, OT Surgery, Patient Care, Radiology, Dep Diagnostics and Trmt/Procedures. This is a mandatory field.

  2. Dietary – Select Dietary to define the charges for the Order Catalogs associated to Dietary Services.

  3. Laboratory - Select Laboratory to define the charges for the Order Catalogs associated to Laboratory.

  4. Miscellaneous – Select Miscellaneous to define the charges for any Order Catalog.

  5. Monitoring - Select Monitoring to define the charges for the Order Types associated to the Order Catalog Monitoring.

  6. Oral Health - Select Oral Health to define the charges for the Order Catalogs associated to Oral Health.

  7. OT Surgery - Select OT Surgery to define the charges for the Order Catalogs associated to Operation Theatre.

  8. Patient Care - Select Patient Care to define the charges for the Order Catalogs associated to Patient Care.

  9. Radiology – Select Radiology to define the charges for the Order Catalogs associated to Radiology department.

  10. Dep Diagnostics - Select Dep Diagnostics to define the charges for the Order Catalogs associated to OPD or Cardiology department.

  11. Trmt/Procedures – Select Treatment Procedures to define the charges for the Order Types associated to the Order Catalog Treatment or Procedures.

 

Order Catalogs

  1. Order Type – Click [List of Values] to select an Order Type. The values defined using the function Order Type in the Order Entry and Result Reporting module get listed. The prices are applicable for the selected Order Type in the Order Category. This is a mandatory field.

  2. Order Category – The Order Categories defined in the Order Entry and Result Reporting module get listed based on the Order Category and the Order Type selected.

  3. Order Type – The Order Types associated to the Order Categories get displayed.

  4. Order Catalog Code - The Order Catalog Codes get displayed on selecting the Order Category and Order Type. The description for the respective Order Catalog Code gets displayed. The item code linked to the Order Catalog Code gets displayed. This is a mandatory field.

  5. Item Code - The Item Code for the selected order type gets displayed.

 

Billing Service by Charge Types

  1. Charge Type – Click [List of Values] to select a Charge Type. The values get listed based on the charge types associated to the Order Catalog in the Order Entry and Result Reporting module.

  2. Description - The description for the Charge Type gets displayed.

  3. Indicator – Select an Indicator from the list. The values are system defined - Service or Panel.

  4. Service – Select Service to associate a Billing Service to a Charge Type. The selected Order Catalog is charged based on the prices defined for the Billing Service.

  5. Panel – Select Panel to associate a Billing Panel to a Charge Type. The selected Order Catalog is charged based on the prices defined for the Billing Panel.

  1. Service/Panel – Click [List of Values] to select a billing service/panel. The values differ in the list based on the Indicator selected. The values defined using the Billing Service or the Billing Panel function get listed respectively. This is a mandatory field.

  2. Description - The description for the selected billing service/panel gets displayed.

  3. Default – Select the check box, if the selected Charge Type has to be default for the selected Order Category. Only a single Charge Type is selected as the default entry, at a time.
