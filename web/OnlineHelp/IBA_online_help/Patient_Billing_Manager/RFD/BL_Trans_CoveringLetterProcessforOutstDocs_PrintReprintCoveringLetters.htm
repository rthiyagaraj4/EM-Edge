Enter the following details to print or to reprint the Covering Letter details:

 

  1. Doc.Ref.Date [From–To] – Click Calendar function to select the Document reference Date. The report generated includes the details of the documents generated during the selected date range.

  2. Doc.Ref.No [From–To] – Click [List of Values] to select a range of Document Reference Number. The Covering Letter is printed for the range of document references selected.

  3. Print Insurance Policy No – Select the check box to print the Insurance Policy Number associated to the outstanding document details, if any.  

  4. Print Covering Letter on Letter Head - Select the check box, if the Covering Letter information has to be generated on the Letter Head.

 

Reprint

  1. As Copy – Select As Copy, if the Covering Letter has to be printed as a copy of the original Covering Letter. You can generate a copy of the Covering Letter when a request is placed.

  2. As Original – Select As Original, if the Covering Letter has to be generated as an Original Copy.

  3. As Per Status – Select As Per Status, if the Covering Letter details have to be generated as per the status.
