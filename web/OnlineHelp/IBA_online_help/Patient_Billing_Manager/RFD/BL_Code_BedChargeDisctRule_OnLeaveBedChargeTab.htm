On Leave Bed Charge Discount Rule

You can define the discount rule on the bed and other charge elements to be calculated for the period when the patient goes on leave.  

 

Enter the following details:

  1. Facility – The name of the logged in facility gets displayed, by default.

 

Leave/Stay Rule

The Leave Stay Rule tab enables you to define the charges for the number of days the patient avails leave from the facility.

 

  1. Leave Start Day – Select a value from the list. The system defined values are Leave Day and Stay Day. Select Leave Day, if the day when the patient goes on leave has to be considered as a leave day, so that the bed charges are not computed for the leave days. Select Stay Day, if the day when the patient goes on leave has to be considered as a stay day and the patient has to be charged for the bed.

  2. Leave on Admission Day – Select a value from the list. The system defined values are Leave Day and Stay Day. Select Leave Day, if the patient goes on leave on the admission day and the bed charge is not to be computed. Select Stay Day, if the patient goes on leave on the admission day and the bed charge has to be computed.

  3. Return Day From Leave – Select a value from the list. The system defined values are Leave Day and Stay Day. Select Leave Day, if the day when the patient returns from leave has to be considered as a Leave Day, so that the bed charge is not computed. Select Stay Day, if the day when the patient returns from leave has to be considered as a stay day, so that the bed charge is computed.  

  4. Return From Leave on the Discharge Day – Select a value from the list. The system defined values are Leave Day and Stay Day. Select Leave Day, if the patient returns from leave on the discharge day. The discharge day is selected as a leave day and the bed charges are not computed. Select Stay Day, if the patient returns from leave on the discharge day, but the day is considered to be a stay day and the bed charges are computed.

  5. Multiple Leaves on a Day – Select a value from the list. The system defined values are Leave Day and Stay Day. Select Leave Day, if the patient has taken multiple leaves on a day and bed charges are not to be computed for the selected day. Select Stay Day, if the patient has taken multiple leaves on a day, but the bed charges are to be computed.

 

Normal Day

You can define the discount rule for the leave days of the patient in the facility on a normal day.

 

Discount Rule

  1. Total Leave Duration for the Day [From–To] - Enter the total number of hours in numeric format. The number of hours in a day is considered as the leave duration. The value has to begin with ‘0’. This defines the discount rule for the elements used during the leave duration on a normal day.

  2. Charge Element – The Charge Elements are system defined – ‘Bed Charge’, ‘Doctor Round Charge’, ‘Consumable Charge’, 'Workup Charge’ 'Serv Chrg Excl Adm and Dischrg Days Charge' and 'ICU Charge'. The charges for these elements for reserved bed are calculated based on the charge type and the discount type selected. The leave bed charge for a normal day’s stay of an inpatient in the facility is computed based on the discount issued and the Billing Service price that is associated  to the Bed Type in ‘Inpatient Billing Details’ function. The service price is valid for the inpatient’s stay excluding the admission and discharge days.

  3. Charge Type – Select a value from the list. The values are system defined – ‘As Per Policy Setup’ and ‘Charge to Patient Only’. The elements are charged based on the policy coverage limit specified, if the value is selected as ‘As Per Policy Setup’. The elements are charged on the policy coverage limit specified, if the value is selected as ‘As Per Policy Setup’ and the patient is insured. If the patient is a non-insurance payer, the amount for the element is charged on the patient on the discharge day for the hours the bed is reserved.

  4. Discount Type – Select a value from the list. The values are system defined – Percentage and Amount. Percentage indicates that the percentage of the actual amount is deducted on the element for deceased patient discharge. Amount indicates that the amount mentioned is deducted from the actual amount for the element during deceased patient discharge.

  5. Discount Value – Enter the discount value in numeric format to indicate the amount to be deducted for the elements, which were reserved until the patient arrival.
