Discharge Clearance Format

The Discharge Clearance Format tab enables you to define formats for discharge clearance slips. The format selected in the 'BL Parameters' function gets displayed. You can load the default format. System displays Hospital Details, Patient Details, Discharge Clearance Details and Text Formatting list values in the text field in each tab. Right-click to select a value from the list. Various formats with several values are displayed on selecting a value from the list. Double-click on a value to select it.

The format lists styles to format the legends in header, footer and body section of a receipt format.

The Discharge Clearance Details format lists legend values to display details on discharge clearance slip generation.

The format lists patient demographic details.

The format lists complete information of the hospital and demographic details.

 

Enter the following details to define a format for Discharge Clearance Slip:

 

  1. Legend For Original - Enter the legend for original Discharge Clearance Slip in alphanumeric format. The legend is displayed in the discharge clearance slip when it is printed.

  2. Legend For Original - Enter the legend for Discharge Clearance Slip in alphanumeric format. The legend is displayed in the discharge clearance slip when it is reprinted.

  3. Load the default Setup - Select the check box to load the pre-defined template. A confirmation message 'Existing Setup Will be Replaced, Continue?' is displayed on selecting the check box. Click [Yes] to load the default template or [No] to cancel the process.

Header

  1. [Line 1- Line 6] - Enter the details in the respective fields. The data recorded gets displayed in the header part of the receipts generated. You can also record the data using the pre-defined formats that are available. Right-click on each field to select a value from the pre-defined format. The options available are Hospital Details, Patient Details, Discharge Clearance Details and Text Formatting. Select a value and the pre-defined values recorded for the selected value are listed. Double-click on the value to select it. The selected value is displayed in the respective field.
