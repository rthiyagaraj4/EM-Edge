Cancel

The patient records that match the search criteria specified are displayed. The patient records that are selected for prepaid letter generation through the 'Create' tab are listed. You cannot update the values that are listed. The cancelled records are available for prepaid letter generation.

 

Enter the following details:

  1. Select All - Select the check box to cancel all the requests that are placed for prepaid letter generation. Clear the check box to clear the selection for cancellation.

  2. Select - Select the check box to cancel the request that is placed for prepaid letter generation. Clear the check box if prepaid letter has to be generated.

  3. A confirmation message 'Do you want to cancel the selected records?' is displayed. Click [Yes] to Cancel or [No] to abort the process.

  4. The selected records are deleted and available for prepaid letter generation under the 'Create' tab.
