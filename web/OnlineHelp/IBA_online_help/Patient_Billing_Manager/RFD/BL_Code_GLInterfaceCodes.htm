Enter the following details to define the GL interface codes:

 

  1. Accounting Entity – The Accounting Entity code defined using the ‘Accounting Parameters’ function in the ‘Financial Setup’ menu in the System Manager module gets displayed. The Accounting Entity is set by default and the field is disabled.

  2. GL Interface Code - Enter the GL Interface Code in alphanumeric format of six characters maximum length. The first character of the GL Interface Code has to be ‘I’, ‘C’, ‘D’ or ‘M’. This is a mandatory field.

  3. I – I represents Service Income Accounts.

  4. C – C represents Cash Accounts.

  5. D – D represents Discount Accounts.

  6. R – R represents Receivable Accounts.

  7. M – M represents Miscellaneous Accounts.

  1. Description - Enter the Description for GL Interface Code in alphanumeric characters of 30 characters maximum length. This is a mandatory field.

 

Chart of A/C

  1. A/C Code 1 – Click [List of Values] to select a value for A/C Code 1. The Chart of Account Codes defined in the Chart of Accounts function get listed. This is a mandatory field.

  2. A/C Code 2 - The data is set automatically in the field A/C Code 2 when A/C Code 1 is set. This is a mandatory field.

  3. Acct Dept - The Account Department is set automatically when A/C Code 1 is set. This is a mandatory field.

 

Product Codes

  1. Group - The appropriate Group Code is set automatically when for A/C Code 1 is selected. This is a mandatory field.

  2. Line - The appropriate Line Code is set when a value for A/C Code 1 is selected. This is a mandatory field.

 

Sub Ledger Codes

  1. Group - The Group Code associated to the selected A/C Code 1 gets displayed.

  2. Line - The Line Code associated to the selected A/C Code 1 gets displayed.

  3. Valid for BL – Select the check box, if the General Interface Code details are valid for Patient Billing Manager module.

  4. GL Interface Type - The description for the selected value appears when the A/C Code1 is selected. The respective data in the General Ledger module for GL Interface Type appears.

 

Descriptions

  1. Chart of A/C - The description for the selected Chart of Account gets displayed. The values get listed from the General Ledger module.

  2. Acct. Department - The description for the selected Account Department gets displayed. The values get listed from the General Ledger module.

  3. Product - The description for the selected Product Codes gets displayed.

  4. Sub Ledger - The description for the selected Sub ledger Codes gets displayed.

 

The data in the above mentioned fields get displayed automatically only when the Chart of Accounts has a valid entry in the General Ledger module.
