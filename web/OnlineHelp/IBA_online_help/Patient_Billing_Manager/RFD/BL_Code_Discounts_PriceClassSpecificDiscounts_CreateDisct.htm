Create Discounts

The Create Discounts tab enables you to define the percentage of discount for a specific Price Class.

 

Enter the following details to define the price:

 

  1. Price Class – Click [List of Values] to select the Price Class. The description for the selected Price Class gets displayed. This is a mandatory field.

  2. Indicator - Select an attribute from the list. The value in the drop-down list is 'Billing Service'. This is a mandatory field.

  3. Billing Service [From-To] – Click [List of Values] to select a range of Billing Service. This is a mandatory field.

  4. Valid [From-To] – Click Calendar function to select a date range. The discount percentage is applicable for the selected Billing Service and the patient class from the date selected. This is a mandatory field.

 

Discount %

  1. IP Rate - Enter the percentage of discount valid for Inpatients. This is a mandatory field.

  2. OP Rate - Enter the percentage of discount valid for Outpatients. This is a mandatory field.

  3. Emergency Rate - Enter the percentage of discount valid for Emergency cases. This is a mandatory field.

  4. External Rate - Enter the percentage of discount valid for External/Referral patients. This is a mandatory field.
