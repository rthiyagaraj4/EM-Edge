Counseling Details

You can associate a template to the patient and record the details based on the questionnaires in the template. You can associate additional counselling details to the template, if the parameter is selected for the respective template. You can exempt the bill amount of a patient and finalize the details or permit the patient to settle the bill amount in installment payments and finalize the details. You have an option to print, view or save the details.

 

Enter the following details to record and finalize the counselling details:

 

  1. Exemption Allowed – Select the check box to exempt the bill details of a patient. Clear the check box, if exemption is not applicable.

  2. Assign to – Select a user from the list. The users associated to the logged in facility get listed. The selected user is authorized to login and exempt the bill details using the ‘Bill Exemption’ function. You can select a value from the list only if the 'Exemption Allowed' check box is enabled.

  3. Installment Allowed – Select the check box to permit the patient to settle the bill amount in installments. Clear the check box, if installment payment is not applicable.

  4. Assign to – Select a user from the list. The users associated to the logged in facility get listed. The selected user is authorized to login and record installment details for the patient using ‘Record Credit Agreement’ function. You can select a value from the list only if the 'Installment Allowed' check box is enabled.

  5. Select Counseling Format – Select a Counselling Format from the list. The values defined using the ‘Setup Counseling Formats’ function get listed. The questionnaires get listed based on the Counselling Format selected. You can associate additional questionnaires, if the value for ‘New Entry Allowed During Trx’ is selected as ‘Yes’ while defining the format.

  6. Copy Counseling From Encounter - Select a value from the list to copy the encounter details from the previous encounter. The details are listed only if the counseling details are recorded for the patient in the previous encounter. The counseling number format is displayed based on the Episode Type, Encounter ID and Visit ID.

  7. Finalize – Select the check box to finalize the details recorded. You can select this check box, if ‘Exemption Allowed’ or ‘Installment Allowed’ check box is selected.

  8. [Preview] – Click [Preview] to print, view or save the details.

  9. The Format ID, Code and Description of the Counselling record, type, value, UOM and  sequence number selected are displayed. If ‘Type’ is ‘Free Text’ for the respective record, detail description is also displayed.
