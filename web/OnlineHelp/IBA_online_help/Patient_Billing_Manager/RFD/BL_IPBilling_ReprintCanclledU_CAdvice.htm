Enter the following details to reprint the cancelled undercharges advice:

 

  1. Episode Type - Select the Episode Type from the list. The values are system defined - “Daycare” and “Inpatient”.

  2. Daycare – Select Daycare to re-print the cancelled bills for Daycare patients.

  3. Inpatient - Select Inpatient to re-print the cancelled bills for Inpatients.

  1. Report Type - Select a Report Type from the list. The values are system defined. The values are “Single Bill - Cancelled”, “Range of Cancelled Bills by Bill No”, “Range of Bills by Cancelled Date”, “Range of Cancelled Bills by Customer” and “All”.

  2. Single Bill - Cancelled - You can select and re-print a single cancelled undercharge advice, which is printed already for an Inpatient or a Daycare patient.

  3. Range of Cancelled Bills by Bill number – You can select to print a range of cancelled undercharge advice by selecting the Bill Numbers. The Bill Number fields under the “From” and “To” sections are enabled.

  4. Range of Bills by Cancelled Date – You can select to print a range of cancelled undercharge advice by selecting the bill generation date. The “Bill Date” fields under the “From” and “To” sections are enabled.

  5. Range of Cancelled Bills by Customer – You can select to print the cancelled undercharge advice by selecting the Customer. The field ‘Customer’ is enabled.

  6. All Range – You can select to print all the Cancelled undercharge advice printed during the specified date range. The Bill Number and the Bill Date fields are enabled on selecting ‘All Range’. You have to specify the Bill Number and the Bill Dates.

  1. Customer – Click [List of Values] to select a Customer. The values defined for the Customer get listed. This field is enabled only when the Report Type is selected as “Range of Cancelled Bills by Customer”.

  2. Bill Number [From-To] - Click [List of Values] to view and to select a list of the bill numbers. A Cancelled Bill Lookup screen appears. Press [F8] or click [Execute Query] and the bills for the selected bill number range gets displayed. Double-click a record to select it. The Cancelled Bill Number associated to the patient ID appears automatically in the field along with the description.

  3. Cancelled Date – You can select a range of Cancelled Date only when the Report Type is selected as “Range of Cancelled Bills by Customer”, “Range of Cancelled bills by Bill Date” and “All”. The Cancelled bill details for the selected date range get listed.

  4. Bill Type - Select a value from the list. The system-defined values are 'Interim Bill - Cancelled', 'Discharge Bill - Cancelled' and 'Undercharge Advice - Cancelled'. The value 'Undercharge Advice - Cancelled' is displayed, by default.

 

Cancelled Bill Details

The Bill Document Type Code, Document Number, Cancelled Date, Cancelled bill amount, patient ID, bill nature, Billing Group, Customer Code, Episode Type, Episode ID and Visit ID  gets displayed based on the search criterion specified.

  1. Patient Name - The patient name associated to the respective patient ID gets displayed.
