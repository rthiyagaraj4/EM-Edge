Enter the following details to exempt the bill amount:

 

  1. Document Ref – Enter the Document Reference details in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  2. Date – Click Calendar function to select a transaction date. The date entered has to be greater than the current system date. This is a mandatory field.

  3. Reason – The reason selected while logging in to exempt the bill details is displayed, by default.

  4. Remarks – Enter the Remarks in alphanumeric format of 100 characters maximum length. This is a mandatory field.

  5. Document No – The Document Type selected for Bill Exemption in 'BL Parameters' function is displayed. The document number is system generated.

  6. Date - The system date and time get displayed automatically.

  7. Current Stage – The first stage defined for the exemption code gets displayed.

  8. Next Stage - The next stage defined for the exemption code gets displayed.

  9. [Authorize] – Click [Authorize] to login  and to exempt the bill amount for the selected patient. An authorization screen pops-up requesting confirmation. The username and password of the logged-in user is displayed, by default. Click [List of Values] to select a reason for Bill Exemption. The values recorded through the 'Reasons for Billing Actions' function with the 'Bill Exemption' check box as selected are displayed.
