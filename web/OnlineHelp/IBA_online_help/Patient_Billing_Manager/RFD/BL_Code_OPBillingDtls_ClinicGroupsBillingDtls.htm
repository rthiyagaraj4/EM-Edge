Clinic Groups Billing Details

A Clinic Group is a cluster of individual Clinics. A Clinic Group defined helps in assessing the revenue generated from the relevant Clinic Group within a Facility.

 

Enter the following details to define the Clinic Groups:

 

  1. Clinic Group – Enter the Clinic Group code in alphanumeric format of four characters maximum length. The Clinic Group code is unique and cannot be changed after creation. This is a mandatory field.

  2. Long Description – Enter the Long Description for the Clinic Group code in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the Short Description for the Clinic Group code in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Max Permitted Elapsed Period since Previous Visit – Enter the maximum time period permitted to the patient to visit the Clinic Group. The maximum time period is calculated since the last visit. Patient is able to register the next visit to the Clinic Group within the time set.

  5. Elapsed Period Unit – Select a value from the list. The values are system defined – 'Days' and 'Hours'.

  6. Days – Select Days, if the patient is able to visit the Clinic Group within the specified number of days from the last visit. If the patient visits after the days specified, a fresh visit has to be registered where the patient receives a new patient ID. The patient is not entitled to any discount or exemptions during the last visit, if any.

  7. Hours – Select Hours, if the patient is able to visit the Clinic Group within the specified number of hours from the last visit. If the patient visits after the hours specified, a fresh visit has to be registered, where the patient receives a new patient ID. The patient is not entitled to any discount or exemptions during the last visit, if any.

  1. Cut Off Time – Enter the Cut Off Time in numeric format. The Cut Off Time indicates the time for the visit to elapse to the relevant Clinic Group. For example, consider the ‘Cut Off Time’ is set to 02:30 hrs and the ‘Max Permitted Elapsed Period since Previous Visit’ is selected as four hours. The visit is considered valid for the selected Clinic upto 06:30 hrs. If the ‘Cut Off Time’ is set to 02:30 hrs and the ‘Max Permitted Elapsed Period since Previous Visit’ is selected as 2 days, the visit is considered valid from 02:30 hrs and is valid for next two days upto the same time.

  2. Exempt Hospital Fee – Select the check box to exempt the hospital fee for the number of exempted outpatient visits to the facility after checkout. Clear the check box, if the hospital fees cannot be exempted for a visit to the respective Clinic. The check box is selected, by default.

  3. Exempt Medical Pers Fee – Select the check box to exempt medical personnel fees for the respective Clinic. Clear the check box, if medical personnel fees cannot be exempted for a visit to the respective Clinic. The check box is selected, by default.

You have to select either ‘Exempt Hospital Fee’ or ‘Exempt Medical Pers Fee’ check box.
