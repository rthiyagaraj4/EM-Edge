Enter the following details to define Financial Reclassification Parameters:

 

  1. Automatic Basic Classification in Use – Select the check box, if financial reclassification is applicable during patient transactions in the facility. The values for the fields in this function have to be entered, if the check box is selected. Clear the check box, if the patient reclassification is not applicable. If the check box is cleared, entry in other fields is not mandatory. This is a one-time setup.

  2. Max-Assessment Period – Enter the maximum assessment period in numeric format of three digits maximum length. The assessment period is specified in months. It defines the maximum period for which the financial reclassification rule is valid.

  3. Pre-Expiry Renewal Period - Enter the number of days in numeric format of two digits maximum length. This indicates the number of days before the expiry date from when the financial details can be renewed.

  4. Max.Period Allowed for Provisional Classification - Enter the maximum period allowed for re-classification in numeric format of three digits maximum length. The period is specified in days. It defines the maximum period for provisional classification.

  5. Child Age Upto - Enter the child age in numeric format of two digits maximum length. This indicates the limit on the child’s age for which reclassification is valid.

  6. Include Dependent Income in Total Income - If this checkbox is selected, the dependent income is included in total income. The total income is sum of patient income and spouse income and dependent income. If this checkbox is not selected, the total income will not include dependent income, so that the total income is sum of patient income and spouse income alone. By default, the checkbox is not selected.

 

Future Classification for Excluded Encounters  

  1. Start Date within Next – Enter the start date for future encounter classification in numeric format of three digits maximum length. This is applicable for future encounters.

  2. Max.Period Allowed number of days - Enter the maximum period in numeric format of two digits maximum length. This indicates the maximum period for which the classification rule is applicable for future encounters.
