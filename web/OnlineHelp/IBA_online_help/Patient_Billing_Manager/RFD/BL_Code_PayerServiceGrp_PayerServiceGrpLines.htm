Payer Service Group Lines

You can define the Payer Service Groups and Payer Service Group Lines using this tab. The services against which the payer settles the bill is defined. The information is also used for bill print.

 

Enter the following details to define Payer Service Group Lines:

 

  1. Payer Group – Click [List of Values] to select a valid Customer Group. The values defined using the function ‘Customer Group’ function in AR Setup menu gets listed. The Service Group settings defined is applicable to the Payer Group selected. This is a mandatory field.

  2. Patient Class - Click [List of Values] to select a valid Patient Class. To select all the Patient Class enter ‘**’. The Service Group settings defined is applicable to the Patient Class associated to the Payer. This is a mandatory field.

 

Payer Service Groups

  1. Service Group - Enter a unique alphanumeric Service Group Code not exceeding two characters. You can associate multiple individual services under a Service Group. This is a mandatory field.

  2. Long Description – Enter a Long Description for the Service Group. The Long Description defines the relevant Service Group Code. This is a mandatory field.

  3. Short Description – Enter a Short Description for the Service Group. This is a mandatory field.

  4. Print Seq No. – Enter the print sequence number in numeric format of three digits maximum length. This indicates the order in which the Payer Service Groups have to be printed in the Invoice report generated.

  5. Mark Up/Mark Down – Select the check box to indicate the Mark up value for a specific Payer Service Group. You can select a single record as the default record to display the mark up value in the report generation.

 

Payer Service Group Lines

  1. Service Group – The Service Group code is set, by default, based on the Payer Service Group selected. To edit and to select the Payer Service Group Line details for any other Service Group, click on the relevant record under Payer Service Groups.

  2. Service Group Line – Enter the Service Group Line Code in alphanumeric format with a maximum of three characters. This is a mandatory field.

  3. Long Description – Enter a Long Description for the Service Group Line. The Long Description defines the relevant Service Group Line Code. This is a mandatory field.

  4. Short Description – Enter a Short Description for the Service Group Line. This is a mandatory field.

  5. Doc Print Seq – Enter the sequence number for each of the Service Group lines. The sequence identifies the order in which the document should be printed. This is a mandatory field.

  6. Non-DRG Related - Select the check box, if the Payer Service Group is Non-DRG related. This is applicable only for inpatient transactions. Clear the check box, if the Payer Service Group is DRG related.
