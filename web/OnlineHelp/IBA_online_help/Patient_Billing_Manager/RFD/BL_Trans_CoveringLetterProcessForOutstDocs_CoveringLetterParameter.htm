Enter the following details to define a Covering Letter:

 

Letter Body Header

  1. Line 1 – Enter the text in alphanumeric format, which has to appear as the header of the letter. This is a mandatory field.

  2. Line 2 - Enter the text, which has to appear as the header in the letter. This is a mandatory field.

  3. Line [3, 4, 5] – Enter the text, which has to appear as the header of the letter.

 

Letter Body Footer

  1. Line 1 – Enter the text in alphanumeric format, which has to appear as footer of the letter body. This is a mandatory field.

  2. Line 2 - Enter the text in alphanumeric format, which has to appear as footer of the letter body. This is a mandatory field.

  3. Line [3, 4, 5] – Enter the text, which has to appear as the footer of the letter body.

 

Letter Footer Note

  1. Line [1-5] – Enter the text in alphanumeric format. The text appears as a note in the letter footer.

  2. Authorized Signatory – Enter the name of the Authorized Signatory. This is a mandatory field.

  3. Designation – Enter the Designation of the Authorized Signatory.

  4. Print Outstanding Documents only in Annexure – Select the check box, if the outstanding documents are to be printed.
