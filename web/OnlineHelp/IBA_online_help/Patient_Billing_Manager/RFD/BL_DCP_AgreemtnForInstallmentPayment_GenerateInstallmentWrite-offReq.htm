Enter the following details to search for the records, which are to be written-off:

 

  1. Outstanding Period – Enter the number of days in numeric format. The outstanding document details for the number of days specified get listed. The outstanding period gets displayed as 0 and 999999, by default.

  2. Option – Select a value from the list. The system defined values are ‘Donot Include Rejections’ and ‘Include Rejections’.

  3. Donot Include Rejections - Select Donot Include Rejections to exclude the documents for which the write-off was rejected.

  4. Include Rejections - Select Include Rejections to list the documents for which the write-off was rejected earlier.

  1. Nationality – Select a value from the list. The values are system defined. Select either “Citizen”, “Non-Citizen” or “All” from the list.

  2. Citizen – Select Citizen to write-off the outstanding documents of the citizens only.

  3. Non-Citizen – Select Non-Citizen to write-off the outstanding documents of the non-citizens.

  4. All - Select All to write-off the outstanding documents of the all the patients in the facility.  

  1. Write-off Process Date – The current system date is displayed as the Write-off Process Date.

  2. Doc No – The Document Number is generated automatically. The ‘Write-off Document Type’ selected under the ‘Write-off’ tab in the BL Parameters function get displayed.

The outstanding document details matching the search criterion and for which the agreement details are created get displayed.

  1. Select All – Select the check box to generate Write-off for the selected records. Select the check boxes manually  to generate write-off for selected records. The reason for write-off gets displayed on selecting the check box. The reason code specified while login to authorize the write-off process gets displayed.

  2. [Process] – Click [Process] to generate Write-off for the selected individual records. A confirmation message gets displayed. Click [Yes] to generate a write-off for the selected records or [No] to cancel the process.
