Enter the following details to associate an Accounting Department and a Service Location to the Facility:

 

  1. Billing Service – Click [List of Values] to select a Billing Service. The description for the Billing Service gets displayed in the adjacent text box.

 

Acct.Dept/Location by Facility

  1. Facility – Select a Facility from the list. The Facilities associated to the logged in User using the Responsibilities for User function in the System Manager module get listed.

Click [Insert Record] to associate an Accounting Department and a Service Location to charge the Billing Service in any other Facility.

  1. Acct Dept - Click [List of Values] to select an Accounting Department. The values defined for Accounting Department in the Application Masters module and associated to the Facility get listed. This is a mandatory field.

  2. Description - The description for the selected Accounting Department gets displayed.

  3. Service Location - Click [List of Values] to select a Service Location. The Service Locations associated to the Facility get listed. This is a mandatory field.

  4. Description - The description for the selected Service Location gets displayed.
