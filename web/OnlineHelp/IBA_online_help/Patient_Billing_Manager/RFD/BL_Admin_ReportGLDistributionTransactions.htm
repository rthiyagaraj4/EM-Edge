Enter the following details to generate the report:

 

Daily/Monthly Report

  1. Daily – Select Daily to print a Daily Report.

  2. Monthly – Select Monthly to generate a Monthly Report.

  3. Transaction Date [From–To] – Click Calendar function to select a Transaction Date. A report is generated listing the transactions posted during the selected date range. This is applicable only if the daily/monthly report is selected as daily. This is a mandatory field.

  4. Processing Period [From–To] – Click Calendar function to select the processing period. A report is generated listing the transactions posted during the selected date range. This is applicable only if the daily/monthly report is selected as monthly.

  5. Report Type - Select a value from the list. The values are system defined - ‘Detail’, ’Summary’ and ‘Both’.

  6. Detail – Select Detail to generate a detailed report.

  7. Summary - Select summary to generate a summary of the transactions during the selected report.

 

Report Order

  • Trx Doc Date – Select Trx Doc Date to sort the report based on the Transaction Document Date.

  • Trx Doc Ref – Select Trx Doc Ref to sort the report based on the Transaction Document Reference details.

 

Posting A/C Range

  1. Chart of A/C [From–To] – Click [List of Values] to select a range of Chart of Account codes. The description for the selected Chart of Account appears in the adjacent text box. The report is generated for the selected Chart of account codes.

  2. Department [From–To] - Click [List of Values] to select a range of Department codes. The report is generated for the selected range of Departments.

  3. Product [From–To] - Click [List of Values] to select a range of Product codes. The report is generated for the selected range of Products. The Product description is displayed in the adjacent text box.

  4. Subledger [From–To] – Click [List of Values] to select a range of Subledger codes. The report is generated for the selected range of Subledger. The description for the selected Subledger is displayed in the adjacent text box.
