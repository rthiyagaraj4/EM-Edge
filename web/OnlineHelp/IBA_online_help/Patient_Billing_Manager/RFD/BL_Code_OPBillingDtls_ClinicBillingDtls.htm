Clinic Billing Details

Clinic Billing Details tab enables you to define the billing details for a Clinic and select the charging stage for an Outpatient in each Clinic.

 

Enter the following details to define billing details for an Outpatient Clinic:

 

  1. Clinic – Click [List of Values] to select a Clinic. The values defined using the function Clinics in the Outpatient Management gets listed. This is a mandatory field.

  2. Billing  Grp Code - Click [List of Values] to select the billing group code. If the Insurance Group radio button in the OP Parameter screen is selected, this field is displayed

  3. Description - The description for the selected Clinic gets displayed.

  4. Visit Type - Click [List of Values] to select the visit type. This field is mandatory.

  5. Description - The description for the selected visit type gets displayed.

  6. Bill Type - Click [List of Values] to select a Bill Type. The values defined using the Bill Type function in the Accounting Entity menu gets listed. This is a mandatory field.

  7. Description -The description for the selected Bill Type gets displayed.

  8. Billing Class – Click [List of Values] to select a Billing Class. The values defined using the Billing Class function in the Billing Rules menu get listed. This is a mandatory field.

  9. Customer Group Code – Click [List of Values] to select the Customer Group Code. If the Insurance Group radio button in the OP Parameter screen is selected, this column is displayed.

  10. Customer Code - Click [List of Values] to select the Customer Code. If the Insurance Group radio button in the OP Parameter screen is selected, this column is displayed.

  11. Description - The description for the selected Billing Class gets displayed.

  12. Days exempted for Revisits - Enter a numeric value with a maximum of three digits to indicate the days exempted for a re-visit to the Clinic.

  13. No of Follow-up Visits Exempted – Enter the number of follow-up exempted in numeric format of two digits maximum length. This indicates the number of exempted visits to the Clinic after patient check out.

  14. Exempt Hospital Fee – Select the check box to exempt hospital fees for the respective Clinic. Clear the check box, if the hospital fees cannot be exempted for a visit to the respective Clinic. The check box is selected, by default. A confirmation message 'Visit charge exempted upto dd/mm/yyyy hh:mm' is displayed during Visit Registration or Checkout in 'Ambulatory Care Management' and 'Accident and Emergency' modules, if the check box is selected.

  15. Exempt Medical Pers Fee – Select the check box to exempt medical personnel fees for the respective Clinic. Clear the check box, if the medical personnel fees cannot be exempted for a visit to the respective Clinic. The check box is selected, by default.

You have to select either ‘Exempt Hospital Fee’ or ‘Exempt Medical Pers Fee’ check box.

  1. Charging Basis by Practitioner – Select the check box, if the Practitioner charges have to be considered for the respective Clinic. Clear the check box, if the Practitioner charges are not applicable.

  2. Hospital Fee Charging Stage – Select a value from the list. The values are system defined – 'At Registration/Assign Practitioner' and 'At Close Consultation'.

  3. At Registration/Assign Practitioner – Select At Registration/Assign Practitioner to charge the patient during Visit Registration or while assigning a Practitioner to the patient in the respective Clinic.

  4. At Close Consultation – Select At Close Consultation to charge the patient when the Consultation to the specific Clinic is closed.

  1. Medical Pers Fee Charging Stage – Select a value from the list. The values are system defined – 'At Registration/Assign Practitioner' and 'At Close Consultation'. This column is visible, only if the check box ‘Consultation Charge in Multiple Stages’ is selected in BL Parameters function under the ‘Outpatient/Emergency/External’ tab. On selecting the check box, professional charge is displayed on recording the financial details in Register Visit, Revise Visit, Close Consultation and Register Attendance, Revise Attendance Details and AE Checkout functions in ‘Outpatient Management’ and ‘Accident and Emergency’ modules respectively.

  2. At Registration/Assign Practitioner – Select At Registration/Assign Practitioner to charge the patient during Visit Registration or while assigning a Practitioner to the patient in the respective Clinic.

  3. At Close Consultation – Select At Close Consultation to charge the patient when the Consultation to the specific Clinic is closed.

  1. [Visit Charge Rule] - Click [Visit Charge Rule] to associate a Visit Type and a Billing Service to the Clinic. The tab is shifted to Visit Charge Rule.
