Enter the following details to amend prices:

 

  1. Facility – Select the Facility from the list. The facilities associated to the logged in User get displayed. The prices defined are valid for the facility selected.

  2. Billing Service – Click [List of Values] to select a Billing Service. The description for the selected Billing Service gets displayed. This is a mandatory field.

  3. Apply To – Select a value from the list. The values are system defined– All, Standard, Price Class and Billing Group. This is a mandatory field.

  4. All – Select All to update either the Price Class, Billing Group or Customer level charges defined for a patient class and associated to a Billing Service.

  5. Price Class – Select Price Class to update the price and discount details defined for a patient class, which is associated to a selected Price Class and a Billing Service.

  6. Billing Group – Select Billing Group to update the price and discount details defined for a patient class, which is associated to a selected Billing Group and a Billing Service.

  7. Standard – Select Standard to update the standard prices defined for a patient class, which is associated to a selected Billing Service and a Facility.

 

Click [Execute Query] or press [F8] to view all the entries associated to the relevant Billing Service. To create a new record, click [Create Record] and enter the following details:

 

  1. Indicator – Select a value from the list. The values are system defined– Standard, Customer, Billing Group and Price Class. This is a mandatory field.

  2. Standard – Select Standard to update the standard prices defined for a patient class and associated to a facility. The Standard prices already defined get listed. To insert a new record, click [Create Record]. This is a mandatory field.

  3. Customer – Select Customer to update Customer Specific Prices defined for a patient class and associated to a facility. The prices already defined get listed. To insert a new record, click [Create Record]. This is a mandatory field.

  4. Billing Group – Select Billing Group to update the Billing Group Specific Prices defined for a patient class and associated to a facility. The prices already defined get listed. To insert a new record, click [Create Record]. This is a mandatory field.

  5. Price Class – Select Price Class to update the Price Class Specific Prices defined for a patient class and associated to a facility. The prices already defined get listed. To insert a new record, click [Create Record]. This is a mandatory field.

  1. Code – Click [List of Values] to select a code. The values in the list differ based on the Indicator selected. This is a mandatory field.

  2. The price and discount percentage defined for a patient class and associated to the selected Billing Service and Billing Group get displayed on selecting Billing Group. To define the price for a new Billing Group, select a value from the list. The values defined using the Billing Group function get listed.

  3. The price and discount percentage defined for a patient class and associated to the selected Billing Service and Facility get displayed on selecting Standard as the indicator. To define a new standard price, select a facility from the list. The Facilities associated to the logged in User get listed.

  4. The price and discount percentage defined for a patient class and associated to the selected Billing Service and Customer get displayed on selecting Customer as the indicator. To define the price for a new Customer, select a value from the list. The values defined using the Customer function get listed.

  5. The price and discount percentage defined for a class and associated to the selected Billing Service and Price Class get displayed on selecting Price Class as the indicator. To define a price for a new Price Class, select a value from the list. The Price Class defined using the Price Class tab in the Price Class Specific Price function gets listed.

  1. Valid From – Click Calendar function to select a date for new records. The price defined is valid for the selected patient class, Indicator and the Billing Service from the date selected. The date gets displayed automatically when the price is defined for the patient class indicator selected and associated Billing Service using the Price Class Specific Prices or Billing Group/Customer Specific Price function. This is a mandatory field.

  2. Valid To - Click Calendar function to select a date for new records. The price defined is valid for the selected patient class, Indicator and the Billing Service until the date selected. The date gets displayed automatically when the price is defined for the patient class indicator selected and associated Billing Service using the Price Class Specific Prices or Billing Group/Customer Specific Price function.

 

Pricing by Billing Class for Billing Service

  1. Billing Class – Click [List of Values] to select a Billing Class. The Description for the selected Billing Class gets displayed. This is a mandatory field.

  2. R/F – Select a value from the list. The values are system defined – Rate and Factor.

  3. Rate – Select Rate, if the service price has to be charged for the patient class associated to the selected indicator and the Billing Service.

  4. Factor – Select Factor and the service price is multiplied with the factor entered for a patient class associated to the selected indicator and the Billing Service.

  1. IP Rate – The price defined for the Inpatients get displayed. You can update the price defined by entering the new price valid for the selected patient class. The text box is enabled, if the selected Billing Class is an Inpatient.

  2. DC Rate - The price defined for the Daycare patients gets displayed. You can update the price defined by entering the new price valid for the selected patient class. The text box is enabled, if the selected Billing Class is a Daycare patient.

  3. OP Rate - The price defined for the Outpatients gets displayed. You can update the price defined by entering the new price valid for the selected patient class. The text box is enabled, if the selected Billing Class is an Outpatient.

  4. Emg. Rate – The price defined for the Emergency patients gets displayed. You can update the price defined by entering the new price valid for the selected patient class. The text box is enabled, if the selected Billing Class is valid for Emergency patients.

  5. Ext. Rate – The price defined for the External/Referral patients gets displayed. You can update the price defined by entering the new price valid for the selected patient class. The text box is enabled, if the selected Billing Class is valid for External/Referral patients.

  6. Min Charge – Enter the minimum charge for the service in numeric format. The price defined for a patient class has to be equal to or higher than the minimum charge. This is a mandatory field.

  7. Base Qty – Enter the Base Quantity towards the service in numeric format. The price for the patient class associated to the Billing Service is calculated as the product of service price and the base quantity.

  8. Type – Select a value from the list to identify the reimbursable type. The system-defined values are – ‘Full’, ‘Partial’ and ‘NA’.

  9. Full – Select Full to refund the entire amount paid by the patient towards the service.

  10. Partial – Select Partial to refund only a fraction of the amount paid towards the service. A part of the amount is deducted for the service availed and the remaining amount is refunded to the patient/payer. You have to specify the amount to be reimbursed in the field ‘Reimbursable Amount’ under the ‘General’ tab. The Charge is specified only if the ‘Charge Type’ is set to ‘Charge’.

  11. Not Applicable – Select NA, if the service is not reimbursable.

  1. Allowed – Select the check box, if the Billing Service is reimbursable for the selected patient class. Clear the check box, if the respective Billing Class is non-reimbursable for the selected patient class.

  2. Reimbursable Amt – Enter the amount, which can be reimbursed to the patient.

  3. Override – Select the check box, if the respective Billing Service is reimbursable for the selected patient class. The override option for the patient class takes precedence over the override check box in the header field. If the service is not reimbursable and if the check box is selected for the patient class, it indicates that the Billing Service is reimbursable for the selected patient class.

  4. Gross/Net Applicability – Select a value from the list. The values are system defined – Gross and Net.

  5. Gross - Select Gross to charge the Gross amount for the service. The charge towards the service is calculated based on the discount and the price defined for the patient class associated to an indicator and a Billing Service.

  6. Net – Select Net to charge the Net amount for the service. The service charge is calculated based on the price defined for the patient class, associated to an indicator and a Billing Service. The discount amount for the service is not considered during bill generation.

  1. Check Cost availability – Select the check box, if the Cost Amount is applicable for the selected patient class, associated to an indicator and a Billing Service. Clear the check box, if the Cost Amount is not considered towards the charge calculation.

  2. Apply Discnt – Select the check box, if Discount is applicable for the selected Billing Class. You can define or update the discount percentage, if the check box is selected. Clear the check box, if the discount is not applicable for the patient class associated to the Billing Service.

 

Applicable Discounts for Service and Billing Class

The text boxes are enabled, if the ‘Apply Disc’ check box is selected.

  1. Discount Indicator – Select a value from the list to issue discount for the Billing Service. The system-defined values are ‘Amount’ and ‘%’. Select Amount to specify the amount that has to be issued as a discount amount on the Billing Service. Select Percentage to specify the percentage of amount that has to be issued as a discount amount on the Billing Service.

  2. IP - (%/Amt) - Enter the value to issue discount towards the service for Inpatients. The price is calculated based on the percentage of amount and the service price defined. If the discount percentage is already defined, the values get displayed. Enter the new discount percentage to update the existing values in numeric format. The value is computed as amount or discount based on the value selected for 'Discount Indicator'. This is a mandatory field.

  3. DC - (%/Amt) - Enter the value to issue discount towards the service for Daycare patients. The price is calculated based on the percentage of amount and the service price defined. If the discount percentage is already defined, the values get displayed. Enter the new discount percentage to update the existing values in numeric format. The value is computed as amount or discount based on the value selected for 'Discount Indicator'. This is a mandatory field.

  4. OP - (%/Amt) - Enter the value to issue discount towards the service for Outpatients. The price is calculated based on the percentage of amount and the service price defined. If the discount percentage is already defined, the values get displayed. Enter the new discount percentage to update the existing values in numeric format. The value is computed as amount or discount based on the value selected for 'Discount Indicator'. This is a mandatory field.

  5. Emg - (%/Amt) - Enter the value to issue discount towards the service for Emergency patients. The price is calculated based on the percentage of amount and the service price defined. If the discount percentage is already defined, the values get displayed. Enter the new discount percentage to update the existing values in numeric format. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.This is a mandatory field.

  6. Ext - (%/Amt) - Enter the value to issue discount towards the service for External/Referral patients. The price is calculated based on the percentage of amount and the service price defined. If the discount percentage is already defined, the values get displayed. Enter the new discount percentage to update the existing values in numeric format. The value is computed as amount or discount based on the value selected for 'Discount Indicator'. This is a mandatory field.

  7. Valid From – Click Calendar function to select a date. The discount percentage defined for the patient class is valid from the date selected. The discount period has to be within the Billing Service period and has to be greater than the system date. This is a mandatory field.

  8. Valid To - Click Calendar function to select a date. The discount percentage defined for the patient class is valid until the date selected. The discount period has to be within the Billing Service period. The date selected has to be greater than the Valid From date. This is a mandatory field.
