The following details are displayed:

 

  1. Rounding Option – Select a Rounding Option from the list. The system defined value is “Rounded to Integer”.

  2. Print Line – The Print Line details for the services associated to the Service Group get displayed. The description for the selected Print Line is displayed.

  3. Print Line Percentage – Enter the discount percentage if a percentage discount is to be given. The percentage entered cannot exceed 100.00. The value entered cannot be negative.

  4. Gross Amount – The Gross Amount for the service gets displayed.

  5. Service Disc Amount – The Service Discount Amount gets displayed.

  6. Print Line Disc % - Enter the discount percentage for a service. The discount amount is computed and displayed based on the discount percentage recorded. If you record the 'Discount Amount' without entering the 'Print Line Disc%', system computes and displays the service level discount percentage automatically. This is a mandatory field.

  7. Discount Amount – The Amount cannot be entered if a discount percentage has been entered. In such a case, amount is computed and displayed. The system computes the discount amount as (Disc %/100.00) * (Gross - Serv Disc). The system also checks to see whether the discount amount entered is greater than the (Gross - Serv Disc). If you record the 'Print Line Disc%' without entering the 'Discount Amount', system computes and displays the service level discount amount automatically. This is a mandatory field. You can set discount on the entire bills where the bill outstanding amount is zero.

  8. Net Amount – The Net Amount is calculated as (Gross Amount - Discount Amount).

  9. GL Interface Code – Click [List of Values] to view and select a valid G/L Interface code. If the code is valid, the short description is displayed for user verification. If print line level discount is entered, a valid G/L Interface code has to be entered to record the discount amount in General Ledger. The interface code has to begin with "D" only.

  10. Rounding Option – Select a Rounding Option from the list. The system defined value is “Rounded to Integer”.

  11. Gross Amount – The total Gross Amount for the services utilized by the patient is displayed.

  12. Overall Disc % - Enter the value in numeric format to issue a bill level discount. The percentage entered cannot exceed 100.00. The value entered cannot be negative. An authorized user is permitted to issue a bill level discount. System computes and displays the overall discount amount automatically based on the discount percentage issued or vice versa.

  13. Disc Amt - The discount amount is displayed.

  14. Overall Discount Amount – The Amount cannot be entered if a discount percentage has been entered. System computes the overall discount amount automatically based on the discount percentage. Discount amount is computed as - (Disc %/100.00) * (Gross - Serv Disc - Man Disc). System also checks to see whether the discount amount entered is greater than the (Gross - Serv Disc - Man Disc). The overall discount percentage is computed automatically if you enter the overall discount amount.

  15. Prepay Adj Amt - The amount adjusted against the prepayment accepted from the patient during the admission gets displayed.

  16. Dep Adj Amt - The amount adjusted against the deposit accepted from the patient during the admission gets displayed.

  17. Manual Discount – The manual discount given (if any) to the patient gets displayed.

  18. Total – The total amount gets displayed.

  19. Rounding Amt - The total rounding amount is displayed.

  20. Bill Total Amount – The total amount after the discount for the specific bill gets displayed.

 

The system will display all the bills generated for the patient for the selected episode including zero bills, negative bills and outstanding bill.

The system will validate the gross bill amount with the discount amount even if the discount is already given. The system provides further discount for that bill if the discount amount is less than gross the bill amount.

The bill outstanding amount will be negative if the outstanding amount of the discounted bill is zero.

The system checks the total discount amount and the gross bill amount. If the discount amount is more than gross amount, the system displays an error message. The total bill discount amount cannot be more than gross bill amount. The negative outstanding bill amount will be refunded to the patient through Settle Bill Outstanding function only.
