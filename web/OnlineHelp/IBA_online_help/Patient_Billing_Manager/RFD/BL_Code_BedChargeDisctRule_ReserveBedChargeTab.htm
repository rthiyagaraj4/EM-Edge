Reserve Bed Charge Discount Rule

You can define the discount on the elements reserved for a patient from the registration date until the date it is occupied.

 

Enter the following details:

 

  1. Bed Type – The Bed Type is displayed as ‘**’, by default. The discount rule is applicable for all the beds in the Nursing Unit. Click ‘Create Record’ icon to define the discount rule for a selected bed in the facility.

  2. Facility – The name of the logged in Facility is displayed.

 

Total Reserved Hours

  1. From – To – Enter the total hours in numeric format. The beginning hour has to start with ‘0’. This defines the discount rule for the hours the bed is reserved in the facility before the patient arrival.

 

Charge Elements

The Charge Elements define the discount on the elements that were reserved until the patient arrival.

 

  1. Charge Element – The Charge Elements are system defined – ‘Bed Charge’, ‘Doctor Round Charge’, ‘Consumable Charge’, 'Workup Charge’ 'Discharge Day Charge', 'Serv Chrg Excl Adm and Dischrg Days Charge' and 'ICU Charge'. The charges for these elements for reserved bed are calculated based on the charge type and the discount type selected. The reserve bed charge is computed based on the Billing Service associated to the Bed Type in ‘Inpatient Billing Details’ function. The discount value set is valid for the services that are associated to the Bed Type in ‘Inpatient Billing Details’ function. The reserve bed charge for an inpatient is computed based on the discount issued. The discount amount is computed on the price defined for the Billing Service that is associated to the Bed Type in the ‘Inpatient Billing Details’ function. The service price is valid for the inpatient’s stay excluding the admission and discharge days.

  2. Charge Type – Select a value from the list. The values are system defined – ‘As Per Policy Setup’ and ‘Charge to Patient Only’. The elements are charged based on the policy coverage limit specified, if the value is selected as ‘As Per Policy Setup’. The elements are charged on the policy coverage limit specified, if the value is selected as ‘As Per Policy Setup’ and the patient is insured. If the patient is a non-insurance payer, the amount for the element is charged on the patient on the discharge day for the hours the bed is reserved.

  3. Discount Type – Select a value from the list. The values are system defined – Percentage and Amount. Percentage indicates that the percentage of the actual amount is deducted on the element for deceased patient discharge. Amount indicates that the amount mentioned is deducted from the actual amount for the element during deceased patient discharge.

  4. Discount Value – Enter the discount value in numeric format to indicate the amount to be deducted for the elements, which were reserved until the patient arrival.
