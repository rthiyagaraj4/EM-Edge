Enter the following details to define a default Billing Group:

 

Default Billing Groups for Encounter Registration

  1. Care Locn.Type – Select the value from the list. The values are system defined – 'Procedure Unit', 'Clinic', 'Daycare' and 'Nursing Units'. The values for Location differ based on the Care Location Type selected. This is a mandatory field.

  2. Procedure Unit – Select Procedure Unit to define a default Billing Group for a Procedure Unit.

  3. Clinic – Select Clinic to define a default Billing Group for a Clinic.

  4. Daycare – Select Daycare to define a default Billing Group for a Daycare Unit.

  5. Nursing Units – Select Nursing Units to define a default Billing Group for a Nursing Unit.

  1. Emergency – Select the check box, if the default Billing Group is applicable to Emergency patients admitted to the selected Location Type.

  2. Location – Click [List of Values] to select a Location Type. The values differ based on the Care Location Type selected. This is a mandatory field.

  3. The Procedure Units defined in the Outpatient Management module get listed, if the Care Location Type is selected as Procedure Unit. The default Billing Group is applicable for the selected Procedure Unit.

  4. The Clinics defined in the Outpatient Management module get listed, if the Care Location Type is selected as Clinics. The default Billing Group is applicable for the selected Clinic.

  5. The Daycare Units defined in the Outpatient Management module get listed, if the Care Location Type is selected as Daycare Units. The default Billing Group is applicable for the selected Daycare Unit.

  6. The Nursing Units defined in the Inpatient and Day Care Management module get listed, if the Care Location Type is selected as Nursing Units. The default Billing Group is applicable for the selected Nursing Unit.

  1. Location Name – The Name of the selected Location gets displayed.

  2. Priority Zone – Select a Priority Zone from the list. The values are system defined– 'Red Zone', 'Yellow Zone', 'Green Zone', 'Brought Dead', 'Unassigned', 'Any' and 'Not Applicable'. This is a mandatory field.

  3. Category - Select the Category from the list. The values are system defined – 'Unrestricted', 'Nationals' and 'Others'.

  4. Unrestricted – Select Unrestricted, if the default Billing Group is valid for all the patients admitted in the selected Location. The patient can be a valid citizen of the country or a non-national.

  5. Nationals – Select Nationals, if the default Billing Group is valid for the citizens of the nation. The default Billing Group is valid for is not applicable for other patients admitted to the location.

  6. Others – Select Others, if the default Billing Group is valid for the non-national patients.

  1. Billing Group - Click [List of Values] to select a Billing Group. The selected Billing Group is set as the default Billing Group. This is a mandatory field.

  2. Description – The Description for the selected Billing Group gets displayed.
