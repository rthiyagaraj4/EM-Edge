Charging Limits

The Charging Limits tab enables you to define the maximum amount that can be charged per encounter for a specific Billing Group and the associated patient category.

 

Enter the following details to define the Charging Limits:

  1. Billing Group – Click [List of Values] to select a Billing Group. The values defined using the Billing Group function get listed. The charging limit is valid for the selected Billing Group and the associated patient category. This is a mandatory field.

  2. Description – The Description for the selected Billing Group gets displayed.

  3. Level of Service – Enter the Level of Service in numeric format not exceeding two digits. The level of service indicates the priority. This is a mandatory field.

  4. Category – Select the Category from the list. The values are system defined – 'Unrestricted', 'Nationals' and 'Others'.

  5. Unrestricted – Select Unrestricted, if the charging limit for the selected Billing Group is valid for all the patients visiting the facility. The patient can be a valid citizen of the country or a non-national.

  6. Nationals – Select Nationals, if the charging limit for the selected Billing Group is valid for the citizens of the nation. The charging limit is not applicable for other patients visiting the facility.

  7. Others – Select Others, if the charging limit for the selected Billing Group is valid for the non-national patients.

  1. Max.Amt per Encounter – Enter the maximum amount to be charged for a particular encounter for the relevant Billing Group and the associated patient category. This is a mandatory field.

  2. [Interface to GL] – Click [Interface to GL] to switch over to the Interface to GL tab.

 

Interface to GL

GL Interface

  1. Excess Charge A/C – Click [List of Values] to select an Account Interface Code. The posting account defined against this GL Interface code is credited while posting billing charges to General Ledger module. This is a mandatory field.

  2. Description – The Description for the selected Account Interface Code gets displayed.

  3. Eff From Date – Click Calendar function to select a date. The charge account selected for the Billing Group is valid from the date selected. The date selected has to be greater than the system date. This is a mandatory field.
