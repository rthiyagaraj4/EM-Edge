Service Charge By Bed Type

The Service Charge by Bed Type tab enables you to define the charges for transfers between the bed types in a Nursing Unit.

 

Enter the following details to define the service charge for a Bed Type:

 

  1. Fr Bed Type – Click [List of Values] to select a Bed Type. The values defined using the Bed Type function in the Inpatient and Day Care Management get listed. This is a mandatory field.

  2. From Bed Type Desc - The description for the selected Bed Type gets displayed.

  3. To Bed Type - Click [List of Values] to select a Bed Type. The values defined using the Bed Type function in the Inpatient and Day Care Management get listed. The value selected for ‘To Bed Type’ has to be greater than the value selected for ‘Fr Bed Type’. This is a mandatory field.

  4. To Bed Type Desc - The description for the selected bed type gets displayed.

  5. Perc/Amt – Select a value from the list. The values are system defined – percentage and amount.

  6. Percentage – Select Percentage to charge the transfers in terms of percentage. The charge for the selected bed type transfer is calculated as the percentage of the service amount associated.

  7. Amount - Select Amount and the price defined for the associated service is charged as the amount for the selected bed type transfer.

  1. Billing Service – Click [List of Values] to select a Billing Service. The values defined using the Billing Services function get listed. A transfer from one bed type to another is charged based on the prices defined for the Billing Service associated.

  2. Per/Amt of Fr or To Charge – Select a value from the list. The values are system defined - To and From.

  3. To – Select To and the bed type selected as ‘To Bed Type’ is charged, that is, the bed type to which the patient is transferred is charged. The bed type from where the patient is transferred is not charged.

  4. From – Select From and the bed type selected as ‘Fr Bed Type’ is charged, that is, the bed type from where the patient is transferred is charged. The bed type from where the patient is transferred is not charged.

  1. %/Charge Amt - Enter the charge amount in numeric format. The field is enabled when the ‘Perc/Amt’ is selected as percentage. The charge for the selected bed type transfer is calculated as the percentage of the service amount associated. This is a mandatory field.
