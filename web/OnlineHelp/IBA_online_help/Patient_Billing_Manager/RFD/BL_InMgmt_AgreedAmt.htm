Enter the following details to record the payment:

 

  1. Payer Group Code - Click [List of Values] to select a Payer Group. The values defined using the ‘Customer Groups’ function under the AR Setup menu gets listed. This is a mandatory field.

  2. Payer - Click [List of Values] to select the Payer. The Customers associated to the selected Customer Group using the function ‘Customer’ under the AR Setup menu gets listed. This is a mandatory field.

  3. Policy - Click [List of Values] to select a Policy. The policies associated to the selected Payer and the Payer Group using the function ‘Setup Service Coverage Detail by Policy’ function in the Insurance Management module get listed. This is a mandatory field.

  4. Applicable for Normal Invoice Generation - Select the check box if the Payer Group, Payer and Policy are valid for normal invoice generation. The check box is selected, by default.

  5. Applicable for Regeneration of Invoice - Select the check box if the Payer Group, Payer and Policy are valid for regeneration of invoice. The check box is cleared, by default.

  6. Varies by Treatment Month - Select the check box to define varying invoice amount for subsequent months. The invoice is generated based on the agreement defined for the treatment. This is valid for the Payer Group, Payer and the associated Policy Type during the period selected. Clear the check box, if the invoice agreement is not required for the treatment. The Payer has to pay the invoice amount based on the agreed amount. On selecting the check box, 'Treatment' tab is enabled using which you can define the agreed invoice amount for subsequent months.

  7. Agreed Amount Applicability – Select a value from the list. The system-defined values are ‘Invoice Level’ and ‘Limited to Specific Payer Service Group’.

  8. Invoice Level – Select ‘Invoice Level’ to record the agreed amount based on the invoice level. The agreed amount is valid for the respective Payer Group, Payer and Policy combination irrespective of the Payer Service Group.

  9. Limited to Specific Payer Service Group/Service/Item – Select ‘Limited to Specific Payer Service Group/Service/Item’ to record the agreed amount for a specific Payer Service Group. The agreed amount is valid for the associated Payer Service Group only. If the Payer Service Group is selected, the agreed amount is valid for the selected Payer Service Group only. You cannot settle the amount against any other Payer Service Group during invoice generation.

  1. Effective From – Click Calendar function to select the date. The Payer pays the agreed amount to the facility for the selected policy coverage from the date selected. This is a mandatory field.

  2. Effective To – Click Calendar function to indicate the date until when the invoice amount is being paid by the Payer to the facility for the selected Policy Coverage.

  3. Agreement Indicator - Select a value from the list. System defined values are - 'Mark Up', 'Mark Down' and 'Both'.

  4. Mark Up Only - Select Mark Up amount to select the highest amount as the Invoice amount. For example, consider the invoice amount to be 15,000 (inclusive of Debit and Credit details) and the agreed amount as 10,000. In such a scenario, invoice is raised for the highest amount i.e15,000.

  5. Mark Down Only - Select Mark Down amount to select the lowest amount as the Invoice amount. For example, consider the invoice amount to be 15,000 (inclusive of Debit and Credit details) and the agreed amount as 10,000. In such a scenario, invoice bill is raised for 10,000.

  6. Both - Select Both to consider Mark Up and Mark Down while raising an invoice.

  1. Agreed Invoice Amt – Enter the amount that the Payer has agreed to pay to the facility for the selected Policy during the period specified.

Service/Item/Payer Service Group Level

  1. Type - Select a value from the list to define coverage amount for an Item, Payer Service Group or Service. The system-defined values are 'Item', 'Payer Service Group', and 'Service'.

  2. Service/Item/Payer Service Group - Select a value from the list. You can define the coverage amount either for the Services, Items or Payer Service Groups.

  3. Rate/Fixed Indicator - Select a value from the list. The system-defined values are 'Rate' and 'Fixed'.

  4. Amount - Enter the amount in numeric format. The amount for the quantity of items differ based on the indicator selected. If the 'Indicator' is selected as 'Rate', the value is computed as (amount*quantity used). If the 'Indicator' is selected as 'Fixed', the amount specified is considered irrespective of the quantity used.

Treatment

This tab is enabled only if the check box 'Varies by Treatment Month' is selected.

  1. Month [From-To] - Enter the month to define the agreed amount. A invoice is generated every month for the agreed amount and posted to the payer/patient.

  2. Agreement Indicator - Select a value from the list. System defined values are - 'Mark Up', 'Mark Down' and 'Both'.

  3. Mark Up - Select Mark Up amount to select the highest amount as the Invoice amount. For example, consider the invoice amount to be 15,000 (inclusive of Debit and Credit details) and the agreed amount as 10,000. In such a scenario, invoice is raised for the highest amount i.e15,000.

  4. Mark Down - Select Mark Down amount to select the lowest amount as the Invoice amount. For example, consider the invoice amount to be 15,000 (inclusive of Debit and Credit details) and the agreed amount as 10,000. In such a scenario, invoice bill is raised for 10,000.

  5. Both - Select Both to consider Mark Up and Mark Down while raising an invoice.

  1. Agreed Invoice Amt – Enter the amount that the Payer has agreed to pay to the facility for the selected Policy during the period specified.
