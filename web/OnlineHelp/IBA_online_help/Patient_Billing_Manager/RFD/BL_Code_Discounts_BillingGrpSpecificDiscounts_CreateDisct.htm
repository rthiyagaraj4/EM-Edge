Create Discounts

The Create Discounts tab enables you to define the discount percentage applicable for the patients associated to the selected Billing Group.

 

Enter the following details to define discounts:

 

  1. Billing Group – Click [List of Values] to select a Billing Group. The description for the selected Billing Group gets displayed. This is a mandatory field.

  2. Indicator - Select an Attribute from the list. The values defined are 'Billing Service'. This is a mandatory field.

  3. Billing Service – Click [List of Values] to select a range of Billing Service. This is a mandatory field.

  4. Valid – Click Calendar function to select a date range. The discount percentage is applicable for the patient category and the Billing Service associated to the Billing Group for the selected date range. This is a mandatory field.

 

Discount %

  1. IP Rate - Enter the percentage of discount valid for Inpatients and the range of Billing Services associated to the Billing Group. This is a mandatory field.

  2. OP Rate - Enter the percentage of discount valid for Outpatients and the range of Billing Services associated to the Billing Group. This is a mandatory field.

  3. Emergency Rate - Enter the percentage of discount valid for Emergency patients and the range of Billing Services associated to the Billing Group. This is a mandatory field.

  4. External Rate - Enter the percentage of discount valid for External/Referral patients and the range of Billing Services associated to the Billing Group. This is a mandatory field.
