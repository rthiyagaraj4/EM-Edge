Enter the following details to view the charges:

  1. Facility – Select a valid Facility from the list. The records can be accessed for the relevant Facility.

 

Search Criteria

  1. Records – Select a value from the list. The values are system defined– 'All Records' and 'Todays Records'.

  2. All Records – Select All Records to display all the records irrespective of the date.

  3. Todays Records – Select Todays Records to display current day’s record.

  4. Modules - Select a value from the list. The values are system defined– 'All Modules' and 'Specific Modules'.

  5. All Modules – Select All Modules to display the records from all the modules.

  6. Specific Modules – Select the Specific Modules to display the records from the selected modules.

  1. Pr.Module – Click [List of Values] to select a Primary Module. The text box is enabled only when the Modules is selected as Specific Modules.

  2. Episode Type – Select an Episode Type from the list. The Episode Type contains system defined values – 'Inpatient', 'Day Care', 'Outpatient', 'Emergency' and 'External'.

  3. Patient – Select either 'All Patient' or 'Specific Patient' from the list. The values are system defined.

  4. All Patient – Select All Patient to display records of all the patients admitted to the facility.

  5. Specific Patient - Select Specific Patient to display records of selected patients admitted to the facility.

  1. Patient No – Enter the Patient ID whose records have to get listed. If the Patient is selected as ‘Specific Patient’, the text field is enabled and you can enter the Patient ID.

  2. Encounter – Select an Encounter from the list. The values are system defined – 'All Encounters' and 'Specific Encounters'.

  3. Specific Encounters – Click [List of Values] to select a Specific Encounter.

  4. All Encounters - Select All Encounters to list all the encounters.

  5. Pr. Key Main – Enter the Primary Key details.

  6. Process – Select Process to proceed with charging of the relevant record. A pop-up message gets displayed inquiring whether you want to charge the item. Click [Yes] to charge and [No] to cancel the transaction.
