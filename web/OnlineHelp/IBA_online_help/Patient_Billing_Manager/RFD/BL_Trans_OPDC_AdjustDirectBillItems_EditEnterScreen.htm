You can update only the amount for the services associated to the patient.

 

  1. Order Catalog – The Order Catalog code and description gets displayed, if any.

  2. Panel – The Billing Panel Code and Description gets displayed, if the Billing Panel is associated to the patient and has to be updated.

  3. Service – The Service code and the description gets displayed for which the selected transaction reference ID was generated.

  4. Included/Excluded as Per Policy – The value is either displayed as Included or Excluded based on the Policy Type associated to the patient, if the patient is an Insurance holder. This is valid only for Insurance Payers.

  5. Rate/Charge – The value gets displayed as ‘R’ or ‘C’, based on the service selected. If the service is Rate based, ‘R’ gets displayed. If the service is charge based, the value is displayed as ‘C’.

  6. Original Qty per Panel – The original quantity of the respective service while defining the panel gets displayed. You cannot update the details.

  7. Revised Qty per Panel – If the service quantity was updated while associating the panel, the revised quantity gets displayed.

  8. Total Qty – The total quantity of the respective service associated gets displayed.

  9. Amount – Enter the amount for the service. You can update only the amount for the service.

  10. Practitioner/Staff – The value is displayed as ‘Practitioner’ or ‘Staff’ based on the value selected for indicator while associating the service to the patient.

  11. Practitioner/Staff ID – The name of the associated Practitioner or Staff gets displayed.

  12. Total Payable – The Total Payable amount towards the service gets displayed. This is based on the entry made in the ‘Amount’ field.

  13. Patient Payable – The amount to be settled by the patient, if the patient is an Insurance holder, gets displayed.

  14. Patient Paid – The amount settled by the patient for the services gets displayed.

  15. Pre-Approval Required – The check box is selected or cleared based on the definition for the Policy Type.

  16. [Accept] – Click [Accept] to save the details.

  17. [Cancel] – Click [Cancel] to cancel the details.

  18. [Add More Service] – Click [Add More Service] to add additional services.

  19. [Remove Service] – Click [Remove Service] to remove the associated services.
