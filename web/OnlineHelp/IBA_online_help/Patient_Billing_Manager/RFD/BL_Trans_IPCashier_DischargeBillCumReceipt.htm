Enter the following to generate discharge bill cum receipt:

  1. Admission No – Enter the Admission Number in numeric format of eight digits maximum length. This field is mandatory. The episode type is displayed.

  2. Patient No. – Enter the patient number in alphanumeric format of 20 characters maximum length. On entering valid patient number, the details of name, Gender and age will be displayed. Patient lookup is provided.

  3. Facility – Select the value from the list. For the logged in user ID, associated facilities will be displayed as list items can be selected.

  4. Bill Generation Option - Select a Bill Generation Option from the list. The system-defined values are 'Both', 'Patient' and 'Customer'.

  5. Inpatient Bill Type – Select a Bill Type to generate the bill. The values are system defined – “Pending Interim Bill”, “Pending Discharge Bill” and “Pending Undercharge Bill”. The Inpatient Bill Type differs based on the 'Discharge Bill Stage' and the 'Default Inpatient Bill Type' selected in 'BL Parameters' function.

  6. If the 'Discharge Bill Stage' is selected as 'After Disch. Adv./Before Discharge' and 'Default Inpatient Bill Type' is selected as 'As Per Patient Status' in 'BL Parameters' function:

=> If discharge advice is generated for the patient, default value for 'Inpatient Bill Type' is set as 'Pending Discharge Bill'.

=> If discharge advice is generated for the patient, default value for 'Inpatient Bill Type' is set as 'Pending Interim Bill'.

  • If the 'Discharge Bill Stage' is selected as 'After Disch. Adv./Before Discharge' and 'Default Inpatient Bill Type' is selected as 'Pending Discharge Bill' in 'BL Parameters' function:

=> If discharge advice is generated for the patient, default value for 'Inpatient Bill Type' is set as 'Pending Discharge Bill'.

=> If discharge advice is not generated for the patient, an error message is displayed indicating the Cashier to generate discharge advice.

  1. Customer – If any Customer is associated with the Patient during registration, the details are displayed. Else, you can click [List of Values] to set the Customer with the Patient record. You can select a Customer from the list when the Bill Generation Option is set to Customer.

  2. Facility - The facility id must be selected for the patient who has undertaken the service in the concerned facility.

  3. Age - The age of the patient will be shown in years, months and the days.

  4. Sex - The Gender of the patient will be indicated in the radio button as Male/Female.

  5. User – The name of the user logged-in to the Cash Counter is displayed.

  6. Cash Counter/Shift – The Cash Counter code along with the Shift details is displayed.

  7. [Admn. Remarks] - Click [Admn. Remarks] to view the remarks that were recorded for a patient during admission to the facility through 'Admit Patient' or any modifications made through the 'Change Admission Details' function. You can only view the remarks and cannot update the existing details.

  8. [Draft Report] - Click [Draft Report] to view an intermediate draft report before generating the discharge advice. A pop-up message is displayed. Click [OK] to view the draft report or [Cancel] to cancel the process.

 

Search Patient Visit

This tab is used to search the inpatient admission for selection.

 

  1. Location Type – The values are system defined – “Clinic” and “Ward”. Select one of the values from the list. The details are displayed for the Location Type selected.

  2. Location Code – Select a Location Code from [List of Values]. The values are populated in the list based on the Location Type selected.

  3. Episode Type – The values are system defined. Select a value from the list.

  4. Option – The values selected for the ‘Inpatient Bill Type’ is defaulted.

  5. Encounter ID – Enter the Encounter ID of the respective patient. The visit details for the specific encounter are displayed.

  6. Encounter Date Range – Click Calendar function to set the date. This field is mandatory. The details for the encounter between the specified date range are displayed.

  7. Patient No – Enter the Patient Number in alphanumeric format of 20 characters, to see the admission of particular patient.

  8. NRIC – Enter the NRIC number for the respective Patient ID.

  9. Customer [From-To] – Click [List of Values] to select the customers range. The search visits of the customers will be displayed.

  10. Ward Code – Enter the Ward code in alphanumeric format of four characters maximum length. This can be selected from the list of valid values. Leave blank if all the records need to be displayed.

  11. [Search] – Click this button to display the details in the detail block. The details displayed will be of Admission Date, Patient No, Ward Code and description, episode type, Admission No and Unbilled amount. To select a particular admission number for raising receipt, double-click on the specific record from the details displayed.

Record Adhoc Discount

You can grant a discount for the individual services availed by the patient. Right-click on a selected record under the Discount Column to grant a discount for the respective service. You can specify the value as either percentage or the actual discount amount.  An authorization screen pops-up on clicking Adhoc discount option. The authorised users only can login to update the discount. The reasons defined using the function Reasons for Billing Actions in the Masters with the ‘Service Level Discount’ as selected gets displayed. The Net Amount for the service is calculated as ‘Gross Amount-Discount’. Click [OK] to login or [Cancel] to cancel the transaction.

The Gross Amount, Net Amount, Additional Amount for the services and Discount Amount if any towards the selected service gets displayed.

  1. User ID – Enter the User ID in alphanumeric format. A authorised user can only login to update the discount details. This is a mandatory field.

  2. PIN – Enter the PIN number for the respective user. This is a mandatory field.

  3. Use Prev Disc Reason – Select the check box to issue a discount for the same reason selected during the previous transaction.

  4. Adhoc Discount Amt – Enter the discount for the selected service in numeric format.

  5. Actual Gross Amt – The gross amount for the service gets displayed.

  6. Discount Reason – Click [List of Values] to select the Discount Reason. The reasons defined using the function ‘Reasons for Billing Actions’ in the Masters with the ‘Service Level Discount’ as selected gets displayed.

Outstanding bills

This page lists all the details of the outstanding bills and generates the consolidated receipts. The details of Facility, where the patient has undergone the services, Document Type, number, outstanding amount and settled amount will be displayed.

Double-click on the detail to make the consolidated receipt.

 

Unbilled Admission

  • The aim of the tab page is to provide the information for the unbilled episodes across the facilities and to bill the corresponding episode ‘Unbilled services’. Click on the Unbilled admission to list the ‘Unbilled admission’.

  • In the first tab page under the ‘Unbilled admission details’ will be ‘Unbilled Admission’, which list all the admissions which are not billed.

  • Select the admission to view all the services for that admission. Click on the sibling tab ‘Unbilled service request of an unbilled admission’. The sibling tab will display all the services for that selected admission.

  • When clicking on the sibling tab ‘Unbilled service request of an unbilled admission’, if there is any ‘Unconfirmed services’, the system will generate the warning and never allow to bill the services for that selected admission.

  1. Facility Id - The facility ID where the patient has undergone the services.

  2. Episode date - The facility id where the patient has undergone the services.

  3. Episode type - The episode type will be ‘O’ for the outpatients, ‘E’ for Emergency and ‘R’ for the ‘External patients’.

  4. Episode no. - The episode number of the patients encounter gets displayed.

  5. Visit no. - The visit number of the patients encounter gets displayed.

  6. Gross Amt - Total gross amount for all the services gets displayed.

  7. Discount Amt - Total discount amount for all the services gets displayed.

  8. Net. Amt - Total Net. Amount for all the services gets displayed.

 

Unbilled service request of an unbilled admission

The aim of the tab page is to provide the information for the unbilled services for the selected episode in the previous sibling tab page.
