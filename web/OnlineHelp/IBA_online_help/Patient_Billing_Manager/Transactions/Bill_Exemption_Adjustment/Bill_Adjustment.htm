Bill Adjustment

From this function you can Adjust the bill amount for a relevant patient. You can Query to view the outstanding bill amount for all the services availed from the facility.

Bill Adjustment function is used to adjust or modify the outstanding bill amount for a patient. You can also view the adjustments against a bill or cancel the adjustments. An authorized User only can adjust the bill amount. The user is able to login and adjust the bill amount only when the ‘Bill Adjustment’ check box is selected in the ‘Users for Authorization/Refund/Cash Counters’ function in the Administration menu. The ‘Bill Adjustment Doc Type’ selected under the ‘Adj/Exmptn’ tab in the BL Parameters function gets displayed. A document reference number is generated on committing the record. You can adjust the amount for the outstanding bills, which are not settled. The bill amount, which is adjusted, gets displayed in different color.

Bill Adjustment

From the Transactions > Bill Exemption/Adjustment, double-click Bill Adjustment Function.

The Bill Adjustment screen gets displayed.

  1. Select a Patient Number for transaction or use the Lookup function to select a patient. You can search for a relevant patient ID in the lookup screen. Click [Enter Query], enter the search criteria and click [Execute Query] or press [F8] to select a patient ID. The patient details get populated in the respective fields.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Authorize] to login and to authorize the bill adjustment for the selected patient.

    Enter the following details:

    1. User ID – Enter the User name in alphanumeric format. This is a mandatory field.

    2. PIN No – Enter the PIN number for the selected patient. This is a mandatory field.

    3. Reason – Click [List of Values] to select a Reason for exemption. The reasons defined using the function ‘Reasons for Billing Actions’ in the Masters with the ‘Bill Exemption’ check box as selected get listed. This is a mandatory field.

     

  4. You can create, cancel and view the bill adjustment details for a patient using the following tab pages:

Create

Outstanding Bills

The outstanding bill details associated to the selected patient get displayed.

  1. Doc Number – The document type and the number for the outstanding document get displayed. The date when the document number was generated for the respective outstanding document also gets displayed.

  2. Episode Type – The Episode Type gets displayed.

  3. Billing Grp – The Billing Group for the selected outstanding document gets displayed.

  4. Cust Code – The Customer Code associated to the selected outstanding document gets displayed.

  5. Outstanding Amt – The outstanding amount for the selected document type gets displayed.

  6.  %age – Enter the percentage of amount in numeric format. The adjustment amount is calculated as the percentage of the actual amount.

  7. Adjustment Amt – The adjustment amount gets displayed automatically based on the percentage and the actual outstanding amount.

  8. Remarks – Enter the remarks in alphanumeric format.

  9. BG – The description for the respective Billing Group gets displayed.

  10. Cust – The description for the respective Customer code gets displayed.

  11. Total – The total amount adjusted against the outstanding bills of a patient gets displayed.

  12. Select – Select the check box to enter the percentage of amount to be adjusted for each outstanding bill. If the check box is cleared, you cannot enter the percentage value to adjust the bill details.

Click  to view the outstanding bill in detail. The Outpatient Query page gets displayed listing the amount against the services availed from the facility.

 

Cancel

Click Cancel to cancel the bill adjustments recorded using the Create tab.

  1. Document No – Click [List of Values] to select and to cancel a document number. The document numbers created using the Create tab for the patients’ get listed. The Patient Name, ID, Age, Gender of the Patient and the Reason selected to cancel the adjustments get displayed.

  2. Cancel Remarks – Enter the cancellation remarks in alphanumeric format of 100 characters maximum length. This is a mandatory field.

Click [Execute Query] or press [F8] to view the adjustment details recorded against an outstanding bill. The outstanding amount to be paid by the patient on cancelling the adjustment against an outstanding bill amount gets displayed.

  1. [Authorize] – Click [Authorize] to login and to cancel the bill adjustments for the selected patient.

A confirmation screen pops-up. Click [Yes] to cancel the adjustments or [No] to abort the process and to exit from the screen.

View

Enter the following details to search for a patient record and to view the exemption details:

  1. Patient No – Enter the Patient ID or click [List of Values] to search for a patient record using the lookup screen. The patient Name, Age and Gender get displayed.

  2. Doc. [From–To] – Click [List of Values] to select a range of documents generated for the adjustments granted to the selected patient.

  3. Doc. Date [From–To] – Click Calendar function to select the date range. The document reference numbers generated during the selected date range for the selected patient get displayed.

  4. [Search] – Click [Search] to view the exemption details and the respective outstanding bill details against which the adjustment is granted.

  1. Click [Commit Form] to record the details or [Clear Record] to clear the entered data.
