Single Bill Adjustment

From this function you can Adjust the bill amount for a relevant patient. You can Query to view the outstanding bill amount for all the services availed from the facility.

Bill Adjustment function is used to adjust or modify the outstanding bill amount for a patient. You can also view the adjustments against a bill or cancel the adjustments. An authorized User only can adjust the bill amount. The user is able to login and adjust the bill amount only when the ‘Bill Adjustment’ check box is selected in the ‘Users for Authorization/Refund/Cash Counters’ function in the Administration menu. The ‘Bill Adjustment Doc Type’ selected under the ‘Adj/Exmptn’ tab in the BL Parameters function gets displayed. A document reference number is generated on committing the record. You can adjust the amount for the outstanding bills, which are not settled. The bill amount, which is adjusted, gets displayed in different color.

    General sub-tab

    1. Patient ID – Enter the Patient ID in alphanumeric format of 10 characters maximum length. The records matching the patient ID entered get displayed.

    2. NRIC (National Registration Identity Card) – Enter the National ID in numeric format of 15 digits maximum length. The records matching the National ID of the patient recorded during registration get displayed.

    3. Passport Details – Enter the passport details.

    4. Title - Enter the Title in alphanumeric format of 15 digits maximum length. The records matching the Title of the patient recorded during registration get displayed.

    6. First Name – Enter the first name of the patient to search for a specific patient record. The records matching the patient’s first name get displayed.

    7. Contact No - Enter the contact number of the patient to search for a specific patient record.

    8. Contact Mode – Select the Contact Mode from the drop-down list. The listed values are E-Mail, Fax, HP, Mobile No, Other, Police, Postal, Primary Resi No, Tel (Home) and Tel (Office).

    9. Nationality – Click [List of Values] to select the nationality.

    10. Birth Date – Select the Birth Date using the calendar function.

    11. Place of Birth – Click [List of Values] to select place of birth.

    12. Gender – Select Gender from the drop-down.

    13. Race - Click [List of Values] to select Race.

    14. Order By – Select Order By from the drop-down list. The listed values are Patient ID, Patent Name, Sex, Date of Birth and NRIC. The patient details are ordered based on the selected value.

    15. Click [Search] to view the patient details or click [Cancel] to close the screen.

Outstanding Bills

  1. Select All – Select this check box to select all the records to be adjusted for outstanding bill.

  2. Select – Select the check box to enter the percentage of amount to be adjusted for each outstanding bill. If the check box is cleared, you cannot enter the percentage value to adjust the bill details.

  3. Doc Number – The document type and the number for the outstanding document get displayed. The date when the document number was generated for the respective outstanding document also gets displayed.

  4. Doc Date – Select the document date using calendar function.

  5. Episode Type – The Episode Type gets displayed.

  6. Billing Grp – The Billing Group for the selected outstanding document gets displayed.

  7. Cust Code – The Customer Code associated to the selected outstanding document gets displayed.

  8. Outstanding Amt – The outstanding amount for the selected document type gets displayed.

  9. %age – Enter the percentage of amount in numeric format. The adjustment amount is calculated as the percentage of the actual amount.

  10. Adjustment Amt – The adjustment amount gets displayed automatically based on the percentage and the actual outstanding amount.

  11. Remarks – Enter the remarks in alphanumeric format.

  12. BG – The description for the respective Billing Group gets displayed.

  13. Cust – The description for the respective Customer code gets displayed.

  14. Total – The total amount adjusted against the outstanding bills of a patient gets displayed.

Click to view the outstanding bill in detail. The Outpatient Query page gets displayed listing the amount against the services availed from the facility.
