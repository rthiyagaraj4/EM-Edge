Re-print IP Service Item List

From this function you can Reprint service item list or lab service item list for an Inpatient in the Facility.

Reprint OP Service Item List function is used to reprint service list or lab service list utilized by an Inpatient in the facility. You can either select to print the laboratory services or other services utilized by an Inpatient. The total amount to be settled, patient payable and payer payable amount are displayed in the report generated. The service item list and lab service item list can be printed only if a bill is generated for the services utilized.

Reprinting IP Service Item List

From the Transactions > IP Billing, double-click Re-print IP Service Item List Function.

The Reprint IP Service Item List screen gets displayed.

  1. Click [List of Values] to select the Bill Document type and number. Services for which the bills are generated are listed with the bill document type and number. This is a mandatory field.

  2. Select the check box 'Print Service Item List' to print the service items utilized by the patient. Clear the check box, if the lab service item list has to be printed.

  3. Select the check box 'Print Lab Service Item List' to print the laboratory service items utilized by the patient. Clear the check box, if other service item list has to be printed.

  4. Click [Run Process] to print the service item list or [Clear Record] to clear the entered data.
