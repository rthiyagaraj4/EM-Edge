Print IP Under Charges Advice – With Settlement

From this function you can Print advice for Inpatient/Daycare patients with the settlement details. A report is generated for the selected Episode Type based on the parameters set.

Print IP Under Charges Advice – With Settlement function is used to print the under charges advice for the selected patients at the Facility. This report does not include the settlement details for the selected patient. You can choose to print from a range of bills based on the Bill Date, Bill Number, Customer or print all the under charges advice generated. An option is also provided to list and print all the under charges advice or only the bills, where the entire amount was not settled during patient discharge. You can either select to print an interim bill, a discharge bill or an undercharge advice without the settlement details.

Print IP Under Charges Advice – With Settlement Details

From the Transactions > IP Billing, double-click Print IP Under Charges Advice – With Settlement Function.

The Print IP Under Charges Advice – With Settlement screen gets displayed.

  1. Select an Episode Type to print the undercharges advice without settlement details for the Inpatients or the Daycare patients in the facility. The undercharges advice generated for the selected Episode Type gets displayed. <Related Field Description>

  2. Click [Process Run] to print the undercharges advice or [Clear Record] to clear the entered data.

  3. A bill print authorization screen pops-up. Enter the User ID, PIN Number and the Remarks to login and to re-print the undercharge advice. Click [OK] to login or [Cancel] to exit from the screen.

  4. A confirmation screen pops-up on clicking [OK]. Click [Yes] to re-print the bills or [No] to cancel the printing process.
