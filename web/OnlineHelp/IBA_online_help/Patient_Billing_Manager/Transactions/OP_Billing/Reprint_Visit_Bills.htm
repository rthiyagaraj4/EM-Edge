Reprint Visit Bills

From this function you can Reprint a copy of the visit bills generated during an Outpatient or an Emergency visit registration. You can re-print the bills printed during visit registration.   

Reprint Visit Bills function is used to re-print the visit bills, printed during visit registration/consultation. The visit bills are re-printed while a request has been made from the patient/payer. The bills are printed during Outpatient visit registration, if the check box ‘Gen Bill at Regn’ is selected for the respective Billing Group and is associated to the Clinic in the facility using the function ‘Visit Regn Bill Rule Setup’. To use this function, the bills have to be printed during registration/consultation. The bill generated indicates the charges for visit registration/consultation in an Outpatient Clinic. An authorized User is permitted to login and re-print the visit bills. The check box ‘Reprint Bill’ has to be selected in the ‘Users for Authorization/Refund/Cash Counters’ function in the Administration menu for the selected User. The bill/receipt format that is printed during patient discharge is based on the format defined for the respective Episode Type through the Bill Print Format function.

Reprint Visit Bills

From the Transactions > OP Billing, double-click the Reprint Visit Bills Function.

The Reprint Visit Bills screen gets displayed.

  1. Select an Episode Type to re-print visit bills. The bills generated for the selected Episode Type gets displayed. <Related Field Description>

  2. Click [Process Run] to print the bill or [Clear Record] to clear the entered data.

  3. A bill print authorization screen pops-up. Enter the User ID, PIN Number and the Remarks to login and to re-print the bill. Click [OK] to login and [Cancel] to exit from the screen.

  4. A confirmation screen pops-up on clicking [OK]. Click [Yes] to re-print the bills listed in the detailed block and [No] to cancel the printing process.

 

An authorized User is permitted to login and re-print the visit bills.

The bills have to be printed during visit registration to use this function.
