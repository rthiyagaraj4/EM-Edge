Confirm Inpatient Service Request

From this function you can Validate the unconfirmed services associated to an Inpatient or a Daycare patient. You can Query to view the unconfirmed services.

Confirm Inpatient Service Request function is used to validate the Billing Services associated to an Inpatient or a Daycare patient in the facility. The services defined with the ‘Autoconfirm’ check box as cleared have to be confirmed manually. An Authorized User has to confirm the services from the list. The Billing Services defined with the ‘Autoconfirm’ check box as selected are confirmed automatically while associating the services to the patient. You cannot update or cancel the services, which are confirmed automatically. Authorized User can only confirm the services. The check box ‘Confirm Service Request’ has to be selected for the selected User associated to a User Group. The authorization to confirm the service is defined in the function ‘Users for Services’.

Confirm Inpatient Service Request

From the Transactions > IP Direct Charges, double-click Confirm Inpatient Service Request Function.

The Confirm Inpatient Service Request screen gets displayed.

  1. Click [Execute Query] or press [F8] to view all the unconfirmed services associated to the patients’ in the facility.

OR

  1. Select an Episode Type from the list. Click [Enter Query] and enter the Patient ID, Visit ID or the Episode ID to confirm the services associated to a patient. Click [List of Values] to search and to select a Patient ID from the lookup  screen. Click [Execute Query] or press [F8] to view unconfirmed services associated to the patient.

  2. The patient ID, respective episode ID, service code and description, UOM, date and time when the service was associated, the transaction reference ID generated for the selected service get displayed along with the serial number and the patient details.

  3. Select the check box ‘Confirm’ to confirm the service or the panel.

  4. Click [Commit Form] to confirm the service or click [Clear Record] to clear the details.
