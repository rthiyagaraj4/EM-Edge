Cancel IP Under Charge Items

From this function you can Cancel the services, which were associated to an Inpatient after the discharge bill generation. When a request is placed to cancel the undercharge items associated to an inpatient, the user has to cancel the service using this function. The services associated to a patient directly can be cancelled using the Patient Billing Manager module.

Cancel IP Under Charge Items function is used to cancel the billing services, which are associated to the Inpatients after the discharge bill generation. The services, which are confirmed by the authorized user are cancelled. An authorized user has to cancel the confirmed billing services. The authorization to cancel the services is set using the function ‘User Groups/Users’ by selecting the check box ‘Cancel Confirmed Service’. The Users associated to the selected User Group are permitted to cancel the services associated to the patient. User has an option to cancel either the confirmed services or the unconfirmed services. The cancelled services are not considered during bill generation. The Billing Services/Panels are associated to the patient using ‘Enter Request – IP Under charge Items’ function.

Cancel Under Charge Items

From the Transactions > IP Under Charges, double-click Cancel IP Under Charge Items Function.

The Cancel IP Under Charge Items screen gets displayed.

  1. Select an Episode Type from the list. The system defined values are Inpatient and Daycare. Select Inpatient or Daycare patient to cancel the services associated to an Inpatient or a Daycare patient admitted to the facility.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

OR

  1. Right click on Right Click for User Options to search for the relevant patient record. You can search for a patient record based on the Patients Lookup, Patient/Admission/Transfer lookup or Admission/Transfer lookup.

  2. Select the check box View Billing to view the billing details.

    The financial details of the patient for the selected episode get displayed. You can also view the unconfirmed services associated to the patient by clicking [Unconfirmed Services]. The Settlement Details, Billing Group details, service charges, gross amount for the services utilized, adjustments or refunds issued, unbilled and outstanding amount gets displayed. The name of the User, who has recorded or modified the financial details gets displayed.

  3. Click [List of Values] in the 'Source Doc.Ref.' field to select and cancel a service. The transaction reference number generated while associating the Billing Services through 'Enter Request - IP Under Charge Items'  function are listed. The transaction reference document reference line number, quantity of service associated, Billing Service or Panel code and description, charge based amount for the quantity of service utilized, bill status, UOM code, staff indicator and staff ID are listed.

  4. Select a transaction document reference number from the list and click [OK] to cancel a Billing Service/Panel. You can search a transaction document reference number in the pop-up screen. Enter the value in 'Find' and click [Find] to query the results. Click [Cancel] to cancel the operation and close the pop-up screen.

  5. The transaction date, Billing Service/Panel code and description, service quantity, UOM, total payable amount, patient payable amount, practitioner/staff indicator and practitioner/staff ID is displayed.

  6. Click [Commit Form] to cancel the services. If the Billing Service/Panel is already confirmed, a confirmation message 'This Service is already confirmed' is displayed. Click [OK] to cancel. Cancelled transaction reference numbers are deleted from the list in 'Source Doc. Ref.'. Click [Clear Record] to clear the entered data.

 

You cannot cancel the Billing Services/Panels, if they are associated using the 'Enter Request - IP' or 'Enter User Dept Request - IP' functions. An error message 'Discharge Bill Services cannot be cancelled using this fn, User Cancel IP function' is displayed.
