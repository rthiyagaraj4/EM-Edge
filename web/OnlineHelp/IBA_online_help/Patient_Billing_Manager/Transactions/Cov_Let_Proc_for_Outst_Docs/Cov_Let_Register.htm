Covering Letter Register

From this function you can Generate a report to view the Covering Letter Register. A Covering Letter Register displays the outstanding document details along with the Covering Letter number dispatched to the patient/customer.  

Covering Letter Register function is used to print, view or save a copy of the Covering Letter Register in the local machine. A Covering Letter Register displays the details about the outstanding documents. The details on the Covering Letter generated and dispatched to the patient along with the reminder letter are also displayed in the report.  

Covering Letter Register

From the Transactions > Reminder Letter Process, double-click Covering Letter Register function.

The Covering Letter Register screen gets displayed.

  1. Select a Report Type for transaction. <Related Field Description>

  2. Click [Run Process] to either View, Print or Save the details in a file.

  3. A log file is generated with the details. Click [OK] to view the report or [Cancel] to exit from the screen
