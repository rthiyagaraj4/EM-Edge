Customer Specific Covering Letter Remarks

From this function you can Define the Remarks for the billing transactions in the facility. You can specify the action for each remark in a Customer Specific Covering Letter. You can Query to view the remarks and the actions defined.  

Customer Specific Covering Letter Remarks function is used to define the remarks to be either displayed or printed on a Covering Letter to be dispatched to a specific Customer. You have an option to select the effective period for which the remarks or actions are applicable.

Defining Customer Specific Covering Letter Remarks

From the Transactions > Covering Letter Process for Outst Doc, double-click Customer Specific Covering Letter Remarks function.

The Customer Specific Covering Letter Remarks screen gets displayed.

  1. Click [Execute Query] or press [F8] to view all the Customers and to select a Customer for whom the details are to be recorded. You can navigate to the next record using the downward arrow key and to the previous record using the upward arrow key.

OR

  1. Click [Enter Query] and select a Customer by clicking [List of Values].

  2. Click Calendar function to select the effective date range. The remarks and action for the selected Customer is valid during the date range selected.

  3. Enter the Serial Number in numeric format of three digits maximum length.

  4. Enter the Remarks and Actions in alphanumeric format .

  5. Select an Action Type from the list. Select Display, if the Remarks are to be displayed. Click Print, if the Remarks are to be printed.

  6. Click [Commit Form] to save the details or [Clear Record] to clear the entered data.

  7. Click [Create Record] to associate remarks or actions for a separate date range.
