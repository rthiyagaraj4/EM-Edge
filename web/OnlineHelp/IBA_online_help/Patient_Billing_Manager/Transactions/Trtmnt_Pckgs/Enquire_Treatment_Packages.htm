Enquire Treatment Packages  

From this function you can Query to view the package details associated to a patient.  

The Enquire Treatment Packages function is used to view the details of the packages associated to a patient using the Subscribe Treatment Packages function. You cannot update the service details associated to the package.

Enquire Treatment Packages

From the Transactions > Treatment Packages, double-click the Enquire Treatment Packages Function.

The Enquire Treatment Packages screen gets displayed.

  1. Click [List of Values] to select a Patient ID. The name of the patient, age and the gender get displayed on selecting the Patient ID. This is a mandatory field.

  2. Select a Billing Group from the list to view the package details associated to the selected patient.

  3. The packages associated to the patient are displayed on the top and the services associated to the package are listed under Package Service details.

  4. The modifications made to the package services are displayed under Package Audit Details.

  5. Click [Print] to print a selected package detail associated to the patient.
