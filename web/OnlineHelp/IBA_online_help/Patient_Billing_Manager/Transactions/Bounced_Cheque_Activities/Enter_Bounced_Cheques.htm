Enter Bounced Cheques  

From this function you can Enter the details on the bounced cheques for a patient transaction. The cheque considered as invalid by a bank due to insufficient amount or incorrect details are recorded using this function. You can Query to view the details recorded.

Enter Bounced Cheques function is used to record the details of a cheque marked as invalid by the Bank. A cheque is marked invalid due to insufficient amount or incorrect details specified in the cheque by the patient or the Customer. The reason why the cheque is considered invalid by the bank has to be recorded and the authorized billing personnel have to approve the details. An authorized User is only permitted to login and to record the bounced cheque details. A message gets displayed requesting confirmation to finalize the document.

Enter Bounced Cheques

From the Transactions > Bounced Cheque Activities, double-click Enter Bounced Cheques Function.

The Enter Bounced Cheques screen gets displayed.

  1. Click [List of Values] to select a Receipt Number for transaction. The cheque receipts lookup screen gets displayed. Double-click on a record to select it.

  2. The original Document Details, Patient and the settlement details get displayed. The details are recorded using the functions Enter Request – IP and Enter Request – OP respectively for Inpatients and Outpatients. A receipt number is generated on committing the details.

  3. Click [Authorize] to login and to record the bounced cheque details.

    Enter the following details:

    1. User ID – Enter the User ID in alphanumeric format. The authorized User is only permitted to login and to update the details. This is a mandatory field.

    2. PIN No – Enter the PIN number recorded for the respective user set while defining the user in System Manager module. This is a mandatory field.

    3. Reason – Click [List of Values] to select a Reason. The reason defined using the function ‘Reasons for Billing Actions’ with the ‘Bounced Cheque’ check box as selected get listed. This is a mandatory field.

  4. Enter the details of the cheque. <Related Field Description>

  5. A confirmation screen pops-up. Click [Yes] to finalize the details or click [No] to cancel the transaction.

 

You have to login to the Cash Counter to record the bounced cheque details.
