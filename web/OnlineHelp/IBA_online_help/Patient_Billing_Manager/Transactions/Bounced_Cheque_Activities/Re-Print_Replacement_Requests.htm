Re-Print Replacement Requests

From this function you can Print the details on the bounced cheques associated to a patient record. You can choose to print either the cheque details, which are bounced and replaced or bounced and not replaced.

Print Bounced Cheque Letter function is used to print the bounced cheque details. You can select to print the cheque details that are bounced and replaced or bounced and not replaced. The destination printer has to be defined to print the output. A report is printed for the selected receipt number or patient ID with the bounced cheque details along with the logo of the facility.

Re-printing Replacement Requests

From the Transactions > Bounced Cheque Activities, double-click Re-print Replacement Requests Function.

The Re-print Replacement Requests screen gets displayed.

  1. Select an option. You can choose to print the bounced cheque details, namely replaced or not replaced.

  2. Click [List of Values] to select a Receipt Number. The receipts generated while recording the settlement details using the functions Enter IP Receipts/Refunds or Enter OP Receipts/Refunds get displayed. This is a mandatory field.

  3. Enter the patient number. The name of the patient gets displayed in the adjacent text box.

  4. Click [Run Process] to print the details or [Clear Record] to clear the entered details.
