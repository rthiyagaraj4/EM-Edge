Update Bank-in Slip

From this function you can Update the Bank-in details after the Bank-in document copy is received from the Bank.

Update Bank-in Slip is used to modify the Bank-in documents received from the bank after verification. The Bank-in documents are created using the function, Generate Bank-in Batch. System displays the pending bank-in documents generated. Against each bank-in batch document, Bank-in document number given by the respective bank has to be entered manually. System defaults the same Bank-in Batch number, you can modify the Bank-in Number.

Updating Bank-in Slips

From the Transactions > Bank in Slip, double-click Update bank-in Slip Function.

The Update Bank-in Slip screen gets displayed.

  1. Click [Execute Query] or press [F8] to view the Bank-in Batch documents created using the Generate Bank-in Batch function. The details get populated in the respective fields.

    The following details are displayed:

    Bank-in Amount by Account Code

    The Settlement Type, Account Code, Department Code and Amount gets displayed. Amount is consolidated and displayed based on the document type.

    Detail by Bank-in Batch No

    The document details associated to the respective Bank-in Batch Number gets displayed.

  2. Click upward arrow or downward arrow in the keyboard to navigate to the previous or the next Bank-in Batch records.

  3. Enter the details in the mandatory (*) fields. <Related Field Description>

  4. Click [Process Bank-in] to update the Bank-in slips.

  5. A slip number is generated on clicking [Process Bank-in].
