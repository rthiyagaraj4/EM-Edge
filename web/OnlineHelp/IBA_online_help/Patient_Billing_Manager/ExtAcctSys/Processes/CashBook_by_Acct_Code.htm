Cash Book by Account Code

From this function you can Generate a report to view the Cash Book entries against an external  account code for a specific date in the facility.  

Cash Book by Account Code function is used to generate a report to view the Cash book entry details by external account code for a given date.  System generates the Cash Book report listing the Account Code and Description, Collection for the day, Collection for the month, Collection for the previous month, Collection for the year, Summary of Collection for the day, Summary of Collection for the month, Summary of Collection for the previous month and the Summary of Collection for the year.

Generating Cash Book entries

From the Transactions > Cash Book, double-click Cash Book by Account Code Function.

The Cash Book by Account Code screen gets displayed.

  1. Click the Calendar function to select the date. The Cash Book entry details made from external account codes for the selected date get listed in the report generated.  This is a mandatory field.

  2. The check box 'Exclude Credit Card Transactions' is selected, by default. You cannot clear the check box. The report generated does not include the Credit Card transaction details.

  3. Click [Run Process] to generate the report. A Report Option gets displayed with the preferences to View, Print and File options.

    Click ' File' to store the output into a file. Depending on the nature of the report, system prompts to select a file format. Choose the appropriate file format from the available list. You can also optionally enter the output file name.

    Click ‘Print’ to print the report on the designated printer. The report is routed to the default printer set to print the report output.

    Click ‘View’ to view the report on the screen. A note 'Process Successful. Click OK to view report' appears. Click [OK] to view report or [Cancel] to close the report viewer.

  4. Click [OK] to generate the report or [Cancel] to cancel the transactions.
