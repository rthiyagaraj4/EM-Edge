Monthly Report Hospital Bill Revenue Collection

From this function you can Generate a report on the revenue generated by the  in the facility.

The Monthly Report Hospital Bill Revenue Collection function is used to generate a report on the revenue generated by the facility in a specific month. You have an option to get a detailed and summarized report. You can also specify the column break order. You have an option to exclude deposit, refund details and exclude bills with adjustments, exemptions and write-off details.

Generating Report on the Hospital Revenue Collection

From the External Accounting System > Reports, Monthly Report Hospital Bill Revenue Collection  Function.

The Monthly Report Hospital Bill Revenue Collection screen gets displayed.

  1. Click the Calendar function to select the month and year. A revenue report is generated for the month selected. This is a mandatory field.

  2. Select a Detail Option from the list. The system-defined values are ‘Daycare & Inpatient’, ‘Emergency & Outpatient’ and ‘External’. A detailed report is generated for the patient class selected. This is a mandatory field.

  3. Select a Summary Option from the list. The system-defined values are ‘Daycare & Inpatient’, ‘Emergency & Outpatient’ and ‘External’. A summarized report is generated for the patient class selected. This is a mandatory field.

  4. The Column Break Order value is displayed as ‘Only Current Period Bill Amount’. The value is set as the default value and cannot be modified.

  5. Select ‘Deposit’ check box to exclude deposit details from the revenue report. Clear the check box if deposit details are to be included in the revenue report.

  6. Select ‘Refund against bill’ check box, if the refund amount issued against a bill has to be excluded from the revenue report generated. Clear the check box, if the refund issued against the bills are to be included in the revenue report.

  7. Select ‘Adjustment’ check box, if the bills with adjustments are to be excluded while generating the revenue report. Clear the check box, if the bills with adjustments have to be included in the revenue report.

  8. Select ‘Exemption’ check box, if the bills with exemptions are to be excluded while generating the revenue report. Clear the check box, if the bills with exemptions have to be included in the revenue report.

  9. Select ‘Write-off’ check box, if the bills with write-off details are to be excluded while generating the revenue report. Clear the check box, if the bills with write-off details have to be included in the revenue report.

  10. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  11. You can view the report, print the details or save the output in a file in the system.
