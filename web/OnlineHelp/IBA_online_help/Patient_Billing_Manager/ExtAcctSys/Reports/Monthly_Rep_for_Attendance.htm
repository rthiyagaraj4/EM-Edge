Monthly Report for Attendance

From this function you can Generate an attendance report in the facility for a specific month.

Monthly Report for Attendance function is used to generate an attendance report for a specific month in the Facility. System generates a report listing episode type, billing information, billing category and deduction details.

Generating Attendance Report

From the External Accounting System > Reports, Monthly Report for Attendance Function.

The Monthly Report for Attendance screen gets displayed.

  1. Select a Facility from the list. The Facilities associated with the relevant user are listed.

  2. Select an Episode Type from the list. The system-defined values are  – ‘Emergency’, ‘Outpatient’, ‘Emergency and Outpatient’, ‘Inpatient’, ‘Daycare’, and ‘Inpatient and Daycare’. The attendance report is generated for the Episode Type selected.

  3. Click the Calendar function to set month and year, the report is generated for the Period selected. This is a mandatory field.

  4. Select a Billing Information type from the list. The  system-defined values are ‘Limited to Attendance/Admission Period’ and ‘Others’. Select ‘Limited to Attendance/Admission Period’ to restrict the attendance report generation. Select ‘Any’ to generate a report irrespective of the attendance or admission period.

  5. Select a Patient Category from the list. The values are system defined – ‘Citizen’, ‘Others’ and ‘Both’. A report is generated for the Patient Category selected.

  6. Select ‘Print Other Deductions?’ check box to include the deductions in the report. Clear the check box, if the deductions are not to be printed in the report.

  7. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  8. You can view the report, print the details or save the output in a file in the system.
