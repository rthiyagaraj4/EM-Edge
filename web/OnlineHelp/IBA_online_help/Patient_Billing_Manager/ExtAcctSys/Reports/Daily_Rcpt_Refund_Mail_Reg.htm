Daily Receipt/Refund Mail Register

From this function you can Generate a report on the receipts generated or refunds issued in the facility.

Daily Receipt/Refund Mail Register function is used to generate a report on the receipts generated or refunds issued in the facility during a specific period.

Printing Receipts or Refund Mail Registers

From the External Accounting System > Reports, Daily Receipt/Refund Mail Register Function.

The Daily Receipt/Refund Mail Register screen gets displayed.

  1. Select a Transaction Type from the list. The system-defined values are ‘Receipt’ and ‘Refund’. Select 'Receipt' to generate a report on the receipts generated for the transactions in the facility. Select 'Refund' to generate a report on the refunds issued in the facility.

  2. Select a date range to generate the report. The receipt or refund transactions during the period selected are listed. This is a mandatory field.

  3. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  4. You can view the report, print the details or save the output in a file in the system.
