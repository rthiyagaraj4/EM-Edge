Bank-In Report

From this function you can Generate a bank-in report for the transactions in the facility.  

Bank-in Report functions enables you to generate a report on the transactions in the facility. This gives an overview of the transactions performed with respect to each external accounting entity.

Generating Bank-in Report

From the Financial Reclassification > Reports, Bank-In Report Function.

The Bank-in Report screen gets displayed.

  1. Click [List of Values] to select a range of Banks associated to the Facility.

  2. Click [List of Values] to select a range of  Bank-in documents.

  3. Click the Calendar function to select the date range. The transactions performed during the selected period gets listed in the report.

  4. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  5. You can view the report, print the details or save the output in a file in the system.
