Transfer - AR Report

From this function you can Generate a Transfer - AR report with outstanding amount details of the patient.

Transfer-AR Report is used to generate the General Due Bill report for a specific period. This report provides detailed information about the outstanding amount of the patient. The system generates report listing the AHID, Episode ID, Patient Name, Bill No, Date/Time, Bill Amount, Pay by Comp. TPA Amount, Pay by Patient, Security Deposit Amt, Company Name and Claim Code/ Policy Code.

Generating a Transfer - AR Report

From the Reports > Revenue Reports, double-click on Transfer-AR Report function.

The Transfer-AR Report screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run Process] to generate a report.

  3. Report Option is displayed with the View, Print and File Preferences on clicking [Yes]. Click [OK] to generate or [Cancel] to abort the process.

    Select View to view the online report.

    Select Print to print the report.

    Select File to save the output in a file in the local system.
