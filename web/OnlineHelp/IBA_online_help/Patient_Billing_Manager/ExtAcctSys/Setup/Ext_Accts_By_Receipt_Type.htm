External Accounts by Receipt Type

From this function you can Define the External Account Codes and Associate it to a Receipt Type. You can Query to view the External Account Codes and the Receipt Type to which it is associated.  

External Accounts identify additional locations where the transactions take place on behalf of the Facility. For Example, a Bank where the transaction takes place can be an external account. Using this functionality, you can create an External Account and associate Receipt Types for a mode of transaction in the External Accounting Entity.

Defining External Accounts and Associating Receipt Types

From the Masters > Accounting Entity Setup, double-click External Accounts By Receipt Type Function.

The External Accounts by Receipt Type screen gets displayed.

  1. Enter the details in the mandatory (*) fields found in the following tabs:

External Accounts by Receipt Type

Enter the following details to associate an Account Code to a Receipt Type:

  1. Receipt Type – Click [List of Values] to select a valid Receipt Type.  This is a mandatory field.

  2. Description – The Description for the selected Receipt Type appears automatically.

  3. Receipt Nature – The Receipt Nature associated to the relevant Receipt Type appears.

  4. Encounter Type – Select a value from the list. The field contains system defined values. The values are – 'Inpatient', 'Day Care', 'Outpatient', 'Emergency', 'External', and 'Unrestricted'. This is a mandatory field.

  5. Category – Select a value from the list. The system-defined values are 'Unrestricted', 'Nationals', and 'Others'. You cannot select 'Nationals' and 'Others', if the 'Dept Identifier' is 'Enterable'. This is a mandatory field.

  6. Account – Click [List of Values] to select an Account Code. The values defined under the 'External Account Codes' tab get listed. This is a mandatory field.

  7. Dept Identifier – Select a value from the list. The system-defined values are 'Fr Source Location' and 'Enterable'. This is a mandatory field.

  8. Billing Service Grp –  Click [List of Values] to select a Billing Service Group from the list. The values defined using the 'Service Groups/Service Classification' function get listed. You can record the Billing Service Group for selected Receipt Natures only.

External Account Codes

Enter the following details to define Account Codes to identify the External Accounting System:

  1. Account Code – Enter the Account Code in numeric format  of five characters maximum length. This is a mandatory field.

  2. Description – Enter the Description for the Account Code in alphanumeric format. This is a mandatory field.

  3. Receipt Cashbook Main Acct Seq. – Enter the Account Sequence Number in numeric format. The sequence number has to be within the range from one to four.

  4. Column Heading – Enter the Column Heading for the relevant Account Code.

  5. Receipt Cashbook Main Acct Seq. – Enter the Account Sequence Number in numeric format. The Number specified can be any digit from one to four.

  6. Column Heading – Enter the Column Heading for the relevant Account Code.

  7. Account Nature – Select a value from the list. The value is system-defined – 'Deposit Related'. You can leave the field value blank, if the Account is not deposit related.

  8. Credit Card Commission - Select this check box against the account codes to be considered for credit/debit card commission. The selected account codes will be displayed in list of values of Account code for credit card commission field in Settlement Type screen.

  1. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records within the facility.
