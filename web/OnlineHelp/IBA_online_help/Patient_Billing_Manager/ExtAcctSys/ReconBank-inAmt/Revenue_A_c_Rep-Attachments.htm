Revenue A/c Reports – Attachments

From this function you can Generate a report to view the reconciliation statement of revenue accounts. You can Print the statement or Save the output in a file.

Revenue A/C Reports – Attachments is used to view or print reconciliation statement of revenue accounts. System generates the revenue account report listing the document type, date and number, debit and credit details recorded while reconciling.

Print Revenue Account Reconciliation Reports

From the External Accounting System > Reconcile Bank-in Amount, double-click Revenue A/C Reports - Attachments Function.

The Revenue A/C Reports – Attachments screen gets displayed.

  1. Click [List of Values] to select a valid Account Code. The report is generated for the Revenue Account Code selected. System displays the Revenue Account Code description for user verification. This is a mandatory field.

  2. Click the Calendar function to select a Document Date. The Document Date selected is set in the form of month and year. This is a mandatory field.

  3. Select a valid Report Option. Valid options are Attachment – A (default) and Attachment – B.

  4. Click [Run Process] to generate the report. A Report Option gets displayed with the preferences to View, Print and File options.

    Click ' File' to store the output into a file. Depending on the nature of the report, system prompts to select a file format. Choose the appropriate file format from the available list. You can also optionally enter the output file name.

    Click ‘Print’ to print the report on the designated printer. The report is routed to the default printer set to print the report output.

    Click ‘View’ to view the report on the screen. A note 'Process Successful. Click OK to view report' appears. Click [OK] to view report or [Cancel] to close the report viewer.

  5. Click [OK] to generate the report or [Cancel] to cancel the transactions.
