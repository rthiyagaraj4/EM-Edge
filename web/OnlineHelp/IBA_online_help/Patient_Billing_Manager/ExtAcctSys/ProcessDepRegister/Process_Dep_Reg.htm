Process Deposit Register

From this function you can Generate a Deposit Register and Create Adjustment Vouchers. You can Query to view the details.

Process Deposit Register function is used to create a deposit register. You can create adjustment vouchers, generate deposit registers and record opening balance for an account code.

Processing Deposit Registers

From the External Accounting System > Process - Deposit Register, double-click Process Deposit Register Function.

The Process Deposit Register screen gets displayed.

  1. Click [Execute Query] or press [F8] to process deposit register for Account Codes. System populates the records associated to the Account Code. You can associate new adjustment vouchers, generate deposit register or create an opening balance.

  2. Enter the details in the fields in the following tabs:

Create Adjustment Vouchers

Enter the following details to create adjustment vouchers:

  1. Trx Date – Click the Calendar function to enter the Transaction Date and update the voucher details. You cannot create duplicate Deposit Registers for the same date.

  2. Dr Amount – Enter the Debit Amount in numeric format for adjustment.

  3. Cr Amount - Enter the Credit Amount in numeric format for adjustment.

  4. Remark – Enter the Remarks in alphanumeric format of 30 characters maximum length.

  5. Document Number – Enter the Document Number in numeric format with a maximum of 12 digits. The default Document Type selected for ‘Deposit Register Doc’ in the ‘Setup External Acct Parameters’ function under the ‘Set Up’ menu gets displayed. You cannot edit the Document Type. You can enter the Document Number, if the ‘System Generated Document Number’ is cleared in the function ‘Maintain Document Type Details’ in Financials Base Manager module. This field is disabled from entry, if the ‘System Generated Document Number’ is selected in the function ‘Maintain Document Type Details’ in Financials Base Manager module. The Document Number gets generated automatically.

  6. Date – System displays the current date, by default. You cannot update the date.

The recorded deposit register adjustment voucher details get listed in the detailed block where it can be viewed.

Generate Deposit Register

The Generate Deposit Register tab enables you to generate deposit registers for a period. You can generate the register for a specific date. The beginning date is set automatically based on the date when the last deposit register was generated. It lists the carry forward amount, credit amount, debit amount, total credit and debit amount and the total balance. The details are displayed on selecting the ‘To’ date.

Enter the following details to generate a deposit register:

  1. Deposit Register Period [From – To] – The ‘From’ date is set automatically based on the date when the deposit register was generated previously. Click the Calendar function to select the ‘To’ date. The details are displayed automatically based on the period selected. The ‘To’ date selected has to be lesser than the system date.

  2. [Generate Deposit Register] – Click [Generate Deposit Register] to generate the deposit register. A Print Confirmation message ‘Do you want to Print?’ appears. Click [Yes] to print or [No] to cancel.

Create Opening Balance

You can create opening balance for an account code using this tab. Opening Balance can be created only for the first record. System generates subsequent records, which can be viewed on clicking the tab.  You cannot create records here.

The following details are displayed:

  1. Account Code – Click [Execute Query] or press [F8] to view the deposit register details. Press upward arrow or the downward arrows in the keyboard to navigate to the previous or the next record.

  2. Date – The date when the deposit register is generated gets displayed.

  3. Dr Amount – The Debit Amount recorded while generating the deposit register on a specific date gets listed.

  4. Cr Amount - The Credit Amount recorded while generating the deposit register on a specific date gets listed.

  1. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records.

 

You can delete only the latest record in the list to cancel the deposit register, if any modifications are to be made. Click [Delete Record] to delete a record. Click [Yes] to delete the latest record or [No] to abort deletion process. On Clicking [Yes] the record gets deleted from the list and details appear under the ‘Generate Deposit Register’ tab, where you can choose the date to re-generate the deposit register. On clicking [Generate Deposit Register] in ‘Generate Deposit Register’ tab, the details are listed under ‘Create Opening Balance’ tab.
