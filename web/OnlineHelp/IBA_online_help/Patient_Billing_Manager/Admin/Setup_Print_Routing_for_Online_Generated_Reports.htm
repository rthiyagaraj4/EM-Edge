Setup Print Routing for Online/Generated Reports

From this function you can Define the destination printer where the report has to be printed when the user selects to print the report. You can Query to view the online report rule defined.

Online reports are the outputs generated by the system on completing the transactions. You can select to either view, print or save the output of the report in a file. A destination printer has to be selected where the reports are printed when you select to print the report. Reports are printed on the selected printer.

Setup Print Routing for Online/Generated Reports

From the Administration, double-click Setup Print Routing for Online/Generated Reports.

The Setup Print Routing for Online/Generated Reports screen gets displayed.

  1. Enter the details in the mandatory (*) fields found in the tab pages. <Related Field Description>

  1. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records within the facility.
