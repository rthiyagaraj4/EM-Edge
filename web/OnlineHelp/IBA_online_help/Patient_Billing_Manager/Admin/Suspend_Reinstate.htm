Suspend/Reinstate

From this function you can either Suspend or Reinstate the master codes in the Patient Billing Manager module.

Suspend/Reinstate function is used to suspend the active codes or reactivate the suspended codes respectively. The user defined codes for a selected function are available in the list when selecting the function name. You can always suspend the code by clicking ‘Suspend’. The suspended codes are not available for transaction. To enable the code, you can reinstate the code by selecting the code and clicking ‘Reinstate’.

Suspending or Reinstating Master Codes

From the Administration, double-click on Suspend/Reinstate function.

The Suspend/Reinstate screen gets displayed.

  1. Select the Master Code from the list and click [List of Values] to select one of the codes to either Suspend or Reinstate. You can either enable or suspend either a single code or a range of codes for a selected master function.

  2. Click [Process Run] to either Suspend or Reinstate the Master Codes or [Clear Record] to clear the entered data.

  3. A message pops-up requesting confirmation to either suspend or reinstate the code, based on the selection. Click [Yes] to either suspend or reinstate the code respectively. Click [No] to cancel the transaction.
