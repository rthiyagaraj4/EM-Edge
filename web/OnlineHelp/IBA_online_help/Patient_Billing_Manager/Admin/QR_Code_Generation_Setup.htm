QR Code Generation Setup

From this function, you can Generate QR code to be printed in the bills.

This function allows to configure QR code values in sequence to print in the bills. The created QR code can be printed on bills based on setup done in BL Parameters.

The logged in user can do the set up separately for each facility. For multiple facility, user must change login to specific facility to make respective configuration.

Generating QR Code Sequence

From the Administration > User Security, QR Code Generation Setup function.

The QR Code Generation Setup screen gets displayed.

  1. Enter the details in the mandatory (*) fields found in the following Tab pages.

Set Sequence tab

Enter the following details:

  1. Seq No – Enter the sequence number.

  2. Value – Select the value from the LOV. This field is mandatory. The system generated list of values are Hospital GSTIN, Customer GSTIN, Patient ID, Encounter ID, Bill Date, Invoice Number, Total Bill Net Amount, HSH Code, Total Service Count, Financial year, GST Amount

  3. Description – On selecting the Value, the description gets displayed.

Create Static Value tab

User can create static QR code values by entering ID, Description and values. The static values are not system generated.

 Enter the following details:

  1. Static Value ID – Enter the ID, maximum 6 characters.

  2. Description – Enter the description value up to 100 characters.

  3. Value – Enter the value in the text field.

Note: If any static parameter selected for QR code sequence, then its Value will be printed on the QR code scan. The system will not allow to delete if any value is in use in QR Code Sequence.

  1. Click [Commit Form] or press [F10] to save the records or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for existing records within the facility.
