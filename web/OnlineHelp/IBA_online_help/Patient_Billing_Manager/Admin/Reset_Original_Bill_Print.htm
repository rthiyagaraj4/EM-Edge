Reset Original Bill Print

From this function you can Reset the original bill printed in a Cash Counter for a relevant transaction.

Reset Original Bill Print function is used to reset the bill details, which are printed. When a patient/customer requests for a copy of the original bill, which is already printed, the report displays a note that the bill is a copy of the original bill. If the patient/customer requests for a bill without copy being displayed on the bill, you have to reset the original bill using this function. The ‘Copy’ option is not displayed on printing the respective bill.

Resetting Original Bills

From the Administration, double-click on Reset Original Bill Print function.

The Reset Original Bill Print screen gets displayed.

  1. Click [List of Values] to select a Bill Type Document Code. The bills generated are listed along with the Document Type, Document Number and the date.

  2. The Bill Document Number is automatically displayed on selecting the Bill Type Document Code. Click [List of Values] to select a Bill Document Number.

  3. The Bill Date is displayed automatically on selecting the Bill Type Document Code and the Bill Document Number.

  4. Click [Update] to update the bill or [Clear Record] to clear the entered data.

 

You cannot reset the bills, which are settled.
