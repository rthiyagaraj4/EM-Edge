User for Services

From this function you can Define the User Groups pertinent to the Facility and associate individual users to the User Group. You can Query to view the User Groups and the associated users.

User for Services function is used to define the User Groups and club users to the User Group. The selected users only are authorized to perform the transactions in the Patient Billing Manager module. A User Group has access to all the transactions in the Patient Billing Manager module, if the check box, “All Service Groups” is selected. The User Group is not listed in the ‘Rate/Charge Update Security by Services’ function, if the check box is cleared.

Defining User Groups/Users

From the Administration, click User Security, double-click on Users For Services.

The Users For Services screen gets displayed.
