Modify Peer Review Radiologist / Authorising Reporter

From this function you can Modify the Peer Review Radiologist or the authorizing reporter associated to a patient record. You can Query for the examination reports of a patient. You can Print the examination Report.

This function allows to modify the peer review radiologist or the authorizing reporter in case the system assigned Radiologist is not present. This function filters only peer review exams. There is no indication that the exam is Peer review.

The exam is sent to Peer review only after the original Exam is resulted and finalized.

This function is applicable only to users with Super user rights.

Modifying a Peer Review Radiologist/Authorising Reporter

From the Administration > Peer Review > select the Modify Peer Review Radiologist / Authorising Reporter function.

The Modify Peer Review Radiologist / Authorising Reporter screen gets displayed.

  1. Enter the details (by default, the logged in facility name appears in both ‘Ordering Facility’ and ‘Performing Facility’ fields) and click [Execute Query] or press [F8] to search for the patient examination reports.

  2. The examination details of the patient are displayed, based on the details recorded using the ‘Report Examination’ and ‘Authorise by Reporter’ functions. The details are displayed under ‘Examination Details’, ‘Views’, ‘Report Text’ and ‘Exam Narration’ tabs.

    Enter the following details:

    1. Exam – The exam code for which the examination report is authorized gets displayed.

    2. Narration – The exam narration recorded gets displayed.  

    Enter the following details to modify the peer review radiologist/authorizing reporter:

    1. Exam Code – The Exam code for which the examination report is authorized gets displayed.

    2. Status – The status of the report is displayed.

    3. Radiologist – The peer review radiologist’s name is displayed. Click [List of Values] to update the name of the Radiologist.  The Original report Radiologist name will not appear in the LoV of Radiologist

    4. Authorising Reporter – Select the authorizing reporter from the list of values.

    5. Print – Select the check box to print the report. The report is printed at the configured printer.

  3. Select the ‘Print’, check box to print the modified examination report. On selecting the check box and clicking [Save], the modified records are saved and the report is printed in the destination printer set.

  4. Click [Request Reason] to view the clinical comments from the clinician.

  5. Click [Save] to modify the Peer view Radiologist or the Reporter or [Clear Record] to clear the entered details.
