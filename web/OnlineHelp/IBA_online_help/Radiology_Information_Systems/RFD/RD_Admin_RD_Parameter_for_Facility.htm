Enter the following details to define RD Parameters for facility:

Operational Settings

  1. No. of Operating Units/Wings – Select the required operational setting. Select the option ‘One’, if the facility has only one Operating Wing or select the option ‘Many’. You cannot change the operational setting from ‘Many’ to ‘One’.

  2. Operating Unit/Wing – Enter the Operating Unit/Wing code in alphanumeric format of two characters maximum length or click [List of Values] to select an Operating Unit/Wing code. It is mandatory to select the Operating Unit/Wing code, if number of wings defined is ‘One’.

 

Registration Settings

  1. Register Referral Patient – Select the check box, if referral patient registration is allowed in the Facility.

  2. Referral Source Mandatory – Select the check box, if recording the referral source is mandatory.

  3. Registration Backdate Limit – Enter the number of days in numeric format of two digits maximum length to indicate the time that is allowed for backdating the radiology registration.

  4. Max Period Ahead to register Exams against original visit:

  5. OP – Enter the hours allowed to register the radiology exams for Outpatients in numeric format of five digits maximum length. This specifies the number of hours the radiology exams can be registered in advance.

  6. AE – Enter the hours allowed to register the radiology exams for Emergency patients in numeric format of five digits maximum length. This specifies the number of hours the radiology exams can be registered in advance.

  7. IP – Enter the hours allowed to register the radiology exams for Inpatients in numeric format of five digits maximum length. This specifies the number of hours the radiology exams can be registered in advance.

  1. Future Booking Days Limit – Enter the number of days in numeric format of two digits maximum length to indicate the time that is allowed for booking the radiology examinations in advance.

  2. Split Partially Unbilled Orders - Select the checkbox to split partially unbilled orders in Register Examinations screen while registering an order. The orders are split such that:

  3. Original order ID is applicable only for all the ‘billed’ exams and exams that do not require billing/payment. The one or more unbilled orders within this order ID are automatically cancelled with the reason “This order is cancelled as it is linked to a new order’”. This cancelled order is displayed within the CA module.

  4. Another new order ID is created for all the ‘unbilled’ exams.

  1. Interface with External Queue Series - Select the check box, if the queue number is mandatory for examination. Selecting this check box indicates that an external Queue Management System (QMS) is interfaced with the EM application and it is mandatory to allocate the queue number from external QMS.

  2. Manual Radiologist Assignment – Select the check box to assign radiologist manually in the Register Examinations screen. When the check box is selected, the Radiologist ID field gets enabled and appears as a lookup in Register Examinations screen so that you can select the required Radiologist from look up and complete the registration. By default, the check box is not selected.

 

Reporting Settings

  1. Allow Preliminary Reporting – Select the check box to allow preliminary reporting.

 

Authorization Settings

  1. Dept. Authorization Required - Select the check box to indicate that Departmental Authorization is required before performing an examination. If the check box is selected, all the examinations to be performed in this facility must be authorized before performing an examination. The status of the examination, which requires departmental authorization, is displayed as ‘PAUTH’ referring to Pending Authorization.

  2. Accept PIN while Authorizing - Select the check box to indicate that the authorizing user must enter the Personal Identification Number to authenticate while authorizing.

 

Others

  1. Track Patient Location – Select the check box to enable tracking of the patient location. The ‘Change Perform Location’ function is accessible if the check box is selected.

  2. Items Used Selection – Select the option from the list. Select ‘Manual’ to manually enter the items, which are used for the examinations. Select ‘Auto populate’ for the system to display the items, which are used for the examinations.

  3. Validate Batch ID for Contrast – Select the check box to indicate that the Batch ID must be validated, if contrasts are used for the examination.

  4. Control Section Access – Select the check box to restrict the control of the section.

  5. Validate MPPS – Select the check box to enable the Enterprise Management applicable to recognize the images captured using MPPS. MPPS is an acronym for Modality Performed Procedure Steps and is an external application, which is interfaced to the Enterprise Management application.

  6. Routine - Enter the Routine in alphanumeric format of 50 characters maximum length.

  7. mAS Auto Calculate – Select this check box to display the milliamps Seconds (mAS) value in the Exposure Factors tab in the Record Exam Completion screen.

  8. Include Technologist Comment In History? - Select this check box to include the technologist comments into the Clinical Event History screen in Clinician Access module.

  9. Manual Allocation For Queue Number -  Select the check box, if the queue number is to be entered manually when the Queue Management System (QMS) is operationally down.

  10. Cosign Required For The Exam Placed ‘From’ Radiology – Select the check box to indicate that the exam placed from Radiology Information Systems as ‘Additional Exam’ or ‘New Exam’ is to be co-signed in Clinician Access module.

  11. If ‘Additional Exam’ order is placed, it has to be co-signed by the Ordering Practitioner.

  12. If ‘New Exam’ order is placed, it has to be co-signed by the Attending Practitioner.

The exams are listed in the ‘Order Co-signing’ function as ‘Ordered from Radiology’ with reasons recorded (if any). You cannot modify the reasons. The exams are displayed with an icon  . Clear the check box to indicate that the radiology exams do not require co-signing and is listed in the ‘Orders’ and ‘Register Order’ function.

Co-signing is a method of intimating the attending practitioner of any orders placed by practitioners who are not directly related to the patient. Co-signing does not delay the processing of the order and is used to purely intimate the attending practitioner of any orders placed against patients who are directly under the care of that practitioner. The check box is selected, by default.

The co-signing is applicable for orders placed for Daycare, Outpatients, Inpatients and Emergency patients. The co-signing is not applicable for orders placed for referral or external patients.

 

Interface to Other Modules

  1. Mandate Result Status when Authorising by Radiologist - Select this checkbox to make the ‘Result Status’ dropdown mandatory. By default this checkbox is not selected.

  2. Interfaced to Patient Billing - Select the check box to interface the Revenue Management module with the Radiology Information Systems module. If the check box is selected, the bill amount of the examination is displayed when the Radiology order is placed.

  3. Charge Recalled Patient – Select the check box to charge the patient, who is recalled for repeating an examination. Patients are recalled for re-examination purposes. Select this check box if the patient must be charged to undergo the same examination.

  4. Charge Consumables -Select this check box to charge the item consumables used in the Record Exam Completion and Record Consumables screen. The item consumables are charged only if the ‘Interfaced to Patient Billing’ check box is selected.

  5. Interfaced to Inventory Management – Select this check box to interface the Inventory Management module with the Radiology module. If this check box is selected, the available stock quantity is displayed when a radiology order is placed.

  6. Note Type – Select the required note type from the lookup. The Notes Type created for the ‘Radiology Clinical Notes’ applicable task will be displayed in the lookup. When the Note Type is selected, the ‘Record Radiology Clinical Notes’ menu option will appear in below RIS functions. This helps users to invoke the ‘Record Radiology Clinical Note’ (3T) screen from RIS screens and record the radiology clinical notes from 2T itself.

• Record Exam Start/Completion

• Authorize by Radiologist

• Modify Authorize Report

• Report / Authorise Unreported Exams

  1. Doc Type – Enter the document type in alphanumeric format of six characters maximum length or click [List of Values] to select the document type.

  1. Interfaced to PACS - Select the check box to interface the Picture Archiving and Communication System (PACS).
