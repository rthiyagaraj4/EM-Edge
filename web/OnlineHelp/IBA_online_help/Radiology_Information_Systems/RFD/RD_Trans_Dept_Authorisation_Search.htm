Enter the following details to query the pending exams:

  1. Ordering Facility – Select the Ordering Facility from the drop-down list to display the orders based on the selected Ordering Facility. The value ’All Facilities’ is selected by default.

  2. Wing – Select the Wing from the list. The value ‘All Wings’ is displayed, by default. The Operating Unit/Wings are defined in the Operating Units/Wings function under the Administration menu.  

  3. Section – Enter the Section or click [List of Values] to select the Section. Sections are displayed based on the selected ‘Operating Unit/Wing’. Sections are associated to the Operating Unit/Wing in the ‘Sections for Operating Unit/Wing’ function under the Administration menu. This is a mandatory field.

  4. Priority – Select a Priority of the exam from the list. The system-defined values are ‘Stat’, ‘Urgent’, ‘Routine’ and ‘All’. The value ‘All’ gets displayed, by default.

  5. Patient ID – Enter the Patient ID in the length as specified in the MP Parameter or right click on ‘Right Click for User Options’ hyperlink to select the patient.  Click ‘Standard Patient Lookup’ to invoke the Patient Lookup screen.

  6. [T] – Click [T] to view the patient tracking location details. This button is enabled if ‘Track Patient Location’ check box is selected in the 'RD Parameter for Facility’ function. On clicking [T], the ‘Change Patient Location’ screen is displayed.

  7. Ordered By – Enter the name of the physician who has placed the order or click [List of Values].

  8. Show – Select a value from the list. The system-defined values are ‘Unauthorized Exams’, ‘Authorized Exams’ and ‘All Exams’. ‘Unauthorized Exams’ gets displayed, by default.

  9. Status – Select a status from the list of values. The system-defined values are ‘Pending Auth’, ‘Replaced’, ‘Rejected’, ‘Cancelled’, ‘Pending/Modified’, ‘All’. ‘Pending Auth’ gets displayed, by default.

  10. Order Date [From / To] – Enter the Order Date range or click on the Calendar function.
