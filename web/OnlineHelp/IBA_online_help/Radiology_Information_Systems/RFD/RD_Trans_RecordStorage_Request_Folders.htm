Enter the following details:

Request From

  1. Request Facility – Enter the Requesting Facility code or click [List of Values] to select the Request Facility. The records are transferred from this facility to the requesting facility. This is a mandatory field.

  2. Request Loc Type – Select the Request Location Type from the list. The listed values are ‘Clinic’, ‘Ward’, ‘Department’ and ‘Physician’. This is a mandatory field.

  3. Request Location – Enter the Request Location code or click [List of Values] to select the Request Location. Based on the Request Location type selected, the Request Locations are displayed. This is a mandatory field.

  4. Request Record Location – Enter the Request record Location or click [List of Values] to select the Request Record Location. This is a mandatory field.

  5. Request By User – The logged in User name is displayed. This is a mandatory field.

  6. Narration – Enter the Narration code or click [List of Values] to select the Narration. On selecting the Narration code, the description of the Narration is displayed. The reason for transferring the record from one location to another is selected. This is a mandatory field.

  7. Patient ID – Enter the Patient ID in the length as specified in the MP Parameter or click ‘Right Click for User Options’ to select the Patient ID, using the Standard Patient Lookup function. This is a mandatory field.

 

Request To

  1. Request to Facility – Select the facility code from the list of values. The name of the facility is displayed in the adjacent field.

  2. Request Period – Enter the period in number of days for which the folder has been requested in numeric format of five digits maximum length.
