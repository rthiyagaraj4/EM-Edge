Enter the following details:

  1. Exam – The code and description of the exam is displayed.

  2. Contrast Type – The Contrast Agent/Drug of the selected exam is displayed. If multiple contrast agents are used, the contrast agent/drug of the selected exam is displayed on the top of the list.

  3. Route – The Route Code of the selected exam is displayed.

  4. Sedative – Select this check box to record the drug as sedative type.

  5. Adverse Reactions – Click [View] to display ‘Adverse Reaction’ search screen. You can view the adverse reactions recorded for the selected exam. [View] is enabled only if adverse reactions are recorded for the exam. Click [Close] to exit from the screen.
