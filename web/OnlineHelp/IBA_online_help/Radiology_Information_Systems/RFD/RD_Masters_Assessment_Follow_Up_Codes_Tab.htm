There are two tabs, namely Followup Types for Assessment Category and Followup Types.

Enter the following details:

  1. Assessment Category Code/Description – Select the Assessment Category from the list of values. The description of the selected Assessment Category is displayed. The Assessment Categories are defined in the same function.

  2. Age Group Code/Description – Select the Age Group applicable for the Follow-up Type. The description of the Age Group is displayed.

  3. Follow-up Code/Description – Select the Follow up code from the list of values. The description of the selected Follow-up is displayed. The Follow-up codes are system defined.

Enter the following details:

  1. Assessment Groups – Click [List Values] to select the Assessment Group from the list of values. This is a mandatory field.

  2. Code – Enter the Follow Up Type code in alphanumeric format of four characters maximum length. This is a mandatory field.

  3. Description – Enter the Follow Up Type description in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  4. Type – Select the Followup Type applicable to the Assessment Group from the list. The listed values are Follow-up and Screening. This is a mandatory field.

  5. Recall Period – Enter the days (or months) within which the patient has to be recalled for reexamination. After an examination is conducted, the report is classified under an Assessment Group. Depending on the Assessment Group, the follow-up examinations are suggested. The recall period is the number of days in which the patient is recalled to the facility for undergoing the follow-up examinations.

  6. Letter Format – Enter the letter format in free text format.

Click [Associated Assessment Category] to go to the Followup Types for Assessment Category tab.
