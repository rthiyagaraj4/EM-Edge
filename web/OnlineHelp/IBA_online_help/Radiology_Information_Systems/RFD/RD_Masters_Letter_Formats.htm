Enter all the details to create a letter format:

  1. Assessment Group – Enter the Assessment Group or click [List of Values] to select it. The description of the Assessment Group is displayed on entering the Assessment Group. The Assessment Groups are defined in the ‘Assessment/Follow up Codes’ function under ‘Result Reporting’ in the Masters Menu. This is a mandatory field.

  2. Assessment Category – Enter the Assessment Category or click [List of Values] to select the Assessment Category. The description of the Assessment Category is displayed on entering the Assessment Category. The Assessment Categories are defined in the ‘Assessment/Follow up Codes’ function under ‘Result Reporting’ in the Masters menu. This is a mandatory field.

  3. Format ID – Enter the Letter Format ID in alphanumeric format of four characters maximum length. This is a mandatory field.

  4. Description – Enter the description for the Letter Format in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  5. Default for assessment category – Select the check box to indicate that the created Letter Format is the default format for the selected Assessment Category.

  6. Text – Enter the Letter Format in free text format. This is saved as a Letter Format for the selected Assessment Category.

More than one letter format can be created for an Assessment Category, but only one Letter Format can be marked as ‘Default Letter Format’.
