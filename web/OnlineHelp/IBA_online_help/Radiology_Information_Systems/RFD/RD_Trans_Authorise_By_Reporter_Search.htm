Enter the following search criteria:

  1. Performing Facility – The logged in Facility Name is displayed, by default.  Select the Performing Facility from the list.

  2. Wing – Select the wing from the list. The list of values gets populated with the operating wings/units created in the ‘Operating Wings/Units’ function.

  3. Section – Enter the Section code or select it from the list of values. The name of the section code is displayed in the adjacent field. The list of values gets populated with the sections created in the ‘Examinations’ function.

  4. Registration Date [From-To] – Click the Calendar function to select the date range. The patient records with the registration date between the selected date ranges are displayed.

  5. Priority – Select the applicable priority for the examination from the list. The system-defined options available are All, Stat, Urgent and Routine.

  6. Patient ID – Enter the Patient ID. On entering the Patient ID, the name of the patient is displayed in the adjacent field.

  7. [T] – Click [T] to view the patient location information. This button is enabled only on selecting the ‘Patient Tracking Information’ check box in the RD Parameters for Facility setup. The current patient location is displayed. You can change the location of the patient, if needed. For example, at the start of the examination, the patient location is displayed as ‘Ward’; after the examination has started, the patient location can be changed to ‘Study location’ indicating that the patient has been brought into the study location for performing the examination.

  8. Right Click for User Options – Right click ‘Right Click for User Options’ option to invoke the ‘Standard Patient Lookup’ or ‘External Patient Lookup’ screen. You can query for a patient or an external patient record using ‘Standard Patient Lookup’ or ‘External Patient Lookup’ screens. Double-click the Patient ID in the displayed list to select the patient record.
