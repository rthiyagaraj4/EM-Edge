Enter the following to create Room Time Table:

  1. Room Code – Enter the Room code or click [List of Values] to select the Room Code. The Study locations (Rooms) are defined in the ‘Study Locations’ function. This is a mandatory field.

  2. Day Code – Enter the Day code or click [List of Values] to select the Day Code. The day codes can be selected from numbers between 1 to 7. 1 indicates Monday and 7 indicates Sunday as per the chronological order of the days of the week. Select the day code as ‘1’ to indicate that the Study Location is operational on Monday or select ‘2’ to indicate that the Study Location is operational on Tuesday and so on. If you want to create the timetable for a Sunday and select the day code as ‘7’, system alerts you with a warning message indicating that Sunday is as holiday if you have declared Sunday as a holiday. Incase the Study Location is operational on all the days of the week, the system displays the day code as ‘0’, by default. This is a mandatory field.

  3. Start Time [HH24:MI] – Enter the Start Time for the Study Location in hh:mm format. This is a mandatory field.

  4. End Time [HH24:MI] – Enter the End Time for the Study Location in hh:mm format. This is a mandatory field.

  5. Time per Slot [HH24:MI] – Enter the standard duration for an appointment in hh:mm format. If the Study Location operates on all the days of the week, this time per slot is applicable for all the days. This is a mandatory field.

  6. Max Patients/Day – The system calculates the total number of appointments based on the start time, end time and the break times and displays the maximum possible number of patients for that day.

  7. Max Overbookings – Enter the maximum number of possible over bookings in numeric format of three digits maximum length.

  8. Narration – Enter the remarks regarding the timetable in alphanumeric format of 60 characters maximum length.

 

Break Timings

  1. Start Time – Enter the break start time in hh:mm format.

  2. End Time – Enter the break end time in hh:mm format.
