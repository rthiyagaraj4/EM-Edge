Enter the following details:

  1. Receiving facility – The Facility ID and the name of the receiving facility is displayed and disabled.

  2. Rec Loc Type/Location – Select the Location Type from the list. The listed values are ‘Clinic’, ‘Ward’, ‘Department’ and ‘Physician’. Based on the selected Location Type, enter the Location code or click [List of Values] to select the receiving Location.

  3. Rec Record Location – Enter the Receiving Record Location code or click [List of Values] to select the Receiving Record Location. On selecting the Receiving Record Location code, the description of the Receiving Record Location is displayed.

 

Folder Details

The Folder details matching the entered criteria are displayed.

  1. Folder No – Enter the number of the folder in the length as specified in the RD Parameter for Facility function or click [List of Values] to select the Folder Number.

  2. Patient ID and Name – Based on the selected Folder Number, the Patient ID and name of the patient is displayed.

  3. Request Reason – The Reason entered while placing the request is displayed.

  4. Issued Facility ID – The ID of the issuing facility is displayed.

  5. Source Type – The Source Type of the requested folder is displayed.

  6. Issue Actual Holder ID/Name – The name of person who has issued the folder is displayed.

  7. Issue Date – The date of issue of the folder is displayed.

  8. Confirm Receipt – Select the ‘Confirm Receipt’ check box to confirm the requesting facility has received the requested folder.
