Enter the following search criteria:

  1. Perf. Facility – Select the facility where the exam is being performed from the list of values. The name of the performing facility where the exam was last conducted is displayed by default.

  2. Wing – Select a Wing from the list. The values defined in the in the ‘Operating Wings’ function.

  3. Section – Click [List of Values] to select a Section or enter the section code. The name of the section code is displayed in the adjacent field. The values defined in the ‘Examinations’ function gets populated.

  4. Registration Date [From-To] – Click the Calendar function to select the date range for the examinations for which the registrations are done.

  5. Registration No – Enter the registration number of the examination.

  6. Patient ID – Enter the patient ID in alphanumeric format of 10 digits maximum length. The name of the patient is displayed in the adjacent field.

  7. [T] – Click [T] to view the patient location information. This button is enabled only on selecting the ‘Track Patient location’ check box in the ‘RD Parameters for Facility’ function. The current patient location is displayed. You can change the location of the patient. For example, at the start of the examination, the patient location is displayed as ‘Ward’, the patient location can be changed to ‘Study location’ once the examination starts, indicating that the patient has been brought into the study location for performing the examination.

  8. Exam – Click [List of Values] to select the examination code. The examination reports are listed based on the Exam selected from the list.

  9. Episode Type – Select the Episode Type from the list. The options available are Inpatient, Outpatient, Referral, External and All.

  10. Priority – Select the priority from the list. The system-defined values are All, Stat, Urgent and Routine.
