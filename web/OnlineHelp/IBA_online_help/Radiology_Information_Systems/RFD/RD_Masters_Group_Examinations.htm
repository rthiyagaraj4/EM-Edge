Enter the following to create a Group Examination:

  1. Group Exam Code – Enter the Group Exam code in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Long Description – Enter the long description of the Group Exam in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description of the Group Exam in alphanumeric format of 15 characters maximum length. The description entered in the long description field is displayed, by default. This is a mandatory field.

  4. Section – Enter the Section code in alphanumeric format of four characters maximum length or click [List of Values] to select the Section. The description of the selected Section is displayed. The Sections are defined in the ‘Sections’ function under ‘Examinations’ in the Masters menu. This is a mandatory field.

  5. Catalog Cross-ref – Enter the Catalog Cross Reference code in alphanumeric format of 15 characters maximum length or click [List of Values] to select the Catalog Cross Reference. The description of the catalog is displayed. The Catalog Cross Reference code is invoked from the Order Catalog function with Order Category type ‘Radiology’ of Order Entry and Result Reporting  module. This is a mandatory field.  

 

Appointment

  1. Appointment reqd – Select the check box to indicate that appointment is required before conducting a group exam. While registering the exam, a message is displayed alerting you that appointment is necessary for the exam. Clear the check box if appointment is not required before performing the examination.

  2. Exam Duration – Enter the time required in the format of hh:mm for performing all the examinations under the Group Examinations function.

  3. Min. Exam Start Time - Enter the minimum exam start time in the format of hh:mm. You are not allowed to book appointments before this time. The minimum exam start time is dependent on the working hours of the Section. If the work hours for a Section are 8 am to 8 pm, the Section is not allowed to start the examinations at 8 am; a minimum exam start time is required to prepare for the examinations. If one hour is required for preparing for the examinations, the minimum exam start time is 9 am.

 

Workload Reporting

  1. Workload Reporting reqd – Select the check box to indicate that the group examination requires workload reporting. Workload reporting is used to report the load on the radiology wing. Some routine examinations are not marked for workload reporting.

  2. Work Units – Specify the units for the workload report in alphanumeric format of six characters maximum length. The work unit is accepted only if ‘Workload Reporting reqd’ check box is selected.

 

Patient Instructions

  1. Patient Instructions – The patient Instructions are defined in the ‘Order Catalog’ function of the Order Entry and Result Reporting module. You cannot make any changes to the defined Patient Instructions. To update the patient instructions, you have to make necessary changes in the ‘Order Catalog’ function. The patient instructions are - By Age/Sex and Standard Instructions. If no instructions are defined in the Order Entry and Result Reporting module, the ‘Not Setup in Catalog’ option is selected.
