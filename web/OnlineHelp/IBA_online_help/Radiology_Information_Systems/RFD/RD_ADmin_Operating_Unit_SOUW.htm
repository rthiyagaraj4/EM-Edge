Enter the following to create a section for any Operating Unit/Wing:

  1. Operating Unit/Wing – Enter the Operating Unit/Wing code or click [List of Values] to select the Operating Unit/Wing.

  2. Section Code – Enter the Section Code or click [List of Values] to select the Section. This is a mandatory field.

  3. Description – The description of the selected Section is displayed. This is a mandatory field.

  4. Prefix – Enter the Prefix that is used in the registration numbering, if numbering is based on location as specified in the parameter. This is a mandatory field.

  5. Next No – Enter the next number length for the Section in numeric format of six digits maximum length. This length is used to generate the registration number and it is incremented by one on generating the registration number. This is a mandatory field.

  6. Max No – Enter the maximum number for the Section in numeric format of six digits maximum length. The length entered is used to restrict the length of the registration number. This is a mandatory field.

  7. Start Time – Enter the working Start Time indicating the start of the working hours for the Section in the format of hh:mm. This is a mandatory field.

  8. End Time – Enter the working End Time indicating the end of working hours in a day for the Section in the format of hh:mm. This is a mandatory field.

  9. Threshold Wait Time – Enter the threshold wait time.

  10. Days Ahead for Appt. Bookings – Enter the number of days allowed to book an appointment in advance for the Section in numeric format of three digits maximum length.

  11. Radiologist ID – Enter the Radiologist ID or click [List of Values] to select the Radiologist ID.

  12. Name – The Name of the selected Radiologist is displayed on selecting the Radiologist ID.

  13. Default Storage Location – Enter the Storage Location code or click [List of Values] to select the Storage Location, which is set as the default Storage Location.

  14. External Queue Series - Select the External Queue Series from the drop-down list. The system defined values are ‘QMS’ and ‘blank space’. This drop-down list is enabled only if the following conditions are satisfied:

  15. 'Assign Queue No. On Registration’ check box in ‘Operating Units/Wings’ screen is not selected

  16. Interface with External Queue Series’ check box is selected in the RD Parameters for Facility screen.

  1. Assign Queue By – This field is enabled only if the value ‘QMS’ is selected in the ‘External Queue Series’ list. Select the required assign queue by value from the drop-down list. The list values include ‘Patient Type’ and ‘Study Location’.

  2. When you select the value ‘Patient Type’ from the drop-down list, the QMS system will generate a Queue number based on the Patient Type.

  3. When you select the value ‘Study Location’ from the drop-down list, the QMS system will generate a Queue number based on the Study Location.

  1. Description - The description of the selected Storage Location is displayed.

  2. Note Type – Enter the Note Type code or click [List of Values] to select the Note Type.

  3. For Reporting - The description of the Note Type for Reporting is displayed on selecting the Note Type.

  4. Investigation – Enter the Note Type code for Investigation or click [List of Values] to select the Note Type.

  5. Follow up Note Type - The description of the Note Type for Reporting is displayed on selecting the Investigation Note Type.

  6. Procedure Unit – Click [List of Values] to select the Procedure Code defined in the Outpatient Management module. The Section Code is associated with the Procedure Unit. This field will appear only if ‘General Schedule’ is selected in the Schedule Type field in RD Parameters.
