Enter the following to create a Section:

  1. Section Code – Enter the Section code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Long Description – Enter the long description of the Section in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description of the Section in alphanumeric format of 15 characters maximum length. The description entered in the long description field is displayed, by default. This is a mandatory field.

  4. Service Code/Service Desc – Enter the Service code or click [List of Values] to select the Service Code. The description of the selected Service Code is displayed on entering the Service Code. The Services are defined in the ‘Service’ function in Application Masters module.

  5. Order Type – Enter the Order Type or click [List of Values] to select the Order Type. The description of the selected Order Type is displayed on entering the Order Type. The Order Types are defined in the ‘OR Order Setup’ function in Order Entry and Result Reporting module. If the Order Type is linked to another Section, on selecting the same Order Type, the system alerts you with a message. Click [OK] to proceed.

  6. Checklist – Enter the Checklist or click [List of Values] to select the Checklist. The checklists are defined in the ‘Checklist’ function. This checklist is applicable for all the examinations under this section. Before commencing the examination, the checklists must be marked as ‘complete’.

  7. Dicom Images – Dicom is a third party imaging system. If this imaging system is interfaced with the Enterprise Management application, the check box is enabled. Select the check box to indicate that the examinations performed in this section use the Dicom images. Clear the check box, if the images captured using this system is not recognized by the Enterprise Management application.

  8. Exposure Factors - Select the Exposure Factor from the list. The list of values is 'Applicable (Mandatory)', 'Applicable (Optional)' and 'Not Applicable'. If you specify the Exposure Factor as mandatory, the system does not allow you to mark the examination as complete until the exposure factor is entered.
