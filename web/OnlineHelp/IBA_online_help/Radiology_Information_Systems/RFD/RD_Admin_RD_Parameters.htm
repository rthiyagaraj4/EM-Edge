Enter the following details to define RD Parameter:

Examination Parameters

  1. Examination Coding Scheme - Select the ‘Examination Coding Scheme’ from the list. The listed system defined values are ‘CPT4’, ‘ICD 9CM’ and ‘User Defined’.

  2. Workload Reporting Units Measured In - Select the unit from the list. The listed system defined values are ‘Korner Units’ and ‘RVU’.

Workload reporting is used to measure the load of work in the Radiology department.

 

Registration Numbering

Based On

  1. Based on - Select the required option to generate the registration number based on the selected option. If ‘Section’ is selected, the registration number is generated based on the ‘Section’. If ‘Storage’ is selected, the registration number is generated based on the Storage Location.

  2. Registration No. Length – Enter the length of the registration number in numeric format. The value cannot be greater than 10. The registration length specified is calculated based on the Length Parameters. You cannot modify the value on saving the record.

 

Length Parameters

  1. Prefix – Enter the Section/Storage Location prefix, which defines the length of the Section code or Location Code. If registration numbering rule is based on Section, the prefix legend is displayed as ‘Section Prefix’ and the Registration Number is generated based on the length specified in the Section Prefix. If registration numbering rule is based on Storage Location, the prefix legend is displayed as ‘Storage Location Prefix’ and the Registration Number is generated based on the length specified in the Storage Location Prefix.

  2. Year – Enter the length of the Year, to be displayed in the Registration Number.

  3. Serial Number – The Serial Number is calculated and displayed based on the Prefix and Year length specified.

The total length must match with the length of the ‘Registration No. Length’ field.

 

Radiation

  1. Measured in – Enter the Unit for measuring the radiation in the numeric format of four digits maximum length.

  2. Diagnosis Coding Scheme – Select the Diagnosis Coding Scheme from the list. The listed system defined values are ‘ICD 10’, ‘ICD 9CM’ and ‘User Defined’.

  3. Schedule Type – Select the Schedule Type from the drop-down list. The listed values are ‘Radiology Schedule’ and ‘General Schedule’. If the value ‘Radiology Schedule’ is selected, the radiology appointments are scheduled in  Radiology  module. If the value ‘General Schedule’ is selected, the RIS appointment scheduling is done in Appointment Scheduling screen in Appointment Scheduling (OA) module. The Schedule Type drop-down list gets disabled once the RIS Appointment Scheduling is enabled in OA.
