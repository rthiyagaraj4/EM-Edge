Examination Codes

From this function you can Define parameters for performing radiology examinations based on the examination coding scheme and Query for the existing Examination Codes .

Examination is the act of examining or doing a checkup. From this function, you can define all the required parameters for performing a particular radiology examination based on examination coding scheme (CPT 4, ICD 9CM or user-defined) set in the RD Parameter. You can associate a section, Organ Site, Exam Category and Order Catalog to the examination. You can define if views/laterality is applicable for the examination, catalog cross-reference and media type for storing the result of the examination. Other details such as appointment for the examination, result reporting and workload reporting of the examination, the instructions to be given to the patient prior/after the examination, the replaceable examination for this examination, report text and location of the documents related to the examination are defined. You can associate the Radiologist and Operating Facility to the examination.

Creating an Examination Code

From the Masters > Examinations, double click Examination Codes function.

The Examination Codes screen gets displayed.

  1. Enter the details in mandatory (*) fields. <Related Field Description>

  2. The Replaceable Exams Tab screen is displayed, by default. Enter the details in the mandatory (*) fields in the following tab pages.

Replaceable Exams

Enter the following details:

  1. Exam Code – Enter the Exam or click [List of Values] to select the Exam Code. This is a mandatory field.

  2. Exam Description – The description of the selected exam code is displayed.

If two examinations are similar in nature and the drugs and contrast agents used are different, consider a scenario where a particular drug is unavailable, the analogous (a similar function with different features) examination can be performed.

Views

Enter the following details:

The Views tab gets enabled only when the ‘Views Applicable’ check box is selected. Using this tab you can define the views applicable to the examination.

  1. Code – Enter the View code or click [List of Values] to select the code. The ‘Views’ are defined in ‘Views’ function. This is a mandatory field.

  2. Description – The description of the selected view code is displayed.

Patient Instructions

If the option ‘By Sex/Age’ is selected under the title Patient Instructions in the header, the following instructions for the exam are displayed based on the Order Catalog code. You cannot modify these values.

Instructions are displayed under the patient instruction based on the Gender and Age of the patient. The instructions are displayed in English and in local language.

The Patient Instructions are set in the ‘Order Catalog’ function of the Order Entry and Result Reporting module. If the instructions are not set, the Patient Instructions Tab is disabled.

Report Text

The Report templates used for reporting are defined using this tab.

Enter the following details to create a Report Text template:

  1. Code – Enter the Report Text code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Long Description – Enter the long description of the Report Text in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description of the Report Text in alphanumeric format of 15 characters maximum length. The description entered in the long description field is displayed, by default. This is a mandatory field.

  4. Text – Enter the content of the template in free text format.

Radiologist by Facility

You can define the default radiologists for the exam across facilities which can be used during registration.

Enter the following details:

  1. Operating Facility ID – Select the Operating Facility from the list. This is a mandatory field.

  2. Radiologist ID – Enter the Radiologist ID or click [List of Values] to select the Radiologist ID. The listed values are defined in the Application Masters module. All the Practitioners who are marked with the Practitioner Type as ‘Radiologist’ are displayed in the list. This is a mandatory field.

  3. Radiologist Desc – The name of the selected radiologist is displayed.

If the ‘Auto Report’ check box in ‘Result Reporting’ heading is selected, this tab does not get displayed.

Documents

Enter the following details:

  1. Patient Information Document – Select the location where the patient information document is stored by clicking  . This displays the path where the document is stored. Click  to view the patient information document.

  2. Post-Exam Instructions Document – Enter the location where the post-exam instructions document is stored by clicking  . This displays the path where the document is stored. Click  to view the post-exam instructions document.

Duplicate Check

Enter the following details:

  1. Duplicate Check – Select the check box to define multiple examination check against an examination.

Order Priority

  1. Routine – Select the check box to prioritise the order for routine patients.

  2. Urgent – Select the check box to prioritise the order for urgent patients

  3. Stat - Select the check box to prioritise the order for stat patients

 

Patient Class

  1. Inpatient - Select the check box to define duplicate check for Inpatients orders.

  2. Outpatient – Select the check box to define duplicate check for Outpatients orders.

  3. Emergency – Select the check box to define duplicate check for Emergercy orders.

  4. Daycare - Select the check box to define duplicate check for Daycare orders.

  5. External - Select the check box to define duplicate check for External orders.

 

Period

  1. Days – Enter the number of days for duplicate check in a number format of three character maximum length.

  2. Hours – Enter the time period for duplicate check in Hrs:Mins format.

  3. Exam Code – Click [List of Values] to select the exam code.

  4. Exam Description – The description of the selected exam code is displayed.

Protocol

You can map an examination with multiple protocols such that it is mandatory to assign the protocol to any one of the Order Priority and Patient Class.

Enter the following details:

  1. Protocol Required – Select this check box to make Protocol selection as mandatory for exams.

Order Priority

  1. Routine – Select this check box to define the protocol selection for routine orders.

  2. Urgent – Select this check box to define the protocol selection for urgent orders.

  3. Stat - Select this check box to define the protocol selection for stat orders.

Patient Class

  1. Inpatient - Select this check box to define the protocol selection for Inpatients.

  2. Outpatient – Select this check box to define the protocol selection for Outpatients.

  3. Emergency – Select this check box to define the protocol selection for Emergency patients.

  4. Daycare - Select this check box to define the protocol selection for Daycare patients.

  5. External - Select this check box to define the protocol selection for External patients.

  6. Protocol ID – Enter the protocol ID  or select it from the [List of Values]. This is a mandatory field.

  7. Protocol Name – Enter the protocol name or select it from the [List of Values].

An error message ‘Protocol ID cannot be null’ is displayed if you save the transaction without selecting the protocol ID. Similarly, error messages will be displayed if you save the transaction without selecting the order priority or patient class.

If you try to delete the protocols used in the transactions, then a warning message ‘Exams which are linked to this protocol in the transactions will be unlinked' is displayed.

 

  1. Click [Save] or press [F10] to save a created Examination Code or click [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records.

The ‘Views’ tab is disabled when the Examination Codes screen gets displayed. On selecting the ‘Views Applicable’ check box, the Views tab is enabled.
