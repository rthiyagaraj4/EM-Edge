Contrast Agent, Drugs/Consumables for Examination

From this function you can Create an association of Contrast Agent, Drugs/Consumables to an Examination and Query the existing Contrast Agent, Drugs or Consumables for examination record.

Contrast Agent is a barium or iodine dye that is injected into the area under investigation. The dye makes the interior body parts more visible on an x-ray film. Drug is a substance used in the diagnosis, treatment or prevention of a disease or as a component of a medication. Consumables are the items or drugs that are consumed.

Creating a Contrast Agent, Drugs/Consumables for Examination

From the Masters > Examinations, double click Contrast Agent, Drugs/Consumable for Examination function.

The Contrast Agent, Drugs/Consumables for Examination screen gets displayed. Consumables tab is enabled, by default.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. The Consumables tab is displayed, by default. Enter the details in the mandatory (*) fields in the following tab pages.

Contrast Agents, Drugs

Enter the following to create Contrast Agent, Drug for the selected examination:

  1. Contrast Agent, Drug Code – Click [List of Values] to select the Contrast Agent/Drug Code. The Contrast Agents/Drugs are defined in ‘Consumables’ function. This is a mandatory field.

  2. Short Description – The description of the selected Contrast Agent/Drug is displayed.

  3. Strength – The Strength of the selected Contrast Agent/Drug is displayed. The Strength of the Drug is defined while creating the Drug in the Pharmacy Management module.

  4. Qty of Admin – The quantity of the Contrast Agent/Drug to be administered during the examination. The Quantity of the Drug is defined while creating the Drug in the Pharmacy Management module.

  5. Normal Admin Qty – Enter the Normal Quantity of the drug administered during the examinations. The quantity of the drug to be administered is defined in ‘Consumables’ function while creating the Drugs/Contrast Agent.

  6. Actual Qty Updatable – Select the check box to indicate that the actual quantity used during the examination can be updated. You can enter the actual quantity of consumption of the contrast agent/drug on completion of the examination. Clear the check box to indicate that the Normal Quantity is considered as the Actual Quantity consumed.

Consumables

Enter the following to create Consumables for Examination:

  1. Consumable Item Code – Enter the Consumable Item Code in alphanumeric format of 20 characters maximum length or click [List of Values] to select the Consumable Item code. The Consumables are defined in ‘Consumables’ function.

  2. Consumable Item Description – The description of the selected Consumable Item code is displayed.

  3. Stock Unit – Enter the Stock Unit for the consumable or click [List of Values] to select the Stock Unit. The Stock Units are created in the Materials Base module. The Stock Unit indicates the method in which the consumable is stored in the stock. Some examples of stock units are tubes, bottles, strips, etc.

  4. Normal Qty – Enter the normal quantity of the consumable normally consumed for an examination.

  5. Actual Qty Updatable – Select the check box to indicate that the actual quantity used during the examination can be updated. You can enter the actual quantity of consumption of the contrast agent/drug on completion of the examination. Clear the check box to indicate that the Normal Quantity is considered as the Actual Quantity consumed.

  1. 1. Click [Save] or press [F10] to save the record or click [Clear Record] to clear the entered data.

  2. 2. Click [Execute Query] or press [F8] to search for an existing Contrast Agent, Drugs or Consumable for Examination record.

The Contrast Agent, Drug tab is enabled if the ‘Drug Administration reqd’ check box is selected while creating the examination.
