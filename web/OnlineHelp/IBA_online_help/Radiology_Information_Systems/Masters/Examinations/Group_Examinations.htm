Group Examinations

From this function you can Group the Examinations and Query for the existing Group Examinations record.

The parameters defined are common to all the grouped examinations. The individual examinations are defined in the ‘Examination’ Master. Grouping of examinations is required when the patient has to undergo more than one examination. In this case, the patient has to undergo all the examinations grouped together. In the Group Examination, individual exams cannot have the ‘Views’ of the exam which was defined while creating the exams. You can associate a Section and Order Catalog to the group examinations. You can also define Catalog Cross-Reference of the group examination. Other details such as appointment for the examination, workload reporting of the examination, the instructions to be given to the patient prior/after the examination and the replaceable examination for this group examination are defined.

Creating a Group Examinationy6

From the Masters > Examinations, double click Group Examinations function.

The Group Examinations screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Descriptions>

  2. The Examinations tab screen is displayed, by default. Enter the details in the mandatory (*) fields in the following tab pages.

Examinations

Enter the following details:

  1. Exam Code – Enter the Exam code or click [List of Values] to select the Exam code. The Exams are defined in the ‘Examination Codes’ function. This is a mandatory field.

  2. Description – The description of the selected exam is displayed.

  3. Exam Sequence – Enter the sequence of the exam in numeric format of three digits maximum length. As this is a group examination, the sequence of the exam in the group has to be specified so that the performing radiologist performs the exams in the set order.

  4. Interval – Enter the required time interval between two exams in the group examination in the format of hh:mm.

Replaceable Exams

Enter the following details to define the Replaceable Examinations:

  1. Exam Code – Enter the Exam code or click [List of Values] to select the Exam code. The Exams are defined in ‘Examination Codes’ function under ‘Examinations’ in Masters menu.

  2. Exam Description – The description of the selected examination is displayed.

Analogous examinations, similar functions with different features, can be replaced with each other.

Patient Instructions

If the option ‘By Sex/Age’ is selected under the title Patient Instructions in the header, the following instructions for the exam are displayed based on the Order Catalog code. You cannot modify the Patient Instructions in the Group Examinations function.

 

Patient Instructions tab display the instructions based on the Gender and Age of the patient. The instructions are displayed in English and in local language.

The Patient Instructions have to be created first in the Order Catalog function of the Order Entry and Result Reporting module. If the instructions are not created, the Patient Instructions tab is disabled.

Radiologist by Facility

This tab is applicable only for users in Medicity.

The radiologist information recorded in this tab gets displayed in the ‘Auto Exam Allocation by Weightage’ master.

Enter the following details:

  1. Operating Facility ID – Select the operating facility ID from the drop-down list. This field is mandatory.

  2. Radiologist ID – Enter the radiologist ID or select it from the lookup. This field is mandatory.

  3. Radiologist Desc – Once the radiologists is selected, the radiologists description gets displayed.

Duplicate records cannot be created.

 

The ‘Radiologist By Facility’ data can be modified any time. However, in case of any deletion/update, system will check for any active allocation is in progress as per ‘Auto Exam Allocation by Weightage’ master. If so, then system prompts the user to change the weightage for that Radiologist to blank in the ‘Auto Exam Allocation by Weightage’ master function and then remove that Radiologist from Exams Master list.

  1. Click [Save] or press [F10] to save a created Group Examination or click [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for an existing Group Examination.
