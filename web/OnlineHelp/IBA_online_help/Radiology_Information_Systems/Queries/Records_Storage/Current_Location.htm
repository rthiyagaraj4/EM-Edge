Current Location

From this function you can Query the current details of a folder.

The details include the current status, the current facility that is in custody of the folder, the current location and the current holder. The exam reports that are included in the folder are displayed.

Querying the Current Details of a Folder

From the Queries > Records Storage, double-click Current Location function.

The Current Location screen gets displayed.

  1. Enter the Patient ID in the length as specified in the MP Parameter or click ‘Right Click for User Options’ and click [Standard Patient Lookup] to select the Patient. The Patient Name, Sex and Age is displayed on selecting the patient. This is a mandatory field.

  2. The logged in facility is displayed, by default.

  3. Select the required folder from the ‘Select Folder’ list. All the folders created for the selected patient are populated in the list.

  4. On entering the criteria, click [Search] or [Execute Query] or press [F8] to display the folder details of the patient.

  5. The Current Location tab is displayed, by default. Click Exams in Folder tab to view the details of exam reports stored in the selected folder.
