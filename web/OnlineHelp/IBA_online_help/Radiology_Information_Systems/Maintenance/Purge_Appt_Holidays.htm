Purge Appointment Holidays

From this function you can Purge the Holidays defined earlier.

Purge Appointment Holidays function enables you to remove historical Holidays from the system up to a specified date. This function must be used periodically to remove old records.

Purging defined Holidays Records

From the Maintenance, double-click Purge Appointment Holidays function.

The Purge Appointment Holidays screen gets displayed.

  1. The Last Purged Holiday date is displayed, by default.

  2. Enter the date in the format of dd/mm/yyyy or select the required date using the Calendar function, up to which you want to purge the Holiday records.

  3. Click [Run Process] to purge the Holiday records.
