Report of Purged Masters

From this function you can View the list of purged masters records.

The Report of Purged Masters function provides an option to Print or Delete the messages.

Viewing list of purged masters

From the Maintenance, double-click Report of Purged Masters function.

The Report of Purged Masters screen gets displayed.

  1. Enter the Master Code in alphanumeric format of six characters maximum length or click [List of Values] to select the required Master Code. This is a mandatory field.

  2. Select the required option. The available options are ‘Report’ and ‘Delete’. Select ‘Report’ to print the messages or ‘Delete’ to delete the messages.

    On selecting the option ‘Delete’, a confirmation message ‘Do you want to confirm’ is displayed. Click [Yes] to delete the purged records or [No] to cancel the process.

    On selecting the option ‘Report’, the ‘Preferences screen displays with ‘View’, ‘Print’ and ‘File’ options. Select ‘View’ to view the report, ‘Print’ to print the report or ‘File’ to generate the report as a file.

  3. Click [Run Process] to Print or Delete the purged records.
