Report/Authorise Unreported Exams

From this function you can Authorise the examination reports which are not finalized. You can  Query for an unauthorized examination reports.

The Report/Authorize Unreported Exams function enables the radiologist to finalize the examination reports. Radiologist can finalize the reports that are automatically generated by the system.

Reporting/Authorizing Unreported Exams

From the Transactions > Result Reporting, double-click Report/Authorise Unreported Exams function.

The Report/Authorise Unreported Exams screen gets displayed.

  1. The pending examination reports, which are generated automatically by the system and not finalized by a Radiologist are listed. Enter the search criteria and click [Search] to search for a specific patient record. Right-click on the ‘Right Click for User Options’ function to search for a patient record using the ‘Standard Patient Lookup’ or the ‘External Patient Lookup’ screen.

  2. Click [Search] after entering the search criteria. The search button now is displayed as [As of (current time)]. Based on the search criteria, list of examinations, which are completed and reported are displayed under ‘Work List’. Details like the exam description, contrast type, registration number, registration date and time, the attending radiologist, section, wing, ordering facility, episode type, doctor and study location are displayed.

  3. Double-click the patient ID and the ‘Authorise by Radiologist’ screen gets displayed.

  4. Click [Patient History] to view the clinical events of the patient.

  5. Click [Examination History] or [Prior Exams] to view the examination details. Examination History screen is displayed listing the patient movement details. The patient details are displayed under ‘Exam Details’, ‘Reviewer Notes’, ‘Relative Follow Up Investigation’, ‘Audit’, ‘Patient Movement’ tabs. You can also view the Report Text recorded for each examination. The details can only be viewed.

  6. Click [Relevant Clinical Summary] to view the clinical comments entered by the radiologist at the time of performing the examination.

  7. The details of the patient’s examination report in the department are displayed under Report Text, Complications, Technologist Comments and other details tabs.  You can update the Report Text and Complications.

    • If PACS is interfaced to the system, [View Image] is enabled under the Report Text tab. Click [View Image] to view the captured image.

    • Record Radiology Clinical Notes – Click the [Record Radiology Clinical Notes] to invoke the ‘Record Radiology Clinical Notes’ window from 3T to record the radiology clinical notes. User can record, sign, modify, mark as In Error, update or forward the radiology clinical note as per the access rights assigned in ‘Note Type Access Rights’ function in CA. Once the notes are recorded, signed or modified it will be displayed in ‘View Chart Summary - Clinical Notes widget’, ‘View Chart Summary - Events occurred in last 24 hrs’ widget and ‘Clinical Event History’ function in CA.

  1. Click ‘ACR Coding’ tab to record the anatomic and pathologic conditions of the patient.  

    Enter the following details:

    1. Anatomic Code/Description – Click [List of Values] to select the anatomic code. The description of the selected code is displayed in the adjacent field.

    2. Pathologic Code/Description – Click [List of Values] to select the pathologic code. The description of the selected code is displayed in the adjacent field.

    The Anatomic codes and the Pathologic codes are created in the ‘ACR codes’ function in the Result Reporting masters function.

    1. Remarks – Enter necessary remarks. This is a mandatory field.

    2. ACR Coding Completed – Select the check box to mark the ACR Coding as Complete for the respective examination.

  1. Click ‘Add to Library’ to add report to the main library from where it can be accessed by other Radiologists.

    Enter the following details:

    1. Reason – Click [List of Values] to select the reason for adding the report to the library. The Reasons are created in the ‘Examinations’ function.

    2. Public – Select the check box to mark the report as ‘Public’. Once the report is marked as public, all the radiologists can view the report.

    3. Privileged Radiologist ID/Name – Click [List of Values] to select the Privileged Radiologist ID/Name. You can associate the Privileged Radiologist ID/Name only if the check box ‘Public’ is cleared Only the radiologists who have been associated here have rights to view the report.

  2. Click ‘Review’ tab to record the review notes and place request to other Radiologist to review the report.

    My Review Notes

    Enter the review notes in free text format.

     

    Request for Review

    If the radiologist wants another review of the report, select ‘Request for Review’ tab and enter the details.

    1. Reason – Enter the reason to forward the report for authorization.

    2. With authorization – Select the check box if the radiologist is authorizing the report and forwarding it to another radiologist for authorization.

    3. One – Select the radio button if only one radiologist is authorizing the report.

    4. Many – Select the radio button if multiple radiologists are authorizing the report.

    5. To be reviewed by – If multiple radiologists are selected and if any radiologist’s review is, pending for authorization, enter the name of the radiologist.

    6. [Confirm] – Click [Confirm] to authorize the examination report.

  3. Select a Type from the list. The system-defined values are ‘Hold’, ‘Prelim’ and ‘Finalize’. Select ‘Hold’, if the examination report is not confirmed. Select ‘Prelim’, if the authorization report is in the preliminary stages. Select ‘Finalize’, to authorize and finalize the examination reports. You cannot change the status from ‘Hold’ to ‘Prelim’, but you can finalize a report for which the status is either ‘Hold’ or ‘Prelim’.

  4. Select the check box ‘Print’ to print examination report. The destination printer to print the report has to be defined. On selecting the check box, the report is sent to the destination printer for printing.

  5. Click [Save] to authorize the completed reports or click [Clear Record] to clear the entered data.

  6. Click [Execute Query] or press [F8] to search for a reported exam.

An error message ‘Not allowed to use this function’ is displayed on accessing the function, if the user is not a Radiologist. The color indicators for Priority [P], Allergy Indicator [A], Mobile Request [M] and Recalled Patient [R] are displayed in the bottom of the Ordered Exams/Appointments screen.
