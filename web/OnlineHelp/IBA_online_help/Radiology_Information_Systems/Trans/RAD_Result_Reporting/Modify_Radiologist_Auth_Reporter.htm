Modify Radiologist/Authorizing Reporter

From this function you can Modify the Radiologist or the authorizing reporter associated to a patient record and Update the examination details. You can Query for the examination reports of a patient. You can Print the examination Report.

The narration entered for an examination report while recording details can be updated using this function. You can modify the name of the Radiologist and Authorizing Reporter who has recorded the examination details. You can update the details for examination reports, which are not authorized.

Modifying a Radiologist/Authorizing Reporter

From the Transactions > Result Reporting, double-click Modify Radiologist/Authorizing Reporter function.

The Modify Radiologist/Authorizing Reporter screen gets displayed.

  1. Click [Execute Query] or press [F8] to search for the patient examination reports. Press [F7] or click [Enter Query] to search for a specific patient record. Enter the details (by default, the logged in facility name appears in both ‘Ordering Facility’ and ‘Performing Facility’ fields) and click [Execute Query] or press [F8].

  2. The examination details of the patient are displayed, based on the details recorded using the ‘Report Examination’ and ‘Authorise by Reporter’ functions. The details are displayed under ‘Examination Details’, ‘Views’, ‘Report Text’ and ‘Exam Narration’ tabs.

    Enter the following details:

    1. Exam – The exam code for which the examination report is authorized gets displayed.

    2. Narration – The exam narration recorded gets displayed. You can update the details.

    Enter the following details to modify the radiologist/authorizing reporter:

    1. Exam Code – The Exam code for which the examination report is authorized gets displayed.

    2. Status – The status of the report is displayed.

    3. Radiologist – The attending radiologist’s name is displayed. Click [List of Values] to update the name of the Radiologist. The list displays only the radiologist names corresponding to the performing location.

    4. Authorizing Reporter – Select the authorizing reporter from the list of values.

    5. Print – Select the check box to print the report. The report is printed at the configured printer.

  3. Select the ‘Print’, check box to print the modified examination report. On selecting the check box and clicking [Save], the modified records are saved and the report is printed in the destination printer set.

  4. Click [Request Reason] to view the clinical comments from the clinician.

  5. Click [Save] to modify the Radiologist or the Reporter or [Clear Record] to clear the entered details.

  6. Click [Execute Query] or press [F8] to search for the existing records.
