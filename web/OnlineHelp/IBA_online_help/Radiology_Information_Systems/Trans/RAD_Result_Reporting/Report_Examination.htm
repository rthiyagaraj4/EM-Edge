Report Examination

From this function you can Prepare the exam result immediately after the exam and Query for the examination reports in the facility. You can Print the examination Report.

The Report Examinations function enables you to record the report for the examinations conducted. A reporter with authorization privileges is permitted to record the examination details. Access to a reporter to record the examination details is given using the function ‘Reporters with Authorisation Privilege’. This function cannot be accessed, if the reporter is not authorized. The details of examinations conducted are listed. You can change the location of examination for a specific patient record, enter the complications observed during examination, record reviewer’s note or view the examination history. The report is entered as free text for each individual examination. Standard predefined text blocks may be invoked and edited while entering the report. This allows you to enter observations and comments during the reporting phase in the Review Comments. Several color codes are used to indicate the result of examination. You can also view the audit report and details on specific audits. Details of previous exams conducted for the patient can also be viewed. The examination reports for a specific patient can be queried based on the Radiologist and the Registration Date.

Recording Report for Examination

From the Transactions > Result Reporting, double-click Report Examination function.

The Report Examination screen gets displayed.  

  1. The examination results for performing facility are listed. Enter the search criteria and click [Search] to search for a specific patient record.

  2. The examination details are displayed on clicking [Search]. The patient details are displayed along with the registration number, appointment number and the Section/Wing. Priority is displayed with the respective color code.

  3. Click [T] to view the tracking location details. You can associate a location to the patient or update the existing location.

    Current location details are displayed. Enter the following details to update the location:

    1. Location Identity/Location – Select a Location Identity from the list. Click [List of Values] to select a location. The values are listed based on the Location Identity selected. This is a mandatory field.

    2. [Change Location] – Click [Change Location] to associate a location.

    3. [Cancel] – Click [Cancel] to exit from the screen.

  4. Click [C] to view the clinical events details. The clinical event details recorded for the Practitioner in Clinician Access module are displayed.

  5. Double-click on a Patient ID to view the examination reports of the selected patient.

  6. The details of the patient’s examination in the department are displayed under Examinations, Views, Exam Narration, Report Text, Complications, Reviewer Notes and Other Personnel tabs. You can update the details only under the Examinations, Report Text, Complications and Reviewer Notes tabs.

    You can record the reviewer comments before/after the examination using this tab.

    1. Exam Code - The examination code and description get displayed.

    2. Complication Type - Click [List of Values] to select and associate a Complication Type. The values defined using the ‘Complication Type’ function get listed.

    3. Complication Management Type/Description - Click [List of Values] to select and associate a Complication Management Type code. The values defined using the ‘Complication Type’ function get listed.

    4. Remarks – Enter the remarks in alphanumeric format. You can enter any additional details about the complications observed during examination.

    Enter the following details:

    1. Exam Code - The examination code and description gets displayed.

    2. Report Text - Enter the Report Text in alphanumeric format.

    3. [Spell Check] – Click [Spell Check] to check the spellings in Report Text entered. You can modify the incorrect spellings using this option.

    4. Primary Diagnosis – Click [List of Values] to select and associate a Primary Diagnosis. The values defined using the ‘Diagnosis Codes’ function get listed.

    5. Result Status – Select a Result Status from the list. There is also a provision of marking the result of the report as ‘Normal’, ‘Abnormal’ or ‘Critically Abnormal’.

    6. Confidential - Select the check box to indicate that the report is to be kept confidential. Only by authorized radiologists can access the reports, which are marked as confidential. The radiologists who have access to confidential reports are defined in the ‘Users for Viewing Confidential Reports’ function.

    Before performing selected Radiology Examinations, a contrast is injected into the body (through the mouth, through enema, intravenously, etc.) for differentiating the body parts distinctly.

    All the exams performed on the patient are displayed under the ‘Examinations’ tab.

    1. Exam Code - The examination code the description is displayed.

    2. Laterality – The laterality is displayed based on the examination code.

    3. Reporter - The name of the Radiologist is displayed.

    4. Authorizing Reporter - The name of the Authorizing Reporter is displayed. Click [List of Values] to select and associate an Authorizing Reporter.

    5. Report Completed – Select the check box to mark the examination report as complete. The Report Text has to be recorded under the ‘Report Text’ tab before selecting the check box. Clear the check box, if the examination report is not complete.  

    6. Print Draft – Select the check box to print the details for the examination report for the respective patient record.

    The Other Personnel tab displays the details of the personnel present during examination.  

    The Views tab displays the views involved in the examination.

  7. Click [Request Reason] to view the details recorded while placing a request for examination.

  8. Click [Examination History] or [Prior Exams] to view the examination details. Examination History screen is displayed listing the patient movement details. Click ‘View Details’ and the patient details are displayed under ‘Exam Details’, ‘Reviewer Notes’, ‘Relative Follow Up Investigation’, ‘Audit’, ‘Patient Movement’ tabs. You can also view the Report Text recorded for each examination. The details can only be viewed.

  9. Click [Save] to record the exam reports or [Clear Record] to clear the entered details.

The color indicators for Priority [P], Allergy Indicator [A], Mobile Request [M] and Recalled Patient [R] are displayed at the bottom of the Ordered Exams/Appointments screen. An error message ‘Not allowed to use this function’ is displayed on accessing the function, if the user is not authorized to access this function.
