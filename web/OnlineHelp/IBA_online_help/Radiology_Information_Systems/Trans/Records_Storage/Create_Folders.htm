Create Folders

From this function you can Create or Modify folders or Create new folder from existing folders.

The Create Folders function is used to create folders for a patient. You can also modify the details of an existing folder and create a new folder from an existing folder.

Creating or Modifying Folders

From the Transactions > Records Storage, double-click Create Folders function.

The Create Folders screen gets displayed with three tabs. Create Folder tab is enabled, by default.

  1. Enter the details in the mandatory (*) fields of the following tabs.

Create Folder

Enter the following details:

  1. Facility ID – The logged in Facility Name is displayed, by default. Enter the Facility Id or click [List of Values] to select the Facility. On selecting the Facility Id, the name of the Facility is displayed. This is a mandatory field.

  2. Patient ID – Enter the Patient ID in the length as specified in the MP Parameter or click ‘Right Click for User Options’ hyperlink to select a patient using the Standard Patient Lookup function. If the folder is already existing for the selected patient, an alert message ‘Folder Already Exists for this Patient, Do you want to continue?' is displayed. Click [Yes] to proceed or [No] to select another patient. This is a mandatory field.

  3. Storage Location – Enter the Storage Location or click [List of Values] to select the Storage Location. This is a mandatory field.

  4. Temporary Folder – Select the check box, if the folder is used temporarily.

  5. Generate Folder No – Select the check box for the system to generate a folder number.

  6. Folder No – Enter the Folder Number as specified in the RD Parameter for Facility function. This field is displayed, if the ‘Generate Folder No’ check box is selected. If the ‘Generate Folder Number’ check box cleared, this field is mandatory.

  7. Created Date – Enter the folder create date in the format of dd/mm/yyyy. This is a mandatory field.

  8. Remarks – Enter the Remarks in the free text format.

 

Exam Details

Enter the exam reports which are to be included in this folder.

  1. Exam Code – Enter the Exam code or click [List of Values] to select the Exam.

  2. Exam Description – On selecting the Exam code, the Exam description is displayed.

  3. Section Code/Description –The Section code and the Section description associated to the selected exam is displayed and disabled.

  4. Performed On – Enter the Date when the Exam is performed or select the date when the exam was performed using the Calendar function.

  5. Media – Enter the Media Type or click [List of Values] to select the Media type on which the report is stored.

Modify Folder Details

Click the ‘Modify Folder Details’ tab and click [Execute Query]. On clicking [Execute Query], the created folders are displayed. Click [Previous Record]/[Next Record] to select the record and make necessary changes. The changes can be done only under the ‘Exam Details’ section.

 

Exam Details

The exam details of the patient are displayed.

  1. Exam Code – Enter the Exam code or click [List of Values] to select the Exam code.

  2. Exam Description – On selecting the Exam code, the exam description is displayed.

  3. Section Code/Description – The Section code and the Section description associated to the selected exam is displayed and disabled.

  4. Media – Enter the Media Type or click [List of Values] to select the Media Type, where the report is stored.

Double-click in the ‘Remarks’ field to open the Editor.

Create New Folder From Existing Folder

Enter the following details:

  1. Facility ID – The logged in Facility name is displayed, by default. Enter the Facility ID or click [List of Values] to select the facility.

  2. Patient ID – Enter the Patient ID in the length as specified in the MP Parameter or click ‘Right Click for User Option’ to select the patient using Standard Patient Lookup function.

  3. Storage Location – Enter the Storage Location or click [List of Values] to select the Storage Location.

  4. Temporary Folder – Select the check box, if the folder is used on temporary basis.

  5. Folder No – Enter the folder number of the selected patient as specified in the RD Parameter for Facility function.

  6. Created date – The system displays the current date as the folder creation date. You can change the date, if necessary.

  7. Remarks – Enter the Remarks to create a new folder from the existing folder.

  8. Select Folder – Select the existing folder from the list, which contains all the records.

  9. Exam Code – Enter the Exam code or click [List of Values] to select the Exam.

  10. Exam Description – On selecting the Exam code, the exam description is displayed.

  11. Section Code/Description – Select the Section code. This section code specifies the section wherein the examination was performed.

  12. Performed On – Enter the date in the format of dd/mm/yyyy or select the date using the Calendar function.

  13. Media – Enter the Media Type or click [List of Values] to select the Media Type where the record is stored.

  14. Select – Select the check box to move this exam folder to the newly created folder.

This tab will function in query mode only.

  1. Click [Save] to create/modify the folders or [Clear Record] to clear the entered details.
