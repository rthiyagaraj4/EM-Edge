Request Folders

From this function you can Request for the folders of a patient from a Facility and Query for the requested folder records.

The Request Folders function enables you to place a request for folders of a patient to a particular facility. During patient’s examination, temporary or permanent folders are created for a patient. The technician can create a new folder for every exam or use the existing folders. The folder requests are placed by Department/Clinics/Wards of the same or different facility.

Requesting Folders

From the Transactions > Records Storage, double-click Request Folders function.

The Request Folders screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [View Folders] to view all the folder details of the selected patient.  

  3. Enter the details in the mandatory (*) fields of the tab pages.

Folder Details

The following details are displayed:

  1. Folder No – The Folder Number created for the patient is displayed. The folders created for the patient are displayed.

  2. Current Facility ID – The facility where the folder is located currently is displayed.

  3. Current Record Location Type – The location where the folder is stored currently is displayed.

  4. Location Type/Location – The Location Type and Location of the folder is displayed.

  5. Folder Status – The status of the folder is displayed.  

  6. Select – Select the check box to request the folder.

Exams in Folder

The following details are displayed:

  1. Date – The Date when the examination was performed is displayed.

  2. Media Type – The Media Type where the report is stored is displayed.

  3. Exam Code – The Examination code of the exam performed is displayed.

  4. Exam Description – The description of the performed exam is displayed.

  5. Registration Number – The Registration Number of the exam is displayed.

  6. Performing Facility – The facility where the examination was performed is displayed.

A folder can contain all the examination reports of a particular patient.

  1. Click [Save] to request a folder or [Clear Record] to clear the entered details.

  2. Click [Execute Query] or press [F8] to search for the existing requested folders.
