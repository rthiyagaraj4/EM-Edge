Issue Folders

From this function you can Issue a folder to the requested location.

The Issue Folders function allows you to issue folders to the Requested Locations. You can issue folders against outstanding requests, that is, the requests placed using the ‘Request Folders’ function or manual requests. Manual request can be made by a patient or by an external source.

Issuing the Folders to requested Location

From the Transactions > Records Storage, double-click Issue Folders function.

The Issue Folders screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [View Outstanding Requests] to view the Outstanding Requests of the selected Study Location. Select ‘Outstanding Request’ to display the ‘Issue Request’ tab.
