Register Outside Reads

From this function you can Record the details of the exams conducted outside the facility.

The Register Outside Reads function allows you to report the examinations that were not conducted in the logged in facility. The examinations are performed outside the facility and the reports are brought into the facility for reporting.

Registering the Outside Reads

From the Transactions > Registration, double-click Register Outside Reads function.

The Register Outside Reads screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Enter the details in the mandatory (*) fields of the tab pages.

Examinations Tab

Enter the following details in the Examination tab:

  1. Exam Code – Enter the Exam code or click [List of Values] to select the Exam code. Examination created in the ‘Examinations’ function are displayed for selection.

  2. Description – On selecting the Exam, the description of the exam is displayed.

  3. Laterality – Select the applicable laterality for the examination from the list. The listed options are Right, Left and Bilateral.

  4. Radiologist ID – Enter the Radiologist ID or click [List of Values] to select the Radiologist ID.

  5. Name – On selecting the Radiologist ID, the name of the Radiologist is displayed.

  6. Study Loc – Enter the Study Location where the exam has to be conducted or [List of Values] to select the Study Location.

  7. Amount - The bill amount is displayed. This column is displayed if  Revenue Management module is interfaced with the Radiology Information Systems module and Requests and Results module for the system to display the bill amount as soon as an order is placed.

  8. [Edit Change] - Click [Edit Charge] to invoke ‘Edit/Enter Charge Details’ screen.

  9. Exam Narration – Enter the Remarks for the Examination.

Views Tab

Enter the following details:

  1. Exam Code – Enter the Exam code or click [List of Values] to select the Exam. On selecting the Exam code, the exam description is displayed.

  2. View Code – Enter the view code applicable for the exam or click [List of Values] to select the View code. The Views are created in the ‘Views’ function.

  3. Description – On selecting the View code, the description of the View is displayed.

  1. Click [Save] or press [F10] to register the details of the exams conducted outside the facility or click [Clear Record] to clear the entered data.
