Register Examinations

From this function you can Register the examinations.

A unique Registration Number is assigned to these examinations. You can register the Examinations for the Order raised, for the booked appointment or direct appointment. On selecting this option, if the Order Entry and Result Reporting module is interfaced, the Outstanding Exams against Orders and Appointments are listed. On selecting the exam, the patient details captured during Order Entry/Booking is made default. In case of direct patients, essential details like the Patient ID, Episode/Visit Number, Referring source, Doctor, and so on. has to be entered. The system displays the Wing code and Section code automatically. The Storage Location is made default by the system based on the Operating Wing and Section. If the registration is based on order, the Section code can be changed but additional exams cannot be added. If the registration is based on appointment, the Section code cannot be changed. If the registration is based on manual request, the section code can be changed but the Operating Wing cannot be changed.

Registering Examinations

From the Transactions > Registration, double-click Register Examination function.

The Ordered Exams/Appointments screen gets displayed.

  1. The Outstanding Exams against Orders and Appointments are displayed or enter the search criteria and click [Search] to select a specific Order.

    Enter the following search criteria:

     

    1. Ordering Facility – Select the Ordering Facility from the drop-down list to display the orders based on the selected Ordering Facility. The value ’All Facilities’ is selected by default.

    2. Wing – Enter the Wing code or click [List of Values] to select a Wing. The name of the wing is displayed. The Operating Unit/Wings are defined in the ‘Operating Units/Wings’ function under the Administration Menu.

    3. Section – Enter the Section code or click [List of Values] to select a Section. Sections are displayed based on the selected ‘Operating Unit/Wing’. Sections are associated to the Operating Unit/Wing in the ‘Sections for Operating Unit/Wing’ function under the Administration Menu.

    4. Priority – Select the priority applicable for the examination from the list. The listed system-defined values are All, Stat, Urgent and Routine. The value ‘All’ is displayed, by default.

    5. Show – Select the option from the list. The listed values are All Requests, Unscheduled Requests and Scheduled Requests. The value ‘All Requests’ is displayed, by default.

    6. Include Future Requests – Select the check box to include the future requests.

    7. Sex – Select the Gender of the patient. The listed values are All, Male and Female. The value ‘All’ is displayed, by default.

    8. Episode Type - Select the Episode Type from the list. The listed values are Inpatient, Outpatient, Referral and All. If the patient is an Inpatient, the Episode Type is ‘Inpatient’; if the patient is an Outpatient, the Episode Type is ‘Outpatient’; if the patient has been referred from another facility, the Episode Type is ‘Referral’. The value ‘All’ is displayed, by default.

    9. P (Priority) – The priority of the pending examination is indicated using colours.

      indicates Urgent

    4. Patient ID – Enter the Patient ID in the length as specified in the MP Parameter or click ‘Right Click for User Options’ to select the patient using Standard Patient Lookup function. On selecting the Patient ID, the name of the patient is displayed.

    5. [T] – Click [T] to view the location information of the patient.

    6. Status – Select the status of the examination from the list. The listed values are Pending Reg, Replaced, Rejected, Cancelled, Pending/Modified and All. ‘All’ gets displayed, by default.

    7. Order/Appointment Date [From-To] – Enter the Order/Appointment date range in the format of dd/mm/yyyy or select the date range using the Calendar function.

    The colour indicators for Priority [P], Mobile Request [M], Recalled Patient [R] and Billing Status [B] are displayed at the bottom of screen. An error message ‘Not allowed to use this function’ is displayed on accessing the function, if you are not authorized to access this function.

  2. On searching, examinations pending for registration are displayed based on the entered criteria.

  3. Double click on the Patient ID and ‘Register Examination’ screen gets displayed.

  4. Right click on the Patient ID to select the Request Comments, Cancel Appointment, View Patient File Details, Assign/View Protocol, Patient Demographics and Radiologist Comments option.

    On selecting the Radiologist Comments option, the Radiologist Comments screen is displayed with the comments recorded for the examination in read-only mode. This option is disabled when no radiologists comments are recorded.

    On selecting Patient Demographics option, the ‘View Patient Details’ screen opens in a new Internet Explorer browser window. You can view the patient demographic details in the View Patient Details tab.

    On selecting the View Patient File Details option, the Patient File Details screen is displayed. The X-ray folder is tracked based on the old medical record file numbers in the Patient File Details screen before exam registration. The View Patient File Details option can be accessed by authorized radiologist only.

    On Selecting the Cancel Appointment option, the Cancel Appointment screen is displayed. This option is enabled only for the patients with appointment.  

    On selecting the Review Comments option, the Clinical Comments, Examination comments and the Item Narration is displayed. The reason for overriding duplicate orders is displayed if it is captured while placing orders in Clinician Access (CA). Click [Close] to exit from the Request Comments screen.

  5. Right click on ‘Order By’ or ‘Attend By’ to view the Physician’s contact details. The name of the Physician and the contact details are displayed.

  6. Click [T] to view the patient tracking location details.  The ‘Change Patient Location’ screen is displayed on selecting a patient.

    Enter the following details to change patient location:

    1. Patient ID – The Patient ID and Patient Name is displayed.

     

    Current Location Details

    1. Current Location Status – The Current Location Status of the patient is displayed.

    2. Location Identity / Location – The location identity or the location of the patient is displayed.

    3. Movement Date Time – The movement date and time of the patient is displayed.

     

    Change Location

    1. Location Identity / Location – Select a location identity / location from the list. The locations created using ‘Study Location’ function are displayed for selection.

     

    Click [Change Location] to record the new location or [Cancel] to retain the old location.

  7. Click [Direct Registration] for register the Exam directly.

  8. Click [Execute Query] or press [F8] to search for an ordered exam, or booked appointment.

The color indicators for Priority [P], Allergy Indicator [A], Mobile Request [M] and Recalled Patient [R] are displayed in the bottom of the Ordered Exams/Appointments screen. The Patient Name is highlighted with the color indicator ‘Red’ for the high risk patient.
