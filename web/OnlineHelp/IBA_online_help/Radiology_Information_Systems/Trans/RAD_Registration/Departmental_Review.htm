Departmental Review

From this function you can Cancel/Replace/Reschedule/Add a patient and Repeat the examination before reporting for authorisation and Query for the desired departmental review record.

The Departmental Review function displays all the registered examinations. You can cancel, replace, reschedule or add additional exams to the registered requests.

Performing a Departmental Review

From the Transactions > Registration, double-click Departmental Review function.

The Departmental Review screen gets displayed.

  1. Click [Execute Query] or press [F8] to display all the registered examinations. Click [Next Record] or [Previous Record] to search for a particular examination.

Or

  1. Enter the criteria and click [Execute Query] to select for a particular registered examinations .

  2. The examination can be Cancelled/Replaced/Rescheduled.

Cancel

Select the 'Cancel' check box to record the Cancellation Reason and cancel the examination.

Replace

  1. Replace Exam – Select the 'Replace Exam' check box to replace the examination.

Enter the following details to replace the examination:

  1. Current Exam – The Current exam code and description is displayed..

  2. Replace Exam – Select the required Examination code. The examination description is displayed in the adjacent field.

  3. Laterality – Select the applicable laterality.

  4. Replacement Reason – Select the Reason for replacing the original examination.

Click [OK] to replace the examination, or click [Cancel].

Reschedule

  1. Reschedule Exam – Select the ‘Reschedule’ check box to reschedule a particular examination.

  2. Enter the reason for rescheduling the examination and click [OK] to invoke ‘Ordered Exams for Appointment’ screen or [Cancel] to exit from the screen.
