Function

From this function you can Query the list of standard functions packaged with Enterprise Management, for any reference purposes.

Enterprise Management is a product consisting of a set of functions of different modules. When the product is packaged, the objects of all the functions of the modules are supplied and appropriate entries for the functions that are supplied are internally maintained. Only the System Administrator can access this function.

Querying a Function

From the SM Admin Setup > Queries, select the ‘Function’ function.

The Function screen gets displayed.

  1. Click [Query] to invoke the Query screen.

  2. Enter the query criteria in the respective fields. <Related Field Description>

  3. On specifying the required query criteria and sort order entry, click [Execute] to select the function records that satisfy the entered query criteria and to display them as per the requested sort order or click [Execute] to display all the records.
