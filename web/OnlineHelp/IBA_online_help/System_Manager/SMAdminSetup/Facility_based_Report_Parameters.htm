Facility based Report Parameters

From this function you can Define a new Facility based Report Parameter and Query for a desired Facility based Report Parameter.

The Facility based Report Parameters function helps in setting up some of the parameters of a report, which has been categorized as facility based.

A report is normally classified as facility based, when the facilities are distinctly identifiable physical locations and distributed. Most of the reports produced at a facility level are classified as facility based and other reports produced at an enterprise level are classified as non-facility based. All the transaction-based reports are classified as facility based reports.

Using this function, a report group can be set as a facility-based report.

Creating a Facility Based Report Parameter

From the SM Admin Setup > Print Routing, select the Report Service function.

The Report Service screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a record or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing record.

  5. Click [Delete] to delete a queried record.
