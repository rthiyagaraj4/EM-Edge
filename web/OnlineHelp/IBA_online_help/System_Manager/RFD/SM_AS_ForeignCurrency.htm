Enter the following details to define a Foreign Currency:

 

  1. Currency Code – Enter the Currency Code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Currency Name – Enter the Currency Name in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  3. Decimal Name – Enter the Decimal Name in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Conversion Rate – Enter the Conversion Rate in numeric format of eight characters maximum length. This is a mandatory field.

  5. Conversion Operator – Enter the Conversion Operator as either ‘*’ or ‘/’.This is a mandatory field.

  6. Number of Decimals – Enter the number of decimals as either two or three. The number of decimals indicates the numerals after the decimal point which can be present in the output during conversion. This is a mandatory field.

  7. Long Description – Enter the long description of the Foreign Currency in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  8. Short Description – Enter the short description of the Foreign Currency in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  9. Main Account Code1 – Select the Main Account Code1 using [?] function. The values created using ‘Chart of Accounts’ function through 'GL Setup' menu in Patient Billing Manager or Accounts Receivable modules are displayed for selection.

  10. Main Account Code2 – Select the Main Account Code2 using [?] function. The values created using ‘Chart of Accounts’ function through 'GL Setup' menu in Patient Billing Manager or Accounts Receivable modules are displayed for selection.

  11. Department Code - The Department Code associated to the selected Main Account code gets displayed. The Department Code is associated to the Main Account using the function 'Create Posting Account Combination' in the Patient Billing Manager or the Accounts Receivable module. This field is enabled for selection on selecting a value for 'Main Account Code2'.

  12. Product Group Code - The Product Group Code associated to the selected Main Account code gets displayed. The Product Group Code is associated to the Main Account using the function 'Create Posting Account Combination' in the Patient Billing Manager or the Accounts Receivable module.

  13. Product Line Code - The Product Line associated to the Product Group Code which is linked to the Main Account Code gets displayed.

  14. Sub Ledger Group Code - The Sub Ledger Group Code associated to the selected Main Account code gets displayed. The Sub Ledger Group Code is associated to the Main Account using the function 'Create Posting Account Combination' in the Patient Billing Manager  or the Accounts Receivable module.

  15. Sub Ledger Line Code - The Sub Ledger Line associated to the Sub Ledger Group Code which is linked to the Main Account Code gets displayed.
