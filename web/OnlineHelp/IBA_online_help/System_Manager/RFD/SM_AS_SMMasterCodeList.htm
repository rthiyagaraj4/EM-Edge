Enter the Report Criteria as follows:

 

  1. Module - Select a module from the list of installed modules.

  2. Function ID [From/To] - Enter the range of Function IDs (pertaining to the module selected) to generate the report. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ Function ID and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Function ID. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value. Click [?] to query the Function IDs set up and required values can be picked up from the list displayed.

  3. Order By – Select the Sort Order as ‘Function ID’ or ‘Function Name’ to generate the report in the required order.
