Enter the following details to create a Responsibility record:

 

  1. ID - Enter the Responsibility ID in alphanumeric format of 20 characters maximum length. A Responsibility ID is a unique value to identify a Responsibility record and cannot be changed after creation. This is a mandatory field.

  2. Name - Enter the Responsibility Name in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  3. Responsibility Group – Select the Responsibility Group from the list of values. The values defined for Responsibility Group gets listed.

  4. Menu - Select the Menu to be associated to the responsibility by clicking [?] function. A root menu only can be associated to the responsibility. This is a mandatory field.

  5. User Desktop - Select the required User Desktop using the [?] function to associate it to the Responsibility. A User Desktop is defined through the ‘Create User Desktop’ function.

  6. Role – Select the Role of the user from the list. The pre-defined values are ‘Medadmin’ and ‘Medusers’. This is a mandatory field.

  7. Remarks - Enter any required Remarks about the responsibility in alphanumeric format of 200 characters maximum length.

  8. Effective (From/To) - Enter the Effective date ranges of the responsibility in dd/mm/yyyy format. ‘From’ date corresponds to the date of the responsibility coming into effect and the 'To' date corresponds to the date, when the responsibility stops being effective.

  9. Enabled - Select the check box to enable the record in the transaction functions. The check box is selected, by default.
