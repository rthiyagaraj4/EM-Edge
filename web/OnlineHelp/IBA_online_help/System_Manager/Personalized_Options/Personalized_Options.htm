Personalize

This section guides you in Setting certain Personalized options for the logged in user.

The users can Personalize their own settings for using technology to accommodate the differences between individuals. From this option, you can change the password of the logged user, set favorites for frequently accessed functions, change the colour theme for the application and change the language.

To Personalize the user options

From the Menu option, select (Personalize) icon.

  1. Click to personalize the logged in user setting.

  2. The Personalize Options is displayed at the bottom of the menu frame, namely:

Change Password

Frequently Assessed Functions

Choose Color Theme

Switch Language
