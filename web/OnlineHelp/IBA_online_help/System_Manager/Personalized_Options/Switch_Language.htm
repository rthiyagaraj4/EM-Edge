Switch Language

From this option, you can Change the language.

The Switch Language enables the user to set the language. The applicable languages are fed through the scripts during the time of installation. The available languages are Arabic, Chinese, English, Equadorian, Russian and Thai.

To Switch Language

From the Personalize Options, select the Switch Language option.

The Switch Language screen is displayed.

  1. Select the language from the list of values.

  2. Click [Apply] to save the language or [Reset] to clear the selected language.
