Choose Color Theme

From this option, you can Choose the color theme to personalize the look and feel.

Using the Choose Color Theme, the logged in user can personalize the background color of the application with the available themes. The available themes are Amazon, Classic Blue, Mystic Green, Rich Gold and Cool Violet.

Choose a Color Theme

From the Personalize Options, select the Choose Color Theme option.

The Choose a theme screen is displayed.

  1. Click [Apply] to select and save the theme.
