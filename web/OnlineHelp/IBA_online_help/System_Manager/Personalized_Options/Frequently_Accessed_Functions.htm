Frequently Assessed Functions

From this option, you can Create frequently accessed function list and set its attributes under Frequently Assessed Functions.

The Frequently Assessed Functions is set to have an easy access to all the frequently used functions.

Setting up a Frequently Assessed Functions

From the Personalize Options, select the Frequently Assessed Functions.

The Frequently Assessed Functions screen is displayed.

  1. Click [Create] to invoke the Create screen.

  2. The Responsibility Name and the Menu name is displayed. You cannot modify these values.

  3. Enter the details to create Frequently Assessed Functions.  

    Enter the following details to create Frequently Assessed Functions:

    Available Functions

    1. Function Name - Enter the function name in alphanumeric format of 30 characters maximum length. From the Sub-menu list of values, select the sub-menu. Click [Search] to search the available function or [Clear] to clear the entered data.

    2. Sub Menu - Select the value from the list. The Sub-menu lists the value where you can select the frequently used functions.

    3. Include - To include the displayed function in the Function Name column, select the desired check box.

     

    The name of the functions which are already added as the Frequently Accessed Functions are displayed. To remove the displayed function in the Function Name column, select the desired check box in the Remove column.

  4. Click [Apply] to save the Frequently Assessed Functions or [Reset] to clear the entered data.
