Add Visit

Add Visit tab is used to register a visit for patients visiting the Location as an Outpatient. You can directly invoke this tab to register visits for Walk in Patients with or without referral. You can also register visits for patients with appointment or invited patients from waiting list using this function. Privilege to use this function is defined in the Operator Station attached to the User in the Application Masters module.

 

Register Visit for walk in patient with or without referral

  1. Enter the search criteria to search for Walk in Patients and click [Search].

  2. If the patient is not found while searching, register the patient from the Patient Search screen.

  3. To register a visit for a referral patient, perform a Referral Search and select the patient. If the selected referral patient is not registered in the facility, register the patient.

  4. If patient has any outstanding amount, the system pops-up a screen for Credit Authorization which displays the outstanding amount.

    1. Patient Remarks – The remarks recorded from the selected patient during bill generation for the previous encounter gets displayed.

    2. Outstanding Amount – The outstanding bill amount from the previous encounters get displayed.

    3. User ID – Enter the User ID. Users with authorization to update the financial details are only allowed to login and modify the billing details. This field is mandatory.

    4. PIN – Enter the PIN for the selected User. This field is mandatory.

    5. Remarks – Enter the remarks in alphanumeric format to record the remarks against the selected patient record.

    Click [Close] to login and update or [Cancel] to exit from the screen.

  5. Patient details are displayed in the Add Visit screen.

Register Visit for patients with appointment

Patients with appointment are selected from the Appointment List tab.

On selecting the patient with appointment, the Appointment details of the patient are displayed in the Add Visit screen.

Register Visit for invited patients from Waiting List

Invited patients from waiting list are selected from the Invitation List tab.

  1. On selecting the Invited patients, the details of the patient are displayed in the Add Visit screen.

  2. The Credit Authorization screen displays if the invited patient has a previous encounter with the facility.

    1. Patient Remarks – The remarks recorded from the selected patient during bill generation for the previous encounter gets displayed.

    2. Outstanding Amount – The outstanding bill amount from the previous encounters get displayed.

    3. User ID – Enter the User ID. Users with authorization to update the financial details are only allowed to login and modify the billing details. This field is mandatory.

    4. PIN – Enter the PIN for the selected User. This field is mandatory.

    5. Remarks – Enter the remarks in alphanumeric format to record the remarks against the selected patient record.

    Click [Close] to login and update or [Cancel] to exit from the screen.

Financial Details Tab

  1. Click [Previous Encounter] to view the previous encounter details of the patient. This button is enabled if the patient has a previous encounter with the facility. You can also view the financial and the bills generated for patient's previous encounter in the facility.

  2. Click [Contact Details] to view or modify the contact details of the patient.

  3. Click [Apply] to register the visit or [Reset] to clear the entered data.

  4. On clicking [Apply], Record Encounter Discount screen is displayed to record discount for an encounter.

  5. You can record discount for a package through the Record Adhoc Discount screen, if a package is associated to the patient.

  6. Bill Settlement Details screen is displayed on recording the discount details. You can update the settlement details before generating the receipt.

  7. Create Patient File screen gets displayed if file management parameters are set for the facility.

  8. Enter the details in the mandatory (*) fields. <Related Field Description>

  9. Click [OK] to generate file number or [Cancel] to cancel file generation.

  10. If the future order exist for the selected patient, the Future Order screen is displayed alerting the user to link the previously placed order with future date.

  11. If internal reports are defined, the Internal Reports screen gets displayed. Click [OK] to generate Internal Reports or [Cancel] to exit from the screen.

  12. Encounter ID, Patient File Number and Visit Charge is generated successfully on registering a visit.

If the selected patient has today’s appointment, appointment details are displayed in the ‘Today’s Appointment’ frame of the Add Visit screen. Click the ‘Appt No’ hyperlink from to display the details in the Visit tab.

The financial details recorded for the patient during registration gets displayed. You can update the financial details by clicking on the 'Financial Details' tab.
