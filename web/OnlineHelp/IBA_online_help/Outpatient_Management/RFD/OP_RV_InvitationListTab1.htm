Enter the following details for querying an invited waiting list patient:

 

  1. Citizen ID - Enter the Citizen ID in numeric format as entered while registration. The ‘Prompt’ specified for the Citizen ID in MP Parameter is displayed as the field legend. The length and format as specified in MP Parameter has to be adhered.

  2. Alternate ID Type(s) - Enter Alternate ID number(s) in the format and length as specified in MP Parameter. Based on the MP Parameter setup, system displays the Alternate ID Type description(s).

  3. Other Alt ID – Select the required Alternate ID from the list of values. The listed values are defined in the function ‘Alternate ID Type’ of MP module. The field ‘Other Alt ID’ is displayed if ‘Accept Other Alt ID’ is set in MP Parameter (Alt ID checks tab).

  4. Other Alt No - On selecting the required Alternate ID type, enter the ID in alphanumeric format of 20 characters maximum length.

The above field names are displayed based on the legends for Alternate ID defined in the MP Parameter.

  1. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in the MP Parameter or select it using the [?] function.

  2. Nationality – Select the required value using the [?] function. The listed values are defined in Master Patient Index module.

  3. Town – Select the Town using the [?] function. The listed values are defined in Master Patient Index module.

  4. Area – Enter the Area in alphanumeric format of 30 characters maximum length or select the area using the [?] function. The listed values are defined in Master Patient Index module.

  5. Region – Select the Region using the [?] function. The listed values are defined in Master Patient Index module.

  6. Specialty – Select the Specialty using the [?] function. The listed values are defined in Application Masters module.

  7. Category – Select the Category using the [?] function. The listed values are defined in Appointment Scheduling module.

  8. Wait List No – Enter the Wait List Number in alphanumeric format of 12 characters maximum length.

  9. Priority - Select the Priority of referral from the list of values. The values are ‘Elective’, ‘Urgent’ and ‘Emergency’.

  10. Location Type - Select a Location Type from the list. The values are 'Clinic' and 'Procedure Unit'. The value 'Clinic' is displayed, by default.

  11. Location - Select a Location using [?] function. Locations are listed based on the selected Location Type. On selecting the Location Type as 'Clinic', Clinics created using 'Clinic' function are displayed for selection. You can register the Visit to a Clinic. On selecting the Location Type as 'Procedure Unit', Procedure Units created using 'Procedure Unit' function are displayed for selection. You can register the Visit to a Procedure Unit.

  12. Invitation Period [From-To] - Enter the ‘From’ and ‘To’ date in date format of dd/mm/yyyy format. The entered ‘From’ date must not be greater than the entered ‘To’ date. The ‘To’ date must not be lesser than the ‘From’ date, or the system prompts with a message. The current date gets displayed, by default.
