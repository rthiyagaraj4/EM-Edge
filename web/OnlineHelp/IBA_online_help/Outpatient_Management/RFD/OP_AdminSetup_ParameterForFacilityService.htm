Enter the following details to create a Parameter For Facility Service record:

 

  1. Facility - Select the Facility from the list. All the facilities of the enterprise are displayed in the list. This field is mandatory.

  2. Service - Select the Service from the list. The listed values are limited to the Services associated to the selected Facility. This field is mandatory.

 

  1. Follow-up - Enter the number of days that must elapse from the date of the last visit in numeric format of six digits maximum length for episodes of the selected Service in a Facility, where the last visit require a follow-up. This value is used to close an Outpatient Encounter of the selected Service in a Facility, based on the elapsed days, by a background function. This field is mandatory.

  2. No Follow-up - Enter the number of days that must elapse from the date of the last visit in numeric format of six digits maximum length for episodes of the selected Service in a Facility, where the patient has no follow up advised, to close forcibly an open episode. This value is used to close an Outpatient Encounter of the selected Service in a Facility, based on the elapsed days, by a background function. This is a mandatory field.

 

Closure Rule – Based on No Shows, Last Visit & Future Appointment

If the Episode closure by value is selected as “No Shows” in the OP Parameter screen, the following fields will be displayed here:

 

  1. Number of Consecutive No shows   - Define the number of no shows to be elapsed after which you  can consider a patient’s episode for closure.

  2. No Visits in Last X Months – Define the minimum number of past months within which if a patient has not made a visit, their episode can be considered for closure.

  3. No Appointments for next X Months - Define the maximum number of future months within which if a patient’s next appointment is not scheduled, their episode can be considered for closure.

Note: Only when all the above mentioned three conditions are met, the system will consider the patient record for batch process to conclude the episode; else the record will not be considered.
