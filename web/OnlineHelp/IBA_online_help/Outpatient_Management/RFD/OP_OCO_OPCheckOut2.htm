Enter the following details to perform the Check out operation:

 

  1. Visit Type - Visit Type of the Visit is displayed and cannot be changed.

  2. Attending Practitioner – If a Practitioner is assigned, then the Practitioner Name and Practitioner type are displayed and cannot be changed. In case the Practitioner is not assigned, then the user can select the required practitioner. This field is mandatory if Practitioner is not yet assigned.

  3. Circumstance of Injury - Select the Circumstance of Injury from the list.

  4. Medical – Check on the box to indicate the case requires only medical attention.

  5. Surgical - Check on the box to indicate the case requires surgical attention.

Disposition

  1. Date/time – The current date/time is displayed as the disposition date and time. Cannot be modified.

  2. Brought Dead - This field is disabled for Outpatients.

  3. MLC – Check on to indicate the patient is registered under Medico Legal Case or clear the check box.

  4. Postmortem Required – Check on if postmortem is required for the deceased body, or clear the check box. This field is enabled only if the ‘Type’ selected is deceased.

  5. Type - Select the Disposition Type from the list. The displayed list is limited to a set of active Disposition Types valid for Visit Conclusion.

  6. Deceased Date/Time – Enter the deceased date and time of the patient. If the ‘Type’ is selected as deceased then this field becomes mandatory.

  7. High Risk Body – Check on to indicate the deceased body is a high risk body, or clear the check box.

  8. Body Release Date/Time – If the ‘Type’ is selected as deceased then this field becomes mandatory and the current system date/time gets populated automatically.

Recall Details

If the Patient is required to come back to the clinic, then the following details can be entered:

  1. Come back After - Click on the check box to indicate whether the Patient has to come back again on a future date. If it is checked, enter the number of days within which the patient has to come back in numeric format of three digits maximum length in ‘Day(s)’, ‘Week(s)’ or ‘Month(s)’. This field is mandatory on checking the ‘Come Back’ check box.

  2. Reason - Enter the Reason for recalling the Patient in alphanumeric format of 200 characters maximum length.

  3. Instructions - Enter the Instructions of Recall to the Patient in alphanumeric format of 200 characters maximum length.
