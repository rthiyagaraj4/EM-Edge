Enter the following query criteria to display a set of Patient Visit records:

 

  1. Date - Displays the current date. It cannot be modified.

  2. Start Time - Displays ‘00:00’ as Start Time, which means that Patients who have registered from ‘00:00’ hours onwards for the current day will be displayed.

  3. View (Single/Multiple) - Single view is selected by the system by default and cannot be modified. The others details allowed as query criteria are Encounter ID, Queue No, Patient ID, Location and Practitioner.

  4. By Visit Type - Click on the check box to group the Patient queue records Visit Type wise.

  5. Previous Days Visits Only – Check the box to include previous days only or clear the check box.

  6. Emergency patients Only – Click on the check box to group the Patient records, which are emergent in nature or clear the check box.

  7. Include Checked out Patients – This option is set as unchecked, that is, not to include the checked out patients and it cannot be modified.

  8. [Refresh] - Click on this button to refresh the Patient queue records. By default, this screen will be refreshed automatically based on the Refresh value parameter setup. This button can be clicked to refresh or re-query the screen manually.
