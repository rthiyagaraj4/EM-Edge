Enter the following detail:

 

  1. Referral Type – The Referral Type is displayed with values ‘Login facility’, ‘Enterprise’ and ‘External’. This field is mandatory.

  2. Healthcare Setting Type – Select the required health care setting from the list of valid values. All the health care setting type created in Enterprise Configuration module will be displayed. If referral type is Login Facility, the system defaults the healthcare setting type of the logged in facility and disables the field. This field is mandatory.

  3. Referred Source – Select the source to which patient is to be referred. If referral type is login facility, then the logged in facility name is defaulted. This field is mandatory.

  4. Specialty – Select the required specialty from the list. This field is mandatory.

  5. Location - Select the location type and location from the list. The listed values for location type are ‘Clinic’, ‘Nursing Unit’, ‘Daycare’ and ‘Procedure Unit’. The list box next to ‘location’ will contain values that correspond to the selected location type.

  6. Service – Select the required service from the list. All services for the selected specialty and location will be displayed.

  7. Referred For – Select the purpose of referral from the list. This field is enabled only when location type is selected as nursing unit. The values listed are ‘Booking’ and ‘Admit’. This field is mandatory if location type is selected as nursing unit.

  8. Practitioner – Enter the Practitioner name in alphanumeric format of 30 characters maximum length. If referral type selected is external, this field shall allow only entry of a practitioner name.  If referral type is of type login facility or Enterprise, the system allows user to select the required practitioner using [?] button. The practitioners listed would be based on the specialty and location selected.

  9. Preferred Date - Enter the preferred date for visit or admission in dd/mm/yyyy format or select the required date using the Calendar function.

  10. Priority - Select the priority of referral from the list. The values are ‘Elective’, ‘Urgent’ and ‘Emergency’.

  11. Referral Notes - Click on referral notes to enter notes about the patient being referred.
