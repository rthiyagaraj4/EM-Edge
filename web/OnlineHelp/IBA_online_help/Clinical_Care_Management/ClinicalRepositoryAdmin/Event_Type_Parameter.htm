Event Type Parameter

From this function you can Build Clinical Repository for events in the Central Repository.

The Event Type Parameter function maintains the clinical repository of events undergone by patients in the facility as well as external patients directly registering to the facility.

Recording Event Type Parameter

From the Clinical Repository Admin,  select Event Type Parameter function.

The Event Type Parameter screen gets displayed.

  1. Select the check box against the required Events under Build Clinical Repository and Build Clinical Repository for Direct Regn.  

  2. Click [Apply] to save the Event Type Parameter record or click [Reset] to reset the selected data.
