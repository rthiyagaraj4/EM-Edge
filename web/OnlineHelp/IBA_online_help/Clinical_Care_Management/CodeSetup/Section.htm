Section

From this function you can Create a new Section applicable to a Section Type and Query for a desired Section.

The Section function allows you to define different types of Sections, which determine the layout of the Clinician Notes. The Sections can be segregated into various types like ‘Free Format’, ‘Result Linked’, ‘Main Section’ and ‘Template Based’.

Creating a Section

From the CA Code Setup, select Section function.

The Section screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Section created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Sections.
