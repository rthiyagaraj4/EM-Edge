Position for Intervention

From this function you can Create an association of positions to interventions and Query for existing records.

This function enables you to associate roles to interventions. The interventions are performed by users belonging to the position mapped to the interventions.

Associating Position to Intervention

From the CA Code Setup, select Position for Intervention function.

The Position for Intervention screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Role Type from the list. This is a mandatory field.

  3. Select the Staff Type from the list. The Staff Types are displayed based on the selected Role Type. This is a mandatory field.

  4. Select the Position using [?] function. This is a mandatory field.

  5. Click [Search] to view the list of interventions.

  6. Select the check box in the column header to select all the interventions or select the check box against the required intervention to be associated to the position.

  7. Click [Apply] to save the association of Positions to Interventions or [Reset] to clear the entered data.

  8. Click [Query] to search for existing 'Position for Intervention' records.
