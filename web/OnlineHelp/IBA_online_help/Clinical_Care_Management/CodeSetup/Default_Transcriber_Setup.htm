Default Transcriber Setup

From this you can Set a Default Transcriber to a Practitioner, Specialty and Patient Class.

Associating a default Transcriber to Practitioner/Specialty/Patient Class

From the CA Code Setup, select Default Transcriber Setup function.

The Default Transcriber Setup screen gets displayed.

  1. Click [Create] to create to invoke the create screen.

  2. Select the Transcriber from the list. The list displays Practitioners defined with Practitioner Type ‘Transcriber’. This is a mandatory field.

  3. Select Group By as Practitioner, Specialty or Patient Class.

  4. The Select Tab is displayed, by default. The screen displays alphabets from ‘A to Z’ and ‘Others’ that are hyperlinked. Each alphabet contains Practitioners or Specialties based on the selected 'Group By' field. The ‘Others’ hyperlink contains Specialties or Practitioners that begin with characters other than alphabets.

  5. Click on the required alphabet and the Specialties or Practitioners beginning with the selected alphabet are listed. Alphabetical display is not applicable if Group By is selected as Patient Class.

  6. Select the check box against the required Practitioners, Specialties or Patient Classes for association. Only one transcriber can be set as default for the selected Practitioners, Specialties or Patient Class.

  7. Click 'Associate' Tab to view the selected Specialties or Practitioners. Clear the check box to dissociate the association.

  8. Click [Apply] to record setting default transcriber for Practitioners, Specialties and Patient Classes.

  9. Click [Reset] to clear the entered data.
