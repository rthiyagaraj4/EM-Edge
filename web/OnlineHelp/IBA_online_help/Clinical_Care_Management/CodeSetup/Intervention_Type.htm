Intervention Type

From this function you can Create Intervention Types to which interventions are associated and Query for existing Intervention Type records.

The Intervention Types are helpful in segregating the total interventions according to the purpose. Grouping individual interventions under the intervention types helps in searching the intervention in the application. Different intervention types for the diagnosis are created which are applicable for the implementation of care to the patient.

Creating an Intervention Type

From the CA Code Setup, select Intervention Type function.

The Intervention Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Intervention Type or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Intervention Type records.
