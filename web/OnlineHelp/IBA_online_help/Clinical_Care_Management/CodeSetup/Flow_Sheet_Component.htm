Flow Sheet Component

From this function you can Create a new Flow Sheet of type New or Charting. You can also Query for an existing Flow Sheet created.

Flow Sheets are used in Patient Chart History function. Flow Sheet Component helps the practitioner to define a customized description of the portion of the history that is to be accessible when this flow sheet is invoked in the patient chart history. On selecting the Flow Sheet and Practitioner Type it is applicable to all the Practitioners and Specialty, by default.

Creating Flow Sheet Components

From the CA Code Setup, select Flow Sheet Component function.

The Flow Sheet Component screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Flow Sheet Component created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Flow Sheet Components.
