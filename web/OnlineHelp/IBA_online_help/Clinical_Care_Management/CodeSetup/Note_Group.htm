Note Group

From this function you can Create and maintain the Note Groups applicable for the facility and also Query for a desired Note Group.

Note Groups form a broader classification level for Note Types. Note Groups are created to associate several Note Types to it. The system validates the usage of the Note Types based on the Note Group to which it belongs. The Applicable Task associated to each Note Group determines the nature of the Note Group and thus the usage of the Note Types associated to the Note Group.

Creating a Note Group

From the CA Code Setup, select Note Group function.

The Note Group screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Note Group created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Note Group.
