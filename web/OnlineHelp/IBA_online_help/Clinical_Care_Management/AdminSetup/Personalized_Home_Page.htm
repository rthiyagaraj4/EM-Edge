Personalized Home Page

From this function you can Customize the components of the clinician home page displayed in the Clinician Access module for a ‘Practitioner’ or for all ‘Practitioners’ belonging to a particular ‘Practitioner Type’.

Clinician Access Home Page is customized for Practitioner(s) based on Practitioner Type.

Personalizing Home Page

From the CA Admin Setup, select Personalized Home Page function.

The Define Personalized Home Page screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Search] to display all Sub Component Descriptions grouped under the Component Descriptions.

  4. Select the Applicable check box against the required Components to associate to the selected Practitioner Type or select the Applicable check box at the header level to select all the components.

  5. Click [Apply] to save the Personalized Home Page or [Reset] to clear the entered data.
