Construct CA Menu

From this function you can Construct the components of the Clinician Desktop Menu and Patient Chart Menu for the Menu ID defined. You can also Assign the iSOFT Enterprise Management default Menu to the defined Menu ID and Preview the Menu structure.

The Construct CA Menu function allows you to design the menu. This function provides for choosing a defined root menu and associating Submenus and functions under these Submenus or directly adding functions under the Root Menu, thus defining the structure of the menu.

Constructing CA Menu

From the CA Admin Setup, select Construct CA Menu function.

The Construct CA Menu screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save constructed CA menu or [Reset] to clear the entered data.

 

The menus defined in ‘Define Menu’ function with ‘Root Menu’ check box cleared are listed as Sub Menus.
