Clinical Event Parameter

From this function you can Define the Clinical Event Parameter.

Using this function, Indicators can be defined for High Value, Low Value, Abnormal Value, Critical Value, Critical Low Value, Critical High Value and Delta Value, which are used for different values in the Chart history of the patient. This function is also used to indicate the categories for which results have to go in for review.

Defining Clinical Event Parameter

From the CA Admin Setup, select Clinical Event Parameter function.

The Clinical Event Parameter screen gets displayed.

  1. Enter the details in the required fields. <Related Field Description>

  2. Click [Apply] to save the Clinical Event Parameter or [Reset] to clear the entered data.
