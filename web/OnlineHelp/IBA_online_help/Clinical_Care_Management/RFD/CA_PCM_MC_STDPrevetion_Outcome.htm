Outcome

  1. Click Outcome from the menu to record the outcome of the event STD Prevention.

  2. The mothercare pending tasks to be recorded for the event STD Prevention is displayed if any. Click [Record] to continue recording the outcome or click [Cancel] to perform the pending tasks for the event. On clicking [Record], the screen to record outcome gets displayed.

  3. The Outcome list box displays Close as the outcome for the event STD Prevention. Select Close as Outcome, the event STD Prevention is closed but the current event for which STD Prevention was recorded is still active. The status shows the event STD Prevention to be closed but other events for the cycle are active.
