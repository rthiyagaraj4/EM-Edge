Enter the following details to create a new Section record:

 

  1. Code - Enter the Section code in alphanumeric format of four characters maximum length. The code must be unique and cannot be changed after creation. This field is mandatory.

  2. Description - Enter the description for the Section in alphanumeric format of 20 characters maximum length. This field is mandatory.

  3. Title – Enter the Title caption in alphanumeric format of 20 characters maximum length.

  4. Type - Select the type of Section from list of valid values. This field is mandatory. The values are 'Free Format', 'Result-Linked', 'Main Section' and 'Template-Based'. This field is mandatory.

  5. Template Type - On selecting the Type as 'Template Based, the Template Type field is enabled. Select the Template Type from the list. The listed vales are 'Note Header', Note Footer', 'Note Body' and 'Report Header'.

  6. Selection Type - This field is enabled if the 'Type' is selected as Result Linked. Select the Selection Type from the list. The listed values are Single and Multiple.

  7. Display Order - This field is enabled if the Selection Type is selected as Multiple. Select the Display Order from the list. The listed values are By Date and By Category.

  8. Result Category - On selecting the Type as 'Result Linked' and Selection Type as 'Single', the Result Category list box is enabled. Select the required value from the list. The listed values are Adverse Event, Blood Transfusion, Clinical Note, Dept Diagnostics, Dietary, Free Documentation, Future Events, Laboratory, Medication, Monitoring, Newborn, Patient Care, Problem/Diagnosis, Radiology/Imaging, Surgery/Procedure and Treatment. This is a mandatory field. If the Selection Type is selected as 'Multiple', [M] is enabled. Click [M] to define Result Categories after recording the Section or while querying for the Section. Based on the Result Category selected, the Configure Display Fields are displayed at the bottom of the screen.

  9. Configure Output - This check box is enabled on selecting Type as Template Based. Select the check box to configure section template output for the Template Based Section.

  10. Enabled - Select the check box to enable the record and clear the check box to disable the record.

 

Characteristics

  1. Image Linked – Select the check box to link the images to the section in the transactions. This field is enabled only if Type is selected as Free Format or Result Linked.

 

Building Notes

  1. Preserve Template Format - Select the check box to maintain the section template format as it is while using in the transactions. This field is enabled if 'Section Type' is selected as Template Based and the 'Template Type' is Note Body.

  2. Include only the filled Components - Select the check box to include only filled components as part of the Section Template. This field is enabled if 'Section Type' is selected as Template Based and the 'Template Type' is Note Body. This check box is selected, by default.

 

Configure Display Fields

  1. Applicability - Click Applicable radio button if the Display Fields are applicable for the Section or click Not Applicable radio button.

  2. Order Sequence - The Order Sequence is displayed as hyperlink. Click the hyperlink to swap it to another order.

  3. Display Fields - Fields which can be configured and displayed in the Note for the selected Result Linked Section Type and Section are displayed. The display fields are based on the selected Result Category.

  4. Select - Select the check box against the required display fields to display them in the Clinical Note on selection of the result. Display fields that are mandatory are selected and disabled, by default.

  5. Future Event Type - If the Result Category is selected as Future Events, select the Future Event Type as Future Orders or Referral from the list that displays. The Display Fields are displayed based on the selected Future Event Type.
