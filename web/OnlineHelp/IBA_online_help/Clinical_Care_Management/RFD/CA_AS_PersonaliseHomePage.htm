Enter the following details to Personalize Home Page:

 

  1. Practitioner Type – Select the required Practitioner Type for which the Home Page is to be customized. This field is mandatory.

  2. Practitioner – Enter the required Practitioner name for whom the Home Page has to be personalized in alphanumeric format or click [?] function and select the required Practitioner.

  3. Desktop - Enter the Desktop Name or select the Desktop using [?] function. The User Desktop created in the Create User Desktop function with the User Type 'Clinician' in System Manager module are displayed. This is a mandatory field.
