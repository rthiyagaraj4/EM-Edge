Enter the following details:

 

  1. History Type – Select the required History Type from the list. This field is mandatory.

  2. Control System – Based on the History Type selected, the values in the Control System list differ. Select a value from the Control system list. This field is mandatory.

  3. Data Set – Select the radio button ‘All’ or ‘Selected’ data set.

The search criteria appear with a top frame that displays all the selected items in the grid. Use the search criteria to add any other item to the selected list.

  1. Search By – Select either Code or Description by selecting the required radio button to search for the Event based on Code or Description.

  2. Search Criteria – Select the Search Criteria as Starts With, Ends With or Contains to search for the required Event.

  3. Search Text – Enter the required text to search for the event in alphanumeric format of 70 characters maximum length. This field is mandatory.

Click [Search] to select the ‘Events’. Select the corresponding check boxes against an Event under the Select column. Select the check box to add and clear the check box to remove an Event. From the selected items list, you can view the added event and the event being removed.
