Outcome

  1. Click Outcome from the menu to record the outcome of the Pregnancy event.

  2. The mothercare pending tasks to be recorded for the event Pregnancy is displayed if any. Click [Record] to continue recording the outcome or click [Cancel] to perform the pending tasks for the event. On clicking [Record], the screen to record outcome gets displayed.

  3. The Outcome list box displays Delivery, Ectopic Pregnancy, Preterm Delivery and Still Birth as the outcomes for the event Pregnancy.

  4. Select Delivery as outcome and a message is displayed indicating the outcome will close the current event and create a new event of recording delivery details for the patient. Click [OK] to proceed. Click [Cancel] to cancel closing the event.

  5. Select Ectopic Pregnancy as outcome and a message is displayed indicating the outcome will close the current event and create a new event of recording Ectopic Pregnancy details for the patient. Click [OK] to proceed. Click [Cancel] to cancel closing the event.

  6. Select Preterm Delivery as outcome, a message is displayed indicating the outcome will close the current event and create a new event of recording preterm delivery details for the patient. Click [OK] to proceed. Click [Cancel] to cancel closing the event.

  7. Select Still Birth as outcome and a message is displayed indicating the outcome will close the current event and create a new event of recording preterm delivery details for the patient. Click [OK] to proceed. Click [Cancel] to cancel closing the event.
