Enter the following details:

 

  1. Maximum Number of Consultation by Same Practitioner - Enter the maximum number of Consultations allowed by the same practitioner.

  2. Maximum Number of Consultation by Specialty - Enter the maximum number of Consultations allowed by a Specialty.

  3. Minimum Duration Required Between Consultations - Enter the minimum duration in minutes required between two consultations. Based on the value entered for this field, the patient should be listed in the IP Consultation - New List during the next consultation. The patient should not be listed in the IP Consultation - New List if the minimum duration between consultations is less than the value entered.   

  4. Back Date Consultation List - Enter the number of days for which the Consultation List can be backdated.

  5. Amend Consultation List - Enter the number of hours for which a temporary Consultation List can be amended. A new list is created only if the current list which is temporarily saved is closed.

  6. Maximum No. of Consultation Chargeable for a given IP encounter is introduced to display the maximum number of consultations that are chargeable per day for an IP encounter. This field is disabled. The total number of Regular Consultations and Referral Consultations entered are displayed.

  7. Regular Consultation - Enter the number of regular consultations that are chargeable per day for an IP encounter.

  8. Referral Consultation - Enter the number of referral consultations that are chargeable per day for an IP encounter.
