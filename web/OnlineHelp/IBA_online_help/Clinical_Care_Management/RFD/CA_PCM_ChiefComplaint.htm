Enter the following details:

 

  1. Complaint – Select the complaint from [?] function. The complaints that appear are defined in the Codes Setup. This field is mandatory.

  2. Complaint Details – The complaint details are displayed. The Complaint Details can be modified.

  3. Onset - Select the onset Date and Time of the Chief Complaint using the Calendar icon. The current system date and time is displayed by default.

  4. Remarks - Enter the Remarks, if any.
