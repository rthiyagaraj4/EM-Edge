Enter the following details to create a Chart Applicability record:

 

  1. Patient Class – Select the Patient Class from the list. The values are All, Daycare, Emergency, External, Inpatient and Outpatient.

  2. Specialty Desc – Enter the required Specialty Description in alphanumeric format or select the Specialty using [?] function.

  3. Practitioner Type – Enter the required Practitioner Type in alphanumeric format or select the Practitioner Type using [?] function.

  4. Location Type – Select the Location Type to be associated to the chart from the list. The values are All, Clinic, and Nursing Unit. By default, All is displayed.

  5. Location - Enter the required Location in alphanumeric format or select the location using [?] function. On selecting the Location Type as Clinic, all the clinics are listed. On selecting the Location Type as Nursing Unit, all nursing units are listed for selection.

  6. Age Group - Enter the required Age Group in alphanumeric format or select using the Age Group from [?] function.

  7. Chart – Enter the required Chart in alphanumeric format or select the Chart using [?] function. All the Charts defined in the Chart Setup function are displayed. This field is mandatory.
