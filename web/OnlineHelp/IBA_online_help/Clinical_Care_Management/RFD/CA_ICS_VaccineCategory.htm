Enter the following details:

 

  1. ID – Enter the Vaccine Category ID in alphanumeric format of 10 characters maximum length.

  2. Long Description – Enter the long description of the Vaccine Category in alphanumeric format of 60 characters maximum length.

  3. Short Description – Enter the short description of the Vaccine Category in alphanumeric format of 40 characters maximum length.

  4. Age Group - Select the Age group from the list. The listed values are defined in the Age Group function of Application Masters module.

  5. Allow Multiple – Select the check box if multiple vaccines can be scheduled at a time. This check box is cleared, by default.

  6. Adhoc -  Select the check box, if the defined Vaccine Category can be used as Adhoc without any schedule. If this check box is cleared, the Vaccine Category is valid only for the specified criteria. This check box is cleared, by default.

  7. Enabled – Select the check box to enable the defined Vaccine Category. The check box is selected, by default. Only enabled Vaccine Categories are available for selection in the required functions.
