Enter the following details to enter Note Parameter:

 

  1. Backdate – Specify the number of days to be backdated in numerical format. This field is mandatory.

  2. XSL Folder - Specify the Location of the stored document by entering the path.

  3. File Size Limit - Enter the Size Limit of the file in numeric format and select the size from the list of values as KB or MB.

  4. Print Discharge Summary Header – Select the check box to enable printing the header of the discharge summary.

  5. Max no of Images allowed per note - Enter the number of images that can be linked to the clinical note in numeric format of two digits maximum length. You can associate multiple images when recording clinical notes based on the limit specified.
