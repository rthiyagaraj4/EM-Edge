Enter the following details:

 

  1. Patient Class – Select the required Patient Class from the list. The listed values are All, Day Care, Emergency, External, Inpatient and Outpatient. By default, the value ‘All’ is selected.

  2. Specialty – Enter the required Specialty in alphanumeric format or click [?] function to search and select the required Specialty.

  3. Practitioner – Enter the required Practitioner name in alphanumeric format or click [?] function to search and select for the required Practitioner.

  4. Summary – Select the required Chart Summary Template from the list. This field is mandatory. The selected Chart Summary Template is displayed at the bottom of the screen.
