Enter the following details to create a new Note Disclaimer record:

 

  1. ID - Enter the Disclaimer ID in alphanumeric format of 10 characters maximum length to identify a Note Disclaimer uniquely. The Note Disclaimer ID cannot be changed after creation. This field is mandatory.

  2. Description - Enter the Description for the Note Disclaimer in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Disclaimer Text – Enter the text for Disclaimer. The Disclaimer text often has a note stating to abstain from using the note or text for any other purpose. For example, the Discharge Summary is not a Medical Report for the Patient and cannot be used in court.

  4. Enabled – Select the check box to enable the record and clear the checkbox to disable the record.

  5. Applicable to Report - Select the check box to make the Disclaimer available in the 'Email Report Control Setup' function.
