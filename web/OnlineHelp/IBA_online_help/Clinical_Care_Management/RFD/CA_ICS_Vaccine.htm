Enter the following details:

 

  1. ID – Enter the ID in alphanumeric format of 10 characters maximum length.

  2. Long Description – Enter the long description of the Vaccine in alphanumeric format of 60 characters maximum length.

  3. Short Description – Enter the short description of the Vaccine in alphanumeric format of 40 characters maximum length.

  4. Can Be Used As Adhoc? - Select the check box, if the defined Vaccine can be used as Adhoc without scheduling. For Eg: Tetanus. This check box is cleared, by default.

  5. Duplicate Check Required? – Select the check box, if the duplicate check is required while administering the Vaccine in Manage Immunization function. The check box is cleared, by default.

  6. Duplicate Check Days – Enter the range in days during which the same Vaccine cannot be administered twice

  7. Duplicate Check Action – Select the Duplicate Check Action value from the list. The listed values are ‘Warn’ and ‘Error’. The value ‘Warn’ is displayed, by default. If the value ‘Warn’ is selected, a warning message is displayed during the Duplicate Check Action. If the value ‘Error’ is selected, an error message is displayed during the Duplicate Check Action.

  8. Enabled – Select the check box to enable the defined Vaccine record. The check box is selected, by default. Only enabled Vaccine records are available for selection in the required functions.
