Enter the following details:

 

  1. Patient ID – The Patient ID of the selected patient is displayed.

  2. Appt No - Enter the Appointment Reference Number in alphanumeric format of eight characters maximum length.

  3. Appt Date From/To - Enter the date range for which the appointments are scheduled in dd/mm/yyyy format or click Calendar function to search for the required date.

  4. Specialty – Select the Specialty from the list.

  5. Location Type – Select the desired value from the Location Type list. The valid values are Daycare Unit, Clinic, and Procedure Unit.

  6. Location – Select the required location, where the patient is scheduled for an appointment from [?] function.

  7. Resource Type – Select the desired Resource Type from the list. The valid values are Practitioner, Equipment, Room and Other.

  8. Resource - Select the required resource against which the Patient is scheduled for an appointment.
