Enter the following details to associate Sections to a Note Type:

 

  1. Note Type – Click [?] function to select the required Note Type. All the Note Types created in 'Note Type' function with status as enabled are displayed. This field is mandatory.

On selecting the required section, click [Search].

 

Available Sections

  1. Section Starts With – Enter the required section starting description in alphanumeric format of 20 characters and click [Search] to search for the required section. By default, the entire sections as defined in the Section function are listed.

  2. Section Description – The description of the section is displayed.

  3. Select - Select the required ‘Section’ to be associated with the ‘Note Type’ by selecting the check box corresponding to the section.

 

Selected Sub Sections

  1. Sequence No – The sequence number of the section is displayed and made hyperlink.

  2. Section Description – The code of the section is displayed.

  3. Chief Complaint - This field is enabled on selecting a Section of Type ‘Free Format’ and ‘Result Linked’, or remains cleared and disabled.

  4. Select – The check box appears selected by default. Clear the check box to exclude a section from the Note Type.
