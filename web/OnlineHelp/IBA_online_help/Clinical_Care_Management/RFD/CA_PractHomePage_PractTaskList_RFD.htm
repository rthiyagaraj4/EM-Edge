Enter the following details:

  1. Location - ‘All’ is displayed, by default. Select the required Location Type from the list. The values displayed are All, Clinic, Day care, Emergency, Nursing Unit and Referral. On selecting the Location Type,  based on the value selected for Location Type, the [?] function displays Locations. If Location Type is selected as ’Clinic’, [?] function displays all the clinics of the logged in facility to which the logged in user has access. If the Location Type is ‘Nursing Unit’, [?] function displays all the Nursing Units of the logged in facility to which the logged in user has access.

  2. Period [From-To] - The date range is displayed, by default. Enter a period in date format of dd/mm/yyyy. This is a mandatory field.

  3. Patient ID – Enter the Patient ID in alphanumeric format of maximum length as specified in the MP Parameter.

  4. Filter – Select the values from the list to filter the orders. The listed values are My Patients, Ordered Patients, My related Patients and All. All is selected, by default.

  5. My Patient – Select ‘My Patient’ from the list to displays all the pending appointments for the attending practitioner.

  6. Ordered Patients – Select ‘Ordered Patient’ from the list to display all the pending appointments for the ordering practitioner.

  7. My Related Patients – Select ‘My Related Patient’ from the list to display all the pending schedules for the patient related to the logged in practitioner.

  8. All – Select ‘All’ from the list to display all the list of patients with pending appointments.

 

Click [Search] to search for the orders or [Clear] to clear the entered details.
