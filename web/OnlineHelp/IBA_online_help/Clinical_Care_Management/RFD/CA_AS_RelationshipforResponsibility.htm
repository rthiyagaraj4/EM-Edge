Enter the following details to create a new Relationship for Responsibility record:

 

  1. Responsibility – Click [?] function to select a Responsibility. All the Responsibilities defined in ‘System Manager’ module is displayed. This is a mandatory field.

  2. Relationship – Click [?] function to select a Relationship. All the Relationship records created in Relationship function with status as enabled are listed. This field is mandatory.

  3. Can Hold – The Can Hold check box indicates that any practitioner assigned to the responsibility selected can hold the selected relationship. Select the check box to hold the relationship.

  4. Can Assign – The Can Assign check box indicates that the practitioner logging in to Clinician Access using the selected responsibility can assign the selected relationship to other practitioners using the function ‘Assign Relationship’ in Clinician Access module. Select the check box to assign the relationship.

  5. Can Terminate – The Can Terminate check box indicates that the practitioner logging in to Clinician Access using the selected responsibility can terminate the selected relationship that has been assigned to other practitioners using the function ‘Assign Relationship’ in Clinical Care Management module. A Practitioner can terminate a relationship, if the relationship is assigned to the practitioner for whom the relationship is being terminated.
