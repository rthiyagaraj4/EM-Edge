Delivery Record

  1. Delivery Type - Select the Delivery Type from the list.  The Delivery Types are defined in the Master Patient Index module with Delivery indicators - Normal, C Section and Others.

  2. LMP - Select the LMP using the calendar function. This is validated based gestation period recorded on the MP Parameter setup. This is a mandatory field.

  3. Attending Practitioner - Enter name of the Attending Practitioner or select using [?] function.

  4. Attending Nurse - Enter name of the Attending Nurse or select using [?] function.

  5. Procedure Classification - Enter the Procedure Classification or select using [?] function. The Procedure Classifications are defined in the Patient Identity Management module.

  6. Delivery Surgical Classification - Enter the Delivery Surgical Classification or select using [?] function. The Delivery Surgical Classifications are defined in the Master Patient Index module.

  7. Vaginal Delivery Type - Enter the Vaginal Delivery Type or select using [?] function. This field is disabled if 'Delivery Type' is selected as a record with Delivery Indicator - 'Normal'.

  8. No. of Birth - Enter the number of births.

  9. Complication in Stage 1, 2 and 3 of Delivery - Select the Complication of Stage 1, 2 and 3 of Delivery from the list. The Complications of Delivery are created in the Complications of Delivery function with 'Type of Indicator' field selected as Mother for each of the complication stages in the Patient Identity Management module. If 'Delivery Type' is selected as a record with Delivery Indicator - 'C Section', these fields are disabled.  

  10. Complication of Delivery - Enter the Complication of Delivery or select using [?] function. The Complications of Delivery are created in the Complications of Delivery function in the Master Patient Index module. This field is disabled if Delivery Type is selected as a record with Delivery Indicator - 'Normal'.

  11. Foetus Complication of Delivery - Enter the Foetus Complication of Delivery or select using [?] function. The Foetus Complications of Delivery records are created in the Complications of Delivery function with 'Type of Indicator' field selected as Fetus in the Master Patient Index module.

  12. Finalize - Select the check box to Finalize the delivery record. Selecting the check box makes the No. of Birth field mandatory.

Delivery Stage 1 Summary

Delivery

  1. Delivery Suite Admission Time - Select the date and time of admission into the delivery suite using the Calendar function.

  2. Date and Time of Labour Pain - Select the date and time of labour pain using the calendar function.

  3. Enema Given - Enter the date and time of when enema was given to the patient. An enema is the procedure of introducing liquids into the rectum and colon via the anus.

  4. Date and Time of Bloody Flux - Select the date and time of bloody flux using the Calendar function. Dysentery (formerly known as flux or the bloody flux) is frequent, small-volume, severe diarrhea that shows blood in the feces along with intestinal cramping and tenesmus (painful straining to pass stool).

  5. Enema Given - Enter the date and time of when enema was given to the patient. An enema is the procedure of introducing liquids into the rectum and colon via the anus.

  6. Character of Bloody Flux - Enter the character of bloody flux. Example: Mucous, Bloody.

  7. Character of Aminiotic Fluid - Select the Character of amniotic fluid from the list. The listed values are Clear, Thin-meconium stained and Thick-meconium stained.

  1. Stage 1 Duration - Enter the first stage duration of labour in minutes.

  2. Foetal Heart Rate – Select the Foetal Heart Rate from the list. The listed values are Late Deceleration, Normal, Prolonged bradycardia, Variable deceleration and Tachycardia.

  3. Labour Analgesia – Select the Labour Analgesia (pain relief given during labour) from the list. The listed values are Pharmacologic and Non Pharmacologic.

Labour Opiate Type

Select the Labour Opiate Type from the following options Morphine, Pethidine, Other and No. Enter any other delivery presentation in the Specify text area, if ‘Others’ check box is selected.

Regional block for labour pain control

Select the Regional block for labour pain control from the following. Epidural and Spinal. Enter any other delivery presentation in the Specify text area, if ‘Others’ check box is selected.

  1. Labour Type - Select the Type of Labour from the list. The values are defined in the Specialty Master Code function of CA Specialty Code Setup. Example: Induced, Spontaneous.

  2. Induction Started - Select the date and time of induction of labour from the calendar function.

  3. Induction Indicators - Select Induction Indicators from the list. The values are defined in the Specialty Master Code function of CA Specialty Code Setup. Example: Prolonged 1st stage labour, Premature rupture of membrane.

  4. Induction Procedure – Select the induction procedure from the list.

  5. 1st Stage Mother Complication - Select the 1st stage mother complication from the list. The values are defined in the Complication of Delivery function in Master Patient Index module.

  6. 1st Stage Fetus Complication - Select the 1st stage fetus complication from the list. The values are defined in the Complication of Delivery function in Master Patient Index module.

Click [Save] to save the record and the record can be modified. Click [Finalize] to save the record permanently and the record cannot be modified. On finalizing the record the date and time of recording will be displayed.

Labour and Delivery

  1. Select Delivery Record from the Labour and Delivery task to record labour and delivery details of the event Delivery. Click [Record] to save the Delivery Record. The Delivery Type field is disabled on selecting the Delivery Record hyperlink from the left side of the screen if it is not yet finalized and other details can be modified. If the record is Finalised, you cannot modify any of the recorded details.

  2. If the Delivery Type is selected as a record with Delivery Indicator as 'C Section', you get the option to Record Procedure.

  3. If the Delivery Type is selected with Delivery Indicator as ‘Normal’ and more than one new born details have to be recorded, Delivery Stage 1 is common to newborns and for each newborn Delivery Stage 2 and Delivery Stage 3 are to be recorded.

Delivery Stage 1

  1. Click Register Baby to register the baby. Enter details is mandatory field and click [Register] to register the baby. The record can be finalized by selecting the Newborn ID hyperlink. Enter mandatory details and click [Finalize] to finalize the record. The newborn ID is displayed on the left side of the screen. If the record is not finalized, text is displayed as (Not Finalized). Select the newborn ID to view details.
