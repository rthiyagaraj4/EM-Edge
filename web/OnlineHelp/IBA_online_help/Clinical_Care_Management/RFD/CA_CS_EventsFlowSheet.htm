Enter the following details:

 

  1. Flow Sheet – Select the required Flow Sheet from the list. All the enabled Flow Sheet components created in the function ‘Flow Sheet Component’ function is displayed. This field is mandatory.

  2. History Type – Select the required History Type from the list. This field is mandatory.

  3. Search By – Select either Code or Description by clicking the required radio button to search for the event based on Code or Description.

  4. Search Criteria – Select the Search Criteria as Starts With, Ends With or Contains to search for the required Event.

  5. Search Text – Enter the required text to search for the event in alphanumeric format of 70 characters maximum length.

On selecting, the required details click [Search] and the following details are displayed

  1. Event Code – The Event Code corresponding to the selected History Type is displayed.

  2. Event Description – The Description of the event is displayed.

  3. Select - Select the check box to select the event to be associated with the selected flow sheet. The selected Event is displayed under the ‘Selected Events’ header. The event selected for inclusion under the 'Selected Events' section can be excluded by clicking ‘Select’ check box.
