Enter the following criteria to filter New Born listing:

  1. Healthy New Borns – Select the check box to display all the healthy newborns born in the hospital. The check box is selected, by default.

  2. Admitted New Borns – Select the check box to display all admitted newborns currently admitted in the hospital. The check box is selected, by default.

  3. Stillborn/Deceased – Select the check box to display stillborn or deceased newborns in the facility. The check box is selected, by default.

  4. Newborn ID – Enter the Newborn ID or select the Newborn ID using [?] function. The value in this field gets populated from the newborn registration function of the Master Patient Index module.

  5. Mothers ID – Enter the Mothers ID or select the Mothers ID using [?] function.

  6. Newborn Encounter ID – Enter the Newborn Encounter ID. The Encounter ID is the ID given for each encounter of the newborn in the facility.

  7. Nursing Unit - Select the required Nursing Unit using [?] function.

Patient List by New Born List

From this function you can View the New born list.

This function provides the listing of newborn individuals in the hospital from different inpatient locations associated to the mother’s Patient ID.

Viewing New Born List

From the CA Transactions > Clinician Access > Desktop Menu > Patient List, select By New Born List.

The New Born List screen gets displayed.

  1. Enter the search criteria.

  2. Click [Search] to display newborns based on the entered criteria.

  3. Place the mouse over the Newborn Name hyperlink to view the Mother details of the newborn.

  4. A ‘*’ symbol is displayed against the newborn record, if relationship between the practitioner and newborn does not exist. Click Newborn Name hyperlink to display the Assign Relationship screen and assign relationship.

  5. For newborns with relationship assigned, Patient Chart is displayed.
