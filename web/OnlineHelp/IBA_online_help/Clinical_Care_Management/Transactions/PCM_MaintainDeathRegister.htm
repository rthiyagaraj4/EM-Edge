In case a sensitive diagnosis needs to be recorded, the logged in user has to be authorized to record such a diagnosis. If an unauthorized user tries to record a sensitive diagnosis, the system stops the user from recording the same.

To record a sensitive diagnosis, click [Authorized Mode], which invokes the Enter/Edit Sensitive Diagnosis screen.

Enter the password in alphanumeric format of eight characters maximum length to edit or enter the sensitive diagnosis. After entering the password, click [OK] to confirm or click [Cancel] to cancel the current job and the control shifts back to the Death Register screen. The [Authorize Mode] is displayed as [Normal Mode].

  1. Diagnosis Code - The code for the diagnosis recorded is displayed. This code is based on the coding scheme chosen in MR parameter.

  2. Description – The description of the diagnosis code as set up in the ICD master is displayed. If an ‘*’ is displayed against the description, this denotes that the diagnosis has been recorded as a primary diagnosis.

  3. Cause of Death – This field appears selected indicating that the diagnosis is identified as the reason for death of the patient. Clear the check box to indicate that the diagnosis recorded is not the cause of death of the patient.

  4. Add to Problem List – Select the check box if the selected diagnosis has to be included in the medical record of the patient.

Maintain Death Register

From this function you can Maintain the death register of a deceased patient.

This function is used to record the cause of death in the eventuality of the patient expiry occurring in the hospital during an active encounter. The function is applicable only after a patient’s encounter is closed with the Disposition Type which has a deceased Indicator.

Maintaining Death Register

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Outcomes, select Maintain Death Register.

The Death Register screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Authorize Mode] to record any sensitive diagnosis.

  3. Click [Add] to add the diagnosis selected to the Diagnosis Details section at the bottom of the screen. The newly added diagnosis is hyperlinked. This part of the screen may already contain diagnosis that has been recorded either during discharge or during an encounter of the patient.

  4. Click [Record] to record save the details entered or [Cancel] to exit from the screen. On clicking [Record], a message to finalize the recording is displayed. Click [OK] to finalize the death details or [Cancel] save the death details without finalizing the record.
