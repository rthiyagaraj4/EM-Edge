Procedure List (V)

From this function you can View the procedures performed for a patient.

This function helps in viewing any previous history of surgeries or procedures performed. The pervious surgery or procedure details helps care providers in getting more information for further patient management.

Viewing Procedure List

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Event Details, select Procedure List (V).

The View Procedure List screen gets displayed.

  1. Click [Search] to view all procedures recorded for the patient or enter the search criteria to view specific records. Click [Clear] to clear the entered data.

  2. On searching, the procedures are displayed based on the search criteria. Select the Code hyperlink to view the procedure details as recorded in the Procedures function for the patient.
