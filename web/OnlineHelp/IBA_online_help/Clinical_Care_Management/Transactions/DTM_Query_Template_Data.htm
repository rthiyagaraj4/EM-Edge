Query Template Data

From this function you can Analyse data entered using Clinical Notes for the selected transaction time period.

This function generates patient list as per the entered criteria for the Clinical Notes data for the selected transaction time period.

Query Template Data

From the CA Transactions > Clinician Access > Desktop Menu > Query Template Data > Select Query Template Data function.

The Query Template Data screen gets displayed.

  1. Enter the Query Description or selecting using [?] function. The Query Description defined in Note Template Query List function are displayed.

  2. The Criteria is displayed, by default.

  3. Enter the Transaction period or select the Transaction Period using the Calendar function. The current system date is displayed, by default. This is a mandatory field.

  4. Select the Gender form the list. The listed values are All, Female and Male.

  5. Enter the Age Range between 0 to 150 and select Days, Months or Years from the list.

  6. The component details selected for the query description are displayed.   

  7. Click [Search] to displayed the results matching with entered criteria or click [Clear] to clear the entered data.

  8. The details of patients matching the entered criteria are displayed and the total patient count is displayed.

  9. Click [Print] to print the details displayed or click [Export to Excel] to export the details to excel sheet.

  10. Click [Criteria] to re-enter the criteria.

  11. Click [Apply] to request for multiple transcriptions.
