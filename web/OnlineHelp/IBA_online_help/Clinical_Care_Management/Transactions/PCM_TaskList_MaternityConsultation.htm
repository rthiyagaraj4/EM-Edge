Maternity Consultation

From this function, you can Maintain care details of patients related to maternity.

The Mothercare module is invoked from the Clinician Access module. Patients have to be registered to a location catering to mothercare specialty. A Practitioner associated to a location of the mothercare specialty can access this function to record Maternity Consultation details by logging into the Clinician Access module. This method of using the Clinician Access module to invoke the Maternity Consultation function is adopted to record the clinical notes and charts for the mother patient encounter.

Recording Maternity Consultation

From the CA Transactions > Clinician Access, Desktop Menu, select required patient > Patient Chart Menu > Charting, select Maternity Consultation function.

The Maternity Consultation screen gets displayed.

  1. Select an Event from the 'Open Event' list. All Specialty Events as defined in the Specialty Event function of CA Specialty Code Setup are defined. The list displays the following Events.

Birth Control

Delivery

Ectopic Pregnancy

Fertility Issues

Menopause

Post Delivery Mother Treatment

Pregnancy

Preterm Delivery

Reproductive Cancers

STD Prevention

Termination of Pregnancy

  1. On selecting an event, various tasks can be recorded for the event in a cycle. There is no specific order in recording events. The relevant event applicable to the patient can be selected for recording. Recording outcomes for the event puts and end to an event or triggers another event for recording. If an event is closed, a new cycle can be created for recording events.

  2. Select the required Event and click [Go].
