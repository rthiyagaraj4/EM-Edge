Schedule Admission

From this function you can Schedule an admission for patients.

This function is used to create booking for patients seeking admission to an inpatient location or nursing unit. The function allows booking to be performed for registered patients of the facility and for referral patients.

Scheduling Admission

From the CA Transactions > Clinician Access > Desktop Menu > Schedules, select Schedule Admission.

The Create Booking screen gets displayed in the Inpatient Booking screen.

  1. Enter the details in mandatory (*) fields. <Related Field Description>

  2. Click [Apply] to save the booking or [Reset] clear the entered data.
