Practitioner for Consultation Orderable

From this function, you can Create an association of a consultation orderable to a Practitioner.

The association is performed by selecting an Order Catalog and associating a Practitioner or selecting a Practitioner and associating an Order Catalog. Once the practitioner is associated to the orderable, the check box is disabled. Only one orderable is associated to a particular practitioner.

Creating an association of Practitioner to Consultation Orderable

From the CA Code Setup > Consultation Setup, select Practitioner for Consultation Orderable function.

The Practitioner for Consultation Orderable screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select Group By as Order Category or Practitioner. Order Category is displayed, by default.

  3. Select the Order Catalog using the [?] function. Order Catalogs created with Order Category Consult Order are displayed for selection. This is a mandatory field.

  4. Select the required alphabet hyperlink under the Select Tab to display Practitioners beginning with the selected alphabet.

  5. Select the check box against the required Practitioner to associate the Practitioner to the selected Order Catalog.

  6. Click the Associate Tab to view the Practitioners associated to the Order Catalog.

  7. Click [Apply] to save the association of Practitioner to Consultation Order Catalog. The Practitioner selected for an Order Catalog is disabled and not allowed for association to another Order Catalog. Click the Associate Tab to view the specialties associated to the Order Catalog. Clear the check box against the required Practitioner to dissociate the Practitioner from the Order Catalog. Click [Apply] to save the dissociation of Practitioner from the Consultation Order Catalog. The Practitioner is then available for association to any other Order Catalog.

  8. If Group By is selected as Practitioner, select the Practitioner using the [?] function. Selecting the Practitioner is mandatory.

  9. Select the Order Catalog using the [?] function. Order Catalogs created with Order Category Consult Order are displayed for selection. This is a mandatory field.

  10. Click [Apply] to save the association of Practitioner to Consultation Order Catalog.
