Identify Main/Duplicate Patients

From this function you can View the duplicate records extracted by the ‘Search Duplicates’ function and Identify Main/Duplicate records of the required groups.

The access to this function is determined by the User Access Rights set for the logged-in User. You cannot invoke this function, if the rights are not defined for the User or the right for the DR process ‘Identify Main/Duplicate Patients’ is set as ‘None’ for the logged in User.

If the privilege level is set as Logged-in facility, you can perform the DR Process of ‘Identify Main/Duplicate Patients’ for the Logged-in facility. If the privilege level is set as All Facilities, you can access the function and can perform ‘Identify Main/Duplicate Patients’ DR Process for all the facilities.

Identifying Main and Duplicate Patients

From the DR Processes, select the Identify Main/Duplicate Patients function.

The Identify Main/Duplicate Patients screen gets displayed.

  1. The Duplicate Groups are displayed at the left hand side of the screen.

  2. Click on the desired Patient ID hyperlink.

Or

  1. Enter the Patient ID and click [Search] to view the duplicate patient details in Group Details frame.

  2. Click [Search Criteria] to view the defined search criteria and [Close] to exit from the Search Criteria screen.

  3. Select at least two patients under the view heading and click [Duplicate Details] to view Possible Duplicate Details. Click Patient ID hyperlink to view the patient details or click [Close] to exit from the Possible Duplicate Details screen.

  4. Click 'E' hyperlink to view previous encounter details of the patient. Click Encounter ID hyperlink to view the encounter details of the selected patient.

  5. Click Patient ID hyperlink to view Patient details. The details entered while registering a patient is displayed.

  6. Identify the patients as Main, Duplicate, Do Not Consider or Exclude. Select the desired option to identity the records.

    Exclude - To exclude the record from merging, select the 'Exclude' check box.

    Do Not Consider - You can discard other Patient IDs from consideration as either Main or Duplicate, by selecting the ‘Do Not Consider’ option against Patient ID(s) displayed. These records are not considered for Merging.

    Duplicate - To select Patient IDs as duplicates select the option ‘Duplicate’ against the Patient ID(s).

    Main - You can identify the Main patient among the displayed records by selecting the ‘Main’ option against the Patient ID. Only one Patient ID can be marked as Main.

  7. Click [Apply] to save the transaction or [Reset] to clear the selected option.

  8. Select the desired record from the view heading to delete the record. Click [Delete] on selecting patient record(s) to delete the record. A pop up message is displayed confirming the deletion. Click [OK] to confirm the deletion or [Cancel] to cancel the deletion.

 

The Diagnosis details screens can be invoked if CA module is operational.
