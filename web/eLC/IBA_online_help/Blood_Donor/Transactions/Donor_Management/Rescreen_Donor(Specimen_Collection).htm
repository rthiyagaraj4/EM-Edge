Rescreen Donor (Specimen Collection)

From this function you can Rescreen the deferred donors' for specimen collection. immunization for a donor.

The Rescreen Donor (Specimen Collection) function enables you to register a deferred donor for rescreening. You can collect specimens for test once the donor is registered through this function. The details that are recorded while deferring the donor are displayed. A confirmation message is displayed when the donor is registered.

Rescreening Donor

From Transactions > Blood Donor, double-click the Rescreen Donor (Specimen Collection) function.

The Rescreen Donor (Specimen Collection) screen gets displayed.

  1. Enter the Donor ID or click [List of Values] to select a Donor ID. The name of the Donor, age and gender are displayed in respective fields.

An error message 'Donor is not Deferred' is displayed if the selected donor is not deferred.

If the specimen is already collected for the selected Donor, a warning message 'Specimen Collected but not sent to Laboratory for Screening' is displayed. Click [Recall] to update the details and register the donor for rescreening or click [New Visit] to create a new visit registration for the donor specimen collection.

  1. The deferral type, unit number and the deferred till date are displayed based on the values recorded during donor deferment.

  2. Click [Antibodies] to view the antibodies that are recorded for the respective donor. This button is enabled only if the details are recorded.

  3. Click [Antigens] to view the antigens that are recorded for the respective donor. This button is enabled only if the details are recorded.

  4. Click [Donor Notes] to view the notes that are recorded for the respective donor. This button is enabled only if the details are recorded.

  5. The name of the logged in user is displayed in 'Collection Details By' field, by default. Click [List of Values] to select any other user.

  6. System Date is displayed as the default collection date. Click the Calendar function to select any other date. The selected date has to be greater than the 'Max. No. of Specimen Collection Behind Days for Rescreen' mentioned in the 'BD Parameters' function and lesser than the System Date.

  7. Click [Save] or press [F10] to save the immunization details, or click [Cancel] to cancel the transaction.

  8. A confirmation message 'Selected Donor Registered for Rescreening' is displayed.
