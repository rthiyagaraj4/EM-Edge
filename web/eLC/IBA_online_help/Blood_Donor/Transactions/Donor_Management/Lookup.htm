Lookup

From this function you can Search for a registered donor based on the demographic details of the donor. You can Register new donors.

The Lookup function enables you to search for a registered donor and to register new blood donors in the facility. Click [Search] to display all the donors or enter the search criteria and click [Search] to search for a specific donor. The statuses of the deceased, inactive and suspended donors are indicated using , and colors respectively. Click [New] to register a new donor.

Enter the search criteria such as Donor Name, Date or Birth, Nationality, etc., or click search without entering any criterion to search for the donor. The status of the donor is displayed in different colors beside the Donor ID. You can update the donor details, if they are not entered amply during registration.

Searching for a Donor

From Transactions > Blood Donor, double-click the Lookup function.

The Lookup screen gets displayed.

  1. Enter the details in the selected field to search for a specific record. You can search for the donor record using the tabs below:

General

Enter the following details to search for a specific donor record:

  1. Source Prefix & Patient ID – Enter a valid Source Prefix or Patient ID defined in Master Patient Index module.

  2. Donor ID – Enter a valid Donor ID of the donor registered in Donor Registration function.

  3. ID No – Enter the valid citizen ID of the Donor in alphanumeric format of 20 characters maximum length.

  4. Passport Number – Enter the Passport number of the patient in alphanumeric format of 30 characters maximum length.

  5. First Name – Enter the First Name of the donor in alphanumeric format of 20 characters maximum length.

  6. Family Name – Enter the Family Name of the donor in alphanumeric format of 30 characters maximum length.

  7. Alt Type/No. – Select the Alternate Type from the list. The Alternate ID type defined in MP Parameter function of the Master Patient Index module gets listed.

  8. Gender – Select the required Gender from the list. The list displays ‘Male’, ‘Female’ and ‘Unknown’.

  9. Birth Date – Click the Calendar function to enter the date of birth of the donor.

  10. Place of Birth – Enter a valid Place of Birth of a donor or click [List of Values] to select the value. The values defined using the Birth Place function in the Master Patient Index module get listed. The respective description of the place gets displayed.

  11. Nationality – Enter the Nationality code of a donor or click [List of Values] to select the nationality code.

  12. Race – Enter the Race code of the donor or click [List of Values] to select the race code.

  13. Order By – Select the value from the list to sort the search output. The valid values are ‘Donor Name’, ‘Sex’, ‘Date of Birth’ and ‘NRIC’.

Additional

Enter the following details to search for a specific donor record:

  1. Contact No. – Enter the Contact Number of the donor. The records matching the contact number gets displayed.

  2. Contact Mode – Select a Contact Mode from the list. The listed values are 'Contact Curr', 'Other Resi No' and 'Primary Resi No'.

  3. Deceased only – Select the check box to search for deceased donors. Clear the check box to search for the donor records who are not deceased.

  4. Suspended only – Select the check box to search for the suspended donors. Clear the check box to search for the donor records which are not suspended.

  5. Inactive only – Select the check box to search for inactive donors. Clear the check box to search for active donors.

  1. Click [Search] or [Execute Query] or press [F8] to search for a registered donor. <Related Field Description>

Click [Search] or [Execute Query] or press [F8] without specifying any of the search criteria to view all the existing donor records.

  1. Click [New] to register a new donor. The Donor Registration screen gets displayed. You can record either the details or right-click on the Donor Registration screen to search for a patient record.

  2. Click [Save] or press [F10] to register the new donor or click [Clear Record] to exit from the Donor Registration screen.
