Deferred Blood Donor

From this function you can Defer a donor temporarily or permanently, based on the lab result .You can Send the blood units of reactive donors for screening.

The Deferred Blood Donor function is used to defer a donor temporarily or permanently based on the lab results. An authorized user has to login with the password to defer a donor. You can search for the donor details or select reactive donors from the list to defer. You can access the donors, who are marked as reactive, send the unit once again for screening, validate and accept the lab results. Defer a donor permanently or temporarily based on the results. The tests that are deferred through the Deferred Donor (Rescreen) function are also listed.

Deferred Blood Donor

From Transactions > Blood Donor, double-click the Deferred Blood Donor function.

The Deferred Blood Donor screen gets displayed.

  1. Enter the password in the Password text box.

  2. Click [Validate] to confirm the password or [Cancel & Exit] to cancel the transaction and quit.

  3. Select a Donor from the list. The system-defined values are ‘All Donors’, 'Re-screening Donors' and ‘Reactive Donors’. You can enter the Donor ID or click [Donor Lookup] to select a donor registered in the facility when the value is selected as ‘All Donors’. On selecting ‘Reactive Donors’ from the list, Select the Donor details screen gets displayed listing the reactive donor details. Select 'Re-screening Donors' to select from the donor list. The mobile drive donors who are deferred through the Record Donation and inhouse deferred donors who are deferred through the Deferred Donor (Rescreen) functions are listed. On selecting a donor from the mobile drive donors' list, the unit number is displayed; System generates sample numbers and labels are printed for each sample. System will not print a label for in-house deferred donor.

    The following donor details are displayed.

    1. Donor ID – The Reactive Donor’s ID is displayed.

    2. Donor Name – The name of the Reactive Donor is displayed.

    3. Donation Date – The date of donation is displayed.

    4. Unit Number – The Unit number is displayed.

    5. [Close] – Click [Close] to close the window.

  4. Press [Tab] to view the donor details. <Related Field Description>

  5. Click [Antibodies] to view antibodies of the donor. This button gets enabled, only if any antibody details are recorded for the donor.

  6. Click [Antigens] to view antigens for the donor. This button gets enabled, only if antigen details are recorded for the donor.

  7. Click [Donor Notes] to add donor notes. This button gets enabled, only if donor notes are recorded for the donor.

  8. Click [List of Values] to select the Deferral Code. The values defined using the ‘Deferred Reasons’ function get listed. The Deferral Type associated to the deferral code is displayed.

  9. Click the Calendar function to specify the date during which the donor is deferred from blood donation. This is applicable only if the deferral type is temporary. Even before the Deferred till date is over, the user with appropriate rights can change the deferred till date. An authorized user with appropriate rights can update the date.

  10. The Screening tests are made default in Screening Tests tab as per the setup made in Screening Test Setup administration function. The details are displayed under the Screening Tests and the Screening Test Results tabs.

    On selecting the donor, the screened results are displayed in Screening Test Results tab, if the tests are conducted and results are released from the lab.

    1. Test Code – The Test code of the unit gets displayed.

    2. Numeric Result - The Numeric Result is displayed, if resulting type is of numerical type.

    3. Test Result – The Test Result recorded during investigation is displayed.

    4. [Comments] – Click [Comments] to view the result comments, if any.

    5. [Validate] - Click [Validate] to validate the test result.

    6. [Accept/Defer Donor] - Click [Accept/Defer Donor] to accept or suspend the donor permanently or temporarily.

    7. [View Lab Results] - Click [View Lab Results] to view the results.

    The following details are displayed under the Screening Tests tab:

    1. Test - The Test code defined in Screening Test Setup function is displayed.

    2. Description – The Description of the respective test code gets displayed.

    3. Send for Screening – Select the check box to send the unit to the laboratory for investigation. Clear the check box if investigation is not required.

    4. [Send for Screening] - Click [Send for Screening] to send the units to the lab for screening. The test units that are sent for screening is accessible through the 'Registration for Screen Units' function for registration. The registered test units are available for result validation. Based on the result, you can defer the donor permanently or unlock the deferral.

  11. A Unit Number is generated with a confirmation message indicating that the tests are sent to laboratory for screening and labels to be stuck on the collected samples are printed.

Perform the following transactions in the Laboratory; Registration for Screen Units, Acknowledge/Verify Specimens and Result Entry by Specimen. On completing the transactions in these transactions, you can view the results using this function.

The following colors indicate the status of the donor:

– Blood Typing Mismatch

– Rejected Tests

– Specimens rejected from Lab

-  Indicates that the donors are awaiting acceptance/deferment
