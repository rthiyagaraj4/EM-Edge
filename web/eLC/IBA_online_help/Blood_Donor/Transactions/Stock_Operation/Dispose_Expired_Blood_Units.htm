Dispose Expired Blood Units

From this function you can permanently Dispose the expired blood units from the existing blood unit stock. You can Query to view the Expired blood units.

The Dispose Expired Blood Units function allows you to dispose the expired blood units from the existing blood stock. You can search for the expired blood products based on the expiry date range. You can dispose a single unit or a list of units using this function.

Disposing the Expired Blood Units

From the Transactions > Stock Operation, double-click the Dispose Expired Blood Units function.

The Dispose Expired Blood Units screen gets displayed.

  1. Select the date range using the Calendar function to dispose the expired blood units. The current system date and time get displayed. This is a mandatory field.

  2. Enter a valid product code or click [List of Values] to select the product code. The values defined using the Products function get displayed.

  3. Enter a valid Fate Code or click [List of Values] to select the Fate Code. The values defined using the Fates function get displayed. This is a mandatory field.

  4. Click [Execute Query] or press [F8] or [Tab] to view the product details.

    The following product details are displayed:

    1. Product Code – The expired product codes within the dates specified in the query criteria are displayed.

    2. Blood Group - The Blood Group of the expired products are displayed.

    3. Rhesus – The Rhesus of the expired products are displayed.

    4. Unit Number – The Unit Numbers of the expired products are displayed.

    5. Expiry Date – The expiry date of the expired product is displayed.

    6. Patient ID – The Patient ID to which the unit was initially allocated is displayed.

    7. Location – The assigned storage location of the unit is displayed.

    8. Description – The corresponding Description of the location is displayed.

    If the check box ‘Effect on Stock’ is selected as ‘Deducted’ while defining the Fates, the selected unit is automatically deducted from the stock and the stock gets updated.

  5. Select ‘Select All’ to dispose all the expired blood units. Select the check box for the respective expired unit to dispose it.

  6. Click [Save] or press [F10] to dispose the selected expired blood units or click [Clear Record] to clear the entered data.
