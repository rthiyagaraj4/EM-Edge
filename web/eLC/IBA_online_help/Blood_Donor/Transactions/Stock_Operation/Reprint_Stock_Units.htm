Reprint Stock Units

From this function you can Reprint the stock details.  You can Query to view the stock unit details.

The Reprint Stock Units function allows you to reprint the details of the Stock. You can query the unit details based on the unit numbers or on the date when the blood units are added to stock. You can choose to print selected blood unit details. Option is also available to specify the number of copies required.

Reprinting Stock Units

From the Transactions > Stock Operation, double-click the Reprint Stock Units function.

The Reprint Stock Units screen gets displayed.

  1. Enter the Unit Number under ‘From’ and ‘To’ to query for unit details matching the search criteria. This is a mandatory field.

  2. Click the Calendar function under ‘From’ and ‘To’ to select the date when the unit is added to stock. This is a mandatory field.

  3. Click [Show Details] to view the stock details matching the query criteria. <Related Field Description>

  4. Select the ‘Select All’ check box to reprint the details of all the units. Select the ‘Select’ check box to reprint the details of individual Stock Units.

  5. On selecting the check box, the number of copies is displayed as ‘1’. Enter the number of copies in numeric format with a maximum of two digits to print the selected stock unit details.

  6. Click [Print] to reprint the details of the selected Stock Units.
