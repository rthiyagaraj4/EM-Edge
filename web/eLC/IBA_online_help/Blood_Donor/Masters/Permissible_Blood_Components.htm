Permissible Blood Components

From this function you can Define the permissible blood components of a blood product by associating the components to the selected product. You can Query for the existing records.

The Permissible Blood Components function enables you to define all permissible blood components that can be prepared from a Blood Product. You can associate one or more products as permissible blood components for a selected product using this function.

Defining Permissible Blood Components

From the Masters, double-click the Permissible Blood Components function.

The Permissible Blood Component screen gets displayed.

  1. Click [List of Values] to select the Product or enter the Product code to associate the components to the Product. The values defined using the ‘Products’ function with the ‘Donated Product?’ and ‘Apheresis Product?’ values set as ‘Yes’ get listed.  

  2. Click [List of Values] to select the Permissible Products. The values defined using the ‘Products’ function get listed.

  3. Click [Save] or press [F10] to save the created records or click [Clear Record] to clear the entered data.

  4. Select the Product from the list and click [Execute Query] or press [F8] to search for the associated blood components.

 

You cannot associate duplicate permissible components to the Product.
