Donor Details

From this function you can Query the personal details of a donor.

The Donor Details function enables you to query about the donor’s personal details. All the details entered during donor registration and pre-donation can be viewed from this function.

Querying the Donor Details

From the Query, double-click the Donor Details function.

The Donor Details screen gets displayed.

  1. Click [Execute Query] or press [F8] to query for all the donor details. You can navigate to the next and previous record using the downward and the upward arrows in the keyboard. <Related Field Description>

  2. Click [Enter Query] and enter the detail in the respective field to search for a specific donor record.

OR

  1. Right-click on the ‘Right Click for User Option’ to search for a specific donor using the Donor Lookup function.

  2. The demographic details, passport number, Status of the donor, the donation process details and contact details of the donor are displayed under the respective tabs.
