Recall Donor for Counseling  

From this function you can Query the details of the donors’ recalled for Counseling. You can View the donor history and Print the counseling list and counseling letter of the donor.

The Recall Donor for Counseling query function populates the records of all donors who have been deferred temporarily or permanently. You can execute query by specifying the donation date range and deferral reason code. You can View the donor history and Print the counseling list and counseling letter of the donor.

Querying the Donors Recalled for Counseling

From Query, double-click the Recall Donor for Counseling function.

The Recall Donor for Counseling screen gets displayed.

  1. Select the Donation Date or click Calendar function to select the date range. The current date is displayed, by default.

  2. Enter the Batch Number to view the donor details associated to the respective batch.

  3. Click [List of Values] to select the Deferral Reason. The description for the selected donor gets displayed.

  4. Click [Execute Query] or press [F8] to search for the donor details. The Donor ID, Donor Name, Donation Date, Donation Type Process, Deferred Date, Deferred Till Date and Deferred Reason are displayed.

  5. Select the check box ‘Select’ to select the individual donor record. You can either view the donor history , print counseling letter or the counseling list of the respective donor.

  6. Select the ‘Select All’ check box to select the donor records either to view donor history, print counseling list or letter.

  7. Click [Donor History] to view the details. The Donor History  screen is invoked with the donor details.

  8. Click [Print Counseling Letter] to print the counseling letter for the donor.

  9. Click [Print Counseling List] to print list of the donors, who are called for counseling.
