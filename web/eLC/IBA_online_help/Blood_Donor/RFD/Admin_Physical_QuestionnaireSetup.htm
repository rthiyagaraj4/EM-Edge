Enter the following details:

 

  1. Valid For – Select the value from the list to specify whether the setup is valid for Questionnaire or Physical Checkup.

  2. Question Code – Enter a unique and valid question code in alphanumeric format of four characters.

  3. Sequence Number – The function automatically updates the sequence number of the question.

  4. Question in English Language – Enter the question in English Language in alphanumeric format.

  5. Question in Local Language – Enter the question in Local Language alphanumeric format.

  6. Sex Specific – Select the gender, if the question is Gender specific. The value is set to 'Both', by default.

  7. Mandatory? – Select a value from the list. The system-defined values are ‘Yes’ and ‘No’. If the question is mandatory, select ‘Yes’. The value is set to ‘No’, by default.

  8. Type of Answer – Select the type of answer expected from the donor, that is either ‘List of Values’ or ‘Numeric’. The value ‘Numeric’ is displayed, by default.

 

Validation for Numeric

 

Valid Range

  1. Default Value – Enter the value that must appear as default value for the respective question. The default value has to be within the lower and the higher value range specified.

  2. Low Value – Enter the lowest value acceptable as the answer.

  3. High Value – Enter the highest value acceptable as the answer.

  4. Units – Enter the value, or click [List of Values] to select the Unit code and the respective description is displayed. The values defined using the ‘Material Setup’ function of the Applications Masters module get listed.

  5. Action - Select the value for action from the list for the specified numerical value. The list displays the values Accept, Reject and Warning.

  6. Test to be Ordered? - Select ‘Yes’ from the list, if a test needs to be ordered for the specified numerical range and action. The value is set to ‘No’, by default.

 

Validation for List of values

When you select the Type of answers as list of values the following fields replaces the existing fields on the screen.

  1. List of values – Enter the value or select it from the [List of Values].

  2. Action – Select the value from the list. The values are Accept, Reject and Warning.

  3. Test to be ordered? – Select the value from the list. The values are Yes and No. Select ‘Yes’ from the list, if a test needs to be ordered.

  4. Default value? – Select the check box if the entered value is to be the default value.
