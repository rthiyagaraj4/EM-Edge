Enter the following details to record items used in the mobile camps:

 

  1. Mobile Item – Enter the Mobile Item code received from the mobile drive or click [List of Values] to select it. The values defined using the Devices function get listed.

  2. Description – On selecting the Mobile Item code, the corresponding Description gets displayed.

  3. Blood Bag – Enter the Blood Bag Code received from the mobile drive in alphanumeric format of four characters maximum length or click [List of Values] to select the Blood Bag. The values defined using the 'Blood Bag' function, are listed for selection. The Blood Bags get listed only for the mobile drive item with the 'Blood Bag' check box selected in 'Mobile Drive Items' masters.

  4. Description – On selecting the Blood Bag code, the respective Description gets displayed.

  5. Blood Group – Enter a valid Blood Group or click [List of Values] to select it. The values defined using the Blood Groups function get listed.

  6. Rhesus – Click [List of Values] to select Rhesus factor. The values defined using the ‘Rhesus’ function get listed.

  7. Component – Enter the Component code received from the mobile drive or click [List of Values] to select it. The values defined using the ‘Permissible Blood Components’ function get listed.

  8. Description – The Description of the selected component gets displayed.
