Enter the following details to record the details to proceed with the blood donation:

 

  1. Consent form reviewed by – The name of the logged in user gets displayed, by default. You can change the name of the practitioner, if you need. This field indicates the name of the practitioner who has reviewed the consent form of the donor.

  2. Reviewed Date – The current system date and time is displayed, by default. You can change the date and time if you need. This field indicates the date and time when the consent form was reviewed.

  3. Print Labels? – Select the check box to print the labels. Clear the check box, if the labels are not required.

  4. Number of Labels – The Number of Labels to be printed is displayed. On selecting the Print Labels? check box, the value entered in BD Parameter gets displayed.

  5. Proceed For Donation – Select ‘Yes’, if the donor is eligible for blood donation. Select ‘No’, if the donor is not eligible for blood donation.

  6. Remarks – Enter the remarks in alphanumeric format, if required.
