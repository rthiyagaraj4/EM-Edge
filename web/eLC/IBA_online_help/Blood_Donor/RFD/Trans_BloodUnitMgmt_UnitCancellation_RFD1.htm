The following details are displayed on entering the Unit Number to be cancelled:

  1. Unit Number – The Unit Number entered in the ‘Barcode Unit Number Product Code’ screen gets displayed, by default.

  2. Product – The Product associated to the Unit Number gets displayed along with the Product description.  

  3. Cancel code – Enter a valid Cancel Code or click [List of Values] to select the Cancel Code. The values defined using the ‘Cancel Codes’ function get listed.

If components are prepared for the unit, this field is not enabled. Select the cancel code from the Component Details tab. If the unit is added to the stock, you cannot cancel the unit, but dispose the unit.

  1. Storage Location – The associated Storage Location is displayed.

  2. [Donor Notes] – Click [Donor Notes] to view donor notes details. The [Donor Notes] is enabled, if Donor Notes are entered for the existing unit.

  3. [Antibodies] – Click [Antibodies] to view the antibody details. The [Antibodies] is enabled, if Antibody details are entered for the existing unit.

  4. [Antigens] – Click [Antigens] to view the antigen details. The [Antigens] is enabled, if Antigen details are entered for the existing unit.
