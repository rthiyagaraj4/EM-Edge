1. Workstation No – The Workstation Number is displayed. You cannot change the values in it.

  2. Report ID – The Report ID is displayed.

  3. Facility – The facility from where the print request has come is displayed.

  4. Destination Type – The Destination Type is displayed. The list items are Department, Clinic and Ward and Location.

  5. Destination Code – The Destination Code is displayed. The criteria for destination code are, for Department - Operating Department Code, for Ward - Ward Code and for Clinic - Clinic Code.

  6. Description – The corresponding Description of the selected destination code is displayed.

  7. Copies – The Number of copies to be printed are displayed.

  8. 1st Printer – The first printer name is displayed.

  9. 1st Printer Start Time – The first printer starting time is displayed in hh:mm format.

  10. 2nd Printer - The second printer name is displayed.

  11. 2nd Printer Start Time – The second printer starting time is displayed in hh:mm format.

  12. 3rd Printer - The third printer name is displayed.

  13. 3rd Printer Start Time – The third printer starting time is displayed in hh:mm format.
