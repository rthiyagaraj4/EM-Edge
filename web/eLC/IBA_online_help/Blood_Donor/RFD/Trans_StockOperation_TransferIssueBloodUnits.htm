Enter the following details:

 

  1. Display – Select the value from the Display list to view the blood unit request details. The Display list displays ‘All’ by default. The system defined values are ‘All’ and ‘Pending For Issue’. Select ‘Pending for Issue’ to view the units that are pending for issue.

  2. Centre Code – The Default Collection/Supplier Centre Code defined in the ‘BD Parameter’ function is displayed. You cannot update the Centre Code.

  3. From Date – Click Calendar function to select the start date. The blood units whose requisition date matches with the date range selected get listed.

  4. To Date – Click Calendar function to select the To date. The blood units whose requisition date matches with the date range selected get listed.

  5. Requisition Number – Click [List of Values] to select the Requisition Number to issue the blood unit.
