Interface

  1. Interfaced to Blood Transfusion – Select the check box, if the Blood Donor Management module needs to be interfaced to the Blood Transfusion Services module.

Blood Units Transfer Parameters

  1. Add Blood Units to BT stock on Transfer? - Select the check box to transfer the Blood Units to BT Stock using the ‘Transfer/Issue Blood Unit’ function under Stock Operations menu in transactions.

  2. Add Blood units to BT stock directly? – Select the check box to add the blood units directly to the BT Stock without using the ‘Transfer/Issue Blood Unit’ function.

  3. Fate Code (Transfer) – Click [List of Values] to select the Fate Code. The system displays the selected value as default, while transferring the Blood Units to BT Stock or disposing the blood unit.

  4. Override Screening? - Select the check box to override the screening of blood units in the BT stock. Clear the check box to do the screening of blood units before adding to the BT Stock.

This check box is introduced so that the screening of the blood units transferred from BD Stock to BT Stock can be ignored, as screening is already done before adding the stock to the BD Stock. This rule is not applicable to the blood units received from the external laboratories.

 

Default Parameters

  1. Print Blood Bag label before adding to Stock? – Select the check box to print the blood bag labels before adding the blood units to the stock.

  2. Display Donor Notes? – Select the check box to display the donor notes in various functions.

  3. National ID Mandatory? – Select this check box to capture the national ID of the patients mandatorily in Donor Registration function.

  4. Use ISBT128 Numbering for Blood Bags? – Select the check box to use ISBT128 numbering for blood bags.

Note: ISBT128 provides a unique identification of any donation worldwide. This is done by using a 13 character identifier built up of three elements, first, identifying the collection facility, second, the year and third, a sequence number for the donation.

  1. Length – Enter the Length in numeric format of two digits maximum length to specify ISBT128 numbering for blood bags. The label numbering cannot cross the digits specified.

  2. Print Blood Bag Label at Pre-Donation – Select the check box to print the blood bag label during pre-donation.

  3. Number of Labels – Enter the numerical value to specify number of labels to be printed in two digits maximum length.

Blood Letting

  1. Generate Unit Number – Select the check box to generate unit numbers automatically.

Note: The system prompts you to enter the unit number in Collection Centre/Supplier or Products functions, if setup is not defined to generate the unit number automatically in BD Parameter.

  1. Length – Enter the Length of the unit number. It is a one-time setup and you cannot make any changes to it later.

  2. Prefix – Enter the Prefix value of the unit number. It is a one-time setup and you cannot make any changes to it later.

  3. Next Number – This field is disabled. When you save the details in pre donation analysis, the Next Number to be generated is displayed.

  4. Cancel Code For Letting – Enter the required value for Cancel Code for Letting, or select it from the list of values. The value selected will be set as the default value in the functions wherever it gets displayed.

  5. Number of Labels – Enter the required number of labels in two digits maximum length.

  6. Prepare Components based on Blood Bag Weight? – Select the check box, if the components have to be prepared according to the blood bag weight rules.

  7. Blood Bag Weighing Units – Enter the default blood bag weighing unit. The list displays the values defined in the Unit of Measurement function.

  8. Mobile Items Entry Sequence – Select the entry sequence for mobile items from the list. The system-defined values are ‘Donation Details’ and ‘Drive Items Register’. If you select Donation Details, the entry sequence is based on the donation details in the Donor Items Received Register function. Select Drive Items Register to list the entry sequence based on the Drive items.

  9. Storage Location – Enter a valid Storage Location code or click [List of Values] to select the default storage location. The values defined using the ‘Storage Location’ function get listed.

  10. Process Blood Unit Based On – Select a value from the list to process the blood unit. The system-defined values are ‘Date Range’ and ‘Batch’.

  11. Reject Storage Location - Click [List of Values] to select the required storage location. The values defined using the Storage Location function get listed.

  12. Fate Code (Autologous) – Click [List of Values] to select a default fate code to be used in the Autologous Type transactions. The values defined using the Fates function get listed.

  13. Collection Centre (Pooled Unit) – Click [List of Values] to select the Collection Centre. The values defined using the Collection Centre/Supplier get listed.

Lab Screening Parameters

  1. Collection/Supplier – Enter a valid Collection/Supplier Centre code or click [List of Values] to select the Collection/Supplier Centre code. The values defined using the function Collection/Supplier get listed. The selected values will be the default value throughout the transactions in the Blood Blood Donor module.

  2. Source – The default Source code to be used in the transactions is displayed. The value is set up in the back-end and disabled.

  3. Screen Donor ID – Enter the Screen Donor ID in alphanumeric format of 30 characters maximum length. The default Donor ID is used while sending the specimens for testing. The donor’s name is masked and this ID is displayed while sending samples to test.

  4. Accept Initial Blood Type Miss match? – Select the check box, if the Initial Blood Type Miss match is to be ignored during the transactions in Blood Unit Screening Result Entry function. If the check box is cleared, you have to change Initial Blood Typing to the resulted Blood Group and Rhesus using the [Change Initial Blood Typing] in Blood Unit Screening Result Entry function.

Reactive Donor Units

  1. Screen Blood Bag? – Select the check box to send the reactive blood unit for screening automatically.

  2. Screen Reactive Tests Only? – Select the check box to screen the tests, which are reactive.

  3. Cancel Code for Blood Bag – Enter the cancel code for the reactive blood bag or click [List of Values] to select the cancel code. The code selected is displayed as the default value while cancelling the blood bag.

  4. Max. Time Limit for recall – Enter the maximum time limit in numeric format of four digits maximum length to recall the reactive donor. Select the required unit of measurement from the list.  

  5. Recall Donor? – Select the check box to recall the donor for donation.

  6. Maximum Acceptable Bleeding Time – Enter the maximum acceptable bleeding time in numeric format of five digits. Select the unit of measurement from the list.
