Enter the following to record donation details:

 

  1. Donor ID – Enter a valid Donor ID or click [Donor Lookup] to select a Donor ID to donate blood. You can process the only those donors who are found fit after the pre-donation analysis.

 

On selection of the Donor ID, the Record Bleeding screen displays the following information:

  1. Personal Details – The Name of the donor, Age and Gender of the donor entered in the ‘Register Donor’ function get displayed.

  2. Unit Details – The Unit Number, Component, Batch number, Blood Bag, No of bags, Venepuncture Site and Blood Bag Lot No entered in the ‘Pre-Donation’ function get displayed.

  3. Donation Details – Donation Type, Donation Process, Blood Group, Rhesus and Type of Bleeding entered in the ‘Pre Donation’ function get displayed.

[Antibodies], [Antigens] and [Donor Notes] get enabled, if corresponding details are recorded for the selected donor.

  1. Volume – Enter the volume of the blood to be donated. This is a mandatory field.

  2. Apheresis Device – Enter the Apheresis device or click [List of Values] to select it. The values defined using the Devices function get listed.

  3. Lot Number – Enter the Blood Bag Lot number in alphanumeric format of 20 characters maximum length.

  4. Kit Number – Enter the Kit Number in the text box in alphanumeric format of 20 characters maximum length.

  5. Operator – Enter the name of the Operator in alphanumeric format of 30 characters maximum length.

 

Donation

  1. Start Time – Click the Calendar function to select the Start Time of donation. This is a mandatory field.

  2. End Time – Click the Calendar function to select the End Time of donation. This is a mandatory field.

  3. Blood Agitator - Enter the logged in user name or click [List of Values] to select the logged in user’s name, who performs the donation process. The values defined using the ‘Users’ function in the System Manager module get listed.

  4. Start By – Enter the practitioner name, who started the donation process or click [List of Values] to select the user name.

  5. End By – Enter the practitioner name, who has ended the donation process or click [List of Values] to select the user name.

  6. Incidence – Enter the Incidence or click [List of Values] to select the incidence. The listed values are created in the Incidence function.

  7. Location – Enter the location of the donation, or click [List of Values] to select the donation location. The values defined using the ‘Bleeding Locations’ function get listed. This is a mandatory field.

  8. Donor Reaction – Enter the Donor Reaction or click [List of Values] to select it, if there are any reactions during the donation. The values defined using the Reaction function get listed.  

  9. Storage Location – Enter the storage location to store the blood or click [List of Values] to select the Storage Location for the donated blood. The values defined using the‘Storage Locations’ function get listed.

  10. Discontinued – Select the check box, if the donation process is discontinued.

  11. Discontinued Reason – Enter the Discontinued Reason code or click [List of Values] to select it. The values defined using the ‘Discontinued Reason’ function get listed.

  12. Remarks – Enter the Remarks on the donation process.
