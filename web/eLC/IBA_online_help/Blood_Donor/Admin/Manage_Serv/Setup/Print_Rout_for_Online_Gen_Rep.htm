Print Routing for Online Generated Reports

From this function you can Define the destination printers for each print document/output depending on the requirements of the specific document/output.

The Print Routing for Online/Generated Reports function is used to define the destination printers for each print document/output. Each report can be routed to multiple printers, depending on the requirements of the specific document/output. This function has the flexibility to define the routing dynamically, based on the Operating Department/Ward/Clinic, which is involved in the transaction, currently being processed. This also provides the facility to print the documents simultaneously at more than one destination and to print more than one copy at any of these destinations. A workstation number can be specified to override the default print routing. The Online reports get printed whenever you give the print command, whereas the Generated Reports get printed only when you run the print services.

Print Routing for Online Generated Reports

From Administration > Manage Services > Setup, double-click the Print Routing for Online Generated Reports function.

The Print Routing for Online Generated Reports screen gets displayed.

  1. Select Online Reports tab and enter the mandatory (*) details. <Related Field Description>  

  2. Select Generated Reports tab and enter the mandatory (*) details.
