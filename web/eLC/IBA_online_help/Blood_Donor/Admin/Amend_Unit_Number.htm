Amend Unit Number

From this function, you can Amend the unit number of a blood unit with an existing unit number.

The Amend Unit Number function is used to change the unit number of a blood unit with an existing unit number. You can amend the unit numbers of a blood unit until it is added to the stock. After the blood unit is added to the stock, you cannot change the unit number.

Amending Unit Number of a Blood Unit

From the Administration, double-click the Amend Unit Number function.

The Amend Unit Number screen gets displayed.

  1. Click the Calendar function to select the donation date range.

  2. Click [List of Values] to select the Mobile Unit Code. The listed values are created in the Mobile Unit function.

  3. Click [List of Values] to select the Batch Number. The listed values are created in Create Batch function.

  4. Click [Display] to show the Unit Number details based on the selected search criteria.

  5. The Unit Number details such as Donor ID, Donation Date, Blood Bag, Mobile Item, Blood Group, Rhesus Code, Donation Type and Unit Number are displayed.

  6. Click [List of Values] to select the new Unit number. The Unit Numbers that satisfy the query criteria are displayed. You can select any Unit numbers from the list.

  7. Click [Amend] to amend the unit number. If you do not change the unit number of the changed blood unit a message ‘Corresponding Unit Number Exists’ is displayed. Click [List of Values] to select the unit number for the blood unit.

  8. When you click [Amend], a message ‘Confirm Amendment of Units?’ is displayed. Click [OK] to amend the unit number or [Cancel] not to amend the unit number. On amending the unit number, the message ‘Units amended successfully’ is displayed.
