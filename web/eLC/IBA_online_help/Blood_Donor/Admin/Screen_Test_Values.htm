Screen Test Values

From this function you can Create the screen test values and Query for the existing records.

The Screen Test Values function enables you to create screen test values. The screen test values are applicable for Apheresis, Blood Bag and Deferred screening, if the check box for each is selected in the Applicable for section in Screening Test Setup function.

Creating Screen Test Values

From the Administration, double-click the Screen Test Values function.

The Screen Test Values screen gets displayed.

  1. Enter a valid test code in alphanumeric format of four characters maximum length. The codes are used in the Screening Test Setup function.

  2. Enter the description of the long name of the test in test code in alphanumeric format of 40 characters maximum length.

  3. Click [Save] or press [F10] to save or click [Clear Record] to clear the entered details.

  4. Click [Execute Query] or press [F8] to search for the existing records.
