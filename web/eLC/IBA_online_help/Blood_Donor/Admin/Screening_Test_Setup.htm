Screening Test Setup

From this function you can Create tests for screening the blood units and Query for the existing records.

The Screening Test Setup function enables you to create tests for screening the blood units. The screening test is applicable for Apheresis, Blood Bag, RH Geno Type and Deferred screening if the check box is selected in the Applicable for section. You can select the desired tests for screening, define type of answers, validations for tests with numeric results (if any) and list of values and actions for the respective values.

Creating Screening Tests

From the Administration, double-click the Screening Test Setup function.

The Screening Test Setup screen gets displayed.

  1. Select the desired value from the list to specify whether the test setup made is valid for ‘Screening’ or ‘Rule Based’ tests.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Save] or press [F10] to save or click [Clear Record] to clear the entered details.

  4. Click [Execute Query] or press [F8] to search for the existing records within the facility.
