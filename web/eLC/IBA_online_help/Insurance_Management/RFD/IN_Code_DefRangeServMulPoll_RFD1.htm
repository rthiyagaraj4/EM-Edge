Enter the following details to copy or delete the services from the range of policies:

 

  1. Exclude/Include Criteria – Select a value from the list. The system defined values are ‘Billing Service’, ‘Enterprise Service Group’, ‘Enterprise Service Group/Line’, ‘Billing Service Group’ and ‘Billing Service Group Classification’.

  2. Billing Service [From-To] – Click [List of Values] to view and to select a valid Billing Service Code. If the code is valid, the short description of the Billing Service code gets displayed. A range of Billing Services can be included or excluded from the Policy Types based on the value selected for Exclude/Include Criteria. This is a mandatory field.

The above legend changes as per the value selected in the Exclude/Include Criteria. The values in the list for the legend too differs based on the value selected for Exclude/Include Criteria.

  1. Excl/Incl Ind – Select a value from the list. The system defined values are ‘Include’ and ‘Exclude’. Select Include and the set of Billing Services are associated to the range of policies selected. Select Exclude and the set of Billing Services are excluded from the range of policies selected.

  2. Override Inclusion/Exclusion – Select the check box to override the Inclusion or the Exclusion criteria. The override performs inverse action based on the value selected for ‘Excl/Incl Ind’.

  3. Patient Class – Click [List of Values] to view and to select a Patient Class. The system defined values are All, Daycare, Emergency, Inpatient, Outpatient and External. Select All and the range of services copied or  deleted for inclusion/exclusion is applicable for all the patient class in the facility.

  4. Co-pay Ind – Select a value from the list. The system defined values are ‘Percentage’, ‘Rate’ and ‘Fixed’. The Co-pay value is set based on the Co-pay Indicator set. The amount to be paid by the co-payer towards the services in the range of policies is calculated based on the Co-Pay Indicator selected. You can select a value from the list, if the ‘Excl/Incl Ind’ is selected as ‘Include’.

  5. Include Policy Level Coverage – Select the check box to include policy level coverage details. The ‘ Co-pay Ind ’ has to be set as ‘ Percentage ’ or ‘ Rate ’ to select the check box and to define the Co-pay value. If the   ‘ Co-pay Ind ’ is selected as ‘ Fixed ’ , the check box is disabled and you cannot record the Co-pay value.

  6. Copay Value – Enter the Copay Value in numeric format. The amount to be settled by the Co-payer is calculated based on the Co-Pay Indicator selected and the value specified for Copay value. The entered value has to be in the range between 1 and 99.

  7. Pre Apprv – Select a value from the list. The system defined values are ‘Required’ and ‘Not Required’. Select ‘Required’ to indicate that an approval is required from the Insurance Company to include or exclude the services to the range of Policies selected. Select ‘Not Required’, if an approval is not mandatory from the Insurance Company to include or exclude the services from the range of policies selected.

  8. Special Service – Select the check box, if special services can be associated to the range of policies selected. Clear the check box and you cannot associate special services to the range of policies.
