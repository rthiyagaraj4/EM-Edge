Enter the following to create Insurance Deductibles:

 

  1. Facility – Select the Facility from the list. The facilities associated to the logged in User get listed. The logged-in-facility is displayed, by default.

  2. Customer Code - Click [List of Values] to view and select a Customer Code. The description for the selected Customer Code is displayed. This field is mandatory.

  3. Policy Type - Click [List of Values] to view and to select a Policy Type code. The description for the selected Policy Type is displayed. This field is mandatory.

  4. Applicable to – Select a value from the list. The system defined values are ‘Outpatient/Emergency’ and ‘Inpatient/Daycare’. The deductible policy is applicable for the selected Patient Class associated to the Customer and Policy in the facility.

  5. Policy Fee Billing Service - Click [List of Values] to view and to select a charge based Billing Service.

  6. Policy Fee Amount – Enter the Policy Fee Amount in numeric format.

  7. Refund to Policy Provider – Select the check box to refund the amount to the Policy provider or the Insurance Company. The amount is refunded to the Policy Provider for the Clinics or the  Nursing Units associated to the Policy Type and Customer combination within the selected facility.

The combination of Customer code, policy type code and illness/visit Indicator must not be a duplicate combination.

Visit/Clinical Details

  1. Clinic Code - Click [List of Values] to view and select to a Clinic Code. The Policy Deductible set up is valid for the selected Outpatient Clinic or Inpatient Nursing Unit within the facility.

  2. Description – The description for the selected Clinic Code or the Nursing Unit Code is displayed.

  3. Visit Type - Click [List of Values] to view and to select a Visit Type. The Visit Type associated to the selected Clinic Code gets listed. The Policy Deductible set up is valid for the selected Visit Type to an Outpatient Clinic within the facility.

  4. Description - The description for the selected Visit Type gets displayed.

  5. Policy Fee Billing Service – Click [List of Values] to view and to select a Billing Service. This indicates the Billing Service which includes the policy coverage amount.

  6. Description – The description for the selected Billing Service gets displayed.

  7. Refund to Policy Provider – Select the check box to refund the amount to the Policy provider or the Insurance Company. Clear the check box, if the refund is not applicable.

  8. Deducted Amount – Enter the amount in numeric value to indicate the deducted amount for the service. It must not be a negative value. The sum of deducted amount of the detail block has to be equal to the Insurance deductible amount of the header block.
