Enter the following selection criteria:

 

  1. Patient ID - Enter a valid patient ID or click [List of Values] to select a valid patient ID. A list of services availed by the selected patient associated to the relevant payer or policy gets listed. If a valid patient number is selected, the patient details are displayed. This is a mandatory field.

  2. Payer - Enter a valid payer ID or click [List of Values] to select a valid payer. The description for the selected payer gets displayed. The values defined using the function 'Customer' gets listed.

  3. Policy - Enter a valid Policy Type code or click [List of Values] to select a valid Policy Type. The description for the selected Policy Type gets displayed.  The values defined using the function Policy Types get displayed.

  4. Encounter Date [From–To] – Click Calendar function to select the date range. The encounter for the selected date range gets displayed. This is a mandatory field.

  5. Option – Select an Option from the list. You can view the details based on the option selected. The values are system defined - 'Summary by Enterprise Service Group', 'Summary by Payer Service Group' and 'By Individual Services'.
