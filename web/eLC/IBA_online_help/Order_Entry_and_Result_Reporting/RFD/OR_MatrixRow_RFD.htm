Enter the following details:

 

  1. Result Matrix - Select the Result Matrix from the list. The values are listed from the matrix row. This field is mandatory.

  2. Code - Enter the Code in alphanumeric format of four characters maximum length.

  3. Long Description - Enter the Long Description in alphanumeric format of 15 characters maximum length.

  4. Short Description - Enter the Short Description in alphanumeric format of 10 characters maximum length.

  5. Sequence No - Enter the Sequence No. in alphanumeric format of four characters maximum length.

  6. Discrete Measure - Enter or select the Discrete Measure by clicking [?] function, click [Search] and a screen gets displayed. All the Discrete Measures set up in the 'Discrete Measure' function in Application Masters module are displayed.

  7. Copy Prev - Select Copy Previous Order to copy the Discrete Measure from the previous row.

  8. 8. Enabled - Select the check box to make the column available for transaction functions. The check box is selected, by default.
