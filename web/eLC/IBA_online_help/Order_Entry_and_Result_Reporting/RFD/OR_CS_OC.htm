Enter the following details:

 

  1. Code - Enter the code for Order Catalog in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Long Description - Enter the long description for the order catalog in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the Short Description for the order catalog in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  4. Order Category - Select the Order Category from the list. The Order Categories are system defined. This is a mandatory field.

  5. Order Type - Select the primary Order Type for the Order Catalog. The values defined using the function Order Types and associated to the selected Order Type get listed. This is a mandatory field.

  6. Activity Type - Select the Activity Type from the list. The Activity Types associated to the selected Order Category gets listed.  

  7. Body Organ - Select the Body Organ from the list. The values displayed in the list are defined using the function 'Image Hotspot Link'. This field is applicable for Department Diagnostics, Radiology and Trmt/Procedure Order Category. The image hotspot link is associated to an Order Catalog using the list.

  8. Index - Select the check box to index the defined Order Catalog. If the Order Catalog is indexed, it has to be associated to a service, that is, the defined Order Catalog gets listed in the function 'Index Order Catalog, where it is associated to a Service'. If the Order Catalog is not indexed, it is not available across the services of the facility.

  9. Enabled - Select the check box to enable the record in the transaction functions. The check box is selected, by default.
