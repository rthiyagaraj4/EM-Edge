Enter the following details:

  1. Fluid – Enter the required fluid name in alphanumeric format of 60 characters maximum length or select the required Fluid using [?] function. This field is mandatory. On invoking the search function, the ‘Drug Search’ screen gets displayed. Specify the required query conditions and invoke the search process. The result screen as shown below gets displayed.

  1. Volume – Measure of the fluid in terms of base unit is displayed as read-only text.

  2. Total Volume – The default value would be one unit of the selected fluid. The Doctor can modify the Total Volume. Total volume is equal to the product of Order Quantity and  Volume/Unit.

  3. IV Admixture – Select the check box to indicate that IV admixture needs to be prepared for the patient. If the ordering time is after the working hours of admixture preparation room, then an alert is displayed. The practitioner is informed that the admixture would be prepared on the following day as dispense location to which the order would be routed is closed for the day.

  4. Build MAR - Medication Administration is applicable for all patient classes depending on the Parameter settings. For inpatients, Build MAR is default checked and enabled. For remaining patient classes based upon parameter settings it will be checked and enabled.

 

Administrative Details

Administration details of the selected IV with Additives/Drugs are to be specified under this section. Enter the following details.

  1. Infusion Rate – The infusion rate is calculated by dividing the Fluid volume with Infuse over value. For ‘Continuous’ orders, ‘Hours’ displays by default that cannot be changed. Infuse over cannot exceed 24 hours for ‘Continuous’ orders.

  2. Order Quantity - The default Order Quantity value will be ‘1’. The Doctor can modify the prescribed quantity of IV fluids within the new Order Quantity field and it will not accept decimal value.

  3. Infuse Over – The ‘Infuse Over’ specified for the selected IV Fluid (in Drug Master Function) is displayed in Hours and minutes format, by default. The Practitioner can specify the required period in numeric format of two digits.

  4. Duration – Enter the duration in numeric format of two digits maximum length. The specified duration has to be greater than or equal to ‘Infuse Over’ value. Duration cannot exceed 24 hours for ‘Continuous’ orders.  

  5. Start Time - The system start date/time are displayed, by default. The practitioner can modify, that is pre-date or post-date the medication starting date/time based on the setting in OR Parameter. This field is mandatory.

  6. End Time – The end date/time is calculated by the system by adding the specified duration with the start time.

  7. Admin Rate - Click on ‘Admin Rate’ to view the IV administration rate. The ‘Admin Rate’ hyperlink displays once the system calculates the ‘Infusion Rate’. The administration rate is calculated by the system and displayed for the selected drugs.
