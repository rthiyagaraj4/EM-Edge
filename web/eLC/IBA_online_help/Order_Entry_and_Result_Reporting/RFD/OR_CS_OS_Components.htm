Component

Order catalogs are associated to the defined Order Set using the Component Tab. The Order Catalogs corresponding to the selected Order Category and Order Type get displayed. You have to select the required catalogs and include them under the defined Order Set. If an Order Set defined initially is selected as the Order Catalog, the Order Catalogs across all the Order Categories are displayed.

Two sub-tabs are provided under the Component tab, Select and Associate. The catalogs are displayed under the select tab. The Order Catalogs associated to the Order Set can be viewed under Associate tab.

 

[Select] Tab

[Select] tab is selected, by default. English alphabets from A to Z are displayed and hyperlinked. Select the required alphabet and order catalogs starting with the selected alphabets are displayed. Order catalogs starting with numeric and special characters are grouped under ‘Others’ hyperlink. The screen as shown gets displayed on selecting an alphabet.

  1. Catalogs - Catalogs starting with the selected alphabet are displayed.

  2. Order Catalog - Order catalogs starting with the selected alphabet are displayed.

  3. Seq No. - Enter the sequence number in numeric format of three characters maximum length. The defined number indicates the order of appearance of the order catalog.

  4. Order Type - Select the order type from the list. The primary order type defined for the displayed order catalog is displayed, by default. The displayed order types are attached to the catalog in the function 'Order Catalog By Order Type'.

  5. Qty - The behavior of the field depends on the characteristics of the order catalog. The value set for the order catalog is inherited and displayed. If the order catalog allows for an update, the quantity value can be changed or the field is defaulted to 'one' and as read-only. Enter the quantity in numeric format of four digits maximum length.

  6. UOM - Click [List of Values] to select a Unit of Measure.

  7. Frequency - The value set for the order catalog is displayed. The frequency values displayed are defined in the function 'Frequency Scope'. Select the required frequency.

  8. Durn value/Durn Type - The duration value set for the catalog is displayed. You can change the value and enter numeric value of three digits maximum length. The duration type as set for the catalog is displayed that cannot be changed.

  9. Time Frame (Hrs) - Enter the time frame in numeric format of three digits maximum length. The value specified indicates execution time of the catalog. For example, during health checkup of a patient, Fasting Blood sugar is the first test performed when the patient reports for testing. Subsequently, PPBS is performed, followed by ECHO and TMT. Intervals at which these tests are to be performed are specified as time frames. In the example, order for order set is placed and FBS to be performed at 8:00 am (ordering time is taken as 00:00 hrs), the time frame is specified as 8 hrs, PPBS at 11:00 am, time frame would be set at 11 hrs and so on.

  10. Required - Select the value from the list provided. Required is displayed, by default. The other values displayed are Optional/Pre selected and Optional/De-selected. If 'Required' is the value, the selection of the order catalog is mandatory while placing an order. If 'Optional/Pre Selected' is the value, it is not mandatory to select the catalog while placing the order. The order catalog will be selected for ordering (check box against the order catalog will be selected), by default. For 'Optional/De-selected' as the value, selection of catalog is not mandatory while placing the order and the check box against the order catalog is cleared while placing the order.

  11. Include - Select the check box to include the order catalog against it. The check box is cleared, by default.

  12. Comments - Enter the comments to be included for the selected order set. On including the required order catalogs, you can view the same under 'Associate' tab.
