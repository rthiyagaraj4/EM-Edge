Enter the following details:

 

  1. Order Category - Select the required Order category from the list. The Order categories that system supports are displayed. This field is mandatory.

  2. Order Type - Select the required Order Type from the list. All the Order Type (enabled) defined in 'Order Type' function appears in the list for the selected order category. This field is mandatory.

  3. Patient Class - Select the required patient class from the list of valid values. These classes are system defined. This field is mandatory.

  4. Performing Location - Select the performing Location, where the order is resulted. The values are displayed from the Performing location master function. The Performing Locations associated to the selected Order Type are listed.

 

Order Form

  1. Print at Destination - Select 'Print at Destination' check box to Print Order Form at the selected destination. The check box is enabled if 'Print at Destination' is selected in the function 'OR Parameter for Facility'.

  2. Print at Source - Select the check box to Print Order Form at order placing department. The check box is enabled if 'Print at Source' is selected in the function 'OR Parameter for Facility'.

  3. Routine - Enter the Printer Identifier (Printer IP or Name used in Enterprise Configuration module) in Loc1 and Loc2 text boxes for printing routine orders. Select Loc1 or Loc2 options from 'Default' heading to set the specified printer as default.

  4. Urg/Stat - Enter the Printer Identifier (Printer IP or Name used in Enterprise Configuration module) in Location 1 and Location 2 text boxes for printing Urgent/Routine orders. Select Loc1 or Loc2 optoin from 'Default' heading to set the specified printer as default.

  5. Print Option – Select the required print option from the drop-down list. The listed values are Confirm Alert, Silent Alert and Offline Alert. Based on the value selected in the Print Option, the Order sheet will be printed while placing orders.

  6. Silent Alert - If you select the value as ‘Silent Alert’, the system prints the Order sheet directly while placing order.

  7. Offline Alert – If you select the value as ‘Offline Alert’, the system will not print directly any Order sheet while placing order.

  8. Confirm Alert – If you select the value as ‘Confirm Alert’, the system prompts you to confirm the printing of Order sheet while placing order.

 

For Pharmacy Order Category, the default value for Print Option drop-down list is ‘Confirm Alert’. While for other order Categories except Pharmacy, the default value for Print Option drop-down list is ‘Silent Alert’.

 

Result Reports

  1. Print at Department - Select the check box to print the Result Reports at Order placing department. This check box is enabled if the 'Print at Department' check box is selected in 'OR Parameter for Facility'.

  2. Urg/Stat - Enter the Printer Identifier (Printer IP or Name used in Enterprise Configuration module) in Loc1 and Loc2 text boxes for printing the result reports of priority urgent/stat. Select Loc1 or Loc2 options from 'Default' heading to set the specified printer as default. You can enter the Location Name if ‘Print at department’ check box is selected.
