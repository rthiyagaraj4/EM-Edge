Instructions

The various instructions used while ordering are defined in the tab.

  1. Prompt Alert - Select the check box to enter the message in alphanumeric format. The alert message is displayed on selecting the Order Catalog while placing orders.

  2. Schedule Applicable - Select the check box to enable the schedule for the selected Order Catalog. A hyperlink ‘Schd’ gets displayed adjacent to the Order Category while placing an order. You can click on the hyperlink to view the details.

  3. Follow Visit Type – Select the Follow Visit Type check box to consider the number of slots entered in the Visit Type. If this check box is selected, Schedule Time field gets enabled.

  4. Schedule Time - Enter the Schedule Time.

  5. Order Priority Applicable – Select the check boxes to indicate that the priority is applicable to the selected Order Catalog - 'Routine', 'Urgent' and 'Stat'. The check boxes are enabled on selecting the Schedule Applicable check box.

  6. Clinical Comment Mandatory - Select the check box to enter clinical comments before placing an order or clear it.

  7. Specialty Specific – Select the check box to check if the selected Order Catalog is Specialty Specific. Clear the check box to check if the selected Order Catalog is not Specialty Specific.

  8. Primary Specialty – Select the Primary Specialty from the [?] function. This lookup function gets enabled if Specialty Specific check box is selected.

  9. Preparation Duration - Enter Preparation Duration in four digits maximum length. Select the Minutes/Hours/Days/Week/month from the adjacent list. This field is enabled, if the Schedule Applicable check box is selected.

  10. Action When Preparation Duration not met - Select a value from the list. The values are system defined - Not Applicable, Warn on Appt and Stop on Appt. This field is enabled, if the Schedule Applicable check box is selected.

  11. Preps - Enter the preparation instructions in alphanumeric format of 500 characters maximum length. You can view/modify the preparation instructions while placing the order.

  12. Department Instruction - Enter the department instruction in alphanumeric format of 500 characters maximum length. You can view/modify the department instructions while placing the order.

  13. Patient Instruction – Enter the instructions in alphanumeric format. These are the instructions given to patients about the orders.  You have to select Standard or By Age/Sex. On selecting the required type of instruction, click [Instructions] and enter the standard instructions. On selecting Age/Sex, a screen pops-up where you can enter the minimum and the maximum age. The instruction is applicable for the selected Gender in between the age range specified. Click Instrn hyperlink to record the instruction. Click [Select] to record the details and [OK] to confirm the instructions. Click [Clear] to reset the data entered. Click [Cancel] to exit from the [Instructions] screen. The instructions recorded cannot be edited while ordering.
