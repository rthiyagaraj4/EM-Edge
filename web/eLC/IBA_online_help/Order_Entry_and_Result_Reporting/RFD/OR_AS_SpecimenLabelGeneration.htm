Enter the following details:

 

  1. Facility - Select the Facility name from the list. The Facilities of the enterprise are listed. This field is mandatory.

  2. Contra Sys Module - The value ‘OR’ is displayed, by default. If Clinical Laboratory module is installed, the value displayed is 'RL'.

  3. Source Type - Select the Source Type from the list. The values displayed are Clinic, Nursing Unit and Referral. This field is mandatory.

  4. Source - Select the Source from the list. The values displayed correspond to the selected Source Type. This field is mandatory.

  5. Specimen Type - Select Specimen Type from the list. The values are populated from the Laboratory module.

  6. Priority - Select the urgency from the list. The values displayed are All, Routine, Stat and Urgent.

  7. Nurse Collect - Select the value from the list. The values displayed are All, Yes and No. ‘All’ is displayed, by default.

  8. Generate Label - Select the check box to generate the label by the system.

  9. Generation Look Ahead (in Mins) - Enter the time of label generation in numeric format of three digits maximum length. If period is specified, the label is generated after the specified time. If period is not specified, it is generated immediately. The field is enabled if 'Generate Label' is selected.
