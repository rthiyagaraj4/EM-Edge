Enter the following details:

 

  1. Order Category - Select the Order Category from the list. The Order Category is system defined. You can select either specific Order Categories or non-specific Order Categories to define a Tick Sheet Section. This field is mandatory.

  2. Tick Sheet - Select the Tick Sheet from the list. The values defined using the Tick Sheet function and associated to the selected Order Category gets listed. This field is mandatory.

 

Section

  1. Code - Enter the Section Code in alphanumeric format of four characters maximum length. This field is mandatory.

  2. Description - Enter the description in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Sequence No - Enter the Sequence Number in numeric format of two digits maximum length. The sequence number indicates the order in which the Sections are to be displayed for a selected Tick Sheet.

  4. Enabled - Select the check box to enable the defined section. The check box is selected, by default.
