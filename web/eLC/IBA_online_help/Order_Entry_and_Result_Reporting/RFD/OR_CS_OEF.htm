Enter the following details:

 

  1. Level - Select the value as either ‘Header’ or ‘Line’. If the value  is selected as 'header', the field mnemonics associated using this function gets displayed in the header section while placing an order. If the value is selected as 'line', the field mnemonics associated using this function gets displayed in the line level (Order Catalog). This is a mandatory field.

  2. Order Category - Select the Order Category from the list. The Order Categories are system defined. This is a mandatory field.

  3. Order Type - The Order Types displayed corresponds to the Level and Order Category selected. If the level is set to 'header', then all the order types are displayed. Select the required order type from the list. If the level is set to 'line', all the order types associated to the selected Order Category are displayed along with 'All'. Select the required value from the list. If 'All' is selected, all the Order Types of the selected Order Category have the associated field items. This is a mandatory field.

  4. Order Catalog - The Order Catalogs displayed correspond to the level, Order Category and the selected Order Type. At header level, the field mnemonics are common to all the Order Catalogs for the selected Order Category and Order Type combination and 'All' is displayed by default. At line level, the Order Catalogs can set to 'All' or specific Order Catalog can be selected. 'All' is displayed, by default. Select the required Order Catalog using [?] function. The [?] function provides Order Catalogs corresponding to the selected Order Type. This is a mandatory field.

  5. [Format Item] – Format Item is used to associate field mnemonics to the Order Entry Format. Click [Format Item], the Format Item screen gets displayed. If you select ‘Field Mnemonic’, [Dependency] gets displayed.

    1. Values - The values are displayed based on the Order Entry Format Item. Based on the values, the selected 'Action' occurs.

    2. Action – Select a value from the list. The values are 'Warning', 'Stop' and 'None'.

    3. Message - Enter Message in alphanumeric characters of 200 maximum lengths. The text entered is displayed as an alert. This field is mandatory, if 'Action' is selected as 'Warning' or 'Stop'.

    4. Alert Dept - Select the check box, if alert has to be sent to the department.

    5. Other Mnemonic - Click on Other Mnemonic to record additional mnemonic items. The format item screen gets displayed and you can add other format items. You can select same Field Mnemonic for different values. If you select the ‘Field Mnemonic’, the Dependency gets displayed. The  Dependency fields that appear are:

    • Operator – By default the ‘Current Date’ gets displayed. From the drop-down list, select any one of the required value: Equal to, Less than or Greater than. The system will validate the selected date with respect to current date.

    • Action – Select the required action to be taken from the drop-down list. The listed values are Warning, Stop and None.

    • Message – Enter the message that needs to appear based on Action selected.

    Based on the defined values, the Date Validation will be done while placing orders in Place Orders screen.

    On selecting Action as 'Warning', the alert message defined in the message box displays while placing an order. Select 'Stop' and the system restrict you from placing an order. If 'None' is selected, the system will not warn or stop on placing the order. On selecting the Order Entry Format Item, based on the values selected another order entry format item gets displayed (the format item displayed will be the format item selected on clicking Other Mnemonic) in Order Entry format while placing the order.

    1. Sequence No. - Enter the sequence number in numeric format of two digits maximum length. The field mnemonics appear in the order recorded.  This field is mandatory.

    2. Field Mnemonic - Select the value from the list. The mnemonics defined using the function Order Entry Format Item and associated to the selected Order Category gets listed. This field is mandatory.

    The value ‘Mobile Required’ is removed from the drop-down list when the Level is selected as ‘Header’, Order Category as ‘Radiology’ and Order Type relevant to Radiology.

    1. Label Text - Enter the label text in alphanumeric format of 50 characters maximum length. This field is mandatory.

    2. Accept Option - Select the accept option from the list. The values displayed are 'Optional', 'Required' and 'Display' only. Required is selected by default for Laboratory Orders and it is disabled to identify mandatory icons while placing orders.

    3. Age Group - Select an Age Group from the list applicable. The values defined using the Age Group function in the Application Masters module gets listed.

    4. Default value - Select a value from the list. This field is enabled on selecting a mnemonic.  The selected value displays as default value for the selected mnemonic in transaction functions. The system accepts values that correspond to the characteristic of the selected mnemonic. You can select or enter default value based on the Field Mnemonic type (order entry format item) selected. The values are defaulted for Laboratory and Radiology category from the respective modules.

  6. Definition Type - The Definition Type is displayed as 'User Defined', by default and you cannot change the value.
