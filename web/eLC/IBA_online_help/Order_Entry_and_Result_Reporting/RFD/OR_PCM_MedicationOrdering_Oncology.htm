Oncology Orders

The clinician prescribes Cyto-toxic drugs meant for use in the oncology clinics/wards using this function. This tab displays only for patients who are registered against the service to which ‘Oncology’ is applicable in Regimen Service master of PH code setup. Regimen typically consists of three to four drugs (all cyto toxic drugs) grouped as an order set along with dosage details. This grouping of drugs to form an Order Set is achieved using the order set features available in Order Entry and Results Reporting module. Dosage for cyto-toxic drugs are calculated based on the Body Surface Area/Weight thus the clinician has to compulsorily enter the height and weight value for a patient before placing an Oncology order. On prescribing the required oncology orders, the prescription is released. Oncology Orders can be placed only after capturing the height and weight values of the selected patient. On entering the height and weight values, the BSA and BMI are calculated. Enter the required header level details as explained under ‘Rx’ tab before prescribing oncology medications.

 

Placing Oncology Orders

 

  1. Enter the Regimen details in the mandatory (*) fields.

    Enter the following details to prescribe oncology medication(s) for the selected patient:

    1. Stage – Select the stage of the problem from the list. All the values defined in ‘Problem Stage’ master in PH code setup are listed in the list.

    2. Intention – Select the intention of the medication from the list. All the values defined in ‘Order Intention’ master in PH code setup are listed in the list.

    3. Regimen – Select the required regimen from the list provided. The regimens listing is based on the following conditions, Service registered against patient encounter must have regimen applicability defined.

    Regimens defined using Order Set functionality under Pharmacy category in the Order Entry and Results Entry module are associated to the service registered against patient encounter.

  2. On entering the Regimen details, the Drug Details/Unit Dose, Fluid Details and Admin Details are displayed.

    Enter the following details:

    1. Infusion Rate – The infusion rate is calculated by dividing the Fluid volume with Infuse over value. The displayed value can be modified. This field is mandatory. Infusion rate is specified in terms of volume against a time period that could be in minutes, hours or days.

    2. Admixture – This field is applicable for drug defined as IV Ingredients. The pharmacist has to prepare an admixture of the selected drug and fluid while dispensing. Select the check box to inform the pharmacist to prepare admixture.

    3. Build MAR - Medication Administration is applicable for all patient classes depending on the Parameter settings. For inpatients, Build MAR is default checked and enabled. For remaining patient classes based upon parameter settings it will be checked and enabled.

    4. Infuse Over – The ‘Infuse Over’ specified for the selected IV Fluid (in Drug Master Function) is displayed in hours and minutes format, by default. The Practitioner can specify the required period in numeric format of two digits. Based on the modified Infuse over value, the Infusion rate is calculated. The Unit of Measurement displayed corresponds to the time component value selected for Infusion Rate (Hours. Minutes or Days).

    The Infusion rate, admixture and infuse over displays only for drugs that are IV fluids or IV ingredients. For drugs that are normal drug, the following details are displayed.

    1. Frequency –The frequency specified for the selected drug in the drug master is displayed. Select the required value from the list. The listed values are Scheduled frequencies associated to the Order Category ‘Pharmacy’. If the selected frequency is scheduled, then [Schedule] is enabled. The button will be disabled for frequency as ‘PRN’ and ‘Once’. Click [Schedule] to view the schedule for the selected Frequency.

    2. Duration – Duration specified for the selected frequency while setting up the selected drug is displayed by default. The duration can be changed only in certain cases. If the selected frequency is of the nature ‘One Time’, then the duration is made default as ‘one’ as read only text. For frequency of nature ‘PRN’, the duration defaults as one day. This value can be changed to the required number of days. The practitioner can enter a value up to 999 days. For frequency of type ‘Frequency’, the drug setting value is displayed. This value can be changed to the required period. If the entered duration exceeds the duration limit defined in Order catalog setup, then the system displays message specified by you and depending on the setting of the order catalog appropriate action to warn or reject or authorization required is performed.

    3. Start Date - The current system start date/time are displayed, by default. The practitioner can modify, that is pre-date or post-date the medication starting date/time based on the setting in OR Parameter. This field is mandatory.

    4. End Date – The end date/time is calculated by the system by adding the specified duration with the start date/time. The end date and time cannot be modified.

    Enter the following fluid details:

    1. Fluid – The fluid specified for the selected drug in ‘Drug’ master function is displayed, by default. The practitioner can select other fluid either by enter the fluid name in alphanumeric format of 40 character maximum length or select it using [?] function. This field is mandatory. The search function list IV fluids that are to be used for cyto-toxic drug preparation. Select the required fluid. Only one fluid can be selected for administration with a drug.

    2. Volume - The fluid volume is made default based on the drug master setup for the selected fluid. The displayed value cannot be modified. This field is mandatory.

    On entering the fluid details, the practitioner has to record the ‘Administration details’ in ‘Admin Details’ part.

    Drugs that constitute the regimen are displayed. Clinician can place Oncology Orders only for orderable set as ‘Order Set’ for the order category ‘Pharmacy’. All the Drugs associated to the selected Order Set are listed in the result set. Drugs(s) associated to the selected order set are displayed in the prescription pad. Appearance of color-coded indicators against certain drugs denoted the following:

    Indicator

    Description

    Allergic –

    The icon indicates the drug as allergic. If the patient is allergic to the selected drug, then this icon displays. The practitioner can prescribe the drug after entering a valid overriding reason.

    On clicking the ‘Allergic’ icon, the ‘Allergy Indications’ window displays. The allergic description is displayed.

    Beyond Drug Dosage Limit -

    This icon indicates exceeding of drug dosage limit of the selected drug. Dosage limit of each drug can be set optionally. The set limit can be specific to age group and gender. Unit and daily dosage limits can be set for the drug. If the prescribed drug exceeds the set limit, then this icon displays against the drug. But, the practitioner can override the same by specifying valid reason.

    Duplicate Drug -  

    This icon displays on the extreme right side of the prescription pad. It indicates drug duplication. Duplication of drug is checked based on generic name. If the selected drug is already a part of patient’s active drug profile, then the prescribing practitioner can override the same by specifying valid reason. If the selected drug falls beyond the active drug profile period, then drug can be prescribed without specifying the reason.

    Click on ‘Duplicate Drug’ hyperlink to view the duplicate medication details.

    Override reason has to be entered for outpatient to prescribe a duplicate drug. But for inpatients and emergency patients, duplicate drug can be prescribed only for STAT frequency.

     

    Calculate By

    The Dosage option is selected, by default. Select the Dosage or Body wt/BSA option.

    Enter the following details:

    1. Body wt/BSA - Select the option to define the additive by Strength and by Quantity.

    2. Dose - Enter the Dose in numeric format of four digits maximum length and select the UOM from the list. The Strength UOM of the Drug is displayed, by default.

    3. /KG or M2 - Based on the Drug Master setup KG or M2 is displayed, by default. If Standard Unit Dose for the selected Item is defined in the Drug Dosage Limit function, where Calculate Dosage By field is selected as 'BSA'; the system automatically displays the value as BSA.

     

    The Total Dose per Administration is displayed. The system displays the dose based on the setup made in Drug Dosage Limit function. The Standard Dose is multiplied with BSA or the weight based on the Calculate Dosage By setup.

    Enter the following details:

    1. Dosage By - Based on the Drug Setup, if ‘Dosage By’ is ‘Strength’, the list contains Strength and Quantity. The practitioner can define dosage details either as Strength or Quantity. If ‘Dosage By’ is ‘Quantity’, the displayed value cannot be changed.

    2. Dosage By: Strength

    This value is displayed provided base strength is defined for the selected drug. The ‘Strength’ field displays the base strength value and unit of measurement. The practitioner can change only the strength value.

    • Dosage By: Quantity

    The base quantity and unit of measurement set for the selected drug are displayed for ‘Dose’ field. The practitioner can change both quantity and unit of measurement. The unit of measurement values is listed based on the following conditions:

    The unit of measurement defined with nature as ‘quantity’ and belonging to the list used for defining prescription base unit.

    The unit of measurement defined with nature as ‘Strength’ and having an equivalent value, which is a quantity.

    The unit of measurement defined as ‘Quantity’ and having an equivalent to a value that is of nature ‘Strength’ and belonging to the list used for defining prescription base unit.

  3. Click [Add New Drug] to add new order for the selected regimen while placing oncology medication orders. This button gets enabled on selecting the regimen from the list. The line level details of the new drug will be displayed under ‘Drug Details/Unit Dose’.

  4. Click [Confirm] to add the details to the prescription pad, [Cancel] to cancel a Drug from the prescription before releasing an oncology prescription, [Reset] to refresh the Drug Details section and bring back the original data, [Clear] to clear the Drug details section to allow you to select another Drug of the regimen.

  5. On confirming a Drug, it is displayed in the prescription pad as shown below with a unique color-code.

  6. Click [Review Rx] to preview the Oncology prescription details.

  7. Click [Release Rx].

  8. Click [Release Rx] to release the Oncology prescription.

  9. You are prompted with a message to print the Oncology prescription details on releasing. Click [OK] to print the prescription or [Cancel] to cancel the printing.

  10. On releasing the prescription, a unique Order ID is generated. Order details can be viewed under ‘Existing Orders’ function.
