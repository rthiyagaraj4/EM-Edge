Enter the following details:

 

  1. Ordering Facility - Select the Ordering Facility from the list. The logged in facility is displayed, by default. Select another Facility within the Enterprise. This is to filter Specimens listing based on Ordering Facility.

  2. Performing Facility - Select the Performing Facility from the list. The logged in facility is displayed, by default. This is used to filter Specimens based on Performing Facility.

  3. Option - Select an Option from the list. The Option ‘New List’ is selected, by default. The other values are Record Collection, Record Dispatch, Reprint Specimen, Reprint Dispatch Slip. This indicates the action to be taken on the specimen listing.

  4. New list – On selection of this option, the system will display orders for which specimen numbers are yet to be generated and printed.

  5. Reprint – This option allows reprinting specimens against orders for which specimen numbers are already generated and printed.  

  6. Record Collection - This option allows stamping the actual date and time of collection against each specimen collected.  

  7. Record Dispatch – This option allows marking all specimens that are sent to the lab as dispatched.

  8. Reprint Dispatch Slip - This option helps to reprint the dispatch slip.

  1. Department Location – Select the Department Location as Wing or Operating Theatre. Select the Location using [?] function based on the selected Department Location Type.

  2. Movement Location – Select the Location Type from the list. The values displayed are All, Clinic, Nursing Unit and Referral. All is displayed, by default. Select the Location using [?] function based on the selected Location Type.

  3. Order Type - The order types displayed correspond to Laboratory order category. Select the order types to specify the range.

  4. Status – Select the status as All, Future, Due and Overdue. All is displayed, by default.

  5. Priority - Select the order priority from the list provided. The values displayed are All, Stat, Urgent and Routine. 'All' is displayed, by default.

  6. Patient ID - Enter the Patient ID in format as defined in Master Patient Index module or use [?] function to search the required patient.

  7. Encounter ID - Enter the Encounter ID in alphanumeric format of 12 characters maximum length.

  8. Period – The date range is displayed, by default. Enter a period in date format of dd/mm/yyyy.

  9. Print Location - Select the Print Location from the list. The list displays Clinic and Nursing Unit. This is used to filter the Care Location based on the Type of Location. This field becomes mandatory on selecting Option as Reprint Dispatch Slip.

  10. Specimen No – Enter the Specimen Number in numeric format of 16 digits maximum length.

  11. Order ID - Enter the Order ID in alphanumeric format of 15 characters maximum length. The order ID is generated by the system.

  12. Ordering Practitioner  – Select the practitioner from the look up field using the function [?] to display the orders pertaining to the selected practitioner.
