Enter the following details:

  1. Report Type - Select the Report Type values from the list. The listed values are system defined. This is a mandatory field.

  2. Grouping Code - Enter the grouping code in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  3. Grouping Description - Enter the grouping description in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  4. Order Category - Select the Order Category from the list. The listed Order Categories are defined in the Order Category function in the OR Code Setup. This is a mandatory field.

  5. Print Report Format - Select the Print Report Format from the list. The listed values are system defined. This is a mandatory field.

Applicable Order Type

  1. ALL - Select the 'ALL' option to display all the details of applicable order Type.

  2. Specific - Select the ' Specific' option to accept the applicable order type in the selected order category.

    • The Applicable Order Types are listed. Select the 'Applicable' check box to accept the applicable order type in the selected Order Category.

  3. Enabled - Select the check box to enable the created record.
