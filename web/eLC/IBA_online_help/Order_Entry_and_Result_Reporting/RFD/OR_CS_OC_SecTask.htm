Secondary Task/Prep

This tab is used to associate Secondary Task/Preparation to be performed after the primary order is placed. The secondary Task/Preparation is applicable for individual Patient Class and the start time of the Secondary Task has to be defined.

 

  1. Type - The Order Catalogs with Order Category 'Treatment/Procedures', 'Miscellaneous', 'Monitoring', 'Dietary' and 'Patient Care' are displayed on selecting 'Type' as Orderable. The Interventions displayed on selecting 'Type' as 'Intervention' are populated from Intervention Setup in Clinician Access module.

  2. Patient Class Applicability - Select the check box corresponding to the Patient Class to apply the Secondary Task/Prep. The Secondary Task/Preparation is applicable to the selected patient class. The values displayed are - 'Day care', 'External', 'Emergency', 'Inpatient' and 'Outpatient'.

  3. [Select] Tab – Click [Select] tab to associate additional Order Catalogs and to define the Order Stage. Select the required alphabet to display Order Catalogs beginning with the selected alphabet.

  4. Order Catalog - The Order Catalogs are displayed in alphabetical order.  

  5. Stat Order Stage - Select 'Order' from the list to place a secondary order prior to the primary order. Select 'Appointment' if the secondary task is not an order, or if it is only a service/preparation to be placed before the primary order. Select 'Not applicable', if the relevant secondary task is not applicable.

In the corresponding text box, enter value in numeric format. The value entered denotes the number of hours or days before which the secondary task is to be carried out.

  1. Urgent Order Stage - Select 'Order' from the list to place a secondary order prior to the primary order. Select 'Appointment', if the secondary task is not an order, or if it is only a service/preparation to be placed before the primary order. Select 'Not applicable', if the relevant secondary task is not applicable to the particular 'Priority' of the order.

In the corresponding text box, enter value in numeric format. The value entered denotes the number of hours or days before which the secondary task is to be carried out.

  1. Routine Order Stage - Select 'Order' from the list to place a secondary order prior to the primary order. Select 'Appointment', if the secondary task is not an order, or if it is only a service/preparation to be placed before the primary order. Select 'Not applicable' if that particular secondary task is not applicable to the particular 'priority' of the order.

In the corresponding text box, enter value in numeric format. The value entered denotes the number of hours or days before which the secondary task is to be carried out.

  2. [Associate] Tab - Click [Associate] to view the selected Order Catalogs in the Associate Tab. You can modify the values in the Tab.
