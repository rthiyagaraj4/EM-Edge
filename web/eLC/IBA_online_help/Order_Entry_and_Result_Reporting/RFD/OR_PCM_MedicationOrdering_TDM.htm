TDM (Therapeutic Drug Monitoring)

A Practitioner can place TDM orders for the selected patient in this tab. The drugs prescribed to the patient that requires monitoring of dosage are listed. The drugs that are to be ordered appear in TDM tab, only if TDM applicable is checked in Drug masters and based on the TDM attributes set in the function ‘TDM Attributes’. Lab orders can be placed for the selected drug. The pharmacy orders with ‘frequency’ as ‘stat’ do not appear in TDM. You need to place an order previously to do the TDM and the patient must be an inpatient. The Discharge Medication check box must be cleared for the TDM.

 

Placing a TDM Orders

  1. Click ‘TDM’ tab to place TDM orders.

  2. Drug profile is made default on selecting the ‘TDM’ tab.

  3. Select ‘ Drug Profile’ option to view the active orders for the selected patient or select ‘ Random Toxicology’ to record the Drugs that are not recorded in Drug Profile.

    If Drugs are Inactive, lab orders can be placed in Random toxicology for the selected Drug.

    1. Click [GO] to display the list of Inactive Drugs or select the Generic Name and the Form of Drug from the list. The Generic Names for which TDM is applicable are displayed and the Form of Drugs defined in the Code Setup functions are displayed. On Selecting the details click [GO], the Drug details are displayed in the Poisonous Drugs frame.

    2. Click Drug Description hyperlink to record the TDM Request details.

      Enter the following TDM request details under ‘TDM Request’ section.

      1. Reason for request – Select the reason for requesting drug monitoring from the list. The listed values are defined in ‘Reason Codes’ function for ‘TDM Orders’. This field is mandatory.

      2. Clinical Comments - Enter the clinical comments in alphanumeric format.

      3. Smoking – Select the check box to indicate that the patient is a smoker.

      4. Takes Alcohol – Select the check box to indicate that the patient consumes alcohol.

      5. Pregnant – Select the check box to indicate that the patient is currently pregnant.

      To place TDM orders, click the required ‘Drug/Dosage’ hyperlink. The TDM attributes set for the selected drug are displayed. The following details are displayed under ‘TDM Attributes’ section.

      1. Date/Time – The scheduled medication date/time is displayed as per the prescription. The schedule times are displayed against each date based on the frequency, duration and order start date.

      2. Dosage – The prescribed dose of the selected drug is displayed.

      3. Sample Types - Select the sample type from the list. The listed values are Peak, Trough and Random. On selecting a sample type, the Sampling timing specified in TDM attributes function is displayed as default. The practitioner can modify the timing; enter the required sampling time in days, minutes or hours.

      4. Sampling Duration – Enter the Sampling Duration in alphanumeric characters of three maximum lengths. Select the Duration in Minutes, Hours or Days.

      5. Lab Orders – Select the laboratory order catalog associated to the selected sample type from the list.

    3. Click [Record] to place TDM Orders or [Clear] to clear the entered details.

    4. The Resulting of the placed orders is recorded in ‘Order Entry and Result Reporting’ module in the function ‘Result Reporting’.

    The orders that are previously placed using Rx get displayed on clicking [GO]. The TDM order can be placed only once for the selected Drug.  On releasing a TDM Order, based on ‘Verify TDM Order’ set in PH Parameter for Facility order status is marked.

    The ‘Frequency’ and ‘Age Group’ should be associated for the drug in the Drug Masters. The Age Group and the Lab Orders must be associated for the selected Drug using ‘TDM Attributes’ Master function. The Lab Orders must be associated for ‘Sample Types’,  ‘Peak’ ‘Trough’, and ‘Random’ using the TDM attribute function. The ‘Age Group’ association helps the Practitioner to place TDM Order for the patient in associated Age Group.
