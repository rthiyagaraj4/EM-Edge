Complete Medication Order

From this function you can Complete medication orders placed for patients.

The pharmacist completes medication orders placed for a patient using this function. This function is available only to practitioner of Practitioner Type ‘Pharmacist’. The pharmacist selects the required medication orders and views them. Orders that are dispensed are set to complete in the function.

Completing Medication Orders

From the CA Transactions > Clinician Access > Desktop Menu > Task, select Complete Medication Order.

The Complete Medication Order screen gets displayed.

  1. Medication Orders for the default date range is displayed or enter the query criteria and click [Search] to display specific orders. Click [Clear] to clear the displayed data.

    1. Location Type - The value displayed is pre-defined.

    2. Location - Pharmacy is displayed, by default.

    3. Order Type - Select the Order Type from the list provided. The values displayed in the list correspond to the 'Pharmacy' order category.

    4. Priority - Select the priority from the list. The values displayed are All, Routine and Urgent.

    5. Period [From-To] - The date range is displayed, by default. Specify the order date range in date format of dd/mm/yyyy.

    6. Patient Class - Select the Patient class from the list. The values displayed are All, Daycare, Emergency, External, Inpatient and Outpatient.

    7. Patient ID - Enter the Patient ID or select Patient ID using [?] function.

    8. Order ID - Enter the Order ID in alphanumeric format of 15 characters maximum length.

    9. Encounter ID - Enter the Encounter ID to filter the drug listing based on Encounter ID.

  2. The Order details are displayed. Click Order Date hyperlink to complete medication orders. Click Order Details hyperlink to view Order details.

    1. The screen displays Order ID, Order Category, Order Type, Priority, and Patient details, Encounter ID, Order Date Time, Order Status, Ordering Location, Ordering Practitioner, Ordering Facility, Performing Facility, Order Format, Clinician Comments and Order Placed By. The screen also displays the Order Catalog, Priority, Frequency, Start Date/Time, End Date/Time, Quantity, Status, Formats and Comments.

    2. Click Format hyperlink to view details in the Order Line Format screen. Click [Close] to close the Order Line Format screen.

    3. Click [Schd] hyperlink to view the Schedule Frequency of the order and [Close] to close the Schedule Frequency screen.

    4. Click Preps, Dept Instrn, Pat Instrn hyperlink to view details if recorded while creating the Order Catalog.

    5. Click [View Audit Trail] to display View Audit Trail screen and view audit trail details. Click the Remark (R) hyperlink to display remarks for the order. Click [Close] to close the remark screen. Click [x] to exit from the View Audit Trail screen.

    6. Click [Close] to close View Order screen.

    1. The Completion – Pharmacy Order screen is displayed.

    2. The screen displays Order ID, Order Category, Order Type, Priority, Patient, Encounter ID, Order Date time, Order Status, Ordering Location, Ordering Practitioner, Ordering Facility, Performing Facility, Order Format, Clinical Comments, Order Placed By and Performing Location.

    3. Select the Complete check box to display the Complete Remarks screen.

    4. The current date and time is displayed. Select the date from the calendar function.

    5. Enter the Remarks. This field is mandatory.

    6. Click [OK] to save and exit from the screen.

    7. Click [Complete] to complete the order or [Close] to exit from the screen without recording completion of the medication order.
