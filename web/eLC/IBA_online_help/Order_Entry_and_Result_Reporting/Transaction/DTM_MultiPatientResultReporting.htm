Multi Patient Result Reporting

From this function you can Perform result reporting for more than one patient at a time.

The patients can be grouped together and the result reporting can be done. The patients can be grouped based on the Order Start Date, Location or the Order Catalog. The Multi Patient Result Reporting is done based on the resulting types set in Discrete Measure function.

Reporting Multi Patient Results

From the CA Transactions > Clinician Access > Desktop Menu > Order Management, select Multi Patient Result Reporting function.

The Multi Patient Result Reporting screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to display the Orders matching the entered criteria or [Clear] to clear the entered details.

  3. On searching, the Orders matching the entered criteria are displayed. The Discrete Measures associated to the Order Catalogs are displayed as tabs. Click on the Discrete Measures tabs to select the orders for resulting.

  4. Select the check boxes under 'Select' to select the orders for resulting. To select all the orders, select the check box beside 'Select' heading.

  5. Click [Result] to record the result or [Cancel] to cancel the request.
