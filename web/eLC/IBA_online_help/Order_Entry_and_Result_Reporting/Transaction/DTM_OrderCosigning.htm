Order Co-Signing

From this function you can Co-sign orders.

Co-signing is a method of intimating the attending practitioner of any orders placed by other practitioners who are not directly related to the patient. Co-signing does not hinder the processing of the order. It is used to purely intimate the attending practitioner of any orders placed against patients.

Co-signing Orders

From the CA Transactions > Clinician Access > Desktop Menu > Pending Activities > Reviews, select Order Co-signing function.

The Order Co-Signing screen gets displayed.

  1. Orders for Co-Signing are displayed for the default date range. Enter the query criteria and click [Search] to display specific orders for Co-signing. Click [Clear] to clear the displayed data.

    1. Location Type – ‘All’ is displayed, by default. Select the required Location Type from the list. The values displayed are All, Clinic, Day care, Emergency, Nursing Unit and Referral.

    2. Location – ‘All’ is displayed, by default. Depending on the value selected for Location Type, the [?] function displays Locations. If Location Type is selected as ’Clinic’, [?] function displays all the clinics of the logged in facility to which the logged in user has access. Similarly, if the Location Type is ‘Nursing Unit’, [?] function displays all the nursing units of the logged in facility to which the logged in user has access.

    3. Patient ID – Enter the patient ID in alphanumeric format of maximum length as specified in the MP Parameter.

    4. Period [From-To] – The date range is displayed, by default. Select the desired Period range using the Calendar function to view the details.

    1. Status by Priority - The number of orders to be cosigned against each priority are displayed. The displayed priorities are hyperlinked. The total number of orders to be cosigned of all the displayed priorities is shown against “All” hyperlink. Select a priority to view orders to be co-signed.

    2. Status by Category – The number of orders to be cosigned against each category are displayed. The displayed categories are hyperlinked. The total number of orders to be cosigned for all the displayed category is shown against “All” hyperlink. Select an order category to view orders to be cosigned.

    3. Cosign – Select the check box to select orders for cosigning. All the check boxes are cleared, by default. To cosign all the displayed orders, select the check box provided before ‘Cosign’ heading.

    4. Priority - The priority of the order is displayed. The priorities displayed are Routine, Stat and Urgent.

    5. Ordered At - The Date and Time of ordering are displayed.

    6. Order Catalogs - The order catalogs are displayed. The catalog is hyperlinked. Click on the hyperlink to view order details.

    7. Ordering Details – The Ordering details displayed are the Patient and Encounter details. The Patient details displayed are name of the Patient, Gender, Age and Patient ID. The Encounter details displayed are Encounter ID and Location details.

  2. Click Order Catalog hyperlink to view the Order details. Click All the pharmacy orders are displayed with dosage and frequency details.

    1. The screen displays Order ID, Order Category, Order Type, Priority, and Patient details, Encounter ID, Order Date Time, Order Status, Ordering Location, Ordering Practitioner, Ordering Facility, Performing Facility, Order Format, Clinical Comments and Order Placed By. The screen also displays the Order Catalog, Priority, Frequency, Start Date/Time, End Date/Time, Quantity, Status, Formats and Comments.

    2. Click Format hyperlink to view details in the Order Line Format screen. Click [Close] to close the Order Line Format screen.

    3. Click [Schd] hyperlink to view the Schedule Frequency of the order and [Close] to close the Schedule Frequency screen.

    4. Click Preps, Dept Instrn, Pat Instrn hyperlink to view details if recorded while creating the Order Catalog.

    5. Click [View Audit Trail] to display View Audit Trail screen and view audit trail details. Click the Remark (R) hyperlink to display remarks for the order. Click [Close] to close the remark screen. Click [x] to exit from the View Audit Trail screen.

    6. Click [Close] to close View Order screen.

  3. Select the check box against the required orders for Co-signing and click [Cosign] to confirm Co-signing or click [Clear] to clear the selected orders.

If 'Cosign Required for the Exams placed from Radiology' is selected in 'RD Parameters for Facility' function, the new/additional orders placed from radiology department are listed in the ordering and attending practitioner's home page. The Order Catalog setup is not applicable for New or Additional Orders placed through Radiology Information Systems module.

The additional orders placed from the following functions Record Exam Start/Completion, Departmental Review, Pending Exam for authorization and Register Examinations of radiology are listed for co-signing.

 

If the same ordering practitioner places the additional orders from Radiology Information Systems module, the order is listed for cosigning for the attending practitioner.

If different practitioner places the additional orders from Radiology, the order is listed for the attending practitioner and ordering practitioner.

 

The radiology icon is displayed and on mouse over the details Ordered from, Reason for ordering if recorded are displayed, these details are static.
