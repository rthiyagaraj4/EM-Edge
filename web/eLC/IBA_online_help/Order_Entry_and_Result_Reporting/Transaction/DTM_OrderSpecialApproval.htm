Order Special Approval

From this function you can Verify orders pending for Special Approval.

This function is used by senior practitioners to verify any orders that require special approval before it is being processed.

Approving Orders

From the CA Transactions > Clinician Access > Desktop Menu > Pending Activities > Verifications, select the Order Special Approval function.

The Order Special Approval screen gets displayed.

  1. Orders pending for Special Approval is displayed for the default date range. Enter the search criteria and click [Search] to display specific orders requiring Special Approval. Click [Clear] to clear the displayed data.

    1. Location Type – All is displayed, by default. Select the required Location Type from the list. The values displayed are All, Clinic, Day care, Emergency, Nursing Unit and Referral.

    2. Location – All is displayed, by default. Select the Location from [?] function. The Locations displayed for selection are based on the selected Location Type.

    3. Patient ID – Enter the Patient ID in alphanumeric format of maximum length as specified in the MP parameter or select the Patient ID from [?] function.

    4. Period – Select the date range using Calendar function to display orders within the specified date range.

    The screen displays orders based on two categories on the left hand side of the screen. Status by Priority and Status by Category.

    1. Status by Priority - The number of orders to be approved against each priority is displayed. The displayed priorities are hyperlinked. The total number of orders to be approved of all the displayed priorities is shown against 'All' hyperlink. Select a priority to view orders to be approved against it.

    2. Status by Category – The number of orders to be approved against each category are displayed. The displayed categories are hyperlinked. The total number of orders to be approved of all the displayed categories is shown against “All” hyperlink. Select a category to view orders to be approved against it.

    The following details are displayed on the right hand side of the screen.

    1. Approve – Select the check box against the required order for approving. To approve all the displayed orders, select the check box provided at the Approve heading.

    2. Priority – The priority of the order is displayed. The priorities displayed are Routine, Stat and Urgent.

    3. Ordered At – The date and time of ordering are displayed.

    4. Order Catalogs – The order catalog is displayed and hyperlinked. Click the hyperlink to view the order details.

    5. Order Details – The Order details displayed are the Name of the Patient, Gender, Age and Patient ID, Encounter Details, Location Type and Location,

    6. Charge Dtls – Click the hyperlinked View/Edit to view or edit the Charge Details of the placed order.

  2. Click Order Catalog hyperlink to view details of the selected Order Catalog. Click [Close] to exit from the View screen and the main screen is displayed.

    1. The screen displays Order details recorded while placing the Order. Click Order Format hyperlink to view details of the Order Format screen.

    2. The Order Catalog, Priority, Frequency, Start Date and Time, End Date and Time, Quantity, Status and Formats are displayed.

    3. Click Format hyperlink to view Order Line Format screen. Click [Close] to exit from the screen.

    4. Click Preps hyperlink to view Preparation Instructions for the patient. Click [Close] to close the Preparation Instruction screen.

    5. Click Dept hyperlink to view Department Instructions. Click [Close] to close the

    6. Click Pat. Instrn hyperlink to view Patient Instruction details. Click [Close] to close the Patient Instruction screen.

    7. Click [View Audit Trail] to display View Audit Trail screen and view audit trail details. Click the Remark (R) hyperlink to display remarks for the order. Click [Close] to close the remarks screen. Click [x] to exit from the View Audit Trail screen. Click [Close] to close the screen. Click [Close] to exit from the View screen.

  3. Click View/Edit hyperlink under Charge Details against the required record to view and record charge details. Click [Cancel] to exit from the Charge Details screen and the main screen is displayed.

    The screen displays Order details recorded while placing the Order. The screen displays the Order Catalog, Total Charge, Patient Payable amount, Patient Paid, Included, Approval Regd and Charge Dtls. Click [Record] to record charge details. Click [Cancel] to close the Charge details.

  4. Select the check box against the required orders to Special Approve.

  5. Click [Approve] to confirm Approval or [Clear] to clear the selected orders.
