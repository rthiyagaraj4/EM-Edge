Review Results

From this function you can Review or Forward Results that are resulted.

Using this function, practitioners can get online status of resulted orders and take decisions on patients. Results are posted for review based on parameters set for Attending and Ordering practitioners.  

System allows Reviewing Result based on the following validations.

If the Ordering Practitioner and Attending Practitioner is Same but the Ordering Practitioner and Reporting Practitioner is Different, the reports are posted to the Ordering Practitioner for Review.

If the Ordering Practitioner and Attending Practitioner is Different and the Ordering Practitioner and Reporting Practitioner is Different, the reports are posted to the Ordering Practitioner and Attending Practitioner for Review.

If the Ordering Practitioner and Attending Practitioner is Different but the Ordering Practitioner and Reporting Practitioner is Same, the reports are posted to the Attending Practitioner for Review.

If the Ordering Practitioner and Attending Practitioner is Same and the Ordering Practitioner and Reporting Practitioner is Same, the reports are not posted for review.

Reviewing Results

From the CA Transactions > Clinician Access > Desktop Menu > Pending Activities > Reviews, select the Review Results function.

The Review Results screen gets displayed.

  1. Orders pending for Review is displayed for the default date range. Enter the query criteria and click [Search] to display specific orders for Review. Click [Clear] to clear the displayed data.

    1. Patient ID – Enter the Patient ID or select the Patient ID using [?] function.  

    2. Normalcy Indicator – Select the Normalcy Indicator from the list. The displayed values are Normal, Abnormal and Critically Abnormal.  

    3. Period – The date range is displayed, by default. Enter a period in date format of dd/mm/yyyy.

    4. Priority – Select the Priority from the list. The values listed are Routine, Stat and Urgent.

    5. Result Type – Select the Result Type from the list. The values listed are All, Blood Transfusion, Laboratory, Radiology, Treatment and Patient Care.  

    6. Event Class – Select the Event Class from the list. The values listed are from the clinical repository.

    7. Order By – Select the value from the list. This is to group the result listing. The values listed are ‘Patient & Event’ and ‘Event’. This field displays ‘Patient & Event’, by default.

    1. Check box – Select the check box to indicate that the Event is reviewed. The check box is cleared, by default.  

    2. Event - The Resulted Event is displayed.  

    3. Result - Depending on the result nature of the clinical event, result values are displayed. Numeric values are listed directly on the screen.

    4. Reported By – The Reporting Practitioner’s name is displayed.  

    5. Reported At – The Date and Time of Reporting is displayed.  

    6. Review Type – Role of the logged in Practitioner with reference to the order is displayed as either Attending or Ordering Practitioner.  

    7. Ordering Practitioner – The name of the ordering practitioner is displayed.

    8. Forwarded By – The name of the person who forwarded the result is displayed.

  2. Click Result hyperlink to view the resulted comments recorded.

  3. Click icon to view past results and compare with current result. A message is displayed. Click [OK] to view the Abnormal Results in the Flowsheet. Click [Cancel] to view the Flowsheet of results recorded. Click icon to view of the graph of results recorded.

  4. Select the check box against the required Results. Remarks hyperlink is displayed. Click the Remarks hyperlink to enter the Remarks. Click [OK] to save the Remarks and exit from the Review Remarks screen. To Review all the displayed orders, select the check box provided in the heading. The Review Remarks screen is displayed to enter the Remarks. Enter the Remarks and click [OK] to save the remarks and exit from the screen.

  5. Click [Complete Review] to complete the review or click [Forward] to forward the report to another practitioner.

  6. On clicking [Forward], Forward to Practitioner screen is displayed. Enter the name of the Practitioner in the Search field and click [Search] to search for the required practitioner. Select the check box against the required Practitioners to forward the Result. Click [OK] to save forwarding the Order or [Close] to exit from the screen. On clicking [OK], a message is displayed ‘Do you want to Complete the Review?’. Click [OK] to complete review.
