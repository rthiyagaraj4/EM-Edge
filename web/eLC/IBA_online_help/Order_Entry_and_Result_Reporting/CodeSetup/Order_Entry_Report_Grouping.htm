Order Entry Report Grouping

From this function you can Generate standard Order Request Reports and Query for the existing records.

Order Entry Report Grouping is used to generate the Order Requests based on the setup made in this function.

Generating Order Request Reports

From OR Code Setup> OR Other Master Setup, select the Order Entry Report Grouping function.

The Order Entry Report Grouping screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to record the generated Order Request Reports or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Order Request Report requests.
