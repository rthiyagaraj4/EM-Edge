Index Order Catalog

From this function you can Create an association of Order Catalogs and Order Sets, which are indexed, to multiple services. You can Dissociate the Order Catalogs and Order Sets associated to the Service and Query for a desired record.

Multiple Order Catalogs are associated to the Service using this function. The services defined in the Application Masters module are listed. The services must be primary services for a Nursing Unit, Clinic, Procedure Unit and Day care unit. The Order Catalogs, where the ‘index’ check box is selected gets listed. You can select an Order Catalog, where the ‘Nature’ is selected as ‘Individual’ or ‘Panel’. You can also associate an indexed Order Set to the Service. This function contains two tabs, namely, Select and Associate. The Select tab is selected, by default and is used to select and associate the Order Catalogs and Order Sets. Using the Associate tab you can view the Order Catalogs and Order Sets associated to the Service.

Creating an association of Order Catalogs to a Service

From the OR Code Setup, select the Index Order Catalog function.

The Index Order Catalog screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select a Service by clicking [?] function. This field is mandatory.

  3. The Select Tab is displayed, by default. English alphabets from 'A to Z' are displayed along with 'Others' and hyperlinked. Click the required alphabet hyperlink to associate Order Catalogs or Order Sets of that particular alphabet to the Service. Click ‘Others’ hyperlink to select and associate Order Catalogs beginning with numeric or special characters.

  4. Select the check box against the Order Catalog or Order Set to associate it to the Service.

  5. Click Associate Tab to view the Order Catalogs or Order Sets associated to the Service. Clear the check box to dissociate relevant Order Catalogs or Order Sets.

  6. Click [Apply] to associate or dissociate the Order Catalog or Order Set to the Service or [Reset] to clear the entered data.

  7. Click [Query] to search for an existing record.

 

The Order Catalogs and Order Sets with the check box ‘index’ selected are listed.
