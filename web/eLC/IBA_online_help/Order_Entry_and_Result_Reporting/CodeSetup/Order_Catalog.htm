Order Catalog

From this function you can Create an Order Catalog and Query for an existing Order Catalog.

The Order Catalog function is used to define various Order Categories and its characteristics.  Individual orderable is defined as a Order Catalog. The Order Catalogs are placed for a clinical analysis of a patient. The cornerstone of an Order Catalog is an Order Category.  The characteristics defined for an Order Catalog influence the ordering and order processing pattern.

Creating an Order Catalog

From the OR Code Setup, select the Order Catalog function.

The Order Catalog screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Enter the details in the following Tab pages:

Characteristics

Timing

Frequency Applicable

Order Reviews

Settings

Instructions

Charting

Secondary Task/Prep

Turn Around Time for Priority

  1. Click [Apply] to save the records or [Reset] to clear the entered data.

  2. Click [Delete] to delete a queried record.
