Wait Time for Pending Orders Tab

From this function you can Define waiting time for colors Green, Yellow and Red for pending Order Catalogs that require appointments.

The color indicators (Green/Yellow/Red) appear on comparing the order start time with the ‘Wait time’ defined for the Order Catalog. The schedule priority check box must be selected for the Order Catalog in the Order Category function. If the time allotted for the Order Catalog exceeds and the order is not processed, you can schedule an appointment in Appointment Schedule module for the pending order in ‘Schedule Appointments’ function. The color indicators displayed in this function under ‘pending order schedule’ tab.

Defining Waiting Time for Pending Orders

From the OR Admin Setup, select the Wait Time for Pending Orders Tab function.

The Wait Time for Pending Orders Tab screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the OR Printing Rule or [Reset] to clear the entered data.
