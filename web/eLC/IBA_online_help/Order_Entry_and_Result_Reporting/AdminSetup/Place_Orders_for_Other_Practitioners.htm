Place Orders for Other Practitioners

From this function, you can Assign rights to Practitioners to place order for another Practitioner.

While placing an order, the logged in practitioner gets a screen wherein the ordering practitioner can be selected.    

Placing Orders for Other Practitioners

From the OR Admin Setup, select the Place Orders for Other Practitioners function.

The Place Orders for Other Practitioners screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the ‘Allowed Type’ from the list of values. The listed values are Practitioner, Practitioner Type and Responsibility. This is a mandatory field.

  3. The Select Tab is displayed, by default on selecting the 'Allowed Type' as Practitioner and Responsibility. The screen displays alphabets from ‘A to Z’ and ‘Others’ which are hyperlinked. Each alphabet contains data relating to the selected 'Allowed Type'. The ‘Others’ hyperlink contains values relating to the selected 'Allowed Type' which begins with characters other than the alphabets. Click on the required alphabet and the values beginning with the selected alphabet for the respective Allowed Type are listed.

  4. Select the check box against the required Practitioner/Responsibility to allow placing orders for other practitioners.

  5. Click 'Associate' Tab to view the selected Practitioners/Responsibilities. Clear the check box to dissociate the association.

  6. If the 'Allowed Type' is selected as Practitioner Type, all the Practitioner Types are listed. Select the check box against the required Practitioner Type to allow practitioners with the selected Practitioner Type to place orders for other practitioners.

  7. Click [Apply] to save the records or [Reset] to clear the entered data.
