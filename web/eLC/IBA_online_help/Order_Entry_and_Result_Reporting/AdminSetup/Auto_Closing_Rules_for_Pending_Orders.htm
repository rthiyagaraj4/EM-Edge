Auto Closing Rules for Pending Orders

From this function you can Define rules for closing pending orders of an Order Type.

The pending orders are closed automatically by the system on elapse of the specified duration. Closing rules are defined for ‘Unprocessed’ and ‘In process’ orders. The unregistered orders are termed as ‘Unprocessed Orders’ and orders with the status ‘Registered’, ‘Partially Resulted’ are considered as ‘In Processed Orders’.

Creating Auto Closing Rules for Pending Orders

From the OR Admin Setup, select the Auto Closing Rules for Pending Orders function.

The Auto Closing Rules for Pending Orders screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Order Category from the list. These values are pre-defined and set. This is a mandatory field.

  3. Select the Order Type from the list. The Order Types defined in the function ‘Order Type’ are listed. This field is mandatory.

  4. Enter the number of days in the field ‘Close Unprocessed Orders after’ in the numeric format of three digits maximum length. Unprocessed orders are closed after the specified period.

  5. Enter the number of future order in three digits maximum length in the field ‘Future Order’.

  6. Select the Close Billed Order check box, to close the billed for the selected order.

  7. Select the Reverse Charge check box to reverse the order.

  8. Enter the number of hours to ignore the order in the field ‘Ignore Orders Placed Within (Hours)’.

  9. Enter the number of days in the field ‘Close In Process Orders after (Days)’ in the numeric format of three digits maximum length. Unprocessed orders are closed after the specified period. 'In Process' orders are closed after the period specified.

  10. Select the Close Billed Order check box, to close the billed for the selected order

  11. Select the Reverse Charge check box to reverse the order.

  12. Click [Apply] to save the Auto Closing Rules for Pending Orders or [Reset] to clear the entered data.

  13. Click [Query] to search for existing Auto Closing Rules for Pending Orders.
