Event wise Segments

Select the ‘Event wise segments’ Tab.

  1. Event Type – Select the Event Type from the list. The list is system-defined. The segments for the selected Event Types are also system-defined. On selecting the Event Type, the Segment ID's are displayed. The optional segments that were ‘Selected’ under the ‘Segments’ tab are also included in the displayed Segments list.   

  2. Segment ID – The ID of the segment is displayed.

  3. Segment Name – The name of the Segment is displayed.

  4. Optional – The check boxes are disabled. Segments may be mandatory or optional in the message. The system displays the optional segments by placing a check mark in the corresponding check box.

  5. Select - Some of the Segments for an Event Type are mandatory (and the ‘Select’ check box for these segments is disabled) and the user cannot delete or disable these mandatory segments. Only optional Segments can be disabled or enabled. Select the ‘Select’ checkbox to enable the required (optional) segments.

  6. Click [Apply] to save the record.
