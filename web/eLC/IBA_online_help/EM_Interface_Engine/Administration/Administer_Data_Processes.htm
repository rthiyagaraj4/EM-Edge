Administer Data Processes

From this function you can Administer Data Processes.

This function is used for submitting, changing, running and removing background jobs. The background jobs or data processes are responsible for building and formatting HL7 interface messages that are required to be send out for various events based on the event occurrence in Enterprise Management.

Administering Data Processes

From Administration, select Administer Data Processes.

The Administer Data Processes screen gets displayed.

  1. The Job Queue is displayed, if recorded.

    This displays the information regarding the currently running processes. The login name of the user, the last execution date with time, the next execution date with time and the status of the process are displayed.

  2. Enter details in the mandatory (*) fields. <Related Field Description>

• Click [Submit] for saving the entered details, or click [Cancel]. Other buttons displayed are explained as below.

• [Change] – Select the required process from the job queue. The process is now displayed in the ‘Process’ field under the title ‘Process Details’. Make the necessary changes for the process by selecting the desired options. Click [Change]. When you click [Change], the changes made are saved automatically.

• [Run] – Select the required process from the job queue. Click [Run] to re-run the process. The status of the process changes accordingly.

• [Remove] – Select the required process from the job queue and click [Remove]. The process no longer appears in the job queue list.
