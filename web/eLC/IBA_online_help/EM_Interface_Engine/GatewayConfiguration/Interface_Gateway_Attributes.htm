Interface Gateway Attributes

From this function you can Define Interface Gateway Attributes. You can Query for an existing record and Delete a queried record.

Defining Interface Gateway Attributes

From Gateway Configuration, select Interface Gateway Attributes Function.

The Interface Gateway Attributes screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Interface Gateway Attributes.

  4. Click [Query] to search for an existing record.

  5. Enter the query criteria and click [Execute].

  6. The ID, Protocol Link Name, Protocol ID and Data Group ID are displayed.

  7. Select the ID hyperlink to view or modify details.

  8. Click [Apply] to save the record.

  9. Click [Delete] to delete a queried record.
