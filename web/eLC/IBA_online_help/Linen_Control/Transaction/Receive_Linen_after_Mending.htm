Receive Linen After Mending

From this function you can Record the linen item details that are received from the agencies after mending. You can View the request details.

The Receive Linen after Mending function is used to record the linen item details, which are received from laundry or external agencies after mending. The quality of the linen items has to be verified. The items can be returned to laundry, if the quality check fails. You have to select a Source Type and Source to record the linen details. The values selected are defaulted and displayed in the transaction functions.

Receiving linen after mending

From the Linen, select the Receive Linen After Mending function.

The Receive Linen After Mending screen gets displayed.

  1. Click the Calendar function to select the Request Date. System calculates and displays the time and date seven days prior to the system date. You can view the requests placed from the date selected. This is a mandatory field. <Related Field Description>

  2. Click [Search] to view the Linen items received after Mending. The Requests placed are displayed in ascending order based on the Request Number.

  3. Click Request Number displayed as a hyperlink to ‘Receive’ or ‘View’ the Request placed for mending linen.

  4. Click ‘Receive’ hyperlink to mark the items as received by the facility after mending.  The Request Number, Request Date, Raised By and Received Date are displayed. The linen items, sent quantity, received quantity, current received quantity and remark are displayed. <Related Field Description>

  5. Click ‘View’ hyperlink to view the details of the request placed for mending linen. You can view details, such as, Request Number, Request Date, Raised By and Request Status. The Linen Item, requested quantity, issued quantity, pending quantity and remark are displayed in tabular format.

  6. The Stock Status and Work Status of the linen items in the laundry are displayed. You can choose to view the status of the linen items in laundry by selecting a source Type and a Source. The current system is displayed. Click [Go] to search for the details in the respective Source and Source Type.
