Issue of Linen

From this function you can Issue new linen items to a Nursing Unit/Clinic/Department based on the request placed. You can View the requests placed for new linen items from a Nursing Unit, Clinic or Department.

The Issue of Linen function is used to issue new linen items to a Nursing Unit/Clinic/Department based on the request placed. The cleaned linen items are issued back to a concerned source based on the request. Multiple linen items can be issued against a single request placed from a source.

Issuing new linen items

From the Linen, select the Issue of Linen function.

The Issue of Linen screen gets displayed.

  1. Click the Calendar function to select the request date. The requests placed on the respective date for the linen items are listed. <Related Field Description>

  2. Click [Search] to search for the requests placed.

  3. The details are displayed with the Request Number as a hyperlink.

  4. Click Request Number to Issue linen items or click View to view the details.

    Click ‘View’ hyperlink to view the details of the request placed for new linen. You can view details, such as, Request Number, Request Date, Source Type, Source, Raised By and Request Status. The Linen Item, Requested Quantity, Issued Quantity, Pending Quantity and Remark are displayed in tabular format.

    The Source Type, Source, Request Number, Request Date, Raised By, Issue Date, Linen Item and Requested Quantity for the selected Request are displayed based on the request placed using the ‘Request for Linen’ function. You cannot modify the data.

    Enter the following details to issue requested linen items.

    1. Issued By – Enter the name of the person in alphabetical format of 30 characters maximum length. This indicates the name of the person who is authorized to issue linen items against the request placed. The name of the logged in user is displayed, by default. This is a mandatory field.

    2. Issued Quantity – Zero is displayed as the initial value. If any partial issues are made against a Request Number, the partial quantity issued gets displayed.

    3. Current Issue Quantity – Enter the quantity in numeric format of 10 digits maximum length. This indicates the number of linen items issued against the request placed.

    4. Remark – Click Remark hyperlink to enter the remarks, if any. Click [OK] to save the record and exit the window.

    5. [Select] - Click [Select] to display the details of Linen Item in a tabular format. The details include Linen Item, Requested Quantity, Issued Quantity, Current issue Quantity and Remark. Linen Items and Remarks are displayed as hyperlinks.

    6. Linen Item - Click Linen Item hyperlink to display the details of the selected linen item, requested quantity, issued quantity, current issue quantity and remark in the respective fields.

    7. Remark - Click Remark hyperlink to view the entered remark. To edit the remark select the Linen Item hyperlink and edit the respective ‘Remark’ field. Click [OK] to save the details.

    8. [Submit] - Click [Submit] to issue new linen items against the request placed from a Source Type. A confirmation message ‘Please note your Request Number’ is displayed. Click [OK] and the control shifts to the Reference Number screen to issue or view the request.

  5. The Stock Status and Work Status of the linen items in the laundry are displayed. You can choose to view the status of the linen items in laundry by selecting a source Type and a Source. The current system is displayed. Click [Go] to search for the details in the respective Source and Source Type.
