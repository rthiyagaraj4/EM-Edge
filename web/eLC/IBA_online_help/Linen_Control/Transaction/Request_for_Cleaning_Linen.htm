Request for Cleaning Linen

From this function you can Place a Request to a laundry department to clean the Linen Items of a Facility. You can Cancel or View the Request List. You can also Confirm the pending count of linen items.

The ‘Request For Cleaning Linen’ function is used to place a request for cleaning soiled, infected or seriously infected linen items by the Nursing Units/Clinics/Departments to a laundry. You are allowed to view, modify or cancel the requests placed according to the requirement. You have the option to segregate the linen items as ‘Soiled’, ‘Infected’ and ‘Seriously Infected’, so that, while cleaning the linen, the laundry department ensures to clean the linen items separately based on the remarks recorded. This function also helps to schedule a delivery date to track the linen items given for cleaning to maintain a record. You can define if the wards should confirm the Dirty Linen counted by Laundry in the ‘Setup Parameters for Facility’ function. The Nursing Unit, Clinic or Department is authorized to confirm the dirty linen counted by the laundry department, if the laundry department is allowed to enter the details of the dirty linen. You have to select a Source Type and Source from where the linen request is placed. The values selected are defaulted and displayed in the transaction functions.
