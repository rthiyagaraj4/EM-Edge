Write-Off

From this function you can Write-Off to write-off the linen that was sent out to external agencies for mending or cleaning and is not received from them. You can View the Stock Status and Work Status of the Linen Items.

The Write-off function is used to write-off the linen items given to the laundry for washing and not received due to loss of linen items, permanently dirty linen, which cannot be re-used after laundry, affected linen with infectious disease, and so on. Linen given for cleaning, but not cleaned properly after repeated washing with proper solutions and chemicals can be written-off using this function. Linen torn while cleaning or that cannot be re-used forever is also written-off using this function. Reference Number is allotted to each written-off linen for record and to keep track of the linen stock within the facility. Linen is categorized under In-use, Soiled or Loaned and is written-off due to various user-defined reasons. Select the source and Source Type to write-off the linen details.

Write-off Linen

From the Linen > Linen Transaction, select the Write-Off function.

The Write-Off screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Submit] to write-off the details.

  3. A confirmation message ‘Please note your Request Number:’ is displayed on confirming clicking [Submit]. Click [OK] after noting the request number.

  4. The Stock Status and Work Status of the linen items in the laundry are displayed. You can choose to view the status of the linen items in laundry by selecting a source Type and a Source. The current system is displayed. Click [Go] to search for the details in the respective Source and Source Type.
