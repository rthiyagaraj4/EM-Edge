Enter the following details to lodge a complaint on a linen item:

 

  1. Complaint Number – The Complaint Number is system generated. The request number is incremented by one when a request is placed successfully to record the complaint. You cannot change the field. This is a mandatory field.

  2. Complaint Date – The system date and time get displayed, by default. The time is displayed in hh:mm format. You cannot change the field. This indicates the date and time when the complaint is recorded. This is a mandatory field.

  3. Complaint Source Type – Select the Complaint Source Type from the list. The system-defined value is Laundry. This is a mandatory field.

  4. Complaint Source - Select the Source using [?] function. Sources associated to laundry get displayed.

  5. Complaint Against Source Type – Select the Source Type from the list. This indicates the Source Type against which the complaint is lodged. The system-defined values are ‘Nursing Unit’, ‘Clinic’ and ‘Department. This is a mandatory field.

  6. Complaint Against Source - Select the Source using [?] function. If the Source Type selected is ‘Nursing Unit’, Nursing Units defined in Inpatient & Day Care Management module get listed. If the Source Type selected is ‘Clinic’, Clinics defined in Outpatient Management module get listed. If the Source Type selected is ‘Department’, Departments defined in Application Masters module get listed.

  7. Complaint Activity – Select the Complaint Activity from the list. The system-defined values are ‘Issue with Cleaning Request’, ‘Request for Mending’, ‘Receive Linen after mending’ and ‘Others’. This is a mandatory field.

  8. Complaint – Select a Complaint from the list. Reasons defined using the function ‘Reason/Complaint Codes’ with the Action Type as ‘Complaint by Laundry’ and ‘Complaint by Ward/Clinic/Department gets listed for selection. This is a mandatory field.

  9. Complaint By - Enter the name of the person in alphabetical format of 30 characters maximum length. This indicates the person who is lodging the complaint from the Laundry to the other Source. The User Name gets displayed as the ‘Complaint By’ name, by default. You can modify this field. This is a mandatory field.

  10. Reference Number - Select the Reference Number using [?] function. Reference Numbers are generated once a request is placed for new linen, cleaned linen, loaned linen, receive linen, issue linen, enter linen, deliver linen, send linen for mending and receiving linen for mending.

  11. Date/Time – The system date and time is displayed, by default. You cannot modify it.

  12. Linen Item – The linen items for which the request is placed for the selected Reference Number are displayed.  Linen items for which standard stock is defined in ‘Standard Stock for Sources’ function get listed.

  13. Quantity – Enter the quantity in numeric format. The quantity refers the number of linen items, which are affected for which the complaints are lodged. The quantity recorded in the requests placed for new linen, cleaned linen, loaned linen, receive linen, issue linen, enter linen, deliver linen, send linen for mending and receiving linen for mending is listed.
