Pending Count Confirmation

This tab is enabled if ‘Wards should confirm the Dirty Linen counted by Laundry’ check box is selected in the ‘Setup Parameters for Facility’ function. You can update the quantity.

  1. The Source Type and Source is displayed based on the selection.

  2. Click the Calendar function to select the request date. This indicates the date when the request is placed for cleaning. System calculates and displays the time and date seven days prior to the system date. This is a mandatory field. <Related Field Description>

  3. Click [Search] to view the pending count confirmation for the selected Source under the related Source Type within a specified date range. The Linen Item details are displayed.

    1. Linen Item – The name of the selected linen item is displayed.

    Request

    1. Date – The date when the linen item was sent for cleaning is displayed.

    2. Quantity – The quantity of linen item sent for cleaning for the selected date is displayed and defaulted. You cannot modify the quantity.

    3. Linen Handed Over By – The name of the person who has handed over the linen item to the laundry is displayed and defaulted. You cannot update the name of the person.

    4. Linen Collected By – The name of the logged in User is displayed and defaulted.

    5. Confirm – Select the check box to confirm the quantity received by the laundry. The number of linen items received is confirmed, if the check box is selected.

    6. Remarks - Click ‘Remark’ displayed as a hyperlink to view the remarks recorded.

    7. Confirmed Date – The date and time when the laundry confirmed the receipt of linen items for cleaning is displayed. You cannot modify the Confirmed Date.

    8. Confirmed By – The name of the logged in User is displayed. Enter the name of the person who is confirming the quantity of the dirty linen item.

  4. Click [Submit] to save the pending count confirmation record.
