Enter the following details:

 

  1. Source Type – The selected Source Type is displayed and you cannot edit this field.

  2. Source - The selected Source is displayed and you cannot edit this field.

  3. Request Date – System calculates and displays the time and date seven days prior to the system date. Click the Calendar function to select a new Request Date. This indicates the date when the request for cleaning the linen is placed. This is a mandatory field.

  4. To - Click the Calendar function to select the Request Date. The current System Date and time are displayed as the ‘To’ Request Date. The cleaning requests placed during the date range selected are displayed.

  5. Linen Handed Over By - Enter the name of the person who had placed the request from the Nursing Units/Clinics/Departments to clean the linen. The User Name is displayed as the name of the person who is responsible for handing over of linen representing the Nursing Unit/Clinic/Department, by default. This can be changed by the user. This is a mandatory field.

  6. Linen Collected By – Enter the name of the person in alphabetical format of 30 characters maximum length. This indicates the person who is responsible to collect  soiled and infected linen from Nursing Units/Clinics/Departments.

  7. Request Status – ‘All Pending Requests’ is displayed, by default. Select a value from the list. The system-defined values are ‘All’, ‘All Pending Requests’, ‘Outstanding’, ‘Partially Issued’, ‘Issued’ and ‘Cancelled’.

  8. All – Select ‘All’ to view all the requests placed. The requests, which are outstanding, partially issued, issued and cancelled, are listed.

  9. All Pending Requests – Select ‘All Pending Requests’ to view the list of pending requests. The requests, which are currently pending are displayed.

  10. Outstanding – Select ‘All Pending Requests’ to view the list of outstanding requests. The requests, which are currently outstanding, are displayed.

  11. Partially Issued – Select ‘Partially Issued’ to display the list of requests for which the linen items are partially issued by the laundry.

  12. Issued – Select ‘Issued’ to display the list of requests for which the linen items are issued by the laundry.

  13. Cancelled – Select ‘Cancelled’ to display the list of requests for which the linen item requests are cancelled.

  1. Linen Item – Select the Linen Item using [?] function. The requests placed to clean specific linen items are listed. You can either view the requests placed or cancel the request for a specific linen item.
