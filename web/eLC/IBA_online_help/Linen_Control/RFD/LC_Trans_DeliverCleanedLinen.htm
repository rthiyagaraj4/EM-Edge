Enter the following details:

 

  1. Source Type – Select a Source Type from the list. The system-defined values are Nursing Unit, Clinic and Department. This indicates the Source Type to which the linen items have to be delivered.

  2. Source - Select the Source using [?] function. If the Source Type selected is ‘Nursing Unit’, Nursing Units defined in Inpatient Management module get listed. If the Source Type selected is ‘Clinic’, Clinics defined in Outpatient Management module get listed. If the Source Type selected is ‘Department’, Departments defined in System Configuration module get listed. This indicates the Source to which the linen items have to delivered from the laundry after cleaning.

  3. To - Click the Calendar function to select the Request Date. The current system date and time are displayed as the ‘To’ date. The requests placed during the selected date range for cleaning request are listed.

  4. Linen Handed Over By - The name of the logged-in user is displayed, by default. This indicates the name of the person who has handed over the linen to the laundry.

  5. Linen Collected By – Enter the name of the person in alphanumeric format of 30 characters maximum length. This indicates the person who is responsible for collecting linen from the Nursing Units/Clinics/Departments.

  6. Request Status – ‘All Pending Requests’ is displayed, by default. Select a value from the list. The system-defined values are ‘All’, ‘All Pending Requests’, ‘Outstanding’, ‘Partially Issued’, ‘Issued’ and ‘Cancelled’.

  7. All – Select ‘All’ to view all the requests placed. The requests, which are outstanding, partially issued, issued and cancelled, are listed.

  8. All Pending Requests – Select ‘All Pending Requests’ to view the list of pending requests. The requests, which are currently pending are displayed.

  9. Outstanding – Select ‘Outstanding’ to view the list of outstanding requests. The requests, which are currently outstanding are displayed.

  10. Partially Delivered – Select ‘Partially Delivered’ to display the list of requests for which the linen items are partially delivered by the laundry.

  11. Delivered – Select ‘Delivered’ to display the list of requests for which the linen items are delivered by the laundry.

  12. Cancelled – Select ‘Cancelled’ to display the list of requests for which the linen item requests are cancelled.

  1. Linen Item – Select the Linen Item for which the request has been placed using [?] function. Linen items for which standard stock is defined in ‘Standard Stock for Sources’ function get listed. This is a mandatory field.
