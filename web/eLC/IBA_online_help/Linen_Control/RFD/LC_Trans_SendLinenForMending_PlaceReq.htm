Place Linen Mending Request

You can record the linen item details, which are sent for mending to the agencies.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Select] to display a Request placed for mending the linen. The selected Linen Item is displayed along with the requested quantity and remarks. Linen Item and Remark fields are displayed as hyperlinks. Click Linen Item displayed as a hyperlink and the selected Linen Item and its requested quantity are displayed in the respective fields for editing.

  3. Click Remark displayed as a hyperlink to view the remarks recorded. To edit remarks, select the Linen Item hyperlink and record the details in the respective ‘Remark’ field. Click [OK] to record the details. Click [Cancel] to cancel the transaction. You cannot select the same linen item while placing a request for mending linen items under a Request Number.

  4. Click [Submit] to save the request for mending linen items.

  5. A confirmation message ‘Please note your Request Number’ is displayed. Click [OK] and the control shifts back to ‘Place Linen Mending Request’ tab to place the next request.
