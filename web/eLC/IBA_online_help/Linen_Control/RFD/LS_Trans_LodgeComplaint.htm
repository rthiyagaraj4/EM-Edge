Enter the following details:

 

  1. Complaint Number - The Complaint Number is system generated. The Complaint Number is incremented for every new request placed. This is a mandatory field.

  2. Complaint Date - The system date and time are displayed as the Complaint Date. The time is displayed in hh:mm format. You cannot change the complaint date. This is a mandatory field.

  3. Complaint Source Type - The selected Source Type is displayed and you cannot edit this field. This indicates the Source Type from where the request is placed. This is a mandatory field.

  4. Complaint Source - The selected Source is displayed and you cannot edit this field. This indicates the Source from where the request is placed. This is a mandatory field.

  5. Complaint Against Source Type – Select the Source Type from the list against which the complaint is lodged. The system-defined values are ‘Nursing Unit’, ‘Clinic’ and ‘Department. This is a mandatory field.

  6. Complaint Against Source - Select the Source using [?] function. If the Source Type selected is ‘Nursing Unit’, Nursing Units defined in Inpatient Management module gets listed. If the Source Type selected is ‘Clinic’, Clinics defined in Outpatient Management module get listed. If the Source Type selected is ‘Department’, Departments defined in Application Masters module get listed.

  7. Complaint Activity – Select the Complaint Activity from the list. The system-defined values are ‘New Linen Issue’, ‘Late Delivery’, ‘Issue with Linen Delivery’ and ‘Others’. This indicates the reason for lodging the complaint. This is a mandatory field.

  8. If Complaint Activity is selected as ‘New Linen Issue’, you can lodge a complaint against the linen item newly issued to the Nursing Unit/Clinic/Department.

  9. If Complaint Activity is selected as ‘Late Delivery’, you can lodge a complaint against the linen items received late after placing a request.

  10. If Complaint Activity is selected as ‘Issue with Linen Delivery’, you can lodge a complaint against the linen item given for cleaning and received defective linen item from the laundry.

  11. If Complaint Activity is selected as ‘Others’, you can lodge any other complaints other than the ones mentioned above against the linen items.

  12. Complaint – Select the Complaint from the list. The ‘Reasons’ defined using the ‘Reason/Complaint Codes’ function with the ‘Complaint By Laundry’ and ‘Complaint By ward/clinic/dept.’ check boxes as selected are listed. This is a mandatory field.

  13. Complaint By – The name of the logged-in User gets displayed, by default. Enter the name of the person in alphanumeric format of 30 characters maximum length. This indicates the name of the person who has lodged the complaint against a source associated to a Nursing Unit, Clinic or a Department. This is a mandatory field.

  14. Reference Number – Select the Loaned Linen Reference Number using [?] function. Reference Numbers are generated in ‘Loan Linen’ tab once a request is placed for lending linen in this function.

  15. Date/Time – The system date and time are displayed, by default. You cannot modify it.

  16. Linen Item - On selecting the Reference Number, the linen item for which the request placed is displayed.

  17. Quantity – On selecting the Reference Number, the linen item and its quantity is displayed in the respective fields. You can enter the quantity of linen item in numeric format. The quantity indicates the number of defective linen items against which the complaints are lodged out of the total linen items recorded in the Reference Number.
