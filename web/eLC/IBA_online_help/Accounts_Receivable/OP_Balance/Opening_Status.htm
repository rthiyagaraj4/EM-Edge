Opening Status

From this function you can Record the Opening Balance details and the last processed document reference details based on a customer and Query for the existing records.

At the time of installation, these values are entered. The values are updated by the system internally, whenever the transactions are posted. The values appeared on the screen are used to analyze the business performance statistics.

Recording the Opening Balance and Last Processed details

From the Op-Bal, double click Opening Status function.

The Opening Status screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Commit Form] or press [F10] to save an Opening Status during the initial setup or [Clear Record] to clear the entered or queried data.

  3. Click [Execute Query] or press [F8] to query the existing records.

 

You can save the record for each customer code during the initial setup only. The system updates the values internally. You can update the Opening Balance and last processed document reference details.
