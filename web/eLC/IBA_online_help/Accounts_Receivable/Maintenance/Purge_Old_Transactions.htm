Purge Old Transactions

From this function you can Delete the old transaction records from the database.

The transaction records include Invoice, Debit Notes, Credit Notes, Adjustments, Receipts and Refunds. The records ranging within the specified criteria can be deleted.

Deleting Old Transaction Records

From the Maint, double click Purge Old Transactions function.

The Purge Old Transactions screen gets displayed.

  1. Enter the Month and Year in the text field. The old transaction records up to the specified periods are deleted. The Month and Year entered must be less than the current processing period. This is a mandatory field.

  2. Select the ‘Purge Transaction Type’ from the list. The listed values are ‘All’, Invoice’, ‘Debit Notes’, ‘Credit Notes’, ‘Receipts’, ‘Refunds’ and ‘Adjustments’. If the value ‘All’ is selected, the transaction records of all Transaction Type records can be deleted.

  3. A message ‘Do you want to confirm?’ gets displayed. Click [OK] to purge the selected Transaction Type or [No] to select another record.
