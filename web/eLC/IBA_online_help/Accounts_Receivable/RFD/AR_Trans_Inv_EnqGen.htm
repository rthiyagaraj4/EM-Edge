Enter the following details:

 

  1. Doc Type Code for Invoices – Enter the Document Type code in alphanumeric format of six characters maximum length or select the required Document Type Code for Invoices from [List of Values]. The entries with code and its description for Invoices get populated when a valid Document Type Codes are set initially in the Financials Base module. This is a mandatory field.

  2. Doc Number/Invoice Date – On entering the Document Type Code, the Starting Document Number set for the selected Document Type in Financial Base module is displayed, by default. The current system date is displayed as Invoice Date, by default. This is a mandatory field.

  3. Bill Type – Select the required Bill Type Code and its description for Invoices from [List of Values]. This is a mandatory field.

  4. Credit Reference – Enter the relevant Credit Reference in alphanumeric format of 15 characters maximum length. The Credit Reference identifies the Document Type for the Customer. This is a mandatory field.

  5. Post Month/Year – The month and year set in AR Parameter is displayed as Current Accounting Year, by default. You can modify the data if the Post Month and Year is greater than Current Accounting Year. This is a mandatory field.
