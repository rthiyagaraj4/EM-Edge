Enter the following details to generate an Official Receipts:

 

  1. Party Type – The options are “Misc” and “Customer”. If the customer is selected, the value for customer code is validated against the Customer code existing in the system. If Miscellaneous is selected, Name and Address are also been entered.  

  2. Doc Type/Number - Select the required Document Type Code from [List of Values]. This is a mandatory field. Document Number is generated by the system, if it is defined for the entered Document Type.

  3. Doc Date - Click Calendar function to select the Document Date. Document date has to be less than or equal to the system date.

  4. Customer Code - Select a valid Customer from [List of Values].  If Party Type selected is “Misc”, Name and Address are entered.

  5. Amount - Enter the Document Amount in numeric format of nine digits maximum length. This is a mandatory field.

  6. Document Ref – Enter the Reference in alphanumeric format of 15 characters maximum length. The default value is populated by the system, if it is of allocated entry.

  7. Cheque No – Enter the cheque number in alphanumeric format of 15 characters maximum length.

  8. Date – Enter the cheque date. Cheque date has to be less than or equal to current date.

  9. Of Bank - Enter the details of Bank and Branch in alphanumeric format of 20 characters maximum length. The Name of the bank in short form and Branch name mentioned in the cheque has to be entered as its value.

  10. Post Month/Year - Enter the Month/Year in mm/yyyy format. This is a mandatory field. The system displays the current processing month and year from AR parameters as default value.

  11. Bank Account - Select a valid Bank Account from [List of Values]. The value of Default Bank Account entered in AR Parameter screen is displayed, by default.

  12. Susp Account – Select a valid Suspense Account from [List of Values]. The value of Default Suspense Account entered in AR Parameter screen is displayed as default.

  13. Narration - Enter the Narration in alphanumeric format of 40 characters maximum length.

  14. [Print & Post] - [Print & Post] is enabled only if in the AR parameter, ‘Online post official Receipt?’ flag is selected. If the check box is not selected, [Print & Post] is not displayed. If you click [Print & Post], the system does the online printing and posting.
