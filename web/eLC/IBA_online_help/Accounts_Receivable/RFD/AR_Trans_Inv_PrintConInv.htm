Enter the following selection criteria:

 

  1. Document Type [From-To] – Enter the range for Document Type or click [List of Values] to select the required Document Type. Both the values are optional. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  2. Document Number [From-To] - Enter the range for Document Number or click [List of Values] to select the required Document Number. Both the values are optional. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  3. Document Date [From-To] - Enter the range for Document Date or select the required date using the Calendar function. Both the values are optional. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  4. Customer – Enter the Customer code in alphanumeric format of eight characters maximum length or select the required Customer Code from [List of Values]. The Customers defined in the Customer function are displayed for selection. If the code selected is valid, the Customer Name is displayed as read-only text for user verification. This is a mandatory field .

  5. Bill Type - Select the required Bill Type from [List of Values]. The entries with code and its short description get populated only when valid Billing Types are set initially in the Patient Billing Manager module. This is a mandatory field.

  6. Cons Doc Type - Select the required Document Type Code from [List of Values]. This field is mandatory.

  7. Statement Date – Enter the date in the format of dd/mm/yyyy or select the required date using the Calendar function to set a valid Statement Date. This field is mandatory.
