Facility Based Report Parameters

You can update the details for the following facility based report parameters:

 

  1. Module Name – Click [List of Values] to select a Module Name. The entries with code and its description are populated only when valid Module Names are set initially in the System Manager Module.

  2. Report Desc – Click [List of Values] to select a valid Report Description. The entries with code and its description are populated only when valid Report Descriptions are set initially in the System Manager module. The description of the report is displayed as the title of the report. This field cannot be changed.

 

Facility Parameters

  1. Facility Based – Click [List of Values] to select a Facility. The entries with code and its description are populated only when valid Facilities are set initially in the System Manager module. For some of the reports, facility based parameters/routing rules are to be set up. This especially happens when the facilities are distinctly identifiable physical locations and distributed. Most of the reports pertaining to a facility can be classified as facility based reports and other reports pertaining to an enterprise are to be classified the other way. If a report is transaction based, it is always a facility-based report also or this flag can be changed.

  2. Report Group – Click [List of Values] to select a Report Group. The entries with code and its description are populated only when valid Report Group is set initially in the System Manager Module. A report can be categorized into one of the valid report groups, if it is not a facility-based report. This field value is not editable for a facility-based report. If a report is not facility based, the corresponding report group can be set up in Facility Based Report Parameters function.
