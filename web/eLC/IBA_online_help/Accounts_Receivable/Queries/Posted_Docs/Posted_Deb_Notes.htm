Posted Debit Notes

From this function you can Query the Debit Notes which are posted to the GL interface.

Posted Debit Notes those pertaining to the period for which the month end process is not complete only gets displayed. The Posted Debit Notes are deleted when the month end process is complete. In addition to the document details, related bills details, user accessed list, text document details are also displayed.

Querying Posted Debit Notes

From the Queries > Posted Documents, double-click Posted Debit Notes function.

The Posted Debit Notes screen gets displayed.

  1. Click [Execute Query] or press [F8] to view the debit notes posted from the Accounts Receivable, Patient Billing Manager and the Laboratory module.

OR

  1. Click Calendar function to select the date range. Click [Execute Query] or press [F8] to view the debit notes posted from the Accounts Receivable, Patient Billing Manager and the Laboratory module during the specified date range.

  2. Click [Refresh] to clear the date recorded.

  3. Click [Debit Note Details] to view the posted debit note details. The Posted Debit Notes screen gets displayed with the debit notes, patient bill, details of the User who has posted the Debit notes and the remarks recorded while posting the debit notes.
