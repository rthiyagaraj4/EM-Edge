Print Credit Notes

From this function you can Print the Credit Notes which are recorded.

Credit Notes can be printed any number of times before or after posting. Once the document is posted, Credit Notes are printed with the legend ‘COPY’. This is an on-line report: the output is produced on a laser printer.

Printing a Credit Note

From the Transactions > Credit Notes, double-click Print Credit Notes function.

The Print Credit Notes screen gets displayed.

  1. Click [List of Values] to select a valid Document Type. Both the values are optional. If ‘FROM’ value is not entered, the system assumes the lowest value for the ‘FROM’ and if ‘TO’ value is not entered, the system assumes the highest value for the ‘TO’ value. When both values are entered, ‘TO’ value must be greater than or equal to the ‘FROM’ value.

  2. Enter the range for Document Number. Both the values are optional. If ‘FROM’ value is not entered, the system assumes the lowest value for the ‘FROM’ and if ‘TO’ value is not entered, the system assumes the highest value for the ‘TO’ value. When both values are entered, ‘TO’ value must be greater than or equal to the ‘FROM’ value.

  3. Enter the range for Document Date. Both the values are optional. If ‘FROM’ value is not entered, the system assumes the lowest value for the ‘FROM’ and if ‘TO’ value is not entered, the system assumes the highest value for the ‘TO’ value. When both values are entered, ‘TO’ value must be greater than or equal to the ‘FROM’ value.

  4. Click [Run Process] to view, print or save the report in a file in the local machine.
