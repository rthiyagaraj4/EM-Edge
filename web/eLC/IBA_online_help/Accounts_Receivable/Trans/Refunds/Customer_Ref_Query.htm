Customer Refund Query

From this function you can Query the details of Refund Request of all Customers.

This is only a query function in case the AR parameter ‘Cheque entry details as part of Refund’ column is selected. If in case the AR parameter ‘Cheque entry details as part of Refund’ is not selected, [Show Details] is displayed to enter the cheque details. There is also the provision given to provide the order in which the query is displayed and whether it has to be in ascending order or descending order. You can query the Customer Refund details based on Cheque Entered, Cheque Not entered or all.

Query Customer Refund Records

From the Transactions > Refunds, double-click Customer Refund Query function.

The Customer Refund Query screen gets displayed.

  1. Select the value from the list. The listed values are All, Cheque Entered and Cheque Not Entered.

  2. Click [Execute Query] or press [F8] to view all the records.

OR

  1. Click [Enter Query], enter the details and click [Execute Query] or press [F8] to view the selected records.

  2. Based on the selection of a value from the list, the records are populated in the Customer detail columns.
