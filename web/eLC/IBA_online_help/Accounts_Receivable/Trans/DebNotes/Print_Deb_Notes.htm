Print Debit Notes

From this function you can Print the Debit Notes which are recorded.

Debit Notes can be printed any number of times before posting or after Posting. Once the document is posted, Debit Notes is printed with the legend ‘COPY’. This is an on-line report; the output is produced on a laser printer.

Printing Debit Notes

From the Transactions > Debit Notes, double-click Print Debit Notes function.

The Print Debit Notes screen gets displayed.

  1. Enter the range for Document Type or click [List of Values] to select the required Document Type. Both the values are optional. If ‘From’ value is not entered, the system assumes the lowest value for the ‘From’ and if ‘To’ value is not entered, the system assumes the highest value for the ‘To’ value. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  2. Enter the range for Document Number or click [List of Values] to select the required Document Number. Both the values are optional. If ‘From’ value is not entered, the system assumes the lowest value for the ‘From’ and if ‘To’ value is not entered, the system assumes the highest value for the ‘To’ value. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  3. Enter the range for Document Date or select the required date using the Calendar function. Both the values are optional. If ‘From’ value is not entered, the system assumes the lowest value for the ‘From’ and if ‘To’ value is not entered, the system assumes the highest value for the ‘To’ value. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  4. Click [Run Process] to view, print or save the report in a file in the local machine.
