Review/Update Debit Note Posting Account

From this function you can Review or Update the Debit Note Posting Account details.

This function is used to view and change the existing posting accounts in Debit Notes. This function cannot be used if AR to GL interface method is ‘Online’ since the General Ledger would have got updated at the time of posting of the debit note itself. Only the debit notes, which have been posted already, can be accessed here for change. Debit Notes can be re-called till the time the current processed month is not closed.

Review/Update a Debit Note Posting Account

From the Transactions > Debit Notes, double-click Review/Update Debit Note Posting Account function.

The Review/Update Debit Note Posting Account screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Commit Form] or press [F10] to save a reviewed or updated Debit Note or click [Clear Record] to clear the entered or queried data.

  3. Click [Execute Query] or press [F8] to search for an existing reviewed or updated posted Debit Note record.
