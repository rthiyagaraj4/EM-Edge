Print Invoices

From this function you can Print the Invoices generated for the Customer Transactions.

Invoices can be printed any number of times before posting if there is no corrections made. Once posted, Invoices cannot be printed. There is an option to either print on pre-printed stationery or normal stationery.

Printing an Invoice

From the Transactions > Invoices double-click Print Invoices function.

The Print Invoices screen gets displayed.

  1. Enter the range for Document Type or click [List of Values] to select the required Document Type. Both the values are optional. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  2. Enter the range for Document Number or click [List of Values] to select the required Document Number. Both the values are optional. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  3. Enter the range for Document Date or select the required date using the Calendar function. Both the values are optional. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  4. Select the check box, if the Invoice is to be printed in pre-printed stationary. Clear the check box, if it is to be printed on normal stationary. Value is cleared, by default.

  5. Click [Run Process] to view, print or save the report in a file in the local machine.
