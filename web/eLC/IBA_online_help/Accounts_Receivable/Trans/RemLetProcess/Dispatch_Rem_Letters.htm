Dispatch Reminder Letters

From this function you can Dispatch reminder letters to the Customers when the payment amount is overdue.

This function is used to dispatch all the Reminder Letters generated for a particular customer within a range of dates for the outstanding amount. The Reminder Letter is dispatched based on the Document Type. Each Reminder Letter includes the Encounter Details indicating the Episode Type and the Episode or the Visit Number of the patient. The Billing Group, Document Type and the Document Number for a specific Encounter are also displayed in the Reminder Letter.

Dispatching a Reminder Letter

From the Transactions > Reminder Letter Process, double-click Dispatch Reminder Letters function.

The Dispatch Reminder Letters screen gets displayed.

  1. Click [List of Values] to select a valid Customer. On selecting a Customer Code, the short description of the Customer is displayed in the adjacent text field. This is a mandatory field.

  2. Click the Calendar function to select valid date range. The Reminder Letters generated for the specific date range is dispatched. This is a mandatory field.

  3. Click [Execute Query] or press [F8] to view the reminder letters which are generated using the function Generate Reminders. <Related Field Description>

  4. Select the check box against a record to dispatch the Reminder Letter for the selected document.

  5. Click [Run Process] to proceed further. Click [Yes] to confirm or [No] to cancel the print. On clicking [Yes], a message is displayed showing that the Dispatch Reminder is processed. Click [OK] to accept the processed records.
