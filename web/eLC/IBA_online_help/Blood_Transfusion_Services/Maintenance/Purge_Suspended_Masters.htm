Purge Suspended Masters

From this function you can Purge the master records that are suspended.

The Purge Suspended Masters function is used to delete the master records, which are in suspended status. If the master codes are already used in any transactions the code cannot be deleted. The deleted master functions are suspended in the database, not deleted. You cannot create a new master function with the same code.

Purging Suspended Masters

From the Maintenance, double-click Purge Suspended Masters function.

The Purge Suspended Masters screen gets displayed.

  1. Click [List of Values] to select a master code. The master codes suspended through the 'Suspend/Reinstate Masters' function are listed.

  2. Click [Commit] or press [F10] to delete the master codes or click [Clear Record] to clear the entered details. A confirmation message is displayed on successfully completing the transaction.
