Acknowledge Blood Bank Order Request

From this function you can Define the batch process to cancel all the incomplete requests before the cut-off date.

This function allows you to acknowledge the blood product volume request made in the blood bank. A discrete measure ‘Blood Volume’ is defined for lab order category in the Order Format screen and it is mapped against all BTS catalogues.

Blood Units Acknowledging Blood Bank Order Request

From the Transactions > BT Operations, double-click Acknowledge Blood Bank Order Request function.

The Acknowledge Blood Bank Order Request screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search]. The orders matching the search criteria gets displayed. The search results include Order ID, Ordered Date, Patient ID, Patient Name, Source Name, Order Catalog Name, Blood Volume (in ml) and Acknowledge check box.

• In CA module, if the Blood Volume is set up in Order Header level and Order Line Level, the system checks for header setup and that Blood volume is displayed.  

• When Blood Volume setup is not done in header lever or header set up value is (null or empty), the system displays Blood Volume setup done in Line level.

• In CA, Blood Volume field allows to enter decimal value but in Acknowledge function the value is round off and displayed (for example, 450.67 to 451 ml).

  1. Select the check box against the required order record to acknowledge the order request. Or, select the ‘Select All’ check box to select all orders for acknowledgment.

  2. Click [Clear] if you want to reset the records displayed in the result area.

  3. Click [Confirm], the selected order gets acknowledged. Once the order is Acknowledged you cannot again modify or revert it. It is a onetime process.

  4. The acknowledged records appear with a blue highlight in Existing Orders and Order Tracking screens.
