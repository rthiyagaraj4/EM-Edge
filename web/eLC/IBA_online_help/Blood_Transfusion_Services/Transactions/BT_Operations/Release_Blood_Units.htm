Release Blood Units

From this function you can Release the blood units that are Allocated or Reserved and Print the details of the specimens to be released.

The Release Blood Units function is used to release the Blood Units, which are currently in Allocated or Reserved Status. You can query for the units that are allocated or reserved based on the specimen allocation or reservation date. You can release a unit or a set of units at a time to the stock. You can also print the specimen details that have to be released.

Releasing Blood Units

From the Transactions > BT Operations, double-click Release Blood Units function.

The Release Blood Units screen gets displayed.

  1. Enter the query criteria to view specimen details for the specified query criteria. <Related Field Description>

  2. Press [Tab] or click [Display] to view the specimen details. <Related Field Description>

  3. Select the check box next to the Storage Location to release the selected specimen. Clear the check box next to the Storage Location not to release the selected specimen.

  4. Select the ‘Select All’ check box to select and release all the specimens. Clear the  ‘Select All’ check box not to select and release all the specimens.

  5. Click [De-Allocate/De-Reserve] to release the allocated or reserved specimens.

  6. A confirmation message ‘De-Reservation/De-Allocation of the selected Units are completed successfully’ is displayed based on the value selected for ‘De-Allocate/De-Reserve’.

  7. Click [Print] to print the specimen details. You can also view the file in a PDF format and save the file in HTML, Delimited, RTF and PDF formats.
