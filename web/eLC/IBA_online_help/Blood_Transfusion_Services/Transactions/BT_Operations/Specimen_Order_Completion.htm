Specimen Order Completion

From this function you can Record completion of orders placed for specimens.  

The Specimen Order Completion function is used to record the completion of an order for specimens. This function allows you to search for the units based on the donation date. You can mark the order as complete for expired units. The specimen validity period is defined in the ‘BT Parameter’ function, based on which the expiry date is calculated. You cannot mark the order as complete for the specimens that are in process. For Example, if a single unit or multiple units of the specimen are in Allocated, Reserved or in Issued stages, even though the specimens have expired, you cannot set the order as complete. The color codes defined at the bottom of the screen indicate the status of the specimen.

– In Process (blood units issued to internal patients)

- Expired Specimen

- In Process, can be completed (blood units issued to external patients)

Specimen Order Completion

From the Transactions > BT Operations, double-click Specimen Order Completion function.

The Specimen Order Completion screen gets displayed.

  1. The current system date is displayed as the Specimen Collection date, by default. Click the Calendar function to select the Specimen Collection date range.

  2. Select a Source Type from the list. The system-defined values are ‘All’, ‘Ward’, ‘Clinic’ and ‘External’. The Source Type is set to ‘All’, by default.

  3. If you select the ‘Source Type’ as ‘Ward’, ‘Clinic’ or ‘External’, additional fields are displayed to specify a range of Wards, Clinics or External sources to list the specimen orders processed in the respective Source Type. Click [List of Values] to select a range of Source Types. The values in the list vary based on the ‘Source Type’ selected.

  4. Press [Tab], press [F8] or click [Execute Query] to view the results matching the query criteria. <Related Field Description>

  5. Select the check box to complete Specimen Order for the selected records. You cannot complete the Specimen Order if any of the units in the specimen is in ‘Allocated’, ‘Reserved’ or ‘Issued’ status.

  6. Click [Update Specimen] to record the completion of the specimen order. A confirmation message ‘Process Completed Successfully’ is displayed.
