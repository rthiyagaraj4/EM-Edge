Dispose Expired Blood Units

From this function you can Dispose the expired blood units permanently from the stock and Print the details.

The Dispose Expired Blood Units function enables you to dispose the expired blood units from the stock. The status of the stock is updated based on the fate code associated to the unit. Stock status is updated if ‘Effect on Stock’ is selected as ‘Deducted’ in ‘Fate Codes’ function.

Disposing Expired Blood Units

From the Transactions > Stock Operations, double-click Dispose Expired Blood Units function.

  1. The Dispose Expired Blood Units screen gets displayed.

  2. The Expiry Date range is set to the current date, by default. Click Calendar function to select the Expiry Date range. The blood units that expired during the selected date range are listed.

  3. Click [List of Values] to select the Code. The list displays the values defined in the Fate Codes function. This is a mandatory field. The dispose code is selected to query the blood units that have the selected Dispose code associated with it.

  4. Click [List of Values] to select the Product Code. The list displays the values defined in the Product Codes function. he product code is selected to query the blood units that have the selected product code associated with it.

  5. Press [Tab] to query the product details matching the query criteria. <Related Field Description>

  6. Select the check box against a specific record to dispose the respective blood unit.

  7. Select the ‘Select All’ check box to select and dispose all the products displayed. If all the check boxes are selected, clicking on ‘Select All’ will clear all the check boxes.

  8. Click [Commit] to dispose the expired blood units or [Clear Record] to cancel the process.

  9. Click [Print] to print the product details. You can also view the file in a PDF format and save the file in HTML, Delimited, RTF and PDF formats.
