Enter the following details:

 

  1. Transaction Date/Time [From-To] – The system Date and Time is displayed, by default. Click the Calendar function to select the date range to view the unit details.

  2. Product Code – Click [List of Values] to select the Product code. The listed values are created in the ‘Product Codes’ function. The units associated to the selected Product code are listed. Leave the field blank to query all the records.  

  3. Centre Code - Click [List of Values] to select the Centre code to search for the blood units from that Collection Centre. The listed values are created in the ‘Collection Centre /Supplier’ function.  Leave the field blank to query all the records.

  4. Barcode Unit Number - Scan the blood unit to record the Barcode Unit Number. The check boxes  'Send to Lab' under 'Units for Screening' and 'Accept' under 'Results from Lab' tab are selected automatically when the Barcode Unit Number is scanned. The units are automatically sent to lab when the details are saved. If there are multiple Barcode Numbers for the same product, you can click [List of Values] to select the Barcode Unit Number.
