Enter the following details to create a Collection Centre or Supplier:

 

  1. Collection Centre/Supplier Code – Enter the Collection /Supplier Code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Long Description – Enter the Long Description of the Collection /Supplier in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the Short Description of the Collection /Supplier in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Address Line [1-4] – Enter the address of the Collection /Supplier in alphanumeric format of 100 characters each maximum length.

  5. Telephone Number – Enter the Telephone Number of the Collection /Supplier in alphanumeric format of 30 characters maximum length.

  6. Person in Charge – Enter the Person in Charge of the Collection /Supplier in alphanumeric format of 40 characters maximum length.

  7. Generate Unit Number - Select the check box to generate Unit Numbers based on the defined Collection /Supplier from where the blood units are received/donated/pooled.

  8. Length – Enter the Length of the unit number in numeric format of 20 digits maximum length. This field is enabled only if the ‘Generate Unit Number’ check box is selected.

  9. Prefix – Enter the Prefix for the unit number in alphanumeric format of six characters maximum length. This field is enabled only if the ‘Generate Unit Number’ check box is selected.

  10. Next Number – The Next Number is generated automatically on generating the unit numbers. This field is disabled and you cannot enter any data.

  11. Number of Labels – Enter the Number of Labels you need to generate/print at pre-donation stage in numeric format of two digits maximum length.
