Enter the following details:

 

  1. Blood Unit Added Date [From–To] – Select the Blood Unit Added Date range to search for the autologous unit details.

  2. Patient ID – Select the Patient ID from the list of values. The list displays a list of patients registered with the logged in facility.

  3. Barcode Unit Number – Scan the Unit Number of the autologous blood unit for which unit has to be released.

  4. Product Code – Select the autologous blood unit Product Code for which Blood Unit release is to be done.

  5. Right Click for user Option – Right-click on the ‘Right Click for user Option’ to select the patient. The Patient Lookup option is displayed. Click Patient Lookup to select the patient. The Lookup screen gets displayed.

  6. Unit Number – Enter the Unit Number of the autologous blood unit for which unit has to be released.
