Enter the following details to format reports:

 

  1. Report Page Length – Enter the numerical value for the number of pages to be printed while printing the report in two digits maximum length. The value set are applicable to all the reports in the Blood Transfusion Services module.

  2. Message Line – Enter the default message line to be printed in the reports.

  3. Footer Line 1 – Enter the default footer 1 to be printed in the report.

  4. Footer Line 2 - Enter the default footer 2 to be printed in the report.

  5. Aliquot Remarks – Enter the default aliquot remarks that appear in aliquot transactions.

  6. Component Tag Legend – Enter the Text to be printed on the tag created while issuing the blood unit in alphanumeric format of 50 characters maximum length.

  7. Crossmatch Tag Legend – Enter the Text to be printed on the tag created while crossmatching the blood unit in alphanumeric format of 50 characters maximum length.

  8. Emergency Tag Legend 1 – Enter the Text to be printed on the tag created while issuing blood unit in emergency in alphanumeric format of 50 characters maximum length.

  9. Emergency Tag Legend 2 – Enter the Text to be printed on the tag created while issuing blood unit in emergency in alphanumeric format of 50 characters maximum length.

  10. Emergency Tag Legend 3 – Enter the Text to be printed on the tag created while issuing blood unit in emergency in alphanumeric format of 250 characters maximum length. The first five lines out of 250 characters gets printed in the report.

  11. Licence in Comp. Rep - Enter the license number to be printed in the Compatibility X-Match Report and Compatibility Report.

  12. Crossmatch Legend in Comp. Rep - Enter the crossmatch name to be printed in the Compatibility X-Match Report and Compatibility Report.

  13. License No – Enter the License number in numeric value format of 40 character length. If there is no content in this field, the system does not print “This unit is not received for HIV, HCV, HBSAG, Malaria and syphilis” in the label.
