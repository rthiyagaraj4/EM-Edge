Results from Lab

Click Results from Lab tab to view the screening results from the lab.

  1. Unit Number – The Unit Number of the blood unit whose screening is performed in the Result Entry by Specimen function in Clinical laboratory module is displayed.

  2. Product Code – The Product Code of the blood unit whose screening is performed in the Result Entry by Specimen function in Clinical laboratory module is displayed.

  3. Description – The description of the Product Code whose screening is performed in the Result Entry by Specimen function in Clinical laboratory module is displayed.

  4. Original Unit Number - The Original Unit Number received from the external source and recorded through the 'Blood Units Entry' function is displayed.

  5. Accept - The check box is selected automatically, if the Barcode Unit Number is scanned.  

  6. Unit Blood Group – The Blood Group of the blood unit whose screening is performed in the Result Entry by Specimen function in Clinical laboratory module is displayed.

  7. Unit Rhesus – The Rhesus factor of the blood unit whose screening is performed in the Result Entry by Specimen function in Clinical laboratory module is displayed.

  8. Storage Location – The Storage Location of the blood unit where the screened blood units are stored is displayed. The values are defined in the Storage Locations function.

  9. Centre Code – The Centre Code of the Storage Location is displayed.

  10. Select All - Select the check box to accept the screening results of all the units from the laboratory. Clear the check box not to accept the screening results of all the units from the laboratory.

  11. Dispose Code - Click [List of Values] to select the Dispose Code to be attached before disposing of the blood units from the list of values, if the unit is mismatching with the specified and resulted blood group.

  12. Accept - Select the check box to accept the lab results.  The check box is selected automatically, if the Barcode Unit Number is scanned.  Clear the check box not to accept the lab results.

  13. On selecting Accept against specific results, a confirmation message ‘Process completed successfully’ is displayed.

  14. Register the Blood unit and perform the screening in the Laboratory. Refer to the Registration, Acknowledge/Verify the Specimen and Result Entry by Specimen functions in Laboratory module.

If the blood type is mismatching, the unit is highlighted with  color.

If the specimen is rejected from the lab, it is highlighted using color.

If the specimen is rejected, it is highlighted with  color.

 

Test Details

  1. Test – The Tests performed on the selected unit in the Result Entry by Specimen function in Clinical laboratory module are displayed.

  2. Details – The details of the test performed in the Result Entry by Specimen function in Clinical laboratory module are displayed.

  3. Result – The result of the test performed in the Result Entry by Specimen function in Clinical laboratory module is displayed.
