Enter the following details to define parameters:

 

Tests for Cross Match

  1. Test – Click [List of Values] to select the test code. The list displays the values defined in the ‘Test Codes’ function in Laboratory module.

  2. Description – The Description of the selected test code is displayed.

  3. In Emergency – Select the check box, if the test is to be performed during emergency  transfusion. If you clear the check box, the test is not performed during crossmatching before the emergency transfusion.

  4. Bill – Select the check box if the billing is required for the test . If the check box is cleared, billing is not performed for the selected test.

  5. Bill - Non Perform – Select the check box if the test has to be billed even when all the tests of the group are not performed. This is applicable in emergency cases when only the mandatory tests are performed before blood transfusion. If you clear the check box, billing is done only for the tests performed.     

 

The saved tests are available for selection in the ‘Acceptable Values for the Tests’ tab.
