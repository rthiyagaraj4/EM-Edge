Enter the following details:

 

  1. Regd Date/Time – The Date and Time recorded while registering the specimen in the Registration function are displayed.

  2. Priority – The Priority of the selected specimen is displayed. The valid values are ‘Routine’, ‘Urgent’ and ‘Stat’. ‘Stat’ means that the blood unit has to be given immediately. Routine means the priority is normal and Urgent means the priority is urgent. The priority is recorded for the specimen in the Registration function in Laboratory module.  

  3. Patient ID – The Patient ID and name of the patient whose specimen and blood typing is to be confirmed is displayed.

  4. Sex –The Gender of the patient recorded while registering the patient is displayed.

  5. Age – The Age of the patient recorded while registering the patient is displayed.

  6. Comments – Click [List of Values] to select the comment from the list of values. The list displays the values defined in the ‘Crossmatch Comments’ function.  

  7. Blood Group – The Blood Group of the patient recorded in the ‘Record Patient Blood Group’ function in Medical Records module is displayed.

  8. Rhesus – The Rhesus factor of the blood group recorded in the ‘Record Patient Blood Group’ function in Medical Records module is displayed. Product/Description – Click [List of Values] to select the Product Code from the list of values. The list displays the values defined in the ‘Product Codes’ function. On selecting the product code, the description of the product code is displayed.

  9. Volume – The Volume and Unit for the product defined in the ‘Product Codes’ function are displayed.

  10. Required Units - Enter the required number of blood units for transfusion in numeric format of two digits maximum length. The volume of the selected specimen is displayed, by default. You can modify the details. This is a mandatory field.

  11. Category/Description – The Category Code and Description for the selected product category defined in the Product category function are displayed.

  12. Required Unit – The volume of the selected specimen is displayed, by default. You can modify the details.

  13. Date – Select the Date upto when the blood units are required at blood transfusion department before transfusing it to the patient. This is a mandatory field.

  14. Expected Transfusion Date – The expected Transfusion date is displayed if the request is placed from the ‘Clinician Access’ module. If you place a request from the Blood Transfusion Services module, you have to enter the expected transfusion date.

  15. Remarks – Enter the Remarks while requesting the blood unit. You can enter the remarks for future reference.

  16. Reject - Select the check box to reject the requested unit. Clear the check box not to reject the requested unit.

  17. Registration – Click [Registration] to add red blood cells if the blood confirmation test is not available for the selected specimen.
