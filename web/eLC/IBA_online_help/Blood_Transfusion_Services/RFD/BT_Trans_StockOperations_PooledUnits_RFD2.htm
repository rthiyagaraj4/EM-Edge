Enter the following details:

 

  1. Unit Number – Enter the new Unit Number for the pooled product in alphanumeric format of 30 characters maximum length. If the ‘Generate Unit Number’ check box is selected in ‘BD Parameter’ function in Blood Donor module, or if the automatic unit number creation rule is applicable you cannot enter the unit number.

  2. Blood Group – The Blood Group of the pooled unit is displayed. Click [List of Values] to associate any other Blood Group.

  3. Rhesus – The Rhesus factor of the selected Blood Group is displayed. You cannot modify it.

  4. Pooled Date – The current system date and time is displayed as the Pooled Date. Click the Calendar function to modify the date and time. The selected date should be equal to or lesser than the current system date and time.

  5. Date of Expiry – The calculated Date of Expiry as per the expiry period defined in the ‘Product Codes’ function is displayed. Click the Calendar function to modify the date and time. The selected date should be equal to or lesser than the current system date and time. The selected date should be equal to or greater than the current system date and time.
