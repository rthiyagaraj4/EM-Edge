Pooled Units

From this function you can Query the Unit Status of a blood unit.

The Pooled Units function enables you to view all the blood units that have been pooled between the specified date range. You can view the pooled unit details such as products pooled and the product that is the outcome of pooling.

Querying Pooled Units

From the Queries, double-click Pooled Units function.

The Pooled Units screen gets displayed.

  1. Select the Date range to search for the pooled units. This is a mandatory field.

  2. Enter the pooled Unit Number to view details. <Related Field Description>

  3. Enter the Product code to search for the pooled unit details.

  4. On selecting the required search criteria, press [Tab] to view the pooled unit details.

  5. Click [Execute Query] or press [F8] to search for the records matching the search criteria.
