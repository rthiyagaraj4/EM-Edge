User Rights

From this function you can Define the user access rights to a User to perform DR processes and Query for a desired User Rights record.

You can perform the DR Processes like Searching, Identify Main/Duplicate, Merge and Administer Search Process from this function. The users are assigned rights to perform DR Processes for the logged in facility or all the facilities.

Defining User Access Rights

From DR Code Setup > DR Code Setup, select the User Rights function.

The User Rights screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the data in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the User Rights defined or [Reset] to clear the entered data.
