Transactions of Merge Process

From this function you can Query and view the details of groups selected for merge process and groups not selected for the merge process.

You can select the groups directly from their respective columns and view the details. The details displayed correspond to the current DR Merge Process.

Querying for Transactions of Merge Process

From DR Queries, select the 'Transactions of Merge Process' function.

The Transactions of Merge Process screen gets displayed.

  1. The Transactions of Merge Process' screen gets displayed with the Merge Process Details.<Related Field description>

  2. Click the ‘Main Patient ID’ hyperlink from the required column (that is, merged groups or not merged groups). The details of the main and duplicate patients of the selected group are displayed in the Group Details area.

You can use the Search Criteria for patients under the merge groups or not merge groups as an alternate method of searching. If the number of groups is more, you can narrow down patient search by specifying the Patient ID as search criteria.

  1. Enter the valid Patient ID in alphanumeric format of length as specified in Master Patient Index module, or click [?] to select the required Patient ID.

  2. On entering a valid Patient ID, click [Search]. For the selected patient, if records exist in the search result, the records details are displayed, or the system displays an appropriate error message.<Related Field Description>

  3. Click [Reset] to clear the Data entered.
