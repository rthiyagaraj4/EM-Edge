Location for Practitioner

From this function you can Create an association of a Practitioner to a specific Location for an effective period. You can Query for a Practitioner associated to a Location and Delete an association if required.

The locations to which practitioners can be associated are specific Clinics, Nursing Units, All Clinics, All Nursing Units, All Locations or based on Practitioner Rights. When a Practitioner is associated to a Location, the Association enables the Practitioner to provide care for patients in those Locations.

Creating an association of Practitioners to Locations

From the CA Code Setup, select Location for Practitioner function.

The Location for Practitioner screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the association of a Practitioner to a Location or [Reset] to clear the selected data.

  4. On clicking [Apply], the Location Type, Location and Effective Date Range are displayed for the selected Practitioner as recorded.

  5. Click [Query] to search for existing records.

  6. Click [Delete] to delete a queried record.
