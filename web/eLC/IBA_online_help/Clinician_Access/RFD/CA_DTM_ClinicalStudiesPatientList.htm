Enter the following details:

 

  1. View – Select the value from the list.. The values listed are ‘All’ and ‘Self Related’.

  2. Restrict To – Select the Restrict To value from the list..

  3. Specialty – Select the Speciality using the [?] function.

  4. Gender – Select the Gender from the list. The listed values are ‘Male’, ‘Female’ and ‘Unknown’.

  5. Age Range – Enter the numeric value of the age in the format displayed.

  6. Period Range – Select the interval date using the calendar function. This is a mandatory field.

  7. Status – Select the Status from the list. The values listed are ‘Open’ and ‘Closed’.

    • If ‘Open’ is selected all the patients with clinical studies are displayed.

    • If ‘Closed’ is selected the list of patients for whom the End Date is recorded in the Clinical Studies function gets displayed.

  8. Research Category – Select the Research Category from the list. The list values are populated from the Research Patient Category function. This is a mandatory field.

  9. Research Category Factors – Select the Research Category Factor from the list. The list values are populated from the Research Patient Category Factor function. This is a mandatory field.

  10. Diagnosis – Select the Diagnosis value from the list. This is a mandatory field.

  11. Group By – Select the Group By from the list. The listed values are ‘Recorded Practitioner’ and ‘Research Category’.
