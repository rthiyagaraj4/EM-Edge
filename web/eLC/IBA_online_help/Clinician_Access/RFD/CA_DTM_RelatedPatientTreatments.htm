Enter the following details:

 

  1. View – Select All or Self Related from the View list.

  2. Restrict To – Select Login Facility or Enterprise from the ‘Restrict To’ list to restrict the searching process.

  3. Specialty – Select the Specialty using [?] function.  

  4. Gender – Select the desired Gender from the list. The valid values are Male, Female, Unknown and All.

  5. Age Range – Specify the desired Age Range to search for a list of patients.

  6. Period Range – Specify the Period Range to search for patient records satisfying the specified date range. This field is mandatory.

  7. Status – Select the desired Status from the list. The valid values are Active and Deceased.

  8. Diagnosis – Select the Diagnosis from the list. Search and select the desired Diagnosis to search for related patient records. The system allows selecting a set of three diagnoses. This field is mandatory.

  9. Group By – Select the desired value from the list to group the search results. The valid values are Patient, Diagnosis, and Disposition Type.
