Enter the following details to Assign CA Menu:

 

  1. Facility – Select the required facility from the list to associate it to a preferred CA Menu. The facilities to which the logged-in user is associated through the ‘Facilities for User’ function in System Manager module are listed. ‘All’ is selected by default.

  2. Patient Class – Select the required Patient Class from the list. The listed values are Day Care, Emergency, External, Inpatient and Outpatient. ‘All’ is selected, by default.

  3. Practitioner Type – Select the required Practitioner Type from the list. All’ is selected, by default.

  4. Practitioner – Enter the required Practitioner name in alphanumeric format or click [?] function to search and select the required Practitioner. All the Practitioners corresponding to the selected Practitioner Type are displayed All’ is selected, by default.

  5. Menu – All the Menus created using ‘Define Menu’ are listed in the list. Select the required ‘Root Menu’ from the ‘Menu’ list. This field is mandatory.

  6. Rename Summary Menu – Enter the required Summary menu name in alphanumeric format of 30 characters maximum length.

  7. Rename Navigation Menu – Enter the required Navigation menu name in alphanumeric format of 30 characters maximum length.
