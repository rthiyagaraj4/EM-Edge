Enter the following fields:

 

  1. Search Criteria – Referred From, Referred To – By default, the ‘Referred From’ check box is selected. You can select the Referred To check box if required.

  2. Period – Displays the past one week duration from current date. This is a mandatory field. You can change the date period if required.

  3. Forward From Facility, Forward To Facility – Select the Forward From and Forward To Facility from the drop-down list. Indicates the facility from which the clinical note referral request is forwarded to the other facility. By default, the logged-in facility is displayed.

  4. Referred From Facility, Referred To Facility – Select the Referred From and To Facility from the drop-down list. By default, the logged-in facility is displayed.

  5. Note Status – Select the Status of the Clinical Notes to be searched. The list includes Authorized, Unauthorized, Transcribed, Modified and All. By default, the value ‘All’ is listed.

  6. Practitioner – Enter the practitioner name or select using the [?] function.

  7. Patient ID – Enter the ID of the patient whose clinical notes has to be searched.

  8. Patient Class – Select the required Patient Class from the list. The list includes All, Daycare, Emergency, Inpatient and Outpatient. By default, the value ‘All’ is displayed.

  9. Referral Status – Select the status of the referred clinical notes from the drop-down list. The list includes All, Reviewed, Not Reviewed. By default, the value ‘All’ is displayed.

  10. Location – Select the required location from the drop-down list. This is a mandatory field. By default, the value ‘All’ is displayed.
