New List

The New List Tab is displayed, by default. The New List Tab displays patients based on the query criteria entered. The patients listed are based on value set for Minimum Duration Required between Consultations in the Consultation Parameter Setup function.

 

  1. Specialty – Select the specialty using [?] function.

  2. Patient ID – Select the Patient ID using [?] function.

  3. Location – Select the Location as Nursing Unit. Based on the location selected, the [?] function displays locations for selection.

  4. Service – Select the Service using [?] function. The Services associated to the selected Location is displayed for selection.

  5. Click [Search] to display patients based on the entered search criteria. Click [Clear] to clear the search.

  6. The Patient details displayed are Name, Age, Gender, Patient ID, Admission Date, Specialty and Location. The check box against the patients are selected, by default. Clear the check box to exclude the patient while generating Consultation List.

  7. On searching Consultation List by Specialty, patients are listed for the selected Specialty and grouped by Practitioner. Select or clear the check box against Patients for consultation under each Practitioner.

  8. The patients that have exceeded the maximum number of Consultation by same Practitioner and Specialty are displayed in a different colour and the check box is disabled.

  9. Patients with Bedside Referral are displayed with a hyperlink. Click the hyperlink to display referral details.

  10. Click [Next] to display the patients for generating Consultation List in the next screen.

  11. Click [Generate List] to generate the Consultation List. Click [Back] to go back to the previous screen.

  12. A Generate List screen is displayed on clicking [Generate List]. Enter the Consultation List name in alphanumeric format of 15 characters maximum length. The List ID is the system date and time on which the Consultation List was generated. Click [OK] and the List ID is displayed in the Existing List Tab. Click [Close] to generate the List ID without the Consultation List name. A generated list cannot be amended. You can edit the charges to be posted for the generated list.

  13. Click [Print] to print the Consultation List along with the new Consultation List name.
