Present Obstetrics Details

Enter the following details:

  1. Partners Name – Enter Partners Name of the patient.

  2. Partners Occupation – Select the occupation of the partner from the [?] function. The occupations are defined in the MP Code Setup of Master Patient Index module.

  3. Married Life – Enter the number of years of married life of the patient.

  4. Consanguineous Marriage – Select either Yes or No from the list box indicating whether the patient had a consanguineous (Related by blood) marriage or not.

  5. Amenorrhea Since – Amenorrhea is the absence of a menstrual period in a woman of reproductive age. Enter the number of days since amenorrhea occurred.

  6. LMP – Select the Last Menstrual Period date from the calendar function.

  7. Gestation Age – On entering the LMP date, the Gestation Age is automatically displayed in days.

  8. EDD – The Expected Date of Delivery (the estimated average date when a woman will give birth) is automatically displayed on entering the LMP date.

  9. Menstrual Cycle – Enter the number of days of menstrual flow and select the menstrual cycle in days from the list box. The list box displays values from 24 – 45 days.

  10. Latest Scan Date – Select the latest date when the patient was scanned from the calendar function.

  11. Corrected EDD – Select the corrected Expected Date of Delivery from the calendar function.

  12. Scan Report – Select the scan report by clicking the icon. The Link Result Documents screen gets displayed. Search and select the scan report from the screen if any.

  13. Precious Pregnancy – Select the check box if the patient has not conceived for long and the precious pregnancy needs extra care. If Precious Pregnancy is selected, a star icon gets displayed next to the event header on recording.

  14. Elderly Primi – Select the check box if the patient has conceived after an age of 35-40 years.

  15. Planned Pregnancy – Select the check box if the patient has a planned pregnancy.

 

Present Obstetrics Details

On recording the Present Obstetrics Details you can view the details captured for that encounter even after the event is completed.

Past Obstetrics History

Click Past Obstetrics History to record past obstetrics details for the event Ectopic Pregnancy.

 

Enter the following details:

  1. Infertility – Select the check box if the patient has a history of infertility.

  2. Years of Infertility – Enter the number of years of infertility.

  3. Contraception – Select the check box if the patient has a history of contraception or birth control.

  4. Contraceptive History – Select the contraceptive history from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  5. Since – Select the date since when the contraceptive was used.

  6. Remarks – Enter remarks of contraception if any.

  7. Click the hyperlink to record more than one contraceptive history details. The + hyperlink displays as hyperlink. Click + hyperlink to record more contraceptive history details and click the – hyperlink to record less contraceptive history details.

  8. Gravida – Enter the number the times the patient has been pregnant in alphanumeric format of 20 characters maximum length. Gravida 0 is a patient, who has never been pregnant. Gravida 1 is a patient, who is pregnant for the first time or has been pregnant one time. Gravida 2, 3 is a patient, who has been pregnant more than one time.

  9. Parity – Enter the number of times the patient has given birth in alphanumeric format of 20 characters maximum length. Para 0 is a patient, who has never given birth. A woman, who has given birth a particular number of time is referred to as Para 1, Para 2, Para 3 and so on.

  10. Abortion – Enter the abortion history of the patient in alphanumeric format of 20 characters maximum length.

  11. Year – Enter the year of past pregnancy.

  12. Pregnancy Event – Select the Pregnancy Event from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  13. Labour Event - Select the Labour Event from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup. Example. Normal, Prolonged

  14. Method of Delivery – Select the Method of Delivery from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup. For example, Normal Vaginal, Breech Delivery, Forceps Delivery.

  15. Peuperium – Select the Peuperium from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  16. Outcome – Select the Outcome of the Pregnancy Event. Example Live Birth, Still Birth.

  17. Sex of the Child – Select the sex of the child as Male, Female or Indeterminate.

  18. Click the hyperlink to record more details. The + hyperlink displays ashyperlink. Click + hyperlink to record more pregnancy event details and click the – hyperlink to record less pregnancy event details.

 

Pregnancy Risk Assessment

Click Pregnancy Risk Assessment to record risk assessment details related to pregnancy for the event Ectopic Pregnancy.

 

Enter the following details:

  1. Medical History – Select any of the check boxes against the following to record risk assessment based on medical history. Anaemia, Diabetes Mellitus, Hypertension,  Epilepsy, Asthma, Tuberculosis, Cardiac Disease, Kidney Disease, Thyroid Disease, Deep vein Thrombosis, Obesity, Major Depression, Anxiety Disorder, Phenyl Ketonuria, Systemic Lupus erythematosis, and Haemoglobinopathies.

  2. Menstrual History – Select any of the check boxes against the following to record Menstrual History. Infertility History, Recurrent Pregnancy Loss.

  3. Tobacco, Alcohol and Substance use – Select  Alcohol, Tobacco and/or Substance Abuse to record risk assessment as a result of it. Enter the required Details. This is a mandatory field.

  4. Sexually Transmitted Diseases – Enter Sexually Transmitted Disease details in the text area provided.

  5. Immunisation – Select any of the check boxes against the following to record risk assessment as a result of immunization. Rubella, Vericella Vaccine, Tetanus Vaccine, Pneumococcal Vaccine, Hepatitis B Vaccine, Influenza Vaccine.

  6. Occupational Exposure – Select the check box against any of the following under occupational exposure to indicate risk assessment. Teratogen Exposure, Metals, Solvents, Plastic, Pollutants, Pesticides, Gases, Radiation.

  7. Family History related risk – Select any of the check boxes to record risk assessment based on family history. Diabetes, Tuberculosis, Chronic Hypertension, Jaundice. Specify if any other family history related risk in the ‘Others’ text area provided.

  8. Screen for Ethnic Back Ground – Select the check box against Sickle Cell Trait, Tay Sach Disease, Blood Dyscrasias and/or Alpha or Beta Thalasemia to record risk assessment.

  9. Risk Assessment Result – Select the Risk Assessment Result as Low Risk, Moderate Risk, High Risk or Others. Enter details if Others is selected. If the result is recorded as High Risk, a High Risk icon is displayed next to the event header on recording.

  10. Remarks – Enter remarks, if any.

 

On entering the required details, click [Record] to save details of first visit recorded for the event Ectopic Pregnancy. Click the required option hyperlink and the details recorded for First Visit of event Ectopic Pregnancy are displayed on the right side of the screen.
