1. Symbol Group ID - Enter the code in alphanumeric format of four ten characters maximum length. This code is to identify the Symbol Group. The entered code must be unique and cannot be changed after creation. This field is mandatory.

2. Symbol Group Desc – Enter the description for the Symbol Group in alphanumeric format of 30 characters maximum length. This field is mandatory.

3. Type ¬ Select the type of the symbol group from the drop-down list. You can select either ‘Note Group’ or ‘Note type’.

• Note Group – If you select ‘Note Group’, you can click [?] and select a note group pick list to which the symbol group will be associated.

• Note Type - If you select ‘Note Type’, you can click [?] and select a note type from the pick list to which the symbol group will be associated.

4. Add- Click this button to add the selected note group or note type to the grid. You can add multiple note groups and types to the grid.

5. Enabled – Select this check box to activate the symbol group. Only active symbol groups will be available within the system.

6. Grid – Displays the added note groups/note type with the following columns:

• Note Indicator – Displays the type.

• Note type/Note Group - Displays the description of the selected note group/note type.

7. Remove – Select this checkbox in the grid to remove the added note group/note type from the grid.
