Enter the following report criteria:

 

  1. Patient ID – The Patient ID gets auto populated.

  2. Nursing Diagnosis – Select the required nursing diagnosis from the lookup.

  3. Status – Select the required status from the drop-down list. By default, the value ‘All’ is displayed.

  4. Goal Statement – Select the required goal statement from the drop-down list.

  5. Intervention – Select the required intervention from the lookup.

  6. Date Range - By default it displays past 3 months date duration with the current system date. This field is mandatory. The ‘To Date’ cannot be greater system date.
