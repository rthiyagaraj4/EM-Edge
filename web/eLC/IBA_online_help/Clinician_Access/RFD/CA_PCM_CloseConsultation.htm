Enter the following details:

 

  1. Visit Type - Visit Type of the Visit is displayed.

  2. Practitioner - The assigned Practitioner Name is displayed.

 

Disposition

  1. Date/Time - The current date/time is displayed as the disposition date and time.

  2. Type - Select the Disposition Type from the list. The displayed list is limited to a set of active Disposition Types valid for Visit Conclusion.

 

Case Definitions

  1. Medical - Select the check box to indicate the case requires only medical attention.

  2. Surgical - Select the check box to indicate the case requires surgical attention.

  3. Circumstance of Injury - Select the Circumstance of Injury from the list.

 

In the Event of Death

  1. Brought Dead - This field is disabled for Outpatients.

  2. MLC - Select the check box to indicate the patient is registered as Medico Legal Case.

  3. Postmortem Required - Select the check box, if postmortem is required for the deceased body. This field is enabled only if the 'Type' selected is deceased.

  4. High Risk body - Select the check box to indicate the deceased body is a high risk body.

  5. Deceased Date/Time - Enter the deceased date and time of the patient. If the 'Type' is selected as deceased, this field becomes mandatory.

  6. Body Release Date/Time - If the 'Type' is selected as deceased, this field becomes mandatory and the current system date/time is populated automatically.

 

Recall Details

If the Patient is required to come back to the clinic, the following details can be entered:

  1. Come Back/After - Select the check box to indicate whether the Patient has to come back again on a future date. If the check box is selected, enter the number of days within which the patient has to come back in numeric format of three digits maximum length in 'Day(s)', 'Week(s)' or 'Month(s)'. This field is mandatory on selecting the 'Come Back' check box. This field is disabled for Emergency patients.

  2. Reason - Enter the Reason for recalling the Patient in alphanumeric format of 200 characters maximum length.

  3. Instructions - Enter the Instructions of Recall to the Patient in alphanumeric format of 200 characters maximum length.

  4. Book Appointment - This button is enabled on selecting the 'Come Back' check box. Appointment is booked in advance for patients who are asked to come back on a future date using this button. On clicking [Book Appointment], diary of the currently registered Location/Practitioner is invoked, if schedule has been generated in 'Appointment Scheduling' module. The system highlights the earliest available slot based on the come back period defined.
