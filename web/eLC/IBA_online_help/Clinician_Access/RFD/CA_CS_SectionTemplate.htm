Enter the following details to create a Section Template:

 

  1. Section - Select the Section from [?] function. This field is mandatory.

  2. Item Type - Select the Item Type from the list. The applicable values are 'Component' or 'Title'. This field is mandatory.

  3. Title Text - This field is enabled and is mandatory for entering the values only on selection of the Item Type as Title. Click [?] function and the Title screen is displayed. Enter the required Title Text in the Message field in alphanumeric format of 200 characters maximum length and click [Record]. The entered text is displayed on the top part of the Title screen. Select the check box under the Select header to display the text in the Title Text field of Section Template screen.

  4. Accept Option – Select an Accept Option from the list. On selecting the Item Type as Component, the list displays ‘Optional’ and ‘Required’. On selecting the Item Type as Title, the list automatically displays ‘Display Only’.

  5. Include for Analysis – Select the check box if the selected Item Type has to be selected for analysis. This field is displayed only if the Item Type is selected as Component.

  6. Component - This field is enabled on selecting the Item Type as Component. Enter the required component details in alphanumeric format of 20 characters maximum length or select the required component by clicking [?] function. All components are defined in the Discrete Measure function of Application Masters module. This field is mandatory.

  7. Prompt – Select the Prompt from [?] function. The Prompt screen displays the prompt or the legend that has been associated with the component. Enter the required Prompt in the Message field in alphanumeric format of 200 characters maximum length and click [Record]. The entered text is displayed in the top part of the Prompt screen. Select the check box under the Select header to display the prompt in the Prompt field of Section Template screen.

  8. Applicable Age Group – Enter the applicable Age Group to be associated to the selected component in alphanumeric format or click [?] function to select the required Age Group. All the Age Groups are defined in ‘Application Masters’ module.

  9. Result Category – This field is enabled on selecting a Component Type defined as ‘Long Text’. Select the required Result Category to be associated to the Long Text from the list. The listed values are Adverse Event, Blood Transfusion, Dept Diagnostics, Dietary, Free Documentation, Future Events, Laboratory, Medication, Monitoring, New Born, Patient Care, Problem/Diagnosis, Radiology/Imaging, Surgery/Procedure and Treatment.

  10. Auto Fill  Active Diagnosis – This check box is visible only when the Item Type is ‘Long Text component’ and Result Category is selected as 'Problem/Diagnosis'. By default, the check box is not selected. Select the check box to display the active diagnosis automatically while Recording Clinical Notes for the Note Type with Template based note and Result Linked note.

  11. Row Position - Enter the Row Position in numeric format of two digits maximum length. The row position denotes the position of the component in the template. This field is mandatory.

  12. Column Position - Enter the Column Position in numeric format of one digit maximum length and the valid values are One, Two and Three. This field is mandatory.

  13. Column Span - Enter the Column Span in numeric format of one digit maximum length and the valid values are One, Two and Three. This field is mandatory.

  14. Label Position - Select the label position from the list. The applicable values for 'Component' are Left. The applicable values for 'Title' are Left, and Center.

  15. Hint Message – Select the Hint message from [?] function.

  16. Text Alignment - This check box appears only when Long and Short text component is selected in ‘Component’ field. Select the check box to perform text alignment for short and long text in discharge summary. By default, the check box is not selected.
