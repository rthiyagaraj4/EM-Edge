Additional Details

  1. Status – By default, the Active radio button is selected in the Create mode. In Modify mode Active, Resolved and In Error radio buttons are displayed. If the status is marked as 'In error', a field Reason is displayed to record the reason for error.

  2. Since – Select the date from the Calendar function, since when the adverse event is active. This field is mandatory.

  3. Reason – Enter the Reason for the error. This gets displayed in the Adverse Event screen.

  4. Certainty – Select the Certainty as ‘Definitive’ or ‘Probable’. By default, Certainty is selected.

  5. Diagnosis - Select the diagnosis from [?] function. If there are any diagnoses recorded that are active for the patient, select the diagnosis. If there are no active diagnoses recorded for the patient, select a diagnosis from the Record Diagnosis screen. If active diagnoses are recorded for the patient, select the diagnosis to associate with the allergy being recorded.

  6. Route of Exposure – Select the route of exposure to the allergy from the list of hard coded values that are Ingestion, Inhalation, Injection, Tropical and Unknown.

  7. Source of Info – Select any of the radio buttons as the source of Information from the options provided. The options are Patient Observation, Allergy Test Result and Staff Observation. Based on the selected option, the field legend changes to enter the details in the respective text area.

  8. Patient Observation - Enter the observation details of the patient.

  9. Treat Advice/Action Taken – Enter the treatment advices given to the patient.

  10. Remarks – Enter the remarks for the allergy being recorded.

 

The Reason gets displayed only when the Status is selected as ‘In Error’. When the status other than In Error is selected, Reason is not displayed.
