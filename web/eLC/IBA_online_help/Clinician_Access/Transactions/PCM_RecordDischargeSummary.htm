Record Discharge Summary

From this function you can Record Discharge Summary for a patient.

Using this function, a practitioner can easily consolidate a summary of events recorded during the encounter. The system allows recording of a discharge summary only after discharge advice is prepared in case of an inpatient encounter. The clinical note that appears for discharge summary recording is based on the note parameter setup.

Recording Discharge Summary

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Discharge Activities, select Record Discharge Summary function.

The Record Discharge Summary screen gets displayed.
