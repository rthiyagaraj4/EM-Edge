Blood Collection Slip

From this function you can Request/Reprint the blood collection slip to collect the reserved blood units from the Blood Transfusion Department.

The Blood Collection Slip function is used to request/reprint the blood collection slip to collect the reserved blood units from the Blood Transfusion department.

Requesting/Reprinting the Blood Collection Slip

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Order Management > select Blood Collection Slip function.

The Blood Collection Slip screen gets displayed.

  1. Select the Blood Collection Slip status from the list. The listed values are ‘Reserved Not Requested’, ‘Reserved already Requested’ and ‘Reserved and Or Requested’. The ‘Reserved Not Requested’ status indicates that blood collection slip is not requested for the patient. You can request and reprint the blood collection slip in this status. The ‘Reserved already Requested’ status indicates that blood collection slip is already requested for the patient. You can reprint the blood collection slip in this status. When you select ‘Reserved and Or Requested’ status, the records with both the above status are displayed.  

  2. Click [Search] to view the Blood Collection Slip details. The details of the blood units issued for the selected patient from the 'Allocate/Xmatch/Issue - Blood Units' function in Blood Transfusion Services module are displayed. <Related Field Description>

  3. Click [Request] to request for the Blood Collection Slip. After you request for the Blood Collection Slip, the unit details are displayed under the ‘Reserved already Requested’ status. On completion of reprinting, the message Process Completed Successfully’ is displayed.  

  4. Click [Reprint] to reprint the Blood Collection Slip. On completion of reprinting, the message Process Completed Successfully’ is displayed.
