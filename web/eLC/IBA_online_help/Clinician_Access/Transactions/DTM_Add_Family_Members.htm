Add Family Member

This screen is used to add family members to the index patient. A family member with respect to whom genogram will be displayed is called as Index Patient. An index patient is represented as Double Circle or Double Square in a Genogram.

Also, you can link the family members to index patient if they are previously registered in the system and having a genogram ID.

 

  1. Select the Genogram of other family member exists option.  By default this option is set as ‘No’.

  2. When the Genogram of other family members exist option is set as ‘No’, the Non-Patient ID field is disabled. First, add the patient as an index patient.

    1. Select the Patient ID of the member to be added from the lookup. The name, age, gender details gets populated automatically. This is a mandatory field.

    2. Click Add Member to add the patient as an Index patient. The Patient Names get added in the Family Members grid. The index patient is represented byicon. Now, you can add the other family member details to the index patient.

    3. Select the Relation Type of the family member to be added to the patient from the drop-down list. The listed values are Children, Couple and Parents. This is a mandatory field.

    4. If the Relation Type is selected as ‘Parents’, then you need to enter the Mother and Father details  in the ‘Family Demographics’ section.

      Enter the father and mother details as follows:

      1. Patient ID – Select the patient ID of the patient’s parent from the lookup, if available. The Name, Date of Birth and Gender get populated automatically.

      2. Name - Enter the name of the patient’s parent. This is a mandatory field.

      3. Date of Birth – Select the date of birth of the patient’s parent by clicking calendar icon. This is a mandatory field.

      4. Gender – The Gender is displayed by default.

      5. Deceased Date/Time – If the parent is deceased, enter the deceased date and time by clicking calendar function.

      6. Deceased Remarks – Enter remarks for the deceased, if required.

      Relation between Father and Mother

      1. Relation - Select the relation between the father and mother from the drop-down list. This is a mandatory field.

      2. Relation Start Date - Select the relation start date of the parents by clicking the Calendar function. This is a mandatory field.

    5. If the Relation Type is selected as ‘Couple,’ then you need to enter the couple details  in the ‘Family Demographics’ section.

      Enter the couple details as follows:

      1. Relation – Select the relation between the couple from the drop-down list. This is a mandatory field.

      2. Patient ID – Select the patient ID of the spouse from the lookup, if available. The Name, Date of Birth and Gender get populated automatically.

      3. Relation Start Date – Enter the relation start date of the couple by clicking calendar function.

      4. Name - Enter the name of the spouse. This is a mandatory field.

      5. Date of Birth – Select the date of birth of the spouse by clicking calendar icon. This is a mandatory field.

      6. Gender – Select the Gender from the drop-down list. The listed values are Male, Female and Unknown. This is a mandatory field.

      7. Deceased Date/Time – If the spouse is deceased, enter the deceased date and time by clicking Calendar function.

      8. Deceased Remarks – Enter remarks for the deceased, if required.

    6. If the Relation Type is selected as ‘Children,’ then you need to select the partner name and ‘No of Children’ from the respective drop-down lists. You need to enter the Child Details  in the Family Demographics section.

      Enter the child details as follows:

      1. Relation – Select the relation of the child with the parents i.e., son or daughter from the drop-down list. This is a mandatory field.

      2. Sibling Position – Enter the sibling position. This is a mandatory field.

      3. Patient ID – Select the patient ID of the child from the lookup, if available. The Name, Date of Birth and Gender get populated automatically.

      4. Name - Enter the name of the child. This is a mandatory field.

      5. Date of Birth – Select the date of birth of the child by clicking Calendar icon. This is a mandatory field.

      6. Gender – Select the gender of the child from the drop-down list. This is a mandatory field.

      7. Deceased Date/Time – If the child is deceased, enter the deceased date and time by clicking Calendar function.

      8. Deceased Remarks – Enter remarks for the deceased, if any.

    7. Click [Add Member]. The new member detail such as Patient ID, Name, Age, Gender, Deceased Date/Time, Relation and Relation With gets added in the ‘Family Members’ 'grid.

    8. If you want to update the member details, double-click the required family member in the grid and corresponding member details gets populated in the respective fields in the ‘Family Member Details’ section. Do the required changes and click [Modify Member].

  3. When the Genogram of other family members exist option is selected as ‘Yes’, then the field ‘Non-Patient ID’ field gets enabled.

    1. Select the patient name of the existing family member from the ‘Non-Patient ID’ lookup.

    2. On selecting the patient name, the family member details will be listed in the ‘Family Members’ grid.

    3. The system allows you to add new members to the existing family members. Select the family member in the ‘Family Members’ grid by highlighting the row. Now, you can add new family members to the highlighted member.

    4. Select the Relation Type from the drop-down list. This is a mandatory field. The listed values are Couple, Children and Parents.

    5. If the Relation Type is selected as ‘Parents’, then you need to enter the Mother and Father details  in the ‘Family Demographics’ section.

      Enter the father and mother details as follows:

      1. Patient ID – Select the patient ID of the patient’s parent from the lookup, if available. The Name, Date of Birth and Gender get populated automatically.

      2. Name - Enter the name of the patient’s parent. This is a mandatory field.

      3. Date of Birth – Select the date of birth of the patient’s parent by clicking calendar icon. This is a mandatory field.

      4. Gender – The Gender is displayed by default.

      5. Deceased Date/Time – If the parent is deceased, enter the deceased date and time by clicking calendar function.

      6. Deceased Remarks – Enter remarks for the deceased, if required.

      Relation between Father and Mother

      1. Relation - Select the relation between the father and mother from the drop-down list. This is a mandatory field.

      2. Relation Start Date - Select the relation start date of the parents by clicking the Calendar function. This is a mandatory field.

    6. If the Relation Type is selected as ‘Couple,’ then you need to enter the couple details   in the ‘Family Demographics’ section.

      Enter the couple details as follows:

      1. Relation – Select the relation between the couple from the drop-down list. This is a mandatory field.

      2. Patient ID – Select the patient ID of the spouse from the lookup, if available. The Name, Date of Birth and Gender get populated automatically.

      3. Relation Start Date – Enter the relation start date of the couple by clicking calendar function.

      4. Name - Enter the name of the spouse. This is a mandatory field.

      5. Date of Birth – Select the date of birth of the spouse by clicking calendar icon. This is a mandatory field.

      6. Gender – Select the Gender from the drop-down list. The listed values are Male, Female and Unknown. This is a mandatory field.

      7. Deceased Date/Time – If the spouse is deceased, enter the deceased date and time by clicking Calendar function.

      8. Deceased Remarks – Enter remarks for the deceased, if required.

    7. If the Relation Type is selected as ‘Children,’ then you need to select the partner name and ‘No of Children’ from the respective drop-down lists. You need to enter the Child Details  in the Family Demographics section.

      Enter the child details as follows:

      1. Relation – Select the relation of the child with the parents i.e., son or daughter from the drop-down list. This is a mandatory field.

      2. Sibling Position – Enter the sibling position. This is a mandatory field.

      3. Patient ID – Select the patient ID of the child from the lookup, if available. The Name, Date of Birth and Gender get populated automatically.

      4. Name - Enter the name of the child. This is a mandatory field.

      5. Date of Birth – Select the date of birth of the child by clicking Calendar icon. This is a mandatory field.

      6. Gender – Select the gender of the child from the drop-down list. This is a mandatory field.

      7. Deceased Date/Time – If the child is deceased, enter the deceased date and time by clicking Calendar function.

      8. Deceased Remarks – Enter remarks for the deceased, if any.

    8. Click [Add Member]. The new member detail such as Patient ID, Name, Age, Gender, Deceased Date/Time, Relation and Relation With gets added in the ‘Family Members’ 'grid.

    9. If you want to update the member details, double-click the required family member in the grid and corresponding member details gets populated in the respective fields in the ‘Family Member Details’ section. Make the required changes and click [Modify Member].

  1. Once the family members are added to the index patient, the genogram is refreshed and displays all the family members of the index patient as a family structure.
