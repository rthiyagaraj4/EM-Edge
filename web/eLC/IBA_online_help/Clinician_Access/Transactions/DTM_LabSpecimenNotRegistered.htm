Lab Specimen Not Registered

From this function you can View lab specimens that are not registered in the laboratory.

This function allows the nurses to view the lab orders which are not registered when the time crosses the grace time defined in ‘Time Frame to alert Urgent/Stat Lab Specimen X minutes’ field in OR Parameter for Facility screen.  The nurses will alert or inquire the lab department why specimens are not registered. Once registration is done, the record gets removed from the Lab Specimen Not Registered function. This is applicable only for urgent and stat orders.

The orders placed only for that location gets displayed for the Nurses in their Desktop menu.

 

This function appears in the Desktop menu only if you configure the ‘Lab Specimen Not Registered’ under the Summary Links section in the Create User Desktop screen in SM module.

Viewing Lab Specimen Not Registered

From the CA Transactions > Clinician Access > Desktop Menu > Summary > Lab Specimen Not Registered function.  

The Lab Specimen Not Registered Summary screen gets displayed.

  1. By default, the system displays the lab specimens that are not registered in the lab based on the ‘Time Frame to alert Urgent/Stat Lab Specimen X minutes’ parameter.  

  2. Enter the query criteria and click [Search] to display the order details of lab specimens that are not registered. Click [Clear] to clear the displayed data.

    1. Patient ID – Enter the patient ID from the lookup.

    2. Specimen No – Enter the specimen number in the text box.

    3. Time Frame to alert Urgent/Stat Lab Specimen – Enter the time range to check for specimen records which are not yet registered. You can enter value from 1 to 9999.

    4. Priority – Select the priority from the drop-down list. The list values include Urgent, Stat and Both. By default, the value ‘Both’ is displayed.

    5. Order ID – Enter the Order ID in the text box.
