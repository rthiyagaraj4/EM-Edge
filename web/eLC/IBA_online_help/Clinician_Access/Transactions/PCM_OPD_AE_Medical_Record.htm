OPD/AE Medical Record

From this function you can View, Record and Print the entire clinical history of a patient from a single screen.

From the OPD/AE Medical Record function, you can view, record and print encounter wise event history in a single page for consultants.

 

This function ensures that the complete clinical history of a patient is available in a single screen. From this function, you can record only for the current encounter and will not be allowed to record for the previous encounters.

 

The Encounter Tabs will display different colours according to the patient class registered against respective encounters. In the Encounter tab header, you can view the Visit Type, Medical Service, Attending Practitioner, Encounter ID and Visit Date.

 

You can view maximum of four encounters in a set and click Next button to access the next set of encounters. On clicking the respective encounter tab, the Patient Clinical Details are displayed in the following order: Adverse Event (Allergies/Sensitivity), Initial Assessment , Personal and Family History, Clinical Note for Physician, Results, Diagnosis, Existing Order, Procedures Details, Medication Certificate, Clinical Note for Nurse, Patient Appointment.

 

These sections are displayed continuously without any scroll bars within the Section details.

The system allows you to select/clear any section of OPD / AE Medical Record for printing. By default, you can print the all the sections of the OPD/AE Medical Record.

Viewing OPD/AE Medical Record

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > select OPD/AE Medical Record icon.

The OPD/AE Medical Record screen gets displayed.

  1. The screen automatically displays the current encounter details of the patient.

  2. Click respective encounter tab to view the current/past encounter details.

  3. On clicking the respective encounter tab, the Patient Clinical Details are displayed in the following order:

Adverse Event  (Allergies/Sensitivity)

This section displays all the active allergies captured for the patient irrespective of the encounter.  You can record allergy details by clicking in the Adverse Event section. The Adverse Event  screen gets invoked where you record the required allergy details.

Initial Assessment  

This section displays the data recorded for vital sign battery and for current Encounter only. The recorded vital signs will be displayed in the main section.

Click icon, the Vital Signs  dialog gets displayed where you can record vital signs.

 

An ‘Add Discrete Measure’ button is available in the Vital Signs dialog to add vital sign during recording of assessment. The adhoc entry of vital signs is applicable only for the selected encounter and not for the vital sign battery.    

Personal and Family History

This section displays the personal and family history of the patients. The Personal & Family History section is a Note Type used to capture patient Personal, Social, Interpersonal and Social History in order to access the patient clinical details.

The Personal and Family History Note Type can be configured such that Personal History event, event details, Other details and Onset Date & Time would be part of Personal History while Relationship, disease, Other details and Onset date and time would be part of Family history.

Click  icon . The Record Social History  dialog gets displayed. You can capture the Personal and Family History of the patient for the current encounter.

Clinical Note for Physician

This section displays only the physician clinical notes recorded for the current encounter for the patient. You can upload image using the Clinical Note upload image functionality. From this section, new clinical notes can be recorded and it will be displayed only after the Clinical Note Entry screen is closed.

Click icon. The Clinical Note for Physician  dialog gets invoked where new clinical notes can be recorded.

Results  

This section displays the lab, radiology and OT results recorded for the patient for the current encounter. When results against any order is updated, this section gets updated.

Click  icon to open the Lab Results dialog to view the lab results for that encounter.

Click  icon to open the Radiology Results dialog to view the radiology results for that  encounter.

Click  icon to open the OT Results dialog to view the OT related results for that encounter.

Diagnosis  

This section displays all the diagnosis recorded for the current encounter. The diagnosis data include Set, Diagnosis/Problem, Diagnosis Description, Diagnosis classification, Practitioner and Recorded date and time.

Click  icon to open the Record Diagnosis  dialog to record the diagnosis for the patient for the current encounter.

Click icon to open the Record Chief Complaint  dialog to record the chief complaints for the patient for the current encounter.

Existing Order

This section displays all pending ordres recorded by nurse for the patient for the current encounter only. From this section, you can place Pharmacy order or any other order for the patient.

Click to open the Existing Orders  dialog to place new orders.

Click icon to open the Pending Orders - Laboratory dialog to view the lab orders pending for that encounter.

Click icon to open the Pending Orders - Radiology dialog to view the radiology orders pending for that encounter.

Click icon to open the Pending Orders - OT dialog to view the OT orders pending for that encounter.

Procedures Details

This section displays all the procedure recorded for the patient for current encounter only. The procedure data include Procedure Code, Description, Date and Time, Findings, Impression and Practitioner.

Click icon to open the Procedures dialog where the Procedure done for the patient can be recorded.

Medication Certificate

This section displays the medical certificate recorded for the patient for the current encounter only. The displayed data includes Date/Time, Note Type, Ref No, Med/Anc Service, Title, Performed By, Authored By and Status.

Click  icon to open the Medication Certificate  dialog where you can capture the medication certificate for the patient.

Clinical Note for Nurse

This section displays the clinical note recorded for the patient for the current encounter only. You can upload image using the Clinical Note upload image functionality. You can record new clinical notes from this section and it gets displayed once the Clinical Note Entry screen is closed.

Click  icon to open the Clinical Notes for Nurse  dialog where you can record the clinical note for the patient.

Click  icon. The Clinical Event History dialog gets displayed; showing the Nursing Notes recorded for that encounter.

Click  icon. The Clinical Event History dialog gets invoked where you can view the clinical data for that encounter.

Patient Appointment

This section displays all the future patient appointment recorded for the patient.

Click  icon, the Book Appointment  dialog gets displayed. You can record new appointments for the patient. The new appointment gets updated in this section once the Book Appointment screen is closed.

  1. Click [Print Preview] to preview the patient clinical history detail in a separate window.

  2. Click [Print] to print the entire patient clinical history. If you want to print only specific section, select the check box against those sections alone and click [Print].
