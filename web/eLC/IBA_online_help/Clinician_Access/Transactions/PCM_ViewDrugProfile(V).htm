View Drug Profile (V)

From this function you can View the drug profile of a patient.

This function helps in viewing details of prescribed or dispensed drugs for the selected patient.

Viewing Drug Profile

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Event Details, select View Drug Profile (V).

The Drug Profile screen gets displayed.

  1. The screen displays details of all active prescribed or dispensed drugs, by default. Enter the search criteria and click [Search] to view details of specific drugs. Click [Clear] to clear the entered data.

    1. Patient ID - The Patient ID of the selected patient is displayed and disabled.

    2. From Date and To Date - Enter the date range or select it from the Calendar function to search for drugs within the specified period.

    3. TPN - Select the check box, if the practitioner wants to view TPN.

    4. Generic Name - Select the Generic Name from [?] function.

    5. Drug Name - Select the Name of the Drug from [?] function.

    6. Patient Class - Select the Patient Class from the list. The Patient Classes listed are Daycare, Emergency, External, Inpatient and Outpatient.

    7. Status - Select the required Status from the list.

  2. On searching, you can see the drug profile of the selected patient based on the entered search criteria.

For all the home leave prescription, an indicator with green background will be displayed so that the pharmacists are aware that the prescription is a home leave prescription. This indicator will appear in Verification, Fill and Deliver Stages.
