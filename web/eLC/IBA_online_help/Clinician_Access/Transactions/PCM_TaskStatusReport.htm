Task Status Report (V)

From this function you can View the task status report.

The Task Status report allows to view the goal statements and tasks for each status for the required date range. You need to configure the report in Construct CA Menu function so that it appears under the Patient Chart.

Viewing Task Status Report

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Chart History, select Task Status Report (V).

The Task Status Report screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to view the records matching the search criteria or [Reset] to clear the entered data.

  3. Click [Print] to generate the report.
