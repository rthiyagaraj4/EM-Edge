Patients By Specific Drug

From this function you can Query for patient record using a specific Drug.

The list of patients can be searched based on the Generic Name or the Drug Name. You can view either the ‘Active’ patients or ‘All’ based on the entered search criteria.

Querying for patients using a specific Drug

From the PH Queries, select the Patients By Specific Drug function.

The Patients By Specific Drug screen gets displayed.

  1. Click [Search] to display all the patient records or enter the search criteria and click [Search] to view the specific Drug details.

    Enter the following search criteria:

    1. Generic Name - Enter the Generic Name in alphanumeric format of 40 characters maximum length or select the required Generic Name using [?] function.

    2. Drug Name - Enter the Drug Name in alphanumeric format of 60 characters maximum length or select the required Drug Name using [?] function.

    3. Patient Class – Select the required Patient Class from the list. The listed values are ‘Daycare’, ‘Emergency’, ‘External’, ‘Inpatient’ and ‘Outpatient’.

    4. Date [From-To] – Enter the date range in dd/mm/yyyy format or select the required date range using the Calendar function.

    5. Status – Select the required status from the list. The listed values are ‘All’ and ‘Active’. The value ‘All’ gets displayed, by default.

  2. On searching, the Drug details are displayed with a unique color code based on the selected status. The status indicators are displayed at the bottom of the screen.

  3. Click [Reset] to clear the specified criteria.
