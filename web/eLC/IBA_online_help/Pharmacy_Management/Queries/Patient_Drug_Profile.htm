Patient Drug Profile

From this function you can Query the Drug Profile of a patient.

The Drug Profile can be searched based on the Generic Name or Drug Name. You can view the ‘Active’ Drug Profile or ‘All’ Drug Profile based on the entered search criteria.

Querying the Drug Profile of a patient

From the PH Queries, select the Patient Drug Profile function.

The Patient Drug Profile screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to display the Drug Profile based on the entered criteria. The Drug Profile, External Drug Profile, Medical Item tabs are displayed.

    Drug Code, Drug Name, Generic Name, Start Date, End Date, Prescribed By and Instructions are displayed

    Drug Profile tab

     

    Prescribed / Dispensed Drug, Status, Route, Dosage Detail, Start Date, End Date Remarks, Patient Class, Prescribed Quantity, Dispensed Quantity, Return Quantity / Date, Admin Quantity, Reissued Quantity, Prescribed By, Ordering Location/Order ID, Facility and Generic Name are displayed.

  3. On searching, the Prescribed or Dispensed Drugs are displayed with a unique color code based on the selected status. The status indicators are displayed at the bottom of the screen. The font and background color of the Route value is based on the Admin Route setup. If the drug is received from the external facility, the ‘Requested for Collection’ is displayed in the Status column against the Drug Name. If the received drug is delivered to the patient, the ‘Issued’ is displayed in the Status column against the Drug Name.

  4. Click 'Remarks' hyperlink against the desired Drug Name to view the split dose details, Duplicate Drug and Dosage Override details, Remarks recorded while generating the prescription. The Remarks window displays these details.  

  5. If an order contains Sliding scale, on clicking 'Remarks' hyperlink, the Sliding scale and Schedule Frequency details are displayed. Also, the sliding scale is displayed on hovering the mouse over or clicking the mouse on the sliding scale text displayed in red under Dosage Details column. Click [Close] to exit from the Remarks screen.

  6. Click ‘Dispensed Locns’ hyperlink to view the Dispensed Location dialog.

  7. Click the Remarks hyperlink in the Remarks column to view the existing remarks for the alternate drug in the Remarks dialog window.

  8. Click [Reset] to clear the specified criteria.
