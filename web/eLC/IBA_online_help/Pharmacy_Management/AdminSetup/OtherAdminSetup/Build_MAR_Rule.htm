Build MAR Rule

From this function you can Define the rules for Medical Administration recording based on patient class, location, priority and form of drug.

This function allows to define the rules for Medical Administration recording and to activate the Build MAR checkbox in the Pharmacy Orders screen.

Defining Build MAR Rule

From the PH Code Setup > PH Other Admin Setup, select the Build MAR Rule function.

The Build MAR Rule screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Build MAR Rule or [Reset] to clear the entered data.

  4. Click [Query] to search for the existing Build MAR Rules. You can modify the MAR Defaulting value in the query mode.
