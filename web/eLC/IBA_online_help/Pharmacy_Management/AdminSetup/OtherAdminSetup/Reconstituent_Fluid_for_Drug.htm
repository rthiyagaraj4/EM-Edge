Reconstituent Fluid for Drug

From this function you can Create an association of Re-constituent Fluid to a Drug and Query for the desired association of Re-constituent Fluid to a Drug.

The Reconstituent Fluids are used with Drugs for preparing Admixtures. The Drugs with physical form ‘Powder’ are listed.

Creating an association of Reconstituent Fluid to a Drug

From the PH Code Setup > PH Other Admin Setup, select the Re-constituent Fluid for Drug function.

The Re-constituent Fluid for Drug screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Search and select the Drug Name using [?] function.

  3. On selecting the Drug Name, the screen displays two tabs, namely, Select Tab and Association Tab. The ‘Select’ tab gets displayed, by default.

Select Tab

Enter the following details:

  1. Name – The constituent Fluid Name is displayed.

  2. Quantity - Enter the quantity in numeric format of two digits maximum length.

  3. UOM - The Unit of Measurement defined for the constituent fluid is displayed.

  4. Stability Info - Click on the ‘ Info’ hyperlink to invoke the Stability Information screen.

    • Enter the Stability Information in alphanumeric format. Click [OK] to record the Stability Information or [Cancel] to exit from the screen.

  5. Default – Select the check box to set the re-constituent fluid as default fluid. Only one fluid can be selected as default.

  6. Select - Select the check box to select the Re-constituent fluid for the selected Drug.

Association Tab

  • Click [Association] to view the selected fluids. You can change the values entered under ‘Select’ using the ‘Association’ tab.

  1. Click [Apply] to save the association of Reconstituent Fluid to a Drug or [Reset] to clear the entered data.

  2. Click [Query] to search for an existing association of Reconstituent Fluid to a Drug.
