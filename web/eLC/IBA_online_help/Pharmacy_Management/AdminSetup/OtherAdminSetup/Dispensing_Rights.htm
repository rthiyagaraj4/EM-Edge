Dispensing Rights

From this function you can Define rights to perform dispensing from a dispensing location for a user and Query for the desired dispensing right records.

The Users of different types are defined in the ‘Application Masters’ module. Users of Practitioner Type ‘Pharmacist’ are assigned rights to perform dispensing functions. These Users are associated to more than one Dispensing Locations.

Defining Dispensing Rights

From the PH Admin Setup > PH Other Admin Setup, select the Dispensing Rights function.

The Dispensing Rights screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the User Name in alphanumeric format of 30 characters maximum length or select the User Name using [?] function.

  3. Click [User Profile] to invoke the User Profile screen.

    Enter the following details:

    1. Login Password – Enter the Old Password in alphanumeric format of 30 characters maximum length. The entered characters are displayed in the form of ‘*’ for security reasons. If you enter incorrect login password, the system does not allow you to change the password

    2. New Password - Enter the New Password in alphanumeric format of 30 characters maximum length. The entered characters are displayed in the form of ‘*’ for security reasons.

    3. Confirm Password - Re-enter the New Password in alphanumeric format of 30 characters maximum length. The entered characters are displayed in the form of ‘*’ for security reasons.

    Click [OK] to confirm User Profile entry or [Cancel] to exit from the User Profile screen.

  4. Select the Dispense Location from the list. The Dispense Locations of type ‘Pharmacy’ defined in the Dispense Location function are listed. This is a mandatory field.

  5. On selecting the Dispense Location, the features of the selected Dispense Location are displayed.

  6. Enter the details in the mandatory (*) fields. <Related Field Description>

  7. Click [Apply] to save the Dispensing Rights defined or [Reset] to clear the entered data.

  8. Click [Query] to search for an existing Dispensing Rights.
