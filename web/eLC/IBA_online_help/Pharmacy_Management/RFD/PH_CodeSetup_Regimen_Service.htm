Enter the following details to associate Regimen, TPN Applicability to a Service:

  1. Indicator – Select the Indicator from the list. The listed values are pre-defined. This field is mandatory.

  2. Service – Enter the Service code in alphanumeric format of four characters maximum length or select the Service code using [?] function.

  3. Regimen – Select the check box to associate the selected Service to the Regimen. The check box is cleared, by default.

  4. TPN – Select the check box to associate the selected Service to TPN. The check box is cleared, by default.

  5. Enabled – Select the check box to enable the association.
