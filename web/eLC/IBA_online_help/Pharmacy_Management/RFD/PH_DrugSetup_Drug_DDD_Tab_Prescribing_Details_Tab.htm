Prescribing Details Tab

The prescribing details for the defined Drug are recorded in this tab. The prescribing details in terms of drug strength, base unit of measurement and stock unit are specified. Optionally, a drug category is attached to the defined Drug.

 

Enter the following Prescribing Details:

  1. Prescribing Unit - Select the Prescribing Unit from the list. The listed Unit of Measurements are based on the following conditions. This is a mandatory field.

    • Unit of Measurements associated to the selected form of Drug.

    • Associated Unit of Measurements must have equivalents Unit of Measurements for which ‘Quantity’ is applicable.

For example, if the form of drug is ‘Syrup’:

    • The form ‘Syrup’ is associated to prescribing base unit such as ‘Tbsp’, ‘tsp’, ‘ml’ and ‘ounce’ in the function ‘Unit for Form of Drug’.

    • The UOMs are defined in the function ‘Unit of Measurement: (Application Masters) as follows:

  1. Strength – Two values are to be specified for the field. Enter the strength of the drug in numeric format of 14 digits maximum length in the first field. Select the Unit of measurement from the list. The listed units of measurement are those with ‘Strength’ as applicable (Unit of Measurement function of Application Masters). Strength is an optional setup. If strength is defined, medication can either be prescribed on strength or quantity. If strength is not specified, the ordering is by quantity. This is a mandatory field.

  2. Dispensing Unit – Select the Dispensing Unit from the list.

  3. Drug Category – Select the drug category from the list. The drug categories defined in the function ‘Drug Category’ is listed in the check box.

Based on the drug category selected, the application displays the drugs in the Drug Category - Query screen during drug search.

  1. Calculate Default Dosage by Age/Weight – Select the check box to calculate the default dosage based on age and weight of the patient during ordering. The system refers to the function ‘Drug Dosage Limit’ and considers the value specified for the age group and ‘Calculate dosage by’ as ‘Weight’. While ordering, the system displays the default dosage considering the age group and weight with the drug dosage limit.
