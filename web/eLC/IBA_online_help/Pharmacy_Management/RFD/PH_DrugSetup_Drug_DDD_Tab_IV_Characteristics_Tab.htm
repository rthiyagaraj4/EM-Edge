IV Characteristics Tab

The IV Characteristics of the defined Drug are specified in this tab. IV Characteristics are optional settings.

 

Enter the following details to define IV Characteristics of the Drug:

  1. IV Ingredients? – If the defined Drug is an Intravenous Ingredient, select the check box. If the defined drug is an IV Ingredient, a compatible fluid can be selected for infusion.

 

On selecting the IV Ingredients? check box, [ Compatible Fluid] is displayed.

Click [Compatible Fluid] to invoke the Compatible Fluids screen.

  • Select the fluid from the Compatible Fluids screen. The listed fluids are Drugs with characteristics defined as ‘IV Fluid’.

  • Select the ‘Default Fluid’ check box to make the Fluid default during IV Ordering.

  • Select the ‘Select’ check box against the desired Fluid to select the fluid for infusion.

 

  1. IV Fluid? – If the defined drug is an IV Fluid, select the check box.

  2. Infuse Over – Specify the infusion duration in hours or minutes by entering the value in numeric format of six digits maximum length. This field is enabled only if ‘IV Fluid’ is selected.

  3. Physical Form? – Select the Physical Form of the drug from the list. The values listed are Liquid and Powder. This field is applicable only for ‘IV Ingredient’.

  4. Use for Oncology Drug Preparation – This function is currently not used in the application.
