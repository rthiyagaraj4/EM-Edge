Enter the following details:

  1. Constituent Generic Name Limiting From - Select the values from the list. The defined electrolytes names are displayed. The listed values are Sodium, Potassium, Acetate, Chloride, Phosphate, Calcium and Magnesium.

  2. Molecule Name Limiting From - Select the value from the list to associate molecule names to the constituent. The molecule names defined for the particular constituent selected in the 'Constituent Generic Name Limiting From' are listed.

  3. Constituent Generic Name Limited To - Select the 'Constituent Generic Name Limited To' values from the list. The defined electrolytes names are displayed. The listed values are Sodium, Potassium, Acetate, Chloride, Phosphate, Calcium and Magnesium.

  4. Molecule Name Limited To - Select the value from the list. The molecule codes defined for the particular constituent selected in the 'Constituent Generic Name Limited To' column are listed. The details of the molecules defined in the TPN Constituent (NSR) setup.

  5. Constituent Concentration From Range Between - Enter the 'Constituent Concentration' From Range in the numeric format of 8 digits maximum length. You can enter the concentration values for the selected Generic Constituent for a range of values. The UOM of the particular 'From Constituent' is displayed and disabled.

  6. Constituent Concentration To Range Between - Enter the 'Constituent Concentration' To Range in the numeric format of eight digits maximum length. You can enter the concentration values for the selected Generic Constituent for a range of values. The UOM of the particular 'From Constituent' is displayed and disabled.

  7. Precipitation Indicator - Select the Precipitation Indicator value from the list. The listed values are 'Warning', 'Error' and 'Information'. If value is selected, the user will be prompted about the precipitation. If value is selected as Error, the user will be prompted about the precipitation. If the value selected is Information, the defined alert message will be only for information. The selected Precipitation Indicators are displayed in the TPN Admixture Preparation function.

  8. Precipitation Alert Message - Enter the Precipitation Alert Message in the alphanumeric format of 300 characters maximum length. You can define the alert message for a particular combination of the constituent molecules and for a specific quantity.

  9. Enabled - Select the check box to enable the defined record in the transactions.

The icon is displayed for adding a new row for duplicating the data in the existing row. This icon is enabled on entering the data in  a row. Click this icon to display the duplicated data.

The icon is displayed for removing the added duplicated row. This icon is enabled for the row which is duplicated from an existing row. Click this icon to delete the duplicated row.
