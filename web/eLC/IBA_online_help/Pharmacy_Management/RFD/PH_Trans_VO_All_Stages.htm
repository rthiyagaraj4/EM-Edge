All Stages

The Pharmacist can perform all the stages in a single function. This is applicable only for Outpatient orders. The value ‘All Stages’ is available only if Multiple Stage is defined for the selected Dispense Location. The dispense medication screen gets displayed depending on the outpatient Pharmacy characteristics of the selected dispensing location.

On selecting ‘All Stages’ in Dispense Medication screen and specifying the required query criteria for outpatient category, the following screen gets displayed.

Pharmacist can Allocate, Fill and Deliver the Drug using All Stages function.

The Verified Orders are displayed on selecting All Stages’.

If another user is accessing the same order, an icon is displayed in the 'User Locked' column. On clicking the Patient ID, a alert message 'This Patient Record is opened by another user. Access Denied' is displayed. Click icon to unlock and proceed.
