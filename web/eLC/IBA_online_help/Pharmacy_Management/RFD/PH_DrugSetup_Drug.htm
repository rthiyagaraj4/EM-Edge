Enter the following details to create a Drug:

  1. Drug Code – The behavior of the field depends on PH Parameter settings. If the generation of Drug Code is set as ‘System Generate Sequence’, this field is disabled and system generates the Drug Code. If the generation of Drug Code is ‘User Enterable’, this field is enabled and you have to enter Drug Code manually. Enter Drug Code in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. In Formulary – Select the check box to indicate that the defined Drug is to be stored in the hospital Pharmacy.

  3. Link to an existing Item – The check box is enabled if ‘Drug and Item Code Link Required’ is applicable in PH Parameter. If the link is not required, the check box is disabled. Select the check box to link an existing Item to the Drug. If link to an existing item is to be established, ‘Item Code’ has to be entered compulsorily.

  4. Item Code - The Item Code field is enterable only when ‘Link to an Existing Item’ is selected. If linking is not allowed, the field is not applicable. Enter the Item Code in alphanumeric format of 20 characters maximum length or select the required code using [?] function. The search screen displays and lists the Items defined as Drugs in ‘Inventory Management’ module ‘Item’ function. If a link between Drug and Item is required, Item Code is mandatory.

  5. Drug Description – The Drug Description gets displayed on selecting Item Code (if applicable). You cannot change the displayed value. If Item Code is not applicable, enter the Drug description in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  6. Generic Name – Enter the Generic Name in alphanumeric format of 40 characters maximum length or select the generic name of the defined drug using [?] function. Generic Names are defined in the function ‘Generic Name’. This is a mandatory field.

  7. Compound – Select the check box, if the defined Drug is a Generic Compound. Generic Ingredients can be specified if the Drug is a Generic Compound.

  8. Form – Select the form from list. The values displayed in the list are defined in the function ‘Form of Drug’. This is a mandatory field.

  9. Route - Select the Route from the list of values. If you want to default the Route at the time of prescription, select the Default Route checkbox and click [Apply].

  10. Default Route – Select the default route of the drug from the list. For the selected Form, the applicable routes as defined in the function ‘Route for Form’ are listed. This is a mandatory field.

  11. Prompt Alert for Preferred Item - Select the check box to prompt the alert ‘Drug Item is excluded from patient coverage or by hospital policy (Patient payable drug). If you would like to choose Preferred drug/item please click’ while placing the orders in Clinician Access module. An icon is displayed beside the Item or Drug if the preference basis is selected in the Patient Billing Manager module for the Item or Drug.
