By Service Tab

You can associate a Service to the required Generic Names using this tab. You can select ‘Clinic’ or ‘Clinic & Nursing Unit’ and select the corresponding Service. On selecting a Service, the Generic Names are displayed. You can associate the required Generic Names to the selected Service.

 

  1. Select ‘Clinic’ option to display the Services of the Outpatient Clinics.

  2. Clinic Nursing Unit – Select ‘Clinic Nursing Unit’ option to display the Services in the facility.

  3. Enter the Service in alphanumeric format of 40 characters maximum length or select the required Service using [?] function. The values fetched are based on ‘Clinic’ or ‘Clinic & Nursing Unit’ and set search criteria. This is a mandatory field.

  4. On selecting the Service, the screen displays alphabets from ‘A to Z’ and ‘Others’. These alphabets are hyperlinked. Each alphabet contains Generic Names under them. The ‘Others’ hyperlink contains Generic Names that start with characters other than the alphabets.

  5. Click on the hyperlinks to display the Generic Names associated to the selected alphabet. The Generic Name details are displayed.

  6. Select the check box to associate the Generic Name against it to the selected Service. The check boxes are cleared, by default.
