Enter the following details to specify the limit:

  1. Non-Standard Regimen – Select the Non-Standard Regimen from the list.

  2. Age Group – Select the Age Group from the list. The listed groups are defined in Application Masters module.

 

On selecting the Age Group, the Gender, Minimum Age, Maximum Age, Age Unit and the TPN Groups and their Generic Constituents are displayed.

Enter the following details to set limits for the TPN Constituents:

 

You have to specify the limit for each constituent. TPN constituents and their generics are displayed.

  1. Fluid – The list of Fluids are displayed.

  2. Unit – The Unit of Measurement applicable to the TPN Constituent is displayed. These UOMs are defined in the function ‘UOM for TPN Constituents’.

  3. Limit - The Limit value is either ‘By Weight’ or ‘By Absolute’. If ‘By Weight’ is displayed, while TPN Medication Ordering the system calculates the weight of the patient with the ordered quantity and compares it with the set limits. If the calculated value is within the specified limits, ordering is allowed. If ‘By Absolute’ is displayed against a TPN constituent, the system evaluates the ordered quantity with the limits set in this function. The ordered quantity must be within the limits specified.

  4. Minimum - Enter the Minimum Dosage Limit for the Constituent in numeric format of eight digits maximum length.

  5. Maximum - Enter the Maximum Dosage Limit for the Constituent in numeric format of eight digits maximum length.

  6. Standard - Enter the Standard Dosage for the Constituent in numeric format of eight digits maximum length. The standard value must be between Minimum and Maximum values defined.

  7. Ceiling – Enter the Ceiling value for the constituent in numeric format of eight digits maximum length.

  8. Usage Indicator – Select the Usage Indicator from the list. The listed values are ‘Applicable’ and ‘To Be Ordered’. The value ‘Applicable’ is displayed, by default.

  9. Click Remarks hyperlink to enter the Remarks. Enter the Remarks in alphanumeric format. Click [OK] to exit from the Remarks screen.

  • The Age Unit value set for the selected Age Group is displayed as read-only text.

  • The Maximum Age set for the selected Age Group is displayed as read-only text.

  • The Minimum Age set for the selected Age Group is displayed as read-only text.

  • If the selected Age Group is specific to a Gender, the same is displayed as read-only text.
