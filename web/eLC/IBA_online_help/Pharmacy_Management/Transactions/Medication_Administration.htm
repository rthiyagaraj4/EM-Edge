Medication Administration

From this function you can Record Medication Administration details for Inpatient/Daycare/Emergency/Outpatients.  

To invoke medication administration function, the following conditions are to be satisfied.

• Applicable for all the Patient Classes

• ‘Medication Administration Required’ is selected in PH Parameter for Facility

• Frequency Schedule specified for IP Orders

The logged in user selects the required Patient Class, specifies the administration status as query conditions. A Nurse or person responsible for Medication Administration performs this transaction. Before performing this transaction, it is recommended to view the Medication Administration Chart for the Location in the ‘Medication Administration Chart’ query function and generate the Medication Administration List for the Location using the ‘Medication Administration List’ report function. You can view the result set as either ‘Single’ or ‘Multiple’. The administration details for the selected patient are listed in multiple rows when the selected view is ‘Single’. The administration details are shown in tabular form with timing/dosage details when the selected view is ‘Multiple’. You must select the timing/dosage to record administration details. The Administration details can be recorded within the permissible period set in the parameter. On recording the Medication Administration details, you can enter the Adverse Drug Reaction details. The provision to record Verbal Order details is provided. User rights to place order directly during Medication Administration is set in Order Entry and Result Reporting module.

Recording Medication Administration Details

From the PH Transactions, select the Medication Administration function.

The Medication Administration screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to display the patient details.

  3. The Medication details are displayed in the Prescription Details frame.

    1. Rx – Displays a dark background when clicked displays the ‘Could not Administer’ drug record. You can make the admin changes and click Apply, the status of the drug is updated to ‘Administered’.

    2. Drugs - Drugs ordered for the selected patient for administration within the specified Period (From and To values) are displayed. The ordered drugs are segregated under IV drugs and Non-IV drugs respectively. Narcotics, Controlled, IV Fluid and IV Ingredients are indicated using color-codes. You can click the icon to view the audit trail of who captured the ‘Could not Administer’ for a drug and who re-administered the drug to the patient.

    3. +Taper hyperlink - The +Taper hyperlink is displayed against all the taper level order Click this hyperlink to administer the patient based medication or medication owned by the patient. The system will copy the original order line.

    4. - Click icon to view the Prescription Instructions recorded while placing the order

    5. Rx – Click on Rx hyperlink to view current prescription details. The drug/dosage details are displayed. Close the screen on viewing the details.

    6. Admin Dose  – The Administration Dosage gets displayed along with Unit of Measurement. The dosage value can be changed to the required value.

    7. Route of Administration - Select the Route of Administration value from the list to filter the Medication Order by Route of Administration.

    8. Schedule Time - The drug administration date and time gets displayed as read-only text.

    9. Admin Date/Time - The schedule date/time gets displayed by default as drug administration date and time. Enter the drug administration date/time in date format of dd/mm/yyyy hh:mm.

    10. Brought by Patient - Upon clicking the Taper hyperlink, the system checks the this check box and the Admin Dtls check box is made mandatory.

    11. Batch ID- Select the Batch ID using [?] function.

    12. Shared drug - When you mouse hover this check box, a tool tip with the name as ‘Shared Drug’ will be displayed. If you select this check box while administering the drug, the administered drug will be considered as a shared drug and no stock will be deducted. Hence, there will be no impact on the billing.  This check box will appear only if the ‘Allow Shared Drug’ check box is selected in the PH Parameter for Facility screen.

    13. Expiry Date – The expiry date gets displayed on selecting the Batch ID.

    14. Admin Dtls – Select the check box to indicate administration of the drug. A mandatory ‘Dtl’ hyperlink gets displayed based on the following conditions.  Click on the hyperlink to enter administration remarks. Enter the remarks and click [OK].

    15. Select– Select the check box to confirm the entered values and to record administration details. If selected, details are stored or details are not stored.

  4. To view other Inpatient details, enter the following details.

  5. Enter the Bed Number in alphanumeric format of eight characters maximum length and click [Locate] to view the Inpatient details for a particular Bed Number.

  6. Enter the Patient ID in format as defined in Master Patient Index module and click [Locate] to view the Inpatient details of a particular Patient.

  7. Select ‘Patient ID’ or ‘Patient Name’ hyperlink to display the Medication Administration details.

You can only view the Bed no, Patient ID, Patient Name, Gender, Age and Nationality.

  1. Scroll down using the scroll bar to view the following links:

    • The text ‘Known Case of Allergy’ is displayed in red when a drug allergy is recorded in the patient record.

    • Click ‘Allergies’ hyperlink to view the Allergy details of the selected patient in the Allergies List dialog.

    • Click ‘Diagnosis’ hyperlink to view the medical problems of the selected patient in the Problem List dialog.

    • Click ‘Current Rx’ hyperlink to view the current medications taken by the patient in the Current Medication dialog.

    • Click ‘Demographics’ hyperlink to view demographics details recorded during patient registration in the Patient Details screen.

    • Click ‘Drug Profile’ hyperlink to view the complete Drug Profile of the selected patient in the Patient Drug Profile screen.

  2. In the Grid, you can view the following columns:

    • Drugs – Displays the drug name. Click to view the drug details in the Drug Detail dialog. Clickto drug information and patient education in a separate <drug name> window. The icon appears only for the drug(s) for which the patient is allergenic. You can click this icon to view the Allergy alert popup.  Click to view the allergy override reason captured while ordering the allergenic drug to the patient in the Override Reason dialog. Click Close to close the dialog

    • Click Rx hyperlink to view Drug/Dosage Details in the Drug/Dosage Detail dialog. The Sliding Scale details are displayed along with existing schedule details on clicking 'Rx' hyperlink. Click [Close] to exit from the screen.

    • Admin Dose - You can change the Admin dose as required.

    • Route of Administration - Select the Route of Administration value from the list to filter the Medication Order by Route of Administration. The available values are As directed and Dilute before use.

    • Scheduled Time – Time scheduled for administering the drug.

    • Admin Date/Time – Displays the current system date and time, by default.

    • Brought By Patient - Select the Brought by Patient check box if the medication is brought by the patient. The Expiry Date field is enabled and you can select the date using the calendar.

    • Batch ID - If the prescribed drug is a “floor stock” drug then a default Batch ID is displayed based on First In First Out Basis(FIFO). You can also click [?] to change the Batch ID by searching and selecting the required batch from the Batch Search dialog.

    • Expiry Date - Expiry Date of the drug is displayed.

    • Admin Dtls - Click Dtl hyperlink to record the administration remarks.

      The Drug Name is displayed as read-only text. Entering Admin details is mandatory.

      1. Remarks - Enter the Remarks in alphanumeric format of 255 characters maximum length. This is a mandatory field.

      2. Administered By - The logged in User Name is displayed as the name of the person administering the Drug.

      3. Changed Admin Dosage Reason - Select the reason for the Admin Dosage Reason change. This is a mandatory field.

      Click [OK] to administer the Drug.

      On clicking Dtl hyperlink for IV Drugs, an additional field 'Next Administration Date/Time' is displayed.

      Witnessed By field is displayed for narcotic and controlled drugs. Enter the person consigning the administration remarks. Enter the name or select it using [?] function. This field is mandatory.

    • Select - Select this check box to select the record.

View Medication Administration Chart

  1. Click [View Medication Administration Chart] to view the Medication Administration Detail dialog.

    Displays the Bed No, Patient ID, Patient Name, Gender, Age, Nationality, Encounter ID as read-only. The system displays administration details based on ‘From’ and ‘To’ values specified in the query criteria. You can view administration details by selecting ‘Past Dose’ or ‘Future Dose’.

    1. Past Dose – Select the date from the list. The past dose value listed are from the medication start date till the current system date.

    2. Future Dose – Select the future date from the list. The future dose values listed are from the next date of current system date till the medication end date.

    3. Date From – The ‘From’ value specified in query criteria gets displayed, by default. Specify the required date or use the calendar function to select the date.

    4. Date To – The ‘To’ value specified in query criteria gets displayed, by default. Specify the required date or use the calendar function to select the date.

Verbal RX

  1. Click [Verbal Orders] to place verbal orders.

  2. The Order Entry dialog is displayed.

This function is applicable for a facility if Medication Administration Required is checked In ‘PH Parameter for Facility’.

  1. After completing the verbal order, click [Release Rx] to display the PIN Authentication dialog.

  2. Select the User ID of the Witness in Witnessed By.

  3. Enter the PIN Number of the witness user.

  4. Click Ok.
