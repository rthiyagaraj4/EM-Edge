Prepare Admixture

From this function you can Verify and Issue the drug which needs to be mixed and processed by the Pharmacist in the Pharmacy.  

Prepare Admixture function filters only those orders which require the Pharmacist to prepare the mixture.

Verifying and Issuing Admixture Drug

From the PH Transactions, select the Prepare Admixture function.

The Dispense Medication login screen gets displayed.

  1. Dispense Medication screen gets displayed.

    Enter the following details to validate dispensing rights:

    1. User Name – The logged in User Name gets displayed as read-only text.

    2. Password – Enter the password to perform dispensing.  In the function ‘Dispense Rights’, password stored for you in User Profile is validated. This is a mandatory field.

    3. Dispense Location – Select the Dispense Location from the list. The locations listed (Pharmacy Type Only) are assigned to the logged in user with rights to perform various dispensing functions. This is a mandatory field.

    4. Category – Based on the Dispense Location the category value gets displayed. The table provides the value(s) listed. Select the required value.  Refer to ‘Dispense Location’ function for more details on characteristics of Dispense Location. This is a mandatory field.

  2. Click [OK] to login to the Prepare Admixture function or [Cancel] to cancel the login process.

  3. The Order Type is displayed as ‘Compounding Order’, by default. The listed Order Types are Compounding Order, IV With Additives (Admx), IV Intermittent (Admx) and Oncology Admixture.

  4. Enter the required criteria based on the selected category and click [Refresh] to display the orders.

  5. For various stages of admixture orders, you can proceed as explained in the Dispense Medication function.

You can select the Product Name for an Admixture Type in the 'Finalize Worksheet for Admixtures' screen in the Fill and All Stages.
