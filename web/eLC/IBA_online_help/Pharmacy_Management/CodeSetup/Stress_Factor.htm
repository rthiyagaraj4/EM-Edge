Stress Factor

From this function you can Create Stress Factor and group it with the Weightage and Query for the desired Stress Factor.

Stress Factor is used to set Weightage against various Stress Factors. The Weightage specified for a Stress Factor is utilized to calculate energy requirement for patient during TPN Ordering.

The system alerts you to enter the mandatory values, if they are not entered. The mandatory field is highlighted with a unique color code. Only 10 records can be added at a time. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate codes with a unique color code.

Creating a Stress Factor

From the PH Code Setup > PH TPN Setup, select the Stress Factor function.

The Stress Factor screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Stress Factor created or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing Stress Factor.

  5. Click [Delete] to delete the queried record.
