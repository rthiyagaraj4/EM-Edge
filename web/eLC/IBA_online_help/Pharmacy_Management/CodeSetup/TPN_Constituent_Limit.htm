TPN Constituent Limit

From this function you can Create TPN Constituent Limit for a particular Age Group.

You can restrict TPN Generic Constituent dosage to a particular Age Group. Age groups are defined in Application Masters module. Select the required Age Group and specify the TPN Constituent Limit.

Creating a TPN Constituent Limit

From the PH Code Setup > PH TPN Setup, select the TPN Constituent Limit function.

The TPN Constituent Limit screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the TPN Constituent Limit created or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing TPN Constituent Limit.
