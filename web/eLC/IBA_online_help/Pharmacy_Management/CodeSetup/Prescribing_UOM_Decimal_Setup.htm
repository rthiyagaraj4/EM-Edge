Prescribing UOM Decimal Setup

From this function you can Define the decimal for a Prescribing Unit of Measure(UOM).

The decimal value and description for the Unit of measure(UOM)  is displayed in the Drug Label and in the Reports.

Creating UOM Decimal Setup

From the PH Code Setup > PH Code Setup, select the Prescribing UOM Decimal Setup function.

The Prescribing UOM Decimal Setup screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Prescribing UOM Code or select the code using [?] function.

  3. On selecting the Prescribing UOM Code, the Short Description and the Long Description of the selected Prescribing UOM Code is displayed in the respective fields.

  4. Enter the Decimal Value, Fractional Value in Label, Fractional Description and Fractional Description Local.

  5. The Enabled checkbox is checked, by default. If the defined decimal for Prescribing UOM is not required, you can clear the Enabled checkbox.

  6. Click [Apply] to save the UOM decimal or [Reset] to clear the entered data.

  7. Click [Query] to search for existing UOM Decimal.
