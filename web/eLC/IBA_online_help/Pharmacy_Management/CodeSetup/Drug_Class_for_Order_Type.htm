Drug Class for Order Type

From this function you can Create an association of Drug Class to Order Type and Query for the desired association of Drug Class to Order Type.

The Drug Class for Order Type facilitates to associate Drug Class to an Order Type. A Drug Class can be associated to more than one Order Type. The Order Types belonging to ‘Pharmacy’ Order Category are defined in ‘Order Type’ function. These values are validated during Medication Ordering.

Creating an association of Drug Class to Order Type

From the PH Code Setup > PH Code Setup, select the Drug Class for Order Type function.

The Drug Class for Order Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Click Select tab to associate Order Type to Drug Class.

  3. Click ‘+’ hyperlink to view the Order Type details. Click [Close] to close the Order Type details screen.

  4. The Order Types defined in the Order Type function are displayed.

  5. Select the Drug Class from the list. The Drug Classes displayed are General, Narcotics and Controlled.

  6. Click ‘Association’ tab to view the associated Order Types. The associated Order Types are displayed with a unique color code and Order Types associated during the current process are displayed below the previously selected values. You can change the associated Drug Class in this tab.

  7. Click [Apply] to save the association of Drug Class to Order Type or [Reset] to clear the entered data.

  8. Click [Query] to search for an existing association of Drug Class to Order Type.
