Enter the following details:

  1. Code - Enter the Code of the purchase unit in alphanumeric format of six characters maximum length.  Once the code is defined and saved, it cannot be changed in Query function.

  2. Long Description – Enter the Long Description of the Purchase Unit in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the Short Description of the selected Purchase Unit in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Global Unit – Select the Global Unit check box to define the status of the purchase unit as global. The selected Purchase Unit is applicable to multiple facilities.

  5. Enabled – Select the Enabled check box to enable the defined Purchase Unit. The check box is selected by default. The enabled Purchase Units are available for selection across the PO module.  Clear the check box, to disable the defined Purchase Unit.

  6. Address Line1, Address Line 2 and Address Line 3 – Enter the Address of the purchase unit in alphanumeric format of 60 characters maximum length in each Address Line.
