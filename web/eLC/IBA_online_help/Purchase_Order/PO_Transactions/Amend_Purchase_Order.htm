Amend Purchase Order

From this function you can Amend the purchase orders.

This function allows you to add more items to the purchase order but you cannot modify the existing items. Only finalized purchase requests can be amended.

Amending Purchase Order

From the PO Transactions, select the Amend Purchase Order function.

The Amend Purchase Order screen gets displayed.

  1. The Purchase Orders created for the current week is displayed by default.

  2. To search any specific purchase order, enter the details in the respective fields. <Related Field Description>

  3. Click [Search]. The purchase orders matching the search criteria is displayed.

  4. Click ‘Purchase Type’ hyperlink to view the purchase order details.

  5. To add more items to the existing list, enter the required item details in the ' Enter Item Details' and click [Add]. The item details are added in the upper part of the screen.

    1. Item – Enter the Item name or select using the [?] function. This is a mandatory field.

    2. Order Qty – Enter the order quantity. This is a mandatory field.

    3. Order UOM – The Order UOM is selected based on the Item selected.  This is a mandatory field.

    4. List Price – Enter the List Price of the item.

    5. Gross Amount – The Gross amount is populated based on the entered Order Qty and List Price.

    6. Discount Type – Select the Discount Type from the list. The listed values are Percentage and Amount.

    7. Discount – Enter the Discount percentage or amount based on the Discount Type selected.

    8. Net Amount – The Net Amount for the item is displayed.

    9. Purchase Price – The Purchase Price of the item is displayed.

    10. Commitment Code – Select the Commitment Code from the list.

    11. Department ID – Select the Department ID from the list.

    12. Account No – Select the Account number from the list.

    13. Acct Description – The Account Description is displayed from the Commitment Code master.

    14. Delivery Schedule – Select the Delivery Schedule for the entire order quantity by clicking the calendar function.

    15. Approval Reference – If the ‘Item Level Approval Required’ check box in PO Parameter is selected, the Approval Reference number has to be entered for each item. The Approval Reference check box is selected by default. Enter the Approval Reference number in alphanumeric format of 20 characters maximum length.

    16. Item Remarks – Click ‘Item Remarks’ hyperlink to enter the item remarks.

  6. After entering all the item details, enter the ‘Additional Discount’ if discount has to be given for the purchase order.

  7. Click [Apply] to amend the purchase order or [Reset] to clear the entered data.

  8. On clicking [Apply], A ‘Confirm Amend’ dialog is displayed.  

  9. Click [Yes].  The amend operation is completed successfully.
