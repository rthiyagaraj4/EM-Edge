Delivery Order

From this function you can Create delivery orders. You can Query for the existing delivery orders.

This function allows to create delivery orders to the Suppliers in two ways:

  • With a Purchase Order – You can raise an order with the purchase order number only. PO Search is mandatory if Purchase Order Mandatory for Delivery Order check box is selected.

  • Without a Purchase Order – Allows you to raise an order directly by adding items and defining the quantity of the required items in the order without a purchase order number.

Creating Purchase Orders

From the PO Transactions, select the Delivery Order function.

The Delivery Order screen gets displayed.

  1. Click [Create] to invoke the Create screen. The Delivery Orders can be created in two ways:

Delivery Order with a Purchase Order Number

  1. Enter the details in the mandatory fields (*). <Related Field Description>

  2. Enter the Item Details .

    1. Item – Enter the Item name or select using the [?] function. This is a mandatory field.

    2. Delivery Qty – Enter the delivery quantity. This is a mandatory field.

    3. Delivery UOM – The Delivery UOM is selected based on the Item selected.  This is a mandatory field.

    4. Unit Cost – Enter the Unit Cost of the item in numeric format.

    5. Net Amount – The Net Amount for the item is displayed.

    6. Item Remarks – Click ‘Item Remarks’ hyperlink to enter the item remarks.

    7. Batch Details – Click ‘Batch Details’ hyperlink to enter the Facility, Store, Expiry Date, Delivery Qty, Free Qty, Sample Qty, Batch ID and Trade ID details.

  3. Click [Add] to add the item to the delivery order. The Gross PO Amount, Net PO Amount gets displayed.

  4. Click ‘Item Code’ hyperlink to modify the item details. On entering the details, click [Modify] to save the changes or click [Cancel] to cancel the changes.

  5. Select the check box to select the Item as part of the delivery order.

  6. Click [Apply]. A ‘Confirm Finalize’ dialog is displayed.  Click [Yes] to finalize the delivery orders. Once the order is finalized it cannot be modified.

  7. Click [No] to Unfinalize the delivery orders.

  8. The Delivery Order Number is generated for the delivery order based on the rule setup.

Delivery Order without a Purchase Order Number

  1. Enter the details in the mandatory fields (*). <Related Field Description>

  2. The items pending for delivery for current date is displayed.

  3. Select the items to be delivered. The items of same Currency can only be selected.

  4. Click [Add to Delivery] to add the item to the delivery order. The Gross PO Amount, Net PO Amount gets displayed.

  5. Click ‘Item Code’ hyperlink to modify the item details. On entering the details, click [Modify] to save the changes or click [Cancel] to cancel the changes.

  6. Select the check box to select the Item as part of the delivery order.

  7. Click [Apply]. A ‘Confirm Finalize’ dialog is displayed.  Click [Yes] to finalize the delivery orders. Once the order is finalized it cannot be modified.

  8. Click [No] to Unfinalize the delivery orders.

  9. The Delivery Order Number is generated for the delivery order based on the rule setup.

  10. Click [Query] to query for the existing purchase orders.
