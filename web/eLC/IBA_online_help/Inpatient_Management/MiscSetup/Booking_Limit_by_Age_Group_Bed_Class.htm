Booking Limit by Age Group – Bed Class

From this function you can Create Booking Limits for an Age Group by specifying the maximum daily booking allowed based on Bed Class and Gender and Query for the Booking Limit set for Age Group, Bed Class and Gender combination.

This function allows you to set limits for controlling bed booking based on the Age Group. The Create screen gets displayed only if the field ‘Check Booking Limit based on’ in IP Parameter function is selected as ‘Age Group – Bed Class’.

Creating a Booking Limit by Age Group – Bed Class

From the IP Setup > IP Misc Setup, select the Booking Limit by Age Group – Bed Class function.

The Booking Limit by Age Group – Bed Class screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Click [?] function to select a Age Group. <Related Field Description>

  3. Click [Search] and enter the maximum daily booking limits for Male, Female and Unknown patients. Select the check box against the Bed Class to record and associate the booking details to the Age Group.

    Select – Select the check box to associate the maximum booking limit defined for a Bed Class to the Age Group. Clear the check box to clear the booking values entered for the bed class.

    Unknown – Enter the maximum number of daily bookings permitted for unknown gender against the bed class for the selected Age Group in numeric format of six digits maximum length.

    Female – Enter the maximum number of daily bookings permitted for female patients against the bed class for the selected Age Group in numeric format of six digits maximum length.

    Male – Enter the maximum number of daily bookings permitted for male patients against the bed class for the selected Age Group in numeric format of six digits maximum length.

  4. Click [Apply] to save the record or [Reset] to clear the entered data.

  5. Click [Query] to search for existing records.

 

Depending on the Age Group selected, the system restricts you to enter the Maximum daily booking limit for a particular bed class, based on the age group setting in Application Masters module.

If the age group entered is Between 11 to 20 years and associated to female gender in Application Masters module, the system allows you to enter the values only in the female column, on selection of a bed class. If the selected age group is not associated with any gender in Application Masters module, the system allows you to enter the values into male, female and unknown columns on selection of the bed class.
