Current Inpatients

From this function you can Query on Inpatients currently admitted in the facility.

The function provides various query criteria to filter the listing. On specifying the query criteria, the result screen displays the details of the inpatients under Specialty/Practitioner combinations. The function provides the option to view the current encounter details of the patients.

Querying for Current Inpatients

From IP Queries> Admissions/Transfers, select the Current Inpatients function.

The Current Inpatients screen gets displayed.

  1. Enter the details in the mandatory fields to query the Current Inpatients. <Related Field Description>

  2. Click [Search] to view the Current Inpatients details or [Reset] to clear the entered search criteria.

  3. On searching, the Current Inpatient details are displayed. The details of the Inpatients are displayed under their respective Specialty/Practitioner combinations. Click the ‘+’ hyperlink to view the Current encounter details of the selected patient.

  4. If a Newborn is registered for the patient, the baby details can be viewed by clicking the hyperlink that appears with the legend ‘View Details’. On clicking the ‘View Details' hyperlink a new window, which displays the list of babies registered for the patient along with the details recorded during the new born registration gets displayed.

  5. Click [Clear] to clear the result screen.
