Referred From

  1. Referral Type - Select the Referral Type from the list as 'Login Facility', 'External' or 'Enterprise'. Depending on the selected value, select the value for the referral source

  2. Referral Source - Select the range of Referral Sources. The corresponding referral sources are available for selection based on the referral type selected.

  3. Specialty - Select the Specialty from the list. The listed values are defined in 'Application Master' module of Enterprise Management.

  4. Service - Select the Service corresponding to the selected specialty from the [?] function.

  5. Location Type/Location - Select the Location Type from the list as either 'Clinic' 'Nursing Unit', 'Procedure Unit' or 'Daycare'. Depending on the selected value, select the Location from the list.

  6. Practitioner - Enter the Practitioner name in alphanumeric format of 30 characters maximum length or click [?] to search for the required practitioner.

 

Referred To

  1. Specialty - Select the Specialty from the list.

  2. Service - Select the Service corresponding to the selected Specialty from the list of valid values.

  3. Location Type/Location - Select the Location Type as Nursing Unit. Select the required Nursing Unit from the list.

  4. Practitioner - Select the Practitioner from the [?] function for the selected specialty.

  5. Patient ID - Enter Patient ID in alphanumeric format of length as specified in MP Parameter setup or select the Patient ID using the [?] function.

  6. Referral ID - Enter the Referral ID in alphanumeric format of length 14 characters.

  7. Priority - Select the Priority as either 'Elective', 'Semi Emergency', 'Emergency' or 'Referral for Admission'.

  8. Status - Select the Status from the list of valid values.

  9. Old Referrals - Select the check box to view details of referrals made before current system date.

  10. Search Name with Local Lang - Select the check box to search patient name in local language. On selecting the check box, 'Legend for Loc Language' and 'Patient Name Derivation Logic For Loc Lang' fields are enabled.
