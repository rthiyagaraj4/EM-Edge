Enter the following details:

 

  1. Nursing Unit – Enter the Nursing Unit in alphanumeric format of 15 characters maximum length, or select the nursing unit using [?] function. This field is mandatory. On invoking [?] function, the Nursing Unit search screen gets displayed.

Enter the required search criteria and click [Search] to display the Nursing Units that satisfy the entered search criteria. If the Nursing Unit description is known, enter the description directly and click [Search]. On selecting a Nursing Unit, following details are displayed in the IP Internal Report Rule screen:

  1. Nursing Unit Type –The Nursing Unit Type associated to the selected Nursing Unit is displayed.

  2. Age Group – The Age Group associated during the creation of Nursing Unit is displayed.

  3. Account Department – The name of the Accounting Department associated to the Nursing Unit is displayed.

  4. Service – The Service attached to the Nursing Unit is displayed.

 

The following Reports are displayed:

  1. Report Name – The online reports associated to the module are displayed.

 

Print On

  1. Booking - Select the check box, if the internal report is to be printed while booking a bed for a patient. The field is enabled for ‘Admission Booking Slip’ and ‘Admission Booking Slip for VIPs’ report.

  2. Admission - Select the check box, if the internal report is to be printed on admitting a patient.  The Admission field is enabled for selected reports only. You can select to either print the reports during admission or clear the check box, if a print is not required during admission. The check box is cleared, by default.

  3. Assign Bed - Select the check box, if the internal report is to be printed on assigning a bed to a patient. The Assign Bed field is enabled for selected reports only. You can select to either print the reports while assigning the bed to the patient or clear the check box, if a printed report is not required. The check box is cleared, by default.

  4. Transfer Patient - Select the check box, if the internal report is to be printed while transferring a patient. The Transfer Patient field is enabled for selected reports only. You can select to either print the reports while transferring the patient or clear the check box, if a printed report is not required. The check box is cleared, by default.

  5. Discharge - Select the check box, if the internal report is to be printed when a patient is discharged. The Discharge field is enabled for selected reports only. You can select to either print the reports while discharging the patient or clear the check box, if a printed report is not required. The check box is cleared, by default.

  6. Select – Select the check box to associate the report to the selected nursing unit. Select the report for association or the report cannot be printed for the selected Nursing Unit.
