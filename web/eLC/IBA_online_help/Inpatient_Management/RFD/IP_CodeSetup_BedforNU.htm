Enter the following details to create a new Bed for Nursing Unit record:

 

  1. Nursing Unit – Select a Nursing Unit from the list of valid values. The Nursing Units whose enabled check box selected while creating a Nursing Unit are displayed in this list. This field is mandatory.

  2. Main Bed No – This field identifies the bed to which a baby’s bed/bassinet is associated. This field is enabled if the Nursing Unit selected has the Allow Newborn Regn field selected. Enter the bed number belonging to the selected Nursing Unit for which the baby’s bed has to be created, or click [?] function to invoke the bed chart, which displays the beds created for the selected Nursing Unit. On selection of a bed, the bed number is displayed in this field.

  3. Room/Cubicle No – Select a Room/Cubicle from the list of valid values. This value depends on the Nursing Unit selected. The Rooms associated to the Nursing Unit are populated (using the function Room/Cubicle for Nursing Unit). If a Main Bed number is identified, the Room/Cubicle to which the main bed belongs to is made default and the field gets disabled. This field is mandatory.

  4. Bed No – Enter the bed number in alphanumeric format of eight characters maximum length. Bed number has to be unique for a Nursing Unit and room combination as it cannot be changed after creation. This field is mandatory.

  5. Census Bed – Select the check box to mark the beds as Census and Non-Census bed at the time of creating the bed and associating the bed to the Nursing Unit. This check box is enabled and selected by default. This field is applicable only for users in Medicity.

  6. Enabled - Select the check box  to enable the Bed for Nursing Unit.

 

Characteristics

  1. Bed Class - Select a valid Bed Class from the list for the Nursing Unit. Only those bed classes, which are enabled during creation, are listed. This is a mandatory field.

  2. Bed Type - Select a valid Bed Type from the list of valid values for the Nursing Unit. Depending on the Bed Class selected, the Bed Type values are populated in the list. This is a mandatory field.

  3. Gender Specific Ind – Select the required value from the list. The values displayed are Male, Female and Unknown. If a Gender is selected for the allotted bed, patients of the specified Gender only can occupy the bed. If the selected Nursing Unit has been defined as Gender specific, the same is displayed in this field and you are not allowed to modify the value.

  4. Specialty – This field indicates whether the bed created is specific to a single specialty or is open to all specialties that are catered by the Nursing Unit.
