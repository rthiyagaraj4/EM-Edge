‘By Nursing Unit’ Tab

 

  1. Nursing Unit – Enter the Nursing Unit in alphanumeric format of 15 characters maximum length, or search for the Nursing Unit by clicking the [?] function provided. This field is mandatory. Enter the required search criteria in the search screen and click [Search] to display the Nursing Units that satisfy the entered search criteria. If the Nursing Unit description is known, enter the description directly and click [Search].

The following details are displayed on selecting the Nursing Unit:

  1. Nursing Unit Type – This field displays the Nursing Unit Type associated to the selected Nursing Unit.

  2. Age Group – This field displays the Age Group associated to the selected Nursing Unit.

  3. Account Department – This field displays the Accounting Department to which the nursing unit is associated.

  4. Service – This field displays the Service associated to the selected Nursing Unit.

  5. Search By – Select the Search criteria as either Practitioner ID or Practitioner Name from the list and enter the required search string in the text box based on the selected criteria.

 

‘By Practitioner’ Tab

 

  1. Practitioner - Enter the name of the practitioner in alphanumeric format of 15 characters maximum length, or select the practitioner by clicking [?] function. This field is mandatory.

Enter the following criteria to search for the required practitioner.

  1. Search By – Select the option as either Description or Code. The ‘Description’ radio button is selected, by default.

  2. Search Criteria – Select the Search Criteria from the list of values provided. The values displayed are Starts With, Ends With and Contains. Starts With is displayed, by default.

  3. Practitioner ID – This field is enabled on selecting the Search By option as Code. Enter the Practitioner ID in alphanumeric format of 15 characters maximum length. The entered ID has to be the same as that created in the function Practitioner in Application Masters module.

  4. Practitioner Type – Select the required Practitioner Type from the list.

  5. Practitioner Name - This field is enabled on selecting the Search By option as Description. Enter the search text in alphanumeric format. Depending on the search criteria selected and search text entered, the results are displayed. If the search criteria is selected as Starts with and search text entered, the result set displayed starts with the entered text. If the search criteria is selected as Ends with and search text entered, the result set displayed ends with the entered text. If the search criteria is selected as Contains and search text entered, the result set displayed contains the entered text.

  6. Specialty – Select the required Specialty from the list.

  7. Gender – Select the required Gender from the list of valid values. The listed values are Male, Female and Unknown.

  8. Position – Select the required position from the list.

The following details are displayed on selecting the Practitioner ID:

  1. Practitioner Type – This field displays the Practitioner Type as selected during the creation of practitioner.

  2. Primary Specialty – This field displays the Primary Specialty of the practitioner.

  3. Search By – Select the Search criteria as either Nursing Unit Code or Nursing Unit Description from the list and enter the required search string in the textbox based on the selected criteria.
