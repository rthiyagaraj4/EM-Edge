Enter the following search criteria:

  1. Nursing Unit - Enter the Nursing Unit or select the Nursing Unit using [?] function.

  2. Priority - Select the priority as 'Urgent', 'Non-urgent' or 'Both'. 'Both' is displayed, by default.

  3. Specialty - Enter the Specialty or select the Specialty using [?] function.

  4. Practitioner - Enter the Practitioner name or select the practitioner using [?] function.

  5. Bed Class – Select the Bed Class from the list.

  6. Bed Type – Select the Bed Type from the list.

  7. Medical Service Group – Select the Medical Service Group from the list.

  8. Booking Type – Select the Booking Type from the list. The listed values are defined in the Booking Type function.

  9. Preferred Date [From-To] - Enter the preferred Admission date range in date format of dd/mm/yyyy or select it using Calendar function.

  10. Booking Ref No – Enter the Booking reference number of the patient.

  11. Referral Source - Select the referral source from the list. The listed values are 'Login Facility', 'Enterprise' and 'External'. On selecting 'Login Facility', the current logged in facility is listed in the adjacent list. On selecting 'Enterprise', all the facilities of the enterprise other than the logged in facility are listed in the adjacent list. On selecting 'External', all the external facilities to which the patient can be referred are listed in the adjacent list. Select the required value as criteria.

  12. Referral ID - Enter the Referral ID in the alphanumeric format of 14 characters maximum length.

 

Patient Details

  1. Citizen ID - Enter the Citizen ID in alphanumeric format of maximum length as specified in the MP Parameter.

  2. Other Alt Type - Select the Other Alt Type from the list. The listed values are Birth Certificate, Passport, Mother's PMI, PMI, etc.

  3. Other Alt ID - Enter the Other Alternate ID in alphanumeric format of maximum length as specified in the MP Parameter.

  4. Previous HN - Enter the Previous Hospital Number in alphanumeric format of maximum length as specified in the MP Parameter.

  5. Government Number - Enter the Government Number in alphanumeric format of maximum length as specified in the MP Parameter.

  6. Driving License No - Enter the Driving License Number in alphanumeric format of maximum length as specified in the MP Parameter.

  7. Expatriate No - Enter the Expatriate Number in alphanumeric format of maximum length as specified in the MP Parameter.

The fields and their order displayed in the screen may vary depending on the MP Parameter Setup.

  1. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or click [?] function to select the Patient ID.

  2. Gender - Select the Gender of the patient from the list. The listed values are 'Male', 'Female' and 'Unknown'.

  3. Search Name with Local lang – Select this check box to search the Patient name with local language.

  4. Name - Specify the name of the patient in alphanumeric format of 40 characters maximum length. The query components and the legends pertaining to the patient name depend upon the set up done in MP Parameter under the 'Legal Names' Tab. The available components for the patient name are Title, First Name, Second Name, Third Name, Family Name and Suffix. The components that are made applicable for the facility appears as query criteria here.
