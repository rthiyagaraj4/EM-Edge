Enter the following details:

 

  1. Admission Date - The date of admission is displayed.

  2. Nursing Unit - The Nursing Unit recorded during admission is displayed. Click [?] function to change the Nursing Unit.

  3. Service - The Service recorded during admission is displayed. Select the Service from the list.

  4. Sub-service - The Sub-service recorded during admission is displayed. Select the Sub-service from the list.

  5. Bed Class - The Bed Class recorded during admission is displayed. Select the Bed Class from the list.

  6. Bed Type - The Bed Type recorded during admission is displayed. Select the Bed Type from the list.

  7. Bed No - The Bed Number recorded during admission is displayed. Click [?] function to change the Bed Number. If the selected bed is associated to a specialty, you cannot change the bed.

  8. Room/Cubicle No - The Room/Cubicle Number of the bed is displayed. If you change the Bed Number, the new Room/Cubicle Number of the bed is displayed.

  9. Specialty - The Specialty recorded during admission is displayed. Click [?] function to change the Specialty. The [?] function displays Specialties associated to the selected Nursing Unit.  if any specialty specific orders are placed or patient transfer is done or if the patient is occupying a specialty specific bed, the system restricts change in specialty. A Bed is associated to Specialty in Bed for Nursing Unit function. An Order is set as Specialty Specific in the Order Catalog function in Order Entry and Result Reporting module.

  10. Admission Type - The Admission Type recorded during admission is displayed. Select the Admission Type from the list.

  11. Ambulatory Status - The Ambulatory Status recorded during admission is displayed. Select the Ambulatory Status from the list.

  12. Expected Discharge Date - The Expected Discharge Date recorded during admission is displayed. Click the Calendar function to change the Expected Discharge Date if there are no bookings done considering the EDD.

  13. Procedure/Surgery Date Time - The Procedure/Surgery Date Time recorded in Admit Patient function or Record Procedure/Surgery Date function is displayed. Click the Calendar function to change the Procedure/Surgery Date Time.

  14. Admitting Practitioner - The Admitting Practitioner recorded during admission is displayed. Click [?] function to change the Admitting Practitioner.

  15. Attending Practitioner - The Attending Practitioner recorded during admission is displayed. Click [?] function to change the Attending Practitioner.

  16. Referral ID – Enter the Referral ID as generated during referral registration or click [?] function to perform Referral Patient Search to locate the referral patient details. The referrals are listed based on the selected location and Patient ID (if any). You can register new referrals only for the selected Specialty and Patient ID and associate to the selected visit.

  17. Reason for Change Admission - Click [?] function to select the Reason for Admission Change. The reasons are created in the 'Reasons' function in the Application Masters module. This is a mandatory field.

  18. Admission Remarks - Enter the remarks, if any. You can record the remarks, if it was not recorded during admission or update the recorded admission details. The recorded remarks can be viewed while generating the discharge bill through the Patient Billing Manager module.

If there are no transfers, orders placed or any service charged,

  • The admission details Nursing Unit, Attending Practitioner, Bed Class, Bed Type, Bed No., Admitting Practitioner, EDD and Specialty can be changed.

If any transfers, orders placed or any service charged,

  • The Nursing Unit details, Attending Practitioner, Bed Class, Bed Type and Bed No are displayed and disabled.

  • Admitting Practitioner, EDD and Specialty are displayed and enabled.

If orders placed or any service charged,

  • EDD, Admitting Practitioner and Specialty can be changed.
