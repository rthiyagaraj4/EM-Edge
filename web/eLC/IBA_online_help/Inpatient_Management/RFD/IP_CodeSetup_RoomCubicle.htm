Enter the following details to create a new Room/Cubicle record:

 

  1. Facility – Select the required Facility from the list of valid values. The listed values contain all the facility(s) that can be accessed by the logged in user. This field is mandatory. The current logged in facility is displayed, by default.

  2. Room/Cubicle No - Enter the Room/Cubicle Number in alphanumeric format of four characters maximum length. A room/cubicle number is used to identify each room/cubicle uniquely. The room/cubicle number has to be unique and cannot be changed after creation. This field is mandatory.

  3. Long Description - Enter the long description of the Room/Cubicle in alphanumeric format of 30 characters maximum length. This field is mandatory.

  4. Short Description - Enter the short description of the Room/Cubicle in alphanumeric format of 15 characters maximum length. This field is mandatory.

  5. Type - Select the Type of Room/Cubicle from the list of valid values. All the care locations created in Application Master (Care Location Type with care location type indicator as Room) gets displayed in the list. This field is mandatory.

  6. Building - Select the Building in which the Room/Cubicle is located from the list of valid values. All the buildings created using the function Building in Facility in Application Masters module are displayed in the list.

  7. Floor - Select the Floor in which the Room/Cubicle is located from the list of valid values. All the floors attached to the selected building in Floor in Facility function of Application Masters module are displayed in the list.

  8. Resident Stay – Select a value to indicate if the room being set is for resident stay for an Inpatient or Daycare patient. The rooms are populated appropriately in the Room/Cubicle for Nursing Unit function when a nursing unit or day care unit is selected.

  9. Tel No - Enter the Telephone Number of the Room/Cubicle in alphanumeric format of 18 characters maximum length.

  10. Extension - Enter the Extension Number of the telephone number of the Room/Cubicle in alphanumeric format of six characters maximum length.

  11. Enabled - Select the check box if the Room/Cubicle is to be enabled or clear the check box, if the Room/Cubicle is to be disabled. the check box is selected, by default.

 

Characteristics

  1. Room/Cubicle Status - Select the privacy level of the Room/Cubicle from the list of valid values. The Privacy Level can be selected only if Daycare Unit or Nursing Unit is selected in the Resident Stay field, or it is disabled. The valid Privacy Levels are Isolation, Private Room, Semi-Private room and Room/Cubicle.

  2. Infectious Disease - Select the check box if the Room allows patients with infectious diseases. This is enabled only if Room/Cubicle status is selected as Isolation.

  3. No of Beds – Enter the number of beds in numeric format of three characters maximum length. This indicates the maximum number of beds that can be accommodated in this Room/Cubicle. This field is mandatory.

  4. Usable by Radiation Implant Patients - Select the check box to indicate whether the Room/Cubicle can be used by Radiation Implant Patients.

  5. Smoking - Select the check box to indicate whether smoking is permitted in the Room/Cubicle.

 

Available Equipment

  1. Oxygen - Select the check box to indicate that Oxygen equipment is available in the Room/Cubicle.

  2. Suction - Select the check box to indicate that Suction equipment is available in the Room/Cubicle.

  3. Vital Signs - Select the check box to indicate that Vital Signs equipment is available in the Room/Cubicle.

  4. Infusion Pump - Select the check box to indicate that Infusion Pump equipment is available in the Room/Cubicle.

  5. IV Pump - Select the check box to indicate that IV Pump equipment is available in the Room/Cubicle.

  6. Electro Encephalogram - Select the check box to indicate that Electro Encephalogram equipment is available in the Room/Cubicle.

  7. Electro Cardiogram – Select the check box to indicate that Electro Cardiogram equipment is available in the Room/Cubicle.

  8. Ventilator – Select the check box to indicate that Ventilator equipment is available in the Room/Cubicle.
