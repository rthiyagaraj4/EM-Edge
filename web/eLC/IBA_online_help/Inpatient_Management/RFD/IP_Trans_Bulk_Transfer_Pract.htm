Enter the following search criteria:

 

Transfer From

  1. Nursing Unit - Enter the Nursing Unit in alphanumeric format or select the Nursing Unit using [?] function. The Nursing Units associated to the Operator Station of the logged in user are listed.

  2. Admission Date [From-To] - Enter the Admission Date range in the format of dd/mm/yyyy or select the Date using the calendar function.

  3. Specialty - Enter the Specialty in alphanumeric format or select the Specialty using [?] function. The Specialties created in the 'Specialty' function of Application Masters module are displayed.

  4. Attending Practitioner - Enter the Attending Practitioner in alphanumeric format or select the Practitioner using [?] function. The Attending Practitioners associated to the logged in facility in the 'Practitioner for Facility' function of Application Masters module are displayed. If 'Specialty' is entered, the Attending Practitioner list displays the names of those practitioners who are associated with that particular Specialty. This field is mandatory.

 

Transfer To

  1. Transfer Type - Select the Type of transfer from the list. The values in the list are defined in the 'Transfer Type' function in IP Code Setup. This field is mandatory.

  2. Attending Practitioner - Enter the Attending Practitioner in alphanumeric format or select the Practitioner using [?] function. The Attending Practitioners associated to the logged in facility in the 'Practitioner for Facility' function of Application Masters module are displayed. This field is mandatory.
