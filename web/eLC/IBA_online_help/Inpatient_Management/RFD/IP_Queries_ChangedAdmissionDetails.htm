Enter the following details:

 

  1. Nursing Unit - Enter the Nursing Unit in alphanumeric format of 15 characters maximum length, or select the Nursing Unit using [?] function.

  2. Admission Date [From-To] - Enter the Admission date range or select the Admission Date using the Calendar function. The current system date is displayed, by default.

  3. Specialty - Enter the Specialty in alphanumeric format of 15 characters maximum length, or select the Specialty using [?] function.

  4. Practitioner - Enter the Practitioner in alphanumeric format of 30 characters maximum length, or select the Practitioner using [?] function.

  5. Patient ID - Enter the Patient ID in alphanumeric format of 10 characters maximum length, or select the Patient ID using [?] function.

  6. Gender - Select the Gender from the list. The listed values are All, Male, Female and Unknown.

  7. Encounter ID - Enter the Patient ID in alphanumeric format of 12 characters maximum length.

  8. Modified By - Enter the name of the user who modified the admission details in alphanumeric format of 15 characters maximum length, or select the user name using [?] function.

  9. Modified Date [From-To] - Enter the Modified date range or select the modified date range using the Calendar function. The current system date is displayed, by default. This is a mandatory field.
