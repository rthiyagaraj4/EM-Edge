Check-out Accompanying Person

From this function you can Check-out an Accompanying Person who has checked in and Enquire about the details of accompanying person.

The lodging charges applicable are calculated. The bill is generated and printed while checking out, if the Patient Billing Manager module is installed. If the Accompanying Person is not checked out while discharging the patient, the same is performed using this function.

Checking out Accompanying Person

From the IP Transactions > Lodger/Accompanying Person, select the Check-out Accompanying Person function.

The Check-out Accompanying Person screen gets displayed.

  1. Click [Search] to display the list of all Accompanying Persons or enter the search criteria and click [Search] to select a specific Accompanying Person. Click [Clear] to reset the records displayed in the result area.

    Enter the following query criteria:

    1. Nursing Unit – Select the Nursing Unit using [?] function.

    2. Accompanying Person ID – Enter the Accompanying Person ID in alphanumeric format of length as specified in the MP Parameter or select using [?] function.

    3. Patient ID – Enter the Patient ID of the Inpatient in alphanumeric format of maximum length as specified in the MP Parameter.

    4. Checked In Date (From-To) – Enter the date range in date format of dd/mm/yyyy or select the date from the calendar function as to when the Accompanying Person was checked-in.

    5. Gender – Select the Gender of the Accompanying Person from the list.

  2. On searching, Accompanying Person details are displayed based on the entered criteria.

  3. Click ‘Accompanying Person Name’ hyperlink to display the Check-out Accompanying Person screen. The screen displays the Accompanying Person Details and Stay Details recorded while checking in Accompanying Person.

  4. Enter the Check out Date and Time. The current system date and time is displayed, by default. The entered value must not be greater than current system date and time and lesser than the checked in date and time.

  5. Click [Check Out] to check-out the Accompanying Person or [Close] to exit from the screen.
