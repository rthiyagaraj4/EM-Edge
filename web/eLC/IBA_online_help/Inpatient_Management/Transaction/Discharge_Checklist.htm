Discharge Checklist

From this function you can Record Discharge Checklist before discharging the patient.

The discharge checklist is used to record if all the activities, like preparing discharge summary, returning the valuables to the patient and so on have been completed before discharging the patient. This function lists out all the patients for whom the discharge advice has been prepared.

Marking of the checklist becomes a mandatory step in the discharge process of the patient if the flag ‘Discharge Check List Applicable’ is selected in the IP Parameter setup.

Preparing Discharge Checklist

From the IP Transactions > Discharge, select the Discharge Checklist function.

The Discharge Checklist screen gets displayed.

  1. Click [Search] to display the list of all Inpatients with discharge advice prepared or enter the search criteria and click [Search] to select a specific Inpatient with discharge advice prepared. Click [Clear] to reset the records displayed in the result area.

    Enter the following query criteria to select the required patient record:

    1. Nursing Unit – Enter the Nursing Unit in alphanumeric format of 15 characters maximum length or select the Nursing Unit using the [?] function. The Nursing Units associated to the Operator Station of the logged in user are listed.

    2. Admission Date From-To – Enter the Admission Date range in the format of dd/mm/yyyy or select the date using the calendar function.

    3. Specialty – Enter the Specialty in the alphanumeric format or select the Specialty using the [?] function. The Specialties created in ‘Specialty’ function of Application Masters module are displayed.

    4. Practitioner – Enter the Practitioner in alphanumeric format or select the Practitioner using the [?] function. The Practitioners attached to the logged in facility in ‘Practitioner for Facility’ function of Application Masters module are displayed.

    5. Encounter ID – Enter the Encounter ID in numeric format of 12 digits maximum length.

    6. Discharge Patient – Select the check box to search discharged patients.

    7. Discharge Date From-To – Enter the Discharge Date range of patients in the format of dd/mm/yyyy. This field is mandatory if ‘Discharge Patient’ check box is selected.

    8. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the Patient ID using the [?] function.

  2. On searching, Inpatient Details are displayed based on the entered criteria.

  3. Click ‘+’ hyperlink to view the Current Encounter Details.

  4. Click the ‘Encounter ID’ hyperlink to display the Discharge Checklist screen.

  5. Enter the details in the mandatory (*) fields of the tab pages.
