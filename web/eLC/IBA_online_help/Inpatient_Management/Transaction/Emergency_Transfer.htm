Emergency Transfer

From this function you can Record the emergency transfer of a patient and Query for a required emergency transfer record.

The ‘Emergency Transfer’ function is used to perform faster transfers in case of critical situations without placing a request for transfer.  This function is controlled by the User access rights.

Recording Emergency Transfer

From IP Transactions > Transfer, select the Emergency Transfer function.

The Emergency Transfer screen gets displayed.

  1. Click [Search] to display the list of all Inpatients for emergency transfer or enter the search criteria and click [Search] to select a specific Inpatient. Click [Clear] to reset the records displayed in the result area.  

    1. Nursing Unit - Enter the Nursing Unit in alphanumeric format or select the Nursing Unit using [?] function. The Nursing Units associated to the Operator Station of the logged in user are listed.

    2. Specialty - Enter the Specialty in alphanumeric format or select the Specialty using [?] function. The Specialties created in the 'Specialty' function of Application Masters module are displayed.

    3. Encounter ID - Enter the Encounter ID in numeric format of 12 digits maximum length.

    4. Patient ID - Enter the Patient ID in alphanumeric format as specified in MP Parameter or select using the [?] function.

    5. Admission Date [From-To] – Enter the Admission Date range in dd/mm/yyyy format or select using the Calendar function.

    6. Practitioner - Enter the Practitioner name or select the practitioner using [?] function.

  2. On searching, the Inpatient details are displayed based on the entered criteria.

  3. Click the ‘+’ hyperlink to view the Current Encounter Details.

  4. Click the ‘Encounter ID’ hyperlink to display the Emergency Transfer screen. The Current Occupancy Nursing Unit details are displayed.

  5. Enter the details in the mandatory (*) fields. <Related Field Description>

  6. Click [Confirm] to record the emergency transfer details for the required patient or [Close] to exit from screen.
