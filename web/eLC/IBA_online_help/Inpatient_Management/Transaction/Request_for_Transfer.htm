Request for Transfer

From this function you can Request for transfer of an Inpatient from one Nursing Unit to another Nursing Unit.

The source Nursing Unit or current Nursing Unit identifies the destination Nursing Unit details along with the Specialty and the priority for the transfer. Request is placed only on those Nursing Units that warrant prior booking before transfer. This setup is observed against the appropriate Nursing Unit using the query mode in the function IP Code Setup ‘Nursing Unit’. The logged in user must have rights to invoke this function. Rights are defined in the Operator Station assigned to the logged in user. If the transferred patient has a baby registered, the system prompts you to de-link the mother and baby link (which is established when the baby is given a bed associated with the mother’s bed.) Only on de-linking the link you are able to proceed with the transfer.

The Bed Class and Bed Type can also be changed while requesting for transfer. In case the flag ‘Request for Bed Class Change’ is selected in the IP Parameter, you have the provision of requesting for a different bed belonging to a different Bed Class. If the user performing the request for transfer has the privilege of confirming such requests, as defined by selecting the flag ‘Bed Class Change’ against the Operator Station to which the logged in user is attached to, the change for Bed Class request is confirmed in this transaction itself and request for the transfer is completed. If the logged in user does not have the privilege, all such requests are routed to the ‘Confirm Bed Class Change’ function where an authorized user confirms the Bed Class change after which the request for transfer is completed.

Request for Transfer

From the IP Transactions > Transfer, select the Request for Transfer function.

The Request for Transfer screen gets displayed.

  1. Click [Search] to display the list of all Inpatients or enter the search criteria and click [Search] to select a specific Inpatient. Click [Clear] to reset the records displayed in the result area.

    Enter the following query criteria:

    1. Nursing Unit – Enter the Nursing Unit in the alphanumeric format of 15 characters maximum length or select the Nursing Unit using [?] function. All the Nursing Units associated to the Operator Station of the logged in user is listed.

    2. Admission Date From-To – Enter the Admission Date range of when the patient was admitted in date format of dd/mm/yyyy or select the date from the calendar function.

    3. Specialty – Enter the Specialty in the alphanumeric format or select the Specialty using [?] function. All the Specialties created in Specialty function of Application Masters module are displayed.

    4. Practitioner – Enter the Practitioner in alphanumeric format or select the Practitioner using [?] function. All the Practitioners attached to the logged in facility in ‘Practitioner for Facility’ function of Application Masters module are displayed.

    5. Encounter ID – Enter the Encounter ID in numeric format of 12 digits maximum length.

    6. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the ID using [?] function.

  2. On searching, Inpatient details are displayed based on the entered criteria.

  3. Click ‘+’ hyperlink to view the Current Encounter Details.

  4. Click the ‘Encounter ID’ hyperlink to display the Request for Transfer screen.

  5. Enter the details in the mandatory (*) fields. <Related Field Description>

  6. Click [Request] to request for transfer from one Nursing Unit to another or [Close] to exit from the Request for Transfer screen.

  7. If the bed is reserved for the selected patient on clicking [Request], an alert message 'The reserve bed will be cancelled on confirmation of this request .Do you want to continue?' is displayed. Click [OK] to continue.

  8. On requesting, you can reserve the existing bed on 'Request for Transfer/Transfer Out'. An alert message 'This bed will be reserved on Transfer In of this Patient. Do you want to proceed?' is displayed. Click [OK] to proceed.

 

If the selected encounter already has a request placed for a transfer to some other Nursing Unit, the record is marked in pink indicating that a request has already been placed for this encounter. Intra-ward transfer is not allowed for such cases. The system prompts you with the appropriate message and does not proceed with the continuation of the transaction.
