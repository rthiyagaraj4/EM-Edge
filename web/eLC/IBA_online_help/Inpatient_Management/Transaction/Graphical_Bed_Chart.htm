Graphical Bed Chart

From this function you can View the graphical representation of each rooms of the selected Nursing Unit.

Graphical Bed Chart is the graphical representation of rooms with the color indicated on each bubble. Each status has a unique color. The color assigned for each status is displayed in the upper part of the screen. Each bubble consist Room No, Patient ID, current Encounter ID, Gender and Age details of the particular patient in the Room/Cubicle.

Viewing Graphical Bed Chart

From the IP Transactions > Bed Maintenance, select the Graphical Bed Chart function.

The Graphical Bed Chart screen gets displayed.

  1. Select the Nursing Unit and Search or Enter the Patient ID for which the graphical representation has to be displayed.

  2. All the rooms associated for that particular Nursing Unit gets displayed.

  3. In the occupied room: Room No, Patient ID, current Encounter ID, Gender and Age details are displayed.

  4. Click Encounter ID hyperlink, a Manage Bed dialog gets displayed.

  5. From the Manage Bed dialog, Assign Treating Practitioner, Prepare Discharge Advice, Cancel Discharge Advice, Admit Patient, Assign Bed/Mark Patient Arrival, Patient Chart  and Discharge Patient functions can be carried through.
