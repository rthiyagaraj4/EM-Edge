Practitioner Transfer

From this function you can Transfer care from one Practitioner to another belonging to the same Specialty or different Specialties within the Nursing Unit.

The role of the previous Attending Practitioner is closed by Practitioner Transfer transaction, and can be observed by invoking the functions 'Assign Treating Practitioner' in Admission and 'Role End Date Time' against the previous Attending Practitioner. The latest Attending Practitioner record is observed with the Role Start Date Time same as the Role End Date Time of the previous Attending Practitioner.

Transferring a Practitioner

From the IP Transactions > Transfer, select the Practitioner Transfer function.

The Practitioner Transfer screen gets displayed.

  1. Click [Search] to display all the Inpatients or enter the search criteria and click [Search] to select a specific Inpatient. Click [Clear] to reset the records displayed in the result area.

    Enter the following query criteria:

    1. Nursing Unit – Enter the Nursing Unit in alphanumeric format of 15 characters maximum length or select the Nursing Unit using the [?] function.

    2. Admission Date From-To – Enter the Admission Date Range in the format of dd/mm/yyyy or select the date using the calendar function.

    3. Specialty – Enter the Specialty in alphanumeric format of 15 characters maximum length or select the Specialty using the [?] function.

    4. Practitioner – Enter the Practitioner name in alphanumeric format of 30 characters maximum length or select the Practitioner using the [?] function.

    5. Encounter ID – Enter the Encounter ID in numeric format of 12 digits maximum length.

    6. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the Patient ID using the [?] function.

  2. On searching, Inpatient Details are displayed based on the entered criteria.

  3. Click ‘+’ hyperlink to view the Current Encounter Details.

  4. Click the ‘Encounter ID’ hyperlink to display the Practitioner Transfer screen. The screen displays the Admission Details, Nursing Unit details, Current Occupancy and the Transfer To details.

  5. Select the Transfer Type from the list. The listed values are defined in the Transfer Type function of IP Code Setup. This is a mandatory field.

  6. Select the required Service from the list. The Services associated to the Nursing Unit are displayed.

  7. Enter the ‘ Transfer To’ details in the mandatory (*) fields.

    1. Specialty – Enter the Specialty to which the patient has to be transferred to from the current Specialty or select the Specialty using the [?] function. This is a mandatory field. The current Specialty is displayed, by default.

    2. Practitioner – Enter the name of the Practitioner or select the Practitioner using the [?] function. The selected Practitioner must be different from the current Practitioner treating the patient. The lookup function lists all Practitioners belonging to the selected Specialty. This field is mandatory.

    3. Medical Team – On selecting the Practitioner this field is displayed. Select the Medical Team from the list. The list displays the Medical Team associated to the selected Practitioner.

  8. Click [Transfer] to transfer care from one practitioner to another or [Close] to exit from the Practitioner Transfer screen.
