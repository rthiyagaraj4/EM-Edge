Specialty for Nursing Unit

From this function you can Create an association of one or more specialties to a particular Nursing Unit and Query for the desired Specialty for Nursing Unit records.

Nursing Units within a facility cater to one or more Specialties. Assigning one or more specialties to a Nursing Unit helps in assigning the specialties for a bed linked to the nursing unit. This association influences admitting patients under a Specialty within a Nursing Unit.

Creating an association of Specialties to Nursing Units

From the IP Setup > IP Code Setup, select the Specialty for Nursing Unit function.

The Specialty for Nursing Unit screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Search] to display Specialties and select the check box against the required Specialties to associate to the Nursing Units.

    1. Specialty –The name of the specialty is displayed.

    2. Select All – Select the check box to indicate applicability of specialties for the selected Nursing Unit. To select all the displayed specialties, select the check box provided in the column header. All check boxes are cleared, by default.

  4. Click [Apply] to save the association of Specialties to a Nursing Unit or [Reset] to clear the entered data.

  5. Click [Query] to search for associated Specialty for Nursing Unit records.
