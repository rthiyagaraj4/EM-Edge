Nursing Unit

From this function you can Create a new Nursing Unit by its location type within the facility applicable to either Daycare/Inpatients and Query for a desired Nursing Unit.

A Nursing Unit is a logical location in an inpatient setting to control the activities of various rooms associated to the Nursing Unit. Nursing Units control the admission process and transfer process against beds in the rooms associated to the Nursing Unit. Attributes like age group and gender restrictions, level of care applicability, inclusion for hospital occupancy calculations, file storage location and accommodation of pseudo beds are maintained at the Nursing Unit level and these influence the operations across all the inpatient beds associated to the Nursing Unit.

Creating a Nursing Unit

From the IP Setup > IP Code Setup, select the Nursing Unit Function.

The Nursing Unit screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the record or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Nursing Units.
