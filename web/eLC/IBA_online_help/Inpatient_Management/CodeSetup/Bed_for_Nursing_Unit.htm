Bed for Nursing Unit

From this function you can Create and associate beds of a particular Bed Class and Bed Type to Rooms/Cubicles within a Nursing Unit and Associate beds to a Main Bed. You can Query for a bed within a Nursing Unit and Delete a desired bed associated to a Nursing Unit.

Beds are organized in a Nursing Unit within Rooms/Cubicles and assigned a unique bed number that cannot be duplicated. Beds defined are specific to an Age Group, Gender and Specialty. From this function you can also attach an additional bed to a main bed. For example, during New Born Admission, the baby is provided with a baby bed attached to the mother bed.

Creating an association of Beds to Nursing Units

From the IP Setup > IP Code Setup, select the Bed for Nursing Unit function.

The Bed for Nursing Unit screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the association of Bed for Nursing Unit or [Reset] to clear the entered data.

  4. Click [Query] to search for beds within the Nursing Unit.

The user is allowed to enable/disable beds that have already been used in transactions in the Query mode. If a bed is occupied by a patient, the system displays a message, ‘Bed is occupied by a patient. Cannot modify details.’ Beds that are disabled and vacant can be enabled at any instant.

  1. Click [Delete] to delete the queried record.
