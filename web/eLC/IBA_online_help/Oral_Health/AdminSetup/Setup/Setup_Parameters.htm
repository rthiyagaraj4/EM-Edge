Setup Parameters

From this function you can Define if the Baseline Chart for a patient is available at the Enterprise level or at the Facility Level.

The Setup Parameters function is used to define if the Baseline Chart created for a patient visiting the facility is accessible throughout the enterprise. Practitioners across the facilities can view and access the Baseline Chart created in a facility and update the existing Chart, if the Chart Level is selected as 'Enterprise'. If the Chart Level is selected as 'Facility', dentist from a facility can view and access the Charts created in another facility, but cannot update the Chart. The charts cannot be closed until it expires based on the validity set in 'Setup Parameters for Facility' function. If the Chart Level is selected as 'Enterprise', values defined using this function are considered whereas if the Chart Level is selected as 'Facility', parameter values defined using the Setup Parameters for Facility function are considered.

Defining Parameter

From the Admin Setup > Setup menu, select the Setup Parameters function.

The Setup Parameters screen gets displayed.

  1. Select a Chart Level from the list. Select Facility, if the Baseline Chart has to accessible to a dentist in a facility where it is created. Select Enterprise, if the Baseline Chart has to accessible to dentists across the facilities in an Enterprise. This is a mandatory field.

  2. Select the check box 'Allow result reporting after recording the performing location details' to record result reporting only after recording the performing location details. Clear the check box, if the user can record the result report without entering the performing location details.

  3. Click [Apply] to save the details or [Reset] to clear the entered data.
