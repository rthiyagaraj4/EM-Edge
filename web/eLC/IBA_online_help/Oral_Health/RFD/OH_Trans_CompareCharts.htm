Compare

Compare tab allows Comparison between two charts. Practitioner can choose to compare between Baseline vs. Baseline, Baseline vs. Evaluation and Evaluation vs. Evaluation. The system accepts the patient Baseline/Evaluation dates to be compared. Click [Refresh] to view the charts based on the parameter selected. You cannot compare the charts, which are closed.

 

The values recorded for the components in the selected charts get displayed. The recorded values are displayed for the selected tooth number. The date when the Evaluation Chart and the Baseline Chart is created is also displayed.

 

The components associated to the selected Chart get listed on the left hand side of the screen. You can select the check box against the components to compare the charts with only the selected components. Click [Compare] to compare the charts with the selected components only.
