Enter the following details to compare charts:

 

  1. Numbering System – The Numbering System selected while defining the parameters in 'Setup Parameters For Facility' function in the Admin Setup gets displayed. This is a mandatory field.

 

Compare

You can compare the values across charts that are marked as Completed.

Compare - Select a value from the list to compare the values in two or three charts. The system-defined values are '2 Charts' and '3 Charts'.

  • Baseline vs. Baseline – Select ‘Baseline vs. Baseline’ to compare a Baseline chart with another.  

  • Baseline vs. Evaluation – Select ‘Baseline vs. Evaluation’ to compare a Baseline chart with an Evaluation Chart.

  • Evaluation vs. Evaluation – Select ‘Evaluation vs. Evaluation’ to compare an Evaluation chart with another.

  1. Across Charts – Select the check box to compare Evaluation Charts or Baseline Charts with each other or across charts.  

  2. Chart – Select a Chart from the list. The Charts defined using the Chart Definition function are listed. This is a mandatory field.

  3. View By – Select a value from the list to view the values recorded in the Chart. The system defined values are ‘Upper Buccal’, ‘Lower Buccal’, ‘Upper Lingual’ and ‘Lower Lingual’.

  4. Baseline Chart – Select a Baseline Chart from the list. The Baseline Charts created and completed using the ‘Baseline Chart’ tab gets listed. This is a mandatory field if the charts selected for comparison are 'Baseline vs. Evaluation' or  'Baseline vs. Baseline'.

The legend changes to ‘Evaluation Chart’ or 'Baseline Chart'  when you select to compare the values between two or three Evaluation Charts or Baseline Charts respectively.

  1. Evaluation Chart - Select an Evaluation Chart from the list. The Evaluation Charts created and completed using the ‘Evaluation Chart’ tab gets listed. This is a mandatory field if the charts selected for comparison are 'Baseline vs. Evaluation' or  'Evaluation vs. Evaluation'.

  2. [Refresh] – Click [Refresh] to view the charts.
