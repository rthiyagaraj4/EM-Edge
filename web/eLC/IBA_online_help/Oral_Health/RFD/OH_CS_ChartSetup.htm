Enter the following details to create a Chart:

 

  1. Chart ID – Enter the Chart ID in alphanumeric format of four characters maximum length. A Chart ID has to be unique and it cannot be changed once defined. This is a mandatory field.

  2. Description – Enter the Description in alphanumeric format with a maximum of 30 characters. This is a mandatory field.

  3. Enabled – Select the check box to enable the record in the transaction functions. The check box is selected, by default.

 

Components

  1. Component – The Components are system defined. You can choose the components, which has to be associated to the Chart ID.

  2. Sequence – Enter the Sequence in numeric format of five digits maximum length. The sequence indicates the order in which the component has to appear on selecting the Chart while using Periodontal Charting.

  3. Min Value – The minimum value is system defined for each component. You can update the minimum value only for the selected components. The minimum value for PD is 0 and CEJ-GM is set to -15. This is used to indicate the condition of the tooth in the Periodontal Charting. The minimum value among PD or CEJ-GM is considered as the minimum value for CAL.

  4. Max Value - The maximum value is system defined for each component. You can update the maximum value only for the selected components. The maximum value for CEJ-GM and Pocket Depth (PD) is set to 15. The maximum value for Clinical Attachment Level (CAL) is 30. You cannot enter the values for Clinical Attachment Level (CAL); it is the sum of the values recorded for CEJ-GM and Pocket Depth (PD). This is used to indicate the condition of the tooth in Periodontal Charting.

  5. Red if > = - The value is system defined for each component. You can update the values only for the selected components. The value recorded is used to specify the condition of a tooth in Periodontal Charting. The component with the value specified indicates that the tooth condition is normal. If the value recorded for the component in the selected chart exceeds the entry recorded, the condition is displayed in red color, which indicates that the tooth condition is severe.

  6. Select – Select the check box to associate the component to the Chart ID. Clear the check box if the component is not valid for the Chart. The check box is cleared, by default.
