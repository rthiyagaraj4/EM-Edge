External Performing Location

From this function you can Create external locations where the orders are performed and Query for the existing external performing locations.

The External Performing Location function is used to create external locations where the orders are performed. The orders can be sent to the external locations to perform the required test.

Creating External Performing Location

From the Code Setup > Others menu, select the External Performing Location function.

The  External Performing Location screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the records or [Reset] to clear the entered details.

  4. Click [Query] to search for existing records.
