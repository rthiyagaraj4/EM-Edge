Replacement GRN

From this function you can Record the replacement details of a returned Item for which ‘Replacement RTV’ is applicable.

The Items for which the Replacement RTV has been assigned are displayed. The goods that are damaged, incorrect or ordered in error and are to be returned to the Vendors can be replaced by the Vendor with the required quantity.

Replacing the Returned Goods

From the MM Transactions, select the Replacement GRN function.

The Replacement GRN screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Modify Document Details screen is displayed or click ‘Document’ hyperlink to record the document details.

    The following details are displayed in the Document section:

     

    1. Document – The default Document Type as set in ‘ST Facility Parameter’ is displayed. Select the Document Type from the list. The documents associated to the transaction ‘Goods Receipt’ is displayed in the list. This is a mandatory field.

    2. Doc No. - The Document Number generation depends on the selected Document Type. You are informed if the Document Number has to be manually generated for the selected Document Type. Enter the Document Number in the numeric format of eight digits maximum length. If the system generates the number, you are not allowed to enter the field value. This is a mandatory field.

    3. Doc Date - The system date is displayed, by default. You can change the Document Date by clicking the Calendar function or enter the date in the format of dd/mm/yyyy. You can modify the date if the ‘Doc Date Modifiable Y/N’ check box is selected in ST Transaction Type function. This is a mandatory field.

    4. Doc Ref – Enter the Document Reference in alphanumeric format of 20 characters maximum length.

    5. Store - The default Store as set in ‘ST Facility Parameter’ is displayed. Select the Store from the list. Only those Stores are displayed for which the User has access and if ‘Goods Receipt’ Transaction function is applicable. This is a mandatory field.

    6. Item Class – Select the Item Class from the list. The listed values are the Item Classes defined in the Code Setup.

    You can modify the document details by clicking on the ‘Document’ hyperlink.

  3. On entering the document details, click [OK] to continue recording details of Item to replace or [Reset] to clear the entered data.

  4. Enter the details in the mandatory (*) fields. <Related Field Description>

  5. Click [RTV REF] to select ‘Replacement GRN’ batch details.

  6. On selecting [RTV Ref], the message ‘RTV Reference data will be refreshed’ is displayed. Click [OK] to refresh the RTV Reference Data or [Cancel] to exit from the dialog. To perform the Replacement GRN function RTV REF is mandatory.

  7. Enter ‘Received Quantity’ for the replacement Item. Select ‘Close’ check box, if the replacement transaction is complete. Select the ‘Select’ check box to record the received item.

Once this is done, the transaction is complete and no more replacements can be entered against this batch.

  1. Click [Close] to close the ‘Replacement GRN Batch Search’ screen. A dialog with the message ‘Unit Cost will be changed based on RTV Cost’ is displayed. Click [OK] to change the Unit Cost based on the RTV Cost or [Cancel] to select another Item.

  2. ‘Associated RTV is enabled. Click ‘Associated RTV’ hyperlink to view the associated RTVs for replacement.

  3. Click [Add] to add Items for replacement or [Cancel] to cancel the selected Item details.

  4. The selected Item details are displayed in the Item Details frame on the left hand side. Click ‘Item Details’ hyperlink to modify the Item replacement details. Click [Modify] to modify the entered details or [Cancel] to cancel the modified changes.

  5. Click [Preview] to view the selected Item details. Click [X] to close the Replacement GRN Preview Details screen.

  6. You can select the ‘Delete’ check box to delete the selected record.

  7. Click [Apply] to replace the Item Returned details or [Reset] to clear the entered data.
