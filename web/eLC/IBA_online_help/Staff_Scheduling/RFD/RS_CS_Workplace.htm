Enter the following details:

 

  1. Location Type - Select the Location Type from the list. The listed values are defined in 'Location Type' function of Application Masters module. If the Location Type is not selected, the Location Type for the workplace appears as General.

  2. Workplace Code - Enter the Workplace Code in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  3. Workplace Description - Enter the Workplace description in alphanumeric format of 20 characters maximum length. This is a mandatory field.

 

Operating hours

  1. Working Days [From/To] - Enter the operating hours of Working Days for the workplace in hh:mm format. [From - To] corresponds to start and end time of the working day in the workplace. The minutes (mi) of end time cannot be entered as 00.

  2. Non-Working Days [From/To] - Enter the operating hours of Non-Working Days for the workplace in hh:mm format. [From - To] corresponds to start and end time of the non-working day in the workplace. The minutes (mi) of end time cannot be entered as 00.

  3. Holidays [From/To] - Enter the operating hours of Holidays for the workplace in hh:mm format. [From - To] corresponds to start and end time of the holiday in the workplace. The minutes (mi) of end time cannot be entered as 00.

  4. Staff Utilization Review Option - Select an option from the list. The listed values are 'Dependency based FTE', 'FTE', 'Staff to Bed Ratio' and 'Staff to Patient Ratio'.

  5. Dependency based FTE - Indicates staff utilization is reviewed based on full time equivalent computed on the basis of patient dependencies.

  6. FTE (Full Time Equivalent) - Indicates staff utilization is reviewed based on Full Time Equivalent basis.

  7. Staff to Bed Ratio - Indicates staff utilization is reviewed on the basis of allocation of staff against beds available in a care location.

  8. Staff to Patient Ratio - Indicates staff utilization is reviewed on the basis of allocation of staff against patients admitted in a care location.

  9. Enabled - Select the check box to enable the Workplace record or clear check box to disable record.

 

Operating days applicability for the work cycle (1 weekly)

This section helps in defining the number of working days, non-working days and holidays that are followed in the workplace. The applicability can be set up for all the days of a week. The operating day's applicability work cycle is on a weekly basis as 'work cycle basis' function is set as 1 weekly in the parameter setup. Currently the Work Cycle Basis is restricted to 1 Week only. The working days, non-working days and holidays set for the workplace are validated during the transaction process for Master Schedule, Generate Work Schedule and Manage Work Schedule.

  1. Default (Mon-Sun) - Each day of the week is defaulted as a Working day.

  2. Week 1 (Mon-Sun) - Select the appropriate value from the respective list to set the day as a Working day, Non-Working Day or Holiday. The listed values are 'W Day', 'NW Day' and 'H Day'.
