Enter the following details:

  1. Shift Pattern ID - Enter the Shift Pattern ID in alphanumeric format of 10 characters maximum length. The shift pattern ID uniquely identifies a shift pattern. This is a mandatory field.

  2. Short Description - Enter the short description of the shift pattern in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  3. Long Description - Enter the long description of the shift pattern in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  4. Location Type - Select the required location type from the list. The listed values are defined in the 'Location Type' function of Application Masters module. If Location Type is not selected, the location type for the shift pattern is displayed as General.

  5. Enabled - Select the check box to enable the skill record or clear check box to disable the record.

 

Work cycle basis: 1 weekly

Based on the value set for 'Work Cycle Basis' in the parameter setup, the value is displayed as read only text. The work cycle basis is set as 1 weekly.

  1. Week 1 - Enter the required shift to be followed in each day of the week in alphanumeric format of one character.

All the shifts and indicators for the shifts are displayed on the right hand side of the screen for easy reference in entering the indicators of the shifts for the work cycle basis.
