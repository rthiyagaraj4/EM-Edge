Enter the following details:

 

  1. Location Type - Select the required Location Type from the list. The listed values are defined in 'Location Type' function of Application Masters module.

  2. Workplace - Select the Workplace using [?] function. If the Location Type is selected, the listed workplaces correspond to the selected location type or all the workplaces created in the ‘Workplace’ function are displayed. This is a mandatory field.

  3. Requirement Period (From/To) - Enter the effective date range of the Requirement Period in date (dd/mm/yyyy) format. The 'From' date should be greater than or equal to the system date. If the requirement period is specified, then the staff requirement for the workplace is considered as variable i.e. the requirement is effective only for the specified date range. If the requirement period is not specified then the staff requirement for the workplace is considered as fixed.

  4. Requirement description - Enter the description of the workplace requirement. This is a mandatory field.

  5. Click [Setup Requirement].

 

Shift/Position

  1. Shift/Position - All productive shifts associated to the workplace is displayed. The shifts are associated to the selected workplace in the function 'Shift for Workplace'. Under each shift the applicable positions are displayed. Positions are associated to a shift in the function 'Shift for Position'.

Workplace requirements are set under three column headers namely Fixed, By Ratio and By FTE. It is mandatory to specify the fixed requirement for the workplace.

 

Fixed

The minimum, optimal and maximum staff requirements for the workplace are specified as fixed requirements.

  • Min refers to the minimum staff requirement for the workplace.

  • Opt refers to the optimal/ideal staff requirement for the workplace. The optimal requirement forms the basis while generating master schedule for the workplace.

  • Max refers to the maximum staff requirement for the workplace.

Enter the fixed requirements corresponding to the position in a particular shift for the selected workplace:

  1. W Day - Enter the minimum, optimal and maximum staff requirements for the workplace on a working day.

  2. NW Day - Enter the minimum, optimal and maximum staff requirements for the workplace on a non-working day.

  3. Holiday - Enter the minimum, optimal and maximum staff requirements for the workplace on a holiday.

 

By Ratio

  1. (S:B) - The number of staffs at a workplace can be dependent on the number of beds, irrespective of the occupancy level of the beds.

  2. (S:P) - The number of staffs at a workplace can be dependent on the number of patients that vary over a shift period.

  3. Enter the requirement for workplace based on Staff to Bed ratio (S:B) and Staff to Patient ratio (S:P) corresponding to the position.

  4. W Day - Enter the S:B requirement and S:P requirement of the workplace for a working day.

  5. NW Day - Enter the S:B requirement and S:P requirement of the workplace for a non-working day.

  6. Holiday - Enter the S:B requirement and S:P requirement of the workplace for a holiday.

 

By FTE

  1. FTE - Requirement can be defined on the basis of Full Time Equivalent value required.

  2. Hours - Requirement can be defined on the basis of total duration required where hours represent duration required to fulfill a single FTE value.

  3. W Day - Enter the FTE requirement of staffs and FTE hours of the workplace for a working day.

  4. NW Day - Enter the FTE requirement of staffs and FTE hours of the workplace for a non-working day.

  5. Holiday - Enter the FTE requirement of staffs and FTE hours of the workplace for a holiday.

  6. Select - Select the check box to indicate the selection of the workplace requirement.

For a selected Position, all the (min, opt and max) should compulsorily have a value greater than zero for WDay/NW Day/Holiday.

For a selected Position, if any one (min, opt or max) has a value greater than zero for WDay/NW Day/Holiday, then all the min, opt and max should compulsorily have a value greater than zero for the WDay/NW Day/Holiday.
