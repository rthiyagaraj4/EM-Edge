Enter the following details:

 

  1. Reason Code - Enter the Reason Code in alphanumeric format of 4 characters maximum length. The reason code is used to uniquely identify the various reasons. This is a mandatory field.

  2. Reason - Enter the reason in alphanumeric format of 30 characters maximum length. The reason records created are displayed as predefined reasons applicable to particular activities in the transactions. This is a mandatory field.

  3. Applicable to - Select a value from the list to which the reason is applicable. The listed values are 'Allocate Shift', 'Cancel Schedule', ‘Cancel Unavailability’ 'Change Shift', 'Overtime', 'Swap Shift', 'Transfer' and 'Unavailability'. This is a mandatory field.

  4. Enabled - Select the check box to enable the reason in the transactions. Clear the check box to disable the record.
