Enter the following details:

 

  1. Facility – Select a facility from the list of facilities in the enterprise level. The listed values are defined in the ‘Facility’ function of ‘Enterprise Management’ module.

  2. Allow Overlap of Shifts in a Workplace – Select the check box to allow overlapping of shifts in a workplace. This is validated in the function ‘Shift for Workplace’. If this check box is cleared, overlapping of different shift timings is not allowed and if shift timings overlap an alert is displayed.

  3. Work Cycle Basis – Select the work cycle basis as 1 weekly from the list. Work cycle basis is displayed as 1 weekly throughout the module. Work Cycle Basis is currently restricted to ‘1 Weekly’. This is a mandatory field.

  4. Period ahead for Work Schedule creation – Enter the number of days in numeric format of three characters maximum length ahead for which a work schedule can be created. This is validated in the function ‘Generate Work Schedule’. This is a mandatory field.

  5. Maximum continuous duration staff can work – Enter the maximum time duration in hh:mm format for which the staffs can work continuously in a day. No working shift is allowed for the staff if the specified time limit is crossed. This is validated in the function ‘Manage Work Schedule’. This is a mandatory field.

 

Staff substitution basis

  1. Within same Grade – Select the check box if substitution of staff is allowed within the same grade or clear the check box if not allowed. This is validated in the function ‘Manage Work schedule’ while swapping staff.

  2. Low to High Grade – Select the check box to allow substitution of staff from low grade to high grade or clear the check box if not allowed. This is validated in the function ‘Manage Work schedule’ while swapping staff.

  3. High to Low Grade – Select the check box to allow substitution of staff from high grade to low grade or clear the check box if not allowed. This is validated in the function ‘Manage Work schedule’ while swapping staff.

Substitution is not allowed, if none of the check boxes are selected.

 

  1. Attendance Feature Required – Select the check box to enable the attendance feature in the facility or clear the check box to disable it. This is validated in the function ‘Record Attendance’.

  2. Max duration from start of shift to mark attendance – Enter the maximum duration from the start of the shift for which the attendance can be marked in hh:mm format. The attendance cannot be marked after the specified time limit. Field is enabled and mandatory if the above field ‘Attendance Feature Required’ is selected. This is validated in the function ‘Record attendance’.

  3. Validate staff access rights by attendance – Select the check box to validate access rights of staff by attendance or clear the check box. If the box is selected, the staff can login to the system only after attendance is marked or access is denied.

  4. Validate staff access rights by work schedule – Select the check box to validate access rights of staff by work schedule or clear the checkbox. If the box is selected, the staff is allowed to work in the assigned workplace only and staff is allowed to login to the system only during working hours scheduled to the staff.

 

Activities allowed on active shift: Duration from start of shift (hh:mm)

  1. Allocate – Select the check box to allow allocation for an active shift or clear the check box. This is validated in the function ‘Manage Work schedule’. On selecting the check box, the maximum duration before which allocation can be performed after the start of shift should be mandatorily entered in hh:mm format.

  2. Change – Select the check box to allow change of shifts within a workplace or clear the check box. This is validated in the function ‘Manage Work schedule’. On selecting the check box, the maximum duration before which change of shifts can be performed after the start of shift should be mandatorily entered in hh:mm format.

  3. Transfer – Select the check box to allow transfer of shifts across a workplace or clear the check box. This is validated in the function ‘Manage Work schedule’. On selecting the check box, the maximum duration before which transfer of shifts can be performed after the start of the shift should be mandatorily entered in hh:mm format.

  4. Swap – Select the check box to allow swapping in a shift or clear the check box. This is validated in the function ‘Manage Work schedule’. On selecting the check box, the maximum duration before which swapping can be performed after the start of shift should be mandatorily entered in hh:mm format.

  5. Overtime – Select the check box to allow overtime for an active shift or clear the check box. This is validated in the function ‘Manage Work schedule’. On selecting the check box, the maximum duration before which overtime can be allotted after the start of shift should be mandatorily entered in hh:mm format.

  6. Leave – Select the check box to allow leave (current date) for an active shift or clear the check box. This is validated in the function ‘Manage Work schedule’. On selecting the check box, the maximum duration before which leave can be recorded after the start of shift should be mandatorily entered in hh:mm format.

  7. Cancel – Select the check box to allow cancellation of shift for an active shift or clear the check box. This is validated in the function ‘Manage Work schedule’. On selecting the check box, the maximum duration before which an allocated shift can be cancelled after the start of shift should be mandatorily entered in hh:mm format.
