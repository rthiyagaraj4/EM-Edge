Allocate

To allocate a new shift to a staff/add another shift to an existing shift select [Allocate] from the menu. You can allocate maximum number of shifts that covers the 'maximum continuous working hours' for the staff as entered in the parameter. Allocation of shifts is also based on shift timings.

 

The following details are displayed if a shift is already allocated:

  1. Workplace - The current workplace is displayed if a shift is already allocated.

  2. Shift - All the shifts scheduled to the staff are displayed.

  3. Start Time - The start time and date of the shift is displayed.

  4. End Time - The end time and date of the shift is displayed.

 

Enter the following details to allocate a new shift:

  1. Schedule Date - The scheduled date is displayed.

  2. Already Allocated Shifts - Already allocated shifts of the staff for the day is displayed.

  3. Shift - Select the shift from the list. The list displays only those shifts that were not displayed in the "Already Allocated Shifts" field legend. On selecting the required shift, its shift indicator is displayed as Productive (Working/On-call) or Unproductive in read only text. This is a mandatory field.

  4. Reason - Select a valid reason from the list. The reasons displayed are defined in the "Reason" function for Allocate Shift. This is a mandatory field.

  5. Remarks - Enter the remarks if required.

  6. Click [Apply] to allocate shift. The system prompts you before finalizing the action. Click [OK] to confirm the action.

For empty cells, only Allocate function can be performed.
