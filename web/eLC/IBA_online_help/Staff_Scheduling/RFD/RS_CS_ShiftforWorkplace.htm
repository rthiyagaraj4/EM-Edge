Enter the following details:

 

'By Workplace' Tab

  1. Location Type - Select the Location Type from the list. The listed values are defined in 'Location Type' function of Application Masters module.

  2. Workplace - Enter the Workplace in alphanumeric format of 20 characters maximum length or select using [?] function. This is a mandatory field.

  3. Click [Show Shifts].

  4. Shift - The shift records created in the 'Shift' function in Application Masters module are displayed.

  5. [From - To] - The start and end time of shift is displayed under respective columns.

  6. Shift Indicator - The indicator specified for the shift in 'Shift' function of Application Masters module is displayed. The displayed indicators are Productive and Unproductive.

  7. Productive Flag - The productive flag set for the shift in 'Shift' function of Application Masters module is displayed. The values are Working and On-call. This is displayed only for productive shifts.

 

Applicable to

  1. Wday - A '"' Tick mark is displayed if the shift is applicable for working days, else 'X' cross mark is displayed.

  2. NWDay - A '"' Tick mark is displayed if the shift is applicable for non-working days; else 'X' cross mark is displayed.

  3. Hday - A '"' Tick mark is displayed if the shift is applicable for holidays, else an 'X' cross mark is displayed.

  4. Select - Select the check box to indicate applicability of the shift for the selected workplace. To select all the displayed shifts, select the check box provided in the column header.  

 

'By Shift' tab

  1. Shift - Select the Shift from the list. The listed values are defined in the 'Shift' function of Application Masters module. This is a mandatory field.

  2. Shift indicator - The Shift Indicator is displayed as productive or unproductive based on the selected shift.

  3. Applicability - The Applicability of shifts to working days, non-working days and holidays are displayed.

If the shift is applicable to Working, Non-Working or Holidays, the check box is selected and if the shift is not applicable, the checkbox is cleared.

  1. Shift Timing - The Shift Timing is displayed based on the selected shift. [From - To] corresponds to start and end time of the shift.

  2. Location Type - Select the required Location Type from the list. The list displays all the Location Types created in the 'Location Type' function of Application Masters module.

  3. Click [Show Workplaces].

  4. Location - The selected Location Type is displayed as defined in the 'Location Type' function of Application Masters module.

  5. Workplace - The Workplaces associated to the Locations Types in the 'Workplace' function are displayed.

  6. Select - Select the check box to indicate applicability of the workplace for the selected shift. To select all the displayed workplaces, select the check box provided in the column header.
