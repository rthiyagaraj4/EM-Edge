Swap

To swap two staffs with the same position within or across workplaces, select [Swap] from the menu.

 

The following details are displayed:

  1. Workplace - The current workplace is displayed.

  2. Shift - The shifts scheduled to the staff is displayed.

  3. Start Time - The start time and date of the shift is displayed.

  4. End Time - The end time and date of the shift is displayed.

 

Enter the following details to swap staffs within or across workplaces:

  1. Schedule Date - The scheduled date is displayed.

  2. Workplace - The current workplace of the staff is displayed.

  3. Staff - The name of the staff is displayed.

  4. Allocated Shifts - All the shifts scheduled to the staff is displayed.

  5. Select - Check the radio button to indicate the selection of the required shift. This is a mandatory field.

  6. Click [Swap With].

  7. Workplace - Select the workplace to which the staff has to be swapped from the list. By default the current workplace is displayed. This is a mandatory field.

  8. Staff - Select the staff to be swapped from the list. The list displays staffs that are associated to the selected workplace in the "Staff for Workplace" function. This is a mandatory field. On selecting the staff, the shifts allocated for the selected staff for the selected workplace are displayed.

  9. Allocated Shifts - Already allocated shift is displayed. Click the radio button to select the shift. This is a mandatory field. An alert is displayed if the shift of staff to be swapped is already assigned to the selected staff.

  10. Reason - Select the reason for swapping from the list. The reason is defined in the "Reason" function for Swap Shift. This is a mandatory field.

  11. Remarks - Enter the remarks if required.

  12. Click [Apply] to save the record. The system prompts you before finalizing the action. Click [OK] to confirm the action.
