Staff Roster

From this function you can Query the allocation details of the staff for different workplaces for specified time durations.

Querying for Staff Roster

From RS Queries, select Staff Roster function.

The Staff Roster screen gets displayed.

  1. Select the required Staff using [?] function. The Role Type of the staff is displayed as 'Practitioner' or 'Other Staff' as read only text. This is a mandatory field.

  2. Enter the date range in dd/mm/yyyy format or select using calendar function, for which the staff roster details has to be queried. This is a mandatory field.

  3. Click [Search] to view the Staff Roster details or [Reset] to clear the entered search criteria.

  4. On searching, all the shifts scheduled for the selected staff in various workplaces for the specified time duration is displayed.
