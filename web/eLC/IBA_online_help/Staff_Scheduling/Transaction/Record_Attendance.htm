Record Attendance

From this function, you can Record attendance of staff members.

This function is used to record the attendance (Step-in and Step-out time) of staff members in the workplace. The 'Attendance Feature Required' field should be checked in parameters to access this function.

Recording Attendance

From RS Transactions, select Record Attendance function.

The Record Attendance screen gets displayed.

  1. Select the required workplace using [?] function. The [?] function displays all the workplaces defined in the 'Workplace' function. This is a mandatory field.

  2. Select the required shift from the list. The list displays all the shifts defined in the 'Shift' function of Application Masters module.

  3. Select the required staff using the [?] function. The Role type of the staff is displayed as read only text.

  4. Click [Show Staff].

  5. Enter the step-in time of the staff for the shift displayed in hh:mi format. Hours (hh) should be entered as a number between 0-23 and Minutes (mi) should be a number between 0-59. The step-in time should be entered within the time specified in 'Maximum duration from start of shift to mark attendance' field of parameters from the start of shift. The system does not permit to record the step-in time after the time limit has crossed as set in the parameters.

  6. Enter the step-out time of the staff for the shift displayed in hh:mi format. Hours (hh) should be entered as a number between 0-23 and Minutes (mi) should be a number between 0-59.

  7. Select the check box to indicate the selection of the staff for the entered timing.

  8. Click [Apply] to save recording attendance.
