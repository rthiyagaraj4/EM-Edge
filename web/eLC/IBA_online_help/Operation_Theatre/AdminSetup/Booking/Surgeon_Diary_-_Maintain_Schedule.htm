Surgeon Diary - Maintain Schedule

From this function, you can Maintain a schedule for the surgeon for a specific time period.

The Surgeon Diary - Maintain Schedule function displays the surgeon's schedule for the selected period generated for the surgeon in the 'Surgeons Diary - Generate Schedule' function.

Maintaining Surgeon Schedule

From the OT Admin Setup > Booking, select the Surgeon Diary - Maintain Schedule function.

The Surgeon Diary - Maintain Schedule screen gets displayed.

  1. Select the Surgeon using the [?] function. This field is mandatory.

  2. Select the date range to maintain schedule.

  3. Click [Search] to view the schedule generated for the surgeon.

  4. Select the check box under Delete to release the schedule.

  5. Click [Apply] to delete the selected schedule.
