Stores for Operating Room

From this function you can Associate Inventory Stores to the Operating Room and Query for associated records.

You can generate a request for items required for an operation in a theatre and the request from the store should be considered from the list of stores associated to the Operating Theatre.

Associating Inventory Stores to Operating Theatre

From the OT Admin Setup > Room Setup, select the Stores for Operating Room function.

The Stores for Operating Room screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Operating Room or select the Operating Room using the [?] function. The listed values are defined in the Operating Room function. This is a mandatory field. On selecting the Operating room, the Inventory Stores which are created across the entity in  ST Code Setup are listed.

  3. Select the Facility Name from the list. The listed values are defined in the Facility function of 'System Manager' module. The value 'All' is selected, by default. On selecting a specific Facility Name, the stores associated to the selected Facility are filtered.

  4. The Facility Name and list of Stores associated to the selected Facility are displayed.

  5. Select the 'Type' of the Store from the list. The listed values are 'Theatre Requesting/Consumption Store' and 'Theatre Issuing Store'. This field is disabled if the Type is already selected for the Store.

  6. Select the 'Select' check box to associate the corresponding Store to the Operating Room.

  7. Select the 'Default Store' check box to default the selected Store for the Operating Room. You can associate only two default stores to a Operating Room.

  8. Click [Apply] to save the association of stores to Operating Room or click [Reset] to clear the entered data.

  9. Click [Query] to search for an existing association of stores to Operating Room records.
