Record Surgeon Notes

Record Surgeon Notes function is used to record the operation notes and to enter a textual description of the diagnosis. The function captures all details of Operation/Procedure.

 

  1. The Order ID, Date and Time of Booking, Surgeon, Specialty, Theatre, Booking Number, Date and Time of the surgery, Accession Number and current date are displayed in the Patient Encounter Banner.

  1. The Anesthesia Type, Anesthetist, Theatre name are displayed.

  2. Select Documentation Level as Specialty or Across Specialties.

  3. Click Pre-Operative Diagnosis to view Pre-Operative details recorded for the patient.

  4. Click Post-Operative Diagnosis to record diagnosis details.

  5. Click Patient History to view patient history.

The patient has to 'Check-In to OR' to perform this function in OT Slate.

 

Operation Details

Operations/Procedures Tab

This tab allows adding operations and removing existing operations in case they are not performed.

  1. Operation - Select the Operation using [?] function. This is a mandatory field.

  2. Description – The description of the Operation is displayed on selecting the operation.

  3. Side Applicable - If Side Applicable field for the Operation is selected in the Operations function, this field is enabled. Select the applicable side from the list. This is a mandatory field.

  4. Remarks - Enter remarks if required.

 

Click [Select] to add the Operation to the screen. Select the Operation Code hyperlink to edit the remarks or click [Cancel] to exclude the operation.

The operation code and description, side applicable, remarks, total charge and the status are displayed when the operation is complete. The total payable amount, patient payable amount and patient paid amount are displayed. If the service is under policy coverage a is displayed. If the service is not covered by the policy a is displayed. Similarly, if the Service requires an approval, it is indicated using symbol. If the service does not requires an approval, symbol is used to indicate the same. Click symbol to edit/enter the charge details. You can include/exclude the service from policy coverage.

These details are displayed in the 'Surgical Accessories' tab when the accessories are recorded.  The equipment, consumable, prosthesis/implant, instrument/CSSD packs, Operation, Implant, Actual Units, Reserved Picklist Units, Requested PickList Units, Remarks, Consumption Store and charge details are displayed. The total payable amount, patient payable amount and patient paid amount are displayed. If the service is under policy coverage a is displayed. If the service is not covered by the policy a is displayed. Similarly, if the Service requires an approval, it is indicated using symbol. If the service does not requires an approval, symbol is used to indicate the same. Click symbol to edit/enter the charge details. You can include/exclude the service from policy coverage.

You cannot enter/edit the charges for Prosthesis/Implants. You can only view the details.

Personnel Details

The Personnel Details tab displays two tabs: Hospital Personnel and Visiting Personnel.

Hospital Personnel

Record the following details:

  1. Specialty - Select the Specialty from the list. The Specialties available in the list are based on Documentation Level field. If Documentation Level is selected as Across Specialties, this field is displayed as All Specialties.

  2. Role - Select the Role from the list. This is a mandatory field.

  3. Name - Select the name of the Role. The Practitioners listed have the Practitioner Types associated to the Role selected.

 

Click [Select] to add Hospital Personnel details to the screen. Select Specialty hyperlink and click [Cancel] to exclude the hospital personnel. Click the ‘View Staff Assignment’ hyperlink to view the staff assignment screen.

 

The Surgeon, Assistant Surgeon 1, Assistant Surgeon 2 and Anaesthetist defined in the Check-In are displayed by default under the Personal Details tab.

Visiting Personnel

Record the following details:

  1. Specialty - Select the specialty from the list. The specialties listed are displayed based on Documentation Level field. If Documentation Level is selected as Across Specialties, this field is displayed as All Specialties.

  2. Visiting Personnel - Enter the details of the visiting personnel in the free text area.

 

Click [Select] to add Visiting Personal details to the screen. Select Specialty hyperlink and click [Cancel] to exclude the visiting personnel from recording.

Surgeon Notes

This tab is used to enter the Surgeon Notes for the Operation. This note is maintained by ‘All Specialties’ or a specific Specialty based on the selected Documentation Level field.

Note Type screen is displayed to select the required Notes. Select the required Note Type from the list to record Surgeon Notes. The Note Types that was associated for Surgeon Notes in the ‘Note Type for Specialties’ function is displayed for selection. Click [OK] and record the notes or click [Cancel] to record proceed with entering details on the screen.

The Surgeon Notes tab displays two tabs:

Notes

  • Click [Surgeon Notes] to select the required Note Type to enter Surgeon Notes. The Note Types that was associated for Recovery Notes in the ‘Note Type for Specialties’ function is displayed for selection.

Wound Details

Record the following details:

  1. Specialty – The Specialty is displayed based on the selected Documentation Level.

  2. Wound Type - Select the Wound Type using [?] function.

  3. Click [Select] to add Wound Details to the screen. Select Specialty hyperlink to edit the Wound Type or click [Cancel] to exclude the Wound Details from recording.

  4. Click Wound Type hyperlink to view the remarks recorded for the selected Wound Type.

Time and Details

This tab is used to record the Operation Timings. These details can be recorded specialty wise or for all Specialties, depending on documentation level selected for the operation. Preparation Time, Anaesthesia procedure timings, Anaesthesia timings and operation timings are accepted in this tab.

Procedure Coding

 

When you select a Speciality to which only one Note-type is mapped, the Surgeon Notes screen is automatically displayed. The surgeon, assistant surgeon and anesthetist details are displayed only if the Display Hospital Personnel Assigned During Booking/Check-In check box in setup is selected.
