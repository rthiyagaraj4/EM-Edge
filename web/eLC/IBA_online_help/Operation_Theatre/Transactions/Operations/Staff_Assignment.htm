Staff Assignment

From this function you can Assign and Modify the Staff to the specific OT room. You can Query the details of staff assigned.

The Staff Assignment function is used to assign the staffs to the specific operation theatre. You can modify the assigned staff details.

Assigning Staffs

From the Operations, select the Staff Assignment function.

The Staff Assignment screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Date using the calendar function.

  3. Select the room name from the drop-down list. The staff details are displayed for the selected date.

  4. Click [Add] to display a blank row.

  5. Enter the details in the fields. <Related Field Description>

  6. Click [Save] to save the staff details

  7. Click [Apply] to save the staff assignment details or [Reset] to clear the entered data.

  8. Click [Query] to display the Modify staff assignment screen.

    Enter the following details:

     

    1. Date [From-To] – Select the date using the calendar function.

    2. Room – select the room from the drop-down list. The staff assignment details are displayed for selected date range.

    3. Role – Select the Role from the drop-down list. The staff roles defined in the Application Masters are listed.

    4. Name – Select the staff name from the drop-down list. The staff names defined in the Application Masters are listed.

    5. Show in widget – Select this check box to show the staff details in the dashboard. This check box is selected by default.

    6. From time – Select the From time the drop-down list.

    7. To time – Select the To time from the drop-down list.

    8. Delete – Click the to delete the selected record.

  9. Click [Audit Trail] to display Audit history screen with history of staff assignment. You cannot edit the data.

 

If you select ‘To date’ is less than ‘From date’, the system displays error message 'From Date' should be less than 'To Date'. If you select a past date in Date field in Create Staff Assignment screen, the system displays ‘Select Current Date or Future Date’.

If the duration between the ‘From date’ and ‘To date’ is greater than 1 month in Modify Staff Assignment screen, the system displays error message ‘Date duration cannot exceed one month’.

If no staffs are assigned and you click save button, the system displays error message ‘Staff not Assigned’.
