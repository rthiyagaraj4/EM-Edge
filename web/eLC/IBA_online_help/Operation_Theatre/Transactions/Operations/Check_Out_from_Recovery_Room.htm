Check Out from Recovery Room

Check Out from Recovery Room function is used to check out patients from Recovery Room.

 

The following details are displayed:

  1. Anesthesia Type – The Type of Anesthesia is displayed as recorded.

  2. Anesthetist – The Anesthetist is displayed as recorded.

  3. Theatre – The Theatre is displayed.

  4. Click Pre-operative clinical comments to view Pre-Operative details recorded for the patient.

  5. Click Verify Checkout Checklist hyperlink to verify check out checklist.

 

Operations/Procedures

  1. The Operation Code, Description, Side Applicable and Remarks are displayed. Click the Pre-operative comments hyperlink to view the remarks recorded for the Operation/Procedure.

 

Check Out from Recovery Room

  1. Recovery Start Time - The system displays the Recovery Start Time as recorded. Recovery End Time - Enter end date and time of recovery. This is a mandatory field.

  2. Check-out - Select the check box to indicate that the patient has checked out from the recovery room.

  3. Check-Out Time – The current time is displayed as the check out time.

  4. Transfer To - Select a transfer location from the list as to where the patient is going to be transferred. The listed values are Nursing Unit, Clinic and Mortuary. This is a mandatory field.

  5. Nursing/Clinic - If Nursing Unit/Clinic is selected in the 'Transfer To' field, then select the Nursing Units/clinics to where the patient is being transferred. The Nursing/Clinic field is disabled if 'Mortuary' is selected in the 'Transfer To' field. This is a mandatory field.

  6. Transfer Time - Enter the date and time of transfer. The current system date and time is displayed, by displayed. This is a mandatory field.

  7. Status - This field is enabled if Clinic/Nursing Unit is selected in the 'Transfer to' field. Select the required Status from the list. The listed values are Intubated and Extubated.

 

Specimen Details Tab

  1. Specimen No./Dtl - Enter the Specimen Number or Details. This is a mandatory field.

  2. Department - Select the Department using [?] function. This is a mandatory field.

  3. Quantity - Enter the Quantity of specimen collected.

  4. Click [Select] to add the specimen details to screen. Select the Specimen No./Dtl hyperlink to modify the quantity or click [Cancel] to cancel recording specimen details.

  5. Click [Apply] to save checking out of Recovery Room.
