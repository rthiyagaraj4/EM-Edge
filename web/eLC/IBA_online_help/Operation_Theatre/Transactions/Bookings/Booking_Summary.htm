Booking Summary

From this function you can View the booking summary report based on the entered criteria.  

The Booking Summary function summarizes the theatre-wise bookings for a given date.

Viewing Booking Summary

From the Bookings, select the Booking Summary function.

The Booking Summary screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run] to get the Report Options screen or [Reset] to clear the entered details.

  3. Select the required report option and click [OK] to view the report, or [Cancel] to abort the process.
