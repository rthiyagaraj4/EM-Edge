Cancel Booking

Cancel Booking function is used to cancel a booking for a patient.

  1. The Surgeon, Specialty, Theatre, Date and Time of the surgery are displayed.

  2. The following details are displayed as recorded while creating a booking - Theatre, Specialty, Assistant Surgeon 1, Assistant Surgeon 2, Anesthetist, Anaesthesia, Patient Position and Source.

  3. Click the Remarks hyperlink to view the remarks recorded while booking.

  4. Click Pre-operative clinical comments to view the pre-operative diagnosis details recorded while booking.

The  icon appears beside the Source field if the long name for the External Referral Source is recorded, in ‘External/Referral Source’ function of Application Masters module. Place the mouse on the  icon to display the long name of the External or Referral Source recorded in ‘External/Referral Source’ function of Application Masters module.

 

Operations/Procedures

  1. The Operation Code, Description, Side Applicable, Duration and Remarks recorded for the Operation/Procedure is displayed. Click the Pre-operative comments hyperlink to view the Operation/Procedure remarks recorded for the Patient.  

 

Cancellation Details

  1. Cancelled By - Select the user cancelling the booking.

  2. Date/Time - The current date and time is displayed as the date and time for cancelling booking.

  3. Reason - Select the Reason using the [?] function. The Reasons for Cancel Booking are defined in the Reason function. This is a mandatory field.

  4. Cancel Order -  This field is initially disabled. This indicates that the booking for the operations is cancelled but the corresponding order cannot be cancelled. Select the check box to cancel the order. This check box is enabled if the Order is placed and booked from Clinician Access module and is disabled if the Order is directly booked from the Booking function in Operation Theatre module.

  5. Informed Ward - Select the check box to indicate that the ward was informed of the cancelled booking.

  6. Informed Ward Date - The current system date and time is displayed as the date and time of informing the ward of the cancelled booking.

  7. Cancel Remarks - Enter the remarks for cancelling booking. This is a mandatory field.

  8. Click [Apply] to save cancelling booking of the operation/procedure.
