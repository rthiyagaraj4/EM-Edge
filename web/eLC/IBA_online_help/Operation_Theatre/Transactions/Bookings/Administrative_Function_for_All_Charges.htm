Administrative Function for All Charges

Administrative Function for All Charges function is used to record the surgical accessories consumed for the Surgery or Patient for scheduled and direct Check-In cases. When the items are finalized in this function, the stock reduces from the Operation Theatre store.

 

Operations/Procedures

You can view the Operations or Procedure charges at the time of finalization.

Click  to view the operation and procedure details.

 

The following details are displayed:

  1. Operation Code – The operation code is displayed.

  2. Description – The description of the operation is displayed.

  3. Side Applicable – The name of the side applicable is displayed.

  4. Remarks – Click ‘Remarks’ hyperlink to view the Operation/Procedure remarks.

  5. Charge - The charges of the operations is displayed. Click   to enter and edit the charges details.

  6. Status – The status of the operation is displayed.

Direct Consumption

You can record and view the details of Consumables used on a patient.

 

Click  to display Consumables Used and View Consumables Used hyperlinks.

 

The following details are displayed:

  1. Consumables Used – Click ‘Consumables Used’ hyperlink to enter the consumables used in the operation.

  2. View Consumables Used – Click ‘View Consumables Used’ hyperlink to view the consumables used in the operation.

Surgical Accessories

Click  against the Surgical Accessories to view the Surgical Accessories details.

 

The following details are displayed:

  1. Accessory Type – The name of the accessory time is displayed and disabled as per the setup in Setup Parameter for Facility function.

  2. Implant – Select the Implant using [?] function. The listed values are created in Prosthesis/Implant function. This is a mandatory field.

  3. Operation – The operation selected in ‘Record Surgeon Notes’ is selected, by default.

  4. Consumption Store – Select the consumption store using [?] function. This is a mandatory field.

  5. Reserved Picklist Units – The number of reserved picklist units are displayed.

  6. Requested Picklist Units – The number of requested picklist units are displayed.

  7. Stock Available – The number of stock available is displayed.

  8. Location - The location code defined in the ‘Instruments/CSSD Packs’ screen in the Code Setup is displayed based on the instrument selected.

  9. Actual Units – Enter the number of actual units in numeric format of three digits maximum length. This is a mandatory field.

  10. Remarks – Click ‘Remarks’ to enter the remarks for surgical accessories in the free text.

Prosthesis/Implant

Click  against the Prothesis/Implant to view the Prothesis/Implant details.

 

The following details are displayed:

  1. Operation – The name of the operation is displayed and hyperlinked. Click the operation name hyperlink to the details of the operation in the Surgical Accessories.

  2. Implant – The name of the Implant is displayed.

  3. Actual Units – The number of actual units are displayed.

  4. Reserved Picklist Units – The number of reserved picklist units are displayed.

  5. Requested Picklist Units – The number of requested picklist units are displayed.

  6. Remarks – Click ‘Remarks’ to view the remarks of the operations.

  7. Consumption Store – The name of the consumption store is displayed.

  8. Charge – The charges of the operation is displayed. Click to enter and edit the charges details.

Consumption Packs

Click  against the Consumption Packs to view the Consumption Packs details.

The following details are displayed:

  1. Operation – The name of the operation is displayed and hyperlinked. Click the operation name hyperlink to the details of the operation in the Surgical Accessories.

  2. Package – The name of the package is displayed.

  3. Actual Units – The number of units are displayed.

  4. Remarks – Click ‘Remarks’ to view the remarks of the operations.

  5. Charge – The charges of the operation is displayed. Click   to enter and edit the charges details.

Equipment

Click    to view the Equipment details.

The following details are displayed:

  1. Click the icon to display the equipment image in an ‘Equipment Image’ dialog. If multiple equipments are used for the surgery/procedure, on clicking the icon , the list of equipments with their Item Code, Item Description and Image Description are displayed in the ‘Equipment’ dialog. Click the Image Description icon to view the image in a new dialog. The image display is based on the OT Code Setup.

  2. Operation – The name of the operation is displayed and hyperlinked. Click the operation name hyperlink to the details of the operation in the Surgical Accessories.

  3. Package – The name of the package is displayed.

  4. Date & Time – Select the Date and Time using the calendar function. The surgical accessory details will be displayed based on the selected date. By default, the current date is displayed.

The following error messages are displayed when the Select button is clicked based on the below mentioned conditions:

If the Date & Time is less than Check-In Date/Time:

‘Date & Time Cannot be Before the Check-In Date/Time <<dd/mm/yyyy hh:mm>>’.

If the Date & Time is greater than One Week from the Current Date:

‘Date & Time Should be Within One Week from Current Date’

  1. Actual Units – The number of units are displayed.

  2. Equipment ID – The name of the equipment id is displayed.

  3. Remarks – Click ‘Remarks’ to view the remarks of the operations.

  4. Charge – The charges of the operation is displayed. Click to enter and edit the charges details.

Instruments/CSSD Packs

Click to view the Instruments/CSSD Pack details.

 

The following details are displayed:

  1. Click the icon to display the instrument image with location code in the ‘Instrument Image’ dialog, If multiple instruments/CSSD Packs are used for the surgery/procedure, when you click this icon, the list of instruments/CSSD Packs with their Item Code, Item Description, Image Description and Location Code will be displayed in the ‘Instrument’ dialog. Click the  icon to view the image with location code in a new dialog. The image display is based on the OT Code Setup.

  2. Operation – The name of the operation is displayed and hyperlinked. Click the operation name hyperlink to the details of the operation in the Surgical Accessories.

  3. Instrument – The name of the instrument is displayed.

  4. Actual Units – The number of units are displayed.

  5. Location - The location code defined in the ‘Instruments/CSSD Packs’ screen in the Code Setup is displayed.

  6. Tray No – The number of the tray is displayed.

  7. Remarks – Click ‘Remarks’ to view the remarks of the operations.

  8. Charge – The charges of the operation is displayed. Click to enter and edit the charges details.

Operation Room Usage

Click  to view the Operation Room Usage details.

The following details are displayed:

  1. Check-in to OR Date/Time – The check-in date and time of the patient in OR is displayed and disabled.

  2. Check-out from OR Date/Time – The check-out date and time of the patient from OR is displayed and disabled.

  3. Chargeable Units – The number of chargeable units are displayed. This is a mandatory field.

  4. Completed – Select the check box to complete the operation room usage. This check box is disabled as per setup in Setup Parameter for Facility function.

Completion Status

The following details are displayed.

  1. Prosthesis/Implants Completed – Select the check box to complete the Prosthesis/Implant. This check is disabled as per the setup in Setup Parameter for Facility function.

  2. Consumables Completed – Select the check box to complete the Consumables. This check is disabled as per the setup in Setup Parameter for Facility.

  3. Equipment Completed – Select the check box to complete the equipment. This check is disabled as per the setup in Setup Parameter for Facility.

  4. Instruments/CSSD Completed - Select the check box to complete the Instrument/CSSD. This check is disabled as per the setup in Setup Parameter for Facility.
