Turnaround Time

From this function you can Generate a Turnaround Time report based on procedure, specialty and surgeon.

The Turnaround Time function is used to generate the Turnaround Time reports based on procedure, specialty and surgeon. This function is enabled only when the Turnaround Time is defined in the Setup Parameters for Facility. The Turnaround Time by Speciality refers to average time taken by surgical procedure within the same speciality in definite time interval. The Turnaround Time by procedure refers to average time taken by surgical procedure indifferent to the speciality or Practitioner/Surgeon in definite time interval. The Turnaround Time by Surgeon refers to average time taken by Practitioner/Surgeon in definite time interval.

Generating a Turnaround Time report

From the Reports> Operation Lists, select the Turnaround Time function.

The Turnaround Time screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run] to generate the report based on the entered report criteria or [Reset] to clear the entered data.

  3. Select ‘Report Option’ to generate the report.

  4. Click [OK] to generate the report and spool it in the file on the print server or [Cancel] to cancel the action.

If the Turnaround Time is not defined at the Setup Parameter for Facility, the system invokes an error message ‘Please set the parameter for Turn around time report’.
