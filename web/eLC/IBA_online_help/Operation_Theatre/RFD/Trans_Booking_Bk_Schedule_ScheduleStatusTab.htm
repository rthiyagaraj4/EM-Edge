Schedule Status Tab

This tab displays the schedule for the next 30 days from the current date for the Specialty/Surgeon/Procedure combination, if generated.

  1. Select required Specialty, Surgeon or Procedure and click [Search] to display the schedule status for the next 30 days from the current date or enter the query criteria to search for a specific schedule.

    1. Specialty - Select the Specialty using [?] function. The Specialties are defined in the 'Specialties' function in Application Masters module.

    2. Surgeon - Select the Surgeon using the [?] function. If Specialty is selected, surgeons associated to the Specialty are displayed for selection. The Practitioners are defined in the 'Practitioner' function of the Application Masters module.

    3. Procedure - Select the Procedure using the [?] function. The listed values are defined in the 'Operations/Procedures' function.

    4. First Available Date - Select the 'First Available Date' check box to view the first available schedule.

    5. Preferred Date - Enter the Preferred Date from which the Schedule Status has to be displayed. The Current Date is displayed, by default.

    6. Preferred Month - Enter the Preferred Month from which the Schedule Status has to be displayed.

    7. Theatre - Select the Theatre using the [?] function. The Theatres are created in the Operating Room function.

    1. Surgeon - The Surgeon for which schedule is generated is displayed against the Theatre.

    2. Theatre - The Theatre for which schedule exists for the surgeon is displayed.

    The status for each day has a distinct color to indicate the schedule status. The color represents:

    •  Schedule - This color indicates that booking is allowed.

    • Fully Booked - This color indicates that the bookings are full and no further bookings are allowed.

    •  Fully Booked (Over Booking Allowed) - This color indicates that overbooking are allowed.

    • Holiday - This color indicates holiday against the displayed date.

    • Unschedule - This indicates that schedule is not generated or defined for the Surgeon/Theatre combination.

    • Booking Authorization Required – This color indicates that the slots are to be authorized.

  2. Click [External Facility] to display the External Facility Booking dialog.

    1. Facility – Select the required facility from the drop-down list. All the active facilities get listed. The logged-in facility is selected by default.

    Click [OK] to display the selected facility as performing location or click [Cancel] to close the dialog.

  3. Select the check box(es) against the theatre to display the day schedule(s) in the Day Schedule Status tab.

By default, first three check boxes are selected. You can modify the check box selection as required. You can select only a maximum of check boxes. The operation schedules for the selected theatre are displayed based on the availability of the surgeons. You can modify the dates as required.

  1. Click to view the schedule for the next period. Click to view the schedule for the previous period.

  2. The schedule for each day shows a distinct color to indicate the schedule status.

  3. Click ‘O’ hyperlink in orange to invoke the Booking List dialog with a message ‘Do you want to continue with overbooking?’. This message is displayed only when you have overbooking rights.

  4. Click [Yes] to start another Overbooking or click [No] you will be redirected to the Schedule Status screen.

  5. Select the corresponding day for which booking is required by clicking the slot hyperlink for a particular date and the control shifts to the Day Schedule Tab.

An alert message ‘This booking utilises reserved slots. Only Authorised users may complete the booking. Do you want to proceed?’ is displayed if the slots allowed for surgeon for surgery defined in the ‘Diary Schedule for Surgeon’ exceeds. Click [Yes] to invoke the Authorize – Slot booking dialog.

  1. User Name -. Select the User Name from the drop-down list. The authorized users defined in the OT Admin Setup are listed.

  2. Password – Enter the Password in alphanumeric format.

  3. Click [OK] to validate the username and password or Click [Cancel] to reset the details entered.

 

  1. Click [Clear] to clear the details displayed.

 

When a theatre is fully blocked, the schedule will be displayed in red background without any count. When a theatre is partially blocked, the schedule will be displayed in red border. This will be applicable for statuses ‘Schedule’, ‘Fully Booked’, ‘Fully Booked (Over Booking Allowed)’. This is based on the ‘Include Blocked Slot in Available Slot Count’ check box selected in Setup Parameters for Facility screen.
