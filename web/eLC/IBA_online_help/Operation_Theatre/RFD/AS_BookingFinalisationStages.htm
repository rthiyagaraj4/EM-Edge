Enter the following details:

 

  1. Stage Code - Enter the Stage Code in alphanumeric format of two characters maximum length. This field is mandatory.

  2. Pre Stage Description - Enter the Pre Stage Description in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Post Stage Description - Enter the Post Stage Description in alphanumeric format of 30 characters maximum length. This field is mandatory.

  4. Serial No. -  Enter the Serial No. for the stage. The maximum length for this field is two characters. This field is mandatory.

  5. Color - Select the Color from the list for the entered Stage. This color displays for the stage in the Transactions.

  6. Text Color - Select the Text Color as Black or White from the list.

  7. Specific Indicator - Select a value from the list. The listed values are All Users and Specific Users.

  8. Applicable Position - This field is enabled only if Specific Indicator is selected as 'Specific Users'. Select the Position using the [?] function.

  9. Action if not Done - Select the action if the stage is not done from the list. The listed values are Alert and Stop. If you select 'Alert', the system gives a warning message indicating that the action is not done, but you can proceed with the booking. If you select 'Stop', the system does not allow you to proceed with the booking.

  10. Pre-Anesthesia Evaluation Stage - Select the check box to indicate that the stage is considered as Pre- Anesthesia Evaluation Stage.

 

Users

  1. User Name - Select the User using the [?] function. This field is enabled if Specific Indicator is selected as Specific Users. This is a mandatory field.

  2. User ID - The User ID of the selected User is displayed.
