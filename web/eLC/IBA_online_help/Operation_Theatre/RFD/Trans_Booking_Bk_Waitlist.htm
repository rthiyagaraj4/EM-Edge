Waitlist Tab

This tab displays orders that were waitlisted while placing the order in the Clinician Access module. The tab also displays orders that were waitlisted in the Pending Orders Tab by selecting the Waitlist option.

  1. Click [Search] to display patients to book appointment or enter the query criteria to select a specific patient to book appointment.

    1. Specialty - Select the Specialty using the [?] function.

    2. Surgeon - Select the Surgeon using the [?] function.

    3. Patient ID - Select the Patient ID using [?] function.

    4. Name - The name of the selected Patient is displayed.

    5. Order ID - Select the required Order ID using the [?] function. Order ID is generated when an Order is placed through the Clinical Care Management module.

    6. Status - Select the status of the booking from the list. The listed values are All, Booked Cases or Unscheduled Cases.

    7. Preferred Surgery Date [From-To] - Enter the Preferred Surgery Date range for which waitlist records have to be displayed. The current date is displayed as the From Date, by default.

    8. Priority – Select the priority from the drop down list. The priority values defined in the masters are displayed in the drop sown.

    9. Category Type - Select the Category Type from the drop down list. The category type values defined in the masters are displayed in the drop sown.

  2. On searching the screen displays the Patient ID, Patient Name, Surgery Type, Procedure, Specialty, Surgeon, Preferred Date, Waitlist Date, Waitlist Days, Status, Pre-operative clinical comments, Priority and Category Type.

  3. Select the Patient ID hyperlink to Book Appointment or Cancel the Waitlist by selecting the required option. Click Book Appointment and the Schedule Status Tab is displayed for booking. Select Cancel Waitlist and Cancel Waitlist screen is displayed to enter the reason to cancel the waitlist. Select the Reason using the [?] function. The Reasons are created in the Reason function. Click [OK] to cancel waitlist. Click [Cancel] to abort cancelling waitlist. The cancelled order is displayed in the Pending Orders tab.
