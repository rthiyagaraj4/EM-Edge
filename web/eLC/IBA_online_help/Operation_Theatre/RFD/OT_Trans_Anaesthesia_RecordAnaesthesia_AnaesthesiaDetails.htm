Anaesthesia Details Tab

On Pre-Anaesthesia Evaluation of the anaesthesia pending orders, the orders are displayed in this tab.

  1. Click [Search] to display the Operations/Procedures or enter the search criteria and click [Search] to select a specific record. Click [Clear] to reset the records displayed in the result area.

    Enter the following search criteria:

    1. Surgery Date [From-To] – Select the surgery date using the calendar function. The current Date is displayed, by default

    2. Anesthesia Ordered Date [From-To] - Select the Anesthesia Ordered Date using the calendar function.

    3. Patient ID – Enter the Patient ID in alphanumerical format of ten characters maximum length or select it using [ ?] function.

    4. Patient Name - Corresponding to the patient identification number selected, the name of the patient is displayed in this field, by default.

    5. Specialty – Select the specialty using [?] function. The listed values are created in Application Masters module.

    6. Surgeon - Select the Surgeon using [?] function. The listed values are created in Application Masters module.

  2. Click the Patient ID hyperlink and a set of options are displayed. Select the required option to view/modify the attributes. The options are:

Patient Demographics

Patient History

Surgery History

Anaesthesia History

Pre-Anaesthesia Evaluation

Re-evaluation

Record Intra Anaesthesia Details

Record Post Anaesthesia Details

  1. Click [Apply] to record the anaesthesia details or [Reset] to clear the entered details.
