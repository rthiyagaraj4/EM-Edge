Enter the following details:

 

  1. Notice Date - Enter the Booking Date for which the report has to be generated. This is a mandatory field. The current date is displayed, by default.

  2. Section Time [From-To] - There are three sections namely, Section 1, Section 2 and Section 3. The From and To time is populated from the 'Setup Parameters for Facility' function for the logged in Facility. You cannot make any changes to the time.

  3. Operating Room - Select the operation room for which the report has to be generated using the [?] function.

  4. Specialty - Select the Specialty for which the report has to be generated using [?] function.

  5. Surgeon - Select the Surgeon for which the report has to be generated using [?] function.

  6. Nursing Unit - Select the Nursing Unit for which the report has to be generated using [?] function.

  7. Assistant Surgeon 1 - Select Assistant Surgeon 1 using [?] function.

  8. Assistant Surgeon 2 - Select Assistant Surgeon 2 using [?] function.

  9. Surgery Type - Select the required Surgery Type from the list. The Surgery Types are created in the Surgery Type function.

  10. Patient Class - Select the Patient Class from the list. The listed values Daycare, Emergency, Inpatient, Outpatient and External are system defined.

  11. Print - Select the Print option from the list. The listed values are Finalised Bookings, Non-Finalised Bookings and All Bookings.
