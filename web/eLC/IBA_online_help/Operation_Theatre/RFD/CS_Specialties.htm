Enter the following details:

  1. Specialty - Select the Specialty using the [?] function. This is a mandatory field. The listed Specialties are defined in the Application Masters module.

  2. Diagnosis Coding Scheme - Select the coding scheme applicable to the Specialty as User Defined, CPT 4, ICD9-CM or ICD-10 PCS. The values are listed from the Medical Records module.

  3. Chart ID - Select the Chart ID using the [?] function. The Chart IDs are created in the Chart Setup function in the Clinician Access module.

  4. Swab & Instrument Count - Select the Swab and Instrument count using the [?] function to associate to the Specialty. The Swab and Instrument Counts are created in the 'Swab Count Template' function.

  5. Medical Service Group - Select the Medical Service Group using the [?] function. The Medical Service Groups are created in the Inpatient Management module.

  6. Anesthesia Review Required - Select the check box if Anesthesia Review is required. On selecting the check box, a Pre-Anesthesia Evaluation is required for all the operations under this Specialty.

  7. Enabled - Select the check box to activate the record.

 

Pre-operative Checklist

  1. Checklist - Select the Pre-Operative Checklist using the [?] function. The Checklists are created in the Check List function.

  2. No. of Verifications - Select the number of verifications to be done for the specialty from the list. You can select upto four verifications. This field is mandatory on selecting a checklist.

  3. Role 1/2/3/4 - The Role fields are enabled based on the selected 'No of Verification'. If 1 is selected as the 'Number of Verifications' then Role1 is enabled.

 

Post-Operative Checklist

  1. Checklist - Select the Post-Operative Checklist using the [?] function. The Checklists are created in the Check List function.

  2. No. of Verifications - Select the number of verifications to be done for the specialty from the list. You can select upto four verifications. This field is mandatory on selecting a checklist.

  3. Role 1/2/3/4 - The Role fields are enabled based on the selected 'No of Verification'. If 1 is selected as the 'Number of Verifications', then Role1 is enabled.
