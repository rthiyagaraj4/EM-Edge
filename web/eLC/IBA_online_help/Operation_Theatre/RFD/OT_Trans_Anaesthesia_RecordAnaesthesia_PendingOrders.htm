Pending Orders

The pending orders for the Operating Theatre available for booking are displayed in this tab.

  1. Surgery Date [From-To] – Select the surgery date using the calendar function. By default, the current date is displayed.

  2. Anesthesia Ordered Date [From-To] - Select the Anesthesia Ordered Date using the calendar function. By default, this field is blank.

 The error message ‘Atleast one field should be entered’ will be displayed when Search button is clicking without entering any search criteria.

  1. Enter the Anesthesia Ordered Date in the format of dd/mm/yyyy or select the date using the Calendar function. The current system date is displayed by default in Anesthesia Ordered Date field. This is a mandatory field.

  2. Click [Search] to display the pending orders or enter the search criteria and click [Search] to select a specific record. Click [Clear] to reset the records displayed in the result area.

    Enter the following search criteria:

    1. Surgeon – Select the Surgeon using [?] function. The listed values are created in Application Masters module.

    2. Patient ID – Select the Patient Identification number using [?] function.

    3. Patient Name – Corresponding to the patient identification number selected, the name of the patient is displayed in this field, by default.

    4. Specialty - Select the Specialty using [?] function. The listed values are created in Application Masters module.

  3. Click the Patient ID hyperlink to display the set of options. The options are:

Patient Demographics

Patient History

Surgery History

Anaesthesia History

Pre-Anaesthesia Evaluation  

In the Pending Orders tab, you may or may not select other options, but you have to select the Pre Anaesthesia Evaluation option mandatorily and enter the pre anaesthesia details for the order to appear in the Anaesthesia Details tab.
