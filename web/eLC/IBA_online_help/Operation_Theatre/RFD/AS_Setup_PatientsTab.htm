Patients Tab

This tab is used to set the automatic reminder similar to appointment slip to be sent to the patients via SMS before the appointment date. The SMS reminder contains the Patient Name, Patient ID, Procedure Name, Day, Date and Time, Booking Number, Doctor’s name and Enquiry Number of Hospital.

  1. Reminder Notification via SMS - Select the check box to send reminder via SMS to the patients about admission.

  2. Days Before Appointment Date – Enter the number of days in three digit maximum length based on which automatic reminder similar to appointment slip is to be sent to the patients via SMS before appointment date. The numeric value range is from 001 to 999. This field becomes mandatory only when the ‘Reminder Notification via SMS’ check box is selected.

  3. Click [Apply] to set the parameter or [Reset] to clear the entered data.
