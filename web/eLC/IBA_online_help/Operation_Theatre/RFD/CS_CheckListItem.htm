Enter the following details:

 

  1. Checklist Item - Enter the code for the checklist item in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Description - Enter the description of the checklist Item in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  3. Checklist Type - Select the Checklist Type from the list. The listed values are Check Box, List Item, Label. This is a mandatory field.

  4. List ID - This field is enabled if the Checklist Type is selected as 'List Item'. The listed values are defined in the List Item function. This is a mandatory field.

  5. Accept Remarks? - Select the check box to enable entering remarks for the checklist item. The remarks are entered in the OT Slate function when the checklists verification is done.

  6. Item Level - Select a Item Level from the list. The listed values 1 - 4.

  7. Enabled - Select the check box to enable the Checklist Item.
