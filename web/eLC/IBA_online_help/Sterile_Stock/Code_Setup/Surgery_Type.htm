Surgery Type

From this function you can Define various Surgery Types and Query for the existing records.

The defined Surgery Types can be associated to a group while defining the Sterile Stock groups. Examples of surgery types are Orthopaedic Surgery or Dental surgery and so on.

Defining Surgery Types

From SS Code Setup, select the Surgery Type function.

The Surgery Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory fields. <Related Field Description>

  3. Click [Apply] to save the defined Surgery Type or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Surgery Type records.

You can view details such as Code, Long Description, Short Description, Surgery Name(Short Desc), Nature and Enabled. The Surgery Name field displays the Surgery Name and Nature displays the nature of surgery, that is, ‘Emergency’ or ‘Elective’.

 

Based on the ‘If the ‘Surgery Type linked to  OT’ parameter set in the Parameter for Facility screen, these two columns will be displayed. You can click the surgery type hyperlink in Code column to view the Surgery Type Detailed View.

 

In the Surgery Type Detailed View screen, you can edit the Short Description and Long description of the surgery type and also enable or disable the surgery type. The Surgery Name field displays the linked surgery name in read-only mode. You can click [?] to search and link to a different surgery name. The Nature field in disabled mode indicates the nature of surgery, that is, ‘Emergency’ or ‘Elective’ as defined for the linked Surgery Name in OT.

 

Only 10 records can be added at a time. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate code with a unique color code. If the mandatory values are not entered, the system alerts you to enter these values. The mandatory field is highlighted with a unique color code.
