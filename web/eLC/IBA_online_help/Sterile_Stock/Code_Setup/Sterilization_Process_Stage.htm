Sterilization Process Stage

From this function you can Define the status for the Items while undergoing the sterilization process. You can also Query for the existing records.

Defining Status for Items undergoing Sterilization Process

From SS Code Setup, select the Sterilization Process Stage function.

The Sterilization Process Stage screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory fields. <Related Field Description>

  3. Click [Apply] to save the defined Sterilization Process Stage or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Sterilization Process Stage records.

Only 10 records can be added at a time. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate code with a unique color code. If the mandatory values are not entered, the system alerts you to enter these values. The mandatory field is highlighted with a unique color code.
