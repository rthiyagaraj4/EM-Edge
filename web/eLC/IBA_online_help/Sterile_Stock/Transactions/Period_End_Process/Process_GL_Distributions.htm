Process GL Distributions

From this function you can Create GL Interface details for the current period’s Sterile Stock transactions.

The transaction records are interfaced to General Ledger module based on the account codes setup in General Ledger module. This function is applicable only when the General Ledger module is interfaced with Sterile Stock module.

Creating GL Interface Details

From SS Period End Process, select the Process GL Distributions function.

The Process GL Distributions screen gets displayed.

  1. The Current Processing Period is displayed and disabled.

    • This period denotes the Sterile Stock module’s processing period. This period must always denote the current processing month and year and not the calendar month and year. This month and year will be automatically updated to the next period during month end processing. The processing month and year cannot be changed, once it is entered and the changes are committed. This is a mandatory field.

  2. Click [Apply] to create General Ledger Interface details for all the transactions in the Sterile Stock module for the specified period.

The ‘Current Processing Month/Year’ option is based on the Period End Scope defined in MM Parameter. If the Period End Scope is defined as Facility Level, enter the Current Processing Month/Year. This is a one-time setup. The Current Processing Year and Month gets updated in the SS Parameter whenever you run the Period End Process.
