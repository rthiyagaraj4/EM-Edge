Print Count Sheet

From this function you can Print count sheet for the Owner/Current Owner/Group Combination as a part of Physical Inventory Process.

The Print Count Sheet function aids in printing count sheet against a particular owner/current owner/group combination to check the items in the stock. On printing a count sheet, the system will allocate a Physical Inventory ID number, which will be referenced for every Physical Inventory Process.

The stock count sheets are printed with a Physical Inventory ID, the owner store, current store, the group type, the group code, group description and blank spaces for writing the physically available tray numbers.

These printed count sheets are handed over to the staff performing the actual physical counting of the items. You must note that the system will not allow functioning of any transactions from the time the count sheet is printed until the variances are posted on the system or the count sheet is cancelled.

Printing Count Sheets

From SS Physical Inventory, select the Print Count Sheet function.

The Print Count Sheet screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run] to print the Count Sheet for the specified Owner/Current/Group combination or [Reset] to clear the entered data.

  3. Select ‘Report Option’ to generate the report.

  4. Click [OK] to Print count sheet for the specified Owner/Current/Group combinations and spool it in the file on the print server or [Cancel] to cancel the action.
