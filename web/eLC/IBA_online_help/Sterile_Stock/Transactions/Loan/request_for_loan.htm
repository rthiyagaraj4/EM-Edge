Request for Loan

From this function you can Record the required quantities of all the chosen sterile groups at once.

This function allows you to loan items between non-owner stores only.

Requesting for Loan

From the SS Transactions > Loan > select the Request for Loan function.

The Request for Loan screen gets displayed.

 

  1. Click [Create] to invoke the Create screen.

  2. Enter the mandatory (*) search criteria fields. Refer < Related Field Description>  

    Enter the following details:

    1. Doc Type – Select the Document Type as defined through the ‘Document Type’ Setup
      in ‘System Manager’ module. This field is mandatory.

    2. Doc No - Enter the number of the Inter-Store loan request document. This field is mandatory.

    3. Doc Date - Select the date when the Inter-Store loan request was placed. This field is mandatory.

    4. Requested By - Select the non-owner store name from which the Inter-Store loan request is being requested. This field is mandatory.

    5. Requested On - Select the non-owner stores from which inter-store loan request is being issued.

    6. Doc Ref - Enter the document reference, if any.

    7. Entry Completed - This checkbox is enabled, by default when the alternate Group is chosen.

    8. Entered By - Displays the name of the user who enters the Inter-Store loan request.

    9. Request Type - Select the type of request.  

    10. Request Group – Click this hyperlink to view the Group item details popup.

    11. Group Search – Displays the Sterile Group Search and Select popup.

    Search results:

    1. Del - By default, this checkbox is always selected for already available Sterile

    2. Groups in the Request for Loan screen. You can also select this check box to mark a particular Sterile Group for deletion. If the Sterile Group is marked for deletion, entering the Request Quantity field is not mandatory.

    3. GroupType – Type of sterile group.

    4. Request Group – Displays the name of the request group. Click this hyperlink to view the Group item details popup.

    5. Authorized Group – Displays only the alternate groups associated with both the ‘Requested On’ and ‘Requested By’ Stores.  You can also choose the alternate groups of the corresponding required sterile groups at once now.  

    6. Surgery Type – Type of surgery.

    7. Surgery Name – Name of the surgery.

    8. Available Sterilized Quantity in : CSSD Chem – Displays the available sterilized quantity of the Alternate Group when the alternate Group is chosen. The Entry Completed checkbox is enabled, by default.

    9. Request Quantity - If the Sterile Group is marked for deletion, entering the Request Quantity field is not mandatory.

     

  3. Click [Group Search]; displays the Sterile group details in the Group Search and Select popup.

    Enter the following details:

    1. Search By - Select Code or Description based on which you want to perform the search.

    2. Search Criteria – Select the search criteria such as Starts With

    3. Search Text – Enter the text based on which you want to perform the search.

    4. Group Type – Select the type of group based on which you want to perform the search.

    5. Surgery Type/Name – Click Lookup to select the Surgery type/name.

    6. Consider Groups in Target Store – Select this check box if you want to view only the Sterile Groups associated with both the Requesting and Issuing Store.

    7. Search – Performs the search based on the search criteria entered.

    8. Reset – Resets the search criteria.

    9. OK – Performs the search based on the search criteria entered.

    10. Close – Closes the Group search and select popup.

      Search results:
       

    11. Code – Click this hyperlink to view the group item details in Group Item Details popup.

    12. Description – Displays the description of the sterile group.

    13. Surgery Type - Displays the type of the surgery.

    14. Surgery Name – Displays the name of the surgery.

    15. GroupType – Displays the type of sterile group.

    16. Select – Multi-select this checkbox to select to view the available sterile quantities at the Requested by Store in the Request for Loan screen.

  4. Enter the search criteria such as Search By, Search Criteria, Search Text, Group Type, Surgery Type/Name and search for Sterile Groups.

  5. Select the Consider Groups in Target Store check box to view only the Sterile Groups associated with both the Requesting and Issuing Store. Else, all sterile groups associated with ‘Requested by’ store Will be taken as search criteria while searching.

  6. Click Search to view the search results such as Code, Description, Surgery Type, Surgery Name and Group Type. The Group Type and the Surgery Type of the searched sterile groups are displayed as additional columns in the search results. The default value for ‘Group Type’ is ‘All’.

  7. Click the Group Code hyperlink to view the Group item Details popup.

  8. Select required multiple records and click OK in the Group Search and Select popup to close it and view the available sterile quantities at the Requested by Store in the Request for Loan screen.

If you don’t select a sterile group in the Groups Search and Select popup, an error message ‘Please select at least one Sterile Group to proceed’ is displayed.

  1. In the Request for Loan screen, only the Alternate Groups associated with both the ‘Requested On’ and ‘Requested By’ Stores are listed in the Authorized Group column. You can change the alternate groups of the corresponding required sterile groups.

  2. When an alternate group is chosen, the Available Sterilized Quantity of the alternate group is displayed. The Entry Completed checkbox is enabled, by default. Enter the Request Quantity.

  3. When an alternate group is chosen, the Available Sterilized Quantity of the alternate group is displayed. The Entry Completed checkbox is enabled, by default.

  4. Enter the Request Quantity.

  5. By default, the Del checkbox is always selected for already available Sterile Groups the Request for Loan screen. This remains applicable even if the particular Sterile Group is marked for deletion in the Request for Loan screen.

If the Sterile Group is marked for deletion, entering the Request Quantity field is not mandatory.

  1. Click the Request Group hyperlink in the Request for Loan screen to view the Group item details popup.

    Enter the following details:

    1. Group Code – Displays the code of the Sterile Group.

    2. Group – Displays the name of the Sterile Group.

    3. Item – Displays the name of the item.

    4. Consumable – Displays X mark for items not selected from the sterile group.

    5. Mandatory Item – Displays Tick mark if the item is mandatory.

    6. Quantity – Displays the number of items requested.

  2. Click Apply.
