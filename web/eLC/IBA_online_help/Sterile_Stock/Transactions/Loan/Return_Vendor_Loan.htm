Return Vendor Loan

From this function you can Return loan of sterile stock from third party suppliers.

This function allows you to return loaned sterile stock to third party suppliers.

Returning Loan to Vendors

From the SS Transactions > Loan > select the Retrun Vendor Loan function.

The Acknowledge Vendor Loan screen gets displayed.

 

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Search] to display the vendor Loan Request Doc No, Doc Type, Vendor Loan Request Doc Date, Request By and Request Type.

  4. Click [Apply]; the Finalize alert message popup is invoked with Yes and No buttons.

  5. Click [Yes] in popup to return loaned sterile stock to third party suppliers.

  6. Click [No] in popup to reset the details.

  7. To delete any record, select the check box against that record and click [Delete]. The selected record gets deleted.

  8. Click [Reset] to clear the entered data.

  9. Click [Query] to search for the existing records.
