Prepare

From this function you can Record the group preparation details and Query for the un-finalized Prepare documents. You can also Delete the un-finalized prepare documents.

The Prepare function facilitates you to record group preparation details into the system. The logged in user must have rights to ‘Prepare Group’ set in ‘SS User Access for Store’ to prepare trays. The details entered during preparation are owner (store), group (Tray/Set/Single/Pack) and quantity. The sum of component cost is displayed as total value.

Defining User Access for Store

From SS Transactions > Tray Maintenance, select the Prepare function.

The Prepare screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Tray details or [Reset] to clear the entered data.

  4. On clicking [Apply], the ‘Confirm Finalize’ dialog is displayed. Click [Yes] to finalize the Prepare details or [No] to finalize the Prepare details in the query mode.

  5. The prepare document number will be generated based on the document type selected for the Prepare Transaction. The Prepare documents pending finalization are available in the query function.

  6. Click [Query] to invoke the query screen.

  7. The un-finalized prepare transactions are displayed based on the default criteria set in the SS Parameter for Facility function or enter the search criteria and click [Search] to select a specific record.

    Enter the following search criteria to get the required result:

    1. Doc Type – Select the prepare document type from the list. The default ‘Document Type’ set in ‘SS Parameter for Facility’ is displayed, by default. This value can be modified.

    2. Doc No. – Enter the prepare document number in numeric format of eight digits maximum length.

    3. Doc Date – Enter the document date in DD/MM/YYYY format or click Calendar function to select the document date.

    4. Store - Select a Store from the list. The stores are displayed based on the restrictions set in ‘SS User Access for Store’ and applicability of the store to perform the specific transaction as set in ‘CSSD Store’.

    5. Group - Click [?] function to search for the required group from the Group Search and Select dialog. In the search results, multi-select the groups, as required and click Ok. Displays the selected groups in the Prepare screen.

  8. Click Doc Type hyperlink to finalize the prepare transaction. The group details are displayed and the Prepare check box is selected, by default. If you want to select more groups, click Group search in the search criteria to search and multi-select other groups from the Group Search and Select dialog. Click Ok; displays the selected groups in the Prepare screen.

    1. Search By – Select either Code or Description to search by item code or item description.

    2. Search Criteria – Select the search criteria such as Ends with.

    3. Search Text – Enter the item code or Item description based on the selection made in Search By.

    4. Store – Displays the name of the store selected.

    5. Search – Searches and displays the search results based on the search criteria entered.

    6. Reset – Resets the values entered in the fields.

    7. Code – Displays the item code as a hyperlink. When clicked, displays the Group Item Details dialog.

    8. Description – Displays the description of the item.

    9. Select –You can multi-select or single-select the check box(es) to select the group(s).

    10. Ok – Completes the group search and selection and displays the search results in the Prepare screen.

    11. Cancel – Cancels the search operation.

  9. Click Group hyperlink to view the group item details in the Group Item Details dialog. Click Close to close the dialog and return to the Prepare screen.

    1. Group Name: Displays the name of the item in the selected group.

    2. Grid – Display the following details in a grid.

      • Item – Displays the name of the item.

      • Consumable - Displays a Tick mark or ‘X’ mark to indicate that the Item is consumable or not.

      • Mandatory – Displays a Tick mark or ‘X’ mark to indicate that the Item is mandatory or not.

      • Quantity – Displays the quantity of the item as entered in the Prepare screen.

      • Close - Closes the dialog.

  10. Modify the Quantity and select the Bin Location.  Both these fields are mandatory.

  11. The Total value is calculated automatically and displayed based on the Quantity entered.

  12. Click [Apply] to finalize the prepare document or [Delete] to delete the selected record.

  13. On clicking [Delete], the ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.

You can modify all the details except the Document Type and Document Number.
