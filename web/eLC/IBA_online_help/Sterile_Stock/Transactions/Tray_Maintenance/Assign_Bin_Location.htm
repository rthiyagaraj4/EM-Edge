Assign Bin Location

From this function you can Assign the Bin Location to various trays prepared.

The Bin Locations are defined in the Bin Location function of Materials Base module. The Bin Locations are assigned while preparing the trays in ‘Prepare Group’ function or can be assigned using ‘Assign Bin Location’ function. All the prepared trays for which Bin Locations are not assigned in the ‘Prepare Group’ function, are displayed by default.

Assigning Bin Location to the Prepared Trays

From SS Transactions > Tray Maintenance, select the Assign Bin Location function.

The Assign Bin Location screen gets displayed.

  1. Click [Query] to display all the finalized prepare documents for which Bin Locations are not assigned or enter the search criteria and click [Search] to select a specific record.

    Enter the following search criteria to display the finalized prepare documents for which Bin Locations are not assigned:

    1. Doc Type – Select the prepare document type from the list.

    2. Doc No – Enter the prepare document number in numeric format of eight digits maximum length.

    3. Doc Date – Enter the document date in DD/MM/YYYY format or click Calendar function to select the document date.

    4. Store – Select a Store from the list. The stores are displayed based on the restrictions set in ‘SS User Access for Store’ and applicability of the store to perform the specific transaction as set in ‘CSSD Store’.

    5. Group - Enter the group in alphanumeric format or click [?] function to search for the required group.

  2. On searching, the finalized prepare documents for which Bin Locations are not assigned are displayed.

  3. Click ‘Doc Type’ hyperlink to display the prepared tray details. The details like Document Type, Document Number, Document Dare, Store, Group, Document Reference, Quantity, and Total Value specified while preparing the tray are displayed. The Tray Numbers are assigned and displayed based on the quantity specified.

  4. Click ‘Tray No’ hyperlink to assign the Bin Location to the selected Tray.

  5. Select the ‘Bin Location’ from the list. This is a mandatory field.

  6. Click [Modify] to modify the assigned Bin Location or [Cancel] to cancel the entered details.

  7. On assigning the Bin Location, the Bin Location is displayed beside the Tray Number. The Assign check box is enabled and selected.

  8. Clear the ‘Assign’ check box to exclude the assigned Bin Location details.

  9. Click [Apply] to record the Bin Location or [Reset] to clear the entered data.

You can assign different Bin Locations to different Trays.
