Washing

From this function you can Record the details required for the washing process and Query for the un-finalized load details. You can also Delete the un-finalized load details.

The ‘Washing’ transaction aids in capturing washing details required for the washing process. Washing is performed in two steps. In the first step, trays that require washing are selected and washing details are specified. A load number is generated and trays loaded are for washing. In the second step, the washing is set to complete after the trays are washed. If the trays require re-washing, they are reloaded under different load number.

Recording Washing details

From SS Transactions > Sterilization Process, select the Washing function.

The Washing screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Get Ster Req Dtls] to get the Sterilization Request details.

  4. On generating the Sterilization Request, all the trays which require washing for the specified conditions are displayed. The displayed details are Group name, Tray number, Sterilization Request Document Type, Document Number and Store. The finalized sterilization request documents (trays) pending washing are displayed in the washing list.

  5. Select the ‘Select’ check box to select the tray for washing. The check box is cleared, by default. To select all the trays for washing, select the check box in the column header.

  6. Click [Apply] to add the trays for Washing or [Reset] to clear the entered details.

  7. The load number is generated on recording the details.

  8. Click [Query] to display all the incomplete load details for which ‘Washing Completed’ check box is not selected while saving the record or enter the search criteria and select a specific record.

    Enter the following search criteria:

    1. Washing Unit - Select the Washing Unit from the list.

    2. Load No. – Enter load number in numeric format of 15 digits maximum length.

    3. Washing Type - Select the Washing Type from the list.

  9. Click Washing Unit hyperlink to complete the washing process.

  10. Select the ‘Select’ check box and click [Apply] to complete the washing process. Click [Delete] to exclude the record from washing.

  11. On clicking [Delete], the ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.

Washing is applicable, if ‘Wash’ check box is selected in the ‘Sterilization Request’ function.
