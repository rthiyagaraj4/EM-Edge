Enter the following details to create SS Parameter for Entity record:

Interface

  1. Billing – Select the check box to interface Patient Billing Manager module with Sterile Stock module. This check box is selected by default, if Patient Billing Manager module is interfaced with Sterile Stock module.

  2. Billing Service – Select the required Billing Service from the list. The listed values are defined in the ‘Billing Services’ function of the Revenue Management module. This field is enabled if ‘Billing check’ box is selected.

  3. General Ledger - The check box is selected if General Ledger module is interfaced with Sterile Stock module. Select the check box to interface General Ledger module with Sterile Stock module.

Default Document Types

  1. Prepare Group – Select the default Document Type for Prepare Group Transaction from the list. All the Document Types applicable at the Entity Level and associated to the Prepare Group transaction in SS Document Type function are displayed in the list.

  2. Request Group – Select the default Document Type for Request Group Transaction from the list. All the Document Types applicable at the Entity Level and associated to the Request Group transaction in SS Document Type function are displayed in the list.

  3. Issue Group – Select the default Document Type for Issue Group Transaction from the list. All the Document Types applicable at the Entity Level and associated to the Issue Group transaction in SS Document Type function are displayed in the list.

  4. Return Group – Select the default Document Type for Return Group Transaction from the list. All the Document Types applicable at the Entity Level and associated to the Return Group transaction in SS Document Type function are displayed in the list.

  5. Request for Loan - Map the default document type value ‘Request for Loan Group’ to this transaction type as mapped in System manager module.

  6. Issue against Loan - Map the default document type value ‘Issue against Loan Group’ to this transaction type as mapped in System manager module.

  7. Return of Loan - Map the default document type value ‘Return of Loan Group’ to this transaction type as mapped in System manager module.

  8. Return Instrument – Select the default Document Type for Return Instrument Transaction from the list. All the Document Types applicable at the Entity Level and associated to the Return Instrument transaction in SS Document Type function are displayed in the list.

  9. Tray Discrepancy – Select the default Document Type for Tray Discrepancy Transaction from the list. All the Document Types applicable at the Entity Level and associated to the Tray Discrepancy transaction in SS Document Type function are displayed in the list.

  10. Write Off – Select the default Document Type for Write off Transaction from the list. All the Document Types applicable at the Entity Level and associated to the Write Off transaction in SS Document Type function are displayed in the list.

  11. Variance – Select the default Document Type for Variance Transaction from the list. All the Document Types applicable at the Entity Level and associated to the Variance transaction in SS Document Type function are displayed in the list.

  12. Process Sterilization Request – Select the default Document Type for Process ROF Transaction from the list. All the Document Types applicable at the Entity Level and associated to the Process ROF transaction in SS Document Type function are displayed in the list.

Other Defaults

  1. Sterile Life – Enter the sterile life period in months in numeric format of three digits maximum length.   

  2. Item Image Path – Enter the item image path in alphanumeric format. The path specifies the location where the image files of the item can be stored. This path is utilized in the function ‘CSSD Item’ to link the defined item to an image.

  3. Write Off Reason - Select the default Write off Reason from the list. The listed values are defined in the function ‘Write off Reason’.

  4. Generate Sterilization Request While Preparing – Select the check box, to generate the sterilization request automatically by the system on preparing trays. If the check box is cleared, you must generate the sterilization request manually on preparing tray(s).
