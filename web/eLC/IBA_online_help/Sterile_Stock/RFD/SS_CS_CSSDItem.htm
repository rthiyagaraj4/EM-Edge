Enter the following details to define Sterile Stock related attributes:

  1. Item – Enter the required Item name in alphanumeric format of 40 characters maximum length or select the Item using [?] function. This is a mandatory field.

The items displayed are defined in Materials Base module ‘Item’ function. The Items for which ‘Sterile Stock Component’ is selected and attributes like Consumable/Chargeable is not defined in CSSD Item function are displayed.

  1. Image File - Enter the file name of the image in alphanumeric format of 200 characters maximum length or click on the ‘Image file’ hyperlink to locate the image file. The Image File is hyperlinked only if the Image File path is defined in the SS Parameter for Entity function. On clicking the ‘Image File’ hyperlink, the system searches for files only in the location specified in the ‘SS Parameter for Entity’.

  2. Consumable – Select the check box to indicate the selected item is consumable.

  3. Consumables – Items consumed by the requesting location (requestor) and not returned to the stores (owner) after usage.

  4. Non-consumables - Items used by the requesting location and sterilized for re-usage. These sterilized items are returned back to the owner stores.

Requesting locations are charged on requesting consumable items that are chargeable.

  1. Chargeable – Select the check box to indicate if the item is chargeable. This check box is enabled only if the ‘Consumable’ check box is selected. Only the Consumable Items can be charged.
