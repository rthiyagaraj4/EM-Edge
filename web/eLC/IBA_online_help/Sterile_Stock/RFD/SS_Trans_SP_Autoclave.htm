Enter the following details to get the sterilization request:

  1. Autoclave Unit – Select the Autoclave Unit from the list. The listed units are defined in the function ‘Autoclave/Washing Unit’ with unit condition as ‘ready to use’. This is a mandatory field.

  2. Service Location – On selecting an Autoclave Unit, the Service Location associated to it is displayed and the value cannot be changed.

  3. Sterilization Type – The Sterilization Type associated to the selected Autoclave Unit is displayed.

  4. Load No. – The Load Number is system generated on recording the sterilization details.

  5. Temperature - The temperature value is displayed on selecting the You can modify the temperature.

  6. Autoclave Date - The system date is displayed.  

  7. Store - Select the owner store from the list. The listed stores have rights to perform ‘Sterilization Request Process’ and assigned to logged in user.

  8. Entered By – The logged in user name is displayed and disabled.

  9. Entry Completed – Select the check box to indicate entry of autoclaving load details is completed. The load details cannot be modified. If entry is not completed, then the trays can be deleted from autoclaving process. The check box is cleared, by default. The value must be specified after selecting trays for sterilization.
