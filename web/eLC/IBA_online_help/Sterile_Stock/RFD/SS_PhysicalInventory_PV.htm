Enter the following details to review and Adjust the stock balance:

  1. Physical Inv ID – Select the Physical Inventory ID from the list. This is a mandatory field.

  2. Doc Type – Select the document type from the list. The document type as set for variance in ‘SS Parameter for Facility’ function is displayed, by default. This is a mandatory field.

  3. Doc No – Enter the document number in numeric format of eight digits maximum length. This is a mandatory field.

  4. Doc Date – Enter the date in dd/mm/yyyy format or select the document date using the Calendar function. The current system date is displayed, by default. This is a mandatory field.

  5. Write off Reason – Select the Write off Reason from the list. This reason will be applicable for missing trays. The Write Off Reasons defined in the Write Off Reason function is displayed in the list. This is a mandatory field.
