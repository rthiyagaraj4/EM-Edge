Enter the following details:

  1. TRN Type – Select the TRN Type from the list. The listed values are ‘Issue Group’ and ‘Return Group’. This is a mandatory field. When the value for the ‘TRN Type’ is selected as Return, the ‘Action Type’ is defaulted to ‘Acknowledge’.

  2. Document Type – Select the Doc Type from the list. All the defined document types are available in the list. This is a mandatory field.

  3. Action Type – Select the Action Type from the list. The listed values are ‘Acknowledge’ and ‘Confirm’.  Select ‘Acknowledge’ to view the records not yet acknowledged or waiting for acknowledgement by the receiving store. Select ‘Confirm’ to confirm the Issue Group. But you cannot confirm a Return Group .

You can acknowledge multiple sterile groups across multiple documents at the same time without performing the search again.  You can acknowledge all documents at once, if required.

  1. From Store – Select the Store name from the list. The ‘From Store’ is mandatory for confirming rejection.

  2. To Store – Select the Store name from the list. The ‘To Store’ is mandatory for acknowledging the issue transactions.

  3. From Doc Date and To Doc Date – Select From and To Dates using the Calendar function.

  4. Doc No – Enter the document number in numeric format of eight digits maximum length.

  5. Doc Ref – Enter the document reference number in alphanumeric format of 15 characters maximum length.

  6. Group – Select the Group using the [?] function.
