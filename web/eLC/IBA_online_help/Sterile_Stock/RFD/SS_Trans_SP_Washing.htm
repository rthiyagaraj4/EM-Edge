Enter the following details to record the washing details:

  1. Washing Unit - Select the Washing Unit from the list.  The washing units are defined in the function ‘Autoclave/Washing Unit’. This is a mandatory field.

  2. Service Location – The Service Location is displayed automatically on selection of the Washing Unit. The displayed value cannot be changed.

  3. Temperature - On selection of the Washing Unit, temperature is displayed. Working temperature of the unit is specified in the function ‘Autoclave/Washing Unit’. You can modify the displayed value and specify the required temperature.

  4. Load No. – The load number is automatically generated by the system on saving the transaction.

  5. Ster Req Doc Type - Select the sterilization request document type from the list. The document type specified in the function ‘SS parameter for Facility’ for process sterilization request is displayed, by default. The values listed are defined in the function ‘SS Document Type’.

  6. Washing Type – Select the washing type from the list.  The washing types are defined in the function ‘Washing Type’. This is a mandatory field.

  7. Store - Select the owner store from the list. The owner stores with rights to perform ‘Sterilization Request Processing’ and assigned to the logged in user are displayed.  

  8. Washing Completed – Select the check box to indicate washing is completed. The check box is cleared, by default. This value is not specified in the first step. Once washing is completed (second step), washing is set to complete for a load.
