Enter the following details to request for sterilization:

  1. Doc Type – Select the sterilization request document type from the list. The values listed are defined in the ‘SS Document Type’ function and the default value displayed is specified in the ‘SS Parameter for Facility’ function for process sterilization request. This is a mandatory field.

  2. Doc No. – Enter the Document number in numeric format of eight digits maximum length if manual entry is applicable for selected document or the system generates the document number on saving the transaction. This is a mandatory field.

  3. Doc Date - The system date is displayed, by default. You can change the document date. Enter the date in date format of dd/mm/yyyy or select the date using the Calendar function. The entered date must not be less than the Processing month and year set in ‘SS Parameter for Facility’ function and not greater than current system date. This is a mandatory field.

  4. Store – Select the store from the list. Only those owner stores are listed from which the logged in user can perform ‘Sterilization Request Process’.

  5. Service Location – The Service Location defined in ‘SS Parameter for Entity’ function is displayed, by default. Select the required Service Location from the list. All the enabled service locations defined in the ‘Service Location’ function are displayed in the list.

  6. Doc Ref – Enter the document reference in alphanumeric format of 15 characters maximum length.

  7. Sterilize – The check box is selected, by default indicating sterilization is required. Clear the check box to indicate sterilize is not required for the prepared Tray.

  8. Wash - The check box is selected, by default indicating washing is required. Clear the check box to indicate washing is not required for the prepared Tray.

  9. Pack - The check box is selected, by default indicating packing is required. Clear the check box to indicate packing is not required for the prepared Tray.

  10. Group Type – Select the group type from the list. The displayed values are Set, Tray, Soft Pack and Single. The value ‘Set’ is displayed, by default.

  11. Group – Enter the group in alphanumeric format of 15 characters maximum length or select the group name using [?] function. This is a mandatory field.

  12. Tray No. – Enter the tray number in numeric format of 12 digits maximum length or select the tray number using [?] function. This is a mandatory field.

  13. Contaminated – Select the check box to indicate the tray is contaminated.

  14. Sterilization Type - This field is disabled if ‘Sterilize’ check box is cleared. If sterilization must be performed, select the required Sterilization Type from the list. The value displayed by default on selection of group corresponds to the tray definition.

  15. Washing Type - This field is disabled if ‘Wash’ check box is cleared. If washing must be performed, select the required Washing Type from the list. The value displayed by default on selection of group corresponds to the tray definition.
