Enter the following details record the Instrument Usage or Discrepancy details:

  1. Store – Select the requestor store from the list. The stores assigned to the logged in user is displayed in the list.  This is a mandatory field.

  2. Group Type – Select the Group Type from the list. The Group Types displayed are Set, Tray, Single and Soft Pack. The value ‘Set’ is displayed, by default.

  3. Group – Enter the Group name or select the Group name using [?] function. All the groups created in ‘CSSD Store’ function are displayed.  This is a mandatory field.

  4. Details - on selecting the Group,  Details hyperlink is enabled. Click 'Details' hyperlink to view the details of Items associated to the selected Group.

  5. Tray No. – Select the tray number using [?] function. This is a mandatory field.

On selecting the tray number, the ‘Store’, ‘Group Type’, ‘Group’ and ‘Tray No’ fields are disabled.

 

Enter the following discrepancy details of the selected tray:

  1. Instrument – The items of the selected tray is displayed. Select the item that differs from the actual tray definition. This is a mandatory field.

  2. Discrepancy Reason - Select the discrepancy reason from the list. The reasons displayed are defined in the function ‘Write Off’. This is a mandatory field.

  3. Remarks – Enter the remarks in alphanumeric format of 100 characters maximum length.

  4. Discrepancy Quantity - Enter the discrepancy quantity in numeric digit of 15 digits maximum length. This is a mandatory field.

  5. Patient ID – Enter the Patient ID using the instrument in the format defined in Master Patient Index module. You can invoke [?] function to search for the required Patient ID. This is a mandatory field.

  6. Patient Name – The name of the patient is displayed on selecting the Patient ID.

  7. Encounter ID – Select the required Encounter ID of the selected patient from the list.
