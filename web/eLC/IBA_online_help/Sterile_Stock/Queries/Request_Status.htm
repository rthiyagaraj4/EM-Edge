Request Status

From this function you can View the current status of a request.

This query function helps in listing the Trays pending for issue or trays that are partially issued to take appropriate measures.

Viewing the current status of a Request

From SS Queries, select the Request Status function.

The Request Status screen gets displayed.

  1. The request status of the all the trays are displayed. Enter the search criteria and click [Search] to view the status of a specific request.

    Enter the following query criteria and search for the required request status:

    1. From Date – Enter the date in the format of dd/mm/yyyy or select the date using the Calendar function.

    2. To Date - Enter the date in the format of dd/mm/yyyy or select the date using the Calendar function. The ‘To date’ must be greater than ‘From date’.

    3. Entry – The value ‘All’ is displayed, by default. Select the values from the list. The listed values are ‘Entry Completed’ and ‘Entry not Completed’.

    4. Authorization – The value ‘All’ is displayed, by default. Select the values from the list. The listed values are ‘Authorized’ and ‘Unauthorized’.

    5. Issue – The value ‘All’ is displayed, by default. Select the values from the list. The listed values are ‘Not Issued’, ‘Partially Issued’ and ‘Fully Issued’.

    6. Cancellation – The value ‘All’ is displayed, by default. Select the values from the list. The listed values are ‘Cancelled’ and ‘Not Cancelled’.

    7. Req By Store - All the Requestor and Owner stores are listed. Select the stores from the list.

    8. Req On Store - All the Owner stores are listed. Select the owner stores from the list.

    9. Request Type – The value ‘Normal’ is displayed, by default. The listed values are ‘Urgent’, ‘Elective’ and ‘All’.

  2. On searching, the request status of a tray is displayed based on the entered criteria. The displayed details are: Doc Type, Doc No, Doc Date, Doc Ref, Req by Store, Req on Store, Entered, Authorized, Issued, Cancelled and Acknowledgement Status.

  3. Click Doc Type hyperlink to view the requested tray details.
