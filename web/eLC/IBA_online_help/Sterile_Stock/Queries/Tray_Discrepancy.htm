Tray Discrepancy

From this function you can View the discrepancy details of the issued trays.

Viewing the Tray Discrepancy details

From SS Queries, select the Tray Discrepancy function.

The Tray Discrepancy screen gets displayed.

  1. The discrepancy details of all the trays are displayed, by default. Enter the search criteria and click [Search] to view the discrepancy details of the selected trays.

    Enter the following query criteria and search for the required tray discrepancy record:

    1. Group Type – The value ‘All’ is displayed, by default. Select the values from the list. The listed values are Set, Tray, Single and Soft Pack.

    2. Group - Enter the group in alphanumeric format of 15 characters maximum length or select the group using [?] function.

    3. Tray No - Enter the tray number in numeric format of 15 digits maximum length.

    4. Discrepancy Entered At - Select the store from the list. All the stores are listed.

    5. Discrepancy – All the discrepancy reasons are listed. Select the reason from the list.

    6. Owner Store – Select the Owner store from the list. All the owner stores are listed.

  2. On searching, the discrepancy details of the selected trays are displayed based on the entered criteria. The displayed details are: Group Type, group, Item, Discrepancy Quantity, Discrepancy Type, Discrepancy Reason, Remarks, Patient ID, Encounter ID, Billing Type, Document Type, Amount, Entered At and Owner details.

    The following tray discrepancy details are displayed:

    The group name is displayed and tray details displayed under the Group name.

    1. Group Type – The group type is displayed and disabled.

    2. Tray No. – The tray number is displayed.

    3. Item - The discrepant item of the group is displayed.

    4. Discrepancy Qty - The discrepancy quantity of the item is displayed.

    5. Discrepancy Type – The type of discrepancy is displayed. The value displayed is Negative or Positive.

    6. Discrepancy Reason – The Discrepancy Reason is displayed.

    7. Remarks – The remarks entered in ‘Tray Discrepancy’ is displayed.

    8. Patient ID - The Patient ID entered while raising the discrepancy is displayed.

    9. Encounter ID - The Encounter ID of the patient is displayed.

    10. Billing Type - The Billing Type as entered in the function ‘Return Group’ is displayed.

    11. Doc Type - The document type selected for transaction is displayed. Discrepancy can be entered using ‘Return Group’ function and ‘Packing’. The document type selected in these functions are displayed, if discrepancy is raised using these functions.

    12. Doc No. – The document number of the selected document type is displayed.

    13. Amount – The product of unit cost of discrepant item and discrepant quantity is displayed.

    14. Entered - The store entering the discrepancy details is displayed.

    15. Owner - The owner store of the tray is displayed.

  3. Click ‘Doc Type’ hyperlink to view the requested tray details.
