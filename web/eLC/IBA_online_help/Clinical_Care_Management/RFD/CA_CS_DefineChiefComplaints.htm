Enter the following details to define a Chief Complaint:

 

  1. Complaint ID - Enter Complaint ID in alphanumeric format of six characters maximum length. This field is mandatory.

  2. Complaint Description – Enter the Complaint Description in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Specialty – Click [?] function to select a ‘Specialty’ to which the ‘Chief Complaint’ is to be associated. On selecting a specialty, the chief complaints are available for patients registered against the selected specialty.

  4. Complaint Text - Enter a detailed Description of the chief complaint in alphanumeric format of 4000 characters maximum length. This field is mandatory.

  5. Enabled – Select the check box to enable the chief complaints in transactions. If the check box is not selected the chief complaints are not available while recording diagnosis.
