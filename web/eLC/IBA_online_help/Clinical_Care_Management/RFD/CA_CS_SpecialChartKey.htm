Enter the following details:

 

  1. Chart Type – Select the Chart Type as Growth Chart from the list. On selecting Growth Chart, the field Chart Group is displayed.

  2. Chart Group – Select the chart Group as 0-36 Months or 2-20 Years, which is system defined from the list.

  3. Type – Select either Interval Based or Static Option Button.

On selecting the Chart Group as 0-36 Months and Interval Based Option Button, the following details are to be entered.

  1. Start Value – Enter a Start Value generate the Growth Chart for the Chart Group of 0-36 Months. This field is mandatory.

  2. End Value – Enter the End Value generate the Growth Chart for the Chart Group of 0-36 Months. This field is mandatory.

  3. Interval – Specify the Interval between the Start Value and End Value.

  4. Unit – Select the Unit as Days, Months or Years from the list.

Click [Generate] to generate the values of the chart.

On selecting the Chart Group as 2-20 Years and ‘Interval Based’ Option Button, the following details are to be entered.

  1. Start Value – Enter the start value to generate the Growth Chart for the Chart Group of 0-20 Years. This field is mandatory.

  2. End Value – Select the end value to generate the Growth Chart for the Chart Group of 0-20 Years. This field is mandatory.

  3. Interval – Select the Interval to generate the Growth Chart. This field is mandatory.

  4. Unit – Select the Unit as Days, Months or Years from the list.

Click [Generate] to generate the Growth Chart.

On selecting the Chart Group as 2-20 Years and ‘Static’ Option Button, the following details are to be entered.

  1. Unit – Select the Unit as Days, Months or Years from the list.

  2. Start Value – Enter the start value in numeric format.

  3. End Value – Enter the end value in numeric format.

Click [Add Rows] to enter five more rows to enter the start and end values.
