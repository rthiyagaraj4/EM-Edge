Outcome

  1. Click Outcome from the menu to record the outcome of the Post Delivery Mother Treatment event.

  2. The mothercare pending tasks to be recorded for the event Post Delivery Mother Treatment is displayed if any. Click [Record] to continue recording the outcome or click [Cancel] to perform the pending tasks for the event. On clicking [Record], the screen to record outcome gets displayed.

  3. The Outcome list box displays Close as the outcome for the event Post Delivery Mother Treatment. Select Close as outcome and a message is displayed indicating the current cycle of recording Post Delivery Mother Treatment event for the patient will be closed and a new cycle has to be created for recording details of any other events.

  4. Click [OK] to close recording events and put an end to the current cycle. In order to record any other events for the patient, a new cycle has to be created by selecting New from the Cycle list box on the top left part of the screen.
