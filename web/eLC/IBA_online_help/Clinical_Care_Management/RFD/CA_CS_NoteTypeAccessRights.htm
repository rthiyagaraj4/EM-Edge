Enter the following details to define access rights:

 

  1. Access Rights For – Select the value from the list. The listed values are Responsibility, Note Type, Practitioner and Specialty. This field is mandatory.

On selecting, the Access Rights For option as ‘Responsibility’, the Responsibility field is displayed.

  1. Responsibility - Select the required responsibility from [?] function. All the responsibilities to which the logged in user has access are displayed. This field is mandatory.

Click [Search] to display a list of all Note Types.

  1. Note Type - All the Note Types created using the Note Type function with status as enabled is displayed.

  2. Role - Select the applicable Role for a Note Type from the list. The values are ‘Transcriber’, ‘Author but without Authorization Facility’, 'Author with Authorization Facility', ‘Reviewer Privilege for Verifying and Authorization’ and 'View only Privilege'.

  3. Default Transcription Notes -  This check box is enabled after selecting the Note Type. The check box is enabled if the 'Transcription Applicable' check box is selected for the Note Group of the selected Note Type. Select the check box to set the Note Type as Default Transcription Notes for the responsibility. You can select only one Note Type as Default Transcription Notes.

or

On selecting, the Access Rights For option as ‘Note Type’, the Note Type field is displayed.

  1. Note Type - Select the required note type from [?] function. All the enabled Note Type records are listed. This field is mandatory.

Click [Search] to display a list of all Responsibilities.

  1. Responsibility - All the responsibilities are displayed.

  2. Role - Select the applicable Role for a Note Type from the list. The values are ‘Transcriber’, ‘Author but without Authorization Facility’, 'Author with Authorization Facility', ‘Reviewer Privilege for Verifying and Authorization’ and 'View only Privilege'.

 

or

On selecting, the Access Rights For option as ‘Practitioner’, the Practitioner field is displayed.

  1. Practitioner - Select the required Practitioner from [?] function. All the enabled Practitioner records are listed. This field is mandatory.

Click [Search] to display a list of all Note Types.

  1. Note Type - All the Note Types are displayed.

  2. Select - Select the required check boxes to associate the Note Types to the Practitioner.

  3. Default Transcription Notes - This check box is enabled after selecting the Note Type. The check box is enabled if the 'Transcription Applicable' check box is selected for the Note Group of the selected Note Type. Select the check box to set the Note Type as Default Transcription Notes for the Practitioner. You can select only one Note Type as Default Transcription Notes.

 

or

On selecting, the Access Rights For option as ‘Specialty’, the Specialty field is displayed.

  1. Specialty - Select the required Specialty from [?] function. All the enabled Specialty records are listed. This field is mandatory.

Click [Search] to display a list of all Note Types.

  1. Note Type - All the Note Types are displayed.

  2. Select - Select the required check boxes to associate the Note Types to the Specialty.

  3. Default Transcription Notes - This check box is enabled after selecting the Note Type. The check box is enabled if the 'Transcription Applicable' check box is selected for the Note Group of the selected Note Type. Select the check box to set the Note Type as Default Transcription Notes for the Specialty. You can select only one Note Type as Default Transcription Notes.
