Enter the following details to create a Location for Practitioner record:

 

  1. Specialty - Select the desired Specialty from the Specialty list. The list contains all the Specialties defined in the Application Master Module with an enabled status.  

  2. Practitioner - Select the required Practitioner using [?] function. This field is mandatory.

  3. Location Type - Select the Location Type to which the Practitioner is to be associated from the list. This field is mandatory.

  4. Location – Click [?] function to select the Location to which the Practitioner is to be associated from the list. The Location values are populated depending on the Location Type selected. This field is mandatory.

  5. Effective From - Enter the Effective from Date in dd/mm/yyyy format. By default, the current system date is displayed. This field is mandatory.

  6. Effective To - Enter the Effective to Date in dd/mm/yyyy format.
