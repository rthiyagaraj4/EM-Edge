  2. Enabled – Select the check box to enable the Quick Link or clear the check box to disable it.

  3. Two sections, 'Linked Functions' and 'Link New Functions' get displayed on selection of 'Adaptive Function To'.

Linked New Functions

  1. Function Ref – Select the function reference using using [?] function. The listed functions are Place Order and Register Patient Referral.  

  2. Description – The description of the function is displayed on selecting function reference. This is a mandatory field.

  3. Click [Add] to add the functions to the ‘Linked Functions’ section. The listed functions will be linked to the adaptive menu.

  4. Click [Clear] to clear the entered data.

Linked Functions

  1. Serial No - The Serial Number of Linked functions is displayed and hyperlinked. Click on the hyperlink to rearrange the sequence.

  2. Description - The description of the Linked Functions is displayed.

  3. Uncheck to Remove - Clear the check box to remove the linked functions. The check box is selected, by default.

  1. Criteria Name - The criterion applicable to the function are displayed.

  2. Actual - Select the Actual options applicable to the criteria names. Select the check box to enter the Actual Value or clear the check box to select the Relative Value.

  3. Actual Value - Select the Actual Value for the Criteria.

  4. Relative Value - Select the Relative Value for the Criteria.

Enter the following details:

 

  1. Quick Link Type – Select the Quick Link Type as Query Filter, Actual Function or External Application or Adaptive Menu. This is a mandatory field.

  2. Function – Select the function using the [?] function. This is a mandatory field. If Quick Link Type is selected as Query Filter, the function of CA module is allowed for selection. On selecting the Function, Define Filter Criteria section is displayed. If Quick Link Type is selected as Actual Function, select the required Module from the Function screen and then select the required function based on the module selected. This field is disabled if Quick Link Type is selected as External Application.

  3. External URL – Enter the external URL. This field is enabled on selecting Quick Link Type as External Application. This is a mandatory field.

  4. Description – Enter the Quick Link description. This is a mandatory field. This field is disabled if the Quick Link Type is selected as Actual Function.

  5. Scope – Select the scope as Function Specific or Global. If Function Specific is selected, the Quick Link being defined is grouped under the function filter. If Global is selected, the Quick Link being defined is grouped under Global filter. If both the check boxes are selected, the Quick Link is grouped under Function Filter and Global Filter. If Quick Link Type is selected as Actual Function or External Application, Global is selected and disabled, by default.

  6. Enabled – Select the check box to enable the Quick Link or clear the check box to disable it.
