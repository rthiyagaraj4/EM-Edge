Enter the following details to create a Clinical Event Parameter record:

 

  1. High Value - Enter the High Value in alphanumeric format of one character maximum length. This field is mandatory. The value entered is an indicator, while recording vital signs for the patient.

  2. Low Value - Enter the Low Value in alphanumeric format of one character maximum length. This field is mandatory. The value entered is an indicator, while recording vital signs for the patient.

  3. Abnormal Value - Enter the Abnormal Value in alphanumeric format of one character maximum length. This field is mandatory. The value entered is an indicator while recording vital signs for the patient.

  4. Critical Value - Enter the Critical Value in alphanumeric format of one character maximum length. This field is mandatory. The value entered is an indicator while recording vital signs for the patient.

  5. Critical Low Value - Enter the Critical low Value in alphanumeric format of one character maximum length. This field is mandatory. The value entered is an indicator while recording vital signs for the patient.

  6. Critical High Value - Enter the Critical Value in alphanumeric format of one character maximum length. This field is mandatory. The value entered is an indicator while recording vital signs for the patient.

  7. Delta Value – Enter the Delta Value in alphanumeric format of one character maximum length. This field is mandatory. The value entered is an indicator while viewing the Delta check results or Laboratory module result in patient history.

  8. Indicator – The Indicators for the values are displayed.

  9. Description – Enter the description of the values.

  10. Colour – Select the required colour for the values using  . The selected colors will reflect for values in transactions like Charting, Clinical Event (Normal and Flowsheet View) and Chart Summary (Last 24 hrs Event).

  11. Symbolic Legends - Select the required Symbolic Legend to indicate High Value, Low Value, Abnormal Value, Critical Value, Critical Low Value, and Critical High Value.

  12. Normalcy Indicator - Select Symbolic Legends or Colour Legends to display the values in the transactions as Symbolic Legends or Colour Legends.

 

History Record Types

Set the applicability of the categories for which the results come for review either for the ordering practitioner or for the Attending Practitioner based on the flag selected.

  1. Ordering Pract - Select the check box, if reviewing is applicable to the Ordering Practitioner. If the check box is selected, the selected results come up for review only for the Ordering Practitioner.

  2. Attending Practitioner - Select the check box, if reviewing is applicable to the Attending Practitioner.
