Enter the following details to create a Flow Sheet Component record:

 

  1. Type - Select the required radio button to choose the Type as ‘New’ or ‘Charting’. On selecting the Type as New, a new flow sheet is created and on selecting the Type as Chart, charts are included in the flow sheet that is viewed in the history.

  2. Flow Sheet ID – Enter Flow Sheet ID in alphanumeric format of 15 characters maximum length. Select the required Chart from [?] function on selecting the Type as Chart. This field is mandatory.

  3. Short Description – Enter the short description for the Flow Sheet Component in alphanumeric format of 20 characters maximum length. The Short Description of the chart is displayed on selecting the Chart. This field is mandatory.

  4. Long Description - Enter the long description for the Flow Sheet Component in alphanumeric format of 30 characters maximum length. The Long Description of the chart is displayed on selecting the Chart. This field is mandatory.

  5. Enabled - Select the check box to enable the Flow Sheet Component in transactions.
