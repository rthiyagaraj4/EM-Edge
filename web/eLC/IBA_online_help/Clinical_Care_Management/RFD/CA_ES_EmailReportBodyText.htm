Enter the following details:

 

  1. Module - Select the required module from the list. This is a mandatory field.

  2. Report - Select the Report from the list. Based on the module selected the Reports that are identified as applicable for Email are displayed. This is a mandatory field.

  3. Report Body Text ID - Enter the Body Text ID for the selected Report. This is a mandatory field.

  4. Report Body Text Description - Enter the description for the Report Body Text. This is a mandatory field.

On selecting the Report, the Patient, Encounter and Email Report Body Text hyperlinks are available for creating the Report Body Text.

  1. Patient - Select the required Patient values to display it in the editor. The values are Patient Name, Registration Date, Gender, Date of Birth, Age, Facility Name, Residential Address, and Nationality.

  2. Encounter - Select the Encounter value and the description is displayed in the editor. The values are Encounter ID, Current Location, Attending Practitioner, Admission Date, Specialty, Patient Class, Room Number, and Bed Number.

  3. Email Report Body Text - Select the values from the predefined system supplied parameter list to display it in the editor.

Click insert table icon to insert tables in the template. Specify the number of Rows and Columns and click [OK] to display the table in the editor.

Click Print Preview icon to preview the Report Body Text.
