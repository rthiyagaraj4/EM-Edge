Enter the following details:

 

  1. Event – The events defined for the chart are displayed.

  2. Value – Enter the values in the fields including discrete measure values with +/- characters.

  3. Unit – The unit associated to the Event is displayed.

  4. Graph – Select the check box to view the graphical representation of the values recorded against the selected Events. You can select more than one Event at a time. Once the Events are selected then click ‘Graph’ to view the graph.
