Enter the following details to administer Vaccines:

 

  1. Vaccine Name – The Vaccine code of the selected Vaccine is displayed if the Vaccine is selected from a Vaccine Category without Adhoc check box selected while creation. Select the required Vaccine Name from the list for Adhoc Schedules selected from the Existing Schedules frame.

  2. Batch ID – Enter the Batch ID in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Manufacturer – Enter the Manufacturer name in alphanumeric format of 100 characters maximum length. This is a mandatory field.

  4. Site Name – The default Site Name associated to the Vaccine in Site for Vaccine function is displayed for Normal Schedules. Select the required Site Name from the list.

  5. Outcome – Select the Outcome from the list. The listed values are ‘Completed’ and ‘Not Administered’. The Outcome is disabled and displayed as Completed for Adhoc Schedules. This is a mandatory field.

  6. Schedule Date – The current system date and time is displayed. You cannot edit or modify the schedule date.

  7. Administer Date - The current system date and time is displayed. You can enter the required Administer date and time in the format of dd/mm/yyyy hh:mm or select the required date and time using the Calendar function.

  8. External Administer – Select the check box if the Vaccine can be administered in an External Facility. If this check box is selected, recording the Batch Number and Manufacturer is not mandatory. The Outcome is displayed as Completed and disabled.

  9. External Facility – This field is enabled on selecting the External Administer check box. Enter the External Facility name in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  10. Administered By – The logged in User Name is displayed, by default. You can enter the required User Name or select the User Name using [?] function. This is a mandatory field.

  11. Remarks – Enter the Vaccine Administration Remarks in alphanumeric format of 200 characters maximum length.
