Enter the following details:

 

  1. Module - Select the required module from the list. This is a mandatory field.

  2. Report - Select the Report from the list. Based on the module selected the Reports that are identified as applicable for Email are displayed. This is a mandatory field.

  3. Subject - Enter the Subject for the Report.

  4. Disclaimer - Select the Disclaimer from the list. The list displays Disclaimers created with the 'Applicable to Report' selected in the Note Disclaimer function of CA Code Setup. Click View hyperlink to view the report.

  5. Verification Required for Confidential Result - Select the check box if verification is required for the confidential reports.

On selecting the Report, select the control setup if applicable.

  1. Applicable to Patient - The check box is selected, by default. If selected, reports to patients with Email ID captured is allowed.

  2. Applicable to Registered Users - The check box is selected, by default. If selected, reports to registered users with Email ID captured is allowed.

  3. Applicable to Adhoc Email Id - The check box is selected by default. If selected, e-mailing reports external Email ID is allowed.

  4. Body Text - Select the Body Text from the list. The listed values are defined in Email Report Body Text function.

  5. Verification Required - Select the check box if the email requires verification before sending. If the check box is not selected and the 'Verification Required for Confidential Results' is selected, the system forces verification of confidential reports before sending.
