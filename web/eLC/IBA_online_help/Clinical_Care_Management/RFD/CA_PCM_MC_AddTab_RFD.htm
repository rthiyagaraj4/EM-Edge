Enter the following details:

 

  1. Note Type -Select the Note Type from the list. The list displays Note Types associated to Note Groups with Applicable Task 'Medical Certificate'.

  2. Note Group - The Note Group associated to the Note Type is displayed and disabled.

  3. Med/Anc Service - The primary service identified for the patient at the time of registering or admitting the patient is defaulted in this field. This is displayed provided the Note Type selected is not specific to any Service. Select another medical service or ancillary service using [?] function. The [?] function displays only those services to which the Note Type is associated in the ‘Note Type by Service’ function otherwise it displays all services. This field is mandatory.

  4. Date/Time – The default value is the system date and time. Enter the date and time of the recording in format dd/mm/yyyy hh:mm. The date/time of recording can be backdated to a period, which is not greater than the value set in the parameter ‘Allowable backdated time’. The date/time cannot be greater than the system date/time as well. This field is mandatory.

  5. Performed By – The system defaults the name of the logged-in practitioner if the responsibility with which the practitioner has logged in has either one of the following privileges.

• Author without authorization facility.

• Author with authorization facility.

• Reviewer privilege for verifying and authorization.

If the privilege given is ‘Transcriber’ then you have to select the name of the practitioner on whose behalf the transcribing is being done. This field is mandatory.

  1. Note Title – Enter the notes title in alphanumeric format of 60 characters maximum length.
