Enter the following details:

 

  1. Group By – Select the required value from the list. The values are Task, Relationship and Responsibility.

If ‘Task’ is selected from ‘Group By’ list, all the Tasks set as ‘Applicable’ in the Application Tasks are displayed for selection.

  1. Task – Click [?] function to select the required Task. Once a value is selected, associate the selected Task to the Responsibility/Relationship individually by selecting the required radio button.

Click [Search] on selecting the Responsibility radio button.

  1. Responsibility – All Responsibilities set as enabled in the System Manager Module are displayed.

  2. Applicable – Select the required value from the list. The values displayed are ‘Applicable’ and ‘Not Applicable’. If a value is not selected, the system considers the value as ‘Not applicable’.

Once the values are selected, click [Apply] to save the association.

Click [Search] on selecting the Relationship radio button.

  1. Relationship – The Relationships set as enabled are displayed.

  2. Applicable – Select the required value from the list. The values displayed are ‘Applicable’ and ‘Not Applicable’. If a value is not selected, the system considers the value as ‘Not applicable’.

Once the values are entered, click [Apply] to save the association.

If ‘Relationship’ is selected from ‘Group By’ list all the relationships set as enabled are displayed for selection.

  1. Relationship – Select from required Relationship from [?] function. Once a value is selected, associate the selected Relationship to the Task and Function individually by clicking the required radio button.

Click [Search] on selecting the Task radio button.

  1. Task – The Application Tasks set as enabled in the Application Task Function are displayed.

  2. Applicable – Select the required value from the list. The values displayed are ‘Applicable’ and ‘Not Applicable’. If a value is not selected, the system considers the value as ‘Not applicable’.

Once the values are entered, click [Apply] to save the association.

Click [Search] on selecting the Function radio button:

  1. Function – The Functions of the Application Tasks set as Applicable in the Application Tasks function are displayed. This helps viewing the Application Tasks in groups under a Function.

  2. Task – The Application Tasks set as enabled in the Application Task Function are displayed.

  3. Applicable – Select the required value from the list. The values displayed are ‘Applicable’ and ‘Not Applicable’. If a value is not selected, the system considers the value as ‘Not applicable’.

Once the values are entered, click [Apply] to save the association.

If ‘Responsibility’ is selected from ‘Group By’ list, all the Responsibilities set as enabled are displayed for selection.

  1. Responsibility – Select the required Responsibility from [?] function. All the Responsibilities set as enabled are listed. Once a value is selected, associate the selected Responsibility to the Task and Function individually by clicking the required radio button.

Click [Search] on selecting the Task radio button.

  1. Task – The Application Tasks set as enabled in the Application Task Function are displayed.

  2. Applicable – Select the required value from the list. The values displayed are ‘Applicable’ and ‘Not Applicable’. If a value is not selected, the system considers the value as ‘Not applicable’.

Once the values are entered, click [Apply] to save the association.

Click [Search] on selecting the Function radio button.

  1. Function – The Functions of the Application Tasks set as Applicable in the Application Tasks function are displayed. This helps viewing the Application Tasks in groups under a Function.

  2. Task – The Application Tasks set as enabled in the Application Task Function are displayed.

  3. Applicable – Select the required value from the list. The values displayed are ‘Applicable’ and ‘Not Applicable’. If a value is not selected, the system considers the value as ‘Not applicable’.

Once the values are entered, click [Apply] to save the association.
