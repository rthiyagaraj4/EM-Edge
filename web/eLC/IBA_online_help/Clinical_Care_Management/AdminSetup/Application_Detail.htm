Application Detail

From this function you can Create an association of the External Application to an Order Category, Order Type, Order Catalog and Facility.

The orderable items which are interfaced with external application are linked using this function.

Defining External Application Details

From the CA Admin Setup, select Application Detail function.

The External Application Detail screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the External Application details or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing associated External Application record.
