Configure Email Format

From this function you can Select the events to be included in a e-mail sent to a patient.

Configure Email Format function is used to configure the list of events to be included in the e-mail that has to be sent to the patient. If patient instructions and clinical comments are recorded while placing order, system generates a PDF and an e-mail is sent to the patient with PDF attached.

Configuring Email Format

From the CA Code Setup > Emailing Setup, select Configure Email Format function.

The Configure Email Format screen gets displayed.

  1. Enter the 'Report' in alphanumeric format of 100 characters maximum length or select it using the [?] function. This is a mandatory field.

  2. Click [Search] to display the events.

  3. Select the 'Select' check box against the events that needs to be displayed in the email. On selecting or clearing the check boxes against the headers, events under the headers get selected or cleared respectively.

  4. Click [Apply] to save the format or [Reset] to clear the selected record.
