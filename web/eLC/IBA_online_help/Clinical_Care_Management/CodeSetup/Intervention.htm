Intervention

From this function you can Create Interventions used in planning care delivery and Query for existing records.

The interventions are grouped under an Intervention Type and you can define the characteristics of the intervention like the duration to perform the intervention etc. Optionally you can map the Intervention to various result types.

Creating an Intervention

From the CA Code Setup, select Intervention function.

The Intervention screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Filed Description>

  3. Click [Apply] to save the Intervention or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Intervention records.
