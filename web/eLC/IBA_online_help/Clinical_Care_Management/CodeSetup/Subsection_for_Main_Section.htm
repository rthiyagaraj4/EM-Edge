SubSection for Main Section

From this function you can Create an association of Sections to a Main Section and Query for Sections associated to Main Sections.

Subsection for Main Section function allows defining different types of sub sections for a section defined as ‘Main Section’. The sub sections include all the sections created of type as 'Result Linked', 'Free Format' and 'Template Based'.

Creating an association of Subsections to a Main Section

From the CA Code Setup, select SubSection for Main Section function.

The SubSection for Main Section screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the association of SubSections to Main Section or [Reset] to clear the entered data.

  4. Click [Query] to search for existing ‘SubSection for Main Section’ records.
