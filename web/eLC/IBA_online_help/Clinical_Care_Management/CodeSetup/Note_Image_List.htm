Note Image List

From this function you can Create a new Image and add or upload the image to associate to a Note.

An image is browsed to Upload or Add depending on the mode specified. Images are associated to Sections in the function ‘Image for Section’ and Image linked Sections are displayed in the Clinical Note function on selecting that particular Note Type to which the sections are associated.

Creating a Note Image

From the CA Code Setup, select Note Image List function.

The Note Image List screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Note Image or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Note Images.
