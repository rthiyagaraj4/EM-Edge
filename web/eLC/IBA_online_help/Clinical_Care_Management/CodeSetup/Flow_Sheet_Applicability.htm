Flow Sheet Applicability

From this function you can Create an association of a Flow Sheet to a Practitioner Type, Practitioner and/or Specialty. You can also Query for an existing associated Flow Sheet record and Delete a queried record.

Flow Sheet Applicability function allows association of a Flow Sheet created in the ‘Flow Sheet Component’ to a Practitioner Type, Specialty, Practitioner or a combination of these factors. The flow sheet is displayed for patients based on these restrictions.

Creating Flow Sheet Applicability

From the CA Code Setup, select Flow Sheet Applicability function.

The Flow Sheet Applicability screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Flow Sheet Applicability created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Flow Sheet Applicability records.

  5. Click [Delete] to delete a queried record.
