Patient Class for Intervention

From this function you can Create an association of patient classes to different Interventions and Query for existing records.

Associating Patient Class to Intervention

From the CA Code Setup, select Patient Class for Intervention function.

The Patient Class for Intervention screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Patient Class from the list. This is a mandatory field.

  3. Click [Search] to view the list of interventions.

  4. Select the check box in the column header to select all the interventions or select the check box against the required intervention to be associated to the patient class.

  5. Click [Apply] to save the association of Patient Class to Intervention or [Reset] to clear the entered data.

  6. Click [Query] to search for existing 'Patient Class for Intervention' records.
