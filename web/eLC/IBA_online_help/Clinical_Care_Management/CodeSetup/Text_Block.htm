Text Block

From this function you can Create a new Text Block by making it specific to a Service, Practitioner, Note Type, Section and Procedure or a combination of all these factors. You can Query for a desired Text Block.

A text block is made applicable to a procedure which is used in the ‘Procedure recording’ function. Practitioners use text blocks as Canned Text in a Free Format Section while recording Clinical Notes or Procedures, thus reducing effort required to type descriptive findings.

Creating a Text Block

From the CA Code Setup, select Text Block function.

The Text Block screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Text Block or [Reset] to clear the entered data.

  4. Click [Query] to search for existing ‘Text Block’ records.
