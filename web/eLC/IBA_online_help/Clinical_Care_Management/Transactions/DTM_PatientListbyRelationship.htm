1. Patient Class – Select a Patient Class from the list of valid values. ‘All’ is displayed, by default. The listed values are Inpatient, Outpatient, Daycare and Emergency.

  2. Relationship - Select a Relationship from the list of valid values. ‘All’ is displayed, by default. All the relationships created using the ‘Relationship’ function (CA Code setup) are displayed in the list.

  3. Relationship Period [From-To] – Enter the start and end relationship period in dd/mm/yyyy format. The To Date entered should be greater than or equal to From Date entered.

  4. Patient ID – Enter the Patient ID in alphanumeric format of maximum length as defined in MP Parameter or click [?] function to search the patient ID.

  5. Attending Practitioner - Enter the Practitioner name or click [?] function to search for the Attending practitioner. All Practitioners are displayed for selection on clicking the [?] function.

  6. Specialty - Enter the Specialty or click [?] function to select the specialty. The Specialties associated to the Practitioner is displayed in the search screen for selection.

  7. Visit Type - Select the Visit Type from the list. This field is enabled only if the Patient Class 'Outpatient'. The Visit Types listed for selection are defined in the Visit Type function of Outpatient Management module.

  8. Alt No – Enter the Alternate Number in alphanumeric format of maximum length as defined in MP Parameter.

  9. Group By Patient Class – The check box is selected and disabled, by default. This facilitates the display of patient list in the order of Patient Class. Clear the check box to display patient list without the Patient Class grouping.

  10. Group By Relationship - The check box is selected, by default. This facilitates the display of patient list in the order of Relationship. Clear the check box to display patient list without the Relationship grouping.

Patient List by Relationship

From this function you can View patient list based on logged-in practitioner and patient relationship.

The function lists all patients On selection of this function from the menu, generates a system query for all episode types by default, for all relationship and displays the patient list for the logged-in user in the descending chronological order of Start Date ordered of Relationship and within Encounter Type.

Viewing Patient List by Relationship

From the CA Transactions > Clinician Access > Desktop Menu > Patient List, select By Relationship.

The Patient List By Relationship screen gets displayed.

  1. Enter the search criteria to search for specific patients and click [Search] to display patients based on the entered criteria.

Or

  1. Click [Search] without entering the search criteria to display all patients based on the relationship with the logged in practitioner.

  2. The patients are displayed under five tabs – Outpatient, Daycare, External, Inpatient and Emergency. Click the required tab to view and select patients under the tabs. Outpatient tab is displayed, by default.

  3. Click the Patient Name hyperlink to display the Patient Chart.
