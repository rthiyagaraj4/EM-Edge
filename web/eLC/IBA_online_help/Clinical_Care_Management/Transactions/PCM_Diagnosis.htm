This section is empty, if the patient has no diagnosis (problems) recorded in the facility or enterprise. The diagnoses displayed are only the active diagnosis for the patient that may have been recorded across encounters. The records displayed are grouped by the diagnosis set.

In the event of a sensitive diagnosis being selected for a patient, the system asks for authorization. This authorization is displayed only if the logged in practitioner has the privilege of recording a sensitive diagnosis as defined in the ‘User Access Rights’ master function in Medical Records module. If the privilege is not given to the logged in user, the system displays an error message and restricts the recording of the sensitive diagnosis for the patient. The authorization is validated by means of providing a password. If the password specified is wrong, the system stops the user to continue with the recording. Once the correct password is provided, the description of the sensitive diagnosis is defaulted onto the corresponding field and the user can record this diagnosis for the patient.

On successful completion of the recording, the record is moved to the ‘Current Diagnosis Profile’ section. The sensitive diagnosis is viewed by clicking [Authorize Mode] and providing the correct password. In such an event, the sensitive diagnosis is displayed under the ‘Current Problem’ section. Click [Cancel] to ignore the reason entered and to abort the process.

Enter the password as specified in the function ‘User Access Rights’ to edit or enter the sensitive. After entering the password click [OK] to confirm or click [Cancel] to cancel the current job and the control shifts back to the Record Diagnosis screen with [Authorize mode] changed to [Normal Mode].

The status of a diagnosis recorded for a patient can be changed based on the periodic assessment of the patient. If the identified diagnosis has been resolved for the Patient, the problem can be closed. To change the status of a diagnosis, select the problem record displayed in the ‘Current Diagnosis Profile’ section by clicking on the code or description hyperlink.

Mark Error

A diagnosis recorded can be marked as an erroneous record, if it was recorded by mistake. To mark a diagnosis record as ‘In Error’ choose the diagnosis from the ‘Current Diagnosis Profile’ section. Select the status as ‘In Error’ and records the ‘As on’ date.

On selecting the status as ‘In Error’, the Remarks field becomes mandatory. Enter the Remarks and click [Record] to save the changes.

Diagnosis

From this function you can Record the Diagnosis details of a patient.

This function is used to record the diagnosis details of a patient through various encounters. The system supports ICD10 standard coding scheme. This function provides a comprehensive search option, which enables the practitioners to record the diagnosis with ease. The diagnosis once recorded is allowed to be amended to indicate the current status of the problem for the patient. Once the problem is resolved, the system allows closing the diagnosis. The system maintains an audit trail based on the various amendments made to a diagnosis once it is recorded. This function can also be used for marking a diagnosis as in error, if it had been erroneously recorded, by providing a reason.

Recording Diagnosis

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Outcomes, select Diagnosis.

The Record Diagnosis screen gets displayed.

  1. The screen displays Diagnosis Details already recorded for the patient.

  2. Enter the details in the mandatory (*) fields under the Update Diagnosis Profile. <Related Field Description>

  3. Click [Collapse] to minimize the Update Diagnosis Profile frame or [Expand] to expand the Update Diagnosis Profile frame.

  4. Click [Authorize Mode] to record sensitive diagnosis details.

  5. Click [Record] to record the diagnosis, [Clear] to clear the entered data or [Cancel] to exit from the Record Diagnosis window.

  6. On recording the diagnosis, the details are displayed on the top half of the screen under Current Diagnosis Profile.

  7. Click Diagnosis Description hyperlink to modify the recorded details.

  8. Click [Record] to save the updated record.
