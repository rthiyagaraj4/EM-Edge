1. Period - The date range is displayed, by default. Enter the period in the date format of dd/mm/yyyy.

  2. Normalcy Indicator - Select the Normalcy Indicator from the list. The listed values are Normal, Abnormal and Critically Abnormal.

  3. Result Type - Select the Result Type From the list. The Order Categories for which 'Result Type Display Control' check box is selected in the Order Category function of Order Entry and Result Reporting module are displayed.

  4. Status - Select the Status from the list. The listed values are All, Pending Review and Reviewed. If the status is All then all result records that are pending for review and reviewed are displayed. If the status is Pending Review then only those records that are Pending for Review are displayed and the checkbox in the result listing is enabled. If the status is Reviewed then all reviewed result records are displayed and the checkbox in the result listing is disabled

  5. Event Class - Select the required Event Class from the list.

  6. Priority - Select the Priority from the list. The listed values are Routine, Stat and Urgent.

  7. Limit to Current Encounter - Select the check box to display results of the Current Encounter only. The check box is disabled, by default.

  8. Order By - Select the value from the list. The listed values are 'Encounter' and 'Event'.  This is to group the result listing. This field displays 'Encounter', by default. If the order by is selected as Encounter and Event then the grouping displays the Facility Name, Encounter ID, Encounter date and time, Specialty and Treating practitioner. If the order by is selected as Event then the grouping displays only the event details.

Results

From this function you can View,  Review and Forward Results that are recorded for patients.

Using this function, practitioners can view and review the results. Based on the results the practitioner can take decisions on patient treatment. Only the practitioners with authorization rights to review results can review the results from the Patient chart menu.

Viewing Results

From the CA Transactions > Clinician Access > Desktop Menu > select required patient, Patient Chart > Comparative View,  select the Results function.

The View Results screen gets displayed.

  1. Enter the query criteria and click [Search] to display specific orders for Review. Click [Clear] to clear the displayed data.

  2. Click icon to view the past results and compare with current result. A message 'Do you want to see Abnormal Result Only?' is displayed. Click [OK] to view the Abnormal Results in the Flow sheet. Click [Cancel] to view the Flow sheet of results recorded. Click Graph icon to view the graph of the results recorded.

  3. Select the check box against the required Results. Remarks hyperlink is displayed. Click the remarks hyperlink to enter the Remarks. Click [OK] to save the Remarks and exit from the Review Remarks screen. To Review all the displayed orders, select the check box provided in the heading.  The review Remarks screen is displayed to enter the Remarks. Enter the Remarks and click [OK] to save the remarks and exit from the screen.

  4. Click [Complete Review] to complete the review or click [Forward] to forward the result to another practitioner.

  5. Click [Forward] to display Forward to practitioner screen. Enter the name of the practitioner in the Search field and click [Search] to search for the required practitioner. Select the check box against the required practitioners to forward the result. Click [OK] to save forwarding the Order or [Close] to exit from the screen. Click [OK] and a message is displayed 'Do you want to Complete the Review?' Click [OK] on the message box to complete review.
