Death Register (V)

From this function you can View the death registration details of the patient.

This function displays death registration details as recorded in the Maintain Death Register function of Clinician Access module.

Viewing Death Register

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Patient Details, select Death Register (V).

The View Death Register screen gets displayed.

The screen displays the Death Registration Details and Diagnosis Details, which includes the cause of death.
