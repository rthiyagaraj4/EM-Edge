Problem List (V)

From this function you can View the problem list of a patient.

This function is used to view all problems related to the health of a patient that has been recorded on current or previous encounters.

Viewing Problem List

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Event Details, select Problem List (V).

The View Problem List screen gets displayed.

  1. The screen displays all the active problems recorded for the patient. Enter the search criteria to view specific problems.

  2. Click [Search] to view problems based on the entered criteria or [Clear] to clear the entered data.

  3. On searching, the details of the diagnoses/problems recorded are displayed. Select the Description hyperlink of the desired diagnosis to view details like Onset, Close, Encounter Dates, Practitioner, Facility, Location, Service, Recorded Date and Recorded/Closed By recorded for the problem. Place the cursor over icon to view complaint details.

 

If any sensitive diagnosis is recorded for the patient, an [Authorize Mode] is displayed. On clicking the [Authorize Mode], the practitioner is prompted to enter the password and on successful entry of the password, the practitioner is allowed view the recorded sensitive diagnosis details of the patient.
