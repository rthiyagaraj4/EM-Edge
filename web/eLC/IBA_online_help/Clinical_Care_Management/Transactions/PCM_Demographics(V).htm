Demographics (V)

From this function you can View the demographic details of a patient.

The demographic details can only be viewed from this function and the details displayed are as recorded while registering the patient or any updates made in the Master Patient Index module.

Viewing Demographics

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Patient Details, select Demographics (V).

The Demographics View screen gets displayed.

The screen displays the following tabs:

• Demographic Details

• Contact Addresses

• Related Contacts

• Documents and Others

• Who Done It

Click the tabs to view patient details as recorded for the selected patient.
