Consultation Parameter Setup

From this function, you can Define the parameter setup for Consultation functions.

Defining Consultation Parameter Setup

From the CA Code Setup > Consultation Setup, select Consultation Parameter Setup function.

The Consultation Parameter Setup screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Apply] to save the Consultation Parameter Setup or [Reset] to clear the entered data.
