Select the Patient Medical Report using [?] function. The [?] function displays Note Type created in the Note Type function of CA Code Setup with Note Groups applicable to Patient Medical Report.

  1. Search By – Select Practitioner Name or Practitioner ID radio button.

  2. Search Criteria – Select the Search Criteria from the list. The listed values are Starts With, Ends With and Contains.

  3. Search Text – Enter the search text to search for the practitioners based on the search criteria specified.

Print/Mail Privilege for PMR

From this function, you can Assign privilege for an user to print or email a Patient Medical Report.

Assigning Print/EMail Privilege for PMR

From CA Code Setup > Patient Medical Report Code Setup, select Print/Mail Privilege for PMR function.

The Print/Mail Privilege for PMR screen gets displayed.

  1. Select Privilege For as Print or E-mail.

  2. Select Patient Medical Report.

  3. Enter the Search criteria to search for Practitioners.

  4. Click [Search] to display Practitioners. The Practitioner ID and Practitioner Name of Practitioners are displayed.

  5. Select the check box against the required Practitioners to assign privilege to Print or E-mail the Patient Medical Report.

  6. Click [Apply] to save the privileges assigned or [Reset] to reset the screen.
