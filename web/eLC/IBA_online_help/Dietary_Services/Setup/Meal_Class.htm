Meal Class

From this function you can Create a Meal Class at the Kitchen level of a hospital and Query for the existing records.

The Meal Class helps in recording various types of meal classes pertaining to a hospital. It represents the different categories of people either the patients, staffs or any visitors to the hospital to whom the meal is served. For example, a VIP visiting a hospital is given a special meal. This helps in tracking the number of meals required for the different class of people.

Creating a Meal Class

From the Setup, select the Meal Class function.

The Meal Class screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a created Meal Class or [Reset] to clear the entered details.

  4. Click [Query] to search for the existing records.
