Daily Meal Statistics

From this function you can Query and view the daily meal statistics of a kitchen in the facility.  

The Daily Meal Statistics function is used to query and view the daily meal statistics based on the Kitchen and the meal type. The query result displays the statistics of the number of meals prepared in the kitchen for a specific meal type. The number of meals supplied to the attendants in the facility is also displayed.

Querying Daily Meal Statistics

From the Queries, select the Daily Meal Statistics function.

The Daily Meal Statistics screen gets displayed.

  1. Select a Kitchen from the list. The Daily Meal Statistics is displayed for the selected Kitchen. This is a mandatory field.

  2. Select the Meal Type from the list. The Daily Meal Statistics is displayed for the selected Kitchen and Meal Type. This is a mandatory field. The values are sorted based on the order of the meal type defined in the Meal type screen.

  3. Click the Calendar function to select the Serving Date. The Daily Meal Statistics is displayed for the selected Kitchen, Meal Type and Serving Date. The system date is displayed as the Serving Date, by default.

  4. Click [Select] to search for the Daily Meal Statistics.

  5. The name of the Nursing Unit to which the meals are served, diet type, total number of meals served and the number of meals served to the attendants are displayed.
