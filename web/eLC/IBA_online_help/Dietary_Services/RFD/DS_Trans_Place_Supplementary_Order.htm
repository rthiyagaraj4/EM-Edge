Enter the following Details

  1. Nursing Unit - Select a valid Nursing Unit from the list. The values defined using the ‘Nursing Units’ function in Inpatient & Day Care Management module get listed. This is a mandatory field.

  2. Admn. Date – Select the admission date range using the Calendar function to select patients admitted to the facility during the specified date range.

  3. Patient ID - Select the Patient ID using [?] function. You can place diet order for the respective patient or view the recorded diet order.

  4. Encounter ID – Enter the Encounter ID to search for the patient.  

  5. Practitioner – Select the Practitioner using [?] function to view the diet orders placed by the Practitioner. Practitioners are associated to the selected Nursing Unit in ‘Practitioner for Nursing Unit’ function in Inpatient & Day Care Management module.

  6. Gender - Select the gender from the list. The system-defined values are male, female and unknown.

  7. Status – Select a value from the Status drop down. Based on the selected value, the diet order is displayed. The listed values are ‘All, ‘Processed by Kitchen’, ‘Rejected by Kitchen’, ‘Accepted by Patient/Attendant’ and ‘Rejected by Patient/Attendant’. By default the value ‘All’ is displayed.
