Generate Meal Plan

You can re-generate a meal plan for a patient using this tab.

 

Enter the following details to generate a Meal Plan:

 

  1. Serving Date [From-To] – Select the serve date range using calendar. This is a mandatory field.

The error message ‘Serving ‘To Date’ should not be less than Current Date’ is displayed if the entered Serving To date is less than current date.

  1. The error message ‘Serving Date Range should be maximum 7 days’ is displayed if the entered Serving date range is greater than 7 days in Generate Meal Plan tab.

  2. Diet Category - Select the Diet Category from the list. The system-defined values are Enternal, Normal and Therapeutic Diet. The Meal Plan is generated for the selected Diet Category.

  3. Kitchen - Select a Kitchen from the list. The values defined using the ‘Kitchen/Kitchen Area’ function get listed. The Meal Plan is generated for the selected Kitchen.

  4. Meal Type - Select a Meal Type from the list. The values defined using the ‘Meal Type’ function get listed. The Meal Plan is generated for the selected Meal Type associated to the kitchen for the selected date. The values are sorted based on the order of the meal type defined in the Meal type screen.

  5. Feed Type – Select the required Feed Type from drop-down list. The list displays the values defined in the Feed Type master.

  6. Nursing Unit - Select a Nursing Unit from the list. Nursing Units are defined in Inpatient and Day Care Management module. The Meal Plans are generated for the selected Nursing Unit.

  7. Meal Class - Select the Meal Class from the list. The values defined using the ‘Meal Class’ function get listed. The generated Meal Plan is applicable for the selected Meal Class.  

  8. Frequency Based – Select the check box to generate the Frequency Based Meal Plan. Clear the check box if the meal plan is non-frequency based. The check box is selected, by default.

  9. Regenerate Meal Plan – Select the check box, if the selected meal plan is to be re-generated for the selected date. The Meal Plan is re-generated on clicking [Run]. You cannot re-generate the Meal Plan, if the check box is cleared.

  10. Diet Type - Select the Diet Type from the list. The values defined using the ‘Diet Type’ function get listed. The Meal Plan is generated for the selected Diet Type.  You can select more than one diet type.

  11. Patient ID – Enter the Patient ID in alphanumeric format of ten characters maximum length or click [?] function to select a Patient ID. The Meal Plan is generated for the selected Patient.

Meal plans are generated for the given criteria, if the values are selected. Meal plan is generated for all Diet Orders, if the Serving Date is selected.
