Enter the following details to create Menu:

 

  1. Menu – Enter the Menu in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Long Description – Enter the long description of the Menu in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description of the Menu in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Diet Type - Select the Diet Type from the list. The values defined using the ‘Diet Type’ function get listed. This is a mandatory field.

  5. Meal Type - Select the Meal Type from the list.  The values defined using the ‘Meal Type’ get listed. This is a mandatory field. The values are sorted based on the order of the meal type defined in the Meal type screen.

  6. Meal Class - Select the Meal Class from the list. The values defined using the ‘Meal Class function get listed. This is a mandatory field.

  7. Effective Date Range [From-To] – Select the date range within which the Menu is effective using the Calendar function. This is a mandatory field.

  8. Enabled - Select the check box to enable the Menu in the transaction functions. The check box is selected, by default.

 

Items for Menu

  1. Item Srl - Enter the Serial Number of the Menu Item in numeric format of two digits maximum length. This is a mandatory field.

  2. Item Type - Select the Item Type from the list. The values defined using the ‘Item Type’ function get listed. This is a mandatory field.

  3. Item - Select the Item using [?] function. The values defined using the ‘Food Items’ function gets listed. This is a mandatory field.

  4. Item Qty - Enter the Item Quantity of the Menu in numeric format of six digits maximum length. The Serving Unit of Measurements defined for the food item in ‘Food Item’ function get displayed next to the quantity of the food item. This is a mandatory field.
