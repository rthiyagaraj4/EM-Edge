Document Meal Served

The meal served to the Staffs is documented for future reference. The details on the Meal Served are recorded for the  meal orders that are placed and served.

 

Enter the following to create Document Meal Served:

 

  1. Diet Request Location – Select the Diet Request Location from the drop down list. Based on the master setup, the location code is selected as “Default diet request location by default. The values defined in the masters are listed in Meal Order.

  2. Order Date From – Click the Calendar function to select the ‘From’ date from when the meals served to staff is to be documented. This is a mandatory field.

  3. Order Date To – Click the Calendar function to select the ‘To’ date upto when the meal served to staff is to be documented.   

  4. Meal Category - Select a Meal Category from the list. The system-defined values are Doctor On-Call, OT/Paramedic Staff, On Call Staff, Staff and Students. This is a mandatory field.

  5. Practitioner Name – This field is displayed on selection of ’Doctor On-Call’ in Meal Category field. Select the required practitioner name from the list. This is a mandatory field.

  6. Diet Type - Select the Diet Type from the list. The values defined using the function ‘Diet Type’ get listed. The Meals served to Staffs are documented for the selected Diet Type.  This is a mandatory field.

  7. Meal Type - Select the Meal Type from the list. The values defined using the function ‘Meal Type’ get listed. The Meals served to Staffs are documented for the selected Meal Type. This is a mandatory field.

  8. No. of Meals – On selecting the date range, meal category, diet type and the meal type, the number of meals associated to the respective Meal Order gets displayed. You cannot modify the quantity. This is a mandatory field.

  9. No. of meals served – Enter the Number of Meals served to the staff in numeric format of six digits maximum length. This indicates the number of meals served against the received Meal Orders.

  10. Total No of meals received – Enter the total number of meal received in numeric format of six digits maximum length.

  11. No of meals received - Enter the number of meals received in numeric format of six digits maximum length. This field is mandatory

  12. Total No of meal rejected - Enter the total number of meals rejected in numeric format of six digits maximum length.

  13. No of meals rejected  - Enter the number of meals rejected in numeric format of six digits maximum length. This field is mandatory.
