Meals

The menu for each diet type is displayed. Based on the Diet Category selected in ‘Diet Order for Patient Attendant’, regular menu details get listed. Click to add or edit items in the selected menu.

 

Edit Menu

 

  1. Menu For Date – Click the Calendar function to select a date. The system date is displayed, by default. You can view, edit or add items to the selected menu within the date range specified in ‘Diet Order for Patient Attendant’. Select a date within the date range and click [Next] to view the recorded Menu for the Meal Type for the selected date. If you select a date after the specified date range, you get an error message ‘Date cannot exceed to Date’.

  2. Next – Click the hyperlink to view the items listed in the consecutive pages based on the Diet Cycle selected.

  3. Diet Cycle – The Diet Cycle for the selected date is displayed. You can select the Diet Cycle from the list and click [Go] to view the Menu for the selected date and Diet Cycle. You can modify the Diet Cycle for a patient by specifying the number. The values displayed are based on the Menu Cycle defined for the selected date in ‘Menu Cycle’ function. You can only select the value from the specified list.

  4. Item – The menu items get displayed based on the Diet Category selected. You can update the item quantity and add items to the selected Diet Category.

  5. Qty. - The quantity specified for the respective item is displayed along with the Serving UOM specified for the respective item using the function ‘Food Item’. You can update the quantity based on which the calorie amount is re-calculated. The Unit of Measurement selected in the ‘DS Parameter’ function gets displayed.

  6. Serve - Select the check box, if the items listed for the specified meal type has to be served. Clear the check box, if the items are not to be served to the patient.

  7. Add Item – Click Add Item hyperlink to associate additional food items to the selected Meal Type. Associate items to the Meal Type using the ‘Add Item To’ option at the bottom of the screen. On clicking the ‘Add Item hyperlink under a specific menu, the name of the Meal Type, where the food item has to be listed is displayed adjacent to the ‘Add Item To’ field.

  8. Add Item To – On clicking Add Item hyperlink, the name of the Meal Type, where the food item is to be added is displayed.

  9. Select Item – Select a food item using [?] function. The values defined using the ‘Food Items’ function get listed.

  10. Qty - Enter the quantity of the item to be served in numeric format. You can modify the quantity under the meal type.

  11. [ADD] – Click [ADD] to associate the food item to the Meal Type. The food item selected gets listed under the respective Meal Type.

  12. [Update] – Click [Update] to save the modified menu details. The additional items get listed under the specific Meal Type.

If the patient dislikes a food item, it is displayed in red color. Click  to collapse the Meals option.

The calorie grand total of all the Meal Types is calculated and displayed.
