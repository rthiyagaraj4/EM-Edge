Enter the following details to create Menu Cycle:

 

  1. Diet Type - Select a Diet Type from the list. The values defined using the ‘Diet Type’ function get listed. This is a mandatory field.

  2. Meal Class - Select the Meal Class from the list. The values defined using the ‘Meal Class’ function get listed. This is a mandatory field.

  3. Menu Schedule Cycle – Select a Menu Schedule Cycle from the list. The system-defined values are from 1-14.  The number defined indicates the respective Menu of a Meal Type applicable for the Reference day selected based on the Meal Plan Date. For example, if Menu Schedule Cycle is defined as 3 and the Reference Day is selected as 2 there are three types of Menu available for a particular Meal Type out of which the third Menu of a particular Meal type is applicable for the second day based on the Meal Plan Date.

  4. Meal Plan Date – Select a date for the Meal Plan using the Calendar function.  The Reference Day Number is calculated based on the Meal Plan Date. For example, if we select Menu Date as 16-03-2009 and Reference Day as ‘2’, the Menu Schedule is applicable from the second day that is 17-03-2009. That is, on 17-03-2009 the third Menu is applicable. On 18-03-2009 the Second Menu is applicable and on 19-03-2009 the first Menu is applicable. The cycle repeats the circle after 3 days.

  5. Reference Day No. – Select a Reference Day Number from the list. The system-defined values are from 1-14.  On selecting a Reference Day Number, the system calculates the day based on the Meal Plan Date. For example, if the Menu Schedule is 3 and Reference Day is 2, the  Menu Schedule Cycle is defined for the third Menu of a particular Meal Type from the second day based on the Meal Plan Date. Next day, the next Menu of the same Meal Type is applicable. On the third day the first Menu of the same Meal Type is applicable. You cannot define the Reference Day Number lesser than the Menu Schedule Cycle.

  6. Enabled - Select the check box to enable the Menu in the transaction functions. The check box is selected, by default.
