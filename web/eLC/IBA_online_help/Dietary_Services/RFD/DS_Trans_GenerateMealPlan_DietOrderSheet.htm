Enter the following details

 

  1. Serving Date – Select the Serving Date from the calendar function. The system date is displayed by default. This is a mandatory field.

  2. Diet Category - Select the Diet Category from the list. The system-defined values are Enternal, Normal  and Therapeutic Diet. The Meal Plan is generated for the selected Diet Category.

  3. Kitchen - Select a Kitchen from the list. The values defined using the ‘Kitchen/Kitchen Area’ function get listed. The Meal Plan is generated for the selected Kitchen.

  4. Meal Type - Select a Meal Type from the list. The values defined using the ‘Meal Type’ function get listed. The Meal Plan is generated for the selected Meal Type associated to the kitchen for the selected date.

  5. Meal Class - Select the Meal Class from the list. The values defined using the ‘Meal Class’ function get listed. The generated Meal Plan is applicable for the selected Meal Class.  

  6. Diet Type - Select the Diet Type from the list. The values defined using the ‘Diet Type’ function get listed. The Meal Plan is generated for the selected Diet Type.  You can select more than one diet type.

  7. Nursing Unit - Select a Nursing Unit from the list. Nursing Units are defined in Inpatient and Day Care Management module. The Meal Plans are generated for the selected Nursing Unit.

  8. Patient ID – Enter the Patient ID in alphanumeric format of ten characters maximum length or click [?] function to select a Patient ID. The Meal Plan is generated for the selected Patient
