Enter the following details to create an Item Type:

 

  1. Item Type – Enter the Item Type in alphanumeric format of four characters maximum length. The Item Type has to be unique and cannot be changed after creation. This is a mandatory field.

  2. Long Description - Enter the long description of the Item Type in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description of the Item Type in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Applicable Diet Category - Select the Diet Category from the list. The values get populated from the Diet Category function. This is a mandatory field.

  5. Commercial Formula – Select the check box to identify the Commercial Formula of the selected Item Type.   The food items get listed in the ‘Commercial Formula’ function. Clear the check box, if Commercial Formula is not applicable for the selected Item Type. On clearing the check box, the item type does not get listed in the ‘Commercial Formula’ function.

  6. Enabled – Select the check box to enable the Item Type in the transaction functions. The check box is selected, by default.
