Enter the following details to record the diet order for attendants:

 

  1. Date [From-To] - Click the Calendar function to set the date range for which the Diet Order is to be placed. The system date is displayed as the ‘From’ date, by default. The date range is restricted while placing a new diet order or modifying an existing order for Inpatients based on ‘Restrict Diet Order Date Range for Inpatients’ parameter configuration done in the DS parameter for facility screen.

  2. Meal Class - Select the Meal Class from the list. Meal Classes are defined in ‘Meal Class’ function. This is a mandatory field.

  3. Diet Type - Select the Diet Type from the list. On selecting the Diet Category, the Diet Types associated to the Diet Category in ‘Diet Types’ function gets listed in alphabetical order . This is a mandatory field.

  4. Attendant Instruction – Enter the attendant instructions in character format of 120 characters maximum length.

  5. ADD ATTENDANT – Click ‘ADD ATTENDANT’ to place diet order for attendants. You can add any number of attendants. If you select the same Date, Meal Class and Diet Type  and try to place another order for an attendant, an alert message ‘This Combination is already exists, Do you want to Continue?’ is displayed. Click [OK] to place the order or [Cancel] to exit from the alert message to select a different date or Meal Class or Diet Type. The selected order is displayed in a tabular format – Period, Meal Class and Diet Type.

  6. REMOVE ATTENDANT – Click 'REMOVE ATTENDANT' to remove the placed diet order for attendants.
