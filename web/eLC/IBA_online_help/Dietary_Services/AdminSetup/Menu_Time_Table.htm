Menu Time Table

From this function you can Define a Menu Time Table for a kitchen in the facility. You can Query for a specific Menu Time Table.

The Menu Time Table helps in defining menu timetable to store information regarding the menu for a particular Diet Type, Meal Class and Meal Type prepared in a Kitchen. The Menu schedule for the selected Meal Class is defined using the Menu Cycle function. This function helps you to maintain a timetable on the entire menu’s in the facility, so that, the information regarding meals to be prepared is listed out previously. Accordingly, the purchases of food items for the preparation of food are also planned in advance. The Menu Time Table is used during generation of Meal Plans for a specific kitchen using ‘Generate Meal Plan’ function in transaction.

Defining a Menu Time Table

From the Admin Setup, select the Menu Time Table function.

The Menu Time Table screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a created Menu Time Table or [Reset] to clear the entered details.

  4. Click [Query] to search for the existing records.
