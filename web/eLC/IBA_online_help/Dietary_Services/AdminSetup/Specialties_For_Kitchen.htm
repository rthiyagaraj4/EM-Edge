Specialties For Kitchen

From this function you can Associate Specialty with a particular kitchen within a Facility. You can Query for a specific Kitchen associated to one or more Specialties.

The Specialties For Kitchen function helps in maintaining multi-kitchen environment. This function helps in associating Specialty to the required kitchens for routing the outpatient meal orders to respective kitchens, if a specific kitchen is allotted to cater exclusively for selected Specialties.

Associating Specialties to a Kitchen

From the Admin Setup, select the Specialties For Kitchen function.

The Specialties for Kitchen screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Kitchen Code from the list. The values defined in the Kitchen/Kitchen Area function get listed. This is a mandatory field.

  3. On selecting a Kitchen Code, the Specialties in the facility are displayed.

  4. Select the check box against the required Specialty to associate it with the selected kitchen. The check boxes are selected by default.

  5. Click [Apply] to associate the Specialties with the Kitchen or [Reset] to clear the entered details.

  6. Click [Query] to search for existing records.
