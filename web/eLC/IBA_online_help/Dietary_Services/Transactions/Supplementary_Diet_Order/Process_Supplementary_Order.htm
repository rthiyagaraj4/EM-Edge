Process Supplementary Order

From this function you can Process supplementary food orders already placed in the system.

The Process Supplementary Order  function is used to allow processing supplementary food orders.

Processing Supplementary orders

From the Transactions, select the Supplementary Diet Order > Process Supplementary Order function.

The Process Supplementary Orders screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to search the supplementary order placed.

  3. In the search results column, select the Status from the drop down and enter the remarks against the required supplementary order.

  4. Click [Apply] to save a processed supplementary food orders or [Reset] to clear the entered details.
