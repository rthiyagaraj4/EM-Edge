Place Supplementary Order

From this function you can Place supplementary orders for Inpatients and their Attendants in a particular Nursing Unit. You can Update the orders placed for a patient. .

The Place Supplementary Order function is used to place supplementary orders from a ward to a kitchen for inpatients requiring meals.

Placing Supplementary orders

From the Transactions, select Supplementary Diet Order > Place Supplementary Order function.

The Place Supplementary Order screen gets displayed.

  1. Enter the details in mandatory (*) fields. <Related Field Description>

  2. Click [Search] to display the patient records based on the entered search criteria.

  3. Click the Patient ID Hyperlink. The Place Supplement Order screen screen gets displayed.

    Enter the following Details

    1. Supplementary Order For – Select ‘Both’ or ‘Patient’ or ‘Attendant’ from drop down list. By default, the value ‘Both’ is selected.

    2. Date [From-To] - Click the Calendar function to set the date range for which the Diet Order is to be placed. The system date is displayed as the ‘From’ date, by default.

    3. Meal Type - Select the Meal Type from the list. The values defined using the ‘Meal Type’ function get listed.

    4. Item Type - Select the Item Type from the list. The values defined using the ‘Item Type’ function get listed.

    5. Food Item - Select the Food Type from the list. The values defined using the ‘Food Type’ function get listed.

    6. Qty – Enter Quantity of the items in numeric format. You can enter only six numeric characters. This is mandatory field.

    7. Instructions – Enter the instructions in alpha numeric format of maximum 255 characters limit.

    8. Select - Click Select to display the supplementary orders based on the search criteria.

    9. View All - Click View All hyperlink to invoke the ‘Supplementary Order Audit Trail’ dialog. This dialog displays Date, Supplementary Order for, Item Type, Food Item, Qty, Meal Type and Instructions of the order

  4. Select the checkbox against the patient records for which the supplementary order has to be placed.

  5. Click [Apply] to save a Diet Order or [Reset] to clear the entered details.
