Generate Meal Plan

From this function you can Generate meal plan for Inpatients and Discharged patients on a particular date and efficiently plan the food preparation activities of a kitchen. You can Print the meal diet list for a specific date.

The Generate Meal Plan function is used to generate default diet order for patients in a facility. You can specify the patient ID to generate a meal plan for a patient who arrives late at the facility. Orders can also be placed for attendants. This function helps in generating meal plan for a patient, for a Kitchen or a Nursing Unit. The system generates unique meal list number to track main, supplement kitchens and provide reprint options by List Number. You can also print supplement list for late admissions.

Generating a Meal Plan

From the Transactions, select the Generate Meal Plan function.

  1. The Generate Meal Plan screen gets displayed with three tabs. Generate Meal Plan tab is enabled, by default.

  1. Click the Calendar function to select the Serving Date. The system date is displayed, by default. You can generate a meal plan for inpatients and discharged patients or print the meal diet list for the specified date. This is a mandatory field.

  2. Click [Run] to generate the Meal Plan or print the meal diet list.

The Meal Plans can be generated for the number of days specified in the DS Parameter.

Meal Plans are generated based on the serving date and the maximum number of days defined in the ‘DS Parameter for Facility’ function.

The Next Meal Plan Number is generated based on the number defined in the ‘DS Parameter for Facility’ function.   

The List number gets generated for each patient record, if the patient ID is specified while generating the Meal Plan for late admissions. The list number can be viewed only on selecting the ‘List Type’ as ‘Specific list’.
