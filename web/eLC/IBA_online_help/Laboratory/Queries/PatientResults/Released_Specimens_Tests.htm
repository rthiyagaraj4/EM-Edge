Released Specimens/Tests

From this function you can Query and Print the released specimens/tests for a given patient number.

This function is used to query the released specimens for specific date range or for any patient number. You can also query for the Released specimen or test with Critical Results.

Querying Released Specimens

From Query > Patient Results, double-click the Released Specimens/Tests function.

The Released Specimens/Tests screen gets displayed.

  1. Select the Released Date From and Released To Date from the calendar. Current system date and time gets displayed by default. This is a mandatory field.

  2. Enter the Patient ID or right-click on the ‘Right Click for User Options’ function to search for a patient record using the ‘Hospital Patient Lookup’ or the ‘External Patient Lookup’ screen. Double-click the Patient ID in the displayed list to select the patient record.

  3. Click ‘Tab’ to view the patient details.

  4. Enter the details in the relevant fields. < Related Field Description>

  5. Click [Display] to view the released specimen records based on the entered query criteria.

  6. Click [Patient Details] to view the patient details of the patient.

  7. Click [Result] or double click on specimen number to view the patient results.

  8. Click [Report] to print the report.
