Source

From this function you can Query the status of all the registered requests and their results for a given source and a period.

The reports for all the tests are shown in the print format report in the screen. This function assists the laboratory personnel in answering queries from various sources regarding the request status.

Querying Source details

From Query > Patient Results, double-click the Source function.

The Source screen gets displayed.

  1. Select the Source Type from the list.

  2. Enter the Source Code or select the Source Code using [List of Values] function. The listed values are defined in Source Code master.

  3. Enter the Source Date Range [Form-To] or select the date using Calendar function. Current system date is displayed, by default.

  4. Click ‘Tab’ to view the list of specimens. Select the Specimen number and click [Result] to view all the results of the specimen with their current statues.
