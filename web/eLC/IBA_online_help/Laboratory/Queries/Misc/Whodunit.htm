Whodunit

From this function you can Query and Print the details for all the activities for a given specimen number.

Whodunit query is used to audit the work carried out by different laboratory personnel for a given specimen number. The typical activities include result entry and result reviews. This inquiry is based on the result log maintained by the system for various activities of a laboratory department.

Querying Whodunit details

From Query > Miscellaneous, double-click the Whodunit function.

The Whodunit screen gets displayed.

  1. Enter the specimen number for which the result activity log is to be queried. This is a mandatory field.

  2. Select the ‘Print Report?’ check box to print the specimen details.

The details of all the result activities for the tests are displayed. The query displays the various stages for each tests, the  User ID of the person who performed the process, the date and time when test was performed, the status of the test and the results for the particular stage.
