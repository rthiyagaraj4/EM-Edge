Referral Patient Lookup

From this function you can Query the patient registration details .

The Referral Patient Lookup query function is used to query the patient registration details based on the patient number assigned by the referral source.

Querying Referral Patient Lookup

From Query > Miscellaneous, double-click the Referral Patient Lookup function.

The Referral Patient Lookup screen gets displayed.

  1. Enter the Referral Patient ID or select the Referral Patient ID using [List of Values] function.

  2. On selecting the Referral Patient ID, details of the patient associated to the Referral ID are displayed.

    The following details are displayed:

    1. Patient ID – The patient identification number is displayed.

    2. Name – The name of the patient is displayed.

    3. Referral Source Code - The referral source code is displayed.

    4. Referral Source Name – The Referral source name is displayed.

    5. Sex – The gender of the patient is displayed.
