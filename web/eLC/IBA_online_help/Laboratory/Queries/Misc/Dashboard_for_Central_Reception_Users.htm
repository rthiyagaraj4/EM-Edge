Dashboard for Central Reception Users

From this function, you can Query the status of orders such as Orders Placed but Sample not collected, Order Collected but not dispatched, Sample dispatched but not received in lab cases.

The Dashboard allows you to query the status of orders as follows:

• Orders Placed but Sample not collected

• Order Collected but not dispatched

• Sample dispatched but not received in lab cases.

• Ordered and not registered by the Lab

The Dash board output gets refreshed in a defined time interval, where the time interval for refreshing the data is entered in ‘Auto Refresh Parameter’ in the Parameter for Facility screen.

Querying Dashboard for Central Reception Users

From Query > Miscellaneous, double-click the Dashboard for Central Reception Users function.

The Dashboard for Central Reception Users gets displayed.

  1.  Enter the Order Date From and To Date in the respective fields. This field is mandatory.

• The system defaults seven days before the current date/system date in the Order From Date.

• The system displays the current date/system date in the Order Date To.

• The Order From and To Date range cannot be more than two months.

  1. Select the required Order Type. By default, the system defaults the order Catalog of the default Department of Lab.

  2. Select the required order status to be queried from the ‘Select’ drop-down list. By default, the value ‘All’ is selected. The list values include:

• Order Placed Sample Not Collected

• Order Collected but not Dispatched

• Sample Dispatched but not registered by the Lab

• Ordered and not registered by the Lab

• All

  1. Click [Display/Refresh] button to view the all the records matching the query criteria. The dashboard output records get refreshed based on the value entered in ‘Auto Refresh Parameter’ in the Parameter for Facility screen.

  2. The query results include Order ID, Order Date, Specimen No, Order Catalog, Order Status, Patient ID, Name and Order Type.

  3. The status of the orders is displayed with their count such as ‘No of Orders Not Collected’, ‘No of Orders Collected but not Dispatched’, ‘No of Orders Dispatched but not Registered in Lab’, ‘No of Order Placed/Collected but not Registered in Lab’, and ‘Total Number of Records’.

When ‘Collection Stage’ is marked as 'Separate' or 'With Order' and order is placed, the Dashboard shows the orders status appropriately.
