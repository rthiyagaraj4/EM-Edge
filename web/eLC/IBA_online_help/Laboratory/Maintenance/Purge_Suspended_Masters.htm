Purge Suspended Masters

From this function you can Delete the master records, which are suspended.

The master codes already used in certain posted data files cannot be deleted. The system will not delete the master codes if there are records existing in the master codes functions. The problems encountered while deleting the records are recorded in messages and these messages are available for view in ‘Report Purged Masters’ report function.

Purging Suspended Masters

From Laboratory Maintenance, double-click Purge Suspended Masters function.

The Purge Suspended Masters screen gets displayed.

  1. Enter the Master Code value in numeric format or click [List of Values] to select the required Master Code.

  2. Click [Save] or press [F10] to save the created Departments records or click [Clear Record] to clear the entered data. The confirmation is displayed to successfully complete the transaction.

  3. The message ‘Do you want to confirm?’ gets displayed. Click [OK] to purge the selected Master Codes, which are suspended or [No] to select another record.
