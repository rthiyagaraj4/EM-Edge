Departmental Parameters for Facility

From this function you can Define the parameters for an individual laboratory department in a facility. You can  Query for the existing records.

This function is used to define and maintain the parameters for a specific department in a facility. You can select a default service and specimen number for the tests performed in the selected department. The name of the person in charge of the department can be recorded along with the telephone number.  

Defining the Departmental Parameters for Facility

From Administration, select the Departmental Parameters for Facility function.

The Departmental Parameters for Facility screen gets displayed.

  1. The General Tab frame is displayed by default. Enter the data in the mandatory (*) fields of General tab.

  2. Click Print/Fax Previous Results on Request tab to invoke the tab frame. This tab is enabled if ‘Print/Fax Previous Results on Request’ check box is selected in General tab.  Enter the data in the mandatory (*) fields of Print/Fax Previous Results on Request tab.

    Enter the following details in the Print/Fax Previous Results on Request:

    1. Dept code – Click [List of Values] to select the department. The department name associated with the logged in user is displayed. If you have access to multiple departments, select the desired department name from Department list.

    2. Description - The corresponding description for the selected Department code is displayed, by default.

    3. Result within Specified Period (days) - Enter the result for specify period to result the specimen.

    4. Print - Select the check box to print the previous results on request.

    5. Fax - Select the check box to fax the previous results on request.

    6. Mail - Select the check box to e-mail the previous results on request.

  3. Click [Commit] or press [F10] to save the defined records or click [Clear Record] to clear the entered data.

  4. Click [Execute Query] or press [F8] to search for the existing records.
