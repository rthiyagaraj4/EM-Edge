Authorized Users

From this function you can Authorize users to access departments and results in a facility. You can Query for the existing records.

Authorized Users function enables you to permit a user to access the departments of a facility. This function is used to define and maintain the laboratory departmental parameters for a specific facility. You can permit a user to modify results or verify the requests in a department. User is authorized to view confidential results using this function.

Defining the Authorized Users

From Administration, double-click the Authorized Users function.

The Authorized Users screen gets displayed.

  1. The logged in facility name is displayed by default. Select the desired Facility name if you have access to more than one facility. This is a mandatory field.

  2. Click [List of Values] to select a user name. The users’ created in ‘Enterprise Configuration’ module.

  3. The General Tab in ‘Department Access’ tab is displayed, by default. Enter the data in the mandatory (*) fields.

    Enter the following details in the General Tab:

    1. Department – Click [List of Values] to select a department. The department name associated to the logged in user is displayed. If you have access to multiple departments, select the desired department name from Department list. This is a mandatory field.

    2. Description - The description of the selected department is displayed, by default.

    3. Verify - Select the check box to enable user to verify the request.

    4. Modify Result - Select the check box to enable user to modify results in the respective department.

    5. Default Department - Select the check box to make the department as default for the user. At least one department has to be a default department.

    6. Name on report - Select the check box to enable the system to print the Name on the report.

    7. Print Draft Result - Select the check box to enable the Printing of Draft results.

    8. Department - Performing Site Access - Select an access type from the list. The system-defined values are ‘Department and All Perf. Sites’, ‘Department and Specific Perf. Sites’ , ‘All Perf. Sites’ and ‘Specific Perf. Site’. This is a mandatory field.

    9. Patient Chart – Select the check box to allow the user to access Patient Chart menu in the Lab module.

  4. Click Department Wise Sites to authorize users to access sites.

    Enter the following details in the Department Wise Sites tab:

    1. Department – The Department, which is selected under General tab, is displayed, by default.

     

    Performing Site

    1. Site – Click [List Values] to select a site. User is permitted access to the selected site in the department. This is a mandatory field.

    2. Description – The description for the selected Perform Site code is displayed, by default.

  5. Click Workgroup tab to authorize user to access a workgroup.

    Enter the following details in the Workgroup tab:

    1. Department - The Department, which is selected under General tab, is displayed, by default.

    2. Workgroup – Click [List of Values] to select the Workgroup.  User is permitted access to the selected Workgroup in the department.

    3. Description - The description of the selected workgroup is displayed, by default.

  6. Click ‘ Result Access’ tab to authorize user to access results.

    Enter the following details in the Result Access tab:

    1. Department – Click [List of Values] to authorize a user to access a department.  The department name associated with the logged in user are listed. If you have access to multiple departments, select the desired department name from the Department.

    2. Description – The corresponding description for the selected department code is displayed, by default.

    3. View Confidential - Select the check box to enable the user to view confidential results.

  7. Click [Commit] or press [F10] to save the created records or click [Clear Record] to clear the entered data.

  8. Click [Execute Query] or press [F8] to search for the existing records.
