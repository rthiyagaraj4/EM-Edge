General Tab

Enter the following details in the General Tab:

  1. Facility Name – The logged in facility name gets displayed, by default. Select the facility name from the list. This is a mandatory field.

  2. Department - Select the Department from the list. The departments defined for the respective facility gets displayed.

  3. Person In Charge - Enter the name of the person in charge of the Department.

  4. Telephone/Extn - Enter the Telephone or the extension number in numeric format of 15 digits maximum length.

  5. Specimen Request Verification Required - Select the check box to verify the specimens that are registered before further processing.

  6. Max. Pending Requests for Verification - Enter the maximum number of pending requests that can wait for verification In Acknowledge verification function.

  7. Usage of Category No - Select the check box to generate the category number in Acknowledge Verification function.

  8. Generate Category No – This check box gets displayed if you select the Usage of Category No function. Select the check box to generate the category number in Acknowledge Verification function.

  9. User Name on the report - Select the check box to enable the system to print the user name on the report.

  10. User PIN Code Applicable - Select the check box to enter the Pin Code to view the confidential results in Confidential reports.

  11. Need Audit Trail (Who done it) - Select the check box get the audit Trial for Who done it in all queries.

  12. Notify Details in Result Report – Select the check box to print the notification details such as Inform By, Inform To, Inform Date, Notify text in the lab result reports.

  13. Print Test Method in Result Report – Select this Print Test Method in Result Report check box to display the test method in the result report.

  14. Legend for Preliminary Result - Enter the Legend for Preliminary result in alphanumeric format of 40 characters maximum length.

  15. Print Urgent/Stat Reports Separately - Select the check box to print Urgent and Stat reports separately.  

  16. Print Result Report only on Completion - Select the check box to print the test result on completion.

  17. Print Urgent/Stat Reports Immediately - Select the check box to print Urgent/Stat samples immediately.

  18. Print/Fax Previous Results on Request - Select the check box to print and fax the previous results on request.

  19. Print Result at Department - Select the check box to print the result at department.

  20. Print Test Ordered In Report - Select the check box to print the entire test ordered in the report.

  21. Print Request Slip - Select the check box to print request slip at the time of specimen registration.

  22. Print/Display Pending Tests of Profile - Select the check box to print and display the pending tests of profile in Print/Display Pending queries.

  23. Legend for Pending Tests - Enter the Legends for pending tests in alphanumeric format of 40 characters maximum length. This field is enabled on selecting 'Print/Display Pending Tests of Profile' check box.

  24. Print Specimen Label Always on Registration – Select the check box to print specimen label while registering specimen in Blood Donor module and at the time of placing laboratory orders in Clinician Access module. Clear the check to print the specimen label, only while placing laboratory orders in Clinician Access module.

  25. Review Results Using Default Ranges - Select the check box to review the results using default range in Result Entry by Specimen.

  26. Include Clinical History in Result Report - Select the check box to include clinical history in the result report.

  27. Print Results of the entire Specimen in the Final Report - Select the check box to print result of the entire specimen in the final report.

  28. Display Patient Markers - Select the check box to display the patient marker in Result Entry by Specimen function.

  29. Specimen Dispose Period days - Enter the specimen dispose period days in numeric format of two digits maximum length.

  30. Check for Unique Specimen Type in Specimen Registration - Select the check box to review unique specimen type while registering specimen.

  31. Service - Click [List of Values] to select a Service. The listed values are created in 'Application Masters' module. This is a mandatory field.

  32. Message Line - Enter the Message line to be printed on the report in alphanumeric format of 200 characters maximum length.

  33. Footer No.1 - Enter the Footer number one details to be printed on the report.

  34. Footer No. 2 - Enter the Footer number two details to be printed on the report.

  35. Print Titles for Num. Result – Select the check box to print the titles for numeric results.

 

Title for

  1. Test - Enter the test in the alphanumeric format of 40 characters maximum length. This field is enabled on selecting the ‘Print Titles for Num. Result’ check box.

  2. Result - Enter the test result in alphanumeric format of 10 characters maximum length. This field is enabled on selecting the ‘Print Titles for Num. Result’ check box.

  3. Test Units - Enter the test unit in alphanumeric format of 10 characters maximum length. This field is enabled on selecting the ‘Print Titles for Num. Result’ check box.

  4. Result Range - Enter the result range in alphanumeric format of 15 characters maximum length. This field is enabled on selecting the ‘Print Titles for Num. Result’ check box.

 

Specimen Numbering

  1. Specimen Prefix - Enter the number to be used as prefix for a specimen belonging to the department in alphanumeric format of two digits maximum length. The number should not begin with zero.

  2. Generate Specimen No – Select the check box to generate the next specimen number for the selected department in Registration function.

  3. Default Specimen No - Enter the default specimen number in numeric format. The length of the specimen number is based on the ‘Specimen No Length’ specified in ‘Lab Parameter’ function. The specimen series has to begin with the specimen prefix specified for the department through ‘Department’ function. The specimen number is validated in transaction functions based on the specimen number specified. This is a mandatory field.

  4. Next Specimen No – The system will default the next specimen number, by default and this field will be disabled.

  5. Last Specimen No Generated Date – The system will default the last specimen number generated date by default and this field will be disabled.
