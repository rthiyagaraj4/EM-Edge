Enter the following details:

  1. Differential Test – Enter a valid Differential Test Code in alphanumeric format of 10 characters maximum length. The system allows you to setup only one record.

  2. Differential Tests – Enter the individual tests in alphanumeric format of 10 characters maximum length that form the differential group test and are to be counted. Enter the keyboard mapping character for each test (in the adjacent field), absolute and percentage units. This is a mandatory field.

  3. Absolute Units – Select the Absolute Unit code in alphanumeric format of 10 characters maximum length.

  4. Percentage Units – Enter the Percentage Units value for a differential test.

  5. Film Comment Tests – Enter the individual tests in alphanumeric format of 10 characters maximum length that form the differential group test and are to be used for film comments reporting.

  6. Support Test – Enter the support test codes (Individual tests) in alphanumeric format of 10 characters maximum length or select it using [List of Values] function.

  7. Result Component Type – Select for the support test the result component that needs to be displayed from the list box. The system allow you to select either the numeric result or the first comment code. This list will get enabled on selecting support test.

  8. NRBC Code – Enter the test code setup for Nucleated Red Blood Cells (NRBC). This test has to be part of the differential group test and has to have been set up in the individual tests.

  9. WBC Code – Enter the test code setup for White Blood Cell (WBC). This test has to have been ordered and processed either as a standalone test or as part of group test for full blood count on the same specimen. In addition, it has to have been included in the Results of Other tests. Depending on the NRBCs seen, the WBC count is corrected.

  10. PLAT Code – Enter the test code setup for Platelets. This test may have been ordered and processed either as a standalone test or as part of group test for full blood count on the same specimen. It has to have been included in the Results of Other tests. If it is set up, the user has an option of amending its result.

  11. Print Result – The result for the tests forming the differential can be reported based on absolute values computed, as percentage or both. The user has a choice of selecting the components, which are needed.

  12. Minimum Cell count – Enter the minimum number of cells to be counted for differential tests.

  13. Maximum Cell count – Enter the maximum number of cells to be counted for differential Tests.

  14. Actual Cell count – Enter the actual number of cells to be counted for the differential tests.
