Enter the following details:

  1. Department - The name of the department associated to the logged in user is displayed. Select the desired department from list, if you have access to multiple departments.

 

Worklists

  1. Name - The Worklists created for the laboratory department is displayed in the order of work list name, number and date.

  2. Number - The corresponding Worklist Number created for the selected laboratory department is displayed.

  3. Date - The dates of creation of the work list are displayed.

  4. Perform Site - The Perform site code associated to the worklist is displayed.

  5. Description - The corresponding description for the selected Perform Site is displayed, by default.
