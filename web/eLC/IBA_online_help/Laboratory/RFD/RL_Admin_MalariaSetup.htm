Enter the following details:

  1. Department Code - The default department name associated with the logged in user is displayed. Select a department from the list to record details for any other department.

  2. MPC Test – Click [List of Values] to select a test code. The test codes defined as valid for MPC test is listed.  

  3. WBC Code – Click [List of Values] to select a WBC Code. The test codes defined as valid for  WBC gets listed.

  4. Key for Counting - Enter the value in alphanumeric format of one character maximum length. This count is used to represent WBC count during result entry.

  5. Parasites Code – Click [List of Values] to select a parasite. The listed values are created in Test Codes function.

  6. Key for Parasites - Enter the value in alphanumeric format to count the Parasites during resulting.

  7. RBC Code - Click [List of Values] to select a RBC Code. The list displays the values defined in Test Codes function.

  8. Parasite Load % Code - Click [List of Values] to select a test code for parasite load percentage. The list displays the values defined in Test Codes masters function.

  9. Support Tests – Click [List of Values] to select the supporting Tests. The list displays the values defined in Test Codes function.

  10. Result Component Type - Select the Result Component Type from the list. The valid values are 'Numeric Result' and 'Result Comment'.

  11. Film Comment Tests - Enter the individual tests that are to be used for film comments reporting.
