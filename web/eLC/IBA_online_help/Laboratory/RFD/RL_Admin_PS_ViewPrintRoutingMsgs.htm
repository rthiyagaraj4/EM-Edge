Enter the following details:

  1. Print Date – The date and time when the error encountered is displayed. The displayed date and time cannot be modified.

  2. Facility – The facility name for the selected message is displayed, by default.

  3. Module – The name of the module from which, the report is generated is displayed.

  4. User ID - The User ID of the logged in user who started the printing is displayed and the value cannot be changed.

  5. Report Name – The name of the report, which caused the error, is displayed. The displayed report name cannot be modified.

  6. Destination – The routing destination type is displayed. The displayed value cannot be modified.

  7. Work Station – The workstation where the message is logged is displayed.

  8. Machine – The name of the computer where the message is logged is displayed.

  9. Printer ID – The Printer ID of the report routing Printer is displayed.

  10. Printer Name – The name of the printer is displayed.

  11. Report File Name – The name of the report output file is displayed.

  12. Print Queue Name – The name of the print queue is displayed.

  13. Error Text – The error message encountered during the process is displayed.
