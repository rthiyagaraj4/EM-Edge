Specimen Type

You can define the required specimen type and volume of specimen required for the test. This sub tab gets enabled on selecting ‘Specimen Required’ check box under ‘General’ sub tab of ‘Attribute’ tab.

  1. Type – Enter the Specimen Type code or click [List of Values] to select a Specimen Type. The list displays the values defined in 'Specimen Type' function.

  2. Description - The corresponding description for the selected Specimen Type code is displayed, by default.

  3. Volume - Enter the required Volume of the specimen required to carry out the test in numeric format of four digits maximum length.

  4. Unit - Enter the unit code or click [List of Values] to select the required Unit. The list displays the values defined in the Units of Measurement function of the 'System Management' module.

 

Volume

Volume tab enables you to specify the volume range of the specimen to be collected for the test based on the age of the individual. You can specify various ranges of age and volume combinations for each specimen type associated to the test.

 

  1. Spec Type - Specimen Type select in the Specimen Type tab is displayed, by default.

  2. Minimum Age - Enter the maximum age range in Year, Months and Days format.

  3. Maximum Age - Enter the minimum age range in Year, Months and Days format.

  4. Volume - Enter the volume required for the selected test in numerical value of five digits maximum length. This volume is viewed by practitioner while placing and collecting specimen that how much specimen can be accumulated.
