Enter the following details:

Derive Code

  1. Derive Code – Enter the derive code or click [List of Values] the created Derive Code. This code cannot be modified or duplicated. This is a mandatory field.

  2. Department Code - The logged in Department Code is displayed, by default. Select required department from the list, if you have access to more than one department. This is a mandatory field.

  3. Long Description - The long description recorded for the selected derive code is displayed, by default. This is a mandatory field.  

  4. Short Description - The Short Description for the selected derive code is displayed, by default. This is a mandatory field.  

  5. Derive Result for Test - Click [List of Values] to select the result comments. The list displays the values defined in ‘Test Codes’ function. The respective description of the test is displayed in the adjacent field. This is a mandatory field.

 

Value

  1. Derive Result Value - Click [List of Values] to select a result value. The list displays the values defined in Result Comments function. This is a mandatory field.

 

Rule Result Detail

  1. Test Code - Click [List of Values] to select a test code that defines the result of the parent test. The list displays the values defined in ‘Test Codes’ function. This is a mandatory field.

  2. Description -The Corresponding Description of the selected Test Code is displayed, by default.

  3. Value – Enter the result comment code or click [List of Values] to select a result comments from the list. The listed values are created in Result Comment Master. This is a mandatory field.
