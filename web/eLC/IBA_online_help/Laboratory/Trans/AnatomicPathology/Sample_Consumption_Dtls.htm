Sample Consumption Details

From this function you can Record the consumption details of the Samples.

Sample Consumption Details function helps you to record the consumption details of Samples. If all samples are recorded as consumed by mistake in the Grossing function, you can rectify using this function.

Sampling Consumption details

From Laboratory Transactions > Anatomic Pathology menu, double-click the Sample Consumption Details function.

The Sample Consumption Details screen gets displayed.

  1. Select the Department from the list. The logged in Department is displayed, by default. Select required department from the list if you have access to more than one department.

  2. Enter the Specimen Number or click [List of Values] to select the Specimen Number to record the consumption details. The system displays a list of specimens for which the sample details are recorded. Press [Tab] to view the specimen details.

  3. Click [Add Clinician History] to view the clinical history of the patients.

  4. Click [Clinician Comments] to view the clinician comments, which was recorded while registering the specimen.

  5. Click [Order Comments] to view the order comments.

  6. Click [Patient Details] to view the patient details.

  7. Enter the details in the following tab pages.

Text Tab

The Text tab displays the grossing details for the selected specimen number. This tab is selected, by default.

Enter the following details:

  1. Grossing Date Time – The Grossing date and time will be displayed.

  2. Re-Grossing Date Time – The Re-grossing date and time will be displayed.

  3. Code - Click [List of Values] to select the grossing code. This code is created in ‘Reflex Text’ Masters function.

Block Tab

Click ‘Block’ tab to view the created blocks.

  1. Block ID – The Block ID is displayed. which is created in the ‘Grossing details’ function. You cannot create a new block if it is consumed.

  2. Number of Tissues – The Number of the Tissues is displayed which is created in the ‘Grossing’ function.

  3. Description – The description of the Tissue is displayed which is recorded while Grossing.

  4. Click [Created Block] to create a new block. If the block is already consumed you are not able to create a new block.

  1. Click [Requesting Task], if the task is to be requested for the sample received.

  2. On entering the details, click [Save] to save the record or [Clear Record] to clear the entered details.
