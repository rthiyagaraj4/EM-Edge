Requesting Task

From this function you can Request the tasks on samples.

Requesting Task function is used to enter the information as part of requesting the tasks on samples. The samples received in Histopathology need to be prepared before the consultants can study them and report the findings. Different types of tasks need to be performed on the received samples. The system allows recording of tasks for each sample received. You can create blocks, slides on the samples.

Tasks are normally requested for Blocks created on the sample. It is possible to request a task for the sample. In certain cases, if Cytology samples are received as slides, it is necessary to request the task on the Specimen directly. The Requesting Task screen includes of the following tabs: Request Task for Block, Request Task for Sample and Request/Record Task for Specimen. The Request Task for Block and Request Task for Sample tabs are enabled, if task is already requested for that sample.

Requesting Task

From Laboratory Transactions > Anatomic Pathology menu, double-click the Requesting Task function.

The Requesting screen gets displayed.

  1. Select the Department from the list. The logged in Department Code is displayed, by default. Select desired department from the list if you have access to more than one department.

  2. Enter the Specimen Number or click [List of Values] to select the Specimen Number to request task or record task for the specimen. The system displays a list of specimens for which the sample details are recorded. Press [Tab] to view the specimen details.

  3. Click [Add Clinical History] to record clinical history details for the patient.

  4. Click [Clinical Comments] to view the recorded clinical history and clinical comments details of the patient. Click [Close] to return to the Sample Details screen.

  5. Click [Patient Details] to view the details of the Patient.

  6. Click [Print Control Slide Label] to print Slide labels.

  7. Click [Specimen Task Details] to view Specimen, Block or Task details. Click [Close] to return to the Requesting Task screen.

  8. Enter the details in the following tab pages.

Request Task for Block Tab

Click ‘Request Task for Block’ tab and the details of the task requested for the block created for the specimen are displayed.

  1. Sample ID – The Sample ID of the specimen is displayed.

  2. Block – The Block sequence number is displayed.

  3. Task Code – Select the Task Code to request a new task which has to be performed on the received block. The list of values displays the values defined in the ‘Individual Tasks’ masters function.

  4. Description – The description of the selected Task Code is displayed.

  5. Task Assigned to – The list displays the value to which the task is assigned. The listed values are ‘Sample’, ‘Blocks’ and ‘Both’.

  6. Slide Level Code – Select the slide level code from the list. The list displays the values defined in the ‘Slide Level’ masters function.

  7. Reject – Select the check box to reject the sample.

  8. Code – If Reject check box is selected, select the Rejection Code from the list. The list displays the values as defined in the ‘Rejection’ masters function.

  9. [Print Task Slip] – Click [Print Task Slip] to print the task slip for the requested task on the blocks.

Request Task for Sample Tab

Click ‘Request Task for Sample’ tab and the details of the task requested for the samples are displayed.

  1. Sample ID – The Sample ID of the Specimen is displayed.

  2. Task Code – Select the Task Code to request a new task which has to be performed on the received block. The list of values displays the values defined in the ‘Individual Tasks’ masters function.

  3. Description – The description of the Task Code is displayed.

  4. Task Assigned to – The list displays the value to which the task is assigned. The listed values are ‘Sample’, ‘Blocks’ and ‘Both’.

  5. Slide Level Code – Select the slide level code from the list. The list displays the values defined in the ‘Slide Level’ masters function.

  6. Reject – Select the check box to reject the sample.

  7. Code – If ‘Reject’ check box is selected, select the Rejection Code from the list. The list displays the values as defined in the ‘Rejection’ masters function.

  8. [Task Assigned Samples] – Click [Task Assigned Samples] to view the details of samples that has been assigned to the task.

  9. [Print Task Slip] – Click [Print Task Slip] to print the task slip for the request.

Request/Record Task for Specimen Tab

Click ‘Request/Record Task for Specimen’ tab to request task for the specimen.

Enter the following details:

  1. Task Code – Click [List of Values] to select the task to be performed on the specimen. The list displays the values defined in ‘Individual Tasks’ masters function.

  2. Description – The description of the Task Code is displayed.

  3. Task Completed – Select the check box if the task is completed as part of requesting.

  4. Slide Level – Select the slide level code from the list. The list displays the values defined in the ‘Slide Level’ masters function.

  5. Reject – Select the check box to reject the sample.

  6. Code – If ‘Reject’ check box is selected, select the Rejection Code from the list. The list displays the values defined in the ‘Rejection’ masters function.

  1. Click [Print Task Slip] to invoke the Preferences screen. Select the desired option to print the task slip.

  2. Click [Task Assigned Samples] and the system invokes Task Assigned Sample/Block Details screen to view task assigned sample or block details. Click [Close] to return to the Requesting Task screen.

  3. Select ‘Task Complete?’ check box to indicate that the requested task is completed. The system invokes Task Completion Details screen. Enter the name of the User who completed the task and the task-completed date. Click [Close] to return to Requesting Task screen.

  4. Select ‘Reject?’ check box and appropriate rejection code to cancel the task.

  5. Click [Save] to save the Requested Task details or [Clear Record] to clear the entered details.
