Specimens for Disposal

From this function you can Mark the specimens for dispose to a specific laboratory department.

The specimens for disposal are based on the parameter setup in Administration under ‘Department Parameter for Facility’. If specimen dispose period is not set in departmental level, the data for specimens of this department will not be displayed for disposal.

Marking specimens for Disposal

From Laboratory > Transactions > Specimen menu, select the Specimens for Disposal function.

The Specimens for Disposal screen gets displayed.

  1. Enter the data in the mandatory (*) fields. <Related Field Description>

  2. Click [Display] to view the specimen details satisfying the query criteria.

  3. Click [Clear] to clear the entered search criteria.

  4. Scan the barcode of the specimen; the specimen number gets populated in the ‘Scan Specimen Number Bar Code’ field. You can also enter the specimen number manually.

  5. If the scanned specimen number is available in the list, the ‘Dispose’ check box against the scanned specimen number gets selected automatically. Else, you need to select the Dispose? check box next to the desired specimen number.

  6. Click [Dispose] to dispose the selected specimen.

  7. Click [Print List] to print the list of specimens.

If the specimen is not available in the disposal list while scanning, then an error message ‘Specimen is not available in disposal list’ is displayed.
