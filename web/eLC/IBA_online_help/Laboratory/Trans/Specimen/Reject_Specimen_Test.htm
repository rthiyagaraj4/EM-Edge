Reject Specimen/Test

From this function you can Reject the specimen or can Test ordered for a particular specimen. You can Reverse the billing of the patient using this function.

Using this function, you can perform query based on the Specimen number to access the specimen details and the details of the ordered test. You can cancel the specimen or the test by providing appropriate reason. The first part is displays the specimen details and the second part displays the test details.

Rejecting Specimen/Test

From Laboratory > Transactions > Specimen menu, select the Reject Specimen/Test function.

The Reject Specimen/Test screen gets displayed.

  1. Enter the specimen number or click [Execute Query] to search and select the desired Specimen number to reject the specimen or the test.

  2. Select 'Reject?' check box in the first part to reject the entire specimen along with the investigations. Select the appropriate Reason for rejection.

  3. Select the 'Reject' check box in the investigations (next to the desired test) to reject a particular test. Select the appropriate reason for rejection.

  4. Click [Notify] to view and record the Result Notification text which need to be notified to other users. If the specimen is resulted, you can only query it and view the notification details. You can also delete the Notify Code by clicking the [Delete] button.

  5. Click [Save] to reject selected Specimen or the Test.
