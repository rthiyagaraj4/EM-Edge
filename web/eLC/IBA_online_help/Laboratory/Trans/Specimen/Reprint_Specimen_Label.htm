Reprint Specimen Label

From this function you can Reprint the Slide, Aliquot and Request Slip along with Specimen label.

Reprint Specimen Label is used to search for the specimen details based on the Patient ID, Specimen range, Specimen Status and Specimen Registration Date. Enter any of the query criteria to view the specimen details. You can print aliquot and slide labels and request slip along with the specimen label for a particular patient. The specimen label includes the First Name, Second Name, Third Name and Family Name of the patient.

Reprinting Specimen Label

From the Transactions > Specimen menu, select the Reprint Specimen Label function.

The Reprint Specimen Label screen gets displayed.

  1. Enter the details in the respective fields. <Related Field Description>

  2. Click [Show Details] to view the specimen details to reprint the specimen labels.

    The following details are displayed:

    1. Specimen No - The outstanding specimen numbers is displayed.

    2. Patient ID – The Patient ID corresponding to the specimen numbers is displayed.

    3. Regd Date Time – The registered date and time of the specimen registration is displayed.

    4. Specimen – Select the Specimen check box to reprint the specimen.

    5. Copies – Enter the number of copies to be printed. One is selected, by default.

    6. Aliquot – Select Aliquot check box, if the specimen requires Aliquot labels to be printed.

    7. Slides – Select Slides check box, if the specimen requires Slide labels to be printed.

    8. Request slip – Select the Request slip check box to print request slip.

    9. Select All – Select the check box to select all specimens from the specimen details.

    10. Deselect All – Select the check box to deselect the selected specimens from the specimen details.

    Click [Show Detail] to display the specimen details for the selected criteria. Click [Print] to print the specimen labels.

  3. Select the corresponding check box against the specimen details to print the specimen, aliquot, slide or request slip.

  4. Click [Print] to print the specified label type for the specimen.
