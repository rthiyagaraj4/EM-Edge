Worklist with Results

From this function you can Print a worklist with results specific to a laboratory department.

System can provide a print out with the results for the specimens and tests that are included on the worklist. The details of the worklist to be printed have to be entered. The system provides a facility to view all the created worklists for the user’s laboratory department. Select the required worklist from the list displayed. After specifying the parameters, confirm whether the printing of the worklist has to proceed.

Printing Worklist with Results

From the Transactions > Worklist menu, select the Worklist with Results function.

The Worklist with Results screen gets displayed.

  1. Select a department from the list. The default department associated to the user and facility is displayed, by default. You can choose any department from the list.

  2. Enter the details in the mandatory fields. <Related Field Description>

  3. On entering the details, press [Tab] to invoke the Preferences dialog box.

    • Select the desired option to print, view or save the result in a file. A confirmation message ‘Process Successful. Press 'OK' to view report’ gets displayed. Click [OK] to view the report or [Cancel] to cancel the process. On selecting print, the results get printed in the designated printer. The results are saved in a file, if the option is selected as ‘Save’.
