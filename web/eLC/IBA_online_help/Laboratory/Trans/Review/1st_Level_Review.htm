1st Level Review

From this function you can Review and Record results of one or more tests of a specimen that have failed validation.

First level review is used to review the abnormal values and validate the results. This function can process specimens that have failed first level validation checks. The tests that have failed the validation are shown in red color and the results are modifiable.

1st Level Review

From Laboratory Transactions > Review menu, double-click the 1st level Review function.

The 1st Level Review screen gets displayed.

  1. Select the desired Department from the list.

  2. Enter the specimen number in alphanumeric format of 13 characters maximum length or select the specimen number using the [List of Values]. This is a mandatory field. The system displays a list of specimens to process that has failed first level validation checks. <Related Field Description>

  3. Click [Modify] to enter/modify the result of all the other remaining tests.

  4. Click [Attach Image] to attach one or more the microscopic images or digital images and documents. To view the uploaded file, click View hyperlink. The Image Links dialog gets displayed. You can delete the uploaded file by selecting the Delete check box against the uploaded file. Click [Apply] to delete the selected image file or click [Close] to close the Image Links dialog.

  5. Click [Release] to release the result.

 

The Numeric Result High/Low column is highlighted in violet if the investigation numeric result is less than the Critical Check value setup in the Test Codes screen.

The specimen records get displayed as per priority (stat, urgent, routine) on pressing F8.
