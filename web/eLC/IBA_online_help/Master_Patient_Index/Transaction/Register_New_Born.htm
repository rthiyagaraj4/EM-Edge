Register Newborn

From this function you can Register a Newborn by capturing delivery details, demographics of parents, and newborn details.

The purpose of newborn registration is to generate unique identifier for every newborn and to maintain birth register used for statistical analysis.

Registering a Newborn

From the MP Transactions, select the Register New Born function.

The Search Patient screen gets displayed.

  1. Search and select the desired female patient to display Register New Born screen based on the MP Parameter Rules.

    The system checks the following rules set in MP Parameter before registration:

    • Age limit specified for the mother patient

    • Gestation period between two deliveries

    If the age of the patient chosen does not fall in the age range specified for the MP Parameter ‘Age limit specified for the mother patient’, the system displays an appropriate error message and does not allow the registration of the baby.

    Further, the system checks if the gestation period for the selected patient falls within the range specified in the MP Parameter ‘Gestation period between two deliveries’, the system displays a message and does not allow the user to continue with the registration process.

    • Period between two deliveries

    If the patient chosen has had a baby delivered and registered in the logged in facility within the period specified in the MP Parameter then the system displays a message as and stops the user from registering another baby for the same patient.

    • Review Mother Baby Link before Registration

    After verifying the above-mentioned rules, based on the ‘Review Mother Baby Link before Registration’ set up in MP Parameter, Mother Baby link window is displayed. If mother baby link is not set, then Register New Born screen is displayed directly.

    The ‘ Mother Baby link’ window provides brief information on the selected mother patient and details of her child/children delivered by her within the enterprise.

    The mother patient details displayed are the Name of the Mother Patient, Age, Gender, and Patient ID. If the mother patient is currently an inpatient, Encounter ID, the Nursing Unit, the Ward and the Bed Details are also displayed.

    On viewing the details, click [Register]. The ‘Register Newborn’ screen is displayed. If there are no babies registered for the chosen patient, the message ‘No child records available for this mother’ is displayed. Click [Ok].

  2. Enter the details in the mandatory (*) fields of the sections. <Related Field Description>

  3. Click [Apply] to save the registration of a Newborn and Create Patient File screen gets displayed or click [Reset] to reset the entered data. Record the patient file details. Click [OK] to generate file number or click [Cancel] to cancel file generation.

  4. Internal Reports screen gets displayed, click [OK] to generate Internal Reports or click [Cancel] to exit from the screen. The patient file label, patient ID card and patient front sheet are printed in the internal reports of Register New born and Maintain Birth Register functions. You can print the Birth Certificate only in the Maintain Birth Register function. The print option of Birth Certificate in MP Internal Reports Reprint function is disabled.

  5. An alert message gets displayed to proceed with Multiple Birth Registration if Multiple Birth check box is selected for the patient and based on the ‘Subsequent Regn of Multiple Birth Within’ field in MP Parameter. Click [OK] to proceed with the new born registration or [Cancel] to cancel the process.

  6. Patient ID and File Number are generated and displayed.

  7. Click [Query] to search for another female Patient.

Newborn registration in stages

  1. If the check box ‘Single Stage New Born Registration Applicable’ is cleared in the MP Parameter, the new born registration is done in two stages.

  2. 1st stage involves recording minimum details with birth date, time and gender. Click [Register] to register the newborn as the 1st stage of registration. The status of the record is displayed in color code. The status of the record is displayed as Registered.

  3. Click the Baby Name hyperlink to populate the Change Patient Details screen to update the New Born details.

  4. Click Newborn ID hyperlink to finalize the record. The message ‘Do you want to finalize?’ gets displayed. Click [OK] and the control shifts back to the registration screen to complete the finalization process. Click [Cancel] to cancel finalizing and the system prompts to print the report.

  5. Click [Audit Trail] to view the Audit Trail details of the newborn registration. The [Audit Trail] is displayed only on the registered or finalized records of the newborn.

  6. In the finalize mode, enter the mandatory details (Length of Baby, Head Circumference, Apgar Score, Patency of Anus) and click [Finalize]. You can print the Birth Certificate after finalization. The record is finalized and status of the record is displayed in color code. The status of the record is displayed as Finalized. Click Newborn ID to view the recorded details of the newborn. Click Task hyperlink to Admit the newborn.

The Gender list is defaulted to ‘Female’ and only Female Patients are listed on searching for a patient in the Patient Search screen.

You can update the Mother details for second child birth and subsequent new born registration if ‘Modify Mother Details in Multiple Birth’s’ check box is selected in MP Parameter screen. The fields ‘Multiple Birth’, ‘No of Births’ and ‘Gestation’ are non-editable.
