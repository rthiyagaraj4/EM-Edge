General Tab

 

  1. Patient ID – Enter the Patient ID in the required format and length as specified in MP Parameter.

  2. Citizen ID – Enter the Citizen ID in the required format and length as specified in MP parameter.

  3. Previous HN – Enter the Previous Hospital Number in alphanumeric format of maximum length as specified in the MP Parameter.

  4. Govt No - Enter the Government Number in alphanumeric format of maximum length as specified in the MP Parameter.

  5. Driv Lic No - Enter the Driving License Number in alphanumeric format of maximum length as specified in the MP Parameter.

  6. Expatriate No - Enter the Expatriate Number in alphanumeric format of maximum length as specified in the MP Parameter.

  7. Other Alt Type – Select the Other Alt Type from the list box. The listed values are ‘Birth Certificate’, ‘Passport’, and ‘Mother’s PMI’, ‘PMI’ etc.

  8. Other Alt No - Enter the Other Alternate Number in alphanumeric format of maximum length as specified in the MP parameter.

 

The ‘Prompt’ specified for the Citizen ID, Other Alt No, Previous HN, Govt No, Driv Lic No and Expatriate No in MP parameter appear as the field legend. User should Note that on entering the Patient ID, the Alternate ID Type(s) is disabled and vice-versa.

 

  1. First Name – Enter the first name in the alphanumeric format of 15 characters maximum length.

  2. Second Name – Enter the second name in the alphanumeric format of 15 Characters maximum length.  

  3. Third Name – Enter the third name in the alphanumeric format of 15 characters maximum length.

  4. Family Name – Enter the family name in the alphanumeric format of 15 characters maximum length.

  5. Soundex - Select the soundex type as either ‘Ethnic’ or ‘Generic’.

  6. Gender - Select the Gender from the list box. The listed values are ‘All’, ‘Male’, ‘Female’ and ‘Unknown’. By default ‘All’ is selected.

  7. Birth Date - Enter date of birth of the patient in dd/mm/yyyy format or select the date using the Calendar function.

  8. By - Select the value as either Date or Year from the list box. If ‘Date’ is selected, then records of patients born on the specified date of birth are displayed in the result set. If ‘Year’  is selected, then records of patients born in the year specified in the ‘date of birth’ column are displayed.

  9. Head Patient ID - Enter the Head Patient ID in alphanumeric characters of 10 characters.

  10. Nationality - Select the Nationality from the list box.

  11. Race - Select the race from the list box.

  12. Area - Select the Area from the list Box.

  13. Order By - Select the value from the list box. The result set is sorted based on the selected value.

 

Additional Tab

Enter the following details

 

  1. Contact No - Enter the Contact Number of the patient in alphanumeric format of 20 characters maximum length. On selecting the contact number the Mode field will be enabled.

  2. Mode - Select the Contact mode from the list box. The field is enabled only on entering the ‘Contact No’ field.

  3. Deceased Only - Check the box, to display only the deceased patients in the search result.

  4. Additional Addr – Enter the Additional Address in alphanumeric format of 30 characters maximum length.

  5. Suspended Only - Check the box, to display only the suspended patients in the search result.

  6. Inactive Only - Check the box, to display only the inactive patients in the search result.

  7. Mother Name - Enter the Mother Name in the text box.

  8. Registration Mode – Enter the required registration mode from the list box. The values listed are ‘Normal’, ‘Emergency’, ‘Rapid’.

  9. Old File No - Enter Old File Number of the patient in alphanumeric format of 15 characters maximum length.
