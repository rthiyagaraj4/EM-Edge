Enter the following details to create Vaginal Delivery Type record:

 

  1. Code - Enter the Vaginal Delivery Type code in the alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Long Description - Enter the long description of the vaginal delivery type in the alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description of the vaginal delivery type in the alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Enabled - Select the check box to enable the record. Only the enabled records are available in the transaction functions.
