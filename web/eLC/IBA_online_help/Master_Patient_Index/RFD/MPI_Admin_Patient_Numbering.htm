Enter the following details to define Patient Numbering Rules:

 

  1. Patient Series - Select the Patient Series from the list. This field is mandatory.  

  2. Site/Facility - Based on the Numbering Control set for the selected Patient Series (while defining the Patient Series); it displays either Site or list of Facilities in the list. This field is mandatory. Select the required value from the list.

 

Patient Series

 

Numbering Control

List value (Site/Facility)

Site

Site name is listed.

Facility

All facilities of the site are listed in the list.

 

Serial Number

 

  1. Start - Enter the starting serial number for the Patient Series. The length of this number depends on the selected Patient Series. If ‘Prefix Required’ for the selected Patient Series is set (in the function Patient Series), the length of start serial number must be two characters less than the Patient ID length, or it can be equal to Patient ID length. This field is mandatory.

  2. Max - Enter the maximum serial number for the Patient Series. The length of this number depends on the selected Patient Series. If 'Prefix Required' for the selected Patient Series is set, the length of maximum serial number must be two characters less than the Patient ID length, or the length can be equal to Patient ID length. This field is mandatory. The specified number must be greater than the start serial number. This field is mandatory.

  3. MR Section - Select the MR Section to be associated with the selected Patient Series from the list. If Numbering Control for the selected Patient Series is by the Site, all the MR Sections of type 'Central' defined for the logged in facility are listed. If Numbering Control for the selected Patient Series is by Facility, then MR Sections (Central) are listed that are defined for the selected facility.

Based on the Patient Series selected the system displays the screen for facility or site.

 

If the Patient Series selected is associated for a site, associate the MR Section with facilities using the following details:

  1. Facility – The name of the facility under the site is displayed.

  2. MR Section – Select the MR Section from the list.

  3. Select – Select the ‘Select’ check box to associate MR Section to the facility.
