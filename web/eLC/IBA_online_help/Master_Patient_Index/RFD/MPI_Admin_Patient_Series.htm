Enter the following details to create a Patient Series record:

 

  1. Series Type - Select the Series Type from the list. The listed values are described below. This field is mandatory.

  2. General Series - Select this option to include the defined Patient Series under General Series group.

  3. New Born Series - Select this option to include the defined Patient Series under Newborn Series group.

  4. Emergency Series - Select this option to include the defined Patient Series under Emergency Series group.

  5. Alternate ID Series - Select this option to include the defined Patient Series under Alternate ID Series group.

  6. National ID Series – Select this option to include the defined Patient Series under National ID Series group.

  7. Unrestricted Series – Select this option to include the defined Patient Series under unrestricted group. Patient Series belonging to this type are available in all the modes of registration.

  8. External Source Series – Select this option to include the defined Patient Series under External Series group.

  9. External Patient/Donor Series - Select this option to include the defined Patient Series under External Patient/Donor Series group.

  1. Code - Enter the Patient Series code in alphanumeric format of two characters maximum length. This field is mandatory.

  2. Long Description - Enter the long description of the Patient Series in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description - Enter the short description of the Patient Series in alphanumeric format of 15 characters maximum length. This field is mandatory.  

  4. Check Digit Scheme - Select the value from the list. The listed values are defined in the function ‘Check Digit Scheme’.

 

Patient ID Generation Rule

  1. Associated Numbering Series - Select the Numbering Series from the list. This field is enabled only if the ‘Single Patient Numbering Applicable’ check box is selected in MP Parameter function. The listed values are the existing Patient Series. If the series is an associated series, the attributes of Patient ID generation rule is disabled.

  2. Generate Patient ID – To Generate Patient ID automatically by the system on registration, select the check box. To generate the Patient ID manually, clear the check box.

  3. Use Alternate ID1 Rule – Select the check box if Alternate ID1 Rule is applicable for the selected Series Type. This check box is enabled if 'Generate Patient ID Using Alternate ID1 Rule' check box is selected in MP Parameter.

  4. Separator Value - Enter the Separator Value in alphabetic format of two characters maximum length. This field is enabled if 'Use Alternate ID1 Rule' check box is selected based on the MP Parameter setup. This value is defined to group the patients according to the Relationship Level.

  5. Separator Position - Select the Separator Position from the list. The listed values are 'Prefix' and 'Suffix'. Based on the position selected, the Separator Value is displayed in the patient identification number.  

  6. Prefix Required - To prefix the Patient Series code to the Patient ID during registration, select the check box.  

  7. Numbering Control (By Site/By Facility/None) - The serial number for the Patient Series is generated and controlled either at Site or Facility. The numbering control depends on the value specified for the field ‘Generate Patient ID’ field. If ‘Generate Patient ID’ is selected, ‘None’ is not applicable. If the field ‘Generate Patient ID’ is cleared, ‘By Site’ and ‘By Facility’ are not applicable.

  8. By Site – Select ‘By Site’, if Patient ID generation is to be controlled by the Site.

  9. By Facility – Select ‘By Facility’, if Patient ID generation is to be controlled by the Facility.

  10. None – This value is selected by default if the ‘Generate Patient ID’ is cleared. Enter the Patient ID manually for the defined series during registration process.

The values ‘By Site’ and 'By Facility’ are available in this function based on the setting in ‘MP Parameter’ (under ‘Patient Administration’ tab > Numbering Control). If Numbering Control is defined ‘By Site’ in the parameter, ‘By Facility’ is not available in this function. Similarly, if ‘By Facility’ is selected in the Parameter Setup, ‘By Site” is not available. If ‘By Combination’ is selected, both ‘By Site’ and ‘By Facility’ are available.

 

  1. Valid for Head Patient ID – Select the check box if the defined Patient Series is valid for Head Patient. Head Patient ID is attached to the patient during registration. Clear the check box, if the defined series is not applicable for head patient. This field is displayed based on the ‘Maintain Family Link’ set in ‘MP Parameter’ (under ‘Legal Names’ tab  Patient Name Derivation Logic).

  1. Patient Name in Loc Lang. Required – Select the check box, if the patient name needs to be specified in Local Language. Clear the check box, if the patient name need not be specified in Local Language.

  2. Email Notification for Appointment Applicable – Select the check box to indicate that the email is applicable for the patient series. This field is enabled only if the Email Applicable parameter check box is selected in the SM Site Installation. This field can be cleared at any time. Based on this setup, the notification through email is sent to patients belonging to that patient series.
