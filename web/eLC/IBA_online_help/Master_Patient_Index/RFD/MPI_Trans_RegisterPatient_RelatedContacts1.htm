Next of Kin

 

  1. Name - Enter the Name of the person to be contacted in alphanumeric format of 80 characters maximum length.

  2. Relationship - Select the Relationship the patient shares with the next of the kin from the list. To Copy Relationship From Demographics Tab, Click copy Head of Family.

  3. Citizen ID – Enter the Citizen ID of the next of kin in the required format as specified in MP Parameter. To Copy Citizen ID From Demographics Tab, Click copy Head of Family.

  4. Previous HN – Enter the Previous Hospital Number in alphanumeric format of maximum length as specified in the MP Parameter.

  5. Govt No - Enter the Government Number in alphanumeric format of maximum length as specified in the MP Parameter.

  6. Driving License No. - Enter the Driving License Number in alphanumeric format of maximum length as specified in the MP Parameter.

  7. Expatriate No. - Enter the Expatriate Number in alphanumeric format of maximum length as specified in the MP Parameter.

  8. Other Alt ID - Select the Alternate ID from the List.

  9. Birth Date - Enter the Date of Birth in date format of dd/mm/yyyy or use the Calendar function to select for the relevant date.

  10. Occupation - Enter the Occupation of the person in alphanumeric format of 30 characters maximum length.

  11. Address Line (1 to 4) – Enter the Address details in alphanumeric format of 30 characters maximum length in each line. The legends are set in System Manager module.

  12. Area - Enter the name of the Area in alphanumeric format of 30 characters maximum length or select it using the lookup function [?]. The field legend is set in System Manager module.

  13. Town - Enter the name of the Town in alphanumeric format of 30 characters maximum length. The entered name must be listed under the selected area. If the selected area has more towns attached to it, select the required town using the lookup function [?]. The field legend is set in System Manager module.

  14. Region – Enter the Region in alphanumeric format of 30 characters maximum length or select it using the function [?]. The area must correspond to the entered region.  The field legend is set in System Manager module.

  15. Postal Code - Enter the Postal Code in alphanumeric format of eight characters maximum length. The entered value must correspond to the selected town/city. Select the required code using the function [?]. The field legend is set in System Manager module.

  16. Country - The Country name gets automatically populated when the Region is selected.

  17. Home Phone No – Enter the Residence Telephone Number of Next of Kin in alphanumeric format of 20 characters maximum length.

  18. Mobile No - Enter the Mobile Number of Next of Kin in alphanumeric format of 20 characters maximum length.

  19. Office Phone No - Enter Office Telephone Number of Next of Kin in alphanumeric format of 20 characters maximum length.

  20. Email - Enter the Email ID of the person in alphanumeric format of 50 characters maximum length

  21. Copy Head of Family – Click on ‘Copy Head of Family’ hyperlink to copy address details of the patient’s Head of Family.

  22. Copy First To Notify – Click on ‘Copy First To Notify’ hyperlink to copy address details of the patient’s first to notify person.

  23. Copy Patient Address - Click on ‘Copy Patient Address’ hyperlink to copy address details of the Patient.

The field legends for Address 1-4, Town, Area, State, and Postal Code are defined in Site function in System Manger module.
