Enter the following details to create an External Referral Source record:

 

  1. Code - Enter the code for the External Referral Source in alphanumeric format of four characters maximum length. An External Referral Source code is to identify the External Referral Source. The External Referral Source code should be unique and cannot be changed after creation. This field is mandatory.

  2. Long Description - Enter the long description of the External Referral Source in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description - Enter the short description of the External Referral Source in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Healthcare Setting Type - Select the Healthcare Setting Type from the list box to which External Referral Source belongs. The options available are as defined using the function Healthcare Setting Type. This field is mandatory.

  5. Organization Type – Select the Organization Type from the list. The listed values are Government and Private. This field is mandatory.

  6. Enabled - Select the check box if the External Referral Source is to be activated and clear this check box if the External Referral Source is to be deactivated. By default, the check box is selected.

 

Characteristics tab

 

Use as External/Referral Source At

 

  1. Registration/Visit/Admission - The individual check boxes control whether this External Referral Source code are used as a Referral Source at Registration, Visit and Admission respectively or not.

 

Use as External/Referral Destination At

 

  1. Visit Conclusion/Discharge - The individual check boxes control whether this External Referral Source Code can be used  as a Referral Destination during Visit Conclusion and Discharge respectively or not.

 

Use as External/Referral Source for Direct Ordering

 

  1. Applicability – Select a value from the list that is predefined. The values listed are Referral, External, Both and Not Applicable. Select a value to indicate the applicability of the External/Referral Source to perform direct ordering for Referral/External/Both Referral and External patients. Select ‘Not Applicable’ if the External/Referral Source is not applicable for Direct Ordering. This field is mandatory.

  2. Associated Enterprise Facility for Referral – Select the facility from the list that is associated to the Enterprise for Referral. The values listed are defined in System Manager Module. If ‘Applicability’ is selected as Referral or Both, this field is enabled. If 'Applicability’ is selected as External or Not Applicable, this field is disabled.

  3. Patient Prefix for identifying Source – Enter the patient prefix for identifying the External/Referral Source in alphanumeric format of two characters maximum length. If 'Applicability’ is selected as Not Applicable, the field is disabled.

  4. Series for Ext Patient Numbering – Select the series for External Patient Numbering from the list. The values are listed from Patient Series function with Series Type as 'External Source Series’ in the Master Patient Index Module. If ‘Applicability’ is selected as Not Applicable, this field is disabled.

 

Allow Direct Orders for

 

  1. Laboratory/Radiology/Pharmacy – Select the check box if the External/Referral Source is allowed direct ordering for Laboratory, Radiology or Pharmacy. If 'Applicability’ is selected as Referral/External/Both, this field is enabled and on selecting ‘Applicability’ as Not Applicable, this field is disabled.

 

Use as External/Referral Source for Tissue Collection

 

  1. Applicability – Select a value from the list that is predefined. The values listed are 'Tissue Collection’ and ‘Not Applicable’. Select a value to indicate the applicability of the External/Referral Source for tissue collection. Select ‘Not Applicable’ if the External/Referral Source is not applicable for tissue collection.

  2. Donor Prefix for identifying Source – Enter the Donor Prefix for Identifying the External/Referral Source in alphanumeric format of two characters maximum length. If 'Applicability’ is selected as Not Applicable, this field is disabled.

  3. Series for Donor Numbering – Select the Series For Donor Numbering from the list. The values are listed from ‘Patient Series’ function with ‘Series Type’ as ‘External Patient/Donor Series’ in the Master Patient Index Module. If ‘Applicability’ is selected as Not Applicable, the field is disabled.

 

Contact Details tab

 

Address Details

 

  1. Address - Enter the address of the External Referral Source in alphanumeric format of 30 characters maximum length in each of the three boxes.

  2. Country - Select the country for the External Referral Source from the list. The values are listed from country function.

  3. Tel No - Enter the telephone number of the External Referral Source in alphanumeric format of 18 characters maximum length.

  4. Fax No - Enter the fax number of the External Referral Source in alphanumeric format of 18 characters maximum length.

  5. Email - Enter the email ID of the External Referral Source in alphanumeric format of 50 characters maximum length.

 

Contact Person Details

 

  1. Name – Enter the name of the contact Person in External referral source in alphanumeric format of 40 characters maximum length.

  2. Title – Enter the title for the name in alphanumeric format of 40 characters maximum length.
