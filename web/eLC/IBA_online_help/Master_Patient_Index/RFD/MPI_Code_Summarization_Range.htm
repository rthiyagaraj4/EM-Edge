Enter the following:

 

  1. Type – Select the Type from valid list. This field is mandatory. The listed options are Age and Weight.

  2. ID – Enter the ID in the alphanumeric format of four characters maximum length. This field is mandatory.

  3. Long Description – Enter the long description in alphanumeric format of 30 characters maximum length. This field is mandatory.

  4. Short Description – Enter the short description in alphanumeric format of 15 characters maximum length. This field is mandatory.

  5. Enabled – Select the check box to enable the record. Only the enabled records are available in the transaction functions.

  6. No of ranges – Specify the required value in numeric format of two digits maximum length. A maximum of 25 range values can be defined. The fields under the column ‘Ranges’ are enabled depending on the value entered in this field.

Ranges

  1. From – Enter the 'From' value in the numeric format of three digits maximum length for 'Age' type or numeric format of six digits maximum length for 'Weight' type.

  2. Unit – Select the unit from the list. The units displayed depend on the range ‘Type’ selected.
