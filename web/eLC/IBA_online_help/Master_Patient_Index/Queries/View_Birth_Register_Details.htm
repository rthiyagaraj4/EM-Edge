View Birth Register Details

From this function you can Query and view the birth register details newborn patients.

This function is used to view the birth register details. The new born details are displayed.

Querying for Birth Register Details

From MP Queries, select the View Birth Register Details function.

The Patient Search screen gets displayed.

  1. Search and select the desired patient to view the Patient details.

  2. Click [Search] to select a specific record and view details or [Clear] to clear the records displayed in the result area.

  3. Click [Reset] to clear the entered data.

 

If 'Accept Patient's name in Local Language' is selected in MP Parameter, you can enter names in local language. The fields displayed in the screen may vary depending on the MP Parameter Setup.

The color indicator  denotes the Finalized records and  denotes the First stage registration records.
