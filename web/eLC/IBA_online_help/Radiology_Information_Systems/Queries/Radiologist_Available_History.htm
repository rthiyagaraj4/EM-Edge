Radiologist Available History

From this function you can Query the history of radiologist available.

This function allows you to query about the Radiologists availability history details. The details include Radiologist ID, Radiologist Name, Radiologist Available From and To date and time.

If a radiologist does not have access to view other Radiologist’s Availability, then he can view only his availability history. The access can be given to radiologist by clicking the 'View other Radiologist's Availability' check box in 'Query other Radiologist/Revert Status' screen.

Querying Radiologist Available History

From the Queries, double -click Radiologist Available History function.

The Radiologist Available History screen gets displayed.

  1. Enter the From and To Dates by clicking calendar icon for which the radiologist availability has to be queried. By default, the current week dates appear.

  2. The Radiologists ID and Name appear by default. Only if you have access to view other radiology details, the system allows you to select other radiologist name from the List of Values.

  3. Click [Search]. The records matching the search criteria appear.

  4. The search result includes Radiologist ID, Radiologist Name, Available From, and Available To columns.

  5. Click [+] View Details button against the required radiologist record to invoke the ‘Radiologist Availability Details’ window.

  6. The details displayed are Radiologist ID, Radiologist Name, Radiologist Available From and To date and time.
