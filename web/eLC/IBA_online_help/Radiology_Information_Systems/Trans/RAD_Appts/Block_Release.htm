Block/Release

From this function you can Block a room or Release a blocked room.

The Block/Release function enables you to block a room for an appointment or to release a blocked room. If the room is blocked, the appointments cannot be booked for that room. The reason for blocking the room has to be provided and the blocking period has to be defined. After a blocked room is released, further appointments can be booked for that room. You can block the rooms, if the room is under renovation, if the equipment in the room is under repair, if only examinations for emergency cases are to be performed and so on.

Blocking/Releasing an Appointment

From the Transactions > Appointments, double-click Block/Release Appointments function.

The Block/Release Appointments screen gets displayed with two tabs. Block tab is enabled, by default.

  1. Enter the details in the mandatory (*) fields of the following tabs.

Block

Enter the following search criteria:

  1. Room Code – Enter the Room code or click [List of Values] to select the Room. Room is a Study Location in an Operating Unit. Study locations are defined in the ‘Study Locations’ function. This is a mandatory field.

  2. Reason – Enter the Reason for blocking the room. This is a mandatory field.

  3. Day Specific – Select the check box if the blocking is day-specific.

  4. Block Date [From-To] – Enter the date range to block the room in the format of dd/mm/yyyy or select the date range using the Calendar function. This is a mandatory field

  5. Slot Time [From-To] – Enter the Slot Time range in hh:mm format. This is a mandatory field.

  6. Weeks – Select the required check box to select the Week to block the room. Select ‘All Weeks’ check box to block the room for all the weeks in the blocking period. The week specific check boxes are disabled on selecting ‘All Weeks’. Select the ‘Week’ check box corresponding to the required week for which the room must to be blocked.

  7. Day – Select the required day from the list. The listed values are Sunday, Monday, Tuesday, Wednesday, Thursday, Friday and Saturday. The rooms are blocked for the selected day and for appointments can be booked on other days.

The ‘Weeks’ and the ‘Day’ fields are enabled only if the ‘Day Specific’ check box is selected. The ‘Weeks’ and the ‘Day’ values are defined for conditional blocking.

Release

The blocked details of the rooms are displayed on selecting ‘Release’ tab. The displayed details are ‘Room code’, ‘Description, Section, Blocked Date, Day for which the room is blocked and the Block Start/End Time.

 

  • Select the ‘Release’ check box corresponding to the room, which needs to be released.

  1. Click [Save] or press [F10] to save the Blocked/Released Appointments or click [Clear Record] to clear the entered data.
