Booking Access Rights

From this function you can Define Booking Access Rights and Query for the existing ‘Booking Access Rights’ records.

The Booking Access Rights function enables you to define access rights for a user for a particular Study Location and Section in a facility. The User is provided access for Booking, Transfer, Cancelling, Overbooking or Forced Booking of Appointments. The user rights are defined for a particular Study Location or it is made applicable across all Study Locations. You can set the Booking Rights for a particular user using ‘Booking Rights for All Study Locations/Specific Sections’ tab. You can define the rights applicable across all the Study Locations/Sections in a facility.

Defining Booking Access Rights

From the Administration > Appointments, double-click Booking Access Rights function.

The Booking Access Rights screen gets displayed with two tabs. Users for Study Locations tab is displayed, by default.

  1. Enter the details in the mandatory (*) fields in the following tabs.

Users for Study Locations

Enter the following details to define access rights to Users for a Study Location:

  1. Study Location – Enter the Study Location code in alphanumeric format of four characters maximum length or click [List of Values] to select the Study Location. On selecting the Study Location code, the description of the Study Location is displayed.

  2. Section – Enter the Section code in alphanumeric format of four characters maximum length or click [List of Values] to select the Section. On selecting the Section code, the description of the Section is displayed.

Users

  1. Users – Enter the User Name or click [List of Values] to select the User from the list.

  2. Book – Select the check box to indicate that the user has rights to book appointment(s) for the selected Study Location/Section.

  3. Transfer - Select the check box to indicate that the user has rights to transfer appointments from/to the selected Study Location/Section.

  4. Cancel - Select the check box to indicate that the user has rights to cancel an appointment for the selected Study Location/Section.

  5. Overbook - Select the check box to indicate that the user has rights to overbook appointment(s) for the selected Study Location/Section.

  6. Force Book - Select the check box to indicate that the user has rights to Force book appointment(s) for the selected Study Location/Section.

Booking Rights for All Study Locations/Specific Sections

Enter the following details to define users for study location:

  1. User Name – Enter the User name or click [List of Values] to select the User. This is a mandatory field.

  2. Book – Select the check box to indicate that the User has rights to book appointment(s) for all the Study Locations/Sections.

  3. Transfer - Select the check box to indicate that the User has rights to transfer appointment(s) from/to the Study Locations/Sections.

  4. Cancel - Select the check box to indicate that the User has rights to cancel an appointment for all study locations/sections.

  5. Overbook - Select the check box to indicate that the user has rights to overbook appointment(s) for all the Study Locations/Sections.

  6. Force Book - Select the check box to indicate that the User has rights to force Book appointment(s) for all the selected Study Locations/Sections.

  7. Options -

  8. All Study locations – Select ‘All Study Locations’ to define access rights to all the Study Locations in the facility.

  9. Sections – Select ‘Sections’ to define access rights to all the Sections in the facility.

  1. Click [Save] or press [F10] to save the defined Booking Access Rights or click [Clear Record] to clear the entered details.

  2. Click [Execute Query] or press [F8] to search for the existing Booking Access Rights records.
