Purge Appointments Statistics

From this function you can Purge the Appointment Statistics report records.

Purging Appointment Statistics records

From the Maintenance, double-click Purge Appointments Statistics function.

The Purge Appointments Statistics screen gets displayed.

  1. The Last Purged Statistics Date is displayed.

  2. Enter the date upto which you want to purge the Appointment Statistics in the format of dd/mm/yyyy or select the required date using the Calendar function.

  3. Click [Run Process] to purge the Appointment Statistics. A confirmation message ‘Do you want to confirm’ is displayed. Click [Yes] to purge the Appointment Statistics Records or [No] to cancel the process.
