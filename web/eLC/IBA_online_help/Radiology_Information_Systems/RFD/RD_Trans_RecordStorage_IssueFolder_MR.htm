Folders are requested electronically or manually. Folders can be issued against manual request. The details are recorded under 'Manual Request' Tab.

 

Enter the following details:

Request From

  1. Request Facility – The logged in facility name is displayed, by default. Enter the Facility ID or click [List of Values] to select the Request Facility. The folders are transferred from this facility to the requesting facility. This is a mandatory field.

  2. Request Loc Type/Location – Select the Request Location Type from the list. On selecting the Location Type, enter the Location code or click [List of Values] to select the Location. Based on the Location Type selected, the Location values are populated.

  3. Request Record Location – Select the Request Record Location.

  4. Request by User – The logged in user name is displayed.

  5. Narration – Enter the Narration code or click [List of Values] to select the Narration. On selecting the Narration code, the description of the Narration is displayed. The Narration defines the reason for transferring the record from one location to another.

  6. Storage Location – Enter the Storage Location or click [List of Values] to select the current Storage Location of the folder.

  7. Patient ID – Enter the Patient ID in the length as specified in the MP Parameter or click ‘Right Click for User Options’ to select the Patient ID using the Standard Patient Lookup function. On selecting the Patient ID, the name of the patient is displayed.

  8. Remarks – Enter the remarks while issuing the folder.

 

Request To

  1. Request to Facility – The logged in facility name is displayed, by default. Enter the Facility ID or click [List of Values] to select the Facility. The folders are transferred to this facility from the requested facility. This is a mandatory field.

  2. Request Period – Enter the period in number of days for which the folder has been requested.

 

Click [View Folders] to view the folder details. The folder details are displayed in the ‘Current Folder Location Details’ section. The Folder Number, Facility ID, Record Location Type, Location, Folder Status are displayed in the ‘Current Folder Location Details’ section. Click [R] to view the recorded remarks. You can edit or modify the entered remarks. Select the ‘Issue’ check box to Issue the Folder to the requested location.
