Enter the following to create a Reason:

  1. Reason Code – Enter the Reason code in alphanumeric format of six characters maximum length. This is a mandatory field.

  2. Description – Enter the description of the reason in alphanumeric format of 230 characters maximum length. This is a mandatory field.

  3. Section Applicable – Enter the Section applicable to the Reason or click [List of Values] to select the applicable Section. On selecting the Section, the description of the selected Section is displayed. The Sections are defined in the ‘Sections’ function.

 

Action Types for Reason Code

  1. Action Type – Select the Action Type from the list. The system-defined values: Transfer Appointment (Individual), Transfer Appointments (Bulk), Cancel Appointment (Individual), Cancel Appointments (Bulk), Cancel Request/Exam, Add to Library, Replace Exams, Request for Exams, Request for Review, Repeat Exam, Revert Exam Status and Change Perform Location are listed. Based on the Action Type selected, the Reasons are displayed in the particular transaction. Only 10 records can be added at a time. This is a mandatory field.

If the Schedule Type in the RD Parameters screen is selected as ‘RIS Schedule’, the Appointment related action types such as ‘Transfer Appointment (Individual)’, ‘Transfer Appointments (Bulk)’, ‘Cancel Appointment (Individual)’, ‘Cancel Appointments (Bulk)’ are displayed. If the Schedule Type in the RD Parameters screen is selected as ‘General Schedule’, the Appointment related action types are not displayed.

  1. Active – Select the check box to associate the Action Type with the reason.
