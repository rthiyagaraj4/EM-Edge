Enter the following details:

  1. Receiving Facility – The Facility Code and description of the receiving facility is displayed.

  2. Rec Loc Type/Location – The Receiving Location Type is displayed. Enter the Location code or click [List of Values] to select the Location.

  3. Rec Record Location – Enter the Receiving Record Location or click [List of Values] to select the Receiving Record Location.

Search

  1. From – Select the Location Type from the list. The listed values are ‘department’, ‘Ward’, ‘Clinic’, ‘Physician’ and ‘Radiology Department’. On selecting the Location Type, enter the Location code or click [List of Values] to select the corresponding Location from where the record is to be returned.

  2. Click [View Folders] to list the folders matching the entered criteria in the ‘Folder Details’ section.

 

Folder Details

  1. Folder No – Enter the Folder Number as specified in the RD Parameter for Facility function or click [List of Values] to select the Folder Number.

  2. Patient ID – The Patient ID is displayed based on the selected Folder Number.

  3. Name – On selecting the Patient ID, the name of the patient is displayed.

  4. Confirm Return – Select the ‘Confirm Return’ check box to confirm the return of the folder.
