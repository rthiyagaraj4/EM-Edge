Enter the following details:

  1. Appt Ref No. – Enter the Appointment Reference Number or click [List of Values] to select the Appointment Reference Number. The Appointment Reference Number is generated while booking the examination. On selecting the Appointment Reference Number, the Episode Number, Registration Date and the Patient ID and Name, Gender and Age of the Patient are displayed.

  2. Registration No. – Enter the Registration Number for the examination in numeric format of 10 digits maximum length.

  3. Registration Date – The scheduled date of the examination is displayed. Enter the Registration Date in the format of dd/mm/yyyy or select the date using the Calendar function, if the actual date of performing the examination is not the same as the scheduled date.

  4. Clinical Events – Click [Clinical Events] to view other details like, Order Number, Catalog Code of the order and associated Clinical Events.

  5. Checklist Completed – Select the check box if the check lists(s) for this examination is completed.

 

Check List

  1. Code – The code of the checklist is displayed.

  2. Description – The description of the checklist is displayed.

  3. Select – Select the check box to indicate that the checklist is completed.
