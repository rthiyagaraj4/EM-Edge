Enter the following to create a Study Location:

  1. Study Location Code – Enter the Study Location code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Long Description – Enter the long description of the Study Location in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description of the Study Location in alphanumeric format of 15 characters maximum length. The description entered in the long description field is displayed, by default. This is a mandatory field.

  4. Operating Unit/Wing – Enter the Operating Unit/Wing or click [List of Values] to select the Operating Unit/Wing. The Operating Unit/Wings are defined in the Operating Units/Wings function. This is a mandatory field.

  5. Section Code – Enter the Section or click [List of Values] to select the Section. Sections are displayed based on the selected ‘Operating Unit/Wing’. Sections are associated to the Operating Unit/Wing in the ‘Sections for Operating Unit/Wing’ function. This is a mandatory field.

  6. Sex Specific – Select the Gender applicable to the Study Location from the list. The listed values are Male, Female and Nonspecific. Select ‘Non specific’ to allow both male and female patients to be examined in the Study Location.

  7. Equipment Code – Click [List of Values] to select the Equipment Code defined in the Equipment Masters in Application Masters. If the Schedule Type in the RD Parameters screen is selected as ‘General Schedule’, the Equipment Code is mandatory and displayed. This will be the linking point for Book Appointment Setup between 2 Tier and 3 Tier.

  8. Operates on all Days - Select the check box to indicate that the Study Location is operational on all the days of the week. This association is used while setting up the Study Location Timetable.

  9. Mobile – Select the check box to indicate that the Study Location is a mobile unit. Some study locations in a wing are mobile study locations to cater to the patients who are unable to come to the Study Location in the facility to undergo the examination.

  10. Appointment Series – Enter the Appointment Series or click [List of Values] to select the Appointment Series. Appointment Series is defined in the ‘Appointments Series’ function under Appointments in the Administration Menu. This is a mandatory field.

  11. Modality ID – Enter the Modality ID in alphanumeric format of 16 characters maximum length. The code entered in the ‘Study Location Code’ field is displayed, by default. On entering the Modality ID, the equipment is attached to the Study Location.  

  12. Check Appt Conflict for Modality – Select the check box to indicate that you have to check for conflicting appointments in other study locations associated with this modality, if more than one Study Locations share an equipment (modality).

The fields ‘Sex Specific’, ‘Operate on all Days’, ‘Appointment Series’, ‘Check Appt Conflict for Modality’, ‘Associated Locations for Modality’ and ‘Preferred Location’ are disabled if the Schedule Type in the RD Parameters is selected as ‘General Schedule’.

 

Exams for Study Location

  1. Exam Code – Enter the code for the Examination to be performed in this Study Location or click [List of Values] to select the Exam Code. Exam Codes are displayed based on the selected ‘Operating Unit/Wing’. The Exam codes are associated to the Operating Unit/Wing in the ‘Exams for Operating Unit/Wing’ function under the Administration Menu. This is a mandatory field.

  2. Exam Description – The description of the selected exam code is displayed.

  3. Preferred Location – Select the check box to indicate that the Study Location is the preferred location for the selected examination. One exam can be conducted in more than one Study Locations, but you cannot specify more than one Study Location as the Preferred Location.
