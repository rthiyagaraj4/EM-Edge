Enter the following details to create a Storage Location:

  1. Location Code – Enter the Location code in alphanumeric format of six characters maximum length. The length of the Location code is defined in the RD Parameter function. This is a mandatory field.

  2. Long Description – Enter the long description of the Location in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description of the Location in alphanumeric format of 15 characters maximum length. The description entered in the long description field is displayed, by default. This is a mandatory field.

  4. Temporary Storage Location – Select the check box to indicate that the Storage Location is a Temporary Storage Location.

 

Registration Numbering

  1. Prefix – Enter the Prefix in alphanumeric format as specified in RD Parameter. If ‘Storage’ option is selected in RD Parameter, the prefix values are recorded in this function. The prefix values indicate the Storage Location and are generated along with the serial number while registering the examination.

  2. Next No. – Enter the next registration number based on RD Parameter. This number is used to assign the registration number for the current registration and it is incremented by the system.

  3. Max No. – Enter the maximum number of registration based on RD Parameter. This number is used to restrict the registration numbering. After reaching the ‘Maximum No.,’ you must initialize the next number.

 

Permanent Folder Numbering

  1. Prefix – Enter the prefix in valid format of four characters maximum length as specified in the RD Parameter function. This prefix is used for permanent folders stored in this storage location.

  2. Next No. – Enter the next number used to assign the Permanent Folder Numbering for the current registration and it is incremented by the system.

 

Temporary Folder Numbering

  1. Prefix – Enter the prefix in valid format of four characters maximum length as specified in the RD parameter function. This prefix is used for the temporary folders stored in this Storage Location.

  2. Next No. – Enter the next number used to assign the Temporary Folder Numbering for the current registration and it is incremented by the system.

  3. Allow Multiple Folders – Select the check box to allow the storage of multiple folders in this Storage Location.
