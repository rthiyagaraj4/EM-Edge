Enter the following details to create a Media Type:

  1. Media Type – Enter the Media Type code in alphanumeric format of 12 characters maximum length. This is a mandatory field.

  2. Long Description – Enter the long description for the Media Type in alphanumeric format of 90 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description for the Media Type in alphanumeric format of 40 characters maximum length. The description entered in the long description field is displayed, by default. This is a mandatory field.

  4. Expected Return Days – Enter the expected return days in numeric format of three digits maximum length.

Expected Return Days define the number of days within which the record has to be returned to the storage location, if the record is moved from the original storage location to another location.
