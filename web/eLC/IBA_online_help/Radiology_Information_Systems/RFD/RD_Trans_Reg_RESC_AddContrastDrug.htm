Enter the following details:

  1. Completed – Select the check box to mark the exam as completed.

  2. Exam – The code and description of the exam is displayed.

  3. Sedative Drug – Select this check box to record sedative drug details for the examination.

  4. Contrast Type – Select the Contrast Type from the list. The system-defined values are ‘Only Plain’, ‘Only Contrast’ and ‘Plain/Contrast’.This list becomes empty and disabled when the Sedative Drug check box is selected.

  5. Anaesthesia Code – This field is enabled only when the ‘Sedative Drug’ check box is selected. Select the anaesthesia code from the drop-down list. This list populates the anaesthesia code and description value from the Anaesthesia master in Application Masters (AM) module.

  6. Contrast Agent / Drug – This field is enabled if Contrast Type is selected as ‘Only Contrast’ or ‘Plain/Contrast’. Enter the Contrast Agent / Drug Code in alphanumeric format of 20 characters maximum length or click [List of Values] to select a Contrast Agent / Drug. This is a mandatory field. When Sedation checkbox is selected, only the sedation drugs list will be populated in the Contrast Agents/Drugs LOV.

  7.  - Select  to view the previous contrast/drugs of the exam.

  8.  - Select to view the next contrast/drugs of the exam.

  9. Normal Qty, Vol – The normal quantity of the contrast/drug is displayed with its volume.

  10. Actual Qty, Vol Administered – The actual quantity of the contrast/drug is displayed with its volume. Enter the actual quantity in numeric format of four digits maximum length.

  11. Strength / Qty, Vol – The strength of the contrast/drug is displayed with its volume. If the actual quantity is changed, the system re-calculates the strength and quantity of the contrast and displays the recalculated strength.

  12. Total Strength – The total strength of the contrast/drug is displayed with its volume. If the actual quantity is changed the system re-calculates the total strength of the contrast and displays the recalculated total strength.

  13. Batch ID – Enter the Batch ID in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  14. Stock Qty – The quantity in stock of the contrast/drug used in the exam is displayed. If the actual quantity is changed the system re-calculates stock quantity of the contrast and displays the stock quantity.

  15. Route Code – Enter the Route Code in alphanumeric format of 90 characters maximum length or click [List of Values] to select a Route Code. The description of the selected Route Code is displayed. The Route Codes are defined in ‘Admin Route’ function in ‘Pharmacy Management’ module. This is a mandatory field.

  16. Administered By – Enter the name of the person who has administered the Contrast/Drug in alphanumeric format of 15 characters maximum length.

  17. Adverse Reaction Code – Enter the Adverse Reaction Code in alphanumeric format of four characters maximum length or click [List of Values] to select the Adverse Reaction. Multiple adverse reactions can be recorded. Three rows are displayed, by default. The fourth row is enabled only on entering the three adverse reaction codes with a scroll bar for navigation. The adverse reactions entered are displayed in ‘Record Consumables’ function to view or edit.

  18. Description – The description of the Adverse Reaction Code is displayed.

  19. Remarks – Double click on the Remarks field to display ‘Adverse Reaction Remarks’ screen. Enter the Remarks in alphanumeric format of 500 characters maximum length. This field is mandatory if adverse reaction code is selected. Click [Close] to exit from the Remarks screen. The remarks recorded are made default in ‘Result Reporting’ function. The remarks entered are displayed in ‘Record Consumables’ function to view or edit.

  20. [Contrasts/Drugs used in request] – Click [Contrast/Drugs used in request] to view the added drugs/contrasts used on request during the examination. Once the details are saved using [Items Used] in ‘Exam Completion’ tab, the details can be viewed in this tab.
