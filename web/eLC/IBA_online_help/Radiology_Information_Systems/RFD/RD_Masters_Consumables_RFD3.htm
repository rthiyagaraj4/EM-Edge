Enter the following to create Drug/Consumable Item Class Code:

  1. Drug/Consumable Item Class Code – Enter the Drug/Consumables Item Class Code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Long Description – Enter the long description of the Drug/Consumable Item Class in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description of the Drug/Consumable Item Class in alphanumeric format of 15 characters maximum length. This is a mandatory field.

If the mandatory values are not entered, the system alerts you to enter them. The mandatory field is highlighted with a unique color code. Only 15 records can be added at a time. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate codes with a unique color code.
