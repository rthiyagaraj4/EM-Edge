Enter the following to create Contrast Agents, Drugs:

  1. Contrast Agent/Drug Code – Enter the Contrast Agent/Drug code in alphanumeric format of 20 characters maximum length. This is a mandatory field.

  2. Long Description – Enter the long description of the Contrast Agent/Drug in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description of the Contrast Agent/Drug in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  4. Contrast Agent/Drug Class – Enter the Contrast Agent/Drug Class in alphanumeric format of four characters maximum length or click [List of Values] to select the Contrast Agent/Drug Class. The description of the selected Contrast Agent/Drug Class is displayed. The Contrast Agent/Drug classes are defined in ‘Drug/Consumable Item Classes’ tab. This is a mandatory field.

  5. Cross-reference Item Code – Enter the Cross-reference item code in alphanumeric format of 20 characters maximum length or click [List of Values] to select the Cross-reference item code. The Cross-reference item codes are invoked from the Sterile Stock module.

  6. Stock Unit – Enter the Unit of Measurement for the consumable or click [List of Values] to select it.

  7. Cross-reference Drug Code – Enter the Cross-reference Drug Code in alphanumeric format of 20 characters maximum length or click [List of Values] to select the Cross-reference Drug Code. The Drug Codes are defined in the Drug Setup function in the Pharmacy Management module.

  8. Prescribing/Administering Unit – The Prescribing/Administering Unit of the selected drug is displayed. This is invoked from the ‘Drug Setup’ function of the Pharmacy Management module. This is a mandatory field.

  9. Sedative Drug – Select this check box to record the drug as sedative.

 

Strength in terms of Prescribing / Administering Unit

Strength, Unit Per

  1. Strength - The Strength per unit is displayed as defined in the ‘Drug Setup’ function of the Pharmacy Management module. The strength is defined in terms of prescribing versus the administration unit. This is a mandatory field.

 

Vol/Qty, Unit

  1. Volume - The quantity of volume per unit is displayed as defined in the ‘Drug Setup’ function of the Pharmacy Management module. This is a mandatory field.

 

Conversion factor

  1. Prescribing/Administering to Stock - The Conversion Factor of the selected Drug is displayed.
