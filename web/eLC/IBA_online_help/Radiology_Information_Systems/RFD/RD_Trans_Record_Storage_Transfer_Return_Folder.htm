Enter the following details:

Transfer From

  1. Facility ID – The logged in Facility Name is displayed.

  2. Folder Location Identity - Select the Folder Location Identity from the list. The listed values are ‘Clinic’, ‘Ward’, ‘Department’, ‘Physician’, ‘Radiology Department’, ‘External’ and ‘Patient’. This is a mandatory field.

  3. Specialty – Enter the Specialty code or click [List of values] to select the Specialty. This is a mandatory field.

  4. Folder Location – On selecting the Specialty, the current location of the folder is displayed.

  5. Transferred By – On selecting the Specialty, the name of the logged in user is displayed.

 

Remarks

  1. Remarks – Enter the Remarks for transferring or returning the folder.

 

Transfer To

  1. Facility ID – Select the facility to which you want to Transfer or Return the folder.

  2. Folder location Identity – Select the Folder Location Identity from the list. The listed values are ‘Clinic’, ‘Ward’, ‘Department’, ‘Physician’, ‘Radiology Department’, ‘External’ and ‘Patient’. This is a mandatory field.

  3. Specialty – Enter the Specialty code or click [List of Values] to select the Specialty. On entering the Specialty code, the description of the selected Specialty code is displayed. This is a mandatory field.

  4. Folder Location - On selecting the Specialty, the current location of the folder is displayed.

  5. Narration – Enter the Narration code or click [List of Values] to select the Narration for transferring the folder.

  6. Remarks – Enter the Remarks to Transfer or Return the folders to another location within the facility.

  7. Click [Show All Folders] to display all the folders from the selected Location.

 

Folder Details

  1. Folder No – The folders in the selected Specialty and Folder Location are displayed.

  2. Patient ID – The Patient ID and Name of the Patient is displayed.

  3. [R] - Click [R] to view the recorded remarks.

  4. Issue – Select the check box to issue the folder.
