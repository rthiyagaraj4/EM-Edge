Enter the following details:

  1. Ordering Facility - The logged in Facility is displayed, by default. Select the Ordering Facility from the list. This is a mandatory field.

  2. Performing Facility – The logged in Facility is displayed, by default. Select the Performing Facility from the drop-down list. This is a mandatory field.

  3. Change To – Select the Facility from the drop-down list. The Performing Facility for orders is changed to the selected Facility. This is a mandatory field.

  4. Wing – Select the Wing from the drop-down list. The Operating Unit/Wings are defined in the Operating Units/Wings function under the Administration menu. This is a mandatory field.

  5. Order Category – The logged in Department Code is displayed, by default.

  6. Order Type – Click [List of Values] to select the Order Type.

  7. Order ID – Enter the Order ID in alphanumeric format of 15 characters maximum length.

  8. Order Date [From-To] – Select the required date range using the Calendar function.

  9. Patient ID - Enter the Patient ID in the length as specified in the MP Parameter or right click ‘Right Click for User Options’ hyperlink and select the Patient ID using Hospital Patient Lookup function.

  10. Reason – Enter the comments for changing the performing facility for the orders in text format of 2000 characters maximum length. This is a mandatory field.
