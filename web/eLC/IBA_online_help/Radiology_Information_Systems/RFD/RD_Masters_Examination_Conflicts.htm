Enter the following details to create Examination Conflicts:

 

Exam 1

  1. Exam Code – Enter the Exam code or click [List of Values] to select the Exam code. The Exams are defined in ‘Examination Codes’ function. This is a mandatory field.

  2. Exam Description - The description of the selected Exam code is displayed.

Exam 2

  1. Exam Code – Enter the Exam code or click [List of Values] to select the Exam code. The Exams are defined in ‘Examination Codes’ function. This is a mandatory field.

  2. Exam Description - The description of the selected Exam code is displayed.

  3. Days – Enter the days of interval required between the two conflicting examinations in numeric format of four digits maximum length.

  4. Hrs:Min – Enter the interval in the format of hh:mm; within which the conflicting examinations must not be conducted.

  5. Stop/Warn – Select the value from the list. The listed values are Stop and Warn. This setup is used while booking an examination. If the two conflicting examinations are booked together without meeting the interval criteria and if you select the ‘Warn’ option, the system displays a warning message while booking. If you select the ‘Stop’ option, the system does not allow you to continue with the booking process.

 

Conflicting Monograph

  1. Monograph – Monograph means information on a subject. Enter the nature of the conflict in free text format.
