Consumables

From this function you can Define various consumables, Contrast Agents and Drugs used in Radiology examinations and Query for the existing Consumable records.

Consumables are the items or drugs that are used once, in performing an exam. You can associate the Cross-reference item code with the items defined in the Sterile Stock or Materials Management module or the Cross reference item code with the drugs defined in the Pharmacy Management module.

Creating a Consumable

From the Masters > Examinations, double click Consumables function.

The Consumables screen gets displayed with three tabs. Contrast Agents, Drugs tab is enabled, by default.

  1. Enter the details in the mandatory (*) fields in the following tabs.

Drug /Consumable Item Classes

Contrast Agents, Drugs

Consumable Items

  1. Click [Save] or press [F10] to save a created Consumable or click [Clear Record] to clear the entered.

  2. Click [Execute Query] or press [F8] to search for existing Consumable records.
