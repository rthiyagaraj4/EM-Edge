Examination Protocols

From this function you can Create protocols to allow radiologist to assign protocols to an examination based on patient conditions while starting the exam.  

This function is used to create protocols to allow the Radiologist to assign the protocols to the examination based on patient conditions while starting the examination so that the Technologist can view/change the protocol details before completing the examination and provide comments for the protocols.

 

You cannot delete any protocol as it can be used in transaction or mapped with the examinations. But you can de-link the protocol from the masters for examination which are not completed.

Creating an Examination Protocol

From the Masters > Examinations > double-click Examination Protocols function.

The Examination Protocols screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Save], or press [F10] to save the protocols created or click [Clear Record] to clear the entered details.

  3. Click [Execute Query] or press [F8] to search for the existing protocols.

If you try to delete any protocol, an error message ‘Linked Protocol cannot be deleted’ is displayed if the protocol is used in the transactions or mapped with the examinations. Similarly, an error message is displayed if you try to add a protocol that already exists.
