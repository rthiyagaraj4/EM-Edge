Exam Categories

From this function you can Create examination categories and Query for the existing Exam category records.

Category is a specifically defined division in a system of classification. From this function, you can create main category and sub-category of radiology examinations. You can associate examinations to the exam categories. These categories are used in statistical reports.

Creating an Exam Category

From the Masters > Examinations, double click Exam Categories function.

The Exam Categories screen gets displayed with two tabs. Main Category tab is enabled, by default.

  1. Enter the details in the mandatory (*) fields in the following tabs.

Main Category

Enter the following to create a Main Category:

  1. Main Category – Enter the Main Category code in alphanumeric format of two characters maximum length. The Main Category code must be fixed two character length. This is a mandatory field.

  2. Long Description – Enter the long description of the Main Category in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description of the Main Category in alphanumeric format of 20 characters maximum length. The description entered in the long description field is displayed, by default. This is a mandatory field.

Category

Enter the following to create a new Category:

  1. Category – Enter the Category code in alphanumeric format of six characters maximum length. The first two characters in the Category code have to be prefixed with the main Category code to which the Category belongs. The Category code must be fixed six character length.

If the first two characters of the Category code do not correspond to any of the main Category codes, the system prompts an error message indicating that ‘The Category Code must have the Main Category code as ‘Prefix’.

  1. Main Category Description – The Main Category description is displayed based on the selected Main Category code.

  2. Long Description – Enter the long description of the Category in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description of the Category in alphanumeric format of 20 characters maximum length. The description entered in the long description field is displayed, by default. This is a mandatory field.

  1. Click [Save] or press [F10] to save a created Main Category or click [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for existing records.

If the mandatory values are not entered, the system alerts you to enter them. The mandatory field is highlighted with a unique color code. Only 10 records can be added at a time. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate codes with a unique color code.
