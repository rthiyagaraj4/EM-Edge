Checklist

From this function you can Create checklist items for a particular exam and Query for the existing Checklists.

A Checklist is a list of items to be checked or verified before performing an action. You can create the list of items such as ornaments, clothes of the patient undergoing an exam and so on. The checklist can be BP checked, weight recorded, and so on. Depending on the type of examination, the checklist list items are grouped under one checklist. Radiologists or Staff Nurses verify the checklist before performing an examination on a patient. You can group checklist items and associate it to a Section.

Creating a Checklist

From the Masters > Examinations, double click Checklist function.

The Checklist screen gets displayed with two tabs. The Checklist tab is enabled, by default.

  1. Enter the details in the mandatory (*) fields in the following tabs.

Checklist

Enter the following details to create a Checklist:

  1. Code – Enter the Checklist code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Description – Enter the description for the Checklist in alphanumeric format of 180 characters maximum length. This is a mandatory field.

  3. Srl. No. – Enter the serial number of the Checklist in numeric format of three digits maximum length. This is a mandatory field.

  4. Checklist Item Code/Description – Enter the Checklist Item code or  click [List of values] to select the Checklist item code. On selecting the checklist item code, the description is displayed. This is a mandatory field.

Checklist Items

Enter the following details to create a Checklist items:

  1. Code – Enter the Checklist item code in numeric format of four digits maximum length. This is a mandatory field.

  2. Description – Enter the description for the Checklist item in alphanumeric format of 180 characters maximum length. This is a mandatory field.

  1. Click [Save] or press [F10] to save a created Checklist or click [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for existing checklist Codes recorded.
