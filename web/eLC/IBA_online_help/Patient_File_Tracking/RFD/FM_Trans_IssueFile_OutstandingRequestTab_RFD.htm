Enter the Requested File Details:

 

  1. FS Location Identity - Select the FS Location identity from the list. The values displayed are Clinic, Nursing Unit, Day Care Unit, Procedure Unit, Practitioner and Department.

  2. FS Location - Select the required FS Location from the list. The list displays all the FS Locations (Permanent, Non-permanent and Temporary locations) for the selected Identity and user rights defined. This field is mandatory.

  3. Reason for Borrowing - On selecting the File Storage Location, the narration specified for the selected FS Location is displayed. Select the applicable reason for borrowing from the list. This field is mandatory..

  4. Requested On Date – Enter the requested date in date format of dd/mm/yyyy or select the date using the calendar function.

  5. Requested By – Enter the name of the person requesting the file or select from the [?] function.

  6. Patient ID – Enter the Patient ID or select it using [?] function. This field is mandatory.

  7. File Type – For the selected Patient ID, the File Types are listed. Select the required type from the list. On selecting the Patient ID, the default File Type associated in the File Type For FS Location is displayed. This field is mandatory.

  8. Volume No - The active Volume no is displayed, by default. Select the required volume number from the list.
