Enter the following details to assign a permanent File Storage Location:

 

  1. Terminal Digit - Enter the Terminal Digit in alphanumeric format of 10 characters maximum length.

  2. File Creation Date - Enter the File Creation Date in dd/mm/yyyy format or use the Calendar function to select the required date.

  3. Patient ID - Enter the Patient ID in alphanumeric format of length as specified in MP Parameter or select the Patient ID using [?] function.  

  4. File Type - On entering the Patient ID, File Types associated to the patient are listed. Select the required File Type. If the patient is associated to a single File Type, it is displayed. This field is mandatory.
