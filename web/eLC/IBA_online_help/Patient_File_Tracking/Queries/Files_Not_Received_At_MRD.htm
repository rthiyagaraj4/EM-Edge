Files not received at MRD

From this function you can Query and view the details of files outstanding to the Permanent File storage location.

Querying for Files not received at MRD

From FM Queries, select the Files not received at MRD function.

The Files not received at MRD screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to select the records that satisfy the query criteria or [Reset] to clear the entered search criteria.

  3. On searching, the file details are displayed based on the receiving FS Locations.

  4. Click [Clear] to clear the result screen.

The orange colour denotes that File is 'Due for return' while the red colour denotes that File is 'Over Due for return'.
