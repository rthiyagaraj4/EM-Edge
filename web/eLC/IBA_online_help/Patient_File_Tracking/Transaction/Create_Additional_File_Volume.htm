Create Additional File Volume

From this function you can Create an additional volume for an existing patient file for a particular patient.

The system displays the current active volume for the selected patient and the new volume number that is to be created.

Creating Additional File Volume

From the FM Transactions, select the Create Additional File Volume function.

The Create Additional File Volume function screen gets displayed.

  1. Enter the Patient ID in alphanumeric format of length as specified in MP Parameter or use [?] function to select the required patient. This field is mandatory.

  2. On selecting the Patient ID, the File Type(s) associated to the patient is displayed. If more than one File Type is associated, select the required File Type from the list. This field is mandatory.

  3. Click [Search] to view the details of the current volume number, status and the new volume number. The displayed values cannot be changed. Click [Clear] to reset the records displayed.

  4. Click [Apply] to save the created additional file volume or [Reset] to clear the entered data.

On recording the transaction, the generated additional volume number is considered ‘Active’.
