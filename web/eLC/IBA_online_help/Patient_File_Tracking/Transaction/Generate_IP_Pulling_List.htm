Generate IP Pulling List

From this function you can Generate the main and supplementary lists of patients expected to visit the hospital.

The permanent File Storage Location of the facility generates a list that contains details of the patients having booking for admission on the required day based on the parameter setup. The File Status are - 'I' - Inside, 'O' - Outside, 'L' - Lost, ' A' - Archived, 'T' - Transit, 'E' - in Other facility and 'R' - 'Returned'.  When the list is generated for the first time, it is considered as the Main list. Subsequent list(s) generated is categorized as Supplementary list.  The list can be generated until the files are tracked to respective Nursing Units. Once the files are auto-tracked, this function cannot be invoked.

Generating IP Pulling List

From the FM Transactions> Track Out to IPW, select the Generate IP Pulling List function.

The Generate IP Pulling List function screen gets displayed.

  1. Select the FS Location from the list. The FS Locations are listed based on the access rights defined for the logged in user. This field is mandatory.

  2. The name of the person generating the pulling list is displayed. In the FM Parameter, if Security Access to be Enforced is applicable, the logged-in User Name is displayed and the field is disabled. In the FM Parameter, if Security Access to be Enforced is not applicable, the field is enabled and you can select the User Name using [?] function.

  3. On selecting the FS Location the query criteria screen gets displayed.

  4. Enter the details in the mandatory (*) fields. <Related Field Description>

  5. Click [Generate] to include files in the pulling list or [Reset] to clear to entered data.

You are informed about the number of files included in the pulling list. Click [OK] to close the message screen. The control shifts back to the main screen.

The generated pulling list provides the files pulled (first time pulling) as Main and subsequent files pulled as Supplementary. You cannot select Main for the Pulling List Type after first pulling list is generated. The system issues an alert message Pulling List Already Generated for Main, Select Supplementary.

Click [OK] and select the Pulling List Type as Supplementary to generate the pulling list. You can generate the pulling list any number of times and all the files pulled after generation of first pulling list are displayed with type as Supplementary.

You must have appropriate rights to invoke this function.
