Link/De-Link/Change Encounter-File Volume

From this function you can Create an association between a patient encounter and the patient file volume. You can modify or cancel an existing association.

This association helps in identifying the file used during the patient encounter. A File Type can be linked to multiple encounters, whereas an encounter can be linked to a File Type only.

Linking/De-Linking/Changing Encounter-File Volume

From the FM Transactions, select the Link/De-Link/Change Encounter-File Volume function.

The Link/De-Link/Change Encounter-File Volume function screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to display the selected encounter details or [Clear] to reset the records displayed in the result area.

  3. On searching, the encounter details of the patient are displayed based on the entered criteria.

  4. Select the File Volume from the list. The File Volume is displayed and disabled for action ‘De-Link’, the File Volume is displayed and available for selection for the action ‘Change’. This field is mandatory.

  5. Select the Link/De-link/Change check box based on the action selected to Link, De-link or Change the association of a patient encounter and the patient file volume.

  6. Click [Apply] to record the association of a patient encounter and the patient file volume or [Reset] to clear the entered data.
