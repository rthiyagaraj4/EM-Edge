Enter the following details to record adhoc discount for a package:

 

  1. + - Click '+' displayed as a hyperlink to view the package details.

  2. Package - The Package code that is subscribed for the patient is displayed.

  3. Sequence - System generated sequence number is displayed as a hyperlink. Click the hyperlink to view package details.

    The package sequence number, episode type, customer group and customer, patient ID and name of the patient is displayed.

    Enter the following details:

    1. %/Amount - Select either Percent or Amount from the list. You can either specify the percentage of base price as the discount amount or specify appropriate amount as the discount amount.

    2. Adhoc Discount Amount - Enter the adhoc discount amount in numeric format.

  4. Description - The description of the associated package is displayed.

  5. Effective [From - To] - The effective date range during which the package is available for the patient is displayed.

  6. Billing Class - The Billing Class associated to the patient is displayed.

  7. Customer Group - The Customer Group associated to the patient is displayed.

  8. Customer - The Customer associated to the patient is displayed.

  9. Package Amount - The amount specified for the package during package definition is displayed. You cannot edit the value.

 

Click [OK] to subscribe the package or [Close] to cancel the transaction.
