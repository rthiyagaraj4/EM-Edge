Package User Access

From this function you can Authorize a user to create, modify, approve, issue refund or discount and cancel a package. You can Query to view the list of users' who are authorized to access package details.

Package User Access function is used to authorize a user to access or modify the package details. It involves multiple operations such as Package definition, Approval for subscription, transfer, discount, refund of services, Adhoc (Manual) refund of package services and Cancellation of packages which requires specific authorization. Only authorized users' are permitted to define, approve, transfer, issue refund or discount on a package. A user can cancel the package associated to a patient, if the user is authorized to cancel. Package authentication is based on the User ID and the User Group ID that is selected through this function.

Authorizing User Access

From the Setup > Package, select Package User Access Function.

The Package User Access screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Query] to view the details recorded.

  3. Click [Apply] to save the details or [Reset] to clear the details.
