Add Bill Settlement Details

You can associate Payers and settlement details by clicking [Add]. You can associate multiple Payers, if the patient has settled the amount using multiple settlement types.

 

Enter the following details to record bill settlement details:

  1. Remaining Settlement Amount - The pending settlement amount is displayed.

  2. Amount Settled - Enter the amount settled by patient for respective Billing Service. You can record multiple payment modes, if the patient pays using multiple settlement types.

  3. Settlement Type - Select a value from the list. The values defined using the 'Settlement Types' function in Patient Billing Manager module gets listed.

  4. Cheque/Card/Bank A/c No. - Enter the cheque, credit card number or the bank account number. This is valid, if the settlement type is Bank Transfer, Cheque, Credit Card or Cashier Cheque. This is a mandatory field.

  5. [Get CC Info] - Click [Get CC Info] to validate Credit Card details. This button is enabled only if the Settlement Type is Credit Card.

  6. Date - Click the Calendar function to select the date. This is a mandatory field.

  7. Bank Name - Click [?] function to select the Bank Name. This field is displayed, if the Settlement Type is Bank Transfer, Cheque, Credit Card or Cashier Cheque. This is a mandatory field.

  8. Branch Name - The Branch name associated to the selected Bank is displayed. This field is displayed, if the Settlement Type is Bank Transfer, Cheque, Credit Card or Cashier Cheque.

  9. Online Approval - The check box is selected, if the 'Settlement Type is 'Credit Card'. You cannot select or clear the check box.

  10. Batch No - Enter the Batch Number in alphanumeric format. This field is displayed, if the Settlement Type is Bank Transfer, Cheque, Credit Card or Cashier Cheque. This is a mandatory field.

  11. Approval Code - Enter the Approval Code in alphanumeric format. This field is displayed, if the Settlement Type is Bank Transfer, Cheque, Credit Card or Cashier Cheque. This is a mandatory field.

  12. Sale Draft No - Enter the Sale Draft Number in alphanumeric format. This field is displayed, if the Settlement Type is Bank Transfer, Cheque, Credit Card or Cashier Cheque. This is a mandatory field.

  13. Cash Given - Enter the amount paid by the patient. Cash given should be greater than or equal to the amount settled.

  14. Cash Returned - The value is displayed automatically, if the Cash Given is greater than the Amount Settled. You cannot edit the value.

The above mentioned fields vary based on the 'Settlement Type' selected.

  1. Payer's Name - The Payer's Name is defaulted to the patient's name. You can update the Payer's Name.

  2. Payer Ref Id/Deposit Ref No. - Enter the Payer's Reference ID or the Deposit Reference Number in alphanumeric format of 30 characters maximum length.

  3. Click [Accept] to record details or [Cancel] to abort the transaction. The Payment Details get listed in the Bill Settlement Details' screen under 'Payment Details'.
