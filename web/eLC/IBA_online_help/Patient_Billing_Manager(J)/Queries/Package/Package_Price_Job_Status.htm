Package Price Job Status

From this function you can Query to view the package Price Job Status.

Package Price Job Status function is used to query the status of the package which have been revised using the package price revision screen. You can view the statuses of the package prices with error and without errors.

Querying Patient Price Job Status

From the Queries > Package, select Package Price Job Status Function.

The Package Price Job Status screen gets displayed.

  1. Click [?] to select the User ID and Job Id.

  2. Select the Process date using calendar function <Related Field Description>

  3. Click [Execute] to search job status of the package price. The Serial Number, Job Id and Job Status are displayed under the Job list section.

  4. Select the radio button before the serial number to view the package details of the patient in the tabs Processed Without Error

    The Following price details are displayed:in the Processed Without Error

     

    The Package Code, Package Description, Std/Customer Price, Bulk/Package Price, Billing Class, Package Price Type, Percentage Price Type, Percentage /Amt, Mark Up/down, Round-Off,  Round Indicator, Effective [From-To]

     

    The following discount details are displayed:

     

    Package Code, Package Desc, Billing Group, Billing Class, Revised Discount Value, Revised Discount Type, Effective [From-To], Error Desc

and Processed With Error.

The Following price details are displayed in the Processed With Error

 

The Package Code, Package Description, Std/Customer Price, Bulk/Package Price, Billing Class, Package Price Type, Percentage Price Type, Percentage /Amt, Mark Up/down, Round-Off,  Round Indicator, Effective [From-To]

 

The following discount details are displayed:

 

Package Code, Package Desc, Billing Group, Billing Class, Revised Discount Value, Revised Discount Type, Effective [From-To], Error Desc

  1. Click [Excel] to export the packages affected in the job to the excel sheet.
