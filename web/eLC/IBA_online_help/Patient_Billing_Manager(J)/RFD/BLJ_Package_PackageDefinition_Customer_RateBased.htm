Enter the following details to define and associate rate based services to the package:

 

  1. Type - Select a Type from the list. The system-defined values are 'Service Classification, 'Billing Service' and 'Service Group'. Select 'Service Group' to define price for a Service Group and Service Classification to define price for a Service Classification. Select 'Billing Service' to define price for a Billing Service.

  2. Billing Service - Click [?] function to select a Billing Service associated to the selected Service Group or Service Classification.

  3. Order Catalog - Click [?] function to select and associate an Order Catalog to the Package. This field is enabled for selection only if the 'Episode Type' is selected as 'OP' and the 'Order Associated' check box is selected. The Order Catalogs associated to the selected Billing Service are displayed based on the definitions in 'Order Catalog Billing Details' function.

  4. Amount Indicator - Select a value from the list. The system-defined values are 'Standard', 'User Defined', 'As Per Rule' and 'Price Class'. Select 'Standard' to use the standard price defined for the Billing Service. Select 'As Per Rule' to use the Customer defined price for the Billing Service. Select 'User Defined' to use the price defined under the 'Package Base Price' tab. Select 'Price Class' to use the price defined in the 'Price Class Specific Price' function. The 'User Defined' indicator is valid only for 'Billing Service' and not valid for 'Service Group' or 'Service Classification'.

  5. Factor/Rate - Enter the factor or rate in numeric format of 12 digits maximum length. The amount is calculated as (Factor*Base Price).

  6. Quantity Limit - Enter the restriction on the quantity of selected Billing Service that can be associated to the package. If the usage of service is more than the limit specified, the service is not considered to be a part of the package and the standard price for the Billing Service is charged to the patient.

  7. %/Amount Indicator - Select a value from the list. The system-defined values are 'Percent', 'Amount' and 'Unlimited'. Select 'Percent' to define the service amount as the percentage of the base price for the selected quantity. Select 'Amount Indicator' to specify valid amount for the service and the quantity specified. If the usage of service exceeds the quantity or the amount specified, standard price of the respective Billing Service is charged to the patient. Select 'Unlimited' if there is no restriction on the amount that can be associated to the respective Billing Service for the quantity utilized.

  8. %/Amount - Enter the value in numeric format based on the value selected for '%/Amount Indicator'. If the usage of service exceeds the amount specified, the standard price defined for the Billing Service is considered. You cannot specify the value, if the indicator is selected as 'Unlimited'.

  9. Round To - Enter the value in numeric format. This indicates the value to which the price can be rounded off.

  10. Round Indicator - Select a value from the round indicator. The system-defined values are 'Up' and 'Down'. Select 'Up' to round off the value to the highest nearing value specified for 'Round To'. Select 'Down' to round off the value to the lowest nearing value specified for 'Round To'. For Example, if the package value is 3,515 and the 'Round To' is set as 10 with the 'Round Indicator' as 'Up', the package price is rounded off to 3520.

  11. Replace - Select the check box if the Billing Service is replaceable with any other service. A legend 'Dtls' is displayed as a hyperlink when you select the check box. A new screen pops-up. Enter the details in the pop-up screen. Click 'Dtls' hyperlink to view the details that are recorded. The check box is enabled only for 'Billing Service' and disabled for 'Service Group' or 'Service Classification'.

    Enter the following details:

    1. Service Code - The selected Service Code is displayed, by default.

    2. Service Description - The description of the selected Service Code is displayed, by default.

    3. Order Catalog Code - The selected Order Catalog Code is displayed, by default.

    4. Order Catalog Desc - The description of the selected Order Catalog Code is displayed.

    5. Replaceable Service - Click [?] function to select a Billing Service with which the existing Billing Service can be replaced.

    6. Order Catalog - Click [?] function to select a Order Catalog with which the existing Order Catalog can be replaced. This field is enabled for selection only if the 'Order Catalog' is associated to the selected Billing Service.

    Click [OK] to save the details or [Cancel] to cancel the process.

  12. Refund Rate % - Select the check box to specify refund rate for the Billing Service. Enter the value in the adjacent text box that can be issued as a refund, if the Billing Service is not utilized when the package is availed by the patient. The check box is enabled only for 'Billing Service' and disabled for 'Service Group' or 'Service Classification'.

  13. Auto Refund - Select the check box if the Billing Service is auto refundable. The amount is automatically refunded, if the Billing Service is not utilized by the patient when the package is availed. The check box is enabled only for 'Billing Service' and disabled for 'Service Group' or 'Service Classification'.

  14. Apply Factor for Service Limit - Select the check box to define the factor available for service limit. This field is enabled for selection only if the 'indicator' value is selected as 'factor through the 'Package Base Price' tab'.

  15. Close Package - Select the radio button to indicate that the package can be closed when the corresponding Billing Service is associated to the patient. The radio button is enabled only for 'Billing Service' and disabled for 'Service Group' or 'Service Classification'.

  16. Service Limit (Dtls) - Select the check box to associate services when the 'Type' is selected as either 'Service Group' or 'Service Classification'. This is not enabled for 'Billing Service'. A legend 'Dtls' is displayed as a hyperlink when you select the check box. A new table appears enabling you to associate Service Classification or Billing Service to the Service Group or Service Classification. Click 'Dtls' hyperlink to view the details that are recorded. The check box is disabled for 'Billing Service' and enabled for 'Service Group' or 'Service Classification'.

  17. Select - Select the check box to associate the Billing Service, Service Group or Service Classification to the package.

  18. [Add] - Click [Add] to associate the Billing Service, Service Group or Service Classification to the Package.

  19. [Delete] - Click [Delete] to delete a Billing Service, Service Group or Service Classification that is associated to the package.

  20. Type - Select a Type from the list. The system-defined values are 'Service Classification' and 'Service Group'. Select 'Service Group' to define price for a Service Group and Service Classification to define price for a Service Classification.

 

The following tabular form is displayed, if the 'Type' is selected as 'Service Group' or 'Service Classification':

  1. Type - Select a Type from the list. The system-defined values are 'Service Classification' and 'Billing Service'. Select 'Service Classification' to define price for Service Classification that is associated to the selected Service Group. Select 'Billing Service' to define price for a Billing Service that is associated to the Service Classification and Service Group.

  2. Billing Service - Click [?] function to select a Billing Service associated to the selected Service Group or Service Classification.

  3. Exclude - Select the check box to exclude Service Code or Service Classification from the package. On selecting the check box, other fields are disabled from selection except 'Select'. Clear the check box, if the Service is associated to the package.

  4. Quantity Limit - Enter the quantity limit in numeric format to indicate the quantity of service that can be associated to the package.

  5. %/Amount Indicator - Select a value from the list. The system-defined values are 'Percent' and 'Amount'. Select 'Percent' to define the package amount as the percentage of the base price. Select 'Amount Indicator' to specify valid amount for the package. The amount indicator is based on the quantity whereas the percent is not dependent on the quantity.

  6. Amount Limit - Enter the amount limit in numeric format. It specifies the limit on the coverage for the respective quantity of service in the package. If the quantity of the service used exceeds the limit, the standard price defined for the service is considered.

  7. Round To - Enter the value in numeric format. This indicates the value to which the decimal value can be rounded off.

  8. Round Indicator - Select a value from the round indicator. The system-defined values are 'Up' and 'Down'.

  9. Replace - Select the check box if the Billing Service is replaceable with any other service. A legend 'Dtls' is displayed as a hyperlink when you select the check box. A new screen pops-up. Enter the details in the pop-up screen. Click 'Dtls' hyperlink to view the details that are recorded. The check box is enabled only for 'Billing Service' and disabled for 'Service Group' or 'Service Classification'.

    Enter the following details:

    1. Service Code - The selected Service Code is displayed, by default.

    2. Service Description - The description of the selected Service Code is displayed, by default.

    3. Order Catalog Code - The selected Order Catalog Code is displayed, by default.

    4. Order Catalog Desc - The description of the selected Order Catalog Code is displayed.

    5. Replaceable Service - Click [?] function to select a Billing Service with which the existing Billing Service can be replaced.

    6. Order Catalog - Click [?] function to select a Order Catalog with which the existing Order Catalog can be replaced. This field is enabled for selection only if the 'Order Catalog' is associated to the selected Billing Service.

    Click [OK] to save the details or [Cancel] to cancel the process.

  10. Refund Rate % - Select the check box to specify refund rate for the Billing Service. Enter the value in the adjacent text box that can be issued as a refund, if the Billing Service is not utilized when the package is availed by the patient. The check box is enabled only for 'Billing Service' and disabled for 'Service Group' or 'Service Classification'.

  11. Auto Refund - Select the check box if the Billing Service is auto refundable. The amount is automatically refunded, if the Billing Service is not utilized by the patient when the package is availed. The check box is enabled only for 'Billing Service' and disabled for 'Service Group' or 'Service Classification'.

  12. Factor Applicable for Service Limit - Select the check box to define the factor available for service limit. This field is enabled for selection only if the 'indicator' value is selected as 'factor through the 'Package Base Price' tab'.

  13. Select - Select the check box to associate the Service Group or Service Classification to the package.

  14. [Add] - Click [Add] to associate the Service Group or Service Classification to the Package code.

  15. [Delete] - Click [Delete] to delete a Service Group or Service Classification that is associated to the package.
