Enter the following details to define the Health Card:

  1. Card Type – Enter the Card Type in the character format of 30 characters maximum length.  This is a mandatory field.

  2. Long Description - Enter the long description for the Card Type in alphanumeric format of 100 characters maximum length.  This is a mandatory field.

  3. Short Description - Enter the short description for the Card Type in alphanumeric format of 25 characters maximum length.  This is a mandatory field.

  4. Card Validity(Days) – Enter the Card Validity in the numeric format of five character maximum length. This is a mandatory field.

  5. No Of Patients Allowed – Enter the number of patients allowed for consultation in numeric format of five characters maximum length. This is a mandatory field.

  6. Card Applicability - Click the Calendar function to select the date range. This is a mandatory field.

  7. Enabled – Select this check box to enable the Card Applicability. This check box is selected by default.

 

Age Group

  1. From - Enter the Days, Months and Years details of the patient’s date of birth in the field D, M and Y.

  2. To – Enter the current date, month and year details in the field D, M and Y.

  3. Price Class – Click [?] to select the price classes. The price class is associated with the billing services.

  4. Membership Fees – Select the membership fees based on the age limit of the patient. If the patient age is greater than sixty years, the membership fee for senior citizen is to be selected.

  5.  Click [Add] to include another membership details in the Age Group section.

 

OP Allotment

  1. No Of OP Allotment - Enter the number of OP Allotment in the numeric format.

  2. Click [Add] to define the number of OP allotment details.

  3. Clinic – Click [?] to select the Clinic. The clinics defined in the Outpatient Management module are listed.

  4. Visit Type – Click [?] to select the Visit type. The visit types defined in the Outpatient Management module are listed.

  5. # of Visit – Enter the number of visits allotted for the patients in a numeric format. The patient is allowed to visit the hospital at the number of times entered using this health card.

 

Applicable Billing Group Details

  1. Billing Group Code – Click [?] to select the Billing Group Code. The Billing Rules defined in the Billing Rule masters are listed.

  2. Description - The Description for the selected Billing Group gets displayed.

 

The sum of number of Visits can not be greater than the No Of OP Allotment.
