Enter the following details to authorize a user:

 

  1. User ID - Click [?] function to select a User ID. The values created through the 'Users' function are listed. This is a mandatory field.

  2. User Name - The name of the selected User ID is displayed. This is a mandatory field.

  3. User Group ID - Click [?] function to select a User Group ID. The values created through the 'User Groups/Users' function are listed. This is a mandatory field.

  4. User Group Name - The name of the selected User Group is displayed.

  5. Package Creation - Select the check box if the user is authorized to define packages through the 'Package Definition' function. Clear the check box if the user is not authorized to define packages.

  6. Package Approval - Select the check box if the user is authorized to approve packages associated to a patient during transactions. Clear the check box if the user is not authorized to approve packages.

  7. Package Transfer - Select the check box if the user is authorized to transfer packages associated to a patient during transactions. Clear the check box if the user is not authorized to transfer packages.

  8. Package Discount -  Select the check box if the user is authorized to issue a adhoc discount on the package associated to a patient during transactions. Clear the check box if the user is not authorized to issue a discount.

  9. Package Refund - Select the check box if the user is authorized to issue a refund on the unavailed services or refundable services that are included in the package. Clear the check box if the user is not authorized to issue a refund.

  10. Package Cancel - Select the check box if the user is authorized to cancel the subscribed package. Clear the check box if the user is not authorized to cancel the subscribed package.

  11. Package Adhoc Refund - Select the check box if the user is authorized to issue a adhoc refund on the unavailed services or the refundable services that are included in the package. Clear the check box if the user is not authorized to issue a refund.
