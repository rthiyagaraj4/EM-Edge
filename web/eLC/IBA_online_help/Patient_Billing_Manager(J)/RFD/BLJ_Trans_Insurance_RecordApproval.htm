Enter the following details to record the approvals:

 

  1. Patient No – Enter a valid patient ID or click [?] to search for a valid patient ID. A Lookup window gets displayed on clicking the [?]. Double-click on a relevant patient record to select it. The patient name, age and gender get displayed along with the Episode Type. This is a mandatory field.

  2. Episode ID – Click [?] to select a valid Episode ID. You have to select an Insurance related encounter to record the approval for the associated material items or the Billing Services.

  3. Payer Group – Click [?] to select the Payer Group. The Payer Groups defined in the function Customer under the ‘AR Setup’ get listed.

  4. Encounter Date [From-To] – Select the Encounter Date range using the calendar function. This is a mandatory  field.

  5. Payer - Click [?] to select the Payer. The Payer Groups defined in the function Customer get listed.

  6. Policy - Click [?] to select the Policy.

  7. Request Status – Select the Request Status from the drop-down list. The listed values are Approved, Rejected, and Pending.

 On entering the required details, the financial details of the patient recorded during admission get displayed. If the financial details were updated, the updated financial details get displayed.

  1. Exclude/Include Criteria – Select a value from the list. The system defined values are ‘Billing Service’, ‘Payer Service Group’ and ‘Payer Service Group/Line’.

  2. Billing Service Code – Click [?] to view and to select a Billing Service Code. The Billing Services defined in the Patient Billing Manager module get listed.

  3. Description – The description for the selected Billing Service gets displayed.

  4. Approval Status – Select a value from the list. The system defined values are ‘Approved’, ‘Pending’ and ‘Rejected’.

  5. Approved – Select Approved to approve the request.

  6. Pending – Select Pending and the service is not associated to the patient encounter.

  7. Reject – Select Reject to reject the request placed.

  1. Approval Number – Enter the approval number in alphanumeric format of 60 characters maximum length. This field is enabled to enter the approval number, only on selecting the ‘Approval Status’ as ‘Approved’.

  2. Service Date – Click Calendar function to record the date when the request for the service was placed. This field is enabled to record the request date, only on selecting the ‘Approval Status’ as ‘Approved’. Request Date must not be more than the Given Date.

  3. Given Date – Click Calendar function to record the date when the request for the service is approved. This field is enabled to record the date, only on selecting the ‘Approval Status’ as ‘Approved’.

  4. Remarks – Enter the remarks in alphanumeric format to indicate the reason.
