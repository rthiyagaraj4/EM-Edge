Terminology Code Evaluation Measures

From this function you can Associate the Evaluation Measure against the Terminology Code.

This function enables you to associate Evaluation Measures against a Terminology code. The system evaluates the measure in the Evaluate Care Plan.

Creating Terminology Code Evaluation Measure

From the CP Codes Setup > CP Code Setup, select the Terminology Code Evaluation Measures function.

The Terminology Code Evaluation Measures screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields.   <Related Field Description>

  3. Click [Associate] to associate the Terminology Code with the Evaluation Measures. The Associated Measures section gets displayed.

    Enter the following details:

    1. ID – Enter the Outcome measure ID for the evaluation measure in alphanumeric format of two characters maximum length. This is a mandatory field.

    2. Description – Enter the Description of the expected outcome in alphanumeric format of 30 characters maximum length.

    3. Cut Off Score – Enter the minimum Cut Off Score value for a particular out come. The minimum value is three.

    4. Sequence No – Enter the Sequence Number of the Outcome measure in alphanumeric format of two characters maximum length. The outcome measure gets arranged in a sequential order.

  4. Click [Add] to add the associated measure on the associated measure grid.

  5. Click ID, which is a hyperlink of the added associated measure to view the entered associated measure values in the respective fields. You can modify the values once again by clicking [Add].

  6. Click [Apply] to create a new Evaluation Measure applicable to the transaction or click [Reset] to clear the entered the data.
