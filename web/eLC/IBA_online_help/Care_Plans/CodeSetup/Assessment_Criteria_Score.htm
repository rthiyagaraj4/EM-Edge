Assessment Criteria Score

From this function you can Associate  numeric values for assessment criteria.

You can allocate the numerical scores in order to classify the results of the assessment in to Normal, Abnormal and Critically Abnormal.

Associating a numeric value to the Assessment

From the CP Codes Setup > CP Code Setup, select the Assessment Criteria Score function.

The Assessment Criteria Score screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Assessment Note from the list. This is a mandatory field.

  3. Select the Assessment Category from the list. The assessment categories attached to the selected assessment note are displayed. This is a mandatory field

  4. The reference ranges for Normal, Abnormal and Critically Abnormal status is displayed.

  5. Click [Search]. The Criteria Description, Type and Response get displayed.

  6. Select the score against the Criteria Description for each response.

  7. Click [Apply] to save the changes or [Reset] to clear the entered data.

  8. Click [Query] to invoke the Query screen and to search for the existing records.
