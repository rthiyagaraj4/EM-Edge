Outcome

From this function you can Create the outcome of the Care delivered.

You can create outcomes that are either Goals to be achieved to solve the problem identified or Expected Outcomes of any intervention performed to achieve the goal. Goals are marked against a problem where as the Expected Outcomes are marked against an individual intervention in order to achieve the goal. You can associate a measure for the outcome.

Creating an Outcome

From the CP Codes Setup > CP Code Setup, select the Outcome function.

The Outcome screen gets displayed.

  1. Click [Create] to define new Outcome.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Outcome created. A dialog box stating ‘Do you want to associate outcome evaluation measures?’ gets displayed.

  4. Click [Cancel] to cancel the recording of the Evaluation measure. Click [OK] to associate evaluation measure the Outcome Evaluation Measure screen gets displayed.

    Enter the following details:

    1. ID – Enter the Outcome measure ID for the evaluation measure in alphanumeric format of two characters maximum length. This is a mandatory field.

    2. Description – Enter the description of the expected outcome in alphanumeric format of 30 characters maximum length.

    3. Cut off score – Enter the minimum cut off score value for a particular out come. The minimum value is three.

    4. Sequence No – Enter the sequence number of the Outcome measure in alphanumeric format of two characters maximum length. The outcome measure gets arranged in a sequential order.

    On entering the required details, click [Apply]. The values entered get displayed on the top frame of the associate evaluation measure. Click [OK].

    Click on the ID, which is a hyperlink to view the values entered. The values can be modified and saved or deleted for the outcome measure created.

  5. Click [Query] to search the existing Outcome record within the facility.
