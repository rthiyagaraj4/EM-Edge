Assessment Based Terminology Code

From this function you can Associate the Terminology Codes to an Assessment Category.

The terminology codes which are associated to the assessment note is displayed when you record data in the note when any criteria value recorded is found as abnormal.

Associating a Terminology Code to an Assessment Category

From the CP Codes Setup > CP Code Setup, select the Assessment Based Terminology Code function.

The Assessment Based Terminology Code screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Search]. Terminology Code and Description gets displayed.

  4. Click ‘Associated’ hyperlink to view the already associated terminology code to the assessment category.

  5. Click any of the alphabets hyperlink, the list of terminology codes starting with the corresponding alphabet are listed.

  6. Select the check box against the Terminology Code to be associated with the Assessment Category or select the ‘Select All’ check box in the column header to associate all the terminology codes by default.

  7. Click [Apply] to save the changes or [Reset] to clear the entered data.

  8. Click [Query] to invoke the Query screen and to search for the existing records.
