Age Group for Assessment Note

From this function you can Associate the Assessment Note to one or more age groups.

This function enables you to associate the assessment note to one or more age groups defined in Application Masters. Assessment notes are used to enter the assessment details. You can Query for the existing records.

Associating an Age Group to an Assessment Note

From the CP Codes Setup > CP Code Setup, select the Age Group for Assessment Note function.

The Age Group for Assessment Note screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Assessment Note using the [?] function. The assessment notes are defined in the Clinician Access module. This is a mandatory field.

  3. Click [Search]. Age Group, Sex Specific, Min Age and Max Age get displayed.

  4. Select the check box against the Age Group to be associated with the Assessment Note or select the ‘Select All’ check box in the column header to associate all the age groups by default.

  5. Click [Apply] to save the changes or [Reset] to clear the entered data.

  6. Click [Query] to invoke the Query screen and to search for the existing records.
