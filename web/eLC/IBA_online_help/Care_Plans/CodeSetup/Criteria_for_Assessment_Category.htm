Criteria for Assessment Category

From this function you can Associate template components in the assessment note to the assessment category selected.

Template component is associated to an assessment category whenever you record any data into the component, there is a score calculated by the system. Eventually the assessment category to which the component is associated is updated with the score.

Associating Criteria to an Assessment Category

From the CP Codes Setup > CP Code Setup, select the Criteria for Assessment Category function.

The Criteria for Assessment Category screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Assessment Note using the [?] function. This is a mandatory field.

  3. Select the Assessment Category from the list. The assessment categories attached to the assessment note is displayed. This is a mandatory field.

  4. Click [Search]. Section, Sub-Section and Criteria are displayed.

  5. Select the check box against the Criteria to be associated with the Assessment Category.

  6. Click [Apply] to save the changes or [Reset] to clear the entered data.

  7. Click [Query] to invoke the Query screen and to search for the existing records.
