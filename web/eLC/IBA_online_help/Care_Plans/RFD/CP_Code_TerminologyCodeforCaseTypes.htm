Enter the following details:

  1. Case Type – Select the Case Type using [?] function. This is a mandatory field.

  2. Case Type Nature – The Case Type Nature selected for the case type in the ‘Case Type’ function is displayed.

  3. Terminology Set – Select the Terminology Set from the list. The terminology sets are populated from the Terminology Set master from the Medical Records module. This is a mandatory field.

  4. Terminology Code Description - Enter the Terminology Code Description in alphanumeric format of 60 characters maximum length.

  5. List value – Select from the list values. The listed values are Starts With, Contains and Ends with.

  6. Type – Select the type from the list of valid values. The listed values are Procedure and Diagnosis.
