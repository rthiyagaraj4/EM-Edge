Enter the following details:

  1. Indicator – Select the Indicator from the option. If the outcome defined is a

    • Goal - Select Goal as the indicator.

    • Expected Outcome - Select Expected Outcome as the indicator.

  2. Code – Enter the Code in alphanumeric format of 10 characters maximum length.

  3. Description – Enter the Description of the expected outcome in alphanumeric format of 60 characters maximum length.

  4. Explanatory text – The text to explain the description of the expected outcome in detail can be entered in the field. Enter the Explanatory text in alphanumeric format of 400 characters maximum length.

  5. Enabled – Select the check box to enable the outcome being defined in the Transactions.
