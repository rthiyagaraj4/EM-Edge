Item

From this function you can Define Inventory Management related attributes for an Item defined in Materials Base with category type as ‘Inventory’ and Query for a desired Item.

The Items can be both medical or non-medical. These Items are further classified as expiry related Items (mostly drugs) or a non-expiry related Items. The Items are classified as manufactured or non-manufactured Item. The Unit Cost of each Item is maintained as an average cost that is calculated whenever stock receipts are generated.

Defining an Item

From the Code Setup, select the Item function.

The Item screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the defined Item or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Item.
