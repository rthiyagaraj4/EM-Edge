Assembly Instructions

From this function you can Create and maintain various Assembly Instructions and Query for the desired Assembly Instructions.

The Assembly Instructions are created for certain composition of Ingredients prepared in terms of decimal values like Preservatives, Calories, Agents and so on. Enterprise Management is accepting only integer values against the ingredients quantity in Assembly Details function.

Creating a Repacking Group

From the Code Setup, select the Assembly Instructions function.

The Assembly Instructions screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory fields. <Related Field Description>

  3. Click [Apply] to save the Assembly Instructions created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Assembly Instruction records.
