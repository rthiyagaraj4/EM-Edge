Item Store Bin Location

From this function you can Associate an Item in a Store to Bin Locations and Query for the desired association of Item in a Store to Bin Location.

The Bin Locations are defined in the Materials Base module of Enterprise Management. The Bin Location is associated to an Item while defining the Item in Materials Base module or you can associate the Bin Location to Item and Store combination in this function.

Creating an Association of Item in a Store to Bin Locations

From the Code Setup, select the Item Store Bin Location function.

The Item Store Bin Location screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Search] to display Bin Locations and select the check box against the desired Bin Location.

  4. Click [Apply] to save the association of Bin Location to Item in a Store or [Reset] to clear the entered data.

  5. Click [Query] to search for associated Bin Location for an Item in a Store records.
