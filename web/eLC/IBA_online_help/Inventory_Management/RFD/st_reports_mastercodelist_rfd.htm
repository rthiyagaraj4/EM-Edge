Enter the following report criteria to generate a report for the selected code:

  1. Item [From-To] - Enter the range of Item Codes for which the report is to be generated in alphanumeric format of 20 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected item code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ item code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Code and select required values.

  2. Alpha code [From-To] - Enter the range of alpha codes for which the report is to be generated in alphanumeric format of 10 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected alpha code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ alpha code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  3. Item Class [From-To] - Enter the range of Item Class for which the report is to be generated in alphanumeric format of four characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Class code and select required values.

  4. Machine Code [From-To] – Enter the range of machine codes for which the report is to be generated in alphanumeric format of 10 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected machine code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ machine code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the machine code and select required values.

  5. Model Number [From-To] - Enter the range of machine model numbers for which the report is to be generated in alphanumeric format of 20 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected model number code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ model number code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  6. Part Number [From-To] - Enter the range of part numbers for which the report is to be generated in alphanumeric format of 20 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected model number code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ model number code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  7. Sterile Type – Enter the sterilization type for which the report has to be generated in alphanumeric format of four characters maximum length or click [?] function to invoke the search screen and you can select the required type by clicking the hyperlink.

  8. Item Analysis – Select the required value for ‘Analysis1’, ‘Analysis2’ and ‘Analysis3’ from the list of valid values.

  9. Expiry Item – Select the required nature from the list of valid values. The lists of valid values are ‘All’, ‘Expiry Items’ and ‘Non Expiry Items’. The value ‘All’ is displayed, by default.

  10. Nature - Select the report nature as, ‘Enabled’, ‘Disabled’ or ‘Both’. The item records with the matching status are included in the report. The value ‘Both’ is displayed, by default.

  11. Order By - Select the Sort Order as ‘Item Code or ‘Item Description’ to produce the report in the required sorting order. The value ‘Item Code’ is displayed, by default.

  12. Report Type – Select the required report type to generate the report from the list of valid values. The lists of valid values are ‘Summary’ and ‘Detail’. The value ‘Summary’ is displayed, by default.
