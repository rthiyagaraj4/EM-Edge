Range of Items

Enter the following details to associate Items to a group of Stores:

 

  1. Store [From-To] – Enter the range of Store codes to which the Items must be associated in alphanumeric format of six characters maximum length or select the Store using [?] function.

  2. Item [From-To] - Enter the range of Item codes, which must be associated to the Store in alphanumeric format of 20 characters maximum length or select the Item using [?] function.

  3. Item Analysis 1 [From-To] - Select the range of Item Analysis 1 values from the list.

  4. Item Analysis 2 [From-To] - Select the range of Item Analysis 2 values from the list.

  5. Item Analysis 3 [From-To] - Select the range of Item Analysis 3 values from the list.

  6. Stock Item – Select the check box, if the Item is a Stock Item in the Store.

  7. Min Stock – Enter the minimum stock of the Item that requires to be Stored in the Store in numeric format of 12 digits maximum length.

  8. Max Stock – Enter the maximum stock of the Item that requires to be stored in the Store in numeric format of 12 digits maximum length.

  9. RO Level - Enter the reorder level of the Item when the Item required to be reordered in the store in numeric format of 15 digits maximum length. The reorder level cannot be lesser than minimum stock level or an error message will be displayed.

  10. Reorder Freq Days – Enter the reorder frequency days in numeric format of three digits maximum length for reordering the Items in the Store.

  11. Req Generation Base – Select the basis for request generation from the list. The values listed are ‘By Stock Level’, ‘By Order’, ‘By Dispense’, ‘By Administration’ and ‘By Manual’. The system displays ‘By Stock Level’, by default. Items having Request Generation Base set to ‘By Manual’ are considered in the ‘Generate Request’ function.

  12. Buffer Percentage - Enter the Buffer Percentage in numeric format. This field is enabled on selecting the ‘Stock Item’ check box.

  13. Lead Days – Enter the lead days for refilling the stock from the Parent Store in numeric format of four digits maximum length. This field is enabled on selecting the ‘Stock Item’ check box.

  14. Min Stock Days - Enter the minimum stock days within which the Items have to be stored in numeric format of four digits maximum length. This field is enabled on selecting the ‘Stock Item’ check box.
