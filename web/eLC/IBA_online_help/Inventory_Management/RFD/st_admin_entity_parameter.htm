Enter the following details to define Entity Parameter rules:

  1. Current Processing Month/Year – This period denotes the Inventory Management module’s processing period. This period should always denote the current processing month and year and not the calendar month and year. This month and year will be automatically updated to the next period during the month end processing. The processing month and year cannot be changed, once they are entered and the changes are committed. This is a mandatory field.

  2. Audit Trail - Click [Audit Trail] to view the Audit Trail details. The Audit Trail screen is displayed. The modified details are displayed. The displayed details are: Modified By, Modified Date/Time, Facility Name, Expiry Warning Days, Transaction across facility and Allow Multi Store Issue.

 

Interface

  1. Billing – Select the check box to enable Patient Billing Manager module of Enterprise Management with the Inventory Management module.

  2. General Ledger – Select the check box to interface General Ledger module of Enterprise Management with the Inventory Management module.

  3. Sale Price Applicable – Select the check box if Sale Price is applicable. This check box is enabled, if ‘Billing’ check box is selected.

  4. Folio At Batch Price – Select the check box to consider the price at Batch Level or clear the check box to consider the price at Item Level. This check box is enabled, if Billing check box is selected.

  5. MRP Greater than Unit Price – Select the check box to accept the MRP if the MRP is greater than the Unit Price. This check box is enabled if ‘Sale Price Applicable’ check box is selected.

  6. PO Applicable – Select the check box to accept PO details in the Goods Received Note function.

  7. PO Module Installed – Select the check box to activate the Purchase Order module. The ‘Interface to External PO’ and ‘PO Applicable’ check boxes are disabled.

  8. Interface to external PO – Select the check box to interface to the external Purchase Order interface.

  9. Generate PO Request By – Select the value from the list. The listed values are ‘None’, ‘ Material Group’, and ‘ Item Class’. The value ‘None’ is displayed, by default.

    If Item Class is selected, the Purchase Order Request is generated based on the Item Class.

    If Material Group is selected, the Purchase Order Request is generated based on the Material Group.

 

Default Stock Parameter

  1. Expiry Warning Days – Enter the expiry warning days in numeric format of three digits maximum length. Based on the value entered, a warning indication is displayed for the Items nearing expiry in the transaction functions.

  2. Transaction across facility – Select the check box to perform transactions across the facilities.

  3. Allow Multi Store Issue – Select the check box to allow issuing Stocks from two Stores for a single requisition. If the check box is cleared, Stocks are issued only from one store.

  4. Restrict Viewing Stock Status - Select the value from the list to restrict the stores from viewing the reports and queries of the other stores. If values 'User Access to Stores' and 'All Stores' are displayed in the list. If the value 'User Access to Stores' is selected, you can view/print stock status details only to stores with privilege rights. If the value 'All Stores' is selected, you can view/print stock status details for all stores. The user access rights for the stores are defined in 'User Access for Store' function.

  5. No of Decimals - Enter the number of decimal value to display in all the transactions functions, reports and queries. You can enter the decimal value between 0 t0 4. If the value entered is 0, the decimal values are not displayed. If the value entered in more than 0, the quantity is displayed in decimals upto number of decimals mentioned in this field.

Inventory Ageing Slots

  1. Age Slots – Enter the default Ageing Slots for the Aging Analysis Report in the text boxes provided in numeric format of three digits maximum length.

  2. Pre Packing – Enter the value 'Standard Formulary - Gale. Internal' in the text area. You can enter up to maximum of 40 alphanumeric characters.

  3. Extemporaneous – Enter  the value ‘Extemporaneous – Ad Hoc' in the text area. You can enter up to maximum of 40 alphanumeric characters.

  4. Standard Formulary – Enter  the value 'Standard Formulary - Gale. Internal', 'Standard Formulary - Gale. External' in the text area. You can enter up to maximum of 40 alphanumeric characters.

  5. Others - EnterYou can record the value ‘Standard Formulary - Extemp' in the text area.  You can enter up to maximum of 40 alphanumeric characters.

  6. Age Slots – Enter the default Ageing Slots for the Aging Analysis Report in the text boxes provided in numeric format of three digits maximum length.

 

‘Extemporaneous’ indicates manufacturing with Ad-Hoc Ingredients. By default, the alphanumeric editable text areas next to each label will be defaulted with the same Preparation Type as the label. If you leave any specific Preparation Type field as blank, it will not appear in the drop-down list.
