Enter the following report criteria to generate the required report:

 

  1. Condemn ID – Enter the required Condemn ID based on which the report has to be generated in numeric format of 10 digits maximum length.

  2. Report Order – Select the required report type from the list. The listed values are ‘By Item Class’ and ‘By Store’ and ‘By Condemned ID’. The value ‘By Item Class’ is displayed, by default.

  3. Item Code [From-To] - Enter the Item Codes range for which the batches that are condemned to be searched in alphanumeric format or select the Item Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  4. Item Class [From-To] - Enter the Item Class range for which the condemned batches are to be searched in alphanumeric format or select the Item Class using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  5. Condemn Date - Enter the Expiry Date range in dd/mm/yyyy format based on which the batches that are condemned are to be searched or select the Expiry Date using the Calendar function. The values are optional.

  6. Store Code [From-To] – Enter the Store codes range for which the condemned batches are to be searched or select the Store Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  7. Batch ID – Enter the Batch ID for the condemned batch in alphanumeric format of 10 characters maximum length.

  8. Remarks – Enter remarks of the condemned batches.
