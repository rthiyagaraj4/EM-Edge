Enter the following details to define attributes for an Item:

 

  1. Code – Enter the Item Code in alphanumeric format of 20 characters maximum length or click [?] function and select the Item. This is a mandatory field.

  2. Medical Item – Select the check box to indicate the Item as a Medical Item or clear to indicate the Item as non-medical Item. Apart from Drugs, Instruments and Medical Supplies are also classified as Medical Items. For example, Catheter, Syringe and Forceps. The check box is selected and disabled for Inventory Management Items.

  3. Drug – The check box is enabled on checking the ‘Medical Item’ check box. Select the check box to indicate the Item is a Drug or clear to indicate that the Item is non-drug Item. This drug code field is enabled, if the Pharmacy Management module is operational and allows entering a valid drug code on selecting this check box.

  4. Expiry – Select the check box to indicate that the defined Item is an expiry based Item. This data element is selected and disabled, if the ‘Drug’ check box is selected.

  5. Use After Expiry – The check box is enabled on selecting the ‘Expiry’ check box. You can select the check box to indicate whether the Item can be used even after its expiry.

  6. Batch ID – Select the Item’s Batch ID applicability from the list. The list of values for Expiry related Items are ‘User Enterable’ and ‘Same as Expiry Date’. The value ‘User Enterable’ is displayed, by default. The list of values for non-expiry related Items are ‘Not Applicable’ and ‘User Enterable’. The value ‘Not Applicable’ is displayed, by default.

  7. Allow Decimal Qty - Select the check box to indicate if the User can enter the quantity in decimals. If the check box is cleared, you cannot enter the quantity in decimals for the Item. Only whole numbers can be entered.

  8. Issue Integer Quantity – Select the check box, if Integer Quantity of the defined Item can be issued or clear the check box to Issue Quantity in fractions.

  9. Label Required – Select the check box to allow generation of bar coded labels for the Item.

  10. Special Req Item – Select the check box, if special request has to be raised while requesting for the Item. Items that require special request will not be available in the transaction ‘Generate Request’ and are available in the transaction 'Request' function request type 'Special Request'.

  11. Fractional UOM – Select the fractional UOM for the Item from the list. The fractional UOM indicates the UOM for slack quantities.

  12. Manufacturing Item – Select the check box to indicate if the Item is a manufacturing Item, that is, Items which can be manufactured internally. This check box is enabled for sole-source Items.

  13. Drug Code – The Drug Code (if any) for the selected Item is displayed based on the ‘Drug and Item Code Link’ as set in PH Parameter as shown in the table.

  2. Phy Count Per Year – Select the required value for Physical Inventory Count Per Year. This selected value displays the number of times physical counting of the Item can be performed in a year. The list of valid values are 1, 2, 3, 4, 6 and 12.

  3. Repack Group – Select the required Repack Group from the list indicating the repacking group for the repack Item. The records created in the function ‘Repacking Group’ is displayed in the list. This value is selected for Items that are repacked internally.

 

Sales Attributes

  1. Sale UOM – Select the required Unit of Measure for the selected item from the list. On selection of the sale UOM, the applicable conversion factor (if any) is displayed. The stock unit of measure is made default.

  2. Min Sales Quantity – Enter the minimum sales quantity in numeric format of six digits maximum length.

  3. Refund Allowed – Select the check box to indicate whether refund for the selected Item is allowed.

  4. Return Allowed - Select the check box to indicate whether returning of the selected Item is allowed.  

 

Other Attributes

  1. Sterilizable – Select the check box, if the selected Item should be Sterilized.

  2. Sterile Type – This field is enabled, if Sterile Stock module of Enterprise Management is installed. On selecting the ‘Sterilizable’ check box, enter the required Sterilization Type in alphanumeric format or select the Sterile Type using [?] function.

  3. ABC Analysis – Select the user defined ABC Classification for the Item. The list of valid values is ‘A’, ‘B’ and ‘C’.

  4. Machine Code – Select the required Machine Code value from the list. This Machine Code field is enabled only for non-medical Items. The non-medical Items are the Items for which Inventory Management check box is cleared in Item function in Materials Base module. That is, if ‘Medical Item’ check box is cleared for the defined Item. The Machine values created in the ‘Machine’ function are displayed.

  5. Part No – Enter the part number for a non-drug Item in alphanumeric format of 30 characters maximum length. This field is enabled for ‘Non-drug’ Item, that is, Items with ‘Drug’ check box cleared.

  6. Consignment – Select the check box to indicate the Item is a Consignment Item.

  7. Insurance - Select the check box to indicate whether the Item is related to Insurance items.

  8. Current Pack Size – Enter the Current Pack Size of the Item in numeric format of six digits maximum length.

  9. Item Image File Name - Enter the image file path.

 

Reorder Attributes

  1. Reorderable – Select the check box if the Item is re-orderable.

  2. Min Stock Level – Enter the minimum stock level that needs to be maintained for the defined Item in numeric format of nine digits maximum length.

  3. Max Stock Level – Enter the maximum stock level that needs to be maintained for the defined Item in numeric format of nine digits maximum length.

  4. RO Level - Enter the Reorder Level for the defined Item in numeric format of nine digits maximum length.

  5. Reorder Quantity - Enter the Reorder Quantity for the defined Item in numeric format of nine digits maximum length.

  6. Lead Days – Enter the Lead Days to be considered for computation of Reorder Level in numeric format of four digits maximum length. The Lead days indicate the number of days for receipt of goods from the supplier after ordering.

  7. Min Stock Days - Enter the Minimum Stock Days for calculating minimum stock level in numeric format of four digits maximum length.

  8. Buffer Percentage – Enter the percentage for buffer (on minimum stock level) to be considered for arriving at reorder quantity in numeric format.
