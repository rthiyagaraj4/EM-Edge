Enter the following report criteria:

  1. Report Type – Select the required report type by selecting the required option. The options are ‘By GRN’ or ‘By Item’.

  2. Item Code [From-To] - Enter the range of Item Codes for which the report is to be generated in alphanumeric format of 20 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Code and select required values.

  3. GRN Doc Type – Select the required GRN Document Type from the list. All the documents associated to the GRN Transaction in ‘ST Facility Parameter’ of ST Admin setup will be displayed in the list. This is a mandatory field.

  4. No. of Copies – Enter the number of copies of the Item Label to be generated in numeric format of 10 digits maximum length. This is a mandatory field.
