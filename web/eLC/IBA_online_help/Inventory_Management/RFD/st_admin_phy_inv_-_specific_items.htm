Enter the following to generate Physical Inventory for specific Items:

  1. Temporary Physical Inv ID – This field is disabled in the Create mode.

  2. Store – Select the required Store from the list. This is a mandatory field.

  3. Item  Class – Enter the search text or select the item class using the [?] function. The code of the selected item class gets displayed. You can enter alphanumeric and special characters up to maximum of four characters. It is not a mandatory field.

  4. Item Code [From–To] – Enter the Item Code range for which the Physical Inventory ID has to be generated or select the Item Code using [?] function. This is a mandatory field. If the Item Class is selected, only the items belonging to the selected item class will be listed in the lookup.  Once the From and To Item fields are selected, the Item Class field gets disabled.

  5. Remarks – Enter the common remarks for the generation of Physical Inventory for specific Items in alphanumeric format of 50 characters maximum length.
