Enter the following patient details in the Patient Details Entry Area:

 

  1. Sale Type – The Sale Type displays the defaulted Sale Type value in the User Access for Store screen. If required, select the Sale Type from the drop-down list. The listed values are Inpatient, Daycare, Emergency, Outpatient, Referral and Direct Sales. The Sale Type  values are displayed based on the ST Store setup.

  2. Discharge – The discharge check box is enabled, if the Sale Type is ‘Inpatient’.

  3. Chargeable – If billing is applicable, Chargeable check box is selected and disabled.

  4. Patient ID – Enter the Patient ID as was generated while registration in length as specified in MP Parameter or select the desired Patient ID using [?] function. This is a mandatory field.

  5. Patient Name – The Patient Name of the selected Patient ID is displayed.

  6. Encounter ID – Select the required Encounter Number from the list. The active Encounters are listed. If the selected Sale Type is ‘Inpatient’, the Inpatient Encounters are displayed in the list or if the selected Sale Type is ‘Outpatient’, the Outpatient Encounters are displayed in the list. If the Sale Type selected is ‘Referrals’ or ‘Direct Sales’, this field is disabled. It is mandatory to enter the Encounter ID for Sale Type 'Inpatient' or 'Outpatient'.

  7. $ - This icon will be visible only for the users who have the rights to view the patient’s current encounter financial details. If the Current Encounter Financial Details check box is selected in the Dispensing Rights screen, this icon with the tool tip  ‘Current Encounter Financial Details’ will be visible. Upon clicking this icon, the patient’s financial details is displayed in a separate Patient Encounter Financial Details dialog.

  8. Patient Class - The Patient Class of the selected patient is displayed on selecting the Encounter ID.

  9. Nursing Unit/Clinic - The Nursing Unit or Clinic of the selected patient is displayed based on the selected Sale Type.

  10. Room No - The Room Number assigned to the selected patient is displayed.

  11. Bed No - The Bed Number assigned to the selected patient is displayed.

  12. Remarks – Enter the required Remarks in alphanumeric format of 200 characters maximum length.

  13. Priority - Select the required value for Priority from the list. The system-defined values are 'Routine' and 'Urgent'. You can filter records based on the priority while finalizing the documents. The value is selected as 'Routine', by default.

  14. Practitioner Name– Enter the short name of the Practitioner in alphanumeric format of 15 characters maximum length or select the Practitioner using [?] function. Practitioners associated to the logged-in facility are displayed. On selecting the Sales Type as ‘Direct Sales’ the Practitioner field changes to a text box. Enter the Practitioner name in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  15. Item – Enter the item or click [?] to select the Item. Displays the Item Search dialog. This field is mandatory.

  16. Quantity – Enter the quantity of the item. This field is mandatory.

  17. Available Qty – Displays the available quantity.

  18. Batch Quantity – Enter the quantity of the batch of items.

  19. Remarks – Enter remarks of the item or click [?] to select the pre-defined remarks from the Remarks dialog.

  20. Grid

    • Delete – Select this check box to delete the item details from the grid.

    • Item Details – Displays the details of the item selected.

  21. Click [Edit/View] to edit or view the item details.

  22. Click [Batch Search] to search batch of items.

  23. Click [Add] to add the item.

  24. Click [Cancel] to cancel the item.
