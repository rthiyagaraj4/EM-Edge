Enter the following details to generate the Item Summary:

  1. Store Code [From-To] - Enter the Store Codes range for which the Item Summary is to be generated in alphanumeric format of four characters maximum length or select the Item Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected store code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ store code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  2. Item Code [From-To] - Enter the Item Codes range for which the Item Summary is to be generated in alphanumeric format of 20 characters maximum length or select the Item Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  3. Item Class [From-To] - Enter the Item Class range for which the Item Summary is to be generated in alphanumeric format of four characters maximum length or select the Item Class using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To Item Class code. When both the values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  4. Stock Item Value - Enter the Stock Item Value range for which the Item Summary is to be generated in numeric format of 15 digits maximum length. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Stock Item Value and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Stock Item Value. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  5. Unit Cost - Enter the Unit Cost range for which the Item Summary is to be generated in numeric format of 15 digits maximum length. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Unit Cost and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Unit Cost. When both the values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  6. Present Count Status – Enter the Present Count Status in numeric format of 15 digits maximum length. The Present Count Status indicates the number of counts done during the current year.

  7. Display Item Quantity - Select the Display Item Quantity from the list. The listed values are 'All', 'Zero Quantity' and 'Quantity > 0'. The value 'All' is displayed, by default.

  8. Expiry Items – Select the required expiry value from the list. The lists of valid values are ‘All Items’, ‘Expiry Items’ and ‘Non-Expiry Items’. The value ‘All items’ for both expiry and non-expiry is displayed in the Item Summary, by default.

  9. Order By - Select the required order by option from the list. The listed values are ‘Store Code’, ‘Item Analysis’ and ‘Item Description’.

 

Analysis

  1. Item Analysis Code 1 – Select the required value for Analysis Code 1 from the list.

  2. Item Analysis Code 2 – Select the required value for Analysis Code 2 from the list.

  3. Item Analysis Code 3 – Select the required value for Analysis Code 3 from the list.

  4. ABC Analysis – Select the required value for ABC Analysis from the list. The lists of valid values are ‘A’, ‘B’ and ‘C’.
