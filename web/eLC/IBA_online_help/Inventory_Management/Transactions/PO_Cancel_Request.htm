PO Cancel Request

From this function you can Cancel a Purchase Order Request.

The Purchase Order request raised to the Oracle Inventory can be cancelled using the Cancel PO Request function. The requests, which are not finalized, can only be cancelled.

Cancelling Purchase Order Request

From the ST Transactions > PO Request, select the PO Cancel Request function.

The PO Cancel Request screen gets displayed.

  1. Click [Query] to invoke the Query screen.

  2. Click [Search] to display the requested Purchase Orders or enter the search criteria and click [Search] to display the required Purchase Order request.

  3. On searching, the Purchase Order Request details are displayed.

  4. Click ‘Doc Type’ hyperlink to view the Item details.

  5. Select the ‘Cancel’ check box beside the Purchase Order Request record to select the record for cancellation. Click ‘Cancel’ check box in the header to select all the requests for cancellation.

  6. Click ‘Remarks’ hyperlink to invoke the Remarks screen. Enter the Cancellation Remarks. This is a mandatory field.

  7. Click [Apply] to cancel the Purchase Order Request or [Reset] to clear the entered data.

  8. On clicking [Apply], the ‘Confirm Cancel’ dialog is displayed. Click [Yes] to cancel the Purchase Order Request or [No] to cancel the operation.
