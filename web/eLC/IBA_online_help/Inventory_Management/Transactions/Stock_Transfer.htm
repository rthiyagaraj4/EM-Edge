Stock Transfer

From this function you can Record the details of stock transferred from one Store to another.

The Items to be transferred should exist in both transferring Store and the transferred Store. As a result of Stock Transfer, the stock moves from one stocking point to another. In the system as a whole there is no net effect on the stock levels. In the transferring Store, the stock levels are reduced. In the receiving Store, the stock levels are increased.

Recording Stock Transfer Details

From the ST Transactions > Transfer, select the Stock Transfer function.

The Stock Transfer screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Modify Document Details screen is displayed, by default or click ‘Document’ hyperlink to modify the document details.

  3. On entering the Document Details, click [OK] to record the Stock Transfer details or [Reset] to clear the entered data.

  4. Enter the Item to transfer to the selected Store in alphanumeric format of 60 characters maximum length or select the Item using [?] function. Click the Information icon near the item name to view either 'Remarks' or 'Specification' or both for the selected item in a popup or as a tool tip. The Information icon will be available only if either 'Remarks' or 'Specification' or both exists for that item. This is a mandatory field.

You can click the information icon next to the Item for viewing the ‘Item Remarks’.

  1. On selecting the Item, the Unit of Measure and the available stock quantity, Cost of the Item is displayed. The Cost of the Item is calculated as rechargeable fractional units for multiple patients.

  2. Enter the Quantity of the Item, which needs to be transferred to the selected Store in numeric format of six digits maximum length. The entered quantity must not be greater than the available quantity of the Item or an error message will be displayed. This is a mandatory field.

  3. On entering the Quantity, the Transfer Quantity is displayed by default. You cannot change the Transfer Quantity.

  4. Click [Edit/View] to transfer the first expiring batch in the slot. The Item Batch details are displayed. The Quantity entered is displayed as the Transfer Quantity. Select the ‘To Store Master Bin Location’ from the list. The Bin Locations associated to the selected ‘To Store’ are displayed in the list.

  5. Click [Batch Search] to view the Batch details. The selected Item Batch details are displayed. Select the check box to select the Batch and click [Close] to exit from the batch search screen. If batch details are selected using [Edit/View], the selected Batch details are refreshed on selecting the new batch.

  6. On selecting the batch, the batch details are displayed in the Batch Entry area.

  7. Select ‘To Store Master Bin Location’ from the list and enter the transfer quantity.

  8. Click [Add] to add the Item for transfer or [Cancel] to cancel the selected Item details.

  9. Click [Stock Level] to view the ‘From’ and ‘To’ stock details.

  10. The selected Item details are displayed in the Item Details frame on the left hand side. Click ‘Item Details’ hyperlink to modify the Item transfer details. Click the Information icon near the item name to view either 'Remarks' or 'Specification' or both for the selected item in a popup or as a tool tip. The Information icon will be available only if either 'Remarks' or 'Specification' or both exists for that item. Click [Modify] to modify the entered details or [Cancel] to cancel the modified changes.

  11. You can select the ‘Delete’ check box to delete the selected record. The Delete check box is selected or cleared by default based on the setup done in Kit Template screen.

  12. Click [Apply] to finalize the Item transfer details or [Reset] to clear the entered data.

  13. The ‘Confirm Finalize’ dialog is displayed. Click [Yes] to finalize the Item transfer details or [No] to finalize the Item transfer details in the query mode.

  14. The ‘Confirm Printing’ dialog is displayed based on the attribute set for the Document Type. Click [Yes] to print the Item transfer details or [No] to cancel the action.

  15. Click [Query] to invoke the Query screen.

  16. The unauthorized Item transfer details are displayed based on the default criteria or enter the search criteria and click [Search] to select a specific record.

    Enter the following criteria to query unauthorized Stock Transfer details:

     

    1. Doc Type – Select the Document Type from the list. The values listed are defined in the function ‘ST Document Type’ and the default value displayed is specified in the function ‘ST Facility Parameter’.

    2. Doc No. – Enter the Document Number in numeric format of eight digits maximum length.

    3. Doc Date - Enter the date in date format of dd/mm/yyyy or select the required date using the Calendar function.

    4. From Store – Select the ‘From Store’ from the list. Only those stores are displayed to which you have access and ‘Stock Transfer’ transaction function is applicable.

    5. To Store – Select the ‘To Store’ from the list. Only those stores are displayed to which you have access and ‘Stock Transfer’ transaction function is applicable.

    6. Item Class – Select the required Item Class value from the list.

  17. On searching, the unauthorized Item transfer details are displayed.

  18. Click ‘Doc Type’ hyperlink to authorize the Item transfer details.

  19. Click [Apply] to finalize the Item transfer details or [Delete] to delete the selected record.

  20. On clicking [Apply], the ‘Confirm Finalize’ dialog is displayed. Click [Yes] to finalize the Item transfer details or [No] to finalize the Item transfer details in the query mode.

  21. On selecting the ‘Delete’ check box, ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.

Sale transaction is processed if the remaining quantity is changed to the patient.

If the patient is not charged for the remaining quantity, then the remaining quantity should be written off through the stock adjustment transaction. The Minimum Stock Level or Reorder Level of the selected item is displayed next to the Item field while performing the transaction. Based on selection of Display Min Level & ROL check box in User Access for Store, this functionality will be enabled.
