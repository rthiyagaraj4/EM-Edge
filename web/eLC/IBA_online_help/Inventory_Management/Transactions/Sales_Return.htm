Sales Return

From this function you can Record the details of Items returned to the Store.

Sales return transactions are performed with respect to Items to be returned to the Store. This function facilitates in capturing details of the Items returned to the Store, updating the patient billing records and stock records.

Recording the details of Items returned to the Store

From the ST Transactions > Sales, select the Sales Return function.

The Sales Return screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Modify Document Details screen is displayed by default or click ‘Document’ hyperlink to modify the document details.

  3. On entering the Document Details, click [OK] to record the Item return details or [Reset] to clear the entered data.

  4. Enter the details in the mandatory (*) fields. <Related Field Description>

  5. Click on Sales Doc No. [?] to open the Sales Return Details dialog. Enter the Item details or Document No. and select the item to be returned and click [Add]. The item to be returned is displayed in the item details frame on the left hand side. The checkbox ‘Select All’ is enabled and selected by default when the  ‘Sales Return: Select all items by default when Sale Document is picked’ check box is selected in ST Facility Parameter screen.

  6. Click 'Item Details' hyperlink to modify the Item details to return.

  7. The [Edit/View] option is disabled.

  8. Click [Batch Search] to select the issued batch. The selected Item Batch details are displayed. Select the check box to select the Batch and click [Close] to exit from the batch search screen. If batch details are selected using [Edit/View], a message ‘Batch Data will be refreshed’ is displayed. Click [OK] to overwrite the existing batch details or [Cancel] to retain the selected batch.

  9. On selecting the batch, the batch details are displayed in the Batch Entry area.

    Enter the following details of the batch:

     

    1. Expiry Date – The Expiry Date of the batch is displayed. You can modify the date by entering the date in dd/mm/yyyy format or select the date using the Calendar function.

    2. Batch ID – The selected Batch ID is displayed. You can enter new Batch ID in numeric format as set in the ‘Batch ID’ label in Item function for expiry and non-expiry based Items.

    3. Trade Name – The selected Trade Name is displayed. You can select the required Trade Name from the list. The Trade Names associated to the selected Item in Materials Base module is displayed in the list.

    4. Available Qty – The Available Quantity of the Item is displayed.

    5. Bin Location – Select the required Bin Location from the list. The Bin Locations associated to the selected Item in ‘Item for Store’ function is displayed in the list.

    6. Ret Quantity - Enter the Return Quantity in numeric format of five digits maximum length. The entered Return quantity must be equal to Item quantity or an error message ‘Quantity has to be less or equal to available quantity’ is displayed.

  10. Click [Modify] to modify the Item return details or [Cancel] to cancel the modified changes.

  11. Click [Preview] to view the Item return details.

  12. Click [Apply] to finalize the Item return details or [Reset] to clear the entered data.

  13. The ‘Confirm Finalize’ dialog is displayed. Click [Yes] to finalize the Item return details or [No] to finalize the Item return details in the query mode.

  14. The ‘Confirm Printing’ dialog is displayed based on the attribute set for the Document Type. Click [Yes] to print the Item return details or [No] to cancel the action.

  15. Click [Query] to invoke the Query screen.

  16. The unauthorized Item return details are displayed based on the default criteria or enter the search criteria and click [Search] to select a specific record.

    Enter the following criteria to query unauthorized Stock Adjustment details:

     

    1. Doc Type – Select the document type from the list provided. The values listed are defined in the function ‘ST Document Type’ and the default value displayed is specified in the function ‘ST Facility Parameter’.

    2. Doc No. – Enter the document number in numeric format of eight digits maximum length.

    3. Doc Date [From-To] - Enter the date in date format of dd/mm/yyyy or select the required date using the Calendar function.

    4. Patient ID – Enter the Patient ID in alphanumeric format as set in Master Patient Index module or select the Patient ID using [?] function.

    5. Item Class – Select the required Item Class value from the list.

    6. To Store – Select the ‘To Store’ value from the list. Only those stores are displayed for which you have access and ‘Sale’ transaction function applicable.

    7. Nursing Unit/Clinic - Select a Nursing Unit/Clinic from the list of values. The Nursing Units/Clinics with unfinalized sales return documents are listed.

  17. On searching, the unauthorized Item return details are displayed.

  18. Click ‘Doc Type’ hyperlink to authorize the Item return details.

  19. Click [Apply] to finalize the Item return details or [Delete] to delete the selected record.

  20. On clicking [Apply], the ‘Confirm Finalize’ dialog is displayed. Click [Yes] to finalize the Item return details or [No] to finalize the Item return details in the query mode.

  21. On selecting the ‘Delete’ check box, ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.
