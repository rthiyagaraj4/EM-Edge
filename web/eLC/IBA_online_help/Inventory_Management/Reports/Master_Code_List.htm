Master Code List

From this function you can Generate reports of the Code Setup functions.

Select the required code and specify the relevant report criteria to generate the report.

You can generate the following Master Code Report:

  • Machine

  • Item

  • Store

  • Item Class for Store

  • Item for Store

  • Assembly Details/MFG Formulary

  • Prepare Location

  • Document Type

  • Item Store Bin Location

  • Repacking Group

Generating the Master Codes Reports

From the ST Reports, select the Master Code List function.

The Master Code List screen gets displayed.

  1. Select the Master Code from the list.

  2. Click [Run] to generate the report or enter the search criteria and [Run] to generate the report based on the entered criteria. Click [Reset] to clear the entered data.

  3. Select ‘Report Option’ to generate the report.

  4. Click [OK] to generate the report and spool it in the file on the print server or [Cancel] to cancel the action.

The Report Criteria fields are displayed based on the selected Code.
