Acknowledge Transactions

From this function you can Query and view the acknowledgement related transactions.  

Acknowledgement Transactions are filtered based on Acknowledgement Date, Drug/Non Drug, Item Code Range, Rejected Quantity, Issuing Store and Receiving Store. This function is used to monitor acknowledged transactions based on a time frame.

Querying Acknowledge Transactions

From the ST Queries, select the Acknowledge Transactions function.

The Acknowledge Transactions screen gets displayed.

  1. Click [Search] to display all the acknowledged transaction details or enter the search criteria and click [Search] to view the required acknowledged transaction details.

    Enter the following search criteria to view acknowledge transactions:

     

    1. Acknowledge Date [From - To] - Enter the Acknowledge Date range in the format of dd/mm/yyyy or select the date using the Calendar function. The current system date is displayed, by default. The 'To' date should not be greater than the 'From' date entered.

    2. Drug/Non Drug - Select the Drug/Non Drug values from the list. Select the value from the list. The listed values are 'Drug', 'Non Drug' and 'Both'. The value 'Both' is displayed, by default.

    3. Item Code [From - To] - Enter the range of Item code for which the acknowledgement details are to be searched in alphanumeric format or select the required Item code using [?] function. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected item code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

    4. Reject Qty - Select the Rejected Quantity from the list. The listed values are 'Zero', 'Non Zero' and 'Both'. The value 'Both' is displayed, by default.

    5. Issuing Store - Enter the Issuing Store name or select the Issuing Store name using [?] function.

    6. Receiving Store - Enter the Receiving Store name or select the Receiving Store name using [?] function.

  2. Click [Reset] to clear the entered data.

  3. On searching, the acknowledged transaction details are displayed based on the entered criteria.
