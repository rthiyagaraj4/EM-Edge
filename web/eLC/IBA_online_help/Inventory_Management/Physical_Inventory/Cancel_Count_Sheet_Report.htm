Cancel Count Sheet Report

From this function you can Generate the canceled count sheet reports.

Generating Canceled Count Sheet Reports

From the ST Physical Inventory, select the Cancel Count Sheet Report function.

The Cancel Count Sheet Report screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [Run] to generate the cancelled count sheet report or [Reset] to clear the entered data.

  3. Select ‘Report Option’ to generate the report.

  4. Click [OK] to generate the cancelled count sheet report and spool it in the file on the print server or [Cancel] to cancel the action.
