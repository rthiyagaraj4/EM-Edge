Re-book Cancelled Appointment

From this function you can Re-book the cancelled appointments.

This function enables you to re-book the cancelled appointment. You can book appointment or Create Waitlist for a patient. Bulk Cancelled appointments cannot be re-booked using this function.

 

Process Flow Diagram

Creating Appointments for cancelled appointments

From the OA Transactions, select the Re-book Cancelled Appointment function.

The Re-book Cancelled Appointment screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Select] to display patients with cancelled appointments. Click [Clear] to clear the displayed data if needed.

  3. Click ‘Appt No’ hyperlink to Book Appointment or Create Wait list.
