Book Appointment

Appointments are booked only for a particular Location/Primary Resource. The appointment details for a patient are captured under three tabs, Patient Details Tab, Secondary Resources Tab and Financial Details Tab.

  1. Select the Visit Type and enter other fields. <Related Field Description>

    In case of schedule generated for a timetable type ‘Slot’, on selection of a visit type from the list, you can select the number of slots from the adjacent list next to Visit Type. You can select the values in the list based on the value set for ‘Maximum Appointment Slots’ in Visit Type for Location of Outpatient Management module. If ‘Overriding Number of slots are permitted while Booking Appointment’ the check box is selected in OA Parameter. If the Overriding Number of slots are permitted while booking Appointment’, the check box is cleared and the values set in ‘Maximum Appointment Slots’ in Visit Type for Clinic of Outpatient Management module is defaulted.

    After you select the slots in the Book Appointment screen for Radiology appointments the Visit Type which is mapped to the Procedure Unit is defaulted. This is possible only if the selected Procedure Unit has a single Visit Type linked. If there is multiple visit types linked to single Procedure Unit and available for booking, the Procedure Unit does not gets defaulted.

    Slot Period:

    If Schedule Time is selected in Order Catalog, then the slot timing is blocked based on the Schedule Time. If Schedule Time is not checked then it considers the number of slots entered against the Visit Type.

  2. Enter the details in the mandatory (*) fields of the Tab pages.

  1. Click [Previous Encounters] to view the previous encounters of the patient if the selected patient had previously received care from the facility.

  2. Click [Other Appts] to view all other appointments of the patient.

    Other Appointment button appears only for patients with current/future day appointments or bookings and having a valid patient ID. When [Other Appts] is clicked, all other appointment of the Patient is displayed. The appointment details are only displayed and cannot be changed.

  3. Click [Recurring Days] to create Recurring Appointments for the patient.

  4. Click [Confirm Booking] to book an appointment. The system allows Overbooking and Forced Booking of appointments.

    The system allows forced in bookings against a particular Location/Resource thus accommodating special cases within a defined appointment schedule for a day. Forced Appointments for a particular Location/Resource can be performed, only if the defined time table is of type ‘Slot’ or ‘Slab’ and value for  ‘No of Forced in Booking’ is set while defining the timetable. Forced Bookings cannot be performed for a future date else the system would prompt the user with a message – Forced Booking possible for future date. User can perform ‘Forced Appointment’ by clicking on any of the already booked time.

    The system allows overbooking against a particular Location/Resource thus allowing certain amount of flexibility in accommodating additional flow of patients within a defined appointment schedule for a day. Overbooking for a particular Location/Resource can be performed, only if value for ‘Maximum Over Bookings’ is set while defining the timetable.

Note: If no translators are available for the patient language, when you click [Confirm Booking], the following message will be displayed. “”Number of Appointments Exceeds Available Translators. Do you want to Continue?”

If you click [OK], the Translator Override Reason dialog is displayed from which you can select one of the reasons for booking the appointment without a translator. You can click Cancel, if you do not want to proceed with the appointment.

  1. If internal reports are defined, Internal Reports screen gets displayed, click [OK] to generate Internal Reports or [Cancel] to exit from the screen.

While rescheduling appointments, the following logic will be applied:

Reschedule within same clinic with Patient Instructions

  • For Patient A, book an Appointment in Clinic A with Patient instructions 1, 2, 3 and instruction 10 added from 8 to 8.30.

  • Reschedule within Clinic A from 10 to 10.30

When you click Edit instructions hyperlink, you will be able to view 1, 2, 3, 10 checked and 4, 5 un-checked.

 

Reschedule within same clinic without Patient Instructions

  • For Patient A, book an appointment in Clinic A without Patient instructions by un-checking 1, 2, and  3 from 8 to 8.30

  • Reschedule within Clinic A from 10 to 10 .30

When you click Edit Instructions hyperlink, you will be able to view 1, 2, 3, 4, 5 un-checked.

 

Reschedule to another Clinic with Patient Instructions

• For Patient A, book an Appointment in Clinic A with Patient instructions 1, 2, 3 from 8 to 8.30

• Reschedule to Clinic B

When you click Edit instructions hyperlink, you will be able to view 7 and 8 checked and 9 and 10 un-checked.

Reschedule to another Clinic without Patient Instructions

• For Patient A, book an Appointment in Clinic A without Patient instructions by un-checking 1, 2, 3 from 8 to 8.30

• Reschedule to Clinic B from 10 to 10.30

When you click Edit Instructions hyperlink, you will be able to view 7 and 8 checked and 9 and 10 un-checked.

Note

 If you do not click Edit patient Instructions and transfer appointment within the Clinic, the instructions which are captured during book appointment will retained.

 If you do not click Edit patient Instructions and transfer appointment to other Clinic, then the Default Patient instructions of the selected Clinic will be considered.

 In case of Bulk transfer within clinic or to other clinic, there will not be any change in Patient Instructions, Irrespective of whether the new clinic supports patient instructions or not.
