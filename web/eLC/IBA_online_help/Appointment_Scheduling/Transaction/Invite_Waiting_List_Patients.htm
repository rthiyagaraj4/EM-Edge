Invite Waiting List Patients

From this function you can Invite waiting list patients or Select Waiting List Patient for inviting and view the status of invited patients from the Waiting List.

Invite Waiting List patients function is used to invite patients on wait lists for Clinics, Nursing Units, Procedure Units or Day Care Units. Using this function, a particular patient is invited for an appointment on a preferred date.

 

Process Flow Diagram

Inviting/Selecting Waiting List Patients

From the OA Transactions, Invite Waiting List Patients function.

Invite Waiting List Patient screen gets displayed.

  1. Select the Specialty to view the patients in Wait list for that particular Specialty or enter the query criteria to search for a particular patient in Wait List. <Related Field Description>

  2. Click [Search] to view patients in waiting list. [Collapse] gets displayed in place of [Search]. Click [Collapse] to view all the query fields.

    1. Select – You can select patient by selecting the ‘Select’ check box provided.

    2. Invitation Date - Select the Invitation Date using the Calendar function. This field becomes mandatory, if the status is selected as ‘Invited’.

    3. Status – Select the status of the Wait List from the list provided. The available values are ‘Selected’ and ‘Invited’. This field is mandatory.

    4. Patient ID – This field displays the Patient ID of the patient if it has been recorded at the time of scheduling the appointment. Click the Patient ID hyperlink to view Discharge Summary recorded for inpatients. If multiple Discharge Summary exists for an inpatient, discharge summary list is displayed hierarchically based on the latest date and time of discharge. The date and time is displayed as a hyperlink. Click on the   hyperlink to view respective Discharge Summary details. The details of the latest discharge summary is displayed.  If the patient has a single Discharge summary, the Discharge Summary Date and Time along with the Discharge Summary Details are displayed in the lower section of Invite Waiting List patients.

    5. Patient Name – This field displays the Patient Name.

    6. Gender – This field displays the Gender of the patient.

    7. Age – This field displays the Age of the patient recorded at the time of scheduling the appointment.

    8. Region – This field displays the Region of the patient

    9. Arrival Date – This field displays the Arrival Date of the patient.

    10. Previous Encounters – On clicking the hyperlink, the details of the previous encounters gets displayed.

    11. Wait list No – This field displays the Wait List number of the patient.

    12. Priority – This field displays the priority of the patient as ‘Urgent’, ‘semi Urgent or ‘Emergency’.

  3. Select the required patient by selecting the check box provided.

  4. Select the Invitation Date to select the Status as ‘Invited’.

  5. Select the status of the wait list as Invited or Selected from the Status list. If the Invitation Date is entered, select ‘Invited’ to invite the patient for appointment or select ‘Selected’ to select the patient for inviting.

  6. Click Patient ID hyperlink to view the Notes recorded for the patient (if any). The color of the record is displayed with a color.

  7. Click Previous Encounter hyperlink to view the previous encounter details of the patient.

    1. Encounter Date/Time – This field displays the date and time when the encounter is registered.

    2. Encounter Type – This field displays the type of encounter which are ‘Outpatient’, ‘Inpatient’, ‘Emergency’ or ‘Daycare’

    3. Encounter ID – This field displays the Encounter Identification number generated during registration.

    4. Location Type – This field displays the location type for the encounter

    5. Location – This field displays the location that the patient visited during the encounter.

    6. Service – This field displays the service that has been rendered during the encounter.

    7. Sub-service – This field displays the sub-service that has been rendered during the encounter.

    8. Room no – This field displays the room that is assigned during the encounter.

    9. Bed No – This field displays the Bed that is assigned during the encounter. This field is applicable only when encounter is of type ‘Inpatient’, ‘Emergency’ or ‘Daycare’.

    10. Practitioner – This field displays the attending practitioner who treated the patient during encounter.

    11. Disposition Date/Time – This field displays the date and time of disposition that is recorded while closing the encounter.

    12. Disposition Type – This field displays the type of disposition that is recorded while closing the encounter.

    13. Financial Details - A hyperlink is displayed to view the financial details during the encounter. This opens only when Billing module is installed and operational in the facility.

    14. Billing Details - A hyperlink is displayed to view the financial details during the encounter. This opens only when Billing module is installed and operational in the facility.

  8. Click [Invitation List] to display the Invitation Status screen and view the status of the invitation list for the current month. Click [Close] to close the Invitation Status screen and the controls shifts back to the Create Wait list screen.

    The status of the invitation list for the next 30 days is displayed. By clicking on , you can check the schedule status for the next month. Similarly, by clicking on , the schedule status for the previous month can be viewed.  

    If Invitation exists for a date, it is displayed as a hyperlink. On clicking the hyperlink, the current day status is displayed. The status has distinct colors to indicate the Invitation status. The color represents:

    •  – This color indicates that the waitlist has been opened.

    •  – This color indicates that the waitlist patient has been selected.

    •  – This color indicates that the waitlist patient has been invited.

    Click [Close] to close the Invitation Status screen and the controls shifts back to the Invite Waiting list Patients screen.

  9. Click [Apply] to invite/select waiting list patients. Click [Clear] to clear the displayed data.
