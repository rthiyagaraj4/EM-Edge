Schedule Appointments

From this function you can View availability of schedules and Perform bookings for future dates. Schedule Appointments function allows querying the earliest available schedule and supports multiple scheduling against a single patient.

An appointment schedule is a planned visit for a future date against a Location or Location and resource combination. An appointment is scheduled against a location with or without assigning a resource based on the attribute of the location. Scheduling an appointment for patients without a Patient ID is allowed by capturing required demographics of the patient. The Schedule Appointment screen gets displayed with three tabs ‘Schedule Status’, ‘Day Schedule’ and ‘Pending Order Schedule’ with ‘Schedule Status’ enabled.

 

When a practitioner opens the Schedule Appointment screen from Clinician Access (CA) module, the schedule available for the current date/time is displayed on the right side of the screen. When the current date schedule is blocked in the Block Schedule function in Appointment Scheduling module, then a  message 'Schedule has been blocked for (date)’ is displayed.  If there is no schedule available for the practitioner, the Schedule Status page will be displayed.
