Enter the following details:

 

  1. Specialty – Select the required specialty against which the appointment waitlist has to be searched from the list. All the enabled specialty records created in ‘Application Masters’ module gets displayed. This field is mandatory.

  2. Waitlist Category - Select the waitlist category using [?] function.

  3. Order Catalog - Select the required Order catalog using [?] function. The value gets populated from the Order Entry & Result Reporting module.

  4. Priority - Select the required priority from the list of valid values. The listed values are Urgent, Semi Urgent and Normal.

  5. Location – You have to select two values for the location. The first list is the Location Type and the next one lists all the Locations of the selected type. The Location Type values listed are ‘Clinic’, ‘Day Care’, ‘Nursing Unit’ and ‘Procedure Unit’. On selecting the required Location Type, the adjacent Location list is enabled with the Location values. The location(s) are displayed based on the selected specialty.

  6. Resource - This field is enabled on selecting a location. On selecting the location the resource type is displayed against the resource field. Enter the required resource name in alphanumeric format of 30 characters maximum length or click [?] function to search for the required resource.

  7. Wait List No – Enter the required waitlist number in alphanumeric format of 12 characters maximum length.

  8. Period – Select the date range for which the waitlist for Patient are created using the Calendar function provided.

  9. Status – This field is displayed and disabled as open.

  10. Elapsed – Select the check box to view elapsed waitlist for the selected specialty, or clear the check box. On selecting the check box, all the waitlist records with preferred date of appointment for waitlist less than the current system date gets displayed.

  11. Patient ID – Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter, use [?] function by specifying the required query criteria and find the Patient ID.

  12. Patient Name – Enter a valid Patient Name in alphanumeric format of 40 characters maximum length. If valid Patient ID is entered or selected using [?] function, system displays the Patient Name corresponding to the entered or selected Patient ID.

  13. Gender – Select the Gender of the Patient whose appointment is to be queried from the list of valid values.

  14. Nationality – Select the required value using [?] function. The listed values are defined in ‘Master Patient Index’ module.

  15. Town – Select the Town using [?] function. The listed values are defined in ‘Master Patient Index’ module.

  16. Area – Enter the Area in alphanumeric format of 30 characters maximum length or select it using [?] function. The listed values are defined in ‘Master Patient Index’ module.

  17. Region – Select the Region using [?] function. The listed values are defined in ‘Master Patient Index’ module.
