Enter the following details to book an appointment:

 

  1. Location – This field displays the Location Type/Location for which the appointment is made.

  2. Resource – This field displays the Resource Class/Resource (primary resource) against which the appointment is made.

  3. Visit Type – Select the Visit Type and enter other fileds.

In case of schedule generated for a timetable type ‘Slot’, on selection of a visit type from the list, you can select the number of slots from the adjacent list next to Visit Type. By default, this drop-down list displays the value defined for ‘Maximum Appointment Slots’ in Visit Type for Location function in the Outpatient Management module will be displayed. You can modify the value, as required.

 

Note:

1. If there are no free slots available, an alert message ’Continuous Slot not Available’ with OK Button will be displayed. Once you click OK button, the drop-down list will display the minimum appointment slot value .

2. The order catalog slot control for radiology appointments will be based on order catalog duration, the default maximum value based on Visit type will not be applicable.

If ‘Overriding Number of slots are permitted while Booking Appointment’ the check box is selected in OA Parameter. If the Overriding Number of slots are permitted while booking Appointment’, the check box is cleared and the values set in ‘Maximum Appointment Slots’ in Visit Type for Clinic of Outpatient Management module is defaulted.

 

  1. Appt Date/Time - The date selected from the schedule is displayed as the Appointment date–and Appointment Time fields cannot be modified for ‘Slot’ and ‘Slab’ types. For ‘Free Format’ type the Appt Time fields can be modified. Enter the time in hh:mm format in both the fields. The first field corresponds to the starting time and the second field corresponds to the ending time. The End Time must not be less than From Time. If the selected schedule time belongs to free format, the end time is displayed based on the selected visit type. That is, for the selected visit type, if ‘Maximum Appointment Duration’ is set up in ‘Visit Type’ function of Outpatient Management module, end time gets displayed based on the setup or the location working time gets displayed. This can be modified.

  2. Preferred Date – Enter the preferred date of appointment in dd/mm/yyyy format or click on the Calendar function to select the required date. On entering the required date, a waitlist booking is created for the patient on the entered date.

  3. Priority – Select the required priority for the Wait List from the list of valid values. The listed values are Urgent, Semi Urgent and Normal.

  4. Reason for Override - This field is available only to capture the reason for override while selecting a practitioner other than the practitioner defined by the system. Enter the reason for override or select using the [?] function. The Override Reason lookup lists the reasons for which the ‘Override Appointment Rule’ check box is selected in the Reason master.
