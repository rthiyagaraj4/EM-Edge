Enter the following details to query the appointments for patients:

 

  1. Search With - Select a value from the list. The values are system defined. Select All to display the appointment details for all the patients. Select Patient ID to search for a relevant patient record for whom the appointment is booked.

  2. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter. In case the Patient ID is not known, the patient ID can be searched using a standard lookup screen by clicking [?] function and by specifying the required query criteria. This field becomes mandatory if ‘Patient ID’ is selected as the ‘Search With’ criteria. The future appointments of the selected Patient ID is displayed.

  3. Patient Name - Enter a valid Patient Name in alphanumeric format. If valid Patient ID is entered or selected, system displays the Patient Name corresponding to the entered or selected Patient ID.

  4. Appt No - Enter the Appointment Reference Number in alphanumeric format of eight characters maximum length, or else the appointment referral number gets  displayed on entering or selecting a Patient ID. This field is mandatory only if ‘Search With’ is selected as ‘All’. The current system date is displayed as the Appointment ‘From’ Date. The second consecutive date of the current system date is displayed as the Appointment ‘To’ Date.

  5. Appt Date [From-To] - Enter the date range for which the appointments for Patient are scheduled in dd/mm/yyyy format or click Calendar function to search for the required date. The date selected has to be equal to or greater than the current system date. This field becomes mandatory if ‘All’ is selected as the ‘Search With’ criteria.

  6. Specialty – Select the specialty from the list of valid values.

  7. Location Type – Select the location type from the list. The listed values are Clinic, Daycare Unit and Procedure Unit. This is a mandatory field.

  8. Resource Type - This field gets enabled only if ‘Patient ID’ is selected in ‘Search With’ field. Select the Resource Type from the list. The listed values are Practitioner, Equipment, Room and Other.

  9. Include Past Appointments - This field gets enabled only if ‘Patient ID’ is selected in ‘Search With’ field. Select the check box to display all the appointments including the past appointments. Clear the check box to display only the present appointments of the selected patient.

  10. Location – This field gets enabled only if ‘Patient ID’ is selected in ‘Search With’ field. Select the location, where the patient is scheduled for an appointment from the list. On selecting the required Location Type, the corresponding Locations are listed. This is a mandatory field.

  11. Resource - This field gets enabled only if ‘Patient ID’ is selected in ‘Search With’ field. Select the required resource against whom the Patient is scheduled for an appointment from the list using [?] function. On selecting the required Resource Type, the corresponding resources are listed. The lookup function gets enabled on selecting a resource type.

  12. Mobile No – Enter the mobile number of the patient for searching the patient.

  13. Facility – Select the facility name from this drop-down list. All the facilities are listed in the drop-down list. By default, The logged in facility is displayed..You can modify the facility name.If you select the value ’Select’ (blank value) in the drop-down list, on click of Search the Appointments pertaining to all the facilities are displayed.

Click  on the respective Column Heading Toolbar to sort the details under the clicked Heading in the ascending order. For example, click on the Appointment Number column heading. All the appointment numbers will be sorted out in the ascending order.

The results display the appointment details such as Specialty, Order Catalog, Referral ID, Patient Instructions, Translator Override Reason, Appt Given by, and status.
