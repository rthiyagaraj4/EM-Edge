Enter the following details:

 

  1. Specialty – Select the specialty from the list of valid values. The listed values are set in ‘Application Masters’ module.

  2. Waitlist Category – Select the category using [?] function. The categories created in the function ‘Wait list category’ in Appointment Scheduling module are displayed.

  3. Order Catalog – Select the required Catalog using [?] function. The values are getting populated from the Order Entry & Result Reporting module.

  4. Priority - Select the required Priority from the list of valid values. The listed values are Urgent, Semi Urgent and Normal.

  5. Location Type – Select the Location Type as Clinic, Daycare Unit, Nursing Unit or Procedure Unit.

  6. Location – Select the required location value from the list of valid values. The locations displayed are based on the Location Type selected.

  7. Resource – This field is enabled on selecting a location. On selecting the location the resource type is displayed against the resource field. Enter the required resource name in alphanumeric format of 30 characters maximum length or click [?] function to search for the required resource.

  8. Wait List No – Enter the Wait List Number in alphanumeric format of 12 characters maximum length.

  9. Period – Enter the preferred ‘From’ and ‘To’ date in dd/mm/yyyy format or select the date using the calendar function. ‘From’ date corresponds to starting date of the range of dates on which the Invitation list details are to be displayed.

  10. Status – Select the status of the Wait List from the list of values provided. The values available are Open, Selected and Invited. ‘Open’ is selected, by default.

  11. Elapsed - Select the check box, if the elapsed Wait List patients are to be displayed.

  12. Patient ID – Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or click [?] function to search for the required Patient ID.

  13. Patient Name – Enter a valid Patient Name in alphanumeric format of 40 characters maximum length. If valid Patient ID is entered or selected using [?] function, the system displays the Patient Name corresponding to the entered or selected Patient ID.

  14. Gender – Select the Gender of the unregistered Patient from the list of valid values. The valid values are Male, Female and Unknown.

  15. Nationality – The default Nationality set in MP Parameter is displayed and disabled if National ID is entered, or select the required value using [?] function.

  16. Town – Enter the Town in alphanumeric format of 30 characters maximum length. The entered Town must be listed under the selected area. If the selected area has more towns attached to it, select the required town using [?] function. The field legend is set in ‘Enterprise Configuration’ module.

  17. Area – Enter the Area in alphanumeric format of 30 characters maximum length or select it using [?] function. The field legend is set in ‘System Manager’ module.

  18. Region – Select the Region using [?] function. The listed values are defined in ‘Master Patient Index’ module.
