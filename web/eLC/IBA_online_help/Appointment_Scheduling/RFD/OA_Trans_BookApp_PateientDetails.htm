Patient Details Tab

This tab is displayed on selecting the time slot/slab from the Day Schedule Tab for booking an appointment.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Contact Details] to enter, view or modify the contact details of the patient.

• If the check box 'Capture Contact details during Appointment' is selected in the OA Parameter function, entering the Primary Residence Number field becomes mandatory.

• If the check box ‘ Email Applicable ’ is selected in OA Parameter function, email address of the patient/customer is mandatory.

• If the check box ‘Mobile Number required during Appointment’ is selected in OA Parameter function, the mobile number of the patient/customer is mandatory.

 

Patient Instructions hyperlink

  1. This hyperlink gets displayed only if the Patient Instructions are defined for the selected Clinic/Procedure Unit in Outpatient Management module. Click Patient Instructions hyperlink to display the Patient Instructions screen.

  2. You can view the Patient Instruction Tab and Procedure Instruction Tab (If Order Catalog with Instruction is selected while scheduling appointment).

    If an Order Catalog is selected for which the order instruction is 'Standard Instruction'  as set in Order Catalog function in 'Order Entry and Result Reporting' module while scheduling appointment, the system validates for the patient ID on click of the patient instruction hyperlink and an alert message is displayed if patient ID is not specified.  If an order catalog is selected for which the order instruction is 'Age/Sex' as set in Order Catalog function in 'Order Entry and Result Reporting' module while scheduling appointment, the system validates for the age and gender of the patient on click of the patient instruction hyperlink and an alert message is displayed if there is no order instructions matching the age and gender of the patient.

  3. Patient Instruction Tab is displayed, by default. Click Procedure Instructions Tab to view details.

    1. Procedure Instructions – This tab gets displayed if the Procedure Instructions are defined for the selected Order Catalog in 'Order Entry and Result Reporting' module and the same Order Catalog is selected while scheduling an appointment.

    2. Description – The description of the procedure Instructions defined in ‘Order Catalog’ function under ‘Instructions’ tab is displayed. The defined instructions can be modified.

    3. Select to Print – Select the check box to print the procedure instruction in the Appointment Slip. The check box is selected for the Order Catalogs with instructions.

    1. Description - The description of the Patient Instructions defined in Clinic/Procedure function is displayed in the order of the serial number.

    2. Select to Print - The Patient Instruction selected as default in Clinic/Procedure Unit function is displayed as selected. Select the check box to print the patient instruction in the Appointment Slip. You can clear the check box to exempt printing the patient instruction in the Appointment Slip.

    3. Click Add Instructions to add more patient instructions to the list. A maximum of 20 Patient Instructions can be recorded. The Patient Instructions defined for the selected Clinic/Procedure are not listed for selection. The system prints the patient instructions in the Appointment Slip.

If the email address is captured during Patient Registration, it is defaulted in the Contacts screen while Booking Appointment and you can modify or edit the email address. Any changes made in the email address while scheduling an appointment is reflected and updated in Master Patient Index module.

• If patient registration is done by selecting ‘Populate From’ as ‘Appointment’ in Register Patient function of Master Patient Index module, on entering the appointment number, the email address is defaulted in the Contact Addresses Tab. You can modify or edit the email address. The recorded email address gets displayed in the appointment screen while booking the next appointment for the patient. The email is sent to the email address that is available in the Appointment Scheduling screen.

• If the email address is not captured during Patient Registration or Appointment Scheduling, and if you try to save the record without entering the email address by clicking [Confirm Booking], an alert message ‘Email ID is not captured. Do you want to proceed?’ is displayed. Click [OK] to proceed without entering the email address or [Cancel] to revert to ‘Book Appointment’ screen and enter the email address to proceed further.

 

Book Appointment for Unregistered Walk-in Patient

Enter the mandatory details of the unregistered patient to proceed with booking.

Book Appointment for Registered Walk-in Patient

Enter the Patient ID. The details of the patient are populated in respective fields and proceed with booking.

Book Appointment for Referral Patients

Enter the Referral ID if the referral patient has a Referral ID. The patient details entered in the ‘ Register Patient Referral’ function are populated in respective fields of the tab. Proceed with the booking process. If the referral patient does not have a Referral ID, register the patient as referral to populate the details of the patient and proceed with booking an appointment.
