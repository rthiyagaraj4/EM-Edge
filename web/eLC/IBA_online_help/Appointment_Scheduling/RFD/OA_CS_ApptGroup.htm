Enter the following details to associate patients with the an appointment group:

 

Note: All fields in this screen are mandatory except Enabled.

 

  1. Group Name – Click [?] and select an appointment group from the Appointment Group webpage dialog. The screen now displays the location, min. patients, max. patients, and the total number of patients already associated with the group, if any. Also, the following patient search fields are now displayed on the screen:

  2. Title – The title of the patient. You can enter Mr, Mrs, Dr, and so on.

  3. First Name – Enter the first name of the patient you want to search for. This is a mandatory field.

  4. Last Name – Enter the first name of the patient you want to search for.

  5. Gender – Select the gender of the patient you want to search for. The available options are “Male”, “Female”, and “Unknown”. This is a mandatory field.

  6. Birth Date – Select the birth date of the patient from the calendar function.

  7. Mobile Number – Enter the mobile number of the patient that you want to search for.
