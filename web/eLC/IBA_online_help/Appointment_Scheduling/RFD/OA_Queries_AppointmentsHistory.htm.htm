Enter the following details to query the appointments history:

 

  1. Appointment Date From - Select the appointment from date by clicking the calendar icon or enter manually in DD/MM/YYYY format. The system allows to select Past Date and Current Date.  

  2. Appointment Date To – Select the appointment to date by clicking the calendar icon or enter manually in DD/MM/YYYY format. The system allows to select Current Date and Future Date.

  3. Specialty – Select the required specialty value from the drop-down list. The value gets populated from Specialty master in Application Masters (AM) module.

  4. Patient ID – Enter the patient ID or search using the [?] function. This field is mandatory.

  5. Location Type – Select the Location Type from the drop-down list. The listed values are Clinic, Daycare Unit and Procedure Unit.

  6. Location – This field gets enabled only when the Location Type is selected. Select the required location using [?] function. The values are displayed based on selected Specialty and Location Type.

  7. Resource Type – This field gets enabled only if any value is selected in the Location field. Select the required Resource Type from the drop-down list. The listed values are Practitioner, Equipment, Room and Other.

  8. Resource – This field gets enabled only when the Resource Type is selected. Select the required resource using [?] function. The values are displayed based on selected Specialty, Location and Resource Type.

  9. Service From and To -  Select the Service From and To value from the drop-down list. The value gets populated from Service master in AM module.
