Existing Holiday

Existing Holiday Tab is used to create holidays. Holidays are marked before generating a schedule for the location. Holidays cannot be generated if a schedule already exists for a location. Holidays are created for Location Types like Clinics, Daycare Units and Procedure Units associated to a location for a date range and a specific reason.

  1. Select the date range for a Location Type and Location. Enter the reason. <Related Field Description>

  2. Click [Select] to list the location for the dates entered.

  3. Select the check box to identify the required date as a holiday. On selecting the select check box, if schedule is already created for that date an alert message 'Clinic Schedule(s) present for this or range of date(s). Do you want to continue?' is displayed. Click [OK] to cancel the schedule for the selected date or [Cancel] to cancel the operation.

  4. Click [Apply].

  5. Click [Query] to search for an existing Holiday.

  6. Click [Delete] to delete a queried record if required.
