Enter the following details to Generate Schedule:

 

  1. Location Type – Select the Location Type as Clinic, Daycare Unit or Procedure Unit.

  2. Location – Select the required Location based on the selected Location Type. This field is mandatory. On selecting the required Location Type, the Locations listed are based on the following conditions:

  3. All the Clinics/Daycare/Procedure units defined under the selected location type that allow appointments (based on the ‘Allow Appointment’ value set in ‘Clinic’, Daycare Unit’ and ‘Procedure Unit’ setup function of EM Outpatient Management module)

  4. All the defined Clinics/Daycare/Procedure units must be associated to the operator station assigned to the logged in user.

  1. Effective From - Enter the date in dd/mm/yyyy or click Calendar function to select the date format from which the schedule is to be generated. This date must be greater than the current date. This field is mandatory.

If it is the first schedule being generated for a Location, the field is left blank. If previously generated schedule is available, it is made default to the lesser of the ‘Effective To Date + 1’ or current date for the same Location. The earliest Effective To Date is taken into account when more than one schedule has been generated for the same Location but for different Practitioners.

  1. Effective To - Enter the date in dd/mm/yyyy format or click Calendar function to select the date up to which the schedule is to be generated. This field is mandatory.

  2. Time Table Available Days - There are seven check boxes indicating seven days of the week. The Location working day(s) are displayed as selected.

  3. All Resources - Select the check box to generate the schedule for all the Resources of the Location selected. The check box is selected, by default. If schedule is not to be generated for all the Resources, clear the check box. If check box is cleared, the Resources associated to the selected Location along with the Timetable available for the Resource gets displayed.

 

Resource Detail

  1. Resource – Resource associated to the location is displayed.

  2. Timetable Available Days - Select the check box to generate the schedule only for the selected days for the Resource. The check box is selected, by default.

  3. Select All – Select the Select All check box to generate schedules for all the resources. Unselect to generate schedules for a particular practitioner. By default, this check box is selected.

  4. Select - Select the check box to indicate that the schedule can be generated for these Resources. The check box is selected, by default.
