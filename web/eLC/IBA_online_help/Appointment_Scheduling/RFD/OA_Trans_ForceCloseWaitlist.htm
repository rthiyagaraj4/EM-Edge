Enter the following query criteria to force close waiting lists:

 

  1. Patient ID - Enter the Patient ID (if known) in alphanumeric format of maximum length as specified in the MP Parameter or select it using [?] function.

  2. Reason for Force Closing – Select the Reason for Force Closing the waiting list from the list of valid values. All the enabled reasons created in ‘Application Masters’ module gets displayed. This field is mandatory.

The number of days after which the waiting list gets elapsed is displayed based on the value set for Cutoff period for Closing Elapsed Waitlist field in OA Parameter setup.
