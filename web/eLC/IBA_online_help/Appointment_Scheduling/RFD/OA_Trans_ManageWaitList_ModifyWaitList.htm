Modify Wait list

Wait list is modified on patient’s request or due to other reasons. This function is used to view and modify details of wait list, reallocate the location and resource for which the wait list is assigned. This function is also used to backdate or postpone the preferred date and reassign a priority for the waitlist.

  1. Select Modify Wait List option from the list of valid values. Modify Wait list screen gets displayed with ‘Wait List Appointment Details’, ‘Patient Details’ and ‘Contact Details’.

  2. Enter details in the mandatory (*) fields. <Related Field Description>

  3. Click [Contact Details] to view the contact details of the patient.

  4. Click [Record] to modify the wait list or [Close] to exit from the screen.

On selecting an elapsed waitlist (elapsed waitlists are those records having preferred date of appointment for waitlist less than the current system date) record for modification, you can modify only the ‘Preferred Date’, all the other details captured during creation is displayed.
