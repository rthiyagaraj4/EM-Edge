Enter the following details to Block/Lift/Cancel the schedules:

 

  1. Action - Select the type of Action that has to be performed on the schedule from a valid list of actions displayed. This field is mandatory. The action selected can be any one of the three types, Block, Lift Block and Cancel.

 

Block Schedule

  • Block - Select this option to block the Appointment Schedules for a Location/Resource for a selected period.

  • Reason – Select the reason from the list of valid values. The list is populated based on the ‘Action’ selected. All the reasons applicable for blocking schedule as set in ‘Other Master setup (Reason)’ of Application Masters module with Block Schedule selected gets displayed. This field is mandatory.

 

Lift Block on Schedule

  • Lift - Select this option to release previously created blocks on Appointment Schedules for a Location/Resource over a selected period.

  • Reason – Select the Reason for lifting block on schedule from the list of valid values. The list is populated based on the ‘Action’ selected. All the reasons applicable for lifting the block on schedule, as set in ‘Other Master setup’ of Application Masters module with lifting block on schedule selected, gets displayed. This field is mandatory.

 

Cancel Schedule

  • Cancel - Select this option to cancel the previously created Location schedules and associated appointments for a Location/Resource over a specified period.

  • Reason - Select the Reason for canceling schedule from the list of valid values. The list is populated based on the ‘Action’ selected. All the reasons applicable for canceling schedule as set in ‘Other Master setup’ of Application Masters module with cancel schedule selected gets displayed. This field is mandatory.

  1. Location Type – Select the Location Type as Clinic, Daycare Unit or Procedure Unit.

  2. Location – Select the required Location based on the selected Location Type using the [?] function. This field is mandatory. On selecting the required location type, the Locations listed are based on the following conditions:

  3. All the Clinics/Daycare/Procedure units defined under the selected Location Type allows appointments if the ‘Allow Appointment’ check box is selected in ‘Clinic’, Daycare Unit’ and ‘Procedure Unit’ setup function of Outpatient Management module.

  4. All the defined Clinics/Daycare/Procedure units must be associated to the operator station assigned to the logged in user.

  1. Resource Type – This field is enabled on selecting a Location. If the Action is selected as ‘Block’ or ‘Lift’, the listed pre-defined value is ‘Practitioner’. If the Action is selected as ‘Cancel’, the listed pre-defined values are Practitioner, Equipment, Room and Other. This is a mandatory field.

  2. Resource – Select the Resource based on the Resource Type selected using [?] function. This field becomes mandatory based on the following condition:

  3. If ‘Check in at Identification’ is set to Resource in ‘Clinic’, ‘Day Care’ and ‘Procedure Unit’ setup (OP module), the resource field becomes mandatory while defining the timetable. If the ‘Check in at Identification’ is set to Non Identifiable, this field is not mandatory.

If the resource is not selected, the slots for all resources get displayed for the selected criteria.

  1. From-To Date - Enter the Date Range for blocking/lifting/canceling in dd/mm/yyyy format or select it using the calendar function. From date corresponds to the date from which the appointment schedules have to be blocked/lifted/cancelled. To date corresponds to the date until when the appointment schedules have to be blocked/lifted/cancelled. This field is mandatory.

  2. Block Time [From-To] – This field gets enabled when Action is selected as ‘Block’. Enter the Block Time range in hh: mi format to block the schedule of a practitioner from booking an appointment for patients.

Example, if the practitioner is scheduled from 08:00 to 12:00 for 7 days and the duration of each slot is 30 minutes; the total number of slots for each day is eight. If the time entered for bulk blocking is from 08:14 to 09:44 for two days, the system considers the highest value of the working slot in the facility as the start time. The lowest value of the working slot in the facility is considered as the end time and system blocks the slot from 08:30 to 09:30, which are two slots per day. As the start time 08:14 is in between the slot time 08:00 to 08:30, the system considers the slot 8:30 as the start time for blocking the appointment. As the end time 09:44 is in between the slot time 09:30 to 10:00, the system considers the slot 09:30 as the end time for blocking the appointment. The system blocks the slots 08:30 - 09:00 and 09:00 – 09:30 for the block time range specified as 08:14 to 09:44.

  1. Day - Select the Day of the week on which the appointment has to be blocked/lifted/cancelled from the list. The listed values are Sunday, Monday, Tuesday, Wednesday, Thursday, Friday and Saturday. You can select random multiple days to block, lift or cancel schedule. Selecting  the respective check boxes provides an option for filtering the days for the specified days in which the practitioner has schedules.

 

On entering, the required criteria click [Select] and an alert message 'Free format schedules will not be blocked' is displayed. Click [OK] to proceed. The generated schedule gets displayed.
