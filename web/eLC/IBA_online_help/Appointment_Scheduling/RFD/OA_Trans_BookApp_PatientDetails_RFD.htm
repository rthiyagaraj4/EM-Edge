Enter the following details to book an appointment:

 

Referral

  1. ID - If appointment has to be made for the referred Patients, Referral ID has to be entered as generated during referral registration or click [?] function to perform Referral Patient Search to locate the referral patient details.

 

Patient Details

  1. Patient ID - Enter the Patient ID (if it is known) in alphanumeric format of maximum length as specified in the MP Parameter. The ID entered must be unique and valid or on clicking [?] function, the Patient Search window is displayed which enables you to select the required Patient ID. In both the cases, remaining fields in Patient Details are populated from the Master Patient Index. If the selected patient has future appointment, a screen gets displayed. The Patient ID field can be left blank, if the Patient has not been registered with the Facility. In that case, minimum demographic details must be entered.

  2. National ID - Enter the National ID in alphanumeric format of maximum length as specified in the MP Parameter. The ‘Prompt’ specified for the National ID in MP Parameter gets displayed as the field legend. The National ID is displayed, if it is entered in the Patient demographic details.

  3. NRIC - Enter the National Registration Identity Card (NRIC) number of the patient. This field is available only when the following setting is done in the ‘MP Parameter’ screen.  

  4. Clear the 'Invoke Routine' check box.

  5. Disable 'Check Digit Scheme' drop-down list.

  6. Select the 'External Interface Required' check box.

On entering the NRIC number, the following fields: Government No, Expiry Date, Prefix, First Name, Family Name, Gender, Birth Date, Residency, Nationality, Race, Address 1, Address 2, Address 3 and Address 4 are populated automatically.

  1. Alternate ID Number - Enter the Alternate ID number in alphanumeric format of maximum length as specified in the MP Parameter. The field title(s) is displayed based on the Alternate ID type(s) set in MP Parameter. Based on the MP Parameter setup, system displays the Alternate ID Type description(s). The Alternate ID is displayed, if it is entered in the Patient demographic details.

  2. Other Alt ID – Select the required Alternate ID from the list. The listed values are defined in the function ‘Alternate ID Type’ of MP module. On selecting the required Alternate ID type, enter the ID in alphanumeric format of 20 characters maximum length. The field ‘Other Alt ID’ is displayed provided ‘Accept Other Alt ID’ is set in MP parameter (Alt ID checks tab).

  3. Patient Name Details - The multi parts of Patient Name are the Title, First Name, Second Name, Third Name, Family Name and Suffix. Based on the setup in MP Parameter (Legal Names), the multi parts of patient name are displayed. The displayed titles and order of appearance are specified in the parameter function. On entering the patient’s name, the system saves the patient name based on the derivation logic defined in MP Parameter and displays them in the text area after Patient ID value.

  4. Gender - Select the Gender of the unregistered Patient from the list of valid values. The valid values are Male, Female and Unknown. This field is mandatory.

  5. Age - Enter the Age of the Patient in years, months and days, if the date of birth is not known. If age is entered, the system automatically calculates the date of birth and displays it in the ‘Birth Date’ field. Either Age or Date of Birth is entered. On entering one of them, the other field is made read-only and disabled.

  6. Birth Date – Enter the Birth Date in dd/mm/yyyy format. If it is entered, system calculates the Patient’s age and displays it in the Age field. Either Age or Date of Birth is entered. On entering one of them, the other field is made read-only and disabled. This field is mandatory.

  7. Birth Place – Enter the Birth Place of the Patient in alphanumeric format of 30 characters maximum length or use [?] function to select the required birth place. If National ID is entered, birthplace is displayed based on the birth place logic defined in the routine associated with the National ID. If the required birth place is not available in the search list, enter the birth place in alphanumeric format of 30 characters maximum length in the text box provided next to the [?] function.

  8. Residency - Select either ‘Citizen’ or ‘Non-citizen’., ‘Citizen’ is selected, by default. If National ID is entered for the patient, ‘Citizen’ is selected and disabled; or both the values are available for selection.

  9. Status - Select either ‘Legal’ or ‘Illegal’ as citizen status. ‘Legal’ is selected, by default. If the patient is a ‘Citizen’, ‘Legal’ is selected and disabled. If the patient is a ‘Non-Citizen’, status can be either ‘Legal’ or ‘Illegal’.

  10. Nationality - The default Nationality set in MP parameter is displayed and disabled, if the National ID is entered, or select the required value using [?] function.

  11. Race - Select the Race from the list. The listed values are defined in ‘Race’ function. This field is mandatory.

  12. Ethnicity – Select the Ethnicity from the list. The listed values correspond to the selected race. This field is mandatory.

  13. Patient Category – Select the value from the list or select using the [?] function. It is not a mandatory field. The Patient Category value is populated based on the setup done in Master Patient Index module.

Others

  1. Reason for Contact - Select the Reason for Contact from the list or select the required reason using [?] function. All the reason(s) created with ‘Book Appointment’ selected in the Reason Master of Application Masters module gets displayed. This field is mandatory.

  2. Receiving Date/Time – Enter the date and time at which the request for appointment was made in dd/mm/yyyy hh:mm format, or click the Calendar function to select the required date. The Receive date/time cannot be greater than the current date/time.

  3. Remarks - Enter the Remarks in alphanumeric format of 60 characters maximum length.

  4. Mode - Select the Mode of Contact from the list of valid values. The mode of contact can be one of the six types ‘By Email’, ‘By Telephone’, ‘By Fax’, ‘In person’, ‘By Web’ or ‘Other’.

  5. Language – Enter the patient language while booking or select using the [?] function. This field enables you to print the online reports in the language preferred by the patient. If the check box ‘Patient Language required during Appointment’ is selected in OA Parameter function, the language of the patient is mandatory.
