Procedure Code for Practitioner

From this function you can Associate procedure codes to the practitioner.

You can link the Practitioner/Procedure code to a particular terminology set based on the Term Codes created in Medical Records module.

The list of terminology set and term codes populate from Medical records Masters for which procedure coding is applicable.

Associating Procedure Code to Practitioner

From the OA Master Setup, select the Procedure Code for Practitioner function.

The Procedure Code for Practitioner screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search]. List of Practitioner name matching the search criteria is displayed when the ‘Link By’ is selected as ‘Procedure Code’. Similarly, the list of Procedure Code is displayed when the ‘Link By’ is selected as ‘Practitioner’.

  3. Select the check box corresponding to the required Practitioner/Procedure Code.

  4. Click [Apply] to associate the procedure code and the practitioner or [Reset] to clear the entered data.
