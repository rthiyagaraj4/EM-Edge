Copy Time Table

From this function you can Copy the schedule of practitioners.

The Copy Time Table helps in defining slots for the resources of a particular location. The time table is scheduled based on the Location, Location Type and Resource selection. This function helps you to maintain the schedules for all the resources in the facility.

Copying the Time Table

From the OA Master Setup, select the Copy Time Table function.

The Copy Time Table Screen gets displayed.

  1. Select the Location Type from the list. The listed values are Clinic, Day Care and Procedure Unit.

  2. Enter or select the Location using the [?] function. According to Location Type selected the Location values are filtered.

  3. Select the Resource value from the list. The listed values are based on Location and Location Type selected.

  4. The time table details for selected day are displayed.

  5. Select the resources for generating time table or check ‘Select All’ check box to select all the resources.

  6. Click [Apply] to copy the schedule or [Reset] to clear the entered data.
