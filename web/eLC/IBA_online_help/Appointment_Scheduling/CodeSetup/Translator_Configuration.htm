Translator Configuration

From this function you can Define the number of translators for other nationals. You can Query for the existing records.

This function allows you to define the number of translators and number of patients for each translator. The system alerts the user while booking a new appointment for other nationals against those translators when the maximum appointment of patients for the translator is reached.

Defining Translator Configuration

From the OA Master Setup, select the Translator Configuration function.

The Translator Configuration screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the translator configuration details or [Reset] to clear the entered data.

  4. Click [Query] to search for the existing translator configuration details.
