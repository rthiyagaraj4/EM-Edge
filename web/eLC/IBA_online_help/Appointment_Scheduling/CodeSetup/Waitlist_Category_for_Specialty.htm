Waitlist Category for a Specialty

From this function you can Create an association of Waitlist Category to a Specialty and Query for a desired Waitlist Category for Specialty record.

Waitlist Category for a Specialty function is used to associate various wait list categories to the specialties. Wait list categories are defined in Waitlist Category function.

Creating an association of Specialties to Waitlist Category

From the OA Code Setup, select the Waitlist Category for a Specialty function.

The Waitlist Category for a Specialty screen gets displayed.

  1. Click [Create] to invoke the create screen.

  2. Enter the waitlist category or select using [?] function. All the Wait list categories created are displayed. This field is mandatory.

  3. Click [Search] to display specialties.

  4. Select the check box against required specialties to associate them to the selected Waitlist Category.

  5. Click [Apply] to associate Specialties to Waitlist Category or [Reset] to clear the entered data.

  6. Click [Query] to search for an existing record.
