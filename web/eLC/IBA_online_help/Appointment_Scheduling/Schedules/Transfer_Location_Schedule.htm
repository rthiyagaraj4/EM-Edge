Location Reschedule

From this function you can perform Reschedule of a particular location to an earlier or future period along with its appointments.

If a schedule needs to be transferred to an earlier or future period, it can be achieved within a single location. Location Reschedule function allows transferring a schedule along with its appointments from one period to another period within the clinic. This depends on whether the transfer is between two periods where timetable is set with common attributes, eliminating the process of canceling the existing location schedules and regenerating them with a different location timetable.

 

Process Flow Diagram

Rescheduling Location

From the OA Schedules, select the Location Reschedule.

Location Reschedule screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Select] to display the schedule details for the days selected.

  3. Select the check box to select the schedule that has to be  rescheduled to another day. If the check box is cleared, the corresponding schedules are not transferred to different resource.

  4. Click [Apply] to transfer the location schedule to any other day.

  5. Click [Generate Report] to generate a report on  rescheduled of location.

 

Select check box is disabled for location schedule having appointments booked with a secondary resource.
