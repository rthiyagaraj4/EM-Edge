Change Resource for Generated Schedule

From this function you can Transfer schedule from one resource to another within a location.

Practitioners belonging to a common specialty and practicing in the same clinic can substitute for each other by transferring a schedule. Practitioners to which the schedule is transferred must not have any existing schedule for the specified period. Schedule that is transferred is specific to a period range and has to be justified by a reason. On transferring the schedule, all appointments already booked gets transferred.

Changing Resource for Generated Schedule

From the OA Schedules, select the Change Resource for Generated Schedule function.

Change Resource for Generated Schedule screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Select] to display the schedule details for the days selected.

    1. From Time – This field displays the start time of an appointment based on the appointment type set up in timetable for scheduling.

    2. To Time – This field displays the end time of an appointment based on the appointment type set up in timetable.

    3. Patient Details – This field displays the details of the patient like Patient ID (Incase of registered patient), Name of the Patient, Gender and Age. This field is blank if no appointments exist.

    4. Visit Type – This field displays the applicable visit type recorded while giving the appointment.

    5. Contact Details – This field displays the contact details of the patient captured while booking an appointment.

    6. Reason For Contact – This field displays the reason for the patient seeking an appointment.

    7. Sec Res – This field displays the secondary resource selected for an appointment with a hyperlink or this column is blank.

  3. Select the check box to select the required schedule to be transferred to another resource. If the check box is cleared, the corresponding schedules are not transferred to any other resource.

  4. Click [Apply] to transfer the schedules to the selected resource.

  5. Click [Generate Report] to generate a report on change of resource.
