Block/Lift/Cancel Schedule

From this function you can Block a schedule, Lift a blocked schedule and Cancel a Schedule for a Resource and Location for a selected period.

A schedule generated against a location can be either blocked or canceled due to various reasons. The task is performed against a location or resource based on a period range and is justified by a reason for auditing. The Block option is used to block the schedule to prevent appointments from being booked. You can block the schedule of a practitioner from booking appointments based on the slots or slabs for a specific date and time range. The Lift option is used to lift the previously blocked schedule for a Resource and Location for a selected period. Once a block is lifted from a Location Schedule, appointments are booked for that Location Schedule. The Cancel option is used to cancel previously created Location Schedules and associated appointments for a Practitioner and Location for a specified period.

 

If there are already appointments booked for a schedule, you can still block the schedule if the parameter Block Schedule even if Appointments exist has been selected in the OA Parameter  screen. Else, you need to either cancel or transfer all existing appointments and only then block the schedule.

Once the schedule is blocked, the schedule will appear in Red colour and you will not be able to perform any action.

Blocking/Lifting/Canceling Schedules

From the OA Schedules, select the Block/Lift/Cancel Schedule function.

Block/Lift/Cancel Schedule screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. The schedule details are displayed based on the query criteria. <Related Field Description>

  3. Click ‘ No of Appts ’ to view the list of appointments for a day.

    1. From Time – This field displays the commencement time of an appointment based on the appointment type set up in timetable for scheduling.

    2. To Time – This field displays the end time of an appointment based on the appointment type set up in timetable.

    3. Patient Details – This field displays the details of the patient like Patient ID (Incase of registered patient), Name of the Patient, Gender and Age. This field is blank if no appointments exist.

    4. Visit Type – This field displays the applicable visit type recorded while giving the appointment.

    5. Reason For Contact – This field displays the reason for the patient seeking an appointment.

    6. Contact Details – This field displays the contact details of the patient captured while booking an appointment.

    7. Sec Res – This field displays the secondary resource selected for an appointment with a hyperlink or this column is blank.

    8. Appt Given By – This field displays the name of the person who has given the appointment.

    9. Appt Given On – This field displays the Date and Time on which appointment is scheduled.

  4. Select the check box against the required schedule to block, cancel or lift a blocked schedule. Select the check box at the header to select all schedules to perform the action. If the check box is cleared, the corresponding schedules are not blocked, cancelled or lifted.

If the time is not entered in the time slot, the slots for the selected date range get displayed based on the selected criteria.

  1. Click [Apply] to Block, Lift the Block or Cancel a schedule.

  2. Click [Generate Report] to generate a report on blocked, cancelled or blocked schedules lifted.

Select check box is disabled for Block and Lift Block functions if appointments are booked against the location.

 

Transferring or cancelling appointments within blocked schedule

1. Navigate to Transfer/Cancel Appointments screen.

2. Select ‘Resource Type’ as Practitioner.

3. Select the practitioner whose schedule has been blocked.

4. Select the appointment date (schedule) that has been blocked.

5. Transfer or cancel the appointments as required.
