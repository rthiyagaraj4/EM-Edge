Appointment Serial Number

From this function you can Define an Appointment Serial Number.

Scheduling an appointment against a clinic generates an Appointment Reference Number, which is a unique identifier for the appointment. Appointment Serial Number function allows defining the range for number generation based on which the system generates appointment numbers in a sequential method. The serial number must not be changed as this can result in duplicate Appointment Reference Number.

Defining Appointment Serial Number

From the OA Admin Setup, select the Appointment Serial Number function.

The Appointment Serial Number Screen gets displayed.

  1. Enter the Next Serial Number in numeric format of 10 digits maximum length. The next serial number corresponds to the next appointment booked. The system increments this number automatically for each appointment booking. This field is mandatory.

  2. Enter the Maximum Serial Number in numeric format of 10 digits maximum length. This field is mandatory.

  3. Click [Apply] to define Appointment Serial Number or [Reset] to clear the entered data.

 

Appointment Serial Number is a one time setup and cannot be changed after setting.
