Wait List Audit Details

From this function you can Query all the details of waitlists applicable to a particular patient.

The function ‘Wait List Audit Details’ is used to query the details of waitlist applicable to a particular patient. A patient is assigned to a waitlist, if appointments are not available,. The system maintains the list of wait listing patients based on priorities; thus, ensuring the confirmation of bookings for the patients based on the criticality or need. Patients on a waitlist are invited for an appointment on preferred date. Healthcare managers can use the function on a daily basis to find the patients in waiting list.

Querying to view the Wait List Audit Details

From the OA Queries, select the Wait List Audit Details function.

The Wait List Audit Details screen gets displayed.

  1. Enter the required wait list number in alphanumeric format of 12 characters maximum length. This field is mandatory.

  2. Select the invited date using the Calendar function.

  3. Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter. In case the Patient ID is not known, a standard lookup screen can be accessed by clicking [?] function and by specifying the required query criteria.

  4. Enter a valid Patient Name in alphanumeric format of 40 characters maximum length . If a valid Patient ID is entered or selected using [?] function, the system displays the Patient Name corresponding to the entered or selected Patient ID.

  5. Click [Reset] to clear the entered data.

 

The start and end time of the block along with the reason for block will be displayed for a Date, Day, Location and Resource combination.
