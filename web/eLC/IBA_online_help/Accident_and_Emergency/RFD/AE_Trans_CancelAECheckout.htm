Enter the following details:

 

Cancel Checkout

  1. Reason for Cancellation – Select the Reason for Cancellation of AE Checkout for the patient from the list. The values created using the function ‘Reason’ under Other Master Setup in Application Masters Module get listed. The reasons assigned to ‘Cancel Checkout’ under Outpatient Management get listed.

  2. Cancelled By – Enter the Practitioner or select the practitioner by using [?] function who is authorized to cancel the AE Checkout. The name of the Practitioner who checked out the patient is displayed, by default.

  3. Remarks – Enter the remarks or instructions for the cancelling AE Checkout in alphanumeric format of not more than 200 characters in length.

  4. Bed/Bay No – Enter or select the Bed/Bay Number by using [?] function to assign a bed for the patient.
