Enter the following details to Reserve Bed:

 

  1. Location - Select a Location from the list to reserve a bay. The list of Clinics displayed is limited to the Clinics of the logged in Facility and only those clinics are populated whose level of care is set to emergency. This field is mandatory.

  2. Priority Zone - Select the required Priority Zone from the list. The selected value indicates the bay defined for the selected zone has to be reserved. The listed values are critical (Red), semi critical (Yellow), non critical (Green) or Unassigned. This field is mandatory.

  3. Treatment Area - Select the Treatment Area to reserve a bay. The Treatment Areas defined for the selected location and priority zone in ‘Treatment Area for Location’ function gets displayed. This field is mandatory.

  4. Bed/Bay No – Enter the Bed/Bay Number in alphanumeric format of eight characters maximum length, or select the required Bed/Bay Number using [?] function. Clicking the [?] function displays a ‘Bed/Bay Availability Chart’ with vacant beds in the selected Location and associated Treatment Area. This field is mandatory.

  5. Reserve Until – Enter the date and time in dd/mm/yyyy format or click the Calendar function to select the date until when the Bed/Bay has to be reserved. This field is mandatory.

  6. Reasons - Enter the reason for Bed/Bay reservation in alphanumeric format. This field is mandatory.
