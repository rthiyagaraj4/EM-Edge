Enter the following search criteria to view the secondary triage history:

 

  1. Patient ID - Enter the Patient ID in alphanumeric format of maximum length. Click [?] function to invoke 'Search Patient' function, using which the required Patient ID can be selected.

  2. Encounter ID - Enter the Encounter ID in numeric format of 12 digits maximum length. Click [?] function to invoke 'Search Visit' function, where the required Encounter ID can be selected.

  3. Location - Select the Location from the list. This field represents the Location to which the Patient has to be registered. All the locations associated to the logged in users operator station with Level of Care as 'Emergency' are displayed.

  4. Date [From - To] - Enter the date range in dd/mm/yyyy format to search for the secondary triage history of the patient for the specified date range. You can also select the date using the Calendar function.
