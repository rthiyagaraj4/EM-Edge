Enter the following details to create the Patient File:

 

  1. File Storage Locn - Select the File Storage Location from the list. The permanent file storage locations of the logged-in facility are available in the list. This field is mandatory. If only one File Storage (FS) Location exists for the logged in facility, all the files created are assigned to this FS Location. If more than one FS Location exists, this field is enabled and the list contains all these FS Locations. Select the required location and assign the patient file. This field appears if 'Patient File Tracking' module is installed and file Storage Location is defined.

  2. File Type – Select the File Type to be associated from the list. The listed types are defined in File Type function. This function is mandatory. The file type defined should be associated to Storage Location using the function ‘File Type for File Storage Location’ in Patient File Tracking module.

  3. Old File No. – Enter the old file number, if it exists for the patient. This field is displayed, if 'Prompt for Old MRN' is selected for the MR section attached to the selected patient series.
