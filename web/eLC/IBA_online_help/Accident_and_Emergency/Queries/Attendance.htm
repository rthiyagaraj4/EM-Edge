Attendance

From this function you can Query the attendance list for a location within the emergency department.

The function provides various query criteria that can be used to filter the listing. This can be used to view the listing of all the patients, who have attended the emergency department with details of their visits and current status.

Querying Attendance

From the AE Queries, select the Attendance function.

The Attendance screen gets displayed.

  1. Select a Location from the list.

  2. Click Calendar function to select the date range in dd/mm/yyyy format. The visit details are limited to the selected date range and the location. The current system date is displayed, by default. This field is mandatory.

  3. Click [Search] to select a specific record or [Clear] to clear the records displayed in the result area.

  4. Click [Reset] to clear the entered data.
