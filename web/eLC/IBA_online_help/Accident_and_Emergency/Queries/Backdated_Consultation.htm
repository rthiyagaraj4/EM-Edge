Backdated Consultation

From this function you can Query the list of patient records with backdated Consultation for required date range.

This function allows you to query the list of patient records with backdated Consultation for required date range or for a specific patient.

Querying Backdated Consultation

From the AE Queries, select the Backdated Consultation function.

The Backdated Consultation screen gets displayed.

  1. Enter the Transaction Date Range (From-To) in the respective fields. This field is mandatory.

• ‘From Transaction’ date should be less than or equal to current date.

• ‘To Transaction’ date should be greater than or equal to From Transaction date.

  1. Select the Patient ID using the lookup if required.

  2. Click [Search] to select the records that satisfy the query criteria or [Reset] to clear the entered search criteria.

  3. The Backdated Consultation details are displayed.

  4. Click [Clear] to clear the result screen.
