AE Bed/Bay Availability Chart

From this function you can Query the online availability of bays or beds within various treatment areas of the emergency department.

The bed availability chart displays all beds or bays within the department and represents the current status of each bed or bay using color indicators.

Querying Bed/Bay Availability

From the AE Queries, select the AE Bed/Bay Availability Chart function.

The AE Bed/Bay Availability Chart screen gets displayed.

  1. The Bed or Bay details are displayed or enter the search criteria and click [Search] to select a specific record.

    1. Select the required Location from the list for which the Bed/Bay has to be viewed.

    2. Select the Treatment Area from the list for which the Bed/Bay has to be viewed. Treatment Areas for the selected location gets listed.

    3. Select the required Bed/Bay classification from the list.

    4. Select the required Bed/Bay status from the list. The list of values are ‘Vacant’, ‘Occupied’, ‘Reserved’ and ‘All’. ‘Vacant’ is displayed, by default.

  2. The beds within a Location/Treatment Area are displayed in cells and are colored uniquely to indicate the status. The color codes represent the following status:

• Vacant Normal Bed/Bay

• Occupied Normal Bed/Bay

• Reserved Normal Bed/Bay

• Vacant Spare Bed/Bay

• Occupied Spare Bed/Bay

On pointing the mouse over a cell, additional details of the bed/bay are displayed as tool tip. The table below provides these details.

 

Status

Details Available

Vacant (Normal Bed/Bay and Spare Bed/Bay)

Bay No, Bay type, Location and Treatment Area.

Occupied (Normal Bed/Bay and Spare Bed/Bay)

Bay No, Bay Type, Location, Treatment Area and Patient ID.

Reserve Bed/Bay

Bay No, Bay Type, Location, Treatment Area and Patient ID.

 

The tool tip details may vary based on user locations.

 

  1. Click [Reset] to clear the entered data.
