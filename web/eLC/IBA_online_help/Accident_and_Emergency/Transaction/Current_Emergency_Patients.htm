Current Emergency Patients

From this function you can Manage the patients by viewing the Patient Queue Status in the Emergency Clinic. You can Record the Arrival of patient, Vital signs, Assign or Reassign practitioner and Check the patient out.

Patient queues comprise of patients at various stages of their interaction with the facility. This function provides a view of patient queue in the emergency department and allows various tasks to be performed on a patient based on the patients queue status. The various tasks performed from this function are Secondary Triage, Assign/Reassign Practitioner, Assign/Reassign Treatment Area, Trauma Assessment, Primary Assessment, Start Consultation, Check Out and View Movement History.

Managing Emergency Patient Queue

From the AE Transactions, select the Current Emergency Patients function.

Current Emergency Patients screen gets displayed.

  1. Patients having their attendance registered are displayed in the Current Emergency Patients screen, by default. Enter the query criteria to select the required patient. The displayed patient records are segregated based on the Location/Practitioner/Room combination. The patient listing is sorted based on the priority zone under the following sub tabs, namely, Critical, Semi-Critical, Non-Critical, Unassigned and All. < Related Field Description >

  2. Click [Refresh] to refresh the Patient records or re-query the screen manually. By default, this screen is refreshed automatically based on the Refresh value parameter setup.

  3. Click to open the ‘Configure Display Order’ dialog where you can configure the column names that you want to appear. On clicking [Save], the column names appear as set in the dialog. Thereby, you can set your own preference of the column to be displayed.   

  4. Click Patient Details hyperlink of a particular patient to perform the various queue management functions.

 

The queue management functions are based on the following characteristics:

  1. If Clinician Access module is not installed, you are able to perform the following set of functions in Accident and Emergency module:

Assign/Reassign Treatment Area

Assign/Reassign Practitioner

Check Out

Movement History

Record Vital Signs

Register Newborn for AE - This function is available only when following conditions are met:

• For Mother Patients (Female patients) only.

• For Mother Patients who completed either ‘Assigned to Treatment Area’ or ‘Consultation Started’. The link is not available for patients in ‘Checked in’ status even if ‘Secondary Triage’ status is performed before performing ‘Assign to Treatment Area’.

• For users linked to the ‘Practitioner Types’ with ‘Register Newborn for AE’ privilege in ‘Access Rights for Queue Management’.

• The ‘Invoke Register Newborn for AE’ checkbox must be selected in AE Parameter

  1. If Clinician Access module is installed, you are able to perform the following set of functions in Accident and Emergency module:

Assign/Reassign Treatment Area

Record Vital Signs

  1. After assigning a practitioner for the patient, the Clinician Access chart menu is displayed. You are able to perform the following functions from the Clinician Access Chart menu:

Primary Assessment

Trauma Assessment

Secondary Triage

Check Out/Close Consultation

 

Only if you move the mouse pointer over any other patient name hyperlinks, the pop-up menu gets disappeared.

When you mouse over on any of the enabled “Patient Name” hyperlink, the previously displayed menu-list disappears and only the previous selected “Patient Name” column will be highlighted in grey colour. You need to click on the “Patient Name” hyperlink to enable the Pop-up menu again.
