Release Deceased/Body Part

From this function, you can Release the deceased and body part from the mortuary.

Once the requested services are rendered within the mortuary, a deceased/body part can be released from the mortuary and handed over to the identified claimant. This function is used to formally release a body from the mortuary. This can be done only when a deceased/body part is moved to an area in the mortuary that has been specifically identified as a Body Release Area. A deceased/body part is released from the mortuary after the claimant details are entered and the burial permit is issued for the deceased.

Releasing Deceased/Body Parts

From MO Transactions, select Release Deceased/Body Part function.

The Release Deceased/Body Part screen gets displayed.

 

The screen consists of 2 tabs:

Deceased Patients

Body Parts

Deceased patients or body parts registered in the mortuary and assigned to the release area of the mortuary through the 'Manage Deceased/Body Parts' function for the current date are listed. The 'Deceased Patients' Tab is displayed, by default.

 

This function refreshes the patient queue list as per the time limit as set in the field 'Refresh Interval' of the MO Parameters.

To release a body from the mortuary, it should be moved to the release area of the mortuary from any other area and burial permit should be generated for the body.
