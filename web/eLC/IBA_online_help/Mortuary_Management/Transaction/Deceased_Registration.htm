Deceased Registration

From this function, you can Register deceased patients to render services in the Mortuary.

Registration is a process used to formally accept deceased patients into the mortuary before rendering services. This function identifies details of the deceased patient like manner of death, condition of the body and identification marks on the body. This function also allows identification of medico legal cases and supports recording of details if police escorts the body. This function allows registration of patients that are deceased and checked for mortuary.

Registering Deceased Patients

From MO Transactions, select Deceased Registration function.

The Deceased Registration screen gets displayed.

  1. Patients discharged or checked out from AE, IP or OP modules for the current date with the discharge type as 'Deceased' are displayed, by default.

  2. Enter the search criteria and click [Search] to select a specific deceased patient. Deceased Patient Details are displayed based on the entered criteria. The status of the patient is indicated by a unique color code. The burial permit of the patient is displayed if Postmortem is not required for the patient or if postmortem recording has been completed and the burial permit is already generated for the patient.

    1. Deceased Patient ID - Enter the deceased patient ID in alphanumeric format 10 characters maximum length or select the deceased patient ID using [?] function. This is a mandatory field.

    2. Burial Permit No - Enter the burial permit number in alphanumeric format of 16 characters maximum length or select the burial permit number using [?] function. The [?] function displays only those deceased patients not registered in the mortuary and for which the burial permit is generated.

    3. Deceased Date - Enter the deceased date range of patients to register in the mortuary. This is a mandatory field. The current date range is displayed, by default.

    4. Location - Select the location from the list where the patient was registered. The listed values are 'Clinic', 'Daycare Unit', 'Procedure Unit', and 'Nursing Unit'. On selecting the required location type, the different locations of the selected location are displayed in the corresponding list to select the specific location.

    5. Brought In Dead - Select the check box to display only 'Brought in Dead' patients for registering in the mortuary.

    6. Medico Legal Case - Select the check box to display medico legal deceased patients for registering in the mortuary.

    7. Clinical Postmortem - Select the check box to display clinical postmortem patients for registering in the mortuary.

    8. Registered Cases - Select the check box to view the patients already registered in the mortuary.

  3. Click '+' hyperlink to View Patient Details.

  4. Click Location hyperlink to view Previous Encounters of the patient.

  5. Click [External Deaths] to register deceased patients brought from other facility. The Deceased Registration screen opens with the Deceased Patient ID enabled to select the deceased patient from different facility.

  6. Click the Deceased Patient Details hyperlink to register the deceased patient in the mortuary.

  7. Click [Apply] to register the deceased patient in the mortuary or click [Query] to go back to the Deceased Registration main screen and query records for which registration is pending.

 

This function refreshes the patient queue list as per the time limit as set in the field 'Refresh Interval' of the MO Parameters.
