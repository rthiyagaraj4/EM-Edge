Body Part Registration

From this function, you can Register various body parts of a patient in the mortuary before rendering services of the mortuary.

Body parts registration can be categorized into one of the following: Amputated Body Part, Dead Fetus or Unidentified Body Part. This function also allows identification of medico legal cases and supports recording details if police escorts the body part. Body Parts can also be referred from some other facilities. The claimant details can be recorded and a burial permit can also be generated for the body part using this function.

Registering Body Parts

From MO Transactions, select Body Part Registration.

The Body Part Registration screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Type of Body Part from the list. The listed values are ' Amputated Body Part', ' Dead Fetus' and ' Unidentified Body Part'.

    On selecting 'Unidentified Body Part', 'Patient ID' field is displayed to select the patient ID of any deceased unknown patient. This is a mandatory field.

    On selecting 'Dead Fetus', a field 'Mother ID No' is displayed to select the patient ID of a live mother registered in the facility of the dead fetus.

    Amputated Body Part belongs to a male or female live patient. On selecting 'Amputated Body Part', a field 'Patient ID' is displayed to select the ID of the patient with an amputated body part.

  3. Select the Patient ID or Mother ID No using [?] function. This is a mandatory field.

    This field is displayed if the 'Body Part Type' field is selected as 'Dead Fetus'. Enter the mother patient ID in alphanumeric format 10 characters maximum length or select the mother patient ID using [?] function. The [?] function permits only female patient Id search. The age for a mother patient is validated as set in the Patient Identity Management module. This is a mandatory field.

    This field is displayed and gets enabled if the 'Body Part Type' field is selected as 'Amputated Body Part' or 'Unidentified Body Part'. Enter the patient ID in alphanumeric format 10 characters maximum length or select the patient ID using [?] function. The [?] function permits selection of male and female patient ID on selecting 'Amputated Body Part' and patient ID of an unknown deceased patient on selecting 'Unidentified Body Part'. This is a mandatory field. Patient ID selected should not be of a deceased patient while selecting the field 'Body Part Type' as 'Amputated Body Part' or 'Dead Fetus'.

  4. The patient details like the name, age, gender and Patient ID are displayed.

This screen consist of 3 tabs

Registration Details

Claimant

Burial Permit

  1. Click [Apply] to register the body part in the mortuary. The 'Internal Report' screen is displayed. Click [OK] to generate the report or label.

  2. A mortuary registration number is generated for the body part based on the numbering control specified in the MO parameter function.

  3. Click [Query] to query on existing records.

    1. Enter the search criteria and click [Search] to display patients for whom body parts were registered in the mortuary.

    2. The status of the patient is indicated by a unique color code.

    3. Select the required ‘Patient Name’ hyperlink to view the details

    4. The Body Part Registration function consists of 3 Tabs namely; Registration Details, Claimant and Burial Permit. The details captured while registering the body part in the mortuary is displayed under respective tabs.

    5. On selecting the hyperlink with the color code status as 'Clinical Postmortem', or 'Medico Legal', the Burial Permit tab is not displayed. The Burial Permit tab appears only after postmortem is recorded or if postmortem is not required for the body part.

    6. You can change all other details except some details that are disabled and displayed as read only text. Details not entered during registration can be recorded and the record can be modified.
