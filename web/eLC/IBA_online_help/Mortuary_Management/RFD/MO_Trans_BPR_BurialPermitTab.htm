Burial Permit Tab

This tab is used to record the burial permit details. Burial Permit Tab is available only if the Postmortem is not required for the deceased patient.

  1. Generate Burial Permit Number - Select the check box to generate the burial permit number by the system.

  2. Burial Permit Number - Enter the burial permit number in alphanumeric format of 16 characters maximum length. This is a mandatory field. This field is disabled if the 'Generate Burial Permit Number' check box is selected.

  3. Date Time of Claiming - Enter the date and time in dd/mm/yyyy hh:mm format of when the body part was claimed or select the date and time using the calendar function. By default, the current date and time is displayed. This is a mandatory field.

  4. Vehicle Number - Enter the number of the vehicle carrying the body part in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  5. Burial Permit Approved By - Enter the name of the practitioner who approved the burial permit in alphanumeric format of 30 characters maximum length or select the required practitioner using [?] function. This is a mandatory field.

  6. Burial Approved Date - Enter the date and time in dd/mm/yyyy hh:mm format or select the date and time using the calendar function of when the burial permit was approved. The entered date cannot be greater than the current system date and time. By default the current system date and time is displayed. This is a mandatory field.
