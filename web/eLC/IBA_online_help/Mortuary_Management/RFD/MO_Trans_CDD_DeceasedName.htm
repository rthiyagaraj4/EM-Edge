Change Deceased Details

Details of the patient already captured earlier are displayed.

  1. Burial Permit No - The Burial permit number is displayed if it was recorded while registering the deceased patient. This field is disabled.

  2. Deceased Patient ID - The selected deceased patient Id is displayed and disabled.

 

This screen comprises of four tabs namely

Registration Details

Organ Donation

Claimant

Burial Permit

The details captured during registering the deceased patient in the mortuary are displayed under respective tabs. Refer Deceased Registration function fields.

 

On selecting the hyperlink with the color code status as 'Clinical Postmortem' or 'Medico legal', the Burial Permit tab is not displayed.

You can change all other details except some details that are disabled and displayed as read only text. Details not entered during registration can be recorded and the deceased patient record can be updated.

 

When the orders placed for  post mortem are cancelled or discontinued in CA, you can release the body to Release Area without a postmortem. This can be done by clearing the ‘Postmortem Required’ check box under Registration Details tab in Change Deceased Details screen. On clearing the check box, the ‘Record Postmortem’ link will not be available in the Manage Deceased/Body Part screen. Thereby, the body can be released to the Release Area without post-mortem.
