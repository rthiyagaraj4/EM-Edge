Body Parts Tab

  1. Enter the search criteria and click [Search] to select a specific body part. Body Part details are displayed based on the entered criteria. The status of the patient is indicated by a unique color code.

    1. Registration No - The mortuary registration number generated on registering the deceased /body part is displayed.

    2. Registration Date/Time - The date and time of registration is displayed.

    3. Deceased Name/Patient Name - The name of the deceased is displayed and hyperlinked in 'Deceased Patient' Tab and the name of the patient whose body part is registered in the mortuary is displayed and hyperlinked in the 'Body Parts' Tab.

    4. Patient ID - The Patient ID generated when the patient was registered in the Master Patient Index module is displayed.

    5. Age - The age of the patient recorded when the patient was registered to the facility is displayed.

    6. Sex - The gender of the patient is displayed.

    7. PM Y/N - If postmortem is required for the deceased/body part, then "Yes" is displayed, else "No" is displayed. The values are displayed based on the Postmortem Required field in Deceased Registration/Body Part Registration function.

    8. Duration in Mortuary - The duration of the deceased/body part in the mortuary is displayed.

    9. Religion - The religion of the deceased/body part is displayed.

    10. Encounter ID - The last encounter ID of the patient is displayed.

    11. Location - The location assigned to the deceased /body part in the area during registration is displayed.

    12. Body Part Type - This is displayed only in the 'Body Parts' Tab. The type of body part registered for the patient is displayed as 'Amputated Body Part/Dead Fetus/Unidentified Body Part'.

    13. Burial Permit No - The Burial Permit Number is displayed if the burial permit was generated for the patient.

     

    1. Registration number - Enter the registration number or select the registration number using [?] function.

    2. Patient ID - Enter the patient ID in alphanumeric format of 10 characters maximum length or select the deceased patient ID using [?] function.

    3. Area Type - Release Area is displayed, by default.

    4. Area - Select the area from the list. The listed values are defined in the Area function.

    5. Postmortem Status - Select the status of postmortem from the list. The listed values are 'Not Applicable' and 'Completed'.

    6. Release Status - The value is displayed as "Not Released" and is disabled, by default.

    7. Date From/To - Enter the date range in dd/mm/yyyy format as to when the deceased patient was released to the release area of the mortuary or select the date range from the calendar function. This field is mandatory. By default the current date is displayed.

    8. Gender - Select the gender from the list.

    9. Burial Permit No - Select the burial permit number using [?] function.

    10. Body Part Type - This field is displayed and disabled as 'Not Applicable' for deceased patients. Select a value from the list for body parts. The listed values are Amputated Body Part, Dead Fetus and Unidentified Body Part.

    11. Refresh Interval - The Refresh Interval is displayed as recorded in the MO Parameter function.

  2. Click the Patient Name hyperlink to release body parts from the mortuary. On selecting the 'Patient Name' hyperlink, a message is displayed if burial permit is not generated for the body part.

    This screen consists of 2 tabs namely

    Release Details

    Claimant

     

    By default, the 'Release Details' Tab is displayed.

    'Release Details' Tab

    All the details displayed were captured while the deceased/body part was registered to the facility.

    1. Burial Permit Issued - Select the check box if burial permit was issued for the patient.

    2. Burial Permit Issue Date/Time - Select the date and time using the calendar function as to when the burial permit was issued. This field is mandatory on selecting the 'Burial Permit Issued' field.

    3. Released By - Enter or select the name of the practitioner who released the body from the mortuary. This is a mandatory field.

    4. Release Date/Time - Enter the date and time when the deceased/body part is released from the mortuary. By default, the current date and time is displayed. This is a mandatory field.

    5. Exportation Required - Select the check box if exportation is required for the deceased/body part.

    6. Unclaimed Body - Select the check box if the deceased/body part patient is an unclaimed body. This field is displayed as selected if it was recorded earlier while registering the deceased/body part in the mortuary.

    'Claimant' Tab

    Click the 'Claimant' Tab to view the claimant details of the deceased/body part which were recorded while registering the deceased/body part to the mortuary. The displayed details can be modified.

  3. Click [Record] in the 'Release Details' Tab to release the deceased/body part from the mortuary.
