Postmortem Number Control

From this function you can Define the format and sequence to generate the Postmortem number.

Every deceased patient registered within the mortuary and has to undergo postmortem is identified using a unique postmortem number. This function is used to define the format and sequence that is used to generate the postmortem number. The format of the number can be prefixed or suffixed by identifiers like the facility ID or the year of generation to avoid exhaustion of the series. You can also define the structure of the number by defining the associate details to the postmortem registration number.

Defining Postmortem Number Control

From the MO Administration, select Postmortem Number Control function.

The Postmortem Number Control gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Apply] to save the defined Postmortem Number Control or [Reset] to clear the entered data.
