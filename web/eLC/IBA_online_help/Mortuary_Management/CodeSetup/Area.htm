Area

From this function you can Create various Mortuary Areas and associate to an Area Type. You can Query for an existing Area.

A mortuary is divided into various areas based on the type of activity performed which influences the workflow to manage deceased and body parts within the mortuary. You can define the capacity limit for accommodating deceased bodies. The system allows identification of mortuary areas for the purpose of postmortem, release and storage of bodies.

Creating an Area

From the MO Code Setup, select Area function.

The Area screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Area or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Area records.
