Reconcile Bank-in by Account

From this function you can Reconcile the Bank-in statements with the Receiver's statement either manually or automatically. You can Query to view the details.

Reconciliation is based on the Account Code and is done by matching the amount from both sides. In manual reconciliation, you can select one to many records and vice versa. The records, which are pending for reconciliation, can be viewed by clicking [Execute Query] or pressing [F8]. System populates the pending records based on the Account Code. If 'Reconciliation Option' is selected as 'Automatic Reconciliation by Bank-in No', system updates the reconciled status to match the records on its process completion. If 'Reconciliation Option' is selected as 'Manual Reconciliation', you can choose individual records from both sides by matching it with the total amount of records selected.

Reconcile Bank-in details based on Account Codes

From the External Accounting System > Reconcile Bank-in Amount, double-click Reconcile Bank-in By Account Function.

The Reconcile Bank-in By Account screen gets displayed.

  1. Click [Execute Query] or press [F8] to navigate and select an Account Code.

  2. Click [List of Values] to select a valid Account Code for which Bank-in reconciled report has to be generated. On selecting a valid Account Code, system displays the description for user verification. This is a mandatory field.

  3. Click the Calendar function to select a month. The pending records are displayed based on the month selected.

  4. Select a Reconciliation Option from the list. The system-defined values are 'Automatic Reconciliation by Document Reference' and 'Manual Reconciliation'. If ‘Manual reconciliation’ is selected, you have the flexibility to choose records for matching from both the sides leveling the amount. In automatic process, system reconciles the pending records automatically.

  5. Select 'Reconcile' check boxes against respective records to reconcile the selected records.

 • Total Credits has to match with the Total Debits.

• Set of pending records on the same side of reconciliation has to be of the same nature.

• Set of pending records on the both sides of reconciliation has to be of different Account Nature.

  1. Click [Reconcile/Reconcile Automatically] to settle and merge the entries. System reconciles the details on clicking [Reconcile/Reconcile Automatically]. When 'Reconciliation Option' selected as 'Manual Reconciliation' system displays the legend as [Reconcile] and if 'Reconcile Option' is selected as 'Automatic Reconciliation by Document Reference' legend is displayed as [Reconcile Automatically].

  2. Click [Close] to exit the window.
