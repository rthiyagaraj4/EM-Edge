Print Deposit Register

From this function you can Print the Deposit Registers which are created.  

The Print Deposit Register function is used to print the details recorded using the Generate Deposit Register function. The details are printed for a specific Account Code, wherein you can choose the period when the deposit register is generated. System lists the date, remarks, document type and number, voucher number, current and cumulative credit and debit details recorded while generating the deposit register.

Print Deposit Registers

From the External Accounting System > Process - Deposit Register, double-click Print Deposit Register Function.

The Print Deposit Register screen gets displayed.

  1. Click [List of Values] to select a valid Account Code. The deposit register details are printed for the selected Account Code. This is a mandatory field.

  2. Click [List of Values] in the ‘To’ field to select the period when the deposit register was generated. The list displays the date, credit and debit balance details. On selecting the value, date gets displayed in the ‘To’ and ‘From’ fields. This is a mandatory field.

  3. Click [Run Process] to print the details. The deposit register details for the respective account code gets routed to the printer defined.
