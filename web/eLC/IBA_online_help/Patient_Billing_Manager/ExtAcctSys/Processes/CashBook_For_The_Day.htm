Cash Book for the Day

From this function you can Generate a report to view the Cash Book entries on a specific date in the facility.  

Cash Book for the Day function is used to generate a report on the cash collection transactions in the facility on a specific date. System lists the Receipt Date, Receipt Number, Receipt Amount, Total Receipt, Details of Payment to Bank, Account Code, Account Code wise amount, Total for the day, Total for month, Total amount for the year, Summary of accounts in the Cash Book generated for a specific date.

Generating Cash Book entries for a Day

From the External Accounting System > Processes, double-click Cash Book For The Day Function.

The Cash Book for the Day screen gets displayed.

  1. Click the Calendar function to select the date for which the report has to be generated. This is a mandatory field.

  2. The check box 'Exclude Credit Card Transactions' is selected, by default. You cannot clear the check box. The report generated does not include the Credit Card transaction details.

  3. Click [Run Process] to generate the report. A Report Option gets displayed with the preferences to View, Print and File options.

    Click ' File' to store the output into a file. Depending on the nature of the report, system prompts to select a file format. Choose the appropriate file format from the available list. You can also optionally enter the output file name.

    Click ‘Print’ to print the report on the designated printer. The report is routed to the default printer set to print the report output.

    Click ‘View’ to view the report on the screen. A note 'Process Successful. Click OK to view report' appears. Click [OK] to view report or [Cancel] to close the report viewer.

  4. Click [OK] to generate the report or [Cancel] to cancel the transactions.
