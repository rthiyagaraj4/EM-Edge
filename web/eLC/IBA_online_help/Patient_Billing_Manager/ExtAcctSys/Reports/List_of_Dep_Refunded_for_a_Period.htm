List of Deposits Refunded for a Period

From this function you can Generate a report on the deposits refunded during a specific period.  

List of Deposits Refunded for a Period function is used to generate a report on the deposits refunded to inpatients during a specific period. The report is generated only for the inpatients where the deposit amount is refundable. System generates a report listing the refund, deposit, bill and patient details.

Listing Deposits Refunded during a Period

From the External Accounting System > Reports, List of Deposits Refunded for a Period Function.

The List of Deposits Refunded for a Period screen gets displayed.

  1. Click [List of Values] to select a range of Patient ID’s. The report generated includes the selected Inpatient details.

  2. Click the Calendar function to select a date range. The deposit amounts refunded to the Inpatients during selected period are listed.

  3. Select a Refund Type from the list. The values are system defined – Cash, Cheque, Credit Card, Autopay, Others and All.

  4. Select a Order By value from the list based on which the report output has to be sorted. The values defined are – ‘Refund Date/Account’ and ‘Refund Date/Account’.

  5. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  6. You can view the report, print the details or save the output in a file in the system.
