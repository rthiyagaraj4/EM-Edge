Top-Up Deposit List for Current Inpatients

From this function you can Print a list of Inpatients who has to pay additional deposit amount to the facility.

The Top-Up Deposit List for Current Inpatients function enables you to view or print a list of current Inpatients who need top-up deposit. System generates a report listing the patient details, account number, ward and bed number, deposit details and remarks recorded during transactions.

Listing Inpatients with a request for top-up deposit

From the External Accounting System > Reports, Top-Up Deposit List for Current Inpatients Function.

The Top-Up Deposit List for Current Inpatients screen gets displayed.

  1. Click [List of Values] to select a range of Billing Groups. The values defined using the Billing Group function gets listed. The report generated includes the list of Inpatients with the selected Billing Group.

  2. Click [List of Values] to select a range of Billing Group Categories. The values defined using the Billing Group Categories function is listed. The report generated includes the list of Inpatients with the selected Billing Group Categories.

  3. Click [List of Values] to select a range of Nursing Units. The values defined using the Nursing Unit function in the Inpatient Management module gets listed. The report generated includes the list of Inpatients admitted in the selected Nursing Units.

  4. Click [List of Values] to select a range of Bed Classes. The Bed Class defined in the Inpatient Management module and associated with the relevant Facility is listed. The report generated includes the list of Inpatients with the selected Bed Classes.

  5. Click [List of Values] to select a range of Inpatient Admission numbers. The report generated includes the list of Inpatients within the range of Admission Numbers selected.

  6. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  7. You can view the report, print the details or save the output in a file in the system.
