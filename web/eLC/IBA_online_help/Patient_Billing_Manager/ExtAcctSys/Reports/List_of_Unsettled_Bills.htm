List of Unsettled Bills

From this function you can Generate a report on the bills which are not settled.

List of Unsettled Bills function is used to view or print the list of unsettled bills. The unsettled bills in the facility are tracked using this function for a particular period. The unsettled bills can represent the refund amount to be issued to the patient or the payer or the vice-versa. System generates a report listing the patient details, bill details, outstanding amount and the reminder amount.

Printing Unsettled Bills

From the External Accounting System > Reports, List of Unsettled Bills Function.

The List of Unsettled Bills screen gets displayed.

  1. Select a valid Episode Type from the list. The values are system defined – ‘All’, ‘Inpatient’, ‘Outpatient’, ‘Daycare’ and ‘Referral’. On selecting the Episode Type as ‘Inpatient’ additional fields called ‘Ward Code’ and ‘Bed Class’ gets displayed as shown in the screen.

  2. Select an Option from the list. The system-defined values are  ‘Hospital Receivable’, ‘Hospital Refundable’, and ‘Both’. On selecting 'Hospital Receivable', a report is generated for all the unsettled bills, which are to be paid by either the patient or the customers. Select 'Hospital Refundable', if a refund is pending to be issued to the patients or the payers. Select 'Both', if the report should contain details of the pending amount the hospital has to receive or vice-versa.

  3. Click the Calendar function to select the bill date range. The unsettled bills for the specific period are listed. This is a mandatory field.

  4. Click [List of Values] to select a range of the Billing Group Categories. The report generated includes the unsettled bills during the specific period. The Billing Group Categories defined in the Masters get listed.

  5. Click [List of Values] to select a range of wards. The report generated includes the unsettled bills associated to the selected wards. This field is available for selection only when the Episode Type is selected as ‘Inpatient’.

  6. Click [List of Values] to select a range of Bed Classes. The report generated includes the unsettled bills for the selected Bed Classes during the specified period. This field is available for selection only when the Bed Class is selected as ‘Inpatient’.

  7. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  8. You can view the report, print the details or save the output in a file in the system.
