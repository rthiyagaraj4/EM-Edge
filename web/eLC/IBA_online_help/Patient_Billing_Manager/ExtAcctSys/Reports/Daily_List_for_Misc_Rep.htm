Daily List for Miscellaneous Receipts

From this function you can Generate a report on the miscellaneous receipts generated in the facility.  

Daily List for Miscellaneous Receipts function is used to get an overview of the miscellaneous receipts generated in the Facility on a daily basis. The report is generated on a daily basis. System generates a report with the patient details, receipt details, Account Code, Narration and the Total Receipt Amount (Date-wise summary, Grand total).

Printing List of Miscellaneous Receipts

From the External Accounting System > Reports, Daily List for Miscellaneous Receipts Function.

The Daily List for Miscellaneous Receipts screen gets displayed.

  1. Click the Calendar function to select the receipt date range. The report generated lists the receipts generated during the specified period. This is a mandatory field.

  2. Select the 'Exclude Credit Card Transactions' check box, if the cash collected by processing the Credit Cards should be included as well. Clear the check box to exclude the credit card transactions from the report.

  3. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  4. You can view the report, print the details or save the output in a file in the system.
