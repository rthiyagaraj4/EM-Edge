Export GL Trans

From this function you can Export the Posting Account details to the General Ledger interface. You can View a report generated displaying the distribution details.

Export GL Trans function is used to export the account details to the General Ledger interface. The billing details are posted using the function ‘Post Billing Transactions’. You have to export the posted or the updated details to the General Ledger interface.

Export GL Trans

From the Period End, double-click Export GL Trans function.

The Export GL Trans screen gets displayed.

  1. Enter the month and the year by clicking the Calendar function. System displays the current processing month and year from the BL Parameters function. The transactions are exported until the specified month and year. If prior period posting is not allowed in General Ledger module, the period selected has to be greater than the GL Current Processing Period. This is a mandatory field. <Related Field Description>

  2. Click [Run Process] to export the data and to generate a report.

 

The Month and Year of ‘Doc Date for Summary J.V’ have to be same as the Transaction Period.

When the Patient Billing Manager module distribution details are transferred to the General Ledger module, the journal voucher number has to be the same as the miscellaneous receipt number generated by the financial accounting cashier in the Patient Billing Manager module.
