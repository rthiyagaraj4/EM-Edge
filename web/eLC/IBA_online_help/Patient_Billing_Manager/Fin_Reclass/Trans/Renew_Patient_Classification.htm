Renew Patient Classification

From this function you can Renew the financial details of a patient in the facility. You can Query to view the Classification Rule. You can Finalize the records.

Renew Patient Classification function enables an authorized user to renew the financial classification details of a patient. You can query to view the finalized patient classification records. The details are renewed for the number of months specified in the parameters beginning from the system date.

Renewing Patient Classification Details

From the Financial Reclassification > Transaction, double-click Renew Patient Classification Function.

The Renew Patient Classification screen gets displayed.

  1. Click [Execute Query] or press [F8] to query the existing records. <Related Field Description>

  2. Click [Authorize] to renew the financial classification details of a patient.

  3. A pop-up appears to enter the details of the User.

    1. User ID – Enter the User ID. The Users with the ‘Reclassification Allowed’ check box as selected in the ‘Users for Services’ function under the Administration function are authorized to finalize the details. This is a mandatory field.

    2. PIN No – Enter the PIN number of the respective User. This is a mandatory field.

    3. Remarks – Enter the Remarks in alphanumeric format. This is a mandatory field.

  4. Click [Commit Form] or press [F10] to save the details.

  5. A message 'Are you sure you want to Renew the Finance Classification Period for this Patient" is displayed. Click [Yes] to renew or [NO] to cancel the process.

  6. A confirmation message 'Classification period has been renewed' is displayed on clicking [Yes].
