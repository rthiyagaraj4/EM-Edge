List of Financial Classification Rules

From this function you can Generate a report to view the financial classification rules define in the facility.

List of Financial Classification Rules function is used to generate a report of the various classification rules defined in the facility. You can view the details for a selected Classification Type and Code.

Listing the Financial Classification Rules

From the Financial Reclassification > Reports, List of Financial Classification Rules Function.

The List of Financial Classification Rules screen gets displayed.

  1. Select the Classification Type from the list. The values are system-defined.

  2. Click [List of Values] to select a Classification Code. The classification codes associated to the Classification Type are listed.

  3. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  4. You can view the report, print the details or save the output in a file in the system.
