Summary Prepaid Report By Program

From this function you can Generate a summarized report for a prepaid program.

The Summary Prepaid Report By Program function is used to generate a summarized report for each prepaid program. The report generated displays the policy and protocols associated to the program, number of patients' using the protocol rule, total protocol amount, actual paid amount, invoice amount, charge amount and the posted amount.

Generating a Summarized Prepaid Report for a Program

From the Prepaid Management > Reports, double-click on Summary Prepaid Report By Program function.

The Summary Prepaid Report By Program screen gets displayed.

  1. Click [List of Values] to select a Prepaid Program. The description of the selected Prepaid Program is displayed. You can leave the Prepaid Program as blank to generate a report for all records.

  2. Click the Calendar function to select the Date range. Enter the visit or admission date and the checkout or discharge date. You can leave the date range as blank to generate a report irrespective of the date range.

  3. Click [Run Process] to generate a report.

  4. Report Option is displayed with the View, Print and File Preferences on clicking [Yes]. Click [OK] to generate or [Cancel] to abort the process.

    Select File to save the output in a file in the local system.

    Select Print to print the report.

    Select View to view the online report.
