Generate Invoice - OP/EM

From this function you can Generate the invoice for Outpatients or the Emergency patients.

The Generate Invoice – OP/EM function enables you to generate invoice for the outstanding bills of the Outpatients or the Emergency patients who are associated to the selected Customer. You can the select the Customer Specific and Facility Specific parameters to dispatch the invoices to the Customers. A bill is printed during invoice generation listing the outstanding payment, if the check box ‘Bill Printing Required’ is selected in ‘Invoice Process Parameters’ function. Invoice posting is mandatory, if the check box ‘GL Interface Distribution required for Outpatients/Emergency patients from Acct Month & Year' and ‘GL Interface Distribution required for External patients from Acct Month & Year’ is selected under the 'Outpatient/Emergency/External' tab in ‘BL Parameters’ function. Invoice is posted to respective GL accounts to process month end. Bills, Receipts and Refund documents are posted only if the Cash Control receipts are generated for the Cashier transactions. A Billing Clerk or Cashier can record Diagnosis Code in the preferred language, if the details are recorded in the local language through the 'Translation Manager' function in System Manager module. The report printed after invoice generation will print the Diagnosis Code and description in the preferred language. A single invoice is generated for multiple bills that are grouped under the same Policy Group and associated to a Payer.

 

The accounts which are credited and debited during bill posting are mentioned below.

=> Bill Posting

The billing receivable account for the respective bill type is debited.

Income accounts for a Billing Service code is credited with the gross amount.

Discount accounts for a Billing Group is debited with the discount amount.

 

=> Receipt Posting

The billing receivable account for the respective bill type is credited when a receipt raised against a bill is posted.

The account defined for the cash counter and the settlement type is debited.

 

=> Refund Posting

The billing receivable account for the respective bill type is debited when a refund raised against a bill is posted.

The account defined for the cash counter and the settlement type is credited.

 

=> Other Receipt Posting

The account defined for the cash counter and the settlement type is debited.

The  account defined against the receipt type is credited.

 

=> Other Refund Posting

The account defined for the cash counter and the settlement type is credited.

The  account defined against the receipt type is debited.

 

=> Cash Counter Receipt

The account defined in the billing parameters will be debited when cash counter receipt transaction is posted.

The account defined for the cash counter and settlement type is credited.

Generating Invoice for Outpatients or Emergency patients

From the Invoice Management > Transaction, double-click on Generate Invoice – OP/EM function.

The Generate Invoice – OP/EM screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [Commit Form] to save the records or [Clear Record] to delete the data entered.
