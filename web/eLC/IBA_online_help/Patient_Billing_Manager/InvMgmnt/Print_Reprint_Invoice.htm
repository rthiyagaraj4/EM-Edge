Print/Reprint Invoice

From this function you can Print or Reprint the invoice, debit notes or the credit notes generated in the facility.

The Print/Reprint Invoice function enables you to generate, view, print or save invoices, debit or credit note reports. You can view the amount against the services utilised and the total amount for which the report is generated. You have an option to select the Payer and the Payer Group to generate the report. The document date range and the invoice number can be selected to generate reports for specific invoice documents generated on a specific date. You can print an original invoice, debit or credit note document or you can reprint a copy of the printed document. The relative weight details, discharge type and the discharge status is displayed in the report generated.

Printing Invoice, Debit or Credit Notes

From Invoice Management > Reports, double-click on Print/Reprint Invoice function.

The Print/Reprint Invoice screen gets displayed.

  1. Select the Transaction Type from the list. The system-defined values are 'Invoice', Debit Note', 'Credit Note' and 'ALL'. 'ALL' is selected, by default. <Related Field Description>

  2. Click [List of Values] to select the Invoice, Debit or the Credit Note document type and number range. You can generate the report for the selected range of values.

  3. Select the 'Reprint' status from the list. The system-defined values are 'As per Status', 'As Copy' and 'As Original'.

  4. Click [Run Process] to either view, print or save the report output in a file.
