Re-print Receipt for Payment Reference Doc

From this function you can Re-print a summarized or a detailed report for the payment received.

The Re-print Receipt for Payment Reference Doc function enables you to re-print receipt for the payments received. You can choose to re-print a receipt for an individual invoice or a consolidated invoice. The payment reference documents, which are not cancelled are only listed for re-printing the receipts. You can choose to re-print all the receipts associated to the payment reference doc or a specific receipt.

Re-printing the receipts

From Invoice Management > Reports, double-click on Re-print Receipt for Payment Reference Doc function.

The Re-print Receipt for Payment Reference Doc screen gets displayed.

  1. Click [Execute Query] or press [F8] to query all the records. You can use the upward and the downward arrow keys in the keyboard to navigate to the previous and the next record.

OR

  1. Click [Enter Query] and enter the details in a specific text box to search for the matching records. Press [F8] or click [Execute Query] to query the specific record.

    Enter the following details to query and re-print the receipts for the selected payment reference document:

    1. Payment Reference Doc - The payment reference document number is displayed on clicking [Execute Query]. Enter the Payment Reference Document number in alphanumeric format of 30 characters maximum length to search for a specific record.

    2. Pmnt Ref Date - The payment reference date is displayed. Click the Calendar function to select the Payment Reference Date and search for a specific record.

    3. Receipt Amount - The amount settled for the services during receipt generation is displayed. Enter the receipt amount to search for a specific record.

    4. Receipt Option - The receipt option is displayed based on the value selected during receipt generation. Select the value as 'Invoice-wise' or 'Consolidated by Customer' to search for a specific record.

    5. Payer Name - The Payer Name recorded during receipt generation is displayed. Enter the payer name in alphanumeric format of 100 characters maximum length to search for a specific record.

  2. The receipts generated are displayed based on the query criteria. The receipt type and number, receipt date and the amount against which the receipt is generated are displayed.

  3. The payment made against the invoices generated are displayed. The document type and number, amount settled, the date when the settled amount is recorded, name of the payer, policy type and the patient ID is displayed along with the episode and the visit ID.

  4. Select the 'Select All' check box to re-print the listed receipts. Select the check box for a specific record to re-print the receipt.

  5. Click [Reprint Receipt] to re-print the selected receipts in the summarized form.

  6. Click [Reprint Receipt in Detail] to re-print the selected receipts in a detailed form.
