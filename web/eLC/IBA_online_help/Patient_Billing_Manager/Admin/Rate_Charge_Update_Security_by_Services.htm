Rate/Charge Update Security by Services

From this function you can Associate services to a User Group. The users associated to a User Group can access the services attached to a User Group. You can View the services associated to a User Group.

Rate/Charge Update Security By Services function is used to associate billing services to a User Group. A User Group is set of Users who has access to the billing services. The Users in a User Group can access the charge based services to perform transactions in the Patient Billing Manager module.  

Defining Users for Rate/Charge Update Security by Services

From the Masters > Administration, click User Security, double-click on Rate/Charge Update Security By Services.

The Rate/Charge Update Security By Services screen gets displayed.

  1. Enter the details in the mandatory (*) fields found in the following Tab pages.

Maintain Services for Rate/Charge Update

Maintain Services for Rate/Charge Update

  1. User Group – Click [List of Values] to select a User Group. The description for the selected User Group gets displayed. This is a mandatory field.

Billing Services

  1. Billing Service – The Billing Services associated to the User Group using the ‘Create Services for Rate/Charge Update’ tab are listed. You can associate additional Billing Services to the User.

  2. Description – The Description for the selected Billing Service gets displayed.

Create Services for Rate/Charge Update

Create Services for Rate/Charge Update

  1. User Group - Click [List of Values] to select a User Group. The description for the selected User Group gets displayed. This is a mandatory field.

  2. Billing Service [From–To] – Click [List of Values] to select a range of Billing Service. The User Group is authorized to access the Billing Services associated.

  3. Analysis Grp1 [From–To] – Enter the range of Analysis Group.

  4. Analysis Grp2 [From–To] – Enter the range of Analysis Group.

  1. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records within the facility.
