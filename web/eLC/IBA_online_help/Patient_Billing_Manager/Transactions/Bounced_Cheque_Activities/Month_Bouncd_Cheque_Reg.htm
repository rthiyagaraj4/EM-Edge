Monthly Bounced Cheque Register

From this function you can Generate a report on the cheques, which have bounced back. You can choose to generate a report for all the cheques, which are bounced back and are replaced in the same month or are not replaced.  

Monthly Bounced Cheque Register function is used to the list the bounced cheque details. You have an option to select the cheques, which are replaced in the same month the cheque has bounced or are not replaced. You can also select to view the report on the bounced, which are not replaced.

Monthly Bounced Cheque Register

From the Transactions > Bounced Cheque Activities, double-click Monthly Bounced Cheque Register Function.

The Monthly Bounced Cheque Register screen gets displayed.

  1. Click Calendar function to select the month in a year. The cheques that have been replaced or not replaced in the selected month or any month get displayed in the report based on report option selected.

  2. Select ‘Replaced/Not Replaced in the Same Month’ to view the cheques, which are either replaced or not replaced in the selected month.

  3. Select ‘Replaced/Not Replaced in Any Month’ to view the cheques, which are either replaced or not replaced in any month.

  4. Click [Run Process] to view, print or save the report output in the local machine.

  5. A log file is generated. Click [OK] to view the report or [Cancel] to exit from the screen.
