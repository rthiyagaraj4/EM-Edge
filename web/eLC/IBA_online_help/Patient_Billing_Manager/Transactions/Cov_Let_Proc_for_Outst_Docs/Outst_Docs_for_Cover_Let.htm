Outstanding Documents For Covering Letter

From this function you can Generate a report to view the documents, where the bill amount is not settled and the reminder letters are not dispatched.

Outstanding Documents For Covering Letter function is used to list out the patient details, where the bill amount is not settled and the reminder letters are not dispatched. A report is generated including the outstanding document details until the date specified.

Outstanding Documents For Covering Letter

From the Transactions > Covering Letter Process for Outst Doc, double-click Outstanding Documents For Covering Letter function.

The Outstanding Documents For Covering Letter screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run Process] to generate a report or [Clear Record] to clear the entered data.

  3. Click View, Print or File to view the report, print or save the output in a file respectively.

  4. A log file is generated. Click [OK] to view the log file or [Cancel] to exit from the screen.
