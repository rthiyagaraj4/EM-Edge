Intercompany Charging Rule

From this function you can Define the charging rule for interfacility dependencies. You can Query to view the existing charging rules.

Intercompany Charging Rule function is used to define interfacility dependencies. You can select the Billing Group, Customer Group,  Customer combination and the facility to which they can be referred.

Defining Intercompany Charging Rule

From the Transactions > Intercompany Charging, double-click Intercompany Charging Rule Function.

The Intercompany Charging Rule screen gets displayed.

  1. Select a Facility from the list. The facilities defined through the 'Facility' function in the Enterprise Configuration module are listed. The patients are referred from the selected facility.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Commit Form] or press [F10] to record the details or [Clear Record] to clear the data.

  4. Click [Execute Query] or press  [F8] to query for the existing records.
