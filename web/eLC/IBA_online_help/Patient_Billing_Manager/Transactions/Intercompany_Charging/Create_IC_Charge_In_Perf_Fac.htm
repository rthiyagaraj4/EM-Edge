Create Intercompany Charges in Performing Facility

From this function you can Define the interfacility charging rule. You can Query to view the existing charging rules.

Create Intercompany Charges in Performing Facility function is used to create intercompany charges in the performing facility as per the Intercompany charging rule. System creates charge in the performing facility at the rates based on the Intercompany charging Rule defined.

Creating Intercompany Charges in Performing Facility

From the Transactions > Intercompany Charging, double-click Create Intercompany Charges in Performing Location Function.

The Create Intercompany Charges in Performing Facility screen gets displayed.

  1. Select the Referred Facility from the list. The facilities defined through the 'Facility' function in the System Manager module are listed. The Referred Facility defined in 'Interfacility Charging Rule' are displayed. The Customer Code and Customer Group are displayed based on the Facility selected and the values associated in 'Interfacility Charging Rule' function.

  2. Click Patient Encounters tab to view the patient encounter details.

  3. Enter the details to search for a specific patient record. Click [Search] to query the patient record or [Clear] to clear the details.

    Enter the following details:

    1. Episode Type - Select an Episode Type from the list. The system-defined values are Inpatient, Daycare, Outpatient and Emergency.

    2. Patient ID - Enter the Patient ID in alphanumeric format of 10 digits maximum length to search for a specific record.

    3. Admission/Tfr. No - The admission or transfer number is displayed on selecting the Patient ID.

    4. Encounter Date [From - To] - Click the Calendar function to select the Encounter Date. The Encounter From Date is defaulted to a week before the current system date.

  4. The patient records matching the search criteria are displayed. Double-click on a specific patient record to view intercompany charges. The details are listed under View Intercompany Charges tab.

    Patient Encounters

    1. Select All/Select - Select the check box to process a single/all patient records.

    2. Patient ID - The Patient ID whose Intercompany charges has to be created in the referred  Facility is displayed.

    3. Patient Name - The name of the patient whose Intercompany charges has to be created in the referred  Facility is displayed.

    4. Episode Type - The Episode Type is displayed as either ‘I’ – Inpatient, ‘O’ - Outpatient, ‘E’ –Emergency and ‘D’ - Daycare.

    5. Encounter ID - The Encounter ID of the patient whose Intercompany charges has to be created in the referred  Facility is displayed.

    6. Location - The Location of the patient whose Intercompany charges need to be created in the Referral Facility is displayed.

    7. Description - The Location Description of the patient is displayed.

    8. Encounter Date - The Encounter Date and Time of the patient whose Intercompany charges has to be created in the referred  Facility is displayed.

    9. Net Charge Amt - The Net Charge Amount of the patient whose Intercompany charges has to be created in the referred  Facility is displayed.

    10. [Process] - Click [Process] to create the Intercompany charges for the Selected patient in the referred Facility.

  5. Enter the details to search for a specific patient record. Click [Search] to query the patient record or [Clear] to clear the details. The details are listed under View Intercompany Charges tab.

    The following details are displayed:

    1. Charge Date/Time - The Charge Date/Time of the Intercompany charges for a Billing Service/Panel/Package is displayed.

    2. Doc Reference - The bill document Reference Number is displayed.  

    3. Service/Panel Ind - Service or Panel is displayed based on whether the patient has availed a Billing Service or Billing Panel.

    4. Panel Code  -  The Panel Code is displayed, if the patient has used a Billing Panel.

    5. Quantity - The quantity of Billing Service/Panel availed is displayed.

    6. UOM - The UOM is displayed based on the Billing Service/Panel availed.

    7. Total Payable - The Total payable amount for the intercompany service is displayed.

    8. Patient Payable - The Patient payable amount for the intercompany services utilised is displayed.  

    9. Patient Paid - The patient paid amount for the services availed is displayed.  

    10. Completed - The check box is selected if the order is complete.

    11. Billed - The check box is selected if the order is billed.

    12. Description  - The description of  the Billing Service/Panel/Package is displayed.

    13. Total Charges - The total payable, patient payable and patient paid amount is displayed.

    14. Episode Type - The Episode Type of the selected patient is displayed.

    15. Customer Group - The Customer group as set in the intercompany charging rule for the referred Facility is displayed.

    16. Billing Group - The Billing Group as set in the intercompany charging rule for the Referred Facility is displayed.

    17. Customer Code - The Customer Code as set in the intercompany charging rule for the Referred Facility is displayed.

    Enter the following details:

    1. Episode Type - Select an Episode Type from the list. The system-defined values are Inpatient, Daycare, Outpatient and Emergency.

    2. Patient ID - Enter the Patient ID in alphanumeric format of 10 digits maximum length to search for a specific record.

    3. Admission/Tfr. No - The admission or transfer number is displayed on selecting the Patient ID.

    4. Encounter Date [From - To] - Click the Calendar function to select the Encounter Date. The Encounter From Date is defaulted to a week before the current system date.
