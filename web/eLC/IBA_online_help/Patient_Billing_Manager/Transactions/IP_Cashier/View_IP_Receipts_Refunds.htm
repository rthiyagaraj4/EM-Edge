View IP Receipts / Refunds

From this function you can View a relevant refund document generated for an inpatient transaction.

View IP Receipts/Refunds function is used to view the receipts/refund documents generated for an inpatient transaction in the Facility. User can view the patient details and cashier details along with the document number generated for the IP receipts or a refund issued.

View IP Receipts / Refunds

From the Transactions > IP Cashier, double-click View IP Receipts /Refunds Function.

The View IP Receipts /Refunds screen gets displayed.

  1. Click [Enter Query] and enter the Patient Number; press [F8] or [Execute Query] to view the details of relevant patients at the Facility.

OR

  1. Press [F8] or [Execute Query] to view the details of all the outpatients.  

  2. The details recorded using the function Enter IP Receipts/Refunds  get displayed.
