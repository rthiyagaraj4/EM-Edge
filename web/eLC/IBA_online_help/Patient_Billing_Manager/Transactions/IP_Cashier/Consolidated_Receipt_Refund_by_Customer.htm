Consolidated Receipt by Customer

From this function you can Query and settle outstanding bills for a payer and generate a consolidate receipt for all the bills.

The Consolidated Receipt/Refund by Customer function is used to search all outstanding bills generated for multiple patients against a specific payer/customer for a defined period.

Querying to view outstanding bills

From the Transactions > IP Cashier, double-click Consolidated Receipts / Refunds by Customer function.

The Consolidated Receipt/Refund by Customer gets displayed.

  1. Enter the details in the mandatory (*) fields to query the booking details. <Related Field Description>

  2. Click [Execute Query] or press [F8] to display the following tabs:

Outstanding Details

The Following details are displayed:

 

  1. Receipt Nature - The Receipt Nature associated to the relevant Receipt Type. You cannot edit the data displayed in this field.

  2. Doc Type – The document type of the receipt based on the document type set for ‘Refund Cash Book Doc’ in the ‘Setup External Account  Parameters’ function under the ‘Set Up’ menu. You cannot edit the data displayed in this field.

  3. Doc Num - The document number generated for the outstanding bill. You cannot edit the data displayed in this field.

  4. Doc Date - The date when the bill was generated for the services utilized by the patient. You cannot edit the data displayed in this field.

  5. Episode No – The Episode Number of the patient. You cannot edit the data displayed here.

  6. Outst Amount – The patient’s outstanding amount gets displayed. You cannot edit the data displayed in this field.

  7. Receipt Nature Description – The description of the selected receipt nature gets displayed. You cannot edit the data displayed in this field.

  8. Receipt Type Description - The description of the selected receipt type gets displayed. You cannot edit the data displayed in this field.

  9. Select – Select this check box to select the bill to be settled..

  10. Settled Amount – Enter the settlement amount to settle the selected outstanding bills.

  11. Rec Type - Click [List of Values] to select a Receipt Type. The receipt types defined in the billing masters are listed.

  12. Acct Code - Click [List of Values] to select an Account Code. The values defined under the ‘External Account Codes’ tab get listed. This is a mandatory field.

  13. Acct Dept - Click [List of Values] to select an account department code.

Settlement Details

The following details are displayed:

 

  1. Total Outstanding Amount – The outstanding amount total gets displayed.

  2. Cash In Hand – Enter the cash available in hand in a numeric format.

  3. Total Amount Settled – Enter the total amount settled by patient.

  4. Settlement Type – Click [List of Values] to select settlement types. The settlement types are ‘Cash’, Cheque and Credit/Debit card.

  5. Cheque/Credit Card No – Enter the cheque/card number, if the settlement type is Cheque/Card.

  6. Date – Enter the date of settlement.

  7. Bank/Card Name – Enter bank or card number, if the settlement type is cheque or card.

  8. Branch Name – Enter the bank branch name, if the settlement type is cheque or card.

  9. Online Approval – Select the check box for online approval for the settlement types.

  10. Batch No – Enter the batch number for outstanding bill stellement.

  11. Approval Code – Enter the Approval Code for settlement, if the online approval is completed.

  12. Sale Draft No – Enter the sale draft number in the alphanumeric format.

  13. Payer’s Name – Enter the patient name

  14. Address – Click [Address] to add the address of the patient.

  15. Payer’s Ref ID – Enter the reference ID of the patient.

  16. Rcvd Through Mail – Select this check box when the reference ID received through email.

  17. Remarks – Enter the remarks about the settlement.

  18. Document Type/No – The type and number of document the get displayed. You cannot edit the data.

  19. Document Date – The document date gets displayed. You cannot edit the data.

  20. Cash Counter – The cash counter number gets displayed. You cannot edit the data.

  21. Post Mth/Yr – The month and year of the document posted get displayed. You cannot edit the data.

  22. Patient Remarks – The patient remarks gets displayed. You cannot edit the data.
