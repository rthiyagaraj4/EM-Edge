Cancel IP Refunds

From this function you can Cancel a relevant refund document generated for an inpatient/daycare patient transaction. A Cashier can cancel only the printed refund documents.

Cancel IP Refunds function is used to cancel the refund documents generated for an inpatient/daycare patient transaction in the Facility. An authorized Cashier has access to cancel the printed documents. The ‘Refunds’ option has to be checked in the function ‘Users for Authorization / Refund / Cash Counters’ listed in the Administration menu for a cashier to login and cancel the refund documents. Cashier can cancel the receipts for the transactions on a relevant day. To cancel the previous day’s transaction, the check box ‘Allow Cancel of Previous Days Cashier Txns’ has to be checked under the tab ‘Receipts/Refund Controls’ in ‘BL Parameters’ function.

Cancel IP Refunds

From the Transactions > IP Cashier, double-click Cancel IP Refunds Function.

The Cancel IP Refunds screen gets displayed.

  1. Click [Enter Query] and enter the Patient Number; press [F8] or [Execute Query] to cancel the refund document for a selected patient. Press [F8] or [Execute Query] to view the details of all the inpatients.  

  2. The receipt details generated get displayed on clicking [Execute Query].

  3. Select Cancel to cancel the refunds. The Cancellation Authorization screen pops-up to validate the user privilege to cancel the document.

  4. Enter the User Id, Pin Number and Cancel Reason. Select a value from the list. The reasons defined using the function ‘Reasons for Billing Actions’ with the Refund Cancellation check box as selected are listed.  

  5. Enter a valid reason for cancellation. During cancel, the updates are just the reverse of that of the receipt update process.

  6. Click [Commit Form] to cancel the receipts or [Clear Record] to clear the data.

                                                                        

You have to login to the Cash Counter to cancel any refund document.
