Enter Request – IP

From this function you can Associate the billing services to an Inpatient or a Daycare patient admitted in the facility.

Enter Request – IP function is used to associate billing services to an Inpatient or a Daycare patient admitted in the facility. A Transaction Reference ID is generated for each service associated to the patient. An authorized user has to associate the services to an Inpatient or a Daycare patient. The services defined with the ‘Direct Entry Allowed’ check box selected are listed. The authorized user has to choose a service from the list. The check box ‘Enter Service Request’ has to be selected for the logged in User in the function ‘Users for Services’ under the Administration menu to associate the services to the patients in the facility. The service price is based on the Multipricing factor, if the service is rate based. If the service is charge based, the charge defined is calculated. The charges for the services associated are calculated automatically and displayed. You cannot update the charges for the rate based services. The total charge is computed and displayed. The Services with the ‘Autoconfirm’ check box selected during definition are confirmed automatically. If the check box is cleared during definition, the services have to be confirmed manually using the function ‘Confirm Inpatient Service Requests’.

Associating a Service

From the Transactions > IP Direct Charges, double-click Enter Request – IP Function.

The Enter Requests – IP screen gets displayed.

  1. Select an Episode Type from the list. The system defined values are Outpatient, Emergency and External. Select Outpatient, Emergency and External patient to associate services to an Outpatient, Emergency or External patient admitted to the facility.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

OR

  1. Right click on Right Click for User Options to search for the relevant patient record. You can search for a patient record based on the Patients Lookup, Patient/Admission/Transfer lookup or Admission/Transfer lookup.

  2. Select the check box View Billing to view the billing details.

    The financial details of the patient for the selected episode get displayed. You can also view the unconfirmed services associated to the patient by clicking [Unconfirmed Services]. The Settlement Details, Billing Group details, service charges, gross amount for the services utilized, adjustments or refunds issued, unbilled and outstanding amount gets displayed. The name of the user, who has recorded or modified the financial details gets displayed.

  3. Click [Future Order] to view any orders placed for future admissions. If any orders are placed for future visits using the external modules, you can view the orders and the details by clicking [Future Order].

  4. Associate the Billing Services or a Billing Panel to the patient. The services or the panels are charged based on the price defined for the Billing Service. <Related Field Description>

  5. Click [Commit Form] to associate the service or [Clear Record] to clear the entered data.

  6. A Transaction Reference ID is generated on successfully associating a service or a panel.
