Cancel Receipts

From this function you can Cancel a relevant receipt generated for an outpatient transaction. The receipts can be either for the payments accepted or for the refunds issued to the outpatients. Cashier has to cancel only the printed receipts.      

Cancel Receipts function is used to cancel the receipts generated for a transaction at the Facility. Cashier can cancel only the printed receipts. The authorized users only can cancel the receipts. The ‘Receipts’ option has to be checked in the function ‘Users for Authorization / Refund / Cash Counters’ in the Administration menu for the cashier to login and cancel the receipts. A cashier can cancel the receipts printed for the transactions on a relevant day. To cancel the previous day’s transaction, check the option ‘Allow Cancel of Previous Days Cashier Txns’ in the ‘BL Parameters’ under the tab ‘Receipts/Refund Controls’. You can cancel the bills associated to the receipts using this function. If there are multiple bills associated to the same receipt, you can select to cancel individual bills. Current receipt is highlighted using color. Current encounter is highlighted using color and other payers associated to the current receipt are highlighted using color.

Cancel Receipts

From the Transactions > OP Cashier, double-click Cancel Receipts Function.

The Cancel Receipts screen gets displayed.

  1. Click [Enter Query] and enter the Patient Number; press [F8] or [Execute Query] to cancel the receipt for a selected patient. Else, press [F8] or [Execute Query] to view the details of all the Outpatients.  

  2. The receipt details generated get displayed on clicking [Execute Query].

  3. Select Cancel to cancel the receipts. The Cancellation Authorization screen pops-up to validate the user privilege to cancel the document.

  4. Enter the User Id, Pin Number and Cancel Reason. Select a value from the list. The reasons defined using the function ‘Reasons for Billing Actions’ with the Receipt Cancellation check box as selected are listed.  

  5. Enter a valid reason for cancellation. During cancel, the updates are just the reverse of that of the receipts updates.

  6. Right-Click on the Bill Number to cancel the respective bill. On right-clicking, the ‘Nullify/Cancel a Bill’ screen gets displayed with the bill number and the type displayed and the ‘Cancel’ check box as selected. Click [Commit Form] to cancel the bill. A warning message ‘Do you want to confirm?’ gets displayed. Click [YES] to cancel or [NO] to abort the transaction. If the bill is associated to another receipt, an error message gets displayed and you cannot cancel the bill. If the bill is associated to another payer, an error message ‘Bill not valid for modification or cancellation’ is displayed.

If the bill is already posted, you cannot cancel the bill. A message ‘Cannot Cancel the Posted Bill’ is displayed.
