Print Receipts

From this function you can Print receipts or an acknowledgement letter for the payments accepted from an outpatient. You cannot print the receipts, which are cancelled.     

Print Receipts function is used to print the receipts in a batch mode. The active receipts against which the payment is accepted are printed. Authorized users only can print the receipts. The Lab Charges are printed under the same group while the drug charges are categorized as essential and non-essential drug groups.  The ‘Reprint Receipt’ option has to be checked in the function ‘Users for Authorization / Refund / Cash Counters’ in the Administration menu for the user to login and print the receipts. The Deposit Reference number is generated along with the receipt details. The receipt also displays the patient payable and the payer payable amount separately in the report. The reimbursable and the non-reimbursable amount gets displayed separately in the report generated. A discount report is also printed along with the bill details, if the patient has availed a discount for the services utilized from the facility.

Print Receipts

From the Transactions > OP Cashier, double-click Print Receipts Function.

The Print Receipts screen gets displayed.

  1. Select an Episode Type for transaction. The values are system defined - Outpatient, Emergency and External.

  2. Click [List of Values] to select a range of document types and numbers. The receipts are printed for the selected document range.

  3. Click Calendar function to select the date range. The documents generated between the selected date range are printed.

  4. Select a Print Option from the list. The values are system defined – ‘Receipts’, ‘Acknowledgment Letter’ and ‘Both’.

  5. Select the Detail Receipt check box to print a detailed report for the selected parameters. Clear the check box to print a summarized report. If the check box ‘Print Legends for Detailed Receipt’ is selected under the ‘Receipt Format’ tab in the ‘Rules for Detailed Bill’ function, the legends are printed in the detailed report.

  6. Select a Group By option from the list to sort the report while printing. The values are system defined – ‘As per Setup’, Service Classification’ and ‘No Grouping’.

  7. Click [Process Run] to print the receipts or [Clear Record] to clear the data.
