Enter OP Receipts/Refunds

From this function you can Generate the receipts for the deposit amount accepted from an outpatient during visit registration. A transaction reference id is generated for every transaction associated to a relevant patient.  

Enter OP Receipts/Refunds function is used to accept a deposit amount from the patient and generate or print a receipt for the transaction. Cashier in the Cash Counter can also issue a refund and generate/print a receipt for every transaction in the Cash Counter. A reference number is generated for every transaction that is committed.

An Acknowledgement Letter is printed for the medical report requests placed to medical board. A request for medical report is placed using the 'Medical Board' function in the Medical Records module. the requestor details has to be recorded along with the payment details. On committing the record with the requestor and payment details, a request number is generated. The request number gets populated in the 'Request ID' field, only if the 'Nature' is selected as 'MI' and the 'Receipt Type' is selected as 'MRR'. The Acknowledgement Letter gets printed, if the check box 'Through Mail' is selected.  

You can issue refunds and generate receipts for the respective transaction through this function. If the orders placed through 'Requests and Results' module are discontinued, the charges that are posted has to be issued as a refund to the patient. The bill/receipt has to be cancelled, before the logged-in user cancels the order when the shift is not closed.

Enter OP Receipts/Refunds

From the Transactions > OP Cashier, double-click Enter OP Receipts / Refunds Function.

The Enter OP Receipts / Refunds screen gets displayed.

  1. Enter a Patient ID or select the Patient ID using the Lookup function.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Commit Form] to print the bills/receipts or [Clear Record] to clear the data.

 

Cashier has to log into the Cash Counter to accept deposits or issue refund using this function.
