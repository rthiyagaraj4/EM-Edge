Generate Reminders

From this function you can Generate reminder letters to be dispatched to the patients/customers owing amount to the facility for the services availed. You can Query to view the unsettled bill amounts for a Billing Group Category by selecting an outstanding amount range and the outstanding period. You can Cancel the Reminder Letters generated for a Billing Group Category.

Generate Reminders function is used to view and generate reminder letters. The document types, where the amount is not settled for the services are displayed along with the patient number and the outstanding amount. You have to select the documents and click [Generate] to create a reminder letter for a respective Billing Group Category. You have an option to select a reminder letter based on the outstanding amount, billing group category and the number of days for which the bill amount is not settled.  

Generate Reminders

From the Transactions > Reminder Letter Process, double-click Generate Reminders function.

The Generate Reminders screen gets displayed.

  1. Select a Billing Group Category. The values defined using the Billing Group Category function get listed. This is a mandatory field.

  2. Select an Outstanding Amount range. The values defined using the Reminder Letter Setup function and associated to the selected Billing group Category get listed. This is a mandatory field.

  3. Select a Reminder Number from the list. The Reminder Number associated to the selected Billing Group Category and Outstanding Amount range get displayed. This is a mandatory field.

  4. The outstanding document details for the selected Billing Group Category get displayed under the following tabs:

Create

The Create tab enables you to select the documents against which the Reminder Letters are to be created for dispatch.

  1. Select All - Select the check box to select all the transactions displayed. The reminder letter is created for the selected outstanding documents. Clear the check box and the records are not selected for reminder letter generation. The check box is selected, by default.

  2. Document No/Date – The Document Number is displayed for the outstanding document with the selected Billing Group Category. The Document Type for the selected Document Number gets displayed. The date when the Bill document date generated gets displayed.

  3. Billing Group – The Billing Group for the selected transaction gets displayed.

  4. Outstanding Amount –The Bill outstanding Amount associated to the Document Type and the respective Document Number get displayed.

  5. Reminder No – The Reminder Number associated to the selected Billing Group Category gets displayed.

  6. Reminder Format – The Reminder Letter associated to the selected Billing Group Category gets displayed.

  7. Patient No – The Patient ID for the outstanding documents get displayed.

  8. Bill/DCP – System displays either ‘B’ or ‘D’ to denote whether the outstanding amount is generated from a Bill or a Deferred Cash Patient.

Encounter Details

  1. Patient Name – The Patient name for the selected patient ID gets displayed.

  2. Episode Type – The Episode type like Inpatient, Daycare, Outpatient, Emergency or External get displayed.

  3. Episode/Visit No – The Episode and the Visit Number associated to the selected Patient get displayed.

Last Delivered Reminder Dispatch Details

  1. Doc No – The system displays the last dispatch document type code and number (If any). The Document Number indicates the details of the reminder letter already dispatched to the patient for the selected Billing Group.

  2. Dispatched Date – The system displays the date when the reminder letter was dispatched.

Descriptions

  1. Billing Group – System displays the Billing Group associated to the patient ID.  

  2. Reminder Format – The description for the selected Reminder Letter gets displayed.

  3. Generate – Click [Generate] to generate the Reminder Letter for the selected outstanding documents. You have to select at least a single record to generate a Reminder Letter.

Cancel

The Cancel tab enables you to de-select or Cancel the bills, which are selected for generating the reminder letter process. The records against which the Reminder Letters are generated get displayed. Select the relevant records and click [Cancel] to cancel the Reminder Letters. You have to select at least a single record to generate a Reminder Letter.

  1. Select the records and click [Commit Form] or [Generate] to generate a Reminder Letter for a Billing Group Category.
