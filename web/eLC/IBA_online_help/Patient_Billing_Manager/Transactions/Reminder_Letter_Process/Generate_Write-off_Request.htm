Generate Write-off Request

From this function you can Cancel the bills, which are not settled by a patient/customer after dispatching reminder letters. Authorized billing personnel have to authorize the write-off request.

Generate Write-Off Request function is used to clear the billing details, which are not settled. Authorized billing personnel has to write-off the billing details, if the patient/customer has not settled the outstanding amount for the services utilized from the facility, after dispatching multiple reminder letters. The User with the authorization to approve the Write-off Request is permitted to login and approve the requests. The check box ‘Write-off Approval’ has to be selected in the function ‘Users for Authorization/Refund/Cash Counters’ in the Administration menu. A document reference number is generated on selecting the records. A screen pops-up requesting confirmation to generate the write for the number of records selected.

Generating Write-Off Request

From the Transactions > Reminder Letter Process, double-click Generate Write-Off Request function.

The Generate Write-Off Request screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Authorize] to login and to approve the write-off process for the outstanding documents. The documents for which the outstanding payments are not settled after dispatching Reminder Letters get listed.  

  3. Select the records and click [Process] to generate a write-off request or click [Clear Record] to clear the entered data.

  4. A confirmation screen appears displaying the number of records selected for approval. Click [Yes] to approve or [No] to cancel and to exit from the screen.
