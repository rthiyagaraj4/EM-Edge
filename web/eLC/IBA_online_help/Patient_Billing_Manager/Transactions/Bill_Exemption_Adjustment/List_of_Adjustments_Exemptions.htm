List of Adjustments/Exemptions

From this function you can Generate a report to view the bill adjustments or exemptions in the Facility for a specific month/year. The Action reasons defined in the Administration menu are listed.

List of Adjustments/Exemptions function is used to view, print or save the adjustment/exemption details made for a specific month. A report generated includes the reason for the exemption or the adjustments made. You have an option to generate a report for either the active documents or the cancelled documents.

List of Adjustments/Exemptions

From the Transactions > Bill Exemption/Adjustment, double-click List of Adjustments/Exemptions Function.

The List of Adjustments/Exemptions screen gets displayed.

  1. Select a Transaction Type from the list. The system defined values are Adjustment and Exemption. Select Adjustment to list either the active or the cancelled adjustment records for the specified month range. Select Exemption to list either the active or the cancelled exemption records for the specified month range.

  2. Select a Status from the list. The system defined values are – Active and Cancelled. Select Active to list either the active adjustment or the exemption records for the specified month range. Select Cancelled to list either the cancelled adjustment or the exemption records for the specified month range.

  3. Click Calendar function to select the month and year range. The canceled or active, adjustment or exemption documents are listed for the specified month and year. The month and year selected has to be equal to or lesser than the current month and year.

  4. Click [List of Values] to select the reason. The documents with the selected reason are listed in the report.

  5. Click [Process Run] to print the details or [Clear Record] to clear the entered data.

  6. Click View, Print or File to view the report, print or save the output in a file respectively.

  7. A log file is generated. Click [OK] to view the log file or [Cancel] to exit from the screen.
