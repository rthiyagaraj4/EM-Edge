Re-Print Adjustment/Exemption Doc

From this function you can Re-print the bill adjustment or the exemption documents. You can choose to print either the Active or the Cancelled documents.

Re-print Adjustment/Exemption Doc function is used to print the adjustment or the exemption documents. The user can select a reason from the list of values and specify the document number range. A report is generated based on the parameters set. A destination printer has to be set for the selected report ID to print the details.

Re-print Adjustment/Exemption Doc

From the Transactions > Bill Exemption/Adjustment, double-click Re-print Adjustment/Exemption Doc Function.

The Re-print Adjustment/Exemption Doc screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run Process] to print the details or [Clear Record] to clear the entered data.
