Cancel Bill Exemption

From this function you can Cancel the bill exemption details of a patient. You can Query to view the exemption details that are already cancelled.

Cancel Bill Exemption function is used to cancel the bill exemption request of a patient. An authorized user only can cancel the bill exemption. The ‘Bill Exemption’ and ‘Bill Adjustment’ check boxes have to be selected in the function ‘Users for Authorization/Refund/Cash Counters’ in the Administration menu for user to login and cancel the bill exemption process. A bill that is transferred to DCP cannot be cancelled.

Cancel Bill Exemption

From the Transactions > Bill Exemption/Adjustment, double-click Cancel Bill Exemption Function.

The Cancel Bill Exemption screen gets displayed.

  1. Select a Patient Number for transaction or use the Lookup function to select a patient. You can search for a relevant patient ID in the lookup screen. Click [Enter Query], enter the search criteria and click [Execute Query] or press [F8] to select a patient ID. The patient details get populated in the respective fields.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Authorize] to login  and to authorize the bill exemption for the selected patient.

    1. User ID – Enter the User name in alphanumeric format. This is a mandatory field.

    2. PIN No – Enter the Pin number for the selected patient. This is a mandatory field.

    3. Reason – Click [List of Values] to select a Reason for exemption. The reasons defined using the function ‘Reasons for Billing Actions’ in the Masters with the ‘Cancel Exemption’ check box as selected gets listed. This is a mandatory field.

  4. You can cancel and view the bill exemption details for a patient using the following tab pages:

Cancel

Click Cancel to cancel the bill exemption recorded using the Create tab.

  1. Document No – Click [List of Values] to select and to cancel a document number. The document numbers created using the Create tab for the patients’ get listed. The Patient Name, ID, Age, Gender of the patient and the reason for exemption get displayed.

  2. Cancel Remarks – Enter the cancellation remarks in alphanumeric format of 100 characters maximum length. This is a mandatory field.

Click [Execute Query] or press [F8] to view the adjustment details recorded against an outstanding bill. The outstanding amount to be paid by the patient on cancelling the exemption gets displayed.

  1. [Authorize] – Click [Authorize] to login  and to cancel the bill exemption for the selected patient.

    1. User ID – Enter the User name in alphanumeric format. This is a mandatory field.

    2. PIN No – Enter the PIN number for the selected patient. This is a mandatory field.

    3. Reason – Click [List of Values] to select a Reason for cancellation. The reasons defined using the function ‘Reasons for Billing Actions’ in the Masters with the ‘Cancel Exemption’ check box as selected gets listed. This is a mandatory field.

A confirmation screen pops-up. Click [Yes] to cancel the exemption or [No] to abort the process and to exit from the screen.

View

Enter the following details to search for a patient record and to view the exemption details:

  1. Patient No – Enter the Patient ID or click [List of Values] to search for a patient record using the lookup  screen. The Patient Name, Age and Gender get displayed.

  2. Doc. [From–To] – Click [List of Values] to select a range of documents generated for the exemptions granted to the selected patient.

  3. Doc. Date [From–To] – Click Calendar function to select the date range. The exemption documents generated during the selected date range for the selected patient gets displayed.

  4. [Search] – Click [Search] to view the exemption details and the respective outstanding bill details against which the exemption is granted.

  1. Click [Commit Form] to record the details or [Clear Record] to clear the entered data.

  2. A Cancelled Exemption Document is printed on cancelling the details.
