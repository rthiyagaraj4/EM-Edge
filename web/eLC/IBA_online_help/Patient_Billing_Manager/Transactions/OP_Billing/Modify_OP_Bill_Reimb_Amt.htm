Modify OP Reimbursable Amt

From this function you can Update the reimbursable amount for a Billing Service after the bill is generated and before the bill amount is settled.

The Modify OP Bill Reimbursable Amount function enables you to update the reimbursable amount in the bill generated for an outpatient. The reimbursable amount for the service is defined in the ‘Billing Services’ function in the Masters. If the ‘override’ check box is selected for the selected service and the selected user is permitted to update the reimbursable amount for the respective service using the ‘Reimbursable Service by User’ tab in the ‘User for Authorization/Refund/Cash Counters’ function, the user can modify the reimbursable amount for an inpatient. The reimbursable amount can be modified for the patients' for whom the bills are generated for the services utilized from the facility, but the settlement is pending.

Modify OP Reimbursable Amt

From the Transactions > OP Billing, double-click Modify OP Reimbursable Amt Function.

The Modify OP Bill Reimbursable Amt screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Commit Form] to save the changes or [Clear Record] to clear the data.
