Reprint Visit Bills – With Settlement

From this function you can Reprint a copy of the visit bills with the settlement details for the selected Episode Type. To reprint the bills, the bill has to be printed during visit registration.

Reprint Visit Bills – With Settlement function is used to re-print the visit bills. You have an option to print a range of bills. A report is generated based on the parameters set. The printed report does not display the settlement details towards a specific visit, but only the bill details encountered for a relevant visit are printed.  An authorized User is permitted to login and re-print the visit bills. The check box ‘Reprint Bill’ has to be selected in the ‘Users for Authorization/Refund/Cash Counters’ function in the Administration menu for the selected User. The bill/receipt format that is printed during patient discharge is based on the format defined for the respective Episode Type through the Bill Print Format function.

Reprinting Visit Bills – With Settlement

From the Transactions > OP Billing, double-click the Reprint Visit Bills – With Settlement Function.

The Reprint Visit Bills – With Settlement screen gets displayed.

  1. Select an Episode Type to re-print visit bills with the settlement details. The visit bills generated for the selected Episode Type gets displayed. <Related Field Description>

  2. Click [Run Process] to print the bill or [Clear Record] to clear the entered data.

  3. A bill print authorization screen pops-up. Enter the User ID, PIN Number and the Remarks to login and to re-print the bill. Click [OK] to login and [Cancel] to exit from the screen.

  4. A confirmation screen pops-up on clicking [OK]. Click [Yes] to re-print the bills listed in the detailed block and [No] to cancel the printing process.
