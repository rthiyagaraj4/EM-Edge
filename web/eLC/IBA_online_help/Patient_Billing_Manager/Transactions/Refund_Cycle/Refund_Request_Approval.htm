Refund Request Approval

From this function you can Approve the refund request raised by the Cashier. You can also Query to view the inactive refund requests, inactive approvals and the requests, which are approved.

Refund Request Approval function is used to view and approve the refund requests raised by the Cashier using the function ‘Request For Refund’. The authorized billing personnel have to approve the refund request placed by the Cashier. You can also view the requests or approvals, which are inactive. Authorized Billing Personnel is only permitted to process the refund request.

Refund Request Approval

From the Transactions > Refund Cycle, double-click Refund Request Approval.

The Refund Request Approval screen gets displayed.

  1. Enter the details in the mandatory fields to search for the records meeting the search criteria. The details are displayed under the following tabs:

Requests for Approval

Inactive Requests, Approvals

Approved Requests

  1. Select the records and click [Commit Form] or click [Process] to record the refund request or [Clear Record] or [Cancel] to clear the entered data.

  2. Click [Execute Query] to view the details in the tabs Inactive Requests, Approvals and Approved Requests.
