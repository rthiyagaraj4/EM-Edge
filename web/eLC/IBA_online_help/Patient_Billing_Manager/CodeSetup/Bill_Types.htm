Bill Types

From this function you can Define various bills to categorize the patient class within the facility. You can specify the remarks to be printed on each Bill Type. You can Query to view the existing Bill Types.

Bill Types function is used to distinguish the bills generated for a patient category within the facility. Cashier has to generate a relevant Bill Type when a patient uses a service pertinent to the patient category. The Bill Types are interfaced to the General Ledger Interface Codes to keep a track of the income generated from each Bill Type.

Defining Bill Types

From the Masters > Accounting Entity Setup, double-click on Bill Types function.

The Bill Type screen gets displayed.

  1. Enter the details in the mandatory (*) fields found in the following Tab pages.

Bill Type

Interface to GL

  1. Click [Interface to GL] to associate the Bill Type to the Control Account Codes for General Ledger.

  2. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for the existing records within the facility.
