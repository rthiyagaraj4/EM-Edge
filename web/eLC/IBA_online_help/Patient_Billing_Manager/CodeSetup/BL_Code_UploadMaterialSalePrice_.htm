Upload Material Sale Price

From this function you can Upload the price for the material items to the database.

The Upload Material Sale Price function is used to upload the prices for the material items from an excel file in the local machine to the database. The excel file with the item and price details are saved in the database with a file extension as ‘.csv’. Using [Validate Data], you can validate the material items in the excel file stored in the database. The application checks for consistency and generates a report on clicking [Validate Data]. The prices stored in the database for the selected items are considered towards bill computation when the ‘Basis’ is selected as ‘Base Price’ in the ‘Setup/Amend Charge Factors for Materials linked Services’ function. To upload the details in the excel file, click [Upload Price]. The file name has to be the same and the existing file is overwritten in the database on uploading the contents. The prices in the excel file against the material items are considered when the Basis is selected as ‘Base Price’ in Setup/Amend Charge Factors for Materials Linked Services function. You can also use the function Material Item Billing Details to define the prices for a material item.

Uploading Material Sale Price

From the Masters > Interfaced Modules Financial Dtls, double-click Upload Material Sale Price function.

The Upload Material Sale Price screen gets displayed.

Upload Item Base Prices

The Upload Item Base Prices tab enables you to upload the prices for the items in an excel sheet to the database. On validating, the items already saved in the excel file in the database are checked for consistency and a report is generated, if an error is encountered.

Enter the following details to upload base prices for material items:

  1. Facility ID - Select the Facility from the list. The facilities associated to the logged in User get listed. The base prices defined are valid for the material items in the selected facility. This is a mandatory field.

  2. Effective From – Click Calendar function to select a date. The prices for the material items are valid from the date selected.

  3. [Validate Data] – Click [Validate Data] to validate the details in the excel file already saved in the database. The Price Id is generated on clicking [Validate Data].

  4. [Upload Price] - Click [Upload Price] to upload the file along with the Price Id to the database.

The data is populated into the database on clicking [Upload Price].

Upload Item Base Prices by Trade/Batch

The Upload Item Base Prices by Trade/Batch enables you to upload the item base prices by trade or batch. The Price ID generated is applicable to the Trade name or a Batch of the items created initially.

Enter the following details to upload the price:

  1. Facility ID - Select the Facility from the list. The facilities associated to the logged in User get displayed. This is a mandatory field.

  2. Effective From – Click Calendar function to select a date. The price defined in the excel file for the batch items or trade names are valid from the date selected.

  3. Price Id – The Price ID for the trade/batch is generated automatically when [Validate Data] is clicked.

  4. [Validate Data] – Click [Validate Data] to validate the details in the excel file already saved in the database. The Price Id is generated on clicking [Validate Data].

  5. [Upload Price] - Click [Upload Price] to upload the base prices defined for a Batch/Trade Name to the database.

The data for the batch items or the trade items get populated into the database on clicking [Upload Price].

  1. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records within the facility.
