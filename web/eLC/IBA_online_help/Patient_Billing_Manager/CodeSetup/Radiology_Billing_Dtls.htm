Radiology Billing Details

From this function you can Define additional charges for consumables and the drugs used.  You can Query to view the records created.    

Radiology Billing Details function is used to define additional charge for the consumables and the drugs used. A Billing Service is associated to the Consumables or the Drug items utilized during examination. The items are charged based on the price defined for the Billing Service associated to the selected consumable or a drug.

Defining Radiology Billing Detailsns

From the Masters > select Interfaced Modules Financial Dtls, double-click Radiology Billing Details  function.

The Radiology Billing Details screen gets displayed.

  1. Enter the mandatory (*) details under the following tabs:

Consumable Billing Details

The Consumable Billing Details tab is used to associate a Billing Service to a Consumable defined using the ‘Consumable Items’ tab in ‘Consumables’ function in the Radiology Information Systems module. A Consumable Item used is charged based on the prices defined for the associated Billing Service.

Enter the following details:

  1. Consumable Pack Code – Click [List of Values] to select a Consumable Pack. The values defined using the ‘Consumables’ function in the Radiology Information Systems module get listed. This is a mandatory field.

  2. Description – The Description for the selected Consumable Pack Code gets displayed.

  3. Service/Panel Ind – Select an Indicator from the list. The values are system defined - “Service” or “Panel”. This is a mandatory field.

  4. Service – Select Service to associate a Billing Service to the Consumable Pack. The charge for the selected Consumable Pack is calculated based on the prices defined for the Billing Service.

  5. Panel - Select Panel to associate a Billing Panel to the Consumable Pack. The charge for the selected Consumable Pack is calculated based on the prices defined for the Billing Panel.

  6. Service/Panel – Click [List of Values] to associate a Service/Panel code. The values defined using the functions ‘Billing Services’ and ‘Billing Panel’ gets listed respectively based on the Indicator selected. This is a mandatory field.

  7. Description – The Description for the relevant Billing Service or Billing Panel gets displayed.

Drugs Billing Details

The Drugs Billing Details tab is used to associate a Billing Service to a Drug defined using the ‘Constant Agents, Drugs’ tab in ‘Consumables’ function in the Radiology Information Systems module. A Drug used is charged based on the prices defined for the associated Billing Service.

Enter the following details:

  1. Drug Code – Click [List of Values] to select a Drug. The values defined using the ‘Consumables’ function in the Radiology Information Systems module gets listed. This is a mandatory field.

  2. Description – The Description for the selected Drug gets displayed.

  3. Service/Panel Ind – Select an Indicator from the list. The values are system defined - “Service” or “Panel”. This is a mandatory field.

  4. Service/Panel – Select Service to associate a Billing Service to the Drug. The charge for the selected Drug is calculated based on the prices defined for the Billing Service. Select Panel to associate a Billing Panel to the Drug. The charge for the selected Drug is calculated based on the prices defined for the Billing Panel.

  5. Service/Panel – Click [List of Values] to associate a Service/Panel code. The values defined using the functions ‘Billing Services’ and ‘Billing Panel’ gets listed respectively based on the Indicator selected. This is a mandatory field.

  6. Description – The Description for the relevant Billing Service or Billing Panel gets displayed.

  1. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records within the facility.
