Maintain Calendar

From this function you can Create and Update a Calendar for a relevant year. You can Query to view the existing Calendar and update the Day Types or the Remarks for a particular Day Type.

The Maintain Calendar function enables you to set up a calendar for a relevant year. To create a full-fledged Calendar, you have to define the Day Types and the Time Types using the functions ‘Day Types’ and ‘Time Types’ respectively. The Maintain Calendar function consists of two tabs, which enables you to create a calendar and maintain it. You can also update a Day Type selected for a relevant Calendar year and the Description for a respective date.

Defining and updating a Calendar

From Masters >  Billing Calendar, double-click Maintain Calendar function.

The Maintain Calendar screen gets displayed.

  1. Enter the details in the mandatory (*) fields found in the following tab pages.

Maintain Calendar

The Maintain Calendar tab enables you to update a Day Type selected for a relevant Calendar year. You can update the Description for the respective date. The Calendar year defined using the Create Calendar tab gets displayed.

Enter the following details to update the Calendar:

  1. Calendar Year – Click [List of Values] to select and update the details of a Calendar Year. Calendar years created using the Create Calendar function gets listed. This is a mandatory field. The date, day type, description and the remarks gets displayed.

You can update only the Day Type and the Remarks for a Day Type. The Description for the respective Day Type is automatically updated when the Day Type is changed.

Create Calendar

Create Calendar tab is used to define a Calendar for a specific year indicating the working days, holidays, working hours and the non-working hours in the facility. The Calendar is created for a specific year by associating the Day Types and marking the holidays in the specific year to indicate that the selected day is a holiday.

Enter the following details to create a Calendar:

  1. Calendar Year – Enter the Year for which Calendar function has to be defined. The data has to be in numeric format not exceeding four digits. This is a mandatory field.

Day Type Details

  1. Day Type – Select a Day Type from the list. The Day Types defined using the ‘Day Type’ tab gets listed. This is a mandatory field.

  2. Description – The Description for the selected Day Type gets displayed.

Holiday Details

  1. Holiday Date – Click Calendar function to select the date. The Holiday Dates for the specific year has to be defined. This is a mandatory field.

  2. Day Type – Click [List of Values] to select a Day Type. This is a mandatory field.

  3. Description – The Description for the relevant Day Type is displayed.

  4. Remarks – Enter the Remarks in alphanumeric format. A valid description for the selected Holiday is to be defined.

  1. Press [F8] or click [Execute Query] to view the Calendar defined..

  2. Modify the details of an individual date for a relevant year, if necessary.

  3. Click [Commit Form] or press [F10] to save a the records or [Clear Record] to clear the entered data.
