Setup Rule Factors

From this function, you can Setup the rules for additional charges for the services, which are availed by a patient during the non-working days or the non-working hours of the facility. You can Query to view the existing details.

You can define the Time Slabs, Time-based facotrs and the Maintain the Time-Based Factors based on which the additional charges are to be computed when the patient is admitted to the facility during holidays or non-working hours.

Defining Rules

From the Masters > Billing Rule, double-click on Setup Rule Factors function.

The Setup Rule Factors screen gets displayed.

  1. Enter the details in the mandatory (*) fields found in the following tab pages.

Create Time-Based Factors

Enter the following details to create time-based factors:

  1. Billing Class – Click [List of Values] to select a Billing Class. The values defined using the Billing Class function gets listed. The additional factor is applicable for the selected Billing Class. This is a mandatory field.

  2. Day Type – Click [List of Values] to select a Day Type. The values defined using the ‘Day Types’ tab in the ‘Day Types/Time Types/Time Slabs’ function gets displayed. The additional factor is applicable for the selected Day Type in the facility. This is a mandatory field.

  3. Time Type – Click [List of Values] to select a Time Type. The values defined using the ‘Time Types’ tab in the ‘Day Types/Time Types/Time Slabs’ function gets displayed. The additional factor is applicable for the selected Time Type in the facility. This is a mandatory field.

  4. Time Slab – [From–To] – The Time Slabs defined for the respective Day Type and the Time Type using the ‘Time Slabs’ tab in the ‘Day Types/Time Types/Time Slabs’ function gets displayed. The value gets displayed automatically on selecting the Day Type and the Time Type and you cannot alter the values.

  5. Billing Service – [From–To] – Click [List of Values] to select a range of Billing Service. The values defined using the Billing Services function gets displayed. The additional factor defined is valid for the range of services availed from the facility during the selected Day Type and Time Type for the respective Billing Class. All the Billing Services are considered, if the Billing Service is not selected.

  6. Option – Select a value from the list. You can enter an alphabet or the service name in the adjacent text box field to select the services for additional factor.

  7. Additional Factor – Enter the additional factor in numeric format of seven digits maximum length. The Additional Factor for the Billing Service is computed as (Billing Service Price)*(Factor).

Create Time Slabs

Enter the following details to create time slabs:

  1. Billing Class [From–To] – Click [List of Values] to select a Billing Class. The values defined using the Billing Class function gets listed. The Time Slab defined is applicable for the selected Billing Class. The ‘From’ field is mandatory.

  2. Day Type Code – Click [List of Values] to select a Day Type Code. The values defined using the ‘Day Types’ tab in the ‘Day Types/Time Types/Time Slabs’ function gets displayed. The Time Slab is applicable for the selected Day Type. This is a mandatory field.

  3. Time Type Code – Click  [List of Values] to select a Time Type Code. The values defined using the ‘Time Types’ tab in the ‘Day Types/Time Types/Time Slabs’ function gets displayed. The additional charge is applicable for the selected Time Type in the facility. This is a mandatory field.

  4. Time Slabs – [From–To] – The Time Slabs defined for the respective Day Type and the Time Type using the ‘Time Slab’ tab in the ‘Day Types/Time Types/Time Slab’ function gets displayed. The value gets displayed automatically on selecting the Day Type and the Time Type and you cannot alter the values.

On recording the details and clicking [Save], a confirmation screen gets displayed.

Maintain Time-Based Factors

Enter the following details to maintain time-based factors:

  1. Billing Class [From–To] – Click [List of Values] to select a Billing Class. The values defined using the Billing Class function gets listed. The Time Slab defined is applicable for the selected Billing Class. Press [F8] or click [Execute Query] to view all time-based factors for the selected Billing Class.

  2. Day Type – Click [List of Values] to select a Day Type. The Day Types associated to the selected Billing Class gets listed.  

  3. Time Type – Click [List of Values] to select a Time Type. The Time Types associated to the Billing Class gets listed.

  4. Time Slabs – [From–To] – The Time Slabs defined for the respective Day Type and the Time Type using the ‘Time Slab’ tab in the ‘Day Types/Time Types/Time Slab’ function gets displayed. The value gets displayed automatically on selecting the Day Type and the Time Type and you cannot alter the values.

Press [F8] or click [Execute Query] to view the Billing Services and the additional factors associated to the Billing Class, Day Type, Time Slabs and Time Type combination. You can also associate additional Billing Services and define the additional factor for the selected Billing Service.

  1. Billing Service - Click [List of Values] to select a Billing Service. The Billing Services defined in the Masters under the ‘Billing Services’ menu gets listed.  You can also select ‘**’ to select and associate All Billing Services to the selected Billing Class. This is a mandatory field.

  2. Description – The Description for the selected Billing Service gets displayed.

  3. Additional Factor – Enter the Additional Factor in numeric format with a maximum of 10 digits. If the additional factor is defined under the ‘Create Time-Based Factors’ tab, you can update the value.

  1. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records within the facility.
