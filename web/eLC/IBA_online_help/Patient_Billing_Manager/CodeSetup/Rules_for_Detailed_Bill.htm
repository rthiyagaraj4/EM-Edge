Rules For Detailed Bill

From this function you can Define a standard format for generating or printing a detailed or a summarized bill. You can Create bill formats for individual payers as well as for multiple payers.

A detailed bill contains detailed information about the services availed from the Facility. The function Rules for Detailed Bill is used to associate a precise Billing Group to a relevant Print Group. When you choose to print a detailed bill for the services availed from the facility, the information regarding the Billing Group and the relevant Print Group associated to the respective Billing Group are also printed.

Defining Rules for a Detailed Bill

From the Masters > select Billing Rule, double-click Rules for Detailed Bill function.

The Rules for Detailed Bill screen gets displayed.

  1. Select a Facility from the list. The system displays the Facility associated with the logged in user. The rules for the detailed bill setup here is pertinent to the selected Facility. This is a mandatory field.

  2. Enter the mandatory details (*) found in the tab pages. The Bill Format tab is enabled, by default.
