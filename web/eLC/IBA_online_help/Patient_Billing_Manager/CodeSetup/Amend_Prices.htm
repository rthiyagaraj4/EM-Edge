Amend Prices – (Any)

From this function you can Update the prices defined for a Price Class or a Customer group, the Standard price for a Service or the prices associated with the Billing Group. You can modify the discount percentage as well. You can Query to view the prices defined for the Billing Classes associated to a Billing Service.

The Amend Prices – (Any) function is used to modify the prices defined for a patient class using the Price Class Specific Prices function. You can update the price and the discount details for a patient class, which are already defined using the functions Price Class Specific Prices and Billing Group/Customer Specific functions.

Amend Prices

From the Masters > Multi Pricing, double-click Amend Prices – Any function.

The Amend Prices - Any screen gets displayed.

  1. Select a Facility from the list.

  2. Select a Billing Service from the list. The price and the discounts defined for the selected Billing Service get displayed. This is a mandatory field. <Related Field Description>

  3. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  4. Click [Execute Query] or press [F8] to search for the existing records within the facility.
