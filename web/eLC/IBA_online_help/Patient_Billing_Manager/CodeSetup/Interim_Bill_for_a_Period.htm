Interim Bill for a Period

From this function you can Generate an Interim Bill for a Period.

The Interim Bill for a Period function is used to generate the interim bills for the inpatients and the daycare patients for whom the discharge bill is not generated.

Generating an Interim Bill for a Period

From the Reports> Revenue Reports, select the Interim Bill for a Period function.

The Interim Bill for a Period function screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Recalculate] to recalculate the patient charges if the patient is moved from one bed class to another bed class.

  3. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  4. Report Option is displayed with the View, Print and File Preferences on clicking [Yes]. Click [OK] to generate or [Cancel] to abort the process.

    Select File to save the output in a file in the local system.

    Select Print to print the report.

    Select View to view the online report.
