Operation Theatre Billing Details    

From this function you can Define the charges for the equipments, consumables, implants and the CSSD packs used in the Operating Theatre. You can Query to view the billing details defined for the equipments.

The Operation Theatre Billing Details function is used to define the charge for the equipments used in the Operation Theatre of a facility. The charge for the consumables and the equipments used in the Operating Theatre are based on either the associated Billing Services or the Billing Panel. The charge for the consumables and the equipments used is calculated based on the price defined for the respective patient class while defining the Billing Service.

Defining Operation Theatre Billing Details

From the Masters > Interfaced Modules Financial Dtls, double-click Operation Theatre Billing Details function.

The Operation Theatre screen gets displayed with Consumable Billing Details tab enabled.

  1. Enter the details in the mandatory (*) fields found in the following Tab pages.

Consumable Billing Details

The Consumable Billing Details tab is used to associate a Billing Service to a Consumable Pack. Consumable Packs are defined in the Operating Theatre module. A Consumable Item is charged based on the prices defined for the associated Billing Service.

Enter the following details to define billing details for the Consumables:

  1. Consumable Pack Code – Click [List of Values] to select a Consumable Pack. The values defined using the function Consumable Packs in the Operating Theatre module get listed. This is a mandatory field.

  2. Description – The Description for the selected Consumable Pack Code gets displayed.

  3. Service/Panel Ind – Select an Indicator from the list. The values are system defined - Service or Panel. This is a mandatory field.

  4. Service – Select Service to associate a Billing Service to the Consumable Pack. The charge for the selected Consumable Pack is calculated based on the prices defined for the Billing Service.

  5. Panel - Select Panel to associate a Billing Panel to the Consumable Pack. The charge for the selected Consumable Pack is calculated based on the prices defined for the Billing Panel.

  1. Service/Panel – Click [List of Values] to associate a Service/Panel code. The values defined using the functions ‘Billing Services’ and ‘Billing Panel’ gets listed respectively based on the Indicator selected. This is a mandatory field.

  2. Description – The Description for the relevant Billing Service or Billing Panel gets displayed.

Equipment Billing Details

Equipment Billing Details tab is used to define the charges for the apparatus used in the Operation Theatre. A Billing Service or a Billing Panel has to be associated to the Equipments. The charge is calculated based on the prices defined for a patient category while defining the Billing Services.

Enter the following details to define the billing details for equipments:

  1. Equipment – Click [List of Values] to select an Equipment. The values defined using the function Equipments in the Operating Theatre module get listed. This is a mandatory field.

  2. Description – The Description for the selected Equipment gets displayed.

  3. Service/Panel Ind – Select an Indicator from the list. The values are system defined -‘Service’ or ‘Panel’. This is a mandatory field.

  4. Service – Select Service to associate a Billing Service to an Equipment. The charge for the selected Equipment is calculated based on the prices defined for the Billing Service.

  5. Panel - Select Panel to associate a Billing Panel to an Equipment. The charge for the selected Equipment is calculated based on the prices defined for the Billing Panel.

  1. Service/Panel – Click [List of Values] to associate a Service/Panel code. The values defined using the functions ‘Billing Services’ and ‘Billing Panel’ gets listed respectively based on the Indicator selected. This is a mandatory field.

  2. Description – The Description for the relevant Billing Service or Billing Panel gets displayed.

  1. CSSD Pack – Click [List of Values] to select an Instrument/CSSD Pack. The values defined using the function Instruments/CSSD Packs in the Operating Theatre module gets listed. This is a mandatory field.

  2. Description – The Description for the selected Instruments/CSSD Pack gets displayed.

  3. Service/Panel Ind – Select an Indicator from the list. The values are system defined - Service or Panel. This is a mandatory field.

  4. Service – Select Service to associate a Billing Service to an Instrument/CSSD Pack. The charge for the selected Instrument/CSSD Pack is calculated based on the prices defined for the Billing Service.

  5. Panel - Select Panel to associate a Billing Panel to an Instrument/CSSD Pack. The charge for the selected Instrument/CSSD Pack is calculated based on the prices defined for the Billing Panel.

  1. Service/Panel – Click [List of Values] to associate a Service/Panel code. The values defined using the functions ‘Billing Services’ and ‘Billing Panel’ gets listed respectively based on the Indicator selected. This is a mandatory field.

  2. Description – The Description for the relevant Billing Service or Billing Panel gets displayed.

The charge is calculated based on the prices defined for a patient category while defining the Billing Services.

Enter the following details to define the billing details for Implants:

  2. Description – The Description for the selected Prosthesis/Implants gets displayed.

  3. Service/Panel Ind – Select an Indicator from the list. The values are system defined - Service or Panel. This is a mandatory field.

  2. Description – The Description for the relevant Billing Service or Billing Panel is displayed.

  1. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records within the facility.
