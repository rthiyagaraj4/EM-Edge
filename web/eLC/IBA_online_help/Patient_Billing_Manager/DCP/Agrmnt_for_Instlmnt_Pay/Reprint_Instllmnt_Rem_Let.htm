Re-print Installment Reminder Letters – Installment

From this function you can Print the outstanding document details against which the reminder letters are dispatched.  

Re-print Installment Reminder Letters – Installment function is used to print the details of the outstanding documents against which the reminder letters are dispatched. The reminder letters dispatched to the patient are printed. The destination printer has to be defined for the selected report group in the System Manager module. The dispatch document type and number generated on dispatching the reminder letters using the function ‘Record Installment Reminder Dispatches’ get listed.

Re-Print Installment Reminder Letters - Installment

From the Deferred Cash Patients > Agreement for Installment Payment, double-click Re-print Installment Reminder Letters – Installment function.

The Re-print Installment Reminder Letters – Installment screen gets displayed.

  1. Click Calendar function to select a date range. The reminder letters dispatched during the selected date range is printed. This is a mandatory field.

  2. Click [List of Values] to select a Dispatch Document Number. The dispatch document number generated on using the function ‘Record Installment Reminder Dispatches’ for the selected set of outstanding records are printed.

  3. Enter the serial number for the dispatched document.

  4. Click [List of Values] to select a valid Patient Number. The reminder letters, which are dispatched for the selected patient are printed.

  5. Enter the cumulative document number for the patient records.

  6. Click [Run Process] to print the reminder letter details, which are dispatched. A confirmation screen pops-up. Click [Yes] to print the details and [No] to cancel.
