Record Installment Reminder Dispatches

From this function you can Generate reminder letters for dispatch to remind a patient to settle the outstanding bill amount. You can also Record and View the undelivered dispatches.

Record Installment Payment function is used to generate a reminder letter to be dispatched to a patient or the sponsor to settle the outstanding installment payment. You have to generate the reminder letter against a record or the selected records and dispatch the letter. An undelivered dispatch is recorded using the ‘Record Undelivered Dispatches’ tab. You have an option to view the undelivered dispatches using the ‘View Undelivered Dispatches’ tab.

Record Installment Reminder Dispatches

From the Deferred Cash Patients > Agreement for Installment Payment, double-click Record Installment Payment function.

The Record Installment Payment screen gets displayed.

  1. Select an Outstanding Amount Range. The Outstanding amount range defined using the Reminder Letter Setup function gets listed. The documents with the selected outstanding amount details get listed. Select All to view and to generate the reminder letters either for the selected documents or for all the documents. This is a mandatory field.

  2. Select a Reminder Number from the list. The number of reminder letters defined for the outstanding amount range using the function Reminder Letter Setup gets listed. Select All to view and to generate the reminder letters either for the selected or for all the documents. This is a mandatory field.

  3. The list of outstanding documents are displayed under the Generate Letters for Dispatch tab. <Related Field Description>

  4. You can record the undelivered dispatched using the Record Undelivered Dispatches tab. <Related Field Description>

  5. The undelivered dispatches are displayed under the tab View Undelivered Dispatches. You can only view the details using this tab.
