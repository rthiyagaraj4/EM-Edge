Audit Trail of Alloc Trans

From this function you can Generate a report to assess the deferred cash patient details with regards to the allocating transaction documents. You can either View or Print or Save the report. A report generated lists out the posted allocating transaction documents only.

Audit Trail of Alloc Trans function is used to generate a report on the allocating transactions recorded using the function ‘Enter Alloc Trans’. A detailed report is generated listing the Document Type/Number, Transaction Type, Date of Transaction, the total amount and the posted date. The documents posted using the function ‘Post Alloc Trans’ are only listed in the report.

Audit Trail of Alloc Trans

From the Deferred Cash Patients > DCP Period End, double-click on Audit Trail of Alloc Trans function.

The Audit Trail of Alloc Trans screen gets displayed.

  1. Click [List of Values] to select a range of documents. The selected Document Type and the Document Number has to be valid. A report is generated for the selected Document Type and Document Number range. The Document Number and the Document Type under the ‘To’ field has to be greater than the Document Type and the Document Number in the ‘From’ field.

  2. Click Calendar function to select a date range. The report generated lists the details on the documents generated between the selected date range. The Document Date under the ‘To’ field has to be greater than the Document Date selected in the ‘From’ field.

  3. Enter the Operator ID in alphanumeric format. The report generated lists the documents generated by the specific Operator.

  4. Click [Run Process] to generate the report. Click View, Print or File to view the output online, print the output or to save the output in a file.  

  5. Click [OK] to generate the output and [C] to cancel the output.
