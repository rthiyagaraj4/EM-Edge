Allocate Open Credits

From this function you can Adjust the open credits for a Document Type. You can either View or Print or Save the report. A report generated lists out the posted allocating transaction documents only.

Allocate Open Credits function is used to adjust the credits for an outstanding bill. You have to select a Document Type for transaction. The selected Document Type numbering is system generated. The credits are adjusted automatically for the selected Document Type based on the date selected for outstanding and open credits.

Allocate Open Credits

From the Deferred Cash Patients > DCP Period End, double-click Allocate Open Credits function.

The Allocate Open Credits screen gets displayed.

  1. Click Calendar function to select a date. A report is generated listing the open credit details until the date selected. The date selected has to be equal to or lesser than the system date. This is a mandatory field.

  2. Click Calendar function to select a date. The report generated lists the outstanding bills until the date selected. The date selected has to be equal to or lesser than the system date. This is a mandatory field.

  3. Click [List of Values] to select a Document Type. The open credits are adjusted for the selected Document Type. The selected Document Type has to be valid and the Document details have to be defined for the selected Document Type.

  4. Click [Run Process] to generate a report. A screen pops-up requesting confirmation. Click [Yes] to process the allocate the open credits and [No] to cancel the process.

 

• You have to select a Document Type where the numbering is system generated. The Document Type numbering is selected as either system generated in the Financials Base Manager module.

• The transaction type has to be defined for the Document Type.

• The Document Types where the Transaction Types are defined as ‘Adjustments or Payments’ get listed.
