Enter Alloc Trans

From this function you can Generate a originate transaction document for a deferred cash patient. You can Query to view the Originating transactions defined.

Enter Alloc Trans is an option provided to generate an allocating transaction document for a cash patient. An allocating document is dependent on the originating transaction document. You have to associate an allocating transaction to a General Ledger Distribution account. You are able to update the general ledger distribution account associated to an allocating transaction document. Examples of allocating transactions are Credit Notes, receipts generated in the facility. Several validations are carried out while associating the transactions to the originating transactions.

  • Insert a new allocating document for a deferred cash patient along with the General Ledger distributions and allocation details.

  • Insert a new General Ledger distribution line and/or new allocation details for an already existing allocating document.

  • Change the details of allocating documents as well as the General Ledger distributions and allocation details.

  • Query on the details of an existing allocating document.

  • Delete an existing allocating document completely with all the General Ledger distributions and allocation details.

  • Deleting a specific General Ledger distribution line is also allowed.

  • Deleting a specific allocation detail is also allowed.

Enter Alloc Trans

From the Deferred Cash Patients > DCP Transaction, double-click on Enter Alloc Trans function.

The Enter Alloc Trans screen gets displayed.

  1. Right click on Click for User Option to query for a patient number.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Commit Form] or press [F10] to save the records or [Clear Record] to clear the data entered.

  4. Click [Execute Query] or press [F8] to query for the existing records.
