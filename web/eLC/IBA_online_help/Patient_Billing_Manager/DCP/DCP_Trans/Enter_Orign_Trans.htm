Enter Orign Trans

From this function you can Generate a original transaction document for a deferred cash patient. You can Query to view the Originating transactions defined.

Enter Orign Trans function is used to generate an originating transaction document for a cash patient. An originating document is an original document, which is referred by the allocating transaction documents. You have to associate an originating transaction document to a General Ledger Distribution account. You have an option to update a General Ledger Distribution Account associated to an Originating Transaction Document. Examples of Originating Transactions are Debit Notes, bills generated in the facility. Several validations are performed while using this function.

  • Insert a new originating document for a Deferred Cash patient along with the General Ledger distributions.

  • Insert a new General Ledger distribution line for an already existing originating document.

  • Change the details of originating documents as well as the General Ledger distributions.

  • Query on the details of an existing originating document.

  • Delete an existing originating document completely with all the General Ledger distributions.

  • Delete a specific General Ledger distribution line is also allowed.

Following validations are carried out for the Document Number:

  • Document Type and Number has to be unique in the transaction entry.

  • Document Type and Number must not be present already in Pending Documents detail.

  • Document Type and Number must not be present in Document Control details.

Enter Orign Trans

From the Deferred Cash Patients > DCP Transaction, double-click on Enter Orign Trans function.

The Enter Orign Trans screen gets displayed.

  1. Right click on Click for User Option to query for a patient number.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Commit Form] or press [F10] to save the records or [Clear Record] to clear the data entered.

  4. Click [Execute Query] or press [F8] to query for the existing records.
