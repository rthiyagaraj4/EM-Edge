Discharge Day Bed Charge Discount Rule

The Discharge Day Bed Charge Discount Rule tab enables you to define the discount rule for the bed types within the facility. The discount rule is applicable for the charge elements like the bed charge, doctor round charge, consumable charge, Workup Charge and the ICU charge. You can set the discount type and the discount value on the charge elements based on which the discount amount is calculated during patient discharge from the facility.

 

Enter the following details:

  1. Bed Type – The Bed Type is displayed as ‘**’, by default. The discount rule is applicable for all the beds in the Nursing Unit. Click ‘Create Record’ icon to define the discount rule for a selected bed in the facility.

  2. Facility – The name of the logged in Facility is displayed, by default.

 

Normal Discharge

Total Stay in Days

  1. From – To – Enter the total number of days in numeric format. The beginning day has to start with ‘0’. This defines the discount rule for the normal discharge of the patient from the facility.

 

Discharge Day Stay in Hours

  1. From – To - Enter the total number of hours in numeric format. The value has to begin with ‘0’. This defines the discount rule for the normal discharge of the patient from the facility.

 

Charge Elements

The Charge Elements define the elements that are to be charged during discharge.

  1. Charge Element – The system defined Charge Elements are ‘Bed Charge’, ‘Doctor Round Charge’, ‘Consumable Charge’, ‘Workup Charge’ ‘ICU Charge’ and 'Discharge Day Charge'. The charges for these elements during a normal patient discharge are calculated based on the charge type and the discount type selected. The discharge day charge for a normal discharge is computed based on the Billing Service associated to the Bed Type in ‘Inpatient Billing Details’ function. The discount value set is valid for the services that are associated to the Bed Type in ‘Inpatient Billing Details’ function.

  2. Charge Type – Select a value from the list. The values are system defined – ‘As Per Policy Setup’ and ‘Charge to Patient Only’. The elements are charged based on the policy coverage limit specified, if the value is selected as ‘As Per Policy Setup’. The elements are charged on the policy coverage limit specified, if the value is selected as ‘As Per Policy Setup’ and the patient is insured. If the patient is a non-insurance payer, the amount for the element is charged on the patient on the discharge day for normal discharge.

  3. Discount Type – Select a value from the list. The values are system defined – Percentage and Amount. Percentage indicates that the percentage of the actual amount is deducted on the element for normal discharge. Amount indicates that the amount mentioned is deducted from the actual amount for the element during normal discharge.

  4. Discount Value – Enter the discount value in numeric format to indicate the amount to be deducted for the elements during normal discharge.

  5. Posting Mode - Select a value from the list. The values are system defined – As Discount and As Net. Select As Discount to post the charge as discount or As Net to post the net charges.
