  2. Charge Element – The Charge Elements are system defined – ‘Bed Charge’, ‘Doctor Round Charge’, ‘Consumable Charge’, Workup Charge’ and ICU Charge. The charges for these elements for reserved bed are calculated based on the charge type and the discount type selected.

  3. Charge Type – Select a value from the list. The values are system defined – ‘As Per Policy Setup’ and ‘Charge to Patient Only’. The elements are charged based on the policy coverage limit specified, if the value is selected as ‘As Per Policy Setup’. The elements are charged on the policy coverage limit specified, if the value is selected as ‘As Per Policy Setup’ and the patient is insured. If the patient is a non-insurance payer, the amount for the element is charged on the patient on the discharge day for the hours the bed is reserved.

  4. Discount Type – Select a value from the list. The values are system defined – Percentage and Amount. Percentage indicates that the percentage of the actual amount is deducted on the element for deceased patient discharge. Amount indicates that the amount mentioned is deducted from the actual amount for the element during deceased patient discharge.

  5. Discount Value – Enter the discount value in numeric format to indicate the amount to be deducted for the elements which were reserved until the patient arrival.

The bed types which are allocated on hospital request are listed for charging basis.

  2. If a patient requests a general bed and due to non-availability a higher bed is assigned by the facility, the services and the bed charges are computed for a general bed.

  3. If the patient occupies a higher bed on hospital request, the charges are computed for the bed type the patient had requested.

AE Bed Charge Rule

  1. Bed Charge Applicable – Select the check box, if AE Bed Charge Rule is applicable. On selecting this check box, the bed charges are calculated for the patients admitted in the Accident and Emergency wards. This is a one time setup.

 

Charge Option

  • Consolidated for Total Stay - Select this option to consolidate all the bed charges and calculate the charges for total stay. Select the option ‘Rate’ to consolidate the charges based on the rate or select the option ‘Time’ to consolidate the charges based on the time.

  • Individual for total stay – Select this option to calculate individual charges for total stay. The options ‘Rate’ and ‘Time’ is not applicable if ‘Individual for total stay’ is selected. Charges computed for different AE bed / bay type occupied by the patient are posted during AE Check-out.

  1. Rate/Time – Select the check box to indicate if the bed charge has to be computed based on rate or time. This is enabled only when the ‘Charge Option’ is selected as ‘Consolidated for Total Stay’.

 

When the Charge option is selected as “Rate” and patient’s stay in individual bed / bay type is consolidated for the number of days, bed charges are computed at the highest rate of the occupied bed / bay type.

If the charge option is selected as “Time” and patient’s stay in individual bed / bay type is consolidated for the number of days, charge is computed based on the maximum time spent on the AE bed / bay type.  

If the patient occupies different bed / bay type with same time, the highest charge of AE bed / bay type is considered for bed charge computation.
