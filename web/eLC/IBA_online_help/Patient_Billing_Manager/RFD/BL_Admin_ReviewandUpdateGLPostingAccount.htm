The following details get populated in the respective text box. You can update the details before the posting:

 

  1. Trx Doc Ref –The transaction document reference for the source transaction gets displayed.

  2. Posting Month/Year - The Posting Month and Year get displayed, by default.

  3. Episode Details – The Episode Details for the document get displayed. It comprises of the Episode Type, Episode Number and the Visit Number.

  4. Patient No – The Patient ID gets displayed.

  5. Patient Name – The Name of the Patient gets displayed.

  6. Sex – Gender of the patient gets displayed.

  7. Age – The Age of the patient is displayed in Years, Months and the Number of days.

  8. Source Doc Ref – The Source Document Reference gets displayed.

  9. Source Date – The Source Document Date gets displayed.

  10. Cash Control Ref – The Cash Control Document Reference gets displayed.

  11. Cash Control Doc Date – The Cash Control Document Date gets displayed.

  12. Trx Type -The Transaction Type code gets displayed.

  13. Document Ref - The Document Reference gets displayed. The Document Reference is used in the bank reconciliation, if the posting account is specified as a bank account.

  14. Distribution Amt - The Distribution Amount gets displayed.

  15. Narration – Narration for the General Ledger distribution gets displayed.

  16. Posting A/C - The existing Posting Account Code gets displayed. Click [List of Values] to view and to select a valid posting account code. The description for the selected posting account code gets displayed. Suspended Posting accounts are not accepted. This is a mandatory field.
