Enter the following to cancel the indirect under charge items associated to a patient:

 

  1. Admission/Tfr. No. - Enter the preferred Admission and the transfer serial number generated during registration for the relevant patient. You can also use the lookup  option to search for an admission and transfer number associated to a patient encounter. This is a mandatory field.

  2. Transaction Date - This is display field. System will default the current date as transaction date. This is required to determine the day type code and the time type code for billing charge calculations.

  3. Patient No. – Enter the Patient Number in alphanumeric format of 20 characters maximum length. Using the lookup, the patient number can be selected. On selecting the valid values of Patient Number or Admission No. and Trf.no. pressing enter or tab key, values of  Patient Name, Sex, Age, Ward No, Doctor, Bed Class, Bed No, Room No. will be displayed by the system.

  4. Line - The serial number for the services associated during associating the service to the patient gets displayed on selecting the transaction reference ID.

  5. Source Doc. Ref. – Click [List of Values] to select a reference ID. The transaction reference ID generated while associating the services to the patient using the functions ‘Enter Request – IP’ and ‘Enter User Dept Request – IP’ gets listed. The transaction date, service code, UOM, service quantity, total payable amount, patient payable amount and the practitioner or staff details associated to the patient during transaction gets displayed.
