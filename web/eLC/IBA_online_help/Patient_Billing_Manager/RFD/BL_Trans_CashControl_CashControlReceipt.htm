Enter the following details to generate Cash Control Receipts:

 

  1. Doc Type/Num – Click [List of Values] to select a Document Type. The system generates the Document Number automatically, if the check box ‘System Generated Document Number’ is selected in the function ‘Document Type Details’ in the Financials Base Manager module. This is a mandatory field.

  2. Receipt Date – The current system date is displayed, by default. Click the Calendar function to select a Receipt Date. This is a mandatory field.

  3. Receipt Amt – The Receipt Amount to be collected from the transactions is displayed.

  4. Cash Counter - Click [List of Values] to select a Cash counter. The transactions in the selected Cash Counter get listed. This is a mandatory field.

  5. Post Month/Year – The Post Month and Year is displayed automatically. Post month and year is as per the receipt date. This is defaulted from the accounting period definition.

  6. Print – Select the check box to print the Cash control receipt.
