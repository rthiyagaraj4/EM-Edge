Enter the following details to define Additional Service Rule for the Bill:

 

  1. Code – The code generated by system is displayed on saving the record.

  2. Cust Grp Code – Click [List of Values] to select the customer group.

  3. Cust code - Click [List of Values] to select the customer code.

  4. Policy Type Code - Click [List of Values] to select the policy type code created for each customer.

  5. Min Amt – Enter the minimum penalty amount for billing.

  6. Max Amt - Enter the maximum penalty amount for billing.

  7. Charging Serv Code - Click [List of Values] to select the charge base service code. The system prints the penalty amount on the bill based on the charge based service code.

  8. Effect From – Enter rule effective start date in the format of dd/mm/yyyy.

  9. Effect To - Enter rule effective end date in the format of dd/mm/yyyy.

  10. Billing Serv Code - Click [List of Values] to select the billing service code for services linked with sale category.

  11. Serv Item Code - Click [List of Values] to select the service item code.

  12. Min Amt – Enter the minimum penalty amount for the item.

  13. Max Amt - Enter the maximum penalty amount for the item.

  14. Charging % - Enter the charging percentage for the item.
