By Billing Group

The By Billing Group tab enables you to define the prices for a Bed or the Medical Personnel in the Facility. In general, the fixed bed charges for a given number of days, additional doctor charges for a given number of days are defined.

 

Enter the following details to define the bed charge rule:

 

  1. Charge Type - Click [List of Values] to select a Charge Type. The values are system - Bed Charges and Medical Personnel Fee. If the Charge Type is selected as Bed Charges, the Doctor Category cannot be set. The Doctor Category can be set only when the Charge Type is set to Medical Personnel Fee. This is a mandatory field.

  2. Bed Charges – Select Bed Charges to associate a Billing Group and a Billing Service to a Bed Charge and to enter the number of discharge days. The Bed Charge for the selected billing group is calculated based on the price defined for the billing service and the number of discharge days. The price for the selected Charge Type is applicable for the number of discharge days. When the days exceed the discharge days, the bed charge is computed as the multiple of the discharge days and the service price. For example, consider the service charge as 100 for a Charge Type and the discharge days is set as two. The price for the Charge Type is 100 for two days. If the discharge days rise to four, the charge is computed as 100*4 = 400. In case, the discharge days rise to three, still the charge is computed as 100*4 = 400.

  3. Medical Personnel Fee - Select Medical Personnel Fee to associate a Billing Group, a Bed Type, a Billing Service to a Doctor and to enter the number of discharge days. The Medical Personnel Fee for the selected billing group is calculated based on the price defined for the billing service and the number of discharge days. When the days exceed the discharge days, the medical personnel fee is computed as the multiple of the discharge days and the service price. For example, consider the service charge as 100 for a Charge Type and the discharge days is set as two. The price for the Charge Type is 100 for 2 days. If the discharge days rise to four, the charge is computed as 100*4 = 400. In case, the discharge days rise to three, still the charge is computed as 100*4 = 400.

  1. Doctor Category - Click [List of Values] to select a Doctor Category. The values are system defined - Associate, Resident and Not Applicable.

  2.  Associate – Select Associate, if the Doctor is not a resident, but an attending practitioner.

  3. Resident – Select Resident, if the Doctor is associated to the facility.

  4. Not Applicable – Select Not Applicable, if the Charge Type is selected as Bed Charges.

  1. Billing Group – Click [List of Values] to select a Billing Group. The Bed Charge defined is valid for the selected Billing Group. This is a mandatory field.

  2. Bed Type - Click [List of Values] to select a Bed Type. The values defined using the bed types function in the Inpatient and Day Care Management module get listed. The Bed Type field is enabled when the Charge Type is set as “Medical Personnel Fee’. If the Charge Type is set as Bed Charges’, the bed type field gets disabled.

  3. Billing Service for Bed Charge - Click [List of Values] to select a Billing Service. The bed charges are computed based on the prices defined for the selected billing service. This is a mandatory field.

  4. Discharge Days - Enter the number of days in numeric format with a maximum of three digits to indicate the days until when the bed or the doctor charge is applicable. This is a mandatory field.

  5. Billing Group - The description for the selected Billing Group gets displayed.

  6. Bed Type - The description for the selected Bed Type gets displayed.

  7. Billing Service - The description for the selected Billing Service gets displayed.
