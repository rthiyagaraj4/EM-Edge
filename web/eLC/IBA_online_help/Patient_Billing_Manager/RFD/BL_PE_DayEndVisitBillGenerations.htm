Enter the following to generate the visit bills:

 

  1. Option – Select a value from the list. The values are system defined - “Patient”, “Payer” and “All”. This is a mandatory field.

  2. Patient – Select Patient to generate visit bills for Outpatients and/or Emergency patients, if the patients settle the bill amount completely.

  3. Payer – Select Payer to generate visit bills for Outpatients and/or Emergency patients, if the Payers settle the bill amount completely. The text box ‘Payer’ gets enabled, if the Option is selected as ‘Payer’.

  4. All - Select All to generate visit bills for Outpatients and/or Emergency patients, where patients and Payers have settled the bill amount for a visit.

  1. Payer – Click [List of Values] to select a Payer. The values defined using the Customer function gets listed. A report is generated listing the visit bill details settled by the selected Payer.

  2. Visit Nature – Select a value from the list. The values are system defined - “Emergency Visits”, “Outpatient Visits” and “Both”.

  3. Emergency Visits – Select Emergency Visits to generate a report on the visit bills for Emergency patients.

  4. Outpatient Visits – Select Outpatient Visits to generate a report on the visit bills for Outpatients.

  5. Both - Select Both to generate a report on the visit bills for Outpatients and Emergency patients.

  1. Printing Not Required for Zero Bills – Select the check box and a report is not generated, if no visit bills are generated in the facility. Clear the check box to generate a report even though no visit bills are generated in the facility.

  2. Process – Select the check box to generate a report. A pop-up message gets displayed requesting confirmation to generate the visit bills.

Click [Yes] to confirm and [No] to cancel the process. A message appears that the process has started. Note that the bills generated during this process are not printed. A screen gets displayed, indicating the number of bills generated in the facility at the end of the day. A pop-up message gets displayed on report generation. Click [OK] to view the report. Click [Cancel] to exit from the screen.
