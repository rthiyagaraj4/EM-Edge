Enter the following details to generate a list of outstanding documents for which the write-off request has been placed:

 

  1. Patient Category – Select a valid Patient Category from the list. The system defined values are “All”, “Nationals” and “Others”. The report is generated for the selected Patient Category.

  2. Status - Select a Status from the list. The system defined values are “Cancelled Approvals”, “Pending Approval”, “Approved” and “Rejection”. The report is generated for on the status selected.

  3. Cancelled Approvals – Select Cancelled Approvals to generate a report for the write-off requests, which are cancelled.

  4. Pending Approval - Select Pending Approval to generate a report for the write-off requests, which are pending to be approved.

  5. Approved - Select Approved to generate a report for the write-off requests, which are approved.

  6. Rejection - Select Rejection to generate a report for the write-off requests, which are rejected.

  1. Write-off Amt [From–To] – Enter the Write-off Amount in numeric format. The report generated includes the write-off details for the amount range specified.

  2. Process Date [From–To] – Click Calendar function to select the Process Date. The report displays the details on the write-off process during the selected date. This is a mandatory field.

  3. Reason [From–To] – Click [List of Values] to select a Reason. The list of records for which the write-off has been requested, approved or rejected with the selected reason range gets displayed.
