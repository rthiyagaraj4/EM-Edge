Enter the following details to define a Bill Format:

Format

  1. Format - Enter the bill/receipt format name in alphanumeric format of 100 characters maximum length. This is a mandatory field. A Format code is unique and cannot be changed after creation.

  2. Format Description - Enter the description for ill/receipt in alphanumeric format. This is a mandatory field.

  3. First Time Print Indicator - Click [List of Values] to select an Indicator.  The values that are defined through the 'Static Information' tab are listed. The selected value is printed when the bill/receipt is printed initially. For Example, you can enter the value as 'Original' to indicate that the printed bill/receipt is a original copy. You can leave the field value as blanks, if the legend should not be printed.

  4. Reprint Indicator - Click [List of Values] to select an Indicator when a copy of the original bill is printed.  The values that are defined through the 'Static Information' tab are listed. The selected value is printed when the bill/receipt is printed subsequently. For Example, you can enter the value as 'Copy' to indicate that the printed bill/receipt is a copy of the original bill. You can leave the field value as blanks, if the legend should not be printed.

Applicable To

Select an Episode Type to use the bill/receipt format. This is a mandatory field.

  1. IP - Select the check box to use the bill/receipt format for inpatient transactions. Clear the check box, if the bill/receipt format is not valid for inpatient transactions.

  2. DC - Select the check box to use the bill/receipt format for daycare patient transactions. Clear the check box, if the bill/receipt format is not valid for daycare patients.

  3. OP -  Select the check box to use the bill/receipt format for outpatient transactions. Clear the check box, if the bill/receipt format is not valid for outpatient transactions.

  4. EM - Select the check box to use the bill/receipt format for emergency patient transactions. Clear the check box, if the bill/receipt format is not valid for emergency patients.

  5. EX - Select the check box to use the bill/receipt format for external patient transactions. Clear the check box, if the bill/receipt format is not valid for external patients.

  6. ALL - Select the check box to use the bill/receipt format irrespective of patient class. Clear the check box, if the bill/receipt format is valid for selected patient class. All the check boxes are selected by default, if you select 'ALL'.

Bill Title

You can select Facility related Column Values that has to appear as the Bill Title.

  1. Portrait – Select this radio button to print the bill in Portrait format. By default, this radio button is selected.

  2. Landscape - Select this radio button to print the bill in Portrait format.

  3. Seq No - Enter the sequence number in numeric format to define the hierarchy of column appearance as the title of the bill/receipt.

  4. Column1 - Click [List of Values] to select the Column that has to be printed on the bill/receipt for the selected sequence number. The listed values are system-defined.

  5. Static Value - Click [List of Values] to select that Static value that has to be printed on the bill/receipt for the selected column. The values defined through the 'Static Information' tab are listed.

Bill Header

  1. Seq No - Enter the sequence number in numeric format to define the hierarchy of column appearance in the header of the bill/receipt.

  2. Column1 - Click [List of Values] to select the first Column that has to be printed on the bill/receipt for the selected sequence number. The listed values are system-defined.

  3. Static Legend - Click [List of Values] to select that Static value that has to be printed on the bill/receipt for first column. The values defined through the 'Static Information' tab are listed.

  4. Static Value - Click [List of Values] to select that Static Value that has to be printed on the bill/receipt for the first column. The values defined through the 'Static Information' tab are listed.

  5. Column2 - Click [List of Values] to select the second Column that has to be printed on the bill/receipt for the selected sequence number. The listed values are system-defined.

  6. Static Legend - Click [List of Values] to select that Static value that has to be printed on the bill/receipt for second column. The values defined through the 'Static Information' tab are listed.

  7. Static Value - Click [List of Values] to select that Static Value that has to be printed on the bill/receipt for the second column. The values defined through the 'Static Information' tab are listed.

Service

  1. Column1 - Click [List of Values] to select a column name for a Service. The system-defined values are listed.

  2. Column [2-7] - Click [List of Values] to select a column names to describe the service details that are to be printed on the bill/receipt for a patient transaction in the facility. The system-defined values are listed.

  3. Summary By Print Group - Select the check box to print bill summary based on the print group. Clear the check box, if a summarized bill is not required for the respective bill/receipt format.

  4. Detail By Print Group Line - Select the check box to print a detailed bill based on the print group line. Clear the check box, if a detailed bill is not required for the respective bill/receipt format.

  5. Detail By Service - Select the check box to print a detailed bill based on the service. Clear the check box if a detailed bill print is not required for the respective bill/receipt.

Note:

You can only select either of the 'Summary By Print Group', 'Detail By Print Group Line' or 'Detail By Service' check boxes.

  1. Subtotal - Select the check box to print subtotal of columns associated to the service description. You cannot select the check box if 'Summary By Print Group' option is selected. Amount Summary is displayed in the detailed bill generated for an inpatient/outpatient based  on the 'Print Group Line' or 'Service'.

  2. Print Doctor - Select the check box to print the Doctor's name in the selected bill/receipt format. Clear the check box, if the Doctor's name is not to be printed.

  3. Print Doctor Specialty - Select the check box to print the Doctor's Specialty in the selected bill/receipt format. Clear the check box, if the Doctor's Specialty is not to be printed.

  4. Print Remarks for Misc Service instead of Description - Select the check box to print remarks for miscellaneous services recorded through the transaction functions 'Enter Request - OP' and 'Enter Request - IP'. Clear the check box to disable the user from recording 'Remarks' for Miscellaneous Services through the transaction functions.

Bill Details

  1. Seq No - Enter the sequence number in numeric format to select the hierarchy to display bill details.

  2. Column1 - Click [List of Values] to select first column name to be displayed in bill/receipt details. The system-defined values are listed.

End of Bill

Enter the values only if the format is corresponding to a receipt. You can leave the fields blank, if the format is valid for a bill.

  1. Seq No - Enter the sequence number in numeric format to select the hierarchy to display receipt details.

  2. Column1 - Click [List of Values] to select first column name to be displayed in receipt details. The system-defined values are listed.

  3. Static Value - Click [List of Values] to select a Static Value for the corresponding Column. The values defined through the 'Static Information' tab are listed.

  4. Justification - Select a value from the system-defined list. You can select the alignment to display receipt details at the end of the bill.
