User Group

Enter the following details to create a User Group:

  1. User Group – Enter the User Group in alphanumeric format with a maximum of 10 characters. This is a mandatory field.

  2. Long Description – Enter the Long Description in alphanumeric format with a maximum of 30 characters. This is a mandatory field.

  3. Short Description – Enter the Short Description in alphanumeric format with a maximum of 15 characters. This is a mandatory field.

  4. All Service Groups – Select All Service Groups and the selected User Group is able to access all the Service Groups.

  5. [<<Users for User Group] – Click [<<Users for User Group] to view the Users associated with the selected User Group.
