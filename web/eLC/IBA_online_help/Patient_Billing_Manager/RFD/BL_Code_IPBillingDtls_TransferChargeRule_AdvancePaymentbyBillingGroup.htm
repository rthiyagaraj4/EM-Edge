Advance Payment by Billing Group

Advance Payment by Billing Group tab enables you to fix an amount to be accepted as prepayment for a Bed Class in a Nursing Unit Type and the associated Billing Group. The prepayment amount is collected during patient registration and the amount is settled against the billing services.

 

Enter the following details to define a prepayment rule:

 

  1. Billing Group – Click [List of Values] to select a Billing Group. The values defined using the Billing Group function get listed. The prepayment rule is applicable for the selected Billing Group. This is a mandatory field.

  2. Description - The description for the selected Billing Group gets displayed.

  3. Nursing Unit Type - Click [List of Values] to select a Nursing Unit Type. The values defined using the Nursing Unit Type function in the Inpatient and Day Care Management module get listed. The prepayment rule is applicable for the selected Billing Group and the Nursing Unit Type. This is a mandatory field.

  4. Bed Class - Click [List of Values] to select a Bed Class. The values defined using the Bed Class function in the Inpatient and Day Care Management module get listed. The prepayment rule is applicable for the selected Billing Group, Nursing Unit Type and the associated Bed Class. This is a mandatory field.

  5. Description - The description for the selected Bed Class gets displayed.

  6. Category – Select a value from the list. The values are system defined value - Nationals, Unrestricted and Others.

  7. Nationals – Select Nationals, if the pre-payment rule is applicable for the citizens admitted to a relevant Bed Class in a Nursing Unit Type with the selected Billing Group.

  8. Unrestricted – Select Unrestricted, if the pre-payment rule is applicable for any patient admitted to a relevant Bed Class in a Nursing Unit Type with the selected Billing Group.

  9. Others – Select Others, if the pre-payment rule is valid for non-nationals.

  1. Pre Payment Amount - Enter the pre-payment in numeric format with seven digits before the decimal and two digits after the decimal. The amount specified indicates the total amount to be collected from the patient in advance. This is a mandatory field. An option to refund the pre-payment amount is added.

  2. Description - The description for the selected Nursing Unit Type gets displayed.
