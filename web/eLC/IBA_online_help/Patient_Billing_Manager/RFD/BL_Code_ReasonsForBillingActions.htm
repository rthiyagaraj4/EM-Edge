Enter the following details to define the Reasons for Billing Actions:

 

  1. Action Reason - Enter the Action Reason code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Description - Enter the description for the selected action code in alphanumeric format. This is a mandatory field.

 

Action Types for Reason Code

  1. Active - Select the check box to associate the action type to an action reason. The action reasons are valid for the selected billing transactions.

  2. Action Type - The Action Types are system defined. Select or clear the check boxes to enable or disable the reasons for the selected transactions in the facility.

 

Action Types

  1. Bill Adjustment - Select the check box, if the action reason is valid for bill adjustments. The action reasons with 'Bill Adjustment' check box as selected, get listed in the 'Bill Adjustment' function during authorization. Clear the check box, if the action reason is not valid for bill adjustments.

  2. Bill Level Discount - Select the check box, if the action reason is valid for bill level discounts. The action reasons with 'Bill Level Discount' check box as selected, get listed in the 'Adhoc Reason For Unbilled Trx - OP' and 'Adhoc Reason For Unbilled Trx - IP' functions respectively. Clear the check box, if the action reason is not valid for bill level discounts.

  3. Bill Exemption - Select the check box, if the action reason is valid for bill exemptions. The action reasons with 'Bill Exemption' check box as selected, get listed in the 'Bill Exemption' function during authorization. Clear the check box, if the action reason is not valid for bill exemptions.

  4. Cancel Adjustment - Select the check box, if the action reason is valid for bill adjustment cancellations. The action reasons with 'Cancel Adjustment' check box as selected, get listed in the 'Cancel Bill Adjustment' function during authorization. Clear the check box, if the action reason is not valid for adjustment cancellations.

  5. Bill Cancellation - Select the check box, if the action reason is valid for bill cancellation. The action reasons with 'Bill Cancellation' check box as selected, get listed in the 'Nullify/Cancel a Bill' and 'Nullify IP Bill' functions under 'OP Billing' and 'IP Billing' menus respectively. Clear the check box, if action reason is not valid for bill cancellations.

  6. Cancel Exemption -  Select the check box, if the action reason is valid for exemption cancellations. The action reasons with 'Cancel Exemption' check box as selected, get listed in the 'Cancel Bill Exemption' function under 'Bill Exemption/Adjustment' menu. Clear the check box, if action reason is not valid for exemption cancellations.

  7. Refund Cancellation -  Select the check box, if action reason is valid for refund cancellations. The action reasons with 'Refund Cancellation' check box as selected, get listed in the 'Cancel IP Refunds' and 'Cancel Refunds' functions under 'IP Cashier' and 'OP Cashier' menus respectively. Clear the check box, if action reason is not valid for refund cancellations.

  8. Receipt Cancellation - Select the check box, if action reason is valid for receipt cancellations. The action reasons with 'Receipt Cancellation' check box as selected, get listed in the 'Cancel IP Receipts' and 'Cancel Receipts' functions under 'IP Cashier' and 'OP Cashier' menus respectively. Clear the check box, if action reason is not valid for receipt cancellations.

  9. Batch Cancellation - Select the check box, if action reason is valid for credit card batch cancellations. The action reasons with 'Batch Cancellation' check box as selected, get listed in 'Credit Card Payment Entry' function under the 'Cancel' tab.  Clear the check box, if action reason is not valid for batch cancellations.

  10. Cancel Invoice - Select the check box, if action reason is valid for invoice cancellations. The action reasons with 'Invoice Cancellation' check box as selected, get listed in 'Cancel Invoice/Note/Credit Note' function in 'Invoice Management' menu. Clear the check box, if action reason is not valid for invoice cancellations.

  11. Installment Payment - Select the check box, if action reason is valid for installment payments. The action reasons with 'Installment Payment' check box as selected, get listed in 'Enter Credit Agreement' function in 'Deferred Cash Patients' menu. Clear the check box, if action reason is not valid for installment payment.

  12. Cancel Installment Payment - Select the check box, if action reason is valid for installment payment cancellation. The action reasons with 'Cancel Installment Payment' check box as selected, get listed in 'Cancel Credit Agreement' function in 'Deferred Cash Patients' menu. Clear the check box, if action reason is not valid for installment payment cancellation.

  13. Cancel Invoice Payment - Select the check box, if action reason is valid for invoice payment cancellations. The action reasons with 'Cancel Invoice Payment' check box as selected, get listed in 'Cancel Receipt for Payment Reference Doc' function in 'Invoice Management' menu. Clear the check box, if action reason is not valid for invoice payment cancellations.

  14. Bounced Cheque - Select the check box, if action reason is valid to record bounced cheque entries. The action reasons with 'Bounced Cheque' check box as selected, get listed in 'Enter Bounced Cheques' function in 'Bounced Cheque Activities' menu. Clear the check box, if action reason is not valid to record bounced cheque activities.

  15. Patient Reclassification - Select the check box, if action reason is valid for patient reclassification. The action reasons with 'Patient Reclassification' check box as selected, get listed in 'Classification Excl-Future Encounters' function in 'Financial Reclassification' menu. Clear the check box, if action reason is not valid for financial reclassification.

  16. Write-off Approval -  Select the check box, if the action reason is valid to approve write-off requests. The action reasons with 'Write-off Approval' check box as selected, get listed in 'Write-off Approvals/Rejections' and 'Generate Installment Write-off Requests' functions under 'Reminder Letter Process' and 'Deferred Cash Patients' menu respectively. Clear the check box, if action reason is not valid for Write-off approvals.

  17. Write-off Cancellation - Select the check box, if the action reason is valid to cancel approved write-off requests. The action reasons with 'Write-off Cancellation' check box as selected, get listed in 'Write-off Cancellation' function under 'Reminder Letter Process' menu. Clear the check box, if action reason is not valid for Write-off cancellations.

  18. Write-off Request - Select the check box, if the action reason is valid while placing write-off requests. The action reasons with 'Write-off Request' check box as selected, get listed in 'Generate Write-off Requests' and 'Generate Installment Write-off Requests' functions under 'Reminder Letter Process' and 'Deferred Cash Patients' menu respectively. Clear the check box, if action reason is not valid for Write-off requests.

  19. Write-off Rejection - Select the check box, if the action reason is valid to reject write-off requests. The action reasons with 'Write-off Rejection' check box as selected, get listed in 'Write-off Approvals/Rejections' and 'Write-off Installment Approval/Rejn' functions under 'Reminder Letter Process' and 'Deferred Cash Patients' menu respectively. Clear the check box, if action reason is not valid for Write-off rejections.

  20. Cancel Discharge Clearance - Select the check box, if the action reasons are valid for discharge clearance cancellation. If the check box is selected, the values are listed in the 'Record Discharge Clearance' function while validating the user. Clear the check box, if the action reason is not applicable for discharge clearance cancellation.

  21. Discharge Clearance W/O Bill - Select the check box, if the action reason is valid for discharge clearances without bill. If the check box is selected, the values get listed in the 'Record Discharge Clearance' function while validating the user. Clear the check box, if the action reason is not valid to record discharge clearance for inpatients' without bill.

  22. Override Reimbursement - Select the check box, if action reasons are valid for overriding reimbursable amount of a patient class. The selected reasons get listed in 'Modify IP Bill Reimbursable Amt' and 'Modify OP Bill Reimbursable Amt'. Clear the check box if the action reasons are not valid for reimbursements.

  23. Downtime Entries - Select the check box, if action reasons are valid for downtime entries. The action reasons are listed in 'Enter OP Receipts/Refunds' and 'Enter IP Receipts/Refunds' function. Clear the check box, if the action reasons are not valid for downtime entries.

  24. Indirect Service Cancellation - Select the check box, if action reasons are valid for indirect service cancellation. Clear the check box, if action reasons are not valid for indirect service cancellation.

  25. Service Level Discount - Select the check box, if the action reason is valid for service level discounts. The selected reasons get listed in 'Adhoc Discount for Unbilled Trx - IP', 'Adhoc Discount for Unbilled Trx - OP', 'Cash Bill-Cum-Receipt' and 'Discharge Bill-Cum-Receipt' function when you right-click on discount amount to issue service level discounts. Clear the check box, if the action reason is not valid for service level discounts.

  26. Cancel Package - Select the check box, if the reason is valid for package cancellation. The selected reasons get listed in 'Cancel Package' function when you cancel the package. Clear the check box, if the reason is not valid for package cancellation.

  27. Cancel Discharge Clearance - Select the check box if the reason is valid for cancelling discharge clearance. The selected reasons get listed in 'Reprint/Cancel Discharge Clearance' function when you cancel the discharge clearance for an inpatient. Clear the check box, if the reason is not valid for discharge clearance.

  28. Service Override Incl/Excl - Select the check box if the reason is valid to override service inclusions/exclusions. The selected reasons are listed while overriding a service inclusion/exclusion in transaction functions.
