Enter the following details to associate a billing service to the blood products:

 

  1. Order Catalog – Click [Execute Query] or press [F8] key to view the Order Catalog details. The description for the respective Order Catalog gets displayed. The values defined using the Order Catalog function in the Order Entry and Result Reporting module gets displayed.

  2. Product Code - The Product Code associated to the Order Catalog gets displayed. The product code generated for the selected Order Catalog using the Blood Transfusion or the Blood Donor module gets displayed.

  3. Pooled Product - The check box is selected, if the product is selected as a pooled product in the Blood Donor module.

  4. Charging Level for Pooled Product – Select a value from the list. The values are system defined - Product Level Charging and Product Constituent Level Charging. If the product is not a pooled product, it can be charged only at the product level. If the product is defined as a pooled product, product constituent level charging is enabled.

  5. Collection Centre - The blood samples collected from the donor is tested and sent to the Collection Centre after systematic examination. The samples collected are stored in the Collection Centre. The values defined using the function Collection Centre/Supplier in the Blood Donor module gets listed. The samples collected in the Collection Centre are charged based on the prices defined for the billing service associated.

  6. Description - The description for the selected Collection Centre gets displayed.

  7. Service Type – Select a Service Type from the list. The values are system defined - Service and Panel.

  8. Service – Select Service to associate a Billing Service to the Collection Centre. The charges are calculated based on the prices defined for the Billing Service.

  9. Panel - Select Panel to associate a Billing Panel to the Collection Centre. The charges are calculated based on the prices defined for the Billing Panel.

  1. Blng Serv Code – Click [List of Values] to select a valid Billing Service or a Billing Panel. The orders placed for a blood product associated to an Order Catalog is charged based on the prices defined for the selected Billing Service or Billing Panel.

  2. Description - The description for the selected Billing Service or Billing Panel code gets displayed.

  3. OP/EM - Select the checkbox to configure billing details for a blood product in  a Panel order. Only one check box can be selected.
