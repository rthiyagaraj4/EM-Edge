Enter the following details to define Support Documents:

 

  1. Support Doc Type Code - Enter the Support Document Type Code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Long Description - Enter the long description for Support Document Type in alphanumeric format of 100 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description for Support Document Type in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  4. Doc Numbering Applicable - Select the check box, if Document Numbering is applicable. Clear the check box, if Document Numbering is not applicable.

  5. Doc Series - Enter the Document Series in alphanumeric format of five digits maximum length. This field is mandatory, if the check box 'Doc Numbering Applicable' is selected.

  6. SS Govt Letter Ref - Enter the SS Government Letter Reference in alphanumeric format of 30 characters maximum length.
