Panel Details

A Panel is a group of individual Billing Services clubbed together. When a Panel is associated to a patient, all the services under the panel are considered valid for transactions. The services associated to the Panel become a part of the patient transaction and the bill is generated for the services under the Panel. You can define the Panel in such a way that either the additional services can be associated with the Panel or existing services can be deleted from the Panel during transactions.

 

Enter the following to set up Panel Details:

 

  1. Panel Code - Enter the Panel Code in alphanumeric format of two characters minimum length and 10 characters maximum length. The first two digits of the code have to be a valid Panel Category code. The Panel Category has to be defined initially to define a Panel Code. This is a mandatory field.

  2. Panel Category - The Panel Category associated to the respective Panel Code gets displayed.

  3. Long Description - Enter the Long Description for the Panel Code in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  4. Short Description - Enter a Short Description for the Panel Code in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  5. Include/Remove Additional Service During Service Entry – Select the check box to either include or exclude any individual services from the panel. Additional services can be associated to the panel and patient using the function, Enter Requests – OP or Enter Request – IP. You are permitted to choose a Billing Panel and associate it to a patient, on selecting Panel in the transaction functions. Clear the check box and additional services cannot be associated to a Panel during transaction functions.

 

Panel Constituents

  1. Srl. No - Enter the Serial number in numeric format of two digits maximum length. This is a mandatory field.

  2. Billing Service – Click [List of Values] to associate valid Billing Services to a Panel. The values defined in the Billing Services function get listed. This is a mandatory field.

  3. Description - The Description for the respective Billing Service gets displayed automatically.

  4. Quantity - Enter the quantity in numeric format of eight digits, where six digits have to appear before the decimal and the other two digits have to appear after the decimal point. The Billing Service price is applicable for the selected Quantity.  An example for the Quantity is 999,999.99. This is a mandatory field.

  5. UOM – The Unit of Measure is set to ‘Number’, by default, when the Quantity is set. You cannot edit the Unit of Measure as it is set by default.

  6. Qty Modifiable – Select the check box, if the Quantity is modifiable. You are able to update the quantity for the service during the transactions, if the check box is selected.
