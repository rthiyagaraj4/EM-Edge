Admission Day Bed Charge Discount Rule

The Admission Day Bed Charge Discount Rule tab enables you to define charging rule, if the patient is admitted and discharged from the facility on the same day. The charging rule is applicable for the bed occupied, doctor's round, consumables used for patient's treatment, Workup and ICU charge. You can set the discount type and the discount value on the charge elements based on which the discount amount is calculated during patient discharge from the facility. This enables you to set parameters to charge a patient, if the patient is discharged from the facility on the same day, but exceeds the grace period set by the facility. This rule is enabled, if the patient is discharged within 24 hours and the check box 'Validate Discharge within 24 HRS' is cleared.

 

Enter the following details:

  1. Bed Type – The Bed Type is displayed as ‘**’, by default. The discount rule is applicable for all the beds in the Nursing Unit. Click ‘Create Record’ icon to define the discount rule for a selected bed in the facility. Click [List of Values] to select a Bed Type.

  2. Facility – The name of the logged in Facility is displayed, by default.

 

Hours Stay

  1. From – To – Enter the total number of days in numeric format. The beginning day has to start with ‘0’.

 

Charge Elements

The Charge Elements define the elements that are to be charged during discharge.

  1. Charge Element – The system defined Charge Elements are ‘Bed Charge’, ‘Doctor Round Charge’, ‘Consumable Charge’, ‘Workup Charge’ and ‘ICU Charge’. The charges for these elements during a normal patient discharge are calculated based on the charge type and the discount type selected.

  2. Charge Type – Select a value from the list. The values are system defined – ‘As Per Policy Setup’, ‘Charge to Patient Only’ and 'No Charge'. The elements are charged based on the policy coverage limit specified, if the value is selected as ‘As Per Policy Setup’. The elements are charged on the policy coverage limit specified, if the value is selected as ‘As Per Policy Setup’ and the patient is insured. If the patient is a non-insurance payer, the amount for the element is charged on the patient on the discharge day for normal discharge. Select 'No Charge', if the respective element is not to be charged.

  3. Discount Type – Select a value from the list. The values are system defined – Percentage, Amount and Not Applicable. Percentage indicates that the percentage of the actual amount is deducted on the element for normal discharge. Amount indicates that the amount mentioned is deducted from the actual amount for the element during normal discharge. Select Not applicable, if discount is not applicable for the selected element.

  4. Discount Value – Enter the discount value in numeric format to indicate the amount to be deducted for the elements during discharge.

  5. Posting Mode - Select a value from the list to indicate the posting mode. System-defined values are 'As Discount' and 'As Net'. Select 'As Discount' to post the values as discount. For Example, if the charge computed is 100/- and discount amount  given is 5/-, then while posting, the amount is displayed as 100/- and the discount amount is displayed as 5/-. Select 'As Net' to post the values as Net Amount. For Example, if the amount computed is 100/- and discount amount given is 5/-, then while posting, the net amount is displayed as 95/-.
