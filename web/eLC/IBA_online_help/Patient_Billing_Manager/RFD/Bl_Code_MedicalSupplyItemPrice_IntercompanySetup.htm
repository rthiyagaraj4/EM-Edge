Intercompany Setup

You can define if the medical item is valid for interfacility dependencies during the selected period.

 

Enter the following details to define intercompany setups:

  1. From Facility – Select a Facility from the list. The facilities associated to the user get listed. The facilities that are defined through the 'Facility' function in System Manager module are listed. This is a mandatory field.

  2. Intercompany to Facility – Click [List of Values] to associate a referral facility. The facilities that are defined through the 'Facility' function in System Manager module are listed. This is a mandatory field.

  3. Valid From - Click Calendar function to select a date. The date selected has to be equal to or greater than the system date. This is a mandatory field.

  4. Valid To - Click Calendar function to select the date to indicate the due date. The selected date has to be greater than the 'Valid From' date.  This is a mandatory field.
