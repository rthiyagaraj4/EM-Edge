Enter the following details to define GL Parameters:

 

  1. Current Month/Year – Enter the current month and the year. The year and the month will be associated with month end processing. The current processing month and year signifies the accounting period for which the GL transactions are being processed. This is a mandatory field.

 

Posting Types

  1. Reverse – Select the check box to define the journal vouchers in reversal type throughout the entire General Ledger module. The check box is cleared, by default. If journal voucher is of Reversal Type, the journal voucher is reversed in the subsequent processing period.

  2. Prior - Select Prior to allow previous period posting process. The account balances for months already closed within the current financial year is updated with new balances. This cannot be extended across financial years.  

 

Summary A/C Update

  1. Online – Select Online to select online transactions. The summary account update is processed online.

  2. Batch – Select Batch to process the transactions in a Batch.

 

Account Code Elements

  1. Department – Select the check box, if the Department Code is a part of valid Posting Account. Department codes are created, if they do not form a part of the account codes defined in the General Ledger module.

  2. Product – Select the check box, if the Product is a part of the valid Posting Account.

  3. Sub-ledger – Select the check box, if the Sub-ledger is a part of the valid Posting Account.

 

Miscellaneous Accounts  

  1. P&L A/c for the Year – Click [List of Values] to select a Profit & Loss Account. The selected account code is affected with the month end "profit/loss for the month" transfer transaction automatically generated by the month end processing function.

  2. P&L Suspense A/c – Click [List of Values] to select a Profit & Loss Suspense Account. The selected account code is affected with the month end "profit/loss for the month" transfer transaction automatically generated by the month end processing function.

  3. Retained Earnings A/c – Click [List of Values] to select a Retained Earnings Account. The selected account code is not affected by the month end profit/loss transfer transactions.

 

Length of Account Code Elements  

  1. Main Account – Click [List of Values] to select a Main Account. The main account codes are selected when the account code is created in the Chart of Accounts. This is a mandatory field.

  2. Sub Account – Click [List of Values] to select a Sub Account. The sub account codes are selected when the account code is created in the Chart of Accounts. This is a mandatory field.

  3. Product Group – Click [List of Values] to select a Product Group. The product group codes are selected when the account code is created in the Chart of Accounts. This is a mandatory field.

  4. Product Line – Click [List of Values] to select a Product Line. The combination of Product Group and Product Line are selected when the account code is created in the Chart of Accounts. This is a mandatory field.

  5. Sub-Ledger Group – Click [List of Values] to select a Sub-Ledger Group. The sub-ledger group codes are selected when the account code is created in the Chart of Accounts. This is a mandatory field.

  6. Sub-Ledger Line - Click [List of Values] to select a Sub-Ledger Line. The combination of Sub-Ledger Group and Line are selected when the account code is created in the Chart of Accounts. This is a mandatory field.

 

Default Values  

  1. Department Code/Desc – Click [List of Values] to select a Department Code. The description for the selected Department Code appears in the adjacent text box. This is a mandatory field.

  2. Product Code/Desc – Select [List of Values] to select a Product Code. The description for the selected Product Code appears in the adjacent text box. This is a mandatory field.

  3. Sub-Ledger Code/Desc – Select [List of Values] to select a valid Sub-Ledger Code. The description for the relevant Sub-Ledger Code appears in the adjacent text box. This is a mandatory field.

  4. Posting Account Delimiter - Enter the value for Posting Account Delimiter. The delimiter is used to display the Posting Account with delimiter distinction between Group and Lines. This is a mandatory field.

  5. Bank Folder Name – Enter the Bank Folder Name in alphanumeric format. This is associated with Bank Statement upload from flat files received from Bank.
