Enter the following details to process interim bills:

 

  1. Ward Code - Click [List of Values] to select the Ward Code. The Wards defined in the Inpatient and Day Care Management module gets listed.

  2. Payable Type - Select a valid Payable Type from the list. The values are system defined - 'Payer', 'Patient' and 'Both'.

  3. Cash Counter - The Cash Counter to which the user has logged in gets displayed.

  4. Patient ID - Enter the patient ID in alphanumeric format of 10 characters maximum length. You can search the interim bills of the selected patient on entering the patient ID.

  5. Payer - Click [List of Values] to select the Payer. This field is enabled for selection only when the 'Payable Type' is selected as 'Payer'.

  6. Special Clinic - Select the check box to view the bills generated for the inpatients' in the special clinics. Clear the check box to view and to process the bills for the inpatients in normal clinics.

  7. Episode ID [From-To] - Click [List of Values] to select the range of Episode IDs. The interim bills processed for the selected episodes are listed.

  8. Payable Amt Exceeding - Enter the amount in numeric format. The records where the outstanding amount is exceeding the amount recorded gets displayed.

  9. Not Generated for Days - Enter the number of days in numeric format. The number of days for which the bills were not generated are listed.

  10. [Search] - Click [Search] to search for the records matching the search criteria.

 

Current Inpatient

  1. Select All - Select the check box to select all the records. Clear the check box to select individual records.

The patient bills along with the Patient ID and name, admission number and date, billing group, payer, policy, unbilled amount, outstanding amount, total amount, episode type, ward code, bed number and the room number are displayed.

  1. Adjust Deposit - Select the check box to adjust the outstanding amount against the deposit collected during the admission. Clear the check box, if the adjustment is not to be made on the deposit amount collected during admission.

  2. Adjust Prepayment - Select the check box to adjust the outstanding amount against the prepayment amount collected during the admission. Clear the check box, if the adjustment is not to be made on the prepayment amount collected during admission.
