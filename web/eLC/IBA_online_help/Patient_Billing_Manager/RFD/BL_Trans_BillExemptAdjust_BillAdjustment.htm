Enter the following details to adjust the bill amount:

 

  1. Document Ref – Enter the Document Reference details in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  2. Total Amount – The total amount gets displayed automatically based on the adjustment amount against the outstanding bills.

  3. Date – Click Calendar function to select a transaction date. This is a mandatory field.

  4. Reason – The reason selected while logging in to adjust the bill details get displayed, by default.

  5. Remarks – Enter the Remarks in alphanumeric format of 100 characters maximum length.

  6. Document No – The Document Type selected for Bill Adjustment in the BL Parameters function gets displayed. The document number is system generated.

  7. Date - The system date and time gets displayed automatically.

  8. [Authorize] – Click [Authorize] to login and to adjust the amount for the outstanding bills.

    Enter the following details:

    1. User ID – Enter the User name in alphanumeric format. This is a mandatory field.

    2. PIN No – Enter the PIN number for the selected patient. This is a mandatory field.

    3. Reason – Click [List of Values] to select a Reason for exemption. The reasons defined using the function ‘Reasons for Billing Actions’ in the Masters with the ‘Bill Exemption’ check box as selected get listed. This is a mandatory field.
