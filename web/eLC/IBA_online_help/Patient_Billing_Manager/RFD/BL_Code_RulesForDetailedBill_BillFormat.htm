Service Grouping

The Bill Format tab enables you to define the bill format for either a Payer Group or a Billing Group.

 

Enter the following details to define a Bill Format:

 

  1. Option – Select either Payer Group or Billing Group from the list. The values are system defined – 'Payer Group' and 'Billing Group'.

  2. Payer Group – Select Payer Group to define the bill format for a Payer Group associated to the Facility.

  3. Billing Group – Select Billing Group to define the bill format for the Billing Groups associated to the Facility.

  1. Payer/Billing Group – Click [List of Values] to select a Payer or a Billing Group. The list gets populated with the Payer details or the Billing Group details accordingly. The values defined using the function Customer under the ‘AR Setup’ menu gets listed when the Option is selected as ‘Payer Group’. The values defined using the Billing Group function gets listed when the Option is selected as ‘Billing Group’. This is a mandatory field.

  2. Patient Class – The Patient Class for the Facility along with the Description gets listed. The description of the corresponding Patient Class is displayed.

 

Bill Formats By Payer

  1. Payer – Click [List of Values] to select a Customer. The bill format defined is applicable for the selected Payer.

  2. Description – The Description for the selected Customer gets displayed.

  3. Group Indicator – Select a Group Indicator from the list. The values differ based on the ‘Option’ selected.

 

If the Option is selected as Payer Group, the following values get listed for Group Indicator:

  • Enterprise Material Group – Select Enterprise Material Group to define the bill format for the selected Material Item group.

  • Enterprise Service Group - Select Enterprise Service Group to define the bill format for the selected Enterprise Service group.

  • Payer Material Group – Select Payer Material Group to define the bill format for the selected Payer Material group.

  • Payer Service Group - Select Payer Service Group to define the bill format for the selected Payer Service group.

 

If the Option is selected as Billing Group, the following values get listed for Group Indicator:

  • Enterprise Material Group – Select Enterprise Material Group to define the bill format for the selected Material Item group.

  • Enterprise Service Group - Select Enterprise Service Group to define the bill format for the selected Enterprise Service group.

  1. Code – Click [List of Values] to select a Code. The Code gets listed based on the value selected for the Group Indicator. The values are system defined.

  2. If the Group Indicator is set to Enterprise Material Group, the Material Items associated to the Facility gets listed. The defined bill format is applicable for the selected Material Item.

  3. If the Group Indicator is set to Enterprise Service Group, the Service Groups associated to the Facility gets listed. The defined bill format is applicable for the selected Service Group.

  4. If the Group Indicator is set to Payer Material Group, the Payer Material Groups defined using the Payer Material Group function under the Billing Services menu gets listed.

  5. If the Group Indicator is set to Payer Service Group, the Payer Service Groups defined using the Payer Service Group function under the Billing Services menu gets listed.

  1. Description – The Description for the selected Code gets displayed.

  2. Print Level Ind – Select a Print Level Indicator from the list. The values are system defined – “Summary” and “Detail”.

  3. Summary – Select Summary to print only an abstract of the bill. On selecting the value as 'Summary', the 'Code Print Indicator' is displayed as 'Don't Print', by default.

  4. Detail – Select Detail to print a detailed bill along with the Payer Service details.  On selecting the value as ' Detail', the 'Code Print Indicator' can be set as either 'Service Item' or 'Mapping Code'.

  5. By Group - Select 'By Group' to print the bill details by service or material group.

  6. By Line - Select 'By Line' to print the bill details based on the line items associated to a service or material group.

  1. Code Print Indicator - Select a value from the list. The system-defined values are 'Service Item', 'Mapping Code' and 'Don't Print'.

  2. Service Item - Select Service Item to display the service item details in a detailed bill. This is valid if the 'Print' value is selected as 'Detail'.

  3. Mapping Code - Select Mapping Code to display service/material item mapping code in a detailed bill. This is valid if the 'Print' value is selected as 'Detail'.

  4. Don't Print - Select Don't Print, if the service/material item details are not to be printed in a detailed bill.  The value is set as 'Don't Print' when the 'Print' value is selected as 'Summary', by default.

 

Bill Format Details

If the Print Level Indicator is set to ‘Detail’ for a Group Indicator, a detailed format is defined for the bills to be printed for the individual services or material items.

 

  1. Indicator – Select an Indicator from the list. The indicator is enabled only when the Group Indicator is selected as ‘Enterprise Service Group’ or ‘Payer Service Group’.

  2. If the Group Indicator is selected as ‘Enterprise Service Group’, the Indicator is set to Enterprise Service Group/Line, by default.

  3. If the Group Indicator is selected as ‘Payer Service Group’, the Indicator is set to Payer Service Group/Line, by default.

  1. Code – Click [List of Values] to select and associate a Print Group code. The values in the list differ based on the Indicator selected.

  2. If the Indicator is selected as ‘Enterprise Service Group/Line’, the values defined using the Enterprise Service Group/Line function gets listed.

  3. If the Group Indicator is selected as ‘Payer Service Group/Line’, the values defined using the Payer Service Group function gets listed.

  1. Description – The Description for the selected Print Group code gets displayed.

  2. Print Level Ind – Select a Print Level Indicator from the list. The values are system defined - ‘Summary’ and ‘Detail’.

  3. Summary - Select ‘Summary’ and the Payer Service Line details are not printed on the bill.

  4. Detail – Select ‘Detail’ and the Payer Service details are printed on the bill.

  1. Code Print Indicator - Select a value from the list. The system-defined values are 'Service Item', 'Mapping Code' and 'Don't Print'.

  2. Service Item - Select Service Item to display the service item details in a detailed bill. This is valid if the 'Print' value is selected as 'Detail'.

  3. Mapping Code - Select Mapping Code to display service/material item mapping code in a detailed bill. This is valid if the 'Print' value is selected as 'Detail'.

  4. Don't Print - Select Don't Print, if the service/material item details are not to be printed in a detailed bill.  The value is set as 'Don't Print' when the 'Print' value is selected as 'Summary', by default.
