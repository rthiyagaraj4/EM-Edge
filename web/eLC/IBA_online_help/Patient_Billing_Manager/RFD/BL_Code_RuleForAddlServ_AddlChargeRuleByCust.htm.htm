You can define additional tax on a Payer/Corporate bill for the services utilized by a patient from the facility. You can select and associate a Service Group, Service Classification or Billing Service to the Payer to define additional tax. The additional tax defined through this tab is not valid on a Patient's bill. For a Corporate bill, additional service charge is valid only for Health check-ups.

 

Enter the following details to define additional charge rule for a Payer/Corporate:

 

  1. Customer Code - The Code is displayed as '**' by default. The values that are defined through the 'Customer' function are displayed. Use the [Upward] or the [Downward] arrow keys in your keyboard to Navigate to the next or previous records. The description of the respective Customer Code is displayed.

Services

  1. Service Indicator - Select a value from the list to define additional charge for a Payer/Corporate bill. The system-defined values are 'Service Classification', 'Service Group' and 'Billing Service'.

  2. Code - Click [List of Values] to select a Code. The values for 'Code' differ based on the Service Indicator selected.

  3. Short Description - The description for the selected Code is displayed.

  4. Effective From - Click the Calendar function to select the date. Additional tax is applicable on the service from the selected date. The date selected has to be greater than the system date.

  5. To - Click the Calendar function to select the date. Additional tax is applicable on the service till the selected date. The date selected has to be greater than the 'Effective From' date.

  6. Rule Applicable - Select the check box to define additional service tax on the respective Service Indicator. Clear the check box to exclude the Service Indicator from additional service charge.

Applicable Rules

  1. Rule Code - Click [List of Values] to select a Rule Code. The 'Rule Code' defined through the 'Additional Service Charge Rule' tab with the 'Rule Type' as 'Main' are listed. Additional Charge on the Payer/Corporate bill is calculated based on the additional charge defined for the respective 'Service Indicator'.

  2. Description - The description of the selected Rule Code is displayed.

  7. list Rule Coverage – Select the value from dropdown list to add or split the tax component. The listed values are ‘As Per Rule’, ‘Patient Coverage’ and ‘Payer Coverage’.

  1. As Per Rule – If you select this value, the rule is split as per the policy.

  2. Patient Coverage - If you select this value, the tax component of both 80% and 20% is added patient portion.

  3. Payer Coverage - If you select this value, the tax component of both 80% and 20% is added payer portion.
