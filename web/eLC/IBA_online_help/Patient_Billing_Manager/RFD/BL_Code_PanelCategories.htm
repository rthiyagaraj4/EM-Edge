Panel Categories

Billing Panel is a "packaged" service, which contains a number of individual billing services. The billing services clubbed together forms a Billing Panel. You have to define the Panel Categories initially and associate individual panels to the panel category. You are able to define the Panel Details only if the Panel Categories are defined. This functionality is also used to insert a new panel and its details or to change the details of an existing Panel Category or to query on the details of an existing Panel Category.

 

Enter the following to create Panel Categories:

 

  1. Panel Category - Enter Panel Category in alphanumeric format of two characters maximum length. This is a mandatory field.

  2. Long Description - Enter the Long Description in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description - Enter a Short Description about the Billing Service in alphanumeric format of 15 characters maximum length. This is a mandatory field.
