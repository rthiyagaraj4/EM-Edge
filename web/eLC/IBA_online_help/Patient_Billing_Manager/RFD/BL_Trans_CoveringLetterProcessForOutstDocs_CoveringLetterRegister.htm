Enter the following details to generate the Covering Letter Register:

 

  1. Report Type – Select a value from the list. The values are system defined.

  2. • Covering Letter - Details – Select Covering Letter – Details, if a detailed Covering Letter has to be generated.

  3. • Covering Letter - Summary - Select Covering Letter – Summary, if a summarized Covering Letter has to be generated.

  4. • Covering Letter by Outstanding Docs - Select Covering Letter by Outstanding Docs, if a Covering Letter has to be generated.

  1. Covering Letter Date [From–To] – Click Calendar function to select the date. The Covering Letters generated between the selected date range get displayed in the report.

  2. Covering Letter No [From–To] – Enter the Covering Letter Number. The Covering Letter details generated for the selected range of Covering Letter Numbers get displayed.  

  3. Billing Group [From–To] – Click [List of Values] to select a range of Billing Groups. The Covering Letter details generated for the range of Billing Groups specified get displayed.

  4. Customer [From–To] - Click [List of Values] to select a range of Customers. The Covering Letters generated for the range of Customers get displayed.

  5. Patient ID [From–To] – Click [List of Values] to select a range of Patient IDs. The Covering Letters generated for the specified range of Patient IDs get listed.  

  6. Group Code [From–To] – Click [List of Values] to select a range of Group Codes. The Covering Letters generated for the range of Group Codes specified get listed.

  7. Module ID – Select a module from the list. The listed values are Both, Patient Billing and Accounts Receivable. The Covering Letters generated in the selected module gets listed.

  8. Episode – Select an Episode Type and enter the Episode ID to view the Covering Letter details. The values are system defined – “Emergency”, “Outpatient”, “Daycare”, “Inpatient”, “External” and “All”.
