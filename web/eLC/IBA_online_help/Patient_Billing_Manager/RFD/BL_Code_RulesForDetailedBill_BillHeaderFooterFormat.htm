Bill Header/Footer Format

The Bill Header/Footer Format tab enables you to define format for a bill header and footer. The format selected in the 'BL Parameters' function gets displayed. Option is also available to load the default format.  System displays Hospital Details, Patient Details, Bill Details, Customer Details and Text Formatting list values in the text field in each tab. Right-click to select a value from the list. Various formats with several values are displayed on selecting a value from the list. Double-click on a value to select it.

The format lists styles to format the legends in header, footer and body section of a receipt format.

The format lists legends related to Customers'.  

 

The format lists legends for bill generation transactions. A Reference ID is generated for bills, if 'Print Bill Using' value is selected as 'Format2' in 'BL Parameters' function. The Cash Counter, Document Number, Document Type Code, Episode Type, Patient Category, Specialty and Year gets displayed for Bill Reference. The mentioned bill reference ID components are valid for Inpatient, Daycare, Outpatient, Emergency and External patient class. This is applicable for Title, Header, Bill Footer and Credit Bill Footer.

If Bill and Receipt is printed as a single document, Bill Reference ID is printed instead of Receipt Reference ID. If there is a Single Reference ID for multiple entries of receipts, the reference ID is printed as asterisks to indicate multiple entries.

The format lists patient demographic details.

The format lists complete information of the hospital and demographic details.

 

Enter the following details to define a Bill Header and Footer Format:

 

Outpatient/Emergency/Referral

Title

  1. Load the default Setup - Select the check box to load the pre-defined template. A confirmation message 'Existing Setup Will be Replaced, Continue?' is displayed on selecting the check box, . Click [Yes] to load the default template or [No] to cancel the process.

  2. Legend for Original Bill - Enter the legend for original bill in alphanumeric format. The value recorded is displayed in the bill during bill generation.

  3. Legend for Reprinted Bill - Enter the legend for reprinted bills in alphanumeric format. The value recorded is displayed in the bill during bill reprints.

  4. Line [1-4] - Enter the values that has to appear in the title of the bill. You can also record the data using the pre-defined formats that are available. Right-click on each field to select a value from the pre-defined format. The options available are Hospital Details, Patient Details, Bill Details, Customer Details and Text Formatting. Select a value and the pre-defined values recorded for the selected value are listed. Double-click on the value to select it.  The selected value is displayed in the respective field.

Outpatient/Emergency/Referral

  1. Cash Bill with Receipt - Enter the legend to be printed for cash bill with receipts valid for outpatient/emergency/referral patients.

  2. Credit Bill - Enter the legend to be printed for credit bills valid for outpatient/emergency/referral patients.

  3. Without Receipt - Enter the legend to be printed for bills without receipts valid for outpatient/emergency/referral patients.

Additional Line

  1. Cash Bill - Enter the additional legends that are to be printed for Cash bills.

  2. Credit Bill - Enter the additional legends that are to be printed for Credit bills.

The options available are Hospital Details, Patient Details, Bill Details, Customer Details and Text Formatting. Select a value and the pre-defined values recorded for the selected value are listed. Double-click on the value to select it.  The selected value is displayed in the respective field.
