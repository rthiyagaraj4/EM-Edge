Enter the following details to query and cancel the payment reference document:

 

  1. Payment Reference Doc - The payment reference document number is displayed on clicking [Execute Query]. Enter the Payment Reference Document number in alphanumeric format of 30 characters maximum length to search for a specific record.

  2. Pmnt Ref Date - The payment reference date is displayed. Click the Calendar function to select the Payment Reference Date and search for a specific record.

  3. Receipt Amount - The amount settled for the services during receipt generation is displayed. Enter the receipt amount to search for a specific record.

  4. Receipt Option - The receipt option is displayed based on the value selected during receipt generation. Select the value as 'Invoice-wise' or 'Consolidated by Customer' to search for a specific record.

  5. Payer Name - The Payer Name recorded during receipt generation is displayed. Enter the payer name in alphanumeric format of 100 characters maximum length to search for a specific record.
