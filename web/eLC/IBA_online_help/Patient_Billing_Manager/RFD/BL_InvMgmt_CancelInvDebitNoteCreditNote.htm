Enter the following details to query the records:

 

  1. Customer - Click [List of Values] to select the Customer. The values defined using the 'Customer' function in the AR Setup menu get displayed. The description for the selected Customer gets displayed.

  2. Policy Type - Click [List of Values] to select the Policy Type. The values defined using the 'Policy Type' function in the Insurance Management module get displayed. The description for the selected Policy Type gets displayed.

  3. Patient ID - Click [List of Values] to select the Patient ID. The patient details for whom the invoice, debit notes or the credit notes are generated in the selected facility get listed. You can select the patient from the list to cancel the details. The age of the patient, gender and the National ID gets displayed on selecting the Patient ID.

  4. Discharge Date [From-To] - Click the Calendar function to select the Discharge Date. The list of patients' discharged between the selected date range get displayed.

  5. Episode Type - Select the respective check box to select the Episode Type. You can select the records of the patient based on the Episode Type selected.

  6. All - Select the check box to select all the Episode Types. Clear the check box to select a specific Episode Type. The check box is selected, by default.

  7. Outpatient - Select the check box to select the Outpatients only. Clear the check box to exclude the Outpatient records. The check box is selected, by default.

  8. Emergency - Select the check box to select the Emergency patients only. Clear the check box to exclude the Emergency patient records. The check box is selected, by default.

  9. Daycare - Select the check box to select the Daycare only. Clear the check box to exclude the Daycare patient records. The check box is selected, by default.

  10. Inpatient - Select the check box to select the Inpatients only. Clear the check box to exclude the Inpatient records. The check box is selected, by default.

  11. External - Select the check box to select the External only. Clear the check box to exclude the External patient records. The check box is selected, by default.

The details are filtered and displayed based on the search criteria specified.
