Interface to GL

Enter the following details to interface the Settlement Type to GL codes:

 

  1. Settlement Type – The Settlement Type selected under the Settlement Type tab gets displayed.

  2. Slmt A/C Int – Click [List of Values] to select and to associate a valid GL Interface code. The code has to begin with "C" to indicate Cash related accounts. This is a mandatory field.

  3. Description – The Description for the selected GL Interface Code gets displayed.

  4. Eff. From Date – Click Calendar function to select a date. The date indicates the day from when the GL interface codes are effective for the settlement type. The date entered has to be greater than the system date and should not overlap with the existing definitions.

  5. Eff. To Date – Enter the date until when the GL Interface codes are effective for the settlement type. The date entered must be greater than the effective from date.
