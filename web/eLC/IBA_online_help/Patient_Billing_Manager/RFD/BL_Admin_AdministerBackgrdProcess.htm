Enter the following details to manage the background process:

 

  1. Facility - The logged in facility gets displayed, by default.

  2. User - Click [List of Values] to select the user. The processes executed during the selected user's shift gets listed.

  3. Process Key - Enter the Process Key to view the processes executed with respect to the selected process key.

  4. Date [From-To] - Click the Calendar function to select the date. The processes executed during the selected date gets listed.

  5. Status - Select a Status from the list. The values are system defined - 'On Error', 'Processed' and 'Not Processed'.

  6. Patient - Click [List of Values] to select a patient from the lookup.

  7. Encounter Id - Enter the Encounter ID of the selected patient to search for the background processes.

  8. [Search] - Click [Search] to search for the records matching the search criteria.

  9. [Purge] - Click [Purge] to delete the selected records.

  10. [Re-Process] - Click [Re-Process] to process the records where an error was encountered.

  11. Select All - Select the check box to select all the records.

  12. User ID - The name of the user during whose shift the process was executed gets displayed.

  13. Process Key - The Process Key which is system generated gets displayed.

  14. Process Date Time - The Date and Time when the process was executed gets displayed.

  15. Patient ID - The Patient ID is displayed.

  16. Encounter ID - The Encounter ID is displayed.

  17. Compute Upto Date - The Date and time until when the process was executed for the selected patient gets displayed.

  18. Process Ind - The process indicator is displayed as 'On-Error', 'Processed' or 'Not Processed'.

  19. Error Text - The error encountered during the processing of selected records gets displayed.
