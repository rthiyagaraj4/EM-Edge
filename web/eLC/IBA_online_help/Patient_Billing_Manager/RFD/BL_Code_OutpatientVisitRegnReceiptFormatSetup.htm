Enter the following details to define a format for visit registration receipts:

 

  1. Facility – Click [List of Values] to select a Facility. The receipt format is valid for the Clinics in the selected facility.

 

Visit Registration Receipt Format Setup

  1. Clinic Code - Click [List of Values] to select a Clinic. The values defined using the Clinic function in the Outpatient Management module gets listed. This is a mandatory field.

  2. Description - The description for the selected Clinic gets displayed.

  3. Visit Type Indicator - Click [List of Values] to select a Visit Type Indicator. The values are system defined - 'Emergency', 'Unrestricted', 'First Visit', 'Follow-up Visit', 'Routine' and 'Series Visit'. This is a mandatory field.

  4. Emergency – Select Emergency to define the receipt format for an emergency visit to the Outpatient Clinic.

  5. Unrestricted - Select Unrestricted to define the receipt format for all the visits to the Outpatient Clinic.

  6. First Visit – Select First Visit to define the receipt format for a first visit to the Outpatient Clinic.

  7. Follow-up Visit - Select Follow-up Visit to define the receipt format for the follow-up visit to the Outpatient Clinic.

  8. Routine – Select Routine to define the receipt format for routine visits to the Outpatient Clinic.

  9. Series Visit - Select Series Visit to define the receipt format for series visits to the Outpatient Clinic.

  1. Description - The description for the Visit Type Indicator gets displayed.

  2. Visit Type Code - Click [List of Values] to select a Visit Type. Visit Type helps in identifying the purpose of the outpatient encounter against a care location. The values defined using the Visit Types function in the Outpatient Management module get listed. This is a mandatory field.

  3. Description - The description for the respective Visit Type gets displayed.

  4. Upto Duration (in hrs) - Enter the duration in numeric format with a maximum of three digits. The time recorded is in hours. The time duration indicates the maximum time period until when the defined visit type is associated to the Clinic in the Facility.

  5. Doc Format – Select a document format from the list. The values are system defined - “Receipt” and “Coupon”. Select “Receipt” to print a Visit Registration Receipt. The Receipt is applicable for the selected Visit Type to the Clinic.
