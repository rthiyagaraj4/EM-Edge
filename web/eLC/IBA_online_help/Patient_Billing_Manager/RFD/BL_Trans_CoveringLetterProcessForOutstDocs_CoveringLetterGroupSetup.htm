Enter the following details to create a Covering Letter Group:

 

  1. Customer Code – Click [List of Values] to select a Customer Code. The values defined using the ‘Customer’ function under the ‘AR Setup’ menu in the ‘Setup’ menu gets listed.

  2. Customer Name – The Name of the relevant Customer gets displayed.

  3. Group Code - Enter the Group Code in alphanumeric format of six characters maximum length. This is a mandatory field.

  4. Description – Enter the Description for the relevant Group Code in alphanumeric format of 40 characters maximum length. This is a mandatory field.

 

Contact Details

  1. Contact Name – Enter the Name of the person to be contacted in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  2. Contact Name Title – Enter the designation of the contact person in alphanumeric format of 40 characters maximum length.

  3. Tel Num – Enter the Telephone Number of the contact person in numeric format of 30 digits maximum length.

  4. Fax Num – Enter the Fax Number of the contact person in numeric format of 30 digits maximum length.

  5. Email ID – Enter the Email ID of the contact person in alphanumeric format of 30 characters maximum length.

 

Mailing Address

  1. Mail Addr Line [1 – 4] – Enter the postal address of the contact person in alphanumeric format of 40 characters maximum length.

  2. Country – Click [List of Values] to select a Country code. The values defined using the function ‘Country’ in the Master Patient Index module get listed. This is a mandatory field.

  3. Postal Code – Enter the Postal Code in alphanumeric format of 20 characters maximum length.

  4. Post Box – Enter the Post Box number in numeric format of 20 digits maximum length.
