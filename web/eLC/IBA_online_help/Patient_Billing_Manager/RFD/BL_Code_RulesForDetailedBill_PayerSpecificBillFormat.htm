Payer Specific Row/Col Format

The Payer Specific Bill Format tab enables you to define a bill format for a Payer. If a Payer Specific Bill Format is not defined, the bill is printed based on the bill format defined using the Multipayer Bill Format tab.

 

Enter the following details to define a Payer Specific Bill Format:

 

  1. Facility – The value is defaulted to the logged in facility, by default. This is a mandatory field.

  2. Payer Code – Click [List of Values] to select a Payer. The values defined using the Customer function gets listed. The bill format is valid for the selected Payer. This is a mandatory field.

  3. Total Additional Charge As - Select a value from the list. The values are system defined – “Column”, “Row” and “Don't Print”. Select 'Column' to print the additional charge in a column. Select 'Row' to print the additional charges in a row. Select 'Don't Print', if the additional charges are not to be printed.

  4. Print Service Date in Detail Format - Select the check box to print service date in a detailed format in the bill generated for payers. Clear the check box, if service date is not required.

  5. Print Service Date with Time - Select the check box to print service date and time in the bill generated for payers. You can select the check box, if 'Print Service Date in Detail Format' check box is selected. Clear the check box, if service date and time is not required.

  6. Print Rate in Detail Format - Select the check box to print service rates in the payer bills irrespective of the service coverage. You can select the check box, if 'Print Service Date in Detail Format' check box is selected. Clear the check box, if service rate is not required in the bills printed for payers.

  7. Print Zero Discount Columns - Select the check box, if the bill printed for payers has to include discount columns, when the patient has not availed any discount from the facility. On selecting the check box, the discount value is printed as zero in the payer bills. The check box is enabled for selection, if the value for 'Total Payable Service Line Discount As' is selected as 'Column'. Clear the check box, if discount columns are not to be printed when there are no discounts.

  8. Total Payable Service Line As - Select a value from the list. The values are system defined – “Gross Amt”, “Net Amt” and “Do Not Print”.

  9. Gross Amt – Select Gross Amt to print the complete bill details along with the discounts amount.

  10. Net Amt – Select Net Amt to print only the net charges in the bill.

  11. Do Not Print – Select Do Not Print, if the details are not to be printed.

  1. Total Payable Service Line Discount As - Select a value from the list. The values are system defined – “Row”, “Column” and “Not Applicable”.

  2. Row – Select Row to print the discount details in a Row.

  3. Column – Select Column to print the discount details in a Column.

  4. Not Applicable - Select Not Applicable, if the discount details are not to be printed.

  1. Bill Adjustments As - Select a value from the list. The values are system defined – “Summarized Row”, “Separate Row” and “Donot Print”.

  2. Summarized Row – Select Summarized Row to print a summary of the payer details in a row.

  3. Separate Row - Select Separate Row to print a summary of the payer details in a separate row.

  4. Donot Print – Select Donot Print, if the details are not to be printed.

  1. Print Bill Discount as Separate Row – Select ‘Print Bill Discount as Separate Row’ to print the bill discounts in a separate row. The check box is selected automatically when the ‘Print Bill Adjustment As’ is set as Separate Row. The check box is enabled only when ‘Bill Adjustments As’ is selected as ‘Summarized Row’ or ‘Separate Row’. The check box is disabled, if the ‘Bill Adjustments As’ is selected as ‘Do Not Print’.

  2. Bill Adjustment as Separate Row - Select ‘Bill Adjustment as Separate Row’ to print the bill adjustment details in a separate row. The check box is selected automatically when the ‘Bill Adjustments As’ is set as Separate Row. The check box is enabled only when ‘Bill Adjustments As’ is selected as ‘Summarized Row’ or ‘Separate Row’. The check box is disabled, if the ‘Bill Adjustments As’ is selected as ‘Do Not Print’.

  3. Bill Exemption as Separate Row - Select ‘Bill Exemption as Separate Row’ to print the bill exemption details in a separate row. The check box is selected automatically when the ‘Bill Adjustments As’ is set as Separate Row. The check box is enabled only when ‘Bill Adjustments As’ is selected as ‘Summarized Row’ or ‘Separate Row’. The check box is disabled, if the ‘Bill Adjustments As’ is selected as ‘Do Not Print’.

  4. Bill Write-Off as Separate Row - Select ‘Bill Write-Off as Separate Row’ to print the bill write-off details in a separate row. The check box is selected automatically when the ‘Bill Adjustments As’ is set as Separate Row. The check box is enabled only when ‘Bill Adjustments As’ is selected as ‘Summarized Row’ or ‘Separate Row’. The check box is disabled, if the ‘Bill Adjustments As’ is selected as ‘Do Not Print’.

  5. Print Zero Adjustments - Select the check box to print zero adjustment details in payer bills. Clear the check box if zero adjustments are not to be printed in payer bills.

  6. Other Payer Amount as – Select a value from the list. The values are system defined – “Summarized Row”, “Separate Row” and “Donot Print”.

  7. Summarized Row – Select Summarized Row to print a summary of the payer details in a row.

  8. Separate Row - Select Separate Row to print a summary of the payer details in a separate row.

  9. Donot Print – Select Donot Print, if the details are not to be printed.

  1. Patient Payable Amount as - Select a value from the list. The values are system defined – “Separate Row” and “Donot Print”.

  2. Separate Row – Select Separate Row to print the patient payable amount in a separate row.

  3. Donot Print – Select Donot Print, if the patient payable amount is not to be printed on the bill.

  1. Net Payable Amount as – Select a value from the list. The values are system defined - “Row”, “Column” and “Both”.

  2. Row – Select Row to print the net amount towards the service in a row.

  3. Column – Select Column to print the net amount towards the service in a column.

  4. Both - Select Both to print the net amount details in a row and column.
