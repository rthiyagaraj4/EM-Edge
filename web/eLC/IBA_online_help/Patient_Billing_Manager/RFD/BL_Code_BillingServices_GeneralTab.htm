General Tab

Enter the following details:

 

  1. Print Group/Line – Click [List of Values] to select a Print Group/Line. The values get listed only if the Print Group/Lines are defined initially. A Print Group/Line is associated to the Billing Service to identify the service and print the details on the bill. This is a mandatory field.

  2. External Bill Type - Click [List of Values] to select an External Bill Type. The Bill Types defined gets listed. The Bill Types are defined under the menu Accounting Entity Setup. This is applicable for Referral patients.

  3. Procedure Coding Scheme - The standard codes for the Diagnosis/Procedure Groups are already set and defined globally by the Practitioners. Examples are ICD-9CM, CPT, etc. These standards can be used to identify the disease the patient is suffering from. The values get listed, if the Procedure code is defined in the Procedure Details function in Medical Records module.

  4. Procedure Code – Click [List of Values] to select a Procedure Code. The Procedure Codes created using the function Procedure Details in the Medical Records module to identify the disease gets listed.

  5. Default Charge – You can define the default charge only when the Rate type is selected as Charge. The Default Charge specified is used to calculate the charge for the charge based services.

  6. Cost Indicator – Select a Cost Indicator from the list. The values are system defined:

  7. %age – Select %age, if a percentage of the specified Cost Amount is to be charged based on Charge or the prices defined using Multipricing function. For Example, if the Cost Amount is set to 50 and the Charge defined for the Service is 100, the charge towards the service is computed as 50% of 100. The charge towards the service utilized is 50.

  8. Amount – Select Amount to define a constant amount. The amount defined for a patient category associated to the Billing Class in the Billing Service function is charged towards the service.

  9. N/A – Select N/A, if the Cost Indicator is not applicable.

  1. Cost Amount – Enter the Cost Amount in numeric format. The Cost Amount is the actual amount towards the Billing Service.

  2. Min. Charge – Enter the minimum charge for the relevant service in numeric format. The minimum charge has to be less than the default charge defined for the charge based service. The Minimum Charge is defined only when the Charge Type is selected as Charge.

  3. Max. Charge – Enter the maximum charge towards the service. The price for a service cannot be higher than the price specified.

  4. Reimbursable Amt – Enter a reimbursable amount towards the service. The amount specified has to be a valid numeral from one to 100. If the Cost Amount is selected as Amount, enter a constant numeral value, which specifies the sum, which has to be deducted from the patient service charge. The Reimbursable Amount specified can be overridden while defining prices under the Standard Price, Price Class Specific Price, Billing Group Specific Price and Customer Specific Price tabs for a specific patient class or all the patient classes in the facility.

  5. Check Cost Availability – Select the check box Cost Availability to consider the Cost Amount during the charge calculation. Clear the check box to consider the actual charge towards the service.

 

Control Settings

  1. Direct Entry Allowed – Select the check box to associate the services to an Inpatient/Outpatient using the transaction functions ‘Enter IP Request’ or ‘Enter OP Request’. You cannot associate the service to an Inpatient/Outpatient using the transaction functions ‘Enter IP Request’ or ‘Enter OP Request’, if the check box is cleared.

  2. Auto Confirm – Select the check box to confirm the services automatically. The services associated to a patient in the transactions using either ‘ Enter IP Request’ or ‘Enter OP Request’ function is confirmed automatically, if the check box is selected. No manual processing is required. Clear the check box to confirm the services using the function manually ‘Confirm Inpatient Service Requests’ or ‘Confirm Outpatient Service Requests’ in the transactions for Inpatients and Outpatients respectively.

  3. Acct Dept from Source – Select the check box to consider the source accounting department. The data set for the parameters under the Account Department gets listed.

  4. Standard Price – Select the check box, if the Standard Price has to be defined for the respective Billing Service. Clear the check box to disable the Standard Price tab. Standard Price is the normal price set for the services using the Standard Price tab.

  5. Price Class Specific Price – Select the check box to enable the Price Class Specific Price tab and define the charge depending on the class of service used from the facility. Clear the check box to disable the Price Class Specific Price tab.

  6. Billing Group Specific Price – Select the check box to enable Billing Group Specific Prices tab and define the prices. Clear the check box to disable the Billing Group Specific Price tab. A price definition is based on the Billing Group selected. The relevant price is charged when the respective Billing Group is selected.

  7. Customer Specific Price – Select the check box to enable the Customer Specific Price tab and define the prices. The price depends on the customer class used. The price is specific to a set of customers.

  8. Screening Services – Select this check box to display the hyperlink Screening Services in the Place order screen of the Clinician Access function.

  9. Nursing Service – Select the check box, if the Nursing Service is a part of the Billing Service. Clear the check box, if the Nursing Service is not a part of the Billing Service.

  10. Materials Item-related service – Select the check box, if the material items sold by the Facility are a part of the Billing Service. Clear the check box, if the material items are not a part of the respective Billing Service. Pharmacy items or Sterile Stocks in a Facility are examples of Materials Item related service.

  11. Miscellaneous Service – Select the check box to associate multiple services to the respective Billing Service. Clear the check box, if the respective Billing Service code cannot be associated to any other service.

 

Doctor’s Fee Related

  1. Read Charges Applicable – Select this check box to calculate the revenue share for the services provided by the Reporting Doctor for each episode.

  2. Community Doctor Service – Select this check box to calculate the revenue share for the services provided by the Reporting Doctor for each episode.
