User for Exemption

Enter the following details to select a User for Exemption:

 

  1. User Id – System displays the ID of the logged in user, by default. The User ID cannot be changed.

 

Exemption Code

  1. Exemption Type – Click [List of Values] to select an Exemption Type. The associated user is able to access the selected exemption code and set the limit amount for a stage level. Authorized user is further able to associate the selected the exemption code to the patient and exempt the bill amount of a patient. This is a mandatory field.

  2. Description – The Description for the selected Exemption Code gets displayed.

 

Privileges by Stages

  1. Stage Level – Click [List of Values] to select a Stage Level. The stages of exemption defined for an exemption code under the tab ‘Stages for Exemption’ get listed. This is a mandatory field.

  2. Description – The Description for the selected Stage Level gets displayed. This is a mandatory field.

  3. Limit Amount  – Enter the limit on the exemption amount in numeric format. The limit specified on the exemption amount is relevant for each stage. This is a mandatory field.

  4. Change Requested Exemption Amt – Select the check box to change the exemption amount in the transaction functions.

  5. Change Type – Select a value from the list. The system-defined values are ‘Increase’, ‘Decrease’, ‘Both’ and ‘Not applicable’. You can either increase or decrease the exemption amount based on the value selected.

  6. Value in % - Enter the value in numeric format to indicate the percentage amount based on which the exemption amount can be changed for the respective stage. You cannot enter the value, if the ‘Change Type’ is selected as ‘Not Applicable’.
