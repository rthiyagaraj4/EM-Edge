Enter the following details to raise a request for refund:

 

  1. Request No. - Enter the value for request number, if the document sequence number setting is set as Manual. The document type is set as manual in the ‘Document Type Details’ function in the Financials Base Manager module. If the check box ‘System Generated Document Number’ is selected in the function ‘Document Type Details’ in the Financials Base Manager module, the document number is generated automatically by the system. This is a mandatory field.

  2. Refund Requests Selection Criteria - The filed contains pre-defined values. The values in the list are “All”, “Refund against Bill & DCP”, “Refundable Deposit”, and “Patient Deposit”. The system will default with the value as “All”. If Refund Request is required to be made for a specific nature, select the specific nature from the list and raise the request.

  3. All – List of records, where the refund is not issued is displayed.

  4. Refund against Bill & DCP - Only the records meeting the criteria are displayed. The refund is issued at the time of discharge and to the deferred Cash patients.

  5. Refundable deposit - Only the patient records, where the deposits are to be refunded are displayed.

  6. Patient Deposit - Select Patient Deposit is the deposit is to be reimbursed to the patient.

  1. Select - Select the check box to process the refund request for the selected documents. Clear the check box, if the refund request is not valid for the selected documents.

The document number, document date, refund amount and the nature of the bill get displayed.

  1. Patient number - The patient number for the associated to the selected document number gets displayed. The name of the respective patient gets displayed. The episode type, episode ID and visit ID are displayed for the documents to be refunded.

  2. Control Voucher No./Date – Enter the Control Voucher number and date. This field is mandatory, if the ‘Control Voucher Entry Mode’ is selected as ‘During Refund Request Entry’ under the ‘Refund Processing’ tab in the BL Parameters function. If the ‘Control Voucher Entry Mode’ is selected as ‘During Approval of Refund Request’, this field is not mandatory.

 

Refund Details

  1. Payee’s Name - Enter the name of the payee to whom the refund has to be issued for the selected document type. This is a mandatory field.

  2. NRIC No - The NRIC number associated to the selected Patient ID gets displayed.

  3. Payee Acc. No - Enter the account number of the payee in numeric format of 20 digits maximum length. The amount issued as refund is deposited in the respective account.

  4. Payee Bank/Branch - Select the Bank/Branch name from the list. The values defined using the Banks and Branches function get listed. The amount issued as a refund is deposited to the payee’s account in the selected Bank or associated Branch.

  5. Remarks - Enter the remarks in alphanumeric format of 50 characters maximum length.

 

Mailing Address

  1. Click to copy from patient - Click the link to copy the patient mailing address. The address details get populated in the respective fields. You can copy the address, if the patient’s address is same as the payee’s address.

  2. Line 1 - 4 - Enter the address in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Post Box - Enter the Post Box number in alphanumeric format of 30 characters maximum length.

  4. Postal Code - Enter the Postal Code in alphanumeric format of 30 characters maximum length.

  5. Country – Click [List of Values] to select a Country. The description for the selected Country gets displayed. This is a mandatory field.

 

Contact Details

  1. Tel No - Enter the telephone number in numeric format of 20 digits maximum length.

  2. Mobile No - Enter the mobile number in numeric format of 20 digits maximum length.

  3. Fax No - Enter the fax number in numeric format of 20 digits maximum length.

  4. Email - Enter the e-mail ID of the payee in alphanumeric format of 30 characters maximum length.
