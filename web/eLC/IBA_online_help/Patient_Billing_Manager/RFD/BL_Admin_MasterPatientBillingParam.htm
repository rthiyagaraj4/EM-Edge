Enter the following details to define Master Patient Billing Parameters:

  1. Default, New Born’s Billing Group as Mother’s Billing Group – Select the check box, if the New Born’s Billing Group has to be same as the Mother’s Billing Group. The ‘New Born’s Registration’ text box is disabled on selecting the check box.

  2. Default Patient Billing Group for Encounter Registration – Select the check box, if the default Patient Billing Group is applicable for Encounter Registration. On selecting the check box, you can select a Billing Group for New Born Registration and the selected Billing Group is made as the default Billing Group for all the new born being registered in the Master Patient Index module.

  3. Default Last Encounter Fin Details During Encounter Registration – Select the check box, if the previous encounter financial details has to be displayed during the encounter registration. Clear the check box and the previous encounter details do not get displayed during the current encounter registration.

  4. Default Patient Fin Details During First Encounter – Select the check box, if the financial details recorded for the patient during first encounter have to be displayed, by default in the external modules. Clear the check box and the details recorded during the first encounter are not displayed.

  5. Provision to select from all Previous Encounters - Select the check box, if the financial details recorded for a patient during the previous encounters have to be displayed during the current encounter registration. Clear the check box and the details recorded during the previous encounters are not displayed.

 

Default Billing Group

  1. New Born’s Registration – Click [List of Values] to select a Billing Group for New Born’s registration. The selected Billing Group is set as the default entry in the Master Patient Index module during new born registration. The text box is enabled on clearing the ‘Default, New Born’s Billing Group as Mother’s Billing Group’ check box.

  2. Emergency Registration – Click [List of Values] to select a Billing Group towards Emergency Registration. The selected Billing Group is set as the default entry during emergency registration for the new born.

 

Default Billing Class

  1. Disaster Billing Group – Click [List of Values] to select the Disaster Billing Group. This billing group is used during emergency.

  2. Quick Adm. Billing Group - Click [List of Values] to select the Quick Admin Billing Group. The billing groups defined in the Encounter Regn. Default Billing Group are listed. This billing group is used during quick admission.

  3. Patient Registration – Click [List of Values] to select a default Billing Class for patient registration. The selected value is set as the default Billing Class during patient registration in the Master Patient Index module. The values defined using the Billing Class function get listed.

  4. New Born Registration - Click [List of Values] to select a default Billing Class for New Born Registration. The selected value is set as the default Billing Class during New Born Registration in the Master Patient Index module. The values defined using the Billing Class function get listed.
