Enter the following parameters to create the Discount Accounts:

 

  1. Billing Group - Click [List of Values] to select a Billing Group. The description for the selected Billing Group appears in the adjacent text box.

 

Click [Enter Query] to select and associate a Bill Type.

 

Discount Account Details

  1. Bill Type - Click [List of Values] to select a Bill Type. The values defined using the Bill Types function gets listed. This is a mandatory field.

  2. Description - The description for the respective Bill Type gets displayed.

  3. GL Interface - Click [List of Values] to select a GL Interface Code. The account code set for discounts in the General Ledger module gets displayed. This is a mandatory field.

  4. Eff. From Date – Click Calendar function to select a date. The Discount Accounts are valid for the selected Billing Group from the date selected. This is a mandatory field.

  5. Eff. To Date - Click Calendar function to select a date. The Discount Accounts are valid for Billing Group until the date selected.

 

Discount Account Interface by Service and Payer

A Billing Service is associated to a Payer and to a Discount Account. You have an option to select a Billing Service and associate a Payer to the selected Billing Service. This indicates that the Discount Account is applicable for the selected Billing Service and Payer combination.

 

  1. Blng Serv Code – Click [List of Values] to select a Billing Service Code. This is a mandatory field.

  2. Description – The Description for the selected Billing Service Code gets displayed.

  3. Payer - Click [List of Values] to select a Payer. This is a mandatory field.

  4. Name – The Description for the selected Payer gets displayed.

  5. GL Interface – Click [List of Values] to select a GL Interface code. The account code set for discounts in the General Ledger module gets displayed. This is a mandatory field.

  6. Description – The Description for the selected GL Interface Code gets displayed.

  7. Blng Serv Indicator –Select the Blng Serv Indicator from the drop-down list. The listed values are ‘Service Group’, ‘Service Classification’ and ‘Billing Service’.

  8. Blng Serv Code – Click [List of Values] to select a Billing Service Code.

  9. Description – The Description for the selected Billing Service Code gets displayed.

  10. GL Interface – Click [List of Values] to select a GL Interface code. The account code set for discounts in the General Ledger module gets displayed.

  11. Description – The Description for the selected GL Interface Code gets displayed.
