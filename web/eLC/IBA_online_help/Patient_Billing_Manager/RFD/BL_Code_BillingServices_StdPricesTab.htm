Standard Prices Tab

Standard Prices tab enables you to set up standard price for the service. The Standard price is valid for the selected facility.

 

Enter the following details to define a Standard Price:

 

  1. Facility – Select the Facility from the list to define the Standard Price for a relevant Billing Class. The facilities defined in the System Manager module and associated to the relevant user get listed. This is a mandatory field.

  2. Valid From – Click Calendar function to select the date. The Standard Prices are applicable for the Billing Classes in the Facility from the date selected. The date set has to be greater than the system date. This is a mandatory field.

  3. Valid To – Click Calendar function to select the date. The Standard Prices are applicable for the Billing Classes in the Facility until the date selected.

  4. Billing Class – Click [List of Values] to select a valid Billing Class and set up a Standard Price for the relevant patient category. The Billing Classes defined under the ‘Billing Rule’ menu in the Masters gets listed.

  5. Description – The Description for the selected Billing Class is displayed.

  6. Cost Ind. – Select a valid Cost Indicator from list. The values are system defined – '%age', 'Amount' and 'N/A'.

  7. Cost Amt – Enter the actual price towards the service. When the selected Cost Indicator is:

  8. %age – A percentage of the Cost Amount is calculated for the standard price. The cost amount is charged towards the service for the relevant Billing Class.

  9. Amt – The amount defined for a patient category associated to the Billing Class in the Billing Service function is charged towards the service.

  10. N/A – Select N/A, if the Cost Amount is not applicable for the selected Billing Class.

  1. IP Rate – Enter the base price to be charged for the inpatients. The inpatients availing the service are charged based on the quantity of the service used.

  2. DC Rate – Enter the base price to be charged for the Daycare patients. The charge towards the service is calculated accordingly based on the quantity used.

  3. OP Rate - Enter the base price to be charged for the Outpatients. The charge towards the service is calculated accordingly based on the quantity used.

  4. Emg. Rate - Enter the base price to be charged for the Emergency cases. The charge towards the service is calculated accordingly based on the quantity used.

  5. Ext. Rate - Enter the base price to be charged for the External or the Referral patients. The charge towards the service is calculated accordingly based on the quantity used.

  6. Min. Charge - Enter the minimum charge towards the service. The charge towards the service will be calculated accordingly based on the quantity used. This is a mandatory field.

  7. Max Charge – Enter the maximum charge towards the service. This is applicable only when the Cost Indicator is set. This identifies the maximum amount, which is charged towards the service when the Cost Indicator is set. You cannot charge an amount beyond the limit specified for the service.

  8. Allowed – Select the check box, if the Billing Service is reimbursable for the selected patient class. Clear the check box, if the respective Billing Class is non-reimbursable for the selected patient class.

  9. Override – Select the check box, if the respective Billing Service is reimbursable for the selected patient class. The override option for the patient class takes precedence over the override check box in the header field. If the service is not reimbursable and the check box is selected for the patient class, it indicates that the Billing Service is reimbursable for the selected patient class.

  10. Reimbursable Amt – Enter the amount, which can be reimbursed to the patient.

  11. Base Qty – Enter the Base Quantity towards the service. The charge is calculated based on the quantity used. This is a mandatory field.

  12. Gross/Net Applicability – Select a value from the list. The values are system defined – 'Gross' and 'Net'.

  13. Gross – Select Gross to charge the Gross amount for the service. The charge towards the service is calculated based on the discount and the price defined for the Billing Class.

  14. Net – Select Net to charge the Net amount for the service. The service charge is calculated based on the price for the Billing Class. The discount amount for the service is not considered during bill generation.

  1. Check Cost Availability – Select the check box, if the Cost Amount is applicable. Clear the check box, if the Cost Amount is not applicable.
