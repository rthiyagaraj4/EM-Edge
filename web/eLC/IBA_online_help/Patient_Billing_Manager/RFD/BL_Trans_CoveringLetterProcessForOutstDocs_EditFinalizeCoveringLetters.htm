You can edit the following details:

 

  1. Foreign Cur.Note – Enter the Foreign Currency Note information in alphanumeric format.

  2. Authorized Signatory – Enter the name of the authorized person to edit and finalize the generated Covering Letters.

  3. Tot Outst Amt – The total outstanding amount from the services gets displayed. The amount gets updated on updating the outstanding amount in the ‘Doc Outst Amt’ field.

  4. No. of Docs – The number of outstanding documents associated to the selected patient get displayed.

  5. Order By – Select an option from the list. The outstanding documents displayed are sorted based on the Order By value selected. The values are system defined.
