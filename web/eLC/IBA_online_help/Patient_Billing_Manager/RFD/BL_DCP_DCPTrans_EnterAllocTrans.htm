Enter the following to create allocating transactions:

 

  1. Doc No - Enter a valid Document Number. A description for the selected Document Type gets displayed. If the document number is system generated, you cannot enter the Document Number. Document Type Code and Document Number have to be unique. This is not pertinent to the Allocation Detail. This is a mandatory field. Following validations are carried out for the Allocating Document Number.

  2. Document Type and Number must not be duplicated in this transaction entry.

  3. Document Type and Number should not be present already in Pending Documents detail.

  4. Document Type and Number should not be present in Document Control details.

  1. Date - The system date is displayed as the document date, by default.  

  2. Trx Type – The Transaction Type is displayed based on the Document Type selected. You cannot update the Transaction Type.

  3. Patient No - Enter a valid Patient Number. Right click on ‘Click for User Option’ to search for a patient record. If the patient number is valid, the patient name, gender, age in years, months and days are displayed. This is a mandatory field.

  4. Doc Ref - Enter the document reference details in alphanumeric format. The Document Reference is considered as the remarks.

  5. Period - The post month and year is defaulted from the Billing Parameters. Click Calendar function to update the period. The post month and year has to be greater than the Billing parameters post month and year.

  6. Amount - Enter the document amount for this document. Zero is a valid entry. If the transaction type is "3" or "5", the amount entered must be negative. If the transaction type is "1", "2" or "6", the amount entered must be positive. You have to take care to put the correct sign in case of transaction type equal to "4", since both signs are acceptable.

  7. Narration - Enter the narration for the allocating document. The narration is considered as the remarks for the selected Document Type.

  8. Details Required – Select the check box to view the Allocating Document details. Clear the check box and the allocating document details are not displayed.  

Allocating Transactions – A/C Lines  

  1. Posting A/C – Click [List of Values] to view and to select a posting account. The description for the selected posting account gets displayed. When the document is posted, the selected posting account is credited or debited (whether the amount is positive or negative respectively) with the line amount.

  2. Description – The Description for the relevant Posting Account gets displayed.

  3. Amount - Enter the amount for the selected GL distribution line. The amount can be either positive or negative irrespective of the Transaction Type.

  4. Narration - Enter the Narration for the General Ledger distribution line.
