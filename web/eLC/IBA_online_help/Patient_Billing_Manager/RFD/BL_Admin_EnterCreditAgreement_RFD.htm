The bill details marked as ‘D’ gets displayed for the selected patient.

 

  1. Print – Select the check box to print the credit agreement details for the selected patient.

  2. Print Service Acceptance – Select the check box to print service acceptance agreement.

  3. Print Agreement Contract – Select the check box to print agreement contract.

  4. Print Installment Agreement - Select the check box to print installment agreement.

  5. Posted – The check box is selected, by default. The details are transferred to the Deferred Cash Patient.

  6. Select – Select the check box to post the details. A log file is generated with the details on posting the details.

  7. [Post] – Click [Post] and the Guarantor’s Details, Counseling Details and the Instalment Plan tabs get enabled. You have to record credit agreement details on posting the details.

  8. Select – Select the check box to record the details. You have to select at least one record to enter the agreement details.

  9. [Enter Agreement] – Select the check box to record the details in the tab pages for the selected patient.
