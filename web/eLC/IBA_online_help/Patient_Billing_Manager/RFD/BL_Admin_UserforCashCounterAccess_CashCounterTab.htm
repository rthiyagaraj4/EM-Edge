Cash Counter Access

The Cash Counter Access tab enables you to associate Users to a Cash Counter in a facility. You have an option to choose the Shift in which the User can work when logged into the selected Cash Counter.

 

Enter the following details to define the Cash Counter Access:

  1. Facility – Select an appropriate Facility from the list. The facilities associated to the logged in User get listed.

  2. Cash Counter Code – Click [List of Values] to select a Cash Counter Code. The User is able to login to the selected Cash Counter in the facility.

 

Process Refund

  1. OP/Emerg – The check box is selected or cleared based on the selection made for the check box ‘OP/Emg’ while defining the Cash Counter.

  2. IP/Day Care – The check box is selected or cleared based on the selection made for the check box ‘IP/Daycare’ while defining the Cash Counter.  

  3. External - The check box is selected or cleared based on the selection made for the check box ‘Ext’ while defining the Cash Counter.

 

Process Receipt

  1. OP/Emerg – The check box is selected or cleared based on the selection made for the check box ‘OP/Emg’ while defining the Cash Counter.  

  2. IP/Day Care – The check box is selected or cleared based on the selection made for the check box ‘IP/Daycare’ while defining the Cash Counter.  

  3. External - The check box is selected or cleared based on the selection made for the check box ‘Ext’ while defining the Cash Counter.  

 

User Details

  1. User ID – Click [List of Values] to select a User ID. The handover limit is valid for the selected user. User can be associated only to a single Cash Counter.

  2. User Name – The User Name for the selected User ID gets displayed.

 

Handover Limit

A handover limit for individual user and counter level is set using this tab. User is either blocked from processing further transactions or the user is alerted when the cash amount grace limit is reached. These fields are enabled only if the 'Handover without CC Logout' check box is selected in 'BL Parameters' function.

  1. Cash - Enter the cash in numeric format. When the transaction amount reaches the cash limit, alert messages are displayed for the respective user in the Cash Counter.

  1. Action - Select an Action from the list. The system-defined values are 'Alert User', 'Block User' and 'No Action'.

  2. Alert User - Select 'Alert User' to alert the Cashier when the Cash limit is reached. You cannot select a Limit Indicator or specify the 'Grace Limit' when the 'Action' is selected as 'Alert User'. Limit Indicator is set as 'NA' automatically. On selecting 'Alert User' an alert message is displayed for every transaction when the handover limit is crossed. The transactions are blocked when the grace limit is reached.

  3. Block User - Select 'Block User' to block the Cashier transactions when the Cash limit is reached. You can set the 'Limit Indicator' as '%' or 'Amount' and specify the 'Grace Limit' when the 'Action' is selected as 'Block User'.

  4. No Action - Select 'No Action' if action type is not valid. An alert or warning message is not displayed when the action is not selected.

  1. Limit Indicator - Select a Limit Indicator from the list. The system-defined values are '%', "Amount' and 'N/A'. You can select the value as '%' or "Amount' only when the 'Action' is selected as 'Block User'.

  2. % - Select '%' to block the cashier transactions when a percentage of the Cash limit is reached.

  3. Amount - Select 'Amount' to block the cashier transactions when the Cash limit reaches the mentioned amount.

  4. N/A - Select the value as 'N/A', if limit indicator is not required. The value is automatically set to 'N/A' when the 'Action' is selected as 'Alert User' or 'N/A'. You can select the value as 'N/A' when you select the 'Action' as 'Block User'.

  1. Grace Limit - Enter the Grace Limit in numeric format of 12 digits maximum length. You can specify the Grace Limit when the 'Action' is selected as 'Block User'. The Grace Limit should not exceed the Cash Limit.

  2. Grace Limit has to be within the range '0-100' when the 'Limit Indicator' is selected as '%'.

  3. Grace Limit should not be greater than the Cash or Cheque limit when the 'Limit Indicator' is selected as 'Amount'.

  4. Grace Limit cannot be recorded when the 'Limit Indicator' is selected as 'N/A'.
