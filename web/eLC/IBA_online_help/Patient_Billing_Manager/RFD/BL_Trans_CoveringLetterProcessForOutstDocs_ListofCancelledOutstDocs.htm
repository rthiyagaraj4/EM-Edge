Enter the following details to list out the cancelled outstanding documents:

 

  1. Module ID - Select a module from the list. The values listed are – Both, Patient Billing and Accounts Receivable. The details are generated based on the module selected.

  2. Covering Letter Date [From–To] – Click Calendar function to select the date. The Covering Letters for the outstanding documents, which were cancelled during the selected date range get displayed in the report.

  3. Billing Group [From–To] – Click [List of Values] to select a range of Billing Groups. The Covering Letters for the outstanding documents, which were cancelled for the selected Billing Group get displayed in the report.

  4. Customer [From–To] - Click [List of Values] to select a range of Customers. The Covering Letters for the outstanding documents, which were cancelled for the selected range of Customers get displayed in the report.

  5. Patient ID [From–To] – Click [List of Values] to select a range of Patient IDs. The Covering Letters generated for the specified range of Patient IDs get displayed in the report.

  6. Doc Type Code [From–To] - Click [List of Values] to select a range of Document Type Codes. The Document type codes for which the Covering Letters were cancelled get displayed in the report.

  7. Episode [From–To] – Enter a range of Episode ID in numeric format. The cancelled Covering Letter details for the outstanding documents of the selected range of Episode numbers get displayed in the report.

  8. Visit Id [From–To] – Enter the range of Visit IDs. The cancelled Covering Letter details for the outstanding documents of the selected range of Visit IDs get displayed in the report.  

  9. Module ID – Select a module from the list. The listed values are Both, Patient Billing and Accounts Receivable. The cancelled Covering Letter details for the outstanding documents associated to the module selected get displayed.

  10. Episode Type – Select an Episode Type and enter the Episode ID to view the Covering Letter details, which were cancelled. The system defined values are “Emergency”, “Outpatient”, “Daycare”, “Inpatient”, “External” and “All”.

  11. 11. Report Sort Order – Select a value from the list to sort the details in the report. The values are system defined. The report is sorted based on the order type selected.

  12. • Billing Group – Select Billing Group to sort the report based on the Billing Group.

  13. • Customer – Select Customer to sort the report based on the Customer details.

  14. • Patient ID – Select Patient ID to sort the report based on the Patient ID.

  15. • Episode ID - Select Episode ID to sort the report based on the Episode ID.
