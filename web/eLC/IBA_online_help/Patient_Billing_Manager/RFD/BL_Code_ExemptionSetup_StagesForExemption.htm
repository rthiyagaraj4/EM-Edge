Stages for Exemption

Enter the following details to define the stages for exemption:

 

  1. Exemption Type – Click [List of Values] to select an Exemption Type. The Action Reasons defined using the ‘Reasons for Billing Actions’ function with Bill Exemption check box selected gets listed.

  2. Description – The Description for the selected Exemption Code gets displayed.

 

Stages for Exemption

 

Stage Level

  1. Stage Level – The priority for each Stage level is set automatically. This is a mandatory field.

  2. Description – Enter the Description for the selected Stage Level. The Stage Level describes the individual stages during counseling. This is a mandatory field.

  3. Status Level – Select a Status Level from the list. The values are system defined – 'Request', 'Verification', 'Approval' and 'Finalize'. This is a mandatory field.

  4. Request – Select Request to indicate that a patient has placed an exemption request.

  5. Verification - Select Verification to indicate that the billing personnel have verified the exemption request.

  6. Approval – Select Approval to indicate that the exemption request is approved.

  7. Finalize – Select Finalize to authorize the exemption request.

  1. Record Counseling During Exemption Stages – Select the check box to record counselling details when the patient is exempted. Clear the check box, if the counselling details are not to be recorded during the exemption stages. The ‘Default Encounter Counseling Detail’ check box gets selected automatically on selecting this check box. You can clear the check box, if the selected record should not be displayed as the default record in the transaction functions.

  2. Default Encounter Counseling Detail – Select the check box, to display the record as the default record in the transaction functions. You can select the check box, only if the check box ‘Record Counseling During Exemption Stages’ is selected.

 

Stages for exemption should have a ‘Request’ and a ‘Finalize’ stage.

A warning message ‘Counseling is not Required For Selected Stage’ gets displayed, if you try to associate Counseling Code without selecting the check box ‘Record Counseling During Exemption Stages’.

 

Counseling by Stage

  1. Counseling Code – Click [List of Values] to select a Counseling Code. The values defined using the ‘Counseling Details’ function gets listed. The selected Counseling Code is associated to the relevant Stage.

  2. Description – The Description for the selected Counseling Code gets displayed.

  3. Mandatory – Select the check box, if the Counseling Code associated to the relevant Stage level is mandatory. Clear the check box, if counseling is not mandatory.
