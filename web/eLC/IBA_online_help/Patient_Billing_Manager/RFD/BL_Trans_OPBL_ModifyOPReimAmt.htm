Enter the following details to modify the reimbursable amount in an outpatient bill:

 

  1. Episode Type – Select an Episode Type from the list. The system-defined values are ‘Outpatient’, 'Emergency' and ‘External’.

  2. Admission/Tfr. No – Enter the admission number and the transfer number in numeric format. If the Admission number of the transfer number is not specified, the number gets displayed on selecting the patient ID.

  3. Patient No – Enter the patient ID in alphanumeric format of 10 characters maximum length. This is a mandatory field. On selecting the Patient ID, the name of the patient, gender and age gets displayed. The outstanding amount, if any, gets displayed along with the respective document type, number, Billing Group and the bill amount. The Billing Services associated to the patient gets listed along with the price defined for the service, the reimbursable type for the service and the date and time when the service was associated to the patient.

You can record the details in the following fields:

  1. Override – Select the check box to override the reimbursable amount. On selecting the check box, an authorization screen gets displayed.

  2. User ID – The User ID gets displayed, by default. You can update the User ID. This is a mandatory field.

  3. Pin No – The Pin number for the respective User gets displayed. You can update the Pin number. This is a mandatory field.

  4. Reason – Click [List of Values] to select the Reason. The reasons defined using the function ‘Reasons for Bill Actions’ in the Master with the ‘Override Reimbursement’ check box as selected gets displayed.

 

Click [OK] to login and override the amount or [Cancel] to cancel the login process.

If the Billing Service is non-reimbursable and the user is not authorised to modify the reimbursable amount, respective error message gets displayed, not permitting the user to proceed further with the transactions.

Overriden Reimbursable Dtls

  1. Reimbursable - Select the check box to reimburse the amount for the selected patient class and episode. You cannot reimburse the amount for the Billing Service, if the check box is cleared.

  2. Type - Select a value from the list. The system-defined values are 'Full', 'Part' and 'NA'. Select a value from the list to reimburse the amount.

  3. Revised Amt - Enter the revised amount in numeric format. The reimbursed amount is valid for the selected Billing Service associated to the respective patient encounter.

Audit Details

An Audit number gets generated for the transaction and gets displayed on clicking [Save] to record the details. The reimbursable amount modified for the selected Billing Services get displayed automatically in the respective fields. You can view the reimbursable details during the selected patient encounter in the facility. The overriding reason selected by the user, user ID, modified date and the reimbursable type details are listed.
