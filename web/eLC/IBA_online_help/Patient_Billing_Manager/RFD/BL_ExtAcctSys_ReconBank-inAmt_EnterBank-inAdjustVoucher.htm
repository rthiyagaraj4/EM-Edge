Enter the following details to record adjustment voucher details:

 

External Account

  1. Account Code –  Click [Enter Query] or press [F7] to search and select a specific Account Code from the [List of Values]. Click [Execute Query] or press [F8] to navigate through the Account Codes. The Account Code indicates the External Accounts against which the adjustment are made.

Click [Execute Query] or press [F8] to view or associate Bank-in Adjustment details.

 

Hospital Bank-in Adjustments

  1. Bank Code – Click [List of Values] to select a Bank to record the adjustment vouchers.

  2. Doc Type – System defaults the document type selected for ‘Bank Recon. Voucher Doc’ specified in the ‘Setup External Acct Parameters’ function. You cannot update the Document Type.

  3. Voucher Number - Enter the Voucher Number with a maximum of eight digits. System generates the Voucher Number, if the ‘System Generated Document Number’ check box is selected in Financials Base Manager module. Unique number has to be entered manually, if the check box is cleared.

  4. Vr.Date – Click the Calendar function to select a Voucher Date to indicate the date when the adjustment voucher is generated.

  5. Mode of Payment – Click [List of Values] to select a Payment Mode. This indicates the settlement type using which the adjustment amount is settled.

  6. Debit - Enter the amount deducted from the relevant Bank after the transactions.

  7. Credit – Enter the amount credited to the Bank after the transactions.

  8. Net Amt - The net amount is displayed automatically. You cannot update the value. Negative sign is  displayed to indicate credit transactions and debit transactions are displayed using positive signs. This is a mandatory field.

  9. Reconciled – You can identify if the record is reconciled in query mode. The check box is selected, if the record is reconciled. The check box is cleared, if the record is not reconciled.
