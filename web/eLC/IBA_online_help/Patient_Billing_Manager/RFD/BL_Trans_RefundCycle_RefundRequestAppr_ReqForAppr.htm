Request for Approval

Enter the following details to place a request:

 

  1. Date [From-To] – Click Calendar function to select the date range. The refund requests placed between the selected date range are listed.

  2. Patient No – Click [List of Values] to select a Patient Number. The refund request raised for the selected patient gets listed.

  3. Request No – Click [List of Values] to select a Request Number. The request numbers generated while placing the request using the function ‘Request for Refund’ gets listed.

  4. Nature – Select a value from the list. The values are system-defined – “All”, “Manual”, “Receipt” and “Bill&DCP”.

  5. All – Select All to view all the records for which the refund request has been approved.

  6. Manual – Select Manual to view all the records for which the refund request has been manually approved.

  7. Receipt - Select Receipt to view all the receipt transactions against which the refund has been approved.

  8. Bill&DCP - Select Bill&DCP to view all the bill and DCP transactions against which the refund has been approved.

  1. Voucher Detail – Select a value from the list. The values are system-defined – “Recorded”, “Not Recorded” and “All”.

  2. Recorded – Select Recorded to view the records for which the Voucher Number is recorded.

  3. Not Recorded - Select Not Recorded to view the records for which the Voucher Number is not recorded.

  4. All – Select All to view all the documents for which the Voucher Number is either recorded or not recorded.

 

The following details are displayed:

  1. Request No – The request number generated while placing the request using the function ‘Request for Refund’ gets displayed. The Serial Number, Date, Refund Amount, Type, Voucher Number, Date, Episode Type, Episode ID, Visit ID along with the patient number and patient name are displayed for the respective request number. You cannot update the values in the field. The refund details and the mailing address are also displayed for the selected record. The details recorded using the function ‘Request for Refund’ gets displayed.

  2. Select – Select the check box to select the record and to approve the refund request.

  3. Voucher No. – Enter the Voucher Number in numeric format to process the request. This field is mandatory, if the ‘Control Voucher Entry Mode’ is selected as ‘During Approval of Refund Request’ under the ‘Refund Processing’ tab in the BL Parameters function. If the ‘Control Voucher Entry Mode’ is selected as ‘During Refund Request Entry’, this field is not mandatory.

  4. Date – Click Calendar function to select the date. The date indicates when the refund request is approved.

  5. Process] –Click [Process] to approve the request.

  6. [Cancel] – Click [Cancel] to clear the records and cancel the transaction.
