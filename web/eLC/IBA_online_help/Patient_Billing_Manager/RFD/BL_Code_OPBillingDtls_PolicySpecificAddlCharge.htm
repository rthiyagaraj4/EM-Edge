Policy Specific Additional Charge

Policy Specific Additional Charge tab enables you to define additional charge for specific policy, payer group and payer combination. The additional charges are applicable, if the patient financial details are modified. The additional charges are calculated based on the parameter defined. The patients referred from other facilities holding the SSU card are charged the doctor fee based on the prices defined for the service which is associated to the policy. If the patient is holding a private insurance, the additional doctor fee that has to be charged on the patient has to be defined.  

 

Enter the following details to define additional charges for specific policies:

 

  1. Billing Group – Click [List of Values] to select a Billing Group. The Billing Groups defined in the Patient Billing Manager module get listed. The Billing Groups with the ‘Settlement Indicator’ as ‘Sponsored Patient’ and the ‘Payer Settlement Type’ as ‘Payment by Insurance Company’ get listed.

  2. Payer Group – Select a Payer Group from the list. The values defined using the function Customer Group in the Patient Billing Manager module get listed.

  3. Payer - Select a Payer from the list. The values defined using the function Customer in the Patient Billing Manager module get listed.

  4. Policy Type Code -  Click [List of Values] to select a Policy Type. The Policy Types defined in the Insurance Management module get listed.

  5. Description – The description for the selected Policy Type gets displayed.

  6. Charge Type – Select a value from the list. The values are system defined – ‘Charge to Patient Only’ and ‘As per Policy Setup’. The charges for the selected policy are computed based on the charge type selected. If the patient is not a insurance payer, the charges are computed based on the standard prices.

  7. Billing Service - Click [List of Values] to select a Billing Service. The additional charge for the policy type is calculated based on the service associated. The values defined using the Billing Service function in the masters get listed.

  8. Description – The Description for the selected Billing Service gets displayed.

  9. Clinic Code - Click [List of Values] to select a Clinic Code. The Clinics defined in the Outpatient Management module get listed.

  10. Description – The Description for the selected Clinics gets displayed.

  11. Visit Type – Click [List of Values] to select a Visit Type. The Visit Types defined and associated to the Clinics in the Outpatient Management module get listed.

  12. Description – The Description for the selected Visit Type gets displayed.

  13. Physician ID - Click [List of Values] to select a Physician. The Physicians defined and associated to the Facility in the Application Masters module get listed.

  14. Physician Name – The name of the selected Physician gets displayed.

  15. Charge Type - Select a value from the list. The values are system defined – ‘Charge to Patient Only’ and ‘As per Policy Setup’. The charges for the selected policy are computed based on the charge type selected. If the patient is not a insurance payer, the charges are computed based on the standard prices.

  16. Billing Service - Click [List of Values] to select a Billing Service. The additional charge for the policy type is calculated based on the service associated. The values defined using the Billing Service function in the masters get listed.

  17. Description – The Description for the selected Billing Service gets displayed.
