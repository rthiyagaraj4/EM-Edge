Enter the following details to query patient records:

 

Episode Type

  1. All - Select the check box to query Inpatient and Daycare patient records matching the search criteria. On selecting the check box, the Inpatient and Daycare check boxes are selected automatically. Clear the check box to clear Inpatient and Daycare check boxes.

  2. Inpatient - Select the check box to query only inpatient records. The check box is selected automatically when you select 'All'. Clear the check box, if the inpatient records should not be listed.

  3. Daycare - Select the check box to query only daycare patient records. The check box is selected automatically when you select 'All'. Clear the check box, if the daycare records should not be listed.

  4. Adm Date [From - To] - Click the Calendar function to select a range of admission dates. The inpatient and daycare patient records who are admitted to the facility during the selected date range are listed.

  5. Discharge Date [From - To] - Click the Calendar function to select a range of discharge dates. The inpatients' or daycare patients' who are discharged during the selected date range are listed.
