Enter the following details to define a Mapping Code and to associate either the Material Items or the Billing Services:

 

  1. Facility ID – Select a Facility from the list. The facilities associated to the logged in User gets listed. The mapping code defined with the Billing Services and the Material Items is relevant to the selected facility.

  2. Service Indicator – Select a Service Indicator from the list. The values are system defined – ‘Billing Service’ and ‘Material Item’.

  3. Billing Service – Select Billing Service to associate Billing Services to the mapping code. The Billing Services associated to the mapping code are printed during bill generation.

  4. Material Item – Select Material Item to associate Material Items to the mapping code. The Material Items associated to the mapping code are printed during bill generation.

 

Mapping Code

Using this tab, you can define Mapping Codes. A Mapping Code is pertinent to a selected set of Billing Services or Material Items. You can define a range of Mapping Codes for different age range. The coverage amount differs for the Mapping Codes of different age range. Mapping Code also varies for a combination of material items based on which the coverage amount differs. You can associate different Mapping Codes to the same Material Item.

  1. Mapping Code – Enter the Mapping Code in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Level Applicable for Coverage – Select the check box to define service or item level coverage for a Mapping Code. The check box is selected, by default. If the check box is selected, you can record coverage limit for mapping code through the ‘Setup Material Item Coverage Detail by Policy’ function in the Insurance Management module. The check box ‘Level Applicable for Coverage’ remains selected under the ‘Service/Items for Mapping Code’ tab when you associate Items or Billing Services to the Mapping Code. Clear the check box, if the level coverage is not applicable for Mapping Code. The coverage level is valid for OT related Material Items only where the Mapping Code is associated to the level of surgery.

  3. Physician & Diagnosis Applicable for Coverage – Select the check box to define if the Physician and Diagnosis charge associated to the Mapping Code is applicable for coverage. The check box ‘Physician & Diagnosis Applicable for Coverage’ remains selected under the ‘Service/Items for Mapping Code’ tab when you associate Items or Billing Services to the Mapping Code. This is valid for non-surgical items and the diagnosis recording is mandatory to recalculate the coverage amount. Clear the check box, if the coverage is not applicable for Physician and Diagnosis charges.  The Physician & Diagnosis coverage level is valid for Material Items only. You can specify the Term Set ID, Term Code and Physician ID in Insurance Management module only if the check box is selected.

You can either select the check box ‘Level Applicable for Coverage’ or ‘Physician & Diagnosis Applicable for Coverage’.

  1. Long Description – Enter the Long Description in alphanumeric format of 100 characters maximum length. This is a mandatory field.

  2. Short Description – Enter the Short Description in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. [Service/Items for Mapping Code] – Click [Service/Items for Mapping Code] to switch to the next tab to associate either Billing Services or Material Items to the mapping code.

 

Service/Items for Mapping Code

Using this tab, you can associate Material Items or Billing Services to the Mapping Codes.

  1. Service/Item Code – The Mapping Code defined and highlighted under the Mapping Code tab gets displayed. You cannot change the Service/Item Code.

  2. Description – The Long Description recorded for the selected Service/Item Mapping Code gets displayed.

  3. Code – Click [List of Values] to select a Code from the list. The values in the list differ based on the Service Indicator selected.

  4. Description – The Description for the selected Billing Service or Material Item gets displayed.

  5. Level Applicable for Coverage – The check box is either selected or cleared based on the setup in the ‘Mapping Code’ tab for respective Mapping Code. You cannot clear or select the check box.

  6. Physician & Diagnosis Applicable for Coverage – The check box is either selected or cleared based on the setup in the ‘Mapping Code’ tab for respective Mapping Code. You cannot clear or select the check box.
