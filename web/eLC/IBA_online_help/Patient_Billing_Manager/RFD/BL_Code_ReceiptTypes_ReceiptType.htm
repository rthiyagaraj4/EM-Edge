Receipt Type

Enter the following details to define Receipt Types:

 

  1. Accounting Entity - The Accounting Entity code defined using the ‘Accounting Parameters’ function in the ‘ Financial Setup’ menu in the System Manager module gets displayed. The Accounting Entity is set by default and the field is disabled.

 

Receipt Type

  1. Receipt Type - Enter the Receipt Type in alphanumeric format of three characters maximum length. This is a mandatory field.

  2. Description - Enter the description in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the Short Description in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Receipt Nature - Select a Receipt Nature from the list. The Receipt Nature is system defined and the values are 'AR Receipt', 'Against Bill', 'Credit Card Company Payment', 'Credit Card Refund Deduction', 'Customer Receipt', 'DCP Receipt', 'Installment Payment', 'Healthcard Receipt', 'Miscellaneous', 'Patient Deposit', 'Prepayments' and 'Refundable Deposit'. This is a mandatory field.

  5. AR Receipt (AR) – Select AR Receipt, if the receipt is generated when the amount is posted to the Accounts Receivable module.

  6. Against Bill (BI) – Select Against Bill, if the patient or a customer clears the amount during bill generation.

  7. Credit Card Company Payment (CR) - Select Credit Card Company Payment, if the Credit Card Company is paying the amount on behalf of the patient.

  8. Credit Card Refund Deduction (CD) - Select Credit Card Refund Deduction, if the receipt is generated for a credit card transaction on deducting the credit card deduction fee.

  9. Customer Receipt (CU) - Select Customer Receipt, if the Customers are settling a part of the bill amount on the patient’s behalf. For example, an Insurance Company paying on behalf of the patient is considered to be a Customer.

  10. DCP Receipt (DC) - Select Deferred Cash patient, if patient does not settle the bill amount during bill generation.

  11. Installment Payment (IN) – Select Installment Payment, if the receipt has to be generated for the installment payments.

  12. Healthcard Receipt (HC) – Select Healthcard Receipt, if the patient has a Healthcard offered by the Facility. Using a Healthcard, patient is able to avail the services at a discounted price.

  13. Miscellaneous (MI) – Select Miscellaneous to indicate any additional services availed from the Facility.

  14. Patient Deposit (AD) – Select Patient Deposit, if the patient has paid a deposit amount.

  15. Prepayments (PR) – Select Prepayments, if the patient has pre-paid a part of the service charges. The prepayments amount received is adjusted with the bill amount.

  16. Refundable Deposit (RD) - Refundable Deposit refers to the amount to be reimbursed to the patient.

  1. Receipt Nature Description - The description for the respective receipt nature appears on selecting the Receipt Nature.

  2. [Interface to GL] – Click [Interface to GL] to switch over to the next tab.
