Billing Grp Specific Tab

The Billing Grp Specific tab enables you to set up prices for various patient categories and associate it to a relevant Billing Group.

 

Enter the following details to define the price:

 

  1. Facility – Select a Facility from the list. The facilities associated to the user get listed. The prices defined for the patient category is applicable only within the facility selected. This is a mandatory field.

  2. Blng Group – Click [List of Values] to associate a Billing Group to the facility. This is a mandatory field.

  3. Description – The Description for the selected Billing Group is displayed.

  4. Valid From - Click Calendar function to select a date. The prices are valid for a Billing Class associated to a Billing Group from the date specified.

  5. Valid To - Click Calendar function to select a date up to when the discount percentage defined for the patient category is valid.

  6. Blng Class – Click [List of Values] to select a valid Billing Class. The Billing Classes defined under the ‘Billing Rule’ menu in the Masters gets listed. The service price for the patient category can be defined only if a valid Billing Class is selected. This is a mandatory field.

  7. Description – The Description for the selected Billing Class is displayed.

  8. R/F – The values are system defined. Select 'R' for rate based and 'F' for factor based service.

  9. IP Rate – Enter the base price to be charged for the Inpatients. The Inpatients associated to the relevant Billing Group and admitted in the respective Facility are charged based on the quantity of the service used.

  10. DC Rate – Enter the base price to be charged for the Daycare patients. The Daycare patients with the selected Billing Group and the respective Facility are charged an amount based on the quantity of the service used.

  11. OP Rate - Enter the base price to be charged for the Outpatients. The charge towards the service is calculated accordingly based on the quantity used.

  12. Emg. Rate - Enter the base price to be charged for the Emergency cases. The charge towards the service is calculated accordingly based on the quantity used.

  13. Ext. Rate - Enter the base price to be charged for the External or the Referral patients. The charge towards the service is calculated accordingly based on the quantity used.

  14. Min. Charge - Enter the minimum charge towards the service. The charge towards the service is calculated accordingly based on the quantity used. This is a mandatory field.

  15. Max Charge – Enter the maximum charge towards the service. This is applicable if the Cost Indicator is set and a maximum amount is charged for the service. The service amount cannot be beyond the limit specified for the service.

  16. Allowed – Select the check box, if the Billing Service is reimbursable for the selected patient class. Clear the check box, if the respective Billing Class is non-reimbursable for the selected patient class.

  17. Override – Select the check box, if the respective Billing Service is reimbursable for the selected patient class. The override option for the patient class takes precedence over the override check box in the header field. If the service is not reimbursable and if the check box is selected for the Patient Class, it indicates that the Billing Service is reimbursable for the selected patient class.

  18. Reimbursable Amt – Enter the amount to be refunded to the patient on producing valid documents.

  19. Base Qty – Enter the Base Quantity towards the service. This is a mandatory field. The charge is calculated based on the quantity used.

  20. Gross/Net Applicability – Select either ‘Gross’ or ‘Net’ from the list.

  21. Gross – Select Gross to charge the Gross amount for the service. The charge towards the service is calculated based on the discount and the price defined for the patient category associated to the Billing Class.

  22. Net – Select Net to charge the Net amount for the service. The service charge is calculated based on the net amount, that is, the discount amount is not considered. The price towards the patient category associated to the Billing Class is only taken into consideration.

  1. Check for Cost Availability – Select the check box, if the Cost Amount is applicable. Clear the check box, if the Cost Amount is not applicable.

  2. Apply Discnt – Select the check box, if a discount is applicable to the selected Billing Class. The Discounts tab is enabled, if the check box is selected. Clear the check box, if the discounts are not applicable.

 

Discounts

The Discounts tab enables you to define the discount percentage applicable to the patient category associated to Billing Class and the Billing Group in the Facility.

 

Enter the following details to define discounts:

 

  1. Billing Group - The Billing Group for which the prices are defined is set, by default. You cannot change the Billing Group, but only define the discounts for the patient category. The description for the selected Billing Group gets displayed in the adjacent text box.

  2. Billing Class - The Billing Class associated to the relevant Billing Group is displayed. You cannot change the Billing Group, but only define the discounts for the patient category.

  3. Valid From – The beginning date set under the Prices tab for the Billing Group is displayed.

  4. Valid To - The end date up to when the prices are valid for the relevant Billing Group is displayed.

  5. Discount Indicator – Select a value from the list to issue discount for the Billing Service. The system-defined values are ‘Amount’ and ‘%’. Select Amount to specify the amount that has to be issued as a discount amount on the Billing Service. Select Percentage to specify the percentage of amount that has to be issued as a discount amount on the Billing Service.

  6. Valid From – Click Calendar function to select a date. The discounts are applicable for the Billing Classes in the Facility from the date selected. This is a mandatory field.

  7. Valid To - Click Calendar function to select a date. The discounts are applicable for the Billing Classes in the Facility up to the date selected.

  8. DC % - Enter the value to issue discount towards the service for Daycare patients. This is applicable only to the Daycare patients associated to the relevant Billing Group and Billing Class for the specified period. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  9. IP % - Enter the value to issue discount towards the service for Inpatients. This is applicable only to the Inpatients associated to the relevant Billing Group and Billing Class for the specified period. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  10. OP % - Enter the value to issue discount towards the service for Outpatients. This is applicable only to the Outpatients associated to the relevant Billing Group and Billing Class for the specified period. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  11. EXT % - Enter the value to issue discount towards the service for External or Referral patients. This is applicable only to the External or the Referral patients associated to the relevant Billing Group and Billing Class for the specified period. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  12. EMG % - Enter the value to issue discount towards the service for Emergency patients. This is applicable only to the Emergency patients associated to the relevant Billing Group and Billing Class for the specified period. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.
