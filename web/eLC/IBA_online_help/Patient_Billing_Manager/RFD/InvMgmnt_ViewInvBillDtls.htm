Enter the following details to view invoice/bill details:

 

  1. Customer - Click [List of Values] to select a Customer. The results are filtered based on the Customer selected.

  2. Policy Type - Click [List of Values] to select a Policy Type. The results are filtered based on the Policy Type selected.

  3. Patient ID - Click [List of Values] to select a Patient ID. The results are filtered based on the Patient ID selected. The name of the patient, age national ID are displayed.

  4. Visit/Discharge Date [From - To] - Click the Calendar function to select the date range. The patient records matching the search criteria are listed.

 

Episode Type

  1. All - Select All to query all patient records. The check boxes for Outpatient, Emergency, Daycare, Inpatient and External are selected automatically when you select 'All.'. Clear the check box to query only selected Episode Types.

  2. Outpatient - Select the check box to query Outpatient records. Clear the check box to exclude Outpatient records.  

  3. Emergency - Select the check box to query Emergency patient records. Clear the check box to exclude Emergency records.  

  4. Daycare - Select the check box to query Daycare patient records. Clear the check box to exclude Daycare patient records.  

  5. Inpatient - Select the check box to query Inpatient records. Clear the check box to exclude Inpatient records.  

  6. External - Select the check box to query External patient records. Clear the check box to exclude External patient records.  

  7. Episode ID - Click [List of Values] to select an Episode ID.

 

Encounter Details for Invoices

The encounter details are displayed. The name of the patient, ID, encounter, account sequence number, priority, Customer code and description, Policy Type and description, Admission and Discharge date are displayed. Click [V] to capture additional invoice information.  

 

Invoice/Debit Note/Credit Note Details

The invoices generated for the outstanding bills of the selected Customer get displayed. The Invoice Number, invoice generation date, payable amount, amount settled, and the payer details are listed. You can also view the document reference details. If the invoices are consolidated, detailed information on the invoices that are consolidated can also be viewed. You can also view the prepaid amount received from a payer for an invoice that is generated. The individual transaction records are displayed along with the document number, document date and document amount. Click [D] to view the Service Details for each bill generated in the facility for the selected Customer.

 

Bill Details

The bill details for the services utilized are displayed. You can select the invoices under ‘View All Transactions’. On clicking a particular record, the bill details associated to the selected invoice are listed under ‘Bill Details’.
