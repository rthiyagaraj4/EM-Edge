Calculation/Recalculation Rules by Billing Group/Category Criteria

Enter the following details to define the inpatient level of service eligibility:

  1. Eligibility in Use –Select the check box, if the eligibility is applicable for the inpatients in the facility. Clear the check box, if the service eligibility is not applicable for the inpatients in the facility. The check box is selected, by default.

 

Calculation/Recalculation Rules by Billing Group/Category Criteria

  1. Billing Group – Click [List of Values] to select a Billing Group. The values defined using the Billing Group function get listed. The service eligibility rule is valid for the selected Billing Group and the associated Billing Services. This is a mandatory field.

  2. Description – The Description for the selected Billing Group gets displayed.

  3. Category – Select the Category from the list. The values are system defined – 'Unrestricted', 'Nationals' and 'Others'.

  4. Unrestricted – Select Unrestricted, if the eligibility rule is applicable to the selected Billing Group and all the patients visiting the facility. The patient can be a valid citizen of the country or a non-national.

  5. Nationals – Select Nationals, if the eligibility rule is applicable to the selected Billing Group and the citizens of the nation. The Billing Group is not applicable to other patients visiting the facility.

  6. Others – Select Others, if the eligibility rule is applicable to the selected Billing Group and the non-national patients.

 

Transfer Recalculation Rule

  1. On Transfer: Recalculate – Select the check box, if the charge has to be re-calculated during patient transfer from one Bed Type to another or from a Nursing Unit to another.

  2. Incl/Excl Basis – Select a value from the list. The values are system defined – 'Include', 'Exclude', 'Specific' and 'Not Applicable'.

  3. Include – Select Include and the bed computation charge is re-calculated when the patient is transferred.

  4. Exclude – Select Exclude and the bed computation charge is not re-calculated when the patient is transferred.

  5. Specific – Select Specific and the bed computation charge is not re-calculated for selected Billing Services only.

  6. Not Applicable - Select Not Applicable and the bed computation charge is not applicable for transfers.

 

Bed Computation Exception Criteria

  1. Billing Service – Click [List of Values] to select a Billing Service. The values defined using the Billing Services function get listed. The bed charges are computed based on the prices defined for the associated Billing Service. The bed computation charges are re-calculated based on the Billing Service and the Billing Class associated to the patient. This is a mandatory field.

  2. Description – The Description for the relevant Billing Service gets displayed.

  3. Billing Class Ind – Click [List of Values] to select a Billing Class. The bed charges are exempted for the selected Billing Class. This is a mandatory field.

 

Recalculation Criteria

  1. Billing Service – Click [List of Values] to select a Billing Service. The values defined using the Billing Services function get listed. The bed charges are re-computed based on the prices defined for the associated Billing Service. This is a mandatory field.

  2. Description – The Description for the selected Billing Service gets displayed.

  3. ICU/CCU Related – Select the check box, if the selected Billing Service is ICU or CCU related. The price for the transfer is re-computed based on the prices defined for the Billing Service, if the check box is selected.

  4. Incl/Excl – Select a value from the list. The values are system defined – 'Include' and 'Exclude'.

  5. Include – Select Include and the bed computation charge is re-calculated when the patient is transferred.

  6. Exclude – Select Exclude and the bed computation charge is not re-calculated when the patient is transferred.
