Update ARV Code

From this function you can Update ARV code of a patient. You can Query to view the existing records.

The Update ARV Code function is used to update the ARV code of a patient in the facility. You can also query to view the ARV code of a patient.

Updating ARV Code

From the SS Invoice Management > Transaction, double-click on Update ARV Code function.

The Update ARV Code screen gets displayed.

  1. Click [List of Values] to select a patient ID. This is a mandatory field. The name of the patient, age, gender and nationality gets displayed on selecting the patient ID.

  2. Enter the ARV Code in alphanumeric format of 20 characters maximum length. This is a mandatory field.

  3. Click [Commit Form] or press [F10] to save or click [Clear Record] to clear the entered data.

  4. Click [Execute Query] or press [F8] to search for the existing records within the facility.
