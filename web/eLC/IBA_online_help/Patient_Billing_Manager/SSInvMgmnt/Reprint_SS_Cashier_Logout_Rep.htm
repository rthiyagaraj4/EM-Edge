Reprint SS Cashier Logout Report

From this function you can Reprint a Cashier Report.

The Reprint SS Cashier Logout Report function enables you to re-print a Cashier report. The Cash Counter login and logout details are listed along with date and time. You can choose to reprint the logout report for a selected Cash Counter shift. The report displays the sponsor's name, Cash Counter code, shift timings, name of the user and total number of receipts, active and cancelled receipts. The receipt numbers are also displayed along with the settlement details.

Re-printing SS Cashier Logout Report

From SS Invoice Management > Reports, double-click on Reprint SS Cashier Logout Report function.

The  Reprint SS Cashier Logout Report screen gets displayed.

  1. Click the Calendar function to select the transaction date. The values in other fields get populated based on the transaction date selected.

  1. Click [List of Values] to select the User ID. The name of the Cashiers' who has performed transaction in the Cash Counter on the selected date is listed.

  2. Click [List of Values] to select the Cash Counter. The login and logout date and time are listed along with the Cash Counter code and name of the Cashier. The 'Login Date & Time' and the 'Logout Date & Time' are displayed automatically on selecting the Cash Counter.

  3. Click [Run] to generate the report.
