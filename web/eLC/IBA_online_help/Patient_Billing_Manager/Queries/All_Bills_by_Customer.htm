All Bills by Customer

From this function you can Query bills against selected customer. You can also search for a bill against a selected customer and display the entire bill with bill status and print status.

The All Bills by Customer is used to Query bills against selected customer and display the entire bill with bill status and print status once customer is selected from customer lookup.

Querying to view all the bills by customer in the facility

From the Queries > Bills, double-click All Bills by Customer function.

The Query All Bills screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Refresh] to view the bill details. The service and the bill details associated to the selected patient get displayed in the respective text fields.

  3. The Bill Details get displayed.

    The following details are displayed:

     

    1. Facility - The name of the Facility where the patient is registered.

    2. Doc Type – The document type defined for the selected patient.

    3. Doc No – The document number. associated with the Doc Type.

    4. Doc Date – The date of the document prepared.

    5. Trx – The transaction type of the patient bill.

    6. Nat – The nature of the patient bill.

    7. Status – The status of the patient bill.

    8. Amount – The Amount of the patient bill.

    9. Dep & Prepayment - The deposits and prepayments done by the patient.

    10. Bill Outst - The bill outstanding amount is displayed. If the patient makes partial receipts, then the amount and the outstanding amounts will be different by the partial receipt amount.

    11. DCP Outst - The Deferred Cash Patient outstanding amount

    12. Printed - If the bills are already printed, this check box gets selected.

    13. Episode Type – The Episode type of the patient.

    14. Episode No – The Episode number

    15. Visit No. – The visit number of the patient.

    16. Bill Trx Type Description – The description of the bill transaction done by the patient.

    17. Bill Nature Type Description – The description of the bill nature.

    18. Bill Status Description – The description of the bill transaction status.

  4. Click [Dtls] to view the transactions which are deferred.

  5. Click [Print] to print the selected records. A printer has to be defined for the selected report in order to print the records.

  6. Click [More] to view the detailed bill information.

  7. Click [Clear Record] to reset the information
