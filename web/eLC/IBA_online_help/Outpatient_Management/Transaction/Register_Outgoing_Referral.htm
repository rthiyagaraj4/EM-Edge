Register Outgoing Referral

From this function you can Create outgoing referral and book appointment for a patient with the encounter.

This function helps in creating outgoing referrals for the patient. When referral is not made in Close Consultation, Discharge Patient, OP Checkout and AE Checkout or Prepare Discharge Advice functions, it can be done in Register Outgoing Referral function.

 

Process Flow Diagram

Registering Outgoing Referral

From the Outpatient Management> OP Transactions, select Register Outgoing Referral function.

The Register Outgoing Referral screen gets displayed.

  1. Search and select the required Patient ID to create a referral. All the encounters of the selected patient gets displayed.

  2. Click Encounter ID hyperlink, the referral section gets displayed.

  3. Enter the Referral Details in the mandatory (*) fields. <Related Field Description>

  4. Click [Record] to save the transaction on entering the required details or [Clear] to clear the entered data. The Internal Reports dialog gets invoked with the option to print the Referral Letter. The Referral ID gets generated for the patient.

Note:

1. If you have selected the Quick Admission check box in this page, the Patient Admission page is displayed when you click this button.  The referral details will be displayed by default, in the page.

2. On completing the admission, the billing details will be displayed in a popup window based on whether the patient is citizen or non-citizen. On clicking OK, the page from where the quick admission process started, will be displayed; for example, AE Checkout page or the OP Check out page.

3. If you are unable to complete the quick admission, the focus will return to the referral page. (See Warnings )The referral page will display the created referral ID. The column Appt No./Encounter ID  will display the hyperlink “Quick Admission”.  

4. You can open the screen anytime and click this hyperlink and complete the Quick Admission process. Once the admission is completed the corresponding encounter ID will be displayed in place of the “Quick Admission” hyperlink.

5. If you try to create a Quick Admission Referral in the Register Outgoing Referral screen for an inpatient, the following error message will be displayed on launch the Admit Patient screen: “This patient is currently an Inpatient”.

Warnings:

1. You cannot admit inpatients; the following error message will be displayed on launch the Admit Patient screen: “This patient is currently an Inpatient”.

2. You cannot admit a patient using an existing Quick Admission Referral number; the following error message will be displayed: “Invalid Referral ID”.

3. If you create a quick admission referral, but then un-check the Quick Admission check box for the particular operation station in Operator Station setup function, you will not be able to admit the patient. The following message will be displayed:

“Operator Station not defined for Quick Admission.”

  1. Click Referral ID hyperlink, the Referral Details dialog gets displayed.

    On click of Referral ID, the Patient ID, Patient Name, NRIC, Alternate ID details, Birth Place, Age, Gender, Birth Date, Residency, Status, Nationality, Race and Ethnicity gets displayed. You cannot change the displayed values.

    Referred From tab

    Referral Letter No, Received Date, Referred Date, Preferred Date, Reason, Type, Referral Source, Speciality, Location Type, Location, Practitioner, Tel No, Mobile No and Pager No gets displayed from the details captured.

    1. [Clinical Details] –Click [Clinical Details], the Clinical Details screen gets displayed. Enter the clinical details of the patient referred.

    2. [Notes] – Click [Notes] to open a screen to enter the referral notes in alphanumeric format and the Notes screen gets displayed.

    3. Other Doc Attached – The details captured in Clinical Details dialog gets displayed.

    Referred To tab

    On click of Referred To tab, the following details: Speciality, Location Type, Location, Practitioner, Priority, Type, Referral Source, Service, Location, Orderable/Procedure, Referred for IP, Healthcare Setting Type gets displayed. You cannot change the displayed values.

  2. Click Book Appointment hyperlink to book appointment for the patient.
