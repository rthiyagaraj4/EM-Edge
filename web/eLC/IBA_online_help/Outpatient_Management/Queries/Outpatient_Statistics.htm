Outpatient Statistics

From this function you can Query all the visits with details of type of visits that are generated either as a monthly listing or as annual listing.

One of the indicators to measure the performance of Outpatient services can be volumes of patients visiting clinics. This function is used to list out information that represent volume of visits registered against a Clinic. The function lists out the total number of visits with visit type details. You can generate either the monthly listing or a annual listing. The list can be used to compare volumes between various clinics across the facility and can be used to allocate resources by forecasting based on historical data.

Querying for Outpatient Statistics

From the OP Queries, select the Outpatient Statistics function.

The Outpatient Statistics screen gets displayed.

  1. Enter the details in the mandatory (*) field or enter the search criteria. <Related Field Description>

  2. Click [Search] to select a specific record or [Clear] to clear the records displayed in the result area.

  3. If the statistics are to be viewed by Services, click [By Services].

  4. If the statistics are to be viewed by Location, click [By Location].

  5. Click [Reset] to clear the entered data.
