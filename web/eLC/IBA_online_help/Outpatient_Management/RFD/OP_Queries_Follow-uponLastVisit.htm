Enter the following details to query the follow-up visits:

 

  1. Date [From-To] - Enter the date range for which the visits are due, in dd/mm/yyyy format or select the date using the Calendar function. The ‘To’ date has to be greater than the ‘From’ date. The current system date is displayed, by default.

  2. Specialty – Enter the specialty in alphanumeric format or select the specialty for which the visits are due, using [?] function to invoke the search screen for selecting the required specialty.

  3. Service - Enter the service in alphanumeric format or select the Service for which the visits are due, using [?] function.

  4. Clinic - Enter the clinic in alphanumeric format or select the Clinic for which the visits are due, using [?] function.

  5. Practitioner - Enter the practitioner name in alphanumeric format or select the Practitioner for whom the visits are to be considered, using [?] function.

  6. Patient ID - Enter the Patient ID in alphanumeric format of 10 characters maximum length or click [?] function to select the Patient ID.

  7. Visit Created By - Enter the name of the logged-in-user in alphanumeric format of 30 characters maximum length or select the logged-in-users name from the list of values using [?] function. The logged-in-users created using ‘User’ function in System Manager module gets listed for selection.

  8. Visit Date – Enter the date of visit in dd/mm/yyyy format or select it using calendar function.
