Enter the following details to create a Queue Numbering Control record:

 

  1. Location Type - Select the Location Type from the list of valid values. All the system defined and user defined Care Location Types associated to Indicator ‘Room’ and ‘Clinic’ in Application Masters module are listed. This field is mandatory.

  2. Location - Select the Location from the [?] function. This field represents the Location for which the Queue Numbering Record is to be defined. The displayed list is limited to the Locations of the logged in facility and to the selected ‘Location Type’. This field is mandatory.

  3. Practitioner - Enter the Practitioner Name in alphanumeric format of 30 characters maximum length or click [?] to invoke a search screen to look for the required practitioner. This field represents the Practitioner who serves the selected Location, for whom the queue numbering record is to be defined. The displayed list is limited to the Practitioners of the logged in facility who serves the selected location. This field is mandatory.

  4. Room – Select the required room for which the queue numbering is defined from the list. All the Rooms associated to the selected location in ‘Resource for Location’ function are listed in addition to a value ‘All’ which can be chosen to indicate a single numbering control for all rooms.

  5. Other Resource – Select the Other Resource type from the list. According to the value selected, the corresponding values get populated in the list box. These values are populated from the Application Masters module.

 

Assigned Queues

  1. Walk-Ins – Select the Walk-Ins from the list. The values get populated based on the ‘Walk-ins’ check box in the ‘Queue Series’ function.

  2. Referral – Select the Referral from the list. The values get populated based on the ‘Referral’ check box in the ‘Queue Series’ function.

  3. Appointment – Select the Appointment from the list. The values get populated based on the ‘Appointment’ check box in the ‘Queue Series’ function.
