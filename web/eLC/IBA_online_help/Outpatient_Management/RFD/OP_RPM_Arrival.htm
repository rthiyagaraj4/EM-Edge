Enter the details to record arrival of a patient to a care location:

 

  1. Location – Select the location to which patient arrival has to be marked from the list boxes provided. The fist list box lists all location types and based on selection of location type, the second list box lists all valid care locations that are derived based on the operator station applicable to the logged in user.

  2. Encounter - Select the Encounter type from list box provided. The valid list of values are ‘Outpatient’, ‘Inpatient’, ‘Daycare’ and ‘Emergency’. Based on the selection, enter the valid Encounter ID to filter the patient listing based on encounter ID.

  3. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in the MP Parameter. The user can also query for the patient details by invoking the [?] function. On entering the relevant details in the lookup screen, the result screen will be populated with patient details that satisfy the entered query details. The user can then select the patient by clicking on the hyperlink of the patient. The control now shifts back to the ‘Arrived’ screen with the patient details populated.

  4. Encounter Date – Enter the date of encounter to list all patients who have had encounters on this date. System defaults current date and can be backdated if required.

  5. Order By – Select the order by from the list box provided. This is to group the listing based on either Encounter date or Encounter Type.

  6. All Encounters – Check the box to include all encounters.
