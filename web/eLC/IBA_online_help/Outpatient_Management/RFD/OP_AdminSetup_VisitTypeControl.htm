Enter the following details:

 

  1. Clinic - Select the Clinic using [?] function. All the Clinics created using 'Clinic' function for the facility are displayed for selection. This is a mandatory field.

  2. Service - Select the Service from the list. All the Services associated to the selected clinic are listed for selection. This is a mandatory field.

  3. No of Follow-up Visits Allowed - Enter the number of follow up visits allowed in numeric format of six digits maximum length.

  4. Duration for Follow-up Visit - Enter the duration for follow up visits in numeric format of six digits maximum length.
