Enter the following details to create a Visit Type record:

 

  1. Code – Enter the Visit Type code in alphanumeric format of two characters maximum length. This code is used to identify the Visit Type of the Outpatient on Registering a Visit. The entered code has to be unique and cannot be changed after creation. This field is mandatory.

  2. Long Description - Enter the long description of the Visit Type in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description – Enter the short description of the Visit Type in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Indicator - Select the Visit Type Indicator from the list of valid values. These values are system defined and the list of values provided are:

  5. First – Initial/new visit for a problem/treatment.

  6. Follow-up – Return/repeat visit as a consequence of the first visit.

  7. Routine – Routine/one-time visit.

  8. Emergency – A visit when a patient requires immediate medical attention; such visits are specific to Clinics, which are set up to handle such emergency situations.

  9. Series – A visit that is part of a recurring series of visits necessary for the healthcare unit/treatment.

  10. Referral for Consult – A visit to the Clinic against an existing encounter pertaining to an Inpatient/Outpatient Service Location.

  1. Max Appt Duration (in minutes) - Enter the Maximum Appointment Duration in numeric format of four digits maximum length. The value set here indicates the maximum appointment time that the patient can be given for the defined Visit Type when an appointment booking is made in the Appointment Scheduling module of Enterprise Management for time table of type ‘Free Format’.

If Maximum Appointment duration value is set up here for a Visit Type and if the time table type for a particular day of a location is set up as ‘Free Format’ (Appointment Scheduling module), then while booking an appointment in Appointment Scheduling module, on entering the ‘From Time’ hyperlink of the appointment, the system calculates the ‘To Time’ by adding the value entered for the selected Visit Type.

  1. Max Appt Slots - Enter the Maximum Appointment Slots in numeric format of three digits maximum length. The value set indicates the Maximum Appointment Slots that the patient can be given when an Appointment Booking is made for a specific Visit Type in Appointment Scheduling module for time table of type ‘Slot’.

If Maximum Appointment Slot value is set up for a Visit Type and if the time table type for a particular day of a location is set up as ‘Slot’, while booking an appointment, on selecting the ‘Visit Type’ of the appointment, the system takes the values of the appointment slots defined.

  1. Orderable Criteria – Select ‘Orderable Criteria’ from the list. The listed values are ‘Required’ and ‘Not Required’. This is used to identify if any Orderables have to be identified against the visit for an outpatient encounter.

  2. Enabled – Select the check box to enable the record and clear the check box to disable the record.

  3. Applicable to Radiology – Select the Applicable to Radiology check box to create Visit Type for booking radiology appointments. This check box is cleared, by default.

  4. Colour Indicator – Select the required colour from the colour palette using [?] function. By default, this field is blank.
