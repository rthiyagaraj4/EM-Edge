Blood Unit Component Preparation

From this function you can Create the components for the blood units and Modify the existing blood components.

The Blood Unit Component Preparation function is used to prepare components for the blood units and change the components that are prepared. You can search for the blood unit by specifying the donation date range or batch number. Select the required blood unit to prepare the component and record the weight of the blood bag. Components are created based on the defined donated product and blood bag weight rules. If the donation time has exceeded, the unit number is highlighted using color.

Preparing Blood Unit Components

From the Transactions > Blood Unit Management, double-click Blood Unit Component Preparation function.

The Blood Unit Component Preparation screen gets displayed.

  1. Enter the query criteria to search for blood units. <Related Field Description>

  2. Click [Display] to view the details of the units satisfying specified query criteria.

  3. Double-click the unit number to select a blood unit. The selected unit details are displayed on the right hand side frame. <Related Field Description>

  4. Click [Create Components] to create components from the selected blood unit. A confirmation message is displayed on the screen. If you have not filled the Measured Weight, an error message ‘Blood Bag Measured Weight Cannot Be null’ is displayed.

  5. Click [Change Components] to change the components of a selected blood unit. <Related Field Description>
