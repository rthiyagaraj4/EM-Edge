View Death Register

From this function you can Query all the patients discharged from the facility with the Discharge Type marked as ‘Deceased’.

This function gives a statistical listing of all the patients discharged from the facility with the Discharge Type recorded as 'Deceased'.

Querying to View Death Register

From the MR Queries, select the View Death Register function.

The View Death Register screen gets displayed.

  1. Enter the search criteria to select a specific record. <Related Field Description>

  2. Click [Search] to display the record or [Clear] to clear the records displayed in the result area.

  3. Click Patient ID hyperlink to view the Death Register Details. The Patient details, Death registration details and Diagnosis details of the patient as recorded in the transactions are displayed.

  4. Click [Reset] to clear the entered data.
