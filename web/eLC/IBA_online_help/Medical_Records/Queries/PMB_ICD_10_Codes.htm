PMB ICD 10 Codes

From this function you can Query the available ICD10 to PMB Code Mapping data.

This function allows you to view available ICD10 (International Classification of Diseases) to PMB (Prescribed Minimum Benefits) Codes mapping data based on the PMB Code or Term Code. The ICD10 to PMB Code Mapping data is uploaded into the EM Database using a backend process.

Querying PMB ICD 10 Codes

From the MR Queries, select the PMB ICD 10 Codes function.

The PMB ICD 10 Codes screen gets displayed.

  1. Enter the required query criteria <Related Field Description>

  2. Click [Execute] to view the PMB ICD 10 Codes matching the query criteria.

  3. Click [Reset] to clear the entered data.
