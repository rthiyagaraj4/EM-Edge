Maternal Deaths

From this function you can Query all the patients discharged with the discharge type as ‘Maternal Death’.

This function gives a statistical listing of all the patients discharged from the facility with the discharge type marked as ‘Maternal Death’.

Querying for Maternal Deaths

From the MR Queries, select the Maternal Deaths function.

The Maternal Deaths screen gets displayed.

  1. Click [Search] to display the records or enter the search criteria to select a specific record. Click [Clear] to clear the records displayed in the result area. <Related Field Description>

  2. Click Encounter ID hyperlink to view the Encounter Details.

  3. Click [Reset] to clear the entered data.
