Tabulation List

From this function you can Create a new Tabulation List applicable to the facility and Query for a desired tabulation list.

Tabulation Lists are used to group Diagnoses/Procedures for statistical analysis. The system allows generation of various types of morbidity and mortality statistical reports based on the tabulation lists created.

Creating a Tabulation List

From MR Code Setup, select the Tabulation List function.

The Tabulation List screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the data in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Tabulation List created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Tabulation List within the facility.
