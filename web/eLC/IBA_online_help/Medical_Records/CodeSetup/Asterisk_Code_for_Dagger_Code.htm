Asterisk Code for Dagger Code

From this function you can Create an association of asterisk codes to a dagger code and Query for the desired asterisk codes associated to a dagger code.

Dagger Codes represent conditions that are considered as the underlying cause of disease. Asterisk codes represent conditions that are the manifestations of the underlying cause. Asterisk Codes for Dagger Codes function aids to associate the ICD Codes indicated as Asterisk codes to those indicated as Dagger.

Creating an Association of Asterisk Codes to a Dagger Code

From MR Code Setup, select the Asterisk Code for Dagger Code function.

The Asterisk Code for Dagger Code screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Dagger Code using [?] function. On clicking the [?] function, an ICD Code Search screen is displayed search and select the required Dagger Code. This field is mandatory.

    The screen displays the tabs 'Custom List', 'Search' and 'Select'.

    Custom List

    The Diagnosis and Procedure groups created are listed in the Custom List from which you can select a diagnosis by clicking the hyperlink on the names of the groups and a custom list gets displayed.

    Click the hyperlink of the Code to select and the selected values appear in the Dagger Code field.

    Search

    Enter the search criteria and click [Search] to display codes based on the entered criteria.  Click the hyperlink of the Code to select it to appear in the 'Dagger Code' field.

    Select

    This is another way to search the Codes where there are four quadrants representing the levels of classification. The first quadrant represents the broadest level of classification. On clicking any of the classifications, all the broadest possible classifications existing in the selected sections appear in the second quadrant. Based on the selected hyperlinks of broader classification, the fourth quadrant will have the final individual diagnoses. Click the hyperlink to select the diagnosis to appear in the Dagger Code field.

  3. Click [Search] to display Asterisk Codes and associate the required Asterisk Codes to the selected Dagger Code.

  4. The Asterisk Codes are displayed in the alphabetical order. These alphabets are hyperlinked. You can select the required alphabet to view the ICD Code in the alphabetical order of ‘Title’.

  5. Select the required Asterisk Codes to associate them to the selected Dagger Code.

  6. Click [Apply] to save the association of Asterisk Codes to the Dagger Code or [Reset] to clear the entered data.

  7. Click [Query] to search for existing Asterisk Codes associated to Dagger Code.

 

Since an underlying cause can have more than one manifestation, it is useful if the possible manifestations are associated to the underlying cause.
