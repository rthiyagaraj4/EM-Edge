Allergy Type

From this function you can Create a new Allergy Type by associating it to an Allergy Class and Query for a desired Allergy Type.

An allergy is a reaction of your immune system to the allergens. An allergen is a substance that can cause an allergic reaction. The allergens are drug, dust, food items and others.

Creating an Allergy Type

From MR Code Setup, select the Allergy Type function.

The Allergy Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the data in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Allergy Type created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Allergy Type within the facility.
