ICD Code for Tabulation List

From this function you can Create an association of ICD Codes to a particular Tabulation List and Query for desired lCD Codes associated to a Tabulation List.

ICD Code for Tabulation List function is used to associate applicable ICD Codes to the tabulation list created. These Tabulation lists are used to generate reports on diagnosing patients.

Creating an association of ICD Codes to a Tabulation List

From MR Code Setup, select the ICD Code for Tabulation List function.

The ICD Code for Tabulation List screen gets displayed.

  1. Click [Create] to invoke the Create screen

  2. Enter the data in the mandatory (*) fields. <Related Field Description>

  3. Click [Search] to display the required ICD Codes. The ICD codes are populated depending on the 'Diag Code Scheme' associated to the Tabulation list. On entering the Search Description and searching for the ICD Code accordingly, the ICD Code gets displayed based on the search description only. If the search is performed irrespective of the Search Description, all the ICD Codes are displayed in alphabetical order. Click the required alphabet hyperlink to select ICD codes available under the selected alphabet.

  4. Select the check box against the required ICD Codes to associate them to the selected Tabulation List.

  5. Click [Apply] to save the association of ICD Codes to the Tabulation List or [Reset] to clear the entered data.

  6. Click [Query] to search for existing ICD Code associated to Tabulation List.
