Recording Status of Medical Report Request

From the MR Transactions > Medical Reports, select the Status tab.

The Status tab of the Medical Report screen gets displayed.

  1. Select the Medical Report Status as ‘Requested’ and click [Search] to display all Medical Report Requests or enter the search criteria and click [Search] to select the required Medical Report Request.

    1. Requester – Select the Requester using [?] function.

    2. Requester Type – Select the Requester Type from the list.

    3. Medical Report Status – Select Requested as the Medical Report Status.

    4. Request Date From-To – Select the date range of request placed.

    Patient Details

    1. Patient ID – Select the Patient ID using [?] function.

    2. Enter the required details in the fields specified as prompt in the MP Parameter of Patient Identity Management  module.

    3. Enter the values for the fields 'Previous HN', 'Government No', 'Driving License No' and 'Expatriate No' which are set in the MP Parameter under 'Alternate ID Checks' in the 'Master Patient Index' module.

  2. Click the Request ID hyperlink to invoke ‘Status of Medical Report Request’ screen and record the status of the Medical Report Request.

  3. Enter the details in the mandatory (*) fields. <Related Field Description>

  4. Click [Record] to save the Status of Medical Report Request or [Close] to exit from the ‘Status of Medical Report Request’ screen.

  5. On recording, if internal reports are defined, the Internal Reports screen gets displayed. Click [OK] to generate Internal Reports or [Cancel] to exit from the screen.
