Recording Status of Medical Board Request

From the MR Transactions > Medical Board, select the Status tab.

The Status tab of the Medical Board screen gets displayed.

  1. Select the Medical Board Status as ‘Requested’ and click [Search] to display all Medical Board Requests or enter the search criteria and click [Search] to select the required Medical Board Request.

    'Status' Tab

    Enter the following details to query on the status of the Medical Board.

    Requester Details

    1. Requester - Select the name of the requester from [?] function.

    2. Requester Type - Select the Requester Type from the list of valid values.

    3. Medical Board Status - Select the status of the Medical Board from the list to view the Medical Report of patients with the selected status. The listed values are predefined. The value is displayed as 'Not Delivered', by default.

    4. Request Date From/To - Select the date range when the Medical Report is requested.

    Patient Details

    1. Patient ID - Select the patient ID of the patient from [?] function to query on the status of the Medical Report.

    2. Gender - Select the Gender of the patient.

    3. Name Prefix - Select a prefix for the name from the list. The listed values are populated from the Master Patient Index module.

    4. Patient Name - Enter the name of the patient in alphanumeric format of 15 characters maximum length.

    5. Suffix - Select a suffix for the patient name from the list. The listed values are populated from the Master Patient Index module.

    6. Citizen ID - Enter the Citizen ID in alphanumeric format of maximum length as specified in the MP Parameter. The 'Prompt' specified as Citizen ID in MP Parameter appears as the field legend. The Citizen ID is displayed automatically on entering Patient ID of the registered patient. For an unregistered patient, this needs to be entered.

    7. Other Alternate Type - Select the other Alternate Type from the list of valid values. This displays all the Alternate ID's that have been defined in the MP Master setup other than for the default Alternate ID's.

    8. Other Alt ID - Enter the other Alternate ID in numeric format of maximum length as specified in the MP Parameter.

    Enter the values in the fields 'Previous HN', 'Government No', 'Driving License No' and 'Expatriate No' as set in the MP Parameter under 'Alternate ID Checks' in the Master Patient Index module.

    On entering the required details, click [Search] to display the results based on the search criteria. The 'Medical Board Status' is displayed as 'Not Delivered', by default, so the search result displays all the Medical Reports that are not delivered.

  2. Click the Request ID hyperlink to invoke ‘Status of Medical Board Request’ screen and record the status of the Medical Board Request.

  3. Enter the details in the mandatory (*) fields. <Related Field Description>

  4. Click [Formation of Medical Board] and the Formation of Medical Board screen is displayed to form the team members of the Medical Board.

  5. Click [Record] to save the Status of Medical Board Request or [Close] to exit from the ‘Status of Medical Board Request’ screen.

    If the 'Application Status' field in the ‘Status of Medical Board Request’ screen was selected as 'Complete' and the 'Formation Status' field in the ‘Formation of Medical Board’ screen was selected as 'Incomplete' or if the Medical Board was not formed, this record is available while selecting the 'Medical Board Status' field as 'Status Completed' in the 'Status' Tab of the 'Medical Board' screen.

    If the 'Application Status' field in the ‘Status of Medical Board Request’ screen was selected as 'Complete' and the 'Formation Status' field in the ‘Formation of Medical Board’ screen was also selected as 'Complete', on clicking [Record], this record is available while selecting the 'Medical Board Status' field as 'Medical Board Formed' in the 'Status' Tab of the 'Medical Board' screen.

  6. On recording, if internal reports are defined, the Internal Reports screen is displayed. Click [OK] to generate Internal Reports or [Cancel] to exit from the screen.
