The following details are displayed:

  1. Registration No - The request ID of the Medical Report is displayed as the registration number.

  2. Registration Date/Time - The date and time of placing the request for Medical Report is displayed.

  3. Click ‘View Encounter Details’ hyperlink to view encounter details.

 

Applicant Information

The applicant information as recorded in the 'Medical Report Request' screen is displayed and disabled. The fields cannot be edited.

  1. Click [Date Details] to view the Request Date and the Application Completion Date. Click [Close] to exit from the Date Details screen.

  2. Click [Charges Details] to view the charge details of the Medical Report as recorded. The charge details are displayed as read-only text. Click [Close] to exit from the Charge Details screen.

  3. Click [Applicant Address] to view the address details of the applicant.  Click [Close] to exit from the Contact Details screen.

  4. Click  [Check List] to view the check lists in read-only mode.  

 

Enter the following details:

  1. Report Type - Select the type of report from the list of valid values. Note Types created with Note Group as Prepare Medical Report in the Clinician Access module are listed for selection. This field is mandatory.

  2. Department/Specialty - Enter the Department or Specialty in alphanumeric format of 15 characters maximum length or select using [?] function. The values are populated from the Application Masters module. This field is mandatory.

  3. Remarks - Enter the Remarks in alphanumeric format of 200 characters maximum length.

  4. Forwarded HOD/Co-ordinator - Enter the name of the HOD or Co-ordinator to whom the request is forwarded or select using [?] function. This field is mandatory.

  5. Required Documents - Displays the ‘Required Documents’ information in disabled state if information is available for that request.

  6. Inform to HOD/Co-ordinator – Enter the practitioner name who has to be informed or search using the [?] lookup. This is a mandatory field.

  7. Forwarded HOD/Coordinator Date –  The current system date and time is displayed in read only text as the date when the Medical Report request is forwarded to the HOD/Co-ordinator.

  8. Requested Practitioner – Displays the name of the practitioner if this information is recorded against that request.

Click [Previous Delivered Medical Report] in the Medical Report Request screen to view the details of Medical Reports already delivered for the patient.
