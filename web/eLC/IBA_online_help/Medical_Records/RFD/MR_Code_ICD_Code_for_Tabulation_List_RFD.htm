Enter the following details:

 

  1. Tabulation List Type - Select the Tabulation List Type as either Disease or Procedure from the list.

  2. Tabulation List - Select the Tabulation List using [?] function. The Tabulation Search screen gets displayed to search and select the Tabulation List.

  3. Diag Code Scheme - The Diagnosis Coding Scheme of the selected Tabulation List is displayed.

  4. Search By - Select an option from the list to search the ICD Code by either Code or Description.

  5. Criteria - Select the Criteria from the list to search for the ICD Codes. The list displays 'Starts With', 'Ends With' and 'Contains'.

  6. Search Description - Enter the Description of search in alphanumeric format of 70 characters maximum length.

  7. Order by - Select the value from the list to denote the Order by which the ICD Codes are to be displayed., the diagnoses to be associated are listed. The listed values are Title and ICD Code. When 'Title' is selected, the ordering is based on the Title of the Diagnosis. If 'ICD Code' is selected, the ordering is based on the Codes of the Diagnoses.
