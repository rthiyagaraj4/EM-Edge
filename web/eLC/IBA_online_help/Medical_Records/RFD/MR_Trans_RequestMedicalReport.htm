Requesting Medical Report

From the MR Transactions, select the Medical Report function.

The Request tab of the Medical Report screen gets displayed.

  1. Select the required Patient ID to request a Medical Report for that patient and click [Search] to display encounters details of the patient or enter the search criteria and click [Search] to display a specific encounter to request for a Medical Report.

    1. Citizen ID - Enter the Citizen ID in alphanumeric format of maximum length as specified in the MP Parameter. The field title(s) is displayed based on the Alternate ID Type(s) set in MP Parameter. Based on the MP Parameter setup, system displays the Alternate ID Type description(s). The Alternate ID is displayed automatically on entering Patient ID of the registered patient. For a previously unregistered patient, this needs to be entered by the user.

    2. Other Alt Type - Select the Other Alt Type from the list of valid values. The listed values are Birth Certificate, Passport, and Mother's PMI, etc.

    3. Other Alt ID - The corresponding Other Alt ID is displayed when an Other Alt Type from the list is selected.

    Enter the values for the fields 'Previous HN', 'Government No', 'Driving License No' and 'Expatriate No' which are set in the MP Parameter under 'Alternate ID Checks' in the 'Master Patient Index' module.

    1. Patient ID - Enter the patient ID in alphanumeric format of 10 characters maximum length or select the patient ID by clicking [?] function. This field is mandatory.

    2. Encounter Period - Enter the encounter period in dd/mm/yyyy format or select the date using the calendar function.

    3. Encounter Type - Select Encounter Type as Day Care, Emergency, Inpatient, Outpatient and External.

    4. Encounter ID - Enter the Encounter ID in alphanumeric format of 12 characters maximum length.

    5. Order By - Select the order in which the records have to be displayed. The listed values are Encounter Date and Encounter Type.

  2. Select the check box against the required encounters to request for Medical Report.

  3. Click [Previous Delivered Medical Report] to view the details of the Medical Reports already delivered for the patient.

  4. Click [Medical Report] to invoke the ‘Medical Report Request’ screen and request for the Medical Report.

  5. Enter the details in the mandatory (*) fields. <Related Field Description>

  6. Click [Record] to save Medical Report Request and a Request ID is generated or click [Close] to exit from the ‘Medical Report Request’ screen without requesting for Medical Report.

  7. On recording, if internal reports are defined, the Internal Reports screen gets displayed. Click [OK] to generate Internal Reports or [Cancel] to exit from the screen.
