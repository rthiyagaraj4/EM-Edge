Factors for Terminology Code

From this function you can Create an association of Factors to a Terminology Code of the Terminology Set and Query for a desired Factosr for Terminology Code records.

The Factors for Terminology Code function is used to limit relevant factors defined to the Terminology Code of a particular Terminology Set. The factors are defined in the 'Factors' function.

Creating an association of Factors to Terminology Code

From MR Code Setup > MR Terminology Setup, select the Factors for Terminology Code function.

The Factors for Terminology Code screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the required Terminology Set ID from the list of valid values. The values are defined in the Terminology Set master. This field is mandatory.

  3. Select the required Terminology Code using [?] function. This field is mandatory.

  4. Click [Search] to display Factors as defined in the Factors function.

  5. Select the check box against the required Factors to associate them to the selected Terminology Code. Select the check box provided in the column header to associate all the displayed Factors to the Terminology Code.

  6. Click [Apply] to save the created Factors for Terminology Code records or [Reset] to clear the entered data.

  7. Click [Query] to search for existing Factors for Terminology Code records.
