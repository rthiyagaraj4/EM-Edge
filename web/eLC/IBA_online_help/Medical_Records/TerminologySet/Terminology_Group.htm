Terminology Group

From this function you can Create a new Terminology Group and Query for a desired Terminology Group record.

The Terminology Group function is used to define various Terminology Groups, where a number of Terminology Codes are associated to a particular Terminology Group based on the selected Terminology Set.

Creating a Terminology Group

From MR Code Setup > MR Terminology Setup, select the Terminology Group function.

The Terminology Group screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Term Code] to search for required Terminology Codes and the selected Terminology Set screen is displayed.

    1. Term Set – The Terminology Set selected is displayed.

    2. Level – Select the level of the Terminology Set as defined in the Terminology Set function.

    3. Search By – Select either ‘Description’ or ‘Code’ as the value.

    4. Term Description – Enter the description for the Terminology Code. This field is mandatory.

    5. Search Criteria – Select the required search criteria from the list. The listed values are ‘Starts With’, ‘Ends With’ and ‘Contains’.

    Click [Search] to display the search result.

    1. Term Code – The Terminology Code as defined in the Terminology Code function is displayed.

    2. Term Code Description – The description of the Terminology Code is displayed.

    3. Code Level – The level of the Terminology Code is displayed.

    4. Select – Select the check box against the required Terminology Code to associate to the Terminology Group.

    Click [OK] to display the selected Terminology Codes in the main screen  or click [Close] to exit from the screen without selecting Terminology Codes.

  4. Enter the serial number against the Terminology Codes in numeric format of three digits maximum length. This field is mandatory.

  5. Click [Apply] to save the Terminology Group created or [Reset] to clear the entered data.

  6. Click [Query] to search for existing Terminology Group records.
