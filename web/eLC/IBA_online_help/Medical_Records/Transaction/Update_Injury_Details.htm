Update Injury Details

From this function you can Update Injury details for an encounter.

From the function you can update injury details like Injury date, Treatment start date and treatment end date for unprocessed encounters.

You can edit the injury details record till the Claim is processed in Billing. Once it is processed, you will not able to modify the injury details for that encounter. This will be applicable for Credit and Insurance patients.

Updating Injury Details

From Medical Records > MR Transactions > select the Update Injury Details function.

The Update Injury Details screen gets displayed.

  1. Enter the query criteria and click [Search]  to select unprocessed encounters records.

    1. Patient ID – Enter the required Patient ID or search and select the patient ID using [?] function.

    2. Patient Class – Select the Patient Class from the drop-down list. The value ‘Select’ is displayed by default.

    3. Encounter ID – Enter the Encounter ID of the patient in the text box.

    4. Encounter Date [From – To] – Select Encounter Date range by clicking the Calendar icon.

    You need to select either the Patient ID or Encounter ID.  Otherwise an error message ‘Specify either Patient ID or Encounter ID’ gets displayed.

  2. Click Encounter ID hyperlink in the search results that appear. The ‘Update Injury Details’ dialog gets displayed.

  3. In the dialog, enter the Injury Date by clicking the Calendar icon. This is a mandatory field.

  4. Enter the Treatment Start Date for that injury by clicking Calendar icon. This is a mandatory field.

  5. Click [Record] to save the entered injury date and treatment start date.

  6. Once the treatment gets completed, search for the respective Encounter and enter the Treatment End Date in the Injury Details dialog and click [Record].

 

• Injury Date cannot be greater than Encounter Start Date.

• Treatment Start Date should be greater than or equal to Injury Date.

• Treatment Start date cannot be greater than current date.

• Treatment End date should be greater than or equal to Treatment Start Date.
