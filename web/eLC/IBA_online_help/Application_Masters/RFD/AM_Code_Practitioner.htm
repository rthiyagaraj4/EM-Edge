Enter the following details to create a Practitioner record:

 

  1. Type - Select a Type of Practitioner from the list of system defined values. Practitioner Type indicates the functional nature of the Practitioner. This is a mandatory field.

  2. ID - Enter the Practitioner ID in alphanumeric format of 15 characters maximum length. A Practitioner ID is unique and is used to identify a Practitioner. This is a mandatory field.

  3. External Practitioner – Select the check box to identify the practitioner as an External Practitioner.

  4. Full Name - Enter the Full Name of the Practitioner in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  5. Short Name - Enter the Short Name of the Practitioner in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  6. Long Name – Enter the long name of the practitioner in this field. You can enter up to 300 characters. This field is optional.

  7. Primary Specialty – Select a Specialty from the list. The values created through the 'Specialty' function with status  as enabled are listed. This is a mandatory field.

Note: If you have assigned and saved a multi specialty as the primary specialty, you cannot modify it.

  1. Enabled - Select the check box to enable the record in the transaction functions. Clear the check box to disable the record. The check box is selected, by default.

 

Identity

 

  1. Gender - Select the Gender of the Practitioner from the list. The list displays Male, Female and Unknown. This is a mandatory field.

  2. Age - Enter the Age of the Practitioner in years/months/days, in the respective text box Y, M and D. Enter the Age, if date of birth is not known and the system calculates the date of birth and displays it in the ‘Birth Date’ field.

  3. Birth Date – Click the Calendar icon to enter the date of birth, if the age is not entered. On entering the Birth Date, system calculates the patient’s age and displays the age of the Practitioner in the Age field. This is a mandatory field.

  4. Birth Place - Enter the Place of Birth of the Practitioner in alphanumeric format of 30 characters maximum length.

  5. Marital Status - Select the Marital Status of the Practitioner from the list of values. All Marital Statuses as defined in the Master Patient Index module is displayed in the list box.

  6. Alias Name - Enter the Alias Name (a name that is assumed temporarily) of the Practitioner in alphanumeric format of 30 characters maximum length.

  7. Citizenship - Select the Citizenship of the Practitioner from the list of values.  All Citizenship records as defined in the Master Patient Index module is displayed.

  8. Citizen ID – Enter the Citizen ID in numeric format of 13 characters maximum length.

The above field names are displayed based on the legends for Alternate ID defined in the MP Parameter.

  1. Other Alt No – Select the required other alternate no value from the drop-down list. The drop-down list value is derived from the ‘Alternate ID Type’ defined in the MP Code Setup which is in enabled mode. By default, the value ‘Select’ is displayed.  The text box next to the ‘Other Alt No’ field is enabled only when the ‘Other Alt No’ field is enabled. You can enter the legend value up to maximum of 20 characters and all special characters are allowed. The captured legend with value is displayed wherever the ‘Other Staff Details’ dialog, and the ‘Practitioner Details’ dialog are invoked in the EM application. When no value is captured, only the legend name ‘Other Alternate No’ is displayed without value.

  2. Credential (1-3) - Enter the Credentials of the Practitioner in alphanumeric format of 20 characters maximum length.

  3. Position - Select the Position of the Practitioner from the list of values. All Positions as defined in the Position function are displayed in the list.

  4. Professional Qualification - Enter the Educational Qualification of the Practitioner in alphanumeric format of 15 characters maximum length.

  5. Empl Status - Select the Status of Employment of the Practitioner from the list of valid values. The listed pre-defined values are Associate, Full Time, Part Time and Visiting Specialist. This is a mandatory field.

  6. APC No. – Enter the Annual Practising Certificate number in alphanumeric format of 20 characters maximum length.

  7. All Facilities - Select the check box to indicate that the Practitioner works for all facilities of the Enterprise. If the check box is selected, the system identifies that the Practitioner is associated to all the Facilities. Otherwise, applicable Facilities of the Enterprise are to be associated to the Practitioner using the function ‘Practitioner for Facility’. This check box is enabled for selection, if the 'External Practitioner' check box is not selected.

  8. Employee  ID – Displays the employee ID of the practitioner, as input into the system via backend. (This field cannot be edited.)

  9. Note: This field is enterable in the Query page, where you can enter the employee ID and search for results. You can enter up to 30 characters, and include the special characters “–“ (hyphen) and “_”(underscore).

  10. Rep Facility – Select the Reporting Facility from the list of valid values. This field lists all the facilities associated to the logged in user.  This check box is enabled for selection, if the 'External Practitioner' check box is not selected.

 

Contact No./Addr

 

  1. Res Tel No - Enter the Residential Telephone Number of the Practitioner in numeric format of 18 characters maximum length.

  2. Office Phone No - Enter the Official Telephone Number of the Practitioner in numeric format of 18 characters maximum length.

  3. Extension - Enter the Extension of Official Telephone Number of the Practitioner in alphanumeric format of six characters maximum length.

  4. Fax No - Enter the Fax number of the Practitioner in numeric format of 18 characters maximum length.

  5. Mobile No - Enter the Mobile Number of the Practitioner in numeric format of 18 characters maximum length.

  6. Pager No - Enter the Pager Number of the Practitioner in numeric format of 18 characters maximum length.

  7. Email ID - Enter the Email ID of the Practitioner in alphanumeric format of 50 characters maximum length.

  8. Pref Mode - Select the Preferred Mode of contact of the Practitioner from the list of valid values.

  9. Address (Residence/Office/Mail) - Enter the Address in alphanumeric format of 30 characters maximum length in each of the four text boxes provided. In the Residence Address fields, enter the address where the Practitioner is presently residing. In the Mailing Address fields, enter the address for sending the correspondences to the Practitioner. In the Office address fields, enter the Practitioner’s office address.

  10. Postal Code (Residence/Office/Mail) – Enter a valid Postal Code for Residence, Office and Mailing Address respectively in alphanumeric format of eight characters maximum length. The Postal Codes gets listed on clicking [?] function, if it is set up through the ‘Postal Code’ function in the Master Patient Index module.

  11. Country (Residence/Office/Mail) – Click the [?] function to select a Country from the list of values. This field displays the values defined in Master Patient Index module.

 

Qualification & Training

Qualification Details

  1. Degree – Enter the Educational Qualification of the Practitioner in alphanumeric format of 60 characters maximum length.

  2. Year of Passing - Enter the year when the Practitioner has completed the degree in numeric format of four digits maximum length.

  3. University - Enter the name of the University in alphanumeric format of 30 characters maximum length.

 

Training details

  1. Details of Training – Enter the Details of Training attended by the Practitioner in alphanumeric format of 100 characters maximum length.

  2. Conducted By – Enter the details of training Conducted By in alphanumeric format of 30 characters maximum length.
