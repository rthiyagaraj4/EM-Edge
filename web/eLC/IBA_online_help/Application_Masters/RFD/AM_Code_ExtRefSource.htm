Enter the following details to create an External Referral Source record:

 

  1. Code - Enter the code for the External Referral Source in alphanumeric format of six characters maximum length. An External Referral Source Code is used to identify the External Referral Source. The External Referral Source Code has to be unique and cannot be changed after creation. This is a mandatory field.

  2. Long Description - Enter the Long Description of the External Referral Source in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the Short Description of the External Referral Source in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  4. Long Name – Enter the Long Name of the External or Referral Source in alphanumeric format of 100 characters maximum length.

  5. Healthcare Setting Type - Select the Healthcare Setting Type from the list to which External Referral Source belongs. The options available are as defined using the function Healthcare Setting Type. This is a mandatory field.

  6. Organization Type – Select the Organization Type from the list. The listed values are ‘Government’ and ‘Private’. This field is mandatory.

  7. Enabled - Select the check box to enable the record. Clear the check box to disable the record. The check box is selected, by default

 

Characteristics Tab

Use as External/Referral source At:

  1. Registration/Visit/Admission - The individual check boxes control if this External Referral Source Code can be used as a Referral Source at Registration, Visit and Admission respectively.

 

Use as External/Referral Destination At:

  1. Visit Conclusion/Discharge - The individual check boxes control if this External Referral Source Code can be used as a Referral Destination during Visit Conclusion and Discharge respectively.

 

Use as External/Referral Source for direct ordering

  1. Applicability – Select a pre-defined value from the list. The values listed are ‘Referral’, ‘External’, ‘Both’ and ‘Not Applicable’. Select a value to indicate the applicability of the External/Referral Source to perform direct ordering for Referral/External/Both Referral and External patients. Select ‘Not Applicable’ if the External/Referral Source is not applicable for Direct Ordering. This field is mandatory.

  2. Associated Enterprise Facility for Referral – Select the Facility from the list that is associated to the Enterprise for Referral. The values listed are defined in System Manager Module. If ‘Applicability’ is selected as, Referral or Both, this field is enabled. If ‘Applicability’ is selected as External or Not Applicable, this field is disabled.

  3. Patient Prefix for identifying Source – Enter the patient prefix for identifying the External/Referral Source in alphanumeric format of two characters maximum length. If ‘Applicability’ is selected as Not Applicable, the field is disabled.

  4. Series for Ext Patient Numbering – Select the series for External Patient Numbering from the list of valid values. The values are listed from the Master Patient Index Module. ‘Patient Series’ with ‘Series Type’ as ‘External source series’ are displayed. If ‘Applicability’ is selected as Not Applicable, this field is disabled.

 

Allow Direct Orders for

  1. Laboratory/Radiology/Pharmacy – Select the check box if the External/Referral Source is allowed direct ordering for Laboratory, Radiology or Pharmacy. If ‘Applicability’ is selected as Referral/External/Both, this field is enabled and on selecting ‘Applicability’ as Not Applicable, this field is disabled.

 

Use as External/Referral Source for Tissue Collection

  1. Applicability – Select a pre-defined value from the list. The values listed are ‘Tissue Collection’ and ‘Not Applicable’. Select a value to indicate the Applicability of the External/Referral Source for tissue collection. Select ‘Not Applicable’ if the External/Referral Source is not applicable for tissue collection.

  2. Donor Prefix for identifying Source – Enter the Donor Prefix for identifying the External/Referral Source in alphanumeric format of two characters maximum length. If ‘Applicability’ is selected as ‘Tissue Collection’, the field is enabled.

  3. Series for Donor Numbering – Select the series for Donor Numbering from the list. The values are listed from the Master Patient Index Module. The ‘Patient Series’ defined with ‘Series Type’ as ‘External Patient/Donor Series’ appears in the list box. If ‘Applicability’ is selected as ‘Not Applicable’, this field is disabled.

 

Contact Details Tab

Address Details

  1. Address [Line 1 to 4] - Enter the Address of the External Referral Source in alphanumeric format of 30 characters maximum length in each of the four boxes.

  2. Town – Enter the City/Town in alphanumerical of 30 characters maximum length. The entered city/town are listed under the selected area. If the selected area has more cities/towns attached to it, select the required city/town using the [?] function. The field legend is set in ‘System Manager’ Module.

  3. Area – Enter the Area in alphanumerical format of 30 characters maximum length or select it using the [?] function. The field legend is set in ‘System Manager’ module.

  4. Region – Enter the state in alphanumerical format of 30 characters maximum length or select it using the [?] function. The area must correspond to the entered state. The field legend is set in ‘System Manager’ Module.

  5. Pin Code - Enter the Postal code of the External Referral Source or click [?] to select the appropriate Postal Code. The values are listed from ‘Postal Codes’ function in Patient Identity Management module.

  6. Country - Select the Country of the External Referral Source from the list of valid values. The values are listed from ‘Country’ function in Master Patient Index module.

  7. Tel No - Enter the Telephone number of the External Referral Source in alphanumeric format of 18 characters maximum length.

  8. Fax No - Enter the Fax number of the External Referral Source in alphanumeric format of 18 characters maximum length.

  9. Email - Enter the Email ID of the External Referral Source in alphanumeric format of 50 characters maximum length.

 

Contact Person Details

  1. Name – Enter the Name of the contact person in External Referral Source in alphanumeric format of 40 characters maximum length.

  2. Title – Enter the Title for the name in alphanumeric format of 40 characters maximum length.
