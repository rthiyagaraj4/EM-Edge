Enter the following details to create a discrete measure panel:

 

  1. Result Type – Select a value for Result Type from the list of valid values. The values are Chart Section, Patient Care Panel and Vital Signs Battery. The Panel created is available for the user according to the Result Type selected. This is a mandatory field.

  2. ID – Enter the ID in alphanumeric format of ten characters maximum length. This is a mandatory field.

  3. Long Description - Enter the long description of the Discrete Measure Panel in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  4. Short Description - Enter the short description of the Discrete Measure Panel in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  5. Service – Select a Service from the list of valid values. The Services defined through the Service function with the Medical Service check box as enabled are listed. This is a mandatory field.

  6. Enabled – Select the check box to enable the record in the transaction functions. Clear the check box to disable the record. The check box is selected, by default.

 

Enter the following details to Add New Panel Element:

 

  1. Discrete ID - Enter the Discrete ID or click [?] function and select the Discrete ID. This field displays the Discrete Measures of all types, which matches the Service selected for the panel.

  2. Description – On selecting the Discrete ID, the field displays the Description for the Discrete Measure.

  3. Mandatory – Select the check box if the field is mandatory or clear the check box. If this check box is selected, you have to compulsorily record the data in the Discrete Measure. Those Discrete Measures, which are mandatory, are accompanied by an asterisk sign.

 

Click [Add] to Add New Panel Element, [Clear] to clear the entered data or [Delete] to delete the record.

On adding a new Panel Element, the details of the Discrete Measure are displayed under Associated Panel Elements.

Click ‘Serial No’ hyperlink to interchange the order of the Panel Elements. The selected record highlights with a unique color code.

Click ‘ID’ hyperlink to modify the Panel Elements. The selected component is displayed under the Modify Existing Panel Element section. Select or clear the Mandatory  check box. Click [Modify] to save the changes, [Clear] to reset the modified data or [Delete] to delete the record.
