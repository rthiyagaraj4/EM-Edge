Enter the following details to create a new Transport Mode record:

  1. Transport Mode - Enter the Transport Mode code in alphanumeric format of 10 characters maximum length. A Transport Mode code is used to identify the Transport Mode. The Transport Mode code has to be unique and cannot be changed after creation. This field is mandatory.

  2. Long Description - Enter the long description of the Transport Mode in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description - Enter the short description of the Transport Mode in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Patient Related - Select the check box to indicate that the mode of transport is utilized for transporting patients. The check box is cleared, by default.

  5. File Related – Select the check box to indicate that the mode of transport is utilized for transporting patient files. The check box is cleared, by default.

  6. Specimen Related - Select the check box to indicate that the mode of transport is utilized for transporting patients’ specimen. The check box is cleared, by default.

  7. By Person – Select the check box to indicate that the mode of transport is manual. The check box is cleared, by default.

 

Medical Report

 

  1. Application Mode – Select the check box to indicate that transport mode is applicable for Application Mode field in ‘Status of medical Request’ screen. The check box is cleared, by default.

  2. Collection Mode – Select the check box to indicate transport mode is applicable for Collection Mode field in ‘Status of medical Request’ screen. The check box is cleared, by default.

  3. Notification Mode - Select the check box to indicate transport mode is applicable for Notification Mode field in ‘Status of medical Request’ screen. The check box is cleared, by default.

If you select any one of the check boxes Application Mode, Collection Mode or Notification Mode, the remaining check boxes get disabled. Similarly, if any one of the check boxes are selected then the check boxes under Medical Report get disabled.

  1. Enabled - Select the check box to enable the Transport Mode record. The check box is selected, by default. Clear the check box to disable the Transport Mode record.
