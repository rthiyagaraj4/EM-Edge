Enter the following details to create a Position record:

 

  1. Position Code – Enter the Position code in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Position Description – Enter the description of the Position in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Role Type – Select a Role Type from the list. The listed values are Practitioner and Other Staff. This is a mandatory field.

  4. Staff Type – Select a Staff Type from the list. The values are displayed based on the Role Type selected. If the Role Type is selected as Practitioner, all the Practitioner Types are displayed. If the Role Type is selected as Other Staff, Other Staff Types defined through the Other Staff Type function are displayed.

  5. Grade - Select a Grade for the position from the list. The listed values are defined through the 'Grade' function. You can associate a position to each grade. You cannot associate a position to multiple grades but a grade can be associated to multiple positions.

  6. Explanatory Text – Enter the explanation in alphanumeric format in the free text area. The Explanatory Text gives a Description of the Position for a Role Type.

  7. Enabled - Select the check box to enable the record in the transaction functions. Clear the check box to disable the record. The check box is selected, by default.
