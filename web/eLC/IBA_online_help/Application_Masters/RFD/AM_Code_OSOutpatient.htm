Outpatient

The tabs displayed are Appointment Details, Visit Detail and Others.

  1. Appointment Details tab is displayed, by default. Select the appropriate check boxes against the required functions to perform those functions of the operator station of type ‘Outpatient’.

    1. Book Appointment – Select the check box to enable the Operator Station to Book Appointments. On selecting the ‘Book Appointment’ check box, the check boxes of the legends ‘Book appointments without Patient ID’, ‘Overbook Appointment’, ‘Allow forced-in Appointment’ and ‘Allow Global Booking’ are enabled. Clear the check box to restrict the Operator Station from booking appointment.

    2. Book Appointment without Patient ID - Select the check box to enable the Operator Station to Book Appointments without Patient ID. Clear the check box to restrict the Operator Station from booking appointment without Patient ID. This check box is enabled only if the 'Book Appointment' check box is selected.

    3. Overbook Appointment - Select the check box to enable the Operator Station to Overbook Appointments. Clear the check box to restrict the Operator Station from overbooking appointments. This check box is enabled only if the 'Book Appointment' check box is selected.

    4. Allow Forced-in Appointment - Select the check box to enable the Operator Station to Allow Forced-in appointments. Clear the check box to restrict the Operator Station from allowing forced-in appointments. This check box is enabled only if the 'Book Appointment' check box is selected.

    5. Allow Global Booking - The check box gets enabled on selecting the 'Book Appointment' check box. Select 'Allow Global Booking' check box to enable the Operator Station to allow Global Booking. Clear the check box to restrict the Operator Station from allowing global booking. This check box is enabled only if the 'Book Appointment' check box is selected.

    6. Transfer Appointment – Select the check box to enable the Operator Station to Transfer Appointments. Clear the check box to restrict the Operator Station from transferring the appointments.

    7. Cancel Appointment – Select the check box to enable the Operator Station to Cancel Appointments. Clear the check box to restrict the Operator Station from canceling the appointments.

    8. Create Wait List – Select the check box to enable the Operator Station to create Wait List in Appointment Schedule module. Clear the check box to restrict the Operator Station from creating Wait List in Appointment Schedule module.

    9. Modify Wait List – Select the Modify Wait List to enable the Operator Station to Modify the Wait List in Appointment Schedule module. Clear the check box to restrict the Operator Station from modifying Wait List in Appointment Schedule module.

    10. Cancel Wait List – Select the check box to enable the Operator Station to cancel the Wait List in Appointment Schedule module. Clear the check box to restrict the Operator Station from canceling Wait List in Appointment Schedule module. The system clears the check box, by default.

    11. Override Allocation Rule for Book Appointment – Select the check box to enable the Operator Station to override the allocation rule while booking an appointment in Appointment Scheduling module. Clear the check box to restrict the Operator Station to override the allocation rule while booking an appointment in Appointment Scheduling module. The check box is cleared, by default.

  2. Click 'Visit Detail' Tab and select the appropriate check boxes against the required functions to perform those functions of the operator station of type ‘Outpatient’.

    1. Register Visit - Select the check box to enable the Operator Station to Register Visit. On selecting the ‘Register Visit’ check box, the check boxes of the legends ‘Register Walk-in’, ‘Register Referral’ and ‘Register Patient from Register Visit’ are enabled for selection. Clear the check box to restrict the Operator Station from registering visits.

    2. Register Walk-in - Select the check box to enable the Operator Station to Register Walk-in patients. Clear the check box to restrict the Operator Station from registering walk-in patients. This check box is enabled only if the 'Register Visit' check box is selected.

    3. Override Walk-in - Select the check box to enable the Operator Station to Override Walk-in patients. Clear the check box to restrict the Operator Station from overriding walk-in patients. This check box is enabled only if the 'Register Visit' and 'Register Walk-In' check boxes are selected.

    4. Register Referral - Select the check box to enable the Operator Station to Register Referral patients. Clear the check box to restrict the Operator Station from registering referral patients. This check box is enabled only if the 'Register Visit' check box is selected.

    5. Register Patient from Register Visit - Select the check box to enable the Operator Station to Register Patients from the Register Visit function in the OP module. Clear the check box to restrict the Operator Station from registering patients while registering a visit for the patient. This check box is enabled only if the 'Register Visit' check box is selected.

    6. Manage Queue – Select the check box to enable the Operator Station to Manage Patient Queue in the Outpatient Management (OP) module. Clear the check box to restrict the Operator Station from managing patient queue.

    7. Cancel Visit – Select the check box to enable the Operator Station to Cancel Visits in Outpatient Management module. Clear the check box to restrict the Operator Station from cancelling visits.

    8. Revise Visit – Select the check box to enable the Operator Station to Revise Visits registered for patients. Clear the check box to restrict the Operator Station from revising visits.

    9. Check Out Patient – Select the check box to enable the Operator Station to Check-out patients after consultation. Clear the check box to restrict the Operator Station from checking out patients.

    10. Reserve Bed/Bay - Select the check box to enable the Operator Station to Reserve Bed/Bay. Clear the check box to restrict the Operator Station from reserving beds/bays.

    11. Release Bed/Bay - Select the check box to enable the Operator Station to Release Bed/Bay that is reserved. Clear the check box to restrict the Operator Station from releasing beds/bays.

  3. Click 'Others' Tab and select the appropriate check box against the required function to perform the function of the operator station of type ‘Outpatient’

    1. Diagnosis Procedure Recoding - Select the check box to enable Diagnosis Procedure Recoding. Clear the check box to restrict Diagnosis Procedure Recoding.
