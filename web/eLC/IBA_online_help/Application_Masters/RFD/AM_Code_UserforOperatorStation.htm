Enter the following details to associate a User to an Operator Station:

 

  1. Facility – Select a Facility from the list.  The logged in facility is displayed, by default. This is a mandatory field.

  2. User - Select the User from the [?] function. A lookup screen gets displayed. Enter the required search criteria and click [Search]. This field displays the records matching the search criteria. Select the User by clicking the 'Code' that is displayed as a hyperlink.  The User must be associated to the Facility through ‘Facility for User’ function in System Manager module. This is a mandatory field.

  3. Operator Station - Select the Operator Station to grant access to the selected User. This field displays all the Operator Stations created through ‘Operator Station’ function. This is a mandatory field.

  4. Effective From - Enter the 'Effective From' Date in (dd/mm/yyyy) format. ‘From date’ corresponds to the date from when the user has access to the Operator Station. The current system date is displayed, by default. This is a mandatory field.

  5. Effective To - Enter the 'Effective To' Date in (dd/mm/yyyy) format. The ‘To date’ corresponds to the date until when the user has access to the Operator Station.
