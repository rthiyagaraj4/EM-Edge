Frequency Schedule

From this function you can Create a Frequency Schedule and Query for a required Frequency Schedule record.

Frequency Schedule function allows you to perform scheduling for the frequencies created in the Frequency Function. Scheduling is done only for those frequencies that are of ‘Frequency’ nature in terms of time. The ‘Scheduled’ check box is selected while defining the frequency in the ‘Frequency’ function. Schedule is defined only in case the duration type is in hours, days or weeks and is not possible for minutes and months. This is an optional setup. This is done at the level of facility.

Creating a Frequency Schedule

From the Frequency Setup > Frequency Setup, select the Frequency Schedule function.

The Frequency Schedule screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the required Frequency from the list. All the frequencies defined of nature ‘Frequency’ with the ‘Schedule’ check box selected is displayed in the list. This field is mandatory.

  3. Select the required facility from the list. The list box displays facility records as created in the Facility function of the System Manager module. This field is mandatory.

  4. On selecting the frequency, the repeat value and its corresponding interval value gets displayed. The display of number of fields in ‘Time-Rounded To’ is based on the repeats value. Enter the valid time in hh:mm format in the respective fields for time.

  5. If the ‘Repeat(s) per’ is set to weeks while creating the Frequency in ‘Frequency’ function, the days of the week are displayed. Select the required day of the week by selecting the check box.

  6. Click [Apply] to save a Frequency Schedule created or [Reset] to clear the entered data.

  7. Click [Query] to search for an existing Frequency Schedule record.
