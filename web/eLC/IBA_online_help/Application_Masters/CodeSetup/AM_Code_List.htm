AM Code List

From this function you can Generate a report on the Master Code lists of Application Masters Module.

AM Code List function is used to generate the Master Code lists of Application Masters module for the enterprise or facility. You can specify the query conditions under the report criteria of AM Code List to generate the reports. The reports are generated as per the query conditions set and order defined.

Generating a Report

From AM Reports, select the AM Code List function.

The AM Code List screen gets displayed.

  1. Select the required Code from the list of AM Master Codes to generate a report.

  2. Select the report criteria.

  3. Click [Run] to generate the report.

  4. Click [OK] to process the report.
