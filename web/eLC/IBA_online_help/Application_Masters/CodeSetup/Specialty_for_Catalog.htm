Specialty for Catalog

From this function you can Create an association of Specialties to an Order Catalog.

This functionality is used to associate the Specialty to the Catalog defined in the Order Entry & Result Reporting module.

Creating an association of a Specialty to Order Catalogs

From the AM Other Setup > Other Master, select the Specialty for Catalog function.

The Specialty for Catalog screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Specialty from the list of valid values. This field displays the Specialties created using the ‘Specialty’ function in Practitioner setup with the enabled check box selected. This field is mandatory.

  3. Select the Order Category from the list of values. The valid list of values are ‘Consumbales’, ‘Dep Diagnosis’, ‘Dietery’, ‘Laboratory’, ‘Miscellaneous’ ‘Monitoring’, ‘Operation Theat’, ‘Oral Health’, ‘Order Set’, ‘Patient Care’, ‘Radiology’ and ‘Trmt/Pprocedures’.

  4. Click [Search] to display the Order Catalogs of the selected Order Category or [Cancel] to clear the entered data.

  5. Select the check box under Include against the required Order Catalogs to associate them to the selected Specialty.

  6. Click [Apply] to save a Specialty for Catalog record created or [Reset] to clear the entered data.

  7. Click [Query] to search for an existing Specialty for Catalog record.
