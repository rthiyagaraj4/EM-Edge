Specialty for External Referral Source

From this function you can Create an association of External/Referral Source to Specialties and Query for a required Specialty for External Referral Source record.

On setting up the various external referral sources, specialties of these sources are to be recorded. This function is used to record specialties of each external referral source.

Creating an association of External Referral Source to Specialties

From the Referral Setup, select the Specialty for External Referral Source function.

The Specialty for External Referral Source screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the External/Referral Source from the list. The listed values are defined in the function ‘External Referral Source’. This field is mandatory

  3. Select the Search By field as Specialty Code or Specialty Description and enter the required text in the text box provided to search for specialties based on the Code or Description.

  4. Click [Search] to search for the specialties.

  5. The Specialties as defined in the ‘Specialty’ function is displayed.

  6. Select the check box against the required Specialty to associate the selected Specialty to the External/Referral Source.

  7. Click [Apply] to save the association of Specialties to External/Referral Source or [Reset] to clear the entered data.

  8. Click [Query] to search for an existing Specialty for External Referral Source record.
