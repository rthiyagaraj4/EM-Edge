Translation Manager

From this function you can Record and Maintain the local language record for each of the master functions that has been created in English language in the corresponding function.

The Translation Manager function helps to record legends in local language for the master functions in a module. You can enter or update the description in local language for each record that has been created in any master function.

Translating Records

From the System Manager, select the Translation Manager function.

The Translation Manager screen gets displayed.

  1. Select the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to list and record the entries in local language or click [Clear] to clear the displayed search results.

  3. Click [Apply] to record the details or [Reset] to reset the details.

 

Through this function, you can translate legends from English to local language for functions in two tier only. The legends for functions in Java version are converted to local language through the resource bundle.
