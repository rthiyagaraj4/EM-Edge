Day of Week

From this function you can Define the Holidays and Working Days of the week at the enterprise level.

Day of Week function is used to specify if a particular day of a week is a working day or a holiday. A day in a week or the days across the month can be set as either a working day or a holiday. The applicability is set across various weeks of the month. This is a generic default setup used across the various modules in Enterprise Management.

Defining Day of Week

From the SM Admin Setup > Enterprise Admin, select the Day of Week function.

The Day of Week screen gets displayed.

  1. Select the values from the list to define a day as a Working day or a Holiday. <Related Field Description>

  2. Click [Apply] to save a setting or [Reset] to clear the entered data.

 

If Day of the week is Sunday and week applicability is selected as ‘All’, all the Sundays are set as holidays, by default.
