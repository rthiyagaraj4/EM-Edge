Printer

From this function you can Create a new Printer by associating a Printer Type and Report Service. You can Query for a desired Printer.

The application server set up by default, functions as the print server. A print server is a server in which the reports are executed and initiated for printing. It is possible to set-up additional print servers also apart from the default print server (which is application server).  

All types of printers like laser, inkjet, label, etc. are supported and the connectivity modes like local, network, etc. are possible. However, this support is available provided the corresponding drivers are available for the server in which these printers are configured.

System Administrator can adopt the following procedure to set-up a printer:

On the print server:

• Printers without network card can be configured on print server itself.

• Printers without network card can be configured on clients connected on the network. Such printers are to be shared and entries are to be created on the print server.

• Irrespective of the printer connectivity, the print drivers are to be loaded on the print server for all the printers.

• If the print server is MS Windows NT based, different models of the printers are to be configured on different NT users. This is due to the bug in the current version of Oracle Reports.

In Enterprise Management:

• All the printers of the print server, which are used to print the reports of Enterprise Management, are to be defined in Enterprise Management.

An entry of a printer typically identifies the printer with an ID, the queue name represents the physical path of the printer, which is used to direct the outputs while printing and the report service to which all the printing jobs of the printer are to be queued.

Creating a Printer

From the SM Admin Setup > Print Routing, select the Printer function.

The Printer screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a record or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing Printer.
