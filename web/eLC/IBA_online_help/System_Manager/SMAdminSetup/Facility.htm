Facility

From this function you can Create a new Facility at the enterprise level, Associate it to a Healthcare Setting Type and Query for a desired Facility.

A healthcare organization is termed as a facility that operates as part of the site where Enterprise Management is installed. A set of facilities defined as part of a site is generally referred to as an Enterprise. This function helps you to set up various facilities required at the level of site.

Creating a Facility

From the SM Admin Setup > Enterprise Admin, select the Facility function.

The facility screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a record or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing Facility.

 

When there are multiple facilities functioning under a site, it is not necessary that they must be physically located together; they can be located in different locations in a distributed manner but connected to the central database at Enterprise Management site.
