Access Rights for Menu - Functions

From this function you can Define and Maintain the Access Rights for Menu Functions by selecting the relevant check boxes.

Access Rights for Menu-Functions is used to define the access rights for a menu function. You can modify the access rights for  a selected menu. The access rights displayed for the first time is based on the default access rights defined for a function in a module.

Defining Access Rights for Menu-Functions

From the SM Admin Setup > Application Security, select the Access Rights for Menu-Functions function.

The Access Rights for Menu-Functions screen gets displayed.

  1. Select a menu from the list. This is a mandatory field. <Related Field Description>

  2. On selecting the menu from the list, click [Search] to display the functions that satisfy the search criteria or click [Clear] to clear the selection. <Related Field Description>

  3. Update the default access rights for menu functions and click [Apply] to save the record.

  4. Click [Reset] to clear the entered data.
