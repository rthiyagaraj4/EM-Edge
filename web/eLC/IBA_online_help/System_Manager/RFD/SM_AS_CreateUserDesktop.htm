Enter the following details to create a User Desktop:

 

  1. Desktop Code – Enter the code of the Desktop in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Desktop Name – Enter the Name of the Desktop in alphanumeric format of 50 characters maximum length. This is a mandatory field.

  3. Enabled - Select the check box to enable the Desktop Summary information for the user in transaction functions.

  4. User Type – Select Clinician or Non Clinician to assign the Desktop Summary information to a Clinician or Non Clinician. Currently the system does not support assigning or defining Summary information for Non Clinicians.

  5. Show Summary Menu – Select 50% or 100% to view the Summary Information in the selected percentage of the screen.

  6. Content Type – Content Type is displayed as Function and disabled.

  7. Module – Clinician Access is displayed and disabled, by default on selecting the User Type as Clinician. Select the required module from the list if the User Type is selected as Non Clinician. This is a mandatory field.

  8. Function – Select the required function of the Clinician Access module if User Type is selected as Clinician. Select the required Function from the list based on the Module selected. This is a mandatory field.

 

Summary Menu Customisation

  1. Default From – Select 'Clinical Desktop – Default' to display the default menu under ‘Selected Items’.

  2. Summary Header – The components for creating the Summary Header are displayed. Select the required component and click the white space under ‘Selected Items’ to display it in the space.

  3. Summary Links – The components for creating the Summary Links are displayed. Select the required component and click the white space in ‘Selected Items’ to display it under the selected Summary Header.

  4. Selected Items – The Summary Headers and Summary Links selected are displayed in the white space. The default menu is also displayed in the area provided on selecting.  

  5. Use the buttons to shift components or delete components from the menu.
