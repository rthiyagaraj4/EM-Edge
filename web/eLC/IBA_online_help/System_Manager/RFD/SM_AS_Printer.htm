Enter the following details to create a Printer record:

 

  1. ID - Enter the printer ID in alphanumeric format of 20 characters maximum length. The printer ID is maintained in upper case. The printer ID is an internal identifier of the printer in Enterprise Management and does not reflect in any way the actual name of the configuration entry of the printer on the print server. The printer ID has to be unique and cannot be changed. This is a mandatory field.

  2. Printer Name - Enter the Name of the Printer in alphanumeric format of 60 characters maximum length. The Printer Name is internal to Enterprise Management and does not reflect in any way the actual name of the configuration entry of the printer on the print server. This is a mandatory field.

  3. Queue Name - Enter the complete network path of the printer through which it is reached as the queue name in alphanumeric format of 60 characters maximum length. The way the network path is to be specified in this field depends on the operating system, where the printer is configured. For example, in Windows based servers, the printer ‘HP6L’, which has been connected to one of the clients (PC) of the registration desk is identified by \\REGDESK03\HP6L, where REGDESK03 is the identification of the client and HP6L is the identification of the printer. This is a mandatory field.

  4. Printer Type - Select the Printer Type from the list. This is a mandatory field. Printer type can be selected from the following standard list:

  5. Laser Printer (S)

  6. Dot Matrix Printer (DMP) (D)

  7. Ink Jet Printer (I)

  8. Line Printer (L)

  9. Label Printer (B)

  1. Report Service – Select a value from the list to associate a valid Report Service to the printer. The print jobs of the printer queued at the selected printer. The ‘Report service’ with 'Enabled' check box as selected during creation are displayed. This is a mandatory field.

  2. Enabled - Select the check box to enable the record in the transaction functions.  The check box is selected, by default.
