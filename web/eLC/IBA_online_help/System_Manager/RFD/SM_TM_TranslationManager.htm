Enter the following details for Translation Manager:

 

  1. Language - Select a local Language from the list. The listed values are generated through scripts during Enterprise Management installation. This is a mandatory field.

  2. Display - Select a value from the list to display all the records, translated records or the non-translated records. This is a mandatory field.

  3. Module - Select a Module from the list. A list of the modules that are installed gets displayed. This is a mandatory field.

  4. Master Name - Select a Master Name from the list. The functions that are listed correspond to the Module selected. This is a mandatory field.

• On selecting the ‘Display’ as ‘Translated’, the master records that are translated to Local Language are displayed. This helps you to edit the Translated Master records.

• On Selecting the ‘Display’ as ‘Non-Translated’, the master records that are not translated to Local Language are displayed. This helps you to record the local language for the corresponding Master records.

• On selecting ‘All’, the master records that are translated and not translated to Local Language are displayed. You can edit the details for translated Master records and record the local language for the non-translated Master records.

• If the language selected is Arabic, you can enter the text in the free text from right to left. On entering the text in the free text area, the translated data will be displayed from right to left.   

• If Master Name is selected as ‘Facility Param’, the following fields are displayed:

  • Facility ID – The Facility ID is displayed.

  • Facility Name – Enter the Facility Name in the language to be translated upto 35 characters maximum length.

 

  1. Search Text – Select the required search criteria from the list. The listed values are ‘Starts With’, ‘Ends With’ and ‘Contains’. Enter a text to query the desired records.
