Enter the following details to create a Facility record:

 

  1. ID – Enter the Facility ID in alphanumeric format of two characters maximum length. A Facility ID is used uniquely to identify the facility. The Facility ID has to be unique and cannot be changed after creation. This is a mandatory field.

  2. GSTIN – Enter the GSTIN value in alpha numeric format with maximum 20 characters in length. Special characters are not allowed. The value must be greater than zero. This field is mandatory. This field is applicable only to customers in India.

  3. Name – Enter the Name of the Facility in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  4. Accounting Entity ID - Select the Accounting Entity from the list. The listed accounting entities are pre-defined. This is a mandatory field.

  5. Category - Select a Facility Category from the list. Facility Categories defined in 'Facility Category' function are listed. Only one Facility Category can be associated to a facility.

  6. Healthcare Setting Type – Select the Healthcare Setting Type of the facility from the list. All the Healthcare Setting types defined with 'Enabled' check box as selected gets displayed in the list of values. This is a mandatory field.

  7. Local Language – Select the Local Language applicable for the facility being set up from the list. The list of languages applicable is provided as a part of the script.

  8. VAT Number (VAT No) – Enter the VAT number in alphanumeric format of maximum 50 characters length. It is a one-time setup. You will not be allowed to edit or modify the VAT No. details in Query mode. This field is applicable only to users in MMS.

 

Address Details

  1. Address Lines 1,2,3,4 – Enter the Address where the facility is situated in the four address lines provided. The legends for the address lines are inline with the setting in ‘Site’ master. Values can be entered in each of the address lines in alphanumeric format of 30 characters maximum length.

  2. Town – Select the Town by using [?] function, or enter the required search criteria and click [Search]. The list of Town names defined in the function ‘Town’ in 'MP code setup' is displayed.

  3. Area - Select Area using [?] function. The list of Area names defined in the function ‘Area’ in 'MP code setup' is displayed.

  4. Region - Select the Region by using [?] function, or enter the required search criteria and click [Search]. The values defined through the function 'Region' in 'MP Code Setup' is displayed.

  5. Postal Code – Enter the Postal Code for the address of the facility in alphanumeric format of fifteen characters maximum length. This legend is displayed based on the Prompt defined in the Function ‘Site’.

  6. Country – Select the name of the Country, where the Facility is located from the list of valid countries displayed. The list of country names defined in the function ‘Country’ in 'MP code setup' is displayed in the list of valid values.

  7. Telephone – Enter the contact Telephone number(s) (a maximum of four) of the facility in alphanumeric format of 20 characters maximum length.

  8. Fax – Enter the Fax number of the facility in alphanumeric format of 20 characters maximum length.

  9. Email – Enter the E-mail ID of the facility in alphanumeric format of 50 characters maximum length.

  10. URL – Enter the Uniform Resource Locator (URL) or homepage address of the facility in alphanumeric format of 50 characters maximum length.

  11. Application Server IP address - Enter the IP address of the application server serving this facility in alphanumeric format of 15 characters maximum length.

  12. Display Site Name – Select the box to display site name at top left of the application screen.

 

Head of the Unit

  1. Name – Enter the Name of the head of the facility in alphanumeric format of 30 characters maximum length.

  2. Designation – Enter the Designation of the person in alphanumeric format of 30 characters maximum length.

  3. License No – Enter the License Number in alphanumeric format of 15 characters maximum length.

 

Logo Details

  1. Facility Logo Image File Name – Enter the name of the image file that is used as the facility logo in alphanumeric format of 40 characters maximum length.

  2. Facility Logo – Click [Browse] to select the image file that has to be displayed as the facility logo.
