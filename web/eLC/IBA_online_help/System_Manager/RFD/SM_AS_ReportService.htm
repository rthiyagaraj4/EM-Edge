Enter the following details to create a Report Service record:

 

  1. ID - Enter the Report Service ID in alphanumeric format of eight characters length. Report Service ID is maintained in upper case and associated to the printer. The Report Service ID has to be unique and cannot be changed after creation. This is a mandatory field.

  2. Name - Enter the Name of the Report Service in alphanumeric format of 30 characters maximum length. This has to be the exact name with which the report service is created on the application server. This is a mandatory field.

  3. Report Server IP Address – Enter the IP address of the server on which the Report Service is configured in alphanumeric format of 15 characters maximum length. It is made default to the login facility’s Application Server IP address, which can be changed.

  4. Remarks - Enter any Remarks pertaining to the report service in alphanumeric format of 200 characters maximum length. You can enter any remarks pertaining to the report service.

  5. Enabled - Select the check box to enable the record in the transaction functions.  The check box is selected, by default.
