Enter the following details to define a Accounting Parameter:

 

  1. Accounting Entity ID – Enter the Accounting Entity ID, which is the logical area, where all the revenues from the hospital are credited to, in alphanumeric format of two characters maximum length.

  2. Accounting Entity Long Name – Enter the long name of the Accounting Entity in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  3. Accounting Entity Short Name – Enter the short name of the Accounting Entity in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  4. No. of Accounting Periods – Enter the number of months that is going to be considered for accounting the cost and revenues from various departments. This is a mandatory field.

  5. Start Fiscal Month – Enter a number corresponding to the calendar year, when the financial year is considered to start for the enterprise. For example 1 stands for January, 2 for February, etc. This is a mandatory field.

  6. Department Length – Enter the number, which controls the maximum length for the department codes that are set up in the enterprise. This is a mandatory field.

  7. Prefix Department by Facility ID – Select the check box to prefix the department code with the Facility ID, when they are being set up in System Configuration module.

  8. Number of Decimals – Enter the number of decimals that are used to display the amount in various transactions and reports in financial module.

  9. Currency Name – Enter a character string that is going to be used as the prefix to the amount that is displayed in the transactions and reports in alphanumeric format of six characters maximum length. This is a mandatory field.

  10. Decimal Name – Enter a character string that represents the decimal units of the currency that is being used in alphanumeric format of six characters maximum length. This is a mandatory field.
