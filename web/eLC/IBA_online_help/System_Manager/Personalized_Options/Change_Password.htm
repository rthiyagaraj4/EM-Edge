Change Password

From the Change Password option, you can Change the password of the logged in user any number of times for security purpose.

The Change Password modifies the passwords and is changed instantly. The Change Password For contains list of values like Login, PIN, Sensitive Diagnosis, High Risk Conditions and Dispensing.

Changing the Password

From the Personalize Options, select the Change Password option.

The Change Password window is displayed.

  1. The logged in User ID is displayed and cannot be changed.

  2. Select a value from the Change Password For where the values are Login, PIN, Sensitive Diagnosis, High Risk Conditions and Dispensing. This is a mandatory field.

  3. Enter the current password of the logged in user in the Old Password field.

  4. Enter the new password in the New Password field. The length of the password must be minimum number of five characters. This is a mandatory field.

  5. Renter the new password in the Re Enter New Password field. This is a mandatory field.

  6. Click [OK] to confirm changing of the password or [Cancel] to cancel the process. On successful validation, the password of the logged in user changes and you are intimated.

The Password Policy legend ‘Password should start with Alphabet and followed either by Numeric values or these Special characters _~$^*`?-[]’ is displayed when the ‘Change Password for’ field value is selected as ‘Login’. This is applicable only when the ‘Enforce Password Policy’ check box is selected in Site screen in SM module.

 

It is mandatory to change password during first login and while logging in for the first time after the password is reset only when the ‘Change Password during First Login’ and ‘Change Password Mandatory’ check boxes are selected in Site screen.
