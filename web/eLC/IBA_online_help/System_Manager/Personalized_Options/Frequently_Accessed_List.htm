Frequently Assessed List

From this option, you can View and Execute the frequently used function that is created in Frequently Accessed Functions  option

The Frequently Assessed List is set to have an easy access to all the frequently used functions.

To access Frequently Assessed List

From the Menu option, select (Frequently Assessed Functions) icon.

  1. Click to access the Frequently Assessed List.

  2. The Frequently Assessed List is displayed at the bottom of the menu frame. The list is based on the settings done in Frequently Accessed Functions Options.
