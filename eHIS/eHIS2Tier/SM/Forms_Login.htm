Logging In

This section guides you through the common process of logging into eHIS.

Logging Into the application

  1. Double-click on the IBAeHIS shortcut.

  2. Enter the User ID, Password and the Domain. Domain indicates the database name.

The entered password is masked for security reasons and displayed as (*).

  1. Click [ Login ] to login to the database or click [ Cancel] to cancel the login process.

  2. Select the Responsibility from the list. The Responsibilities to which the logged in User has access gets listed.

  3. On selecting the Responsibility for the respective User, the facilities associated to the User and the selected Responsibility get listed. Select the required Facility from the list.

  4. Click [OK] to complete the login process or [Cancel] to cancel the login process.

While logging into the application, it is expected that the installation of eHIS has been successfully carried out and the System Administrator has set up the required Facilities, Responsibilities and Users.
