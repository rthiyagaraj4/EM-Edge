Logging In

This section guides you through the common process of logging into Enterprise Management.

Logging Into the application

  1. Switch on the computer, connect to the network and open your Web Browser.

  2. Specify the Destination Path of your server in the Address Bar of the browser window and press [Enter] on the keyboard or click [Go] on the browser window. The login screen gets displayed.

  3. Enter the User Name and Password.

  4. Click [Login] to login to the application or [Cancel] to cancel logging into the application.

The entered password is masked for security reasons and displayed as (●).

  1. Select the required Responsibility from the list. The Responsibilities for which the logged in User has access are displayed in the Responsibility field.

  2. On selecting a Responsibility, based on the User Name and Responsibility, Facilities to which the User has access are listed. Select the required Facility from the list.

  3. On selecting the Facility from the list, click [OK] to complete the login process. To cancel the login process, click [Cancel].

While Logging into the application, it is expected that the installation of Enterprise Management has been successfully carried out and the System Administrator has set up the required Facilities, Responsibilities and Users.
