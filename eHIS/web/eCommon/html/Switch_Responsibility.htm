Switch Responsibility

This section guides you in Switching the Responsibility of the logged in user from the current one to another Responsibility if more than one Responsibility is attached to the user in the System Manager module.

Switching Responsibility

  1. Click [Switch Responsibility] to switch the Responsibility of the logged in user.

  2. Select the Responsibility from the list displayed in alphabetical order. The Responsibilities for which the logged in User has access are displayed in the Responsibility field. The first responsibility is highlighted. Enter the name of the Responsibility in the text box. As you enter the name of the responsibility, the list narrows down in the alphabetical order matching the alphabets e entered in the list box with the name of the responsibility in the list

  3. On selecting a Responsibility, based on the User Name and Responsibility, Facilities to which the User has access are listed. Select the required Facility from the list displayed in alphabetical order. The first facility is highlighted. Enter the name of the facility in the text box. As you enter the name of the facility, the list narrows down in the alphabetical order matching the alphabets entered in the list box with the name of the facility in the list.

  4. On selecting the Facility from the list, click [OK] to switch to another Responsibility or [Cancel] to cancel the process.

  5. On confirmation, the logged in user is successfully switched to the new Responsibility. On cancellation, the logged in user resumes operation with the current Responsibility.
