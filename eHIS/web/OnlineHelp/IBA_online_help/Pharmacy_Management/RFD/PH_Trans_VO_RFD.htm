Enter the required criteria to view Inpatient orders routed to the selected Dispense Location. Select the ‘Category’ as Inpatient and appropriate Dispense Location in the ‘Login Window’.

 

Enter the following criteria for Inpatient category:

 

  1. Period From – Enter the period in date format of dd/mm/yyyy. The search is performed  based on the Prescription Start Date.The orders placed within the specified range are listed.

  2. Ordering Facility - Select the Ordering Facility from the list. The logged in facility is displayed, by default.

  3. Order Date [From-To] – Enter the order date in date format of dd/mm/yyyy. Specify the order date range. The orders placed within the specified range are listed.

  4. Nursing Unit - Enter the nursing unit in alphanumeric format of 20 characters maximum length or select the units using [?] function. This is a mandatory field. All the Nursing Units of the logged in facility are listed in the search screen.

  5. Priority – Select the priority from the list. The listed values are All, Urgent and Routine. ‘All’ gets displayed, by default.

  6. Scope - Select Scope from the list. The listed values are New Prescription, Regular Prescription and Held Orders.

 

Scope

Description

New Prescription

Prescriptions not dispensed

Regular

Prescription Medication orders partially dispensed.

Held Orders

Orders placed on hold during verification.

 

  1. Patient ID – Enter the Patient ID in alphanumeric format of length as specified in MP Parameter or click [?] function to search for the required patient.

  2. Citizen ID – Enter the Citizen ID in the alphanumeric format.

  3. Previous HN – The Previous HN number gets displayed and disabled on entering the Citizen ID.

  4. Dispense Level - Select Dispense Level from the list. The listed values are Order Wise and Patient Wise.

 

Disp Level

Description

Order Wise

Order ID(s) gets displayed followed by the prescribed drugs under each Order ID.

Patient Wise

All the drugs prescribed for the patient irrespective of Order ID are displayed in result section.

  1. Drug/Medical Supply - Select the Drug/Medical Supply value from the list. The listed values are 'All', 'Drugs' and 'Medical Supply'.

  2. Group By Ordering Location - Select the check box to display the prescribed orders by the Ordering Location.

  3. Include Absolute Orders - Select the check box to filter the Absolute Orders. If the check box is selected, the orders prescribed by Absolute, Stat and Narcotic and all Admixtures (excluding TPN Admixture) orders are displayed. If the check box is cleared, the orders prescribed by Stat and Narcotic and all Admixtures (excluding TPN Admixture) orders are displayed. By default, this check box is not selected.

  4. Include Orders by Frequency & Duration  

  5. If the checkbox is selected, the orders prescribed by Frequency and Duration, Stat and Narcotic and all Admixtures (excluding TPN Admixture) orders are displayed. If the 'Calculate By Freq Durn' checkbox is selected then, based on the setup at the parameter level the results are displayed and not by the ordering level.

  6. If the checkbox is cleared, the orders prescribed by Stat and Narcotic and all Admixtures (Excluding TPN Admixture) orders are displayed.

  1. Exclude PRN Drugs - By default, the check box is not selected. If the checkbox is selected, then orders prescribed by PRN Orders will not be displayed. If the checkbox is cleared, then only those orders prescribed by PRN orders is displayed.

  2. Group By - Select the Group By from the list. The listed values are ‘None’, ‘Ordering Location’, ‘Release Date’, ‘Order Date’ and ‘Priority’. When ‘Ordering Location’ is selected, the only value that is available in Sort By is Patient ID.

  3. Sort By - Select the Sort By value from the list. The listed values are ‘None’, ‘Ordering Location’, ‘Release Date’, ‘Patient ID’ and ‘Priority’.

  4. Ascending/Descending - Click Hyperlink to sort the results in ascending or descending order.

  5. Tab Based - Select the check box to divide the results tab as Orders greater than or lesser than the entered time. This check box is disabled when ‘Ordering Location’ is selected in ‘Group By’.

  6. HRS - Enter the time in hours.
