Enter the following TPN Constituent details:

 

TPN Constituents

 

  1. Constituent Generic Name - Select the Generic Name from the list. The listed values are pre-defined. This is a mandatory field.

  2. Short Name - Enter the short name in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  3. Long Name - Enter the long name in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  4. Physical Form - Select the value from the list. The listed values are liquid and solid.

  5. Concentration (gm/100ml) - Enter the concentration in numeric format of two digits maximum length. The specified value is express in terms of the pre-defined Unit of Measurement.

  6. Variance Percentage for Manufacturing - Enter the variance percentage in numeric format of five digits maximum length. During IV Admixture Preparation, you can deviate from the manufacturing standards by the specified percentage only. If variance is not specified, deviation is not allowed.

  7. Enabled - Select the check box to enable the record.
