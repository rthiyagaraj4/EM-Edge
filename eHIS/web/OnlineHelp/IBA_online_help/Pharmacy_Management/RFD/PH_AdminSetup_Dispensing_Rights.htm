Dispensing Controls

The Dispensing Controls for Outpatients and Inpatients are set for the selected User. The displayed fields are enabled or disabled based on the definition of selected Dispense Location. Rights are granted to functions that are enabled.

  1. Change Trade Name – Select the Change Trade Name check box to grant rights to change the Trade Name of a Drug.

  2. Allow Multi Trade – Select the Allow Multi Trade check box to grant rights to allow multiple trades.

  3. Re-route Orders – Select the check box to grant rights to re-route the orders.

  4. Patient History - Select the check box to grant rights to display the Patient History.

  5. Current Encounter Financial Details – Select this check box to view the patient’s current encounter financial details and warn the pharmacist appropriately, if the patient’s insurance policy is expired. By default, this new check box will be unchecked.

Only if this check box  is selected, you can view the patient’s current encounter financial details. Also, you can view a  $ symbol/tool tip as ‘Current Encounter Financial Details’  in the Register Order , Dispense Medication  – (Verification, Allocation, Fill, Deliver, All Stages),  Direct Dispensing, Sales/Patient Issues and Sales Return screens.  

 

OP Sub Process

Rights to invoke dispensing functions applicable to Outpatients are defined.

  1. Registration - Select the ‘Registration’ check box to grant rights to perform registration process.

  2. Verification - Select the ‘Verification’ check box to grant rights to perform verification.

  3. Fill - Select the ‘Fill’ check box to grant rights to perform fill process.

  4. Allocate - Select the ‘Allocate’ check box to grant rights to allocate drugs.

  5. Deliver - Select the check box to grant rights to perform delivery of drugs.

  6. Allow Medication Return after Bill Generation – Select one of the following values from the drop-down list:

    1. Select – This value is selected by default.

    1. Applicable - If you select this value, the system will display all the drugs (bill generated as well as bill not generated) in the Return Medication screen. You will have the rights to return the medication for which the bill is generated..

    1. Not Applicable - If you select this value, the system will display the drugs for which the bill is not generated in the Return Medication screen and you will not have the rights to return the medication for which the bill is generated.

  7. Default Token Series By – This drop-down list will be enabled only if the existing Issue Token check box within the Dispense Location screen is checked. The values in the drop-down are:

  8. Workstation - If this value is selected, the default workstation selected within the Workstations for Token Series screen will be defaulted in the Token Series drop-down list within the Dispense Medication screen.

  9. All - If this value is selected, the same value will be defaulted in the ‘Token Series’ drop down within the Dispense Medication screen.

  10. Individual - If this value is selected, the Token Series drop-down list will be defaulted with ‘Select’ within the Dispense Medication screen.

  1. Order by - This drop-down list will have the following values:

  2. Order Date and Time – Select this value for sorting the patients list based on the patients order date and time.

  3. Register Date and Time - Select this value for sorting the patients list based on the Register date and time.

  4. Patient Arrived Time – Select this value for sorting the patients list based on the patients Arrived time. This value will be displayed in the drop-down list only if the existing check box Generate Token Number before Patient Arrival within the Dispense Location screen is checked.

The default values in the Order By and Token Series drop-down lists in Dispense Medication screen will be populated based on the values selected in Default Token Series By and Order By drop-down lists in the Dispensing Rights screen.

  1. Reissue Medication - Select the check box to grant rights to reissue the medication.

  2. Allow Reprinting of Prescription - Select the check box to allow reprint of prescription.

  3. No. of Reprints Allowed - This field is enabled, if ‘Allow Reprint of Prescription’ is selected. Enter the number of copies to be reprinted in numeric format of two digits maximum length.

  4. Medication Return - Select the check box to grant rights to receive Medication Return by Outpatient.

  5. Default for Fill - Select ‘Against Fill List’ or ‘Without Fill List’ option as required. The setup done in the Dispensing Rights for a particular Dispense Location and a particular User defaults in the   Dispense Medication Login screen.

  6. Default for Deliver - Select ‘Against Fill List’ or ‘Without Fill List’ option as required. The setup done in the Dispensing Rights for a particular Dispense Location and a particular User defaults in the Dispense Medication Login screen.

If there is no setup done in the dispensing rights, then the ‘Against Fill List’ option is selected as default in the Dispensing Medication Login screen.

 

IP Sub Process

Rights to invoke dispensing functions applicable to Inpatients are defined.

  1. Verification - Select the check box to grant rights to perform verification.

  2. Fill - Select the check box to grant rights to perform fill process.

  3. Deliver - Select the check box to grant rights to perform delivery of drugs.

  4. Reissue Medication - Select the check box to grant rights to reissue the medication.

  5. Medication Return - Select the check box to grant rights to receive Medication Return by Inpatient.

  6. Allow Medication Return after Bill Generation – Select one of the following values from the drop-down list:

    1. Select – This value is selected by default.

    1. Applicable – If you select this value, the system will display all the drugs (bill generated as well as bill not generated) in the Return Medication screen. You will have the rights to return the medication for which the bill is generated.

    • Not Applicable - If you select this value, the system prompts a message ‘You do not have sufficient access to return the medication for billed transactions’ during return medication. If you want to do a Ward Return for some of the drugs for which the bill is generated, the system will display the following message ‘Bill has been generated for the drug(s). Would you still like to return the medication?’ with ‘Yes’ and ‘No’ buttons..

These new drop downs lists are enabled only if the Medication Return check box under the OP Sub Process and IP Sub Process sections is selected, respectively.

 

Amend Order Rights

  1. Allow Change of Dose - Select the check box to allow change of dosage during Dispensing Stage.

  1. Unlock Records Locked by Other Users - Select the check box to unlock the records locked by other users. This check box is cleared, by default. The users can unlock the self locked records.

  2. Tracking Outsource Medication - Select this check box to provide access rights for outsourcing the drugs.

 

Access Rights For

  1. Send – Select this check box to provide access rights for sending drugs to external facility. This check box is enabled only if the Tracking Outsource Medication check box is selected.

  2. Receive – Select this check box to provide access rights for receiving drugs from external facilities if the Tracking Outsource Medication check box is selected.

  3. Deliver - Select this check box to provide access rights for delivering drugs to  patients in the specific location in the hospital if the Tracking Outsource Medication check box is selected.

  4. Default Tracking Action – Select the Default Tracking Action in the drop-down list. The Access Rights selected in the Access Rights For section are listed.

  5. Default Referral Facility – Select the Default Referral Facility from the drop-down list. All the Referral Facilities are listed.
