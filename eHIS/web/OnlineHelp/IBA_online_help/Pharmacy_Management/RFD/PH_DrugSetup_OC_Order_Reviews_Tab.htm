Order Reviews Tab

You can specify rules that influence order processing for the various patient classes while ordering the drug.

 

Enter the following order review details:

 

  1. Patient Class – The pre-defined Patient Classes are displayed. The characteristics are defined for each class (if applicable). You can define the characteristics specific to a patient class, which overrides the values set for ‘All’.

  2. Applicability - Select the value as ‘Applicable’ or ‘Not Applicable’ from the list. If Drug category is associated to the drug, the patient classes associated to the drug category are displayed as applicable, or they are not applicable.

  3. Authorization Required – If ‘Applicable’ is selected, the check box against the patient class is enabled. If authorization is required for ordering the drug, select the check box. Ensure that you select this check box to overrule the authorization only for those Order Catalogs previously authorized by Senior Doctor.

  4. Cosign Required - If ‘Applicable’ is selected, the check box against the patient class is enabled. If consigning is required for ordering the drug, select the check box.

  5. Special Approval Required - If ‘Applicable’ is selected, the check box against the patient class is enabled. If special approval is required for ordering the drug, select the check box.
