Enter the following details:

  1. Drug Name - Select the drug name from the Drug Name lookup for which quota limit must be defined.

  2. Specialty - Select the required Specialty from the lookup. The selected speciality will be displayed under ‘Associate Speciality’ header.

  3. Set Limit By - There are two radio buttons: Patient and Quantity. By default, ‘Patient’ will be selected. If required user can change as Quantity.                 

  4. Quota Limit – Enter the quota limit until which user can order the quantity without any approval. Only numerical characters are allowed up to 3 digits. It cannot be zero or blank.

  5. Additional Quota Limit – Enter the additional quota limit until which user can order the quantity with an approval. Once exceeds the additional quota limit, user can order the drug only after approval. Only numerical characters are allowed up to three digits. Zero or blank is not accepted.

  6. Reset – The ‘Rest’ link will be available only if ‘Set Limit By’ is selected as ‘Patient’. On clicking the Rest link, the message ‘The defined quota will be reset from start. Do you want to continue’ gets displayed with OK and Cancel buttons.

a. Ok –  Click OK, the ongoing quota limit count in transaction will get reset and quota calculation will start from first.

b. Cancel – Click the cancel button to reset the quota limit. There will be no audit for who did the reset in front end.

  1. Select – Select the check box against the required record and click [Apply] to associate the specialty to the drug.  

If you click Apply without any speciality associated, then the message ‘At least one Speciality should be included for associating it to a drug’ gets displayed.
