Enter the following search criteria:

  1. Patient ID – Enter the Patient ID in the format as specified in Master Patient Index module or select the required Patient ID using [?] function. This is a mandatory field.

  2. Reissue Drug – Enter the Drug to be re-issued in alphanumeric format of 60 characters maximum length or select the required Drug using [?] function.

  3. Dispense Date [From–To] – The system displays the current system date as the starting dispensing date, by default. Enter the required date range in the format of dd/mm/yyyy or select the required date using the Calendar function. This is a mandatory field.

The drug lookup function lists only those drugs that can be dispensed against damage.

If you try to reissue the medication beyond the number of quantity or number of times that is set in the Drug screen, the following message is displayed:

“The reissue quantity  cannot exceed <<No of Qty set in the parameter>> quantities”

 

You can click OK and the cursor focus will be on the Dispense quantity field.

 

Similarly, if you reissue the medication beyond the number of times that is set in the Drug screen, the following message is displayed:

“The maximum number of reissues has been met. So, cannot reissue the medication.”

 

You can click OK and the cursor focus will be on the Reissue Medication screen.
