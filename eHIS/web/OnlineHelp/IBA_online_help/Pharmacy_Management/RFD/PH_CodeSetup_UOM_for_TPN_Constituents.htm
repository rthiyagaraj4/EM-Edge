TPN Parameter

 

Unit of Measure

  1. Energy – Select the Unit of Measurement from the list. The Unit of Measurement defined in the ‘Unit of Measurement’ function with the Quantity check box selected in ‘Application Masters’ module are listed. The selected UOM is used to express the energy requirement during TPN Ordering. This is a one time setup and this is a mandatory field.

  2. Prescribing Base Unit for Regimen - Select the Unit of Measurement from the list. The Unit of Measurement defined in the ‘Unit of Measurement’ function in ‘Application Masters’ module with the Quantity check box selected are listed. The selected UOM is the Prescribing Base Unit for Non-standard Regimen. This is a mandatory field.

  3. Order Regimen By – Select the Order Regimen By value. The listed values are ‘Age’ and ‘Weight’.

  4. Selection of Preferred Item Fluid Applicable – Select the check box, if Preferred Item Fluid is applicable.

  5. Maximum weight for pediatric patient – Enter the Maximum Weight for Pediatric Patients in numeric format of three digits maximum length.

  6. Minimum weight for adult patient – Enter the Maximum Weight for Adult Patient in numeric format of three digits maximum length.

  7. Expiry period for TPN Admixtures – Enter the Expiry Period for TPN Admixtures in numeric format of two digits maximum length.

 

Quantity Unit of Measure for TPN Constituents

  1. Constituents - The Constituent Group Names and their Independent generics (Generics with UOM flag as ‘I’) are displayed. The Generics are displayed under each Group.

 

Standard Regimen

  1. Regimen Setup - Select the Unit of Measurement for each of the displayed constituent from the list. The Unit of Measurement defined in the Unit of Measurement function in Application Masters module, with Quantity check box selected are listed. The specified UOMs are displayed during Standard TPN Ordering.

 

Non Standard Regimen

  1. Constituents Limit - Select the Unit of Measurement for each of the displayed constituent from the list. The Units of Measurement defined in the Unit of Measurement function in Application Masters module, with Quantity check box selected are listed. The specified UOMs are displayed in the function ‘TPN Constituent Limit’.

  2. Order By - Select the Unit of Measurement for each of the displayed constituent from the list. The Units of Measurement defined the Unit of Measurement function in Application Masters module, with Quantity check box selected are listed. The specified UOMs are displayed in the function ‘TPN Regimen’.

  3. Preparation at Pharmacy - Select the Unit of Measurement for each of the displayed constituent from the list. The Units of Measurement defined the Unit of Measurement function in Application Masters module, with Quantity check box selected are listed. The specified UOMs are displayed in the Transactions function ‘Prepare IV Admixture’.
