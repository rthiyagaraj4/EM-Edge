Timing Tab

Some of the values under ‘Timing’ tab are inherited from the selected Order Type.

 

Enter the following characteristics of Drug:

 

  1. Change Qty/Dose – Select the check box to change the quantity and dose in ‘Setting’ tab.

  2. UOM – The base unit defined for the drug is displayed as read-only text.

  3. Split – To indicate splitting of drug dosage while ordering, select the check box.

  4. Taper Up - To allow tapering up of dosage while ordering, select the check box.

  5. Taper Down - To allow tapering down of dosage while ordering, select the check box.

  6. Sliding Scale – To indicate sliding scale method is adopted during ordering of the drug, select the check box.

  7. Frequency Required - The frequency required is selected, by default. The value cannot be changed. It indicates frequency has to be specified while ordering. The default value is based on the selected order type.

  8. Soft Stop for Inpatients - This field is not applicable for pharmacy Order Category.

  9. Create as Continuing Order – The value is selected by default and disabled. You cannot edit the value. The default value is based on the selected order type.

  10. Continuous Frequency Action - If create as continuing order is applicable, the value for the field is defined as ‘Explode by Day’. You cannot change the value. The value ‘Explode by Day’ indicates orders are generated while ordering, if the start medication day is the current system day. Subsequent order for the next day is placed at midnight.

  11. Allow Refills - Select the check box to allow refills. This field is applicable based on PH parameter setup.

  12. Priority Applicable – Select the applicable default priority while ordering the drug from the list. The listed values are Routine & Urgent, Stat and Routine.

  13. Dispense Auth. Required – Select the check box to indicate whether authorization is required while dispensing the drug. The check box is enabled only when ‘Dispense Authorization’ is applicable for the schedule attached to the drug.
