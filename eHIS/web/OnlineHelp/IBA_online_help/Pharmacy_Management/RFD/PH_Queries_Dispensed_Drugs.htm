Enter the following search criteria:

 

  1. Search By – Select either Drug Name or Trade Name as search by criteria.

  2. Drug Name/Trade Name - Based on the selected ‘Search By’ value, the screen displays either Drug Name or Trade Name as field legend. Enter the Drug Name or Trade Name in alphanumeric format of 40 characters maximum length or select the required Drug Name or Trade Name using [?] function. This is a mandatory field.

  3. Route - Select the Route from the list. The Routes associated to the selected Form of Drug is displayed for selection.

  4. Dispense Date [From-To] - Enter the dispense date range in the format of dd/mm/yyyy or select the required date range using the Calendar function. The To date must be greater than the From date. If From date is not specified, the system considers the lowest date as From date value. If To date is not specified, the system considers the highest date as To date value. If both the values are not specified, the system considers lowest date for From value and highest date for To value. This is a mandatory field.

  5. Dispense Location - Select the Dispense Location from the list. The Dispense Locations defined in the Dispense Location function are displayed.

  6. Billing Group - Select the Billing Group from the list. The Billing Groups defined in the Patient Billing Manager module is displayed for selection.
