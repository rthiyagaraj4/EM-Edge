Patient Order Details

Medication Orders of the patient are displayed in chronological order. The Patient ID, Name, Age and Gender of the selected patient gets displayed.

 

Order Details

  1. Order Type – Order Type selected while ordering gets displayed.

  2. Order ID – The Order ID generated on ordering gets displayed and hyperlinked.

  3. Order Date - The Ordering Date gets displayed.

  4. Location – The ordering location gets displayed.

  5. Practitioner – The ordering Practitioner gets displayed.

  6. Status – The table below lists the color codes for each status.

 

Status

Color coding

Description

New Orders

New orders yet to be dispensed.

Refill

Refill prescriptions.

Partially dispensed orders.

Discharge medication orders.

Regd Pres

Registered orders.

 

  1. Select - Select the check box to consider the order for registration. If the selected patient has multiple orders, the first check box is selected, by default. To consider other orders for registration, select the corresponding check boxes.

  2. Print - For the selected Dispensing Location, if ‘Print Prescription’ is applicable, this field is enabled. Select the check box to print prescription on registering the order.

To view other order details, click the required Order ID hyperlink. The following prescription details are displayed.

  1. Drug Name –

  2. The name of the drug prescribed gets displayed.

  3. The Trade Name and the Drug/Trade Image are displayed with the Drug Name, if Trade Name and Drug/Trade Image is set in the Drug Master Setup.

  4. The external drug information icon   is displayed against the drug name, if the external drug code is interfaced with the drug name.

  5. The Drug Information/Patient Education icon  is displayed against the drug name. Click Drug Information/Patient Education icon  to view the Drug Information and the Patient Education details recorded in the Drug Master Setup.

  1. Dosage Detail – The Strength of the Drug or Quantity, Frequency and prescribed Drug Duration are displayed.

  2. Charge Amount – The amount charged for the prescription gets displayed.

  3. Patient Payable Amount – The Payable amount is calculated and displayed.

  4. Include? - It is used to calculate the Patient Payable Amount based on the Reimbursement, Insurance setup for the drug. Click ‘Include?’ hyperlink, to capture the Insurance or Reimbursement details.

    Enter the following details:

    1. Included? – If Insurance or Reimbursement is included during the Billing Setup for the selected Item, the symbol gets displayed or the symbol gets displayed.

    2. Include? – The check box is enabled, if Insurance or Reimbursement is applicable for the selected Item.

    3. Reason – Select the Reason using [?] function, if the Item is not included and you are forcefully trying to include the Drug for Reimbursement or Insurance.

    4. Approval Reqd – If approval is required for the selected Drug, the symbol is displayed or the symbol gets displayed.

  5. Select – Select this check box to select only the drug(s) that you want to register from a medication order. By default, the checkbox will be selected. You can de-select the drug(s) which are not required to be registered. Once you register one or more drugs from a medication order, the registered drug(s) will not be available for further registration. The Select checkbox of already registered drugs will be in disable mode and the background will be highlighted in green colour.
