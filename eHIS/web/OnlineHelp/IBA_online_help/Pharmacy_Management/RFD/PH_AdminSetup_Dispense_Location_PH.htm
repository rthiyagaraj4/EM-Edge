Type: Pharmacy

On selecting the type as ‘Pharmacy’, the screen as shown is shown by default.

 

Enter the following details to create a Dispense Location of type ‘Pharmacy’:

On selecting the type as ‘Pharmacy’, the following fields are displayed. Enter the following details.

  1. Working Hours [From-To] – Enter the working hours of the Dispense Location in time format of hh:mm. The ‘To’ value must be greater than ‘From’ value.

  2. List box - On selecting the type as ‘Pharmacy’, specified whether the location caters to both or either Outpatients/Inpatients. The list next to the ‘Type’ field displays the following values.

Both

  • On selecting ‘Both’ as value, in the Dispensing Rules both ‘Outpatient Prescription’ and ‘Inpatient Prescription’ tabs are displayed.

OP Pharmacy

  • On selecting ‘OP Pharmacy’, in the Dispensing Rules ‘Outpatient Prescription’ tab is displayed.

IP Pharmacy

  • On selecting ‘IP Pharmacy’, in the Dispensing Rules ‘Inpatient Prescription’ tab is displayed.

  1. Stage – Select the stage from the list. The listed values are ‘Any Stage’, ‘Single Stage’ and ‘Multiple Stage’. This is a mandatory field.

  2. Single Stage – The users associated to the Dispense Location can access only one dispensing stage function at a time.

  3. Multiple Stage – The users associated to the Dispense Location can access multiple dispensing stage functions.

  4. The user rights are defined in the function ‘Dispense Rights’.

  1. Enabled – Select the check box to enable the Dispense Location.

  2. Dispense Level - Select the value from the list. The listed values are Order Wise and Patient Wise. By selecting ‘Order Wise’, all the Order ID's of the selected patient are displayed and on selection of an Order ID, prescribed drugs are listed. If the setup is ‘Patient Wise’, all the orders of the patient are displayed irrespective of the Order ID's.

 

Three tabs are displayed, namely, ‘Outpatient Prescription’, ‘Inpatient Prescription’, ‘Other Details Tab’ based on the Location Type settings.
