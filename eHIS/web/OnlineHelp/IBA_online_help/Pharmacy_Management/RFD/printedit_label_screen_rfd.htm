Enter the following details:

  1. Dispensing Sheet - Select this check box to print the dispensing sheet after dispensing the medication for the Dispense Location.

  2. Dispense Label - Select the check box to print the dispensing label after dispensing the medication for the Dispense Location.

  3. <Default language> drop-down list – Displays the default language as set in the PH Parameter for Facility screen under the Outpatient and Inpatient tabs.

  4. Home Medication Sheet (include Eye Preparation) – Select this option to print the Home medication Summary Report that includes Eye preparation.

  5. Home Medication Sheet (Separate Eye Preparation) - Select this option to print the Home medication Summary Report that includes Separate Eye preparation.

  6. Home Medication Sheet (Only Eye Preparation) - Select this option to print the Home medication Summary Report that includes Only Eye preparation.

  7. Home Medication Sheet (Taper Order) - Select this option to print the Home medication Summary Report that includes Taper Order.

  8. Home Medication Sheet Orientation – This drop-down has values such as ‘Portrait’, ‘Landscape’ and ‘Both’. For inpatient and outpatient, the Home Medication Summary report will print in portrait mode. By default, the value ‘Portrait’ is selected.
