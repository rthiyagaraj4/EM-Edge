Anatomical Class Frame

  1. Click [Add] to invoke the Anatomical Class Insert screen.

  2. Enter the details in the mandatory fields. <Related Field Description>

  3. Click [Record] to save the defined Anatomical Class, [Reset] to clear the entered data or [Cancel] to exit from the screen.

  4. Click ‘+’ hyperlink to view the Therapeutic Class (Main) records.

  5. Click on the defined record and select [Modify Selected Item] to modify the record or [Modify All – Anatomical Class]. The ‘ATC Classification Modify/ATC Classification Modify All’ screen gets displayed.

  6. Enter the details to update the record.

  7. Click [Record] to update the details, [Reset] to clear the entered data, [Delete] to delete the record or [Cancel] to exit from the screen.

The delete option is enabled, if the selected class does not have any sub records grouped under it.

If ‘ATC Classification Modify All’ is selected, a check box is provided to delete the record. Select the ‘Delete’ check box against the desired record to delete the record.
