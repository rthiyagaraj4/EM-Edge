Settings Tab

The default frequency and duration values are specified for the applicable patient class in the ‘Settings’ tab. The standard and maximum duration values are recorded along with action on exceeding the maximum value.

 

Enter the following details to set the Frequency:

 

  1. Patient Class - The applicable Patient Classes are displayed. These classes are pre-defined. The characteristics are defined for each class (if applicable). You can define the characteristics specific to a Patient Class, which overrides the values set for ‘All’.

  2. Applicability – Select the value as either ‘Applicable’ or ‘Not Applicable’. If applicable is selected, define the default duration/frequency values for the Order Catalog.

  3. Age Group - Select the Age Group from the list. The listed values are defined in Application Masters module.

  4. Change Qty/dose – The value ‘1’ is displayed, by default. The quantity/dose value can be changed provided ‘Change Qty/dose’ under ‘Timing’ sub tab (‘Characteristics’ tab) is selected. Enter the value in numeric format of three digits maximum length.

  5. UOM - The base unit defined for the drug displays as Unit of Measurement. The value cannot be changed.

  6. Frequency – Select the Frequency from the list. This is a mandatory field. The values displayed in the list are defined in the tab ‘Frequency Applicable’.

  7. Durn Value/Type - Specify the duration value in numeric format of three digits maximum length. The type value displayed corresponds to the selected frequency as read-only text.

  8. Max Value/Durn Type - Enter the maximum value in numeric format of three digits maximum length. The maximum value that can be ordered for the defined Order Catalog is specified. On entering the maximum value, the duration type corresponding to the selected frequency is displayed. The maximum value must be greater than the Duration value. If drug category is associated to the drug, the maximum prescription duration specified (Associated rules for drug category function) is displayed as default value. The displayed value can be changed.

  9. Action - Select the value from the list. The values displayed are Reject, Warn and Request for Authorize. On selecting the value, appropriate action is performed if maximum value is exceeded. This can be set only for ‘All’ patient class. If Drug Category is associated to the Patient Class, the action defined for the Drug Category is displayed as read-only text.

  10. Msg – Click 'Msg' hyperlink to enter the message to be displayed for the specified action.

    • Enter the required prompt message and click [OK] to shift the control back to the main screen.

    • For other patient classes, if Drug Category is associated to the Drug, the message entered in the category is displayed that cannot be changed.
