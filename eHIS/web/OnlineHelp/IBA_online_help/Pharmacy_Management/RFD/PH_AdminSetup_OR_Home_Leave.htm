Routing Level: Home Leave

Routing Code

The Order Types belonging to Order Category ‘Pharmacy’ are listed. Select the required Order Type. This is a mandatory field.

Performing Facility

The facilities of the enterprise are listed. Select the required facility. This is a mandatory field.

 

Default Dispense Location

Routine

Select the Dispense Location to route routine orders from the selected Location Type and Order Type. The locations listed are defined in the function ‘Dispense Location’. This is a mandatory field.

Stat

Select the Dispense Location to route stat orders from the selected Location Type and Order Type. The locations listed are defined in the function ‘Dispense Location’. This is a mandatory field.

Discharge/Take Home Medication

Select the Dispense Location to route discharge/take home medication orders from the selected Location Type. The locations listed are defined for the selected ‘Location Type’ in the function ‘Alternate Dispense Location’. This is a mandatory field.
