Supportive Drugs Tab

This tab helps users to map more than one supportive drug to main drug. Supportive drugs are used to lessen the harmful side effects of treatment by protecting certain cells or organs. Supportive drugs are also called protective drugs.  

 

In transactions, while placing pharmacy orders, in the Drug lookup while searching for drugs, the system prompts users to automatically add mapped supportive drugs matching to the main drug. This is applicable for Normal Rx type prescriptions only.

 

Adding Supportive Drugs to main Drug:

  1. Drug Name - Select the required drug name from the lookup.  

  2. User can select more than one supportive drug against the main drug and add to the list.

  3. On selecting the supportive drugs, the below columns will be displayed.

• S. No – Displays the serial number series of the supportive drug.

• Drug Name - Displays the short name of the drug.

• Generic Name - Displays the Generic name of the drug.

• Form - Displays the form defined for the selected drug which was defined in PH Drug setup.

• Strength - Displays the strength of the selected drug which was defined in PH Drug setup.

• Admin Route - Displays the Admin Route of the selected drug which was defined in PH Drug setup.

• Action - By default all supportive drugs check boxes are selected.

  1. Click [Apply] button to add supportive drugs to the main drug.

  2. If you want to delete a supporting drug, select the record and click [Delete] button. The system prompts a validation message 'Selected supportive Drugs shall be deleted. Do you want to continue' with OK and Cancel buttons.

  3. Click [OK] to delete the selected supportive drugs from the list

  4. Click [Cancel] button to cancel the delete option.
