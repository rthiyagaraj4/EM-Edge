Enter the following details to create a TPN Regimen:

  1. Standard Regimen – Select the check box to indicate the TPN Regiment as a Standard Regiment. The check box is selected, by default.

  2. PPN – Select the check box to indicate the Regimen as Partial Parenteral Nutrition. Standard TPN with PPN selected is used while ordering Non Standard Regimen.

  3. Regimen Code - Based on the PH Parameter Setup, you are allowed to enter Regimen Code. If allowed, enter the Regimen Code in alphanumeric format of 15 characters maximum length or the Regimen Code depends on the ‘Item Code’. This is a mandatory field.

  4. Long Name - Based on the PH Parameter, the long name is displayed. If link between Drug and Item Code exists, Item description is displayed that can be modified or enter the long name in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  5. Infusion Line – The Infusion Line is set to ‘Central’ and ‘Peripheral’. Select a valid entry from the list. This is a mandatory field.

  6. Single Emulsion - Select the check box, to indicate the regimen is a single emulsion. Clear the check box, if the Regimen is multi-emulsion.

  7. Item Code - Enter the Item Code in alphanumeric format of 10 characters maximum length or select the require Item Code using [?] function.

  8. Prescribing Base Unit - The Prescribing Base Unit defined in the function ‘UOM for TPN Constituent’ is displayed as read-only text.

 

You can invoke the Constituents and Ordering Dispensing Rules tabs only on entering these details.
