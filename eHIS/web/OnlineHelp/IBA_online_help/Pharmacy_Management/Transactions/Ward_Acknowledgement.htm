Ward Acknowledgement

From this function you can Acknowledge Dispense Medication/ Ward Return/ Sale/ Sale Return transaction against the patient ID.

This function allows you to acknowledge the Items dispensed from the pharmacy. This function is applicable for Inpatients only. Once the item is acknowledged, the same item records does not get displayed in “Ward acknowledgement” screen. You can return the items through Ward Return screen.

Acknowledging Ward details

From the PH Transactions, select Ward Acknowledgement function.

The Ward Acknowledgement screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to view the dispensed drugs.

  3. If the status ‘Acknowledged’ is selected in search criteria, the search results column include Dispense No, Dispense Date Time, Dispense Location, Bed No, Nursing Unit, Order No, Order By, Dispense By, Acknowledged By Date and Time.

  4. If the status ‘Not Acknowledged’ is selected in search criteria, the search results column include Dispense No., Dispense Date Time, Dispense Location, Bed No, Nursing Unit, Order No, Order By, Dispense By and Acknowledge check box.

  5. To acknowledge the item, select the Acknowledge checkbox and click [Apply]. The item status gets changed to “Acknowledged”. If you click [Apply] without  selecting the check box, a validation message “Please select the acknowledge item” is displayed. If you select the status as “Acknowledged.

  6. Click [Reset] to clear the entered data.
