Tracking Outsource Medication

From this function you can Send/Receive/Deliver the Medication Orders.

This function is used to send the drugs to the external facility and to receive the drugs from the external facility. The drugs can be delivered to the patients in the specific location inside the hospital.

 

From the PH Transactions, select the Tracking Outsource Medication function.

The Dispense Medication Login screen gets displayed.

  1. Tracking Outsource Medication screen gets displayed.

    Enter the following details:

    1. User Name – The logged in User Name gets displayed as read-only text.

    2. Password – Enter the password to perform dispensing. In the function ‘Dispense Rights’, password stored for the user in User Profile is validated. This is a mandatory field.

    3. Dispense Location – Select the Dispense Location from the list. The locations listed (Pharmacy Type) are assigned to the logged in user with rights to perform various dispensing functions. This is a mandatory field.

    Click [OK] to get the Tracking Outsource Medication screen. Depending on the Dispensing Stage selected, the Tracking Outsource Medication screen gets displayed.

  2. Click [OK] to login to the Tracking Outsource Medication function or [Cancel] to cancel the login process.

  3. Tracking Outsource Medication screen gets displayed.

  4. Enter the details in the mandatory (*) fields. <Related Field Description>

  5. Click [Switch Login] to display Dispense Medication Login screen. Switch Login aids in switching to different Dispense Locations and/or Category from the current login. The Tracking Outsource Medication screen closes and control shifts back to the login screen.

  6. Click [Apply] to save the outsourcing medication orders or [Reset] to clear the entered data.

  7. Click [Query] to search for an existing outsourcing medication orders.
