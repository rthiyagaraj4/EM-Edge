Re-route Prescription

From this function you can Query and Re-route the prescription to other Dispensing Location in the same Facility or another Facility for a patient.  

The re-route can happen within the facility to any other Dispensing Location, or from one facility to other facility.

Re-routing prescription

From the PH Transactions, select the Re-route Prescription function.

The Re-route Prescription screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to display the order details based on the entered criteria.

  3. Click Order ID hyperlink to view the Current Medications details.

  4. The Order Date, Order Status, Current Facility Name, Current Dispense Location details and Prescription type drop-down list are displayed.

  5. 5. The Prescription Type drop down have two values ‘Internal’ and ‘External’. By default, the value ‘Internal’ will be selected. If the default value is selected, the existing functionality of re-routing the prescription to an internal facility will be continued. If you need to refer the prescription to an external facility, you have to select the value ‘External’ in the ‘Prescription Type’ drop down list. Once the External value is selected, the existing Re-Route Facility and the Dispense Location drop down lists will be disabled and you need to check the existing ‘Select’ checkbox and click on the Apply button. A new screen External Facility Details is displayed. The external facilities created within the Application Masters will be listed in the External Facility drop down list. The orders are rerouted in the following criteria:

    The following fields are displayed:

    1. External Facility – Select the external facility to which the order(s) needs to be re-routed from the drop-down list. The external facilities created in the Application Masters module are listed in the drop-down list. This is a mandatory field.

    2. Next Collection Date – Select the next collection date using calendar function. This field is blank by default. This is a mandatory field.

    3. Balance Medication Remarks - Select the Balance Medication Remarks from the drop-down list.

    4. Remarks – Enter the remarks about the balance medication. This is a mandatory field.

    5. Referred by - The logged in pharmacist name will be defaulted within the ‘Referred by’ field.

    6. Print External Description - Select this check box to print the external prescription. By default, this check box is selected.

    7. [Record] – Click this button to save all the entered details.

    8. [Reset] – Click this button to reset all the details.

  • If an order is partially dispensed and the balance medication is re-routed to the external facility, the order status will be changed to ‘Dispensed Complete’. If a fresh order is re-routed to the external facility, the order status will be changed to ‘Dispensed Complete’.

  • If you retrieve the referral prescription from the external facility and re-routes it to the internal facility which was partially dispensed before sending it to the external facility, the status of the order will be changed to ‘Dispensed Partial’.

  • If you retrieve the referral prescription from the external facility and re-routes it to the internal facility for which the dispensing was not done before re-routing it to the external facility, the status of the order will be changed as ‘Ordered’.

  1. Select Re-route Facility Name from the list to re-route the prescription to another facility. By default, the logged in Facility name is displayed.

  2. Select the Re-Route Dispense Location from the list to re-route the prescription to other Dispense Location.

  3. Select the Reason from the list. The listed values are defined in Reason Codes for Applicable Transaction ‘Re-route prescription’.

  4. Select the Dispense Medication check box if the order is an inpatient order. This check box is active only for inpatient orders.

  5. Select the check box to select the order to re-route to other location.

  6. Click [Apply] to re-route the prescription to selected Dispense Location in another facility or [Reset] to clear the entered data.

  7. Click on respective hyperlinks to view the patient’s Allergies, Problems, Drug Profile, Current Rx, Demographics, Dispensed Drugs, Pending Orders and Patient History.

The Pending Orders hyperlink will be displayed in bold and red in colour and it will blink.
