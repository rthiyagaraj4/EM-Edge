Edit Dispense Label

From this function you can Edit the Dispense Label once the Drugs are dispensed.

You can access this function if 'Edit Dispense Label after Dispensing / Delivery' check box is selected in the PH Parameter for Facility function and based on the User Access Rights to the Dispense Location.

Editing Dispense Label

From the PH Transactions, select the Edit Dispense Label function.

The Edit Dispense Label Login screen gets displayed.

  1. Enter the Login Details and click [OK] to login to the Edit Dispense Label function or [Cancel] to cancel the login process.

    Enter the following details:

    1. User Name – The logged in User Name gets displayed as read-only text.

    2. Password – Enter the password to edit the Dispense Label. In the function ‘Dispense Rights’, password stored for the user in User Profile is validated. This is a mandatory field.

    3. Dispense Location – Select the Dispense Location from the list. The locations listed (Pharmacy Type) are assigned to the logged in user with rights to perform various dispensing functions. This is a mandatory field.

  2. On successful login, the Edit Dispense Label screen gets displayed.

  3. Enter the search criteria and click [Search] to display the dispensed drugs.

    Enter the following search criteria:

    1. Patient ID - Enter the Patient ID or select the Patient ID using [?] function.

    2. Dispense No - Enter the Dispense Number is alphanumeric format of 15 characters maximum length.

    3. Dispense Date - Enter the Dispense Date is the format of dd/mm/yyyy or select the date using Calendar function.

  4. Click Edit Label hyperlink to edit the Dispense Label for the dispensed drugs.

    The Patient Details and the Drug/Dosage details of the selected Patient is displayed.

    • Prescribed Drug, Prescribed Quantity (stock unit) and Expiry Date of the selected batch are displayed. The batch expired date gets displayed only for prescribed drug.

    • No of Copies  – The No of Copies of the label depends on the item’s unit of measure. You can also enter the number of copies required. By default, it is set to 1.

    • Patient ID, Patient Name, Prescription Date and Prescribing Practitioner details get displayed.

    Enter the following details:

    1. Auxiliary Instructions (English) – Select the Cautionary Instructions, Special Instructions and Prescription Remarks from the list. Enter the Patient Instructions in alphanumeric format.

    2. Auxiliary Instructions (Local Language) –Enter the Cautionary Instructions, Special Instructions and Patient Instructions in the required format.

  5. On entering the required details, click [OK] to record the instructions, [Cancel] to exit from the Edit Label screen. Click [View Audit Trail] to view the changes made in the Dispense Label. The Date, User and Work Station details are displayed.
