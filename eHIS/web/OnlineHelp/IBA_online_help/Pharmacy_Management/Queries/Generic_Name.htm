Generic Name

From this function you can Query the Drug based on Generic Name.

The query is limited to Generic Names associated with a Drug Class by specifying the required Drug Class.

Querying Drugs by Generic Name

From the PH Queries, select the Generic Name function.

The Generic Name screen gets displayed.

  1. Click [Search] to display all the defined Generic Names or enter the search criteria and click [Search] to select a specific Generic Name.

    Enter the following search criteria:

    1. Generic Name – Enter the desired Generic Name in alphanumeric format of 40 characters maximum length or select the Generic Name using [?] function.

    2. Drug Class – Select the required Drug Class from the list. The listed values are ‘General’, ‘Narcotic’ and ‘Controlled’.

    3. No. of Records – Select the required number to limit the display of the query result.

  2. On searching, the Generic details are displayed based on the entered criteria.

  3. Click [Reset] to clear the specified criteria.
