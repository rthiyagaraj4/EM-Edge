Drugs Price

From this function you can Query the Drug price based on selected Store and Drug.

This function is used to allow the pharmacist to enquire about prescribed drug price details based on selected Store and Drug. This screen facilitates the pharmacist to provide the prescription price to the patients without viewing Dispense screen and assist patients to take decision about taking prescribed drugs or not.

Querying Drugs Price

From the PH Queries, select the Drugs Price function.

The Drugs Price screen gets displayed.

  1. Enter the search criteria and click [Search] to view the Item, Item Description, Batch, Expiry Date, UOM, Issue MRP, Stock UOM, Stock Unit Cost, Store On Hand, Allocated Quantity based on the selected store and drug code range.

    Enter the following details:

    1. Store [From-To] – Select the Store range using lookup function. The store codes defined in the Pharmacy masters are listed.

    2. Drug [From-To] - Select the Drug code using lookup function. The drug codes defined in the Pharmacy masters are listed. This is a mandatory field.

  2. Click [Reset] to clear the specified criteria.
