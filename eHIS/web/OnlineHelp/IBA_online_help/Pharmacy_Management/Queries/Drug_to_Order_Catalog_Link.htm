Drug To Order Catalog Link

From this function you can View details of the Drugs, which are linked or not linked to an Order Catalog.

A Drug is linked to an Order Catalog during its creation. You can view the details of Drugs, which are linked and not linked to the Order Catalog.

Viewing Drugs which are linked or not linked to an Order Catalog

From the PH Queries, select the Drug To Order Catalog Link function.

The Drug To Order Catalog Link screen gets displayed.

  1. Click [Search] to display all the linked or not linked Drugs based on the default option selected or enter the search criteria and click [Search] to display a specific Drug, which is linked or not linked to the Order Catalog.

    Enter the following search criteria:

    1. Drug Name - Enter the Drug Name in alphanumeric format of 60 characters maximum length or select the desired Drug using [?] function.

    2. Drug Class – Select the required Drug Class from the list. The listed values are ‘General’, ‘Narcotic’ and ‘Controlled’.

    3. Linked – Select the option to view the Drugs linked to Order Catalog. This option button is selected, by default.

    4. Not Linked – Select the option to view the Drugs, which are not linked to Order Catalog.

  2. On searching, the Drug details are displayed based on the entered criteria.

  3. Click ‘+’ hyperlink to view the Drug Details.

  4. Click [Reset] to clear the specified criteria.
