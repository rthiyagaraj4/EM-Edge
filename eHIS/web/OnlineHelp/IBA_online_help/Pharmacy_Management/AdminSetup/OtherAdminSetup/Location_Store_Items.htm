Location Store Items

From this function you can Create an association of Item to the Location and Store for an Item Type and Query for the existing association of Item to Location and Store.

Creating an association Item to the Location and Store

From the PH Code Setup > PH Other Admin Setup, select the Location Store Items function.

The Location Store Items screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Search and select the Dispense Location using [?] function. The listed Dispense Locations of the Type ‘Clinic’ and ‘Nursing Unit’ for which is linked to a valid Store are displayed in the list. This is a mandatory field.

  3. On selecting the Dispense Location, the Store associated to the Dispense Location is displayed as read-only text.

  4. Select the Item Type from the list. The listed values are ‘Drug’ and ‘Material Item’. The value Drug is displayed, by default.

  5. Enter the Item Name in the Item Search String field or select the required Item using [?] function. This is a mandatory field.

  6. Click [Search] to display the list of Items. The Items associated to the selected Location Type are displayed.

  7. Select the check box against the Item to associate the Item to the selected Dispense Location.

  8. Click [Apply] to save the association of Item to the Location and Store created or [Reset] to clear the entered data.

  9. Click [Query] to view the existing association of Item to the Location and Store.
