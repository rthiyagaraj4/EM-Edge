Store for Location

From this function you can Create an association of Locations of Type Clinic or Nursing Unit’ to a Store and Query for the association of Store to a Location.

Creating an association of Locations to Store

From the PH Code Setup > PH Other Admin Setup, select the Store for Location function.

The Store for Location screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Search and select the Dispense Location using [?] function. The listed Dispense Locations of the Type ‘Clinic’ and ‘Nursing Unit’ for which is linked to a valid Store are displayed in the list. This is a mandatory field.

  3. On selecting the Dispense Location, the Store associated to the Dispense Location is displayed as read-only text.

  4. Select the Location Type from the list. The listed values are ‘Clinic’ and ‘Nursing Unit’. The value Nursing Unit is displayed, by default.

  5. Click [Search] to display the list of Locations. The Locations associated to the selected Location Type are displayed.

  6. Select the check box against the Location to associate the Location to the selected Store.

  7. Click [Apply] to save the association of Store to a Location created or [Reset] to clear the entered data.

  8. Click [Query] to search for an existing association of Store to a Location.
