Acknowledgement Rights

From this function you can Provide Acknowledge Rights and identify Turnaround Time (TAT) from pharmacy dispensing to acknowledgement in the nursing station

This function is used to provide Acknowledge Rights and identify Turnaround Time (TAT) from pharmacy dispensing to acknowledgement in the nursing station

Providing Acknowledgement Rights

From the PH Admin Setup > PH Other Admin Setup > select the Acknowledgment Rights function.

The Acknowledgement Rights screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the User from lookup function. This field is mandatory field.

  3. Select the Nursing Unit using lookup function. This field is mandatory field.

  4. By default, General, Narcotics, Controlled check boxes are displayed as selected under Authorized Drugs section. Select or clear the required check box to provide access right.

  5. Click [Apply] to save the Acknowledgement Rights defined or [Reset] to clear the entered data.

  6. Click [Delete] to delete the selected records.

  7. Click [Query] to search for an existing Acknowledge Rights.
