Workstations for Token Series

From this function you can Create an association of Workstation to the Token Series defined for a Dispense Location and Query for the desired association of Workstation to the Token Series.

The Workstations are indicators of the system used by the Pharmacist, which are associated to the Dispense Location for generating the Token Series. The Dispense Locations of type ‘Pharmacy’ are available for selection.

Creating an association of Workstations to Token Series

From the PH Admin Setup > PH Other Admin Setup, select the Workstations for Token Series function.

The Workstation for Token Series screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Dispense Location from the list. The locations listed are of type ‘Pharmacy’ and defined in ‘Dispense Location’ function. This is a mandatory field.

  3. Workstations defined for the selected Dispense Location in the ‘Workstation for Dispense Location’ function are listed. This is a mandatory field.

  4. On selecting the required details, the Token Series defined in the Dispense Location function are displayed.

  5. Select the Applicable check box against the desired Token Series, if the Token Series is applicable for the selected Dispense Location and Workstation Combination.

  6. Click [Apply] to save the association of Workstations for Token Series or [Reset] to clear the entered data.

  7. Click [Query] to search for an existing association of Workstations for Token Series.
