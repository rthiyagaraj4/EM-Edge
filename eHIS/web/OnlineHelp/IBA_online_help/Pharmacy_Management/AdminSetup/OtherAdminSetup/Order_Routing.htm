Order Routing

From this function you can Define the default route for prescriptions based on various factors and Query for the desired Order Routing records of different Location Types.

The Order Routing is based on the Ordering Location or based on the Drug or attribute of the Drug that is prescribed such as Drug Class or Order Type. If Order Routing is defined, the system derives the route based on precedence in the following sequence:

• Drug

• Drug class

• Order Type

• Source

• Special Orders

• Medical Item

Defining an Order Routing

From the PH Admin Setup > PH Other Admin Setup, select the Order Routing function.

The Order Routing screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Order Characteristics, Routing Locations, and Default Dispense Location details in the mandatory (*) fields.

    Routing Locations

    1. Routing Level – Select the Routing Level from the list. The listed values are Source, Order Type, Drug Class, Drug, Special Orders, Medical Item and Home Leave . By default, Drug Class is displayed. This is a mandatory field.

    Based on the Routing Level selected, the values are listed in the respective fields. The following Routing Location details are displayed for the selected Location Type.

    • Routing level

    • Routing

    • Performing Facility

    • Default Dispense Location

      • Dispense Location selected to route Routine Orders

      • Dispense Location selected to Route Stat Orders

      • Dispense Location selected to route Discharge/take home Medication Orders

    Enter the following details to define Order Routing:

    1. Ordering Facility – The logged in facility is displayed and disabled. This is a mandatory field.

    2. Location Type - Select the Location Type from the list. The values listed are Clinic and Nursing Unit. This is a mandatory field.

    3. Location – Select the Location from the list. The listed values correspond to the selected Location Type. This is a mandatory field.

  3. Click [Add] to add the Order Routing details or [Clear] to reset the entered data.

  4. On adding, the Routing details are displayed in the Routing Locations frame.

  5. Select the check box against the desired Routing Location.

  6. 6. Click [Apply] to save the defined Order Routing or [Reset] to clear the entered data.

  7. 7. Click [Query] to search for an existing Order Routing defined.

If Order Routing is not specified for a Location Type, the orders are routed to the dispensing locations specified in ‘PH Parameter for Facility’.
