Create User Desktop

From this function you can Create a new User Desktop with the required summary information that are to be displayed at the Home Page. You can Query to view the User Desktops which are created.

The Create User Desktop function helps you to create user-friendly desktops. This function helps you to customize and save the layout and contents based on User Roles or Responsibilities. You can configure the Summary Information and Information Views of the User, which gets displayed in the Home Page of the User.

Creating User Desktop

From the PH Admin Setup > Pharmacist Homepage Setup, select the Create User Desktop function.

The Create User Desktop screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the User Desktop or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing User Desktop record.
