Drug Common Name

From this function you can Create the common name for drugs.

Drug Common Name allows you to create the common name for drugs. You can search the drug and link with the common name. One common name can be linked with multiple drugs and vice versa.

Creating the Common Name for Drugs

From the PH Code Setup > PH Drug Setup, select the Drug Common Name function.

The Drug Common Name screen gets displayed.

  1. Click [Create] to invoke the Create screen. <Related Field Description>

  2. Click [Apply] to save the Drug Common Name details or [Reset] to clear the entered data.

  3. Click [Query] to invoke the Query  screen.

    To query a Drug, enter the following details:

    1. Drug Name – Enter the Drug Name in Alpha Numeric format of 60 characters maximum length, minimum of 3 characters or select the required Drug Name using [?] function.

    2. Search By – Select the option from the drop-down list.

    3. Common Name – Enter the  Common Name or select using the [?] function.

    4. Sort Order  – Select the Sort Order as Drug Name or Common Name as required.

    5. Execute – Click [Execute].  The Drug Name, Common Name and Enabled status are displayed.

    • To modify the Common Name, click Drug Name hyperlink.

    • Make the required changes and click [Apply].
