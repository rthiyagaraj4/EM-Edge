Order Catalog

The following information is displayed as read-only text:

 

  1. Code – The Code specified for the drug is displayed.

  2. Long Description – The long description of defined drug is displayed.

  3. Order Category – The Order Category is set to ‘Pharmacy’ and disabled.

  4. Activity Type – Select the Activity Type of the Order Catalog. The Activity Type displayed corresponds to the Order Type Selected. Activity Types are defined in the function ‘Activity Type’.

  5. Index – Select the check box to index the defined Order Catalog. If the defined Order Catalog is indexed it has to be associated to a service, that is, the defined Order Catalog is available in the function ‘Index Order Catalog’. If the Order Catalog is not indexed, they are available across the services of the facility.

 

Enter the following details to create an Order Catalog:

 

  1. Short Description - Enter the short description in alphanumeric format of 40 characters maximum length. This is a mandatory field. The short description defined is used during medication ordering.

  2. Order Type – Select the required Order Type from the list. The list is displayed based on the Generic Name selected. The order types and generic name share the same Drug Class. This is a mandatory field. The order types are defined in the function ‘Order Type’.

On entering the required details, enter the details in the tabs provided in ‘Order Catalog’ screen.

 

Characteristics Tab

Timing Tab

Frequency Applicable Tab

Order Reviews Tab

Settings Tab

Instructions Tab
