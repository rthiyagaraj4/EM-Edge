Prescription Remarks for Drug

From this function you can Associate multiple prescription remarks for a particular drug and Query for the desired association of Prescription Remarks to a Drug.

These Remarks are displayed for the particular drug during Medication Ordering.

Creating an association of Prescription Remarks for a Drug

From the PH Code Setup > PH Code Setup, select the Prescription Remarks for Drug function.

The Prescription Remarks for Drug screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Drug Name in alphanumeric format of 60 characters maximum length in the Drug Search Text or select the Drug Name using [?] function. This is a mandatory field.

  3. On selecting the drug, the Form of Drug and Administration Route associated to the drug  is displayed. You can select any other route from the list. The option ‘All’ is selected in route list to display all the prescription remarks for the entire routes associated to the form. The Form of Drug and Administration Route are associated to the drug in the Prescription Remarks function.

  4. Click [Search] to display the prescription remarks.

  5. Select the ‘Default Remark’ check box to select the selected Remarks as default remark for the Prescription remark. Only one Remark can be selected as default.

  6. Select the ‘Select’ check box to associate the Prescription Remarks with the selected Drug.

  7. Click [Apply] to save the association of Prescription Remarks for a Drug or [Reset] to clear the entered data.

  8. Click [Query] to search for an existing association of Prescription Remarks for a Drug.
