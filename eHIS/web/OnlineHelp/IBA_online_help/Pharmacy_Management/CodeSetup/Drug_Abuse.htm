Drug Abuse

From this function you can Define drugs to be restricted to a patient while ordering. You can Query for the restricted drug and Delete a restricted drug if required.

This function allows to restrict placing of specific drugs for a patient and unblock if the restriction is not required.  There are two types of Restriction: Drug Class and Drug Name.  When Drug Class option is selected, user can restrict whole class of PH orders from being ordered for the patient. (General, Narcotics and Controlled).

If user restricts the transaction for the defined class or drug, then the defined class or drug cannot be ordered for a patient.

If user does not restrict the transaction for the defined class or drug, then the defined class or drug can be ordered for a patient after the user has entered the override reason.

Defining Drug Abuse

From the PH Code Setup > PH Code Setup, select the Drug Abuse  function.

The Drug Abuse screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the changes done or [Reset] to clear the entered data.

  4. Click [Query] to search for existing records.

  5. Click [Delete] to delete a queried record.
