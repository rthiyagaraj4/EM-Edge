Specialty Event

From this function you can Identify various conditions or events pertaining to the Mothercare specialty module.

The set of Specialty Events in this function are system defined and new events cannot be created. The Specialty Events of module are Birth Control, Fertility Issues, Menopause, Pregnancy, Reproductive Cancers, STD Prevention, Termination of Pregnancy, Delivery, Preterm Delivery, Post Delivery Mother Treatment and Ectopic Pregnancy. The description of any Specialty Event can be modified. The user is also allowed to associate a Specialty Event to a particular age group.

Identifying Specialty Events

From the CA Code Setup, CA Specialty Code Setup, select Specialty Event function.

The Specialty Event screen gets displayed.

  1. Click [Query] to query for system defined Specialty Events.

  2. Enter the query criteria and click [Execute].

  3. The Specialty Events are displayed. The Code, Description, Age Group, Order Sequence, Definition Type and Enabled Status are displayed.

  4. Click the Code Hyperlink to modify the Description, Age Group, Order Sequence and Enabled Status.

  5. Click [Apply] to save the modifications.
