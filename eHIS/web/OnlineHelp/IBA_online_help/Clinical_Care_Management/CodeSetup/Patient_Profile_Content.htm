Patient Profile Content

From this function you can Define the user descriptions for system descriptions that should appear as hyperlinks in the Patient Chart Summary while clicking the Profile hyperlink. You can also record an Order Serial Number as to what order the descriptions should appear in the Patient Profile screen.

User Descriptions are defined for the System Descriptions – Patient Markers, Current Allergies, Vital Signs, Medications, Current Encounters, Procedure Performed, Patient Allergy Signs and Referral Diagnosis.

Defining Patient Profile Content

From the CA Code Setup, select Patient Profile Content function.

The Patient Profile Content screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the fields. <Related Field Description>

  3. Click [Apply] to save the user defined descriptions or [Reset] to clear the entered data.
