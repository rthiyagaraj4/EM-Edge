1. Row Position – Enter the Row Position in numeric format of two digits. The entered value should not be greater than the Number of Rows specified, or the system prompts a message.

  2. Column Position – Enter the Column Position in numeric format of two digits. The entered value should not be greater than the Number of Columns specified, or the system prompts a message.

  3. Row Span – Enter the Number of Rows for the component to stretch. The maximum number of Row Span is three. If a value more than three is entered, the system prompts a message and the user is not able to proceed with entering data further.

  4. Column Span – Enter the Number of Columns for the component to stretch. The maximum number of Column Span is three. If a value more than three is entered, the system prompts a message and the user is not able to proceed with entering data further.

  5. Content Description – Select the required value from the list. The values displayed in the list are Allergies, High Risk, Pending Orders, Encounters, Patient Care Flow Sheet, Current Medications, Triage Details, Active Problems, Procedure Details, Referral Information, Events occurred in last 24 hrs, Pending Schedules and Care Plan Status. The content selected is displayed in the selected row and column position in the selected row and column span in the Patient Chart.

  6. [Clear] – Select the check box adjacent to the component and click [Clear] to clear the data entered in the fields of the selected component.

Chart Summary Template

From this function you can Create a customized Template that can be used as an alternate to the iSOFT default Chart Summary.

This template is created using the components available as a standard set for the Chart Summary. Template is created using all the components or few of the components. This template is built in the form of a matrix as defined by the number of rows and columns and the chart summary components are housed in one or more cells of this matrix. If Chart Summary Template is not defined, the system displays the standard Chart Summary provided by Enterprise Management by default.

Defining Chart Summary Template

From the CA Admin Setup, select Chart Summary Template function.

The Chart Summary Template screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Define].

  4. Enter the Row and Column details.

  5. Click [Apply] to save Chart Summary Template created or [Reset] to clear the entered data.
