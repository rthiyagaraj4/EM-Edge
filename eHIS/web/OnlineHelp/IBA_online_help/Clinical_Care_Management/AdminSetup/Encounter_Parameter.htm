Encounter Parameter

From this function you can Define the number of days for Backward Visits and Backward Discharges.

Using this function you can establish the relationship between Patient and Practitioner through ‘Assign Relationship’ function in Clinician Access module.

Setting Encounter Parameter

From the CA Admin Setup, select Encounter Parameter function.

The Add Encounter Parameter screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Apply] to save the Encounter Parameter or [Reset] to clear the entered data.
