View Relationships (V)

From this function you can View the relationships created for a patient.

This function displays all relationships that the logged in practitioner has with the patient for the current encounter. A practitioner can further view all other relationships that have been created across encounters.

Viewing Relationships

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Patient Details, select View Relationships (V).

The View Patient Provider Relationships screen gets displayed.

  1. The screen automatically displays all the active relationships for the logged in care provider for the selected encounter of the patient.

  2. View the different relationships by selecting appropriate search criteria.

• Select the Nature of Encounter as Life Time, Encounter and All.

• Select Relationships as Self, Others and All.

• Select Status as Active, Expired and All.

• Select Encounters as Selected Encounter and All Encounters.

  1. Click [Search] to display the Relationship details.
