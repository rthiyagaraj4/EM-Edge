1. Self - Self is selected, by default. Select Self to display all the patients for whom the attending practitioner and logged-in practitioner are same. By default, logged-in practitioner is displayed and the [?] function is disabled.

  2. On behalf of - Select 'On behalf of' to request on behalf of another practitioner. Select the practitioner on behalf of whom the logged in practitioner request for transcription of notes. Selecting Practitioner is mandatory.

  3. View - Select an option from the list. The listed values are All, My Patients and My Related Patients. All is displayed, by default. Select My Patients to display patients for whom the logged-in practitioner is the attending practitioner. Select My Related Patients, to display patients for whom relationship exists with the logged-in practitioner.

  4. Specialty - Select the Specialty using [?] function. By default, the primary specialty of the logged-in practitioner is displayed.

  5. Patient ID - Select the Patient ID using [?] function.

  6. Location - Select the Location from the list. The locations are Clinic and Nursing Unit. Select the required Clinic or Nursing Unit using [?] function based on the selected Location Type. Selecting location is not mandatory if View is selected as 'My Patients' or 'My Related Patients'.

  7. Service - Select the Service using [?] function.

Multi Transcription Request

From this function, you can Record multiple transcription requests placed to the transcriber from the transactions.

This function allows you to request transcriptions (of the attending practitioner and on behalf of other practitioners) to transcribers in a bulk. You can select the required transcriber and notes for transcription.

Multi Transcription Request

From the CA Transactions > Clinician Access > Desktop Menu > Pending Activities > Tasks > select Multi Transcription Request function.

The Multi Transcription Request screen gets displayed.

  1. Enter the search criteria and click [Search] to display requests based on the entered criteria.

  2. The search result displays Patient Details, Attending Practitioner, Admission Date, Transcriber and Notes.

  3. Select the check box against the patient details for which the transcription request is required.

  4. By default, transcriber set in the 'Default Transcriber Setup' function for the Practitioner, Specialty and Patient Class is displayed in the Transcriber field. Select the transcriber using [?] function. The list displays all practitioners created with Practitioner Type 'Transcriber'.

  5. The Default Note Type selected in the 'Note Type Access Rights' function for the Responsibility, Practitioner and Specialty is displayed in the Notes field. Select the Notes from the list. The Notes are displayed based on the selected Transcriber.

  6. Click [Apply] to request for multiple transcriptions.
