View Appointments

From this function you can View appointments scheduled for patients.

This function is used to view all appointments booked against the patient across various care locations. Using this function, the practitioners can view the status of appointments and volumes of bookings performed against care locations for patients.

Viewing Appointments

From the CA Transactions > Clinician Access > Desktop Menu > Schedules, select View Appointments.

The View Appointments screen gets displayed.

  1. Enter the details in mandatory (*) fields. <Related Field Description>

  2. Click [Search] to view appointment details of the patient.

  3. The Patient appointment details are displayed based on the entered. If Patient is not registered, Patient ID field is left blank.
