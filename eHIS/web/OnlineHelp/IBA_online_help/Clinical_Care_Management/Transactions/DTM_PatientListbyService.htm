Enter the following criteria to filter patient listing:

  1. Patient Class – Select a Patient Class from the list. ‘All’ is displayed, by default. Other values are Daycare, Emergency, Inpatient and Outpatient.

  2. Medical Service - Select a Medical Service using [?] function.

  3. View List – Select ‘Self Related’ or ‘All’ from the list. ‘Self Related’ is displayed, by default.

  4. Attending Practitioner - Enter the Practitioner name or click [?] function to search for the Attending practitioner. This field is enabled only if the 'View List' field is selected as All. If View List is selected as Self Related, the logged in user is displayed and disabled. All Practitioners are displayed for selection on clicking the [?] function.

  5. Specialty - Enter the Specialty or click [?] function to select the specialty. The Specialties associated to the Practitioner is displayed in the search screen for selection.

  6. Visit Type - Select the Visit Type from the list. This field is enabled only if the Patient Class 'Outpatient'. The Visit Types listed for selection are defined in the Visit Type function of Outpatient Management module.

  7. Unassigned - This check box is enabled on selecting Patient Class as All, Emergency or Outpatient. Select the check box to display the list of patients to whom a care provider is not assigned.

  8. Group By Patient Class –The check box is selected, by default. This facilitates the display of patient list in the order of Patient Class. The check box is cleared to display the patient list without Patient Class grouping.

  9. Group By Medical Service - The check box is selected, by default. This facilitates the display of patient list in the order of Medical Service. The check box is cleared to display patient list without medical service grouping.

  10. Patient ID – Select the Patient ID using [?] function.

  1. National ID (Alternate ID defined in MP Parameter) - Enter the National ID in alphanumeric format as mentioned in MP Parameter of maximum length.

  2. Citizen ID (Prompt entered in MP Parameter) - Enter the patient Citizen ID in alphanumeric format as mentioned in MP Parameter of maximum length.

  3. Past Visits - This check box is enabled only on selecting Patient Class as All, Emergency or Outpatient. Select the check box to list patients with any previous visits.

  4. Period - The field is enabled only when 'Past Visits' check box is selected. The ‘Date' value is displayed as defined in the parameter. Enter the date in dd/mm/yyyy format within the allowable period as defined in the parameter. This is to filter patient listing with encounters within the specified period range.

  5. Discharged - This check box is enabled on Patient Class as All, Daycare or Inpatient. Select the check box to include discharged patients in the list.

  6. Period - The field is enabled only when 'Discharged' check box is selected. The ‘Date' value is defaulted as defined in the parameter. Enter the date in dd/mm/yyyy format within the allowable period as defined in the parameter. This is to filter patient listing with encounters within the specified period range.

  7. Admitted Period – Enter the date range to search for patients admitted within the entered date range. Enter the date in dd/mm/yyyy format within the allowable period as defined in the parameter. This is used to filter patient listing based on admission dates. This field is displayed only for Daycare or Inpatients.

  8. Queue Status – This check box is used to filter patients based on various queue statuses applicable to an Emergency or Outpatient encounter. The various statuses listed are ‘Checked In’, ‘Arrived at NS’, ‘VS Recorded’, ‘Cons in Progress’, ‘Treat/Invest’, ‘Return from Invest’ and ‘Checked Out’.

Patient List by Service

From this function you can View patient list based on the medical service identified against the patient’s encounter.

This function lists patients based on the Medical Service assigned to the patient encounter. On searching, by default all patients are displayed in the descending chronological order of Visit Date or Admission Date within each Medical Service for the episode Type.

Viewing Patient List by Service

From the CA Transactions > Clinician Access > Desktop Menu > Patient List, select By Service.

The Patient List by Service screen gets displayed.

  1. Enter the search criteria.

  2. Click [More Criteria] to enter additional search details.

  3. Click [Less Criteria] to enter minimum search details.

  4. Click [Search] to display patients based on the entered criteria.

  5. A ‘*’ symbol is displayed against the patient name, if relationship between the practitioner and patient does not exist. Click Patient Name hyperlink to display the Assign Relationship screen and assign relationship.

  6. For patients with relationship assigned, Patient Chart is displayed.
