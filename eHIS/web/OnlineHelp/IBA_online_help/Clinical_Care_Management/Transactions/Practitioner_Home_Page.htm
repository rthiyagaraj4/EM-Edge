Practitioner Home Page

From the Practitioner Homepage you can Access patient charts and perform various tasks across patients. You can also View activities that require review and Perform tasks without having to open any specific patient chart.

The Homepage gives a bird’s eye view of all the activities that is pending to be performed and also a listing of the patients that are under the practitioner’s care. The home page consists of various components that constitute the Clinician’s task list. The Practitioner Home Page is the first screen that is displayed when a practitioner logs into Clinicians Access if the homepage has been assigned to the logged in practitioner. The components that are displayed in the homepage are based on the personalization setting against the logged in practitioner. The homepage is assigned to a practitioner using the function ’Personalized Home Page’ function in the admin set up of Clinician Access. If the homepage is assigned for the logged in practitioner, then the ‘Patient List’ function is displayed by default.

Accessing Practitioner Home Page

From the Clinician Access, select the Clinician Access function.

The Practitioner Home Page screen gets displayed.

The Practitioner Home Page screen contains the following:

Name of the logged in practitioner

Icons and functions

Practitioner Task List

The Components of the Practitioner’s Home Page is displayed based on the Settings made in the Personalized Home Page function in CA Admin Setup.

If any one of the Components is selected for the logged in Practitioner in the Personalized Home Page function then those Components are displayed for the Practitioner. If none of the Components are selected for the logged in Practitioner then the Home Page displays components associated to the Practitioner Type.
