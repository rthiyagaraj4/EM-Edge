Enter the following details to Construct CA Menu:

 

  1. Desktop/Patient Chart – Select Desktop or Patient Chart radio button to construct CA Menu specific to Clinician Desktop or Patient Chart.

  2. Desktop – Select ‘Desktop’ radio button to construct a ‘Menu’ specific to the ‘Clinician Desktop’ screen. This field is mandatory.

  3. Patient Chart – Select ‘Patient Chart’ radio button to construct a ‘Menu’ specific to the ‘Patient Chart’ screen. This field is mandatory.

  1. Menu ID – Select Menu ID from the list. The list displays ‘Root Menus’ defined in ‘Define Menu’ function. The menu ID listed corresponds to the option selected as either ‘Desktop’ or ‘Patient Chart’. This field is mandatory.

  2. Menu Options – All the system defined Menus and user defined Sub Menus defined in ‘Define Menu’ function is listed. The menus defined in ‘Define Menu’ with ‘Root Menu’ check box cleared are listed as sub menus in ‘Menu Options’.

  3. - During menu construction, if a submenu/Function is required to be moved up by one position in the structure, the required submenu/function is selected by clicking the same and clicking button to achieve the upward movement.

  4. - During menu construction, if a Submenu/Function are required to be moved down by one position in the structure, the required submenu/function is selected by clicking the same and clicking button to achieve the downward movement.

  5. - This button is used to Delete a Submenu/Function from the structure. The required Submenu/Function is selected by clicking the same and clicking button to delete the component. Note that to delete a ‘Menu’, the functions under the menu should be deleted first.

  6. - During Menu Construction, if a Submenu/Function is required to be placed under another Submenu, the required Submenu/Function is selected by clicking the same and clicking button to achieve the desired movement.

  7. - During Menu Construction, if a Submenu/Function is required to be placed directly under the Parent Menu, the required Submenu/Function is selected by clicking the same and clicking button to achieve the desired movement.

  1. Menu Options – During Menu Construction, any of the displayed Menu Options are selected by clicking the required Menu Option and brought into the Menu Structure space by a single mouse click anywhere in Menu Structure space. Any number of Menu Options can be included.

  2. Function Options – The list of functions displayed correspond to the Menu Option selected as either ‘Desktop’ or ‘Patient Chart’. During menu construction, any of the displayed functions are selected by clicking the same and brought into the Menu Structure space by a single mouse click anywhere in Menu Structure space. Any number of functions can be included.

Click the [Default Menu] to structure the default Menu (Desktop and Patient Chart) of Enterprise Management under the required Menu ID.

The function provides a facility to preview the constructed menu by clicking [Preview].
