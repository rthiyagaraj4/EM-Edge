Enter the following details:

 

  1. Referral Type – The Referral Type is displayed with values ‘Login facility’, ‘Enterprise’ and ‘External’. This field is mandatory.

  2. Healthcare Setting Type – Select the required Health Care Setting Type from the list. All the Healthcare Setting Type records created in Systems Manager module are displayed. If Referral Type is Login Facility, the system displays the Healthcare Setting Type of the logged in facility and disables the field. This field is mandatory.

  3. Referred Source – Select the source from which patient is referred. If Referral Type is login facility, the logged in facility name is defaulted. This field is mandatory.

  4. Specialty – Select the required specialty from the list. This field is mandatory.

  5. Location - Select the Location Type and Location from the list. The valid values for location type are ‘Clinic’, ‘Nursing Unit’, ‘Daycare’ and ‘Procedure Unit’. The list next to ‘Location’ contains values that correspond to the selected Location Type.

  6. Service – Select the required Service from the list. All services for the selected specialty and location are displayed.

  7. Referred For – Select the purpose of referral from the list. This field is enabled and is mandatory if Location Type is selected as Nursing Unit. The values listed are ‘Booking’ and ‘Admit’.

  8. Practitioner – Enter the Practitioner name in alphanumeric format of 30 characters maximum length. If Referral Type selected is External, this field allows only entry of a practitioner name. If Referral Type is of type Login Facility or Enterprise, the system allows selecting the required practitioner using [?] function. The practitioners listed are based on the specialty and location selected.

  9. Preferred Date - Enter the Preferred Date for visit or admission in dd/mm/yyyy format or select the required date using the Calendar icon.

  10. Reason for Referral - Select the Reason for Referring from the list. All the reasons defined with Referral check box selected in 'Other Master setup (Reason)' of Application Master module are displayed.

  11. Priority - Select the Priority of referral from the list. The values are ‘Elective’, ‘Urgent’ and ‘Emergency’

  12. Referral Notes - Click on Referral Notes to enter notes about the patient being referred.
