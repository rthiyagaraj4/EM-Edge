Enter the following details:

 

  1. Chart Type – Select the Chart Type as Growth Chart from the list. This field is mandatory. On selecting the Chart Type, a field Chart Group is displayed.

  2. Chart Group - Select the Chart Group as 0-36 Months or 2-20 Years, which is system, defined from the list. This field is mandatory.

  3. Graph Code – Enter a code for the graph in alphanumeric format of four characters maximum length. This field is mandatory.

  4. Graph Description – Enter the description of the graph alphanumeric format of 30 characters maximum length. This field is mandatory.

  5. X-Axis – Select a value for the X-axis from the list containing the values Key Value, Chest Circum, Weight, Length, Head Circum. This field is mandatory.

  6. Y-Axis - Select a value for the Y-axis from the list containing the values Chest Circum, Weight, Length, Head Circum. This field is mandatory.

  7. Enabled – Select the check box to enable the record.
