Routine Visit

Select Record Routine Visit from the Routine Visit task and enter the routine visit details.

 

Record Routine Visit

Enter the following details:

  1. LMP – The LMP as recorded during the First Visit is displayed. You can modify the LMP based on the parameter set in Specialty Parameter function.

  2. Gestation Age – The Gestation age is automatically displayed based on the LMP. The gestation age is displayed based on the MP Parameter setup. Gestational age is usually considered to be the age of an embryo or fetus (or newborn infant) from the first day of the woman's last menstrual period (LMP).

  3. EDD – The EDD is automatically calculated and displayed based on the LMP.

  4. Corrected EDD – Enter the corrected EDD as required. You can enter the corrected EDD based on the parameter set in Specialty Parameter function.

Record Symptoms

  1. Nausea/Vomiting – Select Present from the list box if Nausea or Vomiting is a symptom of Pregnancy for the patient. Select Absent is Nausea/Vomiting is not a symptom of pregnancy for the patient.

  2. Vagina Bleeding, Constipation, Abdominal Pain, Heart Burn, Back Ache, Swollen Feet – Select the check boxes against the required if it is a symptom of pregnancy.

  3. Vaginal Discharge – Select Present from the list box if Vaginal Discharge exists for the patient. Select Absent if not.

  4. Other (Specify) – Enter any other symptoms of pregnancy other than the ones mentioned.

Record Signs

  1. Temperature – Enter the temperature of the patient in Degrees Celsius.

  2. Pulse – Enter the pulse rate of the patient per minute.

  3. Systolic – Enter the systolic blood pressure in mm/Hg.

  4. Diastolic – Enter the diastolic blood pressure in mm/Hg.

  5. Weight – Enter the weight in kilograms.

  6. Height – Enter the height in centimeters.

  7. Oedema – Select if Oedema (Oedema means too much fluid, mainly water, accumulated in the body) is absent or present from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  8. Type – If Oedema is selected as Present, select the Type of Oedema as Nonpitting or Pitting.

  9. Varicose Veins – Select Varicose Veins (twisted, widened veins caused by swollen or enlarged blood vessels) details for the patient from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  10. Pallor – Select if Pallor (Pallor is a reduced amount of oxyhemoglobin in skin or mucous membrane, a pale color which can be caused by illness, emotional shock or stress, avoiding excessive exposure to sunlight, anaemia or genetics) details for the patient from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

Abdomen Examination

  1. Uterus Palpable – Select the check box if uterus is palpable (part of a physical examination in which an object is felt usually with the hands of a healthcare practitioner to determine its size, shape, firmness, or location).

  2. Uterus Fundal Height – Enter the Uterus Fundal Height (measure of the size of the uterus used to assess fetal growth and development. It is measured from the top of the pubic bone to the top of the uterus in centimeters) in weeks.

  3. Presentation – Select Presentation details of the examination from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  4. Fetal Heart Activity – Select the Foetal Heart Activity from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

Fetal Well Being

  1. Fetal Movements – Select Fetal Movement details from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  2. Fetal Kick Counts – Enter the number of fetal kicks per day.

  3. Immunization – Select Immunization details from the list box. The values are defined in the Specialty Master Code function of CA Specialty Code Setup.

  4. Date – Select the date of immunization from the calendar function.

  5. Click the hyperlink to record more details. The + hyperlink displays as hyperlink. Click + hyperlink to record more immunization details and click the – hyperlink to record less immunization details.

On entering the required details, click [Record] to save details of Routine Visit recorded for the event Pregnancy. The date and time of recording the Routine Visit details is displayed on the left side menu of the screen under Routine Visit task. Click the Date and Time hyperlink and the details recorded for Routine for the event Pregnancy gets displayed on the right side of the screen.

Click Record Routine Visit from the Routine Visit task to record details on a subsequent routine visit.
