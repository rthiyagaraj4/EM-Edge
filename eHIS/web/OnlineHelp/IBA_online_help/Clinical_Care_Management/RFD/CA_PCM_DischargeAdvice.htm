Enter the following details:

 

Discharge Tab

  1. Authorizing Practitioner – Enter the authorizing practitioner in the alphanumeric format as specified or select the practitioner using [?] function. The practitioners listed in [?] function are filtered based on the Specialty of the Nursing Unit. This field is mandatory. The logged in practitioner is displayed as the Authorizing Practitioner, by default.

  2. Expected Discharge Date/Time – Enter the date in the ‘dd/mm/yyyy hh:mm’ format. The date and time entered has to be greater than or equal to system date and time. This is a mandatory field. On load of the screen, the expected discharge date and time is displayed. This displayed value is specified in 'Admit Patient' function or 'Record Expected Discharge' function.

  3. Discharge Type - Select a valid Discharge Type from the list. This is a mandatory field. The list displays values defined in the function 'Discharge Type'. Discharge Types can be set for two categories – Ones with ‘Deceased’ indicator selected (used for patients pronounced dead before/during discharge) and others with the same indicator flag ‘Deceased’ cleared.

  4. Death Date/Time – Enter or select the date from the Calendar function in date format of dd/mm/yyyy hh:mm. This field is displayed if a value with Type Deceased is selected as the Discharge Type. This field is mandatory.

  5. Post-Mortem Requested - Select the check box, if post-mortem is requested for the deceased patient. The field is enabled only if the selected Discharge Type has ‘Deceased’ check box selected during its creation.

  6. Remarks – Enter the remarks if any in the alphanumeric format of 100 characters maximum length.

 

MLC Details

  1. Medico Legal Case - Select the check box, if the encounter is a medico legal case.

  2. Police Report No. – Enter the police report number in alphanumeric format of 30 characters maximum length. This field is enabled only if the medico legal case check box is selected.

  3. Police Station – Enter the police station, where the case has to be registered in alphanumeric format of 30 characters maximum length. This field is enabled only if the medico legal case check box is selected.

  4. Police ID – Enter the Police ID in alphanumeric format of 30 characters maximum length. This field is enabled only if the medico legal case check box is selected.

 

Informed Details

  1. Informed To – Select the valid contact to inform about the patient’s discharge from the list. The listed values are Next to Kin, First to Notify and Others. On selection of either ‘Next of Kin’ or ‘First to Notify’, the button ‘Patient Details’ gets enabled, allowing you to view these details, if they had been entered during the registration process for the selected inpatient. On selecting Others from the list, the Patient Details button is disabled.

  2. Name – Enter the name of the person informed in alphanumeric format of 200 characters maximum length.

  3. Date Time – Enter the informed date and time in the format ‘dd/mm/yyyy hh:mm’, or select the date and time from the Calendar function.

 

Absconding Details

  1. Absconding – Select the check box, if the patient has absconded. This is usually recorded for patients absconded from the facility or those patients that have gone on leave and not yet returned.

  2. From Date/Time – This field is enabled on selecting the Absconding check box. Enter the date and time in date format of dd/mm/yyyy hh:mm, from when the patient is absconding. The entered date and time has to be greater than current system date/time and not be less than admission date/time. This field is mandatory.

  3. Informed Others - This field is enabled on selecting the Absconding check box. Enter the name of the person to whom absconding of the inpatient is reported. This field is mandatory.

 

Other Details Tab

Click Other Details tab to enter the details in this tab.

The Other Details Tab is disabled only for Deceased Patients. The value with indicator ‘deceased’ has to be selected from among the available values within ‘Discharge Type’ list in case of an inpatient that is pronounced dead.

 

Take Home Medicine

  1. New Medicine - Select the check box, if any new medication is to be supplied to the patient on discharge.

  2. Stock Medicine - Select the check box if any stock medication currently being administered to the patient is to be handed over on discharge.

 

Charges Incurred On Discharge Day

These fields are enabled only if the billing module is interfaced with the IP module. To specify whether any charges were incurred on the Discharge day relating to the following services:

Select the check box against Lab, X ray, Scan, Ultrasound, MRI, RT, Physio, ECG, Diet and Pharmacy, if any of these are incurred in the Discharge Day. The check boxes are cleared.
