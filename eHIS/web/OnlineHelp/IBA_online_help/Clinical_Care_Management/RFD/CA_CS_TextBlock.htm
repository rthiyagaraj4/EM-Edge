Enter the following details to create a new Text Block record:

 

  1. ID - Enter the text block ID in alphanumeric format of two characters maximum length to identify a Text Block uniquely. The text block ID has to be unique and cannot be changed after creation. This field is mandatory.

  2. Text Block - Enter a detailed description for the text block in alphanumeric format of 4000 characters maximum length. This field is mandatory.

  3. Service – Enter the required Service in alphanumeric format or select the service using [?] function.

  4. Practitioner – Enter the required Practitioner name in alphanumeric format or select a Practitioner using [?] function.

  5. Note Type – Click [?] function to select a Note Type from the list. All the Note Types created in Note Type function with status Enabled gets listed.

  6. Section – Click [?] function to select a Section from the list. All the Sections created in Section function with status Enabled get listed.

  7. Applicable to Procedures – Select the check box to indicate that the Canned Text is applicable for procedure recording.
