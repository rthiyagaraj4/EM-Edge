Enter the following details:

 

  1. Recipient ID - Enter the ID of the Registered Email Recipient.

  2. Recipient Name - Enter the name of the Registered Email Recipient.

  3. Email ID - Enter a valid Email ID.

  4. Authorize - Select the check box to identify the Recipient as an authorized user and password is generated for the Registered Email Recipient.

  5. Password - Password is generated only for authorised users. The password is displayed on selecting the Authorize check box. Click View hyperlink to view the Effective From Date and the Password in a screen. The effective from date is the date on which the password is generated. Click [OK] to exit from the screen.

  6. Enabled - Select the check box to enable the Registered Email Recipient
