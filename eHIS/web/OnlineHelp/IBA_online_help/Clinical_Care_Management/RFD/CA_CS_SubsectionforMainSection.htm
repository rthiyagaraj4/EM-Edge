Select the following details to associate Subsections to a Main Section:

 

  1. Main Section – Click [?] function to select the required main section. All the sections created of type Main Section in the Section function is listed. This field is mandatory.

On selecting the required section, click [Search].

 

Select Sub Section

  1. Section Starts With – Enter the required section starting description in alphanumeric format of 20 characters and click [Search] to search for the required section. By default, the entire sub sections are listed based on the entered text.

  2. Sub Section Description – The description of the SubSections are displayed.

  3. Select - Select the check box to select the sub sections to be associated to the main section. The selected sections are displayed under the ‘Selected Sub Sections’.

 

Selected Sub Sections

  1. SubSection Description – The description of the selected sub section is displayed.

  2. Order Sequence Number – Enter the order sequence number for the Section.

  3. Select – The check box is selected, by default. Clear the check box to exclude a Subsection from the main section.
