Enter the following details:

 

  1. Application – Select the External Interfaced Application from the list. This field is mandatory.

  2. Order Category – Select the Order Category from the list, which is system defined for the selected external application. This field is mandatory.

  3. Order Type – Select the Order Type from the list of values, based on the Order Category. The Order Type list comes from the Order Entry and Result Reporting module.

  4. Order Catalog – ‘All’ is selected, by default. Select the required Order Catalog from [?] function.

  5. Facility – ‘All’ is selected, by default. Click [?] function to search for the required facility.

  6. Enabled – Select the check box to enable the record in Orders function.
