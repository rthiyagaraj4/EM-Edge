This is a common function, which is used to invoke the menu from the Clinician Desktop or from a Patient Chart by clicking icon on the top left corner of the screen. Once the menu is invoked, the icon displays which is used to hide the menu. The content of the menu is personalized to suit practitioner preferences. If personalized menu is not assigned, the system displays the default menu.

The system provides an option to collapse and expand the menu. The menu is collapsed using icon and the same is expanded using  icon. The menu expands to show all the functions in the menu, by default.

The practitioner home page is invoked from any function within the Clinician Access module by clicking on the Home Page icon  on the top left corner of the screen.

This function is used to select a patient from the patient search option without accessing the patient list option. The patients are retrieved by specifying the required query criteria in the ‘Patient Search’ function. This function is generally used when a patient’s history has to be accessed. This does not require an encounter of the patient to be selected to view the history. The complete history of the patient is accessed if the logged in practitioner has a relationship existing with the patient. The practitioner can alternately set up the relationship if he has the access to assign a relationship to himself. The search patient screen is invoked by clicking ‘Patient without Encounter’ icon    on the top left corner of the operational screen. This Icon is not displayed on selecting specific patients from the Desktop Menu.

This function enables the Practitioner to calculate essential indices of the patient, which may play an important role in the assessment of the condition. Calculators like Cerebral Perfusion Pressure, Mean Blood Pressure, and Cardiac Index can be calculated, which will help the physician in decision-making. Apart from this the Clinical Calculator also enables the Practitioner to calculate the Body Mass Index of the patient, which will help him in deciding the drug dosage. The Practitioner can also convert the measured temperature of the patient from Centigrade to Fahrenheit and vice versa. The Clinical Calculator is an effective tool in the Clinician homepage. The Clinical Calculator function is invoked by clicking the Clinical Calculator icon  on the left corner of the homepage.

Clinical Calculator – Select the Clinical Calculator from the list. The values are

• BMI Calculator

• Red Cell Volume in Adults

• Cerebral Perfusion Pressure

• Creatinine Clearance

• Glomerular Filtration Rate

• Mean Blood Pressure

• Cardiac Index

• Centigrade to Fahrenheit

• Fahrenheit to Centigrade

• Kilogram to Pound

• Pound to Kilogram

Enter the required values in the fields and click [Calculate] to calculate the values.

Click Customise Icons to display the Customise Icons screen. An Icon and the description of the function is displayed on the screen. Select the check box against the required functions. A maximum of 18 functions can be selected. Click [Update] to display the selected Icons on the tool bar. Click the required Icon for easy access to the function.

The Customisation of the Icons are maintained uniquely to the User.

The Switch icon on the top right corner of the screen allows the logged-in user to work on multiple patient charts on the clinician desktop. This function explicitly requires minimum of one chart of the patient to be open. This function enables the Practitioner to toggle between the Active Function of the Clinician Desktop and the Patient Charts.

On clicking Switch icon without selecting a patient chart displays a screen with appropriate message ‘No Patient Chart Opened currently to Switch’.

On clicking the Switch icon the control is automatically send to the Active Function in the Clinician Desktop if the current context of operations is only with one patient chart. When multiple Patient Charts are selected and the user needs to select or switch either to the Clinician Desktop or to any other Patient Chart, Switch Option screen is displayed on clicking the Switch icon showing patient details of patients with open patient chart and Clinician Desktop to select the required option.

Click Close on the top right corner of the screen to exit from the Practitioner’s Home Page and display the Clinician Access module main menu. The system alerts the user on attempting to close the clinician desktop with active patient charts open. The message ‘One or more Patient Chart Open – Please Close before exiting’ is displayed.
