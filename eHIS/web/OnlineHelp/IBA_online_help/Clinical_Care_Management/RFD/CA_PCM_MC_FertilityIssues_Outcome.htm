Outcome

  1. Click Outcome from the menu to record the outcome of the Fertility Issues event.

  2. The mothercare pending tasks to be recorded for the event Fertility Issues is displayed if any. Click [Record] to continue recording the outcome or click [Cancel] to perform the pending tasks for the event. On clicking [Record], the screen to record outcome gets displayed.

  3. The Outcome list box displays Ectopic Pregnancy as the outcome for the event Fertility Issues. Select Ectopic Pregnancy as Outcome and message is displayed indicating the current cycle of recording Fertility Issues event for the patient will be closed and new event will be created for recording Ectopic Pregnancy details for the patient. Click [OK] to close recording Fertility Issues event and the current cycle ends displaying tasks for recording details for the event Ectopic Pregnancy. Click [Cancel] to cancel closing the event and cycle.
