Check In to Recovery Room

Check In to Recovery Room function is used to record checking in details to the Recovery Room.

The following details are displayed:

  1. Anesthesia Type – The Type of Anesthesia is displayed as recorded.

  2. Anesthetist – The Anesthetist is displayed as recorded.

  3. Theatre – The Theatre is displayed.

  4. Click Pre-operative clinical comments to view Pre-Operative details recorded for the patient.

  5. Click Recovery Notes hyperlink to record Recovery Notes. Select the required Note Type from the Note Type screen that is displayed. The Note Types that was associated for Recovery Notes in the ‘Note Type for Specialties’ function is displayed for selection.

  6. Click Record Chart hyperlink to record chart details for the patient.

 

Operations/Procedures

  1. The Operation Code, Description, Side Applicable and Remarks are displayed. Click the Pre-operative comments hyperlink to view the remarks recorded for the Operation/Procedure.

 

Check In to Recovery Room

  1. Recovery Start Time – Enter the start date and time of recovery of the patient. This is a mandatory field. The current system date and time is displayed, by default.

  2. Recovery End Time – Enter the end date and time of recovery of the patient.

 

Untoward Event Tab

  1. Unusual Occurrence – Select the Unusual Occurrence using [?] function. The values are defined in the Unusual Occurrence function. This is a mandatory field.

  2. Click Action Remarks hyperlink to enter action remarks for the unusual occurrence.

  3. Click General Remarks to enter remarks in general for the unusual occurrence.

  4. Click [Select] to add the details to the screen. Select the Untoward Event hyperlink and click [Cancel] to cancel recording Untoward Events.

  5. Click [Apply] to save checking in to Recovery Room.
