O.R List

From this function you can View a list of the confirmed Booked cases across all OTs.

The O.R List function is used at the beginning of each day to view a list of the confirmed Booked cases across all OTs.

Viewing O.R List

From the Bookings, select the O.R function.

The O.R screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run] to get the Report Options screen or [Reset] to clear the entered details.

  3. Select the required report option and click [OK] to view the report, or [Cancel] to abort the process.
