Patient Surgery History

This function allows you to view the Surgery History of the Patient.

The following details are displayed.

  1. Performing Facility - The Facility selected in the External Facility Booking dialog is displayed by default. You cannot edit the data.

  2. Operation - Select the Operation using the [?] function.

  3. From/To Date - Select the date range of the Surgery.

  4. Click [Search] to display the Surgery Details based on the entered criteria. The details displayed are Operation Date, Operation, Surgeon, Status, Details and Notes.

  5. Click Operation Date hyperlink to view the following options:

Patient Demographics

Patient History

View Booking Details

Anaesthesia History

  1. Select the Details hyperlink to display the Surgery Details screen.  

  2. Click Notes to view the details of the notes in the Notes screen.
