Display Mapping

From this function you can Define the display status and color code to be display in the display mapping screen and View the defined display status in the display mapping screen.

The Display Mapping function is used to define display status and color code. You can preview and view the statuses in the dashboard.

Display Mapping

From the Bookings > OT External Display, select the Display Mapping function.

The Display Mapping screen gets displayed.

  1. Click [Add] to display the blank row.

  2. Select OT slate status from the drop-down list. You can select more than one OT slate status. The selected OT slate statuses are displayed.

  3. Select the color using color picker for display status. The selected color will be displayed next to the display status.

  4. Click [Save] to save the details.

  5. Click [Preview] to display preview of the dashboard with display statuses and OT Slate Status.

  6. Click [Display Dashboard] to display the dashboard. This button is enabled only if the dashboard is closed.

  7. Click [Close Dashboard] to close the dashboard display. This button will be enabled only if the dashboard is open.

  8. Click to delete the record.

If the OT slate status has already been selected for a particular OT room then it will be ignored from the current selection and the remaining values selected are displayed in the drop-down list, the system displays error message ‘Duplicate Status Cannot be Selected’.
