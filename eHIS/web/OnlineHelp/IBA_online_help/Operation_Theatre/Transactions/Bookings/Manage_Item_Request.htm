Manage Item Request

From this function you can Query all the booking details. You can Add, Cancel and Print the Item Requests.

Bookings function facilitates viewing all the booking details. The booking details within a specific theatre and with a specific status for a particular period can be viewed by specifying the necessary criteria. The query result provides information on the booking details in a specific Operating Theatre or for all Operating Theatres.

Querying, Adding Cancelling and Printing ItemRequest Details

From OT Bookings, select the Manage Item Request function.

The Manage Item Request screen gets displayed.

  1. Enter the details in the mandatory (*) fields to query the booking details. <Related Field Description>

  2. Click [Search] to view the booking details or [Clear] to clear the records displayed in the result area.

  3. Click Patient hyperlink to view 'Patient Demographics', Print Appointment Slip,  'View Booking Details ' and Manage Item Request.

    Click Print Appointment Slip to invoke the Print Appointment Slip dialog.

    Enter the following details:

    1. Report Name – The name of the report is displayed.

    2. Report option – Select the Report Option from the list. The listed values are ‘View’, ‘Print’ and ‘File’.

    3. If the value ‘View’ is selected, the report is viewed in a PDF/Excel format.

    4. If the value ‘Print’ is selected, the report is printed.

    5. If the value ‘File’ is selected, the report is saved in PDF/Excel format in the system.

     

    The following details are displayed:

    1. Surgeon – The Surgeon assigned for the patient gets displayed.

    2. Specialty - The Specialty assigned for the patient gets displayed. You cannot edit the data

    3. Theatre – The Theatre assigned for the patient gets displayed.

    4. Operation Description – The Operation Description entered in the Manage Item Request – Bookings screen gets displayed.

    5. Anesthetist – The Anesthetist assign for operation gets displayed.

    6. Operation Type – The operation type gets displayed.

    7. Status – The booking status gets displayed.

    8. Source – The source name gets displayed. You cannot edit the data.

    Click [Add] to introduce a new blank row with following details in the Item Items section:

    1. Item Description – Select the description of the Item using [?] function.

    2. Qty – Enter the Item quantity required for operation in the numeric format of 5 character maximum limit.

    3. Requested Date – Select the date using the calendar function. The current date is displayed by default.

    4. Cancel – Select the values ‘Yes’ or ‘No’ from the drop-down list. If the value ‘Yes’ is selected, the Item request is removed. If the value ‘Yes’ is selected, the Item request is retained.

    5. Cancelled Remarks – Enter the remarks for the cancellation in the alpha numeric character of 150 characters limit.

    6. Item Spec Remarks - Enter the remarks for the Item specification in the alpha numeric character of 150 characters limit.

    You cannot add or amend an item request detail if the patient is checked out from OT or Recovery Room and the charge state is over in the Billing module.

  4. Click [Reset] to clear the entered data.
