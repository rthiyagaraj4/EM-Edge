Cancel/Transfer Booking

From this function you can Cancel/Transfer a booking.

The Cancel/Transfer option enables you to cancel/transfer a booked operation. For cancelling an operation, an appropriate reason has to be entered. After the cancellation/transfer, the waitlisted patients can be brought into the vacant slots.  

Transferring/Cancelling a Booking

From the Bookings, select the Cancel/Transfer Booking function.

The Cancel/Transfer Booking screen gets displayed.

  1. Click [Search] to display appointments for the default appointment date or enter the query criteria to search for a specific appointment.

    1. Patient ID - Select the required Patient ID using [?] function.

    2. Name - The name of the patient is displayed based on the selected Patient ID.

    3. Theatre Type - Select the Theatre Type using [?] function.

    4. Theatre - Select the Theatre using [?] function. The operating rooms are created in the Operating Room function.

    5. Specialty - Select the Specialty using [?] function.

    6. Surgeon - Select the Surgeon using [?] function.

    7. Appointment Date [From-To] - The current date is displayed as the appointment From date. Select the date till when the booking records have to be displayed.

    8. Appt Ref. No. - Enter the Appointment Reference Number to display the appointment.

  2. On searching, the appointment details are displayed. The displayed details are Appointment No., Appointment Date, Patient ID, Name, Theatre, From Time, To Time of the slot, Specialty and Surgeon.

  3. Click the Appointment No. hyperlink to Cancel Booking or Transfer Booking.
