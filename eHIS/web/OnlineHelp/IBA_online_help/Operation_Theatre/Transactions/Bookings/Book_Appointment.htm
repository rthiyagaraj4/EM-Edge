Book Appointment

This function is used to create a booking for an Operation for a specific date. The screen is displayed on selecting a Time Slot from the Booking function.

 

Enter the following details:

  1. Booking Date, Start Time, Operating Room, Specialty and Surgeon are displayed on the screen.

  2. Booking Date – The date selected for Booking is displayed.

  3. Start Time – The Start time slot selected for Booking from the Day Schedule Tab is displayed.

  4. Surgery Type – Select the Surgery Type using the [?] function. The Surgery Types are created in the Surgery Type function.

  5. Performing Location – The Facility selected in the External Facility Booking dialog is displayed by default. You cannot edit the data.

  6. Patient Class – Select the Patient Class from the list. The listed values are Daycare, Emergency, Inpatient, Outpatient and External.

  7. Patient ID – Search and Select the required Patient to create a booking.

  8. Gender – The Gender of the patient is displayed.

  9. Encounter ID – Encounter ID is displayed by default only if the selected patient has a single encounter ID. Select the Encounter ID of the patient using the [?] function if there are multiple encounters.

  10. Date of Birth – The date of birth of the patient is displayed.

  11. Operating Room – The Operating Room is displayed.

  12. Specialty – The Specialty for the Surgeon is displayed.

  13. Patient Position – Select the Position of the Patient using the [?] function.

  14. Surgeon – The Surgeon for the Operation is displayed.

  15. Assistant Surgeon – Select the Assistant Surgeon using the [?] function.

  16. Anaesthesia Review – Select the check box to perform the pre-anaesthesia, intra-anaesthesia and post-anaesthesia evaluations for a particular procedure in Booking Verification function. If the check box is cleared, the system does not allow  pre-anaesthesia, intra-anaesthesia and post-anaesthesia evaluations in Booking Verification function. This check box is selected, by default.

  17. Anesthetist – Select the Anesthetist using the [?] function.

  18. Source Type – The Source Type is displayed based on the selected Patient Class and Encounter ID of the Patient. The  icon appears on the screen if the long name for the External Referral Source is recorded, in ‘External/Referral Source’ function of Application Masters module. Place the mouse on the  icon to display the long name of the External or Referral Source recorded in ‘External/Referral Source’ function of Application Masters module.

  19. Anaesthesia – Select the Anaesthesia using the [?] function. The values are created in the Application Masters module.

  20. Remarks – Select the Remarks hyperlink to enter Remarks.

  21. Pre-operative clinical comments – Enter the Pre-operative clinical comments by selecting the hyperlink.

  22. Pre-Operative Diagnosis – Click Pre-Operative Diagnosis hyperlink to record the record the pre-operative diagnosis.   

Operations/Procedures

  1. Operation – Select the Operation using the [?] function. The operations are created in the Operations/Procedures function. This is a mandatory field.

  2. Side Applicable – Select the side applicable for the Operation from the list. The listed values are Left Side, Right Side and Bilateral. This is a mandatory field.

  3. Standard Duration – The Standard Duration of the Operation/Procedure is displayed as defined in the Operations/Procedures function

  4. Surgeon Avg. Time – The average time taken by the surgeon to complete the Surgery is displayed. This Average time is calculated by any previous recording for the same surgery taken by the surgeon entered in for 'Procedure Start Time' and 'Procedure End Time' while recording surgeon notes.

  5. Duration – The duration of the Operation/Procedure is displayed as defined in the Operations/Procedures function. This is a mandatory field.

  6. Pre-operative comments – Enter comments if required.

  7. Click [Select] to display the selected Operation details. Select the Operation Code hyperlink to edit details or cancel the operation.

  8. Click [Apply] to create booking for the Patient and the Create Booking screen is displayed.
