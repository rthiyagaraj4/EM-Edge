Mark For Reschedule

Mark for Reschedule function allows the booking to be rescheduled. This can be done in cases like when the Practitioner feels the patient is not fit for the operation. Once a booking is marked for rescheduling, you cannot change the priority or sequence for that booking. The status for this booking will appear as rescheduled in the ‘Booking Verification’ screen. You have to give an appropriate reason for rescheduling the operation. These bookings will appear in the ‘To be Rescheduled Cases’. You can reschedule or cancel the booking.

 

  1. The Surgeon, Specialty, Theatre, Date and Time of the surgery are displayed. The following details are displayed as recorded while creating a booking - Theatre, Specialty, Assistant Surgeon 1, Assistant Surgeon 2, Anesthetist, Anaesthesia, Patient Position and Source.

  2. Click the Remarks hyperlink to view the remarks recorded while booking.

  3. Click Pre-operative clinical comments to view the details recorded while booking.

The  icon appears beside the Source field if the long name for the External Referral Source is recorded, in ‘External/Referral Source’ function of Application Masters module. Place the mouse on the  icon to display the long name of the External or Referral Source recorded in ‘External/Referral Source’ function of Application Masters module.

 

Operations/Procedures

  1. The Operation Code, Description, Side Applicable, Duration and Remarks recorded for the Operation/Procedure is displayed. Click the Pre-operative comments hyperlink to view the Operation/Procedure remarks recorded for the Patient.  

 

Mark for Reschedule

  1. Reschedule By – The user marking for reschedule is displayed.

  2. Date/Time – The current date and time is displayed as the date of reschedule.

  3. Reason – Select the Reason using the [?] function. The Reasons for Marking for Reschedule are defined in the Reason function. This is a mandatory field.

  4. Click [Apply] to save the operation/procedure as a rescheduled case.
