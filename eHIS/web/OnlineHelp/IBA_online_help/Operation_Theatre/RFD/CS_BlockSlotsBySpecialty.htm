Enter the following details:

 

  1. Operating Room - Select the Operating Room from the list to block a slot. The  listed Operating Rooms are defined in the 'Operating Rooms' function.

 

Block Slots for the Operating Room

  1. Specialty - Enter the Specialty or select the specialty using the [?] function. The listed values are defined in the 'Specialties' function in Application Masters module. While booking an Operation Theatre for an operation, only the specialties defined in this field can be booked. This is a mandatory field. For example, if you have blocked slots for the specialty 'Cardiology' on all days of the week from 8 am to 10 am in Operating Room 2, an orthopedics operation cannot be performed in Operating Room 2 during the blocked time slot.

  2. Day - Select the day for blocking the slots for the specialty. This is a mandatory field.

  3. Start Time - Enter the start time in hh:mm format to block the slot for the specialty. This is a mandatory field.

  4. End Time - Enter the end time in hh:mm format to block slots for the specialty. This is a mandatory field.

  5. Schedule Type - Select the 'Schedule Type' from the list. The options available are, 'All Weeks' and 'Specific Weeks'. If 'Specific Weeks' is selected, you can select the weeks to block slots. If 'All Weeks' is selected, all the 'Weeks' check boxes are selected and disabled.

  6. Weeks - The check boxes are enabled if Schedule Type is selected as 'Specific Weeks'. Select the required check boxes to indicate that the slots are blocked for the specific weeks selected.
