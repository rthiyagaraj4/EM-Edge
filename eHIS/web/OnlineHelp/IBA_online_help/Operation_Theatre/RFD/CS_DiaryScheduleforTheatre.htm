Enter the following details:

 

  1. Operating Room - Select the Operating Room from the list. The schedule created for all surgeons for the Operating Room are displayed. The listed Operating Rooms are defined in the 'Operating Rooms' function. This is a mandatory field.

 

Diary Schedule for Theatre

  1. Surgeon - 'All Surgeons' are displayed and disabled, by default.

  2. Specialty - Enter the Specialty or select the specialty using the [?] function. The listed values are defined in the 'Specialties' function in Application Masters module. This is a mandatory field.

  3. Start Time - Enter the Start Time in hh:mm format for the theatre. This is a mandatory field.

  4. End Time - Enter the End Time in hh:mm format for the theatre. This is a mandatory field.

  5. Day - Select All Days or a specific day to schedule the theatre.

  6. Schedule Type - Select the 'Schedule Type' from the list. The options available are, 'All Weeks' and 'Specific Weeks'. If 'Specific Weeks' is selected, you can select the weeks to block slots. If 'All Weeks' is selected, all the 'Weeks' check boxes are selected and disabled.

  7. Weeks - The check boxes are enabled if Schedule Type is selected as 'Specific Weeks'. Select the required check boxes to indicate that the slots are scheduled for the theatre for the specific weeks selected.
