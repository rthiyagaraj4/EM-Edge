Enter the following details:

 

  1. Booking Date [From-To] – Enter the Booking Date range in the format of dd/mm/yyyy or select the Booking Date range using Calendar function. This is a mandatory field.

  2. Booking Number – Select the Booking Number for the Operation using [?] function.

  3. Theatre - Select a Theatre using [?] function. The theatres defined in the Operating Room function are displayed in the list. This is a mandatory field.

  4. OT Order ID - Select the required Order ID using [?] function. An order ID is generated as soon as an order is placed in the Clinical Care Management module

  5. Anesthetist - Select the Anesthetist using [?] function. The practitioners with practitioner type ‘Anesthetist’, as defined in the Application Masters module are displayed in the list.

  6. Surgeon – Select the Surgeon using [?] function. The practitioners with practitioner type ‘Surgeon’, as defined in the Application Masters module are displayed in the list.

  7. Operation Description – Select the operation description using [?] function. The operation descriptions defined in the Operation masters are listed.

  8. Specialty - Select the Specialty using [?] function. The specialties defined in the Operations master are displayed in the list.

  9. Operation Type - Select the Operation Types using [?] function. The Operations/Procedures Types defined in the Operations function are displayed in the list.

  10. Patient ID – Select the Patient identification number using [?] function.

  11. Patient Name – The Patient Name corresponding to the Patient ID gets populated in this field based on the selected Patient ID.

  12. Gender – Select the Gender from the list. The listed values are Male, Female and Unknown.

  13. Birth Date – Enter the Date of Birth of the patient in dd/mm/yyyy format.

  14. Stages – This text box gets enabled if the selected Status is ‘Booked Cases’. Select the Stages from the list of values.

  15. Order By - Select the ‘Order By’ value from the list. This is a mandatory field.

  16. Source Type – Select the Source Type from the list. The options available are ‘Nursing Unit’, Clinic’ and ‘Referral’. On selecting the Source Type, the Sources for the selected Source Type are available for selection in the adjacent field. The  icon appears beside the Source field if the long name for the External Referral Source is recorded, in ‘External/Referral Source’ function of Application Masters module. Place the mouse on the icon to display the long name of the External or Referral Source recorded in ‘External/Referral Source’ function of Application Masters module.

  17. Status - Select the Status from the list. The system defined values are: All Cases, Booked, Waitlist, Cancelled and Reschedule Cases. This is a mandatory field.
