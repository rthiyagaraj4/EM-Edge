Enter the following details:

 

  1. Booking Date - Enter the Booking Date to view the booking summary. This is a mandatory field.

  2. Operating Room - Select the Operating Room for which the report has to be generated using the [?] function. The Operating Rooms are defined in the Operating Room function.

  3. Specialty - Select the Specialty for which the report has to be generated using [?] function. The Specialties are defined in the Specialties function.

  4. Surgeon - Select the Surgeon for which the report has to be generated using [?] function.

  5. Nursing Unit - Select the Nursing Unit for which the report has to be generated using [?] function.

  6. Surgery Type - Select the Surgery Type from the list for which the list has to be generated. The Surgery Types are defined in the Surgery Type function.
