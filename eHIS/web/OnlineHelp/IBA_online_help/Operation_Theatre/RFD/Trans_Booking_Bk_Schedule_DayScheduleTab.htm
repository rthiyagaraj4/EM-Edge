Day Schedule Tab

The Day Schedule Tab is enabled only on scheduling the appointment through Schedule Status Tab. This tab displays the schedules for a day based on the selected Theatre and Surgeon. The time set for the selected Operating Room is displayed.

  1. Click the required 'From Time' hyperlink to book an appointment for a patient for the selected time slot. The time slot selected cannot be less than the system time. Book appointment screen is displayed to Book an Appointment for the selected Schedule Date.

  2. The day schedule time slots for the theatres selected in the Schedule Status tab are displayed in the Day Schedule tab. Patient booking details, if any, are displayed against the time slots If you select a fully-booked theatre in which overbooking is allowed, the Day Schedule tab is displayed with a new button Over Booking. Click this button to perform overbooking.3 Click [Over Booking] to display booking screen for overbooking.

If no schedules are available for the selected date, a message ‘For the selected date, Schedule is not available’ gets displayed in the Day Schedule Tab.

  1. If the Operating Room permits Over Booking, select the required slot in the Schedule Status Tab, to perform Over Booking.

  2. Click 'Patient ID' hyperlink of an already booked appointment from the day schedule screen to view appointment details of an already booked appointment and Operating Site Identification.
