Enter the following report criteria:

  1. Operation Date [From–To] – Enter the ‘From’ and ‘To’ dates using the calendar function. This is a mandatory field.

  2. Surgeon - Select the Surgeon using [?] function. The practitioners with practitioner type ‘Surgeon’, as defined in the Application Masters module are displayed in the list.

  3. Procedure - Select the Procedure using [?] function. The Procedures created in the MR module are displayed in the list.

  4. Surgery Type - Select the Surgery Type using [?] function. The Surgery Types created in the Operations function under OT Code Setup are displayed in the list.

  5. View By – Select the ‘Group By’ from the list. The listed values are ‘Specialty’, ‘Surgeon’ and ‘Procedure’. If the value ‘Specialty’ is selected, the Turn around Time Report is generated by Specialty. If the value ‘Surgeon’ is selected, the Turn around Time Report is generated by Surgeon. If the value ‘Procedure’ is selected, the Turnaround Time report is generated by Procedure. This is a mandatory field.
