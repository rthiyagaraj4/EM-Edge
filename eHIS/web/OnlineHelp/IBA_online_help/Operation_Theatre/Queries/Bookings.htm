Bookings

From this function you can Query all the booking details.

Bookings function facilitates viewing all the booking details. The booking details within a specific theatre and with a specific status for a particular period can be viewed by specifying the necessary criteria. The query result provides information on the booking details in a specific Operating Theatre or for all Operating Theatres.

Querying Booking Details

From OT Queries, select the Booking function.

The Booking screen gets displayed.

  1. Enter the details in the mandatory (*) fields to query the booking details. <Related Field Description>

  2. Click [Search] to view the booking details or [Clear] to clear the records displayed in the result area.

  3. Click to e-mail or print a reminder letter.

  4. Click Patient hyperlink to view 'Patient Demographics', Print Appointment Slip, 'View Booking Details' and Operating Site Identification.

    Click Print Appointment Slip to invoke the Print Appointment Slip dialog.

    Enter the following details:

    1. Report Name – The name of the report is displayed.

    2. Report Option – Select the Report Option from the list. The listed values are ‘View’, ‘Print’ and ‘File’.

    1. If the value ‘View’ is selected, the report is viewed in a PDF/Excel format.

    2. If the value ‘Print’ is selected, the report is printed.

    3. If the value ‘File’ is selected, the report is saved in PDF/Excel format in the system.

    1. Required – Select the check box to generate the report. This check box is selected by default.

    2. Copies – Enter number of copies to be printed.

    3. Preferred Language – Select the preferred Language from the drop-down list. The online report is printed in the Preferred Language only when the specific report is available in the patient's preferred language. If the online report is not in the patient's preferred language, the online report is printed in English.

    4. Click [OK] to print the online report or click [Cancel] to cancel the print operation.

  5. Click ‘Diagnosis’ hyperlink from the Pre Operative Diagnosis column to view the diagnosis remarks.

  6. Click [Reset] to clear the entered data.
