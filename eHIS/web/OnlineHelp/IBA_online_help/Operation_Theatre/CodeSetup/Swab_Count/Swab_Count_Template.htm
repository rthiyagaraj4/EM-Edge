Swab Count Template

From this function, you can Create a new Swab and Instrument Count Template and Query for the existing records.

Swab Count function enables you create Swab Count Templates by associating items created for the template. Swab and Instrument count is an important aspect in an operation. All the utilized swabs/instruments and the final number of swabs/instruments has to tally with the initial number of swabs/instruments for an operation.

Creating Swab Count Template

From OT Code Setup > Swab Count, select the Swab Count Template function.

The Swab Count Template screen gets displayed.

  1. Click [Create] to invoke the create screen

  2. Enter the data in the mandatory (*) fields. <Related Field Description>

  3. Click [Select] to display the details in the right hand side of the screen. Select the serial number hyperlink to edit the Initial Count or exclude the item from the template.

  4. Click [Apply] to save a created Swab Count Template or [Reset] to clear the entered data.

  5. Click [Query] to search for existing Swab Count Templates.
