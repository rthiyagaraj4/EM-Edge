Consumable Packs

From this function you can Create Consumable packs used during operations and Query for existing records.

A consumable pack contains all the consumable items related to a particular operation. For example, some common items (such as thread and needle) used extensively during operations can be included in a 'General Consumable Pack'. You can create operation related consumable packs. For example, an 'Eye Pack' can contain all the items necessary for an eye surgery.

Creating Consumable Packs

From OT Code Setup > Operations, select the Consumable Packs function.

The Consumable Packs screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Select] to display the items in the right hand side of the screen. Select the Item Code hyperlink to edit the No. of Units or cancel an item from the Pack. Click [Cancel] to clear the selected details.

  4. Click [Apply] to save a created Consumable Packs or [Reset] to clear the entered details.

  5. Click [Query] to search for the existing Consumable Pack records.
