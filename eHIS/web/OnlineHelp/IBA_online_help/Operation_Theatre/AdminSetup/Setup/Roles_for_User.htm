Roles for User

From this function, you can Assign a Role to the Practitioner and Query for existing roles

You can specify and assign the existing roles created in the role function to the Practitioner.

Creating a Role for user

From the OT Admin Setup > Set up, select the Roles for User function.

The Roles for User screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Click [?] to select the User name. The logged in user name is displayed by default.

  3. Click [Search] to display Role and Role types created in the Role function.

  4. Select a value ‘Applicable’ or ‘Not Applicable’ from the Applicability drop-down list. By default, the value ‘Select’ is displayed.

  5. Select ‘Is Default’ radio button next to the role type. The selected role is displayed by default in the OT Slate screen.

  6. Click [Apply] to save created Roles for Users or [Reset] to clear the entered data.

  7. Click [Query] to search for existing Roles.
