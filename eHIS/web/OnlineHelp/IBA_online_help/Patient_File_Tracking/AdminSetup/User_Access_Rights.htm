User Access Rights

From this function you can Create a new user access rights by assigning a File Storage Location to the desired user and assigning rights to invoke the functions applicable to the assigned file storage location. You can Query for a desired User Access Rights and Delete the queried record.

The User Access Rights function facilitates associating the users to file storage location of the facility and assigning rights to invoke the functions applicable to the associated File Storage Location.

Assigning Access Rights

From FM Admin Setup, select the User Access Rights function.

The User Access Rights screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the user access rights assigned or [Reset] to clear the entered details.

  4. Click [Query] to search for existing access rights for the user.

  5. Click [Delete] to delete the queried record.
