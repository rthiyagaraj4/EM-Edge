Confirm OP Pulling List

From this function you can Confirm the files identified for auto-tracking. The confirmed files are tracked to respective Locations.

The files that are currently with the permanent FS Location are confirmed. The Confirmed files are assigned a unique color-coding. If appointments are Canceled, Transferred or Schedule Cancelled, files for these statuses cannot be confirmed. These files are indicated using color-codes. The following files cannot be confirmed for auto-tracking:

  1. Patients who have canceled their appointments.

  2. Clinic schedule canceled.

  3. Patients whose appointments are transferred to other Clinic or Practitioner.

  4. Patient files currently outside the owner Location.

Confirming OP Pulling List

From the FM Transactions > Track Out to OPD, select the Confirm OP Pulling List function.

The Confirm OP Pulling List function screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Add] to view existing files of the patient to Confirm OP Pulling List, [Show All Files] to view all the files from the selected FS Location to Confirm OP Pulling List or [Clear] to clear the entered data.

  3. The Patient File Details are displayed.

  4. Click ‘Enter Remarks’ hyperlink to view and update the remarks. The default remarks set in the FM Parameter or the remarks associated to the FS Location is displayed. Click [OK] to exit from the Confirm Pulling List Remarks screen.

  5. Select the Confirm check box against the desired file to confirm the identified file for auto tracking.

  6. Click [Apply] to save the record or [Reset] to clear the queried results.

  7. On clicking [Apply], the system confirms with you to print the confirm pulling list record. Click [OK] to print the confirm pulling list record  or [Cancel] to cancel the operation.

  8. The system confirms with you to print the tracer card. Click [OK] to print the tracer card or [Cancel] to cancel the operation. The system performs the required activities and control shifts back to the main screen.

To view the confirmed files, enter the query criteria and select the Display as Confirmed. The confirmed files appear with a unique color-coding scheme. You can change the remarks and Confirm value. Only confirmed files are considered for auto tracking.

To view the files, specify the query criteria and enter the Display status as Appt. Cancelled, Schedule Cancelled, Appt. Transferred or File Outside MRD. On executing the query, the relevant files details are displayed.
