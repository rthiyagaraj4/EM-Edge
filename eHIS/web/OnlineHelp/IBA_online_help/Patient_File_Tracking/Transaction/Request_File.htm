Request File

From this function you can Request a Patient File by the file storage locations (both MRD and non-department) from the Medical Record Department.

The system validates the access rights of the logged in user to request for file from the File Storage Location(s). Only a user with the access rights as defined in the User Access Rights function is allowed to invoke this function. All requests are directed only to the permanent File Storage Locations of the logged in facility. Only the permanent FS Locations (with appropriate rights) can request for files from other facilities. If access rights are not defined for the logged in user, system displays an appropriate alert message.

Requesting a File

From the FM Transactions, select the Request File function.

The Request File screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Add] to view all files belonging to the patient or [Clear] to clear the entered data.

  3. Click [Apply] to request the Patient File or [Reset] to clear the entered data.
