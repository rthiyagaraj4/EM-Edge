Issue File

From this function you can Issue patient files from the permanent file storage locations requested by various file storage locations either within the facility or across the facilities of an enterprise.

Issue of files to other file storage locations are made against electronic or manual request. Usage of this function by the logged-in user is determined based on the access rights defined in the FM Parameter. If ‘Security access to be enforced’ is not selected, then, all the users associated to that facility have rights to issue files from the permanent file storage locations of the facility. If ‘Security access to be enforced’ is selected, then based on the rights of the users, permanent FS locations are listed for issuing the files.

Issuing a File

From the FM Transactions, select the Issue File function.

The Issue File screen gets displayed.

  1. Select the issue location from the Issue FS Location list provided. This field is mandatory.

    • If 'Security Accessed to be Enforced' check box in FM Parameter is selected, the File Storage Locations (permanent) to which the logged in user has access to issue files are displayed.

    • If 'Security Accessed to be Enforced' check box in FM Parameter is cleared, all File Storage Locations (permanent) associated with the logged in facility is displayed.

    • If the logged in user is associated to only one permanent File Storage Location, on invoking the function, the Issue File Storage Location and Issued By are displayed with the associated File Storage Location and logged in User Name respectively.

    • If the logged in user is associated to more than one permanent File Storage Locations, they are listed. You can select the required location from the list.

  1. The logged in User Name is displayed in the Issue By field, by default.

    • If 'Security Accessed to be Enforced' check box in the FM Parameter is selected, the User Name cannot be changed.

    • If 'Security Accessed to be Enforced' check box in the FM Parameter is cleared, the User Name can be selected using [?] function.

  1. On selecting the ‘Issue FS Location’ field, three tabs are displayed.

Outstanding Request

Manual Request

Issue File

  1. Click [Apply] to Issue Patient file(s) or [Reset] to clear the selected criteria.

Files not considered for issue under the 'Issue File' tab is displayed under the Outstanding Request tab.
