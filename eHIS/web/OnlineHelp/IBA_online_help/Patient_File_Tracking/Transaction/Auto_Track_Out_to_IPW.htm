Auto Track Out to IPW

From this function you can Issue the confirmed files from permanent FS Location to the respective Nursing Units in which the patients have bookings for admission on the specified date.

On tracking out the files from the Permanent Locations, the receiving locations have to acknowledge the receipt of the files. The File status is updated based on the auto receipt Parameter Setup in the FM Parameter. If Auto Receipt check box is selected, the File Status is ‘O’ - Outside or ‘T’ - Transit.

Auto Tracking out to IPW

From the FM Transactions > Track Out to IPW, select the Auto Track Out to IPW function.

The Auto Track Out to IPW function screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Track Out] to issue the files to the respective Nursing Units or [Reset] to clear the entered data.

If the 'Auto Track out Upon Confirm Pulling List' check box is selected in FM Parameter Setup, on clicking the 'Auto Track Out To IPW' function a pop-up message gets displayed ‘This function is not applicable as Auto Track out is happening upon Confirm Pulling List.’

If the 'Auto Track out Upon Confirm Pulling List' check box is cleared in FM Parameter setup, on selecting Auto Track out to IPW function displays the Auto Track out to IPW screen.
