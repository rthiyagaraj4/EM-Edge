Link/De-Link Encounter-Volume-Documents

From this function you can Create an association between various Document Types and the File Volume for a specific encounter and Cancel an existing association.

This function helps in associating the File Volume and Document details with a particular Encounter ID. The function lists only those File Volumes associated to encounters in the function ‘Link/De-Link/Change Encounter-File Volume’.

Linking/De-Linking Encounter-Volume-Documents

From the FM Transactions, select the Link/De-Link Encounter-Volume-Documents function.

The Link/De-Link Encounter-Volume-Documents function screen gets displayed.

  1. Enter the Patient ID in alphanumeric format of length as specified in MP Parameter or use [?] function to select the required patient. This field is mandatory.

  2. On selecting the Patient ID, the File Type associated to the Patient ID is displayed.

  3. Select the ‘Action’ from the List. The listed values are ‘Link’ and ‘De-link’. This field is mandatory.

  4. Click [Search] to view the details of the encounter linked with the file volume or [Clear] to reset the records displayed in the result area.

  5. If Action is selected as Link, the screen displays 2 tabs.

Encounter Tab

  1. The Encounter Details tab is enabled, by default. The Encounter Details are displayed in the Encounter Tab.

  2. Click Encounter ID hyperlink to navigate to the Documents Tab.

Documents Tab

  1. On clicking ‘Encounter ID’ hyperlink, the Documents tab gets enabled.

  2. Enter the Document ID in alphanumeric format of 15 characters maximum length or select the Document ID using the [?] function. The Document Types created in the Document Type function are listed. Select the ‘Document No’ hyperlink.

  3. On selecting the Document ID, the Document Type associated to the Document ID is displayed.

  4. Enter the Document Date in the format of dd/mm/yyyy or select the Document Date using the Calendar function.

  5. Enter the Document Details in alphanumeric format of 200 characters maximum length.

  6. Click [Add] to display the selected Document details and associate the Document to an Encounter or [Clear] to reset the entered details.

  7. On Adding, click ‘Associate’ check box to associate the Document to the selected Encounter.

  1. If Action is selected as Delink, the screen displays the encounter details.

    1. If the ‘Action’ selected is ‘De-link’, the encounter details are displayed.

    2. Select the ‘De-link’ check box to De-link the associated Document from an Encounter.

  2. Click [Apply] to save Link or De-link the Document to an Encounter or [Reset] to clear to entered data.
