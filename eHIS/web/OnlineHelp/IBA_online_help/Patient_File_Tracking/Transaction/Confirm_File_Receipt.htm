Confirm File Receipt

From this function you can Record the File Receipt confirmation by the receiving File Storage Locations.

This function is applicable, if ‘Auto Receipt Rules’ in the FM Parameter is cleared for all the identities. The file status is marked as ‘In Transit’ for files issued from the FS Locations until the confirmation of the file receipt is received from the receiving FS Locations.

Confirming File Receipt

From the FM Transactions, select the Confirm File Receipt function.

The Confirm File Receipt function screen gets displayed.

  1. Select the Receiving FS Location Identity from the list. The listed identities are the ones with ‘Auto Receive File on Issue to’ checks cleared in FM Parameter function.

  2. Select the Receiving FS Location from the list. The list displays FS Locations corresponding to the selected Identity and with access rights assigned to the logged in user. This field is mandatory.

  3. The System Date and Time is displayed and disabled as the Receiving Date and Time, by default.

  4. Enter the name of the person confirming the file receipt. If the 'Security access to be Enforced' check box is selected in the FM Parameter, the logged in User Name is displayed as read-only text. If the check box is cleared, the logged in user name is displayed, by default. You can select the User Name using [?] function. This field is mandatory.

  5. On selecting the ‘Receiving FS Location’, the fields Patient ID, File Type and Volume Number are displayed.

  6. Enter a Patient ID in alphanumeric format of length as specified in Master Patient Index module or use [?] function to select the required patient. This field is mandatory.

  7. On selecting the Patient ID, the default File Type associated in the File Type For FS Location is displayed. The active Volume Number is displayed, default. If more than one File Type and Volume Number is associated, select the required value from the list.

  8. Click [Add] to view existing files of the patient for confirming the receipt, [Show All Files] to view all the files from the selected FS Location for confirming the receipt or [Clear] to clear the entered data.

  9. Select the ‘Confirm Receipt’ check box against the required files to confirm the receipt. The check box is selected, by default.

  10. On selecting, click [Apply] to confirm the receipt of the desired file(s) or [Reset] to clear the entered data.
