Enter the following details to issue the confirmed files:

 

  1. FS Location - Select the FS Location from the list. The listed locations are populated based on the user access rights. This field is mandatory.

  2. Trackedout By - The name of the person generating the list is displayed. In the FM Parameter, if Security Access to be Enforced is applicable, the logged in User Name is displayed and disabled. In the FM Parameter, if Security Access to be Enforced is not applicable, the field is enabled and you can select the User Name using [?] function.

  3. Nursing Unit Code [From – To] - Enter the Nursing Unit Code range or select the Nursing Unit code using [?] function.

  4. Booking Date - Enter the Booking Date in the format dd/mm/yyyy. The Booking Date is displayed based on the Parameter Setup that can be modified. This field is mandatory.

  5. Print Tracer Card - Select the check box to print the Tracer Card on tracking out the files.
