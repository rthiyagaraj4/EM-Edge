Enter the following details to generate main and supplementary list:

 

  1. Pulling List Type - Select the Pulling List Type from the list. The values displayed are Main and Supplementary. The value ‘Main’ is displayed, by default.

  2. Booking Date - The Booking date is displayed based on the FM Parameter Setup. The displayed Date cannot be changed. This field is mandatory. In FM Parameter function, the value specified for No. of Days upto which to select for booking in file inquiry is considered for generation of pulling list. The current system date and the parameter value are added to arrive at the booking date value in this function. The list is generated for this calculated Date.

  3. Gender - Select the Gender from the list. The values contained in the list are Male, Female, Unknown and All. The value 'All' is displayed, by default.

  4. Nursing Unit [From – To] - Enter the Nursing Unit code range to limit the generation of pulling list or use [?] function to query the Nursing Unit codes.

  5. Practitioner ID [From – To] - Enter the Practitioner ID range to limit the generation of pulling list or use [?] function to query the codes.

  6. Print Pulling Label - Select the Print Pulling Label check box to print the generated list. On generating the list, the list is routed to the specified printer for printing.

  7. Print Identification Label - Select the Print Identification Label check box to print the Identification Label.

  8. Print Tel. No in Pulling List - Select the Print Tel. No in Pulling List check box to include Telephone Number of the patients in the list. This field is enabled only when Print Pulling List is selected.

  9. Print Bar Code - Select the check box to print the Bar Code. This field is enabled only when Print Pulling List is selected.

  10. Order By - The Order By Terminal Digits is displayed by default and disabled if Print Pulling List check box is cleared. If Print Pulling List is applicable, you can select the values from the  list. The listed values are Terminal Digits and Booking time. The generated list is order based on the value specified. On selecting the required order by value, you can specify either Ascending or Descending value from the list.

  11. Print Patient File Request Slip – Select the ‘Print Patient File Request Slip’ check box to print the Patient file request slip for Booking. The Patient file request slip is printed on confirming the booking.
