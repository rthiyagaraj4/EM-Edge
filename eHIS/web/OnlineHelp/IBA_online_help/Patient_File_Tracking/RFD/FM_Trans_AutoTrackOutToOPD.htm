Enter the following details to issue the confirmed files:

 

  1. FS Location - Select the FS Location from the list. The FS Locations are listed based on the access rights defined for the logged in user. This field is mandatory.

  2. Trackedout By - The person responsible for Tracking out is displayed and disabled if the 'Security to be Enforced' check box is selected in the FM Parameter Setup. If 'Security to be Enforced' check box is cleared in the FM Parameter Setup, the field is enabled and you can select the required User Name using [?] function. The person responsible for changing the file status is specified.

  3. Location Type - Select the Location Type from the list. The listed values are Clinics, Day Care and Procedure Unit.

  4. Location Code [From – To] -Enter the range of Location Code in alphanumeric format of four characters maximum length or use [?] function to query and select the Location Codes. This field is optional. The location code must correspond to the selected Location Type.

  5. Appt Date - Enter the Appointment Date in the format dd/mm/yyyy. The Appointment Date displayed is calculated based on the current system date plus one more day. This field is mandatory.

  6. Print Tracer Card - Select the check box to print the Tracer Card on Tracking out the patient files.
