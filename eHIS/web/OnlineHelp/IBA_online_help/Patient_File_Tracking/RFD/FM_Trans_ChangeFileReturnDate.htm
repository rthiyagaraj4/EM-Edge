Enter the following details to change the file return date:

 

  1. Issue FS Location - Select the Issue FS Location from the list. Based on the access rights of the logged in User the values are listed. This field is mandatory.

  2. Changed By - The logged in User Name gets displayed by default, if the 'Security to be Enforced' check box is selected in the FM Parameter Setup. If 'Security to be Enforced' check box is cleared in the FM Parameter Setup, the field is enabled and you can select the required User Name using [?] function. The person responsible for changing the return date is specified.

  3. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the Patient ID using [?] function. This field is mandatory.

  4. File Type - On selecting the Patient ID, the File Type(s) associated to the patient is displayed. If more than one File Type is associated, select the required file type from the list. On selecting the Patient ID, the default File Type associated in the File Type For FS Location is displayed. This field is mandatory.  

  5. Volume No – The active volume number is displayed, by default. Select the Volume Number from the list. This field is mandatory.
