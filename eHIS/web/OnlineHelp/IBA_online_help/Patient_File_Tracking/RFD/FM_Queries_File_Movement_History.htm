Enter the following search criteria:

 

  1. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the Patient ID using [?] function. This is a mandatory field.

  2. File Type - Based on the selected Patient ID, the File Types applicable are displayed. If the selected patient has more than one File Type associated, they are listed in the list. Select the required File Type from the list.

  3. Volume No - Based on the selected File Type, the Volume number(s) is listed. Select the required volume number from the list.

  4. Issue Date [From - To] - Enter the Issue Date range in the format of dd/mm/yyyy. Specify the required date range.
