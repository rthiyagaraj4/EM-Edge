File Type for FS Location

From this function you can Assign a File Type to a selected File Storage Location at facility level and Query for a desired File Type for FS Location record.

File Type for FS Location function enables you to assign various File Types to a selected File Storage Location. You can assign the File Type to a patient based on the File Storage Location in a facility, during patient registration, patient file generation, visit registration, inpatient admission, or emergency registration. A File Type applicable to a facility is associated to various Patient Classes.

Assigning File Types to a File Storage Location

From the FM Code Setup, select the File Type for FS Location function.

The File Type for FS Location screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the File Storage Location from the list. This field is mandatory. The listed locations are defined in the function File Storage Location.

  3. Click [Search] to display the File Types defined in File Type function of Master Patient Index module.

  4. Select the Enabled check box against the required File Types to assign the file type to the selected File Storage Location. The system displays a maximum of 14 File Types per page. If the File Types are more than 14, they are displayed across pages. To select all the File Types displayed across pages, select the Page Select All check box. To select all the File Types in a particular page, select the Select All check box.

  5. Select the default check box and the selected File Type is made default for the File Storage Location. You can select only one File Type can be selected as default for the File Storage Location. The default check box is enabled on associating the File Types to the File Storage Location.

  6. Click [Apply] to save the File Type for FS Location record or [Reset] to clear entered data.

  7. Click [Query] to search for existing File Type for FS Location records within the facility.
