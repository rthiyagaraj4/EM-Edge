Encounter-File Volume Audit Trail

From this function you can Query about the audit trail of transactions done in Link/De-Link/Change Encounter-File Volume function.

This function allows you to search and view the audit trail details of the transaction such Link, De-link or Change the association of a patient encounter and the patient file volume happened in the ‘Link/De-Link/ Change Encounter File Volume’ function for a selected patient and File Type.

Based upon the site setup File Type and File number will display

Querying for Encounter-File Volume Audit Trail

From FM Queries, select the Encounter-File Volume Audit Trail function.

The Encounter-File Volume Audit Trail screen gets displayed.

  1. Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the Patient ID using [?] function. This is a mandatory field.

  2. On selecting the Patient ID, the File Type/File No created for that patient ID gets displayed in the File Type field if available.

  3. Click [Search] to select the records that satisfy the query criteria or [Reset] to clear the entered search criteria.

  4. On searching, the audit trail of transactions done in Link/De-Link/Change Encounter-File Volume function get displayed.

  5. Click [Clear] to clear the result screen.
