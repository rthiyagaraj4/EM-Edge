Down Load Quality Results

From this function you can Generate report of the quality screening test results.

The Down Load Quality Results function is used to generate the report of the quality screening test results. A quality test for a blood product in the stock is ordered through the ‘Blood Unit Quality Screening’ function. Quality screening is done based on the pre-defined Quality Screening Test setup. You can download the screening results from laboratory and generate a report.

Generating Quality Screening Reports

From the Transactions > Stock Operation, double-click Down Load Quality Results function.

The Down Load Quality Results screen gets displayed.

  1. Click the Calendar function to select Quality Screening Date range. System displays the Quality Screening Dates in the range of a month, by default. This is a mandatory field.

  2. Enter the File Name and path to download and generate the report. This is a mandatory field. The system-defined path is displayed, by default.

  3. Click [Generate File] to generate the report.

  4. Errors are displayed in the Error Log text area, if any.

  5. Quality screening reports are displayed in the Quality Data text area, if any.
