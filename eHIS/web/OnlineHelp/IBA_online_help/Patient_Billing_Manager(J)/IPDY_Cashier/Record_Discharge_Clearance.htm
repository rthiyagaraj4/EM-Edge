Record Discharge Clearance

From this function you can Generate discharge clearance slip for an inpatient and day care patient. Also, you can Reprint or Cancel generated discharge clearance slips.

Record Discharge Clearance function is used to generate discharge clearance for inpatient and day care patients. The finance team generates a discharge clearance slip for an inpatient after the discharge advice is generated by the Nurse. Discharge clearance slip is generated irrespective of bill settlement against the discharge bill raised. Users are restricted from placing new orders for the patient after the discharge clearance slip is raised. User can generate a discharge clearance slip only if the discharge advice is raised. You can access this function only if the check box 'Discharge Clearance in Use' is selected in 'BL Parameters' function under the 'Inpatient/Daycare' tab. Authorized users with valid access rights are permitted to generate discharge clearance slip. You can generate discharge clearance slips for patients' with or without discharge bill.

In addition, to generating discharge clearance, user can cancel the generated discharge clearance or reprint the discharge clearance. Users can cancel or reprint the discharge clearance only if the check box 'Discharge Clearance in Use' is selected in 'BL Parameters' function under the 'Inpatient/Daycare' tab. Authorized users with valid access rights are permitted to cancel discharge clearance slips. To reprint the slips, you have to define print routing. The Discharge Document is available for discharged patient's also, so that the document can be reprinted after a patient is discharged from the ward.

Generating Discharge Clearance

From the Transactions > IP/DY Clearance, select Record Discharge Clearance function.

The Generate Discharge Clearance screen gets displayed.

  1. By default, the Generate Discharge Clearance tab get displayed.

  2. Enter the search criteria <Related Field Description>.

  3. Click [Search] to search for patient records matching the search criteria. On clicking [Search], the legend changes to [As On: Current Date Time].

  4. Inpatient records get listed with the Patient ID, Patient Name, Episode ID, Episode Type, Admission Date, Discharge Advice Date, Discharge Bill Document Type and Number and the Patient Outstanding Amount.  

  5. Select the check box against required patient record to generate discharge clearance. Only one patient record can be selected at a time.

  6. An authorization screen pops-up requesting the login details. Enter the details and click [Accept] to generate discharge clearance slip or [Exit] to cancel.

    1. User ID – By default, the current user ID gets displayed only if the logged in user has the privileges.  Enter the user ID in alphanumeric format. This is a mandatory field.

    2. Pin Number – By default the password of the logged user gets displayed if the logged in user has privileges. This is a mandatory field.

    3. Reason - Click [?] to select a reason. The action reasons defined through the 'Reasons for Billing Actions' function with the check box 'Discharge Clearance W/O Bill' as selected are listed. This is a mandatory field.

    4. Remark - Enter the Remarks in alphanumeric format. This is a mandatory field.

    5. Note: Only on entering the valid credentials by authorized user, system will allow the user to generate discharge clearance.

  7. After accepting, click [Generate Discharge Clearance] at the bottom. The Discharge Clearance Slip number gets generated successfully.

 

Reprinting/Cancelling Discharge Clearance

Click [Reprint/Cancel Discharge Clearance] tab to reprint or cancel the generated Discharge Clearance.

 

  1. Enter the search criteria <Related Field Description>.

  2. Click [Search] to search for patient records matching the search criteria. On clicking [Search], the legend changes to [As On: Current Date Time].

  3. Patient records matching the search criteria gets listed. Inpatient records get listed with the Patient ID, Episode ID, Episode Type, Admission Date, Discharge Clearance Date, Discharge Clearance Document Type and Number, Outstanding Amount and the name of the authorized user.  

  4. To reprint any IP discharge clearance, click the Print icon   against the required record. The Reprint ‘Reprint Discharge Clearance and Bill’ window gets displayed with the list of options to select Discharge clearance / Bill / Both.

    • If discharge clearance is selected from the drop-down list, system will invoke discharge clearance report (offline report in PDF) for the selected record.

    • If ‘Bill’ is selected from the drop-down list, system should invoke the bill (Offline) for the selected record, (Offline bill will be in a PDF format ). If no bill exists for the episode, then system will prompt the validation ‘Bill not generated for the selected record – Cannot Print’  

    •  If ‘Both’ is selected, system will display both discharge clearance and bill.

    • Click [Accept] to accept the changes done or click [Exit].

    • On clicking the Reprint icon, discharge clearance report will open in a new window, containing 3 copies in single page (original, copy 1 and copy 2).

No authorization is required for printing discharge clearance.

  1. To cancel any discharge bill, select the check box against patient record. An authorization screen pops-up requesting the login details. Enter the details and click [Accept] to cancel /reprint discharge clearance slip or [Exit] to cancel.

    1. User ID – By default, the current user ID gets displayed only if the logged in user has the privileges.  Enter the user ID in alphanumeric format. This is a mandatory field.

    2. Pin Number – By default the password of the logged user gets displayed if the logged in user has privileges. This is a mandatory field.

    3. Reason - Click [?] to select a reason. The action reasons defined through the 'Reasons for Billing Actions' function with the check box 'Discharge Clearance W/O Bill' as selected are listed. This is a mandatory field.

    4. Remark - Enter the Remarks in alphanumeric format. This is a mandatory field.

    5. Note: Only on entering the valid credentials by authorized user, system will allow the user to generate discharge clearance.

  1. After accepting, click [Cancel Discharge Clearance] button at bottom to cancel generated discharge clearance slip.  The discharge clearance gets cancelled successfully.
