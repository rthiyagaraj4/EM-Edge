Enter the following details:

 

  1. Order Category – Select the order category from the drop-down list. The field is mandatory.

  2. Long and Short Description – Displays the long description and short description of the order category in the respective fields.

  3. Order Type – Select the order type from the drop-down list. This field is mandatory.

  4. Revenue Stage – Select the required revenue stage from the drop-down list.  The list values include Billed, Registered, Select, and Completed.  By default, the value ‘Select’ gets displayed. This field is mandatory.

  5. Valid From – Select the Valid From by clicking the calendar icon. The Valid From date is mandatory.

o Valid From Date should be greater than System date.

o The ‘Valid From’ always accept the (System Date +1) / future dates.
