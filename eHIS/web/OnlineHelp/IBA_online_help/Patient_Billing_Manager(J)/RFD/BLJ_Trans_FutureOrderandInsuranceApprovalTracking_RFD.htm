Enter the following details:

  1. Patient ID – Select the patient ID from the lookup.

  2. Patient Name – On selecting the patient ID, the patient name gets displayed.

  3. Episode Type – Select the required episode type from the list. The list values include All, Inpatient, Outpatient, Daycare, and Emergency. The default value is ‘All’.

  4. Encounter ID – Select the required encounter ID from the lookup.

  5. Specialty – Select the required specialty from the lookup.

  6. Payer Group – Select the required payer group from the lookup.

  7. Payer – Select the required payer from the lookup.

  8. Policy – Select the required policy from the lookup.

  9. Billing Group – Select the required billing group from the lookup.

  10. Service Group – Select the required service group from the lookup.

  11. Service Classification – Select the required service classification from the lookup.

  12. Order Status – Select the required order status from the list. The default value is ‘Select’.

  13. Coding Status – Select the required coding status value from the list.

  14. Insurance Status – Select the required insurance status from the list. The list includes Pending, Approval, Sent to Insurance, Clarification, Rejected and Hold. The default value is ‘All’.

  15. Encounter Date From – To – Select the required encounter date range. Encounter To date cannot be greater than 30 days from Encounter from date. This field is mandatory.
