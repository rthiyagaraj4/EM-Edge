Enter the following details to subscribe the package:

 

  1. Package - Click [?] to select a package. The description of the package is displayed in the adjacent field. The values defined through the 'Package Definition' function are listed.

  2. Billing Class - Click [?] to select a Billing Class. The description of the selected Billing Class is displayed in the adjacent field. The values defined through the 'Billing Class' function are listed.

  3. Payer Group - Click [?] to select a Payer Group. The description of the selected Payer Group is displayed in the adjacent field. The values defined through the 'Payer Group' function are listed.

  4. Payer - Click [?] to select a Payer. The description of the selected Payer is displayed in the adjacent field. The Payers associated to the selected Payer Group are listed.

  5. From Date - Click the calendar function to select the effective date. The package is available for the patient from the date selected.

  6. To Date - The date is computed and displayed automatically based on the 'Valid Days' specified during package definition. You cannot edit the value.

  7. Package Amount - The amount specified for the package during package definition is displayed. You cannot edit the value.

  8. Episode Type - The episode type is displayed based on the encounter selected during package definition.

  9. Practitioner – Enter the practitioner name in alphanumeric format of 30 characters maximum length or click [?] to select the Practitioner Name.

  10. [Add] - Click [Add] to subscribe the package.
