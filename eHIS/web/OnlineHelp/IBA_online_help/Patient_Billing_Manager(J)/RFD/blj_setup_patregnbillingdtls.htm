Enter the following details to define patient registration billing rule:

 

  1. Patient Series – Click [?] function to select a Patient Series from the list. The Patient Series created using the Master Patient Index module gets listed. A patient is charged during registration on associating respective Billing Group, if selected Patient Series is used. This is a mandatory field.

  2. Billing Group - Click [?] function to select a Billing Group from the list. The values defined using ‘Billing Group’ function in Patient Billing Manager module gets listed. A patient is charged during registration, if selected Billing Group is used. This is a mandatory field.

  3. Payer Group - Click [?] function to select a Payer Group from the list. The values defined using ‘Customer Group’ function in the Patient Billing Manager or Accounts Receivable module gets listed. This field is mandatory, if the selected Billing Group is Sponsor/Insurance.

  4. Payer - Click [?] function to select a Payer from the list. The values defined using ‘Customer’ function in the Revenue Management or Accounts Receivable module gets listed. This field is mandatory, if the selected Billing Group is Sponsor/Insurance.

  5. Chargeable at Registration – Select the check box, if the Patient Series and Billing Group combination is chargeable. The patient registration ID charges are computed during patient registration in the 'Master Patient Index' module. Clear the check box, if the Patient Series and Billing Group combination is non- chargeable.

  6. Chargeable at Renewal - Select the checkbox to apply charge for UHID renewal. Once the Patient UHID validity expires, the charge is applicable based on the other combination configured like patient series / billing group / payer group / service/panel. The renewal charges are applied based on the financial details of patient and not based on any encounter. This is applicable only for users in AMRI.

  7. Chargeable at Encounter – Select the check box to charge the patient during encounter registration, if the charge is not computed during registration. The check box is enabled for selection, if the ‘Chargeable at Registration’ check box is cleared. The patient registration ID charges are computed during encounter registration in Admit Patient, Register Visit and Register Attendance functions in the external module. Clear the check box, if the registration charges are to be computed during patient registration.

  8. Billing Service/Panel – Select a value from the list. The system-defined values are ‘Service’ and ‘Panel’.

  9. Service - Select ‘Service’ to associate a Billing Service to the Patient Series and Billing Group combination in the Facility.

  10. Panel - Select ‘Panel’ to associate a Billing Service to the Patient Series and Billing Group combination in the Facility.

  1. Service/Panel Description – Click [?] function to select a Billing Service or a Billing Panel. Patient is charged based on the price defined for the selected Billing Service/Billing Panel. This is a mandatory field. The legend changes to ‘Service Description’ or ‘Panel Description’ based on the value selected for ‘Billing Service/Panel’.
