Enter the following details:

  1. Facility – Select the Facility from drop down. This field is mandatory.

  2. Billing Group – Select the Billing Group using lookup function. This field is mandatory.

  3. Payer Group - Select the Payer Group using lookup function. This field is mandatory.

  4. Payer - Select the Payer using lookup function. This field is mandatory.

  5. Policy - Select the Policy using lookup function. This field is mandatory.

  6. Valid [From-To] – Select the date range using calendar function. The valid from date is greater than current date. Valid To date is equal or greater than current date / Valid from date. This field is mandatory.

  7. Applicable for – Select the Applicable for from drop down. The listed values are Only High Cost Package, Only Low Cost Package and All Packages except high cost.

  8. If you select Only High Cost Package value, the system consider the discount for high cost package only. The system calculates discount based on the value defined in multi package discount.

  9. If the value ‘Only low cost package’ is selected, the system considers the discount for low cost package only . The system calculates discount the low cost package based on the value defined in multi package discount.

  10. If ‘All packages except high cost package’ is selected, the system considers discount for all packages except the high cost package. The system calculates discount for low cost packages based on the value defined in multi package discount.

Allow

  1. Allow Multi Package Discount – Select this checkbox to calculate multi package discount as per the setup enabled.

  2. Allow package discount – Select this checkbox to calculate the package discount as per the setup enabled.

  3. Allow adhoc discount – Select this checkbox to calculate the adhoc discount as per the setup enabled.

New setup multiple package discount is applicable only for the current encounter. The Post multi package discount, discount amount are displayed in Bill cum receipt / Discharge bill cum receipt / Intermediate draft report/ Bills. Gross, Discount and Net amount are displayed correctly. Multi package discount are displayed across BL functions similar to base package discount / customer discount / Package adhoc discount.
