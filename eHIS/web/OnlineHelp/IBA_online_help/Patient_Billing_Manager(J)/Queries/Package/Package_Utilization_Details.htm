Package Utilization Details

From this function you can Query to view the package details that are associated to the patient.

Package Utilization Details function is used to query the package details of a patient. You can view the services that are associated to the package and the status of each package that is associated to the patient.

Querying Patient Utilization Details

From the Queries > Package, select Patient Encounters/Billing Details Function.

The Package Utilization Details screen gets displayed.

  1. Click [?] to select the Patient ID. This is a mandatory field.  

  2. Press [Tab] and the packages associated to the patient are displayed. The name of the patient, gender, age and nationality are displayed.

  3. Click '+' displayed as a hyperlink to view respective package details.

  4. The package code and description, effective date range, billing class, customer group and customer associated are displayed. The amount allocated for each package is displayed.

  5. You can also view the status of the package. Status is either displayed as ‘Open’ and ‘Close’ based on the transactions.

  6. Click Package Sequence Number that is displayed as a hyperlink to view the package service details.
