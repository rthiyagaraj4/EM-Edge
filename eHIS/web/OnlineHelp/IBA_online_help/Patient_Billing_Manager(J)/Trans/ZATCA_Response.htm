ZATCA Response

From this function you can View the response status from ZATCA.

This function allows users to view the response status from ZATCA. User can send the bills to ZATCA manually by clicking on Resend to ZATCA. It will be helpful if user machine gets shut down after generating bill or any mishappening happened during the process or timed out.

 

The below statuses are considered:

• Valid Document

• Accepted Document with warnings

• Invalid Document

Viewing ZATCA Response

From the Transactions, select the ZATCA Response function.

The ZATCA Response screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to view records matching the search criteria.

  3. Click [Reset] to reset the entered search criteria.

  4. The  [Resend to ZATCA] button will be enabled only based on the data populated in screen, else it will be in disabled mode.

  5. Select the required patient record and click [Resend to ZATCA] button to send the bills manually to ZATCA.

After getting ZATCA approval / rejection, cancellation privileges should not be given to users so that they do not cancel bills which is been shared to ZATCA.
