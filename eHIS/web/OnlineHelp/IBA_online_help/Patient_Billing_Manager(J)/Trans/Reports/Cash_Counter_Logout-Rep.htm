Cash Counter Logout – Report

From this function you can Generate a report to view the Cash Counter Report.  

Cash Counter Logout – Report function is used to generate a report on the Cash Counter transactions at the end of the shift. The cashier journal for the respective Cash Counter gets displayed. You can either print, view or save the report in a file. You have to define the printer for the selected report function to print the report.

Generating Cash Counter Logout Reports

From the Reports, select Cash Counter Logout – Report Function.

The Cash Counter Logout – Report screen gets displayed.

  1. Click [?] to select the Cash Counter user. This is a mandatory field. The report is generated for the selected Cash Counter User.

  2. Click [?] to select the Cash Counter. This is a mandatory field. The report is generated for the selected Cash Counter.

  3. Click [?] to select the Login Date and Time. The Login Date and Time when the respective User has logged into the selected Cash Counter gets listed.

  4. Click [?] to select the Logout Date and Time. The Logout Date and Time when the respective User has logged out from the selected Cash Counter gets listed.

  5. Select the option to generate the report. A summarized report is generated listing the Cash Counter transactions.

  6. Click [Run] to generate the report. You can either View, Print or Save the report in a file in the local machine.

  7. Click [Reset] to clear the entered details.
