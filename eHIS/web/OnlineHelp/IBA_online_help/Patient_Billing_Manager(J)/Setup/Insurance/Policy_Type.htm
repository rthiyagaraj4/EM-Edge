Policy Type

From this function you can Create a Policy Type pertinent to the Facility. You can Query to view the existing records.

The function Insurance Policy Types is used to insert a new Policy Type used in the facility. The Policy Type is pertinent to the Facility to which it is associated. The Facilities defined in the System Manager module and associated to the logged in User get listed. The Policy Types defined get populated in the external modules on updating the patient financial details. You can associate the Policy Types to the patients when the selected Settlement Type is Insurance.

Creating Policy Type

From the Setup > Insurance, click the Policy Type function.

The Policy Type screen gets displayed.

  1. The logged in Facility is displayed, by default. You can select a Facility to define the Policy Types pertinent to the respective Facility. The existing Policy Type records are displayed on trying to create a new record.

  2. Click [Create] to create a new Policy Type.

  3. Enter the details in the mandatory (*) fields. <Related Field Description>

  4. Click [Apply] to save or click [Clear Record] to clear the entered data.

  5. Click [Query] to query the policy types.
