Function Privilege Setup

From this function you can Define user wise policy accessibility with view or modify access rights.

The Function Privileges Setup is used to define access right or user privilege for a user for policy level. You can perform policy definition and modification in policy definition or encounter policy definition.

Defining Function Privilege for policy definition, update and association

From the Setup > Insurance, select Function Privilege Setup Function.

The Function Privileges Setup screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Function Privileges Setup or [Reset] to clear the entered data.

  4. Click [Query] to search for existing records.
