Setup/Amend Service Price / Discount

From this function you can Define the discount for each Service wise / Classification wise / Group wise on Total Amount or Payable Amount. You can Query the prices and the discounts defined for a Billing Service associated to the Facility.

The function is used to define a discount for each Service wise / Classification wise / Group wise on Total Amount or Payable Amount. You can associate a charge based Billing Service to the Sale category and define the prices for the patient class.

Amend Service Price/Discount

From the Setup, select Setup/Amend Service Price / Discount function.

The Setup/Amend Service Price / Discount screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the search criteria and click [Search] to display a Price Details.

    The following Price Details are displayed:

    1. Srl No – The serial number is displayed.

    2. Facility – The facility defined during creation is displayed. You cannot modify the data.

    3. Service Ind – The service Ind defined during creation is displayed.

    4. Service Code - The service code defined during creation is displayed.

    5. Serv Description - The service description defined during creation is displayed.

    6. Price Indicator - The price indicator defined during creation is displayed.

    7. Price Ind Code- The price indicator code defined during creation is displayed

    8. Price Ind Description - The price description defined during creation is displayed.

    9. Policy Code- The policy code defined during creation is displayed.

    10. Policy Description - The policy description defined during creation is displayed.

    11. Cust Group Code - The customer group code defined during creation is displayed.

    12. Cust Group Description - The customer description defined during creation is displayed

    13. Srl No - The serial Number is displayed.

    14. Billing Class – The billing class defined during creation is displayed.

    15. Description – The billing class description defined during creation is displayed.

    16. Indicator – The indicator selected at the time of creation is displayed.

    17. IP Rate – The IP Rate defined during creation is displayed.

    18. OP Rate - The OP Rate defined during creation is displayed.

    19. Emg Rate - The emergency rate defined during creation is displayed.

    20. Ext Rate - The external Rate defined during creation is displayed.

    21. DC Rate – The DC Rate is displayed.

    22. Min Charge – The minimum charge defined during creation is displayed.

    23. Max Charge - The maximum charge defined during creation is displayed.

    24. Base Qty - The base quantity defined during creation is displayed.

    25. Applicability - The applicability defined during creation is displayed.

    26. Apply Disc – This checkbox is selected for discounted billing classes.

    27. Valid [From-To] – The validity dates are displayed.

    Enter the following details:

    1. Service Ind - Select an attribute from the list. The value 'Billing Service' is displayed by default. The listed values are ‘Service Group’, ‘Classification’ and ‘Billing Service’. The value ‘Billing Service’ is displayed by default. This is a mandatory field.

    2. Service Code – Click [?] to select Service Code. The service codes defined in the billing masters are listed based on the service indicator selection.

    3. Service Description – The service description is displayed after selecting the service code.

    4. Price Indicator - Select an Indicator from the list. The values are system defined – Standard, Customer Price, Billing Group Price and Price Class Price. The value ‘Standard’ is displayed by default. This is a mandatory field. The value ‘Customer Price’ is displayed by default. This is mandatory field.

    5. Price Ind Code - Click [?] to select a price indicator code from the Price Indicator dialog. The values in the list differ based on the Indicator selected. This is a mandatory field. The price indicator codes defined in the billing masters are listed.

    6. Price Ind Description – The price indicator description is displayed after the price indicator description is selected.

    7. Billing Class - Click [?] to select a Billing Class from the billing class dialog. The billing classes defined in the billing masters are displayed. The value ‘OP’ is displayed by default. This is mandatory field.

    8. Valid [From –To] - Select the valid date range using the calendar function. The price details are displayed within this date range.

    9. Charge Type – Select the Charge Type from drop-down list. The listed values are ‘Rate’ and ‘Charge’. The value ‘Rate’ is displayed by default. The value ‘Rate’ is displayed by default. This is mandatory field.

    10. Disc Exists – Select the check box to display the price details of the discounted records.

    11. Disc Applicability – Select the Disc Applicability from the drop-down list. The listed values are ‘Only Patient’, ‘Only Payer’, ‘Payable Amount’ and ‘Total Amount’. The discount is applicable for the selected value. The value ‘Only Payer’ is displayed by default.

  3. Click [Add] to display Price Details dialog.

    Enter the following details:

    1. Facility – Click [?] to select Facility. The facility defined in the application masters is listed.

    2. Service Indicator - Click [?] to select Service Indicator.

    3. Service Code - Click [?] to select Service Code.

    4. Serv Description – The description of the service code is displayed.

    5. Price Indicator - Click [?] to select Price Indicator.

    6. Price Code - Click [?] to select Price Code.

    7. Price Description – The description of the price code is displayed.

    8. Policy Code - Click [?] to select policy code.

    9. Policy Description – The description of the policy code is displayed.

    10. Cust Grp Code - Click [?] to select customer group code.

    11. Cust Group Description – The description of the customer group code is displayed.

    Price Details

    1. Billing Class - Click [?] to select Billing Class.

    2. Description – The description of the billing class is displayed.

    3. Price Indicator - Click [?] to select Price Indicator.

    4. Check Cost Avl – Select the check box, if the Cost Amount is applicable for the selected billing class, associated to an indicator and a Billing Service. Clear the check box, if the Cost Amount is not considered towards the charge calculation.

    5. Cost Indicator – Enter the cost indicator in the numeric format.

    6. Cost Amt – Enter the cost amount in the cost amount.

    7. Rate – Enter the rate in numeric format. The price is calculated as the product of rate and the billing service price for IP, OP, EM, EX and DC.

    8. Min Charge – Enter the Minimum Charge for the Billing Class.

    9. Max Charge - Enter the Maximum Charge for the Billing Class.

    10. Base Qty - Enter the Base Quantity towards the service. The service charge is calculated based on the price and the quantity of the service used.

    11. Applicability - Select a value from the list. The values are system defined – Gross and Net.

    12. Apply Discount - Select the check box, if discount is applicable to the respective Billing Class associated with the relevant Billing Service. The discount details are displayed only this check box is selected.

    13. Effective (From-To) – Click Calendar function to select a date for new records. The price defined is valid for the selected billing class, Indicator and the Billing Service from the date selected. The date gets displayed automatically when the price is defined initially for the billing class indicator selected and associated Billing Service.

    Discount Details

    1. Discount Indicator – Enter the discount indicator.

    2. Discount In Hours (Upto (Hrs) – Enter the discount in hours and the discount will be calculated based on the time range entered. User can enter only numeric characters starting from 1 to 999.  If the entered number is less than 1 or greater than 999, the system prompts validation message as ‘Range should be within 1 to 999. This field is non mandatory.  This setup is valid based on the effective from date and to date given against the same record.  This is applicable for users in ME Region.

    3. Discount In Hours (% or Amt): Based on the discount indictor selected, user must define % or amount. With % or amount is selected, existing validation handled for other fields in discount section will remain the same for this field also. This is applicable for users in ME Region.

    4. Disc Validity (From-To) – Click Calendar function to select a date range. The discount percentage defined for the billing class is valid only at the date range selected. The discount period has to be within the Billing Service period and has to be greater than the system date.

    5. Discount Percentage – Enter the discount percentage in the numeric format.

    6. Disc Applicability – Select the discount applicability from the drop-down list.

    7. Enable – Select this check box to active the record.

  4. Click [Apply] to amend service Price/Discount. Click [Reset] to reset the entered details.
