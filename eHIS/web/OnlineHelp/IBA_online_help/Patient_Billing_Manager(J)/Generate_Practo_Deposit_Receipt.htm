Generate Receipt for Vendors

From this function you can Generate a Practo Deposit Receipt

The Generate Receipt for Vendors function is used to Generate a Practo Deposit Receipt based on receipt status.

Generating Practo Deposit Receipt

From the Transactions, select Generate Receipt for Vendors Function.

The Generate Receipt for Vendors screen gets displayed.

  1. Click [Create] to display create screen.

  2. Enter the search criteria to select a specific record.

  3. Click [Search] to display the details Patient ID, Patient Name, Date of Birth, Gender, Practo Transaction ID, Practo Receipt No., Practo Receipt Amt in Rs., Practo Receipt Date, EM Receipt No, EM Receipt Date and Comments. or [Clear] to clear the records displayed in the result area.

  4. Select checkbox against the record.

  5. Click [Apply] after to the billing is processed for New Request Status transactions and Receipts are generated. The Apply button is enabled only if at least one record is selected for generating receipts.
