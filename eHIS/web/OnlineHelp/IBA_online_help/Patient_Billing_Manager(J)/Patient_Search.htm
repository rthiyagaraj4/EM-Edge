Patient Search

You can search for the patient records through the search option to modify the unbilled or billed transactions of a patient class in the facility. Enter the mandatory details to search a patient record. The search criteria remains the same for the following functions:

Enter the following details:

  1. Patient ID - Click [?] function or enter the patient ID in alphanumeric format of 10 digits maximum length. This is a mandatory field.

  2. Episode Type - Select an Episode Type from the list. The system-defined values are 'Inpatient' and 'Daycare' patient for Inpatient related transactions and 'Outpatient', 'Emergency' and 'Referral' for Outpatient related transactions.

  3. Episode ID - Enter the Episode ID of the Inpatient or the Daycare patient.

  4. Visit ID - Enter the Visit ID of the Outpatient, Emergency or the Referral patient.

  5. Encounter [From - To] - Click the Calendar function to select the encounter date range. The patient records with the encounter during the selected period are displayed.

  6. Ward -  Click [?] function to select the Ward in which the patient is admitted.

  7. Click [Search] to search the patient records meeting the search criteria or [Clear] to clear the details. Patient Encounter details are displayed on clicking [Search]. Encounter Type, Encounter ID, Admission Date and time, Discharge Date, Primary Privilege, Primary Payer, Location, Status, Room and Bed details are displayed. Encounter ID is displayed as an hyperlink.
