Cash Counter Login

From this function you can Login to an Outpatient, Emergency patient or an External patient Cash Counter at the beginning of the shift in the facility. You can also View the Logged in User/Counter details using this function.  

Cash Counter Login function is used to login to the Outpatient, External or the Emergency patient Cash Counter at the beginning of the shift. The transactions related to the Outpatients External  patients or the Emergency patients have to be performed in the respective Cash Counter to perform billing transactions related to selected patient category. The privilege rights for the respective Cashier have to be defined in the Administration menu. The Cash Counter should have access to process receipts, so the Cashier can generate receipts as well at the respective Cash Counter level.

Login to the Outpatient/Emergency/External Cash Counter

From the Transactions > OP/EX/EM Cashier, select Cash Counter Login Function.

The Cash Counter Login screen gets displayed.

  1. Click [List of Values] to select a Cash Counter. The Cash Counter codes defined using the function Cash Counter gets listed. This is a mandatory field.

  2. The Inpatient Allowed, Outpatient/Emergency Allowed and External Allowed check boxes are selected or cleared based on the parameters defined while creating the Cash Counter.

  3. The Shift Start At is left blanks and the system time is set as the current time, by default.

  4. The name of the logged in User is displayed in the ‘Shift Started By’ field. You cannot update the name.

  5. The current system date and time gets displayed in the Current Date/Time field.

  6. Enter the Opening Balance in numeric format. The Opening Balance indicates the amount in the Cash Counter during the beginning of the shift.

  7. Click [Confirm Login] to login to the Cash Counter or [Reset] to clear the data.

  8. The detail on the user who has already logged into the Cash Counter details gets displayed listing the Cash Counter code and description. You can also view the user name and the login date and time.

An Alert message ‘User is already logged into Cash Counters valid for OP, IP and EXT gets displayed, if you have already logged into the Cash Counter.
