Modify IP Unbilled Reimbursable Amt

From this function you can Update the reimbursable amount for the services utilised by a patient before the bill is generated.

The Modify IP Unbilled Reimbursable Amount function enables you to update the reimbursable amount for the services that are utilised by an Inpatient or a Daycare patient from the facility. An authorized user can reimburse the amount only if the services are not billed. The reimbursable amount for the service is defined in the ‘Billing Services’ function in the Masters. If the ‘override’ check box is selected for the respective service and the user is permitted to update the reimbursable amount for the respective service using the ‘Reimbursable Service by User’ tab in the ‘User for Authorization/Refund/Cash Counters’ function, the user can modify the reimbursable amount for an Inpatient or a Daycare patient.

Modify Inpatient Unbilled Reimbursable Amount

From the Transactions > IP Direct Charges, select Modify IP Unbilled Reimbursable Amt Function.

The Modify IP Unbilled Reimbursable Amount screen gets displayed.

  1. Patient Encounter details are displayed on clicking [Search]. Encounter Type, Encounter ID, Admission Date and time, Discharge Date, Primary Privilege, Primary Payer, Location, Status, Room and Bed details are displayed. Encounter ID is displayed as an hyperlink.

  2. Click the hyperlink to modify unbilled reimbursable amount. The Modify IP Bill Reimbursable Amt screen is displayed listing the List of Services and Audit Details.

    The following audit details are displayed:

    Audit Details

    1. Audit No - An Audit number gets generated for the transaction and gets displayed on clicking [Save] to record the details.

    2. Modified By – The name of the user who has reimbursed the amount is displayed.

    3. Modified Date/Time – The date and time when the details are modified is displayed.

    4. Old Reimbursable – The check box is selected if it is a previous transaction.

    5. Old Reimbursable Type – The value is displayed based on the value selected while reimbursing the bill amount.

    6. Old Reimbursable Amount – The reimbursable amount is displayed based on the amount reimbursed.

    7. New Reimbursable – The check box is selected if it is a new transaction.

    8. New Reimbursable Type – The value is displayed based on the value selected while reimbursing the bill amount.

    9. New Reimbursable Amount – The reimbursable amount is displayed based on the amount reimbursed.

    10. Overriding Reason – The reason selected while recording the reason is displayed.

    The following details are displayed:

    1. Service Description – The services associated to the patient are displayed as hyperlinks. Click the hyperlink to view the audit details for the respective Billing Service.

    2. Item Description – The description of the Item associated is displayed.

    3. Service Date – The date and time when the service is associated to the patient are displayed.

    4. Net Charge – The net charge of the service is displayed. Net charge is recorded while defining the Billing Service.

    5. Defined Reimbursable – The check box is selected or cleared based on the validation set during Billing Service definition. If the service is selected as ‘reimbursable’, the check box is selected or it is cleared. The check box is disabled from selection.

    6. Defined Reimbursable Type – The value is displayed as Part or Full based on the value selected during Billing Service definition. The value changes based on the value selected while modifying the bill reimbursable amount.

    7. Defined Reimbursable Amount – The reimbursable amount for the respective Billing Service is displayed based on the Reimbursable Type. The value changes based on the value selected while modifying the bill reimbursable amount.

  3. Services utilized by the patient  in the facility are listed. The service date, net charge, reimbursable type and amount are listed.

  4. Select the 'Override' check box to modify the reimbursable amount.

  5. An authorization screen is displayed. Enter the details to login and override.

    Enter the following details:

    1. Override - Select the check box to override the reimbursable amount. The Reimbursable check box is enabled for selection on selecting the check box.

    2. Reimbursable – Select the check box to reimburse the amount. The ‘Reimbursable Type’ and ‘Revised Amt’ fields are enabled on selecting the check box.

    3. Reimbursable Type – Select a value from the list. The system-defined values are ‘Full’, ‘Part’ and ‘NA’. Select Full to reimburse full amount or Part to reimburse partial amount. The value is defaulted to NA, if the check box ‘Reimbursable’ is cleared. This field is enabled, if the check box ‘Reimbursable’ is selected.

    4. Revised Amt - Enter the reimbursable amount in numeric format. This field is enabled, if the check box ‘Reimbursable’ is selected. On selecting ‘Reimbursable Type’ as ‘Full’, the net amount of the service is displayed. You can enter the value only when the ‘Reimbursable Type’ is selected as ‘Part’.

    Enter the following details:

    1. User ID - Enter the User ID to login and override. This is a mandatory field.

    2. PIN - Enter a valid PIN to login and override. This is a mandatory field.

    3. Use Previous Reason - This check box is enabled only if the reimbursable amount is overridden for a service associated to the patient. Select the check box to use the previous reason using which the reimbursable amount is overridden. You cannot select the check box when you override the amount for the first time.

    4. Reason - Click [?] function to select a reason. The values defined through 'Reasons for Billing Actions' function with 'Override Reimbursement' check box as selected are displayed. This is a mandatory field. This check box is not available for selection when you reimburse a service initially.

  6. Click [OK] to validate or [Cancel] to abort the process.

  7. Click [Apply] to save the changes or [Reset] to clear the details.
