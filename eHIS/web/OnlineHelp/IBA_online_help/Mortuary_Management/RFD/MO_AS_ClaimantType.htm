Enter the following details:

 

  1. Code – Enter the Claimant code is alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Long Description – Enter the long description of Claimant in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description of Claimant in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Claimant Type/Relationship - Select the Claimant /Relationship from the list. The listed values are Embassy of a Country, Others, Religious Organizations, Under Taker and Welfare Associations. This is a mandatory field.

  5. Enabled - Select the check box to enable Claimant record or clear the check box to disable the record.

 

Address Details

  1. Address line (1-4) - Enter the address details of the claimant in the alphanumeric format of 30 characters maximum length in multiple text areas of the same length.

  2. Town - Enter Town of the claimant or select using the [?] function.

  3.  Area – Enter Area of the claimant or select using the [?] function.

  4. Region – Enter Region of the claimant or select the using the [?] function.

  5. Postal Code - Enter Postal Code of the claimant or select using the [?] function.

  6. Country - Select Country of the claimant from the list. The countries listed are defined in the Master Patient Index module.

 

Contact Details

  1. Tel No. - Enter the telephone number in the alphanumeric format of 18 characters maximum length.

  2. Telex No. - Enter the telex number in the alphanumeric format of 18 characters maximum length.

  3. Fax No. - Enter the fax number in the alphanumeric format of 18 characters maximum length.

  4. Email - Enter the email address in the alphanumeric format of 50 characters maximum length.

  5. Web - Enter the website address in the alphanumeric format of 50 characters maximum length.
