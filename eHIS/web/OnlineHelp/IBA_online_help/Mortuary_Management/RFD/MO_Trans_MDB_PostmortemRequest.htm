Postmortem Request

This sub function is used to place a request to record postmortem for medico legal and clinical postmortem cases if postmortem status is 'Not Requested' for the deceased/body part. On selecting this sub function, the control reverts to the OR module where request for postmortem is done based on the value set for 'Clinical Postmortem' and 'Medico Legal Postmortem' in the MO Parameters. A postmortem request can be placed even before assigning an area for the first time to a deceased/body part.

Postmortem Request option is applicable to deceased/body parts registered in the mortuary to any area except Release Area with 'Postmortem Required' field checked and the postmortem status as Not Requested.  

Click Postmortem Request, the Place Orders screen gets displayed.

Postmortem Request

  1. Select a ‘Deceased Name’ Hyperlink with postmortem status as ‘Not Requested’. The control reverts to the OR module based on the order catalog set for ‘Clinical Postmortem’ and ‘Medico Legal Postmortem’ in the function “MO Parameters”.

  2. Enter the required details. Click [Release] to place a request for postmortem and the control reverts back to the Manage Deceased/Body Part screen based on a confirmation. The screens of the OR module is displayed only if the OR module is installed. Based on the order catalog set for ‘Clinical Postmortem’ and ‘Medico Legal Postmortem’ in the function “MO Parameters”, the screens to place the request for postmortem may differ.

  3. The postmortem status of the deceased patient is now displayed as ‘Requested’.
