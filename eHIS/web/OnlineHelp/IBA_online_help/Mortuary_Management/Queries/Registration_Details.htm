Registration Details

From this function you can Query registration details of deceased patients and body parts.

The registration details displayed are the details that were recorded while registering the deceased patient and body part in the mortuary department.

Querying for Registration Details

From MO Queries, select Registration Details function.

The Registration Details screen gets displayed.

  1. Enter the query criteria and click [Search] to query registration details of a specific deceased/body part.

    1. Registration number - Enter the registration number or select the registration number using [?] function.

    2. Patient ID - Enter the patient ID in alphanumeric format of 10 characters maximum length or select the deceased patient ID using [?] function.

    3. Area Type - Select the area type from the list. The listed values are Postmortem Area, Release Area, Storage Area and Others.

    4. Area - Select the area from the list. The listed values are defined in the Area function. The list also displays 'Unassigned' and 'Unregistered'. 'Unassigned' Area applies to those deceased/body parts that are not yet assigned to any area. 'Unregistered' refers to deceased/body parts that are not yet registered to the mortuary but have placed an order from the "Clinician Access" module.

    5. Postmortem Status - Select the postmortem status from the list. The listed values are 'Not Applicable', 'Not Requested', 'Requested', 'In Progress' and 'Completed'.

    6. Release Status – Select the Release Status from the list. The listed values are Released and Not Released.

    7. Date From/To - Enter the date range in dd/mm/yyyy format to query deceased patients in the mortuary within the specified date range. This is a mandatory field.

    8. Gender - Select the gender of the patient from the list.

    9. Burial Permit No - Select the burial permit number using the [?] function.

    10. Body Part Type - This field is displayed as 'Not Applicable' and is disabled for deceased patients. Select a value from the list for body parts. The listed values are Amputated Body Part, Dead Fetus and Unidentified Body Part.

  2. Deceased/Body Parts assessed for the current date is displayed, by default.

  3. Select the Deceased Name/Patient Name hyperlink in the Deceased Patients Tab and Body Parts Tab to view registration details.
