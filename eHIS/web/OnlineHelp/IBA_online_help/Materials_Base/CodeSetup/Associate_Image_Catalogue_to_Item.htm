Associate Image/Catalogue to Item

From this function you can Associate image and catalogue files to an item.

This function allows you to add image or catalogue files related to an item against the logged in facility. You should ensure that the file name must match with the file name stored in the server path mentioned in the ‘Item Image/Catalogue Path’ field in the Facility Parameter. You can Query for the existing filenames.

Creating an association of Image/Catalogue to an Item

From the Materials Setup, select the Associate Image/Catalogue to Item function.

The Associate Image/Catalogue to Item screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Item Code using the [?] function.

  3. Enter the name of the image file.

  4. Click [Apply] to save the file name or click [Reset] to clear the entered data.

  5. Click [Query] to view or modify the existing file names.

  6. Click [Delete] to delete the image/catalogue associated to the item.
