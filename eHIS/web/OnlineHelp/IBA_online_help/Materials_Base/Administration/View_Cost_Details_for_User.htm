View Cost Details for User

From this function you can Define attributes to view the cost details for a User.

You can define 10 Material Group codes at a time.

Defining Attributes to View Cost Details

From the ST Administration, select the View Cost Details for User function.

The View Cost Details for User screen gets displayed.

  1. Enter the User ID in alphanumeric format of 60 characters maximum length or select the User ID using [?] function. This is a mandatory field.

  2. On selecting the User ID, the name of the selected User is displayed.

  3. Select the ‘View Cost Details’ check box to provide access to view the cost details for the selected User.

  4. Click [Apply] to save the defined record or [Reset] to clear the entered data.

If mandatory values are not entered, the system alerts you to enter them. The mandatory field is highlighted with a unique color code. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate code with a unique color coding.
