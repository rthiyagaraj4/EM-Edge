Enter the following details to query GRN History:

 

  1. Item – Enter the Item in the alphanumeric format of 60 characters maximum length or select the item from the Item Master [?] function.

  2. Supplier – Select the Supplier from the list. This is listed from the Supplier Master Code.

  3. Finalized – Select the required value from the list. All the records which are marked Finalized in the GRN are listed. The listed options are ‘All’, ‘Finalized’ and ‘Unfinalized’. If ‘Finalized’ is selected, all the files which are marked as finalized is displayed. If ‘Unfinalized’ is selected, all the files, which are not marked as finalized, is displayed. If ‘All’ is selected, all the files are displayed both finalized and Unfinalized.

  4. Date [From-To] – Enter or select the date range. Based on the date range the search result gets displayed.

  5. Store – Select the Store from the list.

  6. Replacement GRN – Select the appropriate value from the list. The listed values are ‘All’, ‘Yes’ and ‘No’. If the value ‘All’ is selected, the entire records are displayed, even if Replacement GRN is not recorded in Goods Return to Vender function, If ‘Yes’ is selected, all the records for which Replacement GRN are recorded are displayed. If ‘No’ is selected, all the records for which Replacement GRN are not recorded are displayed.

  7. PO No – Enter the Purchase Order number in alphanumeric format of 10 characters maximum length.
