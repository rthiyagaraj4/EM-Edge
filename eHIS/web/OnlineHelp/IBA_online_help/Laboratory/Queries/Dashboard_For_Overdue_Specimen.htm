Dashboard For Overdue Specimens

From this function you can Query the dashboard for Specimen Overdue.

The Dashboard For Overdue Specimen query function allows you to query the specimens overdue with department wise based on the event master setup.

Querying Dashboard For Overdue Specimens

From the Queries, double-click the Dashboard For Overdue Specimens function.

The Dashboard for Overdue Specimens screen gets displayed.

  1. The Departments for which the Specimens are pending for Registration, Processing, 1st Level Review and 2nd Level Review are displayed, by default.

  2. Click [As on Date and Time] to refresh the Dashboard for Overdue specimen.

  3. Double click on the Pending Specimens for Registration to register the pending specimen, Processing to process the pending specimen, 1st Level Review and 2nd Level Review to perform 1st and 2nd level review of the pending specimen.

  4. Click [Details] to view the specimen details of the selected department.

  5. Click [Execute Query] or press [F8] to query for the Dashboard for Specimen Overdue.
