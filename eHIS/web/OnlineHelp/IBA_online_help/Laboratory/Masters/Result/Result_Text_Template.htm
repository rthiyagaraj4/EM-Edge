Result Text Template

From this function you can Create the Result Text Template created using a word processor specific to a department. You can Query for the existing records.

Result Text Templates are used to associate templates created using word processors. These templates are associated to a department. You can define if the template is a default template and also protect a template from being listed at resulting.

Defining Result Text Template

From the Masters > Result, double-click Result Text Template function.

The Result Text Template screen gets displayed.

  1. Enter the data in the mandatory (*) fields. <Related Field Description>

  2. Click [Edit Template] to invoke the defined word processor template and modify the Template details. Save and close the template file to return to Result Text Template screen.

  3. Click [Commit] or press [F10] to save the created records or [Clear Record] to clear the entered data.

  4. Click [Execute Query] or press [F8] to search for existing records.
