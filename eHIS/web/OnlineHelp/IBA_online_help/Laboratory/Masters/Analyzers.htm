Analyzers

From this function you can Create a cross-reference between tests codes of the Analyzers with the test codes set up in the laboratory module. You can Query for the existing records.

Laboratories have different analyzers or instruments to assist in processing the specimens. These machines perform the specified tests on the specimen. The machines have the capability of interfacing with laboratory information system like Enterprise Management. Each machine has its own protocol for interfacing. The tests done on the machines are defined by the manufacturer and need to co-relate with the tests defined in the laboratory module.

This function is used to provide cross-reference between the test codes of the laboratory analyzers (interfaced with Enterprise Management) with the test codes set up in the laboratory modules. It is also used to define the mode of data download from the laboratory system and the mode of handling the specimens resulted by the analyzers. If more than one analyzer has the ability to perform the same test and is set up as such, the system will download (broadcast) the request to all such analyzers. Once a result has been updated for a specimen, repeated release of the results for the same specimen from the analyzer is ignored.

Defining Analyzers

From the Masters, double-click the Analyzers function.

The Analyzers screen gets displayed.

  1. Enter the data in the mandatory (*) fields. <Related Field Description>

  2. Click [Analyzer Comments] to invoke ‘Setup Analyzer Comments’ screen. Enter a valid Analyzer Comment in alphanumeric format of 40 characters maximum length. These comments are used if you have selected the option to receive the results from the analyzer as comments or [Close] to close the Setup Analyzer Comments screen.

  3. Click [Analyzer Specimen Types] to invoke the ‘Setup Analyzer Specimen Types’ screen or [Close] to close the Setup Analyzer Specimen Type screen.

    Enter the following details:

    1. Specimen Type – Click [List of Values] to select the Specimen Type. The listed values are defined in Specimen Type function. This is a mandatory field.

    2. Description – The description of the selected Specimen Type is displayed, by default.

    3. Analyzer Specimen Type – Enter a valid Analyzer Specimen Type in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  4. Click [Save] or press [F10] to save the created records or click [Clear Record] to clear the entered data.

  5. Click [Execute Query] or press [F8] to search for the existing records.
