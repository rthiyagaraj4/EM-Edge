Process Sheet Template

From this function you can Edit the Process Sheet Template created using a word processor specific to a Facility. You can Query for the existing records.

The process sheet templates are used to store intermediate process information for the specimen in a Word Processor document. Before creating the templates using this function, the template must be created using Word Processor. You can update the existing template using this function.

Creating Process Sheet Template

From Masters, double-click the Process Sheet Template function.

The Process Sheet Template screen gets displayed.

  1. Enter the data in the mandatory (*) fields. <Related Field Description>

  2. Click [Edit Template] to invoke the defined word processor template and modify the Template details. Save and close the template file to return to Result Text Template screen.

  3. Click [Commit] or press [F10] to save the created records or click [Clear Record] to clear the entered data.

  4. Click [Execute Query] or press [F8] to search for the existing records.
