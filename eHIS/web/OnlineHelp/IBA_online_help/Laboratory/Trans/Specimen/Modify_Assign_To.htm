Modify Assign To

From this function you can Modify the Assign To for a specimen.

This function allows to modify the Assign To user for a required specimen.

Modifying Assign To

From Laboratory > Transactions > Specimen menu, select the Modify Assign To function.

The Modify Assign To screen gets displayed.

  1. Enter the details in the respective fields. <Related Field Description>

  2. Click [Search] to view the list of specimen numbers matching the search criteria.  

  3. The search result details include Specimen No, Registered Date Time, Category Number and Assigned User and Assign To column.

  4. Enter the barcode of the specimen number or scan it to navigate directly to entered specimen number in the searched output list.

  5. Select or change the ‘Assign To’ for a specimen number by clicking the [?] icon.

  6. After assigning, click [Save] to accept the changes done
