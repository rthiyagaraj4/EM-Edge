Specimens Linking for Hist

From this function you can Link as well as Release the child specimens associated to the parent specimen without review.

From the function you can link the child specimen to the parent specimen. Also, when the child specimen is linked to the parent specimen, the child specimen can be exempted from review and release by clicking Link and Release button.

 

This function helps to directly link and release the test results (child specimen linked to parent) in Clinician Access without going to First Level Review and Second Level Review.  The linking of results from parent specimen to child is applicable only to Note Type results.  The tests in child specimen should also be configured for Note Type results.

Linking/Releasing Specimens

From Laboratory > Transactions > Specimen menu, select the Specimens Linking for Hist function.

The Specimens Linking for Hist screen gets displayed.

  1. Select the required department from the list. By default, the department name associated with the logged in user is displayed.

  2. Enter the patient ID or search for the required patient ID using 'Right click for User option'.

  3. Click [Display].  The Patient Name, Age and Specimen details  satisfying the query criteria are displayed.

    The Specimen details include :

    1. Facility – Name of the facility where the specimen has been taken is displayed. You cannot modify the data.

    2. Specimen No – The specimen no is displayed. You cannot modify the data.

    3. Specimen Regd Date – Date and time of registration of specimen is displayed. You cannot modify the data.

    4. Category Code – Category Code assigned to the specimen is displayed. You cannot modify the data.

    5. Category Year – Category Year assigned to the specimen is displayed. You cannot modify the data.

    6. Category Number – The category number assigned to the specimen is displayed. You cannot modify the data.

  4. Click [Clear] to clear the entered search criteria.

  5. Select the check box against the specimen record to be linked to the child specimen.

  6. Go to the Current Specimen tab. The current specimen details related to that Patient are listed.

  7. Select the check box against the current specimen that needs to be linked. You have to select only one record as you can link only one specimen to the parent specimen. If you try to select more than record, an error message will appear.

  8. If you want to only link the specimens, click [Link]. The selected child specimen is linked to the parent specimen. You have to go to the Ist Level Review  and 2nd Level Review  screens to review the specimen.

  9. If you want to link as well as release the specimens, click [Link and Release]. The selected child specimen is linked to the parent specimen and the child specimen is exempted from review and release.

  10. A pink colour box appears against the linked specimen record indicating that Result is completed.
