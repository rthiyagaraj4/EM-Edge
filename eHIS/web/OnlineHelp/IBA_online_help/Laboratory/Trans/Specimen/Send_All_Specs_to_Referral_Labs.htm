Send All Specimens to Referral Labs

From this function you can Send all specimens to the Referral Laboratory for investigation.

The referral lab must be linked to the hospital in the Referral Labs setup in Masters, if the hospital is within the organization’s network. You can choose the Laboratory to which specimen can be send, if the investigation is being serviced by more than one hospital. List of referral labs is available.

In case of Group test, you can send only the entire group Test to outside Lab.

Sending All Specimens to Referral Labs

From Laboratory > Transactions > Specimen menu, select the Send All  Specimens to Referral Labs function.

The Send All Specimens to Referral Labs  screen gets displayed.

  1. Enter the data in the mandatory (*) fields. Related Field Description

  2. Click [Patient Details] to view the specimen details satisfying the specified specimen range.

    The following details are displayed:

    1. Specimen No - The Specimen numbers that are to be send to Referred labs are displayed in this field.

    2. Patient ID - The patient ID to which the specimen belongs is displayed.

    3. Test - The Test name, which is referred, is displayed.

    4. Send? – Select the ‘Send?’ check box next to the desired specimen, to send that specimen to a selected referral laboratory.

    5. Select All? - Select the check box to select all the 'Send?' check boxes.

    6. In-house? - Select the ‘In-house?’ check box if you need to process the specimen in-house even though it is referred out test.

    7. Select All? - Select the check box to select all the 'Inhouse?' check boxes.

    You can choose either 'Send' or 'Inhouse' check box but not both the check boxes.

  3. Click [Save] to save the entered details.
