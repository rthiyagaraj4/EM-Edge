Create Batch

From this function you can Create a batch specific to a laboratory department to process the specimen.

The Create Batch function enables you to create a specimen batch specific to a department, perform site and a workgroup combination. To ease the method of processing specimens, the specimens are grouped into different batches. You can record the maximum number of specimens for a specific batch. You can select the status of the specimens that can be a part of the batch. The batch creation process is valid for a specified period. A batch number gets generated, if there are valid specimens in the selected department.

Create Batch

From Transactions > Batch menu, select the Create Batch function.

The Create Batch screen gets displayed.

  1. Enter the data in the respective fields. <Related Field Description>

  2. Click [Create Batch] to invoke the Preferences dialog to print the created batch.

A warning message ‘No specimen for the process’ is displayed, if there are no specimens in the selected department.
