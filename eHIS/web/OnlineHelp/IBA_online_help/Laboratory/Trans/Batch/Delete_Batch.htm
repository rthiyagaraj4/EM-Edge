Delete Batch

From this function you can Delete a batch specific to a laboratory department, once the process is completed.

The Delete Batch function allows you to delete multiple batches that are already created and processed. The batch details are deleted from the database and will not be available for further transactions.

Deleting Batch

From Transactions > Batch menu, select the Delete Batch function.

The Delete Batch screen gets displayed.

  1. The default department associated to the logged in user gets displayed. Select a value from the list to delete the batches generated for any other laboratory department.

    Enter the following details:

    1. Number - The Batch Numbers generated through the ‘Create Batch’ function for the respective department gets listed.

    2. Date - The date on which the batch has been created is displayed.

    3. Perform Site - The perform site code where the batch is processed is displayed.

    4. Description - The Corresponding Description of the selected Perform Site is displayed.

  2. Select the check box ‘Delete?’ next to the desired batch number. You can delete either a single record or multiple records.

  3. Click [Commit] to delete a batch. A confirmation message ‘Do you want to confirm?’ is displayed. Click [Yes] to confirm deletion or [No] to abort the transaction.
