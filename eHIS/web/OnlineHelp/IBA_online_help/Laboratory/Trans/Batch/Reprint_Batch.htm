Reprint Batch

From this function you can Reprint a specimen batch specific to a laboratory.

The Reprint Batch function is used to reprint the details of a batch by selecting the batch number. The printing details will include the Department name, Batch number, Perform Site, Work Group and maximum number of specimens in the batch.  

Reprint Batch

From the Transactions > Batch menu, select the Reprint Batch function.

The Reprint Batch screen gets displayed.

  1. The default department associated to the logged in user gets displayed. You can select any other department to reprint the batch details.

  2. Click [List of Values] to select the Batch Number. The Batch Numbers generated through the ‘Create batch’ function get listed. This is a mandatory field.

  3. The values for Site, WorkGroup, Test Code and the Maximum Specimens in the Batch are displayed automatically based on the values selected during batch generation for the respective batch number. You cannot edit these values.

  4. Click [Print] to reprint the selected batch details.

  5. Select a report option to reprint. You can either view, print or save the report output in a file. Click [OK] to proceed or [C] to cancel.

  6. A pop-up message ‘Process Successful. Press OK to view report’ gets displayed. Click [OK] to reprint or [Cancel] to cancel the process.
