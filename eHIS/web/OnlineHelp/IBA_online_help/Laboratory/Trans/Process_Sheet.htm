Process Sheet

From this function you can Record the processing information for a specimen.

You can record the information such as intermediate steps done in processing the specimen. This information is used while performing the request. The information is recorded in a word processing document. You can define templates for different types of information that needs to be stored for a specimen. These templates can be recalled and used for recording the information.

Recording Processing Information

From Laboratory > Transactions, double-click the Process Sheet function.

The Process Sheet screen gets displayed.

  1. Enter the specimen number for which the process sheet needs to be entered or click [List of Values] to select the specimens for the selected department.

  2. Click ‘Tab’ to view the specimen details. The details recorded while registering the specimen are displayed.

  3. Click [Clinical History] to view the clinical history recorded for the specimen.

  4. Enter the Template Name or click [List of Values] to select the Template Name. The Templates defined in the ‘Process Sheet Template’ masters function are displayed for selection. Click  icon to edit the processing sheet.

    • Template Name - Enter the template name to be used for entering the information for the specimen or click [List of Values] to select from the list of defined templates. Click 'Open' to invoke WP and to start editing the WP document. Save the document in WP on completion. Exit WP program. Click [Save] to save the information within Enterprise Management.

  5. Click [Save] or press [F10] to record the processing information or [Clear Record] to clear the entered details.
