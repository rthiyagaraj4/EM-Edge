Reprint Slide Label

From this function you can Reprint the slide labels.

Reprint Slide Label function is used to reprint the slide labels. The labels can be reprinted based on the tasks ordered for the specimens for whom the slides are created.

Reprinting the Slide Label

From Laboratory Transactions > Anatomic Pathology menu, double-click the Reprint Slide Label function.

The Reprint Slide Label screen gets displayed.

  1. Select the Department from the list. The logged in Department Code is displayed, by default. Select required department from the list if you have access to more than one department.

  2. Click [List of Values] to select the Group Task. The listed values are defined in the Application Masters module.

  3. Enter the Specimen Number or click [List of Values] to select the Specimen Number to record the grossing details. The system displays a list of specimens for which the sample details are recorded. Press [Tab] to view the specimen details.

    1. Specimen Number – The Specimen number of the selected task ordered date range is displayed.

    2. Ordered Date – The ordered date of the Specimen is displayed.

    3. Category Number – The Category Number of the Specimen is displayed. This number is generated in the ‘Acknowledge Verification’ function.

    4. Patient Name – The name of the Patient is displayed.

    5. Task – The Task code which is created in ‘Request Task’ function is displayed.

    6. Identifier – The Sample Identification number created in the ‘Sample Details’ function is displayed.

  4. Click [Display] to view the task details satisfying the query criteria.

    1. Category No [From/To] – Click [List of Values] to select the range of Category .

    2. Task Order Date [Form/To] – Select the Task Order Date range using the Calendar function to get the task details.

    3. Select Option – Select the option from the list. The values displayed are ‘Outstanding’, ‘Complete But not Sent’ and ‘All’.

  5. Select the ‘Print Label?’ check box beside the required task to reprint the label for that task.

  6. Click [Print Task Details] to print the task details.

  7. Click [Specimen Task Details] to view Specimen, Block or Task details. Click [Close] to return to the Reprint Slide Label screen.

  8. Click [Patient Details] to view the details of the Patient.

  9. Click [Print Slide Label] to print the slide label for the selected task.
