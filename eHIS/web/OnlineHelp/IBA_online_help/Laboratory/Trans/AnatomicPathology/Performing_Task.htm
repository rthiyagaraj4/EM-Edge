Performing Task

From this function you can Record the samples, which are consumed.

When the ordered task is completed, the system provides a facility to mark the samples. Once the samples are marked no changes allowed for the user task. If all the materials are sectioned and used, it will be recorded and no additional tasks can be ordered for such samples.

Performing Task

From Laboratory Transactions > Anatomic Pathology menu, double-click the Performance Task function.

The Performance Task screen gets displayed.

  1. Select the Department from the list. The logged in Department is displayed, by default. Select the department from the list if you have access to more than one department.

  2. Click [List of Values] to select the Group Task. The listed values are defined in the Application Masters module.

  3. Enter the Specimen Number or click [List of Values] to select the Specimen Number to perform task for the specimen. The system displays a list of specimens for which the sample details are recorded. Press [Tab] to view the specimen details.

  4. Click [Print Task Details] to print the task details.

  5. Enter the required criteria and Click [Display] to display the performed task details.

    Enter the following details:

    1. Task Ordered Date [From/To] – Enter the Task Ordered Date range or select Task Ordered Date range using the Calendar function.

    2. Select Option – Select the options from the list. The listed values are ‘Outstanding’, ‘Completed but Not Sent, ‘Completed and Sent’ and ‘All’.

  6. Enter the details in the following tab pages.

Task Details Tab

The Task details tab displays the details of the task details for the specimen; this tab is displayed by default.

Task Details:

  1. Specimen Number – The Specimen number is displayed based on the entered search criteria.

  2. Ordered Date – The ordered date of the Specimen is displayed.

  3. Category Number – The Category Number of the Specimen is displayed. This number is generated in the ‘Acknowledge Verification’ function.

  4. Patient Name –The name of the Patient is displayed.

  5. Task – The Task code which is created in the ‘Request Task’ function, is displayed.

  6. Identifier – The Sample Identification number created in ‘Sample Details’ function is displayed.

  7. Task Started – Click ‘Task Started’ check box, ‘Task Start Details’ screen will open. Select the date and time from the calendar function to start the task.

  8. Task Completed – Click ‘Task Completed’ check box, ‘Task Completion Details’ screen will open. Enter the remarks in the remarks text box to complete the task.

  9. Send Task – Click [S] to submit the slide to pathology or consultant.

  10. Print Label – Select the check box to print the slide label.

  11. Select All – Select the check box to select all the tasks.

  12. Click [Specimen Task Details] to view the specimen task details.

  13. Click [Patient Details] to view the details of the Patient.

  14. Click [Print Slide Label] to print slide labels.

Block Consumption Tab

Click ‘Block Consumption’ to mark the block as consumed.

Enter the details in the mandatory fields.

  1. Specimen Number – The Specimen number is displayed based on the entered search criteria.

  2. Category Number – The Category Number of the Specimen is displayed. This number is generated in the ‘Acknowledge Verification’ function.

  3. Patient Name – The name of the patient is displayed.

  4. Sample ID – The Sample ID is displayed.

  5. Block – The Sample Block number created in ‘Sample Details’ function is displayed.

  6. Consumed? – Select the check box to consume the task.

  7. Click [Patient Details] to view the details of the Patient.

Specimen/Sample Consumption Tab

Click ‘Specimen/Sample Consumption’ to view the specimen details. You can select or clear Sample Consumed check box based on the completion of the block.

  1. Specimen Number – The Specimen number is displayed based on the entered search criteria.

  2. Category Number – The Category Number of the Specimen is displayed. This number is generated in ‘Acknowledge Verification’ function.

  3. Patient Name – The name of the Patient is displayed.

  4. Simple ID – The Sample Identification number created in ‘Sample Details’ function is displayed.

  5. Sample Consumed - Select the check box to consume the sample.

  1. On entering the details, click [Save] to record the details or [Clear Record] to clear the entered details.
