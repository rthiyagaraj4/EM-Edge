Result Entry by Differential

From this function you can Result specimens which have peripheral blood film test requested.

Differential is a specific purpose built function to result the specimens, which have peripheral blood film test requested. This function is applicable only to hematology. The keyboard of the computer terminal functions as a cell counter. You can enter the results, as the slide is being analyzed by clicking the keys assigned for each of the differential cell. In addition, the nucleated red blood cells (NRBC) can be recorded in the similar way. The WBC is rectified based on the NRBC and the absolute values for the components of the differential worked out. The results of other user-selected tests, which would assist in resulting of the differential is also displayed.

Result Entry by Differential

From Laboratory Transactions > Result Entry menu, double-click the Result Entry by Differential function.

The Result Entry by Differential screen gets displayed.

  1. Select the desired Department from the list.

  2. Enter the specimen number or select the specimen number using the [List of Values]. On entering the desired details, press 'Tab' to view the specimen details. <Related Field Description>

  3. Enter the WBC (White Blood Cell) counts.

  4. Click [Compute] to calculate WBC count, [Request] to request one more test code, [Incr NRBC] to increase the NRBC count, [Release] to release the result, [Multiple] to multiply the cell count, [NRBC Manual] to enter the NRBC manually.

  5. Click [Hold] to hold the enter result, [Prv Result] to view the previous result, [Direct Result] to enter the result directly, [Cancel] to cancel the result.

  6. Click [Keyboard] to enter the comments in KB text block, [Bypass] to bypass the results. Click [Patient Details] to view the patient details, [Add Clinical History] to record clinical history details for the patient, [Patient Markers] to mark the test for further investigation, [Clinical Comments] to view the comments given while registering the sample.

  1. Click [Process Sheet] to enter the template, [Comments] to view the comments given while registering the sample, [Textual Result] to enter the text, which is entered in Masters; [Save] to save the results or [Clear Record] to clear the entered details.

Results would have to be already available at the Result Entry by Specimen function before user can access this particular function.
