Reticulocytes

From this function you can Define Reticulocytes test setup specific to a laboratory department and Query for the existing records.

This function assists you in directly entering the results as the user views the slide through the microscope. To use the Reticulocytes Resulting function, certain information required by it needs to be set up. This function is used to generate reports and send this report to the lab technician, on identifying infection as per the setup made in this function.

Defining Reticulocytes

From Administration, double-click the Reticulocytes function.

The Reticulocytes screen gets displayed.

  1. Enter the data in the mandatory (*) fields. <Related Field Description>

  2. Click [Commit] or press [F10] to save the created records or click [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for the existing records.
