Result Entry For Batch

From this function you can Record the test results for the specimens that are grouped as  batch by laboratory for reporting.

Result Entry For Batch function allows you to list out specimens that are grouped as batch by laboratory for reporting. You can select the Batch and record the test results for the specimens which are grouped as 'Batch' through the Result Entry function.

Result Entry For Batch

From Laboratory Transactions > Result Entry menu, double-click the Result Entry by Batch function.

The Result Entry For Batch screen gets displayed.

  1. Select the Department from the list. The name of the Department associated to the logged in user is displayed. If the user has access to multiple departments, select the desired department from list.

  2. Enter the Batch Number or click [List of Values] to select the Batch Number. The batch creation date and the corresponding patient details are displayed. Press [Tab] to view the specimen details.

    The following details are displayed:

    1. Perform Site – Enter the Perform site where the specimen to be processed or click [List of Values] to select the Site. The values displayed are defined in Perform Site function in the Masters.

    2. Creation Date – The Creation Date of the batch is displayed.

    3. Work Group – The Work Group associated to the selected batch is displayed.

    4. Maximum Specimens in the Batch – The number of Specimen assigned while creating the Batch are displayed.

    5. Test selection – The type of test is displayed. The values are ‘Outstanding’ and ‘All’.

    6. Batch Selection – The Type of batch that is mentioned while creating the batch is displayed. The values are ‘Any’ and ‘Not in any Batch’.

    7. Date Range [ From – To]  – The ‘From’ and ‘To’ date range for the batch is displayed.

    8. Specimen Number – The specimen numbers satisfying the query criteria are displayed. The color code associated to the specimen displays the status of the specimen.

    9. Specimen Regd. Date Time – The Date and Time of the specimen registration is displayed.

    10. Process Date – The Date on which the specimen is processed is displayed.

    11. Category No – The Category Number associated to the specimen is displayed.

    12. Result Comment – Enter the Result Comment (If any) in alphanumeric format.

  3. Click [Patient Details] to view the corresponding patient details.

  4. Click [Create New Batch] to create a new batch.

  5. Click [Recreate Batch] to edit the created batch.

  6. Click [Result] and the system will take you to the corresponding screen based on the status of the specimen. If the specimen status is ‘Review’, the ‘Review’ screen is displayed or the resulting screen is displayed. You can ‘Review’ or ‘Result’ the specimen.
