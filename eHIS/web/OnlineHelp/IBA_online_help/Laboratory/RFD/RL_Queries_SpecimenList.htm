The specimens which are in process are displayed.

  1. Specimen No – The specimen number is displayed.

  2. Registered Date Time – The Specimen Registration Date and Time is displayed.

  3. Category No – The category number of the specimen is displayed.

  4. Source – The name of the source is displayed.

  5. Test Code – The name of the Test code is displayed.

  6. Result – The status of the result is displayed.

  7. Patient ID – The Patient identification number for whom the specimen is registered is displayed.

  8. Name – The name of the patient is displayed.

  9. Print Outstanding List – Click [Print Outstanding List], if hard copy of the outstanding requests is required.

  10. Enter Result – Click [Enter Result], to result a request from the list, place the cursor on the request to view the result.

  11. View Test Result – Click [View Test Result], to view the test result for a request. (only numeric and the result comment)

  12. Referred Tests – Click [Referred Tests] to view the Pathology Query for the patient.

  13. Interval Tests – Click [Interval Tests] to view interval tests if the test needs to be conducted in specified intervals.

  14. Specimen/Patient Details – Click [Specimen/Patient Details] to view specimen and patient details.
