The Report Ranges for Facilities tab is used to set Report Range, critical Range and Normal Range for Age, sex and Test method for each facility independently.  User can create, edit, and delete the validation values for login facility, also for other facilities without switching facility login.

 

Enter the following details:

  1. Facility - Facility – By default, the login facility is displayed. User can select any other facility from the drop-down list.

  2. Sex - Enter a valid Gender code as either 'M' for male, 'F' for female, or 'U' for Unknown.

  3. Test Method – Click [List of Values ] to select the Test Method.

  4. Maximum Age - Enter the maximum age range in Year, Months and Days format.

  5. Minimum Age - Enter the minimum age range in Year, Months and Days format.

 

Maximum  Age >= Minimum Age.

  1. Normal Range - Enter required Low and High range of values.

  1. Critical Range - Enter the low and high range of values.

  2. Report Range - Enter the Low and High value to be printed on the report for the specified gender and age.
