1. Perform Site - The Site where the test is performed is displayed. These are associated in Test Codes of Application Masters module.

  2. Workgroup - Enter the Workgroup in alphanumeric format of four characters maximum length or click [List of Values] to select the Workgroup. These work groups are created in Application Masters module.

  3. Specimen Type – The code and short description of the specimen type is displayed. You cannot edit the specimen type.

  4. Category No - The category number assigned to the specimen is displayed.

  5. Unit Number - The Unit number is displayed, if the investigation is from Blood Donor and Blood Transfusion Services modules.

  6. Assign To - The name of the technician who is assigned to the selected test in Acknowledge/Verification function is displayed.

  7. Priority - The specimen's priority indicator is displayed. The valid values are 'Routine', 'Urgent' and 'Stat'.

  8. Patient ID – The ID of the Patient is displayed.

  9. Age – The Patient's age is displayed.

  10. Sex- The Patient's gender is displayed.

  11. [Patient Details] – Click [Patient Details] to view the patient admission details.

  12. [Patient Markers] – Click [Patient Marker] to mark the test for further investigation.

  13. [Clinician Comments] – Click [Clinician Comments] to view the comments given while registering the sample.

  14. [Process Sheet] – Click [Process Sheet] to enter the template.

  15. [Add Clinician History] – Click [Add Clinician History] to add clinician comments on the selected patient.

  16. [BB Comments] – This button appears only if the Department is selected as ‘BTS’. Click [BB Comments] to view the blood bank comments for the selected patient. You can add comments if required. The BB Comment codes defined in the Result Notify/BB Comments master appear in the List of Values.
