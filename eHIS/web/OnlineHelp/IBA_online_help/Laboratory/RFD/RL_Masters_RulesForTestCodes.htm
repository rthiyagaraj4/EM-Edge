Enter the following details:

  1. Department - The logged in Department Code is displayed, by default. Select desired department from the list, if you have access to more than one department. This is a mandatory field.

  2. Test Code - Click [List of Values] to view and select a test code. The list displays the values defined in the ‘Test Codes’ function. This is a mandatory field.

 

Rules for Test Code

  1. Rule Code - Click [List of Values] to view and select the required rule code. The list displays the values as defined through Rules masters function. The same rule code is defined for different tests, but duplication is not allowed in the same test-rule. This field is mandatory.

  2. Description - The corresponding description for the selected Rule Code is displayed, by default.
