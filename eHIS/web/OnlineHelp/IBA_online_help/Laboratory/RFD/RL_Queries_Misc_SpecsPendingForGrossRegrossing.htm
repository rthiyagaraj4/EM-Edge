The following details are displayed:

  1. Department – The department name associated with the logged in user is displayed. If the user has access to multiple departments, select the desired department name from Department list.

  2. Option – Select the option from the list. The listed values are Pending for Grossing, Pending for Re-grossing and Both. Both is selected, default.

  3. Registered Date [From-To] – Select the registered date [From-To] using calendar function.

  4. Specimen No [From-To] – Select the Specimen number [From-To] using calendar function.

  5. Specimen No – The specimen number is displayed.

  6. Test Code – The test code and description is displayed.

  7. Sample ID – The sample identification number is displayed.

  8. Anatomy Site – The name of the anatomy site is displayed.

  9. Tissue – The code and description of the tissue is displayed.

  10. Fixative – The code and description of the fixative is displayed.
