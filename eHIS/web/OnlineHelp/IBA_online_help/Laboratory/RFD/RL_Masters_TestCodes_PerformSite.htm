Perform Sites

This tab is used to associate required site to the test in the facility.

 

Enter the following details:

  1. Facility - Select the Facility from the list. The logged in facility is displayed, by default. If you have access to multiple facilities, you can select the required facility from the list.

 

Perform Sites

  1. Site Code - Enter the Site Code or click [List of Values] to select the required Sites. The list displays the values defined in the 'Perform Site' function.

  2. Description - The corresponding Description of the selected site code is displayed, by default.

  3. Default YN - Select the check box to make the required site as a default site.
