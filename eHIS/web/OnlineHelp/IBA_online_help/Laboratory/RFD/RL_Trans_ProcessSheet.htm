Enter the following details:

  1. Perform Site - Enter the Perform Site code in alphanumeric format of four characters maximum length or click [List of Values] to select the Perform Site. The list displays the values defined in the Perform Site masters function.

  2. Department - The department associated to the logged in user is displayed. If the user has access to multiple departments, select the department name from Department list.

  3. Referral Number - The number assigned by the external source to the patient is displayed.

  4. Order No - If the request was registered for an order, the order number is displayed.

  5. Sex – The gender of the patient is displayed.

  6. Episode No – The Episode number of the request is displayed.

  7. Visit No/Tfr No - The Visit or Transfer number for an Outpatient/Inpatient episode is displayed.

  8. Age – The age of the patient is displayed.

  9. Patient ID - The patient identification number is displayed.

  10. No of Samples - The number of samples received is displayed.

  11. Source - The source from where the request originated is displayed. The source codes are Ward/Clinic/External code based on the Episode Type. The description for the source code is displayed.

  12. Copy - The name for which the extra copy of the result is required is displayed.

  13. Doctor - The code assigned to the doctor who requested the tests is displayed. The description for the doctor code is displayed.

  14. Refl Spec No - The specimen number assigned by the external source for the request is displayed.

  15. Spec Type - The specimen type code for the request is displayed. The description for the specimen type is displayed.

  16. Category Number - The category number assigned from the laboratory to the specimen is displayed.

  17. Clinical Comments - The clinical comments recorded for the request is displayed.

  18. Priority – The urgency indicator for the specimen is displayed. The values are Routine, Urgent or Stat.

  19. Col Date - The specimen collection date is displayed.

  20. Recd - The specimen receipt date/time is displayed.

  21. Assign To - If the request is assigned to a specific doctor, the doctors name is displayed.

  22. Regd - The specimen-registered date/time is displayed.

 

Test

The test details of the patient is displayed.

  1. Test Code - The code for the ordered test is displayed.

  2. Description - The description of the test is displayed.
