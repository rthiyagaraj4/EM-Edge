Print Setup

Enter the following details in the Print Setup tab:

  1. Facility Name on Label – Enter the name of the facility in alphanumeric format of 13 characters maximum length to be printed on the specimen label.

 

Laboratory Title

  1. Laboratory Title – Enter the name of the laboratory, which will be printed on the reports.

 

Print Results for

  1. Inpatients – Select the check box to print the Inpatient results on investigation completion.

  2. Outpatients – Select the check box to print the Outpatient results on investigation completion.

  3. Referral - Select the check box to print the Referral patient results on investigation completion.

  4. External - Select the check box to print the External patient results on investigation completion.

  5. Discharged/Deceased – Select the check box to print the Discharged/Deceased Patient results on investigation completion.

  6. Emergency – Select the check box to print the Emergency patient results on investigation completion.
