Enter the following details:

  1. Ext Ref Source/Loc – Enter the External Referral Source code or click [List of Values] to  select the desired code. The list displays the values as defined in the ‘External Referral Source’ function in the Referral Setup function of Application Masters module. This is a mandatory field.

  2. Patient ID - Enter the patient identification number in the format as defined in the MP Parameter of Master Patient Index module. If the patient identification number is not entered, the system generates a patient identification number in a sequence as per the settings made in ‘Patient Series’ function of the Patient Identification Management module.

  3. Series/ID – On entering the external referral source, the Series ID value gets displayed, by default. These values are defined in the Laboratory Parameter.

  4. ID No – Enter the Identification Number of the patient. The Identification Number is defined in the MP Parameter function of the Patient Identity Management module.

  5. Name – Enter the name of the patient.  

  6. Alt Type/No – Select the desired Alternate ID Type from the list. The list displays the values as defined in the ‘Alternate ID Type’ function of MP Code Setup in Master Patient Index module.
