Enter the following details:

 

Department

  1. Department Code – Enter a unique department code in alphanumeric format of one character maximum length. This is a mandatory field.

  2. Long Name – Enter the Long Description for the laboratory department in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Name – Enter the short name description for the Laboratory department in alphanumeric format of 15 characters maximum length.  This is a mandatory field.

  4. Order Type Code – Click [List of Values] to select the Order Type. The list displays the values defined in the ‘Order Type’ function of the Order Entry and Result Reporting module. This is a mandatory field.

 

Sub Department

This tab is visible only for Medicity users.

 

This tab allows you map the sub-departments to the Department and to map the test codes to the sub-departments as required. Based on this mapping, the Lab Result Report displays the Results grouped on the Sub-department code.

 

  1. Department – The department selected in the Department tab gets displayed.

  2. Sub-Dept – Enter the code of the sub-department in alphanumeric format of two characters maximum length. This field is mandatory.  

  3. Short Desc - Enter the short description of the sub-department in alphanumeric format of 15 characters maximum length. This field is mandatory.  

  4. Long Desc - Enter the long description of the sub-department in alphanumeric format of 40 characters maximum length. This field is mandatory.  

 

  •  The Sub-department tab is enabled only for the active departments.

  • A Department cannot have two same sub department codes.
