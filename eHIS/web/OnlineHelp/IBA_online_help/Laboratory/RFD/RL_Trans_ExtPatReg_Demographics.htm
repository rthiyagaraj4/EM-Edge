Demographics Tab

This tab is displayed, by default.

 

Enter the following details:

  1. Title – Select the Prefix from the list. The listed values are defined in the 'Name Prefix' function in Master Patient Index module.

  2. First Name – Enter the First Name in alphanumeric format of 20 characters maximum length as specified in MP Parameter of Master Patient Index module. This is a mandatory field.

  3. Second Name - Enter the Second Name in alphanumeric format of 20 characters maximum length as specified in MP Parameter of Master Patient Index module

  4. Family Name – Enter the Family Name in alphanumeric format of 40 characters maximum length as specified in MP Parameter of Master Patient Index module. This is a mandatory field.

  5. Suffix - Select the Suffix from the list. The values are listed from ‘Name Suffix’ function of Master Patient Index module.

If 'Accept Patient's name in Local Language' is set in the MP Parameter of Master Patient Index module, the patient's name can be entered in local language. If this parameter is not set then ‘Local Language’ details to be entered will not display.

  1. Gender – Select the Gender of the Patient from the list. The listed values are Male, Female and Unknown. This is a mandatory field.

  2. Birth Date – Enter the Birth Date in dd/mm/yyyy format or select the date of birth of the patient using the ‘Calendar’ function. On entering the birth date, the system calculates the Patient's age and displays it in the 'Age (Y/M/D) fields. This is a mandatory field.

  3. Birth Place – Enter the Birth Place of the Patient in alphanumeric format of 30 characters maximum length, or click [List of Values] to select the required birthplace.

  4. Residency – Select either 'Citizen' or 'Non-citizen' from the list.

  5. Category – Select the Category of the Patient from the list. The listed values are defined in the 'Patient Category' function of Master Patient Index module.

  6. Race – Click [List of Values] to select the Race. The listed values are defined in 'Race' function of Patient Identity Management module. This is a mandatory field.

  7. Religion – Click [List of Values] to select the Religion of the Patient. The listed values are defined in the 'Religion' function of Master Patient Index module.

  8. Marital Status – Click [List of Values] to select the Marital Status of the Patient. The listed values are defined in the 'Marital Status' function of Master Patient Index module.

  9. Status – Select either 'Legal' or 'Illegal' as citizen status. If residency is set as 'Citizen', 'Legal' selected and disabled, by default. If the residency is set as 'Non-Citizen', you can select status as either 'Legal' or 'Illegal' from the list.

  10. Nationality – Click [List of Values] to select the ‘Nationality’. The listed values are defaulted from MP Parameter of Patient Identity Management module.

  11. Ethnicity – Click [List of Values] to select the Ethnicity. The listed values correspond to the selected race.

  12. Alias Name – Enter the Alias Name of the Patient in alphanumeric format of 30 characters maximum length.

The fields displayed in the screen may vary depending on the MP Parameter Setup in Master Patient Index module.
