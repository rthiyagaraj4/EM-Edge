Blood Unit Parameters Tab

Enter the following details in the Blood Unit Parameters Tab:

  1. BTS Department – Enter the Blood Transfusion Services department code. The laboratory department is defined in Department function. This is a one-time setup and you cannot edit the records.

 

Antibody Screening Test/Result

  1. Test – Click [List of Values] to select a Test. The list displays the values defined in Test Codes function.

  2. Result – Click [List of Values] to select the Result for Antibody screening to default the values in Antibody identification results in Result Entry by Specimen function. The listed values are created in Antibiotic Code Master in Blood Transfusion module.

  3. Blood Group Test – Click [List of Values] to select a Blood Group. The list displays the corresponding test code comments.

  4. Rhesus Test – Click [List of Values] to select the Rhesus Test. The list displays the corresponding test code comments.

  5. Antibody Identification Test – Click [List of Values] to select the Antibody Identification Test.  The list displays the corresponding test code comments.

  6. RH Pheno Type – Click [List of Values] to select the RH Pheno Type. The list displays the corresponding test code comments.

  7. Blood Typing Confirmation – Click [List of Values] to select the Blood Typing Confirmation code. The list displays the corresponding test code comments.

  8. RBC Phenotype – Select the required test code for the RBC Phenotype from the lookup. The values configured in the ‘RBC Phenotype’ master in BT module appear in this lookup.
