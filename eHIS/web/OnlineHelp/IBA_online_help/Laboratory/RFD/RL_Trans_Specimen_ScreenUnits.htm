Screen Units

Click [Screen Units] to display Blood Donor Outstanding Orders screen.

 

Enter the following details:

  1. Department - Select the department name from the list. The department name associated with the logged in user is displayed. If you have access to multiple departments, select the desired department name from Department list.

  2. Unit No - Enter the unit number in numerical format of 20 characters maximum length.

 

The following details are displayed:

  1. Unit No - The unit number is displayed.

  2. Product Code - The product code is displayed

  3. Order Date - The order date is displayed.

  4. Episode Type - The episode type of the patient is displayed.

  5. Doctor - The code of the doctor is displayed.

  6. Source Type - The source type of the patient is displayed.

  7. Code - The source code of the patient is displayed.

  8. Register? - Select the 'Register' check box to select the blood donor order for registration.

  9. Register All? - Select the 'Register All' check box to select all the blood donor orders for registration.

 

Investigations

  1. Test Code - The test code of the test is displayed.

  2. Description - The description of the test code is displayed.

 

Click [Register Units] to register the selected unit.

 

The Blood Donor Samples are indicated with unique color indicator .

The Blood Transfusion Samples are indicated with unique color indicator .
