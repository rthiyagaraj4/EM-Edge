Enter the following details:

  1. Department – By default, the logged in User's Department gets displayed based on Authorized Users function.

  2. Specimen Reg. From/To Date - By default, the system shows Current Date minus 30 as From date. The Current Date is displayed as To date.  Maximum duration must be 30 days

  3. Specimen No -  Enter the specimen number. If the Specimen No is not valid, an error message gets displayed.

  4. Selection Type - Select the 'Selection Type' from the drop-down list. The listed values are Test, Group Test and Workgroup.

  5. Test – If the selection type is selected as ‘Test’, the field name gets displayed as ‘Test’. Select the required test from the drop-down list. If 'Test' is not entered, the system considers All Test.

  6. Group Test - If the selection type is selected as ‘Group Test’, the field name gets displayed as ‘Group Test’. Select the required group test from the drop-down list.

  7. Work Group - If the selection type is selected as ‘Group Test’, the field name gets displayed as ‘Group Test’. Select the required group test from the drop-down list.

  8. Assigned User -  Select the required assigned user by clicking the [?] function.

  9. Category Number - Enter a valid Category Number.
