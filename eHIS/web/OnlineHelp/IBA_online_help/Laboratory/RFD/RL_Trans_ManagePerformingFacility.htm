Enter the following details:

  1. Ordering Facility - The logged in Facility is displayed, by default. Select the Ordering Facility from the list. This is a mandatory field.

  2. Performing Facility – The logged in Facility is displayed, by default. Select the Performing Facility from the drop-down list. This is a mandatory field.

  3. Change To – Select the Facility from the drop-down list. The Performing Facility for orders is changed to the selected Facility. This is a mandatory field.

  4. Order Category – ‘Laboratory’ is displayed, by default. You cannot change the data.

  5. Order Type – Click [List of Values] to select the Order Type.

  6. Order ID – Enter the Order ID in alphanumeric format of 15 characters maximum length.

  7. Order Date [From-To] – Select the required date range using the Calendar function.

  8. Specimen No – Enter the Specimen Number in numeric format of 20 characters maximum length.

  9. Patient ID - Enter the Patient ID in the length as specified in the MP Parameter or right click ‘Right click for User Options’ hyperlink and select the Patient ID using Hospital Patient Lookup function.

  10. Reason – Enter the comments for changing of the performing facility for the orders in text format of 2000 characters maximum length. This is a mandatory field.
