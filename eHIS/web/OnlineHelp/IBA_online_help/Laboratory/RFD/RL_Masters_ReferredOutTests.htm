Enter the following details:

  1. Department - The logged in Department Code is displayed, by default. Select required department from the list if you have access to more than one department. This is a mandatory field.

  2. Test Code – Click [List of Values] to select a Test Code. The list displays the values defined in ‘Test Codes’ function. This is a mandatory field.

  3. Reference Laboratory – Click [List of Values] to select a Reference Laboratory. The specified test is contracted to the selected Laboratory. The list displays the values defined in the ‘Referral Lab’ function. The short name of the code is displayed for user verification when a valid code is selected. This is a mandatory field.

  4. Result In - Enter the time within which the result has to be reported back from the referral  laboratory. This value is used to calculate the due date of result based on the specimen sent date. This is a mandatory field.

  5. Units - Select a valid Unit from the list. The list displays the valued are system defined.

  6. Remarks - Enter specific remarks pertaining to the test procedure, specimen, transportation, special instructions etc. The first line of remarks is mandatory.

 

Local and Foreign

  1. Currency - Click [List of values] to select the Currency code. The list displays the values defined in ‘Foreign Currencies’ function of the Financials Base Manager module.

  2. Rate - Enter the Rate applicable to the test in numeric format. The test is charged based on the rate defined.
