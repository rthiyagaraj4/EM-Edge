Enter the following details to edit worklist:

  1. Department - The name of the Department associated to the logged in user is displayed. If you have access to the multiple departments, select the desired department name from the list.

  2. WL Name - Enter a valid worklist name, which has been created. Click [List of Values] to view and select the worklist name.

  3. WL Date - Enter the work list date or click [List of Values] function to select a valid date.

  4. WL No. - Enter a worklist number. Click [List of Values] to view and select a valid worklist number which has been created.

  5. Perform Site – Click [List of Values] to select the site. The listed sites are defined in the Perform Site function in the Masters.

  6. Cup – Select the cups as per the test investigations.

  7. Specimen - Enter the Specimen number on selecting the last cup.

  8. Category No – The Category number is displayed, by default.

  9. Ind – Indicator for cup will be displayed as Patient Sample (P) or Quality Sample (Q).

  10. [Clinician Comments] – Click [Clinical Comments] to display the comments, which are entered while registering..

  11. Create Empty Worklist - Click [Create Empty Worklist] to include more than one worklist.
