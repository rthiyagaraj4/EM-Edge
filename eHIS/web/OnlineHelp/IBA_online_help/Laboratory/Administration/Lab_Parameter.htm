Lab Parameter

From this function you can Define Lab parameters applicable to the logged in facility that influence the functioning of the Laboratory module.

This function is used to define or update the parameter details used in the Laboratory module. The criteria for declaring parameter values is to use the default values in transaction functions. The inter module integration setup is set using this function.

The Lab Parameter screen consists of four tabs: General, Blood Unit Parameters, Print Setup, and Letter Formats tabs. You can setup the corresponding details in each tab frame.

Defining Lab Parameters

From Administration, select the Lab Parameter function.

The Lab Parameter screen gets displayed.

  1. The General tab page is displayed by default. Enter the details in the mandatory (*) fields of General tab.

  2. Click ‘Blood Unit Parameters’ tab to define blood bank and blood transfusion related parameters. Enter the details in the required fields of Blood Unit Parameters tab.

  3. Click ‘Print Setup’ tab to define parameters for result printing.

  4. Click ‘Letter Formats’ tab to define various letter formats used in the laboratory department.

    • Format Type – Select the Format type values which are system defined. The created letters are displayed in Recall Patient report. This  letters are dispatched to patient in specific format, if the facility requires clinical history or specimen sample.

  5. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  6. Click [Execute Query] or press [F8] to view the existing records.
