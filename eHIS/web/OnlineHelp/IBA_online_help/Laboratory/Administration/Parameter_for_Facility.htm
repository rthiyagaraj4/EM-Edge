Parameter for Facility

From this function you can Define Laboratory parameters applicable to a specific facility that influence the functioning of the Laboratory module. You can Query for the existing records.

This function is used to define or update the parameter details that are specific to a facility. You can define rules for shift timings, specimen registration details and billing interface for the laboratory. You can also define if the fax results, mail results and print results are valid for a patient class. The letter format required to dispatch reminder letters to patients is also defined. The Parameter for Facility screen consists of General, Print Setup and Letter Format tabs.

Defining Parameter for Facility

From Administration, double-click the Parameters for Facility function.

The Parameters for Facility screen gets displayed.

  1. Select a facility from the list. The facilities associated to the logged in user are listed. This is a mandatory field.

  2. The General tab page is displayed by default.

  3. Click Print Setup tab to define parameters for result printing.

  4. Click Letter Formats tab to define various letter formats used in the laboratory department.

    Letter Formats

    Enter the following details in the Letter Formats tab:

    1. Format Type – Select the desired letter Format Type from the list. The list displays the predefined values.  On selecting the format type, enter the body of the letter format.

  5. Click Logos tab to define the path of the logo file to be used in header and footer of the specimen result report.

  6. Click [Save] or press [F10] to save the created records or click [Clear Record] to clear the entered data.

  7. Click [Execute Query] or press [F8] to search for the existing records.
