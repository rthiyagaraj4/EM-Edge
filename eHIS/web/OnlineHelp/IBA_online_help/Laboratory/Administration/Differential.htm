Differential

From this function you can Define the tests that are used for peripheral Blood Film (Differential) investigation. You can Query for the existing records.

This function is applicable only for Haematology department. The setup for differential count is done using this function. We can set different WBC Cells like Neutrophil, Eosnophil, Lymphosyte, Basophil and Monocyte with the normal value of the cells in percentage with total WBC Count, RBC Cells and Haemoglobin. The system provides a purpose built function for recording the results for Peripheral Blood Film (Differential) investigation. This function assists you in directly entering the results as you view the slide through the microscope. To use the Differential function, certain information required by it needs to be set up. Differential screen consists of the Differential Setup and Define User key Mapping tabs.

Defining Differential setup

From Administration, double-click the Differential function.

The Differential screen gets displayed.

  1. The Differential Setup tab is displayed by default.

  2. Enter the details in the respective fields to setup differential test. <Related Field Description>

  3. Click User Key Mapping tab to invoke the tab page.

  4. Enter the details in the respective fields to define user key mapping for the differential tests.

    Enter the following details:

    1. User – Enter the lab user id in alphanumeric format of 30 characters maximum length to which key mapping needs to be done or select it using [List of Values] function. The listed values are created in ‘System Manager’ module

    2. Differential Test – Enter the Differential test in alphanumeric format of 10 characters maximum length or select it using [List of Values] function for which the key stroke need to defined.

    3. Map Key – Enter the actual key stroke for the tests in alphanumeric format of one character maximum length.

  5. Click [Save] or press [F10] to save the created Differential records or click [Clear Record] to clear the entered data.
