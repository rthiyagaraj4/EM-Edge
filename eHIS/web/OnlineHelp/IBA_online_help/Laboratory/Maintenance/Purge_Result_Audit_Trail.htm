Purge Result Audit Trail

From this function you can Purge or Clear the log records on changes performed on the previous results based on the provided input parameters.

The records created on or before the cutoff date are selected for purging. The log record contains both old and new values. Information not required by the enterprise needs to be cleared on a regular basis. The log records satisfying the input parameters are purged and the query function ‘Who Done It’ will not list any transactions prior to cut off date.

Purging the log records

From Laboratory Maintenance, double-click Purge Result Audit Trail function.

The Purge Result Audit Trail screen gets displayed.

  1. Enter the cutoff date to purge the log records in the ‘Purge up to Date’ field or select the date using the Calendar function.

  2. Click [Save] or press [F10] to purge the log records or [Clear Record] to clear the entered data.
