Diet Category Order Type Link

From this function you can Associate an Order Type to a system-defined Diet Category, if the Order Type is not linked and Query for an existing system defined Diet Category.

The Diet Category Order Type Link function is used to query for the system-defined Diet Categories such as, Enteral Diet, Normal Diet and Therapeutic Diet pertaining to a Hospital. You can associate an Order Type to the Diet Category, if it is not associated. The Diet Statistics are based on the Diet Categories.

Querying a Diet Category Order Type Link

From the Setup, select the Diet Category Order Type Link function.

The Diet Category Order Type Link screen gets displayed.

  1. Click [Query] to search for an existing Diet Category associated to Order Types. The Diet Categories are system-defined. You can select an Order Type from the list and associate it to the Diet Category, if it is not linked. The Order Types defined in the Order Entry and Result Reporting module get listed.

  2. Click [Apply] to save the records or [Reset] to clear the entered details.
