Diet Order Status

From this function you can Query the diet order status for the ward/clinic.  

The Diet Order Status function is used to check the diet order status for the ward/clinic. The system shows the mother accompanied child list count if Mother accompanied child diet is placed for a normal patient and less than six months patient is placed.

Querying Diet Order Status

From the Queries, select the Diet Order Status function.

The Diet Order Status screen gets displayed.

  1. Select the Patient Class from drop down list. The listed values are ‘Inpatients’, ‘Outpatients’, ‘Daycare’, ‘Emergency’ and ‘External’. This is a mandatory field.

  2. The ward is displayed based on selection of Patient class. Click [?] function to select the Clinic in which the patient is admitted. The records matching the selected Clinic get displayed.

  3. Click the Calendar function to select the Serving Date. The system date is displayed as Serving Date, by default.

  4. Click [Select] to search for the Diet Order Status.

  5. Ward Or Clinic Description, Total No. of Patients, Total Orders, Pending Orders, Attendant Orders, Full NBM, Diet Type gets listed.
