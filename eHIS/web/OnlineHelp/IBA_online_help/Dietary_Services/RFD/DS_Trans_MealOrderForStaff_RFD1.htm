Meal Order

You can record the meal order for staff in the facility using this tab.

 

Enter the following to create Meal Order for Staffs:

 

  1. Meal Category - Select a Meal Category from the list. The system-defined values are Doctor On-Call, OT/Paramedic Staff, On Call Staff, Staff and Students. This is a mandatory field.

  2. Practitioner Name – This field is displayed on selection of ’Doctor On-Call’ in Meal Category field. Select the required practitioner name from the lookup. This is a mandatory field.

  3. Diet Request Location – Select the Diet Request Location from the drop down list. Based on the master setup, the location code is selected as “Default diet request location by default. The values defined in the masters are listed in Meal Order.

  4. Meal Category - Select a Meal Category from the list. The system-defined values are On Call Staff, Staff and Students. This is a mandatory field.

  5. Order Date From – Click the Calendar function to select the ‘From’ date from when the selected meal order is applicable for staffs. The current date is displayed by default. This is a mandatory field.

  6. Order Date To – Click the Calendar function to select the ‘To’ date upto when the meal order is applicable for staffs. The current date is displayed by default.

  7. Diet Type - Select the Diet Type from the list. The values defined using the function ‘Diet Type’ get listed. The Meal Orders for Staffs are placed for the selected Diet Type.  This is a mandatory field.

  8. Meal Class - Select the Meal Class from the list. The values defined using the function ‘Meal Class’ get listed. The Meal Orders for Staffs are placed for the selected Meal Class. This is a mandatory field.

  9. Meal Type - Select the Meal Type from the list. The values defined using the function ‘Meal Type’ get listed. The Meal Orders for Staffs are placed for the selected Meal Type. This is a mandatory field.

  10. No. of Orders – Enter the total Number of Meal Orders in numeric format of six digits maximum length. This indicates the quantity of meals to be prepared.

  11. [Select] - Click [Select] to display the entered records in a tabular format on the right hand side of the screen. The Diet Types are displayed as a hyperlink. On clicking the Diet Type hyperlink, Diet Type, Meal Class, Meal Type and number of orders get displayed in the respective fields. You can change the Diet Type, Meal Class, Meal Type and number of orders. Click [Select] to modify the details. Click [Apply] to save the records. The recorded details get displayed while selecting the respective details in ‘Document Meal Served’ tab. Click [Delete] to get dialog with OK and Cancel buttons. Click [OK] button to delete the record.
