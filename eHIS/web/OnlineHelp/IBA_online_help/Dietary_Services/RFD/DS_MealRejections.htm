Related Field Description

  1. Serving [From-To] – Select the serving date range using calendar function. Maximum date range allowed is 90 days. By default, the highest date value of the meal rejection record is displayed in Serving From Date field and Least date value of the meal record is displayed in Serving To Date field.

  2. Patient Class – Select the Patient Class from drop-down. The listed values are ‘All’ ‘Inpatient’, ‘Outpatient’, ‘Emergency’ and ‘Department’. By default, the value ‘All’ is selected.

  3. Acknowledge Status - Select the Acknowledgement Status from drop-down. The listed values are ‘All’ ‘No’ and ‘Yes’. By default, the value ‘No’ is selected.

  4. Patient ID – Select the Patient ID using lookup function. The patient IDs defined in the masters are listed.

  5. Patient Status - Select the Patient Status from drop-down. The listed values are ‘Active’ ‘Discharged’ and ‘All’. By default, the value ‘All’ is selected.
