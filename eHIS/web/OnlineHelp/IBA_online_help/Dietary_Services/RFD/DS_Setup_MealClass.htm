Enter the following details to create a Meal Class:

 

  1. Meal Class – Enter the Meal Class in alphanumeric format of two characters maximum length. The Meal Class has to be unique and cannot be changed after creation. This is a mandatory field.

  2. Long Description - Enter the long description of Meal Class in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description of Meal Class in alphanumeric format of 15 characters maximum length.  This is a mandatory field.

  4. Applicable for Edit Menu – Select the checkbox to display the Edit Menu option for the Meal class in Place Diet Order screen. If you do not select the checkbox, the Edit Menu option is  not displayed  in Place Diet Order screen.

  5. Applicable for Mother Accompany Child - Select the checkbox to display the meal class in the Place Diet Order screen to place the Mother Accompany Child Order. By default, the checkbox is selected.

  6. Default Meal Class For Bed Class – This list box displays the checkboxes with the Bed Classes names defined in the Bed Class screen in IP module.  Select checkbox to associate the meal class with selected bed class. The selected bed class is displayed by default in the Place Diet Order screen.

  7. Enabled – Select the check box to enable the Meal Class in transaction functions. The check box is selected, by default.
