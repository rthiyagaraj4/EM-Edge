Related Field Description

 

  1. Name – Enter Name or select the name using lookup function. The names defined in the master are defined. This is a mandatory field.

  2. User ID – The User ID is displayed based on the selected Name.

  3. Edit Meals – Select this checkbox to provide access right for Edit Meals option in Place Diet Order. By default this checkbox is selected.

  4. Nutrient Supplement - Select this checkbox to provide access right for Nutrient option in Place Diet Order. By default this checkbox is selected.

  5. Food Dislike - Select this checkbox to provide access right for Food Dislike option in Place Diet Order. By default this checkbox is selected.

  6. Special Food Order - Select this checkbox to provide access right for Special Food Order option in Place Diet Order. By default this checkbox is selected.
