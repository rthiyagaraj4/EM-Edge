Enter the following details to define a Diet Type:

 

  1. Diet Type – Enter the Diet Type in alphanumeric format of 20 characters maximum length. The Diet Type has to be unique and cannot be changed after creation. This is a mandatory field.

  2. Long Description - Enter the long description of the Diet Type in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description of the Diet Type in alphanumeric format of 45 characters maximum length. This is a mandatory field.

  4. Age Group – Select the Age Group from the list. The values defined using the ‘Age Group’ function in Application Masters module get listed. This is a mandatory field.

  5. Default Diet Type – Select the check box to set the Diet Type as the Default Diet Type for a particular Age Group. Clear the check box, if the Diet Type is not to be defaulted.

  6. Default Feed Type – Select the default Feed Type from the list. The values defined using the ‘Feed Type’ function get listed. The Default Feed Type value selected gets displayed while placing a diet order using the function ‘Place Diet Order’.

  7. Diet Cycle Type – Select the appropriate value as ‘Repeat’ or ‘One Time’ from the list. ‘Repeat’ is displayed, by default. On selecting ‘Repeat’, the Diet Type can be repeated for the number of times defined by the Dietitian in ‘Menu Cycle’ function. On selecting ‘One Time’, the Diet Type cannot be repeated.

  8. Order Catalog Code – Select the Order Catalog using [?] function to define additional attributes for a Diet Type. The Order Catalogs are defined in ‘Order Catalog’ function in Order Entry and Result Reporting module. This is a mandatory field.

  9. Diet Category - Select the Diet Category from the list. The values defined using the  ‘Diet Category’ function get listed. This is a mandatory field.

  10. Inpatient Diet Type - Select the checkbox to display Inpatient Meal order in the Meal Type drop-down list in Place Diet Order screen for inpatients. By default the check box is selected.

  11. Daycare Diet Type – Select the checkbox to display Day Care Meal order in the Meal Type drop-down list in Place Diet Order screen for day care patients. By default the check box is selected.

  12. Outpatient Diet Type – Select the checkbox to place diet based on the outpatient diet type.

  13. Applicable for MAC -  Select the check box to associate Diet Type with Diet Category for MAC Order. While placing a Diet Order for MAC, only the Diet Category and Diet Type that are applicable for MAC are displayed in the respective drop-down list in the Place Diet Order screen. By default, the check box is selected.

  14. Applicable For Staff Order – Select the check box to associate Diet Type with staff. While placing order in Meal Order for Staff screen, only the Diet Category and Diet Type that are applicable for Staff Order are displayed in the respective drop-down lists. By default, the check box is selected.

  15. Emergency Patient Diet Type - Select the checkbox to place diet based on the Emergency patient diet type.

  16. Enabled - Select the check box to enable the Diet Type in Transaction functions. The check box is selected, by default.
