Menu Cycle

From this function you can Define a Menu Cycle by associating the Menu to Diet Type, Meal Class and Menu Schedule Cycle. You can Query for a specific Menu Cycle based on a particular Diet Type associated to a Meal Class.  

The Menu Cycle function helps in defining Menu Cycle to be followed in the facility. A menu cycle is defined for a particular Diet Type and Meal Class. This function is used to follow a menu cycle for a specific period until the number of days specified. For example, if the menu cycle is specified as 3 and the Reference Day number is 2, this indicates that the respective menu of a Meal Type is applicable from the second day if the meal plan date is defined as the current date.  The second Menu of the same Meal Type is applicable for the third day and the third Menu of the same Meal Type is applicable for the first day. The cycle gets repeated after the number of Menu Schedule Cycle defined. A schedule cycle set up for each diet type and meal class indicates a defined menu cycle it has to follow. You cannot modify the defined Menu Cycle in ‘Place Diet Order’ function in Transactions, but can modify a Diet Cycle for a particular patient in ‘Place Diet Order’ function. You can modify a Menu Cycle only in this function.

Defining a Menu Cycle

From the Admin Setup, select the Menu Cycle function.

The Menu Cycle screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a created Menu Cycle or [Reset] to clear the entered details.

  4. Click [Query] to search for the existing Menu Cycles.
