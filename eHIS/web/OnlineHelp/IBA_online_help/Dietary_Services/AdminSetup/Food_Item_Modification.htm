Food Item Modification

From this function you can modify the food items for multiple diet type in a single update.

The Food Item Modification function allows you to modify the food items for multiple diet type in a single update.

Modifying Food Item Modification

From the Setup, select the Food Item Modification function.

The Food Item Modification screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to display respective menus based on the filter criteria. Menu and Menu description are displayed.

  3. Select the check against the required Menu.

  4. Click [Apply] to replace the existing Diet Type item with the new item or [Reset] to clear the entered details.
