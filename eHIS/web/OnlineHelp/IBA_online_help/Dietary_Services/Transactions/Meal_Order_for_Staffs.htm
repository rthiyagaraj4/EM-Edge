Meal Order for Staffs

From this function you can Define Meal Order for Staffs in the facility. You can also Record the meals served.

The Meal Order for Staffs function is used to place meal order for the staff members, on call staff and the students practicing in the facility. This function provides option to place cumulative meal orders for a specified date range. You can associate the Diet Type, Meal Class, Meal Type and the Number of Orders to a Meal Category to be served for the specified date range.

Defining Meal Orders and Recording details

From the Transactions, select the Meal Order for Staffs function.

  1. Click [Create] to invoke the Create screen.

  2. The Meal Order for Staffs screen gets displayed with two tabs. Meal Order tab is enabled, by default.

Meal Order

Document Meal Served

  1. Enter the details in mandatory (*) fields.

  2. Click [Apply] to save the records or [Reset] to clear the entered details.
