Checklist for Requestor

From this function you can Define the Requestor with the Checklist created for Medical Reports.

This function allows you to map the Requestor with the checklist created in ‘Checklist for Medical Report’ function.

You can map maximum 10 number of checklist against one requestor. The same checklist can be mapped for another requestor also.

Defining Checklist for Requestor

From MR Code Setup, select the Checklist for Requestor function.

The Checklist for Requestor screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Requestor in alphanumeric format of 25 characters or select using the [?] lookup. This field is mandatory.

  3. Enter the text for searching the check list in alphanumeric format in the Search Checklist field. Special characters are allowed. Also, select the Criteria from the drop-down list. The list includes Starts With, Contains, and Ends With. By default, the value 'Starts With' appears.

  4. Click [Search], the checklists matching the search criteria appear.

  5. From the search results, select the check box against the checklist to be mapped with the Requestor and click [Apply]. The selected check lists get mapped to that requestor. You can define maximum 10 number of checklist against one requestor.

  6. Click [Query] to search for existing checklists mapped for the requestor.
