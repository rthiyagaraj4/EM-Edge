Medical Event Type

From this function you can Create a new Medical Event Type applicable to the healthcare facility and Query for a desired medical event.

Medical Event Type function is used to define all the Medial Event Types applicable in the healthcare facility. Using the medical events, care providers can record any significant medical event against a patient and such an event can occur outside the facility.

Creating a Medical Event Type

From MR Code Setup, select the Medical Event Type function.

The Medical Event Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the data in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Medical Event Type created or click [Reset] to clear the entered data.

  4. Click [Query] to search for existing Medical Event Type within the facility.
