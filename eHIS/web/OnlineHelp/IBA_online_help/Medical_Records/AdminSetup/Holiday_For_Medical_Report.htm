Holiday For Medical Report

From this function, you can Create and Query the holidays for required date range so that holidays are considered during Medical Report request processing.

This function allows you to create and query the holidays for required date range so that holidays are considered during Medical Report request processing. You can delete the created holiday record if not required.

Creating Holiday for Medical Report

From MR Setup > select the Holiday For Medical Report function.

The Holiday For Medical Report screen gets displayed.

  1. Enter the From Date by clicking the calendar icon. This field is mandatory. The From Date should be greater than or equal to Current Date.

  2. Enter the To Date by clicking the calendar icon. This field is mandatory. The Date Range cannot be greater than 30 days.

  3. Click [Select].  The Date, Reason for Holiday and Select columns appear.

  4. Enter the reason for holiday in the ‘Reason for Holiday’ column against the respective date. It is mandatory to enter the holiday reason.

  5. Select the check box against the Holiday set record.

  6. Click [Apply] to save the changes done or [Reset] to clear the entered data.

  7. Click [Query] to view as well as to edit the Holidays already created for the Medical Report.

  8. To delete any holiday record, select the check box against the record and click [Delete].
