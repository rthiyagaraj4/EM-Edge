Patient Disposal/Inactive Criteria

From this function you can Set criteria to mark patient ID as disposed or inactive.

This function allows to set the criteria to mark the Patient ID who are not having any active encounter or inactive or no visit, as disposed. Set the criteria by mapping Specialty and Year to be considered while disposing or inactivating patient IDs.  

Setting Criteria for Disposing/Inactivating Patient IDs

From MR Setup > MR Admin Setup, select the Patient Disposal/Inactive Criteria function.

The Patient Disposal/Inactive Criteria screen gets displayed.

  1. By default, the Action ‘Disposal’ is displayed in Action drop-down list. Select Disposal value to set criteria to dispose patient ID or select the Inactive value to set criteria to mark the patient as inactive.

  2. All Specialty names (active and inactive) available in the Facility are listed under the Specialty Name column.

  3. Enter the number of years in the Duration (Year) text box against the required Specialty. You can enter up to 99 years. Only numeric value is allowed. Decimal values are not allowed.

  4. Click [Apply] to save the settings done.
