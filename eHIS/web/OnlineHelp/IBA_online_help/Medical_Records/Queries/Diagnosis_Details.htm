Diagnosis Details

From this function you can Query and get a statistical view of the number of patients for whom a particular diagnosis has been recorded and the status of the diagnosis recorded against each of the patient.

The status of a diagnosis is either Active or Closed. The other details listed are the Patient ID, Patient Name, Birth Date of the Patient, the Onset Date of the Problem/Diagnosis recorded, the Location where it was recorded and the Practitioner who recorded the Diagnosis. The view is filtered based on the query criteria. On selecting a patient, all the diagnoses recorded for that patient are displayed based on the query criteria specified after selecting the patient.

Querying for Diagnosis Details

From the MR Queries, select the Diagnosis Details function.

The Diagnosis Details screen gets displayed.

  1. Enter the diagnosis code in alphanumeric format of five characters maximum length or select the code using [?] function. This field is mandatory. <Related Field Description>

    The screen displays the tabs 'Custom List', 'Search' and 'Select'.

    Custom List

    The Diagnosis and Procedure groups created are listed in the Custom List from which you can select a diagnosis by clicking the hyperlink on the names of the groups and a custom list gets displayed.

    Click the hyperlink of the Code to select and the selected values appear in the Dagger Code field.

    Search

    Enter the search criteria and click [Search] to display the codes based on the entered criteria. Click the hyperlink of the Code to select it to appear in the 'Dagger Code' field.

    Select

    This is another way to search for the Codes, where there are four quadrants representing the levels of classification. The first quadrant represents the broadest level of classification. On clicking any of the classifications, all the broadest possible classifications existing in the selected sections appear in the second quadrant. Based on the selected hyperlinks of broader classification, the fourth quadrant will have the final individual diagnoses. Click the hyperlink to select the diagnosis to appear in the Dagger Code field.

  2. Click [Search] to select a specific record or [Clear] to clear the records displayed in the result area.

  3. Click Patient ID hyperlink to view the Problem List of the patient.

  4. Click [Reset] to clear the entered data.
