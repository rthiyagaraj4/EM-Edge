View Audit Trail for Modified Medical Report

From this function you can View audit trail for modified Medical Reports.

This function allows to check the audit trail for the ‘Modify Requested’ medical records after delivered.

Viewing Audit Trail for Modified Medical Report

From the MR Queries, select the View Audit Trail for Modified Medical Report function.

The View Audit Trail for Modified Medical Report screen gets displayed.

  1. Enter or select the patient ID using the lookup.

  2. Select the Modified Date From and To range by clicking the calendar icon. The date is displayed in DD/MM/YYYY format. This field is mandatory.

  • Modified Date From/To cannot be greater than to system date/time

  • Modified To date should be greater than or equal to From date

  1. Click [Search] to view the modified medical report records or [Clear] to clear the records displayed in the result area.
