View BHT Status

From this function you can View the Bed Head Ticket (BHT) status for discharged patients.

This function is used to query about different BHT status for the patients based in encounter date range and encounter type. The different BHT statuses are ‘Pending’, ‘Overdue’, ‘Delivered’, ‘Received’, and ‘Returned’.

Querying BHT Status

From the MR Queries, select the View BHT Status function.

The View BHT Status screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to view records matching the search criteria.

  3. The search results include Patient ID, BHT status, Encounter Date, Encounter ID, Encounter Type, Location, Discharge Date, Specialty, Practitioner and Diagnosis Details.

  4. Click Encounter ID hyperlink to view the encounter details of the patient in the Encounter Details dialog.

  5. Click Diagnosis Details hyperlink to view the diagnosis details in the Diagnosis Details dialog if available.
