Term Set Criteria for Patient Class

From this function you can Associate a Term Set Criteria for a Patient Class.

You can select the list item for the Patient Class and mark the default for the Patient Class. This function is used for creating the restriction in the list item. Once the default value is marked for the criteria other check boxes are disabled.

Associating Term Set Criteria for a Patient Class

From MR Code Setup > MR Terminology Setup, select the Term Set Criteria for Patient Class function.

The Term Set Criteria for Patient Class screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Search] to display the Patient Class based on the entered criteria.

    • Select - Select the check box to list the values in the Transaction screens.

    • Default - Select the check box to mark the default value for the Patient Class. Once the default value for a criterion is marked for the Patient Class, the check box is disabled.

  4. Click [Apply] to save the recorded details or [Reset] to clear the entered data.

  5. Click [Query] to search for existing Accuracy record.
