Accuracy for Patient Class

From this function you can Create an association of an Accuracy to Patient Classes and Query for a desired Accuracy for Patient Class record.

This function is used to restrict accuracy to a patient class. The user can associate the required accuracy applicable to a particular patient class. The accuracy is defined in the ‘Accuracy’ function.

Creating an association of Accuracy to Patient Class

From MR Code Setup > MR Terminology Setup, select the Accuracy for Patient Class function.

The Accuracy for Patient Class screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Accuracy from the list of valid values. This field is mandatory.

  3. Click [Search] to display Patient Classes.

  4. Select the check box against the required Patient Class if the selected Accuracy is applicable to the Patient Class.

  5. Click [Apply] to save the created Accuracy for Patient Class record or [Reset] to clear the entered data.

  6. Click [Query] to search for existing Accuracy for Patient Class records.
