Diagnosis/Procedure Coding

From this function you can Code a diagnosis or procedure already recorded by a practitioner for an encounter of a patient.

This function is accessed only if the Procedure/Diagnosis Recoding check box is selected in Inpatient and Outpatient tabs in Operator Station and Operator Station Restriction functions in Application Masters module based on the operator station privilege of the user. This function is dependent on the MR Parameter and Recode Privilege Setup function. The status of coding is displayed based on the different symbolic indicators.

Coding Diagnosis/Procedure

From the MR Transactions, select the Diagnosis/Procedure Coding function.

The Diagnosis/Procedure Coding screen gets displayed.

  1. Select the Encounter Type and Status and click [Search] to select encounters based on the entered criteria or enter the search criteria and click [Search] to select a specific encounter.

  2. Click the [+] hyperlink of the required encounter to view the Encounter Details.

  3. Select the check box against the required encounter and click [Select] to display the Diagnosis/Procedure Coding screen.

  4. The diagnosis or procedure recorded for the patient is displayed.
