Create Patient File

From this function you can Create a Patient File or Medical Record for a particular patient.

A patient file is created during Patient Registration, Inpatient Admission or Outpatient Visit Registration. If the patient file is not created during any of these events, this function is used to create the patient file.

 

Process Flow Diagram

Creating a Patient File

From the MR Transactions, select the Create Patient File function.

The Search Patient Screen gets displayed.

  1. Search the required patient and click the Patient ID hyperlink to select the patient to create a Patient File. An alert message gets displayed if the patient file is not available for the logged in facility. Click [OK] to create a patient file and the Create Patient File screen is displayed. Click [Cancel] to cancel creating a file. If Patient File is available for the logged in facility, file details are displayed. Click [Create] to create another Patient File for the patient.

    1. File Type – The File Type of the existing patient file is displayed.

    2. Active Volume Number – The Active Volume Number of the respective File Type is displayed.

    3. File Current Location – The Current File Storage Location of the patient file is displayed.

    4. Current File Status – The Current File Status of the patient file is displayed.

    5. File Created Date/Time – The patient File Created Date and Time is displayed.

    6. File Created By – The User ID of the person who created the patient file is displayed.

  2. Enter the details in the mandatory (*) fields in the Create Patient File screen. <Related Field Description>

  3. Click [OK] to generate a file and Patient File Number is generated or [Cancel] to cancel file generation.

  4. The Internal Reports screen gets displayed as defined for the Patient Series assigned to the selected patient. Click [OK] to generate Internal Reports or [Cancel] to exit from the screen. On generating the reports, the control shifts back to the ‘Search Patient’ function.

 

MR Section has to be defined for the Patient Series of the selected patient to create a patient file.
