Record Patient Medical Events

From this function you can Record the previous medical events of a patient that have occurred earlier outside the facility.

Process Flow Diagram

Recording Patient Medical Events

From the MR Transactions, select the Record Patient Medical Events function.

The Search Patient Screen gets displayed.

  1. Search the required patient and click the Patient ID hyperlink to select the patient and record Medical Events. The Record Patient Medical Events screen gets displayed.

  2. Any Medical Events already recorded for the selected patient are displayed in the top part of the screen.

    1. Event Type – The recorded Event Type for the patient is displayed. These values are defined in the function ‘Medical Event Type’.

    2. Event Details – The hyperlinked Event Details as recorded is displayed. Click the hyperlink to view the details.

    3. Start Date – The Start Date of the recorded medical event is displayed.

    4. End Date – The End Date of the recorded medical event is displayed.

    5. Confirmed – Confirmation of the recorded medical event (if confirmed) is displayed.

  3. Enter the details in the mandatory (*) fields to record a new Medical Event for the patient. <Related Field Description>

  4. Click [Record] to record the Medical Event or [Clear] to clear the entered data. Click [Cancel] to exit from the screen.

  5. On successfully recording the Medical Event, select the Event Type hyperlink to record the End Date of the Medical Event. The medical event details are displayed in the lower part of the screen. You cannot change the displayed values if the details of the Medical Event are confirmed. Click Event Details hyperlink to view the remarks recorded for the patient.
