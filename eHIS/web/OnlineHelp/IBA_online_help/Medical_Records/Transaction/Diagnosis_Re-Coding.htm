Diagnosis Re-Coding

From this function you can Re-code a diagnosis already recorded by a practitioner for an encounter of a patient.

On selecting the Patient ID, the system lists out all encounters that the patient had with the facility. On selecting the desired encounter, the system displays a list of all diagnoses recorded for the chosen encounter. You can select the required diagnosis from the list and re-code with a different diagnosis code.

 

Process Flow Diagram

Re-Coding Diagnosis

From the MR Transactions, select the Diagnosis Re-coding function.

The Diagnosis Re-coding screen gets displayed.

  1. Select the Encounter Period and Encounter Type and click [Search] to display all encounters based on the entered criteria or enter the query criteria and click [Search] to display and select a specific encounter.

    1. Encounter Date – The date of the encounter is displayed for the selected patient.

    2. Encounter Type – The type of encounter of the patient had is displayed.

    3. Encounter ID - The Encounter ID of the patient is displayed. This field is hyperlinked.

    4. Location – The Location where the patient is treated is displayed.

    5. Specialty – The Specialty under which the patient is treated is displayed.

    6. Room – The Room occupied by the patient is displayed.

    7. Bed – The Bed occupied by the patient is displayed.

    8. Practitioner – The name of the Practitioner treating the patient is displayed.

  2. Click the Encounter ID hyperlink of the required encounter to display the Diagnosis Re-coding screen. The active problem recorded for the patient is displayed. Click the Description hyperlink to view details of the re-coded diagnosis, if any.

    Record Diagnosis

    1. Original Diagnosis Code – The Original Diagnosis Code recorded is displayed.

    2. Old Diagnosis Code – The Old Diagnosis Code recorded is displayed.

    3. Re-coded Diagnosis – The Re-Coded Diagnosis code recorded is displayed.

    4. Re-coded By – The name of the person who re-coded the diagnosis is displayed.

    5. Re-coded Date – The date and time of re-coding is displayed.

    6. Remarks – The Remarks recorded is displayed (if any).

    7. Supporting Diagnosis – The Supporting Diagnosis recorded is displayed (if any).

  3. Enter the details in the mandatory (*) fields to re-code diagnosis. <Related Field Description>

  4. Click [Record] to re-code diagnosis details or [Clear] to clear the entered data. Click [Cancel] to cancel recoding diagnosis and exit from the screen.
