The following details are displayed:

 

  1. ID – A unique ID for the Terminology Set is displayed, which cannot be edited. This field is mandatory.

  2. Description - Enter the Terminology Description in alphanumeric format of 60 characters maximum length. This field is mandatory.

  3. Enabled – Select the check box to enable the Terminology Set or clear the check box to disable the Terminology Set. The check box is selected, by default.

 

Term Set Attributes

  1. Diagnosis Specific - Select the check box if the Terminology Set is restricted specifically to Diagnosis.

  2. Procedure Specific - Select the check box if the Terminology Set is restricted specifically for Procedures.

  3. Build Statistics - Select the check box if a statistical report has to be generated for the codes classified under the Terminology Set.

  4. Build Episode of Care - Select the check box if the Terminology Set allows Building Episode of Care.

  5. Classification Levels - Select the number of Levels of Classification for the Terminology Set from the list of valid values. The value selected is one, by default. Depending on the value selected, the system will display number of levels to define the description details and the code length for the Classification Levels.

 

Term Code Control

  1. Notification - Select the check box if the Terminology set consists of Terminology Codes which are notifiable.

  2. Sensitivity - Select the check box if the Terminology Set consists of Terminology Codes which are sensitive in nature.

  3. Age Group - Select the check box if Age Group is applicable for the Terminology Set.

 

General Attributes

  1. Diagnosis Nature - Select the check box if the Nature of Diagnosis is applicable for the Terminology Set.

  2. Severity - Select the check box if Severity is applicable for the Terminology Set.

  3. Diagnosis Factors - Select the check box if the Diagnosis Factors are applicable for the Terminology Set.

  4. Encounter Stage – Select the check box if Encounter Stage is applicable for the Terminology Set.

  5. Onset Type - Select the check box if Onset Type is applicable for the Terminology Set.

  6. Priority - Select the check box if Priority is applicable for the Terminology Set.

  7. Cut Off Period – Enter the Cut-Off period for the Terminology Set in numeric format of three characters maximum length. This field is mandatory.

  8. Diagnosis Classification Applicable - Select the check box if Diagnosis Classification is applicable. Clear the check box if Diagnosis Classification is not applicable for the Terminology Set.

  9. Applicable for Recoders - Select the check box to list the Terminology Set in the Diagnosis/Procedure Set field of Diagnosis/Procedure Recoding function while recoding or creating a new diagnosis/procedure. This is a one time setup.

  10. Diagnosis Classification Mandatory - Select the check box to control each of terminology set to be a mandatory field during Record Diagnosis. This check box is enabled only if the ‘Diagnosis Classification Mandatory’ check box is selected in MR Parameter screen.

  11. Free Text Applicable - Select the check box to indicate free text recording is applicable for the selected Terminology Set while recording diagnosis. This is a one time setup.

Classification Level Details

  1. Levels - The number of Classification Levels are displayed.

  2. Description - Enter the description in alphanumeric format of 60 characters maximum length. This field is mandatory.

  3. Code Length - Enter the code length in numeric format of two digits maximum length. This field is mandatory.
