Enter the following details:

  1. Reference No – Enter the Reference Number in alphanumeric format 10 characters of maximum length. This field is mandatory.

  2. Date – Enter the event date in the dd/mm/yyyy format. This field is mandatory.

  3. Category – Select the category from the list. The list values are pre-defined. The list values displayed are Others, Report, Correspondence and Adhoc. This field is mandatory.

  4. Remarks – Enter the Remarks in alphanumeric format of 2000 characters maximum length. This field is mandatory.
