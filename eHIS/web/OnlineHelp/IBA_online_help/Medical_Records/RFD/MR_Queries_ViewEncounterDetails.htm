Enter the following details to view encounter details of a patient:

  1. Patient ID – Click [?] to search and select the patient ID for which you want to view the encounter details.

  2. Facility – Select a facility from the drop-down list, for which you want to view the encounter details. By default, the logged-in facility is displayed. If you select the value “Select”, all the encounter details for the patient will be displayed, grouped by facility.

  3. Encounter Date [From-To] - Enter the Encounter date period in dd/mm/yyyy format or select the date from the Calendar function.

  4. Encounter Type - Select the Encounter Type from the encounter type list. The listed values are 'Daycare', 'Emergency', 'Inpatient', 'Outpatient' and 'External'.

  5. Encounter ID - Enter the Encounter ID in alphanumeric format of 12 characters maximum length.

  6. Order By - Select the order in which the records have to be displayed. The listed values are 'Encounter Date' and 'Encounter Type'.
