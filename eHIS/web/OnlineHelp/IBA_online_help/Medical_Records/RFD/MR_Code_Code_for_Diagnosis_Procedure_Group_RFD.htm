Enter the following details:

 

  1. Diagnosis/Procedure Group - Select the Group from the list. The groups defined in the function 'Diagnosis/Procedure Group' is populated in the list. This field is mandatory. On selecting the Diagnosis/Procedure Group, the Type, Code Scheme and Facility of the selected Group is displayed.

  2. Search By - Select an option from the list to search for the ICD code by either Code or Description.

  3. Criteria - Select the criteria from the list to search for the ICD Codes. The list displays 'Starts With', 'Ends With' and 'Contains'.

  4. Search Desc - Enter the Description of search in alphanumeric format of 70 characters maximum length.
