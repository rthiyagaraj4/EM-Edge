The following details are displayed:

 

Patient Details

Details like the Patient Name, Age, Gender, etc. of the patient that were captured while registering the patient are displayed. If the patient has not been registered to any of the facilities of the enterprise, the Patient ID is not displayed for the patient.

 

Request Details

The Request details as entered in the 'Requester Details' Tab of the 'Medical Board' main screen is displayed and disabled. The request ID of the patient is displayed as the Registration No. The fields cannot be updated except the 'Consent Letter' field to select the availability of consent letter. Click [Applicant Address] to view the contact address details of the applicant. The contact details screen is disabled and the fields cannot be updated.

 

Charges

The Charge Details are displayed as recorded in the 'Other Details' Tab of the 'Medical Board' main screen. The fields can be updated.

 

Enter the following details:

  1. Application Status - Select the status of the Application as 'Complete' or 'Incomplete'. The listed values are predefined. This field is mandatory. On selecting the value as 'Complete', the 'Application Completion Date' is displayed as the current system date and time in read-only text. Once the status is marked as Complete, the status of the Medical Board is displayed as 'Status Completed'.

  2. Collect Date - Select the Date on which the Medical Report can be collected from the calendar function. This field becomes mandatory on selecting the 'Application Status' as 'Complete' and is disabled on selecting the 'Application Status' as 'Incomplete'.

  3. Required Docs - Enter the details of the Documents required for the Medical Report in alphanumeric format of 200 characters maximum length.
