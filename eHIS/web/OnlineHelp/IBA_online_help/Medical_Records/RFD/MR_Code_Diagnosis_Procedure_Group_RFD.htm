Enter the following details to create a Diagnosis or Procedure Group:

  1. Group Type - Select the required Group Type from the list. The Group Types listed are Diagnosis and Procedure. This field is mandatory.

  2. Code Scheme - On selecting the Group Type value, the Code Scheme is displayed automatically. The Code Scheme is defined for Group Type in MR Parameter setup.

  3. Code - Enter the code for the Diagnosis/Procedure Group in alphanumeric format of four characters maximum length. This field is mandatory.

  4. Description - Enter the description for the Diagnosis/Procedure Group in alphanumeric format of 30 characters maximum length. This field is mandatory.

  5. Specialty - Select the Specialty from the list of valid values. The values are defined in the Specialty function of Application Masters module.

  6. Facility - Select the facility from the list of valid values. The facilities are defined in the Facility function of System Manager module.

  7. Practitioner - Select the Practitioner using [?] function. If the Specialty is selected, the practitioners associated with the selected Specialty are displayed. If Specialty is not selected, all the practitioners of the logged in facility are populated.
