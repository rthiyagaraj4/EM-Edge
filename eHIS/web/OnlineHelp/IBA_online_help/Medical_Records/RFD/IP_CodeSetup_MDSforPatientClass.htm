Enter the following details:

 

  1. Min Data Set - Select the minimum data set from the list. You can select one of the following system-defined values:

- Charts

- Clinical Notes

- Diagnosis

- Orders

By default, ‘Charts’ will be displayed. This is a mandatory field.  

  1. Terminology Set - Select the Terminology Set from the list. The list contains all the Terminology Set defined in the Terminology Set Master. The values are displayed only on selecting 'Diagnosis' from the Min Data Set list. This is a mandatory field.

  2. Patient Class - The system defined Patient Classes are displayed.

  3. Applicable - Select the 'Applicable' check box to select the required Patient Class applicable to the Terminology Set. On selecting the 'Applicable' check box, the 'Mandatory' check box gets enabled.

  4. Mandatory - Select the 'Mandatory' check box to indicate that the selected Min Data Set is applicable and mandatory for the patient class or clear the check box to make it optional.

  5. Stage – Indicates the function for which the criteria for minimum data set is to be checked. The stages for the various patient classes are as follows:

Note The Stage field is a drop-down list, however only the administrators will be able to change any of these values.

 

6. Principal Diagnosis – This check box column is displayed only when you select the Minimum Data Set as ‘Diagnosis’. The check boxes for all the patient classes are selected by default. If they are selected, the system will check whether any principal diagnosis is recorded against the encounter during the applicable checkout or discharge stages. Else, the system will check only if any diagnosis is recorded.
