Enter the following details:

  1. Encounter Type – Select the required encounter type from the drop-down list. The list values include Inpatient, Daycare, Emergency and Outpatient. By default, the value ‘Inpatient’ is displayed.

  2. Nursing Unit – This field appears only if the selected encounter type is ‘Inpatient’ and ‘Day care’. Select the required nursing unit code from the drop-down list.  

  3. Location Type – This field appears only if the selected encounter type is ‘Outpatient’ and ‘Emergency’. Select the required location type from the drop-down list.

  4. Location – This field appears only if the selected encounter type is ‘Outpatient’ and ‘Emergency’. Select the required location using [?] function. The lookup value is populated based on the selected location type.

  5. Specialty – Select the required specialty from the drop-down list.

  6. Admission Date From and To – Enter the Admission From and To date by clicking calendar icon or enter manually in the dd/mm/yyyy format. This field appears only if the selected encounter type is ‘Inpatient’ and ‘Daycare’.

• ‘To Date’ should be Greater than or equal to ‘From Date’.

  1. Visit Date From and To – Enter the From and To Visit date by clicking calendar icon or enter manually in the dd/mm/yyyy format. This field appears only if the selected encounter type is ‘Outpatient’ and ‘Emergency’.

• ‘To Date’ should be Greater than or equal to ‘From Date’

  1. Practitioner – Select the practitioner name using [?] function or enter the practitioner name in 30 characters of maximum length.

  2. Patient ID – Enter the patient ID or search using [?] function.

  3. Encounter ID – Enter the encounter ID in alphanumeric format of 12 characters maximum length.
