Enter the following details to Request for a Medical Report:

 

Applicant Information

  1. Application Type - Select the type of the application from the list. The listed values are populated from the Application Type function. This field is mandatory.

  2. Letter Date/Time - Select the date and time of the letter using the calendar function. The current system date and time is displayed, by default. This field is mandatory.

  3. Applicant Name - Enter the name of the applicant requesting the Medical Report in alphanumeric format of 30 characters maximum length.

  4. Reference No - Enter the Reference Number for the application in alphanumeric format of 15 characters maximum length.

  5. Relationship Type - Select the type of relationship of the applicant to the patient from the list of valid values. The listed values are 'Next of Kin', 'First to Notify', 'Employer', 'Self' and 'Others'. On selecting 'Next of Kin', 'First to Notify' or 'Employer', the 'Applicant Name' field is displayed and disabled if the details were recorded while registering the patient. On selecting the value as 'Self', the 'Applicant Name' field is disabled.

  6. Relationship - Select the relationship of the applicant to the patient from the list. This field is enabled and becomes mandatory only if the 'Relationship Type' is selected as 'Others' or this field is disabled.

  7. Requester Type - Select the type of requester from the list. The values are predefined. This field is disabled if a value is selected for the 'Relationship Type' field.

  8. Requester - Enter the requester in alphanumeric format of 15 characters maximum length or select the name of the requester who requested for the Medical Report from [?] function. This field becomes mandatory on selecting the 'Requester Type'. This field is disabled if a value is selected for the 'Relationship Type' field.

  9. Consent Letter - Select the availability of the consent letter for the request as 'Available', 'Not Available' or 'Not Required' from the list. This field is mandatory.

  10. Attention To - Enter the name of the person whose attention is needed in the preparation of the medical report in alphanumeric format of 50 characters maximum length.

  11. Application No - Enter the application number in alphanumeric format of 15 characters maximum length.

Click [Applicant Address] to enter the contact address of the applicant of the Medical Report. This button is enabled on selecting the field 'Relation Type' from the list of values. Enter the contact address of the applicant and click [Close] to save the details. If the contact details of the applicant are already recorded while registering the patient, it is displayed.

The applicant address cannot be entered if the Requester Type and Requester are selected.

 

Charges

  1. Charges Applicable - Select Yes or No from the list indicating whether charges are applicable to the applicant for the medical report of the patient.

  2. Amount - Enter the amount to be charged in numeric format of 15 characters maximum length. This field is enabled only if the 'Charges Applicable' field is selected as 'Yes'.

  3. Receipt No - Enter the number of the receipt in alphanumeric format of 15 characters maximum length. This field is enabled only if the 'Charges Applicable' field is selected as 'Yes'.

  4. Receipt Date - The current date and time is displayed as the Receipt Date and this field is disabled if the Receipt No is entered.

  5. Additional Charges - Select Yes or No from the list indicating whether any additional charges are applicable to the applicant for the medical report of the patient.

  6. Amount - Enter the amount to be charged in numeric format of 15 characters maximum length. This field is enabled only if the 'Additional Charges' field is selected as 'Yes'.

  7. Receipt No - Enter the number of the receipt in alphanumeric format of 15 characters maximum length. This field is enabled only if the 'Additional Charges' field is selected as 'Yes'.

  8. Receipt Date -  The current date and time is displayed as the Receipt Date and this field is disabled if the Receipt No is entered.

Click ‘View Encounter Details’ hyperlink to view encounter details.

Click [Previous Delivered Medical Report] in the Medical Report Request screen to view the details of Medical Reports already delivered for the patient.
