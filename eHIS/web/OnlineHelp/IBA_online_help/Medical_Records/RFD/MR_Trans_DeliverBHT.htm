Enter the following search criteria:

 

  1. Encounter Type - Select the required encounter type from the drop-down list. The list values include Daycare, Emergency, Inpatient, and Outpatient. This field is mandatory.

  2. Encounter ID - Enter the encounter ID of the patient in the text box in numeric format up to 12 digits maximum.

  3. Location Type - Select the required location type from the drop-down list. The list values include Clinic, Daycare unit, Nursing unit. The values displayed are defined in the ‘Care Location Type’ function in Application Masters module.

  4. Location - The logged in user associated location will be loaded based on selected location type.

  5. Discharge Period [From -To] – By default, the current date is displayed. User can change the date range up to 14 days maximum by clicking the calendar icon. This field is mandatory.

a. To Date should be Greater than or equal to from Date.

b. Date Range cannot be greater than 14 days.

  1. Patient ID -Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the ID using [?] function.

  1. BHT Status - Select the required BHT status from the drop-down list. The list values include ‘Pending’, ‘Overdue’ and ‘Returned’. The default value is ‘Select’.
