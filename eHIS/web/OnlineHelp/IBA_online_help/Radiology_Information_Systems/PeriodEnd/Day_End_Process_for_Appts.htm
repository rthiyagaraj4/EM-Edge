Day End Process for Appointments

From this function you can Close appointments for a particular day.

Day End Process for Appointments function allows you to logically Close appointments for a particular day in the system. This function closes the appointments by posting statistics and purging the appointments that were not completed. The control is updated to reflect the last date that was closed using the Day End processing. The Day End Process for a day must be executed the next day.

Closing the Appointment for a particular day

From the Period End, double-click Day End Process for Appointments function.

The Day End Process for Appointments screen gets displayed.

  1. The Last Processed Day is displayed and disabled.

  2. Enter the date till which the Day End must be processed. The date entered must be greater than the ‘Last Day’ and lesser than the current system date.

  3. Click [Process Run] and all the records for this period are processed.

  4. Click [Run Process] to close appointments for a particular day.

The appointments statistics reports for required dates can be executed only after the Day End for those days has been processed. The List of No Shows report must be generated before the Day End for the required dates is processed.
