Record Radiologist Availability

From this function you can Record the availability or non-availability of radiologists.

This functional allows you to record availability or non-availability of radiologists. The system marks the radiologist as available even after date change until he marks himself not available. Also, you can view the availability of other radiologists and the number of exams assigned to that radiologist.

When the logged in user is not a Radiologist, the system displays an error message “Not allowed to use this function” and closes the function.

Recording Radiologist Availability

From the Transactions > Registration, double-click Record Radiologist Availability.

The Record Radiologist Availability screen gets displayed.

  1. Enter the details in the mandatory (*) fields of the following tab pages.

 

Radiologist Availability

This tab allows the logged in radiologist to mark their availability and non-availability.

  1. Radiologist ID – Displays the logged in radiologist ID

  2. Radiologist Name - Displays the logged in radiologist name.

  3. Limited Availability – By default, the check box is disabled. Once the radiologists name is selected, the check box gets enabled. Select the check box to enter the available date/time of the radiologist.

  4. From-To – Enter the available date/time of the radiologists by clicking the calendar icon.

  5. Available - Click [Available] to mark availability of the radiologists. Once the ‘Available’ button is clicked, the ‘Not Available’ button gets enabled. Once Availability is marked, the Available Radiologist Details tab gets enabled where you can view the details of all available radiologists.

  6. Not Available - Click [Not Available] to mark non-availability of the logged in radiologist. The system displays a warning message ‘Exam assigned to you are pending. Do you want to Continue?’ on clicking [Not Available] button if assigned  exams are pending for that radiologist.

Available Radiologist Details

This tab displays details of all the radiologists who are marked as available.

  1. Radiologist ID - Displays the ID of the available radiologist.

  2. Radiologist Name - Displays the name of the available radiologist.

  3. Available From - Displays the date and time when the radiologist marked the availability.

  4. No of exams assigned - Displays the number of exams assigned against each radiologist. When no exams are assigned for a radiologist, the system displays '0' in the column.

  5. Limited Availability From – Displays the radiologists available from date and time .

  6. Limited Availability To – Displays the radiologists available to date and time.

The system displays only logged in Radiologists availability when no access right is given in for the logged in user in 'Query other Radiologist/Revert Status' screen to view other radiologists details.

 

  1. Click [Save] to mark the changes done or click [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the radiologist records.
