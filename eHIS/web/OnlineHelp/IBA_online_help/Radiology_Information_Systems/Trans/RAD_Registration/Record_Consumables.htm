Record Consumables

From this function you can Record the consumables contrast agents and drugs used for exams and Query for the recorded consumable.

The Record Consumables function enables you to record the consumables, contrast agents and drugs used for the examination.

Recording the Consumables

From the Transactions > Registration, double-click Record Consumables function.

The Record Consumables screen gets displayed.

  1. The Exams for which start date is marked are displayed, by default or enter the search criteria and click [Search] to select a specific Order.

  2. On searching, examinations pending for registration are displayed based on the entered criteria.

  3. To view the demographic details of a patient, right-click the Patient ID and select ‘Patient Demographics’ menu. The ‘View Patient Details’ screen opens in a new Internet Explorer browser window.

  4. Double click on the Patient ID to select the Patient. The 'Record Consumables' screen is displayed. Based on the Patient ID selected, the Registration Number, Registration Date, Section/Wing where the examination is performed, Ordering Facility, Order number, Appointment Reference Number, Episode Number, Name and Gender of the Patient and the name of the attending doctor are displayed.

  5. Enter the details in the following tab pages.

Items Used

This tab enables you to enter the consumables used for the examination. The actual quantity of the consumables used has to be entered.

The following details are displayed:

  1. Exam – The examination code and description is displayed.

  2. Start Date Time – The start time and date of the examination is displayed.

  3. End Date Time – The end time and date of the examination is displayed.

  4. Items Used – Click [Items Used] to enter the consumables used in the examination. The actual quantity of the consumables used up during the examination has to be entered. The normal quantity of consumption of the selected consumable is displayed by the system.

 

After entering the details, click [Save] to save the entered details.

Contrast/Drug Admin

This tab enables you to enter the details of the drugs/contrast agents administered during the examination.

Enter the following details

  1. Exam – The exam code and description for which the drug/contrast agent details is displayed.

  2. Sedative Drug - Select this check box to record sedative drug details for the examination.

  3. Contrast Type – Select the Contrast Type from the list. The available options are ‘Only Plain’, ‘Only Contrast’ and ‘Plain and Contrast’. This list becomes empty and disabled when the Sedative Drug check box is selected.

  4. Anaesthesia Code – This field is enabled only when the ‘Sedative Drug’ check box is selected. Select the anaesthesia code from the drop-down list. This list populates the anaesthesia code and description value from the Anaesthesia master in Application Masters (AM) module.

  5. Contrast Agent/Drug – Select the contrast agent/drug used for the examination from the list of values. The contrast agents and drugs are created in the ‘Examinations’ function. This is a mandatory field.

  6. Normal Qty, Vol – The normal quantity and volume of the contrast agent are displayed. This is the quantity that is generally used.

  7. Actual Qty, Vol Administered – Enter the actual quantity of the contrast agent used during the examination. This is a mandatory field.

  8. Strength/Qty, Vol – The strength per quantity and the volume of the contrast agent/drug are displayed.

  9. Total Strength – The total strength of the contrast agent/drug is displayed.

  10. Batch ID – Enter the batch ID of the drug. This is a mandatory field.

  11. Stock Qty – The available stock quantity of the drug/contrast agent is displayed.

  12. Route – Enter the administration route for the drug or select from the list of values. This is a mandatory field.

  13. Administered By – Enter the name of the practitioner who has administered the drug to the patient.

  14. Adverse Reaction Code – Enter the Adverse Reaction Code in alphanumeric format of four characters maximum length or click [List of Values] to select the Adverse Reaction. Three rows are displayed, by default. The fourth row is enabled only after entering the three adverse reaction codes with a scroll bar for navigation. The Adverse Reactions of the Contrast/Drugs recorded in ‘Record Exam Start/Completion’ gets displayed.

  15. Description – The description of the Adverse Reaction Code is displayed. The description of the Adverse Reactions of the Contrast/Drugs recorded in ‘Record Exam Start/Completion’ gets displayed.

  16. Remarks – Double click on the Remarks field to display ‘Adverse Reaction Remarks’ screen. Enter the Remarks in alphanumeric format of 500 characters maximum length. This field is mandatory if adverse reaction code is selected. Click [Close] to exit from the Remarks screen. The remarks recorded are defaulted in ‘Result Reporting’ function. The remarks recorded in ‘Record Exam Start/Completion’ function are displayed in ‘Record Consumables’ function to view or edit.

  17. Admin Reaction – After the drug is administered to the patient, select the reaction code (if any). The reaction codes are set up in the ‘Examinations’ function.

  18. Drugs Used in Request – Click [Drugs Used in Request] to enter the details of the drugs/contrast agents given to the patient on request.

  19. Previous Drug – While you are entering the drug details, you can click [Previous Drug] to move to the previous drug details form and make necessary changes.

  20. Next Drug – While you are entering the drug details, you can click [Next Drug] to add another drug details.

Under the Contrast/Drug Admin tab, two tabs are available – ‘Enter Drugs’ and ‘View Drugs’.

 

The ‘Enter Drugs’ tab is used to enter the details of the drugs and the ‘View Drugs’ tab is used to view the entered details. First make all the drug entries in the ‘Enter Drugs’ tab and after making the entries, click the ‘View Drugs’ tab to view the list of the entered drugs. If the entered drug is sedative, the Sedative check box will be selected.

  1. Click [Save] to record the consumables, contrast agents and drugs or click [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for a recorded consumable.
