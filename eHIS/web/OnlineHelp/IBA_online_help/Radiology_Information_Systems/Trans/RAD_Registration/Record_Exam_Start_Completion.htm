Record Exam Start / Completion

From this function you can Record the start and completion details of the examinations and Query for the exam status record.

The Record Exam Start/Completion function enables you to record the start and completion of examinations conducted on the patient with details like the Start and End Time, the Technician who conducted the examination, Stock items consumed, and so on.

Recording Exam Start Completion

From the Transactions > Registration, double-click Record Exam Start Completion function.

The Record Exam Start Completion screen gets displayed.

  1. The exams with ‘Pending’ status are displayed or enter the search criteria and click [Search] to select a specific examination.

  2. On searching, examinations pending for completion are displayed based on the entered criteria.

  3. Click [Call Next Patient] to call the next waiting patient for examination based on the study location.

  4. Click [Recall number] to recall the patient who has not turned up for examination after being called using [Call Next Patient].

The [Call Next Patient] and [Recall number] buttons are enabled only if the ‘Assign Queue No. On Registration’ check box in ‘Operating Units/Wings’ screen is not selected.

  1. Double click on the Patient ID and ‘Record Exam Start’ and ‘Record Exam Completion’ screen gets displayed.

    The Exam details are displayed. Click [Patient History] to view the examination details of the patient; click [Examination History] to view the examination history of the patient; click [Prior Exams] to view the prior examination details of the selected patient;  click  [Protocol] to view the protocols assigned to the examination. The Protocol button is available only if protocols are assigned to the examination in the Master setup.

     

    Enter the details in the following tab pages.

    Exam Completion

    Involved Personnel

    Select 'Involved Personnel' tab to view the Involved person details.

    1. Exam Code – The Exam code of the selected Exam is displayed and disabled.

    2. Type – Select the Practitioner Type from the list.

    3. ID – Select the Practitioner ID from the list.

    4. Name – On entering the Practitioner ID, the name of the Practitioner is displayed.

    Technologist/Radiologist Comments

    Enter the Technologist’s or Radiologist comments during the Examination.

    Exposure Factors

    You can record the Exposure Factors that are applicable for the Section in Exposure Factors Tab and are not defined in the Examination Codes function. The system validates the Exposure Factors in Section function if the Exposure Factors are not entered in Examination Codes function.

    1. Exam Code - The Exam Code and Description is displayed and disabled.

    2. KV - Enter the units of Kilovoltage (KV) in numeric format of six digits maximum length.

    3. mA - Enter the units of Milliampere (mA) in numeric format of six digits maximum length.

    4. ms - Enter the units of Millisecond (ms) in numeric format of six digits maximum length.

    5. mAs - If 'mAs Auto calculate' check box is selected in 'RD Parameter for Facility' function,  the milliampere second is calculated by the system and displayed. If ''mAs Auto calculate' check box is cleared in 'RD Parameter for Facility' function, enter the milliamphere seconds (mAs )value in numeric format of 10 digits maximum length.

    6. Radiation in [] - Enter the Radiation in numeric format of nine digits maximum length. The unit of measurement of the radiation as defined in 'RD Parameters' is displayed in the field name [].

    Repeats

    The orders repeated details for the selected patient are displayed.

    Contrast/Drug Admin

    Contrast/Drug Admin Tab

    Select ‘Contrast/Drug Admin’ tab to view or add the Contrast/Drug Admin used during the examination and record the adverse reactions and remarks occurred during a procedure. This tab gets displayed based on the masters setup of ‘Examination Codes’ function if ‘Drug Administration reqd.’ check box is selected and the ‘Contrast Applicability’ defined as ‘Optional’.

    The adverse reactions and remarks recorded in this function can be viewed in ‘Clinical Event History’ of Clinical Care Management module. Multiple adverse reactions can be recorded.

     

    Enter the details in the following tab pages:

    Add Contrasts/Drugs

    View Contrast Drugs

    Report Text

    The Report Text details are displayed.

    1. Exam Code – The Exam code of the selected Exam is displayed.

    2. Report Text – Click [List of Values] to select the report text template.

    3. Type – Select the suitable option from the list. The options are – Prelim, Hold and Finalize.

    4. Print Report – Select the check box to print the report at the designated printer.

    5. Primary Diagnosis – Select the primary diagnosis form the list of values.

    6. Normal/Abnormal – Select if the report is ‘Abnormal’, ‘Normal’ or ‘Critically Abnormal’.

    7. Confidential – Select the check box to mark the report as confidential.

    8. Enter/Edit Result – If you have selected the exam report from the already set report templates, click [Enter/Edit Result] to make modifications in the report.

    9. Spell Check – Click [Spell Check] to check for spelling errors in the report text.

    On entering the Exam Completion details, click [Save] to record the exam completion details.

    The exam registration details are displayed. The displayed details are Patient ID, Name, Registration Number, Registration Date, Exam code and Description, Exam Date and Time, Start Date/Time. Enter the examination Start Date and Time in the format of dd/mm/yyyy hh:mm.

    • Click [Patient History] to view the examination details of the patient; click [Examination History] to view the examination history of the patient; click [Prior Exams] to view the prior examination details of the selected patient.

    • Click ‘Views’ tab to view the ‘View’ details. The Exam code, View Code and description are displayed. The view details are displayed This field is displayed if ‘view’ is applicable to the examination.

  2. Click [T] to view the location information of the patient.

  3. Click [C] to view the Clinical Events for the patient. The Clinical Events screen displays the order number of the examination, the relevant catalog code and description and the Patient Name and ID.

  4. Right click on the Patient ID to select the options 'Call number', ‘Complete’, ‘Change Study Location/Radiologist’, ‘Revert Status’, ‘Cancel’, ‘Revert Status Exam Completed to Exam Started’, ‘Revert Status Exam Completed to Registered’, ‘Reschedule’, ‘Add Additional Exam to Request’, ‘Request Comments’, ‘Inform Porter’, ‘Check Out Patient,’ ‘View Patient File Details’, Assign/View Protocol, 'Patient Demographics' and Record Radiology Clinical Notes.  

    Click the [Record Radiology Clinical Notes] menu to invoke the Record Radiology Clinical Notes window (3T) where radiologists can record their clinical notes.

    User can record, sign, modify, mark as In Error, update or forward the radiology clinical note as per the access rights assigned in ‘Note Type Access Rights’ function in CA.

    Once the notes are recorded, signed or modified then system will display the notes in below CA screens:

    o View Chart Summary - Clinical Notes widget

    o View Chart Summary - Events occurred in last 24 hrs widget

    o Clinical Event History

    Note:

    • For the applicable task – Record Radiology Clinical Notes, user must create the note groups and map to respective note types in Note Group and Note Type setups respectively in CA Clinical Note setup. Those note types must be configured in ‘Note Type’ field in RD Parameter for Facility screen.

    • The ‘Record Radiology Clinical Note’ menu will be disabled when ‘Note Type’ field is blank in RD Parameter for Facility Setup

    On selecting Patient Demographics option, the ‘View Patient Details’ screen opens in a new Internet Explorer browser window. You can view the patient demographic details in the View Patient Details tab.

    On selecting the View Patient File Details option, the Patient File Details screen is displayed. The X-ray folder is tracked based on the old medical record file numbers in the Patient File Details screen before exam registration. The View Patient File Details option can be accessed by authorized radiologist only.

    This option enables you to check out the patient from the ward to the Radiology wing.

    If ‘Patient Tracking’ option is enabled in the Radiology set up parameter for facility function, you are able to track the patient location.

    Select ‘Request Comments’ to view the clinical comments and reason for overriding duplicate orders (if any) as entered while placing order.

    If additional exams are to be conducted for the patient, select this option. Patient ID, Name, Registration Number, and Registration Date gets displayed.

    1. Wing  - Select the required wing from the list. The Operating Unit/Wings defined in the ‘Operating Units/Wings’ function under the Administration menu is displayed.

    2. Section – Enter the section in the alphanumeric format of four characters maximum length or click [List of Values] to select the section. Sections are displayed based on the selected ‘Operating Unit/Wing’. Sections are associated to the Operating Unit/Wing in the ‘Sections for Operating Unit/Wing’ function under the Administration menu.

    3. Exam Code - Enter the Exam code or click [List of Values] to select the Exam code.

    4. Laterality - Select the applicable laterality from the list. The list of system-defined values is, ‘Right’, ‘Left’ and ‘Bilateral’.

    5. Study Location – Select the location where the examination is to be done. This is a mandatory field.

    6. Radiologist ID – Select the radiologist ID from the [List of Values].

    7. Request Reason – Select the reason for adding the exam to the existing order from the [List of Values]. The reasons are defined in the ‘Reasons’ function in the ‘Examinations’ masters.

    8. OK – Click [OK]. The exam gets added to the patient record.

    9. Cancel – Click [Cancel] to cancel the request to add additional exams.

    10. Net Charge – The net charge defined in Patient Billing Manager module gets displayed for view and editing if ‘Interfaced to Patient Billing’ check box is selected in the ‘RD Parameters for Facility’ function.

    As you enter the laterality, the ‘Views’ tab gets enabled. Select the view from the list of values. This field is optional.

    Select ‘Reschedule’ to reschedule the examination with a suitable reason. Patient ID, Name, Registration Number, Registration Date and Exam is displayed. You cannot modify it. Enter the Reason or click [List of Values] to select the Reason. The description of the Reason gets displayed.

    This option appears only when the exam is completed.

    Click this option to revert the exam status from ‘Completed’ to ‘Registered.’

    If you are not having access to revert status, the message ‘RD9343-You are not authorised to revert the exam status’ gets displayed.

    This option appears only when the exam is completed.

    Click this option to revert the exam status from ‘Completed’ to ‘Started.’

    If you are not having access to revert status, the message ‘RD9343-You are not authorised to revert the exam status’ gets displayed.

    The examination can be cancelled with an appropriate reason. The Patient ID, Name, Registration Number, Registration Date and Exam are displayed. Enter the Cancellation Reason code or click [List of Values] to select a Cancellation Reason code from the list. The description of the Cancellation Reason code is displayed.

    If the exam is already ‘Started’, the exam can be reverted back to ‘Registered’ state by clicking this option.

    The Patient ID, the examination description and the current study location and radiologist are displayed. Enter the Study Location and Radiologist or click [List of Values] to select the Study Location or Radiologist under the title ‘Transfer To’ to change the study location and the radiologist. Click [Change] and the study location and radiologist are changed, or click [Cancel].

    On conducting the exam, click ‘Complete’ to complete the examination. Registration Number, Registration Date, Section/Wing, Appointment Reference Number, Order Number, Priority, Ordering Facility ID, Episode Number, Patient ID, Name, Sex, Age, Referral Source, LMP Date (if applicable), Female Patient Pregnant (if applicable), doctor, and storage location is displayed. Enter the Exam Completion details such as Repeat/Complete. Media Type, Media ID, Scan Number, Number of films or images taken, radiologist and examination end time, the number of repeats (if required), the examination report, the drugs and the contrast agents used for the examination, Involved Personnel and Technologist Comments. Click [Patient History] to view the examination history of the patient; click [Prior Exams] to view the exams registered earlier; Click [Request Comments] to view the Clinical Comments, Examination comments and the Item Narration.

    On selecting the Call number option, the selected patient is called for examination for any urgent or special cases irrespective of the queue number assigned to the patient.

  5. Right click on ‘Order By’ or ‘Attend By’ to view the Physician’s contact details.

  6. Click [Statistics] to view the Exam Statistics. Click [>>] to view the Technician Wise Exams completed details for the current date. Click [OK] to exit from the Statistics screen.

    The Technician Name and the No of Exams completed by the Technician is displayed. Click [Close] to exit from the ‘Technician Wise Exams Completed Today’ screen.

  7. Click [Save] to record the start and completion details of the examination or click [Clear Record] to clear the entered data.

  8. Click [Execute Query] or press [F8] to search for an examination status.

The color indicators for Priority [P], Waiting Time [W] (In Minutes), Allergy Indicator [A], Recalled Patient [R], Assistance Required [S] and Mobile Request [M] are displayed in the bottom of the Pending Exams for Authorization screen. The Patient Name is highlighted with the color indicator ‘Red’ for the high risk patient. If any adverse reactions are recorded for any contrasts of an examination you get an alert message ‘Patient has adverse reactions, please check the details’. Click [OK] to proceed with the recording of the adverse reactions for the contrasts/drugs used in the examination. Click [Details] to display ‘Adverse Reaction’ search screen. You can view the administered date, exam, contrast/drug, adverse reactions recorded with remarks for the selected exam. The button gets displayed only if adverse reactions are recorded for the exam. If no adverse reactions are recorded no alert message is displayed. If the radiology appointment is rescheduled in OA, the message ‘This Order has been moved to Pending Order Tab of the Scheduling function in OA module’ is displayed. This message will appear only if the Radiology module is integrated with the Appointment Scheduling (OA) module. The Queue number in Work List table in Record Exam Start/Completion screen is highlighted in red when the patient is called by Call Patient function.
