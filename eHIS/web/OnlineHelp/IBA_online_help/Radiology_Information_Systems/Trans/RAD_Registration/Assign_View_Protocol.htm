On selecting Assign/View Protocol option, the Assign/View Protocol screen is displayed. This option is enabled only if a protocol is already mapped to the examination.

 

Enter the following details:

  1. Protocol ID – Enter the protocol ID or click [List of Values] to select the protocol ID. This protocol list includes the protocols mapped in the masters for the selected examination.  

  2. Name – On entering the Protocol ID, the protocol name is displayed automatically.

  3. Description – On entering the Protocol ID, the protocol description is  displayed automatically.

  4. Select All – Select this check box if you want to copy the selected protocol to other examinations listed in the Order. Or, you can select the check box against the corresponding Exam Code.  

  5. Exam Code, Name – Displays the code and name of the examinations having the same protocol assigned to it in the master.

  6. RAD / Tech ID – Displays the logged-in user ID. This is a non-editable field.

  7. Name – Displays the name of the Radiologist or Technician.

  8. Radiologist / Technician – Displays the designation of the logged-in user i.e., Radiologist or Technologist.

  9. Date – Displays the date and time when the comments are entered. This is a non-editable field.

  10. Comments – Enter the comments, if required. You can enter up to 4000 characters.

  11. OK – Click [OK]. The protocol is assigned to the examination.

Only Radiologist can assign/change the protocols and add comments.  

The Technologist can assign/change the protocol or add comments only while completing the exam.
