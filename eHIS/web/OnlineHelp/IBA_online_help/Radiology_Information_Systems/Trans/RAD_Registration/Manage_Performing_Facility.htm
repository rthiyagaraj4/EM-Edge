Manage Performing Facility

From this function you can Change the performing facility for the orders before the registration stage.

Manage Performing Facility function is used to change the performing facility for the orders in case of equipment breakdown or examinations not available in the hospital.

Managing Performing Facility

From Transactions > Registration menu, double-click Manage Performing Facility function.

The Manage Performing Facility screen gets displayed.

  1. Enter the details in the respective fields. <Related Field Description>

  2. Click [Display] to view the Order ID, Order Type, Patient ID, Name, Order Date, Order Catalog, Order Status, Appointment Reference Number and Performing Facility.

  3. Click [A] to invoke the Appointments for Study Location screen.

  4. Double-click on the Order ID to invoke Appointments for Patient screen.

  5. Select the check box corresponding to the Order ID to change the order to the facility mentioned in the ‘Change To’ field.

  6. Click [Save] or press [F10] to change Performing Facility for the selected orders or click [Clear Record] to clear the entered details.
