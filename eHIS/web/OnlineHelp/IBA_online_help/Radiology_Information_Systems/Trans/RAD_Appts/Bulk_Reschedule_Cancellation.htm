Bulk Reschedule/Cancellation

From this function you can Reschedule or Cancel appointments scheduled for examinations in bulk for the same wing.

The Bulk Reschedule/Cancellation function is used when the machine in the Study Location is not operating, all the examinations scheduled to be conducted needs to be rescheduled to another day or study location or the Radiologist is unable to conduct the examinations due to an emergency. Enter the search criteria and select all the examinations that need to be Rescheduled/Cancelled.

Rescheduling/Cancelling Appointment in Bulk

From the Transactions > Appointments, double-click Bulk Reschedule/Cancellation function.

The Bulk Reschedule/Cancellation screen gets displayed.

  1. Enter the Reschedule/Cancel From details in the mandatory (*) fields.

    Enter the following search criteria:

    1. Transaction Type – Select the Transaction Type from the list. The listed values are ‘Transfer’ and ‘Cancel’. Select ‘Transfer’ to reschedule/transfer the examination in bulk or  ‘Cancel’ to cancel the appointments in bulk. This is a mandatory field.

    2. Operating Unit/Wing – Enter the Operating Unit/Wing or click [List of Values] to select the Operating Unit/Wing. The Operating Unit/Wings are defined in the Operating Units/Wings function under the Administration menu. This is a mandatory field.

    3. Section – Enter the Section or click [List of Values] to select the Section. Sections are displayed based on the selected ‘Operating Unit/Wing’. Sections are associated to the Operating Unit/Wing in the ‘Sections for Operating Unit/Wing’ function. This is a mandatory field.

    4. Study Location – Enter the Study Location or click [List of Values] to select the Study Location. This is a mandatory field.

    5. Appointment Date – Enter the Appointment Date in the format of dd/mm/yyyy or select the Appointment Date using the calendar function.

    6. Reason – Enter the Reason code or click [List of Values] to select the Reason for bulk Transfer/Cancel. The Reason codes with Action Type ‘Transfer Appointments (Bulk)’ and ‘Cancel Appointment (Bulk)’ are displayed for selection and are defined in the ‘Reasons’ masters.

  2. On entering the Reschedule/Cancel From details, click [View Appointments] to view the scheduled appointments for the entered criteria are displayed.

  3. The appointment details are displayed in the Work List section. Select the check box beside the Patient ID to select the exam for transferring/cancelling. You can select multiple exams together. All the ‘Selected’ exams are selected for bulk rescheduling or transfer.

  4. Enter the Study Location or click [List of Values] to select the Study Location. All the selected appointments are rescheduled to the selected Study Location.

  5. Enter the Date in the format of dd/mm/yyyy or select the Date using the Calendar function. All the selected appointments will be rescheduled for this date.

  6. Click [Confirm] to rescheduled or cancel the appointments in a bulk.

Reschedule/Cancel section is applicable for rescheduling the appointments only.
