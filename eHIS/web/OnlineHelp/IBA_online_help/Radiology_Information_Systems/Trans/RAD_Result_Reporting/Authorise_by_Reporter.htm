Authorise by Reporter

From this function you can Authorize the completed results and Query for the examination reports to be authorized by the reporter. You can Print the examination Report.

An authorized reporter reviews the completed examination results of a patient and authorizes the examination results. The results completed and submitted using the ‘Report Examination’ function is listed. An authorized reporter can update the Report Text, Diagnosis/Analysis Codes, add review notes and finally authorize the Result. Only reports that are marked as ‘complete’ in the ‘Report Examination’ function can be authorized.

Authorizing a Completed Result

From the Transactions > Result Reporting, double-click Authorise by Reporter function.

The Authorise by Reporter screen gets displayed.

  1. The examination results, which are marked as ‘Complete’ in the Performing Facility, are listed. Enter the search criteria and click [Search] to search for a specific patient record. Right-click on the ‘Right Click for User Options’ option to search for a patient record using the ‘Standard Patient Lookup’ or the ‘External Patient Lookup’ screen.

  2. The examination reports are displayed on clicking [Search]. The patient records are displayed based on the search criteria.

  3. Double-click on a Patient ID to view the examination reports of the selected patient. Authorise by Reporter screen is displayed with the patient ID, name, exam details, laterality, Radiologist code, wing and registration number.

  4. Click [Examination History] to view the variances recorded for the patient.

  5. Click [Examination History] or [Prior Exams] to view the examination details. Examination History screen is displayed listing the patient movement details. Click ‘View Details’ and the patient details are displayed under ‘Exam Details’, ‘Reviewer Notes’, ‘Relative Follow Up Investigation’, ‘Audit’, and ‘Patient Movement’ tabs. You can also view the Report Text recorded for each examination. The details can only be viewed.

  6. Click [Request Comments] to view the comments recorded for the patient while placing a request for examination.

  7. The details of the patient’s examination report in the department are displayed under Report Text, Complications, Reviewer Notes, Technologist Notes and Other Personnel tabs. You can update the Report Text, Complications and Reviewer Notes.

    Enter the following details:

    1. Reviewer Notes – The reviewer can enter the notes while authorizing the examination reports.

    Enter the following details:

    1. Exam – The exam for which the report is being prepared is displayed.

    2. Complication Type - Click [List of Values] to select and associate a Complication Type. The values defined using the ‘Complication Type’ function get listed. You can associate additional Complication Types to the examination report.

    3. Complication Management Type/Description - Click [List of Values] to select and associate a Complication Management Type code. The values defined using the ‘Complication Type’ function get listed. You can associate additional Complication Management codes to the Complication Types.

    4. Remarks – Enter the remarks in alphanumeric format. You can enter any additional details about the complications observed during examination.

    The Complication Type and the Complication Management Types are defined in the Result Reporting function.

    Enter the following details:

    1. Report Text - Enter the Report Text in alphanumeric format in the provided space.

    2. [Spell Check] – Click [Spell Check] to check the spellings in Report Text entered. You can modify the incorrect spellings using this option.

    3. [Report With] - Click [Report With] to associate the exam code and description to the Report Text.

    4. Primary Diagnosis – Click [List of Values] to select and associate a Primary Diagnosis. The values defined using the ‘Diagnosis Codes’ function, get listed.

    5. Result Status – Select the suitable option from the list. The available options are ‘Normal’, ‘Abnormal’ and ‘Critically Abnormal’. This is a mandatory field.

    6. Confidential – Select the check box to indicate that the report is to be kept confidential. Only by authorized radiologists can access the reports, which are marked as confidential. The radiologists who have access to confidential reports are defined in the ‘Users for Viewing Confidential Reports’ function.

    The names of the personnel, who have conducted the exam, are displayed.

    The technologist comments are displayed. The comments of the radiologist who has reported the exam are displayed.

  8. Select a Type from the list. The system-defined values are ‘Hold’, ‘Prelim’ and ‘Finalize’. Select ‘Hold’, if the examination report is not confirmed. Select ‘Prelim’, if the authorization report is in the preliminary stages. Select ‘Finalize’ to authorize and finalize the examination reports. You cannot change the status from ‘Hold’ to ‘Prelim’, but you can finalize a report for which the status is either ‘Hold’ or ‘Prelim’.

  9. Select the check box ‘Print’ to print the examination report. The destination printer to print the report has to be defined. On selecting the check box, the report is sent to the destination printer for printing.

  10. Click [Save] to authorize the completed reports or [Clear Record] to clear the entered details.

The color indicators for Priority [P], Allergy Indicator [A], Mobile Request [M] and Recalled Patient [R] are displayed in the bottom of the Ordered Exams/Appointments screen. An error message ‘Not allowed to use this function’ is displayed on accessing the function, if the user is not authorized to access this function.
