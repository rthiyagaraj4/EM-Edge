ACR Codes

From this function you can Associate the ACR Codes to examination reports of a patient and Query for a desired report.

The acronym ACR stands for American College of Radiologists. ACR Codes are Diagnosis codes which enables Radiologist to associate the examination reports to  ACR codes. You can view the result of an examination conducted in the facility for a specific patient.

Associating ACR Codes to Reports

From the Transactions > Result Reporting, double-click ACR Codes function.

The ACR Codes screen gets displayed.

  1. The examination reports, which are finalized and authorized by a Radiologist, are listed. Enter the search criteria and click [Search] to search for a specific patient record. Right-click on the ‘Right Click for User Options’ function to search for a patient record using the ‘Standard Patient Lookup’ screen.

    Enter the following search criteria:

    1. Perf. Facility – Select the Performing Facility from the list.

    2. Wing – Select the Wing from the list. The list of values gets populated with the Operating Wings/Units created in the ‘Operating Wings/Units’ function.

    3. Section – Enter the Section code or select it from the list of values. The name of the Section code is displayed in the adjacent field. The list of values gets populated with the sections created in the ‘Examinations’ function.

    4. Registration Date [From / To] - Enter the date range. All the registrations made in this period are displayed in the search result.

    5. Registration No – Enter the registration number of the examination.

    6. Patient ID - Enter the Patient ID. The name of the patient is displayed in the adjacent field.

    7. Right Click for User Options – Right-click ‘Right Click for User Options’ hyperlink to invoke the Standard Patient Lookup screen. Double-click the Patient ID in the displayed list to select the patient.

    8. Episode Type – Select the episode type from the list of values. The pre-defined values are All, Inpatient, Outpatient, Referral and External.

    9. Exam - Select the exam code for which the report has to be prepared. The examination description is displayed in the adjacent field. The list of values gets populated with the exam codes created in the ‘Examinations’ function.

    10. Organ – Select the organ from the list of values. The values are created in ‘Anatomical Site’ function in Application Masters module.

  2. Click [Search] to query for the examination reports. Based on the search criteria, details like Patient ID, Name, Exam, Organ Site, Registration Number, Registration Date, Radiologist, Section, Wing and Ordering Facility are displayed.

  3. Double-click on the Patient ID to select the examination report. ACR Codes screen gets displayed. Patient ID and Name, Episode Number, Exam code and description, registration number and registration date are displayed in ACR Code screen.

  4. Click ‘Result’ tab to view the examination result.

  5. Click ‘ACR Coding’ to associate ACR Codes to examination report of the patient.

    Anatomic and Pathologic Code

    Enter the following details:

    1. Anatomic Code/Description – Select the anatomic code from the list of values. The description of the code is displayed in the adjacent field. The Anatomic Codes created in the Result Reporting function get populated in the list.

    2. Pathologic Code/Description – Select the pathologic code from the list of values. The description of the code is displayed in the adjacent field. The Pathologic Codes created in the Result Reporting function get populated in the list.

    3. Remarks – Enter the remarks.

    4. ACR Coding Completed – Select the check box to mark ACR coding as complete.

  6. Click [Save] to save the records or click [Clear Record] to clear the entered data.

  7. Click [Execute Query] or press [F8] to search for an examination report.

An error message ‘Not allowed to use this function’ is displayed on accessing the function, if the user is not authorized to access this function.
