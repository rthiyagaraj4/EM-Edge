Modify Authorised Reports

From this function you can Modify or Print the examination reports and Query for a desired report.

The Modify Authorized Report function is used to modify and print reports, which have been authorized earlier. You cannot change the original report, but associate additional details to the original examination report authorized by a Reporter or a Radiologist. System automatically inserts a header showing the date of entering the addendum and the Reporter/Radiologist who has created the addendum. Only an authorized Radiologist can modify and print the reports.

Modifying Authorised Reports

From the Transactions > Result Reporting, double-click Modify Authorised Reports function.

The Modify Authorised Reports screen gets displayed.

  1. Click [Execute Query] or press [F8] to query all the examination reports. Click [Enter Query] or press [F7], enter the details and click [Execute Query] or press [F8] to search for a specific examination report.

  2. The examination reports are displayed for patients in the facility. You can update the examination reports under the following tabs:

List of Authorised Exams

List of Authorised Exams

Enter the following details:

  1. Exam Date – The date of the exam is displayed.

  2. Exam Description – The description of the exam is displayed.

  3. Radiologist – The name of the radiologist is displayed.

  4. Authorizing Reporter – Select the name of the authorizing reporter.

  5. Print – Select the check box ‘Print’ to print the authorized exam report. The examination report is routed to the destination printer set.

Report Text

Report Text

Enter the following details:

  1. Reported With – There is a provision to report multiple exams together, that is if multiple exams exist for a request or multiple requests, exist for a patient, other exams is shown in the ‘Report With’ block. To report these exams along with the selected exam, click the [Report With], which will show all the exams for the above request in a separate window. Select the exams, which you want to report together and click [OK].

  2. Report Text – Enter the report text in free format.

  3. Modify Report – Click [Modify Report] to modify the report.

  4. Modify – Click [Modify] to save the modified report.

  5. Primary Diagnosis – Select the primary diagnosis from the list of values. The Diagnosis codes are defined in the Result Reporting function.

  6. Result Status – Select the suitable result status from the list. If the report is normal, select ‘Normal’; if the report ID abnormal, select ‘Abnormal; if the report is critically abnormal, select ‘Critically Abnormal’.

  7. Confidential – Select the check box to mark the report as ‘Confidential’. Once the report is marked as confidential, only users with valid rights are allowed to view this report. The users with rights to confidential reports are defined in the RD Parameters for Facility function.

  8. Record Radiology Clinical Notes – Click the [Record Radiology Clinical Notes] button to invoke the ‘Record Radiology Clinical Notes’ window from 3T to record the radiology clinical notes. User can record, sign, modify, mark as In Error, update or forward the radiology clinical note as per the access rights assigned in ‘Note Type Access Rights’ function in CA. Once the notes are recorded, signed or modified it will be displayed in ‘View Chart Summary - Clinical Notes widget’, ‘View Chart Summary - Events occurred in last 24 hrs’ widget and ‘Clinical Event History’ function in CA.

Complications

Complications

Enter the following details:

  1. Exam – Select the name of the exam from the list of values.

  2. Complication Type/Description – Select the complication type code from the list of values. The description of the complication type is displayed in the adjacent field.

  3. Complication Management Type/Description – Select the Complication Management Type code from the list. The description of the management type is displayed in the adjacent field.

  4. Remarks – Enter remarks, if any.

  1. Click [Request Reason] to enter the request comments of the exam.

  2. Click [Save] to modify the reports or click [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for a modified report.

 

If the existing report is marked as ‘In-Error’ or ‘Modilfy’ in Record Clinical Notes screen, and the report is signed in the Modify Authorised Report screen, the system displays a message ”Please Save the report” when you try to close the report.
