Lay Language Patient Letters

From this function you can Generate and Dispatch follow-up letters to patients for their follow-up tests. You can Query to view the examination reports for follow-up.

The Lay Language Patient Letters function enables you to issue letter(s) to patients for their follow-up tests. Report has to be marked for follow-up at the result reporting stage itself and the letter for follow-up is given to the patient with the report. This function is used to issue letter to the patient, if the patient des not collect the follow-up letter with the report.

Issuing Lay-language patient Letters

From the Transactions > Result Reporting, double-click Lay Language Patient Letters function.

The Lay Language Patient Letters screen gets displayed.

  1. The examinations for which the follow-up letters are not collected by the patient are listed. Enter the search criteria and click [Search] to search for a specific patient record. Right-click on the ‘Right Click for User Options’ function to search for a patient record using the ‘Standard Patient Lookup’ screen.

    Enter the following details:

    1. Perf. Facility – Select the performing facility, which conducted the examination.

    2. Wing – Select the wing in the facility where the exam was conducted. The operating wings created in the Operating Wings function get listed.

    3. Section – Click [List of Values] to select a Section. The values defined using the ‘Sections’ function get listed.

    4. Registration Date [From-To] – Click the Calendar function to select the date range. Examination reports with the registration date matching the date range specified get listed.

    5. Registration No – Enter the registration number in alphanumeric format of 10 digits maximum length. The examination reports matching the specified Registration Number gt listed.  

    6. Patient ID – Enter the Patient ID for whom the exam is booked. Right-click ‘Right Click for User Options’ to select the patient ID from the standard patient lookup.

    7. Episode Type – Select the suitable option from the list. The options available are All, Referral, Outpatient and Inpatient.

    8. Exam – Click [List of Values] to select the examination code for which the follow up investigations are requested.

  2. Click [Search] after entering the search criteria. Based on the search criteria, Patient ID, Name, examination, registration number and date, Radiologist, Section, Wing and Ordering Facility are listed. The check boxes ‘Generated’ and ‘Dispatched’ are selected, if the follow-up letters are generated and dispatched to the patients respectively. The check boxes are cleared, if the follow-up letters are not generated and dispatched.

  3. Double-click the Patient ID to whom the letter has to be issued. You can mail or print the letter.

  4. Click [Save] to issue letters for follow up tests or click [Clear Record] to clear the entered data.
