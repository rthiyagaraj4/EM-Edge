Purge Suspended Codes

From this function you can Purge the suspended master records from the system.

Purge Suspended Codes function is used to remove the suspended master records from the system and the records are not physically available. This batch function carries out a number of validations to determine whether a master code can really be purged. If the master code is present in other master or transaction records, the master code is not purged and an error message giving details of the table where the code is used is recorded by the function. A message is recorded when a code is purged successfully. You can view the errors and messages or print subsequently using the function ‘Report Purged Masters’.

Purging Suspended Codes

From the Maintenance, double-click Purge Suspended Codes function.

The Purge Suspended Codes screen gets displayed.

  1. Enter the Master Code in alphanumeric format of six characters maximum length or click [List of Values] to select the required Master Code. This is a mandatory field.

  2. Click [Process Run] to Purge the suspended master records.
