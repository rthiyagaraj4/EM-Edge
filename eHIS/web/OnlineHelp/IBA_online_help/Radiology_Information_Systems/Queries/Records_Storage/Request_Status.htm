Request Status

From this function you can View the status of all folder requests.

Request Status function enables you to view the status of all folder requests for a time period from a particular location. The request status can be ‘All’, ‘Outstanding’, ‘Issued’ or ‘Rejected’.

Viewing the Status of all folder requests

From the Queries > Records Storage, double-click Request Status function.

The Request Status screen gets displayed.

  1. Select the required Location from the ‘Req Loc Type/Location’ list. Based on the selected Location Type, the Locations are displayed. Enter the Location code in alphanumeric format of 20 characters maximum length or select the Location using the [List of Values].

  2. On selecting the Location, select the Location from where the record or folder has been requested.

  3. Select the Request Status from the list. The listed options are ‘All’, ‘Outstanding’, ‘Issued’ and ‘Rejected’.

  4. Enter the ‘From’ and ‘To’  Date in the format of dd/mm/yyyy or select the required date using the Calendar function.

  5. Enter the Patient ID in the length as specified in the MP Parameter or click ‘Right Click for User Options’ and click [Standard Patient Lookup] to select the Patient. The Patient Name, Sex and Age are displayed on selecting the patient. This is a mandatory field.

  6. On entering the criteria, click [Search] or [Execute Query] or press [F8] to display the request details of the folder.
