Enter the following details to create the Reporting Templates:

  1. Radiologist – Enter the Radiologist code or click [List of Values] to select the Radiologist. The name of the selected Radiologist gets displayed. The Radiologists are defined in ‘Practitioner’ function with the Practitioner Type as ‘Radiologist’ in Application Masters module. This is a mandatory field.

  2. Template By – Select the ‘Template By’ value from the list. The system-defined values are Exam, Section and Multiple Exam.

    On selecting the ‘Template By’ value: ‘Multiple Exam’, the Free Text Templates tab is disabled and the ‘Multi Exam Template’ is enabled.

    This field is displayed, on selecting the ‘Template By’ value: ‘Section’. Enter the Section code or click [List of Values] to select the Section. The Section description is displayed. Free Text Templates tab is enabled on selecting ‘Template By’ value: ‘Section’. You can associate the free text templates to this Section. The Sections are defined in ‘Sections’ function under ‘Examinations’ in Master Menu. This is a mandatory field.

    This field is displayed, on selecting the ‘Template By’ value: ‘Exam’. Enter the Exam code or click [List of Values] to select the Exam. The Exam description is displayed. Free Text Templates tab is enabled on selecting ‘Template By’ value: ‘Exam’. You can associate the free text templates to this exam. The Exams are defined in ‘Examination Codes’ function under ‘Examinations’ in Master Menu. This is a mandatory field.
