Enter the following search criteria:

  1. Ordering Facility – Select the Ordering Facility from the drop-down list to display the orders based on the selected Ordering Facility. The value ’All Facilities’ is selected by default.

  2. Wing – The name of the wing gets displayed. If the displayed wing is not applicable, enter the wing or click [List of Values]. The Operating Unit/Wings are defined in the Operating Units/Wings function under the Administration Menu.

  3. Section – Enter the Section or click [List of Values]. Sections are displayed based on the selected ‘Operating Unit/Wing’. Sections are associated to the Operating Unit/Wing in the ‘Sections for Operating Unit/Wing’ function under the Administration Menu.

  4. Priority – Select the priority of the examination from the list of values. The listed system-defined values are ‘Static’, ‘Urgent’, ‘Routine’ and ‘All’.

  5. Sex - Select the Gender of the patient. The listed system-defined values are Male, Female and All.

  6. Order Date [From-To] – Click calendar function to select the date range for the orders placed. All the pending orders in this date range are displayed.

  7. Episode Type – Select the episode type applicable for booking from the list. The episode type can be ‘Inpatient’, ‘Outpatient’, ‘Referral’, ‘External’ or ‘All’. If you select ‘All’, the result displays Inpatients, Outpatients, Referral and External patients. ‘All’ gets displayed, by default.

  8. Status – Select the Status of the examination from the list. The listed values are Pending All, Pending/Modified, Pending Appointment, Pending Authorisation, Replaced, Rejected, Cancelled and All.  

  9. Patient ID – Enter the Patient ID or click ‘Right Click for User Options’ to select the Patient ID all the pending examinations for the patient are displayed.

  10. Show – Select the option applicable for searching the orders from the list. The listed system-defined values are Unscheduled, Scheduled and All.
