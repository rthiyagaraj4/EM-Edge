Enter the following details:

  1. Section – Enter the section code or click [List of Values] to select the section code. On selecting the Section code, the name of the Section is displayed. The Section list is defined in the Section Masters.

  2. Radiologist ID – Select the required Radiologist ID from the list.

  3. Unassigned to Self Assign – Select this check box to give access to the radiologist to assign the unassigned patients to self. If this check box is selected, the ‘Unassigned’ check box in Authorise/Review by Radiologist function will be selected so that the radiologist can view the unassigned patients in the Work List and assign it to self by selecting the ‘Select All’ check box against each patient record. The radiologist name gets updated in the ‘Radiologist’ column for the selected patients.

  1. Unassigned to Others – Select this check box to give access to the radiologist to assign the unassigned patients to others. If this check box is selected, then the ‘Change To’ drop-down will be available in the Authorise/Review by Radiologist function.  The- radiologist can view the unassigned patient list and select the radiologist name from the ‘Change To’ drop-down list. Then select the ‘Select All’ check box against the patient record to be reassigned. The selected radiologist name gets updated in the ‘Radiologist’ column for the selected patients.

  2. Assigned to Self Assign – Select this check box to give access to the radiologist to reassign the already assigned patients to self.  If this check box is selected, the ‘Change To’ drop-down list and the ‘Unassigned’ check box will not be visible in the Authorise/Review by Radiologist function. The radiologist can view the assigned patient list and reassign the patient to self by selecting the ‘Select All’ check box against the patient record. The radiologist name gets updated in the ‘Radiologist’ column for the selected patients.

  3. Assigned to Others - Select this check box to give access to the radiologist to reassign the patients to others. If this check box is selected, the ‘Change To’ drop-down list and the ‘Select All’ check boxes are  visible in the Authorise/Review by Radiologist function. The radiologist can view the assigned patients in the Work List and select the radiologist name from the ‘Change To’ drop-down list for reassigning.  The selected radiologist name gets updated in the ‘Radiologist’ column for the selected patients.
