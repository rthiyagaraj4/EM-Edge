Enter the following details to create the Default Storage Location for Media Types:

  1. Operating Unit/Wing - Enter the Operating Unit/Wing in alphanumeric format of two characters maximum length or click [List of Values] to select the Operating Unit/Wing. On selecting the Operating Unit/Wing, the description of the selected Operating Unit is displayed. The Operating Unit/Wings are defined in the ‘Operating Units/Wings’ function under ‘Administration’ in Masters menu. This is a mandatory field.

  2. Section – Enter the Section in alphanumeric format of four characters maximum length or click [List of Values] to select the Section. The Sections are displayed based on the selected ‘Operating Unit/Wing’. Sections are associated to the Operating Unit/Wing in the ‘Sections for Operating Unit/Wing’ function in the Administration menu. This is a mandatory field.

 

Location by Media Type

  1. Media Type – Enter the Media Type or click [List of Values] to select the Media Type. The Media Types are defined in the ‘Media Types’ function under ‘Records Storage’ in Masters menu. This is a mandatory field.

  2. Media Description – The description of the selected Media Type is displayed.

  3. Storage Facility ID – Enter the Storage Facility ID or click [List of Values] to select the Storage Facility ID. The Storage Locations are defined in the ‘Storage Locations’ function under ‘Records Storage’ in Masters menu. This is a mandatory field.

  4. Storage Location Code – Enter the Storage Location or click [List of Values] to select the Storage Location Code. The Storage Location code indicates the current storage location of the media type. This is a mandatory field.

  5. Storage Loc Description – The description of the selected Storage Location is displayed.

  6. 8. Initial Storage Facility ID – Enter the Facility ID where the media type has to be initially stored or click [List of Values] to select the Initial Storage Facility ID. This is a mandatory field.

  7. Initial Storage Location Code – Enter the Initial Storage Location code or click [List of Values] to select the Initial Storage Location Code. This field indicates the original storage location of the media type. This is a mandatory field.

  8. Initial Loc Description – The description of the selected storage location is displayed.
