Enter the following details:

  1. Examination – Enter the Examination code in alphanumeric format of 10 characters maximum length or click [List of Values] to select the Examination. The description of the selected Examination is displayed. The Exams are defined in ‘Examination Codes’ function under ‘Examinations’ in the Masters menu.

  2. Section – Enter the Section code in alphanumeric format of four characters maximum length or click [List of Values] to select the Section. The description for the Section is displayed. The Sections are defined in the ‘Sections’ function. This is a mandatory field.

  3. Contrast Agent, Drug Administration required – Select the check box to indicate that the examination requires contrast agents and drugs. On this check box, the Contrast/Agent drugs tab gets displayed in this function Only after selecting the check box,. You can associate the contrast agents or drugs administration to an examination if the ‘Drug Administration reqd’ check box is selected while creating the examination. The check box is selected and disabled.

A contrast agent distinctly distinguishes the body parts.
