Enter the following details:

  1. Storage Location Code – Enter the Storage Location code or click [List of Values] to select the Storage Location. On selecting the Storage Location code, the description of the Storage Location is displayed. This is a mandatory field.

  2. User – The logged in user name is displayed.

  3. Patient ID – Enter the Patient ID in the length as specified in the MP Parameter of click ‘Right Click for User Options’ to select the Patient using Standard Patient Lookup function. This is a mandatory field.

  4. Select Folder – Select the Folder from the list. This is a mandatory field.

  5. Current Folder Status – The system displays the current status of the folder.

  6. Changed Folder Status to – Change the folder status by selecting a suitable option from the list. The available options are ‘Lost’, ‘Inactive’ and ‘Found’.

  7. Changed Folder Status of – Select the ‘Change Folder Status of’ value from the list. The listed values are ‘Folder’ and ‘Exams’. Select ‘Folder’ to change the status of a Folder or select ‘Exams’ to change the status of the exams.

  8. Reason – Enter the Reason for changing the folder location or click [List of Values] to select the Reason.

 

Exam Details

  1. Exam Code – On entering the folder details, the exam code is displayed.

  2. Exam Description – On entering the folder details, the exam description is displayed.

  3. Created date – On entering the folder details, the folder creation date for the exam is displayed.

  4. Confirm – Select the check box to confirm and change the folder status.
