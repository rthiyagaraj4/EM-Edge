Enter the details in the ‘Post-Exams Details/Item Used’, ‘Views’ and ‘Exam Narration’ Tabs.

The recorded narration details for the selected exam is displayed.

This tab is enabled, if ‘Views’ are applicable. The Exam code is displayed. Select the View Code and Description.

Enter the following details:

  1. Repeat/Complete – Select the value from the list. The status ‘Complete’ is displayed, by default.

  2. Auto Receipt – Select the check box to indicate that the examination has been marked for auto report.

  3. Exam Code – Enter the Exam code or click [List of Values] to select the Exam code. The list of values are populated with the examinations created in the ‘Examinations’ function.

  4. Laterality – Select the Laterality applicable from the list.

  5. Study Loc – Enter the Study Location code or click [List of Values] to select the Study Location.

  6. Start/End Date Time – The Exam start date and the end date is displayed in the format of dd/mm/yyyy. Select the exam complete date and time using the Calendar function.

  7. Radiologist – Enter the Radiologist ID or click [List of Values] to select the Radiologist who is involved in completion of the examination.

  8. Media Type - Select the Media Type used for the storage of the record. Click [List of Values] to select the required Media Type. The list of values are populated with the Media Types created in the ‘Records Storage’ function.

  9. Storage Location - Select the Storage Location for the report that is prepared. Enter the Storage Location code or click [List of Values] to select the Storage Location.  

  10. Media ID - Enter the media ID on which the report is stored.

  11. Scan No – Enter the Scan Number, if any.

  12. No. of Films/Images taken – Enter the number of films or images taken during the examination.

  13. Exam Amount – The bill amount is displayed. This column is displayed if  Revenue Management module is interfaced with the Radiology Information Systems module and Requests and Results module for the system to display the bill amount as soon as an order is placed.

  14. [Edit Charge] - Click [Edit Charge] to invoke ‘Edit/Enter Charge Details’ screen.

  15. [Item Used] - Click the [Item Used] to invoke the Items Consumed for Examination screen.

    Enter the following details:

    1. Registration No – The Registration No. in the Record Exam Completion screen is displayed. This is mandatory field.

    2. Examination – The Examination code in the Record Exam Completion is displayed. This is mandatory field.

    3. Completed – Select the check box to complete the examination of the drug items.

    4. Items Used in Request – Click [Item Used in Request] to view the drug/contrast used on request during examination.

    Items for Examination

    1. Consumable Item Code – The Consumable Item Code defined in the Consumable masters is displayed. This is a mandatory field

    2. Description – The description of Consumable Item defined in the Consumable masters is displayed.

    3. Normal - The normal quantity of the contrast/drug is displayed. This is a mandatory field

    4. Actual - – The actual quantity of the contrast/drug is displayed. This is a mandatory field

    5. Batch ID – Click [List of Values] to select the Batch ID.

    6. Click [Save] or press [F10] to register the details of the examination.
