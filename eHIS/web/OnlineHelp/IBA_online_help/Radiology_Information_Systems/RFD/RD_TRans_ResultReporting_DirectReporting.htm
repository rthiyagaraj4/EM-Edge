Enter the following details:

  1. Registration No – Enter the registration number for the examination.

  2. Registration Date – Enter the registration date and time for the examination. This is a mandatory field.

  3. Section/Wing – Enter the Section/Wing wherein the examination is performed. Click [List Values] to select a section/wing. The list of values gets populated with the Sections and Operating Wings created in the Masters setup. This is a mandatory field.

  4. Appt Ref No – Enter the appointment reference number generated during the appointment booking stage.

  5. Order No – Enter the order number for the examination.

  6. Priority – Select the applicable priority for the examination from the list. The options available are Static, Urgent and Routine. These options are system-defined. This is a mandatory field.

  7. Select from Worklist – Click [Select From Worklist] to get the ‘Ordered Exams/Appointments function’ screen. You can select the required exam from the displayed Worklist.

  8. Ordering Facility ID – Select the Ordering Facility ID from the list. The list is populated with the facilities created in the ‘Facility’ function of the System Manager  module.

  9. Episode No – Enter the episode number relevant to the examination.

  10. Patient ID – Enter the Patient ID or right-click ‘Right Click for User Options’ to select the patient from the standard patient lookup.  This is a mandatory field.

  11. Female Patient: Pregnant – Select the check box to indicate that the patient is a female and pregnant.

  12. LMP Date – Enter the Last Menstrual Period Date of the female patient.

  13. Referral Source – Select the Referral Source from the list. Click [List of Values] to select the Referral Source.

  14. Doctor – Enter the Doctor ID or click [List of Values] to select the Doctor. The name of the selected doctor is displayed in the adjacent field. This is a mandatory field.
