Enter the following details:

  1. Group/Examination Code – Enter the examination group code from the LOV. This field is mandatory.

  1. Section – On selecting the examination code, the section is displayed automatically.

  2. Long Description - On selecting the examination code, the long description is displayed automatically.

  3. Short Description - On selecting the examination code, the short description is displayed automatically.

  4. Auto Allocation Type (F/R) – Select the check box if you want to allocate automatically. By default, the check box is cleared. Once the check box is selected, the Period and Units field gets enabled.

  5. Period – Enter the period in numeric format up to 3 digits.

  6. Units – Select the unit value - days, weeks, or month from the drop-down list.

  7. Effective Date [From – To] – Based on the Period and Units selected, the date field gets populated automatically.
