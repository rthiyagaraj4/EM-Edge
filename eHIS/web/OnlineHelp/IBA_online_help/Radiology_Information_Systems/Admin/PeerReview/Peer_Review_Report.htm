Peer Review Report

From this function you can Print both the original result report and the peer review result report.

This function allows to print both the Original result report (O) and Peer Review result report (P) separately. This function is applicable only to users with super user rights.

Printing Peer Review Report

From the Administration > Peer Review > select the Print Peer Review Report function.

The Print Result Report Peer Review screen gets displayed.

  1. By default, the logged in facility name appears in both ‘Ordering Facility’ and ‘Performing Facility’ fields.  You can change the facility from the drop-down list if required.

  2. Enter any one of the value Order ID, Patient ID or Registration No in the respective field.  

  3. Right-click on the ‘Right Click for User Options’ function to search for a patient record using the ‘Standard Patient Lookup’ screen.

  4. Click [Search]. The Examination records matching the search criteria appear.

  5. Click [Prev] and [Next] to navigate to the previous and next registration numbers respectively.

  6. By default, the check box Select [O] is selected for printing i.e., the original result report will be printed.  

  7. If you want to print the Peer review result report, select the Select [P] check box and the Select [O] check box gets cleared automatically.

  8. Click [Run] to generate the report. You can either View, Print or Save the report in a file in the local machine.

  9. Click [Reset] to clear the entered details.
