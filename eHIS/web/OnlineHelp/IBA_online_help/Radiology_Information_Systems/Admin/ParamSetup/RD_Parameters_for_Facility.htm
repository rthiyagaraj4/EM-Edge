RD Parameters for Facility

From this function you can Define Parameter values that suit the requirements of the logged in facility.

RD Parameters for facility function enables you to maintain the Radiology Information Systems module control parameters record applicable at Facility level. These parameters control the behavior of few functions and the parameters need to be set up before any operations are performed. These are crucial details, which are required for Radiology transactions and access to this function must be restricted to authorized users only. On defining the parameter rules, you cannot modify or edit the recorded values.

Defining RD Parameters for Facility

From the Administration > Parameters Setup, double-click RD Parameters for Facility function.

The RD Parameters screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Enter the details in the mandatory (*) fields of the tab pages.

Report Texts

Enter the following details:

  1. Default Auto Report Text – Enter the Default Auto Report Text in the free text format. Double-click the text box to invoke the editor. Click [Close] to exit the editor.

  2. Preliminary Report Text Editor – Enter the Preliminary Report Text in the free text format. Double-click the text box to invoke the editor. Click [Close] to exit the editor.

  3. Department Name in Result Report - Enter the name of the Department to be printed in the result report.

  4. Footer [No.1 – No.3] – Enter the footer details that are displayed in the foot section of the report. By default, the text ‘REPORT IS SIGNED ELECTRONICALLY’ is displayed in this field

  5. Message Line – Enter the message to be displayed in footer section of the report.

Records Storage

The tabs Parameters and Notice Text are displayed on selecting Records Storage Tab. Enter the details in the following tab pages.

Parameters Tab

Notice Text Tab

  • Notice Text – Enter the Notice Text in alphanumeric format or double-click on the text box to invoke the report text editor. Click [Close] to exit the report text editor.

 

Waiting Time

Waitlist Numbering

Enter the following details:

  1. Next Waitlist No. – Enter the Next Wait List Number in numeric format of eight digits maximum length.

  2. Max Waitlist No. – Enter the Maximum Wait List Number in numeric format of eight digits maximum length.

  3. Waitlisting Applicable For:

  4. Inpatient – Select the check box, if the wait listing is applicable for Inpatients.

  5. Outpatient – Select the check box, if the wait listing is applicable for Outpatients.

  6. Daycare Patient – Select the check box, if the wait listing is applicable for Daycare Patients.

  7. Emergency Patient – Select the check box, if the wait listing is applicable for Emergency Patients.

Alert to Patients

Enter the following details:

  1. Reminder Notification via SMS - Select the check box to send reminders about scheduled, cancelled, rescheduled, waitlist appointments to patients via SMS..

  2. Days before appointment - Enter a number of days in three digit maximum length, based on which automatic reminder is to be sent to the patients via SMS before appointment date. This field becomes mandatory only when the ‘Reminder Notification via SMS’ check box is selected. The numeric value starts from 000 to 999.

Alert applicable for

  1. Appointment Scheduled – Select this check box to send  SMS alert about appointment scheduled before the days entered in the ‘Days before appointment date’ to the patient.

  2. Appointment Rescheduled – Select this check box to send  SMS alert about appointment rescheduled to the patient before the days entered in the ‘Days before appointment date’ to the patient.

  3. Appointment Cancelled – Select this check box  to send  SMS alert about cancellation of the appointment before the days entered in the Days before appointment date to the patient.

  4. Waitlist Appointment Scheduled – Select this check box to send  SMS alert about the waitlist appointment scheduled before the days entered in the ‘Days before appointment date’ to the patient.

 

An SMS alert notification will be sent for Bulk Reschedule/Cancellation and Recall Cases. An SMS alert is send if the performing location is changed. The reminder notification contains the Patient Name, Exam Description, Date and Time, RD Location, Appointment Number, Reason for Cancellation/Reschedule, and Enquiry Number of Hospital.

  1. Click [Save] or press [F10] to save the RD Parameters for Facility rules or click [Clear Record] to clear the entered details.
