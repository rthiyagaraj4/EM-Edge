Study Location – Time Table

From this function you can Create a timetable for a study location and Query for the existing timetable.

The Study Location - Timetable function enables you to create the timetable for a Study Location, scheduling a list of events with the time when they take place. You can define the start and end time, break timings, time per slot and the maximum number of over bookings allowed for the Study Location. You can set the timetable for a single day or for all days of the week.

Creating a Study Location – Time Table

From the Masters > Appointments, double click Study Location-Time table function.

The Study Location - Timetable screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Descriptions>

  2. Click [Save] or press [F10] to save a created Study location – Timetable or click [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for the existing records.
