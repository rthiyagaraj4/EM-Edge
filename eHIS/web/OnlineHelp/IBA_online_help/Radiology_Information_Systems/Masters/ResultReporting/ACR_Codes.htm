ACR Codes

From this function you can Create ACR Codes for coding diagnoses, imaging findings and for coding all the imaging procedures.

ACR is an acronym for American College of Radiologists. ACR codes are used for coding diagnoses and imaging findings and for coding all imaging procedures. This is a standard for codes and is predefined. According to the pathological reasons, the Radiologist or the Reporting Technologist can select the code during result reporting.. Radiologists or reporters can view ACR Codes in ‘Authorize / Review by Radiologist’ ‘Report / Authorize Unreported Exams’ and ‘ACR Codes’ function in Transactions.

The ACR Codes defined can be associated to an examination in ‘Authorise/Review by Radiologist’ function and ‘ACR Codes’ function while generating a report for the Radiology Examination.

Creating an ACR Code

From the Masters > Result Reporting, double-click ACR Codes function.

The ACR Codes screen gets displayed.

  1. Right click under the field provided for Anatomic Codes or Pathologic Codes.

Create

You can Create an ACR code by providing description and level of the ACR code.

  1. ACR Code – Enter the ACR Code in alphanumeric format of one character maximum length.

  2. Description – Enter the description of the ACR Code in alphanumeric format.

  3. Level – The level is displayed and defaulted.

  4. Parent ACR – If the record is created under a Parent ACR, the name of the parent ACR is displayed and made default.

Create Next Level

You can Create a sublevel of an ACR code according to the parental level created by providing description and sublevel of the ACR code.

  1. ACR Code – Enter the ACR Code in alphanumeric format of one character maximum length.

  2. Description – Enter the description of the ACR Code in alphanumeric format.

  3. Level – The level is displayed and defaulted.

  4. Parent ACR – If the record is created under a Parent ACR, the name of the parent ACR is displayed and made default.

Copy

You can copy a created ACR code.

  • Right click on the ACR Code and select [Copy].

  • Select the ACR Code and click [Create].

  • Click [Paste] to copy the selected ACR Code.

Modify

You can modify the description of a created ACR code.

  1. ACR Code – The ACR Code is displayed and you cannot modify the ACR Code.

  2. Description – The description of the ACR Code is displayed. You can modify the ACR Code. Enter the description of the ACR Code in alphanumeric format.

  3. Level – The level of the ACR Code is displayed and made default.

  4. Parent ACR – The name of the parent ACR is displayed. You cannot modify the Parent ACR.

Delete

Click Delete to delete a record. You are alerted by a warning message ‘Do you want to delete this record?’ Click [Yes] to delete the record or click [No] to cancel the deletion of a record.

Click  to view all the created ACR Codes under one tree. Click   to come back to the tree option once the ACR Codes are expanded for viewing.

  1. Click [Search] to search for the created ACR Codes.

    Enter the following search criteria:

    1. Search By – Select the required option. The options are ‘Anatomic’ or ‘Pathologic’.

    2. Description – Select the description from the list. The system-defined values are Starts With, Ends With and Contains.

    3. Reset – Click [Reset] to clear the entered data for querying the ACR Code.

    4. Search – Click [Search] to search the created ACR Code. The level and description of the created ACR Code is displayed.

    Click [Close] to exit from the search screen.

  2. Click [Save] or press [F10] to save a created ACR Code or click [Clear Record] to clear the entered data.
