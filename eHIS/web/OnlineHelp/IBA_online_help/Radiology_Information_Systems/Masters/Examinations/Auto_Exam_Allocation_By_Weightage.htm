Auto Exam Allocation By Weightage

From this function you can define and allocate radiology examinations to radiologists based on weightage.

The Auto Exam Allocation by Weightage function is used to define and allocate the examinations to radiologists based on the weightage. Based on the exam allocation, the radiologist name is displayed by default in Register Examination screen. The allocation is based on Facility/Radiologist combination.

 

The sequence for auto-allocation for exams driven with respect to weightage:

1. Auto Exam Allocation By Weightage function

2. Auto Exam Allocation function

3. Examination Codes function

4. Sections for Operating Unit / Wing function

5. Operating Units / Wings function

 

If there is no Radiologists Entry in ‘Exam Auto-Allocation By Weightage’ master and if workload count is equal, then system follows the alphabetical order of the radiologists.

Allocating Examinations for Radiologists based on Weightage

From the Masters, double-click Auto Exam Allocation By Weightage function.

The Auto Axam Allocation By Weightage screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

Weightage tab

This tab allows to enter the weightage for the radiologist.

1. Operating Facility ID, Radiologists ID and Radiologist Description gets populated automatically from the ‘Radiologists By Facility’ tab in Group Examination function. User cannot add or delete these details.

2. Weightage – Enter the weightage to be assigned to the radiologists. If the weightages are same, then system considers the radiologists by alphabetical order while assigning. For weightage configuration both the ‘Auto-Allocate Type’ and ‘Period/Units’ fields are mandatory.

Allocation Sequence tab

This tab allows to enter the sequence to be considered while assigning radiologist.

1. Operating Facility ID, Radiologists ID and Radiologist Description gets displayed from the Weightage tab.  

2. Radiologist Desc – The radiologists description column is highlighted in different colours to indicate whether the radiologist is allocated or not.

3. Sequence – Enter the sequence number to be considered while assigning  radiologists.

  1. Click [Save] or press [F10] to save the data or click [Clear Record] to clear the entered details.

  1. Click [Execute Query] or press [F8] to search for the existing data.
