Views

From this function you can Create a view of the organ/body part and Query for the existing View record.

View is a way of showing or seeing something, from a particular position or angle. The Views function enables you to define the various views of the organ/body part. Views are associated to examinations. A View Code is used to uniquely identify a specific view for an examination. Some examples of views are Lateral View, Axial View, Oblique View, Bennets View, Sky line View and so on.

Creating a View

From the Masters > Examinations, double click Views function.

The Views screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Descriptions>

  2. Click [Save] or press [F10] to save a created View or click [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for existing records.

If the mandatory values are not entered, the system alerts you to enter them. The mandatory field is highlighted with a unique color code. Only 12 records can be added at a time. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate codes with a unique color code.
