File Type

From this function you can Create a new File Type applicable to a Facility and Query for the desired File Type created.

The File Type defined is associated to various Patient Classes. The File Type created is associated to the File Storage Location in the function ‘File Type for FS location’ in the Patient File Tracking module. On association, the system prompts to select the file type for the patient while registration.

Creating a File Type

From the MP Admin Setup > MP Admin Setup, select the File Type function.

The File Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the File Type created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing File Type records.

 

This function is invoked if 'File Maintenance' for the logged in facility is specified as 'Multiple' and 'Keep File Number same as Patient ID' in the function MP Parameter function.
