MR Section

From this function you can Create a new Medical Record Section, define the properties and Query for a desired Medical Record Section.

Medical Record Sections are used to generate medical record numbers during registration in a facility. MR Sections are common across locations (Clinics/Nursing Units) within the facility when identified as Central or specific to a location when identified as Departmental.

The Medical Record number generated is influenced based on the numbering rules and prefix attributes. The MR Section created should be associated to the File Storage Locations in the Code Setup of Patient File Tracking module.

Creating a Medical Record Section

From the MP Admin Setup > MP Admin Setup, select the MR Section function.

The MR Section screen gets displayed.

  1. Click [Create] to invoke the create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the MR Section created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing MR Section records.

 

This function cannot be invoked if 'File Maintenance' for the logged in facility is specified as 'Single' in the function MP Parameter function.
