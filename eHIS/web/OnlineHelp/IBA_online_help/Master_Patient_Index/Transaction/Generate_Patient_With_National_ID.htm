Generate Patient With National ID

From this function you can Generate permanent patient ID for patient with temporary patient ID by adding the National ID details.  

This function allows you to generate permanent patient ID by adding the National ID details for the patient with temporary ID. Once the permanent ID gets generated for the patient, the existing temporary ID becomes invalid.  All the transactions associated with the temporary ID get transferred to the permanent ID. The permanent ID is generated based on Patient Series and it is the next number of existing series.

 

This functionality is applicable only if you do the below configuration in MP Parameter under Patient Administration tab:

• Select the 'Accept' check box under 'National ID Checks' header

• Select the required patient series in the 'With National ID' drop-down list under Patient Series header.

Generating Permanent Patient ID

From the MP Transactions > select the Generate Patient With National ID function.

The Generate Patient With National ID screen gets displayed.

  1. Enter the Patient ID or select the patient ID using the [?] function. It is a mandatory field.

  2. The NRIC field gets enabled once the Patient ID is entered and the cursor focus is moved to NRIC field. It is a mandatory field.

  3. Click [Apply] to generate a permanent ID for the temporary patient or [Reset] to clear the entered data. Once the permanent ID gets generated for the patient, the existing temporary ID becomes invalid.  All the transactions associated with the temporary ID get transferred to the permanent patient ID.

 

The systems prompts an error message when the National ID or Patient Series is not configured in MP Parameter as well as when the patient is deceased, inactive or suspended.
