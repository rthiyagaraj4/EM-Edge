Upload Insurance Document

From this function you can Upload the insurance related documents against a patient encounter.

This function allows the user to upload the insurance related documents for the patient against the encounter. The uploaded documents are shown as attachments in the Insurance Authorization Form. The Uploaded document can be in jpeg, jpg, gif and PDF format.

You can upload five insurance documents in this function for an encounter. The recent uploaded document will be considered as the latest document.

If there is no insurance for the encounter, system will not allow the user to select the insurance and the user cannot upload the document.

Uploading Insurance Documents

From the MP Transactions > select the Upload Insurance Document function.

The Upload Insurance Document screen gets displayed.

  1. Enter the required patient ID or select the patient ID from the lookup. This field is mandatory.

  2. Click [Search]. The system lists all open and closed encounters of the patient along with the patient demographic details.

  3. Against the required Encounter ID, select the Insurance from the drop-down list.

If the payment type is cash, the message ‘The Encounter Payment Type is Cash’ gets displayed below the drop-down list. The Browse and Upload buttons will be disabled against this encounter ID.

  1. Click [Browse] to select the insurance document and click [Upload] to upload the selected documents. This field is mandatory.

  2. Once the document is uploaded for the encounter, the message ‘Operation Completed Successfully’ gets displayed. A hyperlink View gets displayed against the uploaded document.

  3. Click the View hyperlink to open the uploaded documents.
