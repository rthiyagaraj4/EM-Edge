Rapid Registration

From this function, you can Register a new patient in the facility by capturing minimum details. You can also Query for the registered patient details.

Rapid Registration may be performed due to non-availability of resources. Other details of the patient are recorded in the Change Patient Details function and patient demographics are updated.

During registration process, a Patient Series is assigned to the patient based on which the Patient ID is generated. Medical Record Number is generated based on the MR Section linked to the assigned Patient Series. A permanent file storage location is selected for storing the generated file, if Patient File Tracking module is interfaced. On generation of patient file, online reports applicable to the assigned Patient Series are listed to select the required reports by specifying the number of copies to be generated.

Performing Rapid Registration

From the MP Transactions, select the Rapid Registration function.

The Rapid Registration screen gets displayed.

  1. Click [Create] to invoke the create screen.

 

Based on the MP Parameter Setup, perform any one of the following:

Registering Patient Using Alternate ID Check

If external source is interfaced, on entering the Alternate ID, the patient’s demographics available for the entered ID gets displayed in the ‘Register Patient’ screen (the routine interfaces the MP module with the external source).

  1. Click [Create].

  2. If Default Patient Series (MP Parameter > Patient Administration Tab > Patient Series ‘Default’) is set as Alternate ID 1, the Alternate ID entry screen gets displayed.

  3. Enter the details in the mandatory (*) fields. <Related Field Description>

  4. Click [OK] to proceed with Register Patient screen or [Cancel] to cancel the Alternate ID entry. The Rapid Registration screen gets displayed.

 

Registering Patient Using National ID Check

If external source is interfaced, on entering the National ID, the patient’s demographics available for the entered ID get displayed in the ‘Register Patient’ screen (the routine interfaces the MP module with the external source).

  1. Click [Create].

  2. If Default Patient Series (MP Parameter > Patient Administration Tab >  Patient Series 'Default’) is set as National ID, the National ID entry screen (as set in Prompt of MP Parameters) gets displayed.

  3. Enter the details in the mandatory (*) fields. <Related Field Description>

  4. Click [OK] to proceed with Register Patient screen or [Cancel] to cancel the National ID entry. The Rapid Registration screen gets displayed.

 

 

If Default Patient Series (MP Parameter > Patient Administration Tab > Patient Series ‘Default’) is set as General, the Rapid Registration screen displays to register the patient with minimal details.

 

Rapid Registration

The screen displays the following Tabs: Demographic Details, Others, and Financial Details.

 

  1. Enter the details in the mandatory (*) fields of the following tab pages:

 

Demographic Details

  1. By default, this tab is displayed to enter the demographics details of the patient.

  2. Enter details in the mandatory (*) fields. <Related Field Description>

The details are updated in ‘Change Patient Details’ function.

Others

  1. Click Others tab to capture the Next of Kin and Patient Address details.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

Financial Details

  1. Click Financial Details Tab to record the financial details of the patient.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

The details are updated in ‘Change Patient Details’ function.

 

  1. Click [Apply] to save the Patient Registration or [Reset] to clear the entered data.

  2. Create Patient File screen gets displayed, if the file management parameters are set for the facility. Enter the details in the mandatory (*) fields. <Related Field Description>. Click [OK] to generate file number or [Cancel] to cancel file generation.

  3. If internal reports are defined, the Internal Reports screen gets displayed, click [OK] to generate Internal Reports or [Cancel] to exit from the screen.

  4. Patient ID and File Number is generated and displayed.

  5. Click [Query] to search for existing Patient details.
