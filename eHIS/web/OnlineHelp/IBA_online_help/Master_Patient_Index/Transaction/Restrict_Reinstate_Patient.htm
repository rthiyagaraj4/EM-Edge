Restrict/Reinstate Patient

From this function you can Restrict or Reinsatet a patient.

This function allows to restrict or reinstate a patient. It is done at patient level. The restricted patients are visible in new colour code (Black) in the ‘Patient Search’ screen.

Only authorized practitioners can view the clinical data of the restricted patient in CA and PAS screen. The practitioners are given access rights to view the restricted patient’s clinical data in the ‘Access Rights to view clinical data for Restricted Patients' in CA.

Restricting/Reinstating a Patient

From the MP Transactions, select the Restrict/Reinstate Patient function.

The Restrict/Reinstate Patient screen gets displayed with the ‘Patient Search’ dialog invoked.

  1. Search for the required patient record and select it. The selected patient demographics appear at the header.

  2. User can perform two actions:

Restricting a Patient

  1. Status - Displays the current status (reinstate) of the patient. 

  2. Action - Displays the action to be performed when applying change. This field is non-editable and disabled.

  3. Reason - Select the required reason for changing the status from the drop-down list. The reasons are configured in the ‘Reason for Restriction’ master. The default value is 'Select'. This field is mandatory.

  4. Restriction Remarks - Enter the remarks for restricting the patient in alphanumeric format up to 500 characters. This field is non-mandatory.

  5. Click [Apply]. The status of the patient ID gets changed from reinstate to restrict.

Reinstating a Patient

  1. Select a restricted patient ID from the Search Patient dialog. The patient demographics appear in the header of the screen.

  2. Status - Display of the current patient ID status (Restricted)

  3. Audit Trail button – Click this button if you want to know the user and date/time when the ‘Restriction’ action was performed. On clicking this button, the Audit Trail dialog gets displayed with Date/Time, User ID, Action and Reason details.

  4. Action – Displays automatically displaying the action to be performed when applying changes (Reinstate).

  5. Restriction Remark - Displays the restriction remarks saved while restricting the patient in the text box.

  6. Reinstatement Remark – Enter the remarks for reinstating in alphanumeric format up to 500 alphanumeric.

  7. Click [Apply], the status of the patient ID is changed from restricted to reinstate.

  8. While querying patient details in the Patient Search dialog, the patient is now not marked as restricted and black colour indicator is removed.

  1. Click [Query] to invoke the ‘Patient Search’ dialog to search for the next patient.
