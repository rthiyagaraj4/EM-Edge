Birth Place

From this function you can Create and maintain various places of birth applicable to the enterprise and Query for the desired places of birth.

Birth Place of a patient is recorded as a part of demographics during registration.

Creating a Birth Place

From the MP Code Setup > MP Code Setup, select the Birth Place function.

The Birth Place screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Birth Place created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Birth Place records.
