Relationship Level

From this function you can Define Relationship Levels to a Relationship.

The Relationship Level is used while Registering Patients in the Register Patient function.

Defining Relationship Level

From MP Code Setup > MP Code Setup, select Relationship Level function.

The Relationship Level screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the 'Relationship' using [?] function. All the Relationships defined in the Relationship function are displayed for selection. This field is mandatory.

  • If Relationship defined with Level 1 is selected, enter Code and Description for the Relationship. Click [Add] to add the Relationship. Click [Clear] to clear the entered values. On adding, click Code hyperlink to modify the description of the Relationship. Modify the description and click [Modify]. Click [Apply] to save the changes.

  • If Relationship defined with Level 2 is selected, select the 'Linked Relationship' using [?] function. All Relationships defined with Level 1 in the Relationship function are displayed for selection. This field is mandatory. Enter the Code and Description. Select the 'Linked Relationship Level' using [?] function. This field is mandatory. All the Relationship Types added to the selected Linked Relationship in this function are displayed for selection. Click [Add] to add the Relationship or [Clear] to clear the entered values. On adding, click Code hyperlink to modify the description of the Relationship. Modify the description and click [Modify]. Click [Apply] to save the changes.

  • If Relationship defined with Level 0 is selected, you cannot proceed further.

  1. Click [Apply] to save the Relationship Level or [Reset] to clear the displayed data.
