Contact Addresses

 

  • Residence Address - Enter the residential address details of the Patient.

  • Mailing Address - Enter the mailing address of the patient.

  • Contact Modes – Enter the Residence number of the patient.

  • Next of Kin – Enter the address of the ‘Next of Kin’ of the patient.

 

Residence/Mailing Address

 

  1. Address Line (1 to 4) – Enter the Address details in alphanumeric format of 30 characters maximum length in each line. The legends are set in System Manager module.

  2. Area - Enter the name of the Area in alphanumeric format of 30 characters maximum length or select it using the function [?]. The field legend is set in System Manager module.

  3. Town - Enter the name of the Town in alphanumeric format of 30 characters maximum length. The entered name must be listed under the selected area. If the selected area has more Towns attached to it, select the required town using the function [?]. The field legend is set in System Manager module.

  4. Region – Enter the Region in alphanumeric format of 30 characters maximum length or select it using the function [?]. The Area must correspond to the entered Region. The field legend is set in System Manager module.

  5. Pin Code - Enter the Postal Code in alphanumeric format of eight characters maximum length. The entered value must correspond to the selected town/city. Select the required code using the function [?].

  6. Country - Enter the name of the Country in alphanumeric format of 15 characters maximum length or select it using the function [?].

  7. Copy Residence Address – Click on the ‘Copy Residence Address’ hyperlink to copy all the Residence Address details of the patient.

 

Contact Modes

 

The Contact Modes for communication are captured under this column.

  1. Primary Resi No - Enter Primary Residence Number in alphanumeric format of 20 characters maximum length.

  2. Other Resi Number - Enter Other Residence Number in alphanumeric format of 20 characters maximum length.

The field legends for Address 1-4, Town, Area, State, and Postal Code are defined in Site function in System Manger Module.

 

Next of Kin

 

The details of Next of Kin are captured under this tab.

  1. Relationship – Select the Relationship of the patient with the Next of Kin from the list.

  2. Name - Enter the Name of the person to be contacted in alphanumeric format of 80 characters maximum length or Click Copy Patient.

  3. Address Line (1 to 4) – Enter the Address details in alphanumeric format of 30 characters maximum length in each line. The legends are set in System Manager module.

  4. Area - Enter the name of the Area in alphanumeric format of 30 characters maximum length or select it using the function [?]. The field legend is set in System Manager module.

  5. Town - Enter the name of the Town in alphanumeric format of 30 characters maximum length. The entered name must be listed under the selected area. If the selected area has more Towns attached to it, select the required Town using the function [?]. The field legend is set in System Manager Module.

  6. Region – Enter the name of the Region in alphanumeric format of 30 characters maximum length or select it using the function [?]. The area must correspond to the entered state. The field legend is set in Master Patient Index module.

  7. Pin Code - Enter the Pin Code in alphanumeric format of eight characters maximum length. The entered value must correspond to the selected Town. Select the required code using the function [?].

  8. Country - Enter the name of the Country in alphanumeric format of 15 characters maximum length or select it using the function [?].

  9. Telephone Res – Enter the Residence Telephone Number of Next of Kin in alphanumeric format of 20 characters maximum length.

  10. Office Phone No - Enter Office Telephone Number of Next of Kin person in alphanumeric format of 20 characters maximum length.

  11. Copy Patient Address - Click on ‘Copy Patient Address’ hyperlink to copy address details of the Patient for the Next of Kin.

  12. Copy Head of Family – Click on ‘Copy Head of Family’ hyperlink to copy all the Head of Family details of the patient for the Next of Kin.

The field legends for Address 1-4, Town, Area, State, and Postal Code are defined in Site function in System Manager Module.
