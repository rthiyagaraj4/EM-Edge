Enter the following details to create an Organization record:

 

  1. Sub Unit ID – Enter the Sub Unit ID in alphanumeric format of 10 characters maximum length.

  2. Name – Enter the Name in alphanumeric format of 30 characters maximum length.

  3. Contact Name – Enter the Contact Name in alphanumeric format of 10 characters maximum length.

  4. Address Line [1-4] – Enter the address details in alphanumeric format of 30 characters maximum length. The legends are set in System Manager module.

  5. Town – Enter the name of the Town in alphanumeric format of 30 characters maximum length or select the Town using [?] function. On selecting the Town, the Area and Region associated to the Town is displayed.

  6. Area - Enter the name of the Area in alphanumeric format of 30 characters maximum length or select it using [?] function. The field legend is set in ‘System Manager’ module.

  7. Region – Enter the Region in alphanumeric format of 30 characters maximum length or select the Region using [?] function. The Area must correspond to the entered Region. The field legend is set in System Manager module.

  8. Postal Code - Enter the Postal Code in alphanumeric format of eight characters maximum length or select the Postal Code using [?] function. The entered value must correspond to the selected Town.

  9. Country - Enter the name of the Country in alphanumeric format of 15 characters maximum length or select the Country using [?] function.

  10. Home Phone No – Enter the Home Phone Number in alphanumeric format of 20 characters maximum length.

  11. Office Phone No – Enter the Office Phone Number in alphanumeric format of 20 characters maximum length.
