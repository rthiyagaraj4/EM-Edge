Enter the following details as search criteria to select the required Patient ID:

 

  1. Patient ID – Enter the newborn Patient ID in the format as defined in the MP Parameter setup.   

  2. Gender – Select the Gender from the list. The listed values are All, Male, Female and Unknown.

  3. Birth Date – Enter the Date of Birth in the date format of dd/mm/yyyy. Specify the Date of Birth range (From and To date), or use the Calendar function to select the required date.

  4. Mother Patient ID – Enter the Mother Patient ID in the required format or select it using the lookup function [?].The Gender in the search screen is defaulted to ‘Female’ when clicking on the lookup button to search for the Mother Patient ID.

  5. Passport – Enter the Passport details in format as defined in MP Parameter.

  6. Names Search By – Select the criteria from the list box. The values available are Baby Name, Mother Name and Father Name.

  7. Soundex – Select the Soundex type as either ‘Ethnic’ or ‘Generic’.

  8. First Name – Enter the First Name in alphanumeric format of 20 characters maximum.

  9. Second Name – Enter the Second Name in alphanumeric format of 20 characters maximum.

  10. Third Name – Enter the Third Name in alphanumeric format of 20 characters maximum length.

  11. Family Name – Enter the Family Name in alphanumeric format of 20 characters maximum length.

If ‘Accept Patient Name In Local Language’ is selected in MP Parameter, enter names in local language. The fields displayed in the screen may vary depending on the MP Parameter Setup.
