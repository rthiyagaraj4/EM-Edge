Enter the following details to create a MP Terminal Digit record:

 

  1. At the level of – Select the level from the list. The listed values are Enterprise, Facility and MR Section. This field is mandatory.

  2. Facility Name - This field is applicable only when the level is selected as either 'Facility' or 'MR Section'. All the facilities defined in Systems Manager module Facility function are listed. Select the required facility from the list. This field is mandatory.

  3. MR Section – This field is applicable when level is selected as 'MR Section'. Select the MR Section from the list. All the MR Sections defined for the selected facility are listed. This field is mandatory.

  4. Terminal Digit By – Select Terminal Digit from the listed values. The listed values are ‘Group’ and ‘Position’. This field is mandatory.
