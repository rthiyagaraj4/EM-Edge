Documents & Others tab

 

  1. Document Type – Select the Document Type from list. The document types defined in the function 'Document Type' are listed.

  2. Document No – Enter the Document Number in alphanumeric format of 15 characters maximum length. This field is enabled on selecting the document type.

  3. Expiry Date – Enter the Expiry Date in dd/mm/yyyy format or use the Calendar function to select the required date.

  4. Place of Issue – Enter the Place of Issue of the specified document in alphanumeric format of 30 characters maximum length.

  5. Issue Date – Enter the Issue Date of the specified document in dd/mm/yyyy format or select it using the Calendar function.

  6. Browse – Click [Browse] to upload the documents of the patient. The file types supported are JPEG, GIF and PDF.

  7. Scan – Click [Scan] to scan the documents. This button appears and is enabled only when the ‘Scanning Enabled’ option is done in Document Configuration screen in System Manager. On clicking [Scan], the ‘Scanner Defined UI’ dialog appear if ‘Scan UI Enabled’ option is selected in Document Configuration screen.

a. The scanned images get displayed in the top pane. If you want to clear the scanned document, select the scanned document and click [Clear] button.

b. To save the image, click [Save Image as PDF] button, the ‘Uploaded Successfully’ message gets displayed. The list of images scanned/raptured gets added as a PDF/Image file and stored in the server application directory directly or the shared folder as configured in Document Configuration screen.

• If PDF format is configured, multiple scanned pages are collated as multiple pages of a single PDF file.

• If Image format is configured, only the page scanned first will be saved as an Image file while the rest of the pages are ignored.

If both Scan and Browse options are used for the same document, the document uploaded using the Browse option always takes precedence.

During scanning, the scan content of the most recent scan applet instance will only be available. Any earlier scanned data will be overwritten with the most recent scan applet windows content.

  1. Upload Document – The path of the document to be uploaded gets displayed.

  2. View Document – Click View to view the document uploaded. The latest update gets displayed in View Patient Details query

 

Miscellaneous

 

  1. Dependency - Select the Dependency of the Patient from the list. The default value is ‘Unknown’. The listed values are pre-defined. This field is mandatory.

  2. Arrangement - Select the value from the list. This field is mandatory.

  3. Education Level - Select the Education Level of the Patient from the list.

 

Registration Source

 

  1. Informant - Enter Informant name in alphanumeric format of 20 characters maximum length.

  2. Informant Remarks - Enter Informant Remarks in alphanumeric format of 100 characters maximum length.

 

General Remarks

 

  1. General Remarks - Enter General Remarks in alphanumeric format of 500 characters maximum length.
