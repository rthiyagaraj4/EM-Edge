Residence/Mailing Address

 

  • Residence Address - Enter Address Details of the Patient.

  • Mailing Address - Enter the Mailing Address of the patient.

 

  1. Address Line (1 to 4) - Enter the address details in alphanumeric format of 30 characters maximum length in each line. The legends are set in Enterprise Configuration  module.

  2. Area - Enter the name of the area in alphanumeric format of 30 characters maximum length or select it using the function [?]. The field legend is set in System Manager module.

  3. Town - Enter the name of the Town in alphanumeric format of 30 characters maximum length. The entered name of the town must be listed under the selected area. If the selected area has more towns attached to it, select the required City/Town using the function [?]. The field legend is set in System Manager Module.

  4. Region – Enter the name of the Region in alphanumeric format of 30 characters maximum length or select it using the function [?]. The area must correspond to the entered region. The field legend is set in System Manager module.

  5. Postal Code - Enter the Postal Code in alphanumeric format of eight characters maximum length. The entered value must correspond to the selected town/city.

  6. Country - The Country name gets automatically populated when the Region is selected

  7. Contact Person - Enter the Contact Person's Name in alphanumeric format of 30 characters maximum length.

  8. Copy Residence Address – Click 'Copy Residence Address' hyperlink to copy the residence address of the patient as mailing address.

 

Contact Modes

 

The Contact Modes available with the patient for communication are captured under this column. By default, Primary Residence Number, Other Residence Number and Email are displayed for the entry. Additional contact modes can also be recorded.

 

  1. Primary Resi No - Enter Primary Residence Number in alphanumeric format of 20 characters maximum length.

  2. Mobile No - Enter Mobile Number in alphanumeric format of 20 characters maximum length.

  3. Email - Enter the Email address of the Patient in alphanumeric format of 50 characters maximum length.

 

Enter the other contact mode details in the remaining three fields by selecting the appropriate Contact Mode.

 

  1. Other Modes – Select the mode of contact from the list. The listed values are defined in the function 'Contact Modes'. Enter the contact number of the selected mode.  

 

Additional Details

 

  1. Additional Addr - Enter the Additional Address in alphanumeric format of 30 characters maximum length.
