Tender

From this function you can Create Tenders. You can Query for the existing Tenders.

The function allows capturing Tender details.

Creating Tenders

From the PO Transactions, select the Tenders function.

The Tender screen gets displayed.

  1. Click [Create] to invoke the Tender screen.

  2. Enter the details in the mandatory fields (*). <Related Field Description>

    Enter the following details:

    1. Tender Type – Select the Tender Type from the list. This is a mandatory field. Displays the Tender types as defined in the Tender Type Master Setup. The available types are Direct Tender and Open.

    2. Tender No –  This field is in disabled state. Auto-generated based on Document Number Control setup

    3. Tender Date – Select the Date by clicking the calendar function. By default, the current date is displayed. Allows retrospective entry of past dates. Does not allow future dates. This is mandatory.

    4. Purchase Unit – Select the Purchase unit for the tender. This is mandatory.

    5. Tender Ref – Enter reference for the tender, if any.

    6. Additional Details – Click Additional Details hyperlink to view or modify additional details of the tender.

      1. Tender Expiry Date – Select the expiry date of the tender using the calendar function. Tender Expiry Date should be greater than or equal to Tender Date.

      2. Tender Approval Date – Select the approval date of the tender using the calendar function. Tender Approval Date should be greater than or equal to Tender Date but not greater than Tender Expiry Date.

      3. Contract Sign Date – Select the date on which the tender contract was signed using the calendar function. Contract Signed Date should be greater than or equal to Tender Date.

      4. Tender Remarks – Enter the tender remarks in the text box provided. You can enter upto 400 characters as free text.

      5. OK – Click [OK] to close the Additional Details dialog.

       

    7. Supplier Details – Click Supplier Details hyperlink to view or modify the details of the supplier.

      1. Supplier Name – Select the supplier name from the drop-down list.

      2. Contact Person – Displays the name of the contact person. This field will be in disabled state.

      3. Contact Number – Displays the contact number of the contact person. This field will be in disabled state.

      4. Email Id – Email Id of the supplier.

      5. Select – Select one or more check box(es) to choose the supplier for the tender. If you select the check box at the header level, all the check boxes are selected.

      6. Add Row – Click [Add Row] to add a new row for including an additional new supplier.

      7. OK – Click [OK] to close the Supplier Details dialog.

       

    8. Local Currency – The Local Currency for the facility is displayed, by default.

  3. Enter the Item Details.

    1. Item – Enter the Item name or select the Item name from the Item dialog using the [?] function. This is a mandatory field.

    2. Tender Qty – Enter the Tender quantity in a numeric format as positive numerals i.e. greater than  0 only. This is a mandatory field.

    3. Tender UOM – The Tender UOM is displayed based on the Item selected. Select the UOM if multiple values are present. Else if only one value is present, choose the value as the default. This is a mandatory field.

    4. Unit Cost – Enter the unit cost of the item. You can enter upto 3 decimals. Positive numeral 0 is allowed.

    5. Net Amount–The Tender Qty Multiplied by Net Amount is displayed. The field will be in disabled state.

    6. Brand –The brand name is displayed based on the item selection. You can modify the brand name.

    7. Item Rmks – Displays the Item Rmks dialog when clicked. You can click [?] to search and select the Predefined remarks text from the Remarks dialog.

  4. Click [Add] to add the Item details to the Tender Details area. The Select check box is selected by default.

  5. You can click Item Rmks to modify the item remarks in the  Item Rmks dialog.

  6. You can click Item hyperlink to modify the item details. The item details record is selected and the record is displayed with pink background color.

  7. In the Item details area, the Add button changes to Modify button. You can edit the details and click [Modify] to save the changes or click [Cancel] to cancel the changes.

  8. You can select more than one Item records by selecting the Select check boxes. The items are selected to get included in the Tender.

  9. The Net Tender Amt is calculated and displayed.

  10. Click [Apply]. A ‘Confirm Finalize’ dialog is displayed.  Click [Yes] to finalize the tender. Once the tender is finalized it cannot be modified.

  11. Click [No] to unfinalize the tender.

  12. The Tender document number is generated for the Tender based on the rule setup.
