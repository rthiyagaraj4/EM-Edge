Enter the following details:

  1. Purchase Unit – Select the Purchase Unit from the list. This is a mandatory field.

  2. Req By Store – Select the Request By Store from the list. If any store is set as default in the ‘PO User Purchase Unit for Transactions’ screen, then that store is displayed by default. This is a mandatory field.

  3. Request Type – Select the Request Type from the list. This is a mandatory field.

  4. Doc Date – Select the Doc date by clicking the calendar function.

  5. Doc No – Enter the document number. If the document number generation rule is set as ‘Manual entry’, you can enter Doc Number up to 20 characters maximum length.  If the document number generation rule is set as ‘Autogenerate’, then this field is disabled.

  6. Doc Ref – Enter the document reference if any.

  7. Remarks – Click ‘Remarks’ hyperlink to enter purchase request remarks.
