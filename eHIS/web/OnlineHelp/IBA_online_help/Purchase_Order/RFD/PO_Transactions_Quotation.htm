Enter the following details:

  1. Supplier – Select the Supplier from the drop-down list. The Suppliers defined in the PO Code Setup are listed. This is a mandatory field.

  2. Quotation No –The Quotation No generated by system is displayed.

  3. Quotation Date – Select the Quotation Date using the calendar function.

  4. Quotation Ref – Enter the DO Reference number in the alpha numeric format of the maximum character limit.

  5. Quotation Type – Select the Quotation Type from the drop-down list. All the delivery order types are listed. This is a mandatory field.

  6. Valid Upto – Select the Valid Upto date using the calendar function. This is a mandatory field.

  7. Additional Details – Click the hyperlink Additional Details

    Enter the following details:

    1. Payment Terms – Select the Payment Terms from the drop-down list. The Payment Terms defined in the Accounts Receivable Module are displayed.

    2. Delivery Mode – Select the Delivery Mode form the drop-down list.

    3. Terms & Conditions – Select the Terms & Conditions from the drop-down list.

    4. Submission to Advt. – Select the Submission to Advt.date  using calendar function.

    5. Quot. Open Date – Select the Quot. Open Date. using calendar function.

    6. Quot. Closed Date – Select the Quot. Closed Date. using calendar function.

    7. Drug Purchasing Record Doc – Select the date of Drug Purchasing Record Doc. Using the calendar function..

    8. Clerk Record – Select the Clerk Record date using the calendar function.

    9. Initial Evaluation – Select the Initial Evaluation date using the calendar function.

    10. QC Result – Select the QC Result date using the calendar function.

    11. Checked Date – Select the Checked Date using the calendar function.

    12. Meeting Date –Select the Meeting Date using the calendar function.

    13. Record After Meeting – Select the date of Record After Meeting using the calendar function.

    14. Submitted for Approval – Select the Submitted for Approval using the calendar function.

    15. Approved Date – Select the Approved Date using the calendar function.

    16. Contact Person – Enter the Contact Person in the character format.

    17. Quotation Remarks – Enter the Quotation Remarks in the character format of 2000 character maximum character length.

  8. Transaction Currency – The Transaction Currency is displayed. You cannot modify the data.

  9. Exchange Rate - The Exchange Rate is displayed. You cannot modify the data.

  10. Local Currency – The Local Currency is displayed. You cannot modify the data.

 

Item Details

  1. Item – Enter the Item in the character format or click [?] to select the Item. The Items defined in the PO Code Setup are listed. This is a mandatory field.

  2. Quotation Qty – Enter the Quotation Qty of the Item. This is a mandatory field

  3. Quotation UOM – Select the Quotation UOM from the drop-down list. All the units of measure of the Item defined in the PO Code Setup are listed. This is a mandatory field.

  4. Unit Cost – Enter the Unit cost of the item.

  5. Gross Amount – The amount of the Item before discounts is displayed. You cannot modify the data.

  6. Discount Type – Select the Discount Type from the drop-down list. The types of Discount applicable for an item are displayed. The listed values are ‘Percentage’ or ‘Amount’. The value ‘Percentage’ is selected by default.

  7. Net Amount – The net Amount of the item after discount is displayed.

  8. Click the hyperlink More Details.

    Enter the following details:

    1. Item – The Item selected in the Quotation is displayed.

    2. Total Quotation Quantity – The total number of quotation available is displayed.

    3. Item Specification – Enter the Item Specifications in the character format of 2000 maximum character limit.

    4. Delivery Frequency Type – Select the Delivery Frequency Type from the drop-down list. The listed values are ‘Daily’, ‘Monthly’ and ‘Yearly’. The value ‘Yearly’ is selected by default.

    5. No Of Deliveries – Enter the number of deliveries in the number format.

    6. First Delivery Date – Select the First Delivery Date using the calendar function.

    7. Equal Shipment – Select the Equal Shipment from the drop-down list. The listed values are ‘Yes’ and ‘No’. The value ‘Yes’ is selected by default.

    8. Catalogue No – Enter the Catalogue No of the item in the numeric format.

    9. Country Of Origin – Enter the Country name from which the item is to be shipped.

    10. Replaceable – Select the Replaceable from the drop-down list. The listed values are ‘Yes’ or ‘No’. If the value ‘Yes’ is selected, the item is to be replaceable. If the value ‘No’ is selected, the item is not to be replaceable. The value ‘Yes’ is selected by default.

    11. Sample Provided – Select the Sample Provided from the drop-down list. The listed values are ‘Yes’ or ‘No’. The value ‘Yes’ is selected by default.

    12. Sample Qty – Enter the number of Sample Qty required. This field is enabled only if the value ‘Yes’ is selected in the Sample Provided drop-down list.

    13. TradeId – Select the TradeId from the drop-down list.

    14. Manufacturer – Select the Manufacturer from the drop-down list.

    15. Item Remarks – Enter the Remarks in the character format of 2000 character maximum length.

    16. Click [OK] to submit all the details in the Quotation screen.

  9. Click the hyperlink Selection Criteria to invoke Quotation Selection Criteria screen.

    1. Item Catalogue Required – Select the check box to set rule for selecting the quotation based on the item catalogue.

    2. Sample Required – Select the check box to set rule for selecting the quotation based on the sample.

  10. Click the hyperlink Evaluation Factor.

    1. Evaluation factor – Select the Evaluation Factor for the quotation from the drop-down list.

    2. Weightage% – Enter the Weightage for the quotation in a numeric format.

    3. Ranking – Enter the Ranking for the quotation.

    4. Eval. Remarks – Enter the Eval. Remarks of the quotation.

    5. Recommanded – Enter ‘Yes’ to recommend the quotation or ‘No’ to not recommend the quotation.

    6. Recommanded Remarks – Enter the Recommanded Remarks in the character format.

    The Quotn No. and selected item are displayed.

  11. Item Status – Select the Item Status from the drop-down list. The listed values are ‘Selected’ and ‘Rejected’. The value ‘Selected’ is selected by default.

If the PO Module Installed check box under the Entity parameter in the Inventory management module is selected, the Quotation screen is displayed.
