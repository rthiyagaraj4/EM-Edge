Stock Status By Item

From this function you can Generate the list of Stores, where an Item is stored and information on the Bin Location, Quantity of stock available and Reorder Level.

The Stock Status By Item provides additional attributes maintained against the selected Item.

Generating List of Stores Where Item is Stored

From the ST Queries, select the Stock Status by Item function.

The Stock Status by Item screen gets displayed.

  1. Enter the required Item in alphanumeric format of 60 characters maximum length or select the Item using [?] function. This is a mandatory field. Displays the UOM and Expiry of the item.

  2. In the Item Search screen, select the Item and click Search; the item details are displayed in the Stock Status by Item screen.

  3. Click [Search] to display the Stock Status of the selected Item or enter the search criteria and click [Search] to display stock details based on the entered criteria. Refer <Related Field Descriptions>.

  4. Click Store Level Status Tab to view the Stock Status of the Item at each Store. Displays details such as Store, Quantity on Hand, Available Qty, Stock Item, Master Bin Location and RO Level.

  5. Click the Store hyperlink to view the batch details in the Batch search screen. Search for the batch details as required.

You can copy the Item code from the Batch Search dialog even if the Item From field is in disabled state. You can and use it for other actions. For example, pasting to a notepad.

  1. Click More Details Tab to view the additional details of the Item.

  2. Click Consumption Tab to view the consumption details of the Item.

  3. Click [Reset] to clear the entered data.
