Request Status

From this function you can View the current status of a request and Generate the request against a Store or group of Stores or against an Item Class.

This query function also helps in listing the Items pending for issue or items that are partially issued to take appropriate measures.

Viewing Status of a Request and Generating the Request

From the ST Queries, select the Request Status function.

The Request Status screen gets displayed.

  1. Click [Search] to view all the request status or enter the search criteria and click [Search] to view the current status of a request. Click [Reset] to clear the entered data.

    Enter the following search criteria:

     

    1. Request of – Select the required value for requests of from the list. The lists of valid values are ‘All’, ‘Internal Stores’ and ‘External Stores’. The value ‘All’ is displayed, by default.

    2. Doc Type – Select the required document type from the list. All the documents associated to the transaction ‘Requisition Issue’ is displayed in the list. The Document Type, as set in ‘ST Facility Parameter’ will be displayed, by default. This field is mandatory.

    3. Doc No – Enter the document number for which the requisition status is to be viewed in numeric format of eight digits maximum length.

    4. Req By Store – Enter the Request by Store or select the Request by Store using [?] function.

    5. Req On Store – Enter the Request on Store or select the Request by Store using [?] function.

    6. Item Class – Select the required Item Class from the list. You can select an Item Class to search the requisition status generated for a particular Item Class.

    7. Item - Enter the required Item in alphanumeric format of 60 characters maximum length or select the item using [?] function.

    8. Doc Date [From-To] – Enter the document ‘From’ and ‘To’ date for which the request status is to viewed.

    9. Request Status – Select the required request status value from the list. The listed values are ‘All’, ‘Authorized at Re. By Store’, ‘Authorized at Issue Store’, ‘Partially Issued’, ‘Fully Issued’ and ‘Closed/Cancelled’.

    10. Acknowledgement Status - Select the Acknowledgement Status from the list. The listed values are ‘All’, ‘Fully Acknowledged’, ‘Partially Acknowledged’, ‘Acknowledgement Pending’, ‘Partially Acknowledged/Confirmation Pending’, ‘Fully Acknowledged/Confirmation Complete’, ‘Partially Acknowledged/Confirmation Complete’, and ‘Fully Acknowledged/Confirmation Pending’.

  2. Click ‘Doc Type’ hyperlink to view the requested Item details.

  3. Click ‘Request Status’ hyperlink to view the audit information of the request.

  4. The Acknowledgement Status is displayed.
