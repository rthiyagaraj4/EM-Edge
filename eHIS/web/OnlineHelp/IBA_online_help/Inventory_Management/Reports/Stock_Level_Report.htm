Stock Level Report

From this function you can Generate report on the Stock Level of Items in various Stores.

This report is a summary generated to provide information on the value and quantity of stock against Items. The report contains various criteria to generate Stock Level Reports against a class of Items or a group of Items.

Generating the Stock Level Report

From the ST Reports, select the Stock Level Report function.

The Stock Level Report screen gets displayed.

  1. Click [Run] to generate the report or enter the search criteria and [Run] to generate the report based on the entered criteria. Click [Reset] to clear the entered data.

  2. Select ‘Report Option’ to generate the report.

  3. Click [OK] to generate the report and spool it in the file on the print server or [Cancel] to cancel the action.
