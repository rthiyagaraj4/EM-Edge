Cancel Request

From this function you can Cancel item-level Issue Requests or Search and Cancel all pending requests for a specific item across all Issue Requests.

Item Requests can be cancelled provided such requests are not processed in Issue Store. The cancellations have to be justified by a cancellation reason without which system does not allow any cancellation.

Cancelling an Item Request

From the MM Transactions > Issue, select the Cancel Request function.

The Cancel Request screen gets displayed.

  1. Click [Query] to invoke the Query screen.

  2. Enter the  search criteria  and click [Search] to search for Issue Requests containing a one or more stock items; displays the issues requests and stock items that match the search criteria in a grid.

  3. Click Request Issue hyperlink under Doc Type to view the Stock item details. For canceling requests in scenarios, refer the below steps:

    • To cancel all the Issue Requests at a time, select the check box at the column header level.

If the Cancel Check box at the column header level is checked, all the checkboxes at document level (i.e. Issue Requests level) as well as the Line Item level (i.e Stock item level) are also checked.

    • To cancel single/multiple Issue Requests, select the respective check box(es) of the Issue Requests at the Request Issue level.
    • To cancel multiple stock items within an Issue Request, select the required check boxes at the stock item level.
    • To cancel individual stock items within a single/multiple Issue Requests, select the specific stock item from the Item search criteria to view all Issue Requests containing the specific item. Then from the search results that is displayed, click + to expand and select the check box of the stock item from multiple Issue Requests. : The application displays a note in the status bar indicating that: “When searching for Issue Requests using ‘Item’, no other Stock Items contained in the Issue Request ID are displayed”.

Enter the remarks in free text. However, you need not enter the cancellation remarks explicitly at each item level since the cancellation remarks at the Issue Request level is inherited for all the Stock items within it. You can edit the cancellation remarks for all the Stock items, as required. Click [Close] to exit from the Cancellation Remarks dialog.
