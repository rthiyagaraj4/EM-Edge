Stock Adjustment for Disposable Items

From this function you can Adjust the disposable items returned to the pharmacy without Expiry Date, Trade Name and Batch ID.

The quantity of the items returned details can be captured and disposed. The Batch ID generation set up is done before using the Stock Adjustment for Disposable Items.

Disposing the Item

From the ST Transactions > Stock Adjustment, select the Stock Adjustment for Disposable Items function.

The Stock Adjustment for Disposable screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Modify Document Details screen is displayed by default or click ‘Document’ hyperlink to modify the document details.

  3. On entering the Document Details, click [OK] to adjust the Item Quantity or [Reset] to clear the entered data.

  4. Enter the Item for Stock Adjustment in alphanumeric format of 60 characters maximum length or select the Item using [?] function. This field is mandatory.

  5. Enter details in the mandatory (*) fields. <Related Field Description>

  6. Click [Batch Search] to view the Batch details. The selected Item Batch details are displayed. Select the check box to select the Batch and click [Close] to exit from the batch search screen.

  7. On selecting the batch, you are prompted with a message ‘Batch data will be refreshed’, click [OK] to overwrite the existing batch details and the batch details are displayed in the Batch Entry area or [Cancel] to select another batch.

    Enter the following Batch details:

    1. Expiry Date – The Expiry Date of the batch is displayed.  You can modify the date by entering the date in dd/mm/yyyy format or select the date using the Calendar function.

    2. Batch ID – The selected Batch ID is displayed. You can enter new Batch ID in numeric format as set in the ‘Batch ID’ label in Item function for expiry and non-expiry based Items.

    3. Trade Name – The selected Trade Name is displayed. You can select the required Trade Name from the list. The Trade Names associated to the selected Item in Materials Base module is displayed in the list.

    4. Bin Location – The selected Bin Location is displayed. You can select the required Bin Location from the list. The Bin Locations associated to the selected Item in ‘Item for Store’ function is displayed in the list.

    5. Qty – Enter the quantity in numeric format of eight digits maximum length. The entered batch quantity must be equal to Item quantity or an error message will be displayed. You can enter both positive as well as negative quantity.

  8. Click [Add] to add the adjustment details or [Cancel] to cancel the selected Item details.

  9. On adding, you are prompted with a message ‘Average Cost will be changed from 0.0 to cost based on the entered quantity’. Click [OK] to update the Average Cost or [Cancel] to select another Item.

  10. The selected Item adjustment details are displayed in the Item Details frame on the left hand side. Click ‘Item Details’ hyperlink to modify the Item adjustment details. Click [Modify] to modify the entered details or [Cancel] to cancel the modified changes.

  11. You can select the ‘Delete’ check box to delete the selected record.

  12. Click [Apply] to finalize the stock adjustment or [Reset] to clear the entered data.

  13. The ‘Confirm Printing’ dialog is displayed based on the attribute set for the Document Type. Click [Yes] to print the Stock Adjustment details or [No] to cancel the action.

  14. Click [Query] to invoke the Query screen.

  15. The unauthorized Stock Adjustment details are displayed based on the default criteria or enter the search criteria and click [Search] to select a specific record.

    Enter the following to query unauthorized Stock Adjustment Details

    1. Doc Type – Select the Document Type from the list. The values listed are defined in the function ‘ST Document Type’ and the default value displayed is specified in the function ‘ST Facility Parameter’.

    2. Doc No – Enter the Document number in numeric format of eight digits maximum length.

    3. Store – Select the Store from the list. The Stores are with access for ‘Stock Adjustment’ transaction function is displayed.

    4. Doc Date [From-To] - Enter the date in date format of dd/mm/yyyy or select the required date using the Calendar function.

  16. On searching, the unauthorized Stock Adjustment details are displayed.

  17. Click ‘Doc Type’ hyperlink to authorize the Stock Adjustment details.

  18. Click [Apply] to finalize the Stock Adjustment details or [Delete] to delete the selected record.

  19. On clicking [Apply], the ‘Confirm Finalize’ dialog is displayed. Click [Yes] to finalize the Stock Adjustment or [No] to finalize the Stock Adjustment in the query mode.

  20. On selecting the ‘Delete’ check box, ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.
