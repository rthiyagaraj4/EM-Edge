PO Request

From this function you can raise a Purchase Order Request to the Oracle Inventory Interface.

The Purchase Order Request raised in EM Inventory is routed to the Oracle Inventory using the external Purchase Order interfacing. The Oracle Inventory on receiving the request will accept, reject or partially accept the request. The requested Purchase Order is issued or cancelled in the EM Inventory application.

Raising Purchase Order Request

From the ST Transactions > PO Request, select the PO Request function.

The PO Request screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. The Modify Document Details screen is displayed, by default or click ‘Document’ hyperlink to modify the document details.

  3. On entering the Document Details, click [OK] to raise a Request for Purchase Order or [Reset] to clear the entered data.

  4. Select the Request Type from the list. The Request Types are Urgent and Routine.

  5. Enter the Item to raise a Request for Purchase Order in alphanumeric format of 60 characters maximum length or select it using [?] function. This is a mandatory field.

  6. On selecting the Item, the Unit of Measure, Conversion Factor and Quantity on Hand is displayed by default.

  7. Enter the details in the mandatory (*) fields. <Related Field Description>

  8. Click [Add] to add the request details and the specified details are displayed in the upper half of the screen. Click [Cancel] to cancel the selected Item.

  9. Click Item Code hyperlink to modify the Item details. On entering the details, click [Modify] to modify the updated changes or [Cancel] to cancel the changes.

  10. Select the check box to select the Item as part of the request.

  11. Click [Apply] to finalize the Purchase Order request or [Reset] to clear the entered data.

  12. Click [Query] to invoke the Query screen. The Purchase Order requests, which are not finalized, are displayed.

  13. Enter the search criteria and click [Search] to display the un-finalized purchase order requests.

  14. Click Doc Type hyperlink of the Purchase Order request, click ‘Item Code’ hyperlink to Modify.

  15. Click [Apply] to finalize the Purchase Order request or [Reset] to clear the entered data.
