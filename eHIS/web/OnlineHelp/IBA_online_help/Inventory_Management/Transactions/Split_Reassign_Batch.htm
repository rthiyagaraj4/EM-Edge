Split/Reassign Batch

From this function you can Split or Reassign the batch details.

Splitting can be performed to group similar batches expiring on one particular day. For example a batch of 10 quantities, with expiry date 10/10/2003, is split into several batches: a batch of 5 quantities, with expiry date 11/10/2003, a batch of 2 quantities, with expiry date 30/10/2003, and a batch of 3 quantities, with expiry date 10/11/2003.

Splitting or Reassigning Batch Details

From the ST Transactions > Stock Adjustment, select the Split/Reassign Batch function.

The Split/Reassign Batch screen gets displayed.

  1. The Modify Document Details screen is displayed, by default or click ‘Document’ hyperlink to modify the document details.

  2. On entering the Document Details, click [OK] to adjust the Item Quantity or [Reset] to clear the entered data.

  3. Enter the Item to Split or Reassign the Batch Details in alphanumeric format of 60 characters maximum length or select the Item using [?] function. This is a mandatory field.

  4. On selecting the Item, the Unit of Measure is displayed against the Item field and the cost is displayed.

  5. Enter the details in the mandatory (*) fields. <Related Field Description>

  6. Click [Batch Search] to view the Batch details. The selected Item Batch details are displayed. Select the check box to select the Batch and click [Close] to exit from the batch search screen.

  7. On selecting the batch, the batch details are displayed in the Batch Entry area.

    Enter the following details of the batch:

     

    1. Expiry Date – The current date as the expiry date is displayed in dd/mm/yyyy format or select the date using the Calendar function. If the selected Item is expiry based, ‘Expiry Date’ is displayed as the heading or ‘Receipt Date’ is displayed for non-expiry based Items.

    2. Batch ID –Enter the Batch ID in numeric format as set in the ‘Batch ID’ label in Item function for expiry and non expiry related Items.

    3. Trade Name – The Trade Name associated to the selected batch is displayed and disabled.

    4. Bin Location – Select the required Bin Location from the list. The Bin Locations associated to the selected Item in ‘Item for Store’ function is displayed in the list. The current Bin Location is displayed and you can select the required Bin Location.

    5. Qty – Enter the quantity in numeric format of eight digits maximum length. The entered batch quantity must be equal to Item quantity or an error message is displayed.

  8. Click [Add] to add the batch details or [Cancel] to cancel the selected Item details.

  9. The selected Item details are displayed in the Item Details frame on the left hand side. Click ‘Item Details’ hyperlink to modify the Item details. Click [Modify] to modify the entered details or [Cancel] to cancel the modified changes.

  10. You can select the ‘Delete’ check box to delete the selected record.

  11. Click [Apply] to finalize Splitting or Reassigning Batch Details or [Reset] to clear the entered data.

  12. The ‘Confirm Printing’ dialog is displayed based on the attribute set for the Document Type. Click [Yes] to print the Split or Reassigned Batch details or [No] to cancel the action.
