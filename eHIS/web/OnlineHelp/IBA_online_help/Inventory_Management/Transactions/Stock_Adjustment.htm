Stock Adjustment

From this function you can Adjust the Item Quantity in a Store.

The Item Quantity is adjusted by adding an adjustment quantity to the quantity on hand or subtracting an adjustment quantity from the quantity on hand.

Adjusting the Item Quantity

From the ST Transactions > Stock Adjustment, select the Stock Adjustment function.

The Stock Adjustment screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Modify Document Details screen is displayed by default or click ‘Document’ hyperlink to modify the document details.

  3. On entering the Document Details, click [OK] to adjust the Item Quantity or [Reset] to clear the entered data.

  4. Enter the Item for Stock Adjustment in alphanumeric format of 60 characters maximum length or select the Item using [?] function. This field is mandatory.

  5. On selecting the Item, the Unit of Measure and the available stock quantity is displayed against the Item field.

  6. Enter details in the mandatory (*) fields. <Related Field Description>

  7. Click [Batch Search] to view the Batch details. The selected Item Batch details are displayed. Select the check box to select the Batch and click [Close] to exit from the batch search screen.

  8. On selecting the batch, you are prompted with a message ‘Batch data will be refreshed’, click [OK] to overwrite the existing batch details and the batch details are displayed in the Batch Entry area or [Cancel] to select another batch.

    Enter the following details of the batch:

     

    1. Expiry Date – The Expiry Date of the batch is displayed.  You can modify the date by entering the date in dd/mm/yyyy format or select the date using the Calendar function.

    2. Batch ID – The selected Batch ID is displayed. You can enter new Batch ID in numeric format as set in the ‘Batch ID’ label in Item function for expiry and non-expiry based Items.

    3. Trade Name – The selected Trade Name is displayed. You can select the required Trade Name from the list. The Trade Names associated to the selected Item in Materials Base module is displayed in the list.

    4. Bin Location – The selected Bin Location is displayed. You can select the required Bin Location from the list. The Bin Locations associated to the selected Item in ‘Item for Store’ function is displayed in the list.

    5. Qty – Enter the quantity in numeric format of eight digits maximum length. The entered batch quantity must be equal to Item quantity or an error message will be displayed. You can enter both positive as well as negative quantity.

  9. Click [Add] to add the adjustment details or [Cancel] to cancel the selected Item details.

  10. On adding, you are prompted with a message ‘Average Cost will be changed from 0.0 to cost based on the entered quantity’. Click [OK] to update the Average Cost or [Cancel] to select another Item.

  11. The selected Item adjustment details are displayed in the Item Details frame on the left hand side. Click ‘Item Details’ hyperlink to modify the Item adjustment details. Click [Modify] to modify the entered details or [Cancel] to cancel the modified changes.

  12. You can select the ‘Delete’ check box to delete the selected record.

  13. Click [Apply] to finalize the stock adjustment or [Reset] to clear the entered data.

  14. The ‘Confirm Finalize’ dialog is displayed. Click [Yes] to finalize the stock adjustment or [No] to finalize the stock adjustment in the query mode.

  15. The ‘Confirm Printing’ dialog is displayed based on the attribute set for the Document Type. Click [Yes] to print the Stock Adjustment details or [No] to cancel the action.

  16. Click [Query] to invoke the Query screen.

  17. The unauthorized Stock Adjustment details are displayed based on the default criteria or enter the search criteria and click [Search] to select a specific record.

    Enter the following criteria to query unauthorized Stock Adjustment details:

     

    1. Doc Type – Select the Document Type from the list. The values listed are defined in the function ‘ST Document Type’ and the default value displayed is specified in the function ‘ST Facility Parameter’.

    2. Doc No – Enter the Document number in numeric format of eight digits maximum length.

    3. Store – Select the Store from the list. The Stores are with access for ‘Stock Adjustment’ transaction function is displayed.

    4. Doc Date [From-To] - Enter the date in date format of dd/mm/yyyy or select the required date using the Calendar function.

  18. On searching, the unauthorized Stock Adjustment details are displayed.

  19. Click ‘Doc Type’ hyperlink to authorize the Stock Adjustment details.

  20. Click [Apply] to finalize the Stock Adjustment details or [Delete] to delete the selected record.

  21. On clicking [Apply], the ‘Confirm Finalize’ dialog is displayed. Click [Yes] to finalize the Stock Adjustment or [No] to finalize the Stock Adjustment in the query mode.

  22. On selecting the ‘Delete’ check box, ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.

Users with rights to ‘Authorize’ Stock Adjustment in the function ‘ST User Access for Store’ will have the provision to add new batches for an Item. If an unauthorized user attempts to enter a new batch for an Item in the Item Entry Area, the system will prompt you with a message.
