Enter the following details in the Item Entry Frame:

  1. UOM – The Unit of Measure of the Manufacturing Item for which the request was raised is displayed.

  2. Req Qty – The quantities of the Item require to be manufactured as was entered while placing the request is displayed.

  3. Req No – The required number of Items to be manufactured is displayed.

  4. Prepare Location – The Prepare Location identified for the preparation of the Item while generating the manufacturing request is displayed.

  5. Preparation Type – The Preparation Type selected for the Manufacturing Item is displayed.

  6. Preparation Mode – The Preparation Mode selected for the Item is displayed.

  7. Patient ID – The Patient ID is displayed, if the generated request is for Preparation Type ‘Extemporaneous’ and the patient name is displayed against it.

  8. Encounter ID – The Encounter ID of the selected patient is displayed, if the generated request is for Preparation Type ‘Extemporaneous’.

  9. Batch ID - Based on the Prepare Location selected for manufacturing the Item, the Batch ID can be system defined or user-defined. The table describes the generation of ‘Batch ID’ based on the setup in ‘Prepare Location’ function.

If ‘Generate Batch’ check box is selected in ‘Prepare Location’ function, the system automatically generates Batch ID for the manufactured Item on saving the record. The generated Batch ID is arrived at, by concatenating the Batch Series and Batch ID as set in the Prepare Location function.

If ‘Generate Batch’ check box is cleared in ‘Prepare Location’ function, you must enter the Batch ID in alphanumeric format of 10 digits maximum length.

 

  1. Expiry Date – Enter the expiry date in dd/mm/yyyy or select the required date using the Calendar function. This is a mandatory field.

  2. Bin Location – Select the required Bin Location from the list. This is a mandatory field.

  3. Trade ID – The default Trade ID (if any) applicable is displayed.

  4. Process Loss% – The Process Loss percentage as entered in ‘Assembly Details’ function is displayed. You can modify the displayed process loss in numeric format of eight digits maximum length. This filed is mandatory.

  5. Composition for (qty) - The composition value based on the process loss entered is displayed. The number of Ingredients(s) quantity is determined based on the composition quantity. This is a mandatory field.

  6. Remarks – Enter the required remarks for manufacturing the Item in alphanumeric format.

  7. Packed By – Enter the name of the person responsible for carrying out the manufacturing process in alphanumeric format of 10 characters maximum length. The logged in User Name is displayed, by default.

  8. Counter Checked – Enter the counter checked details in alphanumeric format of 10 characters maximum length.

  9. Barcode ID – Enter the Barcode ID of the item in alphanumeric format of 20 characters maximum. If the barcode of the item is scanned, then the barcode ID is populated automatically during transactions.

  10. Label – Select the check box to print the label for the Item.

  11. No of Labels - The value is computed and displayed by adding one to the ‘Actual Quantity’. You can modify this value, if the label check box is selected by entering the required number of labels in numeric format.
