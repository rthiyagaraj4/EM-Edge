Enter the following details to create a Machine record:

  1. Machine Code – Enter the Machine Code in alphanumeric format of 10 characters maximum length.  This is a mandatory field.

  2. Machine Name – Enter the Machine name in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  3. Model No - Enter the Model Number in alphanumeric format of 20 characters maximum length. This is a mandatory field.

  4. Manufacturer – Select the Manufacturer name from the list. All the records created in the Application Masters module are listed.

  5. Technical Specification – Enter the Technical Specification in the text box.

    Click Technical Specification hyperlink to view the entered specifications.

    Click [Close] to close the Technical Specification screen.

  6. Enabled – Select the check box, to enable the defined Machine Code or clear the check box to disable the defined Machine Code. The check box is selected, by default. The enabled Machine Codes are available for selection in the required functions.
