Enter the following details:

  1. Disposable – Select the Disposable check box to indicate an item as disposable item. By default, this check box is unchecked. This can be selected only if Batch ID generation setup is done and Default Duration for Expiry Date for Disposable Drugs without Expiry Date field is entered in Facility Parameter screen.

  2. Positive Adjustment - Select the Positive Adjustment check box to indicate a positive adjustment in the stock or clear the check box to indicate a negative adjustment against the Item. The negative quantity is recorded with a negative sign. If 'Positive Adjustment' is selected, a message 'This is a Positive Adjustment. Do you want to continue?' is displayed on Adding the Item for Stock Adjustment. If 'Positive Adjustment' is cleared, a message 'This is a Negative Adjustment. Do you want to continue?' is displayed on Adding the Item for Stock Adjustment. This field is disabled in the Modify mode.

  3. Adjust Quantity - Enter the quantity of Item to be added or reduced in numeric format of six digits maximum length. This field is mandatory.

  4. Cost - The Cost of the item is calculated as per the formula in 5.x.

  5. Value – Value is the multiplication of Cost and quantity.

  6. Quantity – The Quantity is computed by the system based on the Adjust Quantity entered.

  7. Remarks – The Remarks as set in ‘ST Facility Parameter’ is displayed, by default. Enter the required Remarks in alphanumeric format or select the required Remarks using [?] function. The Remarks created in the function ‘Transaction Remarks’ of Materials Base module is displayed. This is a mandatory field.
