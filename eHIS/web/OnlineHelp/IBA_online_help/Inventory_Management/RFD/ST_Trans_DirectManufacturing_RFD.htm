Enter the following details to add the Items required to manufacture the entered Item:

  1. Prepare Location – Select the required Prepare Location from the drop-down list. The Locations created using the function ‘Prepare Location’ and status as enabled are displayed. This is a mandatory field.

  2. Preparation Type – Select the required Preparation Type from the list. The list of valid values are ‘All’, ‘Pre Packing’, ‘Standard Formulary’, ‘Extemporaneous’ and ‘Others’. The value ‘All’ is displayed, by default. This is a mandatory field.

  3. Preparation Mode – Select the required Preparation Mode from the list. The list of valid values includes ‘All’, ‘Sterile’, ‘Non-sterile’ and ‘Others’. The value ‘All’ is displayed, by default.

  4. Item – Enter the required Item Name in alphanumeric format of 60 characters maximum length or select the Item using [?] function. All the Items defined, as ‘Manufacturing’ and associated to the selected Store are displayed in the search result screen. On selecting the required Item, the corresponding Unit of Measure is displayed against it. This is a mandatory field.

Click ‘Item’ hyperlink to invoke the Item Description screen. Enter the Item Description for the selected Item in alphanumeric format of 200 characters maximum length. Click [OK] to record the entered remarks or [Cancel] to exit from the screen.

  1. Req Qty- Enter the Requested Quantity in numeric format of eight digits maximum length. This is a mandatory field.

  2. External Item – This button is visible only if ‘Extemporaneous’ is selected in the Preparation Type.

    On selection of the Preparation Type as ‘Extemporaneous’ the following fields are displayed.

     

    1. Patient ID – Enter the Patient ID in alphanumeric format as was generated while registering the patient in Master Patient Index module or select the Patient ID using [?] function. This is a mandatory field.

    On entering the Patient ID, the patient name is displayed against it.

    1. Encounter ID – Select the Encounter ID of the patient from the list. The active encounters are displayed in the list.

    Click [External Item] to invoke the Manufacturing Request Details dialog.

  3. Batch ID – Enter the Batch ID for manufacturing the Item. The Batch ID generation is based on the setup in ‘Prepare Location’ function. This is a mandatory field.

  4. Expriy Date – Enter the expiry date in dd/mm/yyyy or select the required date using the Calendar function. This is a mandatory field.

  5. Bin Location – Select the required Bin Location from the list. This is a mandatory field.

  6. Trade ID – The default Trade ID (if any) applicable is displayed.

  7. Process Loss% - Enter the Process Loss percentage in numeric format. This is a mandatory field.

  8. Composition for (Qty) – Enter the composition value based on the process loss. The number of Ingredients(s) quantity is determined based on the composition quantity. This is a mandatory field.

  9. Remarks – Enter the required remarks for manufacturing the Item in alphanumeric format.

  10. Packed By – Enter the name of the person responsible for carrying out the manufacturing process in alphanumeric format of 10 characters maximum length. The logged in User Name is displayed, by default.

  11. Counter Checked - Enter the counter checked details in alphanumeric format of 10 characters maximum length.

  12. BarCode ID – Enter the Barcode ID of the item in alphanumeric format of 20 characters maximum. If the barcode of the item is scanned, then the barcode ID is populated automatically during transactions.

  13. Label – Select the check box to print the label for the item.  

  14. No of Labels – This field gets enabled only if the Label check box is selected. The value is computed and displayed by adding one to the ‘Actual Quantity’. You can modify this value if required.
