Enter the following details:

 

Transfer From

  1. Item – The Item name is displayed, if a request has been selected or enter the required Item to Transfer in alphanumeric format of 60 characters maximum length or select the required Item using [?] function. This is a mandatory field.

  2. On entering the required Item, the Unit of Measure of the Item is displayed against the Item.

  3. Repack Size – Enter the Repack Size of the selected Item in numeric format of 12 digits maximum length.

  4. Quantity - Enter the Quantity of the selected Item in numeric format of 12 digits maximum length.

 

Transfer To

The Item name is displayed, if a request has been selected or enter the required Item to Transfer in alphanumeric format.

  1. Item – The Item used for the product Transfer/Repack while raising the request is displayed or enter the required Item to be transferred in alphanumeric format of 60 characters maximum length or select the Item using [?] function. Only Stock Items can be selected. This is a mandatory field.

  2. Batch ID - Based on the Repacking Group selected for Product Transfer/Repacking, the Batch ID is system defined or user defined. The table describes the generation of ‘Batch ID’ based on the setup in ‘Repacking Group’ function.

If ‘Generate Batch’ check box is selected in ‘Repacking Group’ function, the system generates Batch ID automatically for the Item on saving the record. The generated Batch ID is arrived at by concatenating the Batch Series and Batch ID, as set in the Repacking Group function.

If ‘Generate Batch’ check box is cleared in ‘Repacking Group’ function, you have to enter the Batch ID in alphanumeric format of 10 digits maximum length.

 

  1. Expiry Date – Enter the expiry date in dd/mm/yyyy format or select the required date using the Calendar function. This is a mandatory field.

  2. Trade Name – The Trade Name of the selected ‘Transfer From’ Item is displayed and disabled.

  3. Bin Location – Select the required Bin Location from the list. The Bin Locations associated to the selected ‘Store’ is displayed in the list. This is a mandatory field.

  4. Factor - Enter the dividing factor in numeric format of 12 digits maximum length. This indicates the conversion factor which requires to be entered to transfer the From Item to the To Item. This is a mandatory field.

  5. Expected Quantity - The quantity based on the value entered for ‘Transfer From Quantity’ is displayed and disabled.

  6. Actual Quantity – The same value as ‘Expect Quantity’ is displayed. You can modify the actual quantity by entering the required quantity in numeric format.

  7. Barcode ID – Enter the Barcode ID of the item in alphanumeric format of 20 characters maximum. If the barcode of the item is scanned, then the barcode ID is populated automatically during transactions.

  8. Label – Select the check box to print the label for the Item.

  9. Label with Barcode  – Select this check box to print the label with barcode.

  10. No of Labels – The value is computed and displayed by adding one to the ‘Actual Quantity’. You can modify this value, if the label check box is selected by entering the required number of labels in numeric format.

  11. Expected Balance Quantity – The Balance Quantity of the Item expected after the Product Transfer is displayed.

  12. Actual Balance Qty – Enter the Actual Balance Quantity of the Item in numeric format of eight digits maximum length. The entered balance quantity has to be less than the Repack Factor entered or an error as shown below will be displayed.

 

Remarks

  1. Remarks – Enter the Remarks for the Product Transfer in alphanumeric format of 200 characters maximum length.

  2. Packed By – The logged in User Name is displayed.

  3. Counter Checked – Enter the counter selected details in alphanumeric format of 30 characters maximum length.

  4. Time Taken [HH:MM] – Enter the time taken for the Product Transfer/Repacking in hh:mm format.

  5. Authorized By – Enter the details of the authorizing person in alphanumeric format of 30 characters maximum length.
