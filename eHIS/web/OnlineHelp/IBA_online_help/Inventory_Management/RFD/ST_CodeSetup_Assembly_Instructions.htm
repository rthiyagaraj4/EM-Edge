Enter the following details to create Assembly Instructions records:

 

  1. Code – Enter the code in alphanumeric format of four characters maximum length. The code defined indicates the assembly instructions. Once the code is defined and saved, it cannot be changed in query function. This is a mandatory field.

  2. Description - Enter the description of the Assembly Instructions in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Enabled - Select the check box to enable the defined Assembly Instructions. Clear the check box to disable the defined Assembly Instructions. The check box is selected, by default. Only enabled Assembly Instructions are available for selection in the required functions.

You can define 10 Assembly Instruction codes at a time. You cannot duplicate the code. On saving the record, the system alerts you by highlighting the duplicate codes with a unique color code.
