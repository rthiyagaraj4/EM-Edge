Enter the following stock details:

Search criteria:

  1. Stock Status for Facility - Displays the logged in facility.

  2. Item - Enter the item name or click [?] to search and select the item name from the Item Search screen.

You can copy the Item code from Item Search dialog even if the Item field is in disabled state. You can and use it for other actions. For example, pasting to a notepad.

 

  1. UOM - Displays the unit of measure of the item.

  2. Expiry Y/N - Displays a Tick mark to indicate if the item is an expired item.

  3. Store Code [From-To] – Enter the range of Store Codes for which the Stock Status details are to be searched or select the Store Code using the [?] function. Both the values are optional.

If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

 

If the 'Restrict Viewing Stock Status' privileges are set to 'User access to stores', only the Stores to which logged in user has access are displayed in Store Code [From - To] fields.

 

  1. Stock Level - Select the level of stock in the store from the drop-down list. The listed values are: ‘Zero Stock’, ‘Stock on Hand’, ‘Below Re-order Level’ and ‘All’. The value ‘All’ is displayed, by default.

  2. Total Stock - Displays the total stock available in the store.

  3. Unit Cost - Displays the unit cost of the item.

  4. Cost Value - Displays the cost value of the item.

  5. Stock Reorder Store – Displays the stock to be re-ordered in the store.

  6. Item Class - Displays the class of the item.

  7. Pack Size - Displays the size of the item pack.

  8. YTD Cons Qty - Displays the year-to-date for the quantity consumed.

  9. MTD Cons Qty - Displays the month-to-date for the quantity consumed.

  10. YTD Issue Qty - Displays the year-to-date for the issue quantity.

  11. MTD Issue Qty - Displays the month-to-date for the quantity consumed.

  12. MTD Receipts Qty - Displays the month-to-date for the receipt quantity.

  13. Last issue Date - Displays the last issue date of the route order.

  14. Minimum Level - Displays the minimum level of the route order.

  15. RO Level - Displays the route order level.

  16. Maximum Level - Displays the maximum level of the route order.

  17. Primary Store - Displays the main store in which the stock is available.

  18. Store Level Status - Displays the status of the store level.

  19. Bin Location(First) - Displays the location of the first container which is associated to the selected item.

  20. Include Suspended Batches in Available Quantity –Select this check box if you want to include the suspended batches explicitly; it is unchecked by default.

  21. Search - Click this button to display the search results under the following tabs:

 

Store Level Status Tab:

  1. Store - Click this hyperlink; displays the Batch Search screen where you can view the batch details of the Item in a Store.  The Stores created using the ‘Store’ function are displayed in the list.

  2. Quantity On Hand – Displays the item quantity available on hand.

  3. Available Qty – Displays the item quantity available in the store.

  4. Cost value - Displays the cost of the item.

  5. Stock Item - Displays the item stock. The listed values are ‘Both’, ‘Yes’ and ‘No’. The value ‘Both’ is displayed, by default.

  6. Master Bin Location - Displays the location of the master bin. All the Bin Locations associated to the selected item in ‘Item Store Bin Location’ function is displayed in the list.

  7. RO Level - Displays the Reorder Level for the defined Item in numeric format of nine digits maximum length.

 

More Details Tab:

The details recorded while creating an Item are listed.

  1. Sole Source YN – The Items placed or received from a single manufacturer is called Sole source.  Displays ‘Tick’ mark or X mark to indicate if it is a sole source.

  2. Special Item YN - Indicates if the Invoice Type is valid for Special Items or not.

  3. Item Analysis 3 - Displays the item analysis. The Item Analysis codes are defined in the Item Analysis function.

  4. Manufacturer - Displays the manufacturer name.

  5. Item Analysis 1 - Displays the item analysis. The Item Analysis codes are defined in the Item Analysis function.

  6. Sterilizable YN - Displays ‘Tick’ mark or X mark to indicate if the item is sterilizable.

  7. Supplier - Displays the supplier name.

  8. Item Analysis 2 - Displays the item analysis. The Item Analysis codes are defined in the Item Analysis function.

  9. Item Remarks – Displays the item remarks.

  10. Item Specifications – Displays the item specifications.

 

Consumption Tab:

The details recorded while creating an Item are listed.

  1. Period – Displays the period of item consumption.

  2. Consumption Qty – Displays the quantity consumed.

  3. Consumption Cost - Displays the cost of consumption.
