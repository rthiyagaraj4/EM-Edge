The following report criteria can be specified to generate Monthly Summary Report:

  1. Report Type – Select the required report type from the list. The listed values are ‘Items Class Consolidated’, ‘Items Consolidated’, ‘Store-Item’ and ‘Store-Item Class’. The value ‘Items Class Consolidated’ is displayed, by default.

  2. Store Code [From-To] – This field is enabled for Report Type ‘Store Item’ and ‘Store Item class’. Enter the range of Store Codes for which the report is to be generated in alphanumeric format of six characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Store Code and select required values.

  3. Item Code [From-To] - This field is enabled for Report Type ‘Item Consolidated’ and ‘Store Item’. Enter the range of Item Codes for which the report is to be generated in alphanumeric format of 20 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Code and select required values.

  4. Item Class [From-To] - Enter the range of Item class for which the report is to be generated in alphanumeric format of four characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Class code and select required values.

  5. Month/Year – Enter the accounting period for which the summary report has to be generated in mm/yyyy format. The current processing period is displayed, by default. This is a mandatory field.

  6. Report Option - Select the required report option from the list. The listed values are ‘Both’, ‘Quantity’ and ‘Value’. The value ‘Both’ is displayed, by default.
