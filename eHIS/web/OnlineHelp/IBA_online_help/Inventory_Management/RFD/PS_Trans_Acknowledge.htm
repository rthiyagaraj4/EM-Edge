Enter the following details to search for a record to Acknowledge or Confirm:

  1. TRN Type – Select the required Transaction Type from the list. The Transaction Types 'Req Issue', 'Urgent Issue', 'Issue Return' and 'Stock Transfer' gets  displayed. This is a mandatory field.

  2. Document Type – Select the type of document.

  3. Action Type – Select the Action Type from the list. The Action Types listed are Acknowledge and Confirm. You can confirm trays rejected during acknowledgement of returns by the main store. ‘Acknowledge’ will be the default option. When you choose ‘Confirm’ as the Action type, the ‘From Store’ drop-down list becomes mandatory and the ‘To Store` becomes non-mandatory.  

  4. From Store – Since Returns are always made from the sub-store (i.e., Non-owner Store) to the Owner stores, the From Store drop-down lists only Non-owner Stores. This is a mandatory field.

  5. To Store –The To Store lists only Owner Stores.

  6. Store – Select the Store from the list. All the Stores for which you have access and with ‘Acknowledge’ and ‘Confirm’ rights applicable are displayed. This is a mandatory field. Select the Store from the list. All the Stores for which you have access and with ‘Acknowledge’ and ‘Confirm’ rights applicable are displayed. This is a mandatory field.

  7. From Doc Date- Enter the From date in dd/mm/yyyy format or click the Calendar function to select the required date. By default, the current date is displayed.

  8. To Doc Date - Enter the To date in dd/mm/yyyy format or click the Calendar function to select the required date. By default, the current date is displayed.

  9. Doc No – Enter the document number in numeric format of eight digits maximum length.

  10. Doc Ref – Enter the document reference number.

  11. Group – Select the group to which the document belongs.
