Enter the following details in the Document section:

  1. Doc Type – Select the required Document Type from the list. The documents associated to the transaction ‘Requisition’ are displayed in the list. The Document Type, as set in ‘ST Facility Parameter’ is displayed, by default. This is a mandatory field.

  2. Doc No - The Document Number generation depends on the selected Document Type. If the Document Number has to be generated manually for the selected Document Type, you are informed. Enter the Document Number in numeric format of eight digits maximum length. If the system generates the number, you are not allowed to enter the field value. This is a mandatory field.

  3. Doc Date - The system date is displayed, by default. You can change the document date by entering the date in date format of dd/mm/yyyy or select the required date using the Calendar function. You can modify the date only if the ‘Doc Date Modifiable Y/N’ box is selected in ST Transaction Type. The entered date must not be less than the Processing month and year as set in ‘ST facility Parameter’ and greater than current system date. This is a mandatory field.

  4. Doc Ref – Enter the Document Reference in alphanumeric format of 15 characters maximum length.

  5. From Store – Select the requisition on Store from the list. The Stores for which you have access and ‘Requisition Issues’ transaction function applicable are displayed. This is a mandatory field.

  6. To Store – Select the Requisition by Store from the list. The Stores for which you have access and ‘Requisition’ transaction function applicable are displayed. This is a mandatory field.

  7. Item Class – Select the required Item class from the list. The Item Class associated to the selected Requested By store and Requested On store are displayed in the list.

  8. Req Type – Select the required Request Type from the list. The request is generated based on the value selected. The listed values are ‘Normal Request’ and ‘Special Request’. The value ‘Normal Request’ is displayed, by default.
