The following report criteria can be specified to generate the Ageing Analysis Report:

  1. Report Type – Select the required Report Type from the list. The listed values are ‘By Item’ and ‘By Store’. The value ‘By Item’ is displayed, by default.

  2. Store Code [From-To] - Enter the range of Store codes for which the report is to be generated in alphanumeric format of six characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Store code and select required values. On selecting the Report Type as ‘Item Consolidated’ this field is disabled.

  3. Item Code [From-To] - Enter the range of Item codes for which the report is to be generated in alphanumeric format of 20 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item code and select required values.

  4. Item Class Code [From-To] - Enter the range of Item Class for which the report is to be generated in alphanumeric format of four characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Class code and select required values.

  5. Alpha code [From-To] - Enter the range of Alpha Codes for which the report is to be generated in alphanumeric format of 10 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Alpha Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Alpha Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

 

Expiry Cutoff Days

  1. (Slot Ranges) – Enter the slot ranges in date format of dd/mm/yyyy format to view the expiry of all the Items across a Store or Items within a single Store. You can specify maximum four date ranges.
