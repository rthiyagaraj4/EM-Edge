Enter the following document details:

  1. Doc Type – Select the required Document Type from the list. The documents associated to the transaction ‘Manufacturing Receipts’ are displayed in the list. The Document Type, as set in ‘ST Facility Parameter’ is displayed, by default. This is a mandatory field.

  2. Doc No. - The Document Number generation depends on the selected Document Type. If the Document Number has to be generated manually for the selected Document Type, you are informed to do it. Enter the Document Number in numeric format of eight digits maximum length. If the system generates the number, you are not allowed to enter the field value. This is a mandatory field.

  3. Doc Date - The system date is displayed, by default. You can change the Document Date by clicking on the Calendar function or entering the date in date format of dd/mm/yyyy. You can modify the date only if the ‘Doc Date Modifiable Y/N’ check box is selected in ST Transaction Type for Manufacturing Receipts. This is a mandatory field.

  4. Doc Ref – Enter the Document Reference in alphanumeric format of 20 characters maximum length.

  5. Store – The Store associated to the selected Item and logged in user is displayed. This is a mandatory field.

  6. Item Class - Select the Item Class from the list. The Item Class associated to the selected Store gets listed.
