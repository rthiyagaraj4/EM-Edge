Enter the following details:

 

  1. Positive Adjustment - Select the Positive Adjustment check box to indicate a positive adjustment in the stock or clear the check box to indicate a negative adjustment against the Item. The negative quantity is recorded with a negative sign. If 'Positive Adjustment' is selected, a message 'This is a Positive Adjustment. Do you want to continue?' is displayed on Adding the Item for Stock Adjustment. If 'Positive Adjustment' is cleared, a message 'This is a Negative Adjustment. Do you want to continue?' is displayed on Adding the Item for Stock Adjustment. This field is disabled in the Modify mode.

  2. Adjust Quantity - Enter the quantity of Item to be added or reduced in numeric format of six digits maximum length. This field is mandatory.

  3. Quantity – The Quantity is computed by the system based on the Adjust Quantity entered.

  4. Remarks – The Remarks as set in ‘ST Facility Parameter’ is displayed, by default. Enter the required Remarks in alphanumeric format or select the required Remarks using [?] function. The Remarks created in the function ‘Transaction Remarks’ of Materials Base module is displayed. This is a mandatory field.
