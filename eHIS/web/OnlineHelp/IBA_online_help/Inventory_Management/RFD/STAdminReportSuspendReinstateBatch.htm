Enter the following report criteria to generate report for Suspend/Reinstate Batches:

  1. Item Code [From-To] - Enter the Item Codes range for which the report is to be generated in alphanumeric format of 20 characters maximum length or select the Item Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  2. Store Code [From-To] - Enter the Store Code range for which the report is to be generated in alphanumeric format of six characters maximum length or select the Store Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store Code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  3. Batch ID - Enter the range of Batch ID's for which the report is to be generated in alphanumeric format of 10 characters maximum length. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Batch ID and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Batch ID. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  4. Item Class [From-To] - Enter the Item Class range for which the report is to be generated in alphanumeric format of four characters maximum length or select the Item Class using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  5. Status – Select the required request status from the list. The listed values are ‘All’, ‘Notified’, ‘Suspended’ and ‘Reinstated’. The value ‘All’ is displayed, by default.
