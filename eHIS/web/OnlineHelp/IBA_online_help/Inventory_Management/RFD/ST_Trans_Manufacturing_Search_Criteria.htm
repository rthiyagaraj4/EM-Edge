Enter the following search criteria:

  1. Store – Select the required Store from the list. Only those Stores are displayed for which you have access and ‘Manufacturing Receipts’ transaction function applicable are displayed. This is a mandatory field.

  2. Preparation Type – Select the required Preparation Type from the list. The listed values are ‘All’, ‘Pre Packing’, ‘Standard Formulary’, ‘Extemporaneous’ and ‘Others’. The value ‘All’ is displayed, by default.

  3. Prepare Location – The Prepare Locations created in the function ‘Prepare Location’ are displayed. Select the required Prepare Location from the list.

  4. Preparation Mode – Select the required Preparation Mode from the list. The listed values are ‘All’, ‘Sterile’, ‘Non sterile’ and ‘Others’. The value ‘All’ is displayed, by default.

  5. Date [From-To] – Enter the required ‘From’ and ‘To’ date in dd/mm/yyyy format or select the date using the Calendar function for which the raised request has to be viewed.

  6. Req Status – Select the required Request Status from the list. The listed values are ‘Pending’ and ‘All’. The value ‘Pending’ is displayed, by default.

  7. Req. No. [From-To] – Enter the required ‘From’ and ‘To’ request number in alphanumeric format.
