Enter the following report criteria:

  1. Transaction – Select a Transaction Type value from the list. All the transactions of Inventory Management module are displayed in the list. This is a mandatory field.

  2. Document Type – Select a Document Type from the list. All the document types associated to the selected Transaction type in ‘ST Facility Parameter’ of ST Admin setup will be displayed in the list. This is a mandatory field.

  3. Store Code - Enter the Store Code in alphanumeric format of six characters maximum length or click [?] function to query the Store Code and select required values.

  4. Doc No – Enter the document number for which for which the requisition status is to be viewed in numeric format of eight digits maximum length.

  5. Period [From-To] – Enter the document ‘From’ and ‘To’ date for which the document can be reprinted or click the Calendar function to search for the required date.

  6. Consider Reprint as Original – On generation of one copy of the document subsequent copies generated will be generated as ‘Copy/Duplicate’. Select the check box to consider the report generated as original without the ‘Copy/Duplicate’ mark.
