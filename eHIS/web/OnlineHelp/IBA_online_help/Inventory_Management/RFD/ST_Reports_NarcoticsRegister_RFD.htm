Enter the following report criteria:

  1. Store - Enter the Store code in alphanumeric format of 15 characters maximum length. Click [?] function to query the Store code and select required values. This is a mandatory field.

  2. Item - Enter the required Item in alphanumeric format of 60 characters maximum length or select the Item using [?] function.

  3. Month/Year [From–To] – Enter the ‘From’ and ‘To’ Month and Year in the format of mm/yyyy. This is a mandatory field.
