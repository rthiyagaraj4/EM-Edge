Enter the following details to raise new Manufacturing Request or modify the raised requests:

  1. Req By Store – Select the Required Store from the list. The Stores for which you have access and ‘Manufacturing Items’ transaction function applicable are displayed. This is a mandatory field.

  2. Prepare Location - Select the required Prepare Location from the list. The Prepare Locations created using the function ‘Prepare Location’ with status as enabled are displayed. This is a mandatory field.

  3. Preparation Type – Select the required Preparation Type from the list. The listed values are ‘All’, ‘Pre Packing’, ‘Standard Formulary’, ‘Extemporaneous’ and ‘Others’. This is a mandatory field.

    On selection of the Preparation Type as ‘Extemporaneous’ the ‘Create/Modify’ area displays two additional fields.

     

    1. Item – Enter the name of the required Item to be manufactured in alphanumeric format of 20 characters maximum length. The [?] is disabled for the Preparation Type as ‘Extemporaneous’. This is a mandatory field.

    2. Patient ID – Enter the Patient ID in alphanumeric format as was generated while registering the patient in Master Patient Index module or select the Patient ID using [?] function. This is a mandatory field.

    On entering the Patient ID, the patient name is displayed against it.

    1. Encounter ID – Select the Encounter ID of the patient from the list. The active encounters are displayed in the list.

     

    Click [External Item] to invoke the Manufacturing Request Details screen.

  4. Preparation Mode – Select the required preparation mode from the list. The list of valid values is ‘Sterile’, ‘Non-sterile’ and ‘Others’. The value ‘Sterile’ is displayed, by default. This is a mandatory field.

  5. Item - Enter the required Item Name in alphanumeric format of 60 characters maximum length or select the Item using [?] function. All the Items defined, as ‘Manufacturing’ and associated to the selected Store are displayed in the search result screen. On selecting the required Item, the corresponding Unit of Measure is displayed against it. This is a mandatory field.

  6. Click ‘Item’ hyperlink to invoke the Item Description screen. Enter the Item Description for the selected Item in alphanumeric format of 200 characters maximum length. Click [OK] to record the entered remarks or [Cancel] to exit from the screen.

  7. Req. Quantity – Enter the Requested Quantity in numeric format of eight digits maximum length. This is a mandatory field.
