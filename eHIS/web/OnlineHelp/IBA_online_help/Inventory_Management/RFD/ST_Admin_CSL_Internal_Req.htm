Enter the following report criteria to generate a report on all the items, which require to be procured:

  1. Item Code [From-To] - Enter the Item Codes range for which the report is to be generated in alphanumeric format of 20 characters maximum length or select the Item Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ item code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  2. Item Class [From-To] - Enter the Item Class range for which the report is to be generated in alphanumeric format of four characters maximum length or select the Item Class using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  3. Store Code – Enter the required Store value in alphanumeric format or select the required Store using [?] function. This is a mandatory field.

 

Consumption Criteria

  1. Consumption Period – Enter the consumption period in numeric format of two digits maximum length. This period is used to arrive at the ‘Average Consumption Per Day’. The average consumption per day is computed by the system using the following formula. This is a mandatory field.

                                                       Consumption for the last <n> months

Average Consumption Per Day =  ---------------------------------------------

                                                             No. of Consumed months * 30

  1. Lead Days - Enter Lead the Days in numeric format of four digits maximum length. The entered Lead Days are automatically updated in the Item function on saving the record. The Reorder Level for an Item is computed based on the Lead Days entered.

  2. Include Stock Issues – Select the check box to include Stock Issue Items. The check box is selected, by default.

  3. Min Stock Days - Enter minimum stock days in numeric format of four digits maximum length. The Minimum Stock Level for an Item is computed based on the Minimum Stock Days and Average Consumption per Day.

  4. Buffer % of Min Stock - Enter the percentage of minimum stock in numeric format of four digits maximum length. The Reorder Quantity for an Item is computed based on the Reorder Level, Percentage of Minimum Stock and Minimum Stock Level.
