Enter the following patient details in the Patient Details Entry Area:

 

  1. Sales Type – The Sale Type displays the defaulted Sale Type value in the User Access for Store screen. If required, select the Sale Type from the drop-down list. The listed values are Inpatient, Daycare, Emergency, Outpatient, Referral and Direct Sales.

  2. Discharge – The Discharge check box is enabled, if the Sale Type is ‘Inpatient’.

  3. Chargeable – If Billing is applicable, the Chargeable check box is selected and disabled.

  4. Patient ID – Enter the Patient ID as was generated while registration in length as specified in MP Parameter or select the Patient ID using [?] function. This is a mandatory field.

  5. Patient Name – The Patient Name of the selected Patient ID is displayed.

  6. Encounter ID – Select the required Encounter Number from the list. The active Encounters are listed. If the selected Sale Type is ‘Inpatient’, the Inpatient Encounters are displayed in the list, or if the selected Sale Type is ‘Outpatient’, the Outpatient Encounters are displayed in the list. If Sale Type is selected as ‘Referral’ or ‘Direct Sales’ this field is disabled. It is mandatory to enter the Encounter ID for Sale Type ‘Inpatient’ or ‘Outpatient’.

  7. $ - This icon will be visible only for the users who have the rights to view the patient’s current encounter financial details. If the Current Encounter Financial Details check box is selected in the Dispensing Rights screen, this icon with the tool tip  ‘Current Encounter Financial Details’ will be visible. Upon clicking this icon, the patient’s financial details is displayed in a separate ‘Patient Encounter Financial Details’ dialog.

  8. Remarks – Enter the required remarks in alphanumeric format of 200 characters maximum length.
