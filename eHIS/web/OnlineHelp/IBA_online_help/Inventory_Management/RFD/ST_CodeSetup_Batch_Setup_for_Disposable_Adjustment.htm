Enter the following details:

  1. Store – Select the store from the drop-down list. The default value is ‘Select’.

  2. Batch Series – Enter the batch series in the text box.

  3. Batch ID – Enter the batch ID in the numeric format of maximum 20 characters in length.

  4. Generate Batch – Select the Generate Batch check box to define the rule for generating the batch ID. This check box is cleared by default.

  5. Enabled - Select the check box to define the batch ID generation rule. Clear the check box to disable the defined rule. The check box is selected by default.
