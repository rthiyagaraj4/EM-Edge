Enter the following details:

  1. Phy Inventory ID – Select the required Physical Inventory ID from the list. This is a mandatory field.

  2. Doc Type – Select the required Document Type from the list. The Document Type as set for variance in ‘ST Parameter for Facility’ is displayed, by default. This is a mandatory field.

  3. Doc Date – Enter the document date in dd/mm/yyyy format or select the required date using the Calendar function. The current system date is made default. This is a mandatory field.
