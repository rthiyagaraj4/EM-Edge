Transaction Analysis Report

From this function you can Generate a set of reports to study the various transactions performed across Stores.

The various reports that can be generated using this function include:

Request Statistics

Request Statistics function is used to generate a report to study number of requests made for various Items from a Store. This report can be used to study request trends from various Stores for Items and plan for optimization of stock holding within individual Stores.

GRN Statistics

GRN Statistics function is used to generate a report to study number of receipts made for various Item Receipts within a Store. This report can be used to study inflow of Items within various Stores within a period range and study percentage of finalized receipts within each Store.

Unfinalized Transactions

Unfinalized Transactions function is used to generate a report on transactions that have not been finalized. This report can be used to monitor un-posted transactions based on a time frame and avoid delays in posting transactions.

Stock Status Report for Transacted Items

Stock Status Report for Transacted Items function is used to generate a report to study stock status of Items in a Store. This report can be used to study stock on hand to study stock status in terms of stock on hand, inflow and outflow during a period range.

Facility Wise Yearly Summary Report

Facility Wise Yearly Summary Report function is used to generate a summary of Item transactions within a facility on an annual basis. The summary contains information on number and value of Items received, supplied and disposed along with turn over rate and value of stock at year end.

Yearly Stock Turnover Report

Yearly Stock Turnover Report function is used to generate a report based on annual stock turnover within a facility. This report contains information on number and value of opening stock, closing stock and issues made along with the turnover rate for each year.

MFG Workload Statistics Report

MFG Workload Statistics Report function is used to generate a statistical report based on manufacturing workload within a facility.

Generating Transaction Analysis Report

From the ST Reports, select the Transaction Analysis Report function.

The Transaction Analysis Report screen gets displayed.

  1. Click [Run] to generate the report or enter the search criteria and [Run] to generate the report based on the entered criteria. Click [Reset] to clear the entered data.

  2. Select ‘Report Option’ to generate the report.

  3. Click [OK] to generate the report and spool it in the file on the print server or [Cancel] to cancel the action.
