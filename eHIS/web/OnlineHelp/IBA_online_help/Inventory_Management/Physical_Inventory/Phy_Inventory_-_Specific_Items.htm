Phy Inventory - Specific Items

From this function you can Print Stock Count Sheets for the specified Items as part of Physical Inventory Process.

Physical Inventory – Specific Items function is used for random Physical Inventory Process.

Printing Stock Count Sheets

From the ST Physical Inventory, select the Phy Inventory – Specific Items function.

The Phy Inventory – Specific Items screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Add Items] to include the Items for Physical Inventory Process.

  4. Select the check box against the desired Item to add the Item for Physical Inventory ID generation or clear the check box to exclude the Item for Physical Inventory check.

  5. Click [Apply] to generate a Physical Inventory ID or [Reset] to clear the entered data.

  6. The Confirm Finalize screen gets displayed. Click [Yes] to finalize and generate a Physical Inventory ID and print the count sheet or [No] to temporarily generate a Physical Inventory ID for count sheet preparation. On clicking [Yes], the specified Items are printed and the document will not be available in the query mode of ‘Physical Inventory Entry’.

  7. Click [Query] to search for the Items with temporary ID.
