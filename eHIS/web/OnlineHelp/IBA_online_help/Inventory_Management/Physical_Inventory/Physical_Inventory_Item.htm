Physical Inventory Entry

From this function you can Record the details of the Physical Stock check into the system.

The staff designated to perform the stock check will count the actual physical items against the count sheet provided. The observations are recorded in the count sheet. On recording the observations, you can enter the observed quantity for each Item into the system using this function against a Physical Inventory ID.   

Recording the details of the Physical Stock Check

From the ST Physical Inventory, select the Physical Inventory Entry function.

The Physical Inventory Entry screen gets displayed.

  1. Click [Query] to invoke the Query screen.

  2. Select the Physical Inventory ID from the list. The Physical Inventory ID generated in the ‘Phy Inventory - Specific Items’ function is listed in the list.

  3. On selecting the Physical Inventory ID, the Store value is displayed in the Store field and the Item details of the selected Physical Inventory ID are displayed in the function frame.

  4. Enter the Item Quantity in numeric format of 10 digits maximum length.

  5. Select the UOM value and Base UOM Qty value from respective drop-down lists. The Stock UOM defined in Item for Store, Base UOM and the Equivalent UOM defined in Item UOM Definition screen are listed in UOM field. These fields appear only if ‘Allow Equivalent UOMs for Physical Inventory Entry’ check box is selected in Facility Parameter screen.

  6. Select the Bin Location of the Item from the list. This is a mandatory field.

  7. Enter the New Batch Details in the New Batch Details frame. Click [Add] to add the entered new batch details or [Cancel] to clear the entered values. On entering the new batch details, an information icon is displayed beside the selected batch.

    Enter the following details to add a new batch:

     

    1. Batch ID – Enter the Batch ID of the Item in alphanumeric format of 10 digits maximum length. If 'Display Batch lookup in Physical Inventory Entry' is enabled selected in the Facility Parameter, you can select the batch using the [?] function. Based on the option selected in the 'Applicable Batches In Physical Inventory Entry', the batches are displayed. If 'All Batches' is selected, on searching all the batches are displayed. If 'GRN Batches Only' is selected, only GRN batches are displayed, on searching all the batches are displayed. This is a mandatory field.

    The number of days mentioned in the Default Receipt Cutoff Days for consideration of Batches for Physical Inventory field in Facility Parameter screen will reflect in the Batch Search screen in Physical Inventory Entry transaction. For example, if the receipt cut off days is entered as ‘100’, the Batch Search screen fetches all the batches for the last 100 days from the current date and displays in the search result.

    Batch ID and New Bach check box is enabled or disabled according to value selected in ‘Display Look Up in Physical Inventory Entry’ in Facility Parameter screen.

    1. Item Quantity – Enter the observed value for the Item while performing Physical Inventory in numeric format of 15 digits maximum length. This is a mandatory field.

    2. Bin Location Code – Select the required Bin Location from the list. The Bin Locations associated to the selected Store are displayed in the list. This is a mandatory field.

    3. UOM – Select the UOM value from drop-down list. The Stock UOM values defined in the Item for Store screen, Base UOM and the Equivalent UOM defined in Item UOM Definition screen are listed. This field appears only if ‘Allow Equivalent UOMs for Physical Inventory Entry’ check box is selected in Facility Parameter screen.

    4. Expiry Date – Enter the Expiry Date of the Item in dd/mm/yyyy format or select the Expiry Date using Calendar function. This is a mandatory field.

    5. Trade ID – Select the Trade ID from the list. This is a mandatory field.

    6. New Batch – The new Batch check box is selected, by default and the check box is disabled indicating that the batch is new.

    The Expiry Date, Trade ID and New Batch fields are disabled if Batch Lookup option is enabled.

    The titles in the Batch Search screen are hyperlinked, the results are sorted based on the arrow indicators. Radio button is introduced to select the batch if single batch selection is applicable.

  8. Click [Apply] to record the Physical Inventory transaction or [Reset] to clear the entered data.

Color indicators are provided against the Item name for which batch details are recorded in the Physical Inventory Entry function. The Red Color indicates that no batch details are available for the selected item-store combination, Orange color indicates that the batch details are partially updated, No color indicates that the batch details are not updated and Green color indicates batch details are completely updated.
