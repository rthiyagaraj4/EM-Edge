Define Barcode Rules

From this function you can Define rules for barcode generation.

This function allows generating the barcode manually, setting rules to generate barcode or allow system to generate the barcode automatically. The elements used for barcode generation are Item Code, Batch ID, Expiry Date, Facility ID, Trade ID, Manufacturer ID, Supplier ID, Store Code, Bin Location and MRP.

 

Once the barcode is generated, it is printed on the Item Label, Manufacturing Label and Product Repack Label. The generated barcode is given as input to inventory. The items are identified based on the barcode and the transactions are processed.

Defining Barcode Rules

From the Code Setup, select the Define Barcode Rules function.

The Define Barcode Rules screen gets displayed.

  1. Select the Generate Barcode option.

    1. Manually – Select this option to generate the barcode manually.

    2. System Generated – Select this option to allow the system to generate the barcode automatically.

    3. Predefined Rules – Select this option to set rules to generate the barcode.

    4. On selecting the Predefined Rules option, move the required barcode elements from ‘All Element’ list to ‘Barcode Elements’ list. The Item Code, Batch ID and Expiry Date are mandatory barcode elements.

  2. Click [Apply] to save the defined barcode generation rules or [Reset] to clear the entered data.
