Document Type

From this function you can Associate and maintain Document Types for all Transactions in Inventory Management module and Query for a desired Document Type.

The Document Types function is used to associate a Transaction Type for a Document Type. The Transaction Type identifies the various transactions. A Document Type is categorized using this function. The Document Type associated to a relevant Transaction Type is used in the relevant Transaction functions. The Document Types form a part of the patient file. On invoking the required transaction, the Document Type and the number get generated automatically, or you can enter them manually based on the setup in Financials Base Manager module.

Creating an Association of Document Types to Transaction Types

From the ST Administration, select the Document Type function.

The Document Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory fields. <Related Field Description>

  3. Click [Apply] to save the association of Document Type with Transaction Type or [Reset] to clear the entered data.

  4. Click [Query] to search for existing association of Document Type with Transaction Type.
