Parent Store for Store

From this function you can Associate a Store to Parent Stores and Associate the Parent and Store combination to the Item Class.

The Item Class associated to the selected Stores in the Item Class for Store function are listed for selection.

Creating an association of an Item Class to Store and Parent Store

From the Code Setup, select the Parent Store for Store function.

The Parent Store for Store screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the required Store code in alphanumeric format of 40 characters maximum length or select the Store code using [?] function. The ‘Stores’ displayed in Materials Base module are displayed. This is a mandatory field.

  3. Click [Search] to display the Parent Stores.

  4. Select the check box against the required Parent Store to associate the Parent Store to the selected Store.

  5. Click Parent Store hyperlink to display the Item Classes associated to the selected Store.

  6. Select the check box against the required Item Class to associate the Item Class to the Store and Parent Store combination. The check box is selected, by default.

  7. Click [Apply] to save the association of Item Class to Store and Parent Store created or [Reset] to clear the entered data.
