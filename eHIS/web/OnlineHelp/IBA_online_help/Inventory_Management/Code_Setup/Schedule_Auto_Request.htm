Schedule Auto Request

From this function you can Schedule and Generate automated requests for all items below re-order level and Query for existing Schedule Auto Request record.

From this function you can determine and maintain an optimum level of items in the stock.

Scheduling Auto Request

From the Code Setup, select the Schedule Auto Request function.

The Schedule Auto Request screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory fields. <Related Field Description>

  3. Click Select tab to enter the Item Details.

    Enter the following details:

     

    1. Item Code - Enter the Item Code in alphanumeric format of 20 characters maximum length.

    2. Item Description - Enter the Item description in alphanumeric format of 100 characters maximum length. Select the search criteria from the list. The values displayed are Starts with, Ends with and Contains. The value ‘starts with’ is displayed, by default.

    Click [Search] to search for the Item based on the entered criteria.

  4. On searching, select the required Item.

  5. Click Association tab to view the associated item details.

  6. Click [Apply] to save the Auto Request or [Reset] to clear the entered data.

  7. Click [Query] to search for an existing Schedule Auto Request record.
