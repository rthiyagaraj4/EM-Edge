Item Class for Store

From this function you can Associate Item Class to a Store and Query for a desired association of Item Class for Store.

The Item Class is defined in the Item Class function of Materials Base module. The Item Classes associated to the Store is made available for selection in the required transactions.

Creating an association of Item Class to a Store

From the Code Setup, select the Item Class for Store function.

The Item Class for Store screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the required Store from the list. This is a mandatory field.

  3. Click [Search] to display the list of Item Classes and select the check box against the required Item Class to associate the Item Class to the Store.

  4. Click [Apply] to save the association of Item Class to Store or [Reset] to clear the entered data.

  5. Click [Query] to search for the desired association of Item Class to Store records.
