Enter the following details:

 

  1. Product Code - Enter the Product Code or click [List of Values] to select it. The listed values are created in the Product Code function.

  2. Description - The Description of the selected Product is displayed.

  3. Volume - Enter the volume of the product to be created in numeric format with a maximum of six digits.

  4. Barcode [Unit Number] - Enter the Barcode Unit Number in the text box or scan the product code to select a particular blood group for conversion.

  5. Unit Number - The Unit Number of the Added stock is displayed.

  6. Product Code - The product code of the selected unit is displayed.

  7. Blood Group - The Blood Group of the selected unit is displayed.

  8. Rhesus Code - The Rhesus factor of the selected unit is displayed.

  9. Expiry Date - The Expiry Date of the selected unit is displayed.

  10. Volume - The Volume of the selected unit is displayed.

  11. Volume Units - The Volume units of the selected unit are displayed.
