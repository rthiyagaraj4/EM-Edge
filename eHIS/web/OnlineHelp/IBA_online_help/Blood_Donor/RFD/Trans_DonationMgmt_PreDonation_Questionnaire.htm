Enter the following details to record Questionnaire:

 

  1. Legend Code – Legend Code is displayed. The Legend code is created in the Physical Questionnaire setup function.

  2. Description – The description of the Legend code is displayed. If the question is mandatory, it is highlighted in  color.

  3. Response – Select the Response from the list of values.

  4. Accepted Y/N – Depending on the responses, the value is displayed automatically. You cannot make any changes to the displayed value.

  5. Additional Screening – Select the value from the list. The Listed values are ‘Yes’ and ‘No’. If Additional Screening is required, select ‘Yes’. Select ‘No’, if additional screening is not required.

  6. Performed By – The name of the logged in user gets displayed, by default. Click [List of Values] to change the name of the practitioner. This field indicates the name of the practitioner who has filled the Questionnaire.

  7. Performed Date - The current system date and time is displayed, by default. Click the Calendar function to update the date and time. This field indicates the date and time when the questionnaire was filled.
