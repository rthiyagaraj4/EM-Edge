Enter the following details:

 

  1. Facility – The name of the logged in facility is displayed, by default. You cannot update the value in the field.

  2. Module ID – Enter the module ID, or click [List of Values] to select the Module. This is a mandatory field.

  3. Send By – Select the medium of transfer of the report from the list. The system defined values are ‘fax’ and ‘Mail’.

  4. Report – Enter the report type, or click [List of Values] to select the report type. This is a mandatory field.

  5. File Name – Enter the report file name, or click [List of Values] to select the File Name. This is a mandatory field.

  6. Dest Type – Select the destination type from the list. The listed values are External Referral, Customer and Supplier. These are system-defined values. This is a mandatory field.

  7. Dest Code – Enter the destination code, or click [List of Values] to select the destination code. This is a mandatory field.

  8. Defined routing – Select the check box, if the report is to be sent through the already defined routing or clear the check box.

  9. Mail ID – Enter the email ID or Fax number to which the report is to be sent. If you choose mail the field name is displayed as Mail ID, whereas if you select, Fax in the Sent By option, the field name is displayed as Fax ID.
