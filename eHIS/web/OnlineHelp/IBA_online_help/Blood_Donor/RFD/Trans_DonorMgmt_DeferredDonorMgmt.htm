The donor details are displayed in the respective text boxes on selecting the donor:

 

  1. Donor ID – Enter the deferred Donor ID or click [Donor Lookup] to select the deferred donor. After selecting the Donor ID, press [Tab] to view the donor details.

  2. Donor Name – On selecting Donor ID, Donor Name is displayed.

  3. Age – The Age of the donor recorded during registration is displayed.

  4. Sex – The Gender of the donor recorded during registration is displayed.

  5. Donation Date – The Donation Date recorded during the blood donation is displayed.

  6. Unit Number – The Unit number created during the donation is displayed.

  7. Product – The Product Code associated with the unit number is displayed.

  8. Visit Date – The Visit Date of the donor is displayed.

  9. Reported Date – The Reported date of the donor is displayed.

  10. Deferral Code - Select the Deferral Code as recorded in Deferred Reasons function.

  11. Deferral Type - The Deferral Type associated with the deferral code is displayed.

  12. Deferred Till Date - Enter the Deferred till Date or select the date using the Calendar function. This is applicable only if the type of donation is temporary. Even before the Deferred till date is over, the user with appropriate rights can change the deferred till date. Refer to User Security function in the Administration section.
