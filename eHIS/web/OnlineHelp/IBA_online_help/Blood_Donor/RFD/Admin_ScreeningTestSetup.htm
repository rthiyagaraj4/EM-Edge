Enter the following details:

 

  1. Test Code – Enter a valid test code to in alphanumeric format of 10 characters maximum length.

  2. Long Name – Enter the long name of the test in alphanumeric format of 40 characters maximum length.

  3. Short Name – Enter the short name of the test in alphanumeric format of 15 characters maximum length.

  4. Type of Test – Select the Type of test from the list. The listed values are Blood Group, Rhesus, Antibody and RH Geno Type. This indicates if the defined test code is valid for Blood Group, Rhesus, Antibody or  RH Geno Type.

  5. Sex Specific – Select the gender for which the test is made, if the test is sex specific. The value ‘Both’ is displayed by default. The listed values are ‘Male’, ‘Female’ and ‘Both’.

  6. Suspend Test? – Select the check box to suspend the selected test. You cannot suspend the test if the radio buttons for Blood bag, Apheresis or Deferred Screening are selected. Clear the check box and select the appropriate radio button in Applicable for section to reinstate the selected test.

 

Applicable for

  1. Additional Tests – Select the check box to identify the test as an additional test. The blood units are sent for additional tests through the ‘Send for Additional Screening Tests’ function when the test is recorded as an additional test. You cannot send the blood units for additional tests, if the Apheresis and Deferred Screening check boxes are selected.

  2. Blood Bag – Select the check box, if the selected test code is applicable for Blood Bag screening.

  3. Apheresis – Select the check box, if the selected test code is applicable for Apheresis screening. The setup made reflects in the Donor Apheresis Screening function.

  4. Deferred Screening – Select the check box, if the selected test code is applicable for Deferred Screening. The setup made reflects in the Deferred Blood Donor function.

  5. Type of Answer – Select a value from the list based on the individual test codes selected. The system-defined values are ‘Numeric’ and ‘List of Values’.

 

Validation for Numeric

If the type of answer selected for a test code is numeric, the following fields get enabled:

  1. Low Value – Enter the lowest value acceptable for the test in alphanumeric format of 15 characters maximum length. The low value cannot be bigger than the high value.

  2. High value – Enter the highest value acceptable for the test in alphanumeric format of 15 characters maximum length. The high value cannot be lesser than the low value.

  3. Units – Click [List of Values] to select the units for the low and high values. The values defined using the ‘Units of Measurement’ function get listed.  

  4. Action – Select the action for the test from the list. The valid values are ‘Accept or ‘Reject.

 

Validation for List of Values

  1. List of Values – Click [List of Values] to select a value from the list. The values defined using the ‘Rhesus’ function get listed.

  2. Description – The description for the selected Rhesus gets displayed.

  3. Action – Select a value from the list. The listed values are ‘Reject’ or ‘Accept’. If you select Accept, the result of the test is positive and you can proceed with the transactions. If you select Reject, the result of the test is negative and you cannot proceed with the transactions.
