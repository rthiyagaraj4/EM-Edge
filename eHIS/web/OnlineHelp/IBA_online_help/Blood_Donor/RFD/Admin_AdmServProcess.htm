Select the following details to start the Server Process:

 

  1. Facility – Select the facility from the list. This field gets disabled, if the selected process is not available for the selected facility. If single facility is selected, the server process carries out the request for the selected facility only.

Note: Once the process has started for single facility, it cannot be started for all facilities. Similarly, if process has started for all facilities, it cannot be started for a single facility.

  1. Priority – Select the Priority from the list or select ‘*All’ to select all priorities for the selected facilities. This field gets disabled, if the selected process is of single instance. If single priority is selected, the server process carries out the request for that priority only. If ‘*ALL’ is selected, the server process carries out for all priorities.

Note: Once the process has started for single priority, it cannot be started for all priorities. Similarly, if process has started for all priorities, it cannot be started for a single priority.

  1. Serviced Request(s) - The service requests can be entered for the server process ‘Print Services’ only. Two services named ‘Lab Result Reporting’ and ‘Patient Bill Printing’ are available for selection. Select ‘*ALL’ to process print requests generated from both processes.

Note: This field is disabled for other processes. If ‘Lab Result Reporting’ service is selected, the ‘Print Services’ process prints the requests created by ‘Lab Result Reporting’ based on the priorities defined. If ‘Patient Bill Printing’ service is selected, the ‘Print Services’ process prints the requests created by ‘Patient Bill Printing’ process.
