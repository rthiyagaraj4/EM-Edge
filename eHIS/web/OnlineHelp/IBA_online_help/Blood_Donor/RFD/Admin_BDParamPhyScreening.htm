1. Minimum Age (Male/Female) - Enter the minimum age for male and female donors in numeric format of three digits maximum length.

  2. Minimum Age Exception - Enter the Minimum Age Exception for male and female donors in three digits maximum length. The donors falling under this category need consent of the parents/guardians for donating blood.

  3. Maximum Age (Male/Female) - Enter the maximum age for male and female donors in numeric format of three digits maximum length.  

  4. Maximum Age Exception - Enter the Maximum Age Exception for male and female donors in three digits maximum length. The donors falling under this category need consent of the parents/guardians for donating blood.

  5. Interval between Donations (WB) [Male/Female] Units - Enter the interval value between whole blood donations for male and female donors in numeric format of four digits maximum length. Select the respective unit of measurement from the list.

  6. Interval between Donations (Apheresis) [Male/Female] Units - Enter the interval value between Apheresis blood donation for male and female donors in numeric format of four digits maximum length. Select the respective unit of measurement from the list.

  7. Minimum Weight [Male/Female] Units - Enter the minimum required weight for male and female donors to donate blood. Click [List of Values] to select the required units. This is a mandatory field.

  8. Maximum Weight [Male/Female] - Enter the maximum required weight for male and female donors to donate blood. This is a mandatory field.

  9. Minimum Temperature [Male/Female] Units - Enter the minimum temperature required for male and female donors to donate the blood. Click [List of Values] to select required units. The values defined using the Unit of Measurement function get listed. This is a mandatory field.

  10. Maximum Temperature [Male/Female] Units - Enter the maximum temperature required for male and female donors to donate the blood. The values defined using the Unit of Measurement get listed. This is a mandatory field.

  11. Minimum Systolic [Male/Female] Units - Enter the minimum value of systolic blood pressure for male and female donors. Click [List of Values] to select the unit of measurement. The values defined using the Units of Measurements function get listed. This is a mandatory field.

  12. Maximum Systolic [Male/Female] - Enter the maximum value of systolic blood pressure for male and female donors. This is a mandatory field.

  13. Minimum Diastolic [Male/Female] - Enter the minimum value of diastolic blood pressure for male and female donors. This is a mandatory field.

  14. Maximum Diastolic [Male/Female] - Enter the maximum value of diastolic blood pressure for male and female donors. This is a mandatory field.

  15. Minimum Pulse [Male/Female/Units] - Enter the minimum value of pulse for male and female donors. Click [List of Values] to select the required units they can donate. The values defined using the ‘Units of Measurement’ function get listed. This is a mandatory field.

  16. Maximum Pulse - Enter the maximum value of pulse for male and female donors. This is a mandatory field.

  17. Minimum Hb Value - Enter the minimum value of haemoglobin for male and female donors. Click [List of Values] to select the appropriate units. The list of values displays the data set up in Unit of Measurements' function. This is a mandatory field.

  18. Maximum Hb Value - Enter the maximum value of haemoglobin for male and female donors. This is a mandatory field.

  19. Mandatory for Allogenic – Select this check box to set the physical screening parameter as mandatory field. By default, this check box is not selected.

  20. Mandatory for Autologus - Select this check box to set the physical screening parameter as mandatory field. By default, this check box is not selected
