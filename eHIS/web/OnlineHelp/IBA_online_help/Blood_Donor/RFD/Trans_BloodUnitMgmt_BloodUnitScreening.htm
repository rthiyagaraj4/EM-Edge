Enter the following details to send the blood unit for screening:

 

  1. Unit Source – Select the blood unit source from the Unit Source list. The list displays the values ‘In-house’, ‘Mobile’ and ‘All’. The value ‘All’ is displayed, by default. Select the required value to search for the blood units. When you select ‘In-house’, all the blood units donated at the blood bank get listed. When you select ‘Mobile’, all the blood units donated at the mobile camp get listed. If you want to send all the units for screening, select ‘All’.

  2. Stage – Select the screening level from the Stage list. The listed values are ‘Level 1’ and ‘Level 2’. ‘Level 1’ is displayed, by default.

  3. Batch Number – Enter the Batch Number to search for the blood units based on the batch defined.  

You can query the details based on the Batch Number and Donation Date range.

  1. Donation [From-To] – Enter the Donation Date range or click the Calendar function to select the date range. The blood units, which are donated during the selected date range, are listed.  

  2. Order By – Select Order By from the drop-down list. The system queries and views the unit number /Donation date based on selection of Order By. The Unit number is sorted in ascending order and Donation date is sorted in descending order.

  3. Barcode Unit Number – Enter the Barcode Unit Number in alphanumeric format of 30 characters maximum length. The Barcode Unit Number gets displayed automatically when the blood bag is scanned which is sent for screening.
