Based on the Questionnaire, risk factor or the physical checkup results, the system allows you to defer a donor.

 

Deferred?

  1. Deferral Reason – Click [List of Values] to select the Deferral Reason. The values defined using the Deferred Reason function get listed.

  2. Type - Based on the Deferral Reason selected, the value is displayed as ‘Temporary’ or ‘Permanent’.

  3. Deferred Till Date – Click the Calendar function to select the date during which the donor is not eligible for blood donation, if the donation type is temporary. Even before the deferred till date is over, the user with appropriate rights can change the deferred till date. The access rights to the user are given using the ‘User Security’ function in the Administration section.
