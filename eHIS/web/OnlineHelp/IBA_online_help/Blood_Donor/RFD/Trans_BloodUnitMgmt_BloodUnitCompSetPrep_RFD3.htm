The following details are displayed:

 

  1. Barcode Unit Number – The scanned Bar Code Unit Number is displayed.

  2. Unit Number – The selected Unit number is displayed.

  3. Component – The Component code of the selected unit number is displayed.

  4. Donation/Mobile Drive Date – The Donation Date of the selected blood unit gets displayed. If the donation was done in a mobile donation camp, the mobile donation date is displayed.

  5. Blood Bag – The Blood Bag Code and Description associated to the unit are displayed.

  6. Number of Bags – The Number of the bags required is displayed.

  7. Blood Group – The Blood Group of the blood unit is displayed.

  8. Rhesus – The Rhesus factor of the Blood Group is displayed.

  9. Preparation Date – Select the preparation date of the component from the Calendar function.

  10. Measured Weight – Enter the measured weight of the component in alphanumeric format of eight characters maximum length. The unit of the component is displayed in the next field.

  11. Product Code - The Product Code of the selected unit is displayed.

  12. Description – The respective Description of the product is displayed.

  13. Storage Location – The Storage Location of the unit/product is displayed.

  14. Expiry Date – The Expiry Date of the unit is displayed.
