The following details are displayed:

 

  1. Unit Number – The Unit Number satisfying the selected search criteria is displayed.

  2. Product Code – The Product Code of the blood unit is displayed. The Product Codes are defined in the ‘Products’ function.

  3. Description – The Description of the product as defined in the ‘Products’ function is displayed.

  4. Blood Group – The Blood Group of the selected blood unit is displayed. The blood group is recorded during pre-donation analysis in the ‘Pre Donation’ function.

  5. Rhesus Code – The Rhesus Code of the blood unit recorded during pre-donation analysis is displayed.

  6. Volume – The Volume of the blood unit added to stock is displayed.

  7. Volume Units – The Unit of the Volume is displayed. The units are defined in the ‘Unit of Measurements’ function of Application Masters module.

  8. Expiry Date – The Expiry Date of the unit defined in ‘Products’ function is displayed.

  9. Select - Select the check box against each record to send the unit for quality screening. Clear the check box, if the unit should not be sent for quality screening.

  10. Select All - Select the ‘Select All’ check box to select all the units for quality screening.

  11. [View Quality Tests] - Click [View Quality Tests] to view the list of tests performed during quality screening. The displayed tests are created in the ‘Quality Screening Test Setup’ function.

  12. [Send Selected Units for Screening] - Click [Send Selected Units for Screening] to send the selected blood units for quality screening.
