The following donor details are displayed:

 

  1. Donor ID – The Donor’s ID is displayed. The temporary and permanent deferred donors are listed.

  2. Donor Name – The name of the Reactive Donor is displayed.

  3. Donation Date – The date of donation is displayed.

  4. Sex - The gender of the donor is displayed.

  5. Age - The age of the donor is displayed.

  6. Unit Number – The Unit number is displayed.

  7. Product - The Product associated to the donor is displayed.

  8. Visit Date - The date and time of visit registration is displayed.

  9. Reported Date – The Reported date of the donor is displayed.

  10. Deferral Code - Click [List of Values] to select the Deferral Code. The values defined using the ‘Deferred Reasons’ function get listed. The Deferral Type associated to the deferral code is displayed.

  11. Deferral Type - The Deferral Type associated with the deferral code is displayed.

  12. Deferred Till Date - Enter the Deferred till Date or select the date using the Calendar function. This is applicable only if the type of donation is temporary. The user with appropriate rights can change the deferred till date before the deferred till date is over.

  13. [Antibodies] - Click [Antibodies] to view antibodies of the donor. This button gets enabled, only if any antibody details are recorded for the donor.

  14. [Antigens] - Click [Antigens] to view antigens for the donor. This button gets enabled, only if antigen details are recorded for the donor.

  15. [Donor Notes] - Click [Donor Notes] to add donor notes. This button gets enabled, only if donor notes are recorded for the donor.

 

Screening Tests

  1. Test - The screening test associated to the donor is displayed.

  2. Description - The description of the selected screening test is displayed.

  3. Send for Screening - The check box is selected to indicate that the test is sent for screening. The tests are highlighted in different colors to indicate the status.

  4. [Send for Screening] - Click [Send for Screening] to send the test for screening. The 'Screening Test Results' tab page is enabled on clicking [Send for Screening].

 

Screening Test Results

The values are displayed based on the result received from Laboratory.

  1. Test Code - The test code associated to the donor is displayed.

  2. Numeric Result - The Numeric Result associated to the test code is displayed.

  3. Test Result - The Test Result associated to the test code is displayed. The values displayed are either 'Accept' or 'Reject'.

  4. Comments - Click [.....] to view comments. The test results are highlighted in different colors to indicate their status.

  5. [Validate] - Click [Validate] to validate the process.

  6. [Accept/Defer Donor] - Click [Accept/Defer Donor] to accept or defer the donor based on the screening test results.

  7. [View Lab Results] - Click [View Lab Results] to view the screening test results.
