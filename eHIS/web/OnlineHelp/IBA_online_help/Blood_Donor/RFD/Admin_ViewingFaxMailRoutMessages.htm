View Fax/Mail Routing Messages function displays the following information:

 

  1. Request Date – The Date when the request was raised is displayed.

  2. Facility – The name of the facility from where the requests are raised is displayed.

  3. Module ID – The Module ID entered while placing the request is displayed.

  4. User - The User ID of the logged in user, who raised the request is displayed.

  5. Report File Name - The Report output file name is displayed and the value cannot be changed.

  6. Destination – The Routing Destination Type is displayed.

  7. Work Station – The Workstation Number, where the message gets logged is displayed.

  8. Report - The Report Type is displayed.

  9. Fax/Mail ID - The Printer ID is displayed.

  10. Error Text – The Error Text is displayed.
