Enter the following details to record donor apheresis screening details:

 

  1. Select Donor from – Select the value from the list. The valid values are ‘All Donors’ and ‘Screening in Progress’. Select ‘All Donors’ to select a donor ID from the list of donors registered in the facility. Select ‘Screening in Progress’ to invoke the donor details screen. The donor records that are in screening process are displayed. The color code as defined at the bottom of the screen indicates that the records are waiting for acceptance/rejection. Select the donor by selecting the check box against the Donor ID and the screen gets closed. Click [Close] to close the screen and the selected value is set to ‘All Donors’. Select the Donor ID using the [Donor Lookup].

    The following details are displayed.

    1. Donor ID – The Reactive Donor’s ID is displayed.

    2. Donor Name – The name of the Reactive Donor is displayed.

    3. Donation Date – The date of donation is displayed.

    4. Sample Number – The Sample number is displayed.

    5. [Close] – Click [Close] to close the window.

  2. Screening Date – The system date is displayed, by default. Select the Screening Date using the Calendar function.

  3. Screening Number - The Screening Number of the blood unit sent for Apheresis Screening is displayed. Screening Number is auto generated on sending the blood unit for screening and you can view them only on query mode.

  4. Donor ID – Enter a valid Donor ID or click [Donor Lookup] to select the donor. On selecting the Donor ID, click tab to view the additional details of the donor. You cannot proceed for the apheresis screening, if any unprocessed outstanding samples exist for the donor.

  5. Age – The Age of the donor is displayed based on the details entered during donor registration.

  6. Sex – The Gender of the donor recorded during registration is displayed.
