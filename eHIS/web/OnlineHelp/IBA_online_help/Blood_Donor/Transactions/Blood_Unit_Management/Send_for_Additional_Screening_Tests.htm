Send for Additional Screening Tests

From this function you can Send the blood units for additional screening tests.

The Send for Additional Screening Tests function is used to send the blood units for additional screening tests. Additional tests are performed on the Donor’s Blood Units for maintaining the medical details for the Donor. All the mandatory tests are to be validated before sending the unit to stock. If the results for additional tests are reported after the units are added to stock, validation does not take place. The additional tests are created through 'Screening Test Setup' function and are not mandatory.

Sending blood units for Additional Screening Tests

From the Transactions, Select Blood Unit Management double-click the Send for Additional Screening Tests function.

The Send for Additional Screening Tests screen gets displayed.

  1. Click the Calendar function to select the donation date range.

  2. Click [List of Values] to select the Batch Number. The listed values are created in Create Batch function.

  3. Enter the Unit Number in alphanumeric format of 30 characters maximum length.

  4. Click [Display] to display the Test and Unit details based on the selected search criteria.

  5. Select the ‘Select All’ check box to select the Additional tests or select the tests individually to send the blood unit for additional tests. The tests and test descriptions are displayed.

  6. Scan the Barcode to send a particular unit for additional screening.

  7. Select the ‘Select All’ check box to select the unit numbers to be sent for additional screening or select the unit numbers individually. The Unit numbers, Component, Donation Date, Blood Group and the Rhesus factor of the unit numbers are displayed.

  8. Click [P] and [N] to navigate between the Test Details and Unit Details displayed. The blood units already sent for Screening are displayed in this function.

  9. Click [Send Selected Units for Screening] to send the selected unit numbers for additional screening. A message ‘Selected units Sent to laboratory for Screening’ is displayed.
