Record Donor/Unit Antigens

From this function you can Record the Antigen details for a donor or a blood unit.

The Record Donor/Unit Antigens function enables you to record the antigens for a donor/blood unit. The Antigens are defined in Antigens function. If multiple units are associated to the donor, you can select the unit number to record the antigen details.

Recording Donor/Unit Antigens

From the Transactions > Blood Unit Management, double-click the Record Donor/Unit Antigens function.

The Record Donor/Unit Antigens screen gets displayed.

  1. Enter the Donor ID or click [Donor Lookup] to search and select the donor.

  2. On selecting the donor, click [Execute Query] to view the donor details. <Related Field Description>

    The following details are displayed on selecting the donor:

    1. Sex – The Gender of the donor recorded during registration is displayed.

    2. Age – The Age of the donor recorded during registration is displayed.

    3. Blood Group – The Blood Group of the donor is displayed.

    4. Rhesus – The Rhesus factor of the blood group is displayed.

  3. Select the Unit Number to record the Antigen details.

    The following details are displayed on selecting the Unit Number:

    1. Unit Number – The Unit Number of the blood unit is displayed.

    2. Product Code – The Product Code associated to the blood unit is displayed.

  4. Press [Tab] to associate antigens to the donor. <Related Field Description>

  5. Click [Antibodies] to view the existing Antibody details recorded for the donor. The [Antibodies] is enabled, if antibody details exist for the donor.

  6. Click [Save] or press [F10] to save the recorded Antigen details. Click [Clear Record] to delete the entered data.
