Blood Unit Screening

From this function you can Send the donated blood units to the lab for screening.

The Blood Unit Screening function enables you to send the donated blood units for screening to the lab. You can send the blood units for Level 1 and Level 2 screening. The blood units donated during Mobile Donations and the blood units in the laboratory are sent for screening using this function. You can scan the Barcode Unit Number to select a particular blood unit to be sent for screening.

Sending Blood Units for Screening

From Transactions > Blood Unit Management, double-click the Blood Unit Screening function.

The Blood Unit Screening screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [Display] to view the list of blood units that are available for screening.

    The following details are displayed:

    1. Unit Number – The Unit Numbers are displayed based on the query criteria entered to search for the records. Double-click the Unit Number to view the details of the unit and tests on the right-hand side frame.

    2. Component - The Component code selected while defining the donation details during pre-donation analysis gets displayed against each unit number.

    3. Stage – The screening stage to which the blood unit is to be sent for screening gets displayed. The listed values are Level 1 and Level 2. If the unit has already finished the Level 1 screening, Level 2 is displayed.

  3. Select the check box ‘Select All’ to select and send the listed blood units for screening. To send only a particular record for screening, select the check box against each record.

  4. Double-click the unit number to view the unit details. The unit number, component, blood group, rhesus, donation and mobile drive details are displayed along with the test code details on the right-hand side of the screen.

  5. Click [Send Selected Units for Screening] to send the selected units to the lab for screening.

  6. A confirmation message ‘Selected Units sent to Laboratory for Screening’ is displayed. Click [OK] to select next set of records for screening.
