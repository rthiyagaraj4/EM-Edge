Blood Unit Screening Result Entry

From this function you can Validate the screening results of blood units individually or in bulk.

The Blood Unit Screening Result Entry function allows you to validate the screening results of blood units. You can validate the results either individually or in bulk. Based on the validation, you can accept, reject or override the tests for the blood units. The rejected blood units are sent for second level screening, if the units are reactive. The system highlights the status of the blood units with color codes after the unit numbers are validated, accepted or rejected.

Blood Unit Screening Result Entry

From the Transactions > Blood Unit Management, double-click the Blood Unit Screening Result Entry function.

The Blood Unit Screening Result Entry screen gets displayed.

  1. Enter the query criteria to search for the units. <Related Field Description>

  2. Click [Display] to view the details of the units within the query criteria. <Related Field Description>

  3. Double-click the unit number to select the unit. Details of the selected unit are displayed in the frame on the right-hand side.

  4. Select ‘Send for Second Level’ to send the unit number to next level for review.

  5. Click [Validate] to validate individual unit. A confirmation message ‘Selected Units Validated’ gets displayed on clicking [Validate].

  6. Select the check box to select all the units to be sent for screening. Select the check boxes against the individual units to validate a set of selected units.

  7. Click [Validate All] to validate all selected units. A confirmation message ‘Selected Units Validated’ gets displayed on clicking [Validate All].

  8. Click [Override Screening] to override the screening result of a reactive unit. The [Override Screening] is enabled, if the validated unit is reactive.

  9. Click [Change Initial Blood Typing] to change the initial blood typing. This button is enabled only if the check box ‘Accept Initial Blood Type Miss match?’ is cleared in 'BD Parameters' function. This is valid only if the blood group and Rhesus factor of a blood unit entered in Pre donation is different from what is entered while resulting the blood group in the Result Entry by Specimen function in Clinical laboratory module. On clicking [Change Initial Blood Typing], the message ‘Selected Units Validated’ is displayed.

  10. Click [Accept/Reject Units] to acknowledge or reject the selected unit. A message ‘Selected Units Added/Rejected/Sent for Second Level Testing Respectively’ gets displayed. If the check box ‘Send for Second Level’ is selected, the units are sent for second level review.

  11. Click [Print] to generate the report. The Preferences dialog gets displayed with the preferences to View, Print and File options.

    Click ' File' to store the output into a file. Depending on the nature of the report, system prompts to select a file format. Choose the appropriate file format from the available list. You can enter the output file name.

    Click ‘Print’ to print the report on the designated printer. The report is routed to the default printer set to print the report output.

    Click ‘View’ to view the report on the screen. A note 'Process Successful. Click to view report' appears. Click to view report or to close the report viewer.

  12. Click to generate the report or  to cancel the transactions.

 

The colors at the bottom of the screen indicate the status of the blood unit:

- Specimen(s) Rejected from Lab

- Blood Typing Mismatch

- Rejected Units

- Donation Details Missing

- Initial Blood Typing Mismatch
