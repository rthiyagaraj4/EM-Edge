Pre Donation

From this function you can Perform the pre-donation analysis of the donor and Defer a donor if the donor is not suitable for donation. You can also Print the Questionnaire, Consent Form and Donor Card.

The Pre Donation function enables you to print donor documents such as Questionnaire, Consent Form and Donor Card. You can perform the pre-donation analysis for a registered donor. During pre-donation, record the donation details, fill the questionnaire, enter the physical checkup details, additional physical checkup details and counseling details. If the donor is not suitable for donation during the analysis, defer the donor either temporally or permanently. A donor is not allowed to donate blood more than once in a day. You can record pre-donation for temporary or permanently deferred donors' who have donated blood through the mobile drives.

Performing Pre-Donation

From Transactions > Donation Management, double-click the Pre Donation function.

The Pre Donation screen gets displayed.

  1. Click [Donor Lookup] to select the donor. Press [Tab] to view the selected donor’s details.

    The following details are displayed:

    1. Donation Date – The current date is displayed, by default. Click the Calendar function to update the donation date. This is a mandatory field.

    2. Age - The Age of the donor recorded using the ‘Donor Registration’ function is displayed.

    3. Sex - The Gender of the donor recorded using the ‘Donor Registration’ function is displayed.

  2. Click [Donor History] to view the previous transactions of the donor in the facility. On clicking [Donor History], the Donor History with Unit Details screen is displayed. This button is enabled only if the donor had previous transactions in the facility.

  3. You can record the pre-donation analysis details under the following tabs:

Donation Details

Questionnaire

Physical Checkup

Addn. Physical Checkup

Counseling

Proceed

Deferral Details

  1. Click [Antibodies] to view the details. This button is enabled, if antibody details are recorded for the donor during the screening.

  2. Click [Antigens] to view the details. This button is enabled, if antigen details are recorded for the donor in the Record Donor/Unit Antigens function.

  3. Click [Donor Notes] to view the details. This button is enabled, if donor notes details are recorded for the donor. Donor Notes are recorded in the ‘Record Donor/Unit Antigens’ function.

  4. Click [Print Questionnaire] to print the Questionnaire of the donor.

  5. Click [Print Donor Card] to print the Donor ID card.

  6. Click [Print Consent Form] to print the consent form for the selected donor.

  7. Click [Print Consent Age Exception] to print the Consent form for the donors under the Age Exception Category set in the BD Parameter function. This button is displayed only if the donor’s age falls in the Age Exception Category.

  8. Click [Save] or press [F10] to save the record or click [Clear Record] to clear the entered data.

If the age of the donor does not fall within the exception age range a message, ‘Donor age is below/above exception range’ is displayed. You cannot proceed with the registration.

Users' with the check box 'TMDF - Allow temporary deferred donor to donate' as selected in the 'User Security' function are permitted to allow temporary deferred donors to donate blood.

Users' with the check box 'Allow Mobile Deferred Donor to Donate' as selected  in the 'User Security' function are permitted to allow temporary deferred donors to donate blood.
