Work Status for Donations

From this function you can View and Print the status of the donated blood units over a specified donation period or a batch.

The Work Status for Donation function enables you to view and print the status of the blood donation batch during a selected period. You can search for the status based on either donation date or the batches. You can view the unit details of all the donated units by clicking [Unit Details].

Printing Work Status for Donations

From Transactions > Donation Management, double-click the Work Status for Donations function.

The Work Status for Donations screen gets displayed.

  1. Enter the Donation Date range or click the Calendar function to select the date. This is a mandatory field.

  2. Enter the batch number or click [List of Values] to select it for viewing the details, if the query is based on the batch.

  3. Select a value for 'Order By' from the list to query the Work Status for Donations. The listed values are Unit Number and Donation Date. Select Unit Number to search for the Work Status for Donations based on Unit Number and Donation Date to search for the Work Status for Donations based on Donation Date.

  4. Press [Tab] to view the unit details satisfying the query criteria. <Related Field Description>

  5. Click [Unit Details] to view the details of the selected blood unit.

  6. Click [Print] to print the status details of the donated units.
