Record Donor Immunization

From this function you can Record an immunization for a donor. You can View the Donor History, Immunization details and Print the immunization details of the donor.

The Record Donor Immunization function enables you to record immunization details for a donor. You can view the Donor History, Immunization details and print the Immunization details. Select the elapsed time from the last immunization date to display the donor details to record immunization.

Recording Donor Immunization

From Transactions > Blood Donor, double-click the Record Donor Immunization function.

The Record Donor Immunization screen gets displayed.

  1. Enter the elapsed time interval of the donor from the last immunization date.

  2. Select the Unit from the list. The listed values are ‘Days’, ‘Weeks’ ‘Months’ and ‘Years’. The list displays ‘Days’, by default.

  3. Click [Display] to view the donor records satisfying the specified query criteria.

    The following details are displayed on clicking [Display]:

    1. Donor ID - The Donor ID's within the time specified from last immunization date is displayed.

    2. Name – The respective donors’ Names are displayed.

    3. Immunization – The number of times Immunization is recorded for the selected donor, if any, gets displayed.

    4. Description – The Description of the immunization associated to the respective donor gets displayed.  

    5. Last Immunization Date – The date on which the previous immunization details are recorded is displayed.

  4. Select the check box ‘Select All’ to select the listed donor records for immunization. Select each donor record to select the specific donor record for immunization.

  5. Click [Record Immunization] to record Immunization for a donor.

    Enter the following details to record immunization:

    1. Immunization Code – Click [List of Values] to select the Immunization Code. The values defined using the Immunizations function get listed.

    2. Immunization Date – Click the Calendar function to select the date when the Immunization is given to the donor.

    3. Remarks – Enter the Remarks in alphanumeric format.

    4. [Save] - Click [Save] to record immunization.

    5. [Cancel] - Click [Cancel] to cancel the process.

  6. Click [Print] to print the listed donor records.

  7. Click [View Immunization] to view the existing Immunization details. This button is enabled only if immunization is recorded more than once for the donor.

  8. Click [Donor History] to view the donor history with unit details.

  9. Click [Save] or press [F10] to save the immunization details, or click [Cancel] to cancel the transaction.
