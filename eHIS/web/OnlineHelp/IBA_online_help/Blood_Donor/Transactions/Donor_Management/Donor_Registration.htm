Donor Registration

From this function you can Register a new donor in a facility and Query for a registered donor.

The Donor Registration function enables you to register a new donor in the facility. Every donor has to be registered in the facility before blood donation. You can search for the demographic details based on the patient ID and register a new donation, if the donor had any encounter with the facility formerly. Enter the demographic details to register the donor, if the donor has no previous encounter in the facility as a donor.

Registering a New Donor

From Transactions > Blood Donor, double-click the Donor Registration function.

The Donor Registration screen gets displayed.

  1. Enter the details if the patient has any previous encounters with the facility. <Related Field Description>

  2. Enter the details in the mandatory (*) fields in the following tabs.

Demographics

Enter the following demographic details:

  1. Title – Enter a valid Title or click [List of Values] to select the title. The values defined using the ‘Name Prefix’ function of the Master Patient Index module get listed.

  2. First Name – Enter the First Name of the Donor in alphanumeric format of 20 characters maximum length. This is a mandatory field.

  3. Family Name – Enter the Family Name of the donor in alphanumeric format of 20 characters maximum length.

  4. Gender – Select the Gender of the donor from the list. The valid values are ‘Male’, ‘Female’ and ‘Unknown’. This is a mandatory field.

  5. Age – Select the check box to enter donor’s age in Year/Month/Date format. If you enter age, the Birth Date gets displayed. If you enter the Birth Date, the age gets displayed automatically.

  6. Birth Date – Click the Calendar function to select the date of birth. The age of the donor is automatically displayed based on the entered date of birth. You can select the Age check box and enter years, months and days to display date of birth of the donor. This is a mandatory field.

  7. Language – Click [List of Values] to select the Language defined in the MP Parameter in the Master Patient Index module.

  8. Birth Place – Enter the Birth Place or click [List of Values] to view and select the Birth Place. The values defined using the Birth Place function of the Master Patient Index module get listed.

  9. Marital Status – Enter the Marital Status or click  [List of Values] to select the marital status. The values defined using the Marital Status function of the Master Patient Index module get listed.

  10. Residency – Select the Residency from the list. The system-defined values are Citizen and Non-citizen. This indicates if the donor is a citizen or a non-citizen.

  11. Status – Select the respective Status of the donor from the list. The valid statuses are ‘Legal’ and ‘Illegal’. The Status is displayed as ‘Legal’ by default, if the ‘Residency’ is selected as ‘Citizen’.

  12. Category – Enter the Category of the donor or click  [List of Values] to select the category and its description. The values defined using the Patient Category function of the ‘Master Patient Index’ module get listed.

  13. Nationality – Enter the Nationality of the donor or click [List of Values] to select the Nationality of the donor. The values defined using the Country function of the ‘Master Patient Index’ module get listed. This is a mandatory field.

  14. Race – Enter the Race of the donor or click [List of Values] to select the race of the donor. The values defined using the Race function of the ‘Master Patient Index’ module get listed. This is a mandatory field.

  15. Ethnicity – Enter the Ethnicity of the donor or click [List of Values] to select the ethnicity. The values defined using the Ethnicity function of the ‘Master Patient Index’ module get listed.

  16. Religion – Enter the Religion of the donor or click [List of Values] to select the religion of the donor. The values defined using the Religion function of the ‘Master Patient Index’ module get listed.

  17. Alias Name – Enter the Alias Name of the donor in alphanumeric format of 30 characters maximum length.

Other IDs

Enter the following details under the Other IDs tab:

  1. Passport No – Enter the passport Number of the donor in alphanumeric format of 20 characters maximum length.

Contacts

You can record the contact address, additional details and the employment details of the patient using the contacts tab.

Contact Addresses

  1. Address Line 1 & Text – Enter the Residence and Mailing Address of the donor.

  2. Country – Select the Country of the donor or click [List of Values] to select the country. The values defined using the Country function of the Master Patient Index module get listed.

Contact Modes

  1. Primary Resi No – Enter the Primary Residence Number of the donor. You can also select other contact modes, such as Fax, Telephone, etc.

  2. Other Resi No – Enter the Other Residence Number of the donor, if any. You can also select other contact modes, such as Fax, Telephone, etc.

  3. Email ID – Enter the Email ID of the donor. You can also select other contact modes, such as Fax, Telephone, etc.

Next–of-kin

  1. Name – Enter the Name of the next-of-kin to be contacted in emergency.

  2. Relationship – Click [List of Values] to select the Relationship the donor shares with the next-of-kin.

  3. Birth Date – Click Calendar function to select the date of birth in dd/mm/yyyy format.  

  4. Occupation – Enter the Occupation of the next-of-kin in alphanumeric format of 37 characters maximum length. The list displays the values defined in the Occupation function of the Master Patient Index module.

  5. Alt ID Type/No. – Enter the Alt ID Type/Number of the next-of-kin specified in MP Parameter of the Master Patient Index module.

  6. Alt ID – Enter the Alt ID of the next-of-kin specified in MP Parameter of the Master Patient Index module.

  7. Line 1 – Enter the address of the next-of-kin in alphanumeric format.

  8. Country – Click [List of Values] to select the Country. The values defined using the Country function of the Master Patient Index module get listed.

  9. Home Phone No. – Enter the residence Phone Number of the next-of-kin in alphanumeric format of 18 characters maximum length.

  10. Mobile Phone No. – Enter the Mobile Number of the next-of-kin in alphanumeric format of 18 characters maximum length.

  11. Office Phone No. – Enter the Office Phone Number of the next-of-kin in alphanumeric format of 18 characters maximum length.

  12. Email ID – Enter the Email ID of the next-of-kin in alphanumeric format of 45 characters maximum length.

Employment Details

  1. Organization Name – Enter the Organization Name where the donor is working.

  2. Patient Employee ID – Enter the Employee ID of the donor.

  3. Contact Person Name – Enter the Name of the Person to be contacted in case of emergency.

  4. Occupation Class – Enter Occupation Class of the donor or click [List of Values] to select the Occupation Class. The values defined using the ‘Occupation Class’ function of the Master Patient Index module get listed.

  5. Occupation – Enter the Occupation of the Donor or click [List of Values] to select the Occupation. The values defined using the Occupation function of the Master Patient Index module get listed.

  6. Status – Select the Status from the list. The valid statuses are ‘Full-Time Employed’, ‘Full-Time Student’, ‘Not Applicable’, ‘Part-Time Employed’, ‘Part-Time Student’, ‘Self-Employed’, ‘Unemployed’ and ‘Unknown’.

  7. Line 1 To 4 – Enter the address of the donor in alphanumeric format.

  8. Area – Enter the Area or click [List of Values] to select the Area of the donor. The values defined using the Area function of the Master Patient Index module get listed.

  9. Town – Enter the Town of the donor or click [List of Values] to select the Town of the donor. The values defined using the Town function of Master Patient Index module get listed.

  10. Region – Enter the Region or click [List of Values] to select the Region of the donor. The values defined using the Region function of the Master Patient Index module get listed.

  11. Postal - Enter the Postal Code or click [List of Values] to select the Postal Code. The values defined using the ‘Postal Codes’ function of the Master Patient Index module get listed.

  12. Country – Enter the Country or click [List of Values] to select the Country of the donor. The values defined using the Country function of the Master Patient Index module get listed.

  13. Home Phone No. – Enter the Home Phone Number of the donor in alphanumeric format of 18 characters maximum length.

  14. Office Phone No. – Enter the Office Phone number of the donor in alphanumeric format of 18 characters maximum length.

  1. Click [Save] or press [F10] to register the donor and save the record.

  2. The system prompts a pop-up message ‘Generated Donor ID and Capture Photo?’

  3. Click [Yes] in the message dialog to capture donor photo and the Capture Patient Photo screen appears. Click [No] to save the record without photo.

    1. Web Cam – Select the Web Cam option to capture the donor photo using web cam.

    2. Upload File – Select the upload file option to upload the existing donor photo in the system.

  4. Click [Donor Photo] to change the photo of the existing donors.

    1. Change Patient Photo – Select the Change Patient Photo check box to change the existing donor photo or to add the donor photo in the existing donor record.

    2. Web Cam – Select the Web Cam option to capture the  donor photo using web cam.

    3. Upload File – Select the upload file option to upload the existing donor photo in the system.

  5. Click [Execute Query] or press [F8] to view the existing records.
