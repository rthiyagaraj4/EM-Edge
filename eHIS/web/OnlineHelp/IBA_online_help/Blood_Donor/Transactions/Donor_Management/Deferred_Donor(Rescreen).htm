Deferred Donor (Rescreen)

From this function you can Record specimen details that are to be sent for re-screening of a deferred donor. You can Defer a donor either permanently or temporarily. You can View the unit details, Antibodies, Antigens and Donor Notes that are recorded for the respective Donor.

The Deferred Donor (Rescreen) function is used to defer a donor or record the specimen details of a deferred donor that has to be sent for screening. Authorized users’ are only permitted to defer the donor either permanently or temporarily. The screening results are highlighted in different colors to indicate the status. The donor records that are deferred through the 'Record Bleeding' function are listed.

The check box 'Allow Temporary Deferred Donor to Donate' check box has to be selected in the 'User Security' function for respective user.

Rescreening Blood Units of a Deferred Donor

From Transactions > Blood Donor, double-click the Deferred Donor (Rescreen) function.

The Deferred Donor (Rescreen) screen gets displayed.

  1. Enter the password in the Password text box.

  2. Click [Validate] to confirm the password or [Cancel & Exit] to cancel the transaction and quit.

  3. Select a Donor from the list. The system-defined value is 'Re-Screen Donors'.

  4. A pop-up screen Select the Donor Details is displayed on selecting a value from the list.

    1. Select – Select the check box to select the donor record. The selected donor details get displayed in ‘Deferred Donor (Rescreen).

    2. The pop-up screen displays the donor ID, name of the donor, donation date and the unit number. The donor records that are waiting acceptance or deferment are highlighted using   color.  

    3. [Close] – Click [Close] to exit from the pop-up screen.

  5. Press [Tab] and the donor details are populated on selecting a Donor ID. <Related Field Description>

  6. Click [Antibodies] to view antibodies of the donor. This button gets enabled, only if any antibody details are recorded for the donor.

  7. Click [Antigens] to view antigens for the donor. This button gets enabled, only if antigen details are recorded for the donor.

  8. Click [Donor Notes] to add donor notes. This button gets enabled, only if donor notes are recorded for the donor.

  9. Click [Unit Details] to view the unit details that are recorded for the donor.

  10. Click [List of Values] to select a Deferral Code. The description of the deferral code is displayed.

  11. The Deferral Type associated to the Deferral Code is displayed.

  12. Click the Calendar function to select a Deferral Date. The selected donor is deferred till the date selected.

  13. A warning message is displayed indicating that the donor is deferred either permanently or temporarily based on the value selected.
