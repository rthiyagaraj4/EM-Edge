Physical/Questionnaire Setup

From this function you can Create the list of questions that a donor should answer before blood donation. You can Update the values for the records which are already defined. You can Query for the existing records.

The Physical/Questionnaire Setup function is used to create or update the questions that a donor answers during the Pre-donation process. You can specify whether the question defined is gender specific, mandatory, the type of answer as either numeric or list of values, the valid range of values for the question and action to be taken based on the validation. These specified values appear as default values in the transaction functions.

Physical/Questionnaire Setup

From the Administration, double-click the Physical/Questionnaire Setup function.

The Physical/Questionnaire Setup screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Save] or press [F10] to save or click [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for the existing records.
