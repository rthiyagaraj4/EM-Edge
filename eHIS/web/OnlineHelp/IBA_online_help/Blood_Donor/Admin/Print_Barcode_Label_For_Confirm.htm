Print Barcode Label For Confirm

From this function you can Print a barcode that performs the function of a ‘Confirm’ button in transactions.

The Print Barcode label For Confirm function is used to print a barcode, which is used to confirm records by scanning the barcodes. You can print barcodes that suffice the use of ‘Confirm’ button and scan the barcode instead of clicking on the button in the application. The barcode created is used in Blood Unit Component Stock Entry function to scan the Barcode instead of using [Record Stock].

Printing Barcode label For Confirm

From the Administration, double-click Print Barcode label For Confirm function.

The Print Barcode label For Confirm screen gets displayed.

  1. Enter the value in alphanumeric format of seven characters maximum length.

  2. Click [Run Process] after entering the value. You can choose to view, print or save the output in a file.

    Click [File] to store the output into a file. Depending on the nature of the report, system prompts to select a file format. Choose the appropriate file format from the available list. You can also enter the output file name.

    Click [Print] to print barcode in the designated printer. The barcode value gets routed to default printer, if print routing for respective function is not defined.

    Click [View] to view the barcode on the screen. A report is generated with the barcode value.

  3. After selecting the required preference, click [Accept] to process or click [Cancel] to re-enter the value.
