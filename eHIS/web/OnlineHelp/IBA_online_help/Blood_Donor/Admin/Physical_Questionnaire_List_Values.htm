Physical/Questionnaire List Values

From this function you can Create the list of values required for Physical Checkup and Questionnaire. You can also Update the List of Values created.

The Physical/Questionnaire List Values function enables you to create the list of values valid for Physical Checkup and Questionnaire. These values get displayed as default values in the transaction functions.

Creating Physical/Questionnaire List Values

From the Administration, double-click the Physical/Questionnaire List Values function.

The Physical/Questionnaire List Values screen gets displayed.

  1. Select the desired value from the list to specify whether the defined lists of values are valid for Physical Checkup, Questionnaire or Both.

  2. Enter the list of values in alphanumeric format of 30 characters maximum length.

  3. Click [Save] or press [F10] to save or click [Clear Record] to clear the entered data.
