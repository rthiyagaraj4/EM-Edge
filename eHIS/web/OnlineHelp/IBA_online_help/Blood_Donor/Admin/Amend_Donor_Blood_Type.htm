Amend Donor Blood Type

From this function you can Modify the Blood Group and Rhesus factor details of a donor and Query for the existing records.

The Amend Donor Blood Type function allows you to modify the Blood Group and Rhesus factor details recorded during Pre-Donation. This function is used when incorrect entries are made for a Blood Group and the Rhesus factor. The modifications that are done reflect in the transaction functions.

Amending Donor Blood Type

From the Administration, double-click the Amend Donor Blood Type function.

The Amend Donor Blood Type screen gets displayed.

  1. Click [Execute Query] or press [F8] to search for the existing records. You can also search for the donor by clicking [Donor Lookup]. Click [Execute Query] or press [F8] to query for donor details in the facility. Double-click on the Donor ID to select the donor.

  2. The Blood Group and the Rhesus Code of the donor recorded in the Pre-Donation function gets displayed.

  3. Click [List of Values] to select and associate a Blood Group and the Rhesus Code.

  4. Click [Save] or press [F10] to save the record. A warning message ‘System will amend the Blood Group for the units transferred to stock’ gets displayed. Click [Yes] to amend the Blood Group for the units transferred to stock or [No] to cancel the process. Click [Clear Record] to clear the entered data.

  5. Click [Antibodies] to view the antibodies details of the donor. If Antibody details are entered, this button gets enabled.

  6. Click [Antigens] to view the antigen details of the donor. If antigens details are entered, this button gets enabled.

  7. Click [Donor Notes] to view the recorded donor notes of the donor, if any. If donor notes are entered, this button gets enabled.

 

You can amend the Blood Group only when all components are added to stock.
