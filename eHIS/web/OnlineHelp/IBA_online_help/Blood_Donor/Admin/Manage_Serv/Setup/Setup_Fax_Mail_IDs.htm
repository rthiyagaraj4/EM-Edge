Setup Fax/Mail IDs

From this function you can Create the Fax numbers and email IDs to fax or mail a document/output based on the requirements of the specific document output.

The Setup Fax/Mail IDs function is used to set up fax numbers and email IDs for each document output. Each report can be routed to multiple faxes and mails, depending on the requirements of the specific document/output.

Setting up Fax/Mail IDs

From the Administration > Manage Services > Setup, double-click the Setup Fax/Mail IDs function.

The Setup Fax/Mail IDs screen gets displayed.

  1. Enter the details in the mandatory (*) details in the following tabs:

Fax

Enter the following details:

  1. ID – Enter the Fax ID in alphanumeric format of 20 characters maximum length. This is a mandatory field.

  2. To Number – Enter the Fax Number in alphanumeric format of 20 characters maximum length. This is a mandatory field.

  3. Recipient Name – Enter the Recipient Name in alphanumeric format of 80 characters maximum length. This is a mandatory field.

  4. Company Name – Enter the Company Name in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  5. Fax Service – Enter the Fax Service in alphanumeric format of 50 characters maximum length click [List of Values] to select the Fax Service. The values defined in Setup Fax Service function get displayed.

  6. Alternate Number – Enter the Alternate Fax number in alphanumeric format.

  7. Recipient Designation – Enter the Recipient’s Designation in alphanumeric format.

  8. Recipient Phone Number – Enter the Recipient’s Phone Number in alphanumeric format.

  9. Status – Select the check box to enable the fax number entered. If the check box is cleared the Fax Number is not enabled.

Mail

Enter the following details:

  1. ID – Enter the Mail ID in alphanumeric format of 20 characters maximum length. This is a mandatory field.

  2. To – Enter the Mail ID in alphanumeric format of 20 characters maximum length. This is a mandatory field.

  3. CC – Enter the Carbon Copy ID in alphanumeric format of 20 characters maximum length.

  4. BCC – Enter the Blind Carbon Copy ID in alphanumeric format of 20 characters maximum length.

  5. Status – Select the check box to enable the Email ID entered. If the check box is cleared the Mail ID is not enabled.

  1. Click [Save] or press [F10] to save or click [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records.
