Workstation Based Print Routing/Parameters

From this function you can Define the destination printers for each specific workstation depending on the requirements of the specific report. You can also Define or Update the Workstation Parameters.

The Workstation Based Print Routing/Parameters function is used to define the destination printers for each specific workstation. It provides multiple ways of routing each online and generated report, depending on the requirements of the specific reports. You can define the routing dynamically based on the Operating Department/Ward/Clinic, which is involved in the transaction, currently being processed. For both online and generated reports, this function includes the facility to print the documents simultaneously at more than one destination and to print more than one copy at any of these destinations. On entering the function, all workstation specific routings for the logged on workstation are listed in the second block and all non-workstation specific document routings are listed in the third block.

Workstation Based Print Routing/Parameters

From the Administration > Manage Services > Setup, double-click the Workstation Based Print Routing/Parameters function.

The Workstation Based Print Routing/Parameters screen gets displayed.

  1. The details are displayed in the fields. <Related Field Descriptions>

  2. Click Workstation Parameter tab to invoke the tab frame.

  3. The details are displayed in the fields.

    Workstation Parameters

    1. Parameters Description – The Parameter Description for the workstation parameters is displayed.

    2. Parameter Value – The Parameter Value of the selected Parameter Description is displayed.

    Default Parameters

    1. Parameters Description – The Parameter Description for the workstation parameters is displayed.

    2. Parameter Value – The Parameter Value of the selected Parameter Description is displayed.
