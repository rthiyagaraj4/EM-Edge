View Fax/Mail Routing Messages

From this function you can View the Fax/Mail Routing Messages and the errors occurred while processing the printing service.

The View Fax/Mail Routing Messages function is an administrative function that maintains the Fax/Mail Routing Messages. The errors occurred while sending the Faxes/Mails are displayed in this function. You can view, report and purge the Fax/Mail routing messages.

Viewing the Fax/Mail Routing Messages

From the > Administration> Manage Services > Process Management, double-click View Fax/Mail Routing Messages function.

The View Fax/Mail Routing Messages screen gets displayed.

  1. The Fax/Mail Routing Messages of the current day are displayed.

  2. Click [All Days] to view a list of Fax/Mail Routing Messages. When you click [All Days], the button changes to [Today]. <Related Field Description>

  3. Click [Report] to report the routing messages in the defined media after the confirmation message is acknowledged.

  4. Select a Fax/Mail routing message record and click [Purge] to delete the error message. A warning message gets displayed. Click [Yes] to proceed with the deletion of the record or [No] to cancel the process.

  5. Click [Purge All] to delete all the queried routing messages. A warning message gets displayed. Click [Yes] to proceed with the deletion of the record or [No] to cancel the process.
