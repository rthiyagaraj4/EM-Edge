View Print Routing Messages

From this function you can View the Print Routing Messages and the errors occurred while processing the printing service.

The View Print Routing Messages function is an administrative function that maintains the Print Routing Messages. In Administer Print Services, if a service is on and any problems occur in the process, the error messages occurred while processing the printing service are displayed. You can view, print or purge the print routing messages.

Viewing Print Routing Messages

From the Administration > Manage Services > Process Management, double-click View Print Routing Messages function.

The View Print Routing Messages screen gets displayed.

  1. The print routing messages of the current day are displayed.

  2. Click [T_All_Days] to view all the print routing messages. When you click [T_All_Days], the button changes to [T_Today]. <Related Field Description>

  3. Click [Print] to print the routing messages in defined printer after the confirmation message is acknowledged. You can either view the report, print the output in the associated printer or save the output in a file in the system.

  4. Select a print routing message record and click [Purge] to delete the error message. A warning message gets displayed. Click [Yes] to proceed with the deletion of the record or [No] to cancel the process.

  5. Click [Purge All] to delete all the queried routing messages. A warning message gets displayed. Click [Yes] to proceed with the deletion of all the records or [No] to cancel the process.
