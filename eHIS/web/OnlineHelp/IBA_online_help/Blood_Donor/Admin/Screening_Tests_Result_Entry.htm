Screening Tests Result Entry

From this function you can manually Record the screening test results of blood units sent for lab screening. You can also Query to view the existing records.

The Screening Tests Result Entry function is used to manually record the screening test results of the blood units sent for screening to laboratory. This function is used in emergency cases, when there is a delay in getting the results from laboratory. The unit numbers rejected by lab are highlighted with indicator. The unit numbers with the tests verified are highlighted using indicator.

Recording the Screening Tests Result

From the Administration, double-click Screening Tests Result Entry function.

The following screen gets displayed.

  1. Enter the Unit Number in the Unit Number field. The Donation Date and Screening Type are displayed. The donation date is recorded in the 'Record Donation' function.

  2. The tests performed on the selected Unit number is displayed. The tests are highlighted with the following color indicators.    - Rejected by lab and Verified Results.

  3. Select the ‘Reject Y/N’ check box to select the test to be rejected.  

  4. The Result value of the test is displayed. The listed values are created in the Screening Test Setup function. If the test is based on numeric results, the results are displayed in Num Results field, if the test is based on list of values, the results are displayed in Comment Results field.

  5. Click [Update] to record the screening test results of the blood unit or [Cancel] to clear the data.

  6. Click [Execute Query] or press [F8] to search for the existing records.
