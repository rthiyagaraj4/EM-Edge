Amend Blood Bag Type/Batch Number

From this function, you can Amend the Blood Bag Type/Batch Number of a blood unit. You can Query to view the existing records.

The Amend Blood Bag Type/Batch Number function is used to change the Blood Bag Type or Batch Number of a blood unit. If the selected blood unit is already added to stock, the unit is highlighted with the indicator and if the components are already prepared for the unit, it is highlighted with the indicator.

Amending Blood Bag Type or Batch Number of a blood unit

From the Administration, double-click Amend Blood Bag Type/Batch Number function.

The Amend Blood Bag Type/Batch Number screen gets displayed.

  1. Click the Calendar function to select the donation date range.

  2. Click [List of Values] to select the Batch Number. The listed values are created in Create Batch function.

  3. Click [Display] to show the Unit Number details based on the selected search criteria.

  4. The details such as Donor ID, Donation Date, Blood Bag, Batch Number, Mobile Item, Blood Group, Rhesus Code, Donation Type and Unit Number are displayed. The displayed details are recorded in Record Donation function.

  5. Click [List of Values] to select the new Blood Bag. The Blood Bags created in the ‘Blood Bags’ function are displayed.

  6. Click [List of Values] to select the new Batch Number. The Batch Numbers created in the ‘Create Batch’ function are displayed.

  7. Click [Amend] to amend the Blood Bag and Batch Number. A message ‘Confirm Amendment of Blood Bag Type/Batch Number?’ is displayed. Click [OK] to confirm or [No] to abort the changes. On clicking [OK], a message ‘Blood Bag Type/Batch Number Amended Successfully’ is displayed.
