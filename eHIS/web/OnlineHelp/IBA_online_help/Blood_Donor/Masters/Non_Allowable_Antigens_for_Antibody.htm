Non Allowable Antigens for Antibody

From this function you can Associate the Non-Allowable Antigens to the selected Antibody and Query for the existing records.

The Non Allowable Antigens for Antibody function enables you to associate the non-allowable antigens for an antibody.

Associating Non Allowable Antigens to Antibody

From Masters, double-click the Non Allowable Antigens for Antibody function.

The Non Allowable Antigens for Antibody screen gets displayed.

  1. Click [List of Values] to select the Antibodies. The values defined using the Antibodies function get listed. Click [Execute Query] or press [F8] to view the Non Allowable Antigens, which are associated to the Antibodies.

  2. Enter a valid Antigen code or click [List of Values] to select the Antigen code. The values defined using the Antigen Codes function get listed. The respective Description of the antigen code is displayed.

  3. Click [Save] or press [F10] to save the created records or click [Clear Record] to clear the entered data.

  4. Click [Execute Query] or press [F8] to search for the existing records.
