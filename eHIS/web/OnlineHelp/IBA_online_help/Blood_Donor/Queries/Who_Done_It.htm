Who Done It

From this function you can Query to view the changes made to the donor’s Blood Group and Rhesus factor.

The Who Done It function displays the name of the user who has made changes to the donor’s Blood Group and Rhesus factor. You can also view the Antibodies, Antigens and Donor notes associated to the Unit, if any.

Query Donor Records

From Query, double-click the Who Done It function.

The Who Done It screen gets displayed.

  1. Click [Donor Lookup] to select the desired donor. The Name of the selected donor is displayed. The name, blood group and the Rhesus factor associated to the donor gets displayed.

  2. Click [Execute Query] or press [F8] to search for the updated donor details. The Blood Group, Rhesus Factor, the name of the user and date when the changes are made are displayed.

  3. Click [Antigens] to view the antigens associated to the donor.  

  4. Click [Antibodies] to view the antibodies associated to of the donor.

  5. Click [Donor Notes] to view the donor notes associated to the donor.
