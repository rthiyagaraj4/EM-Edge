Search Duplicates

From this function you can Search the Master Patient Index for possible duplicate registrations by running this function periodically.

The records in the Master Patient Index are compared based on the defined search criteria. The patient records that match the criteria are extracted and stored for you to view and confirm as duplicates. Access to this function is determined by the User Access Rights set up for the logged-in User. You cannot invoke this function if the rights for the logged in user is not defined or if the rights for the ‘Search’ DR Process is set as ‘None’.

If the privilege level is set as Logged-in facility, you can perform ‘Search Duplicate’ process only for the Logged-in facility and the patients registered in the logged in facility are only considered for the Search Process. If the privilege level is set as All Facilities, you can perform the ‘Search Duplicate’ process for all the facilities of the enterprise and registered Patients of all the facilities are considered for the Search Process.

Searching Duplicate Registrations

From the DR Processes, select Search Duplicates function.

The Search Duplicates screen gets displayed.

  1. Select the desired criteria for current search from the Patient Name, Demographics Data Scope of Search and Action Type frames. <Related Field Description>

  2. Click [Audit Data] to invoke Process Audit Data screen and view the status and other details of the last duplicate check process. Click [Close] to exit from the Process Audit Data screen.

  3. Click [Default Criteria] to update the default search criteria defined in the function ‘Default Search Criteria’.

  4. Click [Previous Criteria] to view search criteria defined for the previous search process.

  5. Click [Start Now] to start the duplicate search process or specify date and time and click [Start Later] to start the search process later. Click [Reset] to clear the selected criteria.

 

You are informed about the progress of DR Cycle, if it is in progress on invoking the ‘Search Duplicates’ function from the menu. The Search Process begins when you start the background process using 'Administer background process' function in the System Manager module.
