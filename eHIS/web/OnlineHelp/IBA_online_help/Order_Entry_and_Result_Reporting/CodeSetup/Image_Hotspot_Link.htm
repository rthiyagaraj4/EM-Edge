Image Hotspot Link

From this function you can Create an association of hotspot to an image and Query for the existing image hotspot link. You can also Delete an existing image hotspot link.

The function Image Hotspot Link helps in linking a hotspot to an image. These image hotspot links get listed in the Body Organ field while defining an Order Catalog when the selected Order Category is either Radiology or Departmental Diagnostics. This helps to identify the affected area and place an order accordingly. Hotspot links attached to order catalogs are available during ordering.

Creating an association of Hotspot to an Image

From the OR Code Setup menu, select the Image Hotspot Link function.

The Image Hotspot Link screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Image from the list. The listed values are Radiology Image Front, Radiology Image Back, Dept Diagnostics Image Front and Dept Diagnostics Image Back.. This field is mandatory.

  3. Select a Hotspot from the list. The values defined using the function Image Hotspot is listed. This field is mandatory.

  4. Enter the Image Link in alphanumeric format of 15 characters maximum length. This field is mandatory.  

  5. Click [Apply] to save the record or [Reset] to clear the entered data.

  6. Click [Query] to search for an existing Image Hotspot Link.

  7. Click [Delete] to delete a selected record.
