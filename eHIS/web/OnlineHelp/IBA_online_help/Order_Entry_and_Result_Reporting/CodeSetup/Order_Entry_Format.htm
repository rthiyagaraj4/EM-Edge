Order Entry Format

From this function you can Define a format to record the details for a order entry format item associated to an Order Category. You can Query for an existing order entry format.

Order Entry Format is a collection of ‘Order entry format Item’ that is used to capture additional details when an order is placed. During the initial definition, *ALL is selected as the Order Catalog for the Level, Order Category and the Order Type. The Definition is displayed as User Defined, by default. A Format Item has to be associated to the Order Entry Format. The field mnemonics defined and associated to an Order Catalog using the function Order Entry Format Item function gets listed. You have an option to associate multiple mnemonics and specify the sequence for each mnemonic associated to the Order Catalog. Practitioner records the details in the format defined for an item associated to a Order Category while examining the patient. The order entry format is built using field mnemonics. These formats are user defined. Format is defined either at header level or line level. The Order entry format defined at header level for Order Category and Order type combination is common to all the Order Catalogs under it. The order entry format defined at line level is specific to an Order Catalog or common for Order Catalogs. In header level, order entry format cannot be created for individual order catalog. In line level, order entry format can be created to individual or all order catalog for the selected order type. Select the Level as line and you have an option to associate multiple Order Catalogs.

Catalogs are displayed as an individual Order Catalog. In Line Level the Order ID is same for Order Catalog if the condition specified above is applicable for the Order Catalog; else Order ID is different.  

In Header Level Order Catalogs are grouped into one Orderable for viewing under existing orders tab in Orders function. The Order Catalogs are grouped based on the following conditions. The defined conditions below must be same for the Order Catalog while placing the order.

• Order Category

• Order Type

• Start date time

• Priority

• Frequency

• Duration

• Special approval

• Order authorization

The Conditions Specimen type, Nurse Collect is applicable only for Lab orders. In Header Level the Order ID is unique for the orderable.

Creating an Order Entry Format

From the OR Code Setup, select the Order Entry Format function.

The Order Entry Format screen gets displayed.

  1. Click [Create] to invoke the create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a created Order Entry Format or [Reset] to clear the entered data.

  4. Click [Query] to search for existing record.

  5. Click [Delete] to delete a queried record.
