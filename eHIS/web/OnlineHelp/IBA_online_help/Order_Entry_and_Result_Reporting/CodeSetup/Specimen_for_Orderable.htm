Specimen for Orderable

From this function you can Associate Specimen to Order Catalogs.

Specimen for Orderable function helps you to associate specimens for an order catalog. The function contains two tabs, namely, [Select] and [Associate]. The [Select] tab is selected, by default. Using [Associate] tab you can view specimens associated previously and dissociate the specimens associated to the selected Order Catalog.

Associating Specimen to Order Catalogs

From the OR Code Setup, select the Specimen for Orderable function.

The Specimen for Orderable screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Click [?] function to select an Order Catalog. This is a mandatory field.

  3. Click the required hyperlinks to display the Order Catalogs. The Order Catalog names beginning with the selected alphabet get listed.

  4. Select the check box against the Specimen to associate it to the Order Catalog. Listed Specimens are created in Order Entry Format Item function.

  5. Click [Apply] to save the records or [Reset] to clear the entered data.

  6. Click [Query] to search for existing records.
