Matrix Row

From this function you can Create Matrix Rows and Query for the existing Matrix Rows.

The function Matrix Row is used to create rows that are required to be part of a ‘Result Matrix’. You can define rows to be a part of the result matrix and assign sequence number to each record. A Matrix Row influences reporting format of the result matrix. You can also associate a discrete measure to the Result Matrix.

Creating a Matrix Row

From the OR Code Setup menu, select the Matrix Row function.

The Matrix Row screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a created Matrix Row or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Matrix Row.

 

You can associate 10 matrix row records to the Result Matrix.
