Consent for Catalog

From this function you can Create an association of consent forms to an Order Catalog which is linked to an Order Category and Order Type.

While placing an order, consent has to be taken from the patient for confirming the order. The consent from a patient can be taken in two stages: ‘Before Order Release’ or ‘Before Order Registration’. The Order Catalogs defined with the ‘Consent Required’ check box selected are listed. Click the Order Catalog name displayed as a hyperlink to associate Consent Forms to the Order Catalog.

Creating an association of Consent to Catalog

From the OR Code Setup, select the Consent for Catalog function.

The Consent for Catalog screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to display Order Catalogs. Click the required Order Catalog hyperlink to associate Consent Forms. Click [Clear] to clear the displayed details.

  3. Enter the Consent Details.

    Enter the following details:

    1. Select – Select the check box to select the order sequence and the consent form. The check box is selected, by default.

    2. Order Seq – The Order Sequence is displayed as a hyperlink.

    3. Consent – Select a Consent using [?] function. The Consent Forms defined using the function Consent/Refusal/Referral Notes get listed.

    4. Stage – Select the Stage at which the consent has to be taken from the patient. The values are system defined – ‘Before Order Release’ and ‘Before Order Registration’.

    5. Default Form – Select the check box to indicate that the selected consent form is the default form.

  4. Click [Record] to save the association of consent to catalog or [Cancel] to clear the consent details.

  5. Click [AddMoreRows] to add extra rows to define additional consent forms.
