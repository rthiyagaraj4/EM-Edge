Charge Type

From this function you can Create a Charge Type and Query for an existing Charge Type record.

The Charge Type function is used to define different Charge Types. The Charge Type helps you to get information about the different Charge Types applicable to patients (for example – Non Citizen, Citizen, and patients based on their income) while placing orders. You can create eight Charge Type records at a time using this function.

Creating a Charge Type

From the OR Code Setup menu, select the Charge Type function.

The Charge Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a Charge Type created or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing Charge Type record.

  5. Click [Delete] to delete a queried record.
