Define Pending Order Status for Discharge

From this function you can Define the setup for validating the orders that are pending before discharging a patient.

This function allows validating the pending orders for each order category while preparing discharging advice and during patient discharge. Also, allows viewing the orders that are pending. The pending order validation is done based on the configuration setup done in IP Parameter.

Defining Pending Order Status for Discharge

From the OR Code Setup, select the Define Pending Order Status for Discharge function.

The Define Pending Order Status for Discharge screen gets displayed.

  1. Select the ‘Order Category’ from the drop-down list.  The Order Category list includes the entire Order Category available in the master in the Order Entry and Result Reporting module. This field is mandatory.

  2. On selecting the Order Category, all the Order Status including user-defined status is displayed.

  3. Select the check box corresponding to the Order Status under ‘Prepare Discharge Advice’ for validating pending orders at the time of preparing discharge advice.

  4. Select the check box corresponding to the Order Status under ‘Discharge’ for validating pending orders at the time of patient discharge.

  5. Select  the check box corresponding to the Order Status under ‘AE Check Out’ for validating pending orders during AE check out.

  6. Click [Apply] to save a record or [Reset] to clear the entered data.
