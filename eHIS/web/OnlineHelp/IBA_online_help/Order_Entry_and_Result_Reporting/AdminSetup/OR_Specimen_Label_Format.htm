OR Specimen Label Format

From this function, you can Create OR Specimen Label Formats by specifying the Start and Maximum Number for the Label. You can Query for existing Specimen Label Format records.

Creating OR Specimen Label Format

From the OR Admin Setup, select OR Specimen Label Format.

The OR Specimen Label Format screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Code for the Specimen Label Format. This is a mandatory field.

  3. Enter the Description for the Specimen Label Format. This is a mandatory field.

  4. Enter the Start Number for the Specimen Label Format. This is a mandatory field.

  5. Enter the Maximum Number for the Specimen Label Format.

  6. Select the check box to enable the record. Clear the check box to disable the record.

  7. Click [Apply] to save the Specimen Label Format or [Reset] to clear the details entered.

  8. Click [Query] to query existing records.
