Restrict Placing Orders for Self/ Family members

From this function you can Restrict placing orders for self and family members of practitioners.

This function allows to restrict placing orders for self and family members. This setup allows to configure appropriate restriction (Self or Family members or both) against each order category.

Based on the configuration done, system will restrict order placing and display appropriate validation message while placing orders for self and family members of practitioner.

This setup is applicable while placing orders from New Orders, Addition Search tab, Order Set, Tick Sheet, Image Hotspot, Place External Orders, Pharmacy Orders, Multi Patient Orders, Existing Orders – Copy Previous Orders, Copy Orders.

Restricting Placing Order for Self/Family Members

From the OR Admin Setup > OR Other Admin Setup, select Restrict Placing Orders for Self/ Family Members function.

The Restrict Placing Orders for Self/ Family Members screen gets displayed.

  1. By default, all order categories defined in the application are listed. Against each order category, two check boxes ‘Restrict Placing Orders for Self’ and ‘Restrict Placing Orders for Family Members’ are displayed.

  2. Select the required check box to restrict for Self, Family Member or Both against the required order category.

  3. Use ‘Select All’ check box to select all the categories at a time.

  4. After making the required selections, click [Apply]. The message ‘Operation Completed Successfully’ gets displayed.

  5. For the selected order category and restriction check boxes selected, system will restrict placing orders for self and family members of practitioner.
