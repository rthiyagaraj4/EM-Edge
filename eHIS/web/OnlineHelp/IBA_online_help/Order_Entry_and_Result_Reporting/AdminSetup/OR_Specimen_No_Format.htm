OR Specimen Number Format

From this function, you can Generate multiple formats for Specimen Number.

System defined components are provided to arrange the Specimen Number format. This function allows editing the Specimen number format defined.

Defining Specimen Number Format

From the OR Admin Setup, select OR Specimen Number Format.

The OR Specimen Number Format screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Code for the Specimen Number Format.

  3. Enter the Description for the Specimen Number Format.

  4. Enter a Constant and click [Insert]. The constant is displayed under Specimen No Format.

  5. Select the required Component hyperlink as shown in the top part of the screen. The selected component is displayed under Specimen No Format. The components available for selection are By Year and Julian Date, By Year (YYYY), Date (YYMMDD), Date (YYYYMMDD), Order Type, Running Serial Number and Specimen Type. On clicking Running Serial Number, a list box is displayed under Specimen No Format with values from 2-Digit Seq to 15-Digit Seq. You can select only one Running Serial Number for the Specimen Number Format.

  6. Select the required Placing Option to Replace, Insert Left or Insert Right to a component.

  7. Click [Remove] to remove an added component. Click [Reset] to reset the entered components.

  8. Click [Apply] to save the Specimen Number Format. The details are displayed on the right side of the screen.

  9. Click the Sl No. hyperlink to edit the Specimen Number Format details and then click [Apply] to save the edited details.
