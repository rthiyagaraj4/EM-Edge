To Default Order in One Day and Continuous Order

From this function you can Define order category for each practitioner type.

This function allows to define order category for each practitioner type. The Admin or Hospital staff can default particular order category for respective practitioner type in One Day and Continuous Order transaction.

Defining Order Category to default in One Day and Continuous Order function

From the OR Other Admin Setup, select To Default Order in One Day and Continuous Order function.

The To Default Order in One Day and Continuous Order screen gets displayed.

  1. Select the required facility from the Facility field.

  2. Practitioner Type - The practitioner types defined in Practitioner Type setup in Application Master module gets listed.  

  3. Order Category - By default, the value 'ALL' is considered for all practitioner types. Select the required order category against the practitioner as required.

  4. Once the order category is selected for respective practitioner type, click [Apply].

  5. The selected order category gets defaulted in One Day and Continue Order transaction page while loading for respective practitioner type.
