Allergy List screen

Enter the details:

  1. Active Only – Select this if you want to view only the active allergies of the patient.

  2. All – Select this if you want to view all the allergies of the patient.

 

Adverse Events(Allergies/Sensitivities)

  1. Event Type – Displays the Reaction Date for the drug if the Drug option is selected in Allergy Alert By field in Adverse Events screen.

  2. Allergen – Displays the drug name and description of the drug reaction.

  3. Onset Date – Displays the onset date of the allergy and its severity.

  4. Status – Displays the status of the allergy and body site of reaction.

  5. High Risk Conditions – Displays the high risk conditions of the patient to the drug.

 

Onset Details:

  1. Onset Date – Displays the onset date of the drug allergy.

  2. Practitioner – Displays the name of the practitioner attending the patient.

  3. Facility - Displays the name of the facility to which the practitioner belongs.

  4. Location -  Displays the location of the facility.

  5. Service – Displays the service provided at the facility.

  6. Recorded Date -  Displays the date when the onset details were recorded.

  7. Remarks – Displays remarks of the allergy onset details.

 

Close Details

  1. Practitioner - Displays the name of the practitioner who resolved the allergy of the patient.

  2. Facility - Displays the name of the facility at which the patient’s allergy was resolved.

  3. Allergy Resolve Recorded Date - Displays the date when the allergy was resolved.
