Enter the following details:

 

  1. Order Set - Select the Order Set from the list. The values defined using the Order Set function with the index check box selected get listed. This field is mandatory.

  2. Index Type - Select the Index Type from the list. The values listed are 'By Source', 'Service', 'Practitioner' and 'Diagnosis'. This field is mandatory.

    1. On selecting the Index Type as 'Diagnosis', select the Diagnosis Code from the list. Click [?] function to search and select the required Diagnosis Codes. The Diagnosis Codes defined in the Medical Records module is listed. The Search screen gets displayed. Search for the required Diagnosis Code. This field is mandatory.

    1. On selecting the Index Type as 'Practitioner', click the required alphabet displayed as a hyperlink to list and select the required Practitioners beginning with the selected alphabet.

    2. Select the check box against the Practitioners to associate it to the Order Set.

    3. Click Associate Tab to view the Practitioners associated to the Order Set. You can dissociate the Practitioners by clearing the check box.

    1. On selecting the Index Type as 'Service', click the required alphabet displayed as a hyperlink to list and select the required Services beginning with the selected alphabet.

    2. Select the check box against the Services to associate it to the Order Set.

    3. Click Associate Tab to view the Services associated to the Order Set. You can dissociate the Services by clearing the check box.

    1. Facility - Select the Facility from the list. The facilities of the enterprise is listed. This field is mandatory.

    2. Source Type – Select a Source Type from the list. The values are system defined. The Source Types displayed are All, Clinic and Nursing Unit. This field is mandatory.

    3. Click required alphabet displayed as a hyperlink to list and select the required Locations beginning with the selected alphabet.

    4. Select the check box against the Locations to associate it to the Order Set.

    5. Click Associate Tab to view the Locations associated to the Order Set. You can dissociate the Locations by clearing the check box.
