Click the Order Catalog hyperlink to perform the following functions.

  • View

    On clicking the View option, the ‘View Order’ screen is displayed.

    1. The screen displays Order ID, Order Category, Order Type, Priority, and Patient details, Encounter ID, Order Date Time, Order Status, Ordering Location, Ordering Practitioner, Ordering Facility, Performing Facility, Order Format, Clinician Comments and Order Placed By. The screen also displays the Order Catalog, Priority, Frequency, Start Date/Time, End Date/Time, Quantity, Status, Formats and Comments.

    2. Click Format hyperlink to view details in the Order Line Format screen. Click [Close] to close the Order Line Format screen.

    3. Click [Schd] hyperlink to view the Schedule Frequency of the order and [Close] to close the Schedule Frequency screen.

    4. Click Preps, Dept Instrn, Pat Instrn hyperlink to view details if recorded while creating the Order Catalog.

    5. Click [View Audit Trail] to display View Audit Trail screen and view audit trail details. Click the Remark (R) hyperlink to display remarks for the order. Click [Close] to close the remark screen. Click [x] to exit from the View Audit Trail screen.

    6. Click [Close] to close View Order screen.

  • Redirect Order

    This function is used to redirect an order to a different facility or performing location. This is used if the order cannot be processed in the current facility or performing location where registration is being performed. This function displays based on access rights assigned to the logged in practitioner and is invoked by using the hyperlink on the Order Catalog.

    On clicking the Redirect option, the ‘Change Performing Location’ screen is displayed.

    1. Performing Facility – Select the Performing Facility to which the order has to be redirected. The system defaults the logged in facility. This field is mandatory. The listing of facilities is validated based on facilities assigned to the logged in practitioner and operational status of the Order Entry and Results Reporting Module in the facility. This field is mandatory

    2. Performing Location – Select the Performing Location. The listing is validated based on the performing facility selected.

    3. Click [OK] to confirm redirection of order or [Close] to abort redirecting the order and the control shifts back to the main screen. If the performing location selected is an external location, on completion of redirecting, you are prompted for sending out the order to the external location. Click [OK] and the Send out screen is displayed. You can send the order to another location. The Send Out option displays while clicking the Order Catalog hyperlink in the main screen only if you click [Cancel].

  • Send Out

    This function is used to confirm sending out an order to an external location for processing. This function is applicable only when an order is placed against an external performing location or order is redirected to an external performing location. The function is invoked while clicking Send Out option from the Order Catalog hyperlink. The performing location for the order catalog must be marked as ‘External’ in the master function. The hyperlink gets displayed if an order is associated to External Performing Location.

    On clicking the Send Out option, the ‘Send Out’ screen is displayed.

    1. Referral Facility – Select the facility to where the order is being sent out. The values listed are based on the external referral sources defined in the Application Master Module. This field is mandatory.

    2. Referred Date – The current date and time is displayed as the date and time for referral of order.

    3. Practitioner – The practitioner performing the referral is displayed.

    4. Remarks – Enter remarks on referral if any. This field is optional

    5. Referral Form – Select a Referral Form from the list. The values listed are based on Referral Forms defined in the Master setup. If a Referral Form is selected, a Referral Text hyperlink is displayed. Click on the Referral Text Hyperlink to view content of the referral form.

    6. Click [OK] to close the Referral Form view and the control shifts back to the Send Out screen. Click [OK] to confirm sending out of the order to the facility or [Close] to cancel sending the order.

  • Receive Results

    This function is used to record acknowledgement of results from an external source for orders that were sent out to the external location. The function can be invoked only using the Task Status – Acknowledge the Results.

    On clicking the Receive Result option, the ‘Receive Result’ screen is displayed.

    1. Reporting Practitioner - Enter the name of the Reporting Practitioner. This field is mandatory.

    2. Reporting Date – Enter the date of reporting for the received report. System defaults current date and time and can be modified using the calendar function. This field is mandatory.

    3. Referral Facility – The external source from where results are received is displayed.

    4. Date of Referral - The date when order was send out to the external source is displayed.

    5. Referred Practitioner - The name of practitioner who send out the order to the external source is displayed.

    6. Receiving Practitioner – Enter the name of the practitioner receiving results. System defaults logged in practitioner and can be changed using [?] function. This field is mandatory.

    7. Date of Receipt - Enter the date of receipt. System defaults current system date and time and can be changed using the calendar function. This field is mandatory.

    8. Remarks – Enter remarks on receipt of results.

    9. Click [OK] to finalize receiving results or click [Cancel] to cancel confirmation of receipt. If logged in practitioner has been assigned rights to perform resulting of order selected the system prompts if you want to proceed with result reporting.

    10. Click [OK] to proceed with result reporting, the result reporting screen gets displayed and you can enter the result. Click [Cancel] to complete acknowledgement of results.

    If result is not entered using this function, then the Status of the ‘Send Out’ order will be ‘Send out for external Source’ in result reporting function. On clicking   ‘Receive Results’ hyperlink, you can receive the result for the external order.
