Enter the following details:

 

  1. Order Category - Select the required Order Category from the list. The Order Categories defined in the Order Category function are displayed. This field is mandatory.

  2. Restrict Performing Location - Select the check box to restrict the user from reporting results from the selected Performing Location. On selecting this check box, the [Performing Locations] is displayed.

    1. Select Performing Facility from the list. The facilities defined in System Manager module are listed.

    2. Select the Performing Location from the list. The Performing Location is listed based on the Performing Location selected.

    3. Click Performing Facility hyperlink to select change the associated Performing Location.

    4. Click [Select] to select the Performing Location, [OK] to exit from the screen, [Clear] to clear the entered data or delete the added record, [Close] to exit from the screen.

  3. Order Type - Select the required Order Type from the list. The listed values are ALL and Order Type defined for the selected Order Category. The Order Types associated to the selected Order Category are displayed on selecting 'ALL'. This field is mandatory.

  4. Orderable - Click [Orderable] to select the Orderable. The Orderable’s belonging to the selected 'Order Type' are listed.

    1. You can view Orderable for the selected Order Category. If Order Type is selected individually, all orderable for the selected Order Type gets displayed. On selecting the

    2. Select the Order Type from the List.

    3. Click Select tab to display the Order Catalogs. The English Alphabets from ‘A to Z’, ‘Others’  are displayed and hyperlinks. Order Catalogs beginning with those alphabets are displayed. Click ‘Others’ hyperlink to select Order Catalogs starting with numeric or special characters.

    4. Select the ‘Select’ check box to associate the individual Orderable to the User for Result Reporting.

    5. Click [OK] to save the association and the control shifts back to the User for Result Reporting screen or [Close] to exit the screen without saving the association and the control shifts back to the User for Result Reporting screen.

  5. Privilege - Select the required privilege to be assigned to the selected Practitioner from the list. The listed values are 'Reporting' and 'Reporting and authorizing'. This field is mandatory.
