Order Reviews

You can specify whether Authorization, Cosign and/or Special Approval is/are required while placing orders for the displayed patient class. On selecting the respective check box, system posts the order for review. Practitioner can review the order, only if he has the rights to review.

Enter the following details:

  1. Patient Class - The Patient Classes in the facility is displayed. These classes are system defined. The characteristics are defined for each class, which overrides the values set for 'All'.

  2. Applicability - Select the value as 'Applicable' or 'Not Applicable' from the list to specify the applicability of an Order Catalog to the patient class. On selecting Applicable, the check boxes for authorize, cosign and special approval are enabled. On selecting Not Applicable, the check boxes for authorize, cosign and special approval are disabled.

  3. Authorization Required – Select the check box, if the order requires an authorization for a selected patient class. The check box is enabled depending on the value selected for applicability. Ensure that you select this check box to overrule the authorization only for those Order Catalogs previously authorized by Senior Doctor.

  4. Skip Authorization based on priority – This drop-down list is enabled only if the ‘Authorization Required’ check box is selected for the selected patient class. By default, the value ‘Not Applicable’ is displayed. Select the required priority from the drop-down list. Based on the selected priority, the system will skip the order authorization for specific patient class. This functionality is applicable for both general orders and Pharmacy Orders. If the ‘Skip Authorization based on priority’ drop-down list is not selected, the system will continue to prompt for authorization while placing order.

  5. Cosign Required - Select the check box, if the order requires cosigning for a selected patient class. The check box is enabled depending on the value selected for applicability.

  6. Special Approval Required - Select the check box, if Special Approval is required while placing order for the patient class.
