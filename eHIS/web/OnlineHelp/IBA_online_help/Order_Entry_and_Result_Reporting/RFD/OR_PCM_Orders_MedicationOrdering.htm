Medication Ordering

From this function you can Place Pharmacy Orders and view the existing orders.

Practitioners use the function ‘Place Order’ to prescribe medications for patients. As per the standard medical practice, it is mandatory to display the Practitioner responsible for the prescription; the default attending Practitioner name is displayed. The practitioner can view the patient’s profile such as Height and Weight, if recorded earlier through vital sign function. Options to view Current medication, Drug allergies, and reference to Active problems (if any) of the patients are provided. Medication for a patient can be selected using either the additional search options or using the drug search function. Information on drugs allergic to the patients, drug dosage limits and drug duplication are provided. Unique color-codes are adopted to indicate them. The practitioner has options to communicate/educate/caution the patients, while prescribing medications by entering appropriate remarks at the Prescription level as well as Medication level (Drug level). Dosage for a prescribed drug can be split as per requirement based on the set up. Medication order provides for medication refill, wherein a practitioner can prescribe a drug for longer duration. Prescribed drug can also be tapered up or down as per requirement. Additionally, option to view previously placed orders and copy required drugs from them to the current prescription is provided. The prescribed drug is routed to a dispensing location based on the order routing specified. While prescribing, the practitioner can also view the drug profile of the patient.
