Enter the following details:

 

  1. Order Category - Select the Order Category from the list. The values listed are Laboratory, Monitoring, Oral Health, Operation Theatre, Radiology, Department Diagnosis and Trmt/Procedures. This is based on access rights given to the logged in practitioner.

  2. Order Type - Select the Order Type from the list. The value 'All' is selected, by default. The values for selection are listed based on the selected Order Category.

  3. Activity Type - Select the Activity Type using [?] function. The listed values are based on the Order Type selected.

  4. Priority - Select the priority from the list. The values displayed are Routine, Stat and Urgent. This is used to filter Orders based on Priority.

  5. Location Type - Select the Location Type from the list. The values displayed are All, Clinic, Nursing Unit and Referral.

  6. Location - The values displayed in the list depends on the Location Type selected. If "All" is selected, all locations of the logged in facility are displayed. If Clinic is selected, all the Clinics of the logged in facility are displayed and if Nursing Unit is the selected, all the Nursing Units of the facility are displayed.

  7. Period - The date range is displayed, by default. Enter a period in date format of dd/mm/yyyy.

  8. Patient ID - Enter the Patient ID in alphanumeric format of length as specified in Master Patient Index module or select the Patient ID using [?] function.

  9. Encounter ID - Enter the Encounter ID of the patient in alphanumeric format of 12 characters maximum length.

  10. Order ID - Enter the Order ID in alphanumeric format of 15 characters maximum length.

  11. Task Status - Select the Task Status from the list. The list values are All, External Orders and Acknowledge the Results. 'All' is selected, by default.

  12. All – Allows listing of all orders that are to be registered.

  13. External Orders – Filters listing of orders where the Performing Facility has been identified to be an External Facility. Orders can be either redirected or send out to the external facility identified.  

  14. Acknowledge the Results – Filters listing of orders, which were sent out to an external facility and allows acknowledging the receipt of the result from external source.

  1. Performing Location – Select the Performing Location using [?] function.

  2. Specimen No - Enter the Specimen Number to filter a laboratory order for registration. This field is displayed only if the selected Order Category is Laboratory. This is used to filter order listing based on the Specimen Number generated.
