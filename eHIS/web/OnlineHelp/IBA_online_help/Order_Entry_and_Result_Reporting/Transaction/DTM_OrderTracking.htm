Order Tracking

From this function you can Track various orders placed for patients within a care location and view their status.

The function is also used to reprint order forms for various orders placed for patients.  

Tracking Orders

From the CA Transactions > Clinician Access > Desktop Menu > Order Management, select Order Tracking function.

The Order Tracking screen gets displayed.

  1. By default, the system displays Orders for the default query criteria.

  2. Enter the query criteria and click [Search] to display specific Orders. Click [Clear] to clear the displayed data.

  3. Click Order Catalog hyperlink to view Order details.

    1. The screen displays Order ID, Order Category, Order Type, Priority, and Patient details, Encounter ID, Order Date Time, Order Status, Medication Administration Details, Ordering Location, Ordering Practitioner, Ordering Facility, Performing Facility, Order Format, Clinician Comments, Performing Location and Order Placed By. The screen also displays the Order Catalog, Priority, Frequency, Start Date/Time, End Date/Time, Quantity, Status, Formats and Comments. The accession number of the Order Catalog is displayed on mouse over.

    2. Click Format hyperlink to view details in the Order Line Format screen. Click [Close] to close the Order Line Format screen.

    3. Click [Schd] hyperlink to view the Schedule Frequency of the order and [Close] to close the Schedule Frequency screen.

    4. Click Preps, Dept Instrn, Pat Instrn hyperlink to view details if recorded while creating the Order Catalog.

    5. Click [View Audit Trail] to display View Audit Trail screen and view audit trail details. Click the Remark (R) hyperlink to display remarks for the order. Click [Close] to close the remark screen. Click [x] to exit from the View Audit Trail screen.

    6. Click [Close] to close View Order screen.

  4. Select the check box against the required Orders and click [Print] to print Orders.
