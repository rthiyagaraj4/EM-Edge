Record Consent/Pending Consent

From this function you can Record Consent for patients.

Orders requiring consent from patients in this function. While creating an Order Catalog, patient consent applicability is set and consent form is attached. On ordering these order catalogs, consent forms are printed. The status of the order requiring consent displays as ‘Pend consent’ in the ‘Existing Order’ tab. The practitioner can view orders requiring consent based on the access to the location(s). The function helps in viewing orders requiring consent and updating the status. Provision to reprint consent form and record refusal details are also available in this function.

Recording Consent

From the CA Transactions > Clinician Access > Desktop Menu > Order Management, select Record Consent function.

The Record Consent screen gets displayed.

  1. Orders requiring Consent are displayed for the default date range. Enter the query criteria and click [Search] to display specific orders requiring consent. Click [Clear] to clear the displayed data.

    1. Location - Select the Location from the list. The listed values are All, Clinic, Day care, Emergency, Nursing Unit and Referral. Select the Location based on the selected Location.

    2. Specialty – Select the required specialty from the list and the consents related to that specialty are listed. All the specialties defined in the Application Masters module are listed. The value ‘Select’ is displayed by default.

    It is applicable for Pending Consent in Summary and Record Consent in Practitioner Desktop.

    1. Period [From-To] – The default date range is displayed. Select the date range using the calendar function.

    2. Patient ID - Enter the Patient ID or use [?] function to select for the required ID.

    1. Status by Priority - The number of orders requiring consent for each priority are displayed. The displayed priorities are hyperlinked. The total number of orders requiring consent of all the displayed priorities is shown against 'All' hyperlink. You can click on a priority to view orders requiring consent.

    2. Status by Category - The number of orders requiring consent for each category are displayed. The displayed categories are hyperlinked. The total number of orders requiring consent of all the displayed categories is shown against 'All' hyperlink. You can click on a category to view orders requiring consent.

    3. Consent - Select the check box to select the orderable to record consent details. The 'Consent Details' screen gets displayed. If the Multiple Consent Required check box is selected in the order catalog, a list of consent forms is displayed. Select the required Consent Form from the list.

    4. Required - This field displays when the consent is required. Click View Consents Hyperlink to view the consent details.

    5. Priority - The Order Priority is displayed.

    6. Ordered At - The Ordering Date and Time is displayed.

    7. Order Catalogs - Click the Order Catalogs hyperlink to view the order details or record refusal details.

    8. Order Details - The Order details are displayed for the patient.

  2. Click Required hyperlink to view.

  3. Click Order Catalog hyperlink to View Order details or Record Refusal of consent. All pharmacy orders are displayed with dosage and frequency details.

  4. Click Consent hyperlink or select the check box against the consent to view and modify consent details.

    On Clicking, consent hyperlink, an alert message is displayed ‘Would you like to view the editor in Zoom mode’. Click [OK] to open editor in Zoom mode or [Cancel] to cancel the Zoom mode.

    Select either ‘Self’ or ‘Others’. If the patient is consenting, then patient name is displayed as read only text. If ‘Self’ is selected then patient is giving the consent or patient relative/guardian is giving the consent.

    1. Name – If the patient is consenting, then patient name is displayed as read only text. If relative of the patient is consenting on behalf of the patient, then enter the name and select the relationship shared with the patient.  

    On clicking Copy the Respective fields copied to the Editor Window.

    1. Obtained by Practitioner - The logged in practitioner name is displayed, by default. Select the require name using [?] function. This field is mandatory. The practitioner recording the consent details is specified.

    2. Ordering Practitioner – The ordering practitioner name is displayed and you cannot change the name.

    3. Consented Date and Time - The current system date and time is displayed, by default. Enter the recording date and time.  

    The consent notes attached to the order catalog is displayed. The practitioner can enter additional details. Click [OK] to record the consent.  

    1. Remarks – Enter the remarks if any in alphanumeric format in the text area provided.  

    2. Orderable – The orderable Name is displayed.

    3. Select – Select the hyperlink to display the corresponding consent form.

    Common Lookups

    Common lookups helps you to add various components to the editor window like date, referral facility, Medical Team, Service and Specialty.

    1. Date – On clicking Date, the date function displays and you can select a date.

    2. Referral Facility/Medical Team/Service/Specialty/Practitioner – On Clicking Referral Facility/Medical Team/Service/Specialty/Practitioner Common lookup screen gets displayed to search and select the required data.

    Click [Close], to close the screen.

    1. Paragraph/Font/Size – You can select Paragraph, Font and size for the Details entered in the editor window.

    Click Insert Table to insert tables in the consent form. Click Preview button to preview the consent form. Click [Close] to exit from the Editor Preview screen or [Print] to print the consent form, Click OK to save the consent form and click Zoom In to zoom the screen. Click [x] to exist from the Consent Details screen.

  5. Click [Record Consent] to record the consent for the order.
