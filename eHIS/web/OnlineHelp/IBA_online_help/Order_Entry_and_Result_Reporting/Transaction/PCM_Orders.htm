Place Order

From this function you can Place new Orders for patients and View existing orders.

The order catalog(s) selected in 'Addl Search', 'Tick Sheet', 'Image' and 'Order Set' appear under their respective order categories. Characteristics of the selected order catalog and OR parameter rules influence ordering. Information is collected at header and line level. For each order category, header level details are gathered and order catalog information at line level. On placing an order, order number is generated by the system automatically, which can be viewed under 'Existing Order' tab.

Placing Orders

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Order Management > select Orders function.

The Orders screen gets displayed.

The screen displays two tabs:

Existing Orders

New Orders

 

The patient’s pregnancy details captured in the Capturing Pregnancy Details screen are displayed in red in the Pharmacy Orders screen if the selected patient is pregnant. If the patient is not pregnant, the ‘Pregnant = NO’ is displayed in red.

 

When the ‘Skip Authorization based on priority’ drop-down list is selected in the Order Catalog screen, the system will skip the order authorization for specific patient class based on selected priority while placing order. This is applicable for both general orders and Pharmacy Orders.
