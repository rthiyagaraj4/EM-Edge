Functions by Responsibility Group

From this function you can Associate a set of functions of Enterprise Management modules to a specific Responsibility Group.

Functions by Responsibility Group function is used to associate the functions of various Enterprise Management modules to a particular Responsibility Group.

Associating Functions by Responsibility Group

From the SM Admin Setup > Application Security, select the Functions by Responsibility Group function.

The Functions by Responsibility Group screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select a Responsibility Group from the list. This is a mandatory field. <Related Field Description>

  3. Select a module name and click [Search] to display the functions. To search for a relevant function, enter the function name in the text box. Click [Clear] to clear the entered data. <Related Field Description>

  4. Select the check box under ‘Add’ against the required functions to associate the functions to the Responsibility Group.

  5. Click [Apply] to associate the Functions to the Responsibility Group or [Reset] to clear the entered data.

  6. To disassociate the functions from the Responsibility Group, click [Create] and select the Responsibility Group. <Related Field Description>

  7. Select the check box under ‘Remove’ against the required functions and click [Apply] to disassociate the functions from the Responsibility Group.
