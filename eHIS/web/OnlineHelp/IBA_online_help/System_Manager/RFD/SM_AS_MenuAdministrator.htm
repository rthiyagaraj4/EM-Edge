Enter the following details to create a Menu Administrator:

 

  1. Name – Select the user by clicking [?] function. A lookup screen gets displayed. Enter the required search criteria and click [Search]. All the records matching the entered search criteria are displayed. You can select the ‘Menu Administrator’ by clicking the name which is hyperlinked. This is a mandatory field.

  2. Responsibility Group – Select the Responsibility Group to be associated from the valid list of responsibilities displayed. The entire responsibility group with enabled check box selected is displayed in the list. This is a mandatory field.

  3. Effective (From/To) - Enter the Effective date range of the assignment of responsibility group to a user in dd/mm/yyyy format, or select the date from the Calendar function. The 'From' date corresponds to the date from which the responsibility group assigned to the user comes into effect. The 'To' date corresponds to the date on which the responsibility group assigned to the user comes to an end.

  4. Enabled - Select the check box to enable the record in the transaction functions. The check box is selected, by default.
