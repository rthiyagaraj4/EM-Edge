The following details are displayed on querying:

 

  1. Module -  The name of the selected module is displayed.

  2. Report ID - The system defined IDs for the reports in a relevant module are displayed.

  3. Report Description – The description of each report in the selected module is displayed.

  4. Trans Based – This field indicates whether the report is transaction based by displaying a tick mark. The ‘X’ symbol indicates that the report is not transaction based.

  5. Internal Request – This field indicates whether the report is based on internal request by displaying a tick mark. The ‘X’ symbol indicates that the report is not based on Internal Request.

  6. Facility Based – Select the check box, if the report is facility based. On selecting the check box, the Report Group field is disabled. Clear the check box to enable the Report Group.

  7. Report Group – Select the appropriate Report Group from the list. The list of values created using the Report Group gets displayed for selection. This field is enabled, if the Facility based check box is cleared.

  8. Select – Select the check box to update and save the modifications.
