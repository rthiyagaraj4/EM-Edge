Enter the following details to construct a menu:

 

  1. Menu - Select the required root menu/submenu to be constructed/amended from the list of valid root menu/submenu entries (created through Create Menu function) displayed. On selection of the menu, the current hierarchical form of the menu in terms of the submenus and functions are displayed in the Menu Construction space provided in the right part of the screen. Under ‘List of Submenus’, the list of valid submenus of the system and under ‘Module’, the list of installed modules are also displayed on selection of the menu. This is a mandatory field.

  2. List of Submenus - During menu construction process, any of the displayed submenus can be selected by clicking the same and brought into the menu construction space by a simple mouse click anywhere in that space. Always, the newly included submenu is positioned as the top most entry of the menu under construction. Any number of submenus can be included under a menu. When an already existing submenu is attempted to be included in the menu construction space, this process is not performed and the system does not display any error message.

  3. Module - During menu construction process, when a function is to be included in the menu, the corresponding module is selected from valid list of modules displayed. On selection of a module, all supplied functions of that module are displayed under ‘List of Functions’.

  4. List of functions - During menu construction process, any of the displayed functions can be selected by clicking the same and brought into the menu construction space by a simple mouse click anywhere in that space. Always, the newly included function is positioned as the top most entry of the menu under construction. Any number of functions can be included under a menu. When an existing function at the top level is attempted to be included, this process is not performed and no error message is displayed.

  5. [Up] - During the menu construction process, if a submenu/function is required to be moved up by one position in the tree, the required submenu/function can be selected by clicking on the same and click [Up] to achieve the upward movement. The upward movement of the top most entry of the menu does not perform any action.

  6. [Down] - During menu construction process, if a submenu/function is required to be moved down by one position in the tree, the required submenu/function can be selected by clicking on the same and click [Down] to achieve the downward movement. The downward movement of the last entry of the menu does not perform any action.

  7. [Remove] - During the menu construction process, if an existing submenu/function is to be removed from the menu, the required submenu/function can be selected by clicking on the same and click [Remove] to remove the selected entry.
