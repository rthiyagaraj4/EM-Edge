Enter the following details to create a Facility based Report Parameter record:

 

  1. Module - Select the Module from the list of installed modules. When the Module is selected, all the facility based reports of the selected module are listed in ‘Reports’. This field is mandatory.

  2. Report - Select the report for which the parameter has to be set up at the facility level from the list of reports. The reports with facility based check box selected in report function are displayed. This field is mandatory.

  3. Facility - Select the facility from the list. The report parameter record is set up for the selected report. This field is mandatory.

  4. Report Group - Select a report group from the list. The selected Report is classified under the Report Group selected.  This field is mandatory.
