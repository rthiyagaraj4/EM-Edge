Enter the following fields to search for an online print configuration:

 

1. Module – Select the Module from the list of installed modules.

2. Function Name – You can enter a function name within the module, if you want.

3. Report Description – You can enter the report description, to further narrow down your search.

 

The following fields are displayed in the Query results page of Online Print Configuration:

 

1. Module – This column displays the module of the reports displayed

2. Function Name – This column displays the function name of the reports displayed.

3. Report Description – This column displays the description of the reports displayed. Click the report description hyperlink. The Online Print Configuration  page is displayed

4. Default – This column indicates whether the particular report is printable by default in the internal reports webpage.

 

Enter the following fields to search for an online print configuration:

 

1. Module – The module name of the selected report.

2. Function Name – The function name of the selected report.

3. Report Description – The description of the selected report.

4. Default – Select this check box, if you want to get the report printed by default during patient registration. By default, this check box remains checked.
