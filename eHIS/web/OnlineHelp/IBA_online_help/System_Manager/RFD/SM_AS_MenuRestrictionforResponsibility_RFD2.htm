Enter the following to exclude/remove submenu/function:

 

  1. Exclude – Select the check box to exclude the function from the responsibility.

  2. Access Rights – Select the check box to modify the access rights. The check box is enabled only on selecting the 'Type' as 'Function'. The ‘Access Level’ check box is enabled, if ‘Access Rights’ check box is selected.

  3. Type –  The Type is displayed either as a ‘Function’, ‘Submenu’ or as a ‘Report’.

  4. Name – The Name of the function is displayed.

  5. Access Level –  The default Access Levels - ‘Insert’, ‘Update’, ‘Delete’, ‘Print’ and ‘Query’ are displayed. It displays the default access right applicable to a menu. This field is cleared, masked, disabled and cannot be modified if the 'Access Rights' check box is cleared.

  6. I – Select the check box to grant access rights to insert record using a specific function for the responsibility. Clear the check box to remove the access rights to insert record.

  7. U - Select the check box to grant access rights to update the record using the specific function for the responsibility. Clear the check box to remove the access rights to update record.

  8. D - Select the check box to grant access rights to delete the record using the specific function for the responsibility. Clear the check box to remove the access rights to delete record.

  9. P - Select the check box to grant access rights to print the record using the specific function for the responsibility. Clear the check box to remove the access rights to print record.

  10. Q - Select the check box to grant access rights to query the record using the specific function for the responsibility. Clear the check box to remove the access rights to query record.

  11. Add – This field gets selected or cleared based on the selection of 'Exclude' or 'Add'. You cannot select or clear the check box as it is masked from selection.
