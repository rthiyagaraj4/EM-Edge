Enter the following details to create a Maintain Accounting Period record:

 

  1. Accounting Period Year - Enter a year for the accounting period in numeric format of four digits maximum length. This field is mandatory.

  2. Accounting Period Month – Enter the number of months for the accounting period in numeric format of two digits maximum length. This field is mandatory.

  3. Accounting Period Start Date – Click Calendar function or enter a date in dd/mm/yyyy format to define the start date for the accounting period. This field is mandatory.

  4. Accounting Period End Date – The Accounting Period end date gets displayed automatically based on the year and the accounting period start date selected.

  5. Accounting Period Description – The description of the accounting month is displayed in words followed by the year in century format. An example is JUNE 2008. The month and the year is displayed based on the date selected for accounting period start date.

  6. Calendar Year – The Calendar Year is displayed based on the selected Accounting Period.

  7. Calendar Month – The Calendar Month is displayed based on the selected Accounting Period.
