SS Document Type

From this function you can Associate applicable Document Types to Sterile Stock Transactions and Query for the existing records.

The document types are defined in the Financial Base module. These document types are used in the respective transactions and relevant document numbers are generated on performing the transactions. The document numbers are generated by the system or entered manually based on the setup in Financial Base module.

Associating applicable Document Types

From SS Administration, select the SS Document Type function.

The SS Document Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select a Document Type from the list. All the Document Types defined for Sterile Stock module in Financial Base module are displayed in the list. The Document Types defined at the Entity Level and Facility Level are displayed in the list. This is a mandatory field.

  3. Select a Transaction Type value from the list. All the transactions of Sterile Stock module are displayed in the list. This is a mandatory field.

  4. Click [Apply] to save the associated Document Types or [Reset] to clear the entered data.

  5. Click [Query] to search for existing Document Type records.

Once a Document Type is associated to a Transaction Type and the record is saved, you cannot modify the set values.

Only 10 records can be added at time. You cannot duplicate the code which is having more than one record of the same combination of Document Type and Transaction Type. The system alerts you on saving the record by highlighting the duplicate codes with a unique color code.

 

As a pre-requisite setup, ensure that the document types are setup in the Document Type setup screen under System Manager module. Only then the transaction screens will be displayed as Module name in the Document Type screen  under Sterile Stock module.
