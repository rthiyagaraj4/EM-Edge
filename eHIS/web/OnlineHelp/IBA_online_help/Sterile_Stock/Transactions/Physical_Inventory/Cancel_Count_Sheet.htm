Cancel Count Sheet

From this function you can Cancel the generated count sheet.

The count sheet is cancelled only prior to posting the variances, so that the Items of the selected Store are available for transaction entry. Count sheets can be cancelled for individual groups of the store.

Cancelling Count Sheet

From SS Physical Inventory, select the Cancel Count Sheet function.

The Cancel Count Sheet screen gets displayed.

  1. Click [Query] to invoke the Query screen.

  2. Select the Physical Inventory ID from the list and click [Search] or enter the criteria and click [Search] to display the generated count sheets based on the entered criteria.

    Enter the following search criteria:

    1. Physical Inv ID – Select the Physical inventory ID from the list. This is a mandatory field.

    2. Owner Store – Select the owner store form the list. The owner stores are displayed in the list.

    3. Current Store – Select the current store from the list.

  3. On searching, the Owner Store, Current Store and Group of the generated count sheet are displayed.

  4. Select the ‘Select All’ check box to cancel the count sheet for all the groups or select the individual check box to cancel the count sheet for individual groups.

  5. Click [Apply] to cancel the generated count sheet or [Reset] to clear the entered data.
