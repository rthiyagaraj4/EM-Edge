Tray Usage on Patient

From this function you can Record the tray usage details.

The usage details are recorded by the requestor location. If the Patient ID has already been captured while issuing the tray the same can be altered in case of any error in the recorded patient information.

Recording the Tray Usage details

From SS Transactions > Tray Usage, select the Tray Usage on Patient function.

The Tray Usage on Patient screen is displayed.

  1. Click [Query] to display the issued trays based on the default criteria or enter the search criteria and click [Search] to search for tray usage on patient details. Displays details such as Iss Doc type, Issue Doc no, Req Doc no, Prepare Doc no, GroupType, Group, Tray No, Patient ID and Patient Name.

    Enter the following search criteria to select a specific record:

    1. Store – Displays all the stores assigned to the user. You should have Issue Group access. Selecting the store is mandatory.

    2. Iss Doc Type – Select the Issue group document type from the list. The default ‘Document Type’ set in ‘SS Parameter for Facility’ is displayed, by default. This value can be modified.

    3. Issue Doc No – Enter the issue document number in numeric format of eight digits maximum length.

    4. Group Type – Select the required group type from the list. The listed values are ‘Set’, ‘Tray’, ‘Soft Pack’ and ‘Single’. The value ‘All’ is displayed, by default.

    5. Group - Enter the group name in alphanumeric format or click [?] function to select the required group. All the groups created in ‘CSSD Store’ function are displayed.

    6. Tray No – Enter the tray number or click [?] to search for the required tray number. The issued tray numbers are displayed.

    7. Patient ID – Enter the Patient ID or click [?] to search for the required Patient ID.

    8. Displays Groups linked to Patient – The values are Linked to Patient, Not Linked to Patient, Both (Default Value)

  2. Click ‘Iss Doc Type’ hyperlink to view the tray usage details. The displayed tray details are Doc Type, Doc No, Group Type, Group, Tray No and Bin Location.

  3. The Patient ID and the Patient Name is displayed, if the tray is issued against a patient.

  4. Click [?] to select a change the Patient ID or to assign a new patient for the issued tray.

  5. Click [Apply] to record the tray usage details or [Reset] to clear the entered data.
