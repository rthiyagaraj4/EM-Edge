Request Loan From Vendor

From this function you can Request sterile stock from third party suppliers if the required CSSD Trays are unavailable or if sufficient quantities are not available.

This function allows you to multi-select sterile stock from third party suppliers if the required CSSD Trays are unavailable or if sufficient quantities are not available.

Requesting Loan from Vendors

From the SS Transactions > Loan > select the Request Loan from Vendor function.

The Request Loan from Vendors screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. < Related Field Description>

    Enter the following details:

    1. Doc Type – Select the request document type from the list. The values listed are defined in the ‘SS Document Type’ function and the default value displayed is specified in the ‘SS Parameter for Facility’ function for Request group. This is a mandatory field.

    2. Doc No. – The request document number generation is based on the selected document type. If the document number has to be manually generated for the selected document type, the user is informed. Enter the document number in numeric format of eight digits maximum length. If the system generates the number, you are not allowed to enter the document number.

    3. Doc Date - The system date is displayed, by default. You can change the document date. Enter the date in format of dd/mm/yyyy or select the Date using the Calendar function. The entered date must not be less than the Processing month and year set in ‘SS Parameter for Facility’ and not greater than current system date. This is a mandatory field.

    4. Requested By Store– Select the request store from the list. Only those Stores are displayed for which the User has access. This is a mandatory field.

    5. Requested On Vendor– Select the owner store from the list. Only those Stores are displayed for which the User has access. This is a mandatory field.

    6. Doc Ref – Enter the document reference in alphanumeric format of 15 characters maximum length.

    7. Entry Completed – Select the ‘Entry Completed’ check box to indicate that entry is completed and request document is available for authorizing.  This check box is selected by default.

    8. Entered By – The logged in user name is displayed and disabled.

    9. Request Type - Select the Request Type from the list. The values displayed are Normal, Urgent and Elective. The value Normal is displayed, by default.

    10. Request Group – Click [Group Search] to launch the Group Search and Select dialog to search for Group details.

  3. Click [Group Search] to launch the Group Search and Select dialog to search for Group details. The search records detail include Group Type, Request Group, Surgery Type, Surgery Name, Total Available Quantity in CSSD Main and Store.

    1. Search By – Select the Code or Description radio button. The search results are displayed based on selection.

    2. Search Text – Enter the keyword to search the records.

    3. Search Criteria – Select the criteria from the drop-down. The listed values are ‘Starts With’, ‘Contains’ and ‘Ends With’.

    4. Surgery Type/Name – Click [?] to search and select the CSSD Surgery types from the CSSD Surgery Type – Search and Select dialog.

    5. Search By – Select the search by option. The options available are Code and Description.

    6. Search Text – Enter the search text. If you select the search by option as Code, enter the Code in this text box. Else if you select Description, type the description here.

    7. Click [Search] to display the Code, Description, Surgery Type and Group Type or click [Reset] to clear all values.

    8. Select – You can select the check box against each item. Or, select the ‘Select’ check box at the header level to select all the records.

    9. Click [OK]. The selected records gets displayed in the search results

  4. If you want to delete any record, select the Del check box against that record and click [Delete]. The selected record gets deleted.

  5. Enter the request quantity of the stocks in the text box. This is a mandatory field.

  6. Click [Apply]; the Authorize alert message popup is invoked with Yes and No buttons.

  7. Click [Yes] in popup to apply the changes. The system generates document number and displays the document at the bottom of the screen. Click [No] in popup to reset the details.

  8. Click [Reset] to clear the entered data.

  9. Click [Query] to search for the existing records
