Generate Sterilization Request

From this function you can Generate requests for sterilization of trays.

The Generate Sterilization Request function aids the owner store to request for sterilization. The sterilization request results in generation of documents based on washing, packing and/or sterilization requirements of the selected trays. The request generated is finalized in ‘Sterilization Request’ function.

 

Trays are returned back to the owner location on usage, by the requestors. On returning the trays, the trays must be sterilized for further usage. The owner location requests for sterilization using Generate Sterilization Request function.

Generating Sterilization Request

From SS Transactions > Sterilization Process, select the Generate Sterilization Request function.

The Generate Sterilization Request screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Select Trays] to display the prepared trays.

  3. Select the ‘Select’ check box to select a specific Tray or select the ‘Select’ check box from the header to select all the Trays from the current page.

  4. Click ‘Next’ hyperlink to view and select the Trays from the next page, click ‘Previous’ hyperlink to view and select the Trays from the previous page.

  5. Click [Apply] to generate the sterilization request or [Reset] to clear the entered data.

  6. On clicking [Apply], the document numbers are generated for the selected Trays.

If ‘Generate Sterilization Request’ check box is selected in the SS Parameter for Facility function, the sterilization requests for all the prepared trays are generated automatically. If the ‘Generate Sterilization Request’ check box is not selected in the SS Parameter for Facility function, the sterilization requests for all the prepared trays are generated using ‘Generate Sterilization Request’ function.
