Autoclaving

From this function you can Record the sterilization details of a Tray and Query the un-sterilized trays. You can also Delete the un-sterilized trays.

Sterilization process is recorded in two stages. In Stage 1, the user selects ‘Trays’ and subject them to sterilization process. A load number is generated for the process. In Stage 2, if the sterilization process is successful, the load is finalized (sterilization process is completed) or the trays are re-loaded for sterilization. On sterilization, the load status is checked and sterilization process is completed.

Recording the Sterilization Details

From SS Transactions > Sterilization Process, select the Autoclaving function.

The Autoclaving screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Get Ster Req Dtls] to get the Sterilization Request details. If the sterilization request for the trays require Washing and packing, the records are displayed for Autoclaving only after the Washing and Packing process is finalized.  

  4. On generating the Sterilization Request, all the trays which require Sterilization for the specified conditions are displayed. The displayed details are ROF Document Type, ROF Document Number, Group and Tray Number. The finalized sterilization request documents (trays) pending sterilization are displayed.

  5. Select the ‘Autoclave’ check box to select the tray for sterilization. The check box is cleared, by default. To select all the trays for sterilization, select the check box in the column header.

  6. Click [Apply] to add the trays for sterilization or [Reset] to clear the entered details.

  7. On clicking [Apply], the ‘Confirm Authorize’ dialog is displayed. Click [Yes] to finalize the sterilization details or [No] to finalize the sterilization request details in the query mode.

  8. The load number is generated on recording the details.

  9. Click [Query] to display all the incomplete load details for which ‘Entry Completed’ check box is not selected while saving the record or enter the search criteria and select a specific record.

    Enter the following search criteria:

    1. Autoclave Unit - Select the autoclave unit from the list.

    2. Load No. – Enter the load number in numeric format of eight digits maximum length.

    3. Autoclave Date - Enter the date in date format of dd/mm/yyyy or select the date using the Calendar function.

    4. Store - Select the Store from the list. The Stores are listed based on the rights assigned.

    5. Entry – The value ‘Completed’ is displayed, by default. The other values listed are ‘All’ and ‘Not Completed’. Select the required value from the list.

  10. Click Autoclave Unit hyperlink to complete the sterilization process.

  11. Select the ‘Delete’ check box to exclude the tray from the Sterilization process or click [Delete] to cancel the operation.  

  12. On clicking [Delete], the ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.

  13. Click [Apply] to complete and finalize the sterilization process.

The trays are packed after the Sterilization request is generated. If washing is applicable, the trays are packed after the washing process is completed. The trays are available for packing if packing is applicable for the particular request.

The load details are displayed for the selected load number. You can modify the load status of the records for which entry is completed. The load details entered during stage 1 are displayed and disabled.

Select the load status value from the list. The values listed are Passed and Failed. The value ‘Passed’ is displayed, by default. This is a mandatory field. Passed indicates the load has successfully completed sterilization process and Failed indicates, sterilization process failed for the load.

If sterilization process has successfully completed for the selected load, click [Apply].

If sterilization process has failed for the selected load, click [Apply]. On finalizing, the failed trays are available for re-loading.
