Maintain Sterilization Request Status

From this function you can Maintain a range of user-defined status for the Sterilization Requests.

Maintaining Sterilization Request Status

From SS Transactions > Sterilization Process, select the Maintain Sterilization Request Status function.

The Maintain Sterilization Request Status screen gets displayed.

  1. Click [Query] to display all the trays based on the default criteria or enter the search criteria and select a specific record.

    Enter the following search criteria:

    1. Doc Type – Select the document type from the list. The document type as set for ‘Process Ster Req’ in SS Parameter for facility will be displayed, by default.

    2. Doc No – Enter the sterilization request document number as was generated in sterilization request function in numeric format of eight digits maximum length.

    3. Doc Date – Enter the document date in dd/mm/yyyy format.

    4. Store – Select the required store from the list. The store as set in ‘SS Parameter for facility’ will be displayed, by default.

  2. Click ‘Doc Type’ hyperlink to record the process details. The Document Type, Document Number, Document Date, Document Reference and Store details are displayed and disabled.

  3. Select the Process Stage from the list. The list displays all the user-defined values set for ‘Sterilization Process Stage’ function in the Code Setup.

  4. Enter the Process Remarks in the alphanumeric format.

  5. Click [Apply] to confirm the acknowledgement or [Reset] to clear the entered data.
