Sterilization Request

From this function you can Raise sterilization request for a Group Type and Query for the un-finalized sterilization requests. You can also Delete the un-finalized sterilization requests.

The Sterilization Request function aids the owner stores to raises sterilization requests for a group. The un-sterilized trays returned back to the owner locations are sent to service locations for sterilization. The owner stores place request to perform sterilization and the sterilization process is performed using this function. You can finalize the sterilization requests raised in the ‘Generate Sterilization Request’ function.

The system checks for the following conditions:

  • Owner stores having rights to sterilization request process

  • Owner stores having access to the trays

  • The logged in user must be assigned to these owner stores

Raising Sterilization Request

From SS Transactions > Sterilization Process, select the Sterilization Request function.

The Sterilization Request screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Add] to add the tray for sterilization request or [Cancel] to clear the entered details.

  4. On adding the request, the tray details are displayed in the right hand side of the screen.

  5. Click ‘Group Type’ hyperlink to modify the sterilization request details. Click [Modify] to modify the changed details or [Cancel] to restore the previous details. You can modify the contaminated details, Sterilization Type and Washing Type.

  6. Click ‘Del’ check box to exclude the entered tray details.

  7. Click [Apply] to save the Sterilization Request details or [Reset] to clear the entered data.

  8. On clicking [Apply], the ‘Confirm Authorize’ dialog is displayed. Click [Yes] to finalize the sterilization request details or [No] to finalize the sterilization request details in the query mode.

  9. Click [Query] to display all the un-authorized sterilization requests.

  10. The un-authorized sterilization requests are displayed based on the default criteria set in the SS Parameter for Facility function or enter the search criteria and click [Search] to select a specific record.

    Enter the following criteria to select the record to raise request for sterilization:

    1. Doc Type – Select the sterilization request document type from the list. The values listed are defined in the ‘SS Document Type’ function and the default value displayed is specified in the ‘SS Parameter for Facility’ function for process sterilization request. This is a mandatory field.

    2. Doc No. – Enter the sterilization request document number in numeric format of eight digits maximum length.

    3. Doc Date - Enter the sterilization request document generation date in dd/mm/yyyy format or select the date using the Calendar function.

    4. Store – Select the store from the list.

    5. Service Location – The Service Location defined in ‘SS Parameter for Entity’ function is displayed, by default. Select the service location from the list. All the enabled Service Locations defined in the ‘Service Location’ function are displayed in the list.

    6. Sterilization – Select the sterilization value from the list. The value ‘All’ is displayed, by default. The values displayed are ‘All’, ‘Required’ and ‘Not Required’.

    7. Washing – Select the required value from the list. The value ‘All’ is displayed, by default. The values displayed are ‘All’, ‘Required’ and ‘Not Required’.

    8. Packing – Select the value from the list. The value ‘All’ is displayed, by default. The values displayed are ‘All’, ‘Required’ and ‘Not Required’.

  11. Click ‘Doc Type’ hyperlink to authorize the sterilization request details. You can modify the tray details.

  12. Click [Apply] to authorize the sterilization request for the tray or select the ‘Del’ check box and click [Delete] to delete the sterilization request for the tray.

  13. On clicking [Apply] the ‘Confirm Authorize’ dialog is displayed. Click [Yes] to authorize the sterilization request or [No] cancel the authorization.

  14. On clicking [Delete], the ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.

The sterilization requests generated in the function ‘Generate Sterilization Request’ are available as query result.
