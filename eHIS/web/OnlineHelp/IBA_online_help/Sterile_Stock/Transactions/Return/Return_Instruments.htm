Return Instruments

From this function you can Record the details of the returned instruments used by Patients.

The Return Instrument function aids in recording the returned instruments details of trays used by patients which is returned back to owner stores by the requesting stores. Only returnable items are returned back to the owner store.

Recording Returned Instrument Details

From SS Transactions > Return, select the Return Instruments function.

The Return Instruments screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields of the Return Instruments section and Return Item Details section.

    Enter the following details to record item return details:

    Return Item Details Area

    1. Group/Tray No/Item – The group type, tray number and returnable instruments used by the selected patient are listed. Select the value from the list. Only those items that are marked as returnable are listed. This is a mandatory field.

    2. Details - on selecting the Group,  Details hyperlink is enabled. Click 'Details' hyperlink to view the details of Items associated to the selected Group.

    3. Damaged – Select the check box to indicate that the item is damaged.

    4. Return Qty - Enter the returning quantity in numeric format of 12 digits maximum length. The quantity must not exceed the issued quantity. This is a mandatory field.

    5. Cost – The cost of the instrument is displayed on specifying the quantity.

    6. Contaminated – Select the check box to indicate the item is contaminated.

    7. Remarks - Enter the remarks if any in alphanumeric format.

    Enter the following details to record instrument return details:

    Return Instrument Details Area

    1. Doc Type – Select the return document type from the list. The listed values are defined in SS Parameter for Facility. This is a mandatory field.

    2. Doc No. – The return document number generation depends on the selected document type. If the document number has to be manually generated for the selected document type, the user is informed. Enter the document number in numeric format of eight digits maximum length. If the system generates the number, then user is not allowed to enter the field value. This is a mandatory field.

    3. Doc Date - The system date is displayed, by default. You can change the document date. Enter the date in date format of dd/mm/yyyy. The entered date must not be less than the Processing month and year set in ‘SS Parameter for Facility’ and greater than current system date. This is a mandatory field.

    4. Receiving Store – Select the owner store from the list. Only those stores are listed from which the logged in user can issue tray(s). This is a mandatory field.

    5. Patient ID – Enter the Patient ID in the format defined in Patient Identity Management module or select the Patient ID using the lookup function.

    6. Patient Name - The patient name is displayed on selecting the Patient ID.

    7. Returning Store – Select the request store from the list. Only those stores are listed from which the logged in user can request for tray(s). This is a mandatory field.

    8. Doc Ref – Enter the document reference in alphanumeric format of 15 characters maximum length.

    9. Entered By – The logged in user name is displayed.

    10. Entry Completed – Select the check box to complete the entry. If entry is completed, the return document is available for finalization.

  3. On entering the Item return details, click [Add] to add the item details or [Cancel] to clear the entered details.

  4. Select the ‘Del’ check box to exclude the item details from the list. On saving the record, the item is removed from the list.

  5. Click Group / Tray No / Item hyperlink to modify the Item details and click [Modify] to record the changes or [Cancel] to clear the entered details.

  6. Click [Apply] to record the instrument return details or [Reset] to clear the entered details.

  7. The ‘Confirm Finalize’ dialog is displayed if the return entry is completed. Click [Yes] to confirm the return instrument details or [No] to cancel the action.

  8. Click [Query] to invoke the un-finalized instrument return records which are displayed based on the default criteria set in the SS Parameter for Facility function or enter the search criteria and click [Search] to select a specific record.

    Enter the following search criteria:

    1. Doc Type – Select the return instrument document type from the list.

    2. Doc No. – Enter the return instrument document number in numeric format.

    3. Doc Date – Enter the return instrument document generation date in dd/mm/yyyy format.

    4. Receiving Store - Select the owner store from the list.

    5. Returning Store - Select the requestor store from the list.

    6. Patient ID - Enter the Patient ID in the format as defined in Master Patient Index module or select it using [?] function.

    7. Entry – Select the value from the list. The listed values are All, Completed and Not Completed. The value Completed is displayed, by default.

  9. Click ‘Doc Type’ hyperlink to finalize the instrument return documents. You can modify the instrument return details for the documents for which ‘Entry Completed’ check box is not selected.

  10. Click [Apply] to finalize the instrument return document details or [Delete] to delete the selected record.

  11. On clicking [Delete], the ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.

  12. A document number is generated on recording the instrument return details.

The Instruments for which ‘Returnable Instrument’ check box is selected in the discrepancy details screen of Return Group function can be returned.
