Transaction Remarks

From this function you can enter appropriate remarks while returning sterile groups not associated to the patient.

From SS Code Setup, select the Transaction Remarks function.

The Transaction Remarks screen is displayed.

  1. Select the Transaction Type; displays the Remarks Code and Remarks Description.

  2. Check Select checkbox to select the remarks that have to be set as predefined remarks.

The remarks selected here are displayed as Reason in the Return Group screen.

  1. Click [Apply] to record the transaction remarks details or [Reset] to clear the entered data.
