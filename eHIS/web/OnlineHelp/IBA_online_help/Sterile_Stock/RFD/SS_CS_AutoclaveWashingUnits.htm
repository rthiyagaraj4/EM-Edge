Enter the following details to create an Autoclave/Washing Unit record:

  1. Unit Type – Select the required value to indicate the type of the unit defined from the list. The values displayed are ‘Autoclave’ and ‘Washing’.

  2. Sterilization Type - Select the required sterilization type to be associated to the autoclave unit from the list. This field is enabled only for the Unit Type ‘Autoclave’. The enabled Sterilization Type records are displayed in the list.

  3. Code – Enter the code in alphanumeric format of eight characters maximum length. The code defined indicates the Autoclave/Washing Unit. Once the code is defined and saved, it cannot be modified. This is a mandatory field.

  4. Service Location – Select the Service Location from the list. The defined autoclave/washing unit is associated to the selected service location. The enabled Service Location records are displayed in the list. This is a mandatory field.

  5. Long Description - Enter the long description of the Autoclave/Washing Unit in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  6. Short Description - Enter the short description of the Autoclave/Washing Unit in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  7. Temperature – Enter the temperature of the unit in decimal format of nine digits maximum length.

  8. Unit Condition – Select the working condition of the unit from the list. The values displayed are ‘Ready to use’ and ‘Not ready to use’.

The Units are prepared at different location after setting all the necessary instruments used for the Autoclaving or Washing and tested. If the test is successful, the Unit is ready for use; and if the test is failed and some defects exist, the Unit is not ready for use.

  1. Enabled – Select the check box, to enable the defined Autoclave/Washing Unit. Clear the check box, to disable the defined Autoclave/Washing Unit. The check box is selected, by default. Only enabled Autoclave/Washing Unit are available for selection in the applicable functions.
