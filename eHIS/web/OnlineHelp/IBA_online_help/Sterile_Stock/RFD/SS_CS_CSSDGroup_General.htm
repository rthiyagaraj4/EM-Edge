The General Tab is displayed, by default.

Enter the following General details to define Sterile Stock Groups:

  1. Group - Enter the Group code in alphanumeric format of 10 characters maximum length. The code defined should be unique for each group record. Once the code is defined and saved, it cannot be modified. This is a mandatory field.

  2. Alpha Code – Enter the alpha code for the Group in alphanumeric format of 10 characters maximum length. Alpha code is entered for the purpose of grouping and easy retrieval. For example, you can group all the Group codes starting with ‘A’ under one alpha code. This is a mandatory field.

  3. Long Description – Enter long description of the Group in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  4. Short Description – Enter short description of the Group in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  5. Group Type – Select the Group Type from the list. The listed values are ‘Set’, ‘Tray’, ‘Soft pack’ and ‘Single’. The value ‘Set’ is displayed, by default.

  6. Sub Classification – Select the Sub-Classification from the list. This field is enabled only for the group type ‘Soft Pack’. The sub-classifications are defined in the function ‘Sub Classification’ are displayed.

  7. Non-returnable – Select the check box to indicate group (Tray/Set/Soft Pack/Single) is not required to return back to the owner after usage. The check box is cleared, by default.

  8. Surgery Type – Select the Surgery Type to be associated to the group. The listed values are defined in the function ‘Surgery Type’.

  9. Usage Type – Select the required Usage Type from the list. The listed values are defined in the function ‘Usage Type’.

  10. Based on Checklist - Select the check box if the CSSD Group is defined based on the Check List. On selecting the check box, the Checklist field is enabled and the Item Add details are not available.

  11. Checklist - Select the Checklist using the [?] function. The Check Lists defined in the Check List function are displayed for selection. This is enabled and mandatory field if 'Based on Checklist' check box is selected.

  12. Enabled – Select the check box, to enable the defined group record. Clear the check box, to disable the defined group. The check box is selected, by default. Only enabled group codes are available for selection in the applicable functions.
