Select ‘Additional’ tab to enter the additional details of the defined group.

Enter the following details:

  1. Item Class – Select the Item Class from the list. The Item Classes are defined in the ‘Item Class’ function of Materials Base module.

Only those Item Classes are displayed for which ‘Sterile Stock’ is applicable in the Item Class function.

  1. Item Analysis – Select the Item Analysis from the list. The listed values are defined in the ‘Item Analysis’ function of Materials Base module.

  2. Statistical Group – Select the Statistical Group from the list to be associated to the defined Group. The Statistical Groups are defined in the function ‘Statistical group’.

  3. Washing – Select the check box to indicate washing is required for the defined Group. The check box is selected, by default.

  4. Washing Type – This field is enabled if washing is applicable. You can select the required Washing Type from the list. The listed values are defined in the ‘Washing Type’ function. This is a mandatory field.

  5. Packing – Select the check box to indicate packing is required for to the defined Group. The check box is selected, by default.

  6. Pack Item – Enter the Pack Item required for the group in alphanumeric format of 40 characters maximum length or click [?] function to search for the required Pack Item. The pack items are defined in the ‘CSSD Item’ function.

  7. Pack Item Qty – Enter the packing item quantity in numeric format of 12 digits maximum length. The specified quantity is used for wrapping the items of the group.

  8. Pack Addl. Cost – Enter the additional cost incurred for packing in numeric format 16 digits maximum length. The additional costs are the administrative costs like Electricity, Labor and so on.

  9. Sterilization – Select the check box to indicate Sterilization is required for the group. The check box is selected, by default.

  10. Sterilization Type – This field is enabled if sterilization is applicable. You can select the required Sterilization Type from the list. The Sterilization Types are defined in the ‘Sterilization Type’ function. This is a mandatory field.

  11. Remarks – Enter the remarks for the Group in alphanumeric format of 200 characters maximum length or click ‘Remarks’ hyperlink to enter the remarks. Enter the remarks and click [Close] to exit from the Remarks screen.
