Enter the following details to Generate Sterilization Request:

  1. Doc Type – Select the document type from the list. The values listed are defined in the ‘SS Document Type’ function with the Transaction Type ‘ROF Processing’ applicable. The default value is displayed based on the value specified in the ‘SS Parameter for Facility’ function for Process Sterilization Request. This is a mandatory field.

  2. Doc Date - The system date is displayed, by default. You can change the document date. Enter the date in the format of dd/mm/yyyy or select the required date using the Calendar function. The entered date must not be less than the Processing month and year set in ‘SS Parameter for Facility’ function and not greater than current system date. This is a mandatory field.

  3. Group Type – Select the group type from the list. The displayed values are Set, Tray, Soft Pack and Single. This field is mandatory.

  4. Store – Select the store from the list. Only those stores are displayed for which the user has access and ‘Sterilization Request Processing’ transaction function applicable. This is a mandatory field.

  5. Service Location - The service location specified in ‘SS Parameter for Facility’ function is displayed, by default. Select the Service Location from the list. The Service Locations are defined in the ‘Service Location’ function. This is a mandatory field.

  6. Doc Ref – Enter the document reference in alphanumeric format of 15 characters maximum length.

  7. Discrepancy – The value ‘All’ is displayed, by default. Select the value from the list. The listed values are ‘All’, ‘With Discrepancy’ and ‘Without Discrepancy’.

  8. All - All the un-sterilized trays currently available with the selected owner store is displayed.

  9. With Discrepancy - Only the un-sterilized trays with discrepancies are displayed.

  10. Without Discrepancy - Only the un-sterilized trays without discrepancies are displayed.

  1. Print Sterilization Request – Select the check box to print the sterilization request.  

  2. Sterilization Required – The check box is selected, by default indicating sterilization is required. Clear the check box to indicate sterilization is not required.

  3. Washing Required - The check box is selected, by default indicating washing is required. Clear the check box to indicate washing is not required.

  4. Packaging Label Required - The check box is selected, by default indicating packing is required. Clear the check box to indicate packing is not required.
