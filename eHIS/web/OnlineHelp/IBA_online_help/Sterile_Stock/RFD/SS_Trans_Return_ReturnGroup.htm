Enter the following details to return the trays to the owner:

  1. Doc Type – Select the return document type from the list. The values listed are defined in the function ‘SS Document Type’ and the default value displayed is specified in the function ‘SS Parameter for Facility’ for Return group. This is a mandatory field.

  2. Doc No. – The return document number generation depends on the selected document type. If the document number has to be entered manually for the selected document type, then the user is informed. Enter the document number in numeric format of eight digits maximum length. If the system generates the number, user is not allowed to enter the field value. This is a mandatory field.

  3. Doc Date - The system date is displayed, by default. You can change the document date. Enter the date in date format of dd/mm/yyyy. The entered date must not be less than the Processing month and year set in ‘SS Parameter for Facility’ and greater than current system date. This is a mandatory field.

  4. Return From – Select the returning store from the list. Only those stores are displayed from which the logged in user can perform return group function. This is a mandatory field.

  5. Return To – Select the owner store from the list. Only those stores are displayed from which the logged in user can issue tray(s). This is a mandatory field.

  6. Doc Ref – Enter the document reference in alphanumeric format of 15 characters maximum length.

  7. Entry Completed – Select the check box to indicate that entry is completed and return document can be finalized. The check box is cleared, by default. If entry is not completed, the return document details can be modified in the Query mode.   

  8. Entered By – The logged in user name is displayed and disabled.

  9. Display Groups Linked to Patient  – Displays the sterile groups associated to the patient. The values are All, Yes, No. You can specifically search for Sterile Groups that are associated with patient, or not associated with patient or All the Sterile Groups.
