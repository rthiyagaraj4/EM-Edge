Enter the following details

  1. Document Type – Select the request document type from the list. The values listed are defined in the ‘SS Document Type’ function and the default value displayed is specified in the ‘SS Parameter for Facility’ function for Request group. This is a mandatory field.

  2. Request Doc No. – The request document number generation is based on the selected document type. If the document number has to be manually generated for the selected document type, the user is informed. Enter the document number in numeric format of eight digits maximum length. If the system generates the number, you are not allowed to enter the document number.

  3. Doc Ref - Enter the document reference in alphanumeric format of 15 characters maximum length.

  4. Group – Enter the Group or select the Group from the lookup. The Groups defined in the masters are listed.

  5. From Vendor – Select the Vendor from drop-down list. The vendors defined in the master are listed. This is a mandatory field.

  6. To Store - Select the store from the list. Only those Stores are displayed for which the User has access.

  7. Request Date [From-To] – Select the date using calendar function. Document Date should not be greater than System Date or Request To Date.
