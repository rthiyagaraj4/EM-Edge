Enter the following details to prepare a group:

  1. Doc Type – Select a Document Type for the transaction from the list. The listed values comprises of all the documents associated to the prepare transaction in ‘SS Document Type’. The default Document Type set in the ‘SS Parameter for Facility’ function is displayed. This is a mandatory field.

  2. Doc No. – Enter the Document number in numeric format of eight digits maximum length if manual entry is applicable for selected document or the system generates the document number on saving the transaction. This is a mandatory field.

  3. Doc Date – Enter the document date in DD/MM/YYYY format. This indicates the document preparation date. The document date is made default to system date. The document date can be modified, if required. The modified document date cannot be less than the current processing period as specified in SS Parameter for Facility or cannot be greater than the current system date. This is a mandatory field.

  4. Store – Select the Store from the list. The default value for Store is displayed based on the value set in ‘Store for Prepare Group’ field in ‘SS Parameter for Facility’ function. The stores are displayed based on the restrictions set in ‘SS User Access for Store’ and applicability of the store to perform the specific transaction  as set in ‘CSSD Store’. This is a mandatory field.

  5. Doc Ref – Enter the document reference number in alphanumeric format of 15 characters Maximum Length. This value serves as some reference to help in identification of the document.

  6. Group Search – Click [?] function to search for the required groups from the Group Search and Select dialog. The groups associated to the selected store are displayed. This is a mandatory field. By default all the groups will be selected. If you do not want to dispatch any group for preparation, clear the Prepare check box.   

    1. Search By – Select either Code or Description to search by item code or item description.

    2. Search Criteria – Select the search criteria such as Ends with.

    3. Search Text – Enter the item code or Item description based on the selection made in Search By.

    4. Store – Displays the name of the store selected.

    5. Search – Searches and displays the search results based on the search criteria entered.

    6. Reset – Resets the values entered in the fields.

    7. Code – Displays the item code as a hyperlink. When clicked, displays the Group Item Details dialog.

    8. Description – Displays the description of the item.

    9. Select –You can multi-select or single-select the check box(es) to select the group(s).

    10. Ok – Completes the group search and selection and displays the search results in the Prepare screen.

    11. Cancel – Cancels the search operation.

  7. Quantity – Enter the quantity to be prepared in numeric format of 4 digits maximum length. This is a mandatory field.

  8. Bin Location – Select the Bin Location from the list. The Bin Locations associated to the selected Store in the function ‘Bin Location’ in Materials Base module are displayed. This is a mandatory field.

  9. Total Value – The system calculates total value based on the components of the Tray and quantity specified.
