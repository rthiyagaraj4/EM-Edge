Stock Status By Store

From this function you can View the details of the Items maintained in a Store, information on the value of stock maintained and quantity of stock available.

This function provides details of the total number of trays held, sterilized trays, un-sterilized trays and trays currently at service locations. The tray status is displayed for the respective owner locations. If trays are currently in requestor stores, the stock status is displayed under the requestor stores.

Viewing the Tray details maintained in a Store

From SS Queries, select the Stock Status By Store function.

The Stock Status By Store screen gets displayed.

  1. The tray details of the trays maintained in all the Stores are displayed, by default. Enter the search criteria and click [Search] to view the specific tray details in a Store.

    Enter the following search criteria:

    1. Owner Store – The owner stores are listed. Select the store from the list.

    2. Current Store - Select the current store from the list. All the stores (owners and requestors) are listed.

    3. Group Type – The value ‘All’ is displayed, by default. Select the Group Type from the list. The listed values are Set, Tray, Single and Soft Pack.

    4. Group - Enter the group in alphanumeric format of 15 characters maximum length or select the group using [?] function. The search screen gets displayed and on entering the required conditions, tray details corresponding to the selected ‘Group Type’ are listed. Select the required tray.

  2. On searching, the tray details of the selected store are displayed based on the entered criteria.

    The following details are displayed:

    1. Owner Store – Displays the owner store name.

    2. Current Store - Displays the requestor of the owner store.

    3. Group Type – Displays the group type name and disabled.

    4. Group Code – Displays the group name.

    5. Total Tray(s) – Displays the total number of tray(s) currently held by the store.

    6. Sterilized – Displays the number of trays sterilized.

    7. Unsterilized - Displays the number of trays un-sterilized.

    8. Pending Request Qty - Displays the number of quantity requested that is pending to be issued. The pending quantity includes both the request raised manually as well as auto request.

    9. In Service Location – Displays the number of trays in the service location.

    10. Under Acknowledgement – Displays the number of trays pending for acknowledgement.

    11. Pending Acknowledgement - Displays the number of quantity that is pending to be acknowledged by the current store.

  3. The displayed details are: Group Type, group, Group Type, Owner Store/Current Store, Group Name, Total Tray's), Sterilized, Unsterilized, In Service Location, Pending Request Qty, Under Acknowledgement and Pending Acknowledgement details.
