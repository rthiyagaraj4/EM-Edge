Document/Tray Return Status

From this function you can View both the document-level return details as well as granular Tray level return details along with the acknowledgement details.

You can enter criteria like Date range, ‘From’ and ‘To’ Stores, Entry Completion Status, Finalize status and return document number.

Search for Document/Tray Return status

From the SS Queries, select the Return Status function.

The Return Status screen gets displayed.

  1. Enter the search criteria and click [Search] to view the required document-level return details such as Doc Type, Doc No, Doc Date, Doc Ref, Returned Date, Returned by, ‘From’ and ‘To’ Stores along with the acknowledgement status.

  2. Click on the Document Type hyperlink to invoke the Sterile group-level Return Details dialog. The dialog displays the Document Number, Return Group, Group Type, Return Qty., Acknowledge Qty., Pending Ackn. Qty and Reject Qty.

  3. Click the sterile group name hyperlink in the Return Group column in the Sterile Group Level details dialog to view the tray level details of a specific sterile group. This launches the Tray Level details dialog. The dialog displays the Return Group, Tray No, Patient ID, Bin location, Reason, Status of the Tray Level and Discrepancy.

  4. Click the corresponding Discrepancy link; the ‘Tray Discrepancy’, ‘Return Discrepancy’ options are displayed. Select any one of the options to view the corresponding details in the respective dialog.

The Discrepancy link is enabled only if either Return Discrepancy or regular Tray Discrepancy details are available for that specific tray number, based on the availability of Return Discrepancy, or Tray Discrepancy details.

  1. Click the sterile group name hyperlink in the Tray Level details dialog to view the group item details in the Group Item Details dialog.

  2. Click [Reset] to clear the entered data.
