Operator Station

From this function you can Create an Operator Station at the Facility level which is used for managing work related processes in the facility for Inpatients, Outpatients or both. You can Query for a required Operator Station.

An Operator Station represents a single point of control for a group of locations. It is useful in managing the patient workflow within the facility. Users associated to an Operator Station are allowed to carry out work-related processes such as viewing appointments, registering patients, placing orders, transferring patients, etc. or the set of patients belonging to locations that are associated with the Operator Station. Users are associated to an Operator Station. It is mandatory to setup Operator Stations, associate locations and users to the Operator Station to perform the operations associated to the Operator Station. All these setups are mandatory, before a set of functions related to an Operator Station is to be applied.

Creating an Operator Station

From the Operator Station Setup, select the Operator Station function.

The Operator Station screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter details in the mandatory (*) fields. <Related Field Description>

  3. The tabs for setting Operator Station access are displayed based on selecting the Operator Station Type. On selecting the Operator Station Type as

Inpatient

Outpatient

Inpatient/Outpatient

  1. Click [Apply] to save a Operator Station created or [Reset] to clear the entered data.

  2. Click [Query] to search for an existing Operator Station record.

  3. Click [Delete] to delete a queried record.
