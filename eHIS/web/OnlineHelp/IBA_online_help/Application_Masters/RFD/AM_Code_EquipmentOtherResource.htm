Enter the following details:

 

  1. Facility – Select the required facility from the list. The list box displays facility records as created in the ‘Facility’ function of the System Manager module. This field is mandatory.

  2. Resource Class – Select the required Resource Class from the list. The pre-defined values are ‘Equipment’ and ‘Other’. This field is mandatory.

  3. Resource Type – Select the Resource Type from a list of valid Resource Types under which the Equipment Other Resource is to be created. This field displays the Resource Types created using the ‘Equipment Other Resource Types’ function with the enabled check box selected. This field is mandatory.

  4. Resource ID – Enter the Resource ID in alphanumeric format of 15 characters maximum length. The Resource ID has to be unique and cannot be changed after creation. This field is mandatory.

  5. Long Description – Enter the long description of the Equipment/Other Resource in alphanumeric format of 30 characters maximum length. This field is mandatory.

  6. Short Description – Enter the short description of the Equipment/Other Resource in alphanumeric format of 15 characters maximum length. This field is mandatory.

  7. Applicable to Radiology – Select the check box to create Equipment for booking radiology appointments. This check box is cleared, by default. When Resource Class is selected as ‘Other’ this check box is disabled.

  8. Enabled – Select the check box to enable the Equipment Other Resource record. The check box is selected, by default. Clear the check box to disable the Equipment Other Resource record.

 

Location Details

  1. Building – Select the Building from a list of valid Buildings where the equipment/other resource is located. This field displays the buildings of the facility created using  the ‘Building in Facility’ function under ‘Location Setup’ with the enabled check box selected for the selected facility.

  2. Floor - Select the Floor from a list of valid Floors where the equipment/other resource is located. This field displays the Floors of the selected building created using the ‘Floor in Facility’ function under ‘Location Setup’ with the enabled check box selected for the selected facility.

 

Contact Details

  1. Person Name - Enter the name of the person in charge of the equipment/other resource in alphanumeric format of 30 characters maximum length.

  2. Telephone Number – Enter the Telephone Number of the person in charge of the equipment/other resource in alphanumeric format of 12 characters maximum length.

  3. Resource Narration - Enter the Resource Narration in alphanumeric format of 300 characters maximum length.
