Enter the following details to create a Trade Name record:

  1. ID - Enter the Trade Name ID in alphanumeric format of 20 characters maximum length. A Trade Name ID is used to identify the Trade Name. The Trade Name ID has to be unique and cannot be changed after creation. This field is mandatory.

  2. Long Name - Enter the long name of the Trade Name in alphanumeric format of 40 characters maximum length. This field is mandatory.

  3. Short Name – Enter the short name of the Trade Name in alphanumeric format of 20 characters maximum length. This field is mandatory.

  4. Drug – Select the check box to indicate that the Trade Name is applicable to a Drug or clear the check box if the Trade Name is not applicable to a Drug.

  5. Manufacturer – Click [?] function to invoke a search screen. Enter the required search criteria and based on the search criteria entered, this field displays the Manufacturers created in ‘Manufacturer’ function. Select the required Manufacturer. This field is mandatory.

  6. Enabled – Select the check box to enable the Trade Name record. The check box is selected, by default. Clear the check box to disable the Trade Name record.
