Enter the following details to restrict access to an Operator Station:

 

  1. Facility ID – Select a Facility ID from the list. The logged in facility is displayed, by default. This is a mandatory field.

  2. Operator Station - Select an Operator Station from the list. This is a mandatory field.

  3. User -  Select the appropriate User from the list. This field is enabled only if the required Operator Station is selected. Click [?] function to invoke a search screen. Enter the required search criteria and based on the search criteria entered, this field displays the Users associated to the selected Operator Station. This is a mandatory field.

  4. Location Type - Select a Location Type from the list. The listed types are based on the selected Operator Station. This is a mandatory field.

  5. Location – Select a Location from the list. The listed Locations are displayed based on the selected Location Type. This is a mandatory field.
