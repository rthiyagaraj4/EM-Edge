Enter the following details to define a Medical Team record:

 

  1. Facility – Select the required Facility from the list. The system displays all the facilities associated to the logged in user. The logged in facility is displayed, by default. This is a mandatory field.

  2. ID - Enter the code in alphanumeric format of six characters maximum length. A code is used to identify the Medical Team. The Medical Team code has to be unique and cannot be changed after creation.  This is a mandatory field.

  3. Long Description - Enter the long description of the Medical Team in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  4. Short Description - Enter the short description of the Medical Team in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  5. Specialty - Select the Specialty to be associated to the Medical Team from the list. The list of values displays the Specialty records created through ‘Specialty’ function. This is a mandatory field.

  6. Applicable for (Outpatients/Inpatients) – Select the required check box to indicate that the Medical Team is applicable for Outpatient or Inpatient or both.

  7. Applicable for Medical Report – Select the check box to indicate that the Medical Team is applicable for Medical Reports.  On selecting, the system allows to forward the medical request not only to the HOD/co-ordinator but also to a ‘Medical Report’ team.

  8. Applicable Period (From/To) - Enter the date range for which the defined Medical Team is applicable in ‘dd/mm/yyyy’ format. You can also select the date by clicking on the Calendar function.

  9. Enabled - Select the check box to enable the record. Clear the check box to disable the record. The check box is selected, by default.
