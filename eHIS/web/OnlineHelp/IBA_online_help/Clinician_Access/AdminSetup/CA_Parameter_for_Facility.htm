CA Parameter for Facility

From this function you can Define business logics or rules that influence the transaction function.

CA Parameter for Facility function is used to define parameter values at facility level. The set values influence the CA Transaction functions. Only one record can be created for a facility. It is mandatory to set up the Parameter before any operations are performed as these records store important details about Clinician Access module.

Based on the setup done in CA Parameter for Facility screen, the system prints the site or facility logo for clinical notes from Print Preview and Print in flowsheet or tree view mode.

Defining CA Parameter for Facility

From the CA Admin Setup > CA Parameter Setup, select the CA Parameter for Facility function.  

The CA Parameter for Facility screen gets displayed.

  1. Enter the details in the respective fields. <Related Field Description>.

  2. Click [Apply] to define CA Parameter for Facility values or [Reset] to revert to the previously defined values.
