Capturing Pregnancy Details

From this function you can Capture the Pregnancy details.

This function allows authorized practitioners to capture the pregnancy details. The drugs are to be prescribed to the patients based on the pregnancy details.

Capturing Pregnancy Details

From the CA Transactions, select the Capturing Pregnancy Details function.

The Capturing Pregnancy Details screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to display Patient ID, Patient Name, Gender, Age, Date of Birth, Practitioner Name.

  3. Select the Pregnant check box to enable Last Maturation Period (LMP), Expected Date of Delivery and Actual Date of delivery fields if the patient is pregnant. If the pregnant check box is cleared, the Last Maturation Period (LMP), Expected Date of Delivery and Actual Date of delivery fields are disabled.

  4. Enter the Last Maturation Period (LMP) using the calendar function. The Expected Date of Delivery is automatically displayed. The Expected Date of Delivery is defaulted as Actual Delivery Date. You can modify the Actual Delivery Date.

  5. If the patient record is already recorded in the system, the Pregnant, Last Menstrual Period, Practitioner, Actual Delivery date and Status are displayed automatically.

  6. If you need to modify the Patient record, on clicking Pregnant hyperlink, the Pregnant checkbox is displayed. Select the Pregnant check box to display Last Maturation Period (LMP), Expected Date of Delivery, First Trimester, Pregnancy Stage, Actual Delivery Date, Practitioner Name, Status and Remarks. This check box is displayed only if the patient records are not available in the system.

  7. Click [Record] to record the modified or new pregnant details or click [Clear] to reset the data in the fields.
