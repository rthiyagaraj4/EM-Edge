Pending Doctor Documentation

 

From this function attending practitioner can Review pending documentation of their patients.  

The Pending Doctor Documentation function allows the attending Practitioner of the patient to review the pending doctor documentation of their patients with respective count.  The hyperlink is visible only if there are records.

Once the patient record is saved with Doctor Query check box selected in the Coder Function, then the record will appear in the Pending Doctor Documentation hyperlink with respective count.

View Pending Doctor Documentation

From the CA Transactions > Clinician Access > Summary Menu > Coders Review > select the Pending Doctor Documentation hyperlink.

The Pending Doctor Documentation screen gets displayed.

  1. By default, the patient details for the last 7 days get displayed.

  2. Enter the search criteria if required.

    1. Patient ID – Enter the required patient ID or select it from the lookup.

    2. Period – By default displays the date of last 7 days. You can change the date range if required

    3. Click [Search], the patient records matching the criteria appear.

  3. On clicking the Patient Name hyperlink, the Record Clinical Notes window gets invoked. You can record the clinical notes in the window. (To open the Record Clinical Notes function by default, you need to define it as the first function in ‘CA Menu” assigned to the coder).

  4. To complete the record, select the check box against the patient record and click [Record Completion] button. The record gets removed from Doctor's list in summary.
