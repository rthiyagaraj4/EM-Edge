View Prepared Medical Report

From this function you can View medical reports of ‘Prepared by Practitioner’ status.

This function allows you view the medical reports belonging to status ‘Prepared by Practitioner’.  The system will not display any other medical report status request IDs.

Viewing Prepared Medical Report

From the CA Transactions > Clinician Access > Desktop Menu > Medical Reports, select View Prepared Medical Report.

The viewing Prepared Medical Report gets displayed.

  1. Enter the search criteria and click [Search] to view any specific Request ID's of ‘Prepared by Practitioner’ status or click [Search] to display all Request ID's in Prepared by Practitioner status.

    1. Requestor - Enter the Requestor or select the requestor using [?] function.

    2. Requestor Type – Select the requestor type from the drop-down list.

    3. Request Date [From – To] – Select the date range of request placed.

    4. Due Days <= - Enter the due days in the text box.

    Patient Details

    1. Patient ID - Enter the patient ID in alphanumeric format of 10 characters maximum length or select the patient ID by clicking [?] function.

    2. Gender – Select the gender from the drop-down list. The list values include Male, Female and Unknown.

    3. Other Alt Type - Select the Other Alt Type from the list of valid values. The listed values are Birth Certificate, Passport, and Mother's PMI, etc.

    4. Other Alt ID - The corresponding Other Alt ID is displayed when an Other Alt Type from the list is selected.  

    5. Enter the values for the fields 'MyPR', 'MyKAS', 'Passport' and 'Pension Card' which are set in the MP Parameter under 'Alternate ID Checks' in the 'Master Patient Index' module.

  2. Click Request ID hyperlink, the system displays ‘Receive Medical Report’ dialog.

  3. Click the [View Medical Report] button, the Medical Report dialog gets displayed.

  4. You can only view/print the already recorded medical report. You cannot record or do any other transactions.
