Patient Profile

The Patient Profile screen gets displayed on clicking the Profile icon  in the Patient Chart. The contents of this screen are enabled from the Patient Profile Content function in CA Code Setup. The description of this field is also defined in the same function in CA Code Setup.

  1. Select the check box against the required Profile Content.

  2. Click [Show] to view the details.

  3. Click [Print] to print the details or click [Close] to exit from the screen.

You will be prompted to enter a password to view the high risk conditions recorded for the patient. The password is defined in the Medical Records module.
