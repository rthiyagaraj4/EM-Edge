Coder Pending Records

From this function Coders can View patient records assigned to them.

This function allows the Coders to view the patient records assigned to them. The hyperlink is visible only if there are patient records assigned to them.

View Coder Pending Records

From the CA Transactions > Clinician Access > Summary Menu > Coders Review > select the Coder Pending Records function.

The Coders Pending Records screen gets displayed.

  1. By default, the patient details for the last 7 days get displayed.

  2. Enter the search criteria if required.

    1. Patient ID – Enter the required patient ID or select it from the lookup.

    2. Period – By default displays the date of last 7 days. You can change the date range if required

    3. Click [Search], the patient records matching the criteria appear.

  3. Click Patient ID hyperlink to invoke the Record Clinical Notes function. (To open the Record Clinical Notes function by default, you need to define it as the first function in ‘CA Menu” assigned to the coder).
