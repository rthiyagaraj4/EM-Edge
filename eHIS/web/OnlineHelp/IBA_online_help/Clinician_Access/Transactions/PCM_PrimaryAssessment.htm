Primary Assessment

From this function you can Record and View Primary Assessment details of an Emergency patient.  

Recording Primary Assessment is the first step in an Emergency care. Primary Assessment function is used to record the patient’s condition when brought into an emergency location. Physical condition of the patient, diagnosis details, medication details are all recorded. Primary Assessment also involves evaluating airway, breathing, circulation and nervous system disability of the Patient. Primary Assessment is recorded to identify and treat all serious injuries.

Recording Primary Assessment

From the CA Transactions > Clinician Access > Desktop Menu > select required Emergency patient > Patient Chart Menu > ED Assessments, select Primary Assessment function.

The Record Primary Assessment screen gets displayed.
