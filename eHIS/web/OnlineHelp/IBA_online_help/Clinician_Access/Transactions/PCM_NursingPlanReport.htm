Nursing Plan Report (V)

From this function you can View the nursing plan report.

The Nursing Plan report allows to view the assigned care plan to a patient and status of it. You need to configure the report in Construct CA Menu function so that it appears under the Patient Chart.

Nursing Plan Report icon is introduced in Tool bar to invoke the Nursing Plan Report function. On mouse over, the system displays the tip ‘Nursing Plan Report’.

Viewing Nursing Plan Report

From the CA Transactions, Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Chart History, select Nursing Plan Report (V).

The Nursing Plan Report screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to view the records matching the search criteria or [Reset] to clear the entered data.

  3. Click [Print] to generate the report.
