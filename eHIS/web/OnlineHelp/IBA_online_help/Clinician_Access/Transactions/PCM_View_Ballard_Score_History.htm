View Ballard Score History 

From this function you can View the History of Ballard Score for selected patient for any encounter.

This function allows to view the history of Ballard score for selected patient for any encounter.  The system displays recorded details of all practitioners who have recorded Ballard score for that selected encounter.

Click    icon in the patient chart to invoke the View Ballard Score History screen.

Viewing Ballard Score History

From the CA Transactions > Clinician Access > Desktop Menu >  select required patient > View Chart Summary > View History > View Ballard Score History.

The View Ballard Score History screen gets displayed.

  1. By default, the ballard history details based on Encounter ID and Period gets displayed.

    • Period From - By default, the date populates for the one month. 'To Date' is current date and 'From Date' is previous month date same as current date.

    • Encounter ID - By default, the current encounter ID of the selected patient gets displayed.

  1. If required, you can change the 'Period From' and 'Encounter ID' fields and click [Search] to view data accordingly.

  2. The Ballard Score History details include Practitioner Name, Recorded Date/Time and Encounter ID. Click ‘Recorded Date/Time’ hyperlink to view the already recorded ballard score details with Score and Reference Score.

  3. You can only view the recorded Ballard Score detail and cannot edit it.

  4. Click [Print] button to print the ballard score.
