Add Emotional/Social/Interpersonal Relationships

This screen is used to add emotional, social and interpersonal relationship of the family members to the index patient. Standard Emotional/Social/Interpersonal relationships symbols are used in creating a genogram.

  1. Select the option buttons in the ‘Emotional Relationships From and To’ columns to record relationship between two family members.

  2. Select the Relationship Type from the drop-down list. This is a mandatory field. The listed values are Emotional, Interpersonal and Social.

  3. Select the Relationship from the drop-down list. This is a mandatory field.

  4. Enter remarks if required.

  5. Click [Add]. The relationship detail gets added in the Relationship grid.

  6. If you want to remove the relationship, select the check box against the required relationship and click [Remove]. Once the relationships are added/removed, the genogram gets refreshed.
