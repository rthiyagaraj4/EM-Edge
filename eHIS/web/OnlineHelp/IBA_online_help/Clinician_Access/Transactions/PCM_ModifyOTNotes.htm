Modify OT Notes

From this function you can Modify OT Notes once it is marked as completed for any OT transaction

The Modify OT Notes function is used to modify the OT Notes once it is marked as completed for any OT transaction.

Modifying OT Notes

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Documentation, select Modify OT Notes

The Modify OT Notes screen gets displayed.

  1. By default, the current week date range is displayed in the Duration field.

  2. Select the Note Type from drop-down list. The Note Types applicable for OT surgeon Notes are listed in the drop-down list. By default, the value ‘Select’ is displayed.

  3. Click [Search] to display Operation Date, Note Type, Specialty, Operation Number, Performed By, Authorized By and Status or [Clear] to clear the entered data.

  4. Click Note Type to invoke Surgeon Notes dialog. The OT surgeon Notes completed in OT Slate is displayed on this screen with Authorised status. When you click on ‘Note Type’ hyperlink, you are directed to OT Surgeon Notes screen and allow you to modify the Notes after providing required Remarks for Modification.

  5. Click [Modify] to display a new dialog with message ‘Confirm Modify?’. If you click [OK], the system confirms the modification made in the OT Surgeon Note.

  6. Click [Apply] once the changes are made in the Surgeon Notes. After the Notes modification is performed, the OT Surgeon Notes status is changed to Modified. You can view the Status history by clicking on ‘Status’ hyperlink in Modify OT Notes screen.

  7. Click [Cancel] to exit from the screen.
