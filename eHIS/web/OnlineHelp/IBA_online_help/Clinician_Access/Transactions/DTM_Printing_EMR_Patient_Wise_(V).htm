Printing EMR Patient Wise (V)

From this function you can View and Print the complete electronic medical records of a patient encounter wise.

This function allows you to view and print the complete electronic medical records of a required patient encounter wise.  Printing of EMR patient wise is applicable for both open and closed encounters.

The system does not restrict for any patient to print the EMR whether it is a MLC, deceased patient.

Printing EMR Patient Wise

From the CA Transactions > Clinician Access > Desktop Menu > Schedules, select Printing EMR Patient Wise (V)

The Printing EMR Patient Wise (V) screen gets displayed.

  1. Enter the patient ID or select the required patient ID using [?].

  2. Date [From - To] - By default the system displays 6 months duration from current date i.e., to date. You can change the date if required. This field is mandatory.

  3. Click [Search]. The patient records matching the criteria appear.  

  4. The records are displayed in descending order being the latest encounter at the top and old at the bottom. The system considers both open and closed encounters of selected patient while printing the EMR patient wise. Even deceased patients EMR are also included.

  5. Select the option button against the required Encounter ID. The system displays below Sections to select for printing.

• Blood transfusion/Transfer

• Clinical Note

• Department Diagnosis

• Dietary

• Document Imaging

• Free Documentation

• Laboratory

• Monitoring

• Patient care

• Problem/Diagnosis

• Radiology Imaging

• Surgery / procedures

  1. Select the required Section.

  2. Click [Print] to print the content part for the selected Sections for the selected patient encounter.

  3. Click [View] to view the content that was selected.
