Enter the following details

  1. Patient ID – Enter the Patient ID in the format as specified in Master Patient Index module or select the required Patient ID using [?] function. This is a mandatory field.

  2. Encounter ID – Enter the Encounter ID of the patient in numeric format of 12 digits maximum length or select the Encounter ID from the list.

  3. Clinic / Nursing Unit – Select the Clinic / Nursing Unit using [?] function.

 

If you click the Search button without selecting the Patient ID in the Capturing Pregnancy Details screen, the system prompts a message ‘Please select the Patient ID’. If you click the Record button without entering the Last Menstrual Period date in the Capturing Pregnancy Details screen, the system prompts a message ‘Please enter the Last Menstrual Period’. If you click the Record button without entering the Expected Delivery Date in the Capturing Pregnancy Details screen, the system prompts a message ‘Please enter the Expected Delivery Date’. If you click the Record button without selecting the practitioner name in the Capturing Pregnancy Details screen, the system prompts a message ‘Please select the practitioner’.
