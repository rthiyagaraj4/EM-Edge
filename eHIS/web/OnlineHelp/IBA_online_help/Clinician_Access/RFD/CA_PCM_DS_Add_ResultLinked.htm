Result Linked

  1. Click the Section or Subsection defined with type ‘Result Linked’ from the ‘Notes View’ part of the ‘Record Discharge Summary’ screen.

  2. ‘Link Result Documents’ screen is displayed. Result Documents are linked to the Note Type using this screen. The fields of the screen displayed depend on the ‘Result Category’ associated to the Result Linked Section while creating the Section in the Section function.

  3. Enter the required search criteria to search for the required Result Documents. Select the required Results to be associated to the Note and click [Select] to display the result in the editor screen.

  4. The editor provides the standard features of paragraph setting, font selection, font size selection, cut, copy, paste, bold, italic, underline, undo, redo, foreground color, background color, align, justify, bulleted list, numbered list and clear page option. The tool tip while moving the mouse over the specific icon or the button highlights each of the functionality within the editor.

 

Additional buttons available on the screen are

Canned Text

Canned texts are predefined text blocks used while recording notes that are of free text types. The button is activated based on the type of Section selected and on the type of Medical Service selected. Using this function, a practitioner can copy a predefined text block and later edit it to suit the requirement of the note. The canned text listed is set in the ‘Text Block’ master function in Clinician Access module.

A canned is made specific to a practitioner, service, note, section or a combination of all these factors.

  1. Click [Canned Text] and the Canned Text screen is displayed.

  2. Select – Select the check box against the appropriate text block that needs to be inserted into the editor.

  3. Description - The content of the Text Block is displayed. This field is hyperlinked if the Text Block contains more contents.

  4. Block ID – The Block ID of the Text Block is displayed as defined in the Text Block function of CA Code Setup.

  5. Click [Select] to copy the text to the editor screen. Click [Cancel] to exit from the screen.

  6. Make necessary changes to the text copied to the editor screen. Click [Record] to save a draft of the Note.

Chief Complaints

If a Chief Complaint defined in CA Codes Setup is associated to any Section of the selected Note Type through ‘Section for Note Type’ function of CA Codes Setup, an icon (c) is displayed.

  1. Click [Chief Complaints] to display the Chief Complaints screen.

  2. Scope - Select Scope as Current Encounter, Across Encounter or Open List.

  3. Search Text - On selecting Scope as Open List, entering the text for searching the chief complaint becomes mandatory.

  4. Search Criteria – Select the Search Criteria as Starts With, Ends With or Contains from the list box.

  5. Click [Search] to display Chief Complaints. Select the check box against the required Chief Complaint and click [Select] to copy the Chief Complaint to the editor screen. Click [Cancel] to exit from the Chief Complaints screen.

Link Doc

  1. The [Link Doc] is enabled only on selecting sections that are result linked.

  2. Click [Link Doc] to display the Link Result Documents screen.

  3. Result Documents are linked to the Note Type using this screen. The fields of the screen displayed depend on the ‘Result Category’ associated to the Result Linked Section while creating the Section in the Section function.

  4. Enter the required search criteria to search for the required Result Documents. Select the required Results to be associated to the Note and click [Select] to display the result in the editor screen.

Upload

  1. Click [Upload] to upload files/documents for the selected Note Type.

  2. File Reference – Browse to the path where the files are stored for uploading.

  3. Remarks – Enter remarks if any.

  4. Click [View] to view the file to be uploaded. Click [Record] to upload the file for the Note Type.

Record

The Note created is partially saved by using [Record] after which the Note is still available for editing or deletion.

On entering the required details for the Note Type, click [Record] to save the notes recorded. The recorded Discharge Summary Notes are displayed. Select the Section hyperlink under Notes View on the left hand side of the screen to edit the contents recorded for the Note Type.

Sign

Once a Note is finalized, it is confirmed by using [Sign] after which the Note is available only for modification, forwarding or marking error and cannot be deleted.

  1. Click on [Sign] to sign a Note.

  2. A message is displayed indicating confirmation of signing the Note.

  3. Click [OK] to confirm signing or click [Cancel] to cancel signing.
