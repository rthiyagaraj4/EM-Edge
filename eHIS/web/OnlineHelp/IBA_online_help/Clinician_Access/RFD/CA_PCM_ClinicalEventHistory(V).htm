Enter the following details:

 

  1. Period From-To – Enter the date range in the format dd/mm/yyyy to search for clinical events within the displayed date range.

  2. View Mode - Flow Sheet – Select the option, if the display of events has to be in the form of a flow sheet. The Flow sheet displays events for the period specified. If the period is not specified, the display includes events for the encounters.

  3. View Mode - Tree - Select the option to display events in the tree view.

  4. View Mode – Group – Select the option to display events in group view. This view is applicable only for Event Type as ‘Laboratory’.

  5. Flow Sheets – This list is displayed only when the Flow Sheet radio button is selected. The list values depend on the flow sheets designed by the user in the setup. According to the setup, the parameters included in the flow sheet are different.

  6. View by – This field is displayed only when the Tree radio button is selected. Select either Event or Date radio button to view records based on the selected option.

  7. View – This field is displayed only when the Group radio button is selected. Two option buttons Panel and Individual are available. By default, the Panel option is selected.

  8. Encounter No – The current encounter number of the patient is displayed. View the recorded clinical events for other encounter numbers by specifying the same in alphanumeric format.

  9. Normalcy Indication – Select a value from the list. The listed values are Color, Indicator and Both.

  10. Display By Event - Select this check box to view the clinical details grouped by event. Thereby, you can view all similar details together. This check box appears only in flowsheet view. By default, it is not selected.

[More Criteria] – This action button facilitates the user to narrow down the search results by entering more search criteria. The button is clicked to reveal other criterion. On clicking [More Criteria], the additional criteria are displayed.

The additional criterion is hidden by clicking [Less Criteria].

  1. Patient Class – Select a Patient Class from the list. The listed values are Inpatient, Daycare, Outpatient, Emergency and External.

  2. History Type – Select the History Type from the list. The listed values are Blood Transfusion, Clinical Note, Dept Diagnostics, Dietary, Free Documentation, Laboratory, Monitoring, Patient Care, Problem/Diagnosis, Radiology/Imaging, Surgery/Procedure and Treatment.

  3. Facility – Select the Facility from the list.

  4. Event Class - Select the Event Class from the list. The listed values are Consent, Procedures and Refusal.

  5. Event Group – Select the Event Group from [?] function.

  6. Event Item – Select the Event Item from [?] function.

  7. Abnormal Only – Select the check box to display only abnormal events.

  8. Display Order – Select the Display Order as Ascending or Descending from the list.

  9. Specialty - Select the required speciality from the [?] function.

  10. Attending Practitioner – Select the required attending practitioner name from [?] function. If ‘Specialty’ field is selected, the attending practitioner list related to the selected specialty gets displayed.
