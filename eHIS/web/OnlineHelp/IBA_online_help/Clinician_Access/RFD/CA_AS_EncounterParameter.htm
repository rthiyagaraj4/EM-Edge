Enter the following details to create Encounter Parameter:

 

  1. Backward Visits – Specify the number of days in numerical format of three digits maximum length. Based on the number of days specified, the search for Patient Visits is backdated accordingly in ‘Clinician Desktop’ Patient Search. This field is mandatory.

  2. Backward Discharge – Specify the number of days in numerical format of three digits maximum length. Based on the number of days specified, the search for Discharged Patients is backdated accordingly in ‘Clinician Desktop’ Patient Search.

  3. Establish Relationship – Select ‘Establish Relationship’ to enable the relationship functionality in the Clinician Access module. Leaving the field cleared allows the practitioners to select a patient without having to establish a relationship.

  4. Remarks mandatory to establish without encounter Relationships - Select the check box to make the Remarks field as mandatory in ‘Assign Relationship’ screen.  By default, the check box is cleared. Only if this check box is selected, 'Reason' and 'Reason Text' fields get displayed in the 'Assign Relationship' dialog. This is applicable only if the 'Establish Relationship' check box is selected.

  5. Enable Assign & Confirm in Specialty Consult – Select the check box if you want to assign and confirm the bedside referral request in Specialty Consult screen without going to Self - Consult screen. This check box is not selected by default.

If you select the check box, a popup message with following options appears on clicking the Patient Name hyperlink of the bedside referral request for specialty in the Specialty Consult screen:

  • Assign and Confirm option - When you select the Assign and Confirm option, the bed referral request gets confirmed at the Specialty level and the confirmed patients are listed in the Referral tab in Patient List by Location screen to all practitioners in that specialty.

  • Assign option - If you select the Assign option, the bedside referral request initiated for the Specialty gets transferred to the Self - Consult page. The patient is visible only to the assigned Practitioner in the Referral tab in Patient List by Location screen.
