Result Linked

  1. Click the Section or Subsection defined with type ‘Result Linked’ from the ‘Notes View’ part of the ‘Record Clinical Notes’ screen.

  2. ‘Link Result Documents’ screen is displayed. Result Documents are linked to the Note Type using this screen. The fields of the screen displayed depend on the ‘Result Category’ associated to the Result Linked Section while creating the Section in the Section function.

  3. Enter the required search criteria to search for the required Result Documents. Select the required Results to be associated to the Note and click [Select] to display the result in the editor screen.

  4. The editor provides the standard features of paragraph setting, font selection, font size selection, cut, copy, paste, bold, italic, underline, undo, redo, foreground color, background color, align, justify, bulleted list, numbered list and clear page option. The tool tip while moving the mouse over the specific icon or the button highlights each of the functionality within the editor.

 

Additional buttons available on the screen are

Canned Text

Canned texts are predefined text blocks used while recording clinical notes that are of free text types. The button is activated based on the type of Section selected and on the type of Medical Service selected. Using this function, a practitioner can copy a predefined text block and later edit it to suit the requirement of the note. The canned text listed is set in the ‘Text Block’ master function in Clinician Access module.

A canned is made specific to a practitioner, service, note, section or a combination of all these factors.

  1. Click [Canned Text] and the Canned Text screen is displayed.

  2. Select – Select the check box against the appropriate text block that needs to be inserted into the editor.

  3. Description - The content of the Text Block is displayed. This field is hyperlinked if the Text Block contains more contents.

  4. Block ID – The Block ID of the Text Block is displayed as defined in the Text Block function of CA Code Setup.

  5. Click [Select] to copy the text to the editor screen. Click [Cancel] to exit from the screen.

  6. Make necessary changes to the text copied to the editor screen. Click [Record] to save a draft of the Clinical Note.

Chief Complaints

If a Chief Complaint defined in CA Codes Setup is associated to any Section of the selected Note Type through ‘Section for Note Type’ function of CA Codes Setup, an icon (c) is displayed.

  1. Click [Chief Complaints] to display the Chief Complaints screen.

  2. Scope - Select Scope as Current Encounter, Across Encounter or Open List.

  3. Search Text - On selecting Scope as Open List, entering the text for searching the chief complaint becomes mandatory.

  4. Search Criteria – Select the Search Criteria as Starts With, Ends With or Contains from the list box.

  5. Click [Search] to display Chief Complaints. Select the check box against the required Chief Complaint and click [Select] to copy the Chief Complaint to the editor screen. Click [Cancel] to exit from the Chief Complaints screen.

Link Doc

  1. The [Link Doc] is enabled only on selecting sections that are result linked.

  2. Click [Rx] icon appearing next to the Link Doc button to invoke the Record Diagnosis dialog to record the diagnosis and  then the recorded diagnosis information will be available in Link Doc for long text component.  

  3. Click [Link Doc] to display the Link Result Documents screen.

  4. Result Documents are linked to the Note Type using this screen. The fields of the screen displayed depend on the ‘Result Category’ associated to the Result Linked Section while creating the Section in the Section function.

  5. Enter the required search criteria to search for the required Result Documents. Select the required Results to be associated to the Note and click [Select] to display the result in the editor screen. If Format is selected as ‘Text’, the results get added in text format in the editor screen. Else, if 'Table' format is selected, the results gets added in table format in the editor screen. The default Format option appearing either as ‘Text’ or ‘Table’ depends on ‘Select Text Format by Default for Linked Sections’ check box selection in Note Parameter screen.

  6. If the Result Category is selected as Newborn while creating a section and is associated to the Note Type, Linked Result Document for selecting newborn details is displayed.

    If mother patient is selected, select the required Encounter to display the details.

    The details displayed are Baby Details (Demographic Details, Clinical Details, Vaccination Details and Apgar Score Details), Mother Details and Father Details. Select the check box against the required details or select the check box against the header to select all the details, by default.

    If newborn is selected, the encounter of the newborn is displayed and disabled. The details displayed are Baby Details (Demographic Details, Clinical Details, Vaccination Details and Apgar Score Details), Mother Details and Father Details. Select the check box against the required details or select the check box against the header to select all the details, by default.

    Select the Format radio button as Text or Table to display details in the required format. Click [Select] to display the selected details in the editor. Click [Clear] to clear the selected items or click [Cancel] to exit from the screen.

Upload

  1. Click [Upload] to upload files/documents for the selected Note Type.

  2. File Reference – Browse to the path where the files are stored for uploading.

  3. Remarks – Enter remarks if any.

  4. Select - Clear the check box to dissociate the images that are linked. The check box is selected, by default. You can dissociate the image only if the note is not signed.

  5. Click [View] to view the images that are uploaded. Click [Record] to upload the file for the Note Type.

You can upload multiple image based on the limit specified in 'Note Type Parameter' function.

Record

The Note created is partially saved by using [Record] after which the Note is still available for editing or deletion.

On entering the required details for the Note Type, click [Record] to save the clinical notes recorded. The recorded Clinical Notes are displayed. Select the Section hyperlink under Notes View on the left hand side of the screen to edit the contents recorded for the Note Type.

Sign

Once a Note is finalized, it is confirmed by using [Sign] after which the Note is available only for modification, forwarding or marking error and cannot be deleted.

  1. Click on [Sign] to sign a Note.

  2. A message is displayed indicating confirmation of signing the Note.

  3. Click [OK] to confirm signing or click [Cancel] to cancel signing.
