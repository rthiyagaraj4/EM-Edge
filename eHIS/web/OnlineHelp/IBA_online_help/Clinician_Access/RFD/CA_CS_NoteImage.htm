Enter the following details to create a Note Image List record:

 

  1. Image Reference ID - Enter the Image Reference ID in alphanumeric format of four characters maximum length. This ID is to identify the Image. The entered ID must be unique and cannot be changed after creation. This field is mandatory.

  2. Image Description - Enter the Image Description in alphanumeric format of 18 characters maximum length. The description names the part and position of the body part of which the image is being added.

  3. Mode – Select the radio button against ‘Upload’ or ‘Add’.

  4. Upload - Upload is when you do not know the file location or you need to add it from the local system or from the network linked system.

  5. Add - Add is when you know the path of the file or is present in the image repository folder. Adding an entry for the image into the table, which enables adding an image into the server or repository when needed.

  6. Image FileName - Enter the image filename in alphanumeric format of 40 characters maximum length when the Mode is selected as ‘Add’. When the Mode is selected as ‘Upload’, this field is provided with [Browse] to browse the image from the system located in the image repository folder. This field is mandatory.

  7. Enabled – Select the ‘Enabled’ check box to enable the ‘Note Image List’ in transactions. If ‘Enabled’ is not selected, the ‘Note Image List’ is not available to clinician to add an image.
