Enter the following details to create a new Note Group record:

 

  1. ID - Enter the code in alphanumeric format of four characters maximum length. This code is to identify the Note Group. The entered code must be unique and cannot be changed after creation. This field is mandatory.

  2. Description – Enter the description for the Note Group in alphanumeric format of 30 characters maximum length. This field is mandatory

  3. Applicable Task – Select an Applicable Task from the list of values provided. This field is mandatory. The Applicable Task values are:

  4. OT Notes

  5. Patient Medical Report

  6. Result Reporting

  7. Print Clinical Data

  8. Email Clinical Events

  9. Medical Certificate

  10. Record Discharge Summary

  11. Referral Letter

  12. Trauma Assessment

  13. Primary Assessment

  14. Record Clinical Notes

  15. Assessment

  16. Notifiable Form

  17. Prepare Medical Report

  18. Prepare Medical Report by Board

  19. Mother Consultation

  20. Oral Health Notes

  1. Num of Note Per Encounter - Enter the number denoting the number of Notes that can be created for every Encounter ID. It restricts the Practitioner to make only the specified number of records for an encounter of the patient. The number has to be in the numeric format of four digits maximum length. This field is mandatory.

  2. Precondition for Recording - Select from the list of values the precondition to be applied for the selected Applicable Task. The values in the list are

  3. Discharge Advice

  4. Deceased

  1. Report Header - Enter the header for the Note group in alphanumeric format of 30 characters maximum length. The Description of the Note Group is displayed as the Report Header which can be modified. This field is mandatory.

  2. Transcription Applicable - This check box is displayed only if Applicable Task is selected as Record Discharge Summary, Referral Letter and Record Clinical Notes. Select the check box if transcription is applicable for the selected Note Group based on the Applicable Task.

  3. Cut off Period to allow Transcription - This field is enabled only is Transcription Applicable check box is selected. Enter the cut off period in days to allow transcription from visit registration or discharge. This is a mandatory field.

  4. Enabled - Select the check box to enable the record and clear the check box to disable the record.
