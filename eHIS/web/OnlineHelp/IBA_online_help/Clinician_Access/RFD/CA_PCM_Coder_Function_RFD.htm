Enter the following details:

 

  1. Patient Class – Select the patient class from the drop-down list. The list values include Inpatient, Outpatient, External, Daycare, Emergency, All. By default, the value 'All' is displayed.

  2. Encounter ID – Enter the encounter ID of the patient in the text box.

  3. Patient ID – Enter the patient ID or select it from the lookup.

  4. From and To Date – By default, the current date gets displayed. You can search for date range of 31 days. This field is mandatory.

  5. Coding Status – Select the required coding status from the drop-down list. The list values include New, Doctor Query, Supervisor Query, Hold, Partially Completed, Completed, All, and Supervisor Clearance Completed. By default, the value ‘All’ gets displayed.

  6. View List – Select the required view list value i.e., All or Self Related. By default, the value ‘All’ gets displayed.
