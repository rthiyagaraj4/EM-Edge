Enter the following details to create a Chart Setup record:

 

  1. Chart Code – Enter the Chart Code in alphanumeric format of four characters maximum length. This code is to identify the Chart. The entered code should be unique and cannot be changed after creation. This field is mandatory.

  2. Long Description – Enter a Long Description for the chart in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description – Enter a Short Description for the chart in alphanumeric format of 20 characters maximum length. This field is mandatory.

  4. Chart Type – Select the required Chart Type from the list. The values are ‘Generic Chart’ and ‘Growth Chart’.

  5. Summary Required – Select the Summary Required check box to enable viewing the summary of all the observed discrete records of a patient for that day.

  6. Configuration – Select the required Configuration setup from the list. The values are ‘Variable’ and ‘Fixed’.

  7. Variable – On selecting ‘Variable’, you can modify the scope of the components used for recording for a particular patient at a particular time.

  8. Fixed - On selecting ‘Fixed’, you cannot modify the scope of the components used for recording unless changed in the master Chart Components.

  1. Summary By Day – Select the Summary by Day check box to enable viewing the summary of the discrete recording of a patient per chart per day.

  2. Summary By Shift – Select the Summary by Shift check box to enable viewing the summary of the discrete recording of a patient per chart per shift.

  3. Day Summary Period From – Enter the ‘From’ date in hh:mm format. By default, the date is mapped to 00:00 – 23:59, that is one day.

  4. Past Records Scope – Enter the Scope as the number of days to view the summary in numeric format of two digits maximum length.  

  5. Enabled – Select the check box to enable the record and clear the check box to disable the record.

  6. Shift Summary Required – Select the check box to define shift summary details so that it appears in Record Chart screen. This check box appears only if the ‘Summary By Day’ check box is selected. When you select this check box, the fields ‘No of Shifts’, ‘Shift Duration’, ‘Shift 1’, ‘Shift 2’ and ‘Shift 3’ appear.

  7. No of Shifts – Enter the number of shift required. You can enter only up to 3 shifts and not less than one . This is a mandatory field.

  8. Shift Duration – Enter the duration of each shift. This is a mandatory field.

  9. AM Shift, PM Shift, ON Shift – The number of shifts and the shift timings appear automatically based on the defined ‘No of Shifts’ and ‘Shift Duration’ fields.

  10. Applicable for Interfaced Vital Signs Data - Select the check box to display the selected charts in both table and graph view in Interfaced Vital Signs Data screen under Task List. This check box is applicable only for generic charts and not for growth charts
