Enter the following details:

 

  1. Link Type - Select the Link Type as ‘Filter’ or ‘Global’.

  2. Function - Select the Function from the Function e Dialog that appears, if the Link Type is selected as Filter. This is a mandatory field. If the Link Type is selected as Global, this field is disabled.

  3. Responsibility - Select the Responsibility from the list. The Responsibilities are defined in the System Manager module. This field is mandatory.

  4. Click [Search] to display the Global/Filter Quick Links based on the Link Type selected. Select the check box against the required Quick Link to associate to the Responsibility. Select - Select the check box against the required Quick Link to associate to the Responsibility.

  5. Default – This check box is displayed only if you have selected the Link Type as ‘Filter’ and it is enabled when you select the Quick Link. Select the Default check box against the required Quick Link to make the quick link as default.
