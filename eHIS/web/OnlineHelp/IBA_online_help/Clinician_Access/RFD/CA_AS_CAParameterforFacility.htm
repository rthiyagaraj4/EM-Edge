Enter the following details to define CA Parameter for Facility:

 

  1. Facility - Enter the login facility name in the Facility field.

  2. Logo To Be Considered for Clinical Notes - Select either 'Site' or 'Login Facility' as required.

  3. By default, the option Site is selected, the system considers the logo defined in Site screen while printing the Clinical notes from Print Preview and Print in flowsheet or tree view mode.

  4. If you select the 'Login Facility logo' option, the system considers logo defined in Facility screen in System Manager module. The logo gets displayed in Print Preview, Print of Clinical notes.

  1. Default Time Difference in Interfaced Vital Signs Data  – Enter the time difference between From date and To Date time field in Filter window invoked on clicking Filter hyperlink in Interfaced Vital Signs Data sub menu in Task List screen. This field is mandatory. You can enter minimum of 1 to maximum 999 minutes. By default, it is set as 15 minutes.  This is applicable only for users in MMS.

  2. Default Logged in Practitioner Primary Specialty in Patient List by Location – Select the check box to default the primary speciality of login practitioner in Specialty search lookup in ‘Patient List By Location’ transaction. By default, the check box will not be selected.  

  3. Enable Primary and Secondary Procedure Codes Classification in Record Procedures function – Select the check box to record primary and secondary procedure codes in ‘Classification Type’ field in Record Procedures function. By default, the check box is cleared.

  4. Hide Patients Who are marked as Restricted for Unauthorized Practitioners – Select the check box to hide restricted patient clinical data for unauthorized practitioners in both CA and PAS module. By default, the check box is cleared. For practitioners authorized in ‘Access Rights to View Clinical Data for Restricted Patients’ function, even if the check box is selected, restricted patients clinical data will displayed.
