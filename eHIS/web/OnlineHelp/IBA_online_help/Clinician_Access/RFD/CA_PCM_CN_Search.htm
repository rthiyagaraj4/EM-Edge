Search

This function helps in searching past Clinical Notes of the patient and the required contents of the Note Type can be viewed by clicking on the required Note Type hyperlink.

  1. By default the Notes recorded are displayed for the date range specified in the Search Tab of the Record Clinical Notes screen. Enter the query criteria and click [Search] to display specific Notes. <Related Field Description>

  2. The following details of the Notes are displayed. Date and Time of recording the Note, Last Modified Date and Time, Note Type which is hyperlinked, Reference Number of the document, any attachment uploaded which can be clicked to view, Medical/Ancillary Service, Title, Practitioner who recorded the Note, Practitioner who authorized the Note and Status of the Note.

  3. Click the Note Type hyperlink to view the contents of the Note recorded. The Clinical Note Details screen is displayed with details of the Note.

  4. Click the Status hyperlink to view the clinical note status details in the ‘Clinical Note Status Details’ dialog. The status details include Action Date/Time, Action By, Action Description, Action Remarks and Pre-Image icon. This icon appears only for the second time recorded note. Click the icon to view the previous/first time recorded note details at result area with scroll bar.

 

The Operations that can be performed on a Document is based on the 'User Role' and 'Status' of the Document.

The different User Roles are Transcriber, Author but without Authorization Facility, Author with Authorization Facility, Reviewer Privilege for Verifying and Authorization, and View Only Privilege.

The various Statuses are In Progress, Transcribed, Unauthorized, Authorized, Modified and In Error.

Delete

Notes recorded could have mistakes or would have been recorded wrongly. Delete is performed for Note Types with status as ‘In Progress’.

  1. Click on [Delete] to delete a Note.

  2. A message is displayed indicating the Note will be closed.

  3. Click [OK] to delete the Note or click [Cancel] to cancel deleting the Note.

Update

The Notes that need a constant update for periodic recording can be under this category.

  1. Click on [Update] to update a Note.

  2. Click the required Section hyperlink under Notes View to update the details.

  3. Update the required details.

  4. Click [Record] to save the update.

Sign

This function facilitates signing the selected Note after confirmation.  

  1. Click on [Sign] to sign a Note.

  2. A message is displayed indicating confirmation of signing the Note.

  3. Click [OK] to confirm signing or click [Cancel] to cancel signing.

Close

Click [Close] to exit from the Clinical Note Details screen.

Forward

This function facilitates forwarding documents to other care providers. This function helps in facilitating the practitioners in sharing clinical information with other practitioners selectively for obtaining the opinion on the patient condition.

  1. Click [Forward] to display the Forward Notes To screen.

  2. Select ‘Forward To’ radio button as ‘Practitioner’ to forward the note to a practitioner or select ‘Specialty’ to forward the note to a Specialty.

  3. If ‘Forward To’ is selected as Practitioner, enter the following details. If 'Forward To' is selected as Specialty, enter the following details.

    1. Specialty - Select the Specialty from the list or enter the name of the Specialty. As you enter the name of the Specialty, the list narrows down in the alphabetical order matching the alphabets entered in the list box with the name of the Specialty.

    2. Subject - Enter Subject in alphanumeric format of 200 characters maximum length. This is a mandatory field.

    1. Specialty – Select the Specialty from the list or enter the name of the Specialty. As you enter the name of the Specialty, the list narrows down in the alphabetical order matching the alphabets entered in the list box with the name of the Specialty.

    2. Practitioner - Select the Practitioner from the list or enter the name of the Practitioner. As you enter the name of the Practitioner, the list narrows down in the alphabetical order matching the alphabets entered in the list box with the name of the Practitioner.

    3. Subject - Enter Subject in alphanumeric format of 200 characters maximum length. This is a mandatory field.

  4. Click [OK] to save the record and a confirmation message is displayed or click [Cancel] to exit from the screen.

Reject

This function facilitates rejecting the clinical note.

  1. Click [Reject] to reject the Note. The Remarks dialog gets displayed.

  2. Enter the remarks for rejection in the text box. It is mandatory to record the rejected marks and click [OK].

  3. Once you reject a clinical note, the status appears as ‘Rejected’ in the Record Clinical Notes dialog.

Modify

This function facilitates modifying the contents of the Note, thus allowing corrections to be incorporated in the note. Modification allows either amending the original section or adding an addendum without amending the original section. This depends on the Note Type setting.

  2. Enter the Remarks. This field is mandatory.

  3. Click [OK] to save the record or [Cancel] to exit from the screen.

  4. On clicking [OK], the screen to select the Sections and modify the details recorded are displayed.
