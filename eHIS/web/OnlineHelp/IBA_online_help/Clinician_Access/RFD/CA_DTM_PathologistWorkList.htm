Enter the following details

  1. Lab. Dept - Select the lab department from the drop-down list. The drop-down values get displayed as per user’s logged in Facility mentioned in ‘Departmental Parameter for Facility’ setup.

  2. Registered Date – By default, displays the date range for 30 days. Select the required date by clicking the Date icon. This field is mandatory.  

  3. Specimen Number -  Enter a valid specimen no. of selected Lab. Departtment. If Specimen Number is entered, then Registered From/To Date is not mandatory i.e., either Registered Date or Specimen number is mandatory.

  4. Ordering Facility – By default, displays the logged in facility. Change the ordering facility from the drop-down list if required.

  5. Ordering Practitioner – Select the ordering practitioner from the lookup.

  6. Lab. Order Status – By default, the value ‘All’ is displayed. The list includes only Registered / Review statuses i.e., the system considers only the Order/Specimen Numbers which are in Registered or in Review (1st or 2nd Normal/2nd Abnormal review) status for assignment.

  7. Performing Facility – Select the performing facility from the lookup.

  8. Reporting Facility - If any assigned before (or) Blank

  9. Patient ID -  Enter or select the patient ID from the lookup.

  10. Assigned Pathologist - The drop-down lists the valid Lab user as per the ‘Authorized Users’ master and have access to the Lab. Dept. selected
