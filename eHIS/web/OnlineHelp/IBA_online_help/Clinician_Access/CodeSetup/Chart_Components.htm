Chart Components

From this function you can Create components for a Chart.

Chart Components function displays all the Sections associated to the Chart. You can associate discrete measures to the sections for chart and associate discrete measure panels defined with type ‘Chart Section’ in the Application Masters module for the chart.

Creating Chart Components

From the CA Code Setup, select Chart Components function.

The Chart Components screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Add New] to add a new component under the panel.

    Enter the search criteria and click [Search] on the Add New Row screen. Select the check box ‘Order Sequence No’ to display an additional text box beside the check box. Enter the order number.

    Click Discrete Measure Description hyperlink to view the newly added panel on the chart component screen for the selected chart under the particular panel.

  4. Click [Chart Component Add Panel] to add a new panel for the chart selected.

    Enter the search criteria and click [Search] on the Add New Panel screen. Select the check box ‘Display order sequence’ to display an additional text box beside the check box. Enter the order number.

    Click Panel Description hyperlink to view the newly added panel on the chart component screen for the selected chart.

  5. Click [Apply] to save the Chart Components created or [Reset] to clear the entered data.
