Special Chart Groups

From this function, you can  Enable the defined chart groups while viewing the graph in record chart functionality.

This function is used while recording Clinical Charts in the Clinician Access module.

Enabling Special Chart Groups

From the CA Code Setup, select Special Chart Groups function.

The Special Chart Groups screen gets displayed.

  1. Select the Effective Status check box against the Special Chart Group ID to enable or disable the chart groups to be part of custom view or special view in record chart functionality. By default, the check box is not selected.

  2. Click [Apply] to save the Special Chart groups or [Reset] to clear the entered data.
