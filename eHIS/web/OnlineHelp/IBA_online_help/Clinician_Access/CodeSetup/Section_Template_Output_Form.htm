Section Template Output Form

From this function you can Create Templates for a Section.

Section Template Output Form function helps in preparing templates for Sections. Sections created with the 'Configure Output' check box selected in the 'Section' function are available for creating template output form.

Creating Section Template Output Form

From CA Code Setup > select Section Template Output Form function.

The Section Template Output Form screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the required Section using [?] function. The Sections with 'Configure Output' check box selected in the Section function are displayed. This is a mandatory field.

  3. Select the required Patient values and the description is displayed in the editor. The values displayed are Patient ID, Patient Name, Registration Date, Gender, Date of Birth, Age, Facility Name, Residential Address, Marital Status, Nationality, National ID, RH Factor, Blood Group, Residential Area, Residential Town, Religion, Name Prefix and Region.

  4. Select the Encounter value and the description is displayed in the editor. The values displayed are Encounter ID, Current Location, Attending Practitioner, Att. Pract with Credential, Admission Date, Specialty, Patient Class, Room Number, Bed Number and Service.

  5. Select the required Component values from the list. These listed values are defined in the Section Template function.

  6. Click insert table icon to insert tables in the template. Specify the number of Rows and Columns and click [OK] to display the table in the text area.

  7. Click Print Preview icon to preview the section template output created.

  8. Click [Apply] to save the Section Template Output Form or [Reset] to clear the entered data.
