Generate Reminders

From this function you can Generate reminder letters to be dispatched to the customers.

Reminder Letter Process is to generate and dispatch Reminder Letters to the customers to remind them regarding the pending payment to be made. This function enables you to reject the bills when the payment is not made on time. This function is used to generate Reminder Letters for a range of Customers based on their outstanding amount. The Reminder Letters are generated on the Severity Levels. Each Severity Level is classified based on the number of days due from the four Cut-off dates set for the Reminder Letters and the Document date on which the Reminder Letters are generated. Severity Level 4 indicates that the bill amount is outstanding for a longer period and not cleared. Correspondingly the overdue days and its condition of payment are derived upon each Severity Level. The Report Order can be generated based on Customer Code or Alpha Code. You can modify or generate any number of Reminders before the generated Reminder Letters are dispatched to the Customer. Once the Reminder Letter is dispatched to the Customer, you can generate the next Reminder Letter within the days specified in the AR Parameter. After the Reminder Letter dispatch, you cannot modify or edit the Reminder Letter.

Generating a Reminder Letter

From the Transactions > Reminder Letter Process, double-click Generate Reminders function.

The Generate Reminders screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Process Run] to generate the reminders.

  3. Click [Yes] to confirm or [No] to cancel the process. On clicking [Yes], a message is displayed showing the number of Reminders generated by the system for the selected cut-off date. Click [OK] to accept the generated reminders.
