Report Settings

From this function you can Associate a Report Group to a module in a facility to generate various reports. You can Query to view the facility based report parameters defined.

The Enterprise Management HIS is packaged with a set of standard reports for each of its modules. Entries of all the reports with default characteristics are created as part of the installation process. You can associate report groups of a selected module to multiple facilities. You can also view the characteristics defined for the report ID’s using the Report Settings tab.  

Defining Report Settings

From the Admin, double-click Report Settings function.

The Report Settings screen gets displayed.

  1. Click [Execute Query] or press [F8] to query the existing records.

  2. The details are displayed under the following two tabs:  

Facility Based Report Parameters

Report Settings

  1. Click [Commit Form] or press [F10] to save the updated Facility Based Report Parameters or [Clear Record] to clear the entered or queried data.
