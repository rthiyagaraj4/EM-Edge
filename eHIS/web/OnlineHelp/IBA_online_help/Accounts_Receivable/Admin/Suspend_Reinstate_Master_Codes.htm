Suspend/Reinstate Masters Codes

From this function you can Suspend or Reinstate the active and inactive Masters Codes.

The Suspended/Reinstate Master Codes function is used to suspend the active master codes or re-activate the master codes, which are suspended. Reinstating to active status is subject to the process, if the records are not deleted. The suspended records cannot be accessed for transactions. The records are only suspended, but not deleted from the database. You can use the records for transactions on reinstating the suspended records. The following master codes can be suspended or re-activated:

• Customer Type

• Customer Group

• Payment Terms

• Reminder Letters

• Customer

Suspend/Reinstate Masters Codes

From the Admin, double-click Suspend/Reinstate Master Codes function.

The Suspend/Reinstate Master Codes screen gets displayed.

  1. Select either ‘Suspend’ or ‘Reinstate’. Select ‘Suspend’ to suspend a selected master code. Select ‘Reinstate’ to enable the suspended master codes.

  2. Click [List of Values] to select a Master Code. The records created are numbered in sequence.  

    • Select 1 to enter the Customer Type code in alphanumeric format of two characters maximum length.

    • Select 2 to enter the Customer Group code in alphanumeric format of two characters maximum length.

    • Select 3 to enter the Payment Terms code in alphanumeric format of two characters maximum length.

    • Select 4 to enter the Letter Type Code in alphanumeric format of three characters maximum length.

    • Select 5 to  enter the Customer Code in alphanumeric format of eight characters maximum length.

  3. Press [Tab] or [Enter] key to suspend or reinstate the records associated to the master code selected from the list. A screen gets displayed listing records created for the master code.

  4. Click [List of Values] to select a master code which has to be suspended or reinstated. You can select to either suspend or reinstate the master codes. The description for the selected master code gets displayed.

  5. Click [Close] to exit from the screen.

  6. Click [Commit Form] or press [F10] to save the records or [Clear Record] to clear the entered data.
