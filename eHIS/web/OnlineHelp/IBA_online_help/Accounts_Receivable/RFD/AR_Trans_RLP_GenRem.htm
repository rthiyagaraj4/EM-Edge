Enter the following details to generate Reminder Letter:

 

  1. Report Order - This is a radio button with options ‘Customer Code’ and ‘Alpha Code’. The Report Order is set to ‘Customer Code’, by default. You can change the Report Order to ‘Alpha Code’. On selecting ‘Customer Code’, the reminder letters generated are sorted based on the Customer Codes. On selecting ‘Alpha Code’, the reminder letters generated are sorted based on the Alpha Codes.

  2. Customer Code [From-To] – If the report order is set as ‘Customer Code’, entering of value into this field is allowed. Click [List of Values] to select a valid Customer Code. Both the values are optional. If ‘FROM’ value is not entered, the system assumes the lowest value for the ‘FROM’ and if ‘TO’ value is not entered, the system assumes the highest value for the ‘TO’ value. When both values are entered, ‘TO’ value must be greater than or equal to the ‘FROM’ value.

  3. Alpha Code [From-To] – If the report order is set as ‘Alpha Code’, entering of value into this field is allowed. Click [List of Values] to select a valid Alpha Code. Both the values are optional. If ‘FROM’ value is not entered, the system assumes the lowest value for the ‘FROM’ and if ‘TO’ value is not entered, the system assumes the highest value for the ‘TO’ value. When both values are entered, ‘TO’ value must be greater than or equal to the ‘FROM’ value.

  4. Outstanding Amt. [From-To] – Enter the Outstanding Amount in numeric format of nine digits maximum length. When both the values are entered, ‘TO’ value must be greater than or equal to the ‘FROM’ value. The Reminder Letters are generated for a range of Outstanding Amounts specified. This is a mandatory field.

  5. Document Cut-off Date – Click the Calendar function to select a valid Document Cut-off-date on which the Reminder Letter is generated. This is used to calculate the number of days due based on which the Severity Level can be put on the Reminder Letters for the Customer. The system date is displayed as Document Cut-off-date, by default. This is a mandatory field.

Cut-Off Date

  1. Severity Level (1)/(2)/(3)/(4) – Click the Calendar function to select a valid cut-off date. Once a date is set for the Severity Level 1, the system does not allow you to set an after date of Severity Level 1 for Severity Level 2. This applies for the other Severity Levels also. You can set the dates in the descending order from that set in the Severity Level. This is a mandatory field.
