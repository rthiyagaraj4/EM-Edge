Enter the following  details to generate an Adjustment Document:

  1. Document Type – Click [List of Values] to select the Document Type. The document codes defined with transaction type as ‘Adjustment’ in Accounts Receivable masters screen gets displayed. You can select the code from the lookup.

  2. Customer Code - Click [List of Values] to select the customer codes. The customers codes defined in the Customer screen under the Masters gets displayed.

  3. Past Month/Year - The month and year set in AR Parameter is displayed as Current Accounting Year. You can modify the date.

  4. Open Credit Upto - Select the date in the Open Credit Upto field using the calendar function. All the open credit notes within the selected date will be considered for credit adjustment based on the selected customer code and patient ID.

  5. Outstanding Invoices Upto - Select the date in the Outstanding Invoices Upto field using the calendar function.

  6. Patient ID - Click [List of Values] to select Patient ID.

  7. Automatic Adjustment on Open Credit - Select this check box for automatic adjustment of all the outstanding invoices and open credit notes for customer code and patient ID combination. If you do not select the ‘Automatic Adjustment on Open Credit’, you have to do adjustment  manually

  8. Click  [Adjustment Details]; an Adjustment Details screen gets invoked.

    The following  details are displayed:

     

    Invoice

     

    All the outstanding invoices for the selected customer code and patient ID for the date selected in ‘Outstanding Invoices upto’ field will be listed in the Invoice section.

     

    1. Doc Type – The document type of the invoice is displayed. You cannot modify the data.

    2. Doc Num - The document number of the invoice is displayed. You cannot modify the data.

    3. Outstanding Amt – The outstanding amount of the invoice is displayed. You cannot modify the data

    4. Episode – The episode type of the invoice is displayed. You cannot modify the data.

    5. Total Invoice Amt – The total amount of the selected invoices records is displayed.

    6. Select – Select this check box against the invoices to be adjusted.

     

    Credit Note

     

    All the open credit notes within the selected date in Open Credit Upto for the customer code and patient ID combination will be listed in the Credit Note section.

     

    1. Doc Type – The document type of the credit note is displayed. You cannot modify the data.

    2. Doc Num - The document number of the credit note is displayed. You cannot modify the data.

    3. Outstanding Amt – The outstanding amount of the credit note is displayed. You cannot modify the data

    4. Episode – The episode type of the credit note is displayed. You cannot modify the data.

    5. Total Invoice Amt – The total amount of the selected credit note records is displayed.

    6. Select – Select this check box against the credit notes to be adjusted.

  9. First, select the Credit Notes to be adjusted and then the Invoices by selecting the ‘Select’ check box against the records.

  10. Click [Close]; the Adjustment Details window gets closed.

  11. Click [Save]; the selected credit note amount will be adjusted towards the invoice.
