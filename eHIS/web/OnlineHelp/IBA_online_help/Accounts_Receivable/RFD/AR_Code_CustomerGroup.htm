Enter the following to create Customer Group:

 

  1. Cust Group Code - Enter the Customer Group Code in alphanumeric format of two characters maximum length. The Customer Group Code is unique and cannot be changed after creation. This is a  mandatory field.

  2. Long Description - Enter the long description for the Customer Group in alphanumeric format of 30 characters maximum length.  This is a  mandatory field.

  3. Short Description - Enter the short description for the Customer Group in alphanumeric format of 15 characters maximum length.  This is a  mandatory field.

  4. Cust Group Description to print in Bill – Enter the description of the customer group in alphanumeric format of maximum 100 characters length. The text field is non-mandatory. The value recorded in the text field will be printed in the bill. This is applicable for users in ALNA.

  5. Ref Title - Enter the title for customer's reference in alphanumeric format of 15 characters maximum length. For example, the title for the Customers paying on behalf of the patients can be "Policy Number" for insurance companies; "Company Number" for company sponsored patients. The title for non-patient related customers can be “Purchase Order Number”, etc. You can leave this field blank, if the title for customer reference is not applicable.  

  6. Payment Terms Code – Click [List of Values] to select the Payment Terms Code. The values defined using the Payment Terms function get listed. Short descriptions of the payment terms are displayed for user verification. This is a  mandatory field.

  7. Markup % - The Markup percentage defined for the selected Payment Term Code using the function Payment Terms is displayed. This field cannot be edited.

  8. Chart of A/C – Click [List of Values] to select a Chart of Accounts Code. The values defined using the function Chart of Accounts under the GL Setup menu gets listed. If the selected code is valid, the description for the respective gets displayed for user verification. The description cannot be edited.

  9. Claim ID MCN# Mandatory – Select the check box to make the ‘Claim ID MCN#’ field is mandatory in Financial Details screen. This is applicable for users in Taiba.
