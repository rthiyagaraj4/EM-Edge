Enter the following details to associate transactions to the Outstanding Documents:

 

  1. Document Type – Click [List of Values] to select a Document Type. This field is mandatory. On entering the Document Type, press [F8] or click [Execute Query]. The Document Number, Document Date, Customer Code, Customer Reference, Document Amount, Balance Amount and Narration gets populated based on the records saved using the function Outstanding Documents.

 

Associated Document Details

  1. Doc Type - Click [List of Values] to select a Document Type. This is a mandatory field.  

  2. Doc Number – Enter the Document Number in numeric format of eight digits maximum length.  This is a mandatory field.  

  3. Doc Date – Click Calendar function to select the Document Date. The date indicates when the respective document was created. The document date has to be equal to or greater than the original document date displayed in the header. This is a mandatory field.  

  4. Month/Year – Enter the Post Month and Year in numeric format of two and four digits maximum length. The value has to be lesser than the current processing month and year defined in the AR Parameter. This is a mandatory field. If the header document is a consolidated Invoice, the details created using the Outstanding Documents function get populated in the respective fields along with the outstanding amount. The total Allocated Amount is copied into the ‘Amount’ column. You cannot change the details. If the document is not consolidated, the Amount can be entered manually.

  5. Amount - Enter the Bill Amount in numeric format in the form 999,999,999.99.

  6. Customer Ref – Enter the Customer Reference details in alphanumeric format of 15 characters maximum length.

  7. Narration - Enter the Narration in alphanumeric format of 50 characters maximum length.

 

Patient Bill Details

  1. Document Type - Click [List of Values] to select a Document Type. This is a mandatory field.  

  2. Bill Doc. Number – Enter the Document Number in numeric format of eight digits maximum length. This is a mandatory field.  

  3. Doc Date – Click Calendar function to select the Document Date. This is a mandatory field.  

  4. Patient No – Click [List of Values] to select a Patient Number. This is a mandatory field.  

  5. Allocated Amount – Enter the Allocated Amount in numeric format of nine digits maximum length.
