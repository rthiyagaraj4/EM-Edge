Associated Transactions for Outstanding Documents

From this function you can Associate transactions for Outstanding Documents.

This function is used to club all the outstanding document details associated to a particular Customer. You can associate any number of outstanding document along with the bill number and the month to the Customer. The bill has to be generated using the Patient Billing Manager module. The bills that are not settled by the Customer are clubbed.

Associated transactions may arise due to the following reasons:

  1. An outstanding document recorded using the function Outstanding Documents get displayed. You have to query to navigate through the records. You can associate multiple outstanding records to the existing record.

  2. If the records created using the Outstanding Documents function has to be recorded first, even though it is partially paid.

  3. You have an option to associate additional outstanding documents to the existing outstanding document type. There can be multiple associated transactions for the respective outstanding document.

Associating additional documents to the existing outstanding document type

From the Op - Balance, double-click Associated Transactions for Outstanding Documents function.

The Associated Transactions for Outstanding Documents screen gets displayed.

  1. Click [Execute Query] or press [F8] to query the existing outstanding Document Type.

OR

  1. You can also click [Enter Query] to search for a relevant Document Type. Click [Execute Query] or press [F8] and the outstanding document details get listed.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3.  Click [Commit Form] or press [F10] to save the created records or [Clear Record] to clear the entered or queried data.

  4. Click [Delete Record] to delete an associated document from the outstanding document type. A confirmation screen pops-up. Click [Yes] to delete or [No] to cancel the delete process.
