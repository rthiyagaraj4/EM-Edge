Report Outstanding Documents

From this function you can Generate a report to view the documents with the outstanding amount for the services utilised from the facility.

This function generates a report of all the outstanding documents for a range of Customers within the Facility. The outstanding documents are processed or a report is generated based on the selection.

Generating report on outstanding documents

From the OP-Bal, double-click Report Outstanding Documents function.

The Report Outstanding Documents screen gets displayed.

  1. Click [List of Values] to select a valid Customer Code. Both the values are optional. If ‘From’ value is not entered, the system assumes the lowest value for the ‘From’ and if ‘To’ value is not entered, the system assumes the highest value for the ‘To’ value. When both values are entered, ‘To’ value has to be greater than or equal to the ‘FROM’ value.

  2. Select ‘Process’ option to view the posted transactions or ‘Report’ option to print the list of transactions.

  3. Click [Run Process] to View or Print the Outstanding Document reports.

    On clicking [Run Process], the Preferences screen gets displayed. Select the required Report Option to generate the report. Click [OK] to generate the report and spool it in the file on the print server or [C] to cancel the action.

    The message ‘Do you want to confirm?’ gets displayed. Click [OK] to View or Print the Outstanding Documents reports or [No] to select another criteria.
