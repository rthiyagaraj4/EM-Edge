Category Types for Composition Class  

From this function you can Associate a Composition Class to a pre-defined Treatment or Procedure Category Types and Query for a selected record.

Treatment within a facility may cater to one or more Composition Classes. This function is used to assign one or more Treatment/Procedure Category Types to a Composition Class that helps in assigning the Composition Class for a particular treatment. The compositions, which are a part of the Composition Class, are in turn associated to the Treatment/Procedure Category Types. The Composition Class is applicable for the selected Treatment/Procedure Category Types.

A Composition Class is associated to the pre-defined Treatment/Procedure Category Types. The Treatment/Procedure Category Types are linked to the icons. During transactions, the relevant Composition Class is listed for the selected Treatment/Procedure Category Type based on the association.

Associating Treatment/Procedure Category Type to Composition Class

From the Code Setup > Compositions menu, select the Category Types for Composition Class function.

The Category Types for Composition Class screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Click [?] function to select a Composition Class. The values defined using in the 'Composition Class' get listed. This is a mandatory field.

  3. The pre-defined Treatment/Procedure Category Types are listed. Select the check box to associate selected Treatment/Procedure Category Types to the Composition Class. The system defined values are ‘Advanced Periodontal Treatment’, ‘Bridge’, ‘Crown’, ‘Denture’, ‘Extraction’, ‘Impaction’, ‘Implant’, ‘Maxilla Facial Prosthesis’, ‘Oral Cavity’, ‘Orthodontic Appliance’, ‘Others’, ‘Root Treatment’ and ‘Tooth Treatment’. This defines the composition class for the selected Treatment/Procedure Category Type.

  4. Click [Apply] to save the record created or [Reset] to clear the entered data.

  5. Click [Query] to search for existing records.
