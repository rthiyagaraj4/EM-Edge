Tooth Applicability for Chart

From this function you can Associate components to a dental chart and Query for the selected Charts.

The Tooth Applicability for Chart function enables you to record the mandatory components for a tooth. After completing the dental treatment, the dentist proceeds to record the gum, periodontal ligaments, membranes and bone conditions of a tooth. The details of the components associated to a tooth have to be recorded while entering the details using the Baseline/Evaluation chart in a Periodontal Chart.

Associating Components to a Chart

From the Code Setup > Periodontal Chart Setup menu, select the Tooth Applicability for Chart function.

The Tooth Applicability for Chart screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select a Chart ID from the list. The descriptions of the Chart ID's created using the function Chart Definition get listed.

  3. On selecting the Chart ID, components associated to the tooth are displayed. If the components are not associated, you can select the components from the list and associate it to a tooth. The components associated to the Chart using the ‘Chart Setup’ function are listed. <Related Field Description>

  4. Click [Apply] to save the records created or [Reset] to clear the entered data.

  5. You can update the saved record to select or clear the ‘Required?’ check box. Click on the component name, which is displayed as a hyperlink. The details get populated in the respective fields on clicking the hyperlink. You can either select or clear the check box. Click [Record] to save the changes. Click [Clear] to reset the values in the fields.

  6. Click [Query] to search for the existing records.
