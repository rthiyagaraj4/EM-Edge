Baseline Chart

From this tab you can Create a Baseline Chart to record dental conditions and treatment during the patient's initial visit to the facility. The Baseline Periodontal charting is done during the first visit of the patient to the Dental Clinic. The numbering system selected while defining the parameters is displayed, by default. The charts defined using the function Chart Definition get listed. Upon selecting the chart, the system populates the components set up for the respective chart. You can record the chart for the Upper or the Lower Arch. A Periodontal Chart recording is said to be complete, only when the status is selected as Complete. You can view the summary of the chart defined under 'Summary' in the Restorative Chart.  

 

You can also view the summary of the values recorded against the tooth number while creating the chart. The components associated to the selected Chart also get displayed with the values recorded for each component. Click    adjacent to Other Facility Charts to list Restorative Charts created in other Facilities of the Enterprise for respective patient. This option is available under ‘Baseline Chart’ and ‘Evaluation Chart’ tabs. You can also view the pictorial representation for the values recorded by clicking on the hyperlink View Graph.

The View Graph enables you to view a graphical representation of the selected Charts. Click 'View Graph' hyperlink adjacent to  to view the graph. The graph displays the Patient ID, Chart Name and the Chart Type selected. You can only view the graphical representation and cannot make any changes to the chart. Select the component by clicking on the appropriate radio button to view the graphical representation. Click to view the graphical representation of CAL, CEJ-GM or PD components. The values recorded through the Baseline Chart and the Evaluation charts are displayed using, and images respectively over the tooth range.

 

Six recordings per tooth can be made for the following components:

• Pocket Depth

• CEJ-GM

• Attachment Loss

• Plaque     

• Debris

• Calculus

• Bleeding in Probing

 

The recordings can be made on the buccal surface as well as the lingual surface of the Upper Arch and the Lower Arch. The six surfaces are mesio facial, facial, disto facial. On the lingual side it is the mesiolingual, disto lingual and lingual.

 

Other components such as Mobility, Missing teeth, Furcation, Food Impaction are represented with icons and charting is done with respect to the tooth, which is represented in the Buccal side of the tooth, and single recording is done against each tooth.
