Enter the following details to define OH Parameter for facility:

 

  1. Tooth Numbering System – Select a Tooth Numbering System from the list. The system-defined values are ‘Universal Numbering System’ and ‘FDI Numbering System’. The selected Tooth Numbering System is set as the default numbering system for the Facility. Dentists, in writing or speech, use several different Dental notation systems for associating information to a specific tooth. The most common systems are the FDI World Dental Federation notation and the Universal numbering system. This is a mandatory field.

  2. Age Range To Display Mixed Dentition [From-To] - Enter the Age range in numeric format to indicate the mixed dentition period. A mixed dentition is a situation, when both deciduous and permanent teeth are present. The age range has to be specified in years and days. When a patient with the age range defined, visits the hospital, a respective chart is displayed based on the mixed dentition period. This is a mandatory field.

  3. Deciduous Grace Period (in days) – Enter the number of days up to which, the deciduous chart is displayed. The Deciduous Chart is displayed over and above the age limit set.

  4. Default Chart To Display For Mixed Dentition - Select a value from the list. The system defined values are ‘Deciduous’ and ‘Permanent’. When a patient in the mixed dentition age range visits the facility, the chart is displayed based on the value selected for the default chart.

  5. Deciduous – Select Deciduous and a Deciduous chart gets displayed for the mixed dentition period.

  6. Permanent – Select Permanent and a Permanent chart gets displayed for the mixed dentition period.

  1. Validity Period for Baseline Restorative Chart – Enter the number of days to indicate the validity period of a restorative chart. This indicates that the restorative chart for a relevant patient is maintained up to the number of days specified. If the patient visits the facility once again within the number of days specified, the dentition has to either create a new chart or use the existing chart. If a new chart is created, dentist  has an option to either copy the previous chart details to the new chart or record the details in the fresh chart.

  2. Validity Period for Baseline Periodontal Chart (Warning Days) – Enter the number of days to indicate the validity period of a periodontal chart. This indicates that the periodontal chart for a relevant patient is maintained up to the number of days specified. If the patient visits the facility once again within the number of days specified for the periodontal chart, the dentist  can either create a new or use the existing chart. If a new chart is created, you can either copy the previous chart details to the new chart or record the details in the fresh chart.

  3. Record Gingival Status in Restorative Chart? – Select the check box to invoke gingival status recording in Restorative Chart. Clear the check box, if the gingival status recording is not required in Restorative Chart.

  4. Interface to Billing Module – Select the check box to interface the Oral Health module to the Patient Billing Manager module. The patient bills are generated only if the Patient Billing Manager module is interfaced. Clear the check box, if the association to Patient Billing Manager module is not required.

  5. Gingival Status Recording – Select either ‘By Quadrant’ or ‘By Sextant’ to record the gingival status of the patient. If ‘By Quadrant’ is selected, the Dentist can record the gingival status on the quadrant when the patient arrives at the facility. If ‘By Sextant’ is selected, the Dentist can record the gingival status on the sextant when the patient arrives at the facility. The value for ‘Site’ under the ‘Gingival Status’ tab is set to the value selected. If the check box is selected, you can record Gingival Status in Deciduous Chart and Deciduous Mixed Dentition Chart.

  6. Enterprise Baseline Chart Applicable - Select the check box, to enable dentists to access a single baseline chart throughout the enterprise. Clear the check box to restrict the dentists from accessing the chart created in a facility. On clearing the check box, patient can have a single chart for a visit to each facility for treatments.

 

The Deciduous, Deciduous/Permanent Mixed Dentition and Permanent columns are activated for a Sextant, on selecting the check box 'Record Gingival Status in Restorative Chart?'. If the check box is cleared, the mentioned columns are disabled.

 

Sextant

The upper and the lower arch is divided into six parts called the sextant, namely, Upper Right, Upper Anterior, Upper Left, Lower Left, Lower Anterior and Lower Right. Restriction on the number of tooth differs for each sextant.

 

When the 'Default Chart To Display For Mixed Dentition' is selected as 'Deciduous' or 'Permanent', the maximum number of tooth that can be displayed in each sextant varies.

=> A Deciduous Chart, can have a maximum of six tooth in the Upper and Lower Anterior whereas Upper Right, Lower Right, Lower  Left and Upper Left can have a maximum of two tooth.

=> A Deciduous Mixed Dentition Chart can have a maximum of six tooth in the Upper and Lower Anterior whereas Upper Right, Lower Right, Lower Left and Upper Left can have a maximum of four tooth.

=> A Permanent Mixed Dentition Chart can have a maximum of four tooth in the Upper Right, Lower Right, Lower Left and Upper Left sextants and six tooth in the Upper and Lower Anterior sextants.

=> A Permanent Chart can have a maximum of five tooth in the Upper Right, Lower Right, Lower Left and Upper Left sextants and six tooth in the Upper and Lower Anterior sextants.

 

Max Num. of Tooth (<=)

Enter the number of tooth in numeric format to appear in each Sextant. The number of tooth to be displayed in each sextant varies. This is a mandatory field.

 

Default Tooth No

The values for each of the Sextant are system defined. Select a default tooth number from the list. The default tooth number is highlighted in the default chart selected to record the gingival status. This is a mandatory field.

 

The columns 'Max Num. of Tooth (<=)' and 'Default Tooth No' gets displayed under three separate headings: Deciduous, Deciduous/Permanent Mixed Dentition and Permanent. If the 'Default Chart To Display For Mixed Dentition' is selected as 'Permanent', the column name is displayed as 'Permanent Mixed Dentition'. If the 'Default Chart To Display For Mixed Dentition' is selected as 'Deciduous', the column name is displayed as 'Deciduous  Mixed Dentition'.

 

Quadrant

A Quadrant is divided into four parts namely,  Upper Right, Upper Left, Lower Right and Lower Left. A Dentist can record conditions and treatment against the Quadrant, if a condition has to be recorded. The 'Max Num. Of Tooth (<=)' and 'Default Tooth No' fields are mandatory, if the 'Default Chart To Display For Mixed Dentition' is selected as 'Deciduous'. The specified fields are non-mandatory, if the 'Default Chart To Display For Mixed Dentition' is selected as 'Permanent'.

 

When the 'Default Chart To Display For Mixed Dentition' is selected as 'Deciduous' or 'Permanent', the maximum number of tooth that can be displayed in each quadrant varies.

 

Max Num. Of Tooth (<=)

Enter the number of tooth to appear in each Quadrant. Restriction on the number of tooth that can appear in a Quadrant varies.  

Restriction on the number of tooth in each Quadrant differs for Dental Charts:

  • A Deciduous chart can have a maximum of five tooth in the Upper Right, Upper Left, Lower Right and Lower Left Quadrants.

  • A Deciduous Mixed Dentition chart can have a maximum of seven tooth in the Upper Right, Upper Left, Lower Right and Lower Left Quadrants.

  • A Permanent Mixed Dentition chart can have a maximum of seven tooth in the Upper Right, Upper Left, Lower Right and Lower Left Quadrants.

  • A Permanent Dentition chart can have a maximum of eight tooth in the Upper Right, Upper Left, Lower Right and Lower Left Quadrants.

 

Default Tooth No

The values for each of the Quadrants are system defined. Select a default tooth number from the list. The default tooth number is highlighted in default chart while recording gingival status. The values in the list varies based on the value specified for 'Max Num. Of Tooth (<=)'.
