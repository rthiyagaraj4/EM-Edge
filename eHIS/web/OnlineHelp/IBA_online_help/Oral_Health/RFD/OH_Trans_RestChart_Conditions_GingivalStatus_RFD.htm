Enter the following details to record the Gingival Status:

 

  1. Clinical Group – Select a Clinical Group from the list. The Clinical Group defined in the Code setup is listed. It identifies the Clinical Group, where the patient is treated.

  2. Site – The site value gets displayed as Quadrant or Sextant based on the default value selected for ‘Gingival Status Recording’ in the ‘Setup Parameters for Facility’ function.

  3. Area – Select an Area from the list. The Area differs based on the Site selected. This is a mandatory field. The following values are listed, if the ‘Site’ is set as ‘Quadrant’:

  4. If the Site is selected as ‘Arch’, the Area is set to ‘Arch’. You can select either the Upper Arch or the Lower Arch from the list. The values are system defined.

  5. If the Site is selected as ‘Quadrant’, the Area is set to ‘Quadrant’. You can select the Upper Right, Upper Left, Lower Right or the Lower Left quadrants from the list. The values are system defined.

  6. If the Site is selected as ‘Tooth’, the Area is set to ‘Tooth No’. You can select a tooth from the list. The selected Condition Type is applicable for the tooth selected. The values are system defined.

  7. If the Site is selected as ‘Tooth Range’, the Area is set to ‘Tooth Nos’. You can select a range of tooth from the list using [Ctrl] key. The selected Condition Type is applicable for the selected tooth range. A minimum of two teeth has to be selected for the tooth range. The values are system defined.

The following values are listed, if the ‘Site’ is set as ‘Sextant’:

  • Lower Anterior Sextant – Select Lower Anterior Sextant to record the conditions on the tooth numbers 31, 32, 33, 41, 42 and 43 in the lower sextant.

  • Upper Anterior Sextant – Select Upper Anterior Sextant to record the conditions on the tooth numbers 11, 12, 13, 21, 22 and 23 in the lower sextant.

  • Lower Left Sextant – Select Lower Left Sextant to record the conditions on the tooth numbers 34, 35, 36, 37 and 38 in the lower sextant.

  • Upper Left Sextant – Select Upper Left Sextant to record the conditions on the tooth numbers 24, 25, 26, 27 and 28 in the upper sextant.

  • Lower Right Sextant – Select Lower Right Sextant to record the conditions on the tooth numbers 44, 45, 46, 47 and 48 in the upper sextant.

  • Upper Right Sextant – Select Lower Right Sextant to record the conditions on the tooth numbers 14, 15, 16, 17 and 18 in the upper sextant.

  1. Tooth No – Select a Tooth Number from the list. The numbers in the list differ based on the Area selected. This is a mandatory field.

  2. Condition Type – Select a Condition Type from the list. The values are system defined. The recorded condition is displayed for the selected tooth. This is a mandatory field.

  3. Gingival Status – Select a Gingival Status from the list. The values are system defined. The recorded condition is displayed for the selected tooth. This is a mandatory field. Each Gingival Status is numbered as follows:

  4. 0 – Normal

  5. 1 – Bleeding

  6. 2 – Calculus

  7. 3 – Pocket Depth 4-5 mm

  8. 4 - Pocket Depth > 6 mm

  1. Click [Record] to save the record or [Reset] to clear the data and to enter new Gingival Status. The Gingival Status is displayed on the right hand side of the screen.
