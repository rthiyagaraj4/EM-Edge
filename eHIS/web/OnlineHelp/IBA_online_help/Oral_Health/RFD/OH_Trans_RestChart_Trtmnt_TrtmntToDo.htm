Treatments To Do

The name of the ordered treatment along with the amount for the treatment is displayed with the status as ‘(OS)’, which indicates that the treatment is ordered. You can update status of the selected task.

  1. Click on the individual task associated to a treatment. The recorded data is populated in the respective fields under ‘Treatment’ in the lower part of the Baseline Chart screen.

  2. You can either register or schedule the task by clicking on the respective options displayed as hyperlinks on clicking the task name. You can record the task and schedule a date to perform the task on clicking the Schedule hyperlink. You can plan the treatment, if you directly register the task without scheduling an appointment.

    Enter the following details to register:

    1. Reg Date/Time - The 'Reg Date/Time' in the screen is defaulted to the current system date and time. This is a mandatory field. You can updated only the time.

    2. Status - Select a status from the list. The system-defined values are 'Planned', 'Add' and 'Cancel'. Select 'Planned' to register the task. On selecting Add, two additional fields, 'Tasks Applicable' and 'Sequence' gets displayed. Select a value for Sequence. The system-defined values are 'Row Above' and 'Row Below'. Select 'Row Above' to add the task above the existing task and 'Row Below' to add the task below the existing task. Select Cancel to cancel the task for the selected patient.

    3. Remarks - Click Remarks displayed as a hyperlink to record remarks for the respective task.

    4. Edit/Charges - Click Edit/Charges displayed as a hyperlink to modify the charges.

    5. [Register] - Click [Register] to register the task. [RG] is displayed beside the task name when the registration is complete.

    6. [Reset] - Click [Reset] to reset the details entered.

    Enter the following details to schedule:

    1. Status - Select a status from the list. The system-defined values are 'Ordered', 'Add' and 'Cancel'. Select 'Ordered' to schedule appointment.  On selecting Add, two additional fields, 'Tasks Applicable' and 'Sequence' gets displayed. Select a value for Sequence. The system-defined values are 'Row Above' and 'Row Below'. Select 'Row Above' to add the task above the existing task and 'Row Below' to add the task below the existing task. Select Cancel to cancel the task for the selected patient.

    2. [Schedule] - Click [Schedule] to book appointment and schedule the task.

    3. [Reset] - Click [Reset] to reset the details entered.

  3. Select a value for Status - Ordered, Add and Cancel. Select Ordered to mark the treatment as ordered for the selected patient. Select Add to associate an individual task to the treatment. On selecting Add, two additional fields, 'Tasks Applicable' and 'Sequence' gets displayed. Select a value for Sequence. The system-defined values are 'Row Above' and 'Row Below'. Select 'Row Above' to add the task above the existing task and 'Row Below' to add the task below the existing task. Select Cancel to cancel the treatment for the selected patient.

  4. On selecting the treatment status as ‘Cancel’, additional field called the ‘Cancel Reason’ gets displayed. Enter the reason in alphanumeric format. On cancelling the treatment, the image on the tooth is deleted and the treatment is deleted from the ‘Treatments To Do’ menu on the right hand side of the screen.

The pending treatments are listed on the right-hand side menu with the respective status.

=> You can register a task even though the previous task associated to the treatment is incomplete.

=> You can register multiple tasks without any sequence, but sequence validation is applicable for resulting.

  1. Select the check box ‘Performed in OT’, if the treatment is recorded in Operating Theatre. Clear the check box, if the treatment is not recorded in Operating Theatre. The check box is available at the task level only if the task is registered.

  2. You can also record the components required to perform the treatment. Click Composition Details hyperlink to record the details. You can record the composition details for an individual task, if the task is registered. Composition Details hyperlink is available at the treatment level, if the treatment is registered.

    You can record the components while planning the treatment or completing the treatment. Click Composition Details hyperlink to record the details.

    Composition Details Entry

    1. Composition Class – Select a Composition Class from the list. The Composition Class defined in the OH Code Setup menu gets listed.

    2. Composition - Select a Composition from the list. The Compositions associated to the Composition Class using the function Composition in the OH Code Setup menu are listed.

    3. [Record] - Click [Record] to record the compositions. On clicking [Record], the composition details get listed in the screen.

    4. Select – Select the check box to select and to delete the associated composition details.

    5. [Delete] – Click [Delete] to delete associated the components.

    6. [Clear] - Click [Clear] to clear the entered data.

    7. [Close] - Click [Close] to exit from the screen.

    8. Data Entry Completed – Select the check box, if association of composition class to the task is complete.

    9. Task - The task name is displayed as a hyperlink. The composition details are displayed on selecting the task.

  3. You can record Complication details for an individual task, if the task is complete. Click Complication Details hyperlink to record the details. Complicaiton Details hyperlink is available at the treatment level, if the treatment is registered.

    You can record the complications when the treatment is being done.

    Click on the hyperlink to view the Complications associated to the respective Task. You can record the details until the task is marked as data entry completed. The first task that is associated to the complication is highlighted in bold. When the task is marked as completed, you can only view the details and cannot make any changes.

    Complication Details Entry

    1. Complication – Select a Complication from the list. The Complications  defined in the OH Code Setup menu gets listed. An additional value 'Others (Free Text)' is available in the list apart from the values defined through 'Complication' function. Select 'Others (Free Text)' to record additional details on any complication. You can enter a complication in alpha numeric format with a maximum of 100 characters when the value is selected as 'Others (Free Text)'.

    2. [Record] - Click [Record] to record the complications. On clicking [Record], the details get listed in the screen under Complication Details header.

    3. [Clear] - Click [Clear] to clear the complication that is selected.

    4. Select – Select the check box to select and delete the respective Complication. A confirmation message 'Tasks Data Entry will be completed. Do you want to proceed?' is displayed on selecting the check box. Click [OK] to proceed or [Cancel] to abort the process.

    5. [Delete] – Click [Delete] to delete associated the complication.

    6. [Close] - Click [Close] to exit from the screen.

    7. Data Entry Completed – Select the check box, if association of Complication to the task is complete.

    8. Task - The task name is displayed as a hyperlink. The composition details associated to the task are displayed on selecting the task.

  4. You have to book an appointment for the patient when the treatment being ordered. If the appointment is not scheduling while placing the order, you can schedule an appointment by clicking [Schedule]. Click [Reset] to clear the data. The Book Appointment screen is displayed. Select a convenient slot, record the patient details and confirm the booking in the Book Appointment screen.

  5. An appointment reference number is generated for patient’s appointment on the scheduled date and time.

  6. The name of the task is updated under the ‘Treatments To Do’ menu with the status as [SC], which indicates that the appointment is scheduled and the appointment reference number.

  7. Click on the task name to complete the treatment. The details are displayed under the Baseline Chart with only the Status field as enabled. Select a Status from the list - ‘Planned’ and ‘Mark Error’. Select Planned to mark the status as ‘Planned’ or ‘Mark Error’ to disable the treatment. On selecting Mark Error, a new field, namely ‘Cancel Reason’ gets displayed. Click on the hyperlink to record the reason to cancel the treatment. Click Remarks hyperlink to view the remarks.

An Order Catalog has to be associated to a Specialty Treatment/Procedure. Select the check box ‘Billable’ while defining the Order Catalog in the Order Entry and Result Reporting module. The Order Catalog is associated to a Billing Service in the Patient Billing Manager module based on which the charges are computed for the individual treatments. If the Order Catalog is not billable, the charges are not computed and the order is not placed.

  1. Select ‘Planned’ and click [Register] to register the pending treatment. You can edit the charges by clicking the Edit Charges hyperlink.

  2. A ‘$’ symbol gets displayed against the task name and the status is changed to [RG] on registering the treatment. The total charge of treatment, patient charge and paid amount are listed for each task. The fields 'Included' and 'Approval Required' indicates if the service associated to the treatment includes policy level coverage and if it requires Pre-approval. If the policy level coverage is available and pre-approval required, the value is displayed as 'Yes'. If the policy level coverage is not available and pre-approval not required, the value is displayed as 'No'.

  3. An image  is displayed adjacent to the treatments listed. On mouse over the icon, respective Encounter ID of the patient, Practitioner name and the Department where the treatment is performed are listed. The Department is displayed as ‘OPD’ for Outpatient and Emergency patient classes whereas it is displayed as ‘IPD’ for Inpatient and Daycare patient classes. The department is displayed as ‘OPD + OT’ for Outpatient and Emergency patient classes whereas it is displayed as ‘IPD + OT’ for Inpatient and Daycare patient classes.

  4. Click on the task name to record the treatment start and end date and time. The treatment details get displayed in the respective fields under the Baseline Chart. The Status is set as Registered. Click the Calendar function to specify the date for treatment. The Treatment Start Date/Time has to be greater than or equal to the date and time when the treatment is registered, but lesser than the system date and time. The Treatment End Date/Time has to be equal to or lesser than the system date and time.

  5. Click Edit Charges hyperlink to edit the charges. The price should not exceed the charge of the treatment.

    Enter the following details to update the charge for the service:

    1. Rate – Enter the rate for the service in numeric format. You can update only the Rate field. This is a mandatory field.

    2. [Accept] – Click [Accept] to save the changes. On clicking [Accept], the charges are updated in the Edit/Enter Charge Details.

    3. [Cancel] – Click [Cancel] to exit from the screen.

  6. Click [Result Entry] to record the treatment result. The Result Entry screen gets displayed.

  7. Based on the event status selected, the Status gets displayed adjacent to the treatment name and the default value differs in the Status field under the Baseline Chart. The Composition Details and Complication Details hyperlinks are displayed when the  treatment is planned. Click on the respective hyperlink to record the details.

    4. Select – Select the check box to select and delete the respective Complication. A confirmation message 'Tasks Data Entry will be completed. Do you want to proceed?' is displayed on selecting the check box. Click [OK] to proceed or [Cancel] to abort the process.

    7. Data Entry Completed – Select the check box, if association of Complication to the task is complete.

    8. Task - The task name is displayed as a hyperlink. The composition details associated to the task are displayed on selecting the task.
