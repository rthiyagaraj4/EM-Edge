Bed Availability

From this function you can Query the bed availability details of a facility.

Bed details within a specific location or with a specific status can be viewed by specifying the required criteria. The bed status is represented using color schemes for easier identification.

Querying for Bed Availability Details

From IP Queries, select the Bed Availability function.

The Bed Availability screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [Search] to view the Bed Availability details or [Reset] to clear the entered search criteria.

  3. On searching, the Bed Availability details are displayed.

  4. Click [Clear] to clear the result screen.

The Bed is released when a patient is discharged.

- This icon indicates that discharge bill is prepared for the current inpatient. It will be displayed only when the discharge bill is ready. Otherwise it will not be displayed.
