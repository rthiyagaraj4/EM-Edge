View Prepare Discharge Advice List

From this function you can Query the list of Patients whose discharge advice has been prepared.  

This function allows you to query the list of Patients whose discharge advice has been prepared.  You can query for the required date range.

Viewing Prepared Discharge Advice List

From the IP Queries > Admissions/Transfers, select View Prepare Discharge Advice List function.

The View Prepare Discharge Advice List screen gets displayed.

  1. Enter the details in the fields to query the discharged advice prepared patient list. <Related Field Description>

  2. Click [Search] to view records that satisfy the query criteria or [Reset] to clear the entered details.

  3. On searching, the list of patient records whose discharge advice is prepared are displayed.

  4. Click [Clear] to clear the search result details.
