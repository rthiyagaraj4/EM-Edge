Current Inpatient Count

From this function you can Query on the count of inpatients currently admitted in the facility.

The function provides the option to view the total count of In-Patients, Foreign Patients, Patients Discharge Advice Prepared, Corporate Patients, Insurance Patients, Free Case Patients and Cash Patients currently admitted in the facility.

Querying for Current Inpatient Count

From the IP Queries > select the Current Inpatient Count function.

The Current Inpatient Count screen gets displayed.

  1. Enter the details in the fields to query the current inpatient count. <Related Field Description>

  2. Click [Search] to view the current inpatient count or [Reset] to clear the entered search criteria.

  3. On searching, the total count of In-Patients, details of Foreign Patients, Patients Discharge Advice Prepared, Corporate Patients, Insurance Patients, Free Case Patients and Cash Patients are displayed. Count is displayed as hyperlink; on click of Count hyperlink Patient Details pop up gets displayed.

  4. Click [Clear] to clear the details on the result screen.
