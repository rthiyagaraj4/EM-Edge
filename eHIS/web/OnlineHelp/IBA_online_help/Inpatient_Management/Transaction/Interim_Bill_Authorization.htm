Interim Bill Authorization

From this function you can Authorize IP encounters having unbilled amount for which interim bill needs to be generated.

This function allows to authorize IIP encounters having unbilled amount for which interim bill needs to be generated in Discharge Bill cum Receipt function.

This function is enabled only if the Interim Bill Authorization check box is selected in BL Parameters screen.

Interim Bill Authorization is valid for 1 day only for the encounter. (Authorization is valid till the End of the Day).

After interim bill generation, the same encounter will not be displayed until if the bill is cancelled.

 

Once the authorization is given for the encounter,

• Further services added to the encounter do not need authorization until 11.59 PM.

• Further interim bill generated to the encounter do not need authorization until 11.59 PM.

• However, for the encounter, Interim bill authorization is required every month

• Encounters having unbilled amount of zero do not need authorization. Only encounters having plus values or minus values will be displayed in the screen for authorization.

 

Based on the privilege mapped against the user / group / responsibility for the selected payer group / payer / policy in the Function Privileges Setup for Policy function, authorization is given in the ‘Interim Bill Authorization’ screen.

Authorizing Encounters for Interim Bill generation

From the IP Transactions> Discharge, click the Interim Bill Authorization function.

The Interim Bill Authorization screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search], the IP encounters having unbilled amount for which interim bill need to be generated gets displayed.

  3. Select the check box against the required encounter and click [Authorize]. The Authorization dialog gets displayed.

  4. Enter the User ID, Pin and Reason in the respective fields. These fields are mandatory.

  5. Click [OK]. The selected record gets authorized.

  6. Click [Apply] to complete the transaction or click [Reset] to clear the entered data.
