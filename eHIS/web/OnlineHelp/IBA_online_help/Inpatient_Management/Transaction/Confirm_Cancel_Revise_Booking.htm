Confirm/Cancel/Revise Booking

From this function you can Confirm an existing booking pending for confirmation from the ‘Create Booking’ function, Cancel an existing booking or Revise the details recorded for an existing unconfirmed booking.

Confirm Booking Process

All bookings are confirmed before admitting patients to Nursing Units. While confirming a booking, Nursing Unit details are recorded if not recorded while creating a booking. Some of the details entered during booking can be modified. Once a booking is confirmed, it cannot be revised.

Cancel Booking Process

A booking is cancelled by specifying the reason for cancellation. You cannot modify booking details during a cancellation process.

Revise Booking Process

Booking details are revised but the patient details cannot be revised using this function. The system validates the ‘Grace period for revision of booking’ parameter set in IP Parameter. This ensures that the facility is able to revise the booking for admission within the number of days specified from the preferred admission date of a patient. Any booking is revised only before confirming the booking.

Confirming/Cancelling/Revising a Booking

From the IP Transactions > Booking, select the Confirm/Cancel/Revise Booking function.

The Confirm/Cancel/Revise Booking screen gets displayed.

  1. Click [Search] to display all the patients whose bookings are created but not confirmed in the Create Booking function, or enter the search criteria and click [Search] to select specific patients whose bookings are created but not confirmed. Click [Clear] to reset the records displayed in the result area.

  2. On searching, Booking details of the patient are displayed based on the entered criteria.

  3. Click ‘+’ hyperlink to view the Booking details, Patient details, Nursing Unit details and Bed details of the selected patient as recorded while creating booking. Click [Close] to exit from the Confirm/Cancel/Revise Booking screen.

  4. Click the ‘Booking Ref No’ hyperlink to display the Confirm/Revise/Cancel Booking screen. The screen displays Booking Details, Patient Details, Nursing Unit Details and Bed Details as recorded while Booking a patient for admission.

  5. Click [Confirm Booking] to confirm the Booking, [Revise Booking] to revise the booking details, [Cancel Booking] to cancel the booking, or [Close] to exit from the screen.

    1. On clicking [Cancel Booking], the Confirm/Revise/Cancel booking screen displays the Booking details of the selected patient.

    2. Select a Reason for Cancellation from the list. This is a mandatory field. The reasons for cancellation are defined in the ‘Reason’ function of Application Masters module.

    3. Click 'Patient Instructions' hyperlink, the Patient Instruction dialog gets displayed to add the instructions available in the Application Masters.

    4. Click [Confirm Cancellation] to cancel the booking, [Abort Cancellation] to abort canceling or [Close] to exit from the Cancel Booking screen.

    1. On clicking [Revise Booking], an alert message is displayed ‘The existing bed booking will be released. Would you like to release the existing bed booking?’. Click [OK] to release the bed booking and display the Revise Booking screen or [Cancel] to cancel the Revise Booking process.

    2. The Revise Booking screen displays the Booking details of the selected patient. The Procedure/Surgery Date Time, Pre-Op Preparation Days, Preferred Admission Date/Time, Expected Days of Stay, Priority, Booking details and Nursing Unit details can be revised.

    3. Enter the Reason for Revision or select the Reason for Revision using [?] function. The reasons for revision are defined in the ‘Reason’ function of Application Masters module. This is a mandatory field.

    4. Click 'Patient Instructions' hyperlink, the Patient Instruction dialog gets displayed to add the instructions available in the Application Masters.

    5. Click Financial Details tab to modify the financial details of the patient.

    6. Click [Revise] to revise the booking and [Close] to exit from the Revise Booking screen.

    1. On clicking [Confirm Booking], the Confirm Booking screen displays the Booking details of the selected patient. The Procedure/Surgery Date Time, Pre-Op Preparation Days, Preferred Admission Date/Time, Expected Days of Stay, Priority, Admitting Practitioner and Nursing Unit details can be modified.

    2. Click Financial Details tab to modify the financial details of the patient.

    3. Click 'Patient Instructions' hyperlink, the Patient Instruction dialog gets displayed to add the instructions available in the Application Masters.

    4. Click [Confirm] to confirm the booking and [Close] to exit from the screen.

 

The privilege to use these functions is based on the Operator Station associated to the logged in user.
