Register Newborn for Inpatient

From this function you can Register a Newborn in the logged in facility and generate unique identifier for every newborn.

The function allows capturing delivery details, demographics of parents and vitals of newborn. The purpose of registration is to generate unique identifier for every newborn and to maintain birth register used for statistical analysis.

The logged in user must have the rights to register a newborn for an Inpatient. The rights are defined in the Operator Station assigned to the logged in user in the Application Masters module.

Registering a Newborn for an Inpatient

From the IP Transactions > Admission, select the Register Newborn for Inpatient function.

The Register Newborn for Inpatient screen gets displayed.

  1. Click [Search] to display the list of all female Inpatients or enter the search criteria and click [Search] to select a specific female Inpatient. Click [Clear] to reset the records displayed in the result area.

    Enter the following search criteria:

    1. Nursing Unit – Enter the Nursing Unit in alphanumeric format of 15 characters maximum length or select the Nursing Unit using [?] function.

    2. Admission Date From-To – Enter the Admission Date range in the format of dd/mm/yyyy or select the date using the calendar function.

    3. Specialty – Enter the Specialty in alphanumeric format of 15 characters maximum length or select the Specialty using [?] function.

    4. Practitioner – Enter the Practitioner Name in alphanumeric format of 30 characters maximum length or select the Practitioner using [?] function.

    5. Encounter ID – Enter the Encounter ID in numeric format of 12 digits maximum length.

    6. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the Patient ID using [?] function.

  2. On searching, Admission Details of the female Inpatients are displayed based on the entered criteria.

  3. Click ‘+’ hyperlink to view the Current Encounter Details.

  4. Click the ‘Patient ID’ hyperlink to display Register New Born screen based on the MP Parameter Rules.

    The system checks the following rules set in MP Parameter before registration:

    • Age limit specified for the mother patient

    • Gestation period between two deliveries

    If the age of the patient chosen does not fall in the age range specified for the MP Parameter ‘Age limit specified for the mother patient’, the system displays an error message and does not allow the registration of the baby.

    Further, the system checks if the gestation period for the selected patient falls within the range specified in the MP Parameter ‘Gestation period between two deliveries’, the system displays a message and does not allow you to continue with the registration process.

    • Period between two deliveries

    If the patient chosen has had a baby delivered and registered in the logged in facility within the period specified in the MP Parameter then the system displays a message as and stops you from registering another baby for the same patient.

    • Review Mother Baby Link before Registration

    After verifying the above-mentioned rules, based on the ‘Review Mother Baby Link before Registration’ set up in MP Parameter, Mother Baby link screen is displayed. If mother baby link is not set, Register New Born screen is displayed directly.

    The ‘ Mother Baby link’ screen provides brief information on the selected mother patient and details of her child/children delivered by her within the enterprise.

    The mother patient details displayed are the Name of the Mother Patient, Age, Gender and Patient ID. If the mother patient is currently an Inpatient, Encounter ID, Nursing Unit, Ward and Bed Details are also displayed.

    On viewing the details, click [Register]. The ‘Register Newborn’ screen is displayed. If there are no babies registered for the chosen patient, the message ‘No child records available for this mother’ is displayed. Click [OK].

  5. On selecting the Patient ID hyperlink, the Register New Born screen is displayed.

  6. The Patient Series is displayed by default based on the MP Parameter setup. Select the required series from the list.

    If 'Patient Series Same as Mother Series' check box is selected in the New Born tab of MP Parameter, the mother Patient Series is displayed by default and all the Patient Series defined in ‘Patient Series’ function of Master Patient Index module is listed. Select the required Patient Series from the list.

    If 'Patient Series Same as Mother Series' check box is cleared in the New Born tab of MP Parameter, the default Newborn Series set in MP Parameter is displayed. The list contains Patient Series belonging to ‘New Born Series Type’ and ‘Unrestricted Series’.

  7. The Patient ID is auto generated or has to be entered manually depending on the selected Patient Series. If system generates the Patient ID for the selected Patient Series, you are prohibited to enter Patient ID or the field is enabled to enter the Patient ID. This field is mandatory.

  8. Enter the details in the mandatory (*) fields of Mother Details, Father Details and Newborn Details.

  5. Click ‘Change Patient Details’ hyperlink to invoke ‘Change Patient Details’ function. You can change the newborn details using this function.

  6. On clicking [Apply], Create Patient File screen gets displayed if file management parameters are set for the facility.

  7. Enter the details in the mandatory (*) fields. <Related Field Description>

  8. Click [OK] to generate file number or [Cancel] to cancel file generation.

Newborn registration in stages

  1. If the check box ‘Single Stage New Born Registration Applicable’ is cleared in the MP Parameter, the new born registration is done in two stages.

  2. 1st stage involves recording minimum details with birth date, time and gender. Click [Register] to register the newborn as the 1st stage of registration. The status of the record is displayed in color code. The status of the record is displayed as Registered.

  3. Click the Baby Name hyperlink to populate the Change Patient Details screen to update the New Born details.

  4. Click Newborn ID hyperlink to finalize the record. The message ‘Do you want to finalize?’ gets displayed. Click [OK] and the control shifts back to the registration screen to complete the finalization process. Click [Cancel] to cancel finalizing and the system prompts to print the report.

  5. Click [Audit Trail] to view the Audit Trail details of the newborn registration. The [Audit Trail] is displayed only on the registered or finalized records of the newborn.

  6. In the finalize mode, enter the mandatory details (Length of Baby, Head Circumference, Apgar Score, Patency of Anus) and click [Finalize]. You can print the Birth Certificate after finalization. The record is finalized and status of the record is displayed in color code. The status of the record is displayed as Finalized. Click Newborn ID to view the recorded details of the newborn. Click Task hyperlink to Admit the newborn.

 

Subsequent newborn registration is allowed within the period specified for ‘Subsequent Registration of Multiple Birth within [] hours’ in MP Parameter.

You can update the Mother details for second child birth and subsequent new born registration if ‘Modify Mother Details in Multiple Birth’s’ check box is selected in MP Parameter screen. The fields ‘Multiple Birth’, ‘No of Births’ and ‘Gestation’ are non-editable.
