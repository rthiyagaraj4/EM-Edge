Amend Inpatient Leave

From this function you can Amend the leave details of an Inpatient.

The activities performed by using this function are:

• Re-admit the patient on returning from leave

• Extend the leave period

• Cancel the leave

• Update the patient leave status as no show for the patients not returning from leave.

The transaction types available under ‘Amend Inpatient Leave’ function are:

• Record Return - To record the return of an Inpatient from leave.

• Record No Show - An Inpatient not returning back to the facility on elapse of leave period is recorded as ‘No Show’. The user has to mark the patient as ‘Absconding’ and specify the absconding date.

• Revise Leave - To revise/extend the duration of leave for an Inpatient granted leave.

• Cancel Leave – Leave granted to an Inpatient can be cancelled by the facility. If the bed has been released while recording the leave, previously occupied bed can be assigned to the patient, provided it is vacant, or a new bed is assigned.

Amending Inpatient Leave

From the IP Transactions > Admission, select the Amend Inpatient Leave function.

The Amend Inpatient screen gets displayed.

  1. Click [Search] to display the list of all Inpatients who are on leave or enter the search criteria and click [Search] to select a specific Inpatient who is on leave. Click [Clear] to reset the records displayed in the result area.

    Enter the following query criteria:

    1. Nursing Unit – Select the Nursing Unit using the [?] function.

    2. Specialty – Select the Specialty using the [?] function.

    3. Practitioner – Select the Practitioner using the [?] function.

    4. Admission Date From-To – Enter the Admission Date range in date format of dd/mm/yyyy or select the date using the calendar function.

    5. Encounter ID – Enter the Encounter ID in numeric format of 12 digits maximum length.

    6. Patient ID - Enter the Patient ID in alphanumeric format of  maximum length as specified in MP Parameter or select the Patient ID using [?] function.

  2. On searching, Inpatients with leave details recorded are displayed based on the entered criteria. The details of the Inpatients who are absent is displayed with a unique color.

  3. Click ‘Remarks’ hyperlink to view the remarks recorded while recording Inpatient Leave details.

  4. Click the ‘Patient ID’ hyperlink to display the Amend Inpatient Leave screen. The screen displays Leave Details, Previous Bed Details, Nursing Unit Details, Record No Show Details and Other Details.

  5. Enter the details in the mandatory (*) fields. <Related Field Description>

  6. Click [Record] to record and save the amended leave details or [Close] to exit from the screen.

 

• The Inpatient Leave has to be recorded to amend Inpatient leave.

• Cancellation has to be performed within the duration specified in IP Parameter.

• The function displays all the inpatients granted/proceeded on leave from the various Nursing Units controlled by the Operator Station assigned to the logged in user. The user must have rights to Record/Amend Leave.
