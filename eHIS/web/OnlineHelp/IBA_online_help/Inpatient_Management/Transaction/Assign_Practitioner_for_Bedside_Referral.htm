Assign Practitioner for Bedside Referral

From this function you can Associate a Practitioner for Bedside Referrals.

Associating a Practitioner is optional in 'Register Inpatient Referral' function when the Bedside Referral is registered. This function is used to associate a Practitioner for Bedside Referrals. The Practitioner is optional in 'Register Inpatient Referral' function when the Bedside Referral is registered. Practitioner can be assigned after registering Bedside Referral.

Assigning a Practitioner for Bedside Referral

From the IP Transactions > Inpatient Referral, select the Assign Practitioner for Bedside Referral function.

The Assign Practitioner for Bedside Referral screen gets displayed.

  1. Click [Search] to display the list of unconfirmed Bedside Referral patients for whom Practitioner is not assigned or enter the search criteria and click [Search] to select specific Inpatients having unconfirmed Bedside Referral patients for whom Practitioner is not assigned. Click [Clear] to reset the records displayed in the result area.

    Enter the following search criteria to search the records:

    1. Nursing Unit - Enter the Nursing Unit description in alphanumeric format of 15 characters maximum length or select the Nursing Unit using [?] function.

    2. Specialty - Enter the Specialty in alphanumeric format of 15 characters maximum length or select the Specialty using [?] function.

    3. Admission Date From - To - Enter the Admission Date range in date format of dd/mm/yyyy or select the date using the Calendar function.

    4. Encounter ID - Enter the Encounter ID in numeric format of 12 digits maximum length.

    5. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the ID using [?] function.

    Referred To

    Select criteria that correspond to the referred details recorded in the Register Inpatient Referral function, for a Bedside Referral. The query criteria under this section are:

    1. Specialty - Select the Specialty from the list. The Specialty is defined in Application Masters module.

    2. Priority - Select the Priority from the list. The various priorities are Elective, Emergency and Semi Emergency.

    3. Location - Select the Location Type from the list. The values are Clinic and Daycare. The Locations corresponding to the Location Type is displayed. Select a value from the list of Locations.

    4. Practitioner - Enter the Practitioner name or select the Practitioner using [?] function.

    5. Preferred Date - Enter the Preferred Date in dd/mm/yyyy format or select the Preferred Date from the Calendar function.

    6. Referral ID - Enter the Referral ID in alphanumeric format of 14 characters maximum length. The Referral ID Numbering is defined in Application Masters module.

  2. On searching, unconfirmed Bedside Referral patients for whom Practitioner is not assigned are displayed based on the entered criteria.

  3. Click ‘+’ hyperlink to view the Current Encounter Details.

  4. Click the 'Referral ID' hyperlink to display the Assign Practitioner for Bedside Referral screen. The screen displays Encounter Details, Nursing Unit Details and Referral details as recorded during Inpatient Referral Registration.

  5. Enter the Practitioner Name or select the Practitioner using [?] function. The Practitioners associated to the Nursing Unit of the selected patient are listed. This field is mandatory.

  6. Click [Record] to assign the Practitioner to the Bedside Referral or [Close] to exit from the screen.
