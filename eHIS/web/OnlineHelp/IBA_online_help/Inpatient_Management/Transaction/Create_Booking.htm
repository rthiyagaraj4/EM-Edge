Create Booking

From this function you can Create Booking for a patient seeking admission to an Inpatient Location on a preferred date and time.

Booking is an option for planned admission where a facility can schedule Inpatient and Day Care Management resources based on the number of bookings created/confirmed. Booking is performed for registered patients of the facility, unregistered patients (patients without Patient ID) and for referred patients based on the booking parameter rules defined which influences the booking process. A booking reference number generated is used while admitting the patient. If the Operator Station associated with the logged in user allows booking confirmation on creation, the desired Nursing Unit and/or bed details are recorded along with patient and booking details. Only confirmed bookings are listed for admission.

Creating a Booking

From the IP Transactions > Booking, select the Create Booking function.

The Create Booking screen gets displayed.

  1. The screen displays the Booking and Financial Details tabs (if the field ‘Capture Financial Details’ is selected in IP Parameter function).

  1. Enter the Booking Details and Financial Details in the mandatory (*) fields.

Booking Tab

Financial Details Tab

  1. Click [Apply] to save the booking created or [Reset] to clear the entered data.

  2. If you have Operator Station rights to confirm the booking on creation, click [OK].

  3. Click [Cancel] to confirm booking later.

  4. Booking Reference Number is generated on successfully creating a booking.

 

The logged in user must have rights to create a booking. Rights are defined in the Operator Station assigned to the logged in user.
