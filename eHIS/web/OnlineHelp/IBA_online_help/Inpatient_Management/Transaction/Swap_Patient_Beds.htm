Swap Patient Beds

From this function you can Interchange beds between two patients within a Nursing Unit.

The Bed Class associated to the bed must allow swapping and this control is set for each Bed Class using the Bed Class function in IP Code Setup.

The logged in user must have rights to swap beds. The rights are defined in the Operator Station assigned to the user.

Swapping Patient Beds

From the IP Transactions > Transfer, select the Swap Patient Beds function.

The Swap Patient Beds screen gets displayed.

  1. Enter the Nursing Unit in alphanumeric format of 15 characters maximum length or select the Nursing Unit using the [?] function.

  2. Click [Search] to display the bed details of the selected Nursing Unit or enter the search criteria and click [Search] to select a specific record. Click [Clear] to reset the records displayed in the result area.

    Enter the following query criteria:

    1. Practitioner – Enter the Practitioner name in alphanumeric format of 30 characters maximum length or select the Practitioner using the [?] function.

    2. Specialty – Enter the Specialty in alphanumeric format of 15 characters maximum length or select the Specialty using the [?] function.

    3. Admission Date From-To – Enter the Admission Date Range in the format of dd/mm/yyyy or select the date using the calendar function.

  3. On searching, the occupied bed details are displayed based on the entered criteria.

  4. Click ‘+’ hyperlink to view the Current Encounter Details.

  5. Click ‘Select’ check box to select the required beds for swapping.

If the Bed Class of the selected bed forbids bed swapping, you are notified with an alert message. The system allows selection of only two beds for swapping.

  1. Click [Swap Beds] to display the Swap Patient Beds screen. The screen displays the details of the patients associated to the selected bed.

  2. Select the Reason for Swap from the list. The values of the list are defined in the Transfer Type function in IP Code Setup. This field is mandatory.

  3. Click [Confirm Swap] to confirm swapping of beds or [Abort Swap] to cancel the swapping.
