Daycare Unit for Nursing Unit

From this function you can Create an association of one or more Daycare Units to a particular Nursing Unit created for Daycare Patients and Query for the desired Daycare Unit for Nursing Unit records.

A Daycare Unit is similar to a Nursing Unit in terms of bed setup but works like an outpatient location in terms of working hours. A Daycare Unit shares the characteristics of a Nursing Unit as well as a Clinic. The Daycare Unit is set up both as a Clinic and a Nursing Unit.

Creating an association of Daycare Units to Nursing Units

From the IP Setup > IP Code Setup, select the Daycare Unit for Nursing Unit function.

The Daycare Unit for Nursing Unit screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Search] to display Daycare Units and select the check box to associate to the Nursing Unit.

    1. Day Care Unit – The description of the Daycare Units set up in Day Care Unit Master in Outpatient Management module is displayed.

    2. Select – Select the check box to indicate the applicability of the daycare units for the selected Nursing Unit. To select all the displayed daycare units, select the check box provided in the column header. All check boxes are cleared, by default.

  4. Click [Apply] to save the association Daycare Units to a Nursing Unit or [Reset] to clear the entered data.

  5. Click [Query] to search for associated Daycare Unit for Nursing Unit records.
