Advanced Bed Management Setup

From this function you can Create a high-level layout of the nursing unit floor plan that includes rooms and bed layouts within each room.

Advanced Bed Management is a logical location in an inpatient setting to control the layouts of floors, rooms and beds. You can configure the layout based on the existing layout in hospital and visually achieve the same to be used to represent a proper layout.

Creating an Advanced Bed Management Setup

From the IP Setup > IP Code Setup > Nursing Unit Setup, select the Advanced Bed Management Function.

The Advanced Bed Management Setup screen gets displayed.

  1. Select the ‘Setup For’ value from the drop-down list. The listed values are ‘Configure Floor’, ‘Configure Rooms’, ‘Configure Beds’ and ‘Configure Colors’. This is a mandatory field.

  2. Based on the selected Setup For field, the following setups can be done.

 

Configure Floor

  1. Select the ‘Configure Floor’ value from the Setup For list to configure floors of the facility. The  Nursing Unit field gets displayed.

  2. The Nursing Unit which is defined for the facility in the Inpatient Management module gets displayed.

  3. Select the ‘Nursing Unit’ to which the floor has to be configured. An area with blue fill gets displayed.

  4. You can expand or contract the blue area according to the needs.

  1. Click [Apply] to save floor configuration created or [Reset] to clear the entered data.

Once the rooms are configured for the floor, the floor size cannot be modified.

Configure Room

  1. Select ‘Configure Rooms’ option to configure rooms of the Nursing Unit.  

  2. Select the required‘ Nursing Unit’ from the list. This is a mandatory field.

  3. The bubbles that are yet to be associated to the Nursing Unit gets displayed in the left pane.

  4. Drag and drop the bubbles to layout boundary to configure rooms for the respective Nursing Units. The bubbles can be expanded or contracted according to the needs. The standard size of a bubble cannot be modified.

  1. Click [Apply] to save room configuration created or [Reset] to clear the entered data.

The system will not allow you to reduce the room/cubicle bubble size once Beds are assigned to it.

Configure Beds

  1. Select the ‘Configure Beds’ option to configure beds of the Nursing Unit, the Nursing Unit and Room/Cubicle No fields gets displayed.

  2. Select the Nursing Unit from the list. This is a mandatory field.

  3. Select the Room/Cubicle No value from the list. The rooms for which the beds are yet to be associated gets listed. This is a mandatory field.

  4. The rooms that are to be associated to the Nursing Unit gets displayed in the left pane.

  5. Drag and drop the rooms to layout boundary to configure rooms for the respective Nursing Units. The rooms can be expanded or contracted according to the needs.

  6. The standard size of the room is minimum width of 170 pixels and minimum height of 110 pixels to accommodate at least one bed in the room. This minimum size cannot be modified.

  1. Click [Apply] to save bed configuration created or [Reset] to clear the entered data.

The system will allow you to increase/decrease the bed bubble size if there enough empty space in the layout.

Configure Colors

  1. Select the ‘Configure Colors’ option to configure colors for each status of rooms available. The colors can be configured for statuses which are configured in the Inpatient Management module.

  2. Select the room status, the color palette gets enabled. Select the color for the particular status.

  3. Text Color – Select Black/White option to configure the text color inside the bubbles.

  4. Beds – Height and Width of beds can be configured by entering the respective values in the text box. The minimum and maximum range is displayed beside the text box. The values entered should be within the mentioned range.

  1. Click [Apply] to save color configuration created or [Reset] to clear the entered data.

 

Rules for positioning rooms and beds in the layout boundary

  1. You can drag and drop a bubble into the layout boundary.

  2. You can resize the bubble as required, by dragging the right and bottom side of the bubble.

  3. The bubble can be moved within the free spaces of the layout boundary by dragging the title bar of the bubble.

  4. You cannot move bubble beyond the layout boundary.

  5. Two bubbles should not overlap each other.

  6. You can remove a bubble from layout boundary by clicking X icon on top right corner of the bubble. On clicking X  icon, the particular bubble is removed and moved back to the left pane for repositioning.

  7. You cannot position a bubble in a place that cannot accommodate the standard size of the bubble.

  8. It is not mandatory that all the bubbles needs to be positioned within the layout boundary, you can allocate a bubble allocate the bubble at the later stage.

  9. Once a bubble is moved into the layout boundary, then the same cannot be displayed in Left pane.
