Room/Cubicle for Nursing Unit

From this function you can Create an association of one or more Rooms/Cubicles to a particular Nursing Unit and Query for Rooms/Cubicles associated to a Nursing Unit.

To associate beds to a Nursing Unit, it is essential to associate rooms or cubicles to a Nursing Unit. Attributes like Level of Care is set to Rooms/Cubicles associated to a Nursing Unit providing the flexibility to support various Levels of Care within a single Nursing Unit.

Creating an association Of Rooms/Cubicles To Nursing Units

From the IP Setup > IP Code Setup, select the Room/Cubicle for Nursing Unit function.

The Room/Cubicle for Nursing Unit screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Search], Rooms/Cubicles are displayed. Select a Level of care from the list and select the check box against the Room/Cubicle to associate it to the Nursing Unit.

    1. Room/Cubicle No – This field displays the Room/Cubicle Number of the Room/Cubicle.

    2. Room/Cubicle Description – This field displays the Room/Cubicle Description.

    3. Level of Care – This field displays the Level of Care associated to the Nursing Unit selected. If the Level of Care is recorded when the Nursing Unit is set up, the same is set as default. If the Level of Care is not recorded during creation of Nursing Unit, all the values defined in the Level of Care function is displayed and is available for selection.

    4. Enabled – Select the check box to select the Room/Cubicle to be associated to the Nursing Unit. The check boxes are cleared, by default.

  4. Click [Apply] to save the association of Room/Cubicle to Nursing Unit or [Reset] to clear the entered data.

  5. Click [Query] to search for associated Room/Cubicle for Nursing Unit records.

 

If Daycare Unit is associated to a Nursing Unit, association of Room/Cubicles to the Daycare Unit is possible.
