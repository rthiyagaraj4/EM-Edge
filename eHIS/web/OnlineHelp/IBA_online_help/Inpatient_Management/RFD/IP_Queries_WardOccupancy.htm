Enter the following search criteria:

  1. Nursing Unit [From-To] – Specify the nursing unit range. Select the nursing unit using [?] function or enter the required nursing unit.

  2. Include for Hospital Calculation – Select the check box to include nursing units for hospital calculations. On selecting the box, the nursing units for which the ‘Include for Hospital Occupancy Calculations’ field is checked during creation (IP code setup > Nursing Unit) is included in the search result.

  3. Governate – The drop-down list displays all governorate data as configured including ‘All Governates’ as default option.  Select the required governate from the list. This is applicable only for users in MOH-EGY.

  4. Facility – The drop-down list displays all facilities linked to the selected governorate. The default value is ‘All Access Facilities’ even if the user selects a specific governorate. Select the required facility from the list. This is applicable only for users in MOH-EGY .
