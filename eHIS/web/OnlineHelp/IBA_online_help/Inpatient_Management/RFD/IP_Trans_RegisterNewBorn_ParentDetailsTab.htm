Enter the Father Details:

 

Father Details

  1. Patient ID – Enter father Patient ID or select it using [?] function. On entering the father Patient ID, all demographics recorded during his registration are displayed in the respective fields as read-only text. On entering the father Patient ID, the system verifies the age with that of the minimum age set for father patient in MP Parameter setup.

  2. Name Details – The Name of the father is populated on entering the Patient ID or enter the father’s name details.

  3. Citizen ID - Enter the Citizen ID of the father in the format as specified in MP Parameter.

  4. Other Alt ID – Enter the Other Alt ID of the father in the format specified in MP Parameter.

  5. Previous HN – Enter the Previous hospital number of the father in the format as specified in MP Parameter.

  6. Nationality - Select the Nationality from the list.

  7. Race - Select the required Race from the list.

  8. Govt Employee – Select the check box to indicate whether the father is a government employee.

  9. Religion – Select the Religion from the list.

  10. Occupation Class - Select the Occupation Class of the father from the list.

  11. Occupation - Select the Occupation of the father from the list. The listed values correspond to the selected Occupation Class. If the occupation is not available, enter the same in alphanumeric format of 15 characters maximum length.

  12. Remarks - Enter the Remarks in alphanumeric format of 100 characters maximum length.

The fields and their order displayed in the screen may vary depending on the MP Parameter Setup.
