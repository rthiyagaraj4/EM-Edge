1. Authorized Practitioner – Enter the Authorized Practitioner in the alphanumeric format, or select the Practitioner using the [?] function. This field is mandatory. The Practitioners listed are filtered based on the Specialty to which the patient belongs. The Authorized Practitioner is displayed if it was recorded.

  2. Discharging Practitioner – Enter the Discharging Practitioner in the alphanumeric format or select the Practitioner using the [?] function. This field is mandatory. The Practitioners listed are filtered based on the Specialty to which the patient belongs.

  3. Discharge Type – The Discharge Type selected while preparing the discharge advice is displayed and disabled. This is a mandatory field.

  4. Discharge Status - The Discharge Status captured in Discharge Advice function is displayed.

  5. Expected Discharge Date/Time – The Expected Discharge Date and Time entered while preparing the discharge advice is displayed. The date can be changed, but it must be greater than or equal to the current system date. This field is mandatory.

  6. Discharge Date/Time – The current system date and time is displayed as the Discharge Date and Time. The date can be changed, but it must be greater than or equal to the current system date. This field is mandatory.

  7. Reason For Late Discharge  – Select the reason from the drop-down list.

  8. Other Reason  – This field is enabled if ‘Others’ is selected in the ‘Reason For Late Discharge’ field. You can enter the reason for late discharge to a maximum of 200 characters.

  9. Remarks – Enter the Remarks at the time of discharge.

 

Deceased Details

  1. Deceased – If the selected Discharge Type is ‘Deceased’ while preparing the discharge advice, the check box is selected and disabled.

  2. Death Date/Time – The Date and Time of death is displayed if it was recorded. If the Discharge Type is ‘Deceased’, this field is enabled and becomes mandatory.

  3. Post Mortem Requested – This field gets displayed as selected, if ‘Postmortem Requested’ check box was selected while preparing the discharge advice. The field is enabled only if the selected Discharge Type has ‘Deceased’ indicator selected during its creation.

  4. Body Released to Mortuary - Select the date and time as to when the body of the deceased patient is transferred to mortuary using the calendar function. This field is enabled only if the Discharge Type of the patient is recorded as ‘Deceased’.

  5. High Risk Body – Select the check box to indicate that the discharged Inpatient body is a high risk body.

 

MLC Details

  1. MLC – Select the check box to indicate the encounter of the patient is a Medico Legal Case.

  2. Police Report No. – Enter the Police Report Number in alphanumeric format of 30 characters maximum length. This field is enabled only if the ‘Medico Legal Case’ check box is selected.

  3. Station – Enter the Police Station where the case has to be registered in alphanumeric format of 30 characters maximum length. This field is enabled only if the ‘Medico Legal Case’ check box is selected.

  4. Police ID - Enter the Police ID in alphanumeric format of 30 characters maximum length. This field is enabled only if the ‘Medico Legal Case’ check box is selected.
