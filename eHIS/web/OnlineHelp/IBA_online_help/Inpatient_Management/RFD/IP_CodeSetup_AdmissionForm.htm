Enter the following details to create a new Admission Form:

 

  1. Code - Enter the Code in alphanumeric format of two characters maximum length. This field is mandatory.

  2. Long Description - Enter the long description of the admission form in alphanumeric format of 30 characters maximum length. Special characters are allowed. This field is mandatory.

  3. Short Description - Enter the short description of the admission form in alphanumeric format of 15 characters maximum length. Special characters are allowed. This field is mandatory.

  4. Specialty – Select the required specialty from the drop-down list. Only one specialty can be mapped at a time.

  5. By Default - Select the check box to make the created admission form selected by default in the 'PF' field in Admit Patient function. This check box is cleared by default.

  6. Enabled - Select the check box to enable the admission form or clear the check box to disable the admission form. The check box is selected, by default.
