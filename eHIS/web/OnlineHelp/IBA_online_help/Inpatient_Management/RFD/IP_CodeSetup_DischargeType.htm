Enter the following details to create a new Discharge Type record:

 

  1. Code - Enter the Code in alphanumeric format of two characters maximum length. The code has to be unique within the Facility and cannot be changed after creation. This field is mandatory.

  2. Long Description - Enter the long description of the Discharge Type in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description - Enter the short description of the Discharge Type in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Blocking Type – Choose the Blocking Type applicable for the Discharge Type defined. The values in this field are populated from Blocking Type master. This ensures that the bed from which the patient is discharged is blocked, once the patient discharge process is complete with the discharge type chosen as the one defined.

  5. Discharge Type Indicator – Select the Discharge Type Indicator for the Discharge type defined. The values in the list are Absconded, At Own Risk, Deceased, Discharged/Home and Transfer to Other Hospital.

  6. Maternal Death – Select the check box if the Discharge Type is applicable to maternal death. The check box is enabled only if the value Deceased is selected as the Discharge Type indicator.

  7. Enabled - Select the check box to enable the Discharge Type.
