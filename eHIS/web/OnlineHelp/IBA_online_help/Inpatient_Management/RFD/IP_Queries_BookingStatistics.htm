Enter the following search criteria:

  1. Bookings - Select the booking from the list. The booking can be either daily or monthly. By default, ‘Daily’ is displayed.

  2. Group By - Select the group by from the list. Group by Medical Service Group or Nursing Unit.

Based on the value selected in the Group By column, relevant fields are displayed.

Group By - Medical service group

  1. Medical Service Group - Select the Medical Service Group using [?] function.

  1. Preferred Date [From-To] – Enter the Preferred date/month. Use date format dd/mm/yyyy for Daily Booking and month format of mm/yyyy for monthly booking. The field legends are displayed as either preferred date or preferred month depending on the value selected in the ‘Booking’ field. If the value in the booking field is selected as ‘Daily’, the ‘Preferred Date’ legend is displayed, if the value is selected as ‘monthly’, the ‘Preferred Month’ legend is displayed. This is a mandatory field.

Group By- Nursing Unit

  1. Nursing Unit – Select the Nursing Unit using [?] function.

  1. Bed Class – Select the Bed Class using [?] function.

  2. Preferred Date [From-To] – Enter the preferred date/month. Use Date format dd/mm/yyyy for Daily Booking and month format of mm/yyyy for monthly booking. The field legends are displayed as either preferred date or preferred month depending on the value selected in the ‘Booking’ field. If the value in the booking field is selected as ‘Daily’, the ‘Preferred Date’ legend is displayed, if the value is selected as ‘monthly’, the ‘Preferred Month’ legend is displayed. This is a mandatory field.
