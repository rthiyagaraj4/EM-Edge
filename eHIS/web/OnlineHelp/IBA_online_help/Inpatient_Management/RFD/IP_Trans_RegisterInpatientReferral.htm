Referral Details

 

  1. Bed-side Referral – Select the check box if a bedside referral is to be requested for the patient.

  2. Referral Type – Select the referral type from the list of valid values. The listed types are Login Facility, Enterprise and External. This field is mandatory. On selecting the Bedside Referral check box, Login Facility gets displayed in the field and is disabled.

  3. Healthcare Setting Type – Select the healthcare setting type from the list to which the patient has to be referred to. This field is mandatory. Healthcare setting types are set up using Application Masters module. The values listed in this field are specific to healthcare setting types for IP Referral. If the Bedside Referral check box is selected, the healthcare facility type associated with the login facility is defaulted and disabled. If the referral type selected is Enterprise, all the healthcare setting types defined in the healthcare setting type master function is displayed here except for the one associated with the logged in facility. If the referral type selected is External, all the healthcare setting types to which External Referral Sources associated in the External Referral Source master function are displayed here.

  4. Referred To – If the Referral Type is selected as the Enterprise, the entire active Facilities of the enterprise except the logged in Facility are listed. If the Referred Type is selected as External, all the external referral sources are listed. If the referred type is selected as Login Facility, the logged in facility name is made default. This field is mandatory. Select the required value. If the Bed Side Referral check box is selected, the referred source gets defaulted to the logged in facility name and disabled. Place the mouse on the icon to display the long name of the External or Referral Source recorded in ‘External/Referral Source’ function of Application Masters module. The  icon appears on the screen if the long name for the External Referral Source is recorded in ‘External/Referral Source’ function of Application Masters module.

  5. Specialty – Enter the Specialty in alphanumeric format of 30 characters maximum length or click [?] function to select the Specialty. This is a mandatory field.

If Referral Type is ‘External’, the Specialty field is not mandatory.

  1. Location Type – Select the Location Type from the list. The pre-defined values are listed in the list. This field is mandatory.

  2. Location – If the Referral Type is ‘External’, the location list gets disabled and you have to specify the location in alphanumeric format of 30 characters maximum length in the text box provided. If the referral type selected is ‘Enterprise’, the location type and the location can be identified, while the adjoining text box does not allow any data entry. The values for the location type field are ‘Clinic’, ‘Procedure Unit’ or ‘Day Care’. Depending on the Healthcare setting type, referred source, specialty and location type selected, corresponding locations are populated in the list. You can select the required value from the list. The same holds good if the referral type selected is ‘Login Facility’. This is a mandatory field. If ‘Bedside Referral’ is selected, this field is disabled.

  3. Service – Select the required Service from the list. If the referral type is selected as ‘Login Facility’, the services are listed based on the Specialty, Location Type and Location chosen. If the referral type chosen is ‘Enterprise’, the services are listed based on the Healthcare Setting Type, Referred Source, Specialty, Location Type and Location chosen. If the referral type chosen is ‘External’, all the services set up in the ‘Service’ master function is available in the list. If ‘Bedside Referral’ is selected, this field is disabled.

  4. Practitioner – Enter the Practitioner name or select the practitioner using the [?] function. The list values correspond to the selected ‘Referred Source’. This field is enabled only if the referral type is ‘Enterprise’ or ‘Login Facility’ and if the specialty is selected, or it is disabled. For the Referral type ‘External’ the practitioner name can be entered in the text box provided in alphanumeric format of 30 characters maximum length. If the check box ‘Bedside Referral’ is selected, the ‘Practitioner’ field becomes mandatory.

  5. Tel No - The telephone number of the Practitioner is displayed based on the value recorded for the respective Practitioner. This is a display field only.

  6. Mobile No - The mobile number of the Practitioner is displayed based on the value recorded for the respective Practitioner. This is a display field only.

  7. Pager No - The pager number of the Practitioner is displayed based on the value recorded for the respective Practitioner. This is a display field only.

  8. Referred Date – This is a display field. The Referred Date is defaulted to the system date/time.

  9. Preferred Date – Enter Preferred Date in dd/mm/yyyy format or select the date from the Calendar icon. This field is mandatory. The user can enter preferred date greater than or equal to the system date. If the check box ‘Bedside Referral’ is selected, ‘Preferred Date’ field becomes mandatory.

  10. Reason for Referral – Enter the referral reason for the patient or select the reason from the [?] function.

  11. Priority – Select the Referral Priority from the list. The values displayed are ‘Elective’, ‘Urgent’ and ‘Emergency’. This field is mandatory.

  12. Referring Practitioner –  This field will be displayed only if you have selected the check box Bedside Referral in the same screen. Click [?] and select a practitioner from the lookup.  The default practitioner displayed in this field is based on the following conditions:

a) Attending Practitioner – The name of the attending practitioner will be displayed by default, if the value ‘Attending Practitioner’ is selected in the Default Referring Practitioner in Bedside Referral field of the IP Parameter screen.

b) Logged-in Practitioner - The name of the logged-in practitioner will be displayed by default, if the value ‘Login Practitioner’ is selected in the Default Referring Practitioner in Bedside Referral field of the IP Parameter screen and the logged-in user is a practitioner.

c) Blank Value – A blank value will be displayed by default, if the value ‘Login Practitioner’ is selected in the Default Referring Practitioner in Bedside Referral field of the IP Parameter screen and the logged-in user is a practitioner. In this case, you must select a value from the lookup.

All the practitioners will be displayed in the “Referring Practitioner” lookup.

  1. Referral Notes - Enter the Referral Notes of the Patient in alphanumeric format of 1500 characters maximum length. To enter the referral notes, click [Referral Notes]. Enter the referral notes and close the referral notes screen. The control shifts back to the main screen.

  2. Record Referral Letter Note – This button appears only if the ‘Enable Recording of Referral Letter Note with Referral Request’ check box is selected in IP Parameter. Click [Record Referral Letter Note] to invoke the Record Referral Letter dialog where you can record new referral letter or search and modify the existing referral letters available for the patient. This button is disabled when the ‘Referral Type’ is selected as ‘External Facility’ and it is enabled only when the ‘Referral Type’ is selected as Login Facility or Enterprise.
