Enter the following details to amend the leave:

 

Leave Details

  1. Transaction Type – Select the Transaction Type from the List. The list values are populated based on the leave status of the patient.

  • Record Return – Select the option to record the return or readmit the patient on returning from the leave.

  • Revise Leave – Select the option to revise (extend) the leave period of the selected patient.

  • Record No Show – The option gets displayed in the list only when the selected patient record has exceeded the granted leave period. This can be selected when the inpatient has not returned back to facility even after the granted leave period is exceeded and continues to block the resources.

  • Cancel Leave – The option gets displayed within the cancellation period allowed as set in IP Parameter (Allow Leave Cancellation within [] min’s).

  1. Admission Date/Time – The Admission Date and Time is displayed as read-only text.

  2. Practitioner – The Practitioner who authorized the leave for the selected patient is displayed as read-only text.

  3. Release Bed – If the Bed is released while recording leave, the field is selected and disabled.

  4. Pseudo Bed – The Pseudo Bed assigned to the patient in the function ‘Record Inpatient Leave’ is displayed.

  5. Leave Period From-To Date Time – The leave period as specified in the function ‘Record Inpatient Leave’ is displayed. This field is enabled only when the Transaction Type is selected as ‘Revise Leave’. Enter the revised period in the format of dd/mm/yyyy hh:mm. The entered period must adhere to the rules set in IP Parameter for maximum allowed leave period.

  6. Leave Remarks – The Remarks entered in the function ‘Record Inpatient Leave’ are displayed. You cannot modify the remarks.

 

Previous Bed Details

  1. Bed No – The Bed assigned to the patient before proceeding on leave is displayed.

  2. Room/Cubicle No – The Room/Cubicle Number assigned to the patient before proceeding on leave is displayed.

 

Nursing Unit Details

The Nursing Unit details can be changed only if the Transaction Type is Record Return/Cancel Leave and bed is released on leave or the details are displayed and disabled.

  1. Nursing Unit – The Nursing Unit to which the patient is associated is displayed and cannot be changed.

  2. Bed No – The Bed Number of the Inpatient at the time of issuing leave pass is displayed. If bed is released, select the required bed in the Nursing Unit. You can also select the bed using the [?] function. On selecting the required bed, the bed details are populated on to the Nursing Unit details column. The lookup function is enabled, if the bed is released on leave and Transaction Type is ‘Record Return’/‘Cancel Leave’.

  3. Room/Cubicle No – The Room/Cubicle Number is displayed on selecting the Bed Number.

  4. Bed Class – The Bed Class is enabled when the Transaction Type is set as Record Return/Cancel Leave and bed is released on leave. Select the required Bed Class from the list

  5. Bed Type - The Bed Type is enabled when the Transaction Type is set as Record Return/Cancel Leave and bed is released on leave. Select the required Bed Type from the list.

 

Record No Show Details

  1. Absconding – Select the check box if the patient has left the facility without prior notice and has not returned. This field is enabled only if the Transaction Type selected is ‘Record No Show’. On selecting the check box the Date of Absconding field becomes mandatory.

  2. Date of Absconding – Enter the date in the format of dd/mm/yyyy hh:mm. This field is enabled only if the ‘Absconding’ check box is selected.

  3. Informed Others – Enter the informed and absconding details in the area provided. This field is enabled only if the ‘Absconding’ check box is selected.

  4. Transaction Remarks - Enter the Transaction Remarks for Inpatient leave amendment in alphanumeric format of 500 characters maximum length. This field is enabled for all the Transaction Types.

  5. Date/Time Returned – Enter the date and time in the format of dd/mm/yyyy hh:mm. The patient’s return date and time is recorded. This field is enabled and becomes mandatory only when the Transaction Type is selected as Record Return. On selecting the Transaction Type as ‘Record No Show’, this field is displayed as ‘Record No Show Date/Time’ and is mandatory. This field is not mandatory if the ‘Absconding’ check box is selected.

 

Other Details

  1. Informed To - The Informed To option selected during the 'Record Inpatient Leave/Abscond' is displayed.

  2. Name - The Name entered during the 'Record Inpatient Leave/Abscond' is displayed.

  3. Date/Time - The Date and Time entered during the 'Record Inpatient Leave/Abscond' is displayed.

  4. Police Report No – The Police Report Number entered during the 'Record Inpatient Leave/Abscond' is displayed. This is applicable only for absconding patients.

  5. Station – The name of the Police Station is displayed. This is applicable only for absconding patients.

  6. ID – The ID is displayed. This is applicable only for absconding patients.
