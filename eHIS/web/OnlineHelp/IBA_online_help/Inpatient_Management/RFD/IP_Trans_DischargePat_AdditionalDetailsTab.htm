Bed Block Details

  1. Block Type – The Block Type selected in the ‘Prepare Discharge Advice’ function is displayed.

  2. Block Period – The default Block Period associated with the Block Type in the 'Block Type' function is displayed.

  3. Bed Blocked Until - The date and time until when the bed is blocked after the discharge of the patient is displayed. The date/time is calculated based on the value entered in the ‘Block Period’ field.

  4. Remarks - Enter the Remarks in alphanumeric format of 500 characters maximum length. This field is mandatory.

The Beds that are reserved for a patient are released while discharging.

 

Informed Details

  1. Informed To – Select the required value from the list. Select the person who must be informed about the patients discharge . The listed values are ‘Next of Kin’, ‘First to Notify’ and ‘Others’.

  2. Name – Enter the name of the person to be informed of the patient’s discharge in alphanumeric characters of length 20 characters. This field is displayed as recorded on selecting the ‘Informed To’ field as ‘Next of Kin’ or ‘First to Notify’ which can be edited.

  3. Date/Time – Once the ‘Informed To’ field is selected, this field is enabled and becomes mandatory. Enter the date of informing or select the date using the Calendar function.

 

Absconding Details

  1. Absconding - Select the check box for patients missing from the facility or those who have gone on leave and not yet returned. This check box is selected by default if the absconding details are recorded while preparing the discharge advice.

  2. Date/Time – Enter the date/time from when the patient has been absconding or select the date/time using the calendar function. This field is displayed if it was recorded while preparing the discharge advice.

  3. Informed Others - Enter the name of the person to whom this information has been reported to in alphanumeric characters of length 20 characters. This field is displayed if it was recorded while preparing the discharge advice.
