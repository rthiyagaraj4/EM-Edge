Enter the following details to create a new Transfer Priority record:

 

  1. Code – Enter the Code in the alphanumeric format of two characters maximum length. This field is mandatory.

  2. Long Description – Enter the long description of the priority in the alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description - Enter the short description of the priority in the alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Color Code Scheme – Choose the required color for priority using the color palate . On invoking the color palate, a color scheme table gets displayed. Each transfer priority can be assigned a unique color and the system does not permit reusage of a particular color to another Transfer priority record.

  5. Expiry Required – Select the check box if the created priority is expiry based.

  6. Expiry Period - Enter the expiry period in the numeric format of three digits maximum length. This field is enabled on selecting the Expiry Required check box. The expiry period is set as Hours or Days. This field is mandatory, if Expiry Required check box is selected.

  7. Enabled - Select the check box to enable the priority  or clear the check box to disable the Priority. The check box is selected, by default.
