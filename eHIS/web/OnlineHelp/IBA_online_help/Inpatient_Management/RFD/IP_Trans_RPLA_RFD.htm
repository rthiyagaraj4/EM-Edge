Enter the following details to record Leave/Abscond for the patient:

 

  1. Record – Select the required value from the list. The listed values are ‘Leave’ and ‘Abscond’. If leave has been granted to the patient, select ‘Leave’ or if the patient is missing from the Nursing Unit, select ‘Abscond’. This field is mandatory.

  2. Practitioner – Select the practitioner authorizing the leave using the [?] function. This field is mandatory. The treating Practitioner is displayed, by default. The lookup function displays the Practitioners belonging to the Specialty of the Nursing Unit. The attending Practitioner is displayed and disabled for absconding Inpatients.

  3. Admission Date/Time – The Admission Date and Time of the Inpatient is displayed.

 

Leave Details

If the Nursing Unit specifies releasing bed on leave, the pseudo beds must be created for the Nursing Unit, or the occupying bed is not released.

If ‘Release Bed on Leave’ is specified in the IP Parameter, two fields get displayed, namely, ‘Release Bed’ and ‘Pseudo Bed.

  1. From Date/Time-To Date/Time – Enter the leave duration by specifying the From and To date and time in the format of dd/mm/yyyy hh:mm. The entered ‘From’ date and time cannot be less than the admission date and time. The ‘To’ date entered must be within the permissible leave duration set in the IP Parameter (Max Allowable Leave Period [] hrs). The From Date is displayed as the current date and the To Date is displayed as specified in the IP Parameter, by default. Leave period entry is mandatory. Based on the action specified for exceeding maximum leave, the system either displays an error message or a warning message. The current system date and time is displayed as From Date for abscond patients. The To Date is not displayed for absconding Inpatients. The ‘To’ date is made default to the current system date and time and displayed as read-only text for patients who are absent.

  2. Remarks – Enter the Remarks for Inpatient leave in alphanumeric format of 500 characters maximum length. This field is mandatory.

 

Informed Details

  1. Informed To – Select the required value from the list. Select the person who must be informed about the discharge of the Inpatient. The listed values are ‘Next of Kin’, ‘First to Notify’ and ‘Others’.

  2. Name – Enter the Name of the person who is informed about the patients discharge in alphanumeric characters of length 20 characters. This field is enabled only if a value is selected in ‘Informed To’ field. On selecting the ‘Informed To’ field as ‘Next of Kin’ or ‘First to Notify’, the name of the person is displayed as recorded while registering the patient.

  3. Date Time – Once the ‘Informed To’ field is selected, this field is enabled and becomes a mandatory field. Enter the date or select the date using the calendar function.
