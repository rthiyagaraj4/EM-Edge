Enter the following details to create a new Blocking Type record:

 

  1. Code - Enter the Code in alphanumeric format of two characters maximum length. The Blocking Type code has to be unique and cannot be changed after creation. This field is mandatory.

  2. Long Description - Enter the long description of the Blocking Type in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description - Enter the short description of the Blocking Type in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Blocking Period – Enter a Blocking Period in numeric format of three characters maximum length. The unit of measure for the blocking period can be specified using the list provided. The values in the list are Hours and Days. The blocking period ensures that for the blocking type defined the duration of blocking, for a bed, cannot be greater than the value entered in this field.

  5. Override - Select the check box, if the blocking type can be overridden during admission/transfer. If the check box is selected, the bed blocked using this blocking type can be overridden by another booking/blocking transaction in the blocking transaction. If the check box is cleared, it indicates that the Blocking Type cannot be overridden and the bed has to be released using the Release Blocked Bed function. The check box is cleared, by default.

  6. Enabled - Select the check box to enable the Blocking Type.
