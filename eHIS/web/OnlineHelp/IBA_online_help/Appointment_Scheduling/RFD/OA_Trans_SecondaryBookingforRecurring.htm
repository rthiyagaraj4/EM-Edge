Enter the following details to search for the required record:

 

  1. Location Type – Select the Location Type from the list of valid values. The list displays Clinic, Daycare Unit and Procedure Unit.

  2. Location – Select the required location, where the patient is scheduled for an appointment from the [?] function. The locations are listed based on the selected Location Type.

  3. Resource - Select the required Resource against whom the Patient is scheduled for an appointment.

  4. Resource Type – The Resource Type is displayed on selecting the Location.

  5. Appt No - Enter the Appointment Reference Number in alphanumeric format of eight characters maximum length or the Appointment Reference Number gets displayed on entering or selecting a Patient ID.

  6. Appt Date [From/To] - Enter the date range for which the appointments are scheduled for Patient in dd/mm/yyyy format or click the calendar function to search for the required date. This field is mandatory.

  7. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or click [?] function to search for the required Patient ID.

  8. Patient Name - Enter a valid Patient Name in alphanumeric format. If a valid Patient ID is entered or selected, system displays the Patient Name corresponding to the entered or selected Patient ID.

  9. Gender – Select the required Gender from the list of valid values. The list of values includes Male, Female and Unknown.
