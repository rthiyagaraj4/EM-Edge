Enter the following details to modify the appointment waitlist:

 

  1. Specialty – The specialty against which the wait list was generated for the patient is displayed.

  2. Category – Select the required list category using the [?] function. All the waitlist category defined for the specialty in ‘Wait List Category For Specialty’ of Outpatient Management module are displayed.

  3. Catalog – Select the required list category using the [?] function. All the catalog associated to the specialty in ‘Specialty for catalog’ in the Application Masters module are displayed.

  4. Waitlist No – The Waitlist number of the patient is displayed.

  5. Location – Select the required location from the list. All the location(s) for the displayed specialty are listed.

  6. Resource – Enter the resource name to which the appointment waitlist has to be modified in alphanumeric format or click [?] function to search for the required resource.

  7. Preferred Date – Modify the preferred date for wait list in dd/mm/yyyy format or click the calendar icon to search for the required date.

  8. Status – Select the required status from the list box. The listed values are ‘Urgent’, ‘Semi Urgent’ and ‘Normal’.

  9. Reference No - Enter the Reference Number of the patient referred to the hospital in alphanumeric format of 12 characters maximum length. The Reference Number helps in identifying the patients referred to the hospital for appointment from the list of patients. You can modify a Reference Number recorded earlier. If an appointment is booked for a preferred date, where no slots are available, a waitlist number is created for the preferred date. You can associate the reference number to the patient.  

  10. Reason for Modification – Select the reason for modification from the list. This field is mandatory. All the enabled reasons created in ‘Application Masters’ module are displayed.
