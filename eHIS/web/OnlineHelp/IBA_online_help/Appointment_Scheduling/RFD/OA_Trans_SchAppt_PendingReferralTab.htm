Pending Referral

This Tab displays referral patient registered through the Outpatient Management module. You can book an appointment for a Referral Patient.

  1. Enter the referral details and click [Search] to search to a particular referral patient or click [Search] without entering the referral details in the fields provided to display all referral patients registered in the facility.

    Referred From

    1. Referral Type - Select the Referral Type from the list of valid values as 'Login Facility', 'External' or 'Enterprise'. Depending on the selected value, select the value for the referral source.

    2. Referral Source - Select the range of Referral Sources. The corresponding referral sources are available for selection based on the Referral Type selected.

    3. Specialty - Select the Specialty from the list of valid values. The listed values are defined in 'Application Masters' module.

    4. Service - Select the Service corresponding to the selected specialty from [?] function.

    5. Location - Select the Location Type from the list of valid values as either 'Clinic' 'Nursing Unit', 'Procedure Unit' or 'Daycare'. Depending on the selected value, select the Location from the list of valid values.

    6. Practitioner - Enter the Practitioner name in alphanumeric format of 30 characters maximum length or click [?] to search for the required practitioner.

    Referred To

    1. Specialty - Select the Specialty from the list of valid values.

    2. Service - Select the Service corresponding to the selected Specialty from the list of valid values.

    3. Location - Select the Location Type from the list of valid values as either 'Clinic' or 'Nursing Unit'. Depending on the selected value, select the location from the list of valid values.

    4. Practitioner - Select the Practitioner from [?] function for the selected specialty.

    5. Patient ID - Enter Patient ID in alphanumeric format of length as specified in MP Parameter setup or select the patient ID using the search [?] function.

    6. Referral ID - Enter the Referral ID in alphanumeric format of length 14 characters.

    7. Priority - Select the Priority as either 'Elective', 'Semi Emergency', 'Emergency' or 'Referral for Admission'.

    8. Status - Select the Status from the list of valid values.

    9. Old Referrals - Select the check box to view details of referrals made before current system date.

    10. Search Name with Local Lang - Select the check box, to search patient name in local language. On selecting the check box 'Legend for Loc Language' and 'Patient Name Derivation.

  2. The Referral Details of the referral patients are displayed.

    The query result displayed is categorized based on the priority as Emergency, Regular and Urgent. The other details displayed are:

    1. + - The plus sign is hyperlink which displays the Referral Details.

    2. Status - The Status of the patient is displayed.

    3. Referral ID - The Referral ID of the referred patients are displayed. The IDs are hyperlinked. To update or view the patient record click on the hyperlink.

    4. Referred On - The Referred Date of the patient is displayed in date format of dd/mm/yyyy.

    5. Preferred Date - The Preferred Date of the patient is displayed in date format of dd/mm/yyyy.

    6. Patient Details - The Patient Details displayed are ID of the patient, name, gender, and date of birth.

    7. Referred From - The Referral Source and the specialty are displayed.

    8. Referred To - The practitioner to whom the patient is referred to is displayed.

    9. Notes - The Referral Notes entered for the patient during referral is displayed on clicking the hyperlinked 'Notes'. The user cannot edit it. The screen as shown below will be displayed on clicking the hyperlink 'Notes'.

  3. Click the Referral ID hyperlink to schedule an appointment for the referral patient.

  4. The Schedule Status tab is displayed to continue scheduling an appointment for the referral patient.
