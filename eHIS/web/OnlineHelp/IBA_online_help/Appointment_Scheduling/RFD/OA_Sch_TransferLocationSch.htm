Enter the following details to Transfer the Location Schedules:

 

  1. Location Type – Select the Location Type as Clinic, Daycare Unit or Procedure Unit.

  2. Location – Select the required Location using [?] function based on the Location Type selected. This field is mandatory.

  3. Resource Type – This field is enabled on selecting a location. Select the Resource Type as Practitioner, Equipment, Room or Other. This field is mandatory.

  4. Resource – Select the required Resource using [?] function based on the Resource Type selected.

  5. By - The Transfer of Schedules can be done in one of the two ways mentioned below:

  6. Date - Select this option to transfer the schedules by date. The Day field gets disabled when this option is selected.

  7. Day - Select this option to transfer the schedules by day.

  1. From-To Date - Enter the date range in dd/mm/yyyy format or select the dates using the calendar function. The field legend is displayed as ‘Period’ when the Date option is selected. The field legend is displayed as ‘Date Range’ when the Day option is selected. From date corresponds to the date from which the Schedules/Appointments has to be transferred. To date corresponds to the date upto which the Schedules/Appointments has to be transferred. This field is mandatory.

  2. From-To Day - Select the Day of Week to which the generated schedules have to be transferred from the list. ‘From’ field corresponds to the day on which the Practitioner/Location is not functioning and Schedules/Appointments are to be transferred to a different day. The ‘To’ field corresponds to the day on which the Location/Resource are not functioning and Schedules/Appointments are to be transferred from an earlier day. This field is mandatory if Day option is selected. The field is disabled when the ‘By’ value is selected as ‘Date’.

  3. Reason - Select the reason for change from the list of valid values. All the reasons applicable for blocking, as set in ‘Other Master setup (Reason)’ of Application Masters module get displayed. This field is mandatory.
