Enter the following details to search for the required record:

 

  1. Location Type – Select the Location Type as Clinic, Daycare Unit or Procedure Unit. This field is mandatory.

  2. Location – Select the required Location based on the selected Location Type.

  3. Resource - Enter the resource name in alphanumeric format of 30 characters maximum length or select the resource to whom the schedule have to be assigned using [?] function.

  4. Resource Type – The Resource Type is displayed. Resource Type is displayed if Location is selected.

  5. Reason for Cancellation - Select the Reason For Cancellation from the list of valid values. All the reasons applicable for cancelling as set in ‘Other Master setup (Reason)’ of Application Masters module gets displayed. This field is mandatory.

  6. Appt Date [From-To] - Enter the date range for which the appointments are scheduled for Patient in dd/mm/yyyy format. This field is mandatory.
