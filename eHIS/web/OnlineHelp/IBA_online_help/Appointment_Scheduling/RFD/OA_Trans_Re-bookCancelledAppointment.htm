Enter the following details:

  1. Search With – Select the ‘Search With’ value from the list. The listed values are All and Patient ID. This is a mandatory field.

  2. Location - Select the Location Type/Location where the Patient appointment is cancelled from the list.

  3. Resource – This field is enabled on selecting the Location Type/Location value. Select the Resource against the Patient for whom the appointment is cancelled using [?] function. The Resource Type is displayed against the Resource Type field.

  4. Appt No - Enter the Appointment Reference Number in alphanumeric format of eight characters maximum length.

  5. Appt Date [From-To] - Enter the date range in dd/mm/yyyy format to view  cancelled appointments. This field is mandatory only if ‘Search With’ is selected as ‘All’. The current system date is displayed as the Appointment ‘From’ Date. The second consecutive date of the current system date is displayed as the Appointment ‘To’ Date.

  6. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or use [?] function to find out the Patient ID. This field is mandatory only if ‘Search With’ is selected as ‘Patient ID’. The cancelled appointments of the selected Patient ID are displayed.

  7. Patient Name - Enter a valid Patient Name in alphanumeric format of 40 characters maximum length. If valid Patient ID is entered or selected using [?] function, system displays the Patient Name corresponding to the entered or selected Patient ID.

  8. Gender - Select the Gender of the Patient whose cancelled appointment is to be queried from the list of valid values.

  9. Day of Week - Select the required day of week from the list.

  10. Display only Not-identified Practitioner - Select the check box to display only Not-identified Practitioners. This check box is enabled, if 'Open to all Resources' check box is selected in 'Clinic' function of Ambulatory Care Management module.
