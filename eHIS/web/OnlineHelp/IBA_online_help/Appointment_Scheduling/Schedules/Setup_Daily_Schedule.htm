Setup Daily Schedule

From this function you can Create a timetable and generate a schedule at the same time for a specific date and Query for a desired schedule. You can Delete a Daily Schedule record setup.

Schedule generation is done either based on a period or for a single day. Setup Daily Schedule function allows generating a timetable and a schedule at the same time but is done only for a specific date. This is used to generate a schedule, which is different from the routine schedule and is applicable only for a particular date. The daily schedule setup is created for the current day.

Setting up a Daily Schedule

From the OA Schedules, select the Setup Daily Schedule function.

The Setup Daily Schedule screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Schedule created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Daily Schedule setup within the facility.

  5. Click [Delete] to delete a queried record.
