Holiday

From this function you can Define Holidays for locations within a facility.

Holiday indicates non-availability of certain resources for scheduling. Schedules are not generated for locations marked as holidays. Holidays are generated for a Location for a specific period with a reason. A holiday is allowed to be cancelled if required. The system does not allow entering Holidays for schedules that have been generated. In this case, to mark a holiday, the Schedules have to be cancelled and regenerated. Holidays are displayed in pink color in Schedule Appointment screen. The reason for Holiday is displayed as tool tip. The holiday screen has two tabs, namely, ‘Existing Holiday’ and ‘Holiday for New Location’.

Defining Holidays

From the OA Schedules, select the Holiday Function.

Holiday screen gets displayed.

Click Create to invoke the Create screen with 2 tabs displayed.
