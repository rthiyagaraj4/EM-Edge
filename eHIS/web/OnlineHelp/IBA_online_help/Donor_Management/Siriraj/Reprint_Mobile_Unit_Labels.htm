Reprint Mobile Unit Labels

From this function you can Reprint labels for the blood units that are collected during a mobile donation.

The Reprint Mobile Unit Labels function is used to reprint labels for the blood units that are collected during a mobile collection. You can select the collection or supplier centre, specify the starting number to generate label, count for each label and number of labels and print the unit label. You can reprint the unit labels based on the collection center you select. If you give the range beyond the actual printed labels, a message ‘Unit No is not generated’ is displayed.

Reprinting Mobile Unit Labels

From the Transactions > Donation Management, double-click Reprint Mobile Unit Labels function.

The Reprint Mobile Unit Labels screen gets displayed.

  1. Click [List of Values] to select the Collection or Supplier centre. The list displays the values defined in Collection/Supplier Centre function. The Starting number to print the unit labels is displayed, by default.  

  2. Enter the Number of Labels required to be printed in numeric format of 15 digits maximum length. This is a mandatory field.

  3. The Count for each Label field displays the count. You can enter the count in numeric format of four digits maximum length. The value entered in the Number of labels field in 'BD Parameter' is displayed by default. You can update the value.  

  4. Click [Print Labels] to print the required number of labels.

  5. A confirmation message ‘Process completed Successfully’ is displayed.
