Workstation Based Print Routing/Parameters

From this function you can Define rules to route the transaction based or the non-transaction based reports to a specific printer. You can define the Workstation Parameters to execute a report. You can Query to view the print routing details and the workstation based parameters.

This function is used to define the destination printers for each specific workstation. It provides multiple ways of routing each online and generated report, depending on the requirements of the specific report. For online reports, there is a flexibility to define the routing dynamically based on the Operating Department/Ward/Clinic, which is involved in the transaction, currently being processed. For both online and generated reports, this includes the facility to print simultaneously at more than one destination and to print more than one copy of the report at any of these destinations. Up to three printers can be set up for each destination in a multiple shift environment. On entering the function, all workstation specific routings for the logged on workstation are listed in the second block and all non-workstation specific document routings are listed in the third block. On selecting to print the report, the report output is printed in the destination machine selected for a report group.

The print routing rules are to be defined based on the workstation identifier of the client, which runs EM. For example, in a facility, patient registration is performed from two different locations, say, a male desk and a female desk. If the patient label generated at the male registration desk has to be printed in the label printer located in the male registration area and the patient label generated at the female registration desk has to be printed in the label printer located in the female registration area, you have to set up the work station based print routing rules. In this case, it is possible to identify the clients at the registration desks by their unique identity and the rules are set up to route the patient label to appropriate printers based on their unique identity.

Defining Workstation Based on Print Routing or Parameters

From the Admin, double-click Workstation Based Print Routing/Parameters function.

The Workstation Based Print Routing/Parameters screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  1. Click [Commit Form] or press [F10] to save the records or [Clear Record] to clear the entered or queried data.

  2. Click [Execute Query] or press [F8] to search for the existing records within the facility.
