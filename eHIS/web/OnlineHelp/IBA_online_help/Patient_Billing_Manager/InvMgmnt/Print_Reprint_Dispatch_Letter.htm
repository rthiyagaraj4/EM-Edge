Print/Reprint Dispatch Letter

From this function you can Print the dispatch letters used to post the invoice to the Customers in the facility.

The Print/Reprint Dispatch Letter function enables you to print the dispatch letters, which are finalized. The printed letters can be reprinted based on the request. You have to select either the document date or the document reference number.

Print or Reprint the Dispatch Letter

From the Invoice Management > Reports, double-click on Print/Reprint Dispatch Letter function.

The Print/Reprint Dispatch Letter screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run Process] to either print, view or save the output in the local machine.
