Setup Invoice Additional Info

From this function you can Setup the additional details, which are to be captured while generating an invoice for a Customer. You can Query for the existing details.

You can also setup the data for an Info ID, which can be associated to the Customer while generating an invoice. Atleast one record has to be set as the default data for the Info ID. You also have an option to set up the additional information based on the Payer and the Policy association. The additional information is mandatory, if the additional information set up here is selected as mandatory. The additional information is recorded using this function for the encounters against which the invoices are generated for the bills. The invoices with the additional details are dispatched to the Customer while dispatching the Invoices.

Setup Invoice Additional Info

From the Invoice Management > Setup, double-click on Setup Additional Invoice Info function.

The Setup Additional Invoice Info screen gets displayed.

  1. Enter the details in the following tab pages:

Additional Info Identifiers

Enter the following details:

  1. Info ID – Enter the Info ID in alphanumeric format of 10 characters maximum length. The Info ID is unique and cannot be changed after creation. An Info ID identifies the additional information to be captured.

  2. Info Descriptor – Enter the Info Descriptor in alphanumeric format of 40 characters maximum length.

  3. Data Set Available – Select the check box, if the data set is applicable for the selected Info ID. Clear the check box, if the data set is not applicable for the selected Info ID.

  4. Default Value – Enter the Default Value in alphanumeric format of 100 characters maximum length. This is a mandatory field.

  5. [Setup Data] – Click [Setup Data] to setup data for the selected Info ID. The tab gets shifted to ‘Setup Data for Additional Info Identifiers’.

  6. Explanatory Note – Enter the Explanatory Note for the respective Info ID in alphanumeric format of 200 characters maximum length.

Setup Data for Additional Info Identifiers

Enter the following details:

  1. Info ID – The Info ID selected under the tab ‘Additional Info Identifiers’ appears, by default, if the data is already set for the Info ID. You cannot change the Info ID. The description for the selected Info ID appears in the adjacent text box.

  2. Code – Enter the Code in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Long Description - Enter the Long Description in alphanumeric format of 100 characters maximum length.

  4. Short Description - Enter the Short Description in alphanumeric format of 40 characters maximum length.

  5. Default – Select the check box, if the record has to be the default record for the selected Info ID. You have to select one record as the default record.

Setup Additional Info by Payer/Policy

Enter the following details:

  1. Payer Group Code – Click [List of Values] to select a Payer Group. The values defined using the ‘Customer Groups’ function under the AR Setup menu gets listed. This is a mandatory field.

  2. Payer - Click [List of Values] to select a Payer. The Customers associated to the selected Customer Group using the function ‘Customer’ under the AR Setup menu gets listed. This is a mandatory field.

  3. Policy - Click [List of Values] to select a Policy. The policies associated to the selected Payer and the Payer Group using the function ‘Setup Service Coverage Detail by Policy’ function in the Insurance Management module get listed. This is a mandatory field.

  4. Seq No – Enter the Sequence Number in numeric format of 40 characters maximum length. This indicates the order in which the associated data has to appear while capturing additional details for the Payer while generating the invoice. This is a mandatory field.

  5. Master ID - Click [List of Values] to select a Master ID. The Info ID created using the ‘Additional Info Identifiers’ tab gets listed. This is a mandatory field.

  6. Description – The Description for the selected Master ID gets displayed.

  7. Default Code Description/Text – The Default Value recorded for the Info ID using the ‘Additional Info Identifiers’ tab gets displayed.

  8. Mandatory – Select the check box, if the selected values are to be captured during invoice generation for the selected Policy and Payer association.

  1. Click [Commit Form] to save the records or [Clear Record] to delete the data entered.

  2. Click [Execute Query] to query for the existing records.
