View Prepaid Letter Details

From this function you can Query to view the prepaid letter details.

The View Prepaid Letter Details function is used to view the prepaid letter details. The details that are recorded through the 'Generate Prepaid Letter' function are listed. You can reprint the prepaid letter through this function.

Querying Prepaid Letter Details

From the Prepaid Management > Queries, double-click on View Prepaid Letter Details function.

The View Prepaid Letter Details screen gets displayed.

  1. The logged-in facility name is displayed, by default. The facilities that are associated to the logged-in user are listed. Select the facility name to view prepaid letter details.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Refresh] to query the details.

  4. Click [Reprint Letter] to reprint the prepaid letter.  

  5. The details that are recorded during prepaid letter generation are displayed.

  6. Click [C] to view additional information and diagnosis/procedure details that are recorded for a patient.
