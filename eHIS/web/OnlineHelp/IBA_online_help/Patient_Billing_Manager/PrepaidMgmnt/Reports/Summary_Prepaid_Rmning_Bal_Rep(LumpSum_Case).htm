Summary Prepaid Remaining Balance Report (LumpSum Case)

From this function you can Generate a summarized report on the remaining prepaid amount for a lumpsum deposit type.

The Summary Prepaid Remaining Balance Report (LumpSum Case) function is used to generate a summarized report on the balance prepaid amount that is remaining after deducting the patient charge. The report is generated for lumpsum deposit types only.

Generating a Summarized Prepaid Remaining Balance Report for Lumpsum Deposit Type

From the Prepaid Management > Reports, double-click on Summary Prepaid Remaining Balance Report (LumpSum Case) function.

The Summary Prepaid Remaining Balance Report (LumpSum Case) screen gets displayed.

  1. Click [List of Values] to select a Prepaid Program. The description of the selected Prepaid Program is displayed. You can leave the Prepaid Program as blank to generate a report for all the records that are associated to the selected date range.

  2. Click the Calendar function to select the Register Date range. This is a mandatory field.

  3. Click the Calendar function to select a Transaction Date range. You can leave the Transaction Date as blank to generate a report irrespective of the transaction date.

  4. Select a Transaction Type from the list. The system-defined values are 'All', 'Allocate', 'Receipt', 'Adjust' and 'Post'. The report is generated for the selected transaction type. The value is displayed as 'All', by default. Select 'All' to generate a report for all transaction types.

  5. Click [List of Values] to select a Register Number. You can leave the Register Number blanks to generate a report for all the register numbers that are associated to the selected date range.

  6. Click [Run Process] to generate a report.

  7. Report Option is displayed with the View, Print and File Preferences on clicking [Yes]. Click [OK] to generate or [Cancel] to abort the process.

    Select File to save the output in a file in the local system.

    Select Print to print the report.

    Select View to view the online report.
