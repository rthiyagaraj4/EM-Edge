Edit/Finalize Covering Letters

From this function you can either Edit or Finalize the Covering Letters generated using the function Generate Covering Letters. You can Query the Letter format before the generation.  

Edit/Finalize Covering Letters function is used either to edit or finalize the Covering Letters. You can make any modifications required before finalizing the Covering Letter either for all the outstanding documents or only for selected outstanding documents. You have an option to select the Print Format too for a Covering Letter. The Covering Letters generated using the function Generate Covering Letter get displayed. The documents, which are verified and cancelled are displayed using separate colors.

Edit/Finalize Covering Letters

From the Transactions > Covering Letter Process for Outst Doc, double-click Edit/Finalize Covering Letters function.

The Edit/Finalize Covering Letters screen gets displayed.

  1. Click [Execute Query] or press [F8] to edit and to finalize the Covering Letters generated. The document reference number generated for a specific patient with the outstanding document details gets displayed. The Billing Group and the Customer associated during Covering Letter generation get displayed. The date when the Covering Letter was generated is displayed. <Related Field Description>

  2. You can update the Covering Letter format before finalizing. You can select or clear the check boxes to define the format for the Covering Letter.

    1. Body Header – Select the check box to include the Body Header information in the Covering Letter.

    2. Body Footer – Select the check box to include Body Footer information in the Covering Letter.

    3. Print Text – Select the check box to include the Print Text information in the Covering Letter.

    4. In Foreign Currency – Select the check box to include the Foreign Currency in the Covering Letter generated.

    5. Foot Note – Select the check box to include the Foot Note information in the Covering Letter.

    6. Print Ins Policy– Select the check box to include the Insurance Policy information in the Covering Letter.

  3. The unsettled bills for the services availed from the facility get displayed. You can insert a new outstanding document to the patient record, cancel the associated outstanding documents for which the Covering Letter is generated or edit the outstanding document amount.

    Right click on an outstanding document record to update the outstanding document amount. Enter the new amount for the outstanding amount in numeric format. The amount gets updated in the ‘Tot Outst Amt’ field too.

    Right click on an outstanding document record to cancel an outstanding document. The cancelled document details are displayed using the color .  

    Right click on an outstanding document record to insert a new record. Click [Clear Record] to clear the inserted record.

    The Serial Number, Document Type and Number, date when the bill was generated, patient ID and patient name get displayed. The Customer Code and Group Code associated while generating the Covering Letter get displayed. The Print Text associated to the Group Code and the Customer combination using the function ‘Covering Letter Grouping by Patient’ is displayed.

    1. Select All – Select the check box to edit and to finalize the Covering Letter for all the outstanding documents associated to the patient.

    2. Verified – Select the check box to verify the Covering Letter for the outstanding documents. The document details are displayed using   color on verification.  

  4. Click [Criteria] to view the covering letter parameters defined using the ‘Generate Covering Letter’ function.

  5. Click [Details] to view the Covering Letter Format Details.

  6. Click [Reprint] to print the Covering Letter for the selected patient.

  7. Press downward arrow key to navigate to the next record on committing the details. Click upward arrow key to navigate to the previous record.
