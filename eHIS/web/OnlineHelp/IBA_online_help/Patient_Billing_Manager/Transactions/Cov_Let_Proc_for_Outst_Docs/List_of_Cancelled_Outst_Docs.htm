List of Cancelled Outstanding Documents

From this function you can Generate a report to view the cancelled outstanding documents.  

List of Cancelled Outstanding Documents function is used to print, view or save a copy of the cancelled outstanding document details in the local machine. You have an option to select the parameters based on which the report has to be generated. The report is sorted based on the sort order selected.

List of Cancelled Outstanding Documents

From the Transactions > Reminder Letter Process, double-click the List of Cancelled Outstanding Documents function.

The List of Cancelled Outstanding Documents screen gets displayed.

  1. Select a module ID from the list. The list of cancelled outstanding documents in the selected module gets listed.<Related Field Description>

  2. Click [Run Process] to view, print or save the details in a file.

  3. A log file is generated with the details. Click [OK] to view the report or [Cancel] to exit from the screen.
