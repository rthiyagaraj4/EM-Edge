List of Approved Refunds

From this function you can Generate a report to view the refund requests placed by a Cashier and approved by authorized billing personnel.

List of Approved Refunds function is used to generate a report on the refund requests approved by authorized billing personnel. The patient details, original document number, request number and the total refund amount is displayed in the report.  

List of Approved Refunds

From the Transactions > Refund Cycle, double-click List of Approved Refunds function.

The List of Approved Refunds screen gets displayed.

  1. Click Calendar function to select a date. The system date is displayed, by default. The report is generated for the transactions until the selected date.

  2. Select a value for Order By. The values are system defined - Request Date, Approved Date and Bank. The report output is arranged accordingly depending on the value selected for Order By.

  3. Click [Run Process] to generate the report. You can view, print or save the report output in a file. A log file is generated. Click [OK] to view the log file and [Cancel] to exit from the screen.
