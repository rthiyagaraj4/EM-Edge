List of Refunds Pending for Approval

From this function you can Generate a report to view the refund requests raised by the Cashier and are pending for approval from an authorized billing personnel.  

List of Refunds Pending for Approval function is used to generate a report on the refund requests placed by the Cashier and pending for approval by the authorized billing personnel. A document reference number generated when placing a request is displayed along with the amount to be refunded. The patient details, episode number and the total refund amount is calculated and displayed in the report.

List of Refunds Pending for Approval

From the Transactions > Refund Cycle, double-click List of Refunds Pending for Approval function.

The List of Refunds Pending for Approval screen gets displayed.

  1. Click Calendar function to select a date. The system date is displayed, by default. The report is generated for the transactions until the selected date.

  2. Select a Voucher Detail from the list. The values are system defined - All, Recorded and Not Recorded.

• Recorded – Select Recorded to view the records for which the Voucher Number is recorded.

• Not Recorded - Select Not Recorded to view the records for which the Voucher Number is not recorded.

• All – Select All to view all the documents for which the Voucher Number is either recorded or not recorded.

  1. Select a value for Order By. The values are system defined - Request Date and Original Doc Date. The report output is arranged accordingly depending on the value selected for Order By.

  2. Click [Run Process] to generate the report. You can either view, print or save the report output in a file. A log file is generated. Click [OK] to view the log file and [Cancel] to exit from the screen.
