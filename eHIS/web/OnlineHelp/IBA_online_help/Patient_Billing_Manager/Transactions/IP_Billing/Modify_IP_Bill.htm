Modify IP Bill

From this function you can Update a bill generated for an Inpatient or a Daycare patient.

Modify IP Bill function is used to update the discount amount in a bill generated for a relevant service availed from the Facility. An Authorized User only can login to update the bill for an Inpatient or a Daycare patient. If the check box ‘Bill Discount’ is selected in ‘Transaction Processing Privileges’ tab in the function ‘Users for Authorization/Refund/Cash Counters’ under the Administration menu, respective user is authorized to modify the discounts for a relevant bill. You have an option to modify the print line discount percentage. The system calculates the discount amount and the net amount based on the print line discount percentage set.   

Modify IP Bill

From the Transactions > IP Billing, double-click Modify IP Bill Function.

The Modify IP Bill screen gets displayed.

  1. Select a Patient ID for transaction. The patient name, age and gender of the patient gets displayed.

  2. Select an Episode Type from the list. The values are system defined - Inpatient and Daycare. This is a mandatory field.

  3. Click [List of Values] to select a Bill Number. The Bill Lookup gets displayed with the Patient ID, Episode ID, Visit ID, Document Type Code, Document Number, Document Date, bill total outstanding amount, bill paid amount, bill transaction type code, bill nature code and the bill status gets displayed. Double-click on the relevant record to select it. This is a mandatory field. The Bill Nature, billing Group and the Customer associated to the selected patient, (if any) get displayed.

  4. Click [Authorize] to modify the bill. A screen pops-up requesting the login information. Enter the User ID, PIN number and click [List of Values] to select a reason code. Click [Clear Record] to clear the entered data. <Related Field Description>
