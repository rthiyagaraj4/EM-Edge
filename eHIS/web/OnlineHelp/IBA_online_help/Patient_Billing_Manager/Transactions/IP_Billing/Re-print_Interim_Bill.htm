Re-print Interim Bill

From this function you can Reprint a range of Interim bills for the inpatients/daycare patients in the Facility. A report is generated based on the parameters set. An interim bill has to be printed initially to re-print using this function.

Re-print Interim Bill function is used to re-print the interim bills for an Inpatient or a Daycare patient in the facility. The report generated displays the bill amount and the Billing Group for a selected patient transaction. The name of the consulting doctor is displayed in the inpatient bill when a detail bill print is selected.

The orders for consultation are either placed through the 'Place Order' or the 'IP Consultation List' functions. The name of the ordering practitioner is displayed in the report for orders placed through these functions. The name of the doctor is also displayed for the direct charges that are placed through the 'Enter Request - IP' or the 'Enter Request - OP' functions.  

Re-Printing Interim Bill

From the Transactions > IP Billing, double-click Re-print Interim Bill Function.

The Re-print Interim Bill screen gets displayed.

  1. Select an Episode Type to print interim bills. The bills generated for the selected Episode Type gets displayed. <Related Field Description>

  2. Click [Process Run] to print the bill or [Clear Record] to clear the entered data.

  3. A bill print authorization screen pops-up. Enter the User ID, PIN Number and the Remarks to login and to re-print the bill. Click [OK] to login or [Cancel] to exit from the screen.

  4. A confirmation screen pops-up on clicking [OK]. Click [Yes] to re-print the bills or [No] to cancel the printing process.
