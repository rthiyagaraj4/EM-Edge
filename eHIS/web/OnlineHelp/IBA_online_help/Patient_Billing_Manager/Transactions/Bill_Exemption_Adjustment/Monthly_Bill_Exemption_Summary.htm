Monthly Bill Exemption Summary

From this function you can Generate a summary on the bill exemption details for the entire month. The exemptions verified, approved and finalized by an authorized billing personnel for a specific month is printed.

Monthly Bill Exemption Summary is an option provided to generate a report on the bills exempted in the Facility for a specific month. The report generated lists the exemption requests, which are finalized and approved by authorized billing personnel. The exemption code against which the amount was adjusted is also displayed. The total amount exempted for a patient in the facility is calculated and displayed.

Generate Monthly Bill Exemption Summary

From the Transactions > Bill Exemption/Adjustment, double-click Monthly Bill Exemption Summary Function.

The Monthly Bill Exemption Summary screen gets displayed.

  1. Click Calendar function to select the month and year. This is a mandatory field. The exemption details are generated for the selected month and year.

  2. Click [Process Run] to print the details or [Clear Record] to clear the entered data.

  3. Click View, Print or File to view the report, print or save the output in a file respectively.

  4. A log file is generated. Click [OK] to view the log file or [Cancel] to exit from the screen.
