Record Discharge Clearance

From this function you can Generate a discharge clearance slip for an inpatient.

Record Discharge Clearance function is used to generate discharge clearance slip. The finance team generates a discharge clearance slip for an inpatient after the discharge advice is generated by the Nurse. Discharge clearance slip is generated irrespective of bill settlement against the discharge bill raised. Users are restricted from placing new orders for the patient after the discharge clearance slip is raised. You can generate a discharge clearance slip only if the discharge advice is raised. You can access this function only if the check box 'Discharge Clearance in Use' is selected in 'BL Parameters' function under the 'Inpatient/Daycare' tab. Authorized users with valid access rights are permitted to generate discharge clearance slip. You can generate discharge clearance slips for patients' with or without discharge bill.

Recording Discharge Clearance

From the Transactions > IP Discharge Clearance, double-click Record Discharge Clearance Function.

The Record Discharge Clearance screen gets displayed.

  1. Select a Facility ID from the list. The logged in Facility is displayed, by default. The Facilities associated to the logged in User are listed. This is a mandatory field. <Related Field Description>

You can search inpatient records against which the discharge advice has been raised by selecting the Facility ID and clicking [Search].

  1. Click [Search] to search for patient records matching the search criteria. On clicking [Search], the legend changes to [As On: Current Date Time]

  2. Patient records matching the search criteria gets listed. Inpatient records get listed with the Patient ID, Episode ID, Episode Type, Admission Date, Discharge Advice Date, Discharge Bill Document Type and Number and the Outstanding Amount.  

  3. Select the check box against patient records to generate discharge clearance.

  4. An authorization screen pops-up requesting the login details. Enter the details and click [Accept] to generate discharge clearance slip or [Exit] to cancel.

    Enter the following details to login and generate discharge clearance slip:

    1. User ID - Enter the User ID in alphanumeric format. Authorized users are only permitted to login and generate discharge clearance slip. This is a mandatory field.

    If the user is not authorised, an error message 'The User is not authorised to generate discharge clearance without bill' is displayed. User is permitted to authorize discharge clearance with discharge bill and without discharge bill for inpatients' by selecting the check boxes 'Discharge Clearance With Discharge Bill' and 'Disch Clearance W/O Disch Bill/Outst Bill' respectively through the function 'User Privileges and Cash Counter Access'.

    1. Pin Number - Enter the Pin Number of the respective User. This is a mandatory field.

    2. Reason - Click [List of Values] to select a reason. The action reasons defined through the 'Reasons for Billing Actions' function with the check box 'Discharge Clearance W/O Bill' as selected are listed. This is a mandatory field.

    3. Remark - Enter the Remarks in alphanumeric format. This is a mandatory field.

  1. Click [Commit Form] to cancel the services or [Clear Record] to clear the entered data.

  2. Discharge Clearance Slip number is generated on clicking [Commit Form].
