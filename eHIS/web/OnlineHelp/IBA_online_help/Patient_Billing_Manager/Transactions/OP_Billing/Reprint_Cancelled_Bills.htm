Reprint Cancelled Bills

From this function you can Reprint the Outpatient and Emergency patient bills that are cancelled.

Reprint Cancelled Bills function is used to re-print the bills that are cancelled. You can cancel the Outpatient and Emergency patient bills that are cancelled through the 'Nullify/Cancel a Bill' function. The check box ‘Reprint Bill’ has to be selected in the ‘Users for Authorization/Refund/Cash Counters’ function in the Administration menu for the selected User.

Reprint Cancelled Bills

From the Transactions > OP Billing, double-click the Reprint Cancelled Bills Function.

The Reprint Cancelled Bills screen gets displayed.

  1. Select an Episode Type to re-print cancelled bills. The bills generated for the selected Episode Type gets displayed. <Related Field Description>

  2. Click [Process Run] to print the bill or [Clear Record] to clear the entered data.

  3. A bill print authorization screen pops-up. Enter the User ID, PIN Number and the Remarks to login and to re-print the bill. Click [OK] to login and [Cancel] to exit from the screen.

  4. A confirmation message 'Do you want to confirm?' is displayed. Click [Yes] to re-print the cancelled bills listed in the detailed block or [No] to cancel the printing process.

 

An authorized User is permitted to login and re-print the cancelled bills.
