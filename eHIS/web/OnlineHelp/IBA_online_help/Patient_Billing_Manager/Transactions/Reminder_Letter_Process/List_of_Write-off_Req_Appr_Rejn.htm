List of Write-off Req/Appr/Rejn

From this function you can Generate a report to view the write-off requests either placed/rejected/approved by an authorized billing personnel.

List of Write-Off Req/Appr/Rejn function is used to list out the patient records, where the amount is not settled even though multiple reminder letters are dispatched. Initially a request is placed to write-off the patient records using the function ‘Generate Write-off Request’. An authorized User has to either approve or reject the write-off process. You can generate a report to view all the outstanding documents for which the write-off request has been placed and approved or rejected using this function. The report generated displays the outstanding amount against which the write-off request has been placed, the original Document Type and Number, the reference document details and the reason code for approval or rejection of write-off.

List of Write-Off Req/Appr/Rejn

From the Transactions > Reminder Letter Process, double-click List of Write-Off Req/Appr/Rejn function.

The List of Write-Off Req/Appr/Rejn screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run Process] to generate a report on the records for which the write-off is requested and approved or rejected.

  3. You can either View, Print or save the report output in a file. A log file is generated. Click [OK] to view the report output or [Cancel] to exit from the screen.
