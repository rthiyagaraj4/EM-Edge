Transfer Package

From this function you can Transfer a treatment package that is associated to a patient. You can Query to view the encounter and package approval details.

The Transfer Package function is used to transfer a treatment package that is associated to the patient. You can transfer a package that is open and subscribed to an alternate package that is available. Authorized users are only permitted to transfer the package. The option to transfer an available package to an existing package is available on selection and on recording valid reasons. System recalculates charge for the new package applicable. The services that are associated to the package and are in order stage can be marked for cancellation. The services that are not a part of new package will be charged outside the package on applicable billing rules for the patient. You can view the Receipt/Refund screen to refund the deposit and collect deposit for a new transferred package or adjust the amount for future encounters of a respective patient in the facility. The encounter approval against the transferred package has to be manually updated through [Package Details] option available in the 'Modify Financial Details - Visit' and 'Modify Financial Details - Admission' functions for the respective encounter.

Transfer a Package

From the Transactions > Treatment Packages, double-click the Transfer Package Function.

The Transfer Package screen gets displayed.

  1. Enter the Patient ID in alphanumeric format of 10 characters maximum length or click [List of Values] to select the patient ID. The name, age and gender of the patient gets displayed on entering the Patient ID. This is a mandatory field.

  2. Press [Tab] to select the package that has to be cancelled.

  3. The details are displayed under the following tabs:

Transfer Package

Encounter & Approval Details

  1. Click [Commit Form] or press [F10] to save the records or click [Clear Record] to clear the data.

  2. The package amount is recalculated.
