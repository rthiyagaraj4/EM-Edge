Subscribe Treatment Packages  

From this function you can Subscribe a treatment package for a patient.

The Subscribe Treatment Packages function is used to associate multiple packages to a patient. You can define the period within which the package service has to be availed by the patient. Select a Billing Group that is defined as Package related. On selecting the Billing Group, the Price Classes associated to the Billing Group gets listed. The package amount gets displayed listing the services associated to the respective package.

Subscription of Treatment Packages

From the Transactions > Treatment Packages, double-click the Subscribe Treatment Packages Function.

The Subscribe Treatment Packages screen gets displayed.

  1. Click [List of Values] to select a Patient ID. The name of the patient, age and the gender gets displayed on selecting the Patient ID. This is a mandatory field.

  2. Click [List of Values] to select a Billing Group. The Billing Groups defined with the ‘Package Patient’ check box as selected get listed. The description for the selected Billing Group gets displayed.  

  3. Click [List of Values] to select a Package. The Price Classes associated to the Billing Group selected get listed. The description for the selected Price Class gets displayed.  

  4. Click [List of Values] to select a Billing Class. The patient class for which the package is defined as valid gets listed. The description for the selected Billing Class gets displayed in Billing Class Description field.

  5. Click [List of Values] to select a Customer Group. The description for the selected Customer Group gets displayed in Customer Group Description field.

  6. Click Calendar function to select the subscription validity period. The package is valid for the patient during the period mentioned.

  7. The Package amount gets displayed automatically based on the price defined for the package.

  8. The deposit mandatory check box is either selected or cleared based on the parameter setup for the respective Price Class. If the check box is selected, the ‘Enter IP Receipts/Refunds’ or the ‘Enter OP Receipts/Refunds’ screen gets displayed based on the patient class to record the deposit amount collected from the patient. An error message gets displayed halting the transaction, if the deposit amount is not accepted when the check box is selected. On accepting the deposit and saving the records, a receipt number gets generated. You can also select to print the receipt details. A report is generated displaying the package transaction details.

  9. Click [Commit Form] or press [F10] to save the records or click [Clear Record] to clear the data.

 

The packages associated to the patient are displayed using the ‘$’ symbol in the external modules while recording the financial details.
