Cancel OP Direct Billing Items

From this function you can Cancel the services associated to an Outpatient, Emergency patient or an External patient. You can cancel the services placed from the Patient Billing Manager module.

Cancel OP Direct Billing Items function is used to cancel the Billing Services associated to an outpatient/external patient. The user has an option to cancel either the confirmed services or the unconfirmed billing services. The cancelled services are not considered during bill generation. Authorized Users can only cancel the services. The authorization to cancel the services is set using the function ‘User Groups/Users’ by selecting the check box ‘Cancel Confirmed Service’. The Users associated to the selected User Group are permitted to cancel the services associated to the patient. The services are associated to the patient using the functions ‘Enter Requests – OP’ and ‘Location Billing/Collection’.

Cancel OP Direct Billing Items

From the Transactions > OP Direct Charges, double-click Cancel OP Direct Billing Items function.

The Cancel OP Direct Billing Items screen gets displayed.

  1. Select an Episode Type from the list. The system defined values are Outpatient, Emergency and External.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

OR

  1. Right click on Right Click for User Options to search for the relevant patient record. You can search for a patient record based on the Patients Lookup, Patient/Episode/Visit lookup or Episode/Visit lookup.

  2. Select the check box View Billing to view the billing details.

    The financial details of the patient for the selected episode get displayed. You can also view the unconfirmed services associated to the patient by clicking [Unconfirmed Services]. The Settlement Details, Billing Group details, service charges, gross amount for the services utilized, adjustments or refunds issued, unbilled and outstanding amount get displayed. The name of the User, who has recorded or modified the financial details gets displayed.

  3. Click [List of Values] in the 'Source Doc.Ref.' field to select and cancel a service. The transaction reference number generated while associating the Billing Services/Panels through 'Enter Request - OP' and 'Location Billing/Collection' functions are listed. The transaction reference document reference line number, Source Document Ref Number, Transaction Date, Panel/Service, Service Code, Rate/Charge, quantity of service associated, UOM, Total Payable, Patient Payable, Patient Paid and Doctor ID are listed..

  4. Select this check box to select all the records. You can also select individual record by selecting the check box against each record.

  5. Select a transaction document reference number from the list and click [OK] to cancel a Billing Service/Panel. You can search a transaction document reference number in the pop-up screen. Enter the value in 'Find' and click [Find] to query the results. Click [Cancel] to cancel the operation and close the pop-up screen.

  6. The transaction date, Billing Service/Panel code and description, service quantity, UOM, total payable amount, patient payable amount, practitioner/staff indicator and practitioner/staff ID is displayed.

  7. Click [Commit Form] to cancel the services. A confirmation message 'Cancelling Daily Process Related Charges, Please Confirm' is displayed. Click [OK] to confirm. If the Billing Service/Panel is already confirmed, a confirmation message 'This Service is already confirmed' is displayed. Click [OK] to cancel. Cancelled transaction reference numbers are deleted from the list in 'Source Doc. Ref.'. Click [Clear Record] to clear the entered data.
