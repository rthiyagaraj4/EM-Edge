Acknowledge Handover Request

From this function an authorized user can Acknowledge the cash or cheque handover request. Authorized users' can Approve or Reject the handover request placed.

Acknowledge Handover Request function is used to acknowledge the requests that are placed through the 'Handover Request' function. The Cash Counter transactions that have crossed the Cash and Cheque handover limit are highlighted in red. Authorized users' are permitted to acknowledge the transactions. The users' with the 'Acknowledge Handover Request' check box selected in the 'User Privileges for Cash Counter Access' function are permitted to approve or request the handover. You can approve the handover request for cash or cheque amount that has exceeded the grace limit set for cash or cheque transactions. The requests that are acknowledged or rejected are displayed in the transaction details section of 'Handover Request' function. The details of the Cash Counters that are active in the facility are listed.

Acknowledge Handover Request

From the Transactions > Cash Control, double-click the Acknowledge Handover Request Function.

The Acknowledge Handover Request screen gets displayed.

  1. Select a facility from the list. The facilities associated to the logged in user are listed.

  2. Click [Refresh] to refresh the details.

  3. Last date and time when the records were refreshed are displayed for reference. <Related Field Description>

  4. Right-click on the 'Requested Cash' or 'Requested Cheque' fields to approve or reject the handover request.

  5. Approve/Reject Handover Request screen is displayed with the denomination or Cash Handover details recorded through the 'Handover Request' function. You can update only the status or record remarks through this screen.

    1. Remarks - Enter the remarks in alphanumeric format. You can record additional details. This is a mandatory field if the status is selected as 'Reject'.

  6. Select 'Accept' to accept the handover request. A confirmation message 'Do you want to Accept?' is displayed. Click [Yes] to accept or [No] to cancel. Select 'Reject' to reject the handover request. A confirmation message 'Do you want to Reject?' is displayed. Click [Yes] to reject or [No] to cancel.

  7. Report Option is displayed with the View, Print and File Preferences on clicking [Yes]. Click [OK] to generate or [Cancel] to abort the process.

    Select File to save the output in a file in the local system.

    Select Print to print the report.

    Select View to view the online report.

  8. Click [Close] to exit from the screen.

  9. Click [Commit Form] or press [F10] to record the details or [Clear Record] to clear the entered data.

 

The check box ‘Handover Request Without CC Logout' is mandatory to enable this function.
