Cash Control Receipt  

From this function you can Generate a receipt for the cash collected at the Cash Counter by a cashier. You can Query to view the pending transactions associated to a Cash Counter.

Cash Control Receipt function is used to generate a Cash Control Receipts for the selected transactions performed in the Cash Counter on a selected date. Cashier is able to view the transactions in a selected Cash Counter using this function. The shift ID, the total amount collected in the form of cash, credit amount and the receipt amount are displayed. Cashier has an option to view the Settlement details, the Cash Counter Transactions, the pending transactions at a Cash Counter for a specific date. An option is also provided to view a detailed cash counter report. The Cash Control Receipt is mandatory to post the billing transactions. You can generate the Cash Control Receipts for the transactions across the days only if the check box ‘Span Shift Across Day’ is selected in the BL Parameters. If the check box is cleared, you can process receipts for the transactions on the same date only.

Generating a Cash Control Receipt

From the Transactions > Cash Control, double-click the Cash Control Receipt Function.

The Cash Control Receipt screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. You can process the receipts and view the pending transactions using the following tab pages:

Cash Control Receipt

The details are displayed under the following three tabs:

Create Cash Control Receipt

The following details are displayed:

  1. Opening Time – The opening time of the Cash Counter gets displayed.

  2. Closing Time – The closing time of the Cash Counter gets displayed.

  3. Cash Bills Amount – The system displays the Cash Bill amount for combination of Cash Counter, Transaction Date and Shift ID.

  4. Credit Bills Amount – The system displays the Credit Bill amount for combination of Cash Counter, Transaction Date and Shift ID.

  5. Counter Collection – Enter the amount collected in the Cash Counter at the respective shift. This is a mandatory field.

  6. Receipt Amount – The system displays the Receipt amount based on the records selected.

  7. Credit Card Amount – The system displays the amount collected through Credit Cards based on the records selected.

  8. Total – The total amount collected during the shift for the selected records gets displayed.

  9. Select – Select the check box to generate a Cash Control Receipt for relevant records. You can select the records across the days, only if the ‘Span Shift Across Day’ is selected in the BL Parameters function.

View Settlement Details

The following details are displayed:

  1. Settlement Type – The System displays the Settlement.

  2. Counter Collection – The system displays the amount for the settlement type. The Settlement Types recorded during the transactions get displayed.

  3. Cancelled Amount - The system displays the cancelled amount for the relevant Settlement Type.

  4. Receipt Amount – The system displays the Receipt amount for displayed Settlement Type.

View Cash Counter Trx

The following details are displayed:

  1. Cash Counter Code – The system displays the Cash Counter Code selected in Create Cash Control Receipt Tab. The description for the selected Cash Counter gets displayed.

  2. Doc Date - The system displays the Document Date selected in the Create Cash Control Receipt Tab.

  3. Shift Number – The system displays the shift selected in Create Cash Control Receipt Tab.

  4. Criteria - Select a value from the list. Select Receipts, Refunds or Receipts & Refunds to view corresponding transactions.

  5. Total – The system displays the total amount of Receipts and Refund for the document collected in the Cash Counter.

  6. Control Doc – The system displays the Control Document Type and the number generated for the transaction.

  7. Doc Type – The system displays the original Document Type selected during transaction.

  8. Doc No – The system displays the original Document Number generated during the transaction against the Document Type.

  9. RN – The system displays the Receipt Number for the selected Document Type.

  10. RT – The system displays the Receipt Type for the corresponding Receipt Number.

  11. ST – The system displays the Settlement Type. The Settlement Type indicates the mode of payment for the selected transaction.

  12. Amount – The system displays the Document Amount.

  13. Patient No - System displays the name of the patient from whom the amount is collected during the transactions.

  14. Status – The Status is displayed as either ‘Receipts’, ‘Refunds’ or ‘Receipts and Refunds’ based on the criteria selected.

  15. No Of Document – The system displays the number of documents selection criteria.

  16. Total Amount – The system displays the total amount.

View Pending Transaction

You can view the pending transactions using this tab:

  1. Cash Counter - The system displays the pending Cash Counter where Cash has not been collected.

  2. Shift Id - The system displays the pending shifts where Cash has not been collected.

  3. Doc Date - The system displays the Document Date where Cash has not been collected.

  4. Bill Amount - The system displays the Bill Amount for the combination of Cash Counter, Shift and Document Date.

  5. Receipt Amount - The system displays the Receipt Amount for the combination of Cash Counter, Shift and Document Date.

  6. All Cash Counters – Select the check box to display the pending transactions for all the Cash counters.

  1. Click [Commit Form] or press [F10] to record the details or [Clear Record] to clear the entered data.

 

• The check box ‘Cash Control Receipt in use’ in the BL Parameters has to be selected to generate Cash Control Receipts using this function.

• If the check box ‘Logout Cash Counter with the Closing Bal.Details’ is selected, the Cashier has to use ‘Cash Control Receipt (Advanced) function to generate the receipts.

• Cashier has to login to the Cash Counter to generate the Cash Control Receipts.
