Approve Cash Control Receipts  

From this function you can Authorize the cash control receipts generated initially. The document details to be verified manually are displayed. You can Query to view the document details recorded by the cashier.

The Approve Cash Control Receipts function is used to approve the Cash Control Receipts generated using the function Cash Control Receipts or Cash Control Receipt (Advanced). You can select either a single record or all the records to approve the receipts. You have to select a User from the list. The receipts processed during the selected Cashier’s shift get listed for the selected date.

Approve Cash Control Receipts

From the Transactions > Cash Control, double-click the Approve Cash Control Receipts Function.

The Approve Cash Control Receipts screen gets displayed.

  1. Click [Execute Query] or press [F8] to view and to verify the Cash Control Receipts irrespective of the User and the Cash Counter.

  2. Click [List of Values] to select a User from the list and to approve the receipts relevant to the User.

  3. Click Calendar function to select the Document Date. The receipts generated on the selected date get displayed.

  4. Click [Execute Query] or press [F8] and the Cash Control Receipts details for the selected date gets listed. The Document Type, corresponding Document Number, date when the receipt is generated, total amount collected, credit card amount, where the settlement type is credit card and the balance amount is displayed along with the Cash Counter code and the description.

  5. Select the Verify check box to approve the receipts.

  6. Click [Commit Form] or press [F10] to save the records or click [Clear Record] to clear the entered data.
