Cancel IP Receipts

From this function you can Cancel a relevant receipt generated for an inpatient/daycare patient transaction. The receipts can be either for the payments accepted or for the refunds issued to the inpatients. Cashier has to cancel only the printed receipts.      

Cancel Receipts function is used to cancel the receipts generated for an inpatient/daycare patient transaction in the Facility. An authorized Cashier has access to cancel only the printed receipts. The ‘Receipts’ option has to be checked in the function ‘Users for Authorization / Refund / Cash Counters’ in the Administration menu for a Cashier to login and cancel the receipts. A Cashier can cancel the receipts printed for the transactions on a relevant day. To cancel the previous day’s transaction, the check box ‘Allow Cancel of Previous Days Cashier Txns’ has to be checked under the tab ‘Receipts/Refund Controls’ in the ‘BL Parameters’ function. You can cancel the bills associated to the receipts using this function. If there are multiple bills associated to the same receipt, you can select to cancel individual bills. Current receipt is highlighted using color. Current encounter is highlighted using color and other payers associated to the current receipt are highlighted using color.

Cancel IP Receipts

From the Transactions > IP Cashier, double-click Cancel IP Receipts Function.

The Cancel IP Receipts screen gets displayed.

  1. Click [Enter Query] and enter the Patient Number; press [F8] or [Execute Query] to cancel the receipt for a selected patient. Press [F8] or [Execute Query] to view the details of all the inpatients.

  2. The receipt details generated get displayed on clicking [Execute Query].

  3. Select Cancel to cancel the receipts. The Cancellation Authorization screen pops-up to validate the user privilege to cancel the document.

  4. Enter the User Id, Pin Number and Cancel Reason. Select a value from the list. The reasons defined using the function ‘Reasons for Billing Actions’ with the Receipt Cancellation check box as selected are listed.  

  5. Enter a valid reason for cancellation. During cancel, the updates are just the reverse of that of the receipt update process.

  6. Right-Click on the Bill Number to cancel the respective bill. On right-clicking, the ‘Nullify/Cancel a Bill’ screen gets displayed with the bill number and the type displayed and the ‘Cancel’ check box as selected. Click [Commit Form] to cancel the bill. A warning message ‘Do you want to confirm?’ gets displayed. Click [YES] to cancel or [NO] to abort the transaction. If the bill is associated to another receipt, an error message gets displayed and you cannot cancel the bill. If the bill is associated to another payer, an error message ‘Bill not valid for modification or cancellation’ is displayed.                                                                           

You have to login to the Cash Counter to cancel any receipts generated. You can cancel the IP receipts only when it is not printed and posted.

If the bill is already posted, you cannot cancel the bill. A message ‘Cannot Cancel the Posted Bill’ is displayed.
