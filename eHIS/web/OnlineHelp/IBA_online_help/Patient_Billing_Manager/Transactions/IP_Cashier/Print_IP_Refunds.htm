Print IP Refunds

From this function you can Print the refund receipts at the Facility for a specified date range for either the inpatients or the daycare patients. You cannot print the refund receipts which are cancelled.      

Print IP Refunds function is used to print the documents generated for every refund transaction in the Facility. The refund receipts are printed in a batch mode for a range of Document Types selected. An authorized user only has access to print the refund receipts. The ‘Print IP Refunds’ option has to be checked in the function ‘Users for Authorization / Refund / Cash Counters’ in the Administration menu for the user to login and print the refund receipts.

Print IP Refunds

From the Transactions > IP Cashier, double-click Print IP Refunds Function.

The Print IP Refunds screen gets displayed.

  1. Select an Episode Type from the list. The values are system defined - Inpatient and Daycare.

  2. Click [List of Values] to select a range of document type code and number for the range of the document numbers.

  3. Click Calendar function to select the date range. The refund receipts generated during the selected date range gets displayed.

  4. Click [Process Run] to print the refund receipts or [Clear Record] to clear the data.
