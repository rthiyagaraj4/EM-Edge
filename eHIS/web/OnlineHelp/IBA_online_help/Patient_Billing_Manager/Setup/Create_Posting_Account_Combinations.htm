Create Posting Account Combinations

From this function you can Define a combination of the posting accounts. You can Associate a department along with a Subledger and Product code to the Chart of Accounts.

Create Posting Account Combinations function is used to define a combination of the posting accounts. You cannot define the posting accounts, if the account combination can be used towards bank reconciliation. Account Balances are maintained against the posting account combinations defined. The following validations are made:

• Suspended codes are not considered.

• Chart of Accounts and Sub Ledger code combination is referred to while creating the full posting account involving Sub Ledger codes.

Defining Posting Account Combinations

From the Setup > GL Setup, double-click on Create Posting Accounts Combination function.

The Create Posting Accounts Combination screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run Process] to process the log file or [Clear Record] to clear the entered data.

 

Suspended accounts cannot be combined.
