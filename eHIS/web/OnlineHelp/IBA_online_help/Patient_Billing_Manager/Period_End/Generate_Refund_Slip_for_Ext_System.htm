Generate Refund Slip for Ext. System

From this function you can Generate a refund slip for the refunds issued through an external system.

Generate Refund Slip for Ext. System function is used to generate a refund slip for the refunds issued through the external system such as SAP. The pending deposit amount of a patient is displayed which can be issued as a refund after adjusting the bill amount for the services. A refund slip is generated for the refunds issued through the external system and the details are recorded for future reference in the Patient Billing Manager module. A valid refund Cash Counter has to be defined for the Cashier to login and generate slips for the refunds issued through external system.

If you have not logged into a Cash Counter, an alert message 'This user has not yet logged in' appears on double-clicking the function 'Generate Refund Slip for Ext. System'. Click [OK] to access the function and login to the respective Cash Counter.

If you have not logged into a Cash Counter valid to generate refund slips for external system, an alert message 'User is not Logged into a valid Cash Counter for Refunds interfaced to external system' is displayed. Click [OK] to login to a Cash Counter valid to generate refund slips.

The check box 'External System Specific Refund Counter Only' has to be selected for the respective Cash Counter.  

Generating Refund Slip for External System

From the Period End, double-click Generating Refund Slip for Ext. System function.

The Generate Refund Slip for Ext. System screen gets displayed.

  1. Select a value from the list to generate the refund slips for respective Episode Type. The system-defined values are 'All', 'Inpatient', 'Daycare', 'Outpatient', 'Emergency' and 'External'. <Related Field Description>

  2. Click [Login] to login to a Cash Counter. The Cash Counters defined with the 'External System Specific Refund Counter Only' check box as selected are listed.   Select the Cash Counter ID and click [Commit]. User is logged into the Cash Counter.

  3. Click [Search] to search for the deposit or the bill refund details.

  4. The details are displayed under the following tabs:

Deposit Refund

Deposit Refund

Click 'Deposit Refund' tab to view the deposit details. The unadjusted amount, the document type and the document number generated while accepting the deposit from the patient during admission, patient ID and name along with the date when the document is generated are listed.

  1. Sel - Select the check box 'Sel' to refund the pending deposit amount after settling the bill amount for the services. The bill amount incurred for the services in the facility is adjusted against the deposit amount paid by the patient during admission to the facility.

  2. Based on the Episode Type, the 'Enter OP Receipts/Refunds' or 'Enter IP Receipts/Refunds' screen gets displayed on clicking 'Select'.   

  3. Enter the settlement details in the screen. The Cashier in the Cash Counter should have enough amount to issue a refund. You have to enter the 'Opening Balance' while logging into the Cash Counter. If the amount is not recorded, an error message 'Amount in hand is not sufficient to issue a refund' gets displayed.  

  4. Click [Commit] on entering the details. A message 'Please Confirm the Printer is ready' gets displayed. Click [Confirm] to save or [Cancel] to cancel the transaction.

  5. The receipt type and the receipt number gets displayed on clicking [Confirm]. Click [OK] to save the adjustment details.

Bill Refund

Bill Refund

Click 'Deposit Refund' tab to view the deposit details. The unadjusted amount, the document type and the document number generated while accepting the deposit from the patient during admission, patient ID and name, encounter ID along with the date when the document is generated are listed.

  1. Sel - Select the check box 'Sel' to refund the pending bill amount after settling the bill amount for the services. The bill amount incurred for the services in the facility is adjusted against the deposit amount paid by the patient during admission to the facility. The 'Settle Bill Outstanding' screen gets displayed.

  2. Enter the settlement details in the screen. The Cashier in the Cash Counter should have enough amount to issue a refund. You have to enter the 'Opening Balance' while logging into the Cash Counter. If the amount is not recorded, an error message 'Amount in hand is not sufficient to issue a refund' gets displayed.  

  3. Click [Commit] on entering the details. A message 'Please Confirm the Printer is ready' gets displayed.

  4. The receipt type and the receipt number gets displayed on clicking [Confirm]. Click [OK] to save the adjustment details.
