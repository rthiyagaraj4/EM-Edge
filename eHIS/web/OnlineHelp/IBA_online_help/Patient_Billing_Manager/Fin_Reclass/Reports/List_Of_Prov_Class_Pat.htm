List of Provisionally Classified Patients

From this function you can Generate a report to view the financial classification details with the Classification Type as Provisional.

List of Provisionally Classified Patients function is used to generate a report of reclassification with the Classification Type as Provisional. You can choose to select the patient ID and the registration date.

Listing Provisionally Classified patient details

From the Financial Reclassification > Reports, List of Provisionally Classified Patients Function.

The List of Provisionally Classified Patients screen gets displayed.

  1. Enter the Patient No. in alphanumeric format of 10 characters maximum length.

  2. Click the Calendar function to select the registration date. The patient records for which the financial details are reclassified gets listed.

  3. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  4. You can view the report, print the details or save the output in a file in the system.
