Reclassification Reminder List

From this function you can Generate a reminder list to renew the financial classification rules.

Reclassification Reminder List function is used to generate a report of the financial classification rules that are approaching the expiry date. Based on the report generated, the financial classification rules can be identified and renewed before expiry.

Generating Reclassification Reminder list

From the Financial Reclassification > Reports, Reclassification Reminder List Function.

The Reclassification Reminder List screen gets displayed.

  1. Enter the Patient No. in alphanumeric format of 10 characters maximum length. The patient records for which the financial classification rules are going to expire get listed.

  2. Click [List of Values] to select a range of Billing Groups. The patient records with the selected Billing Groups are listed.

  3. Enter the Cut off days for Expiry in numeric format of two digits maximum length. This is a mandatory field. A report is generated for the financial classification rules expiring within the specified number of days beginning from the current system date.

  4. Enter the number of visits in numeric format of two digits maximum length. The patient records with the specified number of visits to the facility are listed for which the financial classification rules are expiring within the cut off days specified.

  5. Enter the number of admissions in numeric format of two digits maximum length. The patient records with the specified number of admissions to the facility are listed for which the financial classification rules are expiring within the cut off days specified.

  6. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  7. You can view the report, print the details or save the output in a file in the system.
