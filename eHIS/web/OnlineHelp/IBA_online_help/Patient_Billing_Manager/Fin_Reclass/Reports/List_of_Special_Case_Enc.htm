List of Special Case Encounters

From this function you can Generate a report to view the financial classification details with the Classification Type as Special Case.

List of Special Case Encounters function is used to generate a report of reclassification with the Classification Type as Special Case.

Listing Special Case Encounters

From the Financial Reclassification > Reports, List of Special Case Encounters Function.

The List of Special Case Encounters screen gets displayed.

  1. Select the Episode Type from the list. The system-defined values are - Inpatient, Outpatient, Daycare and Emergency.

  2. Click the Calendar function to select the admission or visit date. The patient records for which the financial details are reclassified gets listed.

  3. Click [List of Values] to select the financial reclassification codes with the Classification Type as 'Special Case'.

  4. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  5. You can view the report, print the details or save the output in a file in the system.
