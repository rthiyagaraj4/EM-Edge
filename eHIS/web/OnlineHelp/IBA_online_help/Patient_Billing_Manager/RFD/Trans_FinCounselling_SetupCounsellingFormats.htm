Enter the following details to setup counselling formats:

 

  1. Counseling Format ID – Enter the counselling format in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Long Description – Enter the long description in alphanumeric format of 50 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. New Entry Allowed During Trx – Select a value from the list. The system-defined values are ‘Yes’ and ‘No’. Select ‘Yes’ to permit the user to associate additional counselling details to the format during transaction. Select ‘No’ to restrict the user from associating additional counselling details to the format during transaction.

Counseling Details

  1. Code – Click [List of Values] to select the Counselling code. The values defined using the ‘Counseling Details’ function get listed.

  2. Description – The description for the selected Counselling Code gets displayed.

  3. Type – The type selected while defining the Counselling Code using the ‘Counseling Details’ function get listed.

  4. Default Value – The default value selected for the respective Counselling Code gets displayed. If there are multiple values associated to the counselling code, the values get listed.

  5. Seq No – The sequence number gets displayed automatically.

  6. Mandatory Yn – Select the check box to specify the record as mandatory. Clear the check box, if the specific record is not mandatory.

  7. Detail Description – The text area is displayed based on the ‘Type’. If the ‘Type’ for the respective record is specified as ‘Free Text’, the description recorded using the ‘Counseling Details’ function gets displayed.
