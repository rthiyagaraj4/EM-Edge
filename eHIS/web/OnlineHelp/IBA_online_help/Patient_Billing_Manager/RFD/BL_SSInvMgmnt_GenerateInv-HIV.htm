Enter the following details:

 

  1. Trx Type – The Transaction Type is displayed as Invoice, by default.

  2. Visit Date - Click the Calendar function to select the Visit Date. This is a mandatory field.

  3. Episode Type – Select an Episode Type to search for the respective Episode Types with the outstanding bill details. You can select from the list of outstanding bills of an Episode Type associated to the Customer to generate invoices for the selected Episode Type.

  4. Outpatient - Select ‘Outpatient’ to search for the Outpatient records with the outstanding bills that are associated to the selected Customer.

  5. External – Select ‘External’ to search for the External patient records with the outstanding bills that are associated to the selected Customer.

  6. Emergency – Select ‘Emergency’ to search for the Emergency patient records with the outstanding bills that are associated to the selected Customer.

  1. [Refresh] - Click [Refresh] to view SS Invoice details.

 

Siriraj Invoice Details

The invoice details associated to the patient gets displayed. Invoice Type, Invoice Number, Invoice Date, Policy Type and Invoice Amount is displayed.

 

Service Charge

The services associated to the respective invoice of the patient are listed.

  1. Customer - Click [List of Values] to select a Customer. The values defined using 'Customer' function with the 'Customer Group' as Social Security get listed. This is a mandatory field.

  2. HIV Service Type - The Service Type associated to the respective patient is displayed.

  3. Service - Service associated to the HIV Service Type is displayed.

  4. Description - The description of respective HIV Service Type is displayed.

  5. Amount - Service coverage amount is displayed.

  6. Total - Total amount is displayed.

  7. Select - Select the check box, if the outstanding amount for service has to be included in invoice generated. The Service Type gets displayed in respective columns under 'HIV Services for Invoice' on the right side.

 

HIV Services for Invoice

  1. HIV Service Type - The Service Type associated to the respective patient is displayed.

  2. Service Amount - Coverage amount for the respective service is displayed.

  3. Mark Up/Down Amt/Invoice Amt - The mark up or mark down amount or the outstanding amount for the respective Service Type is displayed.

  4. Total - The total outstanding amount is displayed.

  5. Signatory1 - The value recorded for Signatory1 is displayed.

  6. Title1 - The value recorded for Title1 is displayed.

  7. Signatory2 - The value recorded for Signatory2 is displayed.

  8. Title2 - The value recorded for Title2 is displayed.

  9. [Generate Invoice] – Click [Generate Invoice] to generate invoice for the selected outstanding bills. The Document Type has to be defined for Invoice records using the function 'Document Types by Payers'. A reference number is generated on clicking [Generate Invoice].

  10. [SS Invoice not Required] - Click [SS Invoice not Required], if invoice is not required for the respective patient record. Invoice is not generated for the selected patient records displayed in 'Siriraj Invoice Details'.
