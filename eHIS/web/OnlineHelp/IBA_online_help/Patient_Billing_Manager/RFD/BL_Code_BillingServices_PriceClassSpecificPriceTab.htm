Price Class Specific Price Tab

Price Class Specific Price tab enables you to set up service price for citizens as well as Non-citizens by defining the prices for a Billing Class associated to a facility.

 

Enter the following details to define the price:

 

  1. Facility - Select a Facility from the list. The facilities associated to the relevant user get listed. This is a mandatory field.

  2. Price Class – Click [List of Values] to select a valid Price Class. The Price Class associated to the relevant Facility gets listed. This is a mandatory field.

  3. Description – The Description for the selected Price Class is displayed.

  4. Valid From – Click Calendar function to select a date. The Price Class Specific Prices are applicable for the Billing Classes in the Facility from the date selected. The date selected has to be greater than the system date. This is a mandatory field.

  5. Valid To - Click Calendar function to select a date. The Price Class Specific Prices are applicable for the Billing Classes in the Facility up to the date selected.

  6. Blng Class – Click [List of Values] to select a valid Billing Class. The Billing Classes defined under the ‘Billing Rule’ menu in the Masters get listed. This is a mandatory field.

  7. Description – The Description for the selected Billing Class gets displayed.

  8. R/F – The values are system defined. Select 'R' for rate based and 'F' for factor based service. Rate based indicates that the charges defined for the quantity is considered during bill generation. Factor based indicates that the services are charged based on the factor, which is defined during Multipricing setup.

  9. IP Rate – Enter the base price to be charged for the Inpatients. The Inpatients associated to the relevant Billing Class are charged based on the quantity of the service used.

  10. DC Rate – Enter the base price to be charged for the Daycare patients. The Daycare patients associated to the selected Billing Class are charged based on the quantity of the service used.

  11. OP Rate - Enter the base price to be charged for the Outpatients. The charge towards the service is calculated accordingly based on the quantity used.

  12. Emg. Rate - Enter the base price to be charged for the Emergency cases. The charge towards the service is calculated accordingly based on the quantity used.

  13. Ext. Rate - Enter the base price to be charged for the External or the Referral patients. The charge towards the service is calculated accordingly based on the quantity used.

  14. Min. Charge - Enter the minimum charge towards the service. The charge towards the service is calculated accordingly based on the quantity used. This is a mandatory field.

  15. Max. Charge – Enter the maximum charge towards the service. This is applicable only when the Cost Indicator is set. This identifies the maximum amount, which can be charged towards the service when the Cost Indicator is set. You cannot charge a service amount beyond the limit specified.

  16. Allowed – Select the check box, if the Billing Service is reimbursable for the selected patient class. Clear the check box, if the respective Billing Class is non-reimbursable for the selected patient class.

  17. Override – Select the check box, if the respective Billing Service is reimbursable for the selected patient class. The override option for the patient class takes precedence over the override check box in the header field. If the service is not reimbursable and the check box is selected for the patient class, it indicates that the Billing Service is reimbursable for the selected patient class.

  18. Reimbursable Amt – Enter the amount to be refunded to the patient on producing valid documents.

  19. Base Qty – Enter the Base Quantity towards the service. The charge towards the service is calculated based on the quantity used. This is a mandatory field.

  20. Gross/Net Applicability – Select either ‘Gross’ or ‘Net’ from the list.

  21. Gross – Select Gross to charge the Gross amount for the service. The charge towards the service is calculated based on the gross amount, that is the discount and the price defined for the Billing Class is considered.

  22. Net – Select Net to charge the Net amount for the service. The service charge is calculated based on the net amount that is the discount amount is not considered. Only the price towards the Billing Class is taken into consideration.

  1. Check Cost Availability – Select the check box, if the Cost Amount is applicable. Clear the check box, if the Cost Amount is not considered towards the charge calculation.

  2. Apply Disc – Select the check box, if a discount is applicable to the selected Billing Class. The Discounts tab is enabled, if the check box is selected. Enter the percentage of discounts to be given to the patients associated to the Billing Class. Clear the check box, if the discount amount is not applicable.

 

Discounts

The Discounts tab enables you to define the percentage of discounts for a patient category. The discount percentage defined is applicable for the selected Price Class.

 

Enter the following details to define discounts:

 

  1. Price Class - The Price Class for which the prices are defined gets displayed, by default. You cannot change the Price Class, but only define the discounts for the patient category.

  2. Billing Class - The Billing Class associated to the relevant Price Class is displayed. You cannot change the Billing Class, but only define the discounts for the patient category.

  3. Valid From – The date selected for the 'From' field  under the Prices tab for the Price Class gets displayed.

  4. Valid To - The end date up to when the prices are valid for the relevant Price Class is displayed.

  5. Discount Indicator - Select a value from the list to issue discount for the Billing Service. The system-defined values are ‘Amount’ and ‘%’. Select Amount to specify the amount that has to be issued as a discount amount on the Billing Service. Select Percentage to specify the percentage of amount that has to be issued as a discount amount on the Billing Service.

  6. Valid From – Click Calendar function to select the date from when the discount percentage is valid for the patient category. This is a mandatory field.

  7. Valid To - Click Calendar function to select the date until when the discount percentage is valid for the patient category.

  8. DC % - Enter the value to issue discount towards the service for Daycare patients. The value is computed as amount or discount based on the value selected for 'Discount Indicator'. This is applicable only to the Daycare patients associated to the relevant Price and Billing Class for the specified period.

  9. IP % - Enter the value to issue discount towards the service for Inpatients. The value is computed as amount or discount based on the value selected for 'Discount Indicator'. This is applicable only to the Inpatients associated to the relevant Price and Billing Class for the specified period.

  10. OP % - Enter the value to issue discount towards the service for Outpatients. The value is computed as amount or discount based on the value selected for 'Discount Indicator'. This is applicable only to the outpatients associated to the relevant Price and Billing Class for the specified period.

  11. EXT % - Enter the value to issue discount towards the service for Referral patients. The value is computed as amount or discount based on the value selected for 'Discount Indicator'. This is applicable only to the external or the referral patients associated to the relevant Price and Billing Class for the specified period.

  12. EMG % - Enter the value to issue discount towards the service for Emergency patients. The value is computed as amount or discount based on the value selected for 'Discount Indicator'. This is applicable only to the Emergency patients associated to the relevant Price and Billing Class for the specified period.
