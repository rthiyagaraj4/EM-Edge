Enter the following details to update package details:

 

Package Details

  1. Package Code - The Package associated to the patient during package subscription is displayed.

  2. Package Description - The description of the package is displayed.

  3. Package Seq No - The Package sequence number is displayed.

  4. Eff Date [From - To] - The effective date during which the package is subscribed for the patient is displayed. Click the Calendar function to update the date. The 'To' date changes automatically based on the 'Valid Days' specified during Package Definition.

  5. Billing Class - The Billing Class associated to the patient is displayed.

  6. Cust Grp Code - The Customer Group Code associated to the patient is displayed.

  7. Cust Code - The Customer code associated to the patient is displayed.

  8. Package Amt - The package amount recorded for the respective package is displayed.

  9. Deposit Amt - The deposit amount recorded for the respective package is displayed.

  10. Utilized Amt - The amount utilized by the patient for the services utilized is displayed.

  11. Price Class - The price class associated to the patient is displayed.

  12. Customer Price - The Customer Price defined for the respective package is displayed.

  13. Encounter Associated - The value is displayed as 'Yes' or 'No' based on whether the package is associated during encounter registration or not.

  14. Practitioner Name - Enter the Practitioner Name in alphanumeric format of 30 characters maximum length or click [List of Values] to select the Practitioner Name.

 

Package Service Details

The services associated to the package are displayed.

  1. Indicator - Select an indicator from the list. System-defined values are 'Billing Service', 'Service Group' and 'Service Classification'.

  2. Service Description - The description of the selected service is displayed.

  3. Package Service - Click [List of Values] to select a value. The values differ based on the Indicator selected.

  4. Rate Charge - The value is either displayed as 'R' or 'C' based on the value selected during Billing Service definition.

  5. Order Catalog Code - The Order Catalog code associated to the selected indicator during package definition is displayed.

  6. Rate Ind - Select a rate indicator from the list. The system-defined values are 'Standard', 'Rule Based', 'User Defined' and 'Price Class'. Select a value to specify the rate indicator. The service is charged based on the price defined.

  7. Factor Rate - Enter the value in numeric format. You can specify the factor only for a rate based service. This is not valid for charge based services.

  8. Qty Limit - Enter the quantity in numeric format. The amount limit id valid for the selected quantity of service.

  9. Limit Ind - Select a value from the list. The system-defined values are 'Perc', 'Amount' and 'Unlimited'.

  10. Amount Limit - Enter the amount limit in numeric format. This indicates the amount that can be utilized from the package on the quantity of service used. You cannot enter the amount limit when the 'Limit Indicator' is selected as 'Unlimited' .

  11. Replaceable - Select the check box to select a service with which the selected service can be replaced. You cannot select the check box for a charge based service. The check box cannot be selected when the 'Indicator' is selected as 'Service Group' or 'Service Classification'.

  12. Replaceable Service - Click [List of Values] to select a replaceable service. The values defined through the 'Billing Services' function are listed. The text box is enabled for entry only if the 'Replaceable' check box is selected.

  13. Replaceable Catalog - The order catalog associated to the replaceable service is displayed.

  14. Refund - Select the check box to issue a refund on a rate or charge based Billing Service. You cannot issue refund for a Service Group or Service Classification.

  15. Refund Rate - Enter the refund rate in numeric format. The refund for the selected Billing Service is calculated based on the selected 'Limit Indicator'.

  16. Auto Refund - Select the check box if automated refund is valid for the service. The refund is calculated automatically before bill generation, if the patient has not utilized the service when the check box is selected. Clear the check box to issue a manual refund.

  17. Closed - Select the check box, if the package has to be closed when the service is subscribed.

  18. Utilized qty - The quantity of service utilized by the patient is displayed.

  19. Utilized Amount - The amount utilized for the respective service is displayed.

  20. Appl Factor - Select the check box to define the factor available for service limit. This field is enabled for selection only if the 'indicator' value is selected as 'factor'. Factor can be applied on the service limit only when the amount limit is specified. You cannot select the check box when the 'Limit Ind' is selected as 'Unlimited'.

  21. [Excl] - Click [Excl] to exclude specific services or service classifications from the package. Exclusion is not valid for a rate based service. You can exclude the services from a package only for charge based services.  

    Enter the following details to exclude details from a charge based service:

    1. Service Ind - Select a value from the list. The system-defined values are 'Service' and 'Classification'. Select 'Service' to exclude a service from the package or 'Classification' to exclude a service classification from the package.

    2. Service Code - Click [List of Values] to select a Service code. The values defined through the 'Billing Services' function are listed.

    3. Rate/Chg - The value is either displayed as 'R' or 'C' based on the service definition.

    4. Exclude All - Select the check box to exclude the service from the package.

    5. Quantity Limit - Enter the quantity limit in numeric format. It indicates the coverage for maximum number of services. You cannot specify the limit if the 'Exclude' check box is selected.

    6. Limit Ind - Select a Limit Indicator from the list. The system-defined values are 'Perc' and 'Amount'. The service price is computed based on the quantity limit and the amount limit specified. You cannot specify the limit indicator if the 'Exclude' check box is selected.

    7. Amount Limit - Enter the amount limit in numeric format. You cannot specify the amount limit if the 'Exclude' check box is selected.

    8. Replaceable - Select the check box, if the selected service can be replaced.

    9. Replaceable Service - Click [List of Values] to select the replaceable service. You can select a value only if the 'Replaceable' check box is selected.

    10. Refund - Select the check box to issue a refund if the service is not utilized by the patient.

    11. Refund Rate - Enter the refund amount in numeric format. You can specify a value only if the 'Refund' check box is selected.

    12. Auto Refund - Select the check box if automated refund is valid for the service. The refund is calculated automatically before bill generation, if the patient has not utilized the service when the check box is selected. Clear the check box to issue a manual refund.

    13. Factor - Select the check box to define the factor available for service limit.

    14. Utilized Qty - The quantity of service utilized by the patient is displayed.

    15. Utilized Amount - The amount utilized for the respective service is displayed.
