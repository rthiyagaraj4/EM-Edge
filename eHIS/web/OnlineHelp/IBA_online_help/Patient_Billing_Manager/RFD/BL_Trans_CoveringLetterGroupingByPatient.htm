Enter the following details to create a Covering Letter for the patients in the facility:

 

  1. Patient ID – Click [List of Values] to select a valid Patient ID. This is a mandatory field.

  2. Name – The Name of the selected Patient gets displayed.

  3. Cust Code - Click [List of Values] to select a Customer Code. The selected Patient ID has the general format setup for the Customer associated.

  4. Cust Name – The Name of the selected Customer gets displayed.

  5. Group Code - The Group Code associated to the selected Customer gets displayed. This is a mandatory field.

  6. Description – The Description for the selected Group Code gets displayed on selecting the Group Code.

  7. Print Text – Enter the text in alphanumeric format of 60 characters maximum length.

 

Contact Details

  1. Contact Name – The name of the contact person recorded for the Customer using the function ‘Covering Letter Group Setup’ gets displayed. You cannot update the Contact Name.

  2. Contact Name Title – Enter the designation of the contact person in alphanumeric format of 40 characters maximum length.

  3. Tel Num – Enter the Telephone Number of the contact person in numeric format of 30 digits maximum length.

  4. Fax Num – Enter the Fax Number of the contact person in numeric format of 30 digits maximum length.

  5. Email ID – Enter the Email ID of the contact person in alphanumeric format of 30 characters maximum length.

 

Mailing Address

  1. Mail Addr Line [1 – 4] – Enter the postal address of the contact person in alphanumeric format of 40 characters maximum length.

  2. Country – The country selected for the Customer using the function ‘Covering Letter Group Setup’ gets displayed. You cannot update the country associated.

  3. Postal Code – Enter the Postal Code in alphanumeric format of 20 characters maximum length.

  4. Post Box – Enter the Post Box number in numeric format of 20 digits maximum length.
