Enter the following details to define Enterprise Service Groups/Line:

 

  1. Enterprise Service Group - Enter the Enterprise Service Group code in alphanumeric format of five characters maximum length. The Enterprise Service Group or the Print Lines defined has to be associated to a relevant Billing Service. The Enterprise Service Group and Line details are printed on the bill for the associated service. This is a mandatory field. Some examples of the Enterprise Service Group code are as follows:

  2. 'AD' for Ward Charges

  3. 'HC' for Facility Charges

  4. 'HD' for Facility Doctor's Fee

  1. Long Description - Enter the Long Description for the Enterprise Service Group in alphanumeric format of 80 characters maximum length. This is a mandatory field.

  2. Short Description - Enter the Short Description in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  3. Consider for Income Reporting – Select the check box, if the patient’s fee is considered as the income for the respective facility. The revenue generated is not considered for income reporting, if the check box is cleared. For Example, outside doctor’s fees may be collected by the Facility, but cannot be considered as the income for the facility, since this has to be paid back to the doctor from an external Facility.

 

Print Line Details

  1. Print Line - Enter the Print Line code in alphanumeric format of two characters maximum length. You have to associate several Print Lines to an Enterprise Service Group/Line. These individual Print Lines are associated to a Billing Service. This is a mandatory field.

  2. Long Description - Enter the Long Description in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the Short Description in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Document Print Sequence – Enter a numeric value to indicate the Document Print Sequence. The sequence identifies the order in which the Print Line details appear on the bill. Based on the sequence set, the Print Line is displayed in ascending order in the bill.
