Press [F8] or click [Execute Query] to navigate through the Customer records and associate a bill format. You can associate the Undercharge advice to the patient bill.

Enter the following details to associate multiple bill formats to a Customer/Cash Billing Group:

Customer

  1. Cust Code - The Customer Codes defined through the Accounts Receivable module are listed on navigation.

  2. Cust Group Code - The Customer Group Code associated to the Customer is displayed automatically.

  3. Description - The description of the respective Customer Group is displayed.

  1. Format Code - Click [List of Values] to select a Format Code. The format code that is defined through the 'Bill Format' tab is displayed.

  2. Description - The description of the respective Format Code is displayed.

  3. Bill Type - Select a Bill Type from the list. The system-defined values are 'OP Visit Bill', 'IP Discharge Bill', 'IP Undercharge Bill' and 'External Bill'. The selected Bill Type is displayed for the selected Customer Group. This is a mandatory field.

Cash Billing Group

Press [F8] or click [Execute Query] to navigate through the Billing Group records and associate it to the Customer and bill format.

  1. Blng Grp ID - The Billing Groups defined through the 'Billing Group' function are listed on navigation.

  2. Description - The description of the respective Billing Group is displayed.

  3. Format Code - Click [List of Values] to select a Format Code. The format code that is defined through the 'Bill Format' tab is displayed.

  4. Description - The description of the respective Format Code is displayed.

  5. Bill Type - Select a Bill Type from the list. The system-defined values are 'OP Visit Bill', 'IP Discharge Bill', 'IP Undercharge Bill' and 'External Bill'. The selected Bill Type is displayed for the selected Cash Billing Group. This is a mandatory field.
