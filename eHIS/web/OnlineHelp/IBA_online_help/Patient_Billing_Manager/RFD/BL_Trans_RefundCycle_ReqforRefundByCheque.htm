Enter the following details to generate the report:

 

  1. Request Doc Date - Enter a range of request date in dd/mm/yyyy format. If the date is not specified, the system assumes the lowest value for the ‘From’ field and highest value for the ‘To’ field.

  2. Request Doc Type - Enter the range for Request Document Type. If the document type is not specified, the system assumes the lowest value for the ‘From’ field and highest value for the ‘To’ field.

  3. Request Doc Number - Enter the range for Request Document Number. If the document number is not specified, the system assumes the lowest value for the ‘From’ field and highest value for the ‘To’ field.

  4. Request Serial No. - Enter the range for the Request Serial Number. If the serial number is not specified, the system assumes the lowest value for the ‘From’ field and highest value for the ‘To’ field.

  5. Original Doc Ind - Select a value from the list. The values are system-defined - “Refund against Bill & DCP”, “All” and “Refund against Deposits”.

  6. Original Doc Date [From-To] - Enter the date for the original document date in dd/mm/yyyy format. If the original document date is not specified, the system assumes the lowest value for the ‘From’ field and highest value for the ‘To’ field.

  7. Original Doc Type [From-To] - Enter the range for the original document type. If the original document type is not specified, the system assumes the lowest value for the ‘From’ field and highest value for the ‘To’ field.

  8. Original Doc Number [From-To] - Enter the range for the original document number. If the original document number is not specified, the system assumes the lowest value for the ‘From’ field and highest value for the ‘To’ field.
