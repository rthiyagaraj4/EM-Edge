Enter the following details to print the bill:

 

  1. Ward Code [From-To] – Click [List of Values] to view and to select a range of Ward Codes. The nursing unit types defined in the IP Code Setup in the Inpatient and Day Care Management module is available in the list.

  2. Admission No [From-To] - Click [List of Values] to select a range of Admission Number. The list gets validated with the Admission Numbers generated at the time of Inpatient registration.

  3. Detail – Select the check box, if the report requires the Quantity and the Transaction Date to be displayed for the billing service along with the patient details.

  4. [Search] – Click [Search] to view all the Inpatient details.

  5. Bed Charge Computed Upto Date – The date when the bed charge was computed previously is displayed.

  6. No. of Days to Run – The number of days from for which the bed charges were not calculated is displayed. This represents the number of days pending from the date when the bed charge was calculated initially until the current system date.

  7. [Process Bed Charges] – Click [Process Bed Charges] to process the bed computation charge for all the Inpatients. A message gets displayed saying the bed charge is being computed when the bed charge is successfully computed.

  8. [Recalculate] - Click [Recalculate] to recalculate the charges for the selected patient before draft report generation.

  9. Payable Type – Select a Payable Type from the list. The values are system defined – “Patient”, “Payer” and “Both”.

  10. Payer – Click [List of Values] to select a Payer. A Payer can be selected from the list only when the Payable Type is set to “Payer”.

  11. Payable Amt Exceeding – Enter the Amount in numeric format. A list of all the Inpatients who have an outstanding amount greater than the amount specified gets listed.

 

Current Inpatients

Click on the field under Patient No for the details to load. The inpatient details for the respective Ward Code and the Admission Number will be displayed. Double-click to select a record.
