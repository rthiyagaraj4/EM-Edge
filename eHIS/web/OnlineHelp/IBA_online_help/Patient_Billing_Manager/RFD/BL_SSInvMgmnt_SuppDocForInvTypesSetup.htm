Enter the following details to associate Support Document to an Invoice Type:

 

  1. Invoice Type - Click [List of Values] to select an Invoice Type. The values defined using 'SS Invoice Types' function get listed. This is a mandatory field.

  2. Support Document Type Code - Click [List of Values] to select a Support Document Type.  The values defined using 'Support Document Master' function get listed. This is a mandatory field.

  3. Document Series - The Document Series associated to the selected Support Document Type gets listed.

  4. Support Doc Auth Signatory 1 - Enter the name of the Authorized Signatory in alphanumeric format. This is a mandatory field.

  5. Support Doc Signatory Title 1 - Enter the designation of the Authorized Signatory in alphanumeric format. This is a mandatory field.

  6. Support Doc Auth Signatory 2 - Enter the name of the Authorized Signatory in alphanumeric format.

  7. Support Doc Signatory Title 2 - Enter the designation of the Authorized Signatory in alphanumeric format.
