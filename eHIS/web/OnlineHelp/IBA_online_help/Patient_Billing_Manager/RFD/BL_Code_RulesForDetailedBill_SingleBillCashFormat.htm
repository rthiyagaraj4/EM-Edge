Single Bill Cash Format

The Single Bill Cash Format tab enables you to define a bill format for a Billing Group. If a Payer Specific Bill Format is not defined, the bill is printed based on the bill format defined using the Multipayer Bill Format tab. System displays Hospital Details, Patient Details, Receipt/Refund Details and Text Formatting list values in the text field in each tab. Right-click to select a value from the list. Various formats with several values are displayed on selecting a value from the list. Double-click on a value to select it.

The format lists styles to format the legends in header, footer and body section of a receipt format.

The format lists receipt/refund related transaction details.

The format lists patient demographic details.

The format lists complete information of the hospital and demographic details.

 

Enter the following details to define a Single Bill Cash Format:

 

  1. Facility – The value is defaulted to the logged in facility, by default. This is a mandatory field.

  2. Billing Group – Click [List of Values] to select a Billing Group. The values defined using the Billing Group function gets listed. The bill format is valid for the selected Billing Group. This is a mandatory field.

  3. Total Payable Service Line As - Select a value from the list. The values are system defined – “Gross Amt” and “Net Amt”.

  4. Gross Amt – Select Gross Amt to print the complete bill details along with the discounts amount.

  5. Net Amt – Select Net Amt to print only the net charges in the bill.

  1. Total Payable Service Line Discount As - Select a value from the list. The values are system defined – “Row”, “Column” and “Not Applicable”.

  2. Row – Select Row to print the discount details in a Row.

  3. Column – Select Column to print the discount details in a Column.

  4. Not Applicable - Select Not Applicable, if the discount details are not to be printed.

  1. Reimbursable Amount – Select the check box to include a column in the bill with the column name as Reimbursable Amount. The reimbursable amount for the services has to be specified in the Reimbursable Amount column during bill generation. Clear the check box and the Reimbursable Amount column is not printed on the bill.

  2. Non-Reimbursable Amount - Select the check box to include a column in the bill with the column name as Non-Reimbursable Amount. The non-reimbursable amount for the services has to be specified in the column during bill generation. Clear the check box and the non-reimbursable amount column is not printed on the bill.

  3. Print Bill Adjustments As - Select a value from the list. The values are system defined – “Summarized Row”, “Separate Row” and “Donot Print”.

  4. Summarized Row – Select Summarized Row to print a summary of the bill details in a single row.

  5. Separate Row - Select Separate Row to print a summary of the bill details in a separate row.

  6. Donot Print – Select Donot Print, if the details are not to be printed.

  1. Bill Discount as Separate Row – Select ‘Bill Discount as Separate Row’ to print the bill discounts in a separate row. The check box is selected automatically when the ‘Print Bill Adjustments As’ is set as Separate Row. The check box is enabled only when ‘Print Bill Adjustments As’ is selected as ‘Summarized Row’ or ‘Separate Row’. The check box is disabled, if the ‘Print Bill Adjustments As’ is selected as ‘Do Not Print’.

  2. Bill Adjustment as Separate Row - Select ‘Bill Adjustment as Separate Row’ to print the bill adjustment details in a separate row. The check box is selected automatically when the ‘Print Bill Adjustments As’ is set as a Separate Row. The check box is enabled only when ‘Print Bill Adjustments As’ is selected as ‘Summarized Row’ or ‘Separate Row’. The check box is disabled, if the ‘Print Bill Adjustments As’ is selected as ‘Do Not Print’.

  3. Bill Exemption as Separate Row - Select ‘Bill Exemption as Separate Row’ to print the bill exemption details in a separate row. The check box is selected automatically when the ‘Print Bill Adjustments As’ is set as Separate Row. The check box is enabled only when ‘Print Bill Adjustments As’ is selected as ‘Summarized Row’ or ‘Separate Row’. The check box is disabled, if the ‘Print Bill Adjustments As’ is selected as ‘Do Not Print’.

  4. Bill Write-Off as Separate Row - Select ‘Bill Write-Off as Separate Row’ to print the bill write-off details in a separate row. The check box is selected automatically when the ‘Print Bill Adjustments As’ is set as Separate Row. The check box is enabled only when ‘Print Bill Adjustments As’ is selected as ‘Summarized Row’ or ‘Separate Row’. The check box is disabled, if the ‘Print Bill Adjustments As’ is selected as ‘Do Not Print’.

  5. Net Payable Amount as – Select a value from the list. The values are system defined - “Row”, “Column” and “Both”.

  6. Row – Select Row to print the net payable amount towards the service in a row.

  7. Column – Select Column to print the net payable amount towards the service in a column.

  8. Both - Select Both to print the net amount details in a row and column.

  1. Print Legends – Select the check box to print the legends in the report. Clear the check box, if the legends are not to be printed in the report.

  2. Print Legends for Detailed Receipt – Select the check box to print the legends in the detailed report. Clear the check box, if the legends are not to be printed in the detailed report.
