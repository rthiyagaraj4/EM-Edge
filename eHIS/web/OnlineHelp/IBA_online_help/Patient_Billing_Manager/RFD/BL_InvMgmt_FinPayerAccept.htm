Enter the following details to finalize the payment acceptance:

 

Dispatch Letter

  1. Document Ref Num – Enter the Document Reference Number in alphanumeric format of 30 characters maximum length. The Document Reference Number indicates the number on the acceptance document posted by the Payer with the payment.

  2. Date – Enter the Date in dd-mm-yyyy format. The date indicates when the document was received by the facility.

  3. Customer Code – Click [List of Values] to select the Customer Code. The documents, which are dispatched to the selected Customer, gets listed.

  4. Tot Outst Amt – The total outstanding amount for the services gets displayed. You cannot update the total outstanding amount. It is displayed based on the invoices associated to the dispatch document.

  5. Confirmation Date – Click Calendar function to select the date. This indicates the date when the payer acceptance is confirmed.

  6. Foreign Cur. Note – Enter the Foreign Currency Note in alphanumeric format of 100 characters maximum length.

  7. Authorized Signatory – The name and designation of the Authorised Signatory recorded in the ‘Invoice Process Parameters’ function gets displayed. You can update the name and designation before finalizing the payer acceptance.

  8. No of Docs – The number of documents associated to the dispatch letter gets displayed.

  9. Order By – Select a value from the list. The value selected indicates the order in which the document details are to be displayed. The values are system defined – ‘Doc Date, Doc Type, Doc Num’, Document Not Verified’, ‘Doc Srl No’ and ‘Doc Outst Amt’.

 

Dispatch Letter Print Format

  1. Body Header – Select Body Header, if the Body Header information defined using the function ‘Invoice Process Parameter’ has to be included in the Dispatch Letter.

  2. Body Footer – Select Body Footer, if the Body Footer information defined using the function ‘Invoice Process Parameter’ has to be included in the Dispatch Letter.

  3. Print Text – Select Print Text, if the Print Text information defined using the function ‘Invoice Process Parameter’ has to be included in the Dispatch Letter.

  4. In Foreign Currency – Select Foreign Currency, if the Foreign Currency information defined using the function ‘Invoice Process Parameter’ has to be included in the Dispatch Letter.

  5. Foot Note – Select Foot Note, if the Foot Note information defined using the function ‘Invoice Process Parameter’ has to be included in the Dispatch Letter.

 

Dispatch Document Details

  1. Slno – The Serial Number in which the invoices were generated gets displayed.

  2. Bill Status – The Bill Status is displayed as either Debit Note, Credit Note or Invoice based on the Bill Status.

  3. Doc Number – The document type and the number created for the selected bills while generating the invoice gets displayed.

  4. Doc Date – The date when the invoice document was created for the selected bills get displayed.

  5. Add Text to be Printed for Invoice – The text recorded while generating the invoice gets displayed.

  6. Doc Amt – The Amount for the selected invoice document gets displayed.

  7. Verified – Select the check box, if the documents have been verified.

  8. Rejected by Payer – Select a value from the list. The values are system defined – ‘Accepted’ and ‘Rejected’.

  9. Accepted – Select Accepted, if the Payer has accepted the invoice dispatched.

  10. Rejected – Select Rejected, if the Payer has rejected the invoice dispatched.

  1. Reject/Accept All – Select the check box to mark all the invoices as either rejected or accepted by the Payer.

  2. [Details] – Click [Details] to view the address and the details of the contact person.

  3. [Finalize] – Click [Finalize] to finalize the acceptance of the Customer.

  4. [Close] – Click [Close] to exit from the screen.
