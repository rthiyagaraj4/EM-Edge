Enter the following details to amend charges for the material items:

 

  1. Facility – Select the Facility from the list. The facilities associated to the logged in user get listed. The prices defined are valid for the facility selected.

  2. Billing Service – Click [List of Values] to select a Billing Service. The Billing Services defined as ‘Materials Item-related Service’ get listed. The description for the selected Billing Service gets displayed. This is a mandatory field.

  3. Apply To – Select a value from the list. The values are system defined – All, Standard, Price Class, Billing Group and Price Class. This is a mandatory field.

  4. All – Select All, if the prices defined for a Billing Class are applicable to Standard prices, Price Class, Billing Groups and the Price Class defined for the selected Billing Service associated to the facility.

  5. Standard – Select Standard, if the prices defined for a Billing Class are applicable to Standard prices defined for the selected Billing Service associated to the facility.

  6. Price Class – Select Price Class, if the prices defined for a Billing Class are applicable to Price Class Specific Prices defined for the selected Billing Service associated to the facility.

  7. Billing Group – Select Billing Group, if the prices defined for a Billing Class are applicable to Billing Group Specific Prices defined for the selected Billing Service associated to the facility.

  1. Indicator – Select an Indicator from the list. The values are system defined – Standard, Customer, Billing Group and Price Class. This is a mandatory field.

  2. Standard – Select Standard to define the prices for a Billing Class for a specific period and to associate it to the materials-item linked Billing Services in the facility.

  3. Customer – Select Customer to define the prices for a Billing Class for a specific period and to associate it to the materials-item linked Billing Services in the facility.

  4. Billing Group – Select Billing Group to define the prices for a Billing Class for a specific period and to associate it to the materials-item linked Billing Services in the facility.

  5. Price Class - Select Price Class to define the prices for a Billing Class for a specific period and to associate it to the materials-item linked Billing Services in the facility.

  1. Code – Click [List of Values] to select a code. The values in the list differ based on the Indicator selected. This is a mandatory field.

  2. The Billing Group Specific Price and discount percentage defined for a billing class and associated to the selected Indicator and Material-Linked Billing Service get displayed, if already defined. To define the price for a new Billing Group, select a value from the list. The values defined using the Billing Group function get listed.

  3. The Standard Price and discount percentage defined for a billing class and associated to the selected Indicator and Material-Linked Billing Service get displayed, if already defined. To define a new Standard Price, select a value from the list. The facilities associated to the Standard Price defined in the Billing Services get listed.

  4. The Price Class Specific Price and discount percentage defined for a billing class and associated to the selected Indicator and Material-Linked Billing Service get displayed, if already defined. To define a new Price Class Specific Price, select a value from the list. The values defined using the Price Class Specific Price function get listed.

  5. The Customer Specific Price and discount percentage defined for a billing class and associated to the selected Indicator and Material-Linked Billing Service get displayed, if already defined. To define a new Customer Specific Price, select a value from the list. The values defined using the function Customer get listed.

  1. Description – The Description for the selected code gets displayed.

  2. Valid From – Click Calendar function to select a date for new records. The price defined is valid for the selected billing class, Indicator and the Billing Service from the date selected. The date gets displayed automatically when the price is defined initially for the billing class indicator selected and associated Billing Service. This is a mandatory field.

  3. Valid To - Click Calendar function to select a date for new records. The price defined is valid for the selected billing class, Indicator and the Billing Service until the date selected. The date gets displayed automatically when the price is defined initially for the billing class indicator selected and associated Billing Service.

 

Pricing by Billing Class for Billing Service

  1. Billing Class – Click [List of Values] to select a Billing Class. The Description for the selected Billing Class gets displayed. This is a mandatory field.

  2. Public – Select the check mark to define the public price for the billing class associated to the indicator selected and the Billing Service in the facility. This indicates if the price value is a Government defined price towards the material item or not. The price for the selected Billing Service cannot exceed the Public price defined.

  3. Basis – Select a value from the list. The values are system defined– Average Cost, Base Price and Computed Value.

  4. Base Price – Select Base Price, if the prices for the individual material items are to be calculated based on the price defined in the excel file and saved in the database. The files are imported to the database using the Upload Material Sale Price function.

  5. Average Cost – Select Average Cost, if the individual material items are to be charged based on the prices defined using the function Material Item Charging Rule function.

  1. IP Factor – Enter the factor in numeric format. The price is calculated as the product of factor and the billing service price for Inpatients. The text box is enabled, if the selected Billing Class is inpatient type.

  2. DC Factor - Enter the factor in numeric format. The price is calculated as the product of factor and the billing service price for Daycare patients. The text box is enabled, if the selected Billing Class is Daycare.

  3. OP Factor - Enter the factor in numeric format. The price is calculated as the product of factor and the billing service price for Outpatients. The text box is enabled, if the selected Billing Class is valid for Outpatients.

  4. Emer. Factor – Enter the factor in numeric format. The price is calculated as the product of factor and the billing service price for Emergency patients. The text box is enabled, if the selected Billing Class is valid for Emergency patients.

  5. Ext.Factor – Enter the factor in numeric format. The price is calculated as the product of factor and the billing service price for External/Referral patients. The text box is enabled, if the selected Billing Class is valid for External/Referral patients.

  6. Min. Charge - Enter the Minimum Charge for the Billing Class.

  7. Max Charge - Enter the upper limit for the amount, which can be issued as a refund towards the service.

  8. Base Qty. – Enter the Base Quantity towards the service. The service charge is calculated based on the price and the quantity of the service used.

  9. Gross/Net Applicability – Select a value from the list. The values are system defined – Gross and Net.

  10. Gross - Select Gross to charge the Gross amount for the service. The charge towards the service is calculated based on the discount and the price defined for the billing class, associated to an indicator and a Billing Service in a Facility.

  11. Net – Select Net to charge the Net amount for the service. The service charge is calculated based on the price defined for the billing class, associated to an indicator and a Billing Service in a Facility. The discount amount for the service is not considered during bill generation.

  1. Check Cost Availability – Select the check box, if the Cost Amount is applicable for the selected billing class, associated to an indicator and a Billing Service. Clear the check box, if the Cost Amount is not considered towards the charge calculation.

  2. Apply Disc – Select the check box, if discount is applicable to the respective Billing Class associated with the relevant Billing Service.

 

Applicable Discounts for Service and Billing Class

  1. IP% - Enter the discount percentage intended for Inpatients. The price is calculated based on the percentage of amount and the service price defined. If the discount percentage is already defined, the values get displayed. Enter the new discount percentage to update the existing values in numeric format. This is a mandatory field.

  2. DC% - Enter the discount percentage intended for Daycare patients. The price is calculated based on the percentage of amount and the service price defined. If the discount percentage is already defined, the values get displayed. Enter the new discount percentage to update the existing values in numeric format. This is a mandatory field.

  3. OP% - Enter the discount percentage intended for Outpatients. The price is calculated based on the percentage of amount and the service price defined. If the discount percentage is already defined, the values get displayed. Enter the new discount percentage to update the existing values in numeric format. This is a mandatory field.

  4. Emg% - Enter the discount percentage intended for Emergency patients. The price is calculated based on the percentage of amount and the service price defined. If the discount percentage is already defined, the values get displayed. Enter the new discount percentage to update the existing values in numeric format. This is a mandatory field.

  5. Ext% - Enter the discount percentage intended for External/Referral patients. The price is calculated based on the percentage of amount and the service price defined. If the discount percentage is already defined, the values get displayed. Enter the new discount percentage to update the existing values in numeric format. This is a mandatory field.

  6. Valid From – Click Calendar function to select a date. The discount percentage defined for the billing class is valid from the date selected. The discount period has to be within the Billing Service period and has to be greater than the system date. This is a mandatory field.

  7. Valid To - Click Calendar function to select a date. The discount percentage defined for the patient class is valid until the date selected. The discount period has to be within the Billing Service period. The date selected has to be greater than the Valid From date. This is a mandatory field.
