Enter the following details to define counseling details:

 

  1. Counseling Code – Enter the Counseling Code in alphanumeric format of two characters maximum length. The Counseling Code is unique Code and cannot be changed after creation. This is a mandatory field.

  2. Counseling Legend – Enter a valid description for the Counseling Code. This Counseling Legend indicates the reason for exemption. This is a mandatory field.

  3. Applicable for Transaction – Select a value from the list. The values are system defined – 'Exemption' and 'Instalment Plan'.

  4. Exemption – Select Exemption, if the Counseling Code is applicable only towards any exemption for the patients.

  5. Installment Plan – Select Installment Plan, if the Counseling Code is applicable only towards permitting the patient to pay through installments.

  1. Applicable to Sex - Select a value from the list. The values are system defined – 'Both', 'Male' and 'Female'.

  2. Mandatory – Select the check box, if the Counseling Code is to be made mandatory during Counseling. Clear the check box, if the Counseling Code is not mandatory. The selected Counseling Code is made mandatory in the Exemption Setup function.

  3. Counseling Value Type – Select a value from the list. The values are system defined – 'List Element', 'Number'  and 'Free Text'.  

  4. List Element – Select List Element to associate a list of values to the Counseling Legend.

  5. Number – Select Number to associate numeric details to the Counseling Legend.

  6. Free Text – Select Free Text to record the details in alphanumeric format of 500 characters maximum length.

 

On selecting List Element, enter the following details:

List of Values

  1. Id – Enter a valid ID with a maximum of three characters. This is a mandatory field.

  2. Value – Enter a Value for the ID. The value describes the action type to be associated to the Counseling Code. This is a mandatory field.

  3. Action – Select an Action from the list. The values are system defined – 'Accept', 'Reject', and 'Warning'.

  4. Default – Select the check box, if the selected ID has to be displayed, by default.

 

On selecting Numeric Details, enter the following details:

  1. UOM – Click [List of Values] to associate a Unit of Measure with the Counseling Code. The values are listed, only if the Unit of Measure is defined in the Application Masters module. This is a mandatory field.

  2. Lower Range –Specify the minimum range of unit of measure, which can be associated with the Counseling Code.

  3. Higher Range – Enter the maximum range for the unit of measure details to be associated with the Counseling Code.

  4. Default Value – Enter the default value for the unit of measure. This value will be considered when no other values are defined for the unit of measure.

  5. Out of Range Action Type – The Action Types are system defined – 'Accept', 'Reject', and 'Warning'.

 

On selecting Free Text, enter the following details:

Free Text

  1. Detail Description - Enter the details in alphanumeric format of 500 characters maximum length. You can record additional details that are required while counselling the patient.
