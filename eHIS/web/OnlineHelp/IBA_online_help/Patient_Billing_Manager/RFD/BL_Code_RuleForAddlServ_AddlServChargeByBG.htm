The Addl Service Charge by Billing Group tab enables you to define the additional charges on selected Billing Groups. The patients with the respective Billing Group are charged additional service charges. The effective date for the rule code to be associated with the Billing Group has to be the same as the effective date mentioned for the rule code under the ‘Additional Service Charge Rule’ tab.

 

Enter the following details to define additional service charges by Billing Group:

 

  1. Billing Group – Click [Execute Query] or press [F8] to view and to associate the rule codes to the Billing Groups. The values defined using the Billing Groups function get displayed.

  2. Rule Code – Click [List of Values] to select the Rule Code. The Rule Codes defined using the ‘Additional Service Charge Rule’ tab gets displayed. This is a mandatory field.

  3. Description – The description for the selected Rule Code gets displayed.

  4. Effective [From – To] – Click the Calendar function to select the date. The ‘From’ date selected has to be greater than the system date. The effective date has to match with the date range selected for the rule code under the ‘Additional Service Charge Rule’ tab.

The Sub Rule Types associated to the Main Rule Types linked to the respective Billing Group gets displayed. You can click  to view the sub rule types associated to each main rule.
