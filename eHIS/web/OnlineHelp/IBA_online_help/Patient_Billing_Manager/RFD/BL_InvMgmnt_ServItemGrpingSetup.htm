Enter the following details:

 

  1. Facility ID - The name of the logged-in Facility is displayed, by default. You cannot change the Facility ID.

  2. Payer Group Code - Click [List of Values] to select a Payer Group Code. The description of the selected Payer Group is displayed.

  3. Payer - Click [List of Values] to select a Payer Code. The description of the selected Payer is displayed.

  4. Policy Type Code - Click [List of Values] to select a Policy Type Code. The selected Policy Type is associated to the Policy Group.

  5. Re-Generate Invoice Allowed For OP/AE - Select the check box to re-generate invoice for Outpatients' or Emergency patients'. The check box is selected, by default. Clear the check box to restrict invoice re-generation for Outpatients' or Emergency patients'.

  6. Re-Generate Invoice Allowed For IP/DC - Select the check box to re-generate invoice for Inpatients' or Daycare patients'. The check box is selected, by default. Clear the check box to restrict invoice re-generation for Inpatients' or Daycare patients'.

 

Service/Item Groups

  1. Code - Enter the Service/Item Group code in alphanumeric format of two characters maximum length. This is a mandatory field.

  2. Long Description - Enter the Long Description for Service/Item Group code in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the Short Description for Service/Item Group code in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Cancer Drug - Select the check box, if the Service/Item Group is valid for Cancer Drugs. The check box is cleared, by default.

    Bill Generation:

    • During Payer bill generation, separate bills are generated for normal services and cancer medicine.

    Invoice Generation:

    • A single invoice is generated for UC IP DRG and Cancer Drug bills.

    Invoice Re-generation:

    • Default Payer is displayed as per the setup, but you can update the Payer.

    • Only UC IP Cancer/Femera Drugs are listed for invoice re-generation.

    Debit Note/Credit Note Regeneration:

    • UC IP Cancer Drug Bill is generated after invoice generation for Debit Note/Credit Note Regeneration.

  5. Femera Drug - Select the check box, if the Service/Item Group is valid for Femera Drugs. The check box is cleared, by default.

    For femera there is one drug which is given to patient. Siriraj has contract with UC to invoice for femera drug at an agreed amount. For example, Siriraj can invoice only a fixed amount, that is, 90 Baht per quantity of femera drug for patients with UC Femera policy which covers the Femera drug. Siriraj also has a contract with Drug company where in the invoice for femera drugs is sent  at the cost of normal drugs. Drug company in return supplies drugs in a batch.

  6. Haemodialysis - Select the check box, if the Service/Item Group is valid for Haemodialysis. The check box is cleared, by default.

    This coverage is applicable only for Chronic Dialysis patients.

    Bill Generation

    • Separate bill is generated for each Haemodialysis Billing Service.

    • For Inpatients' an interim bill is generated for each session.

    • For Inpatients' and Outpatients' there will be a single Haemodialysis service for a day.

    • A bill can have single Haemodialysis charge.

    Invoice Generation:

    • Separate invoice is generated for each Haemodialysis service.

You can select any one of the check boxes mentioned above.

  1. Default Regen. Invoice Payer - The selected Payer code is displayed as the default registration invoice payer. Click [List of Values] to select any other Payer. The Payers associated to the selected Payer Group are listed. You can select a default registration invoice paid only if the checkbox 'Re-Generate Invoice Allowed For OP/AE' or 'Re-Generate Invoice Allowed For IP/DC' is selected.

  2. [Service/Items By Group] - Click [Service/Items By Group] to navigate to the 'Service/Items By Group' tab and associate Items to the Service/Item Group.

 

Service/Items By Group

  1. Indicator - The listed values are 'Service' and 'Items'. The value is defaulted to 'Item' if the check box 'Femera Drug' or 'Cancer Drug' is selected. You can select 'Service' or 'Item' if the 'Haemodialysis' check box is selected.

  2. Service/Item - Click [List of Values] to select an item and associate it to the Service/Item Group. This is a mandatory field.

  3. Description -  The description of the selected Service/Item is displayed.

  4. Effective [From - To] - Click the Calendar function to select the date range during which the item is associated to the Service/Item Group. The 'From' date is mandatory and it has to be greater than the current system date.
