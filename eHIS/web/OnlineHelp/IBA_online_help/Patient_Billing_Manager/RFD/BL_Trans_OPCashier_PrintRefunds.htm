Enter the following details to print the refund receipts in batch mode:

 

  1. Episode Type - List of episode types attached are – 1. Outpatient, 2. Emergency and 3.External

  2. Document Number –From - Enter the document type code and number for the FROM range of the document numbers. Leave spaces to select all from the beginning.  Receipt/Refund Look up is provided to select the valid values from the list.

  3. Document Number –To - Enter the document type code and number for the TO range of the document numbers. Leave spaces to select all till the end.  Receipt/Refund Look up is provided to select the valid values from the list.

  4. Document Date –From - Enter the document date for the FROM range of the document dates. Leave spaces to select all from the beginning.

  5. Document Date –To - Enter the document date for the TO range of the document dates. Leave spaces to select all till the end.
