Enter the following details to define the additional service charge rule:

 

  1. Facility – Select the Facility from the list. The facilities associated to the logged in user gets listed. The service charge rule defined is valid within the facility selected.

  2. Rule Code – Enter the Rule Code in alphanumeric format of five characters maximum length. This is a mandatory field.

  3. Long Description – Enter the Long Description for the rule code in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  4. Short Description – Enter the Short Description for the rule code in alphanumeric format of five characters maximum length.

  5. Effective From – Click Calendar function to select the date. The rule code is valid within the selected facility from the date selected. The date selected has to be greater than the current system date. This is a mandatory field.

  6. To – Click Calendar function to select the date. The service charge is valid within the facility until the date selected.

  7. Code – Enter the VAT code in alpha numeric format in three character length to print the code in bills.

  8. Rule Type – Select a value from the list. The values are system defined – Main and Sub. Select Main, if the rule code is a main rule type. You can associate any number of Sub Rule Types to the Main Rule Type. Select Sub, if the Rule Type is a Sub rule code. On selecting the Rule Type as ‘Sub’, additional field gets displayed to associate the Sub Rule Type to a Main Rule Type. Select a Main Rule to which the Sub Rule Type has to be associated. A Main Rule Type can have multiple Sub Rule Types associated to it. The sub rule types associated to the main rule types appear under Associated Charges menu on the screen.

  9. Payer Coverage – Select the check box, if the Payer Coverage is applicable for the rule code. Clear the check box, if the Payer Coverage is not applicable for the selected rule code.

  10. Additional Charge – Enter the additional charge that has to be charged on associating the respective rule code to the patient bills. Select either ‘%’ or ‘Value/Unit’. The percentage of the additional charge is charged as the tax amount.  The value or the unit specified is charged as the additional amount on the total bill amount.

  11. Apply On – Select a value from the list. The values are system defined – ‘Gross Charge’, ‘Net Charge’, ‘Main Rule’ and ‘Sub Rule’. Select ‘Gross Charge’, if the additional amount has to be charged on the gross amount during bill generation. Select ‘Net Charge’, if the additional amount has to be charged on the net amount during bill generation. Select ‘Main Rule’ to charge the additional amount on the main rule. Select ‘Sub Rule’ to charge the additional amount on the sub rule.

  12. Amount Capping Limit - Enter the capping amount in the text field. Only numeric values can be entered up to 10 characters in length. If the rule is associated to a billing service, based on the capping, GST will be calculated and charged. The capping amount is checked for each quantity of the service.

  13. Amount Capping Apply On – Select whether capping amount to be applied to Gross Amount or Net Amount from the drop-down list for GST calculation. The Amount Capping fields are applicable only for Indian customers.

  14. GL Interface Code – Click [List of Values] to select the GL Interface Code. A interface code associated to a rule code cannot be associated to another rule. The values defined using the ‘Chart of Accounts’ function under the ‘GL Setup’ menu gets listed. This is a mandatory field.

  15. Text to Print in Bill – Enter the text in alphanumeric format with a maximum of 50 characters. The value recorded gets displayed on the bill generated.

  16. Payer Coverage - select this checkbox to define payer coverage for facility. By default this checkbox is cleared.
