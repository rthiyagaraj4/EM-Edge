Enter the following details to amend discounts:

 

  1. Type – Select a value from the list. Type is set to Billing Service, by default. This is a mandatory field.

  2. Code - Click [List of Values] to select the Billing Service. You can update the discount details for the selected Billing Service. The Description for the selected Billing Service gets displayed. This is a mandatory field.

  3. Apply to – Select a value from the list to update the discount details. The values are system defined – All, Price Class, Billing Group and Customer.

  4. All – Select All to update the discount details defined for a Price Class, Billing Group and Customer.

  5. Price Class – Select Price Class to update the discount details defined for a Price Class.

  6. Billing Group – Select Billing Group to update the discount details defined for a Billing Group.

  7. Customer – Select Customer to update the discount details defined for a Customer.

 

The following details are displayed:

 

  1. Code – The code gets displayed based on the Indicator selected.

  2. Description – The description for the selected Code gets displayed.

  3. Valid From – The date from when the discount is applicable gets displayed.

  4. Valid To - The date until when the discount is applicable gets displayed.

 

Discount %

  1. IP – The discount percentage defined for the Inpatients gets displayed. Enter the new discount percentage in numeric format. This is a mandatory field.

  2. DC – The discount percentage defined for the Daycare patients gets displayed. Enter the new discount percentage in numeric format. This is a mandatory field.

  3. OP – The discount percentage defined for the Outpatients gets displayed. Enter the new discount percentage in numeric format. This is a mandatory field.

  4. Emg – The discount percentage defined for the Emergency patients gets displayed. Enter the new discount percentage in numeric format. This is a mandatory field.

  5. Ext - The discount percentage defined for the External/Referral patients gets displayed. Enter the new discount percentage in numeric format. This is a mandatory field.
