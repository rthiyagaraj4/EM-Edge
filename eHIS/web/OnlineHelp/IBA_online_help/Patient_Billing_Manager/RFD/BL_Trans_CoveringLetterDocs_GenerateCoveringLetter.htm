Enter the following details to generate a Covering Letter:

 

  1. Cut-off Date – Click Calendar function to select the Cut-off Date. This is a mandatory field.

  2. Module ID – Select a Module from the list. The system defined values are Patient Billing, Accounts Receivable and Both. The Covering Letter is generated for the outstanding documents in the module selected.

  3. Prefix Series - Click [List of Values] to select a Prefix Series. The Prefix selected when defining the Document Numbering gets listed. This is a mandatory field.

  4. Billing Group - Click [List of Values] to select a Billing Group. The outstanding document details for the selected Billing Group get displayed.

  5. BreakDown By Customer & Group – The BreakDown By Customer & Group check box is selected, by default.

  6. Customer [From–To] - Click [List of Values] to select a range of Customers. The outstanding documents associated to the selected Customer range get displayed.

  7. Patient ID [From–To] – Click [List of Values] to select a Patient ID from the lookup. The outstanding documents associated to the selected range of patients’ get displayed.

  8. Episode – Select an Episode Type from the list. The field contains system defined values such as Emergency, Outpatient, Daycare, Inpatient, External and All. Enter a valid Episode number for the selected Episode Type.

  9. Group Code - Click [List of Values] to select a Group Code. The Description for the selected Group Code gets displayed.

  10. Option - Select an Option from the list. The system defined values are Preview and Process.

  11. 11. Body Header – Select the check box, if the Body Header information has to be included in the Covering Letter generated.

  12. Body Footer – Select the check box, if the Body Footer information has to be included in the Covering Letter.

  13. Print Text – Select the check box, if the Print Text information has to be included in the Covering Letter.

  14. Foreign Currency – Select the check box, if Foreign Currency details have to be included in the Covering Letter.

  15. Foot Note – Select the check box, if the Foot Note information has to be included in the Covering Letter.

  16. Signatory – Select the check box, if the Signatory information has to be included in the Covering Letter.

  17. [Letter Format] – Click [Letter Format] to preview the Letter.

  18. [Contact Address] – Click [Contact Address] to record the details of the contact person. The button is enabled only if the ‘Billing Group’ is specified and the option ‘BreakDown By Customer & Group’ is selected. Enter the Contact Address for the selected Billing Group. If the Billing Group is not selected and the field ‘BreakDown By Customer & Group’ is cleared, the Contact address button is not enabled.
