Enter the following details to define the format to print the documents:

 

  1. Facility Id – Click [List of Values] to select a valid Facility ID. The description for the selected Facility ID gets displayed.

  2. Available Document Formats – The Document Formats created initially get listed. Select a valid entry from the list.

 

Executable Names

  1. Executable Name – Click [List of Values] to select a valid Document Type.

  2. Doc Description – The Description for the selected executable name gets displayed.

  3. Report Tool to be Used – Select a valid report tool. The values are system defined – 'Pro*C' and 'Report Writer'.

 

Column Value Setup

  1. Column Description – Click [List of Values] to select a Column Description. The values are system defined – 'Format Type' and 'Logo File Name'.

  2. Column Value – Click [List of Values] to select a Column Value. The values get listed based on the Column Description selected.

 

Online Print Documents

  1. Document Name – Click [List of Values] to select a Document Name.

  2. Executable Name – Enter a valid Executable Name for the Online Print Documents.

 

Online Print Routing

  1. Destination – Select a Destination from the list. The values are system defined – 'Dept', 'Ward', 'Clinic' and 'Locn'. The description for the selected Destination gets displayed.

  2. WS No – Enter the Work Station Number.

  3. Printer – Enter the Printer location.
