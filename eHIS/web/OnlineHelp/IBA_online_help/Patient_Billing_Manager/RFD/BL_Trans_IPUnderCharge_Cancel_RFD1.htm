Enter the following to cancel IP Direct Billing Items:

 

  1. Admission/Tfr. No. - Enter the preferred Admission and the transfer serial number generated at the time of registration for the relevant patient. You can also use the lookup  option to search for an admission and transfer number associated to a patient encounter. This is a mandatory field.

  2. Transaction Date - This is display field. System will default the current date as transaction date. This is required to determine the day type code and the time type code for billing charge calculations.

  3. Patient No. – Enter the Patient Number in alphanumeric format of 20 characters maximum length. Using the lookup, the patient number can be selected. On selecting the valid values of Patient Number or Admission No. and Trf.no. pressing enter or tab key, values of Patient Name, Sex, Age, Ward No, Doctor, Bed Class, Bed No, Room No. are displayed by the system.
