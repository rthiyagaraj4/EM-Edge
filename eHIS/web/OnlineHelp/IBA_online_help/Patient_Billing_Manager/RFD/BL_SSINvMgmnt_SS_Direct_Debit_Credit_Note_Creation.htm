Enter the following details:

 

  1. Invoice Type - Select an Invoice Type from the list. The values defined through the 'SS Invoice Types' function get listed.

  2. Patient ID - Click [List of Values] to select a Patient ID. List of Patients with the Invoices are listed.

  3. Episode ID - Click [List of Values] to select an Episode ID.

  4. Invoice - Click [List of Values] to select a invoice. Invoices generated for the selected patient ID are listed. This is a mandatory field.

  5. Click [Refresh] to view the details. The invoices generated for the respective patient ID and the services associated to the listed invoices are displayed.

  6. [Generate Dbn/Crn] – Click [Generate Dbn/Crn] to generate either the Debit Notes the Credit Notes for the selected outstanding bills. The Document Type has to be defined for the Credit Notes and the Debit Notes using the function ‘Document Types by Payers’. A document reference number gets displayed.
