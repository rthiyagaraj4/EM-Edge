Enter the following details to print undercharge advice:

 

  1. Report Type - Select a report type from the list. The values for Report Type are system-defined. The values are Single Bill, Range of Bills by Bill Number, Range of Bills by Bill Date, Range of Bills by Customer and All Range.

  2.  Select “Single Bill” from the list to print only a single undercharge advice.

  3. Select “Range of bills by Bill Number” from the list and the Bill Number field under the “From” and “To” sections gets enabled. The undercharge advices for the selected range of bill numbers get listed.

  4. Select “Range of bills by Bill Date” from the list and the “Bill Date” field under the “From” and “To” sections gets enabled. The undercharge advice for the selected range of bill dates gets listed.

  5. Select “All Range” from the list and all the fields under the “From” and “To” sections get enabled. The bill number and the bill date are to be specified where the records meeting all the search criteria get listed.

  6. Select “Range of Bills by Customer” from the list and the “Customer” field get enabled. The “Bill Date” field gets enabled, where the bill dates can be specified. The discharge bill details for the selected criterion get listed.

  1. Option - The values for option are system-defined. The values in the list are “All Details” and “No Settlement”. The option is selected as “All Details”, by default. You cannot change the value in the list.

  2. Customer - Select a Customer from the list. The values defined for the Customer in the Masters is available in the list. This field is enabled only when the Report Type is selected as “Range of Bills by Customer”.

  3. Bill Number [From-To] - Click [List of Values] to view and to select a list of the bill numbers. A screen with Bill Lookup appears. Press [F8] or click [Execute Query] and the bills for the selected bill number range gets displayed. Double-click a record to select it. The Bill Number associated to the patient ID appears automatically in the field along with the description.

  4. Bill Date [From-To] – You can select a range of Bill Date only when the Report Type is selected as “Range of Bills by Customer”, “Range of bills by Bill Date” and “All Range”. The discharge bill details for the selected date range get listed.

  5. Bill Type – The Bill Type is displayed as Undercharge Advice, by default. You cannot change the value for Bill Type.

  6. Only Discount to be Print - Select the check box to print discount details in the bill. Clear the check box, if discount details are not to be printed in the bill.

  7. Detailed Bill Print - Select the check box to print a detailed bill. The details are either Grouped or printed as a Line item along with the Code Print Indicator value selected for a detailed bill print in 'Rules for Detailed Bill' function. If the check box is cleared, bill is printed based on the bill format defined through 'Service Grouping' tab in 'Rules for Detailed Bill' function.

  8. [Excel] - Click [Excel] to import the details to an excel file. An authorization screen pops-up. Enter the User Name and Pin number. Click [OK] to import or [Cancel] to exit. An excel file is saved with the bill details in the local system.

The document type, respective document number, date, bill amount, patient number, bill nature, Billing Group, Customer code, Episode Type, Episode ID, Visit ID and the printed status of the bill gets displayed.

  1. Patient Name - The patient name associated to the respective patient ID gets displayed.
