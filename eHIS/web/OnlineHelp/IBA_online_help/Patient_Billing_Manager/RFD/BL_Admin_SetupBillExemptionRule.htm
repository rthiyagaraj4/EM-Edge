Enter the following details to Setup Bill Exemption Rule:

 

Billing Groups for Bill Exemption

  1. Billing Group – Click [List of Values] to select a Billing Group. The bill exemption is valid only for the selected Billing Groups, which are associated to the facility.

  2. Description – The Description for the selected Billing Group gets displayed.

  3. Episode Type – Select an Episode Type from the list. The values in the list are system defined – 'Inpatient', 'Daycare', 'Outpatient', 'Emergency' and 'External'.

  4. Inpatient – Select Inpatient and the bills are exempted for the Inpatients associated to the selected Billing Group in the facility.

  5. Daycare – Select Daycare and the bills are exempted for the Daycare patients associated to the selected Billing Group in the facility.

  6. Outpatient - Select Outpatient and the bills are exempted for the Outpatients associated to the selected Billing Group in the facility.

  7. Emergency - Select Emergency and the bills are exempted for the Emergency patients associated to the selected Billing Group in the facility.

  8. External - Select External and the bills are exempted for the External/Referral patients associated to the selected Billing Group in the facility.

  1. Level of Service – The Level of Service gets displayed in numeric format. The Level of Service specified during Billing Group definition gets displayed. The Level of Service is eligible for Inpatients only.

  2. Billing Group Explanatory Note – The Explanatory Note recorded for the selected Billing Group during definition gets displayed.
