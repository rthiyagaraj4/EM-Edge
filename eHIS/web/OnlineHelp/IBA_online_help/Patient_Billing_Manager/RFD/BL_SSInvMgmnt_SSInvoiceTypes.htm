Enter the details in the following fields:

 

  1. Invoice Type Code - Enter the Invoice Type Code in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Description - Enter the description in alphanumeric format of 100 characters maximum length. This is a mandatory field.

  3. HIV Related - Select the check box, if the Invoice Type is HIV related. Clear the check box, if Invoice Type is not HIV related.

  4. Special Item Only - Select the check box, if the Invoice Type is valid for Special Items. Clear the check box, if Invoice Type is not valid for Special Items.

You can either select HIV Related or Special Item Only check box.

 

Default Doc Type to be Used

  1. Invoice - Click [List of Values] to select the default document type for Invoice. The Document Types with the Transaction Type as 'Invoice/Bill' get listed. This is a mandatory field.

  2. Debit Note - Click [List of Values] to select the default document type for Debit Note. The Document Types with the Transaction Type as 'Debit Note' get listed. This is a mandatory field.

  3. Credit Note - Click [List of Values] to select the default document type for Credit Note. The Document Types with the Transaction Type as 'Credit Note' get listed. This is a mandatory field.

  4. Ref Letter Num - Enter the Referral Letter Number in numeric format of 30 characters maximum length.
