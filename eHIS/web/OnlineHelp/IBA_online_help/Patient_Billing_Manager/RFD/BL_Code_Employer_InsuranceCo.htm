Enter the following details to create an Employer or Insurance Company:

 

  1. Employer/Company Code - Enter the Employer or Insurance Company code in alphanumeric format of eight characters maximum length. The Employer or the Insurance Company Code has to be unique. This field is mandatory.

  2. Insurance – Select the check box, if the Company is an Insurance company. Clear the check box, if the company is not an Insurance Company.

  3. Long Description - Enter a long description in alphanumeric format of 40 characters maximum length. This field is mandatory.

  4. Short Description - Enter a short description in alphanumeric format of 15 characters maximum length. This field is mandatory.

  5. GSTIN – Enter the GSTIN value in alpha numeric format. It is unique, 15-digit number. User can enter text up to 20 characters (maximum). It is a one-time setup field. This field is mandatory only for Credit billing group. It is not mandatory for other billing groups i.e., Cash / Sponsor. This is applicable only for Indian customers.

  6. Billing Group – Click [List of Values] to select a Billing Group. The description for the respective Billing Group is displayed. The data gets listed only if the values are defined using the Billing Group function under the Billing Rule menu. This field is mandatory.

 

Contact Details

  1. Contact Person - Enter the name of the person in alphabetical format of 40 characters maximum length. The Contact Person is the person to be contacted with regards to any queries.

  2. Designation - Enter the designation of the Contact Person in alphabetical format of 40 characters maximum length.

  3. Address Line 1, Line 2 and Line 3 - Enter the address of the contact person in alphanumeric format of 30 characters maximum length.

  4. Postal Code - Enter the Postal Code in alphanumeric format of 30 characters maximum length.

  5. Telephone Number - Enter the Telephone Number of the contact person in alphanumeric format of 30 characters maximum length.
