Visit Charge Rule

Visit Charge Rule tab enables you to define the price for a Visit Type associated to a Clinic. You have to associate a Billing Service or a Billing Panel to a Visit Type. The bill generation for a relevant Clinic depends on the billing services associated for the Visit Type and the Medical Personnel Fee.

 

Enter the following details to define a Visit Charge:

  1. Clinic Code – Click [List of Values] to select a Clinic. The description for the selected Clinic Code gets displayed. This is a mandatory field.

 

Visit Types

  1. Billing Grp Code - Click [List of Values] to select the billing group code. If the Insurance Group radio button in the OP Parameter screen is selected, this field is displayed.

  2. Customer Grp Code - Click [List of Values] to select the Customer Group Code. If the Insurance Group radio button in the OP Parameter screen is selected, this field is displayed.

  3. Policy Type Code - Click [List of Values] to select the Policy Type Code. If the Insurance Group radio button in the OP Parameter screen is selected, this field is displayed

  4. Customer Code - Click [List of Values] to select the Customer Code. If the Insurance Group radio button in the OP Parameter screen is selected, this field is displayed.

  5. Visit Type - Click [List of Values] to select a Visit. The values defined using the Visit Types function in the Outpatient Management module gets listed. This is a mandatory field.

  6. Description - The description for the selected Visit Type gets displayed.

  7. Doctor ID - Click [List of Values] to select a Practitioner. Select ‘**’ to associate all the Practitioners to a Visit Type. This is a mandatory field.

  8. Name – The Name of the selected Practitioner gets displayed. If the Doctor ID is selected as ‘**’, ‘All Doctors’ is displayed in the text box.

  9. Indicator – Select an Indicator from list. The values are system defined – 'Service' and 'Panel'.

  10. Service – Select Service to associate a Service. The Doctor fee is calculated based on the charges defined for the Service.

  11. Panel – Select Panel to associate a Panel. The Doctor fee is calculated based on the charges defined for the Panel.

  1. Billing Service/Panel for Visit Charge - Click [List of Values] to select a Billing Service or Billing Panel. The values defined using the functions Billing Service and Panel get listed. This is a mandatory field.

  2. Description - The description for the selected Billing Service or the Panel gets displayed.

  3. Indicator – Select an Indicator from list. The values are system defined – 'Service' and 'Panel'.

  4. Service – Select Service to associate a Service. The Medical Personnel fee is calculated based on the charges defined for the Service.

  5. Panel – Select Panel to associate a Panel. The Medical Personnel fee is calculated based on the charges defined for the Panel.

  1. Billing Service/Panel for Medical Pers Fees – Click [List of Values] to select a Billing Service or Billing Panel to compute the Medical Personal Fees. The values defined using the function Billing Services with the ‘Doctor’s Share Applicable’ check box as selected gets displayed.

  2. Description – The Description for the selected Billing Service or the Billing Panel gets displayed.

  3. Alternate Visit Charge During Close Consult - This field is enabled only if the 'Billing Service/Panel for Medical Pers Fees' is selected as 'Panel'. Click [List of Values] to select a Billing Service. Billing Services associated to the Billing Panel with the 'Medical Personal Fee' as selected gets displayed. If the medical personal fee is not charged during consultation, then the respective Visit Type to the Clinic is charged based on the associated alternate Billing Service.
