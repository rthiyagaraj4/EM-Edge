The Addl Service Charge by Billing Services tab enables you to define the additional charges based on the Billing Services utilized by the patient from the facility. The effective date for the rule code to be associated with the Billing Service has to be the same as the effective date mentioned for the rule code under the ‘Additional Service Charge Rule’ tab. Also, if the additional charge is defined on the Net charge, it cannot be included in the price.

 

Enter the following details to define the additional charges on the Billing Services:

 

  1. Service – Click [Execute Query] or press [F8] to view and to associate the rule codes to the Billing Services. The values defined using the Billing Services function get displayed.

  2. Rule Code – Click [List of Values] to select the Rule Code. The Rule Codes defined using the ‘Additional Service Charge Rule’ tab gets displayed. This is a mandatory field.

  3. Description – The description for the selected Rule Code gets displayed.

  4. Effective [From – To] – Click the Calendar function to select the date. The ‘From’ date selected has to be greater than the system date. The effective date has to match with the date range selected for the rule code under the ‘Additional Service Charge Rule’ tab.

  5. Included in Price – Select the check box, if the additional charge has to be included in the bill. If the check box is cleared, the additional charge cannot be included in the bill. If the rule code is based on the net charge, the service charge is not included in the price.

  6. All – Select this checkbox to link rule to the billing service that is associated to this patient class.

  7. OP - Select this checkbox to link rule to the billing service that is associated to this patient class.

  8. EM - Select this checkbox to link rule to the billing service that is associated to this patient class.

  9. IP - Select this checkbox to link rule to the billing service that is associated to this patient class.

The Sub Rule Types associated to the Main Rule Types linked to the respective Billing Service gets displayed. You can click  to view the sub rule types associated to each main rule type.
