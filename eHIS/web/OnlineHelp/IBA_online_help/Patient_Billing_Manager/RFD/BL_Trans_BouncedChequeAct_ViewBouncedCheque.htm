Enter the following selection criteria to view the details:

 

  1. Date Selection – Select a value from the list. The system defined values are Bounced Date, Receipt Date and Replacement Date.

  2. Bounced Date [From-To] – Click Calendar function to select the date range. The cheques, which had bounced during the selected date range, get listed. If the date is not specified, system assumes the minimum value for ‘From’ field and the maximum value for ‘To’ field.

  3. Option – The options are All, Not Replaced and Replaced.

  4. Select All to view bounced cheques and the replaced cheque details.

  5. Select Not Replaced to view the bounced cheques, which are not replaced.

  6. Select Replaced to view the bounced cheques, which are not replaced.

  1. Patient No – Enter the patient number in alphanumeric format of 10 characters maximum length. The name of the selected patient gets displayed in the adjacent text box.

  2. Original Doc – Click [List of Values] to select the Original Document Number. The documents against which the bounced cheque details are recorded using the ‘Enter Bounced Cheque’ function get listed. The original document number recorded for the cheques get displayed along with the date.
