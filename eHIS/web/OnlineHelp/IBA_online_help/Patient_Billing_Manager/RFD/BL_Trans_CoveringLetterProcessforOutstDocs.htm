Enter the following details to generate a report listing the outstanding documents for the selected Covering Letter:

 

  1. Cut-off Date – Click Calendar function to select the Cut-off Date. This is a mandatory field.

  2. Module ID – Select a Module ID from the list. The system defined values are Patient Billing, Account Receivable and Both.

  3. Billing Group [From–To] – Click [List of Values] to select a Billing Group. The Outstanding Documents for the selected Billing Group specified get displayed.

  4. Customer [From–To] - Click [List of Values] to select a range of Customers. The outstanding documents associated to the range of Customers get displayed.  

  5. Patient No [From–To] – Click [List of Values] to select a Patient Number from the lookup . The list is generated for the selected patients.

  6. Doc Type Code [From–To] – Click [List of Values] to select the range of Document Type Code. The outstanding document details get listed.

  7. Episode No [From–To] – Click [List of Values] to select a range of Episode Numbers.

  8. Visit ID [From–To] – Click [List of Values] to select a range of Visit IDs.

  9. Report Sort Order – Select a value from the pre-defined set based on which the Report should be sorted. The values in the list are “Billing Group”, “Customer”, “Patient ID” and “Episode ID”.

  10. Episode Type – Select an Episode Type from the list. The system defined values are All, Emergency, Outpatient, Daycare, Inpatient and External.

  11. Report Sort Order – Select the sort order from the list. The report output gets displayed based on the sort order selected. The system defined values are Billing Group, Customer, Patient ID and Episode ID.
