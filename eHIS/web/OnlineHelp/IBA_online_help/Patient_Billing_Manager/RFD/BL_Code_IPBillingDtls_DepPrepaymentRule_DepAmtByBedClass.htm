Deposit Amount by Bed Class

The Deposit Amount by Bed Class tab enables you to fix a deposit amount to be collected from the patients during registration. The deposit amount is applicable for the selected bed class.

 

Enter the following details to set the deposit amount:

 

  1. Bed Class – Click [List of Values] to select a Bed Class. The description for the selected bed class gets displayed. This is a mandatory field.

  2. Basis for Deposit – Select a value from the list. The values are system defined - Lumpsum or Number of Days Stay.

  3. Lumpsum – Select Lumpsum to accept a large amount of money during registration.

  4.  Number of Days Stay - Select Number of Days Stay to accept the deposit based on the number of days the patient is admitted to the hospital.

  1. Calculation Criteria - Select a value from the list. The values are system defined - Fixed Amount, Factored (on bed charge) and Additional Days (bed charge). When the basis for deposit is set to Lumpsum, the Calculation Criteria is set to Fixed Amount, by default. The Calculation Criteria cannot be changed when the Basis for Deposit is selected as Lumpsum.

  2.  Fixed Amount – Select Fixed Amount to enter a fixed deposit amount for a Bed Class.

  3. Factored (on Bed Charge) - Select Factored (on Bed Charge) to enter a deposit amount which has to be calculated based on the bed class charge. Calculation Criteria is set as Factored (on Bed Charge), only when the ‘Basis for Deposit’ is selected as ‘Number of Days Stay’.

  4. Additional Days (Bed Charge) - Select Additional Days (Bed Charge) to enter a deposit amount, which has to be calculated based on the bed, class charge. Calculation Criterion is set as Additional Days (Bed Charge), only when the ‘Basis for Deposit’ is selected as ‘Number of Days Stay’. The bed charge is calculated based on the number of days stay.

 

Additional Deposit Amount for ICU

If the patient is admitted to the Intensive Care Unit, then additional charges are to be fixed. The additional deposit amount defined is applicable only for the ICU. The fields differ based on the ‘Basis for Deposit’ selected.

  1. No. of Days Stay Upto - The number of days is set to unspecified, by default. The deposit amount is applicable when the days of stay are not specific. To fix an amount for specific days, enter the number of days in the text box.

  2. Deposit Amount/Deposit Factor/Additional Days - Enter the amount paid in numeric format with 10 digits before the decimal and two digits after the decimal. The amount indicates the deposit amount to be collected in advance during patient registration. The field legends change based on the value selected for Calculation Criteria.

  3. Deposit Amount – Enter the Deposit Amount when the Calculation Criteria is selected as Fixed Amount.

  4. Deposit Factor - Enter the Deposit Factor when the Calculation Criteria is selected as Factored (on Bed Charge). The deposit amount is calculated based on the amount specified and the bed charge.

  5. Additional Days - Enter the Additional Days when the Calculation Criteria is selected as Factored (on Bed Charge). The deposit amount is calculated based on the amount specified and the bed charge defined.

  1. Indicator – Select a value from the list. The values are system defined - Percentage and Amount.

  2. Percentage – Select Percentage to calculate the price in terms of percentage.

  3. Amount – Select Amount to enter a specific amount to calculate the deposit amount. This field is enabled when the ‘Basis for Deposit’ is selected as ‘Lumpsum’.

  4. Amount/Percentage (Additional Deposit Amount/Percentage for ICU) - Enter the amount in numeric format. If the indicator is set to amount, then enter the amount in numeric format with 10 digits before and two digits after the decimal point. When the indicator is set to percentage, enter the value lesser than 100.

When the Basis for Deposit is set to 'Number of Days Stay', the 'No. of Days Stay Upto' will have additional fields where you have an option to specify additional number of days. The 'No. of Days Stay Upto' can contain a maximum of four digits.
