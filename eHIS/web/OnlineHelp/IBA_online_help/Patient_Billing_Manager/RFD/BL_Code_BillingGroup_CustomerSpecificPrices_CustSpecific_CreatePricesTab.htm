Create Prices

The Create Prices tab enables you to define the prices for the Billing Services associated to the Customer. You have an option to select the Billing Services as reimbursable. You can also define if discount is applicable for a range of Billing Services selected and associated to the Customer. The price defined for a patient class and the associated Price Class gets displayed under the ‘Customer Specific Prices’ tab in the Billing Services function.

 

Enter the following details to define the price:

 

  1. Facility – Select the Facility from the list. The facilities associated to the logged in user get listed. The price defined for a patient class and the associated Billing Service is applicable to the facility selected.

  2. Customer – Click [List of Values] to select a Customer. The description for the selected Customer gets displayed. The service prices defined for a patient class is applicable for the selected Customer in the facility. This is a mandatory field.

  3. Billing Class – Click [List of Values] to associate a Billing Class to the Customer. The values defined using the Billing Class function gets listed. The price defined is valid for the selected patient class. The text boxes to enter the price adjustment details are enabled based on the Billing Class selected. If the Billing Class is selected as ‘All’, all the text boxes are enabled to enter the price adjustment details.

 

From

  1. Billing Service – Click [List of Values] to select a Billing Service. The values defined using the Billing Service function gets listed. The prices defined are applicable for the selected patient class and is valid for the period selected. This is a mandatory field.

  2. Valid From – Click Calendar function to select a date. The Billing Service prices for the selected Customer are valid from the date selected. This is a mandatory field.

 

To

  1. Billing Service – Click [List of Values] to select a Billing Service. The values defined using the Billing Service function gets listed. A range of Billing Services from the one selected in the ‘From’ field to the one selected for ‘To’ field is associated to the Customer. The prices defined are applicable for the selected patient class and is valid for the period selected. This is a mandatory field.

  2. Valid To – Click Calendar function to select a date. The Billing Service prices for the selected Price Class are valid until the date selected.

  3. Type – Select a value from the list to identify the reimbursable type. The system-defined values are – ‘Full’, ‘Partial’ and ‘NA’.

  4. Full – Select Full to refund the entire amount paid by the patient towards the service.

  5. Partial – Select Partial to refund only a fraction of the amount paid towards the service. A part of the amount is deducted for the service availed and the remaining amount is refunded to the patient/payer. You have to specify the amount to be reimbursed in the field ‘Reimbursable Amount’ under the ‘General’ tab. The Charge is specified only if the ‘Charge Type’ is set to ‘Charge’.

  6. Not Applicable – Select NA, if the service is not reimbursable.

  1. Allowed – Select the check box, if the Billing Service is reimbursable for the selected patient class. Clear the check box, if the respective Billing Class is non-reimbursable for the selected patient class.

  2. Reimbursable Amt – Enter the amount, which can be reimbursed to the patient.

  3. Override – Select the check box, if the respective Billing Service is reimbursable for the selected patient class. The override option for the patient class takes precedence over the override check box in the header field. If the service is not reimbursable and the check box is selected for the patient class, it indicates that the Billing Service is reimbursable for the selected patient class.

 

Price Adjustment %

  1. IP Rate – Enter the price adjustment percentage in numeric format. If the selected Billing Class is valid for the Inpatients, the text box is enabled to enter the adjustment percentage for Inpatients in the facility.

  2. DC Rate – Enter the price adjustment percentage in numeric format. If the selected Billing Class is valid for the Daycare patients, the text box is enabled to enter the adjustment percentage for Daycare patients in the facility.

  3. OP Rate – Enter the price adjustment percentage in numeric format. If the selected Billing Class is valid for the Outpatients, the text box is enabled to enter the adjustment percentage for Outpatients in the facility.

  4. Emergency Rate – Enter the price adjustment percentage in numeric format. If the selected Billing Class is valid for the Emergency patients, the text box is enabled to enter the adjustment percentage for Emergency patients in the facility.

  5. External Rate – Enter the price adjustment percentage in numeric format. If the selected Billing Class is valid for the External/Referral patients, the text box is enabled to enter the adjustment percentage for External/Referral patients in the facility.

  6. Minimum Charge – Enter the minimum charge for a range of services and the selected patient class. The charge for the patient class has to be equal to or greater than the minimum charge defined.

  7. Max Charge – Enter the maximum charge for a range of services and the selected patient class.

  8. Base Qty. – Enter the Base Quantity towards the service. The service charge is calculated based on the service price and the quantity of the service used.

  9. Reimbursable Amt – Enter the Reimbursable Amount for the service. The text box is enabled, if the check box ‘Reimbursable’ is selected. The refund amount can be full or partial based on the value selected for ‘Reimbursable Type’.

  10. Gross/Net Applicability – Select a value from the list. The values are system defined – Gross and Net.

  11. Gross - Select Gross to charge the Gross amount for the service. The charge towards the service is calculated based on the discount and the price defined for the patient class.

  12. Net – Select Net to charge the Net amount for the service. The service charge is calculated based on the price towards the patient class. The discount amount for the service is not considered during bill generation.

  1. Apply Discount – Select the check box, if discount is applicable for a range of Billing Services selected.

  2. Check for Cost availability – Select the check box, if the Cost Amount is applicable for the Billing Services associated to the Customer. Clear the check box, if the Cost Amount is not applicable for the Billing Services associated to the Customer. A screen pops-up requesting confirmation. Click [Yes] to save the entries. Click [No] to cancel the entries.
