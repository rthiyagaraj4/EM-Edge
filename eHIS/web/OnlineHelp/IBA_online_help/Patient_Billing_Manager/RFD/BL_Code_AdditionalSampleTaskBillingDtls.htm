Enter the following details to define the charges for Additional Samples/Tasks:

 

  1. Order Catalog Code – Click [List of Values] to select an Order Catalog. The values defined using the function Order Catalog in the Order Entry & Result Reporting module get listed. The description for the selected Order Catalog code gets displayed.

  2. Charge Type Code - Click [List of Values] to select a Charge Type. The values defined using the function Charge Type in the Order Entry & Result Reporting module get listed.

  3. Test Code – The Test Codes defined using the function Test Codes and associated to an Order Catalog get displayed.

  4. Additional Charge Level – Select an Additional Charge Level from the list. The values are system defined - Sample, Task and Both.

  5. Sample – Select Sample to charge the Anatomical Sites. Select Sample to define a service charge for the Anatomical Sites defined in the Laboratory module. The ‘Additional Charge by Sample’ and the ‘Past Settings’ tab gets enabled on selecting Sample from the list.

    When the Additional Charge Level is set to Sample:

    The Additional Charge by Sample tab enables you to define the charge for the Anatomical Sites. The Anatomical Sites are charged based on the prices defined for the associated Billing Service.

    Enter the following details to associate Billing Service to the Anatomical Site:

    1. Donot Charge 1st Sample – Select the check box, if the first sample collected disregard the charges for the first sample. Clear the check box, if the first sample has to be charged.

    2. Anatomical Site – Click [List of Values] to select an appropriate Anatomical Site. The values defined using the function Anatomical Site in the Application Masters module get listed. Select ‘Any Site’ to associate a Billing Service or a Billing Panel to all the Anatomical Sites defined. The selected Anatomical Site is charged based on the price defined for the Billing Service or the Billing Panel associated.

    3. Description - The description for the selected Anatomical Site gets displayed.

    4. Charge? – Select the check box to charge the Anatomical Site. Clear the check box, if the charges are not applicable for the selected Anatomical Site. The ‘Service Ind’ field is enabled only when the check box is selected.

    5. Service Ind – Select a value from the list. The values are system defined - Service and Panel.

    6. Service – Select Service to associate a Billing Service to an Anatomical Site. The values defined using the Billing Service function get listed. An Anatomical Site is charged based on the charges defined for the selected Billing Service.

    7. Panel – Select Panel to associate a Billing Panel to an Anatomical Site. The values defined using the Billing Panel function get listed. An Anatomical Site is charged based on the charges defined for the selected Billing Panel.

    1. Billing Service – Click [List of Values] to select a Billing Service or a Billing Panel. The values defined using the Billing Services or the Billing Panel function get listed respectively based on the Service Indicator selected.

    2. Description - The description for the selected Billing Service or Billing Panel gets displayed.

    Past Settings

    The Past Settings tab enables you to view the previous settings for an Anatomy Site. The date until when the Billing Service/Billing Panel was associated to a relevant Anatomy Site gets displayed.

  6. Task – Select Task to charge either Individual Tasks or Group Tasks defined in the Laboratory module. The ‘Additional Charge by Task’ and the ‘Past Settings’ tab gets enabled on selecting Task from the list.

    When the Additional Charge Level is set to Task:

    Additional Charge by Task

    The Additional Charge by Task tab enables you to define the charge for the Individual or Group Tasks defined in the Laboratory module. The Individual or Group Tasks are charged based on the prices defined for the associated Billing Service.

    Enter the following details to associate a Billing Service to the Task:

    1. Task ID – Click [List of Values] to select a Task ID. The Individual or Group Tasks defined in the Laboratory module get listed. Select ‘Any Task’ to associate a Billing Service or a Billing Panel to all the Tasks defined. The selected Task is charged based on the price defined for the Billing Service or the Billing Panel associated.

    2. Description - The description for the selected Task gets displayed.

    3. Charge? - Select the check box to charge the selected Task. Clear the check box, if the charges are not applicable for the selected Task. The ‘Service Ind’ field is enabled only when the check box is selected.

    4. Service Ind - Select a value from the list. The values are system defined - Service and Panel.

    5. Service – Select Service to associate a Billing Service to a Task. The values defined using the Billing Service function get listed. A Task is charged based on the charges defined for the selected Billing Service.

    6. Panel – Select Panel to associate a Billing Panel to a Task. The values defined using the Billing Panel function get listed. A Task is charged based on the charges defined for the selected Billing Panel.

    1. Billing Service - Click [List of Values] to select a Billing Service or a Billing Panel. The values defined using the Billing Services or the Billing Panel function get listed respectively based on the Service Indicator selected.

    2. Description - The description for the selected Billing Service or Billing Panel gets displayed.

    Past Settings

    The Past Settings tab enables you to view the previous settings for an Individual task or a Group task. The date until when the Billing Service/Billing Panel was associated to a relevant task gets displayed.

  7. Both – Select Both to define a charge for Anatomical Sites, Individual Tasks and Group Tasks. The ‘Additional Charge by Sample’, ‘Additional Charge by Task’ and the ‘Past Settings’ tab gets enabled on selecting Both from the list.

    When the Additional Charge Level is set to Both:

    Select Additional Charge Level as ‘Both’ and the tabs ‘Additional Charge by Sample’, ‘Additional Charge by Task’ and ‘Past Settings’ gets enabled. You can either define the charges for Anatomical Sites and Tasks or view the past settings for the selected Anatomical Sites and Tasks.

    Past Settings

    The Past Settings tab enables you to view the previous settings for an Anatomy Site and a Task. The date until when the Billing Service/Billing Panel was associated to a relevant Anatomy Site or Task gets displayed.
