Enter the following details to associate Customer/Cash Billing Group to a Bill/Receipt Format:

Format

  1. Format Code - The format code that is defined through the 'Bill Format' tab is displayed.

  2. Description - The description of the respective Bill Format is displayed.

Customer

  1. Customer Group Code - Click [List of Values] to select a Customer Group Code. The values defined through the Accounts Receivable module are listed. Select '**' to associate the Customer/Billing Group format to all the Customers.

  2. Customer Code - The Customer Code is displayed automatically on selecting the Customer Group Code. The Customer Code associated to the Customer Group is displayed.

  3. Customer Name - The description of the Customer associated to the Customer Group is displayed.

  4. Bill Type - Select a Bill Type from the list. The system-defined values are 'OP Visit Bill', 'IP Discharge Bill', 'IP Undercharge Bill', 'External Bill, and 'Pharmacy Bill'. The selected Bill Type is displayed for the respective Customer, Customer Group and Payer combination. This is a mandatory field.

  5. Print/Reprint – Select the Print/Reprint value from the list. The system defined values are ‘Print’, ‘Reprint’ and ‘All’. The bills are generated in the bill formats associated with the bill type based on the selection of Print/Reprint.

Cash Billing Group

  1. Billing Group - Click [List of Values] to select the Billing Group. The bill format is valid for the selected Customer, Customer Group and the Billing Group.

  2. Description - The description of the selected Billing Group is displayed.

  3. Bill Type - Select a Bill Type from the list. The system-defined values are 'OP Visit Bill', 'IP Discharge Bill', 'IP Undercharge Bill', 'External Bill' and 'Pharmacy Bill'. The selected Bill Type is displayed for the selected Cash Billing Group. This is a mandatory field.

  4. Print/Reprint – Select the Print/Reprint value from the list. The system defined values are ‘Print’, ‘Reprint’ and ‘All’. The bills are generated in the bill formats associated with the bill type based on the selection of Print/Reprint.

The preview of customer bill is displayed in all bill formats while reprinting the customer bill.
