Payer Item Groups

The Payer Item Groups tab enables you to define the Item Groups and associate it to Payers. A Patient Class has to be associated to the selected Payer. The Item Group defined is valid for the selected Patient Class.

 

Enter the following details to create a Payer Item Group:

 

  1. Payer – Click [List of Values] to select a Payer. The values defined using the Customer function gets listed. This is a mandatory field.

  2. Patient Class – Click [List of Values] to select a Patient Class. The values in the list are:

  3. All Patient Class – Select All Patient Class from the list, if the Payer Material group is valid for all the patient classes.

  4. Daycare – Select Daycare from the list, if the Payer Material Group is valid only for Daycare patients.

  5. Emergency – Select Emergency from the list, if the Payer Material Group defined is valid only for Emergency patients.

  6. External – Select External from the list, if the Payer Material Group defined is valid only for Emergency patients.

  7. Inpatient – Select Inpatient, if the Payer Material Group defined is valid only for Inpatients.

  8. Outpatient – Select Outpatient from the list, if the Payer Material Group defined is valid only for Outpatients.

 

Payer Item Groups

  1. Code - Enter the Payer Item Group Code in alphanumeric format with two characters maximum length. The Payer Item Group Code has to be unique. This is a mandatory field.

  2. Long Description – Enter the Long Description for the Payer Item Group Code in alphanumeric format with 40 characters maximum length. This is a mandatory field.

  3. Short Description – Enter a Short Description for the Payer Item Group Code. The Short Description has to be in alphanumeric format with 15 characters maximum length. This is a mandatory field.

  4. Non-DRG Related - Select the check box, if the Payer Item Group is Non-DRG related. This is applicable only for inpatient transactions. Clear the check box, if the Payer Item Group is DRG related.
