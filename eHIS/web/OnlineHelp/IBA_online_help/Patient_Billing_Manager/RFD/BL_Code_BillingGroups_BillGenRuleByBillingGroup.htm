Visit Bill Generation Rule by Billing Group

The Visit Bill Generation Rule by Billing Group tab enables you to define the bill generation rule for the Billing Group. The Visit Bill Generation Rule is valid for Outpatients and is used to generate visit bills in the Outpatient Management module during visit registration.

 

  1. Generate Bill at – Select the appropriate value from the list. You have an option to define the bill generation rule for a relevant Billing Group. The Bill Generation rule is applicable only for Outpatients with the Settlement Type as Cash. Listed values are:

  2. Visit Registration – Select Visit Registration to print a bill during Visit Registration.

  3. Do not Generate – Select Do not Generate, if a Visit Registration bill is not required.

  1. Settle Bill Amount – Select Settle Bill Amount to capture the Settlement Details during bill generation for a patient for the relevant Billing Group. Clear the check box to generate only the bills without the Settlement Details.

  2. Print Option - Select a Print Option from the list. The values are system defined – 'Print Visit Bill-Cum-Receipt – Automatically', 'Print Visit Receipt – Automatically' and 'Do not print'.

  3. Print Visit Bill-Cum-Receipt – Automatically – Select Print Visit Bill-Cum-Receipt – Automatically to print Bill-Cum-Receipt for the respective Visit automatically. The Print Option is enabled, if ‘Settle Bill Amount’ is selected.

  4. Print Visit Receipt – Automatically – Select Print Visit Receipt – Automatically to generate only a receipt for the visit.

  5. Do not print – Select Do not Print to disable bill/receipt print during visit registration.

  1. Generate Zero Visit Bill – Select Generate Zero Visit Bill to generate a bill with zero amount during visit registration. Clear the check box, if a zero visit bill is not required.

  2. Print Zero Visit Bill – Select the check box to print a Zero Visit Bill during visit registration. The check box is enabled, if ‘Generate Zero Visit Bill’ generation is selected. Clear the check box and the zero visit bill is not printed during visit registration.
