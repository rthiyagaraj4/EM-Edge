Enter the following details to define a Billing Class:

 

  1. Billing Class - Enter the Billing Class code in alphanumeric format of two characters maximum length. This field is mandatory.

  2. Long Description - Enter the Long Description in alphanumeric format of 40 characters maximum length. This field is mandatory.

  3. Short Description - Enter the Short Description in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Patient Class - Select a Patient Class from the list. The values are system defined.

  5. Inpatients – Select Inpatients, if the Billing Class is applicable for Inpatients admitted in the Nursing Unit of a Facility.

  6. Outpatients - Select Outpatients, if the Billing Class is applicable for Outpatients visiting Clinic in a Facility.

  7. Daycare – Select Daycare, if the Billing Class is applicable for Daycare patients admitted in the Clinic of a Facility.

  8. Emergency – Select Emergency, if the Billing Class is applicable for Emergency patients.

  9. External – Select External, if the Billing Class is applicable for Referral or External patients.

  1. Level of Service – Enter the level of service in numeric format of two digits maximum length. The level of service indicates the priority of the selected Billing Group for Inpatients. The Bed Class gets listed in the ‘Hospital’s Request’ field while transferring or admitting the patient in the Inpatient and Day Care Management module, based on the level of service defined here and in the Billing Class function.
