Enter the following details to query patient records:

 

  1. Customer - Click [List of Values] to select the Payer and rollback the pre-invoice generated. The pre-invoice details recorded for the selected payer are listed.  

  2. Policy Type - Click [List of Values] to select the Policy Type and rollback the pre-invoice generated. The pre-invoice details recorded for the selected Policy Type are listed.  

 

Episode Type

  1. All - Select the check box to query Inpatient, Outpatient, Emergency, External and Daycare patient records matching the search criteria. On selecting the check box, the Inpatient, Outpatient, Emergency, External and Daycare check boxes are selected automatically. Clear the check box to clear the selected check boxes.

  2. Outpatient - Select the check box to query only outpatient records. The check box is selected automatically when you select 'All'. Clear the check box, if the outpatient records should not be listed.

  3. Emergency - Select the check box to query only emergency patient records. The check box is selected automatically when you select 'All'. Clear the check box, if the emergency patient records should not be listed.

  4. Inpatient - Select the check box to query only inpatient records. The check box is selected automatically when you select 'All'. Clear the check box, if the inpatient records should not be listed.

  5. Daycare - Select the check box to query only daycare patient records. The check box is selected automatically when you select 'All'. Clear the check box, if the daycare records should not be listed.

  6. External - Select the check box to query only external or referral patient records. The check box is selected automatically when you select 'All'. Clear the check box, if the external or referral patient records should not be listed.

  7. Visit/Discharge Date [From - To] - Click the Calendar function to select a range of visit or discharge dates. The records of patients' who have visited the facility or has been discharged from the facility during the selected date range are listed.
