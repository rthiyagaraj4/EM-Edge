Enter the following details:

  1. Customer Group – Select the Customer Group using lookup function. The customer group defined in the masters are listed. This field is mandatory.

  2. Stmt Duration [From-To] – Select the date range using calendar function. This field is mandatory.

  3. Customer Code – Select the Customer Code using lookup function. The customer code defined in the masters are listed.

  4. Policy – Select the Policy using lookup function. The policies defined in the masters are listed.

  5. Posted Month/Year From - Select the date range using calendar function.

  6. Episode Type – Select Episode type from drop down. The listed values are Outpatient/Emergency, Inpatient/Day care and All. A value All is displayed in drop down by default.
