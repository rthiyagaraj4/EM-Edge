Enter the following details to modify a bill:

 

  1. Rounding Option – Select a value from the list. The values are system defined.

  2. Print Line - The print line code of the bill print line is displayed.. The description for the selected Print Line gets displayed. You cannot change it.

  3. Gross Amt - The gross amount of the print line is displayed for user verification. You cannot change the Gross Amount.

  4. Service Disc Amount - The service discount amount of the print line gets displayed for user verification. You cannot change it.

  5. Print Line Disc% - Enter the discount percentage in numeric format, if bill print line discount has to be given. The percentage entered cannot exceed 100.00. The value entered cannot be negative also.

  6. Discount Amount - The Amount cannot be entered if a discount percentage has been entered. In such a case, amount is computed and displayed. If the discount percentage is not entered, enter the discount amount. The system computes the discount amount as  (Disc %/100.00) * (Gross - Serv Disc). System also checks to verify if the discount amount entered is greater than the (Gross - Serv Disc). You can set discount on the entire bills where the bill outstanding amount is zero.

  7. Net Amt - Net Amount for the line item is displayed for user verification. You cannot change the Net Amount. This is the net of charges specified in the bill print line.

 

Bill Level Discount details

  1. Rounding Option – The values are system defined. Select a value from the list.

  2. Gross Amt - Gross Amt for the bill is displayed for user verification. You cannot edit the gross amount. This is the sum of gross charges specified in the bill print line.

  3. Serv Disc - Service Discount for the bill is displayed for user verification. No changes can be done. The Service Discount is the sum of the service discount amounts specified in the bill print line.

  4. Manual Disc - Manual Discount (Sum of Print Line Special Discounts) for the bill is displayed for user verification. You cannot edit the manual discount. The manual discount is the sum of the print line special discounts specified in the bill print line.

  5. Overall Disc% - Enter the discount percentage in numeric format, if bill level discount is to be given. The percentage entered cannot exceed 100.00. The value entered cannot be negative.

  6. Overall Disc Amt - You cannot enter the amount, if a discount percentage has been entered. In such a case, amount is computed and displayed. You have to enter the discount amount, if the discount percentage is not entered. The system computes the discount amount as - (Disc %/100.00)*(Gross - Serv Disc - Man Disc). System also checks to see if the discount amount entered is greater than the (Gross - Serv Disc - Man Disc).

  7. Prepay Adj Amt – The Prepaid Adjustment Amount is displayed.

  8. Dep Adj Amt – The Deposit Adjustment Amount is displayed.

  9. Manual Discount – If any discount was given to the patient, the detail is displayed.

  10. G/L Interface Code - Click [List of Values] to view and select a G/L Interface code. If overall discount is entered, a valid G/L Interface code has to be recorded to book the discount amount in General Ledger. The interface code has to begin with "D" only. If the code is valid, the short description is displayed for user verification.

  11. Total - The total is displayed for user verification. This total is not the amount payable, but the total is as follows:

  12. Gross - Serv Disc

  13. Man Disc - Overall Disc

  1. Bill Total Amount – The Total Bill Amount after the adjustments made is displayed.

 

The system will display all the bills generated for the patient for the selected episode including zero bills, negative bills and outstanding bill.

The system will validate the gross bill amount with the discount amount even if the discount is already given. The system provides further discount for that bill if the discount amount is less than gross the bill amount.

The bill outstanding amount will be negative if the outstanding amount of the discounted bill is zero.

The system checks the total discount amount and the gross bill amount. If the discount amount is more than gross amount, the system displays an error message. The total bill discount amount cannot be more than gross bill amount. The negative outstanding bill amount will be refunded to the patient through Settle Bill Outstanding function only.
