Enter the following details to define Cash Counter for SS Office:

 

  1. Cash Counter Code - Enter the Cash Counter Code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Doc Type Code - Click [List of Values] to select a Document Type. The values defined using the 'Document Type' function in Financials Base Manager module get listed. This is a mandatory field.

  3. Long Desc - Enter the long description in alphanumeric format of 100 characters maximum length. This is a mandatory field.

  4. Short Desc - Enter the short description in alphanumeric format of 40 characters maximum length. This is a mandatory field.

 

User for Cash Counter

  1. User ID - Click [List of Values] to select a User ID. The selected Users are permitted to access the Cash Counter.

  2. User Name - The name of the selected User gets displayed.

 

Document Types By Cash Counter

  1. Payer Group Code - Click [List of Values] to select a Payer Group Code. The values defined using the 'Customer Group' function gets listed. The description of the selected Payer Group code gets displayed. This is a mandatory field.

  2. Payer Code - Click [List of Values] to select a Payer Group Code. The values defined using the 'Customer' function and associated to the selected 'Payer Group' gets listed. The description of the selected Payer Code gets displayed. This is a mandatory field.

  3. Receipt-OP - Click [List of Values] to select a receipt for Outpatient transactions. The 'Document Types' with the 'Trx Type' as 'Receipts' are listed. The description of the selected Document Type gets displayed. This is a mandatory field.

  4. Receipt-IP - Click [List of Values] to select a receipt for Inpatient transactions. The 'Document Types' with the 'Trx Type' as 'Receipts' are listed. The description of the selected Document Type gets displayed. This is a mandatory field.
