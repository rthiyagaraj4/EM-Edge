Enter the following details to export the details:

  1. Selection Criteria – Select a value from the list. The values are system defined - "Report Only" and "Export to GL".

  2. Report Only - Select Report Only to view a report on the distributions in the Patient Billing Manager module.

  3. Export To GL – Select Export to GL to transfer the data to General Ledger module.

  1. Transactions - Select a value from the list. The values are system defined - "With Cash Control Receipt" and "Others".

  2. With Cash Control Receipt – Select With Cash Control Receipt to post the Cash Control Receipts as well.

  3. Others - Select Other to post other transactions.

  1. Doc Type Code for Summary J. V – Click [List of Values] to select a Document Type. The Document Type Code for Summary Journal Voucher text box is enabled, if ‘Selection Criteria’ is selected as ‘Export to GL’ and ‘Transactions’ is selected as ‘Others’.

  2. Doc Date for Summary J. V – Click Calendar function to select a date. The Document Type Code for Summary Journal Voucher text box is enabled, if ‘Selection Criteria’ is selected as ‘Export to GL’ and ‘Transactions’ is selected as ‘Others’. The month and year selected for the transaction date has to be same as the transaction period.
