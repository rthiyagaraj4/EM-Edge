Enter the following details:

 

  1. Billing Category – Click [List of Values] to select a Billing Group Category. The outstanding document details associated to the selected Billing Group Category for which the write-off has been requested and approved or rejected gets displayed.

  2. Patient Category – Select a valid Patient Category from the list. The values are system defined – “All”, “Nationals” and “Others”. The report is generated for the selected Patient Category.

  3. Status - Select a Status from the list. The values are system defined – “Cancelled Approvals”, “Pending Approval”, “Approved” and “Rejection”. The report is generated for on the status selected.

  4. Cancelled Approvals – Select Cancelled Approvals to generate a report for the write-off requests, which are cancelled.

  5. Pending Approval - Select Pending Approval to generate a report for the write-off requests, which are pending to be approved.

  6. Approved - Select Approved to generate a report for the write-off requests, which are approved.

  7. Rejection - Select Rejection to generate a report for the write-off requests, which are rejected.

  1. Write-off Amt [From–To] – Enter the Write-off Amount in numeric format. The report generated includes the write-off details for the amount range specified.

  2. Process Date [From–To] – Click Calendar function to select the Process Date. The report displays the details on the write-off process during the selected date. This is a mandatory field.

  3. Reason [From–To] – Click [List of Values] to select a Reason. The record for which the write-off has been either requested, approved or rejected with the selected reason range gets displayed.
