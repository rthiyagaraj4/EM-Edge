Enter the following details to create Banks and Branches:

 

  1. Bank Code - Enter a unique Bank Code in alphanumeric format of six characters maximum length. This is a mandatory field.

  2. Bank Name - Enter the Bank Name in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Branch Name - Enter the Branch Name in alphanumeric format of 60 characters maximum length.

  4. Use for Deposit Bank-In – Select the check box, if the amount collected against the receipts are to be deposited in the respective bank.

  5. Credit Card M/c Provided – Select the check box, if the Credit Card Machine is provided by the Bank and interfaced to the selected Bank. When the Cashier swipes the Credit Card the amount gets deposited to the Bank.

  6. Surcharge % on Refund – Enter the amount in numeric format between the range 0 – 99 to deduct an additional amount while issuing a refund. The amount deducted while issuing the refund depends on the surcharge percentage amount defined.

  7. Cheque Type - Select the required Cheque Type from the drop down list. The listed values are ‘Blank’, ‘In-house’, ‘Local’ and ‘Outstation’. This field is  disabled if the Settlement Nature is set to a value other than “cheque”. This is one-time setup.

 

Credit Card Machines Supplied by Bank

  1. Operating Facility ID – Select a Facility from the list. The facilities associated to the logged in User get listed.

  2. Credit Card Machine ID – Enter the Credit Card Machine ID in alphanumeric format of 30 characters maximum length. This is mandatory when the ‘Credit Card M/c Provided’ check box is selected.
