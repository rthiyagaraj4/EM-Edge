Enter the following details to record agreed amount:

 

  1. SS Invoice Type - Click [List of Values] to select SS Invoice Type. The values defined using the 'SS Invoice Types' function get listed. This is a mandatory field.

  2. HIV Related - The check box gets selected, if the SS Invoice Type is defined as 'HIV Related'. The check box is cleared, if the SS Invoice Type is not 'HIV Related'.

 

Agreed Amount for Invoice - Non HIV Related

You can record details in columns only if the SS Invoice Type is non-HIV related.

  1. Effective From - Click the Calendar function to indicate the date for agreed amount for non HIV related invoice types. Date cannot be less than the system date. This is a mandatory field.

  2. Effective To - Click the Calendar function to indicate date upto which the agreed amount is valid for non HIV related invoice types. This is a mandatory field.

  3. Agreed Invoice Amount - Enter the agreed invoice amount in numeric format. It indicates the amount covered by Social Security for the selected Invoice Type.

  4. MarkDown Only - Select the check box to markdown the agreed invoice amount. For example, if the invoice amount is 15,000/- and the agreed amount is 10,000/-, then only the agreed amount is covered by Social Security during patient's bill settlement. Clear the check box, if markup is valid. For example, if the invoice amount is 15,000/- and the agreed amount is 10,000/-, then Social Security pays 15,000/- towards patient's bill settlement.  

 

Agreed Amount for Invoice - HIV Related

You can enter the details in the columns only if the SS Invoice Type is HIV related.

  1. Effective From - Click the Calendar function to indicate beginning date for agreed amount for HIV related invoice types. Date cannot be less than the system date. This is a mandatory field.

  2. Effective To - You cannot enter the effective date. It is based on the 'Effective From' date set for the same HIV Service Type from any other 'Effective From' date. The Effective From date is set as the Effective To date.

  3. HIV Serv Type - Click [List of Values] to select and associate a Service Type. This is a mandatory field.

  4. Description - The description of the selected Service Type is displayed.

  5. Agreed Amount - Enter the agreed amount in numeric format. It indicates the amount settled by Social Security for the services utilized.
