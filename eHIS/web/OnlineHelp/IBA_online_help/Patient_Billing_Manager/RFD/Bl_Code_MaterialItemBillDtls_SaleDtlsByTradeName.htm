Sale Details by Trade Name

The Sale Details by Trade Name tab enables you to define the price for a particular item based on the trade name. The sale details for a patient class is defined and associated to a Trade Name.

 

Enter the following details to define the sale details by trade name:

 

  1. Item Code – Click [List of Values] to view and select an Item Code to define the charges. The items defined in the Inventory Management module get listed. The description for the selected Item Code gets listed.

  2. Item Type – The value gets displayed based on the Item Type selected while defining the Item in the Inventory Management module.

  3. Trade Name – Click [List of Values] to select a Trade Name. The values defined using the function ‘Trade Name for Item’ in Inventory Management module gets listed. The price defined for the patient class is applicable to the selected Trade Name in the Facility. This is a mandatory field.

  4. Facility – Select a Facility from the list. The facilities associated to the logged in user gets listed. The prices defined are valid for the facility selected. This is a mandatory field.

 

Base Prices by Patient Class

  1. Batch ID – Enter a unique Batch ID to identify the prices for the Patient Class. Any number of batches can be created with the base price for the patient class, but for different dates.

  2. Public Price - Enter the Public Price for the trade name in numeric format with a maximum of 10 digits. Public Price indicates the price regulated by the Government for a material item. The charges for the selected trade name cannot be higher than the price regulated by the Government.

  3. Cost Ind – Select a Cost Indicator from the list. The values are system defined - %age, Amount and N/A.

  4. %age – Select %age, if the Cost Amount is applicable and to calculate the material item charge in terms of percentage.

  5. Amount - Select Amount and the price is computed based on the amount defined for the patient category associated to the Billing Service.

  6. N/A – Select N/A, if the Cost Amount is not applicable for the selected material item.

  1. Cost Amt – Enter the Cost Amount in numeric format. If the Cost Indicator is selected as percentage, the amount for the material item is calculated based on percentage of the Billing Service.

Inpatient

  1. Inpatient - Enter the base price to be charged for Inpatients for the selected material item and the associated trade name.

  2. Daycare - Enter the base price to be charged for Daycare patients for the selected material item and the associated trade name.

  3. Reimbursable – Select the check box, if the item is reimbursable.  Clear the check box, if the item is not reimbursable.

  4. Reimbursable Type – The values listed differ based on the value selected for ‘Reimburse’. The values are system defined – “Full”, “Partial” and “Not Applicable”.

  5. Full – ‘Full’ gets displayed on selecting ‘Reimburse Type’ as ‘Fully Reimbursable’ in Materials Base module. Patient receives a complete refund for the selected material item.

  6. Partial – ‘Partial’ gets displayed on selecting ‘Reimburse Type’ as ‘Partial Reimbursable’ in Materials Base module. Patient receives only a fraction of the service amount as a refund.

  7. Not Applicable – ‘Not Applicable’ gets displayed on selecting ‘Reimburse Type’ as ‘Non Reimbursable’ in Materials Base module. Patient does not receive any refund for the material item used.

  1. Amount – Enter the amount that can be issued as a refund towards the trade name selected. The Reimbursable Amount field is enabled when ‘Reimburse Type’ is selected as ‘Fully Reimbursable’ or ‘Partial Reimbursable’ in Materials Base module.

Outpatient

  1. Outpatient - Enter the base price to be charged for Outpatients for the selected material item and the associated trade name.

  2. Emergency - Enter the base price to be charged for Emergency patients for the selected material and the associated trade name.

  3. External - Enter the base price to be charged for External patients for the selected material item and the associated trade name.

  4. Reimbursable Item – The check box is selected, if a refund can be issued for the trade name. The check box ‘Reimburse’ has to be selected in the Materials Base module while defining the item. A refund can be issued towards the trade name only if the check box is selected.

  5. Reimbursable Type – The values listed differ based on the value selected for ‘Reimburse’. The values are system defined – ‘Full”, ‘Partial” and ‘Not Applicable”.

  6. Full – ‘Full’ gets displayed on selecting ‘Reimburse Type’ as ‘Fully Reimbursable’ in Materials Base module. Patient receives a complete refund for the selected material item.

  7. Partial – ‘Partial’ gets displayed on selecting ‘Reimburse Type’ as ‘Partial Reimbursable’ in Materials Base module. Patient receives only a fraction of the service amount as a refund.

  8. Not Applicable – ‘Not Applicable’ gets displayed on selecting ‘Reimburse Type’ as ‘Non Reimbursable’ in Materials Base module. Patient does not receive any refund for the material item used.

  9. Amount – Enter the amount that can be issued as a refund towards the trade name selected. The Reimbursable Amount field is enabled when ‘Reimburse Type’ is selected as ‘Fully Reimbursable’ or ‘Partial Reimbursable’ in Materials Base module.

  10. Effective From – Click Calendar function to select a date. The charges are applicable for the selected material item and the associated trade name from the date selected. The date entered has to be greater than the system date. This is a mandatory field.

  11. Effective To - Click Calendar function to select a date. The charges are applicable for the selected material item and the associated trade name until the date selected. The date entered has to be greater than the selected from date.

  12. Price Id - Enter a Price ID in numeric format with a maximum of three characters. The Price ID has to be unique and it helps in identifying the price for the Material Items and the trade name associated to the Facility.
