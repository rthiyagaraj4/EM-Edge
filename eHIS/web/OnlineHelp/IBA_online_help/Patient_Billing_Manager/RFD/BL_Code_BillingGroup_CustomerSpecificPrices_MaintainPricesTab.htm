Maintain Prices

The Maintain Prices tab enables you to update the prices for the Billing Services associated to the Billing Group. You can update the discount percentage or select discount applicability and enter a new discount percentage. The price defined for a patient class and the associated Billing Group gets displayed under the ‘Billing Group Specific Price’ tab in the Billing Services function.

 

Enter the following details to update the prices defined:

 

  1. Facility – Select the Facility from the list. The facilities associated to the logged in User get listed. The prices defined are valid for the facility selected.

  2. Billing Group – Click [List of Values] to select a Billing Group and to view or update the price or discounts defined for a patient class associated to the Billing Service and the selected Billing Group. Click [Execute Query] to view and update the prices and the discounts defined for the Billing Groups in the facility. The description for the Billing Group gets displayed.

  3. [View Past Prices] - Click [View Past Prices] to view previous prices defined for a patient class, associated to the Billing Services and the selected Billing Group.

 

Billing Service Details

The Billing Services, descriptions and the period until when the Billing Service is associated to the Billing Group get displayed. Click [Create Record] to associate a new Billing Service to the Billing Group.

 

  1. Billing Service – Click [List of Values] to select a Billing Service. The Billing Services, which are already associated to the Billing Group get displayed along with the effective from date. This is a mandatory field.

  2. Description – The Description for the selected Billing Service gets displayed.

  3. Valid From – Click Calendar function to select a date. The Billing Service is valid for the selected Billing Group from the date selected. The dates for the Billing Services, which are already associated, get displayed. The date selected has to be greater than the system date. This is a mandatory field.

  4. Valid To – Click Calendar function to select a date. The Billing Service is valid for the Price Class up to the date selected. The date for the Billing Services, which are already associated, get displayed. The date entered has to be greater than the valid from date.

 

Pricing by Billing Class

  1. Class – Click [List of Values] to select a Billing Class. The values defined using the Billing Class function under the ‘Billing Rule’ menu get listed. Select ‘**’ to define the prices for a patient class in the facility. The prices are valid for the patient class associated to the selected Billing Service and the Billing Group in the facility. This is a mandatory field.

  2. Description – The Description for the selected Billing Class gets displayed. This is a mandatory field.

  3. R/F - Select a value from the list. The values are system defined – Rate and Factor.

  4. Rate – Select Rate, if the service price has to be charged for the patient class associated to the Billing Service and the Billing Group.

  5. Factor – Select Factor and the service price is multiplied with the factor entered for a patient class.

  1. IP Rate/Factor - Enter the rate/factor in numeric format for Inpatients. The text box is enabled only if the Billing Class is selected either as ‘Inpatient’ or ‘All’.

  2. DC Rate/Factor - Enter the rate/factor in numeric format for Daycare patients. The text box is enabled only if the Billing Class is selected either as ‘Daycare’ or ‘All’.

  3. OP Rate/Factor – Enter the rate/factor in numeric format for Outpatients. The text box is enabled only if the Billing Class is selected either as ‘Outpatient’ or ‘All’.

  4. Emg. Rate/Factor - Enter the rate/factor in numeric format for Emergency patients. The text box is enabled only if the Billing Class is selected either as ‘Emergency’ or ‘All’.

  5. Ext. Rate/Factor - Enter the rate/factor in numeric format for External/Referral patients. The text box is enabled only if the Billing Class is selected either as ‘External’ or ‘All’.

  6. Min. Charge - Enter the Minimum Charge for the selected patient class. The minimum charge is applicable only if the Billing Service is defined as Charge based. The price defined is valid for the selected period. The price for a patient class has to be equal to or greater than the minimum charge.

  7. Max Charge – Enter the Maximum Charge for the selected patient class. The charge is applicable for the selected patient class. The price defined is valid for the selected period.

  8. Type – Select a value from the list to identify the reimbursable type. The system-defined values are – ‘Full’, ‘Partial’ and ‘NA’.

  9. Full – Select Full to refund the entire amount paid by the patient towards the service.

  10. Partial – Select Partial to refund only a fraction of the amount paid towards the service. A part of the amount is deducted for the service availed and the remaining amount is refunded to the patient/payer. You have to specify the amount to be reimbursed in the field ‘Reimbursable Amount’ under the ‘General’ tab. The Charge is specified only if the ‘Charge Type’ is set to ‘Charge’.

  11. Not Applicable – Select NA, if the service is not reimbursable.

  1. Allowed – Select the check box, if the Billing Service is reimbursable for the selected patient class. Clear the check box, if the respective Billing Class is non-reimbursable for the selected patient class.

  2. Reimbursable Amt – Enter the amount, which can be reimbursed to the patient.

  3. Override – Select the check box, if the respective Billing Service is reimbursable for the selected patient class. The override option for the patient class takes precedence over the override check box in the header field. If the service is not reimbursable and the check box is selected for the patient class, it indicates that the Billing Service is reimbursable for the selected patient class.

  4. Base Qty - Enter the Base Quantity in numeric format. The price for the service when associated to the selected patient class is calculated as the product of the rate/factor set and the base quantity defined.

  5. Gross/Net Applicability – Select a value from the list. The values are system defined – Gross and Net.

  6. Gross - Select Gross to charge the Gross amount for the service. The charge towards the service is calculated based on the discount and the price defined for the patient class.

  7. Net – Select Net to charge the Net amount for the service. The service charge is calculated based on the price towards the patient class. The discount amount for the service is not considered during bill generation.

  1. Check for Cost – Select the check box, if the Cost Amount is applicable for the selected patient class, associated to the Billing Service and Billing Group. Clear the check box, if the Cost Amount is not considered towards the charge calculation.

  2. Apply Disc - Select the check box, if Discount is applicable for the selected Billing Class. You can define the discount percentage, if the check box is selected. Clear the check box, if the discount is not applicable for the patient class associated to the Billing Service.

 

Applicable Discounts for Service and Billing Class

The text boxes are enabled, if the ‘Apply Disc’ check box is selected.

  1. Discount Indicator – Select a value from the list to issue discount for the Billing Service. The system-defined values are ‘Amount’ and ‘%’. Select Amount to specify the amount that has to be issued as a discount amount on the Billing Service. Select Percentage to specify the percentage of amount that has to be issued as a discount amount on the Billing Service.

  2. IP - (%/Amt) - Enter the value to issue discount towards the service for Inpatients. The price is calculated based on the percentage of amount and the service price defined. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  3. DC - (%/Amt) - Enter the value to issue discount towards the service for Daycare patients. The price is calculated based on the percentage of amount and the service price defined. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  4. OP - (%/Amt) - Enter the value to issue discount towards the service for Outpatients. The price is calculated based on the percentage of amount and the service price defined. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  5. Emg. - (%/Amt) - Enter the value to issue discount towards the service for Emergency patients. The price is calculated based on the percentage of amount and the service price defined. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  6. Ext  - (%/Amt) - Enter the value to issue discount towards the service for External/Referral patients. The price is calculated based on the percentage of amount and the service price defined. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  7. Valid From - Click Calendar function to select a date. The discount percentage defined for the patient class is valid from the date selected. The discount period has to be within the Billing Service period. The date selected has to be greater than the system date. This is a mandatory field.

  8. Valid To - Click Calendar function to select a date. The discount percentage defined for the patient class is valid until the date selected. The discount period has to be within the Billing Service period. The date selected has to be greater than the From date. This is a mandatory field.
