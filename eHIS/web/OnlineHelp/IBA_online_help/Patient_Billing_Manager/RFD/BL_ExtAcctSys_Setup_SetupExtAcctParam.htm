Enter the following details to define external accounting parameters:

 

  1. Facility – The name of the logged-in facility is displayed, by default. You cannot change the name of the facility. The parameters defined are specific to the logged-in facility.

  2. External Account for Receipts in Use – Select the check box to define external accounts. The functions ‘External Accounts by Receipt Type’, ‘Generate Bank-in Batch’ and ‘Update Bank-in Slip’ are enabled on selecting the check box. You cannot access the specified functions, if the check box is cleared.

  3. Bank-in Batch Doc – Click [List of Values] to select a default document type for Bank-in Batch. The Document Types defined using the ‘Document Types’ function in the Financials Base Manager module, and set as valid for ‘Patient Billing’ get listed, if the ‘Transaction Type’ is selected as ‘Miscellaneous’ in the ‘Document Types’ function under the ‘Other Masters’ menu.

  4. Length of External Account Code – Enter the length in numeric format of two digits maximum length. This restricts the length of the accounting code defined using the ‘External Receipts By Receipt Type’ function.

  5. Refund Cashbook Doc - Click [List of Values] to select a default document type for Refund Cashbook. The Document Types defined using the ‘Document Types’ function in the Financials Base Manager module, and set as valid for ‘Patient Billing’ get listed, if the ‘Transaction Type’ is selected as ‘Miscellaneous’ in the ‘Document Types’ function under the ‘Other Masters’ menu.

  6. Cost Centre – Enter the Cost Centre in alphanumeric format of 10 digits maximum length. The value recorded is set as the default value, if the Cost Centre is not selected during transactions.

  7. Bank Recon. Voucher Doc - Click [List of Values] to select a default document type for Bank Reconciliation. The Document Types defined using the ‘Document Types’ function in the Financials Base Manager module, and set as valid for ‘Patient Billing’ get listed, if the ‘Transaction Type’ is selected as ‘Miscellaneous’ in the ‘Document Types’ function under the ‘Other Masters’ menu.

  8. Deposit Register Doc – Click [List of Values] to select a default document type for deposit register. The Document Types defined using the ‘Document Types’ function in the Financials Base Manager module, and set as valid for ‘Patient Billing’ get listed, if the ‘Transaction Type’ is selected as ‘Miscellaneous’ in the ‘Document Types’ function under the ‘Other Masters’ menu.
