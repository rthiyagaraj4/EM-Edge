The following details are displayed:

 

  1. Customer Code – Click [List of value] to select the customer code. The customer code is defined in the Customer function in the Setup.

  2. Bill Date [From-To] – Select the Bill date using the calendar function.

  3. Bill Status – Select the bill status from the drop-down list. The listed values are With Exemption, With Adjustment and With Write-off.

   - This icon is displayed against the Bills with Adjustment

   - This icon is displayed against the Bills with Exemptions

   - This icon is displayed against the Bills with Write-offs.

  1. Patient Number – Press [F9] key to view and select a Patient ID. If the patient number is valid, patient details are displayed for user verification. The patient name, gender, age and National ID is displayed. By default, the bills for all episode and bills of all status are displayed. Query the bills for a particular type of episode, for an episode, for a range of bill dates and for a particular bill status. Upon entering the query criteria, click refresh to populate the details of the bills as per the query criteria. User has an option to re-print the bill.

  2. Patient Name – The patient name gets displayed based on the selection of the patient number.

  3. Sex – Select the gender of the patient from the drop-down list.

  4. Age – Enter the age of the patient in the format of Y/M/D.

  5. National ID – Enter the National ID of the patient.
