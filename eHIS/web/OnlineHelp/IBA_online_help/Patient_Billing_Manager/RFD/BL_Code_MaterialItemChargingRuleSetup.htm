Enter the following to set up Material Item Charging Rule:

 

  1. Sale Category Code - Enter the Sale Category code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Long Description - Enter the Long Description for the sale category in alphanumeric format with a maximum of 30 characters. This is a mandatory field.

  3. Short Description - Enter the Short Description in alphanumeric format with a maximum of 15 characters. This is a mandatory field.

  4. Cost Indicator – Select a Cost Indicator from the list. The values are system defined - Average Cost and Batch Cost. The price computation depends on the cost indicator selected. This is a mandatory field.

  5. Average Cost – Select Average Cost, the material item is charged an average cost if ‘Basis’ is selected as ‘Average Cost’ in the function ‘Setup/Amend Charge Factors for Materials Linked Services’.

  6. Batch Cost - Select Batch Cost, the material item is charged the cost if ‘Basis’ is selected as ‘Batch Cost’ in the function ‘Setup/Amend Charge Factors for Materials Linked Services’. The price for the material item is captured and calculated based on the prices defined in the excel sheet.

  1. Sale Price Rounding – Select a value from the list. The values are system defined - No Rounding, To Next Highest 10, To Next Highest 50 and To Next Highest Integer. This is a mandatory field.

  2. No Rounding – Select No Rounding to convert the decimal amount to a whole amount.

  3. To Next Highest 10 – Select To Next Highest 10 to convert the amount to nearby tens value.

  4. To Next Highest 50 – Select To Next Highest 50 to convert the amount to nearby fifty’s.

  5. To Next Highest Integer – Select To Next Highest Integer to convert the amount to nearby numeral.

  1. Min Sale Price - Enter the minimum sale price in numeric format of four digits maximum length. The amount indicates the minimum value for the sale category.

 

Refund Value

  1. Min Sale Value - Enter the minimum sale value in numeric format of five digits maximum length. A refund is issued on the item price if the minimum sale value for the item is equal to the entered value. A refund towards the sale category has to be lower than the amount specified.

  2. Min Deduction Charge - Enter the minimum deduction charge in numeric format of six digits maximum length. If the material item is returned, a minimum amount specified here is deducted from the actual amount.

 

Price Computation & Associated Billing Service by cost range

  1. Min Cost - Enter the minimum cost in numeric format of six digits maximum length before the decimal point and two digits after the decimal point. The minimum cost mentioned is applicable for the respective sale category. The minimum cost is considered when the price is computed based on the formula defined for the sale category. This is a mandatory field.

  2. Max Cost - Enter the maximum cost in numeric format with a maximum of seven digits before the decimal point and two digits after the decimal point. The maximum cost mentioned is applicable for the respective sale category. The maximum cost is considered when the price is computed based on the formula defined for the sale category. This is a mandatory field.

 

Markup %

  1. IP - Enter the markup percentage for Inpatients in the facility. The data has to be in numeric format with a maximum of three digits before and two digits after the decimal point. This is a mandatory field.

  2. DC - Enter the markup percentage for Daycare patients in the facility. The data has to be in numeric format with a maximum of three digits before and two digits after the decimal point. This is a mandatory field.

  3. OP - Enter the markup percentage for Outpatients in the facility. The data has to be in numeric format with a maximum of three digits before and two digits after the decimal point. This is a mandatory field.

  4. Emg - Enter the markup percentage for Emergency patients in the facility. The data has to be in numeric format with a maximum of three digits before and two digits after the decimal point. This is a mandatory field.

  5. Ext - Enter the markup percentage for External patients in the facility. The data has to be in numeric format with a maximum of three digits before and two digits after the decimal point. This is a mandatory field.

 

Admin Charge

  1. IP - Enter the Admin charge for Inpatients in the facility. The data has to be in numeric format with a maximum of seven digits before and two digits after the decimal point. This is a mandatory field.

  2. DC - Enter the admin charge for Daycare patients in the facility. The data has to be in numeric format with a maximum of seven digits before and two digits after the decimal point. This is a mandatory field.

  3. OP - Enter the admin charge for Outpatients in the facility. The data has to be in numeric format with a maximum of seven digits before and two digits after the decimal point. This is a mandatory field.

  4. Emg - Enter the admin charge for Emergency patients in the facility. The data has to be in numeric format with a maximum of seven digits before and two digits after the decimal point. This is a mandatory field.

  5. Ext - Enter the admin charge for External patients in the facility. The data has to be in numeric format with a maximum of seven digits before and two digits after the decimal point. This is a mandatory field

  6. Billing Service – Click [List of Values] to select a Billing Service. The values defined using the Billing Services function and selected as ‘Materials Item-related Service’ get listed. This is a mandatory field.

 

Price Computation Formula

Enter the price computation formula using the item cost, markup percentage, minimum cost and the admin charge. The computation formula is applicable to the selected record.
