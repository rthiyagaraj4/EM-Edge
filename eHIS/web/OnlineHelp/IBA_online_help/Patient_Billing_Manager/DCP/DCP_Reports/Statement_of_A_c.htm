Statement of A/c

From this function you can Generate a report to view the statements for the deferred cash patients.

The function Statement of Account is used to generate a report on the deferred cash patient details. The document amount recorded, the outstanding amount and the amount adjusted against the patient record are displayed. The invoice generated for the selected patient record is also displayed. You also have an option to generate the report including document details for zero balance. You can either exclude or include the associated transaction details and print only the originating documents.

Statement of Account

From the Deferred Cash Patients > DCP Reports, double-click Statement of A/c function.

The Statement of A/c screen gets displayed.

  1. Click Calendar function to select the Ageing Cut-off Date. You can also generate the report without specifying the other input parameters the system assumes the lowest and the highest value for the input fields under ‘From’ and ‘To’.<Related Field Description>

  2. Click [Run Process] to generate the report. Click View, Print or File to view the output online, print the output or to save the output in a file.  

  3. Click [OK] to generate the output and [C] to cancel the output. A log file is generated. Click [OK] to view the report and [Cancel] to exit from the screen.

 

• The report is in the order of Document Type and Document Number within the main report order specified in the input parameters. However, this is valid only for the detail ageing report.

• If the outstanding amount is credit, ageing is not considered and it is shown separately under the Open Credits column.

• The outstanding amount only is used for ageing analysis. The document amount is not used for ageing analysis purpose.

• Document date is used for ageing along with the ageing cutoff date.

• Original documents with outstanding amount as zero are not considered in this analysis.

• Detail report only shows the original document details and not the associated transaction details.
