Generate Installment Write-off Requests

From this function you can Generate a Write-Off Request against an outstanding installment payment.

The Generate Installment Write-Off Requests function is used to write-off an outstanding installment document for which the payment is not settled. You have to select the document against which the payment is not settled, after multiple reminder letters were dispatched. Authorized Users only can login to generate a write-off request. A document reference number is generated on committing write-off for the selected records.

Generate Installment Write-Off Requests

From the Deferred Cash Patients > Agreement for Installment Payment, double-click Generate Installment Write-Off Requests function.

The Generate Installment Write-Off Requests screen gets displayed.

  1. Select an Outstanding Amount Range. The Outstanding amount range defined using the Reminder Letter Setup function gets listed. The documents with the selected outstanding amount details get listed. Select All to view and to generate the reminder letters either for the selected documents or for all the documents. This is a mandatory field. <Related Field Description>

  2. Click [Authorize] to login and to authorize the write-off process.

  3. Select the records, which are to be written-off. The reason code specified while logging-in to authorize the write-off process gets displayed automatically on selecting the records.

  4. Click [Process] to write-off the outstanding documents. A screen pops-up requesting confirmation. Click [Yes] to write-off the document or [No] to cancel the process.
