Amend Discounts – (Any)

From this function you can Update the discounts defined for a Price Class or a Customer group, discounts for a Standard price defined or the discounts associated to a Billing Group. You can Query to view the discounts defined.

Amend Discounts – (Any) function is used to update the discounts for a Patient Class associated to a Price Class or a Billing Group, a Customer or associated to the standard prices defined. You can update the discount details, which are already defined.

Amend Discounts

From the Masters > Multi Pricing, double-click Amend Discounts – (Any) function.

The Amend Discounts – (Any) screen gets displayed.

  1. Select a Type from the list. This is a mandatory field.

  2. Click [List of Values] to select a Billing Service. The Description for the selected Billing Service gets displayed. This is a mandatory field. <Related Field Description>

  3. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  4. Click [Execute Query] or press [F8] to search for the existing records within the facility.
