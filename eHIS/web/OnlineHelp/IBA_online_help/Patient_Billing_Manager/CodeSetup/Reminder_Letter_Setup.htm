Reminder Letter Setup     

From this function you can Create a Reminder Letter for installment payments or for outstanding bill clearance. You can Query to view the Reminder Letters created.

Reminder Letter Setup function is used to create Reminder Letters to be sent to the patients or the customers to settle the pending bill amount. The Reminder Letters are dispatched with the Covering Letters defined. You can define any number of Reminder Letters and dispatch it to the patients or the customers based on the outstanding amount and the period for which the amount is outstanding. You have an option to set up a Reminder Letter for the outstanding bills or for the installment payments. The bill outstanding reminder letter is pertinent to a selected Billing Group Category.

Defining Reminder Letter Setup

From the Masters > Interfaced Modules Financial Dtls, double-click Reminder Letter Setup function.

The Reminder Letter Setup screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Enter the text that has to appear in the Letter Header, Letter Footer and the body of the letter.

  3. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  4. Click [Execute Query] or press [F8] to search for the existing records within the facility.
