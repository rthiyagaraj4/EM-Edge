Day Types/Time Types/Time Slabs

From this function you can Define Day Type, Time Type and Time Slabs necessary to identify the working days or the non-working days and working hours or the non-working hours in the Facility. You can Query to view the existing records.

The Day Types/Time Types/Time Slabs function enables you to define the working days and the non-working days in the facility. You also have an option to define the working and the non-working hours in a day. If the patient is admitted on a holiday, you can charge additional amount for the patient based on the day type, time type and the time slab. The calendar for a specific year is created based on the day types and the time types defined.

Creating a Day Type, Time Type and the Time Slab

From the Masters > Billing Calendar, double-click Day Types/Time Types/Time Slabs function.

The Day Types/Time Types/Time Slabs screen gets displayed.

  1. Enter the details in the mandatory (*) fields found in the following tab pages.

Day Types

The Day Type differentiates between a Working Day and a Holiday. Day Types are significant to set the Calendar pertinent to the Facility. The symbol ‘H’ represents a Holiday and ‘W’ represents a Working Day. Different Day Types are defined for Holidays and for Working days. For Example, various Day Types are used to define a full working day or a half working day.

Enter the following details to create a Day Type:

  1. Day Type - Enter the Day Type code in alphanumeric format of two characters maximum length. Day Type code is a unique code, set to identify each day type uniquely. This is a mandatory field and the code has to be unique. The Day Type code defined initially cannot be altered. The First character of the Day Type code has to be H or W, which represents either a Holiday or a Working day.

  2. Long Description - Enter the long description of the Day Type in alphanumeric format of 30 characters maximum length. This is a mandatory field. As in the screen, the Long Description provides an explanation for the selected Day Type.

  3. Short Description - Enter the short description of the Day Type in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Working Day – Select the check box, if the mentioned Day Type is a working day. Clear the check box, if the defined Day Type is set to be a Holiday.

  5. [Time Slabs for Day Type] - Click [Time Slabs for Day Type] and the tab gets shifted to Time Slabs. The Time Slabs set is for the selected Day Type.

Time Types

A Time Type identifies the working hours and the non-working hours in the Facility. Based on the Time Types, a patient is charged an additional amount, if admitted during non-working hours. The Time Type is significant to set the Calendar pertinent to the Facility.

Enter the following details to define a Time Type:

  1. Time Type - Enter the Time Type code in alphanumeric format of two characters maximum length. This is a mandatory field and the code has to be unique. The Time Type code defined cannot be changed.

  2. Long Description - Enter the long description in alphanumeric format of 30 characters each. This is a mandatory field.

  3. Short Description - Enter the short description in alphanumeric characters of 15 characters each. This is a mandatory field.

Time Slabs

The Time Slabs help in identifying the working, non-working hours at the Facility for a relevant Day Type. These Time Types are valid for 24 hours.

Enter the following details to create a time slab for a day type:

  1. Day Type – Click [List of Values] to select a valid Day Type. The Day Types defined using the Day Types tab gets listed. This is a mandatory field.

  2. Time Type - Click [List of Values] to select a valid Time Type. The Time Types defined using the Time Types tab gets listed. This is a mandatory field.

  3. Description - The Description for the selected Time Type gets displayed when a valid Time Type is selected from [List of Values].

  4. Time Range From - Enter the time in numeric format of four digits maximum length. The Time slabs for the selected Time Type are considered valid from the time specified. Time has to be entered in 24 hours format. For Example, 8 - 30 AM is specified as 0830 and 5 - 30 PM as 1730. This is a mandatory field. This is a mandatory field.

  5. Time Range To – Enter the time in numeric format of four digits maximum length. The Time slabs for the selected Time Type are considered valid until the time specified. Time must be entered in 24 hours format and the same range is to be followed as mentioned for the ‘From’ range. This is a mandatory field.

  1. Click [Commit Form] or press [F10] to save the records or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for existing records within the facility.
