Order Catalog Billing Details

From this function you can Define the Billing Service details for the Order Categories and define the stages when the patient has to be charged. You can Query to view the Order Catalog Billing Details defined.

Order Catalog Billing Details function is used to charge the Order Categories and the Order Types defined in the Order Entry and Result Reporting module. Either services or panels are associated to an Order Category. The prices for the Order Catalogs associated to the Order Category are charged based on the prices defined for a patient category while defining a Billing Service or a Billing Panel. You have to choose a charging stage for the Order Catalog. The Order Catalog can be charged while registering an order, or during order completion or during result generation.

Defining Order Catalog Billing Details

From the Masters > Interfaced Modules Financial Dtls, double-click Order Catalog Billing Details function.

The Order Catalog Billing Details screen gets displayed.

  1. Enter the details in the mandatory (*) fields found in the following tab pages.

Services for Order Catalog

Charging Stage for Order Catalog

  1. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] key to search for the existing records within the facility.

 

You have an option to update the charging stage for an individual patient class by selecting the check boxes manually.
