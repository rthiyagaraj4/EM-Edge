Bill Print Format

From this function you can Define the bill or receipt format to print bills/receipts for various transactions in the facility. You can also Define a bill format for a Customer and a Billing Group. Apart from these, you can also Define legends that has to be be printed on a bill/receipt. You can Query to view the existing records in the Facility.

The Bill Print Format function is used to define the format of a bill or receipt. The defined format can be associated to a Customer or a Billing Group. Thus, a bill or receipt  is printed in the defined format when corresponding Customer/Billing Group transactions occur. The legends that have to be printed in the bill/receipt can be defined manually. Hence, the legends that are to be printed on the bill/receipt are customizable. You can customize the bill header to print the Nursing Unit/Ward Name, Bed Number, Clinic Name, Primary Specialty along with the Doctor's Name.  

Defining Bill Print Format

From the Masters > Billing Rule, double-click Bill Print Format function.

The Bill Print Format screen gets displayed.

  1. Enter the details in the mandatory fields found in the following tab pages.
