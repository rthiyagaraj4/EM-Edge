Current Inpatient Exceeding Specific Duration

From this function you can Generate a report on the inpatients who are staying in the facility beyond a specific period

Current Inpatient Exceeding Specific Duration function is used to generate a report on the inpatients who are staying in the facility beyond a specific period. The report generated includes the Billing Group, patient details and ward details.

Listing Current Inpatients exceeding the stay duration

From the External Accounting System > Reports, Current Inpatient Exceeding Specific Duration Function.

The Current Inpatient Exceeding Specific Duration screen gets displayed.

  1. Click [List of Values] to select a Billing Group. The Billing Groups associated with the Facility are listed.

  2. Enter the number of days in numeric format with a maximum of 10 digits. Depending on the Number of days set, the report is generated for all the inpatients in the Facility who is exceeding the number of days specified. This is a mandatory field.

  3. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  4. You can view the report, print the details or save the output in a file in the system.
