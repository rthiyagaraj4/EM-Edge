Daily Returns

From this function you can Generate a report on the profit gained by the facility on a particular day.

The income gained from all the resources in the Facility at the end of the day are listed in the report generated. System generates a report listing the receipt date, Operator, daily returns from the selected Cash Counters, collection amount/refund amount, payment mode, returns amount, acknowledged receipt, total returns amount by Payment mode and grand total.

Printing Daily Returns

From the Transactions > Cash Book, double-click Daily Returns Function.

The Daily Returns screen gets displayed.

  1. Click [List of Values] to select a Facility. The Daily Returns from the selected range of Cash Counters in the respective Facility are listed. This is a mandatory field.

  2. Click [List of Values] to select a range of Cash Counters. The Daily Returns from the selected range of Cash Counters are listed.

  3. Click the Calendar function to select the document date range. The daily returns from the respective Cash Counters in the facility during the selected date range are listed in the report. This is a mandatory field.

  4. The values are system defined – 'Receipt', 'Refund' and 'Both'. Select 'Receipt' to generate the Daily Returns based on the Receipts processed at the Facility. Select 'Refund' to generate the Daily Returns based on the Refunds issued in the Facility. Select 'Both', if the report generated has to include daily returns for Receipts and Refunds.

  5. Select 'Exclude Credit Card Transactions' check box, if the cash collected by processing the Credit Cards should be included as well. Clear the check box to exclude the Credit Card transactions.

  6. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  7. You can view the report, print the details or save the output in a file in the system.
