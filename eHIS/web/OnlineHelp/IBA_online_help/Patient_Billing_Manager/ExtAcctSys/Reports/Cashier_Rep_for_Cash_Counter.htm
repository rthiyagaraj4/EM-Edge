Cashier Report for Cash Counter

From this function you can Create report subtitles. You can Generate a report on the transactions performed by a Cashier during a relevant Cash Counter shift.

Cashier Report for Cash Counter function is used to generate a report on the transactions performed by a Cashier during a relevant Cash Counter shift. You can define report subtitles using this function and default a specific subtitle, so that the default subtitle is printed on the Cashier report.

Creating Report Subtitles and Generating Reports

From the External Accounting System > Reports, Cashier Report for Cash Counter Function.

The Cashier Report for Cash Counter screen gets displayed.

  1. Enter the details in the fields in the following tabs:

Setup Report Subtitles

Enter the following details to create a record:

  1. Subtitle Id – Enter the Subtitle ID in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Subtitle Description - Enter the Subtitle Description in alphanumeric format of 100 characters maximum length. This is a mandatory field.

  3. Default – Select the check box, to set the Subtitle as the default record. Clear the check box, if the Subtitle is not the default record. You have to select a record as the default record. The record, which is selected as the default record, is displayed in the ‘Report Subtitle’ field under the ‘Generate Report’ tab.

Generate Report

Enter the following details to generate a report:

  1. Receipt Date [From – To] – Click the Calendar function to select a date range. A Cashier Report is generated for the date range selected. This is a mandatory field.

  2. Cash Counter [From – To] – Click [List of Values] to select a range of Cash Counters. A Cashier Report is generated for the relevant Cash Counter. This is a mandatory field.

  3. Operator – Click [List of Values] to select an Operator. Cashier Report is generated for the selected Operator.

  4. Exclude Credit Card Transactions – Select the check box, if the cash collected by processing the Credit Cards should be included as well. Clear the check box to exclude the Credit Card transactions.

  5. Bank-in Doc – Select a value from the list to generate the report. The listed values are – ‘With Bank-in’, ‘Without Bank-in’ and ‘Both’.

  6. Report Subtitle – The default Subtitle selected gets displayed. Click [List of Values] to select any other Subtitle. The values defined using the ‘Setup Report Subtitles’ tab are listed. The subtitle selected gets displayed as the report header.

  1. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records within the facility.

  3. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  4. You can view the report, print the details or save the output in a file in the system.
