Re-Print Bank-in Document

From this function you can Print the documents created using the Generate Bank-in Batch function.

Re-Print Bank-in Document function is used to print Bank-in Batch documents created using the function Generate Bank-in Batch. You can choose a specific Bank to view the documents generated by the respective Bank or clear the entries and select only the 'Bank-in Batch Date' to print the documents generated by all the banks. System prints the Bank Name, Bank-in Batch Date, Document Number, Total Amount, Payment Mode, Account Code and Description, Account code wise amount, Total amount for account code and Hospital details.

Re-printing Bank-in Documents

From the Transactions > Bank in Slip, double-click Re-print bank in Document Function.

The Re-Print Bank-in Document screen gets displayed.

  1. Click [List of Values] to select a Bank Code. The list displays the Bank Code, Batch Document Type, Batch Document Number and date when the documents are generated. The bank-in batch documents generated by the specific bank are printed. On selecting the Bank Code, date and document type and number gets displayed in ‘Bank-in Batch Date’ and ‘Bank-in Batch Doc Number’ fields respectively.

  2. The Bank-in Batch Date gets displayed automatically on selecting the Bank Code. The batch documents generated by the selected Bank on a specific date are printed.

  3. The Bank-in Batch Doc Number gets displayed automatically on selecting the Bank Code. The batch documents generated by the selected Bank on a specific date are printed.

  4. Click [Run Process] to print bank-in documents.

  5. A screen pops-up requesting confirmation to print the documents.

  6. Click [Yes] to print or [No] to cancel the process. The documents are printed in the default printer on clicking [Yes].
