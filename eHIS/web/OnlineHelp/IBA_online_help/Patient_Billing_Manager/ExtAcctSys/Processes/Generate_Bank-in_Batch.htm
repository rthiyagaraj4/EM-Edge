Generate Bank-in Batch

From this function you can Generate the Bank-in batch details.

Generate Bank-in Batch function is used to create a batch for a number of transactions. The Cash Control Documents are populated automatically. It is also used to process Bank-in Batch documents to generate Bank-in slips. The batch process can be completed only on generating Cash Control Receipts. You cannot create a batch, if Cash Control Receipts are not generated for all receipt transactions.

Generate Bank-in Batch

From the Transactions > Bank in Slip, double-click Generate Bank-in Batch Function.

The Generate Bank-in Batch screen gets displayed.

  1. Click the Calendar function to set the transaction date. Latter date transactions are not allowed for bank-in batch generation, if the previous date transactions are pending. System populates receipt transaction details associated to the date. <Related Field Description>

  2. Enter the maximum number of cheques per Bank-in slip. This indicates the maximum number of cheques that can be sent during Bank-in batch generation process.

  3. Click [Preview Batches] to view the Batches created for a selected transaction date. The Bank-in batch details are populated automatically. System displays the details before the batch is generated.

  4. Click [Execute Query] or press [F8] to search for the Cash Control Receipts.

  5. Click [Generate Batch] to create a batch of Cash Control Receipts.

  6. A document reference number is generated on successful generation of batch.

  7. A print confirmation message 'Do you want to Print?' is displayed. Click [Yes] to print or [No] to cancel the printing process.
