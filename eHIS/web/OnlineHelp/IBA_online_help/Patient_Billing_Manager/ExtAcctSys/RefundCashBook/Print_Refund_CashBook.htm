Print Refund Cashbook

From this function you can Print the Refund Cash Book entries for a specific period.  

Print Refund Cashbook function is used to print or view the Refund cash book, which is generated using ‘Refund Cash Book Process’ function. System generates a Refund Cash Book listing the Debit (Opening Balance) and Credit details. The surplus amount may be issued to the patient during settlement or during discharge based on the policy. This report function enables a user to track the refunds issued in the facility.

The following details are displayed:

  • Date

  • Name as in Receipt

  • Refund Document Type and Number

  • Deposit Number

  • Account code

  • Cash-on-hand

  • Cheque-on-hand

  • Voucher-on-hand

  • Closing Balance Total

The following details are displayed:

  • Cash-on-hand

  • Cheque-on-hand

  • Date

  • Document Number

  • Amount

  • Opening Balance Total

Print Refund Cash Book

From the External Accounting System > Refund Cash Book, double-click Print Refund Cashbook Function.

The Print Refund Cashbook screen gets displayed.

  1. Click [List of Values] to select the Opening Balance Date. This is listed only when the Replacement Voucher is generated for the selected period.

  2. The Closing Balance Date is set automatically when the Opening Balance Date is selected. This is listed only when the Replacement Voucher is generated for the selected period.

  3. Click [Run Process] to print the details. The Refund Cash Book details are printed based on the Opening and Closing Balance Date selected.
