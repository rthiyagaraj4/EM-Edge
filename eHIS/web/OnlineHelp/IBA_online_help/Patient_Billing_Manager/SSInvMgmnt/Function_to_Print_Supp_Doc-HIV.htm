Function to Print the Support Document - HIV

From this function you can Print support documents generated for HIV related services.

The Function to Print the Support Document - HIV function is used to print the support document. You can print only HIV service related documents. The invoice details of a patient with the Internet Send Date recorded through the function 'Update Internet Send Date for the Invoices' are displayed.

Printing Support Document for HIV

From the SS Invoice Management > Transaction, double-click on Function to Print the Support Document - HIV function.

The Function to Print the Support Document -HIV screen gets displayed.

  1. Click the Calendar function to select the Internet 'From' and 'To' Date. This is a mandatory field.

  2. Select the 'Support Doc Type' from the list. The records associated to the selected support document type are listed. This is a mandatory field.

  3. Click [Refresh] to query the records. Patient ID, ARV Code for the respective patient, Invoice Number, date and amount are displayed. HIV Service Types and the net amount for each Service Type are listed when you select the check box. The name and designation of the respective accounts personnel is displayed.

  4. Select All - Select the check box to generate document for the selected patient records.

  5. Click [Generate Document] to generate support document. A document reference number is generated.
