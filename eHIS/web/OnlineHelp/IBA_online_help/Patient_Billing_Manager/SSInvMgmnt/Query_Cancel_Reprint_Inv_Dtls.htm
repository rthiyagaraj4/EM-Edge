Query/Cancel/Reprint Invoice Details

From this function you can Query, Cancel or Reprint invoice or bills that are dispatched to Social Security by the facility.

The Query Invoice/Bill Details function enables you to query either invoice or bill details of a patient associated to Social Security. You can filter the invoice or bill details based on the Episode Type or the Visit Date. The encounter details of patients matching the search criteria gets displayed. The payment made against an invoice generated can also be viewed.

Query/Cancel/Reprint Invoice details

From the SS Invoice Management > Queries, double-click on Query/Cancel/Reprint Invoice Details function.

The Query/Cancel/Reprint Invoice Details screen gets displayed.

  1. Select the Invoice Type from the list.

  2. Click [List of Values] to select a Patient ID. The name, age, gender and national ID of the patient gets displayed on selecting the patient ID. This is a mandatory field.

  3. Click [List of Values] to select an Episode ID, if there are multiple episodes associated to the patient ID. You can view invoice/bills associated to a selected Episode.

  4. Click the Calendar function to select Visit Date Range. The invoice or bills generated during the selected period for the patient is displayed.

  5. Select the check box to select an Episode Type. All check boxes are selected, by default. Clear the check box against an Episode Type to exclude the selected episode from search criteria. Select the check 'All' to search bill or invoices of all Episode Types.

  6. Click [Refresh] to query the records matching search criteria.

  7. The SS Invoice Details and Service Charges associated to the selected Patient ID is displayed.

  8. Select the check box 'Print/Cancel' to print or cancel the respective record. Click [Print] to print or [Cancel] to cancel the invoice details. On clicking  [Print] a dialog box is displayed. Select 'View' to view the details, 'Print' to print the details or 'File' to save the details in a file.

    Authorized Users are only permitted to cancel the invoice details. An authorization screen pops-up on clicking [Cancel].

    Enter the following details to authorize:

    1. User ID - Enter the User ID in alphanumeric format. This is a mandatory field.

    An error message 'This User is not Authorized for Cancel Invoice' is displayed, if the user is not authorized to cancel the invoice. The check box 'Cancel Invoice' has to be selected for the respective user in the 'Users for Services' function in the Administration menu to cancel the invoice. If the check box is cleared, user is not authorized to cancel the invoice.

    1. Pin No - Enter the PIN Number of the respective User.

    2. Reason - Click [List of Values] to select a Reason. The values defined through the 'Reasons for Billing Actions' function with the 'Cancel Invoice' check box as selected are listed.

  9. Click [D] to view the payment details.

  10. The details associated to the selected patient ID during transactions are displayed.
