Generate Debit/Credit Note - HIV

From this function you can Generate Debit Notes or Credit Notes for HIV related services utilized by Outpatients, External or Emergency patients associated to Social Security in the facility.

The Generate Debit Note/Credit Note – HIV function enables you to generate either debit notes or credit notes for HIV related services. You can generate invoices for Outpatients, External or Emergency patients in the facility. A Debit Note is generated when there is an error in the original bill or if a service was not included in the bill during final bill generation. In such cases, debit notes are generated and dispatched to Social Security notifying the bill details for the services utilized. A Credit Note is generated if a refund has to be issued to the Social Security.

Generating Debit/Credit Notes for Outpatients/External/Emergency patients for HIV related services

From the SS Invoice Management > Transaction, double-click on Generate Debit/Credit Note - HIV function.

The Generate Debit/Credit Note - HIV screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Refresh] to query for the invoice details matching the search criteria.

  3. Click [Generate Invoice] to generate debit/credit notes.

  4. Click [SS Invoice not Required] if invoice is not required for the selected patient encounter.
