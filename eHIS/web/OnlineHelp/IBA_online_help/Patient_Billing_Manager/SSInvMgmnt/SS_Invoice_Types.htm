SS Invoice Types

From this function you can Define various Invoice Types and Associate default Document Types for the Invoice Types. You can also Define whether the Invoice Type is valid for Special Items and HIV related. You can Query to view the existing details.

The SS Invoice Types function is used to define various Invoice Types valid for Social Security transactions. The Invoice Types of a patient with the Customer as 'Social Security' are sent to Social Security to claim the bill amount. You can select the default Document Types for the selected Invoice Type. The default Document Type gets selected during transactions, if the Document Type is not selected. You have an option to define if the Invoice Type is related to HIV patients' or special items.

Defining SS Invoice Types

From SS Invoice Management > Setup menu, double-click SS Invoice Types function.

The SS Invoice Types screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Commit Form] or press [F10] to save or click [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for the existing records within the facility.
