SS Direct Debit/Credit Note Creation

From this function you can Generate Debit Notes or Credit Notes for all Invoice Types for a selected Episode Type.

The SS Direct Debit/Credit Note Creation function enables you to generate either debit notes or credit notes for a selected invoice type. You can generate invoices for Outpatients, Inpatients or all Episode Types. A Debit Note is generated when there is an error in the original bill or if a service was not included in the bill during final bill generation. In such cases, debit notes are generated and dispatched to Social Security notifying the bill details for the services utilized. A Credit Note is generated if a refund has to be issued to the Social Security. You can also verify if the support document is printed, or the document is printed or cancelled for the selected Invoice Type.

Creating SS Direct Debit/Credit Notes for Episode Types

From the SS Invoice Management > Transaction, double-click on SS Direct Debit/Credit Note Creation function.

The SS Direct Debit/Credit Note Creation screen gets displayed.

  1. Select the Invoice Type from the list. The values defined using the 'Invoice Types' function get listed.

  2. Click [List of Values] to select a Patient ID. The name of the patient, age, National and Episode ID, gets displayed.

  3. Click [List of Values] to select an Invoice number. This is a mandatory field.

  4. Select either 'All', 'Outpatient' or 'Inpatient' to select the respective Episode Type.

  5. Click [Refresh] to query for the invoice details matching the search criteria.

  6. SS Invoice Details and Service Charge details associated to the patient ID gets displayed.

  7. Enter the details in the mandatory (*) fields. <Related Field Description>

  8. Click [Generate Dbn/Crn] to generate debit/credit notes.
