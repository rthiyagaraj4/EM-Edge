Queue Numbering Control

From this function you can Create an association of Queue Series to a Location and Query for a desired Queue Numbering Control record.

Queue Series created in the Queue Series function is assigned to the queues for Walk-ins, Referral and Appointments. The assigned queues are associated to a specific Location, Practitioner and Resource using this function. Based on the field 'Assign Queue Number By' set as ‘System’ in the OP Parameter function, the values defined in this function are assigned for Queue Numbering.

Creating Queue Numbering Control

From the OP Admin Setup, select the Queue Numbering Control function.

The Queue Numbering Control Screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Descriptions>

  3. Click [Apply] to save a Queue Series associated to a location or click [Reset] to clear the entered data.

  4. Click [Query] to search for existing record.
