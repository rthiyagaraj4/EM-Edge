Resource for Location

From this function you can Create an association of one or more Resources to a Location and Query for a desired Resource for Location record.

Resource for Location function allows you to identify Resources against Locations. Based on the value selected in Resource Class, the values in Resource Type are filtered and populated from Application Masters module. The attributes defined like Working Days and Visit Type Indicators applicable for the Practitioner and Location combination can be updated at any time after creation. The association of a Resource to a Location depends on the Resource Identification Setup in Clinics and is limited to Resources that belong to the Resource Type if defined at the location level and also the Specialty of the Location.

Creating an association of Resource to Location

From the OP Code Setup, select the Resource for Location function.

The Resource for Location screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to associate Resource to a Location or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Resource for Location.

 

The Practitioner is not identified as a Resource Class if the selected Location Type is checked for Open to all Resources.
