Daycare Unit

From this function you can Create a new Daycare Unit by specifying the working days applicable and associating other attributes to the Daycare Unit. You can Query for a desired Daycare Unit.

A Daycare Unit represents a care location to cater to daycare consultations and procedures within a healthcare facility. A Daycare Unit created is used in the ambulatory healthcare setting and pertains to a particular Specialty. Patients visit the Daycare Unit with scheduled appointments or visit as Walk-in patients like in ‘Emergency’ Clinics. The Daycare Unit function allows defining attributes like Type of Visits Applicable, Mode of Registration, Level of Care, Working Days in a Week, Age Group Applicable, Specialty, Department and File Storage Location.

Creating a Daycare Unit

From the OP Code Setup, select the Daycare Unit function.

The Daycare Unit screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a created Daycare Unit or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Daycare Unit.
