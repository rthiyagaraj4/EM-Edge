Visit Rule

From this function you can define the rule for visits, based on the duration between each visit type if a facility follows Practitioner Based visit types.

The Visit Rule function is used to define a specific rule for each visit if the facility adheres to Practitioner Based visit. The function defines at least one rule as default for all the practitioners.

Creating a Visit Rule

From the OP Code Setup, select the Visit Rule function.

The Visit Rule screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Visit Rule created or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing Visit Rule record.
