Enter the following details to query referral details:

 

Referred From

  1. Referral Type/Source - Select the source type from the list as 'Login Facility', 'External' or 'Enterprise'. Depending on the selected value, select the value for the source.

  2. Specialty - Select the Specialty from the list.

  3. Service - Select the service corresponding to the selected specialty from the list.

  4. Location Type/Location - Select the location type from the list  as either 'Clinic' 'Nursing Unit', 'Procedure Unit' or 'Daycare'. Depending on the selected value, select the location from the list.

  5. Practitioner - Select the practitioner using [?] function.

 

If Specialty, Location Type, and Location are selected as query criteria, the Practitioner search list provides Practitioners belonging to the selected query criteria.

 

Referred To

  1. Specialty - Select the Specialty from the list.

  2. Service - Select the service corresponding to the selected specialty from the list.

  3. Location Type - Select the Location Type from the list as either 'Clinic' or 'Nursing Unit'. Depending on the selected value, select the location from the list.

  4. Practitioner - Select the practitioner from [?] function for the selected specialty.

  5. Patient ID - Enter patient ID in alphanumeric format of length as specified in MP parameter set up or select the patient ID using [?] function.

  6. Referral ID - Enter the Referral ID in alphanumeric format of 14 characters maximum length.

  7. Priority - Select the priority as 'Elective', 'Semi Emergency', 'Emergency' or 'Referral for Admission'.

  8. Status - Select the Status from the list.

  9. Old Referrals - Select the check box to view details of referrals made before current system date.

  10. Search Name With Local Lang - Select the check box to view records with the patient name in local language.
