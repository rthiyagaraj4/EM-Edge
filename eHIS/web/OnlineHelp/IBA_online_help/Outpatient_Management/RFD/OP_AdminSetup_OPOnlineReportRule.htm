Enter the following details to create an OP Online Report Rule record:

 

  1. Location - Select the Location using [?] function. This field represents the Location for which the Online Report Rule is to be defined. The displayed list is limited to the set of locations pertaining to the logged in facility. This is a mandatory field. When the rule is to be defined for all the locations of the logged in facility, ‘*ALL’ has to be selected.

  2. Visit Type - Select the Visit Type from the list of valid values. This field represents the Visit Type for which the Online Report Rule is to be defined. The displayed list is limited to the set of Visit Types associated with the selected Location. When the rule is to be defined for all the Visit Types associated with the selected Location, ‘*ALL’ has to be selected. This is a mandatory field.

  3. Report - Select the Report from the list of valid values. This field represents the Outpatient Online Report, which has to be generated. The displayed list is limited to the standard set of OP Online Reports. This is a mandatory field.

  4. Print Applicability - Select the check box to print the report. This flag could be used to change the applicability on temporary basis. The Online Report is generated and printed only when the check box is selected.
