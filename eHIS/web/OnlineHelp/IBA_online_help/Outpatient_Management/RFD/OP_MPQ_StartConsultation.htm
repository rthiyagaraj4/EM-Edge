Start Consultation

The Practitioner uses the Start Consultation function to mark the commencement of consultation. This function is enabled for an assigned patient queue record having status other than ‘Checked Out’ and ‘Consultation in Progress’. After recording consultation, the queue status changes automatically to Record Vital Signs, Patient Chart and Checkout.

  1. Click Start Consultation option from the queue status to display Start Consultation screen.

  2. The Current Status and Start of Consultation details are displayed.

  3. Click [OK] to start the consultation or [Close] to exit from the screen.

The system calculates the consultation start time only after you click Start Consultation.
