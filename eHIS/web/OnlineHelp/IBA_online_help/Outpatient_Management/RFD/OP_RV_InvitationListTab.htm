Invitation List

Invitation List tab is displayed if ‘Allow Visit Registration for Wait list Invitations without Appointment’ is selected in the OP Parameter. This tab displays all the invited patients from the waiting list.

  1. Click [Search] to display the list of invited patients for the current date or enter the search criteria and click [Search] to search for a specific invited patient. Click [Clear] to reset the records displayed in the result area.

  2. On searching, the invitation details of the wait list patients are displayed based on the entered criteria.

Based on the OP Parameter Setup, perform the following:

If ‘Allow Invitation No’ is selected as ‘Display’

  1. On searching, the ‘Invitation Number’ of the Wait List patients are displayed.

  2. Click ‘Wait list No’ hyperlink of the required patient to display the Add Visit screen.

If ‘Allow Invitation No’ is selected as ‘Accept’

  1. On searching, the details of Wait List patients are displayed.

  2. Click ‘Wait list No’ hyperlink to display the Invitation List screen.

  3. Enter the valid Invitation Number in the text box.

  4. Click [OK] to accept the Invitation Number and display the details in Add Visit screen or [Cancel] to exit from the Invitation List screen and display the details of Wait List patients.
