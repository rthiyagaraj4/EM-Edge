Enter the following details to create a Visit Rule:

 

  1. Rule ID: Enter the Rule ID. This is an alphanumeric field which is limited to four characters.

  2. Rule Description: Enter the rule description to define the Rule ID which is limited to 40 characters.

  3. Follow-Up visit: Enter the number of days between first visit and follow up visit.

  4. Secondary Visit: Enter the duration in number of days between first visit and secondary visit.

  5. No of Followup Visits Allowed: Enter the number of follow up visits allowed in numeric format.

  6. No of Secondary Visits Allowed: Enter the number of secondary visits allowed in numeric format.

  7. Allow Cumulative visit in Secondary visit: Select the check box to identify whether the accumulated visits in follow up can be carried to secondary visit.

  8. Enabled: By default, it is selected.
