Enter the following details to create a Resource for Location record:

 

  1. Location – This field has two components where you have has to select a value from list and [?] function to identify the care location. The list is the Location Type and [?] function is used to select all the Locations that come under the selected Location Type. The Location Type values listed are 'Clinic', 'Procedure Unit' and 'Daycare Unit'.  On selecting the required Location Type, the Locations listed are based on the following condition:

  2. All Locations that are created for the logged in facility under the selected Location Type and have effective status as enabled

  3. All the defined Locations must be associated to the Operator Station assigned to the logged in user

  1. Resource Class/Type - Select the Resource Class and Resource Type from the list. The values populated for Resource Class are ‘Practitioner’, ‘Room’, ‘Equipment’ and ‘Others’ for locations, which is based on Ambulatory Level of Care, and values are ‘Room’, ‘Equipment’ and ‘Others’ for locations, which is based on Emergency Level of Care. Based on the value selected in Resource Class, the values in Resource Type are filtered and populated from Application Masters module.

When you select the ‘Location Type’ as ‘Procedure Unit’; and the ‘Location’ as Procedure unit specific to Radiology:

  • Resource Class field is defaulted as ‘Equipment’ and disabled.

  • Resource Name loads the Equipment values specific to Radiology.

  1. Resource Name - This field is enabled on selecting the Location, Resource Class and Resource Type. Enter the resource name in alphanumeric format of 30 characters maximum length or click [?], to invoke a search screen to look for the required resource. The displayed list is based on the following conditions:

  2. Resources that fall under the selected Resource Class and Resource Type

  3. In case of Resource Class ‘Practitioner’, the listing is based on Specialty associated to the Location

  4. In case of Resource Class ‘Room’, the listing is based on Accounting Department associated to the Location

  1. Enabled - Select the check box if the record is to be activated and clear the check box if the record is to be deactivated.

  2. Working days - Click on the appropriate check boxes to indicate the days when the resources are applicable to the Location. Based on this definition, the Appointment Scheduling/Visit Registration transactions allow selection of resources only on applicable days of the week. The day check boxes are selected based on the working days of the Location selected and clear the check box to reallocate the working days for the resource.

 

Applicable Visit Type Indicators

  1. Applicable Visit Type Indicators – Identify the Applicable Visit Type Indicators applicable for the Resource by selecting the appropriate check boxes. The check boxes are enabled based on the Applicable Visit Type Indicators of the Location. Based on this definition, the Resources are listed for an Outpatient encounter during Visit Registration.
