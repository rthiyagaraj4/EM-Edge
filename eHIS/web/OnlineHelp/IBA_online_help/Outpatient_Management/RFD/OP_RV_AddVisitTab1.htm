Enter the following details:

 

  1. Citizen ID - Enter the Citizen ID in numeric format as entered while registration. The 'Prompt' specified for the citizen ID in MP Parameter is displayed as the field legend. The length and format as specified in MP Parameter has to be adhered.

  2. Alternate ID Type(s) - Enter Alternate ID number(s) in the format and length as specified in MP Parameter. Based on the MP Parameter setup, system displays the Alternate ID Type description(s).

  3. Other Alt ID - Select the required alternate ID from the list. The listed values are defined in the function 'Alternate ID Type' of MP module. On selecting the required alternate ID type, enter the ID in alphanumeric format of 20 characters maximum length. The field 'Other Alt ID' is displayed if 'Accept Other Alt ID' is set in MP Parameter (Alt ID checks tab) of Master Patient Index module.

  4. Alternate ID1 - Enter the Alternate ID of the patient. The Alternate ID of the patient is recorded during patient registration. If multiple patients are linked to the same Alternate ID, a screen is displayed with the list of patients registered with the same Alternate ID. The Alternate ID is defined in MP Parameter of Master Patient Index module. The details displayed are Patient ID, Patient Name, Patient Name In Local Language, Medical Record Number, Gender and Date of Birth. Click the Patient ID hyperlink to select a single patient from the list of patients for registering visit. If only one patient is associated to the Alternate ID1 and a patient search is performed based on the Alternate ID1 entered, the system displays the details of the patient in the 'Register Visit' function. If 'Check Unique' check box is selected for the Alternate ID in Alternate ID Checks Tab of MP Parameter function, you cannot associate multiple Patient IDs to a single Alternate ID. The Patient Details for Alternate ID is not displayed while searching for patients based on Alternate ID in 'Register Visit' function.

The field names are displayed based on the legends for Alternate ID defined in the MP Parameter.

  1. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in the MP Parameter. You can query for the patient details by invoking the [?] function. On entering the relevant details in the lookup screen, the result screen is populated with patient details that satisfy the entered query details. You can select the patient by clicking on the hyperlink of the patient. The control now shifts back to the 'Register Patient Visit' screen with the patient details populated. If the required patient record is not found in 'Patient Search' function', the patient has to be registered. In this case, on clicking [Register Patient] from the 'Patient Search' screen, the 'Register Patient' function is invoked, the Patient is registered and on completion of registration, the registered patient details are displayed in 'Add Visit' screen and additional details of the visit can be entered.

  2. Referral ID - Enter the Referral ID in alphanumeric format of 14 characters maximum length. You can query for the patient referral details by invoking the [?] function. On entering the relevant details in the lookup screen, the result screen is populated with patient referral details that satisfy the entered query details. You can select the patient by clicking on the hyperlink of the patient. The control now shifts back to the 'Register Patient Visit' screen with the patient details populated. If the required patient record is not found in 'Referral Search' function', the patient has to be registered. In this case, on clicking [Register Referral] from the 'Referral Search' screen, the 'Register Patient Referral' function is invoked, the Patient is registered and on completion of registration, the registered patient details are displayed in 'Add Visit' screen and additional details of the visit can be entered.
