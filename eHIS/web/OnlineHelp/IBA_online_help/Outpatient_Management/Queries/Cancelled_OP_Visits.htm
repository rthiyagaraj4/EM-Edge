Cancelled OP Visits

From this function you can Query a list of Cancelled OP Visits that were cancelled using the Cancel Visit function.

Querying for Cancelled OP Visits

From OP Queries, select the Cancelled OP Visits function.

The Cancelled OP Visits screen gets displayed.

  1. Enter the query criteria.

    Enter the following query criteria:

    1. Patient ID - Enter the Patient ID or click [?] function to search and select the patient.

    2. Specialty – Enter the Specialty or select using [?] function.

    3. Visit Date From/To – Select the date range using the calendar function. The Cancelled Visits records for the selected date range are displayed.

    4. Service – Enter the Service or select using [?] function.

    5. Location – Enter or select the Location using [?] function.

    6. Practitioner – Enter the Practitioner or select the practitioner using [?] function.

  2. Click [Search] to select the records that satisfy the query criteria or [Reset] to clear the entered search criteria.

  3. The Cancelled OP Visit details are displayed.

  4. Click [Clear] to clear the result screen.
