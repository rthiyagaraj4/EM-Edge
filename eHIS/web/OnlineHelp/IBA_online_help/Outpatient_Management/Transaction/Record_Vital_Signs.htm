Record Vital Signs

From this function you can Record the Vital Signs for patients.

The Vital Signs are measured to obtain a quick evaluation of the person's general physical condition. The list of Vital Signs can be specific to the Location (Clinic) selected if it is set at the Clinic level. The Vital Signs can be recorded any number of times as required. You can either overwrite the previous values or enter new values. Vital Signs are set in OP Parameter, Location (Clinic) or in Visit Type for Location functions.

Recording Vital Signs

From the OP Transactions, select the Record Vital Signs function.

The Record Vital Signs screen gets displayed.

  1. Select the Specialty and click [Search] to display all the patients with visit registered and patients arrival marked or enter the search criteria and click [Search] to select specific patients. Click [Clear] to reset the records displayed in the result area.

    Enter the following details:

    1. Specialty – Select the Specialty from the list.

    2. View (Single/Multiple) - Click on the required option to select the view required. If you select Single View, additional query criteria can be specified to restrict the Patient records. Single is selected, by default.

    3. Location Type - Select the Location Type and Location from the list of valid values. The Locations that are associated to the Operator Station are displayed.

    4. Practitioner - Select the Practitioner using [?] function. The Practitioners associated to the displayed Locations are displayed.

    5. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter.

    6. Gender - Select the required value for gender from the list. The valid values are 'Male', 'Female' and 'Unknown'.

    7. Encounter ID - Enter the Encounter ID in numeric format of 12 digits maximum length.

    8. Queue No - Enter the Queue Number in alphanumeric format of six characters maximum length.

    9. Date – The current date is displayed. It cannot be modified.

    10. Start Time – The Start Time is displayed as '00:00'. The patients registered from '00:00' hours onwards for the current day are displayed. It cannot be modified.

    11. Display – Select the display type as ‘By Visit Type’, ‘Previous Day Visits Only’ and ‘Include Check-out patients’ from the list. Based on the selection the records are displayed.

    12. Appointment Status - Select the required value for appointment status from the list. The valid values are 'Yet to come', 'On time', 'Late' or 'No show'.

  2. Click the ‘+’ hyperlink to view the Patient Movement details.

  3. Click Patient Details hyperlink of a particular patient to display the Vital Signs screen.

  4. Enter the value appropriate to the Vital Signs variables.

  5. Click [Record] to save the Vital Signs details, [Clear] to clear the entered data or [Cancel] to exit from the screen.

 

The Record Vital Signs function is invoked through the ‘Manage Patient Queue’ function or ‘Record Vital Signs’ function.

The variables are defined in the Discrete Measure Panel and the Result Type is defined in the Discrete Measure functions in Application Masters module.

You can view the previously recorded values (if any) for the patient by clicking on the date and time hyperlink and the recorded values are displayed.
