Self Check-in Appointment

From this function you can Check-in your OP Appointments.

This function allows patients to check-in their OP appointments by themselves. Thereby, reduces the number of Reception Staff and to reduce the Queues for registration.

The system only displays the available appointments for the current day for a patient. Also, allows to view the appointment details of the patient.

The system will not consider order catalogue for the self-check in appointment.

Check-in OP Appointments

From the OP Transactions, select the Self Check-in Appointment function.

The Self Check-in Appointment screen gets displayed.

  1. Enter or scan the required patient ID. The system displays the available appointments for the current day for that patient with Register button.

  2. The Patient name and Location details are displayed both in English and local language in separate columns.

  3. Click ‘+’ button to view the appointment details in the Appointment Details dialog.

  4. Click [Register] to check-in for the current day appointment. The Internal Reports dialog gets displayed.

  5. Click [OK] to print the required labels.

Note: While creating appointments in OA, if service is recorded then only the Register button is enabled. If the service is not entered during registration, then the Register button will be disabled.

  1. When the registration is done, the system generates the Encounter ID for the patient.
