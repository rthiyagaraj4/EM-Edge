Cancel Visit

From this function you can Cancel the registered visits.

Visit registration against a Clinic remains canceled for reasons like non-availability of resources or patient request. This function allows canceling a visit registration within a specified time limit. Any Visit stands canceled only on recording a valid reason substantiating the cause for audits later.

Cancelling a Visit

From the OP Transactions, select the Cancel Visit function.

The Cancel Visit screen gets displayed.

  1. Click [Search] to display all the patients with visits registered or enter the search criteria and click [Search] to select specific patients. Click [Clear] to reset the records displayed in the result area.

  2. On searching, the Visit Details of the patient are displayed based on the entered criteria.

    1. Encounter ID – The encounter Identification number generated at the time of registration is displayed.

    2. Regn Date/Time – Registered date and time of the visit is displayed with a hyperlink.

    3. Patient Details – The details of the patient such as patient name, patient ID and date of birth are displayed.

    4. Location – The Location of the Visit is displayed.

    5. Practitioner Name – The Name of the Practitioner associated with the Visit is displayed.

    6. Queue Status - The Queue Status of the visit is displayed.

  3. Click Encounter ID hyperlink to display the Cancel Visit screen. The Queue details are displayed.

    Queue Status

    1. Status - Current queue status of the visit record of the Patient along with the date/time the status was set is displayed.

    2. Room – The room code with a short description is displayed.

    3. Appointment - Indicates if the selected visit record pertains to an Appointment visit or not.

    4. Service/Sub-service – The Service and Sub-service associated with the visit is displayed.

    5. Visit Type – The Visit Type associated with the visit and the visit type indicator is displayed.

    6. Patient Type – The Patient Type under which the patient visit was recorded is displayed.

    7. New Episode - Displays if this visit opened a new Episode.

    8. Episode/Visit No – the Associated Episode/Visit Number is displayed.

  4. Select the reason for cancelation from the list of valid values to cancel a particular visit. All the reasons defined with cancel visit check box selected in ‘Other Master Setup (Reason)’ of Application Masters module are listed for selection.

  5. Click [Apply] to cancel a particular visit and display the Search Visit screen to select the next patient for Cancelation or [Reset] to clear the entered data.

 

Visits can be canceled if Current date is within the allowed number of days relative to the visit date, as per the OP Parameter Setup. Visit status has to be ‘Checked In’ or ‘At Nursing Station’ or ‘Initial Check Up/Vital Signs Recorded’. A Visit with the status ‘Started Consultation’ cannot be canceled. Visits with entries in Medical Records cannot be cancelled.

You cannot cancel a visit if there exists an order associated to the encounter and the orderable is billed on placing the order.

The package associated to the visit has to be cancelled in order to cancel the visit. The individual orders associated to the package can be cancelled, if it is not billed on ordering. The orders that are processed or billed cannot be cancelled.

The services within a package that is not cancelled are charged based on the standard price defined.
