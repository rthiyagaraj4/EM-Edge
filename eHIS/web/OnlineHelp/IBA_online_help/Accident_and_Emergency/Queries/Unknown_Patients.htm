Unknown Patients

From this function you can Query the Unknown patients in a location within the emergency department.

This function is used to query the Unknown patients against a location within the emergency department. The function provides various query criteria used to filter the listing. This is used to view the list of all unknown patients registered in the emergency department with details of their visits and current status.

Querying Unknown Patients

From the AE Queries, select the Unknown Patient function.

The Unknown Patient screen gets displayed.

  1. Select the Location from the list.

  2. Select the Gender from the list.

  3. Click the Calendar function to select the Date range. The date entered has to be in dd/mm/yyyy format. The search results are filtered based on the date range selected. This field is mandatory.

  4. Click [Search] to select a specific record and view details or [Search] to clear the records displayed in the result area.

  5. Click [Reset] to clear the entered data.
