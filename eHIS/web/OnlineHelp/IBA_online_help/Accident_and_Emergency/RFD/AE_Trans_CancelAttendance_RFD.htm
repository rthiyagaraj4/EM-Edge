Enter the following query criteria to search for the required details and to cancel attendance:

 

  1. Start Date - Enter the Start Date in dd/mm/yyyy format or click the Calendar function to search for the required date. The current system date is displayed, by default. The Start Date should be greater or equal to the number of days specified for 'Max No of days for visit revising/cancelling' in OP Parameter. This field is mandatory.

  2. Location - Select a Location using the [?] function for which the visit records are to be selected.

  3. Patient ID - Enter the Patient ID in alphanumeric format, or click [?] function to invoke the Search Patient function to search and select the required Patient ID.

  4. Encounter ID - Enter the Encounter ID of the visit to be selected in numeric format of 12 digits maximum length.

  5. Practitioner - Select the Practitioner using [?] function for whom the Patient visits are to be selected.

  6. Include Treatment Area Assigned Patients – Select the check box to cancel attendance of the patients who has been assigned a treatment area during attendance registration. Only those patients for whom the consultation is not started or no orders placed during attendance registration can be cancelled. You cannot cancel attendance of the patients for whom Secondary Triage has been recorded. Clear the check box, if the patients who are assigned Treatment Area during registration need not be included to cancel attendance. The check box is cleared, by default.
