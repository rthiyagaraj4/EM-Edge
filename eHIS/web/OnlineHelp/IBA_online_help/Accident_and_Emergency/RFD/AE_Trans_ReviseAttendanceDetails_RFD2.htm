Visit Detail Tab

 

When ‘Visit detail’ tab is clicked, the details of the patient as entered during attendance registration are displayed. You cannot modify the Visit details.

 

Arrival Tab

 

Escort Detail

  1. Accompanied By - The name of the person accompanying the patient who is registered is displayed. The list displays all the values set in ‘Escort Type’ under Other Master Setup of Application Masters module.

  2. Relationship – The relationship of the person accompanying the registered patient if already captured is displayed. If the detail is not recorded, select a valid relationship of the person accompanying the registered patient.

  3. Address Line (1 to 4) – The address details are displayed, if already captured while registering the attendance.  If the details are not recorded, enter the address details in alphanumeric format of 30 characters maximum length in each line. The legends are set in System Manager module.

  4. Post Code – The postal code if already captured while registering the attendance is displayed. If the details are not recorded, enter the Postal Code in alphanumeric format of 8 characters maximum length. The entered value must correspond to the selected town. You can select the required code using the [?] function.

  5. Town - Enter the name of the Town in alphanumeric format of 30 characters maximum length. The entered name must be listed under the selected area. If the selected area has more Towns attached to it, select the required town using the [?] function. The field legend is set in System Manager module. The Area and Region associated to the selected Town is displayed by default.

  6. Area - Enter the Area in alphanumeric format of 30 characters maximum length or select it using the [?] function. The field legend is set in System Manager module.

  7. Region – Enter the name of the Region in alphanumeric format of 30 characters maximum length or select it using the [?] function. The area must correspond to the entered Region. The field legend is set in System Manager module.

  8. Postal Code - Enter the Postal Code in alphanumeric format of eight characters maximum length. The entered value must correspond to the selected Town. Select the required code using the [?] function. The field legend is set in System Manager module.

  9. Country - The Country name gets automatically populated when the Region is selected.

  10. Remarks – Enter remarks if any for the arrival details in alphanumeric format.

 

Ambulance Details

 

  1. Name of Escort/Ambulance Service – The name of the Escort/Ambulance Service if already captured in alphanumeric format is displayed.  Enter the Name of the Escort/Ambulance Service in alphanumeric format of maximum 30 characters if it was not captured. This is used to record the name of the escort service or ambulance service if any.

  2. Escort/Ambulance Service Contact No – The Escort/Ambulance Service Contact No. is displayed, if it was already captured while registering the attendance. If the details are not recorded, enter the contact number of the escort service in alphanumeric format of 10 characters maximum length.

  3. Motor Vehicle Reg No – The Motor Vehicle Reg No. is displayed, if it was captured while registering the attendance. If the details were not recoded,  enter the motor vehicle registration number of the vehicle in alphanumeric format of 10 characters maximum length.

 

Related Contacts Tab

 

Next of Kin

  1. Name - Enter the Name of the person to be contacted in alphanumeric format of 30 characters maximum length.

  2. Relationship - Select the Relationship the patient shares with the Next of the Kin from the list.

  3. Citizen ID – Enter the Citizen ID of the Next of Kin in the required format as specified in MP Parameter.

  4. Birth Date - Enter the Date of Birth in date format of dd/mm/yyyy, or use the Calendar function to select for the relevant date.

  5. Occupation - Enter the Occupation of the person in alphanumeric format of 30 characters maximum length. The field names are displayed based on the legends for Alternate ID defined in the MP Parameter.

  6. Address Line (1 to 4) – Enter the Address Details in alphanumeric format of 30 characters maximum length in each line. The legends are set in System Manager module.

  7. Town - Enter the name of the Town in alphanumeric format of 30 characters maximum length. The entered name must be listed under the selected area. If the selected area has more Towns attached to it, select the required town using the [?] function. The field legend is set in System Manager module. The Area and Region associated to the selected Town is displayed by default.

  8. Area - Enter the Area in alphanumeric format of 30 characters maximum length or select it using the [?] function. The field legend is set in System Manager module.

  9. Region – Enter the name of the Region in alphanumeric format of 30 characters maximum length or select it using the [?] function. The area must correspond to the entered Region. The field legend is set in System Manager module.

  10. Postal Code - Enter the Postal Code in alphanumeric format of eight characters maximum length. The entered value must correspond to the selected Town. Select the required code using the [?] function. The field legend is set in System Manager module.

  11. Country - The Country name gets automatically populated when the Region is selected.

  12. Home Phone No – Enter the Residence Telephone Number of Next of Kin in alphanumeric format of 20 characters maximum length.

  13. Mobile Phone No - Enter the Mobile Number of Next of Kin in alphanumeric format of 20 characters maximum length.

  14. Office Phone No - Enter the Office Telephone Number of Next of Kin in alphanumeric format of 20 characters maximum length.

  15. Email – Enter the Email ID of the person in alphanumeric format of 50 characters maximum length.

Employment Details Tab

 

Employer Details

  1. Organization Name - Enter the Organization Name in alphanumeric format of 30 characters maximum length.

  2. Contact Person Name - Enter the name of the Contact Person in the organization in alphanumeric format of 30 characters maximum length.

  3. Address Line (1 to 4) – Enter the Address Details in alphanumeric format of 30 characters maximum length in each line. The legends are set in System Manager module.

  4. Town - Enter the Town in alphanumeric format of 30 characters maximum length. The entered Town gets listed under the selected area. If the selected area has more towns attached to it, select the required town using the [?] function. The field legend is set in System Manager Module.

  5. Area - Enter the Area in alphanumeric format of 30 characters maximum length or select it using the [?] function. The field legend is set in System Manager module.

  6. Region – Enter the Region in alphanumeric format of 30 characters maximum length or select it using the [?] function. The area must correspond to the entered Region. The field legend is set in System Manager module.

  7. Postal Code - Enter the Postal Code in alphanumeric format of eight characters maximum length. The entered value must correspond to the selected Town. Select the required code using the [?] function. The field legend is set in System Manager module.

  8. Country - The Country name gets automatically populated when the Region is selected.

  9. Home Phone No – Enter Residence Telephone Number of the contact person (in the organization) in alphanumeric format of 20 characters maximum length.

  10. Office Phone No - Enter Office Telephone Number in alphanumeric format of 20 characters maximum length.

 

Employee Details

  1. Employment ID - Enter the Employment ID of the Patient in alphanumeric format of 15 characters maximum length.

  2. Occupation Class – Select the Occupation Class from the valid list of values. The listed values are defined in the function 'Occupation Class’ of Master Patient Index Code Setup.

  3. Occupation of the Person - Select the Occupation from the valid list of values. The listed values correspond to the selected occupation class. If the required occupation is not available, enter the same in alphanumeric format of 15 characters maximum length in the text box provided.

  4. Status - Select the Employment Status of the Patient from the valid list of values. The listed values are pre-defined.
