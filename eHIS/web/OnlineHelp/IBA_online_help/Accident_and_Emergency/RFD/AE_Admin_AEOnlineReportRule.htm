Enter the following details to set the AE Online Report Rule:

 

  1. Location – Select the Location from the valid list of values. This field represents the Location for which the Online Report Rule is to be defined. The displayed list is limited to the set of locations pertaining to the logged in facility. This field is mandatory. When the rule is to be defined for all the locations of the logged in facility, ‘*All’ should be selected.

  2. Visit Type - Select the Visit Type from the valid list of values. This field represents the Visit Type for which the Online Report Rule is to be defined. The displayed list is limited to the set of Visit Types associated with the selected Location. This field is mandatory. When the rule is to be defined for all the Visit Types associated with the selected Location, ‘*All’ is selected.

  3. Report – Select the Report from the valid list of values. This field represents the AE Online Report, which is to be generated. The displayed list is limited to the standard set of AE Online Reports. This field is mandatory.

  4. Print Applicability – Select the check box to indicate whether the selected report is to be printed or not. This flag is used to change the applicability on temporary basis.
