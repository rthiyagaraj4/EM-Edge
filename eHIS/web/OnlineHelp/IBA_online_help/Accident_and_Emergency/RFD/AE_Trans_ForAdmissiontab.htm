For Admission

This sub menu is applicable only of users in ML-MMOH sites.

The For Admission, Cancel for Admission, and Modify Referral sub menus will be visible based on the below configuration:

• ‘Enable for Admission’ check box selected in AE Parameter screen.

• For Admission, Cancel For Admission, Modify Referral check boxes selected in Access Rights for Queue Management screen.

 

Once the consultation is started, the ‘For Admission’ sub menu will be visible so that the patient can be moved to 'For Admission' tab with or without a referral.

  1. Click the [For Admission] sub menu, the system prompts message ‘This patient is marked for Admission. Do you want to continue?’ with OK and Cancel buttons.

  2. Click [Cancel] to abort the action and focus is back to the Current Emergency Patients screen.

  3. Click [OK], the system opens the Register Referral Details dialog. User can register the referral and proceed for admission using the referral ID.

  4. Click [Close] button to close the Register Referral Details dialog without registering a Referral. On closing the Register Referral details dialog, the system focus is back to the Current Emergency Patients screen.

  5. When the patient is marked for admission, in the status column the status ‘Waiting For admission’ will appear when the action is done.

  6. The View Details hyperlink will be displayed only when a referral is recorded against the encounter. If the referral is not generated, then View Details hyperlink will not be displayed.

  7. Click 'View Details' hyperlink to view the referral details in the Referral Details dialog.

Note: The patient’s name disappears automatically from “For Admission” tab only if:

• If the patient is checked out from Emergency using AE Checkout.

• If cancel for admission is done using Cancel for Admission hyperlink.

• If Assign Bed / Mark Patient Arrival is done in Inpatient module.

Modify Referral sub menu

The 'Modify Referral' sub menu appears only when the patient has a referral generated against the patient. Also, based on the access rights given for the user in Access Rights for Queue Management function, the sub menu will be visible for the users.

Click the Modify Referral sub menu, the Edit Referral window gets invoked where the referral inputs can be modified against the referral ID.

Cancel For Admission sub menu

Click the 'Cancel for Admission' sub menu to cancel the admission after selecting the reason for cancellation.

  1. Reason for Cancellation - Select the Reason for Cancellation from the drop-down list. The reasons for cancellation are defined and enabled in the ‘Reason’ master in AM module. This field is mandatory.

  2. Remarks - Enter the remarks for cancellation in the Remarks field. This field is mandatory.

  3. Cancelled By - The field is non editable and will automatically record the username who is performing the cancellation. This field is mandatory.

  4. Click the [Cancel for Admission] button, system will cancel the admission and removes ‘for admission’ status and reverts it back to the original status.

  5. Click [Abort Cancel For Admission] to abort the cancellation process and focus is back to the ‘For Admission’ tab in the Current Emergency Patients screen.

  6. When Cancel for admission is done, the patient record will be removed from the ‘For admission’ tab and will remain in their original assigned/unassigned priority zone.
