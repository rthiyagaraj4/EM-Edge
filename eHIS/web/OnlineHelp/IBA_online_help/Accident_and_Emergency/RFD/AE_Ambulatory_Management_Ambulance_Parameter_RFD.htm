Enter the following details to record Ambulance Parameter:

 

Ambulance Odometer

 

  1. Odometer Units – Select the odometer units value from the drop-down list. The list displays the short description of the Unit of Measure values created in the ‘Units of Measure’ screen under the Materials Setup menu in Application Master. This is a mandatory field.  

 

Request Queue Period

 

  1. Display past requests for and Display future requests for - Enter the maximum number of days in three digits. Based on this value, the past and future requested ambulance records will be displayed in the Ambulance Dashboard. You can change the value when required.

 

Incident Register

 

  1. Incident Numbering By - Select the incident numbering value from the drop-down list. The list values include Facility ID, Ambulance ID, Year and Customer Prefix. The selected value will be the prefix for the incident number. By default, the value 'Facility ID' is displayed. This is a mandatory field. It is a one-time setup.

  2. Prefix – This field is available only when the incident numbering is selected as ‘Custom Prefix’. You can enter two digits in alphanumeric format.

  3. Start Number - Enter the number which will be the Incident Start number. This is a mandatory field. You can enter up to nine digits maximum length.

  4. Next Number - Enter the number which will be the subsequent number to be provided for the incident. This is a mandatory field. You can enter up to nine digits maximum length.
