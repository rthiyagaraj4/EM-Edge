GCS for Response

From this function you can Query the existing Responses for defined categories to assign GCS score within an Emergency Location.

The Glasgow Comma Scale (GCS) is used to establish the degree of consciousness in a patient. It is based on the response to the stimuli. Ratings range from 3-15 with 15 being the highest. A patient is assessed against the criteria of the scale and the resulting points give the Glasgow Comma Score (GCS). GCS for Response helps in categorizing different responses into GCS. GCS for Response function is used to record various responses of the patient as per the Standard GCS, which has a scoring for each response, based on Eye Response, Verbal Response and Motor Response.

Querying a GCS Response

From the AE Code Setup > Code Setup, select the GCS for Response function.

The GCS for Response screen gets displayed.

  1. Click [Query] to search for GCS responses. GCS responses are displayed. <Related Field Description>

  2. Click Code hyperlink to modify the description of a particular GCS Response.

  3. Click [Apply] to save the GCS responses or [Reset] to clear the entered data.

 

The GCS for Response values are provided through scripts to restrict you to 15 observations only. As per the Standard Glasgow Comma Scale, the maximum score value for recording responses is 15 and the minimum is 3.
