Resource for Ambulance  

From this function you can Associate the staff and equipment required for the ambulance.

This function allows you to assign the staff and equipment for the ambulance created in the Ambulance function. You can assign the same staff and equipment for multiple ambulances.

Associating Staff/Equipment with Ambulance

From AE Ambulatory Management menu, select the Resource for Ambulance function.

The Resource for Ambulance screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the required Ambulance from the drop-down list. This is a mandatory field.

  3. Select the Staff or Equipment option to associate staff or equipment to the ambulance. By default, the Staff option is selected.

To associate the Staff to the Ambulance:

Select the Staff option and click [Search].  The search results include the ‘Associated Staff’ section and ‘Available Staff’ section.

Associated Staff section - The Associated Staff section includes:

i. The list of staff associated to the selected Ambulance if any.

ii. The ‘Select’ check box against each record is selected by default.

iii. The Display Contact check box is available in each record, you can select this check box against only one record. The selected staff description will be available in the ambulance bubble in the Dashboard.  

iv. If you want to remove the staff associated with the ambulance, clear the ‘Select’ check box and the removed staff name gets added to the Available Staff List.

v. Click [Apply] to save the changes.

You can select Display Contact for only one staff record, if you try to select for more than one record, an error message ‘Only one resource can be marked for contact display’ is displayed.

 

Available Staff section - This section allows you to search for available staff based on Staff Type i.e., Practitioner, External Practitioner and Other Staff. The details displayed in the Available Staff section are Description, Position, and Type.

To associate any staff to the ambulance, select the check box against the required staff record from Available Staff list and the selected name appears in the Associated Staff section,

Click [Apply]. The staff name gets added to the Associated Staff section.

To associate Equipment to the Ambulance:

Select the Equipment option and click [Search].  The search results include the Attached Equipments section and Available Equipments section.

Attachment Equipments – This section lists the equipments attached to the ambulance. The equipment description and Select check box gets displayed.

If you clear the select check box against any equipment record, the equipment associated with the ambulance gets removed and that equipment appears in the Available Equipments section.

Click [Apply] to save the changes done.

 

Available Equipments – This section lists the available equipment that can be assigned to the ambulance.

You can also search for available equipment by entering in the Equipment ‘Starts With’ field and clicking [Search].

To associate any equipment to the ambulance, select the check box against the equipment description, the selected equipment record gets added in the Attached Equipments section.

Click [Apply] to save the changes done.
