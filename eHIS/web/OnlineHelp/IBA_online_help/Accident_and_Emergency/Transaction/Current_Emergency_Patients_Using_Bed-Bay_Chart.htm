Current Emergency Patients using Bed/Bay Chart

From this function you can View patients occupying various Beds or Bays within the Emergency Location and perform various tasks on a patient based on the patient’s status.

Current Emergency Patients using Bed/Bay Chart function is used to view Bed/Bay availability within a treatment area. This function provides listing of patients under two tabs, namely, Patient Management and Unassigned List. The various tasks that can be performed from this function are Secondary Triage, Assign/Reassign Practitioner, Assign/Reassign Treatment Area, Trauma Assessment, Primary Assessment, Start Consultation, Check Out and View Movement History.

Managing Emergency Patient Queue

From the AE Transactions, select the Current Emergency Patients using Bed/Bay Chart Function.

Current Emergency Patients using Bed/Bay Chart screen gets displayed. This screen displays the following two tabs:

Patient Management

Unassigned List

When you try to start consultation for an emergency patient having any unsettled or unbilled due, an error or warning message is displayed. This message is displayed based on the value selected in the ‘Action when unsettled/unbilled amount for the encounter’ drop down list in the AE Parameter screen.
