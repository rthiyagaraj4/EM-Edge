Enter the following details:

 

  1. Facility - Select the facility from the list. The list displays all the facilities defined in the 'Facility' function of System Manager Module. This is a mandatory field.

  2. Location type - Select the Location Type from the list. The list displays all the location types defined in the 'Location Type' function of System Configuration module.

  3. Workplace - Select the Workplace using [?] function. If the Location Type is selected, workplaces listed correspond to the selected location type or all the workplaces created in the 'Workplace' function are displayed.

  4. Group By - Select Group By as 'Workplace' or 'Position' from the list. By default 'Workplace' is displayed.

  5. View options - Select an option from the list. Values for View Options list item get populated based on the 'Group By' value chosen.

If 'Group By' value is chosen as Workplace, the 'View Options' lists

  • Position, Day

  • Shift, Day

  • Shift, Position, Day

  • Position, Shift, Day

If 'Group By' value is chosen as Position, the 'View Options' lists

  1. Workplace, Day

  2. Shift, Day

  3. Shift, Workplace, Day

  4. Workplace, Shift, Day

  1. Duration [From - To] - Enter the date range in dd/mm/yyyy format or select using the calendar function to search for the required date, for which the workplace allocation details have to be queried for a particular Workplace/Position/Shift/Day combination. This is a mandatory field.
