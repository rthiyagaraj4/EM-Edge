Cancel

To cancel a shift scheduled for the staff, select [Cancel] from the menu. Cancellation for more than one day cannot be done using this function. The current shift of the staff cannot be cancelled using this function if the time limit to cancel a shift is over as specified in the parameters.

 

The following details are displayed:

  1. Workplace - The workplace to which the staff is allocated is displayed.

  2. Shift - The shifts scheduled to the staff are displayed.

  3. Start Time - The start time and date of the shift is displayed.

  4. End Time - The end time and date of the shift is displayed.

  5. Schedule Date - The scheduled date is displayed.

 

Enter the following:

  1. Allocated Shifts - The shifts scheduled for the staff are displayed.

  2. Select - Check the box to indicate the selection of the required shift.

  3. Reason - Select a valid reason from the list box. The reason is defined in the "Reason" function for Cancel Schedule. This is a mandatory field

  4. Remarks - Enter the remarks if required.

  5. Click [Apply] to save the record. The system prompts you before finalizing the action. Click [OK] to confirm the action.
