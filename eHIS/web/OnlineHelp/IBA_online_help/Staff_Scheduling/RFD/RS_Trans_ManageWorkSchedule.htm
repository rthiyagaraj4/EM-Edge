Enter the following details:

 

  1. Location type - Select the required Location Type from the list. The listed values are defined in "Location Type" function of Application Masters module.

  2. Workplace - Enter the workplace in alphanumeric format of 10 characters maximum length or select the required workplace using [?] function. If the Location Type is selected, then workplaces listed in the [?] function corresponds to the selected location type or all the workplaces created in the "Workplace" function are displayed if the access rights to allocate and reallocate are set for the workplace in the function "Access Rights". This is a mandatory field.

  3. Role Type - Select the role type as 'Practitioner' or 'Other Staff' from the list.

  4. Staff type - Select the Staff Type from the list. The staff type displayed is based on the selected role type. If Role Type is selected as Practitioner, the list displays the Practitioner Type as defined in the Application Masters module. If Role Type is selected as Other Staff, the list displays the Other Staff Type as defined in the Application Masters module.

  5. Position - Select the position from the list. The listed values are defined in the "Position" function of Application Masters module. If the 'Staff Type' is selected then the positions populated in the list corresponds to the selected Staff Type, or the list box displays all the positions defined in "Position" function in Application Masters module for the selected Role Type.

  6. Staff - Select the required staff using [?] function.

  7. From/To - Enter the date range of the work schedule in date (dd/mm/yyyy) format or select the date using the calendar icon. 'From' date corresponds to the start date when the generated work schedule can be managed. The From date should be greater than or equal to the system date. 'To' date corresponds to the end date to manage the generated work. This is a mandatory field.
