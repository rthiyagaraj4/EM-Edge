Enter the following details:

 

  1. Grade Code - Enter the Grade Code in alphanumeric format of 4 characters maximum length. The grade code uniquely identifies a grade. This is a mandatory field.

  2. Grade description - Enter the Description of the Grade in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Explanatory text - Enter the explanation of the grade in alphanumeric format of 4000 characters maximum length.

  4. Level - Select the level of the grade from the list. On a scale of 1-99, 1 indicates the highest level and 99 indicates the lowest level. The level of grade will be considered during swapping staff in "Manage Work Schedule" in transactions based on the parameter set for 'Staff Substitution Basis'. This is a mandatory field.

  5. Consider for scheduling - Select the check box if the grade record should be considered for scheduling. If the check box is cleared, staffs that need not be scheduled in the transaction process are exempted.

  6. FT work hours - Enter the Full Time working hours for the grade in numeric format of 3 characters maximum length. 'FT work hours' indicates full time working duration expected from the grade. This is calculated on scale of 0-1. This is a mandatory field.

  7. Enabled – Select the check box to enable the grade in the transactions. Clear the check box to disable the record.
