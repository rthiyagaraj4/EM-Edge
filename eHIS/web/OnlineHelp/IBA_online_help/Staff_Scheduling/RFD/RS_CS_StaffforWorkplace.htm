Enter the following details:

 

'By Workplace' Tab

  1. Location Type - Select the Location Type from the list. The listed values are defined in 'Location Type' function of Application Masters module.

  2. Workplace - Enter the Workplace in alphanumeric format of 20 characters maximum length or select using [?] function. This is a mandatory field.

  3. Staff Name - Enter the Staff Name in alphanumeric format of 15 characters maximum length or select using [?] function.

  4. Click [Show Staff].

  5. Role Type - The Role Type is displayed as Practitioner or Other Staff based on selection of the 'Staff Name'.

  6. Name - Based on the selected 'Staff Name', the staff names of Practitioners/Other staff are displayed.

  7. Type - The type of Practitioner or Other Staff is displayed.

  8. Effective [From - To] - Enter the Effective period of the assignment of staff to workplace in date (dd/mm/yyyy) format. [From - To] corresponds to the start and end date of the staff assignment to the workplace. This is a mandatory field. 'To' date can be left unattended, if the end date of staff assignment to the workplace is unknown.

  9. Select - Select the check box to indicate applicability of the staff for the selected workplace. To select all the displayed staffs, select the check box provided in the column header.

 

'By Staff' tab

  1. Staff Name - Enter the Staff Name in the alphanumeric format of 15 characters maximum length or select using [?] function. This is a mandatory field. On selecting the required staff, the 'Role Type' (Practitioner/Other Staff) of staff appears next to the 'Staff Name' textbox as read only text.

  2. Location Type - Select the Location Type from the list. The listed values are defined in 'Location Type' function of Application Masters module.

  3. Click [Show Workplace].

  4. Location Type - The selected Location Type defined in the 'Location Type' function is displayed.

  5. Workplace - The Workplace of the selected location type created in the 'Workplace' function is displayed.

  6. Effective [From - To] - Enter the Effective period of the assignment of staff to a workplace in date (dd/mm/yyyy) format. [From - To] corresponds to the start and end date of the staff assignment to the workplace. This is a mandatory field.  The 'To' date can be left unattended, if the end date of staff assignment to the workplace is unknown.

  7. Select - Select the check box to indicate applicability of the workplace for the selected staff. To select all the displayed workplaces, select the check box provided in the column header.
