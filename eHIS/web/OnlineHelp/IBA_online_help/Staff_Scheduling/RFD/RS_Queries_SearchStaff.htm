Enter the following details:

 

  1. Role Type - Select the Role Type as Practitioner or Other Staff from the list. This is a mandatory field.

  2. Practitioner Type - Select the Practitioner Type from the list if the Role type is selected as Practitioner. If the Role Type is selected as Other Staff, this field is disabled.

  3. Primary Specialty - Select the Primary Specialty of the practitioner from the list. The listed values are defined in the 'Practitioner' function of Application Masters module. If the Role Type is selected as Other Staff, this field is disabled.

  4. Other Staff Type - Select the required staff from the list if the Role Type is selected as Other Staff. The listed values are defined in the 'Other Staff' function of Application Masters module. If the Role Type is selected as Practitioner, this field is disabled.

  5. Staff ID - Enter the Staff ID in alphanumeric format of 15 characters maximum length.

  6. Staff Name - Enter the name of the staff in alphanumeric format of 30 characters maximum length.

  7. Gender - Select the gender of the staff from the list. The listed values are Male, Female and Unknown.

  8. Age - Enter the age of the staff in years in numeric format of two characters maximum length.

  9. Empl Status - Select the employment status of the staff from the list. The listed values are Associate, Full Time, Part Time and Visitor.

  10. Facility - Select the Facility from the list. The listed values are defined in the 'Facility' function of System Manager module.

  11. Experience - Enter the experience of the staff in numeric format of two characters maximum length. Select an option as either Months or Years from the list.

  12. Position - Enter the position of the staff in alphanumeric format of 20 characters maximum length or select the position using [?] function. The positions created in the 'Position' function of Application Masters module are displayed.

  13. Skill - Enter the Skill in alphanumeric format of 20 characters maximum length or select using [?] function. The skills created in the 'Skill' function are displayed.

  14. License - Select the License from the list. The listed values are defined in the 'License' function.

  15. Training - Select the Training from the list. The listed values are defined in the 'Training' function.
