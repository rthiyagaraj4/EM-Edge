Access Rights

From this function you can Create Access Rights for the users to schedule activities and Query for existing records.

Creating Access Rights

From the RS Admin Setup, select Access Rights function.

The Access Rights screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the User or select using [?] function. The Users are defined in the System Manager module are displayed. This is a mandatory field.

  3. Enter the workplace or select using [?] function. The workplace records are defined in the Workplace function.

  4. Click [Show Workplaces] to display the workplaces based on the entered criteria. If the workplace is selected, the selected workplace is displayed. If workplace is not selected, all the workplaces are displayed.

  5. Select the check box to give access rights to setup workplace requirement or clear the check box.

  6. Select the check box to give access rights to allocate staff in the workplace or clear the check box.

  7. Select the check box to give access rights to re-allocate (Swap and Transfer) staff in the selected workplace or clear the check box.

  8. Select the check box to give access rights to cancel a schedule in the selected workplace or clear the check box.  

  9. Select the check box to indicate the selection of the activities of the workplace to the user. To select all the displayed workplaces, select the check box provided in the column header.

  10. Click [Apply] to save the Access Rights or [Reset] to clear the entered data.

  11. Click [Query] to search for existing ‘Access Rights’ records.
