Record Mapping Code/Recalculate Charges

From this function you can Record Mapping Code for a patient encounter or Recalculate the charges.

The Record Mapping Code/Recalculate Charges function is used to record the Mapping Code for a patient encounter in the facility. You can recalculate the charges based on the Mapping Code associated to the patient.

Recording Mapping Code or Recalculating Charges

From the Transactions, double-click Record Mapping Code/Recalculate Charges Function.

The Record Mapping Code/Recalculate Charges screen gets displayed.

  1. Select an Episode Type from the list. The system-defined value is Outpatient.

  2. Click [List of Values] to select the Admission or Transfer number. This is a mandatory field. The ward or clinic to which the patient is admitted, transaction date, gender, age, patient's name and ID and the name and ID of the Physician is displayed.

  3. Enter the Patient ID in alphanumeric format of 10 digits maximum length. The patient ID and name of the patient is displayed automatically when you select the Admission or Transfer number. This is a mandatory field.

  4. Click [Search] to query the records. <Related Field Description>

  5. Click [Recalculate Charges] to recalculate the charges. You can view this button only on querying the records.

  6. An Authorization screen is displayed requesting the login details.

    Enter the following details:

    1. User ID - Enter the User ID in alphanumeric format. Authorized Users' who has access to Patient Billing Manager module can recalculate the charges. This is a mandatory field.

    2. Pin No - Enter valid PIN number. This is a mandatory field.

    3. Remarks - Enter the Remarks in alphanumeric format. This is a mandatory field.

    4. Click [OK] to login and recalculate or [Cancel] to cancel the recalculation process.

  7. Enter the details and click [Commit Form] or press [F10] to save the records.
