Record Approvals for Billing Services

From this function you can Record approvals for the requests placed for the Billing Services.                   

The function Record Approvals for Billing Services enables you to approve the request placed for a Billing Service by a patient. The Billing Services associated to the patient record using the function Encounter Specific Policy Definition – Services gets listed. You have to login to the Cash Counter in order to approve or reject the service request placed for a patient.

Record Approvals for Billing Services

From the Transactions, double-click on Record Approvals for Billing Services function.

The Record Approvals for Billing Services screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Authorize] to login and to authorize the services. You can also reject the services using this function. A screen pops-up requesting you to enter the login information. Authorized Users only are permitted to approve or reject the request for services. Click [Close] to exit from the screen.

    Enter the following details to login and to approve the request placed for Billing Services:

    1. User ID - Enter the User ID in alphanumeric format. This is a mandatory field.

    2. Pin No - Enter the Pin Number for the respective User. The Pin Number is recorded while creating the User in the System Manager module. This is a mandatory field.

    3. Remarks - Enter the Remarks in alphanumeric format. This is a mandatory field.

  3. The requests placed for the Billing Services using the function Encounter Specific Policy Definition – Billing Services gets listed. You can place additional requests for Billing Services, by clicking on [Create Record]. <Related Field Description>

  4. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.
