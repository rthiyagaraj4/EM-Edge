Enter the following details to define the Policy Coverage Limit:

 

Range of Items to be Copied/Deleted for Service Coverage Limits

  1. Material Item Coverage Limit Criteria - Select a value from the list. The system defined values are ‘Sale Category’, ‘Material Item’ and ‘Enterprise Material Group’. Select Material Items to select and to associate a set of Material Items to the range of Policies in the facility. Select Sale Category to select and to associate a set of Sale Categories to the range of Policies in the facility. Select Enterprise Material Group to select and to associate a set of Material Groups/Lines to the range of Policies in the facility.

  2. Override Coverage Limits – Select the check box to override the Coverage limits for the Material Items/Enterprise Material Groups/Lines/Sale Category.

  3. Code [From-To] - Click [List of Values] to view and to select a range of Material Items/Enterprise Material Groups/Lines/Sale Category. The selected range of items or material groups are either included or excluded from policy coverage.

  4. Patient Class – Click [List of Values] to select a Patient Class. The selected items are either excluded or included towards policy coverage for the selected Patient Class.

  5. Amt Per Visit – Enter the coverage amount per Visit. The Insurance Company pays the amount specified towards an Outpatient visit for the selected items associated to the Policy Type.

  6. Amt Per Admission - Enter the coverage amount per Admission. The Insurance Company pays the amount specified towards an Inpatient admission for the selected material items associated to the Policy Type.

  7. Per Inpatient Day – Enter the coverage amount for Inpatients per day. The amount set is specific only to the Inpatients admitted in the facility.

  8. No Of Days – Enter the number of days for which the coverage amount is applicable towards the Inpatient stay in the facility.

 

Range of Policies to be Copied/Deleted

  1. Facility – Select a value from the list. The facilities associated to the logged in user get displayed. The range of services/service groups associated to the range of policies is valid for the selected facility.

 

Query Criteria for Policy Types Selection

  1. Policy Types [From–To] - Click [List of Values] to select a range of Policies. You can specify the search criteria to search for the records matching the search criteria. Select either ‘Starts With’, ‘Contains’, ‘Ends With’ from the list. Enter the search criteria in the adjacent text box.

  2. [Display Records] – Click [Display Records] to view the range of Policies between the selected ‘From’ and ‘To’ entries.

A confirmation message is displayed in red on clicking [Display Records] to search for the Policy Type records. The process status also gets displayed.

  1. Cust Group – The Customer Groups associated to the selected range of Policies get listed.

  2. Cust Code – The description for the selected Customer Group gets displayed.

  3. Policy Type Code – The Policy Types between the selected ‘From’ and ‘To’ policies get displayed.

  4. Description - The description for the selected Policy Type gets displayed.

  5. All – Select the check box to associate all the Customer Groups and the Policy Types to the facility. To select individual records, select the check box adjacent to each record.

  6. Display Existing Items – Select the check box to display the Billing Services/Service Groups associated to the selected Policy Type. The Item code along with the description is displayed.
