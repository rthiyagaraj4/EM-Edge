Enter the following to create copy material item exclusions for Policy:

 

  1. Customer Group - Click [List of Values] to select a valid Customer Group. The Customer Groups defined in the Patient Billing Manager or Accounts Receivable module get listed.

  2. Customer - Click [List of Values] to select a valid Customer. The Customer has to be associated to the Customer Group using the function ‘Setup Service Coverage Detail by Policy’. The Customer code is listed, only if it is defined in the Patient Billing Manager or Account Receivable module. The description for the Customer associated gets displayed.

  3. Policy Type – Click [List of Values] to view and to select a Policy Type code. The Policy Type has to be associated to the Customer Group and the Customer using the function ‘Setup Service Coverage Detail by Policy’. The short description for the selected Policy Type code gets displayed.

  4. Type - Select a value from the list. The system defined values are ‘All’, ‘Item’ and ‘Sale Category’. ‘All’ is displayed, by default. Select All to copy the material items and sale categories associated to the Policy Type. Select Item to copy only the items associated to the Policy Type. Select Item to copy only the sale categories associated to the Policy Type.

 

Item Range

  1. From - Click [List of Values] to view and select the items. The short description for the selected item gets displayed. The values in the list differ based on the ‘Type’ selected.

  2. To - Click [List of Values] to view and select the items. The short description for the selected item gets displayed. The code selected has to be greater than the code selected for the ‘From’ field. The values in the list differ based on the ‘Type’ selected.

 

Copy To Policy

  1. Customer Group - Click [List of Values] to select a Customer Group. The Customer Groups defined in the Patient Billing Manager or Accounts Receivable module get listed.

  2. Customer - Click [List of Values] to select a valid Customer. The Customer code is listed, only if it is defined in the Patient Billing Manager or Accounts Receivable module. The Customer has to be associated to the Customer Group using the function ‘Setup Service Coverage Detail by Policy’.

  3. Policy Type – Click [List of Values] to view and select a valid Policy Type code. The Policy Type has to be associated to the Customer Group and the Customer using the function ‘Setup Service Coverage Detail by Policy’. The short description for the selected Policy Type code gets displayed. This is a mandatory field.
