Release Autologous Units

From this function you can Issue the autologous blood units to be used for the patients.

The Release Autologous Units function is used to issue the autologous blood units to be transfused to the patients. You can query for the units reserved for autologous patients between the date ranges or based on a patient ID or the combination of the product code and unit number details are populated.

Releasing Autologous Units

From the Transactions > Stock Operations, double-click the Release Autologous Units function.

The Release Autologous Units screen gets displayed.

  1. Enter the query criteria. <Related Field Description>

  2. Click [Display] or Press [Tab] to view autologous unit details. <Related Field Description>

  3. Select the check box next to the unit number to release the autologous blood unit.

  4. Select the Select All check box to release all the autologous blood units.

  5. Click [Release Blood Units] to issue the autologous blood units for the patients.
