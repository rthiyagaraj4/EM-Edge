Reprint Unit Labels

From this function you can Reprint the unit labels for a selected range of blood units.

The Reprint Unit Labels function allows you to reprint the blood unit labels for a selected range of blood units. You can search for the unit details based on the Unit number or date on which the units are added to the stock.

Reprinting Unit Labels

From the Transactions > Stock Operations, double-click the Reprint Unit Labels function.

The Reprint Unit Labels screen gets displayed.

  1. Enter the query criteria. <Related Field Description>

  2. Click [Show Details] to view the unit details. <Related Field Description>

  3. Select the ‘Print’ check box next to the unit number and enter the number of copies to be printed.

  4. Select the ‘Select All’ check box to select all the units to reprint the unit labels and enter the number of copies to be printed.

  5. Click [Print] to print the unit details.
