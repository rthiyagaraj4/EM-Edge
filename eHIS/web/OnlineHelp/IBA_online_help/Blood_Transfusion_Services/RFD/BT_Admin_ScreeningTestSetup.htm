Enter the following details to setup screening test:

 

  1. Individual Test Code – The sub tests defined under the selected test in ‘Test Codes’  function in Laboratory module are displayed.  

  2. Type of Answer – The types of answers created for the test is displayed. The valid values are ‘List of Values’ and ‘Numeric’.

 

Validation for Numeric

If the type of answer selected for a test code is numeric, the following fields get displayed:

  1. Low Value – Enter the lowest value acceptable for this test in numeric format of 15 digits maximum length.

  2. High Value – Enter the highest value accepted for this test in numeric format of 15 digits maximum length.

  3. Units – Click [List of Values] to select a unit for low and high values. The list displays the values defined in ‘Unit of Measurement’ function of Application Masters module.

  4. Action – Select the action for the test based on the low value and high value. The valid values are ‘Accept’ or ‘Reject’.  

  5. [>>2>>] - Click [>>2>>] to go back to the previous list.

 

If the type of answer selected for a test code is List of values, the following fields get displayed:

  1. List of Values – Select the List of Values from the list. The ‘Result Comments’ recorded for the respective Test Code through the ‘Result Comments’ tab under ‘Comments’ in Laboratory module are listed.

  2. Action – Select the action for the test from the list. The listed values are ‘Reject’ or ‘Accept’.  If you select ‘Accept’ the results of the test conducted in the laboratory are accepted. If you select ‘Reject’, the results of the test conducted in the laboratory are rejected.  

  3. [>>3>>] - Click [>>3>>] to go back to the previous list.
