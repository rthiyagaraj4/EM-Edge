The following details are displayed:

 

Pooled Units

  1. Product Code - The values defined in the Product Codes function with Pooled Product check box selected get displayed. The list displays the products of allowable categories for pooling. The selected units are pooled together under the main product group head.

  2. Description – The Description of the product is displayed.

  3. Unit No – The Unit Number of the selected blood unit is displayed.

  4. Blood Group – The Blood Group of the selected blood unit is displayed.

  5. Rhesus – The Rhesus factor of the selected blood unit is displayed.
