Enter the following details:

 

  1. Product Code - Click [List of Values] to select the product code to which the selected product is to be converted. The listed values are created in the ‘Product Codes’ function.

  2. Description - The description of the selected product is displayed.

  3. Volume - Enter the volume of the product to be converted in numeric format of six digits maximum length.

  4. Unit Number - The unit number of the selected blood unit is displayed.

  5. Product Code - The product code of the selected blood unit is displayed.

  6. Blood Group - The blood group of the selected blood unit is displayed.

  7. Rhesus Code - The rhesus factor of the selected blood unit is displayed.

  8. Expiry Date - The expiry date of the selected blood unit is displayed.

  9. Volume - The volume of the selected blood unit is displayed.

  10. Volume Units - The volume units of the selected blood unit is displayed.
