Enter the following details to create permitted Blood Type Substitutions:

 

  1. Product – Click [List of Values] to select the Product Code. The listed values are created in the Product codes function. This is a mandatory field.

  2. Patient’s Blood Group – Click [List of Values] to select the Blood Group of the patient. The list displays the values defined in the Blood Group function of the Medical Records module.  

  3. Patient’s Rhesus – Click [List of Values] to select the rhesus factor of the patient. The list displays the values as defined in the ‘Rhesus Factor’ function of the Terminology and Coding Management module.  The Rhesus factor is checked for the compatibility with the donor’s blood group. This is a mandatory field.

  4. Donor’s Blood Group – Click [List of Values] to select the Blood Group of the donor. The list displays the values defined in the Blood Group function of the Medical Records module. The Donor’s blood group is checked for the compatibility with the patient’s blood group, in case of organ transplantation.

  5. Donor’s Rhesus – Click [List of Values] to select the Rhesus factor of the donor. The list displays the values defined in the ‘Rhesus Factor’ function of the Terminology and Coding Management module. The Rhesus factor is checked for the compatibility with the patient’s blood group, in case of organ transplantation.

 

Permissible Blood Type Substitutions

  1. Blood Group – Click [List of Values] to select the compatible Blood group for the selected blood group in Patient’s Blood Group and Donor’s Blood Group functions. The list displays the values defined in the Blood Group function of the Medical Records module.

  2. Description – The Description of the selected Blood Group is displayed.

  3. Rhesus – Click [List of Values] to select the compatible Rhesus factor for the selected blood group.

  4. Description – The Description of the selected Rhesus factor is displayed.
