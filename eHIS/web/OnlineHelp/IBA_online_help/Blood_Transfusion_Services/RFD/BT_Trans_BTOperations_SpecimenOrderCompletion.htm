The following details are displayed:

 

Specimen Details

  1. Specimen Number – The Specimen Numbers meeting the query criteria are displayed.

  2. Specimen Regd Date Time – The Specimen Registered Date and Time are displayed. The specimen is recorded in the Registration function in Laboratory module.  

  3. Patient ID – The Patient ID associated to the specimen number is displayed.

  4. Patient Name – The Name of the patient is displayed.

  5. Sex – The Gender of the patient recorded during patient registration is displayed.

  6. Source Code – The Source Code associated to the specimen is displayed.
