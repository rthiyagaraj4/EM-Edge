Enter the following details:

 

  1. Specimen Date [From-To] –  Click the Calendar function  to select the Specimen Date range to search for the specimens to release the blood units. This is a mandatory field.

  2. De-Allocate/De-Reserve – Select a value from the list. The system-defined values are ‘De-Allocate’ and ‘De-Reserve . This is a mandatory field. You can de-allocate an allocated blood unit to normal status. The blood unit can be allocated to any other patients after de-allocation. You can De-reserve a crossmatched blood unit to normal status. The blood unit can be allocated and reserved to any other patients after de-reservation.

  3. Barcode Unit - Product Code – Scan the barcode of the blood unit to release the blood units matching the search criteria. You can enter the Product code or description to query matching results.
