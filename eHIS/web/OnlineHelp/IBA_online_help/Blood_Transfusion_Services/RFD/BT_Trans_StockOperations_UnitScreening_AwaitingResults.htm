The following details are displayed:

 

Awaiting Results

This tab displays the details of the units, which are awaiting the blood group confirmation from the laboratory. The blood units sent to the lab using Units for Screening tabs is displayed here.

  1. Unit Number – The Unit Number whose results are awaited is displayed.

  2. Product Code – The Product Code of the unit whose results are awaited is displayed.

  3. Description – The description of the Product Code whose results are awaited is displayed.

  4. Original Unit Number - The Original Unit Number received from the external source and recorded through the 'Blood Units Entry' function is displayed.

  5. Blood Group – The Blood Group of the unit whose results are awaited is displayed.

  6. Rhesus – The Rhesus factor of the unit whose results are awaited is displayed.

  7. Center Code – The Centre code of the collection Centre from where the blood units are sent for screening is displayed.

 

Test Details

  1. Test – The Tests to be performed for the selected unit are displayed.

  2. Description – The description of the corresponding test is displayed.
