Enter the following details:

 

  1. Ordered Date From - By default the current date gets displayed. The time is displayed as '00.01' as midnight. This field is mandatory.

  2. Ordered Date To - By default the current date and time gets displayed. You can change the required date. This field is mandatory. Ordered Date range can be 30 days only.

  3. Source Type - Select the required source type from the list. The list value includes All, Ward, Nursing unit. By default, the source type is displayed as 'All'.

  4. Source From - This field gets enabled when the Source Type is selected as Ward or Nursing Unit. Select the Source From value from the list.

  5. Source To - This field gets enabled when the Source Type is selected as Ward or Nursing Unit. Select the Source To value from the list.

  6. Patient ID - Select the patient ID from the lookup. If you enter wrongly, the message 'Invalid Patient ID' gets displayed.

  7. Order Catalog Code - Select the required order catalog code from the LOV. Only BT Order Catalogs (Products) are displayed in the LOV and BTS Department test code are not displayed.
