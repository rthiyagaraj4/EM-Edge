Enter the following details to associate result values:

 

Acceptable Values for the Tests

  1. Test – The list displays the selected value in the Tests for Cross Match tab of this function. You can also click [List of Values] to select it. The list displays all the values created in the Tests for Cross Match tab of this function .

  2. Description – The Description of the Test is displayed.

  3. Result Value/Description – Click [List of Values] to select the result value for the test selected in the Test field. Based on the result values selected in the transactions, the test is validated. The list displays the values defined in the Result Comments tab of ‘Test Codes’ function in Laboratory module. On selecting the result value, the corresponding description is displayed.  

  4. Result Indicator – Click [List of Values] to select a value to specify the result indicator for the result value entered. The valid values ‘Compatible’, ‘Least Incompatible’ and ‘Incompatible’ display the result of the test. The result indicators selected are displayed in the Cross Match Tests screen in Allocate/Xmatch/Issue - Blood Units function.
