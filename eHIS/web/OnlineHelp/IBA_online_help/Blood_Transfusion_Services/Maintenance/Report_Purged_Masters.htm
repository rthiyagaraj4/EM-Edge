Report Purged Masters

From this function you can Print the list of errors recorded during master code deletion process.

Report Purged Masters function is used to print the list of errors recorded during master codes deletion. The errors encountered are recorded in appropriate message form. This report is available for viewing and printing. The error messages can be deleted after viewing, by clicking ‘Delete’ in the ‘Purge’ options.

Reporting Purged Masters

From the Maintenance, double-click Report Purged Masters function.

The Report Purged Masters screen gets displayed.

  1. Click [List of Values] to generate report or delete a master code. The code for the respective master code gets displayed.

  2. Select a value from the list to generate report or delete a master code. The system-defined values are ‘Generate Report’ and ‘Delete’. Select Generate Report to display the errors in the form of a report. Select Delete to delete all the error messages for the selected master code.

    A confirmation message ‘Do you want to Confirm?’ is displayed. Click [Yes] to confirm or [No] to cancel.

    After entering the selection criteria, click [Run Process]. This pops-up a screen prompting for the following preferences:

    • Click ‘View’ to view the report on the screen.

    • Click ‘Print’ to print the report on the designated printer.

    • Click ‘File’ to store the output into a file. Depending on the nature of the report, system prompts to select a file format. Choose the appropriate file format from the available list. You can also optionally enter the output file name.

    • After selecting the required preference, click [Accept] to process or [Cancel] to reenter the selection criteria for the report.

  3. Click [Run Process] to generate report or delete the selected master code.
