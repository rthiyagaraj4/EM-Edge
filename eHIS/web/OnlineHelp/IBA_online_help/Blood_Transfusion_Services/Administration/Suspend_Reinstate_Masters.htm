Suspend/Reinstate Masters

From this function you can Suspend or Reinstate a selected master function.

The Suspend/Reinstate Masters function is used to suspend or reinstate valid records defined for a function in the Master menu in Blood Transfusion Services module. The suspended codes do not appear for selection in the transaction functions.

Suspending or Reinstating Masters

From the Administration, double-click Suspend/Reinstate Masters function.

The Suspend/Reinstate Masters screen gets displayed.

  1. Select the master function to be suspended or reinstated.

The legend varies based on the Master Code selected.

  1. Click [List of Values] to select the master code. The list displays the values defined in corresponding masters of Blood Transfusion Services module. On selecting the value, the description is displayed for verification in the adjacent field.

  2. Select a value from the list. The valid values are ‘Suspend’ and ‘Reinstate’. The value ‘Suspend’ is displayed, by default. If the status for the selected code is ‘Suspended’, the reactivation of such codes is to be done by selecting the value ‘Reinstate’.

  3. Click [Commit] to suspend/reinstate a record. A confirmation message gets displayed when you select ‘Suspend’ or ‘Reinstate’ from the list. Click [Yes] to confirm or [No] to abort the transaction.

  4. A confirmation message ‘Operation Completed Successfully’ is displayed on saving the changes.
