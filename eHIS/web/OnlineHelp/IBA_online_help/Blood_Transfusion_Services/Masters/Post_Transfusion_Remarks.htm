Post Transfusion Remarks

From this function you can Define remarks for post transfusion follow up.  

This function allows you to define the remarks to be used for post transfusion follow up.

You cannot create duplicate codes or delete the existing codes created in the Post Transfusion Remarks master. You cannot edit the suspended codes unless it is activated again.

Defining Post Transfusion Remarks

From the Masters menu, double-click the Post Transfusion Remarks function.

The Post Transfusion Remarks screen gets displayed.

  1. Enter the Remarks Code in alphanumeric format of maximum 5 characters length in the text box. This field is mandatory.

  2. Enter the short description of the remarks in alphanumeric format of 15 characters length. This field is mandatory.

  3. Enter the remarks text in alphanumeric format in the Remarks Text field. You can enter maximum of 2000 characters. Special characters are allowed.

  4. Click [Commit] or press [F10] to save the defined post transfusion remarks or click [Clear Record] to clear the record.

  5. Click [Execute Query] or press [F8] to search for the existing records.
