Units Approaching Expiry

From this function you can Query the status of the blood units that are nearing expiry date and Print the details.

The Units Approaching Expiry function enables you to query the units approaching the expiry date between the selected date ranges. The units approaching the expiry dates in the stock are displayed while searching for the units approaching the expiry. The Blood Group, Rhesus factor product code, unit number and expiry date are displayed. You can also print the details of the expiring units and view the unit details.  

Querying the Units Approaching Expiry

From the Queries, double-click Units Approaching Expiry function.

The Units Approaching Expiry screen gets displayed.

  1. Select the date range using the Calendar function.

  2. Select the Product description from the list of values.

  3. Press [Tab] to view the expiring units. The details are displayed based on the query criteria.

  4. Click [Unit Details] to view the details of the selected unit. The Unit Status screen gets displayed.

  5. Click [Print] to print all the records within the selected date range.   You can also view the file in a PDF format and save the file in HTML, Delimited, RTF and PDF formats.
