Send Linen for Mending

From this function you can Record the linen item details sent to Agencies for mending. You can Cancel the mending requests or View the mending requests placed.

Send Linen For Mending function is used to send the linen items to associated agencies for mending. User has to verify the quality of the linen items received from the laundry after cleaning. The linen items can be returned to the laundry, if the quality check fails. You have to select a Source Type and Source from which the linen items are sent for mending. The values selected are defaulted and displayed in the transaction functions.

Sending linen items for mending

From the Linen, select the Send Linen for Mending function.
