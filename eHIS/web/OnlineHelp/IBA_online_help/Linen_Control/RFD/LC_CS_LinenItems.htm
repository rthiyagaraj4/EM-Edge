Enter the following details to create a Linen Item:

 

  1. Linen Item – Enter the Linen Item in alphanumeric format of four characters maximum length. A Linen Item has to be unique and cannot be changed after creation. This is a mandatory field.

  2. Long Description - Enter the long description of the Linen Item in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description of the Linen Item in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Linen Type - Select the Linen Type from the list. The values defined using the ‘Linen Type’ function get listed. This is a mandatory field.

  5. Disposable Item – Select the check box, if the Linen Item is disposable. The disposable items can be used only once and cannot be reused. Clear the check box, if the Linen Item can be reused.

  6. Enabled - Select the check box to enable the linen item in the transaction functions. The check box is selected, by default.

  7. Washing Instructions – Enter the Washing Instructions in alphanumeric format of 2000 characters maximum length. The Washing Instructions are related to the Linen Item and Linen Type.
