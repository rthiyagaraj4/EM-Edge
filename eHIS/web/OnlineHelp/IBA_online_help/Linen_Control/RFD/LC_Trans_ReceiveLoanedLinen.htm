Enter the following details:

 

  1. Source Type – The selected Source Type is displayed and you cannot edit this field. This is a mandatory field.

  2. Source – The selected Source is displayed and you cannot edit this field. This is a mandatory field.

  3. Loaned To – Select the Source Type from the list to which the linen item is loaned. The system-defined values are Nursing Unit, Clinic, Department and Patient. Nursing Unit is displayed, by default.

  4. Source - Select the Source using [?] function. If the Source Type selected is ‘Nursing Unit’, Nursing Units defined in Inpatient Management module get listed. If the Source Type selected is ‘Clinic’, Clinics defined in Outpatient Management module get listed. If the Source Type selected is ‘Department’, Departments defined in System Configuration module get listed.

  5. Loan Date [From] – System calculates and displays the time and date seven days prior to the system date. Click the Calendar function to select the date when the linen items are loaned to a Nursing Unit, Clinic, Department or patient. This indicates the date when the request for linen item is loaned. This is a mandatory field.

  6. To - Click the Calendar function to select the Date. The current System Date and time are displayed as the ‘To’ Request Date. The requests placed between the date range selected are listed.

  7. Given By - The name of the logged-in user gets displayed as the Given By name, by default. Enter the name of the person in alphabetical  format of 30 characters maximum length. This indicates the name of the person who has loaned the linen items to the selected Nursing Unit, Clinic or Department.

  8. Received By – The name of the logged-in user gets displayed as the Received By name, by default. Enter the name of the person in alphabetical  format of 30 characters maximum length. This indicates the name of the person who has received the loaned linen items from the selected Nursing Unit, Clinic or Department.

  9. Reason for Loan – Select the Reason recorded while loaning the linen items. The ‘Reasons’ defined using the ‘Reason/Complaint Codes’ function with the ‘cancel Reason’ as selected are listed.

  10. Request Status - ‘All Pending Requests’ is displayed, by default. Select a value from the list. The system-defined values are ‘All’, ‘All Pending Requests’, ‘Outstanding’, ‘Partially Issued’, ‘Issued’ and ‘Cancelled’.

  11. All – Select ‘All’ to view all the requests placed.  The requests, which are outstanding, partially issued, issued and cancelled, are listed.

  12. All Pending Requests – Select ‘All Pending Requests’ to view the list of pending requests. The requests, which are currently pending are displayed.

  13. Outstanding – Select ‘Outstanding’ to view the list of outstanding requests. The requests, which are currently outstanding, are displayed.

  14. Partially Issued – Select ‘Partially Issued’ to display the list of requests for which the linen items are partially issued by the laundry.

  15. Issued – Select ‘Issued’ to display the list of requests for which the linen items are issued by the laundry.

  16. Cancelled – Select ‘Cancelled’ to display the list of requests for which the linen item requests are cancelled.  

  1. Linen Item – Select the Linen Item using [?] function. Linen items for which standard stock is defined in ‘Standard Stock for Sources’ function gets listed.

  2. [Submit] - Click [Submit] to acknowledge the quantity of loaned linen received back from the Nursing Unit/Clinic/Department/Patient.
