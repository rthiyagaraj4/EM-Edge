Cleaning Requests List

You can view the Cleaning Requests and cancel the requests placed for cleaning.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to view and modify the Cleaning Requests List placed from a selected Source Type. The Requests placed are displayed in an ascending order based on the  Request Number.

  3. The Request Number is displayed as a hyperlink. Click Request Number to ‘Cancel’ or ‘View’ the request placed for cleaning linen.

  4. Select ‘Cancel’ hyperlink to cancel the request placed for cleaning linen. Request Number, Request Date, Source Type, Source, Linen Handed Over By, Linen Collected By and Request Status are displayed. The linen item requested, requested and delivered quantity is also displayed along with the remarks recorded.

    Enter the following details to cancel:

    1. Cancelled By – The User name is displayed, by default. You cannot edit this field.

    2. Cancelled Date/Time – The system date and time is displayed. You cannot edit the Cancelled Date and Time.  

    3. Cancelled Reason – Click [?] function to select a reason for cancellation. Reasons defined using the function ‘Reason/Complaint Codes’ with the Action Type as ‘Cancel Reason’ get listed for selection.

    4. [Cancel] - Click [Cancel] to cancel the request placed for the cleaning linen.

  5. Select ‘View’ hyperlink to view the details of the request placed for cleaning linen. You can view the Request Number, Request Date, Source Type, Source, name of the users who have handed over the linen to laundry and who has collected the cleaned linen is displayed along with the remarks.
