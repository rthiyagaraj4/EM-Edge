Loaned Linen List

This tab is used to view or cancel the loaned linen. You can Cancel or View the loaned linen item details.

Click ‘View’ hyperlink to view the details of the request placed for loaned linen. You can view details, such as, Loan Reference Number, Loan Date, Source Type, Source, Loan To, Source, Given By, Received By, Reason For Loan, Remark and Request Status. The Linen Item, loaned quantity and received quantity is displayed in a tabular format.

Cancel

The linen requests placed using the ‘Loan Linen’ function are displayed based on the search criteria. You can either cancel or view the request placed for loaned linen items.

  1. Click [Search] to view and modify all the Loaned Linen Requests List placed from the selected Source Type. The Requests placed are displayed in an ascending order of the Request Number.

  2. Click ‘Loan Reference Number’ displayed as a hyperlink to ‘Cancel’ or ‘View’ the request placed for loaned linen.

  3. Click ‘Cancel’ hyperlink to cancel the request placed for loaned linen. Loan Reference Number, Loan Date, Source Type, Source, Loan To, Source, Given By, Received By, Reason for Loan, Remark and Request Status are displayed based on the details recorded using the ‘Loaned Linen’ tab. The Linen Item, Loaned Quantity and Received Quantity are displayed.

    Enter the following details to cancel:

    1. Cancelled By – The User name is displayed, by default. You cannot modify this field.

    2. Cancelled Date/Time – The system date and time is displayed. You can edit the Cancelled Date and Time.  

    3. Cancelled Reason – Select the reason for cancelling the loaned request placed for new linen using [?] function. Reasons associated with the Action Type ‘Cancel Reason’ in ‘Reason/Complaint Codes’ function get listed for selection.

 

Enter the following details:

  1. Source Type – The selected Source Type is displayed and you cannot edit this field. This is a mandatory field.

  2. Source – The selected Source is displayed and you cannot edit this field. This is a mandatory field.

  3. Loan To – Select a Source Type from the list to which the linen item is loaned. The system-defined values are Nursing Unit, Clinic, Department and Patient. Nursing Unit is displayed, by default.

  4. Source - Select the Source using [?] function. If the Source Type selected is ‘Nursing Unit’, Nursing Units defined in Inpatient Management module get listed. If the Source Type selected is ‘Clinic’, Clinics defined in Outpatient Management module get listed. If the Source Type selected is ‘Department’, Departments defined in Application Masters module get listed.

  5. Loan Date (From) - Click the Calendar function to select the Loan Date. The items loaned from the date selected get displayed. The system calculates and displays the time and date seven days prior to the system date as the ‘From’ Date, by default. This is a mandatory field

  6. Loan Date (To) - Click the Calendar function to select the Loan Date. The current System Date and time is displayed as the ‘To’ Request Date. The requests placed until the date selected are listed.

  7. Given By - The User Name gets displayed as the Given By name, by default. You can update the name of the person who has loaned the linen items. Enter the name in alphanumeric format of 30 characters maximum length. The linen items loaned by the person from the selected Nursing Unit, Clinic or Department to the other Source are listed.

  8. Received By – The name of the logged-in User is displayed, by default. You can modify the name of the person who has received the linen items. Enter the name of the person in alphanumeric format of 30 characters maximum length. The linen items collected by the person are displayed.

  9. Reason for Loan – Select the Reason for loaning the linen from the list. The details of the linen items given by selecting the specific reason are displayed.

  10. Request Status - ‘All Pending Requests’ is displayed, by default. Select a value from the list. The system-defined values are ‘All’, ‘All Pending Requests’, ‘Outstanding’, ‘Partially Issued’, ‘Issued’ and ‘Cancelled’.

  11. All – Select ‘All’ to view all the requests placed. The requests, which are Outstanding, Partially Issued, Issued and Cancelled, are listed.

  12. All Pending Requests – Select ‘All Pending Requests’ to view the list of pending requests. The requests, which are currently pending are displayed.

  13. Outstanding – Select ‘Outstanding’ to view the list of outstanding requests. The requests, which are currently outstanding, are displayed.

  14. Partially Issued – Select ‘Partially Issued’ to display the list of requests for which the linen items are partially issued by the laundry.

  15. Issued – Select ‘Issued’ to display the list of requests for which the linen items are issued by the laundry.

  16. Cancelled – Select ‘Cancelled’ to display the list of requests for which the linen item requests are cancelled.

  1. Linen Item – Select the Linen Item using [?] function. The linen items that are loaned get displayed based on the search criteria specified.
