Enter the following details:

 

  1. Received By – Enter the name of the person responsible for receiving the linen after mending against the Request placed for mending linen.

  2. Linen Item – Enter the linen item. The linen item defined in the Mending Request gets displayed, by default.

  3. Sent Qty – The number of linen items sent for mending get displayed, by default. You cannot modify it.

  4. Received Quantity – If any remarks are recorded against the selected Request Number for linen sent to mending, respective quantity gets displayed, by default.

  5. Cur Received Qty – Enter the quantity of linen currently received against the selected Request Number given for mending.

  6. Remark – Click Remark hyperlink to record remarks, if any. Click [OK] to save the record and exit the window.
