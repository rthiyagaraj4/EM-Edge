Enter the following details:

 

  1. Loan Ref. Num – The Loan Reference Number is system generated and incremented for each request placed to record the linen loaned. You cannot change the field. This is a mandatory field.

  2. Loan Date – The system date and time get displayed, by default. The time is displayed in hh:mm format. The Loan Date indicates the date when the request is placed to the loan a linen item from a Nursing Unit/Clinic/Department/Patient. You cannot change the field. This is a mandatory field.

  3. Source Type – The selected Source Type is displayed and you cannot edit this field. The source type indicates the location type from which the request is placed. This is a mandatory field.

  4. Source – The selected Source is displayed and you cannot edit this field. The source type indicates the location from which the request is placed. This is a mandatory field.

  5. Loan To - Select the Source Type from the list to which the linen is to be loaned. The system-defined values are Nursing Unit, Clinic, Department and Patient. Nursing Unit is displayed, by default. This is a mandatory field.

  6. Source - Select the Source using [?] function. If the Source Type selected is ‘Nursing Unit’, Nursing Units defined in Inpatient Management module get listed. If the Source Type selected is ‘Clinic’, Clinics defined in Outpatient Management module get listed. If the Source Type selected is ‘Department’, Departments defined in Application Masters module get listed. This is a mandatory field.

  7. Given By - Enter the name of the person in alphabetical format of 30 characters maximum length. This indicates the name of the person who is responsible for lending linen items from a selected Nursing Unit, Clinic or Department to the another Source. The User Name gets displayed, by default. You can update the name of the person who has loaned the linen items. This is a mandatory field.

  8. Received By – Enter the name of the person in alphabetical format of 30 characters maximum length. This indicates the name of the person who is responsible for collecting the loaned linen items on behalf of the receiving source or patient name. The User Name gets displayed as the Received By name, by default. You can update the name of the person who is responsible for collecting the linen items. This is a mandatory field.

  9. Reason for Loan – Select a Reason from the list to indicate the reason for lending a linen item. The reasons defined using the ‘Reason/Complaint Codes’ function with the Action Types as ‘Cancel Reason’ get listed. This is a mandatory field.

  10. Linen Item – Select the Linen Item using [?] function. Linen items for which standard stock is defined in ‘Standard Stock for Sources’ function get listed. This is a mandatory field.

  11. Loaned Qty – Enter the loaned quantity in numeric format of eight digits maximum length.

  12. Remark – Enter the remarks in text area in a maximum of 100 characters maximum length. Click [OK] to save and return to the main screen.

  13. [Submit] - Click [Submit] to save the linen loaned.
