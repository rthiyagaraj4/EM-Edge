Place Clean Linen Request

This tab is used to place an order for cleaning linen items used in the facility.

 

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Select] to display a Linen Request placed for cleaning the linen.

• If ‘Allow linen more than std. stock qty.’ value is selected as ‘Allow’, the system allows you to place request for linen items more than the standard stock defined in ‘Standard Stock for Sources’ function in Admin Setup for the selected location.

• If ‘Allow linen more than std. stock qty.’ value is selected as ‘Stop’, the system does not allow you to place request for linen items more than the standard stock defined in ‘Standard Stock for Sources’ function in Admin Setup for the selected location. It displays an error message ‘Requested Quantity should be less than or equal to Std. Stock’.

• If ‘Allow linen more than std. stock qty.’ value is selected as ‘Alert’, an alert message ‘The requested quantity exceeds the standard stock for the selected location. Do you want to continue?’ is displayed. Place request for linen items more than the standard stock defined in ‘Standard Stock for Sources’ function in Admin Setup for the selected location. It displays an error message ‘Requested Quantity should be less than or equal to Std. Stock’. Click [OK] to proceed with placing the request for linen items or [Cancel] to exit without placing an order.

  1. The selected Linen Item is displayed along with the requested quantity, category, schedule delivery date and remarks. Linen Item and Remark is displayed as a hyperlink. On clicking the Linen Item hyperlink the selected Linen Item, Category and its requested quantity is displayed in the respective fields for editing. Click Remark hyperlink to view the remarks recorded. You can update the remarks by clicking ‘Remarks’, which is displayed as a hyperlink. Click [OK] to record the details. Click [Cancel] to cancel the request placed for the linen item.

  2. Click [Submit] to save the cleaning request placed for a Source Type. A confirmation message ‘Please note your Request Number’ is displayed on placing the cleaning request. Click [OK] and the control shifts back to ‘Place Clean Linen Request’ tab, where you can place the next request.
