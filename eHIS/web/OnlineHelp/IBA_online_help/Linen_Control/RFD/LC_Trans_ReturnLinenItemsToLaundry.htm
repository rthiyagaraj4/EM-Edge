Enter the following details:

 

  1. Source Type – The selected Source Type is displayed and you cannot edit this field. This indicates the Source Type from where the linen items are returned to laundry.

  2. Source - The selected Source is displayed and you cannot edit this field. This indicates the Source from where the linen items are returned to laundry. This is a mandatory field.

  3. Ref. Num - The Reference Number is system–generated. The number is incremented for every new request. You cannot modify the reference number. This is a mandatory field.

  4. Date – The system date and time is displayed, by default. Click the Calendar function to select a date. This indicates the date when the linen items are returned to the laundry by the respective Source. This is a mandatory field.

  5. Returned By - Enter the name of the person who is responsible to return the linen items to laundry from the Nursing Units/Clinics/Departments. This is a mandatory field.

  6. Received By - Enter the name of the person in alphabetical  format of 30 characters maximum length. This indicates the name of the person who has received the linen items in the laundry from Nursing Units/Clinics/Departments. This is a mandatory field.

  7. Linen Item – Select the Linen Item using [?] function. Linen items for which standard stock is defined in ‘Standard Stock for Sources’ function get listed. This is a mandatory field.

  8. Returned Quantity – Enter the returned quantity in numeric format of eight digits maximum length.

  9. Remark – Enter the relevant remarks in the text area. Click [OK] to save and return to the main screen.
