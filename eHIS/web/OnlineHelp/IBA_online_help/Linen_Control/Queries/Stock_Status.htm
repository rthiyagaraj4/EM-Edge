Stock Status

From this function you can Query for the status of the linen stock in each Nursing Unit, Clinic or Department. You can also View the Stock Status and the Work Status of a Linen item.

This function is used to view the Stock Status by Source or by Linen Item. On selecting the option as ‘By Source’, the linen items pertaining to the selected Nursing Unit, Clinic or Department are displayed with the total quantity classified into the linen category ‘In-Use’, ‘Soiled’ or ‘Loaned’. The total of these categorized types of linen is equal to the Total Quantity of the Linen Item the Nursing Unit or Clinic or Departments hold. The Linen Items pertaining to the selected Nursing Unit, Clinic or Department are displayed.

On selecting the option as ‘By Linen Item’, you can view the stock status of each linen item in the Nursing Units, Clinics and Departments in the facility. The quantity of the selected linen item is displayed with the total quantity categorizing it as With Laundry, Issued Out, In Cleaning and In Mending. The total quantity of the selected linen item in each Nursing Unit or Clinic or Department is displayed and categorized as ‘In-Use’, ‘Soiled’ or ‘Loaned’. The total of these categorized items must match the total quantity in all the Nursing Units, Clinics or Departments. This total in turn must match the total quantity of linen items in the facility.

Querying Stock Status

From the Queries, select the Stock Status function.

The Stock Status screen gets displayed.

  1. You can query the Stock Status of linen items in facility based on 'Source' or 'Linen Item'.

  2. Click [?] to select a Source or a Linen Item.

    Enter the following details:    

    1. By Linen Item - Select the Linen Item using [?] function. Linen items for which standard stock is defined in ‘Standard Stock for Sources’ function get listed.

    The stock of each linen item across all the Nursing Units/Clinics/Departments gets displayed on clicking [Search]. The stock of the selected linen item with laundry, issued outside the Nursing Unit, in cleaning or in mending is displayed with its cumulative total. Details as name of the Source, total number of linen item, the number of linen items in use, soiled or loaned within each Nursing Unit/Clinic/Department get displayed. The figure in the total column indicates the sum of the linen items in use, soiled or loaned.

    Enter the following details:    

    1. Source - Select the Source Type from the list. The system-defined values are Nursing Unit, Clinic and Department. Select the Source using [?] function. If the Source Type selected is ‘Nursing Unit’, Nursing Units defined in Inpatient Management module gets listed. If the Source Type selected is ‘Clinic’, Clinics defined in Ambulatory care Management module gets listed. If the Source Type selected is ‘Department’, Departments defined in Application Masters module gets listed. This is a mandatory field.

    The stock status of linen items in a particular Nursing Unit/Clinic/Department gets displayed on clicking [Search]. Details such as name of the Linen Item, total number of linen item, the number of linen items in use, soiled or loaned get displayed. The figure in the total column indicates the sum of the linen items in use, soiled or loaned.

  3. Click [Search] to view the details based on the search criteria specified.

  4. Click [Query] to select the query criteria either ‘By Source’ or ‘By Linen Item’.

  5. Click [Reset] to clear the details.
