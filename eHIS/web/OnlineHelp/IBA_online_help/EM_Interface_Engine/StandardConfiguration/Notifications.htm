Notification

From this function you can Create/Map the Application ID and Event Type for Track Notification screen.

This function allows to create and edit mapping of 'Application ID', 'Event type' with result columns to be displayed in 'Track Notification' screen.

Creating/Mapping Notifications

From Standard Configuration, select Notification Function.

The Notification screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Application ID – Select the required application ID (laboratory) from the drop-down list. This field is mandatory. By default, the field is blank.

  3. Notification Event Type – Select the required notification event (critical lab result) from the drop-down list. This field is mandatory. By default, the field is blank.

  4. After selecting the Application ID and Notification Event Type field values, the Notification details check boxes are displayed. Select the required check boxes and it will be displayed as columns in the 'Track Notification' screen.
