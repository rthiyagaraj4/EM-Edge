Enter the following details:

 

  1. FS Location – Select the FS Location from the list. The FS Location set in FM Parameter to hold incomplete files is displayed, by default. This field is not displayed, if the ‘Enable Monitor File Deficiency when Incomplete FS Location not defined’ check box in FM Parameter is selected.

  2. Patient ID - Enter the Patient ID in alphanumeric format of length as specified in MP Parameter or use [?] function to select the required patient. This field is mandatory.

  3. File Type - On selecting the Patient ID, the File Type(s) associated to the patient is displayed. If more than one File Type is associated, select the required File Type from the list. On selecting the Patient ID, the default File Type associated in the File Type For FS Location is displayed. This field is mandatory.

  4. Volume No - The active volume number is displayed. Select the required Volume Number from the list.

  5. Encounter ID - Enter the Encounter ID in alphanumeric format of 12 characters maximum length or use [?] function to select for the required Encounter ID. This field is mandatory.

The system issues the file or performs transactions even if the monitor file deficiency status is ‘open’.
