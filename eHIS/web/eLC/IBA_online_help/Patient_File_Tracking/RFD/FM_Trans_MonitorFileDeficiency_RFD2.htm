Enter the deficiency details:

 

  1. Deficiency Type - Select the Deficiency Type from the list. The list values are defined in the function ‘Deficiency’. The Deficiency Type value cannot be changed while Closing/Canceling the deficiency. This field is mandatory.

  2. Responsible Practitioner - Select the Practitioner responsible for the file deficiency using [?] function.  The value cannot be changed while Closing/Canceling the deficiency. This field is mandatory.

  3. Date/Time Reported - Enter the Reporting Date/Time of the discrepancy in the Date format of dd/mm/yyyy hh:mm. The current date is displayed, by default. The entered date must not be greater than the System Date and lesser than Discharge Date. The reporting Date/Time cannot be changed while Closing/Canceling the deficiency.

  4. Deficiency Status - Select the status as Open, Close or Cancel. ‘Open’ indicates the deficiency is unresolved, ‘Closed’ indicates that the deficiency is resolved and ‘Cancel’ indicates the deficiency is cancelled.

  5. Only the Open status is enabled while recording the deficiency. The other status values are disabled.

  6. On Closing/Canceling a deficiency, the open status is selected and disabled. The Close and Cancel status are available for selection.

  1. Date/Time Resolved/Cancelled - Enter the Resolving/Canceling Date and Time in the format of dd/mm/yyyy hh:mm.

  2. Date/Time Resolved/Canceled field is not available while recording the deficiency.

  3. Date/Time Resolved/Canceled field is available only when closing/canceling the deficiency.

  1. Remarks - Enter Remarks in alphanumeric format of 60 characters maximum length.
