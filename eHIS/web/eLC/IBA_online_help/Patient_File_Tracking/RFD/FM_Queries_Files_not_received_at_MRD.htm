Enter the following search criteria:

 

  1. FS Location - Select the FS Location from the list . The list contains all the permanent File storage locations of the logged in facility. The value selected specifies the permanent FS Location to which files are due. This is a mandatory filed.

  2. Outstanding From - Select the facility from the list. The list provides the names of all the facilities of the enterprise.

  3. Outstanding FS Location [From - To] - Select the FS Location for the selected facility from the list provided. The list is populated with values that correspond to the selected Facility. Select the required locations. The query result is restricted to the selected location range.
