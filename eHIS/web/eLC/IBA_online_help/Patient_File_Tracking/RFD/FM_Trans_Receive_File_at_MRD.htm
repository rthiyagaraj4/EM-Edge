Enter the following details to receive files at the medical record department:

Received Details

  1. Receiving FS Location - Select the permanent file storage location from the list. If ‘Security access for users is Enforced’ check box in FM Parameter is selected, the list contains values that are associated to the logged-in user. If security is not applicable, all valid permanent file storage locations of the logged in facility are displayed as list values. This is a mandatory field.

  2. Received By - The logged-in user name is displayed as default value. If ‘Security access for users is enforced’ check box in FM Parameter is selected, the logged-in user name is displayed and disabled or the user name can be selected using [?] function.

  3. Narration - Select the Narration from the list. The narration set during FM parameter setup is displayed, by default. The narrations are defined in the Narration function.

  4. File Status - Select the File Status from the list. The list of received files are sorted by the received order. If the ‘Accept Outside & In Transit Files Directly in Receive File at MRD’ check box in FM Parameter is selected, you can select any values from the list. The status values listed are Returned, Lost, Archived, Outside, and In Transit. If the 'Accept Outside & In Transit Files Directly in Receive File at MRD’ check box in FM Parameter function is cleared, the File status is displayed as ‘Returned’ and displays only the returned files.  

 

Outstanding From

  1. Facility - Select the facility from the list. All the facilities of the enterprise are listed. The logged in facility is displayed, by default.

  2. FS Location Identity - Select the FS Location Identity from the list.

  3. FS Location - Two values are to be specified for FS location. The Identity value and actual FS location. Select the Identity from the list. The listed values are Clinic, Nursing unit, Practitioner, Department, Day care Unit and Procedure Unit. Next, select the actual FS location from the list. The listed values correspond to the selected Facility and Identity.

 

Criteria

  1. Patient ID - Enter the valid Patient ID in alphanumeric format of length as specified in Master Patient Index module or select the Patient ID using [?] function. This is a mandatory field.

  2. File Type /File No. - On selecting the Patient ID, the File Number or the File Type associated to the patient is displayed. If more than one file type is associated, select the required type from the list. On selecting the Patient ID, the default File Type associated in the File Type For FS Location is displayed. This is a mandatory field.

  3. Volume No - The active volume number is displayed, by default. Select the required volume number from the list.
