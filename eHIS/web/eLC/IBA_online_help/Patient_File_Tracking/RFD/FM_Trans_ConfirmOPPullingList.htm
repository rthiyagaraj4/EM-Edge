Enter the following details to confirm the files for auto-tracking:

 

  1. FS Location - Select the permanent FS Location from the list. The listed Locations are populated based on the access rights of the logged in User. This field is mandatory.  

  2. Appt Date - The Appointment Date is displayed based on the FM Parameter setup. The displayed date can be changed to the required date. Enter the future Appointment Date in the date format dd/mm/yyyy or use the Calendar function to select the required date. This field is mandatory.

  3. Location Type - Select the Location Type from the list. The listed values are Clinics, Day Care and Procedure Unit. This field is mandatory.

  4. Location - Select the Location using [?] function or enter the Location in the required format. The specified Location corresponds to the selected Location Type. This field is mandatory.

  5. Resource Class - Select the Resource Class from the list. The listed values are Practitioner, Room, Equipment and Others.

  6. Resource Type - Select the Resource Type range from the list. The listed values correspond to the selected Resource Class.

  7. Resource ID [From – To] - Enter the Resource ID range in alphanumeric format of 15 characters maximum length. The specified values must correspond to the selected Resource Type.

  8. Display - Select the Display status from the list. The values are pre-defined. The values displayed are Not Confirmed, Confirmed, Appt. Cancelled, Schedule Cancelled, Appt. Transferred and File Outside MRD.

  9. Pulling List Type - Select the Pulling List Type from the list. The values displayed are Both, Main and Supplementary.

  10. Gender - Select the Gender from the list. The values displayed are Male, Female, Unknown and All. The value ‘All’ is displayed, by default.

  11. Generated By  - The name of the person confirming files for tracking is specified. If the FM Parameter is applicable (Security to be enforced), only the logged in User Name is displayed as read-only text. If Security to be enforced is cleared in FM Parameter, you can select the Users belonging to the facility.  

  12. Patient ID - Enter the Patient ID in alphanumeric format of length as specified in MP Parameter or use [?] function to select the required patient. This field is mandatory.

  13. File Type - On selecting the Patient ID, the File Type(s) associated to the patient is displayed. If more than one File Type is associated, select the required File Type from the list. On selecting the Patient ID, the default File Type associated in the File Type For FS Location is displayed. This field is mandatory.  

  14. Volume No  - The active volume number is displayed, by default. Select the Volume Number from the list.
