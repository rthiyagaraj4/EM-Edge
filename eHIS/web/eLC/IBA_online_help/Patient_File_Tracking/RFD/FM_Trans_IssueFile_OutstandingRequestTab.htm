Outstanding Request

Files that are requested from various File Storage Locations either within the facility or from other facilities are displayed. Files requested from various File Storage Locations are grouped under 'From Logged in Facility' (Requesting FS Locations within the facility) and 'From Other Facilities' (permanent FS Location of other facilities). The outstanding requests are arranged in the alphabetical order. If the logged in user does not have access to issue files to other facility, 'From other facilities' is not displayed, even though the request for files from other facilities are outstanding.

  1. Enter the details in the required fields. <Related Field Description>

  2. Click [Search] to view a list of Requested Patient Files or [Cancel] to reset the records displayed in the result area.

  3. On searching, the details of the Requested Files are displayed based on the entered criteria.

  4. Click ‘+’ hyperlink to view the Patient Details.

  5. Click ‘View Remarks’ hyperlink to view the Request Remarks. Click [OK] to exit from the Request Remarks screen.

  6. Click ‘Enter Remarks’ hyperlink to enter the Issue Remarks. On entering the Issue Remarks, click [OK] to save and exit from the Issue Remarks screen. The issue remarks cannot be entered for the files with status as Lost, Archived, Returned and In transit. The Remarks is displayed as read-only text.  

  7. Select the ‘Issue?’ check box to select the requested files for Issue. The check box is selected by default, based on the File Status.

    For file status ‘Inside’ the check box is selected, by default. Clear the check box not to issue the file.

    For file status ‘In transit’, ‘Lost’, ‘Archived’ or ‘Returned’, the check box is disabled. These files are not considered for issue and you cannot select the check box.

 

Medico Legal Case

If an encounter is marked as Medico-Legal case in Medical Records module, the files attached to the encounter are indicated using the Medico Legal Case icon. This blinking icon is displayed under the ‘Issue?’ column.

  1. Click [Add for Issue] on selecting the files for issue.

Repeat the process to add files to issue to other requesting FS Locations. Files once selected cannot be cleared on clicking [Add for Issue]. Click [Issue File] tab to view the files considered for issue.
