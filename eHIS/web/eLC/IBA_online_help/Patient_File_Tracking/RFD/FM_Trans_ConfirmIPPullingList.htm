Enter the following details to confirm the files for auto-tracking:

 

  1. FS Location - Select the permanent FS Location from the list. The listed Locations are based on the access rights of the logged in User. This field is mandatory.

  2. Booking Date -  The Booking Date is displayed based on the FM Parameter. The displayed Date can be changed to the required Date. Enter the future Booking Date in the format dd/mm/yyyy or use the Calendar function to select the required Date. This field is mandatory.

  3. Nursing Unit - Select the Nursing Unit from the list. The Nursing Units defined in the Inpatient and Day Care Management module are displayed. This field is mandatory.

  4. Display - Select the Display status from the list. These values are pre-defined. The values displayed are Not Confirmed, Confirmed, Cancelled Booking, Revise Booking and File Outside MRD.

  5. Practitioner ID [From – To] - Enter the Practitioner ID range in alphanumeric format of length as defined in Master Patient Index module or select the ID using [?] function.

  6. Pulling List Type - Select the Pulling List Type from the list. The values displayed are Both, Main and Supplementary.

  7. Generated By - The name of the person confirming files for tracking is specified. If the FM Parameter is applicable (Security to be enforced), only the logged in User Name is displayed as read-only text. If ‘Security to be Enforced’ check box is cleared in FM Parameter, the Users belonging to the facility can be selected.

  8. Gender - Select the Gender from the list. The values displayed are Male, Female, Unknown and All.

  9. Patient ID - Enter the Patient ID in alphanumeric format of length as specified in MP Parameter or use [?] function to select the required patient. This field is mandatory.

  10. File Type - On selecting the Patient ID, the File Type(s) associated to the patient is displayed. If more than one File Type is associated, select the required File Type from the list. On selecting the Patient ID, the default File Type associated in the File Type For FS Location is displayed. This field is mandatory.  

  11. Volume No - Select the Volume Number from the list.
