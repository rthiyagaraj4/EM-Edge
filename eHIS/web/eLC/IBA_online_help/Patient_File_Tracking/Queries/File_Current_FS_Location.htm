File Current FS Location

From this function you can Query and view the current storage location of the file.

The query result lists File Type(s) and Volume(s) for the specified Patient ID. You can select the required File Type/Volume and view the current and previous file status details. The function provides Outpatient Appointment details and Inpatient Booking details (if any) for the selected patient.

Querying for File Current FS Location

From FM Queries, select the File Current FS Location function.

The File Current FS Location screen gets displayed

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to select the records that satisfy the query criteria or [Reset] to clear the entered search criteria.

  3. On searching, the File Type(s) associated with the selected Patient ID, the volume number currently active for the selected Patient ID and the Volume number that is inactive for the File Type are displayed.

  4. Click the ‘Active Volume No’ hyperlink to view the current FS Location of an Active Volume Number.

  5. Select the Inactive Volume number from the list (if any). On selecting a number, the current FS Location is displayed.

  6. Click [Clear] to clear the result screen.
