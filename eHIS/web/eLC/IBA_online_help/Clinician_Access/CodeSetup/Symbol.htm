Symbol

From this function you can Create and maintain the Symbols to be used within text editors and also Query for a desired Symbol.

Symbols are special text characters used within text editors in the system; for example, £, ¥, €, and so on. Symbols are part of a higher level, symbol group. Both symbols and symbol groups are associated with note types and note groups and are used in their respective text editors.

Creating a Symbol

From the CA Code Setup, select Note Type function.

The Note Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Note Type created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Note Type.
