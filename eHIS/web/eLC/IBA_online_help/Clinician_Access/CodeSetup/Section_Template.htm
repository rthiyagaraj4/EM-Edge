Section Template

From this function you can Create a new Section Template for Template Based Sections.

This function is used to build a section template using the template components created in the Discrete Measure Setup of Application Masters module. The Section Templates created are used for recording details on selecting the Template based Sections in the Clinical Note transaction.

Creating a Section Template

From the CA Code Setup, select Section Template function.

The Section Template screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Select [Preview Template] to view the template created at any time. Click [Close] to close the preview screen.

  4. Click [Apply] to save the Section Template or [Reset] to clear the entered data.

  5. On saving the record, the details like Item Type, Title Value, Component and Row/Column/Span/Location are displayed on the top part of the screen.

  6. Click the Item Type hyperlink to modify the details. On modifying any details, click [Modify] to record the modification.

  7. Click Row/Column/Span/Location hyperlink to change the position of the component and the Shift Rows screen is displayed. Enter the number from 1-9 by which the Row has to be shifted and click [Forward] to shift the component ahead by the number of rows entered in the field, [Backward] to shift the component backward and [Cancel] to exit from the screen.

  8. Select [Preview Template] to view the template modified. Click [Close] to close the preview screen.

  9. Click [Query] to search for existing ‘Section Template’ records.
