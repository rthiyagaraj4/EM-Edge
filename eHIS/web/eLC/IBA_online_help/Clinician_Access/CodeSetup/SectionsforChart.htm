Sections for Chart

From this function you can Create an association of Sections to a Chart.

Sections for Chart function enables you to associate the sections defined with Type ‘Chart Section’ in the Discrete Measure Panel function of the Application Masters module for the chart defined in the Chart Setup function.

Creating an association of Sections to a Chart

From the CA Code Setup, select Sections for Chart function.

The Sections for Chart screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the association of Sections to Chart or [Reset] to clear the entered data.
