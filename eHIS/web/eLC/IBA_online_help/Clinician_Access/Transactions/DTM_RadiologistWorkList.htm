Radiologist Work List

From this function you can View the performed radiology orders for the logged in facility and Assign it to required Practitioner of any reporting facility.

This function allows to view the radiology orders performed list specific to logged in facility. The logged in user facility is set as default facility. Also, you can search and view all the facilities radiology orders performed list in the Radiologist Work List. You ca assign any performed radiology order to another facility for reporting.

Viewing Radiologist Work List

From the CA Transactions > Clinician Access > Desktop Menu > Order Management > Radiologist Work List.

The Radiologist Work List screen gets displayed.

  1. By default, the system displays the radiology orders for the current date for the logged in facility.

  2. Enter the mandatory fields (*). <Related Field Description>

  3. Click [Search]. The radiology orders matching the search criteria appear.

  4. In the search results, click ‘Exam Status’ hyperlink of the radiology order to view the ‘Audit Trail’ details in a dialog. To view the Remarks, place cursor on the ‘Remarks’ hyperlink and it appears as a tool tip.

  5. Select the required radiology order using ‘Select’ check box or all orders using the ‘Select All’ check box.

  6. Select the reporting facility which is already associated, so that the performed order is assigned to the practitioner of the reporting facility. This field is mandatory.

  7. Enter the remarks up to maximum 250 characters in the remarks text box if required.

  8. Select the Assigned Radiologist from the lookup. Based on the selected reporting facility, the system loads in the assigned radiologist in the lookup. This field is mandatory.

  9. Click [Assign] to assign the selected performed orders to the radiologists of another reporting facility.
