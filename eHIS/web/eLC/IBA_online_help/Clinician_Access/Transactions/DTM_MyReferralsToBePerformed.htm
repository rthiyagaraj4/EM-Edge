My Referrals To Be Performed

 From this function you can View the referrals done by the referred practitioner and are not yet seen by the other practitioner.

From this function, the referred practitioner can view the referrals done by him and it is not yet seen by the other practitioner. Based on the time duration defined in ‘Time Frame Duration To Consider For My Referrals To Be Performed’ in CA Parameter, the system displays the referrals.

This menu appears under Summary in Desktop menu only if you configure it under the Summary Links section in the Create User Desktop screen in System Manager.

Viewing My Referrals To Be Performed

From the CA Transactions > Clinician Access > Desktop Menu > Summary > Patients, select the My Referrals To Be Performed.

The My Referrals To Be Performed screen gets displayed.

  1. By default, the system displays the referrals not seen by the Referred To practitioner for the current week.

  2. Enter the   search criteria and click [Search] to view specific referrals. Click [Clear] to clear the displayed data.

    1. Patient ID - Select the patient ID using [?] function.

    2. From and To Date - The date range of one week is displayed by default. Enter a period in date format of dd/mm/yyyy.

  3. Click + sign to view the encounter details of that patient record.
