Charts

From this function you can Record Clinical Chart details for a patient.  

The components of a Chart to be recorded are defined in the Chart Component function. The Chart is made applicable to the Patient Class, Specialty, Practitioner Type, Location Type, Location and Age Group in the Chart Applicability function.

You can define the minimum and maximum value as the base value for generating the graph for any chart component in the Discrete Measure screen. If you do not define, the minimum value of the chart component will be zero.

Recording Clinical Chart

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Charting, select Charts.

The Record Charts screen gets displayed.

  1. Select the Chart from the list. The Charts are defined in the Chart Setup function of CA Code Setup. If the Chart selected is a Variable Chart, Define Chart Scope screen is displayed. If the Chart selected is a Fixed Chart, the Events of the Chart as defined in the Chart Components function are displayed for recording. Icon appears near the Chart field. Select the chart value and click icon to add the value as default value. Next time, when you log in the system defaults that value in the Chart field.

    Define Chart Scope screen displays the Events of the Chart. All Events are selected, by default. Clear the Include check box against the required Events to exclude the Events while recording Chart. Click [Apply] to define the scope of the Chart. Click [Cancel] to exit from the screen and the Events for recording are displayed on the main screen.

  2. On selecting Variable Charts, [Modify Scope] is displayed. Click [Modify Scope] to Include or Exclude Events for recording in the Chart.

  3. On selecting Growth Charts, [Past Records] is displayed. Click [Past Records] to view past record details in the Past Records screen.

  4. Enter details in the fields to add a new record. <Related Field Description>

  5. Click [Record] to save the entered details or click [Cancel] to reset the entered details. Click [Print] to print the details after recording values.  The recorded values with the Date and Time of recording are displayed on the right side of the screen with the Symbolic Legends or Colour Indicator as set in the Clinical Event Parameter function. In addition, the shift summary details and timings are displayed if ‘Shift Summary Required’ check box is selected in Chart Setup screen.

  6. Enter the following details to view the values recorded for the patient. <Related Field Description>

  7. Click [Search] to view the recordings in the selected chart for the selected date and time period.

  8. Click the Time hyperlink to record a value as error or replace a value.

    You can record the values as error or replace entered values using this screen.

    1. The name of the Chart is displayed. The date and time of recording and the name of the logged in user marking error or replace details is displayed.

    2. Select the Mark all as Error check box to mark all the values recorded as errors.

    3. Enter the Remarks if any.

    Selected for Error/Amend

    1. Select Error or Replace from the Amend/Error list against the required Event. On selecting Replace, a text box with the value entered is displayed to replace the value.

    2. Enter the Remarks.

    3. Click [Error Details] to view the any previous error details recorded in the Chart Recording Error List screen. Click [Close] to exit from the screen.

    4. Click [Back] at anytime to get back to the main screen and the message ‘The action will clear all the values. Do you want to Continue?’ is displayed. Click [OK] to get back to the main screen or click [Cancel] to stay on the existing screen.

    5. Click [Record] to save the details and the main screen is displayed with the Replaced value, if replaced. If the recorded value is recorded as Error, the value is not displayed in the main screen.

  9. Select the appropriate check box against the Events and click [View Graph] icon to view the graphical representation of the selected events.

    1. Select the check box against the required Events under Standard View on the left hand side of the Graphs screen.  Select the Single or Multiple radio button and click the [View  Graph] under Standard View to view the graphical representation.

    2. Select the check box against the required components and click [New Group] to create a new group.

    3. On selecting Multiple radio button, you can view the graph with multiple and independent Y-axis instead of combining into one axis. Graph is displayed on right side with multiple y-axes. The graph display mode is Single, by default. All the elements are mixed up in single axis and displayed.

    4. Select the X axis and Y axis from the list and click the [View Graph] against Comparison View to view the Comparison Graph.

    5. Select the Growth Chart hyperlinks to view the Graphical representation of the Growth Charts.

    6. Select the required option as 0-36 Months or 2-20 Years by clicking the radio button and select the required value for X Axis and Y Axis. Click [View Graph] against the Custom View to view the graph.

  10. Click [Error Details] to view the error details recorded in the Chart Recording Error List screen.  Click [Close] to exit from the screen.

  11. The Record Charts screen displays the Symbolic Legends or Color Indicators for High, Low, Abnormal, Critical, Critical High and Critical Low as set in the Clinical Event Parameter function of CA Admin Setup.
