Rejected Orders - Laboratory/Radiology

From this function you can View and Review rejected laboratory/radiology orders.

From this function, the practitioner can view the laboratory/radiology orders that are rejected. This function allows to view the reason for rejection and to review it.

Ordering Practitioner or Attending Practitioner have access to review the rejected orders. For other practitioners, it is view only.

Based on location rights of the login practitioner, the system displays the rejected orders for either ordering practitioner or attending practitioner.

Viewing Rejected Laboratory/Radiology Orders

From the CA Transactions > Clinician Access > Desktop Menu > Summary > Orders, select the Rejected Laboratory/Radiology Orders.

The Rejected Orders - Laboratory/Radiology screen gets displayed.

  1. By default, the system displays the lab/radiology orders rejected by the laboratory/radiology team.

  2. Enter the search criteria and click [Search] to view specific rejected orders. Click [Clear] to clear the displayed data.

    1. Patient ID - Select the patient ID using [?] function.

    2. From and To Date - The date range of one week is displayed by default. Enter a period in date format of dd/mm/yyyy.

  3. Click  icon in the Rejected Remarks column against an order to view the reason for rejection in a popup window. The Rejection details includes Specimen Rejection, Date and Time, and the Name of user who rejected it.

  4. Select the check box against the order and click [Reviewed] to mark an order as reviewed. The reviewed order gets disappeared from the list.

If you select the ‘Select All’ check box, only the records in the current page get selected.
