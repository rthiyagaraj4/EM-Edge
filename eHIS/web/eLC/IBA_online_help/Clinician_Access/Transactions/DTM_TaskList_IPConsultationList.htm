IP Consultation List

From this function, you can Consult and Post consultation charges against patient records.

This function allows generating a list of patients seen in a ward and enables you to mark all patients that require consultation. The relevant consultation charges are posted to the respective patient charts. You can charge the patient directly using this function. The consultation charges against each patient are displayed with the provision to edit/discount the charges. You can also cancel charged consultation records.

IP Consultation List

From the CA Transactions > Clinician Access > Desktop Menu > Pending Activities > Tasks > select IP Consultation List.

The IP Consultation List screen gets displayed.

  1. Enter the query criteria and click [Search] to display patients based on the entered criteria or click [Search] without entering the query criteria to display all patients based on the default query criteria.

    1. Self - Self is selected, by default. If Self is selected, the name of the logged in practitioner is displayed.

    2. On behalf of - If On behalf of is selected, select the required Practitioner on behalf of whom the Consultation List has to be generated. Selecting Practitioner is mandatory.

    3. Specialty - If Specialty is selected, select the required Specialty for which the Consultation List has to be generated. Selecting Specialty is mandatory.

    4. Consult Date - The current system date and time is displayed, by default. Select the required date to search for patients to generate the consultation list. A back date is allowed for selection based on the Consultation Parameter Setup.
