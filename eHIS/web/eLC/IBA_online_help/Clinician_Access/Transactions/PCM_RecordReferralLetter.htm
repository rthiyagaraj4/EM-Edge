Record Referral Letter

From this function you can Record Referral Letter for a patient.

This function is used to generate a referral letter at the end of the encounter. Using this function, practitioners can easily consolidate a summary of events recorded during the encounter. The system allows recording a referral letter and printing the same to be handed over to the patient if the patient is being referred to another facility. The clinical note that appears for referral letter recording is based on the note parameter setup.

Recording Referral Letter

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Discharge Activities, select Record Referral Letter function.

The Record Referral Letter screen gets displayed.
