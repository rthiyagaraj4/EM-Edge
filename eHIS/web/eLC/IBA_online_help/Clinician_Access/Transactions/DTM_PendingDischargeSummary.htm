Pending Discharge Summary

From this function you can View pending discharge summary cases for discharged patients whose discharge summary is not prepared within 3 days from Date of Discharge.

This function appears as a hyperlink and displays count only when Discharge summary note is not recorded for any discharged patient within 3 days from the Discharge Date.  This function appears in the Desktop menu only if you configure the ‘Pending Discharge Summary’ under the Summary Links section in the Create User Desktop screen in SM module.

In addition, the system also displays patient status in Red Colour under the ‘Queue Status’ column  in the Patient List by Location screen to indicate that the Discharge summary note has not yet been created for patients who had been discharged.

Viewing Pending Discharge Summary

From the CA Transactions > Clinician Access > Desktop Menu > Summary > Notes, select the Pending Discharge Summary function.

The Pending Discharge Summary screen gets displayed.

  1. By default, the system displays the pending discharge summary cases.

  2. Enter the query criteria and click [Search] to display specific pending discharge summary cases. Click [Clear] to clear the displayed data.

    1. Patient ID - Select the patient ID using [?] function.

    2. From and To Date - The date range of one month is displayed by default. Enter a period in date format of dd/mm/yyyy.

  3. Click the Record/Sign hyperlink, the Discharge Summary Note dialog gets displayed where you can proceed with discharge summary recording and signing.

  4. Once the discharge summary is prepared and signed by the respective practitioner, the patient name is removed from the list. If the discharge summary is only prepared and not signed, the patient name is removed from the list but this summary note appears in the Pending Note Review Summary link.
