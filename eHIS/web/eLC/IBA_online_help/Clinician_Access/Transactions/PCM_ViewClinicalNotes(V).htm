View Clinical Notes (V)

From this function you can View the clinical notes recorded for a patient.

This function helps in viewing all Clinical Notes as recorded in the Clinical Notes function for the patient. Notes recorded by all practitioners belonging to the patient are viewed using this function.

The Quick link feature allows the clinicians to quickly view the notes that they would like to see for the patients. You need to define the quick link filter components in the Define Quick Link screen.

Viewing Clinical Notes

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Event Details, select View Clinical Notes (V).

The View Clinical Notes screen gets displayed.

  1. Click [Search] to display clinical notes within the displayed date range or enter the search criteria and click [Search] to view specific clinical notes.

    1. Note Group – Select the required Note Group from [?] function.

    2. Note Type – Select the Note Type from [?] function.

    3. Performed – Select the practitioner who recorded the Clinical Note from [?] function.

    4. Med/Anc service – Select the Medical or Ancillary Service from [?] function.

    5. Period – Enter the date range in dd/mm/yyyy format to search for clinical notes within the specified period.

    6. Speciality – Select a speciality from the drop-down list. The list displays only those specialities for which the  Practitioner has access. This is not a mandatory field.

    7. Status – Select the status from the list. The list displays Unauthorized, Authorized, Modified and In Error.

    8. Limit to Current Encounter – The check box is selected, by default. Clear the check box to view the notes recorded for past encounters.

    9. Group By/Order By – Select Note Type or Date from the list to view records according to the selected group. Select the Ascending or Descending from the list to display records in the selected order.

    10. Attending Practitioner – Select the required attending practitioner name from [?] function. If ‘Specialty’ field is selected, the attending practitioner list related to the selected specialty gets displayed.

  2. The Clinical Notes are displayed on the left hand side of the screen on searching based on the entered criteria.

  3. Select the Date hyperlink to view the details of the selected Clinical Note.

  4. Click [Print] to print the Clinical Notes of encounters of the selected patient.

  5. Click [Zoom+] to zoom the view of the Clinical Note and [Zoom-] to get back to the original view.
