View Nurse Notes

From this function you can View the nurse notes recorded for a patient.

This function helps in viewing all Nurse Notes recorded in the View Nurse Notes function for the patient. Notes recorded by all practitioners/nurses belonging to the patient are viewed using this function.

Viewing Nurse Notes

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > History, select View Nurse Notes (V).

The View Nurse Notes screen gets displayed.

  1. Click [Search] to display nurse notes within the displayed date range or enter the search criteria and click [Search] to view specific nurse notes.

    1. Note Group – Select the required Note Group from [?] function.

    2. Note Type – Select the Note Type from [?] function.

    3. Performed – Select the nurse who recorded the Nurse Note from [?] function.

    4. Med/Anc service – Select the Medical or Ancillary Service from [?] function.

    5. Period – Enter the date range in dd/mm/yyyy format to search for nurse notes within the specified period.

    6. Status – Select the status from the list. The list displays Unauthorized, Authorized, Modified and In Error.

    7. Limit to Current Encounter – The check box is selected, by default. Clear the check box to view the notes recorded for past encounters.

    8. Group By/Order By – Select Note Type or Date from the list to view records according to the selected group. Select the Ascending or Descending from the list to display records in the selected order.

  2. The Nurse Notes are displayed on the left hand side of the screen on searching based on the entered criteria.

  3. Select the Date hyperlink to view the details of the selected Nurse Note.

  4. Click [Print] to print the Nurse Notes of encounters of the selected patient.

  5. Click [Zoom+] to zoom the view of the Nurse Note and [Zoom-] to get back to the original view.
