Enter the following details to associate Services to a Note Type:

 

  1. Note Type - Select the Note Type from list of valid values. All the Note Types created in Note Type function with status Enabled and ‘Service/Facility Specific’ check box selected gets populated in the list. This field is mandatory.

  2. Department – Select the required department from the list.

Click [Search] and a list of services are displayed. If Department is selected, the services displayed are based on the department. If Department is not selected, all the services are displayed.

  1. Services – The description of the service is displayed.

  2. Select - Select the required Service to be associated to the Note Type by clicking on the Select check box.
