Enter the following details to define CA Parameter:

 

  1. Confidential Pin Required for Printing – Select the check box if you want to confirm PIN to print the clinical details of the patient even if they have access rights for the selected patient. By default, this check box is cleared. If you select this check box, an ‘Authorize Pin Form’ dialog gets invoked on clicking Print button while trying to print the clinical details of the patient from Existing Orders, Order Tracking and View Clinical Event History screen.  This dialog appears even though you have confidential Rights.

  2. Auto Refresh for summary menu (time stamp) – Enter the required time for auto refresh of summary menu as per preference of the facility. The default time is 10 minutes. You can define the time limit from 10 to 99 minutes.

  3. Pending Discharge Summary Grace Period [X] Days -  Enter the grace period i.e., the number of days after the patient is discharged wherein the doctor has not recorded the discharge summary and signed. The patient record gets displayed under the ‘Pending Discharge Summary Patient List’ with count under the Summary menu in Practitioner’s Desktop. Enter numeric digits from 1 to 9. This field is mandatory.

  4. Pop up required to view patient financial details in viewing chart summary of selected patient – Select the check box to display the Financial Details dialog on clicking View Patient Chart Summary or refreshing that page. This dialog displays the outstanding or unbilled amount of the patient across encounters. This field is available only for users in MOHML sites.
