Enter the following details to create a Booking:

 

  1. Procedure/Surgery Date Time – Enter the date of procedure/surgery or select the date by clicking the Calendar function. The Procedure/Surgery Date Time must be greater than or equal to the preferred admission date/time and lesser than the expected days of stay (if entered).

  2. Pre-Op Preparation Days – Enter the number of days for preparation before Operation.

  3. Preferred Admission Date/Time – Enter the preferred date and time for admission in the dd/mm/yyyy hh:mm format or use the Calendar function to select the required date and time. Advance booking is performed within the period specified in IP Parameter ('Max allowable period for advance bed booking [] days') from the current system date and time. This field is mandatory.

  4. Referral ID – Enter the Referral ID of the patient in alphanumeric format of 14 characters or select the referral ID using [?] function to get the ‘Referral Search’ screen.

  5. Medical Service Group – Select an option from the available values in the list. Only those medical service groups that are attached to nursing unit types, which are in turn, attached to a single/multiple nursing units are displayed. The selected medical service group indicates the group under which the patient is being booked. This field is mandatory. For the selected medical service group, if the maximum daily bookings or the maximum monthly bookings are specified, the system checks for the set rules during every booking being made. Once the values set are exceeded, the subsequent bookings are not allowed.

  6. Booking Ref No. – The Booking Reference Number is auto generated or entered manually based on the selected medical service group. If system generates booking reference number for the selected medical service group, the user is not allowed to enter the number. Otherwise, the system prompts the user to enter the booking reference number by making the field entry mandatory. Enter the Booking Reference Number in alphanumeric format of 12 characters maximum length.

  7. Expected Days of Stay – Enter the expected number of days the patient would be staying in numeric format of three characters maximum length.

  8. Expected Discharge Date Time - The Expected Date and Time the patient would be discharged is displayed depending on the Expected Days of Stay entered.

  9. Priority - Select the booking priority from the list. The listed values are ‘Non-Urgent’ and ‘Urgent’.

 

Patient Details

This portion of the screen is disabled and details can only be viewed since the patient is already selected for creating booking.

  1. Contact Details – Click the button to enter the address details of the Patient. If address details are already available for a registered Patient, they are displayed. After entering the contact details, click [Close] to close the contact details screen.

 

Booking Details

  1. Reason for Admission/Booking – Select the reason for admission using [?] function. All the reasons for admission defined in the ‘Complaint’ function of the 'Application Masters' module are available for selection.

  2. View Open Bookings – Click the hyperlink to view open bookings of the patient.

  3. Specialty – Enter the specialty in alphanumeric format of 15 characters maximum length or select the specialty using [?] function. This is a mandatory field. All the specialties from the System Configuration module are displayed. Only those specialties associated to the Nursing Unit(s) belonging to the nursing unit types associated to the selected Medical Service Group is listed.

  4. Admitting Practitioner – Enter the name of the practitioner admitting the patient in alphanumeric format of 30 characters maximum length or select the practitioner using [?] function. The listed practitioners in [?] function correspond to the selected specialty.

  5. Medical Team - Select the medical team from the list. This field is enabled only if the admitting practitioner is selected. The values displayed correspond to the Medical Team to which the admitting practitioner is associated in ‘Practitioner for Medical Team’ in Application Masters module.

  6. Nursing Unit – Select the Nursing Unit from [?] function. The Nursing Unit displayed corresponds to the selected practitioner.

  7. Service – Select the required Service from the list. The primary specialty of the selected Nursing Unit is displayed, by default.

  8. Bed Class – Select the Bed Class from the list

  9. Bed Type – Select the Bed Type based on the selected Bed Class.

  10. Bed No – Select the Bed Number from the [?] function.

  11. Room/Cubicle No – The Room or Cubicle No. is displayed based on the Bed Number selected.

  12. Override – Select the check box if the selected bed is overridable.
