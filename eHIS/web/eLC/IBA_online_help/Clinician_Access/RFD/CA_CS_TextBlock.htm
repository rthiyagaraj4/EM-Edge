Enter the following details to create a new Text Block record:

 

  1. ID - Enter the text block ID in alphanumeric format of two characters maximum length to identify a Text Block uniquely. The text block ID has to be unique and cannot be changed after creation. This field is mandatory.

  2. Text Block - Select the value from the drop-down list. The listed values are Text and Editor.

• If Text is selected, enter the description in alphanumeric format of 4000 characters maximum length.

• If Editor is selected, an Editor hyperlink is displayed. Click the Editor hyperlink to edit the content in text editor. You can edit the font colour, font size, highlight, cut, copy, and paste in the text editor. You can also insert symbols in the text editor by clicking the Symbol  icon.

  1. Service – Enter the required Service in alphanumeric format or select the service using [?] function.

  2. Practitioner – Enter the required Practitioner name in alphanumeric format or select a Practitioner using [?] function.

  3. Note Type – Click [?] function to select a Note Type from the list. All the Note Types created in Note Type function with status Enabled gets listed.

  4. Section – Click [?] function to select a Section from the list. All the Sections created in Section function with status Enabled get listed.

  5. Applicable to Procedures – Select the check box to indicate that the Canned Text is applicable for procedure recording.
