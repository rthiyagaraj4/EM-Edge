Enter the following details:

 

  1. Configure Display Fields - Select the Applicable radio button to select the Result Category. The Not Applicable radio button is selected, by default.

  2. Result Category - Select the required value from the list. The listed values are Adverse Event, Blood Transfusion, Clinical Note, Dept Diagnostics, Dietary, Discharge Summary, Free Documentation, Future Events, Laboratory, Medication, Monitoring, New Born, Patient Care, Problem/Diagnosis, Radiology/Imaging, Surgery/Procedure and Treatment. This is a mandatory field. If the Result Category is selected as Future Events, select the Future Event Type as Appointments, Future Orders or Referral from the list that displays.

  3. Order Sequence - The Order Sequence is displayed as hyperlink. Click the hyperlink to swap it to another order.

  4. Display Fields - Fields which can be configured and displayed in the Note for the selected Result Linked Section Type and Section are displayed. The display fields are based on the selected Result Category.

  5. Select - Select the check box against the required display fields to display them in the Clinical Note on selection of the result. Display fields that are mandatory are selected and disabled, by default.
