Enter the following details:

 

  1. Ordering Facility – Displays the logged in facility as default facility. All the radiology orders performed list specific to that facility gets listed if available without clicking the [Search] button.

  1. Order By – Select the ordered by name from the lookup. The orders are performed only.

  2. Order ID – Enter the required order ID to view the required radiology order.

  3. Ordering Practitioner – Select the required ordering practitioner ID from the lookup.

  4. Order Type – Select the required order type from the lookup.

  5. Catalog – Enter the catalog name. You can search the catalog name using Starts With, Contains, and Ends With.

  6. Performing Facility – Select the performing facility from the lookup

  7. Performed By – Select the Performed By from the lookup.

  8. Performed Date [From and To] – By default, the current date gets displayed. The system displays all performed orders for the current date. You can search for Date range of one month. This field is mandatory.

  9. Reporting Facility – Select the reporting facility which is already associated, so the reported result will be listed.

  10. Assigned Radiologist – Select the assigned radiologist from the lookup. All the assigned orders for the mentioned radiologist get displayed.

  11. Patient ID - Enter the Patient ID if you want to view the of exams for that patient which might be in any exam status.

  12. Exam Status – Select the required exam status from the drop-down list.
