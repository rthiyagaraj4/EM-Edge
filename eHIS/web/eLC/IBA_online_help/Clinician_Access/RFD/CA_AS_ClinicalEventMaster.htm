Enter the following details:

 

  1. History Type – Select the required History Type from the list. This field is mandatory.

  2. Search By – Select either Code or Description by clicking the required radio button to search for the Event based on Code or Description.

  3. Search Criteria – Select the Search Criteria as Starts With, Ends With or Contains to search for the required Event.

  4. Search Text – Enter the required Text to search for the Event in alphanumeric format of 70 characters maximum length. This field is mandatory.

  5. Click [Search] to select the ‘History Types’ by selecting the corresponding ‘Ordering Practitioner’ and/or ‘Attending Practitioner’ check box.

 

History Record Types

Set the applicability of the Order Catalogs for which the results come for review either for the Ordering Practitioner or for the Attending Practitioner based on the flag selected.

  1. Ordering Pract - Select the check box, if reviewing is applicable to the Ordering Practitioner.

  2. Attending Pract - Select the check box, if reviewing is applicable to the Attending Practitioner.
