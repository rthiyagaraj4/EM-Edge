Enter the following details:

  1. Code – Enter the code in alphanumeric format of four characters maximum length. The Code has to be unique and cannot be changed after creation. This field is mandatory.

  2. Description - Enter the description of the Health Risk Factor in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Risk Factor Type – Select Risk Factor Type from the list. The listed values are ‘Age Group’, ‘Allergy’, ‘BMI’, ‘Chief Complaints’, ‘Discrete Measure’, ‘Lab Results’, ‘Order Catalogs’, ‘Pharmacy Items’, ‘Radiology’ and ‘Term Set’.

  4. Enabled - Select the check box if the Health Risk Factor is to be activated. Clear the check box if the Health Risk Factor is to be deactivated. The check box is selected, by default.

According to the Risk Factor Type selected, Risk Factor Indicator fields get displayed.

Age Group

  1. Age Group – Select Age Group from the listed values. This value is generated from Age Group function (Application Masters). This field is mandatory

  2. Risk Factor Indicator – Select Risk Factor Indicator the options are ‘High’, ‘Average’ and ‘Low’.

  3. Remarks – Enter the remarks in alphanumeric format.

Allergy

  1. Adverse Event Type – Select Adverse Event Type from the list. The listed values are populated from the Adverse Event Type function (Medical Records).

  2. Allergen – Select Allergen from the list.

  3. Status – Select Status from the list. The listed values are ‘All’, ‘Active’ and ‘Closed’.

  4. Risk Factor Indicator – Select Risk Factor Indicator the options are ‘High’, ‘Average’ and ‘Low’.

  5. Remarks – Enter the remarks in alphanumeric format.

BMI

  1. Weight – Select Weight using the [?] function.

  2. Height – Select Height using the [?] function.

A grid with values to be entered is displayed.

  1. Gender - Select Gender from the list.

  2. All Ages – Select All Ages check box if it applies for all the ages. This check box is selected by default.

Start and End

  1. Age – Enter the Age in the text box provided in numeric format of maximum 3 digits.

  2. Units – Select Units from the list. The listed values are ‘Days’, ‘Months’ and ‘Years’.

Risk Factor

  2. Value – Enter the Value in text provided.

  3. Risk Factor Indicator – Select Risk Factor Indicator the options are ‘High’, ‘Average’ and ‘Low’.

  4. Remarks – Enter the remarks in alphanumeric format.
