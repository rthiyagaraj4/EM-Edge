By default this tab is displayed on selecting the Trauma Assessment function.

  1. Enter details in mandatory fields. <Related Field Description>

  2. On selecting the Note Type, Sections and Subsections associated to the Note is displayed on the left side of the screen under Notes View. Each of the Section or the Sub Section appears as a hyperlink to invoke the screen for entering the relevant notes.

  3. An icon (c) is displayed if Chief Complaint defined in CA Codes Setup is associated to any section of the selected Note Type in the function ‘Section for Note Type’.

    If a Chief Complaint defined in CA Codes Setup is associated to any Section of the selected Note Type through ‘Section for Note Type’ function of CA Codes Setup, an icon (c) is displayed.

    1. Click [Chief Complaints] to display the Chief Complaints screen.

    2. Scope - Select Scope as Current Encounter, Across Encounter or Open List.

    3. Search Text - On selecting Scope as Open List, entering the text for searching the chief complaint becomes mandatory.

    4. Search Criteria – Select the Search Criteria as Starts With, Ends With or Contains from the list box.

    5. Click [Search] to display Chief Complaints. Select the check box against the required Chief Complaint and click [Select] to copy the Chief Complaint to the editor screen. Click [Cancel] to exit from the Chief Complaints screen.

  4. An icon (i) is displayed for an Image Linked Section and this setup is done in Section function of CA Code Setup by selecting the Image Linked checkbox.

    An Image linked section is used to represent the details that need to be recorded with the help of an Image. The section of the note type is marked as Image linked by selecting the check box ‘Image linked ‘in CA code set up using the ‘Section’ function. The Image is linked to the section using ‘Image for Section’ in CA code setup. If a section of the note type is Image linked, an icon (i) is displayed.

    1. Click the Section or Subsection with (i) icon from the ‘Notes View’ part of the ‘Record Trauma Assessment’ screen.

    2. The editor is displayed to enter details of the Note.

    3. Click [Show Image] to select an image. If an Image is set default to a particular Section in ‘Image for Section’ function, the Image is displayed on selecting that Section and clicking [Show Image]. The tool palate in the image editor is used for marking or recording parts or details of the Image. Click [Show Editor] to switch to the editor screen.

    4. Select an Image from the Available Images list.

    5. Click [Show Both] to display the Text Editor and the Image Editor screen and record details.

  5. The [Upload] button is available at the bottom of the screen if the Note Type has the check box ‘Document Linked Note’ selected in the Note Type function.

    1. Click [Upload] to upload files/documents for the selected Note Type.

    2. File Reference – Browse to the path where the files are stored for uploading.

    3. Remarks – Enter remarks if any.

    4. Click [View] to view the file to be uploaded. Click [Record] to upload the file for the Note Type.

The Sections displayed can be of Types:

Free Format

Result Linked

Main Section

Template Based

  1. Click [Prev Notes] to preview the notes before recording or signing. Close the Preview Notes screen on previewing the contents.

  2. Click [Print Preview] to preview and print the Trauma Assessment Note.
