Intervention Parameters

From this function you can Define a set of parameters that govern the characteristics of interventions.

Defining Intervention Parameters

From the CA Code Setup, select Intervention Parameters function.

The Intervention Parameters screen gets displayed.

  1. Enter the Duration before start time when task becomes due in numeric format of two digits maximum length. Select the duration as day(s), hour(s), minute(s), month(s) or week(s). This is a mandatory field.

  2. Enter the Duration before start time when task becomes overdue in numeric format of two digits maximum length. The duration is for day(s), hour(s), minute(s), month(s) or week(s). This is a mandatory field.

  3. Enter the Duration after end time for marking tasks as expired in numeric format of two digits maximum length. Select the duration as day(s), hour(s), minute(s), month(s) or week(s). This is a mandatory field.

  4. Enter the Backdate limit for resulting of tasks in numeric format of two digits maximum length. Select the duration as day(s), hours(s), minutes(s) or week(s). This is a mandatory field.

  5. Click [Apply] to save the Intervention Parameter or [Reset] to clear the entered data.
