Assign Chart Summary

From this function you can Assign a customized Chart Summary Template to a specific ‘Patient Class’, ‘Specialty’ or ‘Practitioner’ based on which the Patient Chart Summary is displayed in the ‘Clinician Access’ module. You can also Query for associated Chart Summary Template and Delete a queried record.  

The customized Chart Summary Templates are associated to a Patient Class, Specialty or Practitioner to make it applicable only for these defined conditions.

Assigning Chart Summary

From the CA Admin Setup, select Assign Chart Summary function.

The Assign Chart Summary screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save Chart Summary assigned or [Reset] to clear the entered data.

  4. Click [Query] to search for an assigned Chart Summary record.

  5. Click [Delete] to delete a queried record.

Personalize Themes

A new icon to personalize themes is introduced in Assign Chart Summary screen. You can personalise the colour of the widgets using the icon.

  1. Click the icon Personalize Themes dialog gets displayed.

  2. Select the colour theme for the widget.

  3. Click [Default Theme To All Widgets] to set default themes for the widgets.

  4. Click [Apply Theme To All Widgets] to set themes for all the widgets.

  5. Click [Apply Theme] to save the changes made.

  6. Click [Default Theme] to set the default chart summary template.

The selected themes get displayed in Chart Summary screen.
