Practitioner for PMR

From this function, you can Associate a Patient Medical Report to various Practitioners.

Associating Practitioners for PMR

From CA Code Setup > Patient Medical Report Code Setup, select Practitioner for PMR function.

The Practitioner for PMR screen gets displayed.

  1. Select Patient Medical Report.

    Select the Patient Medical Report using [?] function. The [?] function displays Note Type created in the Note Type function of CA Code Setup with Note Groups applicable to Patient Medical Report.

  2. Enter the Search criteria to search for Practitioners.

    1. Search By – Select Practitioner Name or Practitioner ID radio button.

    2. Search Criteria – Select the Search Criteria from the list. The listed values are Starts With, Ends With and Contains.

    3. Search Text – Enter the search text to search for the practitioners based on the search criteria specified.

  3. Click [Search] to search for Practitioners. The Practitioner ID and Practitioner Name of the logged in facility are displayed.

  4. Select the check box against the required Practitioners to associate them to the Patient Medical Report.

  5. Click [Apply] to save the association or [Reset] to reset the screen.
