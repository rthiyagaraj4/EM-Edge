Site For Vaccine

From this function you can Create an association of Vaccine to an Anatomical Site.

The Sites defined in the Anatomical Sites function of Application Master module are available for selection. The Sites which are already associated to the selected Vaccine are displayed on the top half of the screen.

Creating an association of Vaccine to Site

From the CA Code Setup > CA Immunization Code Setup > select the Site For Vaccine function.

The Site For Vaccine screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Vaccine Name in alphanumeric format of 40 characters maximum length or select the required Vaccine Name using [?] function.

  3. Click [Show Detail] to display the Anatomical Sites in the Select Sites frames.

  4. Click [Search] to display all the Sites defined in the Anatomical Site function of Application Masters module or enter the search criteria to select a specific site name. On searching, the Sites are displayed based on the searched criteria.

    1. Site – Enter the Site Name in alphanumeric format of 15 characters maximum length.

    2. Search Criteria – Select the Search Criteria value from the list.  The listed values are Starts With, Ends With and Contains.

  5. Select the check box against the desired Anatomical Site to display the selected Sites in the Selected Sites frame.

  6. Select the Default check box against the desired Anatomical Site to make the Site as default.

  7. Select the check box to select the Site. On clearing the Select checkbox, the Site is eliminated from the Selected Sites frame.

  8. Click [Apply] to save the association of Site to Vaccine or [Reset] to clear the entered data.
