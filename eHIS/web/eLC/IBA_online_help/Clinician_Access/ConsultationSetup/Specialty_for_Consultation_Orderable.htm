Specialty for Consultation Orderable

From this function, you can Create an association of a consultation orderable to a Specialty.

The association is performed by selecting an Order Catalog and associating a Specialty or selecting a Specialty and associating an Order Catalog. Once the specialty is associated to the orderable, the check box is disabled. Only one orderable is associated to a particular specialty.

Creating an association of Specialty to Consultation Orderable

From the CA Code Setup > Consultation Setup, select Specialty for Consultation Orderable function.

The Specialty for Consultation Orderable screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select Group By as Order Category or Specialty. Order Category is displayed, by default.

  3. Select the Order Catalog using the [?] function. Order Catalogs created with Order Category Consult Order are displayed for selection. This is a mandatory field.

  4. Select the required alphabet hyperlink under the Select Tab to display Specialties beginning with the selected alphabet.

  5. Select the check box against the required Specialty to associate the Specialty to the selected Order Catalog.

  6. Click the Associate Tab to view the Specialties associated to the Order Catalog.

  7. Click [Apply] to save the association of Specialty to Consultation Order Catalog. The Specialty selected for an Order Catalog is disabled and not allowed for association to another Order Catalog. Click the Associate Tab to view the specialties associated to the Order Catalog. Clear the check box against the required Specialty to dissociate the Specialty from the Order Catalog. Click [Apply] to save the dissociation of Specialty from the Consultation Order Catalog. The Specialty is then available for association to any other Order Catalog.

  8. If Group By is selected as Specialty, select the Specialty using the [?] function. Selecting the Specialty is mandatory.

  9. Select the Order Catalog using the [?] function. Order Catalogs created with Order Category Consult Order are displayed for selection. This is a mandatory field.

  10. Click [Apply] to save the association of Specialty to Consultation Order Catalog.
