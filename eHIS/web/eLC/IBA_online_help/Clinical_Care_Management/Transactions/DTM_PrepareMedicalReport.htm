  2. Requester Type – Select the Requester Type from the list. The listed values are same as the indicators in the ‘Requester’ function in the Medical Records module.

  3. Request Date [From-To] – Enter the date range of request in the dd/mm/yyyy format.

Patient Details

  1. Patient ID – Enter the Patient ID in alphanumeric format of 10 characters maximum length or select the Patient ID using [?] function.

  2. Citizen ID (Prompt entered in the MP Parameter) – Enter the Citizen ID of the patient in the length as specified in the MP Parameter function.

  3. Alternate ID – Enter the Alternate ID.

Prepare Medical Report

From this function you can View requests made for patients to prepare Medical Report.

A Medical Report is a clinical document that contains the details of the encounters that the patient has had with the logged in facility.

Preparing Medical Report

From the CA Transactions > Clinician Access > Desktop Menu > Medical Reports, select Prepare Medical Report.

The Prepare Medical Report screen gets displayed.

  1. Enter the search criteria and click [Search] to display specific Request IDs or click [Search] to display all Request IDs forwarded to the logged in practitioner to prepare Medical Report.

  2. Click the Request ID hyperlink to display the Prepare Medical Report screen.

  3. Click [Record Notes] to record Notes. Note Types associated to Note Group defined with Applicable Task ‘Prepare Medical Report’ are displayed to record Notes.

  4. Click [View Medical Report] on the Prepare Medical Report screen to view the Prepared Medical Report.

  5. Click [Record] to save the Medical Report prepared.
