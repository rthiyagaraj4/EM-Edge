Define Quick Link

Define a Quick Link with:

Quick Link Type – Query Filter

Function – View Clinical Event History

Description – View History

Scope – Function Specific

Enabled

Period From – Before 1 Year From Current System Date and Time

Period To – Before 1 Day From Current System Date and Time

Specify other filter criteria if required.

A query criteria is defined for the Quick Link ‘View History’ applicable to the function ‘View Clinical Event History’ at Function Level.

Quick Link for Responsibility

Select the following:

Link Type – Filter

Function – View Clinical Event History

Responsibility – Clinician Access

View History – Select check box.

Now, the Quick Link ‘View History’ is applicable to the ‘View Clinical Event History’ function of the responsibility ‘Clinician Access’.

Impact on Transaction

Navigate to Clinician Access > Desktop Menu > Select Patient > Patient Chart Menu > View Clinical Event History.

Or click History Icon from the icon bar.

The quick link works only for this function based on the data setup.

Select ‘View History’ from the list box at the top right corner.

The screen displays the query criteria as set in the function ‘Define Quick Links’ for ‘View History’. Select the required History to view the details.

Click the icon next to the list box.

Select the check box and click [Update] to update the quick link for the user at the function level. Select the default check box and click [Update] to default the quick link for the user on load of the function.

Quick Links

1 What is Quick Links Setup in Enterprise Management?

Quick Links setup enables practitioners or clinicians to view vital information (eg. past results of patients) and access different functions of Enterprise Management quickly from the current function used in the application.

 

2 How to access Quick Links?

Currently, this feature is applicable only to the Clinician Access module. Quick Links can be accessed from the Clinicians Desktop Menu functions and/or Patient Chart Menu functions based on the setup.

 

3 What are the types of Quick Links?

Query Filter - Used to predefine a query criterion to default the query on accessing the quick link.

Actual Function - Used to display the actual function on accessing the quick link.

External Application - Used to display an external application on accessing the quick link.

 

4 What are the Pre-requisites for Quick Links?

Perform the setups in 'Define Quick Link' and 'Quick Link for Responsibility' functions from CA Quick Links Setup under CA Codes Setup.
