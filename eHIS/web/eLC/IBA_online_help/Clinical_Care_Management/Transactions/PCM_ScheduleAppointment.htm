Schedule Appointment

From this function you can Schedule an appointment for a patient.

An appointment schedule is a planned visit for a future date against a location or location and resource combination. This function is used to view availability of schedules and perform a booking for a future date. An appointment can be scheduled against a location with or without assigning a resource based on the attribute of the location. This function provides an intuitive interface to query the earliest available schedule and supports multiple schedules against a single patient. The screen that gets displayed on invoking the function Schedule Appointment is based on type of practitioner who has logged in. The screen that displays on invoking this function depends on whether the practitioner is identified as a primary resource against any care location or not.

Scheduling Appointment

From the CA Transactions > Clinician Access > Desktop Menu > select required patient > Patient Chart Menu > Schedules, select Schedule Appointment.

The Outpatient Appointment screen gets displayed.

  1. If the Calendar is displayed on the screen (this is based on whether the logged in practitioner is identified as the primary resource against the care location), select the required hyperlinked date to view the time slots vacant. Click the required time slot hyperlink to book an appointment.

  2. If the Calendar is not displayed (if the logged in practitioner is not identified as the primary resource against the care location), select the required hyperlinked slot against the required practitioner to book an appointment for the patient.
