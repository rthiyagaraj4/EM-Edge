Section for Note Type

From this function you can Create an association of Sections to a Note Type and Query for Sections associated to Note Types.

Section for Note Type function allows associating different types of Sections to a Note Type. The Sections associated to a Note Type are displayed for recording details while selecting the Note Type in the Clinical Notes transaction.

Creating an association of Sections to a Note Type

From the CA Code Setup, select Section for Note Type function.

The Section for Note Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the association of Sections to Note Type or [Reset] to clear the entered data.

  4. Click [Query] to search for existing ‘Section for Note Type’ records.
