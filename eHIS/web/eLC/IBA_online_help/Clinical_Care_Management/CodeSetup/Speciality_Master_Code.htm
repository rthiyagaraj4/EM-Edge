Specialty Master Code

From this function you can Setup various specialty masters codes required for use in the transactions of this module.

Most of these codes are optional and used whenever, applicable. The master setup function comprises of codes that would be applicable for usage in various events of Maternity Consultation transaction function.

Creating Specialty Master Codes

From the CA Code Setup, CA Specialty Code Setup, select Specialty Master Code function.

The Specialty Master Code screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the required Master Category from the list.

  3. Enter a Code, Long Description and Short Description for the selected Master Category. Select the Enabled check box to enable the record in the transactions.

  4. Click [Add] to add the entered details.

  5. The Specialty Episode Cycle is displayed. The Module, Episode Description and Event Title Description are displayed.

  6. Click the Episode Description to modify the Episode Description and Event Title Description.

  7. Click [Apply] to save the modification.
