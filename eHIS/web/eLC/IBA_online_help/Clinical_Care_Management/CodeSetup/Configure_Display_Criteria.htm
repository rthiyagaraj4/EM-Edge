Configure Display Criteria

From this function you can Configure the display fields for the selected Result Category.

Configuring Display Criteria

From CA Code Setup, select Configure Display Criteria function.

The Configure Display Criteria screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the display criteria or [Reset] to clear the entered data.
