Specialty Parameter

From this function you can Define Specialty parameter for Mothercare Module.

This function is used to define specialty parameter for Mothercare module that controls the duration in which Last Menstrual Period and the Corrected EDD can be modified.

Defining Specialty Parameter

From the CA Code Setup, select Specialty Parameter function.

The Specialty Parameter screen gets displayed.

  1. Mother Care Module is displayed and disabled, by default.

  2. Enter the number of months or encounters (1-9) to restrict modification of LMP/Corrected EDD. Select Months, Encounters or Encounters/Month from the list. If Encounters/Month is selected, a text ‘which ever is less’ is displayed.

  3. Enter the number of days in the format of three digits maximum length beyond or before which you cannot extend the corrected EDD. The system validates with the Gestation Range value defined in MPI Parameter based on which you can modify the entered EDD in Maternity Consultation function.

  4. Click [Apply] to save the defined Parameter or [Reset] to clear the entered data.
