Special Chart Key

From this function you can Create a Growth Chart for a Chart Group on Interval basis or Static.

This function is used while recording Clinical Charts in the Clinician Access module.

Creating Special Chart Key

From the CA Code Setup, select Special Chart Key function.

The Special Chart Key screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Special Chart Key created or [Reset] to clear the entered data.
