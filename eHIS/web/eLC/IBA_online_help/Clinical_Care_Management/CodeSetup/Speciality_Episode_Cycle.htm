Specialty Episode Cycle

From this function you can Define the title ‘Cycle’ in the Maternity Consultation function.

The title description defined in this function is displayed in the Maternity Consultation function while clicking the New Event icon.

Identifying Specialty Episode Cycle

From the CA Code Setup, CA Specialty Code Setup, select Specialty Episode Cycle function.

The Specialty Episode Cycle screen gets displayed.

  1. Click [Query] to query for system defined Specialty Episode Cycle.

  2. Enter the query criteria and click [Execute].

  3. The Specialty Episode Cycle is displayed. The Module, Episode Description and Event Title Description are displayed.

  4. Click the Episode Description to modify the Episode Description and Event Title Description.

  5. Click [Apply] to save the modification.
