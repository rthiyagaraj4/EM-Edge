Vaccine Category

From this function you can Define Vaccine Category for an Age Group and Query for an existing Vaccine Category record.

The defined Vaccine Category is termed as Schedule Name. You can create Vaccine Categories like Newborn, Mothercare, etc.

Defining Vaccine Category

From the CA Code Setup > CA Immunization Code Setup, select the Vaccine Category function.

The Vaccine Category screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the defined Vaccine Category or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing Vaccine Category record.
