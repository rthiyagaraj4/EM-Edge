Immunization Parameter

From this function you can Link the vaccine to schedule and Define the adhoc vaccine category.

Defining Immunization Parameter

From the CA Code Setup > CA Immunization Code Setup, select the Immunization Parameter function.

The Immunization Parameter screen gets displayed.

  1. Enter the ‘Adhoc Vaccine Category for MAR’ or select the vaccine category using [?] function. This is a mandatory field.

  2. Enter the number of days in the numeric format of three digits maximum length. This is a mandatory field.

  3. Click [Apply] to record the defined parameter or [Reset] to clear the entered details.
