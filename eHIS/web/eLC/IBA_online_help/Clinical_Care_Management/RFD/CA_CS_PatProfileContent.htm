Enter the following details to define Patient Profile Content:

 

  1. System Description – The system provided fields are displayed.

  2. User Defined Description – Enter the required user description for the field in alphanumeric format.

  3. Order Srl No – Enter the required sequence number for ordering the descriptions in numeric format of three digits maximum length.

  4. Include – Select the check box to include the fields in the Patient Profile screen of Patient Chart.
