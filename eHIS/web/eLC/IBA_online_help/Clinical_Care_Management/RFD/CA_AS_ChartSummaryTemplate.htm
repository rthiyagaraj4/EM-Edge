Enter the following details:

 

  1. Summary ID – Enter Chart Summary ID in alphanumeric format of 15 characters maximum length or click [?] function to search and select for the summary ID. This field is mandatory.

  2. Summary Desc – Enter Chart Summary Description in alphanumeric format of 50 characters maximum length. This field is mandatory.

  3. No. of Rows – This field decides the Number of Rows displayed in Chart Summary Template. Enter the required number of rows in numeric format of two digits maximum length. The entered value should not be greater than 10, or the system prompts a message. This field is mandatory.

  4. No. of Columns – This field decides the Number of Columns displayed in ‘Chart Summary Template’. Enter the required Number of Columns in numeric format of two digits maximum length. The entered value should not be greater than 10, or the system prompts a message. This field is mandatory.

  5. Enabled – This field is selected making the Template available in the transactions, by default. Clear the check box to disable the Template.
