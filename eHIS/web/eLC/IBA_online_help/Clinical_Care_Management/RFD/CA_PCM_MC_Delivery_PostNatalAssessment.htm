Post Natal Assessment

Select Post Natal Assessment from the Post Natal Assessment task to record post natal assessment details of the event Delivery.

 

Post Natal Assessment

  1. Delivery Place - Select Within Facility or Outside Facility as the delivery place. Specify details if Others is selected.

  2. Check and Verify - Select the check box if post natal assessment has been checked and verified.

  3. Assessment Time - Select the post natal assessment time from the calendar function.

  4. Assessed By - Select the practitioner, who assessed the post natal assessment. All the practitioners defined in the Application Masters module is displayed for selection.

  5. Assessment Location - Enter the location of assessment in the test area provided.

  6. Current BP - Enter the current systolic and diastolic blood pressure in the respective text areas.

  7. Current Temperature - Enter the current temperature of the mother in degrees Celsius.

  8. Current Pulse - Enter the current pulse rate of the mother.

  9. Select the required details under Breasts, Nipples, Uterus Contracted, Vaginal Bleeding, Episiotomy/tear, Urine Output, AntiD Given, Breast Feeding Initiated, Family Planning advice given, Rubella vaccination given, Hepatitis vaccination given.

  10. Remarks - Enter remarks on the post natal assessment for the mother. This is a mandatory field.

  11. Medication Details - Select the Medication details. This is a mandatory field.

  12. Hb - Enter the Hb count in grms/dl. This is a mandatory field.

  13. Click [Record] to save details of Post Natal Assessment recorded for the event Delivery. The date and time of recording the Post Natal Assessment details is displayed on the left side menu of the screen under Post Natal Assessment task.

  14. Click the Date and Time hyperlink and the details recorded for Post Natal Assessment for the event Delivery gets displayed on the right side of the screen.
