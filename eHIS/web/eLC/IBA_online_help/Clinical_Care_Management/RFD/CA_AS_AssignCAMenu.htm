Enter the following details to Assign CA Menu:

 

  1. Patient Class – Select the required Patient Class from the list. The listed values are Day Care, Emergency, External, Inpatient and Outpatient. ‘All’ is selected, by default.

  2. Practitioner Type – Select the required Practitioner Type from the list. All’ is selected, by default.

  3. Practitioner – Enter the required Practitioner name in alphanumeric format or click [?] function to search and select the required Practitioner. All the Practitioners corresponding to the selected Practitioner Type are displayed All’ is selected, by default.

  4. Menu – All the Menus created using ‘Define Menu’ are listed in the list. Select the required ‘Root Menu’ from the ‘Menu’ list. This field is mandatory.
