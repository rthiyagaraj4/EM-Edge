Enter the following details to create a Flow Sheet Applicability record:

 

  1. Practitioner Type – Select for the Practitioner Type from the list, to associate the Flow Sheet Component to the Practitioner Type.

  2. Specialty – Enter the required Specialty in alphanumeric format or select the Specialty using [?] function.

  3. Practitioner - Enter the required Practitioner in alphanumeric format or select the practitioner using [?] function. Practitioners are displayed based on the selected Practitioner Type.

  4. Flow Sheet – Enter the required Flow Sheet in alphanumeric format or select the Flow Sheet using [?] function. All Flow Sheets defined in the Flow Sheet Component function is displayed. This field is mandatory.
