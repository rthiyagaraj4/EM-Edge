Enter the following details:

 

  1. Chart Type – Select the Chart Type as Growth Chart from the list. This field is mandatory. On selecting the Chart Type, a field Chart Group is displayed.

  2. Chart Group – Select the Chart Group as 0-36 Months or 2-20 Years from the list. This field is mandatory.

  3. Element Type – Select the Element Type from the list as Head Circum, Chest Circum, Length or Weight. This field is mandatory.

  4. Gender – Select the gender from the list as Male, Female or All. This field is mandatory.

On selecting the Chart Group, the range of values are displayed to enter the Median, Variance, Trans and Percentiles. The 12 user defined percentiles are displayed, Percentile ‘P’ is hard coded, you can enter the numeric values in the numeric format of 0 to 100.
