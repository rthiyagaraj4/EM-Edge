Enter the following details:

 

  1. Link Type - Select the Link Type as Filter or Global.

  2. Function - Select the Function if the Link Type is selected as Filter. This is a mandatory field. If the Link Type is selected as global, this field is disabled.

  3. Responsibility - Select the Responsibility from the list. The Responsibilities are defined in the System Manager module.

  4. Click [Search] to display the Global Quick Link if the Link Type is selected as Global. Select the check box against the required Quick Link to associate to the Responsibility. If the Link Type is selected as Filter, on clicking [Search], the Quick Links defined for the function are displayed.

  5. Select - Select the check box against the required Quick Link to associate to the Responsibility.

  6. Default - Select the check box against the required Quick Link to default the selected Quick Link in the transaction.
