Enter the following query criteria:

 

  1. Search With – Select either All or Patient ID from the list to search for all patients within a period or a particular patient. This field is mandatory.

  2. Patient ID – Select the Patient ID using [?] function. This field is mandatory if the ‘Search With’ field is selected as Patient ID.

  3. Patient Name – The name of the patient selected is displayed and disabled.

  4. Appt No - Enter the Appointment Reference Number in alphanumeric format of eight characters maximum length, or the Appt Ref No is displayed on entering or selecting a Patient ID.

  5. Appt Date [From-To] - Enter the date range for which appointments for Patient are scheduled in dd/mm/yyyy format or click the Calendar function to search for the required dates. This field is mandatory if the Search With field is selected as All.

  6. Specialty – Select the specialty from the list.

  7. Location Type – Select the Location Type list. The valid values are ‘Daycare Unit’, ‘Clinic’, and ‘Procedure Unit’.

  8. Location – Select the required location where the patient is scheduled for an appointment using [?] function.

  9. Resource Type – Select the Resource Type from the list. The valid values are ‘Practitioner’, ‘Equipment’, ‘Room’, and ‘Other’.

  10. Resource - Select the required resource against which the Patient is scheduled for an appointment.
