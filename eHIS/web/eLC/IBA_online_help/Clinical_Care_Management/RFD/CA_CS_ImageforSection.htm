Enter the following details to create a new Image for Section record:

 

  1. Section – Click [?] function to select a result linked Section from the list. This field is mandatory.

  2. Image Reference – Click [?] function to select an Image reference of the image that needs to be attached to the section.

  3. Default – Select the check box to display the image as default on invoking the section. More than one images are allowed to be associated to the section.
