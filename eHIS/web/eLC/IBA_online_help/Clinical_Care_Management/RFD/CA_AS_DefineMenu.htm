Enter the required details to Define Menu:

 

  1. ID – Enter the Menu ID in alphanumeric format of 10 characters maximum length. This field is mandatory.

  2. Short Description – Enter Short Description for the Menu in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Long Description - Enter the Long Description for the Menu in alphanumeric format of 40 characters maximum length. This field is mandatory.

  4. Root Menu – Select the check box if the Menu is a ‘Root’ Menu. On selecting the Root Menu check box, specify the applicability of the menu by selecting Desktop and/or Patient Chart check box. This field is mandatory. The Desktop and Patient Chart check boxes appear only when Root Menu check box is selected.

  5. Enabled – Select the check box to enable the Menu created.
