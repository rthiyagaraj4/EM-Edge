Enter the following details:

 

  1. Note Group – Select the Note Group from the list.

  2. Note Type – Select the Note Type from the list.

  3. Performed – Select the name of the Practitioner who recorded the Note using [?] function.

  4. Med/Anc Service – Select the Medical or Ancillary Service using [?] function.

  5. Period – The Period is displayed, by default.

  6. Status – Select the status as In Progress, Transcribed, Unauthorized, Authorized, Modified or In Error from the list.

  7. Show Notes – Select Awaiting Review or Awaiting Sign from the list.

  8. Include Others’ Unsigned Notes – Select the check box to include unsigned notes of other practitioners.

  9. Ref No - Enter the Reference Number as defined in the Document Ref Format function of CA Admin Setup.
