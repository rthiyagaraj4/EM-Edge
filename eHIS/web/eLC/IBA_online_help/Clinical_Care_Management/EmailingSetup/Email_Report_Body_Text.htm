Email Report Body Text

From this function you can Create the body text of reports required for e-mailing purposes and Query for a desired Report Body Text record.

More than one Body Text can be defined for a selected Report.

Creating Email Report Body Text

From the CA Code Setup > Emailing Setup, select Email Report Body Text function.

The Email Report Body Text screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Report Body Text created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Report Body Text records.
