Note Parameter

From this function you can Define Note Parameter.

The Note Parameter function identifies the Note Type used in the Clinician Access module to record ‘Discharge Summary’ and ‘Visit Summary’. The ‘Discharge Summary’ and ‘Visit Summary’ are recorded only in the attached Note Type.

Setting Note Parameter

From the CA Admin Setup, select Note Parameter function.

The Note Parameter screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Apply] to save the Note Parameter or [Reset] to clear the entered data.

 

The 'Discharge Summary Note' is prepared for Inpatients and 'Visit Summary Note' is prepared for Outpatients.
