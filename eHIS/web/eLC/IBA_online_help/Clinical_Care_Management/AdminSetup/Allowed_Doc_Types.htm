Allowed Doc Types

From this function you can Define the Document Types allowed to be added or uploaded to the system.

Allow Doc Types function allows you to permit or restrict the type of documents or images that the system allows to be uploaded through various functions. For example, ‘Note Image List’.

Defining Allowed Document Types

From the CA Admin Setup, select Allowed Doc Types function.

The Allowed Doc Types screen gets displayed.

  1. Enter the details in the required fields. <Related Field Description>

  2. Click [Apply] to save the Allowed Doc Types or [Reset] to clear the entered data.
