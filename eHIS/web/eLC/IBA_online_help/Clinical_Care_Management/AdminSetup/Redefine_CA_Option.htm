Redefine CA Option

From this function you can Redefine some of the Clinician Access functions as required.

The system defined Short Descriptions and Long Descriptions of some of the functions available in the Clinicians Desktop menu are displayed for creating user defined Short Descriptions and Long Descriptions.

Redefining CA Options

From the CA Admin Setup, select Redefine CA Option function.

The Redefine CA Option screen gets displayed.

  1. Enter the details in the required fields. <Related Field Description>

  2. Click [Apply] to save redefined CA Options or [Reset] to clear the entered data.
