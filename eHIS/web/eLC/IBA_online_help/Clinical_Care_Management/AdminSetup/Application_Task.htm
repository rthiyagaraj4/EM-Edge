Application Task

From this function you can Set Tasks of the Clinician Assess module applicable to the Enterprise.

The Application Tasks are the various Functions/Actions that form the part of the Clinician Access module. The Application Tasks are inserted through the installation scripts. The privilege status of an Application Task can be set applicable to the facility.

Setting Applicable Task

From the CA Admin Setup, select Application Task function.

The Applicable Task screen gets displayed.

  1. Enter the details in the required fields. <Related Field Description>

  2. Click [Apply] to save Applicable Tasks created or [Reset] to clear the entered data.
