Chart Summary Content

From this function you can Define user descriptions for system descriptions that should appear as components in the Patient Chart Summary.

Chart Summary is a summarized view of a patient’s condition. The function helps defining the user defined descriptions for the components in the Chart Summary. If the user defined descriptions are not defined, the system descriptions are displayed in the Chart Summary.

Defining Chart Summary Content

From the CA Admin Setup, select Chart Summary Content function.

The Chart Summary Content screen gets displayed.

  1. Enter the details in the required fields. <Related Field Description>

  2. Click [Apply] to save Chart Summary Content created or [Reset] to clear the entered data.
