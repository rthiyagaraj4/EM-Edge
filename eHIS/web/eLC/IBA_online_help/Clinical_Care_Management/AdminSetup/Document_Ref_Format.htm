1. Encounter ID – Click Encounter ID hyperlink to add the Encounter ID text box at the bottom of the screen. The Encounter ID of a patient is system defined and is automatically generated while generating Document Reference Number. You can add more than one Encounter ID text box.

  2. Practitioner ID - Click Practitioner ID hyperlink to add the Practitioner ID text box at the bottom of the screen. The Practitioner ID is displayed while generating Document Reference Number. You can add more than one Practitioner ID text box.

  3. Accession No - Click Accession No hyperlink to add the Accession No text box at the bottom of the screen. Enter the Accession Number in the Enter/Edit field and click [Insert] and the Accession No is displayed while generating Document Reference Number. You can add more than one Accession No text box.

  4. Date & Time - Click Date & Time hyperlink to add the Date & Time text box at the bottom of the screen. The current system Date & Time is displayed while generating the Reference Number. You can add more than one Date & Time text box.

Enter / Edit – Enter the Number in alphanumeric format of 40 characters maximum length. Click [Insert] to enter the Number.

Replace – Select the Replace radio button to replace the values in the text box. Select the text box from the bottom of the screen for which the values have to be replaced. Enter the Number in the Enter/Edit field. Click [Insert] to replace the value.

Insert Left - Select the Insert Left radio button to insert a new text box on the left side of the currently selected text box. Enter the value in the Enter/Edit field and click [Insert] to inset the value to the left of the selected text box.

Insert Right - Select the Insert Right radio button to insert a new text box on the right side of the currently selected text box. This radio button is selected, by default. Enter the required value in the Enter/Edit field and click [Insert] to inset the value to the right of the selected text box.

Document Ref Format

From this function you can Define the Reference Format for documents.

A Document Reference Number is unique to documents, patients, visits and records. The Document Reference Number is a combination of Encounter Number, Practitioner Code and Date and Time. This function helps in defining the format to generate a unique reference number for verification purpose, which is displayed on clinical notes.

Defining Document Reference Format

From the CA Admin Setup, select Document Ref Format function.

The Define Header screen gets displayed.

  1. Select the Document Type as Clinical Document or Plain Document.  

  2. Select the required hyperlink under ‘Component Definition’ to define the reference format for the selected Document Type. The selected Component is displayed at the bottom of the screen.

  3. Enter the required value in the Enter/Edit field and click [Insert] to display the value as part of the Reference Number.

  4. Click Replace, Insert Left or Insert Right to arrange the components.

  5. Click [Remove] to remove the text box from the bottom of the screen. Click [Reset] to reset the selected data.

  6. Click [Apply] to save the Document Reference Format or [Reset] the entered details.
