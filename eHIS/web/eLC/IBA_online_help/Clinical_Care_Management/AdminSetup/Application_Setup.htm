View the following details:

  1. External ID – The External ID of the external application is displayed. This is a hyperlink.

  2. External Name –The name of the external application is displayed.

  3. Base URL – The URL given to the particular external application is displayed.

  4. Enabled – Select the ‘Enabled’ check box to make the ‘External Application setup’ appear in the Clinical Event.

External Application

From this function you can View the External Applications.

External Application Masters is set in this function. The list of interfaced applications is queried and corresponding URL to access application is set up at the client site.

Defining External Application

From the CA Admin Setup, select Application Setup function.

The Application Setup screen gets displayed.

  1. Click [Query] to view the Application Setups.

  2. Click to view/modify/enable and disable the particular application.

  3. Click [Apply] to save the External Application details or [Reset] to clear the entered data.
