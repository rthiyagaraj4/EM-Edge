Define Quick Link

From this function, you can Define Quick Links which are function specific and global allowing you to view past results and vital information of patients at a quick glance.

This function enables defining Quick Links for a Query Filter, Actual Function and External Application.

Defining Quick Links

From the CA Code Setup > Quick Links Setup, select Define Quick Link function.

The Define Quick Link screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Quick Link defined or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Quick Link records.
