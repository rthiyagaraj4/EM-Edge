Loan Linen

From this function you can Lend linen items from a Nursing Unit, Clinic or Department to another Nursing Unit, Clinic, Department or to a patient within a Facility. You can Cancel the request placed and View the details.

The Loan Linen function is used to record the details of the loaned linen items in the facility. You can keep track of the movement of linen items between various source types. A request number is generated for each transaction. The name of the person lending and receiving loaned linen between the sources, reason for lending and quantity of the loaned linen items are recorded using this function. You have to select a Source Type and Source to loan a linen item. The values selected are defaulted and displayed in the transaction functions.

Loan Linen Items

From the Linen > Linen Transaction, select the Loan Linen function.
