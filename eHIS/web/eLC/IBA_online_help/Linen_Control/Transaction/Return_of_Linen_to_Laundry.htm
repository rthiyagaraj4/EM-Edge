Return of Linen to Laundry

From this function you can Return the Linen Items to Laundry if they are found to be defective and also View the Stock Status and Work Status of the Linen Items in the laundry.

Return of Linen to Laundry function is used to record the linen item details, which are returned to laundry by the facility. The linen items are returned back to the laundry for cleaning, if the quality is not met. The linen items received, but not intended for the respective Nursing Unit are also returned to the facility and the details are recorded. You can record a note to enter additional details. You have to select a Source Type and Source to return the linen items. The values selected are defaulted and displayed in the transaction functions.

Returning Linen Items to Laundry

From the Linen > Linen Transaction, select the Return of Linen to Laundry function.

The Return of Linen to Laundry screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Select] to record the details of returned linen or [Cancel] to clear the entered or displayed details.

  3. The selected Linen Item is displayed along with the returned quantity and remarks. Linen Item and Remark fields are displayed as hyperlinks. On clicking the Linen Item hyperlink, the selected Linen Item and its returned quantity get displayed in the respective fields for editing. Click Remark hyperlink to view the entered remarks.

  4. To edit the remark, select the Linen Item hyperlink and enter the details by clicking ‘Remark’ hyperlink. Click [OK] to record the remarks. Click [Cancel] to cancel the transaction.

  5. Click [Submit] to save the details.

  6. A confirmation message ‘Please note your Request Number’ is displayed on clicking [Submit]. Click [OK] to record the details and the control shifts back to the main screen to record the details.

  7. The Stock Status and Work Status of the linen items in the laundry are displayed. You can choose to view the status of the linen items in laundry by selecting a source Type and a Source. The current system date is displayed. Click [Go] to search for the details in the respective Source and Source Type.
