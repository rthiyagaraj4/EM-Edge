Standard Stock for Sources

From this function you can Maintain a Standard Stock of linen items used by the Source in a facility. You can Query for the stock details of a source in the facility.

The Standard Stock for Sources function helps to maintain and control the minimum stock of linen items used in the facility. This avoids shortage of linen items in need of urgency and also has a control on the linen items issued to Nursing Units, given to laundry and received back. You can use this function to track the stock details of a linen item in the Nursing Units/Clinics/Department. You can update the standard stock defined for a selected linen item.

Maintaining a Standard Stock for Sources

From the Setup > Code Setup, select the Standard Stock for Sources function.

The Standard Stock for Sources screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Source Type from the list. The system-defined values are ‘Nursing Unit’, ‘Clinic’ and ‘Department’. This is a mandatory field.

  3. Select the Source using [?] function. The system-defined values are - Nursing Unit, Clinic and Department. This is a mandatory field.

    • If the Source Type is selected as ‘Nursing Unit’, Nursing Units defined and associated to the logged-in-facility in Inpatient and Daycare Management module gets listed.

    • If the Source Type is selected as ‘Clinic’, Clinics defined and associated to the logged-in-facility in Outpatient Management module gets listed.

    • If the Source Type is selected as ‘Department’, Departments defined and associated to the logged-in-facility in Application Masters module gets listed.

  4. Click [Search] to display the quantity of standard stock of linen items associated to the selected source and source type.

  5. The values defined and enabled in ‘Linen Item’ function gets listed.

  6. Enter the Standard Stock in numeric format of five digits maximum length. This indicates the standard stock applicable for the selected source in the facility. You can update the stock quantity recorded for a selected linen item in a source.

  7. Click [Apply] to save the stock details or [Reset] to clear the entered details.

  8. Click [Query] to search for the existing records.

The standard stock quantity set is taken as a base to maintain the stock details in the respective source. Based on the value selected for ‘Allow linen more than std. stock qty.’ field in the ‘Setup Parameters for Facility’ function, you can add items to the stock during transactions.
