Enter the following details:

 

  1. Linen Item – Select the Linen Item using [?] function. Linen items for which standard stocks are defined in ‘Standard Stock for Sources’ function get listed. The requests placed by a Nursing Unit/Clinic/Department for the respective linen item are displayed.

  2. Cleaning Date - The system date and time are displayed, by default. Click the Calendar function to enter the Date when the linen is cleaned. This is a mandatory field.

  3. Cleaning Done By - Enter the name of the person in alphabetical format of 30 characters maximum length. This indicates the name of the person responsible for cleaning the linen items. The User Name is displayed, by default. This is a mandatory field.

  4. Remark – Click Remark to enter remarks, if any. Click [OK] to save a remark and exit the window.
