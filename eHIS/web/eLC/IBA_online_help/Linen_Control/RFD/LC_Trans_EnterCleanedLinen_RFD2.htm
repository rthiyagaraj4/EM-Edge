Enter the following details:

 

  1. Linen Item – The name of the cleaned linen item is displayed.

  2. Pending Qty. – The quantity of the linen item pending for cleaning is displayed.

  3. Cleaned Qty. – Enter the quantity of linen item cleaned against the request received from Nursing Unit/Clinic/Department. The pending quantity is displayed, by default.

  4. Quality Passed – Select the check box, if the linen item has passed the quality check after cleaning. On selecting the check box, the fields ‘Nature of Defect’ and ‘Reason For Defect’ are disabled. Clear the check box to record any defects or complaints for the cleaned linen items. You can select the reason from the list.

  5. Nature Of Defect – Select a value from the list. This indicates the nature of defect of the linen item. Reasons defined with the Action Type as ‘Nature of Defect’ in ‘Reason/Complaint Codes’ function get listed for selection.

  6. Reason For Defect – Select a value from the list. This indicates the reason for defect. Reasons defined with the Action Type as ‘Reason for Loaning’ or ‘Reasons for Quality Failure’ in ‘Reason/Complaint Codes’ function get listed for selection.
