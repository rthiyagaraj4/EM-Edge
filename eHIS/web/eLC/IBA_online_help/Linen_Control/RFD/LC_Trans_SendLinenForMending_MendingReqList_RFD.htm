Enter the following details:

 

  1. Request Date – System calculates and displays the time and date seven days prior to the system date. Click the Calendar function to select Request Date. The requests placed during the selected date range are listed. This is a mandatory field.

  2. To - The current System Date and time is displayed as the ‘To’ Request Date. System calculates and displays the time and date seven days prior to the system date. This is a mandatory field.

  3. Agency Code – Select an Agency Code using [?] function. Agency Codes are defined in ‘Agency Codes’ function. The Agencies to which mending requests are placed get listed.

  4. Request Status – All Pending Requests’ is displayed, by default. Select a value from the list. The system-defined values are ‘All’, ‘All Pending Requests’, ‘Outstanding’, ‘Partially Issued’, ‘Issued’ and ‘Cancelled’.

  5. All – Select ‘All’ to view all the requests placed. The requests, which are outstanding, partially issued, issued and cancelled, are listed.

  6. All Pending Requests – Select ‘All Pending Requests’ to view the list of pending requests. The requests, which are currently pending are displayed.

  7. Outstanding – Select ‘Outstanding’ to view the list of outstanding requests. The requests, which are currently outstanding, are displayed.

  8. Partially Issued – Select ‘Partially Issued’ to display the list of requests for which the linen items are partially issued by the laundry.

  9. Issued – Select ‘Issued’ to display the list of requests for which the linen items are issued by the laundry.

  10. Cancelled – Select ‘Cancelled’ to display the list of requests for which the linen item requests are cancelled.

  1. Linen Item – Select a Linen Item using [?] function. Linen Items are defined in ‘Linen Item’ function. Select the linen items for which mending requests are placed.
