Mending Requests List

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to view or cancel the requests placed for mending linen items. The Requests placed are displayed in an ascending order based on the Request Number generated.

  3. Click Request Number displayed as a hyperlink to ‘Cancel’ or ‘View’ the Request placed for mending linen items.

  4. Click Cancel hyperlink to cancel the Request placed for mending linen items. Request Number, Request Date, Raised By and Request Status are displayed. Under the ‘Linen Item’ heading, linen item, requested quantity, issued quantity, pending quantity and remark are displayed.

    Enter the following details:

    1. Cancelled By – Enter the name of the person responsible for cancelling the Request placed for mending linen items.

    2. Cancelled Date/Time – The system date and time is displayed, by default. You cannot edit the Cancelled Date and Time. It indicates the date and time when the linen items are cancelled.

    3. Cancelled Reason – Select the reason using [?] function to cancel the request placed for mending linen items. Reasons associated with the Action Type ‘Cancel Reason’ in ‘Reason/Complaint Codes’ function get listed for selection.

    4. [Cancel] - Click [Cancel] to cancel the request placed for mending linen items.

  5. Select ‘View’ hyperlink to view the request placed for mending linen items. You can view the Request Number, Request Date, Raised By and Request Status. The Linen Items, requested quantity, issued quantity, pending quantity and remarks are displayed. If the linen items are outstanding, the ‘Request Status’ is displayed as ‘OS’. The ‘Request Status’ is displayed as ‘CN’, if the mending request is cancelled.
