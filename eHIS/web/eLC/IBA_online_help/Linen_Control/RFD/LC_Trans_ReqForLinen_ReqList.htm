Requests List

The linen requests placed using the ‘Place Linen Request’ are displayed based on the search criteria. You can either cancel or view the request placed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to view and modify all the Requests List placed from the selected Source Type. The Requests placed are displayed in ascending order of the Request Number.

  3. Click ‘Request Number’ displayed as a hyperlink to ‘Cancel’ or ‘View’ the request placed for a new linen.

  4. Click ‘Cancel’ hyperlink to cancel the request placed for new linen. Request Number, Request Date, Source Type, Source, Raised By and Request Status are displayed based on the details recorded using the ‘Place Linen Request’ tab. The Linen Item, Requested Quantity, Issued Quantity, Pending Quantity and the Remarks are displayed.

  5. Enter a Reason to cancel the request placed.

  6. The User name is displayed, by default. Enter the name of the person responsible for cancelling the Request placed for new linen.

  7. Select the reason for cancelling the request placed for new linen using [?] function. The reasons associated with the Action Type ‘Cancel Reason’ in ‘Reason/Complaint Codes’ function get listed for selection.

  8. The system date and time is displayed. You cannot edit the Cancelled Date and Time.

  9. Click [Cancel] to cancel the request placed for the new linen.

  10. Click ‘View’ hyperlink to view the details of the request placed for new linen. You can view the details, such as, Request Number, Request Date, Source Type, Source, Raised By and Request Status. The Linen Item, requested quantity, issued quantity, pending quantity and remark are displayed in a tabular format.
