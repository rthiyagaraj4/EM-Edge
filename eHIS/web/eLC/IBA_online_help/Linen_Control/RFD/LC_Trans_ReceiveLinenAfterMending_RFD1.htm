Enter the following details to record the linen item details received after mending:

 

  1. To - The current System Date is displayed as the ‘To’ Request Date. Linen mending takes place for all requests placed within the ‘From’ and ‘To’ Request Date.

  2. Raised By - Enter the name of the person who raised a request for mending linen. The User Name is displayed as the person handing over the linen, by default.

  3. Request Status – ‘All Pending Requests’ is displayed, by default. Select a value from the list. The system-defined values are ‘All’, ‘All Pending Requests’, ‘Outstanding’, ‘Partially Issued’, ‘Issued’ and ‘Cancelled’.

  4. All – Select ‘All’ to view all the requests placed. The requests, which are outstanding, partially issued, issued and cancelled, are listed.

  5. All Pending Requests – Select ‘All Pending Requests’ to view the list of pending requests. The requests, which are currently pending are displayed.

  6. Outstanding – Select ‘Outstanding’ to view the list of outstanding requests. The requests, which are currently outstanding, are displayed.

  7. Partially Issued – Select ‘Partially Issued’ to display the list of requests for which the linen items are partially issued by the laundry.

  8. Issued – Select ‘Issued’ to display the list of requests for which the linen items are issued by the laundry.

  9. Cancelled – Select ‘Cancelled’ to display the list of requests for which the linen item requests are cancelled.

  1. Linen Item – Select a Linen Item using [?] function. Linen Items are defined in ‘Linen Item’ function.

  2. [Select] - Click [Select] to display a Request placed for mending the linen. The selected Linen Item is displayed along with the sent quantity, received quantity, current received quantity and remarks. Linen Item and Remarks are displayed as hyperlinks. On clicking Linen Item, the selected Linen Item and its requested quantity is displayed in the respective fields for editing.

  3. Remark - Click Remark hyperlink to view the entered remark. To edit remark, select the Linen Item hyperlink and edit in the respective ‘Remark’ field. Click [OK].

  4. [Cancel] - Click [Cancel] to remove the selected linen item from the table. You cannot select the same linen item while placing a request for mending linen under one Request Number.

  5. [Submit] - Click [Submit] to acknowledge the receipt of linen items after mending. You get a message ‘Please note your Request Number’. Click [OK] and the control shifts back.
