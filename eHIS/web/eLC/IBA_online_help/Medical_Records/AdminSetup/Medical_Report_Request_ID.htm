Medical Report Request ID

From this function, you can Define the numbering sequence to be followed while creating Medical Report Request. You Query for existing records.

This function allows you to define the numbering sequence to be followed for Medical Report Request ID generation.

In Medical Report Request screen, the Medical Request ID gets generated based on Current Year and Next Serial No. The length of the Request Id is based on the Year and Max serial No.

Defining Medical Report Request ID

From MR Setup > MR Admin Setup, select the Medical Report Request ID function.

The Medical Report Request ID screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the define numbering sequence or [Reset] to clear the entered data.

  4. Click [Query] to query about existing records.
