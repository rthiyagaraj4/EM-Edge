Transmit Notifiable Disease Report

From this function you can Record the details related to the communication of a Notifiable Disease that has been recorded for a patient to the relevant health authority.

This function lists out all the encounters (both active and closed) for the selected patient. You can view the notifiable diagnoses that have been recorded for the patient by clicking the Encounter ID hyperlink.

Transmitting Notifiable Disease Report

From the MR Transactions > Notifiable Diseases, select the Transmit Notifiable Disease Report function.

The Transmit Notifiable Disease Report Screen gets displayed.

  1. Enter the Patient ID and click [Search] to display the encounter recorded with a Notifiable disease for the patient or enter the search criteria and click [Search] to display the required encounter.

    1. + - On clicking the hyperlink '+', the current ‘Encounter Details' screen gets displayed. The Encounter Details and Referral Details are displayed. The Stay Details are also displayed if the selected encounter is an inpatient encounter.

    2. Encounter Date - The Date of the encounter is displayed for the selected patient.

    3. Encounter Type - The Type of encounter the patient had is displayed.

    4. Encounter ID - The Encounter ID of the patient is displayed. This field is hyperlinked.

    5. Location Type - The Location Type in which the patient is treated is displayed.

    6. Location - The Location in which the patient is treated is displayed.

    7. Specialty - The Specialty under which the patient is treated is displayed.

    8. Room - The Room that the patient has occupied is displayed.

    9. Bed - The Bed occupied by the patient is displayed.

    10. Practitioner - The name of the Practitioner who is treating the patient is displayed.

  2. Select the Encounter ID hyperlink to display the Transmit Notifiable Disease Report screen. The screen displays the Diagnosis details recorded as Notifiable.

    The following details are displayed:

    1. Onset Date - The Onset Date as entered while recording the diagnosis is displayed.

    2. Code - The Diagnosis Code is displayed.

    3. Diagnosis - The Diagnosis description that has been recorded is displayed. This is a hyperlinked field.

    4. Scheme - The Scheme used to record diagnosis is displayed based on the scheme chosen in MR Parameter.

    5. Primary - If there is a ‘Tick’ sign against the description it means that this has been recorded as the Primary diagnosis and if there is a ’Cross’ sign against the description, it means that this is not the Primary diagnosis.

    6. Significant - If there is a ’Tick’ sign against the description it means that this has been recorded as the significant diagnosis and if there is a ‘Cross’ sign against the description, it means that this is not the significant diagnosis.

    7. Recorded By - The name of the person who recorded the diagnosis is displayed.

    8. Recorded Date - The Date on which the diagnosis was recorded is displayed.

    The details that are displayed in the screen are based on the query criteria entered. The diagnosis is sorted based on the stage at which they have been recorded.

  3. Click the ‘Diagnosis’ hyperlink in the screen to display fields to record Transmission details.

  4. Enter the details in the mandatory (*) fields. <Related Field Description>

  5. Click [Record] to record Transmission details or [Close] to exit from the screen.
