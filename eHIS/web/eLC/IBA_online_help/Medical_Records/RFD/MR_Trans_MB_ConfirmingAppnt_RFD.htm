The following details are displayed:

 

Patient Details

Details like the Patient Name, Age, Gender, etc. of the patient that are captured while registering the patient are displayed. The Patient ID is not displayed if the patient is not registered in the facility.

 

Request Details

The Request details as entered in the 'Requester Details' Tab of the Medical Board main screen is displayed and disabled. The fields cannot be updated.

  1. Click [Date Details] to view the Request Date, Application Completion Date and Forwarded Date. Click [Close] to exit from the Date Details screen.

  2. Click [Charges Details] to view the charge details of the Medical Board as recorded. The charge details are displayed as read-only text. Click [Close] to exit from the Charge Details screen.

  3. Click [Applicant Address] to view the address details of the applicant.  Click [Close] to exit from the Contact Details screen.

 

Enter the following details:

  1. Report Type - The Type of Report as selected in the 'Forward Medical Board Request' screen is displayed and disabled.

  2. Department/Specialty - The Department or Specialty as selected in the 'Forward Medical Board Request' screen is displayed and disabled.

  3. Remarks - The Remarks are displayed as recorded which can be updated.

  4. Collect Date - The Collect Date as recorded in the 'Status of Medical Board Request' screen is displayed.

  5. Forwarded Practitioner - The name of the Practitioner to whom the request is forwarded as selected in the 'Forward Medical Board Request' screen is displayed and disabled.

  6. Intimation Mode – Select the Intimation Mode of the Medical Board Request from the list of valid values. The values in the list of values are 'Personal' and 'Others'.

  7. Date Posted – Enter the date of when the Medical Board Request is posted in dd/mm/yyyy hh:mm format or select the date using the calendar function. If 'Personal' is selected as the ‘Intimation Mode', this field is disabled.

[Address] is enabled only if the ‘Intimation Mode' field is selected. On clicking this button a contact details screen gets displayed. The contact details of the patient as recorded while registering the patient is displayed. The displayed fields can be updated.

  1. Section File No – Enter the section number in numeric format of 20 digits maximum length. If 'Personal' is selected as the 'Intimation Mode', this field is disabled.

  2. Intimated To – If the 'Intimation Mode' is selected as 'Others', this field is disabled. If the 'Intimation Mode' is selected as 'Personal', the name of the applicant is displayed as recorded which can be updated.

  3. Other Alt Type - Select the Other Alt Type of the Patient from the list of valid values. This field is disabled on selecting 'Others' in the field 'Intimation Mode'.

  4. Other Alt ID - Enter the Other Alt ID of the Patient. This field is disabled on selecting 'Others' in the field 'Intimation Mode'.
