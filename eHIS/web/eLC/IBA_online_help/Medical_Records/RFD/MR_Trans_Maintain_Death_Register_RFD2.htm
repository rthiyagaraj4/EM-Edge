1. Diagnosis Set – Select the Diagnosis Set from the list box. The list box displays the Terminology Sets as defined in the Terminology Set function that is associated for a practitioner, facility and specialty.

  2. Diagnosis – Select the diagnosis from the [?] function. Click [!] corresponding to the diagnosis field to view the description of the diagnosis.

  3. Nature – Select the nature of diagnosis as Actual, Potential or Wellness. This field is mandatory.

  4. Accuracy – Select Accuracy from the list box. The Accuracies are defined in the Accuracy function in the MR Terminology Set. On selecting the Accuracy, differential Group hyperlink is displayed. Click the hyperlink to invoke Differential Group dialog. Select the Group 1 radio button or if the diagnosis is to be recorded under a new group then select New Group radio button. On selecting New Group radio button, Group 2 is displayed which can be selected. Click [Close].

  5. Severity – Select Moderate, Mild, Severe or Extreme as the severity from the list box. This field is mandatory.

  6. Priority – Select priority as Normal, High or Critical. This field is mandatory. Select ‘As Evidenced By’ hyperlink to invoke As Evidenced By dialog. The Defining Characteristics are displayed which can be selected. The characteristics are defined as displayed in ‘Factors for Terminology Code’ function. On selecting, click [OK]  or click [Close] to exit from the screen.

  7. Onset Type – Select the Onset Type as Acute or Chronic from the list box. This field is mandatory. By default, Not Applicable is displayed.

  8. Onset Date – Select the onset date of the diagnosis from the calendar function. This field is mandatory.

  9. Since – This field is updated automatically on entering the Onset Date.

  10. Status – Select the Status as Resolved, Revoked or In Error from the list box. By default, Active is displayed.

  11. As On – The current date is displayed for the status.

  12. Include in Current Encounter – This checkbox is selected and disabled which indicates that the diagnosis is included in the current encounter.

  13. Remarks – Enter any remarks if any.
