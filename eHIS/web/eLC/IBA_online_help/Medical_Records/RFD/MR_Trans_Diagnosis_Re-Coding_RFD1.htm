Enter the following details:

  1. National – Enter the National ID in numeric format of maximum length as specified in the MP Parameter or click [M] to select the National ID using the external interface.

  2. Alt ID No. – Enter the other Alt number in numeric format of maximum length as specified in the MP Parameter.

  3. Other Alt Type - Select the other Alternate type from the list. The Other Alternate Types listed include ‘Birth Certificate’, ‘PMI’, ‘Passport’, etc.

  4. Previous HN - Enter the Previous HN number in numeric format of eight characters maximum length.

  5. Government No - Enter the Government number in numeric format of 10 characters maximum length.

  6. Driving License No - Enter the Driving License number in numeric format of 10 characters maximum length.

  7. Expatriate No - Enter the Expatriate number in numeric format of 10 characters maximum length.

  8. Patient ID - Enter the patient ID in the alphanumeric format of 10 characters maximum length or select the Patient ID by clicking [?] function. This field is mandatory.

  9. Search By - Select the Search By value from the list. The listed values are My Level and All. If My Level is selected, the search result is based on the level of privilege the user holds. For instance, if the user with Level 1 privilege selects Search By My Level, the list contains those records, which are pending for level 1 recoding. If you select All, the logged in user can view all the levels of users below the logged in user. A level 3 user can view the records of Level 1, 2 and 3. If a higher level user starts recoding a lower level user’s records, the record can only be viewed for the lower level user. The user cannot make any changes to it.

  10. Encounter Type – Select Encounter Type from the Encounter Type list. The listed values are based on the check boxes selected for ‘Applicable Patient Class for Recoding’ in MR Parameter function. When the Encounter Type is selected, the Discharge Period is disabled and mandatory.

  11. Discharge Period – The Discharge Period is disabled on selecting an Encounter Type.

  12. Specialty - Select the Specialty from the list.

  13. Location - Enter the Location or select the Location using [?] function. Based on the Operator Station setup, the Locations are displayed for selection.

  14. Encounter ID – Enter the Encounter ID in alphanumeric format of 12 characters maximum length.

  15. Order By – Select the order in which the records have to be displayed. The listed values are ‘Encounter Date’ and ‘Encounter Type’.

  16. Status - Based on the 'Search By' value selected, the status values are displayed and the field becomes mandatory. If 'Search By' value selected is 'My Level', the status listed for selection are 'Pending Review', 'Partially Completed', 'Completed' and 'Authorized'. If 'Search By' value selected is 'All', the value 'Completed' is not listed for selection as all the completed records of each level is posted as pending review to the next level.
