The following details are displayed:

 

Request Details

  1. Registration No - The request ID is displayed as the Registration Number.

  2. Registration Date - The Date and Time of placing the Medical Board request is displayed.

  3. Report Type - The Report Type is displayed.

  4. Requested Practitioner - The Practitioner expected to prepare the Medical Report is displayed.

  5. Type of Medical Board - The Medical Board Type selected while placing the request for the patient is displayed.

  6. Click [Date Details] to view the Request Date and the Application Completion Date. Click [Close] to exit from the Date Details screen.

  7. Click [Charges Details] to view the charge details as recorded. The charge details are displayed as read-only text. Click [Close] to exit from the Charge Details screen.

 

Medical Board Members Formation

  1. Medical Board - Select the required Medical Board from the list of valid values. The listed values are populated from the 'Medical Board' function of MR Code Setup. This field is mandatory.

  2. Appt Date - Enter the Appointment Date in dd/mm/yyyy format or select the Date from the calendar function. This field is mandatory if the ‘Formation Status’ field is selected as Complete.

  3. Member Type - Select the Type of members of the Medical Board from the list. The listed values are Internal and External. If Internal is selected, either User or Practitioner can be the members of the Medical Board. If External is selected, the Practitioner is displayed and disabled as the member. This field is mandatory.

  4. Name - Select the Name of the Member Type selected for the Medical Board using [?] function. This field is mandatory.

  5. Role - Select the Role of the member from the list of valid values. The listed values are Chief and Member that are predefined. This field is mandatory.

There can be only one chief in a Medical Board and a minimum of two other members. On selecting more than one member as 'Chief', an alert message gets displayed.

  1. Formation Status - Select the status of formation of the members of the Medical Board as 'Complete' or 'Incomplete'.

On entering the required details, click [Add]. A screen gets displayed with the following details:

  1. Member Type - The type of the member is displayed.

  2. Name - The name of the member is displayed.

  3. Role - The role of the member is selected as 'Chief' or 'Member'.

  4. Select - Select the check box to include the member in the Medical Board. The check box is selected, by default. To exclude a member, clear the check box corresponding to the chief/member.
