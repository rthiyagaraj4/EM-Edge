Enter the following details to query the diagnosis recoding audit details:

  1. Patient ID - Enter the Patient ID in the alphanumeric format of 10 characters maximum length, or select the Patient ID by clicking [?] function.

  2. Location - Select the Location as either Clinic or Nursing Unit from the list. Click [?] function to display the locations based on selection.

  3. Re-coded By - Select the name of the person who recorded the diagnosis details using [?] function.

  4. Orig Diag Code - Select the original diagnosis code using [?] function.

  5. Diag Stage - Select the diagnosis stage as 'Admission', 'Discharge' or 'Intermediate'.

  6. Age (Years) - Enter the Age range in numeric format of three characters maximum length.

  7. Gender - Select the Gender from the list of values.

  8. Encounter Period - Enter the Encounter Period in dd/mm/yyyy format or select the date using the Calendar function.

  9. Encounter Type - Select the Encounter Type as Daycare, Emergency, Inpatient, Outpatient or External.

  10. Encounter ID - Enter the Encounter ID or select the Encounter ID using [?] function.
