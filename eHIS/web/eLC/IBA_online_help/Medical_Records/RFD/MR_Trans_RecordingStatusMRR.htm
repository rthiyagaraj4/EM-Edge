The following details are displayed:

 

  1. Registration No - The request ID of the Medical Report is displayed as the registration number.

  2. Registration Date/Time - The date and time of placing the request for Medical Report is displayed.

  3. Click ‘View Encounter Details’ hyperlink to view encounter details.

 

Applicant Information

The applicant information as recorded in the 'Medical Report Request' screen is displayed and disabled. The fields cannot be edited except 'Consent Letter' field.

Click [Applicant Address] to view the address details of the applicant.

Click   [Check List] to enter the check lists.

Enter the following details::

  1. Consent from Patient/ Next of Kin – Select the value from the list. The listed values are Yes, No and Not Applicable.

  2. Copy of Identification document of the requestor – Select the value from the list. The listed values are Yes, No and Not Applicable.

  3. Marital certificate of the spouse – Select the value from the list. The listed values are Yes, No and Not Applicable.

  4. Statutory Declaration - Select the value from the list. The listed values are Yes, No and Not Applicable.

  5. Consent from Embassy – Select the value from the list. The listed values are Yes, No and Not Applicable.

  6. Payment (Autolink from Billing) - Select the value from the list. The listed values are Yes, No and Not Applicable.

 

Charges

  1. Charges Applicable - Select Yes or No from the list indicating whether charges are applicable to the applicant for the medical report of the patient. This field displays the value if set in the Medical Report Request screen.  

  2. Amount – This field displays charges if set in the Medical Report Request screen and cannot be modified. Enter the amount to be charged in numeric format of 15 characters maximum length. This field is enabled only if the 'Charges Applicable' field is selected as 'Yes'.

  3. Receipt No - This field displays the Receipt No if set in the Medical Report Request screen. Enter the number of the receipt in alphanumeric format of 15 characters maximum length. This field is enabled only if the 'Charges Applicable' field is selected as 'Yes'.

  4. Receipt Date - The Receipt Date as recorded is displayed. Select the date of the receipt using the calendar function. This field is enabled only if the 'Charges Applicable' field is selected as 'Yes'.

  5. Additional Charges - Select Yes or No from the list indicating whether any additional charges are applicable to the applicant for the Medical Report of the patient. This field displays the value if set in the Medical Report Request screen.  

  6. Amount - This field displays charges if set in the Medical Report Request screen and cannot be modified. Enter the amount to be charged in numeric format of 15 characters maximum length. This field is enabled only if the 'Additional Charges' field is selected as 'Yes'.

  7. Receipt No - This field displays the Receipt No if set in the Medical Report Request screen. Enter the number of the receipt in alphanumeric format of 15 characters maximum length. This field is enabled only if the 'Additional Charges' field is selected as 'Yes'.

  8. Receipt Date – The Receipt Date as recorded is displayed. Select the date of the receipt using the calendar function. This field is enabled only if the 'Additional Charges' field is selected as 'Yes'.

 

Enter the following details:

  1. Application Status - Select the status of the application as 'Complete' or 'Incomplete'. The listed values are predefined. This field is mandatory. On selecting the value as 'Complete', the 'Application Completion Date' is displayed as the current system date and time in read only text. The status of the Medical Report is displayed as 'Status Completed'.

If you select the ‘Application Status’ field as ‘Complete’, then you need to record the Check List values to do further transaction. Else, an error message “Checklist is incomplete. Cannot mark Application Status as Complete” gets displayed.

 

  1. Collect Date - Select the date on which the medical report can be collected using the calendar function. This field becomes mandatory on selecting the 'Application Status' as 'Complete' and is disabled on selecting the 'Application Status' as 'Incomplete'.

If  ‘Max No of days for Report Completion’ field is entered in MR Parameter, then when  the ‘Application Status’ is changed as ‘Complete’, the system displays the Collect Date based on the system date and entered parameter value. For example, if the ‘Max No of days for Report Completion’ parameter value is set as "2", the system displays the value as Today+2 days in the Collect Date field.

  1. Requested Practitioner - Enter the requested practitioner in alphanumeric format of 30 characters maximum length or select the practitioner from [?] function.

  2. Required Documents - Enter the details of the documents required for the medical report in alphanumeric format of 200 characters maximum length.

Click [Previous Delivered Medical Report] in the Medical Report Request screen to view the details of Medical Reports already delivered for the patient.
