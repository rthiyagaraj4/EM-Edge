Enter the details to record User Access Rights:

 

  1. User - Select the User Name from [?] function. This field is mandatory.

  2. Facility - Select the facility from the list of valid values. This field is mandatory. The list values displayed correspond to the facility(s) the user has access.

 

Sensitive Diagnosis

To enable the selected User to view or record the Sensitive Diagnosis for Consulting Patient/Other Patient, select the appropriate check boxes:

 

Consulting Patient

  1. Record - Select the check box to provide Access Rights to the User to record Sensitive Diagnosis details of the Consulting Patient.

  2. View - Select the check box to provide Access Rights to the User to view the Sensitive Diagnosis details of the Consulting Patient.

 

Other Patient

  1. Record - Select the check box to provide Access Rights to the User to record Sensitive Diagnosis details of the Other Patient.

  2. View - Select the check box to provide access rights to the User to view the Sensitive Diagnosis details of the Other Patient.

  3. Password - Enter the Password to be used for recording or viewing the Sensitive Diagnosis. This Password can be different from Logging in Password to the application. The password entered is masked for security reasons. This field is mandatory. (On selecting one of the check boxes in Consulting Patient/Other Patient, the mandatory indicator appears '*')

  4. Confirm Password - Re-enter the Password to confirm. This field is mandatory.

National ID

  1. Change National ID - When the check box is selected the ‘National ID’ field in Change Patient Details function is enabled for you to edit else it is disabled.

 

You would be able to change the National ID if the ‘Change allowed for series’ field is selected as ‘All Series’ in MP Parameter.

 

MyHIX

  1. View MyHIX – Select the check box to provide Access Rights to the User to view the MyHIX details of the Patient.

 

Transfer

  1. Cancel Transfer In – Select the check box to provide Rights to the User to access the Cancel Transfer function under IP Transactions.
