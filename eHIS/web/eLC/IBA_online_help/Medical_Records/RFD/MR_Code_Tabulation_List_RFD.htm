Enter the following details to create a Tabulation List record:

  1. Code - Enter a Tabulation List Code in alphanumeric format of four characters maximum length. A Tabulation List code is used to identify the Tabulation List. The Code has to be unique and cannot be changed after creation. This field is mandatory.

  2. Long Description - Enter the long description of the Tabulation List Code in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description - Enter the short description of the Tabulation List Code in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Diagnosis Code Range - Enter the Diagnosis Code Range in alphanumeric format of 50 characters maximum length.

  5. Diagnosis Coding Scheme - Select the Diagnosis Coding Scheme from the list. The values displayed correspond to the Medical Records Parameter Setup. This field is mandatory.

  6. Tabulation List Type - Select the Tabulation List Type from the list. The listed values determine whether the tabulation list comprises of the codes used for recording Diagnosis or Procedures.

  7. Enabled - Select the check box to enable the record or clear the check box to disable the record.
