Enter the following details:

  1. Cross Reference Type - Select the Cross Reference Type from the list. The listed values are Equivalent Code, Dagger and External Cause. This is a mandatory field.

  2. Terminology Set – Select the Terminology Set from the list of valid values. The listed values are defined in the ‘Terminology Set’ function. This field is mandatory. On selecting the Terminology Set, the Terminology Codes are displayed.

  3. Search – Enter the required text and select the required search criteria from the list of valid values. The listed values are 'Starts With', 'Ends With' and 'Contains'.

  4. Term Code - Enter the Term Code in alphanumeric format.

Click [Search] to display the desired Terminology Codes based on the entered search criteria.

The first level of the Terminology Codes is displayed, by default. More option to search for terminology sets are provided in alphabetical order. On clicking any of the alphabet hyperlinks, the various Terminology Codes are displayed and they are hyperlinked.

Click the Description hyperlink and select the Equivalent Term Set.

  1. Equivalent Term Set – Select the Terminology Set from the list of valid values. The listed values are defined in the ‘Terminology Set’ function. This field is mandatory. On selecting the Terminology Set, the Terminology Codes are displayed. If the Cross Reference Type is selected as Dagger or External Cause, the Equivalent Term Set field is disabled.

  2. Search – Enter the required text and select the required search criteria from the list of valid values. The listed values are 'Starts With', 'Ends With' and 'Contains'. Click [Search] to display the desired Terminology Codes based on the entered search criteria.

The first level of the Terminology Codes is displayed, by default. More option to search for terminology sets are provided in alphabetical order. On clicking any of the alphabet hyperlinks, the various Terminology Codes are displayed and they are hyperlinked.

  1. Select - Select the check box corresponding to the required Terminology Code to associate it to the Terminology Set selected on the top part of the screen.
