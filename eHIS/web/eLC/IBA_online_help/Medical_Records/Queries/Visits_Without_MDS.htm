Visits Without MDS

From this function you can Query patient visits for which Minimum Data Set is not recorded.

MDS for Patient Class defines the type of diagnosis scheme applicable as Minimum Data Set for various types of encounters.

Querying Visits without MDS

From MR Queries, select the Visits Without MDS function.

The ‘Visits Without MDS’ screen gets displayed.

  1. Enter the search criteria and click [Search] to display Patient Visits without MDS. <Related Field Description>

  2. Click [Clear] to clear the records displayed in the result area.

  3. Click [Reset] to reset the entered details.
