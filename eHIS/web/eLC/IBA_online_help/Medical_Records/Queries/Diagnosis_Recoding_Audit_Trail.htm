Diagnosis Recoding Audit Trail

From this function you can Query all the changes done to the original diagnosis details of the patient.

The Diagnosis Recoding Audit Trail query function lists the date and name of the person who recorded the changes to the diagnosis details.

Querying for Diagnosis Recoding Audit Trail

From the MR Queries, select the Diagnosis Recoding Audit Trail function.

The Diagnosis Recorded Audit Trail screen gets displayed.

  1. Enter the search criteria to select a specific record. <Related Field Description>

  2. Click [Search] to display the records or [Clear] to clear the records displayed in the result area.

  3. Click Encounter ID hyperlink to view the Encounter Details. The Stay details are displayed in the case of an Inpatient and the Encounter details are displayed in the case of an Outpatient, Emergency, External or Daycare patient.

  4. Click [Reset] to clear the entered data.
