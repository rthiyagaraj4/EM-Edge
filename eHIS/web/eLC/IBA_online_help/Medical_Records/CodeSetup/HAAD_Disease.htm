HAAD Disease

From this function, you can Assign colour to the HAAD disease and map the HAAD disease with the diagnosis code.

This function allows you to assign colour to the HAAD disease and to map the HAAD disease with the diagnosis code. You have the provision to choose the colour code from the colour palette.

 

Only when HAAD Disease check box is selected in Patient Banner/Encounter Context Banner screen in MP Admin Setup and diagnosis mapped with HAAD disease, the assigned HAAD disease colour code gets displayed in the Patient Banner or Encounter banner label.

Assigning/Modifying Diagnosis against HAAD Disease

From MR Code Setup, Code Setup select the HAAD Disease function.

The HAAD Disease screen gets displayed.

  1. By default, the HAAD Disease Code, Description and Colour Code appear. The Colour Code box is displayed as white.

  2. Click the Colour Code box against HAAD Disease and choose the required colour from colour palette. Refer Colour Coding for Identification of Infectious disease

    DISEASE

    COLOUR CODE

    COLOR

    Hepatitis B

    Yellow Dot

      

    Hepatitis C

    Yellow Dot

      

    HIV 

    Red Dot

     

    MRSA-Methicillin Resistant Staphylococcus Aureus

    Blue Dot

     

    MDRO-Multi Drug-Resistant organism

    Green Dot

     

     

  3. Click [Apply] to save the changes done.

  4. To modify the diagnosis against HAAD disease, click the Editable sign . The Term Code and Term Set fields appear and the Editable sign changes into a tick mark  .

  5. Select the required Term set from the drop-down list and select the Term code from the lookup.

  6. Click [Add] and then [Apply] to save the changes made. If you try to link already mapped diagnosis with same/another HAAD disease, then message “Diagnosis is already mapped with disease (disease code name)” gets displayed. There by, restricts you to link mapped diagnosis with same/another HAAD disease.

  7. To delete the diagnosis mapped with HAAD disease, click  near the corresponding Term Set and Term Code. The message “Do you want to Delete the Record?” gets displayed.

  8. Click [OK] and then [Apply] to delete the diagnosis.

  9. Click [Reset] to clear the entered data.
