Recode Privilege Setup

From this function you can Define the privilege setup for recoding and Query for existing privilege setup records.

From this function you can restrict the recoding at multiple levels at user level in various facilities and the privilege to authorize can differ.

Recoding Privilege Setup

From MR Code Setup > Recode Setup, select the Recode Privilege Setup function.

The Recode Privilege Setup screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Select] to add the User to the screen. Click User name hyperlink to modify the selected User or [Cancel] to cancel recording the User.

  4. Click [Apply] to save the defined Privilege Setup or [Reset] to clear the entered data.

  5. Click [Query] to search for existing Privilege Setup.
