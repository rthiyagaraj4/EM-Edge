Allergen

From this function you can Create the actual agent causing an allergy or adverse event and Query for an existing allergen record.

The Allergen causing the Adverse Event is defined using this function by associating. The Allergen is associated to the Reaction Type as created in the Adverse Event Type master function.

Creating an Allergen

From MR Code Setup, select the Allergen function.

The Allergen screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the data in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Allergen created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Allergen records.
