ICD Code

From this function you can Query for a desired ICD code based on the nature of the code whether it is an Underlying cause of the disease or Manifestation of the cause.

The International Classification of Diseases (ICD 9 or ICD 10) code provides the ground rules for codification of Diseases/Procedures while recording Patient Diagnoses/Procedures.

Querying for an ICD Code

From MR Code Setup, select the ICD Code function.

The ICD Code screen gets displayed.

  1. Click [Query] to search for existing ICD Codes.

  2. Enter the query criteria. <Related Field Description>

  3. Click [Search] after entering the query criteria to select the records that satisfy the query criteria.

  4. Click the required ICD Code hyperlink to view details of the ICD Code. The ICD screen is displayed with ICD Code details some of which can be modified.

    The ICD Code details are defined and you cannot edit the values. The details that can be modified are:

    1. Code Type - Select the value from the list. The listed values are Disease and Procedure. This value determines whether the code is applicable during the recording of a diagnosis or procedure.

    2. Gender Specific - Select the value from the list. The listed values are Common, Male and Female. The value entered here limits the code to the selected gender.

    3. Code Indicator - Select the value from the list. The listed values are Dagger, External Cause, Asterisk and Consequence. The value selected determines the nature of the Code, whether it is used to denote an underlying disease, a manifestation, an external cause or a consequence.

    4. Age Specific - Select the value from the list. The listed values are All, Adult, Common, Maternity, New Born, Pediatric and Others.. The value entered in this field limits the code to the Age Group selected.

    5. Long Title - Enter the Long Title in alphanumeric format of 250 characters maximum length.

    6. Sensitive - Select the check box to indicate the ICD code as Sensitive or clear it to indicate as Non-Sensitive.

    7. Notifiable - Select the check box to indicate the ICD code as Notifiable or clear it to indicate as Non-Notifiable.

  5. Click [Apply] to save the modification performed on entering the required details. Click [Cancel] to cancel the modification process and close the ICD Code details screen or [Reset] to clear the entered data.

 

It is recommended that the licensed data pertaining to these codes be procured from an authorized supplier and loaded into the database before commencement of operations. This prevents the need to create or alter data in this master.
