Examination History

From this function you can View all the details of Examinations performed on a patient..

The details like the Drugs/Contrast Agents used for an examination, Exposure Factor details, Review Notes entered by various Reporters/Radiologists, Consumables used for the Examination, the number of repeats for the Examination, the User involved in conducting the examination, Audit details, details of Follow Up Examinations are displayed. You can view the status of an Examination within the Radiology Department. You can retrieve the patient’s old relevant radiology reports across encounters through a keyword search.

Viewing the Examination History of a Patient

From the Queries > Result Reporting, double-click Examination History function.

The Examination History screen gets displayed.

  1. Enter the Patient ID in the length as specified in the MP Parameter or click ‘Right Click for User Options’ and click [Standard Patient Lookup] or [External Patient Lookup] to select the Patient. The Sex and Age is displayed on selecting the patient. This is a mandatory field.

  2. On entering the Patient ID, click [Search] or [Execute Query] or press [F8] to display the examination details of the patient or enter the search criteria and click [Search] or [Execute Query] or press [F8] to display the examination details of the patient.

    Enter the following query criteria:

    1. Facility ID – The logged in Facility is displayed, by default. Select the Facility from the list.

    2. Section – Enter the Section Code in alphanumeric format of four characters maximum length or click [List of Values] to select the Section. On selecting the Section Code, the description of the Section Code is displayed.

    3. Exam – Enter the Exam Code in alphanumeric format of 10 characters maximum length or click [List of Values] to select the Exam . On selecting the Exam Code, the description of the Exam Code is displayed.

    4. Normal/Abnormal – Select the Report type from the list. The listed values are Abnormal, Normal or Critically Abnormal.

    5. Organ Site – Enter the Organ Site in alphanumeric format of six characters maximum length or click [List of Values] to select the Organ Site. The Sites defined in the Anatomical Site function in the ‘Application Masters’ module is displayed for selection. On selecting the Organ Site code, the Organ Site description is displayed.

    6. Radiologist – Enter the Radiologist code in alphanumeric format of 15 characters maximum length or click [List of Values] to select the Radiologist who performed the operation. On Selecting the Radiologist code, the Radiologist name is displayed.

    7. Status – Select the examination status from the list. The system defined values are All, Registered, Exam Started, Exam Completed, Reported – On hold, Reported – Prelim, Reported – Final, Auto Reported, Report Modified and Cancelled.

    8. Date [From-To] – Enter the date range in the format of dd/mm/yyyy or select the required date using the Calendar function.

    9. Primary Diagnosis Code - Enter the Primary Diagnosis Code in alphanumeric format of 60 characters maximum length or click [List of Values] to select the Primary Diagnosis Code for the Examination. The values are listed based on the 'Diagnosis Coding Scheme' setup in RD Parameters.

    If 'User Defined' is selected as the 'Diagnosis Coding Scheme' in RD Parameters, the Diagnosis Codes recorded in Diagnosis Codes function is listed. If 'ICD 10' is selected as the 'Diagnosis Coding Scheme' in RD Parameters, the ICD 10 Diagnosis codes are retrieved from the configured Code Set.If 'ICD 9CM' is selected as the 'Diagnosis Coding Scheme' in RD Parameters, the ICD 9CM Diagnosis codes are retrieved from the configured Code Set.

    1. Exam Performed Date - Enter the date in DD-MON-YYYY format or click the Calendar function to select the date when the exam was performed. The exams performed within the date range specified is listed.

    2. Adverse Reactions – Select a value from the list. The system-defined values are Both, Not Recorded and Recorded.

    3. Not Recorded – Select ‘Not Recorded’ to display all the Examination History in which the Adverse Reactions are not recorded.

    4. Recorded – Select ‘Recorded’ to display all the Examination History where the Adverse Reactions are recorded.

    5. Both – Select ‘Both’ to display all the Examination History reports.

    1. Result Keyword – Enter the Result Keyword in alphanumeric format of 100 characters maximum length. The system displays the Examination History Reports of patients with the queried Result Keyword.

    => You cannot enter single quotes in the free text for the Result Keyword.

    => If the Result Keyword is in double quotes, the system checks for the text including double quotes and filters only the text with double quotes in the reports.

  3. Click to view the Examination Details screen with Reason for proceeding duplicate exams. You can view the protocols assigned to the examination and the comments entered by the Radiologist and Technologist, if protocols are assigned to the examination in the Master setup.

  4. Click to view the image captured during the examination. You can view the image if the PACS application is interfaced with the Enterprise Management application.

  5. The details of the patient’s examination in the department is displayed under Report  Text, Complications and Other Exam Reported tabs respectively.

    In the Other Exam Reported tab, you can view the related examination reports.

    1. Examination Code – The code of the examination is displayed.

    2. Examination Description – The description of the examination is displayed.

    In the Complications tab, you can view the complications reported during/after the examination.

    1. Complication Type - The Complication Type of the examination is displayed.

    2. Management Type  -  The Management Type of the complication is displayed.

    3. Remarks – The remarks (if any) is displayed.

    You can view the examination report details:

    1. Report Text - The details of the Report Status of the examination is displayed.

    2. Result Status – The Result Status of the examination is displayed.

    3. Primary Diagnosis Code – The Primary Diagnosis Code of the examination is displayed.

    4. View Report – Click [View Report], the ‘View Clinical Notes’ screen gets displayed where you can view the report details of the patient.
