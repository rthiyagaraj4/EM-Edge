Add to Library

From this function you can Send the exam reports to the library.

The Add to Library function enables you to send the exam reports to the Library from where the records can be Viewed or reviewed by all the Radiologists in the facility.

Sending Exam Results to Library

From the Transactions > Special Interest Library, double-click Add to Library function.

The Add to Library screen gets displayed.

  1. Click [Search] to display the Exam details of the patients or enter the Search Criteria and click [Search] to display the specific Examination record.

    Enter the following criteria:

    1. Patient ID – Enter the Patient ID in the format as specified in the MP Parameter or right click ‘Right Click for User Options’ and select the Patient ID using Standard Patient Lookup option.

    2. Exam – Enter the Exam code of the report or click [List of Values] to select the Exam. The report pertaining to this exam is sent to the library.

    3. Section – Enter the Section code or click [List of Values] to select the Section, where the exam was conducted.

    4. Radiologist ID – Enter the Radiologist ID or click [List of Values] to select the Radiologist who performed the examination.

    5. Registration No. – Enter the Registration Number of the examination.

    6. Registration Date [From-To] – Enter the date range of the registration in the format of dd/mm/yyyy or select the date range using the Calendar function.

  2. On searching, the exam details are displayed based on the entered criteria. The displayed details are Patient ID, Name, Exam, Organ Site, Section, Radiologist, Registration Number and Registration Date.

  3. Double-click on the Patient ID to invoke ‘ Add to Library’ screen.

  4. Click [Save] to send the reports to library or [Clear Record] to clear the entered details.
