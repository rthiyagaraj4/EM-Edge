Authorise/Review by Radiologist

From this function you can Review or Authorize the pending examination reports and Query for the pending reports.

The Authorise/Review by Radiologist function enables the radiologist to review/authorize examination reports that are marked pending for authorization. The reports that is not marked, as ‘Auto Report’ has to be authorized by a radiologist. The reviewing radiologist can make necessary corrections in the entered Report Text, Diagnosis/Analysis Codes entered and finally authorize the Result Report. If the reviewing radiologist considers a need for a review by another radiologist, the report can be sent for further review. After the review, if the report is marked ‘Finalized’, the report is added to the main library from where the report can be accessed and viewed/analyzed (editing/modifying the report is not permitted) by other radiologists. A Radiologist can view the number of reports pending for authorization, pending for review, awaiting review and reviewed by other Radiologists.

Authorizing or Reviewing the reports by a Radiologist

From the Transactions > Result Reporting, double-click Authorise/Review by Radiologist by Reporter function.

The Authorise by Radiologist screen gets displayed.

  1. The pending examination results, which are authorized by a Reporter in the Performing Facility and assigned to the Radiologist, are listed. Enter the search criteria and click [Search] to search for a specific patient record. Right-click on the ‘Right Click for User Options’ function to search for a patient record using the ‘Standard Patient Lookup’ or the ‘External Patient Lookup’ screen.

  2. Click [Search] after entering the search criteria. The search button now is displayed as [As of (current time)]. Based on the search criteria, list of examinations, which are completed and reported are displayed under ‘Work List’. Details like the exam description, laterality, contrast type, status, registration number, registration date and time, the attending radiologist, section, wing, ordering facility, episode type, doctor and study location are displayed.

  3. Select the ‘Select All’ check box against the patient record in the Work List to assign the patients to different radiologists. This check box is visible based on the privileges assigned to the radiologist in the Access Rights for Assigning Radiologist function.

  4. Double-click the patient ID and the ‘Authorise by Radiologist’ screen gets displayed.

A warning message “Technologists comments are available against this Exam” is displayed when a patient is selected from the Work List. This warning message is displayed only if the Technologists comments are recorded for the exam. This warning message does not stop the workflow.

  1. Click [Patient History] to view the clinical events of the patient.

  2. Click [Examination History] or [Prior Exams] to view the examination details. Examination History screen is displayed listing the patient movement details. The patient details are displayed under ‘Exam Details’, ‘ Reviewer Notes’, ‘Relative Follow Up Investigation’, ‘Audit’, ‘Patient Movement’ tabs. You can also view the Report Text recorded for each examination. The details can only be viewed.

    Request for Review

    If the radiologist wants another review of the report, select the ‘Request for Review’ tab and enter the details.

    1. Reason – Enter the reason for forwarding the report for authorization.

    2. With authorization – Select the check box if the radiologist is authorizing the report and forwarding it to another radiologist for authorization.

    3. One – Select the radio button, if only one radiologist is authorizing the report.

    4. Many – Select the radio button, if multiple radiologists can authorize the report.

    5. To be reviewed by – If multiple radiologists are selected and if any radiologist’s review is pending for authorization, enter the name the radiologist who has to review the report.

    6. [Confirm] – Click [Confirm] to authorize the report. .

  3. Click [Relevant Clinical Summary] to view the clinical comments entered by the radiologist at the time of performing the examination.

  4. Click [ICD10-PCS Coding] to view the procedure codes recorded for the patient during examination.

  5. The details of the patient’s examination report in the department are displayed under Report Text, Complications, Technologist Comments and other details tabs.  

    This tab displays the name of the Radiologist who has conducted the examination, record the technologist comments while preparing the initial report and the views for the examination and reason for proceeding duplicate exams .

    Enter the following details:

    1. Enter the report text in alphanumeric format in the provided space.

    2. [View Image] - If Picture Archiving and Communication System (PACS) application is interfaced to the Enterprise Management application, [View Image] is enabled under the Report Text tab. Click [View Image] to view the image captured during the examination.

    3. [Enter/Edit Result] - Click [Enter/Edit Result] to open the clinical notes of the patient to edit the test results.

    4. [Spell Check] – Click [Spell Check] to check the spelling errors in the entered report.

    5. [Report With] – There is a provision to report multiple exams together, that is if multiple exams exist for a request or multiple requests, exist for a patient, other exams is shown in the ‘Report With’ block. To report these exams along with the selected exam, click the [Report With], which will show all the exams for the above request in a separate window. Select the exams to report together and click [OK].

    • The Report With button is enabled in the Authorise by Radiologist screen to group exams with the reports generated from the Record Clinical Notes screen in the Clinician Access  module. Once the report is signed in the in Record Clinical Notes screen, the value ‘Finalized’ is displayed and disabled in the Type field in the Authorise/Review by Radiologist screen. You cannot edit the date.

    • A warning message “Technologists comments are available against this Exam” is displayed when you click Report With button and select an exam having technologists’ comments. This warning message does not stop the workflow.

    1. Primary Diagnosis – Select the primary diagnosis from the list. The Diagnosis codes are defined in the Result Reporting function.

    2. Result Status – Select the result status from the list. The listed values are Normal, Abnormal and Critically Abnormal.

    3. Follow-up Investigation – Select the check box if the patient requires follow investigations to be conducted.

    4. Confidential – Select the check box to mark the report as ‘Confidential’. Once the report is marked as confidential, only users having rights is allowed to view this report. The users with rights to access confidential reports are defined in the ‘Users for Viewing Confidential Reports’ function.

  6. Click ‘ ACR Coding’ tab to record the anatomic and pathologic conditions of the patient.  

    Enter the following details:

    1. Anatomic Code/Description – Click [List of Values] to select the anatomic code. The description of the selected code is displayed in the adjacent field.

    2. Pathologic Code/Description – Click [List of Values] to select the pathologic code. The description of the selected code is displayed in the adjacent field.

    The Anatomic codes and the Pathologic codes are created in the ‘ACR codes’ function in the Result Reporting masters function.

    1. Remarks – Enter necessary remarks. This is a mandatory field.

    2. ACR Coding Completed – Select the check box to mark the ACR Coding as Complete for the respective examination.

  7. Click ‘Add to Library’ to add report to the main library from where it can be accessed by other Radiologists.

  8. Select a Type from the list. The system-defined values are ‘Hold’, ‘Prelim’ and ‘Finalize’. Select ‘Hold’, if the examination report is not confirmed. Select ‘Prelim’, if the authorization report is in the preliminary stages. Select ‘Finalize’, to authorize and finalize the examination reports. You cannot change the status from ‘Hold’ to ‘Prelim’, but you can finalize a report for which the status is either ‘Hold’ or ‘Prelim’.

  9. Select the check box ‘Print’ to print examination report. The destination printer to print the report has to be defined. On selecting the check box, the report is sent to the destination printer for printing.

  10. Click [Save] to authorize the completed reports or click [Clear Record] to clear the entered data.

  11. Click [Execute Query] or press [F8] to search for a reported exam.

An error message ‘Not allowed to use this function’ is displayed on accessing the function, if the user is not a Radiologist. The color indicators for Priority [P], Allergy Indicator [A], Mobile Request [M] and Recalled Patient [R] are displayed in the bottom of the Ordered Exams/Appointments screen.
