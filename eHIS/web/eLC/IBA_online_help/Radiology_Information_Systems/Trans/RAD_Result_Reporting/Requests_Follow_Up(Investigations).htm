Requests Follow Up (Investigations)

From this function you can Record the remarks for follow-up investigations.

The Requests Follow Up (Investigations) function enables you to record remarks for the follow up investigation requested by the Radiologist.  

Recording Requests for Follow Up (Investigations)

From the Transactions > Result Reporting, double-click Requests Follow Up (Investigations) function.

The Requests Follow Up (Investigations) screen gets displayed.

  1. The examinations for which the follow-up letters are not collected by the patient are listed. Enter the search criteria and click [Search] to search for a specific patient record. Right-click on the ‘Right Click for User Options’ function to search for a patient record using the ‘Standard Patient Lookup’ screen.

    Enter the following search criteria:

    1. Registration Date [From-To] – Click the Calendar function to select the date range. Examination reports with the registration date matching the date range specified get listed.

    2. Section –  Click [List of Values] to select a Section where the examination is booked. The values defined using the ‘Sections’ function get listed.

    3. Exam – Click [List of Values] to select the examination code for which the follow up investigations are requested.

    4. Radiologist – Select the Radiologist who is responsible for the conducting the examination.

    5. Registration No –  Enter the registration number in alphanumeric format of 10 digits maximum length. The examination reports matching the specified Registration Number gt listed.   

    6. Status – Select the current status of the exam request. The options are Outstanding, Pending, Reviewed and All.

    7. Patient ID – Enter the Patient ID for whom the exam is booked. Right-click ‘Right Click for User Options’ to select the Patient ID from the standard patient lookup.

  2. Click [Search] after entering the search criteria. Based on the search criteria, Patient ID, Name, Examination, Registration number and date, Radiologist, Section, Wing, Status and Ordering Facility are listed.

  3. Right-click on the Patient ID to select either ‘Patient History’, ‘Examination History’ or ‘Review’ option. On selecting ‘Patient History’,the clinical events of the patient is displayed. Click ‘Examination History’ to view the examination details of the patient. Click ‘Review’ to get the exam report, the list of complications during/after the examination (if any), the ACR codes and the review notes entered by the authorizing/reviewing radiologist.

  4. Click [Save] to record the remarks or click [Clear Record] to clear the entered data.
