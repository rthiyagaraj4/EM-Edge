Direct Reporting

From this function you can Prepare exam result immediately after the examination of the patient in the Radiology department.

The Direct Reporting function enables the Radiologist to generate the examination report immediately after the exam has been conducted. This is mainly used in emergency cases. A Wing has to be associated to the Workstation.

Direct Reporting

From the Transactions > Result Reporting, double-click Direct Reporting function.

The Direct Reporting screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

Examinations

Examinations

Enter the following details:

  1. Select – Select the check box to report the conducted examination.

  2. Allergic Indicator – Click [Allergic Indicator] to view the recorded patient allergies. You can also add other allergy details of the patient.

  3. Instruction – Click [Instruction] to view the recorded instructions.

  4. Exam Code – Click [List of Values] to select the exam code.

  5. Description – The description of the selected code is displayed.

  6. Laterality – Select the laterality from the list. The pre-defined values are Right, Left and Bilateral.

  7. Exam Date – Select the exam date on when the examination was conducted.

  8. Amount – Enter the amount charged for the examination.

  9. [Edit Charge] – Click [Edit Charge] to edit the charges of the examination (if any).

Views

Views

Enter the following details:

  1. Exam Code – Enter the exam code for which the report is being prepared.

  2. View Code – Select the applicable view code for the examination from the list of values.

  3. Description – The description for the view is displayed by the system.

Exam Completion

Exam Completion

Enter the following details:

  1. Exam Code – Enter the Exam code or select it from the list of values. The list of values gets populated with the examinations created in the ‘Examinations’ function.

  2. Start/End Date Time – Enter the exam start and end date/time. This is a mandatory field.

  3. Radiologist ID – Select the Radiologist ID from the list of values.

  4. Storage Location – Select the storage location for the report that is being prepared. The list of values gets populated with the Storage locations created in the ‘Records Storage’ function.

  5. Mobile Request – Select the check box if examination was performed in a mobile study location.

  6. Study Location – Enter the code of the mobile study location. Among all the study locations, only a few of them are marked as mobile study locations. Select the required code from the [List of Values].

  7. Media ID – Enter the Media ID on which the report is stored.

  8. Scan No – Enter the scan number.

  9. Media Type – Select the media type used for the storage of the record. Click [List of Values] to select the required media type. The list of values gets populated with the media types created in the ‘Records Storage’ function.

  10. No. of Repeats – Select the number of repeats ordered for the examination.

  11. Auto Report – Select the check box to indicate that the examination has been marked for auto report.

  12. Use same folder for all Exams – Select the check box to indicate that all the exams, which have been directly reported, is stored in the same folder.

  13. Items Used – Click [Item Used] for the system to display a list of the consumables used during the examination.

Contrast Drug Admn

Contrast/Drug Admin

Enter the following details:

  1. Exam Code – The Exam code and description is displayed.

  2. Contrasts Agent/Drug – Select the contrast agent/drug from the list of values.

  3. Normal Qty, Vol – The normal quantity of the drug that is generally used during the examinations is displayed.

  4. Actual Qty, Vol Administered – Enter the actual quantity administered during this particular examination.

  5. Strength/Qty, Vol – The strength per volume of the drug is displayed.

  6. Total Strength – Enter the total strength of the contrast agent/drug.

  7. Batch ID – Enter the Batch ID of the drug/contrasts agent. This is a mandatory field.

  8. Stock Qty – The system displays the available quantity of the drug in the stock.

  9. Route Code – Select the Route Code (the way by which the drug has been administered into the body of the patient) from the list of values. The route for a particular drug is set in the Contrast Agents/drugs set up in the Radiology module but the basic setup for the drugs is done in the Drug Setup in the Pharmacy Management module. This is a mandatory field.

  10. Administered By – Enter the name of the practitioner, who has administered the drug to the patient.

  11. Admin Reaction Code – Select the Admin Reaction Code from the list of values. The Administration Reaction Codes are created in the ‘Examinations’ function and the list of values get populated with the codes created.

Involved Personnel

Involved Personnel

Enter the following details:

  1. Exam Code – The exam code is displayed by the system.

  2. Type – Select the required Practitioner Type from the list.

  3. ID – Select the Practitioner ID from the list of values.

  4. Name – The name of the Practitioner is displayed as soon as the Practitioner ID is selected.

Report Text

Report Text

Enter the following details:

  1. Exam Code – The exam code is displayed by the system.

  2. Report Text – Click the list of values to select a suitable report text template.

  3. Type – Select the suitable option from the list. The options are Prelim, Hold and Finalize.

  4. Print Report – Select the check box to print the report at the designated printer.

  5. Primary Diagnosis – Select the primary diagnosis form the list of values.

  6. Normal/Abnormal – Select if the report is ‘Abnormal’, ‘Normal’ or ‘Critically Abnormal’.

  7. Confidential – Select the check box to mark the report as confidential.

  8. Enter/Edit Result – If you have selected the exam report from the already set report templates, click [Enter/Edit Result] to make modifications in the report.

  9. Spell Check – Click [Spell Check] to check for spelling errors in the report text.

Complication

Complications

Enter the following details:

  1. Exam Code – The exam code is displayed by the system.

  2. Complication Type/Description – Select the complication type from the list of values. The Complication Types and the Complication Management Types are created in the ‘Examinations’ function of the Radiology module.

  3. Complication Management Type/Description – Select the complication management type form the list of values.

  4. Remarks – Enter remarks (if any).

  1. Click [Patient History] to view the details of the previous examinations performed on the patient.

  2. Click [Request Comments] to view the clinical comments entered by the radiologist while performing the examination.

  3. Click [Save] to prepare the exam report or click [Clear Record] to clear the entered data.
