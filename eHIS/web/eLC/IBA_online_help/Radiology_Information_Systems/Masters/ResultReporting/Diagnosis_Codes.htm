Diagnosis Codes

From this function you can Create Diagnosis Codes and Query for an existing Diagnosis Code.

You can define the Diagnosis Coding Scheme based on ICD 10, ICD 9CM or User-Defined values based on the RD Parameter setup. Diagnosis codes defined in User-defined Diagnosis Coding Scheme is included in the reports. You can view these diagnosis codes in ‘Authorize/Review by Radiologist’, ‘Report/Authorize Unreported Exams’, ‘Direct Reporting’ and ‘Modify Authorized Report’ functions.

Creating a Diagnosis Code

From the Masters > Result Reporting, double-click Diagnosis Code function.

The Diagnosis Code screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [Save] or press [F10] to save a created Diagnosis Code or click [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for the existing records.

If the mandatory values are not entered, the system alerts you to enter them. The mandatory field is highlighted with a unique color code. Only 14 records can be added at a time. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate codes with a unique color code.
