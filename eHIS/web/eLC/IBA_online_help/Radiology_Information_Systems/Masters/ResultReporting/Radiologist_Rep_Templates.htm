Radiologist Reporting Templates

From this function you can Create a template for generating the Radiologist reports.

A Reporting Template is used for generating the same report for different patients. Radiologists use these templates while reporting a radiology examination. Reports used in common can be stored as a free text template or a word processor template. You can define templates based on Exam, Section or Multiple exams and associate each template to a Radiologist. Any number of templates can be associated to an Exam and a Radiologist. You can view these templates in ‘Authorise/Review by Radiologist’, ‘Report/Authorize Unreported Exams’ and ‘Direct Reporting’ functions in Radiology Transactions.

Creating a Radiologist Reporting Template

From the Masters > Result Reporting, double-click Radiologist Reporting Templates function.

The Radiologist Reporting Templates screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. The Free Text Templates tab is displayed, by default. Enter the details in the mandatory (*) fields in the following tab pages.

Free Text Templates

This tab gets enabled when ‘Template By’ is selected as either ‘Exam’ or ‘Section’.

  1. Code – Enter the code for the Template in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Long Description – Enter the long description of the Template in alphanumeric format of 100 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description of the Template in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Text Block – Enter the Text Block in alphanumeric format of not more than 2000 characters maximum length. This is a mandatory field.

Mutli Exam Template

This tab gets enabled on selecting ‘Template By’ as ‘Multiple Exam’. The report is saved and linked to the exam.

  1. Code – Enter the code for the Multi Exam Template in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Long Desc – Enter the long description of the Multi Exam Template in alphanumeric format of 100 characters maximum length. This is a mandatory field.

  3. Short Desc – Enter the short description of the Multi Exam Template in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Text Block – Enter the Text Block in alphanumeric format of 4000 characters maximum length. This is a mandatory field.

  5. Exam Code – Enter the Exam code or click [List of Values] to select the Exam code. The Exams are defined in ‘Examination Codes’ function under ‘Examinations’ in Master Menu. This is a mandatory field.

  6. Short Description – The description of the selected examination is displayed.

  1. 3. Click [Save] or press [F10] to save a created Radiologist Reporting Template or click [Clear Record] to clear the entered data.
