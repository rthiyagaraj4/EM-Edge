Assessment/Follow up Codes

From this function you can Create the Assessment or Follow up codes and Query for an existing Assessment or Follow up Code.

Assessment/Follow up Codes is used to define the assessment or follow ups for a patient who has undergone an examination. On creating the Assessment groups, you can create the Assessment Category and the Follow up Types. Assessment groups are used at result reporting stage. The reports are classified based on assessment groups. The assessment group to which a report belongs to is an indicative of the exam report. You can view these assessments or follow up codes in ‘Letter Formats’ function.

Creating an Assessment/Follow up Code

From the Masters > Result Reporting, double-click Assessment/Follow up Code function.

The Assessment/Follow up Code screen gets displayed with two tabs. Assessment/Follow up Codes tab is enabled, by default.

  1. Enter the details in the mandatory (*) fields in the following tabs.

Assessment Groups

Assessment/Follow up Codes

  1. Click [Save] or press [F10] to save a created record or click [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records.
