Movement Locations

From this function you can Create the various locations in a facility to where the records are moved for further investigations and Query for the required movement location record.

The Movement Location function enables you to create the various locations in a facility where the records are likely to be moved/shifted to for further investigations. You have to associate the movement location with a source type. Source is the original location from where the record is shifted to another location.

Creating a Movement Location

From the Masters > Records Storage, double-click Movement Locations function.

The Movement Locations screen gets displayed with two tabs. Movement Location tab is enabled, by default.

  1. Enter the details in the mandatory (*) fields in the following tabs.

Movement Location

Enter the following details to create the Movement Location Details:

  1. Movement Location – Enter the Movement Location code in alphanumeric format of 18 characters maximum length. This is a mandatory field.

  2. Long Description – Enter the long description of the Movement Location in alphanumeric format of 90 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description of the Movement Location in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Source Type – Select the Source Type applicable to the Movement Location from the list. The system-defined values are Clinic, Ward, Department, Physician, Patient and External. This is a mandatory field.

Movement Location By Source

Enter the following to create Movement Location Details:

  1. Source Type – Select the Source Type from the list. The system-defined values are Clinic, Ward, Department, Physician, Patient and External. This is a mandatory field.

  2. Source Code – Enter the Source Code or click [List of Values] to select the Source Code. This is a mandatory field.

  3. Short Description – The short description of the selected source is displayed.

  4. Movement Location – Enter the Movement Location or click [List of Values] to select the Movement Location. The description of the Movement Location is displayed.

  1. Click [Save] or press [F10] to save a created Movement Location or click [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records.
