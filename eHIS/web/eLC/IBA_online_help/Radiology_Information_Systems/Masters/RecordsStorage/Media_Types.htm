Media Types

From this function you can Create various Media Types and Query for the existing Media Types.

The defined Media Types are used for storing the Radiology records. Some of the examples of the Media Types are Magnetic Disks, Diskettes, Compact Discs and so on. You can associate a Media Type to an examination.

Creating a Media Type

From the Masters > Records Storage, double click Media Types function.

The Media Types screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Save] or press [F10] to save the Media Type record or click [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for existing Media Types recorded.

If the mandatory values are not entered, the system alerts you to enter them. The mandatory field is highlighted with a unique color code. Only 16 records can be added at a time. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate codes with a unique color code.
