Sections

From this function you can Create Sections in the Radiology department and Query for the existing Section records.

Section is a portion or a division. Sections are named based on the modalities (equipments) used and the specific radiology exams conducted. A Section Code is assigned to each section to uniquely identify a functional section. You can associate each Section with a service, order type and a checklist. If the Section has images to be stored in digital format, you can indicate while creating the Sections.

Each section can have different Study Locations or Rooms. Exam Codes are associated to the Sections indicating the rooms where the exams are conducted. You can view the Sections while associating examinations, contrast agent drugs/consumables for examination, study locations, users for section, booking access rights and departmental authorization. You can define Radiologist report text templates based on the Sections.

Creating a Section

From the Masters > Examinations, double click Sections function.

The Sections screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Descriptions>

  2. Click [Save] or press [F10] to save a created Section or click [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search for an existing record.

If the mandatory values are not entered, the system alerts you to enter them. The mandatory field is highlighted with a unique color code. Only 15 records can be added at a time. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate codes with a unique color code.
