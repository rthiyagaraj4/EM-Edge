Users for Section

From this function you can Create an association of User to the Section and Query for the existing users for section record.

The Users for Section function enables you to specify the users for a particular Section.

Defining Users for Section

From the Administration > Access Privileges, double-click Users for Section function.

The Users for Section screen gets displayed.

  1. Enter the Section Code or click [List of Values] to select the Section Code. On selecting the Section code, the name of the Section is displayed.

  2. Enter the User ID or click [List of Values] to select the User to associate to the Section. On selecting the User ID, the name of the User is displayed.

  3. Click [Save] or press [F10] to define users for section or click [Clear Record] to clear the entered details.

  4. Click [Execute Query] or press [F8] to search for the existing users for section record.
