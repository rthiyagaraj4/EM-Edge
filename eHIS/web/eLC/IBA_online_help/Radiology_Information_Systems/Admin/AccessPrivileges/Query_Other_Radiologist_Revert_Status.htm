Query Other Radiologist/Revert Status

From this function you can Define rights to radiologist to query other radiologist's records in Worklist and to revert exam status.

This function is used to define access rights to the radiologists so that they can query other radiologist's records in Authorise/Review by Radiologist Worklist function. Also, allows to give access rights to the user to revert the exam status from Started to Registered, Completed to Started, and Completed to Registered in Record Exam Start/Completion function.

Defining Access Rights for Querying Other Radiologist Records and Reverting Exam Status

From the Administration > Access Privileges > double-click Query other Radiologist/Revert Status function.

The Query other Radiologist/Revert Status screen gets displayed.

  1. Enter the details in the fields. <Related Field Description>

  2. Click [Save], or press [F10] to define radiologist with assigning rights or click [Clear Record] to clear the entered details.

  3. Click [Execute Query] or press [F8] to search for the existing radiologist with assigning rights.
