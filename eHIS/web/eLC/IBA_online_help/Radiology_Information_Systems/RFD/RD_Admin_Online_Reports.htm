Enter the following details:

  1. Report Name – Enter the report name or click [List of Values] to select the Report Name. This is a mandatory field.

  2. Operating Facility ID – Select the Operating Facility from the list. The Selected report is applicable to the selected facility. This is a mandatory field.

  3. Section Type – Select the Section Type from the list. The listed values are Ward, Clinic, Referral and Section.

  4. Section Code – Enter the Section code or click [List of Values] to select the Section. The Sections are displayed based on the selected Section Type.  This is a mandatory field.

  5. Description – The description of the selected Section code is displayed.

  6. Print – Select the check box to indicate that the report can be printed at the selected Facility/Section.
