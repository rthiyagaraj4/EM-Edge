Enter the following details to Reprint the Exam Labels:

  1. Registration Number - Enter the Registration Number and click [F8] to display the Registration Date, Section/Wing, Appointment Reference Number, Order Number, Priority, Ordering Facility ID, Episode No, Patient ID and Name, Age and Gender of the patient, Ref Source, Doctor, Exam code and description, Ordering Facility.

  2. Required - Select the check box to if reprint the examination label is required.

  3. Print – Select the check box to reprint the examination label printed during exam registration.

  4. No. of Copies – Enter the number of copies required to reprint in numeric format of three characters maximum length.
