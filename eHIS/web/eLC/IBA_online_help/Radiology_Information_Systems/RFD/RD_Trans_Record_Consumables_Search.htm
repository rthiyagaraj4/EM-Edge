Enter the following search criteria:

  1. Wing – Enter the Wing code or click [List of Values] to select a Wing. The name of the wing is displayed. The Operating Unit/Wings are defined in the ‘Operating Units/Wings’ function under the Administration Menu.

  2. Section – Enter the Section code or click [List of Values] to select a Section. Sections are displayed based on the selected ‘Operating Unit/Wing’. Sections are associated to the Operating Unit/Wing in the ‘Sections for Operating Unit/Wing’ function under the Administration Menu.

  3. Study Location – Enter the Study Location code or click [List of Values] to select a Study Location code. The name of the Study Location is displayed.

  4. Patient ID – Enter the Patient ID in the length as specified in the MP Parameter or click ‘Right Click for User Options’ to select the patient using Standard Patient Lookup function. On selecting the Patient ID, the name of the patient is displayed.

  5. [T] – Click [T] to view the location information of the patient.

  6. Status – Select the status of the examination from the list. The listed values are All, Pending, Registered, Exam Started and Exam Completed. The value ‘Pending’ is displayed, by default.

  7. Episode Type - Select the Episode Type from the list. The listed values are Inpatient, Outpatient, Referral, External and All. If the patient is an Inpatient, the Episode Type is ‘Inpatient’; if the patient is an Outpatient, the Episode Type is ‘Outpatient’; if the patient has been referred from another facility, the Episode Type is ‘Referral’. The value ‘All’ is displayed, by default.

  8. Exam Start Date [From-To] – Enter the Examination Start date range in the format of dd/mm/yyyy or select the date range using the Calendar function.
