The received folder details are displayed. The displayed details are Received By User, Outstanding from Facility, Received Date and Time, Narration, Receiving Facility, Received Location Type and Received Record Location.

  1. Outstanding Loc Type/Location – Select the Outstanding Location Type from the list. The listed values are ‘Clinic’, ‘Ward’, ‘Department’, ‘Physician’, ‘Radiology’, ‘Patient’ and ‘External’. Enter the Location code or click [List of Values] to select the Location. Based on the Location type selected, the Location values are populated. The Outstanding Location is where the record is currently present and this record must be returned to the original Study Location.

  2. Outstanding Record Location – Enter the Outstanding Record Location or click [List of Values] to select the outstanding record location.

 

Folder Details

  1. Folder No – The folder number that is issued out from the Study Location on request is displayed.

  2. Patient ID – The Patient ID is displayed.

  3. Name – On selecting the Patient ID, the name of the patient is displayed.

  4. Confirm Receipt – Select the ‘Confirm Receipt’ check box to confirm the receipt of the issued folder.
