The Assessment Groups tab has three tabs, namely Assessment Categories, Sections for Assessment Groups and Assessment Groups.

Enter the following details:

  1. Code – Enter the code for the Assessment Group in alphanumeric format of 12 characters maximum length. This is a mandatory field.

  2. Description – Enter the description of the Assessment Group in alphanumeric format of 120 characters maximum length. This is a mandatory field.

Enter the following details:

  1. Assessment Group – Click [List Values] to select the Assessment Group for which the sections are created. This is a mandatory field.

  2. Section Code/Description – Examinations associated to sections and marked as assessment or follow-up coding required get listed for selection.

  3. Associate – Select the check box to associate the section with the Assessment Group.

Enter the following details:

  1. Assessment Group – Click [List Values] to select the Assessment Group for which the categories are created.

  2. Code – Enter the code for the Assessment Category in alphanumeric format of 12 characters maximum length. This is a mandatory field.

  3. Description – Enter the description of the Assessment Category in alphanumeric format of 120 characters maximum length. This is a mandatory field.

  4. Normal/Abnormal – Select the value from the list if the Assessment Category is applicable for ‘Normal’, ‘Abnormal’ or ‘Critically Abnormal’ reports.

Click [Associated/Follow up Criteria] to get to the Assessment/Follow-up Codes screen.
