Enter the following details:

  1. Performing facility – Select the Performing facility from this drop down list. The performing facilities defined in the radiology masters are listed. You can search the exams within the selected Performing Facility

  2. Ordering Facility - Select the Ordering facility from this drop down list. The ordering facilities defined in the radiology masters are listed. You can search the exams within the selected ordering Facility.

  3. Patient ID – Enter the Patient ID in the length as specified in the MP Parameter or right click ‘Right click for User Options’ hyperlink and select the Patient ID using Standard Patient Lookup function.

  4. Sex – On selecting the Patient ID, the gender of the patient is displayed.

  5. Exam – Enter the Examination or click [List of Values] to select the exam.

  6. Exam Category – Enter the Exam Category code or click [List of Values] to select the Exam Category.

  7. Organ – Enter the Organ code or click [List of Values] to select the organ on which the examination was performed.

  8. Added to Library By – Enter the name of the person responsible for adding the record to the library or click [List of Values] to select the name of the person responsible for adding the record to the library.

  9. Reason – Enter the Reason code or click [List of Values] to select the Reason for adding the record to the library.

  10. Reg. No – Enter the Registration Number of the examination.

  11. Registration Date [From-To] – Enter the date range of the examination in the format of dd/mm/yyyy or select the Registration Date using the Calendar function.

  12. Diagnosis – Enter the Diagnosis code or click [List of Values] to select the Diagnosis.

  13. ACR Codes – Enter the ACR Code or click [List of Values] to select the ACR Code.

  14. Remove All – Select this check box to choose an exam to be removed from the library.

  15. Click [Remove] button to remove the selected exams.
