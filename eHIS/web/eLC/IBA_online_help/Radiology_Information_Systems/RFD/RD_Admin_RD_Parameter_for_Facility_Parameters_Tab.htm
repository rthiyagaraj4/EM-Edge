Enter the following details:

 

Folder Numbering

  1. Patient ID as Folder No. - Select the check box to generate the Folder Number same as the Patient ID.

  2. Folder No. Length – Enter the length of the Folder Number in numeric format of two digits maximum length.

  3. Prefix Facility ID – Select the check box, if the Facility ID is prefixed in the Folder Numbering.

  4. Prefix Storage Location – Select the check box, if the Storage Location is prefixed in the Folder Numbering.

  5. Storage Prefix Length – Enter the Storage Prefix Length in numeric format of two digits maximum length.

 

Transaction Numbering

  1. Next Folder Request Serial – Enter the next folder request serial number in numeric format of five digits maximum length. The Folder Request Number is maintained by the Ward, Clinic, Practitioner or Department for the receiving folders from the Radiology Department.

  2. Next Folder Movement Serial – Enter the next folder movement serial number in numeric format of five digits maximum length. The Folder Movement Number is maintained by the Radiology Department for the issuing folders to the Wards, Clinic, Practitioner or Department.

 

Auto Receipt Rules

  1. Ward – Select the check box, to acknowledge the receipt of the folder in a Ward automatically.

  2. Clinic – Select the check box, to acknowledge the receipt of the folder in a Clinic automatically.

  3. Practitioner – Select the check box to acknowledge the receipt of the folder by a Practitioner automatically.

  4. Department - Select the check box to acknowledge the receipt of the folder in a Department automatically.

 

Auto Return Rules

  1. Ward – Select the check box to acknowledge the return of the folder by a Ward to the Radiology Department automatically.

  2. Clinic – Select the check box to acknowledge the return of the folder by a Clinic to the Radiology Department automatically.

  3. Practitioner – Select the check box to acknowledge the return of the folder by a Practitioner to the Radiology Department automatically.

  4. Department - Select the check box to acknowledge the return of the folder in a Department to the Radiology Department automatically.

  5. Default Narration for Record Return – Enter the Narration code in alphanumeric format of four characters maximum length or click [List of Values] to select a default narration for Record Return. The value set here is displayed as default for all the narrations, which relate to the return of record.

  6. Print Tracer Card on Issue – Select the check box to print the tracer card on issue of the record.

 

Default Locations

  1. Inactive Folder Location – Enter the Storage Location code in alphanumeric format of four characters maximum length or click [List of Values] to select a default Inactive Folder Location. The value set in this field is displayed as the Inactive Folder Storage Location.

  2. Lost Folder Location – Enter the Storage Location code in alphanumeric format of four characters maximum length or click [List of Values] to select a default Lost Folder Location. The value set in this field is displayed as the Lost Folder Storage Location.
