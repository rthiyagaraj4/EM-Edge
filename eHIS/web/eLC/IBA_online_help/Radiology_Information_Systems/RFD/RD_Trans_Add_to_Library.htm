The exam details are displayed on selecting the patient.

Click [Patient History] to view the history of the patient and [Examination History] to view the Examination history of the patient. The Result of the examination is displayed in the ‘Result’ tab; If the Reports are associated to the ACR Codes, it is displayed in the ‘ACR Codes’ tab; if the complications are recorded, it is displayed in the ‘Complications’ tab.

  1. Enter the details in the mandatory (*) fields of Add to Library tab. Add to Library tab is displayed, by default.

 

Add to Library

  1. Reason - Enter the Reason code or click [List of Values] to select the Reason to add the examination to the library. This is a mandatory field.

  2. Public – Select the check box, if all Radiologists can view the Examinations added to the library.

  3. Privileged Radiologist ID/Name – Enter the Radiologist ID or click [List of Values] to select the Radiologist who can view the Examination report from the Library.
