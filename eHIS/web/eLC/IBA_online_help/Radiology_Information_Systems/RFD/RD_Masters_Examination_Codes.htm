Enter the following details to create an Examination Record:

  1. Examination Code – Enter the Examination code in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Description-Long – Enter the long description for the Examination in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description for the Examination in alphanumeric format of 30 characters maximum length. The description entered in the Description-Long field is displayed, by default. This is a mandatory field.

  4. Section - Enter the Section or click [List of Values] to select the Section. The Sections are defined in the ‘Sections’ function. This is a mandatory field.

  5. Laterality Applicable – Select the check box to indicate that laterality is applicable for the examination.

  6. Organ Site – Enter the Organ Site or click [List of Values] to select the Organ Site. The description of the selected Organ Site is displayed. The Organ Sites are defined in the Anatomical Site function in Application Masters module.

  7. Media Type – Enter the Media Type where the radiology records are stored or click [List of Values] to select the Media Type. The description of the Media Type is displayed. The Media Types are defined in ‘Media Types’ function.

  8. Views Applicable – Select the check box to define views for the examination. On selecting the check box, the ‘Views’ tab is enabled.

  9. Exam Category – Enter the Exam Category or click [List of Values] to select the Exam Category. The description of the selected category is displayed. The Exam Categories are defined in ‘Exam Categories’ function. This is a mandatory field.

  10. Catalog Cross Reference – Enter the Catalog Cross Reference or click [List of Values] to select the Catalog Cross Reference value. The description of the catalog is displayed. Every radiology examination has an Order Catalog created in the ‘Order Catalog’ function of the Order Entry and Results Reporting module. The Catalog Cross Reference code is associated to one Examination code only. This is a mandatory field.

 

Patient Instructions

  1. Patient Instruction – The option defined for the order in the Order Catalog function is selected by default and disabled. The patient instructions values are - Sex/Age, Standard or Not Setup in Catalog. You can modify the selected patient instructions option in the corresponding Order Catalog only.

 

Appointment

  1. Appointment reqd – Select the check box to indicate that an appointment is required before performing the examination. While registering the examination, the system prompts a message indicating that a prior appointment is necessary for the examination. Clear the check box if appointment is not required before performing the examination.

  2. Exam Duration – Enter the time required to perform the examination in the hh:mm format. The system displays the number of slots required for the examination based on the exam duration defined, while booking the examination in the Transaction functions.

  3. Min. Exam Start Time – Enter the minimum exam start time. This refers to the minimum preparation time required for the examination. For example, the Study Locations where you perform the examination starts working at 8 am, but some time is required for preparation. The exam starts at 9 am. You are not allowed to book appointments before this time.

  4. Admission reqd – Select the check box to indicate that the patient has to be admitted in the facility for performing the examination.

  5. Drug Administration reqd – Select the check box to indicate that a drug has to be administered during/prior to the examination. If the check box is selected, while placing orders for these examinations, the ‘Drugs’ tab in ‘Contrast Agent, Drugs/Consumables for Examination’ function is enabled for the entry of necessary drugs for the examination. Otherwise, the ‘Drugs’ tab is disabled.

  6. Contrast Applicability – Select the value from the list. The listed values are Not Applicable, Optional and Required. For some exams, contrasts are not applicable and for some exams, contrasts are used to distinguish the body parts distinctly.

 

Result Reporting

  1. Auto Report – Select the check box to generate a report after completing the examination. The Report defined in the ‘Report Text’ tab gets generated automatically as the report after the examination if the check box is selected.

  2. Auto Authorize Normals – Select the check box for the system to authorize the report, if the report is classified as normal. This is used in emergency cases when a radiologist need not be called for authorizing a report.

  3. Assessment/Follow-up coding reqd – Select the check box to indicate that assessment/follow-up coding is required for this examination while reporting.

  4. Inhibit Report – Select the check box to prevent sending the examination report to clinical history.

  5. Record to MR – Select the check box to post the examination under Medical Records.

 

Workload Reporting

  1. Workload Reporting reqd – Select the check box to indicate that this examination has to be included for workload reporting. Workload Reporting is used to report the load on the Radiology wing.

  2. Units – Select the units for the workload report. The unit of measurement defined in RD Parameters gets displayed.

  3. Note Type – If user defined templates are used for reporting instead of free text; select the Note Type in this field. The Note Types are created in the Clinician Access module.

  4. Exposure Factors – Select the Exposure Factor from the list. The list of values are ‘Applicable (Mandatory)’, ‘Applicable (Optional)’ and ‘Not Applicable’. If you specify the Exposure Factor as mandatory, the system does not allow you to mark the examination as complete until the exposure factor is entered.
