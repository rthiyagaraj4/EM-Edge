Enter the following details:

  1. Storage Location – Enter the Storage Location code or click [List of Values] to select the Storage Location. The Storage Locations are defined in the ‘Records Storage’ function.

  2. User – The logged in User Name is displayed.

  3. Request Loc Type/Location – Select the Request Location Type from the list. The listed values are ‘Clinic’, ‘Ward’, ‘Department’ and ‘Physician’. On selecting the Request Location Type, enter the Location code or click [List of Values] to select the Location. Based on the Location Type selected, the Location values are populated.
