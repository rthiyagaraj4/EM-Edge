Enter the following search criteria:

  1. Perf. Facility – Select the facility where the exam is being performed from the list of values. The name of the performing facility where the exam was last conducted is displayed by default.

  2. Wing – Select a Wing from the list. The values defined in the in the ‘Operating Wings’ function.

  3. Section – Click [List of Values] to select a Section or enter the section code. The name of the section code is displayed in the adjacent field. The values defined in the ‘Examinations’ function gets populated.

  4. Registration Date [From-To] – Click the Calendar function to select the date range for the examinations for which the registrations are done.

  5. Registration No – Enter the registration number of the examination.

  6. Patient ID – Enter the patient ID in alphanumeric format of 10 digits maximum length. The name of the patient is displayed in the adjacent field.

  7. [T] – Click [T] to view the patient location information. This button is enabled only on selecting the ‘Track Patient location’ check box in the ‘RD Parameters for Facility’ function. The current patient location is displayed. You can change the location of the patient. For example, at the start of the examination, the patient location is displayed as ‘Ward’, the patient location can be changed to ‘Study location’ once the examination starts, indicating that the patient has been brought into the study location for performing the examination.

  8. Status – Select the status of the examination. The options available are – ‘All’, ‘Exam Completed’, ‘Reported-On hold’ and ‘Reported-Prelim’.

  9. Exam – Click [List of Values] to select the examination code. The examination reports are listed based on the Exam selected from the list.

  10. Episode Type – Select the Episode Type from the list. The options available are Inpatient, Outpatient, Referral, External and All.

  11. Priority – Select the priority from the list. The system-defined values are All, Stat, Urgent and Routine.

  12. Radiologist – The name of the Practitioner associated to the logged in User gets listed. You can select Radiologist from the list to view the examination reports pending for authorization.

If you do not have rights assigned in Query other Radiologist/Revert Status screen to query on other radiologists record, the error message ‘‘RD9344-You are not authorised to change the Radiologist’ gets displayed. Thereby, restricts the change of radiologist in drop-down list.

 

  1. Doctor – Enter the name of the doctor in alphanumeric format of 45 characters maximum length or click [List of Values] to select the doctor. The names of the referring doctors are displayed. On selecting the name of the doctor, the doctor ID is displayed.  The records of all patients are listed in the Work list based on the selected referring doctor.

  2. Unassigned - Select this check box to view all the patients not assigned to a radiologist. If the check box is cleared, all the assigned patients will be listed.  This check box is enabled based on the privileges assigned to the radiologist in the Access Rights for Assigning Radiologist function.

  3. Change To – Select the radiologist name from the drop-down list to reassign the patients to that radiologist. The patients can be assigned and reassigned to different radiologists until the exam report is finalized. This check box is visible based on the privileges assigned to the radiologist in the Access Rights for Assigning Radiologist function.
