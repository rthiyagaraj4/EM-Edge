Enter the following details:

 

Query other Radiology tab

  1. Radiologist ID – Select the required radiologist ID from the [?] lookup.

  2. Short Name – The short name gets populated automatically once the radiologist ID is selected.

  3. Query other Radiologist’s record in worklist – Select the check box to assign privileges to the selected radiologist to query other radiologist record in 'Authorise/Review by Radiologist Worklist' function.

  4. View other Radiologist’s Availability – Select the check box to assign access rights to radiologist to view other radiologist availability details in the ‘Record Radiologist Availability’ screen and in ‘Radiologist Available History’ screen.

 

Revert Status tab

  1. User ID – Enter the User ID or click [List of Values] to select the User.

  2. User Name - The user name gets populated automatically once the user ID is selected.

  3. Exam Started to Registered –  Select the check box to give access to the user to revert exam status from started to registered in ‘Record Exam Start/Completion Worklist’ function.

  4. Exam Completed to Started – Select the check box to give rights to the user to revert exam status from completed to started in ‘Record Exam Start/Completion Worklist’ function.

  5. Exam Completed to Registered - Select the check box to give rights to the user to revert exam status from completed to registered in ‘Record Exam Start/Completion Worklist’ function.
