Cancel OP Checkout

From this function you can Cancel patients checked out from the Outpatient location using the OP Checkout function.

Cancel OP Checkout function is used to cancel a patient checked out from the Outpatient department. This function displays a list of OP Checkout patients based on entered search criteria. Only authorized users can cancel an OP checkout based on the Operator Station Control else a message ‘you are not authorized to cancel check out’ is displayed. The information recorded during the OP checkout is defaulted and disabled in the Cancel OP Checkout screen with option to record the reason for cancelling checkout. OP Checkout cancellation is done only on the same day irrespective of the time the checkout has been recorded for the patient on that date. The name of the practitioner who checked out the patient is displayed.

 

Process Flow Diagram

Cancelling OP Checkout

From OP Transactions, select the Cancel OP Checkout function.

The Cancel OP Checkout screen gets displayed.

  1. Click [Search] to display patients checked out from the Outpatient location on the same date or enter the search criteria to select a specific checked out patient. Click [Clear] to reset records displayed in the result area.

    Enter the following query criteria

    1. Specialty - Select the Specialty from the list. The specialties created using ‘Specialty’ function under Practitioner Setup in Application Masters module with the enabled check box selected gets listed.

    2. Patient ID - Enter the required Patient ID or click [?] function to search and select the practitioner.

    3. Location Type – Select the Location Type from the list. The listed values are Clinic and Procedure Unit.

    4. Location – Enter the Location or select the location using [?] function. The locations are listed for selection based on the selected Location Type.  The locations created using the function ‘Clinic’ and ‘Procedure Unit’ in Outpatient Management module get populated.

    5. Practitioner - Select the required Practitioner using [?] function.

    6. Gender – Select the Gender from the list. The listed values are All, Male, Female and Unknown.

  2. Checkout details of patients are displayed.

  3. Click the ‘Encounter ID’ hyperlink to display the Cancel Checkout screen. The screen displays Disposition, Case Definitions and In the Event of Death details as recorded in the OP checkout function.

  4. Enter the details in mandatory (*) fields. <Related field description>

  5. Click [Cancel Checkout] to record canceling OP checkout. Click [Abort Cancel Checkout] to abort cancelling OP checkout and exit from the Cancel Checkout screen.
