Entered the following details to select the required Appointment Patient records:

 

  1. Citizen ID - Enter the Citizen ID in numeric format as entered while registration. The ‘Prompt’ specified for the Citizen ID in MP Parameter is displayed as the field legend. The length and format as specified in MP Parameter has to be adhered.

  2. Alternate ID Type(s) - Enter Alternate ID number(s) in the format and length as specified in MP Parameter. Based on the MP Parameter setup, system displays the Alternate ID Type description(s).

  3. Other Alt ID – Select the required Alternate ID from the list of values. The listed values are defined in the function ‘Alternate ID Type’ of MP module. The field ‘Other Alt ID’ is displayed if ‘Accept Other Alt ID’ is set in MP Parameter (Alt ID checks tab).

  4. Other Alt No - On selecting the required Alternate ID type, enter the ID in alphanumeric format of 20 characters maximum length.

The above field names are displayed based on the legends for Alternate ID defined in the MP Parameter.

  1. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in the MP Parameter.

  2. Gender - Select the Gender from the list of valid values. The listed values are ‘Male’, ‘Female’ and ‘Unknown’.

  3. Nationality – Select the Nationality from the list of valid values. The listed values are defined in ‘Master Patient Index’.

  4. Patient Details – The multi parts of Patient Name are the Prefix, First Name, Second Name, Third Name, Family Name and Suffix. Based on the setup in MP Parameter (Legal Names), the multi parts of patient name are displayed. The displayed titles and order of appearance are specified in the MP Parameter function.  

  5. Appt No. - Enter the Appointment Reference Number in alphanumeric format of eight characters maximum length.

  6. Location - Select the Location type from the list of valid values and Location from the [?] function. The valid values for Location Type are 'Clinic', 'Nursing Unit', 'Daycare' and 'Procedure Unit'. The listed values for ‘Location' contains values that correspond to the selected location type.

  7. Practitioner – This field is enabled on selecting the Location. Enter the practitioner name in alphanumeric format of 30 characters maximum length or click [?] function to invoke a search screen to look for the required practitioner. The displayed list is limited to the Practitioners who are associated with the Locations of the derived Operator Station.

  8. Location Type - Select a Location Type from the list. The values are 'Clinic' and 'Procedure Unit'. The value 'Clinic' is displayed, by default.

  9. Location - Select a Location using [?] function. Locations are listed based on the selected Location Type. On selecting the Location Type as 'Clinic', Clinics created using 'Clinic' function are displayed for selection. You can register the Visit to a Clinic. On selecting the Location Type as 'Procedure Unit', Procedure Units created using 'Procedure Unit' function are displayed for selection. You can register the Visit to a Procedure Unit.
