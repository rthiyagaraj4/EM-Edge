Enter the following details to define a Queue Series record:

 

  1. Queue ID – Enter the Queue ID in alphanumeric format of maximum length as specified in MP Parameter. This is a mandatory field.

  2. Description – Enter the description of in alphanumeric format of 30 characters maximum length. This is a mandatory field.

 

Applicability

  1. Walk-ins – Select the check box to allow walk-ins to be entertained in the Queue Series.

  2. Referrals – Select the check box to indicate if referrals are allowed in the Queue Series. If the check box is cleared, the Queue Series does not allow referral patients.

  3. Appointments – Select the check box to indicate if appointments are allowed in the Queue Series. If this check box is cleared, the Queue Series does not allow appointment patients.

  4. Queue Prefix – Enter the prefix of the queue in numeric format of two digits maximum length and has to be unique. This is a mandatory field.

 

Number

  1. Start - Enter the starting number of the queue in numeric format of six digits maximum length. This is a mandatory field.  

  2. Next - Displays the next queue number. When the record is created, it is set as the Start queue number and incremented whenever queue is built up.

  3. Reset Number - Enter the Reset Number in numeric format of six digits maximum length. Whenever the queue number reaches the Reset number, queue number gets initialized to the start number.

  4. Enabled - Select the check box to enable the queue in the transaction functions. Clear the check box to disable the queue. The check box is selected, by default.
