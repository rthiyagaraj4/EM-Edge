Enter the following details:

  1. Quick Admission – Select this check box if you want to proceed with quick admission process for the patient in case of emergencies, such as AE checkout. Once this is checked, the screen immediately displays the following additional fields:

 

a. Bed Class – The class of bed; general,

b. Admission Type

c. Bed Type  

 

Also, the following existing fields become mandatory.

a. Service

b. Location

c. Practitioner

  1. Referral Type – Select the Referral Type from the list. The listed types are Login Facility, Enterprise and External. This is a mandatory field.

  2. Healthcare Setting Type – Select the Healthcare Setting Type from the list to which the patient has to be referred to. This  is a mandatory field. Healthcare setting types are set up using Application Masters module. The values listed in this field are specific to healthcare setting types for IP Referral.

  3. If the Referral Type selected is ‘Enterprise’, all the healthcare setting types defined in the Healthcare Setting Type master function is displayed here except for the one associated with the logged in facility.

  4. If the Referral Type selected is ‘External’, all the healthcare setting types to which External Referral Sources associated in the External Referral Source master function are displayed here.

  1. Referred To –

  • If the Referral Type is selected as  ‘Enterprise’, the entire active facilities of the enterprise except the logged in Facility are listed.

  • If the Referred Type is selected as ‘External’, all the external referral sources are listed.

  • If the Referred Type is selected as ‘Login Facility’, the logged in facility name is made default. This field is mandatory. Place the mouse on the icon to display the long name of the External or Referral Source recorded in ‘External/Referral Source’ function of Application Masters module. The   icon appears on the screen if the long name for the External Referral Source is recorded in ‘External/Referral Source’ function of Application Masters module.

  1. Specialty – Enter the Specialty in alphanumeric format of 30 characters maximum length or click [?] function to select the Specialty. This is a mandatory field.

If Referral Type is ‘External’, the Specialty field is not mandatory.

Note:

1. If you have selected the Quick Admission check box in this page, the Patient Admission page is displayed when you click this button.  The referral details will be displayed by default, in the page.

2. On completing the admission, the billing details will be displayed in a popup window based on whether the patient is citizen or non-citizen. On clicking OK, the page from where the quick admission process started, will be displayed; for example, AE Checkout page or the OP Check out page.

3. If you are unable to complete the quick admission, the focus will return to the referral page. (See Warnings )The referral page will display the created referral ID. The column Appt No./Encounter ID  will display the hyperlink “Quick Admission”.  

4. You can open the screen anytime and click this hyperlink and complete the Quick Admission process. Once the admission is completed the corresponding encounter ID will be displayed in place of the “Quick Admission” hyperlink.

5. If you try to create a Quick Admission Referral in the Register Outgoing Referral screen for an inpatient, the following error message will be displayed on launch the Admit Patient screen: “This patient is currently an Inpatient”.

Warnings:

1. You cannot admit inpatients; the following error message will be displayed on launch the Admit Patient screen: “This patient is currently an Inpatient”.

2. You cannot admit a patient using an existing Quick Admission Referral number; the following error message will be displayed: “Invalid Referral ID”.

3. If you create a quick admission referral, but then uncheck the Quick Admission check box for the particular operation station in Operator Station setup function, you will not be able to admit the patient. The following message will be displayed:

“Operator Station not defined for Quick Admission.”

  1. Location Type – Select the Location Type from the list. The pre-defined values are listed in the list. This is a mandatory field.

  2. Location – If the Referral Type is ‘External’, the location list gets disabled and you have to specify the location in alphanumeric format of 30 characters maximum length in the text box provided. If the referral type selected is ‘Enterprise’, the Location Type and the Location can be identified, while the adjoining text box does not allow any data entry. The values for the Location Type field are ‘Clinic’, ‘Procedure Unit’ or ‘Day Care’. Depending on the Healthcare Setting Type, referred source, Specialty and Location type selected, corresponding locations are populated in the list. You can select the required value from the list. The same holds good if the referral type selected is ‘Login Facility’. This is a mandatory field.

  3. Service – Select the required Service from the list.

  4. If the Referral Type is selected as ‘Login Facility’, the services are listed based on the Specialty, Location Type and Location chosen.

  5. If the Referral Type chosen is ‘Enterprise’, the services are listed based on the Healthcare Setting Type, Referred Source, Specialty, Location Type and Location chosen.

  6. If the Referral Type chosen is ‘External’, all the services set up in the ‘Service’ master function is available in the list.

  1. Practitioner – Enter the Practitioner name or select the practitioner using the [?] function. The list values correspond to the selected ‘Referred Source’. This field is enabled only if the Referral Type is ‘Enterprise’ or ‘Login Facility’ and if the specialty is selected, or it is disabled. For the Referral type ‘External’ the practitioner name can be entered in the text box provided in alphanumeric format of 30 characters maximum length.

  2. Referred For - Select the Referred To from the list. The listed values are ‘Admit’ and ‘Booking’.

  3. Preferred Date – Enter Preferred Date in dd/mm/yyyy format or select the date from the Calendar icon. This  is a mandatory field. The user can enter Preferred Date greater than or equal to the system date. If you had checked the Quick Admission check box, the system date will be displayed here by default and you can select a different date, if needed.

  4. Reason for Referral – Enter the referral reason for the patient or select the reason from the [?] function.

  5. Priority – Select the Referral Priority from the list. The values displayed are ‘Elective’, ‘Semi Emergency’ and ‘Emergency’. This  is a mandatory field.

  6. Referral Notes – Click [Referral Notes] to open the Referral Notes dialog. Enter the Referral Notes of the patient in alphanumeric format of 1500 characters maximum length and close the Referral Notes dialog. The control shifts back to the main screen.

  7. Record Referral Letter Note – This button appears only if the ‘Enable Recording of Referral Letter Note with Referral Request’ check box is selected in IP Parameter. Click [Record Referral Letter Note] to invoke the Record Referral Letter dialog where you can record new referral letter or search and modify the existing referral letters available for the patient. This button is disabled when the ‘Referral Type’ is selected as ‘External Facility’ and it is enabled only when the ‘Referral Type’ is selected as Login Facility or Enterprise.
