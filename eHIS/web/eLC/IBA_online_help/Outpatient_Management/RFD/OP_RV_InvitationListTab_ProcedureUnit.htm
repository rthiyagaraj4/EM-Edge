Invitation List Tab

Invitation List tab is displayed if ‘Allow Visit Registration for Wait list Invitations without Appointment’ is checked in the OP Parameter. This tab displays all invited patients from the waiting list.

  1. Click [Search] to display invited patients for the current date. Alternately, enter the query criteria to search for a specific invited patient. <Related Field Description>

  2. Click ‘Waitlist Number’ hyperlink to display Invitation List Dialog.

  3. Enter a valid Invitation number.

  4. Click [OK] to populate patient details in Add Visit Screen.
