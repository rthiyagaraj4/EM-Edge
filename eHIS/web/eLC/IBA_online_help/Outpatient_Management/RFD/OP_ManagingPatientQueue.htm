Enter the following details to search the Patient Queue records, the following query criteria can be entered which are common for both Single and Multiple views:

 

  1. Date - Displays the current date. It cannot be modified.

  2. Start Time - Displays ‘00:00’ as Start Time, which means that Patients who have registered from ‘00:00’ hours onwards for the current day will be displayed. It cannot be modified.

  3. View (single/multiple) - Click on the appropriate radio button to select the view required.

  4. By Visit Type - Click on the check box to group the Patient queue records based on Visit Type.

If the user has selected Single View, following additional query criteria can be specified to restrict the Patient records display:

  1. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter.

  2. Gender – Select the required value for gender from the list box. The valid values are ‘Male’, ‘Female’ and ‘Unknown’.

  3. Encounter ID - Enter the Encounter ID in numeric format of 12 digits maximum length.

  4. Queue No - Enter the Queue Number in alphanumeric format of 6 characters maximum length.

  5. Appt Status - Select the required value for appointment status from the list box. The valid values are ‘Yet to come’, ‘On time’, ‘Late’ or ‘No show’.

  6.  Location - Select the Location type and Location from the list of valid values. The locations that are associated with the Operator Station are only displayed.

  7.  Practitioner - Select the Practitioner using [?] button. The practitioners who are associated with the displayed locations are only displayed.

  8. Specialty – Select the required specialty for which the visit records are to be displayed from the list of valid values.
