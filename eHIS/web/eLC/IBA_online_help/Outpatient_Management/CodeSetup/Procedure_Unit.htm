Procedure Unit

From this function you can Create a new Procedure Unit by specifying the working days applicable and associating other attributes to the Procedure unit. You can  Query for a desired Procedure Unit record.

A Procedure Unit represents a care location, to cater to Outpatient procedures within a healthcare facility. A Procedure Unit is used in the ambulatory healthcare setting and pertains to a particular Specialty. Patients normally visit the Procedure Unit with scheduled appointments or visit as Walk-in patient like in ‘Emergency’ Clinics. The Procedure Unit function allows defining attributes like Type of Visits Applicable, Mode of Registration, Level of Care, Working Days in a Week, Age Group Applicable, Specialty, Department and File Storage Location. Procedure Units can be defined specific to a group of Practitioners or open to all Practitioners within the Specialty.

Creating a Procedure Unit

From the OP Code Setup, select the Procedure Unit function.

The Procedure Unit screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a Procedure Unit created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Procedure Unit.
