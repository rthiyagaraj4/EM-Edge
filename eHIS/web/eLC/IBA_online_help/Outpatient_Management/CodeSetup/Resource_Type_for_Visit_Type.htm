Resource Type for Visit Type

From this function you can Create an association of a Resource Type to a Visit Type and Query for a desired Resource Type for Visit Type record.

Resource Type for Visit Type function is used to identify the various Resource Types required to cater for a particular visit within an outpatient encounter. This also identifies the Resource Types that are mandatory and optional in order to cater for an outpatient encounter.

Creating an association of Resource Type to Visit Type

From the OP Code Setup, select the Resource Type for Visit Type function.

The Resource Type for Visit Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the details from the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Resource Type associated to a Visit Type or [Reset] to clear the entered data.

  4. Click [Query] to search for existing association of Resource Type to Visit Type records.

  5. Click [Delete] to delete a queried record.
