Cancelled OP Checkout

From this function you can Query a list of Cancelled OP checkouts that were cancelled using the Cancel OP Checkout function.

Querying for Cancelled OP Checkouts

From OP Queries, select the Cancelled OP Checkout function.

The Cancelled OP Checkout screen gets displayed.

  1. Enter the query criteria.

    Enter the following query criteria:

    1. Specialty – Enter the Specialty or select using [?] function.

    2. Location Type – Select Location Type from the list. The listed values are Clinic and Procedure Unit.

    3. Location – Enter the Location or select the Location using [?] function.

    4. Practitioner – Enter the Practitioner or select the practitioner using [?] function.

    5. Patient ID - Enter the Patient ID or click [?] function to search and select the patient.

    6. Gender – Select the Gender from the list. The listed values are All, Male, Female and Unknown.

    7. Cancelled Date From/To – Select the date range using the calendar function. The Cancelled OP checkout records for the selected date range are displayed.

    8. Cancelled By – Enter the Practitioner or select the practitioner using [?] function who cancelled the OP Checkout.

  2. Click [Search] to select the records that satisfy the query criteria or [Reset] to clear the entered search criteria.

  3. The Cancelled OP Checkout details are displayed.

  4. Click [Clear] to clear the result screen.
