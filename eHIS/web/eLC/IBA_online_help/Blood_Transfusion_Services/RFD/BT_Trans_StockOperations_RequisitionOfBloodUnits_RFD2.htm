The following details are displayed:

 

  1. Product Code – The Product Code satisfying the specified search criteria is displayed if stock is available for the selected combination in the Collection Centre.

  2. Description – The Description of the product defined in the Product Codes function is displayed if stock is available for the selected combination in the Collection Centre.

  3. Blood Group – The Blood Group of the product defined in the Product Codes function is displayed if stock is available for the selected combination in the Collection Centre.

  4. Rhesus – The Rhesus factor of the blood group defined in the Product Codes function is displayed if stock is available for the selected combination in the Collection Centre.

  5. Reorder Level – The Reorder Level for the product defined in the Product Code function is displayed if the stock available is below Reorder Level.

  6. Current Level – The Current Level of the product in the stock is displayed if stock is available for the selected combination in the Collection Centre.
