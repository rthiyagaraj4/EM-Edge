Enter the following details:

 

  1. Product Code/Description – Enter a unique Product code in alphanumeric format of 10 characters maximum length. Enter the Description for the Product code in alphanumeric format of 40 characters maximum length in the adjacent field. This is a mandatory field.

  2. Short Description – Enter the short description of the Product code in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  3. Category – Click [List of Values] to select the Product category code. The values defined using the 'Product Categories' function get listed. This is a mandatory field.

  4. Screening Required? – Select the check box, if the Product requires blood screening. Clear the check box, if blood screening is not mandatory. If the check box is cleared the blood units are sent to the stock directly on receiving from the external facility or from the blood bank from Donor Management module. If the check box is selected, you have to do the screening in Blood Unit Screening function.

  5. Pooled Product? – Select the check box, if the Product requires pooling. Clear the check box, if the Product does not require pooling. If the check box is selected, the product can be used to pool with other similar products in the Pooled Units function. If the check box is cleared, you cannot pool the product with other products.  

  6. Expiry Period – Enter the Expiry Period of the Product in numeric format of three digits maximum length. The entered value has to be a valid non-zero positive integer. This is a mandatory field.

  7. Units – Select the unit from the list to indicate the expiry period of the Product. The listed values are Days, Months, Weeks, hours, and years. This is a mandatory field.

  8. Max Vol – Enter the Maximum Volume of the particular product that can be collected at a time from a donor in numeric format of five digits maximum length. This is a mandatory field.

  9. Units – Enter the corresponding Units for the maximum volume specified. This is a mandatory field.

  10. Stock Units – Click [List of Values]to select a Stock Unit for the Product. The values defined using the Unit of Measurements function get listed. This is a mandatory field.

  11. Generate Unit Number – Select the check box to generate the Unit Number automatically based on the product code. Clear the check box, if the unit number has to be entered manually for each Product Code.

The system prompts you to enter the Unit Number, if setup is not defined to generate unit number automatically in ‘BD Parameter’, ‘Collection Centre/Supplier’ or ‘Products’ functions. This rule takes precedence in checking the setup. System checks if the set up is defined in Collection Centre/Supplier, then in Products and finally in BD Parameters.

  1. Allow to change Expiry Date? - Select the check box to enable the system to allow you to enter the Expiry date beyond the Product expiry date which is being defaulted.

  2. Expiry Date Midnight - Select the checkbox to provide notification about the expiry date for the blood bank products. The system displays the notification as midnight (23:59) prior the day of expiry date.

  3. Length – Enter the Length of the unit number that you want to generate in the numerical format of two digits maximum length. The specified length of the unit number must not exceed 20. If you use the ISBT, standards, the maximum number you can use is 13. You have to define it in the BD Parameters. This field is enabled only on selecting ‘Generate Unit Number’ check box.

  4. Prefix – Enter the Prefix of the unit number in alphanumeric format of six characters maximum length. This field is enabled only on selecting ‘Generate Unit Number’ check box.

  5. Next Number – The Next Number field is disabled. The next number is created automatically when a new unit number is created. This field is enabled only on selecting ‘Generate Unit Number’ check box.

  6. Number of Labels – Enter the Number of Labels required for the Product in numeric format of two digits maximum length.

  7. Specimen Type – Click [List of Values] to select the Specimen Type. The values defined using the ‘Specimen Type’ function get displayed.

  8. Cross Match Required? – Select the check box to specify whether Crossmatch is required. The value for this field depends on the specimen type selected. If ‘Cross Match Required’ check box is cleared for the selected specimen, you cannot modify it. If ‘Cross Match Required’ check box is selected for the selected specimen, you can modify it and specify other specimen type under ‘Spec Type’ tab. The ‘Spec. Type’ tab is enabled only if the check box is selected.

  9. Max Time out of Blood Bank – Enter the maximum time allowed for the Product in numeric format of three digits maximum length. It indicates the maximum time a Product can be out of the blood bank before transfusion. It must not exceed the specified period. This is a mandatory field.

  10. Units – Select the units from the system-defined list for the maximum time allowed out of the blood bank before the transfusion. The valid values are ‘Minutes’, ‘Hours’ and ‘Day’. This is a mandatory field.

  11. Processing of Unit at issue time? – Select the check box to process the Product during issue. You have to enter the process code while transfusing the blood, if the check box is selected. If you clear the check box, you need not enter the process code during blood unit transfusion. The process codes are created in ‘Process Codes’ function in Blood Transfusion Services module.

  12. Refund Bill on Return? – Select the check box to enable refund of the bill amount on returning the product. Clear the check box, if the bill amount cannot be refunded on returning the product.

  13. Blood Bag Print Indicator - Select a value from the list. The system-defined values are 'Always Print', 'Differs from Original Unit Number' and 'No Print'.

  14. Always Print - Select 'Always Print' to print a Blood Bag Label for all the products associated to the Unit Number when the units are sent to stock through the 'Unit Screening' function.

  15. Differs from Original Unit Number - Select 'Differs from Original Unit Number' to print a Blood Bag Label for all the products where the 'Original Unit Number' received from the external source does not match with the 'Unit Number' that is generated through the 'Blood Unit Entry' function.

  16. No Print - Select 'No Print' if the Blood Bag Label is not to be printed when the units are sent to stock through the 'Unit Screening' function.

  1. Bypass Blood Confirmation – Select the check box to set the selected blood category as non blood cell . The selected non blood cell categories are bypassed from the Result Entry and Confirm Blood Type procedures and are allocated directly.
