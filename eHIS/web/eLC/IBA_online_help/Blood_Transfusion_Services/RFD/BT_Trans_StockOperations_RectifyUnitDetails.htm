Enter the following details to modify:

 

  1. Product Code – Click [List of Values] to select the Product Code to change the blood unit details. The list displays the values defined in Product Codes function.

  2. Unit Number – Enter the unique Unit Number in alphanumeric format of maximum 20 characters maximum length. The details are displayed if you click [Execute Query] after selecting the Product Code. You can modify the details.

  3. Original Unit Number - Enter the Original Unit Number in alphanumeric format of maximum 20 characters maximum length. The details are displayed if you click [Execute Query] after selecting the Product Code. You can modify the details.

  4. Blood Group – Click [List of Values] to select the Blood Group. The list displays the values defined in the Blood Groups function in Medical Records module. The details are displayed if you click [Execute Query] after selecting the Product Code. You can modify the details.

  5. Rhesus – Click [List of Values] to select the Rhesus factor of the blood unit. The list displays the values defined in Rhesus Factor function of the Medical Records module. The details are displayed if you click [Execute Query] after selecting the Product Code. You can modify the details.

  6. Donation Date – Click the Calendar function to change the Donation Date of the selected Unit.

  7. Date Of Expiry – The calculated expiry date of the unit based on the expiry period of the product defined in the ‘Product Codes’ function is displayed. You can modify it without exceeding the Expiry date specified for the selected product in the ‘Product Codes’ function.

  8. Volume – The Volume of the blood unit is displayed. You can modify the details, if required.

  9. Collection/Supplier – The recorded Collection /Supplier is displayed. Click [List of Values] to modify the Collection/Supplier code, if required. The list displays the values defined in the Collection /Supplier function.

  10. Donation Type – The Donation Type recorded is displayed. Select the Donation Type from the list to modify the details. The valid values are ‘Autologous’, ‘Allogenic’ and ‘Allogenic-Directed’. If the Donation Type is Autologous’, and ‘Allogenic-Directed’, you have to enter the Patient ID.

  11. Patient ID – The Patient ID associated with the unit, if the unit is used in any of the transactions is displayed.

  12. Fate – The Fate code and its description are displayed, if fate code is associated to the unit. The fate code is associated to the blood unit in Dispose blood units function.

  13. Last Reserved Date – The last reserved date of the unit is displayed. You cannot modify the details. The last Reserved date is recorded in Allocate/Xmatch/Issue - Blood Units function.

  14. Unit Status – The Status of the Unit is displayed, if it has been used in the transactions. The unit status is updated based on the use of the unit in Allocate/Xmatch/Issue - Blood Units function or Transfusion of Blood Units function.

 

Reserved for

  1. Patient ID – The Patient ID associated with the unit is displayed, if that unit is used in any of the transaction (reserved, issued, transfused, etc.). If multiple patients associated to the same unit the details of the last association is displayed.

  2. Specimen No – The Specimen number of the unit is displayed, if the unit has been used in transactions such as Allocate/Xmatch/Issue - Blood Units, Transfusion of Blood Units, etc.
