Enter the following details:

 

  1. Specimen No - Click [List of Values] to select the Specimen Number. This is a mandatory field.

  2. Priority – The Priority of the specimen is displayed. The valid values are ‘Stat’, ‘Urgent’ and ‘Routine’. The priority is recorded for the specimen in the Registration function in Laboratory module.  

  3. Patient ID – On selecting the specimen number, patient ID and name of the patient are displayed.

  4. Unit Number – The Unit Number associated to the selected specimen number in Pre Donation function in Blood Donor module is displayed.

  5. Comments – The comment recorded during requesting the blood unit in Request Blood Units function is displayed.

  6. Product Code – The product code of the selected specimen is displayed. The Product codes are created in Product Codes function.

  7. Sex – The Gender of the patient recorded during patient registration is displayed.

  8. Age – The age of the patient recorded during the patient registration is displayed. If an asterisk (*) is displayed next to the age, it is a calculated age and not the actual age of the patient.

  9. Blood Group – The Blood Group of the patient recorded in ‘Record Patient Blood Group’ function of the Medical Records module is displayed.

  10. Rhesus – The Rhesus factor of the blood group recorded in ‘Record Patient Blood Group’ function of the Medical Records module is displayed.

  11. Regd Date/Time – The Registered Date/Time of the Specimen is displayed. The specimens are registered in the Register function in Laboratory module.

 

Order Set Details

You have to Place Order for the order sets from the Clinician Access module, Register, Result Entry for the specimen and perform the First level review in the Laboratory module. The order sets are displayed for the specimen in the Order sets section.

  1. Order Set – The Order Set Code if order is placed for the Transfusion Order Set from Clinician Access module is displayed. The Transfusion Order set is defined in the BT Parameters function.

  2. Order Date – The Order Date is displayed.

  3. Confirmed – The check box is selected, if the order is confirmed. You cannot select or clear the check box. It is cleared if the order is not confirmed.  

  4. Order Set – The Order Set Code of the selected Order Set is displayed. The Orders that are awaiting results are highlighted using color.

  5. Order ID – The Order ID associated with the order set is displayed.

  6. Order Date – The Order Date is displayed.

  7. Specimen No – The Specimen number is displayed.

  8. [.....] – Click [.....] to compare the Xmatch/Order Set results. You can compare the results of current specimen number with the new specimen numbers that are associated to the patient record.

  9. Remarks – Enter Remarks in alphanumeric format. Double-click the Remarks field to maximize the Remarks dialog. When the Remarks text exceeds 4000 characters, then the alert message ‘The text after concatenation is more than 4000 characters. Do you want to truncate and proceed? gets displayed.

o If you click [Yes], the text gets truncated by the system up to 4000 characters and displayed in the Remarks field.

o If you click [No], then you can edit the text to accommodate up to 4000 characters.
