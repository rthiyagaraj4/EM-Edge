Enter the following details to define permissible blood type substitution:

 

Neonate’s

  1. Blood Group – Click [List of Values] to select the Blood Group for the newborn baby. The list displays the values defined in the ‘Blood Groups’ function in Terminology and Coding Management module.

  2. Rhesus – Click [List of Values] to select the Rhesus factor of the Blood Group. The list displays the values defined in the ‘Rhesus Factor’ function in Medical Records module.

 

Mother’s

  1. Blood Group – Click [List of Values] to select the Blood Group of the mother. The list displays the values as defined in the ‘Blood Groups’ function in Medical Records module.

  2. Rhesus – Click [List of Values] to select the Rhesus factor of the Blood Group. The list displays the values defined in the ‘Rhesus Factor’ function in Medical Records module.

 

Permissible Blood Type Substitutions

Blood Unit

  1. Blood Group – Click [List of Values] to select the acceptable Blood Group for substitution. The selected blood group is substituted to the actual Blood Group of the newborn when there is a shortage of the same Blood Group. The list displays the values defined in the ‘Blood Groups’ function in Medical Records module.

  2. Rhesus – Click [List of Values] to select the Rhesus factor of the Blood Group. The selected Rhesus factor is substituted to the actual rhesus factor of the newborn when there is a shortage of the same Blood Group. The list displays the values defined in the ‘Rhesus Factor’ function in Medical Records module.
