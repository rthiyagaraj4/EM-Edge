The following details are displayed:

 

Transfusion Start and End Time

  1. Unit Number – The Unit Number of the transfused blood unit is displayed.

  2. Check by Practitioner – Click [List of Values] to select the Practitioner who checked the transfusion process.

  3. Transfusion By Practitioner – Click [List of Values] to select the Practitioner who transfuses the blood unit to the patient. The list displays the list of Practitioners who are authorized to transfuse the blood.

  4. Practitioner ID – Click [List of Values] to select the Practitioner ID who transfuses the blood unit to the patient. The list displays the list of practitioners who are authorized to transfuse the blood unit.

  5. Start Date Time – Click Calendar function to select the transfusion Start Date and Time.  The transfusion start time should not be greater than reserved date time.

  6. End Date Time – Click Calendar function to select the transfusion End Date and Time. The transfusion completed time should not be greater than reserved date time.

  7. Volume – The transfused volume of the blood unit is displayed. You can modify the value if required.

  8. Storage Location – The Storage Location of the blood unit that is transfused is displayed. You can modify the value if required. The listed values are created in the ‘Storage Location’ function.

  9. Reaction – Click [List of Values] to select the Reaction code, if there is any reaction during the transfusion. The list displays the values defined in the ‘Transfusion Reactions’ function. If the selected reaction code is an Adverse Reaction, the ‘Adverse Reaction’ check box is selected by default and the system warns you to place order for transfusion reaction investigation.  You can place the order for transfusion reaction investigation in Post Transfusion Follow Up function.

  10. Description – On selecting the reaction code, the respective Description of the reaction code is displayed.

  11. Adverse Reaction – The selection of the check box depends on the reaction code. If the selected reaction is specified as an adverse reaction in ‘Transfusion Reactions’ function, the check box is selected, by default. The check box is cleared, if the reaction is not an adverse reaction.

  12. Transfusion Remarks – Enter the Transfusion Remarks in alphanumeric format.

  13. Discontinue? – Select the check box, if you want to discontinue the blood transfusion. The discontinuation depends on the reactions of the patient.

  14. Partial Transfusion Reason – Click [List of Values] to select the Partial Transfusion Reason, if the blood transfusion is discontinued. The list displays the values defined in ‘Reason’ function.

  15. Description – The Description for the Partial Transfusion Reason code selected is displayed.
