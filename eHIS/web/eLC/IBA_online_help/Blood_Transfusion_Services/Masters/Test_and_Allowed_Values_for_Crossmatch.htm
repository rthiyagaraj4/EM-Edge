Test and Allowed Values for Cross Match

From this function you can Associate the Tests for the crossmatch process with acceptable values for the test, Define whether the test is an emergency test, it is to be billed  and it is to be billed on non-performance of a test. You can Query for the existing records.

The Test and Allowed Values for Cross Match function enables you to associate the tests that are being performed while cross-matching blood units with acceptable values for tests. The tests in the list are created in the Test Codes function in Laboratory module.  The tests associated in this function are displayed in the Cross Match Tests screen in ‘Allocate/Xmatch/Issue - Blood Units’ function. You can indicate whether billing is mandatory for the tests. In case of emergency transfusions, all the defined tests are not performed, still you can bill all the tests based on the parameter defined.
