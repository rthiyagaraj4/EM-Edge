Neonate Permissible Blood Type Substitution

From this function you can Define Neonate permissible Blood Type Substitutions for a selected Product and Query for the existing records.

The Neonate Permissible Blood Type Substitution function enables you to define the permitted blood substitutions for a selected Blood Group and Rhesus factor of a mother and a newborn baby. This defines the compatibility of the mothers Blood Group with the newborn baby’s Blood Group. The allocation of the blood unit for the newborn baby in Allocate/Xmatch/Issue - Blood Units is based on the setup made in this function. In the first few weeks of the newborn’s life, any group of blood can be substituted irrespective of Blood Group and Rhesus factor. The blood is substituted when there is a shortage of blood or a need of replacement occurs for the newborn baby.

Defining Neonate Permissible Blood Type Substitution

From the Masters, double-click Neonate Permissible Blood Type Substitution function.

The Neonate Permissible Blood Type Substitution screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [Commit] or press [F10] to save the created Permissible Blood Type Substitutions or click [Clear Record] to clear the entered details.

  3. Click [Execute Query] or press [F8] to search for the existing records.
