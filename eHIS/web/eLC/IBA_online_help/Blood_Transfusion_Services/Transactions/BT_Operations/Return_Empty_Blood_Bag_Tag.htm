Return Empty Blood Bag/Tag

From this function you can Return the empty blood bags and tag after blood transfusion. You can also Print the report of the unit statistics. You can Query to view the existing details.

The Return Empty Blood Bag/Tag function is used to return the blood bag and tag after the blood is transfused to the patient. If the blood typing is confirmed in Confirm Blood Typing function, you cannot return the Empty Blood Bag/Tag using this function. The color codes displayed at the bottom of the screen denotes the status of blood units.

– Units issued in emergency

– Autologous Unit

– Stat (to be issued immediately)

– Urgent

– Directed Unit

– Routine

– Multi allocation Unit

Returning Empty Blood Bag/Tag

From the Transactions > BT Operations, double-click Return Empty Blood Bag/Tag function.

The Return Empty Blood Bag/Tag screen gets displayed.

  1. Click [List of Values] to select the specimen to return empty bag or tag.

  2. Click [Execute Query] to view the patient details, product category and unit details. <Related Field Description>

  3. Select a record and click [Return Empty Blood Bag/Tag] to return the empty blood bag or blood tags. The blood bag/tag associated to the selected blood unit is returned. <Related Field Description>

  4. Click [Close] to return the blood bag or blood tag or  [Cancel] to cancel the return process.

  5. Click [Save] to return the empty blood bag or blood tags or [Clear Records] to cancel the process.

  6. Click [Print] to print unit report. You can print unit reports of Allocated, Compatible/Suitable, Issued, Transfused units or All the Units together. A report is generated with the details.

  7. Click [Details] to view the patient details.

  8. Click [Antibodies] to view the antibodies recorded for the patient. The [Antibodies] gets enabled if antibodies are associated to the selected patient.

  9. Click [Transfusion History] to view the previous transfusion details of the patient, if any.

  10. Click [Requirements] to view the transfusion requirements recorded for the patient, if any.

  11. Click [View Incompatible Returns] to invoke Unit History screen to view the incompatible return details. Click [OK] to close the Unit History screen.

  12. Click [Product/Category Unit Details] to view category and product details of the unit. Click [Close] to close the screen.

The [Transfuse], [View Xmatch Results] and [View Multi-Allocation] are disabled in the function.
