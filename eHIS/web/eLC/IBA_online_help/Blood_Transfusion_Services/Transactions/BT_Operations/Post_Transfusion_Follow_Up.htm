Post Transfusion Follow up

From this function you can Record post transfusion reaction remarks for a patient.

The Post Transfusion Follow up function is used to record reaction remarks after blood transfusion to a patient. You can view the results of the order sets ordered from the Clinician Access module as a part of Post Transfusion follow up. The order is placed, if patient develops any kind of reaction during or immediately after the blood transfusion. You can select the order and add remarks, if required. The order sets which are awaiting results are highlighted in color.

Post Transfusion Follow Up

From the Transactions > BT Operations, double-click Post Transfusion Follow up function.

The Post Transfusion Follow up screen gets displayed.

  1. Click [List of Values] to select a specimen number to record post transfusion details.

  2. Click [Execute Query] to view the patient details, Product Category and Unit details. <Related Field Description>

  3. Right-click on the link 'Right Click for User Option' to select the user. The link has two pop-ups, namely, Patient Chart and Patient Specimen History. Click Patient Chart option to open the Patient Chart Screen from Clinician Access and Patient Specimen History to open the Lab screen from the Laboratory module.

  4. Click [Results] to view the test results of original specimen of the unit number with the new specimen number. You can compare the values through this screen.

    The specimen details are displayed in two columns namely 'Xmatch Results' and 'Order Set Results'. The current specimen number and the corresponding test results are displayed under 'Xmatch Results'. New specimen details associated to the patient are listed under 'Order Set Results'.

    Xmatch Results

    1. Specimen No - The current specimen number is displayed.

    2. Unit No. - The unit number that is associated to the specimen number is displayed.

    3. Product Code - The Product Code associated to the Unit Number is displayed.

    4. Test Code - The tests that are associated to the current specimen number is displayed.

    5. Result Value - The Result Value recorded for the respective test is displayed.

    6. Result Indicator - The Result Indicator recorded for the respective test is displayed.

    7. Xmatch Result - The Xmatch Result value is displayed.

    Order Set Results

    1. Specimen No - The Specimen Numbers associated to the patient are listed. The specimen number with pending results are highlighted in color.

    2. Ordered Date - The date and time when the specimen number is generated is displayed.

    3. Results - Select the check box to view the results. The result that is recorded for the respective specimen number is displayed in the text area on selecting the check box. You cannot select the check box, if result is pending.

    4. [Close] - Click [Close] to revert to 'Post Transfusion Followup' screen.

  5. Click [Details] to view the patient details.

  6. Click [Antibodies] to view the antibodies recorded for the patient. The [Antibodies] gets enabled if antibodies are recorded for the selected patient in Blood Units entry or Rectify Blood Units function.

  7. Click [Transfusion History] to view the previous transfusion details of the patient, if any. The Unit Number that is selected in 'Post Transfusion Followup' screen are highlighted in red in the 'Transfusion History' screen to indicate the current Unit Number.

  8. Click [Requirements] to view the transfusion requirements recorded for the patient, if any.

  9. Click [Text] to invoke the Remarks Text dialog where you can select , view or modify the remarks text.

  10. Click [Maximize] to view the remarks in a ‘Transfusion Remarks’ dialog.

  11. Click [Save] to save the remarks entered. You cannot edit the Remarks once saved. Hence, only new Remarks can be added on to the existing remarks in case if the Remarks already is not having 4000 characters saved.

  12. Click [Confirm] to record the Post Transfusion Follow up process. A message 'Process Completed Successfully' is displayed on clicking [Confirm]. The Confirm button is visible to pathologists only if the rights is assigned in User Access screen.

In case if there is no Order set ordered for a Unit which on transfusion has an adverse reaction, the 'Text', 'Maximize', 'Save' and 'Confirm' buttons are disabled.

When the specimen is confirmed, the Text, Save and Confirm buttons are disabled only the Maximize button is enabled.
