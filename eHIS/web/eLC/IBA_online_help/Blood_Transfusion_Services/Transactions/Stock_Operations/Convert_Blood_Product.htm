Convert Blood Product

From this function you can Convert the existing blood units into new units and Print new blood bag labels for the components.

The Convert Blood Product function is used to convert the existing blood products into new blood products. The function allows you to convert the blood component unit to different components and print new blood bag labels for the components. The Unit Number of the converted unit remains the same. The function enables you to enter multiple units to be converted simultaneously. If the Volume is Insufficient, color is displayed against the unit. If the unit is Duplicate or Expired Units or Products, color is displayed against the unit.  

Converting Blood Products

From the Transactions > Stock Operations, double-click Convert Blood Product function.

The Convert Blood Product screen gets displayed.

  1. Click the Calendar function to select the date range when the blood unit is added to the stock. <Related Field Description>

  2. Click [Display] to view blood unit details stored in the storage location.

  3. Enter the converted product details to query specific product records. <Related Field Description>

  4. Click [N] (Next) to navigate between the converted products and the units displayed.

  5. Scan the barcode unit number to convert the selected blood units. You can also enter the barcode.

  6. Click [P] (Previous) to navigate between the converted products and the units displayed.

  7. Select the ‘Select All’ check box to convert all units to a new product. You can select the individual units by selecting the check box against the unit number. The selected products are converted to a new product.

  8. Click [Validate] to validate the selected unit before converting it into a new product. The records that are validated are highlighted using color.

  9. Click [Confirm] to confirm blood unit conversion.

  10. A confirmation message 'Process completed successfully' is displayed when the process is complete. On clicking [Confirm], the results displayed based on the query criteria are cleared. The query remains intact enabling conversion of the same product without re-entering the query criteria. When we click [Display], the Unit List details are re-invoked for conversion.
