BT Parameters

From this function you can Define parameters that influence the functioning of Blood Transfusion Services module.

The BT Parameters function is used to define parameter details used in the Blood Transfusion Services module with respect to blood transfusion, blood unit management and stock operation. You can define default values that are displayed in the transaction functions of the module. The setups defined in this function influence the functioning of transaction functions performed in the module and interfacing to external modules.

Defining BT Parameters

From the Administration, double-click BT Parameters function.

The BT Parameters screen gets displayed with the General tab enabled.

  1. Enter the details in the mandatory (*) fields in the following tabs.
