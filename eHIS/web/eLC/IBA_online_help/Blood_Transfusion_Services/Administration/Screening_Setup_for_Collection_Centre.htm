Screening Test Setup for Collection Centre

From this function you can Associate a Product to a Centre and Restrict blood unit screening received from external facilities. You can Query to view the existing records.

The Screening Setup for Collection Centre function is used to restrict the screening of blood units received from external facilities. The function defines whether a blood product requires tests to be performed before adding it to the stock. If a product from a selected collection centre does not require test to be performed and if the ‘Screening Required’ check box is cleared, the blood unit gets added to the stock directly, on entering the blood units in the Blood Units Entry function. The system takes precedence to the setup in Screening Setup for Collection Centre for validation. If the selected product is not entered in the list in Screening Setup for Collection Centre function and if the ‘Screening Required’ check box in the ‘Product Codes’ function is selected, the product is sent for screening before adding to the stock.

Restricting Screening for Product/Collection Centre

From the Administration, double-click Screening Test Setup for Collection Centre function.

The Screening Test Setup for Collection Centre screen gets displayed.

  1. Click [List of Values] to select the Collection Centre Code from where the blood unit is received. The list displays the Collection Centers defined in ‘Collection/Supplier’ function. The Description of the Collection Centre is displayed.

  2. Click [List of Values] to select the Product code. The values defined through the ‘Products Code’ function are listed. The description of the product code is displayed.

  3. Select the ‘Screening Reqd’ check box, if screening is required for the selected product from the selected Collection Centre.

  4. Click [Commit] or press [F10] to save or click [Clear Record] to clear the entered details.

  5. Click [Execute Query] or press [F8] to search for the existing records.
