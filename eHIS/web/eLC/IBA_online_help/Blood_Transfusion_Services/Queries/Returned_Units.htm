Returned Blood Units

From this function you can Query the issued blood units that are returned to the Storage Location without blood transfusion. You can Print the details.

The Returned Blood Units function enables you to view the blood units are returned to the blood stock after issuing it for blood transfusion. You can query the details based on the date range. The Incompatible emergency units are displayed with yellow color code.

Querying Returned Blood Units

From the Queries, double-click Returned Blood Units function.

The Returned Blood Units screen gets displayed.

  1. Click the Calendar function to select the date range to query the blood unit status. This is a mandatory field.

  2. Click [Display] to view the blood unit status. The Returned Blood Unit details such as Product Returned Date, Description and Date of Expiry, Unit No/Volume, Blood Group, Rhesus, Category/Modified Date, Return Code and Description/Modified by are displayed.

  3. Click [Print] to print the details of the selected blood unit. You can also view the file in a PDF format and save the file in HTML, Delimited, RTF and PDF formats.
