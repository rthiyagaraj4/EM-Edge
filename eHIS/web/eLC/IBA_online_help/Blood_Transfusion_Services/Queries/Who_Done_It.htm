Who Done It

From this function you can Query the User who has modified the blood group and rhesus details of a blood unit.

The Who Done It function enables you to acquire the details of the user who has changed the Blood Group and Rhesus details of a blood unit.  The blood group and rhesus factor of a particular specimen is changed in the Amend Blood Group Rhesus factor function. The changed blood group and Rhesus factor, new blood group and Rhesus factor, the user name and the changed date is displayed.

Querying User Details

From the Queries, double-click Who Done It function.

The Who Done It screen gets displayed.

  1. Click [Execute Query] or press [F8] to view the details of the changes done to the blood group.

  2. Enter the patient ID and click [Display] to query for a particular Patient ID. The details are displayed based on the query criteria specified.
