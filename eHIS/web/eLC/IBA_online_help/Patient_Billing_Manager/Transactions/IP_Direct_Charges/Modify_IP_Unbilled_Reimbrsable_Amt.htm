Modify IP Unbilled Reimbursable Amt

From this function you can Update the reimbursable amount for the services utilised by the patient before the bill is generated.

The Modify IP Unbilled Reimbursable Amount function enables you to update the reimbursable amount for the services that are utilised by an inpatient or a daycare patient from the facility. An authorized user can reimburse the amount only if the services are not billed. The reimbursable amount for the service is defined in the ‘Billing Services’ function in the Masters. If the ‘override’ check box is selected for the respective service and the user is permitted to update the reimbursable amount for the respective service using the ‘Reimbursable Service by User’ tab in the ‘User for Authorization/Refund/Cash Counters’ function, the user can modify the reimbursable amount for an inpatient or a daycare patient.

Modify IP Unbilled Reimbursable Amount

From the Transactions > IP Direct Charges, double-click Modify IP Unbilled Reimbursable Amt Function.

The Modify IP Unbilled Reimbursable Amount screen gets displayed.

  1. Select an Episode Type from the list. The system-defined values are 'Inpatient' and 'Daycare'.

  2. Enter the Patient ID or the Admission/Transfer number. On entering the values, the name of the patient, age and gender is displayed.

  3. The unbilled transactions associated to the respective patient encounter are displayed.

    The following details are displayed:

    1. Service Code - The code of the Billing Service associated to the Inpatient or Daycare patient using the 'Enter Request-IP' function is listed. The service description is displayed.

    2. Item Code - The item associated to the patient through the Patient Billing Manager module or external modules are listed.  The item description is displayed.

    3. Service Date - The date and time when the service is associated to the patient gets listed.

    4. Net charge Amt - The net amount set for the service during definition is displayed.

    5. Reimbursable - The check box is selected, if the Billing Service is defined as 'Reimbursable' during definition. The check box is cleared, if the Billing Service is non-reimbursable.

    6. Type - Type is displayed as 'Full', 'Part' or 'Not Applicable', if the service is defined as fully reimbursable, partially reimbursable or non-reimbursable.

    7. Reimbursable Amt - Reimbursable amount for the respective service is displayed based on the 'Reimbursable Type'.

    8. Total - The total reimbursable amount for the unbilled services associated to the patient are displayed.

    9. Reimbursable Amt - The total reimbursable amount is displayed for the unbilled services associated to the patient.

  4. Select the check box 'Override' to override the reimbursable amount for a selected service. An authorization screen pops-up requesting login details. Enter the details and click [OK] to login and reimburse or [Cancel] to cancel the transaction.

    Enter the following details:

    1. User ID - Enter the User ID to login. This is a mandatory field.

    2. Pin No - Enter the respective PIN Number. This is a mandatory field.

    3. Use Prev. Reason - Select the check box to use the same reason used previously while reimbursing the amount. On selecting the check box the reason selected is displayed in the 'Reason' field.

    4. Reason - Click [List of Values] to select a reason. The values defined through 'Reasons for Billing Actions' function with 'Override Reimbursement' check box as selected are displayed. This is a mandatory field. This check box is not available for selection when you reimburse a service initially.

  5. Select the check box 'Reimbursable' to reimburse the service amount. The Reimbursable Type is displayed on selecting the check box. You cannot select the values as the values are displayed by default, based on the reimbursable type selected during Billing Service definition.

  6. Enter the amount to be reimbursed for the service in numeric format. The value entered is validated against the 'Type' and maximum limit set for the user through ‘User for Authorization/Refund/Cash Counters’ function. The recorded details are displayed under Audit Details when you click on the respective record under 'Unbilled Trx' tab.

    The following details are displayed:

    1. Audit No - The audit number is system generated.

    2. Added By ID - The name of the user who has reimbursed the amount is displayed.

    3. Modified Date - The Date and Time when the amount is reimbursed by the User is displayed.

    4. Old Reimbursable - The check box is selected, if the Billing Service is reimbursable.

    5. Old Type - The Reimbursable Type is displayed based on the type selected during Billing Service definition and previous transactions.

    6. Old Value - The Reimbursable amount for the respective service as per Billing Service definition and the previous transactions are displayed.

    7. New Reimbursable - The check box is selected, if the Billing Service is reimbursable.

    8. New Type - The Reimbursable Type selected to reimburse the amount during the previous transaction is displayed.

    9. New Value - The Reimbursable amount issued in the previous transaction is displayed.

    10. Overriding Reason - The reason selected to override the reimbursable amount is displayed.

  7. Click [Commit Form] to record the details or [Clear Record] to clear the data.
