Cancel IP Indirect Services  

From this function you can Cancel the services associated to an Inpatient or a Daycare patient from the external modules. The services associated to an Inpatient or the Daycare patient using Outpatient Management, Inpatient and Day Care Management, Order Entry and Result Reporting, Laboratory or the Pharmacy modules are cancelled using this function.  

Cancel IP Indirect Services function is used to cancel the Billing Services associated to an Inpatient or a Daycare patient from the external modules. A document reference number generated for the transactions are listed. An authorized user can cancel the transaction by selecting a relevant document reference number. The User associated to the User Group is able to cancel the services only if the check box ‘Cancel Confirmed Services’ is selected using the function ‘User Groups/Users’. User has an option to cancel the confirmed and unconfirmed services associated from the external modules.

Cancel IP Indirect Services

From the Transactions > IP Direct Charges, double-click Cancel IP Indirect Services function.

The Cancel IP Indirect Services screen gets displayed.

  1. Select an Episode Type from the list. The system defined values are Inpatient and Daycare. Select Inpatient or Daycare patient to cancel the services associated to an Inpatient or a Daycare patient admitted to the facility.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

OR

  1. Right click on Right Click for User Options to search for the relevant patient record. You can search for a patient record based on the Patients Lookup, Patient/Admission/Transfer lookup or Admission/Transfer lookup.

  2. Select the check box View Billing to view the billing details.

    The financial details of the patient for the selected episode get displayed. You can also view the unconfirmed services associated to the patient by clicking [Unconfirmed Services]. The Settlement Details, Billing Group details, service charges, gross amount for the services utilized, adjustments or refunds issued, unbilled and outstanding amount get displayed. The name of the User, who has recorded or modified the financial details gets displayed.

  3. Click [List of Values] in the 'Source Doc.Ref.' field to select and cancel a service. The transaction reference number generated while associating the Billing Services/Panels through 'Enter Request - OP' and 'Location Billing/Collection' functions are listed. The transaction reference document reference line number, Source Document Ref Number, Transaction Date, Panel/Service, Service Code, Rate/Charge, quantity of service associated, UOM, Total Payable, Patient Payable, Patient Paid and Doctor ID are listed.

  4. Select a transaction document reference number from the list and click [OK] to cancel a Billing Service/Panel. You can search a transaction document reference number in the pop-up screen. Enter a value in 'Find' and click [Find] to query the results. Click [Cancel] to cancel the operation and close the pop-up screen.

  5. The transaction date, Billing Service/Panel code and description, service quantity, UOM, total payable amount, patient payable amount, practitioner/staff indicator and practitioner/staff ID are displayed.

  6. Click [Commit Form] to cancel the services. A confirmation message 'Cancelling Daily Process Related Charges, Please Confirm' is displayed. Click [OK] to confirm. If the Billing Service/Panel is already confirmed, a confirmation message 'This Service is already confirmed' is displayed. Click [OK] to cancel. Cancelled transaction reference numbers are deleted from the list in 'Source Doc. Ref.'. Click [Clear Record] to clear the entered data.
