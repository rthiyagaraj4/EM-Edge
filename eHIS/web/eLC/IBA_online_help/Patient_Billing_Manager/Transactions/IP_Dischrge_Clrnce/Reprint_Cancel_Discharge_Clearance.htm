Reprint/Cancel Discharge Clearance

From this function you can Reprint or Cancel discharge clearance slips generated for inpatients.

Reprint/Cancel Discharge Clearance function is used either to reprint or cancel the discharge clearances. The discharge clearances generated for inpatients using the function Record Discharge Clearance are listed. You can access this function only if the check box 'Discharge Clearance in Use' is selected in 'BL Parameters' function under the 'Inpatient/Daycare' tab. Authorized users with valid access rights are permitted to cancel discharge clearance slips. To reprint the slips, you have to define print routing. The Discharge Document is available for discharged patient's also, so that the document can be reprinted after a patient is discharged from the ward.

Reprinting or Cancelling Discharge Clearance

From the Transactions > IP Discharge Clearance, double-click Reprint/Cancel Discharge Clearance Function.

The Reprint/Cancel Discharge Clearance screen gets displayed.

  1. Select a Facility ID from the list. The logged in Facility is displayed, by default. The Facilities associated to the logged in User are listed. This is a mandatory field. <Related Field Description>

You can search inpatient records against which the discharge advice has been raised by selecting the Facility ID and clicking [Search].

  1. Click [Search] to search for patient records matching the search criteria. On clicking [Search], the legend changes to [As On: Current Date Time]

  2. Patient records matching the search criteria gets listed. Inpatient records get listed with the Patient ID, Episode ID, Episode Type, Admission Date, Discharge Clearance Date, Discharge Clearance Document Type and Number, Outstanding Amount and the name of the authorized user.  

  3. Select the check box against patient records to cancel discharge clearance generated using the function 'Record Discharge Clearance'.

  4. An authorization screen pops-up requesting the login details. Enter the details and click [Accept] to cancel discharge clearance slip or [Exit] to cancel.

    Enter the following details to login and cancel discharge clearance slip:

    1. User ID - Enter the User ID in alphanumeric format. Authorized users are only permitted to login and generate discharge clearance slip. This is a mandatory field.

    If the user is not authorised, an error message 'The User is not authorised to Cancel discharge clearance' is displayed. User is authorized to cancel discharge clearance for inpatients' by selecting the check box 'Discharge Clearance'' through the function 'User Privileges and Cash Counter Access'.

    1. Pin Number - Enter the Pin Number of the respective User. This is a mandatory field.

    2. Reason - Click [List of Values] to select a reason. The action reasons defined through the 'Reasons for Billing Actions' function with the check box 'Cancel Discharge Clearance' as selected are listed. This is a mandatory field.

    3. Remark - Enter the Remarks in alphanumeric format. This is a mandatory field.

  1. Click [Commit Form] to cancel the services or [Clear Record] to clear the entered data.
