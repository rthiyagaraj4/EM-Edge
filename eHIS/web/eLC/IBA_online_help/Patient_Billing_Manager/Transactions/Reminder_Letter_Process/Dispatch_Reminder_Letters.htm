Dispatch Reminder Letters

From this function you can Dispatch the Reminder Letters for an outstanding amount range and a relevant Billing Group category. You can Query to view the generated reminder letters, which are not dispatched for a selected Billing Group Category and the selected amount range. You can Record the undelivered dispatches using this function.  

Dispatch Reminder Letters function is used to post the reminder letters for a selected Billing Group Category and an outstanding amount range. You can select a Reminder Letter to be dispatched based on the outstanding amount, Billing Group Category and the number of days for which the bill amount is outstanding. You have an option to view the undelivered dispatches. A reference number is generated on selecting several outstanding documents for reminder letter dispatch. The outstanding document details are listed under the tab ‘Record Undelivered Dispatches’.

Dispatch Reminder Letters

From the Transactions > Reminder Letter Process, double-click Dispatch Reminder Letters function.

The Dispatch Reminder Letters screen gets displayed.

  1. Select a Billing Group Category. The values defined using the Billing Group Category function get listed. This is a mandatory field.

  2. Select an Outstanding Amount range. The values defined using the Reminder Letter Setup function and associated to the selected Billing group Category get listed. This is a mandatory field.

  3. Select a Reminder Number from the list. The Reminder Number associated to the selected Billing Group Category and Outstanding Amount range gets displayed. This is a mandatory field.

  4. Click [Refresh] to clear the entered details.

  5. The outstanding document details for which the Reminder Letters are generated get listed. The details are displayed under the following tabs:

Generate Letters for Dispatch

Record Undelivered Dispatches

View Undelivered Dispatches

The View Undelivered Dispatches tab enables you to view the complete range of bills details, which are recorded as undelivered.

  1. Select the records and click [Commit Form] or [Process] to dispatch the Reminder Letters for the selected Outstanding documents.

  2. Click [Clear Record] or [Cancel] to clear the entered data.
