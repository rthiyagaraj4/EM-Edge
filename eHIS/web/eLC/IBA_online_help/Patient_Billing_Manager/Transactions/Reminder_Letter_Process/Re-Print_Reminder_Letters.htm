Re-Print Reminder Letters

From this function you can Generate a report to view the reminder letters dispatched to the patients/customers.

Re-Print Reminder Letters function is used to re-print the Reminder Letters dispatched to a patient/customer. The dispatched reminder letter details are listed in the report.

Re-Print Reminder Letters

From the Transactions > Reminder Letter Process, double-click Re-Print Reminder Letters function.

The Re-Print Reminder Letters screen gets displayed.

  1. Click the Calendar function to select the dispatch date range. The reminder letters dispatched during the selected date range gets listed. This is a mandatory field. <Related Field Description>

  2. Select the documents and click [Run Process] to re-print the reminder letters or click [Clear Record] to clear the entered data.

  3. A confirmation screen appears on clicking [Run Process]. Click [Yes] to re-print the reminder letter or [No] to cancel the transaction.
