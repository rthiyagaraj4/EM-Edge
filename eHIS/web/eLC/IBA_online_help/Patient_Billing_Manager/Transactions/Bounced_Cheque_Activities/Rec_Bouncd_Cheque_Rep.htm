Record Bounced Cheques Replacements  

From this function you can Enter and Record a new cheque detail or the credit card information to clear the outstanding bill details. You can record the new details against a document type. You can Query to view the replacement details already recorded by the user.

Replacement of Bounced Cheque function is used to replace the bounced cheque details with a new cheque or a credit card. You have to replace the document type details with the new cheque or credit card details. A new document number is generated and displayed for the replacement on committing the record. The Credit Card Document Type has to be created initially for the facility. The document types valid for billing transactions are created using the Financials Base Manager module. The document type has to be defined as valid for transactions in the Patient Billing Manager module. The valid documents are associated to a transaction type using the function ‘Document Types’ in the Masters.

Recording Bounced Cheques Replacements

From the Transactions > Bounced Cheque Activities, double-click Recording Bounced Cheques Replacements Function.

The Recording Bounced Cheques Replacements screen gets displayed.

  1. Select a Document Number for transaction.

  2. Click [Execute Query] or press [F8] to view the details. The transaction against which the Settlement Type was recorded as Cheque and if the cheque was considered as invalid by the bank, the details get displayed.  The details recorded using the function Enter Bounced Cheques  get displayed.

  3. Enter the details in the mandatory (*) fields. <Related Field Description>

  4. Click [Commit Form] to record the details or [Clear Record] to clear the entered data.

 

You have to login to the Cash Counter to replace the bounced cheques.
