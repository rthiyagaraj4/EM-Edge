View Bounced Cheques  

From this function you can View the details regarding the cheques marked as invalid by the bank. You can only view the details recorded, but no modifications can be made to the data.

View Bounced Cheques function is used to view the details of the cheque considered as invalid by the Bank due to certain reasons. The details recorded using the function ‘Enter Bounced Cheques’ gets displayed.  

View Bounced Cheques

From the Transactions > Bounced Cheque Activities, double-click View Bounced Cheques Function.

The View Bounced Cheques screen gets displayed.

  1. Select a value for Date Selection from the list. <Related Field Description>

  2. The transaction against which the Settlement Type was recorded as Cheque and if the cheque was considered as invalid by the bank, the details get displayed. The details recorded using the functions Enter IP Receipts/Refunds  or Enter OP Receipts/Refunds  for an Inpatient or an Outpatient gets listed.
