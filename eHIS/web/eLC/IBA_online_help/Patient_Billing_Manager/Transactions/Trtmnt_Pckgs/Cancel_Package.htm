Cancel Package

From this function you can Cancel a treatment package that is associated to a patient. You can Query to view the encounter and package approval details.

The Cancel Package function is used to cancel a treatment package that is associated to the patient. You can cancel a package that is not billed and open. Authorized users are only permitted to cancel a package. The orders that are placed as a part of the package remain associated to the patient records even after cancellation and standard price is computed for the services based on the financial details that are captured during encounter registration. You can cancel the orders that remain unprocessed during package transfer. Option is available to convert the deposit amount accepted against a package to special agency deposit or the amount can be issued as a refund. The approval against the cancelled package is cancelled while the package is dissociated from the patient encounter.

Cancel Packages

From the Transactions > Treatment Packages, double-click the Cancel Package Function.

The Cancel Package screen gets displayed.

  1. Enter the Patient ID in alphanumeric format of 10 characters maximum length or click [List of Values] to select the patient ID. The name, age and gender of the patient gets displayed on entering the Patient ID. This is a mandatory field.

  2. Press [Tab] to select the package that has to be cancelled.

  3. The details are displayed under the following tabs:

Cancel Package

Encounter & Approval Details

  1. Click [Commit Form] or press [F10] to save the records or click [Clear Record] to clear the data.

  2. The package amount is recalculated.

If there are any closed packages that are to be cancelled, an authorized user has to cancel the bill generated for an inpatient or outpatient encounter in the facility. The package is reopened with the message 'Cancelling the Bill will reopen the Package. Confirm'. Confirm to cancel the bill and re-open the package.
