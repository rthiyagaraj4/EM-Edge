Medical Record Approval Request

From this function you can Approve Medical Record Request even if the patient has Outstanding Amount.

The Medical Record Approval Request function is used to approve Medical Record Request even if the patient has Outstanding Amount. This function is accessible to the user only if rights are given in the ‘User Privileges and Cash Counter Access’.

Approving Medical Record Request

From the Transactions, select Medical Record Approval Request function.

The Medical Record Approval Request screen gets displayed.

  1. Click [?] function to select the patient ID. This is a mandatory field.

  2. Click [Search] to search outstanding amount of the selected patient ID across all encounters or [Clear] to clear the entered data.

  3. Select the ‘Approve’ checkbox against the outstanding amount to approve the medical record request of the patient.

• If the patient outstanding is zero, the approve check box is disabled.

• If the patient has been approved already when there is an outstanding amount the check box is selected and disabled.

  1. Click [Apply] to allow approve Medical Record Request for the patient or [Reset] to clear the details.

  2. Once an approval is recorded for the patient, the approval is valid for the number of hours mentioned in the ‘Medical Request Authorization Valid Till’ field in MR Parameter screen from the date/time of the approval.
