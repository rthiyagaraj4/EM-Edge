Track/Finalize Bill Exemptions

From this function you can Approve the billing exemption request raised for a patient. You can Query to view the bill exemption details. The bill exemptions defined initially for a patient can be verified, approved and finalized using this function.

Track/Finalize Bill Exemption function is used to view the exemption details of a relevant patient. The details recorded using the function ‘Bill Exemption’ is verified, approved and finalized by an authorized user. The user also has an option to view and approve the adjustments made against a bill for a patient on producing valid documents. The details are displayed under the respective tabs based on the stages selected for the respective counselling detail. Pending Tasks are displayed in different colours with the number of pending tasks for the logged in user.

Track/Finalize Bill Exemptions

From the Transactions > Bill Exemption/Adjustment, double-click Track/Finalize Bill Exemption Function.

The Track/Finalize Bill Exemption screen gets displayed.

  1. Enter the Patient ID in alphanumeric format of 10 characters maximum length. This is mandatory field.

  2. Click [List of Values] to select an Exemption Code. The exemption request placed for a patient gets displayed.

  3. Select the check box 'My Assignments Only' to view the assignments associated to the logged-in user. Clear the check box to view all the records of the respective patient.

  4. Click [Execute Query] or press [F8] view the exemption details recorded for the selected patient.

  5. The details get displayed under the following three tabs:

Verification

The exemption details recorded using the function Bill Exemption function gets displayed along with the document number generated for the respective transaction. The adjustment amount has to be equal to or lesser than the authorized amount to verify and to approve the exemption.

  1. Select – Select the check box to verify the details for the displayed record.

  2. [Authorize] – Click [Authorize] to login and to verify the bill exemption.

    Enter the following details:

    1. User ID – Enter the User ID in alphanumeric format. The authorized users are only permitted to login and to update the details. This is a mandatory field.

    2. PIN No – Enter the PIN number for the selected User. This is a mandatory field.

The exemption details, adjustment details and the exemption history for the selected patient gets displayed under three separate tabs:

Exemption Dtls

Based on the exemption details recorded using the function ‘Bill Exemption’, you can exempt or adjust the bill amount. You also have an option to view the history. You can update the Bill Exemption details and the Counseling details on confirming the bill exemption. Only those patient records for which the Bill Exemption details were entered initially are available for approval.

Enter the following details to exempt the patient:

  1. Exemption Doc - The exemption document type and number generated while exempting the bill amount using the ‘Bill Exemption’ function gets displayed.

  2. Patient No – The patient ID and name is displayed.

  3. Date – The date when the bill amount is exempted is displayed.

  4. Exemption Reason – The reason selected by the user while exempting the bill is displayed.

  5. Current Stage – The current stage sequence number and description is displayed for the respective exemption code. This is based on the stages defined for the respective exemption code.

  6. Next Stage – The next stage sequence number and description is displayed for the respective exemption code. This is based on the stages defined for the respective exemption code.

  7. Revised Amount – The exemption amount for the respective patient is displayed. You can update the amount.

  8. Accept/Reject – Select Accept, Reject or Reject on close to accept or reject the bill exemption for the selected patient respectively. On selecting the value as 'Accept' and saving the details, system prints the online Exemption Approval Document.  A copy of the same document is also printed with the legend as 'Copy'.

  9. Assign To – Select the user from the list. The name of the logged in user is listed.

Adjustment Bills

You can view the adjustments made to the bill details for the respective patient.

Exemption History

You can view the exemption details for the respective patient. The Counseling details along with the exemption document details are also displayed.

Approval

The exemption details verified using the Verification tab gets displayed. The adjustment amount has to be equal to or lesser than the authorized amount to verify and to approve the exemption.

  1. Select – Select the check box to approve the verified exemption details for the displayed record.

  2. [Authorize] – Click [Authorize] to login and to approve the bill exemption.

    Enter the following details:

    1. User ID – Enter the User ID in alphanumeric format. The authorized users are only permitted to login and to update the details. This is a mandatory field.

    2. PIN No – Enter the PIN number for the selected User. This is a mandatory field.

The fields that are mentioned Exemption Dtls, Outstanding Bills and Activity History tabs under the Verification tab are displayed.

Finalize

The exemption details verified using the Approval tab gets displayed. A Cashier will finalize the bill exemption. The adjustment amount has to be equal to or lesser than the authorized amount to verify and to approve the exemption.

  1. Select – Select the check box to finalize the approved exemption details.

  2. [Authorize] – Click [Authorize] to login and to finalize the exemption details.

    Enter the following details:

    1. User ID – Enter the User ID in alphanumeric format. The authorized users are only permitted to login and to update the details. This is a mandatory field.

    2. PIN No – Enter the PIN number for the selected User. This is a mandatory field.

The fields that are mentioned Exemption Dtls, Outstanding Bills and Activity History tabs under the Verification tab are displayed. System prints the Online Exemption Document on accepting the exemption details and saving the records.

  1. You have to accept or reject the bill exemption recorded for a patient.

  2. You can record the counselling details by associating a Counselling Code to the patient or view the encounter counselling details.

    Record Stage Counseling

    1. Populate Counseling from Encounter -  Select a value from the list. If the counselling details are recorded during encounter registration for the respective patient, the details are listed.

    2. The Counselling Code, description and type are displayed. You can select the UOM and counselling value. Detailed description is displayed, if the value is ‘Free Text’.

  3. Click [Commit Form] to record the details or [Clear Record] to clear the entered data.
