Bill Exemption

From this function you can Enter the necessary details to exempt the bills of a patient. You can Query to view the details already recorded by the user to exempt the bill amount for a patient.

Bill Exemption function is used to record the counseling details and adjust the outstanding bill amount for a relevant patient. An authorized user only can exempt the bill amount. The ‘Bill Exemption’ and ‘Bill Adjustment’ check boxes have to be selected in the function ‘Users for Authorization/Refund/Cash Counters’ in the Administration menu for the user to login and to authorize the bill exemption process. The stage for exemption has to be defined in the Masters and the user assigns an exemption code to the patient during counseling. The counseling details are defined initially in the Masters. The Authorized User also has an option to cancel the bill exemption or view the bill exemption details for a patient. The patient has to produce valid documents, if the bill amount has to be exempted. The stages defined for the selected Exemption Code gets displayed. The ‘Bill Exemption Doc Type’ selected under the ‘Adj/Exmptn’ tab in the BL Parameters function gets displayed. A document reference number is generated on committing the record. You can exempt the amount for the outstanding bills, which are not settled. The bill details against which the exemption is requested, get displayed in different color. You cannot authorize an exemption more than the limit amount recorded using ‘User for Exemption’ tab in the ‘Exemption Setup’ function. The counseling detail has to be recorded, if the Counseling by Stage is mandatory. The pending exemption request is highlighted using color and pending installment request is highlighted using color.

 

Process Flow Diagram

Bill Exemption Request and Completion

From the Transactions > Bill Exemption/Adjustment, double-click Bill Exemption Function.

The Bill Exemption screen gets displayed.

  1. Select a Patient Number for transaction or use the Lookup function to select a patient. You can search for a relevant patient ID in the lookup screen. Click [Enter Query], enter the search criteria and click [Execute Query] or press [F8] to select a patient ID. The patient details get populated in the respective fields.

  2. The patient Episode, National ID and pending Exemption and Installment request assigned to the logged-in user are displayed.

  3. Click the Calendar function to select the Encounter date range. This is a mandatory field. The exemption request placed for the patient and assigned to the logged-in user is displayed.

  4. Click [Search] to query the patient details.

  5. The patient record for which exemption or installment request is placed through 'Record Financial Counselling' function and finalized are listed.

  6. The Counseling Details recorded through the Record Financial Counselling are displayed.

  7. Click [Preview] to generate a report, print the output or save the output in a file.

  8. The patient ID, Encounter Date, Episode Type, Encounter ID, name of the user who has to finalize bill exemption or installment request is displayed. The date when the request is assigned to the user, patient outstanding amount, total outstanding amount for the services utilized, discharge date of the patient, admission or visit status and the location where the patient is admitted is displayed. The check boxes 'Counseling Finalized', 'Installment Payment Allowed' and 'Exemption Allowed' are selected, by default.

  9. Right-click on the user name in 'Assigned To' column beside the 'Exemption Allowed' check box.

  10. The options Exemption Allowed and Request Completion are displayed on right-click.  

  11. The 'Bill Exemption' screen is displayed on selecting Exemption Allowed. Enter the patient number and patient details like the patient's name, age and gender is displayed.

  12. Enter the details in the mandatory (*) fields. <Related Field Description>

  13. You will be directed to 'Record Stage Counseling' tab, if the counselling details are mandatory. Mandatory details can be recorded only on selecting a record for exemption. You can also view the encounter details through the 'View Encounter Counseling' tab.

  1. Click [Authorize] to login  and to authorize the bill exemption for the selected patient.

    1. User ID – Enter the User name in alphanumeric format. This is a mandatory field.

    2. PIN No – Enter the Pin number for the selected patient. This is a mandatory field.

    3. Reason – Click [List of Values] to select a Reason for exemption. The reasons defined using the function ‘Reasons for Billing Actions’ in the Masters with the ‘Bill Exemption’ check box as selected gets listed. This is a mandatory field.

  2. The outstanding bill details are displayed.

    Outstanding Bills

    The outstanding bill details associated to the selected patient get displayed.  

    1. Include Unauthorised Bills – Select the check box to list and exempt unauthorised bills.

    2. Assign To – Select a value from the list to assign the exemption process to a relevant user.

    3. Select - Select the check box to exempt the bill amount. The mandatory counselling details can be recorded, if the check box is selected.

    4. Doc Number – The document type and the number for the outstanding document gets displayed. The date when the document number was generated for the respective outstanding document also gets displayed.

    5. Episode Type – The Episode Type gets displayed.

    6. Billing Grp – The Billing Group for the selected outstanding document gets displayed.

    7. Bill Amt– The pending bill amount of the patient is displayed.

    8. Cust Code – The Customer Code associated to the selected outstanding document gets displayed.

    9. Outstanding Amt Before Exemption – The outstanding amount for the selected document type gets displayed. Right-click on the column to view the patient encounter details or financial details.

    10. %age – Enter the percentage of amount in numeric format. The exemption amount is calculated as the percentage of the actual amount.

    11. Adjustment Amt – The adjustment amount gets displayed automatically based on the percentage of exemption and the actual outstanding amount.

    12. Remaining Amt – The remaining amount that the patient has to pay is displayed.

    13. Remarks – Enter the remarks in alphanumeric format.

    14. BG – The description for the respective Billing Group gets displayed.

    15. Cust – The description for the respective Customer code gets displayed.

    16. Total – The total amount adjusted against the outstanding bills of a patient gets displayed.

    17. Select – Select the check box to enter the percentage of amount. If the check box is cleared, you cannot enter the percentage value to adjust the bill details.

    Click to view the outstanding bill in detail. The Outpatient Query page gets displayed listing the amount against the services availed from the facility. The records with exemption request are highlighted in color.

    Counseling Details

    If the counseling details are not captured for the patient, you can record the details using this tab. If the details are already captured, the recorded details are displayed.

    1. Current Stage – The highest priority set for the stages while defining the exemption setup, gets displayed.

    2. SL No – The serial number gets displayed.

    3. Counseling Descriptor – Click [List of Values] to select a Counseling Descriptor. The counseling details associated to the selected exemption code get listed.

    4. Value – Select a value from the list. The values listed are ‘Yes’ and ‘No’. The values indicate whether the selected Counseling Code is valid for the patient or not.

    5. UOM – Enter the Unit of Measure in numeric format.

    6. Action – Select an Action from the list. The values listed are Accepted, Rejected and Warning. Select Accepted, if the exemption is applicable. Select Rejected to reject the exemption. Select Warning to issue a warning during exemption for the selected patient.

    A message pops-up on saving the transaction to confirm if the 'Request for Financial Counseling' document has to be printed.

  3. A document number is generated on recording the counseling details. Click [OK].

  4. A pop-up screen is displayed requesting confirmation to print the Financial Counseling Report. Click [YES] to print or [NO] to cancel. The report is printed in the destination printer.

  5. Right-click on the column and select 'Request Completion' to complete the request.

  6. A warning message 'Do you want to confirm the request completion?' is displayed.

  7. Click [Yes] to confirm request completion or [No] to cancel the request.
