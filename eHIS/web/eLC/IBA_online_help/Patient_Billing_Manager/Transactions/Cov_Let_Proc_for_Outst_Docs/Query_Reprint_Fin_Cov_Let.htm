Query/Reprint Finalized Covering Letters

From this function you can Reprint the finalized Covering Letters. You can Query to view the finalized Covering Letters.  

Query/Reprint Finalized Covering Letters function is used to query or to print the finalized Covering Letter details. You have an option either to query or re-print the finalized Covering Letters.

Query/Reprint Finalized Covering Letters

From the Transactions > Covering Letter Process for Outst Doc, double-click Query/Reprint Finalized Covering Letters function.

The Query/Reprint Finalized Covering Letters screen gets displayed.

  1. Click [Execute Query] or press [F8] to edit and to finalize the Covering Letters generated. The document reference number generated for a specific patient with the outstanding document details gets displayed. The Billing Group and the Customer associated during Covering Letter generation gets displayed. The date when the Covering Letter was generated is displayed. The total outstanding amount for the outstanding documents gets displayed.

  2. The Covering Letter format selected during finalizing the Covering Letter gets displayed. The check boxes are either cleared or selected for the individual formats based on the format defined using the function Edit/Finalize Covering Letters.

  3. The unsettled bills for the services availed from the facility get displayed. The cancelled and the verified documents are displayed using red and green colors.

  4. The number of outstanding documents associated to the patient gets displayed. You can sort the displays order by selecting a value from the Order By list.  

  5. Click [Criteria] to view the covering letter parameters defined using the Generate Covering Letter function.

  6. Click [Details] to view the Covering Letter Format Details.

  7. Click [Reprint] to print the Covering Letter for the selected patient.

  8. Click [Commit Form] to save the details or [Clear Record] to clear the entered data.

  9. Press downward arrow key to navigate to the next record on committing the details. Click upward arrow key to navigate to the previous record.
