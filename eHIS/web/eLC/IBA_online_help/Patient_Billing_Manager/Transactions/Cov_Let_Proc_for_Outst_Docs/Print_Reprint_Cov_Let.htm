Print Reprint Covering Letter

From this function you can Print or Reprint the Covering Letters generated for the outstanding documents.  

Print Reprint Covering Letter function is used either to print or reprint a copy of the Covering Letters generated for the outstanding documents generated using the function Generate Covering Letter. You have an option to include or exclude the Policy details or the Covering Letter details on the letterhead. The result is displayed based on the sort order selected. You have to define the destination printer to print the report.  

Print/Reprint Covering Letter

From the Transactions > Reminder Letter Process, double-click Print Reprint Covering Letter function.

The Print Reprint Covering Letter screen gets displayed.

  1. Select either the document reference date range or a range of document reference numbers. <Related Field Description>

  2. Click [Run Process] to either View, Print or Save the details in a file.

  3. A log file is generated with the details. Click [OK] to view the report or [Cancel] to exit from the screen.
