Nullify/Cancel a Bill

From this function you can Cancel a bill generated for an outpatient towards a service. You cannot post a bill once it is cancelled.  

Nullify/Cancel a Bill function is used to cancel a relevant bill. The Authorized Users only can login to cancel the bill. If the option ‘Bill’ is checked in ‘Transaction Processing Privileges’ tab in the function ‘Users for Authorization/Refund/Cash Counters’ under Administration menu, the user is authorized to login and cancel a relevant bill. You cannot cancel a bill if the patient is discharged. A cancellation report is printed with the bill details that are cancelled.

Nullify/Cancel A Bill

From the Transactions > OP Billing, double-click the Nullify/Cancel a Bill Function.

The Nullify/Cancel a Bill screen gets displayed.

  1. Enter the Bill document type and number, which has to be cancelled. This is a mandatory field.

  2. Click [Bills] to select an Outpatient or an Emergency patient bill from the lookup. The Bill Lookup appears with the Patient ID, Episode ID, Visit ID, Document Type Code, Document Number, Document Date, bill total outstanding amount, bill paid amount, bill transaction type code, bill nature code and the bill status gets displayed. Double-click on the relevant record to select it. This is a mandatory field. The Bill Nature, billing Group and the Customer associated to the selected patient, if any, gets displayed.

  3. Double-click the relevant record to select it. The Bill document type and number gets displayed in the respective fields.

  4. Select the check box 'Cancel' to login and to cancel the bill or [Clear Record] to clear the entered data. Authorized users are permitted to cancel the bills that are generated.

  5. The check box 'Print Cancelled Report' is selected automatically when the check box 'Cancel' is selected. A cancellation report is printed, if the check box is selected. Clear the check box, if the report cancellation report is not required.

  6. The cancellation authorization screen gets displayed. Enter the User ID, PIN Number and select a reason for cancellation. Click [Accept] to login and to cancel the bill. Click [Cancel] to exit from the screen.

    The values defined through the 'Reasons for Billing actions' function with the 'Bill Cancellation' check box as selected are displayed.

  7. The bills associated to the selected patient are displayed with the check box as selected.

  8. Click [Process] to cancel the associated bills.

  9. A confirmation message 'Do you want to Confirm?' is displayed. Click [Yes] to cancel the selected bills or [No] to cancel the process.

 

The bills against which the payment is made cannot be cancelled.
