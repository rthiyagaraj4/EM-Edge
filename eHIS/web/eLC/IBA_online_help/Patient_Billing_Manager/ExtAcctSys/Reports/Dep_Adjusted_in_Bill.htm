Deposit Adjusted in Bill

From this function you can Generate a report on the deposit amount adjusted against the bill amount.

Deposit Adjusted in Bill function is used to list the patient details where the deposit amount is adjusted against the bill amount. System generates a report listing the patient details, bill details, deposit details and the total refund amount. The report output is based on the discharge date selected.

Generating a Report on the Deposit Adjusted

From the External Accounting System > Reports, Deposit Adjusted in Bill Function.

The Deposit Adjusted in Bill screen gets displayed.

  1. Select a valid Episode Type from the list. The system-defined values are– All, Inpatients, Outpatient, ‘Daycare’, ‘Emergency’ and ‘External’. A report is generated for the selected Episode Type. This is a mandatory field.

  2. Click the Calendar function to select a date range. The report is generated for the selected period. This is a mandatory field.

  3. Click the Calendar function to select a date range to view the deposit adjustment details for patients discharged during the selected period. The report output is filtered based on the discharge period selected. The date selected for 'Discharge To' has to be lesser than the system date and greater than the 'Discharge From' date.

  4. Select 'Exclude' check box to exclude the Receipt Type from the report. If the check box is selected, the report generation task does not consider the deposits adjusted against the relevant Receipt Type. If the check box is selected, the details on the deposit amount adjusted against the relevant Receipt Type are listed.

  5. The nature of the Receipt Type codes defined in the Masters are listed. The description for the relevant Receipt Type is displayed.

  6. Click [List of Values] to select a valid Account Code. The Account Codes are displayed only if the Chart of Accounts is defined in the GL Set up.

  7. Click [Run Process] to generate the report based on the entered report criteria or [Clear Record] to clear the entered data.

  8. You can view the report, print the details or save the output in a file in the system.
