Service/Item Grouping Setup

From this function you can Create a Service/Item Groups and Associate items to the Service/Item Group. You can Query to view the existing records.

The Service/Item Grouping Setup function is used to create a Service/Item Group. You can associate multiple items to a Service/Item Group. A Policy Group is associated to a Payer and Policy Type, which is valid within the facility to which it is associated. A single invoice is generated for multiple items that are grouped under a Service Group that is associated Policy Group. The DRG code has to be captured through the 'Setup Service Coverage Detail by Policy' function in Insurance Management module to generate multiple bills, but a single invoice for the items that are grouped under the Service/Item Group.

Service/Item Grouping Setup

From the Invoice Management > Setup, double-click on Service/Item Grouping Setup function.

The Service/Item Grouping Setup screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Commit Form] or press [F10] to save the records or [Clear Record] to delete the data entered.

  3. Click [Execute Query] or press [F8] to query for the existing records.
