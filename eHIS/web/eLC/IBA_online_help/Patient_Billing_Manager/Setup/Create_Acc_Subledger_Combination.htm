Create Acc Subledger Combination  

From this function you can Define a combination of the Subledger and the charting account code.

Create Account Subledger Combination function is used to associate a Chart of Account to the Subledger account. You can post the Subledger entries only when it is associated to a Chart of Account. The combination defined here is referred in the posting account combination. The Subledger code entered must not be the same as that of the parameter Subledger code. By Subledger, it is meant the combination of Subledger group and the Subledger line code.

The following verification is made during Subledger combination:

• The Chart of Accounts with Subledger applicability selected is considered.

• Suspended Chart of Accounts code and Subledger codes are not considered.

• The values in the range specified are selected for processing.

Defining Account Subledger Combination

From the Setup > GL Setup, double-click on Create Acc Subledger Combination function.

The Create Acc Subledger Combination screen gets displayed.

  1. Click [List of Values] to select a range of Chart of Accounts. The Chart of Accounts defined gets listed. The Account Subledger combination is valid for the range of Chart of Accounts selected.

  2. Click [List of Values] to select a range of Subledger. The Subledgers defined get listed. The Account Subledger combination is valid for the range of Subledger selected.

  3. Click [Run Process] to process the log file or [Clear Record] to clear the entered data.

  4. A screen pops-up requesting confirmation. Click [Yes] to define the combination and [No] to cancel. A log file is generated on clicking [Yes].

  5. Click [OK] to view the log file and [Cancel] to exit from the screen.

 

Suspended accounts cannot be combined.
