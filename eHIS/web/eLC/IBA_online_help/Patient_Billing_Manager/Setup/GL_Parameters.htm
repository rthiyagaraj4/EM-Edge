GL Parameters

From this function you can Define the rules that influence functioning of General Ledger functionalities in the Patient Billing Manager module. The Chart of Accounts, Subledger and the Products along with the Chart of Accounts combination functions effectively in the Patient Billing Manager module when the GL Parameters are defined. You can Query to view the parameter defined.

GL Parameters function is used to define the parameters required for the General Ledger functions to function effectively. You have to select a Code length for the functions in the GL Setup menu, so the Code length does not exceed the limit specified. A default value for department, product and Subledger is also defined. You can either enable or disable the account codes.

Define GL Parameters

From the Setup > GL Setup, double-click on GL Parameters.

The GL Parameters screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Commit Form] or press [F10] to save the details or [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to search existing records.

 

The parameter is a one-time setup and you cannot modify the values defined for the mandatory fields.
