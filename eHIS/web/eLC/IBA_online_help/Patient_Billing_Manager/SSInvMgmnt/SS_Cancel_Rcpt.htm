SS Cancel Receipt

From this function you can Cancel the receipts generated for payment references.

The SS Cancel Receipt function enables you to cancel the receipts which are generated to acknowledge the payment against the services utilized. You can cancel the receipt for each individual invoice or cancel the receipts generated for a consolidated invoice.  The receipts generated for the invoice are listed.

Cancelling the receipts for SS Invoices

From the SS Invoice Management > Transaction, double-click on SS Cancel Receipt function.

The SS Cancel Receipt screen gets displayed.

  1. Click [Execute Query] or press [F8] to query for the payment reference documents. Click upward or downward arrow key in the keyboard to navigate to the previous or the next record.

OR

  1. Click [Enter Query] and enter the details in a specific text box to search for the matching records. Press [F8] or click [Execute Query] to query the specific record. <Related Field Description>

  2. The receipts generated are displayed based on the query criteria. The receipt type and number, receipt date and the amount against which the receipt is generated are displayed.

  3. The payment made against the invoices generated are displayed. The invoice number, amount settled, the date when the settled amount is recorded, name of the payer, SS Invoice Type and the patient ID is displayed along with the episode and the visit ID.

  4. Click [Cancel] to cancel the receipt. An authorization screen pops-up. An authorized user is permitted to login and cancel the receipts.

    Enter the following details to login and cancel the payment reference documents:

    1. User ID - Enter the User ID in alphanumeric format. The Users with the check box 'Cancel Invoice' as selected in the 'User Privileges and Cash Counter Access' function are authorized to cancel the receipts. This is a mandatory field.

    2. PIN No - Enter the PIN Number for the respective User. This is a mandatory field.

    3. Reason - Click [List of Values] to select the Reason. The Reasons defined using the 'Reasons for Billing Actions' function with the 'Cancel Invoice' check box as selected get listed.

    Click [OK] to login or [Cancel] to exit from the authorization screen.

     

  5. The selected receipts are cancelled on clicking [OK].

 

An error message 'Cash in hand is not sufficient to refund the amount' gets displayed while trying to refund using the Cash Counter, if there is no sufficient amount in the Cash Counter.
