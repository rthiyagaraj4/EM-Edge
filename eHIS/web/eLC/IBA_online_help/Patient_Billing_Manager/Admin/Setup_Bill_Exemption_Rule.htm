Setup Bill Exemption Rule

From this function you can Define the bill exemption details for a Billing Group. You can Update the Billing Group associated for the existing records.  

The Bill Exemption Rule is applicable only to the selected Billing Groups in the facility. Authorized User is able to exempt the bills only for the selected Billing Groups associated to an Episode Type in the Facility.

Defining Bill Exemption Rule

From the Administration, double-click Setup Bill Exemption Rule function.

The Setup Bill Exemption Rule screen gets displayed.

  1. Select a Facility from the list. The facilities associated to the logged in User get listed. <Related Field Description>

  2. Click [Commit Form] or press [F10] to save or [Clear Record] to clear the entered data.

  3. Click [Execute Query] or press [F8] to view the existing records within the facility.
