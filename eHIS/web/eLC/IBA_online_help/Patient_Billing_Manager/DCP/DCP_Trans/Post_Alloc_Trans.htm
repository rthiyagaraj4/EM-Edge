Post Alloc Trans

From this function you can Post the allocating transaction documents. You can post only the unposted documents.

Post Alloc Trans function is used to post the unposted allocating transactions. You have to post all the transactions to process a period end. The posted documents cannot be edited. Any modification to the allocating transaction documents has to be done before posting. The posted documents are purged during process end.

Post Alloc Trans

From the Deferred Cash Patients > DCP Transaction, double-click on Post Alloc Trans function.

The Post Alloc Trans screen gets displayed.

  1. Click [List of Values] to view and to select a range of allocating Document Types. The allocating Document Number selected under ‘To’ has to be greater’ than the value selected under ‘From’. The list of documents selected gets displayed.

  2. Click Calendar function to select a range of allocating Document Date. The documents for the selected date range get displayed.

  3. Enter the Operator ID in alphanumeric format. The documents generated by the selected Operator ID get listed.

  4. Click [Run Process] to generate a report or [Clear Record] to clear the data entered. Click View, Print or File to view the output online, print the output or to save the output in a file.  

  5. Click [OK] to generate the output and [C] to cancel the transaction.
