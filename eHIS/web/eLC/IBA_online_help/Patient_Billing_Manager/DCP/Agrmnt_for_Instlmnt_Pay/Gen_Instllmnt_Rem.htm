Generate Installment Reminders

From this function you can Generate a reminder for the outstanding documents, where the patient has agreed to settle the bill amount in installments. You can also Cancel the reminders generated.

Generate Installment Reminders function is used to generate a reminder letter to be dispatched to the patients or the sponsor to remind them to settle the amount agreed to be paid in installments. A reminder letter is generated for the outstanding documents. The generated reminder letter is later posted to a patient/guarantor to remind them to settle the installment amount. You also have an option to cancel the records against which the reminders are generated. You can also view the last delivered reminder letter for a selected document.

Generate Installment Reminders

From the Deferred Cash Patients > Agreement for Installment Payment, double-click Generate Installment Reminders function.

The Generate Installment Reminders screen gets displayed.

  1. Select an Outstanding Amount Range. The Outstanding amount range defined using the Reminder Letter Setup function gets listed. The documents with the selected outstanding amount details get listed. Select All to view and to generate the reminder letters either for the selected documents or for all the documents. This is a mandatory field.

  2. Select a Reminder Number from the list. The number of reminder letters defined for the outstanding amount range using the function Reminder Letter Setup gets listed. Select All to view and to generate the reminder letters either for the selected or for all the documents. This is a mandatory field.

  3. The outstanding document details are displayed under the Create tab.

    The agreement document number, date, installment number for which the amount is outstanding get displayed. The agreement document number generated using the function ‘Enter Credit Agreement’ gets displayed. The patient details and the reminder letter details, which were posted, if any, are also displayed.  

    1. Select All – Select All to select all the records for installment reminder letter generation. To clear all the records, which were selected, select the check box Select All. Select the check box against the individual records to generate the installment reminder letters for the selected outstanding documents. A screen pops-up requesting the confirmation to generate the reminder letters. Click [Yes] to generate or [No] to cancel.

    2. [Generate] – Click [Generate] to generate the relevant reminder letter for the selected records.

  4. The outstanding documents with the installment reminder letters generated get displayed under the Cancel tab. You can cancel the records.

    The outstanding document details for which the installment reminder letter is generated get displayed. The document number, type, date and installment number for which the installment reminder letter is generated get displayed. The patient details and the reminder letter details, which were posted, if any, are also displayed.  

    1. Select All – Select All to cancel the reminder letters for all the records against which the installment reminder letter is generated. To clear all the records, which were selected, select the check box Select All. Select the check box against the individual records to cancel the reminder letters generated for the selected documents.

    2. [Cancel] – Click [Cancel] to cancel the reminder letters generated for the outstanding documents.

  5. Clear [Clear Record] to clear each record from the list.
