List of Installment Write-off/Appr/Rejn

From this function you can either Approve or Cancel a Write-Off request placed.      

The List of Installment Write-Off Appr/Rejn function is used to list out the patient records, where the write-off process has been requested, approved or rejected for unsettled installment payments. Initially a request is placed to write-off the patient records using the function ‘Generate Installment Write-off Request’. An authorized user has either to approve or reject the write-off process. You can generate a report to view all the outstanding documents for which the write-off request has been placed and approved or rejected using this function. The report generated displays the outstanding amount against which the write-off request has been placed, the original Document Type and Number, the reference document details and the reason code for approval or rejection of write-off.

List of Installment Write-Off Appr/Rejn

From the Deferred Cash Patients > Agreement for Installment Payment, double-click List of Installment Write-Off Appr/Rejn function.

The List of Installment Write-Off Appr/Rejn screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run Process] to generate a report on the records for which the write-off has been requested and approved or rejected.

  3. You can either View, Print or save the report output in a file. A log file is generated. Click [OK] to view the report output or [Cancel] to exit from the screen.
