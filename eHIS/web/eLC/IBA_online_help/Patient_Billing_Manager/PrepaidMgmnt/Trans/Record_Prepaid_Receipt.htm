Record Prepaid Receipt

From this function you can Record prepaid amount that is received from the Payer and Generate a receipt for the transaction.

The Record Prepaid Receipt function is used to record prepaid amount received from the Payer. A receipt is generated to confirm the payment. An accountant with valid Cash Counter access is authorized to record prepaid receipt for lumpsum amount received from a Payer. You can generate a receipt only if the prepaid letter is generated.

Recording Prepaid Receipts

From the Prepaid Management > Transaction, double-click on Record Prepaid Receipt function.

The Record Prepaid Receipt screen gets displayed.

  1. Select a Deposit Type from the list. The system-defined values are 'LumpSum Deposit' and 'Deposit Against Patient'.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Commit Form] or press [F10] to save the records or [Clear Record] to delete the data.

  4. A receipt number is generated with the document type that is selected for the respective prepaid code.
