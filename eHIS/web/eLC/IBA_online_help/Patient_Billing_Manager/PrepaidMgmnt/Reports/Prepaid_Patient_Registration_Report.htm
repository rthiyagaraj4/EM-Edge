Prepaid Patient Registration Report

From this function you can Generate a report to list the prepaid patient registration details.

The Prepaid Patient Registration Report function is used to generate a report on the patients' who are registered for the prepaid program. The details that are recorded through the 'Patient Registration for Prepaid Program' function are listed.

Generating a report on Prepaid Patient Registration

From the Prepaid Management > Reports, double-click on Prepaid Patient Registration Report function.

The Prepaid Patient Registration Report screen gets displayed.

  1. Click [List of Values] to select a Prepaid Program. The description of the selected Prepaid Program is displayed. You can leave the Prepaid Program as blank to generate a report for all the records that are associated to the selected date range.

  2. Click the Calendar function to select the Register Date range. This is a mandatory field.

  3. Click [List of Values] to select a Register Number. You can leave the Register Number blanks to generate a report for all the register numbers that are associated to the selected date range.

  4. Click [List of Values] to select a Patient ID. You can leave the Patient ID blanks to generate a report for all the records that are associated to the selected date range.

  5. Enter the National ID in alphanumeric format to generate report for the specific patient record. You can leave the National ID as blank to generate a report for all the records that are associated to the selected date range.

  6. Click [Run Process] to generate a report.

  7. Report Option is displayed with the View, Print and File Preferences on clicking [Yes]. Click [OK] to generate or [Cancel] to abort the process.

    Select File to save the output in a file in the local system.

    Select Print to print the report.

    Select View to view the online report.
