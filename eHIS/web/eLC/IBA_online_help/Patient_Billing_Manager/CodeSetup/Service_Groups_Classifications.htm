Service Groups/Classifications

From this function you can Create the Service Groups pertinent to the Facility. You can Query to view the existing Service Groups or a relevant Service Group.

A Service Group is an association of the Service Classifications and the individual Services offered in the Facility. The purpose of the Service Group/Service Classification is to simplify the transaction process by associating a group of service to the patient rather than the individual services, so the bill computation too becomes simpler.

Defining Service Groups

From the Masters > Billing Services, double-click on Service Groups/Classifications function.

The Service Groups/Classifications screen gets displayed.

  1. Enter the details in the mandatory (*) fields found in the following tabs:

Service Groups

Enter the following details to create the Service Groups:

  1. Service Group - Enter the Service Group in alphanumeric format of two characters maximum length. The Service Group Code has to be unique and cannot be changed after creation. This is a mandatory field.

  2. Long Description - Enter the Long Description in alphanumeric format of 80 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the Short Description in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. [Group based Classifications] - Click [Group based Classifications] to create Service Classifications for the respective Service Group or to view the service classifications recorded. The tab gets shifted to Service Classifications on clicking [Group based Classifications].

Service Classification

Each Service Group can have multiple services associated to it. Each Service Group is a set of individual services the facility provides. For Example, the service Gynaecology may include the Registration fee, Consultation fee, Follow up fee or the ward charges. All the basic classifications become a part of the selected Service Group.

Enter the details mentioned to create a Service Classification:

  1. Service Classification - Enter the Service Classification in alphanumeric format of four characters maximum length. The first two characters of the Service Classification code have to start with the Service Group code. This is a mandatory field.

  2. Long Description - Enter the Long Description in alphanumeric format of 80 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the Short Description in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  1. Click [Commit Form] or press [F10] to save the records or [Clear Record] to clear the entered data.

  2. Click [Execute Query] or press [F8] to search for the existing records within the facility.

  3. Click [Group Based Classifications] to view the individual Service Classifications associated to the Service Group.

 

A Service Classification Code has to begin with the Service Group code.
