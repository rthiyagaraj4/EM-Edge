Report GL Distribution Transactions

From this function you can Generate a report to view the debit and credit posted to the General Ledger interface. You can choose to generate a report on a daily or a monthly basis.    

Report GL Distribution Transactions is an option provided to generate a report to get an overview of the transactions posted to the General Ledger interface. A report is generated listing the debit and the credit details along with the Posting Account combination details.

Report GL Distribution Transactions

From the Period End, double-click Report GL Distribution Transactions function.

The Report GL Distribution Transactions screen gets displayed.

  1. Select either ‘daily’ or ‘monthly’ to generate a report. Based on the selection a daily or monthly distribution list is generated.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Run Process] to generate a report. You can either view, print or save the output in a file.
