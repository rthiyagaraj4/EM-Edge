Cashier Journal

From this function you can Query to view the Cashier transactions for a specific period.  

The Cashier Journal function is used to query on the document details for the transactions in the Cash Counter. You also have an option to view the transactions associated to a Cash Control Receipt. You can choose to view the journal for the receipts, refunds issued, cheques bounced, replacement of bounced cheques for the transaction period or you can view all the Cash Counter transactions. The details of the selected transaction against which the Cashier generated the receipt is displayed. You also have an option to view the deposits accepted from a patient. Cash Counter wise Summary can also be analyzed for each Cash Counter shift.

Querying to view the Cashier Journal

From the Queries > Transactions, double-click Cashier Journal function.

The Cashier Journal screen gets displayed.

  1. Click Calendar function to select the date and time. The transactions processed in the Cash Counter during the selected date and time gets listed. This is a mandatory field.

  2. Click [List of Values] to select the Cash Control Receipts. The Cash Control Receipts generated using the function ‘Cash Control Receipt’ gets listed. The Receipt number, type, date and time when the receipt was generated, Cash Counter where the receipt was generated and the document amount against which the receipt was generated, gets displayed.

  3. Select an option from the list to view the Cashier Journal. The document details are displayed based on the option selected.

    1. All – Select All to view the Cash Control receipts generated in all the Cash Counters for the specified date and time range.

    2. Receipt – Select Receipt to view only the receipts generated in all the Cash Counters for the specified date and time range.

    3. Refund - Select Refund to view only the refunds generated in all the Cash Counters for the specified date and time range.

    4. Bounced - Select Bounced to view only the transactions related to the bounced cheques in all the Cash Counters for the specified date and time range.

    5. Replacement - Select Replacement to view only the transactions where the bounced cheques were replaced. The transactions in all the Cash Counters related to bounced cheque replacement for the specified date and time range gets displayed.

  4. Click [View] against each record to view the Cashier Journal for the specific Cash Control Receipt. The patient details get displayed in the screen Cashier Journal. Click [Deposit Dtls] to view the deposit amount collected from the patient.

  5. Click [Summary] to view the summary of all the Cash Counters in the facility. The Cash Counter shift and the amount collected in each shift is displayed. You can also view the  total amount collected from all the Cash Counter transactions. Click [Back] to exit from the Cash Counter wise Summary screen.

  6. Click [Clear Record] to clear each record from the list.
