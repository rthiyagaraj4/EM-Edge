Estimate Surgery Cost

From this function you can Query on the surgery cost details for a patient class in the facility.

This function is used to evaluate the cost incurred for the services availed by the patient at the facility. You can specify the Billing Group, relevant Billing Service and the Billing Class for which the cost amount has to be displayed. On selecting the parameters, the additional services associated to the patient along with the amount and the percentage of the charges for the service availed are displayed. You can also get an estimation of the deposit amount to be collected from the patients by specifying the deposit percentage. The deposit collection amount entered in the transaction functions gets displayed.  

Querying to estimate the surgery cost details

From the Queries > Transactions, double-click Estimate Surgery Cost function.

The Estimate Surgery Cost screen gets displayed.

  1. Enter a valid Patient ID or click [List of Values] to select a Patient ID. The cost estimation for all the services availed from the facility is displayed. The Additional Service Code,  Description for the Service, charge, Department Service Code, Gross Amount, Discount Amount and the Net amount gets displayed.

  2. Click [List of Values] to select a Billing Group. The values defined using the ‘Billing Groups’ function gets listed.

  3. Click [List of Values] to select a Billing Class. The values defined using the ‘Billing Class’ function gets listed.

  4. Click [List of Values] to select the Order Catalog code. You can estimate the cost for the Billing Services associated to the selected Order Catalog code. The description for the selected Order Catalog Code gets displayed.

  5. The total amount of the main and the associated services gets displayed.

  6. Select the check box 'Srl No' to compute the charges including the amount for the selected Order Catalog code. Clear the check box to exclude the amount from estimate cost computation.

  7. The discount defined for the Order Catalog using the function ‘Operation Theatre – Addl charge’ under the ‘Additional Charge Setup by Surgery’ tab gets displayed. The value recorded for ‘Discount 1%’ is displayed in the Multi Surgery Discount (%) field.

  8. The discount percentage for the main surgery and the associated services get displayed.

  9. The charge amount for the main surgery and the associated services get displayed based on the discount percentage.

  10. Select the check box 'Select All' to calculate the total amount for the selected services. If the check box is cleared, the price for the service is not considered towards the calculation of the total amount.

  11. The total amount is calculated automatically on selecting the check boxes against selected services. This field is non-editable.

  12. Enter the percentage for the deposit amount in numeric format. You can get an overview of the amount to be collected as deposit on specifying the entry.

  13. The suggested deposit amount is calculated automatically on entering the suggested deposit percent amount. The deposit amount is calculated based on the percentage of the total amount. The value is generated automatically on specifying the suggested deposit percent and is non-editable.
