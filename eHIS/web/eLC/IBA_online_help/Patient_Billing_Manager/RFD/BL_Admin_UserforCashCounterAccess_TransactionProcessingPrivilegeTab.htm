Transaction Processing Privileges

The Transaction Processing Privileges tab enables you to authorize the Users to either perform or cancel certain transactions in the facility. You have an option to select a User and choose among the list, the transactions which the user is authorized to perform in the Cash Counter.

 

Enter the following details to associate privileges to a user:

 

  1. User ID – Click [List of Values] to select a User ID. The values defined using the Users function in the System Manager module get listed.

  2. User Name – The name of the selected User gets displayed.

 

Privileges

  1. Fin. Dtls Change – Select the check box to permit the User to update the Financial Details in the Master Patient Index/Inpatient and Day Care Management/Outpatient Management/Accident and Emergency module during admission/registration function. Clear the check box and the User cannot update the financial details of the patients.

  2. Credit Authorization – Select the check box to permit the User to offer credit to the patient. On trying to record/update the financial details of a patient during the transaction function in the external modules, a pop-up message gets displayed requesting for an authorization. Clear the check box and the User is not authorized to offer credit to the patient.

  3. Receipt – Select the Receipt check box to authorize the User to generate and settle receipts in the Cash Counter. Clear the check box to generate receipts in the Cash Counter for the transactions in the facility.

  4. Change Receipt Amount – Select the check box to authorize the User to update the amount in the receipt generated for the transactions. Clear the check box to restrict the User from updating the amount in the receipt generated.

  5. Refund - Select the check box to authorize and settle refund bill for the transactions. Clear the check box to generate a refund bill for the transactions in the facility.

  6. Change Refund Amount – Select the check box to authorize the User to update the refund amount. Clear the check box to restrict the User from updating the refund amount to be issued to the patient during bill generation.

  7. Bill Discount – Select the check box to authorize the User to grant discounts to the patients during bill generation. Clear the check box to restrict the User from issuing a discount on the services utilized by the patient.

  8. Bill Adjustment - Select the check box to authorize the User to update the bill details during bill generation. Clear the check box to restrict the User from updating the bill details during bill generation .

  9. Bill Exemption – Select the check box to authorize the User to exempt the bill details during bill generation. Clear the check box to restrict the User from exempting the bill details during bill generation.

  10. Write-Off Approval – Select the check box to authorize the User to approve the Write-off for the outstanding documents. Clear the check box to restrict the User to approve the write-off process.

  11. Reprint Bill – Select the check box to authorize the User to re-print the bill on request. Clear the check box to restrict the User from re-printing the bills.

  12. Reprint Receipt – Select Reprint Receipt to authorize the User to re-print the receipt on request. Clear the check box to restrict the User from re-printing the receipts.

  13. Reprint Refund – Select Reprint Refund to authorize the User to re-print the refund details. Clear the check box to restrict the User from re-printing the refund details.

  14. Installment Payment - Select the check box to authorize user to accept installment payment. Clear the check box to restrict user from accepting installment payments.

  15. Discharge Clearance With Discharge Bill - Select the check box to generate discharge clearance for inpatients with discharge bill. Clear the check box, if the user is not authorized to generate discharge clearance for inpatients with discharge bill. If the check box is cleared, user is not authorized to generate discharge clearance slip through the function 'Record Discharge Clearance'. An error message 'The User is not authorised to generate discharge clearance without bill' is displayed.

  16. Discharge Clearance W/O Disch Bill/Outst Bill - Select the check box to generate discharge clearance for inpatients without discharge bill. Clear the check box, if the user is not authorized to generate discharge clearance for inpatients without discharge bill. If the check box is cleared, user is not authorized to generate discharge clearance slip through the function 'Record Discharge Clearance'. An error message 'The User is not authorised to generate discharge clearance without bill' is displayed.

  17. Acknowledge Handover Request - Select the check box to authorize respective user to acknowledge handover requests. If the check box is cleared, user cannot approve or reject the handover requests using the function 'Acknowledge Handover Request'. This check box is enabled for selection, if the 'Handover without CC Logout' check box is selected in the 'BL Parameters' function.

  18. Approve O/S Amt. MR Request - Select the checkbox to allow the user to access   Approve Medical Record Request screen under Transactions.

 

Cancellation

  1. Receipt – Select Receipt to authorize the User is authorized to cancel the receipts generated for the transactions in the facility. Clear the check box to restrict the User from cancelling the receipts for any transactions in the facility.

  2. Refund – Select Refund to authorize the User to cancel the refund process. Clear the check box to restrict the User from cancelling the refund process.

  3. Bill – Select Bill to authorize the User to cancel the bills generated. Clear the check box to restrict the User from cancelling the bills generated.

  4. Bill Adjustment – Select Bill Adjustment to authorize the user to cancel any adjustments made in the bill. Clear the check box to restrict the User from cancelling the adjustments made in the bill generated.

  5. Bill Exemption – Select the check box to authorize the User to cancel the bill exemptions. Clear the check box to restrict the User from cancelling the bill exemptions.

  6. Approved Write-off – Select the check box to authorize the User to cancel the approved write-off details. Clear the check box to restrict the User from cancelling the write-off details, which are approved.

  7. Credit Card Batch Payment – Select the check box to authorize the User to cancel any Credit Card batch payment process. Clear the check box to restrict the User from cancelling the Credit Card batch payment process.

  8. Installment Payment – Select the check box to authorize the  User to mark the patients for installment payments. Clear the check box to restrict the User from marking installment payments for the patients in the facility.

  9. Cashier Acknowledgment – Select the check box to authorize user to cancel the acknowledgment using the function 'Acknowledge Cashier Trx' . Clear the check box, if the users are not permitted to cancel the acknowledgment.

  10. Discharge Clearance - Select the check box to permit user to cancel discharge clearance slip. Clear the check box, if the user is not permitted to cancel discharge clearance slip. If the check box is cleared, user is not authorized to cancel discharge clearance slip through the function 'Reprint/Cancel Discharge Clearance'. An error message 'The User is not authorised to Cancel discharge clearance' is displayed.

 

Service Discount Applicability by User

Using this tab, you can define the maximum discount amount and the discount percentage the user can authorize for a Service Group, Service Classification or a Billing Service. You can also define the maximum percentage and amount of discount the respective user can permit for a payer while settling the bill amount.

 

Enter the following details to define the service discount applicability by user:

  1. Service Type – Select a Service Type from the list. The system defined values are – Service Group, Service Classification and Billing Service.

  2. Service – Click [List of Values] to select a Service. The user can authorize the maximum discount amount and percentage on the selected service. The values in the list differs based on the Service Type selected.

  3. Description – The description for the selected Service gets displayed.

  4. Allowed – Select the check box to permit the user to authorize discount on the services and the Payer.

  5. Max Serv Disc(%) – Select the check box to authorize the user to permit maximum discount percentage on the Service Type selected. The user is permitted to authorize the maximum discount percentage on the service type for the selected period.

  6. Max Serv Disc Amt – Select the check box to authorize the user to permit maximum discount amount on the Service Type selected. The user is permitted to authorize the maximum discount amount on the service type for the selected period.

  7. Max Payer Disc(%) for Service – Select the check box to authorize the user to permit maximum discount percentage for a Payer on the Service Type selected. The user is permitted to authorize the maximum discount percentage on the service type for a Payer for the selected period.

  8. Max Payer Disc Amt for Serv – Select the check box to authorize the user to permit maximum discount amount for a Payer on the Service Type selected. The user is permitted to authorize the maximum discount amount to a Payer on the service type for the selected period.

  9. Effective From – Click Calendar function to select the date from when the user is authorized to permit the discount on the Service Type selected. The date selected has to be greater than the system date.

  10. Effective To – Click Calendar function to select the date until when the user is authorized to permit the discount on the Service Type selected. The date selected has to be greater than the Effective From date.

 

Ent Serv Group/Line Discount Applicability for User

Using this tab, you can define the maximum discount amount and the discount percentage the user can authorize for a Enterprise Service Group or for a Enterprise Service Group/Line.

 

Enter the following details to define the service discount applicability by user:

  1. Group Type – Select a Group Type from the list. The system defined values are – Ent Service Group, Ent Service Group/Line.

  2. Group Code - Click [List of Values] to select a Group Code. The user can authorize the maximum discount amount and percentage on the selected service groups. The values in the list differs based on the Group Type selected.

  3. Description – The description for the selected Service Group gets displayed.

  4. Allowed – Select the check box to permit the user to authorize discount on the Group Type selected.

  5. Max Ent Disc(%) – Select the check box to authorize the user to permit maximum discount percentage on the Enterprise Service Group selected. The user is permitted to authorize the maximum discount percentage on the group type for the selected period.

  6. Max Serv Disc Amt – Select the check box to authorize the user to permit maximum discount amount on the Group Type selected. The user is permitted to authorize the maximum discount amount on the group type for the selected period.

  7. Effective From – Click Calendar function to select the date from when the user is authorized to permit the discount on the Group Type selected. The date selected has to be greater than the system date.

  8. To – Click Calendar function to select the date until when the user is authorized to permit the discount on the Group Type selected. The date selected has to be greater than the Effective From date.

 

Reimbursable Service by User

Using this tab, you can define the maximum amount an authorized user can reimburse on a Billing Service or a Service Group or a Service Classification. You can also permit the user to override the reimbursable amount by selecting the check box ‘Override Allowed’. The reimbursable amount is valid for the selected period.   

 

Enter the following details to define the service discount applicability by user:

  1. Blng Serv Type – Select a Billing Service Type from the list. The system defined values are Service Group, Service Classification and Billing Service.

  2. Blng Serv Code - Click [List of Values] to select a Billing Service Code. The user can authorize the maximum reimbursable amount and percentage on the selected service code. The values in the list differs based on the Billing Service Type selected.

  3. Description – The description for the selected Service Type gets displayed.

  4. Max Part Reim Amt – Enter the maximum reimbursable amount in numeric format of 15 digits maximum length. It indicates the amount that can be reimbursed by the user on the selected Service Type.

  5. Override Allowed – Select the check box to permit the user to override the reimbursable amount for the selected Service Type. Clear the check box, if the user is not authorized to reimburse the amount on the selected Service Type.

  6. Max Part Reim(%) – Enter the maximum part in terms of percentage that can be reimbursed to the selected Service type. User cannot reimburse the amount more than the amount specified.

  7. Effective From – Click Calendar function to select the date from when the user is authorized to permit the discount on the Group Type selected. The date selected has to be greater than the system date.

  8. To – Click Calendar function to select the date until when the user is authorized to permit the discount on the Group Type selected. The date selected has to be greater than the Effective From date.

 

Nursing Unit by User

Nursing Unit by User tab enables you to permit an authorized user to query the Nursing Unit details. User can query to view the details only in the associated Nursing Units.

  1. Nursing Unit Code – Click [List of Values] to select a Nursing Unit code. The Nursing Units defined in the 'Inpatient and Daycare Management' module get listed.

  2. Description – The description for the selected Nursing Unit gets displayed.

  3. Default Forecast Days – Enter the default forecast days in numeric format of four digits maximum length. The forecast days specified gets displayed in ‘No. of Days Occupancy Forecast is Required’ in ‘Nursing Unit – Billing Info’ Query function. The number of booking details, expected discharge and expected patients in the specific Nursing Unit are listed for the number of days. You can however change the default value to view forecast for the required number of days.

  4. Default – Select the check box to default the Nursing Unit in the ‘Nursing Unit – Billing Info’ Query function for the respective user. You can select only a single Nursing Unit record as the default value.
