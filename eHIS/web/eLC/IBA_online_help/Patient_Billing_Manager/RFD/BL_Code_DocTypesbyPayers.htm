Enter the following details to define the document types:

 

  1. Customer Group - Click [List of Values] to select a Customer Group. The values defined using the function 'Customer Group' in the 'AR Setup' menu gets displayed. This is a mandatory field. The description for the selected Customer Group gets displayed.

  2. Customer - Click [List of Values] to select a Customer Group. The values defined using the function 'Customer Group' in the 'AR Setup' menu gets displayed. This is a mandatory field. The description for the selected Customer gets displayed.

  3. Counter -  Click [List of Values] to select a Cash Counter. The values defined using the function 'Cash Counters' in the 'Facility Specific Setup' menu gets displayed. This is a mandatory field. The description for the selected Cash Counter gets displayed.

 

General

  1. Interim Bill – Click [List of Values] to select a default document type for Interim bill. The Inhouse bill is applicable for the inpatients. The Document Types defined in the Masters with Transaction Type as Invoice/Bill gets listed.

  2. Discharge - Click [List of Values] to select a default document type for Discharge bill. The Document Types defined in the Masters with Transaction Type as Invoice/Bill gets listed.

  3. U/c Advice - Click [List of Values] to select a default document type for Undercharge Advice. The Document Types defined in the Masters with Transaction Type as Invoice/Bill gets listed. The Undercharge Advice text box has to be left blank, if Inpatients Billing is not enabled.

  4. Zero Amt –IP - Click [List of Values] to select a default document type to generate and print inpatient bills with zero amount. The Document Types defined in the Masters with Transaction Type as Invoice/Bill gets listed. The Zero Bill – IP text box has to be left blank, if zero bill generation/printing is not required.

  5. OP/Emg - Click [List of Values] to select a default document type for outpatient visit or emergency bills. The Document Types defined in the Masters with Transaction Type as Invoice/Bill gets listed. The OP/Emg Bill text box has to be left blank, if Outpatients Billing is not enabled. This covers both Cash memos and acknowledgement slips (for credit patients).

  6. Zero Amt-OP/Emg - Click [List of Values] to select a default document type to generate and print outpatient/emergency patient bills with zero amount. The Zero Bill-OP/Emg text field has to be left blank, if zero bill generation/printing is not required for outpatients/emergency patients. The Document Types defined in the Masters with Transaction Type as Invoice/Bill gets listed.

  7. External - Click [List of Values] to select a default document type for external patient bills. The external bill text box has to be left blank, if referral billing is not enabled. The Document Types defined in the Masters with Transaction Type as Invoice/Bill gets listed.

  8. Zero Amt– Ext - Click [List of Values] to select a default document type to generate and print external patient bills with zero amount. The zero bill – ext text box has to be left blank, if zero bill generation/printing is not required for referrals. The Document Types defined in the Masters with Transaction Type as Invoice/Bill gets listed.

Bill Refund Document

You can select a default document type for bill refund transactions through this tab. The selected document type is valid for the selected Customer Group, Customer and the associated Cash Counter. It is valid for all Customers', Customer Groups and Cash Counters when the value for all fields are selected as '**'.

  1. Refunds - IP - Click [List of Values] to select a bill refund document type for an Inpatient. The Document Types with the 'Trx Type' as 'Invoice/Bill' are listed.

  2. Refunds - OP/Emg. - Click [List of Values] to select a bill refund document type for an Outpatient or an Emergency patient. The Document Types with the 'Trx Type' as 'Invoice/Bill' are listed.

  3. Refunds - Ext. - Click [List of Values] to select a bill refund document type for an External patient. The Document Types with the 'Trx Type' as 'Invoice/Bill' are listed.

  4. Misc. Refunds - Click [List of Values] to select a bill refund document type for miscellaneous transactions. The Document Types with the 'Trx Type' as 'Invoice/Bill' are listed.

Prepaid Deposit Document

  1. Prepaid Lumpsum Deposit - Click [List of Values] to select a default document type for prepaid lumpsum deposits. The document type selected is used during receipt generation through the 'Record Prepaid Receipt' function for lumpsum deposit types.

  2. Prepaid Patient Deposit - Click [List of Values] to select a default document type for prepaid patient deposits. The document type selected is used during receipt generation through the 'Record Prepaid Receipt' function for deposits against patient.
