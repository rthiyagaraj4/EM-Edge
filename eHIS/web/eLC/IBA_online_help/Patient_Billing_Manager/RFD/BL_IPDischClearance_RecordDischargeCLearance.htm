Enter the following details to query inpatient records matching the search criteria:

  1. Nursing Unit - Click [List of Values] to select the Nursing Unit. A list of patients' admitted to the selected Nursing Unit and matching the query criteria are listed.

  2. Discharge Advice Date From/To - Click the Calendar function to select date on which the discharge advice is prepared. The inpatients' with the discharge advice generated during the selected period are listed.

  3. Patient - Click [List of Values] to select the patient ID. A list of inpatients' with the discharge advice are listed. The name of the patient is displayed in the adjacent text box.

  4. Episode ID - Click [List of Values] to select the Episode ID. A list of inpatients' with the discharge advice are listed.

  5. Option - Select an option from the list. The system-defined values are 'With Discharge Bill', 'Without Discharge Bill' and 'Both'. Select 'With Discharge Bill' to generate discharge clearance for inpatients' with discharge bill. Select 'Without Discharge Bill' to generate discharge clearance for inpatients' without discharge bill. Select 'Both' to generate discharge clearance for inpatients' with and without discharge bill.
