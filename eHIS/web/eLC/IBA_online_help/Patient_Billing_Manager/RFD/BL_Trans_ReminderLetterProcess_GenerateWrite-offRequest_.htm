Enter the following details to generate write-off request:

 

  1. Billing Group Category – Select a Billing Group Category. The values defined using the Billing Group Category function get listed. This is a mandatory field.

  2. Outstanding Amount Range – Select an Outstanding Amount range. The values defined using the Reminder Letter Setup function and associated to the selected Billing group Category get listed. This is a mandatory field.

 

Range in Days

  1. Outstanding Period – Enter the Outstanding Period in numeric format. The documents with the outstanding payments are displayed. The Period specifies the number of days for which the balance is not cleared.

  2. Bill Date Range – Click the Calendar function to select the date range. The bills generated during the specified period and are not cleared get listed.

  3. Option – Select a value from the list. The system defined values are “Donot Include Rejections” and “Include Earlier Rejections”.

  4. Donot include Rejections - The documents where the write-off was rejected during earlier transactions are not listed.

  5. Include Earlier Rejections - The documents where the write-off was rejected during earlier transactions are also listed.

 

Inpatient Encounters

  1.  Highest Class Bed Occupied – Click [List of Values] to select a Level of Service. This is applicable for inpatient transactions. You can approve the write-off for the outstanding documents with the highest bed class occupied.

  2. Nationality – Select a value from the list. The values are system defined – “All”, “Non-Citizen” and “Citizen”. This is a mandatory field.

  3. Write-off Process Date – The Write-off Process Date is set as the system date, by default.

  4. Doc No. – The Write-off Document Type selected in ‘Write-off’ tab in the BL Parameters function gets displayed, by default. The Document Number is system generated.

  5. [Authorize] - Click [Authorize] to approve the Write-off Request. An authorization screen appears requesting to enter the login details. The details are displayed based on the entry made during the reminder letter dispatch .

  6. [Process] - Click [Process] to approve the Write-off Request for the selected records. The respective records are dismissed from the database on processing.
