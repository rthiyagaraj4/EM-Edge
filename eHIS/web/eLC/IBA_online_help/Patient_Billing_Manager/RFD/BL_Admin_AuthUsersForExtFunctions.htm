Enter the following details to define authorized users for external functions:

  1. Authorization Type – Select an Authorization Type. Based on the Authorization Type selected, the values are displayed in the list. This is a mandatory field.

  2. User – Select User to permit the selected User to access the functions from the external modules.

  3. Responsibility - Select Responsibility to permit the logged in User to access the functions from the selected Responsibility.

  1. Active – When you select the check box, only the active values for Authorization Type get displayed. When you clear the check box, even the inactive functions are also listed. The check box is selected, by default.

  2. Description/Name – The Description for the selected Authorization Type gets displayed.

  3. Pract Type – The Practitioner Type selected while defining the selected User in the System Manager module gets displayed. You cannot update the Practitioner Type.

 

List of Privileges

  1. Module ID - Click [List of Values] to select a Module ID.

  2. Description – The Description for the selected Module ID gets displayed.

  3. Function ID – Click [List of Values] to select a Function ID. The functions in the selected module get listed. Active or Inactive functions get listed based on whether the check box ‘Active’ is selected.

  4. Function Name – The name of the Function gets displayed.
