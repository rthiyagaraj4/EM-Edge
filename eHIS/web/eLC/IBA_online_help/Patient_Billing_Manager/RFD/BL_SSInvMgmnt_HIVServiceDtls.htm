Enter the following details to associate Service Details:

 

  1. SS Invoice Type Code - Click [List of Values] to select an Invoice Type. The Invoice Types defined using the 'SS Invoice Types' function get listed. This is a mandatory field.

  2. HIV Service Type - Click [List of Values] to select a HIV Service Type. The Service Types defined using the 'HIV Service Type' function get listed. This is a mandatory field.

  3. HIV Related - The check box gets selected if the Invoice Type is HIV related and it is cleared, if the Invoice Type is non-HIV related.

  4. Effective From - Click the Calendar function to select the date from when the Billing Service is valid. This is a mandatory field.

  5. Effective To - You cannot enter the Effective Date. It gets displayed, if you define another 'Effective From' date for the same Billing Service. The 'Effective From' date is considered as the 'Effective To' date.

  6. Blng Serv Code - Click [List of Values] to select a Billing Service code. The values defined using the 'Billing Services' function get listed. This is a mandatory field.

  7. Description - The description of the selected Billing Service is displayed.
