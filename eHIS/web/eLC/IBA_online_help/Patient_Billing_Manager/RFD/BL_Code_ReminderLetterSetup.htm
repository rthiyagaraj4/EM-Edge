Enter the following details to set up Reminder Letter:

 

  1. Applicable for Transaction – Select a value from the list. The values are system defined - 'Bill Outstanding Payment Reminder' and 'Installment Payment Reminder'. The Reminder Letter is set up for the transaction selected. This is a mandatory field.

  2. Bill Outstanding Payment Reminder – Select Outstanding Payment Reminder to setup a reminder letter based on a Billing Group.

  3. Installment Payment Reminder – Select Installment Payment Reminder to set up a reminder letter for installment payments.

  1. Billing Group Category - Click [List of Values] to select a Billing Group Category. The values defined using the function Billing Group Categories gets listed. The Billing Group Category field is enabled when ‘Applicable for Transaction’ field is selected as “Bill Outstanding Payment Reminder”.  

 

Reminder Letter by Outstanding Amount

  1. Lower Limit - Enter the lower limit in numeric format for the outstanding amount. The lower limit can have a maximum of ten digits before the decimal and two digits after the decimal. This is a mandatory field.

  2. Upper Limit - The upper limit is automatically calculated when the Lower Limit and the maximum number of reminders are set.

  3. Max. no of Reminders - Enter the maximum number of reminders in numeric format with a maximum of two digits. The number specifies the number of reminder letters to be dispatched to the patient/customer when the amount is not settled. This is a mandatory field.

  4. Days after Last Reminder for Write-Off - Enter the number of days to write-off the outstanding bill details after the last reminder is dispatched. This is a mandatory field.

 

Details by Reminder No. for Outst. Amt Range

  1. Reminder No – A list of reminder letters are generated by the system based on the value set in ‘Max. no for Reminders’ field. This is a mandatory field.

  2. Description - The description gets displayed automatically.

  3. No of days after Due Date/Last Reminder Date - Enter the number of days in numeric format. The value indicates the number of days from the due date or the last reminder date when the reminder letter has to be dispatched. This is a mandatory field.

  4. Reminder Format – Click [List of Values] to select an appropriate Reminder Format. The values defined using the Reminder Letter Format get listed. This is a mandatory field.

  5. Description - The description for the selected Reminder Format gets displayed.
