Package Related Other Setup

The Package Related other Setup tab enables you to define the additional factor for a Billing Class, define the discount for a Customer, associate charge based service limits or exclude the material items from a package.

 

Additional Factor by Billing Class

You can define the additional charge for a Billing Class based on which the package has to be charged. The additional factor is multiplied with the rate for the Billing Class to compute the price.

 

Enter the following details to define the additional factor:

 

  1. Blng Class Code - Click [List of Values] to select the Billing Class code. This is a mandatory field.

  2. Description - The description for the selected Billing Class gets displayed.

  3. Blng Class Addl Factor - Enter the additional factor in numeric format. This is a mandatory field.

 

Customer Discount

You can define the discount percentage for a specific customer. The customer is charged for the package after deducting the discount amount from the actual package price.

 

Enter the following details to define discount percentage for a Customer:

 

  1. Customer Code - Click [List of Values] to select the Customer code. This is a mandatory field.

  2. Description - The description for the selected Customer gets displayed.

  3. Discount Type - Select a Discount Type from the list. The system-defined values are 'Percent' and 'Value'. Select 'Percent' to enter the percentage of discount amount or 'Value' to enter the discount amount in numeric format. This is a mandatory field.

  4. Discount (%) - Enter the discount percentage or amount in numeric format. You can specify a value upto '100' when the 'Discount Type' is selected as 'Percentage'. During transactions, the 'Discount Amount' should not exceed the discount amount recorded for the Customer. The discount amount is validated against the package amount during transactions. You cannot enter a negative value for discounts. You cannot record duplicate discount values for a Customer. This is a mandatory field.

 

Charge Based Service Limit

You can select and associate the charge based services to the package and specify the limit amount for the charge based service within the package. The charge is computed for the service based on the package setup in BL Parameter.

 

Enter the following details to define the limit amount for the charge based services:

 

  1. Blng Serv Code - Click [List of Values] to select the Billing Service code. This is a mandatory field.

  2. Description - The description for the selected Billing Service gets displayed.

  3. Service Limit Amt - Enter the service limit amount in numeric format. This identifies the amount limit for the service from the package amount.

 

Excluded Material Items

You can select the services, which are to be excluded from the package coverage. The package amount cannot be charged for the excluded material items, but the amount is charged based on the setup for the packages in the BL parameter.

 

Enter the following details to select the Material Items:

  1. Item Code - Click [List of Values] to select the Item Code. This is a mandatory field.

  2. Description - The description for the selected Item Code gets displayed.
