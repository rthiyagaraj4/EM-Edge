Items by Payer Group

The Items by Payer Group tab enables you to associate material items to an Item Group. The purpose of associating items to an Item Group is to define the Payer Group who settles the bill amount for the items utilized from the Item Group by a relevant patient category. The details are printed on the bill during bill generation.

 

Enter the following details to associate material items to a Payer Group:

 

  1. Payer Item Group – Click [List of Values] to select a valid Payer Item Group. A Payer Item Group selected in the ‘Payer Item Groups’ gets displayed.

  2. Non-DRG Related - The check box is selected, if the Payer Item Group is selected as Non-DRG Related under ‘Payer Item Groups' tab. The check box is cleared, if the Payer Item Group is DRG related.

  3. Item Code – Click [List of Values] to select Item Codes from the list. The material items defined in the Material Base module gets listed. You can select and associate any number of Items to the Payer Item Group. This is a mandatory field.

  4. Description – The Description for the respective Item Code is displayed.

  5. Effective [From-To] – Click the Calendar function to select the date range during which the selected item is under selected payer coverage. The coverage amount for the specific Material Item is calculated based on the policy definition for the respective patient class, Payer and Payer Group combination. The respective Material Item is under policy coverage for the date range selected.

  6. Non-DRG Related - Select the check box, if the material item is Non-DRG Related. The DRG Related services are valid for inpatient transactions only.  Clear the check box, if the material items are DRG related.
