Maintain Prices

The Maintain Prices tab enables you to update the prices for the Billing Services associated to the Price Class. You can update the discount percentage or select discount applicability and enter a new discount percentage. The price defined for a patient class and the associated Price Class gets displayed under the ‘Price Class Specific Price’ tab in the Billing Services function. If the selected Price Class code is package related, the package price, quantity limit and the amount limit for the services gets displayed.

 

Enter the following details to update the price:

 

  1. Facility – The logged in Facility name gets displayed, by default.

  2. Price Class Code – Click [List of Values] to select a Price Class and to view or update the price or discounts defined for a patient class associated to the Billing Services and a Price Class. Click [Execute Query] to view and update the prices and the discounts defined for the Price Classes in the facility. The description of the currency associated to the selected Price Class Code using the ‘Price Class Setup’ tab gets displayed adjacent to the Price Class Code.

  3. Package Price - The package price defined under the ‘Price Class Setup’ tab gets displayed.

  4. [View Past Prices] - Click [View Past Prices] to view previous prices defined for a patient class, associated to a Billing Service and the selected Price Class.

The Billing Services, Descriptions and the period until when the Billing Services are associated to the Price Class gets displayed. Click [Create Record] to associate a new Billing Service to the Price Class.

  1. Billing Service – Click [List of Values] to select a Billing Service. The Billing Services, which are already associated to the Price Class get displayed along with the effective from date. This is a mandatory field.

  2. Description - The Description for the respective Billing Service gets displayed.  

  3. Valid From – Click Calendar function to select a date. The Billing Service is valid for the selected Price Class from the date selected. The dates for the Billing Services, which are already associated, get displayed. The date selected has to be greater than the system date. This is a mandatory field.

  4. Valid To - Click Calendar function to select a date. The Billing Service is valid for the Price Class up to the date selected. The date for the Billing Services, which are already associated, gets displayed. The date entered has to be greater than the valid from date.

  5. Qty Limit - The field is displayed only if the selected Price Class is package related. You can enter the quantity of the services that are to be included within the package coverage. If the Amount Limit is not specified, system checks only the quantity limit while associating the services to the patient which are under package coverage. If the quantity of the service is more than the quantity limit specified, the service is charged based on the setup defined in the BL Parameters for the packages.

  6. Amt Limit - Enter the amount limit for the respective service. The price is computed as (Qty Limit)*(Rate defined for the Pricing Class). If the Quantity Limit is not specified, system checks to see if the price for the services utilised exceeds the Amount Limit entered. If the Quantity Limit is specified and the quantity of the services exceeds the quantity specified, the patient is charged based on the setup defined for the packages in the BL Parameters.

 

Pricing by Billing Class for Billing Service

  1. Class – Click [List of Values] to select a Billing Class. The values defined using the Billing Class function under the ‘Billing Rule’ menu get listed. Select ‘**’ to define the prices for all the patient class in the facility. The prices are valid for the patient class associated to the selected Billing Service and Price Class in the facility. This is a mandatory field.

  2. Description – The Description for the selected Billing Class gets displayed.

  3. R/F – Select a value from the list. The values are system defined – Rate and Factor.

  4. Rate – Select Rate, if the service price has to be charged for the patient class associated to the Price Class.

  5. Factor – Select Factor and the service price is multiplied with the factor entered for a patient class.

  1. IP Rate/Factor - Enter the rate/factor in numeric format for Inpatients. The text box is enabled only if the Class selected is an ‘Inpatient’ Class or ‘All’.

  2. DC Rate/Factor - Enter the rate/factor in numeric format for Daycare patients. The text box is enabled only if the Class selected is a ‘Daycare’ Class or ‘All’.

  3. OP Rate/Factor – Enter the rate/factor in numeric format for Outpatients. The text box is enabled only if the Class selected is an ‘Outpatient’ Class or ‘All’.

  4. Emg. Rate/Factor - Enter the rate/factor in numeric format for Emergency patients. The text box is enabled only if the Class selected is an ‘Emergency’ Class or ‘All’.

  5. Ext. Rate/Factor - Enter the rate/factor in numeric format for Referral patients. The text box is enabled only if the Class selected is a ‘Referral/External’ Class or ‘All’.

  6. Min. Charge - Enter the Minimum Charge for the selected patient class. The minimum charge is applicable only if the Billing Service is defined as Charge based. The price defined is valid for the selected period.

  7. Maximum Charge – Enter the Maximum Charge for the selected patient Class. The charge is applicable for the selected patient class. The price defined is valid for the selected period.

  8. Base Qty - Enter the Base Quantity in the numeric format. The price for the service when associated to the selected patient class is calculated as the product of the rate/factor set and the base quantity defined.

  9. Allowed – Select the check box, if the Billing Service is reimbursable for the selected patient class. Clear the check box, if the respective Billing Class is non-reimbursable for the selected patient class.

  10. Type – Select a value from the list to identify the reimbursable type. The system-defined values are – ‘Full’, ‘Partial’ and ‘NA’.

  11. Full – Select Full to refund the entire amount paid by the patient towards the service.

  12. Partial – Select Partial to refund only a fraction of the amount paid towards the service. A part of the amount is deducted for the service availed and the remaining amount is refunded to the patient/payer. You have to specify the amount to be reimbursed in the field ‘Reimbursable Amount’ under the ‘General’ tab. The Charge is specified only if the ‘Charge Type’ is set to ‘Charge’.

  13. Not Applicable – Select NA, if the service is not reimbursable.

  1. Reim Amt – Enter the amount, which can be reimbursed to the patient.

  2. Override – Select the check box, if the respective Billing Service is reimbursable for the selected patient class. The override option for the patient class takes precedence over the override check box in the header field. If the service is not reimbursable and the check box is selected for the patient class, it indicates that the Billing Service is reimbursable for the selected patient class.

  3. Gross/Net Applicability – Select a value from the list. The values are system defined – Gross and Net.

  4. Gross - Select Gross to charge the Gross amount for the service. The charge towards the service is calculated based on the discount and the price defined for the patient class.

  5. Net – Select Net to charge the Net amount for the service. The service charge is calculated based on the price towards the patient class. The discount amount for the service is not considered during bill generation.

  1. Check for Cost – Select the check box, if the Cost Amount is applicable for the selected patient class and the associated Price Class. Clear the check box, if the Cost Amount is not considered towards the charge calculation.

  2. Apply Disc – Select the check box, if Discount is applicable for the selected Billing Class. You can define the discount percentage, if the check box is selected. Clear the check box, if the discount is not applicable for the patient class associated to the Billing Service.

 

Applicable Discounts for Service and Billing Class

The text boxes are enabled, if the ‘Apply Disc’ check box is selected.

  1. Discount Indicator – Select a value from the list to issue discount for the Billing Service. The system-defined values are ‘Amount’ and ‘%’. Select Amount to specify the amount that has to be issued as a discount amount on the Billing Service. Select Percentage to specify the percentage of amount that has to be issued as a discount amount on the Billing Service.

  2. IP - (%/Amt) - Enter the value to issue discount towards the service for Inpatients. The price is calculated based on the percentage of amount and the service price defined. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  3. DC - (%/Amt) - Enter the value to issue discount towards the service for Daycare patients. The price is calculated based on the percentage of amount and the service price defined. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  4. OP - (%/Amt) -  Enter the value to issue discount towards the service for Outpatients. The price is calculated based on the percentage of amount and the service price defined. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  5. Emg. - (%/Amt) - Enter the value to issue discount towards the service for Emergency patients. The price is calculated based on the percentage of amount and the service price defined. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  6. Ext - (%/Amt) - Enter the value to issue discount towards the service for External/Referral patients. The price is calculated based on the percentage of amount and the service price defined. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  7. Valid From – Click Calendar function to select a date. The discount percentage defined for the patient class is valid from the date selected. The discount period has to be within the Billing Service period. The date selected has to be greater than the system date. This is a mandatory field.

  8. Valid To - Click Calendar function to select a date. The discount percentage defined for the patient class is valid until the date selected. The discount period has to be within the Billing Service period. The date selected has to be greater than the From date. This is a mandatory field.
