By Bed Type

The By Bed Type tab enables you to define the prices for a Bed Type in a Nursing Unit. A Billing Class is also associated to the Bed Type. The Billing Class indicates the patient class for whom the Bed Type is applicable. The patient associated to a bed type is charged based on the Billing Service prices.

 

Enter the following details to define the Bed Charge based on the Bed Type:

 

  1. Bed Type – Click [List of Values] to select a Bed Type. The values defined using the function Bed Type in the Inpatient and Day Care Management module get listed. This is a mandatory field.

  2. Description - The description for the selected Bed Type gets displayed.

  3. Billing Class - Click [List of Values] to select a Billing Class. The values defined using the function Billing Class get listed. This is a mandatory field.

  4. Description - The description for the selected Billing Class gets displayed.

  5. Billing Service for Bed Charge - Click [List of Values] to select a Billing Service. The values defined using the Billing Service function get listed. The Bed Type and the selected Billing Class is charged based on the prices defined for the selected Billing Service. This is a mandatory field.

  6. Description - The description for the selected Billing Service gets displayed.
