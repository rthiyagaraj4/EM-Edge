Enter the following selection criteria:

 

  1. Patient No [From–To] - Click [List of Values] to view and to select a range of Patient Number. If the patient number is not selected, the system assumes the highest value for the ‘To’ field and the lowest value for the ‘From’ field.

  2. Doc Nature [From–To] - Enter the Document Nature in alphanumeric format. The report is generated based on the range of Document Nature selected. The Document Nature is recorded for a deferred cash patient during the transactions.

  3. Doc Reference [From–To] - Enter the value for Document Reference. The report is generated for the selected range of Document Reference.

  4. Report Order – Select a valid Report Order from the list. The system defined values are – “Patient Number”, “English Name” and “Local Language Name”. The report output is sorted based on the value selected for Report Order.

  5. Ageing Cut-off Date – Click Calendar function to select the date. The current system date is displayed, by default. The date selected has to be equal to or lesser than the current system date. This date is used to calculate the ageing details of a document, which is outstanding.

  6. Show Associated Transactions – Select the check box to view the associated transactions linked to the originating transaction.

  7. Print Zero Bal Documents – Select the check box and the original documents with zero outstanding balance are also printed. Clear the check box and the original documents with zero outstanding balance are not printed.
