Enter the following to set Workstation based Print Routing/Parameters:

 

Workstation

  1. Workstation No – The system number is displayed as the Workstation Number, by default. You cannot edit the Workstation Number.

 

Print Routing

The Print Routing details defined in the System Manager module are listed. Double-click on the selected record to update the Print Routing details or you can record the destination printer for a selected report ID.

 

Workstation Specific Reports

  1. Report ID – Press [F9] to select a Report ID from the [List of Values]. The entries are populated only when valid Report ID's are created initially under Print Routing Menu in the System Manager Module.

  2. Facility – Click [List of Values] to select a facility. The entries are populated only when valid Facilities are set initially in the System Manager Module.

  3. Dest Type – Click [List of Values] to select a Destination Type. On selecting the Report ID and facility, the related data is populated in the list. The listed items are Department, Clinic, Ward and Location.

  4. Dest Code – Enter the Destination Code in alphanumeric format of six characters maximum length. This field is mandatory. Criteria for destination code can be:

  5. Department - Operating Department Code

  6. Ward - Ward Code

  7. Clinic - Clinic Code

  1. Description – The Description for the selected Destination Code is displayed in the text box.

  2. Copies – The number of copies entered while defining the Print Routing rule in the System Manager module gets displayed. You can update the required number of copies. When the report is generated based on the Print Routing Rule, number of copies is generated based on the entry made.

  3. Printer 1, 2, 3 – The Printer associated to the report while defining Print Routing in the System Manager module gets listed. You can update the printer details be entering a valid Printer name.  

  4. Start Time – Enter the starting time for the associated printers in hh:mm format. The time indicates the time scheduled for the respective printers to carry on with the print routing.

  5. Report Name – On selecting the Report ID and Facility, the Report Name is displayed automatically in the field.

 

Workstation Parameters

The Workstation Parameters are system-defined. You have to update the parametric values to generate reports in the local machine.

  1. Parameter Description – Double-click on the Parameter Description to select it and to update the parametric values. You have to update the parametric values to generate and to view the reports in the local machine.

  2. Parameter Value – Enter the complete path to the directory structure, where the report output has to be stored.
