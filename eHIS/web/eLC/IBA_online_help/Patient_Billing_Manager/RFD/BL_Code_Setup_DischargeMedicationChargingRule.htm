Enter the following to define discharge medication charging rule:

 

Customer Grp

  1. Cust Group Code - Click [List of Values] to select the customer group code.

  2. Customer Code - Click [List of Values] to select the customer code.

  3. Cust Name – Click [List of Values] to select the customer name.

  4. Policy Code – Click [List of Values] to select the policy code.

  5. Blng Class Code - Click [List of Values] to select the billing class code.

  6. Blng Class Description – On selecting the billing class code, the billing class description is displayed.

  7. Apply for Other Payers YN – Select the check box to charge discharge medications for all the payers/payer groups/policies of the patient at the OP rate if the patient has multiple payers/payer group/policies. Clear the check box to charge discharge medication for payers/payer group/policies in the IP rate.

 

Cash Blng Grp

  1. Blng Grp ID - Click [List of Values] to select the billing group ID.

  2. Blng Grp Desc - On selecting the billing group ID, the billing group description is displayed.

  3. Blng Class Code - Click [List of Values] to select the billing class code.

  4. Blng Class Description – On selecting the billing class code, the billing class description is displayed.
