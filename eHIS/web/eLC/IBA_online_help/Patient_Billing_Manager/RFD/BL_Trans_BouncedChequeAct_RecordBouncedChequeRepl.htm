Enter the following details to replace the bounced cheque details:

 

  1. Amount Settled - Enter the amount settled. The amount has to be positive only. If this document is allocating in nature, the amount has to be equal to or lesser than the original document's outstanding amount. For this comparison, the absolute value of the amount is considered. If the document is originating in nature, there is no comparison.

  2. Settlement Type - Click [List of Values] to view and to select a valid Settlement Type code. The description for the selected Settlement Type gets displayed. If the Settlement Type is Cash, the approval code and cheque/card no. fields get disabled. If the Settlement Type is not Cash, the approval code and cheque/card no. fields are enabled for entry.

  3. Cheque/Card No – Enter the Cheque or the Credit Card number, if the Settlement Type is non-cash. For example, if the settlement is by Credit Card, enter the Credit Card identity , and expiry date of the Credit Card. If the settlement type is by Cheque, enter the Cheque details and the date.

  4. Date – Click Calendar function to select the date. This field is mandatory, if the Settlement Type is not Cash. For example, if the settlement is by Credit Card, enter the Credit Card identity  and expiry date of the Credit Card. If the settlement type is by Cheque, enter the Cheque details and the date.

  5. Bank/Card Co. Name - Click [List of Values] to view and to select an approved Credit Card company name or an approved Bank or branch name. If Settlement Type is non-Cash, the Bank and credit card company name is mandatory. For example, if the Settlement Type is Credit Card, enter the Credit Card Company name. If the Settlement Type is Cheque, enter the bank’s name or bank’s branch name.

  6. Online Approval – Select the check box, if the Settlement Type is Credit Card. Clear the check box for any other Settlement Type.

  7. Batch No – Enter the credit card Batch Number in alphanumeric format of eight characters maximum length. This field is enabled only if the Online Approval check box is selected.

  8. Approval Code – Enter the Approval Code in alphanumeric format of 30 characters maximum length. If the Settlement Type is selected as Credit Card, Approval Code of the Credit Card has to be recorded.

  9. Sale Draft No. – Enter the Approval Code in alphanumeric format of 20 characters maximum length. The Sale Draft number is mandatory, if the Settlement Type is selected as Credit Card. The sale draft number is captured while swiping the card to check for the validity.

  10. Payer’s Name - Enter the payer name in alphanumeric format of 40 characters maximum length. The Payer name is valid, if the receipt nature code is miscellaneous (MI).

  11. Through Mail – Select the check box, if the settlement amount against a bill or a receipt is received through post. If the check box is selected, a Medical Report Payment receipt is printed for the selected Document Type.

  12. Remarks - Enter the remarks in alphanumeric format of 50 characters maximum length. This narration is recorded as part of General Ledger distribution and printed as a part of the daily Cash Collection report.
