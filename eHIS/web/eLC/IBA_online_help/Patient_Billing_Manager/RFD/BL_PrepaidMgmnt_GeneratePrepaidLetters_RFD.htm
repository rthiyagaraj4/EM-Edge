Enter the following details:

 

  1. Payer Group - Click [List of Values] to select a Payer Group. The values defined through the 'Payer Group' function are listed. This is a mandatory field.

  2. Payer - Click [List of Values] to select a Payer. The values defined through the 'Payer' function are listed. This is a mandatory field.

  3. Prepaid - Click [List of Values] to select a Prepaid Code. The values defined through the 'Prepaid Payment Setup Rule' function are listed. This is a mandatory field.

  4. Protocol - Click [List of Values] to select a Protocol. The values defined through the 'Prepaid Payment Setup Rule' function are listed. This is a mandatory field.

  5. Register Date [From - To] - Click the Calendar function to select Registration date range. The patient records with the registration date matching the selected date range are listed.

  6. Due Date [From - To] - Click the Calendar function to select due date range. The patient records with the due date matching the selected date range are listed.

  7. Payment No. [From - To] - Click [List of Values] to select a payment number range. The values defined through the 'Prepaid Payment Setup Rule' function are listed. This is a mandatory field.

  8. Patient Id - Click [List of Values] to select a Patient ID. The name of the selected patient is displayed in the adjacent text box.

  9. Register No. - Click [List of Values] to select a Registration Number.
