Visit Exemption Exceptions

Visit Exemption Exceptions tab enables you to define the number of days and the visits exempted for an outpatient visit to the facility after patient checkout. You also have an option to exempt the hospital fee or the medical personnel fee for the patient visits. The specified number of visits are considered free within the number of exempted days. The exemption is limited to the specialty, where the patient visit is registered.

 

Enter the following details:

  1. Visit Exemption Rule – Enter the Visit Exemption Rule Code in alphanumeric format of two characters maximum length.

  2. Description – Enter the description for the respective Visit Exemption Rule code in alphanumeric format.

  3. No of Days Exempted – Enter the number of days in numeric format. It indicates the number of days an outpatient can visit the facility to avail an exemption after the checkout from the facility. The patient visits on the number of days specified are not charged and the number of days are counted from the subsequent day of the patient checkout from the facility.

  4. No of Follow-up Visits Exempted – Enter the number of follow-up visits in numeric format. It indicates the number of visits to the facility within the specified number of days after checkout for which the patient can avail an exemption.

  5. Exempt Hospital Fee – Select the check box to exempt the hospital fee for the number of exempted outpatient visits to the facility after checkout. Clear the check box to charge the hospital fee during the exempted visits.

  6. Exempt Medical Personal Fee – Select the check box to exempt the doctor rounds charge for the number of exempted outpatient visits to the facility after checkout. Clear the check box to charge the doctor rounds during the exempted visits. This checkbox appears if the 'Consultation Charge in Multiple Stages' checkbox is selected in the BL Parameters function. If the 'Consultation Charge in Multiple Stages' checkbox is cleared, this check box is not available for selection. If this check box is cleared, the doctor fee is exempted during the patient’s visit to the facility after check out.
