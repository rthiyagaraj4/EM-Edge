Enter the following details to define document numbering:

 

  1. Letter Prefix Series – Enter the Letter Prefix Series in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Use as Default Prefix Series – Select the check box, if the respective letter prefix series has to be the default value for the covering letters generated. You can specify only one letter prefix series as the default value for the covering letters.  

  3. Long Description – Enter the Long Description in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  4. Short Description - Enter the Short Description in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  5. Year Prefix – Select a value from the list. The values are system defined– “Not Applicable”, “YYYY – Format” and “YY – Format”. The year in the Covering Letter is displayed based on the prefix selected.

  6. Separator Before Year – The separator before Year is set as “/”, by default.

  7. Month Prefix – Select a value from the list. The values are system defined – “Not Applicable”, “Number - Format” and “Character – Format”. The month in the Covering Letter is displayed based on the prefix selected.

  8. Separator Before Month – The separator before Month is set as “/”, by default.

  9. Separator for Serial Number – The separator for serial number is set as “/”, by default.

  10. Starting Number – Enter the Starting Number in numeric format of 10 digits maximum length. The covering letters are numbered based on the format selected. This is a mandatory field.

  11. Next Number – Enter the Next Number in numeric format of 10 digits maximum length. This is a mandatory field.

  12. Last Number – Enter the Last Number in numeric format of 10 digits maximum length. This is a mandatory field.

  13. Recycle Every Year with Starting Number – Select the check box, if the Starting Number has to be recycled every year. If the check box is cleared, the numbering is continued for the successive year.
