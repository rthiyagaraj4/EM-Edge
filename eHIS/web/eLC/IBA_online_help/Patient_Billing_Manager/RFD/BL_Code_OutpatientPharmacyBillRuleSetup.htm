Enter the following to define the bill generation rule for pharmacy items:

 

  1. Facility - Click [List of Values] to select a Facility. The bill format is valid for the pharmacy items utilized from the store in the selected facility.

 

Outpatient Pharmacy Bill Generation/Collection Rule

  1. Billing Group - Click [List of Values] to select a billing group. The values defined using the Billing Group function get listed. This is a mandatory field.

  2. Description - The description for the selected Billing Group gets displayed.

  3. Store - Click [List of Values] to select a store. The values defined using the Store function in the Pharmacy Management module get listed. This is a mandatory field.

  4. Description - The description for the selected store gets displayed.

  5. Generate Bill - Select the check box to generate the bill for the pharmacy items used from the facility.

  6. Settle Bill Amount - Select the check box to generate the bill for pharmacy items when the amount is settled.

  7. Print Option - Click [List of Values] to select a Print Option. The values are system defined and differ when the check box Settle Bill Amount is selected or cleared.

  8. Do not Print – Select Do not Print, if a hard copy of the bill is not required.

  9. Print Visit Bill-Cum-Receipt Automatically - Select Print Visit Bill-Cum-Receipt Automatically to print a receipt and bill automatically when the amount is settled for Pharmacy items.

  10. Print Visit Receipt Automatically - Select Print Visit Receipt Automatically to print a receipt automatically when the amount is settled for Pharmacy items.

  1. Generate Zero Bill - Select the check box to generate a bill with zero amount for the Pharmacy items.

  2. Print Zero Bill - Select the check box to get a printed copy of the bill with zero amount generated for the Pharmacy items.
