Enter the following details to define the legal entities for a practitioner:

 

  1. Practitioner - Click [List of Values] to select the Practitioner ID. The values defined using the 'Practitioner' function in Application Masters module get listed. This is a mandatory field.

 

Applicable Legal Entities

  1. Applicable Legal Entity - Enter the Applicable Legal Entity in alphanumeric format of 30 characters maximum length.

  2. Explanatory Note - Enter the explanatory note in alphanumeric format.

  3. Select - Select the check box to select the record as the active legal entity. On selecting the check box, the fields under 'Active Legal Entity' is enabled. Clear the check box, if the record is not active legal entity.

 

Active Legal Entity

  1. Legal Entity ID - The Applicable Legal Entity gets displayed.

  2. Effective [From - To]  - Click the Calendar function to select the date from when the respective legal entity record is active. You can select only the 'Effective From' date. Based on the start date selected for Legal Entity ID, 'Effective To' date gets displayed. You cannot select the 'Effective To' date. It is set automatically based on the start date selected for the next legal entity record to be active.
