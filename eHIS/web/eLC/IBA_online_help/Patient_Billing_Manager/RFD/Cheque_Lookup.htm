Cheque Lookup

 

You can either click [Enter Query], enter a search criteria and click [Execute Query] or press [F8] to view and to select a record for which the Settlement Type was selected as Cheque or you can click [Execute Query] or press [F8] to view all the records with the Settlement Type as Cheque.

 

The following details get displayed:

 

  1. Doc Date – The date when the document was generated for the service gets displayed.

  2. Doc Desc – The description for recorded during transactions get displayed.

  3. Doc Date – The date from when the document is considered valid gets displayed.

  4. Remarks – The bank or the branch selected for the respective cheque transaction gets displayed.

  5. Doc Type – The document type for the respective transaction gets displayed.

  6. Doc Num – The document reference number generated for the settlement via cheque gets displayed.

  7. Sr No – The Serial number for the selected cheque transaction gets displayed.

  8. Doc Amt – The amount incurred for the service against which the cheque payment was recorded gets displayed.

  9. Patient ID – The ID of the patient who has availed the service and the Settlement Type was recorded as Cheque, gets displayed.

  10. Episode Type – The Episode Type gets displayed for the selected patient.

  11. Episode ID – The Episode ID gets displayed during which the service payment was recorded as cheque.

  12. Visit ID – The Visit ID for the respective patient gets displayed.
