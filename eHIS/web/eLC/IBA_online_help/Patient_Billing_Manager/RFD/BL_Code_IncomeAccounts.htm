Enter the following details to define Income Accounts:

 

  1. Accounting Entity - The Accounting Entity code defined using the ‘Accounting Parameters’ function in the ‘Financial Setup’ menu in the System Manager module gets displayed. The Accounting Entity is set by default and the field is disabled.

  2. Bill Type Code – Click [List of Values] to select a Bill Type. This is a mandatory field.

 

Income Account Details

  1. Selection Criteria – Select a value from the list. The values are system defined – ‘Service Group’, ‘Service Classification’ and ‘Billing Service’.

  2. Service Group – Select Service Group to associate an Income account to the Service Groups.

  3. Billing Service – Select Billing Service to associate an Income account to the Billing Service.

  4. Service Classification – Select Service Classification to associate an Income account to a Billing Service.

  1. Billing Service - Select a valid Billing Service from the list. The values differ based on the Selection Criteria.

  2. Description - The description for the respective Billing Service gets displayed.

  3. GL Interface – Click [List of Values] to select a GL Interface code. The account codes set as income accounts in the General Ledger module gets listed. This is a mandatory field.

  4. GL Interface Description – The Description for the selected GL interface code gets displayed.

  5. Eff. Date From – Click Calendar function to select a date. The selected Income Account is valid for the Bill Type from the date selected. The date selected has to be greater than the system date. This is a mandatory field.

  6. Eff. Date To - Click Calendar function to select a date. The selected Income Account is valid for the Bill Type until the date selected.

 

Additional Charge Interface

You can associate a Billing Service and Income Account code to a Rule Code. The selected Rule Code is considered towards income generation for the respective facility.

 

Enter the following details to define additional charge interface:

  1. Rule Code – Click [List of Values] to select a Rule Code. The Rule Codes defined using the function ‘Rule for Additional Service Charge’ get listed.

  2. Selection Criteria – Select a value from the list. The system defined values are ‘Service Group’, ‘Service Classification’ and ‘Billing Service’.

  3. Service Group – Select Service Group to associate an Income account to the Service Groups for additional charge interface.

  4. Billing Service – Select Billing Service to associate an Income account to the Billing Service for additional charge interface.

  5. Service Classification – Select Service Classification to associate an Income account to a Billing Service for additional charge interface.

  1. Billing Service - Select a valid Billing Service from the list. The values differ based on the Selection Criteria.

  2. Description - The description for the respective Billing Service gets displayed.

  3. Income Acc Int Code – Click [List of Values] to select an Income Account code. The account codes set as income accounts in the General Ledger module get listed.

  4. Description – The Description for the selected Income Account Code gets displayed.

  5. Eff. Date From – Click Calendar function to select a date. The selected Income Account is associated to the Rule Code from the date selected. The date selected has to be greater than the system date. This is a mandatory field.

  6. Eff. Date To - Click Calendar function to select a date. The selected Income Account is associated to the Rule Code until the date selected.
