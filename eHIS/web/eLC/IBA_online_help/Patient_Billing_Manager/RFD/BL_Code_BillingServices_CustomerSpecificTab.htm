Customer Specific Price Tab

The Customer Specific tab enables you to set up prices for various Customers in the facility.

 

Enter the following details to define the price:

 

  1. Facility – Select a Facility from the list. The facilities associated to the user get listed. The prices defined for the patient category is applicable only in the facility selected. This is a mandatory field.

  2. Customer – Click [List of Values] to associate a Customer to the facility. The values are listed only if the ‘Customer’ is defined initially under the ‘Accounting Entity Setup’ menu in the Masters. This is a mandatory field.

  3. Description – The Description for the selected Customer is displayed.

  4. Valid From - Click Calendar function to select a date. The Customer Specific Prices are applicable for the Billing Class in the Facility from the date selected. This is a mandatory field.

  5. Valid To - Click Calendar function to select the date upto when the discount percentage defined for the patient category is valid.

  6. Blng Class – Click [List of Values] to select a valid Billing Class. The Billing Classes defined under the ‘Billing Rule’ menu in the Masters gets listed. The service price for the patient category can be defined only when a valid Billing Class is selected. This is a mandatory field.

  7. Description – The Description for the selected Billing Class is displayed.

  8. R/F – The values are system defined. Select 'R' for rate based and 'F' for factor based service.

  9. IP Rate – Enter the base price to be charged for the Inpatients. The Inpatients associated to the relevant Customer and admitted in the respective Facility are charged based on the quantity of the service used.

  10. DC Rate – Enter the base price to be charged for the Daycare patients. The Daycare patients associated with the relevant Customer and the respective Facility will be charged an amount based on the quantity of the service used.

  11. OP Rate - Enter the base price to be charged for the Outpatients. The charge towards the service is calculated accordingly based on the quantity of the service used.

  12. Emg. Rate - Enter the base price to be charged for the Emergency cases. The charge towards the service is calculated accordingly based on the quantity used.

  13. Ext. Rate - Enter the base price to be charged for the External or the Referral patients. The charge towards the service is calculated accordingly based on the quantity used.

  14. Min. Charge - Enter the minimum charge towards the service. The charge towards the service is calculated accordingly based on the quantity used. This is a mandatory field.

  15. Max Charge – Enter the maximum charge towards the service. This is applicable only when the Cost Indicator is set. This identifies the maximum amount charged towards the service when the Cost Indicator is set. You cannot charge a service price beyond the limit specified.

  16. Type – Select a value from the list to identify the reimbursable type. The system-defined values are – ‘Full’, ‘Partial’ and ‘NA’.

  17. Full – Select Full to refund the entire amount paid by the patient towards the service.

  18. Partial – Select Partial to refund only a fraction of the amount paid towards the service. A part of the amount is deducted for the service availed and the remaining amount is refunded to the patient/payer. You have to specify the amount to be reimbursed in the field ‘Reimbursable Amount’ under the ‘General’ tab. The Charge is specified only if the ‘Charge Type’ is set to ‘Charge’.

  19. Not Applicable – Select NA, if the service is not reimbursable.

  1. Allowed – Select the check box, if the Billing Service is reimbursable for the selected patient class. Clear the check box, if the respective Billing Class is non-reimbursable for the selected patient class.

  2. Reimbursable Amt – Enter the amount, which can be reimbursed to the patient.

  3. Override – Select the check box, if the respective Billing Service is reimbursable for the selected patient class. The override option for the patient class takes precedence over the override check box in the header field. If the service is not reimbursable and the check box is selected for the patient class, it indicates that the Billing Service is reimbursable for the selected patient class.

  4. Base Qty – Enter the Base Quantity towards the service. The charge is calculated based on the quantity used. This is a mandatory field.

  5. Gross/Net Applicability – Select either ‘Gross’ or ‘Net’ from the list.

  6. Gross – Select Gross to charge the Gross amount for the service. The charge towards the service is calculated based on the discount and the price defined for the patient category associated to the Billing Class.

  7. Net – Select Net to charge the Net amount for the service. The service charge is calculated based on the price towards the patient category associated to the Billing Class. The discount amount for the service is not considered during bill generation.

  1. Check for Cost Availability – Select the check box, if the Cost Amount is applicable. Clear the check box, if the Cost Amount is not applicable towards the service.

  2. Apply Discnt – Select the check box, if a discount is applicable to the selected Billing Class. If selected, the Discounts tab is enabled. Enter the percentage of discounts for a patient category associated to the Billing Class.

 

Discounts

The Discounts tab enables you to define the percentage of discounts applicable to the patient category associated to the Billing class and the Billing Group.

 

Enter the following details to define discounts:

  1. Customer - The Customer selected under the Prices tab gets displayed, by default. The description for the Customer gets displayed in the adjacent text box.

  2. Billing Class - The Billing Class associated to the Customer under the Prices tab is displayed, by default. The description for the selected Billing Class gets displayed in the adjacent text box.

  3. Valid From – The From date selected to indicate from when the Customer Specific Prices are applicable for a Customer gets displayed.

  4. Valid To – The To date upto when the prices are considered valid gets displayed.

  5. Discount Indicator – Select a value from the list to issue discount for the Billing Service. The system-defined values are ‘Amount’ and ‘%’. Select Amount to specify the amount that has to be issued as a discount amount on the Billing Service. Select Percentage to specify the percentage of amount that has to be issued as a discount amount on the Billing Service.

  6. Valid From – Click Calendar function to select a date. The Customer Specific Prices are applicable for the Billing Class in the Facility from the date selected. The date set has to be greater than the system date. This is a mandatory field.

  7. Valid To - Click Calendar function to select the date upto when the discount percentage defined for the patient category is valid.

  8. DC % - Enter the value to issue discount towards the service for Daycare patients. This is applicable only to the Daycare patients associated to a relevant Customer and a Billing Class for the specified period. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  9. IP % - Enter the value to issue discount towards the service for Inpatients. This is applicable only to the Inpatients associated to a relevant Customer and a Billing Class for the specified period. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  10. OP % - Enter the value to issue discount towards the service for Outpatients. This is applicable only to the Outpatients associated to a relevant Customer and Billing Class for the specified period. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  11. EXT % - Enter the value to issue discount towards the service for External or Referral patients. This is applicable only to the External or the Referral patients associated to a relevant Customer and a Billing Class for the specified period. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.

  12. EMG % - Enter the value to issue discount towards the service for Emergency patients. This is applicable only to the Emergency patients associated to a relevant Customer and a Billing Class for the specified period. The value is computed as amount or discount based on the value selected for 'Discount Indicator'.
