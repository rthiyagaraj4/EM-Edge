Enter the following details to query prepaid letter details:

 

  1. Payer Group - Click [List of Values] to select a Payer Group. The description of the selected Payer Group is displayed. This is a mandatory field.

  2. Payer - Click [List of Values] to select a Payer. The description of the selected Payer is displayed. This is a mandatory field.

  3. Prepaid - Click [List of Values] to select a Prepaid Code. The description of the selected Prepaid code is displayed. The values that are defined through the 'Prepaid Payment Setup Rule' function are listed.

  4. Protocol - Click [List of Values] to select a Protocol. The patient records with the protocol specified are listed.

  5. Register Date [From - To] - Click the Calendar function to select the register date range. The patient records with the register date matching the search criteria are listed.

  6. Due Date [From - To] - Click the Calendar function to select the due date range. The patient records with the due date matching the search criteria are listed.

  7. Payment No [From - To] - Click [List of Values] to select a Payment Number. The patient records with the payment number matching the search criteria are listed.

  8. Generate Doc Date - Click the Calendar function to select a date range. The documents that are generated during the selected date range are displayed.

  9. Doc Type - Click [List of Values] to select a Document Type. The document types that are generated during prepaid letter generation are displayed.

  10. Patient ID - Click [List of Values] to select a Patient ID. The patient records matching the search criteria are listed.

  11. Register No. - Click [List of Values] to select a register number.
