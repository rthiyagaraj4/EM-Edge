Enter the following details to define the billing details for an external source:

 

  1. Source Code – Click [List of Values] to select a Referral Source. The values defined using the function ‘External/Referral Source’ in the Application Masters module and associated to the logged in facility get listed. This is a mandatory field.

 

Financial/Linked Patient Info

  1. Charge Posting Mode – Select a value from the list. The values are system defined - 'Via Patient Billing Manager module', 'Via Accounts Receivable Module' and 'Not Chargeable'. This is a mandatory field.

  2. Via Patient Billing Module – Select Via Patient Billing Manager module, if the Referral Source has to be charged in the Patient Billing Manager module.

  3. Via Accounts Receivable Module - Select Via Accounts Receivable Module, if the charges for the Referral Source have to be posted to Accounts Receivable module.

  4. Not Chargeable – Select Not Chargeable, if the charges are not applicable for the selected Referral Source.

  1. Country - The name of the country name associated to the created referral source code using the function ‘External/Referral Source’ in the Application Masters module gets displayed.

  2. Doctor – Click [List of Values] to select a practitioner. The values defined using the function Practitioner in the Application Masters module get listed.

  3. Billing Class - The Billing Class is set to external, by default.

  4. Billing Group – Click [List of Values] to select a Billing Group. The values defined using the Billing Group function get displayed. The description for the selected Billing Group get displayed.

  5. Settlement Type - The Settlement Type associated to the respective Billing Group gets displayed. The system defined values are - 'Credit', 'Cash' and 'Sponsored'.

  6. Customer - Click [List of Values] to select a Referral Source. The description for the selected Customer gets displayed. This is mandatory, if the Billing Group is a non-cash Billing Group.

  7. Credit Doc Ref - Enter the credit document reference for the selected Referral Source. This field is mandatory, if the Billing Group is a non-cash Billing Group.

  8. Start Date – Click Calendar function to select the date. The billing details defined are valid for the selected Referral Source from the date selected. This is a mandatory field when the selected Settlement Type is a non-Cash billing group.

  9. End Date - Click Calendar function to select the date. The billing details defined are valid for the selected Referral Source until the date selected. This is a mandatory field when the selected settlement type is a non-Cash billing group.

  10. Remarks - Enter the remarks in alphanumeric format with a maximum of 40 characters.

  11. Bill Type – Click [List of Values] to select a Bill Type. The values defined using the Bill Types function get listed.

  12. Patient Series – Click [List of Values] to select a patient series. The values defined in the Patient Identity Management module get listed. The description for the selected patient series gets displayed.

  13. Name - Enter the name of the patient to be associated with the selected referral source type.

 

Mailing Address/Contact

  1. Address - The address of the Referral Source gets displayed.

  2. Telephone - The Telephone Number of the Referral Source gets displayed.

  3. E-mail - The e-mail address of the Referral Source gets displayed.

 

Contacts

  1. Name - The name of the contact person gets displayed.

  2. Job Title - The job title of the contact person gets displayed.
