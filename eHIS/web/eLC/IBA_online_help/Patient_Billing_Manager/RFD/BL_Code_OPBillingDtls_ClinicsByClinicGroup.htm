Clinics by Clinic Group

The Clinics by Clinic Group tab enables you to associate individual clinics to the Clinic Group. The billing details defined for the Clinic Group is applicable to the associated clinics in the facility. When the income or the discount report has to be generated for a Clinic Group, the individual clinic details are considered towards evaluation.

 

Enter the following details to associate Clinics to a Clinic Group:

 

  1. Clinic Group – The Clinic Group selected in the Clinic Groups Billing Details tab gets displayed. To select a relevant Clinic Group, click [Enter Query] and press [F9] to select a Clinic Group from [List of Values]. To view all the Clinic Groups and to associate Clinics to a Clinic Group click [Execute Query] or press [F8]. Press [F8] to select a Clinic Code.

  2. Clinic Code – Click [List of Values] to select a Clinic Code. A Clinic associated to a Clinic Group cannot be associated to another Clinic Group. This is a mandatory field.

  3. Description – The Description for the selected Clinic Code gets displayed.
