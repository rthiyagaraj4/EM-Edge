The following details are displayed:

 

  1. Customer No – Click [List of value] to select the customer code. The customer code is defined in the Customer function in the Setup.

  2. Facility ID – Click [List of Values] to select the Facility ID from the list. The facilities associated to the logged-in user are displayed.

  3. Patient No - Select a Patient Number for transaction or use the Lookup function to select a patient. You can search for a relevant patient ID in the lookup screen. Click [Enter Query], enter the search criteria and click [Execute Query] or press [F8] to select a patient ID. The patient details get populated in the respective fields.

    Enter the details in the following fields to search for a record matching the criteria:

    General sub-tab

    1. Patient ID – Enter the Patient ID in alphanumeric format of 10 characters maximum length. The records matching the patient ID entered get displayed.

    2. NRIC (National Registration Identity Card) – Enter the National ID in numeric format of 15 digits maximum length. The records matching the National ID of the patient recorded during registration get displayed.

    3. Passport Details – Enter the passport details.

    4. Title - Enter the Title in alphanumeric format of 15 digits maximum length. The records matching the Title of the patient recorded during registration get displayed.

    5. Last Name – Enter the last name of the patient to search for a specific patient record. The records matching the patient’s family name get displayed.

    6. First Name – Enter the first name of the patient to search for a specific patient record. The records matching the patient’s first name get displayed.

    7. Contact No - Enter the contact number of the patient to search for a specific patient record.

    8. Contact Mode – Select the Contact Mode from the drop-down list. The listed values are E-Mail, Fax, HP, Mobile No, Other, Police, Postal, Primary Resi No, Tel (Home) and Tel (Office).

    9. Nationality – Click [List of Values] to select the nationality.

    10. Birth Date – Select the Birth Date using the calendar function.

    11. Place of Birth – Click [List of Values] to select place of birth.

    12. Gender – Select Gender from the drop-down.

    13. Race - Click [List of Values] to select Race.

    14. Order By – Select Order By from the drop-down list. The listed values are Patient ID, Patent Name, Sex, Date of Birth and NRIC. The patient details are ordered based on the selected value.

    15. Click [Search] to view the patient details or click [Cancel] to close the screen.

    Additional sub-tab

    1. Option – The value NORMAL is selected by default. You cannot modify the data.

    2. Deceased Only – Select this check box to query only the deceased patient details.

    3. Suspended Only – Select this check box to query only the suspended patient details.

    4. Inactive Only - Select this check box to query only the inactive patient details.

    Patient Details grid

    1. Patient ID - The patient ID is displayed.

    2. National ID - The national ID of the patient is displayed.

    3. Citizen ID - The citizen ID of the patient is displayed.

    4. Last Name (Loc) - The last name of the patient is displayed.

    5. First Name (Loc) - The first name of the patient is displayed.

    6. Second Name (Loc) - The second name of the patient is displayed.

    7. Third Name (Loc) - The third name of the patient is displayed.

    8. Title (Loc) - The title of the patient family is displayed.

    9. Family Name – The name of the patient family is displayed.

    10. First Name - The first name of the patient family is displayed.

    11. Second Name – The second name of the patient family is displayed.

    12. Third Name – The third name of the patient family is displayed.

    13. Prefix – The prefix number is displayed.

    14. Government No – The Government number is displayed

    15. Expatriate No – The expatriate number is displayed.

    16. Pat Series – The patient series is displayed.

    17. Sex – The patient gender is displayed.

    18. Res Tel Num –The residence telephone number is displayed.

    19. Inpatient – Select this check box if the patient is an inpatient.

    20. Deceased – Select this check box if patient is deceased.

    The following color codes are used to identify the patient status:

      - Deceased patients.

      - Inactive patients.

     - Suspended patients.
