Enter the following details:

 

  1. Episode – Select an Episode from the list. The system-defined values are Daycare, Inpatient, Emergency, Outpatient, External and All. Enter the respective Episode ID on selecting the Episode.

  2. National ID – The National ID of the patient recorded during patient registration is displayed.

  3. Encounter Date [From – To] – Click the Calendar function to select the encounter date. The patient records with outstanding bills are displayed based on the date range selected. This is a mandatory field.

  4. Outstanding Amt Only – Select the check box to access patient records with outstanding amount. Clear the check box to list all the deferred patient records.

  5. Today’s Encounters Only – Select the check box to query for current day’s encounters. Clear the check box to query the encounters irrespective of date.

  6. My Assignments – Select the check box to view the cash patient records deferred by the logged-in user. Clear the check box to view the cash patient records deferred by any user.
