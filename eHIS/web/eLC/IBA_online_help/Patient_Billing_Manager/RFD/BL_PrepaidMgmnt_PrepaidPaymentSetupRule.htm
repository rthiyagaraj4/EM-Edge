Enter the following details to define prepaid codes:

 

  1. Prepaid Code - Enter the prepaid code in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Long Description - Enter the long description for a Prepaid code in alphanumeric format of 100 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description for a Prepaid code in alphanumeric format of 45 characters maximum length. This is a mandatory field.

  4. Lumpsum Applicable - Select the check box if lumpsum amount can be collected for the respective prepaid code from the Payer. Clear the check box if lumpsum amount is not valid for respective prepaid code.

  5. Prepaid Letter - Select the check box if prepaid letter is valid for respective prepaid code. You can send reminder letters to the Payer only if the check box is selected. Clear the check box if prepaid letter is not valid for respective prepaid code.

Signature

  1. Authorized Signature 1 - Enter the name of authorized signatory in alphanumeric format. The name of the authorized signatory is printed on the reminder letter.

  2. Authorized Signature 1 Desg - Enter the designation of authorized signatory in alphanumeric format. The designation of the authorized signatory is printed on the reminder letter.

  3. Authorized Signature 2 - Enter the name of authorized signatory in alphanumeric format. The name of the authorized signatory is printed on the reminder letter.

  4. Authorized Signature 2 Desg - Enter the designation of authorized signatory in alphanumeric format. The designation of the authorized signatory is printed on the reminder letter.

Remarks

  1. [Line 1 - Line 6] - Enter the remarks in alphanumeric format.

  2. Doc Type Code for Prepaid Letter - Click [List of Values] to select a document type for prepaid letter. The description of the selected document type is displayed.

  3. Print Receipt Option - Select a value from the list. The system-defined values are 'Print Receipt Automatically' and 'Marked Only As Printed'. Select 'Print Receipt Automatically' to print receipts automatically for the respective prepaid code. Select 'Marked Only As Printed' to

  4. Print Receipt Legends - Select the check box to print receipt legends. Clear the check box if receipt printing is not required.

Click [Commit Form] or press [F10] to save the records or [Clear Record] to delete the data.
