Multipayer Row/Column Format

The Multipayer Bill Format tab enables you to define the bill format for multiple payers and patients in a facility. When the Payer Specific Format or a Single Bill Cash Format is not defined, Multipayer Bill Format is taken as the default format for bill print in the facility.  

 

Multipayer Bill Format For Payers

 

  1. Print Service Line As – Select a value from the list. The values are system defined – 'Gross Amt' and 'Net Amt'.

  2. Gross Amt – Select Gross Amt to print the complete bill details along with the discounts.

  3. Net Amt - Select Net Amt to print only the net charges on the bill.

  1. Print Service Line Discount As – Select a value from the list. The values are system defined – 'Row', 'Column' and 'Not Applicable'.

  2. Row – Select Row to print the discount details in a row.

  3. Column – Select Column to print the discount details in a column.

  4. Not Applicable - Select Not Applicable to exclude the discount details.

  1. Print Bill Adjustment As – Select a value from the list. The values are system defined – 'Summarized Row', 'Separate Row' and 'Donot Print'. The Bill Adjustment details are printed based on the format selected.

  2. Summarized Row – Select Summarized Row to print a summary of the Bill Adjustment details in a row.

  3. Separate Row - Select Separate Row to print the Bill Adjustment details in a separate row.

  4. Donot Print – Select Donot Print, if the Bill Adjustment details are not to be printed on the bill.

  1. Print Zero Discount Columns - Select the check box, if the bill printed for payers has to include discount columns, when the patient has not availed any discount from the facility. On selecting the check box, the discount value is printed as zero in the payer bills. The check box is enabled for selection, if the value for 'Total Payable Service Line Discount As' is selected as 'Column'. Clear the check box, if discount columns are not to be printed when there are no discounts.

  2. Print Zero Additional Charge Columns - Select the check box to print zero adjustment details in payer bills, if there are no additional charges associated to the patient's bill. Clear the check box if zero adjustments are not to be printed in payer bills.

  3. Print Service Date in Detail Format - Select the check box to print service date in a detailed format in the bill generated for payers. Clear the check box, if service date is not required.

  4. Print Service Date with Time - Select the check box to print service date and time in the bill generated for payers. You can select the check box, if 'Print Service Date in Detail Format' check box is selected. Clear the check box, if service date and time is not required.

  5. Print Rate in Detail Format - Select the check box to print service rates in the payer bills irrespective of the service coverage. You can select the check box, if 'Print Service Date in Detail Format' check box is selected. Clear the check box, if service rate is not required in the bills printed for payers.

  6. Print Bill Discount as Separate Row – Select ‘Print Bill Discount as Separate Row’ to print the bill discounts in a separate row. The check box is selected automatically when the ‘Print Bill Adjustment As’ is set as Separate Row. The check box is enabled only when ‘Print Bill Adjustment As’ is selected as ‘Summarized Row’ or ‘Separate Row’. The check box is disabled, if the ‘Print Bill Adjustment As’ is selected as ‘Do Not Print’.

  7. Print Bill Adjustment as Separate Row - Select ‘Print Bill Adjustment as Separate Row’ to print the bill adjustment details in a separate row. The check box is selected automatically when the ‘Print Bill Adjustment As’ is set as Separate Row. The check box is enabled only when ‘Print Bill Adjustment As’ is selected as ‘Summarized Row’ or ‘Separate Row’. The check box is disabled, if the ‘Print Bill Adjustment As’ is selected as ‘Do Not Print’.

  8. Print Bill Exemption as Separate Row - Select ‘Print Bill Exemption as Separate Row’ to print the bill exemption details in a separate row. The check box is selected automatically when the ‘Print Bill Adjustment As’ is set as Separate Row. The check box is enabled only when ‘Print Bill Adjustment As’ is selected as ‘Summarized Row’ or ‘Separate Row’. The check box is disabled, if the ‘Print Bill Adjustment As’ is selected as ‘Do Not Print’.

  9. Print Bill Write-Off as Separate Row - Select ‘Print Bill Write-Off as Separate Row’ to print the bill write-off details in a separate row. The check box is selected automatically when the ‘Print Bill Adjustment As’ is set as Separate Row. The check box is enabled only when ‘Print Bill Adjustment As’ is selected as ‘Summarized Row’ or ‘Separate Row’. The check box is disabled, if the ‘Print Bill Adjustment As’ is selected as ‘Do Not Print’.

  10. Print Zero Adjustments - Select the check box, if zero adjustments are to be printed in patient's bill. The legend with the value as '0' gets printed. Clear the check box, if zero adjustments are not to be printed in patient's bill, if there are no adjustments made with respect to the patient's bill.

  11. Print Other Payer Amount as – Select a bill format from the list. The values are system defined – “Summarized Row”, “Separate Row” and “Donot Print”.

  12. Summarized Row – Select Summarized Row to print a summary of the payer details in a row.

  13. Separate Row - Select Separate Row to print a summary of the payer details in a separate row.

  14. Donot Print – Select Donot Print, if the details are not to be printed.

  1. Print Patient Amount as - Select a bill format from the list. The values are system defined – 'Separate Row' and 'Donot Print'.

  2. Separate Row – Select Separate Row to print the patient payable amount in a separate row.

  3. Donot Print – Select Donot Print, if the patient payable amount is not to be printed on the bill.

  1. Print Net Amount as – Select a value from the list. The values are system defined - 'Row', 'Column' and 'Both'.

  2. Row – Select Row to print the net amount towards the service in a row.

  3. Column – Select Column to print the net amount towards the service in a column.

  4. Both - Select Both to print the net amount details in a row and column.

Discount Document Setup

Multipayer Bill Format for Payers

  1. Print Discount Amount in Detail Format - Select the check box to print a detailed discount document report for multiple payers associated to a patient. Clear the check box to print a summarized discount document report for multiple payers.

 

Multipayer Bill Format for Patient

  1. Print Discount Amount in Detail Format - Select the check box to print a detailed discount document report for patients. Clear the check box to print a summarized discount document report for patients.
