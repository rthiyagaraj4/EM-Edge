Related Field Descriptions:

 

  1. Invoice Document Type – Click [?] to select Invoice Document Type. This is mandatory field. The Invoice Document Types defined in the Document Type are listed.

  2. Credit Document Type – Click [?] to select Credit Document Type. This is mandatory field. The Invoice Document Types defined in the Document Type are listed.

  3. Patient ID – Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter. This is mandatory field.

  4. Episode ID / Visit ID – Based on the Patient ID, the Episode ID and Visit ID are displayed after entering Patient ID.

  5. Episode Type – Based on the Episode ID, the Episode Type is displayed.

  6. Doc Type/Number – Select a Document Type and Number from [List of Values]. This is a mandatory field.

  7. Doc Date – based on selected Doc Type, the doc date is displayed.

  8. Outstanding Amount – based on selected Doc Type, the outstanding amount is displayed.

  9. Reason Text – Enter the reason in alphanumeric format of maximum length of 50 character. This is mandatory field

  10. Cus Code – Enter the customer code based in which the customer code description is displayed. This is mandatory field.
