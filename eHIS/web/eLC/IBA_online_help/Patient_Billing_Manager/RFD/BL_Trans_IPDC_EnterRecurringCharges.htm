Enter the following to create recurring charges:

 

  1. Panel/Serv – The value gets displayed as ‘S’, by default. ‘S’ indicates ‘Service’. Enter ‘L’ to associate Panels.

  2. Panel/Service – Click [List of Values] to select a Panel or a Service. The description for the selected Billing Service or Panel gets displayed. This is a mandatory field.

  3. Rate/Charge –The value is displayed as ‘R’ or ‘C’ based on the service selected. The value is displayed as ‘C’, if the service is charge based. The value is displayed as ‘R’, if the value is Rate based.

  4. Quantity – Enter the quantity of service in numeric format. This indicates the number of times the service has to be served to the patient.

  5. Charge Amount – The charge amount gets displayed automatically based on the price recorded during service definition. You can update the charge, if the selected service is of charge type.

  6. Start Date – Click the Calendar function to select the date. The service is served to the patient from the date selected. The date selected has to be greater than or equal to the transaction reference date and date until which the recurring charges has been computed.

An error message 'Recurring Charges have been computed till dd/mm/yyyy' is displayed, if the 'Start Date' is lesser than the date until which the recurring charges are computed.

  1. End Date – Click the Calendar function to select the end date. The date selected has to be greater than the current date.
