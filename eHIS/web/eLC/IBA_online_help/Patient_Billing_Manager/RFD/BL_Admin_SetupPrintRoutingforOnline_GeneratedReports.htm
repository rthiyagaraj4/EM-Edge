Enter the following details to set up print routing for online generated reports:

 

Online Reports

  1. Print Name – Click [List of Values] to select the Report. The online reports in the Patient Billing Manager module get listed.

  2. Report Group – A report is categorized into one of the valid report groups in the System Manager module, if the report is not a facility-based report. You cannot select the Report Group for the Facility based reports. If a report is not a facility based one, the corresponding report group can be set up in Facility Based Report Parameters function in the Enterprise Configuration module. This is a mandatory field.

  3. Module ID – The Module ID gets displayed as BL, by default.

  4. Facility Based? – The check box is selected, if the Report is defined as Facility based in the System Manager module. The check box is cleared, if the Report is non-facility based. You cannot select or clear the check box.

  5. Transaction Based? - The check box is selected, if the Report is defined as Transaction based in the System Manager module. The check box is cleared, if the Report is not transaction based. You cannot select or clear the check box.

  6. [Show All Facilities] - Click [Show All Facilities] to list the print routes defined for the report associated to the Report Group in the selected facilities.

  7. [Show Current Facility] - Click [Show All Facility] to list the print route defined for the report associated to the Report Group. The print route defined in the logged-in facility gets listed.

 

List of Destination Printers (Current Facility)

  1. Facility ID – Select the Facility from the list. The facilities associated to the logged in User get listed.

  2. Dest Type - Select the Destination Type from the list. Based on the report for which the print routing rule is being defined, the first part of the destination is shown as ‘Location’ only (if it not transaction based), ‘Location’/‘Clinic’ (if the report is transaction based and the transaction is performed in an outpatient clinic), or ‘Location’/’Nursing Unit’ (if the report is transaction based and the transaction is performed in an Inpatient Nursing Unit).

  3. When the first part of the destination is selected as ‘Location’, enter the location under the ‘Dest Code’ field. Enter the description for the selected Destination.

  4. When the first part of the destination is selected as ‘Clinic’, the valid list of clinics (limited to the selected facility, for facility-based reports) gets listed. Select the Clinic and the description for the selected clinic gets displayed.

  5. When the first part of the destination is selected as ‘Ward’, the valid list of Nursing Units (limited to the selected facility, for facility based reports) gets listed. Select a Nursing Unit and the description for the selected Nursing Unit gets displayed.

  6. When the first part of the destination is selected as ‘Department’, the valid list of Accounting Departments (limited to the selected facility, for facility based reports) gets listed. Select the Department and the description for the selected Department gets displayed.

  1. Dest Code – The Codes are listed based on the ‘Dest Type’ selected.

  2. Description – The description for the selected Destination Code gets displayed.

  3. WS No - Enter the Work Station Number in alphanumeric format.

  4. Copies – Enter the number of copies in numeric format of two digits maximum length. The reports are routed to the associated printer and the copies of the report printed are based on the number specified. The field is defaulted to one on selecting the report. You can update the number of copies. This is a mandatory field.

  5. Printer 1, 2, 3 - Click [List of Values] to select the printer to which the report has to be routed. 1st Printer is mandatory. The number of copies of the report specified is printed in the selected printer.

  6. Start Time – Enter the time, which can be scheduled for the respective printers to carry on with the print routing.
