Exemption Types By Class

You can associate multiple Exemption Types to an Exemption Class.  An Exemption Type defines the reason mainly identifies the modes of Exemptions based on which the patient is counselled and exempted. An Exemption Class is further categorized into Exemption Types. The patient is exempted based on the Exemption Class and Exemption Type associated to the patient while counselling.

 

Enter the following details to define an Exemption Type:

 

  1. Exemption Class – The Exemption Class selected under the Exemption Class tab gets displayed.

  2. Exemption Type – Click [List of Values] to select the Exemption Type. The reasons defined in the ‘Reasons for Billing Actions’ function with the ‘Bill Exemption’ check box as selected get listed.

  3. Description – The description for the respective Exemption Type get displayed.
