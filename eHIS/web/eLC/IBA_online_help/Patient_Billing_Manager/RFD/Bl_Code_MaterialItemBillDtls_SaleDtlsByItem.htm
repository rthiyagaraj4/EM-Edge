Sale Details by Item

Enter the following details to define the sale details by item:

 

The Sale Details by Item tab enables you to define the prices for a material item based on the Sale Category.

  1. Facility – Select a Facility from the list. The facilities associated to the logged in user get listed. The prices defined for a Material Item is valid only within the Facility selected. This is a mandatory field.

  2. Sale Category – Click [List of Values] to select a Sale Category. The values defined using the function ‘Material Item Charging Rule Setup’ get listed. You can update the Sale Category that is associated to the Facility, if required. This is a mandatory field.

  3. Imported Item – The check box is selected, if the selected Item is marked as an imported item while defining the Item in Materials Base module.

  4. Essential Item – The check box is selected, if the selected Item is marked as an essential item while defining the Item in Materials Base module.

  5. Public Price – Enter the Public Price for the sale category in numeric format with a maximum of 10 digits. Public Price indicates the price regulated by the Government for a material item. The charges for the selected material item cannot be higher than the price regulated by the Government.

  6. Cost Ind – Select a Cost Indicator from the list. The values are system defined - ‘%age’ and ‘Amount’.

  7. %age – Select %age, if the Cost Amount is applicable and to calculate the material item charge in terms of percentage of the service price and the associated sale category.

  8. Amount - Select Amount to charge the amount defined for the Billing Service associated to the sale category.

  9. N/A – Select N/A, if the Cost Amount is not applicable for the selected material item and the associated sale category.

  1. Cost Amt – Enter the Cost Amount in numeric format. The Cost Amount is valid when the Cost Indicator is selected as percentage. The service price is calculated as a percentage of the billing service price associated to the sale category.

Inpatient

  1. Inpatient - Enter the base price to be charged for Inpatients for the selected material item and the associated sale category.

  2. Daycare - Enter the base price to be charged for Daycare patients for the selected material item and the associated sale category.

  3. Reimbursable – Select the check box, if the item is reimbursable.  Clear the check box, if the item is not reimbursable.

  4. Reimbursable Type – The values listed differ based on the value selected for ‘Reimburse’. The values are system defined – “Full”, “Partial” and “Not Applicable”.

  5. Full – ‘Full’ gets displayed on selecting ‘Reimburse Type’ as ‘Fully Reimbursable’ in Materials Base module. Patient receives a complete refund for the selected material item.

  6. Partial – ‘Partial’ gets displayed on selecting ‘Reimburse Type’ as ‘Partial Reimbursable’ in Materials Base module. Patient receives only a fraction of the service amount as a refund.

  7. Not Applicable – ‘Not Applicable’ gets displayed on selecting ‘Reimburse Type’ as ‘Non Reimbursable’ in Materials Base module. Patient does not receive any refund for the material item used.

  1. Override – Select the check box to override the reimbursable amount during transactions. If the check box is cleared, you cannot override the reimbursable amount during transactions.

  2. Reimbursable Amount – Enter the amount that can be issued as a refund towards the material item selected. The Reimbursable Amount field is enabled when ‘Reimburse Type’ is selected as ‘Fully Reimbursable’ or ‘Partial Reimbursable’ in Materials Base module.

Outpatient

  1. Outpatient - Enter the base price to be charged for Outpatients for the selected material item and the associated sale category.

  2. Emergency - Enter the base price to be charged for Emergency patients for the selected material item and the associated sale category.

  3. External - Enter the base price to be charged for External patients for the selected material item and the associated sale category.

  4. Reimbursable Item – The check box is selected, if a refund can be issued for the item. The check box ‘Reimburse’ has to be selected in the Materials Base module while defining the item. A refund can be issued towards the material item only if the check box is selected.

  5. Reimbursable Type – The values listed differ based on the value selected for ‘Reimburse’. The values are system defined – “Full”, “Partial” and “Not Applicable”.

  6. Full – ‘Full’ gets displayed on selecting ‘Reimburse Type’ as ‘Fully Reimbursable’ in Materials Base module. Patient receives a complete refund for the selected material item.

  7. Partial – ‘Partial’ gets displayed on selecting ‘Reimburse Type’ as ‘Partial Reimbursable’ in Materials Base module. Patient receives only a fraction of the service amount as a refund.

  8. Not Applicable – ‘Not Applicable’ gets displayed on selecting ‘Reimburse Type’ as ‘Non Reimbursable’ in Materials Base module. Patient does not receive any refund for the material item used.

  1. Override – Select the check box to override the reimbursable amount during transactions. If the check box is cleared, you cannot override the reimbursable amount during transactions.

  2. Reimbursable Amount – Enter the amount that can be issued as a refund towards the material item selected. The Reimbursable Amount field is enabled when ‘Reimburse Type’ is selected as ‘Fully Reimbursable’ or ‘Partial Reimbursable’ in Materials Base module.

  3. Effective From – Click Calendar function to select a date. The charges are applicable for the selected material item and the associated sale category from the date selected. The date entered has to be greater than the system date. This is a mandatory field.

  4. Effective To - Click Calendar function to select a date. The charges are applicable for the selected material item and the associated sale category until the date selected. The date entered has to be greater than the selected from date.

  5. Price Id - Enter a Price ID in numeric format of three digits maximum length. The Price ID has to be unique and it helps in identifying the price for the Material Items and the sale category associated to the Facility.
