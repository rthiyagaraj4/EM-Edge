Enter the following details to re-print the adjusted or the exempted documents:

 

  1. Transaction Type – Select a Transaction Type from the drop-down list. The values are system defined – “Exemption” and “Adjustment”.

  2. Exemption – Select Exemption from the list to print all the exempted bills generated during the specified period.

  3. Adjustment – Select Adjustment from the list to print all the bills, where the amount was adjusted during the specified period.

  1. Status – Select the Status from the drop-down list. The system defined values are “Active” and “Cancelled”.

  2. Active – Select Active to print the exempted and the adjusted bills, which are in an active state.

  3. Cancelled – Select Cancelled to print the adjusted and the exempted bills, which are cancelled.

  1. Reason – Click [List of Values] to select a Reason. All the reason codes are considered, if the reason is not entered.

  2. Date [From–To] – Click Calendar function to select a date range. The adjustment or the exemption bills generated during the specified period are printed. This is a mandatory field.

  3. Document No [From–To] – Click [List of Values] to select a range of Document Numbers. The range of adjustment or the exemption bills generated are printed.
