Deposit Applicability by Nursing Unit

Deposit Applicability by Nursing Unit Type tab enables you to describe if the deposit collection is applicable for a Nursing Unit Type. An individual Nursing Unit Type may or may not have the deposit applicability rule.

 

Enter the following details to define the deposit applicability rule for a Nursing Unit:

  1. Nursing Unit Type – Click [List of Values] to select a Nursing Unit Type. The deposit rule is valid for the selected Nursing Unit Type. The values defined using the function Nursing Unit Type function in the Inpatient and Day Care Management module gets listed. This is a mandatory field.

  2. Description - The description for the selected Nursing Unit Type gets displayed.

  3. Deposit Applicable – Select the check box, the deposit rule is applicable for the selected Nursing Unit Type. Clear the check box, if the selected Nursing Unit Type is not applicable for deposit collection.

  4. Addl ICU Deposit Applicable – Select the check box, if additional amount rule defined under the ‘Deposit Amount by Bed Class’ tab is applicable for the selected Nursing Unit Type. Clear the check box, if the rule is not applicable. The Nursing Unit Type cannot collect additional amount towards any services for the patients admitted in the ICU department.

  5. Collect Deposit on Admission – Select the check box, if the deposit amount has to be collected from the patient during admission. The Admission Deposit screen gets displayed in the function 'Admit Patient' in the Inpatient Management module, where you can record the admission amount collected from the patient, if the check box is selected. If the check box is cleared, you cannot collect deposit from the patient during admission to the selected Nursing Unit.
