Enter the following details to associate a Document Type to a Transaction Type:

 

  1. Doc Type – Click [List of Values] to select a Document Type. The values defined using the Document Type function in the Financials Base Manager module get listed. This is a mandatory field.

  2. Description - The description for the selected Document Type appears automatically.

  3. Trx Type - Select a transaction type from the list. The transaction types are system defined - 'Invoice/Bill', 'Adjustments', 'Receipts', 'Payments', 'Cash Control Receipts' and 'Miscellaneous'. This is a mandatory field.

  4. Invoice/Bill – Select Invoice/Bill to define a document type as applicable for invoice/bill generation.

  5. Adjustments – Select Adjustments to select a document type as applicable for bill adjustments.

  6. Receipts - Select Receipts to select a document type as applicable for receipt generation.

  7. Payments - Select Payments to select a document type as applicable for issue of refunds.

  8. Cash Control Receipts - Select Cash Control Receipts to select a document type as applicable for cash control transactions in the facility.

  9. Miscellaneous - Select Miscellaneous to select a document type as applicable for miscellaneous transactions in the facility.

  1. Abbr. - The abbreviation for the selected transaction type gets displayed.

  2. Based on - The document type is based on the 'Accounting Entity' or the 'Facility'. The values selected during the definition of Document Type in the Financials Base Manager module get displayed. You cannot change the value.
