Remarks by Receipt Nature

The Remarks by Receipt Nature tab enables you to define remarks for the selected Receipt Type. The Remarks gets displayed when the respective Receipt Type is used in the transaction functions.

 

  1. Receipt Nature - Click [List of Values] to view and to select a Receipt Nature Code. The values are system defined - 'AR Receipt', 'Against Bill', 'Credit Card Company Payment', 'Credit Card Refund Deduction', 'Customer Receipt', 'DCP Receipt', 'Installment Payment', 'Healthcard Receipt', 'Miscellaneous', 'Patient Deposit', 'Prepayments' and 'Refundable Deposit'. This is a mandatory field.

  2. AR Receipt (AR) – Select AR Receipt, if the amount collected against the receipt is posted to the Accounts Receivable module.

  3. Against Bill (BI) – Select Against Bill, if the patient or a customer clears the amount during bill generation.

  4. Credit Card Company Payment (CR) - Select Credit Card Company Payment, if the Credit Card Company is paying the amount on behalf of the patient.

  5. Credit Card Refund Deduction (CD) - Select Credit Card Refund Deduction, if a part of the amount is deducted during refund process.

  6. Customer Receipt (CU) - Select Customer Receipt, if the Customers are settling a part of the bill amount on the patient’s behalf. For example, an Insurance Company paying on behalf of the patient is considered to be a Customer.

  7. DCP Receipt (DC) - Select Deferred Cash patient, if patient does not settle the bill amount during bill generation.

  8. Installment Payment (IN) – Select Installment Payment, if the Receipt Type is valid for installment payments.

  9. Healthcard Receipt (HC) – Select Healthcard Receipt, if the patient has a Healthcard offered by the Facility. Using a Healthcard, patient is able to avail the services at a discounted price.

  10. Miscellaneous (MI) – Select Miscellaneous to indicate any additional services availed from the Facility.

  11. Patient Deposit (AD) – Select Patient Deposit, if the patient has paid a deposit amount.

  12. Prepayments (PR) – Select Prepayments, if the patient has pre-paid a part of the service charges. The prepayments amount received is adjusted with the bill amount.

  13. Refundable Deposit (RD) - Refundable Deposit refers to the amount to be reimbursed to the patient.

  1. Description – The Description for the selected Receipt Nature gets displayed.

  2. Receipt Type - Click [List of Values] to select a Receipt Type. The Receipt Types defined gets listed. If the Receipt Type can be set to ‘**’, the Receipt Type is unrestricted and the Receipt Nature is applicable to all the Receipt Types. This is a mandatory field.

  3. Description – The Description for the selected Receipt Type gets displayed.

  4. Receipt/Refund – Select either ‘Receipt’ or ‘Refund’ from the list.

  5. Receipt – Select Receipt, if the remarks for the receipt types are applicable for patient charges.

  6. Refund – Select Refund, if the remarks for the receipt types are applicable for any refund transactions.

  1. Type – Select a Type from the list. The values are system defined – 'External', 'Discharge', 'Inhouse', 'Inpatient', 'Outpatient', 'Undercharge', 'Unrestricted' and 'Emergency'.

  2. External – Select External, if the Receipt Nature and Receipt Type are applicable for external or referral patients.

  3. Discharge – Select Discharge, if the Receipt Nature and Receipt Type are applicable during patient discharge.

  4. Inhouse – Select Inhouse, if the Receipt Nature and Receipt Type are applicable for Inhouse transactions.

  5. Inpatient – Select Inpatient, if the Receipt Nature and Receipt Type are applicable for Inpatient transactions.

  6. Outpatient - Select Outpatient, if the Receipt Nature and Receipt Type are applicable for Outpatient transactions.

  7. Undercharge – Select Undercharge, if the Receipt Nature and Receipt Type are applicable for undercharge transactions.

  8. Unrestricted – Select Unrestricted, if the Receipt Nature and Receipt Type are applicable for unrestricted transactions.

  9. Emergency – Select Emergency, if the Receipt Nature and Receipt Type are applicable for Emergency transactions.

  1. Remarks – Enter the Remarks in alphanumeric format for the Receipt Nature. The Remarks are printed on the Receipts when a valid Receipt Type is selected.
