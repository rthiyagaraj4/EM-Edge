Enter the following criteria to print the count sheet:

  1. Owner [From – To] - Enter the range of owner code to which the count sheet should belong in alphanumeric format of six characters maximum length or select the Owner Store using the [?] function. Both the values are optional. If ‘From’ value is not entered, the system assumes the lowest value for ‘From’ and if ‘To’ value is not entered, the system assumes the highest value for ‘To’ value. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  2. Current [From – To] - Enter the range of current code for which the count sheet belongs in alphanumeric format of six characters maximum length or select the current store using the [?] function. Both the values are optional. If ‘From’ value is not entered, the system assumes the lowest value for ‘From’ and if ‘To’ value is not entered, the system assumes the highest value for ‘To’ value. When both values are entered, ‘To’ value must be greater than or equal to the ‘From’ value.

  3. Group [From – To] - Enter the Group range for which the count sheet belongs in alphanumeric format of 10 characters maximum length or select the Group using the [?] function. Both the values are optional. If ‘From’ value is not entered, the system assumes the lowest value for ‘From’ and if ‘To’ value is not entered, the system assumes the highest value for ‘To’ value. When both values are entered, ‘To’ value should be greater than or equal to ‘From’ value.
