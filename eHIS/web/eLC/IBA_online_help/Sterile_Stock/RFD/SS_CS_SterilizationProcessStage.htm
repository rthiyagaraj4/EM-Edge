Enter the following details to create a Sterilization Process Stage record:

  1. Code – Enter the code in alphanumeric format of four characters maximum length. The code defined should be unique for each Sterilization Process Stage record. Once the code is defined and saved, it cannot be modified. This is a mandatory field.

  2. Long Description - Enter the long description of the Sterilization Process Stage in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description of the Sterilization Process Stage in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Enabled – Select the check, to enable the defined Sterilization Process Stage. Clear the check box, to disable the defined Sterilization Process Stage. The check box is selected, by default. Only enabled Sterilization Process Stage codes are available for selection in the applicable functions.
