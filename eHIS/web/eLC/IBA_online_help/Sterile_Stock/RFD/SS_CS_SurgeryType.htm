Enter the following details to define Surgery Type records:

  1. Code – Enter the code in alphanumeric format of eight characters maximum length. The code defined indicates the Surgery Type. Once the code is defined and saved, it cannot be modified. This is a mandatory field.

  2. Long Description - Enter the long description of the Surgery Type in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  3. Short Description - Enter the short description of the Surgery Type in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Surgery Name – You can click [?] to search and select the corresponding OT Surgery Name from the OT Surgery Name - Search and select popup.

    1. Search By  – Select the criteria by which you want to search for. The options available are Surgery Name and Description.

    2. Search Text – Enter the search text. If you had selected the Search By option as “Surgery Name”, enter the surgery name in this text box. Else if you had selected “Description”, type the description here.

    3. Search - Click this button to view the search results.

    4. Reset - Clears the values entered in the fields.

    5. Surgery Name - Displays the name of the surgery.

    6. Long Description - Displays the short description of the surgery name.

    7. Short Description - Displays the short description of the surgery name.

    8. Nature - Displays the nature of the surgery i.e., Elective or Emergency.

You can link the Surgery Types with the Surgery Names from OT module, based on the setup in Parameter For Facility function. The fields are renamed as shown in the below table to differentiate them:

  1. Enabled – Select the check box, to enable the defined Surgery Type. Clear the check box, to disable the defined Surgery Type. The check box is selected, by default. Only enabled Surgery Types are available for selection in the applicable functions.
