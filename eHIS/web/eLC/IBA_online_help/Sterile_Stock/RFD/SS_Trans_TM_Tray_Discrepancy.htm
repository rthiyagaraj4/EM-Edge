Enter the following details to record tray discrepancy:

  1. Store – Select the owner store from the list. The stores assigned to the logged in user is displayed in the list. This is a mandatory field.

  2. Group Type – Select the Group Type from the list. The Group Types displayed are Set, Tray, Single and Soft Pack. The value ‘Set’ is displayed, by default.

  3. Group – Enter the Group Name in alphanumeric format of 15 characters maximum length or select the Group using [?] function. The Groups associated to the selected Group Type are displayed. This is a mandatory field.

  4. Details - on selecting the Group, Details hyperlink is enabled. Click 'Details' hyperlink to view the details of Items associated to the selected Group.

  5. Tray No. – Enter tray number or select the Tray number using [?] function. The sterilized Trays associated to the selected Group are displayed. This is a mandatory field.

  6. On selecting the tray number, the ‘Store’, ‘Group Type’, ‘Group’ and ‘Tray No’ fields are disabled.

 

Enter the following discrepancy details of the selected tray:

  1. Instrument – The non-consumable items of the selected tray are listed in the list. Select the item(s) that differs from the actual tray definition. This is a mandatory field.

  2. Discrepancy Reason - Select the Discrepancy Reason from the list. The reasons displayed are defined in the function ‘Write Off’. This is a mandatory field.

  3. Quantity - Enter the quantity in numeric digit of 12 characters maximum length. On specifying the numeric quantity, select ‘Negative’ or ‘Positive’ from the list. The value Negative indicates the item is less in number from the actual group definition from the specified quantity and Positive indicates the item is more in number from the specified quantity.

  4. Cost - The cost displayed is the product of specified quantity and cost of the item. You cannot edit the displayed value. The cost of the item is defined in ‘Item’ function of Materials Base module.

  5. Remarks – Enter the remarks (if any).
