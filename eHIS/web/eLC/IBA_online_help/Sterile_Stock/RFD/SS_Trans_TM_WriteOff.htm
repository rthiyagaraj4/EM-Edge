Enter the following details for rejecting the trays:

  1. Doc Type – Select the Document Type for the transaction from the list. The values listed are defined in the ‘SS Document Type’ function and the default value displayed is specified in the ‘SS Parameter for Facility’ function for the Write off transaction. This is a mandatory field.

  2. Doc No. – Enter the Document number in numeric format of eight digits maximum length if manual entry is applicable for selected document or system generates the document number on saving the transaction. This is a mandatory field.

  3. Doc Date - The current system date is displayed, by default. You can change the document date. Enter the date in the format of dd/mm/yyyy or select the date using the Calendar function. The entered date must not be less than the Processing month and year set in ‘SS Parameter for Facility’ and not greater than current system date. This is a mandatory field.

  4. Store – Select the store from the list. Only those stores are listed for which the logged in user has access and ‘Write off’ transaction function applicable. This is a mandatory field.

  5. Doc Ref – Enter the document reference in alphanumeric format of 15 characters maximum length.

  6. Entry Completed – Select the check box to indicate the entry is completed. The check box is cleared, by default. If the ‘Entry Completed’ check box is selected, this indicates that the write off document is available for finalization.

  7. Entered By – The logged in user name is displayed and disabled.

  8. Group Type – Select the Group Type from the list. The values displayed are Set, Tray, Soft Pack and Single. The value ‘Set’ is displayed, by default.

  9. Group – Enter the group name in alphanumeric format of 15 characters maximum length or select the group name using [?] function. This is a mandatory field.

  10. Tray No. – Enter the Tray Number in numeric format of 12 digits maximum length or select the Tray Number using [?] function. This is a mandatory field.

  11. Write Off Cost - The write off cost is calculated by the system automatically and displayed. The displayed cost cannot be modified. The cost is calculated based on the items of the tray, their quantities and unit cost.

  12. Write off Reason - Select the reason for rejecting the tray from the list. The reasons are defined in the function ‘Write off Reason’. This is a mandatory field.

  13. Remarks - Enter remarks for rejecting the tray if any in alphanumeric format.
