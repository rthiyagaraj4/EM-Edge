Enter the following details as entry for Autoclave/Washing Unit Test:

  1. Unit Type – Select the required Unit Type from the list. The listed values are ‘Autoclave’ and ‘Washing’. The ‘Autoclave’ is displayed, by default in the list. This is a mandatory filed.

  2. Autoclave Wash Unit – Select the Autoclave/Wash unit from the list. The values are populated in the list based on the ‘Unit Type’ value selected. On selection of ‘Unit Type’ as ‘Autoclave’ all the autoclave units are displayed in the list or on selection of ‘Unit Type’ as ‘Washing’ all the Washing Units are displayed in the list. This is a mandatory field.

  3. Test Date – Enter the date on which the test will be performed in dd/mm/yyyy format or select the date using the Calendar function. This is a mandatory field.

  4. Entered By – The name of the person entering the details for performing the Autoclave/Wash unit test is displayed.

  5. Entry Completed – Select the check box if the required entry for performing the test is completed.

  6. Autoclave/Wash Test – Select the Autoclave/Wash test from the list. All the tests associated to the selected ‘Unit Type’ are displayed in the list. This is a mandatory field.

  7. Batch ID – Enter the Batch ID in numeric format of 12 digits maximum length. The Batch ID should be entered to uniquely identify the test.

  8. Test Result - The Test Result value is displayed as 'Pass' and disabled.

  9. Result Date - The Test Date is displayed and disabled.

  10. Remarks - Enter the Remarks in alphanumeric format.
