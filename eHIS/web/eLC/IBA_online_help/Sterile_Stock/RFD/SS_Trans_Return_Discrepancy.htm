Enter the following discrepancy details of the selected tray:

  1. Instrument – The items of the selected tray is displayed. Select the item that differs from the actual tray definition. This is a mandatory field.

  2. Discrepancy Reason - Select the discrepancy reason from the list. The reasons displayed are defined in the function ‘Write Off’. This is a mandatory field.

  3. Remarks – Enter the remarks in alphanumeric format of 100 characters maximum length.

  4. Discrepancy Quantity - Enter the discrepancy quantity in numeric digit of 15 digits maximum length. This is a mandatory field.

  5. Patient ID – Enter the Patient ID using the instrument in the format defined in Master Patient Index module. The user can invoke [?] function to search for the required Patient ID. This is a mandatory field.

  6. Returnable Instrument - Select the check box if the tray is a returnable instrument.

  7. Encounter ID – Select the required Encounter ID of the selected patient from the list.

  8. Charge Option - Select the charge option from the list. The values listed are None, Charge and Deposit. The value ‘None’ is displayed, by default.

  9. Cost - The cost of the selected item is displayed and disabled when charge option is selected as ‘Charge’.

  10. Receipt Reference - Enter Receipt Reference in alphanumeric format of 20 characters maximum length if the charge option is selected as ‘Deposit’.
