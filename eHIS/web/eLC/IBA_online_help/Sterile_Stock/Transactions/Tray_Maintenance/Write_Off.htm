Write Off

From this function you can Reject the trays requested from the Owner Store and Query for the un-finalized rejected Trays. You can also Delete the un-finalized rejected trays.

From this function the owner store can write off trays by specifying write off reasons. You can only reject the trays which are currently available with owner store.

Rejecting the Trays

From SS Transactions > Tray Maintenance, select the Write Off function.

The Write Off screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Add] to add the tray to the Write Off list or [Cancel] to cancel the process.

  4. Select ‘Del’ check box to exclude the tray from Write Off list.

  5. Click ‘Group type’ hyperlink to modify the Write Off details. You can only modify the Write Off Reason.

  6. On modifying, click [Modify] to save the modified details or [Cancel] to cancel the process.

  7. Click [Apply] to reject the Tray or [Reset] to clear the entered data.

  8. On clicking [Apply], the ‘Confirm Finalize’ dialog is displayed. Click [Yes] to finalize the Write Off details or [No] to finalize the Write Off details in the query mode.

  9. The Write Off document number will be generated based on the document type selected for the Write Off Transaction. The Write Off documents pending finalization are available in the query function.

  10. Click [Query] to invoke the un-finalized Write Off records which are displayed based on the default criteria set in the SS Parameter for Facility function or enter the search criteria and click [Search] to select a specific record.

    Enter the following search criteria to select a specific record:

    1. Doc Type – Select the write off document type from the list. The default ‘Document Type’ set in ‘SS Parameter for Facility’ is displayed, by default. This value can be modified.

    2. Doc No. – Enter the document number in numeric format of eight digits maximum length.

    3. Doc Date - Enter the write off document generation date in format of dd/mm/yyyy or click Calendar function to select the document date.

    4. Store – Select the store from the list. The stores are displayed based on the restrictions set in ‘SS User Access for Store’ and applicability of the store to perform the specific transaction as set in ‘CSSD Store’. The store value set as default store in ‘SS Parameter for Facility’ function is displayed, by default.

    5. Entry – Select the value from the list. The values listed are ‘All’, ‘Completed’ and ‘Not Completed’. The value ‘Completed’ is displayed, by default.

  11. Click ‘Doc Type’ hyperlink to finalize the Write Off documents. You can modify the Write Off details for the documents for which ‘Entry Completed’ check box is not selected.

  12. Click [Apply] to finalize the Write Off document or [Delete] to delete the selected record.

  13. On clicking [Delete], the ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.

You can delete the Write off documents which are not finalized.
