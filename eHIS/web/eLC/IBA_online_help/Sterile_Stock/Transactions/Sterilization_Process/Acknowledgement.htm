Acknowledgement

From this function you can Acknowledge the Tray details of a service location.

You can search for the sterilization request document of a service location based on date and document reference number. All the dispatched tray details are available in the Sterilization Request document. . You can acknowledge the receipt of the tray by entering the acknowledgement date.

Acknowledging Sterilization Request document(s)

From SS Transactions > Sterilization Process, select the Acknowledgement function.

The Acknowledgement screen gets displayed.

  1. Click [Query] to view the sterilization request documents available for acknowledgement based on the default criteria or enter the search criteria and search for specific records. Click Search to view the search results that match the search criteria.

    Enter the following search criteria:

    1. Ster Req Doc Type – Select the required sterilization request document type from the list. The document type as set for ‘Process Ster Req’ in SS Parameter for facility will be displayed, by default.

    2. Ster Req Doc No – Enter the sterilization request document number in numeric format of eight digits maximum length.

    3. Doc Date – Enter the document date in dd/mm/yyyy format.

    4. Service Location – Select the required Service Location from the list. The Service Location as set in ‘SS Parameter for facility’ will be displayed, by default.

    5. Store – Select the required store from the list. The store as set in ‘SS Parameter for facility’ will be displayed, by default.

    6. Doc Ref– Enter the document reference number in alphanumeric format of 15 characters maximum length.

  2. In the search results, details such as Ster Req Doc Type, Ster Req Doc No, Doc Date, Service Location and Store are displayed.

  3. To acknowledge the sterilization request document of a service location or dispatched tray details, select the required check box under the Select column.

  4. To view the Group and Tray details relevant to the selected document number, click the ’Document type’ hyperlink under Ster Req Doc Type column. An Acknowledge Group Details dialog gets displayed with the Stertilization Request Number, Group, Tray Number, Received Date and Acknowledge check box.

By default, the Acknowledge check box will be selected, if the Sterilization Request document record is selected in Acknowledgement screen.

  1. You can individually select the required trays that need to be acknowledged by selecting the Acknowledge checkbox

  2. Modify the Received Date, if required.

All other details are non-editable. The current system date is displayed as the received date.

  1. Click [Apply] to confirm the acknowledgement or [Reset] to clear the entered data.

The application displays the message ‘Operation Completed Successfully’ in the status bar. The status of the records is marked as ‘Acknowledged’ and the Tray(s) are available as ‘Sterilized’ and can be used for further issue.
