Request Loan From Vendor

From this function you can Request sterile stock from third party suppliers if the required CSSD Trays are unavailable or if sufficient quantities are not available.

This function allows you to multi-select sterile stock from third party suppliers if the required CSSD Trays are unavailable or if sufficient quantities are not available.

Requesting Loan from Vendors

From the SS Transactions > Loan > select the Request Loan from Vendor function.

The Request Loan from Vendors screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Group Search] to launch the Group Search and Select dialog to search for Group details. The search records detail include Group Type, Request Group, Surgery Type, Surgery Name, Total Available Quantity in CSSD Main and Store.

    1. Search By – Select the Code or Description radio button. The search results are displayed based on selection.

    2. Search Text – Enter the keyword to search the records.

    3. Search Criteria – Select the criteria from the drop-down. The listed values are ‘Starts With’, ‘Contains’ and ‘Ends With’.

    4. Surgery Type/Name – Click [?] to search and select the CSSD Surgery types from the CSSD Surgery Type – Search and Select dialog.

    5. Search By – Select the search by option. The options available are Code and Description.

    6. Search Text – Enter the search text. If you select the search by option as Code, enter the Code in this text box. Else if you select Description, type the description here.

    7. Search - Click [Search] to view the search results.

    8. Reset – Click [Reset] to clear the values entered in the fields.

    9. Surgery Type (Code) - Displays the code of the surgery type.

    10. Surgery Type (Short Desc.) - Displays the short description of the surgery type.

    11. Surgery Name - Displays the surgery name.

    12. Surgery Name(Short Desc.) - Displays the short description of the surgery name.

    13. Group Type – Select the Group Type from the list. The values displayed are Set, Tray, Soft Pack, Single and All. The value ‘All’ is displayed, by default

    14. Consider Groups in Target Store – Select this check box to exclude the sterile groups not associated with the ‘Requested On’ store from the search results. The system excludes the sterile groups not associated with the ‘Requested By’ Store by default.

    15. Click [Search] to display the Code, Description, Surgery Type and Group Type or click [Reset] to clear all values.

    16. Select – You select the check box against each item. Select the ‘Select’ check box at the header level to select all the records.

    17. Click [OK]. The selected records gets displayed in the search results

  4. If you want to delete any record, select the Del check box against that record and click [Delete]. The selected record gets deleted.

  5. Enter the request quantity of the stocks in the text box. This is a mandatory field.

  6. Click [Apply]; the Authorize alert message popup is invoked with Yes and No buttons.

  7. Click [Yes] in popup to apply the changes. The system generates document number and displays the document at the bottom of the screen. Click [No] in popup to reset the details.

  8. Click [Reset] to clear the entered data.

  9. Click [Query] to search for the existing records
