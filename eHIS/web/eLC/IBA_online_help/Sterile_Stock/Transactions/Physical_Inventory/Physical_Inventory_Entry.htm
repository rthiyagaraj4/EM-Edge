Physical Inventory Entry

From this function you can Record the details of the Physical Stock check into the system.

The staff designated to perform the stock check will count the actual physical items against the count sheet provided. The observations are recorded in the count sheet. On recording the observations, you can enter the observed quantity for each Item into the system using this function against a Physical Inventory ID.

Recording the details of the Physical Stock Check

From SS Physical Inventory, select the Physical Inventory Entry function.

The Physical Inventory Entry screen gets displayed.

  1. Click [Query] to list all the Physical Inventory entries generated while printing the count sheet in the ‘Print Count Sheet’ function or enter the criteria and click [Search] to select a specific record.

    Enter the following search criteria:

    1. Physical Inv ID – Enter the Physical Inventory ID in numeric format of 10 digits maximum length.

    2. Owner – Select the owner store from the list. The stores defined as owner store in the CSSD Store function are displayed in the list.

    3. Current Store – Select the current store value from the list.

  2. Click ‘Phy Inv ID’ hyperlink to view the details of the selected Physical Inventory ID record. The details displayed are Physical Inventory ID, Count Sheet generation date, Owner store and the Current store.

  3. Enter the details in the mandatory (*) fields to record the Physical Inventory details. <Related Field Description>

  4. On entering the inventory details, click [Add] to add the details and the entered values will be displayed on the right hand side of the screen or [Cancel] to clear the entered values.

  5. Select the ‘Del’ check box to exclude the entered inventory details.

  6. Click ‘Group’ hyperlink to modify the inventory details.

  7. Click [Apply] to record the Physical Inventory transaction or [Reset] to clear the entered data.
