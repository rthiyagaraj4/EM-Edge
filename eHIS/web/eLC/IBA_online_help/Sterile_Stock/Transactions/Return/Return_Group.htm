Return Group

From this function you can Record the details of the Trays returned back to Owner Store.

The issued trays are returned back to the Owner Store by the requestor store. This function aids the User in recording trays returned back by the requestor store to the owner stores. The logged in user must have rights to Return Group transactions from these stores.

Recording Return Details

From SS Transactions > Return, select the Return Group function.

The Return Group screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Select Trays] to invoke the Select Trays screen.

    • The Owner Store, Bin Location, Current Store, Group, Tray no, Patient ID and Used details are displayed. Select the 'Select' check box to select the Tray to return.

    You can mark a Sterile Group as ‘Used’ or ‘Unused’. By default, the Sterile Groups will be marked as ‘Unused’, if the Sterile Group is configured as Non-Returnable in the setup and will be marked as ‘Used’, if the Sterile Group is configured as Returnable in the setup.

    • Click Close to exit from the Select Trays screen.

  4. On selecting the Tray, the tray details are displayed in the right hand side of the Return Group screen. By default, this is unchecked.

  5. Click Group Description hyperlink to modify the tray details.

    1. The Group, Tray No, Bin Location, Reason, Expired, Used, Patient ID, Patient Name, Sterilize, Wash, Pack, Contaminated and Charge Consumables details are displayed. You can click Modify and  modify the Group, Tray No, Bin Location and Patient details.

    You can select more than one group (which been issued to the store) to return at the same time.

    1. The Used check box is selected if the Tray is issued to a patient and the tray is used. Clear the check box if the tray is not used.

    2. Select the 'Contaminated' check box if the tray is contaminated.

    3. The 'Charge Consumables' check box is selected if the consumables of the tray are charged. The check box is cleared if the consumables of the tray are not charged. You can select or clear the check box. By default, this is unchecked.

    4. Enter any remarks about returning the sterile groups. If you have entered predefined text in the Transaction Remarks screen, it will be displayed here. You can click on the Reason hyperlink to view the Reason for return in a popup. If the Reason for return is not available for a specific Sterile Group, the label ‘Reason’ in that row is not hyperlinked. The reason should be included at each row of the group.

    When you click Apply, all unused sterile groups will be returned. Therefore, if you are returning sterile groups not associated with the patient, entering remarks is mandatory.

    1. Click [Discrepancy] to view or enter the discrepancy details. The Discrepancy Details screen is displayed on clicking [Discrepancy]. The discrepancy details recorded in the Instruments Usage/Discrepancy at Units function are displayed.

      1. Enter the details in the mandatory (*) fields. <Related Field Description>

      2. Click [Add] to add the discrepancy details or [Cancel] to clear the entered details.

      3. On adding the discrepancy details, the details are displayed in the right hand side of the discrepancy details screen. Click Instrument Description hyperlink to modify the discrepancy details and click [Modify] to modify the changes.

      4. Click [Close] to exit from the Discrepancy Details screen.

  6. Click [Modify] to record the modified details or [Cancel] to clear the entered details.

  7. Select ‘Del’ check box to exclude the trays from returning.

  8. Click [Apply] to record the return details or [Reset] to clear the entered details.

  9. The ‘Confirm Finalize’ dialog is displayed if the return entry is completed. Click [Yes] to confirm the return record or [No] to cancel the action.

  10. Click [Query] to invoke the un-finalized return records which are displayed based on the default criteria set in the SS Parameter for Facility function or enter the search criteria and click [Search] to select a specific record.

    Enter the following search criteria:

    1. Doc Type – Select the return document type from the list. The value as set in ‘SS Parameter’ for Return Group is made default and displayed.

    2. Doc No – Enter the return document number in numeric format of eight digits maximum length.

    3. Doc Date – Enter the return document generation date in date format of dd/mm/yyyy format.

    4. Return From Store – Select the return from store from the list.

    5. Return To Store – Select the return to store from the list.

    6. Entry – Select the value from the list. The listed values are Completed, Not Completed and All. The value 'Completed' is displayed, by default.

  11. Click ‘Doc Type’ hyperlink to finalize the return documents. You can modify the return details for the documents for which ‘Entry Completed’ check box is not selected.

  12. Click [Apply] to finalize the return document or [Delete] to delete the selected record.

  13. On clicking [Delete], the ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.

  14. A document number is generated on recording the return details.

You can return the tray only if the trays are issued to a patient.
