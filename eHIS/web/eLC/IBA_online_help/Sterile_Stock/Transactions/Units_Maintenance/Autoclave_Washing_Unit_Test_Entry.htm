Autoclave Washing Unit Test Entry

From this function you can Determine the working condition of Washing Unit and Autoclave Unit by performing various tests and Query for the un-finalized Test Entry. You can also Delete the un-finalized Test Entries.

If the outcome of the test results are successful, the unit condition is automatically set to ‘Ready to use’ in the function ‘Autoclave/Washing Unit’ or if the test result has failed, the unit condition is set to ‘Not ready to use’ in the function ‘Autoclave/Washing Unit’ until it passes the specified tests.

Maintaining Sterilization Request Status

From SS Transactions > Unit Maintenance, select the Autoclave Washing Unit Test Entry function.

The Autoclave Washing Unit Test Entry screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Add] to add the tray for Autoclave/Washing Unit Test or [Cancel] to clear the entered details.

  3. On adding, the test details are displayed in the right hand side of the screen.

  4. Click ‘Test’ hyperlink to modify the test details.

  5. The Autoclave Wash test is displayed. You can modify the value.

  6. The Batch ID and the Result Date are displayed and disabled.

  7. The Test Result field is enabled on adding the test details. Select the Test Result value from the list. The listed values are ‘Pass’ and ‘Fail’. The value ‘Pass’ is displayed by default.

  8. Click [Modify] to modify the test details or [Cancel] to clear the entered details.

  9. Select the ‘Del’ check box to exclude the test details.

  10. Click [Apply] to save the test details or [Reset] to clear the entered data.

  11. If entry is completed, the ‘Confirm Authorize’ dialog is displayed. Click [Yes] to finalize the test entry or [No] to finalize the test entry in the query mode.

  12. Click [Query] to display all the un-finalized test entry or enter the search criteria and click [Search] to select a specific record.

    Enter the following search criteria:

    1. Unit Type – Select the Unit Type from the list. The list of values are ‘Autoclave’ and ‘Washing’. The ‘Autoclave’ Unit Type is displayed, by default.

    2. Autoclave/Wash Unit - Select the Autoclave/Wash Unit from the list. The values are populated in the list based on the ‘Unit Type’ value selected. On selection of ‘Unit Type’ as ‘Autoclave’ all the Autoclave Units will be displayed in the list or on selection of ‘Unit Type’ as ‘Washing’ all the Washing Units will be displayed in the list.

    3. Test Date – Enter the date on which the test entry was performed in dd/mm/yyyy format or select the date using the Calendar function.

    4. Entry – Select the entry value from the list. The list of values are ‘All’, ‘Completed’ and ‘Not completed’. The value ‘Completed’ is displayed, by default. Select ‘Completed’ to record the test result for the entered test and ‘Not completed’ to complete the test entry details.

  13. Click Unit Type hyperlink to finalize the test entry. You can modify the test details for which ‘Entry Completed’ check box is not selected.

  14. Click [Apply] to finalize the test entry or select the ‘Del’ check box and click [Delete] to delete the test entry.

  15. On clicking [Apply] the ‘Confirm Authorize’ dialog is displayed. Click [Yes] to finalize the test entry or [No] cancel the action.

  16. On clicking [Delete], the ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.
