Statistical Group

From this function you can Define the reporting groups that provide statistical data. You can also Query for the existing records.

Statistical Group is a user defined reporting group that provides statistical data on number of sterilized trays in a functional unit. Each Sterile Stock Group is associated to a statistical group.

Defining the Statistical Group

From SS Code Setup, select the Statistical Group function.

The Statistical Group screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory fields. <Related Field Description>

  3. Click [Apply] to save the defined Statistical Group or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Statistical Group records.

Tray is the common word used to denote a Set, Tray, Soft pack or Single.
