CSSD Group

From this function you can Define Sterile Stock Groups and their attributes and Query for the existing records.

A Group is a collection of Items/Components with defined set of properties. The properties that are defined for a group are Item Class, Usage Type, Surgery Type, Statistical Group, Sterilization Type, Washing Type and/or Packing Type. The Groups are defined for a Group Type and associated to an Item for a specified quantity. The group types available are Tray/Set/Soft Packs/Single.

The group item definition cannot be altered once the group is prepared. Groups are prepared using the ‘Prepare’ transaction function. You can modify the group details on completing the process which user has defined at the time of creating a Group and on performing the Physical Inventory process for that Group.

Defining Sterile Stock attributes for a Group

From SS Code Setup, select the CSSD Group function.

The CSSD Group screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields of the following tab pages.

General

Additional

  1. On defining the required properties of a group, select the items for the defined group and specify the required quantities.

    1. Item - Enter the required CSSD Item code to be added to the group in alphanumeric format of 40 characters maximum length or click [?] function to search for the required Item. This is a mandatory field.

    2. Item Qty – Enter the quantity of the selected Item in numeric format of 12 digits maximum length. This is a mandatory field.

    On specifying the Item and Quantity, click [Add] to insert the item to the group. On adding, the Item details are displayed on the top half of the screen. Click [Cancel] to clear the recorded values.

    The following details are displayed on selecting the required items.  

    1. Del – Select the check box to delete an Item from the group. The selected Item will be removed from the group on saving the record.   

    2. Item – The Items selected for the group are displayed and hyperlinked. Click on the hyperlink to modify the item quantity. Click [Modify] to modify the Item Quantity or [Cancel] to cancel the process.

    3. Consumable – A ‘’ indicates that the item is consumable and ‘’ mark indicates item is non-consumable.

    4. Chargeable - A ‘’ indicates that the item is chargeable and ‘’ mark indicates item is not charged.

    5. Quantity – The quantity specified for the item is displayed.

  2. Click [Apply] to save the defined CSSD Group or [Reset] to clear the entered data.

  3. Click [Query] to search for existing CSSD Group records.
