Washing Type

From this function you can Define the various types of washing adopted in Sterilization Process. You can also Query for the existing records.

The defined Washing Types can be associated to a group while defining the Sterile Stock groups. These Washing Types are used during the Sterilization Process. Examples for Washing Types are: Hand wash, Ultrasonic wash and so on.

Defining the Washing Types

From SS Code Setup, select the Washing Type function.

The Washing Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory fields. <Related Field Description>

  3. Click [Apply] to save the defined Washing Types or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Washing Type records.

Only 10 records can be added at a time. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate code with a unique color code. If the mandatory values are not entered, the system alerts you to enter these values. The mandatory field is highlighted with a unique color code.
