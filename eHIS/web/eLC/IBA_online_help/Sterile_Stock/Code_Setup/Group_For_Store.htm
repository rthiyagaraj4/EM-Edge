Group For Store

From this function you can Associate a Group to multiple Stores and Query for the existing records.

The Store can be an Owner Store or a Requestor Store. A Store can request only from the associated groups in the transaction functions.

Associating a Group to Stores

From SS Code Setup, select the Group For Store function.

The Group For Store screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Group code in alphanumeric format of 15 characters maximum length or click [?] function to select the Group. This is a mandatory field.

  3. On selecting the required group, the Sterile Stock stores are displayed in the ‘Select’ tab.

  4. Click Select tab to create an association of Group to Store.

  5. Select the ‘Enabled’ check box, to enable the association of the Store to the Group. Clear the check box, to disable the association. The check box is selected, by default.

  6. Select the ‘Select’ check box if the required Store to be associated to the selected Group. The check boxes are cleared, by default.

  7. The Reorder Level column displays the minimum level of stock quantity maintained within the OT Store. This column appears only if the OT store is interfaced with Central Sterile Stock Department (CSSD).

  8. The Reorder Quantity column defines the minimum reorder quantity.

If the Quantity on Hand for a store goes below the defined reorder level, the system will generate a ‘Un-Finalized’ automatic request against the store for the CSSD group. The automatic request quantity will be based on the Reorder Quantity specified.

  1. Click Association tab to view the associated Stores. The associated Stores are displayed with a unique color code and Stores associated during the current process are displayed below the previously selected values. You can change the associated Stores in this tab.

  2. Click [Apply] to save the associated Group to Store combination or [Reset] to clear the entered data.

  3. Click [Query] to search for existing Group to Store combination records.

The applicable Stores associated to the selected Group Name is displayed with a unique color code. You can associate additional Stores if required and view them under the Association tab.

The Stores associated earlier are displayed with a unique color-code in the Association tab. The Stores associated to the selected Group Name currently are displayed with ‘Select’ check boxes selected and with a different color-code (Black Color). You can clear the check box and the selected Store will not be associated to the selected Group Name.
