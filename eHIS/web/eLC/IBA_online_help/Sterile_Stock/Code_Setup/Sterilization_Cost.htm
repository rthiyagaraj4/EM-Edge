Sterilization Cost

From this function you can Define sterilization cost applicable to a Group and Sterilization Type combination and Query for the existing records.

Defining Sterilization Cost applicable to a Group

From SS Code Setup, select the Sterilization Cost function.

The Sterilization Code screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Group code in alphanumeric format of 15 characters maximum length or click [?] function to select the Group. This is a mandatory field.

  3. On selecting the required group, the Sterilization Types associated to the selected groups are displayed.

  4. Select the ‘Sterile Cost’ check box against the Sterilization Type if the sterilization cost is application for the Sterilization Type. The check boxes are cleared, by default.

  5. Select the ‘Previous’ check box to enter the cost of sterilization in numeric format of 16 digits maximum length. This field is enabled on selecting the ‘Sterile Cost’ check box. This is a mandatory field.

  6. The ‘Delete’ check box is disabled for new records.

  7. Click [Apply] to save the defined Sterilization Cost or [Reset] to clear the entered data.

  8. Click [Query] to search for existing Sterilization Cost records.

The Sterilization Types associated to the selected Group are displayed with a unique color coded.  

The ‘Sterile Cost’ check box is not available for associated Sterilization Type.

To delete the Group and Sterilization Type association, select ‘Delete’ check box. The delete check box will be enabled only for sterilization types that are already associated to the selected group. You can associate additional Sterilization Types for the group and sterilization type which is already defined.
