Enter the following details:

  1. Transaction Applicable – Click [?] to select the user name. The users who have access rights for creation, Finalize/Authorize, Cancel and Deletion are listed.

  2. Create – Select this check box to provide user access rights for creating tender.

  3. Finalize/Authorize – Select this check box to provide access rights for finalizing/authorizing the tender.

  4. Cancel – Select this check box to provide user access rights for cancelling tender.

  5. Delete – Select this check box to provide user access rights for deleting tender.
