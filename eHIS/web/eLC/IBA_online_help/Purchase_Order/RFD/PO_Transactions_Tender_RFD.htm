Enter the following details:

  1. Tender Type – Select the Tender Type from the list. This is a mandatory field.

  2. Tender No – Enter the Tender number.

  3. Tender Date – Select the Date by clicking the calendar function. By default, the current date is displayed.

  4. Purchase Unit – Select the Purchase unit for the tender.

  5. Tender Ref – Enter the Tender Reference, if any.

  6. Additional Details – Click Additional Details hyperlink to view or modify additional details of the tender.

    1. Tender Approval Date – Select the approval date of the tender using the calendar function.

    2. Tender Expiry Date – Select the expiry date of the tender using the calendar function.

    3. Contract Sign Date – Select the date on which the tender contract was signed using the calendar function.

    4. Tender Remarks – Enter the tender remarks in the text box provided. You can enter upto 400 characters as free text.

    5. OK – Click [OK] to close the Additional Details dialog.

     

  7. Supplier Details – Click Supplier Details hyperlink to view or modify the details of the supplier.

    1. Supplier Name – Select the supplier name from the drop-down list.

    2. Contact Person – Displays the name of the contact person. This field will be in disabled state.

    3. Contact Number – Displays the contact number of the contact person. This field will be in disabled state.

    4. Select – Select the check box to choose the supplier for the tender.

    5. Add Row – Click [Add Row] to add a new row for including an additional new supplier.

    6. OK – Click [OK] to close the Supplier Details dialog.

     

  8. Local Currency – The Local Currency for the facility is displayed, by default.
