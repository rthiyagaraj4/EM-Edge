TAT Defect Status

From this function you can Generate a Turnaround Time Defect Status report. You can Print the statement or Save the output in a file.

This function is used to generate a Turnaround Time Defect Status report.

Print TAT Defect Status report

From the Reports > General, double-click Patient TAT Defect Status function.

The TAT Defect Status screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run Process] to generate the report. A Report Option gets displayed with the preferences to View, Print and File options.

    Click ' File' to store the output into a file. Depending on the nature of the report, system prompts to select a file format. Choose the appropriate file format from the available list. You can also optionally enter the output file name.

    Click ‘Print’ to print the report on the designated printer. The report is routed to the default printer set to print the report output.

    Click ‘View’ to view the report on the screen. A note 'Process Successful. Click OK to view report' appears. Click [OK] to view report or [Cancel] to close the report viewer.

  3. Click [OK] to generate the report or [Cancel] to cancel the transactions.
