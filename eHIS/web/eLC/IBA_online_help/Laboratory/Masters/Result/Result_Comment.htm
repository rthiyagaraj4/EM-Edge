Result Comment

From this function you can Create Result Comments specific to a Department to assist you while processing the specimen results. You can Query for the existing records.

A department can define the result comments that are used while resulting investigation (tests). These comments assist you in processing the specimen results. During set up or when required, certain result comments linked to the different investigations could be marked for review. If the results for such investigations have the marked comment specified, the request on resulting is held for review.

Creating Result Comment

From the Masters > Result, double-click Result Comment function.

The Result Comment screen gets displayed.

  1. The logged in Department Code is displayed, by default. Select desired department from the list, if you have access to more than one department. This is a mandatory field.

  2. Enter a unique Result Comment code in alphanumeric format of four characters maximum length. This is a mandatory field.

  3. Enter the description for the result comment code in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  4. Click [Save] or press [F10] to save the records or click [Clear Record] to clear the entered data.

  5. Click [Execute Query] or press [F8] to search for the existing records.
