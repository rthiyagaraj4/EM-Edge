Delete Worklist

From this function you can Delete a worklist if the worksheet is processed.  

System displays the worklists, which have been created for the laboratory department. You can mark the worklist for deletion, which are no longer required.

Deleting Worklist

From the Transactions > Worklist menu, select the Delete Worklist function.

The Delete Worklist screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Select the ‘Delete?’ check box to delete the desired worklist. The check box is cleared, by default.

  3. Click [Commit] to delete the selected worklist or [Clear Record] to delete the entered details.
