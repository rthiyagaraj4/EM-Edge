Result Entry by Worklist

From this function you can Record the test results for the specimens that are grouped by a worklist.

Result Entry by Worklist function is used to enter the results for all the specimens grouped together on a worklist. If analyzers were linked to the system, the results from the analyzers will be updated for the specimen and need not be entered again. The specimens on the worklist may be patient sample ‘P’ or quality control ‘Q’ material. Numeric result or coded comment for each test may be entered as results. The system provides an option to release the result for further processing or to hold the result. If the result is released, various validations will be applied to the results as defined in the test setup. On failure of the validation, the specimen will be directed to a review function.

This function works in two stages:

First stage accepts the worklist name, number and date for an existing worklist. The results, if available, for all the tests covered in the worklist are displayed.

Second stage allows the result entry through that worklist.

The result entry process is considered complete only when both the stages are successfully carried out.

Result Entry by Worklist

From Laboratory Transactions > Result Entry menu, double-click the Result Entry by Worklist function.

The Result Entry by Worklist screen gets displayed.

  1. Select the Department from the list. The name of the Department associated to the logged in user is displayed. If the user has access to multiple departments, select the desired department from list.

  2. Enter the details in the (*) fields in Worklist Details section.

    Enter the following details:

    1. Perform Site – The Perform Site name for the work list is displayed.

    2. Work List Name – Enter the name of the work list or click [List of Values] to select the Work list from the list. This is a mandatory field.

    3. Work List Date – Enter the work list creation date or click [List of Values] to select the Work list creation date from the list. This is a mandatory field.

    4. Work List Number - Enter the work list number or click [List of Values] to select the Work list number from the list. This is a mandatory field.

    5. Include Cancelled Specimen/Test – Select the check box to include the cancelled specimen or test in the work list to process the result.

  3. Click [Save] or press [F10] to confirm the Worklist; click [Clear Record] to clear the entered details.

  4. On pressing [Tab] a confirmation message ‘Do you want to confirm?’ is displayed. Click [Yes] to confirm the result entry or [No] to exit.
