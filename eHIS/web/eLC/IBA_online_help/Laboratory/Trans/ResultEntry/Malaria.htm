Malaria

From this function you can Result the test results for the specimens for which Malaria test is requested.

This function is applicable only to Hematology. You can enter the results as the slide is being analyzed by clicking the keys assigned for different cells. If the specimen requested is awaiting any review, an appropriate message is displayed. If request has no outstanding Malaria test, an alert message is displayed. All the test records as defined in the Malaria Parameter are displayed.

Resulting Malaria test Results

From Laboratory Transactions > Result Entry menu, double-click the Malaria function.

The Malaria screen gets displayed.

  1. Enter the search criteria to search for the specimens to record the result. <Related Field Description>

  2. Click [Patient Details] to view the patient details, [Patient Marker] to mark the test as patient marker, [Process Sheet] to add the template, [Add. Clinician Comments] to add any more clinician comments, [Clinician Comments] to view the Clinician comments, which is given while registering; [Comments] to enter the comments, [Textual Result] to enter the text, which is entered in Masters; [Compute] to calculate the result, [Request] to request for more than one test.

  3. Click [Release] to release the result, [Multiple] to extrapolate the count to 500 if the pattern is found uniform, [Direct Result] to view all the direct results, [Hold] to hold if there is any confirmation required, [Prv Result] to view the previous result, [Bypass] to bypass the results, [Cancel] to cancel the entered details and [Keyboard] to enter the comments in KB text block.
