Result Entry by Specimen

From this function you can Record the test results for a specimen and Hold or Release the specimen for further processing.

Result Entry by Specimen function is used to enter the results for the tests for a given specimen number. If analyzers were linked to the system, the results from the analyzers would have been updated for the specimen and need not be entered again. The result for a test can have a numeric value or coded comments/descriptions. All or any combination of these components may be entered as results, based on the test code setup. If released, the various validation rules would be applied to the results as defined in the test code setup. On failure of the validation rules, the specimen would be directed to a review function.

Result Entry by Specimen

From Laboratory Transactions > Result Entry menu, double-click the Result Entry by Specimen function.

The Result Entry by Specimen screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click ‘Tab’ to view the Patient details and test codes.

  3. Click [View Previous Result] to view the previous results.

  4. Enter the test values under Investigations.

    Enter the following details:

    1. Group – The Group test requested to which the test code belongs is displayed. In case of individual test code, group test code is left blank.

    2. Test - The code for the test requested is displayed.

    3. Previous – The previous result of the selected test is displayed.

    4. Num Result – Enter the numeric result if the selected test is numeric.

    5. Comment Result – Select the comment if the selected test is comment result. Click [List of Values] to select the Media Type Screen.

      Enter the following details are displayed:

      1. Media Type - Click [List of Values] to select the Media Type.

      2. Batch Number – Click [List of Values] to select the Batch Number.

      3. Expiry Date – Select the Expiry Date using the calendar function.

    6. Add. Work Count – Specify the count for additional workload done for the test.

    7. Hide Comments? – Select the check box to inhibit comments from the result report.

    8. [B] – Click [B] to view the block details. This is applicable for Histopathology and Cytology only.

  5. Click [Past Numeric Result] to view graphical details of the test.

  6. Click [View/Print Draft Copy] to the entered value in the reports.

  7. Click [Sample Details] to view the sample details. This is applicable for Histopathology and Cytology only.

  8. Click [Comment] to enter multiple comments for the tests. This button is enabled if you have recorded values in ‘Result Comment’ tab.

  9. Click [Text] to enter the clinical notes for the external patients. This button is enabled if you select ‘Text Block’ check box in Test Code Masters.

  10. Click [SNOMED] to enter the SNOMED codes for the selected test codes. This button is enabled if you select 'Nomenclature' check box in Test Code Masters.

  11. Click [Organism] to invoke the Sensitivity Details screen to enter the organism and the antibiotics for the selected tests. This button is enabled if you select 'Culture Test’' check box in Test Code Masters. This is applicable for Microbiology Department only.

  12. Click [User Marker] to mark the specimen for further investigation.

  13. Click [Modify/Addn] to modify the test result if it is already resulted.

  14. Click [Request] to add more investigations to the request.

  15. Click [Release] for further processing or to hold. The system prompts for the user PIN code if set up to authenticate the user. If the results are released the system validates all the results based on the parameters set for each of the test. The validations applied are Result Range Select, Delta Check, Quality Check, Request Comment Check and Result Comment Check

  16. Click [Hold] to record the results and validate them later.

  17. Click [Compute/Validate] to compute the result for the test and display the result.

  18. Click [Cancel] to cancel the processing of the specimen. Any results entered for the specimen will not be recorded.

  19. Click [Attach Image] to attach one or more the microscopic images or digital images and documents. To view the uploaded file, click View hyperlink. The Image Links dialog gets displayed. You can delete the uploaded file by selecting the Delete check box against the uploaded file. The uploaded images can be viewed in Result – Laboratory Order, Order Tracking, View Chart Summary, Events occurred in last 24 hours in Patient Chart, Clinical Event History – Flow sheet, expanded and collapsed mode screens in CA module.

  20. Click [Notify] to view and record the Result Notification text which need to be notified to other users

  21. Click [Apply] to delete the selected file or click [Close] to close the Image Links dialog.

 

The Numeric Result High/Low column is highlighted in violet if the investigation numeric result is less than the Critical Check value setup in the Test Codes screen.
