Result Entry by Assign To/Patient Class/Regn. Date Range

From this function you can Record the test results for the specimens that have been assigned to a pathologist for reporting.

Result Entry by Assign To/Patient Class/Regn. Date Range function allows you to list out specimens that have been assigned to pathologist for reporting. You can select the Specimen to record the test results through Result Entry function.

Result Entry by Assign To/Patient Class/Regn. Date Range

From Laboratory Transactions > Result Entry menu, double-click the Result Entry by Assign To/Patient Class/Regn. Date Range function.

The Result Entry by Assign to/Patient Class/Regn. Date Range screen gets displayed.

  1. Enter the search criteria to search for the specimens to record result.

    Enter the following search criteria:

    1. Department – The name of the Department associated to the logged in user is displayed. If the user has access to multiple departments, select the desired department from list.

    2. Perform Site – Click [List of Values] to select the desired Perform site to process the specimen from a list. The displayed values are defined in the ‘Perform Site’ function in the masters.

    3. Option – Select the desired Option from the list. The valid values are ‘All’, ‘Assign To’ and ‘Non Assign to’. The value ‘All’ is displayed, by default.

    4. Priority – The priority of the Specimen is displayed. The valid values are ‘All’, ‘Routine’, ‘Urgent’ and ‘Stat’. The value ‘All’ is displayed, by default.

    5. Selection – Select the desired value from the list. The valid vales available are ‘All’, ‘Result not Released’, ‘Technical’, ‘Abnormal’ and ‘Normal’. The value ‘All’ is displayed, by default.

    6. Patient Class – Select the required patient class from the drop-down list. The valid values are ‘Inpatient’, ‘Outpatient’, ‘Emergency’, ‘Daycare’, ‘External’ and ‘All’. By default, the value ‘All’ is displayed.

    7. Spec Regd Date [From – To] – Select the specimen registration date and time by clicking the calendar icon.

  2. Click [Patient Details] to view the corresponding Patient details.

  3. Click [Fetch] to view the specimen details satisfying the search criteria or [Clear] to clear the entered details.

    The following details are displayed:

    1. Specimen No. – The specimen numbers satisfying the query criteria are displayed. The color code associated to the specimen displays the status of the specimen.

    2. Registered Date Time – The Date and Time of specimen registration is displayed.

    3. Priority – The Priority status of the specimen is displayed.

    4. Comment – Any request comments entered as part of registration is displayed.

    5. Test for the Specimen - The system displays the ordered Test Codes and the corresponding Description as defined in the Test Codes master function.

  4. Click [Enter Result] and the system will take you to the corresponding screen based on the status of the specimen. If the specimen status is ‘Review’, the ‘Review’ screen is displayed or the resulting screen is displayed. You can ‘Review’ or ‘Result’ the specimen.
