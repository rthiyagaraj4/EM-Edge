Procedure Coding

From this function you can Associate procedure codes to tests.

The Procedure Link can be set as Not Applicable, Auto-Link or Manual-Link in the Order Catalog function of Order Entry and Result Reporting module.

Associating Procedure Codes to Tests

From Laboratory > Transactions > Result Entry menu, select the Procedure Coding function.

The Procedure Coding screen gets displayed.

  1. Enter the details in the relevant fields. <Related Field Description>

  2. Click ‘Tab’ to view the Patient details.

  3. Click [Show Details] to view the Specimen details and Procedure Coding details.

    Specimen details

    1. Specimen No. - The Specimen Number is displayed.

    2. Patient ID - The Patient ID is displayed, for which the specimen is registered.

     

    Procedure Coding

    1. Test (Clinical Event Nomenclature) - The Test associated to the Specimen No is displayed.

    2. Procedure Code (ICD10-PCS Nomenclature) – Select the Procedure Code to associate to the Test.

    3. Group Test Code – The group test code name is displayed.

    4. Group Procedure Code (ICD10-PCS Nomenclature) – The group procedure code name is displayed.

  4. Click [Save] or press [F10] to save the entered details.
