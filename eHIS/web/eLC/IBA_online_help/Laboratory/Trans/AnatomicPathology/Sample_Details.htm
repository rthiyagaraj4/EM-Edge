Sample Details

From this function you can Record the Sample Details of a Patient.

The Sample Details function is used to process the sample details. You can enter the sample details of a patient for the processing. Specimen number for the particular patient sample has to be entered to query the details.

Recording Sample Details

From Laboratory Transactions > Anatomic Pathology menu, double-click the Sample Details function.

The Sample Details screen gets displayed.

  1. Select the Department from the list. The logged in Department Code is displayed, by default. Select desired department from the list if you have access to more than one department.

  2. Enter the Specimen Number or click [List of Values] to select the Specimen Number to record the sample details.

  3. On selecting the specimen number, click [Tab] to view the details of the specimen.

  4. Click [Add Clinical History] to add the details of the clinical history of the patients. These clinical history codes are defined in the Clinical History master function. Click [Close] to return to Sample Details screen.

  5. Click [Clinical Comments] to view the recorded clinical history and clinical comments details of the patient. Click [Close] to return to the Sample Details screen.

  6. Click [Patient Details] to view the details of the Patient.

  7. Enter details in the mandatory (*) fields of the following tab pages.

Sample

  1. Sample ID -   The sample ID for the specimen is displayed on clicking [Add Sample].

  2. Test Code – Select the desired Test Code from the [List of Values] function. The list displays the values defined in Test Codes masters function. This is a mandatory field.

  3. Anatomy Site – The Anatomical site of the specimen from where the sample has been taken has to be entered. Select the desired Anatomy Site from the [List of Values] function. The list displays the Anatomy Sites associated with the selected test code. This is a mandatory field.

  4. Tissue Description – Select the appropriate Tissue Description from the [List of Values] function. The list displays the Tissue Description associated with the selected Anatomy site.

  5. Fixative – Select the appropriate Fixative Code from the [List of Values] function. The list displays the Fixative recodes associated with the selected test code.  This is a mandatory field.

  6. Adequacy – Select the desired Adequacy code from the [List of Values] function. The list displays the values defined in Adequacy masters function.

  7. Reject? - Select the 'Reject?' check box if you want to reject the sample. Specify the appropriate reason for rejection.

  8. Reject Code – Enter the Reject Codes or select it from [List of Values] functions. The listed reject codes are created in Masters.

  9. Click [Add Sample] to add a sample Identification number.

Blocks

Add

  1. Sample ID – The sample ID for the specimen is displayed.

  2. Test Code – Enter the code in alphanumerical format of 10 characters maximum length or select it using [List of Values] function. The listed values are created in Test Code function.

  3. Anatomy Site – Enter the anatomy site code in alphanumerical format of four characters maximum length or select it using [List of Values] function. The listed values are created in Anatomy Site function.

  4. Description – The corresponding description for the select the Anatomy site code is displayed, by default.

  5. Tissue – Enter the Tissue code in alphanumerical format of four characters maximum length or select it using [List of Values] function. The listed values are created in tissue description masters function.

  6. Description – The corresponding description for the select the Tissue is displayed, by default.

  1. Click [Save] to save the recorded sample details or [Clear Record] to clear the entered details.
