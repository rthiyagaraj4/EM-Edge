Specimens Linking Query

From this function you can Query for linked specimens and do result entry for the linked specimens.

From the function you can query for the specimens linked to the parent specimen and do result reporting.

Querying Linked Specimens

From Laboratory > Transactions > Specimen menu, select the Specimens Linking Query function.

The Specimens Linking Query screen gets displayed.

  1. Select the required department from the list. By default, the department name associated with the logged in user is displayed.

  2. Enter the patient ID or search for the required patient ID using 'Right click for User option'.

  3. Enter the category code in the text box.

  4. Click [Display].  The Patient Name, Age and specimen details satisfying the query criteria are displayed. The parent specimen detail is listed in the first section while the child specimen linked to the pnarent is listed in the second section.

  5. Click [Enter Result] against the specimen record to enter results for the child specimen linked to the parent. The colour of the specimen record changes to red if review is done and green if result is completed.

  6. Click [Patient Details] to view the details of the patient.

  7. Click [Patient Marker] to view the details of the patient marker and the results of the marker tests and their results. Or to mark the test for further investigation.

  8. Click [Clinical Comments] to view the clinical comments of the patient.

  9. Click [Add Clinician History] to add clinician comments on the selected patient.
