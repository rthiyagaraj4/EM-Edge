Acknowledge/Verify Specimens

From this function you can Verify and Acknowledge the registered specimen to ensure that the details entered are correct.

This function is used to verify the registered specimen to ensure that the details entered are correct and to generate category numbers (optional). You can assign specimens  to a particular user using this function. This function is not mandatory and can be activated at the laboratory department level, if it is activated then all the requests registered need to be verified before any further processing.

Acknowledge/Verify Specimens

From Laboratory > Transactions > Specimen menu, select the Acknowledge/Verify Specimen function.

The Acknowledge/Verify Specimen screen gets displayed.

  1. Enter the details in the respective fields or click [Execute Query] to search and select the desired Specimen number to display the patient and investigation details to acknowledge or verify. <Related Field Description>

  2. Click [Specimen] to invoke the Specimen Registration screen to process (acknowledge) the specimen.

  3. Enter the Category Code and click [Verify] to verify the specimen. The system generates the category number on verification.

  4. The test details are displayed in the Investigation section.

To enable the Category Code in Acknowledge/Verify function, create a Category Codes in Masters and in Administration setup under Departmental Parameter for Facility select ‘Usage of Category No’ check box.
