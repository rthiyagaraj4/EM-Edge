Record Follow up

From this function you can Record the patients status, reported for Nosocomial Infection during the follow up.

Record Follow Up function is used to record if the patient has any infections before admitting to the facility or hospital. The treatment given thereafter and its follow up.. The record is updated over the period of time based on the follow up stages of the treatment for the infection and is finally marked as completed when the infection is over.

Recording Follow up

From Laboratory Transactions > Nosocomial Infection menu, double-click the Record Follow up function.

The Record Follow up screen gets displayed.

  1. Click [Search] to display all the Outstanding/In-process or enter the search criteria to select a specific patient to record the follow-up status.

  2. On entering the search criteria, click [Search] to display the records based on the entered criteria.

  3. On searching, the Patient ID, Patient Name, Serial Number, Encounter ID, Onset Date, Ward, Specialty, Status, Diagnosis recorded and the recorded Remarks are displayed.

  4. Double click on the Patient ID to select the record and the ‘ Record Follow Up’ screen gets displayed.

  5. Click ‘ Infection Type’ Tab to record the infection type(s) applicable for the patient.

    Enter the following details in the Infection Type tab:

    1. Infection Type Code – Enter the Infection Type code in alphanumeric format of six characters maximum length or click [List of Values] to select the Infection Type code. The values displayed are defined in the Application Masters module. This is a mandatory field.

    2. Description – On selecting the Infection Type code, the description of the selected Infection type code is displayed.

    3. Remarks – Enter the remarks for the infected patient details.

  6. Click ‘ Predisposing Factors’ tab to record predisposing details of the patient.

    Enter the following details in the Predisposing Factors tab:

    1. Predisposing Factors Code – Enter the Predisposing Factors code in alphanumeric format of six characters maximum length or click [List of Values] to select the Predisposing Factors code. The values displayed are defined in the Application Masters module. This is a mandatory field.

    2. Description – On selecting the Predisposing Factors code, the description of the selected Predisposing factors is displayed.

  7. Click ‘ Devices’ tab to record onset devices record of the patient.

    Enter the following details in the Devices tab:

    1. Onset Devices Code – Enter the Onset Devices code in alphanumeric format of six characters maximum length or click [List of Values] to select the Onset Devices code. The values displayed are defined in the Masters.

    2. Description – On selecting the Onset Devices code, the description of the selected Onset Device is displayed.

    3. Date – Enter the Date to record the onset infected date or select the date using the Calendar function.

    4. Remarks – Enter the remarks for the onset infection of the patient.

  8. Click ‘ Antibiotic’ tab to enter the antibiotics that the doctor prescribed to the patient during his/her stay at the hospital as an inpatient.

    Enter the following details in the Antibiotic tab:

    1. Drug – Enter the Drug in alphanumeric format of 20 characters maximum length or click [List of Values] to select the Drug. The values displayed are defined in the ‘Antibiotic/Organism’ function in Masters.

    2. Description – The description for the selected Drug is displayed.

    3. Dose – Enter the dosage for the selected patient.

    4. Route - Enter the Route for antibiotics. The route is defined in the ‘Admin Route’ function of Pharmacy Management module.

    5. Frequency - Enter the Frequency of the antibiotic which needs to be taken per day.

    6. Start Date - Select the Date when the antibiotics are given to the patient using the Calendar function.

    7. End Date - Select the Date when the antibiotics have been stopped for the patient using the Calendar function.

    8. Remarks - Enter the remarks on the patient’s status after administering the antibiotics intake.

  9. Click ‘ Clinical Data’ tab to enter the clinical details.

    Enter the following details in the Clinical Data tab:

    1. Date and Time – Select the date and time of patients follow up using the Calendar function.

    2. Remarks – Enter the clinician remarks of the selected patient.

  10. Click ‘ Specimen Details’ to enter the specimen details.

    Enter the following details:

    1. Specimen No – Enter the Specimen Number or click [List of Values] to select the Specimen Number. This is a mandatory field.

    2. Specimen Type/Description – The type and description of the specimen for the selected specimen, is displayed.

    3. Infection Type Code / Description – The type and description of the Infection Type for the selected specimen, is displayed.

    4. Remarks – Enter the remarks for the selected specimen.

  11. On entering the details, click [Save] to save the record or [Clear Record] to clear the entered details.
