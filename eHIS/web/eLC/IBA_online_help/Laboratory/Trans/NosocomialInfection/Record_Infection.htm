Record Infection

From this function you can Record the details for a patient reported for Nosocomial Infection.

Record Infection function is used to record the details for the patient reported for Nosocomial Infection whether the patient is infected on site or before admission. The record is updated over the period of time based on the follow up stages of the treatment for the infection and is finally marked as completed when the infection is over.

Recording Infection

From Laboratory Transactions > Nosocomial Infection menu, double-click the Record Infection function.

The Record Infection screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. On entering mandatory details, press [Tab] to view the Specimen details.

  3. Enter the details in the following tab pages.

Infection Type Tab

Click ‘Infection Type’ Tab to record the infection type(s) applicable for the patient.

  1. Infection Type Code – Enter the Infection Type code in alphanumeric format of six characters maximum length or click [List of Values] to select the Infection Type code. The values displayed are defined in the Application Masters module. This is a mandatory field.

  2. Description – On selecting the Infection Type code, the description of the selected Infection type code is displayed.

  3. Remarks – Enter the remarks for the infected patient details.

Predisposing Factors Tab

Click ‘Predisposing Factors’ Tab to record predisposing details of the patient.

  1. Predisposing Factors Code – Enter the Predisposing Factors code in alphanumeric format of six characters maximum length or click [List of Values] to select the Predisposing Factors code. The values displayed are defined in the Application Masters module. This is a mandatory field.

  2. Description – On selecting the Predisposing Factors code, the description of the selected Predisposing factors is displayed.

Devices Tab

Click ‘Devices’ tab to record onset devices record of the patient.

  1. Onset Devices Code – Enter the Onset Devices code in alphanumeric format of six characters maximum length or click [List of Values] to select the Onset Devices code. The values displayed are defined in the Masters.

  2. Description – On selecting the Onset Devices code, the description of the selected Onset Device is displayed.

  3. Date – Enter the Date to record the onset infected date or select the date using the Calendar function.

  4. Remarks – Enter the remarks for the onset infection of the patient.

Antibiotics Tab

Click ‘Antibiotic’ tab to enter the antibiotics that the doctor prescribed to the patient during his/her stay at the hospital as an Inpatient.

  1. Drug – Enter the Drug in alphanumeric format of 20 characters maximum length or click [List of Values] to select the Drug. The values displayed are defined in the ‘Antibiotic/Organism’ function in Masters.

  2. Description – The description for the selected Drug is displayed.

  3. Dose – Enter the dosage for the selected patient.

  4. Route - Enter the Route for antibiotics. The route is defined in the ‘Admin Route’ function of Pharmacy Management module. .

  5. Frequency - Enter the Frequency of the antibiotic which needs to be taken per day.

  6. Start Date - Select the Date when the antibiotics are given to the patient using the Calendar function.

  7. End Date - Select the Date when the antibiotics have been stopped for the patient using the Calendar function.

  8. Remarks - Enter the remarks on the patient’s status after administering the antibiotics intake.

  9. Click [Populate from Drug Profile] to find the antibiotics.

  1. On entering the details, click [Save] to save the record or [Clear Record] to clear the entered details.
