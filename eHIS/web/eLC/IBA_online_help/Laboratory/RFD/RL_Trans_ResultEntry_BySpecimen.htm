Enter the following details:

  1. Department - The name of the department associated to the logged in user is displayed. If the user has access to multiple departments, select the desired department name from Department list. This is a mandatory field.

  2. Perform Site - The Perform Site where the test is performed is displayed. The Perform Sites defined in the Masters are displayed.

  3. Specimen No – Enter the Specimen number in alphanumeric format of 13 characters maximum length or select the Specimen Number using [List of Values]. This is a mandatory field. A green colour indicator appears next to the Specimen No if the result is modified for that specimen.

  4. Specimen Type – The code and short description of the specimen type is displayed and you cannot edit the same.

  5. Workgroup – Enter the Workgroup in alphanumeric format of four characters maximum length or select it using [List of Values] function. This work groups are created in the Masters.

  6. Unit Number – The Unit number is displayed, if the investigation is from Blood Donor and Blood Transfusion Services module.

  7. Assign To – The name of the technician who is assigned to the selected test in the Acknowledge/Verification function is displayed.

  8. Priority – The specimen's priority indicator is displayed. The valid values are 'Routine', 'Urgent' and 'Stat'.

  9. Patient ID – The Patient ID will be displayed.

  10. Age – The patient's age is displayed.

  11. Priority Specimen – Select the check box to consider the specimen as high priority.

  12. Sex – The patient's Gender is displayed.

  13. [Patient Details] – Click [Patient Details] to view the patient admission details.

  14. [Patient Markers] – Click [Patient Marker] to mark the test for further investigation.

  15. [Clinician Comments] – Click [Clinician Comments] to view the comments given while registering the sample.

  16. [Process Sheet] – Click [Process Sheet] to enter the template.

  17. [Add Clinician History] – Click [Add Clinician History] to add clinician comments on the selected patient.

  18. [BB Comments] – This button appears only if the Department is selected as ‘BTS’. Click [BB Comments] to view the blood bank comments for the selected patient. You can add comments if required. The BB Comment codes defined in the Result Notify/BB Comments master appear in the List of Values.
