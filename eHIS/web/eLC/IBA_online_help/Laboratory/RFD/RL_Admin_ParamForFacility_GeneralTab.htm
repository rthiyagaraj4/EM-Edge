General

Enter the following details in the General Tab:

  1. Billing Module Interface – Select the check box to enable the billing module interface.

  2. Billing Sample/Task – Select the Billing Sample/Task check box to enable billing for Sample Details and Request Task details. You can select the check box only if the ‘Billing Module Interface’ check box is selected.

  3. Billing External AR Doc – Click [List of Values] to select the external Accounts Receivable document. The list displays the values defined in ‘AR Document Types’ function of the Accounts Receivable module. To bill an external Source with Laboratory orders.

  4. Shift 1 Start Time – Enter the start Time of shift 1 in hh:mm format. The values selected have to be within ‘0’ to ‘23’.

  5. Shift 2 Start Time – Enter the start Time of the shift 2 in hh:mm format. The values selected have to be within ‘0’ to ‘23’.

  6. Shift 3 Start Time – Enter the start Time of the shift 3 in hh:mm format. The values selected have to be within ‘0’ to ‘23’.

  7. Result Report Page length – Enter the value for the report Page length in numeric format of two digits maximum length.

  8. Auto Validate Susceptibility – Select a value from the list to specify auto validation for the susceptibility of Antibiotic/Organism. The system-defined values are ‘KB’, ‘MIC’ or ‘Manual’. To validate the results of Antibiotics.

  9. Allow Specimen Registration without Collection Date – Select if the registration for ordered request can be allowed if Specimen Collection Date/time is not entered as part of specimen collection.

  10. Allow Specimen Registration without Specimen Dispatch – Select the check box to permit specimen registration in the department without specimen dispatch.

  11. Allow Specimen Registration for Inpatient/Outpatient without Order – Select the check box, if you want to allow specimen registration for Inpatient/Outpatient without orders.

  12. Allowed Time to Modify Nosocomial Infection Details – Enter the time allowed to modify Nosocomial Infection Details in hours.

  13. Allow Specimen Registration even if Patient Discharge Bill Generated - Select this check box to allow registration of specimen for an inpatient even if the patient’s Discharge Bill is generated .  When this check box is cleared, an error message is displayed and you cannot register.

  14. Enable View Clinical Event History in Result Entry by Specimen - Select the checkbox to display ‘View Clinical Event History’ button in the Result Entry by Specimen screen. This is applicable only for MLMOH users.

  15. Previous IP Encounter allowed – Enter the number of days for inpatient encounter to allow the encounter to register the specimen for the selected encounter to the entered days.

  16. Max Period Ahead to register Tests against Original Visit [OP-AE] – Enter the number of hours ahead of the Original visit to be allowed to register a test. This is valid for Outpatient Management and Emergency Care Management.

  17. Mode – The Mode drop-down list values are ‘Auto’ and ‘Manual’. Select the required value from the list.  By default, the value ‘Manual’ is displayed.

  18. Hrs, Mts, Secs –  Enter the required time interval in the text field. The value should not be decimal or negative value. This field is enabled only when Mode is selected as ‘Auto’.

If you enter the time value wrongly, the message appears ‘Field must be of form 99’ at the bottom of the screen.  

If you enter the Hrs value as ‘24’, the message ‘Must be in the range 0 to 23’ gets displayed at the bottom of the screen.

If you enter the ‘Mts’ or ‘Secs’ value as ‘60’, the message ‘Must be in the range 0 to 59’ gets displayed at the bottom of the screen.

This field is disabled when Mode is selected as ‘Manual’.
