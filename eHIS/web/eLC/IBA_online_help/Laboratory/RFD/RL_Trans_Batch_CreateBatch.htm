Enter the following details:

  1. Department - The department name associated with the logged in user is displayed. Select the desired department name from the list.

  2. Batch Number - A unique batch number is created on clicking [Create Batch].

  3. Perform Site – Click [List of Values] to select the site where the entire specimen in the batch is going to be processed. The list displays the values defined in ‘Perform Site’ function.

  4. Workgroup – Click [List of Values] to select the Workgroup code. The Workgroups associated to the logged in user and the respective department through the ‘Authorized Users’ function get listed.

  5. Test Code – Click [List of Values] to select the Test code. The values defined through the ‘Test Codes’ function is listed.

  6. Date Range [From – To] – Click the Calendar function to select the date range. The ‘From’ and ‘To’ values are mandatory. The specimens registered during the selected period are considered during batch generation.

 

Selection Criteria

  1. Maximum Specimens In the Batch - Enter the maximum number of specimens that can be accommodated in the batch. The system defined value 50 is displayed, by default.

  2. Specimen Status – Select a value from the list. The system-defined values are ‘All’ and ‘Outstanding’. Select ‘All’ to create batch for all the specimens, select ‘Outstanding’ to create batch only for the specimens with status ‘Outstanding’.

  3. Batch – Select the batch from the list. The system-defined values are ‘ANY’ and ‘Not in any Batch’. Select ‘ANY’ to include any specimen to create the batch or select ‘Not in any Batch’ to create the batch for the specimens which are not in any batch.
