Enter the following details:

  1. Department – The logged in Department Code is displayed by default. Select desired department from the list, if you have access to more than one department. This is a mandatory field.

  2. RBC Test – Click [List of Values] to select a RBC test. The list displays the values defined in Test Codes function.

  3. Reticulocytes Test – Click [List of Values] to select Reticulocytes test. The list displays the values defined in Test Codes function. This is a mandatory field.

  4. Key for counting Reticulocytes – Enter the key character that is used to represent Reticulocytes during result entry. This is a mandatory field.

  5. Key for RBCs – Enter the key character that is used to represent RBCs during result entry. This is a mandatory field.

  6. Print Result – Select the Print Result from the list. The listed valued are ‘Absolute’, ‘Percentage’, ‘Both’. The Reticulocytes count can be reported based on absolute values, percentage or both. The list is set to ‘Absolute’ by default. This is a mandatory field.
