Enter the following details:

  1. Department - The department name associated with the logged in user is displayed. If the user has access to multiple departments, select the desired department name from Department list.

  2. Perform Site - Enter the Perform Site code or select the Perform Site using [List of Values] function. The list displays the values defined in Perform Site masters function.

  3. Include? - Select the values from the list. The listed values are 'Outstanding Test' and 'All Tests'. The value ‘All Tests’ is displayed, by default.

  4. Specimen No [From-To] – Enter the specimen number in numeric format of 15 characters maximum length or select the Specimen Number using [List of Values] function.

  5. Priority - The Priority of the order placed is displayed. The listed values are ‘All’, 'Routine', 'Stat' and 'Urgent'. The value ‘All’ is displayed, by default.

  6. Category No [From-To] - Select the Category code using [List of Values] function to display the selected category task details.

  7. Assign To - The name of the person to whom the selected specimen processing is assigned, is displayed.

  8. Workgroup Code - Select workgroup name from the [List of Values] function. The list displays the values defined in Work Group masters function.

  9. Test Code - Enter the Individual Tests code or click [List of Values] function to view a list of tests created while setting in 'Tests Codes' masters function and select the desired Test Code.

  10. Reference Laboratory - Enter the Reference Laboratory code to which the test is contracted or select the reference laboratory name using [List of Values] function. The list displays the values defined in the Referral Lab masters function. When a valid code is entered, the short name of the code is displayed for user verification.

  11. Source - Select the source from the list. The listed values are 'Ward', 'Clinic' and 'External'. Based on the selected Source, you can enter or select the values using [List of Values] function.

  12. Start and End Date - Enter the start and end date of the specimen result entry from Calendar function.
