Enter the following details:

 

  1. Order No - If the Order Number is available, enter the Order Number, for which the specimen has to be registered.

  2. Perform Site - Enter the Perform Site code or click [List of Values] to select the Perform Site. The listed values are defined in 'Perform Site' function

  3. Department - The department name associated to the logged in user is displayed. If you have access to multiple departments, select the desired department name from list.

  4. Specimen No – The Specimen Number generated for the selected patient is displayed.

  5. Priority Specimen – Select the check box to consider the specimen as high priority. This priority is not related to urgent, stat or routine priority of the specimen.

  6. Facility - The logged in facility name is displayed. If you have access to multiple facilities, select the desired Facility from the list.

  7. Episode No – Select the Episode Type from the list. The listed values are Inpatient, Outpatient, Referral and External. Enter the Episode Number on selecting the Episode Type.

  8. Visit/Tfr No - Enter the visit or transfer number for an Outpatient/Inpatient episode.

  9. Patient ID – The Patient ID is of the patient is displayed.  If the Patient is not registered in the Facility, the [External Patient Registration] screen is displayed.

  10. Sex - Select the Gender of the patient from the list. The listed values are Male, Female and Unknown.

  11. Nationality – Enter the Nationality or click [List of Values] to select the Nationality. The listed values are defined in the Master Patient Index module.

  12. Source – Select the Source from the list, from where the request originated. The source codes are Ward/Clinic/External code based on the selected episode type. The description for the source code is displayed. You can change the displayed Source code for Inpatients and Outpatients. The results are printed on the printer defined for the source.

  13. Age – The Age of the patient is displayed in the format of yy/mm/dd. The age of the patient at the time of booking request is displayed. The age is prefixed with a ‘*’, if at the time of patient registration the age was specified and not the actual date of birth.

  14. Referral No - The number assigned by the external source for the patient is displayed.

  15. Doctor - Enter the code for the doctor who has requested the tests. The description for the selected doctor code is displayed. The doctor associated to the patient is displayed and can be changed for Inpatients and Outpatients.

 

General

  1. Clinical Comments - Enter the clinical comments or click [List of Values] to select the comment codes.

  2. Spec Type - Enter the Specimen Type code. The description for the Specimen Type is displayed.

  3. Assign To - If the request has to be processed by a specific doctor, enter the doctor code or click [List of Values] to select the Doctor.

  4. Priority - The Priority of the order placed is displayed. The valid values are 'Routine', 'Stat' and 'Urgent'.

  5. Print Request Slip – Select this check box if the request slip has to be printed, else clear the check box.

 

Date Time

  1. Collection - The Collection Date and Time of the sample is displayed.

  2. Received - The Received Date and Time of the sample is displayed.

  3. Registered - The Registered Date and Time of the sample is displayed.

 

Others

  1. No. of Samples – The number of samples received for registration is displayed or enter the number of samples received for registration in alphanumeric format of two digits maximum length.

  2. Ordering Doctor Name – The name of the Ordering Doctor  is displayed or enter the Name of the Ordering Doctor in alphanumeric format of 40 characters maximum length.

  3. Copy – Enter the number of counts to print the labels.

  4. Refl Spec No – The Reflex specimen number is displayed, by default.

 

Investigations

  1. Include Suspended Test? - Select the ‘Include Suspended Test?’ check box to include the suspended test for investigation.

  2. Test – Enter the Test name or click [List of Values] to select the Test. The list displays the Test codes defined in the Test Codes master function. The description of the test is displayed in the adjacent field. At least one test must be ordered for the request. Click   to view the order comments recorded for the Test code while booking the order.  

  3. Result - Enter the result for the test if applicable (prompted result). The sign character can be entered if required. The valid characters are +, -, < and >. Based on the setup an error or warning is displayed if the result is not recorded.

  4. Process Date - Enter the process date if the test requires the process date to be entered. If the test registered is defined as an interval test, a screen gets displayed. The interval specimen numbers to be linked together for result reporting can be entered using this screen.

A patient who has a hospital number but is being referred from another source is treated as a referral patient. A patient who does not have a patient number can be registered in the Clinical Laboratory module and assigned a number to be used within the module. Such patients are treated as external patients. You can register external patients by clicking [New Patient].
