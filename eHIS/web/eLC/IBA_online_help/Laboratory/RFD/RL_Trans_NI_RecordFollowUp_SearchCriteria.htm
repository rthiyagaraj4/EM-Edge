Enter the following search criteria:

  1. Serial – Enter the serial number in alphanumeric format of 10 characters maximum length or click [List of Values] to select the serial number. The selected serial number is generated from the ‘Record Infection’ function.

  2. Onset Date [From - To] – Enter the Onset Date when the patient is admitted in the Facility or select the Onset Date using the Calendar function.

  3. Status – Select the Status from the list. The listed values are ‘Outstanding/In-process’, ‘Outstanding’, ‘In-process’, ‘Completed’ and ‘All’. You can select the status based on the requirement.

  4. Ward – Enter the Ward code in alphanumeric format of four characters maximum length or click [List of Values] to select the Ward. These values are defined in the ‘Nursing Unit’ function of ‘Inpatient & Daycare Management’ module.

  5. Specialty – Enter the Specialty code in alphanumeric format of four characters maximum length or click [List of Values] to select the Specialty. These values are defined in the ‘Specialty’ function of ‘Application Masters’ module.

  6. Patient ID – Enter the Patient ID in the length as defined in the MP Parameter function of ‘Master Patient Index’ module or right click on ‘Right Click for User Options’ and select the Patient ID using the ‘Standard Patient Lookup’ option.
