Enter the following details:

  1. Department – The logged in Department Code is displayed, by default. Select required department from the list, if you have access to more than one department.

  2. Test Code – Enter a unique Test Code in alphanumeric format of 10 characters. You cannot modify or duplicate the text code after creation. This is a mandatory field.

  3. Sequence No – Enter a valid Sequence Number of the test code in numerical value of five digits maximum length. This number is used to sequence the test codes in the result reports.

  4. Long Name – Enter the Long Name of test in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  5. Short Name – Enter the Short Name of test in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  6. Sub Department – Click [List of Values] to select the sub department. The sub departments defined in the Department masters are listed.

  7. Numeric Result – Select the ‘Numeric Result’ check box if the result of the test is in numerical format. The system automatically enables Units, Other units, Decimal Places, Rounding of Decimal and Significant Digits fields upon the selection of numeric results. Otherwise, you cannot enter numeric result values at the time of result entry.

  8. Units –Click [List of Values] to view and select the required unit. The list of values displays a list of units created through the ‘Discrete Measures’ function in ‘Application Masters’ module. The selected unit value appears in the report.

  9. Other Units – Enter the unit of measurement code or click [List of Values] function to view a list of units and select the required Other Units. The list of values displays a list of units created through the ‘Discrete Measures’ function in ‘Application Masters’ module. Entered unit value appears in the report.

  10. Decimal Places – Enter the required decimal number required for the numeric results.

  11. Rounding of Decimal – Select ‘Rounding of Decimal’ check box, if the test result requires rounding up of decimal places.

  12. Significant Digits – Enter the number of Significant Digits of the numeric test result. The system allows entering a maximum of one digit. This field gets enabled on selecting the 'Numeric Result' check box else it is disabled.
