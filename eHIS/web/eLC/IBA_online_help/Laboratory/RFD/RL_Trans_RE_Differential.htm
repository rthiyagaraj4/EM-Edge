Enter the following details:

  1. Patient ID – The unique ID assigned to the patient is displayed.

  2. Perform Site – Select the desired Perform site to process the specimen from a list of values. The list displays the values as define through the Perform Site masters function

  3. Sex –  Patient’s gender is displayed.

  4. Age – Patient age is displayed.

  5. Film Comments – Select the file comments from the [List of Values] function. The listed values are created in Administration Setup.

  6. NRBC – It will increase by one every time you click [Inrc NRBC] or you can enter manually.

  7. Retics –  Enter the count for RETICS, if required.

  8. WBC Comment – Enter the comments in alphanumeric format of 30 characters of maximum length.

  9. Cell to be Counted – The maximum number of cell to be counted.

  10. Recalculate WBC Result – Select the check to recalculate the WBC result.

  11. KB -  Enter the valid keystrokes or select it using the [List of Values] function. This Test box will get enabled on clicking the [Keyboard].
