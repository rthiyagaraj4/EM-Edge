Specimen Type

Specimen Type tab enables you to associate specimen types and volume of the specimen-required base on the age range for a group test.  This tab is enabled on selecting the ‘Specimen Type’ check box.

 

Specimen Type

  1. Type – Click [List of Values] to associate a Specimen Type to the Group Test. The values defined through the ‘Specimen’ function get listed.

  2. Description – The corresponding description is displayed, by default.

  3. Volume – Enter the required volume of the specimen in numeric format of four digits maximum length required to carry out the test.

  4. Unit – Click [List of Values] to select the unit.

 

Volume by Age Range

  1. Spec Type - The selected Specimen Type in Specimen Type tab is displayed and disabled, by default.

  2. Minimum Age – Enter the Minimum Age in years, months and days format for the selected group test and specimen type.

  3. Maximum Age – Enter the Maximum Age in years, months and days format for the selected group test and specimen type. The days specified for maximum age has to be greater than the minimum age.

  4. Volume – Enter the Volume of the specimen to be collected for the selected test in numeric format of five digits maximum length. It indicates the volume of a sample to be collected for a defined age range and for a selected specimen type.
