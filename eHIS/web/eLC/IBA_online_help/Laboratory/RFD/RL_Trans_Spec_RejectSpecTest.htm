Enter the following details:

  1. Perform Site - Enter the perform site code or select it using [List of Values] function. The list displays the values as defined in the 'Perform Site' masters.

  2. Department - The department name associated with the logged in user is displayed. If the user has access to multiple departments, select the desired department name from Department list.

  3. Specimen No - Select the desired Specimen Number from the list of values. List displays the specimen numbers belonging to the selected department.

  4. Referral No - The number assigned by the external source for the patient is displayed.

  5. Order No -The order number (if available) for which the specimen has been registered is displayed.

  6. Sex - Gender of the patient gets displayed. Listed values are 'Male', 'Female' and 'Unknown'.

  7. Episode No - Select the Episode type for the patient from the list. The valid values available are 'Outpatient', 'Inpatient', 'Referral' and 'External'. On selecting the episode type from the list, the respective reference number of the patient episode in case of the outpatient and admission number in case of inpatient is displayed.

  8. Visit/Tfr No - The Visit number associated with the Inpatient episode is displayed.

  9. Age - Age of the patient in format of Years, Months and days is displayed. The age of the patient as recorded at the time of patient registration is displayed.

  10. Patient ID - The unique identification number assigned to the patient and the name of the patient is displayed.

  11. No. of Specimen - The number of samples registered for the test are displayed.

  12. Source - The Source Type and the corresponding source code and description is displayed.

  13. Copy - Specified number of copies that you need to print is displayed, if the selected specimen is already rejected.

  14. Doctor - The name of the Doctor who ordered test is displayed.

  15. Refl Spec No - The specimen number assigned by the external source for the request is displayed.

  16. Spec Type - The specimen type code. The description for the specimen type is displayed.

  17. Category No - The Category Number associated with the sample is displayed.

  18. Clinical Comments - The clinical comments associated with the ordered investigation is displayed.

  19. Priority - The specified Priority to process the specimen or investigation is displayed. The valid values are 'Routine', 'Stat' and 'Urgent'.

  20. Col Date - The specimen Collection date (dd/mm/yyyy) and time (hh:mm) gets displayed.

  21. Recd - - Specimen receipt date gets displayed.

  22. Assign To - The name of the user for whom the specimen is assigned to perform investigation is displayed.

  23. Regd - The Registration date (dd/mm/yyyy) and time (hh:mm) of the Specimen is displayed.

  24. Reject? - Select the check box to cancel the request. If the check box is selected, the request is treated as cancelled. Clearing the check box reinstates the request.

  25. Reason - Select the cancellation code from the List of Values. The description for the cancelled code is displayed.

 

Investigations

  1. Test - The code of the test ordered is displayed.

  2. Description - The corresponding Description of the ordered test is displayed, by default.

  3. Reject? - Select the Reject check box to reject the test.

  4. Reason - Select the Reason code from the list of values. On selecting the reason code, system displays the corresponding description for the reason code.

  5. Remarks - Enter the remarks for the investigation.
