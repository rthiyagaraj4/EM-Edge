Logs Tab

Enter the following details:

  1. Request Date – The Request date and time is displayed.

  2. Facility – The name of Facility is displayed.

  3. User ID – The ID of the user who has requested the report generation is displayed.

  4. Machine Name – The Computer Name where the process is running is displayed.

  5. Workstation – The Workstation where the process is running is displayed.

  6. Program Name – The Report generation program name is displayed.

  7. Regenerate Date -  The request date and time when the process is regenerated is displayed.

  8. Click [View Error Log] to view the recorded error message information.

  9. Click [Regenerate] on selecting a request record. This deletes the corresponding error messages and makes the report generation request as active request for regeneration.

  10. Click [Purge] to delete the error log after confirmation message is acknowledged.

  11. Click [Purge All] to delete the error logs after confirmation message is acknowledged.
