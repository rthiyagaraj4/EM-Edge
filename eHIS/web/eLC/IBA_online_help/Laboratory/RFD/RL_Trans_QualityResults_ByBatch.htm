Enter the following details:

  1. Department – Select a Department from the list. The departments to which the user has access gets listed. The Department Name of the logged-in User is displayed, by default.  

  2. Quality Code – Enter the Quality Code in alphanumeric format of four characters maximum length or click [List of Values] to select the Quality Code. The values displayed are defined in ‘Quality Codes’ function in the Laboratory masters. On selecting the Quality Code, the description of the selected Quality Code is displayed.

  3. Manufacturer ID – On selecting the Quality Code, the Manufacture ID and description is displayed.

  4. Batch/Lot No. – Enter the Batch or Lot Number in alphanumeric format of 20 characters maximum length or click [List of Values] to select the Batch or Lot Number. The values displayed are defined in ‘Create Batch’ function of Laboratory Transactions.

  5. Specimen No – On selecting the Batch or Lot Number, the Specimen Number is displayed.

  6. Expiry Date – On selecting the Batch or Lot Number, the Expiry Date is displayed.

  7. Analyzer Code – Enter the Analyzer Code in alphanumeric format of four characters maximum length or click [List of Values] to select the Quality Code. The values displayed are defined in ‘Analyzers’ function in the Laboratory masters. On selecting the Analyzer Code, the description of the selected Quality Code is displayed.

Quality Results

  1. [L] – Click [L] to display the previous test results performed on the same Specimen (if any).

  2. Test Code – Click [Query] or press [F8] to display the Test Code. You cannot modify the Test Code.

  3. Result Date/Time – The current date and time of the system is displayed, by default as the Result Date in dd/mm/yyyy format and time in hh:mm format. You cannot modify the Result Date/Time.

  4. Low Value – On selecting the Quality code, the Low Value defined for the Quality Code in ‘Quality Tests’ function in the masters, is displayed. You cannot modify the Low Value.

  5. High Value – On selecting the Quality code, the High Value defined for the Quality Code in ‘Quality Tests’ function in the masters, is displayed. You cannot modify the High Value.

  6. Result Value – Enter the Result Value in alphanumeric format of five characters maximum length.

  7. Result Status – Select the Result Status as ‘Accepted’ or ‘Rejected’. If ‘Accepted’ is selected, the quality test of the Specimen is accepted. If ‘Rejected’ is selected, the quality test performed on the Specimen is rejected.

  8. Remarks – Enter the remarks in alphanumeric format of 150 characters maximum length.
