Enter the following details:

  1. Task Assigned to - Select the values from the list. The listed values are 'Specimen', 'Sample' and 'Block'.

  2. Test Code - Click [List of Values] to select a Test Code and the respective description created through Test Codes function gets displayed. 'All' is displayed on selecting 'Task Assigned To', by default.

  3. Anatomy Site – Click [List of Values] to select an Anatomy Site. It indicates the Anatomical site of the specimen from where the sample was taken.  The list displays the Anatomy Sites associated with the selected test code. '*All' is displayed on selecting 'Task Assigned to', by default.

  4. Task Code – Click [List of Values] to select a Task Code. The list displays the values defined in the ‘Individual Tasks’ function.

  5. Description – The corresponding description for the selected 'Task' code is displayed, by default.
