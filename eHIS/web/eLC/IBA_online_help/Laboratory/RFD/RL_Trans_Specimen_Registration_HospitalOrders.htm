Hospital Orders

Enter the following details:

  1. Priority – Select the priority from the list. The listed values are ‘All’, ‘Stat’, ‘Urgent’ and ‘Routine’.

  2. Press [F10] to view the order details. The test details are displayed in the ‘Investigations’ tab if the test details are recorded.

  3. Order ID – The Order identification number of the patient is displayed.

  4. Order Date – The order date and time is displayed.

  5. Specimen No – The specimen number is displayed.

  6. Patient ID – The Patient identification number of the patient is displayed

  7. Facility ID – The facility identification number is displayed.

  8. Episode Type – The patient episode type is displayed.

  9. Episode ID – The patient episode identification number is displayed.

  10. Doctor – The name of the patient is displayed.

  11. Source Type – The source type of the patient is displayed.

  12. Source Code – The source code of the patient is displayed.

  13. Perform Site Code – The perform site code of the patient is displayed.
