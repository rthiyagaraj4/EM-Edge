Enter the following details:

  1. Department - The department name associated with the logged in user is displayed. If the user has access to multiple departments, select the desired department name from Department list.

  2. Specimen Number - Scan the Specimen Number through the Barcode Scanner if scanner is interfaced.

Click 'Double Click For User Option' link to select 'Hospital Patient Lookup' and 'External Patients Lookup' links are displayed. Select any of the options to select the Specimen Number.

 

Barcode Scanning

  1. NRIC ID - Enter a valid NRIC ID number to search for the record.

  2. Patient ID - Enter a Patient ID to search for the result from a referral lab.

  3. Due Date [From-To] - Enter the due date or select the due date from the calendar function.

  4. Reference Laboratory - Specify the Referred Lab code to which the specimen has been sent for investigation.

  5. All/Marked as Received/Not received – Select the result status from the list. The listed values are ‘All’, ‘Marked as Received’ and ‘Not Received’.

  6. Send e-mail - Click [Send e-Mail] to send email to concern department.

  7. Specimen No - Specimen numbers that are to be received from referral labs are displayed in this field or one can enter the specimen number as one of the query criteria.

  8. Patient ID - The Patient ID to which the specimen belongs to is displayed.

  9. Patient Name - The Name of the Patient as recorded during registration is displayed.

  10. NRIC/Alternate ID - The NRIC/Alternate ID is displayed, by default.

  11. Due Date - The due date of the specimen is displayed, by default.

  12. Test - The test name, which is referred, is displayed.

  13. Reference Laboratory - The name of the reference laboratory to which the specimen has to be send is displayed.

  14. Date Sent - Date on which the specimen has been sent to referral lab are displayed.

  15. Date Recd - Select Receive date from the calendar function on which the date for the result of the specimen is received from referral lab.
