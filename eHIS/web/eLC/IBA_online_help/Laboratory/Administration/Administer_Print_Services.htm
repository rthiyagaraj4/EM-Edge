Administer Print Services

From this function you can Start or Stop the predefined Enterprise Management Server Processes.

The process with multiple instances can be started in selected facility or for all facilities. Each process can be started as a single or multiple instance depending on the design of the process.

Administer Print Services screen consists of two tabs namely ‘Process’ and ‘Logs’. Process tab allows you to start or stop the server processes. Logs tab allows you to view the process log details and can regenerate the report request and purge the error logs.

Administering Print Services

From Administration > Print Services, select the Administer Print Services function.

The Administer Print Services screen gets displayed.

  1. Enter the details in the available fields in the following tabs:

Process

Logs

  1. Click [Execute Query] or press [F8] to view the records.
