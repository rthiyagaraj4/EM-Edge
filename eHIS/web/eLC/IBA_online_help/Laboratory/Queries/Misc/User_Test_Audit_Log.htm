User/Test Audit Log

From this function you can Query the User/Test Audit Log information.

User/Test Audit log is used to query or view the detail tests or users audit information for the selected date range, User Name, Test Type and Test, the function  lists out all the details.

Querying User/Test Audit Log

From Query > Miscellaneous, double-click the User/Test Audit Log function.

The User/Test Audit Log screen gets displayed.

  1. Enter the details the relevant fields. <Related Field Description>

  2. Click [Fetch] to view the specimen details satisfying the search criteria.

  3. Click [Clear] to clear the entered details.

  4. Click [Who Done It] to view, print or file the report.
