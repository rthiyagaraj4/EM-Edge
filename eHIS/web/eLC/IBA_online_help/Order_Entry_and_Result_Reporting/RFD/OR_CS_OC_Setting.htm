Settings

Applicability of the Order Catalog to various patient classes and frequency settings are specified in the ‘Setting’ tab.

  1. Patient Class - The Patient Classes applicable are displayed. These classes are system defined. The characteristics are defined for each class, which override the values set for 'All'.

  2. Applicability - Select the value as either 'Applicable' or 'Not Applicable'. If applicable is selected for a patient class, the Order Catalog is available to the selected patient class and frequency settings are defined.

  3. Age Group - Select the Age Group to which the Order Catalog is applicable. The values are displayed from Application Masters module.

  4. Change Qty/Dose – The value is displayed as One, by default. The functioning of the field depends on the value specified for 'Change Qty/dose' under 'Timing' sub tab. If the check box is selected under the tab Timing, you can change the quantity/dose. You have to enter the quantity/dose in numeric format of three digits maximum length. If the check box is cleared, 'one' is set as the default value and the field is disabled.

  5. Frequency - Select the Frequency from the list. This field is mandatory, if 'Frequency Required' is applicable defined under 'Timing' sub-tab. The Frequency Required check box has to be selected while defining the Order Type. If the frequency is applicable for the selected Order Type and the check box is selected during definition, the ‘Frequency Required’ check box is enabled under the Timing sub-tab. On selecting the check box, the frequency is set as mandatory in the Frequency tab. The frequencies selected under ‘Frequency Applicable’ sub-tab get listed.

  6. Durn Value/Type – Enter the duration value in numeric format of three digits maximum length. The type displayed corresponds to the nature defined for the selected frequency in the Application Masters module. You cannot update the type.

  7. Max Value/Durn Type - Enter the maximum value in numeric format of three digits maximum length. On entering the maximum value, duration type is automatically displayed corresponding to the specified Durn Value/Type. The maximum value must be greater than the Duration Value.

  8. Action - Select the value from the list. The values displayed are Reject, Warn and Request for Authorize. The field is enabled for selection only when the maximum value/duration type is specified. On selecting the value, appropriate action is performed, if maximum value is violated.

  9. Msg - Enter the message to be displayed for the specified action. On clicking the 'Msg' hyperlink, the screen gets displayed. Click [OK] to save the details and to exit from the screen. This field is mandatory for any Action selected.
