Enter the following details:

 

  1. Ordering Facility - Select the Ordering Facility from the list. The facilities of the enterprise are listed. This is a mandatory field.

  2. Source Type - Select the Source Type from the list. The listed values are Nursing Unit and Clinic. This is a mandatory field.

  3. Source Location - Select the Location from where the order is being placed. The values in the list correspond to the Source Type selected. Clinics and Nursing Units that are created for the ordering facility are displayed. This is a mandatory field.

  4. Order Category - Select the Order Category from the list. The list values are system defined. The Order Categories listed are Laboratory, Radiology, Dept Diagnostics and Trmt/Procedures. The system allows only categories mentioned above for Multi Facility processing. This is a mandatory field.
