Enter the following details:

 

  1. Type - Select the Type from the list. The listed values are ‘Consent Form’, ‘Patient Instructions for Interventions’, ‘Refusal Form’, ‘Referral Form’ and ‘NPB Request Form’.

  2. ID - Enter the ID in alphanumeric format of 10 characters maximum length. This field is mandatory.

  3. Description - Enter Description in alphanumeric format of 30 characters maximum length. Description is a brief explanation given to the Note text that helps in understanding the name and purpose of the note text. This field is mandatory.

  4. Text Type - The values are system defined - ‘Editor' and ‘Plain’. The applicability of the notes is decided by the value selected in the 'Type' field. Select the Text Type using [E] function if the Type is selected as Consent Form and Refusal Form. An editor is available to enter content for the Consent/Refusal/Referral form. Click the hyperlinks available to display the required details on the editor frame. Click to insert table in the editor frame. Click to preview the Consent/Refusal/Referral form. Click  to save the changes.

  5. Text – Enter the required Text if the Type is selected as Referral Form.

  6. Index – This field is displayed if the Type selected is Consent Form. Select the check box to associate Consent Notes to a service.

  7. Enabled - Select the check box to enable the record in the transaction functions. The check box is selected, by default.
