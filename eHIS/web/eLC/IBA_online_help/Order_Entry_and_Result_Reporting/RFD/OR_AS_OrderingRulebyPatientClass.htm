Enter the following details:

 

  1. Patient Class – The system defined Patient Classes are displayed. The displayed values are 'Daycare', 'Emergency', 'External', 'Inpatient' and 'Outpatient'.

  2. Pre-select Order Type – This check box is enabled if 'Order Type by Class' is applicable in OR Parameter.  Select the check box to select Order Type for the Patient Class while placing an Order. If Pre-select Order Type is selected, it is mandatory to select order type for the patient class while placing an order.

  3. Time Based Rule - Select the required time based rule from the list. The listed values are 'Day After discharge', 'Day After Adm/visit', 'Hour After Discharge', 'Hour after Adm/Visit', 'Day After Discharge Advice' and 'Hour After Discharge Advice'. You can place orders after discharge or on a visit.

  4. Expiry Period - Enter the Expiry Period for the selected rule in numeric format of three digits maximum length. This ia a mandatory field.

  5. Future Order Rule - Enter the period in numeric format of three digits maximum length for which the order has to be considered as a Future Order. Select ‘Minutes’, ‘Hours’, ‘Days’, ‘Weeks’, ‘Months’ and ‘Year’ from the list. This is a mandatory field.

  6. Round Off Start Time - Select the check box to consider the start time for the future order from 00:00 or the system considers the start time for future order from the time the order is placed.
