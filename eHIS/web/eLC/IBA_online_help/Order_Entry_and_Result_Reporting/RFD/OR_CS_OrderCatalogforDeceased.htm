Enter the following details:

 

  1. Order Category – Select an Order Category from the listed values. The order categories that are defined in the OR Order Category function gets listed. This is a mandatory field.

  2. Order Type – Select an Order Type from the listed values. The order categories that are defined in the OR Order Type function gets listed.

  3. Order Catalog – Select an Order Catalog from the list using this [?] function.

  4. Post Mortem Applicability – Select the post mortem applicability value from the drop-down list. The listed values are ‘All’, ‘Embalm Registration’, ‘Post Mortem Request’, and ‘Post Mortem Record’. This is a mandatory field.

  5. Post Mortem Type – Select the Post Mortem Type from the list. The listed values are ‘Clinical Post Mortem’, ‘Medico Legal Post Mortem’, ‘Both’ and ‘Not Applicable’. This is a mandatory field. When the Post Mortem Applicability  is selected as ‘Embalm Registration’, the value ‘Not Applicable’ has to be selected in the Post Mortem Type field.

  6. Enable – Select the check box to enable the record for the deceased patients. This check box is selected, by default.

  7. Select row to Remove – Select the check box against the particular record and click ‘-‘ to remove  the selected row.

Click ‘+’ button to add a new row.
