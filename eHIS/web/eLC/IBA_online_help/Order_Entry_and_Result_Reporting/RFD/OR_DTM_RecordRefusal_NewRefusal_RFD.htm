Enter the following details:

 

  1. Encounter ID - Enter the Encounter ID in numeric format of 12 digits maximum length or use [?] function to select the required Encounter ID.

  2. Order ID - Enter Order ID in the required format or use [?] function to select the required order. You can select only one Order.

  3. Refusal Type - If Order ID is selected, the Orderable is displayed and the field is disabled or select either Orderable or Non-Orderable from the list.

  4. Orderable/Non-Orderable - On selecting Order ID, the Orderable is displayed and the field is disabled. If Order ID is not selected, select the required Orderable using [?] function. If Non-orderable is selected, specify the Non-Orderable description. This field is mandatory.

  5. Reason - Enter the reason for refusal. This field is mandatory.

  6. Refusal Date/Time - Current system date and time is displayed, by default. This field is mandatory. Enter the required date and time.

  7. Practitioner - The logged in user is displayed, by default. Select the required practitioner using [?] function.

  8. Refusal ID - Select the Refusal ID and the Refusal Editor screen is displayed to view and edit refusal details. Click Insert Table to add tables to the refusal form. Click Preview to preview the refusal form. Click [Close] to exit from the Editor Preview screen or [Print] to print the refusal form. Click OK to save the refusal form.

  9. A Refusal Text hyperlink is displayed next to the field. Click the hyperlink to view or modify details of the refusal form.   

  10. Click [Record] to record the refusal form and the Existing Refusal tab is displayed or click [Clear] to clear the selected data and display the Record Consent screen.
