Discharge Medication screen

Enter the following details:

  1. Select – Select the check box for the medication you want to give to the patient during discharge.

  2. Drug Name – Name of the drug is displayed. By default all the drugs will be selected. The duplicate drugs or drugs which are already ordered for the patient or any other drugs with same generic are indicated in blue colour hyperlink. The doctor can de-select the drugs by clicking the duplicate drug name hyperlink and the existing Discontinue/Cancel Medication screen is launched. The doctor can discontinue or cancel the drugs and the duplicate drugs will be removed from the Discharge Medication screen.

  3. Dosage Detail – The dosage details of the drug is displayed.

  4. Order Date Time – Displays the date/time when the order was placed and the Rx.

  5. Ordered Location – Displays the location at which the order was placed.

  6. Duration – Select the duration  for all the selected drugs. The Maximum duration for each of the selected drugs will be validated as per the current functionality within the Pharmacy Orders screen.

  7. Start Date - The Start Date and Time will be defaulted with the Expected Discharge Date and Time. But the Doctor can modify the Start/End Date and Time. If the Start Date and End Date are modified, the value of the Duration field will be calculated automatically.

  8. Override Reason - If duplicate drug(s) exist, the doctor needs to capture the override reason mandatorily.

  9. Expected Discharge Date - The Expected Discharge Date will be defaulted with the Discharge date captured while preparing the discharge advice. The Doctor can modify the duration and override reason within the Pharmacy Orders screen.

  10. Ok - After entering the Duration and capturing the override reason, click [OK] and all the drugs will be copied to the Pharmacy Orders screen.

  11. Cancel – Cancels the discharge medication.

The existing validation messages for duplicate drugs will not appear if the drugs are copied. If the Override Reason field is left blank within the new Discharge Medication screen, the following validation message is displayed when OK is clicked:

 

“Override Reason cannot be left blank”
