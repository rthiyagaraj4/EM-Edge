Timing

Some of the values under ‘Timing’ sub-tab are inherited from the selected Order Type. Indexing of an Order Catalog and consent requirement specifications are defined.

  1. Change Qty/Dose - Select the check box, if Quantity/Dose can be changed. Clear the check box and you cannot change the Quantity/Dose. The check box is cleared, by default. On selecting the check box the UOM field gets displayed.

  2. UOM – Click [?] function to select the Unit Of Measure. The Unit of Measures defined in the Application Masters module get listed.

  3. Frequency Required - If the check box is selected, the frequency is mandatory and has to be entered in the 'Settings' tab. If the check box is cleared, frequency value is not mandatory. The field is not applicable for Order Category, Department Diagnostics and Miscellaneous. The check box is enabled, if frequency applicable check box is selected for the selected Order Type.  

  4. Soft Stop for Inpatients - The check box is enabled depending on the Order Category and Order Type selected. If Soft Stop for Inpatient is selected, the duration value is optional. This is applicable only for inpatients. The Soft Stop is applicable only for Order Categories - Dietary, Monitoring, Patient Care and Trmt/Procedure. The Soft Stop for Inpatient check box has to be selected in the Order Type function.

  5. Create as Continuing Order - The value of the field depends on the Order Category and Order Type selected. The values for selected Order Type is inherited. Modification of the value depends on the selected Order Category. If continuous order is applicable for the selected Order Type, a single order is placed for the Order Catalog and it repeats every day. The value for ‘Continuous Frequency Action’ is set to ‘Not Applicable’, by default, if the check box ‘Create as Continuing Order’ is cleared.  If this field is applicable and frequency defined is other than 'Continuous' frequency type, the recurring orders will not be generated on ordering. The field is applicable, if respective fields are selected in order type master function for order category Dietary, Patient Care, Trmt/Procedures and Pharmacy order.

  6. Continuous Frequency Action - If the check box ‘Create as Continuing Order’ is selected, the value for this field is set as ‘explode by Day’. The values displayed are system defined. If the check box ‘Create as Continuing Order’ is cleared, the value is set as  'Not Applicable', by default. The value 'Explode by Day' indicates that the orders for the defined Order Catalog are generated once while ordering and subsequent order for the next day is generated by the system at mid night.

  7. Priority Applicable – Select a value from the list to indicate the priority. The values listed are 'Routine & Urgent', 'Routine' and 'Stat'.
