New Orders

Click New Orders tab to place new orders for the selected patient. Orders are placed using any of the following tabs:

Additional Search

Using the Additional Search tab, you can select Order Catalogs across Categories based on the Parameter Setup. Specify Order Category/Order Type/Order Catalog/Service as criteria and execute the query. You can select the required Catalogs from the query result. On selecting the required Order Catalogs, option to preview and execute Orderables is provided.

‘Addl. Search’ is displayed on selecting ‘New Orders’, by default. If the logged in user and attending practitioner are same, orders can be placed directly using the tab. If practitioner type of the logged in user is ‘Nurse’, the ordering has to be performed on behalf of the ordering practitioner. This concept is explained later.

The system validates these conditions while searching for order catalogs.

In ‘OR Parameter’, if ‘Pre-select Order Category’ is selected, the selection of order category is mandatory.

It then checks, if ‘Order Type by Class’ is selected. If order type by class is applicable, the rules defined in the function ‘Ordering Rules by Patient Class’ are verified.

If Order Category is mandatory, the selected Order Category and type are made default across the other tabs (tick sheet and order set). The selected Order Category is made default under ‘Image’ tab, only if ‘Radiology’ or ‘Department diagnostics’ and ‘Trmt Procedure’ are selected.

  1. Enter the details in mandatory fields.

  2. Click [Search] to display Orders grouped under Order Set, Panel and Individual.

Expiry rules defined for the patient class in the function ‘Ordering Rules by Class’.

Based on the practitioner type, description of the Order Catalogs is displayed. For Physician, Surgeon, Dentist and Psychiatrist direct care synonym is displayed and ancillary department synonym is displayed for radiologist and pathologist. When the Nurse logs in, the short description of the order catalog is displayed. Order catalogs defined as ‘Department only’ under charting tab in order catalog master function appears only for radiologists and pathologists. Order Catalogs applicable to the patient class are listed. Only enabled Order Catalog is displayed.

  1. Select the required Order Set/Panel/Individual Order Catalog. On selecting the check box, prompt message appears if prompt text is entered for the catalog (‘Prompt Alert’ is applicable and prompt text entered in the Order Catalog function).

  2. Click [Preview] to preview the selected orders.

  3. Click [Clear] to clear the displayed Orders.

  4. Click Place Order.

The order catalogs are displayed vertically if the ‘Order Search Results Display in Vertical’ check box is selected in the OR Parameter screen.

Tick Sheet

You can select Order Catalogs through tick sheets. Order Category/Order Type/Tick Sheet is/are specified as search criteria and executed. Order Catalogs associated with the tick sheet are selected for placing order. Non-specific tick sheets can also be selected. Based on practitioner/location/service, tick sheets are available for selection. Tick Sheets not associated to practitioner/location/service (Index check box cleared in the function ‘Tick Sheet’) are available to all the practitioners of the logged in facility. On selecting the required Order Catalogs, option to preview and execute the orderables is provided.

On selecting ‘Tick Sheet’ tab. You can select the required catalogs from the tick sheets. Tick sheets can be specific to order categories. On selecting the order category, the tick sheets associated with it are listed. You can select the required tick sheet. On selecting the required tick sheet, order catalogs under them are displayed for selection. You can select the required catalogs. Based on the setup, non-specific tick sheets (tick sheets not belonging to any category) are available for selection.

  1. Enter the details in mandatory fields.

The tick sheets are displayed based on the following conditions:

By default, all the tick sheets belonging to the selected order category are displayed. If the tick sheets are indexed and associated to practitioner, primary service of the location and ordering Location in the index tick sheet function, the associated practitioner, Primary service of the Location and Ordering Location can access the tick sheet.

If order category is not mandatory, on clicking the ‘Tick Sheet’ tab, tick sheets associated to the ordering practitioner, services of the ordering location (Index tick sheet function) and nonspecific tick sheets are displayed, by default. You can select the required tick sheet.

  1. Select the required Order Catalog. On selecting the check box, prompt message appears if prompt text is entered for the catalog (‘Prompt Alert’ is applicable and prompt text entered in the Order Catalog function).

  2. Click [Preview] to preview the selected orders.

  3. Click [Clear] to clear the displayed Orders.

  4. Click Place Order.

The order catalogs are displayed vertically if the ‘Order Search Results Display in Vertical’ check box is selected in the OR Parameter screen.

Order Sets

Select the required order set for ordering under Order set tab. You can select Order Sets belonging to all/particular diagnosis. Order sets associated to practitioner/location/service are available for selection. On selecting an order set, the order catalogs can be viewed based on the order execution time.

Image

To place imaging orders, select Image tab. Only Radiology, Dep Diagnostics and Order Category can be selected. On specifying order category/order type, image hotspots appear. Select the required hotspot. Orderables linked to the hotspot are displayed. Choose the required orderables and click ‘Place Order’ to execute the Order.

Place Order - To Place Medication Orders. Click Place Order Tab directly.

On selecting the required Order Catalogs, you can place Orders thorough this tab. Required details are entered while ordering. To place Medication Orders, you can select this function directly.

The sequence in which these two tabs are displayed is based on the setup done in the ‘Orders Tab’ master function of OR code setup in the ‘Order Entry and Result Reporting’ module for the logged in practitioner and the function from where these tabs are invoked. If there is no setup specific to the logged in practitioner or from the function from where it is invoked, the display of the tabs is based on the default setup.

The total charge for respective service, patient charge, approval required and value recorded during Billing Service definition are listed. Click View/Edit hyperlink to edit the charges or override the service definition.

 

If the patient is insured and holds ARAMCO policy, the hyperlink Screening Services will be displayed on this screen.

 

The screen also displays the following functions.

  • Drug Profile

  • Allergies

  • Active Problems

  • Demographics

Click to record Order Format details. The Order Format screen displays fields based on the Order Format Setup done in Request and Results module. For Laboratory orders, an Other check box is displayed. Select the 'Others' check box to enter the clinical comments in alphanumeric format of 40 characters maximum length. On selecting the 'Others' check box, the Clinical Comments field is disabled. Clear the 'Others' check box to select the clinical comments using the [?] function.

A visual and auditory warning is displayed when a duplicate order is created. You can override the warning message by entering a justification.

 

On placing the Order, the message “Do you want to Record Consent” appears if the ‘Record Consent Form With Order’ check box is selected in OR Parameter screen. Click [OK], the Consent Order dialog with all the consent requests generated for that patient gets displayed. Select the required order and record the consent.

 

On placing the order, the discount screen or the bill settlement screen is displayed to settle the amount, if the 'Invoke Settlement Screen For Patient Class' option is selected for the respective patient class in the 'OR Parameter For Facility' function.

 

You can continue or cancel an order from the Place Order screen, if the order is duplicate depending on the 'Duplicate Override Reason Required’ check box and  setup made for ‘Type’ under ‘Duplicate Check’ tab in ‘Order Catalog’ function in 'Order Entry and Result Reporting' module. If a duplicate order is placed, ‘Duplicate Check’ screen is displayed listing the Order Catalog. Enter the remarks in the ‘Dup Override Reason’ text box and click [Continue] to place the duplicate order.  Or, you can capture the override reason for each order entry at line level by clicking the ‘Dup Override Reason’ hyperlink against each ‘Duplicate order’. Select the check box ‘Discard’ and click [Continue] to cancel the order.
