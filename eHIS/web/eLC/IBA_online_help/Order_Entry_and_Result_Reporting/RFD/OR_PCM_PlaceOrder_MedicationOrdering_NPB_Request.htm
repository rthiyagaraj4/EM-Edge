NPB Request

Select this tab to place a new Named-Patient-Basis (NPB) request or view the existing NPB requests. All the NPB requests are patient and encounter specific.

 

Placing a New Request

 

  1. Select the New Request option to create a new request.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Confirm] to confirm the new request. A unique NPB Form ID is generated on confirmation.

 

Viewing an Existing Request

 

  1. Select the Existing Request option to view the status of the existing request.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Search]. The NPB requests are displayed based on the search criteria.

  4. Click ‘NPB Form’ hyperlink to open the request form. Enter the details of the request form. This is applicable only for the requests with the status as ‘Requested’.

  5. To cancel a request, select the Cancel check box and select the Reason For Cancellation from the drop-down list. Click [Apply] to cancel the request.

  6. Click ‘Other Details’ hyperlink to view the requested, cancelled, acknowledged, approved and rejected by details in the ‘Request Details’ dialog.

Only the NPB requests with the status ‘Requested’ can be cancelled. The ‘Cancel’ check box and ‘Reason for Cancellation’ drop-down list are disabled for requests with status other than ‘Requested’.

Dispensing of alternate drug/form/strength for NPB drug is not allowed.
