The following details are displayed:

  1. Status – The status of the blood unit is displayed with the color indicator. The colour indicator is displayed for Reserved Not Requested. The colour indicator  is displayed for Reserved already Requested. A [+] icon is displayed to view the request details. Click [+] to view the Request Person and Requested date.

  2. Unit No – The unit number of the issued blood unit is displayed. The following color indicators are displayed for the units.

- Autologous Units

- Directed Units

- Units Issued in Emergency

No Color – Normal Issues

  1. Specimen No – The specimen number of the issued blood unit is displayed.

  2. Reserved Date – The reserved date of the issued blood unit is displayed. The Reservation of the blood unit is done in Allocate/Xmatch/Issue - Blood Units function in Blood Transfusion Services module.

  3. Specimen Registered Date – The specimen registered date of the issued blood unit is displayed.

  4. Product - The product code of the issued blood unit is displayed.

  5. Blood Group – The blood group of the issued blood unit is displayed.

  6. X-Match Status – The cross match status of the issued blood unit is displayed.

  7. Rhesus – The rhesus factor of the issued blood unit is displayed.

  8. Transfuse Expect Date – The expected transfusion date of the issued blood unit is displayed. The expected transfusion date is calculated in Allocate/Xmatch/Issue - Blood Units function in Blood Transfusion Services module.

  9. Sel – Select the check box to request or reprint the collection slip.
