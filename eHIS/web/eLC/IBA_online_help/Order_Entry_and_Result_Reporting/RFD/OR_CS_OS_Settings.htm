Settings

Applicability of the defined Order Set to various patient classes is set under the tab. The screen gets displayed on clicking the Settings tab.

  1. Patient Class - The Patient Classes are system defined. The value set for 'All' patient class is applicable to all the classes. You can define value specific to a patient class which overrides the value set for 'All'.

  2. Applicability - Select the value as either 'Applicable' or 'Not Applicable'. Select Applicable and the Order Set is applicable for the Patient Class. Select 'Not Applicable' and the Order Set is not applicable to the Patient Class.

  3. Age Group – Select the required age group value applicable for that patient class from the drop-down list. The age group enabled in Application Masters module is displayed in the list.
