OR Printing Rules

From this function you can Define printing rules for Order Type and Patient Class combination. You can Query existing printing rule records and Delete printing rule records.

Printing Locations for Consent Forms, Order Forms and Reports are specified based on the order priority.

Defining OR Printing Rules

From the OR Admin Setup, select the OR Printing Rules function.

The OR Printing Rules screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Printing Rule defined or [Reset] to clear the entered data.

  4. Click [Query] to search for existing records.

  5. Click [Delete] to delete the selected record.
