Ordering Rule by Patient Class

From this function you can Define Ordering rules based on Patient Class.

The time based rules and expiry period are defined for Order Category and Patient Classes. These rules are validated in the 'Order' menu in the 'Clinician Access' module.

Defining Ordering Rule by Patient Class

From the OR Admin Setup, select the Ordering Rule by Patient Class function.

The Ordering Rule by Patient Class screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the required Order Category from the list. The Order Categories defined in the ‘Order Category’ function are displayed. This field is mandatory.

  3. On selecting the Order Category, the Patient Class details are displayed.

  4. Enter the details in the mandatory (*) fields. <Related Field Description>

  5. Click [Apply] to record the defined Ordering rules or [Reset] to clear the entered data.

  6. Click [Query] to search for existing Order Rules.

 

All orders of the selected category related to 'Inpatients' can be placed within three days after discharge of the Patient.
