Configure Specimen Label and Number

From this function, you can Configure various Specimen Label Formats and Specimen Number Formats specific to the Facility and Order Types.

Configuring Specimen Label and Number Formats

From the OR Admin Setup, select Configure Specimen Label and Number.

The Configure Specimen Label and Number screen gets displayed.

  1. Select the Facility from the list to associate to the Specimen Number Format. This is a mandatory field.

  2. Select the Specimen No Format from the list of values. The listed values are defined in the OR Specimen Number Format function. This is a mandatory field.

  3. Select the Applicable Order Type by clicking the radio button against All or Specific. If Specific is selected Order Types defined for the Order Category 'Laboratory' are displayed.

  4. Select the check box against the required Order Types to associate to the Specimen Number Format selected.

  5. Click [Apply] to configure Specimen Label and Number formats. Click [Reset] to reset the entered details.

  6. Click [Query] to search for existing records.
