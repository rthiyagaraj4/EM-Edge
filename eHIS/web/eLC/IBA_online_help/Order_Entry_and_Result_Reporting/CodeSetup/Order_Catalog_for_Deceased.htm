Order Catalog for Deceased

From this function, you can Create order catalogs for deceased patients to render services in the Mortuary. You can also Query for the existing records.

Order Catalog for Deceased function is used to create order catalogs which are applicable for deceased patients. This setup is mandatory to place orders for deceased patients. Postmortem Applicability and Postmortem Type is selected to which the order catalog is applicable in Mortuary.

Creating Order Catalog for Deceased

From OR Code Setup > OR Order Setup, select Order Catalog for Deceased function.

The Order Catalog for Deceased screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save an Activity Type created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Activity Type record.
