Performing Locations

From this function you can Create locations additional to the default locations, where an order can be processed. You can Query for the existing Performing Location.

Performing Location is used to define additional locations where an order is processed. The orders are routed to a particular Location based on the ‘Order Category’. For instance, a lab order placed is usually processed in the laboratory, but if it has to be processed in a location other than the laboratory, like in a Cath lab or a Satellite lab, the additional lab locations have to be defined using this function. This location can either be a location external to the enterprise or internal to the facility. Using this function.

Creating a Performing Location

From the OR Code Setup, select the Performing Location function.

The Performing Location screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a created performing location or [Reset] to clear the entered data.

  4. Click [Query] to search for existing performing location.

 

If Laboratory and Radiology Information Systems modules are operational, the ‘Radiology Wing’ and ‘Laboratory Site’ is not listed for Location Type. At a time, you can define and associate 10 Locations to a Location Type.
