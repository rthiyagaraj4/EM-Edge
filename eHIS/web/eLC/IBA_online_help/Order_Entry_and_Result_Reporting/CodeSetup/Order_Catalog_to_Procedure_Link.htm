Order Catalog to Procedure Link

From this function you can Associate an Order Catalog to Procedures and Query for the existing records.

The function Order Catalog to Procedure Link is used to associate an Order Catalog to a Procedure. The Order Catalogs for which the Procedure Link is selected as Manual Link under Charting tab during definition get listed during selection of Order Catalog. The Procedures that are linked to the Order Catalog gets listed for Result Entry in the Result Reporting transaction function. The term set and term code are defined in Medical Records module. You can link 10 Term code and Term set to the Order Catalog in a single action.

Associating an Order Catalog to Procedure

From the OR Code Setup menu, select the Order Catalog to Procedure Link function.

The Order Catalog to Procedure Link screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the record or [Reset] to clear the entered data.

  4. Click [Delete] to delete a queried record.

You can link 10 Term Codes to Term Set in one action.
