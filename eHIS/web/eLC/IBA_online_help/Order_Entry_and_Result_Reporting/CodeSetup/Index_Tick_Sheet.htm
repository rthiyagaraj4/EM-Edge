Index Tick Sheet

From this function you can Associate indexed tick sheet to practitioner, source or service and Query for the existing indexed tick sheet.

The function Index Tick Sheet is used to associate the indexed tick sheets to the Practitioner, Source or Service. The associated practitioner, source or service can only access the Tick Sheet. Tick sheets are indexed during creation by selecting the ‘Index’ check box in the function ‘Tick Sheet’. The function contains two tabs, namely, Select and Associate. Select tab is selected, by default. Using the Associate tab you can view Practitioner/Location/Service associated previously.

Associating an Indexed Tick Sheet to Practitioner/Location/Service

From the OR Code Setup, select the Index Tick Sheet function.

The Index Tick Sheet screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. If you want to Search for a required practitioner, service or location, enter the required search criteria and click [Search]. The names matching the search criteria appear. Else, you can click the required alphabet displayed as a hyperlink. The Service/Practitioner names beginning with the selected alphabet get displayed.  

  4. Select the check box against the Service/Practitioner to associate it to the Tick Sheet. Select a Facility name from the list to associate the Facility to the Tick Sheet and the selected Order Category .

  5. Click [Apply] to associate an indexed Tick Sheet to Practitioner/Location/Service or [Reset] to clear the entered data.

  6. Click [Query] to search for the existing records.
