Image Hotspot

From this function you can Create an Image Hotspot. You can also Delete an existing Image Hotspot.   

For ease of ordering, parts of the body that are most commonly subjected to radiological examination are identified as hotspots. The Image Hotspot function enables you to identify and record a name for the Hotspots.  The hotspots are created for Order Categories - Radiology and Department Diagnostics.

Creating an Image Hotspot

From the OR Code Setup menu, select the Image Hotspot function.

The Image Hotspot screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Image from the list. The listed values are Radiology Image Front, Radiology Image Back, Dept Diagnostics Image Front and Dept Diagnostics Image Back.

  3. On selecting, an Image is displayed to select a particular body part. Click the required body part.

  4. Enter the ID in numeric format of six characters maximum length. This field is mandatory.

  5. Enter the Name for the hotspot in alphanumeric format of 30 characters maximum length. This field is mandatory.

  6. Click [Apply] to save a Image Hotspot created or [Reset] to clear the entered data.

  7. Click [Delete] to delete a selected record.
