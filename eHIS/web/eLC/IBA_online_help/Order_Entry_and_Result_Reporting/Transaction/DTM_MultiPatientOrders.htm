Multi Patient Orders

From this function you can Place a common order for a group of patients having active encounters.

This helps save time spend in placing individual orders for each patient. This function is used when a set of patients requiring a common order are present in a single care location.

Placing Multi Patient Orders

From the CA Transactions > Clinician Access > Desktop Menu > Order Management, select Multi Patient Orders function.

The Multi Patient Orders screen gets displayed.

  1. Enter the Location Type as Clinic, Daycare, Emergency, Nursing Unit or Procedure Unit. Select the Location based on the selected Location Type. Click [Search] to display patients in the selected Location.

Or

  1. Select the Specialty using [?] function and click [Search] to display patients in the selected Specialty.

  2. Select the Patient ID to place order for a particular patient.

  3. The details displayed on clicking [Search] are Patient Details, Visit Admission Date and Practitioner.

  4. Click Existing Orders hyperlink against the required patient to view existing patient orders.

  5. Select the check boxes under ‘Select’ to select the patients for placing new order. To select all the patients, select the check box next to ‘Select’ heading.

  6. Click [Select] to place order.

  7. The duplicate check screen is displayed for the duplicate orders with the list of patients and orders against which there are duplicate orders. The duplicate orders will be displayed with the indication of Stop or Warn based on the setup made in the Order Catalog function. Clear the check box for a selected patient if you do not want to place the duplicate orders or clear the check box for the selected orders and an alert message ‘The Selected Order Catalog Will be removed for all patients’ is displayed. Click [OK] to remove the orders for all the selected patients or [Cancel] to cancel the request.

If the check box is selected for the orders which are marked as ‘Stop’ as per the setup in  Order Catalog function the  system will prompt an alert ‘'The order selected cannot be placed as its set as 'Stop' for Duplicate Orders, uncheck the order to proceed'.

  1. The New Orders Tab is displayed to place orders for the selected patients.
