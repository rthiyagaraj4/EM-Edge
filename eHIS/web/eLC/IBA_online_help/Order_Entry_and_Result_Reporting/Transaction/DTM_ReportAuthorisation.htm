Report Authorization

From this function you can Verify, Amend and Authorize Reports.

From this function, a Senior Practitioner from the resulting department can verify and authorize an order result. The results for which a practitioner can authorize are controlled based on the access rights for the logged in practitioner. The practitioner can amend the result attributes before completing the authorization of an order result. Results are posted for authorization only if the order catalog carries the attribute for result authorization.

Authorizing Reports

From the CA Transactions > Clinician Access > Desktop Menu > Pending Activities > Verifications, select the Report Authorization function.

The Report Authorization screen gets displayed.

  1. Select the Order Category and click [Search] to display orders for report authorization for the default date range. Enter the query criteria and click [Search] to display specific reports. Click [Clear] to clear the displayed data.

    1. Order Type - Select the Order Type from the list. The values displayed in the list correspond to the Order Category displayed and rights set for the Practitioner/Responsibility.

    2. Activity Type – Select the Activity Type using [?] function based on the entered Order Type.

    3. Priority – Select the priority from the list. The values displayed are Routine, Stat and Urgent. All is displayed, by default.

    4. Period [From-To] - The date range is displayed, by default. Enter a period in date format of dd/mm/yyyy.

    5. Order ID – Enter the Order ID in alphanumeric format of 15 characters maximum length.

    6. Patient ID - Enter the Patient ID in alphanumeric format of length as specified in Master Patient Index module or select the Patient ID using [?] function.

    7. Encounter ID - Enter the Encounter ID of the patient in numeric format of 12 characters maximum length.

    1. Order Date – The Order Date and Time is displayed. The value is hyperlinked. On clicking the hyperlink, the report entry screen gets displayed. The clinician can amend the result entry, if required.

    2. Priority – The priority of the order is displayed.

    3. Order Details – The Order Catalog is displayed. The catalog is hyperlinked. Click on the hyperlink to view order details.

    4. Patient Details - The Patient details displayed are Patient Name, Gender, Age and Patient ID. The Encounter details displayed are Encounter ID and Ordering Location.

    5. Authorize – Select the check box to authorize the order result. To check all the orders for authorizing, select the check box provided against the ‘Authorize’ heading.

  2. Click Order Date hyperlink to view the results and then authorize the order.

  3. Click Order Details hyperlink to view order details.

    1. The screen displays Order ID, Order Category, Order Type, Priority, and Patient details, Encounter ID, Order Date Time, Order Status, Ordering Location, Ordering Practitioner, Ordering Facility, Performing Facility, Order Format, Clinical Comments and Order Placed By. The screen also displays the Order Catalog, Priority, Frequency, Start Date/Time, End Date/Time, Quantity, Status, Formats and Comments.

    2. Click Format hyperlink to view details in the Order Line Format screen. Click [Close] to close the Order Line Format screen.

    3. Click [Schd] hyperlink to view the Schedule Frequency of the order and [Close] to close the Schedule Frequency screen.

    4. Click Preps, Dept Instrn, Pat Instrn hyperlink to view details if recorded while creating the Order Catalog.

    5. Click [View Audit Trail] to display View Audit Trail screen and view audit trail details. Click the Remark (R) hyperlink to display remarks for the order. Click [Close] to close the remark screen. Click [x] to exit from the View Audit Trail screen.

    6. Click [Close] to close View Order screen.

  4. Select the check box against the required patient details and click [Authorized] to complete authorization or [Clear] to clear the selected order.
