Return Medication

From this function you can Record details of dispensed drugs returned by patients.

The following rules are verified before drugs are received at the Dispense Location.

• The Dispense Location must allow for return medication.

• Workstation of the Pharmacist must be mapped to the Dispense Location.

• The Pharmacist must have rights to receive returned drugs for the Dispense Location.

Returning Dispensed Medication

From the PH Transactions, select the Return Medication function.

The Return Medication Login screen gets displayed.

  1. Return Medication login screen gets displayed.

    Enter the following details to log into the function:

    1. User Name – The logged in User Name gets displayed.

    2. Password – Enter the password to perform the transaction. In the function ‘Dispense Rights’, password stored for you in User Profile is validated. This is a mandatory field.

    3. Patient Category – Select the Patient Category from the list. The listed values are Outpatient and Inpatient. This is a mandatory field.

    4. Dispense Location – Select the Dispense Location from the list. The locations listed correspond to the selected Patient Category and assigned to the logged in user with rights to receive the Retuned Drugs. This is a mandatory field.

  2. Click [OK] to login to the Return Medication function or [Cancel] to cancel the login process.

  3. On Successful login, the Return Medication screen gets displayed.

  4. Enter the details in the mandatory (*) fields. <Related Field Description>

  5. Click [Switch Login] to switch to different Dispense Location or Category.

  6. Click [Search] to view the dispensed drugs.

  7. Click Drug Profile hyperlink to view the Drug details.

    The ‘Patient Drug Profile’ screen gets displayed. The Practitioner can view the complete information on Drugs Prescribed, Prescribed Quantity, Dispensing Status, Dispensing Quantity, Prescribing Practitioner, Ordering Location and Others under Drug Profile. Option to view the profile based on Patient Class and to cancel or discontinue drug based on status is provided.

  8. Enter the details in the mandatory (*) fields. <Related Field Description>

  9. Click [Proceed] to display the Dispensed Drug Details.

  10. Select the 'Return to Bin Location' and enter the Return Quantity in numeric format. The entered value must not exceed the issued quantity of the batch. Medicines, which are reissued to the patient and which are returned are displayed in the color background.

  11. Click [Return] to return the medication or [Cancel] to cancel the process.

  12. You are prompted with a message ‘Do you want to print Return Medication Voucher?’. Click [OK] to generate Return Medication voucher or [Cancel] to cancel the voucher generation.

It is not necessary to return dispensed drug to the issuing Dispense Location. The patient can return drugs to any Pharmacy for which Return Medication is applicable. The return period after dispensing is specified in the function ‘PH Parameter for Facility’.
