Ward Return

From this function you can Record the returned item details returned by the ward Nurses.

The return details are recorded in the returning store for the Inpatient Pharmacy only. The recorded details are displayed in the Return Medication function for the Pharmacist to acknowledge the returns.

Recording Ward Return details

From the PH Transactions, select the Ward Return function.

The Ward Return Login screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to view the dispensed drugs.

  3. Click Drug Profile hyperlink to view the Drug details.

    The ‘Patient Drug Profile’ screen gets displayed. The Practitioner can view the complete information on Drugs Prescribed, Prescribed Quantity, Dispensing Status, Dispensing Quantity, Prescribing Practitioner, Ordering Location and Others under Drug Profile. Option to view the profile based on Patient Class and to cancel or discontinue drug based on status is provided.

  4. Enter the details in the mandatory (*) fields. <Related Field Description>

  5. Click [Proceed] to display the Dispensed Drug Details.

  6. Enter the Return Quantity. .

  7. Click [Cancel Return] to cancel return request for the medication or [Cancel] to cancel the process..

  8. You are prompted with a message ‘Do you want to print Return Medication Voucher?’. Click [OK] to generate Return Medication voucher or [Cancel] to cancel the voucher generation.
