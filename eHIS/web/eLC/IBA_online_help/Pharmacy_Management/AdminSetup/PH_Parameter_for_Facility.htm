PH Parameter for Facility

From this function you can Define Parameter values that suit the requirements of the logged in facility.

The Parameter values defined at enterprise level are inherited and modified. During Medication Ordering, the system first checks for parameters set at facility level and subsequently at enterprise level. Enterprise level parameters must be set before setting at the facility level.

Defining PH Parameter for Facility Rules

From the PH Admin Setup, select the PH Parameter for Facility function.

The PH Parameter for Facility screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Enter the details in the mandatory (*) fields of the tab pages.
