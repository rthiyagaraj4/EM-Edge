PH Parameter

From this function you can Define business logics or rules that influence the functioning of transactions in the module and the behavior of master codes that depend on the defined values.

Some of the values set in the PH Parameter function depend on installation of ‘Inventory Management’ module.

Parameters are to be defined before beginning usage of the module and are specific to each facility, where they are defined.

The following parameter values are defined in PH Parameter:

• Default Route for IV Administration

• Default Frequency for TPN Order

• Refill Applicability

• Prescription Rules for Outpatient/Inpatient

• Dispensing Rules

• Interfacing of PH module with an external drug database

• Default prompts for Dispense Stages.

Defining PH Parameter

From the PH Admin Setup, select the PH Parameter function.

The PH Parameter screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Enter the details in the mandatory (*) fields of the tab pages.

Prescription Rules Tab

The Prescription Rules applicable for various Patient Classes are specified in the tab. The Practitioner cannot violate the maximum prescription duration specified as prescription rules. These values are inherited by the logged in facility and altered to suit its need. The facilities cannot exceed the specified values.

 

Outpatient Encounters

  1. Max Prescription Duration for Outpatients - Enter the Maximum Prescription Duration for Outpatients (in days) in numeric format of three digits maximum length. The Maximum Prescription Duration for Outpatients defined cannot be exceeded for Outpatients and discharged Inpatients (take home medication). This is a mandatory field.

 

Discharge Medications

  1. Max Prescription Duration for Discharge Medications - Enter the Maximum Prescription Duration for Discharge Medications in numeric format of three digits maximum length. The maximum prescription duration cannot be exceeded for Discharge Medications. This is a mandatory field.

 

Inpatient/Other Encounters

  1. Max Prescription Duration for Inpatients () days – Enter the Maximum Prescription Duration for Inpatients in numeric format of three digits maximum length. The maximum prescription duration defined cannot be exceeded for Inpatients. This is a mandatory field.

  2. First Trimester – Select the First Trimester from the drop-down list. The listed values are 1 to 44.

  3. Second Trimester - Select the Second Trimester from the drop-down list. The listed values are 1 to 44.

  4. Third Trimester - Select the Third Trimester from the drop-down list. The listed values are 1 to 44.

  5. Display of no pregnancy record details in Pharmacy Orders -  Select this check box if you want to display the pregnancy indicator in the Pharmacy Orders screen

  6. Infuse Over Calculate by – Select ‘Duration’ or ‘Quantity’ from the drop-down list. By default, the value ‘Duration’ is selected. If the default value is selected, the existing functionality of IV Order will be continued. If the value ‘Quantity is selected, the text fields Order Quantity and Total Volume are displayed within the Pharmacy Orders screen for IV Rx.

 

If the ‘To’ weeks drop-down list value is less than the ‘From’ weeks drop-down list value, the system will prompt a message ‘The ‘To’ week cannot be less than the ‘From’ week’.

 

If the same week range that is already selected for other trimester is selected, the system prompts a message ‘The same week range is selected for Trimester. Please select different week range’

Dispensing Rules Tab

The Dispensing Rules applicable to the enterprise are defined in this tab. The rules defined pertain to dispense alternate forms of drug, dispensing of multi-strength, Outpatient Pharmacy rules for advance and past orders.

  1. Dispense By Alternate Form – Select the check box to allow Dispensing Alternate Form of the prescribed drug. All other forms of the prescribed drug, but belonging to same route can be dispensed.

  2. By Multi-Strength – Select the check box to allow multi-strength dispensing of the prescribed drug(s). This check box is selected and disabled, if ‘By Alternate Form’ is applicable. Dispensing of multi-strength of prescribed drug but belonging to the same form is allowed.

  3. Allow Alternate Generic Name – Select the check box to allow dispensing of alternate generics of the prescribed Drug.

Outpatient Pharmacy

  1. Allow Dispensing in Advance – Select the check box to allow dispensing of drugs in advance of prescription start date.

  2. No. of Days Ahead – Specify the number of days in advance the drugs is dispensed from the prescription start date. Enter the period in numeric format of three digits maximum length. This field is enabled only when ‘Allow Dispensing in Advance’ is selected. This is a mandatory field.

  3. Allow Dispensing past Orders - Select the check box to allow dispensing of past orders (prescriptions) that have exceeded the medication end date and time.

  4. Up to No. of Days - Specify the number of days in numeric format of two digits maximum length up to which an expired prescription order can be dispensed. The system validates expired prescriptions for dispensing based on the end date and time of the prescription. This field is enabled only when ‘Allow Dispensing past Orders’ is selected. This is a mandatory field.

Drug Database Interface Tab

Pharmacy Module can be interfaced with External Drug Database. The database name is specified in the tab.

  1. Drug Database Interface Required – Select the check box to indicate, if the Drug Database Interface is required.

Process Name Tab

Prompts for various dispensing stages are specified in this tab. These prompts displays in ‘PH Parameter for Facility’ function. The dispensing process stages applicable to the enterprise are specified as prompts.

Prompts for Dispensing Stage

  1. Registration – Enter the prompt text for ‘Registration’ in alphanumeric format of 15 characters maximum length. The entered prompt is displayed in ‘PH Parameter for Facility’ function.

  2. Verification - Enter the prompt text for ‘Verification’ in alphanumeric format of 15 characters maximum length. The entered prompt is displayed in ‘PH Parameter for Facility’ function.

  3. Fill - Enter the prompt text for ‘Fill’ in alphanumeric format of 15 characters maximum length. The entered prompt is displayed in ‘PH Parameter for Facility’ function.

  4. Allocate - Enter the prompt text for ‘Allocate’ in alphanumeric format of 15 characters maximum length. The entered prompt is displayed in ‘PH Parameter for Facility’ function.

  5. Bill/Receipt - Enter the prompt text for ‘Bill/Receipt’ in alphanumeric format of 15 characters maximum length. The entered prompt is displayed in ‘PH Parameter for Facility’ function.

  6. Deliver - Enter the prompt text for ‘Deliver’ in alphanumeric format of 15 characters maximum length. The entered prompt is displayed in ‘PH Parameter for Facility’ function.

IV Order Type Legends

  1. IV With Additives – Enter the alternate name for IV with Additives. The described name appears in Order Type list in under ‘IV Rx’ tab in Prescription writer.

  2. IV Intermittent - Enter the alternate name for IV Intermittent. The described name appears in Order Type list box in under ‘IV Rx’ tab in Prescription writer.

  3. IV Without Additives - Enter the alternate name for IV without Additives. The described name appears in Order Type list in under ‘IV Rx’ tab in Prescription writer.

  1. Click [Apply] to save the PH Parameter rules or [Reset] to clear the entered data.

Only one record of PH Parameter is maintained, hence [Create] is not provided in the function.
