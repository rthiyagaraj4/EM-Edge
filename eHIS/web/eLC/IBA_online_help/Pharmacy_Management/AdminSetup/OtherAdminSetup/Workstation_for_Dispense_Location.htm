Workstation for Dispense Location

From this function you can Create an association of Workstation to a Dispense Location and Query for the desired association of Workstation to a Dispense Location.

The Workstations are indicators of the system used by the Pharmacist, which are associated to the Dispense Location for dispensing. Dispense Locations of type ‘Pharmacy’ are available for selection.

Creating an association of Workstation to a Dispense Location

From the PH Admin Setup > PH Other Admin Setup, select the Workstation for Dispense Location function.

The Workstation for Dispense Location screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Dispense Location from the list. The locations listed are of type ‘Pharmacy’ and defined in ‘Dispense Location’ function. This is a mandatory field.

  3. Enter the Workstation Number in numeric format of 25 digits maximum length. This is a mandatory field.

  4. Select the check box to associate the defined Workstation to the selected Dispense Location.

  5. Click [Apply] to save the association of Workstation to a Dispense Location or [Reset] to clear the entered data.

  6. Click [Query] to search for an existing association of Workstation to a Dispense Location.

The system alerts you to enter the mandatory values, if they are not entered. The mandatory field is highlighted with a unique color code. Only 10 records can be added at a time. You cannot duplicate the code. The system alerts you on saving the record by highlighting the duplicate codes with a unique color code.
