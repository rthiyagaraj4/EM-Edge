Term Code for Drug

From this function you can Create an association of Term Code to a particular Drug.

If the Term Code is set for a particular Drug, the patient must be having at least one Term Code as active in the diagnosis profile or on selecting such drug, you are prompted with an alert and the drug will not be active for selection.

Creating the Term Code for Drug

From the PH Code Setup > PH Code Setup, select the Term Code for Drug function.

The Term Code for Drug screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Drug Name in alphanumeric format of 60 characters maximum length or select the Drug Name using [?] function. This is a mandatory field.

  3. On selecting the Drug Name, the associated Term Codes (if any) are displayed in the Associated Term Codes frame.

  4. Select the Term Set from the list. The Term Sets are defined in the Medical Records module. This is a mandatory field.

  5. Select the Term Code using [?] function. The Term Codes associated with the selected Term Set are displayed in the list. This is a mandatory field.

  6. Select the Enabled check box to enable the Term Code for Drug. The check box is selected, by default.

  7. Click [Apply] to save the association of a Term Code to the selected Drug or [Reset] to clear the entered data.
