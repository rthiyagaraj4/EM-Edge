Pregnancy Category for Drugs

From this function you can Define the pregnancy effects of a drug and Query for the defined Pregnancy Category for Drugs.

You can create pregnancy category as per the local drug authority specifications. A master for defining the pregnancy effects of a drug has been introduced. As per this the user can define the pregnancy effects of a drug as per the local drug control authority. You can define any number of pregnancy categories.

Defining the Pregnancy Effects of a Drug

From the PH Code Setup > PH Code Setup, select the  Pregnancy Category for Drugs function.

The Pregnancy Category for Drugs screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the defined Pregnancy Effects of a Drug or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing Pregnancy Effects of a Drug.
