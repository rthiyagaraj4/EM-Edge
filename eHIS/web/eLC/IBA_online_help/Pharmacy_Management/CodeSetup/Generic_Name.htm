Generic Name

From this function you can Define Generic Name for a Drug Class, if required associate it to the ATC Classification and Query for the desired Generic Name.

The defined Generic Name is associated to a Drug Class and ATC Classification or Pharmacological Classification. The Generic Name can be associated to more than one ATC/Pharmacological Classification.

Defining a Generic Name

From the PH Code Setup > PH Code Setup, select the Generic Name function.

The Generic Name screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [ ATC Classification] to associate the Anatomical, Therapeutic and Chemical Classification if required, to the defined Generic Name.

    A Generic Name can be associated to ATC Classification.

    1. Select the appropriate ATC Classification from the ‘ATC Classification’ screen.

    2. Click ‘+’ or ‘-’ hyperlink to select the Anatomical Class. If the selected Anatomical Class has child records under it, select the required Therapeutic and Chemical Classes. The selected classes are displayed in the ‘ATC Classification Display Frame’.

  4. On selecting the required values, click [Select] ([Select] is enabled only on selection of an Anatomical Class). The ATC Classification screen closes and control shifts back to the Generic Name screen.

  5. Select the check box to associate the ATC Class to the Generic Name. The check box is selected, by default.

  6. Click [Apply] to save the defined Generic Name or [Reset] to clear the entered data.

  7. Click [Query] to search for an existing Generic Name.
