Enter the following details:

  1. Dispensing Location – Select the Dispense Location from the list. Only if the Parameter value Direct Dispensing is set to Yes this function will be applicable. The Dispense Locations will be listed based on the parameter value.

  2. Patient ID – Enter the Patient ID or select the required Patient ID using [?] function. Based on the Parameter Setup, if the Encounter ID is mandatory during Direct Dispensing, the system opens an encounter list screen. The number of encounters and the nature of the encounters (either closed or open) depend upon the parameter values set.

  3. Encounter ID - Click Encounter ID hyperlink and the Encounter ID gets displayed in the respective field.

  4. $ - This icon will be visible only for the users who have the rights to view the patient’s current encounter financial details. If the Current Encounter Financial Details check box is selected in the Dispensing Rights screen, this icon with the tool tip  ‘Current Encounter Financial Details’ will be visible. Upon clicking this icon, the patient’s financial details is displayed in a separate ‘Patient Encounter Financial Details’ dialog.

  5. Ordering Practitioner – The Ordering Practitioner associated to the Encounter ID is displayed.
