By Drug Characteristics Tab

Enter the following search criteria:

 

  1. Search By – Search the required Drug based on the following:

 

  • Generic Name – To search Drug based on Generic Name, select this option.

  • Drug Name - To search Drug based on Drug Name, select this option.

  • Trade Name - To search Drug based on Trade Name, select this option.

  • Common Name - To search Drug based on Common Name, select this option.

  • All - To search drug based on Generic Name, Drug Name, Trade Name, and Common Name.

 

  1. Generic/Drug/Trade/Common Name – Based on the search by option selected, enter the Generic/Drug/Trade/Common Name in alphanumeric format or select the Generic/Drug/Trade/Common Name using [?] function. This is a mandatory field.

  2. Search By - Select the search criteria from the list. The values displayed are Starts with, Ends with and Contains. The value ‘starts with’ is displayed, by default.

  3. Form – Select the required Form of Drug from the list.

  4. Strength – Enter the Strength of the required Drug in numeric format of four digits maximum length. Select the appropriate Unit of measurement from the list for the specified strength.

  5. Route – Select the required Route from the list.

  6. Click [Other Criteria] to perform additional search based on the Other Criteria.

    You can restrict the search result by entering the following additional details:

    Others

    1. Non-Essential/Restricted – Select the required value from the list. The listed values are ‘Both’, ‘Yes’ and ‘No’. The value ‘Both’ is displayed, by default.

    2. Drug Type – Select the required drug type from the list. The listed values are ‘Trial Drug’, ‘Infection Control Drug’ and ‘Vaccine’.

    3. IV Ingredients - Select the required value for IV Ingredient from the list. The listed values are ‘Both’, ‘Yes’ and ‘No’. The value ‘Both’ is displayed, by default.

    4. IV Fluid - Select the required value for IV Fluid from the list. The listed values are ‘Both’, ‘Yes’ and ‘No’. The value ‘Both’ is displayed, by default.

    5. Compound - Select the required value for Compound from the list. The listed values are ‘Both’, ‘Yes’ and ‘No’. The value ‘Both’ is displayed, by default.

    6. Affects Pregnancy - Select the required value from the list. The listed values are ‘Both’, ‘Yes’ and ‘No’. The value ‘Both’ is displayed, by default.

    7. Affects Lab Results - Select the required value from the list. The listed values are ‘Both’, ‘Yes’ and ‘No’. The value ‘Both’ is displayed, by default.

    8. Affects Breast Feeding - Select the required value from the list. The listed values are ‘Both’, ‘Yes’ and ‘No’. The value ‘Both’ is displayed, by default.

    9. Formulary - Select the required value for Formulary from the list. The listed values are ‘Both’, ‘Yes’ and ‘No’. The value ‘Both’ is displayed, by default.

    10. Drug Class – Select the required Drug Class from the list. The listed values are ‘General’, ‘Narcotic’ and ‘Controlled’.

    11. Scope - Select the required scope from the list. The listed values are ‘Facility’, ‘Enterprise’ and ‘Dispense Location’. The value ‘Facility’ is displayed in the list, by default.

    12. Schedule – Select the required value for schedule from the list.

    13. Manufacturer – Enter the name of the Manufacturer in alphanumeric format or select the required Manufacturer using [?] function.

  7. Click [Search] to display the Drug Details based on the selected criteria.

  8. On searching, the Drug details are displayed.

  9. Click ‘+’ hyperlink to view the details of the selected Drug.

  10. Click Drug Name hyperlink to select the required Drug.
