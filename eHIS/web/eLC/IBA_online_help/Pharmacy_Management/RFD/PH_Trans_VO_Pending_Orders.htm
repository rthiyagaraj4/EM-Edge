On clicking ‘Pending Orders’ hyperlink, the below screen gets displayed.

 

The following details are displayed:

  1. Order ID – The Order ID gets displayed.

  2. Drug – The name of the Drug gets displayed.

  3. Start Date - The Start Date of the order gets displayed.

  4. Status - The Order Status of the order gets displayed.

  5. Token Series - If the Token is generated or applicable, the Token Series of the order gets displayed.

  6. Token No. - The Token No of the Order gets displayed.

  7. Token Date - The token generated date gets displayed.

  8. Dispense Location - The Dispense Location of the order gets displayed.

The Refill hyperlink displays if Refill is applicable for the ordered Drug. The Pending Orders hyperlink will be displayed in bold and red in colour and it will blink.
