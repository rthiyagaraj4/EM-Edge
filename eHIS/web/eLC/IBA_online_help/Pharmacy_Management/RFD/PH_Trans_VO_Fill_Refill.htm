The following details are displayed.

  1. No Stock [X] – If the Drug stock is currently not available, the ‘X’ mark gets displayed or the first column remains blank.

  2. Alternate Drug - Click  to launch the Alternate Drugs for <Drug name> screen. Select the required alternate drug by selecting the ‘Select’ checkbox and click OK button. A new ‘Remarks’ window is displayed, only if the option ‘Applicable with Mandatory’ is selected in the PH Parameter for Facility screen. But if the option ‘Applicable with Optional’ is selected, the system will not prompt the user to capture the remarks while dispensing the alternate drug.

  3. Rx – To view the prescribed drug details, click ‘Rx’ hyperlink. The ‘Drug/Dosage Details’ screen gets displayed. The prescribed Drug and its dosage details get displayed.

  4. Drug – The prescribed drug(s) of the selected Order ID gets displayed and hyperlinked.

  5. Allocation - The quantity of drug allocated gets displayed after fill is performed.

  6. Balance Quantity - The Balance Quantity after allocation gets displayed. The value displayed in the () indicates the balance quantity.

To allocate alternate drug/prescribed drug, follow the steps explained below. The first Drug Details of the Order ID is selected and the Store/Item and batch details corresponding to it are displayed, by default. Click on other drug hyperlink and view the details under store/item details and batch details section.

 

Store/Item Details

  1. Store - The store linked to the Dispense Location gets displayed (refer to Dispense Location’ master code for linking store to a Dispense Location). A drug is linked to an Item code in Drug master function (refer to Drug Master). If the item linked to the selected drug is available in the displayed store, the store gets displayed.

  2. Item Code - The item code attached to the selected prescribed/alternate drug gets displayed. If drug to item link is same, the item code gets displayed as read-only text. If a drug is linked to many Items, the list contains all the items linked to the selected drug. Select the required Item Code. This is a mandatory field.

  3. Item Desc - The description of the selected item gets displayed as read-only text.

 

Batch Details Section

The batch details of selected prescribed/alternate drug are displayed in this section. The batches are displayed depending on the prescribed quantity. The batches nearing expiry date get displayed by default, followed by batches with long expiry. The system considers the duration of prescription and excludes any batches with expiry date within the prescription duration.

You can select other batches and allocate the required drug quantity. You can also view other batches and allocate stocks as per requirement using [Stock Availability].

 

  • Click [Stock Availability] to view other batches of the Drug.

  • Click [Preview] to view the allocated details.

  • Enter the Allocated Quantity in numeric format of five digits maximum length.

  • Click [Fill for Drug] on entering the required quantity or [Reset] to clear the entered data.

  • Click [Auto Fill for Drug] to Fill drug stock automatically. Batches and quantities are automatically filled for the prescribed Drugs.

  • Click [Complete Fill] on filling the required batches and quantities.
