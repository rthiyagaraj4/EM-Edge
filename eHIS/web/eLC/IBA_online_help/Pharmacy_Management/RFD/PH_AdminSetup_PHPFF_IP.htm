Inpatient Tab

The Parameter values applicable to Inpatients are specified in this tab. Default dispensing locations for routine, stat orders and discharge/take home medications and Dispense Location controls are entered for the logged in facility.

 

Prescription Rules

  1. Consider Stock – Select the check box to Consider Stock while ordering for drugs.

  2. Allow Without Stock – Select the check box to allow ordering without stock. The check box is enabled only when ‘Consider Stock’ is selected.

  3. Allow Duplicate Drug – Select the check box to allow ordering of duplicate drug. This means, the ordering practitioner can order a drug currently being used by the patient. If the check box is cleared, duplication is not allowed.

 

Order Routing Defaults

  1. Routine Orders – Select the default Dispensing Location to route Routine Orders of Inpatients. The locations are defined in ‘Dispense Location’ function. Dispensing locations of type ‘Pharmacy’ and catering to both Outpatient/Inpatients and exclusively to Inpatients are listed.

  2. Stat Orders - Select the default dispensing location to route stat orders of Inpatients. The locations are defined in ‘Dispense Location’ function. Dispensing locations of type ‘Pharmacy’ and catering to both Outpatient/Inpatients and exclusively to Inpatients are listed.

  3. Discharge/Take Home Medication - Select the default Dispensing Location for discharged patients. The locations are defined in ‘Dispense Location’ function. Dispensing locations of type ‘Pharmacy’ and catering to both Outpatient/Inpatients and exclusively to Inpatients are listed.

 

Dispensing Rules

 

  1. Default Order From Date - Select this check box to dispense the medication in advance of prescription start date.

  2. No of days Ahead - Specify the number of days in advance the drugs can be dispensed from the prescription start date. Enter the period in numeric format of three digits maximum length. This field is enabled only when ‘Default Order From Date’ is  selected. This is a mandatory field. The default number of days ahead value would be ‘0’.

  1. Default Order To Date - Select this check box to dispense the medication for past orders (prescriptions) that have exceeded the medication end date and time.

  2. Up to No of Days - Specify the number of days in numeric format of two digits maximum length up to which an expired prescription order can be dispensed. The system validates expired prescriptions for dispensing based on the end date and time of the prescription. This field is enabled only when ‘Default Order To Date’ is selected. This is a mandatory field. The default Up to number of days value would be ‘30’.

By default, the Default Order From Date and Default Order To Date check boxes will be checked. Based on values entered in these two fields, the Order Date From and To will be defaulted within the Verification, Allocation, Fill and Deliver stages of Dispense Medication screen respectively. The Order From Date would be the system date minus 30 days. The Order Date To would be the current date. But you can modify the dates, as required. There is no change in the current functionality in the different stages of Dispense Medication.

 

Stage

 

  1. Verification – Select the check box to indicate that the verification of drugs is required during dispensing. If the check box is selected, ‘Verification’ is applicable for Inpatient-Dispense Location in the function ‘Dispense Location’.

  2. Fill Period – Enter the fill period in numeric format of two digits maximum length. Specify the fill period as ‘Hours’, ‘Days’ or ‘Prescribe Period’. A fill sheet is generated for the specified period from the start time entered for ‘Continuous/Intermittent Orders Schedule Start’. If ‘Prescribe Period’ is selected, fill sheet is generated for the prescribed period.

  3. Deliver - Select the check box to indicate delivery of drug for Inpatient is required. If the check box is selected, ‘Delivery’ is applicable for inpatient-Dispense Location in the function ‘Dispense Location’.

  4. Enable Daily Sequence Number - Select the check box to enable daily sequence number generation for the selected facility in the Dispense Location.

  5. Include Zero Fill Items in Dispensing - Select this checkbox to displays Zero quantity in Fill stage in Dispense Medication screen.

 

Dispense Location Controls

  1. Exclude Routine Orders Placed before [] hours – Enter the period in numeric format of two digits. Routine orders placed within the specified duration are not listed in the generated fill sheet.

  2. IP Fill Process Schedule Start Time – Enter the start time in time format of ‘hh:mm’. Fill sheet is generated from the specified time.

  3. Default IP Fill Incremental Process With - Select the value from the list  to display the Effective Start Time in the IP Fill Process function. The values ‘System Time’ and '00:01' are displayed in the list. ‘System Time’ is displayed, by default.

 

Labels

  1. Label Format - Select the value from the list. The values listed are ‘Drug wise’ and ‘Combined’.

  • Drug Wise – When the drug is dispensed, the label formats are printed for each drug.

  • Combined – When the drug is dispensed, the label formats are printed for the entire prescription.

  1. Default Language – Select the default language from the drop-down list.

    1. The selected language is displayed, as default, in the Print/Edit Label dialogue window and the Dispense label is  printed in the same language. However, if the language ‘Arabic’ is selected, the Dispense label will be printed in both Arabic and English language. The Patient Name, Dosage Details and Cautionary Instructions will be printed in Arabic and the rest of the details will be printed in English.

    1. If ‘None’ is selected , the Dispense label will be printed according to the default language in the Print/Edit Label dialogue window or based on the logged-in user’s language.

By default, ‘None’ is selected as the default Language

 

Drug Return

  1. Return Medication Restriction – Select one of the following value from the drop-down list. By default, the value ‘Select’ is selected

 

  • Initiate Pre-Discharge – If you select this value from the drop-down list, the system does not allow to return the medication for Initiate Pre-Discharge patients.

  • Prepare Discharge Advice - If you select this value from the drop-down list, the system does not allow to return the medication for Prepare Discharge Advice patients.

  • Discharge Patient - If you select this value from the drop-down list, the system does not allow to return the medication for Discharge patients.
