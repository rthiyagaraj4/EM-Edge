Enter the following details:

  1. Send – Select this radio button to send the drugs to external facility. This check box is selected by default if the field ‘Access Rights For’ is selected as ‘Send’ in the Dispensing Rights screen.

  2. Receive – Select this radio button to receive the drugs from the external facility. This check box is selected by default if the field ‘Access Rights For’ is selected as ‘Receive’ in the Dispensing Rights screen.

  3. Deliver - Select this radio button to deliver the drugs to patient in specific location in the facility. This check box is selected by default if the field ‘Access Rights For’ is selected as ‘Deliver’ in the Dispensing Rights screen.

  4. Order Date [From-To] – Select the Order Date using the Calendar function. This is a mandatory field.

  5. Patient ID – Select the Patient ID using lookup function.

  6. Drug – Select the Drug name using lookup function.

  7. Dispense Location – The Dispense Location selected in the Tracking Outsource MedicationLogin screen is displayed.

  8. Order Type – Select the Order Type from the drop-down list. The Order Type defined in the OR Code Setup in the Order Entry Result Reporting is displayed.

  9. Referral Facility – Select the Referral facility from the drop-down list. This is a mandatory Field.

  10. Remarks – Enter the remarks in the character format of 2000 character maximum length.

  11. Click [Refresh] to display Referral Facility defined in the Dispensing Rights screen and Remarks field to enter comments.

  12. If the Send radio button is selected, the Drug Name, Ordered Date, Ordering Location, Ordering Practitioner name are displayed. Select the check box against the Drug Name to send the selected drug to external facility.

  13. If the Receive radio button selected, the Drug Name, Ordered Date, Ordering Location, Ordering Practitioner name and , Outsource and Received. Select the check box against the Drug Name to send the selected drug to external facility.

    Enter the following received drug details:

    1. Product ID – Enter the Product ID in alphanumeric format of 20 character maximum length. This is a mandatory field.

    2. Product Name – Enter the Product Name in alphanumeric format of 60 character maximum length. This is a mandatory field

    3. Batch ID - Enter the Batch ID in alphanumeric format of 60 character maximum length. This is a mandatory field.

    4. Expiry Date - Enter the Expiry Date in the label of the product.

    5. Quantity - Enter the number of received products in numeric format of five character maximum length. This is a mandatory field.

    6. Ingredients – Enter the Ingredients available in the product in character format of 60 character maximum length. This is a mandatory field.

    7. Click – Click to add the other ingredients of the products. You are allowed to add upto five ingredients.

    8. Remarks – Enter the comments about the received products in character format of 2000 character maximum length.

    9. Click – Click to add the details of the other products. You are allowed to add upto five product details.

    Enter the following outsourcing details:

    1. Sent By –The Personnel name who sends the drug to the external facility is displayed.

    2. Date – The date on which the product is outsourced is displayed. You cannot modify the data.

    3. Remarks – The reason for sending the drug to the external facility is displayed. You cannot modify the data.

  14. If the Receive radio button selected, the Drug Name, Ordered Date, Ordering Location, Ordering Practitioner name, Outsource, Location and Delivered To. Select the check box against the Drug Name to send the selected drug to external facility.

    1. Sent By –The Personnel name who sent the drug to the external facility is displayed. You cannot modify the data.

    2. Date –The date on which the product is outsourced is displayed. You cannot modify the data.

    3. Remarks –The reason for sending the drug to the external facility is displayed. You cannot modify the data.

    4. Product ID –The Product ID in the label of the product is displayed. You cannot modify the data.

    5. Product Name – The Product Name in the label of the product is displayed. You cannot modify the data.

    6. Batch ID - The Batch ID in the label of the product is displayed. You cannot modify the data.

    7. Expiry Date - The Expiry Date in the label of the product is displayed. You cannot modify the data.

    8. Quantity - The number of received products is displayed. You cannot modify the data.

    9. Ingredients –The Ingredients available in the product are displayed. You cannot modify the data.

    10. Remarks –The comments about the products received is displayed. You cannot modify the data.
