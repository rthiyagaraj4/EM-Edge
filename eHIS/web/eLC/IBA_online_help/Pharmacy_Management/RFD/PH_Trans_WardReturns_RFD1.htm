Enter the following details:

  1. Patient ID - Enter the Patient ID in alphanumeric format of length defined in Master Patient Index module or select it using [?] function. This is a mandatory field.

  2. Dispense No. - Enter the Dispense Number in alphanumeric format of 15 characters maximum length.

  3. Drug - Enter the drug name in alphanumeric format of 20 characters maximum length or select it using [?] function.

  4. Cut-off Date - Cut-off date is calculated by subtracting medication return period (specified in PH Parameter for facility) from the current system date, that is, (system date – medication return period). Medications dispensed for the selected Patient ID after the cut-off till the current system date are displayed as query result.

  5. Return Request - select this check box to display the Return requested Date [From-To] date” field. Based on the selected date range, the search results is displayed with the check box for all the line item.
