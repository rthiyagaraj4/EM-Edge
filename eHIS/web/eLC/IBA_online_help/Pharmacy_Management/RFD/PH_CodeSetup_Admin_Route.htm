Enter the following details to create various Administration Routes:

  1. Code – Enter the route code in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Description – Enter the description in alphanumeric format of 20 characters maximum length. This is a mandatory field.

  3. Local Language Description – Enter the Local Language Description in alphanumeric format of 20 characters maximum length.

  4. Route Color- Select the color for the Admin Route. The selected route color defaults in the Pharmacy Orders, Preview Rx, Patient Drug Profile and Medication Administration screens. You can modify the Route Color and Description in the Query mode.

  5. Enabled – To enable the defined route select the check box or clear the check box to disable the defined route. The check boxes are selected, by default.
