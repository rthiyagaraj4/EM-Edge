Enter the following details to stipulate the rules for the Drug Category:

 

  1. Drug Category – Select the Drug Category from the list. The Drug Categories defined in the function ‘Drug Category’ are listed. This is a mandatory field.

  2. Patient Class – Select the Patient Class from the list. The listed values are pre-defined. The value ‘*All’ is displayed, by default. This is a mandatory field.

  3. Max Prescription Duration [] Days - Enter the Maximum Prescription Duration in Days for the selected Drug Category and Patient Class combination. The defined duration must not exceed the value set in PH Parameter Setup.

The table mentions the validations.

PH Parameter - Prescription Rules - Outpatient Encounters/Take Home Medications

 

  1. Action - Select the Action from the list. The values displayed are Warning, Reject and Authorize. If ‘Warning’ is selected (default value), you are warned on exceeding the maximum prescription duration. If ‘Reject’ is selected, you are informed that maximum prescription duration is exceeded and the process stops. If ‘Authorize’ is selected, you are informed that the maximum prescription duration is exceeded and requires authorization.

  2. Message Text – Enter the message text in alphanumeric format of 255 characters. The message text is displayed when the maximum prescription duration is exceeded. The text is displayed on exceeding the maximum duration during medication ordering.
