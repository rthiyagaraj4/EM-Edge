Enter the following details:

  1. Drug – Select the drug name using the [?] function.

  2. Product ID – Select the Product ID from the list. The listed values are ‘First Data Bank’ and ‘CIMS’.

  3. Type – The Type field gets enabled only when the Product ID is selected as ‘CIMS’. Select the type from the drop-down list. The listed values are ‘Product’, ‘GGPI’ and ‘Generic Item’. GGPI is an acronym for Global Generic Product Identifier.

  4. External Drug Code –

  5. If the Product ID is selected as ‘First Data Bank’, enter the external drug code in alphanumeric format of 40 characters maximum length. The lookup function is disabled.

  6. If the Product ID is selected as ‘CIMS’, enter the external drug code in alphanumeric format of 40 characters maximum length or select using the CIMS Lookup.  

  1. Enabled – Select this check box to enable the defined external product link. This check box is selected, by default. Only enabled External Product Links are available in other functions.
