Age Groups Tab

This tab enabled you to link the applicable Age Groups for the defined drug. When no Age Groups are linked, the Drug can be prescribed for all Age Groups. To restrict to specific Age Groups, you must ink the Age Groups to the defined Drug. Overlapping age ranges cannot be linked to the drug. Indication can also be made selectively for forcing the weight recording for the Age Groups. Overlap check is performed with matching Gender of the Age Groups selected. If the Age Ranges for specific Gender (All/Male/Female/Unknown) are found matching and overlapping, an error message indicating the same will be displayed.

 

Adding the Age Group to the defined Drug:

 

  1. Select the required Age Group from the list. The listed Age Groups are defined in the Age Group function of Application Masters module.

  2. On selecting the Age Group, the Gender, Minimum Age, Maximum Age, Age Unit associated to the selected Age Group is displayed as read-only text.

  3. Click [Add] to add the Age Group to the defined Drug or [Reset] to clear the entered data.

  4. On adding the Age Group for the defined Drug, the Age Group details are displayed in the top half of the screen.

  5. Select the ‘Drug Applicable’ check box against the desired Age Group, if the Drug is applicable.

  6. Select the ‘Weight Mandatory’ check box, if weight recording is mandatory while prescribing the Drug.

  7. Select the ‘Height Mandatory’ check box, if height recording is mandatory while prescribing the Drug.

  8. Select the ‘Select’ check box against the desired Age Group if the Age Group is applicable for the defined Drug.
