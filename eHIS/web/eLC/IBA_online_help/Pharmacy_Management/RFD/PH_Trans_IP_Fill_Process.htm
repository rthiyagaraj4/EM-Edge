Enter the following details to Generate IP Fill List:

  1. Dispense Location – Select the location from the list. The Locations of type Pharmacy catering to IP Orders and attached to the Workstation of the logged in user are listed.

On selecting a Dispense Location, the following details are displayed.

 

Medication Details

  1. Orders - Select the value from the list. The listed value is ‘Medication’.

  2. Compound - Select the value from the list. The listed values are Both, Single, and Compound. The value ‘Both’ gets displayed, by default.

  3. Exclude Drugs where Fill Quantity is not by Frequency & Duration - Select the check box to exclude absolute orders.

 

Nursing Unit Details

  1. Ordering Facility - Based on the Order Routing set for the Nursing Unit, you can view the Nursing Unit across the Facility in IP Fill Process function.

  2. Select – Select the value from the list. The listed values are Nursing Unit and Physical Location. The Nursing Unit gets displayed, by default.

    Enter the following details:

    1. Building - Select the building from the list. The buildings listed are defined in Application Masters module. Nursing Unit is attached to a floor in a building (Inpatient Management module). The Nursing Units routing Inpatient orders to the selected dispense unit are considered and building/floor attached to these Nursing Units are listed.

    2. Floor – Select the floor from the list. The listed floors are attached to the selected building.

 

Nursing Unit

  1. Nursing Unit [From-To] - Select the Nursing Unit code range using [?] function. The fill list is generated for orders placed by the selected Nursing Unit on the Dispense Location. The [?] function displays all the Nursing Units routing Inpatient orders to the selected Dispense Location.

 

Fill Interval Details

  1. Process Type – Select the Process Type from the list. The IP Fill Process Type can either be Routine or Incremental. To generate fill list for orders placed after completion of routine process, Incremental process is invoked.

  2. Routine – Fill list is generated only once for the selected Dispense Location and for the set fill interval details.

  3. Incremental – This option helps in generating fill list for orders placed after generation of routine fill list.

 

Process Type: Routine

Schedule Start

Current System date gets displayed and time defined for the selected Dispense Location for ‘Continuous/Intermittent Orders Schedule Start []’ in Dispense Location function gets displayed and disabled.

Fill Period

The value specified for ‘Filling process: Filling period’ for the selected Dispense Location gets displayed. You can change only the fill period before fill list generation. Select the required period from the list. The listed values are based on the value specified for ‘Max Prescription Duration for Inpatients [] days’. The fill list is generated for the specified period. This is a mandatory.

Exclude New Orders Placed Within [] Hrs of Scheduled Start

The value specified for ‘Exclude New Orders Placed Within [] Hours of Scheduled Start’ for the selected Dispense Location gets displayed and disabled.

Effective Start Time

The schedule start time is taken as effective start time for fill list generation. Based on the ‘Default IP Fill Incremental Process With’ setup in PH Parameter for Facility function, you can back time the effective start time up to 00:01 for the displayed date.

End Time

The end time is calculated by adding Fill Period with ‘Effective Start Time’ and displayed.

Process Type: Incremental

Schedule Start

Current System date gets displayed and time defined for the selected Dispense Location for ‘Continuous/Intermittent Orders Schedule Start []’ in Dispense Location function gets displayed and disabled.

Fill Period

The value specified for ‘Filling process: Filling period’ for the selected Dispense Location gets displayed and disabled.

Exclude New Orders Placed Within [] Hrs of Scheduled Start

The value specified for ‘Exclude New Orders Placed Within [] Hours of Scheduled Start’ for the selected Dispense Location gets displayed and disabled.

Effective Start Time

The schedule start time is taken as effective start time for fill list generation.

End Time

The end time is calculated based on the fill period.

 

Other Details

  1. Allocate Batches During Fill – Select the check box to perform batch allocation during IP Filling. If the check box is cleared, the allocation of batches have to be performed manually during IP Filling. This field gets displayed only when ‘Allocate batches During Fill Process’ is selected for the selected Dispensing Location.

  2. Order By for Referencing – Select the value from list. The listed values are Nursing unit, bed number and Nursing Unit and Patient ID.

The Ordering Facility and Nursing Unit details are displayed in the View Fill List screen.
