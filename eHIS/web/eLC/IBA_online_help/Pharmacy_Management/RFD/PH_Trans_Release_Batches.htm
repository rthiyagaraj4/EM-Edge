Enter the following conditions to search for the required details:

  1. Allocated Date [From-To] – Enter the allocated date range in the format of dd/mm/yyyy or select the date using the Calendar function. This is a mandatory field.

  2. Patient ID - Enter the Patient ID in the format specified in Master Patient Index module or select the Patient ID using [?] function.

  3. Batches – Select the required value from the list. The listed values are ‘Allocated’ 'On hold by Clinicians' and ‘Discontinued’.

 

Allocated

The result set displays orders for which batches are allocated.

Discontinued

The result set displays discontinued orders for which batches are allocated.
