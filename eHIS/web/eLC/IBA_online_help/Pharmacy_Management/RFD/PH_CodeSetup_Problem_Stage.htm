Enter the following details to define Problem Stage record:

  1. Indicator – Select the required Indicator from the list. The listed values are pre-defined. This is a mandatory field.

  2. Code – Enter the code in alphanumeric format of eight characters maximum length. This is a mandatory field.

  3. Long Description – Enter the long description in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  4. Short Description – Enter the short description in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  5. Enabled – Select the check box to enable the defined Problem Stage. The check box is selected, by default.
