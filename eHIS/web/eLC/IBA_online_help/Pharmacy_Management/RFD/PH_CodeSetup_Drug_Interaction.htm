Enter the following details:

  1. Drug Interaction By – Select the ‘Drug Interaction By’ from the list. The listed values are ‘Generic Name’ and ‘Drug Name’. If the value ‘Generic Name’ is selected, the interaction set up is based on the generic name of the drug.

    • Generic Name1 and Generic Name2 – Select the Generic Names using the [?] function. These fields are mandatory. [or]

    • Drug Name1 and Drug Name2 - Select the Drug Names using the [?] function. This field is displayed only if ‘Drug Name’ is selected in the ‘Drug Interaction By’ field.

  2. Dosage Form1 – Select the form of drug dosage for interacting drug1. For example, Tablets, Capsules. The default value is ‘All’. This is a mandatory field.

  3. Dosage Form2 – Select the form of drug dosage for interacting drug2. For example, Tablets, Capsules. The default value is ‘All’. This is a mandatory field.

  4. Severity Level - Select the Severity Level from the list. The listed values are 'High', 'Medium' and 'Low'. The severity level indicates the severity of the drug interaction to the patient. This is a mandatory field.

  5. Significant Level – Select the Significant Level from the list. The listed values are from Level 1 to Level 5. This indicates the level of significance attached to the patient’s reaction to allergen. The significant level can be used to alert the physician. Level 1 is the most significant level. This is a mandatory field.

  6. Documentation– Select the documentation from the list. The listed values are ‘Established’, ‘Suspected’, ‘Possible’, ‘Unlikely’ and 'Probable'. This is a mandatory field.

    • If the value ‘Established’ is selected, the drug interaction will definitely occur.

    • If the value ‘Probable’ is selected, the drug interaction may or may not occur.

    • If the value ‘Suspected’ is selected, the drug interaction is suspected to occur.

    • If the value ‘Possible’ is selected, the drug interaction may occur.

    • If the value ‘Unlikely’ is selected, the drug interaction is unlikely to occur.

  1. Restrict Transaction – Select this check box to restrict prescribing the interaction drugs to the patient. If this check box is cleared, the interaction drugs can be prescribed. If this checkbox is checked, the application will not allow the doctor to prescribe the interacting drug. The message ‘Hence, this drug cannot be prescribed’ is shown in the alert message.

  2. Interaction Description – Enter the remarks about drug interaction in 400 characters maximum length. This is a mandatory field.
