Inpatient Prescription Tab

The dispensing rules for Inpatients are specified in this tab. The dispensing rules defined for Inpatient in ‘PH Parameter for Facility’ are made default. You can change these values to suit working of the Dispense Location.

 

Enter the following dispensing rules for Inpatients.

Stage

  1. Verification – The check box is selected, if ‘Verification’ is applicable for Inpatient in ‘PH Parameter for Facility’. It indicates verification of drugs is required during dispensing for inpatients. Clear the check box to indicate ‘Verification’ is not required.

  2. New Prescriptions Only – Select the check box if the Dispensing Location is applicable for New Prescriptions only.

  3. Fill – Select the check box if ‘Fill’ is applicable for the Dispensing Location.

  4. Fill Process - Fill Period – This field is used to compute the period for generating the fill sheet. The value specified in ‘PH Parameter for Facility’ is made default and can be modified but not exceeding the value defaulted. Specify the fill period in numeric format of two digits maximum length. The period can be defined in terms of ‘Hours’, ‘Days’ or can be based on the ‘Prescribe Period’. A fill sheet is generated for the specified period. If the fill period is based on hours or days, the period computed for fill sheet generation shall include the time specified in ‘Continuous/Intermittent Orders Schedule Start’.

  5. Deliver - The check box is selected, if ‘Delivery’ is applicable for Inpatient in ‘PH Parameter for Facility’. It indicates delivery stage is applicable for the Dispense Location. If delivery stage is not applicable to the Dispense Location.

  6. Exclude Routine Orders Placed within [] Hours of Schedule Start – The value specified in ‘PH Parameter for Facility’ is made default. Specify the period in numeric format of two digits maximum length. Routine orders placed within the specified duration are not listed in the generated fill sheet. The parameter value cannot be exceeded.

  7. Allocate – If allocation stage is applicable to the Dispense Location, select the check box. Clear the check box, if allocation is not required.

  8. Auto Dispense At Administration – Select the check box to indicate automatic dispensing of the drug during medication administration. Stock of the drug is updated on administration to the patient.

  9. IP fill process Schedule Start time – The value specified in ‘PH Parameter for Facility’ is made default. Enter the start time in the time format of ‘hh:mm’. Fill sheet is generated from the specified start time.

  10. Allow Medication Return - Select the check box to allow returning of Inpatient Medication. If the check box is selected, it indicates dispensed medication can be returned back to the Dispense Location.

  11. Daily Sequence Number Range - Enter the ‘Display Sequence Number Range’ in numeric format of five digits maximum length. Click on the ‘ View Ranges’ hyperlink to display ‘Defined Sequence Ranges’ screen.  The Daily sequence number is generated at the Dispense Location level, which is mapped to different labels/reports printed in different printer in a Dispense Location for a patient.    

    • The screen displays the sequence number range defined for other Dispense Locations within the facility. This helps in avoiding duplication of sequence number.

 

Printer Location

  1. For Stat Order - Enter the Printer Location for stat orders in alphanumeric format of six characters maximum length.

  2. For Routine Order - Enter the Printer Location for routine orders in alphanumeric format of six characters maximum length.

  3. Fill Period: Fill Without Fill List – Specify the fill period in numeric format of two digits maximum length. The period can be defined in terms of ‘Hours’, ‘Days’ or can be based on the ‘Prescribe Period’. On loading, the ‘Fill Period: Fill Without Fill List’ value is same as the ‘Fill Process: Fill Period’ value but the user can modify as per the requirement, to display in 'Fill Without Fill List' screen only.

  4. Priority: Fill Without Fill List – Select the priority from the list. The listed values are ‘All’, ‘Urgent’ and ‘Routine’.
