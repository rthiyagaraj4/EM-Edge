Enter the following search criteria:

 

  1. ADR Reported Date From and To - Enter the ADR reported date in date format of dd/mm/yyyy or use the Calendar function to select for the required date.

  2. Status – Select the Status from the list . The values are Verified, Unverified, All, Cancel and In-Error. The value Unverified is selected, by default.

  3. ADR No - Enter the ADR number in numeric format of 10 digits maximum length.

  4. Patient ID – Enter the Patient ID in the required format, or select the required patient using [?] function. This is a mandatory field.

  5. Reported By - Enter name of the person recording the ADR in alphanumeric format or select the required name using [?] function.

  6. Signed By - Select the person signing the ADR Report using [?] function or enter the name in alphanumeric format.

  7. Suspected Drug - Enter the suspected drug name in alphanumeric format of 40 characters maximum length or select the suspected Drug using [?] function.

  8. Search – Click this to view the search results. The details displayed in the search results are

    • Patient ID – Identifier of the patient. Displayed as hyperlink. Click the hyperlink to view the ADR details in the Adverse Drug  Reaction screen.

    • ADR No – Adverse Drug Reaction number. Displayed as hyperlink. Click the hyperlink to view the ADR details in the Adverse Drug Reaction screen.

    • Facility Name – Displays the facility name.

    • Reported Date – Displays the Reported date when the ADR was noted.

    • Reported By - Displays the Reported date when the ADR was noted.

    • Signed By - Displays the name of the person who signed the ADR.

    • Suspected Drug – Name of the suspected drug.

    • ADR Type – Type of adverse drug reaction.

    • Status – Status of the adverse drug reaction.

    • Remarks – Displayed as hyperlink only for Cancelled/In-Error ADRs, only if remarks exist. For other ADRs, this will be blank. Click the hyperlink to modify the remarks in the Cancelled/In-Error Remarks popup.
