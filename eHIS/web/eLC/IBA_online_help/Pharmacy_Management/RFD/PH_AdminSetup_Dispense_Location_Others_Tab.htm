1. Online Print Setup - Click ' Online Print Setup' hyperlink to invoke Online Print Setup screen. All the Online Reports applicable for the enterprise are listed.

    All the Online Reports applicable for the enterprise are listed.

    • Select the 'Select All' value form the list. The listed values are 'No Print', 'Print', and 'Print Default'. The list options are available only when more than one report is printing at that stage. On selecting the 'Select All' value, the selected list value is displayed, by default for all reports. You can change the list values individually for the reports.

    • If ‘No Print’ is selected, the print screen will not appear.

     

    Click [OK] to record the selected values or [Cancel] to cancel the details.

  2. Print Dispensing Sheet in Direct Dispensing - Select the check box is selected, if ‘Print Dispensing Sheet’ is applicable in Direct Dispensing. If the check box is selected, it indicates dispensing sheet is printed after dispensing for the Dispense Location.

  3. Print Dispense Label in Direct Dispensing -  Select the check box is selected, if ‘Print Dispensing Label’ is applicable in Direct Dispensing. If the check box is selected, it indicates dispensing label is printed after dispensing for the Dispense Location.

 

Spill Quantity Details

  1. Usage of Spill Quantity during Admixture - Select the check box, if the usage of the Spill Quantity is permitted during admixture. If the check box is cleared, the usage of the spill quantity is not permitted.

  2. Charge Patient for Spill Quantity – Select the check box, if the usage of the Spill Quantity is chargeable.

 

Billing Interface

  1. Bill Stage – The value specified in ‘PH Parameter for Facility’ is defaulted. If Patient Billing Manager module is interfaced, the Bill Stage is applicable. Select the value from the list. The values listed are ‘Not Applicable’, ‘Upon Allocation’ and ‘Upon Delivery’.  

  2. Receipt Stage – The value specified in ‘PH Parameter for Facility’ is made default. If Patient Billing Manager module is interfaced, Bill/Receipt is applicable. The ‘Receipt Stage’ field is enabled if ‘Upon Allocation’ and ‘Upon Delivery’ is selected in Bill Stage field. Select the value from the list. The values listed are ‘Not Applicable’, ‘With Delivery’ and ‘Separate’.
