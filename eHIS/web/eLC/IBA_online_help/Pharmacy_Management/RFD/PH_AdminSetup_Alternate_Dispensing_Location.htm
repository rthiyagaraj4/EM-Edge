Enter the following details for specifying the Alternate Dispensing Location:

Ordering Location Detail

 

  1. Ordering Facility – The logged in facility is displayed, by default.

  2. Ordering Source Type – The Ordering Source Types listed are Clinic and Nursing Unit. Select the required value from the list. This is a mandatory field.

  3. Ordering Source Location – Based on the ‘Ordering Source Type’ selected values are listed. The Clinics of the selected Ordering Facility are listed for Ordering Source Type as ‘Clinic’. All the Nursing Units of the selected Ordering Facility are listed for Ordering Source Type as ‘Nursing Unit’. The values for type ‘Nursing Unit’ gets displayed if the Nursing Units are defined in the Inpatient and Day Care Management module of Enterprise Management. This is a mandatory field.

  4. Priority - Select the Priority from the list. The listed values are 'All', 'Routine', 'Stat' and 'Discharge'.

  5. Routing Level - Select the Routing Level from the list. The listed values are 'Source', 'Order Type', 'Drug Class', 'Drug', 'Special Orders',  'Medical Item', and 'Home Leave’.

  6. Current Dispense Location - The Dispense Location specified for the selected Ordering Source Location in the function ‘Order Routing’ is displayed, by default. Orders are routed to the Dispense Location based on the setup. You can select the required current Dispense Location. This is a mandatory field.

  7. Working Hours From - On selecting the ‘Current Dispense Location’, the Working Hours of the location is displayed.

 

Performing Location Detail

  1. Performing Facility - The logged in facility is displayed, by default. You can select other facility from the list. All the facilities of the enterprise are listed. This is a mandatory field.

  2. Day Type – Select the Day Type from the list. The listed values are ‘Holidays’, ‘Sunday’, ‘Monday’, ‘Tuesday’, ‘Wednesday’, ‘Thursday’, ‘Friday’, ‘Saturday’ and ‘All Days’.
