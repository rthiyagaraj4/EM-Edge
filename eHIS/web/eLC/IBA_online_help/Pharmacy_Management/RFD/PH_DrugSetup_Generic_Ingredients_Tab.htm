Generic Ingredients Tab

If the defined drug is compound, the ingredients of the drug are entered. If ‘Compound’ check box is cleared, you are alerted that the values cannot be entered.

  1. On selecting the Generic Ingredients tab, the Generic Details of the Drug is displayed in top half of the screen.

  2. Click ‘Generic ID’ hyperlink to modify the Generic Details.

  3. Click [Modify] to save the changes or [Reset] to clear the modified changes.

 

To add/delete generic Ingredients to Drugs

 

Enter the following details to add Generic Ingredients to Drug:

  1. Generic Name – Select the Generic Name using [?] function. The Generic Names defined in the Generic Name function are displayed.

  2. Display Sequence No – Specify the Display Sequence Number in numeric format of three digits maximum length. On selecting the Generic Name, the value ‘10’ is displayed by default, which can be changed.

 

Click [Add] to add the Ingredients or [Reset] to clear the entered data. On adding the Generic Ingredients, the Generic Details are displayed in the top half of the screen. Select the check box against the Generic Name to add the Ingredients or clear the check box to remove the Ingredients from the list. The check box is selected, by default.
