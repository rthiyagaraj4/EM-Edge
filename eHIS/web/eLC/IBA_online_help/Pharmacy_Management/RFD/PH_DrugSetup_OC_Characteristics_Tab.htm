Characteristics Tab

A sub-tab ‘Consent/Refusal’ tab is provided under ‘Characteristics’ tab. The Consent/Refusal tab helps you to attach a consent/refusal form to the Order Catalog.

 

Consent/Refusal Tab

Enter the following details to record or refuse the consent:

  1. Consent Required - Select the check box, if patient consent is required from the patient. Clear the check box, if consent is not required. Consent From List, Consent Stage and Consent note text fields are enabled, if the check box is selected.

  2. Consent From List – Select the Consent From List value from the list. The Listed values are ‘Fixed’ and ‘Variable’. The value ‘Fixed’ is displayed, by default.

  3. Consent Stage – Select the Consent Stage as ‘Before order Release’ or ‘Before Order Registration’. ‘Before Order Release’ is selected, if Patient consent is required before order is released by the Practitioner. ‘Before Order Registration’ can be selected, if consent is required before order is registered by the Practitioner.

  4. Multiple Consent – Select the check box, if multiple consent is required.

 

Number of Consent Required

  1. Before Order Release – On selecting the Multiple Consent check box, this field is enabled or this field is disabled with the default value. Enter the value in numeric format of two digits maximum length.

  2. Before Order Registration – On selecting the Multiple Consent check box, this field is enabled or this field is disabled with the default value. Enter the value in numeric format of two digits maximum length.

  3. Consent Note Text – Select the Consent Note Text from the list. This field is enabled only when ‘Consent Required’ is selected. The consent notes defined in the function ‘Consent Note Text’ is listed. The record of type ‘Consent Form’ is populated.

  4. Refusal Note Text – Select the Refusal Note Text from the list. Refusal Note Text is defined in ‘Consent/Refusal/Referral notes’ function. The record of type ‘Refusal Form’ is populated.
