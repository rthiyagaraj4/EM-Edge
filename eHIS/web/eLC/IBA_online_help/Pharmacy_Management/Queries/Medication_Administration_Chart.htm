Medication Administration Chart

From this function you can Query for the Medication Administration details of an Inpatient, Day Care Patient, Emergency Patient and Outpatient.

The Medication Administration Chart displayed corresponds to the Patient ID and Encounter ID combination. You can view the details administered in the Medication Administration using this function. You can view the Medication Administration details by selecting past dose, future dose or by specifying date range.

Querying for the Medication Administration details

From the PH Queries, select the Medication Administration Chart function.

The Medication Administration Chart screen gets displayed.

  1. Enter the Patient ID in alphanumeric format of length defined in Master Patient Index module or select the Patient ID using [?] function.

  2. On selecting the Patient ID, the Encounter ID of the selected patient is displayed.

  3. Click [Search] to display the Medication Administration details of the selected patients.

  4. Select ‘Past’, ‘Future’ Dose, ‘From’, ‘To’ Date, Display Order By and click [GO] to display the medication administration details for that particular period. By default the value Status is selected in the Display Order By drop down .

  5. Click [Reset] to clear the specified criteria.

The past dose list contains dates from the medication start date till the current system date. The future dose list contains dates greater than the current system date till the medication end date.
