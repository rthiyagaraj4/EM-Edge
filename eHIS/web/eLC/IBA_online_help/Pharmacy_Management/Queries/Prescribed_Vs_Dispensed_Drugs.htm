Prescribed vs Dispensed Drugs

From this function you can Query the information on Drugs dispensed against a prescription.

Querying the information on Drugs Dispensed against a Prescription

From the PH Queries, select the Prescribed vs Dispensed Drugs function.

The Prescribed vs Dispensed Drugs screen gets displayed.

  1. Click [Search] to display all the dispensed Drugs or enter the search criteria and click [Search] to display a specific dispensed Drug.

    Enter the following search details:

    1. Order Type – Select the Order Type from the list. The listed values are ‘Routine’ and ‘Urgent’.

    2. Order ID – Enter the Order ID in alphanumeric format of 15 characters maximum length.

    3. Order Status - Select the Order Status from the list. The listed values are Fully Dispensed, Partially Dispensed and Pending.

    4. Order Date [From-To] - Enter the Order Date range in the format of dd/mm/yyyy or select the required date range using the Calendar function.

    5. Patient Class – Select the Patient Class from the list. The listed classes are pre-defined.

    6. Location Type - Select the Location Type from the list. The listed values are Nursing Unit and Clinic.

    7. Location - Enter the required Location or select the Location using [?] function. This field is enabled on selecting a Location Type. The locations that correspond to the selected Location Type are displayed.

    8. Patient ID – Enter the Patient ID in alphanumeric format of length as specified in MP Parameter or select the required Patient ID using [?] function. This is a mandatory field.

    On selecting a Location Type, the corresponding Patient Class gets displayed as read-only text.

  2. On searching, the Dispensed Drugs are displayed with a unique color code based on the selected Order Status. The status indicators are displayed at the bottom of the screen.

  3. Click ‘Order ID’ hyperlink to display the Order Details and Dispensed Details in the Order Details frame and Dispensed Details frame.

  4. Click [Reset] to clear the specified criteria.

On selecting a Location Type, the corresponding patient class gets displayed as read-only text.
