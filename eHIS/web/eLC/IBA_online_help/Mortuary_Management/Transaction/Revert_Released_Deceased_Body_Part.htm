Revert Released Deceased/Body Part

From this function, you can Revert the released deceased and body part back to the mortuary.

This function allows reverting the released deceased and body part back to the mortuary. Once revert release is done, revert record is available in the ‘Release Deceased/Body Part’ function. The associated RFID tag number and Location are defaulted in the ‘Manage Deceased/Body Part’ function if it is not used for any other patient. You can modify the defaulted RFID tag number and Location if required. If the associated RFID tag number and Location is used for another patient, you can assign new RFID tag number and Location.

Reverting Released Deceased/Body Part

From the MO Transactions, select Revert Released Deceased/Body Part function.

The Revert Released Deceased/Body Part screen gets displayed.

 

The screen consists of 2 tabs:

Deceased Patients

Body Parts

 

The 'Deceased Patients' Tab is displayed, by default.
