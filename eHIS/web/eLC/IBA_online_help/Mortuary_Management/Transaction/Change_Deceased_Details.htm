Change Deceased Details

From this function, you can Change the details of registered deceased patients in the mortuary.

Deceased details of patients are vital information maintained in the mortuary. Changes to these details are to be modified regularly by the facility. This function allows amending deceased details of the registered patient. User information performing changes to deceased details of patients is maintained. Details not recorded during other modes of registration are captured in this function and deceased patient profile is modified.

Changing Deceased Details

From MO Transactions, select Change Deceased Details function.

The Change Deceased details screen gets displayed.

  1. Deceased patients registered in the mortuary for the current date are listed.

  2. Enter the search criteria and click [Search] to select a specific deceased patient to change deceased details. Deceased Patient Details are displayed based on the entered criteria. The status of the patient is indicated by a unique color code. The burial permit of the patient is displayed if Postmortem is not required for the patient or if postmortem recording has been completed and the burial permit is already generated for the patient.

    1. Registration No - Enter the registration number or select the registration number of the deceased body using [?] function.

    2. Patient ID - Enter the deceased patient ID in alphanumeric format of 10 characters maximum length or select the deceased patient ID using [?] function.

    3. Area Type - Select the required area type from the list. The listed values are Postmortem Area, Release Area, Storage Area and Others.

    4. Area - Select the area from the list. The listed values are defined in the Area function. The list also displays the 'Unassigned' Area indicating that deceased patients are not yet assigned to any area.

    5. Postmortem Status - Select the status of postmortem from the list. The listed values 'Not Applicable', 'Not Requested', 'Requested', 'In Progress' and 'Completed'.

    6. Release Status - The value is displayed as "Not Released" and is disabled, by default.

    7. Date From-To - Enter the date range in dd/mm/yyyy format within which deceased patients were registered in the mortuary or select the date range using the calendar function. This is a mandatory field. The current date is displayed, by default.

    8. Gender - Select the gender from the list.

    9. Burial Permit No - Select the burial permit number using [?] function.

    10. Body Part Type - 'Not Applicable' is displayed, default.

  3. Click the Deceased Name hyperlink to change the deceased patient details in the mortuary.

  4. Click [Apply] to save the modifications the deceased patient details or click [Query] to go back to the Change Deceased Details main screen and query records for which deceased patient details have to be changed.

 

This function refreshes the patient queue list as per the time limit as set in the field 'Refresh Interval' of the MO Parameters.
