Release Location

From this function you can Release a Blocked Location before the specified blocking period has elapsed.

Releasing a Blocked Location

From the MO Transactions, select Release Location.

The Release Location screen gets displayed.

  1. Click [Search] to display all the blocked locations or enter the search criteria and click [Search] to select a specific blocked location. Click [Clear] to reset the records displayed in the result area.           

    1. Area - Select the required Area from the list. The listed values are defined in the 'Area' function.

    2. Location - This field is enabled on selecting the 'Area'. Select the required location of the selected Area to be released using [?] function.

    3. Blocked Until - Select the date until when the location was blocked using the calendar function.

  2. Select the Release check box against the desired blocked location to release it.

  3. Click [Apply] to save Releasing the blocked location or [Reset] to clear the entered data.
