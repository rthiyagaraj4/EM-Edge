Deceased Patients Tab

  1. Enter the search criteria and click [Search] to select a specific released deceased/body part. Deceased Patient Details are displayed based on the entered criteria. The status of the patient is indicated by a unique color code.

    Deceased Patient Details:

    1. Registration No - The mortuary registration number generated on registering the deceased /body part is displayed.

    2. Registration Date/Time - The date and time of registration is displayed.

    3. Deceased Name/Patient Name - The name of the deceased is displayed and hyperlinked in 'Deceased Patient' Tab and the name of the patient whose body part is registered in the mortuary is displayed and hyperlinked in the 'Body Parts' Tab.

    4. Patient ID - The Patient ID generated when the patient was registered in the ‘Master Patient Index’ module is displayed.

    5. Age - The age of the patient recorded when the patient was registered to the facility is displayed.

    6. Sex - The gender of the patient is displayed.

    7. PM Y/N - If postmortem is required for the deceased/body part, then "Yes" is displayed, else "No" is displayed. The values are displayed based on the ‘Postmortem Required’ field in ‘Deceased Registration/Body Part Registration’ function.

    8. Postmortem Status – The postmortem status is displayed.

    9. Duration in Mortuary - The duration of the deceased/body part in the mortuary is displayed.

    10. Religion - The religion of the deceased/body part is displayed.

    11. Encounter ID - The last encounter ID of the patient is displayed.

    12. Location - The location assigned to the deceased /body part in the area during registration is displayed.

    13. RFID Tag No – The RFID Tag No assigned to the released/body part is displayed.

    14. Body Part Type - This is displayed only in the 'Body Parts' Tab. The type of body part registered for the patient is displayed as 'Amputated Body Part/Dead Fetus/Unidentified Body Part'.

    15. Burial Permit No - The Burial Permit Number is displayed if the burial permit was generated for the patient.

    Search criteria:

    1. Registration No - Enter the registration number or select the registration number of the deceased body using [?] function.

    2. Patient ID - Enter the deceased patient ID in alphanumeric format of 10 characters maximum length or select the deceased patient ID using [?] function.

    3. Area Type - The  area type is 'Release Area', by default.

    4. Area - Select the area from the list. The listed values are defined in the Area function.

    5. Postmortem Status - Select the status of postmortem from the list. The listed values 'Not Applicable', 'Not Requested', 'Requested', 'In Progress' and 'Completed'.

    6. Release Status – 'Released' is displayed by default.

    7. Date From-To - Enter the date range in dd/mm/yyyy format within which deceased patients were registered in the mortuary or select the date range using the calendar function. This is a mandatory field. The current date is displayed, by default.

    8. Gender - Select the gender from the list.

    9. Burial Permit No - Select the burial permit number using [?] function.

    10. Body Part Type - 'Not Applicable' is displayed, by default.

    11. Refresh Interval – The Refresh Interval is displayed as recorded in the ‘MO Parameter’ function.

  2. Click the ‘Deceased Name’ hyperlink. The ‘Revert Released/Body Part’ dialog is displayed.

  3. Select the ‘Reason’ from the drop-down list in the ‘Release Details’ tab. This is a mandatory field.

  4. Enter the ‘Remarks’ and click [Revert Released]. The reverted record is now available in the ‘Release Deceased/Body Part’ function.
