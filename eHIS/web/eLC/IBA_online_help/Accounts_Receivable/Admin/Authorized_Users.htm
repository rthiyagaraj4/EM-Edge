Authorized Users

From this function you can Define the Authorized Users to associate patients to a Customer.

The Authorized User function is used to authorize the users who can associate the patients to the Customer. The authorized users are permitted to associate the patients to avail the benefits of the associated Customer. You can authorize multiple users for this task using this function.

Defining Authorized Users

From the Admin, double-click Authorized Users function.

The Authorized Users screen gets displayed.

  1. Click [List of Values] to select a User ID. The name of the selected User gets displayed in the adjacent text box. The selected user is permitted to associate a patient to the Customer. The Users defined in the System Manager module and associated to the logged-in responsibility and facility gets listed. This is a mandatory field.

  2. Select the check box to permit the selected user to associate patients to a Customer.

  3. Click [Commit Form] or press [F10] to save the records or [Clear Record] to clear the entered or queried data.

  4. Click [Delete Record] to delete a created or queried record.
