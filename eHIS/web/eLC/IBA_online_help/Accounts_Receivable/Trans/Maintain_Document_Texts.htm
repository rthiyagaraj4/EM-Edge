Maintain Document Texts

From this function you can Record the document text for the Customer transactions. You can Query to view the existing details.

This function is used to enter the document text for any transaction pertaining to Accounts Receivable module. Any information, which is associated with the transaction, can be recorded. More details pertaining to the Posted Invoices, Debit Notes, Credit Notes, Adjustments, Receipts and Refunds is obtained by clicking [Execute Query] or pressing [F8]. A similar screen as that of Posted Documents is displayed which includes the details of Posted Documents, Patient Bills, User ID and Document Text. You cannot update the contents of the posted documents. This screen allows you only to view the posted details.

Creating and Maintaining Document Texts

From the Transactions, double-click Maintain Document Texts function.

The Maintain Document Texts screen gets displayed.

  1. Select the Transaction Type from [List of Values]. The valid values are Invoices, Debit Note, Credit Note, Adjustment, Receipts and Refunds. ‘Invoices’ is displayed, by default.

  2. Select a valid Document Type from [List of Values]. This is a mandatory field. Enter the Document Number in numeric format of eight digits.  

  3. Click Calendar function to select the document date. Document date has to be less than or equal to system date.

  4. Enter the Document Text in alphanumeric format of 4000 characters maximum length.  

  5. Clicking [More Details] populates details of posted documents based on the Transaction Type and the Document Type and Number selected.

  6. Click [Commit Form] or press [F10] to save the records.
