Edit List of Receipts

From this function you can Edit a list of Receipts generated for the Customer transactions.

This function is used to check for all the field values entered in Receipts and provide information on Errors, if any encountered for the generated Receipts. The information is provided in the form of report with required details. The report can be generated in Erroneous Documents, Error free Documents and All Documents format.

Editing a Receipt

From the Transactions > Receipts, double-click Edit List of Receipts function.

The Edit List of Receipts screen gets displayed.

  1. Select a valid Document Type Code from [List of Values]. Both the values are optional. When both values are entered, ‘TO’ value must be greater than or equal to the ‘FROM’ value.

  2. Enter the range for Document Number. Both the values are optional. When both values are entered, ‘TO’ value must be greater than or equal to the ‘FROM’ value.

  3. Click the Calendar function to set a valid date. Both the values are optional. When both values are entered, ‘TO’ value must be greater than or equal to the ‘FROM’ value.

  4. Select the appropriate value from [List of Values]. The values listed are Erroneous Document, Error free Documents and All Documents.

  5. Click [Run Process] to view, print or save the adjustment details in a file in the local machine.  

  6. A log file gets generated. Click [OK] to view the log file or [Cancel] to exit from the screen.
