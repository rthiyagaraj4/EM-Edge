Receipt Lines

The entries for the Receipts already allocated but not posted are displayed. This is displayed as “Advance” in Line Type.

Patient Bills

On clicking on any of the Line Type a Bill Wise Allocation window as shown is displayed for selection. Document Date, Document Type, Number, Patient Number, Outstanding Amount, Allocated Amount of each Invoice are displayed for selection.

 

  1. Select - Select the check box to select the Outstanding Amount against which the Receipt Amount is to be allocated.

  2. Options - This is a radio button with options “Select All”, “Clear All”. Click on "Select All", if all the Outstanding Amount are to be selected for allocation else select “Clear All”. Invoices are sorted based on the Options selected. “Unselect All” is selected as the Option, by default.

After selecting the required preference, click [Accept] to process or [Cancel] to re-enter the selection criteria for the Bill Wise allocation. The accepted record is displayed under the Line Type.

  1. Line Type - Select the appropriate Line Type from the [List of Values]. The values are system defined - Pending, Direct and Advance. This field is mandatory.

  2. If the option selected is “Direct”, the user is allowed to enter the Receipts received directly from the Customers.

  3. If the option selected in “Pending”, the user has an option to select the pending Invoices by pressing the “Tab” key. A window as shown with all the Pending Invoices is displayed for selection. Document Type Code, Document Number, Document Date and Outstanding Amount of each Invoice are displayed for selection.

  1. Select – Select the check box to select the pending Invoice for which the Receipt amount is to be allocated.

  2. Options - This is a radio button with options “Select All”, “Unselect All”. Click on "Select All", if all the Invoices are to be selected for allocation else select “Unselect All”.  Invoices are sorted based on the Options selected. “Unselect All” is selected as the Option, as default.

After selecting the required preference, click [Accept] to process or [Cancel] to re-enter the selection criteria for the receipt allocation. The accepted record is displayed under the Line Type.

  1. Doc Type – Select a valid Document Type from the [List of Values]. This field is mandatory.

  2. Number – Enter the Number in numeric format of eight digits maximum length. This can be entered if Line Type is "D" (Direct). If line type is "A" (Advance), the receipt document type and receipt document number is displayed. If line type is "P" (Pending), the outstanding document type and the document number can be selected from the list. This field is mandatory.

  3. Amount - Enter the Amount in numeric format of nine digits maximum length. This can be selected from the [List of Values]. If line type is "P" (Pending), the amount selected must  not be greater than the pending amount of the outstanding document being allocated. If the amount entered is positive, it is shown under Credit Column. If the amount entered is negative, it is shown under Debit Column. This field is mandatory.

  4. Debit – This is a display field. When the amount value entered is greater than zero, the amount is considered to be Debit. The amount is displayed in this field automatically.

  5. Credit – This is a display field. When the Amount value is less than zero, the amount is considered to be Credit. The amount is displayed in this field automatically.

  6. Posting Account – Enter the Posting Account Code in alphanumeric format of 6,4,6,3,4,3,8 characters each maximum length. This field is mandatory. This can be selected from the [List of Values].

  7. Narration - Enter the Narration in alphanumeric format of 50 characters maximum length. The description for invoice line can be entered as narration.
