Enter the following to review/update Credit Note Posting Account:

 

  1. Doc Type/No - Click [List of Values] to select a valid Document Type and Number. Document Number is generated by the system, if it is defined for the entered document type. Press [F8] or click [Execute Query] to view the posting account information created initially for the selected Document Type. You can edit the entries made.

  2. Org Doc – The original document type and number is displayed on the screen. This is a display field.

  3. Amount – This is a display field. The amount gets displayed on the screen.

  4. Posting A/c Old – This is a display field. The values of the entered Posting Account for the Document Type and Number is displayed.

  5. Posting A/c New - Click [List of Values] to select a valid Posting Account. This is a mandatory field.

Credit Note Lines

  1. Posting A/c Old – This is a display field. The entered value of the posting account at lines level is displayed.

  2. Posting A/c New – Click [List of Values] to select a valid Posting Account. This is a mandatory field.
