Enter the following to generate Write-off Request:

 

  1. Customer - Click [List of Values] to select a valid Customer. On selecting a Customer code, the short description of the Customer is displayed. This field is mandatory.

  2. Write-off Request Date – The system date is displayed as Write-off Request Date, by default. You cannot change the Write-off Request Date.

  3. Doc No. – The Document Code set for Write-off Documents in AR parameter is displayed, by default. If the specification for the Document Type states that the document number is system generated in AR Parameters, the document number is generated during transaction. When system generates document number, field is protected from entering.

  4. Outstanding Period (Range in Day) – Enter the range for number of days in numeric format of six digits maximum length. If the data is not entered, the system assumes the Outstanding Period between 0 – 999999.

  5. Outstanding Amount Range – Enter the range for the Outstanding Amounts in numeric format of eight digits maximum length.

  6. Nationality - Select the appropriate Nationality from the list. List contains three values namely, All, Citizen and Non-Citizen. “All” is displayed on the screen, by default. This field is mandatory.
