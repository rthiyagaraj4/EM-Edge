Related Field Descriptions:

  1. Invoice Document Type – Click [?] to select Invoice Document Type. This is mandatory field. The Invoice Document Types defined in the Document Type are listed.

  2. DCP Document Type – Click [?] to select DCP Document Type. This is mandatory field. The DCP Document Types defined in the Document Type are listed.

  3. Customer - Click [?] to select customer. This is mandatory field. The customers defined in the masters are listed.

  4. [Patient Details] – Click this button to invoke Patient Details dialog with Patient ID, Patient Name, Episode Name, Episode ID and Visit ID. You can select the patient ID from the dialog.

 

DCP Details

  1. Patient ID – Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter. This is mandatory field.

  2. Episode ID / Visit ID – Based on the Patient ID, the Episode ID and Visit ID are displayed after entering Patient ID.

  3. Episode Type – Based on the Episode ID, the Episode Type is displayed.

 

Pending Invoice Details

  1. [Document Details] - Click this button to invoke Document Details dialog with Document Type, Document Number, Document Date and Pending Amount. You can select the Documents Type from the dialog.

  2. Doc Type/Number – Select a Document Type and Number from [List of Values]. This is a mandatory field.

  3. Outstanding Amount – based on selected Doc Type, the outstanding amount is displayed.

  4. Pending Amount – based on selected Doc Type, the pending amount is displayed.

  5. Reason Text – Enter the reason in alphanumeric format of maximum length of 50 character. This is mandatory field

  6. Cus Code – Enter the customer code based in which the customer code description is displayed. This is mandatory field.
