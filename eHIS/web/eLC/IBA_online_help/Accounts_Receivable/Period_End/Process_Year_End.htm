Process Year End

From this function you can Process the transactions in the Accounts Receivable module. The documents have to be posted before executing this function.

This function is used to process the transactions for the complete financial year in the Accounts Receivable module. This function carries out all the functions of month end process, which are not executed using the month end process. Year end can be run only for the last accounting period. System performs certain validations before and after processing a year end.

  • Financial Year is incremented by 1.

  • Financial month is set as 1.

  • Initialization of values in Customer master.

  • All transactions pertaining to the financial year are posted to General Ledger.

  • When  interface option is “Batch”, General Ledger distributions and the export to General Ledger module has to be complete.

Year End Process

From the Period End, double-click the Process Year End function.

The Process Year End screen gets displayed.

  1. Select the check box to initialize the Cumulative Sales Amount to zero, when year-end process is completed. If the check box is cleared, Cumulative sales figure is appended.

  2. Select a value from the list to process the records. The options are “Online”, “Batch” and "No Interface". Select one of the options from the list. This indicates whether the interface method to General Ledger is “Online”, “Batch” for the next financial period. Select "No Interface", if the Accounts Receivable module is not interfaced to the General Ledger.

  3. Click [Run Process] to execute the month end process.

  4. A confirmation screen pops-up. Click [Yes] to process the month end or [No] to cancel the process.

  5. A log file with the output is generated. Click [OK] to view the log file or [Cancel] to exit from the screen.

 

You can use this function only for the last accounting period.

All the unposted or uninterfaced transactions have to be posted before execution of this function.

The credit note, debit note, adjustment, receipt, payment, invoice transactions have to be posted. If the details are not posted, you cannot process a month end.
