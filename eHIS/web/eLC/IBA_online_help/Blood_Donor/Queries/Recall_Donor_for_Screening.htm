Recall Donor for Screening

From this function you can Query the details of the donors who are recalled for screening. You can Print the screening letter and the screening list of the donor.

The Recall Donor for Screening query function populates the donor’s details for which the blood donation is found to be reactive. There is also an option to print the screening list and a recall letter sent to the donor. You can also view the Donor History and Unit details from this function.

Querying the Donors Recalled for Screening

From Query, double-click the Recall Donor for Screening function.

The Recall Donor for Screening screen gets displayed.

  1. Click the Calendar function to select a valid Donation Date range. The donation details during the specified date range get listed. The Donor ID, Donor Name, Donation Date, Contact Number and Unit Number are displayed.

  2. Enter the Batch Number to query the donor details with respect to the batch specified.

  3. Select the value from the list. The valid values are ‘Yes’, ‘No’ and ‘Both’. The value is set to ‘Both’ by default. Select 'Yes' to view the donor details for which the recall letter is printed. Select 'No' to view the donor details for whom the screening letter is not printed. If you select 'Both', the details of all the donors are displayed, irrespective of the screening letter status.

  4. Click [Execute Query] or press [F8] to view the details of the donors who are recalled for screening.

  5. The details of the donor who are recalled for screening are printed. If the records are printed, the Unit number is highlighted in  color. If the records are not printed, the Unit number is highlighted in color.

  6. Select the ‘Select?’ check box to select the individual donor record. You can either view the unit details, donor history, print screening letter or the screening list. Select the ‘Select All’ check box to select the donor records either to view the unit details, donor history, print screening letter or the screening list.

  7. Click [Donor History] to view the details of the donor. The Donor History with the Unit Details screen is invoked with the details.

  8. Click [Units Details] to view the unit details. The Unit Details screen is invoked with the details.

  9. Click [Print Recall Letter] to print the recall letter of the donor.

  10. Click [Print Recall List] to print the recall list of the donors.
