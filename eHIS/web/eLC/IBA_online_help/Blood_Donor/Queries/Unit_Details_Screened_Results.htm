Unit Details – Screened Results

From this function you can Query about the blood unit details. You can View the screening results for a specific Unit Number.

The Unit Details – Screened Results function allows you to query about the unit details. The Donor’s response to the Questionnaire, Physical Checkup details and Component details are displayed. You can view the screening result for the selected Unit Number. The components, which are added to stock, are displayed using the color. The reactive components are displayed using the   color in the ‘Screened Results’ tab.

Querying the Unit Details with screening results

From Query, double-click the Unit Details – Screened Results function.

The Unit Details – Screened Results screen gets displayed.

  1. Enter a Unit Number or click [List of Values] to query for a specific record.

  2. The Product Code and the Cancel code are displayed.

  3. Click [Execute Query] or press [F8] to query the details associated to the selected Unit Number.

OR

  1. Click [Execute Query] or press [F8] to query for all the records. You can navigate to the next and the previous record using the downward and the upward arrow keys in the keyboard. Click [Execute Query] or press [F8] to view the unit details of the respective record.

  2. The donor’s response to the questionnaire, physical checkup, screening results and the component details are displayed.

  3. Click [Donor History] to view the donor’s previous transactions in the facility. On clicking [Donor History], the Donor History with Unit Details – Confidential screen is displayed. The confidential transactions performed on the unit number along with the Donor’s previous transaction history is displayed.

  4. Click [Unit Log] to view, print or save the report output in a file.
