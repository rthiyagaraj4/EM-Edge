Donor Letter Statistics

From this function you can Query to view details on the letters sent to the donor.

The Donor Letter Statistics query function allows you query on donor letter statistics. The Donor Letter Statistics function displays the statistics on the letters sent to the donor and number of the times a specific letter has been printed. You can perform the query based on the date range during which the letters are printed for the donor. It is also possible to filter data based on the Donor ID and Letter Type. The available Letter Types are:

• Thank you Letter

• Recall for Donation

• Recall for Screening

• Counseling Letter

• Campaign Letter

You can view the total number of letters printed for each letter type, within the selected date range.

Querying the Donor Letter Statistics

From Query, double-click the Donor Letter Statistics function.

The Donor Letter Statistics screen gets displayed.

  1. Click the Calendar function to select the printed date range. The letters printed during the date range for the selected donor gets displayed.

  2. Click [Donor Lookup] to select the donor.

  3. Select the value from the list to view statistics. The value is set to ‘All’ by default. On selecting the type of letter, details on the specific letter is displayed.

  4. Click [Execute Query] or press [F8] to query the statistics.
