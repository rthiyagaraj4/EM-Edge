BD Parameters

From this function you can Define the rules that influence the functioning of Blood Donor module. You can Query to view the parameters defined.

The BD Parameters function is used to define or update the parameter details used in the Blood Donation Management module with respect to blood donation, blood unit management and stock operation. The values defined are displayed as default in the transaction screens of the module. The setups defined in this function influence the functioning of the transactions done in the module and interfacing to other modules.   

Defining BD Parameters

From Administration, double-click the BD Parameters function.

The BD Parameters screen gets displayed with the General Tab enabled.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click ‘Physical Screening’ tab to define the screening parameter details. <Related Field Description>

  3. Click [Save] to save the created records or click [Clear Record] to clear the entered data.

  4. Click [Execute Query] or press [F8] to view the parameter setup.
