Administer Report Server Fax/Mail Requests

From this function you can View the Fax and Mail requests made and Resubmit the requests.

The Administer Report Server Fax/Mail Requests function enables you to view the fax and mail requests, which are under process, which are sent and those that are in error. You can resubmit the mails and faxes, if a request is raised. The requests that are under process are highlighted in color, whereas the faxed ones are highlighted in and the error ones are highlighted in .

Administering the Report Server Fax/Mail Requests

From the Administration > Manage Services, > Process Management double-click Administer Report Server Fax/Mail Requests function.

The Administer Report Server Fax/Mail Requests screen gets displayed.

  1. The request details are displayed under the following tabs:

Fax

The Fax Tab displays the details of the faxes sent.

  1. Request Date – The request date of the fax is displayed.

  2. Priority - The priority of the fax request is displayed.

  3. Facility – The facility to which the fax request is placed gets displayed.

  4. Module ID – The Module ID related to the fax request is displayed.

  5. User ID – The ID of the user who has placed the fax request is displayed.

  6. Workstation – The workstation from where the fax request is raised is displayed.

  7. Report – The requested report type is displayed.

  8. Report File Name – The report file name is displayed.

  9. Resubmitted date – The date when the report is resubmitted is displayed.

  10. Destination – The destination to which the report is sent is displayed.

Mails

Select the Mail tab to view the details of the mails sent.

  1. Request Date – The date when the request was made for the mail is displayed.

  2. Priority - The priority assigned to the mail request is displayed.

  3. Facility – The facility to which the mail request is placed gets displayed.

  4. Module ID – The Module ID related to the mail request is displayed.

  5. User ID – The ID of the user who has placed the mail request is displayed.

  6. Workstation – The workstation from where the mail request is raised is displayed.

  7. Report – The requested report type is displayed.

  8. Report File Name – The report file name is displayed.

  9. Submitted Date - The date when the report is submitted to server gets displayed.

  10. Resubmitted date – The date when the report is resubmitted to server gets displayed.

  11. Destination – The destination printer to which the report is sent is displayed.

  1. Select a mail/fax request and click [View Report] to view the report of the mail/fax sent.

  2. Select a mail/fax request and click [View Error Log] to view the error logs of the mails/faxes sent.

  3. Select a mail/fax request that is already sent and click [Re Submit] to resubmit the mails/faxes that are sent earlier. The faxed/mailed requests are highlighted in .

  4. Select a request and click [Purge] to delete the error log after the confirmation message gets acknowledged.

  5. Click [Purge All] to delete all the queried mail/fax requests after the confirmation message is acknowledged.
