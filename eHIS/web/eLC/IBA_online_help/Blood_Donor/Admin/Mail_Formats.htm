Mail Formats

From this function you can Create the standard mail formats used in the Blood Donor module.

The Mail Formats function enables you to create mail formats to be used in the Blood Donor module. You can define standard letter formats for Donation Campaign, Donor Consent Letter, Thank You Letter etc. You can also select the variables that are to be included in the mail.

Creating Mail Formats

From the Administration, double-click the Mail Formats function.

The Mail Formats screen gets displayed.

  1. Click [List of Values] to select the required Letter ID. This is a mandatory field.

  2. Enter the body of the letter in the text box.  

  3. Right click in the text box to select the Donor Details to be included in the mail format. The listed Donor Related items are Donor ID, Donor Name, Blood Group/Rhesus, Donation Date, Next Donation Date, Hospital Name, Current Date and Donor Mailing Address.

  4. Double click the selected variable and the variable name gets displayed in the text box. When multiple variables are selected, they are separated by ‘+’ sign.

  5. Click [Save] or press [F10] to save the mail formats or click [Clear Record] to clear the entered details.
