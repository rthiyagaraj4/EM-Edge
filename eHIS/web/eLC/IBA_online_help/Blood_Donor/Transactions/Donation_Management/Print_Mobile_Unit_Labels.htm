Print Mobile Unit Labels

From this function you can Print the labels for the blood units collected during a mobile donation.

The Print Mobile Unit Labels function is used to print the labels for the blood units collected during a mobile donation camp. In Print Mobile Unit Labels function, you can select the collection or supplier centre, specify the starting number to generate the label, specify the count for each label, the number of labels required and print the unit label.

Printing Mobile Unit Labels

From Transactions > Donation Management, double-click the Print Mobile Unit Labels function.

The Print Mobile Unit Label screen gets displayed.

  1. Click [List of Values] to select the Collection/Supplier Center to print unit labels for mobile donation. The values defined using the Collection/Supplier Centre function get listed.

  2. The Starting number to print the unit labels is displayed, by default. You cannot make any changes to this number. The value increases each time the Collection/Supplier Center is selected to print the labels.

  3. Enter the Number of Labels in numeric format of four digits maximum length. The number of labels indicates the maximum number of labels that are required for the selected Collection/Supplier Center. This is a mandatory field.

  4. Enter the number of labels to be printed in numeric format of two digits maximum length. The field displays the number of labels as defined in the BD Parameters. You can change the value, if required.

  5. Click [Print Labels] to print the labels. A confirmation message ‘Process Completed Successfully’ gets displayed on completion of the task.
