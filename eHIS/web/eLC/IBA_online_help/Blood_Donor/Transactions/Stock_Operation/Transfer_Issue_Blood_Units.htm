Transfer/Issue Blood Units

From this function you can Transfer or Issue the blood units from a donor stock to the blood transfusion stock.

The Transfer/Issue Blood Units function enables you to transfer the Blood Units from the Donor Blood Stock for transfusion. You can Transfer/Issue Blood Unit(s) with/without unit requisition to the Lab Center. You can transfer or issue more than one unit simultaneously, if the units belong to the same blood group and product.

Transferring or Issuing Blood Units

From the Transactions > Stock Operation, double-click the Transfer/Issue Blood Units function.

The Transfer/Issue Blood Units screen gets displayed.

  1. Select the Unit Source from the list to transfer or issue blood units. The values displayed are Requisition and Without Requisition. If you select ‘Without Requisition’ from the Unit Source list in the Transfer/Issue Blood Unit screen, the system invokes the blood unit details screen. The list displays ‘Requisition’ by default. The values are system defined. <Related Field Description>

  2. Press [Tab] to view the blood unit request details.

    The following blood unit request details are displayed:

    1. Requisition Number – The Requisition Numbers of the blood units within the queried date range are displayed. The colors displayed against the Requisition Number indicate the status of the request. The valid statuses provided are ‘Unit Issued’-   and ‘Unit Acknowledged’.

    2. Requisition Date – The Date of Requisition to issue the blood unit from the Lab Center is displayed.

    3. Product Code – The Product Code of the requested product is displayed. The product codes are defined in Products function.

    4. Description – The Product Description of the requested product is displayed.

    5. Blood Group – The requested Blood Group of the unit is displayed.

    6. Rhesus Code - The requested Rhesus Code of the unit is displayed.

    7. Requested Quantity – The Requested Quantity of blood units against each requisition is displayed.

    8. Issued Quantity – Enter the Quantity to be issued in the text box.

  3. Double-click on the Request Number to invoke the ‘Blood Unit Details’ screen.

  4. Enter the details in the mandatory fields to view the Blood Unit Details. <Related Field Description>

  5. Select the check box against the unit number to issue the unit.

  6. Click [Issue] to deliver the blood unit or [Cancel] to cancel the transaction.
