Unit Cancellation

From this function you can Cancel a blood unit by attaching a fate code. You can Reject the blood units based on the response of the donor to the questionnaire, physical checkup and blood unit screening result.

The Unit Cancellation function enables you to cancel any blood units by attaching a fate code. You can cancel blood unit based on the donor’s questionnaire, physical checkup or screened results. When you provide Unit Number, the system displays respective donor responses to the Questionnaire, Physical checkup details, test-screening results and component details.

The system displays the ‘Barcode Unit Number Product Code’ screen based on the pre-defined setup. Scan the product or enter the Barcode Unit Number. Click [Display] to get access to the ‘Unit Cancellation’ screen, or [Cancel & Exit] to abort.

Cancelling Blood Units

From the Transactions > Blood Unit Management, double-click the Unit Cancellation function.

The Unit Cancellation function screen gets displayed.

  1. Enter the Barcode Unit Number and click [Display] to view the unit details or click [Cancel & Exit] to cancel the process. <Related Field Description>

  2. Click Donor Response to the Questionnaire tab to view the donor responses.

    The Donor Response to the Questionnaire tab displays the details of the questionnaire answered by the donor during pre-donation analysis.

    1. Question – The questions defined in the Physical/questionnaire Setup function is displayed.

    2. Response – The response entered during the pre-donation analysis is displayed.

    3. Result – The result selected during pre-donation analysis is displayed.

    4. Test Ordered – The list indicates whether any test was ordered to be conducted on the selected donor during the pre-donation analysis.

  3. Click Physical Checkup tab to view the physical checkup details. <Related Field Description>

  4. Click Screened Results tab to view the screened results of the donor. <Related Field Description>

    The following results are displayed:

    1. Test – The Test Code of the ordered Test is displayed.

    2. Result – The Test Result entered during result entry gets displayed.

    3. Accept/Reject – The list displays ‘Accepted’ or ‘Rejected’ based on the result.

    4. Comment – The Comment recorded about the pre-donation analysis gets displayed. A screen pops-up listing the code and the description. Click [Close] to exit from the window.

  5. Click Component Details tab to view the component details of the selected unit.

    The following component details are displayed:

    1. Blood Bag Lot Number – The Blood Bag Lot Number associated to the unit is displayed.

    2. Donated Component – The code and description of the Donated Component are displayed.

    3. Measured Volume – The weight of the donated component is displayed.

    4. Unit Number – The Unit Number from which the component is prepared is displayed.

    5. Sub Component – The code of the sub component prepared is displayed.

    6. Description – The Description of the sub component entered while setting up ‘Permissible Blood Components’ master is displayed.

    7. Volume – The volume of the blood recorded using Blood Unit Component Stock Entry function is displayed.

    8. Units – The Unit of the Volume is displayed.

    9. Expiry Date – The Expiry Date of the product is displayed.

  6. Enter the Cancel Code or click [List of Values] to cancel the component. The description of the selected Cancel Code is displayed. If components are prepared for the selected unit, select the cancel code from this field to cancel the components. The values defined using the Cancellations function get listed.

  7. Select the appropriate storage location for the unit. The Rejected Storage location as defined in the Administration function of this module gets displayed after the cancel code is selected.

  8. Click [Save] or press [F10] to cancel the selected unit or click [Clear Record] to cancel the transaction.
