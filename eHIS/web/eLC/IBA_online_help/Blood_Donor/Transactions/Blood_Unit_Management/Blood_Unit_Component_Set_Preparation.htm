Blood Unit Component Set Preparation

From this function you can Create a set of components from a set of blood units, which are donated in in-house, or during mobile donation.

The Blood Unit Component Set Preparation function enables you to prepare components for a set of units, based on the defined blood-bag weight rule. You can search for the set of units by specifying the donation date range or batch number. You can select either the individual unit or a set of units to prepare the component. The weight of the blood bag has to be measured and recorded. Based on the preset donated product and blood bag weight rules, you can create components.

Component set preparation is not allowed if the selected blood unit component has expired.

The status of the blood units are displayed using the following colors:

– Bleeding Time Exceeded

– Valid Combination  

- Invalid Combination  

– Expired Donated Product

– Time elapsed as per blood bag rule

Preparing Blood Unit Component Set

From Transactions > Blood Unit Management, double-click the Blood Unit Component Set Preparation function.

The Blood Unit Component Set Preparation screen gets displayed.

  1. Enter the query criteria to search for the blood units. <Related Field Description>

  2. Click [Display] to view the details of the units within the query criteria. On clicking [Display], the cursor will be active in the Barcode Unit Number field to record the Barcode number automatically on scanning. After scanning the unit number, the cursor moves to the Weight field. On entering the weight, cursor moves back to the Barcode Unit Number field to enable scanning the next unit number.

    The following details are displayed:

     

    1. Unit Number - The Unit Number gets displayed based on the values entered for search criteria.

    2. Component – The respective Component of the unit is displayed.

    3. Units – The unit of measurement for the selected weight is displayed.

    4. Preparation Date – The preparation date of the blood unit gets displayed.

  3. Enter the Weight of the blood unit. This indicates the volume of the unit that is necessary to prepare the component.

  4. Click [Select Components] to invoke Select Components screen. <Related Field Description>

  5. Select the check box to validate a particular component for component set preparation. Select the check box to validate all the components for component set preparation.

  6. Click [Validate] to validate the selected blood units/components or click [Close] to return to the Blood Component Set Preparation screen.

  7. On validating the components, the color codes describing the status of the unit are displayed against the validated components.

  8. Double-click the unit number to view the details of the unit in the frame. <Related Field Description>

  9. Click [Create Components] to create components for the selected blood unit. The product weight and the measured weight details are mandatory to create the components.

A warning message ‘Units are expired’ is displayed, if the selected units are expired.

  1. Click [Cancel Unit] to cancel the transactions on the selected blood unit.
