Blood Unit Component Stock Entry

From this function you can Record the volume of the blood unit and Add the blood components to the stock.

The Blood Unit Component Stock Entry function is used to record the volume of the units and add components to the stock for which the pre-donation analysis and screening tests are complete. The volume has to be recorded and only then, the blood components can be added to the stock. You can add the Blood Unit Components to the stock individually or select all the products at once.

The products that have reached the expiry date cannot be added to stock.

You have to record the volume first to include the unit in the stock. Enter the volume of the unit first and save it first and then record the unit to stock.

Recording the Blood Unit Components to the Stock

From the Transactions > Blood Unit Management, double-click the Blood Unit Component Stock Entry function.

The Blood Unit Component Stock Entry screen gets displayed.

  1. Enter the query criteria to select the Unit details. <Related Field Description>

  2. Select a value from the system-defined list. The values in the list are – Record Volume and Record Stock. Select ‘Record Volume’ from the list to record the volume. The units for which the volume can be recorded are displayed. Select ‘Record Stock’ to add the unit to the Stock. The units, which can be added to stock, are displayed. You can add a unit to the stock only after recording the volume. The listed values are Record Volume and Record Stock.

  3. Click [Display] to view a list of units within the specified query criteria. <Related Field Description>

  4. On clicking [Display], the cursor will be active in the Barcode Unit Number field to  record the Barcode number automatically on scanning. After scanning the unit number, the cursor moves to the Volume field. On entering the volume, cursor moves back to the Barcode Unit Number field to enable scanning the next unit number.

  5. Double-click the selected unit to view the component details. The unit details get displayed on the right hand side frame. <Related Field Description>

  6. Click [Record Volume] to Record the volume of the units. When the blood unit is scanned, the cursor moves to the Volume field, by default. After the volume is recorded, the curser moves back to the Barcode unit number field for scanning the next blood unit.

  7. Click [<<] to clear the unit details on the right hand frame and select any other unit number to record the volume.

  8. Select the Action as ‘Record Stock’ and click [Display].

  9. Select the unit by selecting the check box against a unit number.  

  10. Double-click the selected unit to view the component details and record the stock.

  11. Select the value from ‘Add to Stock?’ list to add the selected component to the stock.

  12. Click [Record Stock] to add the selected unit component to the stock.

  13. On clicking [Record Stock], the system invokes a warning message. Click [OK] to add the component to the stock or click [Cancel] to cancel the transaction.

  14. Click [Add all selected Units to Stock] to add all the units and their components to the stock.
