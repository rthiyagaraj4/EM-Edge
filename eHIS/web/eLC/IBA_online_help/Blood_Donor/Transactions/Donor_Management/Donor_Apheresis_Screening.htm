Donor Apheresis Screening

From this function you can Send the apheresis blood unit for screening. You can View and Validate the results after screening and Accept or Reject the donor based on the results.

The Donor Apheresis Screening function enables you to send the Apheresis blood for screening, view and validate the results obtained after screening the blood. The Apheresis Screening tests are displayed as per the setup made in ‘Screening Test Setup’ function. The screened results are displayed by default in Apheresis Screening Results tab, once the results are entered and released from the Laboratory. You can accept or reject the donor based on the lab results.

If the Tests are rejected, the test is highlighted with  color. If the Specimen is rejected from the lab, it is highlighted with  color.

If the donors are awaiting acceptance/deferment, a color code is displayed to the Donor ID.  

Donor Apheresis Screening

From Transactions > Blood Donor, double-click the Donor Apheresis Screening function.

The Donor Apheresis Screening screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [Antibodies] to view antibodies associated to the donor. This button gets enabled, if antibody details are recorded for the donor.

  3. Click [Antigens] to view antigens associated to the donor. This button gets enabled, if antigen details are recorded for the donor.

  4. Click [Donor Notes] to add donor notes. This button gets enabled, if donor notes are recorded for the donor.

  5. Click Apheresis Screening Tests tab to send the blood unit for screening.

    This tab displays tests to be done for the unit:

    1. Test Code/Description – On selecting the Donor ID for apheresis screening, the system displays a set of tests defined in the ‘Screening Test Setup’ function.

    2. Result Awaiting – The result of the test is displayed whether it is awaiting the result or not. The valid values are ‘Yes’ and ‘No’.

    3. [Send for Screening] - Click [Send for Screening] to send the blood unit to the lab for screening.

  6. Click Apheresis Screening Results to validate and accept or reject the test result.

    The following details are displayed on clicking the Apheresis Screening Results tab:

    1. Test - The Test Code of the ordered test is displayed.

    2. Numeric Result – The Numeric Result of the test is displayed (if any, as per the ‘Test Codes’ setup made in Lab module).

    3. Test Result – The Test Result reported from the lab is displayed. The values displayed are ‘Accept’ and ‘Reject’.

    4. [Comment] – Click [Comment] to view the comments entered during the test.

    5. [Validate]  - Click [Validate] to validate the test result.

    6. [Accept/Reject] - Click [Accept/Reject] to accept or reject the results from the lab.

    7.  [View Lab Results] - Click [View Lab Results] to view the result details.

  7. Click [Commit] or press [F10] to save the record or click [Clear Record] to clear the entered data.

  8. Click [Execute Query] or press [F8] to view the existing records.
