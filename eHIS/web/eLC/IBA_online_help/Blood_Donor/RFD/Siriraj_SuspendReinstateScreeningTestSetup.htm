Enter the following details:

  1. Test Code – Click [List of Values] to select the test code. The list displays the test codes defined in Test Codes function in Laboratory module.

 

Applicable for

  1. Blood Bag – Select the check box, if the selected test code is applicable for Blood Bag screening.

  2. Apheresis – Select the check box, if the selected test code is applicable for Apheresis screening. The setup made reflects in the Donor Apheresis Screening function.

  3. Deferred Screening – Select the check box, if the selected test code is applicable for Deferred Screening. The setup made reflects in the Deferred Blood Donor function.

  4. Test Code – If the test is a panel test, select associated individual test and respective type of answer for selected tests.

  5. Additional Test – Select the check box to identify the test as an additional test. The blood units are sent for additional tests through the ‘Send for Additional Screening Tests’ function when the test is recorded as an additional test. You cannot send the blood units for additional tests, if the Apheresis and Deferred Screening check boxes are selected.

  6. Performing Facility – The logged in Facility is displayed, by default. Select the Performing Facility from the drop-down list to transfer the order for the blood units. The screening test is performed at the selected facility.

  7. Type of Answer – Based on the individual test codes selected, select the respective type of answers from the list. The valid values are ‘Numeric’ and ‘List of Values’.

 

Validation for Numeric

If the type of answer selected for a test code is numeric, the following fields get enabled:

  1. Low Value – Enter the lowest value acceptable for this test.

  2. High Value – Enter the highest value accepted for this test.

  3. Units – Select the units for the low and high value from the list of values. The list displays the values defined in Units of Measurement function of Application Masters module.

  4. Action – Select either the action ‘Accepted’ or ‘Rejected’ for the test. The values are system generated.

Click [<<2<<] to go back to the previous list.

 

Validation for List of Values

  1. List of Values – Enter the validation for List of Values or select it from the list of values provided.

  2. Action – Select either the action ‘Accepted’ or ‘Rejected’ for the test. The values are system generated.

Click [<<3<<] to go back to the previous list.
