Enter the following details to query and select the unit details:

 

  1. Unit Source – Select the blood unit source from the Unit Source list. The listed values are In-house, Mobile and All. All is displayed, by default.

  2. Component Prepared – Click [List of Values] to select the components to add the component to stock. The units whose components are prepared using the Blood Unit Component Set Preparation function get listed.

  3. Donation [From-To] – Enter the Donation date range or click the Calendar function to select the date. The components prepared within the date range get listed.

  4. Batch Number – Click [List of Values] to select the Batch Number. The Batch Number selected during pre-donation analysis is displayed. The Batch Numbers are created in the Create Batch function.

  5. Barcode Unit Number – Scan the unit or enter the valid Barcode Unit Number in alphanumeric format of 30 characters maximum length to view the details of the unit.
