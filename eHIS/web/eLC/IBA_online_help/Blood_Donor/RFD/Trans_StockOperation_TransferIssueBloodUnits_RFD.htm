The following details are displayed:

 

  1. Product Code – The Product Code of the product requested is displayed. The product codes are defined in Products masters function.

  2. Blood Group – The Blood Group of the unit is displayed.

  3. Rhesus Code - The Rhesus Code of the unit is displayed.

  4. Requested Quantity – The Requested Quantity of blood units against each requisition is displayed.

  5. Issued Quantity – The Issued Quantity of blood units against each requisition is displayed.

  6. Barcode Unit - Product Code – Scan the Bar code or enter Bar code unit number of the blood bag or unit in alphanumeric format of 30 characters maximum length. The bar code and the unit number are displayed automatically while scanning the blood bag.

 

Blood Unit Details

  1. Ordering Facility ID – Select the Ordering Facility from the drop-down list. The blood units are to be transferred to the selected facility. The logged in facility is displayed by default.

  2. Barcode Unit - Product Code – Enter the Barcode Unit and Product Code in alphanumeric format.

  3. Unit Number – The Unit Number of the scanned or entered unit(s) is displayed.

  4. Product Code – The Product Code of the product to which the unit belongs is displayed.

  5. Description – The Product Description of the product is displayed.

  6. Blood Group – The Blood Group of the unit is displayed.

  7. Rhesus Code – The Rhesus Code of the blood group is displayed.

  8. Volume – The volume of the blood unit is displayed.

  9. Volume Units – The unit of volume of the blood group is displayed.

  10. Expiry Date – The expiry date of the blood unit is displayed.
