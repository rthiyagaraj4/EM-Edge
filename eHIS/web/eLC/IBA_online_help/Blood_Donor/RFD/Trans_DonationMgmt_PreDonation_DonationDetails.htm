Enter the following donation details:

 

In-house/Mobile Details

If the age of the donor falls within the exception age, fields Consent By and Consent Date are displayed.

  1. Consent By - Enter the name of the parent/guardian, who signed the Consent Age Exception form in alphanumeric format of 30 characters maximum length. This field is displayed only if the donor’s age falls in the Age Exception Category. This is a mandatory field.

  2. Consent Date - The current date and time is displayed. You can change the date and time, if needed. This field is displayed only if the donor’s age falls in the Age Exception Category. This is a mandatory field.

  3. Batch Number – Enter a valid batch number or click [List of Values] to select the batch number.

  4. Donation In-House – Select the check box if the donation is an in-house donation. Clear the check box, if the donation is conducted in a mobile drive.

  5. Physical Location - Enter the Physical Location of the donation centre or click [List of Values] to select the location code and name. The list of values displays the values defined using the ‘Application Masters’ module.

  6. Mobile Code – Select the Mobile Code if it is a mobile donation. This field gets disabled, if you select the check box ‘Donation In-house?’. The values defined using the Mobile Units function get displayed.

  7. Drive Date – Click the Calendar function to select the drive date, if the donation is a mobile donation. The drive date is set to the current date, by default.

  8. Group Donation Y/N? – Select the check box, if the donation is a Group Donation. Clear the check box, if the donation is not a group donation.

  9. Donation Group – Click [List of Values] to select the Donation Group code and name. This field is enabled if the 'Group Donation Y/N' check box is selected. The list displays the values defined in the Donation Groups masters.

 

Donation

  1. Type – Select the Type from list. The available donation types are:

  2. Autologous – Select Autologous, if the donor donates blood for himself or herself. If the selected type is Autologous, enter the patient ID of the donor.

  3. Allogenic – Select Allogenic, if the donor donates blood for other patients.

  4. Blood Letting – Select Blood Letteing, if the  donated unit is marked for cancellation and is not available for any transaction after the donation. Cancellation code is applied against the unit based on the setup defined in the ‘BD Parameter’ function.

  1. Process – Select the process from the pre-defined list. The list displays ‘Voluntary’, ‘Replacement’, ‘Campaign’ and ‘Directed’. If the selected type is allogenic, you can select any of the following processes:

  2. Voluntary – Select Voluntary, if the donor comes to a blood bank to donate blood voluntarily.

  3. Replacement – Select Replacement, if a donor, preferably patient’s relative/friend, is donating blood reciprocally to the blood given to the patient. Enter the Patient ID on selecting this donation type.

  4. Campaign – Select Campaign, if the donation of blood happens in a campaign raised by the blood bank authorities, or any other organizations.

  5. Directed – Select Directed, if blood is donated by a patient’s close relative. Enter the Patient ID on selecting this donation type.
