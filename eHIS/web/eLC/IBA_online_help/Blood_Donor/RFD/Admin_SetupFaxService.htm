Enter the following details:

 

  1. ID – Enter the Fax ID in alphanumeric format of eight characters maximum length. This is a mandatory field.

  2. Name - Enter the Fax Service name in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Routing Mode – Select the Routing Mode from the list. The listed values are By Mail and By Printer. This is a mandatory field. When you select the value as By Mail, the field name changes to Email Profile and when you select the value as By Printer field name changes to Print ID.

  4. Email User Profile Option - Select the Email User Profile Option from the list. The listed values are Individual and Common. This is a mandatory field. When you select Individual, you can send mail to only one person. This field is disabled, if the value selected in Routing Mode is By Email.

  5. Fax Number Prefix – Enter the Fax Number Prefix in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  6. Company Name Prefix – Enter the Company Name Prefix in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  7. Fax Server IP – Enter the Fax Server IP Address in 1.1.1.1 format not exceeding 15 digits. This is a mandatory field.

  8. Product – The Product name is displayed.

  9. Printer ID or Email Profile – The Email profile created in the Email Profile for users is displayed.

Note: If the routing mode is selected as ‘By Printer’, you can enter only the Printer ID and the Remarks. Enter the Printer ID in alphanumeric format of 15 characters maximum length.

  1. Recipient Name Prefix – Enter the Recipient Name Prefix in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  2. Domain Name – Enter the Domain Name in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  3. Remarks – Enter the remarks in alphanumeric format of 200 characters maximum length.

  4. Status – Select the check box to enable the Fax Service. If the check box is cleared the defined Fax Service will be disabled.
