The details of the selected unit is displayed on the right-hand side frame:

 

  1. Product Code – The Product Code of the unit is displayed.

  2. Description – The description of the product gets displayed.

  3. Volume - Enter the volume of the product in numeric format of five digits maximum length. It indicates the quantity of the selected product.

  4. Add to Stock? – This field is disabled while you are recording the volume. The valid values are ‘Yes’ and ‘No’.

If you have selected the value in 'Action' field as 'Record Volume', this field is disabled. Select the Record Stock value after recording the volume to enable this field.

  1. Storage Location – The Storage Location of the unit is displayed.

  2. Expiry Date - The expiry date of the unit defined in the ‘Products’ function is displayed.
