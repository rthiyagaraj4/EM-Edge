Enter the following to record the Counseling details:

 

Initial Blood Typing

  1. Blood Group – Click [List of Values] to select the Blood Group of the donor. The values defined using the Blood Group function gets displayed.

  2. Rhesus – Click [List of Values] to select the Rhesus factor of the donor. The values defined using the Rhesus function get listed.

  3. Blood Typing Done By - The name of the logged in user gets displayed, by default. You can change the name of the practitioner, if you need. This field indicates the name of the practitioner who has done the blood typing.

  4. Blood Typing Done On – The current system date and time is displayed, by default. You can change the date and time if you need. This field indicates the date and time when the blood typing was done.

  5. Venepuncture Site - Select the Venepuncture site from the list of values. The listed values are created in the Venepuncture function.

  6. Rare Blood Group – Select the check box,  if the blood group of the donor is a rare group. Clear the check box, if the blood group of the donor is not a rare group.

 

Donation Details

  1. Type of Bleeding – Select the type of donation form the list. The listed values are ‘Whole Blood’ or ‘Apheresis’.

  2. Blood Component – Click [List of Values] to select the blood component. The values defined using the Products function get listed.

  3. Blood Bag Lot Number – Click [List of Values] to select the Blood Bag or Lot number. The values defined using the Blood Bags function get listed.

  4. Blood Bag – Click [List of Values] to select the Blood Bag. The values defined using the Blood Bags function get listed.

  5. Unit Number – Enter a valid Unit Number. The Unit Number is generated automatically, if ‘Generate Unit No’ check box is selected while defining the ‘Products’ or ‘Collection Centre/Supplier’ masters or in the ‘BD Parameters’. This rule takes precedence in checking the setup. First, the system checks if the set up is defined in Collection Centre/Supplier, then in Products and finally in BD Parameters. If the check box is not selected in the three functions specified, the unit number is not generated automatically.

  6. [Risk Code] – Click [Risk Code] to add donor risk details. The values defined using the  Risks function get listed.

  7. Counseling Done By – The name of the logged in user gets displayed, by default. You can change the name of the practitioner, if you need. This field indicates the name of the practitioner who has done the counseling of the donor.

  8. Counseling Done On – The current system date and time is displayed, by default. You can change the date and time if you need. This field indicates the date and time when the counseling was done.
