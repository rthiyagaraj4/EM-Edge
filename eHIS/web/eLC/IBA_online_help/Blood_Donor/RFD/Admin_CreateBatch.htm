Enter the following details:

 

  1. Batch Status – Select the batch status from the list. The listed values are ‘All’, ‘Open’ and ‘Closed’.

  2. [Display] – Click [Display] to show the batch details based on the selected search criteria.  

The Reactive Unit Numbers are highlighted with the  colour indicator.

  1. Batch Number – Enter the Batch Number in alphanumeric format of 15 characters maximum length. The Batch Numbers are listed, if you query the details.

 

Valid Period

  1. Date From – Click Calendar function to select the Start date of the Batch. The current time is displayed on selecting the date. You can modify the time as per the requirement.

  2. Valid To – Click Calendar function to select the date until when the batch is valid. The current time is recorded on selecting the date. You can modify the time as per the requirement.

  3. Status – Select the Status from the list. The system-defined values are ‘Open’ and ‘Closed’.

  4. Donation Count – The Number of Donations added to the batch is displayed. The system calculates the number of donations done in the selected batch automatically.
