Enter the following details to select the product:

 

  1. Unit Number – Enter the Unit number or click [List of Values] to select the unit number.

  2. Blood Group – The Blood Group of the unit recorded in the Pre Donation function is displayed.

  3. Rhesus – The Rhesus factor of the blood group recorded in the Pre Donation function is displayed.

  4. Volume – The Volume of the unit recorded in the Record Donation function is displayed.

  5. Donation Date – The Donation date of the unit recorded in the Record Donation function is displayed.

  6. Date of Expiry – The Expiry Date of the Unit recorded in the Products function is displayed.

  7. Dispose Unit – Select the check box to dispose the blood bag. If you do not dispose the blood bag, while splitting it, there will be an empty blood bag in the storage location with the same unit number. Clear the check box, if the blood bag is not to be disposed.

  8. Dispose Code – Enter the Dispose Code or click [List of Values] to select it. The values defined using the Fate Code function get listed. This field is enabled only when the ‘Dispose Unit’ check box is selected.
