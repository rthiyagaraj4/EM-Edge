Online Reports

 

Enter the following details to define Print Routing for Online/Generated Reports:

 

Online Reports tab allows you to define print routing for the online reports.

  1. Print Name – Enter the Print Report name in alphanumeric format of 60 characters maximum length or click [List of Values] to select the Print Name.

  2. Module ID – Enter the Module ID in alphanumeric format of two characters maximum length.

  3. Facility Based? – Select the check box to change the online print check box to facility based.

  4. Report Group – Enter the report group in alphanumeric format of 30 characters maximum length, or select it from [List of Values].

  5. Transaction Based? – Select the check box to change the online print check box transaction based.

 

List of Destination Printers (All Facilities)

  1. Facility ID – Select the value from the list. The Unit of the healthcare organization that functions as part of the site is defined as facility ID.

  2. Dest Type – Select the value from the list. The list items are Department, Clinic and Ward.

  3. Dest Code – Enter the Destination Code in alphanumeric format of six characters maximum length. Criteria for destination code can be, for Department - Operating Department Code, for Ward - Ward Code and for Clinic - Clinic Code. This is a mandatory field.

  4. Description – The Description of the selected destination code is displayed on selecting the Destination Code.

  5. WS No – Enter the Workstation Number in alphanumeric format of 15 characters maximum length. When the report is routed from the selected workstation, it overrides any other setup done for the destination report combination.

  6. Copies – Enter the number of copies in numeric format of two digits maximum length. This is a mandatory filed.

  7. 1st Printer – Enter the first printer name in alphanumeric format of 20 characters maximum length. This is a mandatory field.

  8. Start Time – Enter the starting time of first printer in hh:mm format.

  9. 2nd Printer - Enter the second printer name in alphanumeric format of 20 characters maximum length. This is applicable only when shift system is available.

  10. Start Time – Enter the starting time of second printer in hh:mm format.

  11. 3rd Printer - Enter the third printer name in alphanumeric format of 20 characters maximum length. This is applicable only when shift system is available.

  12. Start Time – Enter the starting time of third printer in hh:mm format.
