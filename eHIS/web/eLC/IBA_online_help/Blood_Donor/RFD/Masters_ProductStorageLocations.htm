Enter the following details:

 

  1. Product Code – Enter a valid Product Code or click [List of Values] to select the Product Code. The values defined using the Products function get listed. This is a mandatory field.

  2. Blood Group – Enter a valid Blood Group or click [List of Values] to select the Blood Group. The list displays the values defined using the Blood Group function. This is a mandatory field.

  3. Rhesus – Enter a valid rhesus factor or click [List of Values] to select the Rhesus factor code. The list displays values using the Rhesus function. This is a mandatory field.

  4. Event – Select the Event Type to associate the storage location. The system defined values are ‘Record Bleeding/Mobile’, ‘Prepare Blood Components’, ‘Cancel Dispose Units’, ‘Blood Unit accept into Stock’ and ‘Accepted Blood Unit not in Stock’. When you associate a storage location with any of these scenarios, the blood is stored in that storage location after the respective process is over. This is a mandatory field.

  5. Storage Location Code – Enter a valid storage location code or click [List of Values] to select a storage location. The values defined using the function Storage Location get listed.

  6. Description – The description for the storage location gets displayed based on the Storage Location code selected.

  7. Default Location Y/N – Select the check box to specify whether the Storage Location is to be set as the default storage location. Clear the check box, if the Storage Location is not to be set as the default storage location.
