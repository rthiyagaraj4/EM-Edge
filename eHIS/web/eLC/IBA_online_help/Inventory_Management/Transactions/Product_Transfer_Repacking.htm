Product Transfer/Repacking

From this function you can Transfer quantities of similar Items with different Unit of Measures.

You can Transfer quantities of one Item to another, in the Stores, wherever stock is available for the ‘From’ Item. This function is applicable only for Stock Items. The Product Transfer can be performed only for a single batch at a time.

Transferring Quantities of Similar Items

From the ST Transactions > Transfer, select the Product Transfer/Repacking function.

The Product Transfer/Repacking screen gets displayed.

  1. Modify Document Details screen is displayed by default or click ‘Document’ hyperlink to modify the document details.

  2. On entering the Document Details, click [OK] to Transfer the Quantities of similar Items or [Reset] to clear the entered data.

  3. Click [Request] to invoke the Product Search Request screen. You can view the Product Transfer/Repacking Request Raised against the selected Store.

    The following details are displayed in the Product Transfer Request search screen:

     

    1. Req Store – The Store selected in the Document header section of the transaction is displayed and disabled.

    2. Req No – Enter the required request number to search for the request in numeric format of eight digits maximum length the requests raised against the selected Store.

    3. [Search] - Click [Search] to display all the requests raised against the selected Store in ‘Product Transfer/Repacking Request’ function.

    4. Click ‘ Workstation’ icon to generate the Product Transfer Request worksheet.

      1. The Report Options screen gets displayed.

      2. Select ‘Report Option’ to generate the worksheet.

      3. Click [OK] to generate the Product Transfer Request worksheet and spool it in the file on the print server or [Cancel] to cancel the action.

    5. Select the check box against the required Item.

    6. Click [Close] to exit from the ‘Product Search Request’ screen.

  4. Click [Delete] to delete the selected the Item.

    On clicking [Delete], the ‘Confirm Delete’ dialog is displayed. Click [Yes] to Delete the Item issue details or [No] to cancel the process. On selecting the ‘Delete’ check box, ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.

  5. Select the ‘Prepare Location’ from the list. The Prepare Locations created in the function ‘Prepare Location’ are listed.

  6. On selecting the Product Transfer Request, the request details are displayed in the ‘Transfer From’, ‘Transfer To’ and ‘Remarks’ frame.

  7. Enter the details in the mandatory (*) fields. <Related Field Description>

  8. Click [Batch Search] to view the Batch Details. Select the check box to select the Batch and click [Close] to exit from the batch search screen.

  9. On selecting the Batch, the Batch details are displayed in the ‘Transfer From’ and ‘Transfer To’ frame.

  10. Click [Apply] to finalize the Item transfer details or [Reset] to clear the entered data.

You can perform ‘Product Transfer/Repacking’ on an already raised request or you can directly perform a ‘Product Transfer/Repacking’ without a request.
