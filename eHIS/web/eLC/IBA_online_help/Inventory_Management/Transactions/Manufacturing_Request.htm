Manufacturing Request

From this function you can raise a Request for a Manufacturing Item.

You can specify the Prepare Location, Type of Preparation, Preparation Mode and the Quantity to be prepared. The logged in user should have access to the Store storing the Manufacturing Items.

Raising Request For a Manufacturing Item

From the ST Transactions > Receipt, select the Manufacturing Request function.

The Manufacturing Request screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [Search] to search the pending manufacturing request.

    1. Select ‘Delete’ check box to delete the record from pending manufacturing list.

    2. Click ‘Manufacturing Item’ to modify the manufacturing request details.

    3. Click [Modify] to modify the changes or [Cancel] to reset the changes.

  3. Enter the details in the mandatory (*) fields to request for a Manufacturing Item. <Related Field Description>

  4. Click [Add] to add the Items to request and the selected Item is displayed in the top half of the screen.

  5. Click [Modify] to modify the changes or [Cancel] to reset the changes.

  6. Select ‘Delete’ check box to delete the request.

  7. Click [Apply] to generate the request or [Reset] to clear the entered data.
