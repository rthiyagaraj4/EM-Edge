Change Bin Location

From this function you can Record the movement of Items within the Bin Locations.

Changing Bin Location

From the ST Transactions > Stock Adjustment, select the Change Bin Location function.

The Change Bin Location screen gets displayed.

  1. The Modify Document Details screen is displayed, by default or click ‘Document’ hyperlink to modify the document details.

    Enter the following details to modify the required document details:

     

    1. Doc Type – Select the required document type from the list. The documents associated to the transaction ‘Stock Adjustment’ are displayed in the list. The Document Type, as set in ‘ST Facility Parameter’ is displayed, by default. This is a mandatory field.

    2. Doc No - The Document Number generation depends on the selected Document type. If the Document Number has to be generated manually for the selected Document Type, you are informed. Enter the Document Number in numeric format of eight digits maximum length. If the system generates the number, you are not allowed to enter the field value. This is a mandatory field.

    3. Doc Date – The system date is displayed, by default. You can select the required date using the Calendar function. The entered date must not be less than the Processing month and year set in ‘ST facility Parameter’ and greater than current system date. This is a mandatory field.

    4. Doc Ref - Enter the Document Reference in alphanumeric format of 15 characters maximum length.

    5. Store – Select the Store from the list. Only those stores are displayed for which you have access and ‘Change Bin’ transaction function applicable. This is a mandatory field.

    6. Item Class - Select the required Item Class from the list.

  2. On entering the Document Details, click [OK] to record movement of Items within the Bin Locations or [Reset] to clear the entered data.

  3. Enter the Item in the lower half of the screen to record movement of Items within the Bin Location in alphanumeric format of 60 characters maximum length or select the Item using [?] function. This is a mandatory field.

  4. On selecting the Item, the Unit of Measure is displayed against the Item field.

  5. Select the Bin Location from the list. The Bin Locations associated to the selected Item and Store is displayed. This is a mandatory field.

  6. Click [Batch Search] to view the Batch details. The selected Item Batch details are displayed. If the batch is expired, notified, suspended or reinstated, a unique indicator is displayed. Click on the indicator to view the Batch History Details. Select the check box to select the Batch and click [Close] to exit from the batch search screen.

  7. On selecting the batch, the Item batch details are displayed in the top half of the screen with unique color indicator.

  8. Click Item hyperlink to modify the Bin Location and Quantity. Click [Modify] to save the changes or [Cancel] to reset the changes.

  9. Click [Apply] to record the movement of Items or [Reset] to clear the entered data.

  10. Select the ‘Delete’ check box and click [Apply] to delete the record.

  11. The ‘Confirm Printing’ dialog is displayed based on the attribute set for the Document Type. Click [Yes] to print the movement of Items within the Bin Locations or [No] to cancel the action.
