Enter the following details:

  1. Item – Select the Item from the list using [?] function. This is a mandatory field. On selecting the item, the Unit of Measure, Conversion Factor and the Item’s year to date Issue Quantity is displayed. If the Item is a Stock Item, the Stock Item check box is selected and disabled.

  2. UOM - Unit of Measurement.

  3. Stock Item - If the Item is a Stock Item, the Stock Item check box is selected and disabled.

  4. Remarks – Remarks about the stock item. Click [?] to select the pre-defined remarks from the Remarks dialog.

  5. Req UOM – Select the required Request UOM from the list. The UOMs are displayed based on the ‘Req By Store’ value. On selecting the required UOM the corresponding conversion factor is displayed against it.

  6. Req UOM Qty – Enter the Request UOM Quantity required by the ‘Req On Store’ in numeric format of 10 digits maximum length. This is a mandatory field.

  7. Req Qty - The entered Request UOM quantity is displayed and disabled.

  8. Issue UOM – Select the required Issue UOM from the list. The UOMs are displayed based on the ‘Req On Store’ value. On selecting the required UOM, the corresponding conversion factor is displayed against it.

  9. Issue UOM Quantity – Enter the Issue UOM Quantity in numeric format of 10 digits maximum length. This is a mandatory field.

  10. Authorize Quantity – The quantity authorized for issue by the ‘Req By Store’ is displayed in numeric format and this field is disabled. This is a mandatory field.

  11. Pend Req Qty – The Item’s pending request quantity of selected request document is displayed.

  12. Pack Size – The current pack size defined in the Item function is displayed.

  13. YTD Iss Qty – Displays the year-to-date of the issue quantity.

  14. View Batch Details – Click View Batch details to view the quantity allotted for each batch.

  15. Add – Adds the details to the upper part of the screen.

  16. Cancel – Cancels the selected item details.

  17. Stock Level – Click to view the Item Stock details.
