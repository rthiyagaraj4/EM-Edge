Enter the following report criteria to generate a report on all the items, which require to be procured:

  1. Store Code [From-To] - Enter the Store Code range for which the report is to be generated in alphanumeric format of six characters maximum length or select the Store code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected store code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ store code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  2. Item Code [From-To] - Enter the Item Code range for which the report is to be generated in alphanumeric format of 20 characters maximum length or select the Item Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  3. Item Class [From-To] - Enter the Item Class range for which the report is to be generated in alphanumeric format of four characters maximum length or select the Item Class using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ item class code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  4. Consumption Period – Enter the Consumption Period in numeric format of two digits maximum length. This period is used to arrive at the ‘Average Consumption Per Day’. The average consumption per day is computed by the system using the following formula. This field is mandatory.

                                                                      Consumption for the last <n> months

Average Consumption Per Day =    ----------------------------------------------

                                                                             No. of Consumed months * 30

  1. Lead Days - Enter the Lead Days in numeric format of four digits maximum length. The entered Lead Days are automatically updated in the Item function on saving the record. The Reorder Level for an Item is computed based on the Lead Days entered as depicted.

Re order Level = Min. Stock Level + (Avg. Consumption Per Day * Lead Days)

  1. Min Stock Days - Enter Minimum Stock Days in numeric format of four digits maximum length. The Minimum Stock Level for an Item is computed based on the Minimum Stock Days and Average Consumption per Day as depicted.

Min. Stock Level = Average Consumption Per Day * Min. Stock Days

  1.  % of Min Stock - Enter the percentage of minimum stock in numeric format of four digits maximum length. The Reorder Quantity for an Item is computed based on the Reorder Level, Percentage of Minimum Stock and Minimum Stock Level as depicted.

Re Order Qty  = Re Order Level + (% of Min stock * Min. Stock Level)

  1. Report Order – Select the required Report Order from the list. The listed values are ‘All Items’, ‘Item Code’ and ‘Short Description’. The value ‘All Items’ is displayed, by default.

  2. Include Stock Issues – Select the check box to consider stock issues while recalculating the stock level.
