Enter the following details to Generate request for Refill of Items:

  1. Doc Type – Select the required request Document Type from the list. The documents associated to the transaction ‘Requisition Issues’ are displayed in the list. The Document Type, as set in ‘ST Facility Parameter’ is displayed, by default. This is a mandatory field.

  2. Doc No. - The Document Number generation depends on the selected Document Type. If the Document Number has to be manually generated for the selected Document Type, you are informed. Enter the Document Number in numeric format of eight digits maximum length. If the system generates the number, you are not allowed to enter the field value.

  3. Doc Date - The system date is displayed, by default. You can enter the date in date format of dd/mm/yyyy or select the required date using the Calendar function. This is a mandatory field.

  4. Req By Store - Select the Requested by Store from the list. Only those Stores are displayed for which you have access and ‘Requisition’ transaction function applicable. This is a mandatory field.

  5. Req On Store - Select the Requested on Store from the list. Only those Stores are displayed for which you have access and ‘Requisition Issues’ transaction function applicable. This is a mandatory field.

  6. Doc Ref – Enter the Document Reference in alphanumeric format of 15 characters maximum length.

  7. Effective Date – The current system date is displayed as the effective date. The system validates and generates a list of Items for which request can be generated against this date.

  8. Item Class – Select the required Item Class from the list. The Item Class associated to the selected Requested By store and Requested On Store is displayed in the list.

  9. Consignment – Select the required value for consignment from the list. The listed values are ‘All’, ‘Consignment’ and ‘Non-Consignment’. The value ‘All’ is displayed, by default.

  10. Item Analysis 1 – Select the required value for Item Analysis 1 from the list.

  11. Item Analysis 2 - Select the required value for Item Analysis 2 from the list.

  12. Item Analysis 3 - Select the required value for Item Analysis 3 from the list.

  13. Selection – Select the required value for selection to generate the request from the list. The listed values are ‘Reorderable Items’, ‘All Items’, 'Stock Item' and 'Non-Stock Item'. The value ‘Reorderable Items’ is displayed, by default.

  14. Item Code [From-To] - Enter the required Item Code in alphanumeric format of 20 characters maximum length or select the Item using [?] function. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both the values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  15. Alpha Code [From-To] - Enter the required Alpha Code in alphanumeric format of 10 characters maximum length. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Alpha Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Alpha Code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.
