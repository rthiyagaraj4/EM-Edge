Enter the required document details by selecting the following:

  1. Doc Type – Select the required Document Type from the list. The documents associated to the transaction ‘Opening Balance’ are displayed in the list. The Document Type, as set in ‘ST Facility Parameter’ is displayed, by default. This is a mandatory field.

  2. Doc No. - The Document Number generation depends on the selected Document Type. If the Document Number has to be manually generated for the selected Document Type, you are informed. Enter the Document Number in numeric format of eight digits maximum length. If the system generates the number, you are not allowed to enter the field value. This is a mandatory field.

  3. Doc Date - The system date is displayed, by default. You can change the document date by using the Calendar function or enter the date in date format of dd/mm/yyyy. You can modify the date only if the ‘Doc Date Modifiable Y/N’ check box is selected in ST Transaction Type for this transaction. The entered date must not be less than the Processing month and year set in ‘ST facility Parameter’ and greater than current system date. This is a mandatory field.

  4. Doc Ref – Enter valid Document Reference in alphanumeric format of 20 characters maximum length.

  5. Store – Select the store from the list. Only those Stores are displayed for which you have access and ‘Opening Stock’ transaction function applicable are displayed. This is a mandatory field.

  6. Item Class – Select the required Item Class from the list. The Item Class associated to the selected Store in ‘Item Class for Store’ function are displayed. The Item Class selection is mandatory on selecting the ‘Item Class Based Y/N’ check box in the function ‘Transaction Type’ in ST Admin Setup.
