Enter the following criteria to generate the Inspection Board Report:

  1. Condemn ID - Enter the Condemn ID in alphanumeric format of 10 characters maximum length.

  2. Item Code [From-To] - Enter the Item Codes range for which the report has to be generated in alphanumeric format or select the Item Code using [?] function. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  3. Item Class [From-To] - Enter the Item Class Range for which the report has to be generated  or select the Item Class using the [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  4. Expiry Date [From-To] – Enter the Expiry Date range for which the report has to be generated in dd/mm/yyyy format or select the required date using the Calendar function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ date and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ date. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  5. Store Code [From-To] – Enter the Store codes range for which the report has to be generated in alphanumeric format or select the Store using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  6. Batch ID – Enter the Batch ID for the Inspection Board Report in alphanumeric format of 10 characters maximum length.

  7. Condemned Date [From-To] – Enter the Condemned Date range for which the report has to be generated in dd/mm/yyyy format or select the required date using the Calendar function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ date and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ date. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  8. Remarks Code [From - Code] - Enter the Remarks Code range in alphanumeric format or select the Remarks Code using the [?] function.
