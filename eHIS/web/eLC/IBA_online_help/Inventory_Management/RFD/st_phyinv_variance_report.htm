Enter the following report criteria:

  1. Physical Inventory ID – Select the required Physical Inventory ID from the list. The Inventory ID's that are not posted for variance calculation are displayed. This is a mandatory field.

  2. Suppress Zero Variance – Select the check box to suppress Items with no variance in the Variance Report.

  3. Report Option – Select the required report option from the list. The listed values are ‘Variance Report at Batch Level’ or ‘Variance Report at Item Level’ ‘Variance Report at Store Level’ and ‘Summary Report at Item Class Level’.

  4. Suppress Summary – Select the check box to suppress the summary in the Variance Report.

  5. Print Nil Stock Items - Select the check box to Print Nil stock Items. If the check box is selected, system lists and prints all items which are nil while generating the Variance Report.

  6. Item Code [From-To] - Enter the Item Codes range for which the Variance Report is to be generated in alphanumeric format of 20 characters maximum length or select the Item Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  7. Item Class [From-To] - Enter the Item Class for which the Variance Report is to be generated in alphanumeric format of four characters maximum length or select the Item Class using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.

  8. Store Code [From-To] - Enter the Store Codes range for which the Variance Report is to be generated in alphanumeric format of six characters maximum length or select the Store Code using [?] function. The values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store Code. When both values are entered, ‘To’ value has to be greater than or equal to the ‘From’ value.
