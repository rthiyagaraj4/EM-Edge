The following report criteria can be specified to generate the Stock Movement Report:

  1. Report Type – Select the required report type from the list. The listed values are ‘By Item’ and ‘By Store’. The value ‘By Item’ is displayed, by default.

  2. Store Code [From-To] - Enter the range of Store Codes for which the report is to be generated in alphanumeric format of six characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Store Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Store Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Store Code and select required values.

  3. Item Code [From-To] - Enter the range of Item Codes for which the report is to be generated in alphanumeric format of 20 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Code and select required values.

  4. Item Class [From-To] - Enter the range of Item Class for which the report is to be generated in alphanumeric format of four characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected Item Class code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ Item Class code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value. Click [?] function to query the Item Class code and select required values.

  5. Alpha code [From-To] - Enter the range of alpha codes for which the report is to be generated in alphanumeric format of 10 characters maximum length. Both the values are optional. If ‘From’ value is not entered, the system assumes a low value for the ‘From’ selected alpha code and if ‘To’ value is not entered, the system assumes a high value for the ‘To’ alpha code. When both values are entered, ‘To’ value should be greater than or equal to the ‘From’ value.

  6. Item Analysis1 – Select the required value for Item Analysis1.

  7. Item Analysis2 – Select the required value for Item Analysis2.

  8. Item Analysis3 – Select the required value for Item Analysis3.

  9. Report Option – Select the required report option from the list. The listed values are ‘All’, ‘Moved Items Only’ and ‘Moved Items Only’.

  10. Exclude the items received within (Months) – Enter the months in numeric format to exclude items received during this period in the generated report.

  11. Order By – Select the required order by option from the list. The listed values are ‘Quantity’, ‘Item Code’ and ‘Item Description’.

  12. Top (Item(s)) – Enter the required value in numeric format of eight digits maximum length to view the top moving items under stock movement.

  13. Movement Cutoff Months (Slot Ranges) – Enter the slots in numeric format where the movement of stock can be viewed based on slots defined. Maximum four slots can be defined.
