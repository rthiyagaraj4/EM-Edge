Item Movement

From this function you can Query the details on stock inflow and outflow of an Item in a Store as a result of various stock transactions for the specified period range.

Item Movement query is generated based on the quantity or value of Item Movement. The query also provides a detailed listing of the transactions against an item in a particular store.

Querying the details on Stock Inflow and Outflow

From the ST Queries, select the Item Movement function.

The Item Movement screen gets displayed.

  1. Enter the details in the mandatory fields. <Related Field Description>

  2. Click [Show Items] to display the details of stock inflow and outflow based on the entered criteria or [Reset] to clear the entered data.

It is mandatory to select values for Item/Item Class for the query type ‘Summary’.
