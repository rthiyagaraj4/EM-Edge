Cancel Count Sheet

From this function you can Cancel the generated count sheet.

The count sheet is cancelled prior to posting the variances, so that the Items of the selected Store are available for transaction entry.

Cancelling Count Sheet

From the ST Physical Inventory, select the Cancel Count Sheet function.

The Cancel Count Sheet screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to display the Items included for Physical Inventory.

  3. On searching, the Item Details of the Items included for Physical Inventory are displayed.

  4. Select the ‘Cancel’ check box against the required Item to cancel the generated count sheet.

  5. Click [Apply] to cancel the generated count sheet or [Reset] to clear the entered data.
