Notify/Suspend/Reinstate Batch

From this function you can Record Notification and  Suspension against one or more batches of inventory related to an Item. You can Revoke the suspended Batch and View the history of the batch.

The Notify/Suspend/Reinstate Batch function provides a list of all batches on selecting an Item.

On suspension, the system marks this batch as a suspended batch in all the Stores, where this Item has been defined as Stock Item. The suspension of batch will not affect the stock position and the accounting. The suspended batch is revoked to make the Items of the batch suitable for issue.

Recording Notification/Suspension and Revoking Suspension

From the ST Administration, select the Notify/Suspend/Reinstate function.

The Notify/Suspend/Reinstate screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [ Show Batch Info] to display the Notified/Suspended/Reinstated batches of the selected Item.

    Displays the following details:

    1. Batch ID – The Batch ID is displayed as a hyperlink. Click the hyperlink to view the facility and store-wise breakup of the batch Available Quantity and Quantity On Hand.

    2. Expiry Date - Displays the expiry date of the item.

    3. Trade Name - Displays the trade name of the item.

    4. Manufacturer Name - Displays the manufacturer name of the item.

    5. Quantity on Hand - Displays the item quantity on hand.

    6. Available Qty  - Displays the item quantity available.

    7. Status -  Displays the status of the item.

    8. Select - Check to select the item.

    9. Remarks – Click the Remarks hyperlink  to view the remarks of the item.

    10. History - The history is displayed  as a hyperlink. Click the hyperlink to view the history of the item.

  3. Select one or more check boxes against the required batch to select the batches. Else click Select at the header level to select all the batches at a time.

  4. Click Batch ID hyperlink to view the facility and store-wise breakup of the batch Available Quantity and Quantity On Hand in the Batch Details dialog.

    Displays the following details:

    1. Item  – Displays the name of the selected item.

    2. Batch ID – Displays the batch ID of the selected item.

    3. Expiry Date - Displays the expiry date of the item.

    4. Trade Name - Displays the trade name of the selected item.

    5. Manufacturer name – Displays the manufacturer name of the selected item.

    6. Quantity on Hand - Displays the item quantity on hand.

    7. Available Qty  - Displays the item quantity available.

    8. Status – Displays the status of the item.

    9. Cancel – Closes the Batch Details popup.

  5. Click Remarks hyperlink to invoke the Remarks screen. The hyperlink is enabled on selecting the Select check box. Click [Close] to exit from the Remarks screen.

    Enter the following details:

     

    1. Remarks – The Remarks is displayed by default with respect to the action selected. Enter the Remarks in alphanumeric format. This is a mandatory field.

    2. Notify/Suspended/Reinstate By – Enter the name of the person performing the action in alphanumeric format of 30 characters maximum length. The logged in User Name is displayed, by default.

    3. Product Recall – Select the check box to recall the product on Suspending the Item. This check box is displayed only if ‘Suspend’ is selected in Select Batch For. If this check box is selected, this batch is listed when you select ‘Reinstate’ also.

    4. Profile – Enter the User's profile in alphanumeric format.

  6. Click History hyperlink to view the batch history of the transactions performed on the selected Item.

    Displays the following details:

    1. Item – Displays the name of the selected item.

    2. UOM – Displays the unit of measure of the selected item.

    3. Batch ID – Displays the batch ID of the selected item.

    4. Expiry/Receipt Date - Displays the expiry date of the item.

    5. Trade Name  - Displays the trade name of the selected item.

    6. Manufacturer name – Displays the manufacturer name of the selected item.

    7. Date – Displays the date and time of the item.

    8. Status – Displays the status of the item.

    9. Facility – Displays the currently logged in facility.

    10. Store – Displays the stores associated with chosen facility.

    11. User Info – Displays the details of the user.

    12. Remarks - Displays the remarks of the item.

  7. Click [Apply] to save the Notification/Suspension/Reinstating or [Reset] to clear the entered data.

Only suspended batch can be reinstated.
