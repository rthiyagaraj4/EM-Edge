Enter the following details:

 

  1. Prosthesis/Implant Code - Enter the Prosthesis/Implants Code in alphanumeric code of four characters maximum length. This is a mandatory field.

  2. Long Description - Enter Long description for the Prosthesis/Implants in alphanumeric format. This is a mandatory field.

  3. Short Description - Enter Short Description for the Prosthesis/Implants in alphanumeric format. This is a mandatory field.

  4. Category - Select the Category using the [?] function. The listed values are defined in the 'Prosthesis/Implants Categories' function.

  5. Type - Select the Type using the [?] function. The listed values are defined in the 'Prosthesis/Implants Type' function.

  6. Specialty - Select a Specialty using the [?] function. The listed values are defined in the 'Specialties' function.

  7. Consignment Item - Select the check box to indicate that the Prosthesis/Implant is a Consignment Item. Consignment Items are items that have to be returned to the facility after use.

  8. Deposit Required - Select the check box to indicate that Deposit is required for this Prosthesis/Implants.

  9. Cross-Ref MM Item - Select the required Item to associate the Prosthesis/Implants using the [?] function. This field is enabled only if the MM module is installed. The Items are created in the Materials Base module. This is a mandatory field.

  10. Enabled - Select the check box to enable the record. Only enabled records are available in the transactions.
