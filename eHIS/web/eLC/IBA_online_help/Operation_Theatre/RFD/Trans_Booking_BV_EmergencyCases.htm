Emergency Cases Tab

This Tab displays orders that were placed with the Surgery Type as Emergency and those orders that were waitlisted in the Clinical Care Management function.

  1. Click [Search] to display Emergency Cases for Operation/Procedure or enter the query criteria to select a specific emergency case.

    1. Surgery Type - Select the Surgery Type from the list. The list displays Emergency Surgery Types.

    2. Specialty - Select the Specialty using the [?] function.

    3. Surgeon - Select the Surgeon using the [?] function.

    4. View - Select a view criterion from the list. The listed values are All, Unassigned and Assigned. All is selected, by default. If All is selected, all the Emergency Cases are displayed. If Unassigned is selected, all unscheduled emergency cases are displayed. If 'Assigned' is selected, all scheduled emergency cases are displayed.

  2. On searching the screen displays the Patient ID, Patient Name, Current Location/Room No/Bed No, Procedure, Surgeon, Surgery Type, Preferred Date, Waitlist Date, Waitlist Days and Status.

  3. The Patient ID is displayed with an icon. Placing the cursor over the icon displays Order Result Status and Additional Information of the Patient.

  4. Click the Patient Name hyperlink to display a list of options. Select the required option to View and record details. The options are:

Assign Theatre

Patient History

Pre-Anesthesia Evaluation

  1. Place the cursor on the Procedure hyperlink to view the Operation/Procedure details.
