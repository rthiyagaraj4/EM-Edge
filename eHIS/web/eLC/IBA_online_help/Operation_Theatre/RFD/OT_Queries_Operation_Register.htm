Enter the following details:

  1. Date [From-To] – Enter the Date range in the format of dd/mm/yyyy or select the operation date range using Calendar function. Current system date is displayed in the ‘Date From’ field, by default. This is a mandatory field.

  2. Theatre Type - Select the Theatre Type using [?] function. You can define the Theatres as OT specific or you can consider all the Theatres.

  3. Theatre - Select a Theatre using [?] function. The Operating Theatres are defined in the Room function under OT Admin Setup. This is a mandatory field.

  4. Specialty - Select the Specialty using [?] function. The Specialties created in the Operations function under OT Code Setup are displayed in the list.

  5. Surgeon - Select the Surgeon using [?] function.

  6. Anesthetist - Select the Anesthetist using [?] function.

  7. Nurse - Select the Nurse using [?] function.

  8. Patient ID - Select the Patient ID using [?] function.

  9. Patient Name - The Patient Name corresponding to the selected Patient ID is displayed in this field, on selecting the Patient ID.

  10. Operation - Select the Operation using [?] function.

  11. Anesthesia Type - Select the Anesthesia type using [?] function.

  12. Implant Used - Select the Implant code using [?] function. The implants created in the Prosthesis/Implants function under OT Code setup are displayed in the list.

  13. Order By - Select the Order By criteria from the list to sort the output. This is a mandatory field.

  14. Source Type – Select the Source Type from the list. Based on the Source Type selected you can select the Source using [?] function. The  icon appears beside the Source field if the long name for the External Referral Source is recorded, in ‘External/Referral Source’ function of Application Masters module. Place the mouse on the  icon to display the long name of the External or Referral Source recorded in ‘External/Referral Source’ function of Application Masters module.

  15. Status Code - Select the Status of the operation from the system-defined list. The system defined values are: All, Checked-In, In Holding Area, In OR, In Recovery Room, Transferred Back to OT, Checked-Out Doc. Incomplete, Checked-Out Doc. Complete and Cancelled. The list displays the value ‘All’, by default. This is a mandatory field.
