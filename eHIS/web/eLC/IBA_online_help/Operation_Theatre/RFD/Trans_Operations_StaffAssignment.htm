Enter the following details:

 

  1. Role – Select the Role from the drop-down list. The staff roles defined in the Application Masters are listed. The selected role is assigned to the selected staff.

  2. Name – Select the staff name from the drop-down list. The staff names defined in the Application Masters are listed.

  3. Show in widget – Select this check box to show the staff details in the dashboard. This check box is selected by default.

  4. From time – Select the From time the drop-down list. The staff will start work in the operation theatre from the selected From time.

  5. To time – Select the To time from the drop-down list. The staff will work in the operation theatre upto the selected To time.

  6. Delete – Click the   to delete the selected record.
