Enter the following details:

 

  1. Generate By - Select Theatre and Surgeon, Theatre or Surgeon from the list.

  2. Theatre - This field is enabled if Generate By is selected as Theatre and Surgeon or Theatre. Select the Theatre using the [?] function to generate schedule.

  3. Surgeon - This field is enabled if Generate By is selected as Theatre and Surgeon or Surgeon. Select the Surgeon using the [?] function to generate schedule.

  4. Days of Week - On selection of the Theatre and Surgeon, the days of the week are populated based on the setup done in 'Diary Schedule for Surgeon' function.

  5. Diary Generated upto - This field displays the last Schedule generated date.

  6. From Date - Enter the date from when the schedule has to be generated. This is a mandatory field.

  7. To Date - Enter the date till when the schedule has to be generated. This is a mandatory field.
