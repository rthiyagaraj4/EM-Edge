Booked Cases tab

  1. Click [Search] to display Patient bookings for Operation/Procedure or enter the query criteria to select a specific booking case. . Patient ID, Patient Name, Current Location/Room No/Bed No, Patient Class , Theatre, Time, Procedure, Change Stage, Surgeon, Priority, Current Stage and Sequence  are displayed in the search results.

    1. Theatre Date - Select the date for which booking has to be verified. The current date is displayed, by default.

    2. Theatre Type - Select the Theatre Type using [?] function.

    3. Theatre - Select the Theatre using [?] function.

    4. Specialty - Select the Specialty using [?] function.

    5. Surgeon - Select the Surgeon using [?] function.

    6. Surgery Type - Select the Surgery Type from the list. The Surgery Types are defined in the Surgery Type function.

    7. Current Stage - Select a Booking stage from the list. The listed values are defined in the Booking Finalisation Stage function.

    8. Change Stage - This list item gets enabled only if a value other than 'All Stages'  is selected as the Current Stage. You can use this field to change the stage of all the records in the result area.

    9. Rearrange Sequence - Select the check box to indicate whether the Booking Sequence has to be rearranged.

    10. ASA Score – Select the ASA Score from the drop-down list. The ASA scores defined in the ASA Score Setup under the Medical Records module are listed in the drop-down box.

The Patient ID is displayed with an  icon. Placing the cursor over the icon displays order Result Status and Additional Information of the Patient.

Once the booking is linked to an open encounter, the Current Location column gets automatically populated in the Booking Verification screen
