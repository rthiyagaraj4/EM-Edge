Setup Parameter

From this function you can Set the Parameters for the various Anesthesia Note Types.

Various Note Types are defined for Anesthesia at various stages of the operation. Pre-Anesthesia Note Type defines the patient's condition before the operation, Intra-Anesthesia Note Type defines the patient's condition while the operation is being preformed and the Post-Anesthesia Note Type defines the patient's condition after the operation is performed.

Setting Anesthesia Parameter

From the OT Admin Setup > Anesthesia, select the Setup Parameter function.

The Setup Parameter screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Apply] to save the parameter setup or click [Reset] to clear the entered data.
