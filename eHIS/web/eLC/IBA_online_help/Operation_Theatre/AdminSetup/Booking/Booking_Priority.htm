Booking Priority

From this function, you can Create a Booking Priority and Query for existing records.  

Booking Priority function allows defining priorities for booking. Booking Priorities are used in the Booking Verification function. If an operation scheduled later is given precedence over an operation scheduled earlier, the booking priority is entered. For example, an emergency operation can be given precedence over an already booked minor operation.

Creating Booking Priority

From the OT Admin Setup > Booking, select the Booking Priority function.

The Booking Priority screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Code and Description of the Booking Priority. These fields are mandatory.

  3. Select the Enabled check box, to enable the record.

  4. Click [Apply] to save the created Booking Priority or [Reset] to clear the entered data.

  5. Click [Query] to search for an existing record.
