Holidays  

From this function, you can Create a Holiday and Query for a Holiday record. You can also Delete records.

Holidays function allows you to set Holidays. A selected holiday can be cancelled at a later stage if required. The system will not allow you to enter Holidays for days that already have generated schedules. In order to mark a holiday on these days, the schedules have to be cancelled and regenerated. In the schedule status screen of the 'Booking' function, all the days that are defined as 'Holidays' appear in a distinctly different color indicating that these days are holidays and bookings cannot be made.

Creating a Holiday

From the OT Admin Setup > Booking, select the Holiday function.

The Holiday screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Holiday using the [?] function. The Holiday Date field is mandatory.

  3. The day of the selected Holiday is displayed.

  4. Enter the Reason for Holiday. The Holiday field is mandatory.

  5. Click [Apply] to save the created Holiday or [Reset] to clear the entered data.

  6. Click [Query] to search for an existing holiday.

  7. Click [Delete] to delete a queried record.
