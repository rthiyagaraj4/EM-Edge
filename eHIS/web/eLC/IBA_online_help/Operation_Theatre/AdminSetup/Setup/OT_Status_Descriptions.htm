OT Status Description

From this function, you can View the various OT Status details.

OT Status Descriptions are pre defined and supplied as part of the OT default scripts. Each status is associated with a background color and text colour. The colors are used in the transactions to indicate the status of the record. As each stage of an operation is completed, the color changes accordingly.

Viewing OT Status Description

From OT Admin Setup > Setup, select the OT Status Descriptions function.

The OT Status Descriptions function screen gets displayed.

  1. Click [Query] to invoke the query screen.

  2. Click [Execute] to display the OT Status Descriptions.

  3. Select the code hyperlink to display details of the OT Status Description.

    1. Status Code - The status code of the Status Description is displayed.

    2. System Defined Description - The system defined description is displayed and disabled.

    3. Description - The description of the status is displayed. You can modify the description. Enter a Description in alphanumeric format of 40 characters maximum length.

    4. BackGround Color - Select the background color from the list. The default value is displayed as 'All'.

    5. Text Color - Select the text color from the list. The listed values are Black and White.

  4. Click [Apply] to save the changes made in the record or [Reset] to clear the entered data.
