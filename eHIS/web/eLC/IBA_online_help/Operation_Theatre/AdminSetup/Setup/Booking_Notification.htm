Booking Notification

From this function you can Set an automatic reminder similar to appointment slip to be sent to the patients via SMS before a specific day range and Select the medical personnel to get an automatic notification of cancellation of surgery via SMS before a specific day range.

The Booking Notification screen has two tabs namely Patients and Practitioner. The ‘Patients’ tab is used to define the number of days based on which the automatic reminder about admission similar to appointment slip is to be sent to the patients via SMS. The ‘Practitioner’ tab allows you to select the medical personnel who has to be notified about cancellation of surgery via SMS. The SMS reminder details include Patient Name, Patient ID, Procedure Name, Day, Date and Time, Booking Number, Doctor’s name, Enquiry Number of Hospital.

Setting automatic notification/reminder

From OT Admin Setup > Setup, select the Booking Notification function.

The Booking Notification screen gets displayed.

The screen displays 2 tabs:

Patients

Practitioner
