Functions

From this function, you can Query for system defined functions and associate certain parameters.

This function enables you to change the system defined descriptions and specify if completion of checklist has to be checked, barcoding is allowed and verification is applicable for the function.  Functions of the Operation Theatre module are supplied with the default OT scripts.

Viewing Functions

From OT Admin Setup > Setup, select the Functions function.

The Functions screen gets displayed.

  1. Click [Query] to invoke the query screen.

  2. Click [Execute] to display the Functions.

  3. Select the Function ID hyperlink to display details of the Function.

    1. Function ID - The system defined Function ID is defined.

    2. System Defined Description - The system defined description of the function is displayed and disabled.

    3. Description - The description of the Function is displayed. You can modify the description. Enter the description in alphanumeric format of 40 characters maximum length.

    4. URL - The URL of the function is displayed and disabled.

    5. Linked Status - The status linked to the Function is displayed and disabled.

    6. Check for Checklist Completed - Select the check box if the system should verify if the checklist is completed.

    7. Allow Barcode - Select the check box to allow barcoding facility. While booking an operation, you need not manually enter the patient details. If a bar-code reader is interfaced to the system, the reader populates the details of the patient.

    8. Verification Applicable - Select the check box if verification of the function is applicable.

  4. Click [Apply] to save the changes made in the record or [Reset] to clear the entered data.
