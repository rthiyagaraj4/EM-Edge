Role Type

From this function, you can Query for a Role Type and Modify the description of the record.

The system defined Role Types indicate the various categories of the hospital personnel. Some of the examples of Role Types are Operating Surgeon, Nurse, etc.

Querying For Role Types

From the OT Admin Setup > Setup, select the Role Type function.

The Role Type screen gets displayed.

  1. Click [Query] to search for Role Types.

  2. Click [Execute] to display the system defined Role Types.

  3. Select the Role Type hyperlink to display the Role Type details.

    1. Role Type - The Type of the Role is displayed.

    2. System Defined Description - The system defined description of the Role Type is displayed.

    3. Description - The description of the Role Type is displayed. You can modify the Role Type in alphanumeric format of 30 characters maximum length.

  4. Click [Apply] to save the changes made or [Reset] to reset the entered details.
