Swab Count Form

Swab Count Form function is used to record the Swab Count Details.

The Order ID, Date and Time of Booking, Surgeon, Specialty, Theatre, Booking Number, Date and Time of the surgery, Accession Number and current date are displayed.

The following details are displayed:

  1. End Time – The Surgery End Time is displayed as recorded.

  2. Surgery Start Time – The Surgery Start Time is displayed as recorded.

  3. Operation – The Operation is displayed. Place the cursor over the Operation field to display the Operation details.

  4. Click [Print] to print the Swab count details.

  5. Click [Add a Swab Count] to add one more Swab Serial Number and the number is displayed which is a hyperlink.

This screen displays 3 tabs:

  • Swab Count

  • Other Details

  • Personnel Details

Swab Count Tab

  1. First Count Completed? - Select the check box to indicate that the first swab count for the operation is completed. Enter the Item details and click [Record].

  2. Addition During OP Completed? – This field is enabled only after recording First Count Completed. Select the check box to indicate if any new addition of Swab/Instruments was made during the operation. Enter the Item details and click [Record].

  3. Second count Completed? - Select the check box to indicate that the second swab/instrument count for the operation is completed. Enter the Item details and click [Record].

  4. Additions after Second Count Completed - Select the check box to indicate that additions were made after the cavity was closed. Enter the Item details and click [Record].

  5. Third count Completed? - Select the check box to indicate that the third swab/instrument count for the operation is completed. Enter the Item details and click [Record].

  6. Additions after third count completed - Select the check box to indicate that additions were made after the cavity was closed. Enter the Item details and click [Record].

  7. Fourth count Completed? - Select the check box to indicate that the fourth swab/instrument count for the operation is completed. Enter the Item details and click [Record].

  8. Additions after fourth  count completed - Select the check box to indicate that additions were made after the cavity was closed. Enter the Item details and click [Record].

  9. Fifth count Completed? - Select the check box to indicate that the fifth swab/instrument count for the operation is completed. Enter the Item details and click [Record].

  10. Additions after fifth count completed - Select the check box to indicate that additions were made after the cavity was closed. Enter the Item details and click [Record].

  11. Final Count Completed - Select the check box to indicate that the final swab/instrument count for the operation is completed. Enter the Item details and click [Record].

  12. Reviewed By - This field is used to print the Surgeon name on the report, which defaults the Surgeon name from the Personnel Details tab. If more than one surgeon name is entered in the Personnel Details tab you can select surgeon name using [?] function. This field is enabled on finalization of final count.

The Swab/Instruments required for the Operations are listed. Enter the required details:

  1. Items - The swabs and instruments are displayed.

  2. First count - Enter the initial (first) count of Swab/Instruments for the operation.

  3. Additions during OP - Enter the number of swabs/instruments added during the operation.

  4. Total count - The total count of the Initial swab/instruments and the number of additional swab/instruments during the operation is displayed.

  5. 2nd Count - Enter the number of Swab/Instruments during the Second count. This field is enabled only if Second Count Completed check box is selected. Entering Remarks is mandatory for recording 2nd count.

  6. Additions (After Second Count) - Enter the number of swabs/instruments added during the operation.

  7. Total Count - The total number of the initial count of Swab/Instruments, number of Swabs/Instruments added during the operation, and number of additional Swab/Instruments after the cavity was closed is displayed. The Final Count is highlighted in red if there is any discrepancy between the Total Count and Final Count.

  8. Third count - Enter the third count of Swab/Instruments for the operation.

  9. Additions (After third count) - Enter the number of swabs/instruments added during the operation.

  10. Total Count - The total count of the third swab/instruments and the number of additional swab/instruments during the operation is displayed.

  11. Fourth count - Enter the fourth count of Swab/Instruments for the operation.

  12. Additions (After fourth count) - Enter the number of swabs/instruments added during the operation.

  13. Total Count - The total count of the fourth swab/instruments and the number of additional swab/instruments during the operation is displayed

  14. Fifth count - Enter the Fifth count of Swab/Instruments for the operation.

  15. Additions (After fifth count) - Enter the number of swabs/instruments added during the operation.

  16. Total Count - The total count of the fourth swab/instruments and the number of additional swab/instruments during the operation is displayed

  17. Final Count - Enter the Final Count after closure of cavity. This field is enabled only if Final Count Completed check box is selected. Entering Remarks is mandatory for recording final count.

  18. View Additions - On clicking the 'Addns' hyperlink, a View Remarks screen is displayed with recorded details of First Count, Additions during Operation, Second Count, Additions after Cl. Cavity completed and Final Count. Click [Close] to exit from the screen.

  19. Remarks – Click the remarks hyperlink and enter remarks if required.

Other Details Tab

This Tab is used to record other swab details.

Basic Set

Enter the basic instrument set used in the operation.

Supplementary Item

Enter the supplementary items used in the operation.

Verify Swab Count

  1. Any Discrepancies in First Count - Select the check box to indicate any discrepancies in the first count.

  2. X-ray Performed - Select the check box to indicate that X-ray was performed for the patient.

  3. Missing Item Found - Select the check box to indicate that a missing item was found.

Remarks

Enter remarks, in free text format if any.

Personnel Details Tab

  1. Specialty - Select the Specialty from the list. The Specialties available in the list are based on Documentation Level field. If Documentation Level is selected as Across Specialties, this field is displayed as All Specialties.

  2. Role - Select the Role from the list. This is a mandatory field.

  3. Name - Select the name of the Role. The Practitioners listed have the Practitioner Types associated to the Role selected.

  4. Click [Select] to add Personnel details to the screen. Select Specialty hyperlink and click [Cancel] to exclude the personnel from recording.

  5. Click [Record] to save Swab Count Details.
