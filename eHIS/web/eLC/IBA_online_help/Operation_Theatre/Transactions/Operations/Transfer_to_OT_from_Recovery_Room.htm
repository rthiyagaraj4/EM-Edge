Transfer to OT from Recovery Room

Transfer to OT from Recovery Room function is used to record transferring details of the patient from Recovery Room to the Operation Theatre.

The Order ID, Date and Time of Booking, Surgeon, Specialty, Theatre, Booking Number, Date and Time of the surgery, Accession Number and current date are displayed.

The following details are displayed:

  1. Anesthesia Type – The Type of Anesthesia is displayed as recorded.

  2. Anesthetist – The Anesthetist is displayed as recorded.

  3. Theatre – The Theatre is displayed.

  4. Click Pre-operative clinical comments to view Pre-Operative details recorded for the patient.

Operations/Procedures

The Operation Code, Description, Side Applicable and Remarks are displayed. Click the Pre-operative comments hyperlink to view the remarks recorded for the Operation/Procedure.

Transfer Details

  1. Theatre Type - Select the Theatre Type from the list. This is a mandatory field.

  2. Theatre – Select the Theatre from the list. The Theatres are displayed based on the selected Theatre Type. This is a mandatory field.

  3. Reason - Select the reason using [?] function. The Reasons are created in the Reason function. This is a mandatory field.

  4. Change Status to - Select the appropriate status from the list. The listed values are Checked In and In OR.

  5. Remarks - Enter remarks if required.

  6. Click [Apply] to save the transfer to OT from Recovery Room.
