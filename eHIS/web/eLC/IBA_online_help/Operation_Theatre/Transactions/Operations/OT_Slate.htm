OT Slate

From this function you can View all the operations scheduled for a particular day and Perform the necessary activities.   

OT Slate function provides a single view of all the operations for a particular day with their colour indicator. A menu is assigned to each stage to perform various activities based on the privileges set for the user/role. The system checks the User ID and password to authenticate the user for performing activities.

Performing OT Slate Activities

From Operations, select the OT Slate function.

The OT Slate screen gets displayed.

The screen displays three tabs:

Slate

Booking Verification

To Be Rescheduled Cases

 

Bookings  passed through the Booking Verification Stage and marked as "Finalised" appear in the Slate tab.

As the operations pass through each stage, the color indicator changes accordingly. The bottom of the OT Slate screen displays the color associated to each stage. Note: The system  displays a warning  message about the existing scheduled operation when a patient already has an existing scheduled and/or unscheduled operation for a particular day and you try to check-in or book the patient for the same day. The warning message is ‘Patient has existing operations for this day, Do you want to continue?’
