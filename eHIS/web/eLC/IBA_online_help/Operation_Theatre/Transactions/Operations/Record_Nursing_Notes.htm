Record Nursing Notes

Record Nursing Notes function is used to record Nursing Notes. The function captures all nursing details of the Operation/Procedure.

The following details are displayed:

  1. Anesthesia Type – The Type of Anesthesia is displayed as recorded.

  2. Anesthetist – The Anesthetist is displayed as recorded.

  3. Theatre – The Theatre is displayed.

  4. Documentation Level – Select Documentation Level as Specialty or Across Specialties.

  5. Click Pre-operative clinical comments to view Pre-Operative details recorded for the patient.

  6. Click the Pre-Operative Diagnosis hyperlink to view the pre-operative diagnosis recorded from the Medical Records, Clinician Access modules with indications. You cannot modify the data.

  7. Click Operative Records to record operative details.

  8. Click Consumables Used hyperlink to details of consumables used.

  9. Click ‘View Consumables Used’ to view the consumables details.

The screen displays the following tabs:

  • Operation Details

    This tab displays the following tabs to record details:

    Operations/Procedures

    Surgical Accessories

    Operations/Procedures Tab

    1. The Operation Code, Description, Side Applicable, Remarks and Charge details are displayed. Click the Pre-operative comments hyperlink to view the remarks recorded for the Operation/Procedure while booking.

    The operation code and description, side applicable, remarks, total charge and the status are displayed when the operation is complete. The total payable amount, patient payable amount and patient paid amount are displayed. If the service is under policy coverage a symbol is displayed. If the service is not covered by the policy a symbol is displayed. Similarly, if the Service requires an approval, it is indicated using symbol. If the service does not requires an approval, symbol is used to indicate the same. Click symbol to edit/enter the charge details. You can include/exclude the service from policy coverage.

    Surgical Accessories Tab

    Click arrow to displayed Prosthesis/Implant details. The Prosthesis/Implant details are displayed, by default.

    1. Accessory Type – The name of the accessory time is displayed and disabled as per the setup in Setup Parameter for Facility function.

    2. Implant – Select the Implant using [?] function. The listed values are created in Prosthesis/Implant function. This is a mandatory field.

    3. Operation – The operation selected in ‘Record Surgeon Notes’ is selected, by default.

    4. Location - The location code defined in the ‘Instruments/CSSD Packs’ screen in the Code Setup is displayed based on the  instrument selected.

    5. Consumption Store – Select the consumption store using [?] function. This is a mandatory field.

    6. Reserved Picklist Units – The number of reserved picklist units are displayed.

    7. Requested Picklist Units – The number of requested picklist units are displays.

    8. Stock Available – The number of stock available is displayed.

    9. Actual Units – Enter the number of actual units in numeric format of three digits maximum length. This is a mandatory field.

    10. Remarks – Click ‘Remarks’ to enter the remarks for surgical accessories in the free text.

    11. Date & Time – Select the Date and Time using the calendar function. The surgical equipments details will be displayed based on the selected date. By default, the current date is displayed.

    The error messages are displayed when the Select button is clicked based on the below mentioned conditions:

    1. If the Date & Time is less than Check-In Date/Time:

    ‘Date & Time Cannot be Before the Check-In Date/Time <<dd/mm/yyyy hh:mm>>’.

    2. If the Date & Time is greater than One Week from the Current Date:

    ‘Date & Time Should be Within One Week from Current Date’

     

    Operation, Implant, Actual Units, Reserved Picklist Units, Requested PickList Units, Remarks, Consumption Store and charge details are displayed. The total payable amount, patient payable amount and patient paid amount are displayed. If the service is under policy coverage a symbol is displayed. If the service is not covered by the policy a symbol is displayed. Similarly, if the Service requires an approval, it is indicated using symbol. If the service does not requires an approval, symbol is used to indicate the same. Click symbol to view the charge details. You cannot include/exclude the service from policy coverage.

    Consumables Packs

    1. Operation – The name of the operation is displayed and hyperlinked. Click the operation name hyperlink to the details of the operation in the Surgical Accessories.

    2. Package – The name of the package is displayed.

    3. Actual Units – The number of units are displayed.

    4. Remarks – Click ‘Remarks’ to view the remarks of the operations.

    5. Charge – The charges of the operation is displayed. Click   to enter and edit the charges details.

    Operation, Package, Actual Units, Remarks and charge details are displayed. The total payable amount, patient payable amount and patient paid amount are displayed. If the service is under policy coverage a symbol is displayed. If the service is not covered by the policy a symbol is displayed. Similarly, if the Service requires an approval, it is indicated using symbol. If the service does not requires an approval, symbol is used to indicate the same. Click symbol to edit/enter the charge details. You can include/exclude the service from policy coverage.

    Equipment

    1. Click the icon to display the equipment image with location code in the ‘Equipment Image’ dialog. If multiple equipments are used for the surgery/procedure, when you clicking this icon, the list of equipments with their Item Code, Item Description, Image Description and Location Code will be displayed in the ‘Equipment’ dialog.

    2. Operation – The name of the operation is displayed and hyperlinked. Click the operation name hyperlink to the details of the operation in the Surgical Accessories.

    3. Package – The name of the package is displayed.

    4. Actual Units – The number of units are displayed.

    5. Equipment ID – The name of the equipment id is displayed.

    6. Remarks – Click ‘Remarks’ to view the remarks of the operations.

    7. Charge – The charges of the operation is displayed. Click   to enter and edit the charges details.

    Operation, Package, Actual Units, Equipment ID, Remarks and charge details are displayed. The total payable amount, patient payable amount and patient paid amount are displayed. If the service is under policy coverage a symbol is displayed. If the service is not covered by the policy a symbol is displayed. Similarly, if the Service requires an approval, it is indicated using symbol. If the service does not requires an approval, symbol is used to indicate the same. Click symbol to edit/enter the charge details. You can include/exclude the service from policy coverage.

    Instruments/CSSD packs

    Click arrow to displayed Instrument/CSSD Packs details.

    1. Click the  icon to display the instrument image in the ‘Instrument Image’ dialog. If multiple instrument/ CSSD Packs are used for the surgery/procedure, on clicking the icon , the list of instruments/CSSD Packs with their Item Code, Item Description and Image Description are displayed in the ‘Instrument’ dialog. Click the Image Description icon to view the image in a new dialog. The image display is based on the OT Code Setup.

    2. Operation – The name of the operation is displayed and hyperlinked. Click the operation name hyperlink to the details of the operation in the Surgical Accessories.

    3. Package – The name of the package is displayed.

    4. Actual Units – The number of units are displayed.

    5. Location - The location code defined in the ‘Instruments/CSSD Packs’ screen in the Code Setup is displayed

    6. Tray No – The number of the tray is displayed.

    7. Remarks – Click ‘Remarks’ to view the remarks of the operations.

    8. Charge – The charges of the operation is displayed. Click to enter and edit the charges details.

    Operation, Instrument, Actual Units, Tray Number, Remarks and charge details are displayed. The total payable amount, patient payable amount and patient paid amount are displayed. If the service is under policy coverage a symbol is displayed. If the service is not covered by the policy a symbol is displayed. Similarly, if the Service requires an approval, it is indicated using symbol. If the service does not requires an approval, symbol is used to indicate the same. Click symbol to edit/enter the charge details. You can include/exclude the service from policy coverage.

  • Personnel Details

    Personnel Details Tab

    This tab displays 2 tabs:

    • Hospital Personnel

    • Visiting Personnel

    Hospital Personnel Tab

    1. Specialty - Select the Specialty from the list. The Specialties available in the list are based on Documentation Level field. If Documentation Level is selected as Across Specialties, this field is displayed as All Specialties.

    2. Role - Select the Role from the list. This is a mandatory field.

    3. Name - Select the name of the Role. The Practitioners listed have the Practitioner Types associated to the Role selected.

    4. Click [Select] to add Hospital Personnel details to the screen. Select Specialty hyperlink and click [Cancel] to exclude the hospital personnel.

     

    Visiting Personnel Tab

    1. Specialty - Select the specialty from the list. The specialties listed are displayed based on Documentation Level field. If Documentation Level is selected as Across Specialties, this field is displayed as All Specialties.

    2. Visiting Personnel - Enter the details of the visiting personnel in the free text area.

    3. Click [Select] to add Visiting Personnel details to the screen. Select Specialty hyperlink and click [Cancel] to exclude the visiting personnel from recording.

  • Time and Details

    Time and Details Tab

    This Tab is used to record the time details of the operation.

    1. Specialty - Select the Specialty from the list. The Specialty is listed based on the selected Documentation Level.

    2. Preparation Time - Enter the preparation time for the operation in minutes.

    3. Anesthesia Procedure Start Time - Enter the Start Date and Time of the Anesthesia Procedure.

    4. Anesthesia Procedure End Time - Enter the End Date and Time of the Anesthesia Procedure. This date and time should be greater than the Start Date and Time.

    5. Anesthesia Start Time - Enter the Start Date and Time of the Anesthesia.

    6. Anesthesia End Time - Enter the End Date and Time of the Anesthesia. This date and time should be greater than the Anesthesia Start Date and Time.

    7. Procedure Start Time - Enter the Start Date and Time of the Procedure.

    8. Procedure End Time – Enter the End Date and Time of the Procedure. This date and time should be greater than the Procedure Start Date and Time.

    9. Click [Select] to add the recorded details. Select the Specialty hyperlink change the time details or click [Cancel] to cancel the time details entered.

  • Specimen Details

    Specimen Details Tab

    1. Specimen No./Dtl - Enter the Specimen Number or Details. This is a mandatory field.

    2. Department - Select the Department using [?] function. This is a mandatory field.

    3. Quantity - Enter the Quantity of specimen collected.

    4. Click [Select] to add the specimen details to screen. Select the Specimen No./Dtl hyperlink to modify the quantity or click [Cancel] to cancel recording specimen details.

  • Disposal Details

    Disposal Details Tab

    This tab is used to record the disposal details.

    1. Disposed Item - Select the Disposed Item using [?] function. This is a mandatory field.

    2. Given By - Enter the name of the person who gave the disposed items. This is a mandatory field.

    3. Received By - Enter the name of the person who received the disposed items. This is a mandatory field.

    Status

    1. Documentation Completed – Select the check box to indicate that documentation is completed for Nursing Notes.

    2. Prosthesis/Implants Completed – Select the check box to complete the Prosthesis/Implant. This check is disabled as per the setup in Setup Parameter for Facility function.

    3. Consumables Completed – Select the check box to complete the Consumables. This check is disabled as per the setup in Setup Parameter for Facility.

    4. Equipment Completed – Select the check box to complete the equipment. This check is disabled as per the setup in Setup Parameter for Facility.

    5. Instruments/CSSD Completed - Select the check box to complete the Instrument/CSSD. This check is disabled as per the setup in Setup Parameter for Facility.

  1. Click [Apply] to save the Nursing Notes. If Documentation Completed check box is not selected, the system prompts you before saving.

 

The surgeon, assistant surgeon and anaesthetist details are displayed only if the Display Hospital Personnel Assigned During Booking/Check-In check box in setup is selected.
