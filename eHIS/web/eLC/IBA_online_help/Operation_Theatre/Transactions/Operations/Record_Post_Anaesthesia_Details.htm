Record Post-Anesthesia Details

Post-Anesthesia function is used to record the Post-Anesthesia details.

The Order ID, Date and Time of Booking, Surgeon, Specialty, Theatre, Booking Number, Date and Time of the surgery are displayed.

The following details are displayed:

  1. Anesthesia Serial No – The Anesthesia Serial Number is displayed.

  2. Accession Number – The Accession Number is displayed.

  3. Referral Source – The referral source of the patient is displayed.

  4. Weight – Enter the Weight of the Patient in Kgs.

  5. Height – Enter the Height of the Patient in Cms.

  6. BSA – The BSA of the Patient is displayed if recorded.

  7. BMI Details – The BMI details of the Patient is displayed if recorded.

  8. Click Post-Anesthesia Details hyperlink to record Post-Anesthesia Details.

  9. Click Record Chart hyperlink to record chart details for the patient.

Anesthesia Particulars

  1. Anesthesia Details – The Anesthesia Details are displayed as recorded.

  2. Start Date/Time – The date and time of starting anesthesia is displayed as recorded.

  3. Finish Date/Time – Enter the finish date and time of anesthesia.

Recovery Particulars

  1. Recovery Particulars – Enter the start date and time of recovery from anesthesia.

  2. Recovery End Time - Enter the end date and time of recovery from anesthesia. This is a mandatory field.

Personnel Details Tab

  1. Role – Select the Role performing post anesthesia from the list. This is a mandatory field.

  2. Name – Select the name of the selected role using [?] function. This is a mandatory field.

  3. Click [Select] to add the details to the screen. Select the Role hyperlink and click [Cancel] to cancel recording personnel details.

Untoward Event Tab

  1. Unusual Occurrence – Select the Unusual Occurrence using [?] function. The values are defined in the Unusual Occurrence function. This is a mandatory field.

  2. Click Action Remarks hyperlink to enter action remarks for the unusual occurrence.

  3. Click General Remarks to enter remarks in general for the unusual occurrence.

  4. Click [Select] to add the details to the screen. Select the Untoward Event hyperlink and click [Cancel] to cancel recording Untoward Events.

Status

  1. Post-Anesthesia Completed - Select the check box to indicate completion of recording Post-anesthesia details.

  2. Click [Apply] to save Post-Anesthesia Details.
