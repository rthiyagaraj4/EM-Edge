Record Intra-Anesthesia Details

Intra-Anesthesia function is used to record the Intra-Anesthesia details.

The Order ID, Date and Time of Booking, Surgeon, Specialty, Theatre, Booking Number, Date and Time of the surgery are displayed.

The following details are displayed:

  1. Anesthesia Serial No – The Anesthesia Serial Number is displayed.

  2. Source ID – The Order ID is displayed.

  3. Accession Number – The Accession Number is displayed.

  4. Referral Source – The referral source of the patient is displayed.

  5. Weight – Enter the Weight of the Patient in Kgs.

  6. Height – Enter the Height of the Patient in Cms.

  7. BSA – The BSA of the Patient is displayed if recorded.

  8. BMI Details – The BMI details of the Patient is displayed if recorded.

Click Intra-Anesthesia Details hyperlink to record Intra-Anesthesia Details.

Click the Pre-Anesthesia hyperlink to view details recorded for pre-anesthesia.

Click Record Chart hyperlink to record chart details for the patient.

 

Anesthesia Particulars

  1. Anesthesia – Select the Anesthesia using [?] function. The Anesthesia is created in the Anesthesia function of Application Masters module. This is a mandatory field.

  2. Anesthesia Start Time – Enter the date and time of starting intra-anesthesia. This is a mandatory field.

  3. Anesthesia End Time – Enter the date and time of ending intra-anesthesia.

 

Operations/Procedures Tab

The Operation Code, Description and Side Applicable are displayed.

 

Personnel Details Tab

  1. Role – Select the Role performing intra anesthesia from the list. This is a mandatory field.

  2. Name – Select the name of the selected role using [?] function. This is a mandatory field.

Click [Select] to add the details to the screen. Select the Role hyperlink and click [Cancel] to cancel recording personnel details.

 

Status

  1. Intra-Anesthesia Completed - Select the check box to indicate completion of recording Intra-anesthesia details.

Click [Apply] to save Intra-Anesthesia Details.
