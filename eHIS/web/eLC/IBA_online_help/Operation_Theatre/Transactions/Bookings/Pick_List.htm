Pick List

From this function you can Generate, Review and Print the pick lists.  

A Pick List is a list of all items and surgical accessories required for an operation. Pick list is generated based on the existing schedules for the selected date. The system looks for the surgeon preferences function first and if it does not exist, the default surgical accessories function defined is taken as the basis for pick list generation. Pick list review allows adding of additional accessories, if required and then the pick list can be finalized after the review is done.

Generating/Reviewing/Printing Pick Lists

From the Bookings, select the Pick List function.

The Pick List screen gets displayed.

The screen displays 3 Tabs:

Generate

Select the required details to generate pick list.

  1. Schedule Date - Select the Date for which the Pick List has to be generated. This is a mandatory field.

  2. Theatre Type - Select the Theatre Type from the list.

  3. Theatre - Select the Theatre from the list. The Operating Rooms are defined in the Operating Room function.

  4. Source Type - Select the Source Type from the list. If the  Source Type is selected, the Patient ID will not be available for selection.

  5. Patient ID - Select the required Patient ID using [?] function.

  6. Patient Name - The name of the patient is displayed based on the selected Patient ID.

Click [Generate] to generate the pick list. The generated pick list is displayed in the Review Tab.

Review

The Pick list generated in the Generate Tab is displayed.

  1. The Review tab displays the Surgery Date, Patient, Surgeon, Booking No and Operation of the generated pick list. An icon  is displayed in the column when the instruments/equipments and packages required for surgery are finalized or requested or partially finalized.  This icon is displayed only if the Operation Theatre (OT) and Sterile Stock module is interfaced.

  2. Select the Operation hyperlink.

  3. The generated pick list from the ‘Generate’ tab is displayed at the bottom of the screen displaying details Item Type, Item and No. of Units/Persons.

 

Surgical Accessories

Package and Implants

  1. Patient ID - The Patient ID of the selected operation is displayed.

  2. Operation - The Operation recorded is displayed.

  3. Booking No - The Booking Number for the Operation is displayed.

  4. Item Type - Select the Item Type from the list. The listed values are Instrument, Equipment, Package, Implant and Resource Type. This is a mandatory field.

  5. Item Description - Select the Item Description using [?] function. The Items are displayed based on the selected Item Type. This is a mandatory field.

  6. No of Units - Enter the Number of Units in alphanumeric format of six characters maximum length.. This is a mandatory field.

  7. Requesting Store - Select the Requesting Store from the list. The stores associated to the theatre and the type selected as 'Theatre Requesting/Consumption Store' in 'Stores for Operating Room' function are displayed. This field is enabled if Item Description is displayed. This is a mandatory field.

  8. Issuing Store - Select the Issuing Store from the list. The stores associated to the theatre and the type selected as 'Theatre Issuing Store' in 'Stores for Operating Room' function is displayed. This is a mandatory field.

  9. QOH of Req Store - The quantities of item available in the requesting store is displayed.

  10. Reserve Quantity - This field is enabled if Requesting Store is enabled. Enter the Reserve Quantity of the Item in numeric format of six digits maximum length.

  11. Request Quantity - Enter the Request Quantity of the Item in numeric format of six digits maximum length.

  12. Select/Deselect All - Select the check box to select the generated pick list. Clear the check box to clear the selection of the Items from the pick list.

Instruments/CSSD and Equipments

  1. Patient ID - The Patient ID of the selected operation is displayed.

  2. Operation - The Operation recorded is displayed.

  3. Booking No - The Booking Number for the Operation is displayed.

  4. Item Type - Select the Item Type from the list. The listed values are Instrument and Equipment. This is a mandatory field.

  5. Item Description - Select the Item Description using [?] function. The Items are displayed based on the selected Item Type. This is a mandatory field.

  6. Quantity Available - The available quantity of the item selected in the ‘Item Description’ field is displayed.

  7. No of Units - Enter the Number of Units in alphanumeric format of six characters maximum length. This is a mandatory field.

  8. Select/Deselect All - Select the check box to select the generated pick list. Clear the check box to clear the selection of the Items from the pick list.

If the requested quantity is greater then the quantity available, the system invokes a pop-up message ‘No sufficient  Quantity, item cannot be finalized. Do you want to send a request to CSSD?’. Click [Yes] to invoke the Request screen in Sterile Stock module or click [No] to display an alert message ‘Equipment/Instrument not requested but added into the pick list; kindly regenerate the Pick list when equipment/Instrument is available’.  The message alert occurs when the Operation Theatre module is interfaced with Sterile Stock module.

Personal Details

  1. Patient ID - The Patient ID of the selected operation is displayed.

  2. Operation - The Operation recorded is displayed.

  3. Booking No - The Booking Number for the Operation is displayed.

  4. Item Type - Select the Item Type from the list. The listed value is Resource Type. This is a mandatory field.

  5. Item Description - Select the Item Description using [?] function. The Items are displayed based on the selected Item Type. This is a mandatory field.

  6. No of Persons - Enter the Number of Persons in alphanumeric format of six characters maximum length. This is a mandatory field.

  7. Select/Deselect All - Select the check box to select the generated pick list. Clear the check box to clear the selection of the Items from the pick list.

 

  1. Select the check box under ‘Finalise’ to finalize the item for the operation.

  2. Click Item hyperlink to display the selected item under Surgical Accessories. Click [Cancel] to delete the selected item from the Surgical Accessories list. Click [Finalise] to finalize the items for operation. A message is displayed on successful finalizing.

  3. New items are added to the list by entering the details under Surgical Accessories.

  4. Click [Select] to add the item.

 

Print

This tab is used to print the pick list details.

  1. Schedule Date - Select the date for which the Pick List has to be printed. This is a mandatory field.

  2. Theatre Type - Select the Theatre Type from the list.

  3. Theatre - Select the Theatre from the list.

  4. Patient ID – Enter the patient ID or select using [?] function.

  5. Patient Name – Patient name is displayed based on the selected Patient ID.

Click [Accept] to print the Pick List.

Select the required report option in the Report Options screen and click [OK].

The ‘Finalised’ items are displayed in the generated report.
