Surgery History

From this function you can Query the details of surgery performed on a patient.

Surgery History query function is used to view the surgery details like: Operation Date, brief description of the operation, the surgeon involved in the operation, the current status of the patient and surgeon notes. The Surgery Status is displayed with a unique colour code.

Querying Surgery History

From OT Queries, select the Surgery History function.

The Surgery History screen gets displayed.

  1. Enter the Patient ID in alphanumerical format of length as specified in the MP Parameter or select the Patient ID using [?] function. This is a mandatory field.

  2. Click [Search] to display the surgery history details for the selected patient or enter the search criteria and click [Search] to display the specific record. Click [Clear] to reset the records displayed in the result area.

    Enter the following search criteria:

    1. Patient ID – Enter the name of the patient in alphanumeric format of 10 characters maximum length or select the Patient ID using [?] function. This is a mandatory field.

    2. Patient Name - Patient Name corresponding to the selected Patient ID is displayed.  

    3. Gender - The Gender of the patient is displayed, by default based on the selected Patient ID.

    4. Birth Date - The Date of Birth of the patient is displayed, by default based on the selected Patient ID.

    5. Facility - Select the Facility ID from the list. The logged in facility name is displayed, by default.

    6. Surgeon – Enter the Surgeon name in alphanumerical format of 15 characters maximum length or select it using [?] function.  

    7. Date [From-To] - Enter the date range to view the surgery history in dd/mm/yyyy format or select the date range using Calendar function.

    8. Operation – Enter the name of the operation or select it using [?] function. The Operations/Procedures defined in the Operations function get populated in the list.

  3. Click ‘Operation Date’ hyperlink to view the details for following options:

Patient Demographics

Patient History

View Booking Details

Anesthesia History

  1. Click ‘Details’ hyperlink to view the surgery history details.

  2. Click ‘Notes’ hyperlink to view the Holding Notes, Surgeon Notes and Recovery notes.

  3. Click [Reset] to clear the entered data.
