Order Type for Specialty

From this function you can Associate Order Types to a specialty and Query for the existing records.

'Order Types for Specialty' function enables you to associate Order Types created in 'Order Type' function of Results and Reports module to a Specialty. The Order Types created with Order Category 'OT Surgery' are listed for association. Order Types associated to a particular Specialty are not available for selection to other Specialties.

Associating Order Types to a Specialty

From OT Code Setup > Operations, select the Order Type for Specialty function.

The Order Type for Specialty screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Specialty from the list. This is mandatory field.

  3. Click [Search] to list the Order Types created with Order Category 'OT Surgery' in the Results and Reports module. Select the check box against the required Order Type to associate to the Specialty.

  4. Click [Apply] to save the association of Order Types to the Specialty or [Clear] to clear the entered data.

  5. Click [Query] to search for existing Order Types associated to a Specialty.
