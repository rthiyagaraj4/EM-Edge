Action Type for Reason

From this function, you can Associate Reasons to various action types and Query for the existing records.

The reasons applicable to the actions like Cancel Appointment, Cancel Check-in Case, Cancel Waitlist, Booking Sequence Change, Transfer/Reschedule Appointment and Transfer back to OT are specified in this function.

Associating Action Type to Reason

From OT Code Setup > Others, select the Action Type for Reason function.

The Action Type for Reason screen gets displayed.

  1. Click [Create] to invoke the create screen

  2. Select Reason Code using the [?] function. The Reasons created in the Reason function are listed for selection.  

  3. Click [Select] to display the Action Types at the bottom of the screen. The Action Types displayed are Cancel Appointment, Cancel Check-in Case, Cancel Waitlist, Booking Sequence Change, Transfer/Reschedule Appointment and Transfer back to OT.

  4. Select the check box against the required Action Type to associate the selected reason.

  5. Click [Apply] to save the Action Type for Reason record or [Reset] to clear the entered data.

  6. Click [Query] to search for existing records.
