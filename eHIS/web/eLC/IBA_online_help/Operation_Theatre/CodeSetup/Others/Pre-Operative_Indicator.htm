Pre-Operative Indicator

From this function, you can Define the Pre-operative indicators and Query for the existing records.

Pre-operative Indicator functions enables defining various individual components which can be grouped together to form the vital signs or patient care panel.

Defining Pre-Operative Indicators

From OT Code Setup > Others, select the Pre-Operative Indicator function.

The Pre-Operative Indicator screen gets displayed.

  1. Click [Create] to invoke the create screen.

  2. Select the required Description using the [?] function. The Discrete Measures are defined in the Discrete Measure function of the Application Masters module.

  3.  The Discrete Measure code of the selected discrete measure is displayed on selecting the description.

  4. Click [Apply] for to save the Pre-Operative Indicator record or [Reset] to clear the entered data.

  5. Click [Query] to search for existing Pre-Operative Indicator records.
