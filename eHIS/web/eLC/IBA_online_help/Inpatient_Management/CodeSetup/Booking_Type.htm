Booking Type

From this function you can Define Booking Types and Query for the existing Booking Types.

The Booking type function is used to create the type of booking performed in the facility. You can associate the Booking type created to the booking indicators ‘Internal Hospital booking’, ‘External Hospital booking’ and ‘New Booking’. The defined Booking Types are associated to the patient, while performing the booking.

Creating Booking Type

From the IP Setup > IP Code Setup> Booking, select the Booking Type function.

The Booking Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Booking Types defined or [Reset] to clear the entered data.

  4. Click [Query] to search for the existing Booking Type records.
