Practitioner for Nursing Unit

From this function you can Create an association of one or more Practitioners to a particular Nursing Unit or associate one or more Nursing Units to a particular Practitioner. You can Query for the desired Practitioner for Nursing Unit records.

A Practitioner is identified as an admitting or attending practitioner only when the practitioner has appropriate admitting rights against a Nursing Unit. Associating practitioners to Nursing Units provide admitting rights for practitioners against those nursing units.

Creating an association of Practitioners to Nursing Units

From the IP Setup > IP Code Setup, select the Practitioner for Nursing Unit function.

The Practitioner for Nursing Unit screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select By Nursing Unit tab to associate one or more Practitioners to a particular Nursing Unit or select By Practitioner tab to associate one or more Nursing Units to a particular Practitioner.

  3. Enter the details in the mandatory (*) fields. <Related Field Description>

  4. Click [Search] to display Practitioners or Nursing Units and select the Practitioners or Nursing Units.

    1. Nursing Unit – This field displays the Nursing Units.

    2. Select All – Select the check box to indicate applicability of the Nursing Units to the selected practitioner. To select all the displayed nursing units, select the check box provided in the column header. All the check boxes are cleared, by default.

    1. ‘+’ - On clicking the hyperlink ‘+’, the Practitioner Details screen gets displayed with details of the practitioner. All the details of the practitioner entered during the creation of practitioner in Application Masters module are displayed under the tabs Identity, Contact No./Addr and Qualification Details tabs.

    2. Practitioner Name – This field displays the names of the practitioners.

    3. Practitioner Type – This field displays the Practitioner Type of practitioners.

    4. Select All – Select the check box to indicate applicability of the Practitioner for the selected Nursing Unit. To select all the displayed practitioners, select the check box provided in the column header. All check boxes are cleared, by default.

  5. Click [Apply] to save the association of Practitioners to Nursing Unit or [Reset] to clear the entered data.

  6. Click [Query] to search for associated Practitioner for Nursing Unit records.
