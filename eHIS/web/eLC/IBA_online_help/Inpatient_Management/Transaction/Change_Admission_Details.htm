Change Admission Details

From this function you can Change the Admission Details of a patient.

This function is used to change the admission details based on the location for operator station.  If the practitioner is associated to more than one specialty, you can change the patient only to those specialties associated to the Nursing Unit. This function is enabled only if the Change Admission Details check box in the Operator Station function, under the Inpatient/Admission tab in the Application Masters function. The user should have the access to the Operator Station to change the details.

Changing Admission Details

From IP Transactions > Admission, select Change Admission Details function.

The Change Admission Details screen gets displayed.

  1. Click [Search] to display the list of all admitted patients or enter the search criteria and click [Search] to select a specific admitted patient. Click [Clear] to reset the records displayed in the result area.

    1. Nursing Unit - Enter the Nursing Unit in alphanumeric format of 15 characters maximum length or select the Nursing Unit using the [?] function.

    2. Admission Date [From-To] - Enter the Admission Date range in format of dd/mm/yyyy or select the date using the Calendar function. The current date is displayed, by default.

    3. Specialty - Enter the Specialty in alphanumeric format of 15 characters maximum length or select the Specialty using the [?] function.

    4. Practitioner - Enter the name of the Practitioner in alphanumeric format of 30 characters maximum length or select the Practitioner using the [?] function.

    5. Encounter ID - Enter the Encounter ID in numeric format of 12 digits maximum length.

    6. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the Patient ID using [?] function.

  2. On searching, the Admission details are displayed based on the entered query criteria.

  3. Click '+' hyperlink to view the Current Encounter Details.

  4. Click the 'Encounter ID' hyperlink to display the Change Admission Details screen. The screen displays Current Admission Details as recorded during patient admission.

  5. Enter the details in the mandatory fields. <Related Field Description>

  6. Click [Record] to save the changed admission details or [Close] to exit from screen.

If service charges exist for the admission of the patient; on click of Encounter ID, you are alerted by a message ‘Service charge exists for this admission. Only Service, Sub-Service, Specialty, Expected Discharge Date and Admitting Practitioner are allowed to Change’.
