Cancel Discharge Patient

From this function you can Cancel the discharged patients, discharged from the Inpatient encounter.

This function allows you to cancel the patient discharge for the current discharge date as well as for the discharges done prior to the current date.

Canceling Patient Discharge

From the IP Transactions > Discharge, select the Cancel Discharge Patient function.

The Cancel Discharge Patient screen gets displayed.

  1. Click [Search] to display the list of discharged patients or enter the search criteria and click [Search] to select a specific discharged patient. Click [Clear] to reset the records displayed in the result area.

    1. Nursing Unit – Enter the Nursing Unit or select the Nursing Unit using the [?] function. The Nursing Units that displays in the lookup are filtered based on the Locations associated with the Operator Station of the logged in User.

    2. Admission Date [From-To] – Enter the Admission Date range in the format of dd/mm/yyyy.

    3. Specialty – Enter the Specialty or select the Specialty using the [?] function.

    4. Practitioner – Enter the Practitioner or select the Practitioner using the [?] function.

    5. Encounter ID – Enter the Encounter ID in numeric format of 12 digits maximum length.

    6. Gender – Select the Gender from the list. The listed values are ‘All’, ‘Male’, ‘Female’ and ‘Unknown’. The value ‘All’ is displayed, by default.

    7. Discharge Date [From–To] – The current system date is displayed, by default. Select the required Discharge Date.

    8. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the Patient ID using the [?] function.

  2. On searching, discharged patient details are displayed based on the entered criteria.

  3. Click ‘+’ hyperlink to view the Current Encounter Details.

  4. Click the ‘Encounter ID’ hyperlink to display the Cancel Discharge Patient screen. The screen displays the Encounter Details, Nursing Unit Details, Discharge Details, Deceased Details, and Additional Discharge Details recorded while discharging the patient. If the Discharge Bill is raised, a message ‘Discharge Bill is Raised. Discharge Cannot Be Cancelled’ is displayed. Cancel the Discharge Bill in Financial Base module to proceed with the Cancel Discharge Patient function.

  5. Select the ‘Reason for Cancellation’ from the list. The Reason is created in the ‘Reason’ function of Applications Masters module.

  6. Enter the ‘Bed No’ or select the Bed Number using the [?] function.

  7. Enter the Cancelled By ID or select the ID using the [?] function.  

  8. Click [Record Cancel Discharge] to cancel the discharge or [Abort Cancel Discharge] to exit from Cancel Discharge Patient screen.
