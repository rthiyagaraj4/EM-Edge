Renew Gate Pass

From this function you can Renew Gate Pass of an Accompanying Person as defined in the IP Parameter function.

Gate passes are issued to people accompanying Inpatients on checking-in to the facility. This function is used to renew the issued gate pass. Renewal rules are defined in IP Parameter.

Renewing Gate Pass

From the IP Transactions > Lodger/Accompanying Person, select the Renew Gate Pass function.

The Renew Gate Pass screen gets displayed.

  1. Click [Search] to display all Accompanying Persons or enter the search criteria and click [Search] to select a specific Accompanying Person. Click [Clear] to reset the records displayed in the result area.

    Enter the following query criteria:

    1. Nursing Unit – Select the Nursing Unit using [?] function.

    2. Patient ID – Select the Patient ID of the Inpatient using [?] function.

    3. Checked In Date [From/To] – Enter the date range in date format of dd/mm/yyyy or select the date from the calendar function as to when the Accompanying Person was checked-in.

    Accompanying Person Details

    1. Citizen ID – Select or enter the Citizen ID and other Alternate IDs in the format as specified in the Master Patient Index module.

    2. Other Alt Type – Select the Other Alt Type from the list. The listed values are Birth Certificate, Passport, Mother’s PMI, PMI, etc.

    3. Other Alt ID - Enter the Alternate ID in alphanumeric format of maximum length as specified in the MP Parameter.

    4. Previous HN – Enter the Previous Hospital Number in alphanumeric format of maximum length as specified in the MP Parameter.

    5. Government No - Enter the Government Number in alphanumeric format of maximum length as specified in the MP Parameter.

    6. Driving License No - Enter the Driving License Number in alphanumeric format of maximum length as specified in the MP Parameter.

    7. Expatriate No - Enter the Expatriate Number in alphanumeric format of maximum length as specified in the MP Parameter.

    8. Person ID – Select the Accompanying Person ID using [?] function.

    9. Gender – Select the Gender of the Accompanying Person from the list. The listed options are Male, Female and Unknown.

    10. Search Name With Local Lang – Select the check box to search the name with the Local Language.

    11. Title – Select the Title from the list. The listed values are defined in the function Name Prefix.

    12. First Name – Enter the First Name of the Accompanying Person in alphanumeric format of 15 characters maximum length.

    13. Second Name – Enter the Second Name of the Accompanying Person in alphanumeric format of 15 characters maximum length.

    14. Third Name – Enter the Third Name of the Accompanying Person in alphanumeric format of 15 characters maximum length.

    15. Family Name – Enter the Family Name of the Accompanying Person in alphanumeric format of 15 characters maximum length. This field is mandatory.

    16. Suffix – Select the suffix from the list. The list of pre-defined suffix is listed in the list.

    The fields and their order displayed in the screen may vary depending on the MP Parameter Setup.

  2. On searching, Accompanying Person Details are displayed based on the entered criteria.

  3. Click ‘+’ hyperlink to view the Current Encounter Details of the patient.

  4. Click ‘Accompanying Person Details’ hyperlink to display the Renew Gate Pass screen. The screen displays Accompanying Person Details, Contact details, Current Validity Details and Renewal Validity Details.

  5. Click [Contact Details] to view the Address Details.

  6. Enter the Renewal Validity Details in the mandatory (*) fields.

    Renewal Validity Details

    1. Period From – The Current Validity Details Period From date and time is displayed, by default. Enter the ‘Period From’ date and time or select the date using the calendar function. The entered date must not be less than the default value. This field is mandatory.

    2. Period To – Enter the period until when the Gate Pass is valid in date format of dd/mm/yyyy hh:mm. This field is mandatory. The entered date must be within the period set in IP Parameter (Allowable Period for Issue/Renewal of Gate Pass []).

    3. Authorized By – Enter the name or select the name of the person who authorized the renewal of gate pass. This field is mandatory.

    4. Remarks – Enter the Remarks in alphanumeric format of 200 characters maximum length.

  7. Click [Renew Gate Pass] to renew the gate pass or [Close] to exit from the screen.

  8. If internal reports are defined, the Internal Reports screen gets displayed. Click [OK] to generate Internal Reports or [Cancel] to exit from the screen.
