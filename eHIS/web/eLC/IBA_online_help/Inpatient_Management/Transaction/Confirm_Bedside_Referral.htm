Confirm Bedside Referral

From this function you can Confirm that the Bedside Referral recorded during the Inpatient referral registration process has been attended to and record the Date/Time when the referral was attended.

When the 'Bedside Referral' check box selected in the 'Register Inpatient Referral' function, you can confirm that the bedside referral recorded during the Inpatient referral registration process has been attended.

Confirming a Bedside Referral

From the IP Transactions > Inpatient Referral, select the Confirm Bedside Referral function.

The Confirm Bedside Referral screen gets displayed.

  1. Click [Search] to display the list of Inpatients for which 'Bedside Referral' check box is selected during Inpatient Referral Registration or enter the search criteria and click [Search] to select specific Inpatients having Bedside Referral. Click [Clear] to reset the records displayed in the result area.

    Enter the following criteria to locate bedside inpatient referrals:

    1. Nursing Unit – Enter the Nursing Unit description in alphanumeric format of 15 characters maximum length or select the Nursing Unit using [?] function.

    2. Specialty – Enter the specialty in alphanumeric format of 15 characters maximum length or select the specialty using [?] function.

    3. Admission Date From–To – Enter the admission date range in date format of dd/mm/yyyy or select the date using the Calendar function.

    4. Encounter ID – Enter the Encounter ID in numeric format of 12 digits maximum length.

    5. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the ID using [?] function.

    Referred To

    Select criteria that correspond to the referred details recorded in the register inpatient referral function, for a bedside referral. The query criteria under this section are:

    1. Specialty – Select the Specialty from the list.

    2. Priority – Select the Priority from the list. The various priorities are ‘Elective’, ‘Emergency’ and ‘Semi Emergency’.

    3. Location – Select the Location type from the list. The values are Clinic and Daycare. The locations corresponding to the Location type are displayed. Select a value from the list of locations.

    4. Practitioner – Enter the Practitioner name or select the practitioner from [?] function.

    5. Preferred Date – Enter the Preferred Date in dd/mm/yyyy format or select the preferred date from the Calendar function.

    Note: You can enter past, current, or future dates in this field and confirm bedside referrals.

    1. Referral ID – Enter the Referral ID in alphanumeric format of 14 characters maximum length. The Referral ID numbering is defined in Application Masters module.

  2. On searching, Bedside Referral details of patients are displayed based on the entered criteria.

  3. Click ‘+’ hyperlink to view the Current Encounter Details.

  4. Click the 'Referral ID' hyperlink to display the Confirm Bedside Referral screen. The screen displays Encounter Details, Nursing Unit Details and Referral details as recorded during Inpatient Referral Registration.

  5. Enter the Bedside Referral Seen Date in dd/mm/yyyy format or select the date from the Calendar function on which the bedside referral patient is seen. This field is mandatory.

  6. Click [Record] to confirm the Bedside Referral or [Close] to exit from the screen.
