Register Inpatient Referral

From this function you can Refer current Inpatients of the logged in facility to other care location types within the facility or to another facility of the enterprise or to an external facility.

Inpatients are usually referred for further treatment/diagnosis/tests/procedures.

Registering an Inpatient Referral

From the IP Transactions > Inpatient Referral, select the Register Inpatient Referral function.

The Register Inpatient Referral screen gets displayed.

  1. Click [Search] to display the list of all Inpatients or enter the search criteria and click [Search] to select specific Inpatients. Click [Clear] to reset the records displayed in the result area.

    1. Nursing Unit – Enter the Nursing Unit description in alphanumeric format of 15 characters maximum length or select the Nursing Unit using [?] function.

    2. Admission Date From-To – Enter the Admission Date range in date format of dd/mm/yyyy, or select the date using the Calendar function.

    3. Specialty – Enter the Specialty in alphanumeric format of 15 characters maximum length or select the Specialty using [?] function.

    4. Practitioner – Enter the Practitioner name in alphanumeric format of 30 characters maximum length or select the practitioner from [?] function.

    5. Encounter ID – Enter the Encounter ID in numeric format of 12 digits maximum length.

    6. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or select the ID using [?] function.

  2. On searching, Inpatient details of patients are displayed based on the entered criteria.

  3. Click ‘+’ hyperlink to view the Current Encounter Details.

  4. Click Encounter ID hyperlink to display the Register Inpatient Referral screen. The screen displays the Encounter Details and Nursing Unit Details as recorded during the patient admission.

  5. Enter the Referral Details in the mandatory (*) fields. <Related Field Description>

  6. Click [Record] to save the transaction on entering the required details or [Close] to exit from the Register Inpatient Referral screen.

  7. On clicking [Record], an alert message gets displayed prompting if another referral needs to be recorded. Click [OK] to save the details entered and the control shifts back to the Register Inpatient Referral screen with all the fields reset allowing you to record another referral for the Inpatient. Click [Cancel] and the control shifts back to the query result screen.

 

It is not mandatory to record all the details during Inpatient referral registration.
