Booking Limit by Nursing Unit – Bed Class

From this function you can Create Booking Limits for a Nursing Unit by specifying the maximum daily booking allowed based on the Bed Class and Gender and Query for the Booking Limit set for Nursing Unit, Bed Class and Gender combination.

This function allows you to set limits for controlling bed booking. The create screen gets displayed only if the field Check Booking Limit based on in IP Parameter function is selected as Nursing Unit – Bed Class.

Creating a Booking Limit by Nursing Unit – Bed Class

From the IP Setup > IP Misc Setup, select the Booking Limit by Nursing Unit – Bed Class function.

The Booking Limit by Nursing Unit – Bed Class screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Search] and enter the maximum daily booking limits for Male, Female and Unknown patients. The Total number of bookings are displayed. Select the check box against the Bed Class to associate to the Nursing Unit.

    Select – Select the check box to enable the booking limit set for a bed class associated to the Nursing Unit. The values specified for the booking limit are validated for the selected Nursing Unit when the patients are booked using Create Booking function. Clear the check box to clear the entered values and to disassociate the bed class from the selected Nursing Unit.

    Total - The total number of daily booking permitted for the bed class (for male, female and unknown patients) is calculated and displayed automatically on setting the values in ‘male’, ‘female’ and ‘unknown’ fields.

    Unknown – Enter the maximum number of daily bed booking permitted for unknown gender patients against each bed class displayed in numeric format of six digits maximum length. Enter the value if the Nursing Unit allows unknown gender patients, or this field gets disabled.

    Female – Enter the maximum number of daily bed booking permitted for female patients against each bed class displayed in numeric format of six digits maximum length. Enter the value if the Nursing Unit allows female patients or this field gets disabled.

    Male – Enter the maximum number of daily bed booking permitted for male patients against each bed class displayed in numeric format of six digits maximum length. Enter the value if the Nursing Unit allows male patients or this field gets disabled.

  4. Click [Apply] to save the records or [Reset] to clear the entered data.

  5. Click [Query] to search for existing records.

 

Depending on the age group associated to the Nursing Unit, the system restricts you to enter the maximum daily booking limit for a particular bed class.

In Application Masters module, if the age group is entered as ‘Between 11 to 20 years’ and is associated to the female gender, the system allows you to enter the values only to the female column on selection of a bed class. If the selected age group is not associated with any gender during setup, the system allows you to enter values into male, female and unknown columns on selection of the bed class.
