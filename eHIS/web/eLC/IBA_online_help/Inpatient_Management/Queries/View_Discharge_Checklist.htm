View Discharge Check List

From this function you can Query the list of Discharge checklists within the facility.

The function provides a set of query criteria to filter the Discharge checklist listing. The query result is categorized on Specialty/Practitioner combination and the Patients with Discharge checklists are grouped under these combinations.

Querying for Discharge Check List

From IP Queries> Admissions/Transfers, select the ‘View Discharge Check List’ function.

The View Discharge Check List screen gets displayed.

  1. Enter the details in the fields. <Related Field Description>

  2. Click [Search] to select the records that satisfy the query criteria or [Reset] to clear the entered search criteria.

  3. On searching, the Discharge checklist details are displayed.

  4. Click the '+' hyperlink, the system displays the Current Encounter Details. The information displayed is categorized as Patient details, Encounter details and Stay details.

  5. Select a record by clicking the patient's ‘Encounter ID’ hyperlink from the main result screen. The Discharge Checklist for the patient is categorized under four headings.

• Discharge Details

• Additional Details

• Health Education

• Appl./Fixtures Documents

  1. Click [Print] to print the Discharge Checklist or [Close] to close the Discharge Checklist screen.

  2. Click [Clear] to clear the result screen.
