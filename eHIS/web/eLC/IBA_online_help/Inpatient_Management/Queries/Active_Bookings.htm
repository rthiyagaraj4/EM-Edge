Active Bookings

From this function you can Query the details on active bookings for Inpatient admissions to the logged in facility.

This function provides various query conditions to filter the result. On specifying the query conditions, the result set provides complete information on the booking details recorded. The preferred Admission date and time, patient details and nursing unit details are displayed. You are provided with booking status to enable resource planning for the anticipated admissions.

Querying for Active Bookings

From IP Queries> Bookings, select the Active Bookings function.

The Active Bookings screen gets displayed.

  1. Enter the details in the mandatory fields to query the Active Bookings. <Related Field Description>

  2. Click [Search] to view the Active Bookings details or [Reset] to clear the entered search criteria.

  3. On searching, the Active Bookings details are displayed.

  4. The result screen is categorized as 'Urgent' and 'Non-urgent'. The Booking reference number of the patient is displayed and hyperlinked. On clicking the 'Booking Ref No' hyperlink, the system displays the Active Booking details. The information screen displayed is categorized into Patient details, Nursing Unit details and Bed Details.

  5. Click [Clear] to clear the result screen.
