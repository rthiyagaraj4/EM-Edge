Enter the following details:

 

  1. Nursing Unit – Enter the Nursing Unit in alphanumeric format of 15 characters maximum length, or search for the Nursing Unit by clicking [?] function provided. This field is mandatory. Enter the required search criteria in the search screen and click [Search] to display the Nursing Units that satisfy the entered search criteria. If the Nursing Unit description is known, directly search without entering any description.

The following details are displayed on the main screen on selecting the Nursing Unit:

  1. Nursing Unit Type – The Nursing Unit Type associated to the Nursing Unit is displayed.

  2. Age Group –The Age Group associated during the creation of Nursing Unit is displayed.

  3. Account Department – The Department to which the Nursing Unit is associated is displayed.

  4. Service – The Service attached to the Nursing Unit is displayed.

  5. Search By – Select the Search Criteria as either Specialty Code or Specialty Description from the list and enter the required search string in the text box based on the selected criteria.
