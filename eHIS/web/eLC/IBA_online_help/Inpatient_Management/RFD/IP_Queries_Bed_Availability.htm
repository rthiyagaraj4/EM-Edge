Enter the following search criteria:

 

  1. Nursing Unit - Select the Nursing Unit from the list. The Nursing Units defined in the Nursing Unit function is displayed for selection. This is a mandatory field if Specialty is not selected.

  2. Specialty - Select the Specialty from the list. This is a mandatory field if Nursing Unit is not selected.

  3. Include Bed to be released in () Hrs - Enter the period in hours in numeric format of five digits maximum length. The result set includes all those beds that would be released within the specified period.

  4. Practitioner - Select the Practitioner using [?] function.

  5. Bed Class - Select the Bed Class from the list.

  6. Bed Type - Select the Bed Type from the list.

  7. Bed Classification - Select the Bed Classification from the list. The values displayed are Babies Beds, Normal Beds, Pseudo Bed and All. The value 'Normal Beds' is displayed, by default.

  8. Bed Status - Select the Bed Status from the list. The values displayed are All, Booked, Vacant, Occupied and Reserved. ‘Vacant’ is displayed, by default.

  9. Gender Type - Select the Gender Type between the two options given. The two options are ‘Patient Gender’ and ‘Bed Gender’. Patient Gender is the gender of the Patient while the Bed Gender denotes the beds assigned for a particular Gender. The Gender Type displayed are Male, Female or Unknown.

  10. Gender - Select the Gender from the list. The values displayed are Male, Female and Unknown.

  11. Bed Availability Date [From-To] - Enter the Bed Availability Date range using the Calendar function.

  12. Age - Enter the Age in numeric format of five digits maximum length and select the units as Days, Months or Years from the list.

  13. Room/Cubicle Status - Select the Room (attached to the nursing unit) with specific privacy level setting from the list. The listed values are Isolation, Private Room, Semi Private Room and Room/Cubicle.

  14. Group By - Select the Group By from the list. You can group by ‘Specialty’, ‘Bed Type’ ‘Room Cubicle’ and ‘Admission Type’. The ‘Admission Type’ is listed only when the ‘Bed Status’ selected is ‘Occupied’ or ‘Reserved’.

If the Gender Type is selected as ‘Bed Gender’ and the corresponding bed gender is selected from the ‘Gender’ list, the search result displays the beds based on the gender of the bed, if any, and not on the gender of the patients.
