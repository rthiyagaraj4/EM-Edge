Admission Details

  1. Admission Date/Time – The admission date and time of the patient is displayed.

  2. Admission Type – The admission type is displayed.

  3. Weight on Admission – Enter the weight of the patient at the time of admission. This field is mandatory if the function name ‘Assign Bed/Mark Patient Arrival’ is selected in the ‘Weight on Admission Mandatory’ drop-down list in IP Parameter. The Unit of measure displayed next to text box is displayed dynamically based on the patient age. The Unit of measure is displayed as ‘Grams’ if age is Less than or equal to 30 days. You can enter 5 digits without any decimals. The Unit of measure is displayed as ‘Kg’ if the patient age is greater than 30 days. You can enter 3 digits before decimal and 3 digits after decimal.

 

Nursing Unit Details

  1. Nursing Unit – Enter the value or select the Nursing Unit from the [?] function. This field is mandatory. The nursing unit is displayed as recorded.

  2. Service – Select the Service from the list. This field is mandatory. The service is displayed as recorded.

  3. Specialty – Enter the value or select the Specialty from the [?] function. This field is mandatory. The specialty is displayed as recorded while admitting the patient.

  4. Admitting Practitioner – The Admitting Practitioner is displayed as read-only text.

  5. Attending Practitioner – Enter the value or select the Attending Practitioner from the [?] function. This field is mandatory. This field is displayed if recorded while admitting the patient.

  6. Medical Team – Select the Medical Team from the list. The Medical Team is displayed if recorded.

  7. Procedure/Surgery Date Time – The date of procedure or surgery to be performed for the patient is displayed as read-only text.

  8. Bed Class – The Bed Class is displayed and disabled if it was recorded while admitting the patient.

  9. Bed Type – The Bed Type is displayed and disabled if it was recorded while admitting the patient.

  10. Bed Number – Enter the Bed Number in alphanumeric format of eight characters maximum length or select the bed number using the [?] function. This field is displayed if it was recorded.

  11. Room/Cubicle Number – The Room/Cubicle Number is displayed once the bed is selected.

  12. Bed Allocation Date/Time - The current date and time is displayed and disabled as the date and time of bed allocation. This field is mandatory.

  13. Arrival Date/Time - The current date and time is displayed and disabled as the date and time of the patient’s arrival. This field is mandatory.
