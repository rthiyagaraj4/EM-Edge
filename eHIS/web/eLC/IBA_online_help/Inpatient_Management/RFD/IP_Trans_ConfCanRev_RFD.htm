Enter the following query criteria:

 

  1. Nursing Unit – Enter the Nursing Unit in alphanumeric format of 15 characters maximum length or select using [?] function. All the Nursing Units associated to the Operator Station of the logged in user is listed.

  2. Priority – Select the priority from the list. The values displayed are Urgent, Non Urgent and Both. Based on the value selected, the booking records created are displayed on query execution.

  3. Specialty – Enter the Specialty in alphanumeric format of 15 characters maximum length or select the Specialty using [?] function. All the specialties defined in Application Masters specialty function are displayed.

  4. Practitioner – Enter the Practitioner in alphanumeric format or select the Practitioner using [?] function. All the practitioners attached to the logged in facility in ‘Practitioner for Facility’ function of Application Masters module are displayed.

  5. Bed Class – Select the Bed Class from the list.

  6. Bed Type - Select the Bed Type from the list. The Bed Types listed are based on the bed class selected.

  7. Medical Service Group – Select the Medical Service Group from the list.

  8. Booking Type – Select the Booking Type from the list. The listed values are defined in the Booking Type function.

  9. Preferred Date From-To – Enter the Preferred Date range in dd/mm/yyyy format or select the date using the Calendar function. The ‘From’ date has to be greater than or equal to the current system date and the ‘To’ date has not to be lesser than the ‘From’ date.

  10. Booking Ref No – Enter the booking reference number in alphanumeric format of 12 characters maximum length.

 

Patient Details

The fields and their order in the Patient Details are displayed as specified in the MP Parameter.

  1. Citizen ID - Enter the Citizen ID and other Alternate IDs in alphanumeric format of maximum length as specified in the MP Parameter. The ‘Prompt’ specified in MP Parameter of Master Patient Index module gets displayed as the field legend.

  2. Other Alt Type – Select the Other Alt Type from the list. The listed values are Birth Certificate, Passport, Mother’s PMI, PMI, etc.

  3. Other Alt ID - Enter the Alternate ID in alphanumeric format of maximum length as specified in the MP Parameter.

  4. Previous HN – Enter the Previous Hospital Number in alphanumeric format of maximum length as specified in the MP Parameter.

  5. Government No - Enter the Government Number in alphanumeric format of maximum length as specified in the MP Parameter.

  6. Driving License No - Enter the Driving License Number in alphanumeric format of maximum length as specified in the MP Parameter.

  7. Expatriate No - Enter the Expatriate Number in alphanumeric format of maximum length as specified in the MP Parameter.

  8. Patient ID - Enter the Patient ID in the format and length as specified in the MP Parameter. The ID entered has to be unique and valid. On clicking [?] function, the ‘Patient Search’ window gets displayed which enables selecting the required Patient ID.

  9. Gender – Select the Gender from the list. The listed options are Male, Female and Unknown.

  10. Search Name with Local Lang – Select the check box to search with the Patient name in local language.

  11. Patient Name Components – The query components and the legends pertaining to the patient name depends upon the setup done in MP Parameter under the ‘Legal Names’ Tab. The available components for the patient name are Title, First Name, Second Name, Third Name, Family Name and Suffix. The components that are made applicable for the facility appears as query criteria. The appropriate patient name can be searched by using the search options 'Starts With', 'Contains' and 'Ends With'.
