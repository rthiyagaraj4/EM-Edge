Enter the following details to create a new Medical Service Group record:

 

  1. Code - Enter the Medical Service Group Code in alphanumeric format of two characters maximum length. A Medical Service Code is used to identify each Medical Service Group uniquely and cannot be changed after creation. This field is mandatory.

  2. Long Description - Enter the long description of the Medical Service Group in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description - Enter the short description of the Medical Service Group in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. Enabled – Select the check box to enable the Medical Service Group.

 

Booking Related Setup

  1. Max Daily Booking – Enter the Maximum Daily Booking for Inpatient/Daycare admissions for the Medical Service Group in numeric format of six digits maximum length. It indicates the maximum allowable bookings per day for the defined medical service group. This field is set to control the number of daily bookings that can be accepted by the facility for a medical service group. If the maximum daily booking exceeds the value set, the system alerts you with a warning, which cannot be overridden. If the field is not set, the system does not validate for the number of bookings made in a day.

  2. Max Monthly Booking – Enter the Maximum Monthly Booking for Inpatient/Daycare admissions for the Medical Service Group in numeric format of six digits maximum length. It indicates the maximum allowable bookings per month for the defined medical service group. This is to control the number of monthly bookings that are accepted by the facility. If the number of maximum monthly bookings exceeds the value set, the system gives a warning to you, which cannot be overridden. If this value is not specified, the system does not check the number of monthly bookings for the Medical Service Group. The value being set for Max Monthly booking must always be greater than the Max Daily Booking.

  3. Generate Booking Number - Select the check box if the system has to generate booking numbers automatically. if the check box is cleared, the booking number has to be entered manually during Create Booking for admission against the selected medical service group., The check box is cleared, by default.

  4. Next No - Enter the Next No. in numeric format of 10 digits maximum length. This field is enabled only if the Generate Booking Number field is selected. The Next Serial Number indicates the next booking number the system generates. The value specified in this field is the first number generated by the system. Subsequently, this is automatically incremented by one by the system for every booking created. If the value is entered, it must be greater than zero.

  5. Max No - Enter the Max No. in numeric format of 10 characters maximum length. This field is enabled only if the Generate Booking Number field is selected. This is the maximum serial number to be generated by the system for the booking number. When the maximum number is reached, no further booking numbers are generated for the defined medical service group. At this stage, the System Administrator has to reset the Next No. appropriately. If the value is entered, it cannot be less than the Next No.
