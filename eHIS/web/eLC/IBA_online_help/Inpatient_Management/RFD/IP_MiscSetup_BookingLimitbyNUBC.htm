Enter the following details to define the booking limit for a bed class in a Nursing Unit:

 

  1. Nursing Unit – Enter the Nursing Unit in alphanumeric format of 15 characters maximum length, or search for the Nursing Unit by clicking [?] function. On invoking [?] function, the Nursing Unit search screen gets displayed.  Enter the required search criteria and click [Search] to display the Nursing Units that satisfy the entered search criteria. If the Nursing Unit description is known, enter the description directly and click [Search]. This field is mandatory.

Enter the following search criteria to select the required value:

  1. Search By – Select either the description or code as the value.

  2. Search Criteria - Select the required search criteria from the list. The listed options are Starts With, Ends With and Contains.

  3. Search Text - Enter the required search string.

On entering the required criteria, click [Search] and the result records are displayed.

The following details are displayed as result set for the specified conditions:

  1. Code – The Code of the Nursing Unit is displayed and hyperlinked.

  2. Description – The Nursing Units are displayed and hyperlinked.

Click the Code or Description hyperlink to select the required value. The selected value gets displayed in the corresponding field.

  • [Reset] – On clicking [Reset], the displayed records are cleared and you can enter the new search criteria to search for the record.

  • [Cancel] – On clicking [Cancel], the search screen gets terminated.

The following details are displayed in the main screen on selecting a Nursing Unit:

  1. Nursing Unit Type – The Nursing Unit Type associated to the selected nursing unit is displayed.

  2. Age Group –The Age Group associated to the selected nursing unit is displayed.

  3. Account Department – The name of the Accounting Department to which the nursing unit is associated is displayed.

  4. Service – The Service associated to the selected Nursing Unit is displayed.

  5. [Search] – Click [Search] to search set the maximum daily booking limit for a male, female and unknown Bed Class in a Nursing Unit.

  6. Bed Class – The Bed Class defined is displayed.
