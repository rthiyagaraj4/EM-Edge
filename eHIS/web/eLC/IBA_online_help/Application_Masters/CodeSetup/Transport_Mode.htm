Transport Mode

From this function you can Create a Transport Mode and Query for a required Transport Mode record.

Various modes of transportation are used to transport patients. The transportation details are recorded while registering the patients. Options to record modes of transport used to transport files and specimen are also provided.

Creating a Transport Mode

From the AM Other Setup > Other Master Setup, select the Transport Mode function.

The Transport Mode screen gets displayed.

  1. Click [Create] to invoke the create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Descriptions>

  3. Click [Apply] to save a Transport Mode created or [Reset] to clear the entered data.

  4. Click [Query] to search for an existing Transport Mode record.
