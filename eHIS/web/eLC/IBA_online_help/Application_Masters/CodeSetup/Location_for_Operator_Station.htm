Location for Operator Station

From this function you can Associate Locations to an Operator Station and Query for a required Location for Operator Station record.

The Location for Operator Station function is used to associate multiple locations to an Operator Station. On setting various Operator Stations of the Facility, locations that are controlled/grouped by each Operator Station is associated to the Operator Station using this function.

Associating Locations to an Operator Station

From the Operator Station Setup, select the Location for Operator Station function.

The Location for Operator Station screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter details in the mandatory (*) fields. <Related Field Description>

  3. Click [Search] to display the Locations based on the entered criteria.

  4. Select the 'Select All' check box to associate listed Locations to the Operator Station. Clear the 'Select All' check box to dissociate the locations from the Operator Station.

  5. Select the check box against the required locations to associate selected Locations to the Operator Station.  

  6. If the 'Allow Global Booking' check box is selected in Operator Station function, 'Global' check boxes are displayed. On selecting 'Select All' check box, 'Global' check box is enabled. Select the required Location to allow global booking for the Operator Station by selecting the appropriate check box. The check box is cleared, by default. If both the 'Select All' and 'Global' check boxes are selected, the Users associated to the selected location for the defined Operator Station is eligible for performing only global booking and not allowed to perform 'Over booking' in Appointment Scheduling module. Global Booking is applicable for Clinics, Procedure Units and Daycare Units. If only 'Select All' check box is selected, the users associated to the Operator Station are eligible to perform normal booking.

  7. Click [Apply] to save a Location for Operator Station record created or [Reset] to clear the entered data.

  8. Click [Query] to search the existing Location for Operator Station record.
