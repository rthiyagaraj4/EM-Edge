Facility for Accounting Department

From this function you can Associate applicable Accounting Departments to a Facility in an enterprise and Query for a required Accounting Department associated to a Facility.

Facility for Accounting Department function is used to associate applicable Accounting Departments to the facilities of an enterprise. An Accounting Department is associated to more than one facility. The Accounting Departments defined through the Accounting Department/Cost Center function are listed for association. Each Accounting Department consists of several sections where the bills are computed for the services utilized.

Associating Accounting Departments to a Facility

From the Service Setup, select the Facility for Acct Dept function.

The Facility for Acct Dept screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the required facility from the list. The logged-in facility is displayed, by default. The list box displays facility records as created in the Facility function of the System Manager module. This is a mandatory field.

  3. Select the Search By field as 'Account Department Code' or 'Account Department Description' and enter the required text in the text box provided to search for Accounting Departments based on Code or Description.

  4. Click [Search] to display Accounting Departments based on the entered search criteria. The Accounting Department records that are created through the 'Accounting Department/Cost Center' function are enabled and displayed, if the 'Search By' field is not specified.

  5. Select the check box against the applicable Accounting Department to associate the selected Accounting Departments to the Facility.

  6. Click [Apply] to save the Accounting Departments associated to a Facility or [Reset] to clear the entered data.

  7. Click [Query] to search for the existing records.
