Enter the following details to associate a Statistics Group to a Program Type:

 

  1. Module - Select the Module from the list of installed modules. This is a mandatory field.

  2. Program Type - Select the Program Type from the list. The pre-defined values are ‘Function’ and ‘Report’. This is a mandatory field.

  3. Program – Select a Program from the list. The values are listed based on the Program Type and Module selected. On selecting the Program Type as ‘Function’, all the functions in the module are listed for selection. On selecting the Program Type as ‘Report’, all the reports in the selected module are listed. This is a mandatory field.

  4. Statistics Group – Select the Statistics Group from the list. The Statistics Group created through the ‘Statistics Group’ function with the ‘Enabled’ check box as selected are listed. This is a mandatory field.

  5. Sequence Number – Enter the Sequence Number in numeric format of three digits maximum length. This is a mandatory field.
