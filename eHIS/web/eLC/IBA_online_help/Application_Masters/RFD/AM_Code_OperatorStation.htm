Enter the following details to create an Operator Station record:

 

  1. Facility – Select a Facility from the list. The facilities associated to the logged-in user through ‘Facilities for User’ function in System Manager module. The system displays the logged in Facility, by default. This is a mandatory field.

  2. Code - Enter the Operator Station Code in alphanumeric format of 30 characters maximum length. An Operator Station ID is used to identify the Operator Station. The Operator Station ID has to be unique. You cannot change it after creation. This is a mandatory field.

  3. Long Description - Enter the Long Description of the Operator Station in alphanumeric format of 30 characters maximum length.  This is a mandatory field.

  4. Short Description - Enter the Short Description of the Operator Station in alphanumeric format of 15 characters maximum length.  This is a mandatory field.

  5. Operator Station Type – Select the Operator Station Type from the list of valid values. The values are ‘Outpatient’, ‘Inpatient’ and ‘Inpatient/Outpatient’.  This is a mandatory field.

  6. Enabled - Select the check box to enable the record. Clear the check box to disable the record. The check box is selected, by default.
