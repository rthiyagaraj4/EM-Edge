Enter the following details to create a Unit of Measure record:

  1. Code - Enter the Code for Unit of Measure in alphanumeric format of four characters maximum length. A Unit of Measure code is used to identify the Unit of Measurement. The Unit of Measure Code has to be unique and cannot be changed after creation. This field is mandatory.

  2. Long Description – Enter the long description in alphanumeric format of 30 characters maximum length. This field is mandatory.

  3. Short Description – Enter the short description in alphanumeric format of 15 characters maximum length. This field is mandatory.

  4. External UOM – Enter the External Unit of Measure in alphanumeric format of 20 characters maximum length.

  5. Enabled - Select the check box to enable the Unit of Measure record. The check box is selected, by default. Clear the check box to disable the Unit of Measure record.

 

Applicability

  1. Purchase – Select the required check box to indicate that the Unit of Measure is applicable to Purchase module. The check box is applicable if Purchase module is installed.

  2. Stock - Select the required check box to indicate that the Unit of Measure is applicable to Inventory Management module.

  3. Sale - Select the required check box to indicate that the Unit of Measure is applicable to Sale related transactions.

  4. Dosage - Select the required check box to indicate that the Unit of Measure is applicable to Pharmacy Management module. Select the check box to enable the Dosage to be displayed in the Pharmacy Management Module..

  5. Strength – Select the check box to indicate that the defined unit is Strength. For example, Milligram, Microgram, etc.

  6. Quantity - Select the check box to indicate that the defined unit is Quantity. For example, Teaspoon, Tablespoon, etc.
