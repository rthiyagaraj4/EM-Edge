Enter the following details to define a Room:

 

  1. Facility – Select a required Facility from the list. This field displays the facilities to which the user is associated through the ‘Facilities for user’ function in System Manager module. The logged in facility is displayed, by default. This is a mandatory field.

  2. Room/Cubicle No - Enter the Room or Cubicle Number in alphanumeric format of six characters maximum length. A Room number is used to identify a Room. A Room number has to be unique and cannot be changed after creation. This is a mandatory field.

  3. Building - Select the Building to which the Room is located from the list. The list of values displays all the buildings created using ‘Building in Facility’ function.

  4. Floor - Select the Floor to which the Room is located from the list. The list of values displays all the floors attached to the selected building in ‘Floor in building’ function.

  5. Long Description - Enter the long description of the Room in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  6. Short Description - Enter the short description of the Room in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  7. Type - Select the Type of Room from the list. All the care locations created through ‘Care Location Type’ function with care location type indicator as ‘Room’ are displayed in the list. This is a mandatory field.

  8. Resident Stay – Select the Resident Stay from the list. The listed values are ‘Daycare’ and ’Inpatient’.

  9. Tel No - Enter the Telephone Number of the room in alphanumeric format of 18 characters maximum length.

  10. Extension - Enter the Extension of telephone number in the room in alphanumeric format of six characters maximum length.

  11. Enabled - Select the check box to enable the record. Clear the check box to disable the record. The check box is selected, by default.

 

Characteristics

  1. Room/Cubicle Status - Select the privacy level of Room from the list. The Room/Cubicle Status is selected only if the field ‘Resident Stay’ is selected. The Privacy Levels are Isolation, Private Room, Semi Private Room and Room/Cubicle.

  2. Infectious Disease - Select the check box to indicate that the Room is meant for patients with Infectious Diseases. This is enabled only if room status is selected as ‘Isolation’.

  3. No. of Beds – Specify the number of beds that can be accommodated in the room in numeric format of three digits maximum length. This field is enabled only when 'Resident Stay' field is selected.

  4. Usable by Radiation Implant Patients - Select the check box to indicate that the room is usable by Radiation Implant patients.

  5. Smoking - Select the check box to indicate that Smoking is permitted in this room.

 

Available Equipment

  1. Oxygen - Select the check box to indicate that Oxygen equipment is available in the room.

  2. Suction - Select the check box to indicate that Suction equipment is available in the room.

  3. Vital Signs - Select the check box to indicate that Vital Sign equipment is available in the room.

  4. Infusion Pump - Select the check box to indicate that Infusion Pump equipment is available in the room.

  5. IV Pump - Select the check box to indicate that IV Pump is available in the room.

  6. Electro Encephalogram - Select the check box to indicate that Electro Encephalogram equipment is available in the room.

  7. Electro Cardiogram - Select the check box to indicate that Electro Cardiogram equipment is available in the room.

  8. Ventilator - Select the check box to indicate that Ventilator equipment is available in the room.
