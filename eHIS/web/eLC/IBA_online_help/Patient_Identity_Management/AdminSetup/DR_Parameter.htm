DR Parameters

From this function you can Define the rules that influence the functioning with respect to searching and merging duplicate records.

This function is used to create and maintain the Duplicate Registration module parameter record. There can be only one parameter record. Parameter values on the search, merge and unmerge processes are stored in this function.

Defining DR Parameters

From the DR Code Setup > DR Code Setup, select the DR Parameter function.

The DR Parameter screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Apply] to save the DR Parameter values or [Reset] to clear the entered data.
