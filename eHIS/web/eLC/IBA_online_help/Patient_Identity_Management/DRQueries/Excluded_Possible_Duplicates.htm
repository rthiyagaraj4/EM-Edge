Excluded Possible Duplicates

From this function you can Query and view the details of Patient IDs that are excluded as possible duplicates.

The patients excluded from future Search Process using the process 'Exclude Possible Duplicates' are viewed in this query function.

Querying for Excluded Possible Duplicates

From DR Queries, select the Excluded Possible Duplicates function.

The Excluded Possible Duplicates screen gets displayed.

  1. Enter a valid Patient ID in alphanumeric format of maximum length as specified in the MP Parameter or click [?] to select the required Patient ID.

  2. On entering the valid Patient ID, click [Execute] to view the Excluded Possible Duplicates for the selected Patient ID or click [Reset] to clear the data entered.

  3. The Query result screen provides details of Patient ID1 and Patient ID2. The patient details displayed are Name of the Patient, Gender and Age.

  4. Click the '+' hyperlink to view the Patient details. The Patient details are displayed under the six tabs, namely, Demographic Details, Contact Addresses, Related Contacts, Documents & Others, Who Done It and Financial Details.

  5. Click [Next] to view the next set of Excluded Possible Duplicate records.

  6. Click [Previous] to view the previous set of Excluded Possible Duplicate records.
