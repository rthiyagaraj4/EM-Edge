Manage Scheduled Request

From this function you can Manage the Pending, Completed and Running jobs submitted through the Merge Duplicates, Merge Duplicate for a Patient.

The Manage Scheduled Request function is used to manage the merge job that is submitted as a job. The jobs submitted in Merge Duplicates, Merge Duplicate for a Patient function with the status Pending, Completed and Running appears in Manage Scheduled Request function. You can change the job's execution time for the process with status Pending. The function helps to remove the process from the job queue and submit it again for merging the records.

Managing Scheduled Request

From the DR Processes, select the Manage Scheduled Request function.

The Manage Scheduled Request screen gets displayed.

  1. Click [Search] to display the scheduled requests or enter the search criteria and click [Search] to select a specific schedule.

  2. On searching, the scheduled request details are displayed based on the entered criteria.

  3. Click the ‘+’ hyperlink to view record details.

  4. Click the desired Scheduled Date hyperlink to enable [Submit], [Change] and [Remove].

  5. Enter the next execution date in the format of dd/mm/yyyy or select the date using the calendar function.

  6. Click [Submit] to submit the job for the Scheduled Date and Time.

  7. Click [Change] to change the job for the Scheduled Date and Time.

  8. Click [Remove] to remove the job from the job queue.

  9. Click [Reset] to clear the selected details.
