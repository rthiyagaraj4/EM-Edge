Others

 

Click  to expand the Others option. Dietitians can view the recorded diagnosis and the food allergy details of a patient by selecting the respective hyperlink. This ready reference helps the Dietitian to track the diagnosis or allergy details recorded by a Clinician or a Nurse in Clinician Access module. A Dietitian can place an order for the patients accordingly after examining the details recorded by the Clinician or a Nurse.

 

Active Problems

Click Active Problems hyperlink to invoke Problem List window from the Clinician Access module. You can choose ‘Active Only’ or ‘All’ to view the current allergies or all allergy details recorded for a patient.

  1. Code - Select a Code from the list. Click [?] function to select a sub-code. The description of the selected sub-code is displayed.

  2. Encounter ID – Enter the Encounter ID in numeric format of 12 digits maximum length or click [?] function to select the Encounter ID. The details recorded during the specific encounter get displayed.

  3. Period – Click the Calendar function to select the date range. The details recorded during the selected date range get displayed.

  4. [Search] - Click [Search] to display the problems recorded in the Clinician Access module or [Clear] to clear the entered search criteria.

 

Allergies

Click Allergies hyperlink to invoke Allergy List window from the Clinician Access module. You can choose ‘Active Only’ or ‘All’ to view the current allergies or all allergy details recorded for a patient. You can view the details in Authorized mode or Normal mode.

Click to collapse the ‘Others’ option.
