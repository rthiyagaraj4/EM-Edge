Enter the following details to place diet order for patients:

 

  1. Nursing Unit - Select a valid Nursing Unit from the list. The values defined using the ‘Nursing Units’ function in Inpatient & Day Care Management module get listed. This is a mandatory field.

  2. Practitioner – Select the Practitioner using [?] function to view the diet orders placed by the Practitioner. Practitioners are associated to the selected Nursing Unit in ‘Practitioner for Nursing Unit’ function in Inpatient & Day Care Management module.

  3. Admn. Date – Select the admission date range using the Calendar function to select patients admitted to the facility during the specified date range.

  4. Encounter ID – Enter the Encounter ID to search for the patient.  

  5. Patient ID - Select the Patient ID using [?] function. You can place diet order for the respective patient or view the recorded diet order.

  6. Gender - Select the gender from the list. The system-defined values are male, female and unknown.

  7. Location – The short description of the location of the patient is displayed.

  8. Status – Select the status from the drop-down list. The listed values are ‘All’, ‘Order Placed’, ‘Order Pending’ and ‘Order Modified’. The value ‘All’ is selected by default.

  9. Search For – Select the value ‘Diet Type Modified’ or Refer to Dietician’ from the drop down list. If you select the Diet Type Modified, the system searches only the diet orders for which ‘Diet Type’ (including ‘Assign Diet Type’) is modified in Place Diet Order. If you select the Refer to Dietician, the system searches only the diet orders that are referred to a dietician, that is, ‘Refer to Dietician’ checkbox is selectedin Place Diet Order.

A new indicator   is displayed in the status column for diet orders where the Diet Type is modified. This is applicable for the Order Placed For Current Date, Order Pending For Patient , Order Modified statuses.
