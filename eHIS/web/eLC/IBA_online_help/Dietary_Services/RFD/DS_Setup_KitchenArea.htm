Enter the following details to associate Kitchen Area to a Kitchen:

 

  1. Kitchen Code – Select a Kitchen from the list. The values are defined using the ‘Kitchen’ tab. On selecting the Kitchen Code, Kitchen Area Code along with the long description, short description and the status recorded for the selected Kitchen Code get displayed in the tabular format. This is a mandatory field.

  2. Kitchen Area Code – Enter the Kitchen Area Code in alphanumeric format of four characters maximum length. This is a mandatory field.

  3. Long Description - Enter the long description of the Kitchen Area in alphanumeric format of 30 characters maximum length.

  4. Short Description - Enter the short description of the Kitchen Area in alphanumeric format of 15 characters maximum length.

  5. [Select] - Click [Select] to display the entered Kitchen Area Code, long description, short description and the status in tabular Format on the right side of the screen. The Kitchen Area Codes are displayed as a hyperlink. On clicking Kitchen Area Code, the Kitchen Area Code, respective long description, short description and the status gets displayed in the respective fields. You can update the long description, short description and the status of the selected Kitchen Area Code. Click [Apply] to save the edited record. Click [Cancel] to remove the selected Kitchen Area from the table, if there are no transactions performed in the Kitchen Area. You can add the Kitchen Areas to the selected Kitchen.
