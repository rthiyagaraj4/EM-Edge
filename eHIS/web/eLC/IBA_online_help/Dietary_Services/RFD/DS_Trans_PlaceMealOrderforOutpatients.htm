Enter the following details to place diet orders:

 

Meal Order For

 

  1. Outpatient – Select the radio button to place meal order for outpatients.

  2. Emergency Patient - Select the radio button to place meal order for Emergency Patients.

  3. Specialty - Select the Specialty using [?] function. The specialities selected in Specialties For Kitchens screen are only listed. This is a mandatory field.

  4. Encounter ID – Enter the Encounter ID in numeric format of 12 digits maximum length.

  5. Location Type - Select the location type from the drop-down list. This field is mandatory. The values listed are ‘*All’, ‘Clinic’, ‘Day Care’ and ‘Procedure Unit’. On selecting the required Location Type, the Locations listed are based on the following conditions:

  6. All the Clinics/Daycare/Procedure units defined under the selected location type that allow appointments (based on the ‘Allow Appointment’ value set in ‘Clinic’, Daycare Unit’ and ‘Procedure Unit’ setup function of EM Outpatient Management module).

  7. All the defined Clinics/Daycare/Procedure units must be associated to the operator station assigned to the logged in user.

  1. Location - Select the Location using [?] function.

  2. Resource Type - Select the Resource Type from the list. The listed values correspond to the selected Resource Class. Specify the Resource Type range to restrict the generation of list.

  3. Resource – Select the Resource using [?] function.

  4. Patient ID – Enter the Patient ID or select the Patient ID using [?] function.

  5. Appointment Date [From-To]– Select the Appointment Date using the calendar function. The current date is displayed by default.

  6. Treatment Area – Select the treatment area from the drop-down list. The treatment area associated with location will be listed out.

You can place meal order one day before the visit day based on the appointment schedule. The meal orders can be auto-generated based on the appointment schedule. You can place meal order for walk-in patient on the date of visit.

  1. Status - Select the status from the drop-down list. The listed values are ‘All’, ‘Order Placed’, ‘Order Pending’ and ‘Order Modified’. The value ‘All’ is selected by default.

  2. Appt No – Enter the appointment number if the patient has visited with appointment.
