Enter the following details to place diet orders:

 

  1. Select Date – Click the Calendar function to select a date from when the meals are to be served. The system date is displayed, by default.

  2. Diet Cycle – The Menu Cycle defined for the selected date, Diet Type and Meal Class is displayed and defaulted. You can edit the Diet Cycle in ‘Menu Cycle’ function for the selected date.

  3. Calorie Grand Total - The sum of the calorie content for the meal type selected is displayed for the selected date. This helps the Dietitian to prescribe a Meal Order for a patient and distribute the calorie content across each Meal type based on the selected food items on a particular day.

 

Select to expand the Meals option and click to add or edit the food items for different Meal Types. Calorie contents are displayed against each food item along with its Energy Unit of Measurement as specified in DS Parameter. The quantity of the food item is displayed along with its Calorie Content and Serving Unit of Measurement as defined in the ‘Food Item’ function. The calorie content is calculated based on the quantity ordered multiplied by the preparation Unit of Measurement, Preparation Factor and Preparation Base Quantity defined for the particular food item in the ‘Food Item’ masters. The total calorie content of each Meal Type is calculated and displayed below each Meal Type. The total Meal Calorie Grand total is displayed on the top.
