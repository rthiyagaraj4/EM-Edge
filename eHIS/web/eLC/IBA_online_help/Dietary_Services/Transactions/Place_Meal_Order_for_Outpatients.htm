Place Meal Order for Outpatients/Emergency Patients

From this function you can Place a meal order for outpatients on a particular date in own facility and across the facilities.

The Place Meal Order for Outpatients and Emergency Patients is used to place the meal order for outpatients within the  facility and across the facilities. This functionality is applicable only for walk-in patients and outpatients having encounters open. It is not applicable for waiting list patients and pending orders.

 

The status of the diet orders is displayed using the following colors:

                             

                  -  Order Placed for current Date

                   -  Order Modified

                  -  Order Pending for patient

Placing a meal order for an outpatient

From the Transactions, select the Place Meal Order for Outpatients/Emergency Patients function.

The Place Meal Order for Outpatients screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to search for outpatients. The Patient ID, Patient Name, Appointment Date/Time, Appt No, Encounter ID and Status are displayed.

  3. Select the check box against the Patient ID to place the meal order. You can select more than one check boxes. By default, the check box is not selected. The check box against the patient ID is displayed only for patients with current and future appointments.

  4. Click [Place Order] to display Place Meal Order for Outpatients dialog. This button is enabled if you select one of the check boxes against the Patient ID. By default, this button is disabled.

    Enter the following details:

    1. Serving Date – Select the serving date using the calendar function. This is a mandatory field.

    2. Kitchen – Select the Kitchen from the drop-down list. This is a mandatory field.

    3. Diet Type – Select the Diet Type from the drop-down list.

     

    Selected Patients

    1. Patient ID – The patient ID entered in the Place Meal Order for Outpatients screen is displayed.

    2. Patient Name - The patient name in the Place Meal Order for Outpatients screen is displayed.

    3. Appointment Date/Time - The Appointment Date/Time entered in the Place Meal Order for Outpatients screen is displayed.

    4. Appt No – The appointment number entered in the Place Meal Order for Outpatients screen is displayed.

    5. Encounter ID – The Encounter ID entered in the Place Meal Order for Outpatients screen is displayed.

    6. Diet Type – The selected Diet Type is displayed.

    7. Breakfast – Select this check box to add breakfast in the meal order. This check box is not selected by default. You can select the check box next the Breakfast label to select all the check boxes

    8. Lunch – Select this check box to add lunch in the meal order. This check box is not selected by default. You can select the check box next the Lunch label to select all the check boxes

    9. Afternoon Tea – Select this check box to add  Afternoon Tea in the meal order. This check box is not selected by default. You can select the check box next the Afternoon Tea label to select all the check boxes

    10. Dinner – Select this check box to add  Dinner in the meal order. This check box is not selected by default. You can select the check box next the Dinner label to select all the check boxes

    11. Food Dislike –Click Edit hyperlink next to the Food Dislike field. The Food Dislikes dialog is displayed.

      Enter the following details:

      1. Item - Click [?] to select the food items defined in the Food Item function.

      2. Click [Select] to add the selected food item under Item section. You can add more than one item under the Item section or click [Cancel] to cancel the food item selection.

      3. Click [OK] to display the food items, which are listed under the Item section, in the Food Dislikes field in the Place Meal Order for Outpatients dialog or click [Close] to close the dialog.

    12. Special Food Item – Click Edit hyperlink next to the Special Food Item field. The Special Food Items for Outpatient dialog is displayed.

      Enter the following details:

       

      1. Item Type – Select the Item Type from the drop-down list. This is a mandatory field.

      2. Item – Select the Item using [?] function. This is a mandatory field.

      3. Qty – Enter the required quantity of the item. This is a mandatory field.

      4. Meal Types – Click this hyperlink to view the meal types in the MealTypes dialog. Select the check box against the required meal type. The selected meal types gets displayed in the Special Food Items for Outpatent dialog.

      5. Click [Select] to display the selected item, Item Type, Qty and Meal Type or click [Cancel] to reset the data.

      6. Click [OK] to display the special Food Items in the Place Meal Order for Outpatients.

    13. Preparatory Instruction - Enter the preparatory instruction while placing a meal order (new/amend). This is also  applicable for Emergency patient meal orders.

    14. Remarks – Enter the remarks about the diet order if required.

  5. Click [Alternate Facility] to view the Alternate Facility Selection dialog with the Facility drop-down list. The logged in facility is displayed by default. You can select the required facility from the drop-down list. This button is displayed only when ‘Enable Multi Facility Meal Order for Outpatients’ check box is selected.

  6. Click [OK] to display an authentication screen with User Name and Password fields. Once you enter the username and password correctly, you will be redirected to a new Place Meal Order for Outpatient screen for the selected facility. You can select the patients and place the meal order.

If you click [OK] button without selecting facility in the Alternate facility selection dialog, an error message ‘Please Select a Facility’ is displayed.

  1. Click [OK] to display the patient details of the selected facility or click [Cancel] to close the Alternate Facility Selection dialog.

If you click [Apply] button without entering any value, an error message ‘Atleast One Record Should Be Updated’ is displayed. If you place meal order for the same patient again by mistake, an error message ‘Meal Order has already been placed for the following patient(s). Saving the changes will override the previous meal order. Are you sure you wish to continue?’ is displayed with Yes or No buttons. Click Yes to override the record or click No to close the error message.
