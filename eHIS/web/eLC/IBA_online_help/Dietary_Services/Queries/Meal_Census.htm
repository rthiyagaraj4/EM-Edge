Meal Census

From this function you can Query and view the meal details prepared and served by the kitchen in a facility.  

The Meal Census function is used to view the meal details such as diet type, meal class, menu and items that are served to the Ward from a Kitchen on a specified date. This function also helps to view the diet type supplied to each patient for a selected date. The food items supplied for each Meal Type can be viewed on clicking   adjacent to each record.

Querying Meal Census

From the Queries, select the Meal Census function.

The Meal Census screen gets displayed.

  1. Select the Kitchen from the list. The Meal Census is displayed for the selected Kitchen. This is a mandatory field.

  2. Select the Ward from the Nursing Unit list. The Meal Census is displayed for the selected Kitchen and Ward Code. This is a mandatory field.

  3. Click the Calendar function to select the Serving Date. The Meal Census is displayed for the selected Kitchen, Ward Code, Serving Date and Ward short description. The system date is displayed as Serving Date, by default.

  4. Click [Select] to search for the Meal Census. The patient ID, Name, Bed Number, Meal Type, Diet Type, Meal Class and the Menu gets listed. The values are sorted based on the order of the meal type defined in the Meal type screen.

  5. Click to view the items consumed by the patient for the selected Meal Type.
