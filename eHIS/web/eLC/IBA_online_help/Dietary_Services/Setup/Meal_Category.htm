Meal Category

From this function you can Associate a system-defined Meal Category to a Diet Type and a Meal Class. You can Query a desired Meal Category associated to a Diet Type and a Meal Class.

Meal Category function is used to maintain categories of meals based on Diet Type and Meal Class like Attendant meals, Inpatient meals, On Call Staff meals, Outpatient meals, Staff meals and Student meals that are applicable in a hospital. Care has to be taken to see that the attendants in the ward are also served meals if ordered. Nursing Units are associated to meal category function in order to specify that the meal type for the attendant are the same as the patient order or it can be a special meal based on the request.

Querying a Meal Category

From the Setup, select the Meal Category function.

The Meal Category screen gets displayed.

  1. Click [Query] to search for an existing system-defined record. <Related Field Description>

  2. Click [Execute] to query the details. A list gets displayed with details such as Meal Category, Diet Type and Meal Class. Meal Category is displayed as a hyperlink.

  3. Click Meal Category hyperlink and a screen gets displayed with two tabs. Meal Category tab is enabled, by default.
