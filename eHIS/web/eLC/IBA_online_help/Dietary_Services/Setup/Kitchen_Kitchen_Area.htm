Kitchen/Kitchen Area

From this function you can Create a main Kitchen in a facility and associate additional sub-units or kitchen areas. You can Query for the specified Kitchens.

The Kitchen/Kitchen Area function is used to define and maintain various kitchens in the facility. This function stores the information of kitchens and the associated kitchen areas. In case of multi-kitchen hospitals, each kitchen caters to a specified number of wards. The system has a provision to allocate wards for each kitchen. Each ward has to be associated to a kitchen. A kitchen can have several other sub-units, where the food items are stored, meals prepared and served to the patients.

Creating Kitchen/Kitchen Area

From the Setup, select the Kitchen function.

The Kitchen/Kitchen Area screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields in the following tabs

Kitchen

Kitchen Area

  1. Click [Apply] to save a created Kitchen/Kitchen Area or [Reset] to clear the entered details.

  2. Click [Query] to search for the existing records.

 

You can create multiple kitchens in a facility, if the check box ‘Multi Kitchen’ is selected in the ‘DS Parameter for Facility’ function.
