Meal Type

From this function you can Create a Meal Type catered by the Kitchen in a hospital and Query for the existing records.

The Meal Type function helps in recording various meal types catered by the kitchen at the facility. It defines the standard Meal Types that are routinely prepared in the Kitchen for consumption. Meal Type function is used to generate a Meal Plan for a patient. A Meal Type identifies the food items that the patient can consume when admitted in the hospital.

Creating a Meal Type

From the Setup, select the Meal Type function.

The Meal Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a created Meal Type or [Reset] to clear the entered details.

  4. Click [Query] to search for the existing records.
