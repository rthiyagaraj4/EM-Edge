Adhoc Discount

From this function you can issue a discount on the individual package that is subscribed and associated to a patient. The discount amount can be recorded separately for a Payer and Patient.

The Adhoc Discount function is used to issue a discount to the patient though the services associated to a package are not utilized.  The discount can be issued in terms of percentage or pre-defined amount. Discount can be issued only on the Open Packages. Adhoc discount can be issued on the package that is non-order catalog related which will populated a window to enter the quantum of discount. Authorized users' are only permitted to issue discount on a package.

Issuing Adhoc Discount

From the Transactions > Package, select Adhoc Discount Function.

The Adhoc Discount screen gets displayed.

  1. Click [?] function to select the patient ID. This is a mandatory field.

  2. Press [Tab] to issue discount on the package. The name of the patient, gender, age and ID is displayed. The patient's name is displayed as a hyperlink. Click the hyperlink to view demographic details.

  3. The package code and description, effective date, billing class, customer group and customer associated to the patient are displayed. The package amount for each package is also displayed.

  4. Click '+' displayed as a hyperlink to view the package details.

  5. Click the sequence number that is displayed as a hyperlink to issue discount on the respective package. The name and ID of the patient, Customer Group and Customer details are displayed.

  6. Select a value for '%/Amount'. Select 'Amount' to record a specific amount as the discount amount or 'Percent' to issue a percentage of the base price as the discount amount.

  7. Click [Apply] to save the changes or [Reset] to clear the details.
