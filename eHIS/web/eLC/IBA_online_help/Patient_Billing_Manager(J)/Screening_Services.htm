Screening Services

The screening services applicable for the selected patient are listed in this screen.

The following details are displayed:

1. Order Catalog Code – A unique code identifying the order catalog is displayed.

2. Order Catalog Description – The description of the order catalog is displayed

3. Applicable Quantity – The quantity of the order applicable for the patient is displayed.

4. Applicable Amount – The amount (charges) applicable for the patient is displayed.

5. Availed Quantity – The quantity of the order already availed by the patient is displayed.

6. Availed Amount – The amount (charges) already availed by the patient is displayed.

7. Balance Quantity – The balance quantity available for the patient is displayed.

8. Balance Amount – The balance amount available for the patient is displayed .

9. Select – Select this check box to select the particular row or order to be displayed in the Pharmacy Orders screen.

 

Note: If patient has already availed an order catalog and there is no balance, then the Select check box will be disabled for that row.

Note: If a practitioner does not have the access to place an order then, the order will be displayed in disabled mode in and the reason will also be displayed.
