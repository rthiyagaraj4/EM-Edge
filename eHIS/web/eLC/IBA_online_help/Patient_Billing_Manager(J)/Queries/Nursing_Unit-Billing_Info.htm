Nursing Unit - Billing Info

From this function you can Query to view the Inpatient billing details in a Nursing unit.

Nursing Unit - Billing Info function is used to query the Current Inpatients, Expected Discharge Patients and Discharged Patients billing details. You also have an option to view the Financial Summary by Billing Group and the Financial Summary by Payer.

Querying Billing Details of the Inpatients in the Nursing Unit

From the Queries, select Nursing Unit - Billing Info Function.

The Nursing Unit - Billing Info screen gets displayed.

  1. Click [?] to select a Nursing Unit. Nursing Units associated to the user using the ‘Nursing Unit By User’ tab of the ‘User for Authorization/Refund/Cash Counters’ function gets displayed. The 'Nursing Unit' set as 'default' under the ‘Nursing Unit By User’ tab gets displayed. This is a mandatory field.

  2. The number of available beds, beds occupied, expected discharges and discharges in the selected Nursing Unit gets displayed.

  3. The results are classified into three tabs Current Inpatients, Excepted Discharge for the Day and Discharged Patients for the Day.

  4. The Encounter ID is hyperlinked and displays financial and settlement details of the patient. The Patient ID is hyperlinked and displays Bill/Payment, Diagnosis and Procedure details recorded for the patient.

  5. The 'Financial Details by Group' and 'Financial Details by Payer' are specific to the Current Inpatients, Excepted Discharges for the Day and the Discharged Patients for the Day.

  6. Enter the number of days in numeric format with a maximum of four digits to view the forecast. The default forecast days recorded through ‘User for Authorization/Refund/Cash Counters’ function for the selected Nursing Unit gets displayed. Click [Go] to view the bookings, excepted discharges and expected inpatients in the Nursing Unit.
