Enter the following details:

 

  1. Facility –Select a facility from the drop-down list. This is a mandatory field.

  2. Billing Group - The description of the Billing Group associated to the respective encounter is displayed. This is a mandatory field.

  3. Payer Group – Click [?]  to search and select a Payer Group for the policy. This is a mandatory field.

  4. Payer – Click [?] to search and select a Payer code for the policy. This is a mandatory field.

  5. Patient Class - Click [?] to search and select a Patient Class. This is a mandatory field.

  6. Policy – Click [?] to search and select a Policy. This is a mandatory field.

  7. Priority - Enter the priority for the selected policy.  

  8. Enabled – Select this check box to enable the Age Group record. This check box is selected by default.
