The following fields are available in the Package Price Bulk Update / Price Revision screen:

 

  1. Std/Customer Price – Select the value ‘Standard Price’ or ‘Customer Price’ from the drop-down list. By default, the value ‘Standard Price’ is selected. This is a mandatory field.

  2. Bulk/ Individual Package – Select the required  value  from the drop down. Based on value selected in the Std/Customer Price, the package will be defined. By default, the value Individual Package is selected. This is a mandatory field.

  3. Billing Class - Select the Billing Class using lookup function. Based on the selected Billing class, the package price is updated.

  4. Package Code – Select the Package using the lookup function. The corresponding package code of the selected package is displayed. This is a mandatory field.

  5. Customer Group – Select the Customer Group using lookup. The customer groups defined in the billing masters are listed in the lookup screen. This field is displayed only when you select the value ‘Customer Price’ in the Std/Customer Price drop down list.

  6. Customer - Select the Customer using lookup. The customers defined in the billing masters are listed in the lookup screen. This field is displayed only when you select the value ‘Customer Price’ in the Std/Customer Price drop down list.

  7. Billing Group – Select the Billing Group using the lookup function. Based on the selected Billing Group, the price discount is updated.
