Enter the following details to define charge based Package Services:

 

  1. Type – Select the Type from the drop-down list. The system-defined values are ‘Billing Service’ and ‘Item’. Select ‘Item’ to define the item along with the billing service. Select ‘Billing Service’ to define and associate the billing service with the package. The ‘Billing Service’ is selected by default.

  2. Billing Service - Click [?] function to select and associate a Billing Service to the package. The description of the selected Billing Service is displayed.

  3. %/Amount Indicator - Select a value from the list. The system-defined values are 'Percent', 'Amount' and 'Unlimited'. Select 'Percent' to define the service amount as the percentage of the base price for the selected quantity. Select 'Amount Indicator' to specify valid amount for the service and the quantity specified. If the usage of service exceeds the quantity or the amount specified, standard price of the respective Billing Service is charged to the patient. Select 'Unlimited' if there is no restriction on the amount that can be associated to the respective Billing Service for the quantity utilized.

  4. %/Amount - Enter the value in numeric format based on the value selected for '%/Amount Indicator'. If the usage of service exceeds the amount specified, the standard price defined for the Billing Service is considered. You cannot specify the value, if the indicator is selected as 'Unlimited'.

  5. Round To - Enter the value in numeric format. This indicates the value to which the price can be rounded off.

  6. Round Indicator - Select a value from the round indicator. The system-defined values are 'Up' and 'Down'. Select 'Up' to round off the value to the highest nearing value specified for 'Round To'. Select 'Down' to round off the value to the lowest nearing value specified for 'Round To'. For Example, if the package value is 3,515 and the 'Round To' is set as 10 with the 'Round Indicator' as 'Up', the package price is rounded off to 3520.

  7. Refund Rate % - Select the check box to specify refund rate for the Billing Service. Enter the value in the adjacent text box that can be issued as a refund, if the Billing Service is not utilized when the package is availed by the patient. The check box is enabled only for 'Billing Service' and disabled for 'Service Group' or 'Service Classification'.

  8. Auto Refund - Select the check box if the Billing Service is auto refundable. The amount is automatically refunded, if the Billing Service is not utilized by the patient when the package is availed. The check box is enabled only for 'Billing Service' and disabled for 'Service Group' or 'Service Classification'.

  9. Factor Applicable for Service Limit - Select the check box to define the factor available for service limit. This field is enabled for selection only if the 'indicator' value is selected as 'factor through the 'Package Base Price' tab'.

  10. Service Limit (Dtls) - Select the check box to associate Items to the rate based services. A legend 'Dtls' is displayed as a hyperlink when you select the check box. A new table appears enabling you to associate or dissociate items from package. Click 'Dtls' hyperlink to view the details that are recorded.

  11. Select - Select the check box to associate the Billing Service, Service Group or Service Classification to the package.

  12. [Add] - Click [Add] to associate charge based services to the Package.

  13. [Delete] - Click [Delete] to delete a service from the package.

 

The following fields are enabled for selection when the 'Service Limit' check box is selected.

  1. Item - Click [?] function to associate an item to the package. The description of the item is displayed. The items defined as 'Package Items' in Inventory Management module are listed.

  2. Exclude - Select the check box to exclude the item from package. All other items associated to the Billing Service are considered to be a part of package except the item that is excluded.

  3. Quantity Limit - Enter the restriction on the quantity of the selected item that can be associated to the package. The item is charged based on the standard price defined if the limit exceeds the value that is specified.

  4. Amount Limit - Enter the amount limit in numeric format. It specifies the amount limit that can be utilized from the package amount for the respective item and the specified quantity. If the amount of the utilized quantity of the item exceeds the limit specified corresponding item is charged based on the standard price defined.

  5. Select - Select the check box to exclude or include the item as a part of the package.

  6. [Add] - Click [Add] to associate or dissociate item from the package.

  7. [Delete] - Click [Delete] to delete the associated item from the package.

You cannot define Charge based services for Order Catalog packages.
