  4. Payer - Select Payer to indicate that the Payer is settling the bill amount. The payer clears the coverage percentage defined and the patient clears the remaining amount. This is the default value.

  3. Policy Settlement applicability – Select this check box to enable policy applicability for the patient.

Outpatient/Emergency /Inpatient/DayCare, External

  1. Credit Auth Required – Select the check box, if the credit authorization is mandatory for the selected Policy Type and the associated Payer. The system prompts you to enter the credit authorization details in the transaction functions while modifying the patient billing details in the external modules and in the Patient Billing Manager module. The prompt is displayed on selecting the relevant Policy Type, Payer and Payer Group.

  2. Credit Auth Entry Mandatory - Select the check box if credit authorization details are mandatoryduring visit registration and emergency registration.

  3. Default Credit Auth Ref as Policy No. – Select the check box to default a part of the Policy Number as the Credit Authorization Reference number. The Policy Number can be a maximum of 30 alphanumeric characters. The first 20 characters out of the 30 alphanumeric characters are considered the Credit Authorization Reference number

  4. Covered – Select this  if the selected Service Policy Coverage for the selected Policy Type is applicable to Outpatients or Emergency patients.

  5. Policy Applicability Across Encounters – Select this check box to set the policy applicability across encounters.

  6. Coverage Limit Type – Select an Outpatient or Emergency Coverage Limit Type from the list. The values are system defined – ‘Restricted’ and ‘Un-Restricted’. The default value is ‘Restricted’.

  7. Restricted – Select Restricted to restrict the Outpatient or Emergency Coverage Limit for the Policy Type.

  8. Un-Restricted - Select Un-Restricted and the Outpatient or Emergency Coverage Limit is not restricted for the Outpatients or Emergency patients associated to the selected Policy Type, Payer and the Payer Group.

  2. Restricted – Select Restricted to restrict the Outpatient or Emergency Approved Days for the Policy Type.

  3. Un-Restricted - Select Un-Restricted and the Approved Days is not restricted for the Outpatients or Emergency patients associated to the selected Policy Type, Payer and the Payer Group.

On selecting Un-restricted, the Outpatients or Emergency Patients visiting the facility can avail any amount from the Insurance Company as the coverage amount.
