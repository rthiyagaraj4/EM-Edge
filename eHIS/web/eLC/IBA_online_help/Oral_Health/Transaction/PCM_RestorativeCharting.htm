Restorative Charting

From this function you can Create a Restorative Chart for recording conditions and treatments for a patient seeking orthodontic treatment.

Using the Restorative Chart, you can record the Dental Conditions/External Treatments, Treatments that are planned, Treatments performed and Clinical Notes/Assessments as related to the various Oral Health Specialties. You can use the chart to record the pre-treatment dental status of a patient. In other words, this function can be used to record the details by the practitioner only for those patients, whose visits are considered as first visit to the facility. A procedure is being performed on the tooth, the dental status of the patient gets updated appropriately and updated dental status is defaulted when this function is invoked subsequently. Once the procedure is performed on a tooth, the dental chart of the patient is displayed only for viewing and no details can either be recorded or modified. The chart gets displayed, based on the parameter setup and the age of the patient. The pre-condition and pre-treatment dental status details of the patient can be recorded in two different ways as follows:

• If the patient is in between the age range of six months – three years a deciduous chart is displayed.

• If the patient is above three years of age, a mixed dentition chart is displayed based on the age range specified in the parameter setup.

• If the age range is greater than the one specified for the mixed dentition, a permanent chart is displayed.

• By selecting the appropriate tooth from the dental chart.

• By selecting the appropriate tooth from the list of valid values.

This function is also used to record the orthodontic conditions. Several recordings are done by clicking on the related options available on the screen. Several customized icons are used to indicate various conditions and treatments. You can customize the icons that has to appear on the Clinician Desktop. The icons, which are selected, get displayed on the desktop along with the selected patient details.

• Record Condition against a tooth

• Record Treatment for the Condition recorded

• Assign Condition/Treatment for an Arch/Quadrant/Tooth Range

• Add New Baseline Charts

• Record Diagnosis

• Record Clinical Notes

• View the Condition/Treatment Recorded

The screen is divided into two parts. The upper part of the screen displays the dental chart and lower part of the screen displays the textual format through which the details can be recorded for the patient. You can record treatment details on clicking the tooth image on screen. An image is displayed in the treatment area on recording the treatment. On clicking the image, you can view the tooth image, treatment category, treatment, tooth number, treatment date, name of the user who has recorded the details along with the remarks. Creating a Restorative Chart

From the CA Transactions > Clinician Access > Desktop Menu > Charting, select Restorative Charting Function.

The Restorative Charting screen gets displayed.

  1. Enter details in the mandatory (*) fields under the following tabs:

Condition

Treatment

  1. Click [Record] to record the condition or the treatment or [Reset] to clear the entered data.

  2. Click to view the image definitions used to indicate treatments and conditions. System-defined icons are used to identify conditions and treatments. The icons for conditions and treatments vary and are displayed on the screen. The respective icons are displayed on selecting a condition or treatment for a tooth.

    There are certain other icons which get displayed on recording a condition/treatment:

    - Partially Errupted Tooth/Deciduous Tooth

    - Mobility

    - Unerupted Tooth

     - Pulp Involvement

    - Peri apical Involvement

     - Root Involvement

     - Caries

    - Cusp Tip

    The icons used to represent various conditions are listed below:

    - Trauma

    - Attrition, Abrasion, Erosion

    - Super Numerary

    - Missing Tooth

    - Partial Eruption

    - Uneruption

    - Impaction Normal

    - Mobility

    - Mal Occlusion

    - Mal Position

    The icons used to represent various treatments are listed below:

    - Extraction Planned

    - Extraction Performed

    - Implant

    - Surgery

    - Denture

    - Bridge

    - Orthodontic Appliance

    - Impaction Treatment

  3. Select the Numbering System from the list to override the Numbering System set in the ‘Setup Parameters for Facility’ function in the 'Admin Setup' menu and use the selected Numbering System. The images in the chart differ based on the Numbering System selected.

  4. Select the date using the Calendar function in ‘Chart as on’ field to view the chart created for the patient on the respective date.

  5. Click to print a dental chart. You can select a date and the numbering system based on which the dental chart has to be printed. A Dental Chart Print screen is displayed with a preview of the tooth images along with the icons, conditions and treatments recorded. Click [Print] to print the selected dental chart or [Cancel] to cancel the print process. The comments that are recorded in an active chart using the Dentist Comments hyperlink is displayed. You have an option to record the comments before printing the chart.

    The chart also displays the external treatments and treatments that are planned and performed. The name of the patient, age and gender are displayed. Details like the name of the user who has created the chart, validity period and the numbering system used to create the chart are displayed. The name of the user is displayed as the 'Signed Name' and the 'Signed Date' is defaulted to the system date. You can enter the Dentist Comments in an active chart before printing the dental chart. You have an option to select language and print chart in the required language.

The print option is available only in the 'Treatment' tab. You can also print the closed Baseline Charts.

  1. Click [Refresh] to create the Chart with the selected Numbering System on the date selected.

  2. Click [Record] to save the details or [Reset] to clear the entered data.

 

When the chart crosses the validity period set in ‘Setup Parameters for Facility’ function, a warning message ‘Restorative Chart validity days are crossed. Do you want to create a new chart?’ gets displayed. You can close the existing chart and create a new chart without any conditions/treatments or copy the conditions/treatments from the old chart to the new chart.
