Dental Chart

Condition

The procedure for selecting appropriate pre-condition information for root, surface and entire tooth is explained:

 

Recording Condition on a Root:

  1. Root – Place the mouse pointer over the appropriate root of the tooth and click the left mouse button. All the possible conditions, those can be recorded on the root level are displayed in a list. The Practitioner needs to select the appropriate Specialty Condition from the list. The listed values are ‘Caries’ and ‘Pulp Involvement’.

 

A screen is displayed listing the possible conditions that can be recorded on root level by clicking the left mouse button over the root. The possible treatments on root level are displayed based on the master setup data. Hence, you have to set up appropriate Specialty Conditions while setting up the master data through OH Code setup function.

 

The Dental Condition tab gets enabled and you can select the area and the dental condition against the area. On selecting the condition from the list, the Condition Type, Site and Area under the Dental Condition tab are populated with the condition selected, the site and the tooth for which the condition is selected. These fields are disabled and you cannot edit the values.

Pulp Involvement is represented in red color, whereas Caries is represented in black color. The image shows the condition recorded on the surface of the tooth. The Dental Condition tab gets enabled, where you can select the area and the dental condition against the area. On selecting the condition from the list, the Condition Type, Site and Area under the Dental Condition tab are populated with the condition selected, the site and the tooth for which the condition is selected. These fields are disabled and you cannot edit the values.

 

Recording Condition on a Crown:

  1. Surface - Place the mouse pointer over the appropriate surface of the tooth and click the left mouse button. All the possible conditions that can be recorded on the crown are displayed as a list. The Practitioner has to select the appropriate Specialty Condition from the list. The listed values are 'Peri apical Involvement' and 'Root Involvement'.

 

When you click the left mouse button over the surface, a screen gets displayed listing the possible conditions that can be performed on surface level. The possible treatments on surface level are displayed based on the master setup data. Hence, you have to set up appropriate treatment categories while setting up the master data through OH Code setup function.

 

Recording Condition on a Tooth:

The tooth number is displayed as a hyperlink. Select the check box adjacent to the tooth number displayed as a hyperlink to view the list and to record a condition on a selected tooth. All the possible conditions, those can be performed on the tooth level are displayed in a list. The Practitioner has to select the appropriate condition from the list. A screen gets displayed listing the possible conditions, that can be recorded on tooth level to identify the severity of dental condition. The possible conditions on tooth level are displayed based on the master setup data. Hence, you have to set up appropriate Specialty Conditions while setting up the master data through OH Code setup function. You can select the check boxes adjacent to the tooth numbers to record the condition on multiple teeth. The listed values are 'Attrition, Abrasion and Erosion', 'Congenitally Missing Tooth', 'Exfoliated Tooth', 'Mal Position', 'Mobility', 'Retained Deciduous Tooth', 'Supernumerary Tooth' and 'Traumatic Tooth'.

 

Completion of data entry:

On completion of data entry, click [Record] to save the record or [Reset] to clear the entered data and to record a new Dental Condition. Multiple entries can be stored against the current encounter, as patient might have undergone multiple conditions for multiple teeth elsewhere prior to the current visit. On clicking [Record], a system defined image gets displayed below the tooth number against which a condition is recorded. When the condition is not recorded, clicking on the surface below the tooth number, displays a message stating that the condition has not been recorded. When you click on the image, a screen gets displayed listing the conditions and the external treatments and the name of the user who has recorded the details.

 

Click History area (row below the tooth number in the dental chart) to view conditions or external treatments that are recorded for the respective tooth. The image for the selected condition, condition type and name, tooth number, date, remarks and the name of the user who has recorded the condition is displayed.
