Enter the following details to create an Evaluation Chart:

 

  1. Numbering System – The Numbering System selected while defining the parameters in 'Setup Parameters For Facility' function in the Admin Setup gets displayed. You can select a different numbering system. On selecting a different numbering system and clicking [Refresh], a message ‘Do you want to keep this Numbering System as your default Numbering System?’ gets displayed. Click [OK] to continue or [Cancel] to continue using the existing numbering system. This is a mandatory field.

  2. Baseline Chart – Select a Chart from the list. The Baseline Charts defined under the Baseline Chart tab for which the status is ‘Complete’ get listed. The name of the user along with the date when the chart was completed gets displayed. This is a mandatory field.

  3. Arch – Select an Arch from the list. You can record the status for the Upper and the Lower Arch. This is a mandatory field.

  4. Status – Select a Status from the list. The system defined values are Active , Completed  and Marked Error . A chart has to go through all the statuses, if has to be completed.

    You can mark the status of an active Chart as Mark Error when the chart is created by error. Select the Status as ‘Mark Error’ from the list. A message ‘Do you want to mark the Chart as error?’ gets displayed. Click [OK] to mark the chart as an error or [Cancel] to maintain the status of the chart. A message ‘Chart is Marked as Error’ gets displayed on selecting [OK]. Click [OK] to save the chart as completed. The charts, which are completed cannot be marked as error. You can mark only the active charts as error.

    You can mark the status of an active Chart as Completed. Select the Status as ‘Completed’ from the list. A message ‘Do you want to mark the Chart as completed?’ gets displayed. Click [OK] to mark the chart as completed or [Cancel] to maintain the status of the chart. A message ‘Chart has completed’ gets displayed on selecting [OK]. Click [OK] to save the chart as completed.

    You can record the details only in an active chart. The recorded details get displayed under the respective tooth number. Only the active charts can be marked as an error or completed.

  5. [Refresh] – Click [Refresh] to refresh the screen. The Baseline Chart gets displayed based on the parameters selected.

  6. [Reset] – Click [Reset] to clear the data entered.

  7. Key To Charting - The components associated to the selected Chart is listed along with the minimum, maximum and the Red if>= values. The Red if>= value is marked in red and displayed within brackets.

  8. Tooth No – Select a Tooth Number from the list to record the components. The tooth range associated to the selected Chart using the function ‘Tooth Applicability for Chart’ in the OH Master function only are listed. This is a mandatory field.

  9. Components – Enter the value in numeric format to indicate the severity. You can enter the numeric value between the minimum and the maximum value recorded for the component. If the value is greater than or equal to the Red if>= value, a message gets displayed indicating that the range of values valid for the selected Component.

  10. [Record] – Click [Record] to record the tooth and the component details. A message gets displayed indicating that the record has been saved.

  11. [Reset] – Click [Reset] to reset the values entered.
