Enter the following details:

 

  1. Allergies – Click on the hyperlink to view the allergy details recorded for the patient.

  2. Active Problems - Click on the hyperlink to view the active problems recorded for the patient.

  3. View/Edit – Click the hyperlink to update or view the charge details for the respective task.

  4. Reporting Practitioner – The name of the associated reporting Practitioner is displayed. Click [?] to associate any other Reporting Practitioner.

  5. Comments – Click Comments hyperlink to record any remarks.

  6. Event – The event is displayed as ‘+/-’. You cannot update the values.

  7. Result – The values are system defined. Select a result from the list. The event status and the name of the practitioner get displayed based on the value selected for result. You can update the Event Status and the Reporting Practitioner.

  8. Event Status – Select an Event Status from the list. The system defined values are ‘In Progress’, ‘Partial’ and ‘Complete’. Select ‘In Progress’, if the treatment is in progress. You can either select the status as ‘Abnormal’ or ‘Complete’ from the screen.

The Status changes to [IM] when the Event Status is selected as ‘In Progress’. The value for ‘Status’ under the Baseline Chart is displayed as ‘In Progress’ on clicking the task details on the right hand side menu.

=> The Status changes to [IR] when the Event Status is selected as ‘Partial’. The value for ‘Status’ under the Baseline Chart is displayed as ‘Partially Resulted’ on clicking the task details in the right hand side menu.

  1. Progress Notes – Click the hyperlink to record Progress Notes. Progress Notes is enabled when a task is registered or at the treatment level when individual tasks associated to the treatment are registered. The notes differ based on the order. You can edit the notes until it is signed-off. You can click the history area to edit or view the notes that are recorded for a tooth. The treatment name is displayed as a hyperlink. Click the treatment name to view the tasks that are associated to the treatment.

An error message is displayed, if you do not record the Progress Notes for a task.

=> An alert is displayed when you record result entry for the last task associated to the treatment, if the notes is recorded, but not signed-off.

=> If the chart is closed and if notes are recorded for the treatment, but not signed-off, you can only view the notes.

=> If progression is recorded, but the notes are not recorded, an error message is displayed to indicate that there are no records.

=> If the notes are not recorded, an error message ‘Progress Notes is not entered for this treatment’ is displayed.

  1. Complication - Click Complications to record complication details.

    You can record the complications when the treatment is being done.

    Click on the hyperlink to view the Complications associated to the respective Task. You can record the details until the task is marked as data entry completed. The first task that is associated to the complication is highlighted in bold. When the task is marked as completed, you can only view the details and cannot make any changes.

    Complication Details Entry

    1. Complication – Select a Complication from the list. The Complications  defined in the OH Code Setup menu gets listed. An additional value 'Others (Free Text)' is available in the list apart from the values defined through 'Complication' function. Select 'Others (Free Text)' to record additional details on any complication. You can enter a complication in alpha numeric format with a maximum of 100 characters when the value is selected as 'Others (Free Text)'.

    2. [Record] - Click [Record] to record the complications. On clicking [Record], the details get listed in the screen under Complication Details header.

    3. [Clear] - Click [Clear] to clear the complication that is selected.

    4. Select – Select the check box to select and delete the respective Complication. A confirmation message 'Tasks Data Entry will be completed. Do you want to proceed?' is displayed on selecting the check box. Click [OK] to proceed or [Cancel] to abort the process.

    5. [Delete] – Click [Delete] to delete associated the complication.

    6. [Close] - Click [Close] to exit from the screen.

    7. Data Entry Completed – Select the check box, if association of Complication to the task is complete.

    8. Task - The task name is displayed as a hyperlink. The composition details associated to the task are displayed on selecting the task.

  2. Composition - Click Composition to record composition details.

    You can record the components while planning the treatment or completing the treatment. Click Composition Details hyperlink to record the details.

    Composition Details Entry

    1. Composition Class – Select a Composition Class from the list. The Composition Class defined in the OH Code Setup menu gets listed.

    2. Composition - Select a Composition from the list. The Compositions associated to the Composition Class using the function Composition in the OH Code Setup menu are listed.

    3. [Record] - Click [Record] to record the compositions. On clicking [Record], the composition details get listed in the screen.

    4. Select – Select the check box to select and to delete the associated composition details.

    5. [Delete] – Click [Delete] to delete associated the components.

    6. [Clear] - Click [Clear] to clear the entered data.

    7. [Close] - Click [Close] to exit from the screen.

    8. Data Entry Completed – Select the check box, if association of composition class to the task is complete.

    9. Task - The task name is displayed as a hyperlink. The composition details are displayed on selecting the task.

  3. Select the Event Status as ‘Complete’ to when the treatment is completed. The treatment is listed under the ‘Treatments Done’ menu.

  4. Click [Record] to record the event status or [Cancel] to cancel the transaction.
