Tasks for Treatments

From this function you can Associate multiple tasks to a Specialty Treatment and Query for a selected association.

Each Treatment Category has its own system defined icons. These icons appear in the baseline chart on selecting a valid treatment type against a tooth. You can easily identify the treatment on a tooth by the dentist  while accessing the chart.

Associating Tasks to Treatments

From the Code Setup > Conditions/Treatments menu, select the Tasks for Treatments function.

The Tasks for Treatments screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Click [?] function to select a Treatment from the list. The Treatments defined in the Specialty Treatments/Procedures get listed. This is a mandatory field.

  3. Click [Search] to associate Tasks to the Treatment. The Tasks defined initially are displayed. These fields cannot be altered. With these fields, Sequence Number and the Select options are displayed.

  4. Click on a Specific Alphabet under the [Select] tab to search and associate the Tasks whose names begin with the alphabets selected.

  5. You can search and associate a specific Task to a Treatment, by entering the Task Code in text box for Task. Click [Search] to select the Task. The Task details get displayed. Specify the sequence number in which the Task has to appear. Select the check box to associate the Task to the Treatment.

  6. Click the alphabet and the Tasks whose name begin with the selected alphabet get listed. You can record the sequence number for the multiple tasks and associate them to the Treatment. <Related Field Description>

  7. Select the check box to associate the Task to the Treatment. Multiple Tasks can be associated to the Treatment, if the check box ‘Treatment has multiple tasks?’ is selected in the function ‘Specialty Treatments/Procedures’. Clear the check box to associate a single Task to Specialty Treatments/Procedures.

  8. Click [Apply] to save the Tasks associated to the Treatments or [Reset] to clear the entered data.

  9. Click [Query] to search for existing records.
