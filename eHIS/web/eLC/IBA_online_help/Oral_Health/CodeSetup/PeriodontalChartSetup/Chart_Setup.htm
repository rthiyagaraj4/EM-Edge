Chart Setup

From this function you can Create a dental chart with selected components and Query for the selected Charts.

The Chart Definition function is used to define a Chart used by the Practitioners for Periodontal Charting. The Chart gets listed in the transactions while defining a Periodontal Chart to record the patient condition and the treatment against the conditions. Several inbuilt components are listed from which you can choose to associate selected components to the chart. You also have an option to record the sequence in which the components has to appear in the chart. The maximum, minimum and the Red if>= values are system defined. You can update the values for CEJ-GM, Clinical Attachment Level (CAL) and Pocket Depth (PD) components. A Practitioner can define multiple Charts and choose to use a selected Chart while recording the conditions and the respective treatment in a Periodontal Chart.

Creating a Chart

From the Code Setup > Periodontal Chart Setup menu, select the Chart Setup function.

The Chart Setup screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the Charts created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Charts.
