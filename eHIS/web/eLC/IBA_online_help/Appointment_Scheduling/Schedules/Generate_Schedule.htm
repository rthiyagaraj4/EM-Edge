Generate Schedule

From this function you can Generate Schedule for a given period based on the timetable set for the location.

Generate Schedule function allows generation of schedules by defining a period range for practitioners of a selected Clinic or for individual Practitioners. If the timetable indicates that a Roster is required for a Practitioner/Location/Day combination, the location schedules cannot be generated unless the roster details are entered.

 

Process Flow Diagram

Generating Schedule

From the OA Schedules, select the Generate Schedule.

Generate Schedule screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Generate] to generate the schedule.

 

The schedules are always generated for a future period for a particular Practitioner, Location and Day Combination. The schedules are not generated on days that have been declared as holidays.
