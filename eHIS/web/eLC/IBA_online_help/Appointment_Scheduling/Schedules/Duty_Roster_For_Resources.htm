Duty Roster for Resources

From this function you can Create a Duty Roster for a Resource against a specific location for a period range and Query for a desired Duty Roster assigned to a Resource. You can Delete a roster assigned to a resource for a specific date.

A Duty Roster for resource is used as a planner for proper utilization of the resource. A Duty Roster is entered for a particular Date/Resource/Location, only if the ‘Roster Required’ check box is selected in the timetable for the particular Resource/Location/Day combination. This function allows the generation of a Duty Roster for a resource against a specific location based on a period range. The generation of the roster depends on the timetable set up for the location and can be deleted and regenerated if there are changes in the timetable.

Creating Duty Rosters for Resources

From the OA Schedules, select the Duty Roster for Resources function.

Duty Roster for Resources screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Select] to display dates between the date range for the roster days selected. The check boxes are selected, by default. Clear the check box against the required date and day to exempt it from associating to the resource.

  4. Click [Apply] to assign the selected Roster days to the Resource.

  5. Click [Query] to search for existing roster day assigned to a Resource.

  6. Click [Delete] to delete a queried record.

 

If the ‘Roster Required’ check box is not selected in the Timetable for Scheduling function for a Practitioner, a message – Roster required is not defined in the Time Table - gets displayed on selection of the Practitioner.
