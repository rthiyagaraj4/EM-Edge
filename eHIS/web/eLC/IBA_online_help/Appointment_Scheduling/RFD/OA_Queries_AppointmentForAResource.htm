Enter the following details to query the appointment for a resource:

 

  1. Location - Select a Location from the list.. The first list is the location type and the next box lists all the locations of the selected type. The Location Type values listed are 'Clinic', 'Day Care' and 'Procedure Unit'. On selecting the required location type, the Locations listed are based on the following conditions:

    • All the Clinics/Daycare/Procedure units defined under the selected location type that allow appointments (based on the 'Allow Appointment' value set in 'Clinic', Daycare Unit' and 'Procedure Unit' setup function of Outpatient Management module).  

    • All the defined Clinics/Daycare/Procedure units must be associated to the operator station assigned to the logged in user.

Note To query for results, you must select a date, and either a location or resource.

  1. Resource -  Select a Resource from the list.  This field is enabled on selecting a location. The first list is the Resource Class and the adjacent box displays all the resource(s) of the selected type. The Resource Class values listed are 'Practitioner', 'Equipment', 'Room' and 'Other'. On selecting the required Resource Class, enter the resource name in alphanumeric format of 30 characters maximum length or click [?] function to search for the required resource. This Resource becomes mandatory based on the following condition: If 'Check in at identification' is set to Resource in 'Clinic', 'Day Care' and 'Procedure Unit' setup (OP module), the resource field becomes mandatory while defining the timetable or if 'Check in at identification' is set to Non Identifiable, this field is non-mandatory.

  1. Date - Enter the date, in dd/mm/yyyy format, for which the appointment has to be queried for a particular Location/Resource combination or click Calendar function to select the required date. This field is mandatory.
