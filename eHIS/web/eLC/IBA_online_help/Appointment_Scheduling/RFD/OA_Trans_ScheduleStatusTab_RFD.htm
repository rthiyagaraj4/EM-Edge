Enter the following details:

 

  1. Specialty - Select the Specialty based on which the Location/Resource Schedule has to be displayed from the list of valid values.

If the patient  has future appointment in the same  specialty, a warning message is displayed.

 

  1. Display Earliest Schedule – The check box is enabled on selecting a Specialty. The check box is selected, by default. You can clear the check box to refrain from viewing the earliest schedule for the location. On selecting the check box, the earliest schedule for the practitioners of the selected specialty gets displayed with a red outline marked on the date cell.

  2. Radiology Appointments – Select the Radiology Appointments check box for booking radiology appointments. If this check box is selected then,

  3. Location Type is selected as ‘Procedure Unit’ by default and disabled,

  4. Resource Type is selected as ‘Equipment’ by default and disabled

  5. Order Catalog field becomes mandatory, displays the order catalogs to the Location selected or all the radiology catalogs assigned to the Procedure Unit of type Radiology.

  6. Only when the check box is selected, the appointment is considered as a radiology specific appointment.

  7. This check box is selected by default when booking is done from Order Entry and Result Reporting and Clinician Access modules.

  1. Location - Select the required Location Type and the corresponding Location value from the list of valid values.

  2. Resource Type – Select the Resource Type as Practitioner, Equipment, Room or Other.

  3. Resource - Select the Resource using [?] function based on the Resource Type selected.

Note:

a. If you had selected the check box Come back in the Op Checkout screen – Recall Details, the Resource Type and Resource fields are populated as follows:

• If the Resource type is ‘Practitioner’, you can select a practitioner from the pick list, which will display the practitioner name by default.

• If the Resource type is ‘Room’, you can select a resource from the pick list, which will be blank, by default.

 

b. If you want to arrange group consultation, you need to select a multi-specialty practitioner as the primary resource.

 

  1. Gender - Select the required Gender from the list. The Gender value is populated from the Gender master.

  2. Position - Select the Position of the Practitioner from the list. The Positions are defined in the Position function of Application Masters module. The Position should be associated to the Practitioner in the Practitioner function of Application Masters module.

  3. [Additional Specialty] - Click [Additional Specialty] to list all the Specialties associated to the Facility. Select the check box against the required Specialties to search for practitioner schedules associated to selected specialties. A maximum of two specialties are allowed to be selected. Click [OK] to display the Specialties in the corresponding field. The Specialties are associated to the Practitioner using 'Specialty for Practitioner' function in System Configuration module. The Practitioners are associated to the Facility using 'Practitioner for Facility' function in Application Masters module.

On selecting the Primary Specialty or Location and the Additional Specialty, the system displays the schedules for the practitioners who are associated with the primary specialty and the additional specialty. Schedules generated for the Additional Specialty is not considered while searching.

 

  1. Show Sch After – Enter the value in numeric format of three digits. The entered value can be in ‘Days’, ‘Weeks’ or ‘Months’. Based on the selection made in the list box, the appointments schedule status gets displayed from the date resulted after adding the system date with the specified values in ‘Show Sch After’.

  2. Preferred Date & Time – Select the preferred date and time of the appointment from the date and time picker fields. The following controls are available for adjustments:

     - Click this button to decrease the date by a day, but not lesser than the current date.

    - Click this button to increase the date by a day.

     - Click this button to increase the time by 1 hour.

     - Click this button to decrease the time by 1 hour.

  3. Visit Type - Select the Visit Type for whom appointments schedule status is to be displayed using the list of valid values.

  4. Order Catalog - Select the required Order Catalog using [?] function. The values are getting populated from the Order Entry & Result Reporting module.

 

If the ‘Schedule Rule Applicable’ check box in OA Parameter is selected, the Schedule Status tab displays the below fields in addition to the above mentioned fields.

Patient Details

  1. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in the MP Parameter or select using [?] function.

  2. Prefix - Select the Prefix from the list. This is a mandatory field.

  3. Gender - Select the Gender of the patient from the list. This is a mandatory field.

  4. First Name - Enter the First Name in alphanumeric format as specified in MP Parameter. This is a mandatory field.

  5. Middle Name - Enter the Middle Name in alphanumeric format specified in MP Parameter. This is a mandatory field.

  6. Last Name - Enter the Last Name in alphanumeric format specified in MP Parameter.

  7. Mobile No - Enter the Mobile Number of the patient.

  8. Nationality - Enter the Nationality of the patient or select using [?] function.

  9. Language - Enter the Language of the patient or select using [?] function.

  10. View History - Click [View History] to view the summary of patient appointment history. The patient appointment details include Patient ID, No of Show Appointments, No of Reschedule Appointments, No of Future Appointments and No of Cancel Appointments.

  11. Clear - Click [Clear] to clear the entered data.

 

Schedule Criteria

  1. Practitioner Unspecified - Select the Practitioner Unspecified check box to apply rule based scheduling even if the practitioner is not specified while booking. The Rule is applied only if the Practitioner Unspecified check box is selected.

  2. Terminology Set - Select the Terminology Set from the drop-down list. The values are defined in the Terminology Set master. Once the terminology set is selected, the Procedure Code field becomes mandatory.

  3. Procedure Code - Enter the procedure code or select using [?] function.
