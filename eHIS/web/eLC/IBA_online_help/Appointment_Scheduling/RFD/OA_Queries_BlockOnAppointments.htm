Enter the following details to query the blocks on appointments:

 

  1. Location Type - Select the required location type and the corresponding location value from the list.

NoteYou need to select either Location or Resource to search for records.

  1. Resource Type - Select the required Resource Type from the list and the Resource text box gets enabled listing the Resources corresponding to the Resource Type selected. You can enter the required resource name in alphanumeric format or click [?] function to search for the required resource.

  2. Day - Select the day of the week when the blocks on appointments for a Location/Resource are to be queried from a valid list of days displayed.

  3. Block Date [From-To] - Enter the date range for which the blocks on appointments for a Location/Resource are to be queried in dd/mm/yyyy format.
