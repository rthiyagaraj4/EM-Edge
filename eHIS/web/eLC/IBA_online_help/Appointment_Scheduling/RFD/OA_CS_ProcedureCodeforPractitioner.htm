Enter the following details:

 

  1. Link By – Select either the Practitioner or Practitioner Code to be linked to the terminology set from the drop-down list.

  2. Terminology Set – Select the Terminology Set from the drop-down list.  The values are defined in the Terminology Set master.  This is a mandatory field.

  3. Specialty – Select the Specialty from the drop-down list.

  4. Procedure Code – This field is displayed, when ‘Practitioner Code’ is selected in the Link By. Enter the Procedure Code in alphanumeric format of five characters maximum length or select using the [?] function. This is a mandatory field.

  5. Practitioner – Enter the Practitioner name or select using the [?] function. This is a mandatory field.

  6. Search By – Enter the search text in the text box provided.
