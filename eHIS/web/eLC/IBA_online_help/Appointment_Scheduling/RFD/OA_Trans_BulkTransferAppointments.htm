Enter the following details to search for the required record:

 

  1. Specialty [From-To] - Select the Specialty based on which the Location/Resource Schedule has to be displayed from the list of valid values. Select the required specialty against which the appointment is booked from the list of values. All the enabled specialty records created in ‘Application Masters’ module gets displayed. This field is mandatory.

  2. Location [From-To] - You have to select two values for the location. The first list is the Location Type and the next one lists all the locations of the selected type. The Location Type values listed are ‘Clinic’, ‘Day Care’ and ‘Procedure Unit’. On selecting the required Location Type, the corresponding Locations are listed. Both the fields are mandatory.

  3. Resource From - This field is enabled on selecting a location. Enter the resource name in alphanumeric format of 30 characters maximum length or click [?] function to invoke a search screen to select the required resource.

  4. Resource To - Enter the resource name in alphanumeric format of 30 characters maximum length or select the resource to whom the schedule have to be assigned using [?] function. This field is enabled on selection of a Resource in ‘Resource From’ field.

  5. Date Range [From-To] - Enter the range for ‘From’ and ‘To’ date in dd/mm/yyyy format or select the required dates using the calendar function. Both the fields are mandatory.

  6. Reason for Revision - Select the Reason For Revision from the [?] function. All the reasons applicable for transferring, as set in ‘Other Master setup (Reason)’ of Application Masters module gets displayed. This field is mandatory.
