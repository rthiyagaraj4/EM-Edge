Enter the following details:

 

  1. Location - Select the Location Type/Location where the Patient is scheduled for an appointment from the list.

  2. Resource – This field is enabled on selecting a Location Type/Location value. Select the Resource against whom the Patient is scheduled for an appointment using [?] function. The Resource class is displayed against the resource field.

  3. Appt No - Enter the Appointment Reference Number in alphanumeric format of eight characters maximum length.

  4. Appt Date [From-To] - Enter the date range for which the appointments for Patient are scheduled in dd/mm/yyyy format. This field is mandatory only if ‘Search With’ is selected as ‘All’. The current system date is displayed as the Appointment ‘From’ Date. The second consecutive date of the current system date is displayed as the Appointment ‘To’ Date.

  5. Patient ID - Enter the Patient ID in alphanumeric format of maximum length as specified in MP Parameter or use [?] function to find out the Patient ID. This field is mandatory only if ‘Search With’ is selected as ‘Patient ID’. The future appointments of the selected Patient ID is displayed.

  6. Patient Name - Enter a valid Patient Name in alphanumeric format of 40 characters maximum length. If valid Patient ID is entered or selected using [?] function, system displays the Patient Name corresponding to the entered or selected Patient ID.

  7. Gender - Select the Gender of the Patient whose appointment is to be queried from the list of valid values.

  8. Day of Week - Select the required day of week from the list.

  9. Display only Not-identified Practitioner - Select the check box to display only Not-identified Practitioners. This check box is enabled, if 'Open to all Resources' check box is selected in 'Clinic' function of Ambulatory Care Management  module.
