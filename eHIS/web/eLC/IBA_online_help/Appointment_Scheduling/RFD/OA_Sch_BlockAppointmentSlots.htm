Enter the following details to Block Appointment Slots or Slabs:

 

  1. Location Type – Select the Location Type as Clinic, Daycare Unit or Procedure Unit.

  2. Location – Select the Location based on the selected Location Type using [?] function. This field is mandatory.

  3. Resource Type – This field is enabled on selecting the Location. Select the Resource Type as Practitioner, Equipment, Room or Other. This field is mandatory.

  4. Resource – Select the Resource based on the selected Resource Type using [?] function. This field is mandatory.

  5. Block Date - Enter the block date and time range for which the slots have to be blocked or canceled in dd/mm/yyyy format or click calendar function to select the required date range. On selecting the date range, the schedules for the start period are displayed. This schedule is used for selection of start time and end time. The schedules for which appointments are not fixed get blocked for the period. This is a mandatory field.

  6. Block Time [From-To] – The Block Time gets displayed on selecting the From and To time from the schedule. From-To time indicates the time when the appointments have to be blocked. This must correspond to the time slot defined in the Location timetable.

  7. Reason – Select the Reason for blocking from the list of valid values. All the reasons applicable for blocking as set in ‘Other Master setup (Reason)’ of Application Masters module with Block Schedule selected gets displayed. This field is mandatory.
