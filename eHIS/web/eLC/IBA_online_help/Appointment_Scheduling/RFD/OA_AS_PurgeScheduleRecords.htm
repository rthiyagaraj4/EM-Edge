Enter the following details to Purge Schedule Records:

 

  1. Location Type – Select the Location Type as Clinic, Daycare Unit or Procedure Unit.

  2. Location - Select the required Location based on the selected Location Type. This field is mandatory.

  3. Resource Type – This field is enabled on selecting a location. Select the Resource Type as Practitioner, Equipment, Room and Other. This field is mandatory.

  4. Resource – Select the Resource based on the selected Resource Type using the [?] function. This field is mandatory.

  5. Date Range [From-To] - Enter the date range in dd/mm/yyyy format or select the required dates using the calendar function. These fields are mandatory.

  6. Time Table Type - Select the Time Table Type from the list of valid values, which is either one of the three types as mentioned below. This field is mandatory.

  7. Slot - A Slot is the time required by the Practitioner to consult each patient.

  8. Slab – A Slab defines the time ranges within a day the Practitioner can see a fixed number of patients. The time taken per patient need not be constant.

  9. Free Format - Free Format is a single time frame for a given day in which a Practitioner can see a fixed number of patients. The time taken per patient need not be constant.
