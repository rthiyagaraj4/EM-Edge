Enter the following details to create Duty Roster for Resources:

 

  1. Location Type – Select the Location Type as Clinic, Daycare Unit or Procedure Unit.

  2. Location – Select the Location based on the selected Location Type using the [?] function. This field is mandatory.

  3. Resource Type – This field is enabled on selecting a location. Select the Resource Type as Practitioner, Equipment, Room or Other. This field is mandatory.

  4. Resource – Select the Resource based on the selected Resource using [?] function. This field is mandatory. The resources are listed based on the following conditions:

  5. If ‘Check in at Identification’ is set to Resource in ‘Clinic’, ‘Day Care’ and ‘Procedure Unit’ setup (OP module), the resource field becomes mandatory. If ‘Check in at Identification’ is set to Non Identifiable, this field is not mandatory.

On selecting the required Practitioner as Resource, the Roster Days of the Practitioner is displayed. All the seven days of the week are displayed and the check box is selected, by default. Clear the required check box against the days of the week to indicate the roster is not required for the cleared days.

  1. Roster Days - The Day of the week for which the roster is required is displayed. Against each day of the week, a check box is provided. The selected check box indicates that the duty roster for that day is required. The check box is selected, by default.

  2. Effective [From-To] - Enter the effective date range of the roster in dd/mm/yyyy format. These fields are mandatory.
