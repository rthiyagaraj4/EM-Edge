Secondary Booking for Recurring

From this function you can Associate secondary resources to a recurring booking created against a series type of visit.

Secondary Booking for Recurring function lists all the appointments created against patients where visit type is a series visit and recurring appointments are confirmed for a patient. This function allows identification of any secondary resources that are required for the recurring bookings.

Associating Secondary Resources to a Recurring Booking

From the OA Transactions, select the Secondary Booking for Recurring.

Secondary Booking for Recurring screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Select] to view all appointments created against patients where visit type is a series visit and recurring appointments are confirmed.

    1. Appt No – This field displays the Appointment Reference Number generated with a hyperlink.

    2. Appt Date – This field displays the date of appointment.

    3. Patient ID – This field displays the Patient ID of the patient.

    4. Patient Details – This field displays the patient details like Name, Gender and Age.

    5. From Time – This field displays the start time of an appointment based on the Appointment Type set up in timetable for scheduling.

    6. To Time – This field displays the end time of an appointment based on the Appointment Type set up in timetable.

    7. Duration – This field displays the duration of the appointment.

    8. Visit Type – This field displays the Visit Type against which the appointment is generated.

    9. Location Type – This field displays the Location Type against which the appointment is generated.

    10. Location – This field displays the location against which the appointment is generated.

    11. Resource Type – This field displays the Resource Type against which the appointment is generated.

    12. Resource Name – This field displays the resource against which the appointment is generated.

    13. Referral ID – This field displays the Referral ID's incase of referred patients.

    14. Order ID – This field displays the Order ID of the referral patient.

  3. Click Appointment Number hyperlink to display the Secondary for Appointments screen with appointment details, patient details, other details and contact details.

  4. Click [Contact Details] to view the contact details of the patient.

  5. Click [Secondary Resources] to display the Secondary Resources screen. Secondary resources screen displays availability of all secondary resources. Select the required time slot/slab. Click [OK] and the control shifts back to the Secondary for Appointment screen or [Cancel] to exit from the screen.

    The Secondary Resources lists all the secondary resources with their availability. The Secondary Resources are listed based on the following conditions:

    • The resource must belong to a Resource Type that has been identified against the visit type (Indicator ‘Series’ in OP module). This association is created using the function ‘Resource Type for Visit Type’ in Outpatient Management module. (Resource Type identified, as primary resource in location setup master of Outpatient Management module is not displayed).

    • The resource must be associated to the selected location using the function ‘Resource for Location’ in Outpatient Management module.

    • Timetable and schedule must be generated for the Location/Resource/Day combination.

  6. Click [Apply] to save the association of secondary resources to a recurring booking.
