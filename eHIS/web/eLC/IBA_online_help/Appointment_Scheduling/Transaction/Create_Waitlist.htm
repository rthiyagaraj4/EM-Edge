Create Wait list

From this function you can Create Wait List for patients in Clinics, Nursing Units, Procedure Units and Day Care Units and view the Invited Waiting List Patients.

Wait listing is used when an appointment is not available on a preferred date for a particular patient. The system maintains the list of wait-listed patients based on the priorities, ensuring the confirmation of bookings for the patients based on the criticality or need. Create wait list screen is used to manually assign patients on a wait list for Clinics, Nursing Units, Procedure Units or Day Care units.

Creating Wait List

From the OA Transactions, select the Create Wait list Function.

Create Wait list screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Invitation List] to display the Invitation Status screen and view the status of the invitation list for the current month. Click [Close] to close the Invitation Status screen and the control shifts back to the Create Wait list screen.

    To View Invited Waiting List Patients:

    1. The Invitation Status screen displays a Calendar showing the status of the current month. Click to check the schedule status for the next month Click to view the schedule status for the previous month.

    2. If Invitation exists for a date, it is displayed as a hyperlink. Click the date hyperlink to view the status of invited waiting list patients.

    3. Invitation status is indicated with a distinct color. The color represents:

    4. –Indicates that the waitlist has been opened.

    5. –Indicates that the waitlist patient has been selected.

    6. –Indicates that the waitlist patient has been invited.

    1. Click [Close] to close the Invitation Status screen and the control shifts back to the main screen.

  3. Click [Apply] to create the wait list and a wait list number is generated.

 

Patients are assigned to a Wait list Category, which becomes mandatory based on the settings in OA Parameters.
