AE Online Report Rule

From this function you can Create Online Report Rules for a Visit Type in an Emergency Location. You can Query for a desired Online Report Rule.

Online reports are outputs generated by the system, on completion of certain transactions. AE Online Report Rule function allows defining the outputs that are required on the completion of registering an attendance. Standard set of such online reports are supplied along with the module. However, the system gives a provision to set up the rules for generating these outputs based on the location and visit type. Based on the rules set up, the reports generation can be totally controlled. The reports generated on registration can be filtered based on the individual clinics and visit types or can be uniform across all clinics and visit types. These rules once defined can be updated or deleted at a later stage.

Creating an AE Online Report Rule

From the AE Admin, select the AE Online Report Rule function.

The AE Online Report Rule screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the AE Online Report Rule created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Report Rules within the facility.

  5. Click [Delete] to delete a queried record.
