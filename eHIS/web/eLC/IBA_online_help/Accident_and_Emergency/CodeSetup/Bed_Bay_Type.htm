Bed/Bay Type

From this function you can Create a new Bed/Bay category for an Emergency Location and Query for a desired Bed/Bay Type.

Bed/Bay Type is an option provided to define Bed/Bay Types across various Emergency Locations within the enterprise. Beds/Bays Types categorize the beds/bays used within an Emergency Location. You can associate multiple beds/bays to a Bed/Bay Type in a Location.

Creating a Bed/Bay Type

From the AE Code Setup, select the Bed/Bay Type function.

The Bed/Bay Type screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the created Bed/Bay Type or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Bed/Bay Type.
