Priority Zone

From this function you can Query a Priority Zone and Define Priority code description, Priority zone tab description and Color Indicator for the Priority Zone.

This function is used to assign priority code description, priority Zone Tab Description and Color Indicator for the Priority Zone. The defined priority zones can be viewed in the Register Attendance, Current Emergency Patients and Revise Attendance Details screens.

Defining Priority Zone

From the AE Code Setup > Code Setup, select the Priority Zone function.

The Priority Zone screen gets displayed.

  1. Click [Query] to get the existing Priority Zone records in query screen.

  2. Click the Priority Zone hyperlink to view the Priority Zone Description and Priority Tab Description. The listed hyperlinks are ‘R’, ‘Y’, ‘G’ and ’U’.

  3. Modify the Priority Zone Description and Priority Zone Tab Description if required. You cannot edit the Priority Zone field.

  4. Click [Apply] to save Priority Zone details or [Reset] to clear the entered data.
