Enter the following details to display a set of Patient Visit records to be checked out:

 

  1. Location - Select the Location from the list of valid values.

  2. Treatment Area - Select the Treatment Area from the list of valid values. The treatment areas associated to the selected location gets displayed.

  3. Practitioner - Select the required Practitioner using [?] function.

  4. Patient ID - Enter the required Patient ID or click [?] function to invoke the search option.

  5. Gender - Select the Gender from the list. The list of valid values is ‘Male’, ‘Female’ and ‘Unknown’.

  6. Previous Day Visit Only - This option is to list out patients from pervious days visits alone. Select the check box to enable listing of previous day’s visits only. If the check box is selected, all the visits registered within the period specified for ‘Cut off period for including previous days visits (Hours)’ as set in AE Parameter are displayed.

  7. Include Checked out Patients - This option is to list out patients who have been checked out from the emergency clinics. Select the check box to include patients who are checked out.
