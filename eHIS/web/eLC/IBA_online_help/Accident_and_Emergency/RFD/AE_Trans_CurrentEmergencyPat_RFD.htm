Enter the following query criteria to search for Patient records:

 

By default, you can view only the below fields:

 

  1. Location - Select the Location from the list.

  2. Treatment Area - Select the Treatment Area from the list of valid values. All the treatment areas associated to the selected location are displayed.

  3. Bed/Bay Type – Select the required bed/bay type from the list.

 

To view additional query fields, click the Maximize/Minimize toggle button  :

  1. Practitioner – Enter the required Practitioner name in alphanumeric format of 30 characters maximum length, or select the required Practitioner using [?] function.

  2. Patient ID - Enter the required Patient ID in alphanumeric format or click [?] function to invoke the search screen and select the required Patient ID by entering appropriate search criteria.

  3. Gender - Select the Gender from the list. The list of valid values are ‘Male’, ‘Female’ and ‘Unknown’.

  4. Previous Day Visit Only - This option is to list out patients from previous days visits alone. Select the check box to enable listing of previous day visits only. If the check box is selected, all the visits registered within the period specified for ‘Cut off period for including previous days visits (Hours)’ as set in AE Parameter gets displayed.

  5. Include Checked out Patients - This option is to list out patients who have been checked out from the emergency clinics. Select the check box to include patients who are checked out.
