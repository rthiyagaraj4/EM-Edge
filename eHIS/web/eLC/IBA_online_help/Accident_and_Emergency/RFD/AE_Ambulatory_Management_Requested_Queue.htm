Requested Queue

The ‘Requested Queue’ section displays the list of requests in queue. The records will be displayed based on date configured for the past and future request date in the Ambulance Parameter screen.

 

• The ambulance requests in ‘Requested’ and ‘Assigned’ status are displayed since the Requested and Assigned check boxes are selected by default. You can select the remaining check boxes to view other status like Completed or Cancelled requests.

• ‘From’ and ‘To’ dates will be available. You can change these dates as required. The selected date range will be displayed till the session is closed.

• You can also search for any request by entering text in the ‘Find’ text box and click [Go]. The search result will appear in the Requested Queue section.

• If you double-click any request record, the details of the selected row will appear in the Request Details section so that you can update the details of that request.

• The Record button will be enabled only when the status is Requested or Assigned. For other status, only Clear button will be enabled.

• In expanded mode, on right-clicking any ambulance request, three menu items are available. They are:

    1. Print

      Click [Print]. A dialog prompts you to continue printing. Click [OK]. The ‘Patient Transport Sheets’ gets printed.

    2. Save Settings

      Select ‘Save Settings’ menu. The ‘Customized Position for Grid table’ dialog gets displayed.

       

      Enter the ID, Description. If you want to use this setting as default value, select the Default check box and click [Submit].  The setting gets saved and appears as drop-down value along with the System Defined button value on the right hand side. If the Default check box is selected, the created ID appears by default whenever the dashboard is launched.

       

      You cannot modify or delete the created ID.

    3. Customize Column Display

      1. Select the ‘Customize Column Display’ menu to customize the columns that needs to be displayed in the Requested Queue section.

      2. On selecting the ‘Customize Column Display’ menu, the ‘Choose which columns you would like to see’ dialog gets displayed.

      3. Click [Show] or [Hide] to include or hide that column in the Requested Queue section.

      4. Click [Close] to close the dialog.

• In collapsed mode, on right-clicking any request, only Print menu gets displayed.
