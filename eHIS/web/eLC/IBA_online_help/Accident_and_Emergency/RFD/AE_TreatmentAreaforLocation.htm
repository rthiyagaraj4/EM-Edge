Enter the following details to create a Treatment Area for Location record:

 

  1. Location – Select the Location from the list to be associate to the Treatment Area. The list of locations displayed are limited to the locations of the logged in Facility. The Locations are defined in the ‘Clinic’ function of Ambulatory Care Management module. The Clinics with Level of Care set to ‘Emergency’ are listed for selection. This is a mandatory field.

  2. Treatment Area Code - Enter the Treatment Area Code in alphanumeric format of two characters maximum length. A Treatment Area Code must be unique and you cannot change the Treatment Area code after creation. This is a mandatory field.

  3. Long Description - Enter the long description of the Treatment Area for Location in alphanumeric format of 30 characters maximum length. This field is mandatory.

  4. Short Description - Enter the short description of the Treatment Area for Location in alphanumeric format of 15 characters maximum length. This field is mandatory.

  5. Gender – Select the Gender from the list, if the Treatment Area restricts admission to emergency patients based on their Gender. The displayed list consists of ‘Male’, ‘Female’ and ‘Unknown’.

  6. Open to all Practitioners of Location – Select the check box, to indicate whether the Treatment Area is open to all practitioners, or clear the check box to indicate that the Treatment Area is restricted to a set of practitioners. If the check box is cleared, you can assign Practitioners to the Treatment Area in the Practitioner for Treatment Area function. This check box is enabled by default, if for the selected Location 'Open to all Resources' check box is selected in the Clinic function, Others tab of Outpatient Management module.

  7. Priority Zone – Select the Priority Zone from the list. This value indicates the zone to which the Treatment Area belongs. A Priority Zone can be critical (Red), semi critical (Yellow), non-critical (Green) or unassigned. Based on the selected Priority Zone, the practitioners can identify the patient records that are segregated with unique colour indicators.  This field is mandatory.

  8. Enabled - Select the check box, to activate the record or clear the check box to deactivate the record. The check box is selected, by default.
