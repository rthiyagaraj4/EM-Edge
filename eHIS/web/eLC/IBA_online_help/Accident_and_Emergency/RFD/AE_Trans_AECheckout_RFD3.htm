Enter the following details to refer the patient to another Facility or Location:

 

  1. Referral Type – The Referral Type is displayed with values ‘Login Facility’, ‘Enterprise’ and ‘External’. This field is mandatory.

  2. Healthcare Setting Type – Select the required Healthcare Setting from the list of valid values. All the Healthcare Setting Type created in Systems Manager module are displayed. If referral type is Login Facility, the system defaults the Healthcare Setting Type of the logged in facility and disables the field. This field is mandatory.

  3. Referred To – Select the source to which patient is to be referred. If Referral Type is login facility, the logged in facility name is made default. This field is mandatory.

  4. Specialty – Enter the Specialty in alphanumeric format of 30 characters maximum length or click [?] function to select the Specialty. This is a mandatory field.

If Referral Type is ‘External’, the Specialty field is not mandatory.

  1. Location – Select the Location type and location from the list of valid values. The valid values for location type are ‘Clinic’, ‘Nursing Unit’, ‘Daycare’ and ‘Procedure Unit’. The list next to ‘location’ contains values that correspond to the selected Location Type.

  2. Service – Select the required Service from the list of valid values. The Services for the selected specialty and location are displayed.

  3. Practitioner – Enter the Practitioner Name in alphanumeric format of 30 characters maximum length. If referral type selected is external, this field shall allow only entry of a practitioner name. However, if Referral Type is login facility type or Enterprise, the system allows you to select the required practitioner using [?] function. The practitioners listed are based on the Specialty and location selected.

  4. Referred For – Select the purpose of referral from the list provided. This field is enabled only when location type is selected as a nursing unit. The values listed are ‘Booking’ and ‘Admit’. This field is mandatory if location type is selected as a nursing unit.

  5. Preferred Date - Enter the Preferred Date for visit or admission in dd/mm/yyyy format or select the required date using the Calendar function.

  6. Reason for Referral – Select the reason for referral from the [?] function.

  7. Priority - Select the Priority of referral from the list provided. The values are Elective, Urgent and Emergency.

  8. Referral Notes - Click on Referral Notes to enter notes of the patient being referred.
