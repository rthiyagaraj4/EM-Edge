Enter the following details to create a Practitioner for Treatment Area:

 

  1. Location – Select the Location from the list to associate to the Bed/Bay. The list of locations displayed are limited to the locations of the logged in Facility. The Locations are defined in the ‘Clinic’ function of Ambulatory Care Management module. If the ‘Open to all Practitioners of Clinic’ check box is cleared in the ‘Treatment Area for Location’ function and ‘Open to all Resources’ check box is cleared in the 'Clinic' function are listed for selection. This is a mandatory field.

  2. Treatment Area - Select the Treatment Area from the list to associate to a Practitioner. The Treatment Areas defined for the selected location in ‘Treatment Area for Location’ function are displayed. This field is mandatory.

  3. Practitioner – Enter Practitioner’s Name in alphanumeric format of 15 characters maximum length or click [?] function to select the Practitioner’s Name. This field is mandatory.

  4. Enabled - Select the check box to activate the record or clear the check box to deactivate the record. The check box is selected, by default.
