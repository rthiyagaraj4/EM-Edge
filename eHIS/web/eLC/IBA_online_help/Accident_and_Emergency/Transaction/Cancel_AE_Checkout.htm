Cancel AE Checkout

From this function you can Cancel patients checked out from the Emergency location using the AE Checkout function.

Cancel AE Checkout function is used to cancel a patient checked out from the Accident and Emergency department. This function displays a list of AE Checkout patients based on entered search criteria. Only authorized users can cancel an AE checkout based on the Application Masters Operator Station Control else a message ‘you are not authorized to cancel check out’ is displayed. The information recorded during the AE Checkout is defaulted and disabled in the Cancel AE Checkout screen with option to record the reason for cancelling checkout. AE Checkout cancellation is done only on the same day irrespective of the time the checkout has been recorded for the patient on that date. The name of the practitioner who checked out the patient is displayed.

Cancelling AE Checkout

From AE Transactions, select the Cancel AE Checkout function.

The Cancel AE Checkout screen gets displayed.

  1. Click [Search] to display patients checked out from the Accident and Emergency location on the same date or enter the search criteria to select a specific checked out patient. Click [Clear] to reset records displayed in the result area.

    Enter the following query criteria

    1. Specialty - Select the Specialty from the list. The specialties created using ‘Specialty’ function under Practitioner Setup in Application Masters module with the enabled check box selected gets listed.

    2. Patient ID - Enter the required Patient ID or click [?] function to search and select the practitioner.

    3. Location Type – The Location is displayed as Clinic and disabled.

    4. Location – Enter the Location or select the location using [?] function. The emergency locations are listed for selection.  The locations created using the function ‘Clinic’ in Outpatient Management module get populated.

    5. Practitioner - Select the required Practitioner using [?] function.

    6. Gender – Select the Gender from the list. The listed values are All, Male, Female and Unknown.

    7. Disaster – Select ‘Yes’ if the patient was registered as a disaster case. Select ‘No’ if the patient was not registered as a disaster case.

    8. Case of Trauma – Select ‘Yes’ if the patient was registered as a case of trauma. Select ‘No’ if the patient was not registered as a case of trauma.

  2. Checkout details of patients are displayed.

  3. Click the ‘Encounter ID’ hyperlink to display the Cancel Checkout screen. The screen displays Disposition, Case Definitions and In the Event of Death details as recorded in the AE checkout function.

  4. Enter the details in mandatory (*) fields. <Related field description>

  5. Click [Cancel Checkout] to record canceling AE checkout. Click [Abort Cancel Checkout] to abort cancelling AE checkout and exit from the Cancel Checkout screen.
