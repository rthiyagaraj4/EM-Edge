Ambulance Dashboard

From this function you can Create and Update Ambulance request, View Requested Queue; View Ambulance Chart in a Dashboard.

This function allows you to create the request for ambulance service, update the status of the ambulance request, manage ambulance details, view ambulance chart.  The Ambulance Dashboard includes three sections namely Requested Queue, Ambulance Chart and Request Details.

 

The ‘Requested Queue’ section displays the list of requests in queue. The records will be displayed based on date configured for the past and future request date in the Ambulance Parameter screen. You can search request for any specific dates. Also, you can print the ‘Patient Transport Sheet’ for any ambulance request.

 

The ‘Ambulance Chart’ section appears next to the’ Requested Queue’ section. This section displays the ambulance details in a bubble format in collapsed mode. By default, the Assigned, Maintenance, and Available status check boxes are selected.  The ambulance status is as follows: Grey for Maintenance, Green for Available, and Pink for Assigned.

 

The ‘Request Details’ section is displayed below the ‘Requested Queue’ and ‘Ambulance Chart’ section in collapsed mode.  The Request Details section is used to raise a new ambulance request or update status of the existing request.

Managing Ambulance Dashboard

From AE Ambulatory Management menu, select the Ambulance Dashboard function.

The Ambulance Dashboard screen gets displayed.

By default, three sections Requested Queue, Ambulance Chart and Request Details get displayed in the Dashboard.
