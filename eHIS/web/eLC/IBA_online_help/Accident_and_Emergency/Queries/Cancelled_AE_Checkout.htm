Cancelled AE Checkout

From this function you can Query a list of Cancelled AE checkouts that were cancelled using the Cancel AE Checkout function.

Querying for Cancelled AE Checkouts

From AE Queries, select the Cancelled AE Checkout function.

The Cancelled AE Checkout screen gets displayed.

  1. Enter the query criteria.

    Enter the following query criteria:

    1. Specialty – Enter the Specialty or select using [?] function.

    2. Location – Enter the Location or select the Location using [?] function.

    3. Practitioner – Enter the Practitioner or select the practitioner using [?] function.

    4. Patient ID - Enter the Patient ID or click [?] function to search and select the patient.

    5. Gender – Select the gender from the list. The listed values are All, Male, Female and Unknown.

    6. Cancelled Date From/To – Select the date range using the calendar function. The Cancelled AE checkout records for the selected date range are displayed.

    7. Cancelled By – Enter the Practitioner or select the practitioner using [?] function who cancelled the AE Checkout.

    8. Disaster – Select Yes or No from the list. The list displays disaster cases if Yes is selected on searching.

    9. Case of Trauma – Select Yes or No from the list. The list displays Trauma Cases if Yes is selected on searching.

  2. Click [Search] to select the records that satisfy the query criteria or [Reset] to clear the entered search criteria.

  3. The Cancelled AE Checkout details are displayed.

  4. Click [Clear] to clear the result screen.
