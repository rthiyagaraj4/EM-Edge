Process Details

Click [New] and enter the following details:

  1. Process – Select the required server process from the list.

  2. Next Process Date – Select the desired date by clicking the icon. This field indicates the next date and time on which the selected process has to be started. Values for these items are defaulted with current date and time. The user is allowed to change the value, if required. While updating the value, it should be greater than the current date and time.

  3. Interval – Enter the interval time (in minutes) for the server processes.

After entering the above details, you can define the options for the new process.

 

Options

  1. Patient Administration – Selecting the check box enables the background process to process only Patient Administration (ADT) related events.

  2. General Order Messaging – Selecting the check box enables the background process to process only the general order messages.  

  3. Observation Reporting – Selecting the check box enables the background process to process only Observation Reporting (LAB) related events.

  4. Scheduling – Selecting the check box enables the background process to process only the scheduling related events.

  5. Master File Notification – Selecting the check box enables the background process to process only the master file notification events.

  6. All – Selecting the check box enables the background process to process all the events.
