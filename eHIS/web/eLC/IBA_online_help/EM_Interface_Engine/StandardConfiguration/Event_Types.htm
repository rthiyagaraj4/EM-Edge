Event Types

From this function you can Create an association of Events to a Message Type and Segments to an Event Type.

HL7 Message is the atomic unit of the data transferred between CSC and interfaced applications. A Message comprises of a group of Segments in a defined sequence. Each Message has a Message Type that defines its purpose. For example, the ADT Message Type is used to transfer a portion of patient administration data. Each Message Type has defined trigger Events that initiate the exchange of messages. All such Message Types and corresponding Event Types supported by CSC are defined in the system through insert scripts.
