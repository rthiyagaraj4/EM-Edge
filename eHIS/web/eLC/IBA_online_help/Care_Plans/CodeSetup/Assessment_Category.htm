Assessment Category

From this function you can Create and maintain assessment categories.

This function enables you to create and maintain various assessment categories to group the assessment criteria. The assessment categories are updated whenever an assessment criterion is updated. You can Query for the existing records.

Creating an Assessment Category

From the CP Codes Setup > CP Code Setup, select the Assessment Category function.

The Assessment Category screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the changes or [Reset] to clear the entered data.

  4. Click [Query] to invoke the Query screen and to search for the existing records.
