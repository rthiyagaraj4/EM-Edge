Stage

From this function you can Define and maintain the various stages applicable for an encounter.

The stage is used while building libraries to identify the Care Plans or Interventions to be performed at different stages of the encounter. You can indicate the applicability of a stage as to when the intervention has to be performed. The various stages applicable are Pre-Admit, Post-Admit, Discharge and Post-Discharge.

Defining a  Stage

From the CP Codes Setup > CP Code Setup, select the Stage function.

The Stage screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save the changes or [Reset] to clear the entered data.

  4. Click [Query] to invoke the Query screen and to search the existing records.
