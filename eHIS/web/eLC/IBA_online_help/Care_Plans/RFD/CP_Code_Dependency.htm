Enter the following details:

  1. Code – Enter the code of dependency in alphanumeric format of four characters maximum length. This is a mandatory field.

  2. Long Description – Enter the long description in alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description – Enter the short description in alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Level – Select the level of dependency from the list. This is a mandatory field.

  5. Reassessment Interval – Enter the reassessment interval in numeric format of three digits maximum length. To specify the measure that supports the reassessment interval entered, select a value from the list. The listed values are ‘Days(s)’, ‘Hours(s)’, ‘Minutes(s)’, ‘Month(s)’, ‘Week(s)’ and ‘Years(s)’.

  6. Guideline – Enter the Guideline in the alphanumeric format of 4000 characters maximum length.

  7. Enabled - Select the check box to enable the dependency defined.
