Patient Series for User

From this function you can Define the patient series to be available for a user during Registration.

From the function you can select the patient series to be available for a user during registration in Register Patient, Rapid Registration, Register Emergency Patients, Register Newborn, and Disaster Patient Registration screens.  

 

Generally, patients are classified under specific groups for the purpose of Identification number generation. The Patient Series types are Alternate ID Series, National ID Series, General Series, Emergency Series, Newborn Series, Unrestricted, External Source Series and External Patient/Donor Series.

 

This function is accessible only if the ‘Patient Series Access by User’ check box is selected in MP Parameter. If this check box is cleared, you will the get error message ‘This function is not applicable since Patient Series Access by User is not checked in MP parameter.

You can also Query for the existing records to know the patient series available for the user.

Defining Patient Series for User

From the MP Admin Setup, select the Patient Series for User function.

The Patient Series for User screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Select the Patient Series from the drop-down list. This is a mandatory field.

  3. Enter the user ID or user name in the ‘Search User’ text box. You can select the search criteria as ‘Starts with’, ‘Contains’ or ‘Ends With’.

  4. Click [Search]. The user details matching the search criteria appear.

  5. The Search Grid displays the User ID, User Name and Select check box. Select the check box against the respective user record so that the selected user will be able to see only that Patient Series during registration.

  6. Click [Apply] to save the changes made or click [Reset] to clear the entered data.

  7. Click [Query] to search for existing records showing the user name and the Patient Series available for that user.

 

When you assign a patient series to a user and then clear the 'Patient Series Access by User' check box in 'MP Parameter', all the patient series will be able for the user during registration.
