1. Citizen ID - Enter Citizen ID in the format and length as specified in MP Parameter. Based on the MP Parameter setup.

  2. Other Alt ID Type – Select the required Alternate ID type from the list. The listed values are defined in the function ‘Alternate ID Type’ of MP module. The field ‘Other Alt ID’ is displayed provided ‘Accept Other Alt ID’ is set in MP Parameter (Alt ID checks tab).

  3. Other Alt No - On selecting the required Alternate ID type, enter the ID in alphanumeric format of 20 characters maximum length.

  4. Previous HN – Enter the Previous Hospital Number in alphanumeric format of maximum length as specified in the MP Parameter.

  5. Govt No - Enter the Government Number in alphanumeric format of maximum length as specified in the MP Parameter.

  6. Driving License No. - Enter the Driving License Number in alphanumeric format of maximum length as specified in the MP Parameter. Also, enter the expiry date of the Driving License number.

  7. Expatriate No. - Enter the Expatriate Number in alphanumeric format of maximum length as specified in the MP Parameter. Also, enter the expiry date of the Expatriate number.

  8. Patient Name – Enter the Patient Name.

  9. Period From/To – Enter the From and To period in dd/mm/yyyy format.

  10. Reference No. - Enter the Reference Number in alphanumeric format of eight characters maximum length.

  11. Location - Select the Location from the list of valid values. The displayed list is limited to the locations controlled by the derived Operator Station.

  12. Gender - Select the Gender from the list. Displayed list consists of ‘Male’, ‘Female’ and ‘Unknown’.

The field names are displayed based on the legends for Alternate ID defined in the MP Parameter.
