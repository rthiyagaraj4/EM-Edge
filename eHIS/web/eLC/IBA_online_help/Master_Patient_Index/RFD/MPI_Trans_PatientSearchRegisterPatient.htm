Advanced Tab

Additional search conditions are specified under ‘Advanced Tab’. The screen as shown gets displayed on selecting the tab:

 

Enter the following details:

  1. Patient ID – Enter the Patient ID in alphanumeric format of 10 characters maximum length.

  2. Citizen ID – Enter the Citizen ID in the required format and length as specified in the MP Parameter.

  3. Previous HN – Enter the Previous Hospital Number in alphanumeric format of maximum length as specified in the MP Parameter.

  4. Government Number – Enter the Government Number in alphanumeric format of maximum length as specified in the MP Parameter.

  5. Driving License No. – Enter the Driving License Number in alphanumeric format of maximum length as specified in the MP Parameter.

  6. Expatriate No - Enter the Expatriate Number in alphanumeric format of maximum length as specified in the MP Parameter.

  7. Other Alt Type – Select the Other Alt Type from the list. The listed values are ‘Birth Certificate’, ‘Passport’, ‘Mother’s PMI’, ‘PMI’, etc.

  8. Other Alt No - Enter the Other Alternate Number in alphanumeric format of maximum length as specified in the MP Parameter.

  9. Service Number - Enter the Service Number in alphanumeric format of 15 characters maximum length.

  10. First Name – Enter the First Name in alphanumeric format of 15 characters maximum length.

  11. Second Name – Enter the Second Name in alphanumeric format of 15 characters maximum length.

  12. Third Name – Enter the Third Name in alphanumeric format of 15 characters maximum length.

  13. Family Name - Enter the Family Name in the alphanumeric format of 15 characters maximum length.

  14. Soundex – Select the Soundex type as ‘Ethnic’ or ‘Generic’ from the list.

  15. First Local - Enter the first name in local language to search for a specific patient record.

  16. Second Local - Enter the second name in local language to search for a specific patient record.

  17. Third Local - Enter the third name in local language to search for a specific patient record.

  18. Family Local - Enter the local name in local language to search for a specific patient record.

  19. Gender – Select the Gender from the list. The listed values are ‘All’, ‘Male’, ‘Female’ and ‘Unknown’. On selecting the gender as ‘All’ the system displays a message ‘Please specify an Additional Criteria’ to query a specific patient record. Patient records are listed based on the specified additional criteria.

  20. Birth Date – Enter Date of Birth of the Patient in DD/MM/YYYY format or select the date using the Calendar function. Select the value as Date or Year from the list. If ‘Date’ is selected, the records of patients born on the specified date of birth are displayed in the result set. If ‘Year’ is selected, the records of patients born in the year specified in the ‘Date of Birth’ column are displayed.

  21. Head Patient ID - Enter the Head Patient ID in alphanumeric characters of 10 characters.

  22. Nationality – Select the Nationality using [?] function.

  23. Race – Select the Race using [?] function.

  24. Area – Select the Area using [?] function.

  25. Membership - Select the Membership from the list. The listed values are Organization Member, Related Member-Family, Related Member-Non Family and Non-Organization Member.

  26. Organization – Select the Organization using [?] function.

  27. Sub-organization - Select the Sub-Organization using [?] function.

  28. Order By – Select the value from the list. The result set is sorted based on the selected value.

 

Click [More Criteria] to enter more search criteria.

  1. Contact No – Enter the Contact Number of the patient in alphanumeric format of 20 characters maximum length. On selecting the contact number, the Mode field gets enabled.

  2. Mode – Select the Contact Mode from the list. The field is enabled only on entering the ‘Contact No’ field. The listed values are Email, Fax, Mobile No and Primary Resi No.

  3. Deceased only – Select the check box to display only the deceased patients in the search result.

  4. Suspended only – Select the check box to display only the suspended patients in the search result.

  5. Inactive only – Select the check box to display only the inactive patients in the search result.

  6. Newborn’s Mother Name – Enter the Mother’s Name of Newborn in alphanumeric format of 30 characters maximum length to search for the newborn baby.

  7. Registration Mode – Enter the required registration mode from the list. The listed values are ‘Normal’, ‘Emergency’ and ‘Rapid’.

  8. Old File No – Enter the old file number in alphanumeric format of 20 characters maximum length, if it exists for the patient. This field is displayed, if 'Prompt for Old MRN' is selected for the MR section attached to the selected patient series.

  9. Current Patient Class – Select the Current Patient Class check box against the modules to indicate that the current patient class belongs to IP, DC, OP or EM.
