Enter the following details to set up MP Audit Trial Parameter:

 

  1. Patient Sex – Select the check box to maintain log information about changes to gender of the patient.

  2. Patient Date of Birth – Select the check box to maintain log information about changes in Date of Birth of the registered patient.

  3. Patient Name – Select the check box to maintain log information about changes in Patient Name of the registered patient.

  4. Alias Name – Select the check box to maintain log information about changes in the Alias Name of the registered patient.

  5. Family link– Select the check box to maintain log information about changes in the Family Link of the registered patient.

  6. Organization Membership - Select the check box to maintain log information about changes in the Organization Membership of the registered patient. The field is enabled if ‘Maintain Organization Membership’ is set in MP parameter.

  7. Address – Select the check box to maintain log information about changes in Address of the registered patient.

  8. Nationality - Select the check box to maintain log information about changes in Nationality of the registered patient.

  9. Citizen ID - Select the check box to maintain log information about changes in Citizen ID of the registered patient.

  10. Occupation & Employer – Select the check box to maintain log information about changes in Occupation and Employer of the registered patient.

  11. Patient Category - Select the check box to maintain log information about changes in the patient category of the registered patient.

  12. Birth Place - Select the check box to maintain log information about changes in the Birth Place of the registered patient.

  13. Suspend/Reinstate patient – Select the check box to maintain log information on patients suspended/reinstated.

  14. Activate/Inactivate Patient – Select the check box to maintain log information on patients activated/inactivated.

  15. Record/Revoke Patient Death - Select the check box to maintain log information on patients recorded as deceased/revoked.

  16. Previous HN - Select the check box to maintain log information about changes in Previous HN details of the registered patient.

  17. Govt No - Select the check box to maintain log information about changes in Govt. No. of the registered patient.

  18. Driving License No. - Select the check box to maintain log information about changes in driving license of the registered patient.

  19. Expatriate No. - Select the check box to maintain log information about changes in Expatriate No. of the registered patient.

  20. Old File No – Select this check box if you want to maintain log information about the old file number updations of the registered patient. This check box will be cleared by default.

  21. Next of Kin - Select the check box to maintain log information about changes in Next of Kin of the registered patient.

  22. First to Notify - Select the check box to maintain log information about changes in First to Notify of the registered patient.

  23. Other Alt No - Select the check box to maintain log information about changes in Other Alternate Number of the registered patient.

  24. Registration Date Time - Select the check box to maintain log information about changes in Registration Date Time of the registered patient.

  25. Marital Status - Select the check box to maintain log information about changes in Martial Status of the registered patient.

  26. Residency - Select the check box to maintain log information about changes in Residency of the registered patient.

  27. Status - Select the check box to maintain log information about changes in Status of the registered patient.

  28. Race -  Select the check box to maintain log information about changes in Race of the registered patient.

  29. Religion - Select the check box to maintain log information about changes in Religion of the registered patient.

  30. Ethnicity - Select the check box to maintain log information about changes in Ethnicity of the registered patient.
