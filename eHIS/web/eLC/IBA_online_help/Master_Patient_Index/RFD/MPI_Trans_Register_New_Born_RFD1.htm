Enter the following details to Register Newborn patient:

 

The mother patient details displayed are Name, Age, Gender and ID.

 

  1. Patient Series – Based on the parameter setup, the patient series are listed. Refer to the table below for the list. This field is mandatory. Select the required series.

  1. Patient ID – The Patient ID is auto generated or has to be entered manually depending on the selected patient series. You are prohibited to enter Patient ID, if system generates ID for the selected Patient Series, or a modal window appears informing to enter ID. This field is mandatory.

 

The function provides three tabs, Mother Details, New Born. Each tab is explained below:

 

Mother Details

 

The delivery details of the mother patient are recorded in this tab. Enter the following details:

 

  1. Mother’s My PR – The Permanent Residency status specified for the mother patient is displayed as read only text.  This field is displayed based on the legends for Alternate ID defined in the MP Parameter.

  2. Mother’s Previous HN - The previous hospital number details specified for the mother patient during patient registration is displayed as read-only text.

  3. Multiple Birth – Select the check box, if the selected mother patient has delivered more than one baby during current encounter.

  4. No. of Births – This field is enabled only if ‘Multiple Birth’ check box is selected. Enter the number of babies delivered by the mother patient in numeric format of two digits maximum length. If ‘Multiple Birth’ is not selected, the number of birth is displayed and disabled as ‘1’.

  5. Gestation – Gestation refers to the duration between the conceiving date and date of delivery. Enter the gestation period in numeric format of three digits maximum length. The entered value must be within the Gestation range specified in MP Parameter or, an alert is displayed. The unit of measure specified in the MP Parameter is displayed as read-only value. This field is mandatory.

  6. Delivery Type – Select the Delivery Type from the list. If the Delivery Type selected is created with Indicator 'Normal', all the three Complication in stages of Labour fields in Mother Details Section of Register New Born function are displayed and Complications of Delivery field is disabled. If the Delivery Type selected is created with Indicator 'C Section', Complications of Delivery field is enabled and all the three Complication in stages of Labour fields are disabled. If the Delivery Type selected is created with Indicator 'Others', both 'Complications of Delivery' field and 'Complications in stages of labour' fields are enabled.

    1. If the Delivery Type selected is created with Indicator 'Normal', all the three Complication in stages of Labour fields in Mother Details Section of Register New Born function are displayed and Complications of Delivery field is disabled.

    1. If the Delivery Type selected is created with Indicator 'C Section', Complications of Delivery field is enabled and all the three Complication in stages of Labour fields are disabled.

    1. If the Delivery Type selected is created with Indicator 'Others', both 'Complications of Delivery' field and 'Complications in stages of labour' fields are enabled.

    1. When the Number of Births is more than one, the Delivery Type field is enabled and you can select the value from the list while registering for each new born.

  7. Gravida – Enter a value in numeric format of two digits maximum length. This field is mandatory.

  8. Parity – Enter a value in numeric format of two digits maximum length. The value of parity cannot be greater than gravida. This field is mandatory.

  9. Attending Practitioner – Enter the name of Practitioner attending the patient in alphanumeric format of 30 characters maximum length or select the practitioner name from the [?] function. This field is mandatory.

  10. Attending Nurse - Select the name of the nurse attending the birth from the list. The Practitioners of type ‘Nurse’ and ‘Mid-Wife’ associated to the logged in facility are listed.

  11. Complications of Pregnancy - Select the Complication of Pregnancy from the list. The listed values are created in Complications of Pregnancy function.

  12. Fetus Complications of Delivery – Select the problem that occurred during delivery from the list. The listed values are defined in the function ‘Complications of Delivery’.

  13. Complications in First Stage of Labour - Select the Complications in First Stage of labour from the list. The listed values are defined in Complications of Delivery in Master Patient Index module.

  14. Complications in second Stage of Labour - Select the Complications of second Stage of labour from the list. The listed values are defined in Complications of Delivery function in Master Patient Index module.

  15. Complications in Third Stage of Labour - Select the Complications of Third Stage of labour from the list. The listed values are defined in Complications of Delivery function in Master Patient Index module.

  16. Delivery Indication - Select the delivery indication from the list. The listed values are defined in the function ‘Delivery Indication’.

  17. Procedure Classification – Select the procedure classification using the [?] function.

  18. Vaginal Delivery Type - Select the Vaginal Delivery Type using the [?] function.

  19. Delivery Surgical Classification – Select the Delivery Surgical Classification using the [?] function.

  20. Remarks – Enter remarks in text format.

 

Antenatal Care Details

 

  1. Care Given – Select the ‘Care Given’ check box to enter the Antenatal Care Details. Once the ‘Care Given’ check box is selected, the ‘Given Location’ field becomes enabled.

  2. Given Location – Enter the Location from which the care is availed. There are two options listed. They are ‘In Hospital’ and ‘External Hospital’.

 

Father Details

 

  1. Patient ID – Enter father Patient ID or select it using the [?] function. On entering the father Patient ID, all demographics recorded during his registration are displayed in the respective fields as read-only text.

On entering the father Patient ID, the system verifies the age with that of the minimum age set for father patient in MP parameter setup. Registration cannot proceed, if found violating.

 

  1. Name – The Name of the father is automatically populated on entering the Patient ID; else, enter the father’s name in alphanumeric format of 80 characters maximum length.

  2. Citizen ID - Enter the Citizen ID of the father in the format as specified in MP parameter.

  3. My Kad /Kid - The compulsory identity document number of the father is displayed.

  4. My PR – The Permanent Resident status and expiry date of the father is displayed.

The above field names are displayed based on the legends for Alternate ID defined in the MP Parameter.

 

  1. Other Alt ID – Enter the Other Alt ID of the father in the format as specified in MP Parameter.

  2. Previous HN – Enter the Previous hospital number of the father in the format as specified in MP Parameter.

  3. Nationality - Select the nationality from the list.

  4. Race - Select the required race from the list.

  5. Govt Employee – Select the check box to indicate the father is a government employee.

  6. Religion – Select the Religion from the list.

  7. Occupation Class - Select the Occupation Class of the father from the list.

  8. Occupation - Select the Occupation of the father from the list. The listed values correspond to the selected occupation class. If the occupation is not available, enter the same in alphanumeric format of 15 characters maximum length.

  9. Remarks - Enter the Remarks in alphanumeric format of 100 characters maximum length.

 

Newborn Details

 

The newborn details are recorded in this tab.

 

  1. Gender – Select the Gender of the baby from the list. This field is mandatory. The listed values are ‘Male’, ‘Female’ and ‘Unknown’.

  2. Birth Date – System date is displayed as birth date. The date cannot be greater than system date and should be within the period specified in MP parameter (Allow Registration within [] of days of birth). Use the Calendar function to select the required date. This is a mandatory field.

  3. Birth Time - Enter the Birth time of the baby in numeric format of hh:mm.

  4. Relationship – This field is enabled on selecting the gender of the newborn. The default relationship specified in MP parameter for the gender is displayed. If value is not set for the gender in ‘MP Parameter’ function, select the relationship of the newborn to the inpatient from the list. The listed values are defined in the master codes of Master Patient Index module.

  5. Outcome – Select either ‘Live’ or ‘ Still Born’ from the list. By default, ‘Live’ is displayed. If the Outcome is selected as Still birth, you cannot modify it in the Maintain Birth Register function. If the default prefix is defined for Still Born outcome, the defaulted prefix is displayed on selecting the value 'Still Born'. The gender is not defaulted. You can change the defaulted prefix.

    [Still Birth Details]

    2. Age – The Age of the Attending practitioner recorded in the Mother details section in Register New born function is displayed. The age of the practitioner is recorded in the Practitioner function in Application Masters module.

    3. License number – The License number of the Attending practitioner recorded in the Mother details section in Register New born function is displayed. The License number of the practitioner is recorded in the Practitioner setup function in Application Masters module.

    4. Attendant at birth – Select the value from the list. The listed values are ‘Accoucheuse’, ‘Traditional Medicine’, ‘Midwife’, ‘Nurse’ ,’Midwife Assistance’ and ‘Physician’. The value ‘Physician’ is displayed by default.

  6. Indicator Mark – This field is enabled only when the Outcome is selected as ‘Still Born’. Select either ‘Fresh’ or ‘Macerated’ from the list.

  7. Born Where – Select the place where the baby was born from the list.

  8. Born At - The location type and location are to be specified. Select the location type from the first list. The listed values are ‘Clinic’, Nursing Unit, Day Care Unit and Emergency. On selecting the Location Type, select the required location. The second list, lists locations corresponding to the selected Location Type. This field is applicable only when ‘Born where’ is set as ‘In Facility’.

  9. Location – Select the location from the list. The list of pre-defined location is listed.

  10. Birth Place – Enter the Birth Place in alphanumeric format of 15 characters maximum length or select it using the [?] function.

  11. Birth Certificate No – Enter the Birth Certificate Number in alphanumeric format of 15 characters maximum length.

  12. Birth Weight – Enter the Weight of the child in numeric format of six digits maximum length. The entered weight is validated against the weight range specified in MP Parameter. This field is mandatory. The displayed unit of measurement is specified in the MP Parameter setup.

  13. Length of Baby – Enter the length of the child in numeric format of six digits maximum length. The entered length is validated against the length range specified in MP Parameter. This field is mandatory. The displayed unit of measure is specified in the MP Parameter setup.

  14. Head Circumference – Enter the head circumference of the child in numeric format of six digits maximum length. The entered value is validated against the head circumference range specified in MP Parameter. This field is mandatory. The displayed unit of measure is specified in the MP parameter setup.

  15. Chest Circumference – Enter the chest circumference of the child in numeric format of six digits maximum length. The entered value is validated against the chest circumference range specified in MP Parameter. The displayed unit of measure is specified in the MP Parameter setup.

  16. Congenital Anomalies – Select the congenital anomalies from the list. The listed values are defined in the function ‘Congenital Anomalies’. This field is enabled only if outcome is selected as ‘Live’.

  17. Complications Of Delivery – Select the complication of delivery from the list. The listed values are defined in the function ‘Complications of Delivery’.

  18. Apgar Score (in 1 min) (in 5 mins) (in 10 mins) – Click  to expand the fields to capture the detailed Apgar Score and  to collapse the fields. On clicking , the Apgar Scores at 1 minute, 5 minutes and 10 minutes get displayed. It is mandatory to capture the Apgar Scores at 1 minute and 5 minutes, if the value in the field ‘Born Where’ is selected as ‘Facility’. The Apgar Score calculated can be modified after finalizing the birth register in the Maintain Birth Register function, if the Change of Data allowed after finalization check is selected in MP Parameter.

Activity

  • Select the Activity rate at 1 minute, 5 minutes and 10 minutes from the list. The listed values are Absent, Arms &legs extended and Active movement with flexed arms& legs. On selecting the values from the list, the Apgar Score is calculated and displayed in the Activity text box. The total score is updated in the Apgar Sign field automatically on selecting the values in the individual Apgar Score fields. You cannot modify the total values. The score for Absent is 0, Arms &legs extended is 1 and Active movement with flexed arms& legs is 2.

Pulse

  • Select the heart beat rate at 1 minute, 5 minutes and 10 minutes from the list. The listed values are Absent, Below 100bpm and Above 100bpm. On selecting the values from the list, the Apgar Score is calculated and displayed in the Heart Rate text box. The total score is updated in the Apgar Sign field automatically on selecting the values in the individual Apgar Score fields. You cannot modify the total values. The score for Absent is 0, Below 100bpm is 1 and Above 100bpm is 2.

Grimace

  • Select the Breathing rate at 1 minute, 5 minutes and 10 minutes from the list. The listed values are No Response, Facial Grimace and Sneeze, Cough, Pulls away. On selecting the values from the list, the Apgar Score is calculated and displayed in the Grimace text box. The total score is updated in the Apgar Sign field automatically on selecting the values in the individual Apgar Score fields. You cannot modify the total values. The score for No Response is 0, Facial Grimace is 1 and Sneeze, Cough, Pulls away is 2.

Appearance

  • Select the Appearance rate at 1 minute, 5 minutes and 10 minutes from the list. The listed values are Blue-Gray Pale all over, Pink Body and blue extremities and Normal over entire body-completely pink. On selecting the values from the list, the Apgar Score is calculated and displayed in the Appearance text box. The total score is updated in the Apgar Sign field automatically on selecting the values in the individual Apgar Score fields. You cannot modify the total values. The score for Blue-Gray Pale all over is 0, Pink Body and blue extremities is 1 and Normal over entire body-completely pink is 2.

Respiration

  • Select the Breathing rate at 1 minute, 5 minutes and 10 minutes from the list. The listed values are Absent, Slow, irregular and Good, Crying. On selecting the values from the list, the Apgar Score is calculated and displayed in the Breathing text box. The total score is updated in the Apgar Sign field automatically on selecting the values in the individual Apgar Score fields. You cannot modify the total values. The score for Absent is 0, Slow, irregular is 1 and Good, Crying is 2.

If any of the individual Apgar Scores at 1 minute or 5 minutes is not captured, an error message ‘Apgar score at 1 min and 5 min cannot be blank’ is displayed.

Vaccination Details

  6. Vitamin K - Select the check box to indicate Vitamin K is administered to the newborn. This field is enabled only if outcome is selected as ‘Live’.

  10. Admission Date/Time - Enter the Vitamin K administration date and time or select it using the calendar function. This field is enabled on checking the ‘Vitamin K’ check box.

  11. Hepatitis B - Select the check box to indicate Hepatitis B is administered to the newborn. This field is enabled only if outcome is selected as ‘Live’.

  12. Vaccination Site - Enter the site of administration in alphanumeric format of 30 characters maximum length. This field is enabled on checking the ‘Hepatitis B’ check box.
