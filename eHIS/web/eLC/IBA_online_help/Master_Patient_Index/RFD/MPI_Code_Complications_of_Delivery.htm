Enter the following details to create a Complication of Delivery record:

 

  1. Code - Enter the Complications of Delivery code in the alphanumeric format of 10 characters maximum length. This is a mandatory field.

  2. Long Description - Enter long description of the complications of delivery in the alphanumeric format of 30 characters maximum length. This is a mandatory field.

  3. Short Description - Enter short description of the complications of delivery in the alphanumeric format of 15 characters maximum length. This is a mandatory field.

  4. Type of Complication - Select the Type of Complication from the list. The listed values are Mother and Fetus. The values created using Mother are listed in the Complications in 1st/2nd/3rd stage of labour fields under Mother Details section in Register Newborn function. The values created using Fetus are listed in the Fetus Complications of Delivery field in the Newborn Details section in Register Newborn function. When the value Fetus is selected, the Complications in 1st/2nd/3rd stage of labour fields get disabled.

  5. Complications in First Stage of Labour - Select the check box if the created Complication of Delivery is a Complication in the First Stage of labour.

  6. Complications in second Stage of Labour - Select check box if the created Complication of Delivery is a Complication in the Second Stage of labour.

  7. Complications in Third Stage of Labour - Select the check box if the created Complication of Delivery is a Complication in the Third Stage of labour.

  8. Enabled - Select the check box to enable the record. Only the enabled records are available in the transaction functions.
