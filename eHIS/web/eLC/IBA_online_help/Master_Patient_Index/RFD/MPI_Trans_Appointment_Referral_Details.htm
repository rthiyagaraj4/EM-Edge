Appointment Referral Details

From this function you can Search records of patients by specifying the search criteria or Register Patient if the records do not exist for the specified criteria.

Searching for Appointment/Referral Patient Records

From the MP Transactions>Register Patient> Create screen, select the desired value from the 'Populate From’ list. Click [?] to invoke Appointment/Referral screen.

The Appointment/Referral Search screen gets displayed.

  1. Enter details in the mandatory (*) fields. <Related Field Description>

  2. Click [Search] to retrieve the Appointment/Referral patient records based on the specified criteria.

  3. The Patient Records are listed for the specified criteria.

  4. Click Reference No hyperlink to select the desired patient record.
