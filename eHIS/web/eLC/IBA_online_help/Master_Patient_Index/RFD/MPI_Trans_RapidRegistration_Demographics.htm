Enter the following details:

 

  1. Patient Series - The default value appears based on the settings in MP Parameter. All the patient series defined in the function 'Patient Series' are listed. Select the required value from the list. Refer to the table below for default value of Patient Series. On selecting the Patient Series, you can capture the mandatory fields which is controlled internally.

 

Value for DEFAULT in MP Parameter (under Patient Numbering tab -> Patient Series)

Default Patient series in Register Patient function

General

Patient series specified for the field 'General' in MP Parameter is displayed as default. The list contains all the patient series defined in 'Patient Series' function. Select the required series.

Alternate ID 1

Patient series specified for the field 'With Alternate Series' in MP Parameter is displayed as default. The list contains all the patient series defined in 'Patient Series' function. Select the required series.

National ID

Patient series specified for the field 'With National ID' in MP Parameter is displayed as default. The list contains all the patient series defined in 'Patient Series' function. Select the required series.

 

Patient ID generation is based on the selected patient series. If the system automatically generates Patient ID for the selected series, you are not allowed to enter the Patient ID.

 

  1. National ID - Enter the National ID in the numeric format as specified in MP Parameter. The displayed field legend corresponds to the set 'Prompt' in MP Parameter.  

  2. Other Alt No - Enter the Other Alternate No from the List.

  3. NRIC - Enter the National Registration Identity Card (NRIC) number of the patient. This field is available only when the following setting is done in the ‘MP Parameter.’   

    • Clear the 'Invoke Routine' check box.

    • Disable 'Check Digit Scheme' drop-down list.

    • Select the 'External Interface Required' check box.

On entering the NRIC number, the following fields: Government No, Expiry Date, Prefix, First Name, Gender, Date of Birth, Residency, Nationality, Race, Religion, Address1, Address 2, Address 3 and Address 4 are populated automatically.

  1. Previous HN – Enter the Previous Hospital Number in alphanumeric format of maximum length as specified in the MP Parameter. Enter the expiry date of the Previous HN.

  2. Form No 60 – Select the value from the drop-down list. The listed values are Yes and No.  By default, the value ‘No’ is selected.

• If the Form No 60 drop-down list is selected as ‘No’, the PAN Card field gets enabled and it is not mandatory.

• If the Form No 60 drop-down list value is selected as ‘Yes’, the PAN Card field is disabled.

  1. Government No - Enter the Government Number in alphanumeric format of maximum length as specified in the MP Parameter. Enter the expiry date of the Government number.

  2. Driving License No. - Enter the Driving License Number in alphanumeric format of maximum length as specified in the MP Parameter. Enter the expiry date of the Driving License number.

  3. Expatriate No. - Enter the Expatriate Number in alphanumeric format of maximum length as specified in the MP Parameter. Enter the expiry date of the Expatriate number.

  4. PAN Card – Enter the PAN Number of the patient in alphanumeric format. This field gets enabled only if the ‘Form No 60’ is selected as ‘No’ and it is not mandatory.  The ‘Alternate ID 3’ is configured as the PAN Card.

The ‘Form No 60’ and ‘PAN Card’ fields are applicable only for users in India. The fields ‘Form No 60’ or ‘PAN Card’ can be recorded for all patients irrespective of whether 'Residency' is selected as 'Citizen' or ‘Non-Citizen’.

  1. Membership - Select the Membership from the list. The listed values are Organization Member, Related Member - Family, Related Member - Non Family, Non-Organization Member. It is mandatory to select the Alternate ID as set in the field 'Alternate ID Type for Membership ID' in MP Parameter function. This field is displayed if 'Maintain Organization Membership' check box is selected in the MP Parameter function.

  2. Organization - Select the Organization using [?] function. The Organization Sub Unit is displayed on selecting the Organization. If more than one Sub Unit is associated to the Organization, select the Sub Unit from the list.

  3. Head of Family Patient ID - This field is displayed based on the ‘Maintain Family Link’ set in MP Parameter. If family link is to be maintained, enter the Head Patient ID in the required format as specified in the MP Parameter, or click on the [?] to select the required Head Patient ID. The Head patient name is displayed next to the field.

  4. Relationship - The Relationship is displayed as set in the 'Relationship for Organization Member' field in the MP Parameter function on selecting the Membership as 'Organization Member'. Select the Relationship to the displayed Head of Family. This field is mandatory.

  5. Relationship Level - This field is displayed on selecting the Relationship with a level defined in the Relationship function. Select the Relationship Level using [?] function. The Relationship Levels defined in the Relationship Level function is displayed. This field is mandatory.

  6. Title - Select the Prefix from the list. The listed values are defined in the function 'Name Prefix'.

  7. First Name - Enter the First Name in alphanumeric format as specified in MP Parameter.

  8. Second Name - Enter the Second Name in alphanumeric format as specified in MP Parameter.

  9. Third Name - Enter the Third Name in alphanumeric format as specified in MP Parameter.

  10. Family Name - Enter the Family Name in alphanumeric format as specified in ‘Legal Names’ tab in MP Parameter. This is applicable irrespective of whether the 'Accept Patient Name in Multi parts' check box, Suffix or Prefix check box is selected or not. But if any other Name component check box is selected other than the ‘Family Name’ check box  in MP Parameter, the Family Name field size will be changed to 20 characters.  This is a mandatory field.

  11. Suffix - Select the Suffix from the list provided. The values are listed from Name suffix function.

  12. Gender - Select the Gender of the Patient from list. The listed values are Male, Female and Unknown. This field is mandatory. Based on the routine defined for the National ID in MP Parameter, the gender is displayed automatically on entering the National ID.

  13. Age & Birth Date & Time – Select the check box and enter the Birth date and time in yyyy/mm/hh format. If the age of patient is entered in years, months and days and not through the calendar function, the system identifies the age of the patient as estimated. If the age of the patient entered while registering is an estimated age, an estimated age indicator (Estd.) is displayed in Patient Context Banner and Patient Encounter Banner across all modules in the functions, where Patient Banner and Patient Context Banner are displayed. If you change the age of the patient using the Calendar function in the Change Patient Details function, the indicator is not displayed in Patient Context Banner and Patient Encounter Banner. On entering the birth date, the system calculates the Patient's age and displays it in the Age (Y/M/D) fields. If National ID is entered, the date of birth is displayed automatically based on the age validation logic defined in the routine associated with the National ID or select the date of birth using the 'Calendar' function. This field is mandatory.

If the hours in 'Display New Born Age in Hrs When age is less than or equal to [] Hours' is recorded, the age for the new born is displayed in hours and the time is displayed in the 'T' text box.

  1. Blood Group - Select the Blood Group from the list. The listed values are defined in the Medical Records module.

  1. RH(D) Factor - Select the RH (D) Factor of the Patient from the list. The listed values are created in Rhesus Factors function in Terminology & Coding Management  module.

  2. Birth Place - Enter the Birth Place of the Patient in alphanumeric format of 30 characters maximum length, or use the Calendar function to select the required birthplace. If National ID is entered, the birthplace is displayed based on the birthplace logic defined in the routine associated with the National ID. If the required birthplace is not available in the search list, enter the birthplace in alphanumeric format of 30 characters maximum length in the text box provided next to the [?] function.

  3. Marital Status - Select the Marital Status of the Patient from the list. The listed values are defined in the function 'Marital Status'.

  4. Residency – The residency of the patient is automatically validated by the system on selecting the nationality of the patient (other than the default nationality) at registration. The system automatically validates the Nationality on selecting the Residency and the ‘Nationality’ of the patient is changed from default value to other value. Select either 'Citizen' or 'Non-citizen'.

  5. Status - Select either 'Legal' or 'Illegal' as citizen status. By default, 'Legal' is selected. If residency is set as 'Citizen', 'Legal' is the only option available. If the residency is set as 'Non-Citizen', the status can either be 'Legal' or 'Illegal'.  

  6. Patient Category - Select the Category of the Patient from the list. The listed values are defined in the function 'Patient Category' and the patient category must be associated to the Membership using ‘Pat Cat by Org. Membership’ master function.

  7. Nationality - The default Nationality set in MP Parameter is displayed. Select the required value using the lookup function [?]. This field is mandatory. If National ID is entered, the default nationality set in MP Parameter is displayed and disabled.

  8. Race – Select the Race from the list. This field is mandatory. The listed values are defined in 'Race' function.

  9. Ethnicity - Select the Ethnicity from the list. The listed values correspond to the selected race.

  10. Religion - Select the Religion of the Patient from the list. The listed values are defined in the function 'Religion'.

  11. Language - Enter the Language in alphanumeric format of 20 characters maximum length or select the Language using [?] function.

The fields displayed in the screen may vary depending on the MP Parameter Setup.
