Organization

From this function you can Define various Organizations applicable during Patient Registration and Query for the desired Organization.

An Organization is created and associated to a Sub Organization. This function is available only if the Maintain Organization Membership check box is selected in the MP Parameter function.

Creating an Organization

From the MP Code Setup > MP Code Setup, select the Organization function.

The Organization screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the Organization ID and the Organization Name.

  3. Select the Enabled check box to enable the defined record. This check box is selected, by default.

  4. Click [Record] to display the Organization Sub Unit frame or [Clear] to reset the entered data.

  5. Enter the details in the mandatory (*) fields in Organization Sub Unit frame. <Related Field Description>

  6. Click [Record] to save the Organization Sub Unit details or [Clear] to reset the entered data.

  7. On recording, the Sub Unit details are displayed in the Sub Unit Details frame.

  8. Click Sub Unit ID hyperlink to modify the Organization Sub Unit Details.

  9. Click [Query] to query and update the existing records.
