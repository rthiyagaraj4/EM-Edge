View Patient History

From this function you can Query and view the changes made in Patient's Demographics/Status.

Changes to Demographics of patients are performed using the function 'Change Patient Details'. The Status changes are performed using 'Activate/Inactivate Patient', 'Suspend/Revoke Patient' and 'Record/Revoke Patient Death' functions. You can select a patient and view the changes in Demographics/Status using this function. Changes to those elements set in the function 'MP Audit Trail Parameter' can only be viewed.

Querying for Patient History

From MP Queries, select the View Patient History function.

The View Patient History screen gets displayed.

  1. Enter the Patient ID in the format as specified in the MP Parameter Setup, or search for the Patient ID using [?] function. This field is mandatory.

  2. Select the History Type to be viewed from the list. The History Types listed are set in the 'MP Audit Trail Parameter' function.

  3. Click [Search] to select a specific record and view details or [Clear] to clear the records displayed in the result area.

  4. Click [Reset] to clear the entered data.

 

If History Type is not selected, all Demographics of the selected patient that are changed and tracked are shown in the result screen.
