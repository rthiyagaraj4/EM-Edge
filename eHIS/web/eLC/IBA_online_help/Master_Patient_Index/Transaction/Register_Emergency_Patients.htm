Register Emergency Patients

From this function you can Register patients brought to the facility under emergency conditions and patients whose identity is not known. You can also Query for the registered patient details.

Registration of patient whose details are not known involves capturing age and gender only. The patient is registered as ‘Unknown’. A default Patient Series as set in MP Parameter is assigned to the patient for Patient ID generation. Medical Record number is generated based on the MR Section associated to the selected Patient Series. On generation of file number, reports applicable to the assigned series are listed for selection. The generated file is assigned a permanent file storage location, if Patient File Tracking module is installed.

If details of the patient are known, a Patient Series is assigned and minimum details are captured. Upon registration, ID is generated based on the assigned Patient Series. Medical Record number is also generated based on the MR section associated to the selected Patient Series.

Registering Emergency Patients

From the MP Transactions, select the Register Emergency Patients function.

The Register Emergency Patients screen gets displayed.

  1. Click [Create] to invoke the create screen.

 

Registering Emergency Patients with details known

  1. Clear the ‘Details Unknown’ check box, if the details of the patient are known.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

Registering Emergency Patients with details unknown

  1. Select the ‘Details Unknown’ check box, if the patient details are unknown. By default, this check box appears selected.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

 

  1. Click [Apply] to save the Emergency Patient Registration or [Reset] to clear the entered data.

  2. Create Patient File screen displays if file management parameters are set for the facility. Click [OK] to generate file number or [Cancel] to cancel file generation.

  3. If internal reports are defined, Internal Reports screen gets displayed; click [OK] to generate Internal Reports or [Cancel] to exit from the screen.

  4. Patient ID and File Number is generated and displayed.

  5. Click [Query] to search for existing Patient Details.
