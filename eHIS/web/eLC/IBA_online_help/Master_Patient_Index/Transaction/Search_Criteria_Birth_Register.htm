Search Criteria Birth Register

Enter the following search criteria:

 

  1. Patient ID – Enter the newborn Patient ID in the format as defined in the MP Parameter setup.

  2. Search Name with Local Language – Select the check box to search for the patient name with local language.

  3. Birth Date – Enter the Date of Birth in the date format of dd/mm/yyyy. Specify the Date of Birth range (From and To date), or use the Calendar function to select the required date.

  4. Gender – Select the Gender from the list. The listed values are All, Male, Female and Unknown.

  5. Mother Patient ID – Enter the Mother Patient ID in the required format or select it using the [?] function. The Gender in the search screen is defaulted to ‘Female’ on clicking [?] function to search for the Mother Patient ID.

  6. Previous Hospital Number – Enter the Previous Hospital Number in alphanumeric format of 15 characters maximum length.

  7. Name Search By – Select the criteria from the list of values. The values available are Baby Name, Mother Name and Father Name.

  8. Soundex – Select the Soundex type as either ‘Ethnic’ or ‘Generic’.

  9. First Name – Enter the First Name in alphanumeric format of 20 characters maximum.

  10. Second Name – Enter the Second Name in alphanumeric format of 20 characters maximum.

  11. Third Name – Enter the Third Name in alphanumeric format of 20 characters maximum length.

  12. Family Name - Enter the Family Name in alphanumeric format of 20 characters maximum length.

 

If ‘Accept Patient’s name In Local Language’ is selected in MP Parameter, enter names in local language. The above field names are displayed based on the legends for Alternate ID defined in the MP Parameter.

 

After entering the required query criteria, click [Search] to get the result.
