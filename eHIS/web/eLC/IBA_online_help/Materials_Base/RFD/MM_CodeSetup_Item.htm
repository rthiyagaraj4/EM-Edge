Enter the following details to create an Item:

 

  1. Item Code – Enter the code of the Item in alphanumeric format of 20 characters maximum length. Once the code is defined and saved, it cannot be changed in query function. This is a mandatory field.

  2. Item Alpha Code – Enter the alpha code for the Item in alphanumeric format of 10 characters maximum length. This is a mandatory field.

  3. HSN Code - Enter the code of the Item in alphanumeric format of 8 characters maximum length. This is a mandatory field. You can only create a HSN code for new Items. This field does not allow special characters. The code is disabled and cannot be modified once it is created. For existing items, the HSN Code field is blank and disabled. You can associate HAS Code with multiple items.

  4. Short Description - Enter the short description of the Item in alphanumeric format of 40 characters maximum length. This is a mandatory field.

  5. Long Description - Enter the long description of the Item in alphanumeric format of 60 characters maximum length. This is a mandatory field.

  6. Click [Show Similar Items] to view the similar Items. The Items are displayed based on the text typed in the description text box, which starts with the same spelling.

  7. Item Remarks – Enter the Remarks of the entered item in alphanumeric format of 120 characters maximum length. This is a mandatory field.

  8. Category Type – Select the Category type as ‘Inventory’ or ‘Non-Inventory’. This is a mandatory field. The value ‘Inventory’ is displayed, by default and disabled if Sterile Stock module is not installed. This is a mandatory field.

  9. Item Specification – Click Item Specification hyperlink to enter the specifications of the Item. On entering the Item Specification, click [Close] to close the Item Specification screen.

  10. Pharmacy Stock Item – The check box is enabled if Inventory Management module of Enterprise Healthcare Information System is installed. Select the check box to define the Item as a Inventory Management Item.

  11. CSSD Component – The check box is enabled if Sterile Stock module of Enterprise Healthcare Information System is installed. Select the check box to define the Item as a Sterile Stock Item.

  12. Material Group – Select the required Material Group from the list. The Material Groups are defined in ‘Material Group’ function. The enabled Material Groups are displayed in the list. This is a mandatory field.

  13. Item Class – Select the required Item class from the list. All the Item Classes with the enabled box selected in the ‘Item Class’ function is displayed in the list. This is a mandatory field.

  14. Unit Cost – The Unit Cost of the Item is derived by the Inventory Management module Transactions and displayed in Unit Cost field. This field is disabled and the values cannot be edited.

  15. UOM – Select the required UOM value from the list. The UOM values defined in the ‘Application Masters’ module are listed. The UOM with ‘Stock’ check box selected are applicable for Inventory Items.

  16. Shelf Life – Enter the shelf life time duration of the item and select the values of Days, Months or Years as the UOM options.  The default shelf life UOM is ‘Months’.

Entering the Shelf Life is optional. But if you enter the shelf life, it is mandatory to choose a UOM value from the time-period drop-down. You can enter only numeric value greater than zero as shelf-life.  The system calculates the Shelf-Life Breach Date by adding the Shelf-Life duration to the Goods Received Note (GRN) Date. If the Shelf-Life Breach Date is lesser than the Expiry Date of the batch of the item received, an alert message is displayed while raising the GRN. You can proceed with the GRN after acknowledging the alert message.

As the storage conditions affect the shelf-life of the items, the system records the ‘Refrigeration Information’ and ‘Disposable Remarks’ of the item in the ‘Item Remarks’ since special care might be necessary while disposing certain types of items. You can view the ‘Refrigeration Information’ and ‘Disposable Remarks’ of the item using the Information Icon  displayed next to the Item Name.

  1. Package Item - Select the check box to include the Item as part of the Package or clear the check box to exclude the Item from the Package. The check box is selected, by default. You can modify the Package Item attributes in the Query mode.

Package is a collection of pre-defined set of services and consultations with a standard price offered by the hospital.

  1. Billable Item – Select the check box, if the Item is chargeable to the patient.

  2. Enabled – Select the check box to enable the defined Item. Clear the check box, to disable the defined Item. The check box is selected, by default. The enabled Items are available for selection in the required functions.

  3. Direct Sales Allowed – Select the check box, if the Item is applicable for Direct Sales.

  4. Sole Source – Select the check box to indicate, if the item is from a single Manufacturer.

  5. Manufacturer – This field is enabled if the ‘Sole Source’ check box is selected. You can select the applicable Manufacturer from the list. The ‘Manufacturer’s’ defined in the Application Masters module are displayed in the list. This is a mandatory field.

  6. Special Item - Select the check box to indicate the Item is a Special Item.

  7. Supplier - Enter the Supplier name to be associated to the Item or select the Supplier using [?] function.

  8. Item Analysis 1 – Select the required value from the list. All the records from Item Analysis master is displayed in the list.

  9. Item Analysis 2 – Select the required value from the list. All the records from Item Analysis master is displayed in the list.

  10. Item Analysis 3 – Select the required value from the list of valid values. All the records from Item Analysis master is displayed in the list.

  11. Imported Item – Select the check box, if the selected Item is brought in from an external source.

  12. Reimburse YN – Select the check box to issue refund towards the selected Item in the respective Facility. If the check box is cleared, no refund is issued towards the service.

  13. Essential Item – Select the check box, if the selected Item is brought in as an essential item in the respective Facility.

  14. Reimburse Type - Select one of the options from the list:

  15. Non Reimbursable - Select the value ‘Not Reimbursable’, if no refund is issued towards the service amount.

  16. Partial Reimbursable – Select the value ‘Partial Reimbursable’, if only a fraction of the service amount is issued as a refund.

  17. Fully Reimbursable – Select the value ‘Fully Reimbursable’, if full amount charged towards the service is refunded.

  1. Country of Origin – Select the Country of Origin from the list. By default, ‘Select’ is displayed. The Country of Origin is captured to identify the country in which the drug is manufactured.
