Enter the following search criteria:

 

  1. Search By – Select the search by option and search either by Item Code, Item Name, Item Remarks, Alpha Code, Trade ID or Trade Name.

  2. Search String – Enter the search string in the alphanumeric format of 60 characters maximum length. The search is based on the string entered.

  3. Category Type – Select the required value for category from the list. The value ‘Inventory’ is displayed, by default.

  4. Special Item – Select the option provided in the list. The listed options are ‘All’, ‘Yes’ and ‘No’. If ‘All’ is selected, all the special items can be queried. If ‘Yes’ is selected, the Special Items are included in the query search and if ‘No’ is selected, the Special Items are not selected in the query search.

  5. Sole Source - The Items placed or received from a single manufacturer is called Sole source. Select the required option from the list. The listed options are ‘All’, ‘Sole Source’ and ‘Non Sole Source’. If ‘All’ is selected, all the manufacturers are displayed.

  6. Item Class – Select the item class from the list. The listed values are the Item Classes defined in the Item Class Master.

  7. Pharmacy Stock Item - Select the required value from the list. The list of valid values includes ‘All’, ‘PS Item’ and ‘Non PS Item’. The value ‘All’ is displayed, by default.

  8. Manufacturer – Select the Manufacturer from the list. Depending on the manufacturer selected the items belonging to the selected manufacturer is queried.

  9. Material Group – Select the Material Group from the list. The Materials Group defined in the Material Group function are displayed.

  10. Search Criteria – Select the search criteria from the search string. The listed values are ‘Starts With’, ‘Contains’ and ‘Ends With’. This helps in tapering the search. Depending on the search string entered and the search criteria selected, the search results are displayed.

  11. Item Analysis 1-3 – Select all the item analysis from the list. The listed values are the item analysis that are defined in the Item Analysis master code.

  12. Nature – Select the nature from the list. The listed values are ‘Enabled’, ‘Disabled’ and ‘All’. The value ‘Both’ is displayed, by default.
