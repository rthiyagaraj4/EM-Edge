Enter the following details to create Item UOM Definition records:

 

  1. Item – Enter the Item description in alphanumeric format of 60 characters maximum length or click [?] function to select the desired Item. The Items created, which are not a Sole source type are displayed. This is a mandatory field.

  2. Base UOM Quantity - Enter the Base UOM Quantity in numeric format of six digits maximum length. This is a mandatory field.

  3. Base UOM Code – On selecting the Item, the Base UOM code associated to the Item is displayed and disabled. The UOM with ‘Stock’ check box selected is applicable for Inventory Items. This is a mandatory field.

  4. Equivalent UOM Quantity - Enter the Equivalent UOM Quantity in numeric format of eight digits maximum length. This is a mandatory field.

  5. Equivalent UOM Code - Select the Equivalent UOM code from the list. The Equivalent UOM codes defined in the Application Masters module are listed in the list. This is a mandatory field.

  6. Enabled – Select the check box to enable the defined Item UOM Definition. Clear the check box to disable the defined Item Definition. The check box is selected, by default. The enabled Item UOM Definitions are available for selection in the required functions.
