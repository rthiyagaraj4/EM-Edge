Transaction Remarks

From this function you can Define the Transaction Remarks and Query for the desired Transaction Remarks.

The defined Remarks are used for Adjustment Remarks, Split Batch and Condemn Expiry Batches. You can define 10 Transaction Remarks codes at a time.

Defining a Transaction Remark

From the Materials Setup, select the Transaction Remarks function.

The Transaction Remarks screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Enter the details in the mandatory fields. <Related Field Description>

  3. Click [Apply] to save the Transaction Remarks created or [Reset] to clear the entered data.

  4. Click [Query] to search for existing Transaction Remarks.
