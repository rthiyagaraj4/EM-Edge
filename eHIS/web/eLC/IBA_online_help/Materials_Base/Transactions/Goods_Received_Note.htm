Goods Received Note

From this function you can Record the details of Items received from a Supplier and Query for the unauthorized Item received details. You can Delete the details of received Item records.

The GRN details are recorded whenever an Item is received from a Supplier, which includes what goods and quantities have been received. The Items include both Medical and Non-Medical Items. Once the GRN document is finalized, it cannot be modified.

 

If the check box ‘Interface to External PO’ in ST Entity Parameter is selected, [Create] will not appear. Only the query mode is applicable in this function. You cannot add any new batches or new Item to the document number selected. You are able to modify the quantity at item and batch level. The PO Number and the PO Document Type code get displayed on the screen.

 

If the 'Interface to External PO’ check box in ST Entity Parameter is cleared, [Create] gets displayed in this function and you can process the transaction as usual. However, the PO Number is disabled and the PO Document Type code does not appear on the screen.

Recording the Items Received details

From the MM Transactions, select the Goods Received Note function.

The Goods Received Note screen gets displayed.

  1. Click [Create] to invoke the Create screen.

  2. Modify Document Details dialog is displayed, by default or click ‘Document’ hyperlink to modify the document details.

    The following details are displayed in the Document section.

     

    1. Document Type – The default Document Type as set in ‘ST Facility Parameter’ is displayed. Select the Document Type from the list. The documents associated to the Transaction ‘Goods Receipt’ are displayed in the list. This is a mandatory field.

    2. Doc No. - The Document Number generation depends on the selected Document Type. You are informed if the Document Number has to be manually generated for the selected Document Type. Enter the Document Number in the numeric format of eight digits maximum length. If the system generates the number, you are not allowed to enter the field value. This is a mandatory field.

    3. Doc Date - The system date is displayed, by default. You can change the Document Date by clicking the Calendar function or enter the date in the format of dd/mm/yyyy. You can modify the date, if the ‘Doc Date Modifiable Y/N’ check box is selected in ST Transaction Type function. This is a mandatory field.

    4. Doc Ref – Enter the Document Reference in alphanumeric format of 20 characters maximum length.

    5. Store – The default Store as set in ‘ST Facility Parameter’ is displayed. Select the Store from the list. Only those Stores for which the User has access and if ‘Goods Receipt’ Transaction function is applicable are displayed. This is a mandatory field.

    6. Item Class – Select the Item Class from the list. The listed values are the Item Classes defined in the Code Setup.

    You can modify the document details by clicking on the ‘Document’ hyperlink.

  3. On entering the Document Details, click [OK] to record the Item received details or [Reset] to clear the entered data.

  4. Enter the details in the mandatory (*) fields. < Related Field Description>

  5. Click [Add] to add Items to the GRN or [Cancel] to cancel the selected Item details.

  6. The selected Item received details are displayed in the Item Details frame on the left hand side. Click ‘Item Details’ hyperlink to modify the Item received details. Click [Modify] to modify the entered details or [Cancel] to cancel the modified changes.

  7. Click [Preview] to preview the selected Item details. On adding the Item Received details, [Preview] is enabled. The details entered while selecting the Item in Goods Received Note is displayed. Click [X] to close the GRN Preview Details screen.

  8. Click [Batch Search] to view the stock position of the item with batch details and bin location.

  9. You can select the ‘Delete’ check box to delete the selected record.

  10. Click [Apply] to finalize the GRN details entry or [Reset] to clear the entered data.

The system calculates the Shelf-Life Breach Date by adding the Shelf-Life duration to the GRN Date. If the Shelf-Life Breach Date is lesser than the Expiry Date of the batch of the item received, an alert message is displayed while raising the GRN. You can proceed with the GRN after acknowledging the alert message.

  1. The ‘Confirm Finalize’ dialog is displayed. Click [Yes] to finalize the GRN details entry or [No] to finalize the GRN details entry in the query mode.

  2. The ‘Confirm Printing’ dialog is displayed. Click [Yes] to print the GRN details or [No] to cancel the action.

  3. Click [Query] to invoke the query screen.

  4. The unauthorized Item received details are displayed based on the default criteria or enter the search criteria and click [Search] to select a specific record.

    Enter the following search criteria:

     

    1. Doc Type – The default Document Type as set in ‘ST Facility Parameter’ is displayed. Select the Document Type from the list. The documents associated to the Transaction ‘Goods Receipt’ are displayed in the list.

    2. Doc No – Enter the Document Number in alphanumeric format of eight digits maximum length.

    3. Doc Date – Enter the document generation date in dd/mm/yyyy format or select the required date using [?] function.

    4. Store – The default Store as set in ‘ST Facility Parameter’ is displayed. Select the Store from the list. Only those Stores are displayed for which the User has access and if ‘Goods Receipt’ Transaction function is applicable.

    5. Supplier – Select the supplier from the list.

    6. Entry Completed – Select the Entry Completed value from the list. The listed values are ‘All’, ‘Completed’ and ‘Not Completed’. The value ‘Completed’ is displayed by default.

  5. On searching, the unauthorized Item received details are displayed.

  6. Click ‘Doc Type’ hyperlink to authorize the Item received details.

  7. Select the ‘Entry Completed’ check box and click [Apply] to finalize the GRN entry or [Delete] to delete the selected record.

  8. On selecting the ‘Delete’ check box, ‘Confirm Delete’ dialog is displayed. Click [Yes] to delete the record or [No] to cancel the action.

If the selected Item in a Store is a Non-Stock Item, the Item received details cannot be recorded.

GRN Transactions that are not finalized can only be deleted.
