Document Details

Enter the following details in the document section:

  1. Doc Type – Select the required Document Type from the list. The documents associated to the selected transaction are displayed in the list. The Document Type, as set in ‘ST Facility Parameter’ is displayed, by default. This is a mandatory field.

  2. Doc No. - The Document Number generation depends on the selected Document Type. If the Document Number has to be manually generated for the selected Document Type, you are informed. Enter the Document Number in numeric format of eight digits maximum length. If the system generates the number, you are not allowed to enter the field value. This is a mandatory field.

  3. Doc Date - The system date is displayed, by default. You can change the Document Date by entering the date in date format of dd/mm/yyyy. You can modify the date only if the ‘Doc Date Modifiable Y/N’ check box is selected in ST Transaction Type for the selected transaction. This is a mandatory field.

  4. Doc Ref – Enter the Document Reference in alphanumeric format of 15 characters maximum length.

  5. From Store – Select the ‘From Store’ from the list. The Stores for which you have access and selected transaction function applicable are displayed. This is a mandatory field.

  6. To Store – Select the ‘To Store’ from the list. The Stores for which you have access and selected transaction function applicable are displayed. This is a mandatory field.

  7. Item Class – Select the required Item Class from the list.

On entering the Document Details, click [OK] to record Item Return details or [Reset] to clear the entered data.
