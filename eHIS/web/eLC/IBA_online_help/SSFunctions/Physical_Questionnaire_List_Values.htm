Physical/Questionnaire List Values

From this function you can Create and Update the list of values for Physical checkup and Questionnaire. You can also Query for the existing records.

The Physical/Questionnaire List Values function is used to create list of values for Physical Checkup or Questionnaire forms. They are questions and answers to be selected based on which the donor gets accepted or rejected during the analysis. The list of values is printed for the Pre Donation Analysis performed in the Pre Donation function.

Creating Physical/Questionnaire List Values

From the Administration, double-click Physical/Questionnaire List Values function.

The Physical/Questionnaire List Values screen gets displayed.

  1. Select the value from the list to specify whether the created list of values are valid for Physical Checkup, Questionnaire or Both.

  2. Enter the list of values in alphanumeric format of 30 characters maximum length.

  3. Click [Commit Form] or press [F10] to save or click [Clear Record] to clear the entered details.

  4. Click [Execute Query] or press [F8] to search for the existing records.
