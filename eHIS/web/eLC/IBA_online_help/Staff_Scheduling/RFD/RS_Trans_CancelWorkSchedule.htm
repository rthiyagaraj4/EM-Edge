Enter the following details:

 

  1. Location type - Select the Location Type from the list. The Location Types displayed in the list are defined in the "Location Type" function of Application Masters module.

  2. Workplace - Enter the workplace or select the workplace using [?] function. If the Location Type is selected, workplaces listed in the [?] function corresponds to the selected location type or all the workplaces created in the "Workplace" function are displayed.

  3. Role Type - Select the role type as 'Practitioner' or 'Other Staff' from the list.

  4. Staff Type - Select the Staff type from the list. The 'Staff Type' displayed is based on the 'Role type' selected. If Role Type is selected as Practitioner, the list displays the Practitioner Type as defined in the Application Masters module. If Role Type is selected as Other Staff, the list displays the Other Staff Type as defined in the Application Masters module.

  5. Position - Enter the position or click the [?] function to search for the required position. If the 'Staff Type' is selected, the positions listed in the [?] function corresponds to the selected Staff Type or the [?] function displays all the positions defined in Position function in System Configuration module.

  6. Staff - Select the required Staff using [?] function.

  7. Cancel Schedule (From/To) - Enter the date range to cancel schedule in dd/mm/yyyy format or select the date using the calendar function. The 'From' date corresponds to the start date of the schedule to be cancelled and To date corresponds to the end date when the schedule will be cancelled. The 'From' date should be greater than or equal to the system date. This is a mandatory field.

  8. Reason for cancellation - Select a valid reason from the list. The reasons displayed in the list are defined in the 'Reason' function for Cancel Schedule. This is a mandatory field.
