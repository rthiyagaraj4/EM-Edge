Change

To change the shift of a staff within the workplace, select [Change] from the menu.

 

The following details are displayed:

  1. Workplace - The current workplace is displayed.

  2. Shift - The shifts scheduled to the staff is displayed.

  3. Start Time - The start time and date of the shift is displayed.

  4. End Time - The end time and date of the shift is displayed.

 

Enter the following details to change a shift within the workplace:

  1. Schedule Date - The scheduled date is displayed.

  2. Allocated Shifts - The shifts scheduled to the staff is displayed.

  3. Select - Check the radio button to indicate the selection of the required shift. This is a mandatory field.

  4. New Shift - Select the new shift from the list box. On selecting the required shift, its shift indicator appears as Productive (Working/On-call) or Unproductive in read only text. This is a mandatory field.

  5. Reason - Select a valid reason from the list. The reason is defined in the "Reason" function for Change Shift. This is a mandatory field.

  6. Remarks - Enter the remarks if required.

If the Maximum continuous working hours of the staff is already covered by the existing shifts, as set in the parameters then the system will not allow you to change the shift.

  1. Click [Apply] to save the record. The system prompts you before finalizing the action. Click [OK] to confirm the action.
