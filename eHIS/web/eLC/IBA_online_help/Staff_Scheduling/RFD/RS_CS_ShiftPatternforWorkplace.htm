Enter the following details:

 

'By Workplace' Tab

  1. Location Type - Select the required Location Type from the list. The listed values are defined in 'Location Type' function of Application Masters module.

  2. Workplace - Enter the Workplace in alphanumeric format of 30 characters maximum length or select the workplace using [?] function. If the Location Type is selected then workplaces listed in the [?] function corresponds to the selected location type, else all the workplaces created in the 'Workplace' function are displayed. This is a mandatory field.

  3. Work cycle - The Work Cycle is displayed as read only text based on the value set for 'Work Cycle Basis' in parameter setup.

  4. Click [Show Shift Pattern].

  5. Select - Select the check box to indicate applicability of the shift pattern for the selected workplace. To select all the displayed shifts patterns, select the check box provided in the column header.

  6. Location Type - The selected Location Type is displayed. If the Location type is not selected then the Location type associated to the selected workplace in 'Workplace' function is displayed along with the General Location Types. Location Types are displayed as General as some of the shift patterns created in the 'Shift Pattern' function are not associated to any specific location types. If the selected workplace is also not associated to any location type then all the location types associated to shift patterns in the 'Shift Pattern' function along with the General location types are displayed.

  7. Shift Pattern - The shift pattern records created in the 'Shift Pattern' function associated to the selected Location type along with the shift patterns not associated to any location types are displayed. If the Location type is not selected, then the shift patterns associated to the Location type of the selected workplace is displayed along with the General shift patterns. If the selected Workplace is also not associated to any location type in the 'Workplace' function then the entire shift patterns created in the 'Shift Pattern' function are displayed.

 

'By Shift Pattern' tab

  1. Location type - Select the required location type from the list. The listed values are defined in 'Location Type' function of Application Masters module.

  2. Shift pattern - Enter the Shift Pattern in alphanumeric format of 10 characters maximum length or select the shift pattern using the [?] function. If the Location Type is selected, the [?] function displays only the shift patterns that are associated to the location type in 'Shift Pattern' function, else all the shift patterns created in the 'Shift pattern' function is displayed. This is a mandatory field.

  3. Click [Show Workplaces].

  4. Location Type - The selected Location Type is displayed along with the general location types. The location types are displayed as general as some of the workplaces created in the 'Workplace' function are not associated to any specific location types. Those workplaces will display the Location Type as General. If the location type is not selected from the list box in this function, then the location type associated to the selected shift pattern in the 'Shift Pattern' function along with the general location types are displayed. If the Shift Pattern is also not associated to any Location type in the 'Shift Pattern' function, then all the location types associated to the workplaces in the 'workplace' function along with the general location types are displayed.

  5. Workplace - The workplaces associated to the selected location type in the 'Workplace' function is displayed along with those workplaces that are not associated to any specific location type. If the Location type is not selected, the workplaces associated to the Location type of the selected shift pattern is displayed along with the General workplaces. If the selected shift pattern is not associated to any location type in the 'Shift Pattern' function then all the Workplaces created in the 'Workplace' function are displayed.

  6. [From - To] - The operating duration of the workplace is displayed as set in the 'Workplace' function.

  7. Select - Select the check box to indicate applicability of the workplace for the selected shift pattern. To select all the displayed workplaces, select the check box provided in the column header.
