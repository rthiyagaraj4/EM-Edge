Record/Cancel Staff Unavailability

From this function you can Record or Cancel the unavailability of staff for a given period by specifying a reason.

In this function, the user can record or cancel leave for a staff for period of time. The Leave option in the function 'Manage Work Schedule' allows recording and cancelling of leave for the current and future date only.

Record/Cancel Staff Unavailability

From the RS Transactions, select Record/Cancel Staff Unavailability.

The Record / Cancel Staff Unavailability screen gets displayed.

 

To Record

  1. Select the required staff using [?] function. The Staff Name is displayed as read only text. The Role Type of the selected staff is displayed as 'Practitioner' or 'Other Staff' as read only text. The Position and Staff Type is also displayed.

  2. Enter the Leave Period in dd/mm/yyyy format. From and To date corresponds to the start and end date of the leave period. From date should be greater than or equal to the system date and To date should be greater than or equal to the From date. This is a mandatory field.

  3. Select the required Reason from the list. The listed reasons are defined in the 'Reason' function for unavailability.

  4. Click [Apply] to record staff unavailability or [Reset] to clear the entered data.

To Cancel

  1. Select the required staff using [?] function. The Staff Name is displayed as read only text. The Role Type of the selected Staff is displayed as 'Practitioner' or 'Other Staff' as read only text. The Position and Staff Type is also displayed.

  2. Enter the Leave Period in dd/mm/yyyy format. From and To date corresponds to the start and end date of the leave period. From date should be greater than or equal to the system date and To date should be greater than or equal to the From date. This is a mandatory field.

  3. The date(s) for which the leave is recorded is displayed in the bottom part of the screen

  4. The date(s) for which the leave is recorded by the staff is displayed.

  5. Select the Reason for Cancelling Leave from the list. The listed values are defined in 'Reason' function if 'Applicable To' is selected as 'Cancel Availability'. This check box is enabled only if Cancel check box is selected.  

  6. Select the check box to cancel the leave for a particular day or select the Cancel check box in the column header to cancel the leave for all the dates.

  7. Click [Apply] to cancel leave or [Reset] to clear the entered data.
