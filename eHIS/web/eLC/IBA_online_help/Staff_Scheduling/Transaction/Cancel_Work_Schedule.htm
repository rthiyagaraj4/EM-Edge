Cancel Work Schedule

From this function, you can Cancel a Work Schedule.

This function allows you to cancel work schedule for a given period on specifying a reason. To perform this activity you should have appropriate access rights as defined in the function "Access Rights" function. The Cancel option in the "Manage Work Schedule" function allows cancellation of shift for the current and future dates only.

Canceling Work Schedule

From RS Transactions, select Cancel Work Schedule function.

The Cancel Work Schedule screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Show Details].

  3. Select the check box to indicate the selection of the staff to cancel work schedule for the specified date range. All the check boxes are selected, by default.

  4. Click [Apply] to Cancel the work schedule.
