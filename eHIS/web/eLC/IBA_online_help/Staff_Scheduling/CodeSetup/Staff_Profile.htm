Staff Profile

From this function you can Create various Staff Profiles and Query for existing Staff Profile records.

This function is used to create and maintain the staff profile. You can refer to the profile of the employee for various purposes. Following are the additional attributes that are linked to create a staff profile.

Skills

Experience

Training

Licensure

When the latest enterprise related position of a staff is recorded, the system should update the 'Job Title' against the corresponding Practitioner/Other Staff setup. The system performs validations to avoid wrong or overlapping entries at suitable stages. The information from functions like 'Skill', 'Skill for Position', 'Training', 'Grade' and 'Licensure' is used to create the profile of the staff.

Creating Staff Profile

From the RS Code Setup, select Staff Profile function.

The Staff Profile screen gets displayed.

  1. Select the required Role Type as 'Practitioner' or 'Other Staff' from the list.  

  2. Select the Staff name using the [?] function. This is a mandatory field.

Skills

  1. Click [Skills] to record the skills of the Staff.

  2. Skill - Enter the Skill or select using the [?] function. The Skills defined in the 'Skills' function are displayed. This is a mandatory field.

  3. Proficiency Level - Select the Proficiency Level from the list. Proficiency level is used for rating the skills with 1 as the lowest proficiency level and 9 as the highest proficiency level. This is a mandatory field.

  4. Click [Record] to save the record. The Skills recorded are displayed on the top part of the screen. Click Skill hyperlink to modify proficiency level.

  5. Click [Delete] to delete the selected record. Click [Clear] to clear the data.

Experience

  1. Click [Experience] to record experience details of the staff until current date.

  2. Position - Enter the position in alphanumeric format of 60 characters maximum length or select the position using [?] function. Multiple current positions for the same staff cannot be created. This is a mandatory field.

  3. Facility Type - Select the required Facility Type from the list as 'Enterprise' or 'External'.

  4. Facility - Select the Facility from the list if the 'Facility Type' is selected as 'Enterprise'. The listed values are defined in the 'Facility' function of System Manager module. If the 'Facility Type' is selected as 'External', the list automatically disables and the name of the external facility can be entered in the textbox provided. This is a mandatory field.

  5. Date [From-To] - Enter the effective date range of experience period of the staff in dd/mm/yyyy format or select the date range using the calendar function. This is a mandatory field. If the 'To' date is not entered, it implies that the staff is currently working in the facility for a position and multiple entries for the same position and date range cannot be made until the existing record is deleted.

  6. Location Type - Select the Location Type from the list. The listed values are defined in 'Location Type' function of Application Masters module.

  7. Workplace - Select the Workplace from the list if the 'Facility Type' is selected as 'Enterprise'. The workplace displayed is based on the Location Type selected. If the 'Facility Type' is selected as 'External', the list automatically disables and the description of the workplace can be entered in the textbox provided. This is a mandatory field.

  8. Grade - Select the grade using [?] function. The listed values are defined in the 'Grade' function. This is a mandatory field.

  9. FTE - Enter the 'full time equivalent' in numeric format of three characters maximum length. The FTE is recorded as a decimal value between 0 and 1. This is a mandatory field.

  10. Remarks - Enter the remarks in the alphanumeric format of 4000 characters maximum length.

  11. Click [Record] to save the experience details.

  12. The recorded details are displayed in the top part of the screen. Click Position hyperlink to modify the experience details. Click Remarks hyperlink to view the recorded remarks. Click [OK] to exit from the screen.

  13. Click [Delete] to delete the selected record. Click [Clear] to clear the data.

Training

  1. Click [Training] to record the training history of the staff.

  2. Date [From-To] - Enter the effective date range of the training period already undergone by the staff in dd/mm/yyyy format or select the date range using the calendar function. This is a mandatory field.

  3. Training program - Select the training program undergone by the staff using [?] function. The [?] function displays all the training programs created in the 'Training' function. This is a mandatory field.

  4. Training type - Select the type of training undergone by the staff from the list. The listed values are 'In-house', 'Local' and 'Overseas'. In-house trainings refer to trainings conducted at the enterprise level. Local trainings refer to trainings conducted within the country. Overseas trainings refer to trainings conducted outside the country.

  5. Organization - Select the organization where the training was conducted from the list if the 'training type' is selected as 'In-house'. The listed values are defined in the 'Facility' function of System Manager Module. This is a mandatory field. If the 'training type' is selected as 'Local' or 'Overseas', the list automatically disables and the name of the organization can be entered in the text box provided.

  6. Certified - Select the check box if the training undergone by the staff is certified.

  7. Remarks - Enter the remarks in the alphanumeric format of 4000 characters maximum length.

  8. Click [Record] to save the training details.

  9. The recorded details are displayed in the top part of the screen. Click Training Program hyperlink to modify the training details. Click Remarks hyperlink to view the recorded remarks. Click [OK] to exit from the screen.

  10. Click [Delete] to delete the selected record. Click [Clear] to clear the data.

Licensure

  1. Click [Licensure] to record the license details of the staff.

  2. Title - Select the license using [?] function. The [?] function displays licenses created in the 'License' function. This is a mandatory field.

  3. ID - Enter a unique ID for the license in alphanumeric format of 20 characters maximum length. This is a mandatory field.

  4. License Type - Select the License Type of the staff from the list. The values displayed are 'In-house', 'Local' and 'Overseas'. In-house refers to license obtained at the enterprise level. Local refers to license obtained within the country. Overseas refers to license obtained outside the country.

  5. Organization - Select the Organization from where the license was obtained from the list if the 'license type' is selected as 'In-house'. The listed values are defined in the 'Facility' function of enterprise Configuration Module. This is a mandatory field. If the 'license type' is selected as 'Local' or 'Overseas', the list automatically disables and the name of the organization from where the license was acquired can be entered.

  6. Issue Date - Enter the Issue Date of the license in dd/mm/yyyy format or select the date of issue using the calendar function. This is a mandatory field.

  7. Valid until - Enter the expiry date of the license in dd/mm/yyyy format or select the date of expiry using the calendar function. If the license has a life long validity, leave the 'valid until' field unattended.

  8. Remarks - Enter the remarks in the alphanumeric format of 4000 characters maximum length.

  9. Click [Record] to save the licensure details.

  10. The recorded details are displayed in the top part of the screen. Click Title hyperlink to modify the License details. Click Remarks hyperlink to view the recorded remarks. Click [OK] to exit from the screen.

  11. Click [Delete] to delete the selected record. Click [Clear] to clear the data.
