Enter the following details to create a new Menu:

 

  1. ID - Enter the menu ID in alphanumeric format of 30 characters maximum length. A Menu ID is a unique value to identify the menu record. For a menu called ‘AM Reports’, the menu ID can be ‘AM_REPORTS’. The Menu ID has to be unique and cannot be changed after creation. This is a mandatory field.

  2. Name - Enter the menu Name in alphanumeric format of 60 characters maximum length. For example, enter ‘AM Reports’ as the name of the menu. The menu name is used for identification during menu construction. While the submenu names constructed at run time gets displayed as a part of the menu, the root menu name does not get displayed anywhere during run time. This is a mandatory field.

  3. Responsibility Group – Select the Responsibility Group from the list of values. The entire responsibility group created using [Create] in 'Responsibility Group' function gets populated in the list.

  4. Root Menu - Select the check box to classify the menu as a Root Menu. Clear the check box to classify the menu as a submenu. The root menus cannot be attached to any other type of menus, whereas the submenus can be attached to other submenus or root menus. The root menus can be linked to a responsibility, which are dynamically constructed at run time.

  5. Adhoc Menu – The check box gets enabled on selecting the Root menu check box. You can select this check box if the defined root menu has to be functional on a temporary basis for a specific period. On selecting the Adhoc Menu check box the Adhoc Menu Effective ‘From’ and ‘To’ date becomes mandatory.

Submenus cannot be defined as Adhoc Menu.

 

Adhoc Menu Effective Date

 

  1. From - Click the Calendar function to select a date or enter the ‘From Date’ in dd/mm/yyyy format. The entered 'From' date has to be greater than or equal to the system date. This is a mandatory field, if the menu is an Adhoc menu.

  2. To – Click the Calendar function to select a date or enter the ‘To Date’ in dd/mm/yyyy format. This is a mandatory field, if the menu is an Adhoc menu.

  3. Tool Used – Select the required Tool to be used from the list. The values are ‘Forms’ and ‘Java’. This is a mandatory field.

  4. Remarks - Enter the required Remarks about the menu in alphanumeric format of 200 characters maximum length.
