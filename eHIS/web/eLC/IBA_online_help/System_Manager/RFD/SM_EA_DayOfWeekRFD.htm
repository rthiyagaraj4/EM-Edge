Select the appropriate values to set the days of the week as a working day or a holiday:

 

  1. Day No. – Each day of the week is given a specific number and displayed. The Day Number cannot be modified.

  2. Day of Week – The Days of the week are displayed.

  3. Week Appl – Select the applicability for the week from the list. The pre-defined values are ‘All’ and ‘Specific’. If ‘All’ is selected, the selected value (‘W’ or ‘H’) is applicable for all the weeks of the month for the specific day.

  4. Default – Select the Default value for the whole week. The listed values are ‘W’ and ‘H’, which means a 'Working' and a 'Holiday' respectively. If the default value selected is ‘W’, all the weeks in the month are set as ‘W’, by default. This is displayed and set as default in the ‘Clinic’ function of Outpatient Management module.

  5. Week1 – Week5 – Select the appropriate value from the respective list. The listed values are ‘W’ and ‘H’.
