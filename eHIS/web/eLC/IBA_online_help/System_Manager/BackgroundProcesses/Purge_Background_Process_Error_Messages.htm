Purge Background Process Error Messages

From this function you can Delete the Error Messages from the database, which are generated by the Background Processes.

The Purge Background Process Error Messages function helps in clearing the Background Process Errors generated after execution of a process during a specified period.

Purging Background Process Error Messages

From the Background Processes, select the Purge Background Process Error Messages function.

The Purge Background Process Error Messages screen gets displayed.

  1. Select a Process from the list. The processes listed are Build Laboratory Statistics, Building HL7 Messages, Charge Recalculation Process, Clinical Repository Process, Creating HL7 events, Duplicate Merge Process, Duplicate Search Process and Loading HL7 Messages. This is a mandatory field.

  2. Click the Calendar function or enter the date in dd/mm/yyyy format to select an effective date range. The error messages generated by the background processes during the selected period are purged. This is a mandatory field.

  3. On entering the purge process details, click [Apply] to clear the background error messages or [Reset] to clear the entered data.
