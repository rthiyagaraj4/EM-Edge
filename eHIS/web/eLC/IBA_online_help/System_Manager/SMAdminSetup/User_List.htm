User List

From this function you can Generate a report, which gives a list of Users set up for the enterprise.

The User List function is used to generate a report on the status of the Users based on the range of values selected. This report also displays the reporting facility of the User, telephone number, fax number, e-mail ID and the effective date range.  

Generating Report on Users

From the SM Admin Setup > Reports, select the User List function.

The User List screen gets displayed.

  1. Enter the details in the mandatory (*) fields. <Related Field Description>

  2. Click [Run] to generate the report based on the entered report criteria or [Reset] to clear the entered data.

  3. Select Report Option to generate the report.

  4. Click [OK] to generate the report and spool it in the file on the print server or [Cancel] to cancel the action.

 

This report is generated only if Patient Billing Manager module is installed.
