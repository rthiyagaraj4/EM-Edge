Responsibility for Facility

From this function you can Associate a Responsibility to a Facility and Query for a desired Responsibility associated a Facility.

The system allows you to associate a responsibility to one or more facilities. For example, the responsibility ‘Blood Transfusion’ can be associated to two facilities, namely, ‘Get Well Clinic’ and ‘Good Health Clinic’. Each responsibility can be associated to multiple facilities and a facility can have multiple responsibilities. This association is used to determine the valid facilities for a selected responsibility.

Associating a Responsibility to a Facility

From the SM Admin Setup > Application Security, select the Responsibility for Facility function.

The Responsibility for Facility screen gets displayed.

  1. Click [Create] to invoke the Create screen.

The Responsibility for Facility is set under two Tabs, namely:

By Facility

By Responsibility

  1. Click [Apply] to save the record or [Reset] to clear the entered data.

  2. Click [Query] to search for an existing record.
