Facility Category

From this function you can Create a new Facility Category used to group a set of identical facilities. You can Query for a Facility Category record.

A Facility Category is used to group a set of identical facilities. The facility category can be a super-specialty, general or epidemic. For example, All super specialty hospitals within an enterprise can be grouped under a Facility Category 'Super Specialty'. A Facility Category is created depending on the bed strength, type of institution, specialization involved, level of care, etc. based on the user's requirement. Facility Category is associated to a facility while creating a facility.

Creating a Facility Category

From the SM Admin Setup > Enterprise Admin, select the Facility Category function.

The Facility Category screen gets displayed.

  1. Click [Create] to invoke the create screen.

  2. Enter the details in the mandatory (*) fields. <Related Field Description>

  3. Click [Apply] to save a created Facility Category or click [Reset] to clear the entered details.

  4. Click [Query] to search for the existing Facility Categories.
