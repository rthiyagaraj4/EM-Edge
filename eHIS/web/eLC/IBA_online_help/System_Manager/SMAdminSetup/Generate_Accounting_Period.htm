Generating Accounting Period

From this function you can Generate an Accounting year for a facility to generate the Profit and Loss account and Balance Sheet.

The Generate Accounting Period function is used to define an accounting year with the beginning date and the number of months for the respective financial year in the facility. For each accounting year, you can define the beginning date and the months to determine the financial year in the facility.

Generating An Accounting Period

From the SM Admin Setup > Financial Setup > select the Generate Accounting Period function.

The Generate Accounting Period screen gets displayed.

  1. Enter a year in numeric format of four digits maximum length for which the accounting period is to be generated. An error message is displayed if the accounting period has already been generated for the respective year. This is a mandatory field.

  2. Click Calendar function or enter the Start Date in dd/mm/yyyy format. The financial year begins with the date selected. This is a mandatory field.

  3. The accounting period is defaulted to 12 months and disabled. The accounting period for the selected year is calculated based on the start date and the number of years entered.

  4. Click [Apply] to generate an Accounting Period or [Reset] to clear the entered data.
