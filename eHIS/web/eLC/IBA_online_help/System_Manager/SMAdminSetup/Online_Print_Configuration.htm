Online Print Configuration

From this function you can Query the set of reports that are configured for online printing.

The Online Print Configuration is used to configure what reports will be printed online by default

Querying an Online Print Configuration

From the SM Admin Setup > Print Routing, select the Online Print Configuration function.

The Online Print Configuration screen gets displayed.

  1. Click [Query] to search for an existing configuration.

  2. Select a module name from the list. <Related Field Description>

  3. Click [Execute] to query for a configuration or all the configurations. The next screen displays the list of reports and their default printing configuration status.

  4. Click [Apply] to save the modifications or [Reset] to clear the data.
