Query on Users

From this function you can Query the list of responsibilities and the associated menu functions for a user based on the associated facilities.

The Query on Users function enables you to view responsibilities, facilities and the menu functions that are associated to the User. System administrator only can access this function.

Querying User records

From the SM Admin Setup > Queries > Query on Users, select the ‘Query on Users’ function.

The Query on Users screen gets displayed.

  1. Click [Query] to search for an existing User.  

  2. Click [?] function to select a User. This is a mandatory field.

  3. On selecting the User, the list of all the responsibilities and facilities that the User has access to is displayed.

  4. On selecting a responsibility from ‘Associated Responsibility’ and a facility from ‘Associated Facility’, the list of menu functions attached to the User are displayed in ‘Associated Menu Function’.

  5. Click [Reset] to clear the entered data.
