Enter the following details to approve or reject the request placed for material items:

 

  1. Exclude/Include Criteria – Select a value from the list. The value is system defined - 'Sale Category'.

  2. Item Code – Click [List of Values] to view and to select an Item Code. The Sale Categories defined using the function ‘Material Item Charging Rule Setup’ in the Patient Billing Manager module gets listed.

  3. Description – The description for the selected Item Code gets displayed.

  4. Approval Status – Select a value from the list. The system defined ‘Approved’, ‘Pending’ and ‘Rejected’.

  5. Approved – Select Approved to approve the request. The text boxes Approval Number, Request Date, Given Date and Remarks are enabled on selecting the Status as ‘Approved’.

  6. Pending – Select Pending and the material item is not associated to the selected patient encounter.

  7. Reject – Select Reject to reject the requests placed for the material items.

  1. Approval Number – Enter the approval number in alphanumeric format of 60 characters maximum length. This field is enabled to enter the approval number, only on selecting the ‘Approval Status’ as ‘Approved’.

  2. Request Date – Click Calendar function to record the date when the request for the material item was placed. This field is enabled to record the request date, only on selecting the ‘Approval Status’ as ‘Approved’. Request Date must not be more than the Given Date.

  3. Given Date – Click Calendar function to record the date when the request for the material item is approved. This field is enabled to record the date, only on selecting the ‘Approval Status’ as ‘Approved’.

  4. Remarks – Enter the remarks in alphanumeric format.
