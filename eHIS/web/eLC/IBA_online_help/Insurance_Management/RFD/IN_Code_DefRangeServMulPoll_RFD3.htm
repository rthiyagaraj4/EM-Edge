Query Criteria for Policy Types Selection

  1. Policy Types [From–To] - Click [List of Values] to select a range of Policies. Specify the criteria to search for the records matching the search criteria. Select either ‘Starts With’, ‘Contains’ and ‘Ends With’ from the list. Enter the search criteria in the adjacent text box.

  2. [Display Records] – Click [Display Records] to view the range of Policies between the selected ‘From’ and ‘To’ entries.

A confirmation message is displayed in red on clicking [Display Records] to search for the Policy Type records. The process status also gets displayed.

  1. Cust Group – The Customer Groups associated to the selected range of Policies get listed.

  2. Cust Code – The description for the respective Customer Group gets displayed.

  3. Policy Type Code – The Policy Types between the selected ‘From’ and ‘To’ policy range gets listed.

  4. Description - The description for the respective Policy Type gets displayed.

  5. All – Select the check box to either associate or dissociate the services from the selected range of policies or copy or delete the coverage limit for the selected services associated to the range of policies. To select individual records, select the check box adjacent to each record.

  6. Display Existing Services – Select the check box to display the Billing Services/Service Groups associated to the selected Policy Type.
