Enter the following details to record mapping code or recalculate charges:

Implants Used in Operations

  1. Oper Code - The Operation Code is displayed, if an operation is recorded for the patient. The details are displayed on selecting the Patient ID or the Admission/Transfer number.

  2. Description - The description of the selected Operation Code is displayed.

  3. Operation No - The operation number that is validated for the selected patient is displayed.

  4. Line No - The line number is displayed based on the operation code associated to the patient.

  5. Mapping Code - Click [List of Values] to select and associate a Mapping Code.

  6. Description - The description of the selected Mapping Code is displayed.

  7. Charge Qty - Enter the quantity of Mapping Code in numeric format. The default charge quantity is mandatory.

  8. Level - The level associated to the Mapping Code during definition through the 'Service/Material Item Mapping' function is displayed.

  9. Accessory Type -  The Implants associated to the Mapping Code is displayed.

  10. Accessory Code - The Accessory Code associated to the selected Mapping Code is displayed.

  11. Description - The description of the selected Accessory Code is displayed.

  12. Issue Qty - The quantity issued against the implants selected are displayed.

 

Charges Distribution

  1. Blng Group - The Billing Group associated to the patient during patient registration is displayed.

  2. Payer - The Payer associated to the patient, if any, is displayed.

  3. Policy Type - The Policy associated to the patient is displayed.

  4. Gross Amt - The Gross Amount for the Implants associated to the Mapping Code is displayed.

  5. Disc Amt - The Discount Amount defined for the selected Payer is displayed.

  6. Net Amt - The net amount is displayed as the difference of the [Gross Amount - Discount Amount].

  7. Serv Qty - The quantity of Material Items that are associated to the patient is displayed.

  8. Mapping Code - The Mapping Code associated to the selected Material Items are displayed.

  9. Description - The description of the Mapping Code is displayed.

  10. Charge Qty - The quantity level recorded for the Billing Group is displayed.

  11. Level - The level captured for the Mapping Code through the 'Service/Material Item Mapping' function is displayed.

  12. Total - The sum of gross amount, net amount and the discount amount are displayed under relevant columns for the Material Items associated to the patient.

 

Other Items

  1. Item Code - The Material Item associated to the Patient for the respective admission is displayed.

  2. Description - The description of the associated  Material Item is displayed.

  3. Served Qty - The item quantity associated to the Patient for the respective admission is displayed.

  4. Physician - The Physician ID captured while placing the order is displayed.

  5. Mapping Code - Click [List of Values] to select the Mapping Code. The Mapping Codes that are associated to the selected Item Code are displayed.

  6. Description - The description of the selected Mapping Code is displayed.

  7. Charge Qty - Enter the quantity of the Material Items in numeric format. This is a mandatory field.

  8. Term Set ID - The diagnosis code recorded for the patient, if any, is displayed. System computes the normal coverage amount if the diagnosis is not recorded.

  9. Term Code - The term code recorded for the patient, if any, is displayed.

  10. Blng Group - The Billing Group associated to the patient is displayed.

  11. Payer - The Payer associated to the patient, if any, is displayed.

  12. Gross Amt - The Gross Amount for the associated Material Item is displayed.

  13. Disc Amt - The discount amount defined for the respective Payer is displayed.

  14. Net Amt - The Net amount is displayed as the difference of [Gross Amount - Discount Amount].

  15. Serv Qty - The quantity of Material Items associated to the patient are displayed.

  16. Mapping Code - The Mapping Code associated to the Material Item is displayed.

  17. Description - The description of the Mapping Code is displayed.

  18. Charge Qty - The quantity of the Material Item associated to the Billing Group is displayed.

  19. Term Set ID - The Diagnosis code recorded for the patient is displayed.

  20. Term Code - The term code associated to the diagnosis that is recorded is displayed.

  21. Total - The sum of gross amount, discount amount and net amount are displayed under relevant columns.

  22. Payer Name - The description of the selected Payer is displayed.

  23. Policy - The description of the Policy associated to the Payer is displayed.
