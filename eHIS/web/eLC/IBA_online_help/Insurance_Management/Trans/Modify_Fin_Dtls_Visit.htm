Modify Financial Details – Visit

From this function you can Update the financial details of an Outpatient, External or Emergency patient. The financial details are updated and considered only for the visit registration. You can Query to view the financial details of the Outpatient, External or Emergency patients.

Modify Financial Details – Visit function is used to update the financial details of an outpatient, external or an emergency patient. The financial details recorded during visit registration are updated. The visit registration charges are calculated based on the updated financial details recorded. Authorized User can update the financial details recorded during visit registration in the facility. The check box ‘Fin. Dtls Change’ has to be selected in the function User Privileges and Cash Counter Access under the Administration menu in the Patient Billing Manager module for the User to login and to modify the details.

Modify Financial Details – Visit

From the Transactions, double-click Modify Financial Details – Visit Function.

The Modify Financial Details – Visit screen gets displayed.

  1. Click [List of Values] to select an Episode Number. The patient ID, name, age, gender, patient registration date and time gets displayed on entering the episode number.

OR

  1. Click [Execute Query] or press [F8] to view all the patient details. Press downward arrow key to navigate to the next record and upward arrow key to navigate to the previous record.

  1. Click [Modify] to login and to update the details. The financial details associated to the patient get displayed. <Related Field Description>

    Enter the following details to login and to modify the financial details:

    1. User ID - Enter the User ID in alphanumeric format. This is a mandatory field.

    2. Pin No - Enter the Pin Number for the respective User. The Pin Number is recorded while creating the User in the System Manager module. This is a mandatory field.

    3. Remarks - Enter the Remarks in alphanumeric format. This is a mandatory field.

  2. Update the details and click [Commit Form] to save the changes or [Clear Record] to clear the details.

  3. Click [Close] to exit from the screen.
